江南雪

作者：沙鸥

皑皑白雪北疆漫，飘到江南又一年。

淡抹浓妆皆美景，天公试比看谁妍？

雪

作者：沙鸥

飞雪狂风同舞蹈，腊梅茶道共香飘。

迎春五彩银装缀，暮色夕霞分外娆。

暗香

作者：沙鸥

千古暗香花独放，凛然不畏雪堆枝。

寒宵同作罗浮梦，孤月清风寄墨诗。

梅

作者：沙鸥

影瘦孤枝独自栽，一花绽后百花开。

傲霜铮骨迷香冷，待等春风送醉来。

/

咏梅

作者：雨载鹰

寂寞深居苦自磨，丹腮玉蕊有谁呵？

芬香褪尽花飞去，冷夜芳心月照柯。

咏梅

作者：雨载鹰

寂寞寒梅志不降，孤将雪夜一肩扛。

春风若解琼英意，早把花声报老桩。

咏雪

作者：雨载鹰

玉屑拍肩心快哉，梨花谁见冷冬开。

扬扬洒洒轻飘絮，却是如期应约来。

咏雪

作者：雨载鹰

寒风止步日初晴，一片苍茫雪扎营。

放眼神州无限美，江山竞是玉铺成。

/

雪馥梅

作者：沧海一声笑

万木萧条腊月天，红梅一树报新年。

枝头朵朵皆春梦，尽把香思付雪前。

雪

作者：沧海一声笑

北风呼啸破天来，千里银花遍地开。

玉洁无需争沃土，人间处处似瑶台。

踏雪寻梅

作者：沧海一声笑

一夜银花遍地开，寒香约我故园来。

深深浅浅因酬梦，不为清高登月台。

梅雪吟

作者：沧海一声笑

有雪无梅尽是愁，有梅无雪空悠悠。

梅开雪怒情方好，雪覆梅花韵自流。


​

咏雪

作者：流星雨

仙女散花飘玉帐，嫦娥引线锈银袍。

真颜素雅山河壮，傲骨豪情展节操。

咏雪

作者：流星雨

瑞雪纷纷诗入画，洒铺满地洁莹花。

轻歌漫舞怀豪志，施展雄威绣锦华。

早春雪景

作者：流星雨

寒风冻雨逞雄威，燕雀归巢蝶止飞。

碧柳苍松披白絮，桃花失意竹枝肥。

鹅毛雪

作者：流星雨

寒来一夜冬风紧，喜看鹅毛落太空。

万树梨花潇洒舞，九千银界似烟笼。

/

小雪

作者：通明

琼花六角舞翩迁，润物无声大德连。

时令循环皆尽谢，笑观俏蕊绽红嫣。

大雪

作者：通明

北风冷冽透衣寒，飘絮飞花舞动欢。

素裹银妆城市美，津门踏雪觅梅丹。

咏梅

作者：通明

红粉娇葩淡雅醇，暗香一缕醉骚人。

瑶池玉女从天降，愿绽芳华笑报春。

腊梅

作者：通明

瑶池仙女降尘埃，玉蕊冰心傲骨裁。

数九寒冬香气散，经风沐雪报春来。

/

雪

作者：霍庆来

风行雪域万云开，漫引骚人笔砚裁。

岂惧寒流侵瘦骨，敲词弄韵意悠哉。

雪

作者：霍庆来

朵朵琼花妆陌野，风行雪域万云开。

幽香一缕飘然过，借问东园又绽梅?

咏梅

作者：霍庆来

玉蕊冰心溢暗香，凌寒傲雪斗风霜。

横斜疏影谁知味，万缕情丝醉羽裳。

咏梅

作者：霍庆来

长在深山远俗尘，晶莹独抱意淳真。

嫣红簇簇迷人眼，醉赏仙姿竞入神。


​

寻梅

作者：九级旋风

雪里芳魂自古吟，娇柔不惧冷风侵。

吾来未做看花客，为学三分傲骨心。

看雪

作者：九级旋风

一夜琼花遍地开，晶莹碎玉满亭台。

凭窗看罢心长叹，恨我今生不是梅。

/

吟雪

作者：心言斋主

漫天柳絮洁无瑕，万壑千村碧玉赊。

借问尘霾欲何往，白云深处是婆家。

咏梅

作者：心言斋主

一剪寒梅抱蕊开，迎风傲雪暗香来。

不和桃李争娇艳，玉骨冰肌独占魁。

/

雪压白梅枝

作者：若桐

城外梅枝裹素条，细分颜色路中标。

无风却见花灵动，早送寒香过雪桥。

驿梅

作者：若桐

悄填冷夜满沟壕，素色妆成似雪涛。

半里寒香邀驿路，一枝梅朵寄丛篙。

/

咏梅

作者：童丽君

香如天脂色尤鲜，不与群芳共斗妍。

独向寒风争气势，霜飞雪舞自怡然。

夜观梅花

作者：童丽君

一阵香风拂面来，南枝破蕾待时开。

霜侵雪染方争艳，万里银妆万树梅。

/

​咏梅

作者：心淡薄冰

萧风瑟瑟百花凋，一阵幽香袅袅飘。

雨打霜欺全不惧，衔冰戏雪乐逍遥。

/

念梅

作者：心淡薄冰

银纱曼舞素山时，一树琼枝绽玉姿。

溪涧淙淙花照影，冰清妩媚惹人痴。


​

咏梅

作者：梁燕妮

凌风冒雪绽苍枝，含韵馨香报故知。

溪畔篱旁冰骨展，何须墨客再吟诗？

雪梅报春

作者：梁燕妮

寒冰冷雪久徘徊，玉树银花迎面开。

端的罗浮情难禁，生机一朵报春来。

/

梅雪吟

作者：白洁

疏影横斜送暗香，琼花素雅碧天扬。

红梅映雪临寒傲，铁骨冰机岁月长。

梅雪吟

作者：白洁

鹅毛曼舞落高天，梅雪相依玉手牵。

含笑临风留倩影，百花争放我为先。

/

咏雪

作者：刘金华

天空玉女下凡来，遍洒银花任意开。

柳柏千身披素袄，春风嘻笑乐心栽。

梅

作者：刘金华

梅开岸上战风寒，覆雪红颜峻岭间。

梦里含情呈爱意，心留傲骨闯难关。

/

雪

作者：书山行者

剪开云朵作霓裳，步履无声入画廊。

曼舞轻歌诗客醉，且观大地换银妆。

咏梅

作者：书山行者

一抹胭红笑满腮，风姿摇曳对窗台。

期君踏雪今朝至，莫让花枝寂寞开。

/

大雪

作者：山石人

纷飞大雪银包裹，肃杀青山带冷风。

我自吟诗冬色里，弦音落在夕阳中。

望梅

作者：山石人

窗前极目岭头枝，万卉千花冻不知。

傲骨清魂犹在守，任凭雪舞冷风驰。


​

红梅

作者：茉莉

当年恨尔来耕种，入梦相逢可未亡？

今有繁花红绽处，香魂一缕耐寒霜。

红梅

作者：茉莉

北风呼啸自疏狂，玉蝶纷飞鸟兽藏。

但有红梅争吐艳，冰魂铁骨傲寒霜。

雪

作者：茉莉

凛冽寒风着力摇，漫天玉蝶任逍遥。

冰心一片藏新梦，飘向人间润物娇。

雪

作者：茉莉

瑟瑟西风一力摇，漫天玉蝶自飘飘。

冰心云后裁新梦，素岭红霞笔墨描。

/

咏梅

作者：高应富

一夜寒风老树红，俏姿映入眼云中。

兴来纵酒坛心漏，提笔狂书装醉翁。

咏雪

作者：高应富

世间乱象每相违，凛冽寒风少见吹。

大雪何尝漫天舞，红梅且恨报春迟。

/

咏梅

作者：城市之恋

风雨无情雪又侵，饱经磨砺亦甘心。

疏枝守得花千朵，赠与骚人作酒吟。

怜梅

作者：城市之恋

寒梅放蕊雪封湖，摇曳枝头花影殊。

入梦芳魂千百态，有谁能解夜时孤？

/

傲雪红梅

作者：五大洲

漫天飞絮路茫茫，村女红装上学堂。

冒雪踏冰驱冷冻，梅花一朵笑迎阳。

飞雪丰年

作者：五大洲

扬扬洒洒絮盈天，地冻天寒白卉妍。

一剪红梅迎日到，漫天飞雪兆丰年。

/

梅

作者：鸽子

冷艳初开冠百芳，瘦枝冰蕊扮红妆。

首均松竹三分傲，更胜兰芝一缕香。

思梅

作者：鸽子

天涯羁旅自徘徊，别却家山久不回。

岁月匆匆春又到，最思故土那枝梅。


​

春雪

作者：张树华

东君有意献琼花，玉絮轻柔暖万家。

梅笑松拔千里锦，多情万物蕴新芽。

咏梅

作者：张树华

多情瘦骨醉寒光，靓影冰心暗吐香。

雪打风吹春早报，虬枝玉树永流芳。

/

咏梅

作者：詹国华

喜鹊登枝玉宇清，寒梅傲骨展娇情。

皑皑白雪银装裹，北国严冬景致盈。

雪

作者：詹国华

九天玄女舞银花，絮落人间千万家。

大地桑榆冰雾挂，三山仙境似蒙纱。

/

咏梅

作者：秦焕举

寒梅不惧冷风徊，独在严冬花自开。

雪覆干枝亲媚骨，飘香旷野待春来。

雪

作者：秦焕举

曼舞银花霄九倾，遮天盖地落无声。

潇潇洒洒千般媚，妆就梅枝朵朵盈。

/

雪梅赞

作者：张庆阳

雪压冬云天地白，红梅一朵凌寒开。

谁人不晓羞花好，若有芳魂也自哀。

冬雪

作者：张庆阳

琼妃思恋世间佳，遍野满山开素花。

情意随心涓万物，迎来春暖舞红霞。

/

梅香

作者：朝进

清风化雨润梅开，吸引花仙下玉台。

雪落琴弦飞日月，黄昏浮动暗香来。

咏梅

作者：朝进

梅曲寒冬笑雪来，梨花泣动石山开。

冰肌玉骨娇姿美，玉雪清香醉翠台。

/

大雪

作者：李冠群

朔气几番凝四野，梨花一夜绽千家。

无香却是真颜色，照我归心越海涯。

飞雪

作者：李冠群

北风卷地舞琼英，六瓣飘飞比羽轻。

斜掠沾衣身作画，纯然无暇暗含情。


​

悟梅

作者：宝世华

寻迹常闻满寺香，惯将五瓣混僧肠。

偏迷热眼观颜色，只怕无缘问古璋。

老梅

作者：宝世华

自处黄昏伴鹤云，寒风拂面洗冬尘。

枯桩不寂虬枝探，几绽孤芳亦动人。

/

傲骨

作者：热冰

红梅傲骨抗严霜，万朵千枝欲泄香。

我不劈寒先吐蕾，群娇哪敢把花扬。

怜梅

作者：热冰

傲骨梅花吐蕾繁，梨花抱絮染心间。

孤红坠落随春去，白雪怜惜泪暗弹。

/

梅语

作者：江蓝

孤山相狎月黄昏，驿外诗翁亦慰魂。

最是润之能懂我，报春天下笑乾坤。

雪声

作者：江蓝

敲窗叩户响沙沙，千里郎归疑到家。

再起开门唯见白，玉寒扑面月西斜。

/

大雪

作者：张晓辉

云生穹顶作冰花，不染纤尘胜白沙。

一夜寒风归故里，扬扬洒洒惠农家。

梅

作者：张晓辉

一剪寒梅树雪中，千劫历尽且从容。

谁人解语愁滋味，煮酒煎茶说落红。

/

初雪

作者：葛霞

梨花一夜九州开，砌玉铺银舞阁台。

添被麦苗寒蝶醉，飞扬六出展奇才。

/

观雪

作者：葛霞

晶莹剔透降尘埃，万种风情带韵来。

素裹银妆仙客至，冰心脉脉惹诗才。

/

梅花

作者：方生

大雪如花遍地开，冬光傍影映阶台。

凌霜几处催春近，一角寒英弄色来。

/

早梅

作者：方生

晨起推窗雪挂枝，边城千树暗枯垂。

寒风吹雁乡关远，幸喜梅花意动时。


​

梅

作者：刘井海

铁骨寒梅傲雪开，冰枝絮叶透香来。

北风冷瑟吹花笑，漫舞娇姿醉玉台。

/

梅

作者：刘井海

银枝玉叶雪冰裁，俏丽多姿蕊竞开。

一笑嫣然红粉面，馨香暗洒透墙来。

/

思梅

作者：赵平

菊后庭园寂寞开，掷杯人月两徘徊。

似嫌尘土沾仙履，冰雪未铺君不来。

/

咏梅

作者：天道酬陈

独凌寒域任冰裁，不与群芳竞绿台；

莫道冬天无暖色，红梅似火为君开。

/

腊梅赞

作者：老冒

西风肃杀少吟哀，自有寒香傲雪开。

莫道无情风雪骤，一枝引领早春来。

/

山寺腊梅

作者：三滇

几树腊黄墙角出，雪花飞舞识香扑。

人踪难渡深山禅，却为玉魂作伴竹。

/

梅雪迎年

作者：张孝华

梅开满岭胜春光，一夜风吹处处香。

十里八村年味足，雪飞万朵赋诗章。

/

雪景

梯航（河北）

塞外冰天雪放歌，江南阡野尚婆娑。

千枝万树琼花放，峻岭崇山景色多。

/

吟梅

作者：董峰

只因笑在寒风里，便扯素袍为嫁妆。

未必幽居深谷处，迎春灿烂满山岗。

/

老梅

作者：方伟

几经苍郁傲冰风，瑞雪含欣再绽红。

梅曲媚浓香四溢，画词诗赋请君翁。


​

梅

作者：流浪的心

老树孤亭百草衰，银沙万里望中来。

冬寒犹在疏枝里，几点深红带雪开。

/

咏梅

作者：博凯

天生不愿展情才，唯寄相思对季开。

待得寒风飞絮尽，冰心好驾碧芳来。

/

傲梅

作者：蓝天

风瑟天寒雪飞扬，铺宣研墨伴梅香。

繁花落尽何争艳？独俏悬崖傲冷霜。

/

落雪

作者：雾松

轻飘白絮挂山川，天落银装掩沃田。

地冻封河年已近，待春唤起万虫眠。

/

雪中行

作者：安宁（汴梁）

人间快事雪中行，玉树蟠枝满汴城。

几日纠缠身外物，皆随飞絮落无声。

/

一支梅

作者：陈慧娟

山明水秀云天外，峻岭摩崖独自开。

唯恐世间春色晚，拈花一笑抱香来。

/

十月雪

作者：心湖

小城十月吼西风，一夜河山尽着戎。

四野茫茫皆絮白，谁家童子戏花丛。

雪后有感

作者：心湖

北风迎面吻肤寒，瑞雪皑皑罩壑峦。

一色河山谁绘就，可将荒漠变银滩。

/

梅

作者：小罗曼罗兰

香浓一朵雪中新，原是瑶台仙子身。

墨客三千心不动，娉婷只为有情人。

/

梅

作者：边义（辽阳）

不慕春光只爱寒，馨香独自向君传。
