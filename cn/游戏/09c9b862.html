怀旧服盗贼攻略大全：怀旧服盗贼究竟有多强？

2020-10-29
| 游戏

盗贼经常从暗处偷袭，率先发动狠毒的近身格斗。在持久战中，他们会利用一套经过精心策划的连续组合攻击削弱敌人的实力，以便施行致命的一击。盗贼在选择目标的时候必须特别谨慎，使他们的组合攻击不至于浪费。如果战斗对他们不利，则必须清楚何时该隐匿或逃逸。

一、 盗贼的基础信息

Ø 可选种族：侏儒，人类，矮人，暗夜精灵，兽人，亡灵，巨魔。

Ø 作用：近战伤害，持续的单体伤害。

Ø 资源：生命值和能量。

Ø 护甲类型：布甲、皮甲。

Ø 武器：匕首，剑，法杖，拳套，枪械，弓，弩。

Ø 属性：敏捷，力量，命中和爆击率。

Ø 优势：潜行，单体伤害，开锁。

Ø 劣势：缺少AOE能力。

Ø 适合的专业：制皮、工程。

三、 盗贼种族推荐

Ø 联盟

n 人类（PVE推荐）：，种族技能剑类武器专精完美契合盗贼。

u 外交：声望获取速度提升10%。

u 人类精魄：精神提高5%。

u 剑类武器专精：剑与双手剑技能提高5点。

u 感知：提升潜行侦测能力，持续20秒，3分钟CD。

n 矮人（PVP推荐）：

u 寻找财宝：使矮人可以感知附近宝藏的位置，并显示在小地图上。

u 冰霜抗性：冰霜抗性提高10点。

u 石像形态：对流血、毒药、疾病效果免疫，护甲值提高10%，持续8秒。

n 侏儒（PVP推荐）：

u 奥术抗性：奥术抗性提高10点。

u 工程学专精：工程学技能提高15点。

u 逃命专精：解除所有定身或减速效果。

n 暗夜精灵（不推荐）：

u 自然抗性：自然抗性提高10点。

u 精灵之魂：死后变成小精灵，移动速度提高50%。

u 迅捷：躲闪机率提高1%。

u 影遁：潜行，10秒CD，不能移动。

Ø 部落

n 兽人（PVE、PVP推荐）：，血性狂暴能让盗贼爆发更高。

u 坚韧：抵抗昏迷效果的机率提高25%。

u 血性狂暴：近战攻强提高25%，持续15秒。受到治疗效果降低50%，持续25秒。2分钟CD。

n 亡灵（不推荐）：

u 暗影抗性：暗影抗性提升10点。

u 水下呼吸：水下呼吸时间延长300%。

u 食尸：只能对半径5码范围内的人型生物或亡灵的尸体使用。每2秒恢复7%生命值，持续10秒。

n 巨魔（不推荐）：，虽然狂暴对盗贼输出还是有提升，但要最大化输出对盗贼还是很危险的。

u 野兽杀手：对野兽的伤害提高5%。

u 再生：生命值恢复速度提高10%。

u 狂暴：攻速提升和施法速度提升10%到30%，生命值越低提升越高。

四、 盗贼练级天赋推荐

剑类天赋（19/32/0）：

如果你喜欢单人游戏的话，推荐使用这套天赋。唯一需要注意的是尽量保持切割buff。



10-41级以战斗天赋为主，增强技能伤害。


42-60级，剩余的点数加到刺杀天赋中，尽快点出强化切割。


匕首类盗贼天赋（15/31/5）：

相比剑类盗贼，匕首盗贼的技能在团队中更有作用，如果你喜欢多人游戏，推荐使用这套天赋。相比剑类盗贼，匕首盗贼能有更好的输出环境，所以放弃了部分的防御性天赋，以增强输出。



10-19级，先点出强化邪恶攻击降低能量消耗，然后再补充另外两个天赋增强凿击的效果以及背后攻击造成的伤害。


20-45级，全都点在战斗天赋，降低疾跑冷却，点出大技能。


46-60级，这时候点刺杀天赋补充爆击和爆击伤害以及切割的持续时间。

六、 盗贼PVE天赋推荐

1、 剑类天赋（19/32/0）



这套天赋适合于怀旧服的每个阶段，你只需要找到一把合适的剑即可轻松地造成高额的伤害。这套天赋侧重于使用邪恶攻击和切割的使用。

2、 匕首类天赋（15/31/0）



这套天赋仅在你有比剑类更好的匕首时才推荐使用。与剑类不同的是，这套天赋你将会以背刺为主要的连击点生成技能。

七、 盗贼PVE输出循环

Ø 剑类武器

n 在隐身的状态下，使用绞喉进行起手。

n 未隐身的状态下，则使用邪恶攻击起手。

n 在1个连击点时使用切割。

n 使用肾上腺素。

n 使用邪恶攻击打满5个连击点。

n 使用切割。

n 循环邪恶攻击，保持切割buff。

Ø 匕首武器

n 在隐身的状态下，使用伏击起手。

n 未隐身的状态下，使用背刺起手。

n 在有1个连击点时使用切割。

n 使用肾上腺素。

n 使用背刺打满5个连击点，如果无法从目标背后使用，则使用邪恶攻击打满连击点。

n 使用切割。

n 循环背刺或邪恶攻击，保持切割BUFF。

十四、其它

1、 毒药/菊花茶职业任务

当盗贼等级到达20级时，可在职业训练师处开启毒药的相关任务，这个任务非常重要，学习毒药能提高盗贼的伤害或者生存能力。在毒药任务期间还会学习到如何制作菊花茶，这是一种盗贼专属的烹饪配方，可以使盗贼立即恢复100点能量值。

2、 搜索/开锁

盗贼在4级和16级时，可以学习搜索和开锁两个技能。

搜索：到你靠近怪物时可以选中怪物使用搜索，可以在不击杀怪物的情况下获取一部分战利品，例如少量的钱、食物、药水等，不过装备这些东西就不要想了。

开锁：开锁则能开启一些被锁上的箱子或门，你也可以通过为别人解锁箱子进行赚钱。与他人交易时，将上锁的箱子放在交易栏中的不可交易栏中，你就可以进行解锁，通过这种方式收取手续费进行赚钱。

3、 专业推荐

对于缺乏AOE输出的盗贼恶言，工程是最好的专业。工程制造的侏儒作战小鸡、铁皮手雷等物品，能很好地弥补盗贼AOE输出的短板。

4、 武器训练师

对于盗贼而言，最好的练级武器即是剑，但盗贼初始只能使用匕首，需要去武器训练师处学习使用剑类。并且武器的提升对于盗贼的收益巨大，应尽可能的提升武器，使得练级更加轻松。

怀旧服里的武器训练师位置如下：

Ø 暴风城（57,57）：匕首、剑、弩、长柄武器、法杖。

Ø 幽暗城（57,32）：匕首、剑、弩、长柄武器。
