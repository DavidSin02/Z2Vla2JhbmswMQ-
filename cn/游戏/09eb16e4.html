《刺客信条：奥德赛》实用铭文全方位攻略

2020-11-14
| 游戏

《刺客信条：奥德赛》中铭文是对玩家们战力提升的一种道具，铭文之间有一个好的搭配才能最大程度的发挥出铭文的效果，那么游戏中什么铭文好用，它又好用在哪里呢？今天小编就来给大家带来游戏实用铭文的全方位攻略，希望能对大家有所帮助。

实用铭文全方位攻略

一、 最实用铭文收集攻略。

本帖将给出最实用的四个【陶片铭文】的收集攻略。这四个铭文分别为【命中+15%肾上腺素】【满血+10%暴击率】【满血+50%暴击伤害】【+20%狂袭伤害】。

我在收集铭文的时候发现，收集陶片的最大困难不是【陶片铭文解密】，而是在整个希腊世界浩渺的【问号】中，找到这些陶片。只要你找到了陶片，即使你不看攻略，解密这个陶片也并不难。并且网上还有较为成熟的攻略。另一方面，如果你全清地图，会发现，很多铭文收集来是没用的。我曾经为了解锁一个中毒伤害+1秒浪费半小时时间!!!因此，我做攻略的目的有两个：

1、 只做那些【最实用】铭文的收集攻略，提高收集效率。

2、 挖掘铭文出处(非解锁地点)，省去清图清问号的麻烦。

下列表格给出这四个【最实用铭文】的出处!我将在所有内容更新完后以截图方式显示每个地点的具体位置(再也不用清问号啦)!

二、 伤害机制及铭文分类。

在之前的【战士大型攻略】贴中，我给出过本游戏伤害的计算公式，但由于一些测试的误差或者未完全拟合，与实际结果有一些偏差。我经过重复测试，进行了修正，新版伤害公式为：

武器面板 * 武器类型面板倍率 * 面板增强 * 伤害类倍率 * 动作值 * 特殊动作增强倍率 * 能力增强倍率 * 对特定敌人增伤倍率 * 暴击倍率 * 特定增伤倍率 * 特别伤害倍率

武器类型面板倍率：刀剑/匕首/锤/斧/矛/杖 15%伤害，刀剑匕首/重型武器/矛杖 10%伤害

面板增强：战士/猎人/刺客被动(40%)，战士/猎人/刺客 13-15%伤害，诅咒(100%伤害25%血量)，15%伤害-15%生命，所有伤害5%，猎人/战士25%伤害-15%防御

伤害类倍率：盔甲穿透，火焰/剧毒伤害，30%元素伤害-30%元素抗性，生命全满30%伤害

动作值：技能伤害倍率/平A伤害倍率

对特殊动作增伤：狂袭增伤(最高200%)/弓蓄力增伤(最高120%)

能力增伤：使用战士/刺客/猎人能力增伤15%，使用XX(如混沌之环，毁灭射击)能力时增伤20%

对特定敌人增伤：雅典/斯巴达/动物增伤(最高200%)

暴击倍率：基础暴击倍率150%，基础爆头倍率250%

爆头伤害为250%+所有暴击爆头词条(包括被动)

暴击伤害为为150%+所有暴击伤害词条(包括被动)

特定条件增伤(分别乘算)：匿踪大师夜晚脱战增伤，第六感增伤，远距离增伤15%(30公尺)

备注：

【特别伤害倍率】：从敌人正背后攻击敌人时出现黄字并且增伤30%。该特别伤害与暴击爆头不能叠加。即，背后攻击出现暴击爆头时，无特殊30%伤害加成。

【爆头伤害基础250%测得依据】：100%爆头暴伤词条+90%被动下爆头为暴击的1.5左右(440%对300%，增加50%爆头词条爆头：暴击增加到5:3左右)。

【暴击伤害基础150%测得依据】：拿弓测得，150%暴伤词条下正好为3倍伤害;0词条被动下为1.5倍伤害。

【远距离伤害15%乘算依据】：30公尺以上有较大概率触发15%乘算增伤(也可能不触发，乘算依据，火焰箭和夺命箭均为1.15倍伤害，夺命箭纯物理加成不吃任何元素加成，倍率相同说明是单独乘算)。远距离增伤与第六感不互相排斥。可被近距离敌人发现并且射击远距离敌人。【远距离增伤词条有bug，超过30%公尺也有概率不触发】

下面的表格给出所有类别伤害相关词条的总结。每种类别内为加算，类别与类别之间乘算。

注意小背伤是人物天生自带的，从背后近战/远程攻击敌人时，不出现猎人暴击爆头，一样跳黄字(俗称小暴击)，伤害为白字的1、3倍。但是可以与战士暴击叠加。大暴击小暴击直接乘算。另外战士能力15%和20%狂袭为加算。狂袭流不推荐出15%战士能力。

三、 基于伤害公式的伤害类铭文收益全解析

I、 非燃烧流极限优化

我们先给出一个火焰战士基础配置铭文：

普罗米修斯剑：35%火焰伤害，盔甲穿透30%

副武器：20%火焰伤害，30%元素伤害

弓：20%火焰伤害，诅咒/满血30%伤害

头盔：无

盔甲：25%暴击伤害

手臂：5%暴击率，50%满血暴击伤害

腰带：5%暴击率，50%满血暴击伤害

鞋子：无

由此我们得到，火焰战士的最低基础伤害倍率为

1、 面板：100% + 40%被动 + 15% + 14*7 = 253%，弓诅咒则为353%

2、 伤害类倍率： 100% + 40%火焰被动 + 35% + 20%*2 + 30%*2 = 275%，弓满血30%则为305%

3、 暴击率：10% + 5%*2 + 10%被动 = 30%

4、 暴击伤害： 150% + 50%被动 + 125% = 325%

由于以上基础倍率按照最低基础值计算，下面是各大词条的最高收益：

1、 10%战士伤害(传说中逆天的双词条)， 10%/253% = 4.0%，如果弓铭文为诅咒，则为2.8%

2、 20%火焰伤害(头盔，腿甲)，20%/275% = 7.2%。如果弓铭文位满血30%，则为20%/305% = 6.6%

3、 满血10%暴击率，(0.4*3.25 + 0.6)/ (0.3*3.25 + 0.7) = 13.4%

4、 满血50%暴击伤害， (0.3*3.75 + 0.7)/(0.3*3.25 + 0.7) = 9.0%

可以看到，收益最高的为【1满血0%暴击率】和【满血50%暴击伤害】。如果装备不够好手臂或者腰带缺少一个50%暴击伤害，则暴击率收益下降，而暴击伤害收益上升，但均大幅高于火焰伤害及战士伤害。

现在我们拿一套经典的“最强”火战士配装举例。主要矛盾集中在头盔和腿甲的词条选择。

原“最强”：头盔20%火焰伤害15%刀剑伤害，腿甲20%火焰伤害50%满血暴击伤害。此时基础为，伤害类345%，暴击伤害325%(假设腰带和手臂只有一个50%暴伤)，暴击率30%。此时将双火焰伤害替换为2个20%满血暴击的收益为：305/345 * (0.5*3.25 + 0.5)/(0.3*3.25+0.7) = 112%。

【也就是说，用2个满血暴击词条替换2个火焰伤害词条，反而多出了12%的期望收益。】

因此新刀剑最强基础配装毋庸置疑：

头盔：15%刀剑伤害，10%满血暴击率

胸甲：25%暴击伤害

腿甲：10%满血暴击率，50%满血暴击伤害

结论是【满血暴击】> 【满血暴伤】 > 【火焰伤害】 >> 【战士伤害】

【重要】!!!：结果是惊人的，以上基础配装，居然不包含任何火焰/剧毒伤害。这意味着你完全可以通用配装，同一套装备走火/毒流。在防具纯堆物理词条的情况下，却达到了收益的最大化。还在吹元素词条吗?火焰/剧毒伤害很厉害吗?真正厉害而通用，必备的防具词条全部是【暴击系的】(物理系词条)。而最垃圾的就是传说中超稀有的【双战士伤害】词条。

下面的表格给出通用战士的各铭文收益，我们主要看【期望乘算收益】一栏，这一栏数值越高，说明这个铭文越重要!所有重要的词条我均已标红。值得注意的是，战士能力伤害15%虽然纯收益为15%，但限制极大，不适合平A燃烧流，因此只在特定情况下做优先选择。狂袭伤害同理，特化过于严重，也是特点情况下选择。具体的词条选择取决于平常的打法。

II、 Dot极限优化

关于这游戏的持续伤害(Dot)，就不得不提及dot的伤害机制。我下面给出dot伤害计算公式：

每跳dot伤害 = 武器面板 * 伤害类倍率中的元素倍率 * dot系数 * 暴击伤害 + dot基础值 * 等级

火附魔的本质是将武器伤害，全部转化为火伤，因此武器面板 * 伤害类倍率中的元素倍率其实就等于实际火面板。(注意30%元素伤害属于元素倍率，但30%满血伤害盔甲穿透不属于元素倍率)。dot系数因dot种类而不同，火dot系数大幅高于毒dot系数。暴击伤害，依旧以期望的形式进行计算。【从该公式得知，堆元素伤害或者面板伤害均可增加dot伤害】。由于我发现诅咒并不是线性增加伤害的(诅咒后4000变5000，增加25%，面板为40%，矛盾)，因此dot里应该有一个受等级加成的基础值。

再来看铭刻配置，头盔铭刻20%火焰伤害，腿部铭刻【50%燃烧率】铭文，因此，剩下的腿甲的词条选择为20%火焰伤害，10%满血暴击，50%满血暴伤。对于dot来说，伤害倍率为100% + 40% + 30% + 35% + 40% + 20% = 265%。20%/265% = 7.5%。收益依旧是不如满血暴击和暴伤的。因此腿甲的最优选择依然是满血暴击暴伤!!!可以看到，dot流的最强基础配装和通用火战士是一模一样的。非常重要!!!

【假设你没有刷到最强通用基础配装】，那么请务必在有词条冲突的时候优先选择10%满血暴击和50%满血暴伤。

四、战士各流派终极配装推荐

分析完各铭文的收益后，我们下面总结【所有输出流战士】的终极配装。配装包括包括毒流和火流，但不包括苟命流。苟命流，逃跑流，脱战流，请使用毒配并自行配置30%防御力，20%全抗性，10%生命等各生存铭文。

在表格中，第二栏为通用词条(也就是一套通用必备的词条)，后面几栏为各战士流派特化的铭文词条。其中标橙的字体为铭刻的金色铭文。值得注意的是，新狮子套的满血5%暴击率25%满血暴伤也是个非常非常强大的铭文(相当于10%满血暴击和50%满血暴伤的综合)

亚特兰蒂斯匕首铭文为：15%猎人伤害，25%暴击伤害，30%盔甲穿透，1个金色铭文位。

对比紫色刀剑/匕首铭文：14%战士伤害，25%暴击伤害，20%剧毒伤害，1个金色铭文位。

【亚特兰蒂斯匕首】相对于紫武器面板优势为5.3%。使用亚特兰蒂斯匕首时，伤害类倍率为(100+25+20*4+30*3)= 295%。剧

毒伤害20%收益为6.8%。盔甲穿透收益为10.1%。14%战士伤害收益为5.5%有诅咒，4.0%无诅咒。即使是【5.5%无诅咒战士伤害+6.8%剧毒】也打不过【5.3%面板，10.1%盔甲穿透】。

因此亚特兰蒂斯匕首虽然是猎人加成，但依旧完胜紫色毒武器!!!另外，亚特兰蒂斯匕首的好处是可以同时携带：盔甲穿透，满血30%，满血5%暴击25%爆伤，30%元素伤害，诅咒，5个中4个金色铭文!!!!【完全不是紫毒武器可以比的】。还在用紫色毒武器的非狂袭战士你们还用吗
