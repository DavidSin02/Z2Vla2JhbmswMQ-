「公主连结Re:Dive」一期专武解析

2020-11-14
| 游戏

文丨生活可以很美好@bigfun社区

说实话，今天的本地化笔记出来的时候惊到我了，完全没想到专武的开启有这么快。不过仔细想想和台服对比也就提前了一期（繁中服是天秤座会战以后开启专武的），相较于国服没有压缩的卡池，提前开启的专武能给会战带来不一样的体验，这样看来国服也有自己的特色了。

然后这篇文章本来是准备放在这期会战之后的，因为我预测国服的专武进度和台服进度差不多，所以现在是知道消息后爆肝出来的。当时准备的是三轮三轮专武的做（留给大家足够的刷碎片时间），现在因为时间比较赶，就先只做第一期了。

一期专武主要为前作和这一作的御三家加上铃铛、充电宝和偶像。

懒人包部分

一、前言

一些基础数据

根据本地化笔记，这次专武总共开放到110级，也就是拉满一个人物需要58个心碎+90个人物碎片+185万MANA。升级满还需要474w5k的MANA。

对于我们来说最缺的资源应该是心碎，在不加倍，不重置的情况下，只靠圣迹调查（心碎本）一天连10个碎片都刷不到。好在同期还会开放露娜塔，露娜塔中同样会提供心碎作为奖励（量也不是特别大）。在这种情况下，我们初期的目前是以开专武为主，不要急着强化满。

专武和RANK、装备一样，并不是拉满最好。如果专武附加的属性有双抗和血量的时候，很多时候也需要卡专武等级。

对环境影响的猜想

专武的提前影响最大的还是公会战，而这主要还是出在怜（剑圣）上。

专武赋予了剑圣破甲的能力。经过了这么多轮会战，大家已经知道了破甲的重要性。不过怜本身的输出并没有很高，破甲数值也有限，所以虽然有了破甲，但是还是很难断言会压缩法队的上场，台服同期法队被打压的主因是TOMO和克总提前（台服美杜莎会战有TOMO，天秤座会战有克总）。而且通过上次会战的魔改我们会发现，国服的魔改还是以砍双抗为主（且砍的魔抗值较多），在这种情况下估计还是以两物理一法为主。

但会战最重要的是你可以不上场，但你不能没有，所以剑圣的练度也会作为一个考核点。

第二个影响是天秤座的云，我们公会的指挥一直说他有印象台服的时候因为云魔抗过低，我们出法刀的时候用中二顶替了一个破甲（黑猫）。而专武赋予了黑猫一技能破甲的能力，是否能让她在天秤的时候有更多的发挥呢？（这里之所以用了问号，是因为我一点都没这个印象啊，我的印象一直是云是物理和法刀混着出。）

二、具体介绍

这部分介绍的顺序是按上代御三家→这代御三家→剩下三人。

折叠内放的是专武和技能的介绍。专武的数据是110级的数据。

日和（猫拳）

喵王腰带：日和配戴的腰带。 有着猛兽睥睨眼神的扣环， 能鼓动斗争本能， 令拳头更有劲道。

专武属性： 物理攻击力607 ；物理暴击129

一技能强化效果：对范围【400】内的敌人造成 【9.0 + 9.0*技能等级 + 0.72 *atk】的物理伤害。

猫拳的专武给了她一技能额外的AOE，属性上增加了物理攻击力和暴击，无需卡专武，可直接升满。

猫拳的专武很一般，设计上确实提高了一定的伤害，但是作为第一批吃螃蟹的人，这种纯伤害技能必然随着超模角色的出现而被环境所淘汰。

虽然我在前文中在公会战方面给了猫拳较高的优先度，但是实际在使用中估计很难有上场的机会。最主要的原因就是我前文提到的国服的环境大概率是两物理一法，而在专武实装的同期另一位顶底物理打手TOMO也会实装。TOMO有个外号叫做0.9狗，这说明了TOMO在输出端的强势。2刀物理刀中狼狗基本定死了，扣去辅助，能上场的物理打手只有三到四个，这基本被三弓和TOMO占领了位置。在这种情况下，猫拳大概率是无法上场的。

建议留好JJC币观望国服改动，之后仓管等六星实装（然后成了JJC一把好手）

优衣

繁花之愿：优衣配戴的发箍。 造型可爱的花蕾丝与装饰， 让守护伙伴的治疗之力更为提升。

专武属性：物理防御33；魔法防御33；魔法攻击力297；回复量上升33

一技能强化效果：【20.0 + 20.0*技能等级 + 1.6 *atk】的魔法伤害。 降低敌方单体1230 【7.5 + 7.5*技能等级】的物理攻击力。 持续：【12.0】 秒。

优衣的专武给了她一技能伤害上更高的倍率，以及额外的降攻效果。不痛不痒的专武效果。

简单来说你所有能用到优衣的场合，她自身都能发挥很好了，不需要再有专武的加持；而你用不到专武的场合给优衣装上专武也无法使用。如果说一定要给她找个穿专武的理由，那就是因为不穿专武升不了六星。

优衣专武优先度低，资源富裕情况下才去考虑。（优衣对不起）

怜（剑圣）

雄风手套：怜爱用的手套。这副攻守兼备的手套所打出的正拳突刺，不论对上什么敌人都能够打倒。

专武属性：HP710；物理攻击力613

一技能强化效果：对敌方单体造成【20.0 + 20.0*技能等级 + 1.6 *atk】的物理伤害。

降低敌方单体【0.6 + 0.6*技能等级】的物理防御力。 持续：【12.0】 秒。

剑圣的专武使她一技能有了破甲效果。大家在看WIKI的时候要注意，有时候剑圣一技能那里会显示两个伤害和两个破甲，数值比较高的是剑圣六星以后的效果。

剑圣因为破甲效果，终于可以登上会战的舞台了。考虑到公会战如果无法出三刀物理的话，那么剑圣的出场其实并没有你们想象中那么多。但无疑剑圣是这期专武中，提升最大的之一。我们当时在台服的时候剑圣主要是卡四星满专，或则五星30专。之后剑圣退出舞台，主要是因为有更好的破甲角色。直到六星之后，剑圣又会重新成为会战的重要角色。

建议可以开始每天刷了，但是不用急着升满，四星开专即可，等会战之后再确定。（反正JJC也用不到剑圣）

可可萝

爱梅斯护符：可可萝配戴的花饰。受到被祀奉为神的伟大存在之加持，花中飘出的温和香气，可以赋予战斗的勇气。

专武属性：物理防御33；魔法防御33；物理攻击力323；回复量上升33

一技能强化效果：对敌方单体造成 【20.0 + 20.0*技能等级 + 1.6 *atk】的物理伤害。

提升自身【11.0 + 11.0*技能等级】的物理攻击力。 持续：【12.0】 秒。

使自己的TP上升提升【15.0 】。 持续：【12.0】 秒。

可可萝的专武使她的一技能能赋予自身攻击力上升和TP上升。

依旧是一个非常保守的专武，并没有给可可萝带来什么质的变化。我们升级专武就仅仅是为了可可萝能在会战中多打一丁点伤害。随着强力角色越来越多可可萝的出场机会会越来越少，直到开启六星之后，可可萝又活跃在各种场合（她的奶实在太足了，然后其他属性又很多）。

初期开专即可，具体看会战实战需求。

凯留（黑猫）

混沌魔典：凯留携带的魔导书。 只要控制书中满载的浩大魔力， 就能自由操控所记载的 强大术式。

专武属性：魔法攻击力607；魔法暴击65

一技能强化效果：对敌方单体造成 【20.0 + 20.0*技能等级 + 1.6 *atk】的魔法伤害。

对范围【200】内的所有敌人造成 【4.5 + 4.5*技能等级 + 0.36 *atk】魔法伤害。

使范围【200】内所有敌人的魔法防御力下降23 【0.1425 + 0.1425*技能等级】。 持续：【12.0】 秒。

黑猫的专武给她的一技能加了一个4.5倍率的AOE和一个比较低的魔抗降低效果。

纵观PCR几次特殊内容的引入（专武和六星），黑猫从中获得收益的方式都是作为第一批成员，拥有了爆炸属性，从数值上比别人优秀。这就导致了将来若出现了数值更加优秀的角色或机制更加优秀的角色，她就很容易被淘汰。

专武黑猫在刚出来的这段时间，主要发挥是在JJC，但是考虑到国服会战法队必不可少，那么即使是公会战玩家对于黑猫专武的需求也比较大。之后七七香出来之后，黑猫的出场次数会再被压缩，不过考虑到一个是塔币，一个初期只能靠母猪石，大部分玩家还是会选择黑猫。

专武拉满，可活跃于JJC和公会战，仓管后等六星翻身

佩可（吃货）

公主之剑：贪吃佩可所使用『皇家装备』 真正的样貌。原本已经足够强大，但若解放了那沉睡之力，将会赋予更强的力量。

专武属性：物理防御65；魔法防御59；回复量上升33

一技能强化效果：提升自身【1.0 + 1.0*技能等级】的魔法防御力。 持续：【12.0】 秒。

回复自身【10.0 + 10.0*技能等级 + 1.2*atk】的HP。

展开可吸收 【22.5 + 22.5*技能等级 】伤害量的物理魔法吸收护盾。

佩可的专武让她一技能的提升魔法防御力和回复的系数都有较大的提升。赋予了新的能力：展开魔法吸收盾。

佩可的专武可以说是“锦上添花”了，佩可本身常用于PJJC的防守队，专武的存在进一步提升了她的肉度。肉、很肉、非常肉，这个就是专武以后的佩可的唯一评价。不过在T上物理T还是布丁，法T还是抖M。佩可主要还是活跃于PJJC苟队阴人的。

打JJC的朋友可以直接拉满，不过初期心碎较缺，先以开专为主。

胡桃（铃铛）

善举之铃：胡桃当作护身符的铃。从庄严雅正的外观到奏出的可爱音调，都能让怯懦的心生出勇气。

专武属性：HP645；物理防御52；魔法防御52；HP自动回复1290

一技能强化效果：提升我方全体【0.75+0.75*技能等级】的物理防御力。持续：【12.0】秒。

提升我方全体【0.75+0.75*技能等级】的魔法防御力。持续：【12.0】秒。

回复我方全体【50.0】的TP。

胡桃的专武提升了一技能原先的系数，同时赋予了小幅度的充电效果。

胡桃这个角色，并不是专武以后一下子就发光发热的，而是慢慢随着环境被人挖掘出来的。主要是胡桃的技能，在初期伤害不是特别高的时候，效果并不是那么直观展现的。随着数据的膨胀人们慢慢发现胡桃的一技能能有效的避免我方角色被秒杀。现阶段的使用还是以PJJC中的苟队为主。

JJC玩家优先度较高，其他的可以把顺序放后面一点

咲恋（充电宝）

荣耀之羽：咲恋爱用的发饰。温柔之中蕴含的强大正是其象征。当翅膀振动时，能令身体从重力解放。

专武属性：HP162；物理攻击力678

一技能强化效果：使该技能的系数1提高【损失的HP*0.4】

对【500】范围内的敌人造成【12.0+12.0*技能等级+0.9*atk】的物理伤害。

咲恋的专武把她的一技能系数从0.9升到了1.2，同时生命值越低伤害越高。

充电宝专武最尴尬的地方在于专武提升的输出能力只能作为一个添头。我们使用充电宝的场合更多是把她作为一个辅助让她给输出充能。这使得充电宝的专武有点类似优衣，但是因为在JJC中的高出场率，充电宝的伤害多少会帮我们打一波奇迹团。

充电宝专武优先度低，有余裕的情况下再升即可。

望（偶像）

闪耀麦克风：望专用的手持麦克风。在可爱的缎带中央画有【慈乐之音】的标志。拿到手上就能向顶级偶像迈进。

专武属性：HP1935；物理防御39；魔法防御39；回避7

一技能强化效果：对范围【280】内的敌人造成【8.0+8.0*技能等级+0.6*atk】的物理伤害。

使范围【280】内的敌人一段时间内无法行动。加速0.0倍。持续：【2.0】秒。

使范围【280】内敌人的物理攻击力降低【1.5+1.5*技能等级】。持续：【12.0】秒。

偶像的专武增加了伤害的倍率，增加了控制时间还加了一个降攻的DEBUFF。

偶像作为一个JJC常客，本身就有非常高的强度和出场率。她的专武对她来说更像是一种锦上添花，专武的属性增加了肉度；附加的控制效果也能更好地恶心对面。不过从实际角度来说，虽然有降攻和击晕效果，但并不会对二号位造成伤害，所以对方在配置阵容的时候还是可以使用单T。

偶像开专后直接拉满，但是不急着投资特别多的碎片。
