输出圣骑士怀旧服PVE指南：感受圣光的平A吧

2020-10-29
| 游戏

一、 开始之前

圣骑士是怀旧服团队中不可或缺的职业之一，强有力的治疗能力及一系列增益光环使其成为团队中一股出色的力量。而在怀旧服中的输出骑士（惩戒系）更像是介于输出与治疗之间的支援型职业。在本篇文章中，我们将针对圣骑士在PVE副本中的能力进行详细的研究。

1、 输出骑士的职业优劣

Ø 优势

n 强大的团队支援能力

n 强大的恢复能力

n 护甲高不易死

Ø 劣势

n 机动性低

n 持续作战能力低

n 伤害技能有限

2、 输出骑士的种族推荐

圣骑士在怀旧服中为联盟专属职业，所以你必须选择加入联盟才能创建圣骑士。

Ø 人类（推荐）

人类是圣骑士最佳的种族选择，种族技能“剑类武器专精”和“锤类武器专精”将能额外增加你的命中率，这是矮人种族无法比拟的。其次是“外交”种族技能能加快声望的获取速度，而“感知”在PVP中的作用也非常不错。

n 外交：声望获取速度提高10%。

n 人类精魄：精神提高5%。

n 锤类武器专精：锤和双手锤技能提高5点。

n 剑类武器专精：剑和双手剑技能提高5点。

n 感知：提高潜行侦查能力，持续20秒，3分钟CD。

Ø 矮人

相比人类，矮人的种族技能对PVE并未有多大的提升。但矮人的种族技能“石像形态”在PVP中应对盗贼的毒药或者战士的出血时有奇效，如果你喜欢PVP的话可以选择矮人，总体来说都不如人类。

n 寻找财宝：使矮人可以感知附近宝藏的位置并显示在小地图上。

n 冰霜抗性：冰霜抗性提高10点。

n 枪械专精：枪械技能提高5点。

n 石像形态：对流血、毒药和疾病效果免疫，护甲值提高10%，持续8秒，3分钟CD。

3、 输出骑士的专业推荐

怀旧服中圣骑最好的专业为工程学，这个专业能为骑士弥补PVE中伤害能力不足的问题，并且还有很多物品能提供不错的效果。

三、 天赋推荐

1、 通用PVE天赋(11/8/32)



这套天赋的加点相对固定，没有什么可以改动的地方。防护天赋树下必须点出“精确”提供命中率。神圣天赋的“神圣之力”可以说是每个圣骑士的必点天赋，而“奉献”能为我们增多一个输出技能。唯一可以更改的地方是惩戒天赋树第二层的“强化十字军印记”，因为在怀旧服有着debuff上限，所以很多时候可能你会用不到十字军印记，这个时候就可以把天赋点加在第一层的“祈福”和第四层“强化惩戒光环”。

2、 命中率达到上限后的天赋改动（18/0/33）



选择这套天赋的前提是你装备堆叠的命中率达到了9%，这时候我们将舍弃防护天赋下的“精确”，取而代之的是神圣天赋下的“精神集中”和“强化圣疗术”，使我们的生存能力更强。


四、 技能循环

输出骑士的循环非常简单，主要由审批和奉献两个技能为主。审判将会是你最主要的输出技能，与命令圣印或复仇搭配时能造成高额的单体伤害。

技能循环如下：

Ø 保持命令圣印

Ø 使用审判

Ø 复仇与奉献搭配使用

Ø 当目标生命值低于20%时，尽量使用愤怒之锤

技能循环不是特别复杂，但是需要注意不能长时间的作战，蓝量根本就撑不住。
