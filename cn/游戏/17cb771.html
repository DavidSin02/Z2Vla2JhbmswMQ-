怀旧服元素入侵事件是什么？元素入侵事件在哪？元素入侵奖励？

2020-10-29
| 游戏

一、 什么是元素入侵？

每隔数日，在卡利姆多的安戈洛环形山、艾萨拉、冬泉谷和希利苏斯就会刷新一些元素入侵开始。事件开始时会出现一个元素位面入口，然后会不断地刷出一些元素入侵者和一个元素首领。

所有的元素都有机率掉落火焰元素之类的元素精华，在元素首领被杀死之前，元素入侵者会不断地刷新，如果你是为了刷元素材料的话，切记千万不要把首领给杀了。当元素首领被击杀后，元素入侵事件结束，剩余的元素入侵者将会继续保留，直到被杀死。

二、 入侵事件类型

卡利姆多的四个地点会分别刷新不同的元素入侵事件，分别对应着四种元素：

Ø 安戈洛环形山——炽热入侵者

Ø 艾萨拉——雷霆入侵者

Ø 冬泉谷——水流入侵者

Ø 希利苏斯——旋风入侵者

三、 炽热入侵者

炽热入侵者是一个绿色的火元素，会刷新在安戈洛环形山火羽山的底部附近。



Ø 火焰男爵查尔

n 火焰男爵查尔是炽热入侵者的首领，免疫火属性攻击。

n 刷新时会大喊：“火山口的居民们！我是来焚烧这片污秽之地的！”

n 还生存时会大喊：“这个世界所谓的英雄在哪里?害怕得不敢出来玩了?”

n 被击杀时会大喊：“这一切终将消耗殆尽。”

Ø 掉落

n 火焰男爵查尔的节杖

n 元素之一（暗月马戏团卡牌）

n 元素灰烬

n 火焰精华


四、 雷霆入侵者

雷霆入侵者是刷新在艾萨拉东北部的土元素，附近还有很多石巨人精英怪，需要格外地小心。



Ø 阿瓦兰奇奥

n 阿瓦兰奇奥是雷霆入侵者的首领，建议组队进行挑战。

n 刷新时会大喊：“你藏在哪里？我找到你了！”

Ø 掉落

n 阿瓦兰奇奥的石皮

n 元素之一（暗月马戏团卡牌）

n 硬石戒指

n 大地精华


五、 水流入侵者

水流入侵者会刷新在冬泉谷的凯斯利尔湖附近。



Ø 泰比斯蒂亚公主

n 泰比斯蒂亚公主是水流入侵者的领袖，免疫冰霜伤害。

n 刷新时会大喊：“以这片冰冷的土地为起点，到我这来吧凡人，我会给你一个快速而痛苦的死亡。”

Ø 掉落

n 泰比斯蒂亚的寒冰项链

n 图样：雷暴手套

n 元素之一（暗月马戏团卡牌）

n 图样：信念手套

n 寒冷戒指

n 水之精华


六、 旋风入侵者

旋风入侵者是刷新在希利苏斯西北部的空气元素。



Ø 烈风掠夺者

n 烈风掠夺者是旋风入侵者的领袖。

n 刷新时会大喊：“烈风掠夺者和他的入侵部队出现在希利苏斯西北部的动乱中。”

Ø 掉落

n 烈风掠夺者腰带

n 图样：雷暴手套

n 元素之一（暗月马戏团卡牌）

n 微风斗篷

n 空气精华
