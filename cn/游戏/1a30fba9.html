2020升降机司机证考试及升降机司机作业考试题库

2020-11-14
| 游戏

题库来源：安全生产模拟考试一点通公众号小程序

2020升降机司机证考试及升降机司机作业考试题库，包含升降机司机证考试答案解析及升降机司机作业考试题库练习。由安全生产模拟考试一点通公众号结合国家升降机司机考试最新大纲及升降机司机考试真题出具，有助于升降机司机考试试题考前练习。

1、【判断题】润滑是起重机司机维护保养工作的主要内容之一。（ √ ）

2、【判断题】特种设备安全监督管理部门在办理《特种设备安全监察条例》规定的有关行政审批事项时,其受理、审查、许可、核准的程序必须公开,并应当自受理申请之日起30日内,作出许可、核准或者不予许可、核准的决定;不予许可、核准的,应当书面向申请人说明理由。（ √ ）

3、【判断题】起吊物件与地面或其它物品相联接，只要物件不超重，就可以起吊。（ × ）

4、【判断题】《中华人民共和国特种设备安全法》规定，特种设备销售单位应当建立特种设备检查验收和销售记录制度。（ √ ）

5、【判断题】块式制动器按工作状态可分常闭式和常开式两种。( )（ √ ）

6、【判断题】《中华人民共和国特种设备安全法》中所称的特种设备是指对人身和财产安全有较大危险性的锅炉、压力容器（含气瓶）、压力管道、电梯、起重机械、客运索道、大型游乐设施、场（厂）内专用机动车辆，以及法律、行政法规规定适用本法的其他特种设备。（ √ ）

7、【判断题】起重机在通过铁路、地下水管、地下煤气管道、电缆等设施时，应铺设木板保护，并不得在上面打转。（ √ ）

8、【判断题】被游翻的物件在重力矩作用下倾倒的同时，大车或小车应向回迅速开动，用以调整车体的位置。（ √ ）

9、【判断题】在紧急情况时，司机应立即打反车制动。（ × ）

10、【判断题】起重钢丝绳的报废标准之一是每一节距内断丝数达到总丝数的10％。（ √ ）

11、【判断题】《中华人民共和国特种设备安全法》规定，特种设备生产、经营、使用单位及其主要负责人对其生产、经营、使用的特种设备安全负责。（ √ ）

12、【判断题】制动器调整工作行程的目的是能获得合适的制动力矩。（ × ）

13、【判断题】起重机变幅应缓慢平稳，严禁在起重臂未停稳前变换档位；起重机载荷达到额定起重量的90％及以上时，严禁下降起重臂。（ √ ）

14、【判断题】起重机的电器设备必须保证传动性能或控制性能准确可靠，在紧急情况下能切断电源安全停车。（ √ ）

15、【判断题】应禁止一切重物从人员上空越过。（ √ ）

16、【判断题】钢丝绳的安全系数是指钢丝绳的破断拉力总和除以许用拉力的商。（ × ）

17、【判断题】在电路中如果发生短路或严重过载时，需要迅速切断电源，担负这一任务的是时间继电器。（ × ）

18、【判断题】起重机司机在操作起重机时，速度要均匀严禁或快或慢和突然制动。（ √ ）

19、【判断题】工作结束应将控制器回到零位，切断总电源，收好工具，关好所以的门窗并加锁。（ √ ）

20、【判断题】正常操作中，严禁反接制动。（ √ ）

21、【判断题】重载下降时，应将手柄快速推到最后一档，当被吊物达到应停位置时，应迅速将手柄由最后一档扳回零位，中间不要停顿。（ √ ）

22、【判断题】起重机上目前使用较广泛的是线接触的钢丝绳。（ √ ）

23、【单选题】特种设备事故在特殊情况下，经负责组织调查的质量技术监督部门批准，事故调查期限可以适当延长，但延长的期限最长不超过( )。（ A ）

A、60日

B、90日

C、180日

24、【单选题】吊装钢丝绳之间的理想夹角是( )。（ B ）

A、30°～45°

B、60°～90°

C、90°～120°

25、【单选题】起升机构制动器的制动轮轮缘厚度磨损超过( ),应更换。（ B ）

A、25％

B、40％

C、50％

26、【单选题】为防止起重机出轨而设置的( )限位装置。（ B ）

A、重量

B、行程

C、力矩

27、【单选题】《中华人民共和国安全生产法》是各级人民政府及其有关部门进行监督管理和行政执法的( )。（ B ）

A、行政规定

B、法律依据

C、有力武器

28、【单选题】作业人员考试两项均不合格或者补考仍不合格者，应当( )考试。（ B ）

A、重新复习参加

B、重新申请

C、三个月内不得重新申请

29、【单选题】起重机司机在吊运作业时必须听从( )。（ C ）

A、企业领导

B、班组长

C、起重指挥

30、【单选题】将原动机的机械能转化为液压能的元件为:( )。（ C ）

A、液压马达

B、液压缸

C、液压泵

31、【单选题】板钩心轴磨损标准达原尺寸的( )时，心轴应报废。（ A ）

A、5%

B、15%

C、50%

32、【单选题】在起重作业中尽量使货物的重心在两根起吊绳的( )。（ C ）

A、左边

B、右边

C、中间

33、【单选题】对己经加工的工件兜翻时，为了防止碰撞，可采用( )的方法。（ C ）

A、铺设垫木

B、使用双小车

C、加挂副绳

34、【单选题】一般主吊装用钢丝绳宜选用( )钢丝绳。（ B ）

A、顺绕编织

B、交绕编织

C、混绕编织

D、复合编织

35、【单选题】采用绳卡连接钢丝绳,绳卡紧固至压扁钢丝绳:( )。（ B ）

A、1／2

B、1／3

C、1／5

36、【单选题】“两小臂水平置于胸前,五指自然伸开,手心朝下,同时水平挥向两侧。”这种手势信号表示( )。（ C ）

A、吊钩上升

B、吊钩下降

C、紧急停止

D、工作结束

37、【单选题】当钢丝绳的外部腐蚀出何种现象时应报废( )。（ C ）

A、出现小于钢丝总数10%的断丝

B、表面干燥

C、表层磨损或锈蚀量达钢丝直径的40%

D、钢丝绳直径减小量小于7%

38、【单选题】起重重物时，如果手柄推到第二档，仍不能起吊，除了电气故障外，很可能是( )。（ B ）

A、停电

B、超载

C、机械故障

39、【单选题】根据规定，钢丝绳表面钢丝磨损或锈蚀量达( )时，应予报废。（ C ）

A、7％

B、10％

C、40％

40、【单选题】吊起的重物落地时，起重机应采用( )。（ B ）

A、快速落地

B、慢速落地

41、【单选题】《中华人民共和国特种设备安全法》规定,特种设备存在严重事故隐患,无改造、修理价值,或者达到安全技术规范规定的其他报废条件的,特种设备使用单位应当履行报废义务,采取必要措施消除该特种设备的使用功能,并向原登记的特种设备安全监督管理部门办理使用登记证书( )手续。（ C ）

A、吊销

B、撤消

C、注销

42、【多选题】符合制动器报废标准的有：( )。（ AB ）

A、裂纹

B、制动带或制动瓦块摩擦片厚度磨损达原厚度的50%

C、弹簧出现弹性变形

43、【多选题】道德不同于法律，不是由国家强制执行的，而是依靠：( )。（ ABCD ）

A、传统习惯

B、内心信念

C、教育示范

D、社会舆论

44、【多选题】为实现平稳起吊，操作中应：( )。（ BCD ）

A、避免用长挂绳

B、禁止快速推档

C、行走机构逐渐加速

D、禁止突然启动

45、【多选题】变幅用的液压油缸按其位置不同可以分为( )。（ ABD ）

B、后倾式
