Everyone needs a gift.

最近空闲的时候又N刷了一次《假如爱有天意》，隔了那么久没看还是被剧情感动到一塌糊涂，尤其是里面俊河把一只萤火虫当做礼物送给珠喜这个片段，真的是浪漫到不行。

这个礼物如果放到现在来看，也还是很能戳中女生的少女心。说到送礼物，像我这种极其注重仪式感的人，每年除了常规的情人节、七夕、520、生日这些，还有各种纪念日，什么相识纪念日，在一起100天，结婚纪念日等等，都愿意花费心思去给对方送上一份心仪的礼物。

当然啦，送礼物环节对于很多人来说，其实是一件挺伤脑筋的事，今天我们就来盘一盘礼物攻略~看看这些清单能不能让你豁然开朗？

👧 GIRLS

01 // 花 卉

如果你是送礼物的新手，花卉一定是不会出错的选择。哪个女孩能抗拒娇艳欲滴的鲜花攻击呢？

男生千万不要害怕送花会被女生觉得俗气，鲜花带来的是美好与活力，作为一份代表心意的礼物再好不过了~

别说送人，我平时逛街看到好看的花束都忍不住买上一把拿回家插在花瓶中，每天早上看到阳光照射在饱满的花朵上，心情都会好很多~

02 // 首 饰

首饰作为礼物也是一个相当不错的选择，而且范围很广，有手链、项链、戒指等等，是精致的女孩们都爱的小物品~

不过作为需要展示品位的装饰品，这个礼物的挑选会比较考验男生的功力，最好是从她平时的穿搭观察风格，才不会踩雷哦！

如果是比较有把握的男生，可以大胆尝试情侣首饰，既有纪念意义，实用性也很强。

03 // 香 水

我就不用说了，香水控一个，谁要是送我香水真的是送到心坎上了。而且我问了一圈身边的女生，不喜欢香水真的只是极少数而已。

香水的香型分很多种，有果香调、花香调、木香调等等，如果平时对香水没有太多研究的话，一定要先了解清楚她喜欢什么香型，毕竟味道这东西真的很私人~

我之前也分享过蛮多自用香水，大家可以去我之前的文章里找找攻略，看看能不能用香水锁住你的那个她。

04 // 口 红

如果你想从化妆品入手，又不知道种类那么多从何下手的话，那就选口红吧！

平时素颜频率再高的女生，包包里也会备上一只口红来提升自己的气色。不用担心分不清色号，网上一搜最热的前十个色号，照买就是了。

在这提个醒，什么芭比粉啊、紫色啊、荧光绿这种比较跳脱的色号，如果你女朋友不是造型师化妆师或者模特的话，最好就不要选择这个作为礼物哈，不要问我是为什么知道的。

05 // 包 包

“包治百病”这个词难道还有人没听过吗？作为造型的其中一个部分，拥有各式各样的包包确实是女生的梦想。

张智霖对袁咏仪的宠爱就是放任她买包，这件事已经在娱乐圈是出了名的，普通人当然不用那么奢侈，但如果在纪念日送给她想要很久的包包，这份用心一定会让她超感动！

我之前也写过不少大牌包包，里面的一些经典款还是很值得入手的，不仅实用，而且还有保值的作用。

👦 BOYS

01 // 篮球鞋

如果另一半是运动型的男生，那篮球鞋绝对不会送错，我身边爱球鞋的男生真的几乎占了八成。

而且男生和女生一样有收藏癖，只不过女生收藏的是口红，男生收藏的是鞋子而已。

自己的女朋友送一双篮球鞋，又满足了自己的需求，还能穿到球场上炫耀一下，简直可以说是男生最爱的礼物之一了。

02 // Switch任天堂

男生永远是小孩，无论到多少岁，游戏对于他们的吸引力永远不会消灭，所以，送一款当下最火的游戏机一定正中下怀！

买两个还可以一起玩，完全是增进感情的利器嘛，现在的工作压力辣么大，周末空闲时间一起在家玩玩游戏机也是一场完美的约会~

别再怪男朋友只顾着打游戏不理你了，加入他不就行了！

03 // 耳 机

电子产品里面比较好送的应该就是耳机了，男生好像普遍对耳机都蛮有追求的，我有个弟弟刚工作拿的第一笔工资就去买了一副死贵死贵的耳机！说是为了提高一下自己的生活品质~

而且我觉得耳机的可选择范围也挺广的，比如airpods、降噪耳机、游戏专用耳机等等，都是男生日常生活会用到的。

尤其是工作之后，一些商务耳机、蓝牙耳机实用性也很强。

04 // 手 表

男生的手表就像女生的包包一样，不仅是为了搭配造型，很多时候也是一种身份的象征。

我也很喜欢给我老公买手表，尤其是在纪念日的时候，手表代表着时间，感觉特别有意义~

如果男朋友是成熟型的商务男士建议选择比较简约一点的机械表，如果是比较开朗休闲的运动型男生，可以选择撞色运动手表~

05 // 领 带

别的不说，男生戴领带的时候实在是太帅了，有一种难以言喻的成熟魅力~

由于男生一般不会像女生一样戴那么多的首饰来装饰自己，而那条领带就是他们最好的装饰品。

不知道别人是什么感受，每次我为我老公挑领带的时候，都会有一种莫名的幸福感~

怎么样？看完我这一份礼物清单，有没有给到你送礼的启发呢？这些都是普遍比较好选，而且不会轻易踩雷的礼物，当然还有一些更有创意更奇异的礼物，下次我再开一篇来好好讲~

其实我觉得送礼物这件事并不难，只要你用心去观察对方的喜好，一般都不会出错，而且收礼物的人想要的只是那一份心意而已，很多时候与价格并没有太大的关系~

礼物是平淡生活的馈赠，可以是一朵鲜花、一个拥抱、一个亲吻，甚至只要一句简单的“我爱你”，就可以在特定的节日里拥有一份属于爱情的欣喜~千万不要吝啬爱意的表达，它就是让生活变得更美好的礼物！
