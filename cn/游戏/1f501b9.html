「DOTA自走棋」自走棋所有野怪物品爆率，你想要神装吗？

2020-10-29
| 游戏


本文转自NGA

想必大家跟作者有类似的经历，一到打野怪就开始忍不住的喊起了“爆爆爆~”，可惜人品实在是有限，经常三波怪一个装备都不给 QAQ，这篇文章给大家简单介绍下物品掉落概率和装备与怪等级的关系，希望大家以后下棋出门就刷新，一会就圣剑- -。

reddit上有一个专门分析自走棋源码的兄弟，他对于卡池、物品掉率、还有每次更新的修改都贴了lua源码验证。物品共分为四级，大致对应游戏里的几种颜色，详细分类如下：

LV1：锁子甲，圆盾，回复戒指，攻击之爪，枯萎之石，短棍，吸血面具，恢复指环，抗魔斗篷，虚无宝石，法师长袍，王冠

LV2：板甲，活力之球，阔剑，秘银锤，标枪，魔力法杖

LV3：恶魔刀锋，振奋宝石，极限法球

LV4：圣者遗物，掠夺者之斧，神秘法杖

野怪分为九级，对应的野怪如下：

LV1：天辉小兵(第一波2只，第二波2只，第三波4只)

LV2：天辉大兵(第二波1只，第三波2只)

LV3：小石头人(第十波2只)，小狼(第十五波4只)

LV4：大石头人(第十波1只)

LV5：大狼(第十五波1只)，夺命双熊(第二十波2只)，小巨魔(第四十波4只)

LV6：愤怒的枭兽(第二十五波2只)，小雷霆蜥蜴(第三十波2只)

LV7：大雷霆蜥蜴(第三十波1只)，大巨魔(第四十波2只)

LV8：黑龙王(第三十五波)

LV9：年兽(第四十五波)，肉山(第五十波)

用波数来重新排序一下野怪：

Wave 1: 2*天辉小兵(LV1)

Wave 2: 2*天辉小兵(LV1)，1*天辉大兵(LV2)

Wave 3: 4*天辉小兵(LV1)，2*天辉大兵(LV2)

Wave 10: 2*小石头人(LV3)，1*大石头人(LV4)

Wave 15: 4*小狼(LV3)，1*大狼(LV5)

Wave 20: 2*夺命双熊(LV5)

Wave 25: 2*愤怒的枭兽(LV6)

Wave 30: 2*小雷霆蜥蜴(LV6)，1*大雷霆蜥蜴(LV7)

Wave 35: 黑龙王(LV8)

Wave 40: 4*小巨魔(LV5)，2*大巨魔(LV7)

Wave 45: 年兽(LV9)

Wave 50: 肉山(LV9)

野怪等级对应的物品掉率如下：

LV1野怪：不掉80%，1级物品20%

LV2野怪：不掉60%，1级物品40%

LV3野怪：不掉50%，1级物品50%

LV4野怪：不掉40%，1级物品40%，2级物品20%

LV5野怪：不掉40%，1级物品20%，2级物品40%

LV6野怪：不掉30%，1级物品30%，2级物品30%，3级物品10%

LV7野怪：不掉20%，1级物品30%，2级物品30%，3级物品20%

LV8野怪：不掉0%，1级物品20%，2级物品40%，3级物品30%，4级物品10%

LV9野怪：不掉0%，1级物品10%，2级物品40%，3级物品30%，4级物品20%

可以看到黑龙王，年兽和肉山这三个boss是一定会掉装备的，而且4级物品只能靠黑龙王，年兽和肉山才能打出来。而巨魔一家虽然是玩家杀手，但只是LV5和LV7的野怪(真的这么弱嘛)，不会爆4级物品。
