神光同步，译名太丑了！谁能想到更好的，因为我也装了台

2020-10-29
| 游戏

以前对主机的RGB光污染不太满意，其实是有原因的。早期各种硬件都是东拼西凑的，有秒杀的，有网购二手的，也有朋友赠送的，总之很杂乱。不同品牌的RGB灯各行其事，互不理睬，一开机灯光是很炫，但是让人眼花瞭乱，真成了“光污染”。最近就想改变一下，整一套一站式RGB解决方案。一站式RGB解决方案，目前来说，市面上很多，就市场占有率，还是几个主板厂商的为主，华硕Aura Sync、技嘉Fusion、微星 Mystic等，除此之外还有海盗船的iCUE。

不过知名最高的就是华硕Aura Sync，中文名为“神光同步”。虽然，我没能想到更好的翻译，但是这个名字实在有些“不对味”。言归正传，这次装机的起因就是，我最近从一位大佬手上收到了一块信仰级主板——华硕ROG MAXIMUS X HERO。

最终配置如下：

CPU Intel I7 8700K 散片 （原有）

主板 华硕（ASUS） MAXIMUS X HERO (WI-FI AC) 玩家国度Z370 M10H

内存 阿斯加特（Asgard） 阿扎赛尔系列 【8GB DDR4 3200频率】*2 （原有）

硬盘 浦科特（PLEXTOR）M9PeG 256G （原有）

显卡 耕升 追风 960 4G （原有）

电源 冷至尊MWE 金牌 550W电源

散热 酷冷至尊冰神G120水冷散热器

机箱 酷冷至尊MasterBox MS600机箱

1、CPU Intel I7 8700K 散片

8700K是近几年英特尔少有的诚意之作，当然得感谢AMD的ZEN系列CPU。因为这都是它们给逼出来的。为了应对AMD的挑战，在七代U上市不到一年，就匆勿推出了八代，从单核性能上看，八代U很像赶工出来的产品，像是直接在七代上增加了两个核心。8700K是6核心12线程，基础频率3.7GHz，睿频4.3GHz，单核最高4.7GHz。总体性能比上一代提升不少，但首发价却相差不大。

不过，有些坑的是，与上两代酷睿采用相同的LGA1151针脚，但必然八代板才能支持（虽然有魔改版的六、七代板支持四核的八代）。还有一点就是，8700K依然是硅脂U，超频还是开盖换液金为好。不过虽然全套开盖设备，操作也很简单，不过还是有句话要提醒大家“开盖有风险”。

2、华硕（ASUS） MAXIMUS X HERO (WI-FI AC) 玩家国度Z370 M10H

华硕ROG MAXIMUS X HERO (WI-FI AC)作为玩家国度系列的一员，定位高端游戏玩家，支持智能超频，让普通游戏玩家可以无障碍体验超频。

华硕ROG MAXIMUS X HERO (WI-FI AC) 正面

华硕ROG MAXIMUS X HERO (WI-FI AC)背面

华硕ROG MAXIMUS X HERO (WI-FI AC)为八代Z370芯片组，1151接口，兼容八代酷睿。

华硕ROG MAXIMUS X HERO (WI-FI AC)提供10相数字供电模组，并为供电模组配备了镂空金属散热片。

四条DDR4内存槽，不过只支持最高双通道。华硕ROG MAXIMUS X HERO (WI-FI AC)为内存设有独立的数供电，超频突破4000Hz不成问题的。当然前提是，你的内存也得支持超频。

主板供电旁边安置着前置USB 3.1跳线接口，支持前置USB 3.1扩展。不过，我的这款酷冷至尊MasterBox MS600机箱并不支持USB 3.1扩展，只有后续自行更换了。

主板提供了两个 M.2 SSD卡槽，其中中部卡槽有一个ROG铭牌，可以作为M.2 SSD的散热片。不过，因为我主硬盘为浦科特（PLEXTOR）M9PeG 256G，自带了散热马甲，拆了不保修，所以只能插在第二卡槽了。

华硕ROG MAXIMUS X HERO (WI-FI AC)提供了共3条PCI-E 全尺寸插槽，其中2条为有Safe Slot金属装甲，另有3条PCI-E X1扩展接口。

华硕ROG MAXIMUS X HERO (WI-FI AC)主板采用了SupremeFX S1220旗舰级音频芯片、日系Nichicon音频电解电容，并配备了EMI声卡防护罩、音频分割线、双耳放等专业音频设计。支持120dB SNR立体声回放以及113dB SNR录制输入。配合华硕的声波雷达技术，可以提供分辨出声源位置，在吃鸡等FPS游戏中帮助玩家提高反应速度。

作为一款信仰级主板，ROG信仰灯是玩家国度系列的一大卖点。华硕ROG MAXIMUS X HERO (WI-FI AC)的“败家之眼”支持1680万色及多种预设灯效模式，信仰十足。AURA SYNC神光同步技术，华硕ROG MAXIMUS X HERO (WI-FI AC)板载了4PIN 5050 RGB灯带接针、3PIN RGB接针，可以通过AURA软件整体设置支持AURA SYNC神光同步技术的其它硬件打造整机光效，比如，酷冷至尊冰神G120水冷散热器 阿扎赛尔系列 【8GB DDR4 3200频率】。

华硕ROG MAXIMUS X HERO (WI-FI AC)除配置了英特尔 Ethernet Connection I219-V千兆网卡外，瑞昱 Realtek RTL8822BE 802.11ac PCIe Adapter无线网卡。无线网卡搭配有双频外置天线，并支持MU-MIMO和蓝牙技术。同时搭配LANGuard和GameFirst技术。

华硕ROG MAXIMUS X HERO (WI-FI AC)背部I/O区域采用一体化设计，预装了一体式的防尘挡板，让装机更加方便。I/O区域设有两个DIY快速按钮，一组DP/HDMI视频输出，8个不同版本r USB接口，一个千兆网络接口及一组音频接口，

3、内存 阿斯加特（Asgard） 阿扎赛尔系列 【8GB DDR4 3200频率】*2 (套条）

Azazel阿扎赛尔系列是阿斯加特推出的高端超频内存系列，我这款是8G*2 DDR4 3200MHz RGB灯条，三星原厂闪存颗粒，幻影黑的金属散热片。内存电压1.2V，CL值为17-17-17-39，支持XMP2.0，轻松超频。和我主板一样属于为有超频需求，却无超频能力的用户而设计。

内存顶部设有RGB灯带，支持1680W色。但重要是支持华硕AURA等主流灯效控制软件，结合华硕主板整体控制灯效变化，提升整机逼格。

4、 浦科特（PLEXTOR）M9PeG 256G

浦科特M9PeG 256G，Marvell 88ss1093主控，512MB缓存，东芝Bics3 3D NAND闪存原片，持续读取约3000MB/s,写入约1000MB/s，还带散热马甲。不过，马甲上贴有保修的封条，也就意味着，保修与主板的玩家国度铭牌，两者之间，只能二选一。我选保修。

5、 耕升 960 4G

很老的显卡了，性能略低于1050TI,但对我这种，基本不玩大型游戏的人来说，是基本够用了。最大的缺点是，功耗比10代的显卡高上不少。

6、 酷冷至尊MWE 金牌直出 550W电源

其实之前，就用酷妈的另一款金牌电源 V650。MWE系列定位稍低于V系列，采用的是台系精选固态电容。

酷冷至尊MWE550有直出、全模组两个版本。我这款是线材直出版，如果想更进一步玩发光线材，可以全模组另行定制线。

酷冷至尊MWE550配备120mm静音散热风扇，静音效果不错，工作时很难听过风扇的声音。

酷冷至尊MWE550的线材除主板24Pin采用蛇皮线包裹设计，其它全部采了扁平线材，背部走线，不易打结。而线材的长度完全支持大机箱走背线。接口上，有8个SATA接口，2个8pin显卡供电接口，1个8pin CPU供电接口，2个大4pin，一个小4pin，对于一般装机都是够用的。

酷冷至尊MWE550拥有80PLUS金牌认证，典型负载效率大于90%。另外，酷冷至尊MWE550的3年质保，是只换新不修，良心大大的。

7、 酷冷至尊冰神G120水冷散热器

散热，要性能，也有逼格，弄水冷是必经之路。酷冷至尊的新一代冰神G120 RGB水冷，支持了最新的Addressable RGB灯效。并支持华硕Aura Sync、微星Mystic Light Sync、华擎RGB LED等软件统一管理。

冰神G120 RGB采用一体式水冷设计（与上一代一致），水泵和风扇都配备了Addressable RGB灯珠，可由主板的灯效软件单独控制。

冰神G120 RGB作为一款120的水冷却配备两个MasterFan均衡风扇，让我有些意外。风扇转速为650-2000RPM，最大风量66.7CFM，工作寿命16万小时。

冰神G120 RGB有两个RGB风扇、一个RGB水冷头，为了方便统一定制灯效，酷冷特意附送了一条一转三ARGB分接线，可以直接插在主板的三针可编程RGB接口上。对于没有可编程RGB接口，酷冷也有WIRED ARGB专属控制器统一控制和定制灯效。

冰神G120RGB采用微流道优化技术(FlowOp)，据官方介绍，这项技术对水冷液的冷热转换过程进行了重新设计，可以将热量迅速传导到散热器内并冷却。

冰神G120RGB的水冷泵采用了分体式喷射型技术，大面积的触面，让它可以适应大尺寸的CPU..

8、酷冷至尊MasterBox MS600机箱

酷冷至尊 MasterBox MS600机箱的外观上采用了侧透设计，而非双侧透。色彩风格采用干净简洁的白色。

机箱尺寸为460 x 213 x 446 mm，内部高度可不秃顶的放入我的T610P。重量为6.35kg。

背部接口区结构和一般机箱一样，采用黑化的钢材，后置风扇，主板接口，PCI扩展，下置电源，分区明确。需求说明的是，PCI扩展区的档板是可拆装设计，方便用户多次折腾显卡等硬件。﻿

顶部是一样的白色面板，主要就是开关功能区。开关功能区以“一”字排列，从左向右依次为重启键、3.5mm音频输出、3.5mm音频输入、电源键、USB3.0*2，硬盘指示灯。﻿

机箱的左侧面是一张一体式钢化玻璃，机箱内部一目了然，适合喜欢光污染的玩家。

酷冷至尊 MasterBox MS600的风道并非常见的前进后出，而是以两侧面的边框处进风，背部出风。

酷冷至尊 MasterBox MS600设有独立的电源仓，风道不与主体干扰。机箱内部的前置的风扇架支持安装两个120/140mm规格的风扇或120/140/240mm水冷。

酷冷至尊 MasterBox MS600机箱的后部预装是一个普通的120mm风扇。作为一款侧透式机箱，标准居然不是RGB就让我有些纳闷，让我很“不开森”。

两个白色的金属片是SSD快装支架，可以把支架取下来，在机箱外部安装好，再固定到机箱上。

9、装机效果

装机是个体力活，中间略过就不细说了（主要是装机，边拍边装太麻烦）直接看成果。出于省事的考虑，我把水冷的风扇安装在了后置风扇位，毕竟原来有没有灯，不谐调。

华硕Aura Sync神光同步，把机箱内的光源统一调和，主板、风扇、冷头、灯条，不再各自为战，提升了观赏的舒适性。

华硕Aura软件预设了恒亮、呼吸、彩色循环等十种灯效，并支持高手DIY玩法。不过，就我而言还是选默认的彩色循环，个人认为这个最漂亮。

总结

最后，还是得总结一下，装机真是一个体力活，装系统更是一个烦心事。好在历经波折，成果还是美好。想想这么累，值吗？从经济的角度，真心不值。花一两百找个师傅上门装一下就好了，自己得花大半天才能完成，而且可能会返工，这时间，干其它的事赚几百元更轻松。但是，看着自己的汗水成果，成就之感爆膨。下次再装机，是自己干，还是找人呢？

在这里，有人想问，8700K+M10H怎么不玩一下超频啊。其实没有必要。8700K+M10H无须超频设置，其实就能自动跑到5.0了。再加上，太晚了，我也想睡了。如果玩，那也是以后的事了。
