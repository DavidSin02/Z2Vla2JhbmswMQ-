《热血传奇》道士技能列表(7~46级）

2020-11-14
| 游戏

治愈术: 根据修炼等级的不同，可以恢复自己或者他人的体力(HP)，每跳5点体力。

Lv.1 : 可以在第 7 级开始修炼

Lv.2 : 可以在第 11 级开始修炼

Lv.3 : 可以在第 16 级开始修炼

精神力战法: : 根据修炼等级的不同，可以提高战斗时的精确度（命中率）跟敏捷（躲闪率）

Lv.1 : 可以在第 9 级开始修炼.

Lv.2 : 可以在第 13 级开始修炼.

Lv.3 : 可以在第 19 级开始修炼.

施毒术: 根据修炼等级及道术的不同，可以延长毒药的有效时间，绿毒减血，红毒减防。道术只影响毒药时间。

Lv.1 : 可以在第 14 级开始修炼.

Lv.2 : 可以在第 17 级开始修炼.

Lv.3 : 可以在第 20 级开始修炼.

灰色毒粉

中毒的目标将会全身发绿，并且会慢慢失去它的体力。制作这种毒药,需要4片食人树的叶子，1个食人树的果实，和2颗毒蜘蛛的牙齿。每份小毒粉可以使用50次。

黄色毒粉

黄色毒粉通过侵蚀目标的武器和防具来减低防御力。毒粉同时还会破坏武器和防具的耐用性。制作这种毒药，需要4片食人树的叶子，1个蝎子的尾巴，以及蛆卵。每份小毒粉可以使用50次。

灵魂火符: 和火球术一样的远程法术，通过驱使护身符来攻击敌人。

Lv.1 : 可以在第 18 级开始修炼.

Lv.2 : 可以在第 21 级开始修炼.

Lv.3 : 可以在第 24 级开始修炼.

召唤骷髅: 一种通过使用护身符和复活力制造骷髅的法术。制造出的骷髅将会成为英雄最忠诚的跟随者。召唤出来的骷髅可以升级，一级骷髅可升3级，二级骷髅可升5级，，三级骷髅可升7级。

Lv.1 : 可以在第 19 级开始修炼.

Lv.2 : 可以在第 23 级开始修炼.

Lv.3 : 可以在第 26 级开始修炼.

隐身术: 隐身术使怪物无法察觉英雄的存在。使用这种法术后采用魔法远程攻击，是一种非常理想的单对单战斗方法（只能针对自己）。

Lv.1 : 可以在第 20 级开始修炼.

Lv.2 : 可以在第 23 级开始修炼.

Lv.3 : 可以在第 26 级开始修炼.

集体隐身术 : 更加强大的隐身术，能够隐藏3X3范围内的全体玩家。集体作战时是一种非常有用的保护措施。

Lv.1 : 可以在第 21 级开始修炼.

Lv.2 : 可以在第 25 级开始修炼.

Lv.3 : 可以在第 29 级开始修炼.

幽灵盾 : 能够提高魔法防御力。他的有效时间随着训练等级的提高而延长（道术影响持续时间）。

Lv.1 : 可以在第 22 级开始修炼.

Lv.2 : 可以在第 24 级开始修炼.

Lv.3 : 可以在第 26 级开始修炼.

神圣战甲术: 能够提高防御力。他的有效时间随着训练等级的提高，（道术影响持续时间）

Lv.1 : 可以在第 25 级开始修炼.

Lv.2 : 可以在第 27 级开始修炼.

Lv.3 : 可以在第 29 级开始修炼.

心灵启示: 能够展示目标（包括玩家和怪兽）体力。现在很少用的技能了，因为一般游戏内挂都有血量显示。

Lv.1 : 可以在第 26 级开始修炼.

Lv.2 : 可以在第 30 级开始修炼.

Lv.3 : 可以在第 35 级开始修炼.

困魔咒: 被限制在咒语中的怪兽因为看不见外面的情况只能绕着圈走。但是当英雄和其他人走进这个六角形的时候这个魔法会消失。当被困住的怪物受到外界攻击时能够躲避。即使英雄和很多人战斗的时候最好是一对一，困魔咒会发光以显示边界。

Lv.1 : 可以在第 28 级开始修炼.

Lv.2 : 可以在第 30 级开始修炼.

Lv.3 : 可以在第 32 级开始修炼.

群体治愈术: 能够恢复自己和周围所有玩家的体力。手残玩家的最爱，尤其是队友被怪物围攻的时候，用它吧，不用再担心奶不到队友了。（同治愈术一样每跳5点）

Lv.1 : 可以在第 33 级开始修炼.

Lv.2 : 可以在第 35 级开始修炼.

Lv.3 : 可以在第 38 级开始修炼.

召唤神兽: 能够召唤一只喷火的强大神兽作自己的随从。升级机制同骷髅。1级神兽攻击单位一格，2-6级攻击前方两码竖线，7级神兽攻击3格。超级肉盾，7级神兽2600点血，是个强有力的队友。有句话怎么说开着？人不如狗，嗯，用在这里最合适了。

Lv.1 : 可以在第 35 级开始修炼.

Lv.2 : 可以在第 37 级开始修炼.

Lv.3 : 可以在第 40 级开始修炼.

气功波: 过去，气功术被认为是一种内功的修炼，而并非应用于实战之中。直到一位方士，在潜心研究后，对气功术进行改良，发明了气功波，从而起到推开周围的怪物而得以防身的作用，道士的抗拒火环，挺有趣的，不像法师的那种无脑推。

Lv.1 : 可以在第 29 级开始修炼.

Lv.2 : 可以在第 31 级开始修炼.

Lv.3 : 可以在第 34 级开始修炼.

无极真气:利用大自然的真气，从而引发体内潜力，不仅可以使自身能力大大提高，还可以将宠物的能量转移给自己。这在武林流派的发展上可以说是一次里程碑。看谁不顺眼，一个无极然后一个毒教他做人，那叫一个酸爽，简直神技，啧啧……

Lv.1 : 可以在第 36 级开始修炼.

Lv.2 : 可以在第 39 级开始修炼.

Lv.3 : 可以在第 42 级开始修炼.

噬血术:驱使护身符，对敌人造成伤害。命中后，还可吸取对方生命，为自己回复一定的血量。

技能施放频率较灵魂火符有所提升，且忽略地形阻碍直接打击目标！

缺点是，属于魔法攻击，会受到魔法躲避的干扰而影响命中率

Lv.1 : 你可以在第 47 级开始修炼.

Lv.2 : 你可以在第 50 级开始修炼.

Lv.3 : 你可以在第 53 级开始修炼.
