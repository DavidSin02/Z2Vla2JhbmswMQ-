为什么说化学起源于西方“炼金术”？中国“炼丹术”表示不服！

2020-10-29
| 游戏

中国古代炼丹术是中国古代化学的重要组成部分 ,炼丹家们做了大量的实验 ,揭示了一些物质的性质，并且发明了火药、颜料等 ,取得了丰硕的成果。古代炼丹家用来炼丹的原料有 60多种 ,在炼丹的化学实践中，人们掌握了很多物质的性质、制法和鉴别方法同时还认识到氧化还原、金属置换和酸碱相互作用等化学反应。对物质性质的认识上 ,人们通过实验 ,掌握了一些物质的化学性质 。

古代炼丹炉-来源于网络

其实，与西方炼金术相比，中国的炼丹术起源更早，并且对阿拉伯炼金术和西欧炼金术产生过很大的影响，也是近代化学的源头之一。然而西方“炼金术”却发展成为近代化学，而中国“炼丹术”却逐渐消失匿迹，为什么会这样，原因可以参考陶果《中国古代炼丹家为何不研究气体 》一文，文中主要通过下面几个原因进行了分析：实验设施 、 思维特征 、 理论基础和炼丹目的等主客观方面。

其实西方很长一段时间对于中国“炼丹术”知之甚少，例如，法国化学家贝尔特罗的《炼丹术的起源》，用大量篇幅向古希腊寻根究底；英国汉学家解释耳斯的《中国与中国人》，说炼丹术是从希腊通过大夏（Bactria）传入中国的，而德国化学家丰﹒利浦曼在他的《炼丹术源流考》一书中，又说炼丹术传自阿拉伯。直到本世纪二三十年代，中国学者王琎、曹元宇等人，陆续发表了一系列中国炼丹术的英文文章，才逐渐引起西方学者的注意，这里面贡献最大的是吴鲁强教授，他将主要的中国炼丹术著作翻译为英文，在国外期刊上发表，才使得炼丹术西来学说在他们心中开始动摇。在这里，我们必须得牢记先辈学者们的贡献。祖国的强大离不开一他们的努力。

中国“炼丹术”的起源

技术基础

炼丹术准确来说起源于“采矿、冶金”，早在原始社会后期，我国有了“冶铜术”到了殷商时代便开始大量使用青铜器，到了春秋战国时期，“冶铁术”逐渐成熟，继而铁器实现了大量使用，之所以有“采矿、冶金”技术对于化学知识的积累，才有可能产生“炼丹术”。

精神基础

当然，光有化学技术是产生不了“炼丹术”的，我们还需要在精神层面上找到一个寄托，古代有一些教派组织，那个时候他们其实还不是真正的道家，主要分布在战国时期燕国齐国一带，他们将神仙学说与阴阳五行说揉和起来，如东汉末年魏伯阳在世界上第一部炼丹术著作《周易参同契》中就有“五行错王,相据以生,火性销金,金伐木荣”和“五行相克,更为父母”之类的话,明确表示炼丹术的理论基础是五行说。

还有一个精神基础就是古代各种神仙说，《山海经》中《大荒西经》记载：有轩辕国，不寿者乃八百岁。《海内经》记载 : 流沙之东，黑水之间，有山名不死之山。《庄子·逍遥游》记载:藐姑射之山，有仙人居焉。肌肤若冰雪，掉约若处子，不食五谷，吸风饮露，乘云气，御飞龙，而游乎四海之外。

有了这些荒诞不经、浪漫无际的异书华章,对仙境对神仙的记述和描绘，让大权在握,贪得无厌的统治者们想人非非，想长生不老，永掌大权，永享受荣华富贵。“上有所好，下有甚焉”，顺应帝王之梦，在地广物丰的神州便催生了一些“方术”之士。 这些“方士”都是学者，既明了满足帝王需要即可成名成家的人生之道，又深谙取类比象的科学之策，于是，他们基于对自然中不腐不朽物质的观察，萌生了这些物质中“不腐不朽“的特质会为人体所吸取而令人也变得同样“不腐不朽”的梦想，通过这些游手好闲,故弄玄虚的方士们为了迎合统治阶级的心理大力鼓吹神仙长生，说的帝王更加心痒痒，在帝王们资金与人力鼎力支持下，更加促进了对于长生不来之术的探索。这方面最大的贡献者就是大秦帝国最高统帅秦始皇。

也许是某天某个方士很是无聊，在海边散步的时候，恰好看到了他们梦寐已久的“蓬莱仙境”，现在我们给它起了一个新的名字，叫“海市蜃楼”，在术士们看来，“蓬莱仙境”是存在的，长生不老的“仙人”也是是存在的，从此掀起了战国时期求仙问药，寻找长生不老之术的高潮，这个有名的术士就叫徐福，而他的大Boss就是大秦帝国最高统帅秦始皇，《史记·秦始皇本纪》载，今闻韩众去不报,徐市(福)等以钜万计,终不得药… …”。

可以看出来，秦始皇派徐福去寻长生不老之药以失败告终。这也为“炼丹术”诞生提供了基础，俗话说，求人不如求己，聪明的术士们在寻仙达到长生不老屡次碰壁以后，继而自己研究“炼丹术”。从此，历史悠久的“炼丹术”诞生了，并且长久不衰，直到清代才逐渐衰败。

中国“炼丹术”发展历史简介

通过秦始皇时期发展，到了西汉时期，汉武帝刘彻求仙药问药更为心切，他效仿秦始皇，又有李少君所提供的封禅不死之术，不仅指派大批船队人海求仙药，还下令在沿海一带修建了许多金碧辉煌的望仙楼，年年亲自东巡，焚香设贡，祈祷神仙赐不死之药。继武帝之后、宣帝、成帝、哀帝等如法效仿，致使迷信神仙之风盛行。

魏晋、南北朝(公元265一617年)是炼丹术最活跃的一个时期，在求仙问药无果的情况下，统治者们终于转变可思路，寻不到没关系，自己动手丰衣足食，他们开始设置仙人博士，建造炼丹炉，开始着手炼制丹药。葛洪就是这时代的代表人物，葛洪字稚川，号抱朴子，是江苏句容人 ，大致生于东晋太康二年(281)，卒于咸康七年(341)年。炼丹术中葛洪是承先启后的杰出人物，要是没有他，可能炼丹之术就在北魏时期告别历史了。

葛洪炼丹-来源于网络

到了唐代，经过魏晋南北朝300多年发展与摸索，唐太宗对于道教方士尤为重视，经常找他们聊聊天，请教一些长生不老之术，也对他们极为信赖，提供有用价值的道士还会封官赐爵，《唐大诏令集》卷九载：诸郡有自古得道升仙之处, 虽令酿祭，犹虑未周，宜每处度道士二人、三人，永修香火。… …天下灵山仙迹，并宜禁断樵采戈猎… …。

盛唐30年,“炼丹术”盛行，并且因此夭亡不少人。如：唐高宗、太宗、宪宗、穆宗、敬宗、武宗、宜宗均是因服用道士炼制的“仙丹” 慢性中毒而亡；昭义军节度使李抱真因服“仙丹”，直至中腹坚硬，胀痛昏死过去。

这些帝王将相对于“炼丹术”的推进着实做了不少贡献，我们可以推测，炼丹术绝对是一个高成本的活动，必须得耗费很多财力人力。换句话说，这玩意不是一般人能够玩的起的，但是有个人，当年也曾赶上流社会的时髦风气而热衷于炼丹，然其晚年却写思归诗一首，凄凉地描写了许多人因服丹药而亡或中毒的境况：

退之（韩愈）服硫黄，一病讫不痊；

微之（元稹）炼秋石，未老身溘然；

杜子（杜牧）得丹诀，终日断腥膻；

崔君（崔元亮）夸药力，经冬不衣棉；

或疾或暴夭， 悉不过中年。

这个人就是唐代诗人白居易，其实除了诗人白居易，这种寻仙炼丹的上流社会的时髦活动怎么能缺了我们的著名诗人李白，这也促进了他很多作品的诞生。

到了宋朝,可能是见识了前朝通过吃丹药长寿者没有，但是吃丹药毒死者甚多，皇帝、道士、百姓都不敢轻易妄动去炼丹、服丹。

明朝时，炼丹术曾一度有东山再起 、 卷土重来之势，原因就是众多皇帝内心的成仙不死之梦又被唤起了，有甚者亲自炼丹误国误民，他们为了成仙，侥幸地大量服用仙药，结果仁宗、宪宗、孝宗、武宗相继服丹身亡。

到清代中叶，上至皇帝，下至百姓，对道士的方术持敬而远之之态，其结果使炼丹术失去了生长的土壤 ，自此一级不振。

对“炼丹术”贡献最大的几位古代“化学家”

自“炼丹术”诞生以来，探索阶段比较长，再加上炼丹术士们有着道教神秘色彩，对于炼丹术的记载很少，特别是实际操作过程的记录，直到公元 142 年，一位炼丹家魏伯阳先生总结之前许多“方士”延续了几百年为帝王效劳的炼丹秘术，写就了一本《周易参同契》，这是世界上最古的论述炼丹术的书。之后，南北朝的陶弘景，晋代医家葛洪着《抱朴子》，唐代药王孙思邈着《丹经内伏硫磺法》，都记载了炼丹术及其临证应用的某些经验。

中国“炼丹术”与西方“炼金术”区别。

欧洲炼丹家可能受到资本思想影响，对于长生不老兴趣不大，但是对于黄金白银却有着浓厚的兴趣，他们做梦都想着发财，所以炼金术的的目的是在“点铁成金”、把贱金属怎样来变成贵金属；

中国炼丹家的目的则是“返老还童”或“ 长生不死”，当然，这中间也有不少贪婪者，想两全其美，他们不仅想长生不死，而且还想荣华富贵，所以这期间也为假黄金的诞生贡献了他们的力量，但其间也有不少是以炼黄白为目的者。

为什么古代道士如此执著于炼制假金——金丹？对于这个问题，东晋著名的炼丹家葛洪在其著作《抱扑子·内篇》有明确的回答：“金丹之为物，烧之愈久，变化愈妙。黄金入火，百炼不消，埋之，毕天不朽。服此二物，炼人身体，故能令人不老不死”。

宋代笔记《渑水燕谈录》记载宋真宗时，有汀州人王挺，得黄白术，“尝以药金、银献上以助国费”。可以看出为了充盈国库，皇帝命人冶炼假金也是有可能的，皇帝也会把假金用来赏赐臣民。据史料记载，西汉一朝，黄金数量猛增，光是皇帝赏赐给臣子的黄金次数就达百余次，数量近90万斤。这么多金子当然不可能都是真的。

唐代小说《广异记》记载，有个叫成弼的人得了可以化铜为金的丹药，唐太宗找到他并授予其五品官，命其专造黄金，制得假金数万斤。这种假金名字叫做大唐金，还流传到国外，外国人将其当成宝物。

以上足矣证明，中国古代的“炼金术”绝对不比国外差，并且比他们的“冶金术”更加成熟。

炼丹术中蕴含的化学反应

炼丹术中经典的“九转还丹”记载了炼丹术中的硫与汞分解以及合成循环的固体化学反应，并从中提出了化学物质守恒的规律，这与现代化学通过氧/汞分解/合成循环的固体化学反应而发现化学物质守恒定律异曲同工。

在炼丹的化学实践中，人们掌握了很多物质的性质、制法和鉴别方法同时还认识到氧化还原、金属置换和酸碱相互作用等化学反应，诱发炼丹化学反应的方法（如研磨、混合、烧焙等）和装置（如炉鼎等），还发明了蒸馏、冷凝、沉淀、过滤等收集化学反应生成物的方法。

对物质性质的认识上 ,人们通过实验，掌握了铜和铁的置换顺序就是一个很好的例子 。早在汉初《淮南万毕术》中就记有“曾青得铁化为铜” ，曾青即硫铜矿物经氧化而形成的次矿物 ，主要成分是 2Cu CO3· Cu(O H) 3 ，和铁发生置换反应。

反应方程式为: 3CuCO3+2Fe+ 2H2O= 3Cu+ 2FeO( O H)+ 3CO2 （气体）

那时候，通过炼丹术，炼丹化学家们不仅创造出了许多而且也从炼丹过程中制造出了许多新的物质，例如黄铜（锌铜合金）、白铜（镍铜合金）、砷白铜（砷铜合金）、白锡银（砷锡合金）、升汞、甘汞、氧化汞、硫化汞、氧化铅、四氧化三铅、三氧化二砷、硫化锡等，这些物质有一些作为药物很早就在中医药学中得到应用。

此“丹砂HgO”非彼丹砂HgS

对于实验的观察与记录最详细的，当属于“化学家”兼炼丹家葛洪，他锐:“丹砂烧之成水银,积变又还成月砂”。从这一点看来葛洪大致做过这样的实验：就是将红色的丹砂(硫化汞)加热使它分离出水银（Hg），而水银加硫黄又能炼成赤色的丹砂。

原来丹砂遇到热可以升华，纯硫化汞（HgS）升华温度为446℃，当温度升的太快则丹砂会分解。

丹砂与空气反应方程式为：HgS（丹砂） + O2（空气） == Hg + S02 产量低

为了提高产量，当隔绝空气以后加热，他们发现汞蒸发为蒸汽，凝结在温度低的上釜上，这也是蒸馏的原型。这个实验必须在相对封闭的环境下进行。

丹砂可逆分解反应方程式：HgS（丹砂）== Hg（气体） + S（气体）。

这时候有一个人叫孤刚子发现在丹砂中加入黄帆[KFe(S04)2﹒12H2O]，黄帆加热分解产生SO3，可以使反应右一，看到这里是不是惊呆了，我们在高三才学的化学知识，他们早就用上了。原理如下所示：

反应方程式：S（气）+ 2SO3（气）=== SO2(气)

综合反应化学方程式：HgS（固）+ 2SO3（气）== Hg（气）+ SO2(气)

这样生成的HgS为黑色，经过升华才会变成红色，这就是所谓的“转”的来历。

而水银（Hg）在温度升高过快也会飞掉，在空气比较充足的，火力不猛的清情况下，又可以变为氧化汞（HgO）。

反应方程式为：2Hg + 02 == 2HgO。

其中氧化汞HgO颜色也为赤红色，跟丹砂（HgS）颜色一样，所以才有葛洪才误以为是丹砂，才有“丹砂烧之成水银,积变又还成月砂”之说。

南宋方士白玉蟾在其 《金华冲碧丹经秘旨》中记录了上火下凝的炼汞法。其反应器分上、下两部分，上部为倒置的石榴状瓷罐，内装丹砂及赤金(红铜)，下部为一高筒状柑祸，内贮华池水(含矿物质的醋)。该体系中加入了铜 。

反应方程式为 :HgS（丹砂）+Cu == CuS + Hg（气）

“还丹”与氧化汞（HgO）

炼丹离不开铅汞二元素，宋王道《金碧古文龙虎经注疏﹒序》中说：“古今学道之士，皆以铅汞为大丹药”

他又说：铅性白也而赤之以为丹,丹性赤也，而白之以为铅”。这说明在那时他就知道铅能变成赤色的四氧化三铅( Pb3O4)，而四氧化三铅,又能分解出白色的铅(Pb)来的道理。

反应方程式为：

3Pb + 2O2 == Pb3O4（黄丹主要成分）

Pb3O4 + 2C == 3Pb + CO2

魏伯阳所说：“汞见火则飞，将欲制之，黄芽（Pb3O4）为根”反应方程式如下，防止汞飞走，让与氧气化合为氧化汞（HgO）。

化学反应方程式：Pb3O4 == 6PbO + 2O2(气)

2Hg（气）+ O2（气）== 2HgO(凝于上釜内壁)

四氧化三铅从何而来？

硫化铅(PbS)是一种最早接触到的含铅矿物质，方铅矿焙烧的化学反应为：

PbS + 302（空气） == 2PbO + 3S02(气体)

唐《丹房镜源》中的造丹法载有硫化铅参加的反应：

Pb + S == PbS(黑色)

3PbS + 8KNO3 + 1/2 O2 == 3PbO(黄色) +3K2SO4 +2KNO2 + 6NO

PbS + 4KNO3 == PbSO4（白） + 4KNO2

3PbO + KNO3 ==Pb3O4（红）+ KNO2

托名苏轼所撰的《格物粗谈》中说“雌黄与胡粉 (PbCO3)同用，则两色俱变”，其反应为：

As2S3 + 3PbCO3 == 3PbS + As2O3(砒霜) +CO2

到宋代硫化铅的医疗效果逐步为医药家所认识，开始利用。

可以看出，所谓的还丹就是指HgO，汉代以来至唐代就以这种丹为主要丹药，很多术士将HgO误以为HgS长期服用，并且引起中毒。唐玄宗时代著名术士张果云：“昔有二人契为兄弟，各将水银一斤，入阳城山中烧。烧三年水银伏火如红玻璃色，光彩可爱，乎为大丹。各服二两。行履之间，须人扶持，不逾百日而死，又何益于身命。”。

可见，这里的水银伏火如红玻璃色就是氧化汞，然后兄弟入山中辛苦烧制三年，终得可爱的大丹，二人各服二两中毒，百日以后嗝屁了。

其实很早以前炼丹师就知道水银的毒性，当然，皇帝肯定也是知道的，很早以前就有用犯人来试吃丹药，但是为什么他们明知有毒，还要吃，这个可能解释起来比较复杂，可能长生不实在是诱惑太大了。再加上是慢性中毒，也就不太在意。

以硝石（KNO3）汞等份蒸之得氧化汞HgO，这个炼丹方法主要产生于宋代，说明宋代还是以HgO为主要服用丹药，这样服用百日，岂有不死之理，估计当时没有少死人。

化学反应方程式：Hg + KNO3 == HgO + KNO2 需要澄清过滤得到HgO，产量不高

由于到唐宋时期，服用丹药盛行，死了很多人，到了明代，丹药逐渐由内服转为外用。作为疡科灵药，陈实功《外科正宗》中俗称“白灵药”具体升炼方法为：

“水银二两，用铅一两化开，投入水银听用，火硝二两、绿矾二两、明矾二两，共碾为末，投入锅内化开，炒干，同水银碾细...”过程比较复杂，一起看看发生哪些反应，具体“白灵药”如何产生。

化学反应方程式：

下罐：2Al2（SO4）3 + 12KNO3 == 6K2SO4 + 2Al2O3 + 12NO2(气) + 3O2(气)

4FeSO4 + 8KNO3 == 4K2SO4 + 2Fe2O3 + 8N02(气) + O2（气）

Hg(固体) == Hg(气体)

上罐：2Hg + O2 == 2HgO

通过化学反应可以看出来所谓的“白灵药”还是HgO,但是这种制作方法已经比唐宋的制作方法先进了不少，并且纯度也要高很多。

雄黄（As2S2）、雌黄（As2S3）炼丹法

两者均为矿物质，我国利用雄黄、雌黄炼丹历史也很悠久，由于色泽金黄，似黄金，魏伯阳在《周易参同契》中说：“金性不败朽，故为万物宝，术士服食之，寿命得长久”。葛洪也强调说 : “夫五谷犹能活人，人得之则生，绝之则死，又况于上品之神药，其益人岂不万倍于五谷耶？… … 黄金人火，百炼不消，埋之毕天不朽 ”，服之则炼人身体，故能令人不老不死，此盖假求于外以自坚固 ”。这些理论都是企图将黄金的抗蚀性移植到人体中去以求长生，于是他们炼制神丹点化药金。

战国时期用于医疗，在《抱朴子·内篇》金丹卷中又戴有雌黄、雄黄的升华法：“取雌黄、雄黄烧其下，其中铜铸以为器，覆之三步醇苦酒中，百日此器皆生赤乳，长数分，或有五色琅歼，取而服之亦能令人长生 ”。

这里的雌黄是指三硫化二砷(As2S3)，雄黄是指二硫化二砷(As2S2) 。这两个硫化物加热后都能升华，赤乳就是升华的晶体。他又说：“饵服雄黄当先以硝石(KNO3)化为水，再凝后服之”。

这又说明他已知道硝酸钾能溶化雄黄为液体的化学知识。在仙药卷中葛洪又有“作雄黄水法”,他说：“治雄黄内生竹篇中，一斤加硝石二两，覆荐上，下封以漆骨丸，内醇苦酒中埋之，深三尺，二十日即化为水，作曾青水及矾石水同法，但各异笛中耳 ”。这也是他懂得醋酸能溶解雄黄为水的化学表现。

雄雌黄色似金，故颇受丹家重视。炼丹家对其进行炼制，得到单质砷，且出现火药的原始配方闭。葛洪《抱朴子内篇》中提出以雄黄与硝石( KNO3)、玄胭肠(猪大肠-含有碳)、松脂三物合炼，经赵匡华等模拟实，采用该法可得到银白如冰霜的单质砷。

具体化学反应为：2As2S2 + 8KNO3 + 7O2 == 2As2O3(砒霜) + 4K2SO4 + 8NO2(气)

As2O3 + 3C == 2As + CO(气体)

这是世界上最早制造出的单质砷（药银）。

在制造所谓的“金、银”过程中，可能无疑中加入硝石量稍大，引发炼丹炉爆作，火药原始配方就此诞生，东汉时期的魏伯阳在其专门论述炼丹术的著作《周易参同契》里就有“药物非种，分剂参差，失其纪纲，飞龟舞蛇，愈见乖张”的记载，形象地记载和描述了炼丹变为火药而爆炸的情形。

唐代药王孙思邈，显然是注意到了炼丹过程中打火的现象，因而将把硫磺、硝石放在硝银锅里炼制而成的药称为“着火的药”。之后，火药曾被古中国人智慧地应用于烟花爆竹、武器、医药和雏形火箭四个领域，并在元朝时期传至西方。

砷能与其他金属形成合金，含量不同，颜色可能各异，这在炼制雄雌黄时就会表现出来，方士们以为制得了黄金、白银，加之他们以为服食黄金可以“假外物以自坚固”，所以对其颇感兴趣,这就是“外丹黄白术”。孤刚子在《五金诀》中指出：“雄黄功能变铁，雌黄功能变锡，砒黄(不纯的天然氧化砷矿石)功能变铜，硫黄功能变银化汞。四黄功亦能变铁为铜，反铜为银，反银为金。如谷作米，是天地之中自然之道”。可见在当时利用四黄使金属转化在方士看来已相当容易。

利用雄雌黄最早得到的“黄金”是含砷量小于10%的黄色铜砷合金，即砷黄铜。葛洪《抱朴子内篇》中载有制造砷黄铜的方法，先将雄黄与牛胆汁做成粉，然后将其与石胆(硫酸铜)、炭末和戎盐 ( NaCl)混入赤土釜中加热，雄黄和石胆被还原成砷和铜，最后形成砷黄铜。苏元朗《宝藏论》中不但载有“金”，而且也载有“银”(砷白铜,含砷量>10%的铜砷合金)的炼制方法，以草药(K2CO3 )与雄雌黄加热合炼,分别制得砷酸钾和亚砷酸钾。其反应大致为:

2As2S2 + 6K2CO3 + 3O2 == 3K3AsO4(砒霜) + K3AsS4 + 6CO2(气)

As2S3 + 3K2CO3 == K3AsO3 + K3AsS3 + 3CO2(气)

再利用熔化的砷酸钾、亚砷酸钾可点化铜成砷黄铜和砷白铜，至于是否能点银成金，还需验证。

神秘的炼丹原料---“秋石”

炼丹大部分原料是天然矿物质，而有一种原料很有意思，那就是“秋石”，最早对于其没有记载，大家都不知道是个什么东东，但是到了唐代以后，终于知道了“秋石”是个什么鬼，原料是人尿等物中提炼出的含有雄性激素的石盐。

下面看看宋初问世《苏沈方良》中记载的具体制作方法，这里摘录其一 :“小便不计多少,大约两桶为一担。先以清水按好皂角浓汁，以布绞去滓。每小便一担捅，入皂角汁一盏。用竹蓖急搅拌，令转千百遭乃止。直候小便澄清，白浊者皆淀底，乃徐徐撒去清者不用，只取浊脚，并作一担捅 ....十数担不过取得浓脚一二斗 ...。 其小便，须是先以布滤过，勿令有滓。...如熬干色末洁白，更准前丁淋，直候色如霜雪即止 ...”。

后面不写了，反正看了这个过程，除了对古人的佩服还是佩服，这个制作工序甚是繁琐，同时也涉及到很多化学实验操作，比如加热蒸发，过滤...

神秘的“六一泥”

古人在炼丹的过程中，已经注意到了隔绝空气（氧气）的重要性，所以丹炉需要密封，所以封丹炉，是炼丹成败的关键之一。因此，方士们对此十分重视，研究和运用了许多方法，其中最有效的方法就是“六一泥 ”。据葛洪 《抱朴子内篇》记载，“六一泥”主要是用：戎盐、卤盐、矾石、牡砺、赤石脂、滑石等药物组成。具体怎么做还没来得及研究，知道的可以补充下。

炼丹术精神

中国古代炼丹家虽然未能达到炼仙丹和炼黄金的幻想目的，却作出了为人民有利的贡献，现代化学的原始萌芽时期。因他们在实践中运用了自然界的许多矿物、通过长时期实验,发现了不少物质的性能和变化，故可利用这些天然财富来为广大人民服务。

他们虽然没有炼成令人长生不死的仙丹，却研究成功为人民治病的药物，如水银、硫黄、砒石等的炼成药物，就是他们的光荣成就。此外，还有火药和许多颜料、合金等物的发明，也与炼丹家的实践有一定关系。

暂且不说炼丹术功效如何，但其中有些积极意义是不可否认的， 那就是炼丹所要达到的目的是长生不老，羽化升天，很大程度上体现着古代人深信人们能摆脱自然界束缚，人的幸福不是建立在来世而是建立在今生。这种以人为本，人定胜天的思想，在鼓励炼丹活动的同时也极大地丰富了人类的想象力。

炼丹家们在条件如此原始的基础上孜孜不倦的探索精神也是非常值得我们学习的，特别是现代科学家们，能够脚踏实地，静心探索可能有所发现，为人类做出更多的贡献。

炼金术在西方国家的兴起和发展。基于欧洲文化中固有的探索和科学精神，古欧洲人意识到并从炼金术中不断总结出了化学原理，出现了像玻意耳、拉瓦锡和道尔顿这样早期的著名化学家，建立了近代化学的基础。
