德鲁伊近战怀旧服攻略：移速最快的输出职业，能跟上我的速度吗？

2020-10-29
| 游戏

一、 开始之前

德鲁伊是游戏中的一个全能职业，既可以作为治疗和坦克，又可以切换为远程近战输出。如果你选择成为一名近战德鲁伊的话，那么你就等于能同时玩两种形态的德鲁伊——熊形态（坦克）、猎豹形态（近战DPS）。近战德鲁伊需依靠猎豹形态专属技能进行输出，所以玩家也喜欢把近战德鲁伊叫做猫德。猫德的技能有点类似于盗贼，需要靠技能生成连击点数，在依靠连击点数施放终结技能。本篇攻略中，我们将针对坦克德鲁伊在PVE副本中的能力进行详细的研究。

1、 猫德的优劣

Ø 优势

n 多功能性

n 实用性强

n 合理搭配天赋能同时玩近战和坦克

n 游戏中移速最快的职业

Ø 劣势

n 伤害偏低

n 输出手法繁琐

n 装备竞争大

2、 猫德的种族推荐

与其它职业不同的是，德鲁伊在游戏中仅两个种族可以选择，联盟为暗夜精灵，部落为牛头人。

3、 猫德的专业推荐

猫德最好的专业是工程学，该专业所制造的物品能提高你的输出能力以及应对一些副本内的困境。其次是制皮专业，该专业所制造的魔暴龙套装，无论是自己用还是出售都是非常不错的选择。同时还能制作最好的皮甲头盔——狼头之盔。

三、 天赋推荐

1、 通用德鲁伊天赋（14/32/5）



这套天赋没有多少可以替换的地方，基本上每个天赋都有实际的作用。首先野性天赋下的31点是必须的，因为该天赋最后一层的“兽群首领”对整个团队的收益非常的大。恢复天赋下5点的“激怒”也会影响你的输出。最后剩下的点数需加在平衡天赋下，“清晰预兆”和“自然变形”能为你节省不少的资源。

2、 DPS/坦克混合天赋（11/35/5）



适用於坦克和输出之间的切换，当然相比单纯的坦克或输出天赋来说都会相对较弱，好处就是既能玩坦克又能玩输出。

四、 输出循环

Ø 单目标

n 精灵之火，如果你是队伍中唯一的德鲁伊，记得时刻监控这个debuff的持续时间。

n 猛虎之怒起手。

n 撕碎，这个将是你的主要输出及连击点生成技能。

n 凶猛撕咬，主要使用的终结技能。

n 在能量不够使用撕碎并且要进行输出时，使用爪击。

Ø 如果你觉得上面的手法还有些麻烦，还有更简单的

n 撕碎X2

n 熊/豹形态切换

n 撕碎X2

n 熊/豹形态切换

n 撕碎

n 凶猛撕咬

n 熊/豹形态切换

Ø 多目标

猫德并没有很好的AOE技能。可以借助工程专业道具增强AOE输出能力。

n 地精工兵炸药。

n 飓风。如果聚集了大量敌人的话，飓风还是能够提供不俗的伤害，但大多数时候用于降低对方的攻速。
