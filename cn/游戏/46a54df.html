so far 不单单指“很远”

2020-10-29
| 游戏

有同学问了一个很好的问题。我们经常能听到 “so far” 这个短语，这是指“到目前为止”的意思，但是你知到怎么去用这个词吗？如果说“到目前为止，我一直工作的很辛苦”，是不是可以用这个词？我们将通过例句讲解 so far 的几个常用的意思和用法。

短语 “so far” 主要有三个含义：表时间、表距离和表程度。

用法讲解（1）—— 表时间

“到目前为止，我一直工作的很辛苦”这句话可以说：

  • So far，I've been working very hard day and night.

此处的so far就是今天我要讲解其三个意思中的第一个。它强调的是时间切入点“到目前为止”，不包括将来。我们看看例句；

  • This is the best score so far, but I will keep trying.（这是迄今为止的最好成绩了，不过我会继续努力的。）
  • So far there's been no news about our moving to Manchester.（到目前为止，没有听到我们搬迁到曼城的消息。）
  • So far, so good. I've got two A's and one B in my exams.（迄今为止， 一切顺利。我的考试两门得A，一门得B。）

请注意，在使用 so far来表达“迄今为止” 的时候，句子一般使用现在完成时态。

用法讲解（2）——表距离

下面我们转看 so far的第二个意思。表示“距离 to this or that point，到某处为止”。请看例句。

  • I can only walk with you so far as I'm running late.（我只能跟你走到这儿了。我要晚了。）

用法讲解（3）——表程度

最后我们一起看so far的第三个意思，表示“程度 to a limited extent”. 表示“有限的、最多不过如此的”。下面看两个例句。

  • I can only take you so far with my advice.（我只能给你这些建议了。）
  • You can only go so far on ten pounds.（靠10镑钱能做得事情太有限了。）

好了。以上给大家讲解了 so far 的三个主要意思：第一表示时间，“迄今为止”；第二，表示距离，“到某处为止”；第三，表示程度，“最多不过如此”。希望大家把例句多看两遍，以便加深印象。
