我的世界1.15.2拔刀剑终极武器合成笔记

2020-10-29
| 游戏

最近在小伙伴的安利下，试玩了一下拔刀剑这个mod。1.15的拔刀剑跟之前版本不太一样，经过一番折腾，终于把五把终极武器给合出来了。

下面这次合成的笔记，于己备忘，于人则是分享：

（1）mod版本号

Forge版本：forge-1.15.2-31.2.0

拔刀剑版本：SlashBlade-1.15-0.0.6

（2）五把终极武器

  • item.slashblade.yamato
  • -Chizuru- Muramasa
  • -Haze- SakuraFox
  • -Weiss- SakuraFox
  • Noble -Tukumo- Violet

（3）六种耀魂

  • Tiny Proud Soul（破碎的耀魂）
  • Proud Soul（耀魂碎片）
  • Blade Soul Ingot（耀魂铁锭）
  • Blade Soul Sphere（耀魂宝珠）
  • Blade Soul Crystal（耀魂结晶）
  • Blade Soul Trapezohedron（耀魂偏方三八面体）

（4）耀魂升级配方

  • 2 x 破碎的耀魂 => 1 x 耀魂碎片（工作台）
  • 2 x 耀魂碎片 + 1 x 铁锭 => 1 x 耀魂铁锭（工作台）
  • 1 x 耀魂铁锭 => 1 x 耀魂宝珠（熔炉）
  • 1 x 耀魂宝珠 => 1 x 耀魂结晶（高炉）
  • 1 x 耀魂结晶 => 1 x 耀魂偏方三八面体（烟熏炉）

（5）九种Blade Component

  • Blade Component -Bamboo-
  • Blade Component -Imitation-
  • Blade Component -White-
  • Blade Component -Iron-
  • Blade Component -Ruby-
  • Blade Component -Tukumo-
  • Blade Component -BlackFox-
  • Blade Component -WhiteFox-
  • Blade Component -Muramasa

（6）Blade Component 合成配方

  • 竹子 + 竹子 + 箱子 + 破碎的耀魂 => Blade Component -Bamboo-
  • 鸡蛋 + 铁粒 + 线 + 箱子 + 黑色燃料 + 纸 + 破碎的耀魂 => Blade Component -Imitation-
  • 耀魂铁锭 + 耀魂铁锭 + 箱子 + 金锭 => Blade Component -White-
  • 烈焰棒 + 耀魂宝珠 + 青金石块 + 箱子 + 煤炭块 + 线 + 金锭 => Blade Component -Iron-
  • 耀魂铁锭 + 耀魂铁锭 + 箱子 + 耀魂结晶 => Blade Component -Ruby-
  • 绿宝石块 + 耀魂宝珠 + 钻石块 + 红石块 + 箱子 + 青金石块 + 铁块 + 耀魂宝珠 + 金块 => Blade Component -Tukumo-
  • 石英块 + 羽毛 + 烈焰粉 + 箱子 + 耀魂结晶 + 小麦 + 黑曜石 => Blade Component -BlackFox-
  • 黑曜石 + 羽毛 + 烈焰粉 + 箱子 + 耀魂结晶 + 小麦 + 石英块 => Blade Component -WhiteFox-
  • 灵魂沙 + 灵魂沙 + 箱子 + 耀魂偏方三八面体 => Blade Component -Muramasa-

（7）Noble -Tukumo- Violet 合成配方

  • 任意木块 + 任意木块 + 木剑 + 破碎的耀魂 => Anonymity -Wood-（工作台）
  • Anonymity -Wood- + Blade Component -White- + 5级经验 => Sharpness -White-（铁砧）
  • Sharpness -White- => Sharpness -White- Broken（疯狂杀怪）
  • Sharpness -White- Broken + Blade Component -Iron- + 10级经验 => SlashBlade --（铁砧）
  • SlashBlade -- + Blade Component -Tukumo- + 20级经验 => Noble -Tukumo- Violet（铁砧）

（8）-Haze- SakuraFox 合成配方

  • 任意木块 + 任意木块 + 木剑 + 破碎的耀魂 => Anonymity -Wood-（工作台）
  • Anonymity -Wood- + Blade Component -Bamboo- + 1级经验=> Anonymity -Bamboo Light-（铁砧）
  • Anonymity -Bamboo Light- + Blade Component -Imitation- + 2级经验 => Noted -Silver Bamboo Light-（铁砧）
  • Noted -Silver Bamboo Light- => Noted -Silver Bamboo Light- Broken（疯狂杀怪）
  • Noted -Silver Bamboo Light- Broken + Blade Component -Ruby- + 15级经验 => Sharpness -nameless- Ruby（铁砧）
  • Sharpness -nameless- Ruby + Blade Component -BlackFox- + 20级经验 => -Haze- SakuraFox（铁砧）

（9）-Weiss- SakuraFox 合成配方

  • 任意木块 + 任意木块 + 木剑 + 破碎的耀魂 => Anonymity -Wood-（工作台）
  • Anonymity -Wood- + Blade Component -Bamboo- + 1级经验=> Anonymity -Bamboo Light-（铁砧）
  • Anonymity -Bamboo Light- + Blade Component -Imitation- + 2级经验 => Noted -Silver Bamboo Light-（铁砧）
  • Noted -Silver Bamboo Light- => Noted -Silver Bamboo Light- Broken（疯狂杀怪）
  • Noted -Silver Bamboo Light- Broken + Blade Component -Ruby- + 15级经验 => Sharpness -nameless- Ruby（铁砧）
  • Sharpness -nameless- Ruby + Blade Component -WhiteFox- + 20级经验 => -Weiss- SakuraFox（铁砧）

（10）-Chizuru- Muramasa 合成配方

  • 任意木块 + 任意木块 + 木剑 + 破碎的耀魂 => Anonymity -Wood-（工作台）
  • Anonymity -Wood- + Blade Component -White- + 5级经验 => Sharpness -White-（铁砧）
  • Sharpness -White- => Sharpness -White- Broken （疯狂杀怪）
  • Sharpness -White- Broken + Blade Component -Iron- + 10级经验 => SlashBlade --（铁砧）
  • SlashBlade -- + Blade Component -Muramasa- + 20级经验 => -Chizuru- Muramasa（铁砧）

（11）item.slashblade.yamato 合成配方

  • 紫珀块 + 紫珀块 + 龙首 + 耀魂偏方三八面体 => item.slashblade.yamato（工作台）
