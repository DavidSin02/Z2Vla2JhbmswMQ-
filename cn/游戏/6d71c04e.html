魔兽世界怀旧服法师攻略大全：怀旧服最强输出职业是怎样的？

2020-10-29
| 游戏

法师会用神秘的咒语摧毁敌人。虽然他们有着威力巨大的攻击性法术，但法师并不强壮，装甲也很单薄，这使得他们面对近距离攻击时非常脆弱。精明的法师会精心使用法术，将他们的敌人限制在一定距离或者控制在适当的位置。

一、 法师的基础信息

Ø 可选种族：人类、侏儒、亡灵、巨魔。

Ø 作用：远程伤害，AOE伤害。

Ø 资源：生命值、法力值。

Ø 护甲类型：布甲

Ø 武器：法杖、魔杖、匕首、剑和副手物品。

Ø 属性：智力、命中、爆击和法术伤害加成。

Ø 优势：出色的单体伤害、AOE伤害和群体控制。通过风筝敌人提高生存能力。

Ø 劣势：缺乏有效的战斗回复能力，法力值就像生命值一样重要。

Ø 特殊能力：可以制作食物和水，并且能为队友开启传送门。

Ø 适合的专业：裁缝、附魔、工程。

二、 法师技能介绍

Ø 攻击法术

n 火球术：火法核心技能，伤害高但消耗也高，能造成额外的火焰伤害dot。

n 寒冰箭：冰法核心技能，伤害高，能对敌人造成减速。

n 火焰冲击：瞬发的火焰伤害技能，适用于PVP。

n 奥术飞弹：法师的第一个引导读条技能，考虑到法力消耗问题，性价比不如火球术和寒冰箭。

n 魔爆术：法师的第一个AOE技能，在学习暴风雪前非常好用。

n 暴风雪：法师最强AOE技能，搭配“强化暴风雪”天赋，在能打伤害的同时还能减速控制敌人。

n 冰锥术：对面前锥形范围内造成高额的冰霜伤害，搭配“碎冰”和“强化冰锥术”，清怪效率非常高。

n 烈焰风暴：AOE火焰伤害，同时还会对受到伤害的敌人施加一个火焰dot。

n 炎爆术（火焰天赋）：法师的代表技能，施法时间有点长，但伤害也是巨额的。推荐在先手或配合变羊术时使用。

n 冲击波（火焰天赋）：非常强大的瞬发AOE法术，造成伤害的同时减速敌人。

n 灼烧：施法时间非常短，搭配“点燃”和“强化灼烧”天赋非常适合在团本中使用。

Ø 防御法术

n 霜甲术：法师可用的第一个魔法护甲法术。提高护甲值，能对任何近战攻击的敌人进行减速。

n 冰甲术：霜甲术的升级版，具有霜甲术的效果并且还额外提供了冰霜抗性。

n 魔甲术：提供额外的魔法抗性，并且能在任意情况下保持30%的法力回复。在法师施放法术时都能保持法力回复。

n 法力护盾：能为法师吸收一定的物理伤害，每吸收1点伤害消耗2点法力值。蓝量消耗巨大，酌情使用。

n 寒冰屏障：能为法师提供短暂的无敌效果，冷却时间过长，可通过“急速冷却”天赋降低冷却时间。

n 寒冰护体：吸收一定的伤害，并且在护盾的持续效果内，法师的施法就不会打断。由于在冰霜天赋最后一层，故只适合冰法。

Ø 辅助技能

n 奥术智慧：为一个目标增加一个智力buff。可通过“奥术光辉宝典”升级为奥术光辉，为整个队伍提供智力buff。

n 变形术：非常实用的控制类法术。

n 冰霜新星：强大的AOE控制法术。结合暴风雪能对任何近战敌人造成毁灭性的打击。

n 缓落术：降低坠落速度，在阿拉希盆地和安其拉非常有用。

n 解除次级诅咒：为目标移除一个诅咒，适合在PVP中应对术士。

n 侦测魔法：可以让你查看目标敌人身上拥有的BUFF，适用于PVP。

n 魔法增效：提高目标受到的魔法伤害和治疗效果。在应对只能造成物理伤害的敌人时非常有用。

n 魔法抑制：与魔法增效的效果相反，降低目标受到的魔法伤害和治疗效果。

n 唤醒：8秒内恢复大量的法力值，在长时间的战斗中非常有用。

n 闪现术：接触控制效果和避免坠落伤害，甚至是赶路的绝佳技能。

n 防护火焰结界：吸收火焰伤害，在应对高火焰伤害的敌人时非常有用。

n 防护冰霜伤害：吸收冰霜伤害，在应对高冰霜伤害的敌人时非常有用。

n 法术反制：法师唯一的打断法术。

n 传送术：允许法师传送至阵营主城。

n 传送门：允许法师创建一个传送门将队友传送至阵营主城。

n 造食术与造水术：能制造出食物和饮料，在练级时能节约买食物和饮料的钱，可通过法师训练师升级技能，以获得最好的食物和饮料。

n 制作魔法宝石：能为法师制造能类似法力药水的法力宝石。

n 气定神闲：能使下一个施法时间低于10秒的技能变为瞬发，搭配寒冰箭、火球术、炎爆术等法师能提供额外的爆发伤害，或者搭配变形术对敌人进行快速控制。

n 奥术强化：为法师提供额外的法术伤害，但法术消耗的法力值也会提高，最好搭配寒冰箭等法力消耗低的法术。位于奥术天赋树最后一层。

n 急速冷却：使用后所有冰霜类法术立即结束冷却。

n 燃烧：激活后提高所有火焰系法术的爆击机率。在火焰天赋树最后一层。

n 暗影抗性:暗影抗性提高10点

n 被遗忘者的意志：激活后对魅惑，恐惧和催眠免疫或解除这些效果，持续5秒，2分钟CD

八、 法师PVE天赋推荐

由于当前版本的后期怪物中多数具有火抗，所以非常不建议玩火焰系的法师。

1、 奥术冰霜系31/0/20

最适合在地下城和团本中使用的天赋，这套天赋能让你持续地使用寒冰箭，而不用过度地考虑法力值的问题，并且由于当前版本的熔火之心以及黑石山的一系列副本怪物都具备火抗，因此冰霜法术是最适合的输出法术。

2、 冰法19/0/32

这套天赋以“深冬之寒”为核心构建，提升所有冰霜法术的暴击率。总体造成的伤害可能不如上一套天赋，但是此天赋树下的“寒冰屏障”和“寒冰护体”也为你提供了一定的生存能力，在团本中能减轻治疗的压力。

九、 法师PVE输出循环

在进入战斗前先确认是否已经有“奥术智慧”BUFF，然后根据BOSS的情况确定是否需要为队友提供“魔法增效”或“魔法抑制”。

Ø 单目标

n 寒冰箭

n 当“深冬之寒”叠到5层后，使用任何能增伤的物品或技能。

n 继续寒冰箭

n 如果有学习“火焰冲击”或“魔爆术”的话可以用于残血收割。

Ø 多目标

n 起手暴风雪

n 当怪物靠近时使用冰霜新星→闪现术拉开距离，或者使用冰锥术。

n 继续暴风雪

n 用魔杖或“魔爆术”进行残血收割。

十、 法师PVE属性优先级

Ø 命中率

n 怀旧服中法术命中率的上限为99%，在我们60级时面对63级团本BOSS时，法术命中率仅为83%。所以我们还需要额外地增加16%的命中率才能保证输出的最大化。

Ø 法术强度

n 法术强度能为提高法术技能的伤害，但是施放时间越短的技能，法术强度的加成就越低。

n 寒冰箭：额外增加81.4%的法术伤害。

n 火球术：额外增加100%的法术伤害。

n 灼烧：额外增加42.8%的法术伤害。

Ø 爆击率

n 由于我们的天赋中点出了“寒冰碎片”，所以爆击率对我们的收益也是非常高的。

Ø 智力

n 法力值可以提高法力值上限和爆击率。

n 1智力=15点法力值

n 59.5智力=1%爆击机率

Ø 法术穿透和精神

n 法术穿透可以降低敌人的法术抗性，精神能提供法力值恢复。

十四、PVP天赋推荐

1、 生存向（17/0/34）

这套天赋偏向于控制对手以及提高自身的生存能力。通过各类冻结效果和法术反制进行控制，依靠碎冰和强化碎冰术进行消耗，寒冰护体、寒冰屏障和奥术活力为法师提供了极强的生存能力。

2、 进攻向（0/24/27）

这套天赋放弃了过多的生存能力，将伤害进行了最大化。冰霜新星+冰锥、冲击波能造成大量的AOE伤害。炎爆术配合变形术，在1V1的情况下非常厉害。

十五、PVP消耗品推荐

Ø 瑟银手榴弹：造成AOE伤害，使对方昏迷。工程制作。

Ø 铁皮手雷：造成AOE伤害，使对方昏迷。工程制作。

Ø 地精工兵炸药：对敌方造成AOE伤害，对自己也造成伤害，极限一换多。工程制作。

Ø 自由行动药水：免疫昏迷和移动控制效果，但不会解除已有的控制效果。炼金制作。

Ø 活力行动药水：免疫并解除昏迷和移动控制效果。炼金制作。

Ø 有限无敌药水：6秒内免疫物理攻击。炼金制作。

Ø 滋补药剂：每5秒驱散身上一个负面效果，持续30秒。炼金制作。

Ø 厚符文布绷带：8秒内恢复2000生命值。急救制作。

十六、其它

1、 专业推荐

怀旧服法师PVE最佳专业是工程和裁缝。怀旧服的工程专业能提供不少伤害道具或者控制道具，而裁缝制作的血藤套装，因为带有命中等属性，对法师前期来说非常好用。

2、 武器训练师

在怀旧服中，法师的初始可用武器仅为法杖和魔杖，需要去武器训练师训练才可以使用单手剑和匕首。需要注意的是，魔杖的近战攻击非常方便你对低血量的怪物进行补刀，所以一把好的魔杖也是很有用的。

怀旧服里的武器训练师如下：

Ø 达纳苏斯57,46：匕首

Ø 铁炉堡62,89：匕首

Ø 暴风城57,57：匕首、单手剑

Ø 奥格瑞玛81,19：匕首

Ø 幽暗城57,32：匕首、单手剑
