古玉热潮遇上「有凤来仪」｜中文大学楚文化玉器展

2020-10-29
| 游戏

近来古玉在市场上愈来愈受欢迎，但收藏鉴赏不能只着眼潮流，一定要以学术为基础。中大文物馆的「有凤来仪」，正展览精选158组曾国和楚国玉器，时代属两周至战国中晚期，所有展品均首次在内地以外展出，想学习玉器知识的朋友可别错过喔。

楚国是西周初年封国之一。经历代楚君开疆拓土、励精图治，跻身「春秋五霸」、「战国七雄」。盛极时，势力遍及南部大半个中国。随之而形成强大而独特的楚文化，于春秋、战国之时，不仅与中原文化南北抗衡，影响更及于长江下游之吴越、黄河下游之齐鲁，流风所及乃至于汉代，盛行长达八百余载。

战国 玉透雕对凤珮；九連墩墓出土；4.8 x 2.5 x 0.5 cm

展览上的玉器均在湖北出土，为湖北省博物馆所藏，曾国的包括著名的曾侯乙、郭家庙；楚国的包括山湾、姚家港、曹家港、九连墩玉器。九连墩为目前出土玉器数量最多的楚国墓葬，而郭家庙墓地于2015和2016年两度入选全国十大考古发现。

战国早期 玉剑；曾侯乙墓出土；33.6 x 5.1 x 0.5 cm

是次展览正以战国中晚期九连墩玉器为中心，旨在探讨和展示楚文化玉器风格特征及其形成过程，以及与中原风格的曾国玉器之关系。展览首次依据展品的出土位置进行展陈及组合，有助参观人士了解楚国葬玉习俗及玉器的功能。

战国 玉龍凤合体珮；九連墩墓出土；6.8 x 2.9 x 0.3 cm

战国 玉三人立豕珮；九連墩墓出土；5.1 x 3 x 0.4 cm

此外，为配合展览，中大文物馆亦特意出版导览手册，并将透过艺术体验工作坊，以剧场导赏结合艺术创作的形式，为公众带来崭新参观体验，加深参加者对楚国文化玉器的认识。所有活动费用全免，唯名额有限，先到先得，额满即止。
