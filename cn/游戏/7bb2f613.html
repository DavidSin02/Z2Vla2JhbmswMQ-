8090年代中国大陆引进的美剧（美国电视剧）（原创）

2020-11-14
| 游戏

1. 《大西洋底来的人》，1977年上映，1980年1月在央视播出，共17集。

中国大陆引进的第一部美剧。

主演: 帕特里克·杜菲 / 阿兰·弗吉 / 贝琳达·蒙哥玛丽 / 琼·玛丽·霍恩 / 维克多·布鲁诺

讲述了一个神秘的晚上，海底巨浪把奇异的生物麦克·哈里斯送到岸上。当医学界视之为死亡而无能为力时，海洋学家伊莉沙白·玛丽博士把他放回海洋，才使他得以复活。麦克接受了一连串的试验，以测验他的速度、灵敏度和力量。

2. 《加里森敢死队》,1967年上映，1980年10月在央视播出，共26集。

中国大陆引进的第二部美剧。央视播了16集后被禁播。

主演: 罗恩·哈珀 / 西萨尔·达诺瓦 / 克里斯托弗·凯瑞/鲁迪·索拉里/布兰登·波恩

剧中讲述了一群来自美国的失足青年，他们行为放荡不羁，身世恶贯满盈，却不断建立功勋，成为英雄的故事。

3. 《240拯救队》，1979年上映，1980年广东电视台引进播出，共13集。

主演: 约翰·贝内特·派瑞 / 马克·哈蒙 / 乔安娜·卡西迪 / 帕梅拉·亨斯利 / 简妮丽·哈里森

应洛杉矶警长紧急服务部的要求。 该团队利用直升机，船只和定制的福特野马，设法从各种危险情况中救助不幸的公民。

4. 《特警队》，1975年上映，80年代由广东电视台引进播出，共12集。

主演: 史蒂夫·福雷斯特/罗伯特·乌里奇/罗德·佩里

该系列的特色是虚构的W.C.P.D.特种武器与战术小组的任务。 他们是一支训练有素，全副武装的警务人员队伍，其目的是在敏感局势和可防御地点，对武装和危险罪犯进行协同攻击。

5.《火星叔叔马丁》，1963年上映，共37集。

主演: 雷·沃尔斯顿 / 比尔·贝克比 / 帕梅拉·布里顿

主要讲述来自火星的Exodus因事故落到地球上，此后跟记者蒂姆·欧哈利一家同住。

6.《糊涂侦探》，1965年上映，共30集。

主演: 唐·亚当斯 / 芭芭拉·费尔顿 / 爱德华·普拉特 / 理查德·高蒂埃 / 布特·穆斯汀

麦克斯维是美国特工局的86号特工，与邪恶的混沌组织做着不懈的斗争。使用特种部队的“独特”武器，与他的搭档-99号特工一次次的击碎了混沌组织毁灭世界的阴谋。

此剧有个万物皆是电话的桥段：皮鞋、皮带、领带扣、钢笔、花瓶、烛台、门把手。。。

7.《时间隧道》，1966年上映，1990年在央视播出，共30集。央视只播出了18集。

主演: 詹姆斯·达伦 / 罗伯特·科尔伯特 / 李·麦瑞威瑟

科学家托尼·纽曼和道格·菲利普斯他们发明了时间隧道，结果一次意外被陷入了时间流中，并出现在著名的不同历史时期。

8.《不可能完成的任务》，1966年上映，共28集。

主演: 格雷格·莫里斯/彼得·卢普斯/彼得·格雷夫斯/芭芭拉·贝恩/马丁·兰道

这是一个由政府间谍和专家组成的团队，他们专门负责解决最高难度的机密事件，不仅任务给的机密，而且成败自行负责，政府完全不承认有这个组织和任务。

9.《星际旅行》，1966年上映，共29集。

主演: 伦纳德·尼莫伊 / 威廉·夏特纳 / 德福雷斯特·凯利 / 尼切尔·尼克斯 / 詹姆斯·杜汉

它是一个描述一艘隶属于星际联邦的星舰企业号上的航员以及他们“勇敢地航向前人所未至的领域”的冒险故事。

10.《代号“耶里哥” 》，1966年上映，1994年上海电视台引进播出，共16集。

主演: 唐·弗兰克斯/ 约翰·莱顿

讲述的是二战时期代号“耶里哥”的三人小组渗透到纳粹占领的敌人后方从事破坏和情报窃取活动。

11.《草原小屋》，1974年上映，1988年上海电视台引进播出，共24集。

主演: 迈克尔·兰登 / 凯伦·格拉斯尔 / 梅丽莎·吉尔伯特 / 林赛·格林布什

描写主人公劳拉在威斯康星的小木屋时，到她在达科他地区结婚所渡过的美好时光，作者以充满诗意的笔调描述了他们纯朴的生活，善良的人性，平凡而动人，吸引了成千上万的读者。

12.《破茧飞龙》，1974年上映，上海电视台引进播出，共22集。

主演：詹姆斯·加纳/小诺亚·比里/乔·桑托斯

关于一个名叫吉姆·罗克福德的前犯罪嫌疑人转为私人侦探的故事。他宁愿逃跑也不愿打架，宁愿钓鱼也不愿工作。 在他的案件中，洛克福德总是得到朋友丹尼斯·贝克尔，安吉尔和贝丝·达文波特的协助。

13.《神力女超人》，1975年上映，上海电视台引进播出，共14集。

神奇女侠电视版，在剧中戴着一副特大的眼镜在国防部任职，亳不显眼，但当国家遇上危急时，她即变成性感的神奇女侠拯救人民。总是穿着一件星旗低胸装在街上奔跑，蔚为奇观。

14.《方舟二号》，1976年上映，共15集。

主演: 特里·李斯特 / 亚当 /乔斯·弗洛雷斯

故事描述的是未来的地球由于污染等原因，人类的文明已经退化，一群科学家建造了一辆高科技的旅行车，环游世界，试图恢复文明。

15.《查理的天使》，1976年上映，1990年上海电视台引进播出，共22集。

主演: 大卫·道耶 / 杰奎琳·史密斯 / 约翰·福赛思

《霹雳娇娃》翻拍自该电视剧，该剧讲述的是三个女人为一个虚构的私人调查中心工作，她们的老板名叫查理，是一个从来不在影片中现身的角色，仅用电话与她们联络。

16.《根》，1977年上映，上海电视台引进播出，共8集。

主演: 玛雅·安吉洛 / 吉-杜·根伯卡 / 摩西·冈恩

改编自黑人作家亚历克斯·哈利所写的同名小说，跨度几代人讲述了他们从被贩卖到寻精神之根的长达半个多世纪的黑人种族血泪史。

17.《绿巨人》，1978年上映，共12集。

主演: 卢·费里诺 / 比尔·贝克比 / 杰克·科尔文

布鲁斯·班纳的父亲在刚出生的班纳身上进行基因改造试验。多年后,班纳在一次试验意外中受到了伽马射线的强力辐射，布鲁斯体内的神秘力量被意外唤醒。从此之后，每当情绪激动时，布鲁斯就会失去自我意识，变身成为绿巨人，并且具有超强的破坏力。

18.《帕尔斯警长》，1979年上映，1986年在央视播出，共13集。

主演: 詹姆斯·厄尔·琼斯 / 李·张伯林 / 汉克·盖瑞特 / 迈克尔·沃伦 / 乔格·斯坦福·布朗

伍迪·帕尔斯在加利福尼亚南部某地经营着一个大城市警察局。他指挥一个由四名特立独行的年轻侦探组成的小队，与妻子巴巴拉过着幸福的家庭生活。

19.《格洛丽亚》，1980年上映，1986年上海电视台引进播出，共4集。

主演: 吉娜·罗兰兹 / 巴克·亨利 / 朱莉·卡门 / 约翰·亚当斯 / 约翰·芬尼根

当一个小男孩的家庭被暴徒杀死时，他们的邻居格洛里亚很不情愿的成为了他的监护人。 两人拥有黑帮想要的一本书， 随后暴徒在整个城市追逐他们。

20.《霓裳恋曲》，1980年上映，上海电视台引进播出，共3集。

主演: 林赛·瓦格纳 / 巴瑞·波斯威克 / 玛丽-弗朗丝·皮西尔 / 金·凯特罗尔

故事讲述了亿万富翁遗孀比利、法国准时装设计师瓦伦丁、停车场服务员/摄影师斯拜德，三个雄心勃勃的年轻人的命运。

21.《鹰冠庄园》，1981年上映，1988年上海电视台引进播出，共18集。

主演: 洛伦佐·拉马斯 / 冀朝理 / 简·怀曼 / 大卫·塞尔比 / 苏珊·沙利文

以加利福尼亚的葡萄园为背景，展现了强大的Gioberti家族之间的冲突，该家族是庞大的Falcon Crest酒庄的所有者。

22.《豪门恩怨》，1981年上映，1991年上海电视台引进播出，共13集。

主演: 约翰·福赛思 / 琳达·伊万斯 / 琼·柯琳斯 / 迈克尔·纳德/ 杰克·科尔曼

一个富裕的丹佛家族在石油行业的传奇。大部分剧情都以卡林顿家族和科尔比家族这两个有争议的家族之间的冲突为特色。

23.《霹雳游侠》，1982年上映，1995年在央视播出，共80集。

主演: 大卫·哈塞尔霍夫 / 理查德·贝斯哈特 / 丽贝卡·霍尔登 / 爱德华·马哈尔

故事讲述了Michael Knight驾驶着具有高度人工智能的跑车KITT，在罪犯横行于法律之上的世界里支持那些无辜及无助的人。

24.《斯蒂尔传奇》，1982年上映，共22集。

主演: 皮尔斯·布鲁斯南 / 斯蒂芬尼·齐姆巴丽斯特 / 詹姆斯·瑞德

斯蒂尔是个潇洒，智慧而且浪漫的侦探。这位神秘的侦探总是和他的助手劳拉在不经意的情况下侦破了一件又一件的案子。而事实上，侦探社的真正操盘手正是劳拉。因为人们总是不不够信任一个女侦探。

25.《曼特·豪斯顿》，1982年上映，1991年上海电视台引进播出，共67集。

主演: 李·霍斯利 / 帕梅拉·亨斯利 / 巴迪·艾布森

曼特·豪斯顿是一位富有的德克萨斯州人，他搬到加利福尼亚来监督他的家人的海上钻井企业，但大部分时间都花在自己的私人调查员爱好上。 这些案件通常涉及谋杀案，利用他多年的经验和调查技巧，在他的助手C.J.的帮助下，休斯顿始终设法将坏人绳之以法。

26.《荆棘鸟》，1983年上映，上海电视台引进播出，共4集。

主演: 理查德·查伯兰 / 蕾切尔·沃德 / 克里斯托弗·普卢默 / 布莱恩·布朗 / 布莱特·卡伦

以早期澳大利亚移民内陆开发为背景。神父拉尔夫与新西兰移民家庭克利里家族交好，家中的小女儿麦琪从少女时期起便爱上了他，神父与麦琪相爱却不能相守。

27.《侠骨柔情》，1983年上映，上海电视台引进播出，共21集。

主演: 凯特·杰克逊 / 布鲁斯·巴克林纳 / 贝弗利·加兰 / 玛莎·史密斯 / 阿尔·罗斯西奥

特工Stetson Lee在一次躲避敌方追捕的途中将证物塞给了正巧路过的家庭主妇Amanda King。在摆脱追踪后，Lee找到了King太太要回证物，哪知King太太玩上了瘾，缠着他要帮助他们破案。迫于无奈，Lee只好答应了她的“无理”要求，两个人开始以奇怪的组合进行合作。

28.《大饭店》，1983年上映，1990年上海电视台引进播出，共23集。

主演:詹姆斯·布洛林 / 康妮·赛勒卡 / 莎莉·贝拉方特

是一部关于加州旧金山著名的圣格雷戈里酒店管理层、员工和客人生活的电视剧。

29.《天龙特攻队》，1983年上映，共13集。

主演: 乔治·佩帕德 / 德怀特·舒尔茨 / 劳伦斯·特劳德 / 德克·本内迪克特 / 梅琳达·库莱亚

在越南服役结束后，约翰·汉尼拔·史密斯上校和他的团队因抢劫河内银行（他们是在命令下抢劫的）而被陷害，并被送进一所美军监狱。他们越狱之后成为雇佣的英雄，在美国或世界上做好人警卫。

30.《天使的愤怒》，1983年上映，共3集。

主演: 杰克琳·史密斯 /肯·霍华德/阿曼·阿桑特

一位无情的律师（莫雷尔）使用一位年轻的助理地方检察官（帕克）来解救他的客户，因此被解雇，几乎被取消律师资格，但她反抗成为顶级律师。

31.《飞狼》，1984年上映，上海电视台引进播出，共11集。

主演: 简·迈克尔·文森特 /艾力克斯·寇德 /欧内斯特·博格宁

斯特林费罗·霍克与多米尼克·桑蒂尼将乘坐先进的战斗直升机“飞狼”执行任务。飞狼直升机全身黑色的，性能极其凶悍，可以说是刀枪不入。

32.《神探亨特》，1984年上映，1992年上海电视台引进播出，共153集。

主演: 弗里德·杜莱尔 / 斯蒂芬妮·克拉默 / 查尔斯·哈拉汉

影片围绕足智多谋的洛杉矶警探瑞克·亨特和他的默契女搭档迪迪麦考尔展开，讲述着洛杉矶警探们捍卫城市安全的故事。

33.《钻石王朝》，1984年上映，上海电视台引进播出，共4集。

主演: 安格哈拉德·瑞丝 / 伊安·查里森 / 切瑞·朗吉

描述了一位淘金者的女儿的传奇故事，幼年的经历即让她懂得权力的重要。她有足够的精神，纷乱的战争年代成为她发迹的最佳时机；她有足够的野心，步步为营直至缔造庞大的钻石王国。在权力的棋盘上，每一个人都是她手中的棋子，无论是她的情人还是她的儿子……

34.《埃利斯岛》，1984年上映，1987年上海电视台引进播出，共7集。

主演: 彼得·里格特 /格雷戈里·保罗·马丁 / 克莱尔·布鲁姆 / 琼·格林伍德 / 艾丽丝·克里奇

为了寻求美国梦，19世纪抵达埃利斯岛（埃利斯岛曾是美国主要的移民检查站）的无数移民中的一些人的生活和挣扎。

35.《成长的烦恼》，1985年上映，1990年上海电视台引进播出，共22集。

主演: 艾伦•锡克 / 乔安娜•柯恩斯 / 柯克•卡梅隆/特蕾茜·格尔德/杰瑞米·米勒

主要讲述了一个普通的美国家庭西佛一家的日常生活。

36.《家族的荣誉》，1985年上映，上海电视台引进播出，共13集。

主演: 埃利·瓦拉赫/肯尼斯·麦克米兰/迈克尔·马德森/汤姆·梅森/迈克尔·伍德

从小一起长大的两个朋友，走上了不同的道路，一个成为了警察，一个加入了黑手党。之后两家人分别加入警队与黑帮，形成了敌对的警察世家与黑帮世家。

37.《浮华世家》，1985年上映，上海电视台引进播出，共49集。

主演: 查尔顿·赫斯顿 / 约翰·詹姆斯 / 凯瑟琳·罗斯 / 爱玛·萨姆斯 / 麦克西维尔·考菲德

《豪门恩怨》的衍生作品，以杰夫·科尔比和法伦·科尔比在加利福尼亚的冒险为特色。杰夫·科尔比接受叔叔杰森·科尔比提出的要为家族企业（ColbyCo）工作的提议。

38.《电脑娃娃》，1985年上映，1991年在大陆播出，共24集。

主演: 蒂芙尼·布里塞特 / 迪克·克里斯帝 / 玛拉·彭宁顿 /杰里·苏普兰

硅谷天才电脑工程师泰德•罗森制造了一个高智能机器人，该机器人拥有人类女孩的外表，智商超高，力大无穷。泰德将这个命名为维姬的机器人作为礼物送给妻子，对外则宣称维姬是儿子乔恩的妹妹。

39.《蓝色月光侦探社》，1985年上映，共66集。

主演: 布鲁斯·威利斯 / 斯碧尔·谢波德

顶级模特麦迪·海耶斯被她的投资顾问出卖了，只剩下房子和破败的侦探社，本打算卖掉侦探社，侦探戴维·艾迪生试图说服她以新老板的身份加入代理公司，就这样成了一对侦探搭档。
