《精灵宝可梦 究极之日/月》新晋训练师指南

2020-10-29
| 游戏

《精灵宝可梦 究极之日/月》攻略索引：

御三家

游戏中玩家最早能够获得三只宝可梦被俗称为御三家，这三只宝可梦必定是草、水、火三种属性，而玩家只能在其中选择一只。按照以往惯例，这三种宝可梦是不可能在野外捕捉到的，想要再获得只能使用通信传送的方式让其他玩家送给你。

《精灵宝可梦 太阳/月亮》中的御三家分别是：

草系，木木枭→投羽枭→狙射树枭

初始技能为“树叶”，属性为草/飞行，特性为茂盛；最终进化属性为草/幽灵，特别技能为“缝影”，受到这招攻击的对手会无法逃走或进行替换。

种族值：HP：78 / 攻击：107 / 防御：75 / 特攻：100 / 特防：100 / 速度：70

  • 特性：茂盛，在体力不支时草系招式攻击增强。
  • 隐藏特性：使用近身招式不会受到自身或目标特性影响。

火系，火斑喵→炎热喵→炽焰咆哮虎

初始技能为“火花”，属性为火，特性为猛火。最终进化属性为火/恶，特别技能为“ＤＤ金勾臂”，可在不受到对手能力变化影响的情况下给予对手伤害。

种族值：HP：95 / 攻击：115 / 防御：90 / 特攻：80 / 特防：90 / 速度：60

  • 特性：猛火，在体力不支时火系招式攻击增强。
  • 隐藏特性：出场时所有攻击范围内的对手攻击下降一级。

水系，球球海狮→花漾海狮→西狮海壬

初始技能为“水枪”，属性为水，特性为激流。最终进化属性为水/妖精，特别技能为“泡影的咏叹调”，受到这招攻击的对手身上的「灼伤」状态将会痊愈。

种族值：HP：80 / 攻击：74 / 防御：74 / 特攻：126 / 特防：116 / 速度：60

  • 特性：激流，在体力不支时水系招式攻击增强，
  • 隐藏特性：所有以声音为基础的招式全部转换为水属性攻击。

捕捉宝可梦

在选好了自己的御三家开始冒险之后，玩家就需要开始捕捉野生的宝可梦来壮大的自己的队伍，完成自己的图鉴。捕捉宝可梦将会需要使用精灵球，而精灵球只能对野生的宝可梦使用。也就是说，你无法捕捉已经有主人的宝可梦。

捕捉宝可梦讲究的就是三个字：快、狠、准……不对，是血量低、有异常、用好球。

所谓血量低，就是尽可能的把野生的宝可梦血量控制在一个较低的水平；有异常，就是为野生的宝可梦施加各种异常状态，比如麻痹，睡眠，混乱等等；用好球，是因为精灵球有很多种，通常越好的精灵球捕捉的概率越高，当然也有一些比较特别的精灵球会针对某些特别种类的宝可梦有加成。

以上几种方法只是能够提高捕捉到宝可梦的概率，并非绝对能够捕获。除非使用独一无二的大师球，否则宝可梦都有一定概率从精灵球中挣脱出来。

最后，在精灵球晃动的过程中，你可以：连按A/B键，狂摇主机，猛点屏幕，跪地祈祷等等，但以上行为均不会有任何加成，恩。

宝可梦进化

进化，是《精灵宝可梦》的一个核心系统。大多数宝可梦在进化之后会改变样子，变强，甚至还会改变属性和特性。而游戏中宝可梦的进化方式也是异常的多样：

等级进化

最常见的进化方式，只要宝可梦达到一定的等级之后就可以进化。不同的宝可梦进化等级各不相同，要求最低的绿毛虫只需要7级就可以进化；而双首暴龙则需要在64级才会进化成为三首恶龙。

道具进化

道具进化最常见的就是使用进化石。比如皮卡丘在正常情况下是不会进化的，只有对它使用雷之石才会进化成为雷丘，而伊布则需要使用火之石、雷之石、水之石才会分别进化成为火伊布、雷伊布、水伊布。

通信进化

部分宝可梦只会在两个玩家之间交换宝可梦时候进化，比如鬼斯通→耿鬼、勇吉拉→胡地、豪力→怪力。值得一提的是，有些需要通信进化的宝可梦还需要携带特定的道具，比如呆呆兽→呆呆王，就需要在通信的时候让呆呆兽携带王者之证才会进化。

亲密度进化

这也是比较常见进化方式。这部分宝可梦只有在和玩家的亲密度达到一定程度，然后在升级的时候才会进化。比如皮丘→皮卡丘。

其他进化

除了上面介绍的几种之外，还有很多其他进化方式，但这一类宝可梦的数量非常少。比如伊布→太阳/月亮，是根据进化时间决定的；笨笨鱼→美纳斯，是根据美丽度决定的；黏美儿→黏美龙，要在下雨时候才会进化。最奇怪的大概还要属好啦鱿→乌贼王，需要在升级时把 3DS 主机倒过来才会进化……

属性相克

在游戏中，不同的宝可梦，不同的招式都会有不同的属性，而属性之间的相性则会影响每次攻击的伤害结算。下面的图片列出了不同属性之间的相互影响。需要注意的是，表格中只包含了一组属性的影响，而非多重属性。具有多重属性的宝可梦则需要选择攻击方的招式的属性和防御方的两个属性对应的倍数相乘。

比如，飞行属性的招式会对虫、草属性神奇宝贝造成2*2=4倍伤害，而若是地面属性招式则只有0.5*0.5=0.25倍伤害。

不过在本作中，玩家如果对战此前已经相遇过的宝可梦，那么招式的效果是会直接显示在旁边的，其描述分别对应：

没有效果：0x

效果不好：0.5x~1x

有效果：1x~1.5x

效果绝佳：1.5x~4x

所以即便是记不住相性表，也可以通过招式上的描述对效果进行大致判断。

宝可梦的能力值

每只宝可梦都有六个能力值，也称六围，分别是HP、攻击、防御、特攻、特防、速度，下面解释这六个属性各自的作用。

  • HP：即宝可梦的血量
  • 攻击：影响物理攻击招式的伤害
  • 防御：减少受到的物理攻击伤害
  • 特攻：影响特殊攻击招式的伤害
  • 特防：减少受到的特殊攻击伤害
  • 速度：速度高的宝可梦优先出手

招式的具体分类可以在招式详情中查看，每个招式都会用上图中的图标表示类型。注意：极少部分招式将会以进攻方的特攻和防御方的防御力计算伤害。

速度的结算非常精确，哪怕是只高出一点，也会比对方先出手。但是，部分招式优先度将会高于速度的计算，比如电光火石。也就是说，即便是对方速度更高，但使用技能优先度更高的一方将会先出手。此外，部分携带道具也会改变出招顺序。

异常状态

《精灵宝可梦 太阳/月亮》依然只有五个异常状态，分别是中毒、灼伤、冰冻、麻痹、睡眠。中了这些异常状态的宝可梦战斗时会在名字旁边写上异常状态的名称，中毒与猛毒都写作中毒，但颜色深浅不同。异常状态无法叠加，也不会相互抵消或替换，战斗结束后异常状态也不会消失。

虽然名为异常状态，但对宝可梦来说，并非所有的异常状态都是负面的，也有能够从异常状态中获得增益效果的宝可梦。而根据自己的需要搭配，也有主动让自己陷入异常状态的战术。此外，异常状态还会受到宝可梦的特性、天气、环境、招式和道具的影响。

灼伤

灼伤状态下的宝可梦每回合会按最大HP的百分比扣除体力，同时物理攻击招式的威力减半。火属性的宝可梦不会进入灼伤状态。

冰冻

冰冻状态下的宝可梦无法使用任何招式。冰属性的宝可梦不会进入冰冻状态，在被火属性的招式攻击后，冰冻状态会解除。同时，每回合开始的时候都有25%左右的概率自动解除冰冻状态。

麻痹

麻痹状态下的宝可梦每回合有25%的概率无法使用招式，同时速度下降50%。电属性的宝可梦不会进入麻痹状态。

中毒

中毒状态下的宝可梦每回合会按最大HP的1/8扣除体力，猛毒第一回合则为1/16的体力，随后每回合加倍。原本钢系和毒系宝可梦是不会中毒的，但由于“腐蚀”特性的引入，理论上所有宝可梦都会进入中毒状态。

睡眠

睡眠状态下的宝可梦无法使用除了“打鼾”和“梦话”之外的任何招式，睡眠状态在2~4回合内随机解除。除了被对手的招式催眠之外，宝可梦也有让自己主动睡眠的招式。自己主动睡眠的宝可梦将会恢复全部HP，清除当前所有异常状态，然后必定会在第三回合醒过来。

能力升降

在对战中，大家经常会看到XXXXX攻击力上升了！XXXX特防大幅上升了等等，这就是战斗中的能力升降变化。

战斗中所有能力变化都分为十二档，上升和下降各六档，普通状态下所有宝可梦都默认处于普通状态，也就是0，此时所有能力都为100%，不同档位具体数据如下：

需要注意的是，当攻击出现“击中要害”的情况时，伤害结算会无视所有的不利条件，即攻击方的攻击和特攻如果小于0档，则按照0档计算，同时防守一方的防御和特防如果大于0档，则按照0档计算。

所有的能力升降在宝可梦回到玩家手中，以及战斗结束之后全部清零。战斗中能力升降可以点击下屏中宝可梦的身边的+号查看。

状态变化

状态变化是区别于上面提到的异常状态的。首先，状态变化可以叠加；其次，大部分都可以自愈；最后，战斗结束时状态变化自动消失。状态变化单场战斗内有效的增益或负面效果，同时还有一些状态变化可以对场地施加，他们会改变一些对战时候的规则。比如，对战场地中存在戏法空间时，速度慢的一方反而会先出手，处于灭亡之歌状态的宝可梦，三回合之后会直接进入濒死状态。

宝可梦的性格

宝可梦的性格将会影响自己的能力值和口味，其中一项的能力值增加10%，另一项则减少10%。比如“勇敢”的性格，会提升10%的攻击，降低10%的速度，喜欢辣、讨厌涩。另外还有5种性格不会影响神奇宝贝的能力值。除了这5种之外，其它20种性格所增加和减少的能力值组合均不相同。

宝可梦的特性

一只宝可梦可能有一到两种特性，通常在捕捉（出生）时就决定了，部分宝可梦在进化后特性会改变，玩家可以在宝可梦的详情中查看每只宝可梦的特性。宝可梦大部分特性都是有益的，少部分特性会有负面效果。同时，部分宝可梦将会有不显示出来的隐藏特性。

这只皮卡丘的特性是静电

Z招式

Z 招式是《精灵宝可梦 太阳/月亮》新加入的系统，属于特别强力的招式、在一次对战中只能使用一次。

要使用 Z 招式，宝可梦需要携带对应属性的 Z 水晶。在道具栏中携带 Z 水晶的时候，系统会告知玩家该水晶对宝可梦是否有效，所以不同担心带错。值得一提的是，部分宝可梦拥有自己的特别Z水晶，比如御三家宝可梦，皮卡丘和卡比兽等等。此外，以不同的招式为基础使用 Z 招式的时候，效果与威力也不相同。

攻击招式→攻击 Z 招式，属于将攻击大幅度强化的Ｚ招式，即便使用“守住”或“看穿”等技能也无法免伤，部分攻击招式甚至还会追加新的效果。

变化招式→变化 Z 招式，变化招式会借由Ｚ力量得到增强，已有的效果将会得到强化，同时也有可能会追加新的效果。

游戏中大部分 Z 水晶都是从各个岛屿的队长和岛主哪里获得的，少部分 Z 水晶会隐藏在地图中。
