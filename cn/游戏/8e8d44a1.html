钱美华(1927—2010 )



宁波宁海人。中国工艺美术大师、北京特级工艺美术大师、高级工艺美术师。北京市珐琅厂第一任总工艺师、科技研究中心主持人。第一批国家级非物质文化遗产景泰蓝制作技艺传承人。师从梁思成、林徽因主研工艺美术，是新中国知识分子从事景泰蓝专业设计的第一人，参与新中国第一份国礼设计。

屠呦呦（1930—）

宁波海曙人，药学家，中国中医研究院终身研究员兼首席研究员，青蒿素研究开发中心主任。屠呦呦是第一位获得诺贝尔科学奖项的中国本土科学家、第一位获得诺贝尔生理医学奖的华人科学家。2017年1月2日，被授予2016年度国家最高科学技术奖。这是国家最高科学技术奖首次授予女性科学家。

金雅梅(1864—1934)

宁波鄞县梅墟宝桥金家村(今属邱隘镇)人。中国第一位女留学生。光绪十四年回国，先后供职厦门、广州、成都、北京等地医院，医术精深，求诊治者甚众。光绪三十四年，得袁世凯资助，创办中国第一所公立护士学校——北洋女医学堂，后又任天津妇女医院院长。

沈贻芗（1900—1989）

宁波奉化溪口亭下村人，幼名遗香。她是中国第一所女子学校——甬江女子中学第一任国人校长，担任时间长达二十多年。曾任奉化县第一届政协委员、民革奉化县委员会委员、宁波“甬港联谊会”副会长。


严彩韵（1902—1993）

宁波慈溪人，生物化学家，是中国最早从事生物化学研究的女学者。曾在联合国儿童基金会、哥伦比亚大学人类营养研究所、纽约圣路加医院等地任职，修订出版《营养概论》。其丈夫吴宪则是中国生物化学领域的先驱，创立了中国的生物化学学科。其中长子吴瑞为著名分子生物学家。


陈亚珠(1936—)



宁波人，中国工程院院士，高电压技术专家、生物医学工程专家。现任上海交通大学教授、博土生导师，生物医学仪器研究所所长。是我国无创伤性医疗技术领域开拓者之一。

陈赛娟（1951— ）

宁波鄞县人。细胞遗传学和分子遗传学专家，中国工程院院士、发展中国家科学院院士。曾任上海交通大学国家重点实验室主任、上海交通大学医学院血研所执行所长、上海市科协主席、全国科协副主席。陈赛娟长期致力于白血病发病机理与治疗研究，建立完整的细胞和分子遗传学技术，实现了当时我国生物医学领域中人类疾病新基因克隆零的突破。

黄量（1920—2013）

祖籍宁波，药物化学家，生于上海。中科院院士。美国康奈尔大学第一位外籍化学女博士。中国第一个以萝芙木总碱为成分的降压药的参研者之一。黄量院士主要从事天然产物及抗肿瘤、计划生育、抗病毒等新药研究。20世纪50年代参与研制成功降压药萝芙木总碱及其利血平等有效成分，抗肿瘤新药N-甲酰溶肉瘤素及剑麻皂甙元的利用，治疗白血病的三尖杉脂碱及其同系物的合成等。设计并指导研制的还有新药维胺酯和甲异靛、抗病毒新药肽丁安等。在国内较早开展手性药物的研究。


张芷芬（1927—）

出生于宁波慈溪龙山。数学家。从事常微分方程定性理论和拓扑动力系统理论的研究，是国内这一领域的开拓者之一。历任北京大学副教授、教授、数学系副主任，北京市数学学会副理事长。专于微分方程的定性理论和动力系统研究。对李奈方程极限环的存在性和唯n性问题有较深造诣。合著《微分方程定性理论》。

陈修良（1907—1998 ）

浙江宁波人，1927年加入中国共产党；毕业于苏联莫斯科中国劳动者共产主义大学；曾任江苏省委妇委书记，《新华报》总编辑，陈修良是我党历史上的第一位大城市的女市委书记；建国后，历任中共上海市委组织部副部长、全国妇联华东区工作委员会副主任、全国妇联第二届执委、浙江省委宣传部代部长、上海社会科学院顾问。陈修良同志的丈夫是新中国首任浙江省省长沙文汉同志。

张鸿眉(1927—2005)

宁波镇海人。1948年，出演其处女作《小城之春》，饰妹妹戴秀。建国后入上海电影制片厂，出演影片《宋景诗》等。1963年起，任导演工作，代表作有《飞向太平洋》，《珊瑚岛上的死光》。中国第一部科幻故事片《珊瑚岛上的死光》导演。

王丹凤（1924—2018）

原名王玉凤，祖籍浙江宁波，出生于上海，著名电影女演员。她在《新渔光曲》一片中主演渔家女一角轰动影坛，被影界称为“小周璇”。出演过《护士日记》、《女理发师》、《桃花扇》、《玉色蝴蝶》等影片。1962年被选为新中国首次推出的“二十二大影星”之一。从影40年，她在银幕上塑造了54个艺术形象，成为高产明星。

周广仁（1928— ）

祖籍宁波，出生于德国汉诺威。著名钢琴家，中央音乐学院终身教授，当今中国在国际乐坛中最具影响和权威的钢琴演奏家、教育家。是第一位在国际钢琴比赛中获奖的中国钢琴家。1955年后在中央音乐学院任教并担任钢琴系副主任、主任等领导工作。曾多次在欧美各国的国际钢琴比赛中担任评委，创办了星海、乐友两所儿童钢琴学校，创办中国国际钢琴比赛和钢琴考级活动，被誉为“中国钢琴教育的灵魂”。

俞丽拿（1940— ）

宁波鄞州姜山镇人，生于上海，中国著名小提琴演奏家。上海市音乐家协会副主席、上海音乐学院教授、上海宁波经促会副会长、俞丽拿小提琴艺术基金董事长。她是小提琴协奏曲《梁山伯与祝英台》最权威的演绎者之一。中国第一次中国音乐家在国际弦乐大赛中获奖；中国第一位中国艺术家以自己名字命名艺术基金“俞丽拿小提琴艺术基金”。

胡晓平(1950— )

宁波北仑柴桥人，生于上海，国际女高音歌唱家。是我国改革开放以来第一位获国际声乐大赛第一名的歌唱家。俄罗斯天文台曾以胡晓平的名字命名一颗小行星，载入史册。2006年底，胡晓平毅然回到故土，参与宁波大学音乐专业的初创，成为第一个实质性助力宁波的著名甬籍音乐家。

张彩珍（1930—）

宁波镇海人，原国家体委副主任、中国奥委会副主席。张彩珍是共和国有史以来第一位在国家体委担任副主任的女性，而且还成为中国历史上第一位获得奥林匹克勋章的女性。

叶佩素(1958— )



宁波北仑人，有“亚洲七项全能女王”之称，现为省体育局训练竞赛处处长。在1978年至1987年间，共获全运会女子跳高冠军2次，田径女子五项全能(破全国纪录)冠军1次，亚运会田径女子七项全能冠军2次。

沈盛妃(1980— )

宁波北仑人，1993年进入宁波市体校进行训练，1997年 被选入浙江省体训队，曾连续夺得1998年曼谷亚运会和2002年釜山亚运会女子七项全能冠军，并多次获得国内各类比赛冠军。中国第一个进入奥运殿堂的女子七项全能选手。

岑伟飞(1967— )

宁波慈溪人，原中国国家地掷球队运动员，现担任镇海炼化培训中心教师。2006年10月21日，岑伟飞在第五届世界女子地掷球锦标赛中一举夺得单打冠军，实现了中国地掷球运动历史金牌的零突破。

杨波(1973— )

宁波人，原中国体操运动员、中国女子体操队队长。世界杯比赛中获平衡木冠军，荣获国际级运动健将称号，并获国家体委授予的体育运动荣誉奖章。1992年，国际体操联合会将她的一个高难度平衡木动作命名为“杨波跳”。

贺慈红(1975— )

宁波北仑人，著名游泳运动员，现为广州军区体工队游泳教练。在1991年至1998年间，共获得全国冠军7次，亚洲冠军6次，世界冠军10次，并7次打破亚洲纪录，4次打破世界纪录。
