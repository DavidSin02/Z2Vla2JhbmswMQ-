《宝可梦剑盾》双打速度压制队伍分享

2020-10-29
| 游戏

　《宝可梦剑盾》中对自己宝可梦能力有自信的玩家们可以选择参加排位赛，组成多种队伍来战胜其他玩家，下面请看由玩家“greed_zzz”分享的《宝可梦剑盾》双打速度压制队伍分享，希望对大家有用。

　　风妖精 @ 气腰

　　特性：恶作剧之心

　　等级：50

　　努力值：252 特攻 / 252 速度 / 6HP

　　胆小(+速度 -物攻) 性格

　　- 顺风

　　- 戏法空间

　　- 挑衅 / 能量球

　　- 月亮之力

　　8代由于出手权的更变判定，感觉风妖精比以前更强了，所以打算建一个依靠速度去强行压制的队伍，那么风妖精就是第一个选进队的。顺风保证出手权，空间在遇到空间队的时候可以逼迫对面猜拳，挑衅是我上9级以后带的技能，一开始带的是能量球为了可以压低没极巨化的海兔兽血线，但遇到蛮多海兔兽都是1KO的。不过打上去以后感觉带多个挑衅稳点就把能量球换成了挑衅，在10级上大师的时候也的确用得上挑衅4战全胜上了大师。

　　铝钢龙 @ 突击背心

　　特性：坚毅

　　等级：50

　　努力值：252 特攻 / 220 速度 / 38特防

　　胆小(+速度 -物攻) 性格

　　- 日光束

　　- 十万伏特

　　- 加农光炮

　　- 流星群

　　一开始看上了这个新特性能无视看我嘛，后面实战的时候发现好像对面用交换场地还能追着点的那只精灵打，而且钢龙的属性不怕仙，弱格斗和地面，铝钢龙自身物耐也高，能吃下大部分精灵无强化情况下的物攻格斗， 极巨后的极巨龙打击面广能降对面一级物攻，极巨钢加自己一级物防还能更进一步增加自己的坦度，遇到了看我嘛加布里姆温的组合可以首发风妖精加铝钢龙去强杀对面布里姆温。日光束也是为了让后面的鳃鱼龙有舒服的输出环境留着清海兔兽用的。

　　颤弦蝾螈 @ 气球

　　特性：庞克摇滚

　　等级：50

　　努力值：252 特攻 / 252 速度 / 6HP

　　内敛(+特攻 -物攻) 性格

　　- 破音

　　- 爆音波

　　- 污泥炸弹

　　- 守住

　　之前看银风聚聚的银风沙发现现在环境里比较强的飞行系有勇士鹰和钢铠鸦，加上4倍弱雷的暴鲤龙也能给队友提速，于是选了这只精灵，特性加破音可以保证雷属性输出有一个可观的表现，极巨化的情况下极巨毒提升特攻也和队伍相性比较合适，4倍弱地所以带气球来保证起码能扛一轮输出不至于直接被一发地震带走。

　　杖尾鳞甲龙 @ 生命宝珠

　　特性：隔音

　　等级：50

　　努力值：252 特攻 / 130 特防 / 116HP / 12速度

　　内敛(+特攻 -物攻) 性格

　　- 波导弹

　　- 鳞片噪音

　　- 加农光炮

　　- 守住

　　在7代刚出来的那个环境下卡噗爹遍地走，四倍弱仙的杖尾鳞甲龙在当时环境下感觉很难出场，但是到了8代卡噗家族留在了阿罗拉，环境里常见的仙系从一开始的仙布到现在布里姆温长毛巨魔还有风妖精，带隔音就是为抗仙布的巨声和与颤弦蝾螈同场的时候颤弦蝾螈无脑爆音波压血线自己不会掉血，而且杖尾鳞甲龙物攻特攻都不错，走特攻不怕鬼火，实战中也遇到由于杖尾鳞甲龙极巨化后极巨技能看不出是物攻还是特攻导致对面对杖尾鳞甲龙用鬼火，一般情况下都是由铝钢龙清完了对面的仙系精灵后看情况上场。

　　鳃鱼龙 @ 围巾

　　特性：强壮之腭

　　等级：50

　　努力值：252 物攻 / 252 速度 / 6HP

　　爽朗(+速度 -特攻) 性格

　　- 鳃咬

　　- 咬碎

　　- 精神之牙

　　- 龙之俯冲

　　刚开始的时候想带的是暴鲤龙，但仔细想了想这个队伍和鳃鱼龙的相性太合适了，不带有点可惜就带上了。无脑鳃咬大家都懂这里就不多说了。

　　洛托姆 (割草机) @ 巧可果

　　特性：漂浮

　　等级：50

　　努力值：252 特攻 / 252 速度 / 6HP

　　内敛(+特攻 -物攻) 性格

　　- 欺诈

　　- 伏特替换 / 十万伏特

　　- 飞叶风暴

　　- 守住

　　洛托姆是最后进队的一只精灵，在它之前的是男爱管侍，想拿爱管侍当一个低配蝶蝶带命玉打输出，特性也能挡击掌奇袭。结果实战里发现站不住，上场打一发精神强念然后就被人送到下面去休息了，而且选出也比较少。后面还遇到了一队腹鼓卡比，发现这个队伍很难解决卡比这个问题，所以选了洛托姆，欺诈腹鼓卡比伤害爆炸不说，飞叶风暴也能帮忙清海兔兽，总体来说还是和队伍风格比较搭的。
