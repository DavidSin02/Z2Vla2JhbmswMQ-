怀旧服增强萨PVE攻略：风怒之力会毁灭你

2020-10-29
| 游戏

一、开始之前

萨满除去不差的输出能力外，还具有优秀的辅助能力，凭借风怒图腾、战栗图腾、清毒图腾等强大的图腾能力，既能保护队友的安全，又能显著提升队友的输出能力。

1、 增强萨满的优劣

Ø 优势

n 风怒武器、风暴打击对增强萨有很好的加成效果

n 最适合施放图腾辅助队友的天赋

n 有着打断和驱散技能

Ø 劣势

n 技能冷却时间较长，持续伤害能力较弱

n 缺蓝

2、 增强萨满种族推荐

Ø 巨魔（PVE推荐）

n 野兽杀手：对野兽的伤害提高5%。

n 狂暴：根据当前生命值提高攻速和施法速度10%-30%。

n 再生：生命值回复速度提高10%。

Ø 兽人（PVP推荐）

n 斧专精：斧和双手斧技能提高5。

n 血性狂暴：近战攻强提高25%，受到的治疗效果降低50%。

n 坚韧：抵抗昏迷效果的机率提高25%。

Ø 牛头人

n 栽培：草药学技能提高15。

n 耐久：生命值上限提高5%。

n 自然抗性：自然抗性提高10点。

n 战争践踏：使半径8码内的敌人昏迷2秒。

3、 增强萨满专业推荐

Ø 元素萨最好的专业就是工程学，通常制造的爆炸物等一些物品在副本内有很大帮助，建议搭配采矿专业，节省购买矿物材料的钱。

Ø 或者学习炼金和草药学，药物可以自给自足，并且还可以赚钱。

二、技能说明

图腾是萨满特有的一种能力，共有四种类型：空气、大地、火焰和水，但每个类型的图腾每次只能存在一个。

Ø 空气图腾

n 风怒图腾：为半径20码内的队友提供风怒效果，主手武器攻击时有20%机率额外进行攻击一次。怀旧服萨满招牌技能，适合所有近战职业，尤其是额外攻击为战士带来的怒气值。

n 风之优雅图腾：使半径20码内的队友增加敏捷。相比战士和盗贼，风之优雅图腾更适合猎人和德鲁伊，因为他们不能受益于风怒图腾。

n 自然抗性图腾：为半径20码内的队友提供自然抗性。

n 根基图腾：每10秒为附近的队友吸收一个有害法术，AOE法术除外。

n 岗哨图腾：监控一定范围内的视野，有点类似于鹰眼术。

n 风墙图腾：减少20码内队友受到的远程伤害。

Ø 大地图腾

n 大地之力图腾：为半径20码内的队友增加力量。适合近战队友。

n 石肤图腾：为半径20码内的队友提供物理伤害减免。在应对BOSS时用处不大，但是在坦克扛着一群小怪时效果显著。

n 地缚图腾：使半径10码内的敌人移动速度降低。风筝怪物时用。

n 战栗图腾：驱散半径30码内队友的恐惧、魅惑及催眠效果，在应对某些boss或PVP时很有用。

n 石爪图腾：吸引8码内的敌人攻击此图腾，单人练级的时候有用，组队时慎用，图腾血量不高，很容易ot。

Ø 水图腾

n 治疗之泉图腾：每隔2秒治疗半径20码内的所有队员。

n 法力之泉图腾：每隔2秒为半径20码内的所有队友恢复法力值。

n 抗火图腾：为半径20码内的队友提供火焰抗性。在熔火之心和黑翼之巢中很有用。

n 祛病图腾：每隔5秒为半径20码内的队友驱散一个疾病效果。

n 清毒图腾：每隔5秒为半径20码内的队友驱散一个中毒效果。

Ø 火焰图腾

n 灼热图腾：频繁攻击20码内的某一个敌人。在长时间战斗中伤害占比较高。

n 火焰新星图腾：图腾持续时间5秒，到时后会对半径10码内的敌人造成火焰伤害。伤害不错，可惜冷却时间有点长。

n 熔岩图腾：每隔2秒对8码内的敌人造成火焰伤害。在火焰新星图腾冷却时可以作为替补使用。

n 抗寒图腾：为半径20码内的队友提供冰霜抗性。在纳克萨玛斯和PVP中应对冰法时有用。

n 火舌图腾：为半径20码内的队友主手武器附加火焰伤害。相比风怒图腾，该图腾作用差远了。

Ø 治疗法术

n 治疗链：为友方目标恢复生命值，然后转而治疗附近的下一个目标，治疗效果每次减少50%，共可治疗3个目标。

n 次级治疗波：快速地为一个目标恢复少量生命值。

n 治疗波：慢速地为一个目标恢复大量生命值。

Ø 伤害法术

n 风暴打击:获得一次额外的攻击机会，使你的下2次攻击对敌人造成的自然伤害提高20%，持续12秒。

n 闪电箭：对单一目标造成伤害。

n 闪电链：对一定范围内的敌人造成伤害，即使是单一目标伤害也比闪电箭要高，但是蓝耗也相对较高。

n 地震术：对敌人造成伤害的同时打断其施法。在PVE和PVP中都很有用，如果你只需要打断效果的话，请勿将该技能升级，避免过高的蓝耗。

n 烈焰震击：对目标造成一个火焰伤害的dot。

n 冰霜震击：对目标造成伤害并降低其移动速度，风筝时很有用。

Ø 功能法术

n 闪电之盾：受到近战或远程伤害时，会对攻击者造成伤害。

n 净化术：移除地方目标的两个增益效果。

n 消毒术：为目标移除一个中毒效果。在清毒图腾无法使用时是一个很好的技能。

n 祛病术：为目标移除一个疾病效果。在祛病图腾无法使用时是一个很好的技能。

n 幽魂之狼：移动速度提高40%，只能在户外使用。无论是快速旅行还是风筝时都是一个非常好的技能。

n 视界术：将你的视野移动至目标区域。在PVP时作为侦查用。

n 水上行走：使友方目标能在水上进行行走。对于在水域多的地图练级时很有用。

n 星界传送：相当于是一个炉石，但是比炉石少了一半的冷却时间。

n 复生：死后能自行复活并恢复少量生命值和法力值。在某些副本中因失误死亡时很有用，避免了多余的赶路时间。

n 先祖之魂：非战斗状态时，可复活一名队友。

三、天赋推荐

风暴打击0/31/20



这套天赋能让你尽可能地持续输出，并且降低了频繁使用图腾所消耗的法力值，非常适合与近战及坦克组队。需要注意的是，由于风暴打击会占用一个DEBUFF的位置，所以在团本中建议替换成“自然迅捷”。

四、技能循环

单目标

Ø 保持者各种增益图腾，在目标范围内施放灼热图腾。

Ø 风暴打击是最主要的攻击手段。

Ø 烈焰震击和地震术能用就用。

多目标

Ø 除了火焰新星图腾，增强萨并没有太好的多目标进攻手段。

五、属性优先级

增强萨最主要的是优先提升武器熟练度，其次是需要6%的命中率，当这两个属性都达标后尽可能多地堆爆击/敏捷。

Ø 武器熟练度

n 武器熟练度不足会降低伤害

Ø 智力

Ø 耐力
