《杀戮尖塔》更新2.0版本 新角色“观者”等大量内容上线

2020-10-29
| 游戏

《杀戮尖塔》2.0版本今日上线，正式推出了第四位角色“观者”，并加入了新卡牌、新遗物、新药水和平衡性调整等内容。

以下是本次更新的详情：

大家好，　　2.0版本更新今天终于发布了！本次更新带来了新卡牌、新遗物、新药水、平衡型调整、重做、bug修复、性能改进和第4名角色。　　您仍然可以在Steam中的BETAS菜单中选择“v1.1 - 2019-17-07”来继续游玩过去的版本。

　　第4名角色　　观者前来评估我们的高塔。她将以神圣的“姿态”与手中的预示之杖来对高塔中的魔物们进行审判。　　和其他角色一样，观者拥有自己的牌组和一些独占的特别遗物。　　感谢所有在beta版中测试第四名角色的玩家门，你们的反馈、数据和讨论帮助我们能够迅速地测试各种新点子和调整这名角色！

　　药水研究所

　　我们这次新增了许多药水，因此也创建了一个单独的页面，让你可以一览所有的药水！　　你可以在主菜单的百科大全中找到药水研究所。　　你甚至可以在其中看到所有新药水和现有药水的稀有度。

　　Mod　　本次更新对游戏本体进行了不少改动，也对Mod支持进行了一些改进。与1.1版兼容的mod很可能需要更新后才能在2.0版本中继续运作，请务必更新！

　　社区同人作品　　本次更新带来了不少新图，但同人绘制的图片也不少！

　　平衡性　　一些稀有度变化、数值变更和各种重做！

  • 召唤铃铛重做。
  • 悬浮风筝重做。
  • 小宝箱重做。
  • 商店遗物的基础价格下调200 -> 150。
  • 黑暗之血遗物加强，治疗量10 -> 12。
  • 永恒羽毛遗物稀有度变更，Boss->罕见。
  • 蜥蜴尾巴遗物稀有度变更，Boss->稀有。
  • 餐券遗物稀有度变更，商店->普通。
  • 星系仪遗物稀有度变更，Boss->商店。
  • 工具箱遗物加强，随机获得无色牌变为三选一无色牌。
  • 玩具扑翼飞机遗物稀有度变更，商店->普通。
  • 白兽雕像遗物稀有度变更，Boss->罕见。
  • 袖箭遗物加强，额外伤害3 -> 4。
  • 火焰吐息卡牌重做。
  • 重编程卡牌重做。
  • 断魂斩+加强，伤害20 -> 22。
  • 计划妥当耗能0 -> 1。
  • 在回合结束时获得格挡的卡牌将会在灼伤等受伤牌之前生效。
  • 普通药水的掉落率下降70% -> 65%。
  • 稀有药水的掉落率提升5% -> 10%。
  • 异蛇之油重做。
  • 鲜血药水回复量加强10% -> 20%。
  • 鲜血药水稀有度调整，罕见->普通。
  • 瓶中精灵加强，回复量10% -> 30%。
  • 集中药水稀有度调整，罕见->普通。
  • 罐装幽灵稀有度调整，罕见->稀有。
  • 毒药水现在变为静默猎手独占。
  • 换脸事件不会再在第二阶段之后出现。
  • 涅奥的无色牌选项现在变为三选一。
  • 多层护甲现在最多999层。
  • 自定模式中的特效现在更新了观者相关的内容。
  • 永存者成就现在需要紫水晶与紫水晶+成就解锁。

　　界面与特效

  • 现在支持4:3显示比例。
  • 虱虫敌人的吐网效果增加了音效。
  • 设定中增加了“字体更大”模式，会将所有文字稍微变大。
  • 卡牌预览中现在可以显示一些相关的卡牌，例如无谋冲锋会显示晕眩。
  • 生成升级卡牌的卡牌/能力/药水现在一律使用+来描述。
  • Boss第一勇士现在会对自己的腰带有反应。
  • 增加了“斩杀”关键词，描述杀死非爪牙敌人时生效的效果。
  • 解锁卡牌和遗物时的字体变大。
  • 在输入设定中鼠标滚轮速度变为3倍。
  • 加入察看按钮，在战斗中选项界面出现时可以察看手牌和战况。
  • 现在在一些遗物界面按0 - 9会播放一些音效。
  • 按ESC键退出其他菜单时不会再自动同时退出卡牌奖励界面。
  • 一场战斗或一局游戏中只触发一次的遗物现在会在触发后变灰。
  • 更改了设定画面中“你需要重启游戏”的文本位置。
  • 历史记录等可以滚动的下拉菜单现在可以使用鼠标滚轮和方向键来滚动。
  • 一些药水有了新的图片。
  • 一些药水现在是彩虹色的了，很酷炫。
  • 罕见药水现在有圈状的粒子效果。
  • 稀有药水现在有金闪闪的特效。
  • 一些能力图标变得更加清晰。
  • 一些遗物图标变得更加清晰，并且查看时会显示高清版本。
  • 一些卡牌/遗物的速度被加快和一致化。
  • 一些有特殊前提条件的卡牌（比如飞身踢）现在会有金色发光效果提示。
  • 悬停在地图图例的图标上时，相关信息会出现在固定位置，不再遮挡地图。
  • 部分遗物、卡牌、药水和能力调整了文本，让表达更为明确。

　　Bug修复

  • 自定模式现在即使在离线状态下进行每日挑战也可以解锁。
  • 飞身踢的结算顺序现在按照卡面描述进行。
  • 蛋系遗物现在在拾取时会反映在同一批卡牌奖励的卡牌上。
  • 蛋系遗物现在会反映在潘多拉魔盒的确认界面上。
  • 混沌药水现在可以在战斗外使用。
  • 修复了武装卡牌不更新升级卡牌描述的bug。
  • 修复了打败特定敌人后，格挡对于荆棘类能力无效的bug。
  • 修复了卡牌总览界面点开时会自动滚屏一点的bug。
  • 修复了一些牌无法被打出时仍然会发光的bug。
  • 修复了在触屏模式下的改名界面中需要点两次确认/取消按钮的bug。
  • 修复了发现卡牌在打开其他菜单后会出现跳过的bug。
  • 修复了分离在对小黑和觉醒者打出两次时会获得两次能量的bug。
  • 修复了蛋系遗物无法升级“留给自己的讯息”事件中卡牌的bug。
  • 修复了无尽模式中精英分数显示不正确的bug。
  • 修复了混沌药水在有添水时也能生效的bug。
  • 修复了ESC键在卡牌奖励画面有可能导致地图变白、隐藏奖励界面等的bug。
  • 修复了逃脱计划在不能抽牌能力生效时依然能够获得格挡的bug。
  • 修复了不惧妖邪+心灵堡垒不能在死亡律动前获得格挡的bug。
  • 修复了势不可当的伤害在杀死一名敌人后不会改变目标的bug。
  • 修复了乱战、破灭、精炼混沌、通晓万物有时在目标死亡时不会打出卡牌的bug。
  • 修复了商人素材初始化时内存溢出的bug。
  • 修复了无尽模式的荒疫在初始化时内存溢出的bug。
  • 修复了在进入房间时生效的遗物，在战斗后奖励界面时能够通过读档再次触发的bug。
  • 修复了怀表遗物在与先见之明等效果互动时过早触发的bug。
  • 修复了弹回卡牌无法对奇怪的勺子遗物影响的卡牌起效的bug。
  • 修复了铜钹遗物的伤害在杀死敌人后不会改变目标的bug。
  • 足跟勾的结算顺序现在按照卡面描述进行。
  • 史莱姆和心脏敌人的音效不会再在游戏后台运行时播放。
  • 铜球的冻结效果现在使用随机数种子了。
  • 转化神龛事件的选项现在使用随机数种子了。
  • 设计师事件的选项现在使用随机数种子了。

　　性能优化

  • 修正了卡牌描述被初始化两次的bug，优化启动时间。
  • 删除一些未使用的字体文件，减少内存使用和启动时间。
  • 变灰的遗物现在使用着色器，而非使用灰度图片来达成效果。

  • 一些时间/日期格式现在可以按语言调整。
  • 增加一些逻辑判断，来为中日韩语言提供更好的卡牌文字格式。
  • 2.0版更新目前有如下语种！
