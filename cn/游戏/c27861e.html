捷克公开赛男双晋级、单打仅3人，燃烧吧小宇宙

2020-10-29
| 游戏

北京时间8月24日至25日凌晨，国际乒联世界巡回赛捷克站，结束了正赛阶段首日的比拼。

以10名小将出战的中国乒乓球队，在头两天的单打资格赛中被淘汰5人（曹巍、宋卓衡、彭飞龙、黄颖琦、张钊），正赛首日再被淘汰2人（李艺杰、袁媛），男队只剩下薛飞、房胤池，而女队更是只有顾若辰一名选手了。

在24日的女双淘汰赛，顾若辰 / 张钊战胜了匈牙利队的柏林特 / 奥拉迪亚。25日下午16点开始的女双1/4决赛，顾若辰 / 张钊对阵埃霍尔姆（瑞典）/ 波塔（匈牙利）跨国组合,1比3失利（9-11,11-4,4-11,8-11）。中国女双止步于8强，未能进入半决赛。

日本组合桥本帆乃香 / 佐藤瞳、早田希娜 / 伊藤美诚、跨国组合埃霍尔姆/ 波塔、 奥地利组合MADARASZ Dora / POLCANOVA 晋级女子双打四强。两对日本女子组合将在半决赛展开内战。

在24日的男双淘汰赛，房胤池 / 薛飞战胜了东道主捷克选手康内克尼 / 波兰斯基；25日下午16点40分，男双1/4决赛，房胤池 / 薛飞对阵伊朗队阿拉米安 / 阿拉米杨。这是一场艰苦的比赛，大比分曾经1比2落后；从第四局开始，薛房组合开始掌控比赛节奏，在悬崖边把比分扳回2平；第五局房胤池发球连得两分10比8拿到赛点；五局的比分依次是13-15,11-8,10-12,11-6，11-8，恭喜房胤池 / 薛飞 ，晋级四强。

房胤池 / 薛飞半决赛的对手是德国的德国弗兰斯卡FRANZISKA /丹麦乔纳森Jonathan。另一场半决赛对手是巴西 卡尔德拉诺 / 法国 西蒙高茨，日本上田仁/吉村真晴组合。

男子单打，薛飞苦战7局，险胜乌克兰选手寇磊；在25日22点，他将面对德国名将波尔。实力与经验的差距决定了这将是一场硬仗，薛飞只能放手去拼。

房胤池4比0大胜奥地利选手“中国女婿”费格尔，他的下一个对手是中国台北队林昀儒。

顾若辰作为女乒最后一位选手，她将在25日23点30分迎战日本队伊藤美诚。

比赛到这个地步，国乒三位已经没有退路。燃烧吧小宇宙，用热血和勇气去冲击，在狭路中迸发激情，发挥出最好的水平，创造奇迹！

乒乓网 胡清源
