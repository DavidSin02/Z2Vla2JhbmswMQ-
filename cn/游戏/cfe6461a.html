没收藏这几家男装店算是白网购这么多年了！改造男友穿搭就靠它！

2020-10-29
| 游戏


偶尔也会收藏关注一些男装店铺。以下这16家是我蛮喜欢的，适合年龄在16岁到28岁的男士穿的，大学生必备哦！网上男装的价格比女装低，按照男生的消费，买贵的衣服大部分都是会去实体店买的，但是网上质量性价比普遍是不错的。

COKEIN古柯男装原创店

描述 4.8 服务 4.8 物流 4.8

这家店铺的三项评分在男装店铺算是高的，风格偏日系，简单干净。T恤的价格在40到80之间，毛衣相对贵些，有的照片拍的很文艺很生活。适合气质偏斯文清爽乖乖的男士穿哦。

二十八间 原创商铺

描述 4.7 服务 4.8 物流 4.8

这家店的外套我很喜欢，衣服风格很阳光刚烈张扬。店铺有几款t恤太花俏，大部分我还是蛮中意的，推荐给心上人吧。

DANNY AND ZEE

描述 4.7 服务 4.8 物流 4.7

店主是个帅帅的女生哦，有的衣服有点挑人，反而帅帅的女生穿更好看。款式很潮，撞衫的概率也不大，潮男必备。

熊猫本benpanda自营店

描述 4.7 服务 4.8 物流 4.7

以平铺为主，点进去看详情页是有模特图的。款式简单，但是衣服上的图案很有意思。T恤居多，价格在50到100之间。如果身边的男性朋友喜欢穿简简单单，不喜欢太繁琐设计的衣服可以把这家店铺安利给他哦

马切达工作室

描述 4.7 服务 4.8 物流 4.8

强烈安利这家店铺，款式很丰富，不单调，衣服风格偏运动外向，有几款短裤真的是超级好看，条纹t恤也很不错。

GBOY潮流日系韩版男装

描述 4.7 服务 4.7 物流 4.7

这家店价格很公道，款式也很简单。T恤大部分偏修身，身边的异性朋友不喜欢宽松的衣服可以考虑这家。

东京衣柜男装工作室

描述 4.7 服务 4.7 物流 4.7

这家店的牛仔裤好评，第一次觉得男生穿破洞牛仔裤那么好看。

港仔文艺男

描述 4.7 服务 4.7 物流 4.7

这家店神店啊，养活了很多男装档口，月销量55万笔。我朋友买过的，性价比还是可以的。适合韩范小男生穿，女生也可以穿的。

Bytehare 兔先森 玩药

描述 4.8 服务 4.8 物流 4.8

这家店的服装搭配做的不错，对于不会搭配的男生可以考虑这家店。这家店跟流氓兔男装店很像，怀疑是不是同一家公司。有几款情侣装蛮有意思的，想买情侣装的可以看下。

几立

描述 4.7 服务 4.8 物流 4.8

想挑选纯色衬衫的可以考虑这家店，衣服有点偏日系清新哦。

Mrlin潮牌铺日系原创

描述 4.7 服务 4.8 物流 4.8

这家店铺是中式复古风格的，想挑一款盘扣衬衫的可以进这家店看下

流氓兔的男装店

描述 4.7 服务 4.8 物流 4.8

这家店拍照方式还有款式都跟Bytehare 兔先森 玩药 是一样的，所以参考价值是一样滴，我买过他们的条纹t恤和裤子，性价比真的是不错的。

SEEZ成衣店

描述 4.8 服务 4.8 物流 4.8

衣服也是偏日系，但没那么斯文，照片拍的不错，喜欢她们在东极岛拍的那期。

白术自制

描述 4.9 服务 4.9 物流 4.9

喜欢中性风格的女生可以看下这家店，店主是个很漂亮又很帅气的女生哦

听风原创日系文艺男装

描述 4.7 服务 4.8 物流 4.8

日系清新风格哦，斯文干净清秀的男生可以看下，衣服偏修身，码数可能有点小，他的白衬衫好评哦。

NOTHOMME

描述 4.7 服务 4.8 物流 4.8

外向张扬气质刚毅的男生可以考虑下哦
