魔兽世界8.3预览：新增同盟种族，神器系统更新，法师塔玩法回归

2020-10-29
| 游戏

魔兽世界8.2.5才上线没多久，8.3的PTR也马不停蹄的到来了。此次8.3更新比8.2更新了更多了内容，新增同盟种族机械侏儒与狐人、黑龙王子携橙披再度归来，法师塔模式将再次回归等一系列重大更新。

本次8.3更新的主要内容有：

Ø 新同盟种族——机械地精、狐人

Ø 新团队副本——尼奥罗萨

Ø 新地图入侵事件与泰坦系统

Ø 法师塔玩法回归

Ø 神器项链更新

Ø 新的秘境词缀

Ø 狼人与地精传承护甲

新同盟种族——机械地精、狐人

此次更新两个阵营将会分别新增一个同盟种族：机械侏儒（联盟）和狐人（部落），之前被猜测会与机械侏儒对位的海地精种族被替换成更萌的狐人。（种族技能名称为暂译）

机械侏儒

Ø 解锁条件：完成“麦卡贡的威胁”成就，锈栓抵抗军声望崇拜。

Ø 可选职业：猎人、法师、武僧、牧师、潜行者、术士、战士。

Ø 种族技能：

n 作战分析：5秒提高一次主属性（120级为50），最多可叠加8次，直到战斗结束后衰减。

n 超级有机光源发明家：召唤一个分身吸引敌方注意力。3分钟CD。

n 骷髅小指：可以打开上锁的箱子和门。

n 工艺大师：你将可以充当铁砧、烹饪用火、矿锄等多种专业工具。

n 重新武装：生命值降低至20%时，立即恢复20%生命值。该效果每1.5分钟触发一次。

Ø 种族坐骑和传承护甲如上图所示。

狐人

Ø 解锁条件：完成“深埋于沙的秘密”成就（沃顿主线任务），沃顿声望崇拜。

Ø 可选职业：猎人、法师、武僧、牧师、潜行者、萨满祭司、术士、战士。

Ø 种族技能：

n 技巧之袋：从你的技巧之袋袋中选择技能，可对友方或敌方使用。1.5分钟CD。

n 翻动包裹：更改技巧之袋内的技能。

n 营地：设定该位置为你的营地，只能在户外使用。5秒CD。

n 返回营地：返回设定的营地。10秒施法时间，1小时CD。

n 防火：减少1%火焰伤害。

n 敏锐嗅觉：降低受到敌人第一次攻击的伤害。

n 狐人生存之道：杀死人形生物后获得额外的战利品。

Ø 种族坐骑和传承护甲如上图所示。

新团队副本——尼奥罗萨

本次新团本将是12个BOSS的大型团队副本，1号BOSS为黑王子拉希奥，尾王为古神恩佐斯。同时，熊猫人团本雷电王座的隐藏BOSS莱登也将作为BOSS出现在新团本中。

介绍语：

沉睡之城尼奥罗萨已然觉醒。经过了无数纪元后，恩佐斯终于再度登上黑暗帝国的宝座。他的大军从黑暗的殿堂中倾泻而出，再次令艾泽拉斯生灵涂炭。整个世界在他们的魔爪下颤抖。于是，部落和联盟的勇士汇成了联军，孤注一掷地深入这支上古大军的大本营，与其进行最终的决战。这场战斗将会决定整个宇宙的命运。

新地图入侵事件与泰坦系统

恩佐斯的使者将会入侵奥丹姆和锦绣谷两地的泰坦设施，届时这两处将会被古神的能量笼罩，玩家需要前往清理这些古神的能量使其恢复如初。不过官方并未对该玩法有进一步的说明，不过暴雪总监在介绍该玩法时，提到了该事件将会改变泰坦铸造系统。（下版本要消灭泰坦造物了吗？#偷笑）

法师塔玩法回归

新的类似于法师塔挑战的玩法“恐怖愿景”。恩佐斯的势力将会入侵奥格瑞玛和暴风城，玩家可以1-5人前往进行挑战。进行挑战时会获得拉希奥提供的帮助，获取并强化橙披，以抵抗古神的入侵。在挑战中还能获得“恩佐斯的腐化碎片”，可用于强化神器项链。目前还未有奖励幻化的消息，不过应该会有与法师塔类似的奖励。

神器项链更新

Ø 神器最高等级将提升至85（原70级）。

Ø 新增1个次插槽。

Ø 新增2个耐力点数。

Ø 新增7个精华。

Ø 新增与恩佐斯相关的巅峰插槽。

新的秘境词缀

新的秘境词缀“方尖碑（Monolithic，暂译）”。在地下城中将会出现一定数量的方尖碑，玩家点击方尖碑后将会进入另一个位面去清除恩佐斯的仆人，如果玩家跳过这些方尖碑的话，这些恩佐斯的仆人将会在尾王出现。

另外，还有一系列的职业改动，由于目前才刚开始PTR测试，后续肯定还会继续进行改动，就不一一列举了。总体来说，武僧三系中，踏风和奶僧得到一定加强，酒仙醉拳相关技能有一定的削弱。鸟德削弱了月火和阳炎。暗牧削弱了吉兆的伤害加成、狂乱合唱（特质）爆击加成、两个dot技能的伤害。

除了上述更新外，拍卖行系统也进行了改动，玩家购物将会更便捷。英雄难度黑海岸也将会在8.3上线。还有一系列新坐骑、新装备等相关信息我们会在后续的文章中一一进行介绍，欢迎关注我们天涯电竞。
