19岁女生李心草死亡事件，立案侦查！出事前这个细节被的哥曝光

2020-10-29
| 游戏

//

昆明刑警再调查涉事酒吧

//

据上游新闻报道，记者在现场看到，昆明警方刑侦人员于14日中午12点30分左右进入热度酒吧，多名刑侦人员进入酒吧后关上卷帘门进行调查。中午1点50分许，昆明警方刑侦人员结束调查，并对酒吧外观等拍照后离开。


警员对上游新闻记者表示，警方目前仍然在对此案进行调查，“尸检结果尽快在做。”对于当天早晨有媒体发布的涉案信息，“我们没有发过。”









10月14日下午2点左右，李心草坠江地、位于昆明市盘龙区桃源街132号的热点网吧，昆明刑警再次赴现场勘查。（来源：上游新闻）

//

出租车司机：两男子劝女孩下车

//

10月13日上午，澎湃新闻（www.thepaper.cn）根据李心草家属提供的车牌号，联系到了9月9日晚李心草乘坐的出租车。

万通出租车公司陆姓出租车司机表示，此前他已经接受过警方对此事的调查。据其介绍，9月9日晚，他开车经桃源街时，见一名女孩（李心草）站在酒吧门口招手叫车，女孩旁边还站着一名男子。随后，他将车开到路边，女孩从后门上车坐在后排位置。

司机称，女孩上车后，前述男子将车门打开，劝女孩下车。随后，另一名男子从酒吧里出来，站在车边。两男子均劝女孩下车，并称“你喝多了，再玩一会儿，等下一起走”。

司机告诉澎湃新闻，女孩上车后，“没有说过一句话，脸上没有任何表情，就呆坐在后排座位上”。司机称，大约坐了两三分钟后，女孩从后排座位的另一侧下车，走向了江边的方向，“（女孩）走的很快的样子，两名男子也跟着女孩向江边方向走去”。

李心草坠江前，

到底发生了什么？

事发后受害方家属

有否与被警方调查的3人有接触

这些问题亟待解答……


10月14日，在接受封面新闻采访时，李心草亲属回应了相关问题。

心存疑虑，无法接受调查结论


李心草母亲


家属表示，他们曾在9月15日，第一次看过那段长达三小时的视频。


他们选择录下所发布的片段，是因为在事发当天9月9日，李心草母亲第一次见在场三人时，涉事男性罗某曾向家属保证，“发誓没有与李心草发生过冲突，没有碰过违禁品，完全没有碰过李心草，也没有过语言刺激和劝酒”。


尝试调解，并未达成任何共识


李心草家属透露，在看到视频前，9月11日警方曾组织李心草一方和涉事同学一方进行调解。


其中，任某及其母亲，和涉事两位男性参与了调解。在协商中，李心草舅妈曾向三人提出80到90万的赔偿金额讨论，“主要用于心草妈妈身体的治疗和外债偿还还有精神赔偿”。

据李心草母亲透露，涉事两位男性以父母不在为由表示回绝，而任某的母亲则一再强调自己是单亲家庭，经济困难。


“她一直在诉苦，说自己的女儿出事之后也很不好过，今后的路也很难走，希望我们能够换位思考，想想他们的难处”。任母的说法彻底惹怒了李心草家属，一行人当场离席，调解草草结束，没有达成任何共识。


李心草母亲仍在和女儿微信“对话”

拒绝调解 否认花钱买热搜


李心草家属表示，在9月15日看到相关视频后，他们拒绝了一切调解事宜，“我们只想要真相”。

10月14日下午，社交媒体流传出一张微信对话截图，称李心草的母亲出钱买热搜。上游新闻就此求证了李心草亲属。多位亲友证实，李心草母亲的微信头像并非网传截图中的头像，“我们从来没有花钱买过热搜，从始至终都只有一个头像。”


社交媒体流传的微信截图，称李心草母亲出钱买热搜，遭李心草家属否认。
