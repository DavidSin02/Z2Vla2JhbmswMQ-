云顶之奕9.14版本必须知道的事：测试排位现问世，鸟盾法师成历史

2020-10-29
| 游戏

国服云顶之奕最近大家正在玩的不亦乐乎之时，国服已经在7月23日开始更新9.14新版本，这一版本可以说是一次大改，主要做的就是针对上一版本中的不平衡现象和大多数bug来进行调整，削强补弱让游戏更加平衡。而相对于现在的环境产生了不小的影响的，如果想要在排位开放后冲击王者，以下这些结合改动分析的干货一定要看完哦。

  1. 新英雄 卡牌大师 崔斯特

再来看看云顶中卡牌的属性与思路

生命值：450 / 850 / 1700

法力值：50

护甲：20(降低16.7%)

魔抗：20(降低16.7%)

攻击力：40 / 72 / 144

攻击速度：0.75

攻击距离：3格

技能：选牌（也就是我们常说的切牌了，不过是随机的）

崔斯特扔出一张卡牌，黄牌伤害并晕眩目标 2/3/4 秒，红牌伤害目标和周围的敌人 100/150/200 点，蓝牌回复他自身和附近友军的法力值 20/35/50 点。

定位：暂时评价为一个合格的打工者，从技能上解读每一张牌，单体控制还可以，红牌伤害太低，蓝牌回蓝作用不明显，都不是那么给力，最大的作用就是作为海盗，前期弥补了船长没来之前的空白期，快速3海盗赚钱是一个新的走势这点毋庸置疑，但是无论是中期的船长还是后期的好运姐都比卡牌实用的多，即使男枪的坦度也是超过卡牌的，而且因为是2费用，如果去追3星就没什么必要了，所以后期一般不会再上场，除非什么海盗法师的野路子也许会用，所以这次的卡牌就是海盗流恰烂钱的福音，对于其他流派选择的可能性不大。

下面介绍一下其他平衡改动，直接加套路分析，具体细节官网有公布：

  1. 恶魔的影响

（1）法力燃烧机率：40/60/80% → 25/50/80%

避免了恶魔打工的局面，2恶魔基本没用，这一改动就是为了让恶魔不再成为陪衬。

（2）蜘蛛：生命和护甲都降低一点，一星和二星的小蜘蛛都减少了一只，小蜘蛛继承恶魔效果，所以小蜘蛛也可以法力燃烧，最重要的就是变成1费了，恶魔终于有1费了，前期打工也好，弄3星做基石都没问题。

（3）剑魔：法力值消耗：100 → 75 伤害：350-850 → 400-1000真正的加强了一波。

（4）寡妇：生命值：550 → 600 处决血限：50% 生命值→ 65% 生命值 处决伤害倍数：300/400/500% → 300/500/700% 伊芙琳在施法期间(0.35秒)会将目标定身防止他们离开技能范围。技能不会空了，非常大的加强，无论对于恶魔还是刺客都算增强。

（5）莫甘娜：当技能范围内没有敌人的时候不再会施放技能，伤害：250-450 → 300-450同样不会空技能了，也成为恶魔中的必选牌了。

（6）火男： 法力值消耗：100 → 125，略微削弱，增加了放两次大招的时间，影响不大，火男不做大哥的阵容一场只能放一次大招，火男大哥有装备不在乎这点蓝。

2.bug修复改动

（1）诺手：当技能范围内没有敌人的时候，不再会施放技能，所以必定吸血，小幅增强。

（2）狐狸： 修复了导致阿狸的技能在其阵亡后无法造成伤害的bug，现在目标在其施法时间内阵亡的话会重新选择，目标技能目标选择：随机敌人→ 当前攻击的目标。云顶之奕第一假赛王的帽子终于可摘掉了。

（3）机器人：修复了一个导致布里茨在施法过程中阵亡会将单位拉出战场的bug，现在附近的友军更有可能选择布里茨拉过来的单位作为目标，小优化，拉人过来围殴就是了。

（4）布隆：修复了布隆的提示说明，使其能够正确的反映出护盾接下来所减免所有伤害的百分比，伤害减免： 60-90% → 70-90%，法力值消耗：75 → 50，技能目标选择： 最近的敌人→ 最远的敌人。之前的布隆就像木桩一样又笨又蠢，这回我们也许能看到打职业的布隆了。

（5）卡特：修复了一个导致卡特琳娜技能无法正确地降低敌人的治疗效果的bug，不太重要。

（6）阿卡丽：修复了一系列会导致其技能偶尔无法造成伤害的bug，当技能范围内没有敌人的时候，不再会偶尔施放技能。不空技能，后排的脆皮们更要颤抖了。

（7）女枪：技能目标选择：随机敌人 → 当前攻击目标，现在在准备动作过程中目标阵亡的话会重新选择目标， 修复了一个导致厄运小姐偶尔会在施法时移动并改变大招的轨迹的bug，法力值消耗：75 → 100。没实际操作一下不好说这个改动是否算改良。

（8）更新了魔法水晶箭弹道的视觉特效，魔法水晶箭现在会播放音效

修复了一个导致从共享选秀中合成升级的英雄有时会复制该英雄的bug

重新连接到云顶之弈游戏中不再会使得游戏结束后的统计无法显示

修复了一系列会导致近战英雄原地不动以及停止移动的Bug

成吨的其他Bug修复已在上方列出

修复了一个BUG，该BUG曾导致：生命值装备给所有2星单位提供了过多的生命值

修复了一个BUG，该BUG曾导致：2星和3星的换形师在变身后从所有来源获得的生命值过多

其他的英雄改动比较小，就不再强调了。

3.装备改动

装备不再受到法术强度的加成，每个中立回合现在都会掉落奖励。奖励包括装备组件，金币和经验值。PvE回合中，如果小兵或野怪没有掉落装备则会掉落金币，真的是非酋的福音呢。 亚龙和远古巨龙不再获得龙类特质的收益，法师流玩家可以开始庆祝了。

（2）金铲子：金铲铲现在会使得其相关的装备组件属性翻倍。例如：幽梦之灵提供40点攻击力

（3）诅咒之刃：现在可以将目标降低至0星阶，修复了一个导致诅咒之刃第二次触发时会重置一名敌方英雄的星阶的bug，新出了拉面熊暴打小朋友玩法，攻速快的，带aoe的都可以用用这个，效果出奇的好。

（4）复活甲：复活延迟：4 秒→ 2 秒，生命值回复：500 → 100， Bug修复：现在守护天使会正确地清除重伤效果。这才是有价值的复活甲，给有格子的大哥来一个没毛病。

（5）离子火花：现在会造成真实伤害而非魔法伤害，修复了一个导致离子火花会造成75点而非200点伤害的bug。这成为这一版本的神器，广大玩家谁用谁知道呀。

（6）救赎：现在在25%生命值时触发，不再对敌人造成伤害。终于不再是自杀救赎了，稍微有点用。

（7）卢安娜的飓风：箭矢数量：2 → 1， 箭矢伤害：50% → 25%，现在附带普攻特效。很多骚套路这回玩不了了。

（8）大天使：双大天使可以叠加了

（9）反甲：反弹伤害：所承受物理伤害的35%→所减免伤害的100%，伤害类型：魔法伤害→ 真实伤害。跟游戏差不多，按照护甲和魔抗的强度来决定反弹伤害，如果是龙王或者龙女，那就变成法师克星了。

（10）基克的先驱： 现在只为持有者左右两格内的英雄提供攻速，攻击速度光环：10% → 15%对于站一排的刺客来说每个人多了半个攻速弓，太合适了。
