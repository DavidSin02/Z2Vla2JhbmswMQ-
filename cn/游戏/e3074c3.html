《狂怒2》PC配置需求公布：最低GTX780，推荐GTX1070

2020-10-29
| 游戏

由B社发行、id Software打造的末日废土题材FPS游戏《狂怒2》公布了PC配置需求。游戏最低配置要3代i5和GTX780（3GB），推荐配置要4代i7和1070，AMD方面则需要R5 1600X+Vega56可以算是中规中矩没有达到夸张的地步。

最低配置：

需要64位处理器和操作系统

操作系统：Windows 7，8.1，10（64位）

处理器：Intel Core i5-3570 或 AMD Ryzen 3 1300X

内存：8GB RAM

显卡：Nvidia GTX 780 3GB 或 AMD R9 280 3GB

存储空间：50 GB可用空间

推荐配置：

需要64位处理器和操作系统

操作系统：Windows 7、8.1、10（64位）

处理器：Intel Core i7-4770 或 AMD Ryzen 5 1600X

内存：8GB RAM

显卡：Nvidia GTX 1070 8GB 或 AMD Vega 56 8GB

存储空间：50 GB可用空间

《狂怒2》是经典游戏《狂怒》的最新正统续作。新作在风格上继续保持废土风和第一人称FPS的玩法，游戏将更加火爆、血腥和疯狂。玩家将进入一个没有社会，没有法律，没有秩序的反乌托邦社会。《狂怒2》将带来真正的开放世界FPS体验，在本作中玩家哪儿都能去、能随意开火，随意引爆轰炸。

《狂怒2》将于5月14日登陆PS4/Xbox One和PC平台。
