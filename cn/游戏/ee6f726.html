2020-10-29

Nike 即将发布全新 React Sertu 鞋款





自 Nike 的 React 技术在 2017 年率先在篮球鞋上应用之后，品牌接连推出了 React Element 87，React Element 55 等优秀鞋款。




而据可靠消息表明，Nike 即将为 React 家族加入新成员，以机能风格为主轴的全新 React Sertu 鞋款。




新的 React Sertu 在鞋面上运用多个区域缝制，以不同物料及制作手法建构，整体观感变得更有细节。




据悉鞋款即将于 Nike 专门店及部分线上店铺如 Sivasdescalzo 推出，售价约为美金 $169。

YEEZY BOOST 350 V2

全新地区限定发售日期确认





不久前 adidas Originals 于官方 IG 账号揭示了YEEZY BOOST 350 V2全新地区限定系列的确切发售讯息。品牌将于 6 月 21 日抢先发布「Antlia」配色反光版本，6 月 22 日才发布「Antlia」不反光版本，并仅在欧洲，俄罗斯和乌克兰发售。




以灰粉色调登场的「Synth」配色，也将于同日期在非洲，亚洲，印度和中东地区发售，同样具有反光及不反光版本。




至于以浅米色 Primeknit 物料构建鞋身的「Lundmark」配色，将会只在美洲地区限定发售，于 7 月 11 日抢先发售满天星版本，于7 月 13 日大规模上架不反光版本。值得一提的是，三双区域限定鞋款皆配备成人、大童、婴儿三种尺码版本。

Off-White™ x Air Jordan 1 有望发售儿童尺码





作为 2017 年最热门的球鞋联名合作项目之一，Off-White™ x Air Jordan 1 从最初代的芝加哥配色，到后来欧洲限定的全白，以及致敬 Michael Jordan 母校的北卡蓝配色，无一不被鞋迷们所喜爱，在转卖市场也拥有惊人的价格。而根据球鞋网站 @OnusInfo 的最新消息表明，这三双在球鞋市场上拥有超高人气的 Off-White™ x Air Jordan 1 极有可能于今年十月迎来儿童尺寸鞋款的发布...实名酸一波弟弟们...
