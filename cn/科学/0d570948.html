华东理工大学刘昌胜院士团队《AFM》：可注射的水凝胶促进软骨和血管的原位形成

2020-11-14
| 科学

骨损伤的高病发率和当前临床治疗的局限性促使人们开发具有高治疗功效的可替代生物材料，来促进骨骼再生。传统的骨修复材料涉及将间充质干细胞直接转化为成骨细胞，然后通过膜内骨化过程（IMO）促进骨形成。尽管该方法在体外具有巨大的潜力，但由于血管生成不足和灌注能力有限，IMO经常出现血管坏死等问题，从而导致成骨失败。当前，大多数骨骼可通过替代性软骨内骨化（ECO）形成。在ECO过程中，软骨对营养和氧气的需求较低，随着软骨细胞的成熟，在后期可为成骨和血管生成提供多种细胞因子。近年来，这种有效的骨再生方法引起了人们极大的兴趣，并且许多研究集中在通过将生物材料与细胞结合，用于体外构建软骨，或用于体内植入来诱导ECO。

华东理工大学刘昌胜院士、袁媛教授等人开发了一种可注射的聚（癸二酸甘油酯）-共聚（乙二醇）/聚丙烯酸（PEGS / PAA）水凝胶，其可诱导低氧环境，并通过原位铁螯合重塑ECO进行骨再生。随着PAA的加入，该水凝胶在注射后呈现出显著的粘弹性和与铁离子螯合的高效率，从而激活了HIF-1α并抑制了炎症反应，从而改善了软骨的早期形成，并在后期促进了血管生成。更重要的是，通过PEGS / PAA水凝胶在整个再生过程中持续稳定地激活HIF-1α，可以实现软骨/成骨的平衡，从而促进ECO的发展。这些发现提供了一种有效的策略，可通过基于生物材料的铁离子螯合和随后的低氧环境来实现原位ECO，这为未来在骨再生中的应用提供了新的思路和概念。该研究以题为“Recapitulation of In Situ Endochondral Ossification Using an Injectable Hypoxia‐Mimetic Hydrogel”的论文发表在《Advanced Functional Materials》上。

【可注射的PEGS水凝胶】

在制备PEGS / PAA水凝胶之前，作者首先优化了PEGS水凝胶作为基质的方案。作者使用来自PEGS预聚物的典型硫醇-Michael加成点击反应制备PEGS水凝胶。考虑到最终可注射的PEGS水凝胶的水溶性，作者选择了30%PEG含量的预聚物，其表现出最好的亲水性和更高的羟基利用率，以确保水凝胶的可注射性和胶凝性。当含量等于或超过40％时，羟基含量将太低而无法形成可用的双键，导致胶凝时间过长。PEGS30预聚物的SEM成像显示，PEGS水凝胶有相互连接的大孔结构，并且可以通过改变硼砂浓度来调整微孔的尺寸。具体来说，将硼砂浓度从1：4增加到1：2，微孔的大小能够从大约6减小到2 µm（图1f）。PEGS水凝胶的体外降解性能表明，随着硼砂浓度的增加，降解速度变慢（图1d）。因此，考虑到降解和凝胶特性，作者最终选择了PEGS30 1：3水凝胶进行后续研究（图1e）。

图1 PEGS水凝胶的表征

【诱导缺氧环境】

HIF-1ɑ是细胞适应过程中低O2感测的关键。但HIF-1ɑ很容易在氧和亚铁离子的存在下被脯氨酰羟化酶羟化，然后迅速被蛋白酶体降解。也就是说，低氧水平或消除铁离子可以有效提高HIF-1ɑ的稳定性。然而，当前仍然缺乏一种可以完全螯合铁离子而不添加缺氧剂的纯生物材料方法，更不用说模拟缺氧环境来重塑和改善ECO，从而实现有效的骨再生。在这项工作中，PEGS / 30PAA在水凝胶中具有更多的可用羧基，并表现出较强的铁离子螯合活性（图2A）。在培养3天和7天后，在PEGS / PAA组的中观察到了较高的HIF-1α蛋白（图2F，H）。术后1周从股骨缺损处去除水凝胶后，作者发现PEGS / PAA水凝胶中铁离子的吸附更加强烈（图2B）。相应地，在骨基质和周围PEGS / PAA水凝胶中的细胞中均观察到了强烈的HIF-1α染色（图2I），这表明该水凝胶在体内模拟出了低氧微环境。

图2 PEGS / PAA水凝胶诱导的低氧微环境

【促进血管生成和重塑ECO】

作者探索了PEGS / PAA对体内血管形成的影响，结果如图3所示。与PEGS组相比，PEGS / PAA组中CD31的表达增加了，这表明有更多的血管再生（图3A，C）。在4周和8周后，PEGS / PAA组的血管生长量明显大于PEGS组。此外，PEGS / 30PAA组表现出最大的血管生长量（图3B）。接下来，作者探索了PEGS和PEGA / PAA水凝胶诱导的体内骨骼再生。在PEGS水凝胶组中，新骨形成低于PEGS / 10PAA或PEGS / 30PAA组，并且仅在材料的外围形成少量新骨组织。相反，在PEGS / PAA水凝胶组中能观察到明显的骨形成。在第8周时，接受PEGS水凝胶的股骨比第4周时显示出更多的矿化，但仍有一些缺陷。而PEGS / PAA水凝胶组的骨缺损在第8周时就已完全修复，并在PEGS / 30PAA组的骨形成最好。这些结果证实，PEGS / 30PAA可以有效促进骨骼再生。

图3 PEGS / PAA水凝胶促进血管和骨生长

总结：作者介绍了一种新型的PEGS / PAA水凝胶，该水凝胶可通过低氧微环境的原位模拟来加速基于ECO的骨再生。具体而言，具有更充分的HIF-1ɑ稳定性的PEGS / 30PAA具有最佳的体内骨再生效率。这项研究提供了有力的证据表明，可以使用原位螯合铁离子来诱导和调节HIF-1ɑ。最重要的是，这项研究中HIF-1ɑ的持续和稳定激活对于在ECO中实现软骨/成骨平衡并从而达到有效的骨修复至关重要。作者的发现对于理解生物材料引起的生物学效应以及协助新型骨再生的发展也具有重要意义。
