什么是全等三角形

能够重合的两个三角形叫做全等三角形

三角形全等的判断定理

SSS（边边边）:三边对应相等的两个三角形相等的两个三角形全等

ASA（角边角）:两个角及其夹边对应相等的两个三角形全等

SAS（边角边）:两边及其夹角对应相等的两个三角形全等

AAS（角角边）:两角及其中一个角的对边相等的两个三角形全等

S:side,边； A：ankle,角

注意：没有ASS,为了不弄错，给如下记忆方法：如果有2个S，必须是2个S夹1个边，或者夹1个角。

通过等边（等腰）三角形构造全等——例题一

等边三角形ABC,DE//BC,延长E，使EF=EC，证明：△ABE≌△AFD


证明思路：

条件（等边三角形ABC和DE//BC）可得AD=AE，∠ADF=∠BAE=60°，可得△ADE为等边三角形。条件（EF=EC），可得AC=DF=AB 根据SAS（AD=AE,∠ADF=∠BAE,DF=AB）,证明得：△ABE≌△AFD

通过等边（等腰）三角形构造全等——例题二

△ABE和△ACF均为等边三角形 ,证明：

（1）△ABF≌△AEC； （2）∠B0E=∠BAE=60°


证明思路：

条件（△ABE和△ACF均为等边三角形）可得AE=AB,AC=AF,∠EAB=∠CAF（两边已在还差一角） ∠EAC = ∠EAB+∠BAC,∠BAF = CAF+BAC,可得 ∠EAC = ∠BAF （相等角+公共角自然相等） 根据SAS易证△ABF≌△AEC

因为全等，∠AEC=∠ABF，令AB和EO交于点G，∠AEC + ∠AGE + ∠EAG = 180°,∠ABF + ∠BGO+ ∠BOE= 180°,易得∠B0E=∠BAE=60°（利用三角和为180°，其中2个角相等，则剩下的角自然相等）

通过等边（等腰）三角形构造全等——例题三

△ABD和△ACE均为等腰直角三角形，证明：

（1）BE=CD （2）BE⊥CD


证明思路：

根据SAS易得△BAE≌△DAC,所以BE=CD


要证BE⊥CD，也就是∠DOB=90°，因为全等，∠EBA =∠CDA,通过：三角和为180°，其中2个角相等，则剩下的角自然相等，可得∠DOB=∠BAD=90°。

通过等边（等腰）三角形构造全等——例题四

如图，△ABC、△CDE都是等边三角形，且点A、C、E在一条直线上，AD与BE、AD与BC、BE与CD分别交于点O、点P、点Q．求证：

（1）、AD=BE （2）、∠AOB=∠ACB=60°

（3）、△PCQ为等边三角形 （4）、PQ∥AE

（5）、AP=BQ （6）、OA=OB+OC



（1）有前面几个例题的练习，易证△ACD≌△BCE,所以AD=BE


（2）因为全等，∠DAC = ∠EBC,易得∠AOB=∠ACB=60°（三角和为180°，其中2个角相等，则剩下的角自然相等）

（3）要证全等三角形，要么证3边相等，要么证2边相等且有1角为60°，条件（点A、C、E在一条直线上和∠BCA=60°，∠DCE=60°）可得∠PCQ=60°，那么只要证PC=QC即可。发现直接证明不行，那就找与这两边有关的三角形是否有全等关系，也就是△PCD和△QCE是否相等。通过（1）中的全等，可得∠PDC=∠QEC,又因为∠PCD=∠QCE=60°,CD=CE，可得△PCD≌△QCE，所以PC=QC，所以△PCQ为等边三角形（两条边相等,一个角为60°）。

（4）因为△PCQ为等边三角形 ，所以∠PQC=60°,因为∠DCE=60°，PQ∥AE(内错角相对两直线平行)

（5）在（3）中已经证明，△PCD≌△QCE，所以PD=QE，因为（1）中证，AD=BE,所以AP=AD-PD,BQ=BE-QE,所以AP=BQ （相等边减去同长边相等）

（6）要证OA=OB+OC，总得把OB和OC整到一条线上去，这样可以通过两条线段相等得出等式成立。所以作CG=CO，交BE于点G。发现△OAC和△GBC应该全等（因为OC=GC，AC=BC，两边已有，还差一个夹角）。所以只要证明∠ACO=∠BCG，即∠OCG=∠ACB=60°即可。CG是自己做出来的，很难证明∠OCG或者OGC为60°，那么只能通过证明∠COG=60°，证明△OCG为正三角形来得到∠OCG=60°。如果∠OCG=60°，之前已经证明∠AOB=60°，所以只要证明∠AOC=60°就行，观察△APB和△CPO，如果它们相似，就可以得到∠AOC=∠ABC=60°。因为之前已经证明，△BPO和△APC中的相应角都对应相等，所以△BPO∽△APC，即PB:PO=PA:PC,因此△BPA∽△OPC(相似判断定理：两边对应成比例，其夹角相等的两个三角形相似)，所以∠AOC=∠ABC=60°，最后反推上去，得到△OAC≌△GBC,最后可得OA=OB+OC。

通过构造等腰三角形，找全等三角形，在证明全等的过程中，通过相似解决了全等三要素中的一个要素，也就是解决了一个角相等的问题。（6）是一个特别好的题目，综合全等和相似。



通过等边（等腰）三角形构造全等——总结

①等腰、等边中有一对边和一对角，只要再找一角或者一边就能证明全等。

②有时候没有等边等腰三角形，自己要学会构造等腰、等边三角形来找全等三角形

倍长中线法构造全等

倍长中线法介绍：将三角形的中线延长一倍，以便构造出全等三角形，对顶角+2对等边，即SAS可证全等。

倍长中线法构造全等——基础模型

△ABC中 ，AD是BC边中线。

模型1：延长AD到E，AD是BC边中线，使DE=AD，连接BE。通过SAS易证△ADC≌△EDC


模型2：作CF⊥AD于F，作BE⊥AD的延长线于E，连接BE。通过ASA易证△FDC≌△EDB


模型3：延长MD到N，使DN=MD，连接CN。通过SAS易证△BDM≌△CDN



倍长中线法构造全等——例题一

已知，如图△ABC中，AB=5，AC=3，则中线AD的取值范围是

解题思路：延长AD到E，使DE=AD。易证△ADC≌△EDC，AB+AC=AB+BE=5+3=8,通过三角形两边和大于第三边和两边差小于第三边可得，5-3<2AD<5+3,可得1<AD<4。



倍长中线法构造全等——例题二

如图，△ABC中，E、F分别在AB、AC上，D是中点，DE⊥DF，比较BE+CF与EF的大小.


例题二

解题思路：

延长ED到G，使ED=DG，连接GC。易证△BDE≌△CDG，可得BF=CG。因为ED⊥FD可得∠EDF=FDG=90°。连接FG，通过SAS易证△EDF≌△GDF，所以FG=EF，可得BE+CF = CG+CF>FG=EF,即BE+CF>EF



倍长中线法构造全等——例题三

已知在△ABC中，AD是BC边上的中线，E是AD上一点，且BE=AC，延长BE交AC于F，求证：AF=EF


证明思路：

延长AD到G，使AD=DG，易证△ADC≌△GDB，可得BG=AC,∠G=∠EAF。因为BE=AC，所以BG=BE，即∠G=∠BEG，所以∠EAF=∠BEG=∠AEF，所以AF=EF。


倍长中线法构造全等——例题四

已知CD=AB，∠BDA=∠BAD，AE是△ABD的中线，求证：∠C=∠BAE


证明思路：

延长AE到F，使EF=AE，连接FD，FC。易证△ABE≌△EFD，得AB=FD，∠B=∠EDF。看到△ADC和△ADF有种莫名的亲切感，因为CD=AB=FD,AD=AD(两边已在差1个夹角就有全等)，所以接下去的关键就是∠ADC=∠ADE，∠ADC=∠B+∠BAD,∠ADF=∠BDA+∠EDF,而∠B=∠EDF,∠BDA=∠BAD,所以∠ADC=∠ADE，然后通过SAS证明△ADC≌△ADF，可得∠AFD=∠C=∠BAE。

倍长中线法构造全等——例题五

在四边形ABCD中，AB//DC，E为BC边的中点，∠BAE=∠EAF，AF与DC的延长线相交于点F。试探究线段AB与AF、CF之间的数量关系。


解题思路：

延长AE和DF交于点G（注意这时候AE暂时不等于EG），因为AB//DC，可得∠BAE=∠G,因为∠BAE=∠EAF，所以∠G=∠EAF，即AF=FG

因为E为BC边的中点，可得BE=EC，∠AEB=∠GEC(对等角)，根据AAS易得△ABE≌△GEC，所以AB=GC=FG+CF=AF+CF,

反思：看似是倍长中线法，实际上不是。因为延长AE的时候，我们没有作AE=EG，虽然可以证明它们相等。为了方便使用AB//DC这个条件，我们选择作AE和DC的延长线，而不是作AE=EG，为何？自行体会！
