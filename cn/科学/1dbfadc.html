功能性消化不良临床常用中成药及注意事项，来了解一下

2020-10-29
| 科学

功能性消化不良是指具有上腹胀、上腹痛、早饱、嗳气、食欲不振、恶心等上腹不适症状，经检查在排除引起这此症状的器质性疾病的一组临床综 合征。功能性消化不良是临床上最常见的一种功能性胃肠病，同时多伴有全身性精神症状，如失眠、焦虑、抑郁、注意力涣散、健忘、神经过敏、头痛等。

一.香砂六君丸

【药物组成】木香、砂仁、党参、炒白术、茯苓、炙甘草、陈皮、姜半。夏辅料:生姜、大枣

【临床应用】 本品适用于功能性消化不良，见胃部胀闷不舒甚或隐隐作痛，呕吐酸水，食欲减退，四肢倦息，舌胖大舌边有齿痕，苔腻的患者。

【注意事项】

1.不适用于口干、舌少津、大便干者。

2.呕叶伴胃部灼热疼痛，口干，大便干者忌用。

3.忌食生冷油腻、不易消化及刺激性食物，宜食清淡易消化之品，戒烟酒。

4.对本品过敏者禁用，过敏体质者慎用。

【用法用量】口服，一次6-9克，1日2-3 次。

二、香砂枳术丸

【药物组成】木香、砂仁、白术(麸炒)、枳实(麸炒)

【临床应用】

1.本品适用于功能性消化不良，见腹部胀满不舒，喜嗳气，嗳气后胀满减轻，食欲欠佳，食少，食后不舒，大便稀或不成形，舌淡的患者。

2.若胃部冷痛，喜温喜按，可与理中丸同服。

3.若患者平时情志不畅，生气或郁闷时易于发病，可与逍遥散同服。

【注意事项】

1.若便秘口苦、舌红苔黄脉滑者，不宜服用本品。

2本品方中有破气之枳实，孕妇慎用。

3.便秘患者慎用。

4.服药期间忌食生冷、油腻食物，宜食清淡易消化之品。

5.舌红无苔、口干咽燥，手足心热，心烦失眠等患者忌服。

【用法用量】口服，一次10克，1日2次。

三、胃苏颗粒(冲剂)

【药物组成】香附、陈皮、枳壳、佛手、香橼、鸡内金（制）、紫苏梗、槟榔

【临床应用】

1.本品适用于功能性消化不良，见胃部胀痛，连及两胁，痛处不固定，喜嗳气，嗳气或排气后胀痛减轻，生气或情绪激动时胀痛明显或加重，胸闷不舒，食少，食后不舒，排便不畅，舌淡红苔薄白。

2.若反酸烧心明显者，可合用左金丸。

3.若胃痛甚，或胃部刺痛、舌质瘀斑者，可合用玄胡止痛片或失笑散。

4.消化性溃疡患者，可合用西药质子泵抑制剂。

【注意事项】

1.孕妇慎用。

2.服药期间情绪稳定，切忌恼怒、抑郁。

3.服药期间偶有口干、嘈杂，不影响服药。

4.服药期间，宜选清淡易消化之品。

5.忌食辛辣油腻及刺激性食品，戒烟酒。

6.肝胃郁热或胃阴亏虚患者慎用本品，前者主要表现为胃部灼热胀痛，生气或情绪激动时疼痛明显或加重，嗳气或排气后减轻，口苦口干;后者主要表现为胃部灼热隐痛，似有饥饿感，口渴喜饮，手足心热，心烦，或口腔溃疡易发，舌红少津。

【用法用量】口服，一次15克，1日3次，15天一疗程，可服1-3个疗程。

四、金佛止痛丸

【药物组成】白芍、延胡索、三七、郁金、佛手、姜黄、甘草

【临床应用】

  1. .本品适用于功能性消化不良、消化道溃疡，慢性胃炎，见胃部胀痛或痛如针刺，痛无定处，连及两胁，拒按，生气或情绪激动时胀痛加重，喜嗳气，嗳气或排气后胀痛减轻，舌苔薄白的患者。
  2. 若反酸烧心明显者，可合用左金丸。3.若情绪不住诱发者，可合用道道散
  3. 若胃部冷痛，得温痛减者，可合用理中丸或虚寒胃痛颗粒。
  4. 消化性溃疡患者，可合用西药质子泵抑制剂。

【注意事项】

  1. 孕妇及妇女月经过多者禁用。
  2. 胃阴虚者表现为唇燥口干、喜饮、大便干结，不宜服用本品。
  3. 不宜在服药期间同时服用滋补性中药。
  4. 忌食辛辣、生冷、油腻食物。

【用法用量】 口服，一次5-10克，1日2-3次。

五、中满分消丸

【药物组成】厚朴(姜炙)、枳实、姜黄、黄芩、黄连、半夏(制)、知母、猪苓、茯苓、白 术(麸炒)、泽泻、陈皮、砂仁、党参、甘草

【临床应用】

  1. 本品适用于功能性消化不良，见脘腹胀痛灼热，烦热，口干口苦，口渴不喜饮，食少腹胀，纳差气短，小便短色黄，大便秘结或稀而不爽，臭秽，舌边尖红，苔黄腻。
  2. 对于食欲不佳，纳差乏力，口苦口渴不喜饮，大便黏滞不爽较重者，选用本品疗效突出。

【注意事项】

  1. 本品祛湿热为主，不宜久用。
  2. 不宜在服药期间同时服用滋补性中药。
  3. 本药内含破气活血之品，孕妇、儿童慎用。
  4. 本品为中满热胀、二便不利而设，若寒湿困脾者不宜使用。
  5. 服药期间饮食宜用清淡易消化之品，慎食辛辣肥腻之物。

【用法用量】口服，一次6克，1日2次。

六、3附子理中丸

【药物组成】附子(制)、党参、白术(炒)、干姜、甘草

【临床应用】

  1. 本品适用于功能性消化不良，见胃部冷痛隐隐，喜温喜按，遇寒后疼痛明显或加重或泄泻，呕吐物清冷，身冷喜暖，四肢不温，大便稀或泻下未消化的食物，口淡不渴，面色白，舌淡的患者。
  2. 若黎明前肠鸣欲泄者，可合用四神丸。
  3. 若腹部疼痛较重者，可合用虚寒胃痛颗粒、气滞胃痛颗粒。
  4. 若呕吐物为清水、肠鸣频繁、小便不畅者，可合用五苓散。

【注意事项】

  1. . 感冒发热病人不宜服用。
  2. 高血压、心脏病、肝病、糖尿病、肾病等慢性病严重者应慎用。
  3. 孕妇、哺乳期妇女、儿童慎用。
  4. 不宜过量或长期服用。
  5. 对本品过敏者禁用，过敏体质者慎用。
  6. 不宜在服药期间同时服用性味苦寒的中药，
  7. . 服药期间宜食用易消化食物，忌生冷油腻之品。

【用法用量】口服，水蜜丸:一次6克;大蜜丸:一次1丸，1日2-3次。

七、香砂养胃丸(颗粒)

【药物组成】白术、木香、砂仁、豆蔻(去壳)、广霍香、陈皮、姜厚朴、醋香附、茯苓、枳实(炒)、姜半夏、甘草

【临床应用】本品适用于功能性消化不良，见胃部胀闷不舒甚或隐隐作痛，呕吐酸水，胃部烧灼感，食欲减退、四肢倦怠。舌胖大舌边有齿痕，苔薄腻的患者。

【注意事项】I

  1. 如果出现胃部灼热，隐隐作痛，口千舌燥等热症表现的患者不宜服用本药。
  2. 不宜长期连续服用本药。
  3. 忌生冷油腻食物。
  4. 本品宜用温开水送服。
  5. 对本品过敏者禁用，过敏体质者慎用。

【用法用量】丸剂:口服，大蜜丸一次9克，1日2次，浓缩丸一次8丸，1日3次，水丸一次9克，1日2次;颗粒剂:开水冲服，一次1袋，1日2次。

八、六味木香胶囊(散)

【药物组成】木香、豆蔻、荜茇、石榴皮、闹羊花、栀子

【临床应用】

  1. 本品适用于功能性消化不良，见胃部堵闷胀满疼痛，或腹胀腹痛，痛无定处，生气或情绪激动时胀痛明显或加重，喜嗳气，恶心欲吐，嗳气或排气后胀痛减轻，情绪郁闷，舌边尖红的患者。
  2. 本品用于中焦气滞、寒热错杂之胃痛效果最佳，主要表现为脘腹胀满，痛处不定，食后腹胀，口渴喜饮，或口中异味，大便稀或不成形，或大便中有未消化的食物，舌淡苔薄黄。
  3. 若大便不通或便秘者，可合用六味安消胶囊。
  4. 若大便稀或不成形、舌苔厚者，可合用枫蓼肠胃康颗粒。
  5. 若反酸烧心，胃部似有饥饿感者，可合用健胃愈疡片。

【注意事项】

  1. 按照规定的用法用量服用，不宜长期服用。
  2. 本品药性偏于温燥，胃阴不足或湿热中阻所致痞满、胃痛者慎用。
  3. 孕妇禁用，儿童、体虚者慎用。
  4. 本品含闹羊花(有毒)，不宜长期或过量服用。一旦出现中毒表现，可按绿豆解毒。
  5. 宜少量多餐，饮食清淡，忌食生冷油腻、辛辣刺激及不易消化食物。

【用法用量】 胶囊剂:口服，一次4-6粒， I日1-2次; 散剂:口服，一次2-3克，1日1-2次。

九、荆花胃康胶丸

【药物组成】土荆芥、水团花

【临床应用】

  1. 本品适用于功能性消化不良，见胃部胀满疼痛，嗳气或排气后胀痛减轻，能食，食后不舒，反酸嗳气，胃中似有饥饿感，口苦，口渴喜热饮，舌淡红苔薄黄的患者。
  2. 本品对于上述症状兼有形体肥胖，胸膈堵闷，常咳嗽咳痰，痰多色白者疗效较佳。
  3. 若症见口干口苦、胁胀胁痛而兼气急易怒，口渴喜凉饮者，可合并应用丹栀逍遥散。
  4. 抗幽门螺旋杆菌治疗时，联合应用荆花胃康胶丸可增强杀菌作用。

【注意事项】

  1. 过敏体质及对本品过敏者不宜服用。
  2. 孕妇忌服。
  3. 饮食宜清淡，忌烟、酒及辛辣、生冷、油腻食物。
  4. 忌情绪激动及生闷气。

【用法用量】口服，一次2粒，1日3次，4周为一疗程。

按症状分类使用:

  1. 上腹胀痛伴平时纳差，面色发白往往用香砂积术丸;
  2. 以上腹部冷胀、痛为主，喜暖、喜按者用附子理中丸;
  3. 上腹胀或痛，伴舌苔黄厚腻，大便腐臭用中满分消丸;
  4. 胃痛隐隐、呕吐酸水用香砂养胃丸;
  5. 上腹胀痛、吞酸嘈杂往往用六味木香颗粒;
