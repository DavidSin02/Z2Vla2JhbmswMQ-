2019 WCLC | 真实世界研究：这三类突变或可预测一二线免疫治疗和靶向治疗的较差获益

2020-10-29
| 科学

由国际肺癌研究协会(IASLC)主办的第20届世界肺癌大会（World Conference on Lung Cancer，WCLC）于2019年9月7-10日在西班牙巴塞罗那召开。口头报告中有两项真实世界研究探索了某些突变与免疫治疗或化疗的应答相关性。详情如下：

LKB1 / STK11突变和LKB1 / KRAS突变与一线免疫治疗和化疗的较差OS和PFS相关

背景

虽然今年在转移性非小细胞肺癌（NSCLC）方面取得了进展，但很多患者对相关治疗不应答。识别和鉴定遗传性标记可以使患者的临床获益最大化，并避免不必要的毒性。LKB1 / STK11突变（LKB1m）和共同发生的KRAS突变/ LKB1缺失（KRAS / LKB1）与免疫治疗的预后不良相关。这项回顾性研究评估了真实世界中LKB1m、KRAS / LKB1突变NSCLC患者接受免疫单药或联合治疗的结果。

方法

研究纳入2013年1月至2017年6月期间开始一线治疗的成人患者，在常规治疗中使用FoundationOne分析数据。根据组织学分型利用多变量Cox比例风险模型评估LKB1m、LKB1 / KRAS与一线和二线治疗总生存（OS）或无进展生存（PFS）的相关性。分层因素为免疫治疗或化疗。

结果

研究纳入2407例患者，其中1847例非鳞状NSCLC患者，开始一线治疗时的中位年龄为66.1岁。328例（13.6％）患者存在LKB1m，157例（6.5％）存在KRAS / LKB1。在接受二线免疫治疗的患者中，与野生型患者相比，LKB1m患者的OS和PFS较差。在一线治疗和非鳞状NSCLC患者中也观察到类似的相关性。在接受化疗的患者中，LKB1m仅与一线治疗时OS结果较差相关。与LKB1m患者相比，KRAS / LKB1突变患者这种相关性更明显。

结论

不论是免疫治疗还是化疗，LKB1m和LKB1 / KRAS与一线治疗的OS和PFS较差相关，并与二线免疫治疗的OS和PFS较差相关。这项真实世界研究的结果支持之前的临床研究结果，并提出了这些突变在一线化疗和一线或二线免疫治疗的独特相关性。

EGFR/HER2 20外显子插入突变对免疫治疗和EGFR TKI应答较差

背景

EGFR/HER2的20外显子框内插入突变（Ex20ins）发生在2%~5％的NSCLC患者中。对于这些突变的患者，目前没有批准的靶向治疗药物。历史对照数据对于开发针对这些罕见靶点的靶向疗法有一定价值。

方法

日本已经建立了全国基因组筛选项目（LC-SCRUM-Japan），用来开发肺癌的靶向治疗。在该项目中，通过第二代测序技术（NGS）分析161种癌症相关基因。使用该项目中的大规模临床基因组数据库评估EGFR / HER2 Ex20in的非鳞状NSCLC患者的疗效和生存率。

结果

研究共分析了3441例晚期非鳞状NSCLC患者的数据。在73例患者中检测到EGFR Ex20ins（2％; M766_A767insASV / A767_S768insSVD / H773_V774insH / D770_N771insNPH / N771_P772insPH /其他= 22/17/5/4/4/21）。在128例患者中检测到HER2 Ex20ins（4％; A775_G776insYVMA / G776delinsVC / P780_Y781insGSP /其他= 95/16/10/7）。患者的中位年龄为62岁。81例（40％）患者为男性，114例（57％）患者从未吸烟。200例（99％）患者被诊断为腺癌，1例是腺样鳞状细胞癌。

基于数据库的数据，EGFR Ex20ins患者的中位OS为22.4（95％CI，15.3~36.8）个月，而HER2 Ex20ins患者的中位OS为18.8（13.6~30.3）个月。在EGFR / HER2 Ex20ins患者中，一线含铂化疗的客观缓解率（ORR）和中位PFS分别为32％和6.0（5.7~7.0）个月。多西他赛联合或不联合雷莫芦单抗的ORR和中位PFS分别为26％和5.1（3.8~5.9）个月。PD-1抑制剂的ORR和中位PFS分别为0％和2.0（1.6~2.6）个月。在EGFR Ex20ins和HER2 Ex20ins患者中未观察到这些药物的疗效的显著差异。在用第一代/第二代EGFR-TKI治疗的19例EGFR Ex20患者中，ORR为5％（M766_A767insASV阳性肿瘤对阿法替尼应答），中位PFS为2.1个月（1.3~4.2）。

结论

EGFR/HER2 Ex20ins阳性NSCLC对PD-1抑制剂和第一代、第二代EGFR TKI显示出较差的应答。这些数据对于评估用于EGFR / HER2 Ex20ins阳性NSCLC的新型靶向疗法的疗效方面具有重要价值。
