2019年大家的愿望清单，你中了几条？

2020-10-29
| 科学



每到年底和新年的开始，大家总要总结一下上一年，看看自己收获了多少，又有哪些未完成的愿望需要在下一年完成的，说来好笑，但是这个确实是很有必要的，没有小目标，你生活怎么才能有动力呢？我们一起来看看大家愿望清单吧！



大众的愿望清单

1. 减肥，保持身材。

2. 锻炼身体，保持健康。

3. 努力工作，涨工资。

4. 找到另一半，早日步入婚姻的殿堂。

5. 生个胖娃娃/抱上大孙子。

6. 出国旅游一次。



女人的愿望清单

1. 自己越来越漂亮，越来越有魅力。

2. 拥有魔鬼的身材，天使的面孔，林志玲的声音。

3. 早日找到男朋友，最好有房、有车、人帅、疼人、工资高。

4. 老公体贴，工作上进。宝宝可爱，成绩优秀。婆婆大气，有共同话语。

5. 买的起漂亮的衣服，用得起高级的化妆品，逛街最好不看价钱。

6. 吃好多好吃的，还能不长肉。熬夜看剧，还长不黑眼圈，穿高跟鞋走路，还不累脚。



男人的愿望清单

1. 自己越来越帅，越来越有魅力。

2. 啤酒肚下去，练出肌肉。

3. 早日找到女友，不用太好看，看着顺眼就行，最好善解人意，身材好。

4. 赚更多的钱，结婚，买房，买车，买奶粉。

5. 大家都尊敬我，说话更有分量。

6. 爸爸妈妈都健健康康的，老婆幸福贤惠，孩子开心懂事。



爸爸、妈妈的愿望清单

1. 孩子可以常回家看看,多往家里打电话。

2. 孩子可以多赚点钱。

3. 可以早点抱上孙子孙女。

4. 儿孙满堂。

5. 身体健康。

6.孩子能感受到父母的不易，早点懂事，让我们省心。



狗狗的愿望清单

1.主人能多陪我。

2.多些肉骨头。



打工人的愿望清单

1. 工资越来越高。

2. 少加班，加班的话能多给加班费。

3. 公司福利多一些。

4. 老板大气不苛刻。



老板的愿望清单

1. 公司效益能越来越好。

2. 员工越来越努力，最好还不提涨工资。

3. 甲方越来越善解人意，不挑剔。

4. 成本越来越低，利润越来越高。



单身狗的愿望清单

1. 找到另一半。

2. 最好可以早点结婚。

3. 结婚后早点生娃。



情侣（夫妻）的愿望清单

1.对方可以爱我更多一点。

2.我们在一起时多创造些美好的回忆，永远不分离。

3.彼此多谅解。

4.不吵架，不冷战，有问题，多沟通。



男神、女神的愿望清单

1. 多读几本书，提高自己的智商、情商。

2. 学习英语，能和外国人顺畅的交流。

3. 学习舞蹈，提高气质。

4. 学会穿搭，做个精致的人。

5. 学会做菜，善待自己的胃。

6. 学习书法，让自己字更美丽。

7. 学会化妆，妆容越自然越好。



学生的愿望清单

1. 少留点作业。

2. 取得优异的成绩，早点毕业。

3. 老师不拖堂。

4. 换一个更温柔更美丽的老师。

5. 爸妈可以不催着我上辅导班。

6. 和喜欢的人坐同桌。



老师的愿望清单

1. 学生都听话，少惹事。

2. 多教出几个优秀的学生。

3. 同学们都能认真听讲，上课的时候不讲话，不调皮。

4. 我讲的课大家都爱听。



房东愿望清单

1. 每个月及时收到房租，不用去催钱。

2. 住房的人爱干净，不养狗，素质高。

设计师的愿望清单

1. 甲方大气豪爽不磨磨唧唧。

2. 少改几次设计稿，早点下班回家吃饭。

3.设计费给的及时，不用一直上门催。



甲方愿望清单

1. 乙方善解人意，做事周到。

2. 乙方审美好，最好和领导想法一致。



环卫工人的愿望清单

1． 大街上，大家可以少扔些垃圾。

2． 最好不要在大街上嗑瓜子。

3． 大风大雨天可以少一些。



每个人，所处的位置不同，所求亦不同，但是大家的方向都是一致的，就是希望自己可以越来越好，新的一年，也祝大家定好自己的目标，向着这个方向去努力，早日达成所愿，心想事成！
