纳米微晶纤维素单宁酸薄膜

本文分析了热处理对壳聚糖纳米复合膜的影响。使用对流烘箱对一系列样品进行热处理，所述样品包括:

纯壳聚糖膜、包埋有纳米微晶纤维素的壳聚糖膜、用单宁酸交联的壳聚糖膜和用纳米微晶纤维素和单宁酸的混合物交联的壳聚糖膜。傅里叶变换红外光谱和 x 光衍射测试显示了热处理薄膜化学相互作用的变化。热处理后的薄膜在吸湿方面有显著的改善。当样品进行热处理时，拉伸强度和杨氏模量分别增加到 7兆帕和 259 兆帕。对于纳米纤维素颗粒，采用透射电子显微镜对悬浮态纤维素颗粒的结构性能进行了检测。

热学性能

假设壳聚糖膜的热处理不会增加膜的结晶度。由于热处理过程中强烈的交联相互作用，结晶度可能会降低。

机械性能.

热处理对壳聚糖膜机械性能的影响如图 4 所示。一般来说，当热处理时，样品 C、CX 和 CN 的拉伸强度和杨氏模量提高，而样品 CNX 经历了轻微的降低(图 4a 和 4c )。样品 C 具有最低的拉伸强度( 36.93 兆帕)、杨氏模量( 721.61 兆帕)和最高的断裂伸长率百分年龄( 43.1 % )。单宁酸和 NCC 的加入增加了所有样品的拉伸强度和杨氏模量，但以断裂伸长率为代价。力学性能改善最大的是拉伸强度为 48.32 兆帕( P < 0.05 )的氯化萘样品，其次是热处理后的氯化萘样品( P < 0.05 )。与纯壳聚糖膜相比，热处理样品氯化萘的杨氏模量最高，为 1080.14 兆帕( P < 0.05 )，未处理样品氯化萘最高，为 1256.48 兆帕( P < 0.05 )。从断裂伸长率观察到的趋势(图 4b )显示，当引入单宁酸时，样品 C 和 CX 的含量大幅降低(所有

结论：

壳聚糖的机械性能和阻隔性能可以通过添加单宁酸、NCC 和热处理工艺来改变。傅里叶变换红外光谱表明热处理引起了壳聚糖膜要改。热处理的 XRD 衍射图显示在 11.4♀处峰变平，在 2θ = 19◦ 到22◦.。热处理后的样品显示出机械强度的显著提高和含水量的降低。这是由于加速交联压缩了结构。这些改进表明，热处理壳聚糖对骨形态发生器翼膜有很大的潜力。
