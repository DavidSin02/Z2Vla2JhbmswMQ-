吃遍魔都五大老牌便利店，才敢告诉你什么值得买！

2020-11-14
| 科学

当小资君每次不停搜刮“萝”森、全“佳”

“seven”-11、喜“事”多，魔都四大便利店时...

内心就会听见四大便利店们的呐喊：“感觉被掏空！”

当妈妈在烧饭时说：

“屋里厢酱油没了，到对面快客帮我买1包回来！”

当每次便利店推文一发送，粉丝的留言里总会有

“怎么没有好德/可的/快客..."等诸如此类的评论

小资君：是呀，什么时候我们只顾着去

那些自带日系风光环，拥有无敌可爱包装

以治愈孤独都市人的新24小时便利店

却忘记了那些老牌便利店啊

其实它们才是和我们一起长大的24小时便利店

只是，那时的我们还不懂

那么，今天就和小资君一起

会会昔日的5位老朋友吧~（惭愧脸）

快客，可的，好德，良友，家乐福easy

不仔细逛不知道

其实好吃的也很多哦

绝对和新便利们有得一拼

亲民的美食list，有点长长长长长

但，还是迫不得待地想安利给小伙伴们啊~

/ 快客 /

加分小亮点：自营小零食系列，面包甜品

因为是百联集团旗下的，店内各种日本、韩国、泰国的进口零食，饮料也不少哦~但令小资君惊讶的有以下两点~

不是每家店都有共享按摩椅的哦！

①：店内居然还有共享按摩椅，想象着一边吃吃零食，一边花点小钱按摩一下，感觉这样的治愈感更实际哇~（毕竟身体能感受得到~）

②：快客的自营零食系列，完全不输全“佳”的呀米将、萝”森和“seven”-11的自营系列零食，铜板的价格也是喜人，近三四十种商品看得小资君统统都想带回家呀！

小Q巧克力威化/2.5元

脆脆威化里夹着奶油般梦幻感的巧克力，清甜不腻~咖滋咖滋的美味根本停不下嘴！外加每包只要2.5元，1包里有10片，幸福感更是抑制不住的upupup~

小Q芝麻多多、葱多多/3.8元

葱多多简直就是椒盐版的桃酥饼，咸香可口~芝麻多多是甜味的，芝麻味很浓郁啊~两款都是酥酥的口感，油腻感不强，老少皆宜~

小Q原味米果 4.5元

小资君的大爱，原味米果里有柿种和花生，55g的分量真的是很感人，香香脆脆的一包够小资君吃3天了~5块钱就可以拥有啦~一般店家都要10元左右了，关键份量也很“羞涩”啊~

小Q香辣小黄鱼/5.5元

鲜辣入味，鱼肉吃起来并不是很干，带着丝丝嚼劲，会让人越吃越有味，如果再来一杯啤酒，就更惬意了~仿佛就像是回到了1988年的双门洞，妈妈们啃着鱼干的小日子里去了~

小Q手工蛋糕片/4.5元

简直就和越南鸡蛋饼干的味道如出一辙~蛋香四溢，酥脆的不要不要的~喜欢哦~

小Q沙琪玛/4.5元

老少皆宜的沙琪玛，小包包里的垫饥小零食必定有它~快客自营的沙琪玛蛋香味很足，甜酥酥哒，不粘牙~

香蕉奶油面包/7.5元

说实话小资君被香蕉味的延伸想象力惊到了！小伙们一定要治愈一下，松软如欧包的外衣包裹着绵绵的香蕉奶油，这个味道简直就是的面包版的香、蕉、先、生！

草莓慕斯蛋糕/10.8元

甜品包装普通，但味道其实一点也不亚于魔都四大便利店，巧克力慕斯、布朗尼都很浓郁哒！但是夏天么，还是推荐一款清爽的草莓慕斯，奶油轻盈不腻，层次感也很强~当办公室下午茶甜品真是绰绰有余啦~

燕麦牛肉三明治/14.8元

枕头形三明治里夹着蔬菜和牛肉，在其他便利店基本很少见吧~蓬松的口感间带着实实在在的满足感~配上燕麦粒，让咀嚼都充满着元气~

/ 良友 /

加分小亮点：老牌新品雪糕、阿波罗冷饮系列，1~4元小甜点

因为和光明合并了，所有一些光明牌新出的冷饮都可以在这里找到，冷柜的雪糕价格区间很大，两三块的都有（毕竟在四大便利店里，雪糕价格都是5元起的，钱包供不起啊~），如果是光明控的小伙伴不妨可以到这里来看看的~

光明熊小白芝士雪糕/8元

第一口就喜欢上的味道，是鲜牛乳做的，虽然容易化，但口感特别醇厚，带着微微的咸，舔上去就像在吃奶盖，也会有幸福的白胡子~把它泡在冷萃里也非常赞哦~

阿波罗雪泥球/7.5元

小资君从小吃到大的，这颗橙色球出现在老牌便利店的概率很高的，橙子味很清甜，挖着吃冰冰爽爽的~就像在吃雪芭~

阿波罗杨枝甘露/6元

因为阿波罗这个牌子主要走的是香港风味，杨枝甘露味的雪糕是一定要舔一根的~清爽味甜，柚子味很清新~化在嘴里就是杨枝甘露！

悦家3D健钙芒果酸奶/3元

包装很有童年感，8090小时候基本都吸过这样的酸奶，只不过价格从原来的1元变到如今3元，价格依旧亲民，味道也没变~酸甜顺滑，没有你要的醇厚感，却有着回忆感~

蝴蝶酥（1元/块）

娜蕾多（2元/块）

蜜蕾方块酥（3.5元/块）

放在收银台前的老式小点心，一包一个，特别可爱~价格在1~4元间不等。蝴蝶酥的黄油味很浓，蜜蕾方块酥就像杏仁排，虽然比不上哈尔滨食品厂，但味道也是可圈可点的~

/ 可的 /

加分小亮点：新上市的国产饮料，阿波罗冷饮系列、八喜甜筒

一般都开在老式居民区附近，以服务周边居民日常需求为主，虽然店面普遍都不大，但日用品，油盐酱醋，大米精制油、酱菜萝卜之类的，总是占有一席之地~为了与时俱进，店内也有盒饭，饭团，甜点之类食物~味道也是可以的~

八喜香草甜筒/6.5元

八喜的雪糕杯见得很多，甜筒了解一下吧~也是一款时常出没在老牌便利店里的冷饮~待微微融化后，顶部的冰激凌就像是奶油蛋糕上的鲜奶油一样轻盈~没如“卡哇伊”多甜筒一样有很多巧克力碎之类的装饰~香草的味道很纯很干净~同款巧克力味甜筒也是如此~

雀巢摩爵扁桃仁香草雪糕/9.2元

一款比星星雨更有满足感的脆皮巧克力雪糕，外加雀巢牛奶品质就不错，奶味很香醇~一口咬下即是脆皮的“咖滋”声，听起来特别性感( ω )~巧克力味浓郁，却没有甜腻感，赞~真心觉得比“某龙”好吃多了~

阿力达百香果棒冰/4元

咬上去就像在吃绵绵雪，百香果的酸甜气息存在感非常强，时不时就会吃到调皮的小黑籽~

喜盈门香蕉牛奶/5元（优惠价）

几乎和韩国某香蕉牛奶味道一模一样~做作的香精味相对还少了一些，香蕉味喝起来很温柔~关键只卖5元/瓶哇~价格是韩国某香蕉牛奶的一半哇~p.s:同品牌的草莓牛奶，也不错哦~

乳小菌白桃味乳酸菌/6.8元

包装很可爱(*▽*)，在老牌便利店也算是少见了吧~味道和小时候喝的乐百氏很像，清清爽爽的酸甜，混合著白桃的味道，很夏天~

BISCOLATA巧克力/5元

像是巧克力软饼或是蛋糕~巧克力一吃就知道品质不错，甜而不腻，不会黏在舌腔上，结果一看成分表果然如此，32%的黑巧克力，不含代可可脂哦~

芒果菠萝班戟9.5/元

可以和街头的港式小店媲美的，蛋皮很薄，奶油绵柔轻盈，配上内馅实实在在的芒果、菠萝果肉，香甜又清爽~

/ 好德 /

加分小亮点：阿波罗冷饮系列、八喜甜筒

好德和可的风格没差别，都是农工商旗下的24小时便利店，精打细算的阿姨妈妈们是普遍消费人群，所以店内基本都是一些乐事、好丽友、上好佳、天喔...之类的零食比较多，偶然会有一些不常见的牌子会出现，味道其实不差的，但总莫名感觉有些山寨（小资君：对不起，我不应该以貌取物~）

卡士达布丁蛋糕/6.2元

包装很少女心，味道也很少女心~外层是松软的面包“羽绒服”，内层的底部是绵柔的蛋糕胚，顶部是一坨可爱的卡仕达奶油~充满着甜蜜的层次感~

欣欣长崎牛奶蛋糕/4.9元

味道很日本某牌的蛋糕也很像啊，绵软的蛋糕胚里带着丝丝蜂蜜般的香甜~一包里有2个独立的小包装~方便携带~

桃李雪之语/4.2元

桃李的面包一向都是很柔软~内馅夹着草莓酱和奶油，每一口都充满着蓬松的清甜感~

名仁玫瑰苏打/4元

带着丝丝乡土气息包装的苏打饮料，苏打的气泡感完全不存在，但这反倒是成就了玫瑰味的清雅微甜~透明色的饮料水，如果换个日式包装的话，透明饮料界的C位之争，应该也有它的一席之地吧！（小资君：国货当自强啊~）

优得宇治抹茶爆米花/8.5元

胜在味道~在各种原味，焦糖味的爆米花世界里，终于有抹茶控的一片天空了~“大风吹~大风吹~抹茶爆米花好美~”

/ 家乐福easy /

加分小亮点：进口零食/饮品/调味品，三明治等简餐

其实家乐福easy不算是老牌，但大卖场般的家乐福绝对是老牌了，只是小时候爱和妈妈逛家乐福的小资君，早已投奔便利店的怀抱了！

同时在B2C，O2O的购物方式的激烈竞争下，家乐福君也在不停地寻求着突破吧，easy算是其中一个改变吧~品类很全，就像一间迷你版的家乐福大卖场，基本可以秒杀掉其他便利店！

夏威夷菠萝火腿卷/16.8元

“明天我还要去买~”比起新四大便利店售卖的各种瘪瘪的卷饼，家乐福的料简直就是plus级别的，卷皮也很紧实Q弹啊，咬下的每一口都非常的有满足感~冷热即可食用~

不过小资君更喜欢吃热的，在平底锅里稍稍烘一下，卷皮会格外香脆~再配个好看的盘子，可以PKwagas家的各种卷饼~XXXXD

尼斯轻食三明治/6.5元

小资君编辑部小伙伴推荐的，吐司很松软，内馅的料也是足足的，吃起来很清爽。降解材质的包装盒也是小资君喜欢它的原因之一，方便携带，不会像新四大便利店的三明治，稍稍一压就变成了一块“饼”-_-||

烤肠/3.5元

应该算是店家的人气小吃了，小时候的台湾香肠味，一口咬下去油润润的，满是肉香味，让小资君恨不得往嘴里再放几粒蒜，那样就口口爆香啦~完全不像四大新便利店的烤肠，带着烟熏的气息~

清美绿豆汤/3.8元

夏天必备的绿豆汤~清美老牌子了，有信任感~汤汁凉爽清甜，绿豆的量也很多，酥酥润润好是饱满呢~另外比心一下包装，比起四大新便利店碗形的汤/粥包装，这里的杯形包装更容易食用，一根吸管就够啦~

劳仑兹番茄味笑脸的薯片/16.5元

颜值高又好吃的薯片，小资君今年追剧的指定薯片就是它啦~可爱的笑脸简直就是薯片界的“鬼脸嘟嘟”让人爱不释手，口感带着松脆的空气感（一般薯片君做不到的），甜甜的番茄味也完全渗透在其每个“小细胞”里了~

/ 魔都老牌便利店vs魔都四大便利店 /

魔都老牌便利店

店员基本都是阿姨店员，

眼神里带着”看破红尘“之感

不说话的样子比较冷漠，有时说活又比较凶

偶尔会和附近居民嘎嘎三湖，这时比较亲切

不太会主动给收银条，怕给自己找麻烦

店内油盐酱醋味、洗衣粉香皂味很重，因为真的很多

夏天售卖的饮品主要是各种矿泉水，盐汽水，酸梅汁

冷柜里的乳制品也基本以光明、伊利，蒙牛为主

...

魔都四大便利店

店员基本都是小年轻

眼神里带着对生活的满腔热情，

态度基本很有礼貌

很会“做生意”，什么2件八折，

送500积分，＋3元得酸奶之类的，可有一套了

会很主动给收银条，但小年轻顾客们基本都不要

一进店就是鲜鲜的关东煮味和香甜的冰激凌味

夏天售卖的饮品是各种网红汽水，咖啡，茶饮

冷柜里的乳制品，品牌多到有些都没听过

连口味也是千变万化的~

最后TIME：)

小资君强行给自己“加戏”

在好吃的list中推荐几款必尝款

刷过路过，真的不要错过哟~
