近6亿中国人失眠，昨晚你也在数绵羊吗？

2020-11-14
| 科学


今天又是一个被闹钟叫起床的周二，下班后的你在干什么？是在不懈追剧还是在刻苦学习？或者是在结束了一天的工作之后早早入睡？

但是随着城市发展的速度越来越迅猛社会竞争和人们的压力越来越大早早地入睡和睡个安稳觉
却更像是一种奢望
入睡困难和良好的睡眠质量已成为人群中最主要的失眠表现而这些失眠人的孤单无助与痛楚只有凌晨的夜最清楚

早睡变成了一种奢侈失眠更是家常便饭每天晚上抱着手机在床上辗转反侧他们为什么不睡因为他们根本睡不着
01准时下班是什么感觉？

@朵朵梓


每天一醒来就是满满的工作项目，从早上9点到晚上12点，都是常有的事。晚上下班晚，睡觉自然也晚：晚上到家随便收拾收拾，就已经一两点了。
自从进入这家公司以来，几乎就没有6点下班过，我都快忘了傍晚6、7点的夕阳和晚霞是什么样子了。



02只有晚上才是属于自己的时间@一Jean
作为一个每天定时定点“搬砖”的社畜，只有在下班后躺在床上那一瞬间，才觉得自己是得到了解放，身体和心灵都是真正属于自己的。
这种时候夜晚的时间就显得特别的珍贵，甚至感觉早睡都是对这段时间的不尊重，所以就一直刷手机，越刷越晚......

这是一个“夜猫子”越来越多的时代
晚上睡不着，早上起不来靠着咖啡撑过每一个工作日睡个好觉，对很多人来说是一种奢望
失眠带来的危害

/ 01 会让人迟钝 /


睡眠不足可损害人的注意力、

警觉性、专注性、推理能力

以及解决问题的能力

这些可导致你的学习和工作效率变低


/ 02 记忆力退化 /
失眠还会使人体的记忆力退化导致患者注意力无法集中，影响工作学习等

/ 03 加速皮肤衰老 /
皮肤在晚10～11点进入保养状态长期失眠会导致植物神经功能紊乱，内分泌失调使皮肤干燥，松弛，弹性降低还有熊猫眼等问题

/ 04 引发严重健康问题 /
经常处于
熬夜、疲劳、精神不振的状况可增加你患有这些疾病的风险人体的免疫力会跟着下降感冒、过敏等就会不期而至找上门

如何防止失眠？
/ 01 适当运动 /
运动的确可以改善睡眠状况因为它能调节植物神经使大脑的兴奋受到转移和抑制所以平时一定要加强体育锻炼，增强体质为了保障良好的睡眠质量每天应该适当进行有氧运动

/ 02 请调节心态和情绪 /
睡前以后最好不要回想烦心事有许多人容易失眠，是因为平时事事存在心里防治失眠，需要自己好好调节心态和情绪23点以后最好不要回想烦心事睡前可以听一些舒缓的音乐

/ 03 合理饮食 /
熬夜的人一般会吃“夜宵”这时不但难以消化隔日早晨还会出现食欲不振造成营养不均衡，引起肥胖所以晚餐不能吃太多也不能吃太少还有，睡前万万不能喝咖啡、浓茶让人兴奋的饮料


/ 04 晚上泡脚利于睡眠 /
热水泡脚可加速脚部血液循环水温以40℃左右为宜每晚用温水泡脚10分钟并用手按摩脚以促进血液循环放松人体，缓解疲劳从而产生困倦感，有助于提高睡眠质量

都没有自己的健康重要不如就从现在开始放下手机，放松心情今晚早睡一小时明天继续加油！
