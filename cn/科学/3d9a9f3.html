塞尔达传说 旷野之息 主线任务流程 图文攻略1

第一座塔Follow the Sheikah Slate / シーカーストーンの示す场所

《塞尔达传说 旷野之息》开篇，林克从重生之祠(The Shrine of Resurrection)苏醒。

在遗迹内获得西卡石板(Sheikah Slate / シーカーストーン)。

离开遗迹的路上有两个宝箱，可以获得衣服和裤子。

用西卡石板打开遗迹大门。沿着出门右转的山坡下去，路上听到旁白，告诉我们前往某地，

地点已在西卡石板的地图上标出(按“—”打开地图 )。山坡上会遇到一位烤火的老人，

可以对话，虽然也问不出什么。篝火上烤着苹果，可以拿走，喜欢生苹果可以在旁边树上踩。

山坡途中有一把斧头，拿走吧。老人身后有一个湖泊，湖中的高台上有一把生锈的剑，也拿走。

湖泊旁边有一座破败的建筑—时之神殿。殿内可获得旅人之弓。

神殿旁边有楼梯，爬上去，在钟楼内可发现一把士兵之弓。如果你能爬上钟楼的顶尖

(掌握耐力的使用技巧)，会发现一只小精灵，它会给你一颗种子：コログのミ / Korok Seed。

种子可以在之后用来扩充武器栏，冒险过程中注意探索。

虽然可以去逛的地方还有很多，但我们还是先去主线目的地。来到西卡石板上指示的地方，

用西卡石板开启机关，一座塔从地底升起。

在塔顶，我们会获得这块地区的地图，登记在西卡石板上。属于你的冒险，现在拉开序幕。

【4个试炼之祠】The Isolated Plateau / 闭ざされた台地

这个主线任务需要玩家完成4个试炼之祠的挑战。根据老人的提示，

试炼之祠的位置可以在初始的高塔上用望远镜查看。4个试炼之祠的位置

其中前往 トゥミンケ之词 / Keh Namut Shrine 需要防寒服。

具体获得办法：武器装备防寒服

完成4个试炼之祠的挑战后，前往时之神殿。

(老人提示你相会地点是在四个祠的交汇位置，把四个祠交叉连线，交点就在时之神殿啦。)

对着女神像祈祷，获得一个心之容器或者一段耐力。之后触发剧情，上到时之神殿的塔楼。

剧情后，获得滑翔伞。离开这片高台，开始海拉尔的冒险吧。

向东，往双子山进发。双子山西侧山脚有一座高塔：双子山之塔。

上塔，打开这个区域的地图。要过双子山，建议走山间峡谷的大路，

途中在河边还能发现一个试炼之祠，其中可以获得一顶帽子，效果是加速攀爬，非常实用。

如果想翻过双子山，也是可以的。双子山南北两个山顶上都有试炼之祠可以挑战。

扩展装备栏的精灵

过双子山，我们按照地图指示，顺着大路朝 Impa 的村子进发。

途中会遇到一个胖嘟嘟的【精灵】ボックリン/Hestu，和他对话，完成一个小支线任务，

把附近的一群怪物打败。

之后用【精灵种子】コログのミ/ Korok Seed 可以和它交易，扩展武器槽。

武器槽的扩展，优先推荐：近战武器 > 弓箭 > 盾。

访问Impa

来到 kakariko / カカリコ 村，按照地图指引前往 Impa 的屋子，即可触发剧情。

剧情之后，我们会获得多个主线任务，在后面的主线攻略中将一一叙述。

在 kakariko / カカリコ 村你可以购买服装、食材。在村子的东北方向，有一片精灵之森，

那里有一个可以帮你强化防具的大妖精，不过你得先给她100卢比

村子东北角有一个试炼之祠，完成它吧，之后方便传送。

【跟随塞尔达的脚步】Locked Mementos/ゼルダの道しるべ去往Hateno / ハテノ村

Impa 交给我们的几个主线任务，推荐优先完成 Locked Mementos/ゼルダの道しるべ。

沿着大陆走即可，一直来到 Hateno / ハテノ村，路线如下图。

途中稍微绕一下，去 Hateno / ハテノ塔，把这片区域的地图打开。

心之容器、耐力的互换

在 Hateno 村里有一尊恶魔石像，可以自去找，也可以让村口的孩子带你去，位置如下。

和石像对话，可以献出一颗心之容器或者一段耐力，获得100卢比。然后你可以花120卢比，

从石像买一颗心或者一段耐力。也就是可以花20卢比，用心和耐力进行置换。

拥有自的房子

Hateno 村里还有一间旧屋正在被拆(就在村中试炼之祠的旁边)，

和拆迁队老板对话，可以选择把这间屋子买下来，价钱：3000卢比，外加30捆柴火。

古代研究所

Hateno 村外，沿着东南方向的山坡一路向上，来到古代研究所(Hateno Ancient Tech Lab)。

和这里的大叔研究员 Symin、萝莉 Purah 对话，开启“传火任务”。

研究所内，门旁边就有一根火把。按照地图指示跑去下图位置引一团蓝色火焰，

然后沿路传火，一直来到研究所外，点燃研究所外的能量炉即可。

为以防万一，把沿途的灯台点燃，可以预防中途意外，即便传火失败也可以从灯台继续，

不用跑回火源。

没火把的时候，用拖把也是可以代替的

点燃蓝色火焰后，古代研究所成为可传送点。

之后和萝莉 Purah 对话，西卡石板追加拍照、相册、图鉴功能。

Purah 会要求玩家给她拍一张照，按L键选择相机图示，给她拍一张，再和她对话即可。

之后在旅途中若是找到古代素材(古代螺丝、古代轴承、古代核心)，

可以回到这里找 Purah 强化西卡石板的功能。

【寻找妖精之泉】Find the Fairy Fountain / 妖精の泉探し

这个主线任务比较简单，算是个插曲。

在 Hateno / ハテノ村 解锁西卡石板的拍照功能后，回到 kakariko / カカリコ村。

和村中女神像边的老人 Pikango/カンギス 对话，他会要求玩家陪他去找妖精之泉。

妖精之泉的位置，我们第一次来 kakariko / カカリコ村 时就找到啦。

陪着老人一路上山去妖精之泉，走到一半老人就走不动，只好委托我们去拍个妖精之泉的照片，

回来给他看。喏，就是这样的位置，拍个照即可。

【前往左拉之乡】Reach Zora’s Domain / ゾーラの里に辿り着け

四神兽的能力分别是：【水】原地复活、【风】平地起风(飞)、【火】防御反射、

【雷】范围雷暴暴。出于保证生存、提高探索效率的考虑，

推荐按照【水→风→火→雷】的顺序去推进这部分主线。

但《塞尔达传说 荒野之息》的开放程度非常之高，你可以根据自喜欢的玩法，选择主线顺序。

好的，我们继续。这是前往水之神兽挑战的前置主线任务。来到地图的东部，位置如下：

先去把这个区域的塔点亮【Lanayru Tower / ラネールの塔】，打开这个区域的地图。

在塔上就能遇到一个左拉族，他会指引你到塔下，在大桥边见到左拉族王子 Sidon / シド。

根据王子的委托，沿着山路一直向索拉之乡进发。

路上会遇到一些蜥蜴弓箭兵，射的还是闪电箭，注意利用场景内的掩体消灭它们。

注意不要踩到水坑里——因为闪电箭射到水里，我们也会被电击……

电系武器对这些敌人有奇效，所以可以捡敌人射空的闪电箭，回射它们。

当然，你也可以迂回绕过这些敌人。

最后来到左拉一族的宫殿，在二楼觐见左拉之王。接下来就要准备去挑战水之神兽。

【水之神兽】Divine Beast Vah Ruta / 水の神獣 ヴァルッタ

左拉之铠和左拉之王对话后，获得装备“左拉之铠”【Zora Armor / ゾーラの铠】，

效果是：游泳加速、在瀑布前按(A)可以直接攀上瀑布。

穿上左拉之铠，按照地图上的标示来到宫殿外和左拉老者【ムズリ/Muzu】对话，触发剧情。

智取雷兽山

按照剧情，你需要收集20支闪电箭。

如果你这时候已经有，那跑一趟雷兽山 【PLOYMUS MOUNTAIN / 雷獣山】 ，

然后从上面直接跳到下一个剧情点即可。如果身上闪电箭的数量不足20，

那就只能去雷兽山收集。雷兽山上有一只半人马型怪物，会射闪电箭，很强，

实力不够别去挑战。而山上的树干上已经射有一些闪电箭，偷偷靠近，拿走即可。

如果闪电箭数量还不够，你就得吸引半人马射箭，躲在树干后面“借箭”，

或者来回跑，捡地上射空的箭。当然，这只半人马也是可以打败的，只要技术够。

打法建议：近身锁定，正对半人马，这样比较方便观察它的攻击，触发林克时间;

如此反复，半人马在HP一半以下时会开始喷火球三连发，弧形移动绕到它身后即可输出。

当然，使用增加防御力/攻击力的料理药物、用左拉之乡里卖的冰霜箭，都可以帮助战斗。

打败半人马可以获得一套不错的装备。收集够20支闪电箭后，

从山上直接跳下，到下一个剧情点。

水之神兽攻略战

这段剧情战斗，需要林克骑在左拉王子身上，靠近水之神兽。

穿上左拉之铠，借助水之神兽身旁喷出的四道瀑布飞流之上(靠近瀑布时按A)，

腾空后在空中开弓，用闪电箭射击红色放光的关键部位，如此重复四次。

在这个过程中，水之神兽会放出冰块追踪袭击我们，此时建议开启结冰能力，

在冰块飞行过程中直接击碎它们(对着冰块发动能力即可，就是变相的射击关卡)。

成功射完四个部位后，进入水之神兽内部。

水之神兽内部解谜

首先，根据提示找到水之神兽内的地图。沿途遇到一堆黑色物体，

用弓箭射爆上面的眼珠即可消除它们。进入大厅后，可以看到水池对面有一扇闸门，

在水中用结冰能力立起冰柱，就能撑起闸门。进去获得地图。

这个水池中有一个宝箱，可以用磁铁能力拉上来，其中是一个古代螺丝。

接下来我们需要开启水之神兽内部的5个控制机构。

控制机构①

从获得地图的房间出来，可以看到对面墙上有一组齿轮，下方水池里有控制机构。

用磁铁能力抓住齿轮组的把手，旋转，控制机构就升上来。

控制机构②

打开地图，把水之神兽的鼻子位置调整到如图所示。

水之神兽内部的水车就开始转动。从侧门来到二楼，可以发现小水车内有一个控制机构，

看准位置，用结冰能力把小水车旁的出水口堵上，小水车停下，进去启动控制装置。

当小水车转动时，可以乘上它的叶片，在最高点时跳出，可滑翔到对面高台上，

上面有个宝箱，内有100卢比。

控制机构③

从二楼侧门出去，来到三楼平台，正对着大水车。第三个控制机构就在大车轮的轴上，

其中还有一个能量球，进入的步骤如下：

如果神兽此时的鼻子位置如下图(从上往下第4格)，

那你在三楼平台看到的水车应该是逆时针转的。

如果水车顺时针转动，那就调整一下鼻子的位置

控制机构的门是关着的，必须等能量球掉入轴心的圆槽，门才会打开。

于是，看准下图所示的时机：圆球在槽内，随水车转动即将滑向外圈，

控制机关的门处于打开状状态。此时用时间凝滞的能力把能量球锁住。

等水车转动，进入控制机构，启动它。

控制机构④

首先，保证大水车还是逆时针转的(从三楼平台上看)。

大水车的叶片上有个宝箱，先把它拿，宝箱中有10支火炎箭。

获得方法：跳上小水车的叶片，再回头跳上大水车的叶片。

看清宝箱的位置，在它的叶片经过之后，跳到下一个叶片上，

顺着水车转动方向追上前一张叶片，就可以找到宝箱。

如果你像我一样，慢一拍，踩上宝箱后第2张叶片，也可以配合水车转动角度，从边缘追上宝箱

拿到宝箱后，再次乘坐大水车，来到三楼的另一个平台，上面有一个守护者，不难，干掉。

踩下平台上的按钮，水口会吐出一道瀑布，靠着登瀑布的能力，这就是条近路

沿着通道前进，可以看到神兽的鼻子。打开地图，把神兽的鼻子降到最低。

沿着神兽鼻子一直走到头。那里有个宝箱，里面是古代的轴承。

此时，再把神兽的鼻子抬起，至少抬到从上向下数第4格，更高也可以。

靠着鼻子上的纹路稳住自的位置，随鼻子的移动调整站位，别掉下去。

等鼻子转倒指定位置后，靠滑翔伞跳下，来到神兽头顶，有一个天窗可以进入。

进入后，先把黑色魔物的眼睛射爆。再次转把手，把整个天窗打开。

可以看到下方的控制机构被火柱包围。打开地图，操作神兽鼻子，放到从上往下数第5格，

鼻子喷水正好能浇灭火柱，启动这个控制机构。

在控制机构旁边，还有一个宝箱，里面是骑士之枪

控制机构⑤

最后一个控制机构在神兽的鼻子尖儿上，到达的方法推荐如下：

先按第④个机构的到达方法，来到神兽头顶。

把神兽的鼻子放到最低点从神兽头顶跳出，乘滑翔伞飞向神兽的鼻尖，落在鼻尖的喷水平台上。

在这里就能看到控制装置。慢慢抬起神兽的鼻子，配合鼻子的移动，调整站位，

可以卡在控制装置背后的矮墙上。

一直把神兽的鼻子抬到从上往下数4、5格的位置，可以很舒服得启动控制机构。

接下来就是 BOSS 战。

BOSS水之加农

BOSS 战在一楼，从神兽头上跳下去即可。

不过别急，先把这里的宝箱拿。空中跳下后，可以看到两边有两团黑色魔物，

空中射爆右手边的眼睛，再飞到左边去拿宝箱即可。宝箱内是10支冰之箭

来到一楼，在核心处确认一下，水之加农(Waterblight Ganon / 水のカースガノン)登场。

第一阶段

BOSS初期使用长枪攻击，招式有：

纵向前突刺。BOSS 是左手持枪，往它右手边走即可。躲开它的枪，可以靠近输出;

前方扇形横扫。拉远距离躲开，或者后空翻躲开;长枪插地，范围光波攻击，

建议拉开距离BOSS 的出招范围较大较远，贴身位置反而空隙较多，建议尽量贴身输出。

第二阶段

当 BOSS 的HP降到一半以下，它会倒挂在空中，并放出追踪冰块来攻击。

场景内水位上涨，活动空间受限。

此时建议：用结冰能力打碎冰块，然后用弓箭攻击 BOSS。

两次命中 BOSS 的眼睛，就能把它打翻在地，此时接近它输出亦可、远程连续射箭攻击亦可。

用时间凝滞能力锁住飞来的冰块，敲打冰块让它回击 BOSS 是很有效的输出手段。

BOSS 除放出冰块之外，还会投枪、挥枪，注意躲避。

在战斗的最后阶段，BOSS 放出的冰块会更多。

应对方法和之前的一样，不过要求玩家更快地处理冰块。

BOSS 战结束后，进入剧情，获得一个心之容器，

以及能力：“英杰ミファーの祈り”，效果是自动复活、增加 HP 上限。

回到左拉之乡，在王座前触发剧情，获得英杰武器：光鳞之枪。

光鳞之枪即便损坏，也可以回到左拉之乡，找这里的工匠修复，

不过需要消耗包括钻石在内的素材。
