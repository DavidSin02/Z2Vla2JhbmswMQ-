新片推荐：《巴霍巴利王 2》，印度版的超级英雄故事

2020-10-29
| 科学

《巴霍巴利王 2 ：终结》

导演：S·S·拉贾穆里

编剧：曼孔布·戈帕拉克里希南 / 马德汉·卡尔基 / C·H·维杰·库马尔 / 马努杰·芒塔希尔 / 维

杰耶德拉·普拉萨德 / S·S·拉贾穆里

主演：帕拉巴斯 / 拉纳·达格巴提 / 安努舒卡·谢蒂 / 特曼娜·芭蒂亚 / 萨伯拉杰

片长：141 分钟（中国大陆）

格式：2D

2015 年，那部荣登印度历史票房榜首的影片《巴霍巴利王：开端》在结尾处给观众留下了一个巨大的

谜团：为什么忠诚的卡塔帕将军会杀死巴霍巴利王？

漫长的 3 年等待后，《巴霍巴利王 2：终结》以前传的形式为观众回溯了属于巴霍巴利的故事前传。

并且不出意外地再次获得了票房冠军。

看过印度史诗《罗摩衍那》或《摩诃婆罗多》的观众不会对《巴霍巴利王 2》的故事感到陌生。这次，

英雄史诗的主角是希瓦的父亲，阿玛兰德拉·巴霍巴利。他出生在皇宫，与表兄巴拉拉德夫德斯一同由

太后抚养大，巴霍巴利从小膂力惊人，心怀正法，最终在一场对阵蛮族的大战中获得了王储身份。

在外出游历中，英雄爱上了诸侯国的公主提婆犀那，这为不甘失败的巴拉拉德夫德斯提供了绝佳的复仇

机会。阴谋、误解和愚昧，最终使英雄死在了亲人的剑下。

如上所述，整部影片描述的内容都颇具神话色彩，因此英雄和大反派的角色也十分套路化，流于表面的

皇室阴谋对于中国观众来说更是司空见惯，但《巴霍巴利王 2》的制作过程和最终效果，都让人感受到

片方的满满诚意，比如片头富有瓷塑质感的几处场景呈现，甚至让人有超级英雄影片的既视感。

事实上，《巴霍巴利王 2》也确实借鉴了好莱坞超级英雄电影的叙事模式。在拍摄前作时，导演 S·S

·拉贾穆里曾表示，在印度极为流行的系列漫画 Amar Chitra Katha正是他投拍影片的契机之一。为了

让孩子们了解国家的历史，出版商 Anant Pai 以漫画的形式重述了印度史诗、神话和民间传说故事。

在漫画中，儿时的 S·S·拉贾穆里看到了神灵们的宫殿、巨大的棕榈树和百发百中的弓箭。

他把这些符号搬到了自己的电影当中。每个属于巴霍巴利的战争片段，观众都不难感受到那种无所不能

的超人感，而在英雄与公主萌生爱情的场景里，那些幽默和近似无厘头的对话，无不来自西式漫画的叙

事风格。

这也是巴霍巴利王系列获得世界吸引力的原因之一。当然，也唯有在这样漫画式的故事里，主人公的全

部表现才能逻辑自洽。

而对于大部分观众来说，古印度充满神秘色彩的英雄传说和极具创意的战争场面已经很有吸引力。另外

，尽管并非出自本意，影片漫画感的角色塑造方式（比如配乐）还为其增添了几分“一本正经搞笑”的

特质。

唯一不足的是，虽然国内版只有 141 分钟（原片 167 分钟），中国观众可能还是会觉得它太长了些。

《香港大营救》

导演：刘一君

编剧：陆信 / 郭子胜 / 刘一君 / 刘云鹏

主演：成泰燊 / 颜丙燕 / 梁俊一 / 霍青 / 王晖

片长：95 分钟

格式：2D

尽管和《明月几时有》讲述了同一历史时期的同一事件，《香港大营救》只能算是一部中规中矩的红色

影片。

1941 年末太平洋战争爆发后，日军在港搜捕文化名人，共产党地下党员与在港各方势力周旋，全城出

动，完成营救。

《香港大营救》是转企改制之后的潇湘电影集团拍摄的“红色潇影“之一。2011 年，他们两部电影被

列入中宣部、广电总局“庆祝中国共产党成立 90 周年国产重点影片”，之后的作品还有《湘南起义》

和《青春雷锋》。

在这种风格定位下，原本充满复杂矛盾的《香港大营救》成了一个极为平面化的故事，导演刘一君在众

多可以观照人性的叙事方式中选择了最平庸的那一种，也就是多线展开的大全景。

而宣传词中多次提及的“40 年代香港场景搭建”和波澜壮阔的大撤退斗争场面，其实和我们常见的谍

战题材电视剧并无二致，实际观影过程中从音轨到字幕的质感，也真的很难感受到片方的五年精心筹备

。

不过，既然能在人民大会堂举办首映礼，它票房应该不会太差吧。
