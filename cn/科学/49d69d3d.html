客家算盘子的简单做法，成本不到5块钱，年菜必备

2020-11-14
| 科学

【客家算盘子】

算盘子，是广东省大埔县颇负盛名的汉族小吃，属于客家菜。它是用煮熟的芋头与木薯粉搓匀做成一粒粒的算盘子样，口感爽滑，是很多客家人逢年过节必不可少的小吃，算盘子“珠圆玉润”的形状，象征着大家团团圆圆；也借着算盘子的名，祈盼自己来年更加精打细算，挣很多钱。

算盘子的芋头也可以换成紫薯等其他食材，主要是看每个人的喜好不同，选择不同，2020鼠年春节降至，学会几道团圆硬菜，在家露一手，全家都夸赞。

客家算盘子的制作方法

食材：

木薯粉 100克 / 紫薯 100克 / 肉末 30克 / 虾皮 30克

包菜 半个 / 胡萝卜 半个 / 葱花 适量 / 生抽 一汤匙 / 玉米油 25毫升

【烹饪步骤】

- step1 -

紫薯去皮洗净切片上锅蒸熟

- step2 -

蒸好的紫薯倒入碗中捣成紫薯泥

- step3 -

紫薯泥中加入木薯粉和80毫升开水揉成不黏手的面团，一定要加刚烧开的开水和面做出来的成品口感才会更加Q软


- step4 -

取一小块面团，用手揉成小圆球后用拇指和食指捏住中间按压成扣子形状，一次可以多做一些放冰箱速冻保存，随吃随取即可

- step5 -

胡萝卜切丁，包菜切丝备用

- step6 -

起锅烧水下算盘子下锅煮至浮起

- step7 -

捞出的算盘子立刻放进冷水中再滤出，这样才会Q弹

- step8 -

锅中下油烧热，放入肉末和虾皮翻炒出香味

- step9 -

倒入算盘子和包菜丝翻炒均匀至包菜变软

- step10 -

调入盐，生抽，葱花

- step11 -

最后大火再次翻炒均匀即可出锅装盘

- step12 -

简单而又新奇的客家算盘子开始品尝吧


【美味呈现】

紫色的算盘子，口感软糯，色泽诱人，在家自制简单方便，一起来品尝这道美味客家菜
