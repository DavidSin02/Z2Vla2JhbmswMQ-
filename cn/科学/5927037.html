青山纸业两股东亏损2亿掐点逃离，16亿年产50万吨原纸项目不佳

2020-10-29
| 科学

近日，有金融媒体爆出青山纸业牛散大股东

牛散方怀月、沈利红清仓计划，

其拟从今年10月29日开始至明年4月26日的半年内，

减持完所持青山纸业5.73%股权。

一解禁就准备清仓，

可见其逃离心情急迫，

或是不太看好青山纸业发展前景。

而青山纸业16.5亿元，

投入年产50万吨食品包装原纸技改工程项目，

三年过去了，仅投入1.2亿元，占比为7%。

今年上半年，该项目亏损1920.98万元。


//

一解禁就准备清仓，亏本也在所不惜

//

9月29日晚，青山纸业披露股东方怀月、沈利红清仓计划，其拟从今年10月29日开始至明年4月26日的半年内，减持完所持青山纸业5.73%股权。

2016年9月，青山纸业实施定增募资21亿元，上述自然人股东各出资3亿元认购了1.02亿股，成为公司持股5%以上的重要股东。

然而，入股三年，今年9月30日解禁，二人就宣告了清仓计划。按照青山纸业9月30日收盘价2.07元/股计算，如果清仓顺利，加上资金成本等因素，二人将合计亏损近2亿元。

刚解禁就宣告清仓足见两股东迫不及待，亏损也要逃离一定程度上说明二人对青山纸业发展前景并不看好。

事实上，青山纸业的经营业绩缺乏亮色。今年上半年，公司实现营业收入13.60亿元，同比下降12.02%，净利润（归属于上市公司股东的净利润，下同）为9284.53万元，同比下降26.86%，扣除非经常性损益的净利润（简称扣非净利润）为5935.61万元，同比下降45.93%，接近腰斩。不仅如此，公司应收账款大幅增长，经营现金流净额正转负，且大幅下降。

备受关注的是，上述青山纸业高达21亿元的募投项目，除了4亿元用于补充公司流动资金外，其余的16.5亿元投入年产50万吨食品包装原纸技改工程项目。三年过去了，仅投入1.2亿元，占比为7%。今年上半年，该项目亏损1920.98万元。

//

两股东亏损2亿也要逃离

//

曾经出资6亿元捧场定增，如今亏损1.2亿元黯然离场。青山纸业回馈投资者的是一个“惨”字了得。

2015年2月12日，青山纸业筹划向不超过10名特定投资者非公开发行7.2亿股股票，募资21亿元。一年之后，定增方案获得证监会放行，并于当年9月28日实施。在这次定增中，方怀月、沈利红出现在发行对象名单中。

彼时，发行股份的价格为2.95元/股，方怀月、沈利红二人分别认购了约1.02亿股，涉及资金合计约6亿元。

公开信息显示，方怀月曾供职于银行，和沈利红一起被市场称之为牛散。

毫无疑问，牛散入股是奔着利益而去的。入股三年，方怀月、沈利红赚了多少呢。

今年9月30日，二人通过定增所持青山纸业股份解除限售，意味着二人所持股份可以通过二级市场减持等途径进行获利了结。

9月29日，青山纸业披露了二人减持计划，即拟在今年10月29日至2020年4月26日减持不超过1.32亿股、约占总股本的5.73%。5.73%，是二人所持青山纸业全部股份，意味着二人要清仓撤退。

二人减持时点可谓掐得非常准。除了一解除限售即披露清仓计划，减持时点定在10月29日开始，至明年4月26日。10月29日，正好是青山纸业三季报披露时间。从踩着这些节点减持看，二人减持似乎有些迫不及待。

然而，急切减持套现，二人不仅不能获利，反而会亏本。

二级市场上，今年4月17日，青山纸业股价为3.63元/股，后复权价为4.36元/股，此后，股价一直在下行。至9月30日，收报2.07元/股，后复权为3.23元/股。

如果方怀月、深红利清仓顺利完成，以2.07元/股为减持均价，合计减持金额为5.48亿元。

2016年至今，青山纸业未曾派发过一次现金红利，仅在今年5月24日实施过一次每10股转3股的送转股。

综上所述，入股三年，牛散方怀月、沈利红不仅未获利分文，反而巨亏0.52亿元。而这，还是不考虑资金成本的情况下。如果以通常创投8%的收益率计算，加上目前0.52亿元的账面浮亏，二人合计亏损1.96亿元。

//

营收净利双降 现金流下降129%

//

牛散清仓逃离心情急迫，或是不太看好青山纸业发展前景。

早在1997年，青山纸业就已在上交所主板登陆，比造纸龙头晨鸣纸业还要早3年。近10年来，青山纸业的经营业绩总体不佳。

经营业绩数据显示，2009年至2015年，青山纸业的扣非净利润一直在亏损。这7年，公司借助非经常性损益在保持净利润为正数，避免退市风险。

2015年，青山纸业定增募资21亿元，其中4亿元补充流动资金，16.5亿元投向年产50万吨食品包装原纸技改工程项目，试图借此打一个翻身仗。

2016年至2018年，青山纸业的营业收入小幅增长，净利润则呈现高速增长势头，净利润分别为4498.85万元、1.05亿元、1.55亿元。这三年，净利润高速增长，与行业去产能、纸价大幅上涨、环保监管趋严等密切相关。

然而，今年上半年，随着造纸行业下游需求疲软，青山纸业产品价格下降，加上公司光电、医药、纸制品加工等行业子公司受经济下行压力影响，经营业绩也在下滑，这些因素叠加，使得青山纸业经营业绩大变脸。其实现营业收入13.60 亿元，比上年同期减少1.85 亿元，同比下降 12.02%。实现利润总额1.15 亿元，比上年同期减少0.41亿元，同比降低 26.47%。净利润为0.93亿元，比上年同期减少0.34 亿元，同比降低下降26.86%。扣非净利润为5935.61万元，比上年同期的1.10亿元减少5042.42万元，降幅高达45.93%。

问题还不在于此，而是公司现金流恶化。上半年，公司经营现金流净额为—9477.94万元，去年同期为3.27亿元，同比减少4.22亿元，下降幅度达128.94%。

经营现金流净额大幅下降，源于应收票据及应收账款回收不力。截至今年6月末，青山纸业的应收票据为3.54亿元，同比增长32.26%，应收账款3.07亿元，较去年同期的1.40亿元增加1.67亿元，增幅为119.64%。

值得一提的是，曾经21亿元的定增募投项目远未达预期。

今年8月27日，青山纸业披露了上半年募集资金存放及使用情况的专项报告，报告显示，公司年产50万吨食品包装原纸技改工程项目计划总投资21.97亿元，拟投入募集资金16.50亿元。其中，项目一期工程3#纸机改造于2015年下半年开工启动第一阶段改造，于2016年下半年先期交付使用。截至今年6月底，累计投入资金1.28亿元。技改工程余下部分，试生产的产品质量经检测结果尚未达到预期，因而项目总体建设进度滞后。

今年上半年，公司实现营业收入13.60亿元，同比下降12.02%，净利润（归属于上市公司股东的净利润，下同）为9284.53万元，同比下降26.86%，扣除非经常性损益的净利润（简称扣非净利润）为5935.61万元，同比下降45.93%，接近腰斩。
