区块链+基因，生命货币的想象空间

2020-11-14
| 科学

一、区块链与基因链

区块链 ( BlockChain ）本质上是一种分布式的数据库，其中区块（Block）和链（Chain）与组成我们以及地球上所有生命的基因，有很多的相似性。

首先，细胞也是一个个的“块”，细胞核里的基因是螺旋式双“链”结构。

其次，身体间的合作也靠“ token”激励完成，比如能让我们感到快乐的多巴胺、内啡肽等。

我常用一个啪啪啪的比喻来解释 token 激励的重要性，啪啪啪是生命系统的挖矿机制，卵子与精子的结合，如同哈希碰撞成功，而成功“爆块”，啪啪啪的激励 token 就是多巴胺、内啡肽等快乐币。以至于，我们现在绝大部分啪啪啪并不是为了“爆块”，而是获取身体奖励的快乐token。也正因此，有些啪啪啪最终促成了真正的目的——繁衍后代。

随着基因测序成本的下降，也有人开始试着用区块链这把锤子去找基因领域中的钉子。

图：人类基因组测序成本2001-2017 来源：Nebula Genomics 白皮书

基因组研究目前最大的问题是：数据量不够。量变才能引发质变，巧妇难为无米之炊。随着基因测序成本的不断下降，让用户个人拥有自身的全基因组数据成为可能，如同移动互联网一样，这将开启一个前所未有的新维度。试想如果人人都有自己的基因组数据之后，将诞生怎样的一个新产业。

二、生命货币的想象空间

我们首先要弄清楚，基因检测在整个行业中的关系，谁需要基因数据。

提起基因、DNA，大部分人的第一印象应该是亲子鉴定。这是最早的应用之一。如今，这项应用在司法领域已经非常成熟，前不久著名的“白银杀人案”告破，也是它的功劳。

目前最成功的商业应用是：无创产前基因检测，通过在孕妇血液中提取胎儿的游离DNA，做一个检测看看是不是唐氏儿。唐氏儿是一种严重的遗传疾病，前些年的人造神话“音乐指挥家”舟舟就是典型的唐氏儿。无创产前是一个很大的市场，A股撑起了华大和贝瑞两家上市公司。

还有就是癌症治疗的靶向用药，也需要做基因检测。癌症是一种基因病，攻克的关键也在基因上。

另外，目前的新药研发对基因数据的需求也很迫。我之前看到过一个新闻说：夏威夷原住民起诉了某家美国药厂，因为这家药厂的治疗心血管疾病的药物对他们无效。照例说无效的药物是不会通过临床测试的，这又是怎么回事呢？原来这种药物在临床测试中对白人、黑人都有效，但偏偏对夏威夷的原住民无效。这源于夏威夷原住民特殊的基因差异。

此外，还有面向大众用户的消费类基因检测。以 23andMe 和 ancestry 为代表，主要提供祖源分析和一些趣味解读。其中 23andMe 还拿到了几种遗传疾病的监测证书。我尝试过一次，几百块钱。其中祖源分析就像一个加强版的DNA鉴定，会告诉你百分之零点几的xx国人成分，百分之几点几的xx民族成分。特别想吐槽一下：国家是政治概念，你护照上写着什么国家就是什么国家；民族更多的是一个文化概念，你身份证上写的什么民族就是什么民族。即使我测出一个百分百的少数民族成分，我去要求改民族和高考加分，估计也没人鸟我。这个产品有点“为赋新词强说愁”的感觉。从用户需求的角度看，更像是一种高科技的体验吧。“酒量”等项目确实很准，至于营养、性格就属于玄学范畴了。

基因的价值，围绕“人”展开，司法、健康、临床、制药。区块链+ token 激励，又能够带来怎样的变革？

先说一个案例，这个例子在几乎所有的基因组+区块链项目白皮书中都有提及。也是神奇。

2015年，制药巨头基因泰克以6000万美元的价格，向基因检测公司 23andMe 购买了 3000 名帕金森病患者的全基因组数据。一方面，需求方付出了巨额资金。另一方面基因数据的所有方——用户，并未获得直接收益。而作为中介方 23andMe 几乎赚取了所有收益。

也就是说：我自己花钱做了基因检测，基因公司把我的数据卖了，而且我还没有得到任何收益？——也就是传说中的被人卖了，还帮人数钱？？

6000万美元买3000个人全基因组数据，平均每人2万美元，折合成人民币要十多万呢，不少了。

如果一定要找价值的话，基因泰克研发出的新药，提供了一个新的治愈可能性，但是你仍要付费的。。。打折？不要想了。

这就给了区块链介入其中的机会，在持之有故，言之成理的项目白皮书中，都介绍了这样一个方案，以 Nebula Genomics 白皮书中的经济模型为例。

图：Nebula Genomics 白皮书中的经济模型，来源：Nebula Genomics 白皮书

用户通过出租自身基因数据获取token奖励，交易记录登记到区块链保存。有效避免被中介公司独吞。

这里还有一个疑似漏洞，基因样本送去测序的时候，测序方总是有一份数据的。那他直接出售这些数据给需求方即可，为什么还要费劲儿从用户说手中买呢？或者说，用区块链做这件事的必要性有哪些：

首先，需求方与测序方也存在一个猜疑链，需求方会担心测序方编造一个数据来糊弄自己。加上区块链之后，需求方至少能够增加一个额外选项。

第二，需求方需要的是不仅仅是一个单纯的基因数据，而且需要这个数据的主人的更多表型信息，比如体检记录、生活习惯等等。显然这不是测序方能够提供的。

第三，之前的用户基因组数据被把持在各个公司手中，想要研究需要支付高额的价格之外，有些公司会因为复杂的控股关系，而故意不与一些研究机构和存在竞争关系的公司合作。这就造成了一个一个的数据孤岛。宝贵的，属于用户的基因数据无法流通。

第四，接上条，这些被大公司把持的数据，本来就应该是用户自己的。是每个人一出生就自带的“生命资产”，决定这些资产的应该是用户本人，而不是那些玩弄注册协议的大公司。

但我们可以看到，在 Nebula Genomics 项目的构想中，第三方公司仍然需要向Nebula Genomics 支付 Fiat money （法定货币）。这个经济模型就很糟糕了，等于第三方用法币买的用户的数据，Nebula Genomics 官方却支付给用户 token 。

对比迅雷链的生态，C端用户向“星域云”共享宽带和存储空间，获取token：链克。B端客户向“星域云”购买CDN和其它云计算服务，支付token：链克。同时，B端客户又向C端用户开放链克购买自身的服务。完整的 token 经济链条，在运转上没有法币的介入。

至少在经济模型上 Nebula Genomics 还需再精进。当然，也说不定他们是故意的，毕竟企业运转的现金流是用法币来衡量的。

除此之外，基因数据也符合区块链介入的一些前提条件：

  • 基因通过基因测序仪数字化，数字化就可以上链；
  • 基因数据人人不同，可以确权；
  • 基因数据本身存在变现价值。

随着越来越多的人意识到自身基因数据的权益。基因数据上链，解决了基因数据的确权问题，让基因数据交易的更加透明性。交易带来的token激励势必会吸引更多人加入，人类基因组数据将愈加丰富。随着数据量的增加，基于基因组的研究成果和应用也将更加丰富，新的生态将逐步构建起来。

未来生态的想象空间：

  • 精准医疗。现代医疗停滞不前，是平均化的分析方法遇到了瓶颈。所谓平均化医学，就是画一条平均线，偏离这条线就有问题。维度单一，且忽略了个体特征。下一步，由个人基因组数据为依托的精准医学，将充分尊重个体的差异的前提下，提供个性化的精准的医疗方案。这个市场预计将几倍于目前的医疗市场。
  • 新药研发。2016 年德勤会计事务所发布的一份研究报告显示：研发一个新药的平均成本为 15.4 亿美元，耗时 14 年。成功的项目收益颇丰，但凤毛麟角，大批药物死在临床前和临床阶段，数亿、数十亿美元打水漂是药物研发的常态。新药研发只有资金实力雄厚的大药企才玩得起。为了对冲风险，新药研发早已是一门复杂的金融生意。结合 token 构建的新经济模型和未来庞大的基因数据库，在提高新药研发精准度的同时，也将降低投资门槛。
  • 众筹研究。理论上基因相近，对某些疾病的风险也相近。目睹过亲属患病情景，势必也担心自己的风险。针对某一类具体疾病或者药物研发方向，利益攸关的用户可以选择以token 众筹的形式支持研究，并通过智能合约约定收益分成。谷歌的创始人布林通过基因检测就发现自己患老年痴呆的风险很高，直接大笔一挥赞助了多个老年痴呆症的项目研究。希望未来，人人都有机会参与。这才是真正的保险。否则到时候光有钱，没办法治也白搭。

三、项目分析

美国·Nebula Genomics

美国，作为目前蓝星科技的第一强国，在制药和基因组研究领域也是首屈一指。区块链+基因自然也不能落后。

前文已提到的 Nebula Genomics 项目，由哈佛大学的遗传学大牛 George Church 发起。白皮书里除了前面吐槽的经济模型之外，整体靠谱，颇有论文范儿。特别是对隐私保护的部分阐述了很多。 Nebula Genomics 项目在中国落地应该会给到其在中国的公司：奕真生物。感兴趣的可以关注。

美国·LunaDNA

有一定的关注度，但社群规模远远比不上Nebula Genomics。官方迟迟没有发布正式版白皮书，之前有流传过一份，但被官方否认了。LunaDNA 自曝是测序仪巨头 illumina 的前员工，并且获得了illumina一定资金的支持。目前了解到的都是概念规划，还没看到落地方案。

印度·Shivom

最近很火，感觉买了币圈很多广告位，但槽点很多。首先，Shivom 白皮书看完给我的印象就是两个字：空谈。核心的地方或一笔带过，或避而不谈——如何测序，如何存储数据，如何构建经济模型——细节统统没有，口号倒是喊了不少，对代币也描绘了很多。我还特意咨询了生物界的专业人士，竟然对 Shivom 白皮书中好几页的核心团队和理事会成员无一耳闻。感觉这是一个为韭菜量身定制的假大空方案。其中，白皮书还有一句说“直接与政府合作以推动对全民进行基因测序”。我就特别想赞美印度人民的天真无邪。

中国

这么时髦的事情，怎么可能没有中国的身影呢。前前后后，我看到过的就有：GHT、GENE、DNAC 等等，其中，有一个 GeneData 的项目，在官网上将“顾问”的人名和照片都搞错了（ http://dnabbs.com/t/topic/41 ），这还是基因领域著名的科学家。后来这个项目被爆各种造假，据说现在转去做交易所了。

靠谱的也有，着重说一下：HGBC，人类基因组价值生态链。我已经买了一个他们的全基因组测序，还没有到货。

相比之下，这个项目更“以用户为中心一些”。而且解决了我关注的几个点：

1、合法合规的重要性。中国的《人类遗传资源管理暂行办法》明确规定了遗传样本未经许可不能出境。基因这么敏感的数据，让外国人来做的可能性为零。加之现在人为割裂的国外互联网和国内互联网，未来区块链内外有别也是定局。基因+区块链只有合法合规的一条道路可以走。

插播一个小故事，数年前，某日本节目通过 DNA 检测发现某家喻户晓的女影星的祖上来自云南某少数民族，于是派出制作组工作人员去云南采集基因样本，进行验证。。。然后，被我英勇的公安民警当成敌特事件，给一网打尽了。。。

2、技术积累。基因+区块链，必须要有基因数据才能玩起来，那么用户以基因数据参与就好了。那些需要“币”才能参与的项目，或许只是想骗取你的币而已。虽然 HGBC 的基因检测盒我还没有收到，但从公开披露的信息看，其前身在基因领域还是有很多积累的。是国内少数几家有能力解读全基因组数据的公司。

3、中国速度。发达国家把路子走通，中国靠人海战术淘汰所有竞争对手，最终搞成白菜价。似乎已经是一条必然路径。什么也拼不过 996，人多就是竞争力。像币圈的主链项目，吹了一整年，才磨磨蹭蹭的准备上测试网络，能不能成功还得等下个月。现在中国的项目，上来就是主链，根本不磨蹭。

4、迅雷链的大太子。基因组数据是一个很大的数据，上链的话对数据存储空间有很大的要求。目前比较热的分布式存储方案是 IPFS ，包括很多基因组项目也都声称要用 IPFS 来存储基因数据。我对此并不看好，IPFS 这种分布式无监管又自带交易系统的存储网络，估计一上线就成为色情网站，自身的稳定性就难以保证。而另一个现成的，在 P2P 技术上绝对领先的是国内的迅雷链。依托于玩客云设备共享的宽带和存储空间，这才是真的分布式存储和落地项目。加上迅雷一直符合国内的监管，HGBC 选用迅雷链做生态应用，非常明智。

5、实践是检验真理的唯一标准。解决具体问题，而不是凹某个主义的造型。区块链带来的去中心化，解决了传统的中心化世界的一些弊端，但不代表去中心化是万能药，也不能把中心化一棒打死。最终还是要落在要解决的具体问题上。如果这个问题是棵树，那么既要中心化的树干，又要去中心化的树叶。

小结一下：

  • 基因生物产业的未来有极大的想象空间
  • 基因与区块链的结合能够助力产业发展
  • 基因数据要在国内做成，必须合法合规
  • 基因区块链项目必须有基因数据才能运转

四、风险

事情虽好，风险也不小。

先分享一个简单的评估方法：风险与收益的四象限分析

图：风险与收益四象限分析

  • 第 1 象限：高风险低收益。借钱是典型，如果对方不还钱，损失很大；如果对方还钱，也基本没有收益。从金钱回报上来看是典型的高风险低收益。
  • 第 2 象限：高风险高收益。真正的投资象限，风险是一道门槛，谁能驾驭风险，谁就能拿到收益。
  • 第 3 象限：低风险低收益。就是存款了；
  • 第 4 象限：低风险高收益。一个美好的愿望，是太多骗局伪装的对象。

我的原则是只在2、4活动，远离1、3。远离 1 很容易，远离 3 看起来很奇怪。其实也很简单，因为有这种好事，早被别人填平了。除非你有某种切实可靠的垄断性的资源，然而这又岂是我等凡辈可企及的。要不然就是没有理解其中隐藏的风险，比如那些被精心设计的骗局。如果有人告诉我有什么无风险高收益的项目，我会绕到后面看看他手里握的是什么镰刀？割韭菜的小镰刀，还是死神用来割脑袋的大镰刀。

第 4 象限是赚不到钱的，那就只有第 2 象限了。在第2象限比拼的就是风险管控能力。

区块链的风险：归零、被盗、漏洞、空气币.... 加上基因之后，还有几个独特的风险，了解一下：

1、隐私

《三体》中描述了一个场景，ETO组织用基因技术制造了一个病毒，这个病毒只对面壁者罗辑一个人进行定向攻击，也就是传说中的“基因武器”。当然这种科幻小说中的场景，如能实现，如博物学家刘大可所言：能拿一摞诺贝尔奖。用此去杀一个人，简直是核武器打蚊子。

更靠谱的担心是，基因信息这种高科技带来的未知性，比如我换某某癌症的风险很高，会不会遭遇基因歧视，影响找媳妇之类的。

对于参与方隐私保护的方案 Nebula Genomics 在白皮书中对此阐述很多，相信区块链运用的零知识证明等加密技术可以解决大部分问题。而我个人之所以选择加入进去，是因为预期收益可以覆盖掉预期损失。

2、生物难题

21世界是生物的世纪，但21世纪有100年。能不能在有生之年看到生物的崛起。是个很大的风险。一直以来，生命科学领域一直都是资本巨头的玩家，有这么一次机会非常难得，长远看能为人类的科研道路贡献一份力量也是好的吧。

3、token价值

首先，经济模型非常重要，这代表着以后你手里的 token 的价值，如上文所述 Nebula Genomics 的经济模型中，做为核心驱动力的需求方，支付的却是法定货币，那用户手中 token 就没有基础的价值锚定。而像 Shivom 这种割韭菜的项目，只说自己的 token 能做一二三四五，但如何保证实现或只字未提，或异想天开。

大部分产业是为房东打工。而 token 是整个生态的核心资产，早期参与者的奖励在未来能否有价值，需要看经济模型是否合理。

其次，当基因数据通过区块链确权给个人之后，生态组织方能否继续把用户组织在一起使用生态中的 token。很考验团队的运营能力。也有可能，随着明星应用的崛起，各种生命类 token 开始自由竞争。去中心化的时代，你无法像法币一样强制大家使用某种 token ，所以早期参与者领到的 token 不一定是最后的胜利者，甚至都不一定有胜利者。像癌症治疗可能会有一个某某癌症的token，某某药物会有某某药物的token。好在它们彼此之间会有兑换流通。所以，基因数据可能才是最核心的资产。这不是一个一劳永逸的事情，时刻关注生态发展变化，多参与新项目和及时调整分配策略，才是正途。

对区块链经济模型的分析、新项目的评估标准等，货币进化论会在未来的文章中详细阐述，欢迎继续关注。
