什么是天敌昆虫？

天敌昆虫是一类长期在农田、林区和牧场中控制着害虫的发展和蔓延的昆虫。分为捕食性天敌昆虫和寄生性天敌昆虫。


+ 捕 食 性 天 敌 +


+ 螳 螂 +




螳螂是比较凶悍的昆虫，它那双锋利无比的“螳螂刀”足以让人望而生畏。

螳螂可捕食多种害虫，如蚊、蝇、鳞翅目幼虫等小型昆虫，蝉、飞蝗、螽斯等大型昆虫。


常见的比如，美丽的薄翅螳Mantis religiosa，在棉田已知可捕食棉铃虫、地老虎、烟青虫、粘虫、盲蝽、棉蚜、叶蝉、蝼蛄、棉小造桥虫。

此外，以“中华”命名的中华大刀螳Tenodera sinensis 以及广捕性的广斧螳（广腹螳螂） Hierodula patellifera、枯叶大刀螳Tenodera aridifolia都是著名的农田益虫。



广腹螳螂


+ 捕 食 性 瓢 虫 +



瓢虫种类很多，全世界约有6000多种，我国记载700余种，其中80%以上为捕食性的，15%为植食性的，另外5%为菌食性的。

我们最为熟悉的瓢虫，无疑是童年伙伴——七星瓢虫，它红红的壳子，黑亮的斑点，十分迷人。难得的是七星瓢虫不仅长得可爱，还是地地道道的捕虫能手：据统计，1只七星瓢虫一天可以吃掉100多只蚜虫，有的地方称其为“麦大夫”。


瓢虫小时候虽说貌丑，但也是捕虫高手。


除了熟悉的七星瓢虫，还有很多捕食性瓢虫。它们分布广泛，可捕食蚜虫、叶螨、粉虱、介壳虫等等，是控制农业害虫的重要天敌。


其中著名的比如：捕食棉蚜、豆蚜、高粱蚜、菜缢管蚜，且千变万化的异色瓢虫Harmonia axyridis：



异色瓢虫


捕食麦长管蚜、禾谷缢管蚜、棉蚜、豆蚜的十三星长足瓢虫（十三星瓢虫）Hippodamia tredecimpunctata ：


十三星瓢虫


捕食棉蚜、麦蚜、玉米蚜、高粱蚜的龟纹瓢虫Propylea japonica：



龟纹瓢虫


此外，黑缘红瓢虫、澳洲瓢虫、红颈瓢虫、中华显盾瓢虫、连斑小毛瓢虫等等都是极其重要的农田天敌昆虫。

+ 步 甲 +



步甲，俗称步行虫，它们大多为捕食性，在自然界消灭害虫方面起着一定作用。中国的金星步甲大量捕食鳞翅目幼虫，是粘虫等害虫的重要天敌，同时还捕杀蜗牛等有害软体动物。



黄缘心步甲


黄缘心步甲Nebria livida能够捕食粘虫、地老虎、菜青虫、蝗虫以及有害的蝽类。



双斑青步甲


双斑青步甲Chlaenius bioculatus，又称黄星步行虫，它初孵幼虫经1个小时爬行后，即可捕食，能捕食比它体大数倍的鳞翅目幼虫，并有群集捕食大型幼虫的特性。

+ 隐 翅 虫 +



隐翅甲科的昆虫的鞘翅很短，后翅隐藏在前翅下。




隐翅甲一部分为腐食性、粪食性的昆虫，可以促进自然界物质的循环；另外一些以菌菇、植物的果实以及花粉等为食物；而大部分以肉食为主，靠捕食农林害虫为生。




比如，黑胫虎隐翅虫Stenus cicindeloides捕食蚜虫、叶蝉、蓟马、鳞翅目昆虫。青翅毒隐翅虫Paederus fuscipes捕食玉米螟、棉叶蝉、棉盲蝽、棉小造桥虫、棉铃虫、棉蚜、蓟马等等。

+ 蠼 螋 +





蠼螋[qú sōu]，是革翅目的杂食性昆虫，别称夹板子、剪指甲虫、夹板虫，蠼螋常常生活在树皮缝隙，枯朽腐木中或落叶堆下，喜欢潮湿阴暗的环境。其实，很多蠼螋是农田中常见的天敌昆虫。




例如，广捕性天敌昆虫的青足小肥螋Euborellia pallipes、日本球螋Labidura japonica能够捕食小老虎、棉铃虫、棉小造桥虫、鼎点金刚钻、斜纹夜蛾、红铃虫、短额负蝗、棉蚜等多类害虫。


+ 胡 蜂 +


胡蜂，可是昆虫界有名的“暴脾气”外加不好惹，不曾想它们也是我们的农田友军。




北方黄胡蜂Vespula rufa喜捕食粘虫、银纹夜蛾；普通长脚胡蜂捕食棉铃虫、棉小造桥虫、豆荚螟、粟灰螟、菜青虫。此外，纹胡蜂、黄斑细脚胡蜂、日本长黄胡蜂等也都是农田益虫。


北方黄胡蜂


+ 蜾 蠃 +


“螟蛉有子，蜾蠃负之”，说的便是蜾蠃[guǒ luǒ]的故事，螟蛉是指鳞翅目昆虫的幼虫，蜾蠃把螟蛉衔回巢中，后作为自己孩子的食粮。



蜾蠃和胡蜂外形有些相似，平时都是自由活动，仅在产卵时才衔泥筑巢（或寻找竹管），之后捕捉鳞翅目幼虫，用自己尾上的毒针麻醉幼虫，在其身上产卵，供后代的食用。可见蜾蠃是鳞翅目的天敌昆虫。



农田常见的比如棕喙蜾赢（黄唇蜾蠃）Rhynchium bruneum，产卵时每粒卵下平均放15条左右棉铃虫等鳞翅目幼虫。前旁喙蜾蠃捕食银纹夜蛾幼虫，四带佳盾蜾蠃和镶黄蜾蠃则捕食菜青虫。



+ 土 蜂 +




土蜂科的土蜂，其幼虫是蛴螬的外寄生者。雌虫在土中挖洞寻找寄主，发现蛴螬先用蜇刺注入毒液，使蛴螬麻醉，再产卵其上，封闭土室。



金毛长腹土蜂


日本土蜂寄生于油桐绿丽金龟；金毛长腹土蜂寄生于方头绿丽金龟、樱桃丽金龟、铜色丽金龟；白毛长腹土蜂寄主为大黑鳃金龟、异丽金龟。

+ 草 蛉 +



草蛉是脉翅目草蛉科的昆虫。说起草蛉我能想到的词汇是端庄、优雅：纤细的身躯、细长的触角、透明的双翅，以及具有金属光泽的复眼，仿佛穿梭于花间的灵秀女子。


只是，草蛉的幼虫——蚜狮，可与亲生父母没有丝毫相像，样貌大不同。既然称为蚜狮，说明它也是蚜虫杀手，据统计，一头蚜狮在整个幼虫阶段能吃掉700头以上的蚜虫，真是好胃口造就好益虫。


草蛉可是真正的捕捉害虫的小能手，不仅是各类蚜虫——棉蚜、菜蚜、烟蚜、麦蚜、豆蚜、桃蚜、苹果蚜、红花蚜，而且粉虱、红蜘蛛等等都是它们的捕食对象。


此外，草蛉还喜欢吃很多种害虫的卵，诸如棉铃虫、地老虎、银纹夜蛾、麦蛾和小造桥虫的卵，都在其食物范围之内。


捕食性草蛉 成虫


+ 吸 食 性 蝽 类 +





蝽，就是我们熟悉的放屁虫或者臭大姐，在我们的记忆中，它们又丑又难看，还生有臭腺，特别不招人喜欢。

然而实际上，蝽也是一个大家庭，全世界已知的就有4万多种，造型繁多、千奇百怪，不仅有食素的蝽，也有不少捕食性蝽。



花蝽科、盲蝽科、姬蝽科、猎蝽科的蝽可捕食叶蝉、飞虱、蚜虫、蓟马、棉叶螨及棉蛉虫等害虫。




花蝽科小花蝽属的蝽，是一类具有重要利用价值的天敌。全世界已知有近百种，我国已知10余种，在我国大部分农田中，常见的优势种主要为东亚小花蝽、微小花蝽和南方小花蝽。小花蝽可捕食蚜虫、蓟马和红蜘蛛，以及红铃虫、造桥虫、棉铃虫等害虫的卵和初孵幼虫。



狡小花蝽 Orius insidiosus


盲蝽科的黑食蚜盲蝽和异须盲蝽捕食棉蚜。

姬蝽科的华姬蝽Nabis sinoferus是杂食性天敌昆虫，在田间可捕食蚜虫、鳞翅目昆虫卵和初孵幼虫等，亦可食用各种花粉。而暗色姬蝽Nabis stenoferus在棉田内主要捕食棉蚜、蓟马、盲蝽若虫以及鳞翅目害虫卵等。



黑食蚜盲蝽


此外，长蝽科的沙地大眼长蝽在棉田捕食棉蚜，而酷似蝉儿的大眼蝉长蝽是北方棉花害虫的主要天敌。蝽科的蠋[zhú]蝽Arma chinensis喜欢榆紫叶甲及松毛虫等鞘翅目和鳞翅目害虫，也是极其知名的天敌昆虫。


大眼蝉长蝽


+ 食 蚜 蝇 +



食蚜蝇，就是Cos小蜜蜂的家伙，它们访花采蜜，穿梭于花丛间，也如同小蜜蜂一般为花儿传粉。

幼虫“食蚜”，虽然模样丑到原地爆炸：蛆形，有一些透明的质感，可以看到内部脏器，乍看好似一坨鸟粪。


但它们面丑心善、技能满满：一般一只幼虫一天可取食百余只蚜虫，是防治蚜虫的好帮手。当然，并非所有的食蚜蝇幼虫都以捕食蚜虫为生，也有些为腐食性或植食性。


食蚜蝇以捕食蚜虫著称，此外，还能捕食介壳虫、粉虱、叶蝉、蓟马和鳞翅目的幼虫等。


比较常见知名的天敌包括：宽尾细腹食蚜蝇、短刺刺腿食蚜蝇、门氏食蚜蝇、短翅细腹食蚜蝇以及狭带食蚜蝇等等。



门氏食蚜蝇


除此之外，捕虫螨、蜘蛛（均不是昆虫）等也是极其重要的害虫天敌。
