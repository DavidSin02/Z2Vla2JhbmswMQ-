保密小测试

（第一期）

一、判断题

1.国家秘密的保密期限，除另有规定外，绝密级不超过30年，机密级不超过20年，秘密级不超过10年。（ ✔ ）

2.科学技术中的秘密事项属于国家秘密。（ ✔ ）

3.在定密的时候，无需对国家秘密载体以及属于国家秘密的设备、产品做出国家秘密标志。（ ❌ ）

4.解密后的国家秘密事项可以公开。（ ❌ ）

5.可以通过普通邮政、快递等无保密措施的渠道传递涉密载体。（ ❌ ）

6.移动存储介质不能在涉密计算机和非涉密计算机之间交叉使用。（ ✔ ）

7.非涉密计算机可以用来存储、处理和传输涉密信息。（❌ ）

8.中央国家机关、省级机关及其授权的机关、单位可以确定绝密级国家秘密。（✔ ）

9.国家秘密载体的复制件应当视同原件进行管理。（✔ ）

10.涉密会议、活动的保密工作原则是谁主办，谁负责。 （ ✔）

二、单选题

11.涉密计算机的放置应与非涉密计算机或设备之间保持多远距离？（ A ）

A.1米 B.2米 C.3米 D.0.5米

12.涉密人员因私出国（境）的证件，应在回国（境）几天内交组织人事部门集中保管。（ B ）

A.5天 B.10天 C.3天 D.12天

13.一般涉密人员的脱密期为多少年？( C )

A.1年 B.3年 C.1年至2年 D.3年至4年

14.秘密级计算机，身份鉴别口令的更新周期为？（D ）

A.40天 B.10天 C.20天 D.30天

15.处理秘密级信息的计算机，口令长度不少于几位？（B ）

A.7位 B.8位 C.6位 D.5位

三、多选题

16.国家秘密的密级分为哪三种？（ABC )

A.绝密 B.机密 C.秘密 D.内部

17.涉密人员按照涉密程度分为哪三类？（ABC )

A.核心涉密人员 B.重要涉密人员 C.一般涉密人员 D.普通涉密人员

18.国家秘密如何解除？（ BC ）

A.随意解密 B.审查解密 C.自行解密 D.不可解除

19.涉密网络“三员”是指？（ ACD ）

A.系统管理员 B.计算机管理员 C.安全保密管理员 D.安全审计员

20.保密工作的基本经验是？（ABCD ）

A.党管保密 B.专兼结合 C.综合防范 D.攻防一体

21.国家秘密的变更是指什么的变更？（ABD ）

A.密级 B.保密期限 C.标识 D.知悉范围

22.哪些机关具有法定定密权？（ ABCD ）

A.中央国家机关 B.省级机关 C.设区市一级机关 D.自治州一级的机关
