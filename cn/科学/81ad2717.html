护肤品成分全索引~成为达人的必经之路

2020-11-14
| 科学

用于护肤品中的成分多达数千种,要一一列举出来非常之难,好在总有一些成分是非常常用的。


下面,我将常见的一百多种成分整理出来,供大家参考。如果你可以对这些成分略有了解,掌握其中的规律,那么理解护肤品的难度就会降低,选择起来也可以抓住关键,知道哪些是自己想要的。


这里面所提到的成分，有自身特性的描述。有其作用的，比如抗衰老，保湿等等。有敏感度的，比如无刺激，或者低刺激等。


使用方法：按成分第一个字的首字母排序：例如：丙二醇，属于B的组别。防晒成分和香精香料在文末处！


很多时候，买家秀和卖家秀是截然相反的效果。但是知道了成分，我们就能知道它的功效是否真实！


比如下图这个精华水的成分表，大家参考这个索引表。大部分的成分都可以查询出来！



A：

1.氨基酸衍生类表面活性剂:如月桂酰甘氨酸钾、月桂酰谷氨酸钠、椰油酰甘氨酸钾、椰油酰谷氨酸钠、椰油酰肌氨酸钠等。此类表面活性剂多用于清洁产品,比皂基更为温和,可以做成弱酸性体系,清洁力和对皮肤的友好度之间达到了较好的平衡,部分成分还有抗菌作用。

海洋精华水成分解析案例

B:

1.变性乙醇(alcohol denatured/ethanol denat. ) :添加了少量的其他物质使之不能食用的乙醇。多数国家对酒类征收重税,变性乙醇能保有酒精的大部分性能,所以仍可用于非食用的领域(燃料、溶剂、清洁剂),这样就不会被课以重税。


2.苯乙基间苯二酚(phenylethyl resorcinol) :即SymWhite377,一种美白成分,作为酪氨酸酶的类似物发挥竞争性抑制酪氨酸酶的作用,可减少黑色素合成,达到美白效果。（活性成分）


3.丙二醇( propylene glycol ) :无色无味的黏稠液体,能与水任意混合,与化妆品成分有很好的相容性。其黏性和吸湿性好,可用作润肤、吸湿和软化剂,也用作香料和防腐剂的溶剂。丙二醇刺激性很低,无毒,但高浓度的丙二醇可以导致皮肤灼热感。


4.吡咯烷酮羧酸(PCA) :表皮丝聚蛋白分解后的产物之一,是天然保湿因子的组成成分。吡咯烷酮羧酸的吸湿性优于甘油、丙二醇、山梨醇等,具有良好的生物相容性和化妆品配方兼容性,也极少导致过敏或刺激反应。


5.吡哆素( pyridoxine ) :即维生素B ,包括吡哆醇、吡哆醛和吡哆胺,具有抗老化、减轻皮肤干燥和抑制皮脂分泌的作用。（活性成分）


6.PCA钠:见吡咯烷酮羧酸,为吡咯烷酮羧酸的钠盐。


7.苯甲酸钠(sodium benzoate ) :对酵母有很强的抑制作用,对霉菌和细菌也一定程度的抑制作用,无毒。（防腐类）


8.苯氧乙醇(phenoxyethanol ) :广谱抑菌剂,对真菌、细菌均有抑制作用。驻留型产品中浓度偏高时皮肤可能会产生灼热感。（防腐类）

D:

1.丁二醇(butylene glycol) :无色无味的黏稠液体,能与水任意混合,是常用的保湿剂。其保湿性能很好,但刺激性较丙二醇低,也没有丙三醇(甘油)黏稠,现在越来越受欢迎。丁二醇和丙三醇及透明质酸钠配合,可取得很好的保湿效果。


2.豆蔻酸( myristic acid ) :即十四酸,主要用于皂类的制作,其皂基有良好的发泡作用。该成分可能具有轻微刺激性和一定的致粉刺性。


3.豆蔻酸异丙酯( isopropyl myristate ) :一种常用的润肤剂,同时其作为酯类,也有较强的溶油作用,因而也常用于卸妆产品中。研究者普遍认为该成分具有致粉刺性。


4.多羟基酸类( polyhydroxy acids, PHAs ) :是第二代a一羟基酸,带有多个羟基,故有更好的保湿作用,同时刺激性更弱,常用的有葡糖酸内酯(gluconolactone )、乳糖酸( lactobionic acid )、麦芽糖酸( maltobionic acid)等。PHAs有抗氧化、抗糖化作用,多用于抗衰老。（活性成分）


5.碘丙炔醇丁基氨甲酸酯( iodopropynyl butylcarbamate) :简称IBPC,是一种温和的、广谱而稳定的抑真菌剂。（防腐类）

鲑鱼精华成分解析案例

F：

1.蜂蜡(beeswax) :是一类固体油脂,可能是人类最古老的化妆品原料,主要用于一些膏状和半固体的产品,如唇膏、发蜡等。蜂蜡刺激性低,具有一定的抗氧化、抗菌能力。


2.泛醇(panthenol ) :即维生素Bs,是一种水溶性的、稳定的、易穿透角质层的小分子物质。泛醇有保湿、减轻皮肤刺激、舒缓、抗炎和止痒作用,常用于保湿和抗敏类产品。（活性成分）


3.泛醌(ubiquinone) :即辅酶Q10 (Co Q10) ,是一种温和而天然的成分,也是极强的抗氧化剂。其颜色淡黄,价格较贵。（活性成分）


4.富勒烯( fullerenes ) :是一种由碳原子构成的球状分子,完全由人工合成。研究发现富勒烯具有极强的抗氧化和清除自由基作用,常用于美白、抗皱、抗衰老产品中,价格极其昂贵。（活性成分）罗伯特·科尔、哈罗德.克罗托

、理查德.斯莫利三人因发现富勒烯而获得1996年诺贝尔化学奖


G:

1.甘油(glycerin) :化学名为丙三醇,味道是甜的,是一种水溶性保湿剂,同时也是皮肤天然保湿成分的一部分,配合透明质酸钠保湿效果更好。甘油是经典保湿成分,价格便宜、刺激性低,效果好,所以保湿类产品中通常少不了它。


2.甘油硬脂酸酯(glyceryl stearate ) :一种常用的乳化剂,但乳化能力不强,一般作为辅助乳化剂。类似的还有甘油异硬脂酸酯、甘油硬脂酸酯柠檬酸酯等,与之配合使用的有PEG-20甘油硬脂酸酯、PEG-30甘油硬脂酸酯等。


3.瓜尔豆胶(guar gum) :瓜尔豆植物种子中发现的成分,是一种多糖,具有吸水能力,能在皮肤表面形成膜。瓜尔豆胶常用作增稠剂和助乳化剂。


4.硅氧烷类(dimethicones/siloxanes ) :即所谓的“硅油” ,是一类物质,包括聚二甲基硅氧烷、三硅氧烷、环硅氧烷等。此类成分非常稳定,低刺激,具有良好的润肤和保湿功能,可以改善矿物油的肤感。

5..有机硅类:包括PEG-10聚二甲基硅氧烷、PEG-9甲醚聚二甲基硅氧烷、PEG/PPG-10/3油基醚聚二甲基硅氧烷、PEG/PPG-20/22丁基醚聚二甲基硅氧烷等。有机硅类和硅氧烷类要区分开来，不要搞混。有机硅类，是表面活性成分。因为都是硅元素类，所以归为G区


6.光甘草定(glabridin) :是光果甘草的提取物,在不产生细胞毒性的条件下对酪氨酸酶的抑制率可达50%,而且已证明比氢醌的抑制能力高16倍。光甘草定非常昂贵,号称“美白黄金”。（活性成分）


7.橄榄油(olive oil) : 榨取自橄榄果的天然油脂,主要含有油酸等成分,通常用作精油的基底油,也用于制作软质的皂类。橄榄油总体上是温和安全的,但也可能有致粉刺性。


8.甘草酸二钾( dipotassium glycyrrhizinate) :著名的抗敏成分,是甘草提取物的衍生物,易溶于水,有较低的pH,对油性皮肤亦有帮助。（抗敏修复类）


9.硅石粉(silica ) :也就是二氧化硅,可以调节产品的黏度,常用作填充料和遮盖成分、润肤剂的载体,可改善肤感。球形的硅石具有多孔性,可以吸附油脂,是一种惰性而安全的成分。（粉体类）


H:

1..红没药醇( bisabolol ):非常经典的抗炎、舒敏成分,提取自洋甘菊或者蓍草,常用于抗敏、抗炎的产品中。（活性成分）


2..黄原胶(xanthan gum) :一种淀粉的衍生物,有极高效的增稠性,颜色透明,温和无刺激,同时具有悬浮作用,可以帮助油脂均匀分散,是一种安全、常用的增稠成分。


3.黄芪提取物(astragalus root extract ) :提取自中药黄芪,具有良好的抗氧化、抗菌能力,常用于美白和抗衰老护肤品。（活性成分）

晶钻面膜解析案例

4.褐藻酸钠( sodium alginate):主要由来源于褐藻(如海带)中的褐藻酸制作而成是褐藻酸的钠盐。褐藻酸钠具有优异的稳定性、溶解性、黏性和安全性,在食品工业、制药工业都得到了广泛应用。化妆品中常用它作为增稠剂。


5.滑石粉(talc ) :来自矿物(硅酸镁),具有非常柔软和滑爽的手感,通常用于彩妆产品中作为填充料。（粉体类）


I


J:

1.胶原蛋白(collagen) :仅存在于动物皮、骨等组织中的重要结构蛋白,应用于护肤品中的一般是水解后的胶原蛋白或者生物发酵产生的胶原蛋白类产物,它们具有良好的生物相容性和保湿性,肤感很好,亦具有很强的吸水能力,故可用作保湿成分和肤感调理剂。


2.角鲨烯(squalene ) :最早提取于鲨鱼的天然油脂类成分,人类皮脂腺也可以自行分泌。角鲨烯具有优良的铺展性和亲肤性,低刺激,没有特殊的气味或颜色,配伍性好，但有致粉刺嫌疑。


3.鲸蜡醇(cetyl alcohol ) :即十六醇,又称棕桐醇,是一种长链脂肪醇,为白色蜡状固体,性质稳定,无特殊气味或颜色。鲸蜡醇是一种优良的多功能化妆品原料,可以作为润肤剂、助乳化剂、增稠剂、发泡剂等,可用椰子油或棕桐油为原料制造,也可以人工合成。


4.聚乙二醇类（PEG)：有PEG-10,PEG-100等多种类型,是一类透明、可吸湿、稳定性好的乙二醇聚合物,低分子量的可以用于保湿,高分子量的则可用于唇膏等较硬的产品。


5.聚乙烯醇( polyvinyl alcohol ):成膜剂,有黏性,可以与水结合,增加产品的稠度,多用于彩妆、指甲护理产品和撕拉式面膜。


6.聚甘油硬脂酸酯( polyglyceryl stearate ) :是聚甘油与硬脂酸的酯类,比较稳定,适应性广,无刺激,食品行业也有应用。常见的有聚甘油-10硬脂酸酯、聚甘油-3硬脂酸酯、聚甘油-10油酸酯等。


7.金盏花提取物( calendula/marigold extract):提取自金盏花的植物成分,有抗炎、舒缓作用,可用于敏感性皮肤和痤疮护理产品中。（抗敏修复类）


8.季铵盐类(quaternium ) :一类具有表面活性的抑菌剂,杀菌能力很强,但某些情况下也可以产生刺激。（防腐类）

磨皮霜成分作用解析案例

9.甲基异噻唑啉酮( methylisothiazolinone) :一种曾经比较流行的抑菌剂,低用量下就能起到极好的抑菌效果,但可惜安全性较差,因此欧盟已经禁用。上海市皮肤病医院的一项实验发现,该成分在测试的各种防腐剂中是不良反应率最高的。（防腐类）


10.己二醇(hexylene glycol/hexanediol) :即“馨醇H6" ,一种保湿剂,也可以作为溶剂,具有抑菌作用,安全性极佳（防腐类）


K：

1.矿脂(petrolatum) :即凡士林,为C16-C32的高碳烷烃(异构)和高碳烯烃的混合物,可用于乳液、膏霜、唇膏、发蜡中,也是各种药物软膏的主要基质成分。该成分是所有油相原料中封闭保湿性最强的,其质地较为黏稠,因此常和其他油相物质混合使用,以改善肤感。


2.抗坏血酸葡萄糖苷(ascorbyl glucoside) :维生素C的衍生物,具有良好的稳定性,但生物利用度也受到了影响,可算是效果和稳定性折中的产物。（活性成分）


3.抗坏血酸棕榈酸酯( ascorbyl palmitate) :也是维生素C的一种衍生物。（活性成分）


L:

1.芦荟(aloe ) :提取自芦荟叶肉,主要成分为芦荟多糖,可保湿、抗炎,并有一定抗菌性,能促进修复。


2.卵磷脂(lecithin) :主要提取自大豆和蛋黄,可能是最温和的乳化剂,亦可作为皮肤调理剂和抗氧化剂。


3.绿茶提取物(green tea extract) :绿茶提取物的有效成分是多酚类,已知可减少DNA损伤、日光炎症和红斑,是极强的抗氧化剂,同时具有美白、控油作用。绿茶提取物是一类天然安全的有效成分,缺点是颜色较深,一般人难以接受其棕黄色的外观。（抗敏修复类）


M:

1.明胶(gellatin ) :胶原蛋白的降解产物,具有良好的水溶性,黏度高,因而也可以起到增稠作用。明胶的来源是动物的皮、骨。


2.米糠油( rice bran oil/acid ) :来自大米米皮,含有多种不饱和脂肪酸,具有润肤作用,据说可以帮助抗衰老和修护皮肤屏障。


3.咪唑烷基脲(imidazolidinyl urea) :应用最广的防腐剂之一,通常与其他防腐剂配合使用以提升防腐效果,在高温下可以释放出更多甲醛。在各种甲醛释放型防腐剂中,咪唑烷基脲是相对容易引起过敏的。（防腐类）


N:

1.尿素(urea) :是皮肤天然代谢的产物之一,也是一种“真正”的保湿剂。不同于一般的吸湿剂,它可以使角质层保持合适的水分含量,同时可以帮助打通细胞的水通道,还可以促进角质的剥落,亦可以促进其他营养成分的吸收。


2.凝血酸(tranexamic acid) :即氨甲环酸,台湾地区称“传明酸" ,是一种具有止血作用的美白成分,常用于黄褐斑皮肤的护理。（活性成分）


3.尿囊素(allantoin ) :最早提取自紫草科植物的根,有抗炎、舒缓和促进伤口愈合的作用,对痤疮亦有改善作用。（抗敏修复类）


4.尼泊金酯类(parabens) :是最古老的防腐剂之一,同时也非常温和,准许使用的包括羟苯甲酯(methyl paraben)、羟苯乙酯(ethylparaben)、羟苯丙酯propylparaben)、羟苯丁酯(butylparaben)等,其中羟苯丁酯可以抗真菌。近年来研究者对其安全性有一些争议,但美国CIR (化妆品成分评估)认为目前为止的证据表明在化妆品中使用这些成分是安全的。（防腐类）


5.柠檬酸(citric acid) :一种果酸(a-羟基酸),常用于pH调节和辅助防腐一般情况下对皮肤安全,但对于敏感、破损的皮肤可能形成刺激。（其他辅助类成分）

P:

1.膨润土(bentonite ) :是一种硅酸铝盐,用于调节化妆品配方的黏度和悬浮性能,具有稳定配方功能。膨润土可以吸水形成胶状物质,不致粉刺,在彩妆产品中应用较多。


2.葡糖酸锌(zinc gluconate ) :易溶的含锌原料。锌是一种对皮肤很重要的离子,具有抗炎、舒缓作用,也可帮助抑制皮脂。葡糖酸锌常用于过敏、痤疮、敏感特应性皮炎等皮肤问题适用的护肤品中。（活性成分）


Q：

1.氢化蓖麻油(hydrogenated castor oil) :蓖麻油有一定气味,难于使用,将其做加氢处理后,颜色变得更白,性质更稳定,气味也得到改善。氢化蓖麻油是一种温和的油脂,能起到润肤作用。


2.氢化橄榄油(hydrogenated olive oil) :与氢化蓖麻油类似。


3.青咖啡果提取物( coffee fruit extract):是一类抗衰老物质,含有大量多酚绿原酸,具有很强的抗氧化能力,能提升皮肤弹性,在护肤领域很有前景。（活性成分）


4.羟（qinag读三声）丙基四氢吡喃三醇(hydroxypropyl tetrahydropyrantriol ) :即“玻色因” ,欧莱雅旗下著名的抗衰老成分。（活性成分）


5.a一羟基酸类(alpha-hydroxy acids, AHAs) :又称果酸,常见的有羟基乙酸(glycolic acid)、乳酸(lactic acid)等,最常用作角质松解剂（去角质剂) ,但是作用不限于此。AHAs (羟基乙酸、柠檬酸、乳酸)还可以促进真皮年轻化。在医学美容领域,高浓度的a-羟基酸类用于换肤。（活性成分）


6.曲酸(kojic acid) ：一种来源于不同的真菌(如曲霉和青霉)的美白成分,可抑制酪氨酸酶的活性,有一定刺激性和致敏性,但对于不耐受氢醌治疗的人很有价值。（活性成分）


7.氢氧化钠(NaOH) :酸碱调节剂,用于调节pH,使过低的pH升高到适当范围。（其他辅助类成分）氢氧化钾(KOH) :同NaOH的作用类似。（其他辅助类成分）

根据防晒成分分析案例


首先，得知道什么是表面活性剂。定义：凡是溶于水能够显著降低水的表面能的物质称为表面活性剂（SAA)或表面活性物质。表面活性剂(surfactant)，是指加入少量能使其溶液体系的界面状态发生明显变化的物质。


其实，我看了这个定义也是云里雾里的。但是，通过推测，应该是加了活性剂之后，护肤品整体的状态发生了变化。比如加活性剂之前可能比较硬，加了之后就软化了。也可能，加之前像水一样，乱流，加了之后，就粘稠一些，易于涂抹。


保湿增稠是什么意思？保湿应该不用多讲，就是使得皮肤变得水润，减少干燥起皮。那么，增稠呢？


增稠是使悬浮于液体中的固体粒子变为浓稠的拢浆而从液体中分离出来的操作即称为增稠，也称为沉淀浓缩或沉降浓缩。这是增稠的定义，说的太严禁了，以至于小编看不懂。不过简单来讲，应该就是从液体里分离出某些成分。比如说酒精和酒的混合物，如何提炼出更加纯的酒精呢？那么可以用蒸馏的方法，只不过蒸馏是物理方法，这里的增稠剂是化学方法。


活性成分又称有效成分，植物活性成分指构成植物体内的物质除水分、糖类、蛋白质类、脂肪类等必要物质外，还包括其次生代谢产物 （如萜类、黄酮、生物碱、甾体、木质素、矿物质等）。这些物质对人类以及各种生物具有生理促进作用，故名为植物活性成分。
