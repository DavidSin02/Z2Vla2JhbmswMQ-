“老胃病”反复发作，医生提醒早诊断早治疗

2020-11-14
| 科学

&nbsp; &nbsp; &nbsp;

&nbsp; &nbsp; &nbsp; &nbsp;.胃十二指肠溃疡患病十年，反复发病、反复住院，近日，六旬“老胃病”患者久病成医、有感而发，写下了近3000字自述病史，提醒大家听医生的话，远离疾病困扰。

&nbsp; &nbsp; &nbsp;【牟大爷：我是多年溃疡病的老患者了，对得病有点心得，有点体会，我就回忆了自己得病的历程，把它整理下文字材料。】

&nbsp; &nbsp; &nbsp;近日，年过六旬的“老胃病”牟大爷因十二指肠球部溃疡合并上消化道出血，再次住进了哈市红十字中心医院普内科。这几天，在医护人员的精心治疗与护理下，牟大爷病情好转，于是他开始在病房创作，将这次的住院经历写下来。

2011年牟大爷就因十二指肠球部溃疡合并上消化道出血在医院住院治疗过，这些年，他也多次因此病住院治疗。根据自身的患病经历，牟大爷反思了自己患病的原因。

&nbsp; &nbsp; &nbsp; 【牟大爷：住院在五次以上，反复发作应该有十次了，主要第一点就是我生活没有规律，暴饮暴食、抽烟喝酒、打麻将，这些缺点我都有，麻将打得特别厉害，有时中午饭、甚至晚饭都不吃，一直玩到二半夜，爱发脾气、精神紧张，普内科医生说，这些都是溃疡病病人发病的原因。】

&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;

&nbsp; &nbsp; &nbsp; &nbsp;最近的一次住院是在今年的十月九号，胃镜检查的结果还是溃疡病发作。哈市红十字中心医院普内科主任李晨玲告诉牟大爷，如此反复发作会局部形成瘢痕，胃的出口和十二指肠会变形，影响食物通过，预后不良！医生的话警示了牟大爷，他再也不敢大意了，这不，牟大爷想了想这些年的病情发作，写出了患病心得进行反思。

&nbsp; &nbsp; &nbsp; &nbsp; 针对牟大爷的情况，医生提醒，消化性溃疡需要重视，临床上许多患者出现反复上腹部疼痛，做胃镜发现是消化性溃疡。当病情比较重或者治疗不及时、长期反复发作，有出血、穿孔、梗阻，甚至癌变的风险。患者应戒烟、酒、茶、辣椒、生冷刺激等伤及胃黏膜、影响脾胃功能的食物，不要暴饮暴食，每餐七分饱。不良的情绪也是消化性溃疡的病因，因此要注意心平气和，平时可以通过散心、转移注意力排解不良情绪。此外，每到秋冬季，很多人借着“抓秋膘”“驱寒”等理由，吃一些高热量的食物，但对于消化性溃疡患者而言，这些食物就很有可能会诱使疾病复发。

&nbsp; &nbsp; &nbsp;【李晨玲：如果有症状当然要及时就诊，没有症状也要定期做胃肠镜检查，这是非常好的早期的进行诊断和治疗最有效的办法。】
