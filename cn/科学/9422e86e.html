气候变化感冒来袭，怎么给小儿选药“拿得准”，让感冒随风而散

2020-11-14
| 科学

气候渐入深秋，寒温失常使得人们容易感冒，尤其是小儿，感冒在西医称为急性上呼吸道感染，中医认为是感受了外邪引起的肺系疾病，所谓外邪，冬春季节主要是指风寒、风热；所谓肺系疾病，指的是以发热、恶寒、鼻塞流涕，咳嗽为特征的病症。

而面对种类繁多的感冒药，怎么才能“拿得准”，不让病情进一步发展呢？

问题的关键是要根据风寒、风热的不同来选药，风寒选择辛温解表的药，风热选择风凉解表的药，弄反了可就犯了“寒寒热热”的错哟！

话虽难懂，道理很俗，打个比喻：弄反了就像冬天里吃冰，夏天里烤火一样。

风寒感冒

风寒感冒的特点：发热，恶寒，无汗，咳嗽，鼻塞，流清涕，口不渴，咽不红。

代表药物--风寒感冒颗粒

成分：麻黄、葛根、桂枝、防风、紫苏叶、白芷、桔梗、苦杏仁、陈皮、干姜、甘草。

功效：解表发汗，疏风散寒。

{ 注意事项 }

1、忌烟、酒及辛辣、生冷、油腻食物。

2、不宜在服药期间同时服用滋补性中成药。

3、风热感冒者不适用，其表现为发热重，微恶风，有汗，口渴，鼻流浊涕，咽喉红肿热痛，咳吐黄痰。

4、糖尿病患者及有高血压、心脏病、肝病、肾病等慢性病严重者、孕妇或正在接受其它治疗的患者，均应在医师指导下服用。

5、服药三天后症状无改善，或出现发热咳嗽加重，并有其他严重症状如胸闷、心悸等时应去医院就诊。

6、按照用法用量服用，小儿、年老体虚者应在医师指导下服用。

7、对本品过敏者禁用，过敏体质者慎用。

8、本品性状发生改变时禁止使用。

【相似品种】

感冒疏风片：1gX12片X2板/盒 麻黄绒、桂枝、苦杏仁、白芍（酒炙）、防风、紫苏叶、独活、桔梗、谷芽（炒）、生姜（捣碎）、大枣（去核）、甘草。

自制葱豉汤：

①配方葱白2根，豆豉10克。

②制法用水500毫升，入豆豉煮沸2～3分钟，之后加入葱白、调料出锅。

③功效解表散寒。

④用法趁热服用，服后盖被取汗。

风热感冒

风热感冒的特点：发热较重、咳嗽、咳痰、鼻塞、流黄涕、口渴，咽喉红肿疼痛。

代表药物——小儿风热清颗粒

成分: 山银花、连翘、板蓝根、薄荷、柴胡、牛蒡子、荆芥穗、石膏、黄芩、栀子、桔梗、赤芍、芦根、苦杏仁（炒）、淡竹叶、枳壳、炒六神曲、僵蚕、防风、甘草。

功效：辛凉解表，清热解毒，止咳利咽。

【相似品种】

余麦口咽合剂 100ml/瓶 余甘子、地黄、赤芍、麦冬、甘草。

感冒清热颗粒 12g*10袋/盒 荆芥穗、薄荷、防风、柴胡、紫苏叶、葛根、桔梗、苦杏仁、白芷、苦地丁、芦根。

桑菊感冒颗粒 桑叶、菊花、薄荷、桔梗、苦杏仁、连翘、芦根、甘草

桑菊感冒片：12片*2板/盒 桑叶、菊花、连翘、苦杏仁、桔梗、芦根、薄荷素油、甘草。

感冒炎咳灵糖浆：100ml/瓶 一点红、一枝黄花、板蓝根、甘草、生姜。

风热咳嗽胶囊 0.32g*20粒/盒 桑叶、菊花、薄荷、桔梗、黄芩、苦杏仁霜、连翘、前胡、枇杷叶、浙贝母、甘草。

小儿感冒颗粒：12g*6袋/盒 广藿香、菊花、连翘、大青叶、板蓝根、地黄、地骨皮、白薇、薄荷、石膏。

风寒风热分不清，选择无明显偏性的药物

代表药物——四季抗病毒合剂

成分：鱼腥草-连翘-菊花、荆芥-紫苏叶、桑叶-杏仁-桔梗、甘草。

功效：清热解毒，消炎退热。

适应症：用于上呼吸道感染，病毒性感冒，流感等病毒性感染疾患，症见头痛，发热，流涕，咳嗽等。

{ 注意事项 }

1．忌烟、酒及辛辣、生冷、油腻食物。

2．不宜在服药期间同时服用滋补性中药。

3．高血压、心脏病、肝病、肾病等慢性病严重者应在医师指导下服用。　

4．发热体温超过38.5℃的患者，应去医院就诊。

5．儿童、年老体弱者应在医师指导下服用。

6．服药3天症状无缓解，应去医院就诊。

7．对本品过敏者禁用，过敏体质者慎用。

8．本品性状发生改变时禁止使用。

9．如正在使用其他药品，使用本品前请咨询医师或药师。
