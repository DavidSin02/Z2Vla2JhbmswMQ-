保险到底保什么，你知道吗？

2020-11-14
| 科学

大家好我是朴松，这篇主要聊一下人身保险所保的东西，也就是“保险标的(dì)”：人的健康与生命。

保险分两种，财产险和人身险。财产险保障的是财物的完好，而人身险保障的是人的健康与生命。

前者大家都有具体的认识，财物是有形的、有价的，而对后者的认识就比较模糊，因为人们只有生病的时候才会意识到健康可贵。


看完这篇文章，相信你对人体的结构、系统会有初步认识，从而知道人身险保的到底是什么。

先上一张图[1]感受下：这张图表现人体结构的层次。人体的基本单位是细胞，细胞构成组织，组织构成器官，以此类推，最终构成人体。


说到人的健康与生命，就不得不提医学。不同于其他学科专注于外在世界，医学专门探索人体内的世界，因而有独特的视角和魅力。



保险有两大重要内容：核保、理赔，其中有很多西医知识，要了解医学就必须了解形态学。西医在全世界普及的重要原因之一，就是立足于形态学的基础。

什么是形态学？简单讲，就是一个东西，它长什么样，长的像什么，乍一听有点好笑，一个东西长什么样，还能成为一门学问？别说，它还真是一门学问，从公元前300多年，西医奠基人、古希腊名医希波克拉底就开始研究和描述心脏的形态，迈出了人类医学的一大步。


如果你还是觉得“形态学”太抽象，就记住那张层次图，那就是形态学。

形态学这方面，大家只需先记住人体有八大系统：运动系统、消化系统、呼吸系统、泌尿系统、生殖系统、脉管系统、神经系统、内分泌系统。


这么多系统，是不是很难记？教你一招助记：

脉管，运动，消化，内分泌，生殖，呼吸，神经，泌尿

简化：脉动化泌，殖呼神泌

再加个谐音：“脉动”花蜜，直呼神秘

——脉动哪天要是出了蜂蜜口味的，我直呼内行，第一个买来尝尝。怎么样，是不是很好记了？


八大系统比较抽象，中国人讲五脏六腑。五脏我们再熟悉不过了：心肝脾肺肾。心、肝、肺、肾分别是脉管系统、消化系统、呼吸系统和泌尿系统的重要器官。其他器官有机会给大家一一介绍。

回到最初的问题，人身险保的是什么，或者说保险责任是什么，就十分清楚了：人体系统在某个层次出现的问题。

举几个例子，层次从小到大，这些问题可能发生在分子（基因）层次，造成所谓的“分子病”，最典型的是地中海贫血（ICD-10 code D56.151）；也可能发生在生殖系统的组织层次，比如宫颈原位癌（ICD-10 code D06.901），也可能发生在消化系统的器官层次，比如胃溃疡（ICD-10 code K25）；还可能发生在泌尿系统层次，比如泌尿系结石（ICD-10 code N20.901）。

有同学问括号里的英文是干嘛的，ICD-10就是保险公司用来审核保险责任的、由世界卫生组织出版的、全球通用的疾病分类手册（第10版），后面的字母+数字就是具体疾病在这本手册中的编码，以后只要谈到具体的疾病，我都会附上编码。至于ICD编码对于保险的重要性，以后跟大家好好聊聊。

总之，无论问题发生在什么层面，最终导致健康和医疗费用的损失，甚至死亡，这些即属于人身险的保障范围。


当然，光谈保险责任不谈责任免除是不负责任的，关于责任免除，今后会单独拿出来讲。

人们往往过于关注外界事物，反而对自身缺乏认识，雅典神庙上镌刻的一句箴言回响千年：“认识你自己”。



现在知道保险到底保什么了，以后跟人聊保险，除了说保“生老病死”，你也能从医学的角度哈拉两句。
