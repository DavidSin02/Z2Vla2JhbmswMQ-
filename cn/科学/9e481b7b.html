城市线性空间营造

汇集与连通——旧金山南城的弹性雨洪管理

地点：美国 加州

设计者：HASSELL+事务所

据worldlandscapearchitect.com讯南旧金山（South San Francisco）位于圣马刁县（San Mateo County），被称作海湾 “工业城”。其主要的高速公路和铁路在连通市中心与该地区的同时，也割裂了该地区与海湾之间的联系。在过去的半个世纪里，该地区的部分社区遭遇洪水泛滥，阻碍了当地工业的发展。如今，包括南旧金山整片海湾区在内的圣马刁县，正面临着海平面上升和地震灾害的风险。HASSELL+事务所为此制定了“汇集与联通”策略，从雨水管理、弹性生态、社区公共开放空间、紧急避灾场所四个方面出发提出弹性网络方案。在方案中，小溪与街道可以转变为雨洪管理与社区雨水汇集的线性廊道，将区域水管理结构从脆弱的环形线路转变为弹性网络，同时，该方案还将为社区带来更多新的城市公园与多功能公共活动场所，提高该区域的恢复力，改善该区域居民的生活品质，为南旧金山海湾地区带来利益。

（冯玮 / 编译）

© https://worldlandscapearchitect.com/collect-connect-resilient-south-city-south-san-francisco-united-states-hassell/#.XSBpHC274Xq

梅尔滨河公园

地点：美国俄亥俄州 辛辛那提市

设计者：Sasaki事务所

据worldlandscapearchitect.com讯梅尔滨河公园（Smale Riverfront Park）位于辛辛那提市中心俄亥俄河（the Ohio River）沿岸，占地约13hm²。

滨河公园不仅为其周边罗布林大桥（the Roebling Bridge）、保罗·布朗体育场（the Paul Brown Stadium）等众多城市地标营建了绿色背景，公园中的滨河长廊、表演舞台、互动水景与儿童活动场地，还为人们提供了聚会、休闲等活动空间，吸引游客并带动区域综合发展。另外，公园的建设完善了俄亥俄州的慢行体系，重塑辛辛那提市中心与俄亥俄河的空间体验。

（贾绿媛 / 编译）

© https://www.gooood.cn/shenzhen-landscape-innovation-of-mixc-world-china-by-canggula-landscape-planning-and-design.html

BIG事务所提出布鲁克林皇后区高速公路更新方案

地点：美国 纽约

设计者：BIG事务所

据worldlandscapearchitect.com讯 布鲁克林皇后区高速公路（The Brooklyn Queens Expressway）由罗伯特·摩西（Robert Moses）于20世纪50年代建造，是纽约市标志性基础设施，已为市民提供交通便利60余年。由于该高速公路部分结构已经出现老化现象，纽约市开始征集重建方案。为此，BIG事务所设计了一套综合方案：建造一条铺有支撑平台结构的道路，平台上的空间设计为新城市公园，并可以为布鲁克林轻轨线路提供空间。平台下的道路则依旧承担其原本的交通功能。该方案在能够满足该区域交通需求的同时，连通布鲁克林社区与滨海地区，实现在布鲁克林城市、高速公路、河流三者之间无缝交织，用简单的结构改造方法，为布鲁克林创造了成本低、可行性强、能为社区带来更多利益的重建方式。

（冯玮 / 编译）

© https://worldlandscapearchitect.com/big-reimagine-the-brooklyn-queens-expressway-as-bq-park/#.XSBpHC274Xp

作为城市项链的江阴绿道环路

地点：中国 江苏

设计者：BAU彼爱游建筑城市设计咨询有限公司

据worldlandscapearchitect.com讯江阴绿道一期北起体育中心，南至永安绿苑，总长约5km的新“绿廊”蜿蜒而行。环路为人们提供了干净舒适的环境与良好的城市空间体验。绿道由四个具有典型特色的区域组成，全程没有红绿灯，架空步道与地面绿道跨越市政道路和东横河无障碍贯通。灯光、WiFi、自行车驿站、指示牌，设施一应齐全，一路揽尽无限风光。整个环城步道采用智慧化的设计元素，如嵌入式隐蔽设计的扶手灯、兼具美观与舒适性能的彩色沥青跑道等。

然而这只是一个开始，未来江阴绿道将串起各个公园，形成环城公园，塑造“城在绿中、人在景中”，生态宜人的城市特色空间。

（郑琦蕾 / 编译）

© https://worldlandscapearchitect.com/bau-creates-an-urban-necklace-in-jiangyin-china/#.XQHxsPZuJPY

洋泾港步行桥

地点：中国 上海

设计者：刘宇扬建筑事务所

据gooood.cn讯洋泾港步行桥位于上海市杨浦大桥旁的洋泾港，西起民生艺术码头，东至洋泾绿地公园，总长度140m，为贯通浦东东岸开放空间的第一座慢行桥梁。

步行桥主体部分采用异型桁架结构，利用高差划分出供骑行、跑步、慢步等使用的慢行空间，以保证通行安全。同时，主桥体优雅的曲线造型与周边景观相呼应，并为慢行体验者营造瞭望向杨浦大桥和西南侧陆家嘴中心建筑群的良好观景视野。步行桥的设计是将结构、功能与造型三者相结合的典范。

（贾绿媛 / 编译）

© https://www.gooood.cn/yangjing-canal-pedestrian-bridge-china-by-atelier-liu-yuyang-architects.htm

荷兰“Ω”型景观桥

地点：荷兰 乌特勒支

设计者：NEXT建筑事务所, 鲁迪建筑事务所

据archdaily.com讯由NEXT建筑事务所与鲁迪建筑事务所（Rudy Uytenhaak）等多家事务所共同设计的景观桥（Dafne Schippers）自2017年建成以来广受市民喜爱，成为连接市中心与新区的绿色景观连廊，将步行空间、自行车道组成的慢行空间与公园和学校融合在一起，形成了一个具有整体性的绿色基础设施项目。

该桥全长110m，横跨丹莱茵运河，桥面最低处距水面9m，最高处距水面35m，通过悬索挂于两座不对称的塔架间，一侧塔架高大，另一座靠近学校的塔架相对体量小，掩映于树列中。大桥可容纳7000余人通行，极大缩短通行时间。

（黄楚梨 / 编译）

© https://www.archdaily.com/919265/dafne-schippersburg-bridge-next-architects?ad_medium=gallery

圣菲利普人行桥

地点：英国 布里斯托尔

设计者：奈特建筑事务所

据archdaily.cn讯圣菲利普人行桥横跨了英国布里斯托尔的埃文河，连接了埃文河两岸不同的高度、风貌的建筑。新步行桥的落成将会改善布里斯托尔火车站前面的庙岛的交通可达性，也为该区域提供了新的发展机遇。

这座钢结构步行桥呈现叉形的平面形态，为行人提供了坡道与楼梯，以满足不同人群的多重需求。桥的两端呈现完全不同的面貌，西边是一堵高大的石墙；东边是一个低矮的天然绿色斜坡，步行桥完美消化了两侧的高差并使其成为一个整体。

（王子尧 / 编译）

© https://www.archdaily.cn/cn/920306/sheng-fei-li-pu-ren-xing-qiao-knight-architects?ad_name=article_cn_redirect=popup

深圳华润万象天地的景观创新

地点：中国 深圳

设计者：仓谷景观规划设计事务所

据gooood.cn讯华润置地大厦由英国Foster+Partners（福斯特）建筑设计事务所担纲整体规划设计，旨在打造成为中国具有昭示性和影响力的商务群落。仓谷景观参与其中华润城前脸-深南大道高新园段片区景观设计。设计者结合上层规划理念提出了缎带及电路板的概念，充分利用了线性的景观设计元素，以场地原有绿色基底为依托，运用多种立体的线性元素雕塑地形，形成带状上扬结构和条带状种植。

（王子尧 / 编译）

© https://www.gooood.cn/shenzhen-landscape-innovation-of-mixc-world-china-by-canggula-landscape-planning-and-design.html

风景园林与艺术交互展示

花园墙

地点：澳大利亚 墨尔本

设计者：雷塔拉克·汤普森事务所（Retallack Thompson）及其他合作建筑事务所

据archdaily.com 讯由雷塔拉克·汤普森事务所及其他合作建筑事务所共同打造的花园墙项目（Garden Wall），将墨尔本维多利亚国家美术馆的室外空间划分为一系列独特的小花园。墙体由网状编织的白色框架组成，与雕塑、景观设施、植物和道路结合，共同形成了独特的感官体验。人们在若隐若现的墙体中穿行，感受着光影与时间的变化。

墙体是最基本的建筑形式。设计师通过对花园空间的分割和重组，以及对路径的处理，使花园式的墙体成为讨论建筑政治维度和边界问题的介入点。

（张沚晴 / 编译）

© https://www.archdaily.cn/cn/918732/hua-yuan-wei-qiang-zhan-lan-guan-jiang-hua-yuan-yin-ni-zai-ban-tou-ming-de-qiang-hou-retallack-thompson-and-other-architects © https://retallackthompson.com/garden-wall/

双层展园

地点：澳大利亚 墨尔本

设计者：穆尔建筑事务所（Muir Architecture）和Openwork景观工作室（Openwork）

据archdaily.com讯“双层展园（Doubleground Pavilion）”是2018年墨尔本维多利亚国立美术馆（NGV）建筑协会展出的作品，该项目由穆尔建筑事务所和Openwork景观工作室共同完成，希望在公共场所寻求一种具有挑战性的介入。人们对建筑的记忆不是由平面或是立面形成的，而是由透视形成的。这一观点促使了花园草坪里折叠墙的诞生，用以重新谱写原维多利亚国立美术馆的建筑画像。设计师采用了拼贴的几何语言，通过限定类似房间、走廊以及场地等主体空间，在公共花园中形成了承担类似建筑的角色的景观。

花园独特的设计形式为公共空间带来更多可能性，也展现了新的建筑语言。

（张沚晴 / 编译）

© https://www.archdaily.com/916967/doubleground-pavilion-muir-plus-openwork

美国梦之柱展览装置

地点：美国 北卡罗来纳州 夏洛特市（Charlotte, NC）

设计者：MARC FORNES / THEVERYMANY设计工作室

据 designboom.com讯“梦之柱（Pillars of Dreams）”是一个轻质的地标式展览亭装置，它看似是由空气填充而成的，实则是由一个连续的超薄铝制结构表皮围合并支撑起来的。该装置的表皮共有两层，白色外表皮隐藏了其内部丰富的颜色。内部的蓝色透过白色外表皮上的开口显露出来。从远处看，整个展览装置结构呈现出一种柔和的色调，但是人们仍然可以看到其内部渐变的色彩。街道上的行人会被其独特的外观吸引，但只有当他们走近，才能了解这个结构，并看到颜色逐渐变深，最终被空间内丰富的色彩所包围。

（郑琦蕾 / 编译）

© https://www.designboom.com/architecture/marc-fornes-theverymany-pillars-of-dreams-05-30-2019/

极小曲面金属空间装置

地点：中国 南京

设计者：刘祥茹、范文、李轩昂、张晋鲁、陈星光、李晓晋、李晓婷、胡萍

据archdaily.com讯2019年南京艺术学院设计学院毕业展主题为“实验·认知”，于5-6月进行集中展览。复合雌蕊馆便是其中一项利用数字化运算制成的极小曲面的金属空间装置。小组成员将两种花的形式整合到设计中，并通过计算机创建“双核”虚拟形式。

通过数字化运算后，利用块切割机，完成397块0.3mm厚的镜面不锈钢板的制作，并通过铆钉连接2200个孔。装置形成了多个流线的环形空间，会跟随昼夜不同的光照情况以及夜间照明补光的不同情境产生极具趣味性的空间与视觉的体验。

（王子尧 / 编译）

© https://www.archdaily.com/918895/the-compound-pistil-pavilion-nanjing-university-of-the-arts
