南昌起义 中共前敌委员会成员和军、师党代表都有谁？

| 科学

​​

南昌起义主要领导人——中国共产党前敌委员会书记周恩来 ;

南昌起义中国共产党前敌委员会成员——李立三 ;

​

南昌起义中国共产党前敌委员会成员——彭湃 ;

​

南昌起义中国共产党前敌委员会成员——恽代英

​

第九军党代表朱克靖 ; 第十一军党代表聂荣臻 ; 第二十四师党代表颜昌颐 ; 第二十五师党代表李硕勋 ; 第二十军党代表廖干吾 ; 第一师党代表方维夏 ; 第二师党代表陈　恭 ; 第三师党代表徐特立 。

时任参谋团秘书长李一氓还回忆说:

南昌起义，师党代表还应加阳翰笙（即欧阳继修）和梅龚彬，当时他们都是师党代表 ;我南昌起义时当参谋团秘书长（参谋团参谋长是刘伯承）。

​

关于参谋团，周恩来曾回忆说：“参谋团主任不是我，是伯承同志。开始，参谋团没有人任参谋长，我就指定刘伯承同志做，起初他谦虚不肯答应，后来我说，一定要你做，他才担任这个职务。”与李一氓同志的回忆是一致的。

李一氓同志(1903年-1990年)，是老一辈无产阶级革命家，马克思主义理论家。他出生于四川省彭州市东大街，1925年加入中国共产党。早年赴法国勤工俭学，追求革命真理。在国内革命战争时期，李一氓同志曾任国民革命军总政治部宣传部科长，南昌起义参谋团秘书长。他参加过红军二万五千里长征，并先后任陕甘宁省委宣传部长，新四军秘书长。抗战胜利后，李一氓同志先后任苏北区党委书记，华中分局宣传部长，大连大学校长等职。李一氓同志是马克思主义理论家，二、三十年代他翻译出版了《马克思与恩格斯合传》、《马克思论文选译》及马克思的《哲学之贫困》等著作。

​

同时又是中国新文化运动先驱者，他是中国左翼文化界总同盟的负责人之一，1928年3月同阳翰笙共同编辑《流沙》。

​

新中国成立后，曾任中国驻缅甸大使，国务院外事办副主任，中联部副部长，中纪委副书记，中顾委常委等。
