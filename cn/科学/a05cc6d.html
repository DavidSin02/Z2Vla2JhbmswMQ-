SolidWorks文件批量转换_低版转高版

2020-10-29
| 科学

1、文件转换

当我们在升级SoildWorks到高版本的时候，其文件的内部结构也随着更新了，即它可以正确打开以前的低版本文件（向下兼容），但不支持低版本打开高版本的文件（向上兼容）。同时，如果我们的设计文件有大量的低版本文件的话，那在用高版本软件打开时，SolidWorks将在内存中把文件的结构和版本转换为当前版本，并在保存时写入硬盘，如果须打开的低版本文件比较多或打开老版大型装配体的话，这将大大增加加载时间。为此，SolidWorks提供了SolidWorks Task Scheduler来对文件和任务进行批量操作。

SolidWorks Task Scheduler——它独立于Solidworks运行，可以按指定的方式在指定时间完成批量任务。

利用任何 SolidWorks 许可，您都可以排定转换文件和分解文件任务。 要排定大多数其它任务或排定已分发的文件转换，您必须具有 SolidWorks Professional、SolidWorks Premium 或 SolidWorks Office 许可。

2、文件批量转换实例

SolidWorks Task Scheduler中的文件转换只是一个代替人工重复操作的批量处理工具，本质上还是会通过启动solidworks依次打开、保存、关闭文件，然后关闭solidworks。所以，这是一个非常原始的方式，减少人工的重复劳动。

步骤1 打开一个低版本文件并关闭所有插件

随意打开一个旧版本文件，可在标准工具栏上看到旧版本文件的提示，然后单击【工具】/【插件】，关闭所有插件以减少加载时间，如下图所示。

不保存，关闭零件和SolidWorks。

步骤2 打开SolidWorks Task Scheduler

从Windows开始菜单中，单击【所有程序】/【SolidWorks<版本>】/【SolidWorks工具】/【SolidWorks Task Scheduler】；对于win10可能是单击【开始】/【SolidWorks<版本>】/【SolidWorks Task Scheduler】，如下图所示。本例采用2013sp5。

步骤3 打开转换文件窗口设置转换选项

在【SolidWorks Task Scheduler】对话框中单击左上角的【转换文件】或菜单单击【任务】/【转换文件】，然后单击【添加文件夹】，选择要转换文件的目录和相关选项，如下图所示。

步骤4 排定任务并转换

采用当前时间，单击【转换】窗口下方的【完成】开始排定任务，如果当前没有其他任务，将立即开始文件转换，如下图所示。

在转换过程中SolidWorks会以缩小窗口的形式多次打开和关闭，如下图所示。

注：1、SolidWorks Task Scheduler中的其他任务的设置与文件转换类似，如任务不能执行，可尝试重装软件或更换系统。

2、关于SolidWorks文件版本间的兼容问题，从2012版开始，SP5版本可打开后续一个主版本的文件，但打开后详细特征设计树，只有一个输入特征。如2012sp5可打开2013SP0-SP5，不能打开2014及以后的，两个主版本必须连续。
