八种几乎绝迹的野生文玩核桃，个个有故事

2020-10-29
| 科学

玩文玩核桃的朋友们，都希望能收藏到一对品相绝佳的野生核桃，在和友人交流的时候还能稍稍显摆一下。然而野生核桃种类繁多，能被大家熟知的却并不多，有些小众品类也几近绝迹，咱们今天来说的就是基本上已经绝迹的九种核桃，快看看您手里是否有一对儿？

一、通天塔

品种：楸子 / 产地：黑龙江 / 称号：通天塔

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：原名粗筋楸子，2013年被毁。

二、金蟾绣花鞋

品种：楸子 / 产地：辽宁丹东 / 称号：金蟾绣花鞋

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：辽宁省楸子名品稀少，能被野生核桃爱好者认可的，其中之一就是被核桃发烧友所津津乐道的沈阳鸡心。而能被楸子爱好者认可并且奉为上品的仅仅就两种，其中一款就是丹东的绣花鞋绝品。

2013年辽宁丹东一位楸子爱好者无意间得到此类单只，当时命名为佛光普照。后经这位丹东核友的寻找，终于收集到了一对，并且发在网上。此款楸子外形兼顾绣花鞋和金蝉的神态，浮雕凸纹，沉水，是一款值得收藏的楸子品种。

三、四座楼

品种：野生麻核桃 / 产地：北京平谷熊耳寨乡 / 称号：四座楼

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：四座楼狮子头老树生长在一个叫牛峪的深山沟里，平谷仅此一颗老树。随着麻核桃价格一路飙升，据传被几户村民所毁。

四座楼不同于南疆之处就是大气且周正，这是野生麻核桃较少见的，四座楼属于百年老树，进入衰弱期后产量极低，愈发显得弥足珍贵。随着这三五年来，嫁接四座楼的大面积种植、上市，四座楼已然成为近几年麻核桃市场领跌的急先锋！

四、凤头灯笼

品种：灯笼 / 产地：秦岭 / 称号：凤头灯笼

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：秦岭垂垂老矣的神树，2015年是自12年后的再次结果。9月初下树，被人拾取，数天后知道此树的一行四人来到树前，发现果子已经被人拾走，愤然毁树。

五、蚂蚁纹

品种：秋子 / 产地：吉林 / 称号：蚂蚁纹

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：蚂蚁纹核桃成名较早，此品种被带至京城，因为筋脉不明显，花纹细碎接近传统的麒麟纹，是首先被麻核桃届广泛所认可的楸子品种。

自赴京一炮而红后，就传说不断。一说，蚂蚁纹树为吉林农村一兄弟所有，但是祸起萧墙，先被砍断一支，近年又报仅剩的一枝又被砍。据说，此树还存在发新芽枝的可能性。

六、门墩狮子头

品种：野生麻核桃 / 产地：不明，据说为天津蓟县 / 称号：（麒麟纹）门墩狮子头

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：传言野生老树生于天津蓟县，12年后就再少见。

七、龙骨

品种：楸子 / 产地：黑龙江 / 称号：龙骨

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：据称产自黑龙江和吉林交界的五常市沙河子林业局，其名字因酷似龙的骨架而来。一般长度在50-60毫米左右，2013年龙骨长度历史性的超过70毫米，传因为招哄抢引起争执，导致树被砍。

2013秋季的龙骨，大多被北京核商收购，市场已经难觅踪影。龙骨主筋瘦骨嶙峋，伴随凸刺纹，像蛟龙出水，又像飞龙盘山，上色快，是款值得楸子爱好者收藏的品种。龙骨分长桩、矮桩和圆形，以图中长桩为代表，分别为不同树的品种，其中图示长桩歪尖的也有称为鱼骨。

八、龙纹柳叶

品种：楸子 / 产地：黑龙江 / 称号：龙纹柳叶

（此处已添加小程序，请到今日头条客户端查看）

江湖传闻：柳叶类楸子中纹路最为粗壮、深邃、复杂的柳叶上品，也是高桩楸子的代表作。据闻于2015年，此树被砍。
