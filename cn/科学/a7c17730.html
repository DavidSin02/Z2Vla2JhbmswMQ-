三分钟读通10位经济学大师，了解马克思、凯恩斯、哈耶克的精华

2020-11-14
| 科学



亚当·斯密：现代经济学之父

我们期望的晚餐并非来自屠夫、酿酒师或是面包师的恩惠，而是来自他们对自身利益的特别关注。——亚当·斯密

他那本可能排名第一的经济学畅销书《国富论》中，最核心的观点就是：让人们追求个人利益将有助于整个社会。

亚当·斯密花了十年时间完成了《国富论》，并在1776年出版了该书。《国富论》被视为全面阐述世界经济理论的第一本书。这是一本为决策者准备的手册，旨在说明社会如何能取得商业成功。全书分五卷，第一卷写劳动生产力，以及各个不同社会阶级之间的劳动力如何分配的问题。第二卷写的是财富的积累。第三卷写的是不同国家如何按着不同的速度发展的问题。第四卷讨论了农业和商业体制的问题。第五卷，写的是国家如何通过税收来增加收入 和如何支出的问题。书中主要涉及六个方面的思想。

  1. “无形之手”其中包括个人自利的选择，形成了最好的经济成果，替代了计划经济。
  2. 劳动分工，根据人们的技能和装备，把任务分配给个人或组织，达到最高的产出水平。
  3. 如果任由企业主实施他们的计划，那么就会形成垄断，也就会抬高市场价格。
  4. 税收制度应该公平、可靠、方便、有效的。
  5. 政府应该用税收来保持经济良好运作，但基础设施的使用者，比如道路使用者，应该支付一定的费用。
  6. 禁止垄断的形成，因为那样会导致更高的价格。


大卫·李嘉图：从移民到绅士

李嘉图是亚当·斯密的迷弟，也是经济学家中的富翁。他去世时留下了4500万英镑的遗产。他闻名于世的经济理论主要是关于价值、租金、贸易及国家债券。虽然有些理论已经过时，但是200年前提出这些理论也是一种重要贡献。

  1. 比较优势：各国应专注于它们最有效的生产，并进口自己不擅长制造的产品。
  2. 李嘉图效应：随工资增长，资本密集型生产的产品，相对于劳动密集型的产品更便宜，因此，对前者的需求量会越来越大。
  3. 李嘉图等价理论：政府借此刺激经济增长是不会成功的，因为人们会意识到，为了偿还国债，他们不得不将来支付更高的关税。



卡尔·马克思：不仅是一位经济学家

各尽所能，按需分配。——卡尔·马克思

我们似乎都很熟悉马克思，他是共产主义的创始人。他不但是经济学家，更是哲学家、革命家和政治活动家。

马克思的三个关键思想：对劳动的剥削是资本主义的基础，资本主义播下了自我毁灭的种子，以及社会主义、共产主义制度最终将取代它。主要观点：

  1. 工人的劳动与劳动权的区别。
  2. 雇佣者通过利用劳动者的工资和他们的劳动价值之间的差距来累积剩余价值的观点。
  3. 资本家保持工资最低化以最大化自己的利润。
  4. 为了更为高效，资本家们不断增加能节省劳动力的机器的投资，以增加剩余价值。
  5. 这导致了“失业后备大军”的产生，其贫困水平不断增加将挑起阶级冲突。
  6. 最终资本主义将会播下自我毁灭的种子。


阿尔弗雷德·马歇尔：划时代的《经济学原理》

马歇尔是经济学家也是数学家，他写的《经济学原理》成了经济学专业学生的标准教材。书中清晰地解释了微观经济学的基本原理，包括供给和需求、边际效用、需求弹性等概念。

  1. 将数学引入经济学，并使其得到进一步发展。
  2. 供给和需求作为剪刀的两个刃，决定了价格。
  3. 图形可以作为经济分析工具和解释工具。
  4. 一个经济互动的局部平衡分析，取决于经济的其他方面处于其他条件不变的情况下。
  5. 给出了边际主义的概念，并对成本或效用对行为的影响进行分析。
  6. 提出了工业区、内部经济和外部经济，这促进了公司发展。



约翰·梅纳德·凯恩斯：升、跌、升、跌

凯恩斯提出在经济衰退时期，国家有必要进行干涉的理论。凯恩斯算是富二代，其父亲曾担任剑桥大学教务长。自己也曾担任过英国财政部驻巴黎和会代表，出版了畅销书《和平的经济成果》，几乎预言了希特勒的崛起和第二次世界大战。凯恩斯引入了“消费倾向”“投资诱因”和“资本的边际效率”等新概念。以下是我们需要关注的要点：

  1. 储蓄和投资并不总是趋于相等，因为它们由不同因素驱动，时间范围也不同。
  2. 产出可能急剧下降，导致许多人失去工作，降低薪酬无法解决问题。
  3. 经济中的总需求在决定总产出时起到了关键作用，所以政府可以采用税收和支出方式，增加需求，改善经济衰退时期的就业。
  4. 人们的不确定性和对未来的预期是经济周期的主要驱动力。



弗里德里希·哈耶克：典型的自由主义者

哈耶克是凯恩斯的反对者，并因此获得了诺贝尔经济学奖。他将开放和自由市场视为组织经济活动的最有效的方式，而且视其为个人自由的护卫者。

  1. 个体是社会根本，产生的社会结果是中央计划者不能复制的。
  2. 政府里的人，尽管意愿很好或者很聪明，但永远不能够得到所有正确的信息，管理好经济这类复杂系统。
  3. 社会最佳结果是被允许做决定的人基于他们的知识所做出的。
  4. 试图让所有人平等，而不是平等地对待他们，这是一种新的奴役形式。


米尔顿·弗里德曼：货币主义之父

和哈耶克一样，弗里德曼热衷于对自由市场的解释，推崇小政府、少监管、纯市场经济的理论。他是理查德·尼克松和罗纳德·里根的顾问。1976年，他因在消费分析、货币供应理论和历史，以及政策稳定的复杂性论证等领域的成就而获得诺贝尔经济学奖。


弗里德曼认为有必要限制政府的角色和范围。他反对歧视同性恋，因为他认为这种偏见会限制一个人作为个体参与到市场的能力。他还认为有偏见的人强加了成本给他或她自己。他支持贩毒和卖淫合法化的自由论者的思想，他也反对美国的兵役。最后，他提出了负所得税的概念，即那些收入水平低于一定标准的人们能从州政府那里收到钱，而那些赚得多的人则需要缴更多的税。



保罗·萨缪尔森：新古典综合经济学

萨缪尔森的《经济学导论》是卖得最好的教材，出售了上千万册，重印了19版，翻译成41种语言。

他发现，在竞争市场内，如果参与者充分获知了所有信息，价格最终呈现出无趋势的随机波动。

  1. 萨缪尔森区分了私人物品和公共物品的差异。公共物品是指某个人的消费不意味着其他人消费的减少。
  2. 世代叠加模型：每个人会生活在三个特殊的时间段：在前两个段，他们工作、挣钱和存钱，最后一个阶段是他们退休后，花钱且无钱可挣。
  3. 经济学家应去找出消费者更偏好这个物品而不是其他物品的本质。


加里·贝克尔：现实世界的经济学

贝克尔将新古典经济理论运用到一系列社会问题的研究中，他的分析基于人类的行为是理性的，人们的行为动机是最大化其快乐感。

  1. 歧视会产生货币影响。
  2. 贝克尔假定犯罪行为是一个理性决定。
  3. 提供让人上瘾的商品的成本—对合法商品抽税，对使用非法物品者加大惩处力度，可能会对减少上瘾者的行为有好的影响。
  4. 家庭经济学是加里·贝克尔的创新。



丹尼尔·卡尼曼：经济心理学家

经济学家考虑人们应该做什么，而心理学家观察人们实际上是怎样做的。——丹尼尔·卡尼曼

卡尼曼作为心理学家却获得了2002年的诺贝尔经济学奖。诺奖对其评价是：“他将心理学研究的观点整合进经济学中，特别是有关人类判断和不确定性决策—由此奠定一个全新研究领域的基础。”他是行为经济学的主要推动者。

他认为：人们的行为经常表现出非理性。提出观点：人们大脑中有两个系统来做决策，系统1借助直觉做出快速、自动和不费力的决定；系统2要运用复杂的推理来做出困难的决定。（《思考：快与慢》）


  1. 不仅仅是一名心理学家，卡尼曼（和特沃斯基）在行为经济学领域也颇有建树。
  2. 他的研究成果颠覆了作为经济学基础的理性行为主流思想的大部分假设。
  3. 他关注人们如何制定决策，现在已经融入政府制定金融决策和规章的过程中。
  4. 创立一个系统性的对启发法和认知偏差的理解，极大地提高了策者理解人们如何做决策，以及避免做出糟糕决定的能力。
  5. 对于良好结果和决策者自身能力的过度自信，引发了金融繁荣泡沫破裂。
  6. 人们往往无法充分考虑他们所承担的风险。
