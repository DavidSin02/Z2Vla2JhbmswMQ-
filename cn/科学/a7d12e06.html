热拉提&热玛吉，傻傻分不清楚？

2020-11-14
| 科学

古往今来，广大女性的美容需求总是层出不穷，有人追求白皙无暇，有人追求红润细腻，但在一点上大家都达成了空前一致：那就是 —— 紧致饱满、没有皱纹的不老容颜。

先来说说皮肤是怎么衰老的：

  • 皱纹

皱纹有大有小，有深有浅，浅层的小干纹一般是角质层干燥导致的，通常做好保湿工作就能缓解。

而较深、固定的皱纹则是因为真皮层的老化。当对皮肤起到支撑作用的胶原纤维和弹性纤维遭到破坏，就会出现肉眼可见的皱纹。

  • 松弛

当皮肤的衰老再发展，胶原蛋白、透明质酸的生产机器 —— 成纤维母细胞，就会因为老化而产能下降。

同时，因为紫外线照射、环境影响、不良作息等因素，最终会导致真皮层胶原蛋白、透明质酸加速流，含量大降。

再加上对表皮有“拽着”作用的胶原纤维、弹性纤维持续遭到破坏，就会体现为皮肤外层的松弛。

  • 下垂

这时候必须提及一个名词：

SMAS

Superficial Musculo-Aponeurotic System

表浅肌肉腱膜系统

这是一层纤维鞘包绕的肌肉筋膜，延伸至真皮纤维连接皮肤，通俗点就是骨肉相连之间的那层筋膜，覆盖了整个面部组织。

筋膜层的衰老，比真皮层的衰老层次更深，明显的表现是整个面部的软组织下滑，就是感觉整个脸都往下垂，出现脸垮的感觉，比皱纹更显老态。

皮肤的老化还有联动性，浅层的衰老会升级成深层的衰老，浅层组织的松弛、位移也会拖着深层组织的更快下垂。

提升、紧致......这些抗衰涉及的名词，实则都是SMAS筋膜层的事儿。真正意义上的医美手段提拉，其实是让皮肤重新被SMAS筋膜层拽住的过程。也就是说提拉紧致效果好不好，取决于有没有作用到SMAS筋膜层。

护肤品是很难触及到这样深层衰老的，好在现在医美比较给力，热拉提、热玛吉这些黄金项目，花费不菲，究竟值不值得做呢？你又适合哪个呢？

热玛吉

// 原理

热玛吉采用的是射频生热激活胶原纤维再生，由外向内层层递减式加热皮肤。

// 深度

热玛吉作用方式主要在真皮层，这也是为什么它收紧皮肤效果肉眼可见，但对深部组织提拉的效果不好。

// 热量

表皮60℃左右，皮肤深层温度大约在50℃。

// 作用方式

全层容积式加热

// 舒适度

有明显疼痛感，需要敷麻药

// 治疗部位

热玛吉不同的探头可以针对不同的治疗区域，大到身体、面部、颈部，小到独有的眼周探头。

// 效果体现

理论而言，治疗后胶原会在3-6个月内逐渐增生，皮肤皱纹得到改善并且收紧。

// 维持时间

官宣热玛吉的作用会保持在2-3年不等，事实上时间和仪器优劣、医生手法密切相关，事实上能保持1-2年左右。

// 副作用

热玛吉治疗头真假难辨，如果真的使用了假冒的治疗头，输出不稳定，可能会造成烫伤、脂肪被破坏、局部凹陷等情况。


热拉提

// 原理

热拉提通过独有的Focused RF聚焦射频技术实现对皮肤分层加热；浅层、中层、深层精准加热。这个技术的厉害之处在于把加热从盲探升级到雷达式定位导航，可以将射频能量准确聚焦，精准加热。

// 深度

浅层促进胶原收缩；中层激活胶原再生；深层可触及到SMAS筋膜层和脂肪层；重现构建弹性纤维网，缩小并减少脂肪细胞。也就是说，热拉提对于皱纹、松弛、下垂都有明显的改善效果。

// 温度

深层温度在60℃-65℃，是刺激胶原的最佳温度，表皮保持在40℃，不宜烫伤。

// 作用方式

分层聚焦加热

// 舒适感

热拉提在舒适度上甩了其他抗衰治疗好几条街，不需要麻药，整体热烫感在承受范围之内。

// 治疗部位

热拉提针对不同治疗部位有对应的手具，身体、颈部、面部、眼周，广泛且不失精准。

// 效果体现

热拉提治疗后即刻就能看到明显的皮肤收紧，之后随着胶原不断增生，提拉紧致效果越明显，轮廓线条感强烈。

// 维持时间

热拉提需要按疗程治疗（3次/疗程），疗程治疗后，可维持1-2年时间。

// 副作用

在FDA记录数上，热拉提的失误率为零，安全性能优秀。但如果操作的手法不对或经验不足，可能会引起烫伤问题，选择正规医院、正规医生操作很重要。

松垂hin厉害、怕疼的盆友选热拉提

热拉提是目前唯一可以用做到SMAS筋膜层的抗衰项目，再加上“隔空加热”模式，实现了舒适提拉、溶脂小脸的功效。

PS:每个人的皮肤厚度和各部位的抗衰诉求是不同的，选择专业医生量身打造治疗方案，选择合适的挡位和深度很重要！不要贪图便宜，那很有可能只是做了热拉提维养版，预防衰老，对已经形成的松垮、下垂基本没有效果。

普通松弛、耐受力强选择热玛吉

热玛吉聚焦真皮层，除皱紧致，一般也不需要疗程治疗，适合生活节奏非常快并且不怕疼的盆友。
