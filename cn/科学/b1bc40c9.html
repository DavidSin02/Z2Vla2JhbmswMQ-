理解支持向量机及示例

2020-10-29
| 科学

简介:

支持向量机是一种监督学习算法。监督学习算法意味着它需要在训练集上标记数据以学习。

支持向量机找到最优的超平面，使类之间的边界最大化。超平面是二维空间的直线，它是三维空间的平面。

优化目标:

构建一个超平面，将类尽可能地与最小的错误分类的数量分隔开，并最大化两个类中最近的数据点之间的距离(差值)。

当数据不是线性可分的时候：

当数据不是线性可分的时候使用内核，这会将2d点转换成n维空间。这是一个复杂的变换，在完成变换之后，超平面的建立方式就是将两个类精确地分隔开来。

数学背后的SVM：

目标函数=正则化+损失函数

调整SVM的参数：

  • C：正则化参数。它还控制平滑决策边界和正确分类训练点之间的权衡。

  • Kernel：它指定要使用的内核类型。它可以是'线性'，'poly'，'rbf'。

  • Gamma：它是核系数。伽马值越高，将尽力准确地按照训练数据集进行拟合，即泛化误差并引起过度拟合问题。

支持向量机的优缺点:

优点:

  • 在高维空间中有效。

  • 使用称为支持向量的训练点的子集更有效。

缺点:

  • 对于很少的异常值，只适用于相对较小的数据集。

  • 过早优化的问题。

Python示例:

In [1]:

import numpy as np

%matplotlib inline

import matplotlib.pyplot as plt

from sklearn import svm, datasets

In [2]:

iris = datasets.load_iris()

X = iris.data[:,:2]

y = iris.target

In [3]:

# Create an event of s=SVM to fit data

# svc = svm.SVC(kernel='linear', C = 1)

svc = svm.SVC(kernel='rbf', C=10)

# svc = svm.SVC(kernel='poly', C=10)

svc.fit(X, y)

Out[4]:

SVC(C=10, cache_size=200, class_weight=None, coef0=0.0,

decision_function_shape='ovr', degree=3, gamma='auto', kernel='rbf',

max_iter=-1, probability=False, random_state=None, shrinking=True,

tol=0.001, verbose=False)

In [5]:

# Plot data points

plt.scatter(X[:, 0], X[:, 1], c=y, cmap=plt.cm.Paired)

plt.xlabel('Sepal length')

plt.ylabel('Sepal width')

plt.title('SVC with linear kernel')

plt.show()

In [6]:

# create a mesh to plot in the dataset

x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1

y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1

h = (x_max / x_min)/100

xx, yy = np.meshgrid(np.arange(x_min, x_max, h), np.arange(y_min, y_max, h))

# prediction coutour plot

plt.subplot(1, 1, 1)

Z = svc.predict(np.c_[xx.ravel(), yy.ravel()])

Z = Z.reshape(xx.shape)

plt.contourf(xx, yy, Z, cmap=plt.cm.Paired, alpha=0.8)

plt.scatter(X[:, 0], X[:, 1], c=y, cmap=plt.cm.Paired)

plt.xlabel('Sepal length')

plt.ylabel('Sepal width')

plt.xlim(xx.min(), xx.max())

plt.title('SVC with linear kernel')

plt.show()
