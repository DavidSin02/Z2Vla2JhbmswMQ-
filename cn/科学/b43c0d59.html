南宁撩螺传说，没有店面的她在北大桥下守候了32年

2020-11-14
| 科学

“对于南宁人来说，撩螺就是夜宵时间永恒的课题”

旧时北大桥下南宁港的码头工人，卸下沉重的货物，总会到附近找个小摊嗦嗦螺卸去一天的疲惫。似乎只要胃满足了，一切温暖和幸福就会包围住全身。

时光荏苒，无数的记忆伴随着岁月消逝，但在南宁北大桥底有一家名叫“螺婆”的小摊，32年里连位置都未曾换过。

不会刻意张扬，就这样安静地在路边守护着属于南宁撩螺界的一枚传说。

神龙见首不见尾

没有店面的32年老字号——螺婆

//

拥有这么多年的历史，螺婆当仁不让是南宁老字号，可它一直以来却连个铺面都没有。

每次去吃，都需要手机定位一旁的北大江北路口公交车站，然后还得注意是桥下靠近朝阳这边的方向，最后再在人群当中认着螺婆的相貌找过去。

这段寻觅过程，生怕出一丁点儿差错。

对于螺婆而言，她这辈子大概都和螺分不开了。每天洗螺、泡干净、加料、煮螺……日复一日，年复一年。

经过我和朋友多次觅食，发现了一个规律，每每到了傍晚六点多，螺婆就会摆出一锅已经煮好的螺和卤好的鸭脚鸭翅，用煤炉微弱的火保温，等待一波又一波客人们的到来。

老南宁的回忆一直都在从未离开

//

住在附近的食客和我说起，他第一次和小伙伴们来这棵大榕树下撩螺的时候还在上住小学，那时候螺虽然只要几毛钱，但一高兴吃嗨了还是会捂捂荷包，怕回家被爸爸妈妈打。

过去在螺婆这里是用小别针折成的铁签来吃螺的，然后吃完插在生姜里也就当是消毒，生姜也因此而变得千疮百孔。

现如今为了干净，小别针也就变成了一次性的竹质牙签，装螺用的瓷碗也变成了不锈钢。虽然和回忆里吃法有了些差别，但也还能适应，毕竟在南宁其他地方也都是用牙签和不锈钢。

4元一碗的螺也有好味道

//

到了物价涨得飞快的现在，看到4元一小碗、5元一大碗的价格，怎么感觉都好像时光在悄悄倒转。不过，和小时候不太一样的是，现在一连来个十碗也不会心疼，这大概就是长大的区别吧。

螺婆这里一碗螺的价钱虽然便宜，但紫苏的量却一点都不吝啬，清新的香气弥漫在每一次咀嚼里，粒粒酸笋的点缀也让石螺原本的土腥消失得差不多。

这使得本身不太入味的螺肉，拥有了一抹独到的鲜美。这也是南宁人爱撩螺的原因之一，独独迷恋这特殊的味道！

吃完一碗螺，再举起碗喝上一口汤，大量石螺化作的螺汤，浓郁的鲜直接充溢到鼻梁。螺婆对于自家的螺汤，也是万般自信：“想喝汤，免费随便加，喝到饱都行。”

嗦完一碗不够尽兴，想再加螺，就直接拿着自己吃完的碗喊句：“阿姆，加螺喂。”

螺婆卤出的鸭脚、鸭翅也如同这里的石螺一般诱惑，淡淡的卤香，微微的咸，吃的就是属于上个世纪未完的古早味。

用牙齿轻轻咬上一口，滑滑的有些韧劲的皮就会接连被撕下，舌头一转，柔软的肉完全被撵开的那刻，酱香和卤香就会在口腔里散开。

旁边的窗台上，螺婆早已为你准备好了几种蘸料，辣椒酱、柠檬酱、黄皮酱、酸醋、酱油一应俱全。

只是简简单单一小碗螺却可以因自己不同而适当的调味而变得丰富多彩。对了，撩过了再多的螺，我也还是觉得再加一点柠檬酱，才会有南宁小时候的滋味。

北大桥底，总有人愿为你不断守候

//

当夜幕完全落下，周围的摊子也纷纷出来，北大桥底的夜宵店们开始热闹非凡。

有人喝着酒猜着码，几个朋友嬉笑吹水，快乐的声音在耳畔此起彼伏，汇成未来对于老南宁的记忆。

看着，螺婆周围一家家店开了又关关了又开，招牌换了一轮又一轮。

可连摊位都没有的她，就在这里对着一锅锅石螺煮上了三十多年的光景。

以前，身边的朋友总说南宁老味道难寻，其实，在不经意之间就有人在街道的一旁，为你留住关于南宁过去的味道。

“友仔，改天去撩螺嘛？”

“走喂！”

螺婆

  • 人均：4元
  • 地址：西乡塘区北大路2号旁（靠近肥佐烧烤）
  • 营业时间：傍晚18:00-凌晨1:30左右
