八月上映的电影，依旧很烂，但能看的比七月多了两部。

2020-11-14
| 科学

本文为微信公号电影天堂原创，转载请联系授权

8月还处于保护月的保护之中，虽然院线依旧拥堵，烂片依旧扎堆。

但毕竟，能看的比七月多了几部。

所以，一起来看下八月影市，小心别让烂片辣了你的眼

﹁

八月院线前瞻

﹂

《爱宠大机密》

上映日期：8月2日

导演: 克里斯·雷纳德 / 雅候·迟内

主演: 路易·C·K / 艾瑞克·斯通斯崔特 / 凯文·哈特 / 艾丽·坎伯尔 / 史蒂夫·库根

类型: 喜剧 / 动画

小黄人团队原班人马制作，选的爱宠都是胖乎乎圆滚滚的，所以不必担心萌不到你。

电影在7月8号北美上映，两天累计一亿票房，打破了由《头脑特工队》创造的票房纪录，成为原创动画最高首周。

媒体影评人放出的评价也尚可， 不求内涵丰富，只图欢乐氛围的话，真的值得一看！

另外，不小心选到国配版也不用过于沮丧，《爱宠》这次的国语配音版除了何炅之外，还请来了陈佩斯老师位反派小白兔配音~

《龙拳小子》

上映日期：8月2日

导演：陶声 王玮主演：刘芮麟 童菲 林秋楠 梁超 吴永伦 余薇薇 王皓祯 陈浩民 金巧巧

类型: 喜剧 / 动作

一个功夫小子拯救了世界的故事。

《我最好朋友的婚礼》

上映日期：8月5日

导演：陈飞宏

主演：舒淇 冯绍峰 宋茜

类型: 喜剧 /爱情

翻拍自好莱坞同名经典喜剧，舒淇接演大嘴朱莉娅·罗伯茨那个角色，宋茜演卡梅隆·迪亚兹演得那个白富美。

珠玉在前，总少不了对比，并且这几年我们翻拍的《新娘大作战》《命中注定》等作品质量都不行。观影需谨慎啊。

《盗墓笔记》

上映日期：8月5日

导演：李仁港

主演：鹿晗 井柏然 马思纯

类型: 奇幻 / 冒险

两大热门IP，去年火的是《鬼吹灯》，今年轮到了《盗墓笔记》。

虽然《盗笔》网剧扑街，但这次影版来自南派三叔徐磊亲自捉刀编剧，让不少书粉稍稍安心。

演员选用鹿晗、井柏然两大人气偶像，票房的号召力不必担心。

质量的话，本片导演为李仁港，他的作品有：鸿门宴传奇、锦衣卫、三国之见龙卸甲。

《谎言西西里》

上映日期：8月5日

中韩合拍片。

导演是凭《翻滚吧，阿信》一片闻名华语影坛的台湾导演林育贤，监制为拍过《蓝宇》的香港著名导演关锦鹏。

幕后大咖来自韩国，《海云台》的摄影、《黄海》的灯光师。

主演周冬雨、阮经天、李准基也都算有演技的。

但看预告片似乎还是“车祸、癌症、治不好”这种韩剧老三宝的梗啊

《古曼》

上映日期：8月5日

导演: 张江南

主演: 韩雪 / 唐宸禹 / 吴艺璇 / 朱圣祎

类型: 惊悚

看海报，看主演。

看导演，

《活宝》

上映日期：8月9日

导演: 陈伟强 / 王惜今

主演: 胡夏 / 黄圣依 / 王惜今 / 陈籽熹 / 朱时茂 /保剑锋

类型: 喜剧 / 爱情

《使徒行者》

上映日期：8月11日

导演: 文伟鸿

主演: 张家辉 / 古天乐 / 吴镇宇 / 佘诗曼 / 许绍雄

类型: 喜剧 / 犯罪

热播电视剧《使徒行者》的电影版本，剧版的评价不错，豆瓣评分8.1

影版换了主角大换血，阵容倒是更强大了，导演还是文伟鸿，如果能保持风格的话还好，不然可能会跟影版《冲上云霄》一样惨淡。

《碟仙诡谭2》

上映日期：8月12日

导演: 雷宇扬

主演: 张韶涵 / 方力申 / 孙艺铭 / 惠英红 / 雷宇扬/ 王子腾

类型: 悬疑 / 惊悚

在你准备看2的时候，建议先看下《碟仙诡谭1》的评分

《微微一笑很倾城》

上映日期：8月12日

导演：赵天宇

主演：井柏然 杨颖 李沁 王思聪 刘循子墨 谭松韵

类型: 爱情

导演赵天宇发挥比较不稳定，履历表里有《双食记》

也有《锺馗伏魔》

不过看到张一白监制可能就更放心一点了是吧~

《危城》

上映日期：8月12日

导演: 陈木胜

主演: 刘青云 / 古天乐 / 彭于晏 / 吴京 / 袁泉

类型: 动作 / 武侠 / 古装

讲述军阀割据的乱世时代，发生在一个小城里的恩怨情仇。

演员上齐聚两岸三地多位大牌，枪战和打戏也是一大看点，看过的盆友反应武戏尚可，文戏略生硬，总体还算差强人意。

Whatever，“当彭于晏颤着大胸骑着马儿向你走来，你还能拒绝什么....”

《我们诞生在中国》

上映日期：8月12日

导演：陆川

解说：周迅

类型: 剧情 / 纪录片

迪士尼跨国团队制作的野生动物题材的纪录片，由陆川执导。

秀美山河，可爱动物，值得一看。

《海洋之歌》

上映日期：8月12日

导演: 汤姆·摩尔

类型: 剧情 / 动画 / 奇幻

主演: 大卫·罗尔 / 布莱丹·格里森 / 丽莎·汉尼根 / 菲奥纽拉·弗拉纳根

奥斯卡最佳动画电影提名。欧州上映以来好评如潮！

豆瓣评分8.8，,好于94%动画片，97%奇幻片！

手绘画风美腻，任意一个截图都可以当屏保。

隔了两年，终于有机会在电影院的银幕上一览风采，我的本月必看。

《美好意外》

上映日期：8月12日

导演：何蔚庭主演：桂纶镁/王景春/欧阳娜娜/王元也/陈坤

类型: 喜剧 / 奇幻

翻拍自韩国电影《妻子小姐》，讲述一位都市白领一夜之间变成普通家庭主妇的故事。

奇幻设定，差不多跟《重返20岁》是同样的路线，电影虽然有桂纶镁和陈坤，但刚被欧阳娜娜的“鹿小葵”荼毒过的观众，估计不想在电影院里再见她一次了。

《勇士之门》

上映日期：8月12日

导演: 马蒂亚斯·霍恩

主演: 赵又廷 / 倪妮 / 尤赖亚·谢尔顿 / 吴镇宇 /戴夫·巴蒂斯塔

类型: 奇幻 / 冒险

中外合拍的奇幻动作冒险电影。

由吕克·贝松编剧及监制。

这个剧情设置，演员搭配，欸？好熟悉，不是《功夫之王》吗？

  • 《勇士之门》：废柴男孩（尤赖亚·谢尔顿 Uriah Shelton 饰）机缘巧合地开启了异世界的大门，卷入了一场惊心动魄的种族斗争。

  • 《功夫之王》：弱鸡少年杰森（Michael Angarano 饰）从某中国古董店得到一根如意金箍棒，借着金箍棒的力量，他竟然回到了几千年前的中国。

《精灵王座》

上映日期：8月12日

导演: 宋岳峰

类型: 动画

人气网游《龙之谷》改编的动画大电影，2014年的首部《龙之谷：破晓奇兵》口碑爆棚，当时被誉为国产3D动画的巅峰之作。

两年之后，原班人马重新打造的续集《精灵王座》，你会去看吗？

《那件疯狂的小事叫爱情》

类型：爱情

导演的上一部作品至今让我心有余悸，

电影质量上，目前北美放出的评论褒贬不一，但是惨淡的8月，难道这不是最值得看的大片吗！
