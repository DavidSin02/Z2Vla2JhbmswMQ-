《DK博物大百科》

作者：英国DK公司 [编着]

出版社：科学普及出版社

原作名：Natural History

译者：王敏洁 / 朱坤 / 吴海峰 / 何长欢 / 张兴春 / 张劲硕 / 林然 / 郑浩然 / 姜景一

简介

我们的杂志就叫《博物》，可能很多人都会有这个疑问：博物是什么意思呢？一般来说，博物的范围主要包括对动物、植物和矿物的基本认识和了解，“博物馆”则是收集、展示“博物”的场地，而这本书就像是一本纸上的博物馆，向我们展示了多样的大千世界。

在这本书的译者团队中，有三位都是我们《博物》的成员，他们是吴海峰、何长欢和特约校对张劲硕。在与他们聊到参与翻译工作的心得时，编辑得知这本书早在2012年就已经开始了翻译工作，而到了2018年年底才上市。

为什么会翻译这么久呢？因为翻译只是其中一个环节，编辑们还需要对所有文字进行科学审校，让译文既能传达原意又不失科学性。举个例子：原文中有一种鹦鹉的图注是“glossy parrot”，如果直译就会变成“光滑的鹦鹉”，但这肯定是不对的。因此要结合这种鹦鹉独特的外观，翻译为“体羽具金属光泽的鹦鹉”才更为合适。

这本书足有7公斤重，当属本月甚至是本年度书单中的NO.1，而它能带给大家的知识量绝对会远超这个数字。

《世界野生猫科动物》

作者： [澳] 卢克·亨特 / [英] 普瑞西拉·巴雷特出版社：湖南科学技术出版社

出品方：中国国家地理译者：猫盟

简介

每每看到“猫”的脸，就觉得造物主真是个大艺术家。有人说，听说自然与人类之间需要一个信使，所以猫科动物就出现了。这当然只是一个美好的想法，但却刚好体现了猫科动物的不同之处。

猫科动物的自由和美，成全了我们作为野生“猫奴”的本命。世界共有野生猫科动物40种，其野性和美令人目眩。40种野生猫科动物，40种荒野家园，这本书会让你更了解这些人类的最美邻居，过足猫奴的瘾。

《奇趣动物宝典》

作者：史蒂夫·詹金斯

科学审校：吴海峰

出版社：北京少年儿童出版社

简介

在本月的书单中，这本可能是最适合小朋友阅读的。首先，它内容的编排方式很特别，不是按照动物类群或者生态环境给动物分类，而是按照小朋友提问的方式来编排的。例如，在动物如何感知这个世界这个部分讲了许多种动物的视觉、听觉、嗅觉等等。

其次，很多这类书中要么是手绘图，要么是摄影作品，而这本书中则是拼贴画。作者用不同材料、质地、颜色的纸张剪出动物的不同部分，之后再把它们一块一块、一层一层地拼贴起来，最后再拍照印在书上。因此，这本书不单单是一本动物图书，更是一本手工书，让小朋友们在学习知识的同时可以收获快乐。

《国家地理珍稀鸟类全书》

作者：[美] 乔尔·萨托 / [美] 诺亚·斯特瑞克

出版社：江苏凤凰科学技术出版社

译者：胡晗 / 王维

简介

在这本书的封面上，最吸引人眼球的恐怕是这只五彩斑斓的鸟，而很多人会忽略了PHOTO ARK 这几个字。

“影像方舟”（PHOTO ARK）是由摄影师乔尔·萨托在十多年前发起的项目，他的目标不是拍摄那些生活在野生环境中的动物，而是拍摄“圈养动物”——生活在动物园和水族馆里的动物。该计划已经在世界各地的动物园和水族馆中拍摄了5400种动物的肖像照，乔尔·萨托表示，在拍完12000种圈养动物之前他不会罢手。

为什么会选择这样的一群动物呢？在乔尔·萨托看来，很多物种已在野外灭绝，而动物园和水族馆成了它们真正的赖以生息繁衍的“方舟”。

《蓝色星球II》

作者：[英]詹姆斯·霍尼伯内/ [英]马克·布朗罗出版社：江苏凤凰科学技术出版社

译者：李蕊 / 印思玫

简介

当年《蓝色星球2》的纪录片刚一推出，就立刻登上了各个播放平台的榜单。它有什么厉害之处呢？在长达4年的拍摄过程中, 纪录片的制作团队共执行了125次的探险，水下拍摄时数长达到6000多个小时，以全新的拍摄技术，呈现出前所未有的画面。如果你是一个痴迷于海洋的人，我觉得你不应该错过这部优秀的作品；而如果你已经看过这部纪录片，这本基于片子的书也是值得一读的。

书中不仅有200多幅来自纪录片的4K高清图片，还展示了众多海洋居民的栖息地原貌，通过讲述令人动容的海洋生命故事，系统地介绍了海洋知识，并独家披露镜头之后的摄制过程。

《濒危》

作者： [英] 蒂姆·弗拉克 摄影 / 乔纳森·贝利 萨姆·威尔斯 撰文

出版社：天津人民出版社

出品方： 后浪

原作名：ENDANGERED

译者： [美] 林肯

简介

准确来说，这本书更像是一本摄影集。蒂姆·弗拉克是一位动物摄影大师，在他的镜头下，动物们或沉默，或静思，或优雅，或深沉，具有一种奇特的美感，而我们也仿佛可以从它们的眼神感受到情感，和它们对话。

人和动物之间到底是怎样的一种关系呢？也许很多科普作者都在用文字向我们讲述他们各自的理解，但有时候最直观的感受来自于视觉。摄影师蒂姆·弗拉克通过记录濒危物种和其生存境遇的特别项目，探索了我们这个时代最紧迫的问题之一。

《BBC科普三部曲》

作者：[英] 保尔·罗斯 安妮·莱金

出版社：重庆出版社

译者：丛言 / 胡娴娟 / 陈瑶

简介

这套书有三本，分别是《地球：行星的力量》、《海洋：深水探秘》和《生命：非常的世界》。先拿《海洋》这本来说，虽然上面也有一本BBC讲海洋的书，但其实它们完全不是从一个角度来解读的，《蓝色星球2》更偏重介绍海洋动物，而《海洋》则是带你探索世界七大洋各自的秘密。

《地球》这本是探索地球如何运转，生命如何孕育，逐一介绍塑造地球的四种力量：火山、大气层、海洋与冰川；《生命》则是为大家展现了世界每个大陆上的动物和植物为了生存而使用的各种策略，看看它们有多厉害吧！
