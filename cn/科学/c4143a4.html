01 水黾属于哪一目？

A 半翅目

B 蚤目

C 同翅目

D 鳞翅目

【答案】A

【解析】水黾是一类适宜在水面上生活的半翅目昆虫。研究表明，水黾的浮力来自它们的3对特殊结构的足，在足的表面生有许多微毛，微毛表面有疏水性的油脂包裹，这种结构的足在水表面张力的作用下产生浮力，可以支撑起水黾瘦小的身体，从而能够使它像船一下漂浮在水面。

02 水黾有几对足？

A 2对

B 3对

C 8对

D 10对

【答案】B

【解析】水黾是一类适宜在水面上生活的半翅目昆虫。研究表明，水黾的浮力来自它们的3对特殊结构的足，在足的表面生有许多微毛，微毛表面有疏水性的油脂包裹，这种结构的足在水表面张力的作用下产生浮力，可以支撑起水黾瘦小的身体，从而能够使它像船一样漂浮在水面。

03 水黾是哪种食性的动物？

A 肉食性

B 植食性

C 杂食性

D 只喝水

【答案】A

【解析】水黾是肉食性昆虫，在水面上滑来滑去主要是觅食，取食在水面活动的其他昆虫或小型动物。

04 龙虱属于昆虫哪一目？

A 食毛目

B 双翅目

C 膜翅目

D 鞘翅目

【答案】D

【解析】龙虱是鞘翅目龙虱科昆虫的统称，它们和其他陆地昆虫一样，也是用气管来呼吸的。它们之所以能在水中呼吸是因为具有特殊结构的翅膀；鞘翅完全革制化，膜质的后翅隐藏于革质的鞘翅之下。

05 关于蜜蜂的“舞蹈语言”，下列叙述错误的是？

A 能传达蜜源的远近

B 能传达蜜源所属花的颜色

C 能传达蜜源的性质

D 能传达到达蜜源所需的能量

【答案】B

【解析】当一只担当侦查任务的蜜蜂发现蜜源后，它会返回巢穴，并在家门口用“舞蹈语言”告诉其他同伴蜜源植物所处的方向和距离。“舞蹈语言”传达的信息包括到达蜜源所需的能量、蜜源的性质及相对太阳位置的方向等。领会含义的工蜂随后会一起出发采蜜。蜜源越近，蜜蜂的舞蹈节奏就越快。而蜜蜂是色盲,它只对蓝色、黄色和白色分得清楚,对红色是盲区。因此，一般不能准确传达蜜源所属花的颜色。

06 如果食物在附近，蜜蜂一般会跳什么样的“舞”来传达信息？

A 三角形舞

B 8字形“摇摆舞”

C “圆圈舞”

D 机械舞

【答案】C

【解析】当一只担当侦查任务的蜜蜂发现蜜源后，它会返回巢穴，并在家门口用“舞蹈语言”告诉其他同伴蜜源植物所处的方向和距离。如果附近有食物，它会跳相对简单的“圆圈舞”；如果食物在远处，它会跳更为复杂的8字形“摇摆舞”。

07 若食物距离蜜蜂很远，它一般会跳什么样的“舞”来告知同伴？

A “圆圈舞”

B 古典舞

C 梯形舞

D 8字形“摇摆舞”

【答案】D

【解析】当一只担当侦查任务的蜜蜂发现蜜源后，它会返回巢穴，并在家门口用“舞蹈语言”告诉其他同伴蜜源植物所处的方向和距离。如果附近有食物，它会跳相对简单的“圆圈舞”；如果食物在远处，它会跳更为复杂的8字形“摇摆舞”。

08 以下哪种昆虫一般不能帮助植物传粉？

A 蜂类

B 蝶蛾类

C 甲虫类

D 蝗虫类

【答案】D

【解析】除了蜜蜂之外，还有其他蜂类，如熊蜂、黄蜂，蚂蚁类，蝶蛾类，蝇类，甲虫类和蓟马类等大量昆虫也为植物提供传粉服务。

09 哪种昆虫是无花果的专一传粉者？

A 熊蜂

B 榕小蜂

C 黄蜂

D 马蜂

【答案】B

【解析】无花果是一种榕树的隐头果实。榕树虽是有花植物，但它的花很小，肉质花序轴向内卷曲成球状的花托，被称为“隐头花序”。深藏的小花，既没有艳丽的色彩，也没有芬芳的香气，而且全部被包裹在了膨大的完全封闭的花托之中，形成隐头果，无法吸引任何传粉昆虫进入其中。然而，这个看似天衣无缝的花托还是留了一个小孔与外界相通，这是为无花果的专一传粉者——榕小蜂量身定制的。

10 如果黑客侵入了数据库或网络操作系统，他不能实现的是？

A 在服务器里盗取密码和文件

B 使计算机病毒迅速传播

C 让计算机表达人类的基因

D 造成整个网络瘫痪崩溃

【答案】C

【解析】如果黑客侵入了数据库或网络操作系统，既可以在服务器里盗取密码和文件，又可以使计算机病毒迅速传播，造成整个网络瘫痪崩溃。
