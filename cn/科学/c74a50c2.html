奇异值分解SVD在PACS图像处理方面应用

2020-11-14
| 科学

这是我前年的一份论文，保存下。

[摘 要] 根据奇异值分解的基本原理及其特点，给出了运用奇异值分解进行PACS图像压缩的方法。通过示例详细说明了该方法进行图像压缩的基本过程，给出了压缩流程，并通过 R编程对实际图像进行处理，表明了该方法的有效性。

[关键词] 图像处理；SVD；图像压缩；奇异值分解

1、 引言

在医疗信息化系统当中，PACS影像分析是医生对患者病情一个非常重要的检查检验方法，尤其是进行双向转诊和远程互诊的时候，需要对图像进行远程传输，高质量的图片固然清晰度较高，但对网络传输成本较大，如果能够有一套方案能够对部分图像进行压缩，而且是可调节压缩比进行，这样就能够确保需要优质图片的时候进行很小压缩或者不压缩，而对于要求质量不高的可以进行深度压缩。基于奇异值分解在理论和实际应用目前都得有较好的发展。本文基于SVD的理论对医学影像图片进行可调节压缩。

2、 SVD理论简介

奇异值分解定理：

设 A&isin;，则存在正交矩阵：

U = [] &isin; ，V = []&isin;，使得A=U=，其中=diag()，并且&ge;&ge;&hellip;&ge;>0，，，&hellip;，是矩阵A的奇异值，r=rank(A)是矩阵A的秩。

奇异值分解的神奇之处在于可以找到任意矩阵的奇异值分解，也即任意一个矩阵A都可以分解为正交矩阵U，对角矩阵，正交矩阵V，其中中的各元素按大小排序就是A矩阵的奇异值，它表示了对应向量的缩放信息，而U矩阵的各列就是矩阵A变换的主轴。

3、图片处理

图像格式包括jpg/jpeg/bmp等格式，图像由像素点构成，每个像素点包含了RGB三原色的基本信息，也即从RGB角度来看，每个点的色彩信息是RGB组成的一个值，一般由16进制来表示，如FFFFFF表示白色，000000表示黑色，每两位代表一个颜色值，如下图1所示：

图1

对原图每个像素点进行分析，得出像素点的组成信息，然后分别形成R、G、B3个矩阵，分别对3个矩阵进行SVD操作，再进行选取合适的奇异值个数k，k值需要小于等r(秩)，k越接近于r，和原图的匹配程度越高。

基于性能的考虑，采用R语言进行处理，R语言在向量处理方面具有先天优势，能够具有最少代码工作量完成复杂向量工作，同时R在各个不同平台都可以运行。为了更加方便，采用RStudio IDE进行开发。在进行开发之前需要安装相关SVD和图像处理的包，如：

install.packages("rARPACK")

install.packages("jpeg")

接下来按照先后顺序进行处理

1） 加载需要的包

library(rARPACK);

library(jpeg);

2）读取指定的影像图片

rawimg = readJPEG("img1.jpg");

rawimg是一个3维数组，包括RGB的结构信息，下图给出详细信息，大小为162.7M

3）进行因式分解，主要用到svds方法，为了方便后面的压缩比率，增加一个可调节参数k，方法详细情况如下

factorize = function(m, k)

{

r = svds(m[, , 1], k);

g = svds(m[, , 2], k);

b = svds(m[, , 3], k);

return(list(r = r, g = g, b = b));

}

这样获得了3组奇异值分解的结果，封装到list列表

4）之后进行数据还原，根据A=U来进行向量运算，函数如下：

recoverimg = function(lst, k){

recoverinner = function(fac, k){

dmat = diag(k);#构造矩阵对角线

diag(dmat) = fac$d[1:k];#对对角矩阵进行赋值

m = fac$u[, 1:k] %*% dmat %*% t(fac$v[, 1:k]); # A=U

m[m < 0] = 0;

m[m > 1] = 1;

return(m);

}

r = recoverinner (lst$r, k); #R分量还原

g = recoverinner (lst$g, k); #G分量还原

b = recoverinner (lst$b, k); #B分量还原

m = array(0, c(nrow(r), ncol(r), 3));

m[, , 1] = r;

m[, , 2] = g;

m[, , 3] = b;

return(m);

}

4、结论

本文讨论了一种基于SVD的图像压缩方法，压缩的基本原理是用较少的压缩矩阵（）代替原始图像矩阵，再使用该压缩矩阵生成近似图像，根据奇异值分解基本原理，可以得到U，V。通过调节不同的k取值来达到不同的压缩比效果，实际工作中根据实际情况进行选择，本文通过完整实验进行了该方法的验证，确实是一种过程简单，行之有效的压缩方法，具有一定的应用前景和扩展程度。
