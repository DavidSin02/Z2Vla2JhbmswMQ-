如何在Python中将列表切成非零整数的连续组！只需一行代码搞定

2020-10-29
| 科学

似乎无法在网上找到有关此问题的线索，也无法自己弄清楚它，因此：

我将如何对列表进行切片，以便返回连续非零整数切片的列表。即：

data = [3, 7, 4, 0, 1, 3, 7]

我想生产：

slices = [[3, 7, 4], [1, 3, 7]]

我尝试了各种遍历该列表的方法，一直倾向于使用一个生成器，该生成器通过测试前后是否存在0来让我知道连续的组何时开始和停止，但是我有些困惑。

解决方案

import itertools[ list(x[1]) for x in itertools.groupby(data, lambda x: x == 0) if not x[0] ]
