“资本老手”易贤忠携百奥泰闯关科创板，长袖善舞引出背后“风云人物”董正青

2020-10-29
| 科学

资本高手易贤忠携未盈利的百奥泰卷土重来战A股，而一切的背后不简简单单是急需补血的公司，还有斥资布局医院的戏码，甚至早年间叱咤风云的董正青。

近日上交所披露，百奥泰生物制药股份有限公司(以下简称“百奥泰”)将于11月20日首发上会。今年10月末的格乐立（百奥泰研制的阿达木单抗）获批为百奥泰其登陆资本市场造了不少声势。

然而这家创新药企业是在易贤忠的裹挟下一步步前进，背后更是隐藏着早年间资本市场的风云人物。

现在提及易贤忠可能很多人会觉得陌生，但要是提起分众传媒的江南春，想必大家肯定都是耳熟能详。而两者之间的渊源要从私有化的分众传媒在2015年借壳七喜控股讲起。而彼时七喜控股的实控人就是易贤忠。

距今四年之久，此次闯关科创板。长袖善舞的资本大佬易贤忠意欲再次登上A股平台“起舞”资本。

//

“资本老手”易贤忠

//

二十多年前，离开国企创业的易贤忠或许也不会想到有一天他会成为搅动资本市场的大佬。

2004年，易贤忠带领七喜控股登陆深交所。随着公司的3C产品在移动互联网浪潮下日渐衰退，上市公司的业绩也是一路下滑。机缘之下，易贤忠的七喜控股成为彼时分众传媒进入A股的踏板。

2013年5月，分众传媒从美股私有化。在对赌之下公司必须在2016年前在A股上市。可是万万没想到的是，2015年分众传媒原本计划借壳的宏达新材，但因后者遭到调查被迫取消了借壳。于是幸运的七喜控股成为分众传媒登上A股的壳。

2015年9月初，七喜控股公告，以截至拟置出资产及负债的方式与分众传媒全体股东持有的分众传媒100%股权的等值部分进行置换。根据评估报告，交易资产分众传媒100%股权的评估值为458.7亿元。这对于业绩颓势尽显的七喜控股无疑是天大的好事。

来源：东方财富

2016年3月8日，在聚光灯下作为第一支回归A股的中概股，江南春也将原七喜控股的名称也将更名为分众传媒。借壳后，上市公司股价大涨，而原实控人易贤忠持有分众传媒2.36%股权，在此后两年间易贤忠陆续高位减持套现十几亿元，挣的是盆满钵满。

然而完成借壳之后，易贤忠也将原七喜电脑资产置出，转到七喜集团（由易贤忠、关玉婵夫妇和儿子易良三人持股100%持股）。与此同时资本市场大获全胜的易贤忠也并没有选择颐养天年，而是开始做大自己旗下其他产业，例如百奥泰。

百奥泰由李胜峰2003年7月28日成立，由于生物医药行业前期投入较大，李胜峰于在2009年末决定引入可以持续提供资金支持的易贤忠等人。与此同时，也包括以0元转让百奥泰60%股权。

易氏家族入主百奥泰9年后，在长袖善舞的易贤忠操盘下百奥泰冲刺科创板。易贤忠也再次踏入阔别几年的A资本市场。

来源：天眼查

七喜集团目前由易贤忠、关玉婵夫妇和儿子易良昱（持股98.53％）三人持股，根据七喜集团官网显示，其目前拥有 IT、地产、大健康三大业务板块，旗下包括两家制药企业、三家医疗器械企业和三家医疗投资运营企业。

而医院建设似乎也变成了七喜集团的主流操作。2016年10月，易氏家族注资50亿元，成立广州圣德医疗管理有限责任公司，这揭开了其医院版图扩张的序幕。此后的2016年11月和12月，常德肿瘤、临沂肿瘤、曲靖肿瘤和河南豫东肿瘤4家医院相继成立。2017年、2018年，易氏家族每年都新增了4家医院。

这些易氏家族不断通过七喜集团在背后扩张医院版图。不难猜出，再次起舞资本市场，易忠贤或将上演一边A股市场融资，一边斥资布局兴建医院的戏码。

市场沉浮几十年，易贤忠对资本运作的老练也非常人能比。易贤忠家族为百奥泰的实控人，合计控制着69.31%股权。掌控一切的易贤忠在上市前突击分红3.5亿。

来源：招股书

实际控制人不具备相关专业知识，公司经营高度依赖核心技术人员，所以便有了“为了吸引、保留和激励人才”合情合理的借口，百奥泰在易忠贤的操控下实施了股权激励措施。这一操作百奥泰股权激励获益最大的也就是这一切的策划者。股权激励支付对象包括实际控制人之一的易贤忠，而且4.46%的股份支付比例在71位股权激励对象中最高。

//

“战友”董正青

//

本次百奥泰上市的联席主承销商为广发证券，这个与易忠贤有着紧密联系的券商再次出现在大众视野。而那个核心人物就是早年间搅动资本市场的风云人物“董正青”。

1993年左右，董正青加入广发证券，十年之后董正青升任广发证券总裁。而在2004年，易贤忠的七喜控股登上资本市场，彼时广发证券为其保荐机构和主承销商。同为湖北潜江人的董正青和易贤忠的渊源从此开始。

2007年，广发证券借壳上市前，董正青因涉内幕交易入狱。四年刑满之后，董正青出狱，将汇天泽作为核心资本运作平台。汇天泽的初始股东为易贤忠和易阳平，这家公司成立的最初目的就是替董正青承接所持广发证券股权，董正青出狱前后，汇天泽的大股东已从易贤忠更换成董正青。此后董正青联手老友旧部频频出入资本市场。

来源：天眼查

在易贤忠将百奥泰招股书递交的那一刻，其与董正青在百奥泰详细的资本往来也被掀开。

根据招股书显示，2016年12月，七喜集团投资3.5亿元，吉富启恒、合肥启兴、汇智富联合向百奥泰投资2.7亿元。而其中吉富启恒、合肥启兴、汇智富均由吉富创投控制。而后者正是昔日广发证券员工持股平台，董正青为吉富创投第一大股东和实际控制人。2018年12月，粤创三号、兴昱投资、浥尘投资、汇天泽向百奥泰合计投资10亿元。其中，董正青持有汇天泽98.88%股权。2019年3月，百奥泰再次增资，兴昱投资、粤科金融集团、中科卓创、汇天泽、吉富启恒、浥尘投资共计投资5亿元。

来源：天眼查

除去易氏家族，百奥泰的主要投资方就是董正青控制的吉富创投和汇天泽。招股书显示，吉富启恒、合肥启兴、汇智富、汇天泽四者为一致行动人。董正青控制的企业合计投入了4.4亿元，合计持股9.2631%，为百奥泰第三大股东。而董正青掌控的汇天泽最重要的医药产业就是投资了老朋友易贤忠的百奥泰。至此十几年的你来我往，两人之间的关系早已可以不言而喻。

而曾经把七喜控股卖给分众传媒借壳的易贤忠，如今再次大张旗鼓的出现在A股资本市场，或许这也将是这位多财善贾的大佬继续利用上市公司来完成自己谋划的开始。
