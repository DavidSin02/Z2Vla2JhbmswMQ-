Chem. Mater | 一锅法原位凝胶体系制备IPN水凝胶用于组织再生

2020-10-29
| 科学

引言

随着社会老龄化人口的增加，骨关节炎在内的软骨疾病急剧增加，对软骨组织再生疗法的需求很高。作为一种广泛使用的人工关节替代治疗方法，自体软骨细胞的植入用于关节软骨的再生已成为一种有效且无创的治疗方法。为了使细胞植入疗法获得高治疗效果，需要将植入的软骨细胞转化为具有高弹性和减震特性的原始透明软骨而不是纤维软骨。目前，需要制备一种支架材料能够用于关节软骨组织再生，并且能为软骨细胞提供良好的培养环境。

因此，Hidenori Otsuka和Shigehito Osawa团队设计了一种一锅法原位凝胶体系用来构建互穿聚合物网络(IPN)，该设计涉及两种类型的凝胶化工艺：RADA16肽的自组装以及壳聚糖(CH)和N-羟基琥珀酰亚胺酯端聚乙二醇(N HS-PEG-NHS)之间共价键的形成。

凝胶动力学表明，RADA16肽网络是独立形成的，随后会在含有这三种组分的混合物中形成CH/PEG网络。在CH/PEG/RADA16中培养软骨细胞表明，IPN结构可以促进嵌入的软骨细胞形成关节软骨的性能。在植入含有软骨细胞的CH/PEG/RADA16的小鼠中，观察到较低的炎症和较高的蛋白质产生，证明了IPN水凝胶设计在组织再生中用作细胞培养支架的可行性。

近期发表在Chem. Mater杂志上题为“Interpenetrating polymer network hydrogels via onepot and in situ gelationsystem based on peptide self-assembly and orthogonal cross-linking for tissueregeneration”的文章，来自东京理科大学的HidenoriOtsuka和Shigehito Osawa团队。


具有互穿聚合物网络（IPN）的CH / PEG / RADA16可注射水凝胶“一锅法”合成的示意图

研究团队首先确定溶液之间的凝胶动力学关系，以评估RADA16肽，CH和PEG混合物以及CH，PEG和RADA混合物这三种组分形成IPN结构的潜力。实验结果表明，采用一锅法和原位凝胶化构建由RADA16和CH / PEG网络组成的IPN水凝胶，RADA16肽网络和CH / PEG网络的分层形成，不会相互影响而抑制其他网络的形成。CH / PEG / RADA16中RADA16的形成过程将比独立的RADA16更快，这是因为在分子拥挤条件下促进了由疏水相互作用，盐依赖性极性和氢键相互作用驱动的蛋白质折叠，所以推测CH和PEG的大分子拥挤效应促进了CH / PEG / RADA16中RADA16的组装。


凝胶化过程中IPN的时间扫描流变学测量

然后，通过在关节软骨中嵌入软骨细胞，证明了CH/PEG/RADA16作为关节软骨再生的细胞培养支架的性能。软骨细胞在CH/PEG/RADA16中呈圆形并且分布均匀，可能有效地诱导透明软骨（这是关节软骨的主要组成部分）。由于存在RADA16肽自组装结构，在CD光谱约220nm处可以观察到用软骨细胞包裹的CH / PEG / RADA16的负棉效应。不管嵌入的细胞是什么，都存在RADA16肽的β-折叠结构的纤维网络，这表明它有可能用作支架。


CH / PEG / RADA16中软骨细胞的分布

最后，将培养好的软骨细胞嵌入的水凝胶外移植到小鼠的背部，引发炎症反应低，蛋白产量高，证明了CH/PEG/RADA16可以作为细胞培养支架在体内应用，结果显示，培养8周后CH/PEG/RADA16中形成了更成熟软骨，促进软骨形成的能力比胶原蛋白凝胶好，支架性能优越。采用一锅法原位凝胶体系构建的IPN水凝胶，是由CH基和RADA16肽网络等糖类组合而成的，能为软骨细胞形成透明软骨提供良好的培养环境，有望成为组织工程系统的新技术。
