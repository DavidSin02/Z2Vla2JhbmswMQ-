F系新10级重坦装甲评测 这台车应该怎么摆姿势？

2020-10-29
| 科学

Hello，各位车长们大家好。在9.21版本的测试服中，法系添加了一条新的重坦线路。该线由7级AMX M4 1945的90mm炮研发，可以看成是7级AMX M4 1945的正统延续，作为传统抗线重坦并列在原弹夹线的一旁。

由于是AMX M4 45的后续车型，三辆新车的乘员也继承了AMX M4 45的配置，为5人车组，车长、炮手、驾驶、通信、装填各一人，同时也与FCM 50t和AMX M4 49两个金币车匹配，想出新线的同志们可以开始着手准备乘员了。今天先给大家带来10级AMX M4 54的测试。

测试服中的F系新重坦升级路线

AMX M4 54火力评价

数据图如下，跟同为用120炮的抗线车超级征服者相比：

AMX M4 54有两门炮可选，一门炮是120炮，与50B接近。弹种AP/APCR/HE，单发400/400/515，穿深264/325/65，基础装填10.9秒，极限装填8.5秒；另一门是130炮，这玩意比较逗比。弹种AP/AP/HE，单发560/560/640，穿深250/280/65，基础装填15.15秒，极限装填12.31秒。不出意外的话，大多数人都会选择120炮，毕竟130炮的穿深太尴尬了。

使用120炮时，穿深中规中矩，DPM一般，只是够用的水准。精度还不错但是火控一般，移动射击是别想了。机动看着还可以，40极速1200马力，但可惜大多时候只能跑到35，也没有比超级征服者好多少。

AMX M4 54静态装甲测试（正面）

首上装甲为220mm/54°，AP等效340，APCR等效360，HEAT等效370。这个水平已经接近E100了，是很不错的装甲。

140工程不论是使用金币弹还是银币弹都不能击穿正面首上装甲。

首下的形状与7201类似，中间有一个比较硬的横条。裤裆125mm/60°，等效240左右，上面的横条为240mm/35°，等效285上下。

140工程264穿的银币弹能够有效击穿正对的裤裆，但是很难击穿横条部分。

建议使用金币弹以保证击穿率。

AMX M4 54静态装甲测试（侧面）

车体侧面仅有60mm，可谓是纸糊，有时被HE击穿也不奇怪。伸缩时千万不要多摆角度，否则断腿，掉血，暴毙，一气呵成。4005也能用AP强制击穿侧面，千万当心。

AMX M4 54有两个小肩膀，正对时非常硬，因为角度非常大。但是问题也随之而来，只要摆一点点角度，比如10度，不说金币弹，大多银币弹都能做到发发穿，更要命的是弹药架就在肩膀后面！

正确的墙角伸缩姿势如下图：

既能藏住脆弱的侧面，又能稍微增大裤裆的角度，大幅降低银币弹的击穿率。

AMX M4 54静态装甲测试（炮塔）

AMX M4 54这辆车据说是应该能够卖头的......那么下面来说一下炮塔装甲。

头上的包当然是一个祖传弱点，基本上打到就穿，可惜包实在太小不好打，而且要是垫一下俯角，在正面根本就看不到。

如果头包不是大问题，那么下面一个问题就比较严重了：空心炮盾怎么办？

AMX M4 54的炮盾中央是空心的，等效大致在300-320的区间，大多金币弹都能有70%以上的击穿率。

所以千万不要低头压炮管，因为这样会导致炮盾被发发穿，这是楔形炮盾天生的缺陷。

测试总结

总结一下打法，正面金币弹打裤裆，如果打不到那就打炮盾，有侧面打就直接打到死。

这辆车总的来说没有什么特点，除开法系能抗线的新鲜劲儿，如果后面的测试没有加强调整的话，正式上线之后很可能也就是3分钟热度（公测服当中仅仅过了一天就没什么人玩了），如果想要去抗线的话，我干嘛不去玩超级征服者啊！
