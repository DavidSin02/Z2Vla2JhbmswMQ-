胃肠镜检查需注意哪些问题

2020-10-29
| 科学

来源： 中国医学科学院肿瘤医院服务号


一、什么是电子内窥镜？


它是一种可插入人体体腔进行直接观察、诊断、治疗的集光、机、电等高精尖技术于一体的医用电子光学仪器。肿瘤医院内镜中心常规开展胃镜、肠镜、无痛胃肠镜的诊疗工作。



二、胃镜适应症：


1.存在上消化道症状（如烧心、吞咽困难、上腹痛、呕吐等），怀疑有食管、胃、十二指肠病变，临床需要确诊者。

2.已确诊的上消化道病变，如消化性溃疡、食管癌、胃癌等疾病治疗后需要随访或观察疗效者

3.消化道出血，病因及部位不明者

4.影像学检查发现上消化道病变，需明确性质者

5.上消化道异物者

6.需内镜下治疗者，如食管胃底静脉曲张行套扎或注射硬化剂治疗，早癌行内镜下黏膜剥离术（ESD）等

7.胃癌家族史及其他胃癌高危人群

8.存在幽门螺杆菌感染，需明确是否有胃黏膜病变者，或需要进行幽门螺杆菌培养及药物敏感性试验以指导治疗者。


三、胃镜禁忌症


绝对禁忌：


  1. 各种原因导致的休克
  2. 严重的心肺疾病、昏迷、癫痫发作、主动脉瘤、脑出血
  3. 怀疑食管、胃、十二指肠急性穿孔
  4. 急性重症咽喉部疾病


相对禁忌：


心肺功能不全，出血倾向伴血红蛋白低于50g/L,高度脊柱畸形，食管或十二指肠巨大憩室等


四、胃镜检查前后的注意事项：


胃镜检查前需注意：


  1. 提前进行腔镜前血清检查的化验
  2. 检查前禁食水六小时，前一天少渣饮食(如稀粥、软饭、烂面条、肉末、菜泥等)，尽量不饮用牛奶或酸奶等影响检查效果的食物，禁烟酒。
  3. 高血压患者可正常服药；如果长期服用阿司匹林、波立维、三七等抗凝成分的中西药提前告知医生。
  4. 就诊时携带以往的就诊病历和检查结果以便医生参考
  5. 检查前应取下义齿（假牙），眼镜等物品，以免影响检查。
  6. 注意调整好心情，保持良好情绪，如有不适要立即告知医生
  7. 检查要求有直系亲属陪同，以便交代病情及签字


胃镜检查后注意事项：


  1. 检查结束后两个小时内不要喝水、进食，以免误入气管引起呛咳或发生吸入性肺炎。
  2. 检查中需要取病理活检，术后两小时宜进食温凉流质饮食，如米汤、菜汤等，并禁止吸烟饮酒，喝浓茶，以减少对胃黏膜创伤面的刺激。
  3. 检查后眼部可能会有疼痛及异物感或出现声音嘶哑，一般3-5天内会好转
  4. 结合身体情况适当休息，不宜做剧烈运动
  5. 检查后三天内可留意大便颜色是否正常，如果出现柏油样黑便，要及时就医。
  6. 无痛胃镜的患者术后不能驾车，不能从事高空、精细作业以防意外。


五、肠镜检查的适应症


  1. 原因不明的下消化道出血，包括便血和持续性便潜血阳性。
  2. 存在下消化道症状，如腹泻、腹痛、便秘、腹部包块、大便习惯改变等可疑结直肠疾病。
  3. 钡剂灌肠或影像学检查发现可疑病变不能定性。
  4. 炎症性肠病做鉴别诊断或需要确定病变范围、严重程度等以及治疗后的复查。
  5. 结直肠肿瘤术后或息肉切除术后的定期随访。


六、肠镜检查的禁忌症


  1. 严重的心肺功能不全
  2. 休克
  3. 大的主动脉瘤
  4. 暴发性结肠炎
  5. 中毒性结肠
  6. 晚期妊娠


七、肠镜检查前后的注意事项：


肠镜检查前注意事项：

  1. 提前进行腔镜前血清检查的化验
  2. 告知患者及家属肠镜检查的相关知识，积极配合医生。
  3. 高血压患者可正常服药；如果长期服用阿司匹林、波立维、三七等抗凝成分的中西药提前告知医生。
  4. 就诊时携带以往的就诊病历和检查结果以便医生参考
  5. 肠道准备工作：

（1）检查前2-3天开始少渣饮食，检查前一晚无渣饮食，检查当天禁食，但可喝糖水。

（2）肠镜下午检查：检查当日早晨7：00开始服用和爽3袋，当日11:00服用西甲硅油15ml。

（3）肠镜上午检查：前晚晚餐后2小时服用和爽3袋，3小时服完，检查当日6:00服用西甲硅油15ml。

（4）对于习惯性便秘的患者需在检查前1～2天服用缓泻剂或用开塞露塞肛，排出干结的宿便。

（5）服用洗肠剂注意速度，避免过急导致呕吐，并认真观察粪便直至成清水样。



6.检查要求有直系亲属陪同，以便交代病情及签字


肠镜检查后注意事项

  1. 检查后会出现轻微腹胀、疼痛等不适，与术中注气有关，一般会自行缓解
  2. 患者无腹痛、腹胀可安排进食，适当休息。
  3. 无痛肠镜的患者术后不能驾车，不能从事高空、精细作业以防意外。
  4. 进行活检的患者三天内勿剧烈活动。


八、无痛胃肠镜检查注意事项：


  1. 胃痛胃肠镜术前要求同普通胃肠镜检查。
  2. 预约前需患者本人进行麻醉评估，可去止痛门诊评估麻醉禁忌症，无禁忌症签字盖章后正常预约。
  3. 无痛胃肠镜禁忌症包括（1）患有严重的呼吸系统疾病（2）张口障碍、颈项或下颌活动受限、矮胖体型者会影响呼吸道开放，不宜做无痛内镜（3）非呼吸系统疾病如心力衰竭、心肌梗死、中风、昏迷、肝肾衰竭、严重贫血以及高龄衰弱等一般情况太差者（4）消化系统疾病如食管、贲门、幽门和肠梗阻、呕吐呕血者存在潴留液反流误吸者，应先进行胃肠减压（插胃管）等预处理，排出潴留液后进行无痛内镜（5）孕妇不宜做无痛内镜
  4. 麻醉方式采用的是静脉麻醉，术后24小时不可开车，不可进行登高等运动，不能签署重要文件等。


九、胃肠镜检查预约方式：


  1. 门诊患者在门诊开具申请并缴费后直接到诊断楼二楼与门诊楼连接处腔镜预约窗口进行预约
  2. 住院患者直接到预约窗口办理。


十、胃肠镜怎么取结果


胃肠镜报告三个工作日，病理结果五个工作日后会发往病案室，保存在患者病例中，门诊患者可前往并复印结果，或医生直接从HIS系统中调阅；住院患者直接发往病房，无需家属或患者个人领取。要后续内镜治疗患者，可以预约内镜门诊号，由内镜医生进行后续诊疗评估及预约。
