最近富民乡街子都有些啥？

2020-10-29
| 科学

作为一名地方自媒体小编

被各种各样的问题包围

是避免不了的一件事

每天上班打开后台消息列表

各式各样的问题

真的问得小编开始怀疑人生



前两天的问题当中

就有微友问小编

最近富民乡街子上都有些什么特产？

这个......

把很少上街的小编难住了

这不，今天亲自走一趟

嘿嘿，大兄dei

你要的答案在这里

富民乡街子，“土货”应有尽有

而且不仅只是土，有的还很“野”

什么！不信？

看图

▼

先让你看看有多“野”



/ 老董果 /

乍眼一看以为是酸多依，原来是“老董果”（不知道字对不对），富民人都喜欢拿来泡酒。查询度娘，好像学名叫“刺玫果”。


/ 酸多依 /

这才是酸多依，是不是跟上面的很像呢？在富民，有人喜欢拿来泡酒。


说到酸多依，小编家里喜欢把新鲜酸多依切丝切碎后，凉拌火烧的新鲜红辣椒，那味道...啊呀呀...太！下！饭！啦！或者把酸多依用盐水泡起来，把酸水当酸醋食用，“多依牌”天然酸醋，你值得拥有！


/ 卡卡果 /

这个又不知道字对不对了，对此比较陌生，据说也是泡酒搭档。



/ 鹿心草 /

别名：鹿仙草。据说是一味药材。可以煮鸡炖汤，也可以泡酒。


/ 野坝子 /

嘿嘿，这个就再熟悉不过了，富民人也叫苦里坝，居家必备凉茶饮品。



/ 八月瓜 /

小编表示对这个极其陌生，据说是一种野生水果，有谁吃过？


/ 野生葛根 /

比起种植的葛根，野生葛根虽然口感略苦，容易嚼渣，但是葛根粉含量高，清凉效果也更好。


/ 野生山药 /

野生山药也没有吃过，不知道跟富民本地种植的山药有什么区别。卖主说非常非常难挖！


/ 老人头 /（野生菌）

这个季节居然还会有野生菌？据卖主介绍说，也叫“山脑包”。好吃吗？



/ 鸡嗉子 /

鸡嗉子是一种野生的果子，熟透了的鸡嗉子果吃起来很香甜，老人说，把果核一起吞了可以消食？跟鸡啄食沙粒石子同理？问了度娘：其有清热解毒，利胆行水，消积杀虫的药用价值。富民人也是会用来泡酒。




唉！这一堆堆的，实在是不认识了......居然很多东西都不认识！

其实呢，小编认为，“野味”这样的东西吧，不论是具有养生价值的药材，还是山珍野味，大部分购买者都是经济状况稳定，寻求品质天然健康、养生意识高的人才懂享受的。像小编这样的人，最多也就买点蛤蟆叶、蒲公英、荠菜（还不一定买到野生的呢）这一类了野菜。所以，不知道也是可以理解的。


上河院菜市场，一位大妈卖的野菜，有蒲公英、小鹅菜、荠菜，确定都是野生的。

以上就是今天拍到的“野”货，当然了，这只是小编发现的一部分而已。好了，接下来看看富民最近街上都还有些啥？



汁水充盈的大梨，糖心的红苹果......

lia了一圈，就拍到这些，重点的基本都拍到了，至于很多细化的，等着感兴趣的你来慢慢发现！

看到这里，也别再问小编这个多少钱、那个怎么卖了，一句话：绝对新鲜，物有所值。自！己！来！逛！

为了拍照，被有的摊贩怼“拍了照片买一点嘛！”拍了这么多，逢拍必买的话，小编我买得完吗？主要是没钱！没钱！没钱！

总之，不管是什么样的摊贩，都是为了生活，都祝大家生意兴隆！也希望小编的这篇文章，会让更多的昆明周边乃至朋友来富民赶乡街子！
