关于​Flexbox你需要了解的5种技术

2020-11-14
| 科学

Flexbox是一个用于设计用户界面的css标准。使用各种flexbox属性，我们可以构建页面，从小积木中构建页面，然后轻松地定位并调整我们想要的任何方式。这种方式提供的网站和应用程序响应性好，并且适合所有屏幕大小。

在本文中，我们将讨论五种解决常见css布局问题的方法。我们还包含了一些实际例子来展示这些技术应用于实际生活场景中。

1.。创建等高列

起初这似乎并不是一项艰巨的任务，但是制作相同高度的列可能会非常烦人。简单设置min-height不会工作，因为一旦列中内容的数量开始不同，其中一些将会增长，其他一些将保持较短。

使用flexbox来修复这个问题并不容易，因为创建这样的列在默认情况下具有相等的高度。我们所要做的就是初始化flex模型，然后确保弯曲方向和对齐-项目属性具有默认值。

css：.container { /* Initialize the flex model */display: flex; /* These are the default values but you can set them anyway */弯曲方向: row; /* Items inside the container will be positioned horizontally */对齐-项目: stretch; /* Items inside the container will take up it's entire height */}
HTML：<div class="container"><!-- Equal height columns --><div>...</div><div>...</div><div>...</div></div>

2.。重排元素

不久前，如果我们必须动态地更改某些元素的顺序，我们可能会尝试一些css代码，退出时退出，然后用javascript进行操作。然而，使用flexbox，这个任务归结为应用一个单一的css属性。

它叫order就像它的名字一样，它的一切都尽可能简单明了。它允许我们交换任意数量的flex项目，并更改它们出现在屏幕上的顺序。它唯一的参数是一个整数，确定那个元素的位置-低数意味着更大的优先级。

css：.container{ display: flex;}/* Reverse the order of elements */.blue{ order: 3;}.red{ order: 2;}.green{ order: 1;}
HTML：<div class="container"><!-- These items will appear in reverse order --><div class="blue">...</div><div class="red">...</div><div class="green">...</div></div>

3.。水平垂直定心

垂直中心在css是其中一个问题，让我们问自己：如此琐碎的事情怎么会这么复杂呢？什么？而且真的是。如果你是谷歌垂直居中css，大量不同的技术将会出现，其中大多数都会涉及表格或转换-那些不是用来制作布局的东西。

Flexbox提供了一个更容易解决这个问题的方法。每个弹性布局都有两个方向(x轴和y轴)以及两个不同的属性来进行它们的对准。通过中心位置，我们可以将任何元素都定位在其父容器中间。

css：.container{ display: flex; /* Center according to the main axis */justify-content: center; /* Center according to the secondary axis */对齐-项目: center;}
HTML：<div class="container"><!-- Any element placed here will be centered both horizonally and vertically --><div>...</div></div>

4.。创建全响应网格

大多数开发人员在创建响应网格时依赖于css框架。自举是最流行的一个，但是有数百库可以帮助您完成这项任务。他们通常工作得很好，而且有很多选择，但往往相当重。如果你是一个diy的人或者不想为网格实现一个完整的框架，那么你就已经覆盖了！

在网格中的一行只是一个容器。display:flex.。它内部的水平列可以是任意数量的元素，它的大小是通过挠曲.。flex模型适应了视窗大小，所以这个安装在所有设备上都应该看起来很好。但是，如果我们决定屏幕上没有足够的空间，我们可以轻松地将布局转换成垂直的空间，并且使用媒体查询。

css：.container{ display: flex;}/* Classes for each column size. */.col-1{ 挠曲: 1;}.col-2{ 挠曲: 2;}@media (max-width: 800px){ .container{ /* Turn the horizontal layout into a vertical one. */弯曲方向: column; }}
HTML：<div class="container"><!-- This column will be 25% wide --><div class="col-1">...</div><!-- This column will be 50% wide --><div class="col-2">...</div><!-- This column will be 25% wide --><div class="col-1">...</div></div>

5.。创建完美的粘性脚注

Flexbox也有一个容易解决这个问题的办法。当构建包含粘性页脚的页面时，通过在flex元素中完成整件事情，我们可以确保我们的页脚始终停留在页面底部。

应用display: flex对于body标记，我们可以使用flex模式属性构建整个页面布局。一旦完成了这个网站，网站的主要内容可以是一个flex项目和另一个脚注，这使得操作它们的定位非常容易，并将它们精确地放置在我们想要的地方。

css：HTML{ height: 100%;}body{ display: flex; 弯曲方向: column; height: 100%;}.main{ /* The main section will take up all the available free space on the page that is not taken up by the footer. */挠曲: 1 0 auto;}footer{ /* The footer will take up as much vertical space as it needs and not a pixel more. */挠曲: 0 0 auto;}
HTML：<body><!-- All the page content goes here --><div class="main">...</div><!-- Our sticky foooter --><footer>...</footer></body>
