Java中的条件运算符

条件运算符（ ? : ）也称为 “三元运算符”。
语法形式：布尔表达式 ？ 表达式1 ：表达式2
variable x = (expression) ? value if true : value if false
运算过程：如果布尔表达式的值为 **true ，则返回 表达式1 的值，否则返回 表达式2
