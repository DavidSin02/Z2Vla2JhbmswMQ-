PTT VBA｜统一或指定幻灯片进行文本和段落格式化

2020-11-14
| 科技

整个Office系列软件都可以支持VBA二次开发，应当说，EXCEL对VBA支持最好，其次是Word，对于PPT来说，支持是较差的。

首先PPT不支持录制宏操作，其次，提供的对象的属性成员和成员方法也很欠缺。从其提供的帮助文档就可以看出：https://docs.microsoft.com/zh-cn/office/vba/api/overview/powerpoint

现在的任务是，对全部幻灯片及每一张的全部对象统一进行文本和段落格式化。

代码逻辑很简单，就是遍历全部幻灯片及每一个幻灯片的形状对象。

For Each oSlide In oPres.Slides……NextFor Each oShape In oSlide.Shapes……Next

然后对每一个对象应用属性。

只要熟悉相应对象语法即可。

运行效果：

当然，如果你只想设置其中的某一张幻灯片，或其中的第j个对象，可以把上面代码的两个循环去掉，具体指定幻灯片和第j个对象即可：

 i = oPres.Slides.Count j = 0'设置最后一个幻灯片或指定幻灯片的第一个对象的字体和段落格式 Set oSlide = oPres.Slides.Item(i) Set oShape = oSlide.Shapes.Item(1)

附代码1：

Sub allSlideAllShapes()

On Error Resume Next

'对象和变量声明

Dim oPres As Presentation

Set oPres = Application.ActivePresentation

Dim oSlide As Slide

Dim oShape As Shape

Dim tr As TextRange

Dim i As Long, j As Long

'遍历全部幻灯片及每一个幻灯片的形状对象

For Each oSlide In oPres.Slides

For Each oShape In oSlide.Shapes

With oShape '设置文本框的宽度和位置，适合只有一个文本框的，

'如果有多个，下面三行代码要注释掉，不然重叠到一起了

'.Left = 45

'.Top = 45

'.Width = 625

End With

If oShape.TextFrame.HasText = msoTrue Then

Set tr = oShape.TextFrame.TextRange

With tr.Font

.NameAscii = "宋体"

.NameFarEast = "宋体"

.Size = 28

.Color.SchemeColor = ppBackground

.Color.RGB = RGB(Red:=0, Green:=0, Blue:=0)

.Bold = msoFalse

End With

tr.ParagraphFormat.SpaceWithin = 1.1 '设置行距

Set tr = Nothing

End If

'oShape.Fill.Background '文本框背景色用幻灯片背景填充

'oShape.TextFrame.TextRange.IndentLevel = 1

Next

Next

'对象删除

Set oShape = Nothing

Set tr = Nothing

Set oSlide = Nothing

Set oPres = Nothing

End Sub
