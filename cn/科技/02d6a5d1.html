免费且支持 218 种格式的全能格式转换工具，这款神器你值得收藏

2020-11-14
| 科技

在线工具拥有无需下载、即点即用的优势，所以我们常会借助在线工具进行临时的格式转换工作。但大部分的格式转换工具仅支持少量格式，像支持图片转换的工具，就未必支持视频或文档的转换。如果你正在寻找一个支持齐全的文件格式的转换工具，不想每次再去寻找单独类型的转换工具，那么不妨试试今天推荐的 CloudConvert。

CloudConvert 最引以为豪的就是它支持 218 种文件格式的转换。无论是压缩文件、音视频文件还是文档、网页，CloudConvert 都能帮你完成转换。你可以在 这里 查看 CloudConvert 完整的格式支持列表。

支持的文件类别

在 CloudConvert 的网站上，你能看到它「Convert Anything to Anything」的标语。事实上，CloudConvert 不但支持转换同类别的文件格式（比如将 JPG 转换为 PNG），它甚至可以跨类别转换，比如将 DOCX 文档转换成 PNG。

选择文件格式

除了提供在线转换服务外，CloudConvert 还提供了 API 接口让我们扩展它的功能。你可以将它接入你编写的程序，或是通过命令行完成批量转换。如果你是自动化工具 Zapier 的用户，你甚至可以直接将其与 Zapier 整合，而无需编写任何代码。

强大的 API 支持

ConvertCloud 在 GitHub 上公开了多种语言所编写的官方程序包，你可以在 CloudConvert 的 API 页面 查看第三方程序包、应用实例及官方文档。

CloudConvert 不但支持从本地直接上传文件，还可以通过链接获取网络文件。在注册账户后，你还可以连接至 Dropbox、Google Drive 等云盘，直接从云盘获取文件。免费用户每天只能享有 25 分钟的转换时间，且文件不得大于 1GB。如果你有更多的需求，可以单独购买转换时间或进行订阅，你可以在 这里 查看收费标准。

除此之外，CloudConvert 还提供了相应的 iOS 客户端，你可以在 App Store 免费下载使用。虽然 CloudConvert 目前还没有开发桌面端应用，但是它也提供了一个 Chrome 应用，感兴趣的朋友可以从 Chrome Web Store 下载。
