程序设计篇之选择结构程序设计

2020-10-29
| 科技

1.关系表达式

①关系运算：是对两个量进行“比较运算”；

②关系运算符：<、<=、>、>=、==、!= ；

③优先级别：算术运算符、关系运算符、赋值运算符（由高到低）；

④关系表达式：用关系运算符将两个表达式（算术、字符表达式）连接起来的有意义的式子；

说明：数值量比较（数字的大小），字符量比较（ASCII的大小）；

⑤关系表达式的值：真（1）、假（0）；

⑥结合规则：左结合。

2.逻辑表达式

①逻辑运算符：&&(与)、||(或)、!(非)；

②优先级别：!(非)、算术运算符、关系运算符、&&、||、赋值运算符（由高到低）；

③逻辑量：当一个量（可以是基本类型的常量或变量）用作判时，

c编译系统认为：非0--真、0--假。

④逻辑表达式：用逻辑运算符将关系表达式或逻辑量简介起来的有意义的式子（单个逻辑量、关系表达式是逻辑表达式的特例）；

⑤逻辑表达式的值：真（1）、假（0）。

3.if语句

①if语句的三种形式：

1)条件执行 if(e) A;

2)分支选择 if(e) A; else B;

当e为真值（非0）时执行A，否则执行B或后续语句。A或B都可以是单一语句，也可以是复合语句；else部分不能独立存在，即else前一定有一个“;”，它必定是if语句的一部分。

注意：在c语言中，表达式e的值为非0时，系统均按“真值”处理。

3)else if形式（阶梯式的if-else语句）

if(P1) S1

else if(P2) S2

...

else if(Pn) Sn

else Sn+1

系统从上到下的逐个判断条件P，一旦发现某条件Pi满足时，则执行与它有关的语句Si，并跳过其它剩余的阶梯。若所有条件均不满足，执行最后一个else语句或后续语句。

注意：一般else与最近的if匹配。

②if语句的嵌套：else应与最近的if匹配；

由此可看出：通过加“{}”,可改变else的层位，从而改变程序的执行流程。

③条件运算符与条件表达式

条件运算符： ?和: /*唯一的三目运算符*/

条件表达式：e1 ? e2 : e3

执行过程：计算e1；e1值为非0（真），计算并返回e2的值否则计算并返回e3的值；

运算顺序：高于赋值运算，低于算术、关系、逻辑运算；

条件运算符的结合方向是“右结合”。

4.switch语句

亦称多分支选择语句。它比用嵌套的if语句实现多路分支问题，其程序结构清晰、易读。

①格式：

switch(e)

{

case c1: 语句1

case c2: 语句2

...

case cn: 语句n

[default: 语句n+1]

}

其中：表达式e可以是整型、字符型、枚举型；

常量表达式ci必须与表达式类型一致（整型与字符型通用）；

②执行过程：计算表达式e的值；若与常量表达式ci值一致，则从语句i开始执行；直到遇到break语句或switch语句的"}"；若与任何常量表达式值均不一致时，则执行default语句，或执行后续语句。

注意：常量表达式ci仅起语句标号作用，不作求值判断；常量表达式的值必须是唯一的，没有先后次序；多个case语句可共用一组执行语句
