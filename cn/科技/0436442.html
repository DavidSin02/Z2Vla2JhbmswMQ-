2017十大高分韩国电影排名 十部高人气韩国电影推荐

2020-10-29
| 科技

说起韩国电影，人们最常提到的关键词便是“感人”了，无论是《熔炉》还是《素媛》，不让人掉下几公升的泪水不罢休，但2017年的韩国高分电影却刷新了鹿豹座小编的旧观念，越来越多出色的类型电影让人们眼前一亮，大呼过瘾。接下来鹿豹座小编就为您整理出2017十大高分韩国电影，快来看看吧。

军舰岛

豆瓣评分：7.6

导演: 柳升完

主演: 黄晸玟 / 苏志燮 / 宋仲基 / 李贞贤 / 李璟荣

类型: 剧情 / 动作 / 历史

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-07-26(韩国)

片长: 132分钟 / 151分钟（导演版）

影评：典型的人物、善恶结构，体现了商业电影的典型性。考虑到导演的地位，充其量也就是一般之作。 崔七星和吴末子之间的爱情线、李姜煜和李素姬之间的父女爱，算是照顾到了各个年龄层，而且考虑出演演员宋仲基光复军所属的特殊要员朴武英一角的人气，票房会取得成功的。

之后

豆瓣评分：7.5

导演: 洪尚秀

编剧: 洪尚秀

主演: 金敏喜 / 权海骁 / 赵允熙 / 金玺碧 / 纪柱峰

类型: 剧情 / 爱情

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-05-22(戛纳电影节) / 2017-07-06(韩国)

片长: 92分钟

影评：洪尚秀电影中最具魅力的典型人物：不断试图逃避却又不断把自己推入淤泥的男性，突降凌辱却尚能维持无辜本质的局外女孩，跳开时空的限制，记忆与现实以暧昧的方式相互交融，虽然免不了自我重复之嫌，但电影人完全追随自我当下生活状态的又何尝不是一种勇气和理智。

王者

豆瓣评分：7.4分

导演: 韩在林

编剧: 韩在林

主演: 赵寅成 / 郑雨盛 / 裴晟祐 / 金雅中 / 柳俊烈

类型: 剧情 / 犯罪

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-01-18(韩国)

片长: 134分钟 / 157分钟(加长版)

影评：这是一部剧情连接最多韩国总统的韩影：《王者》的剧情故事表面上并不复杂，但细细咂摸之下，就会发现它的剧本野心很大——其时间轴长达30年（1983-2012），以电视新闻、条幅等多种形式，呈现出了7名、8任韩国总统的真实影像，他们分别是：全斗焕、卢泰愚、金泳三、金大中、卢武铉、李明博、朴槿惠。

青年警察

豆瓣评分：7.4

导演: 金柱焕

编剧: 金柱焕

主演: 朴叙俊 / 姜河那

类型: 喜剧 / 动作

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-08-09(韩国)

片长: 109分钟

影评：大陆也应该有这种剧本成熟的青春商业片，认真推敲每一个梗，该笑时笑，该紧张时紧张，该感动时感动——总之，让观众完全融入，记起自己的热血时光。双男主就像TVB学警吴卓羲陈键锋的设置，一个莽撞一个谨慎，超互补，各种互动也太有爱。

被操纵的城市

豆瓣评分：7.3分

导演: 朴光铉

编剧: 朴光铉

主演: 池昌旭 / 沈恩京 / 吴政世 / 金相浩 / 安在洪

类型: 动作

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-02-09(韩国)

片长: 126分钟

影评：片中好比游戏虚拟世界一样的现实情况和主演们华丽的动作戏抢人眼球。受时长所限，真相似乎反转得太快，影片反而给人另一种被操纵的影像的感觉，如果拍成电视剧，可能会让观众更从容地审视，审视影像，也审视自己的情绪。

一天

豆瓣评分：7.1

导演: 赵善浩

编剧: 赵善浩

主演: 金明民 / 卞耀汉 / 申惠善 / 赵恩恒 / 刘在明

类型: 剧情 / 悬疑 / 惊悚

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-06-15(韩国)

片长: 90分钟

影评：用轮回、投胎、灵魂这些纯粹东方的精神符号去解构纯西方的时间循环的概念。一天既是循环的一天又是片中死去孩子的名字，而和解之后因罪活下来的两个孩子一个承载了一天的肉体，一个承载了一天的灵魂，更多的是最后突出那一丝的人性。人生就是一场因果，每天都是现场直播轮回。结尾就是点睛之笔，编剧实在是太强了！

独自在夜晚的海边

豆瓣评分：7.1

导演: 洪尚秀

编剧: 洪尚秀

主演: 金敏喜 / 徐永嬅 / 郑在咏 / 文成根 / 权海骁

类型: 剧情

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-02-16(柏林电影节) / 2017-03-23(韩国)

片长: 101分钟

影评：这部新片是一种考验，也是一次坦率的问询。大体上存在两种观影视角：一是将其带入洪尚秀私生活，从台词中追索出导演心中的蛛丝马迹；二是由作品本身而非作者角度出发去思考。这两种立场其实互相抵触。了解八卦的人在观影之初，很可能会截取其中的个人化片断，猜想导演在借台词为自己发声。

不汗党

豆瓣评分：6.9

导演: 卞成贤

编剧: 卞成贤

主演: 薛耿求 / 任时完

类型: 剧情 / 动作 / 犯罪

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-05-17(韩国)

片长: 120分钟

影评：相比其他同类黑色电影，这部的摄影照明剪辑都好出彩，尤其是照明，光线的调度非常惊艳。演技当然也都在线。而故事方面，虽然反转无数次但其实还是蛮俗套的，《监狱》加《新世界》，这样的故事在韩国电影里真的已经太多了。真心求各位编剧以后不要再重复了。

蚯蚓

豆瓣评分：6.8

导演: 尹学烈

编剧: 金严昊 / 尹学律

主演: 金正均 / 吴艺雪

类型: 剧情 / 犯罪

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-04-20(韩国)

片长: 102分钟

影评：最黑暗的地方如果能被分享出来，那么即是光明。想起前年的那部「韩公主」，看得我心都要碎了。昨晚睡前看了林奕含事件的新闻，校园暴力和性侵就像龙卷风，而受害者像身处的暴风眼，束手无策、无法逃脱。当这个世界真正变得好起来，从来都不是柔弱者开始变得强大， 而是因为强大者成功地控制了自己。

爸爸是女儿

豆瓣评分：6.7

导演: 金惠荣

编剧: 崔允美 / 金娜里 / 乔晟佑 / 金吉森

主演: 尹宰文 / 郑素敏 / 申久 / 李日花 / 李美度

类型: 剧情 / 喜剧 / 奇幻

制片国家/地区: 韩国

语言: 韩语

上映日期: 2017-04-12(韩国)

片长: 115分钟

影评：个是改编自日本的电视剧《父女七日变》，电视剧没看过。上次看的韩国翻拍日本版电影是《幸运钥匙》（《偷钥匙的人》），看完再看日版，觉得韩版比较好看。片头树叶飘下，一下让人梦回《阿甘正传》。灵魂交换的梗其实真很老旧了，男女互换也很常见，不过父女互相印象中是第一次看到。搞笑的情节不多，桥段设计也没什么太大亮点，在韩国电影来说，这个片子有点平庸了，想起了《奇怪的她》。
