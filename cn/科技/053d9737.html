几个超级无聊的网页，真的比你还无聊

2020-11-14
| 科技

在小狐上班时没有思绪的时候，常常会对着电脑屏幕发呆！

那么在发呆的时候，小狐就会想今天要写什么，才能引起你们的注意！后来。。。小狐想到了！就是我在发呆时对着屏幕的这些网页。。。

想选题的时候

就会对着电脑屏幕，滑着鼠标滚轮，一直滑啊！滑啊！因为小狐知道这个网页划不到尽头。

在滑动鼠标滚轮的过程中，能给予小狐部分解压和灵感！总觉得手动一下才踏实！其实看似无底洞一般的无聊网页，是一位小伙伴他为了试验 CSS 在网页上到底能有多大，就制作了一个世界上最长的网页 —— The World Highest Website ！

这个网页的全部长度，或者更确切的说是网页上下翻转的高度，大约是 18.939 公里，通过 CSS 的页面压缩功能，让这个文档存在于网页之中，不得不说制作这个网页的朋友个使用 CSS 很熟练，同时他也应该是世界上最无聊的人了。

而这个网页，正如我上面所说的！它是有旅程终点的，只要按动键盘的 Ctrl + Home 和 Ctrl + End ,就能快速在网页里到达置顶和到达终点。

而起点和终点的彩蛋文字还是很励志的。

吃午饭的时候

而一个上午过去了！小狐还是没有找到选题，为了避免被狐友们发现小狐一早在划水，于是小狐只能设置一个快速而又有说服力的屏保，再去吃午饭。这时候小狐就用到这个网页：

里面有 10 款屏保，种类为 Windows 安装升级的界面，和 OSX 的界面！而简单的点开，按 F11 进入全屏和退出全屏即可，超级方便！

还有 B 格满满的 OSX ！

效果怎么样？是不是很真实！反正小狐就经常挂着 OSX ！应该谁也想不到一个人无聊到经常升级系统吧！

当然！还有杀手锏 ，在各个系统的升级或者开机界面，按空格可触发各个系统崩溃的界面，比如上面我们熟悉的新盖中盖 “ 蓝瓶 ” ！还有 OSX 的也有！

午休的时候

吃完饭午休的时候，小狐会带着耳机听一些 Future Bass 的雨！这个时候请戴上耳机食用。

开玩笑开玩笑！其实这是一个听下雨的网页播放器！里面播放的是下雨时的水滴声和打雷的轰鸣声！很舒服而又身临其境的感觉，一定要去试试！午休听着这个，小狐很容易进入睡眠状态。恰巧你在上班思考时，也能给予你不错的灵感。

继续想选题的时候

怎么办？还没想到！玩个音乐小游戏解解压放轻松就很容易想到了吧！

一款非常二次元的网站，界面很简单！按下 START 直接可以进入游戏界面开始， F11 全屏。

别看界面很炫酷！其实里面的音乐更炫酷，按下画面不同的区域有不同的人声声音，无意中就成了一首曲！这个一定要戴耳机食用呀。

快下班的时候

已经下班了！该怎么办？那小狐就以这个划水篇直接提交吧！
