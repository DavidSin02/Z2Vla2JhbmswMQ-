Typora-编写博客格式化文档的最佳软件

Typora 不仅是一款支持实时预览的 Markdown 文本编辑器，而且还支持数学公式、代码块、思维导图等功能。它有 OS X、Windows、Linux 三个平台的版本，是完全免费的。

作为技术人员或者专业人员，使用Markdown语法进行写作，可以提高文档的编写能力，而且格式也比较统一，支持编辑Markdown的软件也比较多，任何电脑都很容易查看或者编辑。

偏好设置

通过打开文件-偏好设置。Typora 中提供了大量有关 Markdown 的设置功能，你可以设置几乎完全适合自己的 Markdown 编辑器。下面我将介绍一些常用的功能亮点。

图片插入

Typora 的图片插入功能是广受好评的，你可以在Typora中直接使用右键复制粘贴将网络图片、剪贴板图片复制到文档中，或者拖动本地图片到文档中，Typora 会自动帮你插入符合 Markdown 语法的图片语句，并给它加上标题。

复制图片

而且也完全可以使用图床来保证文档在复制分享后图片仍能正常显示。

Typora 支持在拖动或复制粘贴网络图片后自动将其保存到本地。你可以在"文件-偏好设置-图像"中选择复制到哪个路径，什么情况下需要复制。

网络图片

这一功能保证了即使网络图片源失效了，你还有本地的备份可用。同时也能使你的文件夹更合理、完整。

实时预览

Typora提供实时预览功能，编写文字时会按照格式实时预览样式，而且Typora 保留了「源代码模式」。你可以通过视图-源代码模式或左下角的</>按钮进入。

侧边栏

Typora 会根据你 Markdown 标记的 H1、H2、H3…… 各级标题为你呈现大纲，类似导航。

空格与换行

空格：在输入连续的空格后，Typora 会在编辑器视图里为你保留这些空格，但当你打印或导出时，这些空格会被省略成一个。可以在源代码模式下，为每个空格前加一个\转义符，或者直接使用HTML风格的&nbps来保持连续的空格。

智能标点

智能标点是比较喜欢的功能。它可以自动帮你将特定引号转化为自己更喜欢的样式的引号。 这样就避免使用不同输入法或者不同文档引起的格式不一致的问题。
