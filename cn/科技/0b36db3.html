Excel如何设置表格边框样式，记住这几个关键属性就懂了

2020-10-29
| 科技

Excel表格边框的设置，对于制作一个漂亮的工作表，十分有用。

由于审美的差异，我们通常做的表格都是以黑色边框白底为主，或者说，有些人根本不知道除了黑白表格还有带颜色的。

如何设置表格边框属性呢，下面介绍一下，边框线型设置也就是Border样式。

比如粗细，单线还是双线，点还是点线结合，等等。

不同线的形状在应用中可以加强或减弱某些内容的突出显示或隐藏效果。

也是对表格视觉效果上的处理，进行一个最优化的组合。

最好学过一些简单的图形图象设计方面的知识，更加有用。

下图分别为单元格上、下、左、右设置不同的线型，举了一个例子。

线条样式

样式用LineStyle属性来设置

例如：

range("A1").Borders.LineStyle = xlContinuous'实线

把A1单元格边框设置为实线，其它如颜色和粗细均为默认值。

可参考边框线条样式表来设置不同的样式效果。

边框粗细

range("A1").Borders.Weight = xlHairline '细线

边框的组细用Weight属性设置。

参数可查看下图进行对比设置。

代码

函数调用方式

代码写到一个OLEobject按钮里面了，也可以放到任何触发的事件里面进行调用。

Private Sub CommandButton1_Click()			Me.UsedRange.Clear'清除表里的内容			setBordersStyle'调用设置边框过程End Sub

边框设置过程（设置边框上下边框）

Private Sub setBordersStyle()    Dim b As Border    Dim r As Range    Set r = Me.Range("A2:F2")    r.ClearFormats    Set b = r.Borders(xlEdgeTop)        With b            .Color = RGB(255, 0, 0)          '  .LineStyle = xlSolid            .LineStyle = xlDot          '            r.Value = "上线框_点划"            r.Columns.AutoFit        End With    Set b = r.Offset(1, 0).Borders(xlEdgeBottom)        With b            .Color = RGB(0, 0, 0)            .LineStyle = xlDouble            .Weight = xlThick            .TintAndShade = 0.1            r.Offset(1, 0).Value = "下边框_双线"        End WithEnd Sub

总体来说比较简单，但是通常在工具栏里也可以找到相应设置。
