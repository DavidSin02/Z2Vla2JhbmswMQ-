【iOS 开发 – SwiftUI 基础 2/12】运算符

2020-10-29
| 科技

这一节，我们学习一下SwiftUI中所使用的运算符，包括以下几种类型：

  • 算术运算符
  • 比较运算符
  • 逻辑运算符
  • 位运算符
  • 赋值运算符
  • 其他运算符

接下来，我们通过例子来具体讲解一下相关运算符：

注意⚠️：Swift中运算法的两侧需要同时使用空格，或者同时不使用空格，否则会报错！

1: 算术运算符

// 算术运算符let a = 10let b = 100let c = a + b // 加法，c = 110let d = b - a // 减法，d = 90let e = a * b // 乘法，e = 1000let f = b / a // 除法，f = 10let g = a % b // 取余，g = 10

2: 比较运算符

let a = 4let b = 6// == 用来判断两侧的取值是否相等// != 用来判断两侧的取值是否不等print(a == b) // 输出 falseprint(a != b) // 输出 true// >, < 分别判断两侧取值是否大于或者小于右侧取值print(a > b) //输出 falseprint(a < b) //输出 true// >=, <= 分别表示 大于等于 和 小于等于print(a >= b) //输出 falseprint(a <= b) //输出 true

3: 逻辑运算符

// 逻辑运算符包括 或 (||)、与 (&&)、非 (!)var a = truevar b = falseprint(a || b) // trueprint(a || a) // trueprint(b || b) // falseprint(a && b) // falseprint(a && a) // trueprint(b && b) // falseprint(!a) // falseprint(!b) // true

4: 位运算符

// 位运算符是将变量转换成二进制格式，然后按照对应位进行操作，包括 位与、位或、取反、异或、左移、右移。 其中异或的计算法则为："同出零，异出一".var a = 24 // 二进制 0001 1000var b = 14 // 二进制 0000 1100print(a & b) // 8 对应二进制 0000 1000print(a | b) // 28 对应二进制 0001 1100print(a ^ b) // 20 对应二进制 0001 0100print(~b) // 243 对应二进制 1111 0011print(a << 1) // 48 对应二进制 0011 0000print(a >> 1) // 12 对应二进制 0000 1100

5: 赋值运算符

严格来说赋值运算符只有等号 （=），但是在实际使用中，为了简化代码书写，Swift语言允许我们将上述提到的各种算术运算法、位运算符与等号相结合，构成新的赋值运算符，例如 +=，实际使用中 先操作 前面的加法，后赋值。详见下面的例子：

var a = 3var b = 5b += a // b = a + b, b = 8b -= a // b = b - a, b = 2// 其他组合依次类推

6: 其他运算符

// 1: 区间符号 "...", "<..", "..<"，表示一个连续的区间for i in 1 ... 3 {  print(i)}// 上面这段代码输出： 1，2，3。for i in 1 ..< 3 {  print(i)}// 上面这段代码输出： 1，2。// 2. 取负 "-"var a = -5let b = -aprint(b) // 5// 3. 正号 "+"var c = 5let d = +cprint(d) // 5// 4. 三元运算符 ？：// 问号冒号运算符与C语言中的规则是一样的，当问号前的变量为True的时候，输出冒号前的变量，当问号前的变量为False的时候，输出冒号后的变量，let a = 3let b = 5print((a > b) ? a : b) // 5

这一节，我们系统梳理了SwiftUI中所涉及到的运算符，这些运算符不是特别容易记忆，在后续的使用中，我们可以在本文或者官方文档中查询具体的使用方法。

此外，运算符的优先级中，大的原则是高优先级的先算，低优先级的后算，其中单目的运算符优先级最高（正负号），然后是先乘除后加减，先算术>移位>位运算>逻辑运算。 同等级，从左往右按结合顺序计算。
