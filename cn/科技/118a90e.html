90后文物修复师，化腐朽为神奇，赞！

2020-10-29
| 科技

（更多精彩，欢迎关注：知羹）

首先，请仔细看这两张图：

这是一件名为《锺馗骑驴图》古画的修复前后对比图，在看第一张图时完全想象不到它原本破烂、满是霉斑的样子。

还有这些：

民国 /《虎》

原画直接从中间裂开了一个大口子，老虎头部也有许多缺失，修复后几乎看不出裂开的痕迹，虎头部分做到了完整的还原。

宋代 / 吉洲窑窑变釉茶盏

看完这两张对比图，不禁想到了那句古诗「粉骨碎身全不怕」。修复后茶盏不仅平整光滑，而且连釉面的光泽度也丝毫不差。

清代 / 字汇（亥部）

虫蛀、污渍、褶皱......这本古籍到底经历了什么？修复之后，边页变得平整，也保留了原本的历史感。

众多被修复的文物中最久远的一件是下面这张，距今至少有1000年的历史了。

▽

五代 / 越窑青釉水盂

这些被“救”活的文物，都出自南京艺术学院14级文物鉴赏与修复专业。

根据种类，又分为书画、古籍和陶瓷三个方向，每个方向都有其神奇之处。（神奇在哪？往下看）

书画 /////

《民初百子闹春图》

年代：清末民初 / 作者：曾博

边缘充满褶皱，还有不少已经凋落，污渍的多年堆积也使整幅画面模糊不清。

修复后，终于看得清画面上的“百子”，就连放爆竹、打闹的细节动作都十分清晰。

《猛虎图》

年代：近代 / 作者：华文韬

透过破碎的画也能感受到老虎的霸气，但就是破碎太多，虎爪和虎头都有明显缺块。（破成这样，山大王不要面子的嘛？）

破碎处修复得毫无痕迹，就连老虎的胡子都根根分明，霸气！

《虎》

年代：民国 / 作者：李婕仪

又是一只虎，但这只虎好像是在寻找什么。

整幅画面变得干净了许多，最吸睛的是虎头部分，本来缺一块的虎眼也做到了毫无PS痕迹。

《大展宏图》

年代：现代 / 作者：沐晶晶

这幅画的整画幅都充满着褶皱，画的右上部分还缺了很大一块。

修复后不仅画面变得平整了很多，而且还细心地将右上方的缺失部分做旧，以免与整幅画不搭调。

《溪山行旅图》

年代：现代 / 作者：邱骆圆

这幅山水画的顶部直接被消失了，右侧也有许多陈年的污渍。

修复后画面变得清楚了许多，前景的挑担小人也十分可爱。

《锺馗骑驴图》

年代：清末 / 作者：水豪杰

这是最幸运的一幅画了，主体部分还算完好，丢失的部分刚好避开了画作的主体。

在修复后，画作大量的留白给观者十足的想象空间，画中的钟馗还有一些萌萌的。

《巧姐》

年代：现代 / 作者：陶颖

保存还算完好，但细节部分缺失较多，从图片上根本无法分辨巧姐的五官，背景也模糊不清。

看完修复后的图才知道原来是巧姐在劳作，裙摆上的吊穗以及花纹都跃然纸上。

《硕果累累》

年代：民初 / 作者：陶源

哪来的有心人，正正好好将画作分成了十六份。画布不仅泛黄，部分破损处还丢失了。

原来这幅画作原本的样子这么白皙，修复后几乎看不出原本的裂痕，好棒！

《策杖访友图》

年代：近代 / 作者：王铭钰

老友、仙鹤、童子，在霉斑和破碎的痕迹下只能隐约透露出来...

可是在修复者的巧手下，画作不仅完整了，身后童子、身下仙鹤，更体现出了原本老者访友的意境。

《西施浣纱图》

年代：清代 / 作者：熊沛霖

这幅画作破损时的颜色本没有这么鲜艳，但在修复后，鲜艳的色彩和绿色的前后景将西施的美描绘了出来，一颦一笑皆为绝美～

《东风浩荡春似锦》

年代：近代 / 作者：姚昊鹏

这仿佛是加上了「旧画」滤镜，原色被污渍挡住了不少，也缺少了春的味道。

摘掉“滤镜”后，画作更加明朗鲜艳，两只可爱的小鸟也在花枝头间嬉戏。

《张仲林书法》

年代：现代 / 作者：尹秋力

这实在是太有视觉冲击力了！这一堆破烂的纸张到底是什么？

“拼”完之后呈现出的是80年代的书法作品。

* 其实以上有许多的书画作品原本都是这样的，修复的每一个步骤都充满着挑战。就比如这里的“拼图挑战”。

《常建樟书法》

年代：清末民初 / 作者：陈俊彦

这幅书法作品虽然能看清出字迹，但多年的色素沉积使作品不够干净，也掩盖了原本的字之美。

修复后的作品，比划清晰可见的浑厚，就连印章也有边有角。

《贵》

年代：现代 / 作者：潘思梵

最喜欢的就是这一幅了，因为它“贵”哈哈哈哈。纸张下端十分薄，还有许多零零散散的碎片。

修复之后，比划间的藕断丝连流畅，充满线条美，“贵”字，活了！

古籍 /////

古籍的繁复，是在于一整本每一页地修复。

细节处连古籍的托纸材质、书芯最大宽、PH值等都要一一测量，精准到毫厘。

《古本祭文书》

年代：清代乾隆 / 作者： 王泽钰

就比如这本古籍的封面就缺失了很大一块，角落处也有很多褶皱。（强迫症绝对忍不了！）

原本灰尘污染严重，染色严重的古籍，在修复后还有一种通透的美感。

《花样集》

年代：清代道光 / 作者：陈凯奕

这本古籍出现了一定程度上的脆化，每张书叶都存在虫蛀，污染面积也超过了1/3。

修补后补全了被虫蛀的地方，也最大程度地降低了污染面积。

《民歌集锦》

年代：清代 / 作者： 韩芳

这本古籍的书叶纸张厚度为0.064cm，称作薄如蝉翼也不足为过了。

修复者用了宣纸作为托纸，为古籍增添了一份保护衣。

《花样集锦》

年代：清代 / 作者： 李璐

这本古籍在保存过程中，染上了油渍、水渍、霉斑，在大部分书叶死角都存在着斑驳。

修复采用了“金镶玉”的装帧方式，用多出来的托纸（图中滴白色部分）保护原本的古籍。

黄兆奎书《画禅室随笔》

年代：清代 / 作者：李盼

换了另一种装帧方式，再加上修复者精心挑选的书套，是不是很炫酷？

《明代罗教宝卷》

年代：明代 / 作者： 彭晨

这本古籍字迹部分保存还算完整，但折痕明显，夹杂着很多污渍。

修复后将所有的文字部分保存完整，在后面附上了一层宣纸保护原件。

《管韫山全集》

年代：清代 / 作者： 王芳

修复者正在拿毛笔修补虫蛀处，左手是补纸，每一页都要如此，想象一下......

《新刻异说南唐演义全传》

年代：清代 / 作者： 王婷

这本线装古籍的书芯最大长为22.6cm，书芯最大厚为0.5，书本的脆化程度也十分严重，看图！

《禁扁 丁、戊》

年代：清代康熙 / 作者： 夏艳琳依

古籍的书衣有破损，书口也出现了断裂。

修复后的内页部分完好无损，并且书的切面十分平整，有一种新书做旧的感觉哈哈哈。

《字汇 （亥部）》

年代：清代乾隆 / 作者： 杨梦桐

污渍太多，连书本上的字都快看不见了。

《四书六经义》

年代：清代光绪 / 作者： 张辰芳

经历过时间的冲刷之后，产生了脆化，感觉随便一翻就会“粉骨碎身”...

只能说：修得这么好，太不容易了！

《澄衷蒙学堂字课图说》

年代：民国 / 作者： 张晓楠

古籍的书叶一角被老鼠啃出了一个梯形，还有不少书叶掉落。

修复后还原了古籍的历史感，在书角处还做了弧边处理。

陶瓷 /////

作为三个方向里最特殊的存在，陶瓷修复会根据陶器本身运用不同的技术手段，对比也是十分有视觉冲击力。

《宋湖田窑印花花卉纹盘》

年代：宋代 / 作者：杨旸

陶瓷碎成了八块，在第一步拼接是一定要先拼锐角，才能有利于修复的平整。

听修复者说：在你没有把它拼起来之前，你永远不知道它是个啥！

《清景德镇青花花卉纹盘》

年代：清代 / 作者：陈雨婕

由于青花产生不稳定的原因，修复青花瓷器的难度在陶瓷方向的同学心中堪比高考数学最后一题的难度。

《宋代青白釉花口瓜棱瓶》

年代：宋代 / 作者：郝梦

这个青白釉花口瓜棱瓶只剩下了底座和瓶身的部分，瓶口部分缺失。

瓶口在修复者的观察走向以及参考历史文献后，做成了波浪形，静态的瓶身有了动态的美感。

《宋湖田窑兽首三足炉》

年代：宋代 / 作者：李雅倩

小尺寸的香炉在古代往往用于焚香，炉口部分缺失近三分之二。

用石膏修补完之后，还原了三足炉的原样，古器在修复后反倒有了一种反差萌的感觉？

《南宋龙泉窑青瓷凤耳瓶》

年代：南宋 / 作者：李颖

青瓷瓶的瓶身上方一处脱釉，在修复后，完全看不出来，也太赞了。

《清康熙青花缠枝花卉纹碗》

年代：清代 / 作者：栾燕菲

这怎么这么像小时候家里的青花碗？（别做梦了）

清代的青花碗在碗侧丢失了一块碗壁，确实的青花部分也需要修复者重新填色，我现在已经开始怀疑自己的视力了。（为啥我对照第一章都看不出第二个缺失了哪？）

《宋当阳峪窑绿釉刻花花卉纹瓷枕》

年代：宋代 / 作者：裴雪敏

数了一下，大大小小一共27块，光看碎片完全看不出这是个啥。

拼完之后是个瓷枕，绿白的经典配色，加上藤枝的花纹，更增加一份东方神韵。

《清景德镇窑白釉铺首炉》

年代：清代 / 作者：屈婉婷

破碎的瓷器的边缘都不平整，用石膏还原后的拼接感还有另一种特殊的美感。

《宋代吉州窑窑变釉茶盏》

年代：宋代 / 作者：戎悦

又是一个让人怀疑视力的作品，光看第二张图真是想象不出它原来的样子。修复后的釉面质感很好的掩盖了裂缝处痕迹。

《建窑黑釉瓷碗 》

年代：宋代 / 作者：阮博

这是修复中的样子，破碎处已经用石膏进行填补。

碗内的金属光泽也拿颜料做了补充，与原物做到尽量的协调补充。

《清代豆青釉青花五福捧寿纹盘》

年代：清代 / 作者：史雪枫

寿纹盘是一个对称的圆盘，对照两张图，你能找出缺失的部分吗？

《清雍正景德镇窑青花过墙云龙纹盘》

年代：清雍正 / 作者：孙含蕙

这个作品除了完整了整个盘子，龙爪处也做了一定的补充。

《清雍正景德镇民窑青花云龙三现纹盘》

年代：清雍正 / 作者：朱黎

这个瓷盘与上一个有一定的相似之处，但不同的是修复者运用了金箔和黑色的颜料，体现出了另一种美感。

《宋吉州窑窑变釉碗等》

年代：宋、元 / 作者：卞丽君

修复中....

五个瓷碗各有不同（最喜欢蓝的那个），所缺的部分也用了最适合的修复手法，保存了原件的历史感。

《宋吉州窑黑釉木叶纹盏》

年代：宋代 / 作者：孙悦

中间这一块深浅不同的黑色就是碗身的碎片，你看出来了吗？

从一块碎片到一个碗的故事...... 太神奇了！

《民国粉彩福寿纹三多盘》

年代：民国 / 作者：陶思妤

瓷器中少见的彩色，这件粉彩盘的色彩清爽，是瓷器中少有的小清新了，完整后的效果更加惊艳。

《清末粉彩万寿无疆纹盖碗》

年代：清末 / 作者：王琰

这件纹盖碗采用了偏现代化的修复手法，用金箔覆盖裂痕处，既保存了原本的样子，又给予了文物新生。

《明龙泉窑莲瓣纹盘》

年代：明代 / 作者：王杨硕

可以从碎片中看出，盘底缺失了一块，釉面基本保存完整。

修复后，盘内的花纹与原本做到了对齐。裂缝之间的空隙也填补到了最小。

《汉红彩陶罐》

年代：汉代 / 作者：吴学翰

作为这些文物中较古老的存在，陶罐保存至今已经少之又少。

修复时更保存了罐子原本的历史感，人工做出了斑驳和时间累积的污渍，给予历史最大的尊重，点赞！

《清光绪民窑青花山水人物纹》

年代：清光绪 / 作者：项影

这件瓷器虽然破损部位较少，但修复效果十分惊艳，告诉缺口你也找不出系列。

《宜兴紫砂金缮锔钉杯》

年代：现代 / 作者：杨晓熠

喜庆的橘红色加上圆滚滚的杯型，看着还有些可爱，修复者还加了朵金色的梅花，俏皮中不失优雅。

《宋白釉菱纹花口尊》

年代：宋代 / 作者：周禺含

修复者有心的将所有的碎片编上号，以免拼接时出现差错。

瓶身的花纹和光泽感都完好无缺（石膏的运用是用来保证文物的真实性），整个瓷瓶呈流线型，庄重感十足。

写在最后

每一个文物都是经过了层层把关，PH值不均匀就一遍一遍地过滤，拼的不平整就重新来直至光滑，光揭背这一项程序能进行一个月......

正是因为他们的出现，那些原本破旧的文物，才有了新的生命和更高的艺术价值。

我只能说，文物鉴赏与修复专业，你们太太太太赞了！
