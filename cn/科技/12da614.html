Linux这5款微型发行版，体积小+精简，比win7运行还快，值得安装

2020-10-29
| 科技


（此处已添加圈子卡片，请到今日头条客户端查看）


Seth Kenlon曾经说过：“这些微型 Linux 发行版可以让你的老爷机复活，可以启动一个损坏的系统，或者是确保在公共电脑上进行安全的操作。”众所周知，能够为我们所用的Linux发行版有很多，但大家是否听说过Tiny Core、Mint等等发行版呢？



这些发行版体积小，使用一整套完整的操作系统驱动，只需一台不到1GB和512MB内存的计算机即可，堪称是终极的黑客作风！那么接下来老毛桃就跟大家分享5款微型Linux发行版。



1、Linux Mint


Linux Mint是一种基于Ubuntu 或 Debian 开发出的 Linux发行版 操作系统，它继承了Ubuntu 的众多优点，同时也在 Ubuntu 的基础上加入很多自己优秀的特性。为了追求易用性和良好的人机交互体验，Linux Mint 集成了可以开箱即用的大量多媒体等软件包。因此它的 Live ISO 体积超过了 ubuntu 的本地安装镜像，甚至大于 950MB。



此外，为了迎合不同用户的口味，Linux Mint 官方提供了 4 种不同的桌面环境版本可以供你选择下载，分别是Cinnamon、MATE、KDE和Xfce，感兴趣的小伙伴可自行了解。如果你正在为台式机、笔记本选用一款桌面级的 Linux 发行版，那么 Linux Mint 可能会让你感觉到相见恨晚！它简单易用，桌面优雅舒适，速度流畅且非常稳定。


2、Tiny Core


Tiny Core Linux是一款体积极小且高度可扩展的微型Linux发行版，终端版本只有 11 MB，图形界面版本只有 16 MB。由于设计得极端精简，甚至安装完整操作系统的应用程序都没有被包含在内，安装后我们需要通过以太网下载需要的应用程序。



3、SliTaz Linux


SliTaz Linux的镜像文件大约在51MB左右，虽比Tiny Core的大了四倍，但比较于Windows系统也算是微型系统了。即便微型，但它还是包含了一整套出色的驱动程序和应用程序，能够为大家提供的应用程序能够满足大多数基本的需求。



4、Porteus Linux


Porteus是微型 Linux 中镜像文件最大的一个，最小的镜像文件大约在270MB，最大的则有350MB，但是这些额外的空间都被用来确保一个非常顺畅的 Linux 桌面环境的体验。我们可以在MATE、LXQT、LXDE、OpenBox、XFCE、Cinnamon 或者 KDE 里选择自己喜欢的桌面环境。在此老毛桃给大家推荐MATE 或者是 KDE 桌面，它们可以为你提供熟悉的桌面环境体验。



5、Bodhi Linux


Bodhi Linux系统以占用系统资源小，速度快的Enlightenment窗口管理器为基础，打造了一个优雅干净，来自未来的桌面环境。关于应用程序，大家可以在Bodhi Linux的库中找到内置的应用程序，可供使用的可谓是数不胜数！



以上5款微型发行版体积小且精简，如果你有台旧电脑，不妨试试？
