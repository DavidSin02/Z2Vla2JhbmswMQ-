分享一个漂亮的查询表，自带切片图表，查询很给力！

2020-11-14
| 科技

工作中，经常会遇到多个表数据合并进行汇总的问题，一般都是先进行表格合并，再进行数据汇总统计。有时候数据汇总统计时不一定要对表格进行合并，也可直接对多个表数据进行查找汇总，这样就省去了多表合并的麻烦。今天给大家分享一个无需合并直接对多表数据进行汇总显示的查询表，通过几个小技巧的运用，查询起来非常高效便捷。

先看效果

除查询表外，其他均为基础数据表，每个表均由单位名称和年度月份作为表单名，数据不需提前合并。在查询表里将表单名分解为单位和年度月份作为查询条件，另外将文件里人员名单也作为查询条件。

功能小技巧

1、首先通过下拉菜单选择查询单位，即所属国家。

小技巧：上图中查询单位的下拉菜单选项直接由代码生成，省去了辅助数据区的设置，表格更加干净精炼。重点是当选择了查询单位，比如选择“蜀国”后，另外三处输入框查询员工、开始期号、结束期号均自动生成对应的二级下拉菜单，同时查询员工框显示第一个员工姓名，开始期号显示第一期工资日期，结束期号显示最后一期工资日期。于此同时激活查询功能，将工资信息分期列到表中并在表末进行汇总合计。

主要代码：

Private Sub Worksheet_Activate() Call 查找单位列表(Range("d3"))End SubSub 查找单位列表(rg As Range)Set d = CreateObject("Scripting.Dictionary")For i = 1 To Sheets.Count If Not IsError(Application.Find(201, Sheets(i).Name)) Then  If Mid(Sheets(i).Name, Application.Find(201, Sheets(i).Name), 3) = "201" Then nh = Left(Sheets(i).Name, Application.Find("201", Sheets(i).Name) - 1) d(nh) = "" End If End If Next s = Join(d.keys, ",") With rg.Validation .Delete .Add Type:=xlValidateList, AlertStyle:=xlValidAlertStop, Operator:= _ xlBetween, Formula1:=s End With Set s = NothingEnd Sub

2、选择查询员工输入框可通过二级下拉菜单进行选择输入，当员工人数较多时，下拉菜单很长，且由于菜单间距较小，为避免选错。在表单右侧列出了当前单位有工资信息的所有员工姓名，直接点击名字，查询员工工资。

小技巧：为了直观显示当前查询工资的员工姓名，右侧列表中将当期员工姓名涂色进行标记，下拉输入框的员工姓名和右侧选择姓名同步对应，即当从下拉菜单中选择了某员工时，右侧该员工名字随之也被涂色标记。

主要代码：

 arr = Split(s, ",") hzbb.[k7].Resize(100, 1).ClearContents hzbb.[k7].Resize(UBound(arr) + 1, 1) = Application.Transpose(arr) hzbb.[i3] = hzbb.[k7] Call 赋色(hzbb.[k7])

小技巧：当需要按顺序依次查询工资时，为更加便捷，设置了通过点击右侧的向上、向下图标，依次选择员工姓名进行查询的功能。

主要代码：

Sub 上一个员工() Set hzbb = Sheets("汇总报表") If hzbb.[k7] = "" Then Exit Sub For i = hzbb.[k60000].End(3).Row To 7 Step -1 If hzbb.[i3] = hzbb.Cells(i, 11) Then If i = 7 Then i = hzbb.[k60000].End(3).Row + 1 hzbb.[i3] = hzbb.Cells(i - 1, 11) Exit Sub End If NextEnd SubSub 下一个员工() Set hzbb = Sheets("汇总报表") If hzbb.[k7] = "" Then Exit Sub For i = 7 To hzbb.[k60000].End(3).Row If hzbb.[i3] = hzbb.Cells(i, 11) Then If i = hzbb.[k60000].End(3).Row Then i = 6 hzbb.[i3] = hzbb.Cells(i + 1, 11) Exit Sub End If NextEnd SubSub 赋色(rg As Range) Set hzbb = Sheets("汇总报表")  hzbb.Range("K7:k15").Interior.ThemeColor = xlThemeColorLight2 hzbb.Range("K7:k15").Interior.TintAndShade = 0.599993896298105 hzbb.Range("K7:k15").Font.ColorIndex = xlAutomatic rg.Interior.ThemeColor = xlThemeColorAccent6 rg.Interior.TintAndShade = -0.249977111117893 rg.Font.ThemeColor = xlThemeColorDark1End Sub

3、右侧设置自动生成图表，图表标题、数据内容根据左侧表格数据联动，通过选择不同的人名，左侧汇总工资和右侧图表随之更新，实现高版本excel里图表的切片器功能。

Sub 显示图表() Set hzbb = Sheets("汇总报表") Set ab = hzbb.Range("m3:w15") Set tb = ActiveSheet.ChartObjects With tb .Visible = True .Left = ab.Left .Top = ab.Top .Width = ab.Width .Height = ab.Height ' .Name = "图表1" End With ActiveSheet.ChartObjects(1).Activate With ActiveChart .SetSourceData Source:=hzbb.Cells(8, 2).Resize(hzbb.[b30000].End(3).Row - 7, 8) .ChartType = xlColumnClustered '选择柱形图样式 .ChartStyle = 29 '选择绿色图表类型 .SeriesCollection(1).XValues = "=汇总报表!$D$7:$I$7" '选择X轴项目 .ApplyLayout (5) '选择有顶标题和下面列表的图表类型 .ChartTitle.text = hzbb.[i3].Value '设置顶部标题 .SetElement (msoElementPrimaryValueAxisTitleNone) '取消Y轴标题 .ChartTitle.Font.Name = "楷体"End WithEnd Sub

4、设置开关按钮控制图表显示或隐藏。

Private Sub onoff_Click()If onoff.Value = False Thenonoff.Caption = "图表隐藏"ActiveSheet.ChartObjects.Visible = FalseElseonoff.Caption = "图表显示"ActiveSheet.ChartObjects.Visible = TrueEnd IfEnd Sub

总的来说，本表看似简单，但细节处理很到位，自动生成一、二级下拉菜单、自动生成图表、图表联动、设置切片器等功能均为实际工作中有一定难度的功能。至于主功能方面，需根据公司的具体要求专门设置，难度相对较小。
