安卓APP开发 | 简单学Java从编程入门开始-基本数据类型

2020-11-14
| 科技

新手在接触Java编程时，除了要熟悉代码中关键字的使用方法外，想要操作程序中的数据，必须要了解其代码中基本的数据类型。通过对代码中数据类型的熟悉，我们才能知道如何去传值和改变数据内容。

数据类型能够对程序中的数据进行归类，让我们开发者对数据的内容有更精确的操作。希望大家在使用的时候能够分清楚传值该用哪种数据类型。

基本数据类型的定义和变量

1.什么是基本数据类型？

基本数据其实就是某件事物的最基本信息。比如描述一个人的时候，我们可以通过他的年龄、姓名、性别、是否结婚、收入等基本信息来确定一个人的最基本信息。

我们可以看到“人”的基本数据类型：

/** 人的基本数据类型 **/public class Person{      int 年龄;     char 姓名;     boolean 是否结婚;    double 收入;}

从上面的代码，我们可以看到人的每个基本信息中，它对应的数据类型关键字的变量声明是不一样的，比如年龄使用的是int，姓名使用的是char，不同的数据内容采用不同的数据类型。

2.什么是基本数据的变量？

基本数据变量主要分为全局变量和局部变量。

/**个人信息**/public class Person{     int 年龄 = 13;  //全局变量    public void addAge(){			//个人更改年龄的方法函数  	年龄 = 56；			//局部变量    char 姓名 = 'L'；//局部变量  }  }

所谓的全局变量，就是在一个类中的公有变量，在类中的所有方法都可以调用这个变量。

局部变量，则是在类中某一个方法或某一位置发生数据改变的变量，它可以是全局变量的局部变化，也可以在代码的函数中去定义变量。

基本数据类型的分类

对于Java语言而言，它的基本数据类型如下：

Java语言的基本数据类型

1.整数类型：byte、short、int、long

2.浮点类型：float、double

3.字符类型：char

4.布尔类型：boolean

根据上面的基本数据类型的分类，我们可以看到Java语言的数据类型主要分为4种。

其中，整数类型的取值为整数，比如1，2，3，4，5等；

浮点类型的取值为小数，比如1.12，3.145，3.148等；

字符类型的取值为单字符，比如'X','Y','x','y';

布尔类型的取值为布尔值，布尔值即true或false，表示是和否。

/**个人信息**/public class Person{	  byte b = 23;				//字节整数类型，取值整数，默认值为0  	short s = 3344;			//短整型，取值整数，默认值为0   	int	i = 3333;				//整型，取值整数，默认值为0	long l = 32322;			//长整型，取值整数，默认值为0L	double d = 333.33;	//双精度浮点型，取值为小数，默认值为0.0d	float	f = 333.33f;	//单精度浮点型，取值为小数，值得后面加小写f,默认值为0.0f		boolean b ;					//布尔类型，取值为布尔值，有判断两面性，默认为false  }

注意：布尔类型的值一般在没有赋值的情况下，默认值为false.

结语

在进行Java语言编程时，程序中很多代码都是围绕数据传递进行，这要求我们编程人员需要对代码中的数据类型要非常熟悉才行，保证传递的数据内容能被正常传递。我们在后续的数据库学习中，也要会定义表中字段的数据类型，保证数据库服务端和程序中的数据类型必须一致。

Java语言的基本数据类型总共就8种，在使用时要加以区别，以免关键字的变量声明错误，希望大家能够认真了解Java语言的基本数据类型，这是数据传递的最核心部分。
