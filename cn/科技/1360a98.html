圣美歌EA400.4S四路功放 让车友越来越烧得起的“纯后级”高端货

2020-10-29
| 科技

DSP作为音频处理器以及DSP功放推动系统的玩法越来越盛行，音改市场上的整体玩法开始走向后级处理器设计或搭配上的简化，也就是把前级处理部分全交由DSP处理。不少老江湖既明白车厢环境实现理想音频的复杂化，同时也深知对于功放来说前级调控越少距离发烧就越近。市场上不少单路和二路功放早已出现纯后级产品，而为了照顾通用性四路功放就少见这类产品。

为照顾车友或者爱好者对发烧汽车音响的追求，SOUNDMAGUS圣美歌早段时间就推出了M1纯后级Hi-End单路放大器，音质更是不容置疑地得到认可。贯彻让车友越来越烧得起的思路，圣美歌最近也推出旗下EA系列高端发烧音频功放的改进版——全新的EA400.4S四路功放。

EA400.4S在输入端和电路方面的设计做出了不少改进，不能说其直接以纯后级方向发展，但对比“前者”EA400.4来说，精简优化让功放变得更为高效，也更贴近车友最理想的发烧功放趋向。功放的整体设计没有大面积升级改进，保留了圣美歌EA系列的黑色拉丝面板设计，系列十分优秀高效的体积得以保留，接近一张A4纸大小的体积是EA功放一大亮点。

从输入端看，我们会发现EA400.4S对比老款优化精简了不少。左侧是四声道的RCA输入端子，以及四/二声道输入转换按钮，按下功放只进行二声道输入，不按下就是一台四路，原来在下方的输入电平按钮取消。功放设计为纯以RCA低电平输入信号，所以老款上的REM和Audio等自动信号开关也被取消。右侧的1、2声道调节上，可以选择全通或者高通，频响范围在50Hz-500Hz可调以及GAIN增益可调。下侧的3、4声道调节上，可选高通、全通和带通，高通全通频响为0Hz-300Hz，带通则在50Hz-500Hz之间可调，同样提供GAIN增益可调，此外还预留了超低音线控插口。

对比老款EA400.4输入端

EA400.4S连两侧的金属倒流散热格栅也与老款同出一彻，拆开底板会发现内部电路是用上双面玻纤电路板，以SMD音频专用贴片元件保证超低内阻内扰以及维持高保真输出。功放在4Ω状态下可提供100W每通道的输出，2Ω下为每通道120W，进行桥接输出则可达两通道220W。大型的“牛”使用MCU控制数字电源平台，保证大动态输出也有足够的电流稳定供应。附近分布的则是大容量电解电容，以及MOSFET场效应管。应对AB类电路加上发烧音质处理上可能产生的高热量，EA400.4S加入完善保护功能让功放在过载、欠压、过压以及过热等情况下也能稳定工作。

360汽车网点评：乐迷都懂后级处理越是减少前级各种调控和处理，越是“简洁”的电路处理出来的效果越能接近理想的发烧音质。圣美歌EA400.4S设计上就为了更好推动四声道的扬声器，特别越发流行的全主动分频玩法。为顾及玩法多样，特别针对超低音桥接输出设计，一台功放完全玩转2.0、2.1声道系统，对应市场、对应车厢环境、对应音质诉求，三方面设计得出平衡。

SOUNDMAGUS圣美歌EA400.4S四路功放参数 / 输出功率：100W×4@4Ω、120W×4@2Ω / 桥接功率：220W×2Ω / THD失真：0.08%@1kHz、4Ω / 频率响应：20Hz-20kHz（-3dB） / 信噪比：＞98dB / 1、2通道滤波范围：50Hz-500Hz（全通、高通） / 3、4通道滤波范围：0Hz-300Hz（全通、高通）、50Hz-500Hz（带通） / 阻尼系数：＞50@1kHz、4Ω / 负载阻抗：4声道@2Ω、2声道@4Ω / 最大音乐功率输出电流：33A@14.4V / 尺寸：188mm×56mm×320mm
