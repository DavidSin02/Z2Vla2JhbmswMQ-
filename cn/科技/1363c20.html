2020高压电工考试题库及高压电工模拟考试

2020-10-29
| 科技

题库来源：安全生产模拟考试一点通公众号小程序

2020高压电工考试题库及高压电工模拟考试，包含高压电工考试题库答案解析及高压电工模拟考试练习。由安全生产模拟考试一点通公众号结合国家高压电工考试最新大纲及高压电工考试真题出具，有助于高压电工模拟考试系统考前练习。

1、【判断题】 交流高压真空接触器的分、合闸动作是由真空开关管内的动触头完成的。（ √ ）

2、【判断题】 双绕组变压器的额定容量即为绕组的额定容量。（ √ ）

3、【判断题】 国家规定要求:从事电气作业的电工,必须接受国家规定的机构培训、经考核合格者方可持证上岗。（ √ ）

4、【判断题】 在电路中,电能的单位常用单位是kW？h,并且,1kW？h的电能俗称为1度电。（ √ ）

5、【判断题】 接触电压是指人站在带电外壳旁(水平方向0.8m处),人手触及带电外壳时,其手、脚之间承受的电位差。（ √ ）

6、【判断题】 线路金具是指连接和组合线路上各类装置,以传递机械、电气负荷以及起到某种防护作用的金属附件。（ √ ）

7、【判断题】 安装接线图是以屏面布置图为基础,以原理图为依据而绘制成的接线图,是一种指导屏柜上配线工作的图纸。（ √ ）

8、【判断题】 电流流过人体的路径,左手至脚对人体的伤害程度最大。（ √ ）

9、【判断题】 在断路器异常运行及处理中,值班人员发现设备有威胁电网安全运行且不停电难以消除的缺陷时,应在今后供电部门线路停电时及时配合处理。（ × ）

10、【判断题】 合理的规章制度是保障安全生产的有效措施,工矿企业等单位有条件的应该建立适合自已情况的安全生产规章制度。（ × ）

11、【判断题】 带负荷拉合隔离开关等,会产生强烈的电弧,将人严重灼伤。（ √ ）

12、【判断题】 为了保证电压质量合乎标准,往往需要装设必要的有功补偿装置和采取一定的调压措施。（ × ）

13、【判断题】 继电保护的可靠性是指发生了属于它该动作的故障,它能可靠动作;而在不该动作时,它能可靠不动。（ √ ）

14、【判断题】 在高压电气设备(包括线路)上工作,需要全部停电或部分停电的工作应使用第一种工作票。（ √ ）

15、【判断题】 避雷器与被保护设备并联连接。（ √ ）

16、【判断题】 架空线路杆塔的横担上下倾斜、左右偏歪不应大于横担长度的2%。（ √ ）

17、【判断题】 雷雨天气,需要巡视室外高压设备时,应穿绝缘靴,并不得靠近避雷器和避雷针。（ √ ）

18、【判断题】 一般地,电力系统的运行电压在正常情况下不允许超过最高工作电压。（ √ ）

19、【判断题】 用数字万用表测量直流电压时,极性接反会损坏数字万用表。（ × ）

20、【判断题】 过负荷、频率降低、单相断线均属于电气设备故障。（ × ）

21、【判断题】 绝缘靴(鞋)要放在柜子内,并应与其他工具分开放置。（ √ ）

22、【判断题】 对电力系统进行无功补偿可提高功率因素。（ √ ）

23、【判断题】 将电气设备由一种工作状态变换到另一种工作状态所进行的一系列操作称为倒闸操作。（ √ ）

24、【判断题】 隔离开关手力操动机构在操作时,操作质量不受操作人员技术水平的影响。（ × ）

25、【判断题】 操作中如发生疑问,可按正确的步骤进行操作,然后把操作票改正过来。（ × ）

26、【判断题】 油箱是油浸式变压器的外壳,变压器的器身置于油箱内,箱内灌满变压器油。（ √ ）

27、【单选题】 继电保护的()是指保护快速切除故障的性能。（ A ）

A、速动性

B、选择性

C、可靠性

28、【单选题】 手车式开关柜断路器手车在试验位置时摇不进的原因之一是()。（ C ）

A、接地开关在分闸位置

B、断路器在分闸位置

C、断路器在合闸位置

29、【单选题】 人体触电时,根据触电的体位不同,电流通过人体最危险途径是()。（ A ）

A、左手至双脚

B、右手至双脚

C、右手至左手

30、【单选题】 电对人体的伤害,主要来自()。（ A ）

A、电流

B、电压

C、电磁场

31、【单选题】 相对编号的常用格式是()。（ A ）

A、设备名称-接线端子号

B、设备编号-端子排号

C、设备编号-接线端子号

32、【单选题】 两个电阻值为R的等值电阻并联,其等效电阻(即总电阻)等于()Ω。（ C ）

A、1.5R

B、R

C、0.5R

33、【单选题】 在腐蚀性较强的场所引下线应适当()或采用其他防腐措施。（ C ）

A、减小截面

B、缩短长度

C、加大截面

34、【单选题】 控制电缆的编号“2UYH”表示该电缆归属于()。（ B ）

A、220kVII段电压互感器间隔

B、35kVII段电压互感器间隔

C、35kVII段母线间隔

35、【单选题】 通过电磁感应现象可以知道,线圈中磁通变化的频率越快,线圈的感应电动势()。（ C ）

A、越小

B、不变

C、越大

36、【单选题】 10kV变、配电所应在()上装设阀型避雷器。（ A ）

A、每组母线和每回路架空线路

B、每组母线和单回路架空线路

C、单组母线和每回路架空线路

37、【单选题】 当变压器二次绕组开路,一次绕组施加额定频率的额定电压时,一次绕组中所流过的电流称()。（ A ）

A、短路电流

B、整定电流

C、励磁电流

38、【单选题】 低压配电线路的导线宜采用()。（ C ）

A、三角、水平混合排列

B、垂直排列

C、水平排列

39、【单选题】 阀型避雷器阀电阻片具有()特性。（ C ）

A、线性或非线性

B、线性

C、非线性

40、【单选题】 在转移工作地点时,()应向工作人员交代带电范围、安全措施和注意事项。（ C ）

A、工作许可人

B、工作票签发人

C、工作负责人

41、【单选题】 架空导线型号LGJ-35/6表示的含义为()。（ B ）

A、钢芯铝绞线,电压等级为35kV,额定电流6k

B、钢芯铝绞线,铝线部分标称截面为35

C、铝芯钢绞线,电压等级为35kV,额定电流6k

42、【单选题】 使人体能够感觉到的最小电流称为()。（ B ）

A、持续电流

B、感知电流

C、致命电流

43、【单选题】 值班人员发现任何异常现象应及时消除,不能及时消除时,除及时报告上级领导外,还应记入运行记录簿和()。（ B ）

A、检修记录簿

B、缺陷记录薄

C、事故记录薄

44、【单选题】 隔离开关按安装地点分类可分为()和户外式。（ A ）

A、户内式

B、山地式

C、平原式

45、【单选题】 测量电路中电流时,电流表应()在该电路中。（ B ）

A、并联

B、串联

C、混联

46、【单选题】 当杆塔离道路较近,不能就地装设拉线时,一般采用()。（ B ）

A、张力拉线

B、水平拉线

C、普通拉线

47、【单选题】 变压器的一、二次电压一般允许有()的差值,超过则可能在两台变压器绕组中产生环流,影响出力,甚至可能烧坏变压器。（ C ）

B、±1%

C、±0.5%

B、屏面布置图

C、安装接线图
