R&S罗德与施瓦茨信号发生器自检自校准方法汇总-安泰维修中心

2020-11-14
| 科技

R&S罗德与施瓦茨信号发生器在信号质量、灵活性和可用性方面表现优异。因此定期就要对仪器进行自检自检准操作，安泰信号发生器维修中心分享罗德与施瓦茨信号发生器自检自校准方法：


信号发生器
一、SMW200A/SMBV100B/SMA100B/SMB100B
<1>自校准
System Config > Setup > General > Internal Adjustments>Adjust All
<2>固件检查、升级或修复
System Config > Setup > Instrument Assembly > Versions / Options

>Firmware>FW version显示firmware版本。

存储firmware的U盘插在信号源上开机启动，即可自动安装。

二、SMBV100A/SMA100A/SMB100A/SMF100A
<1>自校准
[Setup] >System > Internal Adjustments>Adjust All
<2>固件检查、升级或修复
[Setup]>Software / Options显示firmware版本。
存储firmware的U盘插在信号源上开机启动，即可自动安装。

以上有关信号发生器自检自校准操作方法由西安安泰信号发生器维修中心整理发布，更多有关仪器维修知识欢迎访问西安安泰仪器维修网，关注公众号：安泰测试
