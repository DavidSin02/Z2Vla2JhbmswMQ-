大揭秘！冠希连晒三次的 Jordan 新鞋！小编提前上脚

2020-11-14
| 科技

上个月初，冠希在个人 Instagram 突然晒出一双带有「000 23」字样的神秘新鞋。「全新联名」、「冷门鞋型」以及「疯狂暗示」等一系列流言席卷网络。

众说纷纭之中，随着整双鞋的全貌释出，谜题也正式揭晓：来自 Jordan Brand 打造的全新鞋型 Jordan Delta React。

作为一双全新 Lifestyle 系列鞋款，能够受到冠希如此青睐，甚至得到：“我认为它用设计证明了自己不凡的身份。” 这样的超高评价。

今天，小编就将为你揭开这双鞋的神秘面纱！究竟实物怎么样？上脚怎么样？想必看完你就能够找到答案！

▼ 小编上脚实拍

01 挑战全新造型

全新 Jordan Delta React 设计初衷在于打造一双适合全天候舒适穿着的鞋款。

设计围绕 “飞翔” 的理念进行创作，低帮造型设计能够充分保障脚踝处的灵活需求。

鞋面借鉴航天科技和设计，采用各种高端材料结合打造。多种材质的拼接效果，呈现出丰富层次感。

搭载全掌 Nike React 泡棉，为双脚带来极佳的舒适感觉。

外底采用了篮球鞋经常使用的人字纹图案，在致敬篮球运动的同时，为穿著者提供充分的防滑效果。

02 做工精细，细节出众

作为一双极具科幻感的全新鞋型，Jordan Delta React 在细节方面的刻画同样值得一看。首先便是整双鞋的用料，鞋身大面积采用致密的织物材质打造鞋面。

鞋头外侧部分采用被透明橡胶包裹，内里采用类似牛皮纸一般的材质，呈现出复古做旧质感。

侧边点缀一小块 3M 反光饰条，「000 23」字样也与冠希当时晒出的实物保持一致。

▼ 「000 23」字样

鞋身中部应用航天材质，不规则的褶皱很容易令人想到太空服手套的造型，这一点也与 Nike Mars Yard 3.0 相类似。

同时粗犷的线条缝线将不同材质拼接组合，淡蓝色防撕裂鞋面也体现出十足科技感。

▼ 鞋身材质丰富

厚实的海绵鞋舌上布满孔洞设计，不仅复古质感出众，同时也为透气效果起到一定作用。顶端部分采用外露式设计，颇有几分解构风格。

▼ 海绵鞋舌填充厚实

手绘风格的 Jumpman Logo 以及「000 23」字样点缀鞋舌标签，彰显整双鞋的主题。

▼ 鞋舌蓝色 Jumpman Logo

后跟位置同样采用透明橡胶材质包裹，圆形孔洞设计呼应鞋舌，蓝色线条勾勒出经典飞翼 Logo 的轮廓。

▼ 后跟蓝色线条飞翼 Logo

中底内侧带有「react」字样，表明整双鞋搭载 React 缓震科技，充分保障脚感。

▼ 中底 React 标签

外底纹路格外清晰，后跟位置一个大号 Jumpman Logo 亮明品牌身份。

▼ 外底 Jumpman Logo

最惊喜的是，Jordan Brand 为全新 Delta React 配备了软木塞鞋垫，这一点怕是要打动不少人的心！

▼ 软木塞鞋垫

03 脚感软弹，造型百搭

整双鞋上脚的第一感受便是来自 React 缓震科技的软弹感觉，不会有过度塌陷感，同时也能够给双脚提供不错的反馈。

外底浅黄色橡胶环绕 React 缓震中底 ，同时在侧边部分还有向上延伸的设计，为整双鞋提供支撑。

清晰的外底纹路为整双鞋带来充足的抓地效果，抬脚间露出的 Jumpman Logo，彰显 Jordan 品牌身份。

海绵鞋舌填充厚实，同时圆形孔洞设计的存在也充分起到透气效果。

鞋后跟故意做歪的拉环符合人体工学设计，更加体现了性能优先的设计考量。

04 本月即将发售

小编上脚亲测整双鞋尺码不偏，按个人正常尺码入手即可。

目前，Jordan Brand 官方已经确定，小编上脚的这双 Jordan Delta React SP 将于 4 月 20 日正式在国内发售。

随后，5 月 1 日还将推出火星鞋配色以及黑白配色等更多抢眼配色！

想要入手的朋友届时一定不要错过！
