2016年造价工程师《工程计价》真题与答案

一、单项选择题(共72题，每题I分，每题的备选选顶中，只有I个是符合题意)

1.【题干】关于我国建设项目投资，下列说法中正确的是(　)

【选项】

A.非生产性建设项目总投资由固定资产投资和铺底流动资金组成

B.生产性建设项目总投资由工程费用、工程建设其他费用和预备费三部分组成

C.建设投资是为了完成工程项目建设，在建设期内投入且形成现金流出的全部费用

D.建设投资由固定资产投资和建设期利息组成

【答案】C

【解析】A错误，不包含流动资金;B错误，应该包含流动资金;D错误，固定资产投资包含建设投资和建设期利息。

【考点】我国建设项目投资及工程造价的构成。

2.【题干】已知生产某非标准设备所需材料费、加工费、辅助材料费、专用工具费合计为30万元，废品损失率为10%，外购配套件费为5万，包装费率为2%，利润率为10%，用成本估算法计算该设备的利润值为(　)元。

【选项】

A.3.366

B.3.370

C.3.376

D.3.876

【答案】C

【解析】{[30×(1+10%)+5]×(1+2%)-5｝×10%=3.376

【考点】国产非标准设备原价的计算。

3.【题干】进□设备的原价是指进口设备的(　)

【选项】

A.到岸价 B.抵岸价 C.离岸价 D.运费在内价 【答案】B

【解析】进口设备的原价就指进口设备的抵岸价。

【考点】

4.【题干】某县城建筑公司承建市区某建筑工程，2015年8月编制估算的税前造价为，已知6000万元，已知营业税率为3%，市、县城市维护建设税税率分别为7%、5%，教育附加率和地方费附加率分别为3%、2%。该工程的结算造价为(　)万元

【选项】

A.6198.0 B.6201.2 C.6204.8 D.6208.6

【答案】D

【解析】6000/(1-3%-3%×7%-3%×3%-3%×2%)=6208.6

【考点】税金的计算.

5.【题干】根据我国现行建筑安装工程费用项目构成的规定，下列费用中属于安全文明施工费的(　)

【选项】

A.夜间施工时，临时可移动照明灯具的设置、拆除费用

B.工人安全防护用品的购置费用

C.地下室施工时采用的照明设施拆除费

D.建筑物的临时保护设施费

【答案】B

【解析】A属于夜间施工增加费;C属于非夜间施工照明费;D选项属于建筑物的临时保护设施费。

【考点】措施费的构成。

6.【题干】关于国外建筑安装工程费用的计算，下列说法正确的是(　)

【选项】

A.分包工程费不包括分包工程的管理费和利润　B.材料的预涨费计入管理费

C.开办费一般包括了工地清理费及完工后清理费 D.现场管理费属于开办费

【答案】C

【解析】A错误，分包工程费包括：分包报价、总包管理费和利润。材料预涨费属于材料费的一部分，B错误。

【考点】

7.【题干】下列与建设用地有关的费用中，归农村集体经济组织所有的是(　)

【选项】

A.土地补偿费 B.青苗补偿费 C.拆迁补偿费　D.新菜地开发建设基金

【答案】A

【解析】B选项青苗补偿费归青苗所有者;C拆迁补偿费归被拆迁人;D选项新菜地开发建设基金归政府。

【考点】建设用地费的构成。

8.【题干】关于工程建设其他费用，下列说法中正确的是(　)

【选项】

A.建设单位管理费一般按建筑安装工程费乘以相应费率计算

B.研究实验费包括新产品试制费

C.改扩建项目的场地准备及临时设施费一般只计拆除清理费

D.不拥有产权的专用通信设施投资，列入市政公用设施费

【答案】C

【解析】A选项建设单位管理费的计算基础为工程费用;B选项研究实验费不包括新产品试制费;D选项不拥有产权的专用通信设施投资列入专利及专有技术使用费。

【考点】工程建设其他费用。

9.【题干】某建设项目静态投资为10000万元，项目建设前期年限为1年，建设期为2年，第一年完成投资40%，第二年完成投资60%。在年平均价格上涨率为6%的情况下，该项目价差预备费应为(　)万元。

【选项】

A.666.3 B.981.6 C.1306.2　D.1640.5

【答案】C

【解析】10000×40%×[(1+6%)1+1-0.5-1]+10000×60%×[(1+6%)1+2-0.5-1]=1306.2

【考点】价差预备费的计算。

10.【题干】某项目建设期为2年，第一年贷款3000万元，第二年贷款2000万元，贷款年均衡发放，年利率为8%，建设期内只计息不付息。则建设期利息为(　)

【选项】

A.366.4　B.449.6 C.572.8　D.659.2

【答案】B

【解析】第一年利息为3000/2×8%=120万;第二年利息为(3000+120+2000/2)×8%=329.6 两年合计利息为449.6万

【考点】建设期利息计算。

11.【题干】下列说法中，符合工程计价基本原理的是(　)

【选项】

A.工程计价的基本原理在于项目划分与工程量计算

B.工程计价分为项目的分解与组合两个阶段

C.工程计价包括工程单价的确定和总价的计算

D.工程单价包括生产要素单价、工料单价和综合单价

【答案】C

【解析】A错误，工程计价的基本原理在于项目的分解与组成;B错误，工程计价分为工程计量和工程计价两个环节;D错误，工程单价包括工料单价和综合单价。

【考点】工程计价的基本原理。

12.【题干】关于工程定额，下列说法正确的是(　)

【选项】

A.劳动定额主要表现形式为时间定额，机械消耗定额主要表现形式为产量定额

B.企业定额是企业内部使用的预算定额

C.补充定额只能在指定范围内使用

D.地区统一定额应依据地区工程技术特点、施工生产和管理水平独立编制

【答案】C

【解析】A错误，劳动定额主要表现形式为时间定额，机械消耗定额主要表现形式为时间定额;B错误，企业定额是企业内部使用的施工定额;D错误，地区统一定额主要是考虑地区性特点和全国统一定额水平做适当调整和补充编制的。

【考点】工程定额体系。

13.【题干】根据《建设工程工程量清单计价规范》(GGB505-2013)下列关于工程量清单项目编码的说法中，正确的是(　)

【选项】

A.第三级编码为分部工程顺序码，由三位数字表示

B.第五级编码应根据拟建工程的工程量清单项目名称设置，不得重码

C.同一标段含有多个单位工程，不同单位工程中项目特征相同的工程应采用相同编码

D.补充项目编码以"B"加上计量规范代码后跟三位数字表示

【答案】B

【解析】A错误，第三级编码由二位数字表示;C错误，特征相同的工程不得有重码;D错误，补充项目编码以上计量规范代码加"B"和三位数字表示。

【考点】分部分项工程量清单。

14.【题干】对于不能计算工程量的措施项目，当按施工方案计算措施费时，若无"计算基础"和"费率"数值，则(　)

【选项】

A.以定额计价为计算基础，以国家、行业、地区定额中相应的费率计算金额

B.以"定额人工费+定额机械费"为计算基础，以国家、行业、地区定额中相办计算金额相应费率计算金额

C.只填写"金额"数值，在备注中说明施工方案出处或计算方法

D.备注中说明中计算方法，补充填写"计算基础"和"费率"

【答案】C

【解析】对于不能计算工程量的措施项目，当按施工方案计算措施费时，若无"计算基础"和"费率"数值，则只填写"金额"数值，但应在备注中说明施工方案出处或计算方法。

【考点】措施项目清单的编制。

15.【题干】采用工程量清单计价的总承包服务费计价表中，应甶投标人填写的内容是(　)

【选项】

A.项目价值

B.服务内容

C.计算基础

D.费率金额

【答案】D

【解析】投标时，总承包服务费的费率及金额由投标人自主报价，计入投标总价中。

【考点】总承包服务费。

16.【题干】下列机械工作时间中，属于有效工作时间的是(　)

【选项】

A.筑路机在工作区末端的调头时间

B.体积达标而未达到载重吨位的货物汽车运输时间

c.机械在工作地点之前的转移时间

D.装车数量不足而在低负荷下工作的时间

【答案】B

【解析】B选项为有根据的降低负荷下的工作时间。

【考点】工作时间分类。
