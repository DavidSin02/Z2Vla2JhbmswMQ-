一闪一闪亮晶晶的，除了星星还有它们

2020-11-14
| 科技

阳光明媚的春夏季节，喜欢各种亮晶晶的玻璃食器。

晶莹剔透的模样足以让人赏心悦目，扑面而来的是清爽的春夏气息。微风习习的清晨，或是烈日当空的午后，一杯自己调配的果汁、饮品，一小碗雪糕、甜点，慢悠悠品尝，应该是许多人关于夏天的记忆。


亮晶晶的玻璃食器在日本称为「硝子」，和普通玻璃不同的是，硝子是用日本独有的玻璃工艺制作出来的，它不仅摸起来更加温润细腻，而且不全是透明无色，有的还色彩斑斓，形态万千，如同工艺品一般。

硝子在日本家庭中扮演着重要的角色，从早到晚，一日三餐，甚至是日常的生活用品都能见到它的身影。


而诞生于 1899 年的广田硝子，是东京历史最古老、也是最具代表性的玻璃制造商之一，制作出许多美观且有质感的玻璃器皿。


:: 坚持手工制作的硝子品牌 ::


广田硝子成立于明治维新时期，当时日本在东京品川创立了官营的「品川硝子制造所」，自此东京下町的玻璃制造厂商纷纷起家，将西方传来的玻璃制造技术与日本美学意识相融合，制作日本独特的玻璃设计，其中就包括广田硝子。

创始人是当时年仅26岁的广田金太先生，在东京市田町建立了一家专门批发玻璃制品的小商铺，主要贩卖玻璃食器，容器和玻璃灯罩等，打下了坚实基础后于1915年在墨田区开了一家属于自己玻璃工厂「江东硝子工厂」，也就是后来的广田硝子。


到了第二代继承人荣次郎，他吸取了大正与昭和时期的新技术，创造出与其他公司完全不同的产品，使得广田硝子开辟出属于自己的道路。

尽管经过动荡的战乱时期，但广田硝子依旧小心翼翼地延续下来，成为了一个见证过东京的动乱与繁荣，也见证过不同时代风貌的古老品牌。不管时代如何变更，广田硝子120多年来始终贯彻了代代相承的「追求传达人之感性，器之温度」的理念，坚持手工制作玻璃制品，这些产品随着广田硝子的基调延续下来。



广田硝子相信亲手制作的器物会带来机器生产所不能共鸣的温度和情感，也期望以传统技术为支持，创造出新产品。


:: 独有的江户切子 ::


在《日本人家必备的杯子，竟然是「吹」出来的》一文中，我们曾介绍过硝子的制作工艺：每一只硝子诞生的背后，都经历了烧、熔、吹、切、磨的工艺历程，饱含了手工艺人繁复的心思和高超的水准。

但正如广田硝子现任社长广田达朗先生所说的「每家公司的生存之道，就在于它与众不同的地方」，广田硝子最引以为傲的，是其独有的玻璃制作技能「江户切子」，这是日本现存的唯一从江户时代流传下来玻璃切割技术。


江户切子原本是指在江户时代流行的一种手工玻璃，因工艺精美华丽而闻名，后演变为一种制作工艺的称呼。


在固态的硝子冷却之后才进行加工，由匠人全程手工在表面刻磨出墨线，宽深的纹路被砂轮雕刻完毕后再用细轮勾勒出纤细，修整后抛光打磨出精美的花样和纹理。整个过程没有草稿，一气呵成，不允有误。



而江户切子最大的特征就是彩色的表面玻璃非常轻薄，如果要成功地裁切，动作必须非常谨慎且细心，若稍有闪失，内层透明玻璃就会出现在表面上。

这是对手艺人是极高考验，据广田硝子的匠人说，要能成为独当一面的专家，负责所有的工程至少需要10年的时间，这也是为什么硝子在日本到处可见，但真正做出艺术的却屈指可数。


独特的技术造就了江户切子与众不同的美丽，不用上各种各样的颜色，仅仅通过切割的反射就能让玻璃绽放出如同钻石的光芒，展现了日式宁静、质朴通透的美学精神。



:: 各不相同的系列产品 ::


虽说最广为人知的是江户切子技艺，但实际上，广田硝子的每个不同系列产品都有自己的独特的工艺技巧，也展现出了不同的样貌，各具特色。


// 昭和时代系列


广田硝子目前一条重要的产品线是对品牌之前非常经典且优秀，却由于各种各样的问题而不得不停产的产品进行「复刻」，其中就包括昭和时代系列。

昭和时代系列是以昭和时期在吃茶店所使用的非常盛行的咖啡杯为灵感进行复刻的。

©️禾希有物


这款昭和时代的复古咖啡杯采用当时的手压技术，由手工吹制而成，用耐热玻璃制作，即便是热饮也能放心使用。

它兼具了美观和实用，复古的琥珀色低调而时髦，玻璃晶莹剔透，底部还有精细的雕刻，质感十足，捧在手上时触感温润。


©️禾希有物


无论是当作咖啡杯、酒杯或水杯都可以，优秀的品质也让它成为送礼的佳选。


// 雪花系列


相比之下，雪花系列是看起来更加新潮和浪漫。采用昭和时期留下的模具制作，有杯子和甜品碗的组合，边缘的乳白色是由人工烘烤而成，与透明的食器互相映衬，仿佛是片片雪花飘落而至，添加了一份柔和。

©️禾希有物


杯子线条分明，宛如一朵郁金香般修长立体，透明的杯身就如夏天一般清凉怡人，最适宜承装夏天的饮品、果汁；甜品碗小巧迷你，与小勺搭配使用，用来吃雪糕最好不过，也可以用来装小点心或配菜。

©️禾希有物


©️禾希有物


// 雪月花系列


这个系列最引人注意的是器皿上浮现的一个个小圆点，当阳光照射进来，有种光彩四溢的迷离感。这是以日本经典的「霰纹」作为主题创作的，透过将高温的玻璃流入模具中，并加压制成形，也是广田硝子独特的技法之一，相比吹制玻璃更加坚固，花纹也清晰可见。


©️禾希有物


杯子和碗碟可搭配成套，会是你餐桌上两眼的一员，即使是喝水也很好看，装和式、洋式的小点心都非常适合。

©️禾希有物


// 酒器套组系列


酒器套组是广田硝子一直制作的经典畅销款，倒上清酒或者自己酿泡的果酒，与家人、好友小酌一杯。

「浅樱」以柔软而微微发光的樱花为主题，酒壶酒杯呈淡粉色，下半部分有如雾霾妆的渲染，握在手中是磨砂质感，杯底还有花蕾的印刻，仿佛手中拿着一朵樱花，置身花丛中一般。

©️禾希有物


瓶身透明可见酒色，浪漫的粉红色装饰春天空间，除了自用也适宜送礼，有着爱与希望的寓意。

©️禾希有物


「吹雪」则是以冬日飞雪作为主题，瓶身杯身均经过磨砂处理，瓶子是不规则的凹凸设计，造型别致外更是便于手握，杯口设计贴合嘴形。整体素雅简洁却不失优雅，在家也能感受日式料理店的氛围。

©️禾希有物

©️禾希有物


// 酱油瓶系列

酱油瓶系列拥有「缓柔细出、急停立止、一滴不漏」三大令广田硝子引以为傲的特质。

©️禾希有物


千筋透明款酱油瓶是椭圆的瓶肚，持握舒适，由层层线条设计组成。倒酱油时，扭紧瓶盖，细缓流出，不多不少，也不会把酱油流出瓶身。除了酱油外，也可以放入醋、白酒或其他调味品。

©️禾希有物


广田硝子的每一款玻璃器皿总能给人带来第一眼的惊喜，光是放在桌上看着就觉得心情愉悦。陷于美貌，忠于质感，只有拿上手方能感受到质感的不同寻常，细心爱护，便能够长久相伴。
