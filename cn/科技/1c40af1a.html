如何在 Mac 上安全彻底的卸载软件？Mac 清理软件整理

2020-10-29
| 科技

在 Mac 上删除软件的方法是选中需要删除的软件，右键选择移到废纸篓就完成了删除。但是这种删除方式并不能完全卸载软件，电脑上可能会残留很多文件和日志碎片，久而久之电脑的存储空间会变少，运行速度也会变慢。

整理了几款 Mac 下好用的卸载软件，使用下来体验都非常不错，推荐给大家。

AppCleaner

这是 Mac 上超级轻量级的免费卸载软件，没有多余附加的功能，只专注于卸载软件及其依赖文件。使用方式非常简单，只需要将软件拖入到 AppCleaner 的窗口中即可。

如果你只需要一款单一的卸载软件功能，AppCleaner 是首选。

CleanMyMac X

CleanMyMac X 是 Apple 官方认证的恶意软件防御应用，只会删除能够 100% 确定可以安全删除的文件。所以在安全性上能够保障你的文件，不会造成误删操作。CleanMyMac X 简单易用，只需要选择相应软件，点击卸载即可。

CleanMyMac X 是付费的，一次性购买的价格为 223 元，也不是非常贵，这是我目前在使用的清理软件，可以非常方便的清理软件和内存。

MacBooster

MacBooster 相比于 CleanMyMac X 少了粉碎文件功能，如果平时没有粉碎文件的需求，那么 MacBooster 也是不错的选择。

MacBooster 界面的风格和 CleanMyMac X 差不多，都采用了左右两侧功能区和展示区分开的风格。

垃圾清理、内存清理、隐私保护、应用卸载都不在话下。

柠檬清理

柠檬清理是腾讯出品的一款免费的 Mac 清理软件，是新晋 Mac 清理软件，相比于 MacBooster 和 CleanMyMac X，柠檬清理完全免费，并且没有捆绑其他软件销售。如果你不想付费，柠檬清理是非常不错的选择。

柠檬清理在清理能力上并不逊色于 CleanMyMac X 和 MacBooster，柠檬清理具备摄像头和麦克风保护功能，能够很好地保护隐私。在系统维护、恶意软件扫描上，柠檬清理相比于 CleanMyMac X 还比较缺失。

如果你想要一款免费并且简约的清理软件，柠檬清理是首选。

ccleaner-mac

ccleaner 在 Windows 系统比较出名，下载量非常高，之前我在使用 Windows 系统时，使用的就是这款清理软件。每当 C 盘变红色时，我就会调出我的 ccleaner，一番操作后，我的 C 盘就变为了蓝色，世界立马安静了。

ccleaner-mac 是后来专门为 Mac 推出的清理软件，如果你之前是 Windows 下 ccleaner 的重度用户，想要直接上手 ccleaner-mac 就比较适合。
