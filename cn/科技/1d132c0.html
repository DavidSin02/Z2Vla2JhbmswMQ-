命题点15 化学式的意义及其相关计算、化合价

(时间：30分钟　分值：28分)

一、选择题

1. (2016沈阳)P2O5的化学名称是( )

A. 二磷化五氧 B. 五氧化二磷

C. 氧五化磷二 D. 磷二化氧五

2. (2016苏州)下列物质化学式书写正确的是( )

A. 铝：AL B. 氯化钾： KCl2

C. 硫酸铜：CuSO4 D. 四氧化三铁：O4Fe3

3. (2016合肥38中一模)NaNO2有剧毒，外貌酷似食盐且有咸味，NaNO2中N元素的化合价为( )

A. －3 B. ＋1 C. ＋2 D. ＋3

4. (2016阜阳市颍泉区二模)铜是生活、生产中重要的金属，常见的黄铜矿中，主要成分是CuFeS2，已知铜元素在该物质中的化合价为＋2价，则该物质中铁元素的化合价为( )

A. ＋2 B. 0 C. ＋3 D. ＋4

5. 钛铁矿的主要成分的化学式为FeTiOx，其中铁元素和钛元素的化合价均为＋3。则x为( )

A. 2 B. 3 C. 4 D. 6

6. (2016泉州)下列物质中，氯元素的化合价最高的是( )

A. ClO2 B. HClO

C. NaCl D. Cl2

7. (2016合肥45中一模)生日蛋糕中的奶油含有对人体有害的反式脂肪酸，其化学式为C18H34O2。下列说法正确的是( )

A. 反式脂肪酸是氧化物

B. 反式脂肪酸由碳、氢、氧三种元素组成

C. 反式脂肪酸由54个原子构成

D. 反式脂肪酸中碳、氧元素的质量比是9∶1

8. 风油精是家庭常备药，常用于蚊虫叮咬、晕车等引起的不适。它的有效成分之一是丁香酚(C10H12O2)，下列有关丁香酚的说法正确的是( )

A. 丁香酚的相对分子质量是164 g

B. 一定条件下，丁香酚可以燃烧生成CO2和H2O

C. 丁香酚中C、H、O元素质量比为10∶12∶2

D. 丁香酚的一个分子中含1个氧分子

9. (2016阜阳二模)安徽黄山盛产茶叶，茶叶中含有茶氨酸(C7H13O3N2)、锌、硒等多种成分。茶树适宜在pH为5～6的土壤中生长。下列说法不正确的是( )

A. 茶树不宜在碱性土壤中生长

B. 茶氨酸由碳、氢、氧、氮四种元素组成

C. 茶叶中所含茶氨酸属于有机物，锌、硒属於单质

D. 一个茶氨酸分子中含有25个原子

10. (2016南昌)南昌汉代海昏侯墓出土的文物中，保存完好的玉器非常罕见。对玉石主要成分(化学式为NaAlSi2O6)的描述正确的是( )

A. 含有3个氧分子

B. 相对分子质量为94

C. 钠元素的质量分数为13.4%

D. 钠元素与铝元素的质量比为23∶27

11. (2016黔南州)高铁酸钾(K2FeO4)是一种集吸附、凝聚、杀菌等功能为一体的新型高效水处理剂，可将水中的三氯乙烯(C2HCl3)除去85.6%，下列说法正确的是( )

A. K2FeO4是氧化物

B. C2HCl3中三种元素的质量比是2∶ 1∶3

C. K2FeO4中Fe元素的化合价为＋3价

D．三氯乙烯中碳元素的质量分数约为18.25%

12. (2016孝感)我国科学家用滤纸和二氧化钛(TiO2)薄膜制作出一种新型"纳米纸"，又在纳米纸上"铺"一层"萘胺"(C10H9N)染料，制成一种试纸，用于检测食品中亚硝酸盐浓度的高低。下列说法正确的是( )

A. 二氧化钛中Ti的化合价为＋2价

B. 萘胺中氮元素的质量分数约为11.9%

C. 这种试纸是一种新型有机化合物

D. 萘胺中碳、氢、氮三种元素的质量比为120∶9∶14

二、填空和简答题

14. (2016邵阳)按要求用化学用语填空：

(1)两个硫原子____________。

(2)铝离子____________。

(3)－1价的氯元素________。

(4)碘酒中的溶剂________。

1. B　【解析】由两种元素组成的化合物的名称，一般读作某化某，有时还要读出化学式中各种元素的原子个数，因此，P2O5读作五氧化二磷。故选B。

2. C　【解析】铝是金属单质，用铝的元素符号表示化学式，即Al，A错误；氯化钾中钾元素为＋1价，氯元素为－1价，化学式应为KCl，B错误；根据化合物中各元素正负化合价代数和为零，判断硫酸铜化学式正确，C正确；四氧化三铁中铁是金属元素，写在前面，氧为非金属元素，写在后面，D错误。

3. D　【解析】亚硝酸钠中钠元素显＋1价，氧元素显－2价，设氮元素的化合价为x，根据化合物中正负化合价的代数和为零，有1＋x＋(－2)×2＝0，解得x＝＋3。

4. A　【解析】CuFeS2中铜元素显＋2价，硫元素显－2价，设铁元素的化合价为x，根据化合物中正负化合价的代数和为零，有2＋x＋(－2)×2＝0，解得x＝＋2。

5. B　【解析】根据化合物中正负化合价的代数和为零，有3＋3－2x＝0，解得x＝3。

6. A　【解析】根据化合物中各元素正负化合价的代数和为0的原则，ClO2中O显－2价，Cl显＋4价；HClO中H显＋1价，O是－2价，Cl显＋1价；NaCl中Na显＋1价，Cl显－1价；Cl2中Cl显0价；因此，A中氯元素的化合价最高，故选A。

7. B　【解析】氧化物是由两种元素组成，且其中一种元素是氧元素的化合物，反式脂肪酸由碳、氢、氧三种元素组成，A错误，B正确；反式脂肪酸由分子构成，一个反式脂肪酸分子由54个原子构成，C错误；反式脂肪酸分子中碳原子与氧原子的个数比为9∶1，由于不同原子的质量不同，所以元素的质量比不等于原子的个数比，D错误。

8. B　【解析】相对分子质量的单位是"1"，常省略不写，A错误；根据质量守恒定律和物质燃烧的规律可知，丁香酚充分燃烧生成水和二氧化碳，B正确；丁香酚分子中碳、氢、氧元素的原子个数比为10∶12∶2，由于不同原子的质量不同，所以元素的质量比不等于元素原子个数之比，C错误；丁香酚能用化学式表示，属于纯净物，所以丁香酚中不可能含有氧分子，D错误。

9. C　【解析】pH小于7显酸性，即茶树适合在偏酸性土壤中生长，A正确；根据茶氨酸的化学式可知，茶氨酸由碳、氢、氧、氮四种元素组成，B正确；根据茶氨酸的组成可知，茶氨酸属于有机物，但锌和硒指的是元素，不是单质，C错误；一个茶氨酸分子中含7＋13＋3＋2＝25个原子，D正确。

11. D　【解析】氧化物是由两种元素组成且其中一种元素是氧元素的化合物，K2FeO4由三种元素组成，不是氧化物，A错误；C2HCl3中三种元素的质量比为(12×2)∶1∶(35.5×3)＝24∶1∶106.5，B错误；K2FeO4中K显＋1价，O显－2价，设Fe的化合价为x，根据化合物中各元素的化合价代数和为0，可得(＋1)×2＋x＋(－2)×4＝0，x＝＋6价，C错误；三氯乙烯中碳元素的质量分数为eq \f(12×2,12×2＋1＋35.5×3)×100%≈18.25%，D正确。

12. D　【解析】氧元素显－2价，设钛元素的化合价是x，根据在化合物中元素正负化合价代数和为零，可得x＋(－2)×2＝0，则x＝＋4价，A错误；该"萘胺"中氮的质量分数为eq \f(14,12×10＋1×9＋14)×100%≈9.8%，B错误；二氧化钛不含碳元素，不属于有机物；"萘胺"(C10H9N)是含有碳元素的化合物，属于有机物，这种试纸是由二者组成的混合物，C错误；萘胺中碳元素、氢元素和氮元素的质量比为(12×10)∶(1×9)∶(14×1)＝120∶9∶14，D正确。

13. C　【解析】根据乙酸分子的模型可以写出乙酸的化学式，化学式为C2H4O2；由化学式可知，乙酸是由碳、氢、氧三种元素组成的化合物，乙酸的相对分子质量＝12×2＋1×4＋16×2＝60；乙酸中碳元素的质量分数＝eq \f(12×2,60)×100%＝40%，C错误；乙酸分子中碳、氢、氧的原子个数比为2∶4∶2＝1∶2∶1。

14. (1)2S　(2)Al3＋　(3)Ceq \o(l,\s\up6(－1))　(4)C2H5OH

【解析】(1)硫原子用符号"S"表示，2个硫原子在前面加上数字"2"。(2)每个铝离子带3个单位的正电荷。(3)氯的元素符号为"Cl"，再在元素符号正上方标上"－1"。(4)碘酒中的溶液是酒精，化学式为C2H5OH。
