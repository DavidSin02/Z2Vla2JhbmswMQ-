注册消防工程师-〖技术实务〗气体灭火系统模拟题

2020-10-29
| 科技

1、【单选题】二氧化碳气体灭火系统不适用于扑救（ ）

A 电气火灾

B 固体表面火灾

C 金属氢化物火灾

D 灭火能量切断气源的气体火灾

2、【单选题】某自备发电机房采用七氟丙烷灭火系统进行保护，系统的设计浓度为9.1%，则该系统的灭火浓度可以为（ ）

A 7%

B 9.1%

C 10.01%

D 11.83%

3、【单选题】某博物馆文物的展览部分，设计采用气体灭火装置进行保护。下列关于该区域的喷头设置，说法错误的是（ ）

A 展厅喷头的安装高度为6m

B 喷头的安装高度距顶板0.4m

C 喷头在防护区内均匀布置

D 展柜喷头的安装高度为1.3m，保护半径为5m

4、【单选题】某电子计算机房设置了七氟丙烷灭火系统，七氟丙烷灭火剂喷放时间不应大于（ ）S；灭火浸渍时间应采用（ ）min

A 5， 5

B 6， 10

C 8， 5

D 10，10

5、【单选题】关于气体灭火系统管道应满足的要求，下列说法不正确的是（ ）

A 启动气体的管道宜采用铜管

B 输送气体灭火剂的管道应采用无缝钢管，无缝钢管应进行内外防腐处理

C 管道的公称直径为80mm时，宜采用法兰连接

D 管道的公称工作压力，不应小于在最高环境温度下所承受的工作压力

6、【单选题】某大型城市综合体中的变配电间，计算机主机房、通讯设备间等场所内设置了组合分配式七氟丙烷气体灭火系统。下列关于该系统组件的说法中，错误的是（ ）

A 集流管应设置安全泄压装置

B 选择阀的公称直径应和与其对应的防护区灭火系统的主管道的公称直径相同

C 输送启动气体的管道应采用铜管

D 选择阀应在容器阀动作之前或同时打开

7、【多选题】某住宅小区的变配电间，建筑面积为300㎡，净空高度为3m，划分为一个防护区并采用无管网七氟丙烷气体灭火装置进行防护。委托消防检查机构对该气体灭火系统进行检测，下列检测结果中，符合现行国家消防技术标准要求的有（ ）

A 该防护区的装置动作响应时间差为1s

B 防护区内设置8台预制灭火装置

C 防护区门口未设手动与自动控制的转换装置

D 储存容器的充装压力为4.2MPa

E 该系统未设置备用量

8、【多选题】某市规划建设大型数据中心，共4层，层高5m，每层建筑面积2200㎡。由于该建筑内平常有人值班，该信息机房内设计采用IG541组合分配式气体灭火系统。下列有关该机房气体灭火的设计不符合规范要求的是（ ）

A 该机房内灭火系统的启动方式有自动启动和手动启动两种启动方式

B 信息机房内设有可开启式甲级防火窗，但未设置机械排风装置

C 该建筑内共划分12个防护区

D 该系统不可以设置为无延迟的喷射方式

E 系统储存容器采用焊接容器或无缝容器

==============

答案

1 C

【解析】气体灭火系统的适用范围

适用于扑救：

（1）电气火灾

（2）液体表面或可融化固体火灾

（3）固体表面火灾

（4）灭火前可切断气源的气体火灾

不适用于扑救：

（1）含氧化剂的化学制品

（2）活泼金属

（3）金属氢化物

（4）能自行分解的化学物质

2 A

【解析】根据《气体灭火系统设计规范》3.3.1七氟丙烷灭火系统的灭火设计浓度不应小于灭火浓度的1.3倍，惰化设计浓度不应小于惰化浓度的1.1倍。9.1%/1.3=7%，故选A。

3 D

【解析】喷头的安装高度和保护半径

（1）最大保护高度不宜大于6.5m

（2）最小保护高度不应小于0.3m

（3）喷头安装高度小于1.5m时，保护半径不宜大于4.5

（4）喷头安装高度不小于1.5m时，保护半径不应大于7.5m

（5）喷头宜贴近防护区顶面安装，距顶面的最大距离不宜大于0.5m

4 C

【解析】教材P240，七氟丙烷灭火系统在通信机房和电子计算机房等防护区，设计喷放时间不应大于8s，在其他防护区，设计喷放时间不应大于10s；灭火浸渍时间的规定：木材、纸张、织物宜采用20min，通信机房、电子计算机房应采用5min，其他固体表面火灾宜为10min，气体和液体火灾不应小于1min。

5 C

【解析】教材P246，公称直径小于等于80mm的管道，宜采用螺纹连接，公称直径大于80mm的管道宜采用法兰连接。

6 C

【解析】

（1）为了防止储存器发生误喷射，在集流管末端设置安全泄压装置

（2）选择阀的公称直径应和与其对应的防护区灭火系统的主管道的公称直径相同

（3）输送启动气体的管道宜采用铜管

（4）输送气体灭火剂的管道可采用无缝钢管、不锈钢管

（5）选择阀应在容器阀动作之前或同时打开

7 BE

【解析】《气体灭火系统设计规范》GB50370-2005 3.1.14、3.1.15

详细【解析】根据题意可知，该系统为预制式气体灭火系统。根据3.1.14、3.1.15“一个防护区设置的预制灭火系统，其装置数量不宜超过10台，同一防护区内必须能同时启动，动作响应时间差不得大于2s”A项错误，B项正确；预制系统储存容器的充装压力不应大于2.5mpa，D项错误；预制系统不需要设置备用量，E项正确。

8 ACE

【解析】根据《气体灭火系统设计规范》5.0.2管网灭火系统应设自动控制、手动控制和机械应急操作三种启动方式，故A项不符合规范要求；6.0.4灭火后的防护区应通风换气，地下防护区和无窗或设固定窗扇的地上防护区，应设置机械排风装置，排风口宜设在防护区的下部并直通室外，故B符合要求；根据3.2.4采用管网灭火系统时，一个防护区的面积不宜大于800㎡，且容积不宜大于3600m³，本题根据面积2400/800=3个，根据体积2400×5/3600=4个，每层划分为4个防护区，该建筑4层×4个=16个防护区；根据5.0.3对于平时无人工作的防护区，可设置为无延迟的喷射，故D项符合要求；根据4.3.1，IG541气体灭火系统的储存容器应采用无缝容器。
