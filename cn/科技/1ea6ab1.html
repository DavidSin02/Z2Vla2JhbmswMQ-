剑指 Offer 51. 数组中的逆序对

2020-10-29
| 科技

在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组，求出这个数组中的逆序对的总数。例如输入数组[7,5,6,4]，则有逆序对[7,5]，[7,6]，[7,4]，[5,4]，[6,4]。一共6个逆序对。

这个是我一个曾经的面试题，当时只要求说思路，没让写代码，这次记录一下代码。

首先，看到逆序对，第一反应应该就是冒泡排序中数据交换的次数，但是冒泡排队时间复杂度较高。再次观察，我们可以思考到另一种排序方案，归并排序。以数组[7,5,6,4]为例子，具体思路如下图：

1、对数组的拆分，排序合并。

2、在每一个合并步骤时，两边数组都是有序的，当找到第一个逆序对时，右边数组后面的数据与当前比较的左边数组的数据全部可生成逆序对。

3、归并排序完成，逆序对个数计算也完成。时间复杂度优于冒泡排序。

具体代码如下：

def mergesort(list):    if len(list) <= 1:        return list, 0    mid = len(list)//2    left_li,l_cnt = mergesort(list[:mid])    right_li,r_cnt = mergesort(list[mid:])    #排序所需的辅助空间    temp = []    cnt = l_cnt + r_cnt		#the index of the left_list and the index of the right_list    lindx, rindx_p = 0, 0     #Merging the ordered arrays    while lindx < len(left_li) and rindx_p < len(right_li):        if left_li[lindx] > right_li[rindx_p]:         		temp.append(left_li[lindx])            cnt += len(right_li[rindx_p:])            lindx += 1        else:            temp.append(right_li[rindx_p])            rindx_p += 1    temp += left_li[lindx:]    temp += right_li[rindx_p:]    return temp, cntif __name__ == "__main__":    l=list(map(int,input().split()))    list,res=mergesort(l)    print(res)
