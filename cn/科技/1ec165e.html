DevexpressGridControl常用属性设置看这篇就够了

2020-10-29
| 科技


GridControl常用属性设置

1.隐藏最上面的 GroupPanel

gridView1.OptionsView.ShowGroupPanel=false;

2.数据只读

gridView1.OptionsBehavior.Editable=false;

或者设置成一次选择一行，并且不能被编辑

gridView1.OptionsBehavior.Editable = false;

this.gridView1.FocusRectStyle =

DevExpress.XtraGrid.Views.Grid.DrawFocusRectStyle.RowFocus;

this.gridView1.OptionsSelection.EnableAppearanceFocusedCell = false;

3.不显示 MasterDetailView

gridView1.OptionsDetail.EnableMasterViewMode=false;

4.修改最上面的 GroupPanel 内容

gridView1.GroupPanelText="分组面板";

5.设置数据源:

gridControl1.DataSource = dt;

绑定每一列的 FiledName--属性。对于 Oracle 数据库,所有的在设计器里写的程序中

的字段名必须大写,否则可能绑定不上字段,sqlserver 没有这个限制。

6.读写拷贝权限设置

只读不可拷贝：

ColumnViewOptionsBehavior.Editable = False

只读可拷贝:

ColumnViewOptionsBehavior.Editable = True

OptionsColumn.AllowEdit = True

OptionsColumn.ReadOnly = True

可编辑:

ColumnViewOptionsBehavior.Editable = True

OptionsColumn.AllowEdit = True

OptionsColumn.ReadOnly = False

7.设某一列文字和标题局中显示

gridView1.Columns[0].AppearanceHeader.TextOptions.HAlignment =

DevExpress.Utils.HorzAlignment.Center;

gridView1.Columns[0].AppearanceCell.TextOptions.HAlignment =

DevExpress.Utils.HorzAlignment.Center;

8.去掉某一列上面的自动筛选功能(Filter)

gridView1.Columns[0].OptionsFilter.AllowAutoFilter = false;

gridView1.Columns[0].OptionsFilter.AllowFilter = false;

gridView1.Columns[0].OptionsFilter.ImmediateUpdateAutoFilter = false;

9.设置冻结列(左冻结)

gridView1.Columns[0].Fixed= DevExpress.XtraGrid.Columns.FixedStyle.Left;

10.得到单元格数据(0 行 0 列)

string text=gridView1.GetRowCellDisplayText(0, gridView1.Columns[0]);

string value = gridView1.GetRowCellValue(0, gridView1.Columns[0]);

11.设置单元格数据(将 0 行 0 列的单元格赋值 123)

gridView1.SetRowCellValue(0, gridView1.Columns[0], "123");

12.手动添加 dev 的列

DevExpress.XtraGrid.Columns.GridColumn Col1=new

DevExpress.XtraGrid.Columns.GridColumn ();

Col1.FieldName="FID";

Col1.Visible=true;

Col1.VisibleIndex=gridView1.Columns.Count;

gridView1.Columns.Add(Col1);

13.删除: (修改了 dgvdel 里的 datagridviewdel 方法)

public static void datagridviewdel_Dev(DevExpress.XtraGrid.Views.Grid.GridView

Mydgv)

{

if (MessageBox.Show("你确定要删除选中的记录吗？", "删除提示",

MessageBoxButtons.YesNo, MessageBoxIcon.Warning,

MessageBoxDefaultButton.Button2, 0, false) == DialogResult.Yes)

{

int iSelectRowCount = Mydgv.SelectedRowsCount;

if (iSelectRowCount > 0)

{

Mydgv.DeleteSelectedRows();

}

}

}

14.新增: (对于新增,其本身的 AddNewRow 方法就可以做到)

private void btn_add_Click(object sender, EventArgs e)

{

gridView1.AddNewRow();

}

具体如果对于新加行还有什么特别的设置,可以在它 gridView1_InitNewRow 事件中

填写:

private void gridView1_InitNewRow(object sender,

DevExpress.XtraGrid.Views.Grid.InitNewRowEventArgs e)

{

ColumnView View = sender as ColumnView;

View.SetRowCellValue(e.RowHandle, View.Columns[0],

gridView1.GetRowCellValue(gridView1.GetRowHandle(gridView1.RowCount - 2),

gridView1.Columns[0])); //复制最后一行的数据到新行

View.SetRowCellValue(e.RowHandle, View.Columns[1],

gridView1.GetRowCellValue(gridView1.GetRowHandle(gridView1.RowCount - 2),

gridView1.Columns[1])); //复制最后一行的数据到新行

}

15.检查数据的有效性

在 gridview 的 ValidateRow 事件中加入检查代码：

#region 检查数据

private void gridView1_ValidateRow(object sender, ValidateRowEventArgs e)

{

GridView view = sender as GridView;

view.ClearColumnErrors();

if (view.GetRowCellValue(e.RowHandle, "ReceiveDate") == DBNull.Value)

{

e.Valid = false;

view.SetColumnError(view.Columns["ReceiveDate"], "必须指定日期");

}

}

#endregion

调用 gridview.UpdateCurrentRow()方法执行检查

16.数据库图片转换并显示

/// <summary>

/// 由资源文件获取图片

/// </summary>

/// <param name="key"></param>

/// <returns></returns>

byte[] GetImage(int key)

{

Image img = DevExpress.Utils.Controls.ImageHelper.CreateImageFromResources(

string.Format("RiverSys.Resources.{0}.gif", key.ToString()),

typeof(RiverInfos).Assembly);

return DevExpress.XtraEditors.Controls.ByteImageConverter.ToByteArray(img,

ImageFormat.Gif);

}
