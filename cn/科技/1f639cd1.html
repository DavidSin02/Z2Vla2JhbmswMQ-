发射器、接收器和突发编码器，美国CIA模块化短波间谍无线电设备

| 科技


六十年代美国中情局研发的模块化短波间谍电台RS-59

RS-59是一种模块化短波（SW）间谍无线电设备，由美国中央情报局（CIA）的技术服务部门（TSD）于20世纪60年代末开发。该套装包括一个RR-59接收器，一个RT-59发射器，一个CK-8脉冲编码器和几个接口和附件。尽管没有关于这一组的详细信息，但从Stasi 2档案中可以看出，在冷战期间，前DDR（东德）已经抓获了一套完整装备。

RS-59是一款低功率中程无线电台，适用于长达600英里的距离。它与现有的远程RS-49互补，后者具有更高的输出功率。 RS-59由中央情报局内部完全开发，由中央情报局和其他人建造。

下图显示了RT-59发射器，它是目前保存在集合中的RS-59组的唯一部分。该装置尺寸为105 x 38 x 63 mm，重量仅为328 g。所有连接（天线除外）均通过右前方的滑动连接器进行。

RS-59的开发和改进从1966年到1976年进行了10年。RR-59接收器是第一个发布的单元，原型于1966年5月上市。那时，RT-59发射机和RS-59电台的其余部分“预计很快”。为了减少互连电缆的数量，同时由（未知）外部方开发了一种新型的高质量薄型滑动连接器。

在接下来的几年中，由于电子部件的短缺，RS-59接收机组件的生产被推迟了几次。此外，组件被改进了几次，导致型号中的附加标识符，例如，RR / C-59和RT-59A。

1976年，推出该套装10年后，RT-59发射器由改进的RT-59A继承，提供10瓦的输出功率。到1976年6月，前30个单位由中央情报局内部生产，还有50个单位正在进行中，另外还有专门建造的CC-59便携箱。其他地方将生产一批另外75个单位。几年前（1974年）开发了自动20瓦3-30 MHz版RT / B-59。






全套装备

下图概述了构成RS-59间谍无线电台的所有组件。左半部分由无线电设备的实际部分组成，而右半部分则显示用于以非常高的速度发送预编码消息的CK-8突发发射机，以便通过手段最小化无线电测向（RDF）拦截和发现的风险。


完整的RS-59间谍电台。

RS-59接收机的单元（左）通过每个模块一侧的滑动触点相互连接。电池组和左前方的电源PSU清晰可见这些触点。 RR / C-59接收器有时作为独立接收器提供。


完整的RS-59套装公文包。

上图是在前斯塔西（MfS）的档案中找到的。它显示了一个普通的公文包，其中一个RS-59无线电台配有CK-8脉冲编码器（GRA-71）和电缆，存放在保护性泡沫嵌体的切口中。在冷战期间，它在DDR中被没收。根据手写的斯塔西笔记，它来自美国特勤局INSCOM 1 。

发射器RT-59

发射器尺寸为105 x 38 x 63 mm，与接收器大致相同，重量为328克。它产生的最大输出功率约为5瓦，覆盖频率范围3-16 MHz，分为四个频段，可通过BAND选择器选择。


上图显示了一个典型的RT-59装置，因为它是在1966年设计的。左上方是两个弹簧式端子，用于连接线天线（ANT）和合适的平衡器（GND）。天线端子右侧的4位LOAD选择器用于选择天线电路的最佳匹配，使用小型仪表作为指示器。

合适的HC-25 / U晶体可安装在右前方的2针插座中，而BAND选择器则设置为相应的频率范围：3-5,5-8,8-12或12-16 MHz 。

右上角是MODE开关。在最低位置（RECEIVE），接收器被启用（通过PSU）。在中心位置，发射器被启用，但不直接产生信号。在此位置，发射器受突发发射器的控制。为了调节变送器的最大输出功率，将开关推入（并保持）到KEY位置，并使用微型仪表作为指示器调节TUNE（在左下角）以获得最大功率。

前表面右侧的结构是专门设计的10向滑动连接器，可与PSU上的类似连接器配合使用。通过该连接器，向发射器提供电力，发射器控制接收器，并且突发编码器控制传输。在两个短边的中心是圆形防潮压力膜。更高版本的发射器提供10瓦（RT-59A）甚至20瓦（RT / B-59）。

接收机RR-59


接收器尺寸为100 x 45 x 22 mm。它支持多达10个晶振控制通道，这些通道预先安装在控制面板左侧的可拆卸盖子下面。实际频率写在盖子上的标签上。此外，外部晶振可以安装在左前方的插座上，在这种情况下，11位通道选择器（插座右侧）设置为0。

该设备有一个2针插座，用于连接附带的耳机，两个线端子用于连接天线和右上角的平衡器（接地）。接收器覆盖3至24 MHz的频率范围，分为三个频段。频段选择器（位于右前方）也用作电源开关。在最左边的位置，接收器关闭。

右侧的滚花旋钮（TUNE）用于将RF电路调谐到所选频率。其余两个旋钮用于音量（GAIN）和Beat Frequency Oscillator（BFO）设置。

虽然这在右侧图像中不可见，但据信应该向设备后部的滑动触点供电。接收器可以直接由电池或电源PSU（独立）供电，也可以通过RS-59接口单元（作为整个站的一部分）供电。

突发编码器CK-8

为了最大限度地降低敌方监测站拦截的风险，或通过无线电测向（RFD）发现，通常的做法是通过使用所谓的突发编码器结合使用，尽可能短。突发发射器（也称为键控器）。

RS-59配备了CK-8键控器/编码器，这是众所周知的GRA-71的CIA版本。它使用磁带盒，通过此处显示的CO / B-8字母编码器在其上记录预编码（通常加密）的消息。

为此，将CA / A-3B盒式磁带连接到编码器的磁头上。然后将字母表盘设置为消息的第一个字母，然后拉动手柄。然后将这封信记录在磁带上。对每个字母重复该过程，直到记录整个消息。







一旦将信息记录（编码）到磁带上，磁带就连接到KE-8B键控器上，而键控器则通过小型AA-59接口（和RS-59单元）连接到RT-59发送器。在重绕磁带并卷起键控器的弹簧机构后，按下一个按钮以高速发送记录的信息，称为突发传输。

内部电路

发射器由一块由实心铝块铣制而成的框架组成，其中前面板构成一个整体部件。该装置采用铝制外壳封装，通过三个凹入式螺钉固定到位：一个位于底部，一个位于前部，一个位于右侧。

一旦拆下这三个螺钉，就可以取下外壳。这可能需要一点力，因为壳体可以由于在装置内部使用厚的保形涂层而被束缚。


移除外壳后，内部会露出，如上图所示。左边是印刷电路板（PCB），带有大部分电子部件。右半部分包含控制器，功率放大器（PA）晶体管和一些无源元件。右前方是天线匹配器，由多抽头环形线圈和可调电容器组成。







环形线圈的抽头连接到控制面板上的4位带选择器，而电容器（TUNE）用于最大化调谐电路的效率，从而最大化发射器的功率输出。右侧是一个微型继电器，用于在控制面板上的3位MODE开关的控制下在发送和接收之间切换。
