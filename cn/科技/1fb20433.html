DIY｜装修边角料变身家居好物！省出一个亿！

2020-11-14
| 科技

装修边角料别浪费，只需一些简单的工具和非常容易买到的辅助材料，就能做出实用的家居好物呢！

1. 折叠沙滩椅 难度系数 一星

材料和工具：

切出33 x 38厘米的胶合板 2块

直径4mm，长度1.5米的聚丙烯绳 1根

直径4mm,长度1m的弹力绳2个

直径4mm,长度0.5m的弹力绳 1根

直径5mm的钻头和钻机

磨料板

操作步骤：

在每块板上钻12个孔，直径为5mm

将直径4mm,长度0.5m的弹力绳插入板孔，制作椅子的铰链。

在弹力绳的两端系一个结。

将两根直径为4毫米，长度为1米的弹性绳插入板的孔中来制作两个十字架。

翻转两块板。

在两根弹性绳的两端系上一个结。

将直径4毫米，长度1米50的聚丙烯绳插入板的孔中，制作成椅子的提手。

用绳子的两端进行渔夫结。

完成！还可以在十字叉内装杂志、毛巾等小物品。走到哪里随时打开就可以坐啦，非常方便！

​

2. 碗碟架 难度系数 两星

材料和工具：

塑料托盘 1个

21 x 21 mm, 长2.5 m的木条

S形钩

不锈钢餐具筒 1个

手锯

螺丝刀

3mm木钻

5mm金属钻

尺子、铅笔

使用手锯和斜接箱切割木条：

- 2个长度为20厘米

-5个长度为32.5厘米

涂上颜色

在32.5厘米的木条上，用4毫米木钻钻5个孔，拧紧螺栓。

使用4 x 30 mm螺钉组装成下图样式

托盘中钻5mm的孔，一侧3个，距离边缘10cm，另一侧5个，用于沥水。

在不锈钢餐具筒上穿一个孔。

用4 x 35 mm的螺丝将钢板拧到板条框架上，把餐具筒用S形钩挂在上面。

完成！这款2合1滤水碗碟架既美观又巧妙，十分节省空间，可谓厨房必备良品！

​

3. 雨伞架 难度系数 两颗星

材料和工具：

L.80 X L.60厘米，厚18mm的木板1块

长1m，直径22mm的光滑山毛榉细木棍18个

清漆、彩漆若干，螺钉若干

电钻，尺子，铅笔，磨砂纸等

在L.80 X L.60厘米的18mm板上切割出：

- 2个33 x 33厘米的正方形

- 2个33 x 6.5厘米的长方形

- 2个29.4 x 6.5厘米长方形

将长1M,22 X 22mm粗的山毛榉细木棍切成两半。

在两个33 x 33厘米的正方形板上，留出边缘4cm，每隔5厘cm打一个孔，一共36个。

作为底部的正方形板，用直径4mm的钻头，作为顶部的正方形板，用直径22mm的钻头。

组装木板做成底座。

将底座涂成灰色，将16个山毛榉细木棍涂成孔雀蓝色，再涂一层无色清漆。

将36个细木棍拧入底座顶部的孔中，然后将它们从底座下方拧入预孔中。

完成！这个功能与装饰兼备的雨伞架适用于办公室等公共场所，用于家庭中不妨开发它的其他功能，去掉几个细木棍也可以的。

​

4. 储物小边桌

材料和工具：

18毫米胶合板31.4 x 28.1厘米 4个

18毫米胶合板53.8 x 53.8厘米 1个

10毫米胶合板53.8 x 53.8厘米 1个

木制凳脚直径50 x H.80 mm x W.50 mm 4个

油漆、密封胶、螺丝、木栓若干

钻头、刷子、砂纸、刀具等

切割32件28.1厘米的木条，

在两侧涂上喜欢的颜色。

切割胶合板，尺寸如下：

将4个31.4 x 28.1厘米的胶合板装到侧面标记处，用螺丝拧紧固定。

根据标记预先刺穿底部和顶部，将32个木条装到侧面，用钉子固定。

在上面一个面的边框两侧中间位置各打一个3mm的孔，将最后一块板盖在上面，用木销钉固定，木销钉随时可以拔出，方便边桌储物。

将桌脚拧在家具下面，还可以刷一层无色清漆或你喜欢的颜色。

完成！这款矮桌既可以作为小边桌使用，还兼具储物功能，甚至可以拿来充当临时凳子，非常实用的小家具！
