高考化学一轮提分练：考点6-附加条件的离子共存解答

2020-10-29
| 科技

一、题干中有附加条件的离子共存判断

1．(2017·四川双流中学高三第一次月考)在pH＝1的无色透明溶液中，能大量共存的离子组是( )

A．Al3＋、Ag＋、NO3(－)、SO4(2－)

B．Mg2＋、NH4(＋)、NO3(－)、Cl－

C．Cu2＋、Cl－、NO3(－)、Cl－

D．K＋、Na＋、ClO－、S2－

2．(2017·衡阳八中高三第二次月考)某溶液能与锌片作用产生H2，则该溶液中可以大量共存的离子组是( )

A．Na＋、NO3(－)、K＋、Fe2＋

B．Fe2＋、Na＋、SO4(2－)、K＋

C．K＋、I－、NO3(－)、H＋　

D．NH4(＋)、NO3(－)、Na＋、HCO3(－)

3．(2017·定州中学高三第一次月考)下列各组离子，在溶液中能大量共存，加入NaOH溶液后加热既有气体放出又有沉淀生成的一组是( )

A．Ba2＋、NO3(－)、NH4(＋)、Cl－

B．Ca2＋、HCO3(－)、NH4(＋)、OH－

C．K＋、Ba2＋、Cl－、HSO3(－)

D．Mg2＋、NH4(＋)、SO4(2－)、K＋

4．(2017·枣阳一中高三开学考试)由水电离出来的氢离子浓度为10－13 mol·L－1的溶液中，下列各组离子一定不能大量共存的是( )

A．Na＋、NO3(－)、Al3＋、K＋

B．Na＋、ClO3(－)、ClO－、ClO4(－)

C．K＋、Cr2O7(2－)、I－、Na＋

D．Na＋、S2O3(2－)、SO4(2－)、NH4(＋)

5．(2017·河北武邑中学高三上学期周考)在溶液中加入足量Na2O2后仍能大量共存的离子组是( )

A．NH4(＋)、Ba2＋、Cl－、NO3(－)

B．K＋、AlO2(－)、Cl－、SO4(2－)

C．Ca2＋、Mg2＋、NO3(－)、HCO3(－)

D．Na＋、Cl－、CO3(2－)、SO3(2－)

二、选项中有附加条件的离子共存判断

6．(2017·凯里一中高三上学期月考)常温下，下列各组离子一定能在指定溶液中大量共存的是( )

A．使pH试纸变蓝色的溶液中：K＋、Al3＋、NO3(－)、SO4(2－)

B.c(H＋)(Kw)＝1×10－12 mol·L－1的溶液中：NH4(＋)、Cu2＋、Cl－、SO4(2－)

C．有较多Fe3＋的溶液中：Na＋、SCN－、K＋、Na＋、CO3(2－)

D．与Al反应能放出H2的溶液中：Fe2＋、K＋、NO3(－)、Cl－

7．(2017·葫芦岛市六校协作体高三期初)常温下，下列各组离子在指定溶液中一定能大量共存的是( )

A．使甲基橙变红的溶液中：Na＋、Cl－、CH3 COO－、Fe2＋

B．水电离的OH－浓度为10－12 mol·L－1的溶液中：Mg2＋、NH4(＋)、AlO2(－)、SO3(2－)

C．无色溶液中：Al3＋、NO3(－)、Cl－、SO4(2－)

D．c(K2C2O4)＝0.5 mol·L－1溶液中：Na＋、K＋、MnO4(－)、Cl－

8．(2017·扬州中学高三开学测试)25 ℃时，下列各组离子在指定溶液中一定能大量共存的是( )

A．0.1 mol·L－1 AlCl3溶液中：H＋、Na＋、Cl－、SO4(2－)

B．能使紫色石蕊溶液变蓝的溶液：Ag＋、Fe3＋、Br－、SO4(2－)

C．能使淀粉碘化钾试纸显蓝色的溶液：K＋、SO4(2－)、S2－、SO3(2－)

D．由水电离产生的c(H＋)＝10－12 mol·L－1的溶液中：NH4(＋)、SO4(2－)、HCO3(－)、Cl－

9．(2017·赣州十四校高三上学期期中联考)在不同温度下，水溶液中c(H＋)与c(OH－)有如图所示关系。下列条件关于离子共存说法中正确的是( )


A．a点对应的溶液中大量存在：Fe3＋、Na＋、SCN－、SO4(2－)

B．b点对应的溶液中大量存在：Na＋、Fe3＋、HCO3(－)、I－

C．c点对应的溶液中大量存在：Na＋、Ca2＋、Cl－、CO3(2－)

D．d点对应的溶液中大量存在：Na＋、K＋、SO3(2－)、Cl－

10．(2016·河北高三入学考试)下列各组离子在指定溶液中一定能大量共存的是( )

A．25 ℃时，c(H＋) /c(OH－)＝1×10－12的溶液中：Na＋、SO3(2－)、ClO－、CO3(2－)

B．能与金属铝反应放出氢气的溶液中：K＋、Fe2＋、Cl－、NO3(－)

C．0.1 mol·L －1Ba (OH)2溶液中：Al3＋、NH4(＋)、NO3(－)、HCO3(－)

D．0.1 mol·L－1 AlO2(－)溶液中：Na＋、K＋、SO3(2－)、CO3(2－)
