VBA｜使用和定制Excel内置对话框

2020-11-14
| 科技

在Excel中，对话框可以实现应用程序与用户的交互。在VBA中，Excel提供了调用这些内置对话框的接口，使开发者可充分利用内置对话框实现更多的操作。

1 用FindFile方法打开文件

Sub 打开工作簿()

Dim wb As Workbook

Set wb = ActiveWorkbook

If Application.FindFile Then '显示“打开”对话框

MsgBox "已打开工作簿文件！"

wb.Activate

Else

MsgBox "打开工作簿失败！"

End If

End Sub

'上面代码运行后，显示“打开”对话框，选择一个工作簿，单击“打开”后即可打开该工作簿

2 使用对话框获取单个文件名

2.1 使用GetOpenFilename方法显示“打开”对话框，获取选择的文件名

Sub 获取单个文件名()

Dim strFilt As String

Dim strTitle As String

Dim strFname As Variant

strFilt = "文本文件(*.txt),*.txt," & _

"Excel文件(*.xls;*.xlsx;*.xlsm),*.xls;*.xlsx;*.xlsm," & _

"所有文件(*.*),*.*"

strTitle = "打开Excel文件"

strFname = Application.GetOpenFilename _

(fileFilter:=strFilt, _

Title:=strTitle)

If strFname = False Then

MsgBox "没选择文件！"

Else

MsgBox "选择的文件是：" & strFname

End If

End Sub

2.2 使用GetSaveAsFilename方法打开“另存为”对话框获取文件名

Sub 获取保存文件名()

Dim fn As String

fn = Application.GetSaveAsFilename( _

fileFilter:="Text Files (*.txt), *.txt")

If fn <> "False" Then

'在此处编写保存文件的代码

MsgBox "保存文件为：" & fn

End If

End Sub

3 使用对话框获取多个文件名

Sub 获取多个文件名()

Dim strFilt As String

Dim strTitle As String

Dim strFname As Variant

Dim i As Integer

Dim strMsg As String

strFilt = "文本文件(*.txt),*.txt," & _

"Excel文件(*.xls;*.xlsx;*.xlsm),*.xls;*.xlsx;*.xlsm," & _

"所有文件(*.*),*.*"

strTitle = "打开Excel文件"

strFname = Application.GetOpenFilename _

(fileFilter:=strFilt, _

Title:=strTitle, _

MultiSelect:=True) '允许选择多个文件

If Not IsArray(strFname) Then '返回值不是数组

MsgBox "没选择文件！"

Else

For i = LBound(strFname) To UBound(strFname)

strMsg = strMsg & strFname(i) & vbCrLf

Next

MsgBox "选择的文件是：" & vbNewLine & strMsg

End If

End Sub

4 使用Dialogs集合可以调用所有Excel内置对话框

Excel2007中有1101个内置对话框，如下面代码可以调用“对齐"标签页：

Sub 对齐对话框()

Application.Dialogs(xlDialogAlignment).Show

End Sub

5 设置内置对话框的初始值

在VBA中，可以通过更改对话框的选项参数，定制用户需要的对话框：

Sub 设置字体1()

Application.Dialogs(xlDialogFontProperties).Show _

"黑体", "粗体", 20, True, True, False, , _

True, 5, 4, True, 3

End Sub

Sub 设置字体2()

Application.Dialogs(xlDialogFontProperties).Show _

arg1:="黑体", arg2:="粗体", arg3:=20, _

arg4:=True, arg5:=True, arg7:=True, arg8:=True, _

arg9:=5, arg10:=4, arg11:=True, arg12:=3

End Sub
