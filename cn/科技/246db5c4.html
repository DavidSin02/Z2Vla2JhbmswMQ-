C++｜区分两个概念，你就能彻底了解指针概念

2020-11-14
| 科技

在区分两个概论前先了解“存储程序”概念：

世界上的第一台电子计算机的程序是通过扳动开关和插拨电线实现的，自然，这样操作费时费力且无法实现自动化。后来冯诺依曼提出了“存储程序”的概念：把程序（指令序列）存储到一个存储器中，程序执行过程中产生的数据也同样的存储到存储器中，存储器中的数据和指令可以通过地址随机访问，指令可以顺序执行，可以跳转（由此实现分支和循环）。

存储器中的数据可以是只读的，也可以是可读可写，后者对于存储单元地址的命名就是变量（毕竟存储地址抽象难记），为方便起见，对变量的操作就是操作由变量名对应的地址所指向的一组内存单元所对应的数据值。变量名显式使用的是其数据值，地址值只是隐含地表示（可用符号&取出其地址）。

如果想显式地用一个变量来表示一个内存单元的地址，这样的变量就是指针变量，指针变量存储一个指针（地址）。指针变量与普通变量在隐式或显式地表示数据和地址时刚好相反，指针变量是显式地表示一个地址，隐式地表示一个地址对应的数据值（使用解引用符号*）。

1 指针声明与指针解引用

对于符号“*“，不同的上下文有不同的含义，可以是指针声明，也可以是指针引用：

int a ＝3;int* p=&a; // 符号*前有数据类型时，是指针声明(*p)++; //符号*前无数据类型时，是指针的解引用，用于更新指针指向的地址对应的数据值

2 指针本身的地址、指针的地址值、指针的地址值指向的数据值

了解以上三个概念，分别看以下代码实例及注释：

#include <iostream>using namespace std;void main(){	int arr[]={3,5,7,9,11};	int* p=arr; // 符号*前有数据类型时，是指针声明	cout<<"指针本身的地址值"<<&p<<endl; // ① 指针本身的地址值	for(int i=0;i<sizeof(arr)/sizeof(arr[0]);++i)	{		(*p)++; //符号*前无数据类型时，是指针的解引用，用于更新指针指向的地址对应的数据值		cout<<p<<"：";// ② 指针的地址值		cout<<*p<<" "; // ③ 指针的地址值所对应的数据值		p++; //指针单独做左值时(无符号*)，是指针的地址值（不是地址指向的值）更新，表示指针的移动；	}	cout<<p<<endl;	cout<<endl;	char* str = "abcdefghij";	char* pc = str;	while(*pc++);//先pc++	cout<<pc-str<<endl; system("pause");}/*指针本身的地址值0012FF300012FF34：4 0012FF38：6 0012FF3C：8 0012FF40：10 0012FF44：12 0012FF4811*/
