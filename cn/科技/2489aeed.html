PHP变量

PHP 是一门弱类型语言，我们不必向 PHP 声明该变量的数据类型，PHP 会根据变量的值，自动把变量转换为正确的数据类型。

PHP 变量规则：

  • PHP 变量以 "$" 符号开始，$符后面便是变量名；
  • PHP 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ），且必须以字母或者下划线字符开始，不能以数字开头；
  • PHP 变量名中不能包含有空格；
  • PHP 是严格区分大小写的，变量$y 和 $Y 是两个不同的变量；
  • PHP 没有单独声明变量的命令，变量在第一次赋值的时候就被自动创建；
  • PHP 赋一个字符串给变量时，需要在字符串两侧加上引号。

实例：

<!DOCTYPE html><html><head> <meta charset="utf-8"></head><body><?php$i = 36;$I = "Lucifer";$f = 123.45;echo "\$i的类型和值：";var_dump($i);echo "<br>\$I的类型和值：";var_dump($I);echo "<br>\$f的类型和值：";var_dump($f);?></body></html>

PHP 变量作用域：

  • local：函数内部声明的变量拥有 LOCAL 作用域，只能在函数内部进行访问。
  • global：函数之外声明的变量拥有 Global 作用域，只能在函数以外进行访问。
  • static：调用该函数后，static局部变量不会被删除，该变量仍然是函数的局部变量。
  • parameter：通过调用代码将值传递给函数的局部变量。

在所有函数外定义的变量拥有全局作用域（global），在函数内部声明的变量是局部变量（local），这两种作用域是默认存在的，不需要特意声明，局部变量仅能在函数内部访问，全局变量仅能在函数外部访问。要在一个函数中使用全局变量，需要使用global关键字。

实例：

<!DOCTYPE html><html><head> <meta charset="utf-8"></head><body><?php$i = 36;$I = "Lucifer";$d = 2643699.89;//这里的变量为全局变量，无法直接在函数中使用function a(){ global $I; //使用global关键字来使用全局变量 $s = 26; //这里是局部变量，无法在函数以外使用 echo $I."今年".$s."岁了！"; echo "<br>";}function b($f){ //这里的$f是传递的参数 global $I; echo $I."的房子价值".$f."元！"; echo "<br>";}a();b($d);//将$d作为参数传递给函数b?></body></html>
