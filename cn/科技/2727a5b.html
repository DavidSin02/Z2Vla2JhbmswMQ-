如何在win10中ubuntu子系统安装桌面环境

2020-10-29
| 科技

win10内置ubuntu子系统安装桌面版教程

注意！！！：

本教程仅作为普及，不代表个人观点!!!。

(1)必备条件：

win10已安装WSL，并可以运行Bash。

补充：

WSL：windows subsystem for linux。

如果没有安装wsl，请参考：

手把手教你启用win10系统中内置的Linux子系统

(2)思路之曲线救国：

WSL没有x-windows来提供图形化界面的支持，但我们可以在windows下安装x-windows，然后在这个桌面环境下显示linux应用，从而达到间接使用ubuntu-desktop。

(3) 操作步骤

1、 安装x-windows

可供选择安装的X-Windows有多个，这里我们使用VcXsrv。

官网：https://sourceforge.net/

下载好软件，在windows下安装好。

首次启动自动进入界面设置后，按下图设置：

选择one large window

Display number设置成0

以下安装其它默认即可

2、 在我们的Ubuntu系统中安装桌面环境

打开Bash，安装ubuntu-desktop、unity、ccsm。

sudo apt-get install ubuntu-desktop unity compizconfig-settings-manager

软件包比较大，稍稍等待几十分钟...

当然这里也是看个人的网速......

如安装遇到问题，可能是源的问题，请参考：

Ubuntu更换为阿里云软件源

3、 配置compiz窗口管理器

(1)启动之前安装的x-windows

(2)在Bash中执行如下命令

export DISPLAY=localhost:0

sudo ccsm

(3)

此时在X-windows中，会弹出ccsm的配置界面，勾选需要的Desktop组件。

这里注意要勾选Desktop中的Ubuntu Unity Plugin，其它默认就好。

(4)

关闭ccsm，在bash中启动compiz，加载Unity-desktop。

打开VcXsrv，我们的ubuntu界面就出现在面前了
