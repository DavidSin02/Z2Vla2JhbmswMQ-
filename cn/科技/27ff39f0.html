第三节 平整度

1. 平整度是指道路表面相对于理想平面的竖向偏差。

2. 路表的不平整会增大行车阻力，使车辆产生附加振动，造成行车颠簸，影响乘客舒适性。

3. 振动作用还会对路面施加额外冲击力，从而加剧路面和汽车机件损坏和轮胎磨损，增大油耗。

4. 不平整的路表会积滞雨水，不仅加速路面损坏，也给行车带来安全隐患。

5. 常见的平整度测试设备有3m直尺、连续式平整度仪、颠簸累积仪、激光平整度仪四种，可分为断面类和反应类两大类。

1.1 3m直尺测定平整度试验方法

1. 原理：用3m直尺基准面距离路表面的最大间隙（以mm计）反映路基路面表面的凹凸情况。最大间隙值越大，说明路表面越不平整。

2. 本方法适用于测定压实成型的路面各层表面的平整度，以评定路面的施工质量，也可用于路基表面成型后的施工平整度检测。

1.2 连续式平整度仪测定平整度试验方法

1. 原理：按一定采样间距量测路表面与八轮仪机架的基准点之间距离的单向偏差（凸起或凹下），以一定长度区间的标准差σ（以mm计），反映路面的平整度。标准差越大，路表面越不平整。

2. 本方法适用于测定路表面的平整度，评定路面的施工质量和使用质量，但不适用于在已有较多坑槽、破损严重的路面上测定。

3. 用连续式平整度仪测定平整度时，通常由小面包车或其他小型汽车牵引连续式平整度仪工作。

1.3车载式颠簸累积仪测定平整度试验

1. 原理：测试车以一定的速度在路面上行驶，由于路面凹凸不平，引起汽车的激振，通过测量车后轴与车厢之间的单向位移累积值VBI（以cm/km计）来表征路面的平整度状况。

2. 累积值VBI越大，说明路面平整度越差，行车舒适性越不好。

3. 本方法适用于各类颠簸累积仪在新建、改建路面工程质量验收和无严重坑槽、车辙等病害的正常行车条件下连续采集路段平整度数据。

4. 本方法的数据采集、传输、记录和处理均由专用软件自动控制进行，可根据相关关系方程式自动换算并输出国际平整度指数IRI。

5. 测试系统由承载车辆、距离测量装置、颠簸累积值测量装置和主控制系统四部分组成。

6. 承载车、测试速度30-80km/h、最大测试幅值±20cm、垂直位移分辨率1mm、距离标定误差＜0.5%、系统工作环境温度0-60℃。

1.4车载式激光平整度仪测定平整度试验方法

1. 激光平整度仪为应用激光测距及加速度惯性修正技术测量路面纵断面高程计算路面国际平整度指数IRI的设备。

2. 本方法适用于各类车载式激光平整度仪在新建、改建路面工程质量验收和无严重坑槽、车辙等病害及积水、积雪、泥浆的正常通车条件下连续采集路段平整度数据。

3. 本方法的数据采集、传输、记录和处理分别由专用软件自动控制进行。

4. 测试系统由承载车辆、距离传感器、纵断面高程传感器和主控制系统组成。承载车、测试速度30-100km/h、采样间隔≤500mm、传感器测试精度0.5mm、距离标定误差＜0.1%、系统工作环境温度0-60℃。

第四节 强度及模量

1. 路基路面各结构层的回弹模量是路面结构设计的重要参数，因其现场测试方法复杂，不要求直接检验，可通过现场检测弯沉的方法间接检验。

1.1土基现场CBR值测试方法

2. 原理：在公路路基施工现场，用载重汽车作为反力架，通过千斤顶连续加载，使贯入杆匀速压入土基。路基强度越高，贯入量达到规定值所施加的荷载越大，即CBR值越大。

3. 本试验采用的主要仪具包括荷载装置、现场测试装置、贯入杆、承载板和贯入量测定装置。

1.2动力锥贯入仪测定路基路面CBR试验方法。

1. 动力锥贯入仪（简称DCP）是一种轻型轻便的地基土原位测试的触探仪。

2. 原理：用一定质量的锤从一定高度落下，打击立在路基路面上的锥杆，测定锤击数与锥杆的贯入量；通过贯入度（即平均每次锤击的贯入量）与CBR值的相对关系式，推算路基路面的CBR值。

3. 本方法适用于动力锥贯入仪现场快速测定或评估无结合料材料路基、路面的强度。

4. 常见动力锥贯入仪的标准落锤质量为8kg，落距为575mm，锥头锥尖角度一般为60°，最大直径20mm。实际使用中，对细粒土的检测效果较好。

1.3承载板测定土基回弹模量试验方法

1. 原理：在现场土基表面，通过采用刚性承载板，对土基逐级加载、卸载的方式，测出每级荷载下相应的土基回弹变形，再根据弹性半空间体理论计算求得土基的回弹模量。

2. 本试验主要仪具有加载设施、现场测试装置、刚性承载板、路面弯沉仪、液压千斤顶、秒表、水平尺。

1.4贝克曼梁测定路基路面回弹模量试验方法

1. 原理：在土基或者厚度小于1m的粒料整层表面，用弯沉仪测试数个测点的回弹弯沉值，根据圆形均布荷载作用下的弹性半无限体理论，计算求得该材料的回弹模量值。

2. 本方法同样适用于旧路表面测定路基路面的综合回弹模量。

3. 仪具主要有标准车和路面弯沉仪，路面弯沉仪由贝克曼梁、百分表及表架组成。

1.5回弹仪测定水泥混凝土强度试验方法

1. 适用于在现场对水泥混凝土路面及其他构筑物的普通混凝土抗压强度的快速评定，所试验的水泥混凝土厚度不得小于100mm，稳定应不低于10℃。

2. 回弹仪测定水泥混凝土强度时，先选择测区并布置测点。

3. 在测定过程中，回弹仪的轴线应始终垂直于混凝土表面，按操作规定，测定每个该点的回弹值。

1.6超声回弹法检验路面水泥混凝土抗弯强度试验方法

1. 水泥混凝土路面的混凝土抗弯强度是指标准条件下的梁式试件龄期28d时的抗弯强度。

2. 本方法适用于回弹仪、低频超声仪在现场对水泥混凝土路面按综合法快速检测，并利用测强曲线方程推算混凝土的抗弯强度。

3. 本方法适用于视密度为1.9-2.5t/m³，板厚大于100mm，龄期大于14d，强度已达到设计抗压强度80%以上的水泥混凝土。

1.7射钉法快速检验水泥混凝土强度的试验方法

1. 本方法采用发射枪使射钉射入混凝土，以射钉外露长度代表贯入阻力，通过相关关系快速评定水泥混凝土的硬化强度。

2. 可用于快速评定新浇混凝土的硬化强度，以检测现场混凝土的匀质性，了解质量低劣的部位或范围。

3. 不适用于施工质量的评定验收与仲裁。

4. 适用于抗压强度不大于50MPa，且厚度不小于15cm的水泥混凝土。
