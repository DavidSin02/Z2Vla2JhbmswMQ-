面试宝典之LeetCode算法题 - 2.两数相加

2020-11-14
| 科技

题目描述：

给定两个非空链表来表示两个非负整数，其中的数字是按逆序方式存储的，并且每个节点只存储一位数字（0 <= Node.val <= 9）。例如数字123表示为3->2->1。假定除了数字0之外，这两个数不会以0开头；链表的长度在1到100之间。

将这两个整数加起来，可以得到它们的和，请用上述链表的方式来输出它们的和。


例子1：

输入：l1 = [2, 4, 3], l2 = [5, 6, 4]

输出：[7, 0, 8]

342 + 465 = 807

注意链表是逆序存储的，这两个数字是342和465，而342 + 465 = 807，所以输出7->0->8。


例子2：

输入：l1 = [0], l2 = [0]

输出：[0]


例子3：

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]

输出：[8,9,9,9,0,0,0,1]


思路：

链表长度可能到100，即可能有100位数，所以不要转换为数字再相加（会溢出或丢失精度）。

参照例子1中的图，可以用列竖式的方式计算两个数字的和，用代码来模拟这个过程。

由于是逆序存储的，列表的头节点存储的是个位数字，竖式计算也是从个位数开始的。只需将对应位上的2个数字相加，即可得到结果中相应位的数字。如果有进位，计算下一位的时候把进位也算上。

两个数的位数可能不一样长，计算到后面如果只有一个数字了，只需把另一个数字看成0来计算即可。


JavaScript代码实现：

/** * Definition for singly-linked list. * function ListNode(val, next) { *     this.val = (val===undefined ? 0 : val) *     this.next = (next===undefined ? null : next) * } *//** * @param {ListNode} l1 * @param {ListNode} l2 * @return {ListNode} */var addTwoNumbers = function(l1, l2) {    var n1 = l1; // 当前的数字    var n2 = l2;        var result; // 结果链表的头节点    var previous; // 结果链表中当前位的前一个节点    var carry = 0; // 进位        while (n1 != null || n2 != null) {        var val1 = n1 != null ? n1.val : 0;        var val2 = n2 != null ? n2.val : 0;        var sum = val1 + val2 + carry; // 相应位置的数字相加，不要忘记加上进位        var val = sum % 10; // 得到对应位上和的数字        carry = sum >= 10 ? 1 : 0; // 更新进位        var current = new ListNode(val);        if (result == null) {            // 链表的头节点            result = current;        }        if (previous != null) {            // 把当前节点加入链表            previous.next = current;        }        previous = current;        n1 = n1 != null ? n1.next : null; // 指向下一位的节点        n2 = n2 != null ? n2.next : null;    }        if (carry > 0) {        // 最后可能还有一个进位        previous.next = new ListNode(carry);    }        return result;};
