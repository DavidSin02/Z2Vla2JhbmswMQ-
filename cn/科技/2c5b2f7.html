平遥第三年，片单太硬了

2020-10-29
| 科技

10月10日，第三届平遥国际电影展即将拉开帷幕。

回想两年前，这个深植于中国内地的国际影展，刚刚带着西北的烟火气在全球电影人面前亮相。对故乡山西饱含无限深情的贾樟柯，给了平遥电影展一个定位：“这将是一个尊重电影、分享文化、尊重影迷、尊重电影人劳动成果的电影展。”

在过去的两年里，这个目标的完成度可以说是有目共睹的。

创始人贾樟柯与影展艺术总监马可·穆勒

对观众来说，这里可以享受到国际电影节带来的视觉饕餮盛宴；而对一些青年电影人来说，这是一次向大众介绍自己的宝贵机会。每年10月的平遥，正成为越来越多国内电影人和影迷的必选行程。

1

发出来自中国的声音

说平遥，就不能不说贾樟柯。这个国际影展的从无到有，离不开这位国际名导的促成。

贾樟柯曾说，自己人生中的重要时刻，都是各式各样的电影节赋予的。凭借《小武》，贾樟柯在柏林电影节一鸣惊人；凭借《三峡好人》，他在威尼斯擒获金狮；凭借《天注定》，他拿下戛纳电影节最佳编剧奖。其作品更曾五次入围戛纳主竞赛单元。


如今举办平遥国际电影展，贾樟柯为最初的理想找到了更广阔的场景,并让世界电影听见了更多来自中国的声音。


影展获得了导演李安的授权，将两个竞赛单元分别命名为“卧虎”和“藏龙”。其中，“卧虎”单元致力发掘优秀的新人新作，在全球范围选择新导演的处女作或第二部电影；而“藏龙”单元关注华语新人，放映华语新导演的新锐作品。

过去两年，从平遥走出来的获奖电影如《骑士》《大世界》《嘉年华》《过昭关》和《过春天》等，都成为备受影迷群体推崇的口碑之作。

78岁的杨太义凭《过昭关》获颁费穆荣誉-最佳男演员

新人辈出的华语电影界，正在平遥发出自己的声音。这样的声音，我们不希望错过。

2

“大家和大家”

第三届平遥国际电影展官方海报“电影小星球”，寓意电影是一个独特神奇的星球，所有电影创作者、电影观众都是这个星球上的居民。

“这个星球里有游荡的年轻人，有闪闪发光的创意。”贾樟柯说，“因为十天的电影展，平遥也变成了一个星球。”

而本届电影展主题“大家和大家”，则是电影展的另一重含义。如果你曾经来过平遥影展，一定明白这其中的意味。

从第一年的“平遥元年”，到第二年的“从电影到市集”，每年平遥影展的主题都别有趣味。两千七百年的平遥古城，与艺术家族的幼子不期而遇，仿佛一场火星撞地球的奇妙体验。

平遥影展根植于一个北方古城，“大格局、小身段”是它的特色之一。影展主要集中在一个内部空间，由平遥柴油机厂旧厂房改造的电影宫，很有复古工业感。

电影宫内景

各个影厅几乎只有上下台阶的路程，几天下来，不但能认识一帮影迷朋友，更可以邂逅不少电影主创和业界大佬。和他们在同一影厅看电影。



可容纳近500人的“小城之春”影厅

“大家”，是引领一个行业前行的先行者；“大家”，则是让电影走进千家万户的传播者。“大家与大家”，缺一不可。而他们的相遇，正是在这座神奇的北方古城。

3

这个片单有点硬

本届平遥影展的首批片单公布之后，吸引了不少关注，共发布来自26个国家和地区的52部影片信息。影展开幕前，还将公布最后一批官方单元入围片单。

在官方单元的49部展映影片中，28部为全球首映，36部为亚洲首映，中国首映率则达到100%。在这份目前的展映片单里，还能看到许多今年欧洲三大的口碑之作。

其中“卧虎”单元的入围影片，包括今年戛纳“金摄影机奖”得主《我们的母亲》，在威尼斯首映的、小田切让自编自导的首部长片《一个船伕的故事》；新加坡导演陈哲艺，也带着多伦多归来的《热带雨》亮相平遥。

《一个船伕的故事》

而关注华语新人新作的“藏龙”单元，更有演员祖峰执导、此前入围戛纳的《六欲天》，入围莫斯科、慕尼黑电影节的《蕃薯浇米》，都将在平遥进行亚洲首映。

《蕃薯浇米》

此外，由李仁港执导的《攀登者》将亮相“影迷嘉年华”。本届影展青年导师谢飞的《黑骏马》2K修复版、张艺谋《红高粱》4K修复版将在电影展进行展映。

其中值得关注的几部电影，这就为您做个简单的介绍，准备去平遥的朋友一定别错过这份片单，这些很可能是今明两年的必看片哦~



《我们的母亲》

Nuestras Madres

入围单元：卧虎

导演 / 编剧: 塞萨尔·迪亚兹

主演:Armando Espitia / Emma Dib / Aurelia Caal

制片国家/地区: 危地马拉

电影获得今年戛纳的金摄影机奖（最佳处女作）。故事以2013年的危地马拉内战期间的大屠杀为背景，法医中心的年轻人类学家伊涅斯托的工作，是确认在战争中失踪的受害者的身份。


在跟一位老妇人的谈话中，伊涅斯托意外发现了一条引向他父亲行踪的线索。而后者是在内战中失踪的一名游击队员，从而引起了一系列关于身份、家庭、群体的问题。

女导演塞萨尔·迪亚兹长期在法国学习电影，曾拍了很多纪录短片，这种强纪录的平实质感也表现在电影中。


《一个船伕的故事》

ある船头の话

入围单元：卧虎

导演 / 编剧: 小田切让

主演: 柄本明 / 村上虹郎 / 川岛铃遥 / 伊原刚志

类型: 剧情

制片国家/地区: 日本

小田切让的导演处女作，这个名头已经让《一个船伕的故事》在今年威尼斯广受关注。

《一个船伕的故事》主要讲述在日趋商业化的世界里，一些传统古老的事物逐渐消失。主人公是一位载人渡河的船伕，在他平淡的生活中总会遇到各式各样的人，带来各种故事。一部探讨人性的典型日式影片。

电影的阵容也让不少人感叹小田切让的圈内人缘。参演电影的卡司还包括村上虹郎、苍井优、浅野忠信、永濑正敏、桥爪工等众演员，而这些人或先或后都和小田切有过合作。摄影为杜可风，配乐则是日本音乐大师细野晴臣。

一直被誉为“圈中宅男”的小田切让，此前在接受采访时的一段话，直接在影视圈里投放了一颗炸弹：“我之前的健康检查结果并不好，夸张地说，我要重新考虑自己剩下的时间了，果然还是很想拍电影。”

尽管让叔语气平和，表示并不很严重，但不管是粉丝，还是相关影视人，对此还是震惊了一番，纷纷担忧地表示，“让叔，请保重身体啊！”

《热带雨》

入围单元：卧虎

导演 / 编剧: 陈哲艺

主演: 许家乐 / 杨雁雁

类型: 剧情

制片国家/地区: 新加坡 / 中国台湾

在今年的多伦多电影节上，新加坡导演陈哲艺执导的最新电影《热带雨》，是电影节“站台单元”中唯一入围的华语片。虽然没有成为最后赢家，但依然得到了场刊3分以上的好成绩，妥妥的年度华语佳作。

陈哲艺在2013年的金马奖上，以处女作《爸妈不在家》一举拿下最佳影片，还在戛纳电影节斩获了金摄影机奖。以这样的起点再度出发，《热带雨》天然吸引了无数注意力。

《热带雨》的故事依然发生在新加坡。正值雨季，身为中学国文老师的女主人公，生活上也在经历“雨水”冲刷：与丈夫因为无子关系愈发疏冷、照顾瘫痪在床的公公、学生对国文课的不屑一顾、以及她和一位男学生的暧昧关系……

所有这些，随着雨季的到来，让女主人公的心里也越发潮湿。虽然线索很多，但整部电影并不松散显得涣散，为主人公心事的展开提供了更多侧面。观众会看到的是整个华人世界的类似问题，不禁回味自己身边种种。

《蕃薯浇米》

入围单元：藏龙

导演: 叶谦

主演: 归亚蕾 / 杨贵媚 / 班铁翔

类型: 剧情

制片国家/地区: 中国大陆

《蕃薯浇米》去年在平遥拿下发展中电影计划的“无限自在”荣誉，可谓土生土长，今年重回平遥，俨然进入了收获的季节。

电影由归亚蕾和杨贵媚主演，寡妇林秀妹抚养两儿成家后，发现自己变得“无关紧要”；而小她十岁的唯一老姐妹青娥突然离世。给了她巨大触动。已过七旬的她想要再次证明自己价值与存在感。

该片使用全闽南语拍摄，展示泉州特有的地域文化和风土人情。片名“蕃薯浇米”指的是是闽南百姓吃的地瓜稀饭，寓意“生命的真谛就藏在平凡生活之中”。



《追凶十九年》

入围单元：藏龙

导演: 徐翔云

主演: 王泷正 / 宋宁峰 / 黄璐 / 国永振

类型: 剧情 / 悬疑 / 犯罪

制片国家/地区: 中国大陆

一部硬核犯罪动作片，张猛监制，导演徐翔云此前曾经是爆款犯罪网剧《法医秦明》《十宗罪》的编剧，这是他的导演处女作。

1999年起的数年内，一向安宁平静的弘州市先后发生多起针对年轻女性的强奸命案，受害者的皮肤均遭凶手割下奇异的血雏菊形状。警察刘一波和搭档何晨奉命侦查，这场斗智斗勇的追凶之旅，一追就是十九年。

一场十九年的追凶之旅，将会杂糅怎样复杂的情绪和仇恨，着实令人期待。除了文艺片女神黄璐，在网剧领域树立了良好口碑的王泷正、宋宁峰，表现也相当值得期待。

《六欲天》

入围单元：藏龙

导演: 祖峰

主演: 祖峰 / 黄璐 / 陈明昊 / 田雨

类型: 剧情 / 犯罪

制片国家/地区: 中国大陆


今年入围戛纳一种关注单元的《六欲天》，是演员祖峰的导演处女作。剧本原名为《热》，祖峰看到之后深为打动决定自导自演。故事也同样为犯罪题材，关于一次警察查案背后的背叛与救赎。



因女友自杀而患上抑郁症的警察阿斌（祖峰饰），在查案时与同样患上抑郁症的李雪（黄璐饰）相识，彼此怜悯，渐渐亲近。当在城市漂泊的少女婷婷（张倩如饰）也和阿斌产生说不清的牵绊。 他们期待在这个世界找到光明。

该片作为聚焦抑郁症的电影，是现代都市快节奏下人所经历的环境和内心压力变化。人与人之间的情感纠葛，在追寻案件真相的过程中映射个体与社会的关系。

《灼人秘密》

入围单元：首映

导演: 赵德胤

主演: 吴可熙 / 宋芸桦 / 夏于乔

类型: 剧情

制片国家/地区: 中国台湾 / 德国 / 马来西亚

曾经执导《再见瓦城》的赵德胤，是台湾较受关注的新生代导演凭该片于今年首度入围戛纳的电影；而女主角吴可熙已经跟她合作了4部影片，此前还主演过大热的《血观音》，她和赵德胤联名作为本片的编剧。

这是一个乡下女孩到城市追梦的故事。女孩妮娜（吴可熙饰）为了追求演员梦独自北上，虽如愿获得电影《谍恋》演出，却因重重行业怪象几度濒临崩溃。


当她因家中变故不得不回到家乡，她遇上昔日好友Kiki（宋芸桦饰），发生了一连串奇异事件，而这似乎都和当初与她竞争角色后突然消失的女孩有关。


电影据说是以韦恩斯坦案为原型。相比之前的赵德胤，这部明显更戏剧化了，风格也偏心理惊悚片，其中的长镜头使用颇为惊艳，对于娱乐圈的许多呈现讽刺而有力量。

《我失去了身体》

I Lost My Body

入围单元：影展之最

导演: 杰赫米·克拉潘

主演: 维克图瓦尔·杜·布瓦 / 帕特里克·德阿萨姆曹

类型: 剧情 / 动画

制片国家/地区: 法国


今年戛纳影评人周单元大奖得主，这是戛纳推出全球新锐导演的长片处女作或第二部作品的一个单元。这部杰赫米·克拉潘导演的长片处女作，被许多影评人评为今年“戛纳最佳”。

电影呈现出非常奇瑰的想象力。影片分两条线索，一条线颇具cult片气质，讲述一只断手从警察局或者医院的冰箱里逃脱，一路经历千难万险穿越半个巴黎、回到主人身边。

另一条线则是主人的年少回忆、父母车祸去世后，自己在巴黎的生存挣扎和美好而痛苦的初恋故事。

影片的画风简约却有气质，音轨和配乐非常出彩，尤其是很多细节的想象力令人惊叹，导演被认为有着令人惊叹的才华。那只游荡的动画手，情感之丰富真实过大部分真人电影。



《大西洋》

入围单元：影展之最

导演 / 编剧: 玛缇·迪欧普

类型: 剧情

制片国家/地区: 塞内加尔 / 法国 / 比利时


塞内加尔裔法国导演玛缇·迪欧普电影《大西洋》，在今年戛纳拿下象征二奖的评委会大奖，算得上爆冷了，毕竟这不是一部广受关注的作品。但场刊2.8的高分，已经足够说明质量。


该片翻拍自同名短片，讲述的是一个非洲非法移民的故事，有趣的是，电影里使用了极为奇妙的奇幻设定，非洲风情的神秘主义气息非常迷人，甚至被拿来对比《幸福的拉扎罗》。

虽然被认为非洲的元素有些过剩，但电影本身对于阶级和社会正义的探讨有着非常有趣的视角。女主角的情感线索处理得也极为细腻和克制，足够深刻，却也足够好看。




《气球》

入围单元：影展之最

导演：万玛才旦

主演：索朗旺姆 / 金巴 / 杨秀措

类型：剧情

制片国家/地区：中国大陆

万玛才旦的《气球》入围了今年威尼斯电影节的一种关注单元，电影被认为像是以往的万玛才旦和《撞死了一只羊》的一次融合。

比较《撞死了一只羊》，《气球》更具有故事性，主人公达杰一家因一只避孕套，卷入了一系列尴尬而又难以抉择的事件当中。通俗易懂的同时又不失轮回和梦境等超现实笔触，让整个故事都显得更有灵性。

据说在威尼斯，电影让不少女性观众都流泪了，看来故事倒是颇具普世性。万玛才旦的作品，当然期待。


以上这份片单，虽然部部都优中选优，还是不可能把今年平遥的好作品说完。首批公布的这些片单，几乎每一部的介绍都让人心痒痒的。

包括今年的 “回顾”单元，以“印度新电影”为主题推出的十部印度片，喜欢的观众也绝对不容错过。

印度电影史的里程碑《云遮星》(1960)
