《道路工程》第七章路基路面现场测试第一二节知识点

2020-10-29
| 科技

全国公路水运试验检测工程师——道路工程 第七章 路基路面现场测试 第一二节知识点分享！

第一节 几何尺寸

1. 路基路面现场测试内容主要包括几何尺寸、压实度、平整度、弯沉值、抗滑性等方面。

2. 纵断高程和横坡一般用水准仪检测。

3. 中线偏位用经纬仪检测，宽度和边坡可用尺量。

4. 路面厚度可采用挖坑法或钻芯取样进行检测，也可以采用短脉冲雷达进行无损检测。

5. 路面结构层厚度的检测一般与压实度同时进行，当用灌砂法进行压实度检测时，可通过率取试坑的深度，从而得到结构层的厚度；当用钻芯取样法检测压实度时，可直接取芯样的高度。

1.1路面钻芯取样方法

1. 钻孔采取芯样的直径不宜小于最大集料粒径的3倍。

2. 准备工作：确定路段、按随机选点方法确定取样位置、将取样位置清扫干净。

3. 由于试验需要不能用水冷却时，应采用干钻孔，此时为保护钻头，可先用干冰约3kg放在取样位置上，冷却路面约1h，钻孔时通以CO2等冷却气体以代替冷却水。

1.2路基路面几何尺寸测试方法

1. 对路基路面各部分的宽度、纵断面高程、横坡及中线偏位等几何尺寸进行检测，可供道路施工过程、交竣工验收及旧路调查使用。

2. 路基路面宽度测试步骤：用钢尺沿中心线垂直方向水平量取路面各部分的宽度，对高速公路及一级公路，准确至0.005m；对其他等级公路，准确至0.01m。测量时钢尺应保持水平，不得将尺紧贴路面量取，也不得使用皮尺。

·路基宽度为行车道与路肩宽度之和，当设有中间带、变速车道、爬坡车道、紧急停车带时，尚应包括这些部分的宽度。路面宽度包括行车道、路缘带、变速车道、爬坡车道、硬路肩和紧急停车带的宽度。

3. 纵断面高程测试步骤：将精密水准仪架设在路基路面平顺处调平，将塔尺竖立在中线的测定位置上，以路线附近的水准点高程作为基准。测记测定点的高程读书，准确至0.001m。连续测定全部测点，并与水准点闭合。

4. 中线偏位测试：有中线座标的道路，对高速公路及一级公路，准确至5mm，对其他等级公路，准确至10mm；无中桩座标的低等级道路（应首先恢复交点或转点，实测偏角和距离，然后采用链锯法、切线支距法或偏角法等传统方法敷设道路中线的设计位置，量取设计位置与施工位置之间的距离，即为中线平面偏位△CL，准确至10mm）。

5. 路面横坡测试：路基横坡为跳槽中心线与路槽边缘两点高程差与水平距离的比值，以百分率表示。

1.3挖坑及钻芯法测定路面厚度试验方法

1. 厚度是路面施工质量控制及施工验收的关键项目。

2. 路面各结构层的厚度是影响路面结构强度和使用性能的重要因素。

3. 路面各层施工过程中的厚度检验及工程交工验收检查通常采用挖验或钻取芯样方法量测，但此法会给路面造成一定的损伤。

4. 挖坑或钻孔的填补：挖坑及钻芯法测定路面厚度的试验方法必然会在路面上留下一定面积的坑孔。为了保持路面的完整性，需要及时填补所有坑孔，避免成为路面损坏的隐患。

·1）适当清理坑中残留物，钻孔时留下的积水应用棉纱吸干；2）对无机结合料稳定层及水泥混凝土面板，应按相同配合比用新拌的材料分层填补并用小锤压实，水泥混凝土中宜掺加少量快凝早强剂；3）对无结合料粒料基层，可用挖坑时取出的材料，适当加水拌合后分层填补，并用小锤压实；4）对正在施工的沥青路面，用相同级配的热拌沥青混合料分层填补，并用加热的铁锤或热夯压实，旧路钻孔也可用乳化沥青混合料修补；5）所有补坑结束时，宜比原面层略鼓出少许，用重锤或压路机压实平整。

1.4短脉冲雷达测定路面厚度试验方法

1. 雷达测试路面结构层厚度的基本工作原理：利用雷达波（电磁波）在不同物质界面上的反射信号，识别分界面，通过电磁波的走时和在介质中的波速推算相应介质的厚度。

2. 短脉冲雷达是目前公路行业路面厚度无损检测应用最广泛的雷达，它具有测值精度高、工作温度等特点。

3. 雷达测试系统由承载车、天线、雷达发射接收器和控制系统组成。

4. 雷达发射的电磁波在道路面层传播过程中会逐渐衰减。

5. 雷达最大探测深度是由雷达系统的参数以及路面材料的电磁属性决定的。

6. 为了满足测试准确度和垂直分辨率的要求，用于检测路面厚度的雷达天线频率一般为1.0GHz以上。

7. 天线频率与可探测深度是成反比的。最小分辨层厚不超过40mm。

8. 芯样标定对于数据解析起着重要作用，因为检测过程中仪器仅仅记录了雷达波在结构层上下表面之间的走势，而不是厚度。

9. 为了获得路面厚度，需要知道路面材料的介电常数。

10. 通常采用在路面上钻芯取样方法以获取路面材料的介电常数，或者计算出雷达波在同样材料中的传播速度，推算出层间厚度。

11. 路面材料的介电常数会随集料类型、沥青产地、密度、湿度等而不同。

12. 测试过程中应根据实际情况增加芯样钻取数量，以保证测试厚度的准确性。

13. 本方法适用于新、改建路面工程质量验收和旧路加铺路面设计的厚度调查。

14. 对于材料过度潮湿或饱水以及有高含铁量的矿渣集料的路面不适合用本方法测试。

15. 准备工作：距离标定、安装雷达天线、开机预热、参数设置等。

第二节 压实度

1. 碾压是路基路面施工的重要环节，压实质量与路基路面的强度、刚度、稳定性和平整度密切相关，压实度是路基路面施工质量检验的关键项目。

2. 对于路基土、粒料类基层或底基层、无机结合料稳定类基层或底基层，压实度时筑路材料压实后的干密度与标准最大干密度之比，以百分率表示。

3. 标准最大干密度需要在施工前通过室内重型击实试验或振动压实试验得到。

4. 压实后的最大干密度通常采用挖坑灌砂法或环刀法或核子密度仪法现场检测，其中核子密度仪法是无损检测方法。

5. 对沥青混合料面层，压实度指现场实际达到的密度与标准密度的比值。

6. 标准密度可采用沥青混合料的马歇尔击实法获得的试验室标准密度，也可以采用最大理论密度或试验段密度，不同的标准密度对应不同的压实度要求。

7. 现场实际达到的密度可采用在现场钻芯并测定芯样密度的方式获得，也可以通过现场无核密度仪无损、快速地测定。

8. 马歇尔击实试件和钻取芯样的密度可采用表干法、蜡封法和水中重法测定。

1.1挖坑灌砂法测定压实度试验方法

1. 基本原理：在压实层挖坑取出材料，烘干，获得干质量；利用密度确定的砂灌到坑里，以置换坑的体积，从而得到压实材料的干密度，进而计算得到压实度。

2. 本方法适用于在现场测定基层（或底基层）、砂石路面及路基土的各种材料压实层的密度和压实度检测。但不适用填石路堤等有大孔洞或大孔隙材料的压实度检测。

3. 用挖坑灌砂法测定密度和压实度时，应符合下列规定：1）当集料的最大粒径小于13.2mm、测定层的厚度不超过150mm时，宜采用直径100mm的小型灌砂筒测试。2）当集料的最大粒径等于或大于13.2mm，但不大于31.5mm，测定层的厚度不超过200mm时，应用直径150mm的大型灌砂筒测试。

4. 仪器：灌砂仪、试样盘、天平或台秤（称量10-15kg，感量不大于1g）、含水率测定器具、量砂、盛砂的容器、其他（凿子、螺丝刀、铁锤、长把勺、长把小簸箕、毛刷等）。

5. 注意：1）必须保证标准最大干密度的试验材料与现场压实层填料是同种材料，计算得到的压实度才有意义。2）量砂要规则。3）每一次量砂，都必须测定松方密度，量砂筒下部圆锥体内砂的数量也应该每次重新标定。因此量砂宜事先准备较多数量。4）地表面处理要平整。5）在挖坑时试坑周壁应竖直，避免出现上大下小或上小下大的情形，这样就会使检测密度偏大或偏小。6）压实度应某测试层整体厚度范围内的压实质量，因此，挖坑深度应等于测试层厚度，即应达到该测试层的层底。

1.2核子密度湿度仪测定压实度试验方法

1. 核子密度湿度检测仪是利用同位素（伽马源和中子源）的放射原理，在施工现场快速地检测路基土和路面材料的密度和含水率的电子仪器。

2. 本方法适用于现场用核子密度湿度仪以散射法或直接透射法测定路基或路面材料的密度和含水率，并计算施工压实度。

3. 本方法用于测定沥青混合料面层的压实密度或硬化混凝土等难以打孔材料的密度时，宜使用散射法；用于测定土基、基层材料或非硬化水泥混凝土等可以打孔材料的密度及含水率时，应使用直接透射法。

4. 在表面用散射法测定时，所测定沥青面层的层厚应根据仪器的性能决定最大厚度。

5. 用于测定土基或基层材料的压实密度及含水率时，打洞后用直接透射法所测定层的厚度不宜大于30cm。

6. 本方法属非破坏性检测，允许对同一个测试位置进行重复测试，并监测密度和压实度的变化，以确定合适的碾压方法，达到所要求的的压实度。

7. 仪器的标定：标定过程就是将仪器在一系列密度和湿度已知的标准材料块（标定块）上进行检测，在每一标定块上，在每一个检测深度上确立标准密度值和湿度值与射线计数率之间的对应关系。一般情况下，每12个月以内要对核子密度湿度仪进行一次标定。

8. 标定后的仪器在测量标准材料块上的密度时，其检测结果与标准密度值的误差不应超过±16kg/m³。

9. 注意事项：放射性物质对人体有害，应注意安全防护；有些因素会影响核子密度湿度仪的测试结果，应注意避免干扰。

1.3环刀法测定压实度试验方法

1. 原理：在测试层内通过带刃环刀取出固体体积的土样，烘干，获得干质量，从而得到压实材料的干密度，进而计算得到压实度。环刀容积通常为200cm³，环刀高度约5cm。

2. 本方法适用于测定细粒土及无机结合料稳定细粒土的密度。

3. 对无机结合料稳定细粒土，其龄期不宜超过2d，且宜用于施工过程中的压实度检验。

4. 主要仪具有人工取土器或电动取土器、天平等。

1.4钻芯法测定沥青面层压实度试验方法

1. 沥青混合料面层的压实度是按施工规范规定的方法测定的混合料试样的毛体积密度与标准密度之比值，以百分率表示。

2. 本方法适用于检验从压实的沥青路面上钻取的沥青混合料芯样试件的密度，以评定沥青面层的施工压实度。

1.5无核密度仪测定压实度试验方法

1. 无核密度仪利用发射的电磁波在材料中的能量吸收和损耗检测材料的密度。

2. 材料对电磁波吸收能量和损耗取决于材料的介电常数。

3. 无核密度仪测定压实度的方法是间接地、无损的。

4. 本方法适用于现场无核密度仪快速测定沥青路面各层沥青混合料的密度，并计算施工压实度，但测定结果不宜用于评定或仲裁。

5. 主要仪具是无核密度仪和标准密度块。探头、探测深度≥4.0cm、测量时间1s、精度0.003/cm³、操作环境温度0-70℃、测试材料表面最高温度150℃、湿度98%且不结露。
