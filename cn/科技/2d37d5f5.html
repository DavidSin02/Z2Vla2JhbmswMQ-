七款常见手机输入法测评

2020-11-14
| 科技

手机输入法的好用与否可能有时候会对我们使用手机带来不同的体验。

今天想给大家来一个七款常见手机输入法的简短测评，希望可以帮助朋友找到属于自己的输入法。

百度输入法

鼎鼎大名的百毒开发出来的输入法

分类：全拼输入法/双拼输入法/手写输入法/笔划输入法

优点：

  1. 词库较为丰富

  2. 皮肤自定义性高，预制皮肤多

  3. 联想方面做得还可以

  4. 弹窗算是国产比较好的

缺点：

  1. 安装包过于臃肿，有很多多余的功能

  2. 有较多多的广告

  3. 纠错不是很好

  4. 动画效果有点生硬

  5. 请求了过多多余的权限

讯飞输入法

这款输入法应该是国产输入法里面评价最好的吧，老罗的发布会火了一款输入法。

分类：全拼输入法/双拼输入法/手写输入法/笔划输入法

优点：

  1. 语音输入非常好用，现在更是加如了多种方言。

  2. 键盘手写

  3. 表情包非常丰富

  4. 可以更改各家键盘的布局

缺点：

  1. 主页不美观

  2. 滑动输入木有了

  3. 纠错智能性一般

谷歌拼音输入法

谷歌大法就是好，这句话也不是没有道理。相信很多人都用过这款输入法，简约不臃肿。

优点：

  1. 纯净，无广告

  2. 滑动输入功能强大

  3. 外观简洁

  4. 纠错功能强大

  5. 较为全面的支持了多种双拼方案

  6. 手写识别较强

缺点：

  1. 国内无云词库

  2. 国内无语音输入

  3. emoji和颜文字不支持自定义，无剪切板

  4. 预制皮肤较少

QQ输入法

分类：全拼输入法/双拼输入法/笔划输入法/五笔输入法/手写输入法

优点：

  1. 浮空键盘

  2. 皮肤非常全面，而且预制皮肤多

  3. 符号部分分类很好

  4. 滑动键盘移动光标的功能不错

  5. 云联想也可以

缺点：

  1. 纠错一般

  2. 依然不支持滑动

  3. 安装包还是大了点

  4. 符号界面存在一定问题，会导致有的符号变形

手心输入法

一款不错的国产输入法

分类：全拼输入法/双拼输入法/手写输入法/笔画输入法/五笔输入法

优点：

  1. 有较强的皮肤自定义功能

  2. 纯净无广告

  3. 纠错功能较为强大

  4. 云同步词库

  5. 剪贴板功能较为强大

缺点：

  1. 不支持按键全时弹窗

  2. 不支持单手模式

  3. 动画效果略微生硬

搜狗输入法

可能说是用户最多的手机输入法吧

分类：全拼输入法/双拼输入法/手写输入法/笔划输入法

优点：

  1. 词库功能非常强大

  2. 拥有较为全面的皮肤

  3. 手感做的还算不错

  4. 拥有较好的语音输入功能

缺点：

  1. 过于臃肿，安装包体积过大

  2. 有广告

  3. 请求了过多不合理的权限

  4. 较为容易卡顿

必应输入法

微软开发的输入法

分类：全拼输入法/双拼输入法

优点：

  1. 滑动输入较为强大

  2. 纠错功能较为强大

  3. 有较强的光标移动控制器功能

  4. 纯净无广告

缺点：

  1. 容易卡顿

  2. 键盘高度不支持自定义

  3. 不包含手写和笔划输入

  4. 键盘可自定义性极差

  5. 不支持单手模式

今天的测评就到这里，朋友们可以在评论区留言，大家一起讨论。
