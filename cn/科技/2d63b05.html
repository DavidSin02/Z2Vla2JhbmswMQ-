作为EMUI10的亲兄弟，谁说Magic UI 3.0没有游戏小窗口？

2020-10-29
| 科技

荣耀V30 Pro升级Magic UI 3.0以后，真真体验到了智慧生活的便捷，比如多屏协同，全屏滑动翻译等功能，大大方便了小编的生活，但Magic UI 3.0还有些隐藏的功能。

虽说官方没有宣传，但并不代表手机没有，比如说，升级到EMUI10或者Magic UI 3.0的用户，都应该用上了游戏空间，对于它增加的屏蔽指关节，以及优化外设连接，都是明面上的功能。
很多喜欢手游的用户，都希望有聊天窗口能悬浮在游戏界面的功能，这样即能不错过游戏，又能不错过信息。

但不可否认，很多已经使用EMUI10或者Magic UI 3.0系统的用户，到现在的都没发现有这个功能，其实，这个功能已经添加，只是你没发现而已。
假如现在你玩游戏，并且没有开启防打扰，那么QQ消息弹出来，就会显示上图这种图像，这里其实末尾多了一个像窗口的东西。这个时候，你点击一下，QQ消息窗口就会悬浮在游戏界面，如下图显示一般：

当然，开始弹出的QQ消息大概占据了游戏大半个屏幕，这样肯定会有点大是不是，没关系，EMUI10以及Magic UI 3.0系统，也给我们提供了压缩窗口变小的功能。比如，你窗口在左边，那么按住窗口右侧边缘，推向左侧即可缩小窗口。

你也许不想把聊天窗口放在边缘，想按这个大小居中显示，那么调整好大小之后，按住窗口中心，移动即可，非常方便。

以上这些其实都不是最重要的功能，毕竟，你聊天窗口悬浮，是用来聊天的，必定会输入文字，所以，当你点击输入框时，也会如下图所示，弹出输入法：

需求更多的花粉，肯定还会关注这样悬浮后，如果关闭窗口，或者我想跳转到聊天界面，不想悬浮聊天了。

很简单，悬浮的窗口，不管是QQ，亦或是微信，右上角都有两个按钮，一个正方形的用于跳转到聊天界面，一个用于关闭悬浮窗口。

虽然说，官方从来没有宣传过这种悬浮窗口，但功能已经应用到EMUI10或者Magic UI 3.0，但有人要说连这都没有，恐怕又要被啪啪打脸了，毕竟华为隐藏的技术太多，稍不注意，可能掉入陷阱哦！
