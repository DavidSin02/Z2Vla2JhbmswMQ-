纸箱厂心态要崩！停机潮提前

2020-11-14
| 科技

理文提前发布春节停机计划，

2020年第一波涨价潮预定？

三级厂心态要崩！

//

停机台数多，发布时间早，理文CEO李文斌签发2020春节停机计划

//

11月5日，理文造纸CEO李文斌签发了2020年春节理文集团停机计划表：从2020年1月12日起到1月31日止，在春节前后，理文国内外六大生产基地共有14台纸机将错时停机，停机时间在6-20天不等（详情请点击：【头条】理文六大基地14台纸机陆续停机，纸价再涨100元！几乎全国纸厂都在涨）。

理文停机计划会不会发布太早了？

有业内人士提出疑问，离2019年结束还有将近两个月，理文在这时候公布春节停机计划，会不会太早了？

对比2019年的停机计划，理文造纸今年发布停机计划的时间很有意思，2018年理文公布停机计划的时间是11月29日（玖龙是11月18日），即使把今年春节提前的时间算上，理文此举也显得过于着急。

另一方面，今年停机台数也比去年多，2020年停机台数高达14台纸机，其中单广东理文就有5台纸机停机：一台停机长达20天，三台停机15天，只有一台停机较短，停机6天。

//

理文如此早、大规模的停机计划或引起突发性暴涨？纸箱厂心态要崩

//

新一年涨价已预订？

而对于三级厂来说最大的疑虑是，广东理文如此早、如此大规模的停机计划，会不会提前烘托出原纸库存不足的气氛？更严重的是，会不会引起突发性的暴涨？这种突发性的涨价，最受伤害的可能就是夹在造纸厂和终端客户间的三级厂了。

值得一提的是，2018年，由于玖龙理文过早宣布停机计划，有媒体评价认为此举透露出了纸业环境的寒冷，那么今年又是什么情况？冰川时代吗？（瑟瑟发抖）

或许正是如此吧。

有数据显示，截至2019年10月，全国箱板瓦楞纸总产量不足，3100万吨，比去年同期减少39万吨；总销量不足3200万吨，同比跌幅超过3.5个百分点。

而形成对比的是库存天数，今年10月箱板瓦楞纸平均库存天数为11天，对比2018年的库存天数是8天，三个指标的表现均差于2018年同期。

基于这些数据，容小编以小人之心度君子之腹一下：这次的停机计划有没有可能人为制造一起短暂的原纸缺货？

当然，这得看其他造纸厂肯不肯跟，如果其他纸企大力度跟停，那么在纸企恢复开机到库存重新建立的空窗期内，可能真的就会造成原纸短缺，继而进一步引起阶段性的大规模涨价吧。

原纸一涨，纸板是不是也会涨？ 2020年新春开工，“全国人民”就要“喜迎一波涨价潮”，估计心态都要崩了吧…

目前来看，其它规模纸厂并没有公布相应春节停机计划，但考虑到规模纸企素来有春节停机计划的传统……我们持续关注。
