各大电影节青睐的佳片，每一部都不容错过

2020-10-29
| 科技

又到了一年一度的颁奖季，除了欧洲“三大电影节”的精选，还有许多年底的颁奖大热门都将陆续登场。凤凰卫视电影台特别精选了近年入围各大电影节甚至获奖的经典佳作，不仅有意大利国宝级女神莫妮卡·贝鲁奇、鬼才导演昆汀·塔伦蒂诺的惊世神作，下面这些好片也不容错过！

01

西西里岛的美丽传说



导演: 朱塞佩·托纳多雷

主演: 莫妮卡·贝鲁奇 / 朱塞佩·苏尔法罗

类型: 剧情 / 战争 / 爱情

地区: 意大利 / 美国

播映日期: 10月27日 21:15 / 10月28日 16:00




该片获得柏林国际电影节金熊奖最佳影片提名，讲述了二战时期意大利西西里岛上的美丽少妇玛莲娜的故事。1941年春末，13岁的少年雷纳多初次见到玛莲娜。她的美丽让男人心醉、女人羡妒，彻底征服了西西里岛。雷纳多也对玛莲娜情难自已，他不仅跟其他青少年们一起搜寻玛莲娜的诱人丰姿与万种风情，还悄悄地成为她不知情的小跟班，如影随形地窥视她的生活。她摇曳的倩影，她聆听的音乐，她贴身的衣物都成为这个被荷尔蒙淹没的少年，最真实、最美好的情欲幻想。然而，透过雷纳多的眼，人们也看到了玛莲娜掉进了越来越黑暗的深渊之中。

02

非常背叛


导演: 保罗·马祖斯基

主演: 雪儿 / 查兹·帕尔明特瑞 / 里安·奥尼尔

类型: 剧情 / 喜剧

地区: 美国

播映日期: 10月2日 21:15 / 10月3日 18:00




结婚二十年来，玛格丽特因为婚姻不幸福与自己的助理外遇了。发现自己妻子外遇的杰克雇用的杀手东尼闯进玛格丽特家将她全身捆绑，并斥责她不该背叛丈夫。而后两人竟不知不觉地谈论起“忠贞”的意义，后续情势发展更趋古怪。杀手东尼竟被玛格丽特迷住并陷入荒淫之中，究竟是谁在勾引谁？东尼最终又是否能完成任务？这一段古怪脱序的谋杀情事，最后会如何完结呢？

03

四个房间



导演: 阿利森·安德斯 / 亚历山大·洛克威尔 / 罗伯特·罗德里格兹 / 昆汀·塔伦蒂诺

主演: 安东尼奥·班德拉斯 / 萨尔玛·海耶克 / 麦当娜 / 昆汀·塔伦蒂诺 / 布鲁斯·威利斯

类型: 喜剧

地区: 美国

播映日期: 10月31日 21:15




影片讲述了酒店服务生特德服务的四个房间里发生的一系列怪事：一群女巫准备在当晚复活他们的女神，而她们却弄丢了至关重要的精液，小侍应生临时客串，勇敢献身；善妒的丈夫怀疑妻子与他人有染，遂将妻子绑在椅子上，倒霉的侍应生则被当作奸夫，手枪抵头，命悬一线；富翁夫妻参加新年宴会，将子女留在房间中。两个淘气的小家伙展开了一场大战，却看到了恐怖的一幕；几名好莱坞演员凑在房间中玩打火机游戏，每人打十下，十下连续打着就赢得一部跑车，反之便要切掉一根手指。侍应生被临时找来充当刽子手。


04

危险关系



导演: 昆汀·塔伦蒂诺

主演: 帕姆·格里尔 / 塞缪尔·杰克逊 / 罗伯特·福斯特

类型: 剧情 / 惊悚 / 犯罪

地区: 美国

播映日期: 10月25日 11:20 / 10月26日 04:10




该片改编自美国小说家埃尔穆尔·伦纳德的小说《神奇之力》，获得柏林国际电影节金熊奖最佳影片提名，塞缪尔·杰克逊凭借此片获得银熊奖最佳男演员。影片讲述了空中小姐杰奇暗地里一直利用职务之便，帮助军火商奥德尔运送赃款，并从中获取暴利。一次意外中，杰奇的“生意”曝光了，她本人亦遭到逮捕。奥德尔知道后派马克斯杀人灭口。让奥德尔没有想到的是，马克斯竟然爱上了杰奇，两人一拍即合，策划了一场阴谋将奥德尔耍得团团转。当奥德尔发现自己受骗时，事态的发展早已经脱离了他的掌控。

05

寻找费里尼


导演: 塔伦·莱克斯顿

主演: 玛丽亚·贝罗 / 克塞尼亚·索罗

类型: 剧情 / 冒险

地区: 美国 / 意大利

播映日期: 11月13日 21:15 / 11月14日 15:25


本片改编自替卡通《辛普森一家》巴特·辛普森配音的美国知名配音演员南希·卡特赖特年轻时的寻梦故事。影片讲述了露西从小在母亲的保护下长大，母女俩最大的休闲娱乐是一起看通俗感人的爱情电影。在某次失败的工作面试后，露西走进电影院，费里尼的电影正在播映。费里尼的电影与过去她的世界截然不同，一个她未曾感受过的世界充满奇想与欲望。于是，她决定离开美国，前往意大利，踏上寻找费里尼的旅程……

06

嗅探员



导演: 阿穆尔·古普特

主演: 帕什瓦·达锐沃 / 库什米特·吉尔

类型: 喜剧 / 家庭

地区: 印度

播映日期: 11月14日 21:15 / 11月15日 17:45




影片讲述了八岁的小男孩桑尼因为天生没有嗅觉而感到非常落寞和自卑。在一个平常的上学日，桑尼在探索生物教室的过程中，意外打翻了神秘的药水。吸入药水释放的气体之后，桑尼意外获得了异常灵敏的嗅觉。他不但可以闻出味道，而且能闻出人们的情绪以及感情，比如悲伤，喜悦等；更能分辨出人们刚刚去过哪里，将要去哪里。桑尼凭借比狗更灵敏的嗅觉，成为了小镇上的民间侦探。这时小区里出现了一个偷车贼，桑尼帮助大家找寻失踪的汽车，没想到却揭开了一件更大的盗车案。

07

杰出公民


导演: 加斯顿·杜帕拉特 / 马里亚诺·寇恩

主演: 奥斯卡·马丁内兹 / 达迪·布列瓦

类型: 剧情 / 喜剧

地区: 阿根廷 / 西班牙

播映日期: 11月24日 21:15 / 11月25日 15:35



本片获得威尼斯国际电影节主竞赛单元金狮奖提名，主角奥斯卡·马丁内兹凭借此片荣获沃尔皮杯最佳男演员。影片讲述了小说家丹尼尔荣获诺贝尔文学奖后，登上了事业的顶峰。成功给他带来了巨大的财富，却让他担忧名声和财富会夺走他文学创作上的灵感与锋芒。五年后，这位定居巴塞罗那的作家依然炙手可热。他让助手礼貌地回绝一堆盛情邀请，直到收到了一封来自家乡萨拉斯的信。丹尼尔所有创作的灵感全部来自这片故土，却已经四十年没有回乡。令人意外的是，他接受了小镇的邀请，踏上前往接受“杰出公民”奖的旅途。


08

脆弱地带


导演: 罗宾·史威考德

主演: 布莱恩·科兰斯顿 / 詹妮弗·加纳 / 比佛莉·德安姬罗

类型: 剧情

地区: 美国

播映日期: 11月8日 11:50 / 11月9日 06:20




在曼哈顿工作的律师霍华德拥有美丽的妻子和一对可爱的双胞胎女儿。他们住在郊外的独栋别墅，过着美国传统精英阶层的典型生活。然而霍华德厌烦了这样朝九晚五的生活。直到某天，下班回家的火车突然出轨，也诱发了霍华德开启他的脱轨人生。于是，他决定躲在家对面的阁楼，给家人造成消失的假像。可是他不知这个作死的行为将带给他严重的后果。


09

希望的另一面


导演: 阿基·考里斯马基

主演: 韦勒·维坦恩 / 卡蒂·奥廷宁

类型: 剧情 / 喜剧

地区: 芬兰 / 德国

播映日期: 12月6日 12:40 / 12月7日 07:25


该片获得柏林国际电影节金熊奖最佳影片提名，导演阿基·考里斯马基凭借此片获得银熊奖最佳导演。影片讲述了叙利亚难民哈立德偷渡到芬兰赫尔辛基请求庇护的故事。因家乡战乱、房屋被炸，叙利亚人哈立德逃难到其他国家，途中遭遇驱逐、殴打，还和妹妹走散。但移民中心并没有打算收留他，下令将他遣送回国。哈立德不得不逃出警察视线，开始了在芬兰东躲西藏的生活。


10

狂鼠



导演: 约瑟夫·哈德

主演: 约瑟夫·哈德 / 皮娅·希尔泽伽 / 耶尔格·哈特曼

类型: 剧情 / 喜剧 / 犯罪

地区: 奥地利 / 德国

播映日期: 12月27日 12:55 / 12月28日 07:45


本片荣获柏林国际电影金熊奖最佳影片提名。影片讲述了维也纳一家报社的乐评编辑格奥尔格对工作一向游刃有余，受到同事们的爱戴和敬仰。可是报社突然裁员，格奥尔格被解雇了。格奥尔格决定向他的前上司复仇。起初他只是肆意破坏他的私人物品，之后愈演愈烈。有一天，格奥尔格遇到了他儿时的同学埃里希。埃里希曾经是学校的狠角色，现在也是无业游民。两人一拍即合，决定修复维也纳普拉特主题公园内年久失修的过山车“狂鼠”。

11

重返蒙托克




导演: 沃尔克·施隆多夫

主演: 斯特兰·斯卡斯加德 / 尼娜·霍斯

类型: 剧情

地区: 德国 / 爱尔兰 / 法国

播映日期: 12月11日 21:15 / 12月12日 17:50


该片荣获柏林国际电影节提名，讲述了作家马克思·左恩来到纽约参加自己新书的发布会。书中描写了一段17年前在这座城市发生和终结的感情。机缘巧合，他遇到了这位被他写进书里的女人丽贝卡。她来自东德，如今已是一位成功的律师，在纽约生活了20年。二人决定在一起度一个周末。冬天长岛尽头蒙托克的海滩上，两把躺椅架在寒风中，两个在时光里错过的人再度回到这里，带着无尽的希望和失意。

12

两张彩票


导演: 保罗·内戈埃斯库

主演: 多里安·波古策 / 德拉戈什·布库尔

类型: 剧情 / 喜剧

地区: 罗马尼亚

播映日期: 12月8日 21:15 / 12月9日 18:10


该片获得北京国际电影节天坛奖提名，讲述了一个小城镇的三名男子寻找遗失的彩票的过程中发生的有趣故事。迪奈勒是一名修车工人，幸运之神似乎从未光顾过他。客户总是对他的工作不满意，他穷的连酒都买不起，妻子也因此离开了他。然而，有一天，迪奈勒买的彩票意外中了大奖，这让他重新燃起了生活的希望。但他似乎高兴过了头，弄丢了那张彩票。于是迪奈勒和他的朋友思莱、帕皮里优一起踏上了寻找彩票的旅程。
