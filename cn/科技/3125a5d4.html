武汉上周土拍成交12宗地，蔡甸区4宗地总溢价率高达8.45%

2020-11-14
| 科技

5月15日-5月21日成交的12宗土地可谓相当有看点，现场挂牌成交10宗地，网上挂牌成交2宗地；其中蔡甸区4宗，东西湖区3宗，武汉经济技术开发区2宗，江岸区1宗，洪山区1宗，汉阳区1宗；总成交面积66.92公顷，总成交价703232万元。其中有3宗地有溢价，其余9宗地均以底价成交。P(2019)53溢价率2.78%，P(2019)54溢价率49.59%，这两宗地均为加油站用地被中国石化拍得；P(2019)56溢价率3.71%，被自然人唐涛拍得。下面我们来具体看看成交情况。

P(2019)45号地块成交信息

地块信息：武汉经济技术开发区165R2地块

出让方式：网上挂牌

起拍价：11252万元，成交价：11252万元

成交人：湖北省工业建筑集团有限公司


地块要求：

竞买人须承诺本宗地用于规划建设企业总部或区域总部，开发自持比例不低于总建筑面积的40%，即32683平方米，自持部分在项目竣工验收后10年内不得整体或分割转让、销售；

竞买人须承诺建设不少于1家（含1家）中国合格评定国家认可委员会评定认可的检测实验室，建筑面积不少于2000平方米；

竞买人须承诺在项目建成后引入院士工作站，搭建科技孵化平台。

P(2019)46号地块成交信息

地块信息：江岸区花桥村K2地块北侧

出让方式：网上挂牌

起拍价：7386万元，成交价：7386万元

成交人：武汉格律诗房地产开发有限公司

地块要求：

该地块应与花桥村K2地块统一规划，控制花桥村K2地块项目住宅建筑高度不大于140米；配建大学生租赁房；应采取装配式建筑建造。

P(2019)47号地块成交信息

地块信息： 武汉经济技术开发区10C36、11C36地块

出让方式： 现场挂牌

起拍价： 178444万元，成交价：178444万元

成交人： 中交城乡开发建设有限公司


地块要求：

竞得人对商务金融用地及商服用地的开发自持比率不低于总建筑面积的50%，自持部分在项目竣工验收后10年内不得整体或分割转让、销售，其中商务金融用地自持建筑面积不小于80000平方米，商服用地自持建筑面积不小于81448平方米（以上建筑面积均以相关职能部门批准的为准）。

竞买人须承诺在商务金融用地中引入一家注册资本金不低于伍拾亿元人民币的企业集团总部，该企业为国有资本投资试点企业。

竞得人须在C地块内配建1所6个班幼儿园。

P(2019)48号地块成交信息

地块信息： 东西湖区柏泉街张柏路以东、外环线以南

出让方式： 现场挂牌

起拍价： 90580万元，成交价：90580万元

成交人： 武汉原绿创变置业有限公司

地块要求：

竞得人须向东西湖区引进一家投资总额不低于400亿元的工业项目。

该项目是省级重点建设工程的配套项目，意向竞买人在竞买报名前，需向东西湖区商务局咨询相关政策，充分了解项目基本情况及后续建设要求。

地块内中小学由竞得人建设，建成后其用地及产权无偿移交给东西湖区教育局。

居住地块内需要布局9班幼儿园一座，用地规模为3780平方米，建筑面积为2730平方米。

P(2019)50号地块成交信息

地块信息： 洪山区狮子山街王家湾

出让方式： 现场挂牌

起拍价： 4710万元，成交价：4710万元

成交人： 湖北润源房地产开发有限公司

地块要求：

该地块竞得人须引进一家国内省级或以上畜牧科研单位，并将建筑面积不少于10000平方米的建筑用于该单位的科研楼。

该地块需与周边地块统一规划，建筑设计需满足畜牧科研工作的需求。

P(2019)51号地块成交信息

地块信息： 汉阳区江堤中路与下马湖路交汇处

出让方式： 现场挂牌

起拍价： 11240万元，成交价：11240万元

成交人： 湖北青莲饮食服务有限公司

地块要求： 地块内配建一处标准化菜市场，建筑面积不少于1500平方米。

P(2019)53号地块成交信息

地块信息： 蔡甸区奓山街三一八国道与双丰四路交汇处以东（奓山街奓山村）

出让方式： 现场挂牌

起拍价： 3600万元，成交价：3700万元，溢价率：2.78%

成交人： 中国石化销售股份有限公司湖北武汉石油分公司

P(2019)54号地块成交信息

地块信息： 蔡甸区奓山街九康大道与千子山大街交汇处以东（奓山街千山村、边江村）

出让方式： 现场挂牌

起拍价： 3650万元，成交价：5460万元，溢价率：49.59%

成交人： 中国石化销售股份有限公司湖北武汉石油分公司

地块要求：53、54两宗地均只能建设加油加气站配套设施。

P(2019)55号地块成交信息

地块信息： 蔡甸区蔡甸街莲花湖大道与茂兴路交汇处以西(蔡甸街孙家畈村）

出让方式： 现场挂牌

起拍价： 18690万元，成交价：18690万元

成交人： 武汉钰源置业有限公司

地块要求：

该地块由蔡甸区房屋征收管理办公室回购所有居住计容面积，即：还建建筑面积控制住124570平方米以内（也就是地块所有建面全部回购），回购价为人民币5600元/平方米。

该项目建设的产权调换房交付时间不晚于国有建设合用地使用权出让同签订之日起18个月。

地块位于蔡甸区的老城板块，区域配套较为成熟，地铁4号线延长线规划2019年底通车，距离本地块仅700m。

P(2019)56号地块成交信息

地块信息： 蔡甸区蔡甸街成功大道与公园路交汇处以东(蔡甸街成功村）

出让方式： 现场挂牌

起拍价： 27385万元，成交价：28400万元，溢价率：3.71%

成交人： 唐涛

地块要求：

竞买人须承诺：该项目建设的产权调换房交付时间不晚于国有建设用地使用权出让合同签订之日起18个月。

该地块由蔡甸区房屋征收管理办公室回购所有居住计容面积，即：还建建筑面积控制住182546平方米以内（含配套服务设施建筑面积），回购价为人民币5600元/平方米；

该地块配建一所9班幼儿园，建筑面积不低于2730平方米；

P(2019)57号地块成交信息

地块信息： 东西湖区临空港大道以东、金山大道以北

出让方式： 现场挂牌

起拍价： 282350万元，成交价：282350万元

成交人： 武汉耀星房地产开发有限责任公司

地块要求：

该地块为龚家湾旧城改造项目一期，竞买人竞得后应参与后期的拆迁退地及安置房建设工作，并与旧改主体（东西湖区径河街道办事处）签订相关协议。

竞得人承诺所建商业建筑面积用于引入：不少于50个餐饮、零售、娱乐、超市、百货等国际品牌入驻；不小于3.6万平方米的五星级酒店，并需由1家全球酒店行业权威媒体美国《HOTELS》杂志会刊公布的全球前30名国际知名酒店管理公司下属运营机构经营管理。由东西湖区商务局监管落实。

竞买人承诺自持商业地块内不少于13.6万平方米商业及配套，自竣工验收之日起满五年，且承诺完成上述要求后才能分割或整体销售。

需布局9班幼儿园2所，每所用地面积约为3780平方米，建筑面积约为2730平方米。

小结： 成交的12宗土地蔡甸区4宗地成最大看点，其中自然人唐涛拿到的P(2019)56 蔡甸街成功村还建房地块。临近蔡甸大街，交通条件较好。周边有状元府、越秀翰悦府等大盘环绕，目前均价在7000元/平方米左右，周边商超以及学校资源也很丰富，且纯住宅地也是不可多得的好地块。
