极致性能、无限扩展，说的就是 ALIENWARE Area-51m

2020-10-29
| 科技

来自数字尾巴

热爱外星人的玩家可能不会对 Area-51m 感到陌生，17 年前的外星人正是凭借着它闯入了游戏笔记本领域，从此开启了游戏笔电的探索之路。如今回过头来看这款游戏本，无论是外形设计还是内部配置，处处都书写着 ALIENWARE 笔电另类、张狂、强悍的风格。

而在今年的 CES 2019 展会上，ALIENWARE 将最新的旗舰游戏本命名为 Area-51m，这也不仅让人浮想联翩：除了浓浓的致敬意味，背后的含义到底是什么？如今，随着 ALIENWARE Area-51m 到达尾巴编辑部，我也希望能从中找到答案。

充满未来感的设计构成了 Area-51m 的第一印象

「未来感」是我第一眼见到 Area-51m 时，脑海中即刻浮现的字眼。不同于此前的扁平设计，Area-51m 从 ALIENWARE 早期的工业设计和科幻文化中汲取灵感，引入了全新的「Legend」设计风格，再加上常用于航天科技领域的镁合金机身，最终造就了 Area-51m 更具现代化、也更具科幻色彩的外观。

就拿机身后方的凸出式底座来说，Area-51m 在此处加入支持 ALIENFX 灯效系统的环形流光灯带，配合上蜂窝状的通风孔，从侧面看的话，和科幻电影中经常出现的飞行推进器颇有几分相似，这也是 Area-51m 的外观设计中我认为最吸睛的一点。

而为了迎合新的设计语言，Area-51m 的 A 面也变得更加简洁，除了标志性的可发光外星人 LOGO 带来的超高辨识度之外，右下角还能看到代表 Area-51m 的 A51 字样，这同样让我想起了科幻电影中各类行动代号。另一方面，前面提到的圆环状灯带可以实现和 LOGO 的联动，你可以在 ALIENWARE COMMAND CENTER 中自定义灯效 ，打造属于你自己的外星人游戏本。

ALIENWARE Area-51m 的 B 面的屏幕由于引入了窄边框的设计，所以带来了不同以往的视觉效果。个人觉得在如今的游戏本领域，窄边框设计虽然越来越普遍了，但如何协调好窄边框和其他的设计就显得尤为重要，像有些笔电为了实现所谓的三边窄边框，不惜将前置摄像头挪到了底部或者采用外置的方式，这样的窄边框对于用户来说到底值不值可能就要打个大大的问号了。

反观 Area-51m 将三边框控制在不影响前置摄像头正常使用的宽度上，即兼顾了美观同时还不影响前置摄像头的正常使用；同时，底部边框也压缩到几乎只留有「ALIENWARE」字样，这也让它的屏占比几乎压缩到了极致，加上屏幕与边框间的衔接过渡非常自然，远远望去，会由衷地发出惊叹声！

要说 Area-51m 变化最小的地方可能就是 C 面了，依旧配备了快捷按键和全键盘，而由于有类肤涂层的覆盖，所以担心游戏过程中手汗打滑的玩家可以安心了，整个 C 面摸起来不算特别细腻，但在镁合金机身的作用下，有些金属独有的冰冷触觉。

同时 Area-51m 的 C 面也贯彻了「灯」的主旨，包括键盘、触控板以及位于顶部的电源 LOGO 二合一键都支持 ALIENFX 情绪同步灯效。熟悉外星人游戏本的玩家可能都知道其灯效的优点是比较柔和，亮度分布也比较均匀，基本不会出现明显的断层，配合上 ALIENWARE COMMAND CENTRE 的话能够实现多种自定义效果，但从我的经验来看，要在百万中色彩中调出好看的搭配的话，还是需要有一定的色彩审美的。所以相比之下，我个人更喜欢统一的纯蓝色、纯黄色、或者纯红色灯效，一来不用费心的搭配色彩，二来白色机身也能够将灯效更好地呈现出来。

便携性方面，像 Area-51m 这类游戏本几乎只能得个及格分，但转念一想，其实 Area-51m 的目标用户可能本来就不会是那些追求极致便携的群体，他们想要的可能更加完整的游戏体验，哪怕牺牲一定的便携性，但 ALIENWARE 其实也给了玩家一定的选择空间。为什么这么说呢？因为需要完全发挥 Area-51m 的全部性能的话，需要两个电源适配器（330W+180W）同时上阵，但你也可以只接入一个体积稍小的 180W 适配器来获得基本的性能，这种情况下完成一些基础的操作完全不是问题。

强悍性能和无限扩展才是 Area-51m 的大杀器

除了以上提到的外观变化，ALIENWARE Area-51m 真正让玩家感到兴奋的地方可能在于发生在内部的「质变」，可以说这是一台从诞生之初就含着金钥匙的游戏本，之所以这么说不仅因为 Area-51m 是市面上第一台搭载 i9-9900K 桌面级 CPU 的游戏本，更是因为它支持完全扩展的特性能够让玩家自主更换 CPU 和 GPU，拒绝处理器和显卡迭代造成的性能落伍。

不同于其他搭载移动级处理器的游戏本，手上的这台 Area-51m 搭载了英特尔 i7-9700K 桌面级处理器，在性能上有着一定的先天优势，例如采用 14 纳米制程、TDP 达 95W 等。和前一代的 i7-8700K 相比，i7-9700K 多了两个核心数，并且基础频率和最高睿频也更高一些，而这种变化对于游戏体验来说也更加有利，从 CINBENCH R15 的多次跑分来看，ALIENWARE Area-51m 第一次、第五次和第十次的数据差距不大，这也在一定程度上说明整机运行的稳定性：

  • 多核 1459，单核 205（第一次）
  • 多核 1426，单核 202（第五次）
  • 多核 1416，单核 199（第十次）

当然了，作为一颗带着 K 后缀的桌面处理器，i7-9700K 允许玩家对其进行超频，经过 ALIENWARE COMMAND CENTER 的超频测试之后，ALIENWARE Area-51m 大致能够维持在 5.0GHz 左右，而此时随着 CPU 温度的提升，机身风扇的转动也慢慢快了起来。但让我感到满意的是，即使是在风扇全速运转的情况下，ALIENWARE Area-51m 机身的噪音依旧保持在一个很小的范围，这对于那些像我一样，打游戏不喜欢戴耳机的群体来说十分地友好。

另一方面，手上的这台 ALIENWARE Area-51m 还搭载了 RTX 2080 显卡，新一代 R 卡除了带来比桌面版 GTX 1080 更强的性能之外，最大的亮点可能就是新加入了实时光线追踪 RTX 和 DLSS 深度学习超级采样技术。从 3DMark 的跑分来看，ALIENWARE Area-51m 上的 RTX 2080 的表现还算不错，总分和桌面级 RTX 2080 咬得很紧：

而在游戏实战中，我也试着用 ALIENWARE Area-51m 跑了几款支持实时光线追踪和 DLSS 的游戏，具体帧数如下：

▎《战地 Ⅴ》

  • 开启 DXR，平均帧数为 94
  • 开启 DXR、DLSS，平均帧数为 119
  • 关闭 DXR、DLSS，平均帧数为 134

▎《地铁：离去》（Extreme 画质下）

  • 关闭 RTX，平均帧数为 124
  • 光追效果高，关闭 DLSS，平均帧数为 87
  • 光追效果最高，关闭 DLSS，平均帧数为 81
  • 光追效果最高，开启 DLSS，平均帧数为 104

▎《Apex 英雄》

  • 默认分辨率、最高画质下且关闭垂直同步，平均帧数为 198

可能是为了缓解机身散热的负担，这一次 ALIENWARE Area-51m 的屏幕支持 1080P 分辨率，但刷新率提高到了 144Hz，并同时支持 G-Sync 技术，从上面测试的结果来看，这种改变带来的变化是十分显著：一方面，如果同时支持高分辨率和高刷新率，那么对于显卡的压力将十分巨大，而牵一发动全身地也会对机身的散热提出更高的要求。

但降低一定的分辨率并提高玩家感知更加明显的刷新率将明显降低显卡的压力；另一方面，在 ALIENWARE Cryo-Tech v2.0 散热技术的加持下， ALIENWARE Area-51m 通过双进气口和双排气口通风、高压驱动风扇、铜质散热翅片组、导热管等散热设计，进一步带出机身内的热量，实测在玩家经常接触的 WASD 区域，温度仅在 45℃ 左右，而掌托的位置由于金属的特性，甚至能够感知到冰凉的触觉。

最后，别忘了 ALIENWARE Area-51m 还是一台支持完全自主升级的游戏本，通过戴尔的 DGFF 技术，ALIENWARE Area-51m 将双芯片与主板独立开来，支持 CPU 和 GPU 的自定义升级，例如你可以自行更换至其他支持 Z390 芯片组的 CPU，也可以通过板载显卡模块更换功能来升级 GPU，而这也意味着你的游戏本将不再和「落伍」挂钩，通过更换最新的 CPU 和显卡，你的 Area-51m 将具备与时俱进的性能，这也解决了游戏本最大的槽点——不够完善的扩展性。

整合一体机，一直是笔记本既定的方式，其实是一定程度上保障了其稳定性，而以往我们在绝大多数品牌产品上最多也只能是「升级内存」「加装硬盘」，为了防止人为拆解，保修政策依然还是自主拆机之后不再享受官方的保修政策。而在ALIENWARE Area-51m 见到这样连CPU和GPU均可更换的理念，确实很大胆很有前瞻性，让你真正意义上实现自由DIY笔记本。

当然，还少不了这些秀肌肉的「黑科技」

之前在体验 ALIENWARE 17R5 和 ALIENWARE m15 时，我就对外星人自带的 ALIENWARE COMMAND CENTER 积累了不少好感，主要是因为这个软件无论是界面的 UI 设计还是丰富的功能都做得十分精致，特别是针对超频部分的功能，以往玩家要对电脑进行超频的话，需要跑到 BIOS 里面设置，对于不懂基本知识的玩家来说难免存在过高的门槛。而 ALIENWARE COMMAND CENTER 通过可视化的 UI 和简易的表现方式，将复杂的超频步骤尽可能简化，普通玩家也能一键上手，实在是赞！

另一方面，ALIENWARE COMMAND CENTER 还支持更改 ALIENFX 情绪同步光效，而除了实现常规的多灯效联动和特定游戏特定灯效之外，ALIENWARE Area-51m 还藏着另外一个大杀器。

没错，它就是「情绪同步反馈」：通过感知游戏过程中的关键情节或者灯光效果，ALIENWARE Area-51m 机身的灯效能够产生同步的效果，让玩家感官与游戏达到 100% 的同步率。举个例子，在玩《地铁：离去》的时候，当我打开武器的灯光时，键盘的灯效会伴随着一同亮起，而当我的血条低至一定数值，或者游戏角色面临危险时，Area-51m 机身的灯效会相应地泛起闪动的红色光效，就好像你和游戏角色有着一样的感知一样，十分炫酷，通过下面这张 GIF 图来感受一下！

此外，ALIENWARE Area-51m 还继承了前几代的眼球追踪技术，借助 Tobii 眼球追踪装置，Area-51m 衍生出了许多玩法，其中基础的比如它会检测你在不在电脑前，不在的话自动调暗屏幕亮度节省电量，如果检测到你在电脑前的话，又会将屏幕自动调亮；此外，还可以利用「眼神」来「触控电脑」。简单来说就是依靠深根在系统层面的 Tobii 眼球追踪装置，你可以利用眼神来切换多任务页面，当你按下 ALT 并同时看着某一个任务页面时，它将自动切换到相应的页面，省去了手动操作鼠标的步骤。更甚者，在 Tobii Experience 应用中，列出了支持眼球追踪的游戏，例如《刺客信条：奥德赛》《孤岛惊魂》等等。

既是致敬过去，也是瞻望未来

17 年前，ALIENWARE 凭借着一台 Area-51m 叩开了「游戏本的大门」，而如今全新的 Area-51m 将外星人最新的科技贯彻地淋漓尽致，称之为拥有强悍台式机性能的未来 PC 也不为过。

在这几天的体验之后，我渐渐明白了 ALIENWARE Area-51m 之所以取了这样一个名字的原因：在它身上，诸如科幻的外观设计、散热设计、史无前例的自由扩展性等都让我看到了外星人对于为玩家创造极致体验的执着追求，而这些特性不正是那些追求极致体验的玩家们一直向往的吗？

另外，ALIENWARE 还为玩家提供了多款专业外设：AW959 专业级电竞鼠标、拥有 12000 DPI 与 5 档飞敏设置、全键无冲等精致设计。AW988 专业电竞耳机，其 40mm 驱动单元的设计与 20Hz~20KHz 频响范围，确保音质得到最佳还原。AW3418DW 游戏显示器，支持NVIDIA® G-Sync™技术。众多的配件加持，能让你更加全情地投入游戏世界。
