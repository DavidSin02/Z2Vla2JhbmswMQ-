可以揣口袋的德生PL-330收音机：初步印象，功能概述和操作

德生PL-330收音机：初步印象，功能概述和操作


文：Jaap de Goede


作者在没有英语手册的情况下操作德生 PL-330


介绍


受父亲的启发，我喜欢听无线电广播已经有60个年头了。这包括收听短波传输。在我的收藏中，有很多收音机，包括基于SDR的接收器。德生PL-330是具有SSB和数字读数功能的短波便携式收音机。


我想分享一下我在PL-330上的操作经验，并且在整个过程中，我将把这个收音机与我的收藏中的类似收音机进行比较：Eton Satellit，XHDATA D-808和CountyComm GP5-SSB（又名Tecsun PL-365） 。


需要明确的是，此处使用的PL-330是中国国产版本，可能于2020年7月生产。在撰写本文时，尚无英文手册。同时，德生PL-990的英文手册有助于弄清楚PL-330的许多功能。较新的制造批次可能包含其他固件，这可能会更改操作方式。


大小和声音


PL-330的体积约为18立方英寸，小于20立方英寸。它很容易适合我的牛仔裤的口袋。20立方英寸（非正式）将其归类为Ultralight DX收音机。四个中最小的收音机是具有14立方英寸的GP5-SSB。Eton Satellit是这四台收音机中最大的一个，这些收音机的大小可以由眼睛决定，但也可以由耳朵决定。


PL-330具有数字音量控制，显示屏上会显示音量。矛盾的是，其他三个收音机具有模拟音量控制。所有收音机均具有用于立体声耳机的3.5毫米插座。PL-330的一个不错的功能是FM立体声解码器只能与插入式耳机配合使用。其他FM仍然处於单声道状态。我认为这是一个非常聪明的功能，因为如果FM立体声接收效果不好，为什么立体声解码器会降低扬声器的单声道声音？


个人音频质量排名（从最佳到最差）：

1. Satellit

2. PL-330和D-808

3. GP5-SSB


电源和电池


PL-330配有1000 mAh的BL-5C电池。电池可以通过微型USB端口在收音机中充电。无论收音机是否打开，显示屏都会永久显示电池状态。USB使从任何外部USB充电器或外部USB电池充电非常容易。这里是四台收音机的功率的快速比较：



我没有运行时数据。我只是在收音机的显示屏上指示时充电，并携带备用USB电池为收音机充电。



天线和背光


所有四台收音机都具有可扩展的鞭状天线（您还会期望什么）。另外，所有设备都有一个3.5毫米的插座，用于外接天线。唯一带有天线衰减器开关的收音机是Satellit。


所有四个收音机都有背光。但是其他收音机都没有的功能是PL-330能够在“始终打开”或“自动关闭”之间切换背光。只需按住5按钮即可在两种模式之间切换。为了进行比较，Satellit具有所有四个中最好的照明显示，而D-808太亮了。


支持的无线电频段


PL-330支持以下无线电频带：

  • 长波（153-513 kHz）
  • 中波（520-1710 kHz或522-1620 kHz）
  • 短波（1711-29999 kHz）
  • FM广播（64-108 MHz）


无线电频段和解调选择


要启用或禁用LW频段，必须在无线电关闭电源时长按2按钮。当无线电打开电源并且启用了LW时，您可以通过短按MW / LW按钮在LW和MW之间进行选择。


关闭电源后，长按3按钮可以切换美国或世界其他地区的MW频段计划。无线电开启后，您可以通过短按MW / LW按钮在MW之间进行选择。


要选择区域FM频段计划，您应该在收音机关闭时长按0按钮。收音机打开后，可以通过短按FM / ST选择FM波段按钮。


收音机打开后，可以通过短按<或>按钮选择SW波段。


在LW中，MW和SW频段的解调可以是AM，SSB和AM同步（SYNC）。短按SSB按钮可在AM和SSB之间切换。长按SSB按钮可在AM / SSB和AM同步模式之间切换。


SW频段内的一个不错的功能是您可以快速跳到预定的广播或HAM频段。在AM和SSB模式下，您可以按<或>按钮跳至所需的频段。可以选择以下频段并在显示屏中指示：


•AM（广播）波段：120m，90m，75m，60m，49m，41m，31m25m，22m，19m，16m，13m，11m


•SSB（HAM）频段：160m LSB，80m LSB，60m，LSB，40m LSB，30m USB，24m USB，20m USB，17m USB，15m USB，12m USB，10m USB


手动调整和步进


一个（数字）TUNING拨盘可用于任何收音机波段中的所有调谐操作。它仅在频段上调和调低，没有其他功能。（与其他三个带多功能拨盘的收音机相比，使用PL-330进行调谐非常容易。）根据所选频段和解调情况，可以通过“ STEP”按钮更改调谐步进。


在用于AM解调的LW和MW频段中，可以通过短按STEP按钮在1 kHz和10/9 kHz之间切换步进。


在用于AM解调的SW中，可以通过短按STEP按钮在1 kHz和5 kHz之间切换步进。


在用于SSB和同步模式的SW波段中，可以通过短按STEP按钮在10 Hz，1 kHz和5 kHz之间切换步进。我的其他三台便携式收音机，但仅实际频率的正负1 kHz（以10Hz为步长）。PL-330在SSB中具有连续微调功能；您可以以10 Hz为步长上下调谐整个SW频段。但是，我注意到如果您快速转动TUNING拨盘，步长可以从10 Hz更改为50 Hz。


在FM频段中，可以通过短按STEP按钮在100 kHz和10 kHz之间切换步进。


带宽设定


对于AM和SSB，您可以通过按AM BW按钮，将VOLUME拨盘转到所需的带宽并再次按AM BW按钮来更改带宽。


•LW / MW频段，AM带宽选择：2.5 kHz，3.5 kHz，5.0、9.0 kHz kHz

•SW波段，AM带宽选择：2.5 kHz，3.5 kHz，5.0 kHz

•SSB带宽选择：0.5 kHz，1.2 kHz，2.2 kHz，3.0 kHz，4.0 kHz


轻松调整模式或ETM和内存


我决定购买PL-330是因为我在GP5-SSB（PL-365）方面的良好经验和不良经验。PL-365超级便携，我认为ETM非常适合偶尔短波收听。但是，如果没有直接输入频率，GP5-SSAB就是不时操作的噩梦。现在，PL-330具有增强版的ETM：ETM +。没有其他电台具有这种功能，我认为ETM +很棒。


ETM +提供24个ETM库，这些库是根据小时自动选择的。您可以每小时按住ETM按钮来启动（新的或更新的）短波扫描。它将显示“ E”以及24小时格式的小时，例如E00到E23，具体取决于时间。当您快速按下ETM按钮时，它会根据时间调用存储库。例如，当您在15：24h快速按下ETM按钮时，它将调出库E15。然后，您可以通过转动TUNING拨盘来选择存储在存储区中的存储位置。



要在ETM和调整模式之间切换，请按ETM按钮。


除了ETM存储器外，PL-330还具有很多存储器，但缺少alpha标签。我真的很难记住我在那些记忆中的记忆。我几乎不使用它们。四个带有alpha标签的唯一一个是Eton Satellit，这使事情变得容易得多。


通过使用自动调整和存储（ATS）功能，可以将内存自动填充到与ETM库分开的库中。全部四个收音机都有ATS，我认为这仅适用于FM。它的工作原理如下：


1.按FM / ST。选择FM波段

2.按住FM / ST。启动ATS

3.使用TUNING拨盘从存储的FM电台中选择


要在内存和频率调整模式之间切换，只需按VM / VF按钮。


BUG或功能所限


使用“显示”按钮，您可以在以下之间更改显示的右上部分：


•信号强度

•时钟

•预设（仅在VM模式下）

•警报时间


几秒钟后，显示屏始终返回信号强度。除非长按DISPLAY按钮选择了时间。但是，在进行任何操作（包括音量更改）后，显示屏将恢复信号强度。我不确定这是功能还是错误。


使用耳机并更改音量时，有时声音会被切断。通过再次更改音量，它又回来了。似乎是一个BUG，而不是功能。


我尝试了AM同步模式。就像在卫星上一样：可怜。我建议隐藏此模式或使其正常工作。


通用串行总线


用USB电缆将收音机连接到PC时，其操作系统不会显示任何连接信息。如果收音机除了充电之外还具有USB逻辑，那么连接信息就会显示出来。我认为没有办法通过USB更新固件。


隐藏功能


没有英文手册，甚至没有中文手册，我也听不懂所有功能。


收音机电源关闭时：

•按住VF / VM将显示所有项目。

•按住8可以在“ HH：MM”和“ HH：MM：SS”之间切换显示时钟

•按住MW / LW波段中的3，在内部铁氧体和外部鞭状天线之间切换

•按住Enter键显示“ dEL ALL”，可能删除所有（全部除外）？

•按住M显示数字，不知道。

•按住0表示“ PO []”，不知道。


似乎可以组合显示固件版本。


缺少功能


欢迎使用RDS显示。即使PL-330与D-808和Satellit共享无线电芯片，PL-330也不在其显示器上显示FM电台数据系统（RDS）。背光按钮非常适合在黑暗中进行操作。（尽管您很快就会习惯按钮的布局。）


我没有错过的功能


四台收音机都没有DAB +或HD功能。对于DAB +接收，我使用Sony XDR-S41D，但在欧洲无法接收高清广播。DAB +和高清广播可能会使广播方式变得更昂贵，而且无论如何，我还是希望拥有良好的软件性能。


升级固件的可能性将很方便。但是我了解成本的限制以及无线电广播的负担。


Satellit和D-808支持空中频段（108-135 MHz）。偶尔听一个频道没关系，但是忘了频道扫描。为此，我有Bearcat UBC XLT125 VHF / UHF扫频仪。


结论


就像我刚开始时一样，我偶尔也是短波听众。我真的无法说出这四台收音机之间收音机性能的差异。当然，最好的收音机是随身携带的收音机。由于其尺寸大，使用单个调谐转盘易于操作以及具有ETM +之类的功能，PL-330可能是目前最好的便携式短波接收器（对我而言？）。感谢德生！


Jaap de Goede


2020年10月
