键帽有点甜哦 Ducky 3108S机械键盘图赏简评

2020-10-29
| 科技

上个月外设天下为大家带来了Akko X Ducky ZERO 3108机械键盘的评测，这款键盘在键帽工艺上有很大进步，在外观上使用了更具质感和美感的金砂黑配色，辨识度颇高，可谓是“黑出新意”。在这之后，Akko X Ducky又推出了3108S机械键盘，也就是今天的主角。3108S与3108相比增加了单色蓝光，并且配备了极具特色的PBT键帽，从外观上很容易让人联想到经典的三明治饼干奥利奥。我们现在就为大家带来这款键盘的图赏简评。

Akko X Ducky ZERO 3108S机械键盘包装

Akko X Ducky ZERO 3108S机械键盘包装

Akko X Ducky ZERO 3108S机械键盘的包装与3108完全一致，都是Ducky最新的家族式包装，就连包装正面的产品型号都是3108，所以如果把3108和3108S放在一起，不看铭牌贴纸完全无法区分。

Akko X Ducky ZERO 3108S机械键盘包装

Akko X Ducky ZERO 3108S机械键盘附件

Akko X Ducky Zero 3108S机械键盘附送了防尘罩、备用键帽、鸭头拔键器、说明书和质保卡，由于3108S的外包装和3108没有区别，所以笔者特别检查了说明书，好在说明书是明显不同的，对3108S的灯光调节进行了介绍，两者之间的一大差别就是灯光方面。

Akko X Ducky ZERO 3108S机械键盘外观

Akko X Ducky ZERO 3108S机械键盘外观

Akko X Ducky Zero 3108S机械键盘的外壳采用了跟3108同样的金砂黑配色，它是以黑色为底，掺杂金砂进行喷漆，使得键盘表面有细目颗粒反光状态。这款金砂黑明显降低了产品的塑料感，比纯黑色更具质感以及美感。

Akko X Ducky ZERO 3108S机械键盘外观

Akko X Ducky Zero 3108S机械键盘与3108的最大外观特点就是键帽了，它采用了特制的PBT二色热升华键帽，黑白配色。其中键帽的顶面是白色透光的，侧面均为黑色不透光的，它的字符通过热升华工艺印制键帽上，非常有特点，是跟大家喜欢吃的三明治饼干奥利奥神似，从视觉效果上给人一种甜甜的感觉，好想扭一扭，舔一舔，再泡一泡。因此它被大家戏称键盘中的“奥利奥”也算是名至实归了。

Akko X Ducky ZERO 3108S机械键盘外观

Akko X Ducky ZERO 3108S机械键盘外观

从侧面看这款这键盘观赏性极佳，它的键帽给人一种将键盘一刀切开的感觉，对于看惯了涂层、双色注塑和二色成型键帽的用户来说，这款键帽还是很有新意的，笔者在看到这款键盘的新闻时就被它深深地迷住了，想赶快见到实物。

Akko X Ducky ZERO 3108S机械键盘底面

Akko X Ducky ZERO 3108S机械键盘撑脚

Akko X Ducky ZERO 3108S机械键盘撑脚

Akko X Ducky ZERO 3108S机械键盘撑脚

Akko X Ducky ZERO 3108S机械键盘线材接口

Akko X Ducky Zero 3108S机械键盘在防滑脚垫、3向出线口、二段式撑脚和线材接口方面与3108是一致的，它的防滑性不错，线材长约1.8m，材质为橡胶线，粗细适中，比较柔软，它配置了屏蔽磁环但接口并没有采用镀金工艺。

Akko X Ducky ZERO 3108S机械键盘外壳

Akko X Ducky ZERO 3108S机械键盘外壳

Akko X Ducky ZERO 3108S机械键盘图案

Akko X Ducky Zero 3108S机械键盘的外壳右下角印有Akko以及Ducky的logo，与3108不同的是空格键侧面的ZERO图案，不光是在图案的样式上有所区别，3108S更是把图案的颜色从白色更改为更贴近金砂黑的金色，跟产品的整体风格更加贴合。

Akko X Ducky ZERO 3108S机械键盘键帽

Akko X Ducky ZERO 3108S机械键盘键帽

Akko X Ducky ZERO 3108S机械键盘键帽

Akko X Ducky Zero 3108S机械键盘的键帽字符采用了热升华工艺，它的字体比较美观，黑色与白色的键帽表面对比十分明显，看起来非常的清晰。不少玩家很喜欢热升华工艺，因为它能提供美观的字符、平整的手感和更长久的使用寿命，当然热升华键帽的成本也是比较高的，而这款酷似甜品的热升华PBT键帽，制作工艺要更复杂，成本自然也更高。

Akko X Ducky ZERO 3108S机械键盘键帽

Akko X Ducky ZERO 3108S机械键盘键帽

这款键帽的表面细腻平整，很有质感，手感出众，但是因为平整的关系，它的摩擦力不大，触感比较光滑。从背面看，这款键帽做工扎实，内部干净整洁，边角也没有毛边和粗糙的现象，它的加强筋图案也非常有特点，从透光性来说，键帽的整个顶部都是透光的，但实际上并没有特别透光，相信也是考虑到热升华字符的背光效果才这么设计的。

Akko X Ducky ZERO 3108S机械键盘轴体

Akko X Ducky ZERO 3108S机械键盘轴体

Akko X Ducky Zero 3108S机械键盘在3108的基础上增加了蓝色的背光，因此采用了轴体倒装的方式，笔者手中的这款为cherry青轴，大键位为卫星轴设计，在手感上与3108一致，按键稳定，没有明显的晃动，大小键位的手感很接近，一致性较好。

Akko X Ducky ZERO 3108S机械键盘灯光

Akko X Ducky ZERO 3108S机械键盘灯光

Akko X Ducky ZERO 3108S机械键盘灯光

Akko X Ducky ZERO 3108S机械键盘灯光

Akko X Ducky ZERO 3108S机械键盘采用了蓝色背光，因为它的键帽顶面是全透的，所以视觉效果上要比单纯的字符透光更加夸张，在白天和灯光环境中，它的亮度不错，但是受到相机白平衡的影响，拍摄样张有些偏紫。

Akko X Ducky ZERO 3108S机械键盘灯光

Akko X Ducky ZERO 3108S机械键盘灯光

在黑暗环境中拍摄就没有白平衡问题，它的视觉感受很奇妙，有些满天星辰的感觉，仿佛置身在夜空中。

Akko X Ducky ZERO 3108机械键盘有常亮、呼吸、波浪、跑马灯、雨滴、单点亮、爆炸和极光这几种灯光模式，灯光亮度为7阶调节，从灯光效果来说，它的亮度不算均匀，能够看出LED灯的位置，好在它的灯光流畅度还是不错的。

Akko X Ducky ZERO 3108S机械键盘灯效

Akko X Ducky ZERO 3108S机械键盘灯效

从整体视觉效果来讲这款键盘是非常有特点的，笔者是非常喜欢它酷似奥利奥的外观，但是打开灯光后，这款键盘原来甜甜的感觉就被很像满天星辰的灯光所取代，别有一番风味，对于追求完美的用户来说，键帽的透光性和灯光的均匀度还不够完美，这也是受到PBT材质的影响。当然键帽在美观度和手感方面是非常出色的，相信对键帽的颜值和手感都有追求的玩家们一定会喜欢这款产品的。
