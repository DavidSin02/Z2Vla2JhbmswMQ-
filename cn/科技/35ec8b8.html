佳能宣布EOS-1D X Mark III开发，将会在10月24日至26日的PHOTOPLUS上进行展示。

新品特性：

1）全新CMOS，更高感光度；

2）可以拍摄10bit HEIF照片；

3）4K / 60p / YCbCr 4:2:2 / 10bit / Canon Log以及RAW格式进行内录；

4）AF/AE追踪功能时，16fps（光学取景方式）20fps（实时取景方式）连拍；

5）使用RAW格式进行连拍时，新机型的一次最多连拍张数将提升5倍；

5）新的自动对焦感应器，其中心像素区域的有效像素数量比前代机型增加约28倍；

6）深度学习技术，将进一步提升对焦追踪能力，在实时取景拍摄方式下，自动对焦区域将扩展至图像感应器约100%（垂直方向）以及约90%（水平方向），525个对焦框。

7）无线文件传输器WFT-E9速度提高2倍；

8）按钮背光；

9）相机存储卡将由CFast2.0改为CFexpress

10）电池效能更高，能拍摄更多照片。

佳能正式发布RF70-200mm F2.8 L IS USM，与EF 70-200mm f/2.8 L IS III USM相比在长度和重量上分别大幅缩减约27%和28%，1片玻璃模铸非球面，3片UD镜片，1片超级UD镜片，双Nano USM对焦马达。最近对焦距离约0.7m，放大倍率在70mm端以及200mm端分别约为0.11X和0.23X，5档防抖效果，滤镜规格为77mm。镜头最大直径及长度分别为约89.9mm及约146.0mm，重量约1070g。

正在开发中的EOS-1D X Mark III预计将具备以下功能：

1. 新开发的CMOS图像感应器以及影像处理器，可实现更高画质和更高的连拍速度

（1）佳能正在开发中的CMOS图像感应器以及影像处理器将实现比前代机型更高的静态照片拍摄感光度。新机型将支持以10bit的HEIF格式2保存静态照片，同时还将支持以4K / 60p / YCbCr 4:2:2 / 10bit / Canon Log以及RAW格式进行视频机内录制。

（2）无论采用光学取景还是实时取景方式拍摄，都可实现AF/AE追踪功能，并实现比前代机型更高的连拍速度。当启动AF/AE追踪功能时，该相机将实现最高约16张/秒（光学取景方式）以及20张/秒（实时取景方式）连拍表现。此外，当使用实时取景方式拍摄时，拍摄者能够选择使用机械快门或电子快门进行拍摄。

（3）相对前代机型，当使用RAW格式进行连拍时，新机型的一次最多连拍张数将提升5倍。

2. HEIF（High Efficiency Image File高效率图像文件）格式。相对JPEG格式，可实现更高的动态范围和色彩还原表现。

2. 借助新开发的自动对焦感应器以及自动对焦算法，将实现更精准的自动对焦表现

（1）将采用新型的自动对焦感应器，其中心像素区域的有效像素数量比前代机型增加约28倍。当通过光学取景器拍摄时，自动对焦感应器获得的高分辨率信号将使高对焦精度成为可能。此外，进一步扩展的自动对焦亮度范围将适用于光学取景拍摄以及实时取景拍摄。

（2）当使用光学取景或实时取景方式拍摄时，新的自动对焦算法将带来更稳定的对焦表现，同时由于新算法应用了深度学习技术，将进一步提升对焦追踪能力。

（3）在实时取景拍摄方式下，自动对焦区域将扩展至图像感应器约100%（垂直方向）以及约90%（水平方向）。此外，当相机自动选择时，最多525个对焦框可以用于自动对焦。

3. 提升数据传输能力以满足专业用户需求

（1）正在开发中的无线文件传输器WFT-E9（无线认可审批中）配合EOS-1D X Mark III使用时，将实现比前代机型WFT-E8（2016年4月上市）快2倍以上的无线图像数据传输速度3。此外，WFT-E9还可配合EOS C500 Mark II电影摄影机（将于2019年12月底上市）使用。

佳能RF70-200mm F2.8 L IS USM所拥有的70mm至200mm焦段，可以兼顾体育题材、生态摄影、人像、风光等多种题材影像创作。同时全部焦段内的F2.8恒定最大光圈以及9片叶片圆形光圈结构，也能获得美丽柔滑的大幅虚化效果，同时能使拍摄获得更高的快门速度，特别是在暗光线环境下能进一步提升拍摄成功率。

作为佳能配备F2.8恒定最大光圈镜头的新一代产品，佳能RF70-200mm F2.8 L IS USM镜头得益于EOS R系统的大口径卡口和短后对焦距离优势，并通过应用多种佳能先进的光学技术，从而在整个变焦范围内，实现了从画面中心到边缘画质的再一次提升。该镜头采用13组17片光学结构，借助1片玻璃模铸非球面镜片和1片UD（超低色散）非球面镜片，实现了球面像差校正和变焦镜头小型化。高效排列的3片UD镜片和1片超级UD镜片，以大幅校正色像差，减少从画面中心到边缘的色晕，实现了高画质。此外，该镜头还应用了佳能SWC亚波长结构镀膜。SWC镀膜具有出色的抗反射性，特别是对于入射角较大的光，可以减少在具有大曲率镜片的边缘发生眩光和鬼影的可能性。

佳能RF70-200mm F2.8 L IS USM镜头的另一突破是成为佳能首款采用双Nano USM对焦马达的镜头。该镜头通过两个Nano USM对焦马达分别驱动浮动镜片组和对焦镜片，实现电子浮动对焦控制，在不牺牲紧凑外形的前提下，实现安静、节能和高速的操作，同时有效抑制了对焦的呼吸效应，在进行短片拍摄时，可在进行焦点偏移操作时，获得更稳定的构图。此外，电子浮动对焦控制的应用还显著缩短了最近对焦距离。RF70-200mm F2.8 L IS USM在全部焦段内的最近对焦距离仅为约0.7m，放大倍率在70mm端以及200mm端分别约为0.11X和0.23X，能够获得更强大、更突出的特写摄影表现力。

在拍摄静态照片时，RF70-200mm F2.8 L IS USM镜头与EOS R或EOS RP相机协作，可实现双重检测IS，在镜头IS功能工作同时，还可以通过相机CMOS图像感应器的图像信息中检测抖动量。通过比较和分析这两类信息，可以高精度地检测和校正单靠镜头难以检测到的低频抖动（由于呼吸等引起的机身慢速抖动），从而实现最大相当于约5级2快门速度的手抖动补偿效果。而在进行短片创作时，同样可通过相机的图像信息来检测镜头IS功能未进行校正的模糊量，同时通过RF卡口更强大的通信能力，可以对相机上的短片五轴数码IS和具备IS功能的RF镜头进行协同控制，实现组合IS功能，获得更强大的影像稳定效果。

RF70-200mm F2.8 L IS USM镜头使用的滤镜规格为77mm。镜头最大直径及长度分别为约89.9mm及约146.0mm，重量约1070g（仅镜身）。

RF85mm F1.2 L USM DS，

平滑散焦呈现柔美的焦外虚化效果

焦外虚化效果是拍摄人像的关键，但用户对虚化形态的喜好各不相同，没有正确的答案。因此，在RF85mm F1.2 L USM基础上，佳能又推出支持平滑散焦的RF85mm F1.2 L USM DS。这一新镜头继承了RF85mm F1.2 L USM出色光学设计的同时，追求平滑的焦外虚化效果，为追求焦外虚化之美的专业摄影师以及热爱人像创作的摄影爱好者带来新选择。

佳能RF85mm F1.2 L USM DS

佳能RF85mm F1.2 L USM DS镜头样片

为实现柔美的焦外虚化效果，RF85mm F1.2 L USM DS采用了DS镀膜设计。DS镀膜是佳能的一种蒸镀多层镀膜技术。该结构中心的透射率较高，边缘的透射率较低，因此可实现平滑的焦外虚化效果。同时，由于该镜头采用将DS镀膜应用于镜片的前表面和后表面的双面覆盖设计，可以使整个边缘的透射率降低，从而柔和地虚化整个焦外虚化图像的轮廓，以更大限度地发挥DS镀膜的效果。同时，通过使用F1.2大光圈和DS镀膜，即使在画面的边缘也能获得显著而平滑的焦外虚化效果，因此十分适用于对前景和背景进行柔和的焦外虚化拍摄创作。此外，DS镀膜带来的焦外虚化平滑散焦效果会随光圈值的变化而产生变化，在F1.2大光圈下，DS镀膜的平滑散焦效果会更明显。镜头由于DS镀膜边缘透光率下降，全开光圈时镜头亮度比F1.2低约1⅓级。相机通过升级至新版固件可实现合适的自动曝光控制。

RF85mm F1.2 L USM DS非常适合在装饰性照明等点光源背景下拍摄，可以柔和平滑地虚化点光源的光斑轮廓以创造迷人的焦外虚化效果，或者在复杂背景下突出拍摄对象的拍摄，例如柔和地虚化重叠的树木，降低背景和前景的复杂度，使拍摄对象更加突出。

RF85mm F1.2 L USM DS继承了RF85mm F1.2 L USM出色的光学设计，除了通过DS镀膜获得平滑散焦的焦外虚化效果外，还具有高分辨率、高对比度和色像差抑制表现，实现了从画面中央到边缘的高画质。基于佳能EOS R系统大卡口直径以及短后对焦距离的先天优势，镜头得以在光学素质以及设计空间上实现更大提升。镜头光学结构为9组13片。采用了BR镜片及UD（超低色散）镜片的组合有效抑制色像差，大幅减少了大光圈镜头易产生的色晕。同时，还在光圈前方配置了1片研磨非球面镜片，并通过镜片位置的优化与合理配置，实现了F1.2大光圈下画面中心到边缘整体的高画质表现。此外，镜头还采用了防反射效果好的ASC镀膜，可提高镜片的透射率，有效抑制画面内光源造成的眩光与鬼影，降低逆光对成像的影响，带来清晰通透的照片效果。

RF85mm F1.2 L USM DS采用9片圆形光圈叶片设计，最近对焦距离仅为约85cm，最大放大倍率为0.12X，因此可以实现更近距离的人像或静物特写捕捉，进一步提升拍摄的灵活度。该镜头使用滤镜规格为82mm。镜头最大直径及长度分别为约103.2 mm及117.3 mm，重量约1195g。

更多共同特性

两款镜头同样配有RF镜头上新增的控制环，通过控制环，可以根据使用需求，将该控制环设定为曝光补偿（Ev）、感光度数值（ISO）、光圈值（Av）或快门速度（Tv）等常用拍摄参数的调整。借助这一设计，用户可以将视线更专注地聚焦在取景器内，无需转移视线就能方便、快速地完成设定拍摄参数的调整，进一步提升了拍摄体验。

RF70-200mm F2.8 L IS USM及RF85mm F1.2 L USM DS镜头均采用了防水滴防尘结构3，从而实现在严苛拍摄环境下的高可靠性。特别是RF70-200mm F2.8 L IS USM镜头，由于采用外变焦类型，该镜头为此特别设计了通风路线，抑制水汽吸入，并采用特殊防尘材料，以阻隔灰尘渗入。同时，两款镜头也加入了具有更好憎油憎水性的防污氟镀膜，在带来良好透光率的同时，可更加轻松地擦除附著于镜片上的水滴或油渍4。
