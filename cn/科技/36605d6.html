使用飞车应注意哪些事项?

知识点 : 送电线路

架空线连接前后应做哪些检查?

答：应做如下检查：

(1)被连接的架空线绞向是否一致。

(2)连接部位有无线股绞制不良、断股、缺股现象。

(3)切割铝股时严禁伤及钢芯。

(4)连接后管口附近不得有明显松股现象。

试题编号 : 05

知识点 : 送电线路

紧线时耐张(转角)塔均需打临时拉线，临时拉线的作用及要求各是什么?

答：(1)作用：平衡单边挂线后架空线的张力。

(2)要求：

①对地夹角为30°～45°。

②每根架空线必须在沿线路紧线的反方向打一根临时拉线。

③转角杆在内侧多增设一根临时拉线。

④临时拉线上端在不影响挂线的情况下，固定位置离挂线点越近越好。



试题编号 : 06

知识点 : 送电线路

钢筋混凝土杆地面组装工艺包括哪些方面?

答：包括以下内容：

(1)组装横担：可将横担两端稍微翘起10～20mm，以便悬挂后保持水平。

(2)组装叉梁：先安装四个叉梁抱箍，将叉梁交叉点垫高，其中心与叉梁抱箍保持水平，再装上、下叉梁。

(3)部分螺栓连接构件不宜过紧，以防起吊时损坏构件。

(4)绝缘子组装后在杆顶离地时再挂在横担上，以防绝缘子碰伤。

(5)金具、拉线、爬梯等尽量地面组装，减少高空作业。安装完毕后，进行各部分全面检查。

知识点 : 送电线路

怎样用异长法观测并计算运行线路的弧垂？

答：（1）在观测档的任一杆塔的导线悬垂线夹下方选取一垂直距离a，绑一弛度板。

（2）另一人持另一块弛度板登观测档的另一杆塔，观测该档弧垂。

（3）上下试凑，使两弛度板与弧垂点恰成一直线，此时绑好第二块弛度板，量取至该导线悬垂线夹的距离b。

（4）由弧垂计算公式计算出导线的弧垂f。

试题编号 : 08

知识点 : 送电线路

使用线材时应做哪些检查？

答：线材的技术特征、制造规范而应符合国家标准，在取得合格证和出厂证明后尚需进行下列外观检查：

试题编号 : 09

知识点 : 送电线路

杆塔编号一般有何规定？

答：（1）线路杆塔的编号一般以送电端出线杆塔为1号，从送电端依次编至受电端杆塔。

（2）完整的杆塔编号应包括表明电压等级、线路名称、杆塔号码。耐张转角换位杆塔应涂相色漆。

（3）用油漆在杆塔上的编号或挂牌，以离地面2.5-3.0m为宜，字体不宜过小，面向线路附近巡线通道或大路。

（4）双回路的双杆塔应在每一基杆塔上分别刷印杆上相应线路的名称和杆号。若双回路为单杆或铁塔，则应将线路名称和杆号分别印在同一杆塔的两边外侧或铁塔的两侧塔腿上。

架空线路为什么会覆冰？

答：形成覆冰层的原因是由于在架空线路上附着水滴，当气温下降时，这些水滴便凝结成冰，而且越结越厚。

(2)引起断线、断股。

(3)杆塔倒塌、变形及横担损坏。

(4)绝缘子及金具损坏。

(5)对跨越物的危害。

为什么要尽可能地降低接地电阻的数值？

答：因为接地电阻愈小，雷击时引起的过电压愈小，防雷效果就愈好，所以要尽可能地降低接地电阻数值。

试题编号 : 13

知识点 : 送电线路

弧垂的大小对线路的安全运行有何影响？

答：弧垂的大小直接影响到线路的安全运行：

（1）弧垂过小，导线受力增大，当张力超过导线许可应力时，会造成断线。

（2）弧垂过大，导线对地距离过小而不符合要求，在有剧烈摆动时，可能引起线路短路。

试题编号 : 14

知识点 : 送电线路

导线为什么要进行换位？

答：(1) 架空线路三相导线在空间排列往往是不对称的，引起三相系统电磁特性不对称。

(2) 三相导线电磁特性不对称引起各相电抗不平衡，从而影响三相系统的对称运行。

(3) 为保证三相系统能始终保持对称运行，三相导线必须进行换位。

试题编号 : 17

知识点 : 送电线路

制动钢丝绳受力情况是怎样变化的?如何有效防止制动钢丝绳受力过大?

答： (1) 制动钢绳在杆塔起立开始时受力收紧。

(2) 随着杆塔起立角度增加，抱杆的支撑力逐渐减小，制动绳索的受力逐渐增大。

(3) 当抱杆失效时，抱杆的支撑力消失，制动绳的受力最大。

(4) 为防止制动钢绳受力过大，在抱杆失效前，应调整制动绳长度，使杆根坐入底盘。

试题编号 : 18

知识点 : 送电线路

使用倒落式抱杆整体组立杆塔时如何控制反向临时拉线?

答：(1) 随着杆塔起立角度的增大，抱杆受力渐渐减小。

(2) 在抱杆失效前，必须带上反面临时拉线。

(3) 反面临时拉线随杆塔起立角度增加进行长度控制。

(4) 当杆塔起立到70°时，应放慢牵引速度。

(5) 当杆塔起立到80°时，停止牵引，用临时拉线调直杆身。

试题编号 : 19

知识点 : 送电线路

拉线在水泥杆上的固定应符合哪些规定？

答：拉线在水泥杆上的固定必须符合以下规定：

（1）楔形线夹和UT线夹卡其舌板与拉线应接触紧密，在拉线正常受力情况下应无滑动现象，钢绞线的端头应装在受力面的一侧。

（2）钢绞线断头露出规定：上端头应为300mm，可用不小于2.0mm的单股镀锌铁线绑扎。下端头露出应为500mm，且用不小于3.2mm的单股镀锌铁线绑扎。上下端绑扎一般为50mm。

（3）拉线的交叉处不得互相摩擦。

（4）拉线调好后，楔形线夹和UT线夹露出的丝扣长度应为10-30mm。

（5）拉线包箍距杆顶200mm。

（6）水平拉线在跨越公路及其他交通道路时，其对地距离一般不小于6m。

（7）在居民区若拉线高于带电导线时，应加装拉线绝缘子，且必须使拉线绝缘子位于带电部位以下。一般情况下，拉线绝缘子安装在距拉线上端1-3m处。

架空线弧垂观测档选择的原则是什么?

试题编号 : 24

知识点 : 送电线路

简要解释档距、水平档距、垂直档距、代表档距和临界档距的含义。

答：(1)档距：是指相邻两杆塔中心点之间的水平距离。

(2)水平档距：是指某杆塔两侧档距的算术平均值。

(3)垂直档距：是指某杆塔两侧导线最低点间的水平距离。

(4)把长短不等的一个多档耐张段，用一个等效的孤立档来代替，达到简化设计的目的。这个能够表达整个耐张段力学规律的假想档距，我们称之为代表档距。

(5)临界档距：是指由一种导线应力控制气象条件过渡到另一种控制气象条件临界点的档距大小。

什么是混凝土的和易性?和易性对混凝土构件质量有何影响?

答：(1)定义：混凝土的和易性是指混凝土经搅拌后，在施工过程中干稀均匀的合适程度。

(2)质量影响：

①和易性差的混凝土影响构件内部的密实性。

②和易性差的混凝土影响构件表面的质量。

③和易性差的混凝土影响构件棱角的质量。

放线前应进行哪些准备工作？

答：（1）根据架线方向及长度布置线轴的安装位置。

试题编号 : 32

知识点 : 送电线路

现浇基础施工时，如遇特殊情况中途中断混凝土浇灌，应如何处理?

答：原则上现浇基础的混凝土应一次连续浇成，不得中断。特殊情况下在浇制同一基础混凝土时遇特殊情况中断超过2h，不得继续浇灌。若要继续浇灌必须待混凝土的强度达到1.18MPa后，将连接面打毛，并用水清洗，然后浇一层厚为10～15mm与原混凝土同样成分的水泥砂浆，再继续浇灌。

知识点 : 送电线路

从哪些方面检查现浇混凝土的质量?

答：从以下几方面进行检查：

(1)坍落度不得大于配合比设计的规定值。

(2)配比材料误差应控制在施工措施规定范围内，并严格控制水灰比。

(3)混凝土强度检查以现场试块为依据。

试题编号 : 34

知识点 : 送电线路

现场浇筑基础混凝土的养护应符合哪些规定?

答：应符合以下规定：

(1)浇筑后应在12h内开始浇水养护，当天气炎热、干燥有风时，应在3h内浇水养护。浇水次数以保持混凝土表面始终湿润为准。

(2)混凝土养护日期不少于5昼夜。

(3)基础拆模经表面检查合格后应立即回填土，并继续浇水养护。

(4)采用养护剂养护时，应在拆模并经表面检查合格后涂刷。涂刷后不再浇水。

(5)日平均气温低于5℃时不得浇水养护。

试题编号 : 35

知识点 : 送电线路

基础分坑前杆塔中心桩位置在复测中出现哪些情况应予以纠正?

答：(1)横线路方向偏差大于50mm。

(2)顺线路方向偏差大于设计档距的1%。

(3)转角杆的角度偏差超过1′30″。

(4)标高与设计值相比偏差超过0.5m。

试题编号 : 36

知识点 : 送电线路

架空线路耐张段内交叉跨越档邻档断线对被交叉跨越物有何影响

答：其影响如下：

(1)交叉跨越档邻档断线时，交叉跨越档的应力衰减，弧垂增大。

(2)交叉跨越档因弧垂增大，使导线对被交叉跨越物距离减小，影响被交叉跨越物及线路安全。

(3)对重点交叉跨越物必须进行邻档断线交叉跨越距离校验。

试题编号 : 37

知识点 : 送电线路

架空线路的垂直档距大小受哪些因素的影响?其大小影响杆塔的哪种荷载?

试题编号 : 46

知识点 : 送电线路

混凝土中水泥的用量是否越多越好?为什么？

答：(1)混凝土水泥用量不能过量，应符合配合比设计要求。

(2)混凝土中水泥用量过多会使混凝土在硬化过程中因水分蒸发引起水泥体积过量收缩，造成混凝土开裂、露筋等缺陷，从而影响混凝土的强度。

为何拌制混凝土尽可能选用较粗的砂

答：其原因如下：

(1)较粗的砂粒，其单位体积的表面积小，易与水泥浆完全胶合。

(2)砂粒表面胶合的水泥浆充分，有利于提高混凝土强度。

(3)有利于降低水泥的用量，减小水灰比。

试题编号 : 48

知识点 : 送电线路

如何更换直线杆塔?

答：(1)为了减少停电时间，通常在原直线杆前后1～2m处先竖立新杆，利用旧杆挂起滑轮，即可吊立新杆。

(2)在立新杆前，先将旧杆上悬垂线夹拆去，使导线悬空并用粽绳将导线吊起。架空地线暂不撤除起支持作用。

(3)新杆立好后填土夯实，将导地线接于新杆上，调整两档弧垂相等即可固定导线。

(4)利用新杆挂滑轮拔去旧杆，拔杆前先将旧杆前方泥土挖成斜槽，挖至杆根，然后将钢丝绳端结扎旧杆横担附近，牵引绞磨后慢慢松出牵引绳使电杆慢慢落地。

(5)若新杆不能利用，则可用人字抱杆进行施工。

试题编号 : 49

知识点 : 送电线路

为什么导线截面积要按机械强度的要求校验其截面？

答：由于架空线路架设在大气中，导线要经受各种外界不利条件的影响，如果选用导线截面小，机械和电气过载能力小，易发生事故，为保证安全运行，导线应有一定的机械强度，即导线的截面不宜过小。概括起来说，在高压线路上：

（1）不许使用单股导线。

（2）铝及铝合金、钢芯铝绞线的最小截面为35平方毫米以上。

（3）其他材料导线的最小截面积为16平方毫米。

试题编号 : 50

知识点 : 送电线路

简要说明导线机械特性曲线计算绘制步骤。

答：其计算绘制步骤如下：

(1)根据导线型号查出机械物理特性参数，确定最大使用应力，结合防振要求确定年平均运行应力。

(2)根据导线型号及线路所在气象区，查出所需比载及各气象条件的设计气象条件三要素。

(3)计算临界档距并进行有效临界档距的判定。

(4)根据有效临界档距，利用状态方程式分别计算各气象条件在不同代表档距时的应力和部分气象条件的弧垂。

(5)根据计算结果，逐一描点绘制每种气象条件的应力随代表档距变化的曲线和部分气象条件的弧垂曲线。
