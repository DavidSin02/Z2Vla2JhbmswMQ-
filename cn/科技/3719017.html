Akko Ducky 3108地平线键盘评测：永无止境

2020-10-29
| 科技

“海的思念绵延不绝 终于和天 在地平线交会”这是笔者在上学时听的歌曲中的一句歌词，好像早已忘记的歌和歌词，当我在拿到并看到手中的这款Akko X Ducky 3108宝石蓝Horizon地平线机械键盘时，脑中浮现的除了有地平线的样子外，就是这首歌了。

相信大家对Akko（艾酷）X Ducky推出的3108系列机械键盘早已不再陌生，因为这个系列早已推出了不少产品，比如：3108金砂黑机械键盘、3108S机械键盘、3108S Newbee战队限定版机械键盘、3108 PBT侧刻机械键盘等等。而这次，它又以Ducky今年的主题色“宝石蓝”为基色，并搭配上Hardcap全新的配色“地平线”PBT二色键帽，打造出了一款既沉稳又不失活泼的机械键盘。

本次，外设天下除了收到了Akko X Ducky 3108 Horizon地平线机械键盘外，还收到了Horizon地平线配色的108键PBT二色键帽，对于两种风格略有不同的“地平线”，你更喜欢哪种呢？赶紧来看下吧~

Akko X Ducky 3108地平线键盘的包装展示

Akko X Ducky 3108地平线键盘的包装展示

　　Akko X Ducky 3108 Horizon地平线键盘的包装依然采用了Ducky最新的家族式包装，整体的像素块设计使它看起来很有时尚感。包装的正反面都有着银色亮面的大大LOGO及型号名称，反而对键盘的特点介绍就少的可怜了。

Akko X Ducky 3108地平线键盘的包装展示

Akko X Ducky 3108地平线键盘的配件展示

　　它的内包装还算不错，保护比较到位，附件也非常全面、实用，除了键盘本体外，还有着1个全尺寸防尘罩、1本说明书、1张保障卡、8颗PBT二色键帽，以及1个鸭头拔键器。

Horizon地平线配色的108键PBT二色键帽的包装展示

Horizon地平线配色的108键PBT二色键帽的包装展示

　　Horizon地平线配色的108键PBT二色键帽的包装看起来很简单，只是透明的塑料外壳，也没有赠送额外的拔键器。

Akko X Ducky 3108地平线键盘的外观展示

　　3108系列是Akko与Ducky合作后推出的全新入门级机械键盘，是为代替原有的2108系列，3108延续了窄边框的经典设计，比2108更窄、更精致、也更美观。该键盘的尺寸约为：440mm×140 mm×41mm，重量约为：1.2KG。

Akko X Ducky 3108地平线键盘的外观展示

Akko X Ducky 3108地平线键盘的外观展示

　　键盘整体配色为纯蓝色时的照片，就是那套Horizon地平线配色的108键PBT二色键帽的样子。带有一颗金黄色键帽的照片，就是这款3108地平线刚被取出时的样子。

Akko X Ducky 3108地平线键盘的外观展示

Akko X Ducky 3108地平线键盘的官方宣传图

　　不过，笔者还是更喜欢现在这样将附赠的键帽全部都搭配好的样子。因为它的外壳为宝石蓝色，键帽又为深蓝+宝石蓝的配色，再加上那些金橘色与晴空蓝的混色搭配字体，以及能起到画龙点睛的金黄色键帽，就使它很形象地还原了太阳初升，海平面的状态，故命名为“地平线”。

Akko X Ducky 3108地平线键盘的细节展示

Akko X Ducky 3108地平线键盘的细节展示

　　Akko X Ducky 3108 Horizon地平线键盘的外壳很漂亮，宝蓝色的配色，再加上那星星点点的珠光感觉，很有那种大海在阳光照耀下泛着波光的感觉。而且为了再次提升键盘的辨识度，在键盘下边框的右侧还有着Akko X Ducky的LOGO。

Akko X Ducky 3108地平线键盘的细节展示

　　该键盘为108键配列，所以在右上方还有着4颗多功能按键，分别为：计算器按键、静音按键、声音减小按键、声音增大按键。

Akko X Ducky 3108地平线键盘的细节展示

Akko X Ducky 3108地平线键盘的细节展示

　　键盘的背面虽然没有过多的繁琐设计，但是很人性化。它的背面一共有4小条防滑橡胶，并且即便不将撑脚打开，它也有着一定的高度设计。

Akko X Ducky 3108地平线键盘的细节展示

Akko X Ducky 3108地平线键盘的细节展示

　　同时，它的撑脚还为两段式设计，再加上它的三向导线槽设计，更加突显它人性化的一面了。

Akko X Ducky 3108地平线键盘的细节展示

　　由于键盘本身就带有一定坡度，所以只将最小的撑脚打开即可。对比笔者来说，总感觉最高的撑脚实在是有些高，坡度太明显了，不过这也得看每个人的使用习惯和键盘的摆放了。

Akko X Ducky 3108地平线键盘的细节展示

　　它的线材为黑色的橡胶材质，柔软度还不错，长度为1.8m，并且像防拉扯设计、屏蔽环、防尘罩、镀金接口、绑线带等均有，配置还是很齐全的。

Akko X Ducky 3108地平线键盘的键帽展示

　　Akko X Ducky 3108地平线键盘的键帽是在与“铠砚Hardcap”合作突破PBT二色完整字形（就是O、D、B、P等字符为全闭口设计）技术后，推出的首款配色产品。当然，这种配色除了用在这款3108上以外，键帽也同时提供零售版本，比如与键盘同时寄来的那款Horizon地平线配色的108键PBT二色键帽，而它除了108键配列外，还单独又推出了加入鸡年11Key的大全套套装。

Akko X Ducky 3108地平线键盘的键帽展示

Akko X Ducky 3108地平线键盘的键帽展示

　　据官方介绍说，这款Horizon地平线配色的键帽是经过第三方检测机构测试的，它拥有高达85%的PBT含量，能保证它不易打油的特性。这款键帽虽然看起来会有些类似磨砂质感的晒纹，但它的整体手感还是很顺滑、很舒适的，而且键帽的整体也很干燥，不是那种黏腻腻的手感。

Akko X Ducky 3108地平线键盘的键帽展示

　　键帽颜色的主体为深邃的深海蓝底色，而它的字体却为棱角分明的金橘色与安静的晴空蓝混色搭配，再加上细字体及全闭口式设计，使它的整体观感很漂亮。

Akko X Ducky 3108地平线键盘的轴体展示

　　它的轴体采用了极具信仰的CHERRY原厂轴体，笔者手中的这款为CHERRY红轴，而它还有着青、茶、黑三种轴体。关于CHERRY红轴的手感，相信也不用笔者再多说什么了吧，直上直下的轻盈手感，还是很适合玩游戏使用的，不过由于这款键盘带有钢板，所以在与无钢板的CHERRY红轴相比时，它的手感就要略硬一些了。

这款键盘没有背光灯，所以在轴体上方也并没采用LED灯珠，但它却预留了灯位，可支持玩家们自己进行DIY改灯。

Akko X Ducky 3108地平线键盘的大键位展示

　　它的大键位采用了卫星轴设计，整体的调校非常不错，一致性也很好，但可惜它的声音会略微有一点点大。

Akko X Ducky 3108地平线键盘的无冲测试

　　经过测试，这款Akko X Ducky 3108地平线键盘支持全键无冲，所以以后喜欢打拳皇类型游戏的玩家们就可以放心使用了。

Akko X Ducky 3108地平线键盘的游戏测试

　　在游戏测试中，这款键盘的表现还是非常不错的，键帽的触感平滑、舒适，轴体的回弹速度快、力度轻，能够迅速按出技能对应的按键。在打字方面，它的手感也同样很好，笔者的这篇评测就是用它敲出来的，不过对于带钢板的红轴来说，其实笔者更喜欢无钢板的红轴，毕竟那样会感觉更绵软一些，不过笔者猜肯定会有不少人喜欢这种带钢板的感觉。

型号：3108（非背光）

颜色：宝石蓝/地平线配色

按键数：108键

键帽：PBT二色（85%含量）

轴：Cherry MX

LED灯光：无，预留灯位，支持DIY改灯

全键编程：支持，Ducky Macro V2.0

多媒体按键：支持

无冲：Ducky N-Key Rollover，支持全键无冲切换

Win键屏蔽：支持

尺寸：440mm×140 mm×41mm

重量：1.2KG

产品优点：

+ 键盘带有两段式撑脚及三向导线槽；

+ PBT键帽手感舒适，做工精细，不易打油；

+ 键帽字体为工艺难度较高的闭口式设计；

产品缺点：

- 空格键声音略大。

Akko X Ducky 3108地平线键盘拥有全新且漂亮的配色，在创意方面也是很有新意的，而且不得不说的是Akko X Ducky所采用的键帽做工确实是越做越好，哪怕只是和半年前的产品做对比，也能看出它的变化之处。

就像“地平线”一样，是个神秘、永远让人向往，但却始终没有尽头的路，而好的设计与好的工艺亦是如此，希望Akko X Ducky能在未来创造出更多工艺精良、赏心悦目的好产品。

　　这款地平线键盘的价格为599元；Horizon地平线配色的基础版108键为169元（不含黄色按键部分）；Horizon地平线配色的完整版加入了鸡年11Key套装，大全套零售价199元，对这款键盘或是对键帽感兴趣的玩家们可以多多关注下。
