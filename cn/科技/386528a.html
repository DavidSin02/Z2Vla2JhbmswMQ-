Windows Core OS 最新消息：移植 Win 10 操作中心

2020-10-29
| 科技

微软 Windows Core OS 又有新消息。

关于微软正在打造的新一代操作系统 Windows Core OS（WCOS）的消息没有停过，但是官方又没有正面给出回应，外界一直通过各种渠道在了解该操作系统。

去年 1 月份，微软正在开发 WCOS 的消息被披露，从披露的消息来看，WCOS 是一个跨 PC、平板、智能设备与移动等各种物理端的操作系统，它采用模块化设计，使用 Composable Shell（C-Shell/CShell）模块化交互界面，以实现跨平台界面。围绕 CShell，WCOS 衍生出了众多不同的版本（子系统/模块），最早为人熟知的是：

  • AndromedaOS，它支持双屏，用于折叠屏的 Surface 设备中；
  • Polaris，它完全基于 UWP 和现代硬件定制，用于 PC 端；
  • Centaurus，用于更大的双屏移动设备。

除此之外，有开发者发现了其它相关版本代码以及其适用场景：

  • Oasis，适用于 HoloLens 等混合现实设备；
  • Aruba，适用于 Surface Hub 等巨屏设备；
  • Lite（Santorini），轻量型精简版，类似 Windows 10 S，只能运行 Microsoft Store 的 UWP/PWA 应用。
  • ……

值得一提的是其中的 Windows Lite OS，因为这个版本已经被曝光得很详实，我们之前就报导过：Windows Lite OS 代号为“Santorini”，基于 Windows Core OS，它的界面与 Chrome OS 很相似。

这些消息让大众对 Windows Core OS 有了一定的了解，但微软官方从来也没有松过口，没有正面介绍过 WCOS。

到了去年 8 月份，在微软招募的软件工程经理岗位说明中明确写到：SiGMa Device Platform (DPlat)团队拥有 Windows Core OS（WCOS）的定义和交付权利，而该系统将适用于所有新设备。所招募工程师主要工作是在新硬件上整合软件体验，这就相当于证实了 WCOS 的存在，同时微软团队正在进行积极的研发。

今年 1 月份的时候，又有消息指出 Windows Core OS 包含了开源组件，这让其模块化（子系统化）的猜想进一步得到证实。

而近日，有网友发现微软某工程师的 Linkedin 档案上有关于开发 WCOS 的记录：开发 Action Center 的 Windows Core OS（WCOS）变体（Develop the Windows Core OS (WCOS) variation of Action Center）。

Action Center 其实是 Windows 10 中的操作中心，它是一个快速操作系统通用功能的便捷入口，同时也是一个系统消息通知中心。该工程师的一个职责应该正是将 Windows 10 的这一功能移植到 Windows Core OS 中。

披露出来的种种信息在不断丰富我们对于 WCOS 的想象，就像文章开头说的，WCOS 应该是一个集合了众多“版本”Windows 子系统（模块）的操作系统，并且其中还有部分子模块是开源的。

但是必须强调的是，这一切都是外界的小道消息，官方还没有明确的公告，所以 Windows Core OS 究竟存不存在，是不是这些消息中所描述的样子？暂时不得而知。另外，据传言，该操作系统的子系统 Polaris 将会在今年发布，具体情况如何，我们拭目以待。
