企业风险评价，（SCL）安全检查表，Excel工作表

2020-10-29
| 科技

说实在的，在很多中小企业里，风险评价在企业运行过程中，还是一项比较概念化的工作，明确地说就是不被重视，似乎有没有都无所谓。

初次接触风险评价，感觉十分陌生，不知道从何入手，什么都不知道，由于要做企业标准建设，不可避免要做风险评价管理工作，包括风险评价制度建立，表格建立，会议记录，评价结果等等一系列内容。

其实风险评价是建立在对整个企业特别熟悉和了解的基础之上来做的一项工作，不然真的不知道如何来评价，在第三方的帮助之下，这件工作就比较顺利展开了。

这里表格是以安全检查表SCL方式进行评价，内容基本涵盖了所有工作项目，其实这个工作依然停留在纸面，要真正落实到工作当中，我想还是一个漫长的过程。

评价要很多内容，在评价管理制度里面明确了，这里不做分析，最终呈现的结果就是这几张表格，往往也就是这几张表格，真正反映企业评价后是一个现状。

最终要做出一个综合性评价结果，包括一些可能存在的风险，和预防解决办法等等。

总之风险评价就是要一些未雨绸缪的工作，通常是看不到实际效果的。确切地说，如此评价出来的结果，往往作为一种企业规避问题的一种参考，很有必要。

本来做一个窗体来，对评价过程进行描述，没有实现，下一次再做。窗体式管理和显示对于Excel来说是一个很简单的事情，也相对容易实现。

这些代码是对评价结果进行颜色标识，以不同颜色对结果分别显示，看图可以理解，很容易实现。​

颜色代码

Private Sub Worksheet_Activate()Dim varr()varr = Array("低风险", "可接受", "一般风险", "较大风险", "重大风险")Dim R As Integer, g As Integer, b As IntegerDim Ro As Range, Rr As RangeSet Rr = Range("K4:K" & Range("K65535").End(xlUp).Row)For Each Ro In Rr    With Ro        Select Case .Value                Case varr(0)                        R = 201                        g = 252                        b = 211                Case varr(1)                        R = 121                        g = 212                        b = 111                Case varr(2)                        R = 211                        g = 182                        b = 181                Case varr(3)                        R = 211                        g = 112                        b = 111                Case varr(4)                        R = 251                        g = 202                        b = 11                Case Else                   Exit Sub        End Select                .Interior.Color = RGB(R, g, b)    End WithNext RoEnd Sub
