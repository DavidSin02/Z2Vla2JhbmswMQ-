13.56MHz、125KHz、915M、2.4G门禁读卡器

2020-10-29
| 科技

DAIC-MJ-RW是 13.56MHz 非接触式射频 IC 卡专用读卡器，采用高集成度的射频芯片、及自带多奥看门狗电路 CPU 芯片设计。读卡器性能稳定，读卡种类多，抗干扰能力强。可支持 ISO14443A/B、ISO15693 国际协议标准，可读取 MF 卡、二代证、CPU 卡等。可提供韦根 26、韦根 34 通讯接口，适用于多奥门禁、考勤、收费、防盗、巡更等各种射频识别应用领域。

非接触式射频 ID 卡专用用远距离读头，釆用多奥自主研发的 125KHz 解码电路和译码算法，具有卡号识别准确﹑抗干扰能力强、性价比高等特点。读 ID 远距离能够达到 90~120CM 的距离。可应用于门禁，考勤，停车场等领域。

DAID-MJ-RW是 125KHz 非接触式射频 ID 卡韦根门禁读头器，釆用先进的射频接收线路设计，通过嵌入式微控制器结合高效译码算法，完成对 64bits Read-Only uem4100 兼容式 ID 卡的接收。本门禁读头具有工作电流小、读卡稳定性高、读卡速度快、性价比高，卡号识别的准确率 100% 等特点。适用于DUOAO门禁、考勤、收费、防盗、梯控等各种射频识别应用领域。

多奥LEGIC 13.56MHZ 非接触式射频 IC 卡专用读卡器，是目前国际上读卡种类最多的读卡器之一。它的读卡芯片采用了 LEGIC 公司设计的高集成度读卡专用芯片 SM-4200，可支持国际上所有 13.56MHZ 非接触卡协议。并且模块的控制芯片自带DUOAO看门狗和电压监控电路，使得该读卡器具有支持多种卡协议、卡安全性高、性能稳定、功耗低等特点。读卡器同时拥有 W26/34,RS485 多种接口可选，非常适用于门禁、考勤、收费、酒店梯控等各种射频识别应用领域。
