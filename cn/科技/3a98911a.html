1

智能化

汽车智能化的终极目标是实现所有条件下都能完成所有的驾驶任务，即完全的自动化。在SAE关于自动驾驶的五级分层定义里，L3以下称为辅助驾驶（ADAS）。目前奔驰、宝马、Tesla上量产应用如ACC、AEB、LKA等均属此类，典型配置包含：1个77GHz雷达+2个24GHz雷达+ME前视+12个超声波 + 全景环视 + 域控制器。

尽管在自动驾驶传感器的三大玩家（视觉、毫米波和激光雷达）里，毫米波雷达仍因其全天候、结构紧凑及长距离检测等优势备受关注。其能够实现对被测物体相对距离、现对速度、方位的高精度测量。其中24GHz多用于短/中距测量，77GHz用于中/长距离测量。

汽车毫米波雷达种类

当下国内汽车毫米波雷达还处于产业发展初期，与国外差距较大，在设计仿真、开发验证及生产测试方面面临诸多挑战。Keysight提供的软硬件组成的系统级解决方案，在雷达基带算法、系统架构设计、雷达信号的生成和分析、目标的仿真、天线测试方面为产业提供全方位的技术服务。

// 设计模拟及评估 //

Keysight 的 SystemVue 软件包含丰富的汽车雷达库，结合毫米波汽车雷达系统仿真模板，能够快速搭建用户自定义的汽车雷达系统，并可通过场景仿真模型构建真实汽车雷达工作场景。

在雷达射频微波电路上，Keysight 的 ADS能够进行精确仿真，将 MMIC、LTCC、键合线、封装、接头、天线等进行有机集成。

使用 Keysight 的 EMPro，能够方便的将波导功分网络、波导至同轴转换、同轴至微带转换以及微带功分网络、微带辐射单元等在一个环境中进行整体仿真、加快仿真效率。

Keysight仿真设计软件在汽车毫米波雷达中的应用方案

// 产品测试及验证 //

通过使用信号产生套件（Signal Studio）搭配任意波形发生器（AWG）及高性能信号源（PSG），可以轻松产生FMCW、Pulse等雷达测试信号。利用高性能信号分析仪（UXA）搭配矢量信号分析软件（VSA）可以完成精确的波形分析。

对于汽车防撞雷达这样工作环境复杂的系统，任何单一的测试项目都无法代替系统性的测试验证。通过Keysight的雷达目标模拟器，可以轻松在实验室环境下对所探测的目标进行距离、目标RCS以及相对速度的模拟，避免传统的外场易受天气及不确定性等因素，以保证测试验证的一致性。最终可以轻松架设24GHz、77GHz、79GHz及更高频段的雷达测试环境。

Keysight毫米波雷达综合测试系统

通过以上设计评估、信号产生与分析和系统功能验证（目标模拟）等一系列的开发和验证工作之后，将为我们的自动驾驶带来更多的安全性，并确保其往更高的层面升级。

2

网联化

对于更高级别的自动驾驶，SAE所定义的L4-L5自动驾驶，则离不开汽车的网联化。V2X，全名vehicle to everything，包含V2I（车路）、V2P（车人）、V2N（车网）和V2V（车车）四大部分，涵盖汽车与其他几乎所有目标进行通信的协议，是实现辅助驾驶甚至无人驾驶的核心技术。

现在已被业内公认的未来提高用户驾驶感受的一个必要实现途径，同时也逐渐成为汽车在市场上的一大卖点。V2X早期主要是基于DSRC，专用短距离通信技术，后期随着蜂窝移动通信技术发展又出现了C-V2X。从技术持续迭代、可靠性和信号传输时延等来说，C-V2X有较大优势。

然而汽车的应用环境极其复杂，无论是DSRC还是C-V2X其信息交互的速率和可靠性的要求与手机等产品不可同日而语。Keysight的UXM网络模拟器、信道仿真及虚拟路测系统可以更完整更可靠的验证车联网模块，从测试测量维度加速V2X相关技术的发展。

Keysight 智能汽车应用复杂电磁环境模拟系统

在智能化、网联化的浪潮里，ADAS甚至自动驾驶技术的不断革新、高品质车载娱乐影音、OTA升级、V2X、云计算等一系列技术的发展，已经远远超出了CAN或FlexRay等传统车载总线的数据传输能力。

汽车行业需要的高可靠性、高传输带宽、低成本、低电磁辐射和时延等车载网络系统的需求，极大推进了汽车以太网的发展。而为了确保以上的特性，需针对物理层的一致性、兼容性和功能的测试，对网络进行压力测试并针对高低温、高低湿度、噪音或低电压等恶劣条件进行测试。

Keysight 可以为您提供完整的L1-L7测试解决方案，100% 覆盖使用 BroadR-Reach 和 100Base-T1。让您轻松应对产业发展带来的挑战，推动自动驾驶和网联化早日实现。

新能源化

在汽车工业智能化、网联化革命的同时，越来越多的传统汽车厂宣布停止或计划停止内燃机研发、使用更加绿色环保的电驱动系统取而代之，并从简单的电池发展成为完整的独立电网。无论您正在进行何种创新，如研发新的智能电网技术、测试电动汽车、制造电池或用于电源管理，能源效率和安全都决定着您的创新能否成功。

Keysight提供全面的电池、BMS、充电桩、车载充电机和逆变器等测试方案，严苛模拟验证电驱动系统的工作环境、状态，将效率和安全放在第一位，推动绿色能源的健康发展。同时为新能源汽车客户提供全面可定制化的方案。
