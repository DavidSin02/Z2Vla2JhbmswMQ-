零基础怎样学好Oracle数据库

2020-11-14
| 科技

Oracle数据库作为Java软件开发中非常重要的一环,一定要非常熟练的掌握。那么作为零基础，如何来学习Oracle数据库呢?下面就随北京学佳澳小编一起来了解一下。

首先了解下基础概念

学习Oracle你首先得了解这几个名词:数据库，数据库管理系统，数据库系统

数据库(DataBase DB)：是存放数据的仓库,所有的数据在计算机存储设备上保存,而且所有保存的数据会按照一定的格式进行保存。

数据库管理系统(DataBase Manager System DBMS)：科学的组织和存储数据,可以高效的获取和维护数据。

主要功能:数据操作功能,数据库的事务管理和运行管理,数据定义功能,数据组织,存储和管理,数据库的建立和维护功能。

数据库系统(DataBase System, DBS)：是指在计算机系统中引入数据库后的系统,一般由数据库,数据库管理系统,应用系统,数据库管理员构成。

Oracle的组成、工作过程与原理又是什么?

由Oracle数据库和Oracle实例组成，包括逻辑结构和物理结构。

物理结构:指的是数据库中的一组组操作系统的文件

分为三类:

数据文件: 用于存储数据库数据,如表,索引数据 .dbf文件

控制文件: 记录数据库物理结构的二进制文件 .ctl文件

日志文件: 记录对数据库的所有修改信息,用于故障恢复 .arc

逻辑结构:是指数据库创建之后形成的逻辑概念之间的关系

数据库- 表空间 - 逻辑对象(模式) - 数据段 - 数据区间 - 数据块

那Oracle数据库又有哪些特点呢?

Oracle数据库作为一种对象关系型的数据库管理系统,具有以下几个特点:

1.多用户/海量数据的事务处理

2.数据安全性和数据完整性 性能优越

3.分布式数据处理

4.可移植性(跨OS)

接下来是实例了!

[Oracle数据库基础用户]

sys - manager - sysdba

打开数据库服务器/ 关闭数据库服务器 / 备份数据库 / 恢复数据库 / 日志归档 / 会话限制 / 管理表/视图

system - change_on_install - sysdba / sysoper

打开数据库服务器/ 关闭数据库服务器 / 备份数据库 / 恢复数据库 / 日志归档 / 会话限制

普通用户(scott)

默认的普通权限用户/ EMP / DEPT

连接身份:

normal:普通身份

sysoper:操作员身份，可以打开，关闭，备份，恢复数据库等权限

sysdba:管理员身份

[表、表空间、用户基础操作语法]

-- Oracle 创建表空间/用户/授予权限

-- 创建临时表空间/表空间语法:

create[temporary] tablespace TABLESPACE_NAME

[tempfile | datafile] 'FILE_PATH'

size 10m

autoextend on;

-- 删除表空间语法:

drop tablespace TABLESPACE_NAME

including contents and datafiles;

-- 创建用户和密码并指定表空间语法:

create user USERNAME identified by PASSWORD

default tablespace TABLESPACE_NAME

temporary tablespace TEMP_TABLESPACE_NAME;

-- 删除用户语法:

drop user USERNAME cascade;

cascade:删除这个用户以及这个用户下的所有对象了

-- 给用户授予权限语法:

grant AUTH_NAME to USERNAME;

-- 撤销权限语法:

revoke AUTH_NAME from USERNAME;

-- 修改用户密码语法:

ALTER USER USERNAME IDENTIFIED BY PASSWORD;

-- 锁定用户语法:

alter user USERNAME account lock;

-- 解锁用户语法:

alter user USERNAME account unlock;

-- 创建表/数据

create table 表名 (

字段名字段类型(容量),

字段名字段类型(容量)

);
