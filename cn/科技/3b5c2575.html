文本编辑器EmEditor快捷键指令

2020-11-14
| 科技

EmEditor是一款非常实用且专业的文本编辑器，该软件内含众多便于文本编辑的功能。帮助用户以轻松、便捷、高效的方式处理文本文件，受到众多用户的喜爱。

对于一名快节奏的用户而言，深度运用EmEditor快捷键指令是必备的工作方式之一。相对于通过鼠标一步步操作而言，运用EmEditor快捷键指令工作将会更加省时省力，快捷方便。

接下来，软发网将详细为大家汇总文本编辑器EmEditor快捷键指令。

新建文本：Ctrl+N - 创建一个新的文本文件

打开：Ctrl+O - 打开一个已存在的文件

保存：Ctrl+S - 保存当前文件

打印：Ctrl+P - 打印当前文件

还原默认工作区：Alt+0 - 还原一个已保存的默认工作区状态

保存默认工作区：Ctrl+Shift+0 - 保存默认的工作区

保存并关闭：Ctrl+E - 保存并关闭当前文件

关闭：Ctrl+F4, Ctrl+W - 关闭当前文件

保存并全部关闭：Ctrl+Shift+E - 保存并关闭所有打开的文件

全部关闭：Alt+Shift+X - 关闭所有打开的文件

撤消：Alt+Backspace, Ctrl+Z - 撤消上一次的操作

重做：Ctrl+Y - 重做上一次撤消的操作

剪切：Shift+Delete, Ctrl+X - 剪切所选内容或当前行至剪贴板

剪切行：Ctrl+L - 剪切选取的行或当前行至剪贴板

复制：Ctrl+C, Ctrl+Insert - 复制所选内容或当前行到剪贴板

复制引用并取消选择：Ctrl+Shift+Q - 将选区复制引用,粘贴至剪贴板并取消选择

粘贴：Ctrl+V, Shift+Insert - 在光标位置插入剪贴板中的内容

粘贴为系统默认编码：Alt+Ctrl+V - 用系统默认编码粘贴剪贴板上的内容

粘贴为引用文本：Ctrl+B - 插入剪贴板中的内容为引用文本

粘贴包含换行符：Ctrl+J - 插入剪贴板内容以及换行符

循环粘贴：Ctrl+Shift+V, Ctrl+Shift+Insert - 在光标位置处插入剪贴板中的历史内容

删除：Shift+Backspace, Delete - 删除所选内容或删除当前光标右侧的一个字符

删除左侧字符：Backspace - 删除所选内容或删除当前光标左侧的一个字符

删除行：Ctrl+Shift+L - 删除选取的行或当前行

删除单词：Ctrl+Shift+Delete - 删除当前光标所在位置的单词

删除右侧单词：Ctrl+Delete - 删除当前光标右边的单词

删除左侧单词：Ctrl+Backspace - 删除当前光标左侧的单词

右移一个字符：Right - 将光标向右移动一个字符

左移一个字符：Left - 将光标向左移动一个字符

右移一个单词：Ctrl+Right - 将光标向右移动一个单词

左移一个单词：Ctrl+Left - 将光标向左移动一个单词

移至行首：Alt+Ctrl+Home - 移动光标到当前行的行首

移至行首或文本起始位置：Home - 移动光标到当前行首个非空的字符位置

移至逻辑行行首：Alt+Home - 移动光标到当前逻辑行的行首

移至行末：End - 移动光标到当前行的行末

移至逻辑行行末：Alt+End - 移动光标到当前逻辑行的行末

查找配对的括号：Ctrl+] - 移动光标到配对的圆括号/方括号

查找匹配标记：Ctrl+. - 移动光标至匹配的标记

制表符或增加行缩进：Tab - 插入制表符或增加当前行缩进

左移制表符或减少行缩进：Shift+Tab - 将光标向左移动一个制表符或减少行缩进

上一行：Up - 移动光标到上一行

下一行：Down - 移动光标到下一行

上一页：Page Up - 移动光标到上一页

下一页：Page Down - 移动光标到下一页

文档顶部：Ctrl+Home - 移动光标到当前文档的顶部

文档底部：Ctrl+End - 移动光标到当前文档的底部

上一逻辑行：Ctrl+Page Up - 移动光标到上一个逻辑行行首

下一逻辑行：Ctrl+Page Down - 移动光标到下一个逻辑行行首

窗口顶部：Alt+Page Up - 移动光标到当前窗口顶部

窗口底部：Alt+Page Down - 移动光标到当前窗口底部

交换开始位置和结束位置：Shift+F8 - 交换选区中的开始位置和结束位置

全选：Ctrl+A - 全选

选择字符：F8 - 切换字符选择模式

选择行：Ctrl+F8 - 切换行选择模式

垂直选择：Ctrl+Shift+F8 - 切换垂直选择模式

选择单词：Alt+F8 - 选择当前光标所在位置右侧的单词

往右扩展一个字符：Shift+Right - 把选区往右扩展一个字符

往左扩展一个字符：Shift+Left - 把选区往左扩展一个字符

往右扩展一个单词：Ctrl+Shift+Right - 把选区往右扩展一个单词

往左扩展一个单词：Ctrl+Shift+Left - 把选区往左扩展一个单词

扩展选区到上一行：Shift+Up - 把选区向上扩展一行

扩展选区到下一行：Shift+Down - 把选区向下扩展一行

扩展选区到上一页：Shift+Page Up - 把选区向上扩展一页

扩展选区到下一页：Shift+Page Down - 把选区向下扩展一页

扩展选区到行首或文本起始位置：Shift+Home - 把选区扩展到当前行的行首或文本起始位置

扩展选区到逻辑行行首：Alt+Shift+Home - 把选区扩展到当前逻辑行的行首

扩展选区到行末：Shift+End - 把选区扩展到当前行的行末

扩展选区到逻辑行行末：Alt+Shift+End - 把选区扩展到逻辑行的行末

扩展选区到文档顶部：Ctrl+Shift+Home - 把选区扩展到文档顶部

扩展选区到文档底部：Ctrl+Shift+End - 把选区扩展到文档底部

扩展选区到配对的括号：Ctrl+Shift+] - 把选区扩展到配对的圆括号/方括号

扩展选区到配对的标记：Ctrl+Shift+. - 把选区扩展到配对的标记

取消：Esc - 取消所选内容

大写：Ctrl+Shift+U - 把选定文本全部转换为大写字符

小写：Ctrl+U - 把选定文本全部转换为小写字符

日期与时间：Shift+F5 - 插入日期与时间

插入抑音符：Ctrl+` - 在输入字符 a、e、i、o、u、A、E、I、O、U、` 或 " 时在该字符上插入抑音符

插入锐音符：Ctrl+' - 在输入字符 a、d、e、i、o、u、y、A、D、E、I、O、U、Y、' 或 " 时在字符插入锐音符

插入扬抑符：Ctrl+Shift+6 - 在输入字符 a、e、i、o、u、A、E、I、O 或 U 时在该字符上插入扬抑符

插入波形符：Ctrl+Shift+` - 在输入字符 a、n、o、A、N或O 时在该字符上插入波形符

插入分音符：Ctrl+Shift+; - 在输入字符 a、e、i、o、u、y、A、E、I、O、U 或 Y 时在该字符上插入分音符

插入上圆圈符：Ctrl+Shift+2 - 在输入字符 a、A 或空格时在该字符上插入上圆圈符

插入连字符：Ctrl+Shift+7 - 在输入字符 a、o、s、A 或 O 时在该字符上插入连字符

插入下加符：Ctrl+, - 在输入字符 c 或 C 时在该字符上插入下加符

插入划线符：Ctrl+/ - 在输入字符 o、O 或 c 时在该字符上插入划线符

插入变音符：Alt+Shift+6 - 在输入字符 a、i、o、u、A、I、O 或 U 时在该字符上插入一个变音符

插入长音符：Alt+- - 以键入字母的方式插入一个长音符

插入倒转的问号：Alt+Ctrl+Shift+/ - 插入一个倒转的问号

插入倒转的感叹号：Alt+Ctrl+Shift+1 - 插入一个倒转的感叹号

插入版权符：Alt+Ctrl+C - 插入版权符号

插入注册符：Alt+Ctrl+R - 插入注册符号

插入商标符：Alt+Ctrl+T - 插入商标符号

插入欧元符：Alt+Ctrl+E - 插入欧元符号

插入特殊字符：Ctrl+Shift+I - 插入一个特殊字符

在上方插入一行：Shift+Enter - 在当前光标位置上方插入新的一行

在下方插入一行：Ctrl+Shift+Enter - 在当前光标位置下方插入新的一行

重复行：Ctrl+Shift+Y - 重复当前逻辑行

插入/改写：Insert - 切换插入/改写模式

向上移动行：Ctrl+Shift+Up - 将已选中的行或当前行上移一行

向下移动行：Ctrl+Shift+Down - 将已选中的行或当前行下移一行

编号：Alt+N - 在当前光标或垂直选区中插入编号

半角/全角转换：Ctrl+T - 转换选定文本中的半角和全角字符

将所有制表符转换为空格：Ctrl+Shift+T - 转换整个文档中制表符为空格

移动到上次编辑位置：Ctrl+Shift+Z - 移动光标到上次编辑位置

跳转至标签：F10 - 跳转至标签

跳转：Ctrl+G - 跳转到指定行

切换输入法：Ctrl+F11 - 为远东语言启用或禁用输入法编辑器

跳转到下一个：F4 - 跳转到输出栏或语法检查器中的下一个位置

跳转到上一个：Shift+F4 - 跳转到输出栏或语法检查器中的上一个位置

后退：Alt+Left - 移动光标至上一位置

前进：Alt+Right - 移动光标至下一位置

设置或清除书签：Ctrl+F2 - 为当前行设置书签,或清除当前行的书签

清除当前文档中的所有书签：Ctrl+Shift+F2 - 清除当前文档中的所有书签

转到本组文档的下一个书签：F2 - 转到当前群组中的下一个书签

转到本组文档的上一个书签：Shift+F2 - 转到当前群组中的上一个书签

下一个节点：Alt+Down - 当显示大纲视图时移动光标到下一个节点

上一个节点：Alt+Up - 当显示大纲视图时移动光标到上一个节点

扩展到下一个节点：Alt+Shift+Down - 当显示大纲视图时扩展选区到下一个节点

扩展到上一个节点：Alt+Shift+Up - 当显示大纲视图时扩展选区到上一个节点

下一单元格：Alt+Shift+Right - CSV、TSV或DSV模式下跳转至下一单元格

上一单元格：Alt+Shift+Left - CSV、TSV或DSV模式下跳转至上一单元格

下一拼写错误：Ctrl+M - 跳转至文档内的下一拼写错误

上一拼写错误：Ctrl+Shift+M - 跳转至文档内的上一拼写错误

查找：Ctrl+F - 查找一个字符串

查找下一个：F3 - 查找下一个匹配

查找上一个：Shift+F3 - 查找上一个匹配

查找下一个字：Ctrl+F3 - 查找指定文本的下一个匹配结果

查找上一个字：Ctrl+Shift+F3 - 查找指定文本的前一个匹配结果

添加下一个匹配结果：Ctrl+R - 添加选取指定文本的下一个匹配结果

选择所有匹配结果：Ctrl+Shift+A - 选择指定文本的所有匹配结果

替换：Ctrl+H - 替换一个字符串

取消查找高亮：Alt+F3 - 清除搜索字符串的高亮

查找下一个Unicode字符：F9 - 查找下一个无法转换为要保存的编码的Unicode字符

查找上一个Unicode字符：Shift+F9 - 查找上一个无法转换为要保存的编码的Unicode字符

取消Unicode高亮：Alt+F9 - 取消高亮显示无法转换为要保存的编码的Unicode字符

在文件中查找：Ctrl+Shift+F - 在多个文件中搜索匹配的字符串

将焦点设置在筛选工具栏：Ctrl+Shift+D - 把焦点设到筛选工具栏上

将焦点移至查找工具栏：Ctrl+D - 将焦点移至查找工具栏

不换行：Ctrl+1 - 以不换行的方式呈现每一行

按字符换行：Ctrl+2 - 按指定的字符换行

按窗口换行：Ctrl+3 - 按窗口大小换行

按页面换行：Ctrl+4 - 按页面宽度换行

切换输出栏：Alt+2 - 显示或隐藏输出栏

全屏：F11 - 切换全屏模式

重设字体大小：Ctrl+0 - 重设显示EmEditor字体大小

字符代码值：Ctrl+I - 显示 Unicode 字符值

缩小：Alt+O - 缩小显示一个较大的区域(打印预览)

放大：Alt+I - 放大显示一个较小的区域(打印预览)

重新扫描：F5 - 重新扫描并刷新所比较的文档

上一更改：Shift+F7 - 跳转至上次更改

下一更改：F7 - 跳转至下一更改

上一更改的行：Ctrl+Shift+F7 - 跳转至上一更改的行

下一更改的行：Ctrl+F7 - 跳转至下一更改的行

开始/停止宏录制：Ctrl+Shift+R - 开始或停止宏录制

运行宏：Ctrl+Shift+P - 运行宏

查找宏关键字：F1 - 查找选定宏关键字的帮助主题

当前配置属性：Alt+Enter - 更改当前选定配置的属性

选择配置：Shift+F11 - 显示一个弹出菜单来选择配置

快速启动：Ctrl+Q - 快速启动一个指定的命令

快速启动 - 命令：Ctrl+Shift+C - 显示快速启动窗口并选择命令模式

快速启动 - 选项：Ctrl+Shift+O - 显示快速启动窗口并选择选项模式

快速启动 - 符号：Ctrl+Shift+S - 显示快速启动窗口并选择符号模式

重复上一命令：Ctrl+Shift+G - 重复上一个已执行的命令

用户菜单(1)：Ctrl+K - 显示用户菜单(1),被用作两次按键快捷方式的第一个快捷键

终止工具任务：Ctrl+Break - 终止当前正在运行的工具任务

还原窗口位置：Ctrl+9 - 将窗口还原到上次保存的位置

分割：Ctrl+F12 - 分割当前窗口到窗格中

移除分割：Shift+F12 - 移除分割

切换水平分割：Ctrl+- 切换水平窗口分割

下一个窗格：F6 - 切换到下一个窗格

上一个窗格：Shift+F6 - 切换到上一个窗格

向上滚动一行：Ctrl+Up - 将文档向上滚动一行

向下滚动一行：Ctrl+Down - 将文档向下滚动一行

下一个：Ctrl+Tab - 切换到下一个文档

上一个：Ctrl+Shift+Tab - 切换到上一个文档

右边的文档：Ctrl+F6 - 切换至标签页右边的文档

左边的文档：Ctrl+Shift+F6 - 切换至标签页左边的文档

搜索帮助主题：Ctrl+F1 - 显示EmEditor帮助主题

打开EmEditor软件Windows资源管理器：Alt+Shift+E - 使用该文件夹作为起始路径打开 Windows 资源管理器

以上即为常用EmEditor快捷键指令，希望对大家学习或使用EmEditor软件有所帮助。
