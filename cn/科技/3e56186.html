洗衣

01 ) 神奇魔法水去渍

日本主妇之间流传着一个神奇的魔法水，只用家里随处可得的三项物品

衣服汗渍、酱汁、汤水或酒，几乎90％的污渍都能去除

不少主妇都用这魔法水清洁衣服污渍，相对轻松多了。

像网红主妇Yura也特别拿这魔法水做实验，结果沾在衣服上的葡萄汁清除得相当干净。

发明魔法水的人叫做横仓靖幸，以经营洗衣店为生，他家的洗衣店已经有70多年的历史，每年都处理来自各地的5000多件衣服。

为了清理各种难解污渍，经过多次开发研究，这神奇的魔法水才终于面世。

魔法水只用了三项物品

氧系漂白剂

小苏打

中性的洗碗剂

他说，“用于破坏染色的氧系漂白剂、增强漂白效果的小苏打，以及防止浮起污渍重新粘覆的中性洗碗剂，这三项清洁剂混合在一起能达到强力去渍的效果。

魔法水

中性洗碗剂 3滴

用于衣服的氧系液体漂白剂 3茶匙

小苏打 1茶匙

※多次实验后的黄金配比，建议不要更动比例。

※请注意不要使用含氯漂白剂，混合后可能会产生有毒气体。

混合后搅拌，注意轻轻搅拌5次即可。

因为这是利用漂白剂、小苏打与洗碗精混合后的化学反应来除渍，大力搅拌会加速化学反应，反而会减低去渍效果。

在沾了番茄汁的衣服涂上魔法水。

顺利去除污渍，是不是非常有效！

这边给大家小提醒，像酱油、酱汁这种有色液体，涂上魔法水前，先在衣服下垫一条毛巾。在溶解污渍时，才不会沾染到衣服背面。

同时利用牙刷或棉棒拍打，有助加速污渍溶解。

用魔法水去除污渍后，记住要先用水冲洗干净，再放入洗衣机，否则衣物上残留的漂白剂可能扩散到其他衣物。

TIPS 魔法水使用注意事项

1. 碱性碳酸氢钠（小苏打）和酸性漂白剂经过3小时后化学作用就会完全中和，因此去渍的效果也会消退，所以每次使用前都必须重新制作一次。

2. 泥渍、墨水渍无法用魔法水去除。

3. 不能水洗的衣服，建议不要用魔法水，以免造成褪色。

02 ) 去除衬衫、T恤汗渍

袖子和衣领上的黄色污渍是人体“皮脂”的污渍，经过汗液氧化所残留下来的，所以是属于一种油渍，建议使用去除油污的“固体肥皂或洗碗剂”来清洗。

▲ 图片来源/INS@fashion_care_eikosha

若是陈年汗渍，建议改用氧系漂白剂，这具有分解粘性皮脂污渍的能力。将漂白剂直接涂在衣领上，然后用洗衣机清洗即可。

▲ 图片来源/INS@fashion_care_eikosha

03 ) 去除茶渍

由于红茶是将热水倒入茶叶中来提取红茶，本身有易溶于水的特性。

准备物品

中性洗碗剂

牙刷或棉棒

像红茶、咖啡、可乐等有色液体，一旦沾了水就容易往外扩散，因此在清除茶渍前，先在衣服背面垫上毛巾，再将洗碗剂涂在污渍上。

▲ 图片来源/ever sense inc

由于污渍容易扩散，小心地用手指搓洗，或是用牙刷、棉棒拍打。

若是痕迹过于久远，则建议再加上氧系的漂白剂加强。

可乐、咖啡、酱油同样是可溶于水的特性，所以也可以一样的方法清洗。

TIPS

沾到红茶、咖啡、可乐等污渍的当下，不能立即用水洗，否则就会看到一圈扩散的污渍。

在这种情况下，建议用手帕或手巾轻轻拍打即可，最好是从污渍背面拍打擦拭，在处理的过程中，也要注意不能搓揉，以免污渍扩散。

回家后再立即用上述的清洁步骤清洗。

06 ) 去除墨水笔渍

▲ 图片来源/INS@cojicaji

准备物品

还原型漂白剤

水盆

在水盆先倒入一公升约40°C热水，再倒入半瓶盖的漂白剂。

▲ 图片来源/INS@cojicaji

将有污渍的区域泡进水里。

▲ 图片来源/INS@cojicaji

浸泡后能看见墨水立即消融。建议静置30至60分钟，让漂白剂继续发挥效用。

▲ 图片来源/INS@cojicaji

笔渍几乎都消除了！

▲ 图片来源/INS@cojicaji

若家中没有还原型的漂白剂，建议将污渍先泡热水30分钟，不要超过浸泡时间，否则墨水会向外扩散染色。

接着再用肥皂搓揉清洗。

▲ 图片来源/INS@ever sense inc

TIPS

油性墨水较难以漂白剂清洗，建议改用酒精涂抹，再用牙刷拍打擦拭，就能有效去除笔渍。

晾衣速干

01 ) Z字型晾衣

采用Z字型晾衣的方式，这样放得下更多衣服。

02 ) 袖子拉开吊挂

把袖子拉开前后吊挂，这样能让衣服完全展开，加速干燥。

▲ 图片来源/INS@0505_pic


03 ) 衣服摊开平放

像毛衣、针织这类的衣服，会因为衣服自身重量而被拉长，建议改用摊开平放的方式。

04 ) 连帽衣帽子撑开晾干

连帽衣的问题在于帽尖处不容易干燥，因此用这自带隔板的衣架，能把帽子撑开，衣服干的更快。

05 ) 外套、裤子挂成筒状

利用衣架隔开变成筒状，不让衣服湿湿的贴在一起。

▲ 图片来源/INS@lecottage_yokohama

▲ 图片来源/INS@kokoro_life

外套用这种宽版衣架撑开，让空气在内部顺利流通，有助加速干燥。

▲ 图片来源/INS@lecottage_yokohama

若家中没有这种宽版衣架，就用挂袜子的衣架，再利用晾衣夹撑开。

▲ 图片来源/INS@jigestagram

TIPS

晾在室内最好搭配除湿机，降低空气湿度，有助衣服晾干。
