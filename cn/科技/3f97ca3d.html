四核APU加持2GB独显！ThinkPad E475评测

2020-10-29
| 科技

作为 ThinkPad 品牌销量担当的 E 系列一直有着出色的性价比，今天的主角 ThinkPad E475 就是一台采用 AMD 四核处理器与 AMD Radeon 独立显卡组合的 14 英寸笔记本，标准的小黑外观设计配合具有强大性能的硬件组合给 E475 带来了更大的想象空间，伴随着 ThinkPad 25 周年的纪念，我们一起来看看这台配备了 APU 的 ThinkPad 有什么与众不同。

纯黑色商务风外观设计

在外观上，ThinkPad E475 继承了 ThinkPad 一贯沉稳低调的设计风格，通体的黑色设计充满了商务范，顶盖角落里银色的 ThinkPad 标识凸显了它的高贵血统，可发光的红色小红点让人轻松地了解它的实时运行状态。Lenovo 的标识则镶刻在机身左下角。

ThinkPad E475 采用了一块 14.0 英寸 1920×1080 分辨率屏幕，屏幕上方配有一枚 720P 摄像头和一组双阵列麦克风，方便在视频会议或是日常视频聊天中使用。

ThinkPad E475 延续了 ThinkPad 一贯舒适的小红点键盘，按键的间距较长，反馈速度很快，日常打字不会有任何压力，配合小红点在一定程度上完全可以替代鼠标。

ThinkPad E475 的机身右侧配有一个 USB 2.0 接口，一个 USB 3.0 接口，一个耳机/麦克风二合一接口，一个 SD 读卡器。

它的机身左侧配有一个电源接口，一个 RJ45 网络接口，一个 HDMI 接口，一个 USB 3.0 接口。

在轻薄本阵营中，ThinkPad E475 的接口配置绝对算得上是丰富，尤其是 RJ45 网络接口和 SD 读卡器的配置能够秒杀众多轻薄本。

APU+Radeon 独显带来更多想象

ThinkPad E475 有多种配置可选，我们拿到的机器搭载了 AMD A10-9600P 四核处理器，16GB DDR4 2133MHz 内存，256GB SSD，显卡为具有 2GB 显存的 AMD Radeon R5 M430 独立显卡。下面我们将进行一系列的测试。

CPU-Z 性能测试

AMD A10-9600P处理器采用四核心架构设计，主频 2.4GHz-3.3GHz，在 CPU-Z 的性能测试中，它的单线程分数为 157，多线程分数为 481。

显卡规格介绍

ThinkPad E475 也采用了双显卡的组合，A10-9600P 中集成了 Radeon R5 GPU，GPU 的最高频率为 720MHz。此外，ThinkPad E475 还配备了同样基于 AMD GCN 架构设计的R5 M430 独立显卡，与APU内置R5核显组成双显卡交火，交火后的显卡型号为Radeon R6 M435DX，拥有 320 个计算单元，核心频率 1030MHz，显存频率1000MHz，显存容量 2GB DDR3。

GPU 性能基准测试

性能方面使用 3DMark Cloud Gate 进行测试，其最终图形成绩为 8991，可以充分满足日常游戏需要。

鲁大师整机性能测试

在鲁大师的整机性能测试中，ThinkPad E475 的成绩为 79737，其中处理器性能为 23236 分，显卡性能为 15864 分，内存性能为 6744 分，磁盘性能为 10650 分，表现非常不错。

PCMark 8 家庭测试

在注重多媒体的家庭娱乐测试中，ThinkPad E475 得到了 2451 分。

PCMark 8 工作测试

在更偏向日常工作应用的 Work 测试中，ThinkPad E475 得到了 3371 分。两次测试充分体现了 ThinkPad E475 足以应对日常的应用操作。

游戏测试

本次游戏测试我们选择了主流的《英雄联盟》，将画质调至中等，关闭垂直同步功能，使用 Fraps 记录运行帧数。

在整个游戏运行过程中，ThinkPad E475 维持在 40 帧左右，最终的平均帧数为 47，即便是大规模团战场景也非常稳定，可以说是轻松应对。

除了性能，AMD 还有这两件黑武器

低负载的 4K 视频播放

APU 的一大优势在于它在解码 4K 视频时的超高效率，A10-9600P 支持 HEVC 硬件加速、AMD 完美图像和 AMD 视频稳定技术，在播放 4K 超清视频时 CPU 的使用率非常低，在保证流畅播放的同时还可以降低功耗，保证整机的续航时间。

我们使用迅雷影音来进行测试，开启内置的硬件加速功能播放著名的《美女吃烤鸭》4K 视频片段，迅雷影音的 CPU 占用率仅为 5.1%。

在播放《台北101》4K 视频时，迅雷影音的 CPU 占用率仅为 5.8%！在高清视频硬解码的测试中，得益于 AMD A10-9600P 出色的性能表现，ThinkPad E475 可以说是毫无压力。

VSR 带来更大视野

近年来随着 4K 技术的不断崛起，越来越多的消费者在购买电脑时也会考虑 2K 甚至 4K 屏幕的配置，但是超清屏幕笔记本的价格一直居高不下，这给预算有限的消费者带来了很多困扰。ThinkPad E475 虽然只配备了 FHD 分辨率的屏幕，但是用户可以通过内置的 VSR 功能一键将屏幕分辨率“升级为 2K”。

操作方法很简单，在 AMD Radeon 设置中点击“显示器”选项卡，接着将“超虚拟分辨率”功能打开，屏幕闪烁几次后即可在 Windows 的设置中找到更高的分辨率选项。

其原理也不难理解，此举并没有改变屏幕的物理分辨率，而是让系统将 4K 的图像以 4K 的标准输出到 FHD 屏幕当中，由于图像本身的清晰度变高了，所以在人眼视觉中图像就会变得更加清晰了。

4K 屏幕的好处有很多，比如显示的图像更加细腻，同一窗口下可以显示更多的内容、平铺更多的应用程序，同样的 Excel 表格可以一次性显示更多的行列内容等等，这都是 FHD 分辨率很难达到的效果。AMD 通过这一技术让普通分辨率的屏幕也能有高分屏的待遇，着实是“小钱办大事”的典范。

综合体验出色，同价位中值得考虑

作为面向主流价位的产品，ThinkPad E475 保持了 ThinkPad 品牌一贯沉稳的设计基调，纯黑色的外观是它的外表，但绝对不是它的全部，出色的键盘手感、丰富的接口配置共同奠定了它优质的使用感受。

AMD 四核 APU 与 Radeon 独显的加持更为 ThinkPad E475 带来了全新的性能感受，无论是日常游戏娱乐还是办公作业都游刃有余，AMD 出色的 4K 硬解码功能让流畅且节能的超清视频播放成为可能，让你能够更轻松、更简单地播放 4K 大片；VSR 更是以独特的思路解决了屏幕物理分辨率有限的问题，在 FHD 屏幕上就能够体验到媲美 4K 屏幕的使用体验，大幅度提高办公效率。总而言之，ThinkPad E475 是同价位中值得考虑的一款产品。■
