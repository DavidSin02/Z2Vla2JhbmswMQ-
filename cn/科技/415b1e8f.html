遍历

通过for ... in ...:的语法结构，我们可以遍历字符串、列表、元组、字典等数据结构。

1.1 遍历字典的所有key

>>> info = {'name': 'xiaoming', 'age': 20, 'hometown': '广州'}>>> for i in info.keys():... print(i)... nameagehometown>>>

1.2 遍历字典的左右value

>>> info = {'name': 'xiaoming', 'age': 20, 'hometown': '广州'}>>> for i in info.values():... print(i)... xiaoming20广州>>>

1.3 遍历字典元素

>>> info = {'name': 'xiaoming', 'age': 20, 'hometown': '广州'}>>> for i in info.items():... print(i)... ('name', 'xiaoming')('age', 20)('hometown', '广州')

1.4 遍历字典的key-value

>>> for key,value in info.items():... print(key,value)... name xiaomingage 20hometown 广州>>>

共有操作

2.1. 合并操作 +，

适用于字符串，列表，元祖

>>> str1 = 'hello'>>> str2 = 'world' >>> str1 + str2 # 字符串'helloworld'​>>> [1,2,3] + [4,5,6] # 列表[1, 2, 3, 4, 5, 6]​>>> ('a','b')+('c','d') # 元祖('a', 'b', 'c', 'd')

2.2. * 复制

表示倍数，适用于字符串、列表、元组

>>> str1 = 'hello'>>> str1*3'hellohellohello' # 字符串>>> [1,2,3]*3[1, 2, 3, 1, 2, 3, 1, 2, 3] # 列表>>> (1,2,3)*3(1, 2, 3, 1, 2, 3, 1, 2, 3) # 元祖

2.3. in 判断元素是否存在

适用于字符串、列表、元组字典

>>> 'my' in 'my name is lilei'True>>> 1 in [1,2,3]True>>> 'age' in {'name':'xiaoming','age':18}True
