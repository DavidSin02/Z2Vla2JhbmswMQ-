分身变永动的劫？拉克丝伤害变十倍！云顶之弈 10.5 版本前瞻

2020-10-29
| 科技

3 月 2 日，云顶之弈的设计师在外网上传了云顶之弈 10.5 版本的前瞻视频，我们游犀社区的游犀攻略君对其做了简单的翻译，小犀牛也在这里分享给大家～

以下内容由游犀攻略君翻译自云顶之弈设计师视频：

羁绊

【剑士】额外攻击触发机率：35% -> 40%

【雷霆】暴击或受到暴击时对敌方造成的伤害：80/250/500 -> 80/250/550

【极地】眩晕时造成的额外魔法伤害：75/175/350 -> 75/175/475

另外，云顶官方在透露版本前瞻时表示会对狂战士进行调整，但是设计师的视频并未提及，最终调整还是以实际官方公告为准。

英雄平衡性

1 阶

【黛安娜】 护盾持续时间：3s -> 4s，创造烈焰法球数： 3/4/5 -> 3/4/6

【蕾欧娜】 屏障持续时间：5s -> 4s

【婕拉】 初始 / 最大法力值： 0/75 -> 0/65 ， 召唤火焰喷射者的数目 2/3/4 -> 2/3/5

2 阶

【雷克塞】 狂怒之咬真实伤害： 250/550/850 -> 250/550/1000

【辛德拉】 法球伤害： 175/350/600 -> 225/375/650

【亚索】 初始 / 最大法力值：0/85 -> 0/90

3 阶

【蒙多医生】 毒云造成伤害： 50/100/150 -> 50/100/200

【伊泽瑞尔】 寒冰射击伤害： 225/250/900 -> 250/450/900

【卡尔玛】 鼓舞攻速： 40%/60%/80% -> 35%/50%/100%

【魔腾】 治疗生命值： 40%/45%/50% -> 50%/60%/70%

4 阶

【迦娜】 季风治疗值： 20%/30%/100% -> 25%/35%/200%

【图奇】 物理攻击伤害： 65 -> 70

5 阶

【劫】 分身法力消耗提高： 50/25/0 -> 50/25/-125

7 阶

【拉克丝】 终极闪光伤害： 550/900/9999 -> 550/900/99999

以上便是小犀牛为各位带来的云顶之弈 10.5 的版本前瞻了，不知道各位对于这个 10.5 版本是否满意呢？
