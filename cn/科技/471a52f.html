洗衣服再也不用心烦，小污渍也能迅速祛除，超全去污大法拿走不谢~

2020-10-29
| 科技

秋冬最烦的事情，洗衣服算一个，特别是小块的污渍，丢进洗衣机又怕洗不干净，手洗又太累，到底怎么办才好呢？

所以今天这篇文章给大家整理了超全去污大法，各种各样的污渍都不怕，随手就能高效去污，这份清洁干货请收好！

油渍污染

/ 土豆＋食盐 /

油 渍

前两天有条牛仔裤报废了，因为吃烧烤的时候被油溅了一裤腿，用了各种方法都没洗干净，最后居然被朋友拿了一个土豆轻轻松松就擦干净了。

先在污渍区域上撒上一些食盐，再拿一个土豆，切开土豆以后，用新鲜土豆的那一面擦拭污渍区。

最后用湿毛巾把污渍区域擦干净，晾干，油污就消失不见啦！土豆中含有淀粉，具有很强的污渍吸附能力，而且也不会对衣服材料有伤害，处理容易掉色或者面料比较娇贵的衣服的时候，可以使用这个方法哦！

/ 爽身粉 /

油 渍

当然还有一种超简单的方法，就是宝宝用的爽身粉啦！用爽身粉覆盖住油污区域，再用洗衣液搓洗，也可以消除污渍哦，不过这种方法比较适合刚蹭上的新鲜油污，那种很难祛除的油渍还是用第一种方法吧！

食物残渣

除了油污之外，“漏嘴巴”对掉落在身上的各种食物残渣如何处理也很有心得，这几个小妙招，必须要学会！

/ 小苏打 + 醋 + 肥皂液 /

酱料渍、果汁渍

平常爱吃各类炸酱面拌面的你，是不是偶尔也会出现这种不小心把酱料蹭到衣服上的尴尬情况呢？

这个时候用小苏打+醋就能解决，将小苏打完全覆盖污渍区域，静置5分钟左右。

最后用肥皂液和醋小心的擦拭污渍区域，一般小块的污渍用牙刷蘸点醋直接刷两遍就能搞定。

再按照正常程序洗一遍衣服就干净了，是不是跟新的一样呢！

/ 醋 + 洗洁精 /

番茄酱渍

吃薯条时番茄酱沾到衣服的人都知道，番茄酱特别难洗，很容易留下红红黄黄的印子，然后一件衣服又不能穿了。

其实番茄酱有个非常简单的处理方法，就是：醋+洗洁精。

用醋沾湿衣服上的污渍区，浸透几分钟以后涂上洗洁精，然后用小刷子轻轻擦拭一下，再按照正常的方式清洗就不会留下痕迹啦！

/ 防锈剂 /

酱料渍、油渍、锈渍

对于难洗的酱汁渍、染色的锈渍，推荐大家不妨试试专业防锈剂。一些电器设备的除锈剂，如WD40，也具有神奇的清洁效果哦~

不管是番茄酱还是甜面酱，在牛仔裤这种比较容易褪色的衣服上都不太适合酸洗，所以家里有这种防锈剂的可以拿出来用了，直接喷到污渍处，等待5分钟以后再照常丢进洗衣机清洗。

完全不会有褪色的痕迹，而且污渍也消失的干干净净。WD40非常适合用在纤维、尼龙等材质上，所以不如在家里备上一瓶试试看吧！

酒水污渍

/ 食盐 + 开水 /

红 酒 渍

对付酒渍，有一个超级好用的办法，3秒搞定，连洗都不用洗。

拿一个大一点碗，或者找一个能架起衣服的洗手台，在酒渍区域上撒点盐，然后在污渍处倒上开水，污渍就顺着水直接流下去了。

这个真的非常实用，不用整件衣服都过水，最后只用把浸湿的地方晒干，或者用吹风机吹干就可以再上身了。

/ 过氧化氢 + 肥皂液 /

红 酒 渍

家里有桌垫那种比较厚布料蹭到酒渍或者饮料渍好像也很难洗，布料本身就非常难揉搓，这个时候可以用过氧化氢+肥皂液试试。

蘸取过氧化氢和肥皂液混合的清洁剂，直接浸湿桌布上脏污的地方，等待五六分钟左右。

最后再丢到洗衣机里过水清洗，最后晾干就跟新的一样啦！

/ 白 醋 /

咖 啡 渍

喝咖啡的时候没拿稳，直接洒在了衣服上，最后把那件衣服手洗了两遍才弄干净，特别麻烦。

咖啡渍的克星就是白醋，直接用白醋浸泡一下污渍区域，然后过水清洗就可以了，这个方法真的减少了不少洗衣工作量！

/ 小苏打＋白醋 /

除 油 去 黄

对于颜色更深的污渍，还可以用小苏打+醋混合的方式，先把小苏打洒在污渍表面，然后再用小刷子蘸取白醋轻轻擦拭。

最后直接照常过水，一遍就能洗出原本颜色啦！

生活污渍

/ 阿司匹林 + 水 /

汗 渍

爱出汗的人，腋下也会经常残留微微泛黄的汗渍，别的颜色有时候也看不出来，但是如果爱穿浅色衣服的话就比较明显。

其实这种汗渍是最好解决的，把家里的阿司匹林碾碎了以后加水。

直接涂抹在洗不干净的汗渍区域上，静置5分钟左右，再拿去清洗。

汗渍毫无残留，白衣服也能跟新的一样啦！

/ 白 砂 糖 /

甲 油 污 渍

不管是什么年龄段的女人，都爱涂指甲油，但是指甲油瓶子太小了，不小心碰到就是灾难。

对付这种地板上斑斑点点的指甲油，其实有个很有用的江湖救急法，拿一勺白砂糖，覆盖住污渍区，等几分钟直接扫掉。

血泪经验告诉你们，千万不要试图用湿纸巾直接擦地板！会留下非常难清理的污渍，怎么都扣不掉的那种！

/ 醋 + 卸甲水 /

化 妆 品 污 渍

对于经常穿白衬衣的女生来说，衣领上总是会沾到很多污渍，特别是粉底液或者防晒霜这种不太好洗还防水的污渍，这个时候可以用卸甲油+稀释过的醋擦一擦。

基本上能够恢复如新，对化妆品污渍这个方法更有效，不用使劲揉搓就能洗的干干净净。

/ 发胶喷雾 /

口 红 污 渍

当然如果家里有头发定型喷雾的话，搞定这类化妆品污渍更简单，连清洗都不用。

把定型喷雾直接喷到污渍上，然后用湿抹布擦一擦，分分钟就干净了，这个救急也非常好用，特别是小范围的污渍，清洁效果立竿见影。
