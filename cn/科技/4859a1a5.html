前端开发：JavaScript 断言工具

2020-11-14
| 科技

此文为大家推荐几款JavaScript 断言工具，希望对大家有所帮助。

1、Sinon.JS（对 JavaScript 进行 spies、stubs 和 mock 测试。）

2、should.js（适用于 Node.js 的 BDD 式断言工具。）

3、chai(适用于 node.js 和浏览器的 BDD / TDD 断言框架，并能搭配其它测试框架使用。)

4、expect.js（简约的、适用于 Node.js 和浏览器端的 BDD 式断言工具。）

5、blanket（一个简单的代码覆盖率检测库。）

6、JSCover（是一个检测 JavaScript 程序代码覆盖率的工具。）
