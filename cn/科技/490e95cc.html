高速公路快充网络虽完善，但节假日还是别让纯电动车跑高速

2020-11-14
| 科技

决定续航里程的除了产品本身的性能技术之外，还有低温和高速行驶两大用车环境因素。气温过低的情况下，电池活性受影响，加上开暖气，特别费电，续航里程大幅度折损。

至于高速行驶，电池系统提供输出以“高放电”模式工作，续航里程大幅度折损。姑勿论电动车高速续航里程的折损，如果高速公路充电足够方便，纯电动车长途奔袭不是什么问题。

关键就在于高速公路充电设备的建设程度，据悉，为服务电动汽车用户城际出行， “九纵九横两环”高速公路为骨干网架的高速公路快充网络已建成。


加上以广东为基地的南方电网，纯电动汽车主销区域的高速公路快充网络已经相对完善了。近日，晓电籍着国庆假期上高速公路溜达了一圈，发现好像不是那回事儿。


//

电桩是在的，多半是坏的

//

其实，网络上关于高速公路快充的内容已经有很多，还整理为“充电攻略”，不少还是“亲身经历”。问题是，晓电看了这些“攻略”后发现，落差还是有的。

十一小长假从广州出发，向邻省福建行驶。第一个充电站正建设中，围蔽起来暂不能使用。第二个充电站，4个电桩3个故障。第三个充电站稍微好点，同样4个电桩，2个故障。

后面还有几个充电站，已建成但未投入使用。充电站之间平均相隔50公里左右，总共约200公里，如果急需充电补给的电动车早已趴窝在高速公路上等待拖车救援了。

还有就是，晓电进入福建省后不久，碰巧遇到一个蔚来汽车的换电站，有趣的是工作人员开着江淮电动车过去进行设备检修。


//

没有停车位谈何充电？

//

多个高速公路服务区的电动车充电站，要么无法使用（设备故障），要么还是无法使用（传统燃油车占用停车位）。前者可以加强人手对设备维修维护，后者则需要更多合理的指引和规范。

燃油车占位问题十分突出，让新能源车车主伤透脑筋。日常生活当中，由于燃油车“霸占”充电车位，新能源车主只能“干瞪眼、直跺脚”。高速公路充电站就更不用说了。

清明节假期，刚结束的五一小长假，十一假期进行时，高速公路免费通行，不少市民自驾出游。导致高速公路服务区成了大型停车场。

只见整个服务区的所有充电桩密密麻麻的被传统燃油车占满了，而且还有许多车子排着长龙等进服务区。虽然，急需充电的电动车车主有权让燃油车挪位腾出充电车位，问题是燃油车能挪到哪去呢？动都动不了！


//

最后说一句

//

随着的科技技术的发展，现在电动车的续航里程平均每半年可能会以50公里的速度在增长，国内NEDC综合续航里程超过500km的产品比比皆是。

NEDC综合续航里程达到600km，甚至700km，只是时间的问题，电动车六七百的续航能力已经达到传统燃油车的续航水平了。

既要马儿跑，又要马儿不吃草，这是不现实的。电动车的续航里程不断突破，充电设备的普及应用是至关重要的使用保障。

哪怕电动车续航里程不高，或者高速和低温的里程折损太大，只要充电足够便利，电动出行依然是最好的体验。
