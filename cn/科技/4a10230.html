一、离子反应与离子方程式


1．离子反应

(1)电解质在溶液中反应的实验探究

按下图所示完成实验，并回答问题：

①图甲实验的现象是无明显变化，因为溶液K＋、Cl－、Na＋、SO42—相互之间不发生化学反应。

②图乙实验的现象是有白色沉淀生成，其原因是溶液中Ba2＋与SO42—结合生成硫酸钡。

③结论：电解质在溶液中的反应实质是离子之间的反应。

(2)离子反应：有离子参加或有离子生成的化学反应。


2．离子反应方程式

(1)离子反应常用离子方程式来表示。离子方程式是用实际参加反应的离子符号来表示反应的式子。

(2)离子方程式的书写步骤

以氯化钡和硫酸钠在溶液中反应为例：

①写：化学方程式(必须根据实验事实)

BaCl2＋Na2SO4===BaSO4↓＋2NaCl；

②拆：把易溶于水、易电离的物质拆成离子的形式

2Na＋＋SO42—＋Ba2＋＋2Cl－===BaSO4↓＋2Na＋＋2Cl－；

③删：删去方程式两边不参加反应的离子

SO42—＋Ba2＋===BaSO4↓；

④查：检查方程式两边各元素的原子个数和电荷总数是否相等、反应条件、沉淀符号、气体符号等。

(3)书写下列反应的离子方程式：

①NaOH溶液与稀硫酸：OH－＋H＋===H2O；

②KOH溶液与稀盐酸：OH－＋H＋===H2O；

③BaCl2溶液与稀硫酸溶液：Ba2＋＋SO42—===BaSO4↓；

④Ba(NO3)2溶液与硫酸钠溶液：Ba2＋＋SO42—===BaSO4↓；

⑤碳酸钠与稀硫酸：CO32—＋2H＋===CO2↑＋H2O

⑥碳酸钙与稀盐酸：CaCO3＋2H＋===Ca2＋＋CO2↑＋H2O。


【归纳总结】

(1)书写步骤：

“写(基础)→拆(关键)→删(途径)→查(保证)”。

(2)书写关键：

①要将易溶于水、易电离的电解质写成离子的形式；

②单质、沉淀、气体、难电离的物质(如水、弱酸、弱碱)等写成化学式。


【深度思考】

(1)以H＋＋OH－===H2O为例说明离子方程式表示的意义是什么？

提示　①表示某一个具体的化学反应，如NaOH＋HCl===NaCl＋H2O。

②表示同一类反应，如：强酸与可溶性强碱生成水和可溶性盐的反应。

(2)①酸碱中和反应的离子方程式都是H＋＋OH－===H2O吗？

②H＋＋OH－===H2O仅表示酸碱中和反应吗？

提示　①硫酸与氢氧化钡溶液反应的离子方程式为：2H＋＋SO42—＋Ba2＋＋2OH－===BaSO4↓＋2H2O。

②硫酸氢钠(NaHSO4)溶液与氢氧化钠溶液反应的离子方程式为：H＋＋OH－===H2O。

(3)判断下列反应的离子方程式正误。

①稀硫酸滴在铜片上：Cu＋2H＋===Cu2＋＋H2↑(　　)

②稀硫酸与氢氧化钡溶液混合：SO42—＋Ba2＋===BaSO4↓(　　)

③稀硝酸滴在大理石上：CaCO3＋2H＋===Ca2＋＋H2CO3(　　)

④氧化铁与稀盐酸混合：Fe2O3＋6H＋===2Fe3＋＋3H2O(　　)

⑤氢氧化铜和稀盐酸反应：OH－＋H＋===H2O(　　)

⑥镁和硝酸银溶液反应：Mg＋Ag＋===Mg2＋＋Ag(　　)

提示　①×　②×　③×　④√　⑤×　⑥×

解析　①Cu的活泼性小于H2的活泼性，Cu不能和稀硫酸反应生成H2，故①错误；②稀H2SO4和Ba(OH)2溶液混合，除SO42—和Ba2＋反应，还有H＋和OH－反应，离子方程式为：2H＋＋SO42—＋Ba2＋＋2OH－===BaSO4↓＋2H2O，故②错误；③反应生成的H2CO3不稳定，分解成CO2和水，故③错误；④Fe2O3为氧化物，在离子方程式中不能拆分为离子，因此④正确；⑤Cu(OH)2在离子反应中，不能拆分为离子，应为Cu(OH)2＋2H＋===Cu2＋＋2H2O；⑥中离子反应，电荷不守恒，应为Mg＋2Ag＋===2Ag＋Mg2＋。


【归纳总结】

离子方程式正误判断——“五看”

(1)看是否符合客观事实。

(2)看拆分是否正确。

(3)看是否符合原子守恒和电荷守恒。

(4)看是否漏写离子反应。

(5)看是否符合题设条件。


二、离子反应发生的条件


1．实验探究

(1)上述三个实验的现象：①生成蓝色沉淀；

②溶液红色消失；③产生气泡。

(2)写出上述三个实验中反应的离子方程式：

①Cu2＋＋2OH－===Cu(OH)2↓；

②H＋＋OH－===H2O；

③CO32—＋2H＋===CO2↑＋H2O。


2．复分解型离子反应发生的条件


3．离子共存

(1)几种离子在溶液中能否大量共存，取决于它们之间是否发生反应。

(2)在溶液中若离子之间不发生反应，则能大量共存；否则不能大量共存。

(3)溶液中离子不能大量共存的判断

①离子间反应生成沉淀：如Ca2＋与CO32—、Ag＋与Cl－、Mg2＋与OH－等。

②离子间反应生成气体：如H＋与CO32—、H＋与HCO3—等。

③离子间反应生成难电离的物质：如H＋与OH－、H＋与CH3COO－、OH－与NH4+等。

(1)离子反应有复分解反应和置换反应等多种类型。判断下列置换型离子反应能否发生，若能，写出相应的离子方程式。

②否　③能；Fe＋Cu2＋===Fe2＋＋Cu

②含大量Ca2＋的溶液中，SO42—可以大量共存(　　)
