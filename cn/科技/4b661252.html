dos快捷键教程，windows控制台程序，cmd命令行窗口命令快捷键

2020-10-29
| 科技

大家好，我是老盖，首先感谢观看本文，本篇文章做的有视频，视频讲述的比较详细，也可以看我发布的视频。

今天我们学习dos里边比较常用的几个快捷键，F3 F7 ↑↓ 键 Tab 键 Ctrl + C。

先介绍一下F7这个命令，F7能调出以往输入的命令，我们先随便输入几个命令dir cls cd..，然后按f7，就会出现以前输入的命令菜单，就可以方便的根据菜单中的命令直接选择后回车运行。

F3，这个快捷键是调出上一个输入的命令。

上下箭头 ↑↓ 键快捷键，可以上下调出以前的命令。

我们再看一下ctrl + c这个快捷键，它可以中断运行的DOS命令，比如我用ping 192.168.1.1 -t，它会一直在执行这个命令，如果想中断的话，可以使用ctrl + c 就可以中断这个命令。

在介绍一下tab这个快捷键，它可以补全单词，在当前目录中有个文件夹叫Administrator。

我输入cd admin，然后按tab键，它会自动补全这个单词英文文件夹。

再举一个tab自动切换补全例子，比如当前目录有三个文件夹。

PerfLogs

Program Files

Program Files (x86)

我输入cd p。

按一个tab键，第一次会自动匹配第一个PerfLogs文件夹。

再次按tab会自动匹配第二个Program Files文件夹。

第三次按tab键会匹配第三个Program Files (x86)文件夹。

我们再按一下f7，这样能看到刚才输入的几个命令全部都在菜单中方便使用。

今天学的这几个dos快捷键我们经常会用到，大家可以多练习一下。
