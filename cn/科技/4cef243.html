最新Linux 除了能够伪装成Win 10，还能让手机变桌上电脑？

2020-10-29
| 科技

Kali Linux官网宣布了2019年第四个也是最后一个版本——Kali Linux 2019.4。相比之前有较大的变动，增加了很多全新的模式，例如全新的桌面环境Xfce、Kali Undercover模式（能够伪装成win10）、NetHunter Kex 。


对Gnome说再见，迎接 Xfce

在最新的版本中， Kali完全舍弃了Gnome桌面而转向了Xfce。该最新迭代用 Xfce 替换了其主要桌面环境 Gnome。官方认为，Gnome虽然功能很强大，有很多非常棒的特性，但对于大部分Kali用户而言，要想的只是能够一次运行多个终端窗口的窗口管理器和一个网页浏览器。再见 Gnome，你好 Xfce

其次，Kali需要支持从低端到高端不同的硬件设备，包括树莓派。Kali致力于轻量级的解决方案，希望能够在不同的设备上都尽可能提供一致的使用体验。基于这些考虑，Gnome则显得有些臃肿，故而选择更为轻量的Xfce作为默认桌面环境。

但同时Kali Linux团队也表示：

“我们对用户界面的更新感到非常兴奋，我们认为您会喜欢它的。但是，由于用户界面变化巨大，如果您不想离开Gnome，请不要担心。我们仍然有一个Gnome为您量身打造的产品，已经进行了一些更改。随着时间的流逝，我们将对所有桌面环境进行更改。”


新的 Xfce 用户界面 (UI) 是可运行于各层级 Kali 安装上的轻量级 UI。其功用在于无需变动即可处理普通用户的各种需求。该界面采用大众熟悉的标准 UI 概念以确保毫无学习曲线。且其现代 UI 元素充分运用屏幕空间，外观十分精美。


将Linux 变成成Win 10

官方制作了一个“ Kali Undercover”模式，能够快速将默认主题切换为类似Windows 10桌面环境，非常逼真，这样可以放心的“工作”。再次运行脚本就可以重新切换回之前的主题。

该模式主要是为了将Xfce桌面环境转变为Windows外观的桌面而设计的，其目的是为那些在公共场所和不想让人们说他们使用的是Linux发行版，而是Windows操作系统的人们设计。由于多数人经常看到的是 Windows 和 macOS 设备，因此如果在办公室走廊或其它公共场所看到有人在使用带有标志性的龙图标的 Kali Linux 可能会引起猜疑。


装进口袋的全功能ARM 台式机

Kali Linux推出了一个重磅功能：Android手机上完整的Kali桌面——NetHunter Kex
名为 NetHunter Kex 的新功能使用户可在 HDMI 输出上连接安卓设备，还可使用蓝牙键盘和鼠标，在安卓智能手机上获得 “完整无删减的 Kali 桌面”。这其实有点类似现在智能手机上内置的桌面模式，提供轻量级的电脑工作模式。

Kali Linux 团队称：只要手机性能够强大，这就好像一台恰好能装进口袋的全功能便携式 ARM 台式机。


专注在“容错”的BTRFS

Kali Linux 文档中新增将 BTRFS 用作根文件系统的内容，赋予用户在升级后回滚文件系统的能力。Kali Linux允许用户在安装过程中将Btrfs文件系统用于根分区。

BTRFS 是个写入时复制 (CoW) 的文件系统，旨在解决 Linux 文件系统中缺乏缓冲、快照、校验和及集成多设备跨越的问题。各大公司均对其代码库有所贡献，包括 Facebook、Red Hat、Oracle SUSE 等。

在虚拟机里尝试新东西的时候，往往会做个快照，万一问题时可以简单回滚到已知良好状态。但在裸机上运行 Kali 就没那么容易做到这一点了。你要么超级小心谨慎，要么一旦出问题就陷入无尽的手动清理中。但有了 BTRFS，即便在裸机安装上都可享有同样的快照功能！

可安装Powershell

PowerShell 是跨平台自动化与配置工具，优化处理结构化数据（如 JSON、CSV、XML 等）、REST API 和对象模型，包含一个命令行 shell、一个关联脚本语言和一个处理 cmdlet 的框架。Kali 官方 git 库已于 2019 年 6 月置于 GitLab 上。

在今年11月份的更新中，powershell已添加到主存储库中，现在只需要运行：
apt update && apt -y install powershell
就能让你的系统拥有强大的PowerShell。


​

Kali Linux是一个高级渗透测试和安全审计Linux发行版。作为使用者，你可以把它理解为一个特殊的Linux发行版，集成了精心挑选的渗透测试和安全审计的工具，供渗透测试和安全设计人员使用，也可称之为平台或者框架。

Kali Linux自带安全工具集，它将所带的工具集划分为十四个大类，这些大类中，很多工具会重复出现，因为这些工具同时具有多种功能，比如nmap既能作为信息搜集工具也能作为漏洞探测工具。另外，除了这些系统推荐的工具，我们也可以自行添加新的工具源，丰富工具集。

Kali linux可以说是网络安全人员的专用系统，它是BackTrack系列的升级版。

听过一个笑话：当别人问你为什么要使用Linux时，一个很好的理由是当有人要求你去他家里修理电脑并且你不知道问题的原因时，这时候你就借口说你不使用Windows。你不了解Windows而无法修复它。
