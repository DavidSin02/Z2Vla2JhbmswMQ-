1.CLO 3D

1.1关于CLO的历史

CLO 软件的雏形源于一位对面料模拟技术充满热情的计算机博士Jaden。他在校期间开发了这款软件的第一个原型。当看到学生们都开心地通过该程序制作3D服装时，Jaden决定将其公开。


软件公开后，在最早一批软件使用者中，有一位cosplay艺术家敏锐地捕捉到了这款软件的潜力，认为这款软件会给虚拟cosplay艺术带来巨大的改变，并建议为此建立一个社群。很快，这款软件便从这个由超过200万虚拟cosplay艺术家的社群中收获到了巨大的肯定。2009年，针对这个群体的软件应运而生，这就是第一代的Marvelous Designer。随后在2011年，由Marvelous Designer制作了第一部电影《丁丁历险记》。


在CG和游戏行业不断发展的同时，这家公司也并没有忘记自己最初的目标——开发一个专门针对服装行业的程序。2010年，在结合了无数设计师和打版师的意见之后，第一代CLO正式发布。CLO的开发是为了更加精准地实现虚拟服装在外观和功能上与物理服装的无缝转换。


经过10年的发展，这两个程序都越来越成熟，Marvelous Designer已经更新到了版本8.0，而CLO也更新到了版本5.0。如今CLO更是推出了CLO-SET和Benefit by CLO这两个平台，来进一步完善虚拟服装这一市场。


1.2 CLO的功能

说到CLO和Marvelous Designer的区别，CLO如是答道：“虽然两者的核心技术相同，但Marvelous Designer主要用于CG、动画和游戏行业。而CLO更倾向于时尚和服装行业，它可以应用于服装设计开发过程的各个阶段。这两者的主要区别在于CLO导入和导出的DXF-AAMA或DXF-ASTM 文件可在生产时发送给生产厂商。”


AAMA和ASTM其实都是DXF文件，只不过是适配不同软件的DXF文件。我们也许知道，只要是DXF文件就可以导入修改。我最开始接触的打版软件是PAD，当时并不知道如何在CLO中打版，在这两个软件中不断使用切换的过程中我才发现，原来DXF文档可以直接在适配软件中通用，至少和PAD的转换中是非常成功的，其他的打版软件我没有实战经历，但应该也问题不大。


CLO的一大特点就是可以流畅地完成3D打版和2D打版之间的转换。设计师，打版师既可以导入已有的DXF文档对其进行修改，也可以直接在CLO中从零起草新的平面版型，还可以在虚拟模特上完成立体裁剪，而3D和2D的同步呈现也使得整个过程更为高效。如果真的要从操作上来讲，我还是更喜欢PAD，所以一开始有点不愿意放弃，很执着地使用PAD。后来学习CLO打版之后发现，虽然从打版上来说有一点点逊色（也可能是我个人的使用习惯造成的），但是它能2D3D实时同步这一点真的很方便。


CLO的特点包括无限设计，实时呈现，更加精准的属性以及更加简便的流程。


2.Style 3D

Style 3D是浙江凌迪数字科技有限公司（原上海凌笛数码科技有限公司）开发的数字化服务平台，可以连接面料供应商、设计部门和合作客户。通过面料数字化、服装建模和远程推款，实现快速开发打样和审版。

图片来源：Style 3D微信公众号


1.面料数字化


下面介绍三种获得面料贴图的方法：


Step1:

选择合适大小的面料样品，保证面料扫描之后的花型纹理。


Step2：

A.使用专业的面料扫描仪，可以完整获得面料的纹理贴图和法线贴图，效果逼真，但仪器价格昂贵，不易养护（参考官方面料扫描介绍）。

B.使用日常办公的彩色平板扫描仪（要求扫描精度不低于300dpi）获得面料的表面纹理，成本低，但面料仿真效果不好（参考官方面料扫描介绍）。

C.手机拍照，获得面料表面纹理。使用方便，但效果差，容易产生色差且面料丝缕方向不正，有一定透明度的面料无法塑造透明效果；


这个扫描过程很快，整体不会花很多时间，上传平台后大家就可以一起使用了。在第二步，我们公司主要用A方法，前几天我去我们公司设计部体验过面料扫描仪，扫描出来的效果的确清晰，但在扫描的时候，面料不能拉伸，容易造成纹理变形，丝缕不正。


而C方法手机拍照，在我之前做作业时，就遇到手边刚好有一块面料可以用于我正在做的模型，就自己用手机拍照试了一下。用这种方法，如果把方向找对，纹理拍清楚，效果也是不错的。


2.服装建模


服装建模主要分为三个步骤：

Step1：在传统平面制版软件中绘制纸样，导出DXF格式文件，纸样导出时去除缝头，只留净纸样。Style 3D不能直接打版，需要先用其他打版软件做好，然后导出DXF。导出的时候一定要去掉缝头。


Step2：在Style 3D建模软件中打开DXF格式文件，选择与待建模服装尺寸匹配的模特，缝制纸样，通过模拟状态下的拉扯调整褶皱效果，然后添加面料、印花图案、明线、纽扣、拉链等。添加缝线后，为了拉出符合实际效果的褶皱，对模型做各种拉扯的过程很花时间。在这一步，我个人经验是用好软件的冷冻、硬化、失效还有固定针等功能。


部件模型做好以后，如果还要修改其他地方，一定要先冷冻再继续模拟，不然后果很崩溃。譬如我们刚开始学习软件时，做的一件戴帽卫衣，当时用了固定针做帽子的造型。刚开始我们经常会忘记冷冻，忘记退出模拟，就把固定针取了，结果就是好不容易做好的帽子造型全塌。


Step3 保存文件，并上传至Style 3D网络平台，备注好内部编号、服装分类、创建人等信息。


注：这里也可以根据设计部门的人员职能，选择在Style 3D中制作白坯上传至云端，设计者或客人从网络平台进入，添加面料、印花图案等细节。

3.远程推款

远程推款可以根据客人是否有平台账号来选择不同方法：


A.客人有Style 3D网络平台账号，可直接进入平台查看建模，并自行改变颜色、面料、印花等，查看不同的效果。查看后若有问题可直接在平台上留言，提出修改要求，由建模人员线下修改后再上传。


B.客人没有Style 3D网络平台账号，可由建模人员做好模型以后，导出不同角度的图片或者gif格式的动图，发送给客人查看。动图会360度旋转，旋转速度在导出时可以自己调整。

二、3D软件带来的便利

1.新品开发

传统的新品开发，在大货开裁之前需要经过头样到船样的多个打样修改的环节。中间涉及面辅料、印绣花的跟进，会出现某个环节跟不上的情况。外贸企业打样中须向客人寄样衣，等待客人收到样衣后给出修改意见，耗时久。


对于印花，在模拟时因为可以调整大小和位置，所以可以帮助用户更好地判断自己想要的比例。对于刺绣类更加昂贵的工艺，这样的模拟更可以减少因不确定而引发的浪费。我们部门打样的服装有很多印绣花，经常听到业务员吐槽印花的花型开版费用，而且有时候印绣花还会出问题，譬如可能颜色印错、方向印错，从而影响之后的打样进度。前一段时间我们部门接了一个几百件的小订单，花片裁下以后送去印花，印回来发现方向不对，跟单要求全部再裁，重印。但是这些问题，使用软件都可以避免，因为在软件里我们就可以改变印花的效果。

图片来源：庄主使用课程资料自制


以上三张图的图案效果从上至下，分别是消光、丝绸和金属。


相比传统样品开发，3D软件速度更快，成本更低。纸样做好后，不需等待面辅料，不需裁剪，就可直接在虚拟模特上模拟。在纸样和数字化面辅料完成的情况下，简单款式半小时甚至十几分钟就可以完成；复杂样衣（例如羽绒服）熟练工半天可完成，新手两天左右。建模完成后不需寄快件，直接向客人发送3D模型，客人和设计师、版师可实现实时沟通，有问题直接修改，不需多次做修改样，省时省力。

2.线上批版（有Style 3D的账号）

上面提到过Style 3D将制作好的模型上传后，客人可以根据自己的喜好，在面料库和图案库里下载喜欢的面料或图案，然后填充在模型上做并查看效果。并且可以直接在平台上做查看后的留言，对局部不满意，也可以添加热点后留言。对比传统的寄样衣，等修改意见，互动的速度加快很多。


3.一版多用

我们在做样品开发时，经常会遇到一个款式需要做不同颜色、不同面料的多个样衣，供客人选择。我目前有三个大批推销样的单子，都是一套纸样，做多个颜色的样衣，每个颜色的数量几件到十几件不等（同尺码）。而使用3D数字样衣，只需要建模一次，在做添加面料时，修改颜色，即可看到不同颜色的效果。这里还可以添加热点，直接指出具体哪个地方有什么问题，做什么修改。（点击3D样衣建模查看）


这个链接是在Style3D的微信公众号找到的。通过这个链接，大家可以体验一下在同一个版型上进行不同的面料切换。


除了制作整件样衣，3D软件也可以把一件衣服拆分成领、袖、大身等几个部分，做部件的建模。将这些部件拼合在一起，就是一件完整的样衣。

1.虚拟模特

1.1 Style 3D

目前根据我自己的使用体验来看，Style 3D的虚拟模特还存在以下一些亟待改善的问题：

A.软件自带模特数量有限，只有女165、男175、130男女大童和90男女小童，若没有定制新模特，所能做的数字样衣尺码有限。对于大童的样衣，在最初打样时，我们公司按照140的尺码制版做样衣，做3D建模时，打样的尺码和模特尺寸不符，建模就没有用。


B.软件自带模特尺寸与实际人体尺寸有差异。在最初我用公司自己的样板做模型时，发现上衣的袖子袖口和裤子脚踝处总会堆积很多衣褶，之后点开虚拟模特的尺寸详情，臂长等一些长度尺寸对比实际人体尺寸都短几公分，造成建模效果差。


关于这个问题，刚开始的时候，我以为是我们尺寸的问题。后来专门上天猫找同类型的服装尺寸，对比的结果是，模特尺寸与实际人体尺寸存在差异。针对以上两个问题，我们曾咨询过凌迪的老师，得到的回复是，要么按照软件已有模特尺寸制版，要么出钱做新的虚拟模特。但虚拟模特的建模成本高，据说3000元一个。Style 3D的网络平台上也有人台库， 但目前官方人台库的模特无法查看详细的尺寸信息。


对于以上两个问题，我们公司目前的应对办法如下：


A.大童样衣打样时按照140打样，然后推码，得出130码的纸样，然后再做数字样衣；

B.明确做数字化样衣的目的是追求3D模型的效果，在不改变结构的情况下，可以调整纸样尺寸，保证数字样衣的穿着效果。


如此一来，我们所看到的数字样衣并不是尺寸正确、符合客人标准的样衣。客人看中样衣下单后，还需调整纸样至符合目标消费人群的尺寸。个人觉得，这种做法不仅未达到3D建模改变传统生产流程、加快生产速度的目的，反而多增加了一个环节，给自己增加工作量。


1.2 CLO的虚拟模特

而上述提到的问题可以在CLO软件中很好的避免。

在CLO中，用户可以根据自己的需求调整虚拟模特，下图是来自于CLO编辑虚拟模特的截图。我们可以看到这上面的参数非常详细，基本上制衣需要的数据都有。不过CLO最开始默认的模特设置和真人不太符合，除了自己调整之外，CLO SET里也有更接近现实比例的人台出售，价格是30美金一个。

图片来源：庄主自制


CLO有一个很有用的功能是压力模拟，当一个版制作好穿在虚拟模特身上之后，该功能会提醒用户哪些地方太紧或者根本不能穿。


下面是我之前学习3D立体裁剪时做的一个小练习：当这件紧身小背心选择较硬的牛仔面料时，大家可以看到红色部分是“不能穿”需要调整的部分，以及黄色部分为“过紧部分”。我做的一个简单调整是把面料换成了针织的，可以看到换了面料之后，因为针织面料的弹性，不可穿部分也随机消失。小范围是可以用这种方法调整模特尺寸，但调整数值太大会造成模特走形。但从我知道的例子来看，在CLO上修改尺寸后，暂时还没遇见过这样的问题。CLO上还可以直接改变面料类型，改变面料后压力也随之改变。

2.面辅料的仿真效果

2.1 Style 3D的数字面料

面料的扫描结果是一张图片（纹理贴图和法线贴图），格式要求jpg、png或jpeg。给数字样衣添加面料，其实就是把图片附到纸样表面。因此，在做模型时，软件里的版片是没有模拟面料性能的。虽然有经、纬向强度等参数调节，但这些参数的调节依赖个人对面料性能的掌握。


我们公司主要做针织类服装，休闲类还好，但合体的服装，例如贴体的内衣，在打样时是不收省的，依靠针织面料的自身弹性达到上身的合体效果。因此，在软件中建模时，如果用未经修改的纸样，袖窿处就会有多余的量，与实际样衣不符。

图片来源：庄主自制


这件吊带是我之前做的模型，原样是针织的面料，紧身型，穿在身上的时候，因为针织面料的弹性，胸部会很服帖。但在3D软件中，如果没有修改纸样，是达不到贴体的效果的。针对这个问题，做样板的时候注意放量即可。


在软件中，面料除了纹理贴图，还有法线贴图。法线的作用是加深面料的模拟纹路，使得模拟效果更逼真。但是法线贴图目前只能用专业的面料扫描仪才能获得，我们公司用的仪器是爱普生。

3.虚拟市场和现实的结合

在CLO的例子中，它一直在完善着自己的虚拟市场，虽然呈现的效果和未来潜力都令人激动，不过目前不管是CLO SET还是Benefit by CLO都还有待更多的商品进行填充。目前CLO SET能买到的虚拟面料，虽然从参数和功能效果上也一定是符合现实的，但如果真的要使用，却很难获得百分百相同的面料。而购买来的面料，也会因为缺乏具体的参数信息，从而导致可能无法完全在软件中重现。如果能打开虚拟和现实面料市场的通道，对于用户来说是极大的福利。


4.关于学习

最后想分享的这点虽然算不上面临的问题，但也是客观存在的事实。虽然3D软件的操作已经尽量简化，在一些细微的调整上是对所有人都很友好的，但是要从零开始制作出好的设计和版型，也还是需要非常强的打版知识储备，以及对于电脑知识的运用。


就CLO的例子来讲，如果想要开发设计一个全新的无限接近实物的样品，整个过程也是需要投入大量的时间（但却是非常节约材料和空间），对电脑配置也是有相应的要求。另外，虽然在CLO中可以模拟出很多辅料，但它也不是什么都可以做，比如一些特别独特的辅料细节也需要运用到其他软件来配合。此外，style3D目前不支持A卡，想要学习的朋友们请注意。

庄主总结

一、服装行业现有3D软件概况介绍

1.CLO 3D

（1）关于CLO的历史

Clo 软件的雏形源自于一位对面料模拟技术充满热情的计算机博士Jaden。

（2） CLO的功能

无限设计，实时呈现，更加精准的属性以及更加简便的流程。

2.Style 3D

Style 3D是浙江凌迪数字科技有限公司开发的数字化数字化服务平台，连接面料供应商、设计部门和合作客户，通过面料数字化、服装建模和远程推款，实现快速开发打样和审版。


二、3D软件带来的便利

1.新品开发

相比传统样品开发，3D软件速度更快，成本更低。纸样做好后，不需等待面辅料，不需裁剪，可直接在虚拟模特上模拟。

2.线上批版

Style 3D将制作好的模型上传以后，客人可以根据自己的喜好在面料库和图案库里下载喜欢的面料或图案在模型上做填充。

3.一版多用

使用3D数字样衣，只需要建模一次，在做添加面料时，修改颜色，即可看到不同颜色的效果。

4. CLO相关实例

（1）CLO SET

CLO SET是一个实现全供应链沟通的平台，全面连接产业链中各种角色，用户可以在上面完成数字采购，同时在服务器上生成3D图像。

（2） Benefit by CLO

Benefit by CLO 是一个在线的虚拟试衣平台。


三、3D在现阶段的应用中存在什么问题

1.虚拟模特

（1）Style 3D的局限

A.软件自带模特数量有限，只有女165、男175、130男女大童和90男女小童。

B.软件自带模特尺寸与实际人体尺寸有差异。

（2）CLO 3D

在CLO中，用户可以根据自己的需求调整虚拟模特，除了自己调整之外，CLO SET里也有更接近现实比例的人台出售，价格是30美金一个。

2.面辅料的仿真效果

（1）Style 3D数字面料

面料的扫描结果是一张图片（纹理贴图和法线贴图），格式要求jpg、png或jpeg，给数字样衣添加面料，其实就是把图片附到版片表面。在软件中，面料除了纹理贴图，还有法线贴图。法线的作用是加深面料的模拟纹路，使得模拟效果更逼真。

（2）CLO数字面料

对于CLO的数字面料，软件中已经自带了比较全面的面料库，并且提供了详细的可调整的参数。因此用户可以参考自己想要的面料，在软件中重现出这样的面料。不仅是平面面料的纹路，还包括可以重现羽毛，珠绣，刺绣等。另外在CLO SET里也会不定期的跟新新的面料，大家觉得自己调太麻烦也可以选择直接购买。
