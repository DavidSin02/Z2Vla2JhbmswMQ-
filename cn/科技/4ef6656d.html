武义90后女孩成功出道，带火了家里的炸串店！网友：这周就去吃

2020-11-14
| 科技

以前，老沈看到甜甜的女孩子都会说：“你吃可爱多长大的吧”。

但是现在，老沈觉得甜甜的女孩子都像小彩虹徐梦洁一样，是吃根建烧烤长大的！

徐梦洁爸爸妈妈经营了十多年的根建烧烤冷饮，在武义当地早就小有名气。

但是创造101播出后，武义吃货们私藏这家的烧烤小店再次人气暴涨。微博、豆瓣......全是网友们探店打开后的repo。最关键的是，大家都表示：根建烧烤真的很好吃啊！

那老沈可就坐不住了，连忙买票出发去武义！下面吃货老沈就放上新鲜出炉的探店攻略，前排说一句，小彩虹的爸爸妈妈真的是hin可爱没错了~

交通指南

根建烧烤所在的武义县位于浙江金华，从杭州出发有高铁直达。票价77元，一个多小时就能到。

老沈想了想，从滨江出发到城西也要这么久，对杭州上班族来说，用周末的时间出游完全足够。

武义北站下车后，建议直接打车去根建烧烤。根建烧烤开在熟溪北路上，是武义县比较繁华的地段。

送我们过去的出租车司机是武义当地人，听说我们是去根建烧烤，告诉我们一定要点藕饼（就是徐梦洁吐槽包起来很难受的那个藕饼）。

司机还让我们问问老板是不是要涨价。在这里老沈先替彩虹爸带句话，根建烧烤不会涨价，大家放心大胆地点！

我们12点45分从杭州出发，到根建烧烤才2点多。但是店里已经陆续来了客人，不出半个小时，就没什么座位了。

彩虹爸说，以前都是下午4点多营业，现在好多客人都是从上海、杭州过来的，得赶下午的车回去，所以就改成2点多营业，不能让人家白跑。

下午三点左右，店里就坐满了。

推荐菜品

走进店里，彩虹爸妈已经忙开了。根建烧烤更像是炸串，一口大锅热着油，彩虹爸一个人负责炸，忙得不可开交。

问他为什么不找个人来帮忙，他说自家的炸串好吃是因为火候掌握得比较好，哪个串先下，哪个串后下都有讲究。这个技术没有十几年练不出来，只能自己上。

店里炸串的品种不多，几乎没有素菜。点菜的方式也很复古，所有食材就放在台面上，要吃什么直接跟老板说。碰到人多到时候，就要比谁嗓门大了……

老沈点了两份鸡爪、两串藕饼、两份香肠和一串芋头。炸中翅、臭豆腐、螺蛳肉等没点上，留着下次再吃。

/ 鸡爪 /

小彩虹在节目里说过，她们家的炸鸡爪是先卤后炸。鸡爪上桌后，油光锃亮的色泽就让人食欲大增。

鸡爪完全入味，口感酥烂，不用动手就能剔下肉。酱料是秘制的，店里大部分的炸串都用同一种酱料，甜辣鲜香。觉得味道不够重的话，还可以蘸一下大蒜叶做的绿色调味酱。

/ 藕饼 /

刚到店里的时候老沈就看到边上坐着三个女孩子在包藕饼（往藕片里填肉馅）。彩虹爸爆料，徐梦洁十多岁就开始帮店里包藕饼，每次包藕饼都要包一大盆，所以才“包得难受”。

藕饼表皮炸得酥脆，内里又很软糯，肉馅填得满满当当。虽然炸串，但一点都不油。只是用料太大方了，暴风式扫完一整盘藕饼，竟然有点饱！

/ 香肠 /

不知道是不是童年滤镜加成，这盘炸香肠真的深得我心。香肠炸完后酥香可口，刷了一层薄薄的甜面酱，完全就是以前放学后最爱吃的那种油炸点心！

/ 炸翅中 /

炸翅中是坐在旁边的小姐姐推荐的，她专程从上海过来打卡。翅中炸得酥软，没有撒很多调味料，一口咬破表皮，带着鲜香的热浪冲到鼻尖，很过瘾了。

必点冷饮

彩虹爸一边忙着炸串一边招呼我们坐下来喝点冷饮。根建烧烤的全名是根建冷饮烧烤。别看炸串的品类不多，冷饮的种类可是多到挑花眼！

有银耳汤、绿豆汤、橘子糖水这些常见的，也有清凉糕、青草腐之类比较有地方特色的。

老沈数了数大概有16种可以选，价格在7元到8元一碗不等。老沈横扫9碗，最后Pick了下面几种。

/ 清凉糕 /

清凉糕是金华地区的特色冷饮，冲着“清凉糕”这个名字，老沈立马就要了一碗。

一碗清凉糕里，糕体软糯Q弹，入口甘甜，糖水部分好像加入了薄荷，入口丝丝清凉。老沈原本想尝一下拍个照的，结果一口气全吃完了……

后来有朋友告诉我，金华本地人吃清凉糕喜欢往里面加一点点醋，口感和降暑效果都会更好。

/ 青草腐 /

青草腐用仙草做底，撒了芝麻糖粉和薄荷。

整碗冷饮的加分点就在芝麻糖粉上，芝麻香混着青草香，和甜品店里烧仙草的口感完全不同，清爽不腻。

/ 白药酒 /

彩虹妈妈倾力推荐的白药酒，其实就是冰酒酿。浓稠润口，甜而不燥，发酵后的酒味比较浓郁，平时喜欢吃醪糟、酒酿的可以试试。

/ 橘子糖水 /

橘子糖水和黄桃糖水也是人气选手，老沈要了一碗，黄澄澄的，拍照很好看。味道就像大家小时候都吃过的那种糖水味道，没有什么复杂的调味剂，简简单单一碗糖水，却让人觉得超级亲切。

因为店里没有菜单，所以菜品没有标出价格。但是老沈吃了那么多东西，总共才花了90块左右，其中还包括了9碗冷饮，算下来真的不贵了。

除了推荐好吃的，老沈还总结了几点注意事项，给准备去打卡的老铁们参考。

温 馨 提 示

下午2:00——4:00之间店里的人会比较少。

点单的时候告诉老板要吃什么就行，不要自己拿。

调味是固定的，不能吃辣的话要主动告诉老板少放点辣。

因为没有菜单，所以点完单要记一下自己点了什么，方便结账买单。

住宿推荐武义唐风酒店，是离根建烧烤比较近的温泉酒店。

攻略都放到这儿啦。最后给大家爆个料，在参加创造101前，小彩虹徐梦洁还参加过《热血街舞团》。

彩虹爸说，徐梦洁是个很听话的孩子，同时也很有自己的想法。原本希望她接烧烤店的班，现在家里人都支持她去做自己喜欢的事。

老沈也祝这个努力又可爱的姑娘，花路越走越顺啦~
