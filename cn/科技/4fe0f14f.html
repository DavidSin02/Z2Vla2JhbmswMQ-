「浩博电池资讯」系统介绍最常见的两种高能绿色电池

2020-11-14
| 科技

本篇将对现在市面上最为常见的两种高能绿色 充电 电池 锂离子 电池 和 镍氢 蓄 电池 做一个体系的知识说明，以供我们对这两种 电池 有个全面的了解以及在选购 电池 对比角度与 电池 运用留神方面做个参看。

一、界说

锂离子 电池 （Li-ion Batteries）是以含锂的化合物作正极，以炭资料为负极的碱性 蓄 电池 。

镍氢 蓄 电池 （Nickel-Metal Hydride battery）正极活性物质首要由镍制成，负极活性物质首要由贮氢合金制成的一种碱性 蓄 电池 。

二、原理

锂离子 电池 首要依托锂离子在正极和负极之间移动来作业。在充放电进程中，Li+ 在两个电极之间往复嵌入(Intercalation)和脱嵌：充 电池 时，Li+从正极脱嵌，经过电解质嵌入负极，负极处于富锂状况；放电时则相反。

1、正极反应：放电时锂离子嵌入，充电时锂离子脱嵌。

充电时：LiFePO4 → Li1-xFePO4 + xLi+ + xe−

放电时：Li1-xFePO4 + xLi+ + xe− → LiFePO4

2、负极反应：放电时锂离子脱插，充电时锂离子刺进。

充电时：xLi+ + xe− + 6C → LixC6

放电时：LixC6 → xLi+ + xe− + 6C

镍氢 蓄 电池 在 电池 充电时，氢氧化钾（KOH）电解液中的氢离子（H+）会被释放出来，由这些化合物将它吸收，避免构成氢气（H2），以坚持 电池 内部的压力和体积。当 电池 放电时，这些氢离子便会经由相反的进程而回到原本的当地。 http://www.dghoppt.cn/ ​

1、充电时

正极反应：Ni(OH)2 + OH- → NiOOH + H2O + e-

负极反应：M + H2O + e- → MH + OH-

总反应：M + Ni(OH)2 → MH + NiOOH

2、放电时

正极：NiOOH + H2O + e- → Ni(OH)2 + OH-

负极：MH + OH- → M + H2O + e-

总反应：MH + NiOOH → M + Ni(OH)2

三、特性

1、电压：锂 电池 的电压是 镍氢 蓄 电池 电压的3倍。

锂离子 电池 电压 = 3.6V

镍氢 蓄 电池 电压 = 1.2V

2、按质量核算， 锂离子 电池 有更高能量密度。

锂离子 电池 能量∕重量 = 150～200Wh/kg(540～720kJ/kg)

镍氢 蓄 电池 能量∕重量 = 60-120 Wh/kg（瓦特小时/千克）

3、按体积核算， 锂离子 电池 有更高能量密度。

锂离子 电池 能量∕体积 = 250～530Wh/L(0.9～1.9kJ/cm3)

镍氢 蓄 电池 能量∕体积 = 140-300 Wh/L（瓦特小时/升）即 504-1188kJ/kg（千焦耳/千克）

4、输出功率： 锂离子 电池 具有更大输出功率。

锂离子 电池 功率∕重量 = 300～1500W/kg(@20秒)

镍氢 蓄 电池 功率∕重量 = 250-1000 W/kg

5、自放电率： 锂离子 电池 具有更低自放电率

锂离子 电池 自放电率 < 5％～10％/月，一般每月在2%以下（可恢复）

镍氢 蓄 电池 自放电率 = 一般每月 2-30%，见温度而定，低自放电型号为每年10-30%

6、充放电功率： 锂离子 电池 充放电率更高。

锂离子 电池 充放电功率 = 100%

镍氢 蓄 电池 充放电功率 = 66%

7、循环运用次数：两种 电池 循环次数靠近，循环运用寿命都比较长。

锂离子 电池 充放电循环次数 = 500 -2000次（磷酸铁锂 电池 能够抵达2000次以上）

镍氢 蓄 电池 充放电循环次数 = 500 -1800次

8、作业温度规划：两种高功用 电池 的温度规划都很宽，相较而言在 低温 环境中， 镍氢 蓄 电池 的放电特性更佳。

锂离子 电池 作业温度规划 -20℃～60℃

镍氢 蓄 电池 作业温度规划 -30℃～50℃（ 低温 镍氢 电池 可在 低温 -40℃中正常运用） ​

​

四、利益

1、比能量高：一次充电可行使的距离长，两者的比能量都十分高，相对而言， 锂离子 电池 （当时比能量最高的 蓄 电池 ）＞ 镍氢 蓄 电池 ；

2、比功率高：在大电流作业时能平稳放电，加快爬坡才华好，两者的比功率都十分高，锂离子优势显著一点；

3、自放电小：二者都具有此利益， 锂离子 电池 略胜一筹；

4、重量轻、体积小：同型 电池 镍氢 蓄 电池 重量更轻、体积更小，只是在输出平等电压的状况下（ 锂离子 电池 电压为 镍氢 蓄 电池 的3倍），使得单个 电池 组合时数目 锂离子 电池 可减少1/3，因而使成型后的 电池 重量和体积 锂离子 电池 更小、更轻。

5、回想效应低： 锂离子 电池 无回想效应因此在这个方面具有更大的优势；

6、 低温 放电功用好：在 低温 状况下 镍氢 蓄 电池 更强悍；

7、循环寿命长：二者循环运用次数都十分多，很合算；

8、无环境污染：两者都是绿色环保 电池 ， 镍氢 蓄 电池 被称作是最环保的 电池 ，回收再用的功率 锂离子 电池 略胜一筹；

9、标明容量高： 镍氢 蓄 电池 标明容量可达2900mAh（毫安培－小时/毫安时），但仍不及 锂离子 电池 高；

10、可快速放电：二者都具有这种利益， 锂离子 电池 才华较强一些；

11、充电时间短：二者都有此利益。

别的， 镍氢 蓄 电池 还具有安全可靠，免维护的特色，被称作是最安全的 电池 ，而 锂离子 电池 的安全功用还有待大大提高；而另一方面 锂离子 电池 因为完全没有回想效应，在运用上更为便利简略，而 镍氢 蓄 电池 却因为剩下电压的原因而无法避免进行守时放电处理操作。

六、问题

两者有个一同的缺陷，那就是制作工艺要求严峻，导致本钱比较高，价格不便宜。相对而言，锂 电池 仍是更合算一点。

七、运用

镍氢 蓄 电池 在笔记本电脑、便携式摄像机、数码相机及电动自行车等领域得到了广泛运用。

锂离子 电池 广泛运用于消费电子产品、军工产品、航空产品等。

八、总述

锂离子 电池 和 镍氢 蓄 电池 都是绿色 电池 ，不会因扔掉而形成二次污染，这契合环保理念，一同它们都可再生运用，契合持续发展的观点，此外，我们在 电池 职业技术现已处于国际领先水平，而且这些绿色 电池 出口前景十分好，虽然现在价格有点高，有些小问题还需求更进一步的得到解决与改善，不过跟着我们国家科研人员以及、企业研发部门人员的一同极力，很快就能抵达优质廉价、高效节能、安全安稳、绿色环保这些要求，而像这种绿色高能且极具开发潜力的 电池 值得我们大力提倡与推行。
