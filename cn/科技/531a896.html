CivilFEM薄壳结构五种不同网格划分实例

2020-10-29
| 科技

几何结构

CivilFEM几何模型


结构元素： 薄壳结构


网格划分一：Edge Length with Mesh Transition Parmeter: 1

网格定义参数：

Mesh control: Edge length

Element Type: Triangle

Type of algorithm: Advancing front

Mesh transition parameter: 1

Edge size: 1m

网格划分结果：

网格划分二: Edge Length with Mesh Transition Parameter: 2

网格定义参数：

Mesh control: Edge length

Element Type: Triangle

Type of algorithm: Advancing front

Mesh transition parameter: 2

Edge size: 0.3m

网格划分结果：


Mesh Transition Parameter值表示从中部网格尺寸和边缘尺寸的比值，大于1，中部网格尺寸大于边缘网格尺寸；小于1，中部网格尺寸小于边缘网格尺寸。


网格划分三: No. of Division, Overlay Meshing with no bias

网格定义参数：

Mesh control: Number of division

Element Type: Quadrilateral

Type of algorithm: Overlay quad

Divisions U direction: 40

Divisions V directin: 20

Bias Factor U: 0

Bias Factor V: 0

网格划分结果：


网格划分四: No. of Division, Overlay Meshing with bias

网格定义参数：

Mesh control: Number of division

Element Type: Quadrilateral

Type of algorithm: Overlay quad

Divisions U direction: 40

Divisions V directin: 20

Bias Factor U: 0.5

Bias Factor V: 0

网格划分结果：


Bias Factor为网格细分偏差系数，在-0.5到0.5之间，正值表示正方向上最末端网格越密，0值表示网格在该方向上均布。


设置两个方向上的偏差系数

Bias Factor U: 0.5

Bias Factor V: 0.5


Bias Factor U: -0.5

Bias Factor V: -0.5

Edges size: 0.2m

Curvature Check

Max deviation: 0.1

Min element size: 0.2
