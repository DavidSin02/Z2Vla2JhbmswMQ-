这种对讲机不要操作证也不用执照，普通人也能用的公众对讲机介绍

2020-11-14
| 科技

最近很多朋友咨询小汪，玩对讲机都需要考操作证办理执照吗？有没有可以不需要操作证和执照的对讲机呢？你别说，还真有，那就是公众对讲机。所谓公众对讲机是指，发射功率不大于0.5W，工作于指定频率的无线对讲机。

（小米米家对讲机包含公众频率）

目前，在我们国家设置和使用公众对讲机，不需领取电台执照 ， 免收频率占用费。但是公众对讲机是禁止在机场和飞行器上使用的 。 还有就是公众对讲机只能和公众对讲机进行通讯，公众对讲机是不能和业余无线电台以及其他非公众对讲机频率的电台进行通讯的。就像我们业余无线电台不能和非业余无线电台进行通联是一个道理。

业余无线电呼号

目前公众对讲机的工作频率为（单位：MHz）: 409.7500 ; 409.7625 ; 409.7750 ; 409.7875 ; 409.8000 ; 409.8125 ; 409.8250 ; 409.8375 ; 409.8500 ; 409.8625 ; 409.8750 ; 409.8875 ; 409.9000 ; 409.9125 ; 409.9250 ; 409.9375 ; 409.9500 ; 409.9625 ; 409.9750 ; 409.9875 （共20个频点）。

参数方面要求：调制方式：F3E；有效发射功率（EIRP ）：< 0.5W；发射频率容限：<+5ppm；发射机杂散辐射：<50uW；接收机杂散辐射：<20nW；信道间隔：12.50kHz

使用人或使用单位只有在购买了符合规定的公众对讲机时，才可以不必领取执照、考取操作证或者交纳频率使用费，但此处的公众对讲机是不能和非公众频率进行通联的，大家务必要注意。

小米米家对讲机和泉盛UV2对讲机

其他一些需要注意的地方：

首先就是大家在选购公众对讲机的时候一定要选择有核准码的对讲机，购买之前一定要咨询商家。其次就是已经国家无线电管理机构型号核准的公众对讲机，任何单位和个人不得擅自更改使用频率、加大发射功率(包括额外加装射频功率放大器)，不得擅自外接天线或改用其它发射天线，或改变原设计特性及功能。最后就是针对0.5瓦的功率，有的朋友会质疑会不会太小，在这里小汪要告诉大家，对讲机的通联距离跟功率确实是有一定的关系，但是只要环境好，通讯电台之间没有阻挡，0.5瓦也是可以通联很远的距离的！最后祝大家通联愉快，Happy New Year！73！
