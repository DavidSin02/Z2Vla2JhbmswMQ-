什么是平均收益？平均收益率与几何平均值之间的差异

2020-11-14
| 科技

什么是平均收益?

平均收益是一段时间内产生的一系列收益的简单数学平均值。平均收益的计算方法与为任何一组数字计算简单平均的方法相同。将数字加在一起成为一个总和，然后将总和除以集合中数字的计数。

平均收益公式

\ text {平均退货} = \ dfrac {\ text {退货总额}} {\ text {退货数量}}平均回报=退货数量返回的总和﻿

如何计算平均收益

有几种回报方法和计算方法，但对于算术平均回报，一种方法是将回报之和除以回报位数。

平均收益告诉你什么?

平均收益告诉投资者或分析师过去股票或证券的收益是多少，或者公司投资组合的收益是什么。这与年化收益率不同。平均收益忽略了复利。



重要要点

  1. 平均回报是一系列回报的简单数学平均值。
  2. 它可以帮助评估证券的过去表现或投资组合的表现。
  3. 几何平均值始终低于平均收益。

如何使用平均收益示例

平均收益率的一个例子是简单的算术平均值。例如，假设一项投资在整个五年的时间内每年获得以下收益：10%，15%，10%，0%和5%。要计算这五年期间的平均投资回报率，将五个年度回报率相加，然后除以5。这将产生8%的年平均回报率。

或考虑沃尔玛(纽约证券交易所：WMT)。沃尔玛股价在2014年回报9.1%，在2015年下跌28.6%，在2016年上涨12.8%，在2017年上涨42.9%，在2018年下跌5.7%。这五年间，沃尔玛的平均回报率为6.1%或30.5%除以5年。

计算增长回报

简单增长率是起点和终点或余额的函数。它是通过从起始值中减去终止值，然后除以起始值来计算的。计算公式如下：

\ begin {aligned}&\ text {增长率} = \ dfrac {\ text {BV}-\ text {EV}} {\ text {BV}} \\&\ textbf {其中：} \\&\ text { BV} = \ text {开始值} \\&\ text {EV} = \ text {结束值} \\ \ end {aligned}​增长率=BVBV-EV哪里：BV=初始价值电动车=结束值﻿

例如，如果您向一家公司投资10,000美元，并且股票价格从50美元增加到100美元，则可以通过将100美元和50美元之间的差额除以50美元来计算回报。答案是100%，这意味着您现在有20,000美元。

平均收益率与几何平均值之间的差异

当查看平均历史收益时，几何平均值是一种更精确的计算。几何平均值始终低于平均收益。使用几何平均值的好处之一是不需要知道实际的投资额。该计算完全着眼于收益数据本身，并且在考察两个或更多个投资在不同时间段内的表现时，呈现出“从苹果到苹果”的比较。

几何平均收益率有时称为时间加权收益率(TWRR)，因为它消除了随着时间的流逝各种资金流入和流出而对增长率产生的扭曲影响。

另外，货币加权收益率(MWRR)结合了现金流量的大小和时间安排，因此它是衡量已收到存款，股利再投资，利息支付或已提款的投资组合收益的有效度量。货币加权收益等于内部收益率，其中净现值为零。

使用平均收益率的局限性

简单的收益平均值很容易计算，但不是很准确。为了更准确地计算收益，分析师和投资者还经常使用几何均值或货币加权收益。
