跟我学python(4)——turtle画哆啦A梦

2020-10-29
| 科技

关于turtle的知识，基本的大家都已经学过了，今天我们就绘画一幅画吧！基础的方法我已经在后面加注释啦！大家如果看不懂的话可以留言呀！

今天我们要画出可以哆啦A梦！如图所示：


python4-0


首先我们要进行最开始的设置，设置画笔，以及画笔的粗细，背景的颜色，图片中的背景是白色。我们选择浅蓝色。

初始设置：

import turtle

turtle.screensize(400,400,'skyblue')

pen=turtle.Pen()

pen.speed(5)

pen.pensize(5)

pen.hideturtle()

turtle.done()温馨提示：这句要放在最后呀！

接下来我们开始设置满脸，需要设置填充颜色。代码如下：

pen.fillcolor('#00A1E8') #选择填充的颜色，可以是#00A1E8的格式，也可以是颜色名称，比如上面的skylue,颜色可以百度

pen.begin_fill()#开始填充

pen.circle(130)#画一个半径130的圆

pen.end_fill()#结束填充

pen.pensize(3)#设置画笔粗细

pen.fillcolor('white')#设置填充颜色

pen.begin_fill()#开始填充

pen.circle(100)#画一个半径100的圆形

pen.end_fill()#结束填充

这时候就画出了哆啦A梦的脸

pen.begin_fill()

a = 0.4

for i in range(120):

if 0 <= i < 30 or 60 <= i < 90:

a = a+0.08

pen.lt(3) #向左转3度left的缩写

pen.fd(a) #向前走a的步长 forward的缩写

else:

a = a-0.08

pen.lt(3)

pen.fd(a)

pen.end_fill()
