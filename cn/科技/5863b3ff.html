如何设置用户只能通过网页只读访问 ProjectWise？

2020-11-14
| 科技

在平常的项目中，往往有一些项目或文件的安全级别比较高，只允许外部用户通过只读权限进行访问，而不能在 ProjectWise 中进行创建、修改或删除操作，如果通过设置权限的方式进行设置或归类，工作量又比较大，还可能出现误操作。因此，ProjectWise 提供了一个设置，即针对某些特定用户，只允许其通过 ProjectWise 的 Web View Server 进行登录，而不能通过客户端或 Web Server 登录。


Web View Server 的功能与 Web Server 有什么不一样的呢？

它们都是网页端，都是通过浏览器就可以访问 ProjectWise 数据。但使用 Web View Server 登录的用户，对于 ProjectWise 中的文件只有只读功能。


想要使用 Web View Server，需要我们的管理员先在服务器IIS中部署 ProjectWise Web View Server。


部署完成后，当用户通过 Web View Server 网页访问时，权限会自动变成只读权限。


在 ProjectWise 当中，用户是可以通过多种终端访问到 ProjectWise。如果只需要某个特定用户只能通过 Web View Server 网页访问到到 ProjectWise，就还需要在 ProjectWise 的管理员端，对其账号进行设置。


设置完成后，该用户就无法通过 ProjectWise 客户端和 Web 端进行登录了，而只能通过 ProjectWise 的 Web View 端进行访问。
