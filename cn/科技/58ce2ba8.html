新版Kali 2020 详细图文安装教程，手把手教你搭建黑客攻击平台

2020-10-29
| 科技

Kali Linux 2020.1是目前最新的版本，相对于之前的2019版进行了大量的改动，除了组件的常规更新外，还包括默认使用标准的非特权用户（非root用户），官方期望以此来增加系统额外的安全性

新版的安装可能会遇到一些列的问题，比如中文乱码、无图形化操作界面等

除此之外，有些同学还觉得不默认用root用户，觉着不舒服，就像失去了灵魂

今天我就给大家带来一篇详细图文安装教程，并附上解决这些问题的办法

kali是什么？

Kali Linux是基于Debian的Linux发行版， 设计用于数字取证操作系统，是黑客常用的攻击平台

Kali Linux预装了超过300个渗透攻击工具，包括常用的信息收集工具nmap 、数据包修改工具Burpsuite 、系统攻击工具metassploit，以及无线攻击工具Aircrack-ng等

Kali Linux可直接在官网下载，该系统是永久免费的

点进去后，有很多选择项，可以根据自己需要进行选择

一般情况下，我们选择第一条64位的ISO系统镜像即可

虚拟机配置

一般情况下，我们不会将Kali安装到物理机上，而是安装在虚拟机里，方便管理也不影响正常使用

虚拟机管理软件常见的有VMware、Parallels Desktop（专用于MAC）、Virtual Box

使用哪一个虚拟机管理软件都不影响Kali的安装

这里我以VMware下的安装为例

新建虚拟机

选择典型安装就行

接着选择安装程序光盘映像文件（ISO）

并在选项中选定我们下载的ISO文件位置

选择Linux下的Debian 64位版本

自定义虚拟机名称和虚拟机的存放位置，建议不要放在C盘

最大磁盘大小就是指你分配给kali虚拟机使用的磁盘的大小

一般设定为40G就足够了，毕竟很少人在里面存放过多的资料文件

下面的虚拟磁盘拆分，这个看个人电脑性能与说明，自己选择

这一块自定义配件还需要我们修改一下

关于内存与处理器的设置，这需要看你物理机的情况了

如果你内存只有4G那你可以分配1G给Kali，或者更少

如果你内存只有8G那你可以分配2G给Kali

网络适配器我们先设定为桥接模式，这里很重要，我们需要在安装时联网

顺便提一下，虚拟机管理软件提供了三个网络模式

桥接模式，相当于虚拟机作为一台单独的设备直接联网

NAT模式，共享你物理主机的IP，分配了个子网IP给虚拟机，能否上网取决于物理机

仅主机模式，只能和你的物理机通讯，无法上网

设置好后，点击完成

Kali的安装

点击开启虚拟机后，开始正式安装

选择第一项，图形化安装

选择系统默认语言，我们选择中文简体，点击continue继续

地区选择

键盘选择

系统开始加载组件进行安装

设置你系统的主机名，可以随意设置

配置网络地址，这里一般人不需要设置，点击继续

设置普通用户名，文章前面也说了，新版Kali的改动之一，默认使用普通用户

这里我以头号黑客的拼音作为用户名thhk

点击继续

设置你的用户密码，两次输入一样的

例如这里我设置为123456

正在自动设置时钟，加载组件等

选择默认

继续默认

选择是

开始安装基本系统

这里不用设置代理，点击继续

点击返回，换一个源试试，这里安装失败很正常，可能是网络原因，也可能是防火墙拦截了

还是不行的话，就再次返回，选择不进行网络安装

所有选项都可以勾选，一定要勾选Xfce，这个是安装图形化界面的软件包，后面再去安装会非常麻烦

下面开始下载安装

如果在开始虚拟机配置那里，不配置联网，那么这里很多软件包是无法下载的，包括Xfce，所以进去就只有字符操作界面

继续默认

选择安装的硬盘

又进入自动安装步骤

很快便安装完成了，点击继续，进行重启

点击Enter继续

成功进入到Kali系统

到这一步，虽然还有些小问题，但系统已经算是安装成功了

如果习惯的英文操作的同学，到这里已经可以正常使用Kali了

优化Kali

  • 更换为root用户

在空白屏幕处点击右键，根据图标就可以知道哪个是新建终端命令

命令行执行

sudo passwd root

会先提示输入当前普通用户的密码，执行sudo命令需要

这里我需要输入123456

然后再设置root密码，紧接着输入两次相同的密码

这个密码就是你设置的root密码

这里我依次输入test123和test123来作为我root密码

设置完成后，继续输入

su - root你设置的密码

即可进入root权限

  • 修改源地址

从国内的代理站点下载软件要快许多

终端输入指令

vim /etc/apt/sources.list

输入i进入编辑模式

空白处添加阿里源

deb https://mirrors.aliyun.com/kali kali-rolling main non-free contrib


点击esc，输入:wq

退出并保存

# 替换好源后，更新一下源就可以了

sudo apt clean && sudo apt update -y

  • 中文乱码

虽然刚才已经说了使用英文，系统已经可以正常使用

但为了以后方便使用，我们还是需要将该问题解决

在终端输入指令，安装中文字体

apt update -y && apt install xfonts-intl-chinese ttf-wqy-microhei

输入reboot，重启系统

可以看到我们的中文乱码已经解决啦

除此之外，我们还可以直接使用root账户登录，解决了某些同学不喜欢普通用户的烦恼

这里我输入我刚才设置的root和test123

就可以root用户登录系统啦
