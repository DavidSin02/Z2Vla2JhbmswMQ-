技术推荐|工业固废钢渣

2020-11-14
| 科技

一、钢铁行业钢渣处理现状

目前，我国出渣量与钢铁产量的比例基本上维持在100kg/ t240kg/t 范围内、就此计算，仅2017年，我国产钢渣约1.3亿吨左右。而我国实际钢渣的综合利用率不足30%, 导致我国钢渣堆存量已累计超过10亿吨 。钢渣的处理与资源利用及零排放已成为困扰各个钢铁企业可待续发展和迫切需要解决的问题。

钢渣中含有大量的金属铁和废钢，钢渣中的含铁物料约10% - 15%，通过磁选以后，可以提取回收各个粒级的废钢铁 ，回收的废钢铁产品根据粒度不同可分为渣钢、粒铁和磁选粉。

目前，由钢渣回收的渣钢由于品位高主要是直接返回转炉炼钢，粒铁和磁选粉采用返回高炉重复利用。

在实际生产中，钢渣回收的废钢铁一般根据不同规格大小和含铁成分不同选择不同的用途：

1、含铁量在80%以上，规格>50mm的钢渣渣钢供应给炼钢当废钢消耗使用 ；

2、含铁量在60%以上，规格在s5omm之间的钢渣粒铁供应给炼铁直接入炉使用；

3、含铁量在60%以下，规格在1smm的钢渣磁选粉供应给烧结当矿粉使用。

二、钢渣处理工艺介绍

目前国内钢渣主要处理工艺有：热泼法、风埣法、滚筒法、热闷法。其中热泼法、滚筒法、热闷法最为常用，在此对其工作原理和优缺点进行简单介绍。

(1 ) 渣线热泼法

将钢渣倾翻 ，喷水冷却3-4天后使钢渣大部分自解破碎 ，运至磁选线处理。

此工艺的优点在于对渣的物理状态无特殊要求、操作简单、处理量大。

其缺点为占地面积大、浇水时间长、耗水量大，处理后渣铁分离不好、回收的渣钢含铁品位低、污染环境、钢渣稳定性不好、不利于尾渣的综合利用。

( 2 ) 渣跨内箱式热泼法

该工艺的翻渣场地为三面砌筑并镶有钢蚽的储渣槽，钢渣罐直接从炼钢车间吊运至渣跨内，翻入槽式箱中，然后浇水冷却。

此工艺的优点在于占地面积比渣线热泼小、对渣的物理状态无特殊要求、处理最大、操作简单、建设费用比热闷装置少。

其缺点为浇水时间24h以上、耗水量大、污染渣跨和炼钢作业区、厂房内蒸汽大、影响作业安全。钢渣稳定性不好、不利于尾渣综合利用。

( 3 ) 滚筒法

高温液态钢渣从溜槽流淌下降时，被高压空气击碎，喷至周围的钢挡板后落入下面水池中。

此工艺的优点在于流程短、设备体积小、占地少、钢渣稳定性好、渣呈颗粒状、渣铁分离好、渣中f—CaO含量小于4% (质量分数，下同）、便于尾渣在建材行业的应用。

其缺点为对渣的流动性要求较高、必须是液态稀渣、渣处理率较低、仍有大量的干渣排放、处理时操作不当易产生爆炸现象。

( 4 ) 热闷法

待熔渣温度自然冷却至300- 800°C 时，将热态钢渣倾翻至热闷罐中 ，盖上罐盖密封，待其均热半小时后对钢渣进行间歇式喷水。急冷产生的热应力使钢渣龟裂破碎，同时大量的饱和蒸汽渗入渣中与f —CaO、 f —MgO发生水化反应使钢渣局部体积增大从而令其自解粉化。

此工艺的优点在于渣平均温度大于3oo·c均适用，处理时间短（10-12h）,粉化率高（粒径20mm以下者达851Jfi-), 渣铁分离好，渣性能稳定，f—Cao、f—MgO含量小于 2%, 可用于建材和道路基层材料。

其缺点为需要建固定的封闭式内嵌钢蚽的热闷箱及天车厂房、建设投入大、操作程序要求较严格、冬季厂房内会产生少量蒸汽。

三：钢渣处理工艺共性问题

1、处理过程中造成二次污染，主要是水污染。

2、由于工艺原因造成回收产品的品味相对较，经济效益差，再次利用率相对较低。

3、尾渣含铁率偏高，不仅造成资源浪费，更造成尾渣很难被高附加值利用。

四、钢渣处理方面政策解读

国家《环境保护税法》2018年1月1日正式启用，其中对钢铁企业产生且未处置的钢渣每吨征收25元的环保税，对露天堆存的钢渣雨季所造成的钢渣渗滤液每吨征收300元的环保税。

这对于钢铁企业来说，既是机遇更是挑战，高效处理钢渣有助于降低企业生产成本，延伸资源利用的深度，真正实现绿色环保型循环经济。
