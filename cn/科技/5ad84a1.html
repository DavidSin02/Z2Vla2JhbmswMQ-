Windows Core OS再遭曝光：可轻松移植到Xbox

2020-10-29
| 科技

全新设计的Windows Core OS系统已经不止一次地被曝光，甚至出现在过微软官方的文件代码里，但是微软从未正面评论这一系统，尤其是什么时候发布？用于什么设备？

事实上，有关于Windows Core OS系统的消息2017年就出现了，据称这是一套UI界面风格更现代化、功能模块较为简单的系统平台，面向双屏折叠Surface、Xbox、Surface Hub、Hololens等终端设备，Windows Lite精简系统也是其衍生品。

现在有人发现，微软产品经理Naim Ayat的领英档案里赫然出现了Windows Core OS的身影，他正是这一团队的成员。

按照他的描述，Windows Core OS系统是Windows 10、Windows MR HoloLens、Xbox软件系统未来迭代的基石，但是很快，这句话就被删除了，等于坐实。

Windows 10、Windows MR、Xbox等字样已被删除

Xbox Series X现在运行的系统叫Xbox OS，内核与Windows 10完全相同，但是既然Windows Core OS本身就是模块化设计，不再依赖于硬件，那么移植到未来的Xbox上也很轻松。

同时，Windows Core OS既然也是Windows 10今后演进的基础，那就意味着很长时间里都不会有Windows 11。

另外，Naim Ayat还透露他目前正在从事Windows 10X相关工作，这套针对Surface Neo等双屏笔记本设计的系统，将在今年年底的购物季面世。
