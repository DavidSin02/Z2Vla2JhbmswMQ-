手语文法的结构

2020-10-29
| 科技

一、手语文法的结构

1.关于手语的文法：自然手语的文法与我们惯用的中文文法不同，熟习自然手与文法是一件很重要的，一如我们学习英文文法，一定样遵照它的文法结构系统一样。

一般手语文法的顺序： 主词→名词→动词→形容词→副词→助动词→疑问词

2.无论上面顺序如何变动，均万变不离其宗，活用为主。基本上手语不论句子的构成，名词或主词均先比出来，之后才比动词及其他词。

例一：

a. a. 我是郑照斌---我 /郑照斌 【助动词可有可无】

b. b. 我就是郑照斌---我 / 郑照斌 / 是【强调】

c. c. A：郑照斌是谁？---郑照斌 / 谁？

B：郑照斌是我。---郑照斌 / 我。

d. d. A：你 / 郑照斌？

B：我 / 郑照斌 / 是（或不是）

e. e. A：你 / 手语名字 / ……？

B：不是，手语名字 / 郑照斌 / 我

例二：

a. a. 那个男生长得高大英俊---那 / 男 / 高 / 英俊

b. b. 那个高大英俊的男生---那 / 高 / 英俊 / 男

例三：

a. a. 昨天我搬家了，今天整理家具---昨天 / 我 / 家 / 搬 / 完了，现在 / 家 / 东西 / 整理

3.疑问句的手语，均置于整个句子之后。

例一：

a. a. 有没有𫓨？---𫓨 / 有没有？

b. b. 有没有𫓨借我？---𫓨 / 借我 // 𫓨 / 有没有？

c. c. 有没有𫓨借我买表？---我 / 表 / 买 / 𫓨 / 借我 / 𫓨 / 有没有

例二：

a. a. 为什么你要欺负我？(欺负我 / 为 / 什么)

b. b. 周末放假，你想去那里玩？(星期六 / 放假 ， 想 / 玩 / 那 / 处)

3.手语的方向及方位要清楚指明是何人何地，完全以空间指示来表达。

例一：

a. a. 我喜欢你---我 / 喜欢 / 你

b. b. 我喜欢你---我→你 / 喜欢

c. c. 你喜欢他吗？---你→他 / 喜欢？

例二：

a. a. 他认识我，但他不知道我的名字

那 / 男 / 见 (右→我) / 认识 // 我 / 名字 / 那 / 一窍不通

b. b. 他认识我，但他知道我名字吗？

那 / 男 / 见 (右→我) / 认识 // 我 / 名字 / 那 / 知道？

4.助动词的使用，一般均用在句尾。

a. a. 你会打电脑？---电脑 / 会？

b. b. 我和我女朋友要去看电影---我 / 女 / 朋友 / 我俩 / 电影 / 看 / 要

c. c. 我没空，不能去看你---我 / 时间 / 没有 // 探访 / 不能

5.动词的方位。

a. a. 我请客，你出𫓨？---请客 (你→我) / 出𫓨 (你→右)

b. b. 他告诉我，我在告诉你---告诉 (他→我) / 完了 // 告诉 (我→你)

c. c. 你不要骗我哦---骗我 / 不

d. d. 我跟他说好了，一起出国去玩---我←→他 / 谈 / 达成协议 // 我俩 / 出国 / 玩

6.手语具有同时性。【表情】

a. a. 明天我去他家。

明天 / 去(右手，我→他家)

家(左手，他处不动)

b. b. 你不认识他吗：

你 (眼神)

不认识 (右手)

他 (左手)

吗 (抬眉)

7.表达数量时，名词 + 数量，单位词省略。

a. a. 我家有两只狗，三条鱼---我 / 房子 / 狗 / 二 / 鱼 / 三 / 有

b. b. 我有六辆汽车，八部飞机---我 / 汽车 / 六 / 飞机 / 八

c. c. 他讨了四个老婆---他 / 娶 / 四

8.在手语里面，很注意表达手语的方位及动作方向及时间的次序，如此才能表达每一人物或事件的相互关系，也就是说，手语具有时间域和空间域。

a. a. 由台北搭飞机到高雄，再由高雄搭船到香港

台北（胸前）/ 飞机（至左）/ 高雄（左）/ 船（至右）/ 香港 （右）

b. b. 我坐飞机到日本再转机到纽约

搭飞机 / 起飞（由胸前→右）/ 日本 / 转到 / 起飞（右→左）/ 纽约

c. c. 我打电话给他，但没人接听

我 / 打电话（至左）// 人（左）/ 接电话（至胸前）/ 没有

9.比较性手语

a. a. 他的朋友多，我的朋友较少

b. b. 去的人渐渐多了，来的人少了
