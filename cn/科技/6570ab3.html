挤压丝锥定义

切削丝锥主要是通过切除金属的方式来获得内螺纹而，挤压丝锥是通过丝锥上螺纹棱带挤压内孔金属，使金属变形后充满丝锥牙槽，丝锥退出后形成内螺纹。挤压丝锥适合在塑性材料上攻丝，如铝合金、铜合金、不锈钢、低碳钢等。使用挤压丝锥形成的内螺纹粗糙度高、精度高、强度大。需要注意的是挤压丝锥要求的底尺寸不同于切削丝锥攻丝的底孔。

分享了一张挤压丝锥的结构图，大家可以看下，写的很清楚。

挤压丝锥结构图


挤压丝锥主要由工作部分和夹持部分组成，工作部分分为挤压锥部和校准部。挤压锥部为锥形，主要对内螺纹进行挤压，以便使工件材料逐步产生塑性变形形成螺纹; 校准部的径向剖面一般是多棱形。径向剖面多棱形随锥度部分各剖面直径的变化而变化，但剖面形状基本不变，校准部的第一扣牙用于使被挤压螺纹最终成形，后边各扣牙主要用于丝锥导向和保证自动进给。挤压丝锥的夹持部分即锥柄的结构类似于切削丝锥，挤压丝锥是一种先进的高效内螺纹加工刀具，它是通过使工件材料产生塑性变形而形成螺纹牙型。与切削丝锥相比，挤压丝锥具有以下特点:


挤压丝锥和切削丝锥加工后截面对比


( 1) 加工的螺纹精度高，表面粗糙度高;

( 2) 加工的螺纹具有较高承载能力;

( 3) 加工中不产生切屑，易于实现自动化加工，并使加工过程比较干净;

( 4) 生产辅助工时短，生产效率高;

( 5) 强度高于一般丝锥，不易折断;

( 6) 挤压丝锥不易磨损，使用寿命长;

( 7) 非常适合加工碳素结构钢、低合金结构钢、不锈钢、各种铝合金、铜、镍及其合金等有色金属及低强度黑色金属塑性材料。
