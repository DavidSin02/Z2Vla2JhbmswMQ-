Python的输入与输出

一、输出格式美化

Python常见的有两种输出值的方式：表达式语句和print()函数。而第三种方式是使用文件对象的write()方法，标准输出文件可以用sys.stdout引用。

如果你希望输出的格式更加多样，可以使用str.format()函数来格式化输出值。

如果你希望将输出的值转换成字符串，可以使用repr()函数或str()函数来实现。

其中:

  • str()函数返回一个用户易读的表达形式
  • repr()函数产生一个解释器易读的表达形式

例如：

s1 = '大家好，我正在学习Python！'print(str(s1)) # 大家好，我正在学习Python！print(repr(s1)) # '大家好，我正在学习Python！'# 三种方法输出1~10的平方和立方# rjust()可以将字符串靠右，并在左边填充空格 括号内为输出值的位数for m in range(1,11):    print(repr(m).rjust(2), repr(m*m).rjust(3), repr(m*m*m).rjust(4))# 0，1，2分别对应format中的三个值，:2d表示两个宽度的十进制，依次是三个和四个宽度for n in range(1,11):    print('{0:2d} {1:3d} {2:4d}'.format(n, n*n, n*n*n))# 使用f-string的方式格式化输出for p in range(1,11):    print(f'{str(p).rjust(2)} {str(p*p).rjust(3)} {str(p*p*p).rjust(4)}')# zfill() 会在数字的左边填充0  括号内为输出值的位数for q in range(1,11):    print(repr(q).zfill(0), repr(q*q).zfill(3), repr(q*q*q).zfill(4))

二、format

括号及里面的字符(称为格式化字段)将会被format()中的参数替换。花括号{}中的数字用于指向传入对象在format中的位置。

例子：

print('{0}和{1}是电影的男生角和女生角'.format('张国荣','毛舜筠')) # 张国荣和毛舜筠是电影的男生角和女生角print('英雄本色演员有{0},{1},和{2},其中，{2}是个内应'.format('张国荣','周润发','狄龙')) # 电影英雄本色的主要演员有张国荣,周润发,和狄龙,其中，狄龙是个内应# 也可以使用关键字参数print('{0}教会了{1}的{kongfu}'.format('风清扬','令狐冲', kongfu='独孤九剑')) #风清扬教会了令狐冲的独孤九剑[25]# :后传入一个整数，可以保证该域至少有这么多的宽度，用于美化格式很有用import math# 保留PI小数点后三位print('圆周率取三位小数：{:.3f}'.format(math.pi)) #圆周率取三位小数：3.142# 对字典中的数值，使用**输出d1 = {'百度':100, '阿里':200,'拼多多':300, '今日头条':400,'京东':'W'}print('百度:{百度:2d}; 阿里:{阿里:3d}'.format(**d1)) # 百度:100; 阿里:200

三、input

Python提供了input()内置函数从标准输入读入一行文本，默认的标准输入是键盘, 其中，input可以接收一个Python表达式作为输入，并将运算的结果返回

示例

print('=================代码演示input====================')name = input('请输入你的姓名：')print(f'你输入的名字是：{name}')

四、文件的读和写

  • open(): 将会返回一个file对象，基本的语法格式如下：
open(filename, mode)# filename: 包含了你要访问的文件名称的字符串值# mode: 决定了打开文件的模式，有只读，写入，追加等方式。默认方式是只读(r)

在Python中，打开文件的模式主要如下表所示：

文件读写mode的模式

  • 文件对象的常用方法

通过open(filename, mode, encoding)可以创建一个文件对象，那么，创建好的文件对象有哪些常用的方法呢？

    • read():读取一个文件的内容，调用read(size)，这将读取一定数量的数据，然后作为字符串或字节对象返回， 其中，size是一个可选的数字类型的参数。当size被忽略时或者取负值时，那么，该文件的所有内容都将被读取并返回。

read()方法

    • readline(): 会从文件中读取单独的一行。换行符为'\n'。如果返回一个空的字符串，说明已经读取到了最后一行。

readline()

    • readlines(): 将返回该文件包含的所有行,返回列表。

readlines()

    • write(string): 将string写入到文件中，然后返回写入的字符数量，如果写入的不是字符串，则需要先进行转换。
# 打开一个不存在的文件, 写入字符串, 会先创建该文件with open('mynewfile.txt','w+') as f:    f.write('风清扬最喜欢令狐冲，所以教他独孤九剑!')# 写入一个非字符的时候，需要使用str()将其转换成字符形式，否则会报错with open('mynewfile.txt','a+', encoding='utf8') as f:    t1 = ('岳不群', 100, '令狐冲', 200, '东方不败', {'name':'鹿鼎记', 'author':'金庸'})    # f.write(t1) # TypeError: write() argument must be str, not tuple    f.write(str(t1))
    • tell(): 返回该文件对象当前所处的位置，它是从文件开头开始算起的字节数。
    • seek(offset, from_what): 如果要改变文件的当前位置，可以使用该函数，其中，对于from_what: 0表示开头(默认值)，1表示当前位置，2表示文件的结尾，例如：
seek(x,0): 从起始位置即文件首行字符开始移动x个字符；seek(x,1): 表示从当前位置向后移动x个字符seek(-x,2): 表示从文件结尾往前移动x个字符
    • close()：当你处理完一个文件后，调用close()方法来关闭文件并释放系统的资源，如果尝试再调用该文件，则会抛出异常。
