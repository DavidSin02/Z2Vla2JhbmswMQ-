Nature Highlight：信息码存储，静态到动态的飞跃

2020-11-14
| 科技

信息码利用其自身的形状能够存储特定的信息，例如，一维条形码和二维码分别用线条和特定的几何图形带代表存储的信息，彩色三维码是建立在传统黑白二维码基础之上发展而来的一种全新图像信息码，它由红蓝绿黑四种颜色的矩阵而构成。以往的这些信息码一般只能存储特定的信息，是一种静态存储，其内容不能改变。

美国德克萨斯大学奥斯汀分校（University of Texas at Austin） Jonathan L. Sessler教授（通讯作者）和吉晓帆博士（论文第一作者），江苏大学龙凌亮教授（共同通讯作者）和浙江大学黄飞鹤教授（共同通讯作者）合作，基于多重荧光的超分子聚合物水凝胶制备了响应型的荧光三维码，分别用物理和化学的方式实现了其存储信息内容的转变。他们将聚丙烯酰胺通过N,N’-亚甲基双丙烯酰胺（化学）和大环／阴离子复合物（物理）共同交联形成水凝胶，引入不同的荧光基团形成了红蓝绿三种荧光颜色的超分子聚合物水凝胶（图一）。由于大环／阴离子这一非共价键交联不仅存在水凝胶的内部，而且也分布在其表面，这三种荧光水凝胶能够通过界面的非共价键作用力组装在一起形成多重荧光的超分子聚合物水凝胶，再通过调整三种水凝胶的数量和位置分布就可以在黑色基质上形成不同的荧光三维码（图二a，如Code A, Code B和 Code C)。

图一、无荧光水凝胶G0, 蓝色荧光水凝胶G1, 绿色荧光水凝胶G2, 非响应型红色荧光水凝胶G3-1, 以及对氨气具有响应的红色荧光水凝胶G3-2的化学结构和卡通示意图。

相对于传统的信息码，这些荧光三维码有新的特点（图二b），例如：（1）各种荧光颜色只在紫外灯照射下才显示，其存储的信息能够得到保护； （2）非共价键交联具有可逆性，荧光三维码被破坏后可以进行自修复；（3）各种荧光会在添加特定的金属离子后发生淬灭，其存储的信息能够在需要的时候得到删除。

图二、卡通示意图：a) 荧光水凝胶G1, G2, 和 G3(G3-1和 G3-2)通过界面的非共价键作用力组装荧光三维码。b) 对Code A进行信息保护，信息自修复和信息删除。c) 通过物理和化学方法，对信息码存储的信息进行转换。

更有趣的是，其存储的信息可以进行转换（图二c），例如，（1）这种荧光三维码是三种荧光水凝胶通过界面可逆的非共价键作用力组装在一起而形成，所以通过可逆的拆分／重组装，即物理方法可以实现Code B到 Code A的转换，其存储信息也从Info B变为Info A；（2）引入一个具有对碱性物质响应性的红色荧光水凝胶，其荧光颜色能够在接触碱性气体（如氨气）后从红色变成蓝色，这样Code C能够在氨气条件下转化成 Code A，即化学方法实现了存储信息从Info C到Info A的转换。因此，这项工作分别用物理和化学的方法实现了信息码存储信息的内容转变。

另外，由于水凝胶材料自身的柔软性以及水凝胶之间超分子组装的稳定性，这种荧光三维码可以作为一种具有信息存储功能的可穿戴材料。

此项工作发表在Advanced Materials上， 此外，本文还入选Nature Highlight栏目。
