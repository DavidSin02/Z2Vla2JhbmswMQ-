第一章 项目介绍

1.1 项目背景

XX医院是一所按照三级甲等综合医院标准设置，是一座集医疗、科研、教学于一体的综合医院，在设置53个临床科室规范下，整合成18个临床中心。XX医院设置床位3000张，其中医疗床位2000张，康复床位1000张，由VIP区和国际特需区组成。（该部分自行根据客户项目情况修改完善！）

随着居民对医疗服务要求的提高，国家医疗制度改革的深化，国家对医疗卫生系统信息化建设投资的不断增加。现如今许多医院在完成基础设施建设后，其投资的重点向智慧医院方向发展。随着医改的逐步深入，医疗卫生信息化的建设步伐也在不断加快，医疗卫生信息化建设的核心是实现病人和医疗机构之间，医院各个科室之间，医院之间，医院与社区、医疗保险部门、卫生行政部门等之间的信息共享，信息化不仅能提升医生的工作效率，使医生有更多的时间为患者服务，有助于提高患者满意度和信任度，树立良好的医院形象。因此，医院决定建设无线网络来实现医院医疗卫生信息化。

1.2 项目需求

项目建设具体需求如下：

1.无线覆盖：需对医院大厅、走廊、病房等区域进行无死角全覆盖，且在无线终端移动过程中，终端能自动切换，实现无感知漫游，实现无线网络线上预约门诊，在线查询体检报告，支付宝在线缴费。

2.认证营销：需提供多种认证登入方式，例如：微信、短信等，并且能够与院内微信公众号/官网相结合，在为病人和家属提供简单方便的无线连接方式的同时，实现微信吸粉及信息发布。

3.统一管理：医院存在弱电网络安防网络及计算机网络，需要对全网进行统一的平台化管理。

4.运维需求：能够实现对全网简单高效运维，降低服务成本。

5.审计要求：无线网络要满足公安部对公共场所网络审计要求。

6.售后服务: 网络建设方需提供高质量的售后服务，保障在无线网络出现故障时，能够得到快速解决，避免无线网络出现长时间故障，因此需要本地化的售后服务机制。

第二章 总体设计

2.1 设计原则

根据医院的无线网络建设现状和实际需求，所设计网络需要完美符合院方需求，在此基础上，所设计的方案还需要突出性价比和可用性，保障在未来3-5年内无线网络符合网络的发展趋势并具有充分的可扩展性。

2.2 网络构架


网络架构拓扑图

2.3 点位设计

第三章 方案特色

3.1 综合管理平台

ü 采用强劲的X86架构，集无线控制器、有线控制器、用户认证、营销推送、客流分析、上网行为管理、流量控制、上网行为审计、网络管理于一体，大大简化网络结构，极大的降低了建设成本、运维成本。

ü 具有丰富的用户认证功能，支持Portal、微信、短信、APP、二维码等认证方式。

ü 高密优化、智能负载均衡、无缝漫游等，全面保证用户上网体验。

ü 提供完整的安全防护体系，保障用户数据安全。

ü 提供可视化管理界面、热点地图、微信小程序等方式，帮助学校有效管理无线网络，让无线网络管理更加简单、方便。

ü 内置AP热点地图，实时查看网络运行状态；支持控制器集群统一管理；IT/营销分权、分级管理。

行为管理路由器

ü 不同于传统基於单用户的流控，志成科技还提供了基于应用的流控以及基于用户数的流量，可根据不同的应用为不同的用户划分流量带宽和根据不同时间不同用户数量分配流量带宽，保证带宽的最大化利用率。

ü 可精准识别1900多种网络应用，并且根据用户、用户组、时间段、接入位置、终端类型等条件进行上网权限设置。

ü 无须单独购买VPN设备，支持点对点、网对网VPN模式组建专用网络，实现分院与总部间的资源远程共享和安全数据传输，避免信息泄露。

ü 灵活的WAN口配置，从容满足多种外网线路方案，支持多条运营商线路同时接入。

智能交换机

ü 所有交换机网口均采用3μm镀金工艺，耐磨、抗氧化、易导电，减少网络卡顿，提高用户体验。

ü 顶级的高端high tg/170主板、中端高频主板，远超FR4普通主板。“沉金工艺”，晶体结构更致密，不易产成氧化，在转发性能上远超一般的“镀金工艺”。

ü 遵循EEE节能标准，能够根据端口的实际流量自动调整能源消耗，可以迅速在全速运行和低功率闲置模式之间转换，为用户节省30%的功耗

ü 支持端口供电优先级，保证网络关键节点的持续供电，当有大电流和其它电力故障时，将自动启动短路功能切断供电，防止烧坏设备并可避免由线路故障或安装错误引起的网络故障。

3.1.1无线+有线


结合志成科技打造的全息AI综合运维管理平台，集成了有线网络管理和无线网络管理的所有功能，将无线优化、认证营销、安全加密等功能融合进一台设备之中化繁为简，实现只需一台设备即可实现有线和无线的统一管理，无论是无线开局调试还是有线和无线网络的后期维护一台设备即可搞定。

3.1.2计算机网络+安防网络

结合志成科技打造的全息AI综合运维管理平台，可实现计算机网络和安防网络的统一管理运维，一键管理计算机网络交换机、路由器以及安防网络交换机等通讯设备，满足集成化程度日益提高的弱电网络建设要求。


3.1.3网络设备+终端设备


传统网管缺乏对终端的监管，随着业务系统的多样化，终端设备状态管控的需求越来越急迫，终端设备跟网络设备的联动性对于网络故障的处理尤为重要，我司全息AI综合网络运维平台能够将网络中的终端设备一网打尽，无论是计算机网络中的台式/笔记本电脑、无线AP，还是安防网络中的摄像机、门禁、可视对讲等平台都能统一管理。

3.1.4云端管理+移动运维


志成科技不仅能够在本地对网络设备进行图形化管理，在外网接入的前提下，同样能够在云端平台对内网产品进行运维管理。极大的方便酒店网络管理人员随时随地管理维护酒店网络，提高处理问题时的响应速度。


设备可对接到云平台，运维人员可登入到云端，实时监测平台数据，可用手机，Ipad，笔记本等各种终端来管理。

运维管理人员通过移动设备（包括手机、笔记本电脑等）可在任何时间、任何地方通过云平台监控所有区域设备状态和管理。如有告警信息，则会通过短信或微信的形式推送到手机端，让我们在第一时间可以掌握网络的运行情况，并且解决故障不再需要亲临现场，完全可以远程解决。为运维人员提供一个极度方便的运维平台。


3.2 全局可视

3.2.1拓扑可视化


支持发现路由器、交换机、AP、摄像机、NVR、服务器等各类设备，自动生成逻辑拓扑图，全网架构清晰可见，实现真正的可视化运维管理，让非网络专业人士经过简单的培训也能轻松管理维护网络。

3.2.2场景可视化


通过导入项目的平面图纸甚至是项目的平面照片，然后标记出AP点位，就能够直观了解无线AP和网络设备在真实场景的部署位置，使得信道的部署和优化清晰可见，同时AP的功率大小、周边的干扰情况、终端的负载数及是否有弱终端等信息，均可通过平台把全网无线的运行状态尽收眼底，极大的方便了后期故障排查及扩容调研。

3.2.3管理可视化


全息AI综合运维管理平台支持基于百度地图的实时物理位置呈现，可通过云端跳转实现远程移动运维，云端采用“宏观到微观”的设计理念，支持省、市、县、乡镇街道等多级地理空间的管理方式，可以让用户的项目实现全局、可视化、集中管理，并且支持多级账号权限管理架构，可以实现集团账号、代理账号、普通账号三级架构的管理模式，让运维管理打破空间距离的限制，极大的缩减了运维成本以及提高了运维效率。

3.3 智能运维

3.3.1故障智能告警


全息AI综合网络运维平台的提供故障的智能告警，包括：网络设备/终端设备的状态告警（在线/离线）、链路流量异常告警、终端流量的异常告警、对于无线提供信道冲突告警、终端负载告警、弱终端告警、钓鱼AP告警以及网络非法接入、网络环路、ARP欺骗、IP地址冲突等网络类故障告警，基于可视化的逻辑组网拓扑，能够及时精准的定位出网络故障的具体位置，让网络故障和风险无处遁形。

3.3.2故障的智能分析


在故障出现后，全息AI综合运维管理平台会基于AI人工智能和深度学习技术，结合我司售后专家团队，丰富的现网问题处理经验及知识库，对故障现象进行智能分析，输出问题处理的预案，给故障的智能处理献计献策。

3.3.3故障的智能处理

多智能业务云管理平台支持用户黑白名单设置，可一键阻止非授权用户连入无线网络，也可以一键允许授权用户无须认证直接连入无线网络。

4.2 产品介绍

4.2.1 上网行为管理路由器


RE系列上网行为管理路由器是针对政府、教育、企业、酒店等行业在智能流控、上网行为管理方面的需求推出的新一代网络产品。它采用业界领先的电信级网络处理器并搭配高速DDR3内存，内置高精准DPI识别引擎，可快速高效识别HTTP协议、网络游戏、网络电视、网络音乐、网络电话、P2P下载、聊天软件、股票交易、移动应用等协议。基于精准的DPI识别，可实现各种应用调度、智能均衡技术，让企业的网络带宽真正的用到实处，同时在保证用户上网体验的前提下对企业安全进行一系列的上网行为管控。该系列路由产品搭载志成科技具有完全自主知识产权的网络操作系统，用户在未来可通过我司APP商店或第三方路由模块开发商随意打造自己的路由器系统。

产品特点：

n 基于APP功能模块的TOS网络操作系统

TOS是我司具有完全自主知识产权的网络操作系统，是一套基于模块化、可扩展性极强的软件框架，各功能模块从软件内核到WEB界面实现了高度的模块化。TOS软件先进的架构允许我们采用互联网产品开发的典型模式：在线增量更新，在确保主体稳定的情况下快速演进,我们可只更新用户关心的一小部分功能或者APP，避免整体更新固件版本带来的各种潜在风险，大幅降低用户的运营风险。

n 全千兆硬件架构，高速稳定的性能

RE系列路由器采用业界领先的电信级网络处理器，最高主频达1.6GHz，同时配备高速的DDR3内存，结合我司独有的硬件加速技术，快速识别应用层协议并执行迅速有效的应用调度，能将吞吐量及带机量提高数倍。宽频电信级电源设计，采用电信级的开关电源，具有防雷设计、防过压设计、防浪涌设计，电压可适应110～240V大范围，保证在不良自然天气及电压不稳环境下网络的正常运行。

n 基于应用层的智能流控和应用分流

通过核心高精准DPI引擎，深度识别内网七层应用程序，如游戏、网页、P2P等应用，通过内置的智能弹性流控功能给每个用户执行最为合理的带宽调度。用户只需设置好广域网带宽，设备会根据网络使用情况，时时优化带宽资源，保障时时敏感业务优先转发。用户也可手动指定具体应用程序走某条外线出去，甚至可以指定应用程序的某些特定端口分流走不同的外线出去。

n 支持多WAN接入和智能选路

单机最多支持四WAN接入，可组合E-24WAN将一个WAN口扩展成为23个WAN口同时使用，极大的满足国内部分区域带宽紧张的运营情况。在多条外线之间可实现智能选路，即“电信流量走电信线路，联通流量走联通线路”；对于同一ISP多条线路，可以实现线路叠加，即将流量均衡分配到各条线路；对于线路带宽不一样的多条ISP线路而言，可以通过权重模式将流量分配到合适的线路以充分利用多条线路的带宽。同时系统支持自动更新运营商IP地址库信息，以保障各ISP之间智能选路的准确性。

n 支持上网行为管理

支持URL关键字过滤、文件后缀名过滤、邮箱过滤、网络发帖过滤，让企业员工上班时间只能访问工作相关资源；

支持QQ黑白名单，让企业员工上班时间只能登陆工作QQ，提高工作效率；

支持迅雷、BT、电驴、FlashGet等P2P软件过滤，保障企业网络带宽利用；

n 支持内网防火墙功能

支持五元组ACL，用户可通过源IP地址、源端口号、目的IP地址、目的端口号、四层协议号来任意配置所有的访问控制列表；

支持IP/MAC绑定、ARP攻击防御、APR信任机制；

支持连接数限制、MAC过滤；

可防止短包、防止碎片包；

支持DDoS攻击防御；

第五章 成功案例

5.1 无锡市惠山区第三人民医院

无锡市惠山区第三人民医院位于无锡市惠山区玉祁（街道）镇，地处无锡市西北之郊，西接常州，北临江阴，京杭大运河，京沪铁路贯境而过，沪宁高速公路设有道口。水陆交通便利，是个气候宜人，物产丰饶，人文荟粹，具千年历史江南名镇。玉祁镇亦为无锡市经济重镇，年总产值达300亿以上。

惠山区第三人民医院，目前正在按照国家二级综合性公立医院异地新建中，建设规模2.7万平方米，占地面积达45亩。预计2017年中期搬迁。现有职工180人左右，拥有CT、CR、彩超、全自动生化仪、电子胃镜、层流手术室等先进设备。服务人口10万左右，增长较快。


客户需求

惠山区第三人民医院，目前正在按照国家二级综合性公立医院异地新建中，新址网络设计需要重新部署包含有线网络及无线网络，其中无线网络是开放给病人及病人家属使用的，针对于无线需要采用高性能设备、达到室内无线全覆盖效果，并且能管控用户上网行为、保证内网数据安全。

解决方案

采用RE9500行为管理路由器作为出口路由，出口路由器下连万兆三层核心交换机S5300-32F-4TF，无线管理控制器M5旁挂在核心交换机S5300-32F-4TF上，核心交换机采用双线汇聚方式下联全千兆管理型交换机S3500-26G-2F千兆电口/光口，采用多台P2000及P3000系列POE交换用于AP供电、数据传输及有线网络设备传输。

其中RE9500行为管理路由器，基于精准的DPI识别，可实现各种应用调度、智能均衡技术，让网络带宽真正的用到实处，同时在保证用户上网体验的前提下对企业安全进行一系列的上网行为管控。在核心交换机和接入交换机上开启环路保护、端口隔离和广播风暴抑制。实现有效隔离非法DHCP，防止私接路由器造成的不良后果，给予用户良好的wifi体验。减少广播风暴以及ARP病毒的爆发，优先保护服务器区域的网络安全。无线控制器内置云智能管理平台，可管控内网所有TG设备，自动生成网络拓扑图，远程访问所有TG设备，故障实时告警，实现远程运维，减低运维成本；具有多种无线认证方式功能，TG云平台可实现多种无线推广业务。

使用效果

整体网络实现了千兆主干+双千兆汇聚+千兆接入的组网架构，能够充分保障汇聚、核心对大流量业务的承载能力， 行为管控路由器自带的应用调度、智能均衡技术充分利用外网带宽并合理的调节终端带宽资源，同时内网所有设备都支持web网管，极大的方便了后期维护操作，并且留下了再次扩容的接口，为后期实际应用扩容提供了保障。

5.2 宝鸡中心医院

项目背景

宝鸡市中心医院始建于1951年，前身为中国人民解放军西北军区第七陆军医院，后移交地方几经隶属更为现名。经60余年建设发展，现已成为集医疗、教学、科研、预防、康复、保健为一体的综合性三级甲等医院，先后荣获全国百佳医院、全国首批百姓放心示范医院、全国卫生系统先进集体、陕西省文明单位、陕西省先进基层党组织等称号。涌现出全国优秀共产党员、党的十五大代表、全国先进工作者王云侠，全国卫生系统先进个人贺志新、顿国亮，全国五一巾帼标兵杨凤玲等一大批先进模范人物。

医院占地170亩，编制床位1600张，现有职工2400余人，设临床、医技科室55个，年接诊患者72.23万余人次、出院患者5.95万余人次、完成手术2.9万余例。现有中高级专业技术人员791人，博、硕士研究生225人；国家级学会常委2人、委员11人，省级学会副主委9人、常委39人；宝鸡市“三五人才工程”3人；市级有突出贡献拔尖人才18人；省、市级劳模8人。是国家住院医师规范化培训基地、国家药物临床试验机构，陕西中医药大学、西安医学院非直属附属医院，宝鸡市基层卫生人员培训基地、宝鸡市全科医生培训基地。是四川大学华西医院/华西第二医院区域联盟网络医院、第四军医大学西京医院军地共建医院、北京阜外心血管病医院宝鸡技术协作中心。


客户需求

1.医院室内全面无线覆盖，为终端提供高品质的无线接入点

2.医生和护士需移动办公，通过WiFi网络对终端用户进行认证

3.实现大厅、病房、门诊等地随时随地接入WiFi

4.整个网络可分为应用层、核心层、传输层和客户终端几部分共同构成

解决方案

网络接入为P2000M系列POE交换机连接WA3122i AP，根据用户数量及上网需求流量大小，决定采用我司S5300-32F-4TF作为总核心交换机，医院各楼层POE交换机到核心使用千兆单模光纤互联，核心交换机到出口路由采用超六类网线互联，出口采用RE8500上网行为管理路由器。

宝鸡中心医院组建不再受到有线传输介质布线的限制、接口数量大、灵活方便，满足移动医疗的需求，架设稳定的无线网络WIFI，极大地提高工作效率、方便医护人员、为病房提供稳定的无线网络，为医院信息化的发展起到非常大的推动作用。
