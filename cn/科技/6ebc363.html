02、案例分析

试题编号 : 01

知识点 : 送电线路

某220kV线路开关B相故障跳闸，重合不成功，高频距离、高频方向保护动作跳闸，当时天气为大风大雨并伴有打雷。随后，强送成功。检查发现#265杆B相（中相）导线在防震锤处附近有明显放电痕迹，横担、接地引下线、避雷线及连接金具未发现明显放电痕迹。#265杆型为ZLH1-20.45型直线杆，位于山顶，绝缘子串为13片LXY1-70型绝缘子，架空地线为双地线，保护角为13.92o。接地电阻值为13欧。试分析线路跳闸原因，提出防范措施。

引起绝缘子闪络的可能性是雷电过电压、操作过电压和污闪。从当时的天气情况看，线路跳闸是雷击引起。水平排列的中相闪络，可以肯定雷击性质为反击。重合不成功是由于重合闸时间短、重合闸时放电通道的空气绝缘尚未恢复。该杆位于山顶，容易遭受雷击，接地电阻不大，保护角也符合规程规定而且雷击性质为反击，因此应采用加强绝缘或加装线路避雷器措施。

试题编号 : 02

知识点 : 送电线路

某500kV线路开关跳闸，重合成功，线路B相故障。天气为阴天，无雷电大风。检查发现#190塔B相（中相）导线、悬垂线夹、横担下层塔材有放电痕迹，在横担下层塔材面上还发现有鸟毛被烧成灰的痕迹，铁塔拉线上把上有大量鸟粪痕迹，在横担上层塔材面上安装有防鸟刺。横担下层塔材放电痕迹的投影与导线距离为60cm。#190塔为拉V塔，位于一小片甘蔗地内，地形为平原，四周空旷无树，在距该塔减号侧约500米处有一大水塘。试分析线路跳闸原因，提出防范措施。

根据天气为阴天、无雷电大风，排除雷击和大风引起外物短路的可能；根据天气为阴天和放电通道为空气的情况，排除污闪的可能；从周围环境情况看，该段落属于鸟害区，从现场放电痕迹和鸟粪、被烧的鸟毛情况，可以认定此次跳闸为鸟粪所致，即鸟粪造成导线对铁塔横担的综合间隙距离不足引起放电短路跳闸。该塔已采取了防鸟害措施，即在横担面上安装有防鸟刺，但没有考虑到鸟仍可站立在中相横担的下层塔材上。对该线路铁塔的中相横担的下层塔材上补装防鸟刺；清查鸟害区的线路杆塔横担是否还存在类似绝缘子上方鸟仍可站立的情况，如有则补装防鸟刺或加装驱鸟装置。

试题编号 : 03

知识点 : 送电线路

x月x日，当地天气寒冷并伴有雨夹雪，某110kV线路开关跳闸，重合不成功，A相接地。护线员汇报：#65～#66杆间避雷线覆冰有5～6cm，造成右避雷线下沉对右导线的安全距离不足（约20cm）而对导线放电，并造成避雷线在距#65塔约150米处断股4股；#65～#66左避雷线弧垂增大，与左相导线相平，空间距离约1.4m；电杆、铁塔、绝缘子均严重复冰。#65～#67为一个耐张段，均处于大山区，#65为耐张塔（JG2-15），#66为直线双杆（ZL2-18），#65～#66档距较大（619m），#66～#67档距较小（237m），#66杆往#65倾斜约5°。该线路是某县唯一的供电电源，停电对当地影响很大，需临时恢复供电。请作出处理决定并说明理由。

首先线路覆冰已大大超过了设计标准，严禁上杆塔工作。处理方案是：采用射绳枪、小火箭抛绳设备、遥控飞艇、弓箭、弹弓、绝缘杆或其他方法把细绳抛过杆塔，拖引麻绳，再拖引钢丝绳，对杆塔避雷线挂点处加临时拉线加固；以前面同样方法把细绳抛过右相导线，拖引麻绳，采取绝缘绳串接复合绝缘子后把导线拉到符合送电的安全距离后送电。

试题编号 : 04

知识点 : 送电线路

人们担心：变电站使用玻璃绝缘子自爆时炸坏设备。请根据玻璃绝缘子的运行情况做出回答，并谈一下玻璃绝缘子的特点。

这是对玻璃绝缘子的误解。玻璃绝缘子的“自爆”又称“自破”，并非“自爆炸”，是绝缘子玻璃体内部应力的平衡受到破坏而使玻璃体自行粉碎，不产生横向的力，也没有产生横向力的能量。绝缘子玻璃体粉碎后，其碎片作为自由落体跌落地面，最大碎片约2cm，一般为1cm米下，碎片的重量和势能能量很小，而且碎片刚度低、易碎，因此不会对设备造成危害。玻璃绝缘子已在变电站中使用，没有出现玻璃绝缘子自爆时炸坏设备的情况。玻璃绝缘子有内部缺陷（例如玻璃体内残存有杂质）或受大的外力冲击时，绝缘子玻璃体外层压应力和内部张应力的平衡受到破坏，内部的张应力得到“释放”，使玻璃体破裂粉碎，这就是玻璃绝缘子的“零值自破”。零值自破是钢化玻璃绝缘子一大特点，也是一大优点。玻璃绝缘子的自破，多发生在施工和运行初期，随着时间的延长，自破率逐年下降并趋于稳定。只需凭借目测就可方便地检测出老化的绝缘子。就国产绝缘子而言，玻璃绝缘子的年自破率比瓷绝缘子的年零值率低。玻璃绝缘子的耐电弧能力比瓷绝缘子强。玻璃绝缘子是不可击穿的，玻璃绝缘子零值自破的特点使它的外绝缘水平永远小于内绝缘水平而不击穿，具有零值绝缘子的玻璃绝缘子串，当发生污闪或雷击闪络时，短路电流在零值绝缘子铁帽和钢脚的外部通过而不经过铁帽内部（绝缘子自破后其铁帽和钢脚间的空间距离短），这就大大减少玻璃绝缘子掉串的概率。零值自破后的残余强度仍然很高，均大于额定负荷的标准值，因此不会造成导线落地事故。

试题编号 : 05

知识点 : 送电线路

某110kV线路开关跳闸，重合成功。当时天气晴好。巡视发现#21钢管塔C相（右下相）绝缘子有轻微放电痕迹（球头上有放电痕迹，复合绝缘子伞群无闪络痕迹），C相导线上有少量鸟粪痕迹，且发现C相的横担内有鸟窝及大量杂物、杂草，现场实测杂草有1米长。经询问附近群众得知，当时线路有放电引起的火光及声响，且有鸟飞进横担内。请分析原因，提出反事故措施。

从当时的天气情况、现场情况及附近群众所观察到的现象可以判断：此次跳闸是由于鸟在杆塔横担内筑巢，叼入杂物及杂草时对导线的安全距离不足引起。暴露问题：钢管塔的横担是中空的，且在横担的杆塔侧没有封口，鸟类易在横担内筑巢。防范对策：统计现有钢管塔的线路名、数量及杆号，制定钢管塔横担口封堵方案、对钢管塔横担口实施封堵，或加装驱鸟装置；对于新建工程，制定钢管塔的技术规范书时要求横担封口。

试题编号 : 06

知识点 : 送电线路

在对线路导线接头进行测温时，发现导线接头温度比环境温度低。请作出判断分析和工作指导。

测量数据明显不对。造成这一原因可能是：测量距离太远，超过了被测目标（导线接头）相对应的有效测量距离；仪器的有关参数如辐射率的调整失当；仪器损坏。先就近测量其他物体的温度以判断仪器的好坏；正确调整测量参数；根据仪器距离系数计算出接头的有效测量距离，把仪器拿到接头的有效测量距离内对导线接头进行测温。

试题编号 : 07

知识点 : 送电线路

在对线路进行接地电阻测量时，接地摇表摆动不定。请作出判断分析和工作指导。

接地摇表摆动不定，可能的原因是：地极打入深度不够或因松动、土壤电阻率过高而与土壤接触不好；仪器损坏；附近有变电站或其他电力设施在地中产生较大的地中电流。先对摇表进行开路和短路测量检查以判断仪器的好坏，如损坏，则更换好的仪器；把地极全打入地中并用水淋地极及其周围。如这样处理后，接地摇表仍然摆动不定，则是地中电流影响，需用大电流法或变频接地电阻测试仪测量。

试题编号 : 08

知识点 : 送电线路

在对某线路的一基杆塔进行接地电阻测量时，测量值很小（小于1Ω），再反复进行几次测量，各测量值变化大。请作出分析判断和工作指导。

测量值很小（小于1Ω），其原因有：杆塔位于土壤电阻率很小的地区；接地网在地下或基础仍与铁塔相连；接地极打对了接地网。各复测值间变化大，其原因有：电压极和电流极与铁塔间的距离太小，从而打在接地网的半径范围内，在每次复测时，电压极和地网中心的距离与电流极和地网中心的距离的比例变化大；各个方向的土壤电阻率差别大；该接地网为引外式接地网，从而地网中心不在铁塔处。综合以上分析，出现题目中现象的原因是电压极和电流极与铁塔间的距离太小、不满足DL/T887-2004《杆塔工频接地电阻测量》要求。测量人员应按DL/T887-2004要求，把电压极P和电流极C分别布置在离杆塔基础边缘2.5L和4L处，L为杆塔接地装置放射形接地极的最大长度。

试题编号 : 09

知识点 : 送电线路

有一铁塔，采用重力式基础，露出柱体为0.8mx0.8m，底宽度2.5mx2.5m，埋深3m，土质为可塑土（上拔角α为20°，摩擦角β为30°，容重γ0为16kN/立方米），陡坡40°，现距离基础柱体边缘1.8m处出现边坡塌方，塌方长度8m、高度1.8m，已看见基础。请提出基础处理方案。

该基础为重力式基础，因此基础边坡塌方不影响基础的上拔稳定性。基础边坡塌方的高度为1.8m，小于基础埋深3m，因此基础边坡塌方不影响基础的下压稳定性。基础处理的关键是控制基础边坡继续塌方，当塌方高度大于基础埋深时，将影响到基础的下压稳定性。故应采用以下方案：在基础的上边坡开设排水沟；回填土，夯实。


试题编号 : 10

知识点 : 送电线路

有一铁塔，采用重力式基础，露出柱体为0.8mx0.8m，底宽度2.5mx2.5m，埋深3m，土质为可塑土（上拔角α为20°，摩擦角β为30°，容重γ0为16kN/立方米），陡坡50°，现距离基础柱体边缘1.8m处出现边坡塌方，塌方长度8m、高度3.5m，已看见基础。请提出基础处理方案。

该基础为重力式基础，因此基础边坡塌方不影响基础的上拔稳定性。基础边坡塌方的高度为3.5m，大于基础埋深3m，因此基础边坡塌方已影响基础的下压稳定性。基础处理的关键是恢复已塌方的边坡、控制基础边坡继续塌方。故应采用以下方案：在已塌方的基础边坡砌筑护坡；在基础的上边坡开设排水沟。

试题编号 : 11

知识点 : 送电线路

有一铁塔，采用阶梯式基础，因处于边坡上，为保护植被及增加基础的稳定性，设计时已加大埋深（正常埋深2.5m，现埋深3.5m），露出柱体为0.8mx0.8m，底宽度2.5mx2.5m，土质为硬塑土（上拔角α为25°，摩擦角β为35°，容重γ0为17kN/立方米），陡坡52°，现距离基础柱体边缘1.5m处出现边坡塌方，塌方长度10m、高度1m。请提出基础处理方案。

该基础为阶梯式基础，依赖基础上方的土中来保证基础的上拔稳定性。基础边坡塌方处已位于基础的上方，基础的抗拔力已有所减少，但由于基础已加大埋深1m，而现塌方高度正好1m，因此基础塌方后基础的上拔稳定性仍然得到保证。基础处理的关键是控制基础边坡继续塌方。故应采用以下方案：在基础的上边坡开设排水沟；回填土，夯实。

试题编号 : 12

知识点 : 送电线路

在用火花间隙法对线路瓷绝缘子进行零值测量时，有一组人测量了3基杆绝缘子，每片绝缘子均没有响声，然后换一组人测量时，每片绝缘子均有响声。请作出分析判断和工作指导。

用火花间隙法测量零值瓷绝缘子时，为提高测量零值瓷绝缘子的准确率，火花间隙在带到现场使用前应进行间隙放电试验，调整好间隙，在每次上杆塔测量时，要用塞尺检查间隙的变化情况。有一组人测量每片绝缘子均没有响声，可能是间隙发生了变化，间隙距离增大了；换一组人测量时每片绝缘子均有响声，可能是间隙发生了变化，间隙距离减小了。应检查间隙距离是否符合要求。另外，如果绝缘子增加较多，每片绝缘子的分布电压则减小较多，如仍用常规绝缘子串的间隙进行测量，也可能出现每片绝缘子均没有响声的情况，测量时需注意这一情况，根据电压分布调整相应的间隙距离。

试题编号 : 13

知识点 : 送电线路

某线路登杆塔检查发现有几基杆塔绝缘子串的U形环磨损严重（已磨损了直径的一半）。请分析原因，提出防范措施。

原因分析：U形环磨损严重，外部原因是由于导线受到不规则的上下左右振荡，使金具和绝缘子串受到很大的冲击拉力造成，这常发生在特殊地形地段，如风口；内部原因是金具的连接灵活性差，使金具受到较大的弯矩，久而久之，金具就严重磨损，甚至疲劳断裂。防范措施：首先调查特殊地形杆塔，调查金具的磨损情况以找出磨损严重杆塔。然后对特殊地形和磨损严重杆塔，把金具更换为高强度的金具，或更换为双挂点双串，同时将绝缘子与横担的连接用两个U形环连接以增加金具的连接灵活性。

试题编号 : 14

知识点 : 送电线路

有一耐张转角塔JG2-18，运行若干年后内倾5‰。请提出处理方案。

试题编号 : 18

知识点 : 送电线路

有一耐张塔JG2-18，经竣工验收检查发现内倾3‰。请分析判断其原因并提出处理方案。

造成铁塔内倾的原因有：基础预偏不够，钢材不满足设计要求，铁塔螺栓松。铁塔内倾的原因分析判断：先检查钢材是否满足设计要求，如果规格偏小或强度偏低，则铁塔内倾是钢材不满足设计要求造成的。再测量基础预偏是否符合要求，如果基础预偏符合要求，则铁塔内倾是由于铁塔螺栓松造成的；如果基础预偏不符合要求，则比较基础减少的预偏量与铁塔的内倾量再判断。如果基础减少的预偏量大于铁塔的内倾量，则铁塔内倾是基础预偏不够造成的；如果基础减少的预偏量小于铁塔的内量，则铁塔内倾是由于基础预偏不够及铁塔螺栓松共同造成的。处理方案：如果钢材不满足设计要求，则更换钢材。如果基础预偏不够，则用千斤顶顶起塔基，垫高基础，再进行基础处理。如果铁塔螺栓松，则打好每根导线的反向临时拉线，对地夹角不大于30°并尽量小，调节临时拉线使铁塔外倾，然后逐段松螺栓和紧螺栓。如果同时存在基础预偏不够及铁塔螺栓松，则打好每根导线的反向临时拉线，对地夹角不大于30°并尽量小，调节临时拉线使铁塔外倾，然后垫高基础，并逐段松螺栓和紧螺栓。

利用人字抱杆起吊杆塔，抱杆座落位置及初始角的大小对起吊工作有何影响?

答：利用人字抱杆起吊杆塔时，若抱杆座落过前，初始角过大，在起吊过程中，易造成抱杆脱帽过早，增大主牵引绳的牵力，并且起吊过程中，稳定性差。抱杆座落过后，初始角小，起吊过程中易造成抱杆脱帽晚。这种情况初始时主牵引力和抱杆受垂直下压力都会增大，特别是在抱杆强度不足的情况下，会造成抱杆变形破坏，对起吊工作是不利的。所以，在施工中抱杆的初始角和座落位置一定要按施工设计进行。

知识点 : 工器具知识

简述钳形电流表的用途和工作原理。

答：通常用普通电流表测量电流时，需要切断电路才能将电流表或电流互感器一次绕组串接到被测电路中。而使用钳形电流表进行测量时，则可在不切断电路的情况下进行测量。其工作原理如下：钳形电流表由电流互感器和电流表组成。互感器的铁心有一活动部分，并与手柄相连，使用时按动手柄使活动铁心张开。将被测电流的导线放入钳口中，放开后使铁心闭合。此时通过电流的导线相当于互感器一次绕组，二次绕组将出现感应电流，其大小由导线的工作电流和圈数比确定。电流表是接在二次绕组两端的，因而它所指示的电流是二次绕组中的电流，此电流与工作中的电流成正比。由于将归算好的电流标在电流表的刻度上，当导线中有工作电流通过时，和二次绕组相连的电流表指针便按比例发生偏转，从而指示出被测电流的数值。
