前言

本文主要介绍的是C语言位运算的一道题，这是微软笔试题中的一道比较简单的的题目，希望对于广大读者学习C语言有一些帮助。深入了解C语言小知识，看题讲程序作用：

int func(x) {     int countx =0;     while(x)     {           countx ++;           x = x&(x-1);      }     return countx; }

知识解析

  1. &：按位与运算
  2. a&b：先把a和b转换为二进制然后 按照每一个二进制位按照逻辑与的运算规则去运算
