8个Windows骚操作，学会瞬间提升工作效率

2020-10-29
| 科技

大家晚上好呀~


今天小Wo要给大家分享的是 8 个关于 Windows 的骚操作，每一个都好用好哭！如果你全部知道，真的就比很多人厉害 10 倍！


01

拖一拖，居然可以保存图片？

每当我们遇到好看的图片时，常规操作是 「右键」-「图片另存为」，选择文件夹保存。其实不用这么麻烦，直接拖拽就可以保存图片了！

将图片拖拽到屏幕右下角，屏幕会立刻返回桌面，继续拖动鼠标把图片放在桌面即可。


02

1 秒返回桌面：Win + M

用电脑时，窗口越开越多，看得人心烦意乱，注意力分散。按住 「Win + M」 键就可以把当前所有窗口收起来~

除了 「Win + M」 ，你也可以使用「Win + D」，同样可以秒回桌面，这组快捷键的优势在于：重复按一次，就能恢复被最小化的窗口。


如果你想保留当前窗口，那就按「Win + Home」 ！


03

快速找回误关网站：Ctrl + H

一不小心误关了正在浏览的网页怎么办？ 「 Ctrl + H 」帮你快速找回。


使用快捷键「 Ctrl + H 」，跳转到历史记录页面，很容易就能找到刚才打开的网站。


04

快速查找所需内容：Ctrl + F

找回误关网站后，怎么定位到刚才正在看的部分呢？滚动鼠标滑轮也太慢了！这时候小Wo教你一招，「Ctrl + F」快速查找所需内容。

例如，我刚才正好看到这篇文章的配色篇，「Ctrl + F」调出搜索框，输入 「配色」，按回车键，自动定位到关键词~

PS：这个快捷键不仅适用于网页，Office 也同样适用哦~


05

分屏：Win + 方向键

相信你一定遇到过，打开了太多的程序，难以找到自己需要的窗口的情况。

每次切换都要使用 「Win + Tab」，着实麻烦。其实 Windows 早就为我们提供了分屏的功能。

具体操作是：

「Win」+ 「←/→」：在屏幕左/右占据一半

「Win」+ 「↑」：最大化

「Win」+ 「↓」：最小化



这样就可以一边编辑文档，一边使用浏览器查询资料了~ 对于经常要写 PPT、写论文的人来说简直太友好了！


06

快速预览文件内容：Alt + P

如果文件保存时间太长或没有命名，很容易忘记它是什么，这时我们可以通过快捷键 「Alt + P」，召唤出右侧的 「文件预览窗格」，快速预览文件主要内容。


这个功能支持查看 Word、Excel、PPT、TXT，连视频和 AI 都能预览！但是不支持PDF。以后找文件的时候再也不用一个一个打开看啦~



如果想根据时间查找文件，按 「Alt + Shift + P」，就可以看到文件的修改日期、大小等基本资料，而且所有文件均可查看！



07

屏幕录制：Win + G

经常演示操作的小伙伴，录屏功能自然必不可少。很多人会选择第三方录屏工具，其实Windows自带的 Xbox 录屏，非常好用~不需安装，操作简单，容易上手

打开想录制的窗口，按下 「Win + G」。如果无法调出，请在 「设置」- 「游戏」- 「游戏 DVR」，打开录屏。

PPT by @秦阳团队


录制完成后视频文件默认为 mp4 格式，保存在默认文件夹。这个功能简单直白，录制后自动保存，几乎没有学习成本。


08

剪辑神器：照片

视频不可能录的尽善尽美，大多数时候都需要剪辑，一般人的需求无非就是剪一下视频长度、或者给视频配个 BGM。如果是这种简单的需求，无需 Pr，用好 Windows 自带的「照片」软件就够了。

功能1：简单视频剪辑，无需安装 Pr

比如我手里有这样一段视频，在一开始的时候总是有一段黑影，我想把它去掉

这时候，选择用 「照片」应用打开这段视频，找到左上角的「剪裁」功能。


首先把蓝色圆点拖到黑影结束的地方，再把左边的白色圆点挪到画面开始的地方，这样就把能开始的一段黑影剪裁掉了，点击「保存副本」搞定！

视频剪裁完后，往往还需要添加背景音乐。我们可以一次找到「创建带有文本的视频」-「背景音乐」。这样一段视频就简简单单完成剪辑啦~


除了剪视频， 「照片」还有两个令人惊艳的功能：


功能2：简简单单修图，无需动用 PS

说到修图，大家立马想到美图秀秀、PS。殊不知，在你的 Windows 电脑里安安静静躺着的「照片」完全可以满足简单的需求。

拉伸、旋转、翻转、剪裁、加滤镜......这些常规的修图操作用 「照片」可以轻松实现。

功能3：超炫酷的 3D 效果，照片里也有

如果你是一个会玩的人， 一定会喜欢新增的「3D 功能」！

比如下面这张动图，就用到了照片其自带的「科幻传送门」、「竖直闪电」、「翩翩起舞」3 个 3D 效果，是不是非常炫酷呢？

其实在「效果」一栏里，这样的 3D 素材还有很多，如果能跟视频很好地配合起来，绝对是一波抢眼的骚操作！

除此之外，助攻姐还发现了一个宝藏——「3D资源库」，里面全是3D素材！

这些素材直接在 「照片」操作比较麻烦，但是如果你放到 PPT 里，就能做出这样炫酷的页面！！！
