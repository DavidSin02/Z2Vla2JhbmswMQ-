LeetCode算法第76题：最小覆盖子串

2020-10-29
| 科技

问题描述：

给你一个字符串 S、一个字符串 T，请在字符串 S 里面找出：包含 T 所有字母的最小子串。

示例：

输入: S = "ADOBECODEBANC", T = "ABC"输出: "BANC"

说明：

如果 S 中不存这样的子串，则返回空字符串 ""。

如果 S 中存在这样的子串，我们保证它是唯一的答案。

思路：

可以在字符串 S中通过滑动的方式来查找从起始位置开始包含字符串 T的最短的字串，然后和已有的字串进行比较，找到最小的字串。因为需要查找最小字串，因此字串的起始字符必然包含在字符串 T。可以定义一个变量map来记录字符串 T中每个字符包含的数量，变量total来记录字符串 T字符的的数量。

定义一个指针标识字符串 S中字串的结束位置，如果指针所在字符包含在字符串 T中，map中记录的对应个数减一，同时当这个记录数不小于0 时，total 减一。当total 大于 0 也就是字符串 S的字串不能覆盖字符串 T的时候，指针右移。当total等于0时，可以从起始位置到指针当前位置获取一个覆盖字符串 T的字串。如果字符串 S遍历未完成，可以移动起始位置，直到新的起始位置满足条件，即起始位置的字符包含在字符串 T中，同时map中对应的个数大于0，此时重复移动指针获取覆盖子串的操作，知道遍历完成。

Java代码：

public String minWindow(String s, String t) { Map<Character,Integer> map = new HashMap<>(); int total = 0; String result = null; for(char c : t.toCharArray()){ map.put(c,map.getOrDefault(c,0) + 1); total ++; } int start = -1; for(int i = 0; i < s.length(); i++){ char c = s.charAt(i); if(null != map.get(c)){ if(start < 0){ start = i; } int cnt = map.get(c); if(cnt > 0){ total --; } map.put(c,cnt -1); if(total == 0){ result = null == result? s.substring(start,i+1) : (result.length() < (i -start + 1) ? result : s.substring(start,i+1)); for(int j = start;j <= i; j++){ char ch = s.charAt(j); if(null != map.get(ch)){ if(total == 0){ int ct = map.get(ch); map.put(ch,ct+1); if(ct == 0){ total ++; } }else{ start = j; break; } } } } } } return result; }
