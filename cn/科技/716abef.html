PS1迷你主机游戏棒 - PlayStation游戏库无限扩容神器实测！

2020-10-29
| 科技

1994年12月，日本发售的PlayStation初代主机，正式吹响了索尼进军游戏业界的号角。说起PlayStation，其实有一段颇为有趣的历史呢：

最初，是任天堂请求索尼为其家用游戏机开发一款CD-ROM光盘游戏的附件，这个附件暂定名为Play Station或SNES-CD。为此两家当时还成立了一家合资企业。然而在签订合作协议之后，索尼决定自行开发一款不仅能玩超任游戏还能播放CD光盘的娱乐设备，说干就干，索尼1991年5月在CES上正式公布了这款娱乐设备——Play Station！

时任任天堂社长的山内溥在索尼公布Play Station原型机之后，意识到任天堂和索尼的合作协议实际上赋予了索尼对所有超任主机游戏的所有权和控制权。于是，山内溥果断取消SNES-CD的开发计划，并中止了和索尼的合作。双方中止合作之后，任天堂顺利的找到了飞利浦来开发CD附件；而另一边索尼邀请世嘉开发主机，最终被世嘉断然拒绝，理由是索尼根本不懂如何制造游戏机和游戏。

把我忽悠来，却不带我玩了？！索尼一怒之下，决定自己干了。推翻了超任主机的设计，完全删除卡槽，采用3D多边形图形开发。最终PlayStation成功发售。自此索尼一发不可收拾，苦心耕耘，并在接下来的20多年，将PlayStation打造成为全世界最受欢迎的主机系列之一！

PlayStation系列的开山鼻祖！

而今，索尼将这款经典的PlayStation初代主机Mini化重置，命名为PlayStation Classic，年轻一代的玩家现在也能感到到原汁原味的PS1游戏了。虽然外观一样，但是中那个机器盖是不能打开的，机器内置了20款数字版的经典游戏，包括老玩家耳熟能详的《最终幻想7》、《山脊4》、《贴铁拳3》……

MINI版比原版缩小了45%

PlayStation Classic主机采用了1GB RAM，比老任家Mini FC和Mini SFC的256MB大得多，但是游戏却仅仅提供了20款，这让很多玩家望而却步了，相对于700多的售价来言，20款游戏有些诚意不足啊。

动手能力强的小伙伴可能已经自行研究灌游戏的方法了，不过这种方法我们就不推荐了。毕竟不是正常手段嘛，而且我真的看不懂那些教程啊，看起来就很复杂。万一再把主机折腾坏了呢？

是时候考虑下 TrueBlue MINI了。一款适用于PlayStation Classic主机的游戏棒。简单来说，TrueBlue MINI是一款内置PS1游戏的U盘，只需要将其接入到PlayStation Classic主机上就能直接玩到多达101款经典PS1游戏。重要的是，U盘内置系统，不需要任何操作，也不会对主机核心做任何修改。真正的安全绿色无污染呢！

加上Hub之后，双人游戏也不受影响

这游戏阵容，比起原版的20个可强太多了

TrueBlue MINI一共有三款，True Blue Mini - Fight Pack ( 32 GB-内置了64款经典游戏 ) / True Blue Mini - Meth Pack ( 64 GB-内置了101款经典游戏 ) / True Blue Mini - Crackhead Pack ( 64 GB-内置了101款经典游戏 )，这其中两款64G收录的101款游戏是完全不同的，可以根据自己的喜好来挑选。（游戏列表可以在官网查看，点击进入）

打开包装之后，就能看到TrueBlue的真容了！

基于U盘打造的蓝色TrueBlue，此外包装中还附送了一个4口HUB线（底部还有一个）。

单人游戏的话，安装非常简单，TrueBlue游戏棒直接插2P口，然后开机玩就行。

如果需要双人游戏，也很方便，需要将Hub接入2P的USB接口，再将TrueBlue游戏棒和2P插上Hub口上，然后开机玩就支持双人了。

总结：TrueBlue游戏棒，一款完美适配索尼PS1经典迷你主机的配件。即插即用，安全简便，内置游戏多。可以说是PlayStation情怀的完美护航者了。
