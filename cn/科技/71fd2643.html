关于npm install升级失败的问题

2020-11-14
| 科技

由于前一段时间，我们对客户的ecshop4.0手机端进行二次开发使用到了vue打包命令，当初使用升级命令“npm install”还可以对npm进行升级了

但今天在开始使用这个命令的时候，发现已经不能使用了，每次升级都提示“Cannot download "https://github.com/sass/node-sass/releases/download/v3.13.1/win32-x64-57_binding.node":”。网上打开对应的链接，资源也不存在了。所以网上有的教程说，在浏览器中输入对应的资源下载链接，把资源下载下来，然后放到对应的目录下，修改资源路径的方法已经不可能使用了。

我们是使用下面这种方法进行升级的：

使用淘宝镜像

npm install -g cnpm --registry=https://registry.npm.taobao.org cnpm install node-sass（我是使用这种方法升级成功的。）

OR

npm set sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

第二种方法至于能不能用，还需要你们进行测试看看了

在此记录一下解决方法，以方便遇到同样问题的小伙伴们解决问题
