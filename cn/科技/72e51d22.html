Win10 Win7，转换磁盘模式，实用教程

2020-11-14
| 科技

经常遇到硬盘模式不对，无法安装的事情，网上五花八门的太多，我今天来讲一个简单，易懂的

当然针对于装机的小伙伴，亲测，苹果，或者特殊电脑都试用

比如下图错误

这个时候 按住Shift +f10 可以调试出DOS界面

命令我大概讲下

打开Dos界面后

输入 diskpart 磁盘

List disk 所有盘

Select disk 0 (根据自己硬盘选择）

Clean 清除

convert mbr (gpt) 自由选择 win7 win10模式

好了以后，格式刚才的分区，就可以安装了
