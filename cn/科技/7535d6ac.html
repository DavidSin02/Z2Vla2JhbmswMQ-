超实用快照功能，帮你快速打造完美照片

2020-11-14
| 科技


爱上修图，从泼辣开始。泼辣修图每日教程， 助你从小白成为大师。

泼辣修图5.4版之后添加了许多有趣的新功能，包括「倒影」、「像素化」、「快照」等，为了让大家对这些功能有快速了解，泼辣修图编写了系列教程，本篇为大家介绍「快照」功能。

熟悉Photoshop的朋友对「快照」这个词并不陌生，不过与PS中建立图像处理记忆点的功能不同，在泼辣修图中，你可以把快照理解为「合并」功能，即将当前的所有操作，尤其是涉及图层、局部调整等添加的各类元素，完全合并为一张图片，就像你全新打开一张图片一样。

这样做的好处是，允许你进一步作为一个整体，调整图像效果，这为图层创意、二次修改、和添加多重滤镜提供了极大的便利！因为之前为了实现类似效果，你必须将图片保存副本，然后再重新打开。

泼辣修图中的「快照」工具，可以非常方便的在作品中叠加效果，今天大家就来跟泼辣熊学学这个功能如何使用吧！

//

工具档案

//

泼辣修图是一款全平台的专业摄影后期软件，支持手机和电脑多平台系统，曾获得苹果AppStore年度最佳应用，还是微软Windows 10中文系统的预装软件。本教程非特指均为手机版软件截图演示。

• 工具位置：按钮：任意工具→历史记录→快照；

• 主要作用：将当前图像效果合并，课直接继续调整；

• 应用场景：拓展参数、叠加滤镜效果；

• 出现版本：泼辣修图V5.4版之后新增功能；

• 是否收费：免费工具，非会员可自由使用；

打开泼辣修图，点击按钮：「任意调整」→「历史记录」→「快照」（如上图所示），即可找到快照工具的位置。

//

适用场景

//

一、扩展参数范围

在泼辣修图中，所有参数的调节程度用数字表示，一般最大值为100。如果此时调整到了极限还没有达到满意效果，想继续调整为150、200甚至更高时怎么办？

这时我们就可以利用快照功能，就像打开一个新的副本文件一样，再次调整这个我们未调整好的参数啦！

二、叠加滤镜效果

在加载滤镜时，如果加载的滤镜并没有达到我们想要的效果，我们可以通过快照功能多次叠加滤镜以增强效果。

原图

添加一次滤镜

快照后二次添加滤镜

三、添加图层后进行全局调整

在泼辣的图层功能中，当我们添加图层后，大家会发现这时我们对照片调整，我们所添加的图层并不会产生变化，这时我们就可以通过快照功能对添加好图层照片进行全局调整了！

//

使用教程

//

这里我们使用泼辣修图中的「HSL」来给大家演示一下快照的神奇之处吧，打开泼辣修图，打开我们要调整的照片！

选择全局调整-HSL-对画面种不同的的颜色进行统一，紫色、红色、橙色、黄色、都向洋红色靠拢，绿色、青色、蓝色都向绿色靠拢，这样画面就只剩下大面积的两个色相！

然后我们找到历史记录-选择快照功能-对画面进行合并调整的操作！

然后我们再次打开基础调整，找到HSL对画面的色彩再次进行调整，让绿色偏青色，洋红偏橙色，其余颜色向这两个颜色靠拢！

这样我们就可以得到非常和谐统一的青色与橙色了！

//

注意事项

//

使用快照工具后你将无法对图层等内容进行单独调整了，所以在使用之前，请确保你所有的调整不会再针对局部元素，当然你可以通过历史工具无限撤消步骤。快照工具是高手进阶的绝佳兵器，希望你能完全掌握。
