前端必备：CSS所有选择器

2020-10-29
| 科技

选择器是最长用到的属性，一般都会用到也算比较熟。但是有都布局或样式需求比较特殊，可能需要JS计算才能完成，但是有好多可以用选择器就可解决。本次整理出来这么多选择器也是一次踩坑了，为了避免再次掉进同一个坑就把所有的选择器整理出来了；有需要的可以保存留着以解燃眉之急，或推荐给刚入行或基础薄弱点的朋友，赠人玫瑰手留余香；

属性选择器

1.属性选择器 [属性 ~= value]

例：属性中包含独立的单词为value

div [class ~= haha]{

width: 100px;

height: 100px;

background-color: cadetblue;

}

<div id="div1">

<p class="p1 haha"></p>//正确

<p class="p1"></p>

</div>

例：属性中做字符串拆分，只要能拆分出来value这个字符串就行；

color: red;

}

3. 属性选择器 [属性 |= value]

例：属性中必须是完整且唯一的单词，或者是 - 分割开

div [class |= ggg]{

font-size:20px;

color: blue;

}

<div id="div3">

<div class="rrr-ggg">1</div>//错误

<div class="rrr ggg">2</div>//错误

<div class="ggg">3</div>//正确

<div class="ggg-rrr">4</div>//正确

</div>

4. 属性选择器 [属性 ^= value]

例：属性中前几个字符是value就行

div [class ^= en]{

width: 100px;

height: 40px;

background-color: coral;

margin-top:20px;

}

<div id="div4">

<div class="en"></div>//正确

<div class="enttt"></div>//正确

<div class="ensss"></div>//正确

</div>

e.g:[lang^=en] <p lang="ennn">

5. 属性选择器 [属性 $= value]

例：属性中后几位字符是value就行

div [class $= yuan]{

width:200px;

height: 50px;

margin-top:10px;

background-color: cadetblue;

}

<div id="5">

<div class="yuanddd"></div>

<div class="ddddyuan"></div>//正确

<div class="jljlj-yuan"></div>//正确

</div>

eg:[src$=".pdf"] <img src="test.pdf">

21. ::before和::after

这两个主要用来给元素的前面或后面插入内容，这两个常和”content”配合使用，使用的场景最多的就是清除浮动。

伪元素选择器

1、伪类 和 伪元素的区别

伪类：匹配的是元素(不同状态或结构的)

伪元素：匹配的是元素中的一部分内容(首字符，首行文本)

2、语法

1、:first-letter 或 ::first-letter

匹配 某元素的 首字符

2、:first-line 或 ::first-line

匹配 某元素的 首行文本

3、::selection

匹配 被用户选取的 部分

3、: VS ::

在 CSS3 之前，伪元素 使用的是 :

:first-letter

:hover

在 CSS3 之后，伪元素 一律使用 ::

::first-letter

:focus
