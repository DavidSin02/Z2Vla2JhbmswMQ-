这里把第一期贴出来给大家方便一起阅读。「音乐杂谈41」混音选择困难，Waves压缩器全介绍与贴心使用心得

很多人都有一个疑问，其实，为啥会有这么多的压缩器还有均衡器？这个问题有点笼统对吧。我能理解他们想问的意思是，为啥同样都是压缩器，为啥有这么多不同的，还有均衡器也是，其实原理不都是差不多嘛？

黑人？号 均衡器？？？哪个好？？？

压缩器来来去去，主要也就是 阈值，压缩比，attack，release，makeup gain，甚至有的没有这么多，就像LA-2A，只有一个 input和一个output。那为啥有这么多不一样的呢？如果你看过上一篇章，就应该了解，不同的压缩器其实会针对不同的情况来应用的，或者说，同一个情况，用不同的压缩器，也会得到有一些些不同的结果。当然，完全相同参数的情况下，其实压缩的结果是差不多的，不过会有很多细节的差别。例如 插件的最后染色不同，纯算法的可能会更干净，等等。

那么均衡器的话，也是一样，但是均衡器的话，主要是有图示均衡器和参量均衡器的区别，另外，均衡器本身也有analog染色的问题，以及操作性的问题。所以，即便是均衡器，一样也是根据自己的需要，来选择适合的均衡器使用。

那么本篇正式开始了

这一次，我也同样挑出18个均衡器来讲解，其中一个是压缩器。为啥有个内奸？呆会讲到了我们再说。那么均衡器的原理什么的估计也不用多啰嗦了吧，这个玩意儿也叫等化器，其实主要就是针对声音的不同频率的增益进行调整。

在混音中，均衡器 （EQ：equalizer）的应用，我个人认为：主要是有三个。

第一个是，修复性均衡，就是频率成分太多的，影响声音的正常状态的，需要调整。这个一般以衰减为主，就是把多余的弄小，也包括高低切滤波，去掉不要的成分。

第二个是，调节性均衡，就是已经听起来差不多了，但是放到完整音乐中又觉得不足的，那这个就需要提升或者衰减来调整，以便适应音乐，以及准确定位该声音。

第三个是，艺术处理均衡，这个就是基于上两个都做好以后，自己主观认为这个声音的哪些频率还要做些处理，让他听起来更接近自己主观喜欢的感觉。通常也是提升衰减都可以。

那么我们接下来看第一个。

第一个，Waves的元老 Q10 equalizer

Waves Audio由Gilad Keren和Meir Sha'ashua于1992年创立。同年晚些时候，Waves发布了第一款产品Q10 Paragraphic Equalizer。Q10是音频行业第一款商用音频插件。在25周年的时候，给重新换了一个界面，相对更加友好。

元老级 Q10 的界面

Q10可以说是陪伴了我们这一代的混音师整整20年了。我第一次用waves就是特别喜欢Q10和C1(压缩器)，功能不用多说了吧，曲线方式可选，±Gain，两头无极扫频，频点随便跑，超大的Q值可调范围。简直就是EQ中的典范，最多可调节点10个，在过去可以说只有多余，不会不够用，即便现在也经常有这种感觉。Q10其实还带有好几个插件，包括Q2 Q4 Q6 Q8。其实就是通道数量，最多是Q10。Q4，那就是4个节点。

25周年纪念版界面

全新25周年纪念版还增添了这几个功能：比例Q值滤波器、可选频段控制、双倍精度处理。可以说直接如虎添翼。

Q10是 mix非常全功能的EQ了。我现在主要是用来作为修复性EQ使用，非常精准。

第二个，GEQ，图示均衡器的样式典范

为啥说是样式典范呢？因为30通道的图示均衡器真的不管是软硬件，都是那么经典，需要的频段全部都分好了。曲线你怎么画就怎么得到结果。这个是图示均衡器的特点，即便没有示波器，看推子都大概知道均衡线的样子。这个均衡有 classic 和 modern 两个模式，下图都有。

两个模式的区别是，classic用的是比例型Q值滤波器，即在均衡量增大时滤波器的带宽将会变窄，modern的是频率无交叠的平顶滤波器，避免了频段间重叠带来的相互影响，从而能够提供平直的阶梯式的频率响应。

除逐个设置各频段的均衡量外，你也可直接在触摸屏上或用鼠标画出所需的均衡曲线。此外，GEQ内置的实时分析仪也可以方便地对两个声源进行差异比较。而GEQ实际上主要是用在现场比较适合，尤其是使用waves的服务器系统，可以触屏直接调节。在现场调节简直就是超级方便。不过呢，EQ生来平等，即便是在混音中，也是一样出色。

第三个，API 系列的三兄弟

双胞胎之一的API 550A均衡器基于60年代后期的API传奇产品进行建模，在相互交叉重叠的三个区域中各提供了15个频点，以5步递进式进行增益调节。它的高频和低频区域可以根据使用需要选择钟型或者搁架式均衡，同时带通滤波器也可以独立应用到参数设定当中。

API 550A 和 API 550B

双胞胎之二的API 550B以四段交叉频点而闻名，也是解决问题和润色声音的利器。它的每段均衡中都有7个可选频点，最大能跨越5个倍频程。550B的“比例Q值”功能能够自动在较小设定时以更宽的Q值滤波，或在大范围提升或衰减时以更窄的Q值进行工作。它甚至还能够让你撤销先前的处理，影响甚至逆转对声音的已有改变。

API-560

双胞胎另外还有一个兄弟 API-560 基于1967年的经典硬件建模，跟550B一样有“比例Q值”功能，作为图示EQ，虽然没有GEQ那么多频段，但是作为快速处理，绝对足够。

我在处理要快速解决，并且要求并不是非常高的商业mix，就非常喜欢API的这种EQ，因为数值是定死的，不是无极拧，那么其实更加容易得到想要的差不多的结果，不至于会一直纠结是100还是101这样的问题。

第四个，Waves EQ中的心外科手术医生---F6

F6可以说只有一点比不上Q10的就是少了4个通道。但是却在功能上碾压Q10。其操作以及调节的精准程度，如果说Q10是普通的外科手术，那F6做的就是心外科手术或者是脑科手术。它拥有6层参数完全可调的浮动滤波处理频带，每层频带内都带有先进的均衡器和压缩/扩展器控制，同时还包括中央/侧向（Mid-Side）处理选项等等。可以说对于声音的均衡调节非常非常细腻。

带上了压缩器和扩展器，意味着，某些频段的声音，除了增减增益，还可以让其稳定或者根据阈值以及比值自动调节。还有MS制式的处理，如果一般处理算是平面的，那么MS制式可以说是在三维空间中处理EQ了。

不用多说，我如果要处理细致的声音，或者要处理很棘手的某些频段稳定却又在某些频段动态过分的声音，一定选F6就没错了。

第五个，这个是奸细，他不是EQ，C4和C6多段压缩器

这个其实在压缩器篇章已经详细讲解过了。就不多讲了。其实一定程度上来说，也可以作为一个4段EQ来用，而且还跟F6一样，带有特定频段压缩处理。可以说，是F6的一个阉割版吧。。反正对于频段如果动态不对，那么可以通过C4C6来处理。就像我老师说道的，哪里不对，就调哪里，压住，就乖乖了。

第六个，文艺复兴系列的 EQ

他跟之前压缩讲到的文艺复兴的压缩和人声压缩是一套的，这个系列主要的特点是声音有老旧的味道，非常温暖。作为算法效果器，模拟出物理建模的感觉，也算是相当牛逼的了。调节方面其实跟Q10非常相似，有2段，4段和6段选择。

这个EQ可以说是中规中矩了。不过有能带来温暖染色，所以做老旧风格挺适合的。关键是跟其他文艺复兴系列一样，简便。

第七个，调音台经典之作 VEQ3 和 VEQ4

VEQ4是基于广受欢迎的1081调音台模块建模的经典四频带均衡器，有着1081调音台特别染色的感觉，我觉得对比文艺复兴那种温暖，这个才更加像机器的染色。调节方面扫频有些限制，类似于API550的操作。也好，非常便利，毕竟选出来的频点都是非常接近我们需要的，增益是无极拧。也有VU去看输出量。

VEQ4

而V-EQ3，则是结合了两款最让世人渴求的调音台模块的所有最优特质——即1066和1073调音台，这款三段均衡器，有着经典的可调截至频率的高通滤波器，对混音和母带都是非常理想。使用V-EQ3能以最轻松的方式获得1960和1970年代那饱满经典的声音，有人说绝杀文艺复兴的感觉哈。不过我觉得这个不能这么比较，看音乐的风格需要吧。

VEQ3

总之，VEQ3和VEQ4，都是经典复刻，对于一些经典风格重现是很好的选择。我也比较喜欢用于bass以及钢琴的调节，或者是地鼓和军鼓的调节。

第八个，两个省吃俭用的，数字均衡 EMO

他们一个是均衡emo-q4，一个是滤波器emo-f2，主要是非常的省资源，非常不占用CPU。作为现场或者直播，或者录屏，都是很不错的选择，延迟超低。作为均衡的调节，其实无非就是频率，增益，Q值，曲线。那么EMO还能给你分开左右处理，可以区分开左右声道的均衡。那还是相当厉害的了。

EMO-Q4

尤其是作为现场使用，非常的直观得到我们要的效果，也很容易调节，尤其是使用触屏系统的话，操作非常人性。像我同事说的，该有的都有了。

EMO-F2

第九个 Kramer HIS 英伦风味

这个跟Kramer PIE 是一套的，也是非常英伦风的经典之作，很好地再现了Helios调音台的伦敦风味。并且带有话放失真模拟，英国味的高频临场，肥厚的低频增益。有点老滚石乐队的味道。我的话主要是用这个来做一些比较俏皮风的或者英伦或者rnb类的作品用的多，当然其实也可以用的，看心情吧。

第十个，SSL调音台中的现代经典 SSL-G EQ

建模自SSL G系列硬件EQ292简单易用。没有任何可视化的东西，纯粹凭借耳朵听到的调。有一个低切，还有一个analog的开关，想要保留ssl的染色，就可以打开。

这个EQ我一般是用于艺术处理了。又要染色，又要调下EQ，就可以考虑，尤其是工程里面其他地方也都用上SSL的东西的话。毕竟是G，那基本上如果工程中决定了用G的染色，那么就可以考虑用这个。或者想单独对某个东西进行染色也行，反正艺术处理嘛。

第十一个，基于 NEVE1073的建模 Scheps-73

基于经典的1073调音台的均衡器及话筒前置放大器模块建模而成，具有相互交叉重叠的频段，忠于1073调音台的原始参数并精确模拟它的运行状态：固定的12kHz高频段，可选并带有衰减和增益控制的低、中频段以及一个每倍频程18dB衰减的高通滤波器。另外，在原始1073设计图中发现的只在稀少的1078通道条上可操作的10kHz频点，也有史以来第一次被建模并融合到插件之中。

一直以来，我都觉得NEVE的那种温暖是一种失真感，用于摇滚或者比较强烈的风格特别搭配。尤其是1073这种经典台子简直就是为摇滚风格而生的感觉。所以失真吉他和bass，绝对可以考虑用他，声音会变得更加饱满以及有力量感。

第十二个 Abbey Road TG Mastering Chain

这个机子其实是一台母带处理器，以EMI TG12410母带控制台为原型建模——从1970年代初至今，它在Abbey Road录音棚所有的母带制作中都不可或缺。有着经典的 Abbey Road的染色，以及模块化处理，其中包含有均衡器。

不多介绍，这个插件主要是用于bus处理，以及母带处理。参数调节都是不用特别大动作就可以得到满意的效果。

第十三个，PuigTec EQs

这个是包含两个均衡器，有侧重中频的PuigTec MEQ-5和宽频段的PuigTec EQP-1A，

当然，一看就知道，是基于硬件物理建模的插件，这套插件强力点是，可以在同一个频点进行增益和衰减，从而产生一个共振搁架。如果你觉得声音某个频点想提升，但是提了又觉得满，不提又不够，而且还不是度的问题，那么同频点再衰减形成的共振搁架就能让你得到想要的效果。很好使的东西。

第十四个，Manny Marroquin EQ

MMEQ是一个套装里面的东西，把 Manny Marroquin的个人对于不同设备使用习惯做出了集合，每一个频率截止点的设置都源自于不同的设备，它们包括Quad 8、Motown、Neve 1073、API 550B、Avalon 2055以及SSL 9080 XL K。简直就是插件火锅，但是对于要处理一些快速工程，想立竿见影，是好东西。

第十五个，RS56 Passive EQ

基于传奇的RS56 Universal Tone Control被动式均衡器建模的插件，它最早在1950年代初期被著名的Abbey Road录音棚用于唱盘刻制，即现在我们所知的“母带工程”的一部分。不久之后，Abby Road年轻的工程师们开始将它用于录音室录音，因为它具有非常强大的声音操控能力，远超当时调音台上的仅有低频和高频控制器的两段式均衡器。这一特性也让RS56获得了“频响曲线塑形器”的绰号。

强大之处在于左右分开的均衡处理是当时的超先进技术，并且能够非常容易得到想要的声音，并且同样带有 Abby Road的染色。

第十六个，H-EQ Hybrid Equalizer

H-EQ可以说是 H系列中很厉害的家伙，其实我觉得他有点类似于F6的物理化版本。它是物理建模一个英式均衡器，却带有美式均衡器的元素。吸血鬼也是混血最厉害，这个可以说真的很猛。染色方面跟整个H系列一样，有多种选择，真方便的。

功能上，跟F6有的一拼的就是同样有MS制式调节，可以得到更加立体的声音。

第十七个，一个万能工具刀一般的EQ

AudioTrack是一个通道条，其实有点类似于SSL通道条，但是这个是超级省资源的货，跟emo类似吧。这个是算法通道条，包含有压缩和均衡和门限。主要是处理一些快速的内容特别好用。

第十八个，母带上的均衡器 Linear Phase EQ

Linear Phase EQ是一款具有线性相位的均衡器，在保证原始音乐的平衡同时，以绝对零相位偏移对谐波频谱提供精准的控制。母带均衡器插件五个用户自定义频段，具有线性相位分频器低频段组件与全频段组件最适用于原声乐器、立体声素材、高瞬态音频轨道以及母带制作九种线性相位滤波器类型。不过嘛，其实要做简单母带，有更多好的选择。不过作为刚刚打开工程，临时在总线做一个简单的调节，还是很方面用的。

总结一下

那么上面基本上已经把waves的独立使用的均衡器插件都简单的概括了一遍，那么大家应该都大概了解每个均衡器都有自己的特点了吧。这个毕竟也只是一个概括讲解，实际上真要知道是不是适合某个音频处理，还是得自己慢慢多试试，这个是基本功呢，就跟学乐器练琴一样。需要时间，也需要精力去做。包括对于不同厂牌的均衡器，也一样是通过多使用来总结感觉。

其实，最终要用哪个均衡器去处理，都是基于自己要有明确的目的，就是，当你得到一个声音，你想要把这个声音做到什么样的变化，出来什么样的结果，要心里有数。然后根据这个结果，来推导自己需要哪个均衡器来处理。那么这样，你就得知道，不同的均衡器，能做到什么样的效果，根据这个自己试验过的经验，来判断哪个均衡器最接近自己要的效果，然后插上去调整。未必一次就中，但是起码是要有方向去做，而不是胡乱去试。

特别似曾相识的总结对吧？对的，其实均衡器和压缩器最终要怎么用，都是一模一样的，需要自己调试过，记住了效果以后，凭借经验去选择使用。多锻炼耳朵吧。
