一、指针的使用

C语言中提供了一个“间接访问运算符”（也称间接地址运算符）：“*”，如果指针变量中保存了确切的地址，就可以使用“*”来引用该地址的存储单元。

1、通过指针来引用存储单元

（1）指针的自加自减

Int *p, i=100, j;

p=&i;

j=*p;//等价于：j=i=100;

j=*&i

j=*p+1;

几种常见的指针操作，见图20-1。

图20-1，指针的自加自减运算

在重申一下，运算的时候，先看运算符的优先级，顺序是高优先级到低优先级，优先级相同的情况下，看结合性顺序。

我们通过程序201.C验证一下我们的分析是否正确！

/******************************目标： 指针的自加自减运算，核心知识点是优先级和结合性*******************************/#include <stdio.h>int main(){ int m=100,n=200,k=300,j=400; int sz[3]={1,2,13}; int *p; p=&m;

printf("*p=%d\np=%d\n",*p,p);

 *p=*p+1; printf("执行*p=*p+1后:\n*p=%d\np=%d\n\n",*p,p);//实际上m的值变成了101，但是p保存的地址还是m的地址 p=&n; printf("*p=%d\np=%d\n",*p,p); ++*p; printf("执行++*p后:\n*p=%d\np=%d\n\n",*p,p);//实际上m的值变成了101，但是p保存的地址还是m的地址 p=&k; printf("*p=%d\np=%d\n",*p,p); (*p)++; printf("执行(*p)++后:\n*p=%d\np=%d\n\n",*p,p);//实际上m的值变成了101，但是p保存的地址还是m的地址 p=sz; printf("*p=%d\np=%d\n",*p,p); *p++; printf("执行*p++后:\n*p=%d\np=%d\n\n",*p,p);//实际上m的值变成了101，但是p保存的地址还是m的地址 getchar();}

 #include <stdio.h>int main(){  int **p,*s, k=20; printf("k=%d\n",k); s=&k; printf("执行s=&k;:\n*s=%d\ns=%d\n",*s,s); p=&s; printf("执行p=&s;:\n*p=%d\np=%d\n",*p,p); **p=30;// printf("执行**p=30;:\n**p=%d\np=%d\nK=%d\n&k=%d\n",**p,p,k,&k); getchar();}

通过以上程序，配合图10-2,，你肯定彻底弄懂了**p的过程，其实类似于你外地有一个朋友，你想给他寄点东西，但是你并不知道他的地址，而你同学却能找到他，因此只能先把东西寄给你的同学，然后通过同学转交给这位朋友，重点还是优先级、结合性、地址这基本的东西。

2、指针的移动

指针的移动就是对指针变量加上或减去一个整数，或者通过赋值，使指针指向其他单元，因此只有指针指向一串连续的存储单元时，指针的移动才有意义。

Int a[5] , *p,*q;

P=&a[0];

P=a;

q=p+2;

p++

q++
