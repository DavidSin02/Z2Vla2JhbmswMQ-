LG发布G8X和全新改进的Dual Screen，比折叠屏手机还香？

2020-10-29
| 科技

图片来源：Kyle Wiggers / VentureBeat

IFA 2019来了，这只能意味着一件事：连续一周电子产品大量涌现。 LG，就像它经常做的那样，在LG G8X(LG G8的继任者)基础上推出了一款不断改进的手机。与之同时发布的配件反而更有趣：Dual Screen产品的升级版，在几个方面都优于原产品。

G8X和改进后的Dual Screen定价尚未公布，但我们将继续关注更多细节。 LG预计两款产品将在第四季度上市。

LG G8X ThinQ

G8X比G8稍高(前者159.3毫米，后者151.9毫米)，更重(前者192克，后者167克)，但宽度(71.8毫米)和厚度(8.4毫米)相同。而且，它同样耐用。与G8一样，它的防刮机身也通过了IP68认证，可以在5英尺深的水下防水30分钟，而且它的防护等级为MIL-STD-810G，这意味着它已通过从肩高处掉落至粗糙表面的跌落测试。

不幸的是，与G8和LG V50一样，后盖有一种廉价的空心感，这是LG的Boombox扬声器技术的特点，它使用后盖和扬声器之间的内部空间作为共振室。

在上述扬声器的右侧是睡眠/唤醒按钮以及SIM卡插槽，左侧是分叉音量按钮和Google Assistant按键。

将G8X翻个身，让我们看看屏幕，顶部有一个凹口。与LG以前的G系列和V系列手机一样，它也被LG所谓的新第二屏幕(New Second Screen)选择性地隐藏了起来。新第二屏幕会改变通知阴影的颜色，以匹配黑色边框。在这里，你还可以找到Crystal Sound，它是一种微小的致动器，通过振动手机屏幕的表面以产生声音，与底部发射的Boombox扬声器相结合。LG声称他们能够提供具有“更高清晰度”和更好平衡(每个扬声器1.2W)的音频，而不是竞争对手智能手机上的立体声解决方案。

值得称赞的是，LG采用32位Hi-Fi Quad数字音频转换器(DAC)，可以轻松放大放大器和算法，以提高视频、歌曲和播客的音质。 G8X还支持DTS：X 3D环绕立体声技术，它模拟了一个7.1通道的系统，无论有没有耳机;声音预设和左/右平衡;通过主质量认证(MQA)从受支持的应用程序(如Tidal和Deezer)获得高质量、有损的音频压缩;和高通的AptX和AptX HD技术，通过蓝牙实现低延迟流媒体。

显示屏

当你不用G8X听歌时，可能会盯着它的6.4英寸，19.5：9宽高比屏幕。它比G8中的OLED面板大0.3英寸，分辨率为2,340 x 1,080像素，最大亮度约为600尼特。

这并未打破记录，而且从某些方面来说，它比G7的MLCD+ 3120 x 1440屏幕有所下降。G7的MLCD+ 3120 x 1440屏幕在最高亮度时可以达到700个尼特，或者每次三分钟最多可以达到1000个尼特。但从好的方面来说，G8X的显示器有一个AI引擎，可以根据屏幕上显示的内容自动切换颜色配置文件。或者，它提供了几种模式的选择 - Eco(节能模式)，电影，体育，游戏，照片，Web和专家 - 以及控制滑块(设置>显示>屏幕颜色)，调整色温，RGB级别和gamma。

G8X支持高动态范围(HDR)视频，电视节目和电影，与非HDR同类产品相比，它具有更高的亮度，更宽的色域和更好的对比度。这里的特色是HDR10，它涵盖100%的DCI-P3和Rec 2020色彩空间 - 数字电影放映机和大多数4K超高清电视、电脑显示器的标准 - 总共10.7亿种颜色(每种原色的1,024种色调) )。

值得注意的是LG决定抛弃G8X的飞行时间传感器，该传感器可用于生物识别身份验证。相反，该公司选择了指纹传感器嵌入显示器下方。

相机

与其前代产品一样，G8X配备了两个后置摄像头，当像Galaxy Note10这样的三摄像头手机成为常态时，这几乎是一种不合时宜的选择。无论好坏，LG似乎倾向于为其V系列预留三摄像头。

G8X的1300万像素广角传感器(具有f / 2.4光圈，1微米传感器尺寸和136度视野)补充了1200万像素传感器(f / 1.4光圈，1.4微米传感器尺寸，以及78度视野)。在G8X的正面，有一个32万像素的摄像头(f / 1.9光圈，0.8微米传感器尺寸，79度视野)，是G8的1200万像素自拍相机的升级版。

在视频方面，G8X能以60帧/秒(或高达1,080p，每秒240帧)的速度记录高达4K分辨率，通过电子图像稳定功能实现稳定记录。

LG一如既往地在G8X的相机应用程序中加入了许多“智能”功能，比如人工智能合成(AI Composition)和夜景模式(Night View)。人工智能合成最多能识别三个人，并相应地调整焦距。还有聚光灯(Spotlight)，可以从不同的角度添加灯光;自动拍照(Auto Shot)，利用人脸检测自动自拍;视频人像(Video Portrait)，使用深度和距离分析“双FOV”技术微调散景(模糊背景的效果，同时保持焦点在前景)。

G8X的Reflector Mode旨在通过曝光灯光颜色控制，确保即使在黑暗环境中也能拍摄清晰锐利的人像，而人工智能动作拍摄(AI Action Shot)则以1/480秒的快门速度识别目标，并对运动进行补偿，以保持目标的聚焦。至于4K延时，它允许你创建4K延时视频。G8X的Steady Cam提供了增强的图像稳定性，LG声称优于市场上大部分电子图像稳定解决方案。

此外，还有一种模式可以自动调整麦克风以进行ASMR录音。

性能

支持G8X的工作负载与三星Galaxy S10系列中的系统级芯片(SoC)相同：高通的骁龙855。这是G7的骁龙845的重大升级，LG的新旗舰产品充分利用了这一点它。

这款7纳米8核芯片整体速度提高了45%，这要归功于64位Arm Cortex基于高通公司内部的Kryo 485处理器的设计。(LG将SoC与6GB RAM和128GB存储配对，后者可通过microSD扩展至2TB。)四个核心处理繁重的工作——一个主核的频率为2.84GHz，三个效率核的频率为2.42GHz——而四个效率核的频率为1.8GHz，可以兼顾要求较低的任务。与此同时，骁龙 855的GPU Adreno 640图形性能比骁龙845的Adreno 630快20%，支持Vulkan 1.1和自定义算法等API，这些算法旨在将帧数减少90%以上。

同样，电池续航时间不会令人失望。 G8X的4,000mAh大电池 - 比G8的3,500mAh还高 -在混合使用的情况下，可以使用一整天。当电池电量为零时，由于高通的Quick Charge 3.0，它能在15分钟内提供高达50%的电量，因此G8X可以快速充电，或者你可以通过Qi进行无线充电。

软件

如果你接触过G8或V50，那么你很快就会意识到G8X的软件几乎是复制的。

LG在安卓系统上的外观已经不像以前那么霸道了，而且它的外观设计更加简洁，图标也更新了。例如，电话拨号器中的选项卡(拨号、通话记录和联系人)已经迁移到显示屏的底部，这样单手就可以更容易地访问它们，并且子标题也被加粗，以便更多地关注键菜单。

Boombox Show，显示符合播放音乐节奏的闪光灯和屏幕视觉效果，有点浮夸。 LG的游戏工具也是如此，它覆盖了一些快捷方式，比如截屏、缩放应用程序、游戏期间提醒禁用、隐藏主页按钮、调整分辨率和帧率。

在G8X相机应用程序上，谷歌的物体识别Google Lens总是在菜单顶部，它可以识别一长串不断增长的东西。除了电话号码，日期，地址，家具，名人，服装，书籍，电影，音乐专辑，视频游戏，地标，兴趣​​点和著名建筑外，Lens还可以扫描条形码和二维码;将事件添加到日历;从名片导入联系信息;从Wi-Fi标签中提取网络名称和密码;并识别葡萄酒和咖啡等饮料。最近的更新使它能够做得更多，例如从打印输出，名片和小册子中复制和粘贴文本，并将衣服和“家居装饰品”与来自网络的结果相匹配。

对于不太直观的查询，可以使用Google Assistant。与V50和G8一样，G8X也有一个专门的按键，可以从任何地方启动语音助手，包括锁屏(双击可以启动Google Lens)。LG与谷歌紧密合作，为ThinQ设备开发了20多个定制命令，包括V50，比如“OK Google，广角拍照”和“OK Google，打开Cine Video相机”。

Dual Screen

LG Dual Screen在2019年的移动世界大会上被宣传为LG V50的字面延伸。翻盖式外壳增加了一个显示器，其外观或多或少类似于外部显示器，在没有内置解决方案的不可移动的大块空间的情况下，赋予了额外的屏幕空间。它在实践中运作良好，但令人沮丧地受到限制 – 这是在最近的测评中提到的几个缺点之一。幸运的是，LG似乎把这些批评放在心上。

全新改进的Dual Screen - 尺寸为165.96 x 84.63 x 14.99毫米，重134克，配备全高清19.5：9 OLED(2,340 x 1,080像素，每英寸403像素)显示屏 - 可以360度折叠，覆盖G8X的后盖。这比前一代产品更好，其铰链不能倾斜超过180度。或许更好的是，Dual Screen的G8X兼容版本在前面板顶部中嵌入了1.2英寸单色屏幕，默认情况下显示时间、日期、电池电量、通知和其他相关信息。

前一代Dual Screen依靠POGO引脚和无线技术的组合进行数据传输和供电，而这款最新型号通过USB-C连接到主机智能手机。这赋予了更大带宽的优势，从而为真正的扩展屏幕功能奠定了基础。这是一个实验性的功能，但新的Dual Screen可以扩展谷歌浏览器等应用程序，无论是横向还是纵向，这种体验让人想起中兴Axon M和Kyocera Echo。

可能是预见到了批评，LG表示工作很谨慎，以确保双屏面板与G8X显示屏的色温和色调相匹配。这是令人鼓舞的，可以注意到的是，在测评中，双屏幕偏向较冷一端的频谱，相比之下，LG V50屏幕有着温暖的轮廓。

Dual Screen的其他漂亮功能包括音量调节工具，可让你在一个屏幕上让应用程序静音，同时增加相反屏幕上应用程序的音量，以及相机取景器功能，旨在被拍摄之前更容易预览照片。现在只需将三指滑动手势，就可以从一个显示屏交换到另一个显示屏，G8X键盘上的专用按钮即可对双屏幕上任何内容的屏幕截图，并将其转换为优化后的图片，用于发送信息。

说到键盘，利用LG的Dual Screen开发工具的应用程序，可以更容易地将其转换为横向屏幕的内容(例如，视频、电子邮件和文档)，使其使用起来不那么麻烦。应用程序现在也可以更好地利用Dual Screen的空间。例如，Game Launcher中的新游戏手柄创建工具可让你选择任何虚拟按钮、操纵杆和肩部按钮，并根据需要调整它们的大小。图库应用程序在左侧显示缩略图，在右侧显示相应的全屏照片。Whale是Android上的第三方网络浏览器，已针对G8X和双屏幕进行了优化。

和以前一样，专用的Dual Screen屏幕设置提供对粒度控件的访问，包括辅助面板的亮度滑块和壁纸选择器。 它也是指定在辅助屏幕打开时打开哪个应用程序(如果有)的地方。
