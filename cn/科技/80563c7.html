一天一个算法题

2020-10-29
| 科技

题目：

给你一个数N，拆成3部分，要求每一部分都不是3的整数。随便输出一个可能解即可。

分析：

题目很简单，要求都不被3整除，不被3整除的数有以下特性：各位数字上面加起来的和不被3整除。可是又要一下子构建3个，看起来这个题目非常困难。

解

事实上，只要输出一个可行解，我们不妨让3个数字为 1,1,N-2, 看看这个是否满足答案，也就是N-2是否满足不被3整除。

如果N-2满足被3整除，那么N-3一定不被3整除，所以结果可为 1,2,N-3.

代码

#include <iostream>using namespace std;int main(){ long n; cin >> n; cout << 1 << " "; if ((n - 2) % 3 == 0) { cout << 2 << " " << n - 3 << endl; }else { cout << 1 << " " << n - 2 << endl; }}
