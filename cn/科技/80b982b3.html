把 C 盘已安装的软件转移到其他盘

2020-11-14
| 科技

当电脑系统出现问题时，想重装系统，但是C盘里的软件重装系统后，自己又不知道从哪里下载或者不会安装等，如果把C盘里的软件直接剪切或拷贝显然是不行的，这么做的话，大多软件都会直接打不开。此时就需要“FreeMove”来帮你解决。

首先，打开百度，在搜索框里输入“FreeMove”点击搜索并下载。


下载完以后双击打开即可，这是一款免安装的小软件。


如果你需要移动的软件是PS，并且在C盘下，那么点击Browse，找到系统盘中的文件夹 Program Files，然后找到PS 的安装文件包，点击确定。在 To 当中，通过同样的方式找到其他盘的软件安装目录。比如，我们需要 PS 移动到 D 盘下 Program Files 文件夹当中。


之后，点击 Move 即可。根据软件大小不同，移动所需的时间不同。

移动完成之后，试试重新打开 PS ，确定是没有问题的。你可以在 D 盘下 Program Files 文件夹当中找到移动后的 PS 文件包。回到原来的 C 盘文件夹下，这里其实仍然有一个 PS 文件包的软链接。单击右键，属性，则会显示文件包目前所在的位置。

当然，你也可以在移动的同时，点击 FreeMove 左下方的隐藏复选框。隐藏掉原文件夹。
