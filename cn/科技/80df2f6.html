c++基础之「多态」

2020-10-29
| 科技

什么是多态?

同一操作作用于不同的对象，可以有不同的解释，产生不同的执行结果，这就是多态性。简单的说就是基类的引用指向子类的对象。

为什么要用多态呢?

1.封装可以隐藏实现细节，使得代码模块化

2.继承可以扩展已存在的代码模块(类)，代码重用

3.解决项目中紧耦合的问题，提高程序的可扩展性

多态的实现

1.重载

静态多态，就是重载函数，编译时已经生成

2.虚函数

简单地说，每一个含有虚函数(无论是其本身的，还是继承而来的)的类都至少有一个与之对应的虚函数表，其中存放着该类所有的虚函数对应的函数指针。

struct B

{

long b;

virtual void foo(){}

virtual void bar(){}

};

其内存布局

vptr --------> B::foo

b B::bar

struct D:public B

{

long d;

virtual void bar(){}

virtual void quz(){}

};

其内存布局

vptr ---------->B::foo

b D::bar

d D::quz

其中:B的虚函数表中存放着B::foo 和B::bar两个函数指针，D的虚函数表中存放的既有继承自B的虚函数B::foo,又有重写了基类虚函数B::bar的D::bar,还有新增的虚函数D::quz
