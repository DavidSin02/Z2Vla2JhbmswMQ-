业余无线电入门如何选购设备，你需要多大的功率

2020-11-14
| 科技

在业余无线电里，有两个极端，有的朋友热心于 QRO ，功率输出最高可达法定限制的最高值 1000W ，有的热衷于QRP，5W 的输出或者更少。大多数业余无线电爱好者的功率都介于 20~100W 之间，100W 功率是比较典型的收发机功率。

有的时候必须使用大功率才能保持稳定的通联，大功率在某些波段的远距离通信竞赛中更有竞争力。多数情况下，100W 的功率足以提供非常好的通联效果。有些业余无线电爱好者认为廉价的低功率短波设备非常有吸引力，特别是那些以套件形式出现的，既锻炼动手能力，又可以获得成就感。

虽然我一直说 QRP 很简单，值得尝试，确实值得尝试。但是，对于刚刚开始尝试短波的初学者，QRP 经常会带来严重的致命性打击。无论如何，在你尝试 QRP 之前，我都推荐你首先积累一些 100W 功率通联的经验，这样一来你就知道在短波频段上会有哪些现象。它们和 UV 段的情况会有很大的不同，短波上功率因素对信号的影响比 UV 段要弱一些，或者说提升功率对于通联效果的改善没有 UV 段那么明显。

当你试图在 UV 段呼叫几百公里远的电台时，功率会比较重要，当使用性能优良的天线进行本地通信时，典型的 5W 手持设备或 25W 的车载电台的功率完全够用。

如果你的兴趣点是 FM 语音、 APRS 或者与本地业余无线电爱好者进行数据无线电通信，那么一部 UV 双段的收发机就可以满足你的需求。FM 收发机有手持设备，也有车载移动设备。如果你有 12V 的电池或合适的电源，那么在家也可以使用车载电台作为固定台。

在家里使用的话，还可以选购性能更加强大，功能更丰富的固定式收发机。比如我目前使用的八重洲 FT-991 和 icom IC-7100 ，作为短波机它们应该还算便携，不过我主用 UV 段全模式。手台出于便携性的考虑，天线不可能做的很大，车载电台会好一些，家用设备天线就可以搞的很大，我目前在用 V 段 5 单元八木，U 段 2X10 单元八木配合云台进行水平，仰角旋转。

手持式收发机俗称手台，对讲机。手持式收发机的工作功率一般介于 0.05~5W，甚至更高。在此不建议使用很大的功率，满足使用即可，低功率也会节省电池的电量，延长使用时间。大多数手持设备都提供高/低功率开关。好的二手手持收发机的价格通常低于 1000 元，而全新的手台的起步价格通常为 1500。当然了，也有很多廉价设备可以选择，比如宝锋，欧讯，泉盛之类的，全新的设备价格在 100-1500 左右。如果你在汽车内部使用手持设备，你很肯定会对原装天线的性能非常失望，此时在车外加装一个移动式天线将会有好得多的性能。

UV 段车载电台的功率一般介于 5- 50W 之间，请注意法律法规允许最大为 25W ，使用天线一般为卡边或吸盘安装移动天线。在使用中继的情况下，任何大于 10W 的功率很可能都是没必要的。二手的车载电台价格通常在 500 以上，视具体情况而定。全新的车载电台起步价格大约为 1300 元，功能更多的车载电台价格高达四五千的也有。

固定电台收发机，基地台。使用电源或电池供电的固定式基站无线电台。它也可以断开连接，搬入汽车以便进行移动式通联。利用电池对设备进行供电还有一个优点，在停电的时候可以继续工作，我这里常备一个 13.8V 40Ah 磷酸铁锂电池。此类设备多数是全波段全模式设备，包含短波，6 米，2 米和 70 厘米波段。如果你只打算在家进行短波通联，这些可不是好的选择。很多朋友不喜欢这些大而全的设备，我却对它们情有独钟，我喜欢 UV 段的全模式操作。但是，国内的爱好者好像更喜欢短波多一些，对于 UV 段的操作只限于中继，其它的内容涉及的比较少。所以我的全模式电台，多数时候用于操作业余卫星。

VHF 段数据包通信无线电。最常见的就是 APRS 自动分组数据交换系统，这可以实现短信息发送，当然几乎所有的设备仅能发送字母和数字。你需要一部 2 米波段的电台进行此项操作，除此之外你还需要一台计算机或终端，还有一个终端节点控制器，也就是 TNC。有一些比较高端的手台内置了 TNC ，可以直接连接电脑使用，有的手台内置了 APRS 功能，在离开电脑的情况下可以直接进行 APRS 操作。

记住，高功率操作可能会产生射频安全隐患。在车载电台上，要把天线放置在远离车厢的位置。对于更高功率的手台来说，尽量使用手咪和天线延长线，以保证天线远离你的双眼和头部。永远都不要忽略这个问题。

短波设备

使用短波电台，以及架设一个短波天线，比 UV 段更具有挑战性。短波设备通常涉及多种模式和波段，设备价格也更加昂贵。短波频段的天线，比手台天线和车载天线大的多。和 UV 段不同，短波的天线更加多样化，各种不同的电台也更多。需要考虑的问题也更多，例如在业余无线电方面的预算有多少，架设天线的尺寸和位置有没有什么限制。多数业余无线电爱好者没有办法架设庞大的天线，他们使用室内天线或者隐藏的天线，工作起来困难重重。

在电台选择上，我不建议没有任何动手维修能力的朋友购买古董电台，虽然它们很便宜，一般花不到 1000 元，你不知道它经历过什么，不知道它何时会突然罢工。上面说过了，我的选择是全波段的设备，多数朋友认为这样的设备性能不好，认为设备需要专用。我不觉得全波段设备有什么不好的，各种电台都有它存在的价值和适用的人群。说实话我买了它们是玩 UV 段的，短波属于附送的功能，而且我短波很少使用超过 5 W 的功率，虽然它们两个都是 100 W 功率的电台。请注意，不是 FT8 模式，不是 CW 模式，我使用的是 SSB 语音模式。

我的以前天线仅有一个简易的单波段 DP ，去年使用 5W 功率在 20 米波段通联了国内 1234567 区的电台，还有一些日本电台。今年我换上了一根 4 波段端馈天线，经常使用 2.5W ，1W 功率和国内以及周边国家的 HAM 进行 SSB 语音通联，使用 5W 还通联了 10717 公里的美国亚利桑那电台。有朋友说我电磁环境好，好啥啊，各波段常年 S9+。短波内容少一些，我对短波没多大兴趣，我觉得 UV 段的挑战性更大。我喜欢玩不一样的东西，短波玩的人太多了。
