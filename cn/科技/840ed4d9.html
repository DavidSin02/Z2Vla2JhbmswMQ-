以下为各题代码：

1、

#include <stdio.h>#include <math.h>int main(){	double a,b,c,disc,x1,x2,p,q;	scanf("%lf%lf%lf",&a,&b,&c);	disc=b*b-4*a*c;	p=-b/(2.0*a);	q=sqrt(disc)/(2.0*a);	x1=p+q;	x2=p-q;	printf("x1=%0.2f\nx2=%0.2f\n",x1,x2);	return 0;}

运行结果1

2、

#include <stdio.h>int main(){	double a=3.0;	printf("%10.7f\n",a/7);	return 0;}

运行结果2.1

printf("%f\n",a/7);

基本型%f，不指定数据长度，系统处理的一般方法是：实数中整数部分全部输出，小数部分输出6位。

运行结果2.2

3、

将第二题中double改为float，即为本题答案。

float型数据只能保证6~7位有效数字。因此虽然程序输出了6位小数，但从左面开始第7位数字以后的数字并不保证是绝对正确的，可以自己试一试。
