除了PVC、PE、PP，电线电缆还有哪些材料？

2020-10-29
| 科技

1聚氯乙烯(PVC )

是一种极性极大的热塑性(塑料分热塑性和热固性)非晶体高分子电解质，所以机械性能好，介电常数大，在20C ，1MHz 下介常为40.~4.6。

含卤47%，难燃，配方成塑料后可燃，密度1.35g/cm立方硬质和软质密度不同，绝缘性能好，耐气候性强，耐化学性能，收缩率相当低，一般为0.2~0.6%。

耐温等级分70C ，90C ，105C ，CM ，CMR ，CMP ，对于105C 耐温等级及以上的PVC 材料应进行交联处理，就是通过电子加速器对材料的辐射使材料从直链状的线性结构变为网状结构，从而提高其耐热稳定性及机械性能。

主要用于绝缘和护套，因为PVC 的流动性能较差较少用于注塑。

2聚乙烯(PE )

分高密度聚乙烯(HDPE ：适合高速挤出);中密度聚乙烯(MDPE );低密度聚乙烯(LDPE ：通长用在电力电缆);线性低密度聚乙烯(LLDPE )。

易燃，吸水性小，介质损耗角和介电常数小，在20C ，1MHz 下无发泡介常为2.28~2.35，绝缘性能比PVC 好。填充性较差，通常用于通信线缆的绝缘或电力电缆护套中。

密度0.917~0.975g/立方厘米。为适应需要，我们国家早已研发出XLPE 。

3聚丙烯(PP )

力学性能和电性能与PE 非常相似，但耐环境应力比PE 好，缺点是铜对PP 有腐蚀作用，就是采用镀锡的方法效果也不是很好。

密度0.895g/立方厘米，在20C ，1MHz 下介电常数2.55。

4聚酰胺(PA )

也称尼龙，具有耐磨性，耐油性和高机械强度、耐热性能好，融熔温度高，通常均化段温度达300 C 。

吸水性强，介电常数大，绝缘性能较差，在20C ，1MHz 下介常为，一般不作绝缘使用，燃烧是发出动物体烧焦的气味。

其型号有尼龙6、9、11、12、66，610、612，还有我国自己研发的尼龙1010。

尼龙在制造线材是有3个缺点：

1)反白、这是由PA 的性质决定的。

2)附着力较差。

3)吸水性较大，在押出过程中容易起泡，因此在押出前应对材料进行预热，温度80~105C。

5氟塑料

在工业上常用的氟塑料是聚四氟乙烯(PTFE )，但在线缆用氟塑料上最常用的是聚全氟乙丙烯(FEP )，他们的密度在2.08~2.20之间。

除此之外，常用的氟塑料还有乙烯-四氟乙烯共聚物(ETFE )、聚偏氟乙烯(PVDF )。

密度在1.70~1.75之间。耐热等级至少200C 以上，除此之外，也具有非常好的耐寒性能。

耐化学腐蚀、吸水性小，在20C ，1MHz 下介电常数2.6。是目前在耐高温方面最好的两种塑料之一(另一种是硅橡胶电缆)。

通常用在高挡通信电缆、航空航天、雷达、原子反应堆用耐辐射电缆、车辆用电缆等等、地质探测电缆、测井电缆。

但缺点是工艺加工较困难，押出线材时使用无模具押出、耐电晕性较差、塑料对铜有腐蚀作用，必须镀锡，其次价格较高。

6聚酯

本材料的主要用途不在于做绝缘或护套，它在制造电线电缆过程中主要作用是用作线芯的包裹，同时还常和铝箔并用(涂在铝箔上)组成铝塑复合带，充分说明聚酯具有很好的绝缘性能。

7低烟无卤电缆料(LSOH )

低烟指的是燃烧产生的烟雾透光率》65%的塑料，有的公司甚至能控制在87%以上，无卤指的是进行腐蚀气体释放测试时1gLOSH 料在935C 高温30分钟所产生的气体质量在2mg —5mg 内，如果在5mg —15mg 范围内，我们称之为低卤。

现今流行的LOSH 料主要是以PP 、PE 、EVA 为主料，阻燃剂Mg (OH )2

、AL (OH )3为主，辅以SbO2阻燃剂，主要使其起协同效应，减少MgOH2的用量，Mg (OH )2用量大，且相容性、分散性差，抗冲击能力差，是导致LOSH 料机械性能较差的主要原因之一，因此必须加入偶联剂，使混合物能紧密结合，提高其延伸率和抗长强度，常用的偶联剂有硅烷类偶联剂。

同时此塑料非PVC 为主料，没有必要添加增塑剂，所以LOSH 具有非移形性，同时LOSH 料的热稳定性好，不必添加稳定剂，在一般PVC 的配方中，稳定剂和色粉是环保料的2大主要敌人!

配方例子PP+EVA(VA 含量20—30%)+ Mg (OH )2+偶联剂+滑剂+弹性剂+其他特殊添加剂FR LOSH 料我们常称之为热塑性低烟无卤阻燃聚烯烃，是一种新兴的环保电缆料，我们坚信，在10 年以后PVC 电缆料将会被我们淡忘!

PS：TPE 为热塑性弹性体，各地区的叫法不同，它是某些塑料的总称。
