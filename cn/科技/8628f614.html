世界各国核武器规模和核弹分类情况介绍

2020-11-14
| 科技

核武器，分几个阶段，原子弹，氢弹，中子弹（受控可调当量核效应弹）

弹头有效负荷相同时氢弹威力是原子弹10倍

美国现役最大（民兵3）单弹头100万吨当量，700个运载工具，1500枚核弹头

俄罗斯白杨M为100万吨当量，700个运载工具，1500枚核弹头

英国核力量在某种程度上是美国核力量的延伸，因为它的核弹是美国提供的，它有四艘弹道导弹核潜艇，一艘核潜艇有16枚核导弹，一枚导弹三个分弹头，实际上它总共处于作战状况下的核弹头应该是192枚，单弹头30万当量

法国拥有348枚核弹头，包括288枚海基核弹（部署在四艘核潜艇上），50枚空基核巡航导弹，10枚空基核弹。单弹头均不超过100万吨。

中国东风5A为300万吨(20枚)，东风31A为100万当量（估计40枚），东方21为30万当量（估计100枚）中国核弹头数量估计350~400枚与法国相当。

印度核弹头超过100枚，烈火5导弹携带1.5吨核弹头，当量20万吨，助爆型原子弹

巴基斯坦核弹头100枚左右

以色列核弹头100枚左右

朝鲜估计现在有造2~3枚小型化的能力

巴西既不承认有核武器，又不同意进行核查

伊朗不承认开发核武器，但又想保留核能力，并不与国际原子能机构完全合作

日本、德国等国家也具备核武器开发能力，但因政治原因而没有相关项目

有效杀伤距离 = C * 爆炸当量^(1/3)...... (C为比例常数,^(1/3)为求立方根)

一般取比例常数为1.493885

当量为10万吨时,

有效杀伤半径 = 1.493885 * 10^(1/3) = 3.22千米

有效杀伤面积 = pi * 3.22 * 3.22 = 33平方千米

当量为30万吨时

有效杀伤半径 =4.7千米

有效杀伤面积 =70平方千米

当量为100万吨时,

有效杀伤半径 = 1.493885 * 100^(1/3) = 6.93千米

有效杀伤面积 = pi * 6.93 * 6.93 = 150平方千米

有效杀伤半径 = 1.493885 * 300^(1/3) = 10.00千米

有效杀伤面积 = pi * 10.00* 10.00= 314平方千米

当量为1000万吨时,

有效杀伤半径 = 1.493885 * 1000^(1/3) = 14.93千米

有效杀伤面积 = pi * 14.93 * 14.93 = 700平方千米

朝鲜发展的是原子弹里面最初级的枪式钚弹，最大可能当量不超过4万吨。印度那个就是这种，印度爆了好像有1,2万吨。 但是这里，实际爆炸的当量跟装药当量是不一样的。比如设计装药当量是2万吨，炸出来实测1万吨，那就是50%的核燃料炸了，如果设计是2万吨，只炸了1000吨，那就只有5%的核燃料炸了，这叫不成功，顶多算爆轰，不算核爆炸。是技术不成熟的表现。

高级一点的是铀弹，这叫内爆式，复杂程度一下高了几十倍，这个当量最大好像能到25万吨，准确数字记不得了，但肯定不到50万吨。但到10万吨以上就会做得很大，飞机不好带，火箭不好打的。中国自己上来就搞的这个，跳过了钚弹阶段。

这两个阶段都是原子核裂变，能量小，污染大，所以叫原子弹，都是反应简单，只求杀伤力和污染，力求当量越大越好，杀伤范围越大。是核武的初级阶段。 这个阶段不存在小型化，因为技术根本做不到。

第二个阶段是氢弹，这个阶段，是聚变弹，是用原子弹引爆氢弹，所以，这种蛋蛋，里面有个原子弹，有个氢弹，很难做小，这才是小型化的难题，但只是体积小型化，而不是杀伤力小型化。

基本上，氢弹的威力是无上限的，从20万吨到1亿吨的都有过，老毛子就爆过设计一亿，裁剪到五千万的。 太阳实际就是个大氢弹。

再一步就是中子弹（或受控可调当量可调核效应弹）。这个阶段，还是氢弹，但反应方向可能是爆炸冲击波杀伤力，可能是放射线杀伤力，目标不同，对杀伤力的要求也不同，朝着可作战术武器的方向去的。这个阶段也要求小型化，低成本化，也会要求小杀伤力，但这个目标是用核弹大常规战争，低污染。

现在各个核大国在搞的更高阶的是非核弹引爆的氢弹，中子弹。这个能做得特别小，很考教技术。而原子弹，因为其裂变原理带来的污染，怎么改小当量，都会被认为是战略武器，不可能当战术武器使用。
