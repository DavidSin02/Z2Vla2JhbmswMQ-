微软：用户安装第三方杀毒软件后Defender将自动关闭

2020-11-14
| 科技

IT之家 8 月 27 日消息 默认情况下，Windows Defender 安装在 Windows 10 的所有版本和版本上。其总是在后台运行，以检测、隔离和删除恶意软件感染、勒索软件攻击和其他威胁。

一些用户认为，当他们打算安装另一个杀毒软件应用时，他们需要禁用 Windows Defender。该理论显然是错误的，Windows Defender 不会与另一个杀毒程序发生冲突。

微软一直允许用户进入 Windows 注册表，启用一个名为 “DisableAntiSpyware”的选项。一旦创建了这个数值为 “1”的条目，微软就会关闭 Windows Defender。

▲注册表编辑器

随着本月对 Windows Defender 的更新，微软表示，他们已经删除了那个永久禁用 Defender 的注册表选项，除非你正在运行 Windows Server 版本。

该公司现在已经更新了文档，说明如果用户安装了杀毒软件，Windows Defender 将自动关闭。如果你卸载了杀毒软件，Windows Defender 将再次开启。

“这一变化不会影响第三方杀毒软件与 Windows 安全应用的连接。这些仍将按预期工作，”微软指出。

虽然这个注册表设置受到 Windows Security 应用的保护，但只有在 Windows 10 家庭版和专业版的新版本中，才有都有 Tamper Protection 篡改保护功能。IT之家了解到，此前曾有恶意软件滥用编辑注册表，并关闭 Windows 10 的防病毒保护来入侵计算机。

这是停用 DisableAntiSpyware 选项的另一个很好的理由，也是微软防止恶意软件对 Windows 的安全性进行不必要的更改的正确举措。
