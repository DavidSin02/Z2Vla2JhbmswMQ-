产品规格: 2m x 1m x 1m, 3m x 1m x 1m, 4m x 1m x 1m, 2m x 1m x 0.5m, 4m x 1m x 0.5m，还可按客户要求进行生产。表面保护状态有热镀锌,镀锌铝合金,涂PVC等。
