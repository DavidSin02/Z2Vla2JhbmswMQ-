2020年一级造价工程师考试《水利计量》真题及答案解析

　　一、单选题

　　1、自稳时间很短，规模较大的各种变形和破坏都可能发生的水工隧洞围岩类型是( ) 。

　　A. II

　　B. II

　　C. IV

　　D. V

　　答案: C

　　解析: P15

　　2、土的物理力学性质指标中，通过试验测定指标换算获得的是()。

　　A.土的天然密度

　　B.含水率

　　C.土的容重

　　D.干密度

　　答案: D

　　解析: P4-5。试验测定指标:土的天然密度(重度)、土的含水率、土粒比重:换算指标:土孔隙比、饱和度、饱和重度、干重度、干密度。

　　3、根据含水层的空隙性质，可将地下水分为孔隙水、裂隙水及()。

　　A.包气带水

　　B.潜水

　　C.岩溶水

　　D.承压水

　　答案: C

　　解析: P23。 根据含水层的空隙性质，地下水又分为孔隙水、裂隙水和岩溶水三个亚类。

　　4、下列不属于岩浆岩的( ) 。

　　A.花岗岩

　　B.玄武岩

　　C.安山岩

　　D.片麻岩

　　答案:D

　　解析: P10。 岩浆岩分侵入岩和喷出岩。侵入岩分深成岩(如花岗岩、正长岩、闪长岩、辉长岩等) :浅成岩(如花岗斑岩、闪长玢岩、辉绿岩、脉岩等) :喷出岩(流纹岩、粗面岩、安山岩、玄武岩、火山碎屑岩等)。

　　5、碳素钢牌号Q235, Q代表( ) 。

　　A.抗拉强度

　　B.屈服强度

　　C.质量等级

　　D.含碳量

　　答案: B

　　解析: P70。Q235-B.F，表示屈服强度为235MPa,质量等级为B级的沸腾钢。

　　6、一黏性土样的液性指数为0.55，土样状态为( )。

　　A.坚硬

　　B.硬塑

　　C.可塑

　　D.软塑

　　答案: C

　　解析: P7。

　　7、在堆石料原岩的质量技术指标中，一般要求软化系数大于( )。

　　A 0.55

　　B.0.65

　　C 0.75

　　D 0.85

　　答案: C

　　解析: P111

　　8、工程中目前应用最广泛的水泥类型()。

　　A.矿渣水泥

　　B.硅酸盐水泥

　　C.铝酸盐水泥

　　D.铁铝酸盐水泥

　　答案: B

　　解析: P62。水泥品种繁多，目前常用的是硅酸盐水泥。

　　9、工程现场用标准贯入试验测砂土密实度，标准锤击数为13，砂土的密实度为( ) 。

　　A.松散

　　B.稍密

　　C.中密

　　D.密实

　　答案: B

　　解析: P6。

　　10、水利工程建筑材料中绝热材料是指常温时导热系数不大于( ) w/ (m*k)。

　　A.0.350

　　B. 0. 175

　　C.0.05

　　D. 0.075

　　答案: B

　　11.某水库工程总库容为1亿m3,年引水量为1亿m3,供水对象的重要性为比较重要，该水库的工程等别为( ) 等。

　　A. I

　　B. II

　　C. III

　　D. IV

　　答案: B

　　解析:P125。根据下表的指标分属机构不同等别时，整个枢纽的等级应以其中的最高等别为准。

　　12、混凝土中加入减水剂的作用是( ) 。

　　A.调节混凝土拌合的流变性

　　B.调节混凝土强度

　　C.调节混凝士凝结时间

　　D.调节混凝土水化热

　　答案: A

　　解析: P92。 改变混凝土拌合物流变性能的外加剂，包括各种减水剂、引气剂及泵送剂。

　　13、一般水工混凝土抗冻等级分为( ) 个等级。

　　A.4

　　B.5

　　C.6

　　D.7

　　答案: C

　　解析: P132.根据混凝土试件在28d龄期所能承受的最大冻融循环次数分为F50、F100、 F150、F200、F250 及F300六种等级。

　　14、硅酸盐水泥的初凝时间不得早于( ) 。

　　A.45min

　　B.90min

　　C.180min

　　D.390min

　　答案: A

　　解析: P63。 硅酸盐水泥的初凝时间不得早于45min,终凝时间不得迟于6.5h;普通硅酸盐水泥初凝时间不得早于45min,终凝时间不得迟于10h。

　　15、载重汽车按吨位分，总重量为( ) t 属于中性卡车。

　　A. <1.8t

　　B.1.8~6t

　　C. 6~14t

　　D.14~100t

　　答案: C

　　解析: P267。 按承载吨位分为微型卡车、轻型卡车、中型卡车、重型卡车、超重型卡车。微型卡车总质量小于1.8t，轻型卡车总总量1.8~6t,中型卡车总质量为6~14t，重型卡车总质量为14~ 100t，超重型卡车总质量为100t以上。

　　16、 水工建筑物按使用期限分为( )

　　A.永久性建筑物和临时性建筑物

　　B.永久建筑物和次要建筑物

　　C.主要建筑物和次要建筑物

　　D.主要建筑物和临时性建筑物

　　答案: A

　　解析: P123。 水工建筑物按使用期限分为永久性建筑物和临时性建筑物。

　　17、蓄滞洪区的堤防级别宜为( )级。

　　A.1

　　B. 2

　　C.3

　　D.4

　　答案: B

　　解析: P127。 蓄滞洪区安全区的堤防永久性水工建筑物级别宜为2级，对于安置人口大于10万人的安置区，经论证后堤防永久性水工建筑物级别可提高一级。

　　18、修建丁坝、顺坝等，将水流挑离提岸，以防止水流、波浪或潮汐对堤岸的冲刷的是( )。

　　A.坝式护岸

　　B.墙式护岸

　　C.坡式护岸

　　D.平顺护岸

　　答案: A

　　解析: P191。 坝式护岸是指修建丁坝、顺坝等，将水流挑离提岸，以防止水流、波浪或潮汐对堤岸的冲刷，这种形式多用于游荡性河流的护岸。

　　19、坝体混凝土抗压强度设计龄期- -般采用( ) 。

　　A.28d

　　B. 90d

　　C.180d

　　D.360d

　　答案: B

　　解析: P131。 坝体混凝土抗压设计龄期- -般采用90d，最多不宜超过180d。

　　20、土质斜墙土石坝，斜墙上下游面应分别设置( )。

　　A.保护层、过滤层

　　B.垫层过、滤层

　　C.保护层、反滤层

　　D.垫层、反滤层

　　答案: C

　　解析: P142。

　　21、沥青混凝土面板、砂砾土坝为调节坝体变形，面板下面应设置( ) 。

　　A.垫层和过渡层

　　B.垫层和反滤层

　　C.过滤层和反滤层

　　D.整平层和垫层

　　答案: A

　　解析: P144~ 145.沥青混凝土斜墙铺设在垫层上，垫层的作用是调节坝体的变形。过渡层主要对其两侧土料的变形起协调作用。

　　22、拱坝坝高大于( ) 米为高坝。

　　A. 30

　　B.50

　　C.70

　　D.90

　　答案: C

　　解析: PI51。 按坝高分类，大于70m为高坝，30 70m为中坝，小于30m为低坝。

　　23、.... 正槽溢洪道控制堰体轴线与( ) ( 题目不全)

　　A.坝轴线正交或近于正交

　　B.坝轴线平行或近于平行

　　C.泄槽轴线正交或近于正交

　　D.泄槽轴线平行或近于平行

　　答案: C

　　解析: P154。

　　24、习惯将用于软土地层的隧道掘进机称为( )。

　　A.敞开式TBM

　　B.双护盾TBM

　　C.单护盾TBM

　　D.盾构机

　　答案: D

　　解析:习惯将用于软土地层的隧道掘进机称为盾构机，将用于岩石地层的称为TBM。

　　25、根据工程习惯主厂房以发电层楼板面为界，分为上部结构和下部结构。下部结构除发电机层楼板外，均为大体积混凝土，包括( ) 风罩、蜗壳外围混凝土、尾水管外围混凝土、防水墙等。

　　A.引水钢管镇墩

　　B.厂房基础

　　C.尾水边墙

　　D.机墩

　　答案: D

　　解析: P169。下部结构除发电机层楼板外，均为大体积混凝土结构，包括机墩、风罩、蜗壳外混凝土、尾水管外围混凝土、防水墙等。

　　26、将上下游水位分为若干级，形成梯形水位跌落的鱼道形式分为( )。

　　A.斜槽式

　　B.水池式

　　C.隔板式

　　D.仿生态式

　　答案: C

　　解析:隔板式鱼道是在水池式鱼道的基础上发展起来的，利用横隔板将鱼道上下游总水位差分为若干级，形成梯级水面跌落，故又称梯级鱼道。

　　27.平洞石方开挖是指水平夹角小于或等于( )的石方洞挖工程。

　　A. 5°

　　B. 6°

　　C. 7°

　　D. 8°

　　答案: B

　　解析: P386。 平洞石方开挖是指水平夹角小于或等于6°的石方洞挖工程。

　　28、将截流材料从龙口的一嘴向 另一端或从两端向中间抛投进占，逐渐束窄龙口，直到全部截断的方法是( )

　　A.平堵法

　　B.下闸截流

　　C.立堵法

　　D.定向爆破截流

　　答案: C

　　解析: P290。立堵法是将截流材料从龙口的一端向另一端或从两端向中间抛投进占，逐渐东窄龙口，直到全部截断。

　　29、正铲挖掘机的铲土动作形式特点是( ) 。

　　A.向前向上，强制切土

　　B.向后向下，强制切土

　　C.向后向下，自重切土

　　D.直上直下，自重切土

　　答案: A

　　解析: P255. 正铲挖掘机铲土动作形式特点是:前进向上， 强制切土。

　　30、格宾石笼、格宾垫是近些年发展起来的生态格网结构，其格网材料一般采用( ) 。

　　A.热镀锌钢筋

　　B.热镀锌铅丝

　　C.热镀锌低碳钢丝

　　D.热镀锌高碳钢丝

　　答案: C

　　解析: P194。 格宾石笼、格宾垫的网材称为格网，一般采用PVC覆层或无PVC覆层的经热镀工艺进行抗腐处理的低碳钢丝等编制。钢丝宜为热镀锌低碳钢丝、铝锌混合稀土合金镀层低碳钢丝，以及经高抗腐处理的以上同质钢丝等。

　　31、铜芯交联聚乙烯绝缘聚氯乙烯护套，钢带铠装电力电缆表示为( )。

　　A. VV22

　　B. ZLL32

　　C. YJU22

　　D. VIV42

　　答案: C

　　解析: P233.交联聚乙烯绝缘电力电缆: YJV 表示铜芯交联聚乙烯绝缘聚氯乙烯护套电力电缆:YJILV22表示铝芯交联聚乙烯绝缘聚气乙烯护套，钢带铠装电力电缆。

　　32、用浆液填充混凝土与围岩或混凝士与钢板之间的空隙和孔洞，以增强围岩或结构的密实性。

　　A.固结灌浆

　　B.接触濯浆

　　C.回填灌浆

　　D.接缝灌浆

　　答案: C

　　解析: P316。回填灌浆是用浆液填充混凝土与围岩或混凝土与钢板之间的空际和孔洞，以增强围岩或结构的密实性。

　　33、以下( ) 不属于水利水电工地的“综合加工厂”。

　　A.混凝土预制件厂

　　B.钢筋加工厂

　　C.木材加工厂

　　D.汽车修配厂

　　答案: D

　　解析: P362。水利水电工地的混凝土预制件厂、钢筋加工厂和木材加工厂等,可视原材料供应、交通运输，场地面积即施工管理等条件分开设置或联合设置。在场地条件允许的情况下，宜联合设置，简称为“综合加工厂”。

　　34、下列不属于二次设备安装内容的是( ) 。

　　A.调试

　　B.防火封堵

　　C.调线

　　D.元器件整定

　　答案: B

　　解析: P239.二次设备安装归纳起来有以下内容:立盘、配线、继电器和元器件的整定、调试。

　　35、碾压混凝土坝浇筑方式( ) 。

　　B.通仓厚层浇筑

　　C.通仓薄层浇筑

　　D.分块浇筑

　　36、某水电站规划装机容量为1200W，,运行水头范围为16米，电站拟选用4台相同容量的水轮发电机组，最合适的水轮机型式是( )。

　　A.混流式

　　B.轴流转浆式

　　C.灯泡贯流式

　　D.轴伸贯流式

　　答案: C

　　37、水利工程中常用施工机械胶带输送机属于( ) 。

　　A.土石方机械

　　B.混凝土机械

　　C.起重机械

　　D.运输机械

　　41.在地质条件较好的地下厂房中，项拱开挖采用中导洞先行掘进，两侧扩挖跟进的方法，一般中导洞超前( ) 。

　　43、开挖坡高小于5m的坚硬黏性土土质边坡时可选容许坡度为( ) 。

　　46、当周围没有山坡可依靠，塔顶又设有操纵闸门井平台和启闭机室，并建桥与岸边或坝相连通的隧洞深孔进口形式有( ) 。

　　C.岸塔式进口

　　D.斜坡式进口

　　47、闸门安装好后，应在无水状态下做全行程

　　A.机组试运行

　　B.启闭试验

　　C.耐压试验
