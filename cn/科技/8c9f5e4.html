【理论基础】

1.mysql自带了不少函数，但是不一定都能满足我们的需求，所以我们可以用function来自定义函数，完成一定的功能，并返回结果。

2.语法:

begin

SQL语句...

end

【案例操作】

1.￥￥创建一个计算斐波那契数列的函数￥￥

create function f_factorial (num int) returns int

begin

declare result int default 1;

while num>1 do

set result=result*num;

set num=num-1;

end while;

return result;

end

【存储函数与存储函数的区别】

1.存储过程可以有多个in,out,inout参数，而存储函数只有输入参数类型，而且不能带in.

2.存储过程实现的功能要复杂一些；而存储函数的单一功能性（针对性）更强。

3.存储过程可以返回多个值；存储函数只能有一个返回值。

4.存储过程一般独立的来执行；而存储函数可以作为其它sql语句的组成部分来出现。

5.存储过程可以调用存储函数。函数不能调用存储过程。
