电力拖动的运动方程式

在电力拖动中涉及的相关物理量有转矩，转动惯量，角速度和角加速度。G*D的平方为飞轮矩，飞轮矩与转动惯量有关。最简单的电力拖动系统为单机单轴系统，多机多轴系统也存在但是较为复杂。

在多轴系统中还涉及到了折算：如负载转矩的折算，飞轮矩的折算，平移部件质量m的折算。

生产机械地负载转矩特性

1、恒转矩负载特性

当转速变化时，负载转矩的大小保持不变，称为恒转矩负载。

（1）反抗性恒转矩负载： Tr 的大小与 n 无关，但是 Tz 的方向始终与 n 相反。

（2）位能性恒转矩负载： Tz 的大小及方向均与 n 无关，而且 Tz方向始终指向使位能部件的势能减小的方向。

2. 恒功率负载特性

负载转矩与转速成反比。

3. 通风机负载特性

负载转矩与转速平方成正比。

他励直流电机的制动

1.制动运行 其他不变，电磁转矩与转向相反

2. 能耗制动 保持If大小方向不变，将电枢回路从电网脱离经制动电阻 Rz闭合。

3. 反接制动 保持 If 大小方向不变，令电枢经制动电阻 Rz 而反接于电网。

4. 回馈制动 保持 If大小方向不变，将电枢反接，从而使位能性负载以较高的转速稳速下放。

他励直流电机的过渡过程

1. 当电力拖动系统的负载、参数或运行方式等发生变化时，由于系统具有惯性（机械惯性、电磁惯性与热惯性等），使之从一个稳定运行状态变为另一个稳定运行状态时，系统各物理量的改变不可能瞬时完成，必须经一个连续变化的过程，称为过渡过程。在此过程中，各物理量随时间的变化规律称为拖动系统的动态特性 。
