Unison Unico CD Uno CD唱盘

2020-10-29
| 科技

意大利音响厂牌Unison Research不但以管机著称，而且他们还喜欢将意大利最为擅长的木工运用在管机箱体上，因此Unison Research的管机运作时，其机箱流线型的金属零件反射着真空管炽红的光芒，加上以木工打造的流线型面板，带来听觉与视觉双重享受，也赋予Unison Research管机名贵的印象。

不过，近几年来，音响市场逐渐萎缩，年轻族群的聆听习惯也与先前的世代大不相同，许多Hi-End厂牌为了吸引年轻族群的目光，纷纷推出转型之作，Unison Research也不例外，自2000年起，他们邀请意大利帕多瓦（Padua）大学的Leopoldo Rossetto教授，参与「Unico」系列的开发计划，除了推出多台管晶混合设计的放大器，更考量到现在有多种音乐格式在市场流通的状况，推出结合CD播放机、USB DAC与无线蓝牙的数字讯源器材。这次试听的主角——Unico CD Uno，便是其中一例。

Unico CD Uno：CD/USB/蓝牙三合一

根据原厂的说明，这台Unico CD Uno是延续 CD Duo设计推出的最新CD播放机。纯铝合金打造的箱体，中央可以显示音档取样率的白色128 X 64 OLED显示屏幕，两旁的电源以及基本播放功能按键，还有抽屉式的CD承盘，各部外观与尺寸，几乎与CD Duo一模一样。不过，Unico CD Uno比起Unico CD Duo少了AES/EBU XLR数字输入端子，保留了较为常用的USB Type B与Toslink光纤数字音讯输入端子，以及一个S/PDIF同轴端子，而且还具备连接手机十分便利的蓝牙3.0无线连接功能。

Unico CD Uno内部电路，可以看出DAC电路与模拟电路位于不同的电路板上，减少互相干扰的机会发生。（图片由垣庆提供）

这台CD播放机采用奥地利公司StreamUnlimited生产的8829-CD承盘，我在使用时，发现此承盘进出顺畅，不会有抖动的情形发生。Unico CD Uno外观虽然看起来传统，但是播放功能丝毫不落伍。其DAC电路搭载ESS Sabre32 9018 DAC芯片，并还有储存自写程式的Xilinx Spartan FPGA芯片，做为数字音讯的处理引擎，时脉则由石英震荡器负责，以减少Jitter的产生。

在此电路设计之下，Unico CD Uno的USB输入可以支持至384 KHz/32 bits的PCM档以及11.2 MHz/1 bit的DSD档，而通过Toslink光纤输入，也可支持至176.4kHz/24 bits的PCM档，以及2.8MHz/1 bit的DSD档，让用户得以聆听市面上大多数的Hi-Res音档。

Unico CD Uno的DAC电路搭载ESS Sabre32 9018 DAC芯片，并还有储存自写程式的Xilinx Spartan FPGA芯片，做为数字音讯的处理引擎。（图片由垣庆提供）

此外，Unico CD Uno也提供三种数字低通滤波选择，包含滚降斜率陡直、线性相位的「F1」、滚降斜率陡直、最小相位的「F2」、滚降斜率平缓、线性相位的「F3」，这三种数字滤波分别对时间一致性、相位一致性以及暂态细节有较为理想的效果。我在试听时，试着切换这三种滤波器，发现其对声音的影响并不大，很难说孰优孰劣，不过，若是觉得音乐略为刺耳，选用「F1」与「F2」可以达到较为舒适的听感。

搭载ECC82真空管

由于代理商特别嘱咐，我无法将Unico CD Uno的箱体打开，观看内部电路布局，不过根据原厂提供的资料，Unico CD Uno的DAC电路采用8组稳压电源供应。在真空管电路上，搭载了1支12AU7/ECC82真空管，采图腾柱式（totem pole，也就是台湾俗称的推挽式）设计，跟搭载4支12AX7/ECC83的CD Duo相比，换管更为容易，省去配对的烦恼。电路上也分别为真空管的阴极与栅极设立独立的稳压，维持电路的稳定。

最后，Unico CD Uno的缓冲及采用了德仪出产的NE5532AD OP芯片作为I/V转换，并搭载威玛（WIMA）电容以及威士（Vishay）的电阻，这些都是许多发烧友心仪的好声元件。玩家还可通过Unico CD Uno的功能选单，选择关闭真空管，让Unico CD Uno以纯电晶体方式发声，就算真空管寿命已尽，来不及换管，也不用担心这台器材无法发声运作。

Unico CD Uno 搭载了1支12AU7/ECC82真空管，并采用威玛（WIMA）电容以及威士（Vishay）电阻等发烧元件。

由以上的介绍不难看出，Unico CD Uno表面上看起来虽然是传统的CD播放机，但是具备了可以与手机直接连线的蓝牙功能，U也能满足现在大多数玩家播放数字流以及手机音乐的需求，价格上也比CD Duo少了5万元之谱，是Unison Research贴近年轻、中阶玩家的数字讯源力作。

Unico 90：可支持Bi-Wiring的管晶混合综扩

我本来以为代理商只会单独送来Unico CD Uno到我们办公室进行评测，没想到代理商还同时送来同系列的Unico 90真空管综扩以及体积十分庞大的Max 2落地喇叭，与Unico CD Uno一起搭配试听，阵仗之大，让我感到受宠若惊。Unico 90厂商定价为195,000元，比起Unison Research先前推出的放大器相比，价格也是亲民许多。

虽然Unico 90少了Unison Research管机向来惯有的木板装饰，让这台真空管综扩看起来朴素不少，不过Unico 90重达20公斤，机箱体积看起来也十分庞大，让人觉得分量十足，再加上雾面处理的灰色铝合金面板，平实中亦散发精致的质感。

Unico 90中的电路可分成输入级与输出级两大部分。输入级为具有真空管的「图腾柱式」推挽电路，其中在每个声道中，搭载一枚ECC83三极管以及双并联ECC81真空管。输出级各声道搭载了三对由国际整流公司（International Rectifier）所生产的HEXFET MOSFET电晶体作为功率放大元件，并采用兼具音质与效率的A/B类混合设计，在小功率输出时，以纯A类扩大方式运作，若是输出功率超过一定的瓦数，那么就会以效率较高的A/B类混合方式运作，以传达较高的输出功率。如此设计下，让Unico 90在8欧姆承阻下输出功率可达100W，在4欧姆承阻下，输出功率可达180W。

这台放大器也与Unico CD Uno一样，具有弹性极高的搭配性能，除了在背板上搭载3组非平衡输入端子以及2组平衡输入端子，每个声道更具备2组喇叭输出端子，可支持Bi-Wiring连接方式，提供喇叭绝佳的驱动力。为了让Unico 90有着自然舒适的音色，原厂在里面不采用整体负回授电路，只使用局部负回授电路。除此之外，Unico 90还具备前级电路bypass功能，搭配其它前级单纯作为后级使用，更具备超低音输出，为2.1声道玩家提供便利的连接方式，可说是一台设计精良、进退皆宜的综合放大器。

Max 2：号角高音加布边振盆低音，为管机而生

号角喇叭与真空管放大器为许多发烧玩家心目中的梦幻搭配，而Unison Research为了搭配自家的真空管放大器，便推出了两款两音路号角喇叭——Max 1与Max 2。这次送来与Unico CD Uno搭配的Max 2，采落地式式设计，Max 2高达101公分，再加上宽障板的设计，让Max 2的宽度也达43.5公分，而箱体深度也达53公分，让Max2看起来颇具份量，箱体容积高达130公升。

Max 2的中高音单元采用1.4寸钛振膜压缩式单元，以铝质线圈推动，兼具质地轻盈、散热性佳的特性，为单元带来绝佳的反应速度，并减少热能产生的失真。在单元前方，搭载有60度乘以50度的恒定角度椭圆形号角（Constant Coverage ESS Horn），以抑振效果绝佳的聚胺甲酸酯（polyurethane）铸造而成，如此一来，压缩式单元在发声时，号角不易随声波产生共振，可以避免不必要的音染。

Max 2的高音单元搭载椭圆形号角，以聚胺甲酸酯铸造而成，并以60度乘以50度的扩散角度传导音波。

低音单元则采用1枚口径达15寸的单元，其振膜以植物纤维以及碳纤维混合材质制程，而且表面具有多道同心圆皱折，以增强单元振膜的韧度，提升单元的功率承受能力。此外，该低音单元采用有「Ｗ」三道皱褶形状的布质悬编，以51%的天然棉以及49%的聚酯纤维制成。这样的悬边设计，除了提升单元的功率承受能力外，更重要的是还能减少单元的冲程距离，进而提升单元的反应速度，以重现更为清晰的细节。

至于冲程距离虽然减少，但该单元本身口径高达15寸，能推动充足的空气量，传达的声音能量不至于有所减损。除此之外，该低音单元的磁力系统以强力的钕磁铁制成，并搭载直径达10公分的线圈以及铝质套筒，让该单元可以产生800瓦的连续输出功率。在该单元的磁铁周围，还搭载有两个特制个金属环，原厂声称这样的设计可以调节磁力线的变化，减少谐波以及内调失真。

Max 2低音单元口径达15寸的单元，其振膜以植物纤维以及碳纤维混合材质制程，而且表面具有多道同心圆皱折，以增强单元振膜的韧度，提升单元的功率承受能力。此外，该低音单元采用有「Ｗ」三道皱褶形状的布质悬编，以51%的天然棉以及49%的聚酯纤维制成。低音反射孔位于箱体前方，摆放时不用担心后墙会影响到低音单元的运作。

与Opera联手打造

此对喇叭分频点设于800Hz，分音电路采取不对称的设计，低通滤波的滚降速度为每八度12dB，高通滤波的上升速度为每八度6dB，分音电路采用意大利SET制造的印刷电路板，并采用无氧铜绕制的电感、Mundorf电容以及MOX电阻。在Max 2的后方，还有一个切换开关，可以改变高通滤波器的斜率，近而改变高音频段的EQ。

Max 2的箱体与Max 1一样同样采用反射式设计，而且反射孔为于前障板下方， 如此一来让反射孔不会与后墙过近，让喇叭摆位更为自由。至于箱体本身以及前障板分别以25mm、30mm厚的MDF板制成，而箱体内部采取龙骨支架结构，以强化箱体的抑振能力，而箱体内壁则涂有开放式聚胺甲酸酯（open cell polyurethane）泡棉，减少箱体内音波反射产生不必要的共振。

Max 2箱体外观以色泽华美的上漆原木贴皮处理，前障板则以合成皮革包覆，这样的设计，不禁让我联想到一个意大利喇叭厂牌——Opera推出的喇叭，事实上，Unison Research与Opera早在1998年为「Audio Research in Audio」集团收购，Max 2身上会有Opera的影子，一点也不意外。除此之外，Max 2扎实的箱体设计，以及几乎与低音单元直径等宽的前障板，相信也都承袭自Opera的设计理念。

如同文艺复兴绘画般的高贵气息

我一开始将Unico CD Uno与Unico 90器材，直接插上排插播放音乐。不知道是不是因为号角喇叭效率较高，对于前端器材较为敏感，或是当时排插还连接着数台其它器材发现音质明显较为粗糙、毛躁，不似这套系统该有的水准。后来我将Unico CD Uno、Unico 90器材接上Linear Acoustic Fuzzy电源，原有的毛躁听感顿时减少许多，音质变为细致柔和。由此可见，聆听这套系统时，电源器材必须审慎选用，否则会容易觉得这套系统不耐听，导致「误杀忠良」的错误判断。

试听时，我以CD做为主要播放素材。我首先播放由英国知名女高音Emma Kirkby所演唱蒙特维尔地牧歌第七册中的〈Con Che Soavita, Labra Odorate〉（Decca 4557182），Emma Kirkby的歌声纯净无暇，控制力犹如长笛一般精准。通过这套系统播放，我可以立刻感受到这位女高音歌声的圆润高雅，以及大键琴如同珠玉般闪耀的光芒，加上人声高雅细腻的强弱变化中，流露出雀跃兴奋的喜悦之情，彷佛可以看到Emma Kirkby微笑唱着歌。

在音乐播放的过程中，我发现无论是人声或是其他伴奏乐器，声音直接，反应敏锐，而各种声音出声后绵延的残响，更强烈地渲染试听室中的气氛，彷佛把视听室重新打造成音乐厅一般，让我如临现场，直接感受音乐家的动人演出。这样活灵活现的声音表现，还有绝佳的扩散性，向来是号角喇叭的特色，不过聆听许久，我发现人声的结像过于突出，音乐情绪也略微兴奋，这时我跑道喇叭后方一看，发现原来高音的EQ切换钮停留在+档的地方。我将EQ调至0档，再把音量旋钮调高至5度10度，会发现声音更为干净，音场更为深邃，每个声音不仅听起来散发独特的香味以及珍珠般温润的光泽色彩，整体音乐诠释更为深情、高雅。

接着，我播放列夫席兹（Konstantin Lifschitz）弹奏的巴哈《郭德堡变奏曲》（Orfeo C864141A）。这张他于2015年的录音，相较于他于1995年为Denon所录制的同曲目演奏，可以用「从心所欲不逾矩」形容，除了节奏分配更加自由灵活，弹至弱音之处，探微赜奥；弹至强音之处，大鸣大放。

通过这套系统聆听，我不但听到琴音的色泽犹如珍珠般多彩润泽，而且在播放至弱音乐段，可以清楚听见键触之间的强弱差异微妙精细，可以明显感受到演奏家在乐曲上精雕细琢的细密心思，键触起落细节历历在目，而且细节清晰，起落灵巧；播放至强音乐段，我更感受到毫无掩饰的情绪传达，还有列夫席兹剧力万钧的惊人弹奏气势，如此可见这套系统非但音色高贵细腻，而且动态极为宽广，情绪表现十分生动活泼。

情绪渲染力强烈

聆听上面的古乐以及钢琴独奏，我可以确定这套系统可以完美呈现高贵的气质，这让我不由得好奇播放摇滚乐会呈现何种听感。于是，我播放品客佛洛伊德《Wish You Were Here》（EMI– 50999 0 82613 2 8 ）中的〈Shine on your crazy diamond〉。通过这套系统聆听，乐曲一开始，如同海鸥鸣叫般的电子音效尽管音量微弱，但是线条感特别凝聚、清晰，电风琴的声音中微微的Vibration特效，亦能清楚呈现，随后出现的电吉他弹奏，滑指效果也相当鲜明，让整个音乐听起来特别扣人心弦，将我带入一个未知的想像世界。

这套系统亦能展现摇滚乐中该有的魄力，像是乐曲中脚踩鼓所营造的节奏，听起来噗通有力，小鼓音色清脆，收放自如，中段电吉他亦能表现出暴力、粗犷的质感，并没有因为这套系统采用真空管以及号角拉吧的古典配置，而有所减损。至于人声演唱，通过这个系统播放，也能展现激昂的情绪，尤其David Gilmour唱至「You were caught on the crossfire of childhood and stardom」一句，发「caught」中「k」这个子音时，可以明显感受歌者舌根用力顶住软腭，制造强烈的气流爆破声，彷佛喇叭就是歌者的嘴巴，感觉有气流以及从其喷出，能量直接而强烈。乐曲播放至11:30的地方，空心吉他的弹奏声响出现在右侧，管乐器吹奏出如同咆勃爵士乐，则位于中央前方，爵士鼓的打击生则结像于中后方，营造出栉比鳞次的透视感，让我看见品客佛洛伊德刻意创造的超现实音乐景象。

最后，我播放普契尼歌剧《蝴蝶夫人》（Decca 4175772）中第三幕最后，由蝴蝶夫人自杀前所演唱的咏叹调〈Con amor muore chi non può serbar vita con onore〉。这首乐曲中，开场低音弦乐器与管乐器以极为沉重的重量，预示蝴蝶夫人即将悲剧性的结束自己的生命，尽管通过Max这对喇叭播放，表现众多低音乐器齐奏丝毫不费吹灰之力，但是乐器声音出手直接，而且节像在前，再加上背景变换的声响还有其他演员的脚步声结像位于喇叭的后方，让试听是充满凝重的低气压，以及躁动不安的空气分子，感染力时分强烈。女高音Mirella Freni以激昂的歌声，唱出对男主角Pinkerton的不满控诉，通过这套系统的播放，抖音如同电波一般触动我心，听起来慷慨激昂，情绪丝毫没有任何保留。

4分16秒，蝴蝶夫人自杀后出现的大锣声，更是形体庞大，尾韵不绝，而且又能呈现出金属打击乐器迷人的光泽感。最后Pinkerton赶到现场，看见死去的蝴蝶夫人时，乐团演奏的能量高涨至顶峰，通过这套系统聆听，更能感受到宽阔的音场以及深邃的舞台透视感，气势滂礡。由此可见，这套系统表现大编制、大场面的音乐，不但举重若轻，而且音乐中血肉灵魂丝毫没有减损，不会让人觉得只有靠蛮力发声。

在试听这套系统时，PS Audio Stellar DAC正好也在我们公司的试听室，于是我将CD Uno与PS Audio的Stellar（音量开到全满）两相比较，通过USB播放上面CD所转出的数字音档。比较之中，我发现这Stellar在细节清晰度方面比CD Uno略胜一筹，但两者之间实际并不会差距太远，通过Unico CD Uno聆听，同样可以听见小提琴擦弦以及歌唱流窜在唇齿间的气流声，只是Unico CD Uno对于细节的呈现方式更为柔和。在音色方面，相较于Stellar的中性，CD Uno或许因为有真空管的加持，音色总是带有丝绸般华美、高贵的质感，其所带来的情绪感染力也极为深刻，这种高贵细腻、舒适柔和的质感，实在让人不忍换下。

由于Unico 90综扩与Max 2喇叭上皆有两对喇叭端子，我便尝试看看以Bi-wiring接法耹听。本来期待而来的是更为庞大、有力的低音，但迎面而来的是更干净的背景、更深邃的音场、更精准的控制力，音乐表现层次大为提升，舍不得换回一般喇叭接线方式了。

让人大喊「Bravo」的意式绝美风格

这次试听的Unico CD Uno，虽然是Unison Research的入门级讯源机种，朴实的铝合金箱体，不同于Unison Research器材予人精雕细琢、工艺精湛的印象，但是他所带来的深刻情绪、高贵气质还有不拘一格的表现力，已在我心中留下无可抹灭的美好印象。将其搭配Unico 90综扩以及Max 2，全套的意式风情，加上经典的管机与号角喇叭搭配，更为音乐带来生动无比的气氛，让我彷佛坐在史卡拉歌剧院中，观赏一场风味正宗的意大利歌剧演出，聆听完毕，让我忍不住起身鼓掌，为这些来自意大利的「声乐家」高喊Bravo!
