Hornsea Project One海上风电场敷设首条输出电缆

2020-10-29
| 科技

【电缆网讯】荷兰海上解决方案供应商Tideway已为Ørsted拥有的1.2吉瓦Hornsea Project One海上风电场敷设了首条海上输出电缆。

Hornsea Project One海上风电场敷设首条海上输出电缆

Hornsea Project One风力发电场将位于约克郡海岸120公里外，将由三条高压海底电力电缆从三个海上变电站向岸边输送电力。

Tideway通知称，正在使用Ocean Yield公司的电缆敷设船“连接器”执行第一部分的电缆敷设。

“连接器”敷设船有两个大转盘，可容纳6000吨和3000吨电缆。对于这项工作，Tideway公司为输出电缆项目设计并安装了一个定制的后甲板电缆敷设层，取代原有的敷设系统。

Tideway的工作范围还包括电缆安装、工程设计、除石、预挖沟、电缆接入变电站，交叉安装、海上连接和电缆埋设。

Tideway项目经理Ken Swaegers表示：“这对我们来说是一项非常令人兴奋的工作，同时也是开展海上风电最大型建设项目的绝佳机会。由于它的规模，我们有若干船只在现场运行。并在夏季，我们自己的全新电缆敷设船Living Stone也会加入。Living Stone将配备创新的双轨道电缆安装系统，由两条电缆高速公路组成，一条用于铺设电缆，另一条可同时准备下一条电缆，显著提高工作效率。”

Hornsea Project One董事Duncan Clark称：“很高兴Tideway能够调动‘连接器’敷设船来满足我们对这个项目的具体要求，同时期待Living Stone来到现场帮助建设这个非凡的风电场项目。 ”

2020年投入运营后，Hornsea Project One的容量将超过1吉瓦，并将能够为超过100万个家庭提供清洁电力。
