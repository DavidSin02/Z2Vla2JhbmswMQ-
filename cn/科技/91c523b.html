规范学习内容

GB50202-2002 《建筑地基基础工程施工质量验收规范》

1、 灰土地基、砂和砂石地基、土工合成材料地基、粉煤灰地基、强夯地基、注浆地基、预压地基，对地基强度或承载力进行检验，每单位工程不应少于3点，1000m2以上工程，每100m2至少应有1点，3000m2以上工程，每300 m2至少应有1点，每个独立基础下至少应有1点，基槽每20延长米应有1点。

2、 水泥土搅拌桩复合地基、高压喷射注浆桩复合地基、砂桩地基、振冲桩复合地基、土和灰土挤密桩复合地基，水泥粉煤灰碎石桩复合地基及夯实水泥土桩复合地基，承载力检验，数量为总数的0.5%~1%，不少于3处，单桩强度检验数量为总数的0.5%~1%，不少3根。

3、 灰土地基有配合比、分层回填压实系数检测。

4、 砂石垫层有配合比、分层回填压实系数检测。

5、 石灰有检测试验报告，有石灰粒径、涂料有机质含量试验。

6、 砂石垫层、砂石有复试报告，有砂石料、有机质含量、含泥量、粒径等检测，水泥土搅拌桩承重取90d试件。

7、 粉煤灰地基，粉煤灰有复试报告，有粉煤灰粒径、氧化铝及二氧化硅含量、烧矢量等检测。

8、 灌注桩每浇筑50 m3必须有1组试件，小于50 m3的桩，每根桩必须有1组试件。

9、 静力压桩硫磺胶泥半成品应每100kg做一组试件（３件）。

10、成品桩有产品合格证。

11、回填土有配合比、分层回填压实系数检测、填筑厚度使用柴油打夯机分层厚度200~250mm。冬季回填厚度应比常温减少20%~50%。

12、每层回填土均应做隐蔽，并按每层分划检验批进行验收。

GB50203-2002 《砌体工程施工质量验收规范》

1、 水泥进行安定性、强度复检。水泥检验批规定：按水泥厂年生产能力规定。

l 120万t以上，以不超过1200t为一编号；

l 60万~120万t，以不超过1000t为一编号；

l 30万~60万t，以不超过600t为一编号；

l 10万~30万t，以不超过400t为一编号；

l 10万t以下，以不超过200t为一编号；

2、 M5以上混合砂浆，砂含泥量不应超过5%。

3、 砂浆中掺入有机塑化剂应有砌体型式检验报告。（塑化剂：如微沫剂，是松香、碱及加水熬制成，其润滑作用）

4、 砂浆试块不超过250 m3砌体的各类型及强度等级的砌筑砂浆，每台搅拌机至少抽检一次；在砂浆搅拌机口随机取样制作砂浆试块（同一盘砂浆只应制作一组试块）。

5、 蒸压（养）砖的产品龄期不应少于28d。

6、 砖抽样：烧结砖15万块、多孔砖5万块、灰砂砖及粉煤灰砖10万块、烧结空心砖强度试验，按3万块为一批，蒸压加气砼块同品种、同规格、同等级的砌块，以1万块为一批、轻骨料砼小砌块同密度、强度、质量同一生产工艺，以1万块为一批。

7、 砼小型空心砌块所用产品龄期不小于28d。

8、 同一产地的石材至少应抽检一组。

9、 用轻骨料砼小型空心砌块或蒸压加气砼砌块砌筑，墙底应砌烧结普通砖或多孔砖，或普通砼小型空心砌块，或普通砼小型空心砖砌块，或现浇砼坎台等，高度不易小于200mm。

10、空心砖、轻骨料砼小型空心砌体的砌体灰缝应为8~12mm。蒸压加气砼砌体的水平灰缝厚度及竖向灰缝宽度分别宜为1.5mm和20mm。抽查10%，且不应少于3间。

11、填充墙、梁、板底砌筑，应待填充墙砌筑完至少间隔7d后再补砌。

12、冬期施工砂浆试块留置，除按常温规定要求外，尚应增留不少于1组与砌体同条件养护的砌块，28d后检测强度。

13、当采用掺盐砂浆法施工时，宜将砂浆强度等级按常温施工的强度等级提高一级。

关于术语：

1、 型式检验

确认产品或过程应用结果适用性所进行的检验。

2、 原位检测

采用标准的检测方法，在现场砌体中选样进行非破损或为破损检测，以判定砌体砂浆（砼）和砌体实体（砼）强度的检测。

★ 砌筑砂浆的强度等级是以70.7×70.7×70.7（mm）立方体砂浆试块（一组为6块）拆模后，再继续标准养护至28d后的抗压强度来确定的，标准养护的条件是：①水泥混合砂浆应为温度20±30C，相对湿度60~80%；②水泥砂浆和微沫砂浆应为温度20±30C，相对湿度90%；③养护期间，试件彼此间隔不少于10mm。

6、 砼中使用水泥按同一生产厂家、统一等级、同一品种、同一批号且连续进场的水泥，袋装不超过200t为一批，散装不超过500t为一批进行复试。使用的水泥严禁是氯化物水泥。

7、 外加剂有出厂合格证、出厂检验报告和复试。

8、 含氯化物的外加剂应有检测，掺和料（粉煤灰）复试。

9、 粗细骨料进行复试，400m3（并提供质量证明书）。

10、宜采用饮用水，当采用其他水源时有水质试验报告。

11、砼配合比有开盘鉴定，开始生产时应至少留置一组标养试件。

12、砼拌制前，应测定砂、石含水率，每工作班检查一次。

13、砼强度试件，在砼浇筑地点随机抽取

l 每拌制100盘且不超过100 m3的同配合比的砼，不少于一次。

l 每工作班拌制的同一配合比的砼不足100盘时，不少于一次。

l 当一次连续浇注超过100 m3时，同一配合比的砼每200 m3取样不得少于一次。

l 每一楼层、同一配合比的砼，取样不少于一次。

l 每次取样至少留置一组标养试件，同条件按实际需要。

14、砼养护

l 浇筑完毕后12h内对砼加以覆盖并保温养护。

l 普通砼7d，抗渗砼14d以上。

15、砼同条件试件符合第10章及附录D规定。

16、结构实体检验

l 结构安全的重要部位进行；

l 内容：砼强度、钢筋保护层厚度；

l 以同条件养护的试件强度为依据检验砼强度，或采用非破损或局部破损的检测方法；

l 当同条件养护试件强度的检验结果符合GB107有关规定，砼强度判定合格；

l 对钢筋保护层厚度的检测，符合附录E规定；

l 当以上不合格，应委托相应资质等级检测机构进行检验。

17、质量不符合要求，应按下列规定进行处理。

l 经返工、返修或更换构件、部位的检验批，应重新进行验收。

l 经有资质的检测单位检测鉴定达到设计要求的检验批，应予以验收。

l 经有资质的检测单位检测鉴定达不到设计要求，但经原设计单位核算并确认仍可满足结构安全和使用功能的检验批，可予以验收。

l 经返修或加固处理能够满足结构安全使用要求的分项工程，可根据技术处理方案和协商文件进行验收。

l 通过返修或加固处理仍不能满足安全使用要求的，应不予以验收。

GB50207-2002 《屋面工程质量验收规范》

1、 确定屋面防水工程防水等级，共4级，一般高层建筑为二级。

2、 有屋面施工方案。

3、 专业防水队伍有资质，作业人员由当地建设行政主管部门颁发的上岗证。

4、 屋面工程用的防水、保温隔热材料应有产品合格证和性能检测报告，并进行复试，符合附录A附录B要求。

5、 屋面找平层材料有配合比报告，复试报告。

6、 防水层有雨后或淋水、蓄水检验，淋水持续2h后进行，蓄水不应少于24h。

7、 细石砼防水层材料要求：

l 水泥用量不得少于330ｋｇ／ｍ３，水灰比不应大于0.55，强度不应低于C20。

l 含砂率宜为35%~40%，含泥量不应大于2%。

l 粗骨料含泥量不应大于1%。

l 灰砂比宜为1：2~1：2.5。

8、 瓦屋面应做雨后或淋水试验。

9、 架空屋面隔热制品质量要求：

l 非上人屋面粘土砖不应低于Mu10。

l 砼板强度不低于C20，钢丝网片。

10、屋面工程隐蔽验收记录：

l 卷材、涂膜防水层的基层；

l 密封防水处理部位；

l 细部做法（天沟、檐沟、泛水、檐口、水落口、变形缝、伸出屋面管道、防水构造），并附图；

l 卷材、涂膜防水层的搭接宽度和附加层；

l 刚性保护层与卷材、涂膜防水层之间的隔离层。

GB50208-2002 《地下防水工程质量验收规范》

1、 确定地下工程防水等级，共4级。

2、 防水工程施工方案，有主体及细部构造的防水技术要求。

3、 专业防水队伍有资质，主要施工人员应持建设行政主管部门或其他指定单位颁发的职业资格证书。

4、 防水材料有产品合格证书和性能检测报告，并进行抽样复试。（附录Ａ、Ｂ）

5、 安全功能检测：地下室防水效果检查记录（附录C）。

6、 方水砼所用材料符合规定：

（1）、水泥强度不应低于32.5级；

（2）、碎石或卵石颗粒宜为5～40mm，含泥量不得大于1.0％，泥块含量不得大于1.0％；

（3）、砂宜用中砂，含泥量不得大于3.0%，泥块含量不得大于1.0%；

（4）、水不含有害物质的洁净水；

（5）、外加剂符合国家或行业标准一等品级以上的质量要求；

（6）、粉煤灰级别二级以上，掺量不宜大于20％；硅粉不大于３％。

7、配合比符合以下规定：

（1）、试配要求的抗渗水压应比设计值提高0.2Mpa；

（2）、水泥用量不得少于300ｋｇ／ｍ３；掺有活性掺和料时，水泥用量不得少于２80ｋｇ／ｍ３；

（3）、砂率宜为35％～45％，灰砂比宜为1：2～1：2.5；

（4）、水灰比不得大于0.55；

（5）、泵送时入泵坍落度宜为100~140mm。

8、拌制和浇注过程符合规定。

9、连续浇筑砼每500m3留置一组抗渗试件，且每项工程不得少于两组。

10、地下防水隐蔽工程验收记录应包括以下主要内容：

l 卷材、涂料防水层的基层；

l 防水砼结构和防水层被掩盖的部位；

l 变形缝、施工缝等防水构造的做法；

l 管道设备穿过防水层的分固部位（包括埋件：模板用螺栓）；

l 基坑的超挖和回填；

12、所用材料符合下列规定：

l 水泥强度不应低于32.5级；

l 砂采用中砂，粒径3mm以下，含泥量不得大于1%，硫化物和硫酸盐含量不得大于1%；

l 水无害洁净水；

l 聚合物乳液外观质量，无颗粒、异物和凝固物；

l 外加剂为一等品以上。

13、按每一层进行检验批验收和填写隐蔽记录。

14、水泥砂浆防水层施工需提供：外加剂、掺和料合格证、检验报告，复试报告，有配合比报告。

15、防水层保护层应进行隐蔽验收，应注意保护层与防水层之间应无空隙。

16、后浇带应在两侧砼龄期达到42d后再施工，应采用补偿收缩砼，强度不得低于两侧砼。

GB50209-2002 《建筑地面工程施工质量验收规范》

１、水泥砼和水泥砂浆试块组数，按每一层（或检验批）建筑地面工程不少于1组。建筑面积大于1000m2时，每增加1000 m2应增做1组试块；小于1000 m2按1000 m2计算，改变配合比时，亦应制作试块组数。

2、防水隔离层应作隐蔽验收，防水砂浆铺涂超过套管上口、靠墙，应高出面层200~300mm。

3、厕浴间四周做高度不小于120mm的砼翻边。

4、防水材料铺设后，必须蓄水检验，水深20~30mm，24h内无渗漏为合格。

5、

l 水泥砼面层强度等级不小于C20，水泥砼垫层兼面层强度等级不小于C15。

l 水泥砂浆面层强度不应小于M15。

l 水磨石面层拌和料有配合比单，强度检测报告，面层结合层水泥砂浆强度不少于M10，体积比宜1：3。

l 水泥钢（铁）屑面层抗压强度不小于４０Ｍpa，结合层体积比1：2，强度不小于M15，钢（铁）屑（1~5mm）有材质合格证及检测报告。

l 防油渗面层强度不小于C30，防油渗涂料抗拉粘结强度不应小于0.3Mpa。

6、

l 不发火（防爆的）面层，选用沙应质地坚硬，表面粗糙，粒径宜为0.15~5mm，含泥量不大于3%，有机物质不应大于0.5%，有砂检测报告。

l 面层分格嵌条为不发生火花材料，有材质合格证明文件，检测报告。

l 面层强度有配合比通知单，检测报告。

l 不发火（防爆的）面层试件在金刚砂轮上做摩擦试验，不少于50各试件选10个。

7、 板块面层，水泥类基层抗压强度不得少于1.2Mpa。

8、 板块有材质合格证明文件及检测报告，大理石、花岗岩石等天然石材有有关材料物质限量规定的检测报告。

9、 地面个构造层均由隐蔽验收及其他验收文件。

GB50210-2001 《建筑装饰装修工程质量验收规范》

1、 抹灰工程应对水泥凝结时间和安定性进行复试。

2、 门窗工程复试项目：

l 人造木板的甲醛含量；

l 外墙金属窗、塑料窗的抗风压性能、空气渗透性能和雨水渗漏性能。

3、 门窗工程隐蔽工程：

l 预埋件和锚固件；

l 隐蔽部位的防腐、填嵌处理。

4、 特种门（人防门）由生产许可证、产品合格证和性能检测报告。

5、 玻璃应有产品合格证书、性能检测报告。

6、 吊顶材料有产品合格证书、性能检测报告和复验报告，对人造木板有甲醛含量复试。

7、 吊顶工程隐蔽；

l 吊顶内管道、设备的安装及水管试压；

l 木龙骨防水、防腐处理；

l 预埋件或拉结筋（预埋件、吊杆、防锈处理）；

l 吊杆安装；

l 龙骨安装；

l 填充材料的设置。

8、 轻质隔墙材料产品有合格证书、性能检测报告、复试报告。

9、 轻质隔墙隐蔽项目：

l 设备管线的安装及水管试压；

l 木龙骨防火、防腐处理；

l 预埋件或拉结筋；

l 龙骨安装；

l 填充材料的设置。

10、饰面板（砖）有产品合格证书、性能检测报告、复试报告。

11、其后置埋件有现场拉拔检测报告。

12、外墙饰面砖样板件的粘结强度检测报告：

外墙饰面砖粘贴前和施工过程中，均应在相同基层上做样板件，并对样板件的饰面砖粘结强度进行检验，其检验方法和结果判定应符合《建筑工程饰面砖粘结强度检验标准》（JGJ110）。

条文说明：（8.1.7）在（JGJ126-2000）中6.0.6条第3款规定：“由于该方法为破坏性检验，破损饰面砖不易复原，且检验操作有一定难度，在实际验收中较少采用，故本条规定在外墙饰面砖粘贴前和施工过程中均应制作样板件并作粘贴强度试验。”

13、饰面板（砖）复验：

l 室内用花岗石的放射性；

l 粘贴用水泥的凝结时间、安定性和抗压强度；

l 外墙陶瓷面砖的吸水率；

l 寒冷地区外墙陶瓷面砖的抗冻性。

JGJ126-2000中规定外墙饰面砖用材料复试：

l 水泥、沙、胶粘剂等均应复试；

l 外墙饰面砖复检项目：a、尺寸 b、表面质量 c、吸水率（Ⅱ区≤6%） d、抗冻性。

14、饰面板（砖）隐蔽项目：

l 预埋件；

l 连接节点；

l 防水层。

15、幕墙工程应收集及复试项目见《幕墙工程》章与JGJ/T139-2001。

16、室内环境质量在竣工验收前应做检测。
