VBA简单入门09：End属性获取最后的单元格信息

2020-11-14
| 科技

1、End属性是单元格对象Range的属性，它返回工作表中（行列）的最后一个非空单元格，并常常用来构建动态的单元格数据范围。

如，工作表中只有A1:C13是数据区域。此时该工作表的最后一个非空单元格的相关信息可以用End属性来获取。

A列最后一个非空单元格地址：返回 : "$A$13"

a = Range("a1048576").End(xlUp).Address

Range("a1048576")表示单元格 A1048576,而End是属于Range单元格对象的属性。

可以说白一点：Range("a1048576").End(xlUp)就是以单元格 A1048576 为基点，从 【A1048575 开始】 (A1048576即使非空也不在查找范围内)在同一列（A列）往上查找，找到第一个非空单元格就停止，并返回该单元格。所以上面的代码，返回的是A13单元格。

如将代码改为：Range("a20").End(xlUp).Address，返回同样结果。

原因只不过是，一个从A1048575开始，一个从A19开始，虽然基点不同，但往上查找到的第一个非空单元格都是A13。

2、既然可以往上查找，那就同样可以往下、往左、往右查找。

在写代码的时候，就有提示：

xlup上； xldown下； xltoleft左； xltoright右

它们的相同点是，作为基点的单元格都不算。

在数据比较规范的时候，无论是xlup还是xldown，都能得出同样的结果。

假如，A16单元格非空，那么上图两句代码返回的结果将不一样。

xlup：向上找，找到第一个非空单元格就停止。

xldown：向下找，遇到空单元格就停止。

xltoleft和xltoright同理。

下列代码返回规范数据情况下（只A1:C13有数据）当前活动工作表的数据范围的相关信息。

假如，当前的数据表格需每天更新数据（添加行），该动态单元格范围一般可以这样表示：

Range("a1:c" & 数值变量) 或 Range("a1",单元格变量)，如下面代码都可以表示单元格A1:C13

行数 = Range("a1048576").End(xlUp).Row

Range("a1:c" & 行数).Select

或

Range("a1", Range("c1048576").End(xlUp)).Select

当数据自动增加的时候，单元格范围会自动增加。如新增3行数据。请看gif演示。

4、Excel版本问题造成的错误。

Excel03版只有65536行，Excel07版后有1048576行，所以Range("a1048576")在03版中会出错。

所以通用的写法是：Cells(Rows.Count, 列)或Cells(行, Columns.Count)来代替Rang对象。

下图中，Rows.Count在03版是65536，在07版或以上是1048576，再也不担心版本问题而出错。

【小练习】提取唯一值（去重复）

要求：将A列的英雄姓名去除重复值，提取唯一值到D列。

此题目可以用菜单栏的删除重复值操作(或字典）来实现更简单，但这里使用的是循环的方法。

Sub A()

Dim R As Long

Dim R1 As Long

Dim B As Boolean

Dim k As Long

R = Cells(Rows.Count, 1).End(xlUp).Row 'A列最后非空行行号

Range("d2:d10000") = "" '先将D列数据清空

[d2] = [a2] '第一个数据必然是唯一的，放入D2单元格

k = 2

For i = 3 To R '遍历英雄姓名(第2个数据起)

B = False '此值改变用来标记是否有重复

R1 = Cells(Rows.Count, 4).End(xlUp).Row 'D列最后非空行行号

'............................................................................................................................................

'//注意R1数值的变化，随着提取的唯一姓名个数增加而增加

For j = 2 To R1 'A列姓名每一个都遍历D列的所有英雄姓名，查看是否在D列存在（重复）

If Cells(i, 1) = Cells(j, 4) Then '如果A列姓名在D列能找到，表示重复

B = True '布尔变量为TRUE表示重复

Exit For '确定重复后退出循环

End If

Next

'.............................................................................................................................................

'//判断是否有重复，并输出数据

If B = False Then '判断B的值，如False，则是唯一值

k = k + 1 '计数

Cells(k, 4) = Cells(i, 1) '数据放到D列

End If

Next

End Sub
