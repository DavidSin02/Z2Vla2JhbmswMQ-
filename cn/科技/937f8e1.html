Python｜查找列表元素（线性查找法或二分查找法）

2020-10-29
| 科技

list类模块已经实现有index方法来查找并返回匹配列表中某个元素的下标，也支持成员运算符in和not in来判断一个元素是否在列表中。

lst = [2,9,5,4,8,1,6]

print(lst.index(8)) # output：4

print(8 not in lst) # output: False

当然，也可以自定义元素查找并返回元素下标，以下阐述的的是线性查找法和二分查找法。

1 线性查找法

线性查找法是指按顺序挨个将需要查找的key与列表中的每一个元素比较，直到匹配到逻辑相等为止。

# The function for finding a key in the list

def linearSearch(lst, key):

....for i in range(len(lst)):

........if key == lst[i]:

............return i

....return -1

lst = [2,9,5,4,8,1,6]

print(linearSearch(lst, 8)) # putput:4

2 二分查找法

二分查找法的思路要复杂一点。对一个已经排序的列表，不断将一个区间分为两个子区间（包括匹配子区间），将需要查找的关键字与区间的中间元素比较，直到匹配到逻辑相等为止。

用一个简单的实例手动模拟实际操作的过程：

lst = [2,4,7,10,11,45,50,59,60,66,69,70,79]

key = 11

将上面图解的过程用Python代码进行一般化实现：

附代码：

# Use binary search to find the key in the list

def binarySearch(lst, key):

....low = 0.................... # 列表中当前要查找的第一个下标

....high = len(lst) - 1........ # 列表中当前要查找的最后第一个下标

....while high >= low:

........mid = (low + high) // 2 # 中间元素下标

........if key < lst[mid]:.... # 满足条件,则定义前半段为子区间

............high = mid - 1

........elif key == lst[mid]: # key与中间元素相等,则返回中间元素的下标值

............return mid

........else: .... # 满足条件,则定义后半段为子区间

............low = mid + 1

....return low - 1 .... # Now high < low, key not found

lst = [2,4,7,10,11,45,50,59,60,66,69,70,79]

print(binarySearch(lst, 11))....# output: 4
