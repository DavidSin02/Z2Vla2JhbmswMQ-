2020烟花爆竹经营单位安全管理人员模拟考试及考试

2020-10-29
| 科技

题库来源：安全生产模拟考试一点通公众号小程序

2020烟花爆竹经营单位安全管理人员模拟考试及烟花爆竹经营单位安全管理人员证考试，包含烟花爆竹经营单位安全管理人员模拟考试答案解析及烟花爆竹经营单位安全管理人员证考试练习。由安全生产模拟考试一点通公众号结合国家烟花爆竹经营单位安全管理人员考试最新大纲及烟花爆竹经营单位安全管理人员考试真题出具，有助于烟花爆竹经营单位安全管理人员模拟考试软件考前练习。

1、【判断题】1.1级建筑物为建筑物内的危险品在制造、储存、运输中具有整体爆炸危险或有迸射危险。（ √ ）

2、【判断题】烟花爆竹类法规也分为行政法规和地方性法规这两种。（ √ ）

3、【判断题】危险场所电气线路的电线和电缆的额定电压不得低于300V/500V。（ × ）

4、【判断题】1.1级、1.3级厂房的门应采用向内开启的平开门。（ × ）

5、【判断题】有升空迸射危险的生产厂房与相邻厂房的门、窗不宜正对设置。（ √ ）

6、【判断题】使用皮带传动时,皮带(特别是平板皮带)与皮带轮摩擦产生静电。（ √ ）

7、【判断题】危险是指系统中存在导致发生不期望后果的可能性超过了人们的承受程度。（ √ ）

8、【判断题】本单位主要负责人具有组织制定并实施单位的生产、经营安全事故应急救援预案的职责。（ √ ）

9、【判断题】推行安全管理现代化不仅是社会主义建设的需要,也是企业管理改革的重要组成部分。（ √ ）

10、【判断题】轻质易碎屋盖是当建筑物内部发生事故时,不仅具有泄压效能,且破碎成小块,减轻对外部影响的屋盖。（ √ ）

11、【判断题】一起伤亡事故的发生往往是两类危险源共同作用的结果。（ √ ）

12、【判断题】库房(仓库)内半成品与未成箱成品堆垛的高度不应超过1.0m。（ × ）

13、【判断题】任何单位或者个人对事故隐患或者安全生产违法行为,均有权向负有安全生产监督管理职责的部门报告或者举报。（ √ ）

14、【判断题】1.3级建筑物应设置防护屏障。（ × ）

15、【判断题】烟花爆竹生产应按产品类型设置生产线。（ √ ）

16、【判断题】防护土堤的顶宽不应大于1.0m。（ × ）

17、【判断题】防护土堤的底宽应根据不同土质材料确定,但不应小于防护土堤高度的1.5倍。（ √ ）

18、【判断题】办公室、配电间属于生活辅助用室。（ × ）

19、【判断题】企业要经常性开展安全隐患排查,并切实做到整改措施、责任、资金、时限“四到位”。（ × ）

20、【判断题】含爆炸药、笛音药的半成品,应采用筑(压)方法封口。（ × ）

21、【判断题】事故险情和支援请求的报告必须严格按照分级响应的原则逐级上报。（ × ）

22、【判断题】危险品运输通廊的承重及围护结构宜采用不燃烧体。（ √ ）

23、【判断题】有易燃易爆粉尘散落的工作间宜用笤帚清扫。（ × ）

24、【单选题】县级以上地方各级人民政府应当根据本行政区域内的安全生产状况,组织有关部门按照( )原则,对本行政区域内容易发生重大生产安全事故的生产经营单位进行严格检查;发现事故隐患,应当及时处理。（ C ）

A、职权划分

B、出现划分

C、职责分工

25、【单选题】通常用于烟火药的安定剂的是( )。（ B ）

A、石蜡

B、碳酸氢钠

C、硬脂酸

26、【单选题】负责烟花爆炸生产企业安全生产许可证的颁发管理的部门是( )。（ A ）

A、省安全生产监督管理部门

B、市安全生产监督管理部门

C、省公安厅

27、【单选题】烟花爆竹工厂1.3级厂房的人均使用面积最小为( )m²。（ C ）

A、3.5

B、4.0

C、4.5

28、【单选题】烟花爆竹工厂危险品总仓库区内,1.1级成品仓库单库存药量最大为( )kg。（ B ）

A、20000

B、10000

C、15000

29、【单选题】手工模具压药,每栋工房定员( )人。（ C ）

A、3

B、2

C、1

30、【单选题】烟花爆竹工厂中易燃易爆粉尘较少的工作间内墙面应刷油漆墙裙的高度为( )。（ B ）

A、1.5m以下

B、1.5至2.0m

C、2.0m以上

31、【单选题】对重大危险源的生产经营单位,要配备( )和必要的应急救援器材和设备。（ A ）

A、应急救援人员

B、应急救援器材

C、消防器材

32、【单选题】职工发生事故伤害或者按照职业病防治法规定被诊断、鉴定为职业病,所在单位应当自事故伤害发生之日或者被诊断、鉴定为职业病之日起( )日内,向统筹地区社会保险行政部门提出工伤认定申请。（ B ）

A、15

B、30

C、60

33、【单选题】作业环境中主要危险、有害因素为生产性粉尘危害等,不是控制和防止粉尘的具体方法是( )。（ C ）

A、粉碎、筛选时,尽可能密封,防止扬尘

B、筛选、配药、混合药剂落下距离尽量小

C、环境的湿度保持相对干燥

34、【单选题】职业病危害事故处理工作应当按照有关规定结案的时间为小于( )天。（ C ）

A、30

B、60

C、90

35、【单选题】烟花爆竹工厂中为F0类危险场所(黑火药除外)服务的排风室危险场所的类型为( )类。（ B ）

A、F0

B、F1

C、F2

36、【单选题】生产经营单位的主要负责人未履行《安全生产法》规定的安全生产管理职责，导致发生生产安全事故，尚不够刑事处罚的，对生产经营单位主要负责人给予撤职处分，并在( )年内不得担任任何生产经营单位的主要负责人。（ B ）

A、3

B、5

C、7

37、【单选题】烟花爆竹工厂危险品总仓库区内,1.1级成品仓库单库存药量最大为( )kg。（ C ）

A、30000

B、25000

C、20000

38、【单选题】安全生产管理基本概念要求最大限度地减少劳动者的工伤和( )。（ A ）

A、职业病

B、意外伤害

C、死亡

39、【单选题】阻止空气进入燃烧区或用惰性气体稀释空气,使燃烧因得不到足够氧气而熄灭的灭火方法叫( )。（ A ）

A、窒息灭火法

B、冷却灭火法

C、隔离灭火法

40、【单选题】对怀孕( )个月以上的女职工,不得安排其延长工作时间和夜班劳动。（ B ）

A、6

B、7

C、8

41、【单选题】生产经营单位的主要负责人,对本单位的( )安全生产工作负责。（ C ）

A、主要

B、全部

C、全面

42、【单选题】烟花爆竹工厂运输通道的防护土堤端部需设挡土墙时,其结构宜为( )。（ B ）

A、砖头结构

B、钢筋混凝土结构

C、夯土结构

43、【单选题】某公司23号火箭安装车间有20名职工在进行火箭安装操作，其中有一道工序是将用薄膜袋装好的炸药和亮珠装入纸筒内，然后盖上一个纸巴巴，盖纸巴巴要用木棒顶进纸筒内，而用薄膜袋装炸药和亮珠，其危险性与裸露药物是一样的。在盖纸巴巴的过程中，职工刘某因用力过大、操作不当，致使薄膜袋内药物产生摩擦产生爆炸，从而引发车间内火箭半成品燃爆、乱射，职工所领药包发生爆炸。事故造成两栋1.3级组装车间炸毁，4人重伤，10人轻伤。擅自改变工房危险等级是( )。（ B ）

A、允许的

B、禁止的

C、视情况而定

44、【单选题】下列灭火剂中属于液体灭火剂的是( )。（ B ）

A、泡沫灭火剂

B、水

C、二氧化碳

45、【单选题】静电非导体体电阻率值大于或等于( )Ω·m。（ C ）

A、1.0×10¹²

B、1.0×10¹¹

C、1.0×10¹⁰

46、【单选题】生产经营活动中发生的造成人身伤亡或者直接经济损失的事件被定义为( )。（ C ）

A、生产事故

B、生产安全事故

C、安全生产事故

47、【单选题】烟花爆竹生产企业的领导(主要负责人)对安全生产具有( )。（ B ）

A、行政责任

B、领导责任

C、法律责任

48、【单选题】下列选项中( )应当按照保障安全生产的要求,依法及时制定有关的国家标准或者行业标准。（ B ）

A、劳动部门

B、国务院有关部门

C、国家相关安全监督管理机构

49、【单选题】烟花爆竹工厂水暖干燥时,烘房温度最高为( )℃。（ A ）

A、60

B、70

C、80

50、【单选题】《职业病防治法》是为了( )和消除职业病危害,防治职业病,保护劳动者健康及其相关权益,促进经济发展,根据宪法而制定。（ C ）

A、预防、遏制

B、预防、减少

C、预防、控制
