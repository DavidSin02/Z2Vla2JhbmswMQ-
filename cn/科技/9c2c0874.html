分支嵌套结构

计数循环

循环某一个范围，例如从 0 循环到 10。

其中: 起始数字可省略，默认值为 0。

步长可省略，默认值为 1。 另外，循环范围是起始数字~(结束数字-1)，即不包含结束数字。

for x in range([起始数字],结束数字,[步长]):    <循环内容>

列表循环

在循环列表时，in 后面是装有列表的变量名称。

循环次数等于列表的长度，x 是每次循环时列表对应位置的值。

books = [‘西游记’,’水浒传’,’红楼梦’,’三国演义’]for x in books:    print(x)

文字的循环

在循环文字时，in 后面是文字的变量。 循环次数等于文字的字符数，x 是每次循环时对应位置的字符。

for x in ‘我爱中国’:    print(x)

while 循环

当判断条件满足时才会执行。

while <判断条件>:    <循环内容>

break 和 continue

break:停止循环

continue:跳过本次循环

循环结构的嵌套

多个循环之间可以相互嵌套，从而形成更加复杂的逻辑结构。

如下为“99 乘法表”的实现程序:

try:<可能出现异常的语句>except ZeroDivisionError:<出现除 0 错误后才会执行的语句>except SyntaxError:<出现语法错误后才会执行的语句>...except :<出现异常后，但不是上述异常类别时才会执行的语句>for x in range(1,10):    for y in range(1,x+1):        print(“x”,”*”,y,”=”,x*y, end=””)    print()

try...except...

此语句进行异常处理，即出现异常时 except 语句块将会执行。

try:    <可能出现异常的语句>except ZeroDivisionError:    <出现除 0 错误后才会执行的语句>except SyntaxError:    <出现语法错误后才会执行的语句>...except :    <出现异常后，但不是上述异常类别时才会执行的语句>

try...except...else...

此语句包含 else 语句块，即没有出现异常的时候将会执行。

try:    <可能出现异常的语句>except:    <出现异常后执行的语句>else:    <没出现异常时执行的语句>

try...except...finally

此语句块包含 finally，为无论是否出现异常均会执行的语句块。

try:    <可能出现异常的语句>except:    <出现异常后执行的语句>finally:    <无论是否出现异常，均为执行的语句>

Turtle 绘图

Turtle(海龟)绘图为入门库，用于进行简单的编程绘图，需要掌握的知识点如下:

# 导入海龟import turtle# 设置背景色 turtle.bgcolor('black') # 设置窗体大小 turtle.setup(800,600) # 获取海龟的笔pen = turtle.Pen()# 更改笔的粗细 pen.width(10)# 更改笔的颜色 pen.color('red')# 更改绘画速度 pen.speed(10)# 绘制直线 pen.forward(100)# 左/右转弯 pen.left(90) pen.right(75)# 抬笔/落笔 pen.up() pen.down()# 设置笔的位置 pen.setposition(0,0) # 设置笔的方向 pen.setheading(90) # 设置填充颜色 pen.fill_color('pink') # 开始填充 pen.begin_fill()for x in range(4):     pen.forward(60)    pen.right(90) #结束填充pen.end_fill()# 注册鼠标点击事件 def click(x,y):    print('你点击的位置是:',x,y) turtle.onscreenclick(click, btn=1) # 注册键盘监听事件def key():    print('你按下 a 键了') turtle.onkeypress(key, ‘a’) turtle.listen()# 隐藏笔的形状 pen.hideturtle() #绘图完成turtle.done()
