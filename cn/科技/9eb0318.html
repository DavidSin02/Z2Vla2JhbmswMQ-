甘草提取物

光果甘草根提取物是从豆科植物光果甘草（学名：Glycyrrhiza Glabra 英文名：Lycoris）的根中提取得到的提取物。

光果甘草根提取物由于是天然成分，其成分组成会随着国家、地域和时期的不同发生变化，主要由以下成分构成：

甘草亭酸

光甘草定

甘草苷

异甘草苷

甘草西定

甘草根糖苷

三萜系化合物

类黄酮

甘草，物如其名，甜度是砂糖的50倍，自古以来用于传统医学。

希腊自然科学家泰奥弗拉斯托斯曾将甘草根用于治疗哮喘、干咳和肺病，中医也将甘草用于治疗咽痛、哮喘、胃溃疡和十二指肠溃疡，在药方中起协调作用。

在药理作用方面，具有消炎、祛痰、缓和粘膜刺激的作用，近年来被发现具有抗病毒作用和免疫活化作用。

甘草的作用中心是三萜皂甙的甘草甜素，协同其他成分提高生物活性。

添加到化妆品中时，因其具有非常优秀的抗炎症作用，还有缓和刺激、促进创伤愈合、抗过敏、抗菌、抑制色素沉著作用，经常添加到皮肤护理化妆品、头发护理产品、身体护理产品、手部护理产品、防晒产品、洁面产品、面膜等产品中使用。

光果甘草根提取物的色素沉着抑制作用机制是阻碍酪氨酸酶活性，首先我们先了解一下色素沉着（黑色素黑化）的方式，如下图。

经紫外线照射后的皮肤，首先紫外线信息到达基底层的黑素细胞，紫外线信息一到达黑素细胞，酪氨酸酶活性化，与黑素细胞内的酪氨酸结合后变化成多巴，然后活性化后的多巴与酪氨酸酶结合后变化成多巴醌，进而发展为黑化黑色素。

关于光果甘草根提取物的酪氨酸酶活性阻碍作用，1997年通过使用蘑菇酪氨酸酶后光果甘草根提取物的量比较检测酪氨酸酶的抑制率，含量分别为0μg/mL、3.3μg/mL、333μg/mL的光果甘草根提取物使用蘑菇酪氨酸酶后验证酪氨酸酶活性阻碍作用，3.3μg/mL和333μg/mL的阻碍率分别是25%和68%。

从结果看来，光果甘草根提取物与其他99种植物提取物相比，酪氨酸酶活性阻碍作用更强。


如前文报告所说（文献1:2008），光果甘草根提取物是通过较高的酪氨酸酶活性阻碍作用来抑制色素沉着。

另外，光果甘草根提取物的美白作用是通过光甘草定进行的。

实际上使用光果甘草根提取物的产品种类、数量和添加量，请看以下2007-2008年海外调查结果报告。

光果甘草根提取物的安全性

（毒性 刺激性 过敏性）

目前就光果甘草根提取物的安全性来说，基本上没有皮肤刺激性，有非刺激性和引起极其轻微的眼睛刺激性的可能，没有出现皮肤过敏的报告，所以安全性上没有问题。但是，有时会主观性地感受到轻度刺激（刺痛感）和燃烧感，而且有极小引起接触性皮炎的可能性，所以还是需要注意一下。

此项结论依据如下。

皮肤刺激性和皮肤过敏性

《Cosmetic Ingredient Review》的《Safety Assessment of Glycyrrhiza Glabra (Licorice) Rhizome/root, Glycyrrhiza Glabra (Licorice) Leaf Extract, Glycyrrhiza Glabra (Licorice) Root, Glycyrrhiza Glabra (Licorice) Root Extract, Glycyrrhiza Glabra (Licorice) Root Juice, Glycyrrhiza Glabra (Licorice) Root Powder, Glycyrrhiza Glabra (Licorice) Root Water, Glycyrrhiza Inflata Root Extract, and Glycyrrhiza Uralensis (Licorice) Root Extract》资料中记载了如下内容。

[人体试验]

在9名受试者的背部涂抹0.02mL含光果甘草根提取物（～900ppm）的实验物质，进行48小时封闭斑贴试验，去除贴片30分钟后对皮肤反应进行评分，观察到3名受试者出现极其轻微的红斑。

[人体试验]

在106名日本人受试者的健康皮肤上用光果甘草根提取物含量为0.0001%的16种产品进行反复伤害斑贴实验（HRIPT），诱导阶段进行48~72小时共计9次封闭斑贴试验，经过12~24天未处置时间后，在未处理部位进行48小时激发斑贴实验，48小时和96小时后对皮肤反应进行评分，结果有1人对1项产品出现接触性过敏症状的征兆，除此之外其他受试者没有出现任何刺激和过敏征兆。出现接触性过敏症状征兆的实验者进行第二次试验时，再次出现延迟过敏的症状。

[人体试验]

自我感觉为敏感肌的30名日本女性受试者和自我感觉为敏感肌的28名白种人、越南、韩国、中国等各色人种的受试者对含有0.0001%光果甘草根提取物的3种产品进行安全性实验，将其中1种产品从第1天到第14天作为夜间一般护理涂抹在整张脸上，从第15天到第28天各色人种受试者使用第2种产品，日本人受试者使用第3种产品，第14天和第28天进行临床性皮肤评分，结果客观上和主观上皮肤刺激分数都没有增加。

[人体试验]

68名日本女性受试者对含有0.0001%光果甘草根提取物的3种产品进行安全性实验，第1天到第4天在受试者单个眼部区域使用第一种产品，1天2次，另一眼部区域使用第二种产品，1天1次。第5天到第18天，受试者全脸涂抹第一种产品，1天2次。第19天到第32天，受试者在早上全脸涂抹第一种产品，傍晚全脸涂抹第三种产品。第2、3、4、18、32天时进行临床性皮肤评分，眼部区域经眼科检查后观察到临床性、非常轻微的眼睑充血，68人中有28名受试者（41%）在两周内使用其中一种产品后出现轻微的刺痛和燃烧感。再次涂抹新增加的产品时，5名（7%）受试者出现轻度刺痛和燃烧感。通常使用其中一种产品时，脸部出现轻度的主观感觉可能性较高，敏感皮肤和患有过敏性皮炎的皮肤可能会出现中度~重度的刺激感，客观上没有出现炎症。

[人体试验]

36名女性受试者（敏感肌15人、色素沉着过多14人、雀斑7人）在脸部色素区域涂抹光果甘草根提取物含量为0.8%的淡斑制剂，第4、8、12周进行临床性评分，结果在观察期间均未观察到出现红斑和干燥，客观上和主观上都没有增加刺激性。

[人体试验]

104名无皮肤疾病的健康受试者在诱导阶段使用20μL光果甘草根提取物含量为0.0001%的产品进行斑贴试验，每周3次，连续使用2周，去除各个贴片后直到进行下一个斑贴实验期间对实验部位进行评分。经过1~15的休息期间后，在实验部位进行48小时激发斑贴试验，去除贴片24小时后对皮肤反应进行评分，结果没有出现刺激和过敏的征兆。

[人体试验]

106名受试者在诱导阶段使用含有0.8%的光果甘草根提取物和其他5种植物提取物的载体溶液进行封闭Finn Chamber试验，每周3次，连续使用3周，经过10~15天休息时间后进行激发斑贴实验，24小时和48小时后观察结果，没有出现刺激和过敏的征兆。


仅从实验数据看来，光果甘草根提取物对健康的皮肤和敏感肌基本上都没有皮肤刺激性和皮肤过敏性，一般情况下基本上没有皮肤刺激性和皮肤过敏性。

但是，主观上有时可能会感觉到轻度的刺激感和燃烧感，也有极小引起接触性过敏皮炎的可能性，需要注意一下。

眼部刺激性

《Cosmetic Ingredient Review》的《Safety Assessment of Glycyrrhiza Glabra (Licorice) Rhizome/root, Glycyrrhiza Glabra (Licorice) Leaf Extract, Glycyrrhiza Glabra (Licorice) Root, Glycyrrhiza Glabra (Licorice) Root Extract, Glycyrrhiza Glabra (Licorice) Root Juice, Glycyrrhiza Glabra (Licorice) Root Powder, Glycyrrhiza Glabra (Licorice) Root Water,Glycyrrhiza Inflata Root Extract, and Glycyrrhiza Uralensis (Licorice) Root Extract》资料中记载了如下内容。

[动物试验]

在6只兔子单眼内滴入0.1mL光果甘草根提取物含量为0.15%的玉米油，不洗眼，24小时、48小时和72小时时进行观察，实验物质对兔子眼睛来说没有刺激性，所以该物质不是眼睛刺激剂。

[动物试验]

在3只兔子单眼的结膜囊内滴入0.1mL光果甘草根提取物含量为1%的甘油，第3天没有出现影响，直至观察到第7天，第1天少量泪液流出后又恢复正常，1小时内观察到轻微的眼睛肿胀，直到第2天才消除，所有的兔子在1一小时内都观察到轻度角膜混浊，直到第1天才完全消除，所以该试验物质有轻微的眼睛刺激性。

[in vitro试验]

用培养的正常人体角质细胞中生成的角膜样本，将100μL含量为10%的光果甘草根提取物进行不同时间的暴露，Draize分数都为0，所以光果甘草根提取物没有刺激性。


仅从实验数据看来，光果甘草根提取物为非刺激性和具有轻微眼睛刺激性，所以眼睛刺激性方面为非刺激性，有引起轻微眼睛刺激性的可能。

关于化妆品毒性判定百科的毒性判定


就已翻阅的化妆品毒性判断百科辞典看来，光果甘草根提取物属于△(∗1)类，安全性方面没有问题。

∗1 毒性判断百科辞典将毒性水平分为“无毒”、“△”、“■”、“■■”，2~3个“△”等于1个“■”，“■”是毒性很强的标识，产品的毒性成分“■”累计四个以上，则表示不可使用。

光果甘草根提取物也被分类在抗炎症成分、美白成分、抗菌成分中。
