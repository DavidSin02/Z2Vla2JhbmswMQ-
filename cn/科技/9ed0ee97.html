逻辑函数中IF函数判断是与不是，NOT函数对参数值求反

2020-11-14
| 科技

今日和大家继续分享逻辑函数中的IF和NOT，这两个函数的应用非常广泛，很多接触EXCEL函数的朋友最初都是从IF函数开始的，这个函数在EXCEL中的地位是永远撼不动的。下面给大家详细地讲解。这是逻辑函数的第三讲。

一．IF 判断是与不是。

函数定义：IF函数一般是指Excel中的IF函数，根据指定的条件来判断其"真"（TRUE）、"假"（FALSE），根据逻辑计算的真假值，从而返回相应的内容。可以使用函数 IF 对数值和公式进行条件检测。

使用格式：

IF(logical_test,value_if_true,value_if_false)

格式简义：IF(条件,与条件一样时运算这个,与条件不同时运算这个)

参数定义：logical_test条件必需，计算结果可能为 TRUE 或 FALSE 的任意值或表达式

value_if_true可选。logical_test参数的计算结果为TRUE时所要返回的值。

value_if_false可选。logical_test参数的计算结果为FALSE时所要返回的值。

注意点：

a).IF 函数语法具有下列参数(参数:为操作、事件、方法、属性、函数或过程提供信息的值.) b).2003版最多嵌套7个IF函数.2007版最多嵌套64个IF函数。

c)在多条件情况时，可以多层次套用IF函数，即 IF（条件1，条件1为真的结果，（IF（条件2，条件2为真的结果），...条件n为真的结果，条件n为假的结果）））注意左括号的数量要和右括号的数量一致，否则无法计算。

二．NOT 对参数值求反

函数定义：对参数值求反.当要确保一个值不等于某一特定值时,可以使用NOT函数

使用格式：NOT(logical)

格式简义：NOT(要判断的单元格)

参数定义：Logical为一个可以计算出TRUE或FALSE的逻辑值或逻辑表达式.

注意事项：

a)如果逻辑值为FALSE,函数NOT返回TRUE；

b)如果逻辑值为TRUE,函数NOT返回FALSE.

c)NOT函数是用于对参数值求反的一种Excel函数。当要确保一个值不等于某一特定值时，可以使用NOT函数。简言之，就是当参数值为TRUE时，NOT函数返回的结果恰与之相反，结果为FALSE。比如NOT(2+2=4)，由于2+2的结果的确为4，该参数结果为TRUE，由于是NOT函数，因此返回函数结果与之相反，为FALSE.

c) NOT函数的函数结构为：NOT(logical)，参数Logical 为一个可以计算出 TRUE 或 FALSE 结论的逻辑值或者逻辑表达式。如果A1=10、A2=15，那么公式"=NOT(A1<A2)"的计算结果为FALSE。其使用方法也与AND函数基本相同.

三 实例

如果产品名称为洗衣机,且颜色为白色，其调价后为原来单击加50元,如果产品名称为洗衣机，且颜色为白色以外的其余色，其调价后为原来单击加200元,其余产品价格不变

在下面的公式中

=IF(NOT(LEFT(A5,3)="洗衣机"),C5,IF(AND(LEFT(A5,3)="洗衣机",NOT(B5="白色")),C5+200,C5+50))

用到了IF函数，NOT函数和AND函数以及IF函数的嵌套。来实现①如果产品名称为洗衣机,且颜色为白色，其调价后为原来单击加50元,②如果产品名称为洗衣机，且颜色为白色以外的其余色，其调价后为原来单击加200元,③其余产品价格不变。上面共讲了三个条件：

①如果产品名称为洗衣机,且颜色为白色，其调价后为原来单击加50元,

②如果产品名称为洗衣机，且颜色为白色以外的其余色，其调价后为原来单击加200元,

IF(AND(LEFT(A5,3)="洗衣机",NOT(B5="白色")),C5+200,C5+50)其中判断条件是AND(LEFT(A5,3)="洗衣机",NOT(B5="白色")) 是一个AND函数，当为真时C5+200；不为真时C5+50，

③其余产品价格不变IF(NOT(LEFT(A5,3)="洗衣机")，C5)条件NOT(LEFT(A5,3)="洗衣机")为真时，C5成立。

好了上面的公式已经非常详细的讲解了。
