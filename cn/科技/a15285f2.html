变压器铁芯的优点及影响质量的因素有哪些？

2020-11-14
| 科技

影响变压器铁芯质量的因素:

① 设计 ② 原材料：牌号、厚度不均、绝缘膜、波浪度。 ③ 剪切质量：毛刺、片宽、片长、角度。 ④ 叠装质量：接缝、器身垫脚高度、心柱级支撑块、夹紧、扎紧、心柱垂直度、地面水平度、磕碰摔打、片弯曲半径。

主要设备工具：纵剪线、横剪线、剪床、冲床、自动叠装机、料料叠装台、千斤顶、定位棒盘、扳手、夹具、收紧带电木板水平尺吸尘器、外卡钳、盘升降架、 C 型夹具、液压机、液压缸等。

铁心：铁心担负磁路重任，也是变压器的骨架，产品各部件固定（绕组、绝缘件、引线、开关等）。

夹件：（对夹件制造的要求） ① 具有足够的强度 ② 焊线强度 ③ 加工精度 ④ 焊接处不可形成“袋状” ⑤ 月支板之间必须垂直 ⑥ 压钉螺母焊后的中心线垂直肢板

⑦ 高低压下夹件肢板必须在同一平面上，尤其是 A 、 C 端部 ⑧ 压钉螺母要用三锥过一次 ⑨ 连接部位不涂漆，保证接触良好 ⑩ 夹件倒角

铁心选片的优点： ① 提高叠片工序的效率 ② 检查废片提前补充不足片，保证每级的厚度和层数

铁心叠装工艺要点准备：夹件、夹件绝缘、枕木、横梁、拉板、压钉、垫脚绝缘等。 ( 1 ) ．放置叠片：放在液压升降台上 ( 2 ) ．布台：找平，使铁心中心对准台中心 ( 3 ) ．放千斤顶位置，留拉带、夹紧空间 ( 4 ) ．将千斤顶固定在台上 ( 5 ) ．垫脚与短臂距离

铁心必须一点接地铁心与金属件必须相连接接地，使同一电位（零电位），铁心两点接地相当于铁心两端短路，会产生一定电流，导致损耗增加，严重时会熔断接地片，铁心悬浮绝不允许。
