关于华为EMUI 10.0和Magic UI 3.0软件：完整更改日志和已知问题

2020-10-29
| 科技


华为发布EMUI 10.0和Magic UI 3.0软件

EMUI 10.0和Magic UI 3.0 beta程序已经在多款华为和荣耀智能手机上启动，并且华为荣耀手机的顶级产品阵容正在获得稳定的EMUI 10.0和Magic UI 3.0系统软件更新。既然我们已经多次讨论了EMUI 10.0和Magic UI 3.0软件最新升级所带来的功能和更改，那么现在该向您展示完整的更改日志以及EMUI 10.0和Magic UI 3.0所附带的已知问题了。

EMUI 10.0 / Magic UI 3.0的已知问题

有些问题可能会影响您的日常使用。如果您经常使用相关的应用或功能，请不要安装此更新。


华为EMUI 10.0和Magic UI 3.0软件

1.某些银行应用程序（banking app）与Android 10操作系统不兼容。只有发布了兼容的应用程序版本后，才能解决此问题。

2.一些即时消息应用程序与Android 10操作系统不兼容，并且在发送图像时可能会遇到问题。需要重新启动应用程序以解决此问题。

3.如果在安装此更新之前为即时消息应用程序禁用了声音和振动警报，则默认情况下将为来自即时消息应用程序的通知启用静音模式。可以根据您的喜好在应用程序设置中修改警报模式。

4.对于启用了“ App twin”功能的应用程序，此更新可能允许twin应用程序读取“消息传递”内容。禁用并重新启用App twin功能以解决此问题。

5.在播放PlayerUnknown的Battlegrounds（PUBG）时，在某些情况下屏幕可能会闪烁。设置非HDR分辨率或重新启动游戏即可解决此问题。

6.一段时间后，三星手表可能会自动从您的手机断开连接。此问题将在更高版本中解决。

7.用两根手指捏住30倍变焦有时可能会失败。拖动缩放滑块或重新启动相机即可解决此问题。

EMUI 10.0 / Magic UI 3.0更改日志

EMUI 10.0 –此版本旨在让您提前体验新的软件功能，从而为我们提供进一步改进软件反馈意见的机会。以下是此版本的已知问题，可能会影响您的日常使用。如果您经常使用相关的应用程序或功能，请不要安装此更新。

Magic UI 3.0 –旨在进一步简化和增强用户体验的质量。它为用户提供了全新的，经过微调的用户界面，旨在促进更流畅，更直观的用户体验，同时包括其增强的数据和隐私保护还带给您更安全的体验。

用户界面：复杂而令人叹为观止：

–杂志设计：

结合了杂志式的布局，以提供更舒适和真实的阅读体验。

–莫兰迪（Morandi）颜色：

低调的灰色调使显示屏更柔和，更舒适。

–动画：

动态自然的动画，用于页面过渡，应用程序启动和其他情况。

性能：平稳；


华为荣耀手机

– GPU Turbo：

改善游戏玩法时的流畅性和稳定性。

–手机克隆：

提高连接和克隆速度。

安全性：更安全的数据和隐私安全性：

–正式验证的微内核

使用华为自己开发的微内核设计，该设计被认证为最安全的商用OS内核。

软件更新安全性：更安全的数据和隐私经过正式验证的微内核使用华为自己开发的微内核设计，该设计被认证为最安全的商业OS内核。

–分布式技术：

提高与其他华为设备的连接以及在其他华为设备之间传输文件和存储数据时的安全性

- 隐私保护

支持应用沙箱功能，以实现标准化的应用权限管理以及更安全的数据和隐私保护。

媒体和娱乐：更有趣

– AppAssistant dropzone

为AppAssistant添加一个放置区，使您可以快速访问服务，例如游戏加速和请勿打扰。

安全：

带来安全性增强。

更新说明：

1.此更新不会删除您的个人数据，但是我们建议您在更新之前备份所有重要数据。

2.更新后，某些与Android 10操作系统不兼容的第三方应用程序可能无法正常运行。建议您将这些应用更新为应用商店中的最新版本

3.如果在更新过程中遇到任何问题，请拨打华为客户服务热线或访问华为客户服务中心寻求帮助。

4.您的主屏幕布局可能会稍微重新排列。

5.此系统版本可以回滚到HiSuite中的EMUI 9.1的正式版本。
