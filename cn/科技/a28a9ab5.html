二、关键对象/方法的分析

经过了 十 期教程，大家对 VBA 估计有了初步的认识。从这期开始，第二部分改为关键对象/方法的分析，会对一些比较常用，但又不容易掌握的对象/方法进行汇总和讲解。

1. Dictionary

一种比较常用的对象，储存数据的方式为：关键字(称为 Key) - 条目(称为 Item)。

属于 Dictionary 常用的属性、方法包括：

  • Add，添加一对相对应的关键字和条目到 Dictionary 对象。
  • Exits，如果在 Dictionary 对象中指定的关键字存在，返回 True，若不存在，返回 False。
  • RemoveAll，删除所有关键字和条目对。
  • Count，返回 Dictionary 对象中的条目数。
  • Keys，返回一个数组，该数组包含一个 Dictionary 对象中的全部已有的关键字。
  • Key，重新设置一个 Key 的 Key 值。

2. Dir

返回文件名、目录名或文件夹名称。

常见的用法举例：

  • Dir "C:\WINDOWS\WIN.ini" ——当 文件存在时，返回 "win.ini"，否则，返回""。
  • Dir "C:\WINDOWS\*.ini" ——返回第一个 ini 文件
  • Dir ——返回同样查询条件时的下一个文件，如果没有，返回 ""。
  • Dir "c:\", vbDirectory ——以字母为顺序，返回第一个 C 盘下面的第一个文件或目录
  • Dir "c:\" ——以字母为顺序，返回第一个 C 盘下面的第一个文件

3. For Each

  • 用于数组或者集合，针对一个数组或集合中的每个元素，重复执行一组语句。
  • 如果用于数组，数组元素必须是Variant 变量。
  • 更常用的用法是用于 VBA 定义的一些集合，比如 shapes，Worksheets，Workbooks，Range等。
  • 可以用 Exit For，强制退出循环。

4. Application

代表整个 Microsoft Excel 应用程序，非专业人员，不可能把 Application 的全部方法和属性都掌握。日常运用，只需要知道几张常用的方法/属性就可以：

  • Application.WorksheetFunction，调用 Excel 的公式，这里的 Application 或者 WorksheetFunction 可以省略其中一个，比如用 WorksheetFunction.Match 或者 Application.Match 都是可以的，但不能直接x写 Match。
  • Application.InputBox，弹出窗口输入，这里的 Application 也可以省略，直接写 InputBox 也可以。
  • Application.Transpose，行列转置，这里的 Application 不能省略，否则会出错。

三、执行思路及过程分析

【1】获取所有子目录

第一个 If 函数，对 A1 路径格式进行修正，以 \ 结尾。

接着创建一个字典 d，并且把 A1 的路径作为 Key 写入字典。后面对 d 仅用到它的 Key，所以，Item 取 “”就可以。

Zi，是否进行子目录查找的标志，取 1 表示查询子目录。

外层的 Do While 的循环，是对目标目录以及所有子文件夹的循环。第一次循环，对 A1 的目标目录执行 Dir，带上参数 vbDirectory，表示匹配的范围，包括该目录下面的子目录。

进入到内层的 Do While 循环。通过 fname = Dir 不断获取目标目录下面的文件或者子目录。

这两个嵌套的 If 条件，其中外层的，涉及到一个小知识——在任何一个目录（或者说文件夹）中，都有两个隐含目录："."表示当前目录，".."表示当前目录的上一级目录，我们把这两个目录都忽略掉。对于获取到的普通的文件或文件夹，会进入到内层的 If 条件，这里用到了 GetAttr 这个函数——

GetAttr 返回的是一个整数，表示文件、目录、或文件夹的属性。

另外，还用到了 vbDirectory 这个常数，这个常数，其实就是 16 。

如果 GetAttr 返回的是 16，那就表示这是个子目录，子目录录入到字典中去。

完成一次目录的查找后，内层 Do While 循环跳出，i 自加 1 。

此时，如果在上次目录的查找中，有找到新的子目录，d 的元素就增加了，这样虽然 i 自加 1，但 d.Count 也增加了，所以外层循环仍未能跳出，继续对新找到的子目录进行同样的查找操作，直到所有的子目录都完成查找为止。

【2】获取所有符合条件的文件名

通过 InputBox，弹出窗口，进行获取查找文件的后缀，参数 Default 表示默认值。

For Each 循环，它循环的数组是字典 d 的 Keys，也就是目标目录和所有子目录。

通过 Do While 循环，对目标目录和子目录下面的所有文件进行匹配。满足查询类型的所有文件的文件名，以字符串的形式，保存在 ms1 和 ms2 中。其中 ms1 只保存文件名，ms2 保存 路径+文件名。

result1 和 result2 分别保存 文件名 和 路径+文件名，这里，用到了两个函数——

Split，以“，”为识别符，将ms转换为一维数组。

一维数组取值到 Excel 表格的行很容易，但取值到列比较麻烦，所以用 Transpose，将一维数组转置为二维数组。

最后，把 result1 和 result2 填入到指定的区域，这个区域的定义，用了一个 Resize 函数。
