各种插口傻傻分不清 ？堂宝宝来给你理一理！

2020-11-14
| 科技

现在人手一两部手机相对来说很普遍了，就拿我来说吧，一部 iPhone 肯定是不够，因为续航实在堪忧。经常半天都不到，电量就不够 20%了。备一个充电宝插着线充电玩手机实在是难受啊。

于是我又入了一部安卓手机，身上还要常备一个满电充电宝，这样才刚刚满足要求。那么大家身边都是备用几根线呢？

iPhone 的 Lightning 线，安卓的 USB Type-C 线和Micro USB 2.0 线。

这都已经有三种接口了。经常用的人还好，但是平时只用一种接口的人，经常会有傻傻分不清的情况。

3.5mm / 2.0mm 接口

这种带着大屁股的电源适配器可以说是童年回忆了。这种接口虽然通用，但是考虑到不方便和不安全的因素，现在已经被淘汰掉了。

Mini USB

以前的 手机、MP3、照相机使用最多的就是这种接口了。但是很显而易见一点是，你现在已经很难这种接口用在高端设备上了。

两个原因，一是充电太慢，数据传输速度也是慢得不行。第二点是太厚了！

Micro USB 2.0

Micro USB 被抛弃的原因是因为太厚，那 2.0 版本要改变，第一条思路肯定是做薄。

USB Type - C

这种插口应该是目前安卓高端机上使用最多的接口了。对比 Micro USB 2.0 ，Type - c 有一种显而易见的好处：正反插都行。

Lightning

我想肯定有小伙伴们会想问，为什么苹果和安卓的充电接口不能通用？一个原因：苹果不想。苹果不管是用系统上、硬件上，都不屑于与安卓“ 同流合污 ”。

但是就设计上来说，Lightning 的接口还是很完美的，不管是用充电速度，传输速度上来说，都是可以个 Type - C 媲美的。

堂宝宝今天为大家讲解了 5 种接口，不知道大家现在分清楚了没有？
