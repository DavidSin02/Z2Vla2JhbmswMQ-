软妹子带你玩转Python数据可视化，座标轴方向及范围设置

2020-10-29
| 科技

在实际工作的过程中，针对不同的需求，我们需要对图像的座标轴方向或者范围进行自定义。比如，在一文中我们介绍了如何改变座标轴的位置，而当Y轴位于右侧或者X轴位于顶部的时候，我们通常需要将X轴或Y轴的方向进行改变；我们也有可能会遇到数据量很大但是只需要显示某个区间内的数据的情况，此时我们便需要对座标轴范围进行设置。

座标轴方向设置

座标轴的方向设置通常有两种方法：第一种为通过axes.invert_xaxis和axes.invert_yaxis方法对X轴和Y轴的方向进行改变；第二种为通过axes.set_xlim和axes.set_ylim方法对X轴和Y轴的方向进行改变。

1、axes.invert_xaxis和axes.invert_yaxis方法

axes.invert_xaxis()

axes.invert_yaxis()

axes.invert_xaxis和axes.invert_yaxis方法不需要参数，当需要改变座标轴方向时，直接调用该方法即可。

方法实例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

n_groups = 11

index= np.arange(1,n_groups)

means_men = np.random.randint(0,100,10)

means_women = np.random.randint(0,100,10)

fig, ax = plt.subplots(nrows=1,ncols=2,sharey=True)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,wspace=0.05)

ax[0].set_facecolor('None')

ax[0].yaxis.set_ticks_position('right')

ax[0].set_yticks(np.arange(1,n_groups))

ax[0].set_yticklabels(('A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J'))

ax[0].invert_xaxis()

ax[0].barh(index,means_women,color='lightgray')

ax[0].set_title('Women',color='w')

ax[1].set_facecolor('None')

ax[1].barh(index, means_men, color='lightgray')

ax[1].set_title('Men', color='w')

plt.show()

2、axes.set_xlim和axes.set_ylim方法

axes.set_xlim(left, right)

axes.set_ylim(bottom, top)

axes.set_xlim和axes.set_ylim方法的参数分别为X轴左端点值、右端点值和Y轴底部端点值、顶部端点值。

当需要改变X轴的方向的时候，只需将axes.set_xlim方法的left参数值设置为座标轴最大值、right参数值设置为座标轴最小值即可；想要改变Y轴的方向，只需将axes.set_ylim方法的bottom参数值设置为座标轴最大值、top参数值设置为座标轴最小值即可。

方法实例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

n_groups = 11

index= np.arange(1,n_groups)

means_men = np.random.randint(0,100,10)

means_women = np.random.randint(0,100,10)

fig, ax = plt.subplots(nrows=2,ncols=1,sharex=True)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,hspace=0.12)

ax[0].set_facecolor('None')

ax[0].set_xticks(np.arange(1,n_groups))

ax[0].set_xticklabels(('A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J'))

ax[0].bar(index,means_women,color='lightgray')

ax[1].set_facecolor('None')

ax[1].xaxis.set_ticks_position('top')

ax[1].set_ylim(100,0)

ax[1].bar(index,means_men,color='lightgray')

plt.show()

座标轴范围设置

座标轴范围的设置通常有两种方法，第一种为通过axes.set_xbound和axes.set_ybound对X轴和Y轴进行分别设置，第二种为通过axes.set_xlim和axes.set_ylim方法对X轴和Y轴进行分别设置。

这两种方法虽然都能改变座标轴刻度范围，但是在使用的时候却有差别。

1、axes.set_xbound和axes.set_ybound方法

axes.set_xbound(lower, upper)

axes.set_ybound(lower, upper)

axes.set_xbound和axes.set_ybound方法有两个参数值，这两个参数值不分先后，大的值代表座标轴的最大值，小的值为座标轴最小值。

方法实例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

x = np.linspace(0.0, 5.0)

y = np.cos(2 * np.pi * x) * np.exp(-x)

fig, ax = plt.subplots(nrows=1,ncols=2)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,wspace=0.12)

ax[0].set_facecolor('None')

ax[0].plot(x,y,'o-',color='gold')

ax[1].set_facecolor('None')

ax[1].plot(x,y,'o-',color='gold')

ax[1].set_ybound(-0.3,0.5)

plt.show()

2、axes.set_xlim和axes.set_ylim方法

axes.set_xlim(left, right)

axes.set_ylim(bottom, top)

axes.set_xlim和axes.set_ylim方法我们已经在座标轴方向中讲过，他们的参数分别为X轴左端点值、右端点值和Y轴底部端点值、顶部端点值，所以只要给定参数值就设定好了座标轴范围。

方法实例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

x = np.linspace(0.0, 5.0)

y = np.cos(2 * np.pi * x) * np.exp(-x)

fig, ax = plt.subplots(nrows=2,ncols=1)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,hspace=0.12)

ax[0].set_facecolor('None')

ax[0].plot(x,y,'o-',color='gold')

ax[1].set_facecolor('None')

ax[1].plot(x,y,'o-',color='gold')

ax[1].set_xlim(1,4)

plt.show()

需要注意的是

  1. 当我们需要使用axes.set_xscale方法改变X轴比例尺时，axes.set_xbound方法和axes.set_xlim方法必须在axes.set_xscale方法之后使用才能正常显示。

  2. 当我们需要使用axes.invert_xaxis方法对座标轴方向进行改变时，使用axes.set_xbound方法并不会对axes.invert_xaxis方法产生影响；而axes.set_xlim方法与axes.invert_xaxis方法互相影响，位置靠后的代码效果会覆盖位置靠前的代码效果。

对比实例：

1、左图axes.set_xbound方法在axes.set_xscale方法之后使用，右图axes.set_xbound方法在axes.set_xscale方法之前使用

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

x = np.linspace(0.0, 5.0)

y = np.cos(2 * np.pi * x) * np.exp(-x)

fig, ax = plt.subplots(nrows=1,ncols=2)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,wspace=0.12)

ax[0].set_facecolor('None')

ax[0].plot(x,y,'o-',color='gold')

ax[0].set_xscale('log',basex=2)

ax[0].set_xbound(0,4)

ax[1].set_facecolor('None')

ax[1].plot(x,y,'o-',color='gold')

ax[1].set_xbound(0,4)

ax[1].set_xscale('log',basex=2)

plt.show()
