2020升降机司机模拟考试题库及升降机司机模拟考试系统

2020-10-29
| 科技

题库来源：安全生产模拟考试一点通公众号小程序

2020升降机司机模拟考试题库及升降机司机模拟考试系统，包含升降机司机模拟考试题库答案解析及升降机司机模拟考试系统练习。由安全生产模拟考试一点通公众号结合国家升降机司机考试最新大纲及升降机司机考试真题出具，有助于升降机司机模拟考试软件考前练习。

1、【判断题】从业人员在作业过程中，应当遵守本单位的安全生产规章制度和操作规程。（ √ ）

2、【判断题】对起重机作全面检查，在确认不影响工作的情况下，方可推合保护柜总刀闸。（ √ ）

3、【判断题】起重机作业时，吊物不准长时间停留在空中，吊物未卸去司机不得离开司机室。（ √ ）

4、【判断题】有主、副两套起升机构的起重机，应同时使用两套机构进行作业。（ × ）

5、【判断题】起重机司机操作应按指挥信号进行，对紧急停车信号不论何人发出都应立即执行。（ √ ）

6、【判断题】起重机上发生火灾的原因主要是电气设备绝缘老化短路引起的火花。（ √ ）

7、【判断题】制定《中华人民共和国特种设备安全法》最重要的目的是制裁各种特种设备安全违法犯罪行为。（ × ）

8、【判断题】《中华人民共和国特种设备安全法》规定，县级以上地方各级人民政府对本行政区域内特种设备安全实施监督管理。（ × ）

9、【判断题】《中华人民共和国特种设备安全法》规定：未经定期检验或检验不合格的特种设备，不得继续使用。（ √ ）

10、【判断题】钢丝绳严重的腐蚀还会引起钢丝绳弹性的降低。（ √ ）

11、【判断题】描述导线中流过的电量大小的单位是“安培”。（ √ ）

12、【判断题】特种设备安全监督管理部门应当制定特种设备应急预案。（ √ ）

13、【判断题】在电路中如果发生短路或严重过载时，需要迅速切断电源，担负这一任务的是时间继电器。（ × ）

14、【判断题】司机对任何人发出的危险信号均应执行。（ √ ）

15、【判断题】持证作业人员违章操作或者造成特种设备事故的，应当注销《特种设备作业人员证》。（ √ ）

16、【判断题】《起重机械安全监察规定》中规定，起重机械安装、改造、维修许可证有效期为2年。（ × ）

17、【单选题】从事特种设备作业的人员，经上岗前的培训和考核取得( )后，方可独立操作。（ C ）

A、上岗证

B、技能证

C、特种设备作业人员证

18、【单选题】起升机构制动器的制动轮轮缘厚度磨损超过( ),应更换。（ B ）

A、25％

B、40％

C、50％

19、【单选题】特种设备作业人员考核发证工作应当遵循( )的原则。（ A ）

A、便民、公开、高效

B、公开、公正、公平

C、便民、高效、公开

20、【单选题】稳钩操作时，跟车速度( )。（ B ）

A、不宜太快

B、不宜太慢

C、较难确定

21、【单选题】起重机运行轨道的终点设置的终点档架，习惯上称:( )。（ B ）

A、锚定

B、止挡

C、扫轨板

22、【单选题】一般吊装用钢丝绳之间的夹角应不大于:( )。（ B ）

A、90°

B、120°

C、150°

23、【单选题】起升机构在突然提升、突然制动时会产生( )，有可能造成起重机倾翻。（ C ）

A、向心力

B、离心力

C、惯性力

24、【单选题】中华人民共和国特种设备安全法》规定，国家对特种设备的生产、经营、使用，实施( )、全过程的安全监督管理。（ A ）

A、分类的

B、全面的

C、部分的

25、【单选题】起重机整机工作级别符号为:( )。（ B ）

A、S

B、A

C、Gn

26、【单选题】钢丝绳表层磨损或锈蚀量达( )%时，应报废。（ A ）

A、40

B、20

C、10

27、【单选题】在起重作业中尽量使货物的重心在两根起吊绳的( )。（ C ）

A、左边

B、右边

C、中间

28、【单选题】为防止起重机被大风吹动或刮倒而安装的安全装置是( )。（ B ）

A、制动装置

B、夹轨器

C、安全开关

29、【单选题】起重机工作完毕后，应将空钩提升到：( )。（ A ）

A、接近上限位

B、中间

C、放在地面

30、【单选题】超负荷限制器是在起重机起升时超负荷起保护作用,这时起重机会( )。（ B ）

A、鸣铃

B、切断起升电源

C、指示灯亮

31、【单选题】卷筒轴平时检查不便，应在起重机( )时检查。（ C ）

A、小修

B、中修

C、大修

32、【单选题】减速箱油量要( )。（ C ）

A、充足

B、少量

C、适中

33、【单选题】《中华人民共和国特种设备安全法》第十三条规定，特种设备生产、经营、使用单位及其主要负责人对其生产、经营、使用的特种设备( )负责。（ D ）

A、经济运行

B、高效率

C、节能

D、安全

34、【单选题】《中华人民共和国特种设备安全法》所称的县级以上地方特种设备安全监督管理部门在现行体制下是指( )。（ B ）

A、安全生产监督管理部门

B、质量技术监督部门

C、交通部门

D、劳动部门

35、【单选题】《特种设备作业人员证》每( )年复审一次。（ B ）

A、2

B、4

C、6

36、【单选题】“小臂向侧上方伸直，五指自然伸开，以腕部为轴转动”，这种手势信号表示( ) 。（ A ）

A、吊钩上升

B、紧急停止

C、工作结束

37、【单选题】二氧化碳灭火器使用时，应( )操作，喷嘴要对准火源根部。（ C ）

A、倒置

B、放平

C、垂直

38、【单选题】起重重物时，如果手柄推到第二档，仍不能起吊，除了电气故障外，很可能是( )。（ B ）

A、停电

B、超载

C、机械故障

39、【单选题】电动机轮胎吊的操作机构通常用( )。（ C ）

A、凸轮控制器

B、液压推杆

C、联动台

40、【单选题】大车行走机构通常能在( )米／秒风速下安全运行至锚定坑位置进行锚定。（ B ）

A、28

B、25

C、30

41、【单选题】卸扣按其外形分为：( )。（ A ）

A、直形

B、方形

C、四边形

42、【单选题】《中华人民共和国特种设备安全法》规定,向负责特种设备安全监督管理的部门办理使用登记,取得使用登记证书。( )应当置于该特种设备的显著位置。（ C ）

A、使用证书

B、登记证号

C、登记标志

D、登记机关名称

43、【多选题】事故简要经过:物资公司仓库内,一台3吨电动单梁正在吊运物品,突然电动葫芦上的钢丝绳导绳器脱落,砸倒了在地面作业的一名工人,头部血流满面,重伤送医。事故调查:经查,吊钩滑轮罩壳变形严重,导绳器紧固螺丝螺纹明显磨损。安全装置有效,通过安全检验,在有效期内使用。事故原因分析:( )（ ABCD ）

A、导绳器固定螺丝松动

B、维保不到位,缺乏日常检查

C、使用时存在斜拉斜吊等违章操作

D、企业的安全责任制没有得到落实

44、【多选题】起重机作业时，应根据( )，调整起重臂长度和仰角。（ BC ）

A、重物的体积

B、重物的重量

C、提升高度

D、吊运环境

45、【多选题】保障安全生产的原则是( )。（ ABC ）

A、经常维护保养设备

B、按照标准进行操作

C、整洁有序的作业环境

D、熟练的操作技能

46、【多选题】下列属于特种设备的有：( )。（ ABD ）

A、起重机械

B、电梯

C、冲压设备

D、压力容器

47、【多选题】燃烧的基本条件是：( )。（ BCD ）

A、一定温度

B、可燃物

C、助燃物

D、着火源

48、【多选题】( )和( )有权对于违反《中华人民共和国特种设备安全法》规定的，向负责特种设备安全监督管理的部门和有关部门举报。（ AD ）

A、任何单位

B、操作人员

C、管理人员

D、个人

49、【多选题】提高起重机稳定性的基本方法有：( )。（题中没有表明是何种起重机，方法也是不一样的，建议删除）（ ABC ）

A、降低重心

B、加大配重

C、增大支承面积

D、降低工作速度

50、【多选题】严禁在安装好的起重机的金属结构上悬挂( )。（ ABC ）

A、标语牌

B、广告牌

C、回转要平稳

D、挡风物件
