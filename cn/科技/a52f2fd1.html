常见几种地板木材大盘点，轻松解决您的选择困难症

2020-11-14
| 科技



许多人在选购实木地板时会犯难，面对多种木材制作的实木地板，由于不能清楚各种材料的特点，挑选起来实在是有心无力。小编今天为您收集了16款木材优缺点，让你轻松选择不纠结！

1、番龙眼


优点

木材具金色光泽，纹理直，结构细致均匀；重量及强度中等，硬度中至略硬。

油漆、胶拈性好，不易翘裂，耐腐及抗虫性强；部分番龙眼实木地板为红褐色，纹理清晰，适合打造欧式、中式风格家居。


缺点

番龙眼实木地板不能用于地暖，否则有可能变形或开裂。

2、橡木


优点

具有比较鲜明的山形木纹，并且触摸表面有着良好的质感；韧性极好，可弯曲，造型美观；质地坚实，稳定性好，使用年限长；档次较高，适合欧式、中式古典风格，显厚实感，有红木的端庄沉稳价格却更亲民。


缺点

未脱净水则易变形开裂。

在选择橡木地板时应该选择大品牌。例如未来家地板采用了瞬间干燥技术，能够保持原木天然纹路和色泽的优美，透气不渗水，让地板通畅呼吸。

3、柚木


优点

柚木被誉为“万木之王”，能防潮、防虫、防蚁，尤其耐腐；柚木稳定性好，其香味对中老年人的脑神经系统有益；油斑随阳光的作用会逐渐消褪，颜色则更加美丽。


缺点

柚木珍贵稀有，价格在三千至上万元一平方。

4、桦木


优点

桦木植物资源十分丰富，所以将它作为原材料的地板一般价格比较便宜；桦木的颜色浅淡，可以进行多种加工，一般颜色清透自然，十分百搭。


缺点

桦木的木质较为松软，耐磨性差，多用作复合地板。

5、二翅豆


优点

二翅豆因其纹理如龙身凤尾而被中国人称为龙凤檀，其纹理清晰，似龙似凤，是一种美好的象征；木质坚硬，色泽沉稳，高贵典雅，颜色红，非常适合中国古典式的装修风格。


缺点

二翅豆地板稳定性差，容易变形，花纹大，有色差；木材密度高材质硬，所以地板两端头容易出现暗裂，哑光工艺比淋漆更能掩盖暗裂痕迹。北方气候不宜铺装二翅豆实木地板。

6、白蜡木


优点

色彩淡雅，纹理夸张绚丽，质感好，美观个性、富有艺术气息；触感柔和，即使在冬天，也不会令人觉得冰冷生畏；主要呈奶白色以及淡粉色，很适合田园风格及现代简约风格的装修。


缺点

白蜡木密度较低，硬度差，木质是偏软，所以耐磨度差，要特别注意保养。

7、花梨木


优点

材质坚硬、沉重，纹理清晰，切削面光滑，耐腐蚀，涂饰、胶合性较好。


缺点

颜色较淡，结构中等，不易干燥。

8、相思木


优点

木材细腻、密度高，强度及抗冲击韧性好，很耐腐；该木材稳定性好、韧性强、耐腐蚀、缩水率小；心材呈黑褐色或巧克力色，结构均匀，生长轮明显且自然，高贵典雅。


缺点

易变形，保养起来较困难。

9、枫木


优点

纹理美丽多变、细腻，显得安静高雅；韧性佳，软硬适中，做成木地板很实用；颜色比较强，可以让房间看起来整洁不拥挤。


缺点

枫木地板制作成的木地板颜色比较浅，不耐脏，需要勤快打理；木质硬度适中，不耐磨。

10、圆盘豆


优点

圆盘豆地板颜色比较深，分量重；圆盘豆地板密度大，比较坚硬，抗击打能力很强；在中档实木地板中，圆盘豆地板的稳定性能是比较好的。


缺点

木种颜色比较深，边材和心材颜色差的很大，所以圆盘豆地板存在色差较大。

密度太大以至于做成地板脚感会略差一些，好象踩在石头上，建议老人和儿童不要选这个品种。

11、松木


优点

松木，因为容易干裂和有松脂渗出，所以本身并不是做地板的好材料。经过特殊的工艺，把过量松脂脱胎烘干，可加工成有靓丽色泽、刚柔并济并的优质木材。松木地板环保、耐腐，自然木节很适合田园风格，松木气味儿还有利于人体健康。


缺点

松木木质软，易开裂变形，含水率高也容易导致开裂；松木讲究纯自然色，需好好保养，否则容易变色。

12、柞木


优点

材质坚硬，结构粗，强度高，着色、涂饰性好，易干燥。


缺点

加工困难，胶合性差，易开裂。

13、水曲柳


优点

材质略重硬，花纹美丽，结构粗，易加工、韧性大，涂饰胶合性好。


缺点

腐蚀能力弱，易受常见家具甲虫蛀食，不易干燥，易产生翘裂。

14、榆木


优点

材质花纹美丽，结构粗，加工性、涂饰胶合性好。


缺点

干燥性差，易开裂翘曲。

15、黑胡桃木


优点

贵重木材，木纹美观大方，黑中带紫，典雅高贵。


缺点

由于色相重，要求油漆成膜后必需透明度高；木纹比较深，要求透明底漆的填充性好，封闭性强；面板加工时有染色或胶合板中存在褪色物，时间一久容易褪色。

16、榉木


优点

材质坚硬，纹理直，结构细、耐磨有光泽、干燥时不易变形，加工、涂饰胶合性较好。


缺点

比多数硬木都重，在窑炉干燥和加工时容易出现裂纹



这下大家都对这些木材有了基本的了解了吧，快趁着假期来选购地板吧
