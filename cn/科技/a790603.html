2、server_name配置

1）、准确匹配

server {

listen 80;

server_name www.shenmazong.com;

#...

}

2）、通配符开始

server {

listen 80;

server_name *.shenmazong.com;

#...

}

3）、通配符结束

server {

listen 80;

server_name www.shenmazong.*;

#...

}

4）、正则表达式

server {

listen 80;

server_name ~^(?.+)\.shenmazong\.org$;

#...

}
