箱型图

我们想看看连续变量在某一个分类变量的不同类别上的分布，一个常用的方法就是画箱型图，如果有不清楚箱型图是什么的同学请看我之前的文章或者去问度娘：


箱型图详解

  • 实例操练

使用的数据集为自带的diamonds数据集，我们用箱型图画出不同的cut上钻石价格的分布，代码如下：

ggplot(data = diamonds, mapping = aes(x = cut, y = price)) +  geom_boxplot()


再画一个：这次还是用自带的mpg数据集，我们画一下不同种类的汽车hwy的分布（大家可以自行查看变量是啥意思哈，我就不写了），代码如下：

ggplot(data = mpg, mapping = aes(x = class, y = hwy)) +  geom_boxplot()


上图中的箱图有点乱，这个时候我们其实想能不能将我们图中的箱图按照某种顺序排一排，比如按照hwy的中位数给所有箱图排序：

ggplot(data = mpg) +  geom_boxplot(mapping = aes(x = reorder(class, hwy, FUN = median), y = hwy))


这一下就好多了，注意到要实现排序，我们用到了reorder方法，这个方法的第一个参数是要排序的变量，第二个参数是排序依照的标准变量，第三个参数则是排序依照的方法。

如果你的x轴的变量名比较长，可以考虑将图像反转90°，用coord_flip()就行，代码如下：

ggplot(data = mpg) +  geom_boxplot(mapping = aes(x = reorder(class, hwy, FUN = median), y = hwy)) +  coord_flip()
