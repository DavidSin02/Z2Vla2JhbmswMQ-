教你从头学前端开发——第一个网页（一）

2020-11-14
| 科技

大家好我是一个不那么正经的“没有格子衫的程序猿”，本人的职业是一名专业的程序员，目前在南宁的一家软件公司任职前端工程师。从今天开始，我将会向大家介绍怎么从零开始成长成为一名根正苗红的前端攻城狮。

上一期我带大家简单认识了什么是前端开发，那这期咱们开始学习前端开发，写我们的第一个网页。

首先和大家说一下，在我们每次打开的网站中，眼前所呈现的图文页面都是有HTML、CSS、和JavaScript语言编写的。HTML是负责页面的标签框架，CSS是负责页面的样式美化，而JavaScript则是负责页面的逻辑功能等；这三大语言简单通俗的理解就是HTML是房子的框架，未经过装修的毛坯房；CSS则是给房子装修，给墙体上油漆、铺地板砖，布置桌椅板凳什么的；而JavaScript就像是给房子装上水管电线。

那么本期我们先开始教大家怎么用HTML给网页搭建框架。HTML中文名称是：超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。HTML严格意义上来说它并不是计算机编程语言，不像java，JavaScript，PHP，C#这类的语言。HTML 是运行在浏览器上的，由浏览器来解析。不用服务端来支撑，所以你想运行HTML文件，只需用浏览器打开文件即可。

接下来我们进入重点，写我们的第一个网页，编辑器我本次用的是Sublime Text。对于编辑器这块大家可以根据自己的习惯来选用，没有太多的限制。我平时用的就是Sublime Text和Visual Studio Code这两款。

打开Sublime Text然后我们新建文件，将文件命名为test.html，然后保存到本地中，注意HTML文件的后缀名是.html或者.htm。

在test.html文件中，我们写上以下代码：

<!DOCTYPE html><html lang="en"><head>		<meta charset="UTF-8">		<title>Document</title></head><body>		<p>你好，世界！</p></body></html>

然后找到test.html文件所在位置，右键用浏览器打开它，然后就可以看到页面上有“你好，世界！”，至此，一个最简单的网页就写好了。

下面给大家介绍代码的构成。<!DOCTYPE html> 声明为 HTML5 文档，它是指示 web 浏览器关于页面使用哪个 HTML 的版本进行编写的指令。<html> 元素是 HTML 页面的根元素，lang="en"中的lang 属性规定元素内容的语言。en是指英文，这样有些中文浏览器在打开这个页面的时候就会提示是否翻译成中文，如果想设置为中文，将lang属性改成“zh”即可，既lang="zh"。在<head> 元素中我们可以写文档的元数据，如meta，title，样式或者引入依赖等等。<meta charset="utf-8"> 定义网页编码格式为 utf-8。<title> 元素描述了文档的标题，该标题会被作为浏览器标签的标题。<body> 元素包含了可见的页面内容，所有的页面内容我们将会在这个标签中去编写，会在里面添加标签或者文字。<p> 元素定义一个段落。

以上就是一个最简单的网页的编写过程，对于前端来说，HTML是最基础的，其他功能都将围绕这个基础展开。那么下一期我会以一个复杂点的页面示例，和大家说说HTML的标签和CSS样式。欢迎大家关注下期的内容。
