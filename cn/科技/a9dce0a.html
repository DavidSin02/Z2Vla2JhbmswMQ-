极致不妥协的开发工具 HP ZBook Studio工作站评测

2020-10-29
| 科技

说到工作站，相信大家都不会陌生。在以前，工作站给大多数人的感觉都是厚重的台式机的形象，搭配了专业的图形设计卡，性能十分强劲。但随着技术的进步，移动工作站也逐渐撑起了自己的一片天，而且随着传统办公场景的逐步瓦解，现在的人们更希望能抱着一台笔记本电脑即可在任意一个场所完成原有的工作，因此在这种移动化办公场景下，人们就对移动设备的便携性、外观设计、性能等多方面设计提出了更高的要求。但符合这样的工作站比较少见。

但凡事皆有例外，笔者手上这台HP ZBook Studio G7就在专业级别的可靠性以及安全性之上，与极致的移动性相结合，契合了当代年轻人的审美、多场景办公需求。相信大多数创意工作者在接触到HP ZBook Studio G7的第一刻也会怀疑手中的是一款笔记本而不是工作站，那么ZBook Studio G7究竟能否称得上是“极致不妥协的开发工具”呢？不妨通过下面的评测一看究竟。

HP ZBook Studio G7工作站

闲话少说，先简单说下配置：

处理器：Intel酷睿i9-10885H

显卡：NVIDIA Quadro RTX 3000 Max-Q

内存：32GB 双通道 DDR4

硬盘：2TB PCIe NVMe M.2 SSD

在如此多料的堆叠下，ZBook Studio G7带来了更加可靠的专业性能。当然，按照惯例，首先来看看ZBook Studio G7的外观设计吧！

01化繁为简 别具匠心的产品设计

HP ZBook Studio G7机身由CNC特种铝合金打造而成，军工标准设计使得机器更加耐用。颜色采用深灰配色，整机具有天然的缎触感，搭配一体式机身，让它看起来足够稳重大气。A面logo从原有的“HP”改为了Z系列工作站特有的“Z”型logo，增加了辨识度，也更有一丝时尚感。

这样的设计，也许是考虑到了年轻一代创意工作者对于移动办公的需求，们时常会在不同的办公场景中穿梭，例如咖啡厅、飞机场、会议室等等，因此便携性逐渐成为十分重要的购买考量因素。此外，沉稳又不乏时尚感的设计也为使用者在无形中提升了气场。

值得一提的是，HP ZBook Studio G7机身整体重量不足1.8Kg，厚度仅为17.5mm，相比上一代的HP ZBook Studio G5，更加轻薄，紧凑程度提升 22%，显得更加小巧精致。15.6英寸的大小也符合当代专业人士的使用需求。

得益于机身的材质，HP ZBook Studio G7可以轻松单手打开，打开后就会被其屏幕设计迷到：高达87%的屏占比，100% DCI-P3s色域，600尼特防眩光4K IPS显示屏带来极致的视觉体验，准确呈现全光谱色彩，支持10亿种屏幕显示色彩，效果惊艳的Dream Color技术让创作不再成为难题。

HP ZBook Studio G7的键盘采用剪刀机制，键程为1.3毫米，可带来上佳的响应体验。此外，消音橡胶垫和防震支架可免除干扰，手感上乘。通过HP Programmable Key APP对按键进行可编程设置，用于打开文件、网站、软件等等。放泼溅设计可有效防止液体侵入机器内部，严格的质量品控让按键可靠耐用。

不过看到这里，可能有些专业人士不免会担忧，在如此纤薄的机身中已经塞入了性能强大的硬件，散热到底能不能hold住？答案当然是肯定的。超薄的液晶聚合物风扇和3面排气设计带来的是极佳的散热解决方案，不仅如此，机器内部的液冷均热板会使液体蒸发，促进GPU和CPU的热量散发，而且会让蒸气散布在整个腔室内，以此优化散热效果。这一切的设计，都让ZBook Studio G7可以持久而稳定的运行，助力专业人士高效地完成创意设计。

常言道，细节决定成败。创意族群的职业特性要求其所使用的工作站必须提供足够丰富的外置接口，以满足文件传输、投屏、多屏等办公需求。HP ZBook Studio G7配备了一个SD读卡器插口，一个Mini DisplayPort 1.4接口，两个带Thunderbolt的USB Type-C接口和一个USB 3.1 Gen1 Type A接口和一个耳麦接口，足够设计工作者扩展存储大量文件。

值得一提的是，ZBook Studio G7采用了Bang & Olufsen音响技术，通过4个独立放大器（2个高音扬声器和2个低音扬声器），带来丰富的环绕音效；150Hz滚降低音让音乐播放更原汁原味。此外HP ZBook Studio还通过了众多ISV（独立软件厂商）认证，软硬件无缝衔接，即使是素材量庞大的设计任务，也能保证顺畅可靠的运行，有助于设计师们在日常工作中提升办公效率。
