PTT VBA｜幻灯片进行文本和段落格式化（一个或两个文本框）

2020-11-14
| 科技

ppt和excel、word一样都可以用vba操作。

正和excel中有Excel→Workbook→Worksheet→Range这样的父子关系对象一样。

ppt也有PowerPoint→Presentation→Slide→shape这样的父子关系对象。

其中PowerPoint表示ppt应用程序，Presentation对象表示所有打开的ppt演示文稿、Slide对象表示演示文稿中的幻灯片。

所有形状组成shapes对象。

所有的幻灯片组成了Slides集合对象。

所有的演示文稿组成了Presentations集合对象。

前面一篇《PTT VBA｜统一或指定幻灯片进行文本和段落格式化》的操作是不考虑幻灯片的标题与内容的关系，统一进行格式操作，不是很妥当，并且没有位置的考量。

现在的任务是，只考虑一个幻灯片中有一个或两个文本框的情形，做不同的文本与段落的格式化，并统一文本框的放置位置。

运行效果：

如果不想对每一张幻灯片进行操作，只想对最后一张或指定的一张进行操作的话，只需将遍历的代码去掉，并指定幻灯片即可：

For i = 1 To oPres.Slides.Count……Next

替换为：

i = oPres.Slides.CountSet oSlide = oPres.Slides.Item(i)

附代码1：

Sub allSlides1or2shape()On Error Resume NextDim oPres As PresentationSet oPres = Application.ActivePresentationDim oSlide As SlideDim oShape As ShapeDim tr As TextRangeDim sText As StringDim i As Long, j As LongDim a As Long, b As LongDim fjd As IntegerDim high1 As Integer, high2 As Integer'循环每页幻灯For i = 1 To oPres.Slides.CountSet oSlide = oPres.Slides.Item(i)Set oSlide = oPres.Slides.Item(i)fjd = oSlide.Shapes.Count'判断每页内是一个还是有2个文本框If fjd = 2 Then '两个文本框设置为上下为标题+内容结构'把顶点值小的文本框仍然设置在上high1 = oSlide.Shapes.Item(1).Tophigh2 = oSlide.Shapes.Item(2).TopIf high1 < high2 ThenSet oShape1 = oSlide.Shapes.Item(1)Set oShape2 = oSlide.Shapes.Item(2)ElseSet oShape1 = oSlide.Shapes.Item(2)Set oShape2 = oSlide.Shapes.Item(1)End IfWith oShape1'设置文本框的宽度和位置.Left = 45.Top = 45.Width = 625End WithIf oShape1.TextFrame.HasText = msoTrue Then '字体设置Set tr = oShape1.TextFrame.TextRangeWith tr.Font.NameAscii = "宋体".NameFarEast = "宋体".Size = 32'.Color.SchemeColor = ppBackground.Color.RGB = RGB(Red:=165, Green:=42, Blue:=42).Bold = msoCTrueEnd WithSet tr = NothingEnd IfSet oShape1 = NothingWith oShape2.Left = 50.Top = 100.Width = 625.TextFrame.TextRange.ParagraphFormat.SpaceWithin = 1.1'设置行距End WithIf oShape2.TextFrame.HasText = msoTrue ThenSet tr = oShape2.TextFrame.TextRangeWith tr.Font.NameAscii = "宋体".NameFarEast = "宋体".Size = 28.Bold = msoCFalseEnd WithSet tr = NothingEnd IfSet oShape2 = NothingElse'单个文本框设置为内容结构Set oShape = oSlide.Shapes.Item(1)With oShape'设置文本框的宽度和位置.Left = 45.Top = 45.Width = 625.TextFrame.TextRange.ParagraphFormat.SpaceWithin = 1.1'设置行距End WithIf oShape.TextFrame.HasText = msoTrue Then'设置文本框的字体和大小Set tr = oShape.TextFrame.TextRangetr.Font.NameAscii = "宋体"tr.Font.NameFarEast = "宋体"tr.Font.Size = 28'tr.ParagraphFormat = 10Set tr = NothingEnd IfSet oShape = NothingEnd IfSet oSlide = NothingNextSet oPres = NothingEnd Sub
