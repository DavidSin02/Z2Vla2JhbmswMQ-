拿了红点奖的岩板长什么样？

2020-10-29
| 科技

近日，德国红点奖公布了2020年获奖名单。陶城君统计发现，今年共有岩板、瓷砖、卫浴、马桶和水龙头等7款产品获奖。

其中，意大利Florim和土耳其Kale的岩板产品获得品牌与传达大奖；中国Tsang Ching Pun设计公司的Clean Seat公共马桶获得产品设计概念大奖；Albarello公司Ignorance Is Bliss瓷砖、Duravit Viu / XViu卫浴、Aquno Select M81厨房水龙头、Axia系列水龙头等产品获得产品设计大奖。

岩板设计获得品牌与传达大奖‍

红点奖共有4大奖项，包括产品设计大奖、品牌与传达设计大奖、设计概念大奖和当代好设计奖。其中，2020年品牌与传达设计大奖共有6992件参赛作品，有854个品牌和传达作品获得了红点奖，占参赛作品的13%。

陶城君检索发现，除了卫浴产品外，意大利Florim和土耳其Kale2家岩板企业的产品获得该奖项。

Florim集团Policroma岩板

意大利Florim集团的新品Policroma系列瓷砖获此殊荣。由设计师Cristina Celestino创作的Policroma系列岩板将古典建筑经典与极具现代感的表达方式相结合，重新定义了瓷砖产品。Policroma岩板呈现了大理石和Marmorino风格的瓷砖表面。

仿大理石效果是Policroma系列内涵的关键，Policroma系列复兴了一种越来越稀少的材料。Cristina Celestino表示，它并没有提出“代替天然大理石”的口号，而是通过模仿，将大理石和Marmorino灰泥相结合，以一种对比强烈、具有现代感的方式再现了具有百年历史的大理石。

设计所选用的大理石类型各具特色。比如，Verde Alpi大理石是经常出现在米兰大教堂入口大厅的一种瓷砖；Breccia Capraia珍稀大理石则存在于意大利托斯卡纳区的极少数地点；Cipollino大理石是特殊的绿红色品种，带有螺旋状纹理；Rosa Valtoce大理石则来自阿尔卑斯山脉附近的皮埃蒙特的一个采石场。

Policroma系列的众多元素突出了瓷砖加工工艺的重要性——通过各种模块的自由组合，打造出各种半圆造型，或者通过小尺寸设计的大量复制，使最初只是用作点缀的细节化身为整个设计的“主角”。它以一种质朴但又并非极简主义的方式来设计墙地面，简单而不失智慧，低调的空间和前卫的宣言，充分体现了“米兰style”。

土耳其Kale“无限”展馆

意大利建筑设计公司Paolo Cesaretti Arch为土耳其Kale陶瓷集团设计的临时展馆Bespoke Romance获得了2020年红点奖“品牌与传达设计”奖项。

“无限”概念是该展馆的设计灵感，两个融汇的圆代表Kale集团文化的双重魅力。展馆的圆形外墙设计大胆、紧凑，同时采用镂空设计，汇聚了国际化企业Kale集团的众多品牌和特色。展馆外墙由数百片竖立的Kalesinterflex超薄岩板建成。展厅内部的展示区也使用了这种岩板，用于室内家居装饰，包括厨房、客厅和浴室，向参观者充分展示了Kale集团产品的特性和功能。

灯光的巧妙运用凸显了展厅的独特形状，让整个展厅带有一种抽象的美感。环形小路上的是家装和商业产品展示区。由经过切割的瓷砖制成的装饰面板竖立在天花板和地板之间。此外，展厅内宽敞的休息区也非常引人注目，其将优雅时尚的酒吧和可于集体工作的会议区空间融合在一起。

瓷砖、卫浴、马桶、水龙头获得设计大奖‍

2020年红点设计大奖共有超过6,500件参赛作品，仅76件作品获得了红点最佳设计奖，获奖率仅为1.2%，小米、联想等多个中国企业品牌也在榜单中赢得一席。

陶瓷卫浴行业中，Albarello的Ignorance Is Bliss瓷砖、Duravit Viu / XViu卫浴、Aquno Select M81厨房水龙头、Axia系列水龙头等产品获得该奖项。

Albarello的Ignorance Is Bliss瓷砖

金属是造成全球污染的原因之一，而且由于其不可再生性，资源是非常有限的。在这种背景下，“Ignorance Is Bliss”这款瓷砖百分百使用工业金属废料作为陶瓷颜料为陶瓷上色，促进了工业金属废物进行合理地循环利用。

Duravit Viu / XViu卫浴系列

Viu陶瓷系列、XViu家具系列以及配套的浴缸作为一个完整的浴室系列，整体设计散发着前瞻性的魅力。该系列作品被定位为提升“后工业化”的优雅，设计中体现出柔软有机的形式与精确的几何碰撞出的火花，在细节上也高度完善考究。

此外，该浴室系列由高级陶瓷、玻璃、漆面金属和优质木材制成，各种不同原材料表面的组合和鲜明反差是该系列的一大特色。

洗漱区域是设计时的一大重点，该区域采用了杜拉维特(Duravit)的C粘合专利技术，为用户提供友好舒适的使用体验。

产品外形是严格的矩形，而内部则具有柔和的有机曲线，洗手盆宽敞而搁板收纳区域宽阔。金属控台，梳妆台，镜子，镜子柜和独立式浴缸等系列产品，通过C型粘合有机融合，并运用香槟色或黑色哑光的金属材质被塑造成醒目的V型，外观独特。

通过不同材料的组合，打造出同色调和反差色的不同配色方案。高档实木，清漆和装饰饰面等可搭配出从“银色”到“深色”的多种样式。

Aquno Select M81厨房水龙头

Aquno Select M81系列产品的设计中，将厨房冷热水混合龙头作为优化工作流程的重点。旨在用于大尺寸的水槽，该产品由拱形出水口和赏心悦目的几何图形组成。

这个水龙头引人注目的外观与厨房氛围融为一体。其多功能性在于包含一个可拉出式的喷头，以及一个精细的平板，在其下方有一个突出的边缘导流，让水以一种创新的方式流出。

使用者可以选择花洒或喷射模式出水，也可以用底部的混合龙头水平喷水，这样水流更低也更接近台盆。

根据不同的烹饪步骤，这种由单个超细微喷雾组成的水流可以更温和地冲洗脆弱的食物，同时在清洗大物体时也不会溅水，另外洗手时也更节水。结合特制的多功能过滤器，这款水龙头让备置食物的过程变得简单而优雅——以一种全新的方式来感受水流。

Axia系列水龙头

Axia系列以其全新的现代极简风格令人印象深刻。它的设计表达出设计师不断探索突破轻薄和纯粹的边界。

Axia的外观令人难以忘怀，一件式铸造的6.5毫米高的超薄出水口，标榜着这一系列不容小觑。这种设计美感创造出富有表现力且功能强大的卫浴配件，完善起室内的整体现代设计。

该系列产品的一大亮点是洗脸池或浴缸所使用的壁面配件，该配件的非水平设计明显打破了传统布局。墙式龙头的非杠杆手柄位于喷嘴下方的斜对角位置。大转盘和长长的出水管以精致微妙的平衡感打造出轮廓精美鲜明的产品，给人以轻盈的感觉。

Axia龙头具有防滑手柄和精刻凹槽，操作简单直观。先进的渐进式陶瓷滤筒可精确地输送水。另外，墙式龙头还具有精心设置的外环，只需一个动作就能完全控制水流，从关闭到水温最热。

中国马桶获得概念大奖

2020年红点设计概念大奖获奖共有参赛作品达4170件，仅42件作品获奖，比例仅占所有参赛作品的1%。其中，中国Tsang Ching Pun设计公司的Clean Seat公共马桶获奖。

Clean Seat的马桶盖会在第一个传感器(位于马桶盖的前部)感应到有人靠近时自动打开。第二个传感器感应到使用人离开后，马桶盖随即关闭并自动上锁。当马桶盖上锁时，马桶将开启自洁功能。经过这一系列的清洗消毒，在下一位使用者使用时，马桶座将再次达到了干净卫生的状态。
