【学习笔记】Android开发之kotlin语言（一）

2020-11-14
| 科技

2011年JetBrains公布了第一版的Kotlin语言，2012年将其开源，早期的Kotlin语言关注度不高，2016年JetBrains发布了Kotlin的1.0正式版，2017年Google正式宣布Kotlin成为Android开发的一级开发语言，Android Studio也加入了对Kotlin的支持。

相比较于Java语言，Kotlin语言的语法更简单，对于同样的功能使用kotlin语言，使用kotlin语言的代码量会比Java减少50%甚至更多；相比于Java的老语法，kotlin还增加了许多新式语法；kotlin还在安全性上下了功夫，杜绝了空指针这个全球崩溃率最高的异常。

《第一行代码Android》--郭霖


Kotlin Logo

fun main(){    val a=5    a = a+5    println(a)}

关于函数：

看一下这个题：构造一个函数，计算两个整数a,b的最大值。

用Java怎么写？

public TestJava{    public static void main(String[] args){          int a=5;         int b=6;          System.out.println(TestJava.add(a,b))；    }     public static double add(int a,int b){        return a>b?a:b    }}

用kotlin呢

fun main(){      val a=5      val b=6      println(add(a,b))}fun add(a:Int,b:Int) :Int =  if(a>b) a else b 

在kotlin语言中，当函数的代码只有一行时候，就不必写函数体了。


If 语句：

还是上面那个例子：

 switch(grade)      {         case 'A' :            System.out.println("优秀");             break;         case 'B' :           System.out.println("还可以");            break;         case 'C' :            System.out.println("良好");            break;         case 'D' :            System.out.println("及格");            break;         case 'F' :            System.out.println("你需要再努力努力");            break;         default :            System.out.println("未知等级");      }

在kotlin函数中，if的用法和Java是完全一样的，但是kotlin中的if是可以有返回值的这个返回值是每一个条件的最后一行代码的返回值举个例子：

val max = if (a>b){        a    }    else{       b  }

when语句

kotlin中有个独特的 when()语句，在功能上接近Java的switch语句，先用Java举个栗子：

 switch(grade)      {            case 'A' : System.out.println("优秀");                  break;            case 'B' : System.out.println("还可以");                break;            case 'C' :  System.out.println("良好");                break;            case 'D' :  System.out.println("及格");                 break;            case 'F' :   System.out.println("你需要再努力努力");                 break;            default :   System.out.println("未知等级");      }

是不是嫌长了，而且每一个case都有一个break，有点烦，when语句是这样用的：

when(grade){   'A' -> println("优秀")    'B' -> println("还可以")    'C' -> println("良好")   'D' -> println("及格")   'E' -> println("你需要再努力努力")    else -> println("未知等级")}

是不是简单多了，匹配值-> { 执行代码 }
