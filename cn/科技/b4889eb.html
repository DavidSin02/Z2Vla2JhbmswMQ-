JavaScript基础之入口函数-2020版


1.入口函数

 window.onload = function(){     内部放js}

这个函数的意思就是说，当我们页面加载完毕之后（就是说等页面的结构 样式 节点等加载完毕），采取执行函数体里面的js部分。也就是说可以使用此方法，就不用考虑JS语句的位置了，放在获取元素变量之前也行。

2.Js的书写位置

Js 的书写位置非常的自由。 也可以参照 css 的位置来分类。

2.1 行内式

一般情况，单双引号是一样 的 但是出现 了包裹的情况。

我们一般采取的是 外双内单 的格式。

2.2 内嵌式

使用 <script> 标签，在 HTML 页面中插入 JavaScript，。

合理放置脚本位置，以优化加载体验，js脚本放在 <body>标签闭合之前。

那些老旧的实例可能会在 <script> 标签中使用 type="text/javascript"。现在已经不必这样做了。JavaScript 是所有现代浏览器以及 HTML5 中的默认脚本语言。

2.3 外链式

<script type=”text/javascript” src=”xx.js”></script>

注意：这对标记之间不能写任何的东西。


如果涉及dom操作，且js在需要操作的dom元素之前取值，可以将相关代码放到window.onload函数里调用，这样就不会报错.

如果没有dom操作，放到前面逻辑没有问题，只是可能会影响性能和体验
