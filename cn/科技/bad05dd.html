EXCEL制作单元格颜色随机闪烁的字体

2020-10-29
| 科技

大家好,我是头条作者:小嘟嘟熊,这篇文章向大家分享如何用EXCEL自作单元格文字颜色随机闪烁,只需要17行代码,编写完了在EXCEL界面选中哪个单元格,哪个文字就用0.5秒速度闪烁,挺好玩了,感兴趣的朋友可以复制代码去试试:

Dim selCols As Integer

Dim selb As Boolean

Private Sub sleep(ByVal value As Single)

Dim tim As Long

tim = Timer + value

Do While Timer < tim

DoEvents

Loop

End Sub

Private Sub Worksheet_SelectionChange(ByVal Target As Range)

selCols = Target.Column

Do

DoEvents

sleep 0.5 '0.5切换颜色

Target.Cells(selCols).Font.Color = RGB(Rnd() * 255, Rnd() * 255, Rnd() * 255)

Loop '随机颜色'

End Sub
