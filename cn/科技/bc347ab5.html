建站经验丰富设置是否显示商务通对话框

2020-11-14
| 科技

企业网站的策划运营者们经常会遇到这样的问题，有时候需要将商务通弹出的对话框显示，有时候需要把它隐藏

比方有活动时显示对话框，活动过期时将对话框隐藏掉。这个需求比较棘手，需要有懂技术的人员长期手动维护js脚本才行

今天专门做网站的兼职小哥教您一招，您使用这个方法，即使是不懂技术的营销策划人员也可以轻易维护网站，设置对话框是否显示了。

var swtID=setInterval("update_swt()", 100);

function update_swt() {

if (typeof(LR_UserInviteDiv) != "undefined") {

LR_UserInviteDiv="<div style=\"position:relative;\"><a href=\"javascript:void(0);\" onclick=\"openZoosUrl();LR_HideInvite();return false;\"><img src=\""+imagepath+"meirongSwt.png\" width=\"422\" height=\"220\" border=\"0\" /></a><a style=\"position:absolute;top:0px;right:0px;z-index:999;width:30px;height:30px;\" href=\"javascript:void(0);\" onclick=\"LR_RefuseChat();LR_HideInvite();return false;\"></a></div>";

clearInterval(swtID);

}

}

商务通代码我们仍然破解的用官方推荐的商务通代码替换掉弹出框图片，因为商务通的默认图片实在是太难看了，您找商务通的客服人员让他们给换的话，价格还很高，5年前换一个对话框图片都是400起步，不知道现在是什么价格了。所以做为网站运营者的我们，只有选择撸起袖子自己干了！如果您遇到了类似问题欢迎在评论区给我留言，我来帮您处理！

以上代码设置是邀请框的显示时间，间隔多少秒弹出来。最上面is_show变量我们加了一个变量，我们正是要用这个变量控制对话商务通对话框是否显示弹出对话框。您一定看出来了，这是织梦后台，现在80%以上的企业网站都是基于织梦开发的。用织梦后台做网站的时候，建议选一个好的主机，阿里云主机是不错的选择，然后注意一些安全选项，织梦安全建站方面的注意事项，我会在稍后分享给大家。
