荣耀30系列首搭Magic UI 4.0，可升鸿蒙2.0

2020-10-29
| 科技

在今日举行的华为2020开发者大会上，华为消费者业务软件部总裁王成录在华为开发者大会上发表了《连接无限可能-全场景终端软件发布》主题演讲，EMUI 11和Magic UI 4.0将同步开启内测，Magic UI 4.0计划将于9月中旬开启内测招募活动，荣耀30系列和荣耀V30系列成为首批搭载机型，更多机型在后续升级计划中，而支持EMUI 11和Magic UI 4.0的机型未来都可以升级到鸿蒙操作系统2.0。


技术共享同源，荣耀创新引领产业

众所周知，脱胎于华为的荣耀，一直与华为保持着技术的共享与同步更新。特别是在软件系统方面，从荣耀Magic UI发布开始，荣耀每一代产品所搭载的系统都与华为最新系统同步更新。EMUI 11和Magic UI 4.0同步开启内测正是如此。

值得一提的是，此次备受大家关注的Magic UI 4.0在分布式技术、UX设计、隐私安全保护、AI智慧等方面都有重大升级。

最新发布的Magic UI 4.0，由于采用分布式技术，打破了设备间、应用间的隔离，将带来更多全场景业务体验。具体表现在不同设备之间的无缝连接和体验，像在多屏协同时，实现隐私投屏、跨屏拖拽等；在畅连通话中，实现与智慧屏进行拟人化交互，手机可以通过畅连消息对智慧屏进行互动和控制等。

在UX设计上，Magic UI 4.0不仅带来更多艺术风格的主题和DIY的AOD，并且继续将人因研究深入用到近百个用户场景上，特别是在动效和多感官协同上，为用户带来全新视听触交互体验。

而Magic UI 4.0还支持AI字幕（直出中文字幕+外语转中文字幕）和系统级翻译（浏览外文文章和图片轻松读懂内容）功能，大大提升了用户的体验。

除了软件系统外，荣耀还与华为保持技术同源，在手机的研发生产上与华为使用同一套技术标准，坚持以自研为第一推动力，带来各种极致黑科技。

以智能手机用户的痛点需求拍照为例，从2014年荣耀6 Plus推出世界首个双摄拍照系统，让手机能拍出背景虚化的大光圈照片；到2017年底，荣耀率先将AI技术应用于荣耀V10、荣耀10上，针对不同场景中的不同元素进行最优的色彩和光影优化；再到2019年荣耀V30系列引入联合SONY共同开发的RYYB颜色滤波器，以及通过荣耀30系列确立的超大底、超大像素、超感光、超长焦四位一体的系统级影像技术发展方向，荣耀已经完成从手机摄影发展的第一阶段的拿来主义、第二阶段的关键技术自研但仍需供应链技术反哺跃升到了对光学和算法全面系统的底层研究，旨在构筑一体化摄影系统的第三阶段，并将继续引领手机摄影的技术与应用趋势。

然而，除了手机的影像技术，荣耀早在 2016 年就推出一款主打人工智能体验的手机——荣耀Magic，给现在的手机 AI 应用场景奠定了方向。在手机散热方面，荣耀更是业界首创了“THE NINE 液冷散热技术”，第一次在手机上实现了 PC 级的散热水平。

当然，强大的技术实力还体现在方方面面，比如业内公认的自研麒麟芯片、GPU Turbo技术、方舟编译器、超级蓝牙、WiFi6+、40W超级快充、系统级影像技术等，使荣耀在市场极具竞争力，占据行业第一梯队席位，同时使得手机用户享受到最新技术带来的新鲜体验。

种种过往的事实证明，荣耀在与华为技术同源共享的同时，已经打造出属于自己的技术创新核心竞争力，并不断引领行业的发展。

双品牌战略，荣耀持续领先市场

除了技术同源和共享外，华为、荣耀还坚持双品牌战略，其中互补性和差异性让双品牌稳步发展的同时，又彰显出不同的个性和发展方式。例如荣耀的品牌定位是年轻人的科技潮牌，将“年轻”、“潮流”融入品牌和产品中；华为的品牌定位则是全球标志性的科技品牌。

以具体的产品荣耀30系列为例，虽然其和华为P40系列，都是主打潮流、摄影，采用相同的麒麟芯片，硬件配置几乎相同，但荣耀30的设计语言更加大胆，时尚、张扬和自信，除了引领潮流的钛空银和霓影紫配色，更是首次将荣耀HONOR大Logo放大，排列在相机矩阵的延长线上，辨识度极高。

此外，从荣耀30系列发布会“潮流大秀”的形式，同样体现出荣耀的科技潮牌定位。不同于业内多见的中规中矩的发布形式，荣耀将30系列发布会变成潮流秀，把手机的概念从工具延伸成时尚个性单品。

提及潮流，其实早在2017年，荣耀9就首创15层工艺3D全面激光玻璃，使得玻璃机身在光线下折射出金属光泽版的炫光，搭配奶奶灰配色，并借此引爆了年轻消费者。而2018年荣耀10首创的变色极光镀膜，利用光程差的原理，当光线直射在机身，不同的角度，不同的光线，都会呈现出不同的独特色彩，更是以引领潮流惊艳年轻用户。

正是由于双品牌战略，让荣耀和华为占据国内市场近半份额，尤其是在国内，甚至国际形势的影响，整个手机市场都处于一个相对低迷的环境之下，华为、荣耀反而是持续呈现逆势增长，例如今年发布的荣耀30系列，自发布以来一直是畅销爆款，首销线上就达到全平台销售额1分钟破3亿的好成绩；6.18期间，荣耀30斩获6.1-6.18京东平台手机累计安卓手机销售冠军实现6连冠，助力荣耀手机连续7天蝉联全平台累计手机销量冠军。

总结：荣耀依托技术溢出和潮流设计在各档位中布局打造最高价值产品，凭借与华为技术同源的先天优势，逐步打造出属于自己的技术创新核心竞争力， 同时又在做着“年轻人最感兴趣的事儿”，成功吸引转化大量其他手机品牌粉丝和大批追逐科技潮牌的年轻用户，用出色实力不断撬动年轻用户市场，期待荣耀未来带来的更多惊喜。
