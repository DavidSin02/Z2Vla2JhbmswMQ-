21世纪最畅销的F-16战机 机背保形油箱是标配 战力不输欧洲四代半

2020-11-14
| 科技

对于很多中国的军迷而言，F-16是一款经典第四代战斗机，这款战机自从1978年装备美国空军后，就成为全球多个国家和地区空军关注的焦点。从上世纪80年代初开始出口到欧洲多国，随后几十年在全球扩散。目前总产量接近5000架，装备全球30多个国家和地区，今天我们重点聊聊其中一个型号。

进入21世纪后，美国推出了当时最强的F-16E/F战机，也在就是大家常说的F-16E/F BlocK 60多用途战斗机。这款专为阿联酋国空军开发，与传统型号的区别是：换装了当时最新的AN/APG-80有源相控阵(AESA)雷达、全新航空电子设备、 保形油箱(CFT)，全新的通用电气F110-GE-132发动机。 总数一共80架，花费64亿美元。

以F-16E/F战机的性能，与欧洲三款四代半战机相比，其多用途作战能力也不落后。外界很多人以为，2000年后，这款战机也会像其他F-16战机一样大卖。然而最终这款战机只生产了80架就停产。让人想不到的是，F-16C/D的另外一款F-16C/D Block 52+却成为近20年最畅销的F-16战机。本文所有配图均为波兰空军F-16C/D Block 52战机，所有图片均为航空新视野原创摄影。

F-16C/D Block 52战机作为新一代的战斗机与其他型号相比有很多特点。首先是装备的国家多，例如希腊，波兰，埃及，以色列，新加坡，伊拉克，摩洛哥，印度尼西亚，巴基斯坦等国均装备大量的F-16C/D Block 52战机，或者一些性能相当于F-16C/D Block 52战机的F-16改进型号。

F-16C/D Block 52战机，也是目前装配机背保形油箱最多的F-16战斗机。大家有兴趣注意观察以上几个国家，很多时候保形油箱都是标配。这一定程度上影响了战机的机动性能。但是美国空军自用的F-16C/D Block 52战机并没有见到安装保形油箱的照片。这应该跟美国空军自身强大的空中加油能力有关。与这款战机性能相似的F-16C/D Block 50战机也很少见到这种情况。

F-16C/D Block 52战机均采用当时普惠公司最新的F100-PW-229IPE发动机。这台发动机最大加力推力达到129千牛。在经济性，可靠性以及寿命这些关键指标，至今依然是一个很高的水平。而传统的F-16C/D Block 50战机，采用的是通用电气F110-GE-129发动机，量大加力推力达到131千牛，例如驻日美军的F-16CM/DM战机和自卫队的F-2战机均采用这一等级的发动机。

关于这两型发动机的外观区别，网络上有很多对比的照片。笔者曾经发过很多F-16CM/DM和F-2战机的高清照片，有兴趣的看看那些战斗机的发动机喷口就明白了。这也是分辨这两个批次F-16战机最大的外观区别。

在航电上，F-16C/D Block 52战机最初使用的是美制的APG-8(V)5 多普勒雷达。近几年很多F-16C/D Block 52战机均升级到F-16C/D Block 52+战机的标准。也就是雷达也升级到最新的APG-8(V)10 。据称新的升级可以保证战机的空对空搜索范围提高到1.5倍以上。同时对地攻击能力也有很大的提高。虽然这款雷达并非有源相控阵雷达，但是依然不可轻视。

通过F-16C/D Block 52战机的改进，可以看到F-16战机虽然是一款很老的第四代战斗机，但是在设计之初就采取模块化的设计，特别是航电系统方面。另外战机的机载武器，也经过很多的升级，以对空导弹为例，从早期的AIM-9L近距空空导弹，到现役最新的AIM-9X和AIM-120C空空导弹。其空战能力都得到了显著的提高。这些较新批次的F-16战机，面对台风，阵风单机性能，谁强谁弱，谁又说得清楚呢？

目前多个国家和地区，计划把现役的F-16C/D战机，甚至还有一些F-16A/B战机均升级到F-16V的标准，主要的升级是采用全新的AN/APG-83有源相控阵雷达。新的任务计算机和电子战套件。全新的玻璃化座舱......这些升级是否给我国数百架早期的歼-10A和歼-10S战机带来什么启示呢？我国歼-10B/C的未来是否又有更强的升级呢？我们拭目以待。
