使用机器学习中的NLTK库通过5个步骤进行文本摘要生成

2020-10-29
| 科技

机器学习取决于 IT、数学和自然语言的交集，在大数据应用程序中会通常用到机器学习。本文将讨论 Python 编程语言和 NLTK 库，然后将它们应用于一个机器学习项目：通过文本提起摘要。NLTK被称为“使用Python进行计算语言学教学和工作的极好的工具”和“使用自然语言的极好的库”。

1. 创建词频表

我们从文本中为词频表创建一个字典。为此，我们应该只使用不属于stopWords 数组的单词。Python代码如下：

def _create_frequency_table(text_string) -> dict: stopWords = set(stopwords.words("english")) words = word_tokenize(text_string) ps = PorterStemmer() freqTable = dict() for word in words: word = ps.stem(word) if word in stopWords: continue if word in freqTable: freqTable[word] += 1 else: freqTable[word] = 1 return freqTable

我们将这个方法应用于text_string，它可以是新闻文章，书籍页面或电子邮件等。词频表应该看起来像上面的图像。

3.给句子打分:词频

我们使用Term Frequency方法对每个句子进行评分。

基本算法：通过单词对句子进行评分，在句子中添加每个不间断单词的频率。

def _score_sentences(sentences, freqTable) -> dict: sentenceValue = dict() for sentence in sentences: word_count_in_sentence = (len(word_tokenize(sentence))) for wordValue in freqTable: if wordValue in sentence.lower(): if sentence[:10] in sentenceValue: sentenceValue[sentence[:10]] += freqTable[wordValue] else: sentenceValue[sentence[:10]] = freqTable[wordValue] sentenceValue[sentence[:10]] = sentenceValue[sentence[:10]] // word_count_in_sentence return sentenceValue

请注意，我们的分数算法的一个潜在问题是长句比短句更有优势。

为了解决这个问题，我们将每个句子得分除以句子中的单词数。您可以使用任何其他归一化方法。

请注意，这里的sentence[：10]是任何句子的前10个字符，这是为了在保存字典的键时节省内存。字典看起来像下面。

4. 找到阈值

在这里，我们将句子的平均分数视为阈值。您可以使用其他方法来计算阈值。

def _find_average_score(sentenceValue) -> int: sumValues = 0 for entry in sentenceValue: sumValues += sentenceValue[entry] # Average value of a sentence from original text average = int(sumValues / len(sentenceValue)) return average

5. 生成摘要

基本算法：如果句子分数大于平均分数，则选择摘要的句子。

def _generate_summary(sentences, sentenceValue, threshold): sentence_count = 0 summary = '' for sentence in sentences: if sentence[:10] in sentenceValue and sentenceValue[sentence[:10]] > (threshold): summary += " " + sentence sentence_count += 1 return summary
