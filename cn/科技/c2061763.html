教你用云态服务器搭建一个无视攻击的网站

2020-11-14
| 科技

很多朋友都被攻击所困扰，这次教大家怎么用云态的服务器搭建一个无视攻击的网站。云态的服务器有首月半价优惠，很实惠

首先安装一个宝塔

Centos安装命令：

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh

Ubuntu安装命令：

wget -O install.sh http://download.bt.cn/install/install-ubuntu.sh && sudo bash install.sh

Debian安装命令：

wget -O install.sh http://download.bt.cn/install/install-ubuntu.sh && bash install.sh

Fedora安装命令:

wget -O install.sh http://download.bt.cn/install/install.sh && bash install.sh

安装好宝塔后安装lnmp

然后安装云锁（屏蔽CC攻击和其他常见攻击）

X86：

wget http://download.yunsuo.com.cn/v3/yunsuo_agent_32bit.tar.gz && tar xvzf yunsuo_agent_32bit.tar.gz && chmod +x yunsuo_install/install && yunsuo_install/install

X64：

wget http://download.yunsuo.com.cn/v3/yunsuo_agent_64bit.tar.gz && tar xvzf yunsuo_agent_64bit.tar.gz && chmod +x yunsuo_install/install && yunsuo_install/install

在官方注册一个帐号：

http://v3.yunsuo.com.cn/register/step1

然后安装云锁PC控制端添加服务器：

http://download.yunsuo.com.cn/v3/%E4%BA%91%E9%94%81%E5%AE%A2%E6%88%B7%E7%AB%AF(%E5%AE%89%E8%A3%85%E5%9C%A8PC%E4%B8%8A).exe

添加好服务器后就可以在使用网页版管理

http://v3.yunsuo.com.cn/login

然后去cloudflare注册一个帐号（用于防御DDOS）

https://dash.cloudflare.com/signup

注册完成后点击Add Site添加一个域名

选择免费方案

然后把域名NS改为

cruz.ns.cloudflare.com
pete.ns.cloudflare.com

如果不会更改，还请自行百度

添加完成后，会受到右键

添加域名的时候记得保持小云朵是黄色的,黄色为开启
