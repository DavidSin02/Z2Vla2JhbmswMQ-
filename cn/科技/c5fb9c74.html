Abaqus 2019 有限元分析软件

2020-11-14
| 科技

Abaqus 是一款功能强大的用于有限元的分析软件，其应用汽车、消费品、航空航天、能源和生命科学等多个领域。目前软件的最新版已经发布，新版在诸多功能上进行了加强，Abaqus 提供了涵盖工业应用的广阔频谱常规和复杂的工程问题的强大而完整的解决方案。Abaqus 的探索真实世界的物理行为，产品和材料，以提高性能，可靠性和安全性，同时减少开发时间和成本。是一款对未来具有战略性的产品。


SIMULIA逼真的模拟解决方案，SIMULIA使虚拟测试成为一种标准的业务实践，可以提高产品性能，减少物理原型并推动创新。搭载了3D体验平台，SIMULIA提供逼真的模拟应用程序，使用户探索的产品，自然与生活真实世界的行为。SIMULIA应用程序在投入物理原型之前，加快了评估材料和产品的性能，可靠性和安全性的过程。SIMULIA提供先进的仿真产品组合，用于多物理场，流程集成和优化：Abaqus FEA，fe-safe，Isight，Tosca，Simpack，Simpoe和SIMULIA SLM。此外，SIMULIA V5和V6解决方案可为目前在CATIA V5或V6环境中工作的设计人员和设计工程师提供逼真的仿真。面向仿真用户和企业的3DEXPERIENCE平台提供了演化功能，以扩展当前的专家工具集。结合SIMULIA的技术Abaqus，fe-safe，Isight和Tosca来展示真实模拟的全部潜力。使用Extended Packaging可以访问所有“产品组合的力量”技术。

软件功能

静态应力/位移分析：包括线性，材料和几何非线性，以及结构断裂分析等

动态分析粘弹性/粘塑性响应分析：粘塑性材料结构的响应分析

热传导分析：传导，辐射和对流的瞬态或稳态分析

质量扩散分析：静水压力造成的质量扩散和渗流分析等

耦合分析：热/力耦合，热/电耦合，压/电耦合，流/力耦合，声/力耦合等

非线性动态应力/位移分析：可以模拟各种随时间变化的大位移、接触分析等

瞬态温度/位移耦合分析：解决力学和热响应及其耦合问题

准静态分析：应用显式积分方法求解静态和冲压等准静态问题

退火成型过程分析：可以对材料退火热处理过程进行模拟

海洋工程结构分析：

对海洋工程的特殊载荷如流载荷、浮力、惯性力等进行模拟

对海洋工程的特殊结构如锚链、管道、电缆等进行模拟

对海洋工程的特殊的连接，如土壤/管柱连接、锚链/海床摩擦、管道/管道相对滑动等进行模拟

水下冲击分析：

对冲击载荷作用下的水下结构进行分析

柔体多体动力学分析：对机构的运动情况进行分析，并和有限元功能结合进行结构和机械的耦合分析，并可以考虑机构运动中的接触和摩擦

疲劳分析：根据结构和材料的受载情况统计进行生存力分析和疲劳寿命预估

设计灵敏度分析：对结构参数进行灵敏度分析并据此进行结构的优化设计

软件除具有上述常规和特殊的分析功能外，在材料模型，单元，载荷、约束及连接等方面也功能强大并各具特点：

材料模型：定义了多种材料本构关系及失效准则模型，包括：

弹性：线弹性，可以定义材料的模量、泊松比等弹性特性

正交各向异性，具有多种典型失效理论，用于复合材料结构分析

多孔结构弹性，用于模拟土壤和可挤压泡沫的弹性行为

亚弹性，可以考虑应变对模量的影响

超弹性，可以模拟橡胶类材料的大应变影响

粘弹性，时域和频域的粘弹性材料模型


安装步骤

1、准备好Abaqus 2019的安装包，打开文件夹“_SolidSQUAD_”，解压“_SolidSQUAD_.7z”；

2、打开解压之后的“_SolidSQUAD_”，打开文件夹“Server”；

3、解压“SSQ_UniversalLicenseServer_Core_20180127074300.zip”和“SSQ_UniversalLicenseServer

_Module_DSSimulia_20180127185300.zip”；

4、打开解压之后的“SSQ_UniversalLicenseServer_Core_20180127074300”文件夹和“SSQ_UniversalLicenseServer_Module_DSSimulia_20180127185300”；

5、复制文件夹“Vendors”，粘贴到文件夹“SolidSQUAD_License_Servers”；

6、复制粘贴完成之后的文件夹“SolidSQUAD_License_Servers”，粘贴到C盘根目录之下；

7、打开粘贴之后的文件夹“SolidSQUAD_License_Servers”，右击“install_or_update.bat”，以管理员身份运行；

8、出现以下红框内容，表示SolidSQUAD_License_Servers已经开启，如果想要关闭服务，右击“uninstall.bat”，以管理员身份运行，出现以下红框内容，表明SolidSQUAD_License_Servers已经关闭；

9、需要对Abaqus 2019创建环境变量，跳过FlexNET License server的安装，打开“此电脑”，点击“属性”；

10、点击“高级系统设置”，点击“环境变量”；

11、在“系统变量”点击“新建”，变量名输入“NOLICENSECHECK”，变量值输入“true”，输入完成之后点击“确定”；

12、确认无误，点击“确定”；

13、返回Abaqus 2019安装包，解压“DS.SIMULIA.SUITE.2019.WIN64.iso；

14、打开文件“1”，右击“setup.exe”，以管理员身份运行；

15、进入Abaqus 2019安装环境，点击“下一步”；

16、选择Abaqus 2019的安装组件，默认即可，FlexNET License server是不安装的，点击“下一步”；

17、确认Abaqus 2019的安装组件信息，点击“安装”；

18、首先先安装Abaqus Simulation Services，修改其安装位置，点击“下一步“；

19，确认Abaqus Simulation Services安装目录，确认无误点击“下一步”；

20、点击“安装”，Abaqus Simulation Services进入安装；

21、Abaqus Simulation Services正在安装；

22、Abaqus Simulation Services安装完成，点击“关闭“；

23、进入Abaqus Simulation Services CAAPI的安装，

24、查看Abaqus Simulation Services CAAPI的安装位置，点击“下一步“；

25、确认Abaqus Simulation Services CAAPI安装目录，确认无误点击“下一步”；

26、点击“安装”，Abaqus Simulation Services CAAPI进入安装；

27、Abaqus Simulation Services CAAPI正在安装；

28、Abaqus Simulation Services CAAPI安装完成，点击“关闭“；

29、点击“浏览”，将Abaqus 2019安装包中的镜像文件夹“DS.SIMULIA.SUITE.2019.WIN64”中的文件夹“2”进行添加，添加完成后，点击“确定”；

30、进入Abaqus CAE 2019的安装环境，修改其安装位置，点击“下一步”；

31、设置Abaqus CAE 2019的许可证类型，选择“SIMULIAFLEXnet”，点击“下一步”；

32、在Abaqus CAE 2019许可证界面中的【License Server 1】输入“27800+@+计算机名称”，点击“下一步”；

33、进入Abaqus解算器的安装，默认其安装位置，点击“下一步”；

34、设置Abaqus CAE 2019的文件保存位置，默认保存在C盘，可根据实际情况修改其文件默认保存位置，也可默认原设置路径，点击“下一步”；

35、设置Abaqus CAE 2019外部插件目录路径，可修改其目录路径，也可默认设置，点击“下一步；

36、默认选择“Connect to searchable documentation on help. 3ds. com”，点击“下一步”；

37、设置Abaqus CAE 2019的数据保存地址，这里修改其保存位置，基本上所有的数据保存地址都是可以默认的，点击“下一步”；

38、确认Abaqus CAE 2019的安装程序信息，点击“安装”；

39、Abaqus CAE 2019正在安装；

40、Abaqus CAE 2019安装完成，点击“关闭”；

41、进入fe-safe2019的安装环境，修改其安装位置，点击“下一步”；

42、选择fe-safe2019的安装部件，这里默认设置，点击“下一步”；

43、确认fe-safe2019的安装部件信息，点击“安装”；

44、fe-safe2019正在安装；

45、fe-safe2019安装完成，点击“关闭”；

46、进入Tosca 2019的安装环境，修改其安装位置，点击“下一步”；

47、选择文档配置——“为本地安装的文档输入文件路径”，点击“下一步”；

48、设置“SIMULIA文档路径”，默认保存到C盘，这里设置为F（修改盘符级即可），点击“下一步”；

49、点击“继续”；

50、选择Tosca2019的安装部件，这里默认设置，点击“下一步”；

51、可根据实际情况选择接口类型，这里默认选择“ABAQUS接口”，点击“下一步”；

52、这里不做修改，点击“下一步”；

53、确认Tosca 2019的安装部件信息，点击“安装”；

54、Tosca 2019正在安装；

55、Tosca 2019安装完成，点击“关闭”；

56、点击“浏览”，将Abaqus 2019安装包中的镜像文件夹“DS.SIMULIA.SUITE.2019.WIN64”中的文件夹“”进行添加，添加完成后，点击“确定”；

57、进入Isight 2019的安装环境，修改其安装位置，点击“下一步”；

58、选择Isight 2019的安装部件，这里默认设置，点击“下一步”；

59、不勾选“Launch TomEE Configuration Ultility”，点击“下一步”；

60、点击“下一步”；

61、选择“Enter a file path for locally installed documentation”，点击“下一步”；

62、设置“SIMULIA文档路径”，默认保存到C盘，这里设置为F（修改盘符级即可），点击“下一步”；

63、点击“Continue（继续）”；

64、确认Isight 2019的安装部件信息，点击“安装”；

65、Isight 2019正在安装；

66、Isight 2019安装完成，点击关闭；

67、Abaqus 2019软件全部安装完成，点击“关闭”；

68、打开开始菜单，点击运行“Abaqus CAE”，查看其破解情况；

69、打开开始菜单，点击运行“fe-safe2019”，查看其破解情况；

70、打开开始菜单，点击运行“Design Gateway”，查看其破解情况；

71、打开开始菜单，点击运行“Tosca Structure GUI”，查看其破解情况；
