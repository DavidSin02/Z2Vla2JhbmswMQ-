Excel VBA 学习 FillFormat 对象、ColorFormat 对象

2020-11-14
| 科技

代表形状的填充格式。形状可以有纯色、渐变、纹理、图案、图片或半透明填充。

方法

OneColorGradient 方法：将指定填充设置为单色渐变。

Patterned 方法 ：将指定填充设置为一种图案。

PresetGradient 方法：将指定填充设置为一个预设的渐变。

PresetTextured 方法 ：将指定填充格式设为预设纹理。

Solid 方法：将指定的填充格式设置为均匀的颜色。可用本方法将带有渐变、纹理、图案或背景的填充格式转换为单色的填充格式。

TwoColorGradient 方法：将指定填充设置为双色渐变。

UserPicture 方法 ：用图像填充指定的形状。

UserTextured 方法 ：用一张图像的平铺形式填充指定形状。 如果您想要用一张大图像填充形状，请使用**UserPicture** 方法。

属性

Application 属性：在不使用对象识别符的情况下使用时, 此属性返回代表 Microsoft Excel 应用程序的**Application** 对象。 如果与对象识别符一起使用，则此属性返回代表指定对象的创建者的 Application 对象（可以将此属性与 OLE 自动化对象一起使用以返回该对象的应用程序）。 只读。

BackColor 属性 ：返回或设置一个**ColorFormat** 对象，代表指定的填充背景色。

Creator属性：返回一个 32 位整数，该整数指示在其中创建了此对象的应用程序。 只读 Long。

ForeColor 属性 ：返回或设置一个**ColorFormat** 对象，代表指定的前景填充色或纯色。

GradientAngle 属性：返回或设置指定填充格式的渐变填充的角度。 读/写

GradientColorType 属性：返回指定填充的渐变颜色类型。 只读的**MsoGradientColorType**。

GradientDegree 属性 ：返回指定底纹单色填充的渐变程度的浮点值从 0.0 （暗） 到 1.0 （亮）。 只读单个。

GradientStops 属性 ：返回渐变填充的结束点。 只读。

GradientStyle 属性 ：返回指定填充的渐变样式。 只读**MsoGradientStyle**。

GradientVariant 属性 ：从 1 到 4，作为整数值返回指定的填充格式的底纹变量。 此属性的值对应于填充效果对话框中过渡选项卡上的渐变变量 （从左到右，从上到下编号）。

Parent 属性：返回指定对象的父对象。

Pattern 属性：返回或设置一个**MsoPatternType** 值，该值代表填充图案。

PictureEffects 属性：返回一个对象，该对象代表指定填充格式的图片或纹理填充。 只读

PresetGradientType 属性 ：返回指定填充的预设渐变类型。 只读**MsoPresetGradientType**。

PresetTexture 属性 ：返回指定填充的预设纹理。 只读**MsoPresetTexture**。

RotateWithObject 属性 ：返回或设置与对象应旋转的填充样式。 读/写**MsoTriState**。

TextureAlignment 属性 ：返回或设置指定FillFormat对象的文本对齐方式。 读/写。

TextureHorizontalScale 属性 ：返回或设置为水平缩放FillFormat对象的文本值。 读/写 Single。

TextureName 属性 ：返回指定填充的自定义纹理文件的名称。 只读 String。

TextureOffsetX 属性 ：返回指定填充的偏移量 X 值。 读/写单个。

TextureOffsetY 属性 ：返回指定填充的偏移量 Y 值。 读/写单个。

TextureTile 属性 ：返回指定填充的纹理平铺样式。 读/写**MsoTriState**。

TextureType 属性 ：返回指定填充的纹理类型。 只读**MsoTextureType**。

TextureVerticalScale 属性 ：返回指定填充的纹理垂直缩放。 读/写单个。

Transparency 属性 ：返回或设置指定填充的透明度作为值从 0.0 （不透明） 到 1.0 （清除）。 读/写 Double。此属性的值仅影响均一颜色填充和线条的外观。对图案线条及渐变、图案、图片和纹理填充的外观无效。

Type属性：返回一个**MsoFillType** 值, 该值代表填充类型。

Visible 属性：返回或设置**MsoTriState** 值，它确定对象是否可见。 读/写。

ColorFormat 对象

代表单色对象的颜色、带有渐变或图案填充的对象的前景或背景色，或者指针的颜色。

Application 属性：在不使用对象识别符的情况下使用时, 此属性返回代表 Microsoft Excel 应用程序的**Application** 对象。 如果与对象识别符一起使用，则此属性返回代表指定对象的创建者的 Application 对象（可以将此属性与 OLE 自动化对象一起使用以返回该对象的应用程序）。 只读。

Brightness 属性：返回或设置指定对象的发光度。 读/写

Creator 属性：返回一个 32 位整数，该整数指示在其中创建了此对象的应用程序。 只读 Long。

ObjectThemeColor 属性：返回或设置一种映射到主题配色方案的颜色。 读/写**MsoThemeColorIndex**。

Parent 属性：返回指定对象的父对象。 只读。

RGB 属性：返回或设置一个Long值, 它代表指定颜色的红-绿-蓝值。

SchemeColor 属性 ：返回或设置一个Integer值, 它代表颜色对象的颜色, 作为当前颜色中的一个索引。

TintAndShade 属性：返回或设置一个变亮或加深颜色的Single 。

​Type 属性：返回一个**MsoColorType** 值, 它代表颜色格式类型。
