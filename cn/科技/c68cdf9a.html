胡凤芩老师：快递行业成本管理（非财）

2020-11-14
| 科技

【课程收益】

1．帮助快递从业人员快速认识行业成本管理，从中找到自己涉及到的成本管理领域范围；

2.帮助学员掌握营业网点成本的构成与控制方法，从而降低营业网点的经营成本；

3.帮助学员学会从物流快递业全方位、全角度的管理去降低成本；

4.把握成本管理与企业现实要求中一些冲突处理；

【课程对象】快递营业网点经理、营业部负责人、主管

【培训方式】课程讲授+案例分析+工具+模版

【课程时间】2天

【课程大纲】

一、 认识快递行业成本的构成与分类

1. 引言：如何做到越“省”越“赚”钱？

2. 关于成本的认知

3. 关于成本管理的“指导思想”

4. 关于成本控制认知

5. 快递行业成本构成

－什么是快递的显性成本

－什么是快递的隐性成本

6. 快递行业的常用分类认知

－主营业务成本

－营业费用

－管理费用

7. 为什么要从管理的角度认识成本管理？

二、 如何在企业中建立成本控制意识、路径与制度

1. 观点：没有做不到的，只有想不到的。

2. 在企业当中培养成本意识，做到各个部门无缝对接

3. 在企业当中建立成本控制路径，帮助企业梳通关节

－案例：超级“凶”的财务主管

－要求员工在特定的路径内执行

－梳通企业各个节点的成本问题

4. 在企业当中建立制度，帮助企业持续进行成本控制

－成立成本库

－组织成本绩效评定（KPI和BSC）

5. 成本控制一句话：没有规矩，不成方圆

三、 快递营业网点成本的构成与控制

1. 案例：著名快递企业的成本控制带来巨额利润

2. 快速梳理快递营业网点的成本构成与控制环节

－一一梳理，不遗漏

3. 如何在企业当中一一实现成本控制

1) 人工成本是重中之重（工资、提成、福利设计原则）

2) 油耗是一笔大支出

3) 路桥费是天天支出

4) 车辆保险费是否可以降一降

5) 车辆维修费如何保持安全最高水准

6) 物料管理是看不见的成本消耗（采购、发放、使用）

7) 水电费是不是负担

8) 场地租凭是否折价统计（专指邮政）

9) 通讯费用在行业中的占比

10) 赔偿费用是否做到可控性

11) 企业固定资产及低值易耗品如何管理

4. 营业网点成本控制与管理带来的最终意义

四、 快递企业成本管理的注意事项

1. 讨论：成本管理的宗旨是什么？

2. 快递成本管理的三项基本原则

－服务质量原则

－业务量稳定原则

－企业发展原则

3. 成本管理与满意度之间的关系

－员工与企业之间如何取得平衡

－客户与企业之间如何取舍

4. 成本控制与企业收入增长的比例

－是不是收入越高，成本投入越高

5. 结论：成本管理是为企业服务的

五、 如何从管理上降低快递企业成本

1. 案例分析：成本管理与“5S”的神秘关系

2. 通过快递标准化作业降低成本

－餐饮、快消的标准化作业给我们带来的启示

－快递标准化作业的投资

－标准化作业带来持续回报

3. 通过提高客户服务水平降低成本

－从企业到客户到作业者之间的协调

－找到客真正满意的标准，不浪费

4. 通过流程的优化来降低成本

－上一道工序影响下一道工序的质量

－分析寄递流程来降低成本

－如：通过削减问题件（包裹）降低成本

－营业网点的问题件（包裹）带来巨大损失

－如何降低问题的发生

－如何及时处理问题，降低成本

5. 降低成本的一些手段介绍

－借助现代信息系统降低成本

－利用外包降低成本

－建立路线优化机制

－通过数据化管理降低成本

－通过效率化的配送降低成本

－提升仓储管理降低成本（参考）

－分部门进行降低成本项目比赛

6. 回顾：没有最好的，只有继续改善。
