业余无线电：新手入门如何选择设备

2020-10-29
| 科技

很多爱好者刚开始玩无线电的时候不知道如何选择设备，下面BG8BXM跟大家介绍如何合法使用业余无线电，以及如何选择手台、车台、天馈等等（个人见解，仅供参考。）

合法使用业余无线电设备

根据《中华人民共和国无线电管理条例》第二十七条设置、使用无线电台（站）应当向无线电管理机构申请取得无线电台执照，第二十九条 申请设置、使用业余无线电台的，应当熟悉无线电管理规定，具有相应的操作技术能力，所使用的无线电发射设备应当符合国家标准和国家无线电管理的有关规定。

哪种条件的对讲机不需要执照？

2001年12月，信息产业部无线电管理局发布通知：低功率民用对讲机市场全面放开。通知的具体内容是：设置和使用公众对讲机，不需领取电台执照，免收频率占用费。

公众对讲机是指：发射功率不大于0．5W，工作于指定频率的无线对讲机，其无线电发射频率、功率等射频技术指标须符合通知要求。

工作频率（单位：MHz）: 409.7500 ; 409.7625 ; 409.7750 ; 409.7875 ; 409.8000 ; 409.8125 ; 409.8250 ; 409.8375 ; 409.8500 ; 409.8625 ; 409.8750 ; 409.8875 ; 409.9000 ; 409.9125 ; 409.9250 ; 409.9375 ; 409.9500 ; 409.9625 ; 409.9750 ; 409.9875 （共20个频点）

使用公众对讲机有以下几点需要特别注意：

1、公众对讲机禁止在机场和飞行器上使用，禁止与公众电话网、公众移动通信网互联。

2、不得擅自更改发射频率、加大发射功率（包括额外加装射频功率放大器），不得擅自外接天线或改用其它发射天线。

3、使用时不得对各种合法无线电业务产生有害干扰，一旦发现有害干扰，应立即停止使用，并采取措施消除干扰后方可继续使用。

如何选择设备？

1：购买有核准代码的电台：因业余无线电爱好者使用的设备均需要办理电台执照，所以在选购设备时需首选具备核准代码的电台。

核准码查询网址：www.srrc.org.cn/WP_Search.aspx

具有核准代码的电台，在机器串号区域有标注

2：根据自己的经济能力选择：有些爱好者一直犹豫是否加入这个爱好群体，是因为有个误区，以为业余无线电是“无线垫”，BG8BXM认为其实不是的，看你怎么玩，想怎么玩，如果经济拮据，一个国产手台1百多块可以在U/V段中继上CQ和通联，也可以在此基础上自制一个八木天线进行业余卫星通信或接收空间站的SSTV(慢扫描电视)，觉得手台功率小，家里可以买一台二手的GM300或1907一样的可以坐台，短波电台也是，网上有很多经典的二手机器如725A，FT-80C等等成色好一点的也就1000块钱，所以想象比现实可怕，只要你想做没人能阻止你。

这位爱好者的通信车酷吧

如何选择手台（手持电台：对讲机）？

手持电台的种类特别多，功率以2-7w居多，没有阻挡的条件下直呼距离可以达到3-10公里，如果有一定的高度外接车载或玻璃钢天线可以达到几十公里，通过中转台（中继台）也能完成几十上百公里的通联，我觉得在功率上5w就可以了。

一、从置频方式来讲分为写频机和手置频机；

1、写频机是固定频道，以16频道居多，很多机器没有液晶屏幕，功能单一，需要通过电脑写频，适用于只用固定频率的爱好者，不适合经常客席异地工作的爱好者，每次出差前先把该区域的频率先写一遍，确实不太方便，常规机型价格从100块到几千块不等；

常规写频机的控制只有音量开关旋钮和频道旋钮

2、手置频机是指可以通过电台自带的液晶屏幕，对菜单进行频率及其他功能的设置，可以根据自己的需要马上进行调整，非常方便，常规机型价格从100块到几千块不等；

最便宜的双段手置频机器之一，也有DMR数字版

二、从频率范围来讲，分为单段机、双段机或多段全段机；

1、单段机：指的是单一频率范围收发的机型，比如29MHz、50MHz、V段144MHz，U段430MHz等等，U、V段单段机在写频机中居多，也有手置频的单段机；

2、双段机：大多时候指的是支持在V段144MHz，U段430MHz等业余频率收发的机器，有单显单待机的也有双显双待的（双显双待指：两个频率待机，监听两个频率）。

FT-4X：不到600块的双段单显双段手台

国产欧讯的KG-UV9Dplus在业余无线电圈口碑不错，UV双段双显，多频段接收（含航空频段，有核准

三、从模式分类来讲，分为模拟机和数字机；

1、模拟机：指的是收发模式为FM模式，由于模拟对讲机抗干扰低，容易被窃听，信道资源有限，根据国家的相关管理政策，模拟对讲机讲逐步被淘汰。由于数字模式各厂家不统一，而且数字机价格稍贵，所以目前市面上销售的大多还是模拟机。

2、数字机：指的是用数字技术进行设计的数字对讲机。数字对讲机则是将语音信号数字化，要以数字编码形式传播，也就是说，对讲机传输频率上的全部调制均为数字。为了使模拟对讲机能平稳过度，所以现在很多数字机都兼容模拟模式。但是数字模式各厂家没有统一，目前市面上常用的数字机模式有：C4FM,D-Star,DMRPlus,dPMR和P25。所以在选购数字机的时候，根据自己地区的应用情况，如果你当地是C4FM的中继，那么久选择C4FM的机型，如果是DMR的，就选择DMR的机器。

ID-31E U段D-STAR防水GPS数模手台，价格约1700左右（有核准码）

P25数字机：XTS5000

四、如果要玩APRS，建议买带GPS的机型；

FT2D触摸屏C4FM数模手台，多频段接收，带APRS（有核准码）

TH-D74，APRS,D-STAR数模双用手台

如何选择车台（车载电台）

车载台U/V段机一般是25瓦~50瓦，车载短波台5瓦~200瓦都有，价格从几百块到几千块不等，除了手持电台的几种类别，车载台分为面板一体机和面板分体机。

面板一体机：面板与机身不能分离，价格相对便宜一点，以单段机居多，在车上安装不太美观但适用于作为基地台，不适合外观控和强迫症的爱好者。

GM300只有二手货，不到500块就能买到25W或45W带信令版本（无核准码））

销量最高的单U段面板一体机之一：FT1907，购买时注意区分仿版（有核准码）

M8668i数模两用车台(有核准码)

面板分体机：指面板可以和主机进行分离安装的车载电台，很多爱好者喜欢将主机装在驾驶位或副驾驶位的座椅下面，也有些爱好者将主机装在后备箱，面板通过吸盘或支架装在中控台不阻挡视线又方便操作的地方，通过一根数据线将主机与面板连接就行，手咪也可以通过延长线搞定。

双段双显C4FM数模车台，带GPS可玩APRS，第1\2代产品不太稳定，价格4千左右（有核准码）

如何选择短波电台

操作短波电台必须持有B类或C类操作证书且在国家规定的频率范围内进行工作，A类操作证书设台半年后可以申请参加B类能力验证。

短波电台分为基地台和便携（车载）台，原身功率从5W~200W不等，价格也是从1000元至几万不等。

一、QRP短波电台：有专业的QRP机，发射功率不超过5W，当然其他大功率机器可以通过调整功率作为QRP电台使用。

千元级别的QRP短波电台

FT-818，可在U/V段收发

国外爱好者的电台通信箱，所有设备都在一个箱子里

二、车载（便携）短波：除了前面介绍的IC-7100和FT-857以下机型也是不错的选择

协谷G90S 20瓦便携短波电台，面板可分离，国外爱好者评价也挺高的（有核准码）

如何选择天线

一、手台天线，长一点的天线肯定比短的好，如果是定制天线，国内爱好者BG7IOO的产品挺不错的，其他成品天线有拉杆的或鞭状的，建议不超过40cm。

bg7ioo的手台定制天线

要进行业余卫星通信，需要自制或购买八木天线：

玩卫星通信及接收空间站SSTV的，建议使用定向八木天线

二、车载台天线，根据你的电台情况，如果是单段机建议购买单段天线效率更高，双段机就购买双段天线，如果是短波电台可以配多波段的天线，不想太张扬也可以配2根单段的车载短波天线（14MHz和7MHz），icom的机器可以用BI7KJB的KB2，挺不错的。

健博通的单段天线不到100块

常规的双段天线以老鹰和新名的居多，再贵一点的就是钻石的。

性价比高的双段天线：770，价格不到150

车载移动台不建议使用U/V段玻璃钢天线，效率不高。

ICOM车载短波天线价格要1万多

BI7KJB生产的KB2车载短波天线，使用ICOM或柯顿电台可以自动调谐

车载电台天线配合卡片或吸盘使用，行李架安装可以配自动放倒器，越重的天线对卡片和吸盘的质量要求越高。

三、基地台U/V段天线：

可以使用车载台天线替代，但是建议使用玻璃钢全向天线，长度在2米~4米的最好，国产品牌健博通、华鸿以及任丘天悦的都不错，也有很多爱好者DIY的。

四、便携短波天线：

便携短波天线比较多的选择，长线天线、倒V天线只要有撑杆或树干就可以架设，如果是在空旷地pac-12这款GP天线以及便携正V天线也是不错的选择。

Gipsy吉普赛5-50MHz卷绕式水平偶极天线 DIPOLE便携短波天线

野外利用撑杆架设倒V天线

五、基地短波天线：

（不管是便携还是固定台的天线，很多国内爱好者研发的产品都很不错，性价比高）

短波天线的种类比较多，常见的有垂直天线（GP），偶极天线（DP）以及八木天线（YAGI），基地短波天线的选择根据自己的架设条件进行选择，如果条件允许，建议架设八木天线。

GP天线是垂直的，适合DX通信，但同一架设环境比DP天线的底噪高1-2个点。

K4GP，四波段GP天线，我现在家里就使用的这款天线，价格1千块左右

DP天线有很多演变：比如倒V、正V、单极、偏馈等等，基地台天线架设较多的是正V及倒V天线。

bi7kjb开发的KB2-V4四波段正V天线，价格约900左右，可以固定架设也可野外设台使用

八木天线属于定向天线，体积比较大，对架设场地要求高，有很好的方向性，较DP天线有更高的增益，常配合旋转器使用，价格上也不一

BG7PNV开发的A5DX短波3波段5单元八木天线，价格3千多块，相比进口八木实惠不少

如何选择馈线

馈线在业余无线电系统中相当的重要，选择好馈线能提高你的收发效率，选择适当的线径可以避免驻波太大，导致机器发热和长时间工作烧毁电路功放系统。

业余无线电常用的同轴线有50欧、75欧、100欧三种标称阻抗。 业余通讯常用50欧，剩下的就是选择馈线的粗细（-3 -5 -7等），所以在你购买馈线的时候经常看到的规格是50-3，,50-5或50-7等等。

馈线的直径选择与频率、功率与馈线长度有关，比如100W短波设备如果20米内可以使用-3或-5的馈线，如果距离太长建议使用-7的纯铜线

50瓦以内UV设备建议使用-5的馈线，距离太长以及中继台建议使用馈管。

车载电台可以使用双屏蔽的50-3的特氟龙馈线

野外设台建议购买野外专用的超柔纯铜馈线。

八哥电缆销售的50-3纯铜多芯超柔馈线，适合野外使用
