基本技能学了都能干嘛？？？

office三大件就不用说了，日常办公出个合同，搞个报表，做个路演都用得到。

下面要说的是Web，UI和视频拍摄剪辑技能：

  • Web：

可以做网站，后期实现技术，以及个人IP的呈现一部分要沉淀到aurislee.com 这个网站上（网站，微博，知乎，头条系，站酷，追波，油管，公众号，都是要铺一遍的）网站分为桌面端和移动端，主要就是电脑和手机，实现技能基本就是展示型的网站，博客类的，所以做响应式就可以，不需要做自适应的，花那个力气没必要，开发技术还能干什么？做APP，现在flutter很多，我准备掌握的就是这个，谷歌的东西，精品，做小程序，微信小程序，抖音小程序，这种内嵌在社交软件里的app，那便携性自不必说，免下载直接使用，这是对大部分用户友好的

  • UI：

这里指的UI不只是用户界面，是包含产品思路，后期运营在内的东西，毕竟一个产品，你能一点点的把他做出来，为什么做他，后期怎么实现盈利，肯定都是提前想好了的，所以以UI代替，UI能做什么？做海报，做淘宝美工，做收学徒的课表，做上面web要用到的界面，等等

  • 视频拍摄和剪辑：

拍摄和剪辑，是两个大的门类，拍摄的技术水平，直接影响到后期视频的表达和质量，巧妇难为无米之炊，拍摄占七，剪辑为三，我虽然没有开始学习，我个人是这么认为的

拍摄：拍摄的过程一定是带着剪辑思路去拍的，不可能傻拍，乱拍，拍摄要考虑的是构图和光线，色彩这个阶段考虑不到，都是后期制作加进去的

剪辑：新人的理解就是，给想表达的东西化妆，别小看化妆，这个化妆能把你想表达的内容，在关键节点上的轻重缓急深一步的渲染，对比，共鸣等等，本来平淡的笔直高速路，变成了拉力赛，转角遇到爱那种感觉，在掌握了基本的专场 音乐音效后，就可以按照自己的意图创作了，能做什么，做教材，做产品宣传片，饮料的也好，汽车的也罢，做企业活动记录，做婚庆，都可以

  • 技能点：
  1. 前后台页面 [ html css javascript ]，我们平时浏览的和管理员管理网站用的页面
  2. 后端逻辑开发 [ Javascript php mysql redis ]
  3. 运维 [ linux php mysql nginx redis ]
  4. UI [ AI PS Indd XD ]
  5. APP [ react flutter ]
  6. 剪辑 [ pr(me) ae lr ps ]
  7. 网站在搜索引擎方面还要做好sitemap和自动化的站内定向锚文本，也就是常说的SEO，因为通过主动搜索来的客户，意向度非常高

健康财富篇

还得增加一下财商，确定一个观点，钱是省出来的。再就是学习保险知识，买保险是亏钱的，买的是保障和强制储蓄两个功能。

这东西就是，没事的时候是骗子，买错了是骗子，发现有用的时候后悔没多买点，人性而已

保险分为重疾险，医疗险，意外险和寿险、年金险，家里的经济支柱，首先就要配置足够的保险。那么重疾险+住院医疗险+意外险+定期寿险都是必要的，小孩子呢，因为他没有经济收入，寿险就不需要了，所以是重疾+住院医疗+意外险就可以。老人呢，首要考虑的是住院医疗险和意外险。要是想在大病这一块有保障的话，还可以考虑防癌险， 就是针对于恶性肿瘤这一块的。

优先考虑的就是医疗和重疾，重疾贵一些，因为是直接赔付的，医疗是报销医疗费用的。
