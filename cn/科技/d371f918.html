cTrans云传输系统

2020-11-14
| 科技


   作为新型企业级数据传输服务，cTrans云传输系统可大大降低企业实施与使用专业的数据传输解决方案的成本与门槛，帮助企业从IT基础设施的层面，快速建设数据传输管控能力，实现数据传输质的飞跃，持续推动业务效率提升：

1.无论是本地服务器还是云端的数据，都能够无障碍流动，解决场景数据流转问题；

2.统一管理企业分散的存储资源，帮助企业更好地利用已有的数据资产；


3.实时监控传输任务，确保整个传输过程“可见、可控”；


4.审计全网传输历史，定期生成统计报表，便于用户存档和分析。

   基于 cTrans 高速云传输机制构建的 cTrans 云分发系统，主要包括中央节点资源及任务管控软件、边缘节点传输客户端软件两个部分。主要功能包括节点管理、用户管理、资源管理、任务管理、日志管理、参数设置等功能模块。cTrans云分发系统广泛适用于广电、媒体、互联网、军队、科研机构等特定行业和专业领域，并可在资源及数据的存储层面，与相关领域的业务系统（例如：媒资管理系统、地理信息系统等）进行整合。云分发系统具有传输速度快、传输成功率高、运行安全稳定、扩展性强、易于部署维护等优点。

cTrans组件

  云传输系统主要由文件管理应用平台cTrans Web.Front、云传输管控平台cTrans Web.Console、浏览器传输插件cTrans Plugin、高速传输引擎cTrans Engine、应用集成接口工具Transfer API等组件组成。根据不同的应用场景，以不同组件进行组合使用，使用更加灵活，配置更加简便。

　　cTrans Engine高速传输引擎

高速传输引擎是整个传输系统的核心组件，也是传输软件的主要组成部分。它为传输客户端与业务系统提供上传下载工作的服务与支持，支持高量并发任务，拥有高运作性能，为业务系统提供安全、可靠的高速文件传输服务。

　　浏览器高速传输插件cTrans Plugin

浏览器高速传输插件，是一款轻量级、高性能、可自动识别安装的文件传输插件。它适用于各种主流通用浏览器，有效解决了无法通过浏览器上传、下载大文件与海量文件的问题，大幅度提升了浏览器以及各类Web 系统的文件传输能力。

　　文件管理应用平台cTrans Web.Front

cTrans Web.Front是面向终端用户，提供多场景、多形态的传输应用服务平台。提供文件的归档管理、用户账户安全管理、分组配置管理、传输下载管理等。搭配浏览器高速传输插件及高速传输引擎，结合数据库系统和用户的存储系统，实现传输管理和资源管理等功能。

　　全局数据传输调度管控平台cTrans Web.Console

cTrans Web.Console面向IT管理员，提供在多节点间自动化调度和管理文件传输的服务。内置各种调度策略、传输流程策略、审计日志策略，方便用户建立各种情景下的节点数据应用管理。

　　应用集成接口工具Transfer API

Transfer API是面向研发人员提供的一系列开发工具和集成方法，支持各种开发平台，可以和用户现有的应用系统做无缝的集成兼容，方便用户构建整体信息化建设。

cTrans功能

基于UDP的高性能数据传输 

cTrans云传输解决方案基于高速传输协议——UDP，可以满足传输所需的高速度、可预测性和安全可靠性。cTrans云传输系统通过多种手段提升传输性能，大大提升数据送达时效性，打破了IT基础设施间的物理隔阂，破除阻碍数据流动的各层面障碍，使得数据交换流畅进行，并使大量业务应用场景变得可行：

1.为私有云、公有云以及混合云，提供数据传输与调度管控服务

2.以SaaS模式 (Transfer-aaS, Transfer as a Service)，提供传输应用服务；

3.以PaaS模式 (MFT-aaS, Managed File Transfer as a Service)，提供传输中间件服务；

4.支持私有化部署。



cTrans云传输解决方案

基于多种技术的数据处理应用

   cTrans云传输系统大大降低了对于大数据移动、管理和使用的门槛，使用多种特有手段保障大数据迁移过程中的可靠性和时效性。用户无需担心持续积累的爆发数据量，通过大文件自适应分段技术与海量文件虚拟拼接技术，使得处理大数据就像常规数据一样简单。



　　同时，在传输策略随时动态调整的复杂情况下，cTrans依然可以保证传输的正确性和安全性，支持断点续传和错误重传等，使得对于大文件的传输体验和普通文件一样简单便捷。


　 智能断点续传

　　对于常规文件、巨型文件和海量文件，自动匹配不同机制的断点续传策略，无论是出错自动重传时还是人工操作时，都可以进行断点续传。

　　出错自动重传

　　当出现传输错误时，会自动启动重传机制，最终确保传输文件的正确性。

　　自定义处理脚本

　　cTrans云传输系统支持用户通过自定义脚本，在文件传输任务执行前后，触发和执行自定义的数据处理或消息通知等指令，实现业务流程自动化。

　　API接口服务

　　cTrans云传输系统提供基于RESTful通信格式的API接口，利用这些接口，开发者和第三方应用可以进行数据传输指令下达、传输进度状态跟踪等操作。

  

基于银行级秘钥交换机制的数据加密

   cTrans云传输系统采用银行级的秘钥交换机制和数据加密算法，支持文件传输内容加密，在数据收发两端之间形成加密隧道，在传输的多个环节进行一致性校验，确保文件传输的一致性，形成覆盖全局的安全加密数据传输可信网络，以安全可靠的方式进行通信及数据传输的秘钥交换，防止数据泄露。同时，集中审计分散的数据传输行为，做到数据流动可追溯，确保安全合规。

产品特性

 

集中管控

cTrans云传输系统跨过单点独立传输工具各自为政的初级阶段，将分散的数据存储和传输行为进行统一管理，包括传输节点与存储节点集中管理、远程安全数据管理、全文传输行为实时调控等，将传输行为纳入到管控范围之中，集中进行数据管理和传输调度，实现无人值守、全流程自动化的数据传递，确保数据移动安全合规，轻松掌握全网数据交换态势。

流程自动化

cTrans云传输系统打通业务运转全部流程，形成稳定的自动化数据交换流，通过自动化的传输触发机制以及可自定义的传输前后数据处理及通知策略，驱动业务大数据便捷流动和自动处理，提升数据处理时效，大大提高业务运转效率。

高效传输

cTrans云传输系统利用企业现有的IT基础设施，基于UDP协议，形成针对大容量数据的传输调度能力，并充分利用企业网络资源，进行大文件及海量文件的高速传输。

灵活响应

cTrans云传输系统具有灵活的应用架构，简单的集成方式，高度的可配置性，可迅速为企业现有IT架构提供数据传输助力：实时、周期、按需等灵活的传输触发机制，流量控制、内容过滤、事件通知等灵活的传输配置选项，API、插件、脚本等多种集成方式。

强扩展性

cTrans云传输系统广泛支持企业内部、外部不同的数据传输需求场景，同时能够完成不同场景之间的相互联通，具有强大的横向拓展和纵向升级的能力，用以应对企业将来未知的数据传输挑战：多种传输方案按需组合，异构传输场景之间的横向联通，跟随企业IT建设规模持续扩容。

统一建设

cTrans云传输系统方案针对企业可能面临的不同场景、不同系统的数据传输问题，一次性解决企业当前多环节的不同问题，同时可以持续响应企业对于数据传输的新挑战：包括系统级和用户级多种场景的整体传输，以及一次性解决不同系统类似场景的问题。


cTrans价值

　　提升企业IT建设层次

cTrans云传输系统补全企业IT基础架构中的数据传输能力，使IT 能够更好地满足日益增长的数据交换需求，支持 IT组织跟上新技术和业务需求的步伐，提升企业IT建设综合层次。

　　降低企业运营成本

cTrans云传输系统快速响应业务对于数据传输持续变化的需求，大大降低业务迭代周期。形成贯穿企业核心业务流程的自动化数据流动和数据处理，减少IT资源分配和人工投入，大幅度提升业务服务质量水平。

　　规范数据传输管理

通过cTrans云传输系统，企业使用一套系统即可完整掌握传输系统内所有资源文件和传输节点的实时情况，按需调度数据的流转，避免传输过程不可见、不可控的风险，确保数据传输管理规范化。

　　挖掘数据应用价值

cTrans云传输系统帮助打破企业内部的数据孤岛，真正使企业存储在本地服务器、云端等各个节点位置的数据有效连接，无论是跨国还是跨云的数据交换都变得简单易行，大幅度提升企业数字资产的价值，从而实现更出色的生产力和持续优化。
