垂直鼠标好在哪？细品罗技 MX Vertical 人体工学鼠标独到之处

2020-10-29
| 科技

一向以人体工学见长的罗技所推出的MX系列鼠标一直受到商务办公人群的喜爱，旗下包括了MX Master系列、MX Anywhere系列以及MX Ergo轨迹球。我个人在用的就是MX Anywhere 2S，它基本成为了我日常出行办公必备的工具。

前作中的MX Ergo轨迹球造型更加别致，除了增加万向轨迹球外，还可以实现20°的角度调节。

商务办公群体应该都会注意到，长时间使用鼠标后，手腕部分关节疼痛、麻木，拇指肌肉感觉无力，需要一段时间的缓和按摩才能缓解。这是由于普通鼠标是平握，支点是手腕下部，压迫了腕管中的正中神经，使神经传导被阻断。

为了应对“鼠标手”这一传统鼠标使用痛点，罗技（Logitech）在去年8月推出了MX系列旗下第一款无线垂直鼠标 MX Vertical 。这款鼠标可以同桌面形成 57°的夹角，使其相比普通鼠标减少 10% 的手部肌肉活动，从而减轻用户使用鼠标的手部疲劳。那么这款主打人体工学的垂直鼠标——罗技 Logitech MX Vertical，是否能够如期所宣传的为使用者带来更加健康的手部姿势，减轻使用者的腕部压力，敬请跟随我的视角了解一番吧。

首先，我们来看看产品开箱，罗技 MX Vertical 采用了比较典型的罗技包装，正面印有产品样图。

半开放式的开箱效果，可以让用户快速掌握产品的外观，贴合的透明保护层能够让购买者上手感受一下尺寸。

包装的背面是 罗技 MX Vertical 的产品信息，以及主打卖点等。

【全家福】：可以看到除了 罗技 MX Vertical 本体外，配备了一个无线发射器、USB-C充电数据线以及众多的产品保修卡说明书。

罗技 MX Vertical 的充电数据线为USB-C接口，这点好评，之前我的MX Anywhere 2S也只是Microusb接口。同时在颜色上同鼠标保持了一致，这根数据线也可以将 罗技 MX Vertical 转化为有线鼠标使用，在鼠标没电时可以救急。

因为 罗技 MX Vertical 是有线、蓝牙、2.4G三模连接可选，所以自然也配备了一个2.4G的无线发射器，不过比较可惜的是它不能存放在鼠标内部，而是单独收纳，所以存在一定的遗失风险，建议还是插在常用设备上。

终于到了外观展示环节了，通过下图360°的动图展示，罗技 MX Vertical 之所以被称作垂直鼠标，也是因为其独特的造型。鼠标的中心并非如普通鼠标般平放在桌面上，而是像斜向上倾斜，让手部自然的贴合。

从 罗技 MX Vertical 的正面视角可以看到左侧内凹，右侧圆润，典型的右撇子设计，为手心预留了大量支撑区域，拇指则紧紧贴合内侧，正面下方还有个USB-C接口，与充电数据线配合使用。

罗技 MX Vertical 正面的夹角是57°，我经过反复的感受后才发现精妙之处，这个角度恰恰是我手部自然放松的区间。想像一下你手握一瓶脉动，垂直握住时手腕向外弯曲，当你在喝脉动时，手腕向内是不是最舒服的角度。

这应该也是罗技经过反复测量设计后的结果，根据罗技的说法，MX Vertical 的设计旨在模拟“自然握手位置”。这有助于在长时间的计算过程中减少前臂张力，与普通鼠标相比，倾斜的 罗技MX Vertical 可以减少高达10%的肌肉活动。

右测视角，罗技 MX Vertical 有点像一个伫立的河蚌，右上角设计着鼠标的左右键位和滚轮。侧面的手部贴合区域采用了橡胶材质，同时这里采用的是棱角分明的纹理，相比于光滑的弧面有助于提升质感，增大摩擦。

罗技 MX Vertical 的顶部是一个长条形平面，左边印有Logi标志，右侧则是一个可自定义的多功能按键，默认用以切换两档dpi（鼠标灵敏度）。

从 罗技 MX Vertical 的背面视角可以看到这里设计了一个LED指示灯、两个可自定义的多功能键。

罗技 MX Vertical 的底面有点像是达芬奇画过的鸡蛋，两个弧形的脚贴实现了整个鼠标在平面上的划动。底面的顶部是电源开关，中部是传感器，其下是设备切换键（3个LED），最下方是鼠标的认证及系列号等内容。

由此可见，罗技 MX Vertical 在按键和接口的分布定位上是与MX Master系列、MX Anywhere系列基本保持一致的。

【上手感受】：罗技 MX Vertical 的上手感受主要表现为三个阶段，初次上手时会有一种不适应感，毕竟以欧美人手型设计的鼠标始终会有些偏大，不过好在按键的位置均较为靠后，使用没有问题。经过大概半小时的使用后，我已经能够熟练掌握 罗技 MX Vertical 的使用技巧。

使用一天以后，才发现 罗技 MX Vertical 相比于普通的鼠标，长时间使用也不会有过多的负担。我已经在让师弟重点感受 罗技 MX Vertical 了，准备带他跳坑入手一个。

罗技 MX Vertical 的滚轮部分和我的 罗技 MX Anywhere 2S 较为接近，力度适中，较为容易上手。

鼠标的定位感上，因为 罗技 MX Vertical 体积偏大，还需要我的不断适应。让我拿来玩游戏，我肯定是拒绝的，但是当个生产力工具，我还是很欢迎的。

因为 罗技 MX Vertical 涉及到了多功能键的自定义、鼠标分辨率的调节、Logitech Flow技术，所以它也能够适配 Logitech Options 软件，通过官网下载安装即可。

首先，Logitech Options 可以对鼠标的滚轮按键、顶部按键以及左侧按键一共四个键位进行自定义。

顶部按键较为独特，默认是指针速度（DPI）的两档切换，最大支持4000dpi。

左侧两个键位我倾向于设置为音量+-，毕竟翻页有了滚轮，多媒体是我常用的场景。

其次，Logitech Options 也能够调节 罗技 MX Vertical 的平滑滚动、滚轮方向等。

最后， 罗技 MX Vertical 支持自家的Logitech Flow技术，也就是在多个设备无线切换，这种功能要求每个设备上均安装了Logitech Options，并同鼠标配对。

例如很多程序猿小哥哥们，经常拥有多个屏幕进行代码编辑处理，此时拥有一个可以支持Logitech Flow跨屏幕操作的鼠标就算的上不错的生产力工具了。

【鼠标操控】：为了测试不同分辨率（DPI）下的 罗技 MX Vertical ，只好将DPI换为最高、中等、最低三档进行了下图测试。实际的感受是中档以下蚊香图和井字都非常好画，但是在最高

【充电续航】：罗技 MX Vertical 搭载了240mAh的可充电电池，完整充电一次能够使用4个月，并且 罗技 MX Vertical 也支持快充功能，充电一分钟可提供3小时使用时间。

实际体验是我从开箱到使用了一周之后，Windows的蓝牙面板上面数据显示仍然是100%的电量标志，由此可以判断出 罗技 MX Vertical 的续航应该真的很强，这点非常贴合商务人群不喜充电的需求。

【办公场景】：对于需要长时间使用鼠标的用户而言，罗技 MX Vertical 就可以发挥巨大作用了，一方面帮助用户减轻手部负担，另一方面提升效率。

对于我们这些需要长时间端坐在电脑前研读文献的学生党而言，罗技 MX Vertical 都算的上得心应手。无论是文献的上下翻页还是具体内容的标识都可以尽在掌握。唯一可能比较遗憾的就是，价格对于学生党不够友好，希望能搞点学生优惠吧。

经过本文的测评之后，我们最后聊聊 罗技 MX Vertical 的产品优势和值得改进之处。

【产品优势】：

1.罗技 MX Vertical 的人体工学设计，相比于普通鼠标，能够有效减少使用者手部的负担，保证长时间工作。

2.罗技 MX Vertical 的按键丰富，配合 Logitech Options 可以实现诸如多功能键、DPI两档切换、Logitech Flow等精细化的功能体验。同时，罗技 MX Vertical 支持无线、蓝牙、有线三种模式的连接，可以满足多种使用场景。

3.罗技 MX Vertical 续航表现卓越，同时支持快充，对于在外旅行的用户算的上利好的消息。

【值得改进】：

1.罗技 MX Vertical 的初期上手会有点难度，主要是定位的把控还需要一定的习惯过程。

2.罗技 MX Vertical 的体积稍大，可以考虑为手小的用户如女性群体提供尺寸更加小巧的版本，也便于外出携带。

3.罗技 MX Vertical 的定价仍然高昂，建议下调定价空间，让更多用户有机会享用。

测评到此结束，我是Geek研究僧，一枚爱好测评的研究生。
