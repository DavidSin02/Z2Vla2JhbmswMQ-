「Leetcode笔记」第2题：两数相加

2020-11-14
| 科技

两链表的指针同时一个一个顺位前进，相加大于10时令carry=1，并进到下一位

还得注意链表一长一短，空链表，末尾额外进位的情况

他这里也是使用虚拟头（dummyHead），返回 dummyHead.next

public ListNode addTwoNumbers(ListNode l1, ListNode l2) {    ListNode dummyHead = new ListNode(0);    ListNode p = l1, q = l2, curr = dummyHead;    int carry = 0;    while (p != null || q != null) {        int x = (p != null) ? p.val : 0;        int y = (q != null) ? q.val : 0;      注意上面三句的写法        int sum = carry + x + y;      这个sum是当前位的相加值，所以下面第二句就可以这样写        carry = sum / 10;        curr.next = new ListNode(sum % 10);        curr = curr.next;        if (p != null) p = p.next;        if (q != null) q = q.next;      也注意上面三句的写法    }    if (carry > 0) {        curr.next = new ListNode(carry);    }    return dummyHead.next;  他这里也是用 dummyHead.next}
