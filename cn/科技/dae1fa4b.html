新型铸铁检查井与路面同步施工技术

2020-11-14
| 科技

摘要

为保证车辆和行人在公路和市政道路新改建、养护施工时可以正常通行，根据道路设计和施工作法，将检查井产品的底层支座进一步分解为下支座和底支座，既能施工便捷又能正常通行车辆和行人。

关键词: 道路; 检查井; 多层支座; 分层; 同步施工

传统和现行铸铁检查井即单层支座检查井、前期创新的双层及多层支座铸铁检查井的底层支座用水泥砂浆或水泥混凝土固定后，因为需要水泥砂浆或水泥混凝土养生而不能马上铺筑同层沥青混凝土，底层支座因上表面达到和将要铺筑的底层沥青混凝土上表面平齐而高出基层 60mm 以上，严重影响道路上车辆和行人的正常安全通行，行车也会碾压导致底层支座位移。现结合传统和现行单层支座铸铁检查井、双层及多层支座铸铁检查井作法进一步创新，将底层支座一分为二为下层支座和底座，从而适应道路施工和养护时车辆、行人的安全通行。

1 铸铁检查井安装传统作法

1. 1 传统和现行铸铁检查井作法和缺陷

道路沥青混凝土路面结构层都是按照设计作法分层施工，而传统和现行的铸铁检查井却只有一层，无法与道路结构层分层同步施工，如双层沥青混凝土路面结构与传统和现行的铸铁检查井的施工工艺是: 将铸铁检查井顶面与下面层沥青混凝土路面调整水平后用水泥砂浆或水泥混凝土临时固定铸铁检查井→固定层达到可施工强度后，铺筑并压实下面层沥青混凝土→ 挖出铸铁检查井→砌筑井筒涨高井，调平固定铸铁检查井，养生→铺筑并压实表面层沥青混凝土。

传统和现行的单层支座铸铁检查井作法有以下缺陷:

( 1) 铸铁检查井与路面沥青混凝土层分层施工不能保持同步;

( 2) 有技术间隙时间长，固定支座的砂浆或现浇水泥混凝土需等 3 ～ 5 天达到一定施工强度后才能进行面层沥青混凝土铺筑;

( 3) 支座每次固定后铺筑沥青混凝土前都高出一个即将铺筑的沥青混凝土厚度，影响道路车辆和行人安全通行，支座高出后也容易被撞和碾压产生高差和水平位移;

( 4) 反挖刚铺筑完的新料后又采用新材料回填修补，既浪费了材料又产生渣土垃圾，且只能小型机具配

合人工施工，有接茬整体性差还不美观;

( 5) 支座用不易结合成整体的一层表面层沥青混凝土和一层砂浆或水泥混凝土固定铸铁检查井，在车辆荷载和热胀冷缩作用下易发生开裂脱落病害。

1. 2 前期创新的铸铁检查井的作法和缺陷

前期创新的新型道路铸铁检查井具有了较大的创新和很大程度上方便了施工，但底层沥青混凝土铺筑前固定的底层支座高出基层不低于 60mm，严重影响车辆和行人在道路上安全通行，如图 1 所示。

2 发明和改造

为完善和改进传统和现行单层支座铸铁检查井、前期创新的双层和多层支座铸铁检查井的缺陷和调整 铸铁检查井的作法，特进一步优化新型铸铁检查井的 产品作法和施工工艺。

2. 1作法

新型铸铁检查井由上支座、若干中支座、下支座、和底座组成，各自的作法和组装作法详见图 2 ～ 图 6。

依据沥青混凝土路面结构分层层数和每层厚度来增加或减少底座上中支座和上支座的数量来满足道路的相应需求:

底座和井盖组合为传统和现行的铸铁检查井;

上支座、底座和井盖组合为单层支座铸铁检查井， 适用於单层沥青混凝土面层道路的检查井;

上支座、1 层中底座和井盖组合为双层支座铸铁检查井，适用于双层沥青混凝土面层道路的检查井;

上支座、2 层中底座和井盖组合为双层支座铸铁检查井，适用于三层沥青混凝土面层道路的检查井;

上支座、多层中底座和井盖组合为双层支座铸铁检查井，适用于多层沥青混凝土面层道路的检查井。

三层沥青混凝土路面的道路铸铁检查井由上支座、中支座、下支座和底座组成，组装后的平面图和剖面见图 6 所示。

上层支座伸入下层支座的部分称为连接板或内伸 板，在满足连接固定的基本功能后为减少材料使用节约 成本可将连接板做成有孔洞的片状或柱状作法，下端既 可以延伸到下层支座支撑台上，又可以悬空，如图8 所示。

2. 2与路面结构层同步施工工艺

新型铸铁检查井与沥青混凝土路面层分层同步施工的施工工艺如下所示:

( 1) 铺筑完基层或铣刨完底层沥青混凝土后，在基层预制钢筋混凝土模块预留槽内用预埋筋和焊接件固定底座，使底座的上表面与基层顶面平齐，浇筑固定材料，盖上井盖和钢板养生，底座顶面与基层顶面平齐不影响道路基层上通行车辆和行人;

( 2) 养生期满，取走钢板，有封层时施工封层，然后取出井盖，将下支座和井盖安装到底座的支撑台上， 铺设底层沥青混凝土并碾压密实;

( 3) 待底层沥青混凝土温度低于 50℃ 后，取出井盖，连接固定中支座后盖上井盖，接着进行道路的上一层沥青混凝土的铺设和碾压密实;

( 4) 根据需要重复步骤( 3 ) 直到铺筑完中层沥青混凝土;

( 5) 待中层沥青混凝土温度低于 50℃ 后，取出井盖，连接固定上层支座后安装井盖并固定，进行道路的表面层沥青混凝土的铺设和碾压密实。

新型铸铁检查井产品和施工工艺的核心是: 根据设计的道路沥青混凝土面层层数和层厚选择相应高度的各层支座，在满足上支座的上表面与表面层沥青混凝土上表面平齐后其余支座的上表面不高于同层沥青混凝土上顶面，以保证同步施工的实现。如 4cm 表面层沥青混凝土 + 6cm 中层沥青混凝土 + 8cm 底层沥青混凝土 + 1cm 封层路面结构，如采用井盖外沿厚 4cm 的常用井盖则分层可为: 5. 2cm + 5. 8cm + 8cm + 底层支座，如采用井盖外沿厚 3cm 的井盖则分层可为:

4. 2cm + 6. 8cm + 8cm + 底层支座高度; 5cm 表面层沥青混凝土 + 6cm 中层沥青混凝土 + 8cm 底层沥青混凝土 + 1cm 封层路面结构，采用井盖外沿厚 4cm 的常用井盖则分层可为: 5. 2cm + 6. 8cm + 8cm + 底层支座。

以三层沥青混凝土的道路为例说明三层支座检查井的施工工艺，详见图 9。

新型多层支座铸铁检查井的施工工艺过程有以下积极和有益的效果:

( 1) 内优外美: 摊铺机、压路机一次性施工完成， 没有二次作业和修补接缝，提高了道路压实度、整体性 和美观性;

( 2) 提前工期: 无涨井等待技术间歇，铸铁检查井施工与路面结构层的分层同步施工，多层沥青混凝土路面可紧跟连续铺筑;

( 3) 节材环保: 井周不再反挖和修补，既节省材料又不会产生建筑垃圾;

( 4) 便捷施工: 在下次道路大修时铣铇完路面后没有涨井技术间歇，只要取下各层支座再分层安装回去即可铺筑各层沥青混凝土，特别适合快速开放交通的城镇道路需求;

( 5) 安全平稳通行: 每层施工后均是平面，不影响交通并通行平稳安全。

2. 3原材料

新型检查井不局限于灰口铸铁、球墨铸铁生产加工，也可扩展到复合新型材料等原材料制作。

3 新型检查井的创新点和优点

与道路同步施工的新型检查井在具有提高质量、提前工期、节材环保的效果的同时，又有了新的有益创新:

( 1) 根据城镇道路施工不中断交通的实际需求优化铸铁检查井底支座为下支座和底座，既方便了施工又保障正常安全通行;

( 2) 针对道路使用期内经常有日常养护修补和小中大修及新改建反复拆装的特性，铸铁检查井井盖、上下层支座间的采用了多种可方便安拆的连接方式;

( 3) 在满足铸铁检查井使用功能的情况下优化了结构作法，节约了原材料和费用;

( 4) 为增强铸铁检查井的稳固性改进了底座锚固的作法。

4 适用范围

改进后的新型铸铁检查井针对不同层数的沥青混凝土路面结构做了相应组合匹配，单层、双层、三层机三层以上分别适用于新铺筑的单层、双层、三层及三层以上沥青混凝土面层的道路检查井。

新型铸铁检查井根据需要可以添加成防滑、防盗、防响、防坠落和防位移的五防检查井。如井盖上浇铸 各专业花纹图案或特制的艺术图案，可以起到防滑、美 观效果; 通过锁具或连杆将井盖固定或连接到支座上， 起到防盗效果; 将井盖和支座通过卡钉和卡槽连接或安放橡胶垫圈，可以起到防响效果; 将底层支座的底板内伸进检查井一定长度，可以防止子井盖，起到防坠落 效果; 将底层支座通过预埋筋和焊筋固定到井筒上，能起到防位移效果。

5 结语

新型铸铁检查井依据同步施工原理和一边施工一边通行的需求完善创新出了新的作法，依然具有提高质量、提前工期和节能环保、方便施工和安全通行的优点，具有推广应用的价值。
