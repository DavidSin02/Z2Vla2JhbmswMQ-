javascript关于鼠标右键和左键的小实例「313」

2020-10-29
| 科技
  • 今天我想到了windows自带的小游戏扫雷，于是想到了其中的鼠标左右键操作，这里做了一个简单的实例，按钮点鼠标左键变红色，点鼠标右键变绿色，大家可以参看一下，更多文章请关注我的头条。

一、能简化的我们都简化了，这里我先做出六个按钮，并给它们添加样式。

二、还是得把这六个按钮装在一个div容器里，等下点击就用得上了。

三、创建两个样式，一个是红色背景，一个是绿色背景。

四、开始创建点击事件，看看点击效果。

五、我们要明确哪个按钮被点击，测试一下效果。

六、成功，我们换成加样式，先加鼠标左键点击添加红色背景样式。

七、添加鼠标右键的点击事件。

八、到第七步，你觉得应该完美了，但还是会发现，变色后，再点绿色变不成红色，红色的右击却能变绿色，还是没能完美，那是因为它们里有两个样式，于是样式就近原则，解决这个问题的方法就是在变样式前，先把原有的样式清空。

<!DOCTYPE html><html lang="en"><head> <meta charset="UTF-8"> <title>Document</title> <style> button{ width: 50px; height: 50px; /* 按钮宽高为50 */ } .tred{ background: red; } .tgreen{ background: green; } </style></head><body> <div class="box"> <button></button> <button></button> <button></button> <button></button> <button></button> <button></button> </div> <script> var ebox = document.querySelector(".box"); // 取得box的元素对象 ebox.onclick = function () { var tg = event.target; // 取得被点击的元素对象 tg.className = ""; // 原有样式为空 tg.classList.add("tred"); // 点击对象添加样式tred } // box的点击事件 ebox.oncontextmenu = function (e) { // 这里是右击时系统弹出 e.preventDefault(); // 这里是让系统原有的右键菜单失效 var tg = event.target; // 取得被点击的元素对象 tg.className = ""; // 原有样式为空 tg.classList.add("tgreen"); // 点击对象添加样式tgreen } </script></body></html>
