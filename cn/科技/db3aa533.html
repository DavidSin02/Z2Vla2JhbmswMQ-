京时观音王柠檬茶

买 一 送 一

◎ 佛山首家400㎡独栋“ 制茶工厂” ◎

TA是佛山首家独栋的【宋代美学茶馆】，就开在岭南天地！凭借着独特的新宋茶饮，俘获了一大批忠实粉丝，最近还给大家带来了夏日“解暑神器”！



400㎡的独栋空间

足足有400㎡的茶馆，分为上下2层，在当代设计中融入了宋代美学，极力打造极致的茶生活空间。

茶馆的名字也蕴含着独特意味，京时之“京”，取于宋朝都城“汴京”，京时之“时”，是汴京时间，天然古茶与现代茶饮的融合，赋予京时之茶以今时之味。



顶尖的制茶工艺

茶馆对茶有着细致的追求，特别引进国内顶尖的全自动炒茶机，炒制、烘焙、过筛、醒发、摊晾...每步工序都十分严谨，只为保留茶叶的原香。

① 炒制

② 烘焙

③ 过筛

④ 摊晾

正是这样严格的制茶工序，加上对天然古茶和现代茶饮的潜心研究，才让茶馆的每一款茶饮都拥有自家独特的味道。

◎ 店里头号“茶魁”，至佳解暑神器 ◎

京时观音王柠檬茶

这杯让人你从夏日中彻底解放的京时观音王柠檬茶，稳坐茶馆头号“茶魁”的地位，一登场就就能感受到它的冰凉气息~

// 现炒的浓香型观音王 //

作为一家把茶叶做进骨子里的茶馆，京时采用全自动的炒茶机，用120°的精准温度炒制80分钟，将清香型的铁观音烘焙成浓香型~



制炒后的铁观音，油润匀整香气悠长，远远地都能闻到醇厚的茶香。



// 纯手工捶打的柠檬 //

特别选用3种不同类型的柠檬，将香水柠檬、黄柠檬、青柠檬混合在一起，调配出独特的柠檬香。


每片柠檬精切成0.3cm，然后纯手工捶打，倒入新鲜煮好的茶底手摇4次，茶的微涩和柠檬的酸味完美融合




入口那刻，嘴巴就感受到冰爽的口感，醇厚香韵的茶香给混合著清新的柠檬香，越喝越畅快，唇齿间得到大大满足~


这样热辣辣的夏天，想续命真的要靠这杯京时观音王柠檬茶，趁着现在买一送一，大家赶紧安排上！



◎ 这些特色新式茶饮，必须安排上 ◎

阳山蜜桃乌龙冰茶


提起茶馆里的“颜王”，这款阳山蜜桃乌龙冰茶必须拥有名字！只要瞄它一眼，就被这桃红色给治愈了！


阳山蜜桃可是中国国家地理标志产品！现切的阳山水蜜桃肉粒，倒入水蜜桃果泥制成圆滚滚的冰球，在配上乌龙茶汤，这也真的太撩人了吧！


用力地吸上一口，满嘴都是清新的乌龙茶香，还能嚼到水蜜桃冰沙果肉，口感超级特别！

台湾凤梨金凤冰茶


采用台湾进口的凤梨，大大颗的凤梨冰球加上清香的金凤茶底，拿到手上就想立刻开喝~


第一口果断要尝这颗凤梨冰球，入口冰冰凉很是刺激，香甜的凤梨果肉和茶香弥漫开来，满嘴巴都是甘香～


传统手工花茗茶


除了新式茶饮，这里的手工花茗茶也是深客官喜爱！传统的茶具和手工的冲泡，让传统茶添上几分独特味道~


选茶、鲜调、洗茶...每一步都是现场纯手工操作，连泡茶的时间也分毫不差地精确到3分钟，才能完美保留花茗茶的风味。




现泡的金盏菊乌龙花茶，喝起来清香甘甜，花香与茶香混合在一起，不浓不烈刚刚好~


另外，手工花茗茶还有各种不同的搭配哦，大家可以根据自己的喜好选择~


冷泡水果茶


每天新鲜制作的冷泡水果茶，每一瓶都浸泡10个小时，有颜值更有内涵哦~


红颜冰粉


手工制作的冰粉，面层铺满了玫瑰花瓣，入口冰凉Q弹带着淡淡的玫瑰花香。

径山绿茶冰淇淋

加入径山绿茶制作的冰淇淋，入口浓浓的茶香味，大夏天的舔一口冰淇淋，真是幸福感爆棚。


茶馆里除了有现制的茶饮、小吃...还有各种关于茶的手工制品哦，客官们可以随时把它们抱回家哦~



◎ 中式韵味的装修，感受茶文化 ◎


中式风的装修搭配木质的桌椅，相比一般的茶饮店，这里更有一种中式茶文化的韵味~


桌子上整齐摆放着的茶叶和器皿，一点一滴都能感受到浓郁的中国茶的深邃。


坐在茶馆的任意一个角落，点上一杯独特的茶饮，这样的体验真的别具一番风味~


觅食小分队探店小结：

1. 茶馆的茶饮真的很特别，茶味比一般的店更点香浓醇厚。

2. 所有的茶叶都是现场的，进到茶馆就会闻到浓浓茶香~

3. 个人很喜欢这样的中式装修，在这里喝着新式茶饮蛮有有氛围的。

4. 茶馆还有各种手工制品，喜欢的客官可以来逛逛哦~
