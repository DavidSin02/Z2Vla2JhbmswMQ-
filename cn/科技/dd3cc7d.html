介绍一个软件是很吃力不讨好的事儿，思来想去很久，作为徐涛老师的铁粉，旺仔男仔男孩，我「周又土 - 铁旺」（以下简称「铁旺」）决定以独特的方式表达对老师的感激之情，以考研政治的学习为例子将所有 MarginNote 3 的使用重点串起来讲。（准备了很久久 T T）

考研政治的 3 个要诀和 2 个题型

天才第一步，先要想清楚：我们首先对考研政治的考试有一个全面的认识：学政治 95% 的人都是为了在考试中取得个高分。那么，为了应对考试，就要对不同的题目采取不同的学习方法。好在政治只有两种题型，1 种是选择题，另 1 种是分析题。

同时结合涛涛所说的「三个要诀」来进行操作：

要决一：听

  • 听重点（划出涛涛说的重点）
  • 听阐述（老师的解释，各种形象生动的做包子作为例子的阐述）
  • 听逻辑（时间线的串联，每个运动的土地法的变化）

所以我们用 MarginNote 3 做笔记 + 思维导图整理。

要决二：练

  • 强化班 → 真题集（讲完一个模块就可以开始练题，一个月就可以做完两遍）→ 习题集
  • 试卷上的大部分题要你深度理解才能做出来
  • 一本真题，一本习题，各练两遍，错题集，只练选择题，不练分析题

所以我们用 MarginNote 3 做错题集。

要决三：背

  • 用心背书
  • 灵活运用，题目千变万化，答案始终如一

所以我们用 MarginNote 3 + Anki 背诵。

其中，选择题侧重于听和练，而分析题侧重于背。

MarginNote 3 与「政治选择题」

听课，梳理，做笔记

在听课的时候使用「文档模式 / 学习模式」，对于能够进行文字选择的 PDF 文件，可以直接通过划线、划重点、高亮选中文字来创建卡片。

而对于不能进行文字选择的 PDF 文件，用 iPad 和 Apple Pencil 通过「写字笔」工具划出重点，手写笔记。

各种手写、打字、录音、截图

可是有时候我们需要整段的书上的文字，同时也需要对一个知识点进行详细的笔记，比如在讲剩余价值的时候，涛涛讲了很长时间的做包子的例子 （很生动，很喜欢，听完很饿），讲了很多内容，但是树上的笔记记不下该怎么办呢！Don’t worry，土土有好办法！

我们可以先框选内容，点击批注符号，就可跳出一个卡片的界面，然后进行文字输入、手写、录音、截图。

铁旺这边截了涛涛的考研视频的图片，同时手画了做包子的时间图、打字的笔记。（不好意思有点丑，给自己看，效率第一！）

说实话这段知识是很绕，听都是能听懂，和别人复述一遍的时候自己还是讲不清楚，所以铁旺我在这个知识点上花了很多时间。

「链接」和「合并」

细心地你一定会发现铁旺这有个「笔记链接」。

当你的笔记想和另一个比价产生关联的时候，通常的做法有两种：1. 创立链接；2. 合并笔记。

「链接」适合距离比较远的分散知识，建立完成后有「虚线的剪头」和「右上角的数字标识」。

「合并笔记」适合距离比较近的相关知识，合并成一条笔记的示例如下：

构建思维导图

进入「学习模式」后，屏幕会分成左侧的提纲 \ 思维导图区域以及右侧的原文区域。

当我们做成了一张张卡片之后，所有的知识点还是分散在书中，我们打开思维导图功能，就能将这一张张卡片构建出一个完美的思维导图框架！

克制的颜色，统一的规则

在思维导图中，你选中文本的颜色就是卡片的颜色，MarginNote 3 在颜色方面给了用户一些的自由度，但铁旺我还是建议大家极其「克制」地使用各种颜色，这样不易造成不同颜色、工具的混用，最后屏幕上花花绿绿的一片，反倒让找不到重点。

事不过三，所以我一般只使用五种颜色：饱和度低的黄、绿、紫、红、蓝。同时也定下了一些使用规则：

  1. 黄色用于涛涛强调的重点知识
  2. 绿色用于背诵的知识
  3. 红色用于超重点，难点，疑问点
  4. 蓝色用于书上老师整理的重点表格
  5. 紫色用于题外话

聪明的你可以和铁旺我不一样，但是一定要有自己统一的规则。

标签也要克制

标签的功能也要很克制地使用。我会在整理分类筛选的时候用标签，有时候老师推荐了什么电影或者书我都会用标签标注一下。总的来说，颜色和导图会更加地直观。

「只显示标题」让脑图更加清晰

为了观看方便，可以将显示调整成「只显示标题」，让脑图更加清晰：

听着涛涛的强化课慢慢地积累，我们最终能画出整本政治书的思维导图！而且是属于我们每个人自己的思维导图！

刷题

我们用涛涛的真题集和习题集来练习题目。

框图梳理线索、做错题集

错题是我们最大的财富，错题集必不可少，但是重新把题目和答案抄在本子上实在是费时费力，无纸化学习的好处就是能够大大的节省了抄题目浪费的时间。我们可以将错题框选，然后加入我们错题集的框框中。

可以修改树形，将属性思维导图修改成框架图，这样所有的错题就被框在了一起。

MarginNote 中提供了「重组」功能，选择多张卡片使用重组，就会将卡片放置在一个半透明的选框中，好似他们被归置在某个区域中，可以视作一个集体，之后复习错的题目就会巨方便！（PS：对于总分结构的文章，将主题相似的卡片重组成一个单独的框框，再将多个组或卡片混合成树状结构的导图，效果比直接用导图好得多。）

政治题目因为有很多需要将零散的知识结合串联起来的题目，比如各种土地法的梳理、各种毛爷爷的文章的整理、中共每一次会议的主题特点，涛涛都会要我们拿出笔记本来做笔记~ 我们就可以用 MarginNote 3 来做某个线索的梳理，让散落一地的知识串联起来！

多本教材一张脑图！

根据课程的进度，需要添加很多新书，MarginNote 3 的思维导图笔记本可以添加多个文档，其中的卡片都会显示在同一个思维导图中。这样我们可以不断丰富一门课唯一思维导图!

同时也把所有教材放在了一起，方便批量管理和主题阅读。

新建自定义卡片

在整理的过程中，常常发现已有的卡片中没有适合作为标题和关键节点的笔记，此时可以点击思维导图中的添加按钮，手动添加一些卡片。

大纲也清晰

换一种有条理的方式查看，类似于幕布。

「源文件 + 脑图 + 大纲」3 合 1

在学习模式下，MarginNote 3 支持通过左侧边缘滑动来拉出大纲，可以同时看到脑图、大纲、源文件，让你不用再像二代一样来回在大纲和脑图中点击切换。

帮你 OCR 好了

MarginNote 3 还新增了 OCR 功能，可以将扫描版 PDF 中的图片内容转换为文字，免去了手动输入的麻烦，同时便于搜索和进一步编辑

「搜索」眼里出西施

最后在无数知识点的热带雨林，你可以用搜索来查找你的关键词，有时往往会让你有意想不到的效果 —— 没想到这个知识点和这个知识点都有同样的关键词！

MarginNote 3 与「政治分析题」

分析题很简单：分类，背诵。

分析题我们用到《考前预测必背 20 题》这本书。

小手一抖做卡片

一张张添加卡片太麻烦可以选择「自动添加到卡片组」。

有时候有时候做卡片一个个手动输入问题答案也很麻烦，框选小黄书的「问题部分」做成一张卡片，框选「答案部分」做成一张卡片，然后按顺序选中「问题部分」+「答案部分」点击「合并」，MarginNote 3 会默认第一张图片作为问题，其他的所有部分为答案。

我通常会在卡片、脑图、和文档中来回切换，在脑图中理解它的知识结构，在文档中理解上下文，再进行背诵。

还是暂时导出 ANKI 背诵比较好

因为截止目前，MarginNote 3 的复习功能还未完善，所以我们选择导出 Anki 来进行背诵复习。

Anki 是一个很强大的软件，这里就不介绍了，是最好的背诵类软件了，用就完事儿了！安卓免费，iOS￥168，macOS 免费。

（PS: 在复习模式里面才能导出到 Anki 背诵哦）

Anki 的工作原理就是按照艾宾浩斯背诵记忆曲线进行问答式（Q & A） 背诵学习。MarginNote 3 的背诵曲线目前不会发生变化，而 Anki 会更根据你反馈的答案难易程度来改变重复的天数。
