模块进阶

Python有一套很有用的标准库(standard library)。标准库会随着Python解释器，一起安装在你的电脑中的。 它是Python的一个组成部分。这些标准库是Python为你准备好的利器，可以让编程事半功倍。

常用标准库

hashlib

import hashlibm = hashlib.md5()   #创建hash对象，md5:(message-Digest Algorithm 5)消息摘要算法,得出一个128位的密文print m             #<md5 HASH object>m.update('itcast')  #更新哈希对象以字符串参数print m.hexdigest() #返回十六进制数字字符串

应用实例
用于注册、登录....

import hashlibimport datetimeKEY_VALUE = 'Itcast'now = datetime.datetime.now()m = hashlib.md5()str = '%s%s' % (KEY_VALUE,now.strftime("%Y%m%d"))m.update(str.encode('utf-8'))value = m.hexdigest()print(value)
