简单、同步剪贴板历史与常用语工具

2020-10-29
| 科技

Rainclip 是一款简单的剪贴板工具，它可以帮你保存近期的剪贴板内容，以供再次使用，可以将常用的文字保存起来，可随时调用，还能对剪贴板内容进行二次利用。

对于剪贴板应用，2T的要求是：够快、够稳定，然后如果漂亮点就最好了。

就是用上来说，Rainclip 可以通过 Ctrl + Shift + V 呼出，响应也算是非常快速了，此时可以直接输入内容进行搜索：

并且可以多次按下此快捷键进行不同分类 Snippet 的切换。Rainclip 支持将常用短语、文本保存在 Snippet 中，供未来随时使用。

不过 Rainclip 的设置页面还是丑的可以。

设置页面里面的 Actions，是一个支持 js 脚本的自定义功能，目前列出了几个：

  • 将字母转换成全大写
  • json 格式化
  • 解析 HTML 代码
  • 插入当前时间
  • 生成随机密码
  • 创建 GUID
  • 获取 Unix 时间戳

不过这个用起来2T还是有点疑问，因为每一个脚本都可以设置快捷键，但2T完全不知道如何激活这个快捷键。倒是每一个 Actions 都可以在剪贴板列表中启用。

最后，无论如何，有一个历史剪贴板工具，绝对可以让效率翻倍。

关于同步，Rainclip 支持 Mac 与 Windows 电脑，需要 Google 账号同步，这个只能会的同学上了。

Rainclip 可以在官网下载rainclip.net
