代码模板（二）data模块：dataset

这一篇介绍代码中的数据部分，也就是data文件夹下的函数：

这一篇只介绍了如何定义数据模块，下一篇介绍如何在训练中加载使用

首先，创建一个data文件夹，在里面新建

  • __init__.py文件以便作为包导入
  • base_data_loader.py文件
  • base_dataset.py文件作为基类

其次，开始写每一个函数

  • __init__.py文件：可以直接为空，也可以根据需要添加内容
  • base_data_loader.py文件（基本不用改）

  • base_dataset.py文件下面基本不用改

  • voc_dataset.py文件：为简洁，有些功能函数可新建一个文件image_folder.py，定义在其中

要求输出：

img:图片数据,path：图片路径

思路：

1、获取数据集路径2、获取图片路径，整理为一个有序列表3、根据图片路径获取处理后的图片数据4、返回图片数据和图片路径,或者还需要label，需要什么就将需要的数据整理成希望的格式，最后按return {'img': img, 'paths': path} 返回

代码：voc_dataset.py

import os.pathfrom data.base_dataset import BaseDatasetfrom data.image_folder import make_dataset, get_transform #导入定义的函数from PIL import Imageclass VocDataset(BaseDataset): # 这里为简洁可将函数定义在data文件夹下的image_folder.py文件中 def initialize(self, opt): # opt为定义的命令行参数参数 self.opt = opt self.dir = os.path.join(opt.dataroot) # 数据集的路径 self.paths = make_dataset(self.dir) # 获取图片路径列表 self.paths = sorted(self.paths) # 对图片路径列表排序 self.transform = get_transform(opt) # 对数据进行处理，裁剪翻转等 def __getitem__(self, index): path = self.paths[index] img = Image.open(path).convert('RGB') # 使用Image.open（）获取图片数据 img = self.transform(img) # 对图片进行处理，例如裁剪图片，或者数据扩增 return {'img': img, 'paths': path} # 返回数据 def __len__(self): # 获取数据集大小 return len(self.paths) def name(self): # 获取数据集名称 return 'VocImageDataset'

其中定义函数的image_folder.py文件

  • 数据集整理部分
import osimport os.path# 图片文件的判别IMG_EXTENSIONS = [ '.jpg', '.JPG', '.jpeg', '.JPEG', '.png', '.PNG', '.ppm', '.PPM', '.bmp', '.BMP',]def is_image_file(filename): return any(filename.endswith(extension) for extension in IMG_EXTENSIONS)# 将路径下所有图片文件的路径添加在一个列表中def make_dataset(dir): images = [] assert os.path.isdir(dir), '%s is not a valid directory' % dir # 判断是否为文件路径 for root, _, fnames in sorted(os.walk(dir)): # 获取文件夹路径和所有文件名 for fname in fnames: if is_image_file(fname): # 判断是否为图片文件 path = os.path.join(root, fname) # 将图片文件名和文件夹路径生成图片路径 images.append(path) # 将图片路径添加至列表中 return images
