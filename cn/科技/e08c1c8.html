微信支付、微信提现、提现到银行卡的RSA公钥加密

2020-10-29
| 科技

由于微信支付最近用的挺多的，最近花了点时间把微信相关的接口封装成了一个npm,直接安装就能使。本模块集成了大部分微信支付、提现等模块的接口，采用async、await的方式调用，使用者不用再考虑参数加密发送，密钥发送方式、xml怎么解析、json怎么转成xml等一系列麻烦事。

注：

noncestr（可以自定义）、sigin不需要传入，所必须的参数值请看微信文档，sign_type支持MD5,HMAC-SHA256,不传默认MD5

微信支付

安装

yarn add node-wxpay3 或者 npm i node-wxpay3 --save

支付产品api

名称（方法名）

统一下单（unifiedorder）

回调验证（callback_check）

查询订单（orderquery）

关闭订单（closeorder）

申请退款（refund）

查询退款（refundquery）

下载交易账单（downloadbill）

下载资金账单（downloadfundflow）

交易保障（report）

拉取订单评价数据（batchquerycomment）

付款码支付（micropay）

撤销订单（reverse）

付款码查询openid（authcodetoopenid）

例子：

const { WxPay } = require('node-wxpay3');

const fs = require('fs');

const wxpay = new WxPay({

'appid': 'appid',

'mch_id': '商户id',

'partner_key': '', // partner_key为商户平台设置的密钥key

'pfx': fs.readFileSync('./apiclient_cert.p12'), // 证书

});
