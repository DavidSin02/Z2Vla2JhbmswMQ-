最好的DJ音箱

2020-11-14
| 科技

作为表演DJ，要把音箱放在和打碟机、混音装置同等重要的位置。诚然，混合音轨需要一些技巧，但是如果没有高质量的音箱将这些音乐播放出去，那么舞池里的人群可能会跳着跳着就睡着了。无论您是常驻DJ还是巡演DJ，选出最好的DJ音箱都不是件容易的事儿，因为可选的音箱种类实在是太多了，今天我们将会就这点给大家提供些建议，希望这些建议会对您有所帮助。

我们还会对产品的功能做详细的介绍，来帮助您做出选择。因为音箱之间的差异可能非常细微，所以这会是您挑选出最适合您的DJ音箱的有利的工具。

注意：我们希望我们的建议对您来说更加实用，所以我们不会仅简单的罗列要推荐的音箱，我们的推荐不会只考虑音箱本身。以下推荐的产品排列顺序不分先后，基本上每组音箱都能满足您的所有需求。

超低音音箱很重要

任何想要购买新的DJ音箱的DJ都可能正在寻找一款完全匹配的超低音音箱。 如果您没有超低音音箱，您一定要考虑购买一台。 低频之于电子音乐或嘻哈音乐就像牛奶之于麦片一样必不可少。 由于波长较长，超低音音箱还可以为您的声音增添更多魅力，而且超低音音箱能让人们更有跳舞的欲望。 因此，我们在下面的大多数推荐中都加入了一个超低音音箱。

即插即用

除了完善您的音域之外，DJ音箱还需要一些额外的配件，这些配件通常不会和音箱随附。下面的每组推荐中都有用于将音箱堆叠到超低音音箱，在任何阶段互联它们以及在演出后保护它们等用途的配件。

需要关注的点

  • 有源/无源—虽然我们首推有源音箱，但市场中也有许多无源音箱可供选择。如果您选择使用无源音箱，那么您还要为它配备一个供电的外置放大器，所以在大多数情况下，有源音箱是一个更好的选择，因为它能为您的运输减负。
  • 功率—影响音箱实际音量输出（额定声压级或SPL）的重要因素是音箱的功率输出，功率以瓦特为单位。根据RMS(均方根)或峰值功率的不同，额定功率会有很大的差异。RMS是放大器的连续功率的额定值，而峰值则是在非常短的时间内(有时是几分之一秒)最高输出功率的额定值（没有功率损耗）。 如果您要比较规格，请确保进行对比的是RMS与RMS或峰值功率与峰值功率， 否则功率比较毫无意义。 相比较而言，RMS是衡量真实功率更好的指标。
  • 材质—如今的音箱箱体不仅可以用木头做，还可以用注塑塑料(ABS)，ABS材质的箱体结构更稳定并且能降低音箱的总体重量，是用于巡演的音箱的最佳选择。
  • 重量—音箱的重量是巡演DJ需要考虑的重要因素。尺寸比较大的音箱在之前是获得比较大的音量的有利工具，但现在却不然。找到一个强大而不庞大的音箱是最理想的。
  • 输入/输出—音箱输入和输出设置因音箱而异。 关于输入，大多数音箱都有1/4″插孔，XLR插孔或这两者都适用的combo插孔; 有些音箱也有蓝牙无线输入或专有的物理输入（speakON或类似的东西，通常在无源音箱中出现）。 许多有源音箱还带有XLR格式的输出或直通选项，用于连接链中的其他音箱。 某些音箱可能还会有以太网插孔，用于连接软件以进行配置和其他控制选项。 要注意检查输入和输出接口，以确保音箱与您现有的需求和装备匹配。
  • 便携性—为了便于移动，有些音箱上面自带握把和脚轮。刚开始您可能并不会觉得这种设计有多重要，但当您移动音箱成百上千次之后，您就会知道这种设计的好处了。
  • 成本—您的预算是需要您着重注意的，但最好不要因为价格牺牲质量。否则，您可能很快就需要再买一套新的音箱。
  • 箱体和外观—您打算如何和在哪里使用您的新DJ音箱?是在室内、室外还是两者皆有? 是在婚礼还是俱乐部? 音乐风格是嘻哈，电子，还是流行? 当您在购买音箱时，确保您已经明确其使用的场景，根据需求进行购买。

EV的ZLX和ELX系列是性价比很高的DJ音箱组合。ZLX-12P是一款有源的双功放（1000瓦峰值和250瓦RMS额定功率）音箱，配有12英寸低频驱动器和1.5英寸钛合金压缩高音单元， ZLX使用涂层聚丙烯复合塑料制做箱顶和侧面手柄，这种材质比木质材料更轻（仅34磅），更便于运输。 在后面板上，您可以看到一个简单的输入和输出设计，其中包含麦克风，乐器或电路板馈送的组合输入。 此外，ZLX-12P还具有LCD显示屏，可轻松调节信号处理。

ELX118P有源超低音音箱具有700瓦的峰值功率和能处理134dB最大声压级的18英寸EVS-18K低音单元。ELX118的箱体是15毫米彩绘胶合板材质，这种材料可以减少长期的磨损造成的箱体损坏。 如果您在设置中需要一些额外的低音，可以切换后面板上的boost开关，并且如果您过度使用了您的音箱，限制器指示灯会给您发出警告。

QSC组合

3 x QSC K10.2

3 x QSC K10 Tote

2 x QSC KS212

2 x QSC KS212C

QSC在此类应用中的口碑不断增长，其K系列最受欢迎， K.2就是一个很好的例子。 K10.2音箱是配有双功放，10英寸喇叭和1.4英寸高音单元的组合，具有令人印象深刻的2000瓦D级功率模块 – 功率是之前K系列音箱型号的两倍。 K.2 系列可实现多样化功能及灵活性应用。可对多个参数进行配置，实现多种功能：舞台监控、舞曲播放、乐器扩声、手持话筒扩声等；同时，可对应用场景进行储存，以便于调用定制化参数配置，如输入通道类型、延迟、均衡、分频及所选界面模式等。单凭这些，这个系列的音箱就能成为DJ们的最佳选择。

这些音箱是演出DJ的绝佳选择的另一个原因是那对儿KS212有源超低音音箱。KS212内置双12英寸低音单元，每只单元带六段带通滤波器 ，由3600W功放模块推动，同时由系统DSP处理功能控制，使设备前方投射的声能比后方多15dB。 该产品除了配备有高级DSP 处理功能以外，也可对场景参数进行储存调用（通过显示屏、控制面板操作）， 满足用户在不同使用环境下的使用需求。

JBL组合

2 x JBL SRX835P

1 x JBL SRX818SP

2 x On-Stage Stands SS7745

DJ和PA音箱领域中最广为人知的品牌之一就是JBL。 其SRX835是一款有源，配有15英寸和6.5英寸喇叭、顶部配有3英寸高音单元的音箱。 SRX835的这种设计可以使其覆盖几乎整个人类听觉频率范围（33Hz至21kHz）。 它们有2000瓦峰值功率、1500瓦连续额定功率，这对于功能强大的Crown Audio音箱来说非常有意义。 至于设置，SRX835具有灵活的板载DSP预设和可调参数。 不过，多亏了可以完全控制您的高端PA系统的Audio Architect，HiQnet Motion Control iOS和其他独立应用程序，JBL对音箱设计的发展更进一步。

SRX818有源超低音音箱非常强大。 每个818都有750瓦连续额定功率，1000瓦峰值D类额定功率，最大声压级输出为135dB。 它由与835相同的软件控制，当串联使用时，您可以对系统进行最终的声音控制。 这也意味着您可以随身携带这个系统到任何需要它的地方，并且它在任何地方都能听起来很棒。

Bose组合

Bose F1 PA Package

Bose是家庭和商用音箱创新的领导者，其F1系列音箱就是一个很好的例子。 F1系列非常适合演出DJ，因为它方便，便携的设计和灵活的应用程序能满足各类场地需求。例如，F1 MODEL 812音箱配有八只Bose 专利的 2.25 英寸单元、100°水平波导、一只高功率 12 英寸低音单元和较低的分频点，F1 MODEL 812 音箱具有出色的最大声压级，性能表现明显优于传统音箱。 如需进一步提升低频响应，可选的 Bose F1 超低音音箱能将大型低音音箱的所有性能融入到更加紧凑的设计之中。Bose 专利的音箱安装支架集成在超低音音箱箱体之上，便于快速安装。支架内含线槽，音频与电源接线可整齐地隐藏于其中，整体视觉效果更美观。

便携式婚礼DJ

Yamaha StagePas 600BT便携式PA系统

如果您正在寻找一个配有快速设置和包含混音器的婚礼DJ音箱组合，那么Yamaha StagePas 600BT组合是您的理想之选。 600BT具备两个美观且轻量化的音箱以及一个可拆卸的有源调音台，同时还包含一对音箱线和一根电源线。 10通道调音台包含4个 mic/line 输入以及6个 line 输入。输入端口3和4采用combo接口，因此可以接插XLR或者1/4插口的线缆。此外，600BT还设置了立体声 RCA 接口，立体声 mini 接口以及 1 个可用于连接智能手机、平板或者电脑的无线蓝牙播放接入功能。Yamaha StagePas 600BT价格也很实惠，是一款很棒的婚礼DJ设备。

一个非常经济的选择

Mackie SRM450组合

我们推荐的的经济型DJ设备是Mackie SRM450音箱，音箱支架和电缆。 SRM450是1000瓦，12英寸有源音箱，内置双声道调音台。 为了便于调整声音，SRM具有可选择的音箱语音选项，如PA，DJ，监听和独奏模式。 为了保持声音干净，每个音箱都采用了专利的声学校正算法、双路拨号交叉、相位校正和一个易于使用的自动反馈抑制器进行优化。 与随附配件搭配使用，无论是对于初学者或者是老手来说，SRM450都是一个非常不错的选择。
