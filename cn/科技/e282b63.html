Windows 10更新Bug禁用了文件资源管理器右键单击

2020-10-29
| 科技

IT之家11月28日消息 Windows 10版本1909（2019年11月更新）正在逐步推送。2019 Windows 10更新十一月版是一个小更新，它有望带来更好的性能，但这并不意味着一切都会顺利进行。

正如我们上周报道的那样，2019 Windows 10更新十一月版中的错误Bug让文件资源管理器冻结并最终崩溃。用户搜索文件、文档或图片时，文件浏览器的搜索栏将无响应。

虽然尚不清楚是什么原因导致文件资源管理器出现问题，但这似乎是对微软新的统一搜索体验的不良反馈。

现在2019 Windows 10更新十一月版引入了另一个烦人bug，用户无法在搜索窗格中右键单击。CTRL + V仍然有效，但是资源管理器会挂起几秒钟，然后用户才能输入文本。

根据10月22日创建的另一个帖子，该问题首先由微软社区测试人员报告，然后该公司向消费者发布了2019年11月更新。

“我有完全相同的问题：当我右键单击资源管理器搜索框时，我什么也没得到，没有粘贴也没有复制。用户单击后得到的只是过去搜索词的历史记录，甚至无法删除。”某用户表示。

文件资源管理器搜索框中的右键单击支持是一项必不可少的功能，用户显然很生气。不幸的是，目前还没有针对该错误Bug的解决方法，但你可以使用传统的Windows搜索来查找文件。
