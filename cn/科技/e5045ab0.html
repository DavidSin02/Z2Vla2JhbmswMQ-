第四步：数据类型？什么鬼？

了解数据类型之前，我们得先知道什么是数据

数据：是指对客观事件进行记录并可以鉴别的符号，是对客观事物的性质、状态以及相互关系等进行记载的物理符号或这些物理符号的组合。它是可识别的、抽象的符号。

好吧，这样的定义看着醉醉的，简单点说吧，就是你能在电脑上看到的、看不到的一切具有实际意义的东西都叫做数据，比如说你现在看到的这篇文章，他就是数据，上面的图片，他也是数据，为了对这个不同的数据进行区分和处理，这里就有了【数据类型】这个词，也就是用来对不同的数据进行分类之后，便于计算机对相应的数据进行处理。

那么，有哪些数据类型呢？简单来说，有以下这么几种：

图片来自www.runoob.com

在这里我们就说简单的几个，int，doule，float，char

  1. int：整型，就理解为整数吧
  2. float：单精度浮点型，就理解为位数比较少的小数呗
  3. doublle：双精度浮点型，额这个就理解为位数比较长的小数
  4. char ：字符型 ，就是这样的'a','b'这些都叫字符

上面说的这些都叫基础数据类型，还有一种比较高级的叫引用数据类型，具体的理解会在后面有专门的解释，现在大概可以理解为，由多种基础数据类型组成的复杂数据类型，可以是系统中已经定义好的，也可以后面我们自行定义这样的数据类型

第五步：哪这些数据类型怎么用呢？

先看这一段代码：

int a = 100;

double b = 100.00;

float c = 100.00f;

char d = 'a';

String e = "HelloWorld";

这几句代码的意思是定义变量，基本格式是这样：

【数据类型】 变量名 ；

如果是这样：

【数据类型】 变量名 = xxxxx ；则是在定义变量后，并给他赋值

这样，我们就可以将数据保存起来，在后续我们需要使用的地方进行使用了。

第六步：保存的数据怎么用呢？

用处挺多的，简单的来说一下运算符这个东西吧，挺复杂的

//赋值运算符

a = 250;//在这里，【=】 就是赋值运算符 作用是将等号 右边 的值赋给 左边 的变量

//四则运算符 之 加

b = a + 100;//在这里，【+】 就是加法运算符 作用是将加号 左右两边的值相加

//四则运算符 之 减

b = a - 100;//在这里，【-】 就是加法运算符 作用是将加号 左右两边的值相减

//四则运算符 之 乘

b = a * 100;//在这里，【*】 就是加法运算符 作用是将加号 左右两边的值相乘

//四则运算符 之 除

b = a / 100;//在这里，【/】 就是加法运算符 作用是将加号 左右两边的值相除
