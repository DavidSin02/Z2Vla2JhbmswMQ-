常用资源网站大全

2020-10-29
| 科技

资源搜索：

  • Gitlogs -- Gitlogs 是专门针对 GitHub 项目的搜索引擎，我们通过他可以快速找到想要项目。
  • 鸠摩搜书 -- 可以用来找一些技术文档手册，很多在百度网盘里。
  • Stack Overflow -- 大部分编程问题都能在这里找到答案。
  • Github -- 最大的开源项目资源网站。
  • Iconfinder -- 用来查找一些类似 logo 都小图标。
  • TinEye -- 用图片来检索图片，我们可以上传图片或输入图片的 URL 来检索。
  • SemanticScholar -- 一个免费学术搜索引擎，其检索结果来自于期刊、学术会议资料或者是学术机构的文献。
  • LibreStock -- 可以检索一些优质的高清图片。
  • CC Search -- CC Search 上搜索到的图片资源都是无版权的，我们可以免费的使用。
  • Pexels -- 高质量的图片网站，可以免费使用。
  • Unsplash -- 免费高清素材网站。
  • The App Store -- 一个针对苹果手机、iPad、Mac 设备的应用搜索工具。
  • Ebooke -- 一个基于互联网并提供免费电子图书下载的搜索引擎网站。
  • Goobe -- 为程序员服务的互联网搜索引擎

    • 图片和文档资源

  • TinyPNG -- 免费的在线图片压缩工具，压缩率高，无损画质，直接拖拽使用，很方便。
  • picdiet -- 图片压缩工具，能极速压缩 80% 的图片大小，而不损害其质量。
  • SmartResize -- 这个工具基于 JavaScript+HTML5 编写而成的，是一个批量裁剪缩放图片的神器！
  • Convertio -- 在线转换文件的简单工具。支持超过309种不同的文档、图像、电子表格、电子书、文档、演示文稿、音频和视频格式。
  • Online converter -- 可以转换视频、音频、图片、文档等。
  • PDF to Word Converter -- PDF 文档转化工具。
  • PDF24 -- 免费且易于使用的在线 PDF 工具。
  • 在线文件转换器 -- 免费在线转换视频,在线转换音频,在线转换图形,在线转换文档和压缩。
  • Xmind -- 跨平台、跨终端，拥有全功能的思维导图软件。
  • ProcessOn -- 免费在线作图，实时协作，支持流程图、思维导图、原型图、UML、网络拓扑图、组织结构图等。
  • MindMaster -- 多平台思维导图软件，可用于Windows，Mac和Linux等桌面环境，也可以在线使用或在苹果，安卓等移动端上使用。
  • MindMeister -- 在线思维导图工具，已有超过1000万个用户。
  • 百度脑图 -- 简洁的在线思维导图工具。
  • Gliffy -- 可以在线创建流程图、UML图、线框图、BPNM、组织架构图等。
  • 印象笔记 -- 在你的手机和电脑上，使用印象笔记来捕捉、培养和共享灵感。
  • 有道云笔记 -- 提供了PC端、移动端、网页端等多端应用，用户可以随时随地对线上资料进行编辑、分享以及协同。
  • OneNote -- 微软出品等云笔记。
  • 幕布 -- 在幕布中，大纲笔记与思维导图可以一键转换
  • 为知笔记 -- 一键收藏、全端全文检索、多级目录、Markdown
  • 石墨文档 -- 一款轻便、简洁的在线协作文档工具
  • Google keep -- Google 云笔记平台。
  • Simplenote -- 可帮助您轻松记录笔记、列表、想法等。

    • 编程工具

  • RegExr -- 正则表达式在线测试工具。
  • Regulex -- 正则表达式在线测试工具。
  • rextester -- 在线执行 PHP、Python、C、Java 等各种语言代码。
  • Ideone -- 在线执行 PHP、Python、C、Java 等各种语言代码。
  • GDB online Debugger -- 在线执行 PHP、Python、C、Java 等各种语言代码。
  • JSFiddle -- 前端代码在线执行工具。
  • CodePen -- 前端代码在线执行工具。
  • JSON 解析器 -- 在线 JSON 工具，可以格式化 JSON 数据。
  • ToolFK -- 程序员在线工具
