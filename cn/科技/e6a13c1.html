2019年必看的22部爽片，先马后看！

2020-10-29
| 科技

尽管离2019年还有大半年，但按照好莱坞「先定档，再开拍」的成熟行规，19年全年主要大片的档期，大多数都已经安排上啦。

所以扒叔特地为你们整理了，明年最值得期待的那些好莱坞顶尖大片。

一共有22部，先马再看！

1

《地狱男爵》1月11日上映

导演: 尼尔·马歇尔

主演: 大卫·哈伯 / 伊恩·麦柯肖恩 / 米拉·乔沃维奇 / 金大贤 / 萨莎·莱恩

《地狱男爵3》流产后，该系列迎来了重启版，新《地狱男爵》目前被定为了R级，并敲定由《汉尼拔》、《西部世界》的导演尼尔·马歇尔执导，《怪奇物语》里的警官大卫·哈伯担任新男爵。

期待值★★★★

2

《玻璃先生》1月18日上映

导演: M·奈特·沙马兰

主演: 詹姆斯·麦卡沃伊 / 安雅·泰勒-乔伊 / 布鲁斯·威利斯 / 塞缪尔·杰克逊

别被片名唬住了，其实它是24重人格精分电影《分裂》的第2部。

相信一美精湛的演技（肉体）依然是本片最大亮点。

期待值★★★★

3

《X战警：黑凤凰》2月14日上映

导演: 西蒙·金伯格

主演: 苏菲·特纳 / 詹姆斯·麦卡沃伊 / 迈克尔·法斯宾德 / 詹妮弗·劳伦斯 / 杰西卡·查斯

这应该是很多人最期待的X战警系列电影了。目前关于《黑凤凰》的消息并不多，但据说《黑凤凰》将一改《天启》那样轻松的故事，回归《第一战》那种严肃黑暗的氛围。

不过很多人期待的还是可以看到一美和老万了吧。

期待值★★★★★

4

《驯龙高手3》3月1日上映

导演: 迪恩·德布洛斯

主演: 杰伊·巴鲁切尔 / 克里斯汀·韦格 / 凯特·布兰切特 / 杰拉德·巴特勒

《驯龙高手3 》由梦工场动画出品，该系列是影史上最受欢迎的动画系列之一，而这部备受期待的新片是该系列的第三部。

需要注意的是续集里连无牙仔都有女朋友啦！

期待值★★★★★

5

《惊奇队长》3月8日上映

导演: 安娜·波顿 / 瑞安·弗雷克

主演: 布丽·拉尔森 / 裘德·洛 / 安妮特·贝宁 / 嘉玛·陈 / 塞缪尔·杰克逊 / 本·门德尔森 / 李·佩斯

不夸张的说《复联3》都算是为这部电影做的铺垫，

而《惊奇队长》在《复联4》之前上映前，也说明，她真是全村的希望了！

期待值★★★★★★★

6

《小飞象真人版》3月29日上映

导演：蒂姆·波顿

主演: 伊娃·格林 / 迈克尔·基顿 / 科林·法瑞尔 / 丹尼·德维托 / 道比·欧帕瑞

迪士尼经典动画《小飞象》于1974年首映，经过76年的时间，记忆中的小飞象又回来了。

真人版同样讲述了马戏团里的一只小象因耳朵巨大被动物和人类嘲笑，但最终小象发现自己耳朵可以飞翔，并一跃成为马戏团明星……

总之蒂姆·波顿执导的电影一定要期待！

期待值★★★★★

7

《沙赞》4月5日上映

导演: 大卫·F·桑德伯格

主演: 马克·斯特朗 / 扎克瑞·莱维 / 杰克·迪伦·格雷泽 / 亚瑟·安其

沙赞是DC漫画旗下一位人气极高的超级英雄，

原漫里他从小是个孤儿，以卖报为生。

后来他遇到了一个古老的巫师并赐予了他来自神话英雄的六种超能力：智慧、力量、毅力、能力、勇气和速度。所以每当他高喊“Shazam(沙赞)，巴特逊就可以变身为超级英雄！

事到如今，只希望DC别把这张好牌打烂啊。

期待值★★★★★

8

《复仇者联盟4》5月3日上映

导演: 安东尼·罗素 / 乔·罗素

主演: 乔什·布洛林 / 布丽·拉尔森 / 小罗伯特·唐尼 / 克里斯·海姆斯沃斯 / 克里斯·埃文斯 / 斯嘉丽·约翰逊 / 马克·鲁弗洛 / 塞巴斯蒂安·斯坦 / 杰瑞米·雷纳 / 查德维克·博斯曼 / 保罗·路德 / 凯伦·吉兰 / 唐·钱德尔 / 戴夫·巴蒂斯塔 / 伊万杰琳·莉莉 / 格温妮斯·帕特洛 / 乔恩·费儒 / 泰瑞·诺塔里 / 泰·辛普金斯 / 真田广之 / 斯坦·李

不多说，这部《复活者联盟》只要国内同步上映，就让我们看爆好吗！

期待值★★★★★★★★★★

9

《疾速追杀3》5月17日上映

导演: 查德·斯塔赫斯基

主演: 基努·里维斯 / 哈莉·贝瑞 / 约翰·雷吉扎莫 / 兰斯·莱迪克 / 伊恩·麦柯肖恩

《疾速追杀3》的故事会承接第二部，续集中有人向里维斯饰演的杀手开出了价值1,400万美元的“追杀令”，因此他开始亡命天涯。

期待值★★★★

10

《阿拉丁真人版》5月24日上映

导演: 盖·里奇

主演: 梅纳·玛索德 / 娜奥米·斯科特 / 威尔·史密斯 / 马尔万·肯扎里 /

作为迪斯尼一系列经典动画真人翻拍计划之一的《阿拉丁》由英国名导盖·里奇执导，故事还是那个故事，并且影片仍旧会保留一些1992年动画版的经典音乐元素。

期待值★★★★★

11

《哥斯拉2》5月31日上映

导演: 迈克尔·道赫蒂

主演: 米莉·波比·布朗 / 维拉·法米加 / 凯尔·钱德勒 / 章子怡

作为传奇影业怪兽宇宙的重要一环，《哥斯拉2》的故事承接《金刚：骷髅岛》里的“帝王计划”展开。

并且本次哥斯拉的老对手拉顿、摩斯拉、三头龙基多拉都可能登场。

值得一提的是，本次新加盟的演员还有章子怡以及《怪奇物语》小11。

期待值★★★★

12

《玩具总动员4》6月21日上映

导演: 乔什·库雷

主演: 汤姆·汉克斯 / 琼·库萨克 / 蒂姆·艾伦 / 邦尼·亨特 / 安妮·波茨

时隔20年，玩具总动员系列终于迎来了第4部。

第四部会讲述胡迪和牧羊女的爱情故事，而观众喜欢的巴斯光年以及玩具小伙伴们的生活也会翻开全新篇章。

期待值★★★★★

13

《蜘蛛侠2》7月5日上映

导演: 乔·沃茨

主演: 汤姆·赫兰德 / 赞达亚 / 雅各布·巴特朗

别紧脏！《蜘蛛侠2》紧接《复联4》剧情，小蜘蛛肯定是要复活的，只是没有正式海报，先拿这个饭制的顶着

目前的得知的消息是，续集中杰克·吉林哈尔有望加入，并饰演大反派神秘客。而在第一部中有精彩表现的“秃鹫”迈克尔·基顿也将回归。

嘻嘻，明年暑假又能吸荷兰弟啦！

期待值★★★★★★

14

《壮志凌云2》7月12日上映

导演: 约瑟夫·科辛斯基

主演: 汤姆·克鲁斯 / 方·基默

1986年上映的《壮志凌云1》是一部空战题材的影片，同时也是汤姆·克鲁斯的成名作。

33年后，飞行员Maverick回归。续集背景设定在现代，在无人机科技发达的世界中，飞行员老派的飞行方式变得极为过时，他们的地位也岌岌可危。

期待值★★★★★

15

《狮子王真人版》7月19日上映

导演: 乔恩·费儒

主演: 唐纳德·格洛弗 / 碧昂丝·诺尔斯 / 詹姆斯·厄尔·琼斯

《狮子王真人版》由拍过《奇幻森林》的导演乔恩·费儒操刀，并且该片貌也是由纯CG特效完成制作，听说还会采用VR技术。

毕竟是迪士尼的经典老IP，可以期待一下的！

期待值★★★★★

16

《新变种人》8月2日上映

导演: 约什·布恩

主演: 安雅·泰勒-乔伊 / 麦茜·威廉姆斯 / 艾莉丝·布拉加 / 查理·希顿

二十世纪福斯准备拓展“X战警”系列，所以立项了《新变种人》

《新变种人》算是《X战警》漫画比较冷门的分支，角色都是X教授和万磁王的学生，能力都比较特别。可以观望一下。

期待值★★★★

17

《霍伯斯与肖》8月2日上映

导演: 大卫·雷奇

主演: 道恩·强森 / 杰森·斯坦森

这部电影其实就是《速度与激情外传》

影片故事也将围绕着卢克·霍布斯（道恩·强森）和迪卡·肖（杰森·斯坦森）展开

电影临近上映应该会改回速激系列的名字，毕竟这个IP在国内的吸精能力是有目共睹的。

期待值★★★★

18

《好莱坞往事》8月9日上映

导演: 昆汀·塔伦蒂诺

主演：莱昂纳多·迪卡普里奥、布拉德·皮特、阿尔·帕西诺、詹妮弗·劳伦斯、萨缪尔·杰克逊

《好莱坞往事》是昆汀·塔伦蒂诺的第九部电影作品，这部电影的亮点全体现在阵容上了，这个要看！

期待值★★★★★★

19

《小丑回魂2》9月6日上映

导演: 安德斯·穆斯切蒂

主演: 杰西卡·查斯坦 / 比尔·斯卡斯加德 / 詹姆斯·麦卡沃伊

《小丑回魂2》设定在第一部过去的27年后，片中的小演员都长大了。

目前已确认一美会扮演成年版的比尔，杰西卡·查斯坦扮演成年的贝弗莉，比尔·哈德尔扮演成年的Richie。当然，四弟将会继续回归扮演小丑潘恩怀斯！

期待值★★★★

20

《神奇女侠2》11月1日上映

导演: 派蒂·杰金斯

主演: 盖尔·加朵 / 克里斯汀·韦格 / 佩德罗·帕斯卡

派派复活了！昨天《神2》发布官方片场照，在《神1》里明明和飞机一起炸了的男主史蒂夫竟然又回到了片场

，回归应该是没跑了。

而这次续集的故事背景将设置在上世纪80年代美苏冷战时期，戴安娜的对手也来自来自苏联。

妇女能顶半边天真不是随便说说的，明年第2期待的超级英雄片，看起来！

期待值★★★★★★

21

《冰雪奇缘2》11月27日上映

导演: 克里斯·巴克 / 珍妮弗·李

主演: 伊迪娜·门泽尔 / 克里斯汀·贝尔 / 乔什·加德

《冰雪奇缘2：家人的思念》讲述了在长期关闭的城门再度开启后，安娜和艾丽莎一同度过的第一个圣诞节的故事。

而我更期待的是，这次将会有比经典的《Let it go》更棒的插曲出现！

期待值★★★★★

22

《星球大战9》12月20日上映

导演: J·J·艾布拉姆斯

主演: 黛西·雷德利 / 亚当·德赖弗 / 奥斯卡·伊萨克 / 约翰·博耶加

据外媒NY Daily报道，传奇角色尤达大师将回归到《星球大战9》和之前一样，他将继续以原力魂魄导师的形式出现。不多说，明年票房大牛应该就是星战8和复联4拼了，迪士尼真是笑开了花。

期待值★★★★★

明年最值得期待的22部大片都在这了，

光是超级英雄片就有8部！而动画也是明年的重头戏。

可以预见，未来的北美票房年冠，迪士尼又或成为最大赢家。
