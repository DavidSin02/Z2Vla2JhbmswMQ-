A、阮籍 B 、嵇康 C 、刘伶 D 、向秀

A、“大元” B 、“解元” C、“进士” D、“榜眼”

A、李白 B 、杜甫 C 、李商隐 D 、 李清照

A、自叙帖 B 、自言帖 C 、中秋帖 D 、黄州寒食帖

A、史记 B 、汉书 C 、三国志 D 、资治通鉴

A、李商隐 B 、李贺 C 、李白 D 、李煜

Ａ、周 B 、元 Ｃ、商 D 、夏

A、永济 B 、桑落 C 、汾阳 D 、绵竹

A、孟子 B 、韩非子 C 、荀子 D 、老子

A、嵇康 B 、曹植 C 、山涛 D 、阮瑀
