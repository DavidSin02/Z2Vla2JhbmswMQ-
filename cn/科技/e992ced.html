戳戳戳戳戳，她用一根针记录最柔软的生活

2020-10-29
| 科技





认识囍囍是在朋友的工作室，她低头戳羊毛，偶尔轻轻说一两句话，几乎从热闹的房间里隐形。

桌面上放着一些软蓬蓬的鸟、奇怪的虫子、一只穿了雨靴的鸭子……所有奇奇怪怪的小物件，全都是羊毛毡。

囍囍是她们的主人，戳羊毛毡是她的工作和生活。






囍囍是湖南人，80后，因为工作来到武汉。她的微信签名写着“万物都萌”，热爱把所见万物戳成可爱的羊毛毡。

囍囍戳各种好吃的

街角小摊卖的烤红薯，表皮焦脆，掰开还散发热气 ，还有炒栗子，糖葫芦，甜甜软软的冬季三件套。




吃过的雪糕棒留下来，自己创造新鲜口味，永远不会融化的俏皮可爱 。


夏天的最后一块西瓜 ，尖尖上那口最甜，再吃得等到明年夏天。


快餐店的汉堡 总是图文不符，自己戳一块超大鸡排，一口下去绝对满足。



她也戳一些“有的没的”

在故宫看展览，喜欢一只清代胭脂红釉碗，自然不能带走，自己戳一个 。



水彩颜料用完，再戳一盒，一不小心还有猫咪乱入。



路过公园，偶遇长椅上温馨画面，用羊毛毡复制出来 ⛄。




从初中时就喜欢《犬夜叉》，去年到日本旅行前戳了一只玩偶同行，会有路人惊呼，“inuyasha”！


囍囍以前是墙绘师，武汉的许多商场、学校都有她和同伴的作品。

“ 有一天我发现了羊毛毡，你只需要一坨羊毛，一根针，一块泡沫垫子，就可以做出一切来。”

绘画功底让囍囍总能很好地观察和重塑这个世界，生活里不大不小的碎片，她都用羊毛毡捕获。



囍囍的工作桌也很简单，和许多手工艺人不同，她不囤积材料，也没有复杂的工具，一张白桌放着全部必需品。

问囍囍做羊毛毡的要领，她说就一个字，戳。戳针上的细小倒刺能让羊毛纤维融合在一起，最后成为想要的形状，她就这样不声不响地坐在桌前，一戳一整天。

“ 不像一些手工，铺好大的摊子，都不知道从哪里开始，收也要收半天，太麻烦了，羊毛毡就是简单，我喜欢简单。”



△ // 囍囍在这张小桌上创造着奇妙的世界

作品分享到朋友圈和微博，慢慢有人想买，囍囍干脆把戳羊毛毡彻底变成了自己的职业。

囍囍的客户通常是家人、情侣、主人与宠物。像摄影师按下快门，囍囍把这些柔软而美好的故事，戳进一团团彩色的羊毛里。




戳一幅客户作品，通常需要半个月，根据尺寸、要求不同，价格400元起。


△ // 每年都向囍囍定制一幅作品的客人，羊毛毡里的宝宝在悄悄成长。


△ // 一家六口，用羊毛毡做全家福，比拍照片更温暖。


△ // 新婚夫妻的“婚纱照”，囍囍戳了个红红的“囍”字。


△ // 和宠物合影的妹子，囍囍按她的要求，加入了从猫咪身上梳下来的毛。

“ 不停戳羊毛的时候，手动得很快，但时间好像就慢下来了，感觉自己的灵魂飘走，放空的感觉超爽！”

但自由职业也有它的烦恼，囍囍要用订单养活自己，创作想法列了一大串，却只能在订单之余，拼命挤时间来实现。



△ // 囍囍会戳出身边的朋友，给纯色衣服戳装饰，或者戳一只小鸟，陪自己去旅行。

“ 我有太多想要做的了，一些情绪、回忆，生活里的小细节，秋天的风景，一场电影什么的，不过，这就像月亮和六便士，说到底还是得先搞定生活。”

不做羊毛毡的时候，囍囍会陪自己的暹罗猫“卡妹”一起玩，用羊毛戳了很多小球球扔在家里的角落给卡妹抓来抓去，或者把卡妹身上梳下来的毛收集起来，做成挂件挂在包包上。


△ // “今天也要开心鸭！”


△ // 囍囍戳出来肉呼呼的小“卡妹”


△ // 人物眉眼的小细节，也是用羊毛细心戳成

家乡在湖南，囍囍一个人住在武汉，自由安排的生活里，非常偶尔的瞬间，她也会有这座拥挤城市里所有年轻人共有的“三十岁焦虑”。

她也像上班族一样，肩颈痛得厉害，办了健身卡，开始认真养生；头痛过户口、房子、收入的问题；觉得自己不能太佛系，期待恋爱，想要多出去走走。

不过大部分时候，囍囍还是会轻声细语地说，人生短短，做有趣的事，自在就好。




△ // 囍囍在镰仓的海边
