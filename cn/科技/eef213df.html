为什么我们需要RF 24-105mm F4-7.1 IS STM镜头

2020-11-14
| 科技

佳能RF镜头系统发布后，F2级别的变焦镜头以及F1.2级别的定焦镜头吸引了市场上绝对的关注，超群的大光圈规格和优秀的画质表现，让人们对RF镜头群充满了厚望。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


不过在针对专业摄影师和发烧友的范畴之外，佳能仍然没有忘记为RF系统推出易用、轻松、高性价比的入门级RF全画幅镜头，RF 35mm F1.8镜头就是其中的典型代表，而在定焦镜头之外，今年佳能带来了一支轻量化、性价比高的标准变焦镜头RF 24-105mm F4-7.1 IS STM。


轻量化标准变焦镜头


大部分普通相机用户，他们对于镜头的要求并不高，并不需要4000线对的分辨率，也不需要十几组的结构来矫正像差和色散，他们需要的是一支素质在水准之上，又非常轻便，可以让自己轻松使用的镜头。


EOS RP / RF 24-105mm F4-7.1 IS STM / F8 1/2000 ISO100


RF 24-105mm F4-7.1 IS STM就是定位于此的一支广角到中长焦的标准变焦镜头，最突出的特点是其重量仅为395g，差不多是F4级别24-105mm镜头的一半重量。同时镜头的和长度，也有不同程度的缩减，这支镜头最大直径为76.6mm，收纳长度为88.8mm，在RF镜头群中可以说是最为轻盈的变焦镜头，整体镜头设计思路就是给用户提供最大程度的便携。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/500 ISO100


从同类镜头的规格来说，这支镜头是重量、体积最为轻盈的一枚24-105mm变焦镜头，搭配EOS RP这样的轻量化全画幅机身，整体重量不超过1000g，对于旅行、日常拍摄来说携带非常轻松。


焦段覆盖范围广泛


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


24-105mm的镜头焦距覆盖范围颇为「万金油」，从广角到中长焦，包含了24mm广角，35mm、50mm常用焦距，85mm、105mm中长焦距，可以应对人像、静物、风光、建筑甚至微距。


EOS RP / RF 24-105mm F4-7.1 IS STM / F5.6 1/2000 ISO100


广角端24mm是这支镜头比较常用的焦段，在拍摄风光、建筑时，能提供84度左右的广阔视野，广角端最大光圈为F4，全开光圈也有不错的画质表现，应对旅行、日常拍摄绝无问题。这也是佳能RF镜头群中，可以以最轻、最高性价比的方式体验到全画幅+24mm广角的镜头。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


长焦端105mm可以提供中长焦的视角压缩效果，利用率同样很高。长焦端最大光圈为F7.1，在白天拍摄时有充足的光线，实际使用并不会感到困扰，可以保持较高速度快门，并且长焦端画质搭配EOS RP机身同样在水准之上。


5挡手持防抖机构


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


这支镜头搭载了IS防抖机构，CIPA标准下提供5挡的补偿效果，理论上长焦端可以使用1/4秒左右的快门速度手持拍摄。实际测试1/50秒的快门速度手持画面非常可靠，降低到1/25秒左右，仍然有非常好的补偿效果。防抖机构的加入，让这支镜头在暗光环境下，尤其是长焦端的使用，有了更大的空间。


STM对焦马达


EOS RP / RF 24-105mm F4-7.1 IS STM / F8 1/1000 ISO100


RF 24-105mm F4-7.1 IS STM镜头使用了步进式对焦马达，对焦安静迅速，合焦行程平顺，应对日常静态摄影并无压力的前提下，可以当作视频用镜头，在拍摄视频时对焦也不会发出声音，且对焦变化表现柔和。


极为优秀的微距性能



这支镜头虽然定位入门，但却具备了非常有趣的近摄能力，自动对焦状态下广角端24mm最近对焦距离约为0.2米，而如果使用手动模式下则可以缩进到0.13米，长焦端最近对焦距离为0.34米，在使用广角端时有0.5倍的放大倍率，从理论上来说可以称之为一枚微距镜头了。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/500 ISO100


105mm长焦端下，最近对焦距离为0.34米，可以极为迫近主体，配合7片圆形光圈，虽然只是一枚入门标准变焦镜头，也可以拍摄出不错的浅景深凸显主体的照片。


为什么我们需要一支轻量化变焦镜头


RF 24-105mm F4-7.1 IS STM的发布实际上有些出乎人们的意料，在F2变焦、F1.2定焦的路线图中，这支定位比RF 24-105mm F4L IS USM更加亲民的镜头实际上更适合入门级相机用户。


一览镜头图谱，这枚24-105mm镜头是最为轻量化、最为紧凑的全画幅变焦镜头之一，搭配EOS RP可以给用户提供最小负担的全画幅变焦镜头组合，手持这套组合的摄影师可以利用手里的相机和镜头，去拍摄风光、建筑、人文纪实、人像甚至微距静物。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/250 ISO400


在无反范畴内，性能、素质超越单反的镜头有许多，但同时不少顶级无反镜头在便携性上反而有所下降，RF24-105mm F4-7.1 IS STM这样使用起来更轻松的镜头，更可以让入门级用户，喜欢上用相机拍照，也会提高相机用户的使用兴趣。
