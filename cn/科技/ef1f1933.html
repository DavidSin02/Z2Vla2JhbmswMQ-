复制构造函数与析构函数

2020-11-14
| 科技

复制构造函数：

复制构造函数是一种特殊的构造函数，其形参为本类的对象引用，作用是用一个已存在的对象去初始化同类型的新对象

如果没有为类声明拷贝初始化构造函数，则编译器自己生成一个隐含的复制构造函数，这个构造函数会用作为初始值的对象的每个数据成员的值，初始化将要建立的对象的对应数据成员，在很多情况下已经很完美了，所以不用自己声明复制构造函数

析构函数：

相当于是“反·构造函数”，完成对象被删除前的一些清理工作

除了不能有返回类型，不能return以外，还不能有参数传递

在对象的生存期结束的时刻系统自动调用它，然后再释放此对象所属的空间

如果程序中未声明析构函数，编译器将自动产生一个默认的析构函数，其函数体为空

样例：（代码中有注释）


  2. #include<iostream>
  3. #include<algorithm>
  4. using namespace std;
  5. class Point
  6. {
  7. public:
  8. Point(int x, int y); //构造函数
  9. Point(const Point &b); //复制构造函数，使用const是为了避免在初始化新对象过程中把原有的对象给改了
  10. ~Point(); //析构函数
  11. void Show();
  12. private:
  13. int *x, *y;
  14. };
  15. Point::Point(int a, int b)
  16. {
  17. x = new int;
  18. y = new int;
  19. *x = a, *y = b;
  20. }
  21. Point::Point(const Point &b)
  22. {
  23. cout<<"调用复制构造函数，为指针x和y申请内存"<<endl;
  24. x = new int;
  25. y = new int;
  26. *x = *b.x, *y = *b.y;
  27. }
  28. Point::~Point(void)
  29. {
  30. cout<<"调用析构函数，释放内存"<<endl;
  31. delete x;
  32. delete y;
  33. }
  35. void Point::Show()
  36. {
  37. cout<<'('<<*x<<", "<<*y<<')'<<endl;
  38. }
  39. void Display(Point a) //形实结合时，即用实参去初始化实参时，也会发生一次复制构造
  40. {
  41. a.Show();
  42. // 注意如果函数的返回值是类的对象，函数执行完成返回主调函数时，将使用return语句中的对象初始化一个临时无名对象，传递给主调函数，此时也会发生复制构造
  43. }
  44. int main(void)
  45. {
  46. Point p(3, 5);
  47. Point q(p);
  48. Display(p), Display(q);
  49. return 0;
  50. }
  51. /*
  52. 调用复制构造函数，为指针x和y申请内存
  53. 调用复制构造函数，为指针x和y申请内存
  54. (3, 5)
  55. 调用析构函数，释放内存
  56. 调用复制构造函数，为指针x和y申请内存
  57. (3, 5)
  58. 调用析构函数，释放内存
