想让你的工作更有效率？推荐7款Windows上的任务管理工具

2020-10-29
| 科技

此前，很少在 Windows 上接触任务管理类的工具，我更倾向使用跨平台的笔记类应用，比如有道云笔记和 Google Keep。不过，自从成为少数派签约作者后，这一习惯有了变化，经常遇到需要快速记录选题灵感或者写作思路，并且根据时间制定写作计划、进度跟踪，逐渐感觉到笔记类应用已经不能高效地满足我在任务管理上的需求，所以逐渐开始接触了解任务管理工具。

最早接触任务管理概念是在 Android 移动平台，第一款深入使用的任务管理应用是 Any.do ，当初任务管理给我留下了两大主要概念：

  • 列表式事项管理；

  • 到期提醒功能。

凭着这懵懂的概念知识，让我们正式开始寻找 Windows 平台上好用的任务管理工具：

Todolist

Todolist 是一款相当「标准」的任务管理工具，支持的平台最为全面，包括了 Web、Mac、Windows、iOS、Android 以及 Chrome/Safari/Firefox，甚至还可在 Apple Watch、Android Wear 以及 Outlook/Gmail 电邮端使用这款优秀的应用。

Todolist 有专门为 Windows 10 系统准备了 UWP 安装版本，也有 exe 版本，很好地满足了不同 Windows 系统平台的需求。

经过这段时间的使用，Todolist 具备以下优势：

  • 自然语义添加任务。除了「正儿八经」地输入任务、设置到期时间、提醒时间、添加标签/优先级外，用户还可以直接通过自然语音的输入方式添加任务，Todolist 将会自动帮助你添加到期时间的任务事项；

  • 支持项目、标签和过滤器三大形式管理任务内容。项目管理方式类似文件夹，标签代表着任务的属性/定义，而过滤器被视为 Todolist 的一大特色，可按照任务分配给我、优先级别、没有到期的条件快速定位不同的任务；

  • 视觉化工作效率。Todolist Karma 数值将由 Todolist 对用户完成任务的效率进行打分，随着时间的推移追踪用户的效率并将工作效率趋势视图化；

  • 更详尽的任务描述。Todolist 不仅支持文字来描述创建的任务，还可让用户自行添加 PDF、数据表格和照片附件来完善任务细节。

Todolist 提供了多个不同的付费方案，分别有免费基本版、高级个人版（28.99 刀/ 年）、商务团队版（28.99 刀/用户/年），其中多形式提醒任务到期、为任务细节添加附件、标签和过滤器都属于高级版提供的附件功能。Todolist 应用本身具备完善的功能，推荐给追求实用、全面，同时又有多平台同步数据需求的用户，通过 DrManhatt 的还没找到合适的任务管理工具？免费领 Todoist 一年高级会员试试 可免费领取 Todoist 一年高级会员资格。

滴答清单

滴答清单所支持的平台与 Todolist 类似，可谓是跨平台的「巨无霸」，涵盖了 Web、macOS、Windows、iOS、Chrome、Firefox 以及 Apple Watch、Android Wear 智能设备平台。滴答清单更专注于多方式创建任务事项和任务提醒的环节上，其优势主要体现在：

多方式创建任务。滴答清单支持以自然语言，语音输入，Siri 助手创建任务，微信端便捷创建任务的方式。应用还专门提供了专属邮箱，类似todo+XXX@mail.dida365.com的电邮地址，或者使用你的注册邮箱发送任务到 todo@mail.dida365.com，即可创建任务到滴答清单的收集箱。

智能提醒。滴答清单具有多闹钟提醒、自定义重复功能和稍后提醒功能，比如多闹钟提醒，用户可以分别设置提前一天、提前 5 小时、提前 1 小时等不同时间点的闹钟；自定义重复提醒，有助于较少用户频繁在固定时间设置同样的任务，譬如每周六下午有固定的足球友谊赛，即可设置为每周六下午 2 点提醒有足球赛事，而不需要每周六前专门设置重复的任务内容。

双向备份。滴答清单支持将所有的清单、任务、子任务、文本、提醒生成备份，将以 csv 文件保存相关数据，也支持导入其他应用程序备份，包括 Wunderlist、Omnifocus、Toodledo、iCal。

双向订阅。滴答清单目前支持订阅其他日历的事件到滴答清单，反过来，也支持将滴答清单以 URL 链接方式添加到 Google Calendar, Outlook 等日历应用。

Windows 平台的特色功能

滴答清单有针对 Windows 平台开发了特色功能，有打开/隐藏主程序、添加任务的快捷键，同时还提供了任务列表和日历视图的桌面小部件，帮助用户快速创建和查看任务。

除了以上特点外，滴答清单还提供了统计、摘要的选项帮助对创建的任务进行一次回顾和覆盘的特色功能。不过诸如订阅日历、日历视图、自定义智能清单、高级搜索、清单和任务动态等功能均属于高级付费版（139 元/年），普通用户版本在清单数量、任务数量、提醒数量、子任务数量、共享成员数量、附件数量均有所「缩水」。

Trello

加入少数派后，首次接触了团队协作工具 Trello，卡片组成的看板形式成为极具特色的设计风格。Trello 本身具备到期提醒的功能，我派作者 Jiahao 在[用 Trello 搭建任务系统]中，分享了如何使用以看板为基础的 Trello 搭建任务系统。Trello 在跨平台上虽然没有达到 Todolist 和滴答清单那样「变态」，但依然支持 Web、iOS、Android、macOS、Windows 以及 Chrome、Firefox、IE11/Edge、Safari 浏览器平台，目前 Trello 仅支持 Windows 10 系统。

对于很多习惯了主动通知提醒，和列表管理任务的用户，Trello 在消息通知和看板管理形式上存在不足的地方。而我运用 Trello 进行任务管理上会专门建立一个 Todo 的任务板（关于写作计划类），任务板下会有六个列表，分别是：

  • 历史选题箱：存放除了今天收集的选题，再使用标签分类优先级别；

  • 今天收集选题：存放今天当天收集的写作选题；

  • 正在开展准备：存放已经开展素材收集的选题，并且在选题描述中添加大纲内容；

  • 已动笔：存放已经开始写作的选题，将文章内容的一二级标题以清单形式展示，并标准哪部分已经完成了；

  • 推迟完成：存放由于其他因素影响了写作进度需要推迟完成的选题；

  • 归档箱：顾名思义就是存放已经完成的选题。

在使用 Trello 大半年的时间里，深刻感受到其看板管理写作上的优势，使用 Trello 搭建一套很完整的任务管理系统，将有别于其他任务管理工具以列表型进行展示管理，Trello 以看板作为核心特点，结合本身在团推协作、标签、添加到期时间、与第三方软件联动等功能，推荐给围绕 Trello 打造一套自己工作流的用户。

Microsoft To-Do

今年年度最佳榜单中获得「最三心两意的应用开发商」，非微软莫属（传送门 → 获奖原因），Microsoft To-Do 就是其中的代表作，支持 Web、Android、iOS、Windows UWP 平台，几大平台的体验保持了很好的一致性，UWP 应用界面可以视为移动端的放大版本。

Microsoft To-Do 的优势是：

  • 我的一天。打开应用主界面，率先聚焦的是「我的一天」界面，我的一天是 To-Do 应用默认内置的任务清单，显示了今天待办的任务，帮助用户快速定位今天当天需要办理的任务事项；

  • 重复提醒。用户可以对任务设置重复提醒的时间，提供了每 1 天、工作日、每 1 周、每1 年或者自定义重复周期，避免多次手动添加需要重复提醒的任务；

  • 简单清爽的界面。我在 Windows 和 Android 端都有使用 Microsoft To-Do 应用，简单流畅的操作方式留下了不错的印象，To-Do 还支持不同清单设置不同的皮肤，增强明显的视觉效果。

Microsoft To-Do 应用提供了轻量化的任务管理，数据同步上并没有受到国内特殊环境所影响，还支持 Outlook 任务中同步事项、清单，有简单的到期提醒、搜索待办事项和笔记内搜索的功能，推荐给深度使用微软系列产品的用户。

Moodo

Moo.do 是一个类似于 Workflowy 的无限层级的纯文本任务管理工具，正如特伦在「Moo.do 中文入门手册：增强版 Workflowy，一个无限层级的纯文本任务管理工具」提到的，Moo.do 的 slogan 仿似 Workflowy：「Organize your way」，同样拥有无限层级的树形结构，但它也拥有任务管理、多面板查看、日历等多种增强功能，就像是一个增强版的 Workflowy。Moo.do 除了类似 Workflowy 的无限层级以外，Moo.do 的多面板、通知、日历、优先级等功能让它也可用于任务管理，其创新性的地方体现在：

  • 无线层级的列表功能。区别于 Workflowy 只能拥有一个文档，Moo.do 支持创建多个 Document 文件夹，用户新建一个新文档时，将呈现一张白纸，让用户无限层级地创建个人任务事项；

  • 多功能用途的 Pane 面板。Pane 是一个平铺展示的窗格，可让用户将多个列表以「平行视角」的形式展示出来，用户无需多次跳转即可查看多个子目录内容。除此之外，Moo.do 还提供了被称为 Calendar Pane、Archived Pane 的多功能用途面板，比如 Calendar Pane 是一个专用于日程安排的 Pane，可将用户在任务事项中的内容按时间显示出来，也可以手动拖动条目到 Calendar Pane，实现快速为任务条目插件日期的目的；

  • 高度整合 Google 服务。Moo.do 支持 Google Calendar、Gmail、Google Drive、Google Contacts 之间无缝同步数据，并且 Moo.do 仅支持使用 Google 账号进行登录，应用的数据会保存在用户个人的 Google Drive 账户中，拥有一定的安全性。

Moo.do 极具创新性地使用类似 Workflowy 的功能，围绕无限层级列表，多功能 Pane、@ 和 #结合使用，帮助用户构建一个更加详尽的任务安排系统。推荐给不需要华丽界面，更愿意自己「把握」任务安排的用户。

Remember The Milk

Remember The Milk 作为有着十年历史的任务管理工具，支持 macOS、Windows、Linux、iOS、Android、BB10、Apple Watch、Outlook，以及包括 Chrome、Firefox、Safari、Edge 浏览器平台，其优点包括了强大的快捷键、高效的日期识别、支持子任务，用户可以通过手机，邮件，IM 等很多方式添加任务，后期则支持电子邮件、IM 提醒、谷歌日历、手机软件、手机短信等方式提醒用户。

Remember The Milk 的优势：

  • 使用文件夹，标签，上下文，子任务等来组织任务事项；

  • 可设置任务的优先级、完成时间、剩余时间、是否重复、列表、标签以及更多内容；

  • 高级搜索。支持使用标识符进行搜索，搜索你的任务和笔记，并保存你喜欢的搜索内容作为智能列表；

  • 使用文件夹，标签，上下文，子任务等来组织任务事项；

  • 自定义排序和分组。可按照任务添加日期、优先级、名称和列表名称的方式进行排序和分组；

  • 强大的任务提醒。支持通过电子邮件、IM 软件、短信、Twitter、手机或者系统桌面通知的方式提醒任务；

  • 电邮通知每日任务摘要。用户可以将今天到期或者已经开始的任务、明天/未来 7 天开展的任务等定制的任务设置成每天发送邮件的方式。

Cortana

Cortana 作为最具 Windows 10 系统特色的功能之一，Cortana 支持手动或者语音方式添加任务事项，只需键入「提醒我...」或者说「你好小娜，提醒我...」就可以开始使用。Cortana 创新性提供基于时间、位置、照片的方式提醒，其中照片提醒功能，用户将照片添加到提醒中，例如孩子最喜爱的游戏，这样 Cortana 就会在下次购物时作出提醒。

由于 Google Assistant、Apple Siri、Amazon Alxea 并未登录 Windows 系统，Cortana 支持 Windows、Android、iOS 平台，有着跨平台的优势，支持简单的任务内容、时间提醒功能，语音录入任务备忘事项有不错的识别准确度。

总结 | 工具和任务管理理念同在

由于 Any.do（我首款使用的 todo 工具）、Checkvist（强大的 Open API 接口）、Google Keep（更像「便条纸」） 、Toodledo （老牌工具） 暂时没有官方的 Windows 客户端，Wunderlist 在被微软收购之后基本处于停摆状态，所以它们都不在这次挑选推荐的应用列表范围内，但这些工具都提供了很好的 Web 端和移动版本，欢迎大家在体验之后留言评论它们的亮点。

苹果 macOS/iOS 平台上有好几款被视为「神器」的任务管理工具，比如 OmniFocus、Things、TaskPaper、2Do，而 Windows 平台上也不缺乏一些相当优秀的任务管理工具，它们能够提供给你多样的选择，有完整任务管理系统的 Todolist、主打语音创建事项的 Cortana、无限层级形式的 Moodo。

如果你希望对任务管理工具、任务管理理念和方式有更深入的理解学习，少数派针对不同的任务管理软件推出了一系列栏目，深入浅出地带你走进任务管理的世界，让你更科学、更系统地做好任务管理，欢迎加入我 π 付费教程。
