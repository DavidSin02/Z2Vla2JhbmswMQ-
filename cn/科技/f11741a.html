自控猫｜仪表设计统一规定18-控制阀选型

2020-10-29
| 科技

王玉虎

仪表专业设计统一规定 仪表选型-调节阀

1) 一般选用 Globe 阀。根据工艺介质及操作条件特殊要求可选用偏心旋转阀、单座调节阀、角型阀、蝶型阀、球阀、三通阀等形式。

2) 8”和 8”以下的调节阀优先选用单/双座或套筒式 Globe 调节阀。10”和 10”以上口径或低差压情况，采用蝶型阀或偏心旋转阀。对于介质中含有固体粉末或粘度较大的场合，采用Ｖ型球阀或偏心旋转阀。对于噪声较大的情况，采用笼式阀或低噪音调节阀。根据工艺介质及操作要求也可选用角型阀、三通阀、波纹管密封阀等。对于高压差、闪蒸、空化、腐蚀、高噪声等情况，可选用特殊的阀芯和阀体设计。

3) 调节阀选用等百分比特性、线性特性或近似等百分比特性。等百分比特性调节阀最小流量开度不应小于 10%；正常流量开度 60%~80%；最大流量开度不应超过 90%；线性特性调节阀最小流量开度不应小于 15%；正常流量开度 50%~70%；最大流量开度不应超过 80%.

4) 调节阀设计应使调节阀下游 1 米处和管道表面 1 米处的噪声等级不超过 85dBA。 间歇使用或紧急操作的调节阀在上述位置的噪声不超过 105dBA。调节阀的降噪措施优先选用低噪音阀内件、改变阀门结构形式等在调节阀内部节流降噪的方法。

5) 根据工艺要求(在 PID 图上表示)，可选用带手轮调节阀，或设旁路阀及前后切断阀。原则上，SIS 用的调节(开关)阀不设手轮，工艺专利商有特殊要求的除外。

6) 安全仪表系统用、要求严密关闭的阀门选用单座阀、球阀、蝶阀等。选用气缸执行器时带弹簧返回装置，气源故障时阀应处于安全位置。安全仪表系统及顺序控制用阀门应装有限位开关，限位开关为接近式，接点类型为干接点，且至少为单刀双掷（SPDT）。

7) 阀体材料应按配管材料等级规定要求选取，阀体材料应不低于碳钢。

8) 阀内组件和阀座材料一般为 316SS。高差压、高温等场所应选用与配管等级一致的材料。在闪蒸、空化或严重冲刷以及高温、高压差或强腐蚀性场合，应在阀内件表面堆焊或喷涂硬质合金等材料。

9) 当使用温度超过 204℃或低于-18℃时，阀应带散热片或延长颈型。

10) 调节阀泄漏等级通常为 CLASS IV （ANSI/FCI 70-2），当工艺要求为严密关闭 （TSO）时，应选用 CLASS VI 或 CLASS V。开关阀泄漏等级应至少为 CLASS V。

11) 执行器通常选用气动薄膜式，在要求大推力的场合可选用单动作或双动作气缸执行器。

对于气源故障时要求开或关的场合，推荐优先选用带弹簧返回的薄膜式或气缸执行器。选择执行机构时，应基于仪表空气的最小压力进行输出力（或力矩）计算。

12) 在气源故障时，应保证阀门处于“故障安全”位置。一般情况下，阀门的联锁位置和气源故障位置一致。必要时使用储气罐以确保联锁时位于正常位置。电磁阀失电(非励磁)联锁。

仪表空气储罐的制造应符合《固定式压力容器安全技术监察规程》的有关规定，并应根据调节阀的安全完整性等级确定是否配置压力开关。

13) 一般情况宜选用单层填料函结构；对于低温、高温或高毒性介质，应选用双层填料函结构。介质温度在-18℃～204℃范围内时，宜选用 V 型 PTFE 填料；介质温度低于-18℃或高于 204℃场合应选用柔性石墨填料或其他合适材料的填料。

14) 一般选用 4~20mA DC 带 HART 协议输出的智能型电/气阀门定位器；也可选用电/气转换器，与气动阀门定位器配合使用。一般选用正作用式电/气阀门定位器。

15) 电磁阀选用 24VDC 低功率（低于 4W） ，316SS 材质。完全密封，高负载电磁线圈，绝缘等级为 H（高温） 。三通或四通电磁阀应选用长期带电型。

16) 自力式调节阀只适用于 N2、空气、燃料气、蒸汽和其它辅助用流体等调节要求不严的场合。

17) 压缩机防喘振调节阀：

a) 除非数据表中有明确说明， 输出信号到 4mA 或电磁阀失电时， 阀门从全关到全开的时间不应超过 2 秒；输出信号到 20mA 或电磁阀励磁时，阀门从全开到全关的时间不应超过 5 秒。

b) 防喘振调节阀宜选用线性流量特性的调节阀。

18) 用于输送含颗粒或易暴聚介质时，开关阀宜选用旋塞阀、柱塞阀或 V 形球阀。

19) 一般情况下，阀门应采用法兰连接。用于烃类或其他可燃介质的角行程阀门宜选用法兰连接型；当 DN>300 时，蝶阀及偏心旋转阀也可选用支耳（Lug）连接型，不得选用无法兰对夹（Wafer）连接型。连接法兰等级应与配管材料等级规定一致。

仪表专业设计统一规定 仪表选型-开关阀

1. 开关阀

1) 除非有特殊要求，开关阀的尺寸与管线尺寸相同，并且为全通径。开关阀的要求基本可参照调节阀。

2) 开关阀的阀门形式选择原则：

a) 低压气体及液体选用球阀或闸阀；

b) 含有固体颗粒介质的场合选用球阀；

c) 在大口径、低压差及含有悬浮颗粒物介质的场合选用蝶阀；

d) 高压、高压差的场合选用球阀、平行闸阀；

3) 开关阀的泄漏等级为 ANSI/FCI 70-2 CLASS V 或 CLASS VI；

4) 开关阀的动作时间应满足工艺要求，当工艺无特殊要求时，电磁阀失电后对应开关阀的全开或全关时间不超过 3 秒/每英寸口径，或者小于 1 秒/每英寸口径，阀体尺寸（英寸）（以秒计）。任何阀的最大行程时间为 30 秒。紧急切断阀最大行程不超过 10 秒。

5) 开关阀执行机构的输出力矩应至少留有 30%的安全系数。

6) 开关阀配置全开及全关阀位开关，SIS 用的阀门应配阀位开关。

7) 开关阀的阀体应符合 API 607 或 API 6FA 耐火试验标准。
