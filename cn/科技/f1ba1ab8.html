动图｜一文看懂材料的强度、硬度、弹性、韧性、延展性

2020-11-14
| 科技



0 1

强度


材质要能经得起应用场景中受力的考验，不弯、不断、不碎、不变形。



0 2

硬度


较硬的材质一般更抗刮擦，耐用，抗撕裂和压痕。



0 3

弹性


材质吸收受力、能在不同方向弯曲并且能够恢复到原来的状态能力。



0 4

成型性能


是否方便加工成永久形状，柿子虽然软可以随便捏但是捏完就完了。另外一个极端的例子钻石硬度太高，也不适合用来加工。钻石可以用来加工别的。



0 5

延性


长度方向上的受力变形能力。橡皮筋的弹性很好。材料方面热塑性弹性体一般都具有不错的延性。



0 6

抗拉强度


未发生断裂或者折断之前的变形能力。



0 7

延展性


未出现裂纹前，材质在各个方向上能够改变形状的能力 ，考验的是材料再次塑性的能力。



0 8

韧性


材质抗冲击能力，突然敲打一下，不会断裂或者破碎 。



0 9

导电性能


这个不需小编多说。正常情况下，导电性好的材料导热性能也不错。
