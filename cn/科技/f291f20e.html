你的办公效率低是有原因的，这些常用的电脑快捷键都掌握了吗？

2020-11-14
| 科技

*本文来自花粉俱乐部论坛-玩机技巧官方

HUAWEI MateBook X Pro 已上市有一段时间，受到很多商务人士和文字工作者的喜欢。键盘是大家使用最多的部件，但是大家真的了解华为MateBook X Pro 的键盘吗？一些可提高办公效率的热键和快捷键组合，大家都知道吗？

不知道的话，就赶紧洗耳恭听吧！

一、超大触控，高效键盘，跟鼠标说再见

华为MateBook X Pro 大幅度加大多手势触控板操作面积，带来更加广阔的触控板操作区域，减少束缚感，提高精度，优化防误触体验。

同时， 华为MateBook X Pro进一步优化键盘键程和键帽尺寸，带来更愉悦的键盘敲击体验。全新的全尺寸键盘可以实现防泼溅，如果不小心把水洒在键盘上了，关机后将键盘上的水倒出擦干静置些许时间就可以正常使用。

二、华为设计师脑洞大开，摄像头变按键

华为MateBook X Pro键盘与此前上市的 MateBook 系列相比，最大的亮点就是隐藏式摄像头。笔记本的摄像头一般都会内置于屏幕上，但是华为MateBook X Pro 的设计师却另辟蹊径，将摄像头移到了键盘上。这样的设计，不仅让人感觉安全无忧，同时也提高了屏幕的屏占比。

使用摄像头时，按一下摄像头就会弹起，再按一下摄像头就隐藏起来。

三、热键优先，多种设置一键搞定

HUAWEI MateBook X Pro 键盘上的F1-F12键默认是热键优先，只需按一个键，就可以执行对应的功能，有效提高办公效率。比如在热键优先状态下，按F1键是降低屏幕亮度，按 F9 键可一键开启/关闭WI-FI，按F10键可一键开启电脑管家。

四、两种方式，热键与功能键可轻松切换

在一些场景下：比如游戏中，要把 F1、F2等键作为快捷功能键使用；在浏览网页中，要用F5键刷新网页。要怎么切换热键与功能键呢？很简单，有两种方式可以选择：

方式1：按下 Fn 键，当 Fn 键指示灯亮起，表示已将 F1、F2 等键锁定为功能键模式。这时候，就可以将F1、F2作为功能键使用。想要切换回热键优先，只需再次按 Fn 键， Fn 键指示灯熄灭表示已经为默认的热键优先。

方式2：通过华为电脑管家设置“功能键优先”。打开华为电脑管家，点击右上角的设置图标，选择“设置中心 > 功能键”，勾选“功能键优先”后，F1、F2 等键将作为功能键使用；如要切换回默认状态，在这个界面勾选“热键优先”即可。

五、快捷键组合，实用便捷姿势又帅

NO.1 Win + D

上班时，稿子憋不出来，放松一下心情说不定不错。找个段子看看，找首歌听听……结果眼睛一撇，老大向你走过来了！肿么办？

别紧张，按下Win + D 快捷键组合，秒回你一本正经的桌面。（真实目的是教你怎么在打开一堆应用时，快速回到桌面滴~）

老板走了之后，想切回刚才的网页？再按一次 Win + D 就可以了！

NO.2 Win + L

上班的时候需要临时离开座位，又不想别人看到你的电脑屏幕，怎么办？按下Win + L这个组合键，电脑即刻锁屏。但是记住哦，电脑得先设好密码，要不然锁了也白锁。

NO.3 Win + E

打开了一堆应用，又需要打开储存在“我的电脑”中的文件怎么办？按下 Win + E ，就能秒出“我的电脑”界面，有没有觉得自己姿势超帅~

NO.4 Ctrl + N

在网上搜到很多有用的资料，为了方便日后查看，分文档储存下来比较好。怎么新建文档？教你个简单办法，在打开 Word 文档时，按Ctrl + N，秒秒钟快速新建多个文档！

NO.5 Ctrl + S

呕心沥血写的文案、做的PPT，按Ctrl + S就能一键保存，无需再从键盘切换到鼠标去点“保存”按钮。

NO.6 Ctrl + Shift + N

学会了新建文档，新建一个文件夹还会难吗？答案是不难，按下组合键“Ctrl + Shift + N”立即出现一个新建的文件夹，并且文件夹名称框处于激活状态，输入你想设置的名字就OK了。

NO.7 Ctrl + Shift + Esc

整个电脑整个卡死怎么办？同时按下 Ctrl + Shift + Esc 就可以调出任务管理器，然后再把不需要的程序关掉就好！

NO.8 Shift + 任意字母

这个快捷方式可以快速切换大小写，同时按下 Shift + 任意字母，输出的是与当前大小写输入状态相反的字母。例如小写状态下，按下Shift + m ，就会输出大写的“M”，这个功能在输入有大写字母的英文单词或句子时，是不是特别方便呢？

NO.9 Home、End、Page Up、Page Down

用 Word写东西或者是敲代码时，熟练使用 Home、End、Page Up、Page Down 键，事半功倍。但 MateBook X Pro 没有这几个键怎么办？同时按下 Fn + 上/下/左/右方向键，也可以有这四个按键的作用啦！

Fn + 左方向键：跳转至行首/顶端；

Fn + 右方向键：跳转至行末/底部；

Fn + 上方向键：向上翻页；

Fn + 下方向键：向下翻页。

NO.10 Ctrl + P

日常工作，大家通常有打印文档的需求。有没有打印的快捷操作呢？有，按下Ctrl + P，然后回车就可以了。如果要打印多份呢？Ctrl + P，输入数字（要打印的份数），然后按回车键就好了。

NO.11 Ctrl + W/Alt+F4

工作很卖力，不知不觉开了N多网页，怎么关闭？

方法1：在打开的网页中，按Ctrl + W，可以关闭当前网页；按住Ctrl键不松，连接点击W键，页面就能连续关闭。

方法2：在打开的网页中，按Alt + F4组合键，一下全关闭。不过使用此组合键要注意：有的浏览器不给提示的情况下所有网页就都关闭了；有的浏览器会弹出一个提示对话框，询问你是否关闭所有选项卡。

掌握热键/功能键的切换、快捷键组合的操作，办公效率提升5倍！5倍！5倍！
