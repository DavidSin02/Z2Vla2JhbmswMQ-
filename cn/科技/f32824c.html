「操作技巧」工具提示在不同条件显示不同颜色

2020-10-29
| 科技

前几天有一位用户问我：“如何让工具提示在不同条件显示不同颜色？”。

今天我们就拿 Tableau内置的数据源示例-超市举例：

当子类别的利润≥0，工具提示中的利润为绿色；

当子类别的利润<0，工具提示中的利润为红色。

效果图如下：

操作核心

创建两个计算字段，让它们在不同条件下分别显示。来跟着我一步步操作吧~

操作步骤

第一步：制作柱状图

将字段[子类别]放到列，将字段[利润]求和放到行。

第二步：创建两个计算字段

第1个名称：利润工具提示正

代码如下

IF SUM([利润]) >= 0

THEN SUM([利润])

ELSE NULL

END

第2个名称：利润工具提示负

代码如下

IF SUM([利润]) < 0

THEN SUM([利润])

ELSE NULL

END

第三步：修改工具提示

1.将上述创建的两个计算字段，分别拖入标记卡中的“工具提示”

2.再点击工具提示修改文字，修改样式如下所示：

3. 把鼠标放到标记上，弹出的工具提示已经能自己变色了哦~
