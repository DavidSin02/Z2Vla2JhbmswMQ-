新一代超广神镜 适马14-24mm F2.8全面评测

2020-11-14
| 科技

十多年前，尼康发布了一代神镜——AF-S 14-24mm f/2.8G ED，在尼康新一代14-24mm久久不来之际，我们终于再度见证了另一支“大灯泡”的降临，适马于今年2月份，正式发布了其新款Art系列超广角变焦镜头14-24mm f/2.8 HSM Art，毕竟尼康14-24mm F2.8算是声名远扬，用过的人都说好，这也让我们对适马的新一代超广角变焦14-24mm f/2.8 HSM Art充满了兴趣。

自第一款适马ART系列镜头上市之后就广受好评，适马该系列的大部分镜头都拥有非常具备竞争力的价格，并且在性能方面也足以与同焦段原厂镜头一战。作为适马ART系列的新一代广角变焦镜头，适马14-24mm f/2.8 HSM Art镜头不但采用了f/2.8的恒定大光圈设计，还使用了更为复杂的11组17片光学结构，其中包含多达3片FLD镜片、3片SLD镜片以及3片非球面镜片，其中一片非球面镜片还是直径达80mm的玻璃模铸非球面镜片，可谓用料十足。除此之外，新款镜头还内置有9片圆形光圈叶片，并采用了HSM自动对焦马达，最近对焦距离仅为26cm。

适马将该镜头售价控制在万元内——￥8499，相较于佳能的旗舰级超广角镜头EF 11-24mm F4L USM目前1.6万元的市场价格来说，适马14-24mm f/2.8 HSM Art的售价基本便宜了一半。虽然佳能的11-24mm具备更广的焦段，但适马14-24mm f/2.8拥有更大的光圈，如果两者具备相近的光学素质，显然适马更具吸引力一些。而对于尼康的AF-S 14-24mm F2.8G ED来说（目前京东方面售价￥9999），适马不仅在价格上拥有一定的优势，经过了十年的光学发展，适马新品显然具备不小的竞争力。下面，我们先从外观来看一下这款“神灯”。

镜头包装盒+镜头+镜头盖

适马14-24mm f/2.8 HSM Art的包装箱采用适马一如既往的黑白设计，简洁大气。包装盒白底黑条的部分写着“14-24mm F2.8 DG”表示着包装箱里主人的身份，包装箱上的银灰色的圆形“A”标展现了其纯正的ART系列镜头血统。

正面+镜头盖

适马14-24mm f/2.8 HSM Art装上镜头盖显得无比的硕大，镜身部分是一层细腻的磨砂涂层，增加手感的同时可以有效地防止指纹。为了增添镜头的工业艺术感，这支镜头还在镜筒上以亮银色的圆形“A”标标识ART系列镜头血统，镜身后部更是采用了与众不同的光滑漆面设计。

适马14-24mm f/2.8 HSM Art采用了花瓣形且带有螺纹消光设计的遮光罩，可以很好的抑制画面光晕、避免杂光进入镜头，同时还能阻挡雨雪溅落、保护相机和镜头免遭意外碰撞。其次巨大的镜头盖也能很好的保护镜头，这就为适马14-24mm f/2.8 HSM Art提供了双重保护。

适马14-24mm f/2.8 HSM Art搭配5000万像素的佳能EOS 5Ds使用

适马14-24mm f/2.8 HSM Art为满足5000万像素以上的机身画质需求所设计，所以笔者在后续的拍摄中搭配了5000万像素的佳能EOS 5Ds使用。

·镜头的参数解析

适马14-24mm f/2.8 HSM Art采用11组17片光学结构，9片圆形光圈设计，其中包含3片FLD镜片、3片SLD镜片和3片非球面镜片，其中更采用了一片大口径的非球面镜片。通过不同位置有效配置FLD和SLD这些高折射率镜片的位置，进一步提升了FLD、SLD镜片的矫正效果。这种光学系统使横向色差和其他光学像差最小化，从而提高了镜头的图像质量。整组镜组最前面的镜片采用了一片大口径非球面镜片。通过将这片超大非球面镜片的威力与其他的镜组相结合，有效地将畸变降至1%以内。

·14mm和24mm端各挡光圈色散情况表现

笔者在做适马14-24mm f/2.8 HSM Art这支镜头的色散测试时，找了好几组高反差的位置进行拍摄，拍摄完在相机内进行放大观察，均未发现有明显的色散表现，当时真的感叹适马14-24mm f/2.8 HSM Art这支镜头包含的三片FLD镜片和三片SLD镜片发挥的作用十分明显，将高反差边缘的紫边几乎全都消除了。

以上这组框选出来的选区，是出现了紫边现象的位置，不过也只在14mm的焦段下开到大光圈时有一道紫边的存在，当光圈收到F5.6时，这道紫边就不见了。而在24mm焦段时，笔者发现确实没有找到紫边的存在，由此可见适马14-24mm f/2.8 HSM Art镜头对于色散出色的控制能力。

·镜头炫光控制表现

·评测总结：

一直以来在14-24mm这个焦段，尼康的14-24mm f/2.8都算是名声在外，用过的人都说好，不过该镜头到现在推出也将近有11年之久，很多对这个焦段有需求的用户，一直在期待这个焦段能有全新的镜头出现。因此，此次适马推出同类型的镜头——14-24mm f/2.8 HSM Art，可以说对大家都非常具有吸引力。适马作为一个以“黑科技”著称的光学厂商，一直善于高锐度的镜头设计。不过作为一支广角端的风光镜头，我们还是很关心除锐度以外的色彩、畸变以及抗眩光能力。

这些方面，在测试环节还有样片展示中，我们已经看到了适马14-24mm f/2.8 HSM Art这支超广角变焦镜头在各方面都是可圈可点的。成像色彩很接近于肉眼在现场看到的画面；畸变控制方面做到了14mm端近乎“零畸变”的水准；抗眩光方面，除了在直射阳光且特定角度的情况下不可避免地出现以外，其余场景再难出现；色散的控制方面也是相当强，拍摄高反差场景时完全不用担心色散了。并且这次适马14-24mm f/2.8 HSM Art的主打高画质的，笔者在拍摄的过程中感受颇深。

价格方面，适马将该镜头售价控制在万元内——￥8499，相较于同类产品佳能的EF 11-24mm F4L USM近两万块的市场价格来说，虽然佳能的11-24mm具备更广的焦段，但适马14-24mm f/2.8拥有更大的光圈。通过前面的测试，适马甚至还在畸变、暗角、色散控制等多个方面超越了对方，并且还便宜了将近一万块的价格，显然适马14-24mm会更具吸引力。而对于尼康的AF-S 14-24mm F2.8G ED来说（目前京东方面售价￥9999），适马不仅在价格上有一定优势，适马新品显然具备相当的竞争力。

总之，就目前适马14-24mm f/2.8 HSM Art的表现来看，对广角镜头有需求的用户来说，它的确是一款非常值得推荐的超广角变焦镜头。
