「中考化学」常见气体的检验方法 含反应方程式

2020-11-14
| 科技

本文总结了：H2、CO、Cl2、O2、CO2、HCl、H2S、SO2、N2、NO、NO2等多种常见气体的检验方法，方便中考的学生进行系统复习。

1.H2

将待检气体通过灼热的CuO，CuO由黑色变为红色，并将产物通入无水硫酸铜中，无水硫酸铜变蓝，则证明通入CuO的气体是H2。

H2+CuO =（加热）Cu + H2O

CuSO4+5H2O =CuSO4·5H2O

2.CO

让待检气体在空气中燃烧，在火焰上方罩一干燥的小烧杯，烧杯上无水珠生成，然后将产物与澄清的石灰水接触，澄清的石灰水变浑浊，则证明燃烧气体为CO。

2CO + O2 = 2CO2

CO2 + Ca(OH)2 = CaCO3 + H2O

3.CO2

将气体分别通入澄清的石灰水和品红溶液中，如果澄清的石灰水变浑浊，且品红不褪色，则通入的气体为CO2。

CO2 + Ca(OH)2 = CaCO3 + H2O

4.CH4

让待检气体在空气中燃烧（火焰为淡蓝色），在火焰上方罩一干燥的小烧杯，烧杯上有液滴生成，然后将产物与澄清的石灰水接触，澄清的石灰水变浑浊，则证明燃烧气体为CH4。

CH4 + 2O2→（点燃）CO2 + 2H2O

CO2 + Ca(OH)2 = CaCO3 + H2O

5.N2

将点燃的镁条放在待检气体中燃烧，燃烧之后，向燃烧产物中加入适量的水，如果有白色的不溶固体和能使润湿的红色石蕊试纸变蓝的刺激性气味的气体产生，则证明待检的气体为N2。

3Mg + N2 =（点燃）Mg3N2

Mg3N2 + 6H2O =3Mg(OH)2↓+ 2NH3↑

6.NO

直接将待检气体的瓶盖打开，如果在瓶口有红棕色气体产生，则证明待检气体是NO。

2NO + O2 = 2NO2 (红棕色)

7.NO2

将待检气体溶于水中，若待检气体由红棕色变为无色，且水溶液也成无色，则证明待检气体为NO2。

3NO2(红棕色) + H2O =2HNO3 + NO(无色)

8.O2

将带火星的木条伸入待检气体中，如果带火星的木条复燃，则证明待检气体为O2。

C + O2 = CO2

9.H2S

（1）闻待检气体的气味，如果有臭鸡蛋气味，则证明待检气体为H2S。

（2）用湿润的Pb(CH3COO)2试纸靠近盛待检气体的瓶口，如果Pb(CH3COO)2试纸变成黑色，则证明待检气体是H2S。

Pb(CH3COO)2 + H2S = PbS↓+ 2CH3COOH

10.SO2

将待检气体通入品红溶液中，如果品红溶液褪色，且受热后又恢复原色，则证明待检气体是SO2。

11.Cl2

（1）观察待检气体的颜色，如果呈黄绿色，则证明待检气体是Cl2。

（2）将氢气点燃并伸入待检气体中，如果氢气继续燃烧，并产生苍白色火焰，同时，在瓶口附近有白雾产生，则证明待检气体为Cl2。

H2 + Cl2 =（点燃）2HCl

（3）将湿润的KI淀粉试纸靠近成待检气体的瓶口，如果KI淀粉试纸变蓝，则证明待检气体为Cl2。（注：当有几种氧化性气体同时存在时，则此法不适用）

2KI + Cl2 = KCl +I2
