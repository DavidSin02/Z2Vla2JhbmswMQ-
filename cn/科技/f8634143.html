简介

form-generator 开源Element UI表单设计及代码生成器

github 地址：

https://github.com/JakHuang/form-generator

github star: 134+

最新版本：0.1.0，MIT 协议

主要技术栈

vue2

vue-cli4

element-ui

vuedraggable

功能列表

  • 可视化设计表单
  • 一键生成原生的vue单页表单代码，100%拓展性
  • 在线编辑器修改生成的代码，并实时预览
  • 表单栅格化布局
  • 支持表单校验
  • 快速查阅组件的官方文档
