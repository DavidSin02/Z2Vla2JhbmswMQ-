掌握「双原生 ISO」，提升动态范围

2020-10-29
| 科技

其实对于设备规格的争议，始终没有一定的答案或对错，不少人执著于更高的画素，但对于务实的影像工作者来说，更在意的会是色彩科学或是编码器、纯净度、帧频…等等。而不久的将来，可能将多出一个让人考虑的主要因素，那就是「双原生 ISO」。

以往的光线控制，都优先以快门与光圈决定，ISO 能低就低，免得影响到影像的纯净度。一般人认为：ISO 是感光组件对于光线的灵敏度。其实更确切的来说，它是感光单位感受到光线撞击后的增益值(Gain)。

单一与双原生 ISO 的讯号处理方式

单原生 ISO

传统摄影机的感光组件都是单原生 ISO。

当感光元件接收到光线后，会先经过一个模拟放大器（这决定了原生的 ISO）进行放大与降噪处理，再将其送至 ADC （Analog to Digital Converter，模拟转换器）转换为数字讯号后，再经过数字讯号放大器，依相机当时设定的 ISO 进行数字增减益（Gain）。

因模拟讯号的增益值是固定的，若曝光不足，就得完全仰赖数字讯号的增益，效果当然没有前面模拟转换器的效果好，而且同时也会放大处理过程中因为电流而产生的噪讯。这也是为什么当我们拉高摄影机的ISO时，画面的纯净度会受到影响的原因。

双原生 ISO

在具备双原生 ISO 能力的条件下，当你切换至高 ISO 的模式，光的讯号会经过「第二组」较强的模拟转换器（也就是第二原生 ISO）进行放大与降噪，接着才送到 ADC 进行数字讯号生成，最后才进行数字讯号的 ISO 增减益。

因此在光线不足的状态下，有了第二个较高的原生 ISO 可选择，对于数字讯号再增益时的噪讯压力，可说是减少许多。

所以具备双原生 ISO 的摄影机，相较于传统摄影机，多了高原生 ISO 的加持，对于低光环境拍摄下的高 ISO 噪讯抑制，将会比平常靠数字增益提升 ISO 的摄影机表现更好。

双 ISO 如何影响动态范围

例如：BMPCC 4K 具备了双 ISO 的设计，当机身的 ISO 调高时，就会自动采用较强的模拟讯号增益器，也就是从原生的 ISO 400 切换至原生的 ISO 3200。而此切换，除了影响到噪讯程度，也会影响到动态范围。

在下面的图表中，可以看到 BMPCC 4K 在各 ISO 值时的动态范围。中间的分线就是所谓的中间灰(middle gray)，线以上的米白区块以及线以下的橘色区块，分别代表了亮部与暗部的动态范围。从图表可见在不同的 ISO 值中，动态范围的表现也不尽相同。

举例来说，在 ISO 100 的设定时，中间灰以上的亮部动态范围区域只有2文件(stops)、低于中间灰的暗部动态范围则有 11.1 档。当 ISO 由 1000 提升至 1250 时，BMPCC 4K 变更为采用 ISO 3200 的原生讯号进行减益，这时可以看到中间灰以上的亮部动态范围骤减、暗部的动态范围则相反呈现增加的现象。

也许你觉得奇怪，当机身设定切换到 ISO 1250 时，为什么是采用原生 ISO 3200而不是ISO 400 ? 其实这表示厂商在设计测试感光元件时，比起「原生 ISO 400，数位增益至 ISO 1250」，「原生 ISO 3200，数字减益至 ISO 1250」表现得会比前者更理想。所以就 BMPCC 4K 来说，原生 ISO 的切换界线就在 1000 与 1250 之间。

BMPCC 4K 双 ISO 实测

以往的认知或许是停留在「ISO 影响的画面亮度」，而对于双原生 ISO 的认知则停留在「在需要高感光的环境下得到更佳的纯净度」。经过以下说明，从此该了解 ISO 同时也影响到整体曝光的动态范围。

低光噪讯测试

摄影师 Hess使用了 BMPCC 4K，采用3种不同的 ISO 值进行 Log 低光源拍摄，测试在中灰色以下的噪讯程度。

在 ISO 400 时可见暗部的高纯净度

切换至 ISO 1000 时，暗部噪讯明显提升，

但只要再往上提升至 ISO 1250(采原生 ISO 3200)，暗部纯净度又再次回到了理想的状态，且动态范围达10文件。

高光动态范围测试

接下来这个例子，Hess 测试了高亮度的拍摄，以测试 BMPCC 4K在亮部的动态范围表现。通常我们为了保留最理想曝光的影像，都会尽可能的去降低 ISO 值。但以下的例子可能会颠覆你的认知：

以ISO 125拍摄

以ISO 1000拍摄

即使中间灰的设定都相同，但ISO 250 的画面，已经丢失不少亮部的细节，回过头去看上面的图表可得知：在 ISO 250 的状态下，中间灰以上的动态范围只有 2.3 档。反倒是将 ISO 设定在 1000 时，中间灰以上的动态范围达 5.3 档，可以有效保留较充足的亮部细节。

以 BMPCC 4K来说，当拍摄高光占多数的画面、想保留最广阔的亮部动态范围时，最佳选择为 ISO 1000 ; 若想保留最广阔的阴影动态范围，最佳选择为 ISO 100，但前提是光源足够，如果光源不足，则建议使用 ISO 1250。(以上建议不包含噪讯考虑)

结论

之后遇到有高光背景居多的场景，还要一味降 ISO 吗? 恐怕使用较高的 ISO、搭配其他方式削减曝光量，动态范围的表现会更理想。

当然，每一台机器的曝光结果都有所落差，熟悉你手中的设备在各级 ISO 的表现，并仔细去体验，肯定会比死记这些评测实际许多。
