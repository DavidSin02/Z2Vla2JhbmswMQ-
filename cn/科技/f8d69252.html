缝纫基础知识（十一）技巧篇

2020-11-14
| 科技

九、闭合辅件

1.搭钩及襻

（1）怎样装腰头搭钩和襻

A 将搭钩放在腰头搭接部分的下侧，距里侧边缘约0.3cm处，穿过每个孔眼缝3-4针将搭钩定位，注意不要缝到服装正面。（下图）

B 将搭钩重叠到需要装襻的部分，标记处襻的位置，在襻的每个孔里缝3-4针。（下图）

C 圆领钩和钩眼用于不搭接的腰头。（下图）

（2）如何制作线环扣眼

A 用针引双股线穿过织物边缘，缝两道基底针迹，长度按纽扣直径需要来定。（下图）

B 使针眼从基底针迹线下面和线环中穿过。（下图）

C 针穿过线环，抽紧线环扣住基底针迹线，重复过程，只到完全盖住基底针迹。（下图）

D 缝两针短小的回针针迹以便锁住针迹线，修剪掉线头。（下图）

2.钮孔

（1）确定钮孔长度

A 测量钮孔的宽度和高度，这两个测得值之和加上0.3cm就是机制钮孔的合适长度。（下图）

B 实验钮孔，在一块碎布上剪一切口，长度为钮孔长度加上0.3cm，如果纽扣能顺利穿过该孔，长度就合适了，就可用此长度在服装、贴边和衬头上缝制钮孔。（下图）

（2）标示钮孔

A 将纸样放在服装上面，纸样的接缝线和服装门襟边缘对齐，用别针穿透纸样，把纸样慢慢从别针头上方拉出。（下图）

B 用别针、水溶性标记笔或胶带等在织物上做钮孔标记。（下图）

（3）机制钮孔

A 怎样缝制钮孔

缝制钮孔压脚装置：将织物置于锁钮孔压脚下面，起点和机针对齐，压脚中心位于中心标记的上方，机器会按下面四步进行缝制。（下图）

包缝钮孔：1）以窄之字形针迹缝钮孔，剪开钮孔并修剪松散纱线；2）重新将钮孔准确定位，调整加大之字形针迹宽度，再次锁钮孔，让宽之字形针迹包住窄之字形针迹。（下图）

一步缝制的钮孔装置（下图）

通用缝制钮孔附件装置（下图）

B 怎样剪开钮孔

在钮孔两端的锁眼针迹上插入别针（下图）

用尖头剪刀或剥线器插入钮孔中心，小心地剪至一端头，再剪至另一端头。（下图）

在切口边缘涂上防脱散剂（下图）

3.纽扣

（1）标示纽扣位置

A 标示钮孔位置时，把服装的门襟一侧搭在另一侧上，对齐中心线，用别针别住两侧衣片，使其闭合。（下图）

B 插入别针直穿透钮孔，到下层织物，在钮孔中心的位置。（下图）

C 小心地将钮孔提升到别针上方，在别针尖处插入穿有线的缝衣针钉纽扣。（下图）

（2）如何钉有脚纽扣

A 剪一节线，长76cm左右，把线拉过蜂蜡，增加线的强度，将线对折。（下图）

B 在织物正面，纽扣下以短小针迹固定线，引针穿过脚孔，向下插缝衣针入织物，并引线穿过，如此重复4-6次。（下图）

C 在纽扣下打结或缝几针短小针迹，以便在织物中栓住线头，剪去线端头。（下图）

（3）用手工钉有眼纽扣

A 像钉有脚纽扣那样穿针引线，把纽扣放在别针标示位置，使纽扣上的孔眼与钮孔平行，从下面使针穿透织物并一直向上穿过纽扣的一个孔眼，插针入另一孔眼并穿透各层织物。（下图）

B 将一根牙签或火柴插在线和纽扣之间形成脚，在每个孔眼内缝3-4针，将针和线都引到服装正面纽扣的下方打结，去除牙签。（下图）

C 让线围绕纽扣针迹线缠绕2-3圈再打结，修剪掉线头。（下图）

（4）用缝纫机钉有眼纽扣

A 装上钉纽扣压脚和专用针板盖住送布牙或卸下送布牙，用密实的之字形针迹钉纽扣。（下图）

B 把纽扣放在压脚下，转动手轮使机针下降，插入一个孔眼的中心，降下压脚。（下图）

C 插入牙签或火柴，利用之字形针迹宽度调节器将针迹宽度调到与纽扣孔眼间的间距宽度相等，慢慢地操作直到调至正确宽度，缝上6针或更多针。（下图）

4.暗扣

怎样安装普通暗扣

（1）将一颗暗扣的凸出部分放在搭接片的反面，距边缘0.3-0.6cm，用单股线穿过每个孔眼缝针迹，只穿过贴边和衬头织，切勿穿透到服装正面，缝两针短针迹固定住线。（下图）

B 在下搭接片的正面标示暗扣另一部分的位置，把别针固定住暗扣凹进的那部分。（下图）

C 使暗扣凹进部分的中心，位于标记上，所缝针迹要穿透各层织物。（下图）
