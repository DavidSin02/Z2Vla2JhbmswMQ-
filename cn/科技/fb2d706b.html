用另一种视角（走势的自同构性）来看待线段划分

2020-11-14
| 科技

线段划分，历来都是一个非常复杂的问题，自缠师在世时，他就写了4课来专门说线段划分的问题，就更别说现在了。真可谓是群魔乱舞，各说各话。在这里，我就简单分享一些我自己的关于线段划分的认知和经验，至于有没有道理，那是你自己的事情了。


关于阅读原文，我经常都说我们要能够学会和缠师的原文形成一种对话的阅读关系。当然首先是能够正确理解各个定理本身的意义，其次还要用"因果"的眼光去看待这个定理本身，这样我们就会问缠师很多问题。

如：为什么缠师要把定理写成这样？为什么还有些什么东西没有写？继而引申出的问题是既然"特征序列包含"这么重要，为什么不在线段分解定理中有所体现？


凡事总有原因，如果你认为缠师就是一个骗子，那你看到这里应该关掉这个页面，然后将我拉黑。如果不是，那么搞清楚这些问题背后的原因，这样会有助于我们对缠论的理解，更有助于我们对线段的划分和应用。


线段划分缠师给出了几种方式，具体每种方式的解读我们这里不详细论述，我们从另一个视角去观察和看待这个问题。


我认为线段划分实际上很简单，把握一个最基本的原则就是："走势必完美。"线段破坏线段实际上就是走势必完美的形态学体现。缠师说过，缠论我们真正利用的是什么？就是走势的自同构性，我们来回顾一段原文：

股票走势，归根结底是不可复制的，但股票走势的绝妙之处就在于，不可复制的走势，却毫无例外地复制着自同构性结构，而这自同构性结构的复制性是绝对的，是可以用本 ID 的理论绝对地证明而不需要套用任何诸如分形之类的先验数学理论。这种同构性结构的绝对复制性的可绝对推导性，就是本 ID 理论的关键之处，也是本 ID 理论对繁复、不可捉摸的股票走势的绝妙洞察之一。


试问，自同构性是在动力学上用的还是在形态学上用的呢？答案当然是都在用。所以走势的自同构性，本质是走势的结构，那么我们是否可以理解为动力学和形态学不过是对这个结构而言的不同观察视角？我可以用自同构性来解释线段划分吗？而这就是我们今天要讨论问题了。


为了解决这个问题，我们先用动力学的内容把结构画出来出来分析一下：

假定这绿色每一个走势都同一个级别的走势，那么一眼一看我们就知道这是一个盘整，那既然是一个盘整，那我们这里先用"因果"的眼光分析一下这个走势的前因及后果。

前因：站在本级别视角而言，对于1-2-3之前一定是一个完成的走势类型。

后果：站在本级别视角而言，对于1-2-3之后也一定开始一个新的走势类型。也就是说，至少在下一个走势类型中其中一定有一个同级别的走势4-5-6。或者是4回到2的区间范围内之后5出新高。或者是4没回到6回到2的区间范围。

接下来，我们再看缠师65课的"线段分解定理"：

缠中说禅线段分解定理：线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。而只要构成有重叠部分的前三笔，那么必然会形成一线段，换言之，线段破坏的充要条件，就是被另一个线段破坏。


我们先白话梳理一下这段定理的含义：

1、 线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。

连续重叠的三笔其中一笔对前线段的特征序列缺口造成破坏。

2、 而只要构成有重叠部分的前三笔，那么必然会形成一线段。

这句话有个意思就是说，一个线段最基本的构成至少有三笔，那么这三笔只要和前线段的重叠部分重叠，那么必然会形成一线段。

3、换言之，线段破坏的充要条件，就是被另一个线段破坏。

也就是说新线段的产生必然是对旧线段的破坏，同时我们要注意：充要条件。充分且必要的条件。但我们要体会，这个充要并不是指一定，这里有是差别的。差别在于达到了这个充分且必要的条件，你可以划分一个线段，也可以不划分一个线段。

（读到这，也许您会觉得要么是缠师说话不严谨，或者是我瞎胡乱解读。实则不然，我们要明白，我们是用一个主观认知去人为的划分一个客观随机走势。所以这个定理如何适用，还看我们个人当下如何处理。）

说到这里，我想我们对比上面动力学和形态学的图，我们就不难发现，有些东西是一样的。有些东西又不一样。那么又回到我们之前的那个话题：所以走势的自同构性，本质是走势的结构，那么我们可以理解为动力学和形态学不过是对这个结构而言的不同观察视角。
