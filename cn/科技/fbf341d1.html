摩擦对于螺纹联接的影响

2020-11-14
| 科技

螺纹联接的原理是，施加在螺栓或螺母用的扭矩转换成夹紧力而对联接进行紧固。在螺纹联接中，摩擦的影响至关重要。这里我们将要讨论摩擦对于螺纹联接的影响，可能导致的问题，怎么做可以解决或消除这些问题。

根据ISO 16047：2005中的10.2 和10.3中公式[1]，推算出扭矩与夹紧力的计算公式：

T=F x (0.16 x P + 0.577 x d2 x µth + 0.5 x µb x Db) （1）

T=0.16 x P x F + 0.577 x d2 x µth x F + 0.5 x µb x Db x F （2）

T：安装扭矩；F：夹紧力；P：螺距； d2：螺牙直径；µth ：螺纹副摩擦系数；Db：支承面摩擦直径；µb：支承面摩擦系数。

从公式（2）可以看出，螺纹联接中，施加在紧固件上的安装扭矩分成了三部分[2]（图1）：

第一部分：用于产生夹紧力（0.16 x P x F），称为夹紧扭矩（Tp）；

第二部分：用于克服内外螺纹副的摩擦（0.577 x d2 x µth x F），称为螺纹副摩擦扭矩（Tth）；

第三部分：用于克服紧固件与被紧固零件的支承面的摩擦（0.5 x µb x Db x F ），称为支承面摩擦扭矩（Tb）。

安装扭矩的分配比例是，10%的安装扭矩转换成夹紧力来对零件进行紧固，40%的安装扭矩用来克服螺纹副部分的摩擦，50%的安装扭矩用来克服支承面的摩擦 [3]。可以看出，摩擦对于螺纹紧固联接的重要性，90%的安装扭矩用来克服螺纹联接中的摩擦。

图1：螺纹联接示意图

螺纹联接的实际安装过程中，往往由于支承面或者螺纹副的摩擦的变化造成各种失效问题。摩擦变小会导致螺栓在安装过程中出现拉伸变形或者拉断的问题。相反的，摩擦提高会导致联接的夹紧力降低而出现联接松动或者异响问题等等。在这里，我们统称为联接失效。

什么原因导致螺纹联接中摩擦的变化？进而导致什么问题问题？如何避免？

1) 零件（包含紧固件和被连接零件）的生产和表面处理

紧固件的冷墩精度，螺纹加工精度，牙侧粗糙度和支承面（包含紧固件和被连接零件的）的粗糙度影响到螺纹的的摩擦系数，加工精度越高，表面越光洁，摩擦系数越小，反之则摩擦系数越大。

螺栓和螺母的表面处理，可以改变螺纹和支撑面的摩擦系数。现在常见的表面处理类型有电镀锌，电镀锌镍合金，锌铝涂覆，镀铜，磷化等，不同表面处理的摩擦系数不同。

从公式(1)可以看出，摩擦系数的大波动会导致夹紧力的大波动，进而引发联接失效问题。

相反，摩擦系数的变化范围越小，夹紧力波动就越小。目前可以在紧固件表面时，增加额外的润滑工艺上，对摩擦系数范围进行调整和控制，各大主机厂摩擦系数的要求一般控制在±0.03。同时，定安装扭矩时，尽量使用模具状态下的零件（包含紧固件和被连接零件）进行试验，以求最真实地模拟量产阶段的安装状态。这样，定出来的安装扭矩才可靠。

2) 轴向力

按我们以往的经验，两种固定的材料，表面的状态不变，那么这两种的摩擦系数是不变的。但是在紧固件的摩擦系数试验过程中，我们会发现紧固件的总摩擦系数是会改变的，而且呈现的规律为摩擦系数随轴向力的加大而降低，最后趋于稳定。如图2所示，M12螺栓轴向力与总摩擦系数的关系。

载荷是通过接触面积的大小和变形状态来影响摩擦力。

常规方法加工的粗糙表面, 摩擦总是发生在一部分接触峰点上。接触点数目和各接触点尺寸将随着载荷而增加,最初是接触点尺寸增加, 随后载荷增加主要引起接触点数目增加。由于实际接触面积与载荷的非线性关系,使得摩擦系数随着载荷的增加而降低。[4]

3) 装配条件

a. 零件组装过程中，有润滑油或者润滑脂混入导致摩擦系数降低。导致螺栓的拉伸或断裂失效。所以在没有特别要求的情况下，请避免在螺纹联接的安装环境中出现润滑油或者润滑脂。

b. 异物（如铁屑，焊渣以及零件表面涂层破坏堆积等）夹杂。这种情况可能会导致的问题有两种：一种是由于异物夹杂导致支承面和螺纹副摩擦提高，相应的夹紧力降低，联接出现松弛的问题。

第二种是，零件组装的时候，螺纹副出现卡滞和滑牙，螺栓头部或者螺母还没有与支承面接触安装就停止。

有异物夹杂的装配，可能还会伴随着两个种现象：一种是“嚣叫”，紧固件拧紧过程中出现很刺耳的声音。一种是“粘滑效应”，就是在紧固件拧紧的过程中，出现拧紧突然停止（粘），然后很快拧紧又继续（滑），拧紧曲线显示的是忽高忽低（见图4）。这是因为在装配的过程中，异物的夹杂导致支承面和螺牙配合处的摩擦增大，某一瞬间需要输入的扭矩很高而导致拧紧停顿，可实际上拧紧并没有完成。这种情况发生的时候，联接很容易出现松动。

对于焊渣和涂层这种异物的覆着，我们可以涂一层特殊的涂层（PTFE）在螺纹上以防止焊渣和涂层的附着，从根本上解决问题。另外一种方法是在外螺纹上涂一层润滑剂来降低摩擦系数，也可以解决出现的问题。另外我们也可以通过调整安装工具的转速来解决“粘滑效应”问题。如图4所示，在工具的安装转速为55rpm的时候，“滑移效应”出现，当工具的安装转速调整为25rpm的时候，“滑移效应”消失。不过，通过提高还是调低工具转速来解决“滑移效应”，需要通过试验来定。

图4：粘滑效应

4) 紧固件安装工具的转速

安装工具的转速对摩擦系数也是有影响的。通常，安装工具转速高的话，联接的摩擦系数会降低。在制定螺纹联接的安装工艺时，我们应该通过一系列的试验来确定安装工具的转速。正常的生产时，不要随便调整安装工具的转速，以防止影响联接的稳定性。

5) 零件的锈蚀

零件的腐蚀造成摩擦系数的变化。零件腐蚀之后，会在零件表面产生一些白锈或红锈或者因锈蚀造成零件表面凹凸不平，导致摩擦系数提高，最终可能导致联接松弛，或者“滑牙”的问题出现。在零件安装前，发现零件有锈蚀的时候请不要用于装配。

上面介绍了影响螺纹联接中摩擦的各种因素，可以看出螺纹联接中，零件本身的状态，安装条件以及安装工艺等都会对联接中的摩擦造成影响。

虽然螺纹联接中的高摩擦会降低联接中的夹紧力，但是只要设计合理，在保证足够夹紧力的情况下，我们可以 通过提高摩擦来防止螺纹联接的旋转松动问题。螺纹联接的旋转松动是指内外螺纹之间发生在松开方向的相对转动时发生的松动。防止旋转松动的方法之一就是增大被联接件表面的摩擦系数[5]。
