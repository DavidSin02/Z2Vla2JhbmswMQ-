【诗月亮“每日一诗”诗与评精品展播】第117期:《峰》

2020-10-29
| 科技

诗月亮临屏诗群“每日一诗”第117期

诗题：《峰》

出题人：爱情石

/

1.峰

文/黄少先

你高耸入云

巍然屹立在群山之巅

春夏秋冬千万年

爱情石点评：诗人黄少先的诗唯美大气，不仅高耸入云，巍然屹立，且立于天地之间千万年！很棒！开了好头！

/

2. 峰

文/顺其自然(甘肃)

高耸在群山之巅

呐喊

震荡天宇

不畏严寒酷暑

怀豁达的胸襟

笑看四季胜景

阳光沐你成长

蓝天伴你入眠

虫鸟为你吟唱

溪水为你奏乐

深谷是你的骄傲

翠郁是你的外衣

坚守一片丹心

在最高深的境界里

迎风顶立

陶醉在云海里

爱情石点评：女诗人顺其自然的诗秀气，清雅，从女性角度来描述的“峰”有另外一种精彩，且在最高深的境界里，迎风顶立，陶醉在云海之中。美!

/

3.峰

文／风（浙江）

塔尖上的孤独

云知道，天知道

极高之处

往往，身不由己

更有虚无之风

成为，峰的立体

天下山峰

高低不同

你有你的峰

我有我的峰

其实，峰

是一个载体而已

爱情石点评：诗人风从哲思的角度，从深层次把握心中的“峰”写得很棒，且令人眼睛一亮，哲理之光给人的启迪，远远超过“峰”，显示诗人作为兰苑文学第一批签约作家的独立思辩能力，精彩！

/

4.峰

文/荷影双双

你一直是我瞧盼的顶

努力的攀登

守候着你的魂

爱情石点评：诗人荷影双双的微诗写得很坦诚且深入，努力攀登(显示诗人的辛勤！)守候着你的魂(写得相当深情！)为诗人点赞喝彩!

/

5.峰

文/牛亚梅

每天 给自己一个高度

不懈的攀爬

达顶

爱情石点评：诗人牛亚梅以“峰”作为奋斗目标，写出另一种精彩，心中有梦，总会有实现之日，微诗以小见大，且文字凝炼！棒!

/

6.峰

文/邹以浩

你高高地挺立

俯视着山川大地

你一身傲骨

但却没有傲气

因为你知道

离开脚下的土地

只是一堆堆土石

爱情石点评：诗人邹以浩的小诗融入诗人独特的体验，有哲理启迪，发人深省。精彩！

/

7.峰

文/吴延安

打一天

与蓝天相识

就把梦想写进了云里

攀登了多少年

山有高度

梦想无边

无法抵达

才萌生了一种力

挺拔如峰

爱情石点评：诗人延安的诗因为“无法抵达/才萌生了一种力/挺拔如峰”诗人的诗，的确有其独特之处。只要坚持下去，一定有更多收获！

/

8.峰

文/成永军

云山雾罩

胜似仙境

各大门派争夺之地

技高一筹

压倒群雄

实力决定高度

要想真正成为霸主

有坚如磐石的基础

高山仰止的气势

一览众山小的胸怀

爱情石点评：诗人成永军最后领略到一览众山小的情怀，应该为诗人点赞喝彩。就是要写出与众不同，才能更体现自己的精彩！希望今天的“峰”成为诗人的一个良好开始。加油！

/

9.峰

文/侯国廷/山东

敞开胸怀

接受蓝天的检阅

征服 一个又一个高度

爱情石点评：诗人侯国廷以“峰”自喻，敞开胸怀，自觉接受蓝天的检阅，征服一个又一个高度！“峰”的风彩尽显，为诗人点赞！

/

10.峰

文/幸福小妇人

心徜徉蔚蓝

云柔柔作伴

梦 在众生仰望的高度实现

爱情石点评：诗人幸福小妇人的诗“心”，“云”，“梦”映衬了“峰”的威仪，梦的实现，其实就是众生仰望的高度实现。祝福诗人！

/

11. 峰

文/ 芸 艸

仰望峨眉、我把自己站成山脚下的仙姝，

等了一个世纪的轮回，

才有机会、沿着崎岖的山路前行，

去寻找那灵秀空旷的山峰中的妙缘；

峰、突兀在金顶的斜坡、雷洞坪的侧面，

当你看清楚了它的容颜、它已隐去了真名，

透过千变万化的云蒸霞蔚、已模糊了视线，

它习惯了镇守了千年的山川、孤独的守望；

轻轻靠近你、在奇花异草里前路漫漫，

在古道里探秘、看见杜鹃啼血的形式，

往前一步是藤蔓的匍匐、却牵住千仞的尺壁，

深谷悠悠、疑是满盈却是虚无的悬空；

从此、把峰口的珙桐花刻在心底，

一生一世、只需一次相见就足够，

高高在上、你的样子，

像峨眉山洗象池的皎月、长长久久。

爱情石点评：诗人芸艸的诗，“峰”既是一种高度，诗人用其女诗人的视角融入了诸多元素，意象之美，烘托出意境之美。最后一句“像峨眉山洗象池的皎月、长长久久。”令人向往！真的了不起!

/

12.峰

文/徐小平

努力托起负重的身躯

高些再高些

心中的信念一直都在

与云拥抱

与鹰接吻

做一个逍遥的君子

沙石不断的扯裂

所有的痛

都在仰望中融化在

深蓝里的天空

爱情石点评：诗人徐小平的清新小诗，最打动我的便是诗的后三句“所有的痛/都在仰望中融化在/深蓝里的天空。”非常棒！祝贺祝福诗人!

/

13.峰

文/郑瀚玮

十万风雪来战

险处烟云四起

勇者必胜

爱情石点评：诗人郑瀚玮的微诗，体现一种狭路相逢勇者胜的精神！“峰”的神勇跃然纸上，是一首很棒的微诗。祝贺诗人！

/

14.峰

文/东荣

洒脱而温柔的女神

腾云驾雾的汉子

一直追求你

爱情石点评：诗人东荣人格诗化，将“峰”赋予男子汉的生命的色彩，腾云驾雾追求心中的“女神”很浪漫，很洒脱！棒!

/

15.峰

文/星空物语

沉默中愈加伟岸

站在父亲的肩头

挥一把云彩 望断天涯路

爱情石点评：诗人星空物语的微诗也是以“峰”喻人，写出了属于“峰”独有的精彩，“挥一把云彩/望断天涯路”。出彩！

/

16.峰

文/无弦

鸟声中醒来

当朝霞落入眼眸

一颗露珠，于掌心中颤抖

爱情石点评：诗人无弦的微诗带着几分朦胧，体现一个女诗人的细致入微，以一颗露珠折射阳光。诗人的“峰”虽然不说，已然在诗人心中。巧妙！

/

17.峰

文/赵书周

多少次在你面前驻足，仰望

无限风光

令人神往

想飞上你

却没翅膀

你依然骄傲

不为任何所动

不会府下身来

为弱者卑躬

我才明白

原来可以用双脚丈量你的高度

以及这座山与那座山的距离

峰，便是我的高度

爱情石点评：诗人赵书周以小诗来写“峰”总体还是不错的。诗人自信“原来可以用双脚丈量你的高度/以及这座山与那座山的距离/峰，便是我的高度”同时体现了自信与自我意识的觉醒。棒!

/

18.峰

文/一树繁花

可以沉默到不闻不问

所有的高高低低 都在心里

不媚不俗 不倨不傲

爱情石点评：诗人一树繁花写得既坦诚又有几分含蓄，可以沉默/高低在心，也可以“不媚不俗，不倨不傲”。棒！

/

19.峰

文/冷月无声

风儿轻拂 云为裳

笑看世间沧桑

岿然屹立 亿万年

爱情石点评：诗人冷月无声用拟人夸张手法，将“峰”写得非常精彩。“风”，“云”意象用得精准，笑看(恰到好处!)世间沧桑，岿然不动，亿万年！(夸张用到极至无穷尽！真的非常棒！)

/

20.峰

文/心静如水

岁月

沉淀了你的厚重

在

绵延云端的高度

俯视着万物

爱情石点评：诗人心静如水的诗从岁月入诗，沉淀了你的厚重，“峰”的形象跃然纸上，且在“绵延云端的高度/俯视着万物(俯视用得好！烘托“峰”的高度与威仪)全诗意境优美，精彩！

/

21.峰

文 /紫霞

云堤满山坳

放眼峰的高度

在眼里更在心里

爱情石点评：诗人紫霞的诗意象用得棒，意境营造得美！语言富有张力，是一首挺美的微诗！今天微诗开局，又以微诗收官。二十一位诗人从不同角度将“峰”的精彩演绎出来，真的挺好的。统一为诗人们点赞!祝贺祝福!
