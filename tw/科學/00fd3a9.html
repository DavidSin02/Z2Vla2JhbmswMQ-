【圖像處理】OpenCV系列十八 --- 幾何圖像變換函數詳解(二)

| 科學

一、linearPolar()函數

1、函數原型

void linearPolar(InputArray src,  OutputArray dst, Point2f center,  double maxRadius,  int flags);

2、函數功能

將一幅圖像映射到極座標空間中。

3、參數詳解

  • 第一個參數，InputArray src，原圖像，即輸入圖像；
  • 第二個參數，OutputArray dst，目標圖像，與原圖像有一樣的尺寸與類型；
  • 第三個參數，Point2f center，變換的中心座標；
  • 第四個參數，double maxRadius，逆向變換中的最大變徑；
  • 第五個參數，int flags，插值方法的組合。

Note:該函數不支持就地操作。

二、logPolar()函數

1、函數原型

void logPolar(InputArray src,  OutputArray dst, Point2f center,  double M,  int flags);

2、函數功能

將圖像映射到半極座標空間；

3、參數詳解

  • 第一個參數，InputArray src，原圖像，即輸入圖像；
  • 第二個參數，OutputArray dst，目標圖像，與原圖像具有相同的尺寸和類型；
  • 第三個參數，Point2f center，轉換的中心座標，其中輸出的精度是最大的；
  • 第四個參數，double M，圖像幅度的比例參數，決定了用多大的邊界圓的半徑進行轉換；
  • 第五個參數，int flags，插值方法與映射模式的組合；

Note:該函數不支持就地操作。

三、warpPolar()函數

1、函數原型

void warpPolar(InputArray src,  OutputArray dst,  Size dsize, Point2f center,  double maxRadius,  int flags);

2、函數功能

將一幅圖像映射到極座標或者半極座標空間中；

極座標映射與半極座標映射的比較：

warpPolar函數的映射模式可以是極座標映射，也可以是半極座標映射，通過flags參數來進行指定映射的模式！系統默認的模式是極座標映射模式。半極座標映射用來模仿人類的中心視覺，在視覺聚焦的地方可以看的很清晰，非聚焦的地方會比較模糊，通過弱化聚焦以外的區域，來簡化圖像，方便對圖像進行處理。

3、參數詳解

  • 第一個參數，InputArray src，原圖像，即輸入圖像；
  • 第二個參數，OutputArray dst，目標圖像，與原圖像具有相同的類型；
  • 第三個參數，Size dsize，目標圖像的尺寸，根據這個參數來設計目標圖像的大小；
  • 第四個參數，Point2f center，圖像進行變換的中心座標；
  • 第五個參數，double maxRadius，要進行變換的邊界圓的半徑，決定了圖像逆向幅度的大小；
  • 第六個參數，int flags，插值的方式與極座標變換的模式的組合；
  • 極座標變換的模式有以下幾種：

(1)極座標變換(WARP_POLAR_LINEAR)

(2)半極座標變換(WARP_POLAR_LOG)

(3)對圖像進行逆向映射(WARP_INVERSE_MAP)

Note:

(1)該函數不支持就地操作；

(2)為了計算角的大小和角度，內部採用了cartToPolar函數，測量角度在0到360之間，精度約為0.3度；

(3)在實現過程中，該函數使用了remap函數，因為當前的實現限制，圖像的大小應小於32767x32767。

綜合實例

對於linearPolar()函數與logPolar()以及warpPolar()函數的使用

#include <opencv2/imgproc.hpp>#include <opencv2/highgui.hpp>#include <iostream>using namespace cv;int main(int argc, char** argv){ // 半極座標轉換圖像 Mat log_polar_img,  // 極座標轉換圖像 lin_polar_img,  // 逆半極座標轉換圖像 recovered_log_polar, // 逆極座標轉換圖像 recovered_lin_polar_img;   // 創建窗體 namedWindow("Linear-Polar", WINDOW_AUTOSIZE); namedWindow("Log-Polar", WINDOW_AUTOSIZE); namedWindow("Recovered Linear-Polar", WINDOW_AUTOSIZE); namedWindow("Recovered Log-Polar", WINDOW_AUTOSIZE); // 移動窗口的顯示位置 moveWindow("Linear-Polar", 20, 20); moveWindow("Log-Polar", 700, 20); moveWindow("Recovered Linear-Polar", 20, 350); moveWindow("Recovered Log-Polar", 700, 350); // INTER_LINEAR 線性插值 // WARP_FILL_OUTLIERS 填充所有目標圖像像素。 // 如果其中一些對應於源圖像中的異常值， // 則將其設置為零。 int flags = INTER_LINEAR +  WARP_FILL_OUTLIERS; // 加載圖像 Mat src = imread("lena.png");  if (src.empty())  { printf("image error!"); return 0; }  // 原圖像的中心座標 Point2f center((float)src.cols / 2,  (float)src.rows / 2); // 最大的半徑 double maxRadius = 0.7*min(center.y, center.x); // direct transform // linear Polar warpPolar(src,  lin_polar_img,  Size(),  center,  maxRadius,  flags);  // semilog Polar warpPolar(src,  log_polar_img,  Size(),  center,  maxRadius,  flags + WARP_POLAR_LOG);   // inverse transform warpPolar(lin_polar_img,  recovered_lin_polar_img,  src.size(),  center,  maxRadius,  flags + WARP_INVERSE_MAP); // inverse transform warpPolar(log_polar_img,  recovered_log_polar,  src.size(),  center,  maxRadius,  flags + WARP_POLAR_LOG + WARP_INVERSE_MAP);  // 從極座標系到直角座標系 Mat dst; if (flags & WARP_POLAR_LOG) dst = log_polar_img; else dst = lin_polar_img; //get a point from the polar image int rho = cvRound(dst.cols * 0.75); int phi = cvRound(dst.rows / 2.0); double angleRad, magnitude; double Kangle = dst.rows / CV_2PI; angleRad = phi / Kangle; if (flags & WARP_POLAR_LOG) { double Klog = dst.cols / std::log(maxRadius); magnitude = std::exp(rho / Klog); } else { double Klin = dst.cols / maxRadius; magnitude = rho / Klin; } // 極座標的點轉換為直角座標的點 int x = cvRound(center.x + magnitude * cos(angleRad)); int y = cvRound(center.y + magnitude * sin(angleRad)); drawMarker(src, Point(x, y), Scalar(0, 255, 0)); drawMarker(dst, Point(rho, phi), Scalar(0, 255, 0)); imshow("Src image", src); imshow("Log-Polar", log_polar_img); imshow("Linear-Polar", lin_polar_img); imshow("Recovered Linear-Polar", recovered_lin_polar_img); imshow("Recovered Log-Polar", recovered_log_polar); waitKey(0); return 0;}

四、resize()函數

1、函數原型

void resize(InputArray src,  OutputArray dst, Size dsize,  double fx = 0,  double fy = 0, int interpolation =  INTER_LINEAR);

2、函數功能

對圖像的尺寸進行調整，使其縮小或放大為指定的大小；

Note:

(1)如果沒有考慮目標圖像(dst)初始化的大小和類型，則目標圖像的大小和類型由原圖像(src)，dsize,fx,fy等參數決定，例如：

// 指定fx,fy的值，//由函數自動計算目標圖像的大小resize(src, dst,  Size(), 0.5, 0.5,  interpolation);

(2)預先創建dst圖像的大小與類型，例如：

// 創建一個800x600的8位單通道的圖像Mat dst(800, 600, CV_8UC1); resize(src, dst, dst.size(), 0, 0, interpolation);

3、參數詳解

  • 第一個參數，InputArray src，原圖，即輸入圖像；
  • 第二個參數，OutputArray dst，目標圖像，其類型與原圖像一致，尺寸可以進行自定義設置，或者通過參數src.size(),fx,fy等參數自動計算出來；

五、remap()函數

1、函數原型

void remap(InputArray src,  OutputArray dst, InputArray map1,  InputArray map2, int interpolation,  int borderMode = BORDER_CONSTANT, const Scalar& borderValue = Scalar());

2、函數功能

對圖像進行幾何變換；

3、參數詳解

  • 第一個參數，InputArray src，原圖像，即輸入圖像；
  • 第二個參數，OutputArray dst，目標圖像，圖像的尺寸的大小與map1相同，圖像的類型與原圖像一致；
  • 第三個參數，InputArray map1， (x，y)點或僅x值的第一個映射，其類型為CV_16SC2、CV_32FC1或CV_32FC2；有關將浮點表示形式轉換為固定點以提高速度的詳細信息，可以看我上一節所講的convertMaps函數；
  • 第四個參數，InputArray map2，y值的第二個映射，其類型分別為CV_16UC1、CV_32FC1或None(如果map1為(x，y)點，則為空映射)；
  • 第五個參數，int interpolation，插值的方法，這個函數不支持INTER_AREA插值；
  • 第六個參數，int borderMode = BORDER_CONSTANT，邊界模式；當邊界模式為BORDER_TRANSPARENT時，源圖像中的“異常值”的像素在目標圖像中不會被函數修改；
  • 第七個參數，const Scalar& borderValue = Scalar()， 在常量邊界情況下使用的邊值；默認情況下，它是0。

void warpAffine(InputArray src,  OutputArray dst, InputArray M,  Size dsize, int flags = INTER_LINEAR, int borderMode = BORDER_CONSTANT, const Scalar& borderValue = Scalar());

2、函數功能

用指定的卷積核對圖像進行仿射變換；

3、參數詳解

  • 第一個參數，InputArray src，原圖像，即輸入圖像；
  • 第二個參數，OutputArray dst，目標圖像，與原圖像具有同樣的尺寸與類型；
  • 第三個參數，InputArray M，2 x 3的變換矩陣；
  • 第四個參數，Size dsize，目標的圖像的尺寸大小，根據這個參數來指定目標圖像的大小；

  • 第六個參數，int borderMode = BORDER_CONSTANT，邊界模式，當邊界模式為BORDER_TRANSPARENT時，源圖像中的“異常值”的像素在目標圖像中不會被函數修改；
  • 第七個參數，const Scalar& borderValue = Scalar()， 在常量邊界情況下使用的邊值；默認情況下，它是0。

七、warpPerspective()函數

1、函數原型

void warpPerspective(InputArray src,  OutputArray dst, InputArray M,  Size dsize, int flags = INTER_LINEAR, int borderMode = BORDER_CONSTANT, const Scalar& borderValue = Scalar());

2、函數功能

用指定的卷積核對圖像進行透視變換；

當flags設置為WARP_INVERSE_MAP時，使用以下的卷積對圖像進行透視變換，

透視變換

否則，先使用逆變換進行轉置，然後用上面的公式代替M，並且此函數不支持就地操作；

3、參數詳解

  • 第一個參數，InputArray src，原圖，即目標圖像；
  • 第二個參數，OutputArray dst，目標圖像，與原圖具有同樣的尺寸和類型；
  • 第三個參數，InputArray M，3 x 3的轉換矩陣；
  • 第四個參數，Size dsize，目標圖像的大小，根據這個參數來設計目標圖像的大小；
  • 第五個參數，int flags = INTER_LINEAR，將常用的插值方法(INTER_LINEAR、 INTER_NEAREST)與可選標誌（WARP_INVERSE_MAP）進行組合，如果與可選標誌結合的話，將M設置為逆變換(dst --> src);
  • 第六個參數，int borderMode = BORDER_CONSTANT，邊界模式(BORDER_CONSTANT、BORDER_REPLICATE)；
  • 第七個參數，const Scalar& borderValue = Scalar()， 在常量邊界情況下使用的邊值；默認情況下，它是0。
