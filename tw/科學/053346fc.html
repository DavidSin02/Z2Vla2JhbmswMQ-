路基季節性施工

一、路基雨期施工

1.雨期施工地段的選擇

(1)雨期路基施工地段一般應選擇丘陵和山嶺地區的砂類土、碎礫石和岩石地段以及路塹的棄方地段。

(2)重黏土、膨脹土及鹽漬土地段不宜在雨期施工；平原地區排水困難，不宜安排雨期施工。

3.雨期填築路堤

( 1 ) 填料應選用透水性好的碎石土、卵石土、砂 礫 、石方碎渣和砂類土等。利用挖方土作填料，含水率符合要求時，應隨挖隨填，及時壓實。含水率過大難以晾晒的土不得用作雨期施工填料。

( 2 ) 每一填築層表面應做成2% ~ 4% 雙向路拱橫坡以利於排水，低窪地帶或高出設計洪水位0.5m以下部位應選用透水性好、飽水強度高的填料分層填築，並及時施作護坡、坡腳等防護工程。

( 3 ) 雨期填築路堤需借土時，取土坑的設置應滿足路基穩定的要求。

( 4 ) 路堤應分層填築，並及時碾壓

4.雨期開挖路塹

( 1 ) 挖方邊坡不宜一次挖到設計坡面，應預留一定厚度的覆蓋層，待雨期過後再修整到設計坡面。

( 2 ) 雨期開挖路塹，當挖至路床頂面以上300 ~ 500mm時應停止開挖，並在兩側挖好臨時排水溝，待雨期過後再施工。

(3)雨期開挖岩石路塹，炮眼應儘量水平設置。

二、路基冬期施工技術

在季節性凍土地區，晝夜平均溫度在－3℃以下，連續10天以上時，進行路基施工稱為路基冬期施工。當晝夜平均溫度雖然上升到－3℃以上，但凍土未完全融化時，亦應按冬期施工。

2.路基工程不宜冬期施工的項目

（1）高速公路、一級公路的土路基和地質不良地區的的公路路堤不宜進行冬期施工 。土質路堤路床以下1m 範圍內，不得進行冬期施工。半填半挖地段、填挖交界處不得在冬期施工。

（2）剷除原地面的草皮、挖掘填方地段的臺階。

（3）整修路基邊坡。

（4）在河灘低窪地帶將被水淹的填土路堤。

4.冬期填築路堤（16多）

( 1 ) 路堤填料應選用未凍結的砂類土、碎 石 、卵石土、石渣等透水性好的材料，不得用含水率大的黏質土。

( 2 ) 填築路堤應按橫斷面全寬平填，每層鬆鋪厚度應比正常施工減少20% 〜 30% ，且鬆鋪厚度不得超過300mm。當天填土應當天完成碾壓。

( 3 ) 中途停止填築時，應整平填層和邊坡並進行覆蓋防凍，恢復施工時應將表層冰雪清除，並補充壓實。

( 4 ) 當填築高程距路床底面1m 時 ，碾壓密實後應停止填築，在頂面覆蓋防凍保溫層 ，待冬期過後整理復壓，再分層填至設計高程。

5.冬期施工開挖路塹表層凍土的方法

（1）爆破凍土法：冰凍深度達1m以上

（2）機械破凍法：1m以下的凍土層，具體方法：凍土犁、凍土鋸和凍土鏟等。

（3）人工破凍法：當冰凍層較薄，破凍面積不大，具體方法：日光暴晒法、火燒法、熱水開凍法、水針開凍法、蒸汽放熱解凍法和電熱法等方法脹開或融化冰凍層，並輔以人工撬挖。

6.冬期開挖路塹（11年單）

( 1 ) 挖方邊坡不得一次挖到設計線，應預留一定厚度的覆蓋層，待到正常施工季節後再修整到設計坡面。

( 2 ) 路基挖至路床頂面以上1m時，完成臨時排水溝後，應停止開挖，待冬期過後再施工。
