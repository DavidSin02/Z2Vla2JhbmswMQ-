都是玻尿酸你買對了嗎？3類精華，適合不同人，可別盲目跟風買

2020-10-29
| 科學


玻尿酸你一定不陌生吧？

這年頭，愛護膚的女孩誰還沒聽過玻尿酸？

恐怕沒有誰的護膚品裡會找不到玻尿酸的成分吧。

可是大家真的都瞭解玻尿酸嗎？恐怕不是。

很多人的瞭解僅僅停留在"補水作用"而已。就不說水光針、填充的作用，單說市面上這麼多玻尿酸產品如何挑一款適合自己的，很多人就沒考慮過。

今天我就拿三款自用的玻尿酸產品舉例說明，如何挑選適合自己的產品。

不過嘛……先來補習一下玻尿酸的基礎知識。


玻尿酸護膚：你需要了解的基礎知識



  1. 玻尿酸或稱透明質酸是一種直鏈高分子多醣
  2. 廣泛存在於人體中，皮膚內位於真皮層
  3. 可吸收1000倍於其重量的水分，起到儲水功能
  4. 天然透明質酸在人體內的半衰期僅為1-2天
  5. 玻尿酸的效果與分子大小和濃度有關
  6. 高分子量玻尿酸（如透明質酸鈉）質地濃稠，吸水力強但無法穿透表皮，可滋潤角質層、鎖住水分、減少經表皮水分流失
  7. 中分子量玻尿酸可滲透至真皮層，鎖水能力相對較弱，且容易揮發，但與皮膚手提結合後可起到長效保溼
  8. 寡聚透明質酸可深入真皮層，起到深層保溼、抗衰的效果
  9. 作為護膚品，0.025~0.05%濃度的高分子量玻尿酸就可起到保溼作用；但如果把1%的高分子量玻尿酸直接塗抹在皮膚上，反而會更幹

這些知識點跟使用玻尿酸護膚有關，那些和醫美整容有關的考點暫時按下不表。

發現沒有？

雖然玻尿酸吸水能力很強，但並不見得是萬能的保溼成分——透皮率低、半衰期短。

雖然玻尿酸護膚很有必要，但並不見得濃度高、分子量大就是好——環境溼度不夠就會倒拔幹。

所以，曾經紅極一時的「太陽社玻尿酸原液」並不是日常護膚的優質選擇。

市面上琳琅滿目的玻尿酸精華，你也需要根據自己的膚質從成分表入手，進行選擇。


護膚達人、沙漠幹皮：可選擇單品玻尿酸原液

/ 代 / 表 / 產 / 品 /


「Timeless」1%玻尿酸原液

核心成分：水、1%玻尿酸、防腐劑

規格：79rmb / 30ml

產地：美國

如果你是個經驗豐富、專業知識紮實的成分黨，可以考慮選擇玻尿酸原料桶，和其他的單品成分配合使用。

但如果你只是普通消費者，我勸你不要輕易嘗試。

我買過唯一的單品玻尿酸原液就是「Timeless」這款，它跟「太陽社」最大的區別是流動性較好。根據1%的濃度和流動性表現可以推測，使用的玻尿酸分子量約100萬左右。

玻尿酸原液雖然能吸水，卻無法穿透表皮層，不能當做精華液來使用。

正確的使用方法是加入到補水能力不夠強的護膚品/化妝品中使用。

也可以與其他的活性成分一同使用，比如與角鯊烷、神經酰胺配合，模擬類似天然保溼因子的效果。

但問題是，如果要七八瓶一起用DIY，那還不如買一瓶現成的保溼精華。


長效補水、抗初老：可選擇多重玻尿酸精華

/ 代 / 表 / 產 / 品 /


Bielenda Serum facial hidratante

碧蓮達玻尿酸賦活基底精華

核心成分：甜菜鹼、甘油、棕櫚酰三肽-5、玻尿酸、水解葡糖氨基聚糖、透明質酸鈉、維生素B5、乳酸

規格：149rmb / 30ml

產地：波蘭

事實上，現在有很多保溼產品用的都是多重玻尿酸的配方。

把不同分子量的玻尿酸按一定的比例配合到一起，就能均衡吸水能力、滲透力、持久度。

之前回購過的兩次的「頤蓮多重透明質酸原液」就是代表之一。

但今天用碧蓮達"小藍瓶"來說明。

這是我幾個月前和"小金瓶"一起買的，只是我先開了"小金瓶"。因為"小金瓶"是煙酰胺+VC+檸檬酸的配方，日夜可用，剛好用來修護日晒暗沉。

但其實"藍金組合"一起用就可以比較全面地照顧到20+的肌膚。因為"小藍瓶"的定位是肌底液，主攻補水保溼，成分表的前幾位都是實打實的活性成分。

甘油、玻尿酸、水解葡糖氨基聚糖（GAGs）和玻尿酸鈉都具有極佳吸水功能，其中玻尿酸和透明質酸鈉就是大、中分子量復配的組合。

甜菜鹼、維生素B5還具有抗靜電的作用，可見是為幹皮研製的。

另外棕櫚酰三肽-5就是抗衰老常用的小分子肽。它可以促進膠原蛋白（I型、II型、IV型）的合成和細胞外基質蛋白的生成，增加肌膚的彈性、提升肌膚含水量。

可以說，"小藍瓶"就是為抗初老量身定製的，預防缺水性老化。這個配方作為肌底精華，水狀質地好吸收、不容易打架，不同膚質還可以根據自己的需求疊加其他精華。

比如混合肌膚或有祛黃提亮需求的，先用碧蓮達的"小藍瓶"再用"小金瓶"就能照顧到全部需求。

類似的玻尿酸精華包括前面提到的「頤蓮多重玻尿酸原液」有個共性：質地水潤稀薄，復配兩種以上的玻尿酸。可能補水的效果並不明顯，但長期使用有種"潤物無聲"的效果，可以有效預防因皮膚缺水而帶來的衰老風險。

唯一要注意的是，它們基本不含封閉劑。所以護膚的最後一道工序一定要使用具有封閉性的保溼面霜/乳液。


問題肌膚：可選擇含玻尿酸成分的精華

/ 代 / 表 / 產 / 品 /

LA BIOSTHETIQUE LA CAPSULE HYDRATANTE CEL

貝伊絲 保溼精華膠囊

核心成分：硅類、透明質酸鈉、甘露聚糖、鹽角草萃取、迷迭香萃取、海茴香萃取、

規格：990rmb / 60粒

產地：德國


雖然皮膚乾燥是比較主流的肌膚狀況，但很多時候我們會面臨其他苦惱。

諸如水油不均衡帶來的油膩、黑頭、痤瘡問題，脆弱、敏感肌膚的修護問題，還有去皺抗衰的訴求。

這時候我們首選的精華液可能不是單純的保溼功效，而會看中其他的功效。比如很多人喜歡的煙酰胺、視黃醇、果酸、VC等等。

但是，如果肌膚也處在缺水狀態，這些功效成分並不能發揮最大作用。

畢竟護膚的基礎是補水保溼。所以我會建議你選擇合理搭配了中等濃度玻尿酸的精華液，換言之，玻尿酸的名字應該出現在成分表的前50%的位置。

以「貝伊絲保溼精華膠囊」為例，它的成分可以分為三大類：硅類封閉劑、透明質酸鈉和甘露聚糖為主的保溼劑以及抗氧化、收斂效果的活性成分。

其中具有封閉性的硅類含量最多，除了提供順滑的手感外，可以在皮膚表面建立封閉層防止水分散發；接下來就是保溼劑透明質酸鈉了。

雖然並不是水潤質感，但補水保溼的框架搭建得很完整，然後再配合葵花籽油、阿魏酸乙酯、維生素E抗氧化，迷迭香、海茴香、鹽角草收斂肌膚，起到調節水油平衡的效果。

在產品宣傳中，玻尿酸並不是主打成分，可實際上它的含量並不少，且使用的是分子量大的透明質酸鈉、滋潤角質的效果比較明顯。

和碧蓮達小藍瓶相比，「貝伊絲保溼精華膠囊」更側重調理而非肌底補水，所以更適合混合肌膚和皮脂膜受損的幹皮使用，卻不適合痘痘肌膚，因為會有閉口風險。

如果是中性或偏油肌膚，「貝伊絲保溼精華膠囊」可作為妝前精華，有平滑肌膚的功效。但如果你是沙漠皮，一定要先用水分或肌底精華滋潤皮膚後再用。


KEYPOINTS & WARNING

三種類型的玻尿酸精華分析到這裡。

再來總結下：

  • 單品玻尿酸原液不建議直接用來護膚，而要和其他產品搭配使用；膠水質地不建議選購，具有流動性的長效保溼效果更好。
  • 水樣流動的多重玻尿酸精華是適合各種膚質的百搭產品，即時效果不明顯但長期使用，具有對抗乾燥老化的功效。
  • 肌膚問題比較多的時候，選擇針對性的精華液要參考玻尿酸成分的含量，選擇成分表中玻尿酸排名中等靠前的才能較好地調理肌膚。

最後，如果你足夠細心，可以看到開頭的玻尿酸基礎知識中並沒有提到「低分子量玻尿酸」。

原因是低分子量的玻尿酸對肌膚損傷會產生炎症反應，原本是有促進癒合的作用，但如果長期使用反而會加速衰老哦。

如果你經常長痘、面部有損傷，那就一定要避開「水解透明質酸」這樣的成分了。
