《集合啦！動物森友會》20種素材島地形及資源一覽

2020-11-14
| 科學

文丨harrychengx@bigfun社區

1. 普通島 #1

首次探訪素材島會遇到的島型，非常普通。有懸崖、自島果樹、椰樹、河流、池塘。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：9%

2. 普通島 #2

和普通島#1除了沒有池塘，懸崖面積更大外沒有區別。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：9%

3. 螺旋島

地形為螺旋狀，資源跟普通島相同。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：9%

4. “指尖陀螺”島

因島上的池塘形似指尖陀螺而得名。無河流，資源與普通島相同。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：9%

5. 石頭島

島分三層，無河流。底層為普通樹，中層為自島果樹，頂層有5塊礦。

  • 樹：自島果樹
  • 花：普通
  • 礦：素材 - 5塊
  • 蟲：鼠婦、蜈蚣
  • 魚：普通
  • 出現機率：9%

6. 竹子島

島上的樹都是竹子，其他資源與普通島相同。

  • 樹：竹子
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：10%

7. 水果島

水果島上有19棵同種非自島果樹。

  • 樹：非自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：9%

8. 異色花蝴蝶島

島中央池塘一圈為異色花，島上只刷新蝴蝶等花卉吸引的昆蟲。

  • 樹：闊葉樹
  • 花：異色花
  • 礦：普通 - 4塊
  • 蟲：蝴蝶為主
  • 魚：普通
  • 出現機率：2%（每天1次）

9. 鈴錢島

島中央有5塊可以敲出鈴錢的礦。

  • 樹：自島果樹
  • 花：普通
  • 礦：鈴錢 - 6塊
  • 蟲：鼠婦、蜈蚣
  • 魚：普通
  • 出現機率：5%

10. 蠍子島

島四個角各有一個懸崖，島上多花，礦能敲出鈴錢，昆蟲為蠍子。

  • 樹：闊葉樹
  • 花：普通
  • 礦：鈴錢 - 7塊
  • 蟲：蠍子
  • 魚：普通
  • 出現機率：2%（每天1次）

11. 狼蛛島

島上昆蟲只有狼蛛。

  • 樹：自島果樹
  • 花：普通
  • 礦：素材 - 7塊
  • 蟲：狼蛛
  • 魚：普通
  • 出現機率：2%（每天1次）

12. 闊葉樹島 #1

島上沒有河流、沒有池塘、只有闊葉樹和椰樹，昆蟲多為甲蟲類昆蟲。

  • 樹：闊葉樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：甲蟲居多
  • 魚：普通
  • 出現機率：2%（每天1次）

13. 闊葉樹島 #2

島型與竹子島相同，只是樹木都是闊葉樹，昆蟲多為甲蟲類昆蟲。

  • 樹：闊葉樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：甲蟲居多
  • 魚：普通
  • 出現機率：1%（每天1次）

14. 大魚島 #1

顧名思義，大魚島只會出大魚影。島上還會出現異色花。

  • 樹：自島果樹
  • 花：異色花
  • 礦：普通 - 4塊
  • 蟲：螳螂/蘭花螳螂/椿象/人面椿象/七星瓢蟲
  • 魚：大型魚類
  • 出現機率：3%（每天1次）

15. 大魚島 #2

島上只會出大魚影。花較少。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：大型魚類
  • 出現機率：2%（每天1次）

16. 水蟲島

島上有一條捲曲的河流，昆蟲只有水生昆蟲。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：紅蜻蜓/綠胸晏蜓/豆娘/無霸勾蜓/水黽/龍蝨/田鱉
  • 魚：普通
  • 出現機率：5%

17. 垃圾島

島上只能釣上垃圾，昆蟲只有水生昆蟲。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：紅蜻蜓/綠胸晏蜓/豆娘/無霸勾蜓/水黽/龍蝨/田鱉
  • 魚：垃圾
  • 出現機率：5%

18. 背鰭島

背鰭島上可發現所有的魚類，可以捕獲所有種類的鯊魚和海洋翻車魚。湖中島上的形狀也像背鰭。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：鯊魚/雙髻鯊/鯨鯊/鋸鯊/翻車魚/吸盤魚
  • 出現機率：1%（每天1次）

19. 瀑布島

島上瀑布、懸崖眾多，資源為普通資源。

  • 樹：自島果樹
  • 花：普通
  • 礦：普通 - 4塊
  • 蟲：普通
  • 魚：普通
  • 出現機率：5%

20. 金礦島

島上有一塊矩形池塘，中央有一塊可以敲出金礦的礦石。昆蟲為蠍子。

  • 樹：自島果樹
  • 花：普通
  • 礦：金礦 - 1塊
  • 蟲：蠍子
  • 魚：普通
  • 出現機率：1%（每天1次）
