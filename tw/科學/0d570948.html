華東理工大學劉昌勝院士團隊《AFM》：可注射的水凝膠促進軟骨和血管的原位形成

2020-11-14
| 科學

骨損傷的高病發率和當前臨床治療的侷限性促使人們開發具有高治療功效的可替代生物材料，來促進骨骼再生。傳統的骨修復材料涉及將間充質幹細胞直接轉化為成骨細胞，然後通過膜內骨化過程（IMO）促進骨形成。儘管該方法在體外具有巨大的潛力，但由於血管生成不足和灌注能力有限，IMO經常出現血管壞死等問題，從而導致成骨失敗。當前，大多數骨骼可通過替代性軟骨內骨化（ECO）形成。在ECO過程中，軟骨對營養和氧氣的需求較低，隨著軟骨細胞的成熟，在後期可為成骨和血管生成提供多種細胞因子。近年來，這種有效的骨再生方法引起了人們極大的興趣，並且許多研究集中在通過將生物材料與細胞結合，用於體外構建軟骨，或用於體內植入來誘導ECO。

華東理工大學劉昌勝院士、袁媛教授等人開發了一種可注射的聚（癸二酸甘油酯）-共聚（乙二醇）/聚丙烯酸（PEGS / PAA）水凝膠，其可誘導低氧環境，並通過原位鐵螯合重塑ECO進行骨再生。隨著PAA的加入，該水凝膠在注射後呈現出顯著的粘彈性和與鐵離子螯合的高效率，從而激活了HIF-1α並抑制了炎症反應，從而改善了軟骨的早期形成，並在後期促進了血管生成。更重要的是，通過PEGS / PAA水凝膠在整個再生過程中持續穩定地激活HIF-1α，可以實現軟骨/成骨的平衡，從而促進ECO的發展。這些發現提供了一種有效的策略，可通過基於生物材料的鐵離子螯合和隨後的低氧環境來實現原位ECO，這為未來在骨再生中的應用提供了新的思路和概念。該研究以題為“Recapitulation of In Situ Endochondral Ossification Using an Injectable Hypoxia‐Mimetic Hydrogel”的論文發表在《Advanced Functional Materials》上。

【可注射的PEGS水凝膠】

在製備PEGS / PAA水凝膠之前，作者首先優化了PEGS水凝膠作為基質的方案。作者使用來自PEGS預聚物的典型硫醇-Michael加成點擊反應制備PEGS水凝膠。考慮到最終可注射的PEGS水凝膠的水溶性，作者選擇了30%PEG含量的預聚物，其表現出最好的親水性和更高的羥基利用率，以確保水凝膠的可注射性和膠凝性。當含量等於或超過40％時，羥基含量將太低而無法形成可用的雙鍵，導致膠凝時間過長。PEGS30預聚物的SEM成像顯示，PEGS水凝膠有相互連接的大孔結構，並且可以通過改變硼砂濃度來調整微孔的尺寸。具體來說，將硼砂濃度從1：4增加到1：2，微孔的大小能夠從大約6減小到2 µm（圖1f）。PEGS水凝膠的體外降解性能表明，隨著硼砂濃度的增加，降解速度變慢（圖1d）。因此，考慮到降解和凝膠特性，作者最終選擇了PEGS30 1：3水凝膠進行後續研究（圖1e）。

圖1 PEGS水凝膠的表徵

【誘導缺氧環境】

HIF-1ɑ是細胞適應過程中低O2感測的關鍵。但HIF-1ɑ很容易在氧和亞鐵離子的存在下被脯氨酰羥化酶羥化，然後迅速被蛋白酶體降解。也就是說，低氧水平或消除鐵離子可以有效提高HIF-1ɑ的穩定性。然而，當前仍然缺乏一種可以完全螯合鐵離子而不添加缺氧劑的純生物材料方法，更不用說模擬缺氧環境來重塑和改善ECO，從而實現有效的骨再生。在這項工作中，PEGS / 30PAA在水凝膠中具有更多的可用羧基，並表現出較強的鐵離子螯合活性（圖2A）。在培養3天和7天后，在PEGS / PAA組的中觀察到了較高的HIF-1α蛋白（圖2F，H）。術後1周從股骨缺損處去除水凝膠後，作者發現PEGS / PAA水凝膠中鐵離子的吸附更加強烈（圖2B）。相應地，在骨基質和周圍PEGS / PAA水凝膠中的細胞中均觀察到了強烈的HIF-1α染色（圖2I），這表明該水凝膠在體內模擬出了低氧微環境。

圖2 PEGS / PAA水凝膠誘導的低氧微環境

【促進血管生成和重塑ECO】

作者探索了PEGS / PAA對體內血管形成的影響，結果如圖3所示。與PEGS組相比，PEGS / PAA組中CD31的表達增加了，這表明有更多的血管再生（圖3A，C）。在4周和8周後，PEGS / PAA組的血管生長量明顯大於PEGS組。此外，PEGS / 30PAA組表現出最大的血管生長量（圖3B）。接下來，作者探索了PEGS和PEGA / PAA水凝膠誘導的體內骨骼再生。在PEGS水凝膠組中，新骨形成低於PEGS / 10PAA或PEGS / 30PAA組，並且僅在材料的外圍形成少量新骨組織。相反，在PEGS / PAA水凝膠組中能觀察到明顯的骨形成。在第8周時，接受PEGS水凝膠的股骨比第4周時顯示出更多的礦化，但仍有一些缺陷。而PEGS / PAA水凝膠組的骨缺損在第8周時就已完全修復，並在PEGS / 30PAA組的骨形成最好。這些結果證實，PEGS / 30PAA可以有效促進骨骼再生。

圖3 PEGS / PAA水凝膠促進血管和骨生長

總結：作者介紹了一種新型的PEGS / PAA水凝膠，該水凝膠可通過低氧微環境的原位模擬來加速基於ECO的骨再生。具體而言，具有更充分的HIF-1ɑ穩定性的PEGS / 30PAA具有最佳的體內骨再生效率。這項研究提供了有力的證據表明，可以使用原位螯合鐵離子來誘導和調節HIF-1ɑ。最重要的是，這項研究中HIF-1ɑ的持續和穩定激活對於在ECO中實現軟骨/成骨平衡並從而達到有效的骨修復至關重要。作者的發現對於理解生物材料引起的生物學效應以及協助新型骨再生的發展也具有重要意義。
