美國航母艦載機編制大調整，擴編F-35C，首次列編無人加油機成亮點

2020-10-29
| 科學

美國海軍作戰部長（CNO）辦公室負責航空作戰的總監格雷戈裡·哈里斯少將9月11日對外發布了未來艦載機聯隊（CVW）的初步編成構想，該文件將提交專業人士進行研討。

該文件顯示，美國海軍未來艦載機聯隊中，戰鬥/攻擊機的數量仍將保持44架不變，但其構成將發生改變，原定的10架F-35C Block 4將增至16架，編為1箇中隊；原定的34架F/A-18E/F Block III將削減至28架，編為3箇中隊。剩餘艦載機包括：5~7架EA-18G電子戰飛機、5架E-2D預警機、6~10架MH-60直升機、3架CMV-22B傾轉旋翼機、5~9架MQ-25A無人加油機。

要說美國海軍這次航母艦載機編制調整的看點，主要有兩個：一個是F-35C第五代戰鬥機擴編，另一個則是航母歷史上首次列編無人加油機。那麼，美軍為什麼要對現有的航母編制“動這兩刀”呢？

增五代機、減四代半，意在提升航空作戰能力

美軍計劃中的航母艦載機編制調整，將F-35C增加到16架，“四代半”的F/A-18E/F Block III則相應削減6架，這是為什麼？

F/A-18E/F Block III是波音公司在“超級大黃蜂” F/A-18E/F基礎上升級改進的最新版本，大幅強化了其信息化作戰能力，且具備了一定的隱身能力，堪稱目前現役“四代半”戰鬥機中的翹楚。它與F-35C分工十分明確，F/A-18E/F Block III主要充當“攻擊機”角色，執行對地/對海的縱深攻擊任務，而F-35C則依靠其強大的隱身性能和綜合作戰能力，擔任防空/制空作戰的主力。通俗點說，F-35C負責迎擊和對抗其他空中來襲戰鬥機，執行制空作戰任務，而F/A-18E/F Block III則在其掩護下，執行對面攻擊任務。

F-35C為什麼增加？這是因為據美國海軍評估，因為潛在競爭對手開始大量研製、生產和列裝第五代戰鬥機。此前美國航母面對的主要是裝備四代機的對手，無論負責防空作戰的F-35C還是縱深打擊的F/A-18E/F Block III，都具備一定的對抗優勢。而在五代機面前，F/A-18E/F Block III則處於明顯劣勢，航母必須裝備更多的F-35C才能保障自身平臺的安全。

列編無人加油機，解放“超級大黃蜂”

而列編MQ-25A“黃貂魚”這種世界上現役的第一種艦載無人加油機，則是出於提升空中加油能力和解放“超級大黃蜂”戰鬥機的考慮。

在美軍以“大黃蜂”和“超級大黃蜂”作為航母艦載戰鬥機主力的時代，為了延長艦載機滯空任務時間，保證其有效航程，大量的艦載戰鬥機被分配了作戰之外的一項重要任務——“夥伴加油”，據美國海軍統計，在實戰和演習中，有至少20~30%即四分之一左右的艦載戰鬥機架次，是掛載四個大型副油箱，飛的“夥伴加油”任務。這就意味著實際上只有四分之三的艦載戰鬥機投入了“作戰任務”。

在美國航母進入F-35C時代後，鑑於F-35C需要保持隱身能力、執行防空/制空作戰任務，不可能攜帶外掛副油箱執行夥伴加油任務，則佔據艦載戰鬥機總數三分之二的“超級大黃蜂”的夥伴加油任務相對而言比此前更為繁重——每次作戰得有將近一半的“超級大黃蜂”戰機用於執行夥伴加油任務，能夠執行縱深打擊任務的戰機數量銳減，相當於嚴重削弱了航母戰鬥群的打擊能力。

為了解放“超級大黃蜂”，MQ-25A“黃貂魚”應運而生，它採用隱身飛翼式佈局，內部裝載空間有很大優勢，載油量比起需要加掛副油箱的“超級大黃蜂”只多不少，在航母上列編5~9架MQ-25A，足可以完成此前需要10多架“超級大黃蜂”才能完成的空中加油任務，可以將“超級大黃蜂”從繁重的夥伴加油任務中解放出來，將寶貴的飛行架次用於執行縱深打擊任務。這將有效提升航母的對面打擊能力。同時，還能節省艦載戰鬥機飛行員的體力和精力。

除此之外，MQ-25A實際上還擔負了另一個重要的作戰任務——偵察監視。MQ-25A採用飛翼式佈局，其隱身性能可能還在F-35C之上，它攜帶探測偵察傳感器，充當航母戰鬥群的耳目，為F-35C和“超級大黃蜂”提供情報信息和目標指示。這種多用途能力，肯定備受航母編隊指揮官的歡迎。

結語

美軍對於航母艦載機編制的調整，表明艦載機隱身化、逐步五代機化，是裝備發展的方向，美軍的這一動向，值得我們借鑑和思考，對於將來我們如何為自己的003新型航母規劃編制科學實用的艦載機組成方案，有著非常重要的意義。值得注意的是，美軍擴編和列編的這兩種新機，恰恰都是我軍目前艦載機編隊所欠缺的機種。我們是否應該像美軍那樣積極研製五代艦載機和艦載無人加油機？這無疑值得我們去思索。
