什麼是“天下第一雄關”完整的軍事防禦體系？

2020-10-29
| 科學

在匈奴語中，“嘉峪”意為“美好的峽谷”。嘉峪關因嘉峪山而得名，從漢代的“玉石障”，五代的“天門關”，到元代正式更名為“嘉峪山”，取“林泉秀美，澗壑寂寥”之意。

從1539年初建到築成一座完整的長城關隘，嘉峪關等待了168年。嘉峪關關城及其兩翼長城和沿線的烽燧墩堡修建完成後，便形成了南起祁連山、北銜黑山峭壁，長15公里的明牆暗壁。關城依山傍水，南部的討賴河谷構成關防的天然屏障。於是，烽燧、墩臺縱橫交錯，攻防兼備，構成了這道完整嚴密的軍事防禦體系，被世人譽為“天下第一雄關”。

航拍關城日出 攝影 / XIAO

這一次我們不再走馬觀花，如普通遊客那樣匆忙，選擇嘉峪關關城-長城第一墩-懸壁長城一日遊，而是連續三天天未亮就出發，分別在這三個極具代表性的地方等待日出，然後或是細細漫步關城內外，或是在懸壁長城停留感受，或是靜下心來聆聽博物館講解員的追本溯源。偏偏天公作美，讓這幾個清晨萬里無雲，我們置身其中，與這“天下雄關”、“連陲鎖陰”一同從黑夜中甦醒，於天地、自然的一呼一吸間感受嘉峪關的無窮魅力。

第一部分：嘉峪關關城

東有山海關，中有鎮北臺，西有嘉峪關。

自西周起，延續不斷2000多年的修築，50000多千米的長度，讓長城成為了中國乃至全世界修建時間最長、工程量最大的古代防禦工程。而萬里長城沿線上分佈著十三個關隘，其中規模最大的有兩座：一座是東端的山海關，另一座便是西端的嘉峪關，後者較前者猶有過之，所以嘉峪關算是長城上最大的關隘。

明長城示意圖 圖片來自網絡

相對地理位置 高德地圖截圖

嘉峪關距今有631年的歷史，比山海關早建9年。

明徵西大將軍馮勝收復河西后，選址於此建關。這兒地勢險峻，是嘉峪關西5公里處最狹窄的山谷，城關兩側的城牆橫穿沙漠戈壁，東連酒泉、西接玉門、北連黑山懸壁長城、南接天下第一墩。作為明長城最西端的關口，“河西咽喉”可不是浪得虛名。

它不僅是明代長城沿線建造規模最為壯觀，保存程度最為完好的一座古代軍事城堡，還是古代"絲綢之路"的交通要塞，素有中國長城三大奇觀之一的美稱。

整個嘉峪關體現出一種莊嚴神聖的規則感。它由內城、外城、城壕三道防線形成重疊並守之勢，壁壘森嚴，五里一燧，十里一墩，三十里一堡，一百里一城。

關城則是以內城為主，全長640米，佔地2.5萬平方米，城高10.7米。內城開東西兩門，東為“光化門”，意為紫氣東昇，光華普照；西為“柔遠門”，意為以懷柔而致遠，安定西陲。門臺上建有三層歇山頂式建築。東西門各有一甕城圍護，西門外有一羅城，與外城南北牆相連，有“嘉峪關”門通往關外，上建嘉峪關樓。嘉峪關內城牆上還建有箭樓、敵樓、角樓、閣樓、閘門樓共十四座，關城內建有遊擊將軍府、井亭、文昌閣，東門外建有關帝廟、牌樓、戲樓等。

公元1495年，嘉峪關樓重新修建。清同治末年，左宗棠駐節肅州之時，又曾修整關牆和關樓，並親筆題下“天下第一雄關”的匾額，高懸關樓。

在嘉峪關還流傳著這樣一個與古代工匠有關的故事：明朝修嘉峪關時，主管官員給工程主管人出難題，要求他預算用材必須準確無誤。在工匠們的幫助下，工程主管人進行了精確的計算。結果工程竣工時，所備的磚瓦木石恰恰用完，只剩下一塊城磚，稱為“最後一塊磚”，這塊磚目前仍存放在會極門(西甕城門)門樓簷臺上。

關城內還有一個建得很高卻沒有樓梯的戲臺。關於建得高有一種說法是：由於當時戲子的地位很低，這麼做是為了不讓人們看到戲子的腳。其實，在建好之初，在兩側是有樓梯的，只是歲月變遷，我們現在能夠看到的，只有一座高高在上的戲臺。戲臺的後臺是封閉的，戲子從樓梯走上去演戲，演完戲之後，去後臺換服裝。據傳說，在嘉峪山下，無警時，西域來人和出關使者常常在此聚會、歌舞，並有“西域來人漢家使，嘉峪山前舞月圓”的詩句。

嘉峪關關城 攝影 / XIAO

關城外的蘆葦蕩 攝影 / XIAO

關城的日出是讓人難忘的，帶著入秋的涼意，陽光透過自成一線的三座城樓的窗櫺傾瀉下來，而後天空變得洗練，直到遠方的祁連雪山也清晰可見，登上城牆，塞外風光盡收眼底，城外的一片蘆葦蕩更是這金戈鐵馬中的柔情。

到了黃昏的時候，坐東朝西的嘉峪關城樓被落日的餘暉鑲上一層金邊，過往的遊客在駝鈴陣陣中，慢慢走向遠方的天際線。

關城黃昏 攝影 / XIAO

第二部分：北有懸壁長城

“萬里長城萬里關，迭障黑山暗壁懸”，向北約8公里，是緊靠黑山的懸壁長城。因城牆自山上陡跌而下，在山脊上似長城倒掛，鐵壁懸空，封鎖了石關峽口，故稱“懸壁長城”。

日出前的懸壁長城像是一條等待甦醒的長龍，我們拾級而上，向東南望去便可見整個嘉峪關市。

懸壁長城 攝影 / XIAO

懸壁長城 攝影 / XIAO

天色漸漸明朗，可見懸壁長城上的牆體，出乎意料的非常完整。

實際上現存的750米長城是經1987年重修過的，其中有231米城牆懸掛於高150米，傾斜度為45度的山脊上，片石、土層厚度如舊。在牆頭加築“垛牆”和“宇牆”，首尾各加築一墩臺。而原來的牆體現只剩下短短一截，底下4米，上寬2米，高0.5－6米不等。

如今，每逢冬季雪後，都可到懸壁長城下的滑雪場滑雪。

第三部分：南有長城第一墩

“清流討賴臨危壁，大嶺祁連斷戰氛。鎧甲冰河聞拆鬥，旌旗旭日舞鵬鯤”。關城向南7.5公里便是長城第一墩了。顧名思義，它是明代萬里長城的第一座墩臺（從西向東），是明代長城的西端起點，也是嘉峪關長城防禦體系的重要組成部分。

墩臺位於討賴河邊近80米高的懸崖之上。

討賴河，自冰溝口出祁連山，從西南往東北流經茫茫戈壁，在漫長的地質歷史中慢慢下切，形成規模宏大的戈壁峽谷奇觀，窄處不足一丈，寬處百丈有餘，平均谷深60米以上，堪稱“戈壁高峽”。

長城第一墩 攝影 / XIAO

玻璃觀景臺 攝影 / XIAO

長城第一墩博物館 攝影 / XIAO

那時的長城第一墩，擔負著傳遞關南及祁連山諸口軍事信息的任務，設有塢、驛馬、驛駝、食宿、積薪等物，配備有一定數量的守兵。如今的長城第一墩有綜合服務區、討賴河滑索、討賴客棧、天險吊橋、“醉臥沙場”雕塑群、“中華龍林”等符合歷史角色的特色景緻。

軍營 攝影 / XIAO

鐵索橋 攝影 / XIAO

討賴客棧 攝影 / XIAO

初到關城時，還未被所見震撼，而到了屹立於討賴河大峽谷之上長城第一墩，寒風拂面，蒼涼感油然而生，想起那首著名的"黃河遠上白雲間，一片孤城萬仞山。羌笛何須怨楊柳，春風不度玉門關。"這玉門關外戈壁荒灘,連春風都不願眷顧。

而後呢？清代左宗棠抬棺出征、收復新疆，並帶領他的軍隊到處種樹,種了幾十萬株,便有了“左公柳”，更有其得力大將楊昌浚的詩“大將籌邊尚未還,湖湘子弟滿天山。新栽楊柳三千里,引得春風度玉關。” 連春風都不願眷顧的戈壁荒漠，就這樣偏偏被楊昌浚的詞給引出了玉門關。

自北向南的懸壁長城、嘉峪關關城、長城第一墩作為不可或缺的隘口，和烽燧、墩臺、長城一起構成了嚴密的防禦體系，在萬里長城與絲綢之路的交匯處，發揮著關鍵作用，豐富了我們中華民族的文明。
