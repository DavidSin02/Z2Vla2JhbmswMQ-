寶寶也要拍X光片？有輻射怎麼辦？

2020-11-14
| 科學

其實，偶然一次X光檢查對嬰兒沒有害處。

說起輻射，不得不提心吊膽的。手機、電視的那丁點兒輻射都怕傷著寶寶，一去醫院，聽說要給寶寶拍X片，瞬間懵住了……

聽著好危險的樣子！事實又如何呢？

提問

寶寶得了肺炎，醫生讓拍X片檢查，聽說有輻射的，對寶寶會不會有害處？可以不做嗎？

回答

答：任何檢查都可能有負效應。

若只有通過這些檢查才能弄清孩子的問題，且不做檢查，存在的問題對孩子可能會有較大傷害，就必須接受檢查。

不要因擔心X光可能對嬰兒有損傷，而躲避。其實，偶然一次X光檢查對嬰兒沒有害處。

（詳細請參考《崔玉濤圖解家庭育兒10：小兒常見病》）

科學家庭育兒（ID：kexueyuer2012）翻閱《崔玉濤圖解家庭育兒【最新升級版】》全套書籍後，就寶爸媽們關心的X光檢查的相關問題進行了補充回答。

不要懼怕常規 X 光檢查

當孩子出現外傷、肺部感染等時，醫生會建議 X 光檢測。很多家長擔心 X光的放射損傷。

現在醫院內使用的都是數字 X 攝像機，放射線非常有限。常規X 光照相對孩子損傷極微。

1.對放射線的感性認識

一位外國專家曾經這麼形容一次 X 光照相可能帶來的損傷——一次常規 X光照相接受到的射線如同乘飛機從紐約到舊金山在空中接受到的射線。

雖然我們不願意讓孩子接受射線，但也不必恐慌偶爾的 X 線檢查。

2.需保護的部位

常規 X 光照相是針對性很強的操作。X 光照相時，人體重要部位一定要做好防輻射的保護，這是必須進行的保護。保護部位包括：頸部（保護甲狀腺）和下腹部（生殖器官部位）。任何醫院的放射科都有遮蓋的鉛衣，專業人員會幫助覆蓋。

3.關於X 線透視檢查

再有，不是特殊情況，不是非常必要，不建議接受 X 線透視檢查。因為每次 X 線透視檢查的 X 光量為一次 X 光片的數十至數百倍。

我認為：應該聽從醫生的建議，如果需要，應該讓嬰幼兒接受 X 光檢查。不要因小失大，畢竟醫生建議接受 X 光檢查，說明還是相當有必要的。關愛孩子不是躲避一切可能的損傷，而是躲避嚴重的損害。
