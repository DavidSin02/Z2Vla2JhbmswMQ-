VS code 編輯器（文件或者文件夾的右鍵打開菜單的顯示）

2020-10-29
| 科學

使用 VSCODE的過程中非常好奇， 為什麼沒有 在文件或者文件夾上右鍵可以使用 VSCODE打開的快捷鍵。

自己傻逼了， 安裝的時候粗心了， 因為如上圖默認是 不選中的， 所以就看不到了。

解決的方式 是 可以自己重新安裝， 然後選中選項。

或者使用下面的cmd 修改註冊表（不熟悉的不推薦啊）

vsCodeOpenFolder.reg 文件


  2. Windows Registry Editor Version 5.00
  4. ; Open files
  5. [HKEY_CLASSES_ROOT\*\shell\Open with VS Code]
  6. @="Edit with VS Code"
  7. "Icon"="C:\\Program Files (x86)\\Microsoft VS Code\\Code.exe,0"
  9. [HKEY_CLASSES_ROOT\*\shell\Open with VS Code\command]
  10. @="\"C:\\Program Files (x86)\\Microsoft VS Code\\Code.exe\" \"%1\""
  12. ; This will make it appear when you right click ON a folder
  13. ; The "Icon" line can be removed if you don't want the icon to appear
  15. [HKEY_CLASSES_ROOT\Directory\shell\vscode]
  16. @="Open Folder as VS Code Project"
  17. "Icon"="\"C:\\Program Files (x86)\\Microsoft VS Code\\Code.exe\",0"
  19. [HKEY_CLASSES_ROOT\Directory\shell\vscode\command]
  20. @="\"C:\\Program Files (x86)\\Microsoft VS Code\\Code.exe\" \"%1\""
  23. ; This will make it appear when you right click INSIDE a folder
  24. ; The "Icon" line can be removed if you don't want the icon to appear
  26. [HKEY_CLASSES_ROOT\Directory\Background\shell\vscode]
  27. @="Open Folder as VS Code Project"
  28. "Icon"="\"C:\\Program Files (x86)\\Microsoft VS Code\\Code.exe\",0"
  30. [HKEY_CLASSES_ROOT\Directory\Background\shell\vscode\command]
  31. @="\"C:\\Program Files (x86)\\Microsoft VS Code\\Code.exe\" \"%V\""

注意 VSCODE的安裝路徑這裡使用的是默認的 C:\\Program Files (x86)\\Microsoft VS Code\\ 你要換成你安裝的路徑才對。

然後雙擊這個文件就行了， 如果有問題就以管理員的身份運行。

然後就可以使用了。
