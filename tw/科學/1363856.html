二十四個魚類冷知識

2020-10-29
| 科學

第一線實踐養魚深知水產從業不易！

"水花魚"務實不誤導不誇大不炒作！

01魚想吃海膽和貝殼怎麼辦？一些隆頭魚類會利用石塊作為砧板，砸開貝類或海膽堅硬的外殼。

楔斑豬齒魚，隆頭魚科豬齒魚屬下的一個種，花了一個小時敲打貝殼。圖片：BBC America‏ / Twitter

02珊瑚礁區的頂級捕食者石斑魚在捕獵的時候，有時會邀請裸胸鱔一起合作捕獵。這樣做可以提高它們倆的捕食成功率。

石斑魚：雞？裸胸鱔：雞！圖片：edyong209 / Youtube

03裸胸鱔（Gymnothorax spp.）在捕食的時候，有時咬住獵物的同時會在自己尾部打一個結。當這個結迅速地傳到它們頭部的時候，裸胸鱔就可以藉助繩結帶來的壓力將獵物撕咬碎。這是…...打…...打…...打劫（結）？

像不像你家主子在玩毛球。圖片：TGI Maldives/ Youtube

04達摩鯊（Isistius sp.）被稱為小餅乾模具鯊魚（Cookie cutter shark）。這是因為它們會利用鋒利的牙齒在大魚或者鯨魚表面咬出一個圓形小餅乾的形狀。

場面過於血腥，請快速觀賞。圖片：Wikimedia Commons

05泥鰍（Misgurnus spp.）會在氣壓降低時變得躁動不安，因此也被稱為氣象魚（Weatherfishes）。

縱帶泥鰍。圖片：Harka, Akos / Wikimedia Commons

06鰻形細潛魚（Encheliophis gracilis）會鑽入海蔘的肛門中，並生活在海蔘體內，吃海蔘的性腺和呼吸樹。這還不是最過分的，過分的是，有時甚至可以在海蔘體內找到成對的鰻形細潛魚。

07翻車魨（Mola mola）在海面上晒太陽並不是因為它們抑鬱了，它們有時候就是想趴在海面當一條隨波逐流的鹹魚。

看什麼看？沒見過晒太陽啊。圖片：Richard Dolan / Two Oceans Aquarium

08金魚的記憶遠不止7秒。在實驗中如果它們被電，那麼在接下來至少一個月內它們都會記得躲避電流的方法，這記憶力可比很多人強。

金魚：我其實很記仇的，望周知。圖片：Wikimedia Commons

09茴魚（Thymallus grubii）聞起來有茴香的味道，而胡瓜魚（Osmerus eperlanus）聞起來有黃瓜的味道…...

10盲鰻可以在短時間內產生大量黏液，它們產生的粘液在幾分鐘之內就可以充滿一個8升的水桶，這些黏糊糊的粘液可以嚇退捕食者，保護它們不被捕食。

為了不先把自己黏住，打著結擺脫粘液的盲鰻。圖片：Lukas Böni et al. / Scientific Reports(2016)

11許多南北極生存的魚類（比如南極冰魚），體內都含有抗凍蛋白(AFPs)，它能吸附在冰晶表面限制冰晶生長，這樣一來在寒冷海水中的它們就不容易結冰了。

獨角雪冰魚（Chionodraco hamatus）：省下買羽絨服的錢了，美滋滋。圖片：Wikimedia Commons

12異鱗蛇鯖（Lepidocybium flavobrunneum）因為體內含有一種熔點高，且不溶於水的蠟質。對於消化不好的人來說，食用過多會上吐下瀉，屁股流油。

我看哪個敢吃我！圖片：NSW Fisheries, Eden / Australian Museum

13中秋之夜是中國人團圓的節日。而在海洋中，每到月圓之夜卻是很多動物大喜的日子。比如一些石斑魚，就會選擇在滿月的時候集群交配。

月黑風高的夜晚......你猜它們在幹什麼？眼帶石斑魚（Epinephelus striatus）。圖片：Doug Perrine Photography

14棘尾深海鰩（Bathyraja spinosissima）卵的孵化期長達1000多天，但它們會選擇把卵產在熱液口旁，利用深海熱液口的溫度加速卵的孵化。

看起來像粽子葉的那些就是它們的卵，棘尾深海鰩可太南啦！圖片：EVNautilus / Youtube

15體型較小而不受雌性青睞的雄性小眼長臂蝦虎（Pomatoschistus microps），會趁其它魚交配的時候偷偷釋放自己的精子，來保證自己的基因得以延續。

16小丑魚、石斑魚和隆頭魚等魚類擁有變換性別的能力。小丑魚通常就是由雄性轉變為雌性。而石斑魚類和隆頭魚類大多數情況是由雌性轉變為雄性。

有點眼熟？對，沒錯，《海底總動員》的主角尼莫就是小丑魚。一想到尼莫會變性就覺得毀童年丨Nick Hobgood / Wiki Media Commons

17虎鯊會產下螺旋形的卵鞘，但這些卵鞘在剛產下時是柔軟淡褐色的，過一段時間後顏色會逐漸變深且變得堅硬。

佛氏虎鯊（Heterodontus francisci）的卵鞘，螺旋形有助於鯊魚媽媽把卵藏在裂縫中。圖片：devra / Flickr

18一些花斑溪鱂（Kryptolebias marmoratus）可以自我繁殖。在花斑溪鱂的群體中大約有95%的個體是雌雄同體的，剩餘的為雄性。雌雄同體的花斑溪鱂可以自我繁殖，也可以和雄性交配產生後代。

19雌性窄頭雙髻鯊（Sphyrna tiburo）可以不依賴雄性進行而產下後代，這種現象叫做孤雌生殖。

世界上第一次被發現會吃素的鯊魚也是它。鯊魚屆緊急召開新聞發佈會，表示不認識它。圖片：sciencemag.org

而另外一些魚類，比如秀美花鱂（Poecilia formosa），它們的卵受精雖然需要雄性，但雄性的精子只負責激活胚胎髮育過程，不提供任何遺傳物質，這種發育過程叫做雌核發育。反正在一些魚類裡，生孩子這事可以不需要男人。

20目前知道的壽命最短的魚是大印磯塘鱧（Eviota sigillata），它們最長的壽命記錄只有59天。在孵化後會首先度過三週的浮游期，緊接著的兩週內可以快速達到性成熟。在生命最後的三週半，雌魚會產下卵由雄魚照顧，直到雄魚生命的最後一刻 。

大印磯塘鱧：我真的還想再活五百年！圖片：Randall, J.E / Wikimedia Commons

21目前知道的壽命最長的魚（也是最長壽的脊椎動物）是小頭睡鯊（Somniosus microcephalus），它們的生長非常緩慢。科學家利用碳同位素測年法測定小頭睡鯊的年齡，其中年齡最大的一尾為392±120歲，也就是說它們的壽命至少可以達到272歲，最多可以達到512歲。如果按512計算，這條小頭睡鯊和《西遊記》的作者吳承恩是同一年出生的。

吳承恩啊？隱隱約約有聽說啦！圖片：oceanexplorer.noaa.gov

22世界上最大的魚類是鯨鯊，它們可以達到20 米長，大概六七層樓高，體重可以達到34噸；最大的硬骨魚類是翻車魨，體長可以達到3.3米，體重可以達到2噸；而最長的硬骨魚類是皇帶魚，可以達到11米長。名叫“地震魚”的它們雖然帶來不了地震，但絕對會給你震撼。

1996年在美國加州捕獲的皇帶魚，長7.0米。圖片：Wikimedia Commons

23世界上目前捕撈量最大的魚類是黃線狹鱈（Gadus chalcogrammus），2016年的捕撈量是347.6萬噸。我們常吃的深海鱈魚堡、包括蟹肉棒都是用它做的。

黃線狹鱈：我們即將失去夢想。圖片：Wikimedia Commons

24世界上目前養殖量最大的魚類是草魚（Ctenopharyngodon idella），2016年的養殖量是606.8萬噸，佔世界魚類養殖量的11%。這個知識點可以在吃烤魚、脆魚的時候拿出來擺。

別的不認識，草魚這個老夥計可太熟悉了。圖片：Peter Halasz / Wikimedia Commons
