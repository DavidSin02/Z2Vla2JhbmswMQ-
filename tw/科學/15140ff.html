最全的鍵盤各鍵及組合鍵功能說明，建議收藏

2020-10-29
| 科學

鍵盤是最常用的輸入設備，通過鍵盤可以將漢字、英文字母、數字、標點符號等輸入到計算機中，從而向計算機發出命令、輸入數據。那麼鍵盤分為幾個功能區？各鍵的功能是什麼呢？組合鍵又有什麼功能呢？下面就分別予以說明 (以win10 OS 、聯想臺式機鍵盤等為例)。

一、鍵盤分區

一般的標準鍵盤分為功能區、主鍵盤區、控制鍵區和數字鍵區。

二、鍵盤各鍵功能


1.功能鍵區

ESC：取消光標所在行

F1：系統默認幫助信息

F2：選中文件及文件夾重命名

F3：資源管理下搜索文件功能

F4：顯示"我的電腦（win xp）"或"計算機（win7）"或"此電腦(win10)"和"Windows 資源管理器"中的"地址"欄、瀏覽器地址欄

F5：刷新鍵

F6：在窗口或桌面上循環切換屏幕元素。

F7：DoS下專用功能

F8：電腦啟動，BIOS啟動項設置

F9：Excel 中計算公式

F10：激活菜單欄（當可用時）

F11：全屏模式

F12：瀏覽器下調出調試窗口、Word 另存文檔


2.控制鍵區

Num Lock指示燈：小鍵盤數字鎖定，亮起表示數字輸入狀態

Caps Lock指示燈：大小寫鎖定，亮起表示輸入大寫

Scoll Lock指示燈：滾動鎖定指示燈，亮起表示頁面滾動狀態

PriScSyRp：屏幕拷貝鍵（截全屏鍵）

Scoll Lock：滾動鎖定，在需要翻頁的時候，按下看看，有奇效

PauseBreak：暫停鍵

Insert：插入字符開關鍵

Home：行首鍵，光標移至行首位置

PageUp：向上翻頁鍵盤

Delele：字符刪除鍵，當前光標後面的字符刪除

End：行尾鍵，光標移至行尾位置

PageDn：向下翻頁鍵盤

四個方向鍵：上下左右移動光標

3.數字鍵區

NumLocK：鎖定數字小鍵盤

Del：字符刪除鍵，當前光標後面的字符刪除

Enter:回車鍵

Home：行首鍵，光標移至行首位置

End：行尾鍵，光標移至行尾位置

PgUp：向上翻頁鍵盤

PgDn：向下翻頁鍵盤

Lns: Insert：插入字符開關鍵

  1. 主鍵盤區



Tab：跳轉功能

Backspace （有的鍵盤為向左的箭頭）：回格鍵或刪除鍵

CapsLock：字母大小寫切換

Shift：按住Shift鍵，在選擇文件可以連續選擇

Ctrl：按住Ctrl鍵，在選擇文件可以不連續多個選擇

Alt：關閉當前的菜單

Enter：回車鍵，換行作用

Windows：單按顯示或隱藏開始菜單

空格鍵右側（Windows鍵與Ctrl鍵之間）頁面上一個斜箭頭那個鍵：同鼠標右鍵功能

三、組合鍵

1. Windows組合鍵

Windows + F：查找所有文件

Windows + Ctrl+ F：查找計算機

Windows+E：打開資源管理

Windows+R：打開運行對話框

Windows+D：顯示或隱藏桌面

Windows+M：最小化所有窗口

Windows+Shift+M:撤消最小化所有窗口

Windows+L：鎖屏（切換用戶）

Windows + P：選擇演示顯示模式

Windows+U：打開設置顯示窗口

Windows+ F1: 顯示"幫助"

Window+Ctrl+D：創建新的虛擬桌面

Window+Ctrl+F4：關閉當前虛擬桌面

Windows+Ctrl+左箭頭鍵：轉到左側的虛擬桌面

Windows+Ctrl+右箭頭鍵：轉到右側的虛擬桌面

Windows+左箭頭鍵：將當前窗口移到左側。

Windows+右箭頭鍵：將當前窗口移到右側。

Windows+向上箭頭鍵：將當前窗口移到到頂部

Windows+向下箭頭鍵：將當前窗口移到到底部

Windows+Tab：打開一個新的任務視圖界面，顯示此虛擬桌面上的所有當前窗口

Windows + pause break：顯示"系統屬性"對話框

Windows + F：打開"反饋中心"

Windows + K ：打開"連接"快速操作

Windows+ A： 新通知

Windows+Alt+D ：顯示和隱藏桌面上的日期和時間

Windows + X ：打開"快速鏈接"菜單

Windows+H ：麥克風

Windows+I：打開"Windows設置"窗口

Windows+O ：鎖定設備方向

Windows+S： 打開快速搜索

Windows+T： 在任務欄上循環切換應用

Windows + V：打開剪貼板

Windows + 逗號 (,): 臨時快速查看桌面

Windows + Shift + V: 以相反的順序循環切換通知

Windows + Z: 顯示在以全屏模式呈現的應用中可用的命令

Windows + Ctrl + F: 搜索電腦（如果你位於網絡中）

Windows + 數字: 打開桌面，並啟動固定到任務欄的位於該數字所表示位置的應用。如果該應用已在運行，則切換到該應用

Windows + Shift + 數字: 打開桌面，並啟動固定到任務欄的位於該數字所表示位置的應用的新實例

Windows + Ctrl + 數字:打開桌面，並切換到固定到任務欄的位於該數字所表示位置的應用的最後一個活動窗口

Windows + Alt + 數字: 打開桌面，並打開固定到任務欄的位於該數字所表示位置的應用的跳轉列表

Windows + Ctrl + Shift + 數字: 打開桌面，並以管理員身份打開位於任務欄上給定位置的應用的新實例

Windows + Home: 最小化除活動桌面窗口以外的所有窗口（在第二道筆劃時還原所有窗口）

Windows+ Shift + 向上鍵: 將桌面窗口拉伸到屏幕的頂部和底部

Windows+ Shift + 向下鍵: 在垂直方向上還原/最小化活動桌面窗口，同時保持寬度不變

Windows+ Shift + 向左鍵或向右鍵: 將桌面中的應用或窗口從一個監視器移動到另一個監視器

Windows + 空格鍵: 切換輸入語言和鍵盤佈局

Windows+ Ctrl + 空格鍵:更改為以前選擇的輸入

Windows + 加號 (+) 或減號 (-): 使用"放大鏡"放大或縮小

Windows + Esc :退出"放大鏡"

2. Ctrl L組合鍵

Ctrl+A：全選

Ctrl+B：整理 收藏夾

Ctrl+C：複製

Ctrl+D：刪除/添加收藏 (在Word中是修改選定字符格式,在IE及傲遊中是保存為收藏)

Ctrl+E： 搜索助理 在Word中是段落居中

Ctrl+F： 查找

Ctrl+ G： 在Word中是定位

Ctrl+H ：歷史記錄 在Word中是替換

Ctrl+I：收藏夾

Ctrl+J：IE7.0裡的源

Ctrl+ K 在Word中是插入鏈接

Ctrl+L＝Ctrl+O：打開

Ctrl+ M： 網頁搜索框中的確定（等於回車）在Word中是調整整段縮進

Ctrl + N：新窗口

Ctrl + P ：打印 在Word中是等於Ctrl+Shift+F12

Ctrl + R ：刷新 在Word中可使段落右對齊

Ctrl + S ：保存

Ctrl+TIE7以上打開新選項卡

Ctrl+U：在Word中是給所選內容加下劃線

Ctrl+V：粘貼

Ctrl+W：關閉當前窗口

Ctrl+X：剪切

Ctrl+Y：恢復上一步操作

Ctrl+Z：撤消上一步操作

Ctrl+ 向右方向鍵：將插入點移動到下一個單詞的起始處

Ctrl+ 向左方向鍵：將插入點移動到前一個單詞的起始處

Ctrl+ 向下方向鍵：將插入點移動到下一段落的起始處

Ctrl+ 向上方向鍵：將插入點移動到前一段落的起始處

Ctrl+ Shift +方向鍵：突出顯示一塊文本

Ctrl+ Esc ：顯示"開始"菜單

Ctrl+F4： 在允許同時打開多個文檔的程序中關閉當前文檔

Ctrl+Tab： 在選項卡之間向前移動。

Ctrl+Shift +Tab： 在選項卡之間向後移動

ctrl+鼠標左鍵： 可以臨時允許彈出窗口程序

Ctrl+Home ：當前窗口或文檔的最上端

Ctrl+End ：當前窗口或文檔的最下端

Ctrl + Shift + 左鍵單擊某程序圖標： 以管理員身份運行

3.Alt組合鍵

Alt+PrtScSysRq：截當前窗口屏幕

Alt+Enter：查看所選項目的屬性／切換全屏

Alt+F4 ：關閉當前項目或者關閉計算機

Alt+F8 ：在登錄屏幕上顯示密碼

Alt+空格鍵：為當前窗口打開控制菜單

Alt+Tab：在打開的應用之間切換

Alt+Esc：以項目打開的順序循環切換項目

Alt+帶下劃線的字母： 執行該字母對應的命令

Alt+Enter：顯示選定項的屬性

Alt+向左鍵：後退

Alt+向右鍵：前進

Alt+Page Up：向上移動一個屏幕

Alt+Page Down：向下移動一個屏幕

4. Shift組合鍵

Shift+ F10: 顯示選定項的快捷菜單

Shift+任意箭頭鍵: 在窗口中或桌面上選擇多個項目，或者在文檔中選擇文本

Shift + Delete: 無需先將選定項移動到"回收站"，直接將其刪除

Shift + Tab :在選項上向後移動

Shift+左箭頭：選擇光標左側的文本

Shift+右箭頭：選擇光標右側的文本

Shift+單擊某個任務欄按鈕 :打開某個應用，或快速打開應用的另一個實例

Shift+右鍵單擊某個任務欄按鈕: 顯示該應用的窗口菜單

Shift+右鍵單擊某個已分組的任務欄按鈕: 顯示該組的窗口菜單

Shift +Ctrl+ +Esc：打開任務管理器

shift+crtl：windows系統默認的輸入法切換，這也是最常用的shift鍵組合

Shift+空格：輸入法中半全角切換

Shift+右擊：右菜單打開方式

Shift+多級文件：關閉這些文件

Shift+印有兩層符號的鍵：輸入上面這層符號，不按shift輸入的就是下面這層符號
