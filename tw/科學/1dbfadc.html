功能性消化不良臨床常用中成藥及注意事項，來了解一下

2020-10-29
| 科學

功能性消化不良是指具有上腹脹、上腹痛、早飽、噯氣、食慾不振、噁心等上腹不適症狀，經檢查在排除引起這此症狀的器質性疾病的一組臨床綜 合徵。功能性消化不良是臨床上最常見的一種功能性胃腸病，同時多伴有全身性精神症狀，如失眠、焦慮、抑鬱、注意力渙散、健忘、神經過敏、頭痛等。

一.香砂六君丸

【藥物組成】木香、砂仁、黨蔘、炒白朮、茯苓、炙甘草、陳皮、姜半。夏輔料:生薑、大棗

【臨床應用】 本品適用於功能性消化不良，見胃部脹悶不舒甚或隱隱作痛，嘔吐酸水，食慾減退，四肢倦息，舌胖大舌邊有齒痕，苔膩的患者。

【注意事項】

1.不適用於口乾、舌少津、大便幹者。

2.嘔葉伴胃部灼熱疼痛，口乾，大便幹者忌用。

3.忌食生冷油膩、不易消化及刺激性食物，宜食清淡易消化之品，戒菸酒。

4.對本品過敏者禁用，過敏體質者慎用。

【用法用量】口服，一次6-9克，1日2-3 次。

二、香砂枳術丸

【藥物組成】木香、砂仁、白朮(麩炒)、枳實(麩炒)

【臨床應用】

1.本品適用於功能性消化不良，見腹部脹滿不舒，喜噯氣，噯氣後脹滿減輕，食慾欠佳，食少，食後不舒，大便稀或不成形，舌淡的患者。

2.若胃部冷痛，喜溫喜按，可與理中丸同服。

3.若患者平時情志不暢，生氣或鬱悶時易於發病，可與逍遙散同服。

【注意事項】

1.若便祕口苦、舌紅苔黃脈滑者，不宜服用本品。

2本品方中有破氣之枳實，孕婦慎用。

3.便祕患者慎用。

4.服藥期間忌食生冷、油膩食物，宜食清淡易消化之品。

5.舌紅無苔、口乾咽燥，手足心熱，心煩失眠等患者忌服。

【用法用量】口服，一次10克，1日2次。

三、胃蘇顆粒(沖劑)

【藥物組成】香附、陳皮、枳殼、佛手、香櫞、雞內金（制）、紫蘇梗、檳榔

【臨床應用】

1.本品適用於功能性消化不良，見胃部脹痛，連及兩脅，痛處不固定，喜噯氣，噯氣或排氣後脹痛減輕，生氣或情緒激動時脹痛明顯或加重，胸悶不舒，食少，食後不舒，排便不暢，舌淡紅苔薄白。

2.若反酸燒心明顯者，可合用左金丸。

3.若胃痛甚，或胃部刺痛、舌質瘀斑者，可合用玄胡止痛片或失笑散。

4.消化性潰瘍患者，可合用西藥質子泵抑制劑。

【注意事項】

1.孕婦慎用。

2.服藥期間情緒穩定，切忌惱怒、抑鬱。

3.服藥期間偶有口乾、嘈雜，不影響服藥。

4.服藥期間，宜選清淡易消化之品。

5.忌食辛辣油膩及刺激性食品，戒菸酒。

6.肝胃鬱熱或胃陰虧虛患者慎用本品，前者主要表現為胃部灼熱脹痛，生氣或情緒激動時疼痛明顯或加重，噯氣或排氣後減輕，口苦口乾;後者主要表現為胃部灼熱隱痛，似有飢餓感，口渴喜飲，手足心熱，心煩，或口腔潰瘍易發，舌紅少津。

【用法用量】口服，一次15克，1日3次，15天一療程，可服1-3個療程。

四、金佛止痛丸

【藥物組成】白芍、延胡索、三七、鬱金、佛手、薑黃、甘草

【臨床應用】

  1. .本品適用於功能性消化不良、消化道潰瘍，慢性胃炎，見胃部脹痛或痛如針刺，痛無定處，連及兩脅，拒按，生氣或情緒激動時脹痛加重，喜噯氣，噯氣或排氣後脹痛減輕，舌苔薄白的患者。
  2. 若反酸燒心明顯者，可合用左金丸。3.若情緒不住誘發者，可合用道道散
  3. 若胃部冷痛，得溫痛減者，可合用理中丸或虛寒胃痛顆粒。
  4. 消化性潰瘍患者，可合用西藥質子泵抑制劑。

【注意事項】

  1. 孕婦及婦女月經過多者禁用。
  2. 胃陰虛者表現為脣燥口乾、喜飲、大便乾結，不宜服用本品。
  3. 不宜在服藥期間同時服用滋補性中藥。
  4. 忌食辛辣、生冷、油膩食物。

【用法用量】 口服，一次5-10克，1日2-3次。

五、中滿分消丸

【藥物組成】厚朴(姜炙)、枳實、薑黃、黃芩、黃連、半夏(制)、知母、豬苓、茯苓、白 術(麩炒)、澤瀉、陳皮、砂仁、黨蔘、甘草

【臨床應用】

  1. 本品適用於功能性消化不良，見脘腹脹痛灼熱，煩熱，口乾口苦，口渴不喜飲，食少腹脹，納差氣短，小便短色黃，大便祕結或稀而不爽，臭穢，舌邊尖紅，苔黃膩。
  2. 對於食慾不佳，納差乏力，口苦口渴不喜飲，大便黏滯不爽較重者，選用本品療效突出。

【注意事項】

  1. 本品祛溼熱為主，不宜久用。
  2. 不宜在服藥期間同時服用滋補性中藥。
  3. 本藥內含破氣活血之品，孕婦、兒童慎用。
  4. 本品為中滿熱脹、二便不利而設，若寒溼困脾者不宜使用。
  5. 服藥期間飲食宜用清淡易消化之品，慎食辛辣肥膩之物。

【用法用量】口服，一次6克，1日2次。

六、3附子理中丸

【藥物組成】附子(制)、黨蔘、白朮(炒)、乾薑、甘草

【臨床應用】

  1. 本品適用於功能性消化不良，見胃部冷痛隱隱，喜溫喜按，遇寒後疼痛明顯或加重或洩瀉，嘔吐物清冷，身冷喜暖，四肢不溫，大便稀或瀉下未消化的食物，口淡不渴，面色白，舌淡的患者。
  2. 若黎明前腸鳴欲洩者，可合用四神丸。
  3. 若腹部疼痛較重者，可合用虛寒胃痛顆粒、氣滯胃痛顆粒。
  4. 若嘔吐物為清水、腸鳴頻繁、小便不暢者，可合用五苓散。

【注意事項】

  1. . 感冒發熱病人不宜服用。
  2. 高血壓、心臟病、肝病、糖尿病、腎病等慢性病嚴重者應慎用。
  3. 孕婦、哺乳期婦女、兒童慎用。
  4. 不宜過量或長期服用。
  5. 對本品過敏者禁用，過敏體質者慎用。
  6. 不宜在服藥期間同時服用性味苦寒的中藥，
  7. . 服藥期間宜食用易消化食物，忌生冷油膩之品。

【用法用量】口服，水蜜丸:一次6克;大蜜丸:一次1丸，1日2-3次。

七、香砂養胃丸(顆粒)

【藥物組成】白朮、木香、砂仁、豆蔻(去殼)、廣霍香、陳皮、姜厚朴、醋香附、茯苓、枳實(炒)、姜半夏、甘草

【臨床應用】本品適用於功能性消化不良，見胃部脹悶不舒甚或隱隱作痛，嘔吐酸水，胃部燒灼感，食慾減退、四肢倦怠。舌胖大舌邊有齒痕，苔薄膩的患者。

【注意事項】I

  1. 如果出現胃部灼熱，隱隱作痛，口千舌燥等熱症表現的患者不宜服用本藥。
  2. 不宜長期連續服用本藥。
  3. 忌生冷油膩食物。
  4. 本品宜用溫開水送服。
  5. 對本品過敏者禁用，過敏體質者慎用。

【用法用量】丸劑:口服，大蜜丸一次9克，1日2次，濃縮丸一次8丸，1日3次，水丸一次9克，1日2次;顆粒劑:開水沖服，一次1袋，1日2次。

八、六味木香膠囊(散)

【藥物組成】木香、豆蔻、蓽茇、石榴皮、鬧羊花、梔子

【臨床應用】

  1. 本品適用於功能性消化不良，見胃部堵悶脹滿疼痛，或腹脹腹痛，痛無定處，生氣或情緒激動時脹痛明顯或加重，喜噯氣，噁心欲吐，噯氣或排氣後脹痛減輕，情緒鬱悶，舌邊尖紅的患者。
  2. 本品用於中焦氣滯、寒熱錯雜之胃痛效果最佳，主要表現為脘腹脹滿，痛處不定，食後腹脹，口渴喜飲，或口中異味，大便稀或不成形，或大便中有未消化的食物，舌淡苔薄黃。
  3. 若大便不通或便祕者，可合用六味安消膠囊。
  4. 若大便稀或不成形、舌苔厚者，可合用楓蓼腸胃康顆粒。
  5. 若反酸燒心，胃部似有飢餓感者，可合用健胃愈瘍片。

【注意事項】

  1. 按照規定的用法用量服用，不宜長期服用。
  2. 本品藥性偏於溫燥，胃陰不足或溼熱中阻所致痞滿、胃痛者慎用。
  3. 孕婦禁用，兒童、體虛者慎用。
  4. 本品含鬧羊花(有毒)，不宜長期或過量服用。一旦出現中毒表現，可按綠豆解毒。
  5. 宜少量多餐，飲食清淡，忌食生冷油膩、辛辣刺激及不易消化食物。

【用法用量】 膠囊劑:口服，一次4-6粒， I日1-2次; 散劑:口服，一次2-3克，1日1-2次。

九、荊花胃康膠丸

【藥物組成】土荊芥、水團花

【臨床應用】

  1. 本品適用於功能性消化不良，見胃部脹滿疼痛，噯氣或排氣後脹痛減輕，能食，食後不舒，反酸噯氣，胃中似有飢餓感，口苦，口渴喜熱飲，舌淡紅苔薄黃的患者。
  2. 本品對於上述症狀兼有形體肥胖，胸膈堵悶，常咳嗽咳痰，痰多色白者療效較佳。
  3. 若症見口乾口苦、脅脹脅痛而兼氣急易怒，口渴喜涼飲者，可合併應用丹梔逍遙散。
  4. 抗幽門螺旋桿菌治療時，聯合應用荊花胃康膠丸可增強殺菌作用。

【注意事項】

  1. 過敏體質及對本品過敏者不宜服用。
  2. 孕婦忌服。
  3. 飲食宜清淡，忌菸、酒及辛辣、生冷、油膩食物。
  4. 忌情緒激動及生悶氣。

【用法用量】口服，一次2粒，1日3次，4周為一療程。

按症狀分類使用:

  1. 上腹脹痛伴平時納差，面色發白往往用香砂積術丸;
  2. 以上腹部冷脹、痛為主，喜暖、喜按者用附子理中丸;
  3. 上腹脹或痛，伴舌苔黃厚膩，大便腐臭用中滿分消丸;
  4. 胃痛隱隱、嘔吐酸水用香砂養胃丸;
  5. 上腹脹痛、吞酸嘈雜往往用六味木香顆粒;
