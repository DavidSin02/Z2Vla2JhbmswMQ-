特殊路基施工技術02

7.剛性樁（預製樁和灌注樁）

剛性樁適用於處理深厚軟土地基上荷載較大、變形要求較嚴格的高路堤段、橋頭或通道與路堤銜接段。

8.爆炸擠淤爆炸擠遊法適用於處理海灣灘塗等淤泥和淤泥質土地基。處理厚度不宜大於15m。爆炸施工前應進行現場勘察及爆炸安全區的安全檢查，從事爆破工作的施工單位應取得當地公安部門核發的爆破作業許可證，從事爆破工作的人員應持證上崗（人員管理）。

起爆前必須在起爆點外佈設警戒線，警戒距離不得小於300m

9.路堤地基隔離牆

路堤地基隔離牆適用於相鄰兩路堤之間，或已建成路堤與拓寬路堤之間出現相互干擾，對地基滲流、變形、穩定等產生不利影響情況下的隔離。

隔離牆按其作用與功能可分為防滲型隔離牆和支擋型隔離牆。相鄰路堤，當待建的路堤採用降水預壓、真空預壓、強化固結等地基處理方法，或採用深井降水等工程措施時，宜設置防滲型隔離牆；其他情況下隔離新老路基相互干擾宜設置支擋型隔離牆。

10. 強夯和強夯置換強夯法適用於處理碎石土、低飽和度的粉土與黏性土、雜填土和軟土等地基。強夯置換法適用於處理高飽和度的粉土與軟塑、流塑的軟黏土地基，處理深度不宜大於7m

強夯處理範圍應超出路堤坡腳，每邊超出坡腳的寬度不宜小於3m。強夯置換處理範圍應為坡腳外增加一排置換樁。

採用強夯法處理軟土地基時,應在地基中設置豎向排水體。對於地下水位較高的地基,強夯前應採取降水措施,將地下水位降至加固層深度以下。強夯置換樁頂應鋪設一層厚度不小於0.5m的粒料墊層,墊層材料可與樁體材料相同,粒徑不宜大於100mm。

強夯和強夯置換應進行試夯，每個試夯區場地面積不小於500m2，試夯確定單擊夯擊能、夯擊次數、夯擊遍數、間歇時間等。

1)強夯施工規定

(1)強夯前應在地表鋪設一定厚度的墊層，墊層材料宜與樁體材料相同。

(2)強夯宜分為主夯、副夯、滿夯三遍實施。第一遍主夯完成後，第二遍的副夯點應在主夯點中間穿插佈置；副夯點與主夯點的佈置間距及夯擊能級應相同。滿夯夯點應採用彼此搭接1/4連續夯擊，滿夯能級可採用主夯能級的1/3～1/2。

（3）兩遍夯擊之間應有一定的時間間隔，間隔時間應根據土中超靜孔隙水壓力的消散時間確定；當缺少實測資料時，可根據地基土的滲透性確定。對於滲透性較差的黏性土地基，間歇時間不應少於21d;對於粉性土地基，間歇時間不應少於7d;對於滲透性好的地基，間歇時間不宜少於3d。

(4 )強夯夯點的夯擊次數,應按試夯得到的夯擊次數和夯沉量關係曲線確定,並應滿足下列要求：

①當單擊夯擊能小於2000kN·m時,最後兩擊的平均夯沉量不宜大於50mm;

當單擊夯擊能為2000-4000kN·m時,最後兩擊的平均夯沉量不宜大於100mm;

單擊夯能大於4000kN·m時,最後兩擊的平均夯沉量不宜大於200mm。

2強夯置換施工規定

(1)強夯置換前應在地表鋪設一定厚度的墊層，墊層材料宜與樁體材料相同。

(2)強夯置換夯點的夯擊次數應通過現場試夯確定，並滿足下列要求：

①置換樁底應達到設計置換深度（樁長度），宜穿透軟土層。

②累計夯沉量應為設計樁長的1.5～2.0倍。

③最後兩擊的平均夯沉量應滿足強夯施工相關規定的要求。

(3)強夯置換應按照由內向外、隔行跳打的方式施工。

11. 軟土地區路堤施工要求

( 1 ) 軟土地區路堤施工應儘早安排，施工計劃中應考慮地基所需固結時間。

( 2 ) 填築過程中，應嚴格控制填築速率，並應進行動態觀測。

( 3 ) 施工期間，路堤中心線地面沉降速率24h應不大於10 ~ 15mm，坡腳水平位移速率24h應不大於5mm。應結合沉降和位移觀測結果綜合分析地基穩定性。填築速率應以水平位移控制為主，超過標準應立即停止填築。

( 4 ) 橋臺 、涵洞 、通道以及加固工程應在預壓沉降完成後再進行施工。

( 5 ) 應按設計要求的預壓荷載、預壓時間進行預壓。堆載預壓的填料宜採用上路床填料，並分層填築壓實。

12. 舊路加寬軟基處理要求

( 1 ) 軟基路段路基加寬臺階應開挖一層、填築一層，上層臺階應在下層填築完成後再開挖，臺階開挖應滿足臺階寬度和新老路基處理設計要求。

( 4 ) 舊路加寬路段應同步進行拼寬路基和老路基的沉降觀測，觀測點宜佈置在同一斷面上。觀測點設置宜為老路路中、老路路肩、拼寬部分中部、拼寬部分外側。老路路中 、老路路肩沉降觀測點設置可採用在路表埋設觀測點的方法，拼寬部分宜採用埋設沉降板的方法。
