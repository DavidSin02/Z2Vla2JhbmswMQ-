2019 WCLC | 真實世界研究：這三類突變或可預測一二線免疫治療和靶向治療的較差獲益

2020-10-29
| 科學

由國際肺癌研究協會(IASLC)主辦的第20屆世界肺癌大會（World Conference on Lung Cancer，WCLC）於2019年9月7-10日在西班牙巴塞羅那召開。口頭報告中有兩項真實世界研究探索了某些突變與免疫治療或化療的應答相關性。詳情如下：

LKB1 / STK11突變和LKB1 / KRAS突變與一線免疫治療和化療的較差OS和PFS相關

背景

雖然今年在轉移性非小細胞肺癌（NSCLC）方面取得了進展，但很多患者對相關治療不應答。識別和鑑定遺傳性標記可以使患者的臨床獲益最大化，並避免不必要的毒性。LKB1 / STK11突變（LKB1m）和共同發生的KRAS突變/ LKB1缺失（KRAS / LKB1）與免疫治療的預後不良相關。這項回顧性研究評估了真實世界中LKB1m、KRAS / LKB1突變NSCLC患者接受免疫單藥或聯合治療的結果。

方法

研究納入2013年1月至2017年6月期間開始一線治療的成人患者，在常規治療中使用FoundationOne分析數據。根據組織學分型利用多變量Cox比例風險模型評估LKB1m、LKB1 / KRAS與一線和二線治療總生存（OS）或無進展生存（PFS）的相關性。分層因素為免疫治療或化療。

結果

研究納入2407例患者，其中1847例非鱗狀NSCLC患者，開始一線治療時的中位年齡為66.1歲。328例（13.6％）患者存在LKB1m，157例（6.5％）存在KRAS / LKB1。在接受二線免疫治療的患者中，與野生型患者相比，LKB1m患者的OS和PFS較差。在一線治療和非鱗狀NSCLC患者中也觀察到類似的相關性。在接受化療的患者中，LKB1m僅與一線治療時OS結果較差相關。與LKB1m患者相比，KRAS / LKB1突變患者這種相關性更明顯。

結論

不論是免疫治療還是化療，LKB1m和LKB1 / KRAS與一線治療的OS和PFS較差相關，並與二線免疫治療的OS和PFS較差相關。這項真實世界研究的結果支持之前的臨床研究結果，並提出了這些突變在一線化療和一線或二線免疫治療的獨特相關性。

EGFR/HER2 20外顯子插入突變對免疫治療和EGFR TKI應答較差

背景

EGFR/HER2的20外顯子框內插入突變（Ex20ins）發生在2%~5％的NSCLC患者中。對於這些突變的患者，目前沒有批准的靶向治療藥物。歷史對照數據對於開發針對這些罕見靶點的靶向療法有一定價值。

方法

日本已經建立了全國基因組篩選項目（LC-SCRUM-Japan），用來開發肺癌的靶向治療。在該項目中，通過第二代測序技術（NGS）分析161種癌症相關基因。使用該項目中的大規模臨床基因組數據庫評估EGFR / HER2 Ex20in的非鱗狀NSCLC患者的療效和生存率。

結果

研究共分析了3441例晚期非鱗狀NSCLC患者的數據。在73例患者中檢測到EGFR Ex20ins（2％; M766_A767insASV / A767_S768insSVD / H773_V774insH / D770_N771insNPH / N771_P772insPH /其他= 22/17/5/4/4/21）。在128例患者中檢測到HER2 Ex20ins（4％; A775_G776insYVMA / G776delinsVC / P780_Y781insGSP /其他= 95/16/10/7）。患者的中位年齡為62歲。81例（40％）患者為男性，114例（57％）患者從未吸菸。200例（99％）患者被診斷為腺癌，1例是腺樣鱗狀細胞癌。

基於數據庫的數據，EGFR Ex20ins患者的中位OS為22.4（95％CI，15.3~36.8）個月，而HER2 Ex20ins患者的中位OS為18.8（13.6~30.3）個月。在EGFR / HER2 Ex20ins患者中，一線含鉑化療的客觀緩解率（ORR）和中位PFS分別為32％和6.0（5.7~7.0）個月。多西他賽聯合或不聯合雷莫蘆單抗的ORR和中位PFS分別為26％和5.1（3.8~5.9）個月。PD-1抑制劑的ORR和中位PFS分別為0％和2.0（1.6~2.6）個月。在EGFR Ex20ins和HER2 Ex20ins患者中未觀察到這些藥物的療效的顯著差異。在用第一代/第二代EGFR-TKI治療的19例EGFR Ex20患者中，ORR為5％（M766_A767insASV陽性腫瘤對阿法替尼應答），中位PFS為2.1個月（1.3~4.2）。

結論

EGFR/HER2 Ex20ins陽性NSCLC對PD-1抑制劑和第一代、第二代EGFR TKI顯示出較差的應答。這些數據對於評估用於EGFR / HER2 Ex20ins陽性NSCLC的新型靶向療法的療效方面具有重要價值。
