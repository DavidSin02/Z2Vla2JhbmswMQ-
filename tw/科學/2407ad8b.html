玉米田莖葉處理除草劑的安全使用技術

2020-11-14
| 科學

1.防除禾本科雜草為主的除草劑的安全使用技術

煙嘧磺隆（玉農樂）是目前玉米田莖葉處理防除禾本科雜草為主的唯一的一種除草劑品種。除草效果好，安全性好，成本較高。

防治對象　一年生禾本科和若干闊葉雜草。

使用技術　玉米4葉期，稗草2～3葉期莖葉處理，每公頃用有效量40～50g。

注意事項　①不同玉米品種對藥劑的敏感性有差異，其安全性順序為馬齒型＞硬質型＞爆裂型＞甜型。甜玉米或爆裂玉米制種田、自交系對該劑敏感，勿用。②一般玉米2葉期前及10葉期以後，對該藥敏感。③用有機磷藥劑處理過的玉米對該藥劑敏感，兩藥劑的使用間隔期應為7d左右。煙嘧磺隆可與菊酯類農藥混用。

2.防除闊葉雜草為主的除草劑的安全使用技術

（1）莠去津

防治對象　稗草、狗尾草、牛筋草、馬齒莧、反枝莧、檾麻、龍葵、酸漿屬、酸模葉蓼、柳葉刺蓼、豬毛菜等雜草，對馬唐、鐵莧菜等防效稍差。

使用技術　玉米4葉期，雜草2～3葉期莖葉處理，每公頃用有效量1140～1425g。

注意事項　莠去津持效期長，易對後茬作物造成藥害，當施藥量每畝有效成分200g時，後茬只能種玉米和高粱，敏感作物如大豆、穀子、水稻、甜菜、油菜、亞麻、小麥、大麥、西瓜、甜瓜、蔬菜等均不能種植。

（2）2，4-滴丁酯

防治對象　對玉米田所有闊葉雜草特別是已經萌發的均有良好的效果。如：播娘蒿、藜、蓼、芥菜、離子草、繁縷、反枝莧、葎草、問荊、苣蕒菜、刺兒菜、蒼耳、田旋花、馬齒莧等闊葉雜草，對禾本科雜草無效。

使用技術　玉米3～5葉期，每公頃用有效量432～675g，莖葉噴霧處理。

注意事項　①必須在玉米3～5葉期用藥，過早或過晚都會造成玉米藥害。②2，4-滴丁酯乳油揮發性強，對棉花、油菜、向日葵等雙子葉作物十分敏感，易造成飄移性藥害。所以施藥時應選在無風或小風時，應留寬20m以上的隔離帶。噴霧器的噴頭最好戴保護罩，避免飄移。③噴施過2，4-滴丁酯器械要用鹼水徹底清洗。

（3）2甲4氯

防治對象　柳葉刺蓼、酸模葉蓼、藜、反枝莧、蒼耳、問荊、芥菜、田旋花、苣蕒菜、檾麻、馬齒莧、大巢菜、牛繁縷、刺兒菜、鴨跖草、地膚、豚草等。

使用技術　玉米3～5葉期，每公頃用有效量390～487.5g，莖葉噴霧。

注意事項　①必須在玉米3～5葉期用藥，過早或過晚都會造成玉米藥害。②雙子葉植物對其比較敏感，施藥時儘量避開雙子葉作物田，應選擇無風的天氣施藥，噴霧器的噴頭最好戴保護罩，避免飄移。③用藥後應徹底清洗施藥器械。

（4）滅草鬆（苯達鬆、排草丹）

防治對象　蒼耳、反枝莧、凹頭莧、刺莧、蒿屬、刺兒菜、大薊、狼把草、鬼針草、酸模葉蓼、柳葉刺蓼、節蓼、馬齒莧、野西瓜苗、豬殃殃、向日葵、辣子草、野蘿蔔、豬毛菜、刺黃花稔、苣蕒菜、繁縷、曼陀羅、藜、小藜、龍葵、鴨跖草（1～2葉期效果好，3葉期以後藥效明顯下降）、豚草、遏藍菜、旋花屬、芥菜、檾麻、野芥、蕓薹屬等多種闊葉雜草。

使用技術　玉米3～5葉期，每公頃用有效量1080～1440g，莖葉噴霧。

混用　滅草鬆可與煙嘧磺隆混用以擴大殺草譜。

（5）碸嘧磺隆（寶成）

防治對象　用於防除玉米田中一年生或多年生禾本科及闊葉雜草，如田薊、鐵薺、香附子、皺葉酸模、阿拉伯高粱、野燕麥、止血馬唐、稗草、多花黑麥草、檾麻、反枝莧、豬殃殃、虞美人、繁縷。

使用技術　玉米2～4葉期，每公頃用有效量5～15g，莖葉噴霧處理。

混用　碸嘧磺隆若與莠去津或噻吩磺隆（噻磺隆）混用，不僅可擴大殺草譜，還可提高對闊葉雜草如藜、蓼等的防除效果。

注意事項　①使用碸嘧磺隆前後7d內，儘量避免使用有機磷殺蟲劑，否則可能會引起玉米藥害。②碸嘧磺隆應在玉米4葉期前施藥，如玉米超過4葉期，單用或混用對玉米均有藥害發生，藥害症狀表現為拔節困難，株高矮小，葉色淺，發黃，新葉捲縮變硬，有發紅現象，隨加強玉米肥水管理，10～15d可恢復正常。③甜玉米、爆裂玉米、糯玉米及制種田不宜使用。

（6）甲基磺草酮（米斯通、硝磺草酮）

防治對象　主要防治蒼耳、檾麻、藜、莧、蓼、龍葵、三裂葉豚草等大多數田間重要闊葉雜草及幼齡稗、馬唐、狗尾草、臂形草等部分禾本科雜草，它防除闊葉雜草的效果優於禾本科雜草。

使用技術　玉米3～5葉期，每公頃用有效量100～150g，莖葉噴霧處理。

混用　甲基磺草酮可與煙嘧磺隆、碸嘧磺隆、莠去津、溴本腈等混用，以擴大殺草譜。

（7）苯吡唑草酮（苞衛）

防治對象　主要防治馬唐、稗草、牛筋草、野黍、狗尾草、藜、蓼、檾麻、馬齒莧、蒼耳、龍葵等一年生禾本科及闊葉雜草，苯吡唑草酮加入莠去津後有顯著的增效作用，除了對上述雜草具有優異的防效外，還可以對惡性闊葉雜草如刺兒菜、苣蕒菜、鐵莧菜、鴨跖草具有良好的防除效果。苯吡唑草酮對近年來危害嚴重、耐藥性極強的狗尾草、馬唐、牛筋草、野黍等禾本科雜草也有較好的防效。

使用技術　在雜草2～5葉期，雜草出齊時（老少三輩均出來）噴藥。每畝5mL苞衛＋90mL專用助劑＋70g　90%莠去津水分散粒劑，對水15～20L。也就是一套苯吡唑草酮（苞衛）＋助劑＋莠去津組合噴1畝地。在氣候較為乾旱或草齡較大的情況下，為了保證除草效果，需要適當加大用藥量。每畝7.5mL苞衛＋135mL專用助劑＋105g　90%莠去津，對水15～20L，也就是三套苯吡唑草酮（苞衛）＋助劑＋莠去津組合噴2畝地。

苯吡唑草酮（苞衛）可以被雜草的葉片、根和莖吸收，並在植物體內向上和向下雙向傳導，間接影響類胡蘿蔔素的合成，干擾葉綠體在光照下的合成與功能，最終導致雜草嚴重白化、組織壞死，雜草死亡。

雜草受藥害後的典型症狀是雜草心葉白化。在20℃下，一般噴藥2～4d就可以見效。所以苯吡唑草酮（苞衛）是目前殺草速度最快的苗後除草劑。

一般的玉米苗後除草劑只能在玉米的2～6葉期用藥，超過此時期用藥就會有明顯的藥害。苞衛不受此用藥時期限制，玉米超過6葉，甚至達到12葉期也能使用苞衛，沒有藥害。

田間試驗表明，苯唑草酮對絕大多數類型的玉米（常規大田玉米、甜玉米、爆裂玉米、糯玉米）均表現出非常好的安全性，是目前國內外市場上所有玉米苗後除草劑品種中最安全的除草劑。所以苞衛可以十分安全地用於常規玉米、糯玉米、甜玉米、爆裂玉米、飼料玉米田防除各種雜草。

注意事項　①苞衛單劑對後茬作物幾乎沒有任何影響，因此下一年要種植較為敏感的經濟作物並對後茬殘留有嚴格的限制時可以採用苞衛單劑除草。②苞衛是超高效、超低量苗後噴霧除草劑，對每畝噴液量和噴霧器械有較為嚴格的要求。對噴霧器的霧化效果要求高，霧滴要細。也就是高濃度、細霧滴。每畝對水量15～20L；每公頃對225～300L水。噴液量遠遠高於苞衛的最佳對水量會嚴重影響苞衛的除草效果。

3.玉米封壟後除草劑的安全使用技術

經過前期各種雜草防除措施後，玉米田後期還會長出很多雜草，這些雜草雖然對當茬玉米產量沒有太大影響，但防除掉這些雜草對減少下茬雜草基數具有重要作用。推薦使用百草枯。

防治對象　可防除各種一年生雜草；對多年生雜草有強烈的殺傷作用，但其地下莖和根能萌出新枝；對已木質化的棕色莖和樹幹無影響。

使用技術　玉米封壟後，每公頃用有效量300～600mL，對水375kg，作行間保護性定向噴霧。

注意事項　①百草枯為滅生性除草劑，在玉米生長期使用，切忌汙染作物，以免產生藥害。②配藥、噴藥時要有防護措施，戴橡膠手套、口罩，穿工作服。如藥液濺入眼睛或皮膚上，要馬上進行沖洗。③使用時不要將藥液飄移到果樹或其他作物上。
