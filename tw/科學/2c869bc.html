2019年大家的願望清單，你中了幾條？

2020-10-29
| 科學



每到年底和新年的開始，大家總要總結一下上一年，看看自己收穫了多少，又有哪些未完成的願望需要在下一年完成的，說來好笑，但是這個確實是很有必要的，沒有小目標，你生活怎麼才能有動力呢？我們一起來看看大家願望清單吧！



大眾的願望清單

1. 減肥，保持身材。

2. 鍛鍊身體，保持健康。

3. 努力工作，漲工資。

4. 找到另一半，早日步入婚姻的殿堂。

5. 生個胖娃娃/抱上大孫子。

6. 出國旅遊一次。



女人的願望清單

1. 自己越來越漂亮，越來越有魅力。

2. 擁有魔鬼的身材，天使的面孔，林志玲的聲音。

3. 早日找到男朋友，最好有房、有車、人帥、疼人、工資高。

4. 老公體貼，工作上進。寶寶可愛，成績優秀。婆婆大氣，有共同話語。

5. 買的起漂亮的衣服，用得起高級的化妝品，逛街最好不看價錢。

6. 吃好多好吃的，還能不長肉。熬夜看劇，還長不黑眼圈，穿高跟鞋走路，還不累腳。



男人的願望清單

1. 自己越來越帥，越來越有魅力。

2. 啤酒肚下去，練出肌肉。

3. 早日找到女友，不用太好看，看著順眼就行，最好善解人意，身材好。

4. 賺更多的錢，結婚，買房，買車，買奶粉。

5. 大家都尊敬我，說話更有分量。

6. 爸爸媽媽都健健康康的，老婆幸福賢惠，孩子開心懂事。



爸爸、媽媽的願望清單

1. 孩子可以常回家看看,多往家裡打電話。

2. 孩子可以多賺點錢。

3. 可以早點抱上孫子孫女。

4. 兒孫滿堂。

5. 身體健康。

6.孩子能感受到父母的不易，早點懂事，讓我們省心。



狗狗的願望清單

1.主人能多陪我。

2.多些肉骨頭。



打工人的願望清單

1. 工資越來越高。

2. 少加班，加班的話能多給加班費。

3. 公司福利多一些。

4. 老闆大氣不苛刻。



老闆的願望清單

1. 公司效益能越來越好。

2. 員工越來越努力，最好還不提漲工資。

3. 甲方越來越善解人意，不挑剔。

4. 成本越來越低，利潤越來越高。



單身狗的願望清單

1. 找到另一半。

2. 最好可以早點結婚。

3. 結婚後早點生娃。



情侶（夫妻）的願望清單

1.對方可以愛我更多一點。

2.我們在一起時多創造些美好的回憶，永遠不分離。

3.彼此多諒解。

4.不吵架，不冷戰，有問題，多溝通。



男神、女神的願望清單

1. 多讀幾本書，提高自己的智商、情商。

2. 學習英語，能和外國人順暢的交流。

3. 學習舞蹈，提高氣質。

4. 學會穿搭，做個精緻的人。

5. 學會做菜，善待自己的胃。

6. 學習書法，讓自己字更美麗。

7. 學會化妝，妝容越自然越好。



學生的願望清單

1. 少留點作業。

2. 取得優異的成績，早點畢業。

3. 老師不拖堂。

4. 換一個更溫柔更美麗的老師。

5. 爸媽可以不催著我上輔導班。

6. 和喜歡的人坐同桌。



老師的願望清單

1. 學生都聽話，少惹事。

2. 多教出幾個優秀的學生。

3. 同學們都能認真聽講，上課的時候不講話，不調皮。

4. 我講的課大家都愛聽。



房東願望清單

1. 每個月及時收到房租，不用去催錢。

2. 住房的人愛乾淨，不養狗，素質高。

設計師的願望清單

1. 甲方大氣豪爽不磨磨唧唧。

2. 少改幾次設計稿，早點下班回家吃飯。

3.設計費給的及時，不用一直上門催。



甲方願望清單

1. 乙方善解人意，做事周到。

2. 乙方審美好，最好和領導想法一致。



環衛工人的願望清單

1． 大街上，大家可以少扔些垃圾。

2． 最好不要在大街上嗑瓜子。

3． 大風大雨天可以少一些。



每個人，所處的位置不同，所求亦不同，但是大家的方向都是一致的，就是希望自己可以越來越好，新的一年，也祝大家定好自己的目標，向著這個方向去努力，早日達成所願，心想事成！
