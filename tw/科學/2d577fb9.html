管會視野2.0：存貨成本計算的兩種方式，吸收成本法和變動成本法

2020-11-14
| 科學

Hi，小夥伴們大家好！

儘管以鍾南山院士為首的科研團隊正在不分晝地夜對新型冠狀病毒展開研究，但是截止至當前，我國還沒有治癒新冠肺炎的“神藥”。

現在還沒到我們放鬆警惕的時候，但是老師從網絡中瞭解到有些景區在這樣的緊要關頭，竟然以“免費”為噱頭吸引遊客。老師對於這一現象十分憤慨，想跟大家分享幾點心得：

  • 景區營業不必急於一時，好景不怕等人，更值得我們等！
  • 景區是人流極易聚集之地，景區責任方不應僅為短期私利，而讓舉國防疫控疫的辛苦付出付之東流。
  • 相信很多人也和老師一樣被迫宅家很久了，看到天氣晴好，忍不住想出去看看，殊不知室外更容易接觸到肆意橫行的病毒。

所以，請小夥伴們繼續將“宅”精神發揮到底，儘量減少不必要外出，保護好自己和家人就是在為疫情防控出力！今天，小夥伴們就跟老師一起來學習存貨成本計算的兩種方式——吸收成本法和變動成本法吧！

變動成本法

變動成本法是指將所有“變動”製造相關的成本計入存貨成本，即存貨成本包括直接人工、直接材料、變動製造費用，而將固定製造費用作為期間費用在利潤表中全額扣除。這種方法扣除了不可控的固定製造費用，更符合企業內部的績效考核，因此適用於企業對內管理用財務報表。

吸收成本法

吸收成本法（完全成本法）是指將所有與製造相關的成本都計入存貨成本，即存貨成本不僅包括直接人工、直接材料、變動製造費用，還包括固定製造費用。這種方法符合美國聯邦稅務局與公認會計原則( GAAP)對存貨成本的要求，因此企業對外財務報告都使用吸收成本法。

從上圖看出變動成本法和吸收成本法之間的差異在於對固定間接製造費用的處理上。吸收成本法將固定間接製造費用作為產品成本。變動成本法則將固定間接製造費用視作期間費用在利潤表中全額扣除。

在實際題目中，固定製造費用一般以“單位固定制造成本”的方式給出，“單位固定制造成本”是基於預計固定製造費用總額和預計產能計算出的標準固定製造費用。而實際固定製造費用總額和預計固定製造費用總額可能存在差異。

在變動成本法下固定製造費用按實際發生額全額結轉到利潤表，不需要考慮兩者差異調整的問題；而在吸收成本法下，如果兩者差額較小，則由當期銷貨成本承擔；如果兩者差額較大，則由當期銷貨成本和期末存貨共同承擔。

那麼，這兩種方式在計算上有什麼區別呢？我們通過一道例題來了解


例題：根據以下信息，分別用變動成本法和吸收成本法計算期末存貨成本，並編制利潤表。

注：本題中不涉及實際固定製造費用總額和預計固定製造費用總額的差異調整。

變動成本法：

（存貨成本只包括變動製造成本，即直接人工、直接材料和變動製造費用，固定製造費用全額結轉到利潤表當期費用化）：

變動製造成本總額 = 1,000 * $35 = $35,000

期末存貨成本 =（1,000 - 750）* $35 = $8,750

結轉到利潤表的總變動成本 = 隨銷售結轉到利潤表的變動製造成本 + 變動銷售成本 = 750 * $35 + 750 * $25 = $45,000

邊際貢獻 = 銷售收入 - 總變動成本 = 750 * $150 - $45,000 = $67,500

結轉到利潤表的固定成本 = 固定制造成本 + 固定銷售成本 = 1,000 * $20 + $20,000 = $40,000

變動成本法下的營業利潤 = 邊際貢獻 - 固定成本 = $67,500 - $40,000 = $27,500

吸收成本法：

（存貨成本包括變動製造成本，即直接人工、直接材料、變動製造費用和固定製造費用，即當生產量大於銷售量時，一部分固定製造費用隨銷售結轉到利潤表的銷貨成本中，一部分固定製造費用隨期末存貨結轉到資產負債表的存貨金額中，直到這部分存貨在以後會計期間被出售時，相關固定製造費用再結轉到利潤表）：

當期生產成本總額 = 變動製造成本總額 + 固定制造成本總額 = 1,000 *（$35 + $20）= $55,000

結轉到利潤表的銷貨成本 = 750 *（$35 + $20）= $$41,250

期末存貨成本 = $55,000 - $$41,250 = $13,750

毛利 = 銷售收入 - 銷貨成本 = 750 * $150 - $41,250 = $71,250

銷售成本總額 = 變動銷售成本 + 固定銷售成本 = 750 * $25 + $20,000 = $38,750

吸收成本法下的營業利潤 = 毛利 - 銷售成本總額 = $71,250 - $38,750 = $32,500

案例分析：

變動成本法下營業利潤為$27,500，而吸收成本法下營業利潤為$32,500，兩者相差$5,000。小夥伴們想一想這個差額是怎樣造成的呢？其實是因為當期的生產量大於銷售量，變動成本法下固定制造成本全額結轉到利潤表，而吸收成本法下的固定制造成本一部分結轉到利潤表，一部分則留存在期末存貨中結轉到了利潤表。因此這就導致了吸收成本法下的營業利潤更大，利潤差異 = （生產量 - 銷售量）* 單位固定制造成本 = （1,000 - 750）* $20 = $5,000

當生產量小於銷售量時，變動成本法下當期固定制造成本全額結轉到利潤表，而吸收成本法下除了當期固定制造成本結轉到利潤表，還有期初存貨中留存的固定制造成本也結轉到了利潤表，這會導致變動成本法下的營業利潤更大；當生產量等於銷售量時兩種方法下都是將當期的固定制造成本結轉到了利潤表，此時利潤相等。

由此，我們得到的結論如下：

  • 生產量 ＞ 銷售量，吸收成本法營業利潤 ＞ 變動成本法營業利潤
  • 生產量 ＜ 銷售量，吸收成本法營業利潤 ＜ 變動成本法營業利潤
  • 生產量 = 銷售量，吸收成本法營業利潤 = 變動成本法營業利潤

吸收成本法下的利潤 - 變動成本法下的利潤 = （生產量 - 銷售量）* 單位固定制造成本

也正是由於吸收成本法將固定制造成本計入存貨成本，所以也給了企業操控利潤的機會。企業可以在銷售量不變的情況下，僅僅增加產量，就可以使得更多的固定制造成本被留存到期末存貨中，則當期銷貨成本降低，當期毛利增加，當期利潤增加。當將績效考核與利潤相關聯時，相關負責人更會有意識地通過過量生產來“虛高”利潤。而過量生產會導致大量的存貨積壓，存貨相關的儲存保險成本、損毀風險都會增加，企業會因為囤貨虛高利潤一時爽，而後患無窮！

除了以上增加產量來操控利潤之外，一些利慾薰心的企業管理者還可能會為了個人業績，不惜犧牲企業整體的利益，有意識地去選擇生產固定制造成本更高的產品，而不以產品的盈利能力作為生產判斷依據。這是因為增產固定制造成本更高的產品，可以使更多的固定制造成本留存到期末存貨，即更大程度的“虛增”利潤。當然以上這些“虛高”利潤的做法都是不可取的，短期的飲鴆止渴只會讓企業付出更大的代價！
