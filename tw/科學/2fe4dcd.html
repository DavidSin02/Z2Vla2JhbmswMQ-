關鍵詞:殼聚糖膜熱處理

納米微晶纖維素單寧酸薄膜

本文分析了熱處理對殼聚糖納米複合膜的影響。使用對流烘箱對一系列樣品進行熱處理，所述樣品包括:

純殼聚糖膜、包埋有納米微晶纖維素的殼聚糖膜、用單寧酸交聯的殼聚糖膜和用納米微晶纖維素和單寧酸的混合物交聯的殼聚糖膜。傅里葉變換紅外光譜和 x 光衍射測試顯示了熱處理薄膜化學相互作用的變化。熱處理後的薄膜在吸溼方面有顯著的改善。當樣品進行熱處理時，拉伸強度和楊氏模量分別增加到 7兆帕和 259 兆帕。對於納米纖維素顆粒，採用透射電子顯微鏡對懸浮態纖維素顆粒的結構性能進行了檢測。

熱學性能

假設殼聚糖膜的熱處理不會增加膜的結晶度。由於熱處理過程中強烈的交聯相互作用，結晶度可能會降低。

這表明熱處理對殼聚糖的分子結構有影響。此外，熱處理樣品顯示強度在2 θ= 19◦ 到

22◦. 然而，對於所有熱處理過的薄膜，觀察到的峰沒有明顯的變化。

機械性能.

熱處理對殼聚糖膜機械性能的影響如圖 4 所示。一般來說，當熱處理時，樣品 C、CX 和 CN 的拉伸強度和楊氏模量提高，而樣品 CNX 經歷了輕微的降低(圖 4a 和 4c )。樣品 C 具有最低的拉伸強度( 36.93 兆帕)、楊氏模量( 721.61 兆帕)和最高的斷裂伸長率百分年齡( 43.1 % )。單寧酸和 NCC 的加入增加了所有樣品的拉伸強度和楊氏模量，但以斷裂伸長率為代價。力學性能改善最大的是拉伸強度為 48.32 兆帕( P < 0.05 )的氯化萘樣品，其次是熱處理後的氯化萘樣品( P < 0.05 )。與純殼聚糖膜相比，熱處理樣品氯化萘的楊氏模量最高，為 1080.14 兆帕( P < 0.05 )，未處理樣品氯化萘最高，為 1256.48 兆帕( P < 0.05 )。從斷裂伸長率觀察到的趨勢(圖 4b )顯示，當引入單寧酸時，樣品 C 和 CX 的含量大幅降低(所有

樣品)。然而，添加氯化萘和單寧酸(樣品氯化萘和氯化萘)不會影響斷裂伸長率。這一趨勢類似於先前的研究( Rubentheren 等人)。，2015 b )。對於未處理的薄膜，拉伸強度和硬度的增加是由於單寧酸和 NCC 的加入。殼聚糖膜交聯導致形成更穩定的。

結論：

殼聚糖的機械性能和阻隔性能可以通過添加單寧酸、NCC 和熱處理工藝來改變。傅里葉變換紅外光譜表明熱處理引起了殼聚糖膜要改。熱處理的 XRD 衍射圖顯示在 11.4♀處峰變平，在 2θ = 19◦ 到22◦.。熱處理後的樣品顯示出機械強度的顯著提高和含水量的降低。這是由於加速交聯壓縮了結構。這些改進表明，熱處理殼聚糖對骨形態發生器翼膜有很大的潛力。
