線性方程組的解集及其幾何意義

2020-11-14
| 科學

我們解決現實問題時可以自由選取其中任意一個作為模型。 我個人認為，線性方程組是最“質樸”的形式； 向量方程則是與幾何建立了關係，這將方便我們進行更直觀的推理； 矩陣方程則是向量方程的一種“封裝”，是向量方程的一種抽象，它將具體的向量形式隱藏，提供給我們一個簡潔的 API 形式——矩陣。 未來將要介紹的很多概念就是基於對這一層封裝的研究，如果到時候我們發現某個概念理解有困難，不妨轉換思路到向量方程或線性方程組的形式進行分析。

此外，我們之前還進行了關於線性方程組解集的討論，在這章我們對其進一步探討。

注意，這裡講齊次方程組和非齊次方程組的解有一個前提，就是非齊次方程組首先要是有解的，如果 0 變成 b 導致方程組沒有解，那麼也就不能用齊次方程組的解集平移了。

結合之前總結的齊次線性方程組解的性質，當方程組含有 p 個自由變量時，齊次方程組的解集是 p 個向量的張成空間，而非齊次方程組解集只是這個空間進行了平移（前提是非齊次方程組有解），並沒有改變這個空間的基本性質（比如空間的維度）。

三、列空間

矩陣

的各個列向量線性組合組成的集合，就是 A 的列空間。 記作 Col A ，即

Col A = Span { a 1 , a 2 , ⋯ , a n }

這個列空間，我們應該不陌生了，上一章中很多時候都是把矩陣看成列向量的排列，考慮 A x = b 的解的情況時其實就是在列向量中進行分析的。 列空間在分析矩陣中各列向量的線性相關性時很有幫助： 只有各列線性無關時，這 n 個列才能張成 n 維空間，這時就說這個矩陣的秩為 n ； 而假如這裡面有 1 列和其他某列線性相關，那麼這 n 個列就只能張成 n − 1 維空間，這個矩陣的秩就是 n − 1 ； 也就是說， 矩陣的秩說明了這個矩陣的列向量最多能張成多少維 。

如下圖中， A = [ a 1 a 2 a 3 ] ，由於有兩個向量線性相關，導致 3 個列向量只能張成 2 維，因此 A 的秩為 2。 所以 A x 得不到任意三維向量 b ，也就是 A x = b 並不對所有 b 成立（只有 b 是 A 列空間中的向量時才成立）。

更進一步，非齊次線性方程組 A x = b 中，如果 A 已知， x 和 b 未知，此時我們關注的問題是 A 的列向量能張成多少維；如果 A 和 b 已知，我們關注的問題就是 A 中 n 個列向量如何線性表示能表示成 b ，這時候我們如果提前知道 A 的列空間達不到 b 的維數，那麼這些列向量就一定無法線性組合出 b 。

四、零空間

齊次方程 A x = 0 的全部解組成的集合，稱為矩陣 A 的零空間，記作 Nul A 。

當 A 中的列向量線性無關時， A x = 0 只有零解，這時 A 的零空間就是 0 ； 而只要 A 中的列向量線性相關， A x = 0 就存在非零解，這時 A 的零空間就是一個維度大於 0 的空間。

關於列空間和零空間的討論先在這裡打住，之後會進一步討論它們之間的關係和各自的意義。 目前只要知道列空間是由 A 的列向量張成的，而零空間的意義更隱晦一些，是 A x = 0 的所有解組成的空間。從列空間能看出 A 各列的線性相關關係，列向量越相關，列空間維度越低。 從零空間也能看出 A 各列的線性相關性，列向量越相關，零空間維度越高。 而負責量化描述 A 列向量有多麼線性相關的，是一個叫做 秩 的東西。
