NumPy隨機數

隨機數常用於蒙特卡羅法、隨機積分等方面。然而，真正的隨機數很難獲得，實際中使用的都是偽隨機數。大部分情況下，偽隨機數就足以滿足我們的需求。當然，某些特殊情況除外，如進行高精度的模擬實驗時。對於NumPy，與隨機數有關的函數都在random子程序包中。

提示：

NumPy核心的隨機數發生器是基於梅森旋轉算法的，詳見https://en.wikipedia.org/wiki/Mersenne_twister$2。

我們既可以生成連續分佈的隨機數，也可以生成非連續分佈的隨機數。分佈函數有一個可選的size參數，它能通知NumPy要創建多少個數字。我們可以用整型或者元組來給這個參數賦值，這時會得到相應形狀的數組，其值由隨機數填充。離散分佈包括幾何分佈、超幾何分佈和二項式分佈。連續分佈包括正態分佈和對數正態分佈。

  • 用二項式分佈進行博弈

二項式分佈模擬的是在進行整數次獨立實驗中成功的次數，其中每次實驗的成功機會是一定的。

假如我們身在17世紀的賭場，正在對8片幣玩法下注。當時流行用9枚硬幣來玩。如果人頭朝上的硬幣少於5枚，那麼我們將輸掉一個8分幣；否則，我們就贏一個8分幣。下面，我們開始模擬這個遊戲，假設我們手上有一千硬幣作為初始賭資。我們將使用random模塊提供的binomial()函數進行模擬：

提示：

完整的代碼詳見本書代碼包中的headortail.py文件。

import numpy as npfrom matplotlib.pyplot import plot, showcash = np.zeros(10000)cash[0] = 1000outcome = np.random.binomial(9, 0.5, size=len(cash))for i in range(1, len(cash)):　　 if outcome[i] < 5:　　　　　　 cash[i] = cash[i - 1] - 1　　 elif outcome[i] < 10:　　　　　　 cash[i] = cash[i - 1] + 1　　 else:　　　　　　 raise AssertionError("Unexpected outcome " + outcome)print outcome.min(), outcome.max()plot(np.arange(len(cash)), cash)show()

若要了解binomial()函數，請看下列步驟。

1．用binomial()函數。

將數組初始化為零，即現金餘額為零。調用binomial()函數，將size參數設為10000，這就表示我們要擲10000次硬幣。

cash = np.zeros(10000)cash[0] = 1000outcome = np.random.binomial(9, 0.5, size=len(cash))

2．更新現金餘額。

利用拋鄭硬幣的結果來更新cash數組。顯示outcome數組中的最大值和最小值，只要保證沒有罕見的異常值即可。

for i in range(1, len(cash)):　　 if outcome[i] < 5:　　　　 cash[i] = cash[i - 1] - 1　　 elif outcome[i] < 10:　　　　　　cash[i] = cash[i - 1] + 1　　 else:　　　　　　raise AssertionError("Unexpected outcome " + outcome)print outcome.min(), outcome.max()

  • 正態分佈採樣

連續分佈是通過 概率密度函數（pdf） 進行建模的。在特定區間發生某事件的可能性可以通過概率密度函數的積分運算求出。NumPy的random模塊提供了許多表示連續分佈的函數，如beta、chisquare、exponential、f、gamma、gumbel、laplace、
lognormal、logistic、multivariate_normal、noncentral_chisquare、
noncentral_f、normal等。

利用NumPy的random子程序包中的normal()函數，可以把正態分佈以直觀的形式圖示出來。下面給出隨機產生的數值的鐘形曲線和條形圖（完整的腳本詳見本書代碼包中的normaldist.py文件）：

import numpy as npimport matplotlib.pyplot as pltN=10000normal_values = np.random.normal(size=N)dummy, bins, dummy = plt.hist(normal_values, np.sqrt(N), normed=True, lw=1)sigma = 1mu = 0plt.plot(bins, 1/(sigma * np.sqrt(2 * np.pi)) * np.exp( - (bins - mu)**2 / (2 * sigma**2) ),lw=2)plt.show()

隨機數可以按照正態分佈的要求生成，它們的分佈情況可以用條形圖來顯示。若要繪製正態分佈，請執行下列步驟。

1．成數值。

藉助於NumPy的random子程序包提供的normal()函數，可以創建指定數量的隨機數。

N=100.00normal_values = np.random.normal(size=N)

下面使用matplotlib來繪製條形圖和理論上的pdf，這裡中心值為0，標準差為1：

用SciPy進行正態檢驗

正態分佈被廣泛用於科學和統計學領域，按照中心極限定理，隨著獨立觀測的隨機樣本數量的增加，它們會趨向呈正態分佈。正態分佈的特性已經為大家所熟知，並且它還非常便於使用，不過，它需要滿足許多必要條件，如數據點的數量要足夠大，並且還要求這些數據點必須是相互獨立的。工作中，我們需要養成檢查數據是否符合正態分佈的好習慣。正態檢驗的方法有很多，其中有一些已經在scipy.stats程序包中實現了。本節將用到這些檢驗方法，這裡的樣本是取自https://www.google.org/flutrends/data.txt$2的流感趨勢數據。這裡我們對原始文件進行了簡化處理，僅留下日期和來自阿根廷的數據兩列，下面給出示例行：

Date,Argentina29/12/02,05/01/03,12/01/03,19/01/03,26/01/03,02/02/03,136

這些數據還可以從本書代碼包中的goog_flutrends.csv文件中找到，和前面一樣，這裡也按照正態分佈對這些數據進行抽樣。得到的數組的大小與流感趨勢數組相同，如果順利通過正態檢驗，就以它為金標準（the golden standard）。

提示：

這裡的代碼取自代碼包中的normality_test.py文件。

import numpy as npfrom scipy.stats import shapirofrom scipy.stats import andersonfrom scipy.stats import normaltestflutrends = np.loadtxt("goog_flutrends.csv", delimiter=',', usecols=(1,), skiprows=1, converters = {1: lambda s: float(s or 0)}, unpack=True)N = len(flutrends)normal_values = np.random.normal(size=N)zero_values = np.zeros(N)print "Normal Values Shapiro", shapiro(normal_values)print "Zeroes Shapiro", shapiro(zero_values)print "Flu Shapiro", shapiro(flutrends)printprint "Normal Values Anderson", anderson(normal_values)print "Zeroes Anderson", anderson(zero_values)print "Flu Anderson", anderson(flutrends)printprint "Normal Values normaltest", normaltest(normal_values)print "Zeroes normaltest", normaltest(zero_values)print "Flu normaltest", normaltest(flutrends)

作為一個反面例子，下面使用的這個數組跟前面提到的填滿零的數組具有同樣的大小。實際上，當處理一些小概率事件（如世界性疫情的爆發）時，就可能遇到這種數據。

在這個數據文件中，一些單元是空的；當然，這種問題經常會碰到，所以必須習慣於首先清洗數據。我們假定正確的數值是0，下面使用一個轉換器來填上這些0值：

flutrends = np.loadtxt("goog_flutrends.csv", delimiter=',',　　usecols=(1,), skiprows=1, converters = {1: lambda s:　　float(s or 0)}, unpack=True)

夏皮羅-威爾克檢驗法可以對正態性進行檢驗。相應的SciPy函數將返回一個元組，其中第一個元素是一個檢驗統計量，第二個數值是 p 值。需要注意的是，這種填滿了零的數組會引發警告。事實上，本例中用到的3個函數在處理這個數組方面確實有困難，同時還引發警告。下面是得到的結果：

Normal Values Shapiro (0.9967482686042786,　　0.2774980068206787)Zeroes Shapiro (1.0, 1.0)Flu Shapiro (0.9351990818977356, 2.2945883254311397e-15)

這種用零填充的數組看起來有點怪，雖然會收到相關的警告，但是大可以忽略不計。這裡得到的p值類似於本例中後面的第三個檢驗的結果。分析結果基本上是一樣的。

Anderson-Darling檢驗可以用來檢驗正態分佈以及其他分佈，如指數分佈、對數分佈和岡貝爾（Gumbel）分佈等。相關的SciPy函數涉及一個檢驗統計量和一個數組，該數組存放了百分之15、10、5、2.5和1等顯著性水平所對應的臨界值。如果該統計量大於顯著性水平的臨界值，就可以斷定它不具有正態性。我們將得到下列數值：

Normal Values Anderson (0.31201465602225653, array([ 0.572,　　 0.652,　　0.782,　　0.912,　　1.085]), array([ 15. ,　　10. ,　　 5.　　,　　 2.5,　　 1. ]))Zeroes Anderson (nan, array([ 0.572,　　0.652,　　0.782,　　 0.912,　　1.085]), array([ 15. ,　　10. ,　　 5. ,　　 2.5,　　 1.　　]))Flu Anderson (8.258614154768793, array([ 0.572,　　0.652,　　 0.782,　　0.912,　　1.085]), array([ 15. ,　　10. ,　　 5. ,　　 2.5,　　　　1. ]))

對於這種用零填充的數組，我們沒有什麼好說的，因為返回的統計量根本就不是一個數值。就像我們期望的那樣，這個金標準數組是符合正態分佈的。然而，流感趨勢數據返回的這個統計量大於所有的有關臨界值，因此我們可以肯定它不符合正態分佈。這3個測試函數中，這個看起來是最容易使用的一個。

SciPy的normaltest()函數還實現了D'Agostino檢驗和Pearson檢驗功能。該函數返回的元組和shapiro()函數一樣，也包括一個統計量和p值。這裡的p值是雙邊卡方概率（two-sided Chi-squared probability），卡方分佈是另一種著名的分佈，這種檢驗本身是基於偏度和峰度檢驗的z分數的。偏度係數用來表示分佈的對稱程度的，這樣，由於正態分佈是對稱的，所以偏度係數為零。峰度係數描述的是分佈的形狀（尖峰，肥尾）。這種正態分佈的峰度係數為3，超額峰度的係數為0。以下是本次檢驗獲得的數值：

Normal Values normaltest (3.102791866779639, 0.21195189649335339)Zeroes normaltest (1.0095473240349975, 0.60364218712103535)Flu normaltest (99.643733363569538, 2.3048264115368721e-22)

因為這裡處理的是p值的概率，所以它越大越好，最好接近1。對於這種用零填充的數組，會得到很奇怪的結果，不過，由於我們已經得到提示，所以對於這個特殊數組來說，結果是不可信的。此外，如果p值大於等於0.5，則認為它具有正態性。對於金標準數組，我們會得到一個更小的數值，這說明我們需要進行更多的觀察，這一點留作作業，請讀者自行練習。
