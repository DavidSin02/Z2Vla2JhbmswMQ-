第1則

-THE FIRST-

畜牧業對飼料工業技術需求

1畜產品市場需求變化

當前，養殖業進入集約快速發展階段，畜禽養殖數量和畜產品產量已經接近峰值，截至 2016 年全國豬肉產量5 299 萬 t，禽肉產量 1 888 萬 t，禽蛋產量 3 095 萬 t( 陶莎，2017)。2030 年前後我國將進入人口老齡化高峰，人口紅利即將消失，對動物性食品的消費量將逐漸降低。此外，城鄉居民傳統的“價格優先”消費觀念正在向“價格與質量並重”轉變 ( 辛翔飛等，2015)。上述的發展變化都對飼料工業提出了更高的要求。

由於我國畜產品生產質量控制方面技術不足，畜產品貿易一直處於逆差狀態 ( 胡梅梅等，2016)。另外，低抗、無抗飼養已成為大勢所趨，且個性化特色畜產品市場需求增加。因此，保障畜產品的安全與品質亟需技術創新。過去大家基本處於觀望狀態，但是現在如果還是觀望，而不去超前地思考與創新，被淘汰的風險就會大大增加。

2

環保計入養殖成本

目前，龐大的畜禽養殖量帶來的環境負擔問題愈顯突出，發展循環低碳養殖已經刻不容緩。養殖企業除承擔傳統的動物、飼料、設備、設施和人員等成本外，還將承擔環境成本，需要投入和技術創新。低排放飼料產品市場需求 ( 如低抗、低礦物元素、低氮和低有機物的飼料產品 ) 很快會出現。我們一直認為有機肥發酵之後再使用到土地中是很好的模式，但事實上，現在的有機肥和過去的有機肥不同，連續在土地上施用 3 年以上，就會導致土地重金屬超標，從而致使蔬菜重金屬也超標。以此倒推，一定會對畜禽糞便的重金屬含量作出要求，而最終會倒逼飼料生產低重金屬含量的飼料產品。

3

進一步降低養殖飼成本

我國平均每出欄 1 頭育肥豬的飼料消耗量比國際先

進水平多 10% 以上，肉雞和蛋雞多 5% 左右 ( 陶莎，2017)。飼料佔養殖成本比例最大 (70% 以上 )，但是可挖掘空間也最大，提高動物飼料利用效率依然是提升養殖效益的關鍵。規模化和集約化養殖模式的推進，以及養殖企業逐漸可以承擔部分飼料加工的任務，為進一步提升飼料利用率奠定了基礎。生物技術的發展和飼料生產工藝的變革使大幅度提升養殖效益變為現實。

第2則

-THE FIRST-

飼料工業對技術創新依賴性增強

1飼料加工企業需要安全度過微利時代

飼料行業已經開啟了微利時代，大多數飼料企業應該如何度過這個時代是很嚴肅的問題。飼料總需求基本趨於穩定，通過簡單的數量擴增已經很難再獲得較大的增長，必須從質上進行突破。飼料產業規模效益已得到充分發揮，飼料產品的利潤也在逐年降低，飼料企業間的競爭空前激烈，僅依靠規模已難以維持持續發展。產業鏈閉合和加工智能化，減少用人 ( 配方師、銷售、財務、工人等 ) 產生的

經濟效益十分有限，與巨大的飼料產值相比，這些利潤空間十分有限，難以維持未來的持續發展。因此，我認為不要過分地迷信智能化、信息化、網絡化，重中之重還是要把佔比 70% 的成本降下來，這才是核心問題。

2

緩解飼料資源短缺必須依賴技術創新

據海關數據顯示，2017 年進口大豆 9 554 萬 t，創歷史最高紀錄。2017 年魚粉進口量為 157.51 萬 t( 佔國內魚粉消費量的 80% 以上 )，較 2016 年全年進口增加 53.81萬 t，創 8 年來最高位。我國 2017 年進口玉米 283 萬 t。2017 年全年高粱、大麥、幹木薯進口量分別為 505.7 萬 t、886.4 萬 t 和 784.3 萬 t。糧食加工副產品糠麩、糖渣、粕類進口總量 255.1 萬 t。飼料資源短缺將是長期限制我國畜牧業發展的瓶頸問題，導致飼料成本上升。

3

現有飼料加工工藝對利用率的改善已十分有限

制粒能夠促進動物生長 ( 提高 5%~10%)，膨化工

藝可使細胞壁破裂，從而提高飼料消化率。受現有飼料加工與消費方式的條件性 ( 時間短、水分低、高能耗和保質期長 ) 限制，利用飼料加工工藝進一步提升飼料營養價值的空間已十分有限。必須思考“生產力與生產關係”的矛盾性。

4

飼料加工與消費模式需要顛覆性創新

飼料需要一定的保質期，再到養殖戶進行使用，在

這個過程中所有的條件和現有的生產模式限制了新的加工工藝的發展。因為含水量高就會縮短保質期，保質期短就會出現各種投訴問題。所以，我們現在要重新思考“生產力與生產關係”之間的關係，必須依靠新思維、新方式、新模式來解決這些問題。利用技術創新深挖飼料原料的潛在價值，提高可消化與不可消化養分的利用，彌補進口價格差。飼料企業亟需“創新型的飼料加工技術和消費模式，深度改善飼料營養價值和產品特性”，尋求

一種新的增長點，否則將會被迫出局。開發安全、健康、高效和環保型飼料產品，適應養殖業市場需求，提升畜牧業經濟的競爭力。

5

飼料加工技術創新是生存之本

創新需要高研發投入和長時間積累，飼料行業技術壁壘不斷提升，技術創新已成為新時期飼料企業所面臨的最嚴峻問題。

第3則

-THE THIRD-

生物技術在改善飼料營養價值的潛質

我認為生物飼料在飼料工業中的潛質是巨大的，如何有效地對其進行利用是核心的問題。

1酶製劑在飼料工業中的潛質遠未得到充分發揮

因微生物酶的作用效果易受動物消化道內環境(溫度、pH、蛋白酶和金屬離子等 ) 和飼料加工工藝 ( 高溫制粒 ) 影響較大，其潛在價值遠未得到充分發揮。若在酶最適條件下對飼料 ( 營養成分和抗營養成分 ) 進行深度預消化處理，可有效避免動物消化道內化境和加工制粒對酶的不利影響，充分發揮酶的高催化效率與潛質，深度改善飼料營養價值。

從而提升酶的效率，使其不受催化環境條件 ( 溫度、pH)、時間和劑量限制 ( 高劑量的蛋白酶可能會出現負面效果 )。實際上就是把飼用酶變成了工業酶。

2

動物消化生理特點決定需要對一些飼料原料進行預

處理

我認為，預消化飼料是一種新的飼料加工方式，建議大家從內心深處去接受這種概念。其實預消化和過去的粉碎、混合等常規環節差不多，但是運用方式有所不同，略微複雜一些而已。

為什麼要運用飼料預消化技術呢？這和過去添加酶製劑的道理基本一致。首先，無論是幼齡動物還是成年動物，食糜在消化道中停留的時間基本是固定的，消化酶的分泌量也基本是固定的，因此動物的消化能力也是固定的。這就需要在體外條件下，人為地延長消化的時間，而這個時間可以無限延長，預處理 12 h、24 h，甚至更長。這樣就解決了動物本身的缺陷。

第二，動物育種為追求高屠宰率，動物消化器官與

體重比明顯降低，相反採食量卻有較大幅度地增加，這樣導致食糜在消化道內的停留時間過短，消化液的分泌量與採食量之間明顯不成比例，進而導致養分消化率降低。因此也需要對飼料進行提前預處理，從而減輕動物消化道的負擔。

第三，與植物組織化學結構的複雜性相比，動物消化道內消化酶分泌的種類遠遠不足。特別是一些抗營養因子成分不僅不能被消耗，而且還會導致其他養分消化率降低。預消化技術不僅會提高營養成分的消化利用率，也會充分利用非營養成分 ( 如結構性碳水化合物等 )。

第四，一些難以消化的物質，通常會引起較為強烈的食後增熱，從而增加動物的體消耗，提前預消化處理會使飼料養分更易消化，進而降低動物的體消耗。

第五，消化率不同，飼料在動物消化道內蛋白質週轉率可能存在差異。低消化率飼料原料不僅是其為動物提供的養分能力較低，還可導致動物體能量和蛋白流失較大 ( 蛋白質週轉 )，因此利用消化預處理技術可改善這些成分的消化率。

第六，利用生物預消化技術可採用微生物發酵產生多種酶製劑，預消化是降低飼料成本的關鍵技術。使用微生物發酵產酶的效率非常高，而使用的碳源 ( 能量 ) 成本又比較低，而動物消化道產酶消耗的是蛋白質、葡萄糖，對比二者的性價比顯而易見。因此，使用多種消化酶進行體外預消化，可以降低動物的消化負擔，降低消化成本。

第4則

-THE FOURTH-

飼料預消化處理技術關鍵點

1飼料生物預消化處理擬解決的關鍵問題

①把飼料酶變成“工業酶”。現在酶製劑的研發都需要耐酸、耐熱，克服動物體內的環境條件限制。而體外預消化可以使酶製劑的研發不必跨越那麼大的障礙，降低了酶製劑的研發瓶頸。

②提高易消化營養物質的消化速度和吸收效率。動物的消化道長度有限，一定要讓營養成分充分吸收。

③在體外條件下，延長難消化物質 ( 慢速消化澱粉

和一些蛋白質 ) 的消化時間，對其進行深度處理，提高消化率。

④破壞抗營養因子 ( 植酸和蛋白酶抑制等 )。

⑤開發不可消化物質的潛在價值。比如纖維，通過預消化處理將部分纖維類物質轉化為可吸收 ( 葡萄糖 ) 或後腸可發酵 ( 木糖 ) 形式的單糖。

⑥通過對纖維的預處理可使纖維轉變為短纖維，更容易在後腸發酵，產生更多的 VFA。

⑦提高飼料利用率 ( 養分利用率和養分平衡 )，降低養殖成本。提高了飼料的消化率和吸收率是一方面，我認為更重要的就是改善了營養平衡。預消化處理過的蛋白質和沒有經過處理的蛋白質的消化率和氨基酸的吸收率不同，經過預消化處理之後的氨基酸平衡一定會發生很大的變化。

⑧挖掘動物生長潛力，可以將日糧的營養水平提高，進一步提升動物的生長速度。但是面臨一個問題，我們現行的飼養標準和飼料營養價值表怎麼辦？可能會進行一些修改。尤其是對於豬來說，還有很多遺傳潛力沒有發揮出來，如果採用預消化處理，或將進一步縮短出欄時間。

2

飼料生物預消化處理的理論基礎

①深入瞭解植物組織化學結構，找出潛在挖掘關鍵點。

②破壞這些組織結構需要的關鍵酶及其協同效應。

③預消化處理終產物的控制。

④體外預消化處理的物理和生物手段如何有機結合。

第5則

-THE FIFTH-

植物組織化學結構的複雜性

1纖維多糖

01

植物纖維多糖

結構多糖包括纖維素、半纖維素、果

膠。半纖維素又包括 β －葡聚糖、阿拉伯木聚糖、甘露聚糖等。“We'll eat trees in the future”這是我在國外一個網站上看到的一句話，我相信這個觀點。隨著生物技術的發展，完全可以把纖維素轉化成葡萄糖。以後我們所食用的葡萄糖不一定來源於甘蔗或甜菜，也可能來源於樹木。希望大家對未來的預消化技術充滿信心。生物技術的發展一定會對動物營養技術產生強大的推進力。

02

纖維分解酶

纖維分解酶主要包括 3 種 ：外切 β-1,4

葡聚糖苷酶 (Exo β-1,4-glucanases，CBH)、內切 β-1,4葡聚糖苷酶 (Endo β-1,4-glucanases，EG) 和 β-1,4 葡萄糖苷酶 (β-1,4-glucosidases，BG)。在此過程中，半纖維素酶也參與其中，協同水解纖維素周圍的半纖維素，進而有利於纖維素的水解。我認為，無論是預消化還是飼料酶技術，一定是很多種酶相互協同才能實現目標。

03

參 與 纖 維 素 水 解 的 纖 維 素 酶

纖 維 素 內 切 酶(Endoglucanases，EG) 通過隨機方式水解纖維素鏈非結晶區，為纖維二糖水解酶產生新的末端，同時消除纖維二糖水解酶進一步水解纖維素的障礙 ；纖維二糖酶(Cellobiohydrolases，CBH I 和 CBH II) 分別從葡聚糖鏈的還原末端和非還原末端持續水解纖維素 ；葡萄糖苷酶(Glucosidase) 主要功能為水解葡萄糖苷鍵，釋放出葡萄糖作為產物。

厭 氧 微 生 物 分 解 纖 維 素 的 特 點 ：纖 維 小 體(cellulosome) 是厭氧生物存在的一種纖維素酶系，是多種纖維素酶、半纖維素酶依靠錨定 - 粘附機制形成的一種多酶複合體結構。

04

參與纖維素水解的非纖維素酶蛋白

非水解性質的解鏈因子或解氫鍵酶在天然纖維素的降解中具有重要作用，它們能打開纖維素鏈間和鏈內的氫鍵，使高度結晶的纖維素轉變為無序的非結晶形式，然後在 3 種纖維素酶組分的協同水解下被分解為葡萄糖或纖維糊精。

溶 多 糖 單 加 氧 酶 (lytic polysaccharide monooxygenases，LMPO)，推測 AA9 家族和部分 AA10家族的溶多糖單加氧酶通過氧化方式將結晶區和非結晶區

纖維素鏈斷裂。

伸展素和類伸展素因子蛋白 ( 如溶脹素 )，可破壞纖維素氫鍵，降低結晶度，增加纖維素酶可接近性。

同時溶脹素還能對木聚糖酶水解玉米芯的木聚糖具有協同作用。對這些蛋白的需要量有時等於甚至高於酶的用量，獲得大量的這些蛋白，缺乏一定的實用性。

這些往往是微生物本身具有的特質，野生菌固態發酵酶製劑更具優勢，而在基因工程菌的液態發酵中均未予以考慮。

青綠飼料一定會進入飼料配方。我們完全可以用生物的手段將青綠飼料進行處理，將來有可能會出現豬場做青貯。

2

澱粉

01

飼料的澱粉結構

澱粉顆粒是一種半結晶的聚合物，其包含結晶區和無定形區 (Hanxz 等，2001)。澱粉顆粒中形成結晶區的主要為支鏈澱粉，支鏈澱粉分子相互聚集形

成的雙螺旋結構是結晶的基礎。而直鏈澱粉是組成澱粉顆粒無定形區的主要成分 (Veelaert 等，1995)。

02

澱粉酶

03

澱粉的結構與可消化性

研究發現，澱粉顆粒越小，其比表面積越大，酶解效率越高 ；澱粉顆粒的消化速率係數與其尺寸大小呈負相關 ( 詹錦玲等，2017)。不同來源的澱粉在分子結構、晶型及與水結合形式等方面都不同，最終影響其消化速率，又將澱粉分為快速消化澱粉、慢速消化澱粉和抗性澱粉 (RS)。

抗性澱粉是由分子量較低、幾乎不含分支的直鏈澱粉分子組成。直鏈澱粉特別是分子量偏低的直鏈澱粉分子可形成良好有序的排布狀態 ( 類似結晶纖維素結構，靠分子間的氫鍵結合 )，可能是形成抗性澱粉顆粒結晶區的主要成分，從而體現出較高的抗消化性。週中凱 (2013) 證實了直鏈澱粉（特別是分子量偏低的直鏈澱粉分子）可形成良好有序的排布狀態，可能是形成抗性澱粉顆粒結晶區的主要

成分，從而體現出較高的抗消化性。這些鏈的形成主要是靠氫鍵，我一直也在思考是否有能打開氫鍵的酶，如果有這種酶，破壞很多化學結構就會變得更加容易，但是一直沒有找到。那麼，是否有其他能夠起到替代作用的酶，這是值得我們思考的問題。

另外，還有新玉米的問題。新玉米在成熟的過程中，有很多直鏈澱粉還沒有組裝到支鏈澱粉上，是以遊離的狀態存在，而且這些直鏈澱粉的分子量又不高，在制粒糊化的過程中，正好提供了一個重排的機會，可能會形成抗性澱粉。個人認為，解決新玉米問題還是應考慮其中碳水化合物的分子大小和結構。抗性澱粉可被後腸微生物降解產生揮發性脂肪酸，如何打開抗性澱粉分子間氫鍵，是否存在抗性澱粉酶？

玉米澱粉和蛋白質組成的組織構架，特別是醇溶蛋

白的含量，對玉米澱粉的消化率有較大影響。在進行預消化處理時，如何較好地運用蛋白酶、澱粉酶值得我們思考。

3

大豆蛋白

大豆蛋白根據生物學功能可分為 ：代謝和貯藏蛋白 ；根據溶解方式可分為 ：白蛋白 ( 易溶於水，佔 5%) 和球蛋白 ( 不易溶於水，佔 95%)。

沉降係數 ：大豆蛋白可分為 4 個組分，即 2S、7S、11S 和 15S。

抗原性蛋白包括大豆疏水蛋白、大豆殼蛋白、大豆

抑制蛋白、大豆空泡蛋白、大豆球蛋白和 β- 伴大豆球蛋白等。其中，大豆球蛋白和 β- 伴大豆球蛋白是大豆中免疫原性最強的兩種抗原蛋白，二者約佔大豆蛋白的70%。

01

大豆中的主要蛋白組分

02

大豆抗原

01

大豆球蛋白

分子量為 320~360 kDa，是一個環狀六聚體，具有 6 個酸性亞基 (A1~A6) 和 6 個鹼性亞基(B1~B6)，酸鹼亞基之間依靠二硫鍵 (A—S—S—B) 聯接，二硫鍵的鍵能很強，因此大豆蛋白的結構非常穩定 ( 見圖2)。β- 巰基乙醇等還原劑可將大豆球蛋白亞基和多肽分開。酸性亞基比鹼性亞基更容易被胃蛋白酶水解 (Lynch，1981)。

02

β- 伴大豆球蛋白

是分子量為 180 000 的三聚體，由 α、α' 和 β 3 個亞基組成，3 個亞基均含有 4%~5%碳水化合物是糖蛋白 ；且含硫氨基酸少，不含半胱氨酸，與球蛋白比的熱穩定性和凝膠能力也較差 ；α 亞基致敏性最強；對熱、酸、鹼的抵抗能力強，對胃蛋白酶不敏感 ( 見圖 3)。我推測，糖蛋白很可能是影響動物對其消化效果的一個重要因素。在微生物表達的過程中，酵母糖基化後就會很穩定，耐熱性會提高，因此糖蛋白可能與 β- 伴大豆球蛋白的穩定性有很大關聯。

03

大豆抗原的穩定性

大豆球蛋白和 β- 伴大豆球蛋白對仔豬消化道內酶的敏感程度存在差異，大豆球蛋白的酸性多肽鏈和 β- 伴大豆球蛋白的 α、α' 亞基易於被降解，而大豆球蛋白的鹼性多肽鏈和 β- 伴大豆球蛋白的 β 亞基不易被降解。β- 伴大豆球蛋白在胃蛋白酶、胰蛋白酶的連續消化下，產生大量抗酶解的肽段，其中有 3 條主要的抗酶解肽段，且均具有免疫活性 ( 王俊，2012)。研究表明，用不同的酶處理大豆蛋白的效果不同。鹼性蛋白酶和木瓜蛋白酶的效果明顯好於胰蛋白酶 ( 見圖 4)。

04

酶解產生的新肽段

各種酶水解大豆蛋白後會出現一些新的肽段 ( 楊國龍，2006)。利用胰蛋白酶水解大豆球蛋白會增加大豆蛋白的聚合。主要是球蛋白中的親水多肽將

疏水多肽包裹在蛋白質的內部。當親水多肽最先被水解後，則疏水多肽增加，且很容易聚合在一起 (BAS，2008)。Dreau 等 (1994) 研究表明，熱乙醇處理能顯著增加大豆抗原對胃蛋白醇和胰蛋白酶的敏感性。

4

胰蛋白酶抑制劑

01

Kunitz 型

Kunitz 型抑制劑是由 181 個氨基酸所組成

的單肽鏈，包含 2 個二硫鍵，分子質量約為 20 ku ；其過敏原性更強，是一種典型的絲氨酸蛋白酶抑制劑，主要抑制胰蛋白酶。

02

Bowman-Birk 型

BBI 是由 71 個氨基酸所組成的單肽鏈，包含 7 個二硫鍵，其分子質量約為 8 ku，是一種絲氨酸蛋白酶抑制劑，有 2 個具有不同特異性的功能性抑制

域，分別抑制胰凝乳蛋白酶和胰蛋白酶。BBTI 除了對胰蛋白酶有較強的抑制活性外，對於熱、酸的鈍化和胃蛋白酶的水解作用都具有很強的抗性。

03

胰蛋白酶抑制劑的抑制機制

兩種抑制劑的存在不改變胰蛋白酶的 Km 值，但其 V max 隨抑制劑濃度的增加而下降。表明其對胰蛋白酶的抑制作用是一種非競爭抑制作用 ( 黃惠華，2005)。I-KSTI 和 BBTI 對物理和化學致鈍作

用 ( 如超聲波場 ) 的抗性差別很大，這些差異的主要原因在於其分子結構上的差異。

5

大豆凝集素

大豆凝集素較其他豆科家族凝集素穩定，由四聚體降解為單體需要更高的能量，主要原因是大豆凝集素分子中的兩個單體一單體非典型界面間存在大量氫鍵 (Sinha 等，2007)。動物的消化酶中主要是胃蛋白酶對大豆凝集素有一定程度的降解作用，但需要一定時間，也與底物濃度密切相關。胰蛋白酶對完整的大豆凝集素幾乎不產生降解作用 ( 王利民，2007)。大豆凝集素與寡糖結合的特異性與組成凝集素糖結合區域的保守氨基酸殘基有關 (Rao 等，1998)。

6

玉米蛋白

根據傳統的奧斯本 - 門德爾 (Osborne-Mendel) 分

離法，清蛋白類 (albumins) 溶於水，加熱凝固，為強鹼、金屬鹽類或有機溶劑所沉澱，能被飽和硫酸銨鹽析 ；球蛋白類 (glubulins) 不溶於水，溶於中性鹽稀溶液，加熱凝固，為有機溶劑所沉澱，添加硫酸銨至半飽和狀態時則沉澱析出 ；醇溶蛋白類 (prolamins) 不溶於水及中性鹽溶液，可溶於 70%~90% 的乙醇溶液，也可溶於稀酸及稀鹼溶液，加熱凝固。該類蛋白質僅存在於穀物中，如

小麥醇溶蛋白 ；谷蛋白類 (glutelin) 不溶於水、中性鹽溶液及乙醇溶液中、但溶於稀酸及稀鹼溶液，加熱凝固。該蛋白也僅存在於穀類籽粒中，常與醇溶谷蛋白分佈在一起，典型的例子是小麥谷蛋白。不溶於水的蛋白用酶消化會有一些困難。

01

玉米蛋白的組成

玉米蛋白粉 (CGM) 的主要成分是玉米蛋白 (55%~65%)，其中玉米醇溶蛋白佔總蛋白的

65%~68%。玉米醇溶蛋白含有大量疏水性氨基酸和含硫氨基酸，如穀氨酸 (21%~26%)、亮氨酸 (20%)、脯氨酸(10%)、丙胺酸 (10%)，決定了它是一種疏水性很強的蛋白，不溶於水，溶於 60%~95% 的醇類水溶液中。但是它缺乏賴氨酸、天冬氨酸及色氨酸等酸性和鹼性氨基酸。

02

玉米醇溶蛋白的結構

玉米醇溶蛋白中 70% 的氨基酸殘基呈 α- 螺旋結構，13% 氨基酸殘基呈轉角結構，其餘的氨基酸殘基呈較鬆散、隨機的螺旋構型。多肽鏈間通過

次級鍵相互組合形成複雜的四級結構，且玉米醇溶蛋白質分子可進一步聚合成聚合體，結構複雜，高度緊密。玉米醇溶蛋白是高度緊密及疏水的“醇溶蛋白體”，蛋白結構比較穩定，對蛋白酶的“酶解進攻”具有很強的抵抗力。因此，玉米蛋白粉不宜過多添加。

03

玉米醇溶蛋白的改性

玉米醇溶蛋白難溶於水，懸浮在水溶液中與蛋白酶接觸的機率小。因此，傳統酶解方法存在產物轉化率低、酶解時間長，蛋白酶利用率低、產物

活性低等問題。必須對天然的玉米醇溶蛋白進行必要的預處理，使其由多肽鏈摺疊和盤曲構成特有的、緻密的結構變得鬆散，蛋白內部的酶切位點得以外露以利於與蛋白酶的結合。擠壓膨化技術可以改善玉米蛋白的吸水性、乳化性等物理性質，也可以改變蛋白質的空間構象，增加其對蛋白酶的敏感性。

7

小麥蛋白

01

小麥蛋白的組成

小麥中含有小麥麵筋蛋白質，約佔麵筋乾重的 85％以上，含有較多的疏水性氨基酸，分子內

疏水作用區域較大 ；其中主要是麥膠蛋白 ( 醇溶蛋白 ) 和麥谷蛋白 ( 谷蛋白 )，麥膠蛋白的含量約比麥谷蛋白少 10%。麥醇溶蛋白分子量小，分子呈球形，由於其分子中的 -S=S-都分佈於分子內部，因此為麵糰提供了延伸性 ；而麥谷蛋

白是由數條亞基通過分子間 -S=S- 交聯，形成的纖維狀大分子，使麥谷蛋白不易流動，為麵糰提供彈性。

02

小麥蛋白的結構

小麥蛋白中是麥谷蛋白和麥醇溶蛋白交織在一起，需要協同地處理兩種蛋白。不同的蛋白質結構對蛋白及氨基酸的消化率影響存在差異 ( 見表 3)。

8

挖掘植物蛋白潛質關鍵

①首先要打開分子間的二硫鍵，因為二硫鍵是維持蛋白質三級結構、四級結構最主要的化學鍵。然後破壞 α-螺旋和 β- 摺疊等空間結構，將其延展成一級結構。

②水解凝集素的關鍵保守區氨基酸肽鍵。

③球蛋白和凝集素等蛋白質中含有一定的糖，是否會對蛋白質的水解產生影響。

④破壞醇溶蛋白的疏水氨基酸的肽鍵 ( 角蛋白酶 )。

⑤選擇特殊微生物蛋白酶，進行協同應用。我們在試驗過程中發現，P-Trx 不僅能顯著促進 P-Ker水解羽毛速度 (8 h 羽小枝水解 )，且有助於 P-Ker 完全水解羽毛。在 P-Ker 和 P-Trx 兩種酶的協同作用下，天然羽毛的水解效果更好 ( 見圖 5)。同時，蛋白酶組合酶、胰蛋白酶和糜蛋白酶協同作用，會提高豆粕的水解效果(見表4)。

此外還發現，寡肽酶與角蛋白酶協同，會大大提高酪蛋白的水解程度。

第6則

-THE SIXTH-

飼料預消化與生物發酵的優劣比較

生物發酵的過程也是一個對飼料進行預消化處理的過程，但生物發酵與生物酶解優缺點各異。二者均可對飼料原料進行一定程度的預處理、預消化。微生物發酵還可以產生有機酸等有益的微生物代謝產物以及大量的有益菌，而生物酶解預消化不具有此功能。生物發酵穩定性和終點控制有一定的難度，因為大部分採用固態發酵，生物酶解比較容易掌握。生物發酵存在一定的養分消耗，生物酶解

預消化無養分消耗。生物發酵工藝較複雜，生物酶解預消化較為簡單，但均存在雜菌控制風險。深度預消化處理，適度生物發酵將是更好的選擇。還有一點值得注意的是，烘乾工藝如何選擇？預處理之後存在很多遊離氨基酸和遊離糖，應該如何控制美拉德反應，因此烘乾工藝是非常關鍵的問題。

第7則

-THE SEVENTH-

飼料生物加工的未來發展趨勢

1

清楚動物蛋白質消化的自然規律與現實訴求，探索動物消化內水解酶的不足 ( 量和種類 )。

2

圍繞動植物性飼料組成和結構做文章，重點關注不同原料的空間結構和化學鍵類別，制定相應的改性措施。

3

開發新型的微生物水解酶，消除多種抗營養因子的同時，提高不易和不可消化組分的潛在營養價值。

4

生物預消化的過程就是改性的過程，如何改性？改性哪些點？如何評價？均需配套的科學方法。

5

野生菌固態發酵酶製劑在未來的飼料預消化處理中存在巨大潛質，特別是在處理纖維等難以被單胃動物消化利用的飼料組分方面，可獲得更多可利用形式的養分。

6

生物預消化是一種全新的飼料加工模式，是飼料工業未來新的經濟增長點，但需要科學接受，更需要不斷科學地發展和完善。
