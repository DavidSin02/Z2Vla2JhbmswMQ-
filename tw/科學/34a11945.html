常用除草劑安全使用技術，百合科菜田除草劑的安全使用技術

2020-11-14
| 科學

近年由於社會經濟的發展和生活水平的提高，促使城市郊區和一些特種蔬菜產區的蔬菜集約化生產發展迅速，種植面積不斷擴大。據統計，全國每年種植蔬菜面積630多萬公頃。

我國的蔬菜種植，以氣候、地理差異，大致可劃分為4種類型、7個區域。

1.單主作類型（無霜期90～165d）

（1）東北鬆嫩流域區，包括吉林、黑龍江和內蒙古大興安嶺以東。

（2）青藏高原區，包括青海、西藏、四川西北部和新疆部分地區。

（3）蒙新高原區，包括內蒙古中西部、甘肅北部、新疆大部。

2.雙主作類型（無霜期165～240d）

華北及東北黃遼流域區，包括遼寧、河北、天津、北京、山東、河南、山西、陝西及甘肅南部。

3.三主作類型（無霜期240～340d）

中南部江淮流域區，包括江蘇、上海、浙江、江西、湖南、湖北、安徽、陝西漢中盆地、四川、貴州及廣西、廣東、福建的北部。

4.多主作類型（全年基本無霜）

（1）華南丘陵區，包括廣西、廣東、福建的大部、雲南南部、海南和臺灣。

（2）昌滇山谷區，包括雲南部分地區和昌都南部。

由於各蔬菜種植區的地理位置、土壤及氣候條件、栽培方式和耕作制度的不同，導致伴生在菜田的雜草也有明顯差異。概括起來，分佈較為廣泛的主要雜草有稗、馬唐、牛筋草、狗尾草、千金子、雀麥草、畫眉草、早熟禾、看麥娘、雙穗雀稗、狗牙根、藜、蓼、莧、萹蓄、馬齒莧、繁縷、牛繁縷、鐵莧菜、通泉草、雀舌草、薺、風花菜、漆姑草、附地菜、婆婆納、豬殃殃、香附子、三稜草、龍葵、蒼耳、田旋花、車前、苣蕒菜、苣蕒菜、加拿大蓬、空心蓮子草等。

在蔬菜田推廣化學除草，比人工和機械除草經濟、省工、徹底。但由於蔬菜種植種類多、面積小、週期短、換茬快、栽培方式複雜，要求選擇與使用除草劑，必須嚴格依據蔬菜的種類、種植時期、栽培方式、土壤質地、水肥管理不同等多種因素綜合考慮，既要考慮對當茬雜草的防除作用，又要考慮殘留對後茬作物有無延續藥害。

一、百合科菜田除草劑的安全使用技術

百合科蔬菜主要指大蒜、洋蔥、韭菜、香蔥和蘆筍等五種。

（一）大蒜田除草劑的安全使用技術

大蒜是我國主要的出口創匯蔬菜之一，在全國各地均有分佈。大蒜生育期長，從播種到收穫為240d左右，雜草為害十分嚴重。據調查，大蒜田常見的雜草種類有看麥娘、硬草、棒頭草、檾麻、日本看麥娘、早熟禾、狗尾草、馬唐、牛筋草、幹金子、畫眉草、牛繁縷、繁縷、豬殃殃、婆婆納、蒼耳、通泉草、碎米薺、泥胡菜、小藜、馬齒莧、凹頭莧、刺莧、一年蓬、小飛蓬等，雜草已成為影響大蒜產量、質量以及出口率的主要障礙。

（1）敵草胺

防治對象　稗草、馬唐、狗尾草、野燕麥等一年生禾本科雜草及某些闊葉雜草如藜、豬殃殃、馬齒莧、苣蕒菜等。


使用技術　播後苗前土壤處理，每公頃用有效量750～1125g。

注意事項　土壤表面溼潤有利於提高防除效果。播種後應儘早施藥。

（2）乙草胺


使用技術　播後苗前土壤處理，每公頃用有效量562.5～750g。

（3）二甲戊靈

防治對象　稗草、馬唐、狗尾草等禾本科雜草及藜、反枝莧等闊葉雜草。

注意事項　藥後混土可提高防除效果。

（4）草酮


防治對象　稗等一年生禾本科雜草和藜、田旋花等闊葉雜草。

使用技術　播後苗前至立針期土壤處理，每公頃用有效量375～562.5g。

注意事項　地膜大蒜要降低用量，使用262.5～337.5g/畝。


防治對象　龍葵、蒼耳、藜、馬齒莧、柳葉刺蓼、酸模葉蓼、繁縷、檾麻、反枝莧、凹頭莧、錦葵、千里光、辣子草、看麥娘等。

使用技術　大蒜播後芽前到立針期或大蒜2葉1心到4葉期，雜草4葉期以下土壤處理，每公頃用有效量176.25～211.5g。

（6）精喹禾靈


防治對象　一年生禾本科雜草，如稗草、狗尾草等。提高劑量時，對多年生雜草，如蘆葦、狗牙根等也有效。

使用技術　大蒜生長期、禾本科雜草3～5葉期時對準雜草莖葉噴霧，每公頃用有效量37.5～52.5g。

注意事項　①儘量避免藥液霧滴沾到蒜苗的莖葉上。②低溫時用藥防效差。

（7）氯氟吡氧乙酸


使用技術　大蒜生長期，闊葉雜草2～5葉期時對準雜草莖葉噴霧，每公頃用有效量150～300g。

（二）洋蔥田除草劑的安全使用技術

1.直播或育苗洋蔥除草劑的安全使用技術

（1）二甲戊靈（除草通）

防治對象　稗草、馬唐、狗尾草等禾本科雜草及藜、反枝莧等闊葉雜草。

使用技術　播後苗前土壤處理，每公頃用有效量742.5～990g。

注意事項　沙土應適當減少用藥量。溼度大有利於提高防除效果。

（2）撲草淨

防治對象　一年生禾本科、莎草科雜草和闊葉雜草。

使用技術　播後苗前土壤處理，每公頃用有效量562.5g。

注意事項　撲草淨在沙質土上不能用。

（3）仲丁靈（地樂胺）

防治對象　稗草、狗尾草、馬唐、牛筋草、反枝莧、皺果莧、馬齒莧等。

使用技術　播後苗前土壤處理，每公頃用有效量1440～1875g。

（4）乙氧氟草醚（果爾）

防治對象　龍葵、蒼耳、藜、馬齒莧、柳葉刺蓼、酸模葉蓼、繁縷、檾麻、反枝莧、凹頭莧、錦葵、千里光、辣子草、看麥娘等。

使用技術　洋蔥3葉期后土壤處理，每公頃用有效量141～176.25g。

注意事項　洋蔥3葉期前施藥，葉片上可能會出現斑點，但不影響最終產量。

（5）滅草鬆（苯達鬆）

防治對象　蒼耳、莧、蓼、馬齒莧、豬毛菜、豬殃殃、藜、鴨跖草、龍葵、檾麻等多種闊葉雜草。

使用技術　洋蔥3葉期後對準雜草莖葉噴霧，每公頃用有效量720～1440g。

注意事項　洋蔥2葉期前用藥，會產生藥害，造成洋蔥葉尖乾枯，應予避免。

（6）精喹禾靈

防治對象　一年生禾本科雜草，如稗草、狗尾草、馬唐、野燕麥等。提高劑量時，對多年生雜草，如蘆葦、狗牙根等也有效。

使用技術　禾本科雜草3～5葉期時對準雜草莖葉噴霧，每公頃用有效量22.5～37.5g。

2.移栽洋蔥田除草劑的安全使用技術

（1）二甲戊靈

防治對象　稗草、馬唐、狗尾草、牛筋草、早熟禾等禾本科雜草及藜、反枝莧等闊葉雜草。

使用技術　移栽前土壤處理，每公頃用有效量990～1237.5g。

注意事項　沙土應適當減少用藥量。溼度大有利於提高防除效果。

（2）撲草淨

防治對象　一年生禾本科、莎草科雜草和闊葉雜草。

使用技術　移栽前土壤處理，每公頃用有效量562.5～750g。

注意事項　撲草淨在沙質土上不能用。

（3）地樂胺

防治對象　稗草、狗尾草、馬唐、牛筋草、反枝莧、皺果莧、馬齒莧等。

使用技術　移栽前土壤處理，每公頃用有效量1440g。

（4）乙氧氟草醚

防治對象　龍葵、蒼耳、藜、馬齒莧、柳葉刺蓼、酸模葉蓼、繁縷、檾麻、反枝莧、凹頭莧、錦葵、千里光、辣子草、看麥娘等。

使用技術　移栽後6～10d土壤處理，每公頃用有效量176.25～264.375g。

（5）草酮

防治對象　稗等一年生禾本科雜草和藜、田旋花等闊葉雜草。

使用技術　播後苗前至立針期土壤處理，每公頃用有效量375～562.5g。

注意事項　整地後應及早施藥，移栽時儘量不要翻動土層。

（6）滅草鬆

防治對象　蒼耳、莧、蓼、馬齒莧、豬毛菜、豬殃殃、藜、鴨跖草、龍葵、檾麻等多種闊葉雜草。

使用技術　洋蔥3葉期後對準雜草莖葉噴霧，每公頃用有效量720～1080g。

注意事項　洋蔥2葉期前用藥，會產生藥害，造成洋蔥葉尖乾枯，應予避免。

（7）精喹禾靈

防治對象　一年生禾本科雜草，如稗草、狗尾草等。提高劑量時，對多年生雜草，如蘆葦、狗牙根等也有效。

使用技術　禾本科雜草3～5葉期時對準雜草莖葉噴霧，每公頃用有效量37.5～52.5g。

注意事項　低溫時不宜使用本劑。

（三）韭菜田除草劑的安全使用技術

韭菜栽培有育苗韭菜和老根韭菜之分，使用除草劑時必須特別注意這一區別。

1.育苗韭菜田除草劑的安全使用技術

育苗韭菜，因前期生長緩慢，受雜草為害程度較重、時間較長。採用化學除草，一般要在播後苗前和苗後進行兩次。

使用技術　播後苗前土壤處理，每公頃用有效量495～990g。


注意事項　沙性土壤用低劑量。播種時避免露籽。


防治對象　一年生禾本科雜草，如稗草、野燕麥、看麥娘等。提高劑量時，對多年生雜草，如蘆葦、狗牙根等也有效。


（3）烯禾定

防治對象　一年生禾本科雜草，如稗草、野燕麥等。提高劑量時，對多年生雜草，如白茅、匍匐冰草、狗牙根等也有效。

使用技術　禾本科雜草3～5葉期時對準雜草莖葉噴霧，每公頃用有效量93.75～140.625g。


注意事項　①施藥時嚴禁藥液飄移到鄰近的禾本科作物上。②空氣溼度大可以提高防效，故以早晚施藥較好。

2.老根韭菜田除草劑的安全使用技術

老根韭菜，採取化學除草是在每次收割後，先鬆土和清除田間大草，待韭菜傷口癒合後長出新葉時澆一次水，然後將除草劑噴灑於土表。

（1）氟樂靈


使用技術　老根韭菜每次收割傷口癒合後，在韭菜行間定向噴霧，每公頃用有效量720～1080g

注意事項　新植韭菜慎用。

（2）二甲戊靈

防治對象　稗草、馬唐、狗尾草等禾本科雜草及藜、反枝莧等闊葉雜草。

使用技術　老根韭菜貼地收割後出土前土壤處理，每公頃用有效量495～742.5g。

（3）乙氧氟草醚（果爾）


防治對象　龍葵、蒼耳、藜、馬齒莧、柳葉刺蓼、酸模葉蓼、繁縷、檾麻、反枝莧、凹頭莧、錦葵、千里光、辣子草、看麥娘等。

使用技術　老根韭菜貼地收割後出土前土壤處理，每公頃用有效量176.25～264.375g。

（四）香蔥田除草劑的安全使用技術

香蔥有直播小蔥、移栽溝蔥、根茬蔥。

（1）二甲戊靈（除草通）

防治對象　稗草、馬唐、狗尾草、牛筋草、早熟禾等禾本科雜草及藜、反枝莧等闊葉雜草。

使用技術　香蔥播後苗前、移栽前或根茬蔥返青前土壤處理，每公頃用有效量495～742.5g。

注意事項　①沙土應適當減少用藥量。②溼度大有利於提高防除效果。

（2）敵草胺

防治對象　一年生禾本科雜草如稗草、馬唐、狗尾草等及一些闊葉雜草如藜、豬殃殃、馬齒莧、苣蕒菜等。

使用技術　香蔥播後苗前、移栽前或根茬蔥返青前土壤處理，每公頃用有效量750～937.5g。

（3）精吡氟禾草靈（精穩殺得）、精氟吡甲禾靈（高效蓋草能）、精喹禾靈（精禾草克）

防治對象　一年生禾本科雜草，如稗草、野燕麥、看麥娘等。提高劑量時，對多年生雜草，如蘆葦、狗牙根等也有效。

使用技術　禾本科雜草3～5葉期時對準雜草莖葉噴霧，每公頃用有效量精吡氟禾草靈112.5～157.5g，精氟吡甲禾靈40.5～64.8g，精喹禾靈37.5～56.25g。

注意事項　①天氣乾旱或草齡較大時應增加藥量。②若噴藥時土壤乾燥，可適當增加藥量，或在灌水後施藥，以提高其防除效果。③施藥時注意風速、風向，不要使藥液飄移到附近禾本科作物田，以免造成藥害。④在禾本科和闊葉雜草混生的田塊，可與防除闊葉雜草的除草劑搭配使用。

（4）烯禾定（拿捕淨）

防治對象　一年生禾本科雜草，如稗草、野燕麥等。提高劑量時，對多年生雜草，如白茅、匍匐冰草、狗牙根等也有效。

使用技術　禾本科雜草3～5葉期時對準雜草莖葉噴霧，每公頃用有效量93.75～140.625g。

注意事項　①施藥時嚴禁藥液飄移到鄰近的禾本科作物上。②空氣溼度大可以提高防效，故以早晚施藥較好。

（五）蘆筍田化學除草技術

蘆筍為多年生宿根性草本植物，大壟栽培，取食嫩芽。

（1）嗪草酮

防治對象　一年生闊葉雜草為主，如反枝莧、藜、蓼、繁縷等。提高施用劑量可防除鐵莧菜、水棘針、檾麻、卷莖蓼等。

使用技術　早春或蘆筍收割後雜草出苗前土壤處理，每公頃用有效量375～562.5g。

注意事項　賽克津的用量，視土壤質地、有機質含量及氣溫而增減，在有機質低於2%的沙質土上不宜用。

（2）敵草胺

防治對象　稗草、馬唐、狗尾草等一年生禾本科雜草及某些闊葉雜草如藜、豬殃殃、馬齒莧等。

使用技術　早春或蘆筍收割後雜草出苗前土壤處理，每公頃用有效量750～1125g。

（3）莠去津（阿特拉津）

防治對象　一年生闊葉雜草和禾本科雜草，如馬齒莧、反枝莧、龍葵、稗草、狗尾草、牛筋草等。

使用技術　蘆筍嫩莖出土前，雜草出苗前土壤兼莖葉處理，每公頃用有效量1140～1425g。

注意事項　蘆筍嫩莖出土期間不能施藥。

（4）麥草畏

防治對象　多種闊葉雜草，如藜、蓼、莧、菊科雜草及問荊、田旋花等。對闊葉小灌木也有防效。

使用技術　闊葉雜草2～4葉期時對準雜草莖葉噴霧，每公頃用有效量144～288g。

注意事項　施藥時應降低噴頭高度，進行行間定向噴霧，避免藥液直接接觸蘆筍。
