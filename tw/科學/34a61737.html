物理層

物理層的基本功能-P58

  1. 向數據鏈路層提供比特流傳輸服務。
  2. 根據所使用傳輸介質的不同，制定相應的物理層協議，規定數據信號編碼方式、傳輸速率，以及相關的通信參數。
  3. 屏蔽物理層所採用的傳輸介質，通信設備與通信技術的差異性。

信息、數據、信號間的關係-P62

信 息 -> 數 據（eg：01串） -> 信 號（eg：光電信號）

串行通信和並行通信-P64

  • 串行通信：一個字符的二進制代碼按由低位到高位的順序依次發送。
  • 並行通信：一個字符的8位二進制代碼同時通過8條並行的通信信道發送，每次發送一個字符代碼。

單工、半雙工、雙工通信-P64

  • 單工：信號只能向一個方向傳輸，任何時候都不能改變信號的傳送方向。
  • 半雙工：信號可以雙向傳送，但是必須交替進行，一個時間只能向一個方向傳送。
  • 雙工：信號可以同時雙向傳送。

同步技術-P65

  • 位同步
  • 字符同步

常用的傳輸介質-P66

  • 雙絞線
  • 同軸電纜
  • 光纖
  • 無線
  • 衛星通信

製作雙絞線時直通線和交叉線的線序-實驗冊

標準[568A]：綠白-1，綠-2，橙白-3，藍-4，藍白-5，橙-6，棕白-7，棕-8；

標準[568B]：橙白-1，橙-2，綠白-3，藍-4，藍白-5，綠-6，棕白-7，棕-8

  • 直通線：兩頭都是[568A]或者兩頭都是[568B]，（兩頭相同，用於連接電腦和交換機和交換機之間的連接）
  • 交叉線：一頭[568A]一頭[568B] ，（兩頭不同，用於兩臺計算機直連或者兩個Hub 和 uplink口相連）

光纖通信的基本原理-P67

光纖通過內部的全反射來傳輸一束通過編碼的光信號。

單模光纖與多模光纖的區別-P68

多模光纖：光信號與光纖軸成多個可分辨角度的多路光載波傳輸。（反射多次，粗，我們用的，光源為發光二極管）

單模光纖：光信號僅與光纖軸成單個可分辨角度的多路光載波傳輸。（近似直射，細，適合遠距離傳輸，光源為激光二極管，貴）

模擬數據的編碼方法-P75

  • 振幅鍵控
  • 移頻鍵控
  • 移相鍵控

波特率的概念-P76

波特率（調製速率）描述的是碼元傳輸的速率（所以又被稱為碼元傳輸速率），單位是波特（Baud）等價於，碼元/s。

PS：碼元個數也可以被稱為脈衝個數或者信號變化的次數

波特率與比特率的關係-P76

比特率：單位時間內數字通信系統所傳輸的 二進制碼元 個數，單位bps等價於bit/s。

$$S = B log_2k$$比特率 = 波特率 X 單個調製狀態對應二進制位數k*

PS:在二進制碼元情況下，比特率在數值上等於波特率。

eg:一個碼元攜 n bit信息量，則 M Baud的碼元傳輸速率 <=> M × n bit/s

PS: 相數 <=> 進制數

脈衝編碼調製（PCM）的過程-P78

是模擬數據數字化的主要方法。

  • 採樣
  • 量化
  • 編碼

香農定理的基本內容-P81

PS：信息論裡面也講了

香農公式：$$C_{max} = W × log_2(1+S/N)$$信噪比$$（dB）= 10log_{10}(S/N)$$

W為信道帶寬，所以要提高最大數據傳輸速率，就應該設法提高傳輸線路的帶寬或提高所傳信號的信噪比。

多路複用技術的分類及各自特點-P82

  1. 時分多路複用（TDM）
  2. 頻分多路複用（FDM）
  3. 波分多路複用（WDM）
  4. 碼分多路複用（碼分多址CDMA）（與正交頻分複用）

ADSL的特點-P89

  1. ADSL在電話線上同時提供電話與Internet接入服務
  2. ADSL提供的非對稱帶寬特性
  3. ADSL結構

HFC的特點-P91

  1. HFC下行信道與上行信道頻段劃分有多種方案（既有帶寬相同的對稱結構又有非對稱結構）
  2. 用戶端的電視機與計算機分別接到線纜調制解調器（下行有線電視信道傳輸的電視節目，下行數據信道傳輸數據，上行數據信道傳輸的數據傳輸到頭端）
  3. HFC的頭端又被稱為電纜調制解調器終端系統。
  4. 小區光纖結點將光纖幹線和同軸電纜相互連接。
  5. HFC對上行信道與下行信道的管理是不相同的。
