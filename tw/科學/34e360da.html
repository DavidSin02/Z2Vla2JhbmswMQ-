吃遍魔都五大老牌便利店，才敢告訴你什麼值得買！

2020-11-14
| 科學

當小資君每次不停搜刮“蘿”森、全“佳”

“seven”-11、喜“事”多，魔都四大便利店時...

內心就會聽見四大便利店們的吶喊：“感覺被掏空！”

當媽媽在燒飯時說：

“屋裡廂醬油沒了，到對面快客幫我買1包回來！”

當每次便利店推文一發送，粉絲的留言裡總會有

“怎麼沒有好德/可的/快客..."等諸如此類的評論

小資君：是呀，什麼時候我們只顧著去

那些自帶日系風光環，擁有無敵可愛包裝

以治癒孤獨都市人的新24小時便利店

卻忘記了那些老牌便利店啊

其實它們才是和我們一起長大的24小時便利店

只是，那時的我們還不懂

那麼，今天就和小資君一起

會會昔日的5位老朋友吧~（慚愧臉）

快客，可的，好德，良友，家樂福easy

不仔細逛不知道

其實好吃的也很多哦

絕對和新便利們有得一拼

親民的美食list，有點長長長長長

但，還是迫不得待地想安利給小夥伴們啊~

/ 快客 /

加分小亮點：自營小零食系列，麵包甜品

因為是百聯集團旗下的，店內各種日本、韓國、泰國的進口零食，飲料也不少哦~但令小資君驚訝的有以下兩點~

不是每家店都有共享按摩椅的哦！

①：店內居然還有共享按摩椅，想象著一邊吃吃零食，一邊花點小錢按摩一下，感覺這樣的治癒感更實際哇~（畢竟身體能感受得到~）

②：快客的自營零食系列，完全不輸全“佳”的呀米將、蘿”森和“seven”-11的自營系列零食，銅板的價格也是喜人，近三四十種商品看得小資君統統都想帶回家呀！

小Q巧克力威化/2.5元

脆脆威化裡夾著奶油般夢幻感的巧克力，清甜不膩~咖滋咖滋的美味根本停不下嘴！外加每包只要2.5元，1包裡有10片，幸福感更是抑制不住的upupup~

小Q芝麻多多、蔥多多/3.8元

蔥多多簡直就是椒鹽版的桃酥餅，鹹香可口~芝麻多多是甜味的，芝麻味很濃郁啊~兩款都是酥酥的口感，油膩感不強，老少皆宜~

小Q原味米果 4.5元

小資君的大愛，原味米果裡有柿種和花生，55g的分量真的是很感人，香香脆脆的一包夠小資君吃3天了~5塊錢就可以擁有啦~一般店家都要10元左右了，關鍵份量也很“羞澀”啊~

小Q香辣小黃魚/5.5元

鮮辣入味，魚肉吃起來並不是很乾，帶著絲絲嚼勁，會讓人越吃越有味，如果再來一杯啤酒，就更愜意了~彷彿就像是回到了1988年的雙門洞，媽媽們啃著魚乾的小日子裡去了~

小Q手工蛋糕片/4.5元

簡直就和越南雞蛋餅乾的味道如出一轍~蛋香四溢，酥脆的不要不要的~喜歡哦~

小Q沙琪瑪/4.5元

老少皆宜的沙琪瑪，小包包裡的墊飢小零食必定有它~快客自營的沙琪瑪蛋香味很足，甜酥酥噠，不粘牙~

香蕉奶油麵包/7.5元

說實話小資君被香蕉味的延伸想象力驚到了！小夥們一定要治癒一下，鬆軟如歐包的外衣包裹著綿綿的香蕉奶油，這個味道簡直就是的麵包版的香、蕉、先、生！

草莓慕斯蛋糕/10.8元

甜品包裝普通，但味道其實一點也不亞於魔都四大便利店，巧克力慕斯、布朗尼都很濃郁噠！但是夏天麼，還是推薦一款清爽的草莓慕斯，奶油輕盈不膩，層次感也很強~當辦公室下午茶甜品真是綽綽有餘啦~

燕麥牛肉三明治/14.8元

枕頭形三明治裡夾著蔬菜和牛肉，在其他便利店基本很少見吧~蓬鬆的口感間帶著實實在在的滿足感~配上燕麥粒，讓咀嚼都充滿著元氣~

/ 良友 /

加分小亮點：老牌新品雪糕、阿波羅冷飲系列，1~4元小甜點

因為和光明合併了，所有一些光明牌新出的冷飲都可以在這裡找到，冷櫃的雪糕價格區間很大，兩三塊的都有（畢竟在四大便利店裡，雪糕價格都是5元起的，錢包供不起啊~），如果是光明控的小夥伴不妨可以到這裡來看看的~

光明熊小白芝士雪糕/8元

第一口就喜歡上的味道，是鮮牛乳做的，雖然容易化，但口感特別醇厚，帶著微微的鹹，舔上去就像在吃奶蓋，也會有幸福的白鬍子~把它泡在冷萃裡也非常贊哦~

阿波羅雪泥球/7.5元

小資君從小吃到大的，這顆橙色球出現在老牌便利店的概率很高的，橙子味很清甜，挖著吃冰冰爽爽的~就像在吃雪芭~

阿波羅楊枝甘露/6元

因為阿波羅這個牌子主要走的是香港風味，楊枝甘露味的雪糕是一定要舔一根的~清爽味甜，柚子味很清新~化在嘴裡就是楊枝甘露！

悅家3D健鈣芒果酸奶/3元

包裝很有童年感，8090小時候基本都吸過這樣的酸奶，只不過價格從原來的1元變到如今3元，價格依舊親民，味道也沒變~酸甜順滑，沒有你要的醇厚感，卻有著回憶感~

蝴蝶酥（1元/塊）

娜蕾多（2元/塊）

蜜蕾方塊酥（3.5元/塊）

放在收銀臺前的老式小點心，一包一個，特別可愛~價格在1~4元間不等。蝴蝶酥的黃油味很濃，蜜蕾方塊酥就像杏仁排，雖然比不上哈爾濱食品廠，但味道也是可圈可點的~

/ 可的 /

加分小亮點：新上市的國產飲料，阿波羅冷飲系列、八喜甜筒

一般都開在老式居民區附近，以服務周邊居民日常需求為主，雖然店面普遍都不大，但日用品，油鹽醬醋，大米精製油、醬菜蘿蔔之類的，總是佔有一席之地~為了與時俱進，店內也有盒飯，飯糰，甜點之類食物~味道也是可以的~

八喜香草甜筒/6.5元

八喜的雪糕杯見得很多，甜筒瞭解一下吧~也是一款時常出沒在老牌便利店裡的冷飲~待微微融化後，頂部的冰激凌就像是奶油蛋糕上的鮮奶油一樣輕盈~沒如“卡哇伊”多甜筒一樣有很多巧克力碎之類的裝飾~香草的味道很純很乾淨~同款巧克力味甜筒也是如此~

雀巢摩爵扁桃仁香草雪糕/9.2元

一款比星星雨更有滿足感的脆皮巧克力雪糕，外加雀巢牛奶品質就不錯，奶味很香醇~一口咬下即是脆皮的“咖滋”聲，聽起來特別性感( ω )~巧克力味濃郁，卻沒有甜膩感，贊~真心覺得比“某龍”好吃多了~

阿力達百香果棒冰/4元

咬上去就像在吃綿綿雪，百香果的酸甜氣息存在感非常強，時不時就會吃到調皮的小黑籽~

喜盈門香蕉牛奶/5元（優惠價）

幾乎和韓國某香蕉牛奶味道一模一樣~做作的香精味相對還少了一些，香蕉味喝起來很溫柔~關鍵只賣5元/瓶哇~價格是韓國某香蕉牛奶的一半哇~p.s:同品牌的草莓牛奶，也不錯哦~

乳小菌白桃味乳酸菌/6.8元

包裝很可愛(*▽*)，在老牌便利店也算是少見了吧~味道和小時候喝的樂百氏很像，清清爽爽的酸甜，混合著白桃的味道，很夏天~

BISCOLATA巧克力/5元

像是巧克力軟餅或是蛋糕~巧克力一吃就知道品質不錯，甜而不膩，不會黏在舌腔上，結果一看成分表果然如此，32%的黑巧克力，不含代可可脂哦~

芒果菠蘿班戟9.5/元

可以和街頭的港式小店媲美的，蛋皮很薄，奶油綿柔輕盈，配上內餡實實在在的芒果、菠蘿果肉，香甜又清爽~

/ 好德 /

加分小亮點：阿波羅冷飲系列、八喜甜筒

好德和可的風格沒差別，都是農工商旗下的24小時便利店，精打細算的阿姨媽媽們是普遍消費人群，所以店內基本都是一些樂事、好麗友、上好佳、天喔...之類的零食比較多，偶然會有一些不常見的牌子會出現，味道其實不差的，但總莫名感覺有些山寨（小資君：對不起，我不應該以貌取物~）

卡士達布丁蛋糕/6.2元

包裝很少女心，味道也很少女心~外層是鬆軟的麵包“羽絨服”，內層的底部是綿柔的蛋糕胚，頂部是一坨可愛的卡仕達奶油~充滿著甜蜜的層次感~

欣欣長崎牛奶蛋糕/4.9元

味道很日本某牌的蛋糕也很像啊，綿軟的蛋糕胚裡帶著絲絲蜂蜜般的香甜~一包裡有2個獨立的小包裝~方便攜帶~

桃李雪之語/4.2元

桃李的麵包一向都是很柔軟~內餡夾著草莓醬和奶油，每一口都充滿著蓬鬆的清甜感~

名仁玫瑰蘇打/4元

帶著絲絲鄉土氣息包裝的蘇打飲料，蘇打的氣泡感完全不存在，但這反倒是成就了玫瑰味的清雅微甜~透明色的飲料水，如果換個日式包裝的話，透明飲料界的C位之爭，應該也有它的一席之地吧！（小資君：國貨當自強啊~）

優得宇治抹茶爆米花/8.5元

勝在味道~在各種原味，焦糖味的爆米花世界裡，終於有抹茶控的一片天空了~“大風吹~大風吹~抹茶爆米花好美~”

/ 家樂福easy /

加分小亮點：進口零食/飲品/調味品，三明治等簡餐

其實家樂福easy不算是老牌，但大賣場般的家樂福絕對是老牌了，只是小時候愛和媽媽逛家樂福的小資君，早已投奔便利店的懷抱了！

同時在B2C，O2O的購物方式的激烈競爭下，家樂福君也在不停地尋求著突破吧，easy算是其中一個改變吧~品類很全，就像一間迷你版的家樂福大賣場，基本可以秒殺掉其他便利店！

夏威夷菠蘿火腿卷/16.8元

“明天我還要去買~”比起新四大便利店售賣的各種癟癟的捲餅，家樂福的料簡直就是plus級別的，卷皮也很緊實Q彈啊，咬下的每一口都非常的有滿足感~冷熱即可食用~

不過小資君更喜歡吃熱的，在平底鍋裡稍稍烘一下，卷皮會格外香脆~再配個好看的盤子，可以PKwagas家的各種捲餅~XXXXD

尼斯輕食三明治/6.5元

小資君編輯部小夥伴推薦的，吐司很鬆軟，內餡的料也是足足的，吃起來很清爽。降解材質的包裝盒也是小資君喜歡它的原因之一，方便攜帶，不會像新四大便利店的三明治，稍稍一壓就變成了一塊“餅”-_-||

烤腸/3.5元

應該算是店家的人氣小吃了，小時候的臺灣香腸味，一口咬下去油潤潤的，滿是肉香味，讓小資君恨不得往嘴裡再放幾粒蒜，那樣就口口爆香啦~完全不像四大新便利店的烤腸，帶著煙燻的氣息~

清美綠豆湯/3.8元

夏天必備的綠豆湯~清美老牌子了，有信任感~湯汁涼爽清甜，綠豆的量也很多，酥酥潤潤好是飽滿呢~另外比心一下包裝，比起四大新便利店碗形的湯/粥包裝，這裡的杯形包裝更容易食用，一根吸管就夠啦~

勞侖茲番茄味笑臉的薯片/16.5元

顏值高又好吃的薯片，小資君今年追劇的指定薯片就是它啦~可愛的笑臉簡直就是薯片界的“鬼臉嘟嘟”讓人愛不釋手，口感帶著鬆脆的空氣感（一般薯片君做不到的），甜甜的番茄味也完全滲透在其每個“小細胞”裡了~

/ 魔都老牌便利店vs魔都四大便利店 /

魔都老牌便利店

店員基本都是阿姨店員，

眼神裡帶著”看破紅塵“之感

不說話的樣子比較冷漠，有時說活又比較凶

偶爾會和附近居民嘎嘎三湖，這時比較親切

不太會主動給收銀條，怕給自己找麻煩

店內油鹽醬醋味、洗衣粉香皂味很重，因為真的很多

夏天售賣的飲品主要是各種礦泉水，鹽汽水，酸梅汁

冷櫃裡的乳製品也基本以光明、伊利，蒙牛為主

...

魔都四大便利店

店員基本都是小年輕

眼神裡帶著對生活的滿腔熱情，

態度基本很有禮貌

很會“做生意”，什麼2件八折，

送500積分，＋3元得酸奶之類的，可有一套了

會很主動給收銀條，但小年輕顧客們基本都不要

一進店就是鮮鮮的關東煮味和香甜的冰激凌味

夏天售賣的飲品是各種網紅汽水，咖啡，茶飲

冷櫃裡的乳製品，品牌多到有些都沒聽過

連口味也是千變萬化的~

在好吃的list中推薦幾款必嘗款
