手把手教你在Linux系統終端輸出的內容如何保存到文件裡面

2020-10-29
| 科學



在Linux，macOS，* BSD或類似Unix的操作系統上使用BASH / KSH / CSH / TCSH時，如何將終端輸出保存到文件中？

>標準輸出

2>錯誤輸出

<標準輸入

如何將終端輸出保存到文件

使用>標準輸出符號將輸出信息導入time.txt文件，例如：

[root@localhost ~]# date > time.txt[root@localhost ~]# cat time.txtTue May 19 14:09:57 CST 2020

<以上代碼可複製粘貼，可往左滑>

將內容輸入重定向到命令中

使用<標準輸入符號將文件輸入到命令匯中，例如：

[root@localhost ~]# cat < /etc/passwd[root@localhost ~]# wc -l < /etc/passwd

<以上代碼可複製粘貼，可往左滑>

將輸出內容追加到文件

使用>會覆蓋文件，使用>>會文件內容後面追加內容。例如：

[root@localhost ~]# echo '------------' >> time.txt 

<以上代碼可複製粘貼，可往左滑>

將錯誤輸出到一個文件

例如，將錯誤信息輸出到err.log，正確的結果輸出的info.txt裡面，可以這樣做：

[bpang@localhost ~]$ find / -iname "*.conf" 2> err.log > info.txt

<以上代碼可複製粘貼，可往左滑>

例如，將錯誤信息和正確的結果放在同一個文件裡面，可以這樣做：

[bpang@localhost ~]$ find / -iname "*.conf" &> info.txt # 或者[bpang@localhost ~]$ find / -iname "*.conf" > info.txt 2>&1

<以上代碼可複製粘貼，可往左滑>
