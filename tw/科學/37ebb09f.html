NTR-032、MGB-007——為什麼任天堂設備代碼像老司機車牌號

2020-11-14
| 科學

題圖及圖1來自掌機收藏家源大頭的頭很大親自提供的高清無碼大圖，更多掌機收藏請戳他的微博，在此特別感謝！

翻開你手上任天堂主機、掌機背面一堆電氣規格、各國電氣認證的銘牌位置，或者遊戲包裝盒甚至是手柄、附帶配件上面某個不起眼的位置，總有一個讓人摸不清頭腦很像某國電影番號的字符：RVL-001、WUP-001、WAP-002、CTR-001、NTR-032、UTL-004、AGS-101 ... 它們前面的字母都是什麼意思？（參考答案後附）

本文以大致的發售時間為順序，整理了任天堂自 FC 以來主機和掌機的型號代碼命名簡史和已知來由，其中還有不少鮮為人知的冷知識，也從側面反映了開發團隊的核心設計理念以及繁重工作之外的一些小巧思。

  • Famicom (FC)：型號代碼 HVC，意為 Home Video Computer 家用視頻電腦，也是內部開發名。歐美版的型號就叫 NES。

  • Famicom Disk System （FC磁碟機）：型號代碼 FMC，FMC 應該是 Family Computer 的縮寫。另外還有一個比較少用的代碼 FSC 就不太清楚意思了，大家可以猜測一下（詳見 Product Codes: FMC and FSC）

  • Gameboy (GB)：型號代碼 DMG，意為 Dot Matrix Game 點陣液晶遊戲，這一段在初代 GB 機身正面就可以找到，液晶屏上印著 Dot Matrix with Stereo Sound（2.6寸超大超省電超瞎眼“特朋瓏”點陣液晶屏！4通道“杜此”環繞立體聲！）。

    - 其實 GB 有五個音頻通道，兩路矩形/脈衝波、一路4位PCM、一路噪音，與FC類似。另外第五路是外部音頻通道，支持從卡帶讀取額外的音頻數據，但實際上當年沒卡帶用過（詳見 8bit Collective Chiptune Wiki）

    - 喇叭其實是單聲道的，插耳機才有立體聲效果

    - DMG 在內部因為不被看好曾經被叫做 Dame Game 駄目ゲーム，差不多就是完蛋遊戲的意思

Super Famicom （SFC超任）：型號代碼 SHVC，Super Home Video Computer。歐美版 SNES 的型號叫 SNS。SFC 的內部開發名為 SFX，包括索尼給任天堂做的 SFC Super Disc CD-ROM 機器（也就是 PlayStation）也屬於 SFX 家族，叫 SFX-100。

Virtual Boy (VB)：型號代碼 VUE，意為 Virtual Utopia Experience 虛擬烏托邦體驗，碉堡了。內部開發代號 VR32，因為 VB 是 32 位的主機。

  • Nintendo 64 (N64)：型號代碼 NUS，意為 Nintendo Ultra Sixty-four，內部開發代號是 Project Reality 真實計劃，當時任天堂是在 Nintendo 64 和 Ultra 64 中選了一個作為正式發售名。N64 的磁碟機 N64DD 的型號為 NUD，意為 Nintendo Ultra Disk 任天堂究極磁碟。

  • Game Boy Pocket (GBP)：型號代碼 MGB，意為 Mini Game Boy 迷你GB。帶背光的 GB Light 也是用的 MGB，型號是 MGB-101。

  • Game Boy Color (GBC)：型號代碼 CGB，意為 Color Game Boy 彩色GB。

Game Boy Advance (GBA)：型號代碼 AGB，意為 Advanced Game Boy 進擊的遊戲男孩（誤）。

有的資料認為 GBA 內部開發代號是 Project Atlantis 亞特蘭蒂斯計劃，但實際上這是另外一臺 GB 後繼機種的開發代號，該項目於 1995 年底開始運行，1996 年開始有外界傳聞並且還得到了任天堂的證實。

但是由於 GBP 和核彈級遊戲《精靈寶可夢》於 1996 年開售，任天堂認為 GB 系列還能再戰十年，所以在不斷推遲 Project Atlantis 的發售計劃後最終取消。

該項目的經驗也沒有被用於 GBA 的開發，因為按當時的一些傳聞，它的性能遠強於 GBA，ABXY按鍵，屏幕也更大，電池也能使用超過 30 小時，機能堪比 N64，任天堂也出於性價比的關係終止了銷售計劃。

要知道當時 GBP 售價僅僅 6800 日元（當時約合人民幣 450 元），這麼一臺機器如果發售不知道會是怎麼樣的天價，而 GBA 正式開發直到 GBC 發售一年後才開始。（詳見 Nintendo’s Original Game Boy Successor Was Not The GBA）

- 任天堂硬件開發部門的桑原雅人曾經在 GDC2009 大會上公開了 Project Atlantis 的外形，圖片中左邊是 Project Atlantis，右邊白色的是 NDSL。

GameCube (NGC)：型號代碼 DOL, 意為 Dolphin 海豚，這也是內部開發名（最早是叫 N2000），系統 OS 也叫 Dolphin OS，主機內 GPU 的名字叫 Flipper 腳蹼/鰭足，而 IBM 開發的 CPU 內部開發名叫 Gekko 月光。另外關於這個開發名還有一些有意思的細節（詳見 Are there any references to old codenames (ex. Dolphin, Ultra 64, NX) found in Nintendo hardware or software? • r/nintendo）：

- 取名為海豚有可能是寄希望其積極向上的形象可以扭轉 N64 銷售的頹勢

-《大金剛64》的100%隱藏結局動畫中，有一段 Cranky Kong 當導演的試鏡場面，而試鏡的牌子上印著海豚的圖片，暗示著為 NGC 上的續作試鏡，但在本作之後 Rare 就再也沒做過大金剛正統作品。（油管視頻鏈接）

- 《陽光馬力歐》裡的島嶼叫 Isle Delfino，Delfino 是意大利語“海豚”的意思，而且島嶼本身也是海豚形狀的

- 《皮克敏》裡船長奧利瑪的飛船叫 S.S. Dolphin

- 《塞爾達傳說：穆吉拉的面具》中的天文臺裡，可以找到地板上的海豚圖案

- 《任天堂大亂鬥Melee》裡，奈斯 Ness 用的悠悠球牌子叫 Dolphin Loop

- 文末會有一條關於 NGC 的 BONUS 彩蛋

陽光馬里奧的 Isle Delfino

Game Boy Advance SP (GBA SP)：型號代碼 AGS，意為 Advanced Game Boy Special 進擊的遊戲男孩~特別篇~（誤

- 還是在 GDC2009 上，桑原雅人公開了帶有觸摸屏的 GBA SP 的原型機照片

Nintendo DS (NDS)：型號代碼 NTR，意為 Nitrogen 氮氣，內部開發代號叫 Nitro。

其實 NDS 的內部開發代號有三個，依次是 Iris、Nitro、DS，Iris 是作為 GBA 後續機種最初期的開發代號，在雙屏設計出現後，這個名字就被 Nitro 和 DS 取代，Nitro 本來也可能被用作正式發售名稱，猜測是希望其含義（氮氣佔空氣的78%）讓該機能夠大賣，但最終任天堂還是選擇了更直白的 DS（Dual Screen 或者 Developers' System），而 NDS 確實也完成了 Nitro 的寓意，銷量超過一億五千萬臺，是有史以來成績最好的掌機。（詳見 Iwata Asks - WarioWare D.I.Y.）

  • Game Boy Micro (GBM)：型號代碼 OXY，意為 Oxygen 氧氣。因為該機是在 NDS 之後才發售的，基本上和 NDS 是同時期開發，所以這兩個機型的內部開發代號取了氮氣和氧氣這兩個在空氣中佔據絕大部分的氣體。

  • Wii：型號代碼 RVL，意為 Revolution 變革，也是內部開發名。在當年 E3 公開了 Wii 這個讓人摸不著頭腦的最終銷售名稱替代 Revolution 時，歐美玩家都炸了。

  • Nintendo DS Lite (NDSL)：型號代碼 USG，意為 Usugata 薄型（日語）。

  • Nintendo DSi (NDSi)：型號代碼 TWL，意為 Twilight 暮光。

  • Nintendo DSi XL/LL (NDSi XL/LL)：型號代碼 UTL，意為 Ultra Twilight 究極暮光。

  • Nintendo 3DS (3DS)：型號代碼 CTR，意為 Citra/Citrus 柑橘、檸檬的詞頭。

  • Nintendo 3DS XL/LL (3DS XL/LL)：型號代碼 SPR，意義不詳。

  • Wii U：型號代碼 WUP，意為 Wii U Project，內部的開發代號是 Project Cafe。

  • Nintendo 2DS (2DS)：型號代碼 FTR，意義不詳。

  • new Nintendo 3DS (new 3DS)：型號代碼 KTR，可能是 Kontrolle 的縮寫。內部開發代號 Project Kontrolle，Kontrolle 在德語裡是“控制、檢查”的意思。

  • new Nintendo 3DS XL/LL (new 3DS XL/LL)：型號代碼 RED，可能取自 Infrared。內部開發代號是 Project Infrared 紅外線計劃。

  • new Nintendo 2DS XL/LL (new 2DS XL/LL)：型號代碼 JAN，意義不詳。

  • Nintendo Switch：型號代碼 HAC，意義不詳。我們都知道內部開發名是 NX，但是很明顯 HAC 和 NX 看不出來有什麼關係，很多人猜測 HAC 的意思是 Handheld And Console（掌機+主機），這也能解釋得通，雖然任天堂只承認它是主機。

鑑於篇幅有限，就不介紹每個代碼下的詳細設備清單了，有興趣的朋友可以到 NinDB: The Unofficial Nintendo Archive 自行查找硬件以及配件的具體型號。

可以看到，新的設備由於發售年代較近，關於內部研發的資料就更少，所以並不清楚某些代碼的含義，只能希望將來任天堂的開發團隊能夠在各種場合為大家介紹這些開發祕聞。

而且如果你有模擬器、破解經驗的話，會發現這些代號經常用於模擬器名稱或者破解屬於中，例如 Dolphin Emulator（模擬 NGC 和 Wii 的海豚模擬器）、CTR NAND（3DS破解術語）、TWLoader（DS樣式GUI）等。

參考答案：略（中學參考書的套路你們不會現在還上當吧！）
