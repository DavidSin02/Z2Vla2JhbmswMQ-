三年級數學知識點總結-03——倍的認識與多位數乘一位數

2020-11-14
| 科學

撰文：茂喵喵

審核：仲修


一、倍的認識

1、倍的意義：一個數是由幾個另一個數組成的或者一個數是幾個另一個數的和，就叫做這個數是另一個數的幾倍。如：8是2的4倍，8=2+2+2+2，8是4個2的和。

2、求一個數是另一個數的幾倍：用除法，一個數÷另一個數=幾倍。如：求20是4的幾倍？因為20÷4=5，所以20是4的5倍。

3、求一個數的幾倍是多少：用乘法。如：8的9倍是多少？因為8×9=72，所以8的9倍是72.

4、倍數關係沒有單位。

5、求比一個數的幾倍多幾的數是多少，用乘法與加法。如：比9的5倍多6的數是多少？因為9×5+6=51，所以所求的數是51.

6、求比一個數的幾倍少幾的數是多少，用乘法與減法。如：比8的6倍少5的數是多少？因為8×6-5=43，所以所求的數是43.


二、多位數乘一位數

1、多位數乘一位數（進位）的筆算方法 ：相同數位對齊，從個位乘起，用一位數依次去乘多位數每一位上的數，哪一位上乘得的數積滿幾十，就向前一位進幾，與哪一位相乘，積就寫在哪一位下面。

2、一個因數中間有0的乘法 ：

① 0 和任何數相乘都得 0 ；

②因數中間有0，用一位數去乘多位數每一位數上的數，與中間的 0 相乘時，如果後面沒有進上來的數，這一位上要用 0 來佔位，如果有進上來的數必須加上。

③一個因數末尾有 0 的乘法的簡便計算：筆算時，可以把一位數與多位數 0 前面那個數字對齊，再看多位數的末尾有幾個 0 ，就在積的末尾添上幾個 0.

3、① 0 和任何數相乘都得 0 ；

② 1 和任何不是 0 的數相乘還得原來的數。

4、三位數乘一位數：積有可能是三位數，也有可能是四位數。
