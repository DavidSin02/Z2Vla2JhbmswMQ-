近6億中國人失眠，昨晚你也在數綿羊嗎？

2020-11-14
| 科學


今天又是一個被鬧鐘叫起床的週二，下班後的你在幹什麼？是在不懈追劇還是在刻苦學習？或者是在結束了一天的工作之後早早入睡？

但是隨著城市發展的速度越來越迅猛社會競爭和人們的壓力越來越大早早地入睡和睡個安穩覺
卻更像是一種奢望
入睡困難和良好的睡眠質量已成為人群中最主要的失眠表現而這些失眠人的孤單無助與痛楚只有凌晨的夜最清楚

早睡變成了一種奢侈失眠更是家常便飯每天晚上抱著手機在床上輾轉反側他們為什麼不睡因為他們根本睡不著
01準時下班是什麼感覺？

@朵朵梓


每天一醒來就是滿滿的工作項目，從早上9點到晚上12點，都是常有的事。晚上下班晚，睡覺自然也晚：晚上到家隨便收拾收拾，就已經一兩點了。
自從進入這家公司以來，幾乎就沒有6點下班過，我都快忘了傍晚6、7點的夕陽和晚霞是什麼樣子了。



02只有晚上才是屬於自己的時間@一Jean
作為一個每天定時定點“搬磚”的社畜，只有在下班後躺在床上那一瞬間，才覺得自己是得到了解放，身體和心靈都是真正屬於自己的。
這種時候夜晚的時間就顯得特別的珍貴，甚至感覺早睡都是對這段時間的不尊重，所以就一直刷手機，越刷越晚......

這是一個“夜貓子”越來越多的時代
晚上睡不著，早上起不來靠著咖啡撐過每一個工作日睡個好覺，對很多人來說是一種奢望
失眠帶來的危害

/ 01 會讓人遲鈍 /


睡眠不足可損害人的注意力、

警覺性、專注性、推理能力

以及解決問題的能力

這些可導致你的學習和工作效率變低


/ 02 記憶力退化 /
失眠還會使人體的記憶力退化導致患者注意力無法集中，影響工作學習等

/ 03 加速皮膚衰老 /
皮膚在晚10～11點進入保養狀態長期失眠會導致植物神經功能紊亂，內分泌失調使皮膚乾燥，鬆弛，彈性降低還有熊貓眼等問題

/ 04 引發嚴重健康問題 /
經常處於
熬夜、疲勞、精神不振的狀況可增加你患有這些疾病的風險人體的免疫力會跟著下降感冒、過敏等就會不期而至找上門

如何防止失眠？
/ 01 適當運動 /
運動的確可以改善睡眠狀況因為它能調節植物神經使大腦的興奮受到轉移和抑制所以平時一定要加強體育鍛煉，增強體質為了保障良好的睡眠質量每天應該適當進行有氧運動

/ 02 請調節心態和情緒 /
睡前以後最好不要回想煩心事有許多人容易失眠，是因為平時事事存在心裡防治失眠，需要自己好好調節心態和情緒23點以後最好不要回想煩心事睡前可以聽一些舒緩的音樂

/ 03 合理飲食 /
熬夜的人一般會吃“夜宵”這時不但難以消化隔日早晨還會出現食慾不振造成營養不均衡，引起肥胖所以晚餐不能吃太多也不能吃太少還有，睡前萬萬不能喝咖啡、濃茶讓人興奮的飲料


/ 04 晚上泡腳利於睡眠 /
熱水泡腳可加速腳部血液循環水溫以40℃左右為宜每晚用溫水泡腳10分鐘並用手按摩腳以促進血液循環放鬆人體，緩解疲勞從而產生睏倦感，有助於提高睡眠質量


不管工作有多忙，生活有多難

都沒有自己的健康重要不如就從現在開始放下手機，放鬆心情今晚早睡一小時明天繼續加油！
