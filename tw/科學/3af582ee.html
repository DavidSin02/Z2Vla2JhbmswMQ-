linux基本命令歸納

2020-10-29
| 科學

一，系統命令

1，poweroff：直接關機

2，shutdown：

-h：將系統關機；

-k：只是送出信息給所有用戶，但不會實際關機；

-r：shutdown之後重新啟動；

-t<秒數>：送出警告信息和刪除信息之間要延遲多少秒。

3，reboot：重啟

4，halt：關機（僅限系統管理者）

5，who：顯示系統所有使用者

6，whoami：顯示當前用戶

7，w：顯示當前用戶信息

二，系統設置

1，clear：清空屏幕

三，系統管理

1，su：切換到其他用戶

2，date：顯示或設定系統的日期與時間

3，cal：當前日曆，或者指定日期的日曆

4，alias：設置指令的別名

5，unalias：取消指令別名

四，磁盤管理

1，pwd：顯示當前工作目錄

2，ls：顯示指定工作目錄下內容

3，cd：切換當前工作目錄

“ ~ ”也表示為 home 目錄 的意思

“ . ”則是表示目前所在的目錄，

“ .. ”則表示目前目錄位置的上一層目錄

4，mkdir：創建目錄

-p 遞歸創建目錄

5，rmdir：刪除目錄

-p 遞歸創建目錄

5，du：用於顯示目錄或文件的大小

-m或--megabytes 以1MB為單位。

-s或--summarize 僅顯示總計

-k或--kilobytes 以1024 bytes為單位

-b或-bytes 顯示目錄或文件大小時，以byte為單位

6.dd：備份文件

if=文件名：輸入文件名，默認為標準輸入。即指定源文件

of=文件名：輸出文件名，默認為標準輸出。即指定目的文件

7，stat：顯示文件詳細信息

五，幫助指令

1，man：查看Linux中的外部指令幫助、配置文件幫助和編程幫助等信息

2，help：查看內部命令的幫助信息

3，whatis：查詢一個命令執行什麼功能，並將查詢結果打印到終端上

六，文件管理

1，whereis：命令用於查找文件

-b 　只查找二進制文件。

-B 只在設置的目錄下查找二進制文件

-f 　不顯示文件名前的路徑名稱

-m 　只查找說明文件

-M 　只在設置的目錄下查找說明文件

-s 　只查找原始代碼文件

-S 　只在設置的目錄下查找原始代碼文件

-u 　查找不包含指定類型的文件

2，cp：複製文件或目錄

3，mv：為文件或目錄改名、或將文件或目錄移入其它位置

4，touch：創建文件

5，rm：刪除文件

-f 即使原檔案屬性設為唯讀，亦直接刪除，無需逐一確認

6，cat/tac：顯示文件的內容/逆序顯示文件的內容

-n 有1開始對所有輸出的行數編號；
-b 和-n相似，只不過對於空白行不編號；
-s 當遇到有連續兩行以上的空白行，就代換為一行的空白行

7，more/less：分頁顯示內容

8，head/tail：從頭部/尾部截取文檔內容

-c 顯示的字節數

-n 顯示文件的尾部 n 行內容

9，cut：截取列內容

-d ：自定義分隔符

-f ：與-d一起使用，指定顯示哪列

10，locate：查找符合條件的文檔

-b 匹配路徑名下文件

-c 只輸出找到的數量

11，find：在指定目錄下查找文件

12，file：file命令用於辨識文件類型

-b 　列出辨識結果時，不顯示文件名稱

13，ln：文件創件連接

-b 刪除，覆蓋目標文件之前的備份
-d 建立目錄的硬連接
-f 強行建立文件或目錄的連接，不論文件或目錄是否存在
-i 覆蓋既有文件之前先詢問用戶
-n 把符號連接的目的目錄視為一般文件
-s 對源文件建立符號連接，而非硬連接

七，文檔編輯

1，sort：針對文本文件的內容，以行為單位來排序

-b 忽略每行前面開始出的空格字符

-c 檢查文件是否已經按照順序排序

-d 排序時，處理英文字母、數字及空格字符外，忽略其他的字符

-f 排序時，將小寫字母視為大寫字母

-m 將幾個排序好的文件進行合併

-M 將前面3個字母依照月份的縮寫進行排序

-n 依照數值的大小排序

-u 意味著是唯一的(unique)，輸出的結果是去完重了的

-r 以相反的順序來排序

2，uniq：命令用於檢查及刪除文本文件中重複出現的行列

-c 在每列旁邊顯示該行重複出現的次數

-d 僅顯示重複出現的行列

-f 忽略比較指定的欄位

-u 僅顯示出一次的行列

3，wc：計算文件的Byte數、字數、或是列數

-c 只顯示Bytes數

-l 只顯示行數

-w 只顯示字數

4，grep：查找文件裡符合條件的字符串

-c 計算符合範本樣式的列數

-n 在顯示符合範本樣式的那一列之前，標示出該列的編號

八，特殊字符

1，“ ^ ” 行首符

2，“ % ” 末尾符

3，“ | ” 管道符
