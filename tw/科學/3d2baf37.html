Numpy 多維數組創建

2020-11-14
| 科學

說明：文章所有內容均截選自實驗樓教程【Numpy 使用教程】（教程鏈接：https://www.shiyanlou.com/courses/912），如果對Numpy感興趣的可以上實驗樓查看教程；

如果你使用 Python 語言進行科學計算，那麼一定會接觸到 Numpy。Numpy 是支持 Python 語言的數值計算擴充庫，其擁有強大的高維度數組處理與矩陣運算能力。除此之外，Numpy 還內建了大量的函數，方便你快速構建數學模型。

Numpy 多維數組

1 ndarray 介紹

在 python 內建對象中，數組有三種形式：

list 列表：[1, 2, 3]

Tuple 元組：(1, 2, 3, 4, 5)

Dict 字典：{A:1, B:2}

其中，元組與列表相似，不同之處在於元組的元素不能修改。而字典由鍵和值構成。

python 標準類針對數組的處理侷限於 1 維，並僅提供少量的功能。

而 Numpy 最核心且最重要的一個特性就是 ndarray 多維數組對象，它區別於 python 的標準類，擁有對高維數組的處理能力，這也是數值計算過程中缺一不可的重要特性。

Numpy 中，ndarray 類具有六個參數，它們分別為：

shape：數組的形狀。

dtype：數據類型。

buffer：對象暴露緩衝區接口。

offset：數組數據的偏移量。

strides：數據步長。

order：{'C'，'F'}，以行或列為主排列順序。

下面，我們來了解創建 ndarray 的一些方法。在 numpy 中，我們主要通過以下 5 種途徑創建數組，它們分別是：

  1. 從 Python 數組結構列表，元組等轉換。

  2. 使用 np.arange、np.ones、np.zeros 等 numpy 原生方法。

  3. 從存儲空間讀取數組。

  4. 通過使用字符串或緩衝區從原始字節創建數組。

  5. 使用特殊函數，如 random。

2 從列表或元組轉換

在 numpy 中，我們使用 numpy.array 將列表或元組轉換為 ndarray 數組。其方法為：

numpy.array(object, dtype=None, copy=True, order=None, subok=False, ndmin=0)

其中，參數：

object：列表、元組等。

dtype：數據類型。如果未給出，則類型為被保存對象所需的最小類型。

copy：布爾來寫，默認 True，表示複製對象。

order：順序。

subok：布爾類型，表示子類是否被傳遞。

ndmin：生成的數組應具有的最小維數。

下面，通過列表創建一個 ndarray 數組：

import numpy as npnp.array([[[1, 2, 3],[1, 2, 3],[1, 2, 3]],[[1, 2, 3],[1, 2, 3],[1, 2, 3]],[[1, 2, 3],[1, 2, 3],[1, 2, 3]]])

或者是列表和元組：

import numpy as npnp.array([(1,2),(3,4),(5,6)])

3 arange 方法創建

除了直接使用 array 方法創建 ndarray，在 numpy 中還有一些方法可以創建一些有規律性的多維數。首先，我們來看一看 arange()。arange() 的功能是在給定區間內創建一系列均勻間隔的值。方法如下：

numpy.arange(start, stop, step, dtype=None)

你需要先設置值所在的區間，這裡為 ``[開始， 停止），你應該能發現這是一個半開半閉區間。然後，在設置step步長用於設置值之間的間隔。最後的可選參數dtype 可以設置返回 ndarray` 的值類型。

舉個例子：

import numpy as np# 在區間 [3, 7) 中以 0.5 為步長新建數組np.arange(3, 7, 0.5, dtype=float32)

4 linspace 方法創建

linspace方法也可以像arange方法一樣，創建數值有規律的數組。linspace 用於在指定的區間內返回間隔均勻的值。其方法如下：

numpy.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None)

start：序列的起始值。

stop：序列的結束值。

num：生成的樣本數。默認值為50。

endpoint：布爾值，如果為真，則最後一個樣本包含在序列內。

retstep：布爾值，如果為真，返回間距。

dtype：數組的類型。

舉個例子：

import numpy as npnp.linspace(0, 10, 10, endpoint=True)np.linspace(0, 10, 10, endpoint=False)

5 ones 方法創建

numpy.ones 用於快速創建數值全部為 1 的多維數組。其方法如下：

numpy.ones(shape, dtype=None, order='C')

其中：

shape：用於指定數組形狀，例如（1， 2）或 3。

dtype：數據類型。

order：{'C'，'F'}，按行或列方式儲存數組。

舉個例子：

import numpy as npnp.ones((2,3))

6 zeros 方法創建

zeros 方法和上面的 ones 方法非常相似，不同的地方在於，這裡全部填充為 0。zeros 方法和 ones 是一致的。

numpy.zeros(shape, dtype=None, order='C')

其中：

shape：用於指定數組形狀，例如（1， 2）或3。

dtype：數據類型。

order：{'C'，'F'}，按行或列方式儲存數組。

舉個例子：

import numpy as npnp.zeros((3,2))

7 eye方法創建

numpy.eye 用於創建一個二維數組，其特點是k 對角線上的值為 1，其餘值全部為0。方法如下：

numpy.eye(N, M=None, k=0, dtype=<type 'float'>)

其中：

N：輸出數組的行數。

M：輸出數組的列數。

k：對角線索引：0（默認）是指主對角線，正值是指上對角線，負值是指下對角線。

舉個例子：

import numpy as npnp.eye(5, 4, 3)

8 從已知數據創建

我們還可以從已知數據文件、函數中創建 ndarray。numpy 提供了下面 5 個方法：

frombuffer（buffer）：將緩衝區轉換為 1 維數組。

fromfile（file，dtype，count，sep）：從文本或二進制文件中構建多維數組。

fromfunction（function，shape）：通過函數返回值來創建多維數組。

fromiter（iterable，dtype，count）：從可迭代對象創建 1 維數組。

fromstring（string，dtype，count，sep）：從字符串中創建 1 維數組。

舉個例子：

import numpy as npnp.fromfunction(lambda a, b: a + b, (5, 4))
