浮點型數據的算術運算

浮點型數據同樣可以應用算術運算符進行算術操作，同整型數據運算一樣，運算結果的類型和參與計算中取值範圍最大的數據的類型一致。如若存在double型的數據進行運算，則運算的結果為double型，否則若存在float型數據，那麼結果就為float型。

當浮點型數據進行運算時，計算機計算出的小數點後的n位，與實際的數值會有一定的誤差，它只能儘量地去接近實際值。

例如，計算float型數據4.0減去浮點型數據2.9的結果，在數學計算中該結果為1.1，但計算機計算出的結果只能儘量地去接近1.1。

float f=4.0f-2.9f;

System.out.println(f); //輸出結果為：1.0999999

例 在項目中創建類Arith3，在主方法中進行浮點型數據除以0或除以0求餘的運算，並輸出計算結果。

publicclass Arith3 {

publicstaticvoid main(String[] args) {// 主方法

float f1=1.234f/0; //計算正浮點數除以0

float f2=-5.678f/0; //計算負浮點數除以0

float f3=1.234f%0; //計算正浮點數除以0求餘

float f4=-5.678f%0; //計算負浮點數除以0求餘

System.out.println(f1); //輸出結果為：Infinity

System.out.println(f2); //輸出結果為：-Infinity

System.out.println(f3); //輸出結果為：NaN

System.out.println(f4); //輸出結果為：NaN

}

}

自增和自減運算符

自增、自減運算符是單目運算符，可以放在操作元之前，也可以放在操作元之後。操作元必須是一個整型或浮點型變量。自增、自減運算符的作用是使變量的值增1或減1。放在操作元前面的自增、自減運算符，會先將變量的值加1（減1），然後再使該變量參與表達式的運算。放在操作元后面的自增、自減運算符，會先使變量參與表達式的運算，然後再將該變量加1（減1）。

例應用兩種自增運算符的輸出結果。下面的代碼：

int i=1;

System.out.println("使用i++的結果：");

System.out.println(i++);

System.out.println(i);

i=1;

System.out.println("使用++i的結果：");

System.out.println(++i);

System.out.println(i);

比較運算符

比較運算符屬於二元運算符，用於程序裡的變量之間、變量和自變量之間以及其他類型的信息之間的比較上。比較運算符的運算結果是boolean型。當運算符對應的關係成立時，運算結果是true，否則是false。所有比較運算符通常用在條件語句中作為判斷的依據。比較運算符共有6個，如表1所示


例 在項目中創建類Compare，在主方法中創建整型變量，使用比較運算符對變量進行比較運算，將運算後的結果輸出。

publicclassCompare { //創建類

publicstaticvoid main(String[] args) {

int number1 = 4; //聲明int型變量number1

int number2 = 5; //聲明int型變量number2

System.out.println("number1 > number2的返回值為：" + (number1 >number2));

/*依次將變量number1與變量number2的比較結果輸出*/

System.out.println("number1 < number2的返回值為：" + (number1 <number2));

System.out.println("number1 == number2的返回值為：" + (number1 ==number2));

System.out.println("number1 != number2的返回值為：" + (number1 !=number2));

System.out.println("number1 >= number2的返回值為：" + (number1 >=number2));

System.out.println("number1 <= number2的返回值為：" + (number1 <=number2));

}

}
