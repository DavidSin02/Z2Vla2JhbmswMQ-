新片推薦：《巴霍巴利王 2》，印度版的超級英雄故事

2020-10-29
| 科學

《巴霍巴利王 2 ：終結》

導演：S·S·拉賈穆裡

編劇：曼孔布·戈帕拉克里希南 / 馬德漢·卡爾基 / C·H·維傑·庫馬爾 / 馬努傑·芒塔希爾 / 維

傑耶德拉·普拉薩德 / S·S·拉賈穆裡

主演：帕拉巴斯 / 拉納·達格巴提 / 安努舒卡·謝蒂 / 特曼娜·芭蒂亞 / 薩伯拉傑

片長：141 分鐘（中國大陸）

格式：2D

2015 年，那部榮登印度歷史票房榜首的影片《巴霍巴利王：開端》在結尾處給觀眾留下了一個巨大的

謎團：為什麼忠誠的卡塔帕將軍會殺死巴霍巴利王？

漫長的 3 年等待後，《巴霍巴利王 2：終結》以前傳的形式為觀眾回溯了屬於巴霍巴利的故事前傳。

並且不出意外地再次獲得了票房冠軍。

看過印度史詩《羅摩衍那》或《摩訶婆羅多》的觀眾不會對《巴霍巴利王 2》的故事感到陌生。這次，

英雄史詩的主角是希瓦的父親，阿瑪蘭德拉·巴霍巴利。他出生在皇宮，與表兄巴拉拉德夫德斯一同由

太后撫養大，巴霍巴利從小膂力驚人，心懷正法，最終在一場對陣蠻族的大戰中獲得了王儲身份。

在外出遊歷中，英雄愛上了諸侯國的公主提婆犀那，這為不甘失敗的巴拉拉德夫德斯提供了絕佳的復仇

機會。陰謀、誤解和愚昧，最終使英雄死在了親人的劍下。

如上所述，整部影片描述的內容都頗具神話色彩，因此英雄和大反派的角色也十分套路化，流於表面的

皇室陰謀對於中國觀眾來說更是司空見慣，但《巴霍巴利王 2》的製作過程和最終效果，都讓人感受到

片方的滿滿誠意，比如片頭富有瓷塑質感的幾處場景呈現，甚至讓人有超級英雄影片的既視感。

事實上，《巴霍巴利王 2》也確實借鑑了好萊塢超級英雄電影的敘事模式。在拍攝前作時，導演 S·S

·拉賈穆裡曾表示，在印度極為流行的系列漫畫 Amar Chitra Katha正是他投拍影片的契機之一。為了

讓孩子們瞭解國家的歷史，出版商 Anant Pai 以漫畫的形式重述了印度史詩、神話和民間傳說故事。

在漫畫中，兒時的 S·S·拉賈穆裡看到了神靈們的宮殿、巨大的棕櫚樹和百發百中的弓箭。

他把這些符號搬到了自己的電影當中。每個屬於巴霍巴利的戰爭片段，觀眾都不難感受到那種無所不能

的超人感，而在英雄與公主萌生愛情的場景裡，那些幽默和近似無厘頭的對話，無不來自西式漫畫的敘

事風格。

這也是巴霍巴利王系列獲得世界吸引力的原因之一。當然，也唯有在這樣漫畫式的故事裡，主人公的全

部表現才能邏輯自洽。

而對於大部分觀眾來說，古印度充滿神祕色彩的英雄傳說和極具創意的戰爭場面已經很有吸引力。另外

，儘管並非出自本意，影片漫畫感的角色塑造方式（比如配樂）還為其增添了幾分“一本正經搞笑”的

特質。

唯一不足的是，雖然國內版只有 141 分鐘（原片 167 分鐘），中國觀眾可能還是會覺得它太長了些。

《香港大營救》

導演：劉一君

編劇：陸信 / 郭子勝 / 劉一君 / 劉雲鵬

主演：成泰燊 / 顏丙燕 / 樑俊一 / 霍青 / 王暉

片長：95 分鐘

格式：2D

儘管和《明月幾時有》講述了同一歷史時期的同一事件，《香港大營救》只能算是一部中規中矩的紅色

影片。

1941 年末太平洋戰爭爆發後，日軍在港搜捕文化名人，共產黨地下黨員與在港各方勢力周旋，全城出

動，完成營救。

《香港大營救》是轉企改制之後的瀟湘電影集團拍攝的“紅色瀟影“之一。2011 年，他們兩部電影被

列入中宣部、廣電總局“慶祝中國共產黨成立 90 週年國產重點影片”，之後的作品還有《湘南起義》

和《青春雷鋒》。

在這種風格定位下，原本充滿複雜矛盾的《香港大營救》成了一個極為平面化的故事，導演劉一君在眾

多可以觀照人性的敘事方式中選擇了最平庸的那一種，也就是多線展開的大全景。

而宣傳詞中多次提及的“40 年代香港場景搭建”和波瀾壯闊的大撤退鬥爭場面，其實和我們常見的諜

戰題材電視劇並無二致，實際觀影過程中從音軌到字幕的質感，也真的很難感受到片方的五年精心籌備

。

不過，既然能在人民大會堂舉辦首映禮，它票房應該不會太差吧。
