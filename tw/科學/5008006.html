物質跨膜運輸的實例

一、選擇題

1．如圖為顯微鏡下某植物細胞在30%蔗糖溶液中的示意圖。下列敘述中錯誤的是( )

A．若將細胞置於清水中，A仍保持不變

B．若該細胞處於40%蔗糖溶液中，B/A值將變小

C．B/A值能表示細胞失水的程度

D．A、B分別表示細胞和液泡的長度

答案：D

解析：本題考查植物細胞的質壁分離過程，意在考查考生對實驗現象的理解分析能力。圖示為處於質壁分離狀態的植物細胞。因細胞壁的伸縮性很小，故若將該細胞置於清水中，A仍將保持不變。若該細胞處於40%的蔗糖溶液中，與圖示相比，A不變，但B因失水更多而變小，所以B/A值將變小，可見B/A值能表示細胞失水的程度。圖中A表示的是細胞長度，但B表示的並不是液泡的長度，而是失水後的原生質體的長度。

2．觀察植物細胞的質壁分離與復原，常用紫色洋蔥做材料，這是因為( )

A．紫色洋蔥容易發生質壁分離

B．細胞液呈紫色便於觀察

C．液泡較大便於觀察

D．細胞膜呈紫色便於觀察

答案：B

3．“人造細胞”是用羊皮紙或火棉膠等半透性膜材料做成的一個密閉的囊，囊裡面裝了等量的不同濃度的蔗糖溶液。假設有三個這樣的“人造細胞”，分別用A、B、C表示，它們的蔗糖溶液濃度分別是10%、20%和30%，其中A、B細胞的外面套一金屬網籠，C細胞的外面沒有套金屬網籠，如下圖。把三個“人造細胞”同時放在蔗糖溶液濃度為15%的玻璃槽中，經過一段時間後，三個“細胞”的變化分別是( )

A．A變大、B稍微變小、C不變

B．A變小、B稍微變大、C不變

C．A變小、B稍微變大、C變大

D．A變大、B稍微變小、C變大

答案：C

解析：在滲透作用過程中，水分子總是從低濃度→高濃度，因此在該裝置中，水分子的運動方向應為A→玻璃槽，玻璃槽→B，玻璃槽→C，因此，A變小，B、C變大；但是，由於B濃度小於C濃度，且B的外面有金屬網籠，因此B的體積稍微變大。

4．下列哪種細胞被放入高滲溶液中將會出現質壁分離現象( )

A．剛撕下的蠶豆葉表皮細胞

B．尚未萌發的菜豆胚芽細胞

C．用95%乙醇浸泡過的洋蔥鱗片葉上表皮細胞

D．剛切下的小麥根尖生長點細胞

答案：A

解析：能發生質壁分離的細胞必須具備如下幾個條件：①有生命的活細胞；②有細胞壁；③有大液泡。題中列出的幾類細胞中，B、D兩項的細胞均無大液泡，C項的細胞經95%的乙醇浸泡，已喪失生命活力。因此，都不可能發生質壁分離。剛撕下的蠶豆葉表皮細胞，儘管葉片不完整，但細胞是完整的，有生命力，具有大液泡。因此，將A項細胞放入高滲溶液中將出現質壁分離。

5．下列說法中，哪些是植物細胞發生質壁分離的原因( )

①外界溶液濃度小於細胞液濃度

②細胞液濃度小於外界溶液濃度

③細胞壁的伸縮性大於原生質層的伸縮性

④原生質層的伸縮性大於細胞壁的伸縮性

A. ①④ B．②④

C．②③ D．③④

答案：B

6．在缺少醫藥的戰爭年代，常常利用高濃度的鹽水對傷員的傷口做消毒殺菌處理，原因是( )

A．鹽水中的氯有殺菌作用

B．鈉具有殺菌作用

C．高濃度的鹽水中水分少，不利於細菌生長

D．細菌會由於滲透作用而脫水死亡

答案：D

7．通過質壁分離實驗可以鑑定( )

①細胞的死活　②原生質層與細胞壁伸縮性大小的差異　③原生質層相當於半透膜　④水分子進出細胞的方式　⑤細胞壁由纖維素構成　⑥細胞膜是單層膜

A．①②③ B．②④⑥

C．②③⑤ D．①②⑥

答案：A

8．夏天，很多人都愛吃拌黃瓜，細心的同學會發現雖然拌黃瓜時加了鹽和醋及其他一些調味品，黃瓜的外表面是有味的，但黃瓜果肉內卻是淡的。這是因為( )

A．拌的時間短

B．細胞壁阻止鹽進入細胞內

C．細胞膜有選擇透過性

D．鹽溶在醋中，無法進入細胞內

答案：C

9．將紫色洋蔥表皮細胞置於30%的蔗糖溶液中，光鏡下所觀察到的現象是( )

①液泡逐漸變小，顏色由淺變深　②液泡逐漸變小，顏色由深變淺　③原生質層與細胞壁逐漸分離形成球形小團　④細胞膜、液泡膜結構明顯　⑤原生質層與細胞壁之間是蔗糖溶液。

A．①③ B．①③④

C．②③ D．①③⑤

答案：A

解析：因外界蔗糖溶液濃度大於細胞液濃度，洋蔥表皮細胞將不斷失水，使得液泡變小，而細胞液中色素不能隨水透過原生質層，因此顏色加深。由於液泡和原生質層的不斷收縮，可看見③現象的出現。光鏡下既看不清膜結構又辨不清蔗糖的成分。

10．以下哪種情況會發生滲透作用( )

①幹種子萌發時的吸水　②萎蔫的青菜放進清水中變得硬挺　③根毛細胞吸水　④水由氣孔進入外界環境　⑤插在花瓶中的玫瑰花枝由瓶中吸水　⑥洋蔥表皮細胞放在鹽水中發生質壁分離　⑦兔的血液紅細胞放在鹽水中發生皺縮

A．①②③⑤⑦ B．②③⑤⑥⑦

C．②③④⑥⑦ D．②③⑥⑦

答案：D

解析：本題考查對滲透原理掌握情況以及對不同條件下植物、動物細胞吸水、失水的分析能力。滲透作用發生的條件是：一是具備半透膜，二是半透膜兩側具有濃度差。而動物細胞的細胞膜、植物細胞的原生質層都是選擇透過性膜，所以動、植物細胞的滲透吸水和失水，決定於細胞膜(或原生質層)內外兩側溶液的濃度差。植物細胞形成液泡以後，細胞液的濃度大於外界溶液濃度時可以通過滲透作用吸水，動物細胞雖無液泡和細胞液，但細胞內的細胞質也具有一定的濃度能發生滲透作用。幹種子細胞沒有形成液泡，屬於吸脹吸水；水由氣孔進入外界是由於蒸騰作用；插在花瓶中的玫瑰花枝條吸水，是通過枝條中的導管的毛細現象吸水；洋蔥表皮、根毛細胞、青菜葉屬於成熟植物細胞有液泡，當它們與外界相接觸時能發生滲透吸水的是②③，發生滲透失水的是⑥⑦。

11．把一個已發生質壁分離的植物細胞投入清水中，細胞的吸水能力將( )

A．逐漸增強 B．逐漸降低

C．保持不變 D．先升高後降低

答案：B

解析：細胞吸水能力的大小取決於原生質層兩側的濃度差的大小。已發生質壁分離的植物細胞的原生質層兩側的濃度差較大，當把該細胞放入清水中，隨著水分子不斷進入到細胞內，原生質層兩側的濃度差逐漸減小，細胞的吸水能力逐漸降低。

12．如圖表示多種植物成熟細胞，在不同濃度的蔗糖溶液中質壁分離的百分率，圖中曲線表明這些植物細胞( )

A．細胞液濃度＞蔗糖溶液濃度

B．細胞液濃度＜蔗糖溶液濃度

C．有不同濃度的細胞液

D．細胞膜有不同的選擇透過性

答案：C

解析：當細胞外濃度大於細胞液濃度時，成熟的植物細胞會發生質壁分離現象。此題中在蔗糖溶液濃度較低時(如0.2)只有一少部分細胞發生了質壁分離，此時若升高外界溶液濃度(如升到0.3)會發現又有一部分本來沒有發生質壁分離的細胞發生了質壁分離，這就說明外界溶液濃度較小時之所以不能使那一部分細胞發生質壁分離，主要是因為當時的外界溶液濃度小於細胞液的濃度，而外部溶液濃度較低時(如0.2)之所以可以使部分細胞發生質壁分離，是由於當時的外界溶液濃度已經超過了那一部分細胞的細胞液濃度。因此可以推斷：不同種類的植物成熟細胞有不同濃度的細胞液。

13．將已發生質壁分離的細胞再置於較低濃度的蔗糖溶液中，觀察到質壁分離復原，直到細胞體積穩定下來，這時對恢復原狀的細胞來說，可以肯定的情況是( )

A．細胞液和外界溶液濃度相同

B．細胞液濃度高於外界溶液濃度

C．水分子不再出入細胞

D．水分子出入細胞處於動態平衡

答案：D

14．將人體哺乳動物的紅細胞分別浸入下列a、b、c三個不同濃度溶液的燒杯內，一段時間後，實驗結果如圖所示，則a、b、c溶液濃度大小比較正確的是( )

A. a>b>c B. a>c>b

C. a<b<c D. a<c<b

答案：D

二、非選擇題

15．用2mol·L－1的乙二醇溶液和2mol·L－1的蔗糖溶液分別浸浴某種植物細胞，觀察質壁分離現象，得到其原生質體體積的變化情況如下圖所示。請據圖回答：

(1)原生質體體積A→B段的變化說明：在該段時間內水分從原生質體__________，細胞液濃度__________；

(2)在1分鐘後，處於2mol·L－1蔗糖溶液中的細胞，其細胞液濃度將__________，此時，在細胞壁與原生質層之間充滿了__________，要使該細胞復原，應將其置於__________中；

(3)在1分鐘後，處於2mol·L－1乙二醇溶液中的細胞，其原生質體體積的變化是由於__________逐漸進入細胞內，引起細胞液濃度__________；

(4)並不是該植物的所有活細胞均可發生質壁分離，能發生質壁分離的細胞還必須具有__________。

答案：(1)滲出　變大　(2)不變　2mol·L－1蔗糖溶液　清水或低於細胞液濃度的其他溶液中　(3)乙二醇　變大　(4)大液泡

解析：(1)AB段原生質體體積逐漸變小，故水分由原生質體滲出，細胞液濃度勢必變大。(2)1分鐘後，處於2mol·L－1蔗糖溶液中的原生質體體積不變，故其細胞液濃度不變。此時，在細胞壁與原生質層之間充滿了2mol·L－1的蔗糖溶液。要使該細胞發生質壁分離復原，須將其放入清水或其他濃度低於細胞液濃度很多的溶液中。(3)在1分鐘後，處於2mol·L－1乙二醇溶液中的細胞原生質體體積變大，是由於乙二醇進入細胞內引起細胞吸水，發生了細胞質壁分離復原，此時細胞的液泡中由於進入乙二醇，其溶液濃度會最終變大。(4)發生質壁分離的細胞必須具有中央大液泡，才有利於其發生滲透作用和原生質體體積的變化。

16．圖示洋蔥表皮細胞質壁分離圖，請據圖回答。

(1)此實驗所用的材料具體說它是__________細胞。

(2)原生質層包括[　]__________、[　]__________和[　]__________。

(3)植物細胞質壁分離的發生是由於外界溶液的濃度比細胞液的濃度__________，原生質層的__________性比細胞壁大。

答案：(1)紫色洋蔥外表皮細胞　(2)2　細胞膜　3　細胞質　5　液泡膜　(3)大(或高)　伸縮
