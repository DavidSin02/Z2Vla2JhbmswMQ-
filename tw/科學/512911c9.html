目 錄（一） 基本常識

　　（二）、主要技術參數及其作用：

１、什麼是汽車的車輪定位 現代汽車的車輪定位是指車輪、懸架系統元件以及轉向系統元件，安裝到車架（或車身）上的幾何角度與尺寸須符合一定的要求，保證汽車行駛的穩定性和安全性，減少汽車的磨損和油耗。

　　２、四輪定位維修的好處？

增加行駛安全直行時方向盤正直 轉向後方向盤自動回正減少汽油消耗 ３、什麼情況下，需要進行四輪定位?每行駛10000公里或六個月後直線行駛時車子往左或往右拉直行時需要緊握方向盤直行時方向盤不正感覺車身會漂浮或搖擺不定前輪或後輪單輪磨損安裝新的輪胎後碰撞事故維修後換裝新的懸掛或轉向有關配件後新車每行駛3000公里後

　　1． 前束(Ｔoe)：

　　從汽車的正上方向下看，由輪胎的中心線與汽車的縱向軸線之間的夾角稱為前束角。

　　輪胎中心線前端向內收束的角度為正前束角，反之為負前束角。

　　總前束值等於兩個車輪的前束值之和，即兩個車輪軸線之間的夾角。

　　☆作用：

消除車輪外傾造成的不良後果.車輪外傾使前輪有向兩側張開的趨勢,由於受車橋約束，不能向外滾開，導致車輪邊滾邊滑，增加了磨損，有了前束後可使車輪在每瞬間的滾動方向都接近於正前方，減輕了輪轂外軸承的壓力和輪胎的磨損。２、外傾（Camber）：

　　從汽車正前方看，汽車車輪的頂端向內或向外傾斜一個角度，稱為車輪的外傾。

通常情況下汽車的側傾角為外傾。用偏離垂直線所傾斜的角度來表示，如果頂端向外傾斜則稱為正外傾角，如果向內傾斜則稱為負外傾角。

　　☆作用：

為了增加汽車直線行駛的安全性。當具有外傾角時，可使車輪在轉向時偏移量減小，所以能減少轉向力；另外，由於主銷外傾，在垂直載荷作用下產生一施加於軸心上的分力，使車輪向內壓在軸承上，以防止車輪甩脫。３、主銷後傾角（Kingpin caster angle）：從汽車的側面看，主銷軸線（或車輪轉向軸線）從垂直方向向後或向前傾斜一個角度稱為主銷後傾或前傾。在縱向垂直平面內，主銷軸線與垂線之間的夾角，稱為主銷後傾角。向垂線後面傾斜的角度稱為正後傾角，向前傾斜的角度稱為負後傾角。通常汽車行駛過程中，主銷後傾角應為正值。主銷後傾角的獲得一般是在安裝時，通過懸架元件相互位置來保證的。☆作用：是當汽車直線行駛偶然受外力作用而稍有偏轉時，主銷後傾將產生車輪轉向反方向的力矩使車輪自動回正，可保證汽車直線行駛的穩定性。後傾角越大車速越高，穩定力矩越大，但後傾角不宜過大,否則在轉向時會導致轉向沉重,主銷後傾角是在前橋連同懸架安裝到車架時而形成的.

　　４、 主銷內傾角（Kingpin inclination angle）：

　　從汽車的正前方看，主銷（或轉向軸線）的上端略向內傾斜一個角度，稱為主銷內傾。

在汽車的橫向垂直平面內,主銷軸線與垂線之間的夾角稱為主銷內傾角。

　　☆作用：

　　是使車輪在受外力偏離直線行駛時，前輪會在策略作用下自動回正。

主銷內傾還可減少前輪傳至轉向機構上的衝擊，並使轉向輕便。但內傾角不宜過大，否則在轉向時,會使輪胎磨損加快,主銷內傾角一般在前軸製造時形成。

　　１、 推進線(Thrust line):

汽車後輪總前束的夾角的平分線就稱為汽車的推進線。２、 推進角（Thrust angle）：推進線與幾何中心線之間的夾角稱為推進角。3、左橫向偏置角（Left Lateral offset）：

　　左側前後輪中心的連線與推進線之間的夾角稱為左橫向偏置角。

　　當左後輪比左前輪更向外偏時，此狀態下規定左橫向偏置為正值，反之，當左前輪比左後輪更向外偏時，此狀態下規定左橫向偏置為負值。

　　如果汽車的規格值中，汽車的前後軸距已經知道，則左橫向偏置既可以用角度值來表示，也可以轉換成長度值來表示。

　　4、右橫向偏置角（Right Lateral offset）:

　　右側前、後輪中心的連線與推進線之間的夾角稱為右橫向偏置角。

當右後輪比右前輪更向外偏時，此狀態下規定右橫向偏置為正值，反之，當右前輪比右後輪更向外偏時，此狀態下規定右橫向偏置為負值。如果汽車的規格值中，汽車的前後軸距已經知道，則右橫向偏置既可以用角度值來表示，也可以轉換成長度值來表示。

　　5、 輪距差：（Track Width Difference）

　　左側前後輪中心連線與右側前後輪中心連線之間的夾角稱為輪距差。

當後輪距寬度比前輪距寬度大時，此狀態下規定輪距差為正值，反之，當後輪距寬度比前輪距寬度小時，此狀態下規定輪距差為負值。如果在汽車的規格值中，汽車的前後軸距已經知道，則輪距差既可以用角度值來表示，也可以轉換成長度值來表示。

　　6、 軸偏置（Axle Offset）：

　　輪距差角的平分線與推進線之間夾角稱為汽車的軸偏置。

當汽車的後輪軸向右偏時，此狀態下規定軸偏置為正值，反之，當汽車的後輪軸向左偏時，此狀態下規定軸偏置為負值。如果在汽車的規格值中，汽車的前後輪距已經知道，則軸偏置既可以用角度值來表示，也可以轉換成長度值來表示。

　　7、前退縮角（Front Set Back）：

　　兩前輪中心的連線與推進線的垂線之間的夾角，稱為前退縮角。

當右前輪在左前輪後面時，此狀態下規定汽車的前退縮角為正值，反之，當右前輪在左前輪前面時，此狀態下規定汽車的前退縮角為負值。如果在汽車的規格值中，汽車的前輪距已經知道，則前退縮角既可以用角度值來表示，也可以轉換成長度值來表示。

　　8、後退縮角（Rear Set Back）：

　　兩後輪中心的連線的垂線與推進線之間的夾角，稱為後退縮角。

當右後輪在左後輪後面時，此狀態下規定汽車的後退縮角為正值，反之，當右後輪在左後輪前面時，此狀態下規定汽車的後退縮角為負值。如果在汽車的規格值中，汽車的後輪距已經知道，則後退縮角既可以用角度值來表示，也可以轉換成長度值來表示。

　　9、包容角（Included angle）：

從汽車正前看，主銷的軸線和車輪的軸線之間的夾角就稱為包容角。它在數值上等於主銷的內傾角和輪胎的側傾角之和。

　　10、轉向前展（Toe-Out-On-Turn）：

在方向盤轉正的情況下，汽車的兩個前輪的前束值都為零時，當方向盤轉動到一側最大轉向角的狀態，此時汽車的兩個前輪端面之間的夾角就稱為汽車的轉向前展。

　　四輪定位­

　　*定義：經專用檢測儀器檢測判斷出四輪各角度與原設計和出廠時調整值的差值，再用專用零件或工具進行調整校正，使之與原設計和出廠調整值相同，或在公差範圍內的全過程。­

　　*功能：設計出汽車四輪各種角度，是為了保證汽車行駛的平穩性、安全性和經濟性。不會造成汽車跑偏、發飄、側滑、轉向不穩、輪胎磨損、及懸掛系統零件磨損加劇。­

　　二、主銷後傾角 (Caster)



　　※定義：上球頭向前或向後的傾斜度，或者支柱頂端與下球頭所夾的角度。

※功能：影響汽車行駛的穩定性和自動回正能力。※0主銷後傾角：從汽車的一側觀察，上球頭或支柱頂與下球頭在一個平面。※負主銷後傾角：從汽車的一側觀察，上球頭或支柱頂端朝下球頭方向向前。※症狀：1、旋轉一週後，自動回正能力缺乏。2、汽車在高速行駛時需小心注意。※後果：使汽車跑偏。舉例：左側前輪正主銷後傾角設為1/2°，右側前輪為1—1/2°，這種車將會向左偏。三、車輪外傾角 (Cember)



　　※定義：車輪的上沿向內或向外傾斜的角度。

※功能：調整車輛在負載時，使受力點作用於輪胎中心消除跑偏， 減少輪胎磨損。※0車輪外傾角：車輪處於垂直於水平面的位置。※正車輪外傾角：當車輪上沿向外傾斜，或偏離引擎。※過大的正車輪外傾角會引起：1、磨損輪胎外側。2、額外增加懸掛部件磨損。3、造成跑偏。※負車輪外傾角：當車輪上沿向內傾斜，或朝向引擎。※過大的負車輪外傾角會引起：1、磨損輪胎內側。2、額外增加懸掛部件磨損。3、造成跑偏 不相等的車輪外傾角：造成跑偏。舉例：左側前輪正車輪外傾角設為1°，右側前輪設為 1/2°，這種車將會向左偏。

　　四、退縮角 (Setback)



　　※定義：一邊車輪比另一邊車輪後退。（如圖三所示）

引起原因：1、廠家原因（廠家生產問題）。2、碰撞。正常情況下靠近0.5"的退縮角是沒有任何問題的。五、前束角 (Toe)



　　※定義：描述從車的正上方看，車輪的前端和車輛縱線的夾角，（如圖四所示）

※功能：減少輪胎磨損和滾動摩擦。正前束：車輪前端向內傾（內八字）。負前束：車輪前端向外傾（外八字）。使用情況：1、在比較滑的場地可以使用少量的前輪正前束，前輪使用正前束可以幫助你的車在加速時保持穩定。但這樣做也會減少車子在進入彎角時的轉向反應，同時增加加速出彎時的轉向反應。2、如果需要在進入彎角時獲得更多的轉向，可以使用負前束。但這樣做會使車子在加速時、或者通過起伏路面時，變得不穩定，偏離直線。3、在平路房車的情況，推薦使用0度到1角的負前束。4、越大角度正前束或負前束，越會減低車輛在直線行走的速度。5、不應使用過大的正前束或者負前束。6、正前束或負前束可以配合Ackerman使用。可以使用一定量的負前束配合較小的Ackerman效應，增加轉向前段（開始入彎）反應並減少轉向後段（在彎中）的反應；使用一定量的正前束，並配合較大的Ackerman效應，減少轉向前段反應並增加轉向後段的反應。六、轉向角 (Turning Angle)



　　※定義：汽車轉向時，前輪所轉的角度。

※功能：1、阻礙輪胎側滑。2、減少輪胎磨損。3、減少汽車轉向時的響聲。七、推進角



　　※定義：車子行進的方向是所謂的推進線，執行後輪個別前束

測量的時候，必須以車子的中線為參考，計算出一個後輪個別 前束的平均值，並畫出一條一條與後輪輪軸中央垂直的線，推 進線就出來了，推進線與幾何中線之間的夾角就是推進角。※推進角造成的影響：（1）輪胎磨損 （2）轉向輪失效 （3）跑偏 （4）車身不規則的直行 （5）方向盤偏斜 ※推進角的修正方式：（1）原車調整器；（2）在輪軸與輪轂之間加裝楔形墊片；（3）凸輪或其它加裝調整器；八、內傾角 (SAL) ※定義：由車輛前方觀察，轉向軸線與車輛垂直線的夾角大多數車輛來說是不可調整的角度。九、包容角 (Toe) ※定義：主銷內傾角與車輪外傾角之和稱為包容角。十、摩擦半徑 (Serrb Radius) ※定義：在地面上觀察，主銷內傾角的延長線與地面的交點輪胎中心線延長與地面的交點的距離稱為摩擦半徑。主銷內傾角的延長線與地面的交點在輪胎中心線延長線與地面的交點在輪胎中心線延長線與地面的交點內側，稱為正摩擦半徑：反之稱為負摩擦半徑。※功能：增加操縱穩定性、轉向後自動回正能力。
