機器學習：監督學習模型SVM（支持向量機）概述

2020-10-29
| 科學

理論

該算法的主要任務是找到最正確的線或超平面，它將數據分為兩類。SVM是一種接收輸入數據並返回這樣一條分界線的算法。請考慮以下示例。假設我們有一個數據集，我們想要對藍色圓圈中的紅色方塊進行分類和劃分（例如，正面和負面）。這項任務的主要目標是找到將兩個類分開的“理想”線。

找到將數據集劃分為藍色和紅色類的完美線或超平面。

乍一看，這並不困難，對吧？

但是，正如您所看到的，沒有一條特定的線可以解決這樣的問題。我們可以選擇無數個這樣的行來劃分這兩個類。SVM究竟如何找到“理想”線，“理想”在理解中意味著什麼？

您認為哪條線最好呢？

如果你選擇黃線，我祝賀你：這是算法選擇的那條線。在這個例子中，我們可以直觀地理解黃線是分開的，因此比綠色更好地分類兩個類

在綠線的情況下 - 它太靠近紅色類。儘管它正確地對當前數據集中的所有對象進行了分類，但 對於不熟悉的數據集，它不會表現得那麼好。解決分類類問題是機器學習的主要任務之一。

SVM如何找到最佳線

SVM算法的設計使得它在圖上尋找直接位於最接近的分界線上的點。這些點稱為支持向量。然後，算法計算參考矢量和分割平面之間的距離。這個距離稱為間隙。該算法的主要目標是最大化間隙距離。最好的超平面是這樣的超平面，其間隙儘可能大。

很簡單不是嗎？

但是讓我們考慮以下示例，使用更復雜的數據集，無法線性劃分。

顯然，這個數據集不能線性劃分。我們無法繪製一條直線來對這些數據進行分類。但是，可以通過添加一個額外的維度（我們稱之為Z軸）來線性劃分此數據集。想象一下，Z軸上的座標由以下約束控制：

z =x²+y²

因此，縱座標Z由點到軸的起點的距離的平方表示。

下面是Z軸上相同數據集的可視化。

現在可以線性劃分數據。假設品紅線分離數據z = k，其中k是常數。如果z =x²+y²，那麼k =x²+y²- 圓的公式。通過這種方式，我們可以使用此轉換設計我們的線性分離器，返回到原始樣本維數。

因此，我們可以通過向其添加額外維度來對非線性數據集進行分類，然後使用數學變換將其恢復為原始形式。但是，不是所有數據集都可以輕鬆地轉換這種轉換。幸運的是，在sklearn庫中實現此算法為我們解決了這個問題。

超平面

現在我們熟悉了算法的邏輯，讓我們繼續討論超平面的正式定義

超平面是n維歐氏空間中的n-1維子平面，其將空間劃分為兩個獨立的部分。

例如，假設我們的線被表示為一維歐幾里德空間（即，我們的數據集位於一條直線上）。在這條線中選擇一個點。這一點將數據集（在我們的例子中是線）分成兩部分。線有一個度量，點有0個度量。

對於我們之前遇到的二維數據集，分界線是相同的超平面。簡單地說，對於n維空間，有一個n-1維超平面，將這個空間分成兩部分

Python編碼

import numpy as np X = np.array([[-1, -1], [-2, -1], [1, 1], [2, 1]])y = np.array([1, 1, 2, 2])

這些點表示為X的數組，以及它們所屬的類作為y的數組。現在我們將用這個樣本訓練我們的機器學習模型。在本例中，我設置了分類器（kernel）的“kernel”的線性參數。

from sklearn.svm import SVC clf = SVC(kernel='linear')clf = SVC.fit(X, y)

預測新對象的類

prediction = clf.predict([[0,6]])

參數調整

參數是您在創建分類器時傳遞的參數。下面我列出了一些最重要的可定製SVM選項：

“C”

此參數有助於調整“平滑度”與訓練樣本對象分類準確度之間的細線。“C”值越高，訓練樣本的對象就越正確分類。

在此示例中，我們可以為此特定樣本確定若干決策閾值。注意線（在圖上表示為綠線）決策閾值。它非常簡單，因此，幾個對象被錯誤地分類。這些被錯誤分類的點在數據中稱為異常值。

我們還可以調整參數，最終得到一條更彎曲的線(淺藍色的決策閾值)，它將正確地分類訓練樣本的所有數據。當然,在這種情況下,我們的機器學習模型可以推廣的可能性和在新數據顯示同樣好的結果是災難性的。因此，如果您試圖在訓練機器學習模型時達到準確的目標，您還應瞄準更均衡的目標。

C越大，相當於懲罰鬆弛變量，希望鬆弛變量接近0，即對誤分類的懲罰增大，趨向於對訓練集全分對的情況，這樣對訓練集測試時準確率很高，但泛化能力弱。C值小，對誤分類的懲罰減小，允許容錯，將他們當成噪聲點，泛化能力較強。

gamma

在官方文檔中，SciKit Learn庫聲明gamma確定數據集中每個元素在定義“完美線”時會產生多大影響。伽瑪越低，即使是離分界線很遠的那些元素，也會參與選擇這條線的過程。但是，如果伽馬值很高，那麼算法將僅“依賴”那些最接近線本身的元素。如果將伽瑪級別設置得太高，則只有最接近該線條的元素才會參與確定線條位置的過程。這將有助於忽略數據中的異常值。SVM算法的設計使得在做出決策時相對於彼此最緊密的點具有更大的權重。然而，通過“C”和“gamma”的正確調優，就有可能得到一個最優的結果，這將構建一個忽略離群值的線性超平面，從而更易於泛化。
