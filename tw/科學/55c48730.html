2020-11-14
| 科學

儀器簡介：SVM™3001+Abbemat

斯塔賓格黏度儀和折光儀連用系統

安東帕新款斯塔賓格SVM™系列黏度儀：

它操作特別簡便測量快速，節能環保，它為我們的客戶提供市場上的特殊解決方案：不僅測量黏度，更可通過單次少量注射直接獲得多個參數。從運動黏度、動力黏度再到密度測量以及更多—更多豐富的選擇。SVM™ 3001 或 SVM™ 4001 測量的每一個參數均遵循以下業界標準。

  • 運動黏度 (NB/SH/T 0870-2013、ASTM D7042、EN16896、DIN 51659)
  • 密度 (GB/T 29617-2013、EN ISO 12185、ASTM D4052、IP 365)
  • 動力黏度 (NB/SH/T 0870-2013、ASTM D7042)
  • 黏度指數 (VI) (ISO 2909, ASTM D2270)
  • API 度 (ISO 91, API 2540, ASTM D1250, IP 200)
  • 賽氏黏度 (ASTM D2161)


高性能系列折光儀能夠在原料加工，中間體到成品的檢測過程中實現完美的常規分析和質量控制。它們健全的設計和簡明易懂的操作使Abbemat 300/500 成為實驗室得力助手。對於折光率的準確度，Abbemat 300為0.0001nD，Abbemat 500為0.00002 nD。
