一，首先來了解一下plot（x,y ）函數吧

  1. x和y為長度相同的向量
  2. plot（x,y ）是繪製二維圖形的最基本函數
  3. 例矩陣x=[1 2；3 4] ; y=[5 6; 7 8];plot（x,y )
  4. 運行一下上例
x=[1 2;3 4] ; %給定X矩陣y=[5 6; 7 8]; %給定X矩陣plot(x,y ) %繪製圖形

運行結果

6.修改這條線顏色

  1. matlab 曲線參數設置表

x=[1 2;3 4] ; %給定X矩陣y=[5 6; 7 8]; %給定X矩陣plot(x,y,'-.or' ) %繪製圖形%用點畫線，圓圈點形，紅色修飾

2.看結果（發現變成了點畫線，紅色修飾）

二，一起實戰吧！

1．生成1×10 維的隨機數向量a，分別用紅、黃、藍、綠色繪出其連線圖、杆圖、階梯圖和條形圖，並分別標出標題“連線圖”、“杆圖”、“階梯圖”、“條形圖”。 繪製函數曲線，要求寫出程序代碼。

①連線圖

a=rand(1,10)plot(a,'r')title('連線圖')

②杆圖

a=rand(1,10)stem(a,'y')title('杆圖')

④階梯圖

a=rand(1,10)stairs(a,'b')title('階梯圖')

⑤條形圖

a=rand(1,10)bar(a,'g')title('條形圖')

2. 在區間[0:N/20]均勻的取50個點，構成向量t （N=3位數）； 在同一窗口繪製曲線y1=sin(2*t-0.3)； y2=3cos(t+0.5)；要求y1曲線為紅色點劃線，標記點為圓圈；y2為藍色虛線，標記點為星號。 分別在靠近相應的曲線處標註其函數表達式。

函數功能

subplot是MATLAB中的函數。

使用方法：subplot（m,n,p）或者subplot（m n p）。

subplot是將多個圖畫到一個平面上的工具。其中，m表示是圖排成m行，n表示圖排成n列，也就是整個figure中有n個圖是排成一行的，一共m行，如果m=2就是表示2行圖。p表示圖所在的位置，p=1表示從左到右從上到下的第一個位置。

在matlab的命令窗口中輸入doc subplot或者help subplot即可獲得該函數的幫助信息。

linspace是MATLAB中的函數

用法：linspace(x1,x2,N)

功能：linspace是Matlab中的均分計算指令，用於產生x1,x2之間的N點行線性的矢量。其中x1、x2、N分別為起始值、終止值、元素個數。若默認N，默認點數為100。

在matlab的命令窗口下輸入help linspace或者doc linspace可以獲得該函數的幫助信息。

①代碼

clear all;t=linspace(0,135/20,50)y1=sin(2*t-0.3);y2=3*cos(t+0.5);subplot(1,2,1) %分成兩個區域，即1*2plot(t,y1,'-.or')text(5,0.8,'sin(2*t-0.3)') %確定表達式的位置subplot(1,2,2)plot(t,y2,'--b*')text(1.5,2.4,'3*cos(t+0.5)') %確定表達式的位置
