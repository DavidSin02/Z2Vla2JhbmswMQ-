承前自定義功能區菜單及工具欄(2/3)，上回書說到自定義工具欄。

4. 自定義工具欄

1. 打開一個新的表格文件，另存為【Excel加載宏(*.xlam)】or【Excel 97-2003加載宏(*.xla)】

2. 添加【模塊】，複製下面代碼到模塊內。

Auto_Open顧名思義，以後用來自動運行的，myButton跟myG是定義按鈕及組的，目前的功能區按鈕跟組的界限也不是很明顯，CreateLineStyleButton的參數較多，分別是按鈕所在的工具欄、鼠標懸停顯示的註釋、按鈕在工具欄的位置、調用的宏名稱、使用圖標序號。

3. Arial9就是一個規範字體的宏範例，具體的不多講，在Auto_Open位置按F5即可見工具欄添加至【加載項】了，試試看。

Private Sub Auto_Open()

CreateLineStyleButton

End Sub

Private Sub myButton(myCmd As String, myname As String, _

mynum As Integer, mycom As String, mid As Integer)

On Error Resume Next

Set newButton = Application.CommandBars(myCmd).Controls.Add( _

Type:=msoControlButton, Before:=mynum)

With newButton

.Style = msoButtonIcon

.Width = 30

.BeginGroup = False

.Caption = myname

.OnAction = mycom

.FaceId = mid

End With

End Sub

Private Sub MyG(myCmd As String, myname As String, _

mynum As Integer, mycom As String, mid As Integer)

On Error Resume Next

Set NewG = Application.CommandBars(myCmd).Controls.Add( _

Type:=msoControlButton, Before:=mynum)

With NewG

.BeginGroup = True

.Style = msoButtonIcon

.Width = 30

.Caption = myname

.OnAction = mycom

.FaceId = mid

End With

End Sub

Private Sub CreateLineStyleButton()

On Error Resume Next

Dim mynum As Integer, myname As String, mycom As String, mid As Integer

Dim myPosition As Variant

myPosition = msoBarTop

Application.CommandBars("nAzyLineStyle").Delete

Application.CommandBars.Add(Name:="nAzyLineStyle", _

Position:=myPosition).Visible = True

Call MyG("nAzyLineStyle", "Arial9", 1, "Arial9", 289)

Call MyG("nAzyLineStyle", "AuditLineUp", 2, "Audit_Line_Up", 391)

Call myButton("nAzyLineStyle", "AuditLineDown", 3, "Audit_Line_Down", 117)

End Sub

Private Sub Arial9()

' Cells.Select

With Selection.Font

.Name = "Arial"

.Size = 9

.Strikethrough = False

.Superscript = False

.Subscript = False

.OutlineFont = False

.Shadow = False

.Underline = xlUnderlineStyleNone

.ColorIndex = xlAutomatic

End With

End Sub

OK，做好了自己常用的工具欄後，請放在固定的位置，我們可以通過加載項設置讓它在每次Excel啟動時自動運行。

4. 【選項】-【加載項】-管理Excel加載項，點擊【轉到】按鈕

通過【瀏覽】找到自己的工具欄xla或xlam文件，ok，設置好了，以後每次都會自動掛起您的常用工具欄了，事後，文件如果挪動了位置系統就找不到了。

5. 後臺自定義菜單

這個少數人才會用到，簡單講幾句就略過了，自定義菜單對xml文件規範性要求較高，因此想要涉獵這塊的同學，先下載參數附件。

百度網盤下載參數文件：微信公眾號，搜【財會人nAzy】，同課程內有鏈接地址。

效果，打開文件即可在功能區植入自己定義的菜單欄，使用大圖標。

1. 將excel文件（一般為xlsm，因為要有宏功能啊），後綴名修改為rar，通過壓縮包軟件打開後，將參數文件植入壓縮包內，原有的覆蓋或者刪除掉。

2. 刪掉rar後綴，打開xlsm文件即可見菜單欄出現。

3. 下圖是菜單定義customUI.xml文件內容，及負責指向的.rels文件，這兩個文件都需要通過【打開方式】-【記事本】可見內容。
