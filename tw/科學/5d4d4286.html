區塊鏈+基因，生命貨幣的想象空間

2020-11-14
| 科學

一、區塊鏈與基因鏈

區塊鏈 ( BlockChain ）本質上是一種分佈式的數據庫，其中區塊（Block）和鏈（Chain）與組成我們以及地球上所有生命的基因，有很多的相似性。

首先，細胞也是一個個的“塊”，細胞核裡的基因是螺旋式雙“鏈”結構。

其次，身體間的合作也靠“ token”激勵完成，比如能讓我們感到快樂的多巴胺、內啡肽等。

我常用一個啪啪啪的比喻來解釋 token 激勵的重要性，啪啪啪是生命系統的挖礦機制，卵子與精子的結合，如同哈希碰撞成功，而成功“爆塊”，啪啪啪的激勵 token 就是多巴胺、內啡肽等快樂幣。以至於，我們現在絕大部分啪啪啪並不是為了“爆塊”，而是獲取身體獎勵的快樂token。也正因此，有些啪啪啪最終促成了真正的目的——繁衍後代。

隨著基因測序成本的下降，也有人開始試著用區塊鏈這把錘子去找基因領域中的釘子。

圖：人類基因組測序成本2001-2017 來源：Nebula Genomics 白皮書

基因組研究目前最大的問題是：數據量不夠。量變才能引發質變，巧婦難為無米之炊。隨著基因測序成本的不斷下降，讓用戶個人擁有自身的全基因組數據成為可能，如同移動互聯網一樣，這將開啟一個前所未有的新維度。試想如果人人都有自己的基因組數據之後，將誕生怎樣的一個新產業。

二、生命貨幣的想象空間

我們首先要弄清楚，基因檢測在整個行業中的關係，誰需要基因數據。

提起基因、DNA，大部分人的第一印象應該是親子鑑定。這是最早的應用之一。如今，這項應用在司法領域已經非常成熟，前不久著名的“白銀殺人案”告破，也是它的功勞。

目前最成功的商業應用是：無創產前基因檢測，通過在孕婦血液中提取胎兒的遊離DNA，做一個檢測看看是不是唐氏兒。唐氏兒是一種嚴重的遺傳疾病，前些年的人造神話“音樂指揮家”舟舟就是典型的唐氏兒。無創產前是一個很大的市場，A股撐起了華大和貝瑞兩家上市公司。

還有就是癌症治療的靶向用藥，也需要做基因檢測。癌症是一種基因病，攻克的關鍵也在基因上。

另外，目前的新藥研發對基因數據的需求也很迫。我之前看到過一個新聞說：夏威夷原住民起訴了某家美國藥廠，因為這家藥廠的治療心血管疾病的藥物對他們無效。照例說無效的藥物是不會通過臨床測試的，這又是怎麼回事呢？原來這種藥物在臨床測試中對白人、黑人都有效，但偏偏對夏威夷的原住民無效。這源於夏威夷原住民特殊的基因差異。

此外，還有面向大眾用戶的消費類基因檢測。以 23andMe 和 ancestry 為代表，主要提供祖源分析和一些趣味解讀。其中 23andMe 還拿到了幾種遺傳疾病的監測證書。我嘗試過一次，幾百塊錢。其中祖源分析就像一個加強版的DNA鑑定，會告訴你百分之零點幾的xx國人成分，百分之幾點幾的xx民族成分。特別想吐槽一下：國家是政治概念，你護照上寫著什麼國家就是什麼國家；民族更多的是一個文化概念，你身份證上寫的什麼民族就是什麼民族。即使我測出一個百分百的少數民族成分，我去要求改民族和高考加分，估計也沒人鳥我。這個產品有點“為賦新詞強說愁”的感覺。從用戶需求的角度看，更像是一種高科技的體驗吧。“酒量”等項目確實很準，至於營養、性格就屬於玄學範疇了。

基因的價值，圍繞“人”展開，司法、健康、臨床、製藥。區塊鏈+ token 激勵，又能夠帶來怎樣的變革？

先說一個案例，這個例子在幾乎所有的基因組+區塊鏈項目白皮書中都有提及。也是神奇。

2015年，製藥巨頭基因泰克以6000萬美元的價格，向基因檢測公司 23andMe 購買了 3000 名帕金森病患者的全基因組數據。一方面，需求方付出了鉅額資金。另一方面基因數據的所有方——用戶，並未獲得直接收益。而作為中介方 23andMe 幾乎賺取了所有收益。

也就是說：我自己花錢做了基因檢測，基因公司把我的數據賣了，而且我還沒有得到任何收益？——也就是傳說中的被人賣了，還幫人數錢？？

6000萬美元買3000個人全基因組數據，平均每人2萬美元，摺合成人民幣要十多萬呢，不少了。

如果一定要找價值的話，基因泰克研發出的新藥，提供了一個新的治癒可能性，但是你仍要付費的。。。打折？不要想了。

這就給了區塊鏈介入其中的機會，在持之有故，言之成理的項目白皮書中，都介紹了這樣一個方案，以 Nebula Genomics 白皮書中的經濟模型為例。

圖：Nebula Genomics 白皮書中的經濟模型，來源：Nebula Genomics 白皮書

用戶通過出租自身基因數據獲取token獎勵，交易記錄登記到區塊鏈保存。有效避免被中介公司獨吞。

這裡還有一個疑似漏洞，基因樣本送去測序的時候，測序方總是有一份數據的。那他直接出售這些數據給需求方即可，為什麼還要費勁兒從用戶說手中買呢？或者說，用區塊鏈做這件事的必要性有哪些：

首先，需求方與測序方也存在一個猜疑鏈，需求方會擔心測序方編造一個數據來糊弄自己。加上區塊鏈之後，需求方至少能夠增加一個額外選項。

第二，需求方需要的是不僅僅是一個單純的基因數據，而且需要這個數據的主人的更多表型信息，比如體檢記錄、生活習慣等等。顯然這不是測序方能夠提供的。

第三，之前的用戶基因組數據被把持在各個公司手中，想要研究需要支付高額的價格之外，有些公司會因為複雜的控股關係，而故意不與一些研究機構和存在競爭關係的公司合作。這就造成了一個一個的數據孤島。寶貴的，屬於用戶的基因數據無法流通。

第四，接上條，這些被大公司把持的數據，本來就應該是用戶自己的。是每個人一出生就自帶的“生命資產”，決定這些資產的應該是用戶本人，而不是那些玩弄註冊協議的大公司。

但我們可以看到，在 Nebula Genomics 項目的構想中，第三方公司仍然需要向Nebula Genomics 支付 Fiat money （法定貨幣）。這個經濟模型就很糟糕了，等於第三方用法幣買的用戶的數據，Nebula Genomics 官方卻支付給用戶 token 。

對比迅雷鏈的生態，C端用戶向“星域雲”共享寬帶和存儲空間，獲取token：鏈克。B端客戶向“星域雲”購買CDN和其它雲計算服務，支付token：鏈克。同時，B端客戶又向C端用戶開放鏈克購買自身的服務。完整的 token 經濟鏈條，在運轉上沒有法幣的介入。

至少在經濟模型上 Nebula Genomics 還需再精進。當然，也說不定他們是故意的，畢竟企業運轉的現金流是用法幣來衡量的。

除此之外，基因數據也符合區塊鏈介入的一些前提條件：

  • 基因通過基因測序儀數字化，數字化就可以上鍊；
  • 基因數據人人不同，可以確權；
  • 基因數據本身存在變現價值。

隨著越來越多的人意識到自身基因數據的權益。基因數據上鍊，解決了基因數據的確權問題，讓基因數據交易的更加透明性。交易帶來的token激勵勢必會吸引更多人加入，人類基因組數據將愈加豐富。隨著數據量的增加，基於基因組的研究成果和應用也將更加豐富，新的生態將逐步構建起來。

未來生態的想象空間：

  • 精準醫療。現代醫療停滯不前，是平均化的分析方法遇到了瓶頸。所謂平均化醫學，就是畫一條平均線，偏離這條線就有問題。維度單一，且忽略了個體特徵。下一步，由個人基因組數據為依託的精準醫學，將充分尊重個體的差異的前提下，提供個性化的精準的醫療方案。這個市場預計將幾倍於目前的醫療市場。
  • 新藥研發。2016 年德勤會計事務所發佈的一份研究報告顯示：研發一個新藥的平均成本為 15.4 億美元，耗時 14 年。成功的項目收益頗豐，但鳳毛麟角，大批藥物死在臨床前和臨床階段，數億、數十億美元打水漂是藥物研發的常態。新藥研發只有資金實力雄厚的大藥企才玩得起。為了對衝風險，新藥研發早已是一門複雜的金融生意。結合 token 構建的新經濟模型和未來龐大的基因數據庫，在提高新藥研發精準度的同時，也將降低投資門檻。
  • 眾籌研究。理論上基因相近，對某些疾病的風險也相近。目睹過親屬患病情景，勢必也擔心自己的風險。針對某一類具體疾病或者藥物研發方向，利益攸關的用戶可以選擇以token 眾籌的形式支持研究，並通過智能合約約定收益分成。谷歌的創始人布林通過基因檢測就發現自己患老年痴呆的風險很高，直接大筆一揮贊助了多個老年痴呆症的項目研究。希望未來，人人都有機會參與。這才是真正的保險。否則到時候光有錢，沒辦法治也白搭。

三、項目分析

美國·Nebula Genomics

美國，作為目前藍星科技的第一強國，在製藥和基因組研究領域也是首屈一指。區塊鏈+基因自然也不能落後。

前文已提到的 Nebula Genomics 項目，由哈佛大學的遺傳學大牛 George Church 發起。白皮書裡除了前面吐槽的經濟模型之外，整體靠譜，頗有論文範兒。特別是對隱私保護的部分闡述了很多。 Nebula Genomics 項目在中國落地應該會給到其在中國的公司：奕真生物。感興趣的可以關注。

美國·LunaDNA

有一定的關注度，但社群規模遠遠比不上Nebula Genomics。官方遲遲沒有發佈正式版白皮書，之前有流傳過一份，但被官方否認了。LunaDNA 自曝是測序儀巨頭 illumina 的前員工，並且獲得了illumina一定資金的支持。目前瞭解到的都是概念規劃，還沒看到落地方案。

印度·Shivom

最近很火，感覺買了幣圈很多廣告位，但槽點很多。首先，Shivom 白皮書看完給我的印象就是兩個字：空談。核心的地方或一筆帶過，或避而不談——如何測序，如何存儲數據，如何構建經濟模型——細節統統沒有，口號倒是喊了不少，對代幣也描繪了很多。我還特意諮詢了生物界的專業人士，竟然對 Shivom 白皮書中好幾頁的核心團隊和理事會成員無一耳聞。感覺這是一個為韭菜量身定製的假大空方案。其中，白皮書還有一句說“直接與政府合作以推動對全民進行基因測序”。我就特別想讚美印度人民的天真無邪。

中國

這麼時髦的事情，怎麼可能沒有中國的身影呢。前前後後，我看到過的就有：GHT、GENE、DNAC 等等，其中，有一個 GeneData 的項目，在官網上將“顧問”的人名和照片都搞錯了（ http://dnabbs.com/t/topic/41 ），這還是基因領域著名的科學家。後來這個項目被爆各種造假，據說現在轉去做交易所了。

靠譜的也有，著重說一下：HGBC，人類基因組價值生態鏈。我已經買了一個他們的全基因組測序，還沒有到貨。

相比之下，這個項目更“以用戶為中心一些”。而且解決了我關注的幾個點：

1、合法合規的重要性。中國的《人類遺傳資源管理暫行辦法》明確規定了遺傳樣本未經許可不能出境。基因這麼敏感的數據，讓外國人來做的可能性為零。加之現在人為割裂的國外互聯網和國內互聯網，未來區塊鏈內外有別也是定局。基因+區塊鏈只有合法合規的一條道路可以走。

插播一個小故事，數年前，某日本節目通過 DNA 檢測發現某家喻戶曉的女影星的祖上來自雲南某少數民族，於是派出製作組工作人員去雲南採集基因樣本，進行驗證。。。然後，被我英勇的公安民警當成敵特事件，給一網打盡了。。。

2、技術積累。基因+區塊鏈，必須要有基因數據才能玩起來，那麼用戶以基因數據參與就好了。那些需要“幣”才能參與的項目，或許只是想騙取你的幣而已。雖然 HGBC 的基因檢測盒我還沒有收到，但從公開披露的信息看，其前身在基因領域還是有很多積累的。是國內少數幾家有能力解讀全基因組數據的公司。

3、中國速度。發達國家把路子走通，中國靠人海戰術淘汰所有競爭對手，最終搞成白菜價。似乎已經是一條必然路徑。什麼也拼不過 996，人多就是競爭力。像幣圈的主鏈項目，吹了一整年，才磨磨蹭蹭的準備上測試網絡，能不能成功還得等下個月。現在中國的項目，上來就是主鏈，根本不磨蹭。

4、迅雷鏈的大太子。基因組數據是一個很大的數據，上鍊的話對數據存儲空間有很大的要求。目前比較熱的分佈式存儲方案是 IPFS ，包括很多基因組項目也都聲稱要用 IPFS 來存儲基因數據。我對此並不看好，IPFS 這種分佈式無監管又自帶交易系統的存儲網絡，估計一上線就成為色情網站，自身的穩定性就難以保證。而另一個現成的，在 P2P 技術上絕對領先的是國內的迅雷鏈。依託於玩客雲設備共享的寬帶和存儲空間，這才是真的分佈式存儲和落地項目。加上迅雷一直符合國內的監管，HGBC 選用迅雷鏈做生態應用，非常明智。

5、實踐是檢驗真理的唯一標準。解決具體問題，而不是凹某個主義的造型。區塊鏈帶來的去中心化，解決了傳統的中心化世界的一些弊端，但不代表去中心化是萬能藥，也不能把中心化一棒打死。最終還是要落在要解決的具體問題上。如果這個問題是棵樹，那麼既要中心化的樹幹，又要去中心化的樹葉。

小結一下：

  • 基因生物產業的未來有極大的想象空間
  • 基因與區塊鏈的結合能夠助力產業發展
  • 基因數據要在國內做成，必須合法合規
  • 基因區塊鏈項目必須有基因數據才能運轉

四、風險

事情雖好，風險也不小。

先分享一個簡單的評估方法：風險與收益的四象限分析

圖：風險與收益四象限分析

  • 第 1 象限：高風險低收益。借錢是典型，如果對方不還錢，損失很大；如果對方還錢，也基本沒有收益。從金錢回報上來看是典型的高風險低收益。
  • 第 2 象限：高風險高收益。真正的投資象限，風險是一道門檻，誰能駕馭風險，誰就能拿到收益。
  • 第 3 象限：低風險低收益。就是存款了；
  • 第 4 象限：低風險高收益。一個美好的願望，是太多騙局偽裝的對象。

我的原則是隻在2、4活動，遠離1、3。遠離 1 很容易，遠離 3 看起來很奇怪。其實也很簡單，因為有這種好事，早被別人填平了。除非你有某種切實可靠的壟斷性的資源，然而這又豈是我等凡輩可企及的。要不然就是沒有理解其中隱藏的風險，比如那些被精心設計的騙局。如果有人告訴我有什麼無風險高收益的項目，我會繞到後面看看他手裡握的是什麼鐮刀？割韭菜的小鐮刀，還是死神用來割腦袋的大鐮刀。

第 4 象限是賺不到錢的，那就只有第 2 象限了。在第2象限比拼的就是風險管控能力。

區塊鏈的風險：歸零、被盜、漏洞、空氣幣.... 加上基因之後，還有幾個獨特的風險，瞭解一下：

1、隱私

《三體》中描述了一個場景，ETO組織用基因技術製造了一個病毒，這個病毒只對面壁者羅輯一個人進行定向攻擊，也就是傳說中的“基因武器”。當然這種科幻小說中的場景，如能實現，如博物學家劉大可所言：能拿一摞諾貝爾獎。用此去殺一個人，簡直是核武器打蚊子。

更靠譜的擔心是，基因信息這種高科技帶來的未知性，比如我換某某癌症的風險很高，會不會遭遇基因歧視，影響找媳婦之類的。

對於參與方隱私保護的方案 Nebula Genomics 在白皮書中對此闡述很多，相信區塊鏈運用的零知識證明等加密技術可以解決大部分問題。而我個人之所以選擇加入進去，是因為預期收益可以覆蓋掉預期損失。

2、生物難題

21世界是生物的世紀，但21世紀有100年。能不能在有生之年看到生物的崛起。是個很大的風險。一直以來，生命科學領域一直都是資本巨頭的玩家，有這麼一次機會非常難得，長遠看能為人類的科研道路貢獻一份力量也是好的吧。

3、token價值

首先，經濟模型非常重要，這代表著以後你手裡的 token 的價值，如上文所述 Nebula Genomics 的經濟模型中，做為核心驅動力的需求方，支付的卻是法定貨幣，那用戶手中 token 就沒有基礎的價值錨定。而像 Shivom 這種割韭菜的項目，只說自己的 token 能做一二三四五，但如何保證實現或隻字未提，或異想天開。

大部分產業是為房東打工。而 token 是整個生態的核心資產，早期參與者的獎勵在未來能否有價值，需要看經濟模型是否合理。

其次，當基因數據通過區塊鏈確權給個人之後，生態組織方能否繼續把用戶組織在一起使用生態中的 token。很考驗團隊的運營能力。也有可能，隨著明星應用的崛起，各種生命類 token 開始自由競爭。去中心化的時代，你無法像法幣一樣強制大家使用某種 token ，所以早期參與者領到的 token 不一定是最後的勝利者，甚至都不一定有勝利者。像癌症治療可能會有一個某某癌症的token，某某藥物會有某某藥物的token。好在它們彼此之間會有兌換流通。所以，基因數據可能才是最核心的資產。這不是一個一勞永逸的事情，時刻關注生態發展變化，多參與新項目和及時調整分配策略，才是正途。

對區塊鏈經濟模型的分析、新項目的評估標準等，貨幣進化論會在未來的文章中詳細闡述，歡迎繼續關注。
