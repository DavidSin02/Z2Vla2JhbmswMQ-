什麼是天敵昆蟲？

天敵昆蟲是一類長期在農田、林區和牧場中控制著害蟲的發展和蔓延的昆蟲。分為捕食性天敵昆蟲和寄生性天敵昆蟲。


+ 捕 食 性 天 敵 +


+ 螳 螂 +




螳螂是比較凶悍的昆蟲，它那雙鋒利無比的“螳螂刀”足以讓人望而生畏。

螳螂可捕食多種害蟲，如蚊、蠅、鱗翅目幼蟲等小型昆蟲，蟬、飛蝗、螽斯等大型昆蟲。


常見的比如，美麗的薄翅螳Mantis religiosa，在棉田已知可捕食棉鈴蟲、地老虎、煙青蟲、粘蟲、盲蝽、棉蚜、葉蟬、螻蛄、棉小造橋蟲。

此外，以“中華”命名的中華大刀螳Tenodera sinensis 以及廣捕性的廣斧螳（廣腹螳螂） Hierodula patellifera、枯葉大刀螳Tenodera aridifolia都是著名的農田益蟲。



廣腹螳螂


+ 捕 食 性 瓢 蟲 +



瓢蟲種類很多，全世界約有6000多種，我國記載700餘種，其中80%以上為捕食性的，15%為植食性的，另外5%為菌食性的。

我們最為熟悉的瓢蟲，無疑是童年夥伴——七星瓢蟲，它紅紅的殼子，黑亮的斑點，十分迷人。難得的是七星瓢蟲不僅長得可愛，還是地地道道的捕蟲能手：據統計，1只七星瓢蟲一天可以吃掉100多隻蚜蟲，有的地方稱其為“麥大夫”。


瓢蟲小時候雖說貌醜，但也是捕蟲高手。


除了熟悉的七星瓢蟲，還有很多捕食性瓢蟲。它們分佈廣泛，可捕食蚜蟲、葉蟎、粉蝨、介殼蟲等等，是控制農業害蟲的重要天敵。


其中著名的比如：捕食棉蚜、豆蚜、高粱蚜、菜縊管蚜，且千變萬化的異色瓢蟲Harmonia axyridis：



異色瓢蟲


捕食麥長管蚜、禾穀縊管蚜、棉蚜、豆蚜的十三星長足瓢蟲（十三星瓢蟲）Hippodamia tredecimpunctata ：


十三星瓢蟲


捕食棉蚜、麥蚜、玉米蚜、高粱蚜的龜紋瓢蟲Propylea japonica：



龜紋瓢蟲


此外，黑緣紅瓢蟲、澳洲瓢蟲、紅頸瓢蟲、中華顯盾瓢蟲、連斑小毛瓢蟲等等都是極其重要的農田天敵昆蟲。

+ 步 甲 +



步甲，俗稱步行蟲，它們大多為捕食性，在自然界消滅害蟲方面起著一定作用。中國的金星步甲大量捕食鱗翅目幼蟲，是粘蟲等害蟲的重要天敵，同時還捕殺蝸牛等有害軟體動物。



黃緣心步甲


黃緣心步甲Nebria livida能夠捕食粘蟲、地老虎、菜青蟲、蝗蟲以及有害的蝽類。



雙斑青步甲


雙斑青步甲Chlaenius bioculatus，又稱黃星步行蟲，它初孵幼蟲經1個小時爬行後，即可捕食，能捕食比它體大數倍的鱗翅目幼蟲，並有群集捕食大型幼蟲的特性。

+ 隱 翅 蟲 +



隱翅甲科的昆蟲的鞘翅很短，後翅隱藏在前翅下。




隱翅甲一部分為腐食性、糞食性的昆蟲，可以促進自然界物質的循環；另外一些以菌菇、植物的果實以及花粉等為食物；而大部分以肉食為主，靠捕食農林害蟲為生。




比如，黑脛虎隱翅蟲Stenus cicindeloides捕食蚜蟲、葉蟬、薊馬、鱗翅目昆蟲。青翅毒隱翅蟲Paederus fuscipes捕食玉米螟、棉葉蟬、棉盲蝽、棉小造橋蟲、棉鈴蟲、棉蚜、薊馬等等。

+ 蠼 螋 +





蠼螋[qú sōu]，是革翅目的雜食性昆蟲，別稱夾板子、剪指甲蟲、夾板蟲，蠼螋常常生活在樹皮縫隙，枯朽腐木中或落葉堆下，喜歡潮溼陰暗的環境。其實，很多蠼螋是農田中常見的天敵昆蟲。




例如，廣捕性天敵昆蟲的青足小肥螋Euborellia pallipes、日本球螋Labidura japonica能夠捕食小老虎、棉鈴蟲、棉小造橋蟲、鼎點金剛鑽、斜紋夜蛾、紅鈴蟲、短額負蝗、棉蚜等多類害蟲。


+ 胡 蜂 +


胡蜂，可是昆蟲界有名的“暴脾氣”外加不好惹，不曾想它們也是我們的農田友軍。




北方黃胡蜂Vespula rufa喜捕食粘蟲、銀紋夜蛾；普通長腳胡蜂捕食棉鈴蟲、棉小造橋蟲、豆莢螟、粟灰螟、菜青蟲。此外，紋胡蜂、黃斑細腳胡蜂、日本長黃胡蜂等也都是農田益蟲。


北方黃胡蜂


+ 蜾 蠃 +


“螟蛉有子，蜾蠃負之”，說的便是蜾蠃[guǒ luǒ]的故事，螟蛉是指鱗翅目昆蟲的幼蟲，蜾蠃把螟蛉銜回巢中，後作為自己孩子的食糧。



蜾蠃和胡蜂外形有些相似，平時都是自由活動，僅在產卵時才銜泥築巢（或尋找竹管），之後捕捉鱗翅目幼蟲，用自己尾上的毒針麻醉幼蟲，在其身上產卵，供後代的食用。可見蜾蠃是鱗翅目的天敵昆蟲。



農田常見的比如棕喙蜾贏（黃脣蜾蠃）Rhynchium bruneum，產卵時每粒卵下平均放15條左右棉鈴蟲等鱗翅目幼蟲。前旁喙蜾蠃捕食銀紋夜蛾幼蟲，四帶佳盾蜾蠃和鑲黃蜾蠃則捕食菜青蟲。



+ 土 蜂 +




土蜂科的土蜂，其幼蟲是蠐螬的外寄生者。雌蟲在土中挖洞尋找寄主，發現蠐螬先用蜇刺注入毒液，使蠐螬麻醉，再產卵其上，封閉土室。



金毛長腹土蜂


日本土蜂寄生於油桐綠麗金龜；金毛長腹土蜂寄生於方頭綠麗金龜、櫻桃麗金龜、銅色麗金龜；白毛長腹土蜂寄主為大黑鰓金龜、異麗金龜。

+ 草 蛉 +



草蛉是脈翅目草蛉科的昆蟲。說起草蛉我能想到的詞彙是端莊、優雅：纖細的身軀、細長的觸角、透明的雙翅，以及具有金屬光澤的複眼，彷彿穿梭於花間的靈秀女子。


只是，草蛉的幼蟲——蚜獅，可與親生父母沒有絲毫相像，樣貌大不同。既然稱為蚜獅，說明它也是蚜蟲殺手，據統計，一頭蚜獅在整個幼蟲階段能吃掉700頭以上的蚜蟲，真是好胃口造就好益蟲。


草蛉可是真正的捕捉害蟲的小能手，不僅是各類蚜蟲——棉蚜、菜蚜、菸蚜、麥蚜、豆蚜、桃蚜、蘋果蚜、紅花蚜，而且粉蝨、紅蜘蛛等等都是它們的捕食對象。


此外，草蛉還喜歡吃很多種害蟲的卵，諸如棉鈴蟲、地老虎、銀紋夜蛾、麥蛾和小造橋蟲的卵，都在其食物範圍之內。


捕食性草蛉 成蟲


+ 吸 食 性 蝽 類 +





蝽，就是我們熟悉的放屁蟲或者臭大姐，在我們的記憶中，它們又醜又難看，還生有臭腺，特別不招人喜歡。

然而實際上，蝽也是一個大家庭，全世界已知的就有4萬多種，造型繁多、千奇百怪，不僅有食素的蝽，也有不少捕食性蝽。



花蝽科、盲蝽科、姬蝽科、獵蝽科的蝽可捕食葉蟬、飛蝨、蚜蟲、薊馬、棉葉蟎及棉蛉蟲等害蟲。




花蝽科小花蝽屬的蝽，是一類具有重要利用價值的天敵。全世界已知有近百種，我國已知10餘種，在我國大部分農田中，常見的優勢種主要為東亞小花蝽、微小花蝽和南方小花蝽。小花蝽可捕食蚜蟲、薊馬和紅蜘蛛，以及紅鈴蟲、造橋蟲、棉鈴蟲等害蟲的卵和初孵幼蟲。



狡小花蝽 Orius insidiosus


盲蝽科的黑食蚜盲蝽和異須盲蝽捕食棉蚜。

姬蝽科的華姬蝽Nabis sinoferus是雜食性天敵昆蟲，在田間可捕食蚜蟲、鱗翅目昆蟲卵和初孵幼蟲等，亦可食用各種花粉。而暗色姬蝽Nabis stenoferus在棉田內主要捕食棉蚜、薊馬、盲蝽若蟲以及鱗翅目害蟲卵等。



黑食蚜盲蝽


此外，長蝽科的沙地大眼長蝽在棉田捕食棉蚜，而酷似蟬兒的大眼蟬長蝽是北方棉花害蟲的主要天敵。蝽科的蠋[zhú]蝽Arma chinensis喜歡榆紫葉甲及松毛蟲等鞘翅目和鱗翅目害蟲，也是極其知名的天敵昆蟲。


大眼蟬長蝽


+ 食 蚜 蠅 +



食蚜蠅，就是Cos小蜜蜂的傢伙，它們訪花采蜜，穿梭於花叢間，也如同小蜜蜂一般為花兒傳粉。

幼蟲“食蚜”，雖然模樣醜到原地爆炸：蛆形，有一些透明的質感，可以看到內部臟器，乍看好似一坨鳥糞。


但它們面醜心善、技能滿滿：一般一隻幼蟲一天可取食百餘隻蚜蟲，是防治蚜蟲的好幫手。當然，並非所有的食蚜蠅幼蟲都以捕食蚜蟲為生，也有些為腐食性或植食性。


食蚜蠅以捕食蚜蟲著稱，此外，還能捕食介殼蟲、粉蝨、葉蟬、薊馬和鱗翅目的幼蟲等。


比較常見知名的天敵包括：寬尾細腹食蚜蠅、短刺刺腿食蚜蠅、門氏食蚜蠅、短翅細腹食蚜蠅以及狹帶食蚜蠅等等。



門氏食蚜蠅


除此之外，捕蟲蟎、蜘蛛（均不是昆蟲）等也是極其重要的害蟲天敵。
