三、路堤施工技術

填料種類：填土、填土石、填石

2）填土路堤的填築技術

（1）填築方法

①水平分層填築法

②縱向分層填築法

③橫向填築

④聯合填築

①水平分層填築法：填築時按照橫斷面全寬分成水平層次，逐層向上填築。是路基填築的常用方法。

②縱向分層填築法：依路線縱坡方向分層，逐層向上填築。常用於地面縱坡大於12%、用推土機從路塹取料、填築距離較短的路堤。缺點是不易碾壓密實

③橫向填築：從路基一端或兩端按橫斷面全高逐步推進填築。由於填土過厚，不易壓實，僅用於無法自下而上填築的深谷、陡坡、斷巖、泥沼等機械無法進場的路堤

④聯合填築：路堤下層用橫向填築而上層用水平分層填築。適用於因地形限制或填築堤身較高，不宜採用水平分層填築或橫向填築法進行填築的情況。單機或多機作業均可，一般沿線路分段進行，每段距離以20～40m為宜，多在地勢平坦，或兩側有可利用的山地土場的場合採用

（2）機械填築路堤作業方式

①推土機填築路堤作業方式

推土機作業方式通常是由切土、推土、堆卸、空返等四個環節組成。而影響作業效率的主要是切土和推土兩個環節。推土機作業效率取決於切滿土的速度、距離，以及推土過程中切滿刀片中的土散失量和推運速度。

路塹（P3）：切土、運土、卸土、倒退、空回等過程組成一個循環，影響效率的是切土和運土兩個環節。

3).土質路堤壓實施工技術要點

（2）碾壓一段終了時，宜採取縱向退行方式繼續第二遍碾壓，不宜採用掉頭方式。故壓路機始終要以縱向進退方式進行壓實作業。（直）

（3）在整個全寬的填土上壓實，宜縱向分行進行，直線段由兩邊向中間，曲線段宜由曲線的內側向外側。（低到高）

4).土質路堤施工規定

（1）性質不同的填料，應水平分層、分段填築、分層壓實。同一水平層路基的全寬應採用同一種填料，不得混合填築。每種填料的填築層壓實後的連續厚度不宜小於500mm。路基上部宜採用水穩性好或凍脹敏感性小的填料。有地下水的路段或浸水路堤，應填築水穩性好的填料

（2）在透水性差的壓實層上填築透水性較好的填料前，應在其表面設2%～4%的雙向橫坡，並採取相應的防水措施。不得在透水性好的填料所填築的路堤邊坡上覆蓋透水性差的填料。

（3）每種填料的鬆鋪厚度應通過試驗確定。

（4）每一填築層壓實後的寬度不得小於設計寬度。

（5）路堤填築時，應從最低處起分層填築，逐層壓實

6）填方分幾個作業段施工時，接頭部位如不能交替填築，先填路段應按1:1~ 1:2坡度分層留臺階；如能交替填築，則應分層相互交替搭接，搭接長度不小於2m。

( 7 ) 填土路堤施工過程質量控制：施工過程中，每一壓實層均應進行壓實度檢測，檢測頻率為每1000m2 不少於2點 。壓實度檢測可採用灌砂法、環刀法等方法，檢測應符合現行《公路路基路面現場測試規程》 JTG E60-2008的有關規定。施工過程中，每填築2m宜檢測路線中線和寬度。

試驗長度不小於200m

2、應進行試驗段的路基：特填高新、路基和路堤

口訣：速度送水喝個遍

2.填石路堤施工技術

1)填石路堤施工工藝流程

四級施工臺階：在路基面以下0.5m為第一級臺階，0.5～1.5m為第二級臺階，1.5～3.0m為第三級臺階，3.0m以下為第四級臺階

四個作業區段：填石區段、平整區段、碾壓區段、檢測區段

2.填石路堤施工技術

2）填築方法

（1）豎向填築法（傾填法）

（2）分層壓實法（碾壓法）

（3）衝擊壓實法

（4）強力夯實法

2）填築方法

（1）豎向填築法（傾填法）：主要用於二級及二級以下，且鋪設低級路面的公路，也可用在陡峻山坡施工特別困難或大量以爆破方式挖開填築的路段；以及無法自下而上分層填築的陡坡、斷巖、泥沼地區和水中作業的填石路堤。該方法施工路基壓實、穩定問題較多。

（2）分層壓實法（碾壓法）：是普遍採用並能保證填石路堤質量的方法。高速公路、一級公路和鋪設高級路面的其他等級公路的填石路堤採用此方法。

（3）衝擊壓實法：利用衝擊壓實機壓密填方，它具有分層法連續性的優點，又具有強力夯實法壓實厚度深的優點。缺點是在周圍有建築物時，使用受到限制

（4）強力夯實法：強力夯實法用起重機吊起夯錘從高處自由落下，利用強大的動力衝擊，迫使岩土顆粒位移，提高填築層的

密實度和地基強度。

①強力夯實法簡要施工程序：每一分層連續擠密式夯擊，夯後形成夯坑，夯坑以同類型石質填料填補

3 ) 填石路堤施工要求

( 1 ) 填石路堤應分層填築壓實。在陡峻山坡地段施工特別困難時，三級及三級以下砂石路面公路的下路堤可採用傾填的方式填築。

( 2 ) 巖性相差較大的填料應分層或分段填築，軟質石料與硬質石料不得混合使用。

( 3 ) 填石路堤頂面與細粒土填土層之間應填築過渡層或鋪設無紡土工布隔離層。

( 4 ) 壓實機械宜選用自重不小於18t的振動壓路機。

（5）填料由土石混合材料變化為其他填料時，土石混合材料最後一層的壓實厚度應小於300mm，該層填料最大粒徑宜小於150mm，壓實後，該層表面應無孔洞。

（6）中硬、硬質石料填築路堤時，應進行邊坡碼砌，碼砌邊坡的石料強度、尺寸及碼砌厚度應符合設計要求。邊坡碼砌與路基填築宜基本同步進行。

（7） 採用易風化岩石或軟質岩石石料填築時，應按設計要求採取邊坡封閉和底部設置排水墊層、頂部設置防滲層等措施。

（8 ) 填石路堤施工過程質量控制：施工過程中每一壓實層，應採用試驗路段確定的工藝流程、工藝參數控制，壓實質量可採用沉降差指標進行檢測。施工過程中，每填高3m宜檢測路基中線和寬度。

（9）不同強度的石料，應分別採用不同的填築層厚和壓實控制標準。填石路堤的壓實質量標準宜採用孔隙率作為控制指標。孔隙率的檢測應採用水袋法進行。

（5）填料由土石混合材料變化為其他填料時，土石混合材料最後一層的壓實厚度應小於300mm，該層填料最大粒徑宜小於150mm，壓實後，該層表面應無孔洞。

4.高路堤施工技術

路基填土邊坡高度大於20m的路堤稱為高路堤。高路堤填料宜優先採用強度高、水穩性好的材料，或採用輕質材料。受水淹、浸的部分，應採用水穩性和透水性均好的材料。

高路堤施工要求：

( 1 ) 高路堤段應優先安排施工，宜預留1個雨季或6個月以上的沉降期。

( 2 ) 高路堤施工中應按設計要求預留高度與寬度，並進行動態監控。

( 3 ) 高路堤宜每填築2m衝擊補壓一次，或每填築4 ~6m強夯補壓一次。

( 4 ) 高路堤填築過程中應進行沉降和穩定性觀測。

( 5 ) 在不良地質路段的高路堤填築，應控制填築速率，並進行地表水平位移監測，必要時應進行地下土體分層水平位移監測。

5.粉煤灰路堤施工技術

粉煤灰路堤可用各級公路路堤填築，不得用於高速公路、一級公路的路床和二級公路的上路床。粉煤灰的使用可減輕土體結構自重，減少軟土路堤沉降，提高土體抗剪強度。

粉煤灰路堤一般由路堤主體部分、護坡和封頂層以及隔離層、排水系統(17多)等組成，其施工步驟與土質路堤施工方法相類似，僅增加了包邊土和設置邊坡盲溝等工序

粉煤灰施工要求：

( 1 ) 大風或氣溫低於0℃ 時不宜施工。

( 2 ) 有顯著差別的灰源應分別堆放，分段填築。

( 3 ) 路堤髙度超過4m時 ，可在路堤中部設置土質夾層。

( 4 ) 粉煤灰路堤應進行包邊防護，包邊土應與粉煤灰同步施工，寬度宜不小於2m。

( 5 ) 施工過程中，作業面應及時灑水潤溼，並應合理設置行車便道。

( 6 ) 施工間歇期，作業面應灑水潤溼，並應封閉交通；間隙期長時，應在粉煤灰壓實層頂面覆蓋封閉土層。

( 7 ) 粉煤灰路堤壓實度標準應通過試驗路段確定

6. 臺背與牆背填築施工技術

1 ) 臺背與牆背填築施工要求

（1）二級及二級以上公路應按設計做好過渡段，過渡段路堤壓實度應不小於96% ；二級以下公路的路堤與回填的聯結部，應預留臺階。

（2）臺背和錐坡的回填宜同步進行

( 3 ) 臺背與牆背1.0m範圍內回填宜採用小型夯實機具壓實。

( 4 ) 分層壓實厚度宜不大於150mm，填料粒徑宜小於100mm, 涵洞兩側回填填料粒徑宜小於50mm，壓實度應不小於96% 。

( 6 ) 涵洞兩側應對稱分層回填壓實。

( 7 ) 回填部分的路床宜與路堤路床同步填築。

( 8 ) 臺背與牆揹回填，應在結構物強度達到設計強度的75%以上時進行。

2 ) 臺背與牆背填築填料要求

填料宜採用透水性材料、輕質材料、無機結合料穩定材料等，崩解性岩石、膨脹土不得用於臺背與牆背填築。
