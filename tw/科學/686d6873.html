C語言編寫程序輸出10個整數中最小值或最大值

這是一個比較基礎，同時也包含C語言中諸多特點的程序，仔細體會有很多的格式規則和技巧。其中包括：輸入語句，輸出語句，數組，分支結構，循環結構。希望廣大C語言學習者能夠從這個程序中感受到C程序的魅力。

10個數中最小值程序

#include <stdio.h>//加載標準輸入輸出頭文件。

int main()//定義整型主函數。

{

int a,b,x[10],min;//定義整型變量a,b,min和一維數組x[]。

for(a=1;a<=10;a++)//外層循環，用來控制從鍵盤輸入整數的個數。

{

printf("請輸入第%d個整數:",a); //屏幕輸出的提示信息。

scanf("%d",&x[a]);//鍵盤輸入整型數據到數組中。

printf("\n");//換行

}

min=x[0];//將第一個數組值複製給變量min.

for(b=1;b<=10;b++)//內層循環，控制數組元素進行比較。

{

if (min>x[b])//最小值變量min與數組元素進行比較。

min=x[b];//min變量始終為最小值。

}

printf("10個數中最小值為：%d",min);//輸出最小值。

return 0;

}
