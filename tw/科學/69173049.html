語用基礎5練＋成語＋小說閱讀

一、語言文字運用

[2018·全國卷Ⅰ]閱讀下面的文字，完成1～3題。

“大洋一號”是中國第一艘現代化的綜合性遠洋科學考察船。自1995年以來，這艘船經歷了大洋礦產資源研究開發專項的多個遠洋調查航次和大陸架勘查多個航次的任務。今年，它又完成了歷時45天、航程6208海里的綜合海試任務。對不熟悉的人而言，(　　)。在這裡，重力和ADCP實驗室、磁力實驗室、地震實驗室、綜合電子實驗室、地質實驗室、生物基因實驗室、深拖和超短基線實驗室等各種實驗室　　　　，分佈在第三、四層船艙。由於船上配備了很多先進設備，人不用下水就能進行海底勘探。比如，深海可視採樣系統可以將海底微地形地貌圖像傳到科學考察船上，猶如有了千里眼，海底世界可以　　　　，並可根據需要　　　　地抓取礦物樣品和採集海底水樣；深海淺層巖芯取樣鑽機可以在深海底比較堅硬的岩石上鑽取巖芯。

“大洋一號”的遠航活動，與鄭和下西洋相呼應。600年前，偉大的航海家鄭和七下西洋，在世界航海史上留下了光輝的一頁。600年後，“大洋一號”不斷進步，　　　　，在《聯合國海洋法公約》的法律框架下，探索海洋奧祕，開發海洋資源，以實際行動為人類和平利用海洋作出了中國人民的貢獻。

1．文中畫橫線的句子有語病，下列修改最恰當的一項是(　　)

A．這艘船經歷了大洋礦產資源研究開發專項的多個遠洋調查航次和大陸架勘查多個航次的調查。

B．這艘船執行了大洋礦產資源研究開發專項的多個遠洋調查航次和多個大陸架勘查航次的任務。

C．這艘船經歷了大洋礦產資源研究開發專項的多個遠洋調查航次，完成了多個航次大陸架勘查任務。

D．這艘船執行了大洋礦產資源研究開發專項的多個遠洋調查航次，完成了多個大陸架勘查航次的任務。

答案：B

解析：本題從搭配不當、語序不當的角度設誤。此句的主幹是“這艘船經歷任務”，顯然“經歷”和“任務”搭配不當，應該是“執行……任務”，而“多個遠洋調查航次和大陸架勘查多個航次”是並列短語，作“任務”的定語，“多個遠洋調查”對應的是“多個大陸架勘查”，“多個”應該放在“大陸架”前修飾“大陸架”，所以B項是完全正確的。

2．下列在文中括號內補寫的語句，最恰當的一項是(　　)

A．“大洋一號”的實驗室很多，就像迷宮一樣

B．“大洋一號”有十幾個像迷宮一樣的實驗室

C．走進“大洋一號”，猶如進入了一座迷宮

D．進入迷宮一樣的“大洋一號”，會分辨不出方向

答案：C

解析：本題答題時要注意前後文的照應，以及語意的提示。上文“對不熟悉的人而言”，陳述的對象是“不熟悉的人”，A項和B項的主語是“大洋一號”，和上文的陳述對象不一致，排除A項和B項。C項，“迷宮”和下文的“在這裡”相呼應，排除D項，選C項。

3．依次填入文中橫線上的成語，全都恰當的一項是(　　)

A．一應俱全　一覽無餘　易如反掌　東山再起

B．應有盡有　一覽無餘　輕而易舉　再接再厲

C．一應俱全　一目瞭然　輕而易舉　東山再起

D．應有盡有　一目瞭然　易如反掌　再接再厲

答案：B

解析：本題考查根據具體語境辨析近義成語。一應俱全：一切齊全。應有盡有：應該有的全都有了，表示一切齊備。橫線前有“各種實驗室”，其中“各種”和“一應”語義重複。在這裡應選用“應有盡有”。一覽無餘：一眼看去，所有的景物全看見了，形容視野廣闊，沒有阻礙，把事物或景象都看在眼裡，沒有遺漏；也指事物簡單，看得十分清楚；或建築物的結構沒有曲折變化；或詩文內容平淡，沒有回味，不用說就可以明白；或形容道理很明顯。一目瞭然：一眼就能看清楚。語境強調對海底世界的觀察沒有遺漏，應選用“一覽無餘”。易如反掌：像翻一下手掌那樣容易，形容事情極容易辦。輕而易舉：形容事情很容易做。語境是說礦物樣品和海底水樣很容易能得到，用“輕而易舉”更恰當。東山再起：指失勢之後重新恢復地位。再接再厲：一次又一次地繼續努力。這裡沒有“失勢”的意思，應選用“再接再厲”。

4．[2018·全國卷Ⅰ]下面是某校一則啟事初稿的片段，其中有五處不合書面語體的要求，請找出並作修改。

我校學生宿舍下水道時常堵住。後勤處認真調查了原因，發現管子陳舊，需要換掉。學校打算7月15日開始施工。施工期間正遇上暑假，為安全起見，請全體學生暑假期間不要在校住宿。望大家配合。

答：　

答案：(示例)①“堵住”改為“堵塞”；②“管子”改為“管道”；③“換掉”改為“更換”；④“打算”改為“計劃”；⑤“正遇上”改為“正值”。

解析：這是一則啟事，應該採用書面語，題幹中也有“不合書面語體的要求”的提示。“堵住”“管子”“換掉”“打算”“正遇上”這些都是口語，為了和啟事的書面語風格一致，應將它們改為相應的書面語。

5．[2018·全國卷Ⅰ]下面是某校為教師編寫個人專業發展規劃而提供的流程圖，請把這個圖轉寫成一段文字介紹，要求內容完整，表述準確，語言連貫，不超過90個字。

答：　

　

答案：(示例)編寫教師個人專業發展規劃首先要進行環境分析和自我分析，在此基礎上進行個人定位並設置發展目標，然後制訂達成目標的操作策略，最後展開評估與信息反饋，再據此進一步修訂。

解析：首先，明確所給題乾的中心詞：編寫教師個人專業發展規劃。然後，仔細分析流程圖，分出寫作層次。第一層是環境分析和自我分析，第二層是個人定位和發展目標，第三層是操作策略，第四層是評估及反饋。作答的時候要注意各層次的邏輯順序，然後使用一些關聯詞串聯句子，使其連貫、通順。

二、成語強化練

6．下列各選項中，加點的成語使用恰當的一項是(　　)

A．在這個項目中，他一絲不苟，管窺蠡測，為下一步的研究作了充分的數據準備。

B．如何使運動員在賽場上創造佳績，許多有識之士提出了美芹之獻，使運動員能夠輕裝上陣。

C．國際形勢日趨複雜，美日兩國正不斷擴大其軍事、政治影響，對此，俄羅斯和中國有必要一個鼻孔出氣，結成牢固的戰略聯盟。

D．隨著人類基因組計劃的進展，生物芯片技術應運而生，並以完整的技術身份促進了基因組學的發展，帶動了生物芯片技術的產業化。

答案：D

解析：D項，應運而生：隨著某種形勢而產生。A項，管窺蠡測：從竹管裡看天，用瓢測量海水。比喻對事物的觀察和了解很狹窄，很片面。不合語境。B項，“美芹之獻”指地位低微的人提出的好意見，用以自謙所獻菲薄。這裡使用對象錯誤。C項，“一個鼻孔出氣”指觀點態度完全一致，多用貶義，感情色彩不合適。

7．下列各句中，加點的成語使用恰當的一項是(　　)

A．學校準備組織教職工去西藏旅遊，大家興致勃勃，小張更是推波助瀾，積極鼓動年輕人搞生態自助遊。

B．辛棄疾的《永遇樂·京口北固亭懷古》信手拈來古人古事入詞，達到了天衣無縫的境地，可謂化典入詞的範例。

C．你剛大學畢業，進公司才三個月就被提升為部門主管。雖然機會難得，但你最好還是敬謝不敏，先在基層鍛鍊幾年再說。

D．論文抄襲的事件在學術圈外的領域也頻頻發生，這就不能不發人深思：不管哪行哪業，動輒將論文作為職稱晉級、業績考核的敲門磚，是否完全合適？

答案：D

解析：D項，比喻藉以謀取名利的工具或手段。A項，推波助瀾：比喻促使或助長事物(多指壞的事物)的發展，使擴大影響。用在此處感情色彩不當。B項，信手拈來：隨手拿來。多形容寫文章時詞彙或材料豐富，不費思索，就能寫出來。其後不能跟賓語。C項，敬謝不敏：表示推辭做某件事的客氣話。用於此處謙敬錯位。

8．下列各句加點詞語使用不恰當的一句是(　　)

A．隨著課改的逐步深入，面對教材的多樣化，有的專家提出了變“教課本”為“用課本教”的理念，可謂切中肯綮。

B．丁俊暉本賽季狀態不錯，拿下了溫布利大師賽的冠軍，三大賽事中僅僅剩下世錦賽沒能染指。

C．過於膨脹的心態並不符合國家利益，國家利益來自於冷靜的理性，而不是膨脹的情緒，中國在當今國際社會仍然需要韜光養晦，而不是顯山露水。

D．通信產業利潤豐厚，吸引了很多廠商進入，但是林子大了，什麼鳥都有，因此消費者可得擦亮眼睛。

答案：B

解析：B項，“染指”比喻分取非分的利益，為貶義詞，不合語境。

三、文學類文本閱讀

[2018·全國卷Ⅰ]閱讀下面的文字，完成9～11題。

趙一曼女士

阿　成

偽滿時期的哈爾濱市立醫院，如今仍是醫院。後來得知趙一曼女士曾在這裡住過院，我便翻閱了她的一些資料。

趙一曼女士，是一個略顯瘦秀且成熟的女性。在她身上瀰漫著拔俗的文人氣質和職業軍人的冷峻。在任何地方，你都能看出她有別於他人的風度。

趙一曼女士率領的抗聯活動在小興安嶺的崇山峻嶺中，那兒能夠聽到來自坡鎮的鐘聲。冬夜裡，鐘聲會傳得很遠很遠。鐘聲裡，抗聯的兵士在森林裡烤火，烤野味兒，或者唱著“火烤胸前暖，風吹背後寒……戰士們喲”……這些都給躺在病床上的趙一曼女士留下清晰回憶。

趙一曼女士單獨一間病房，由警察晝夜看守。

白色的小櫃上有一個玻璃花瓶，裡面插著丁香花。趙一曼女士喜歡丁香花。這束丁香花，是女護士韓勇義折來擺放在那裡的。聽說，丁香花現在已經成為這座城市的“市花”了。

她是在山區中了日軍的子彈後被捕的。濱江省警務廳的大野泰治對趙一曼女士進行了嚴刑拷問，始終沒有得到有價值的回答，他覺得很沒面子。

大野泰治在向上司呈送的審訊報告上寫道：

趙一曼是中國共產黨珠河縣委委員，在該黨工作上有與趙尚志同等的權力。她是北滿共產黨的重要幹部，通過對此人的嚴厲審訊，有可能澄清中共與蘇聯的關係。

1936年初，趙一曼女士以假名“王氏”被送到醫院監禁治療。

《濱江省警務廳關於趙一曼的情況》扼要地介紹了趙一曼女士從市立醫院逃走和被害的情況。

趙一曼女士是在6月28日逃走的。夜裡，看守董憲勳在他叔叔的協助下，將趙一曼抬出醫院的後門。一輛僱好的出租車已等在那裡。幾個人上了車，車立刻就開走了。出租車開到文廟屠宰場的後面，韓勇義早就等候在那裡，扶著趙一曼女士上了僱好的轎子，大家立刻向賓縣方向逃去。

趙一曼女士住院期間，發現警士董憲勳似乎可以爭取。經過一段時間的觀察、分析，她覺得有把握去試一試。

她躺在病床上，和藹地問董警士：“董先生，您一個月的薪俸是多少？”

董警士顯得有些忸怩，“十多塊錢吧……”

趙一曼女士遺憾地笑了，說：“真沒有想到，薪俸會這樣少。”

董警士更加忸怩了。

趙一曼女士神情端莊地說：“七尺男兒，為著區區十幾塊錢，甘為日本人役使，不是太愚蠢了嗎？”

董警士無法再正視這位成熟女性的眼睛了，只是哆哆嗦嗦給自己點了一顆煙。

此後，趙一曼女士經常與董警士聊抗聯的戰鬥和生活，聊小興安嶺的風光，飛鳥走獸。她用通俗的、有吸引力的小說體記述日軍侵略東北的罪行，寫在包藥的紙上。董警士對這些紙片很有興趣，以為這是趙一曼女士記述的一些資料，並不知道是專門寫給他看的。看了這些記述，董警士非常向往“山區生活”，願意救趙一曼女士出去，和她一道上山。

趙一曼女士對董警士的爭取，共用了20天時間。

對女護士韓勇義，趙一曼女士採取的則是“女人對女人”的攻心術。

半年多的相處，使韓勇義對趙一曼女士十分信賴。她講述了自己幼年喪母、戀愛不幸、工作受欺負，等等。趙一曼女士向她講述自己和其他女戰士在抗日隊伍中的生活，有趣的、歡樂的生活。語調是深情的、甜蜜的。

韓護士真誠地問：“如果中國實現了共產主義，我應當是什麼樣的地位呢？”

趙一曼女士說：“你到了山區，一切都能明白了。”

南崗警察署在趙一曼女士逃走後，馬上開車去追。

追到阿什河以東20多公里的地方，發現了趙一曼、韓勇義、董憲勳及他的叔父，將他們逮捕。

趙一曼女士淡淡地笑了。

趙一曼女士是在珠河縣被日本憲兵槍殺的。

那個地方我去過，有一座紀念碑。環境十分幽靜，周圍種植著一些松樹。

我去的時候，在那裡遇到一位年邁的老人。他指著石碑說，趙一曼？我說，對，趙一曼。

趙一曼被槍殺前，寫了一份遺書：

寧兒：

母親對於你沒有能盡到教育的責任，實在是遺憾的事情。

母親因為堅決地做了反滿抗日的鬥爭，今天已經到了犧牲的前夕了。

母親和你在生前是永久沒有再見的機會了。希望你，寧兒啊！趕快成人，來安慰你地下的母親！我最親愛的孩子啊！母親不用千言萬語來教育你，就用實行來教育你。

在你長大成人之後，希望不要忘記你的母親是為國而犧牲的！

一九三六年八月二日

(有刪改)

9．下列對小說相關內容和藝術特色的分析鑑賞，不正確的一項是(　　)

A．小說以“趙一曼女士”為題，不同於以往烈士、同志、英雄等慣常用法，稱謂的陌生化既表達了對主人公的尊敬之意，又引起了讀者的注意。

B．“通過對此人的嚴厲審訊，有可能澄清中共與蘇聯的關係”，這既是大野泰治向上級提出的建議，也暗示他已從趙一曼那裡得到有價值的回答。

C．“他指著石碑說，趙一曼？我說，對，趙一曼。”兩個陌生人之間有意無意的搭訕，看似閒筆，實則很有用心，說明趙一曼仍活在人們的記憶裡。

D．醫院是“我”與趙一曼的連接點，小說由此切入主人公監禁期間鮮為人知的特殊生活經歷，在跨越時空的精神對話中再現了趙一曼的英雄本色。

答案：B

解析：B.由“偽濱江省警務廳的大野泰治對趙一曼女士進行了嚴刑拷問，始終沒有得到有價值的回答”可知，“也暗示他已從趙一曼那裡得到有價值的回答”錯誤。

10．小說中說趙一曼“身上瀰漫著拔俗的文人氣質和職業軍人的冷峻”，請結合作品簡要分析。

答：　

　

答案：①文人的氣質：喜歡丁香花，情趣不俗；時常深情、甜蜜地回憶戰鬥生活，文雅浪漫；用大義與真情感化青年，智慧過人；②軍人的冷峻：遭嚴刑拷打而不屈服，意志堅定；笑對即將到來的死亡，從容淡定；充滿母愛又不忘大義，理智沉穩。

解析：本題從文人的氣質、軍人的冷峻兩方面作答。趙一曼女士文人的氣質主要體現在：喜歡丁香花；用通俗的、有吸引力的小說體記述日軍侵略東北的罪行；用深情、甜蜜的語調講述在抗日隊伍中的生活。軍人的冷峻主要體現在：不屈服於嚴刑拷問；用軍人敏銳的眼光觀察、分析可爭取的董憲勳和韓勇義；笑對死亡，用飽含深情的語言寫下遺書，卻不忘大義。由這些分析，總結答案即可。

11．小說中歷史與現實交織穿插，這種敘述方式有哪些好處？請結合作品簡要分析。
