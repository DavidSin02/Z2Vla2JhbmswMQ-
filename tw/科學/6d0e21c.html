連載 | 02 Daniel Quillen 在代數 K-理論領域的貢獻

2020-10-29
| 科學

Eric Friedlander，美國南加州大學數學教授。

Daniel Grayson，美國伊利諾伊大學香檳分校數學教授。

編者：Eric Friedlander，Daniel Grayson

譯者：王勃

Daniel Quillen (1940 —2011) 是菲爾茲獎獲得者，在代數、幾何及拓撲方面都頗有建樹。特別是在 1967 至 1977 這十年間，他完成了一系列卓越的論文，從而開創了全新的數學領域。Quillen 的思想能為數學家提供靈感與幫助，他的數學論著使得這些領域變得更加清晰和明確。儘管 Quillen 天賦異稟，但是熟悉他的人都對其慷慨和謙遜印象深刻。能夠得到他的指導， 並且有機會去學習其卓越的學術成果，是我們的榮幸。對於 Quillen 的逝去，我們悲痛萬分。

這篇紀念文章共由 12 名作者合作完成，其中包括 Quillen 的同事、合作者、學生和家人。Graeme Segal 對 Quillen 數學上的貢獻做了全面而具有深度的總結；Hyman Bass 則重點論述了 Quillen 在代數 K-理論領域的貢獻；Quillen 的合作者 Joachim Cuntz 和 Jean-Louis Loday 介紹了他們與 Quillen 合作完成的關於循環同調的工作；Quillen 在牛津的同事 Michael Atiyah 和 Ulrike Tillmann，以及 Quillen 在哈佛的同事 Barry Mazur，講述了他們與 Quillen 共事時的往事；Dennis Sullivan 和 Andrew Ranicki 回憶了他們以前與 Quillen 在數學上的交流；Quillen 的學生 Ken Brown 和 Jeanne Duflot 則展現了學生視角下的 Quillen；本文的最後一部分來自於 Quillen 的妻子 Jean Quillen，她和 Quillen 共同育有六個孩子。Jean 描繪出了數學奇才 Quillen 在家庭中作為一個居家男人的形象。

02

Hyman Bass¶

¶Hyman Bass 是密歇根大學數學教授。

Hyman Bass 重點論述了 Quillen 在代數 K-理論領域的貢獻

我和 Quillen 共有兩次機會一起工作，兩次都是如花園一樣的工作環境， 卻都令我們做出了非凡的成果。一次是在 1968 —1969 年間，我們都參加了 IHES 的 Grothendieck 研討會以及法蘭西學院的 Serre 講座。另一次機會則是在 1972 年夏季，在西雅圖 Battelle 紀念研究所舉辦的關於代數 K-理論的為期兩週的會議。

在巴黎的日子裡，Quillen 顯示出他的人格魅力：紳士、謙遜、仔細，一 頭銀髮所無法改變的孩童般的外表，以及充實的家庭生活。在那個絢麗而輝煌的數學時代，Quillen 似乎傾聽得更多，他只在有非常重要的內容時才會講兩句。Quillen 隨後的工作展示出他是一位耐心的傾聽者。

然而在 Battelle 會議上，Quillen 一反常態地成為了會議的中心。此次會議由於 Quillen 的工作，可以看作是代數 K-理論的分水嶺。在下文中，我會回憶當時的場景和氣氛，以及 Quillen 所帶來的思維方法和結果的根本性改變，這也為 Quillen 帶來了 1978 年的菲爾茲獎。

K-理論的靈感來源於 Grothendieck 關於K(X) 群生成代數簇X上廣義 Riemann-Roch 定理的介紹 [4]。這激發 Atiyah 和 Hirzebruch 創立拓撲K-理 論，X是拓撲空間，K(X) 成為一個Kn(X)(n≥ 0) 群的廣義同倫理論的零 次項K0(X) [1]。

設 X是一個仿射概形，X= Spec(A)，Grothendieck 構造K(X) 所用到 的代數向量叢會與有限生成投射A-模對應起來 [11]。如果X是緊 Hausdoff 空間並且A=C(X) 是環上的連續函數，則對拓撲K(X) 也成立 [12]。這意 味著定義了有限生成投射A-模的 Grothendieck 群4)K0(A)，且該定義對任何 環A均成立（可以不是交換環）。這種超越代數的（也是非幾何的）一般性 並不是無用的，因為拓撲學家在一些基本群π的整群環

π的K0群的同倫 理論中已經遇到了阻礙 [13]。

4)這裡變成下標是因為X與A之間的逆變。

很自然地就會聯想到，拓撲 K-理論所對應的由群Kn(A)(n> 0) 組成 的類似的代數理論。結論並不是顯而易見的，但一些特別的方法可以成功處理前兩步。首先是K1(A) =GL(A)/E(A) 的定義 [3]，這裡GL(A) 是 無限一般線性群，E(A) 是其換位子群，由GL(A) 中的初等矩陣構成 [14]。K1的定義最好包含與K0自然函子的關係以及與拓撲的聯繫，這裡簡單 同倫論中的 Whitehead 撓率不變量位於形如W h(π) =K1(Zπ)/(±π) 的群 中 [14]。

當 A是數域的整數環時，K0就和理想類群相關，K1則和單位組成的群 相關，並且相對K1群蘊含了SLn(A)(n≥ 3) 的同餘子群這一古典問題的答 案 [2]。

第二步則是 K2(A) =H2(E(A),) 的定義 [6]，它是泛中心擴張St(A)

E(A) 的核，這裡“Steiberg 群”St(A) 由基本生成元及其之間 的關係表示。這個定義依然具有很好的函子性質，數域中的計算顯示出其與 顯式互反律的深刻聯繫。另外，Hatcher [5] 發現K2與假同位素問題的拓撲上的聯繫。

儘管這些代數上的概念 K0,K1和K2只是通往未知一般性理論的第一 步，它們已經展示出了與代數拓撲、代數幾何、數論以及我們上面沒有提到 的算子代數之間足夠有趣的聯繫，因此完全得出代數K-理論似乎是水到渠成 的。事實上，一些人 (Gersten, Karoubi-Villamayor, Swan, Volodin) 已經嘗試創造了高等代數K-函子，但是他們並不能完全搞清楚函子的特性以及之間的 關係，他們也沒有計算一般環A上的函子。

這就是 1970 年左右代數 K-理論的發展情況：一個依然在孕育之中的理論，但是已經有了一些令人感興趣的應用，吸引了不同領域數學家的注意。我將對這些理論的發展者和實踐者做一個總結。西雅圖的 Battelle 紀念研究所召開了為期兩週的會議，包括 Spencer Bloch, Armand Borel, Steve Gersten, Alex Heller, Max Karoubi, Steve Lichtenbaum, Jean-Louis Loday, Pavaman Murthy, Dan Quillen, Andrew Ranicki, Graeme Segal, Jim Stashe, Dick Swan, John Tate, Friedhelm Waldhausen 以及 Terry Wall 在內的 70 位數學家參加了會議，正如我在會議文集的序言中所提到的 [BC]：

“······ 很多數學家儘管有著不同的研究背景和研究興趣，但是都對代數 K-理論很感興趣。也許這些數學家的聚會比新元素的發現更重要。無論如何， 將這麼多數學家聚在一起是令人滿意的，特別是其中一些數學家根本沒有其他機會在一起進行學術交流。數學家們在舒適而放鬆的環境中彼此交流，推動了數學和人類的發展。”

我認為，這次會議超越了預期，取得巨大的成功。絕對可以把這次會議稱作代數 K-理論的盛典，而 Quillen 則是當之無愧的巨星。Quillen 給出了兩個高等代數K-理論的成功構造，一個是 Battelle 會議前提出的“+-構造”， 另一個則是在會議現場提出的“Q-構造”。

“+-構造”定義了

(A) = πn(BGL(A)+)n≥ 1,

這裡 BGL(A)+表示分類空間BGL(A) 的一個修正，擁有相同的倫型，基本群為

K1(A) =GL(A)/E(A).

Quillen 進一步又驗證了

π2(BGL(A)+) =K2(A),

從而進一步說明了定義的合理性。Quillen 在 Adams 猜想的證明過程中指出 [7]，在有限域中，BGL(Fq)+的倫型與

Φq− Id : BU → BU

的纖維倫型相同，這更加驗證了定義的正確性，同時也完整地給出了有限域 K-理論的計算結果。

“+-構造”是一個質的飛躍，但仍然存在兩個不足。首先，它並沒有直接解釋 K0(A)。其次，也是更重要的，它並沒有帶來在低維 Kn(n= 0, 1, 2) 中 有效的簡單計算工具。為了克服這一點，根據 Grothendieck 原始定義的思想， 必須對有正合序列的加性範疇 C定義

(C) (n≥ 0)。（對於環A，為了得 到 (A)，必須把C弄成有限生成投射A-模的範疇。）而“Q-構造”完成 了這一點，它定義了 (C) =πn+1(BQC)，這裡BQC是 Quillen 發明的新範疇QC（Q-構造）的分類空間（每個範疇都有定義），一系列精妙的定理 和基礎的方法使得該定義更加明確：

  • 一致性：(A) = Kn(A), n =0, 1, 2;(A) = (A), n≥ 1。從而 定義 Kn(A) 等於 (A) 對所有的n≥ 0 成立。


  • 可解性：如果 C′ ⊆C，並且每個範疇都有一個有限C′ -分解，則Kn(C′ ) →Kn(C) 是同構。


  • 可分性：如果 C′ ⊆C，並且每個範疇有一個有限C′ 子商纖維，則Kn(C′ ) →Kn(C) 是同構。


  • 局部化：如果 C是一個 Abel 範疇，C′ 是一個 Serre 子範疇，則存在 一個與Kn(C), Kn(C′ ), Kn(C/C′ ) 相關的局部化正合序列。


  • 倫型不變：如果 A是諾特的，則 (A[t]) = (A)，這裡 (A) 是 有限生成A-模的範疇的K-理論（當A正則時，由可解性，與Kn(A) 相同）。


  • 基本定理：如下是一個自然正合序列，0 → Kn(A) →Kn(A[t]) ⊕Kn(A[t−1]) →Kn(A[t;t−1]) → Kn−1(A) → 0。


  • 代數幾何：許多定理可以應用到概形的 K-理論，包括計算以及同周氏環的關係。

這些都詳細地收錄於 Quillen 63 頁的雙欄論文中 [8]。這是一篇令人震驚的論文，裡面有概念、技巧以及應用。通常許多數學家十餘年才能完成的成果，Quillen 在一篇文章中就全部體現。Quillen 不但促成了代數 K-理論從孕 育到成熟這一質的飛躍，而且還做了更多。在一次講座中，Quillen 用巧妙的新方法給出了一個關於代數整數環上K-群的有限生成性的完整證明 [9]。後 來，Quillen 又一次用精妙的技巧證明了所謂的 Serre 猜想，即多項式代數上 的投射模是自由的 [10]。

數學家總是被刻畫為理論創新者以及問題解決者，Quillen 則是兼有兩者。正如 Grothendieck 一樣，Quillen 並不是直接地面對難題迎難而上，而是對一般性的概念做出分析和改進，使得論證具有數學的必然性。然而如果說 Grothendieck 是瓦格納的風格，則 Quillen 更接近於莫扎特。Quillen 謙遜而和藹，是一位偉大的創作家，陶冶著他的聽眾。Quillen 從不留下多餘而浮華的內容，以致人們並不想去改變而只是想繼續深入學習 Quillen 的理論。能夠與 Quillen 交往並見證他的這些成果是我個人的榮幸與樂趣。
