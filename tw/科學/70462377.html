GTA5富貴險中求：組織行動指南

2020-11-14
| 科學

除了在洛聖都走私特種貨物外，「富貴險中求」還提供了新的老大工作與老大挑戰，為老大與保鏢的事業開啟嶄新局面。雖然 CEO 盡享所有身為老大的好處，包括從下屬及特種貨物任務中獲益，保鏢及副手也可以參與老大工作及挑戰，得到豐厚薪水及額外獎勵。

老大工作

身為新崛起的 CEO 與老大，您可以指派「老大工作」給組織成員。而所謂的老大工作，就是在自由模式下進行的一系列多人合作差事。老大工作必須經由互動選單中的「保鏢事務所：老大」選項來啟動。只要老大需要，保鏢與副手都會收到遊戲內簡訊或自保鏢事務所應用程式送出的邀請，要求他們參加。老大工作提供許多機會，讓玩家狂賺 GTA 遊戲幣和大量聲望值，同時敵對組織也會趁機偷襲，藉此分一杯羹。我們將在此提供一些祕訣，讓您的組織在從事全新老大工作時可以守住每一分利益：

獵殺專員

任務目標：與您的組織攜手合作，在時限內殺掉地圖上多個戒備森嚴的定點或移動的目標。

為了讓您的隊員能像精密儀器般按計畫行事，您可以分派他們鎖定早已選定的目標，並銘記下列訣竅：

  1. 所有潛在目標及守衛的生命值與武裝皆已增加，若您徒步接近，他們可輕易將您擊倒在地。為避免定點目標發現您的蹤跡，您可採用遠處攻擊的方式，無聲無息地解決掉他們。確認目標死亡後請儘速離開現場，因為您的通緝等級將會迅速升高。

  2. 追擊位於載具內的目標時，您可以追蹤他們的移動路線，思考如何伏擊；在後頭追逐他們，可能會導致您受到大量子彈的熱情招呼。

  3. 您可以使出各個擊破的策略。由於地圖上有多個目標等您拿下，因此將所有資源投入單一目標是相當不智的舉動。針對移動目標，您可將隊伍分成兩人一組，掌控更多移動路線，並派出一名能幹的殺手拿下定點目標。

  4. 為因應各種突發狀況，請讓空中及武裝載具隨時待命，再開始這項工作。

  5. 玩家可以透過萊斯特，或使用老大 / CEO 的「賄賂當局」能力來消除通緝等級。請注意，即使「賄賂當局」能力已啟動，攻擊中的執法單位仍會讓您冠上通緝等級。

  6. 身為敵對玩家，保護目標不受攻擊組織的獵殺則可為您賺得獎勵。

在「獵殺專員」中，您可從遠處獵殺目標，讓他們無法察覺您的存在。

空運貨物

任務目標：您的組織必須取得一架運兵直升機，並利用直升機運送貨櫃到指定投放點。

指派您最棒的駕駛員來進行這項任務，讓其餘成員負責抵擋攻擊。我們同樣提供幾點建議幫助您：

  1. 在運送過程中，您的貨物隨時都可能受到毀損而失去現金價值，所以別拖延 - 直接前往投放區，儘可能賺取最大利潤。

  2. 仔細偵查戒備森嚴的地區後再搶奪貨物，除了打倒守衛外，還要炸燬備有武器的載具，讓他們無法追趕或扭轉情勢。

  3. 或者，您也可以幹掉所有守衛，並讓一位組織成員控制攻擊直升機，如此您便有額外的空中支援護送您前往投放區。

  4. 空中支援是運送過程的核心助力，因為在飛行過程中，您不只要揹負無法消除的 3 星通緝等級，敵對組織也會接獲您行動的情報，進而設法擊下您以獲取獎勵。

  5. 一旦您成功運送貨物，請立即迅速撤退，因為運兵直升機將啟動自爆程序，掩蓋的犯罪證據。

  6. 若運送貨物的運兵直升機被摧毀，不屬於任何組織或以敵對組織身分參與的玩家，即可獲得部分貨物價值做為獎勵。

在「空運貨物」裡，一旦成功運送貨物，請立即迅速撤退，因為運兵直升機將啟動自爆程序，掩蓋您的犯罪證據。

陸上貨運

任務目標：您的組織必須取得一輛大型拖車，並駕駛卡車將它送達投放點。

即使您的大卡車在高速公路上凶猛無比，能迅速衝過最靠近您的追兵，遵循下列祕訣仍是明智選擇：

  1. 前往取得拖車時，利用附裝甲的交通工具抵禦追兵無疑是個好主意。若您前進的路上缺少支援，請隨時注意拖車周圍有沒有全副武裝的卡林鐵尼高。

  2. 攻擊者的車輛可能會在您卡車的前方，拖慢您的前進速度。記得將視角稍微向前傾斜調整，才能確實掌握他們的位置，避免他們阻礙您的動線。

  3. 如果您的卡車承受過多損害，隨時都會爆炸，您得立刻制定策略，規劃路線好替換一輛卡車。這些卡車會以藍色圖示顯示在您的地圖上。

  4. 此外，若是您的卡車著火且瀕臨爆炸邊緣，記得分離拖車後將卡車駛離，避免遭炸燬的卡車殘骸擋住新卡車的去路。

  5. 您可考慮讓一名副手或保鏢駕駛一輛額外的卡車，當做突發狀況時的備用車輛。

  6. 如果敵對組織或自由模式的玩家成功運送貨櫃，他們將能分到最高獎勵的一部分。

在「陸上貨運」裡，如果您的卡車車著火，請記得將拖車與卡車分離，並將卡車駛離，方便替換一臺新的卡車。

老大挑戰

您可以在互動選單的「保鏢事務所」選項中找到「老大挑戰」；老大可以發起「老大挑戰」，讓組織進行良性內部競爭並獲取獎勵。

在「老大挑戰」中獲勝的人自然可以獲得獎勵，而參加老大工作的保鏢，每分鐘都可獲得 GTA 遊戲幣以及聲望值做為參加獎勵。

挑戰設置完成後，老大可選擇在一旁觀看保鏢的表現，或直接參與其中。

提領一空

任務目標：所有組織內玩家要在規定時間內較勁，看誰能入侵最多自動提款機。

拿了錢就跑吧！搶奪四周可用的最快速交通工具，並留心下列祕訣，增加您的籌碼：

  1. 注意背後。請與其他組織成員保持距離。在入侵提款機時，您很容易就會成為攻擊靶子，所以開始入侵工作前務必注意這一點。這樣同時也可確保無人能搶先入侵您的目標提款機。

  2. 為保持領先，留神注意其他停在自動提款機前的組織成員，干擾他們的行動，確保自己是率先前往下個入侵地點的人。

  3. 持續查看地圖，除了注意下個距離最近的自動提款機地點外，還要對附近人士全神戒備，並找出自動提款機分佈密度最高的區域。

  4. 如果您不想讓其他人在地圖上發現您的行動，您可以打電話給萊斯特，付錢讓他幫您在雷達中暫時消失，這招在您決定好下手的提款機時特別好用。

  5. 時間是整場挑戰賽的關鍵。在二星通緝等級啟動前，您有 60 秒時間進行自動提款機入侵程序，所以請務必請求最快的載具，進行高速移動。

  6. 遭到擊殺時將會損失金錢，所以請儘量避免惹禍上身。

在「打撈行動」中，您的體力與氧氣會逐漸耗盡。記得尋找其他裝有水中呼吸器的箱子，延長在水中活動的時間。

打撈行動

任務目標：所有組織內的玩家將配備水中呼吸器，比賽誰能在水中收集最多檢查點。

在水中存活可不像表面看來這麼平靜。潛水前，請閱讀下方的實用祕訣：

  1. 使用刀子阻止對手靠近檢查點，算是比較公平的方式。老大與 CEO 可在老大選單中的「管理」選項開啟「友方傷害」，讓整場比賽既刺激又有趣。

  2. 您不需要從海岸一路游到檢查點。注意地圖上的小海鯊圖示，讓前進速度增快吧。

  3. 儘管戴上水中呼吸器能讓您在水中待久一點，但您的體力及氧氣仍會逐漸耗盡。記得尋找其他裝有水中呼吸器的箱子，延長在水中活動的時間。

  4. 檢查點遍及四處，遊往較少人佔據的領域可不是什麼丟人之舉。在其他人你爭我搶的同時，您可以輕鬆無阻地收集檢查點。

  5. 若意圖潛入更深的水域收集檢查點，請記得再三確認您的氧氣與體力是否充足，才能安心收集檢查點並返回水面。別過於貪心，否則將面臨失敗收場的命運。

武器升級

激烈火拼時，裝填彈藥的時機可能會影響您的生死，畢竟沒人想在身上只剩幾發子彈時被殺個措手不及。值得慶幸的是，武裝國度已針對 8 種火力最強的武器提供全新造型的大容量彈匣。這些特殊彈匣能容納更多子彈，讓您不用在短時間內重新裝填，這項優勢能在武力配備較弱的對手面前發揮決定性作用。

關於特種貨物的祕訣和技巧

您的組織已花了幾天時間熟悉特種貨物任務，這裡有些祕訣讓您的任務過程進行得更加流暢：

  1. 透過保鏢事務所特種貨物交易網路，留心其他在您戰局內的 CEO 及副手，謹慎規劃任務時機。

  2. 確定您已僱用足夠的副手，好因應運輸的貨物量 - 運送三個箱子的風險將比運送一個高，因此需要更多人手進行保護及拾取行動。

  3. 拾取箱子時，您可直接走過去，如要節省時間及避免遭受攻擊，您也可以駕駛載具前往拾取 - 請記住，您一次只能拿一個箱子。

  4. 做好萬全準備 - 配備齊全的帕格薩斯載具，如兀鷹攻擊直升機，隨時都可能派上用場。遇到麻煩時，適時利用您的 CEO 能力避開警察，並隱藏組織及貨物的光點，以保護特種貨物的安全。

  5. 敵對玩家無法攻擊倉庫，但可伏擊運有特種貨物的交通工具。

  6. 您在倉庫儲備的貨物越多，賺得的利潤也就越可觀。一次販售大量貨物可吸引到給錢大方的有錢買家，但風險也隨之增高。
