喜報：恭喜騰訊雲 API 網關獲得國內首張可信雲認證證書

2020-11-14
| 科學

2020 年 10 月 21 日，“2020 雲原生產業大會”在北京召開。本屆大會由中國信息通信研究院主辦，中國通信標準化協會雲計算標準和開源推進委員會承辦。會上發佈了雲原生領域評估結果、雲原生應用優秀案例等一系列重磅成果。



在此次大會上，騰訊雲 Serverless 團隊的 API 網關產品獲得可信雲最高級認證證書！代碼編號：No.GW-0001。今年是信通院首次推出 API 網關評估，騰訊雲 API 網關憑藉著可靠穩定的服務和優秀的產品功能，成為最早通過首批 API 網關評估的產品。



騰訊雲 API 網關是騰訊雲在騰訊網關 Tencent Gateway、負載均衡 CLB 等產品的成功經驗上，開發推出的一種 API 託管服務，提供 API 的完整生命週期管理，包括創建、維護、發佈、運行、下線等。騰訊雲 API 網關與雲函數 SCF、微服務框架 TSF、雲服務器 CVM、容器服務 TKE 等雲產品打通，用戶可使用 API 網關封裝自身業務，將數據、業務邏輯或功能安全可靠的開放出來，用以實現自身系統集成、以及與合作伙伴的業務連接。騰訊雲 API 網關還繼承了騰訊網關的高性能、高可靠的優勢，提供強負載能力，輕鬆應對大流量。

目前，騰訊雲 API 網關作為騰訊雲 Serverless 家族的重要一員，已為數十萬名開發者，數千家企業提供接入層服務，幫助其開放業務能力，其中包括騰訊視頻、萬達、麥當勞、碧桂園、江娛互動等知名企業。未來我們將持續投入，一如既往地為廣大開發者提供優質的雲服務。
