VBA/Excel實例系列——奇偶判斷

2020-11-14
| 科學

Part 1：邏輯過程

  1. 採用一個函數，傳入需要檢查的整除
  2. 函數內部採用求餘數運算，餘數為0則為偶數，反之奇數
  3. 其中求餘VBA函數為Mod
  4. 官方幫助文件：Integer 變量存儲為 16位（2 個字節）的數值形式，其範圍為 -32,768到32,767 之間
  5. 所有超過Integer範圍的會報錯

Part 2：代碼

Function f01_奇偶判斷(num As Integer) Rem>>判斷數字的奇偶性 Rem>> Dim remainderNum remainderNum = num Mod 2 If remainderNum = 0 Then f01_奇偶判斷 = "偶數" Else f01_奇偶判斷 = "奇數" End IfEnd Function

Part 3：調用該函數

 num = 11 Status = f01_奇偶判斷(num) If Status = "偶數" Then MsgBox num & "是偶數" Else MsgBox num & "是奇數" End If
