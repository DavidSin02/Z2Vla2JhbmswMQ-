接下來介紹的是QPanda 2的算法組件之一：泡利算符類。

泡利算符類

泡利算符是一組三個2×2的么正厄米復矩陣，又稱酉矩陣。我們一般都以希臘字母σ（西格瑪）來表示，記作σxσx，σyσy，σZσZ。在QPanda中我們稱它們為X門，Y門，Z門，它們對應的矩陣形式，如下表所示。

泡利算符的運算規則如下：

1.泡利算符與自身相乘得到是單位矩陣



2.泡利算符與單位矩陣相乘，無論是左乘還是右乘，其值不變



3.順序相乘的兩個泡利算符跟未參與計算的泡利算符是 ii 倍的關係



4.逆序相乘的兩個泡利算符跟未參與計算的泡利算符是 −i−i 倍的關係


接口介紹

根據泡利算符的上述性質，我們在QPanda中實現了泡利算符類PauliOperator。我們可以很容易的構造泡利算符類，例如：

using namespace QPanda;// 構造一個空的泡利算符類PauliOperator p1;// 2倍的"泡利Z0"張乘"泡利Z1"PauliOperator p2("Z0 Z1", 2);// 2倍的"泡利Z0"張乘"泡利Z1" + 3倍的"泡利X1"張乘"泡利Y2"PauliOperator p3({{"Z0 Z1", 2},{"X1 Y2", 3}});// 構造一個單位矩陣，其係數為2，等價於PauliOperator p4("", 2);PauliOperator p4(2);

其中PauliOperator p2("Z0 Z1", 2)表示的是 2σz0⊗σz12σ0z⊗σ1z。

註解

構造泡利算符類的時候，字符串裡面包含的字符只能是空格、 XX ，YY ，ZZ 中的一個或多個，包含其它字符將會拋出異常。另外，同一個字符串裡面同一泡利算符的比特索引不能相同，例如：PauliOperator("Z0 Z0", 2)將會拋出異常。

泡利算符類之間可以做加、減、乘等操作，計算返回結果還是一個泡利算符類。

using namespace QPanda;PauliOperator a("Z0 Z1", 2);PauliOperator b("X5 Y6", 3);PauliOperator plus = a + b;PauliOperator minus = a - b;PauliOperator muliply = a * b;

泡利算符類支持打印功能，我們可以將泡利算符類打印輸出到屏幕上，方便查看其值。

using namespace QPanda;PauliOperator a("Z0 Z1", 2);std::cout << a << std::endl

我們在實際使用的時候，常常需要知道該泡利算符類操作了多少個量子比特，這時候我們通過調用泡利算符類getMaxIndex接口即可得到。如果是空的泡利算符類調用getMaxIndex接口則返回0，否則返回其最大下標索引值加1的結果。

using namespace QPanda;PauliOperator a("Z0 Z1", 2);PauliOperator b("X5 Y6", 3);// 輸出的值為2std::cout << a.getMaxIndex() << std::endl;// 輸出的值為7std::cout << b.getMaxIndex() << std::endl;

如果我們構造的的泡利算符類，其中泡利算符的下標索引不是從0開始分配的，例如PauliOperator b("X5 Y6", 3)調用getMaxIndex接口返回的使用的比特數是7，其實 只使用了2個比特。我們如何才能返回其真實用到的比特數呢。我們可以調用泡利算符類裡面remapQubitIndex接口，它的功能是對泡利算符類中的索引從0比特開始分配映射， 並返回新的泡利算符類，該接口需要傳入一個map來保存前後下標的映射關係。

using namespace QPanda;PauliOperator b("X5 Y6", 3);std::map<size_t, size_t> index_map;auto c = b.remapQubitIndex(index_map);// 輸出的值為7std::cout << b.getMaxIndex() << std::endl;// 輸出的值為2std::cout << c.getMaxIndex() << std::endl;

實例

以下實例主要是展示PauliOperator接口的使用方式。

#include "Operator/PauliOperator.h"int main(){  QPanda::PauliOperator a("Z0 Z1", 2);  QPanda::PauliOperator b("X5 Y6", 3);   auto plus = a + b;  auto minus = a - b;  auto muliply = a * b;  std::cout << "a + b = " << plus << std::endl << std::endl;  std::cout << "a - b = " << minus << std::endl << std::endl;  std::cout << "a * b = " << muliply << std::endl << std::endl;  std::cout << "Index : " << muliply.getMaxIndex() << std::endl << std::endl;   std::map<size_t, size_t> index_map;  auto remap_pauli = muliply.remapQubitIndex(index_map);  std::cout << "remap_pauli : " << remap_pauli << std::endl << std::endl;  std::cout << "Index : " << remap_pauli.getMaxIndex() << std::endl;  return 0;}
