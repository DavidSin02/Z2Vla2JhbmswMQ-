護膚品成分全索引~成為達人的必經之路

2020-11-14
| 科學

用於護膚品中的成分多達數千種,要一一列舉出來非常之難,好在總有一些成分是非常常用的。


下面,我將常見的一百多種成分整理出來,供大家參考。如果你可以對這些成分略有了解,掌握其中的規律,那麼理解護膚品的難度就會降低,選擇起來也可以抓住關鍵,知道哪些是自己想要的。


這裡面所提到的成分，有自身特性的描述。有其作用的，比如抗衰老，保溼等等。有敏感度的，比如無刺激，或者低刺激等。


使用方法：按成分第一個字的首字母排序：例如：丙二醇，屬於B的組別。防晒成分和香精香料在文末處！


很多時候，買家秀和賣家秀是截然相反的效果。但是知道了成分，我們就能知道它的功效是否真實！


比如下圖這個精華水的成分表，大家參考這個索引表。大部分的成分都可以查詢出來！



A：

1.氨基酸衍生類表面活性劑:如月桂酰甘氨酸鉀、月桂酰穀氨酸鈉、椰油酰甘氨酸鉀、椰油酰穀氨酸鈉、椰油酰肌氨酸鈉等。此類表面活性劑多用於清潔產品,比皁基更為溫和,可以做成弱酸性體系,清潔力和對皮膚的友好度之間達到了較好的平衡,部分成分還有抗菌作用。

海洋精華水成分解析案例

B:

1.變性乙醇(alcohol denatured/ethanol denat. ) :添加了少量的其他物質使之不能食用的乙醇。多數國家對酒類徵收重稅,變性乙醇能保有酒精的大部分性能,所以仍可用於非食用的領域(燃料、溶劑、清潔劑),這樣就不會被課以重稅。


2.苯乙基間苯二酚(phenylethyl resorcinol) :即SymWhite377,一種美白成分,作為酪氨酸酶的類似物發揮競爭性抑制酪氨酸酶的作用,可減少黑色素合成,達到美白效果。（活性成分）


3.丙二醇( propylene glycol ) :無色無味的黏稠液體,能與水任意混合,與化妝品成分有很好的相容性。其黏性和吸溼性好,可用作潤膚、吸溼和軟化劑,也用作香料和防腐劑的溶劑。丙二醇刺激性很低,無毒,但高濃度的丙二醇可以導致皮膚灼熱感。


4.吡咯烷酮羧酸(PCA) :表皮絲聚蛋白分解後的產物之一,是天然保溼因子的組成成分。吡咯烷酮羧酸的吸溼性優於甘油、丙二醇、山梨醇等,具有良好的生物相容性和化妝品配方兼容性,也極少導致過敏或刺激反應。


5.吡哆素( pyridoxine ) :即維生素B ,包括吡哆醇、吡哆醛和吡哆胺,具有抗老化、減輕皮膚乾燥和抑制皮脂分泌的作用。（活性成分）


6.PCA鈉:見吡咯烷酮羧酸,為吡咯烷酮羧酸的鈉鹽。


7.苯甲酸鈉(sodium benzoate ) :對酵母有很強的抑制作用,對黴菌和細菌也一定程度的抑制作用,無毒。（防腐類）


8.苯氧乙醇(phenoxyethanol ) :廣譜抑菌劑,對真菌、細菌均有抑制作用。駐留型產品中濃度偏高時皮膚可能會產生灼熱感。（防腐類）


C


D:

1.丁二醇(butylene glycol) :無色無味的黏稠液體,能與水任意混合,是常用的保溼劑。其保溼性能很好,但刺激性較丙二醇低,也沒有丙三醇(甘油)黏稠,現在越來越受歡迎。丁二醇和丙三醇及透明質酸鈉配合,可取得很好的保溼效果。


2.豆蔻酸( myristic acid ) :即十四酸,主要用於皁類的製作,其皁基有良好的發泡作用。該成分可能具有輕微刺激性和一定的致粉刺性。


3.豆蔻酸異丙酯( isopropyl myristate ) :一種常用的潤膚劑,同時其作為酯類,也有較強的溶油作用,因而也常用於卸妝產品中。研究者普遍認為該成分具有致粉刺性。


4.多羥基酸類( polyhydroxy acids, PHAs ) :是第二代a一羥基酸,帶有多個羥基,故有更好的保溼作用,同時刺激性更弱,常用的有葡糖酸內酯(gluconolactone )、乳糖酸( lactobionic acid )、麥芽糖酸( maltobionic acid)等。PHAs有抗氧化、抗糖化作用,多用於抗衰老。（活性成分）


5.碘丙炔醇丁基氨甲酸酯( iodopropynyl butylcarbamate) :簡稱IBPC,是一種溫和的、廣譜而穩定的抑真菌劑。（防腐類）


E

鮭魚精華成分解析案例

F：

1.蜂蠟(beeswax) :是一類固體油脂,可能是人類最古老的化妝品原料,主要用於一些膏狀和半固體的產品,如脣膏、髮蠟等。蜂蠟刺激性低,具有一定的抗氧化、抗菌能力。


2.泛醇(panthenol ) :即維生素Bs,是一種水溶性的、穩定的、易穿透角質層的小分子物質。泛醇有保溼、減輕皮膚刺激、舒緩、抗炎和止癢作用,常用於保溼和抗敏類產品。（活性成分）


3.泛醌(ubiquinone) :即輔酶Q10 (Co Q10) ,是一種溫和而天然的成分,也是極強的抗氧化劑。其顏色淡黃,價格較貴。（活性成分）


4.富勒烯( fullerenes ) :是一種由碳原子構成的球狀分子,完全由人工合成。研究發現富勒烯具有極強的抗氧化和清除自由基作用,常用於美白、抗皺、抗衰老產品中,價格極其昂貴。（活性成分）羅伯特·科爾、哈羅德.克羅託

、理查德.斯莫利三人因發現富勒烯而獲得1996年諾貝爾化學獎


G:

1.甘油(glycerin) :化學名為丙三醇,味道是甜的,是一種水溶性保溼劑,同時也是皮膚天然保溼成分的一部分,配合透明質酸鈉保溼效果更好。甘油是經典保溼成分,價格便宜、刺激性低,效果好,所以保溼類產品中通常少不了它。


2.甘油硬脂酸酯(glyceryl stearate ) :一種常用的乳化劑,但乳化能力不強,一般作為輔助乳化劑。類似的還有甘油異硬脂酸酯、甘油硬脂酸酯檸檬酸酯等,與之配合使用的有PEG-20甘油硬脂酸酯、PEG-30甘油硬脂酸酯等。


3.瓜爾豆膠(guar gum) :瓜爾豆植物種子中發現的成分,是一種多糖,具有吸水能力,能在皮膚表面形成膜。瓜爾豆膠常用作增稠劑和助乳化劑。


4.硅氧烷類(dimethicones/siloxanes ) :即所謂的“硅油” ,是一類物質,包括聚二甲基硅氧烷、三硅氧烷、環硅氧烷等。此類成分非常穩定,低刺激,具有良好的潤膚和保溼功能,可以改善礦物油的膚感。

5..有機硅類:包括PEG-10聚二甲基硅氧烷、PEG-9甲醚聚二甲基硅氧烷、PEG/PPG-10/3油基醚聚二甲基硅氧烷、PEG/PPG-20/22丁基醚聚二甲基硅氧烷等。有機硅類和硅氧烷類要區分開來，不要搞混。有機硅類，是表面活性成分。因為都是硅元素類，所以歸為G區


6.光甘草定(glabridin) :是光果甘草的提取物,在不產生細胞毒性的條件下對酪氨酸酶的抑制率可達50%,而且已證明比氫醌的抑制能力高16倍。光甘草定非常昂貴,號稱“美白黃金”。（活性成分）


7.橄欖油(olive oil) : 榨取自橄欖果的天然油脂,主要含有油酸等成分,通常用作精油的基底油,也用於製作軟質的皁類。橄欖油總體上是溫和安全的,但也可能有致粉刺性。


8.甘草酸二鉀( dipotassium glycyrrhizinate) :著名的抗敏成分,是甘草提取物的衍生物,易溶於水,有較低的pH,對油性皮膚亦有幫助。（抗敏修復類）


9.硅石粉(silica ) :也就是二氧化硅,可以調節產品的黏度,常用作填充料和遮蓋成分、潤膚劑的載體,可改善膚感。球形的硅石具有多孔性,可以吸附油脂,是一種惰性而安全的成分。（粉體類）


H:

1..紅沒藥醇( bisabolol ):非常經典的抗炎、舒敏成分,提取自洋甘菊或者蓍草,常用於抗敏、抗炎的產品中。（活性成分）


2..黃原膠(xanthan gum) :一種澱粉的衍生物,有極高效的增稠性,顏色透明,溫和無刺激,同時具有懸浮作用,可以幫助油脂均勻分散,是一種安全、常用的增稠成分。


3.黃芪提取物(astragalus root extract ) :提取自中藥黃芪,具有良好的抗氧化、抗菌能力,常用於美白和抗衰老護膚品。（活性成分）

晶鑽面膜解析案例

4.褐藻酸鈉( sodium alginate):主要由來源於褐藻(如海帶)中的褐藻酸製作而成是褐藻酸的鈉鹽。褐藻酸鈉具有優異的穩定性、溶解性、黏性和安全性,在食品工業、製藥工業都得到了廣泛應用。化妝品中常用它作為增稠劑。


5.滑石粉(talc ) :來自礦物(硅酸鎂),具有非常柔軟和滑爽的手感,通常用於彩妝產品中作為填充料。（粉體類）


I


J:

1.膠原蛋白(collagen) :僅存在於動物皮、骨等組織中的重要結構蛋白,應用於護膚品中的一般是水解後的膠原蛋白或者生物發酵產生的膠原蛋白類產物,它們具有良好的生物相容性和保溼性,膚感很好,亦具有很強的吸水能力,故可用作保溼成分和膚感調理劑。


2.角鯊烯(squalene ) :最早提取於鯊魚的天然油脂類成分,人類皮脂腺也可以自行分泌。角鯊烯具有優良的鋪展性和親膚性,低刺激,沒有特殊的氣味或顏色,配伍性好，但有致粉刺嫌疑。


3.鯨蠟醇(cetyl alcohol ) :即十六醇,又稱棕桐醇,是一種長鏈脂肪醇,為白色蠟狀固體,性質穩定,無特殊氣味或顏色。鯨蠟醇是一種優良的多功能化妝品原料,可以作為潤膚劑、助乳化劑、增稠劑、發泡劑等,可用椰子油或棕桐油為原料製造,也可以人工合成。


4.聚乙二醇類（PEG)：有PEG-10,PEG-100等多種類型,是一類透明、可吸溼、穩定性好的乙二醇聚合物,低分子量的可以用於保溼,高分子量的則可用於脣膏等較硬的產品。


5.聚乙烯醇( polyvinyl alcohol ):成膜劑,有黏性,可以與水結合,增加產品的稠度,多用於彩妝、指甲護理產品和撕拉式面膜。


6.聚甘油硬脂酸酯( polyglyceryl stearate ) :是聚甘油與硬脂酸的酯類,比較穩定,適應性廣,無刺激,食品行業也有應用。常見的有聚甘油-10硬脂酸酯、聚甘油-3硬脂酸酯、聚甘油-10油酸酯等。


7.金盞花提取物( calendula/marigold extract):提取自金盞花的植物成分,有抗炎、舒緩作用,可用於敏感性皮膚和痤瘡護理產品中。（抗敏修復類）


8.季銨鹽類(quaternium ) :一類具有表面活性的抑菌劑,殺菌能力很強,但某些情況下也可以產生刺激。（防腐類）

磨皮霜成分作用解析案例

9.甲基異噻唑啉酮( methylisothiazolinone) :一種曾經比較流行的抑菌劑,低用量下就能起到極好的抑菌效果,但可惜安全性較差,因此歐盟已經禁用。上海市皮膚病醫院的一項實驗發現,該成分在測試的各種防腐劑中是不良反應率最高的。（防腐類）


10.己二醇(hexylene glycol/hexanediol) :即“馨醇H6" ,一種保溼劑,也可以作為溶劑,具有抑菌作用,安全性極佳（防腐類）


K：

1.礦脂(petrolatum) :即凡士林,為C16-C32的高碳烷烴(異構)和高碳烯烴的混合物,可用於乳液、膏霜、脣膏、髮蠟中,也是各種藥物軟膏的主要基質成分。該成分是所有油相原料中封閉保溼性最強的,其質地較為黏稠,因此常和其他油相物質混合使用,以改善膚感。


2.抗壞血酸葡萄糖苷(ascorbyl glucoside) :維生素C的衍生物,具有良好的穩定性,但生物利用度也受到了影響,可算是效果和穩定性折中的產物。（活性成分）


3.抗壞血酸棕櫚酸酯( ascorbyl palmitate) :也是維生素C的一種衍生物。（活性成分）


L:

1.蘆薈(aloe ) :提取自蘆薈葉肉,主要成分為蘆薈多糖,可保溼、抗炎,並有一定抗菌性,能促進修復。


2.卵磷脂(lecithin) :主要提取自大豆和蛋黃,可能是最溫和的乳化劑,亦可作為皮膚調理劑和抗氧化劑。


3.綠茶提取物(green tea extract) :綠茶提取物的有效成分是多酚類,已知可減少DNA損傷、日光炎症和紅斑,是極強的抗氧化劑,同時具有美白、控油作用。綠茶提取物是一類天然安全的有效成分,缺點是顏色較深,一般人難以接受其棕黃色的外觀。（抗敏修復類）


M:

1.明膠(gellatin ) :膠原蛋白的降解產物,具有良好的水溶性,黏度高,因而也可以起到增稠作用。明膠的來源是動物的皮、骨。


2.米糠油( rice bran oil/acid ) :來自大米米皮,含有多種不飽和脂肪酸,具有潤膚作用,據說可以幫助抗衰老和修護皮膚屏障。


3.咪唑烷基脲(imidazolidinyl urea) :應用最廣的防腐劑之一,通常與其他防腐劑配合使用以提升防腐效果,在高溫下可以釋放出更多甲醛。在各種甲醛釋放型防腐劑中,咪唑烷基脲是相對容易引起過敏的。（防腐類）


N:

1.尿素(urea) :是皮膚天然代謝的產物之一,也是一種“真正”的保溼劑。不同於一般的吸溼劑,它可以使角質層保持合適的水分含量,同時可以幫助打通細胞的水通道,還可以促進角質的剝落,亦可以促進其他營養成分的吸收。


2.凝血酸(tranexamic acid) :即氨甲環酸,臺灣地區稱“傳明酸" ,是一種具有止血作用的美白成分,常用於黃褐斑皮膚的護理。（活性成分）


3.尿囊素(allantoin ) :最早提取自紫草科植物的根,有抗炎、舒緩和促進傷口癒合的作用,對痤瘡亦有改善作用。（抗敏修復類）


4.尼泊金酯類(parabens) :是最古老的防腐劑之一,同時也非常溫和,准許使用的包括羥苯甲酯(methyl paraben)、羥苯乙酯(ethylparaben)、羥苯丙酯propylparaben)、羥苯丁酯(butylparaben)等,其中羥苯丁酯可以抗真菌。近年來研究者對其安全性有一些爭議,但美國CIR (化妝品成分評估)認為目前為止的證據表明在化妝品中使用這些成分是安全的。（防腐類）


5.檸檬酸(citric acid) :一種果酸(a-羥基酸),常用於pH調節和輔助防腐一般情況下對皮膚安全,但對於敏感、破損的皮膚可能形成刺激。（其他輔助類成分）

O


P:

1.膨潤土(bentonite ) :是一種硅酸鋁鹽,用於調節化妝品配方的黏度和懸浮性能,具有穩定配方功能。膨潤土可以吸水形成膠狀物質,不致粉刺,在彩妝產品中應用較多。


2.葡糖酸鋅(zinc gluconate ) :易溶的含鋅原料。鋅是一種對皮膚很重要的離子,具有抗炎、舒緩作用,也可幫助抑制皮脂。葡糖酸鋅常用於過敏、痤瘡、敏感特應性皮炎等皮膚問題適用的護膚品中。（活性成分）


Q：

1.氫化蓖麻油(hydrogenated castor oil) :蓖麻油有一定氣味,難於使用,將其做加氫處理後,顏色變得更白,性質更穩定,氣味也得到改善。氫化蓖麻油是一種溫和的油脂,能起到潤膚作用。


2.氫化橄欖油(hydrogenated olive oil) :與氫化蓖麻油類似。


3.青咖啡果提取物( coffee fruit extract):是一類抗衰老物質,含有大量多酚綠原酸,具有很強的抗氧化能力,能提升皮膚彈性,在護膚領域很有前景。（活性成分）

根據防晒成分分析案例


首先，得知道什麼是表面活性劑。定義：凡是溶於水能夠顯著降低水的表面能的物質稱為表面活性劑（SAA)或表面活性物質。表面活性劑(surfactant)，是指加入少量能使其溶液體系的界面狀態發生明顯變化的物質。


其實，我看了這個定義也是雲裡霧裡的。但是，通過推測，應該是加了活性劑之後，護膚品整體的狀態發生了變化。比如加活性劑之前可能比較硬，加了之後就軟化了。也可能，加之前像水一樣，亂流，加了之後，就粘稠一些，易於塗抹。


保溼增稠是什麼意思？保溼應該不用多講，就是使得皮膚變得水潤，減少乾燥起皮。那麼，增稠呢？


增稠是使懸浮於液體中的固體粒子變為濃稠的攏漿而從液體中分離出來的操作即稱為增稠，也稱為沉澱濃縮或沉降濃縮。這是增稠的定義，說的太嚴禁了，以至於小編看不懂。不過簡單來講，應該就是從液體裡分離出某些成分。比如說酒精和酒的混合物，如何提煉出更加純的酒精呢？那麼可以用蒸餾的方法，只不過蒸餾是物理方法，這裡的增稠劑是化學方法。


活性成分又稱有效成分，植物活性成分指構成植物體內的物質除水分、糖類、蛋白質類、脂肪類等必要物質外，還包括其次生代謝產物 （如萜類、黃酮、生物鹼、甾體、木質素、礦物質等）。這些物質對人類以及各種生物具有生理促進作用，故名為植物活性成分。
