依頓電子（603328）：國內PCB行業領軍企業，盈利能力遙遙領先

2020-11-14
| 科學

1、國產PCB領軍企業

公司是國內印刷線路板行業的領軍企業，國內前十大 PCB 廠商之一，連續多年入選行業研究機構N.T.Information 發佈的世界 PCB 製造企業百強以及中國印製電路行業協會（CPCA）發佈的中國 PCB 百強企業，並被 CPCA 評為優秀民族品牌企業。根據N.T.Information 發佈的 2016 年全球百強 PCB 製造商排名，按年銷售收入排名公司位於全球第 36 位。

公司主營產品線路板按層數可分為雙面板、四層板、六層板、八層及以上板，產品廣泛應用在汽車電子、通訊設備、消費電子、計算機、工業控制等下游行業產品上。

公司客戶群分佈廣泛，優質客戶眾多，主要客戶覆蓋汽車、通訊、消費電子等眾多領域。公司業務佈局具有前瞻性，主動調整客戶結構迎合汽車電子需求風口，目前汽車電子佔營收比例在 PCB 同行中處於領先水準。

得益於業務結構不斷優化，公司盈利能力位於 PCB 行業領先水平。2017 年公司實現主營業務收入 32.86億元，同比增長 11.97 %；毛利率 33.37 %，同比上升 4.8 %，創開業以來新高，主要通過調節客戶結構改善盈利。

2、盈利能力行業領先，產能釋放支撐業績增長

近年來，隨著汽車電子、新能源汽車、人工智能、小基站等高成長性領域的快速發展，PCB 行業已經進入新一輪的景氣週期。針對行業環境的變化，公司戰略上立足於“質量至上、成本領先、技術領先”的方針，在多個方面努力提升自身實力。

公司的主營業務為高精度、高密度雙層及多層印刷線路板的製造和銷售。公司線路板產品可分為四層板、雙面板、六層板、八層板及以上。其中雙面板應用於汽車業務、充電器領域為主；四層、六層板應用於通訊和汽車業務和電腦鍵盤領域；八層以及以上板主要服務通訊基站領域。公司技術力量雄厚，具備各類印刷線路板的生產能力，產品最小孔徑可達 0.10mm，最小線寬可達 0.05mm，最高層數可達 24 層。

在產品應用領域方面，公司努力提高汽車、醫療及工業等高毛利 PCB 產品的份額。2017年，鍵盤產品業務和汽車業務發展勢頭良好，鍵盤產品銷量同比增長 210%，汽車產品銷量同比增長 34%。公司對產品結構優化效果較為明顯。

公司 2017 年產能為 324 萬平方米，2018 年一季度募投新增產能 110 萬平方米設備基本準備完畢。隨著下游市場特別是汽車智能化和舒適度提升導致汽車 PCB 面板用量加大以及公司近年持續加大在汽車領域的投入，新增產能有望順利消化，快速提升公司業績。

3、PCB 景氣上行

印製電路板作為電子設備的基礎組件，被譽為“電子產品之母”。印製電路板的下游產業涵蓋計算機及其周邊、消費電子、網絡通訊、醫療、汽車電子、軍事、航天科技產品等領域，目前 尚沒有能夠替代印刷電路板的成熟技術和產品，不可替代性是印製電路板製造行業得以始終穩固發展的要素之一。經過幾十年的發展，PCB 行業已成為全球性大行業，PCB 產業產值佔電子元件產業總產值 10%以上，是電子元件細分產業中比重最大的產業。近年來，在 3C 行業穩定發展的 基礎上，隨著汽車電子、新能源汽車、人工智能、小間距 LED、高端服務器、小基站等高成長性 領域的快速發展，PCB 行業已進入新一輪的景氣週期。

PCB 產值增速和全球經濟增速有較強相關性。在全球經濟復甦的大背景下，全球 IT 市場同步恢復，PCB 作為其配套行業其市場需求與 IT 行業的市場需求成正比例關係。近年來全球經濟扭轉疲態，復甦力度走強，歐美經濟快速回暖。2017 年全球經濟增速達 3.0%，明顯高於近十年 2.3% 的平均水平，也高於 1980-2017年 2.8%的平均水平。相應，2017年 PCB 市場回暖，實現 552.77 億美元，同比增速為 1.86%，為三年來最高增速。

汽車電子類、通訊設備類需求在新能源汽車、5G 商用等刺激下有望顯著上升，成為 PCB行業景氣度上行的有力支持。Prismark 預測 2018 至 2022 年期間全球 PCB 產值將保持年均複合增長率 3.2%的速度增長。從最近臺灣 PCB 大廠月度營收數據來看，整體 PCB行業持續復甦，景氣度有望維持。

目前，全球印製電路板產業的發展已形成中國大陸、臺灣地區、日本、韓國、美國、歐洲和東南亞地區等主要生產中心，其中亞洲地區（含中、日、韓及東南亞）佔全球印製電路板總產值的比例超過 90%。

從近十年產業轉移趨勢看，中國大陸已經成為全球 PCB 產值主要集中地，佔全球產值比50%以上。2016 年-2017 年，中國 PCB 整體產值保持平穩增長，年增速約為 3%，明顯高於全球 PCB 產值整體增速，產業持續往中國大陸轉移。從世界 PCB 產值地位上看，中國大陸廠商主要面臨與臺灣廠商的競爭。陸資 PCB 企業提升技術、生產水準，逐漸縮短甚至超趕臺資同行是突破成長空間的關鍵所在。

4、公司未來看點

公司作為國產 PCB 大廠的領軍代表，有望引領 PCB 國產全面崛起進程。公司優勢主要集中在技術、客戶及產能三大方面：

技術優勢：持續的工藝改進與具備全面的生產技術是公司長期發展的核心競爭力和重要保障。公司技術力量雄厚，具備各類剛性印刷線路板的生產能力。在產品研發方面，公司持續完善科技創新體系，增強自主創新能力和技術競爭力。2017 年，公司研發投入為 10322.77 萬元，佔公司全年營業收入的 3.14%。

客戶優勢：公司不斷開拓高端 PCB 產品市場，努力擴大汽車、醫療及工業等高附加值 PCB 的份額佔比，從而提高公司訂單質量、增強公司的盈利能力。公司自 2013 年開始起著力優化產品訂單結構，積極拓展中高端客戶，至 2017 年產品結構已從中低端轉為中高端，同時擁有了大量的中高端客戶群。其中主要客戶包括 Jabil（捷普）、Flextronics（偉創力）、緯創、華為、 Lite-On (旭福)、Delphi(德爾福)、Hon Hai(鴻海)、Preh（普瑞）、Continental（大陸）、Astec（雅達）、 Bose 和 Valeo（法雷奧）。此外公司十分注重在汽車領域的開拓，目前為止公司的公司汽車業務主要客戶有法雷奧、博世、大陸電子、德爾福、李爾、普瑞。目前對法雷奧的出貨佔到公司汽車業務的 3 成；公司 2017 年開始開拓其他車系客戶例如日系小糸。汽車電子領域對產品性能穩定性要求較高，易與客戶形成較強黏性，推高公司利潤。公司與優質汽車客戶率先建立的穩定合作關係為公司未來發展奠定了良好的市場基礎。

新增產能：公司 2017 年產能為 324 萬平方米。公司募投新增產能 110 萬平方米項目，經過 4 年建設，目前設備基本準備完畢。同時公司擬對 “年產 45 萬平方米 HDI 印刷線路板項目”進行變更，將該項目尚未使用的募集資金變更用於“年產 70 萬平方米多層印刷線路板項目”建設。新產能逐步投放助力公司更上新臺階。

5、厚積薄發，公司汽車電子業務進入收穫期

公司在上市初期生產消費電子、計算機、通信等產品的 PCB 為主，汽車 PCB 僅 11%左右。公司從2014年開始佈局汽車業務，通過不斷改善產品結構，汽車類業務在2014年-2017年間保持 30%左右的年增長率。2017年汽車電子佔營收比例已提高到 36%，實現營收約12 億元，在產品結構中份額最大。

公司生產的汽車 PCB 多為四層板，毛利高於普通的四層板甚至六層板，相對於普通 PCB有 5%-10%的溢價。公司的通過縮減低毛利的 PC 和通訊類產品佔比、增加汽車板和優質消費電子客戶業務佔比，不斷調整產品結構。汽車 PCB 的增速和利潤提高明顯，無論是從份額、利潤還是市場需求上，都是拉動公司未來業績增長的主要動力。

公司在汽車電子領域主要客戶以 Valeo（法雷奧）、Bosch（博世）、Continental（大陸）、Delphi(德爾福)、Preh（普瑞）、Bose（博世音響）等德系和美系車為主，2017 年開拓了Koito（小糸）等日系客戶。公司汽車板產品以四層板為主，主要應用到 LED 車燈、車載音響娛樂系統、車載空調、ECU（電子控制單元）、BCM （車身控制模塊）以及動力系統等領域。

目前公司的汽車產品種類儲備較多，除了新能源汽車的 BMS（電池管理系統）領域，其它領域都有涉及。汽車電子有著相對較高的進入壁壘，而一旦進入供應鏈後合作關係也會相對穩定，一般不會輕易更換供應商，有利於公司獲得長期且穩定的訂單。雖然汽車電子新客戶的認證週期較長，但老客戶的新產品導入週期則相對較短。公司在汽車電子領域經營多年，並在 2014 年開始主動調整客戶結構逐漸向汽車電子傾斜，憑藉技術與時點優勢切入較早，公司已完成眾多知名汽車零部件供應商的認證工作併成為 Tier-1穩定供應商，正進入業務收穫期。

在產能方面，自上市以來公司主要在調整產品結構，實際新增產能十分有限。2017 年，公司產能為 324萬平米。經過了 3 年的建設，IPO 募投 110 萬平方米多層印刷線路板項目已於 2018 年一季度設備安裝到位，市場開拓到位即可投產。該項目產能為 110 萬平方米/年，預計將帶來 30%以上產能擴張。藉助新增產能，公司有望持續發力車載 PCB市場。

6、投資建議

公司是國內印刷線路板行業的領軍企業，國內前十大 PCB 廠商之一。公司多年來前瞻性對客戶結構做出調整，精準針對汽車電子市場開拓優質客戶。公司毛利率和淨利率分別高達 33.37%、16.83%，在全球 PCB 行業中，公司的盈利能力名列前茅。隨著新增產能投放，公司有望進入新一輪高速增長期。

PCB 產值增速和全球經濟增速有較強相關性。在全球經濟復甦的大背景下， PCB 景氣度有望持續上行。公司及時擴張產能，有望受益國產 PCB 產業的全面崛起，在市場開拓、技術積累等領域打開新局面，產業地位更上臺階。

近年來汽車電子在整車的應用場景不斷豐富。汽車電子配套的 PCB 需求大幅增長，成為整體 PCB 行業的重要需求支撐點。公司前瞻性佈局汽車電子客戶，目前開拓汽車電子客戶涵括 Valeo（法雷奧）、Bosch（博世）、Continental（大陸）、Delphi(德爾福) 、Preh（普瑞）、Bose（博世音響）、Koito（小糸）等汽車電子巨頭。目前公司汽車電子業務佈局領先同行，業務已經進入收穫期，盈利能力有望持續提升。

計公司 2018 年-2020 年實現利潤 6.96/ 8.57/9.39 億元，對應 PE 分別約為 14X、11X 、10X，我們看好整體PCB 行業的景氣上行和公司汽車電子業務發力，給予“強烈推薦”評級。
