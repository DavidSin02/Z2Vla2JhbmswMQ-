尿八項尿分析原理及其臨床意義

2020-10-29
| 科學



一、葡萄糖（GLU)


[反應原理]：採用酶法測試，用兩種酶，分別是葡萄糖氧化酶和過氧化氫酶。主要反應過程如下：


現在幾乎所有的試紙都是採用酶法，因為酶法具有特異性強、靈敏度高、反應時間短等優點。不同型號的試紙所選用的指示劑有所不同。


[臨床意義]：


1. 生理性糖尿為一過性糖尿，是暫時性的，排除生理因素後恢復正常。主要有三種：①飲食性糖尿，即在短時間內服用大量糖類，引起血糖濃度過大；②應急性糖尿，在腦外傷、腦血管意外、情緒激動、劇烈運動週期性四肢麻痺等情況下，延腦糖中樞受刺激，使腎上腺激素或胰島素分泌異常，可出現暫時性的糖尿；③妊娠中後期多可見糖尿。

2. 病理性糖尿也可分為三種：①真性糖尿，既胰島素的分泌量相對或絕對不足，使血糖濃度超過腎糖閾尿糖檢查不僅可以診斷糖尿病，還可以指導臨床醫生決定胰島素的用量、判斷療效；②腎性糖尿，即腎小管對葡萄糖的重吸收功能減退，新生兒的近曲小管功能未完善也能出現糖尿；③其他糖尿，如生長激素過多（肢端肥大症）、甲狀腺激素過多（甲亢）、腎上腺激素過多（嗜鉻細胞瘤）、皮質醇（Cμshing綜合症）、胰高血糖素等都可使血糖濃度高過腎糖閾而出現糖尿；另外，肥胖病、高血壓也可能出現糖尿。

[注意事項]：

1. 因為尿糖分析試紙的後一步反應是氧化還原反應，當尿液中含有比色素還原能力更強的物質時，可使測試結果偏低甚至出現假陰性。如尿液中含有維生素C時就能使測試結果偏低甚至假陰性。

2. 抗生素對班氏法糖定性、糖定量測定結果都有一定的影響，而對幹化學法的測試結果無影響。尿液存放時間過長也能使尿糖被細菌分解使濃度下降，但含有抗生素時幾乎不下降。

3. 高濃度的酮體尿可引起假陰性；尿液比重增高，可降低試劑帶對尿糖的敏感性；服用大量左旋多巴時，該藥物的代謝產物會對測試反應產生抑制作用，使測試結果偏低或出現假陰性；尿液被過氧化物或次氯酸鹽等強氧化性物質汙染的時候能出現假陽性。

4. 試劑帶採用的使酶促反應，測定的結果和溫度及時間有關，因此應在規定的溫度和時間內測定。



二、膽紅素(BIL)、尿膽原(URO)


[膽紅素反應原理]：在酸性條件下，重氮鹽作用於膽紅素的中央，使其斷開並與重氮鹽偶合形成2分子的偶氮膽紅素，從而產生顏色變化。


[尿膽原反應原理]：尿膽原分析試紙的反應原理一般有兩種，一種是尿膽原在酸性條件下與對-二甲氨基苯甲醛發生醛化反應(即尿膽原與醛縮合生成紅色的縮醛化合物，既常見的歐氏試劑)。另一種是重氮鹽法，即尿膽原在酸性條件下，與重氮鹽偶聯生成紫紅色的偶氮化合物。


[臨床意義]：


1. 膽紅素的檢測對肝膽系統疾病的診斷有重要價值。尿膽原更加靈敏的反映肝功能。

2. 膽紅素的檢測有助於診斷黃疸。在敗血症、蠶豆病、異型輸血等情況下使紅細胞大量破壞，產生溶血性黃疸，此時雖然膽紅素大量增加，但大部分是間接膽紅素，因此，尿中的膽紅素還是陰性。

3. 膽素原族的檢測可以敏感的反映肝細胞功能，臨床經驗表明，在病毒性肝炎早期未出現黃疸前，尿中的膽素原族就已經明顯增加。與膽紅素結合可以為診斷黃疸的類型提供依據。黃疸大致可分為三類：①肝前性黃疸，或稱溶血性黃疸，②肝原性黃疸，或稱肝細胞性黃疸，是在感染（如病毒性肝炎）、中毒及肝硬化等情況下，肝細胞或肝內毛細膽管大量病變，使肝細胞對膽紅素攝取、結合、轉運及排洩障礙所致；③肝後性黃疸，或稱梗阻性黃疸，由於結石、腫瘤或先天性膽道閉鎖等原因造成總膽管梗阻所致。

[注意事項]:

1. 標本必須新鮮，以免膽紅素被氧化成膽綠素，強烈的陽光會加速此反應。放置時間長可使尿膽原氧化成尿膽素。

2. 尿液中含有高濃度的維生素C或亞硝酸鹽時，會抑制重氮偶合反應，可能出現假陰性結果。當病人接受大劑量氯丙嗪治療或尿中含有鹽酸苯偶氮吡啶的代謝產物時，可呈假陰性。

3. 由於分析試紙無尿膽原陰性的梯度，因此，分析試紙不能用來檢測尿膽原的減少或消失。

4. 尿液中的一些內源性物質如膽色素原、吲哚、膽紅素等，可是測試結果出現假陽性；一些藥物如吩噻嗪等可產生顏色干擾。另外，尿液中含有大量維生素C或亞硝酸鹽時，可抑制重氮偶合反應，使測試結果偏低甚至出現假陰性。


三、酮體(KET)


[反應原理]：尿酮體包括乙酰乙酸、丙酮、β-羥丁酸，後者雖不屬於酮類，但經常與前兩者伴隨出現，因而統稱為酮體。反應原理是在鹼性條件下，尿中的乙酰乙酸、丙酮與亞硝基鐵氰化鈉反應，生成紫紅色的複合物。這種測試方法對乙酰乙酸的敏感度為5～10mg/dl，對丙酮的敏感度為40～70mg/dl，並且不與β-羥丁酸反應。


[臨床意義]：


1. 糖尿病酮症酸中毒。糖利用減少，脂肪分解產生過量酮體，尿酮體的檢查對未控制或治療不當的糖尿病出現酸中毒或昏迷的診斷很有價值，可以與低血糖、心腦疾病酸中毒或高血糖滲透性糖尿病昏迷相區別。

2. 感染性疾病（如肺炎、傷寒、敗血症、結核等發熱期），嚴重嘔吐、腹瀉，長期飢餓、禁食，全身麻醉後等都可能出現酮尿。另外婦女孕期因妊娠反應嘔吐多、進食少，體脂肪代謝明顯增多，也能出現酮尿。

3. 氯仿、乙醚麻醉後、磷中毒等情況也能出現酮尿。

4. 服用雙胍類降糖藥如降糖靈後，由於藥物抑制細胞呼吸，也可能出現酮尿。

[注意事項]：

1. 由於尿酮體中的丙酮和乙酰乙酸都是揮發性物質；乙酰乙酸受熱易分解成丙酮；尿液被細菌汙染後，酮體消失，因此，尿樣必須新鮮，檢測應該及時，以免測試結果偏低或出現假陰性。

2. 幹化學法測定酮體時對乙酰乙酸的敏感度約是丙酮的7～10倍，因此，與其他的檢測方法存在一定的差別 。


四、潛血（BLD）


[反應原理]：尿液潛血分析試紙的反應原理是利用血紅蛋白中的亞鐵血紅素的假過氧化物酶活性催化分解過氧化物，產生新生態的氧，氧化指示劑，使指示劑顯色，從顯色的強度可以得知尿液中血的濃度。


[臨床意義]：


1. 尿液中混有0.1%以上的血液時，便呈肉眼血尿的特徵，血量在此以下便只能用潛血反應或尿沉渣鏡檢才能證明。血尿常見於尿路炎症（急性腎炎、腎結核、尿道炎等）、結核、腫瘤,有白細胞時則表示有炎症；蛋白陽性、尿沉渣中有腎上皮細胞、管型等時，應考慮腎炎；特別是有紅細胞管型時，是腎實質出血的佐證。

2. 血紅蛋白尿見於發作性血紅蛋白尿症，還見於各種中毒、感染、鏈球菌敗血症、瘧疾（黑水熱）、灼傷、溶血性輸血反應等情況。

[注意事項]：

1. 成年女性的經血常可引起測試結果出現假陽性，因此應採取必要的採尿措施以減少汙染。

2. 分析試紙不僅可以測試紅細胞，還能測試血紅蛋白，因此，當血紅蛋白破裂時，可引起試紙測試結果與鏡檢結果的不一致，應加以區別。

3. 尿液中含有對熱不穩定酶、肌紅蛋白或菌尿，可引起假陽性；尿液中含有大量維生素C時，可以抑制後一步反應的進行，使測試結果偏低甚至出現假陰性，因此，測試時應使用具有抗維生素C干擾能力的分析試紙。


五、酸鹼度（pH）


[反應原理]：pH試紙的應用是非常廣泛的，pH的反應原理是基於pH指示劑法，目前，一般的尿液pH分析試紙中含有甲基紅[pH4.2(紅)～6.2(黃)]，溴甲酚綠[pH3.6（黃）～5.4（綠）]溴百里香酚藍[pH6.7(黃)～7.5(藍)]，這些混合的酸鹼指示劑適量配合可以反映尿液pH4.5～9.0的變異範圍。


[臨床意義]：


正常尿液呈弱酸性，約為6.0，因食物的不同，pH的變化範圍為4.5～8.0。pH測定結果的判斷比其他實驗結果的判斷要難一些，因為尿液pH的波動幅度大而快，而正常尿和病人的尿的pH沒有明顯的差別，所以pH單獨使用的意義不大。但與其他臨床資料配合起來分析，則可成為重要的資料。


[注意事項]：


1. 檢測時尿標本必須新鮮，放置過久細菌分解尿液成分可導致尿液pH改變。大多數情況下，細菌分解尿素產生氨，使尿液呈鹼性；少數情況下，細菌也分解尿液成分產生酸性物質，使尿液pH偏酸。

2. 當腎臟分泌的尿液含有過多的碳酸氫鹽和碳酸緩衝對時，放置時間過長，使尿液中的二氧化碳自然擴散到空氣中，使尿液pH增高。

3. 在測定過程中，應嚴格按照規定的時間將試劑帶浸泡尿液標本中，浸泡時間過長，尿pH呈減低趨勢。


六、蛋白質(PRO)


[反應原理]：蛋白誤差法，某種特定的pH指示劑陰離子受蛋白質陽離子吸引，進一步電離，使指示劑的變色點發生改變，這種方法稱為蛋白誤差法。尿試紙中含有四溴酚藍（TBPH)指示劑和pH為3的緩衝體系，TBPH的變色點在pH3～4之間，它在尿中解離成陰離子，與蛋白質陽離子結合後呈黃色。


[臨床意義]：檢驗尿液蛋白是臨床尿液常規化學檢查之一，尿液蛋白試製主要檢測的是尿液中的白蛋白。正常情況下，健康成人每日排出的蛋白質含量極少（約為30～120mg），一般的常規定性方法都不能檢測出來。一旦尿液蛋白含量>100mg/L或>150mg/24h尿，尿液蛋白質定性實驗便呈陽性反應而稱為蛋白尿。蛋白尿主要反映腎小球（管）損害及腎小球濾過率的增加。


[注意事項]：


1. 尿液標本必須新鮮，變質的尿液會使尿液的pH值產生變化。尿液本身過酸會出現假陰性、過鹼會出現假陽性，緩衝體系適合檢測的pH範圍為3～7.5。例如含有奎寧、奎寧丁和嘧啶等藥物時，尿液呈鹼性（pH>8.0），超過了試紙本身的緩衝能力，可能出現假陽性結果。

2. 尿液分析試紙中的TBPH對於白蛋白的敏感性遠遠超過其他蛋白，因此，在尿液中僅含有其他種類的蛋白時，該方法的測試結果可能為陰性。

3. 試紙條不能長時間浸泡於尿液中，否則由於試劑溶於尿液中導致失效而出現假陰性。

4. 多種物質（大多為藥物）對尿蛋白的測定結果都有影響，如青黴素可以使測試結果偏低甚至出現假陰性；季胺鹽、PVP（聚乙烯吡咯烷酮）、喹啉等可使試紙出現假陽性；某些洗滌液汙染尿液時，測試結果會偏低。另外，大量飲水會稀釋尿液，可能造成漏檢。


七、亞硝酸鹽（NIT)


[反應原理]：尿液中的亞硝酸鹽與試紙塊中的對氨基苯砷酸或磺胺發生重氮化反應，生成重氮鹽，生成的重氮鹽再與試紙上的N-1-萘基乙二胺鹽酸鹽或四氫苯並喹啉-3-酚偶合生成紅色的偶氮化合物（蓋氏試劑法）。


[臨床意義]：正常情況下，尿液亞硝酸鹽的定性實驗一般為陰性。當泌尿系統受到感染時，由於細菌還原硝酸鹽生成亞硝酸鹽，因此檢測結果為陽性，常見於大腸埃希氏菌引起的泌尿系統感染。尿液亞硝酸鹽檢測結果為陽性時，預示著尿液中的細菌數量在10萬/ml以上。


[注意事項]：


1. 當尿液中缺少硝酸鹽時，即使有細菌感染也會出現陰性結果；尿液在體內的留存時間太短會因為硝酸鹽來不及還原而得到陰性結果。

2. 留取標本的樣杯必須清潔，並及時送檢，以免存放時間過長使細菌生長而出現假陽性結果。

3. 使用利尿劑後，尿中的亞硝酸鹽含量降低，可能出現假陰性；硝基呋喃可降低反應的靈敏度；非那吡啶可引起假陽性；使用抗生素後，細菌被抑制可出現假陰性；尿液中含有大量維生素C時，也可能出現假陰性。

4. 高比重尿可降低測試反應的靈敏度，尿中的亞硝酸鹽離子小於1.0mg/l時，可能出現假陰性結果。
