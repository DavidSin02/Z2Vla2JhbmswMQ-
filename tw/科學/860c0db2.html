支持向量機（SVM）

支持向量機（Support Vector Machine，常簡稱為SVM）是一種監督式學習的方法，可廣泛地應用於統計分類以及迴歸分析。支持向量機屬於一般化線性分類器，這種分類器的特點是他們能夠同時最小化經驗誤差與最大化幾何邊緣區，因此支持向量機也被稱為最大邊緣區分類器。

如上圖所示，SVM的目的是尋找一個最優分割面，使得分類分類間隔最大化。而最優分割面求解原則可以大致歸納如下：

最優決策面能夠容忍更多噪聲—>所有樣本與分割超平面的距離儘可能遠—>最差的樣本（離分割超平面最近的樣本）與分割超平面的距離要儘可能遠。

邊際（margin）是什麼？

邊際就是某一條線距離它兩側最近的點的距離之和。

SVM 將會尋找可以區分兩個類別並且能使邊際（margin）最大的超平面（hyper plane），即劃分超平面。

所有坐落在邊際超平面上的點被稱作支持向量(support vectors)

支持向量（support vector）就是剛好貼在邊際所在的平面上的點，它們是用來定義邊際的，是距離劃分超平面最近的點。

作用： 支持向量因為支撐了邊際區域，並且用於建立劃分超平面。

注意： 支持向量可能不止一側一個，有可能一側有多個點都貼在邊際平面上。

在線性不可分的情況下，數據集在空間中對應的向量無法被一個超平面區分開，如何處理？

兩個步驟來解決：

  • 利用一個非線性的映射把原數據集中的向量點轉化到一個更高維度的空間中（比如下圖將二維空間中的點映射到三維空間）
  • 在這個高維度的空間中找一個線性的超平面來根據線性可分的情況處理

比如想要將紅點和藍點變成線性可分的，那麼就將映射 y=x 變成映射 y=x^2,這樣就線性可分了。



支持向量機（SVM）特性

（1）訓練好的模型的算法複雜度是由支持向量的個數決定的，而不是由數據的維度決定的。所以 SVM 不太容易產生 overfitting。

（2）SVM 訓練出來的模型完全依賴於支持向量，即使訓練集裡面所有非支持向量的點都被去除，重複訓練過程，結果仍然會得到完全一樣的模型。

（3）一個 SVM 如果訓練得出的支持向量個數比較少，那麼SVM 訓練出的模型比較容易被泛化。

代碼實現

SVM.py

print(__doc__)# 導入相關的包import numpy as npimport pylab as pl  # 繪圖功能from sklearn import svm# 創建 40 個點np.random.seed(0) # 讓每次運行程序生成的隨機樣本點不變# 生成訓練實例並保證是線性可分的# np._r表示將矩陣在行方向上進行相連# random.randn(a,b)表示生成 a 行 b 列的矩陣，且隨機數服從標準正態分佈# array(20,2) - [2,2] 相當於給每一行的兩個數都減去 2X = np.r_[np.random.randn(20, 2) - [2, 2], np.random.randn(20, 2) + [2, 2]]# 兩個類別 每類有 20 個點，Y 為 40 行 1 列的列向量Y = [0] * 20 + [1] * 20# 建立 svm 模型clf = svm.SVC(kernel='linear')clf.fit(X, Y)# 獲得劃分超平面# 劃分超平面原方程：w0x0 + w1x1 + b = 0# 將其轉化為點斜式方程，並把 x0 看作 x，x1 看作 y，b 看作 w2# 點斜式：y = -(w0/w1)x - (w2/w1)w = clf.coef_[0]  # w 是一個二維數據，coef 就是 w = [w0,w1]a = -w[0] / w[1]  # 斜率xx = np.linspace(-5, 5)  # 從 -5 到 5 產生一些連續的值（隨機的）# .intercept[0] 獲得 bias，即 b 的值，b / w[1] 是截距yy = a * xx - (clf.intercept_[0]) / w[1]  # 帶入 x 的值，獲得直線方程# 畫出和劃分超平面平行且經過支持向量的兩條線（斜率相同，截距不同）b = clf.support_vectors_[0] # 取出第一個支持向量點yy_down = a * xx + (b[1] - a * b[0])b = clf.support_vectors_[-1] # 取出最後一個支持向量點yy_up = a * xx + (b[1] - a * b[0])# 查看相關的參數值print("w: ", w)print("a: ", a)print("support_vectors_: ", clf.support_vectors_)print("clf.coef_: ", clf.coef_)# 在 scikit-learin 中，coef_ 保存了線性模型中劃分超平面的參數向量。形式為(n_classes, n_features)。若 n_classes > 1，則為多分類問題，(1，n_features) 為二分類問題。# 繪製劃分超平面，邊際平面和樣本點pl.plot(xx, yy, 'k-')pl.plot(xx, yy_down, 'k--')pl.plot(xx, yy_up, 'k--')# 圈出支持向量pl.scatter(clf.support_vectors_[:, 0], clf.support_vectors_[:, 1],           s=80, facecolors='none')pl.scatter(X[:, 0], X[:, 1], c=Y, cmap=pl.cm.Paired)pl.axis('tight')pl.show()

總結

SVM優點

- 可用於線性/非線性分類，也可以用於迴歸，泛化錯誤率低，也就是說具有良好的學習能力，且學到的結果具有很好的推廣性。

- 可以解決小樣本情況下的機器學習問題，可以解決高維問題，可以避免神經網絡結構選擇和局部極小點問題。

- SVM是最好的現成的分類器，現成是指不加修改可直接使用。並且能夠得到較低的錯誤率，SVM可以對訓練集之外的數據點做很好的分類決策。

SVM缺點

- 對參數調節和和函數的選擇敏感。
