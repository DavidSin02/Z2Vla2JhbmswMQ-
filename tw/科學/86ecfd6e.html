“老胃病”反覆發作，醫生提醒早診斷早治療

2020-11-14
| 科學

&nbsp; &nbsp; &nbsp;

&nbsp; &nbsp; &nbsp; &nbsp;.胃十二指腸潰瘍患病十年，反覆發病、反覆住院，近日，六旬“老胃病”患者久病成醫、有感而發，寫下了近3000字自述病史，提醒大家聽醫生的話，遠離疾病困擾。

&nbsp; &nbsp; &nbsp;【牟大爺：我是多年潰瘍病的老患者了，對得病有點心得，有點體會，我就回憶了自己得病的歷程，把它整理下文字材料。】

&nbsp; &nbsp; &nbsp;近日，年過六旬的“老胃病”牟大爺因十二指腸球部潰瘍合併上消化道出血，再次住進了哈市紅十字中心醫院普內科。這幾天，在醫護人員的精心治療與護理下，牟大爺病情好轉，於是他開始在病房創作，將這次的住院經歷寫下來。

2011年牟大爺就因十二指腸球部潰瘍合併上消化道出血在醫院住院治療過，這些年，他也多次因此病住院治療。根據自身的患病經歷，牟大爺反思了自己患病的原因。

&nbsp; &nbsp; &nbsp; 【牟大爺：住院在五次以上，反覆發作應該有十次了，主要第一點就是我生活沒有規律，暴飲暴食、抽菸喝酒、打麻將，這些缺點我都有，麻將打得特別厲害，有時中午飯、甚至晚飯都不吃，一直玩到二半夜，愛發脾氣、精神緊張，普內科醫生說，這些都是潰瘍病病人發病的原因。】

&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;

&nbsp; &nbsp; &nbsp; &nbsp;最近的一次住院是在今年的十月九號，胃鏡檢查的結果還是潰瘍病發作。哈市紅十字中心醫院普內科主任李晨玲告訴牟大爺，如此反覆發作會局部形成瘢痕，胃的出口和十二指腸會變形，影響食物通過，預後不良！醫生的話警示了牟大爺，他再也不敢大意了，這不，牟大爺想了想這些年的病情發作，寫出了患病心得進行反思。

&nbsp; &nbsp; &nbsp; &nbsp; 針對牟大爺的情況，醫生提醒，消化性潰瘍需要重視，臨床上許多患者出現反覆上腹部疼痛，做胃鏡發現是消化性潰瘍。當病情比較重或者治療不及時、長期反覆發作，有出血、穿孔、梗阻，甚至癌變的風險。患者應戒菸、酒、茶、辣椒、生冷刺激等傷及胃黏膜、影響脾胃功能的食物，不要暴飲暴食，每餐七分飽。不良的情緒也是消化性潰瘍的病因，因此要注意心平氣和，平時可以通過散心、轉移注意力排解不良情緒。此外，每到秋冬季，很多人藉著“抓秋膘”“驅寒”等理由，吃一些高熱量的食物，但對於消化性潰瘍患者而言，這些食物就很有可能會誘使疾病復發。

&nbsp; &nbsp; &nbsp;【李晨玲：如果有症狀當然要及時就診，沒有症狀也要定期做胃腸鏡檢查，這是非常好的早期的進行診斷和治療最有效的辦法。】
