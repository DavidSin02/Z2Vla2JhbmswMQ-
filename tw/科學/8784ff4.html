第2節 現代生物技術在育種上的應用

(時間：30分鐘　滿分：50分)

一、選擇題(共6小題，每小題4分，共24分)

1．切取某動物合成生長激素的基因，用某種方法將此基因轉移到鯰魚的受精卵中，從而使轉基因鯰魚比同類個體大3～4倍，此項研究遵循的原理是( )。

A．基因突變，

―→RNA―→蛋白質

B．基因工程，―→tRNA―→蛋白質

C．細胞工程，―→RNA―→蛋白質

D．基因重組，―→RNA―→蛋白質

解析　轉基因動物是將特定基因轉入某動物體內並使其表達，產生相應性狀。

其表達的過程即為轉錄和翻譯。

答案　D

2．轉基因耐貯藏番茄的作用機理是 ( )。

A．抑制生長素的合成

B．抑制代謝，代謝減弱

C．抑制細胞呼吸

D．封閉乙烯合成途徑中關鍵酶的基因

解析　乙烯是一種與生物成熟有關的植物激素。生物耐貯藏即不令它成熟，故使乙烯合成途徑中的關鍵酶基因封閉，不合成乙烯即可。

答案　D

3．科學家將一段控制某藥物蛋白合成的基因轉移到雞胚胎細胞的DNA中，發育後的雌雞就能產出含該藥物蛋白的雞蛋，這些雞蛋的蛋清中都含有大量的藥物蛋白；而且這些雞蛋孵出的雞，仍能產出含該藥物蛋白的雞蛋。以下描述不正確的是 ( )。

A．這些雞是轉基因工程的產物

B．這種變異屬於可遺傳的變異

C．該過程運用了胚胎移植技術

D．該技術可定向改造生物

解析　這些雞因為具有了外源基因，所以可以稱為轉基因工程的產物；由於外源基因是導入到雞胚胎細胞中的，所以這種變異是可遺傳的變異；基因工程最大的優勢就是可以定向地改變生物的性狀；但是雞屬於鳥類，是卵生動物，所以不經過胚胎移植技術。故選C。

答案　C

4．要獲得一個雜種植株，下列步驟正確的是 ( )。

A．分離原生質體―→誘導原生質體融合―→組織培養―→雜種植株

B．分離原生質體―→原生質體融合―→雜種植株

C．獲取細胞―→原生質體融合―→組織培養―→雜種植株

D．獲取細胞―→原生質體直接融合―→組織培養

解析　植物細胞融合前需先用酶解法去除細胞壁獲得原生質體，再誘導原生質體融合，再生出細胞壁，獲得雜種細胞，經植物組織培養即可獲得雜種植株。

答案　A

5．為獲得性狀優良的作物新品種，下列問題不是體細胞雜交時必須考慮的是

( )。

A．親本細胞間的生殖隔離

B．雜種細胞愈傷組織的誘導和再分化

C．親本細胞融合方法

D．選擇具有優良性狀的親本類型

解析　植物細胞雜交可以克服遠緣雜交不親和的障礙，實現種間的基因交流。因此，兩個物種的植物雜交，傳統有性雜交雖然不可能進行，但植物細胞雜交卻可以實現。故選A。

答案　A

6．乳腺生物反應器可用於 ( )。

A．動物抗病育種 B．生產藥用蛋白

C．治療人類疾病 D．分泌大量乳汁

解析　乳腺生物反應器是讓某些特定基因在哺乳動物體內表達，使這些哺乳動物的乳汁中含有人類所需要的物質，如激素、抗體和酶類。

答案　B

二、非選擇題(共2小題，共26分)

7．(14分)下圖為植物體細胞雜交過程示意圖。據圖回答：

(1)步驟①最常用的方法是________，目的是得到________。

(2)步驟③一般常用的化學試劑是________。

(3)在利用雜種細胞培育成為雜種植株的過程中，運用的技術手段是________，其中步驟④是________過程，步驟⑤是________過程。

(4)若遠源雜交親本A和B都是二倍體，則雜種植株為________倍體。

(5)從理論上講，通過此方法得到的雜種植株的育性為________。若運用傳統有性雜交方法能否實現此方法達到的結果？________。說明理由____________________________。

解析　植物細胞融合前需要先去掉細胞壁，人工誘導原生質體間融合的基本方法可以分為物理法和化學法，化學法一般是用聚乙二醇(PEG)。體細胞雜交成功，雜種體細胞遺傳物質加倍，傳統雜交育種種間存在生殖隔離。

答案　(1)酶解法　原生質體　(2)聚乙二醇(PEG)

(3)植物組織培養　脫分化　再分化　(4)四　(5)可育　不能　因為不同種生物之間存在生殖隔離

8．(12分)閱讀下列資料，回答有關問題。

通過基因改良，科學家培育出保鮮時間更長的番茄，不受除草劑影響的農作物和生長更快的魚。如今，實驗室正在培養基因工程昆蟲。

這種生物技術昆蟲——含有水母基因的粉紅色棉鈴蟲蛾的首次實地放飛計劃將要進行。水母基因使這種幼蟲能夠發出熒光，使科學家更容易跟蹤及監視其行為。

科學家們已做好下一步的準備：對轉基因棉鈴蟲蛾進行放飛實驗。這種蛾被棉農稱為"終結者"，它沒有繁殖能力，但仍具有與野生昆蟲相同的交配能力。目的是讓它與野生昆蟲交配，使野生蛾無法繁殖。

(1)水母基因導入棉鈴蟲蛾體內後，其表達效果是________。在此性狀表達過程中，實質上是有相應蛋白質的合成，這一事實表明不同生物共用一套________，這為________提供了又一證據。

(2)在培育轉基因蛾的基本操作中，所用基因的"剪刀"是________，基因的"針線"是________，基因的"運載工具"是________。

(3)與雜交育種、誘變育種相比，通過基因工程來培育新品種的主要優點是________和________。

解析　利用基因工程的方法把水母的基因拼接到運載體上，導入棉鈴蟲蛾的體內，使棉鈴蟲能發光，便於跟蹤。而且使棉鈴蟲不能繁殖，但能交配，使後代不能繁殖。雜交育種、誘變育種不能定向改變生物的遺傳性狀，而且性狀穩定慢。基因工程能定向改變生物的遺傳性狀，使性狀穩定快。

答案　(1)使棉鈴蟲蛾的幼蟲能夠發生熒光，失去繁殖能力　遺傳密碼　生物進化　(2)限制性核酸內切酶　DNA連接酶　載體　(3)定向　穩定遺傳
