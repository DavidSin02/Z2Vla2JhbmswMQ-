西部地區將會鼓勵哪些化工項目投資，名單出爐

2020-10-29
| 科學

農信傳媒

8月12日，國家發改委會同有關部門對《西部地區鼓勵類產業目錄（2014年本）》進行了修訂，形成了《西部地區鼓勵類產業目錄（2020年本，徵求意見稿）》，現向社會公開徵求意見，時間為2020年8月12日至2020年9月11日......

西部地區鼓勵類產業由兩部分組成：

一是國家現有產業目錄中的鼓勵類產業，包括《產業結構調整指導目錄》和《鼓勵外商投資產業目錄》(均按最新修訂版本執行)

二是西部地區新增鼓勵類產業，在《西部地區鼓勵類產業目錄》中按省、自治區、直轄市分列確定(如所列產業被國家相關產業目錄明確為限制、淘汰、禁止等類型產業，其鼓勵類屬性自然免除)。

西部地區新增鼓勵類化工產業如下：

重慶市

1. 化工新材料生產(《產業結構調整指導目錄》限制類、淘汰類項目除外)

2. 新型化合物藥物或活性成分藥物、高附加值製劑產品的原料藥及其關鍵中間體開發和生產

四川省

1. 核級石墨開發及生產

2. 4萬噸/年以上炭黑新工藝開發及應用

3. 3000噸/年以上氧化釩清潔生產技術開發及應用（廢水、廢渣零排放）;釩製品先進製造技術開發及應用（釩基合金、釩基功能材料、釩精細化工產品）

4. 石墨烯和納米碳材料、細結構石墨、生物炭、鋰電池負極等新型碳材料的開發及生產

5. 生物乙醇制乙烯（以糧食為原料的除外）

6. 高性能稀土永磁、發光、催化、合金等稀土功能材料及器件的開發、生產及應用

貴州省

1.“磷—電—化”一體化資源綜合利用(《產業結構調整指導目錄》限制類、淘汰類項目除外)

2. 重晶石精深加工新產品開發及生產

3. 白雲石精細加工及綜合開發利用

4. 己二酸與尼龍 66 生產

5. 天然植物精細化工產品的開發（香精、香料、化學藥、化工產品中間體）

6. 膜用新材料研發製造及膜元件自動化生產

7. 煤炭地下氣化開採技術開發與應用，煤炭地下氣化發電成套設備製造

8. 氫加工製造、氫能燃料電池製造、輸氫管道、加氫站等涉氫產業

雲南省

1. 石油精細化工產品開發及生產(《產業結構調整指導目錄》中限制類、淘汰類項目除外)

2. 水溶性肥料、微量元素肥料、微生物肥料等新型肥料的開發及生產

3. 生物乙醇制乙烯（以糧食為原料的除外）

4. 生產天然橡膠深加工產品如淺色標準膠、子午線輪胎專用膠、恆粘膠、高等級輪胎專用膠、脫蛋白橡膠系列產品，鼓勵建設大型子午線輪胎生產線，進行特種專用天然橡膠或膠乳產品的研製與開發

5. 雲南滇中鋰資源開發利用及產業鏈發展

西藏自治區

1. 鹽湖資源綜合開發利用(《產業結構調整指導目錄》限制類、淘汰類項目除外)

陝西省

1. 大型煉油、乙烯、芳烴生產裝置生產的有機化工原料就地深加工(《產業結構調整指導目錄》限制類、淘汰類項目除外)，新型、環保型油田化學品（二氧化碳及新型試劑驅油驅氣）技術開發及生產，均聚甲醛、6 萬噸/年及以上二甲基甲酰胺、6 萬噸/年及以上聚乙烯醇等精細化工產品生產

2. 生物法、化學法生產可降解材料，包括聚乳酸、丙內酯基可降解材料、聚丁二酸丁二醇酯、聚羥基烷酸等，以及可降解高分子材料與澱粉共混的環境友好材料，新型炭質吸附材料，生物化學品等

3. 百萬噸級大型乙烯、芳烴、千萬噸級大型煉油等重大煤化工、石油化工、電站用裝備關鍵用泵、控制閥、調節閥的研發及製造

4. 天然氣調峰工廠、地下儲氣庫及應急儲氣調峰設施

5. 化工新材料、高端專用化學品的製造及深加工

6. 進行煤炭分質利用，生產烯烴系、芳烴系及烯烴、芳烴結合產品

甘肅省

1. 利用天然氣替代化工及有色金屬材料深加工中使用的重油、柴油、煤技術開發及應用（天然氣產業政策限制、禁止類項目除外）

2. 電子級氟化氫（HF）生產(《產業結構調整指導目錄》限制類、淘汰類項目除外)

3. 石墨烯和納米碳材料、細結構石墨、生物炭、鋰電池負極等新型碳材料的開發及生產

4. 粉煤灰生產新型油田固井減輕劑

5. 高爐解毒鉻浸出殘渣技術開發及應用

青海省

1. 鉀、鈉、鎂、鋰、硼、鍶、溴、碘、銣、銫等鹽湖資源綜合利用、系列產品開發及副產物利用(《產業結構調整指導目錄》限制類、淘汰類項目除外)

2. 稀鹽酸脫析制氯化氫

3. 六水氯化鎂連續脫水制無水氯化鎂

4. 以各類無機鹽產品生產過程中所產生的固體廢棄物、副產物為主的循環利用

5. 地下深層承壓滷水資源勘探、研究及開發

寧夏回族自治區

1. 20萬噸/年及以上醋酸乙烯、6萬噸/年以上聚乙烯醇、10萬噸/年以上聚碳酸酯（PC）、6萬噸/年以上環氧乙烷、6萬噸/年以上聚甲醛、雙加壓法硝酸生產工藝、電子級高純化學品生產，單套裝置5萬噸以上使用迴轉窯的石灰氮、使用液下投料低溫循環分離法萬噸以上單氰胺、2萬噸以上雙氰胺生產(《產業結構調整指導目錄》限制類、淘汰類項目除外)

2. 大型煤化工、石油化工、火電核電站、油氣儲運、海洋工程等領域用耐高壓、耐高低溫、低噪音等高端智能控制閥和智能定位器的研發及製造

3. 煤炭氣化及尾氣綜合利用、液化合成等潔淨煤技術開發及設備製造

4. 生物發酵、化學合成氨基酸及相關衍生物的開發及生產

新疆維吾爾自治區（含新疆生產建設兵團）

1.大型煉油、乙烯、芳烴生產裝置生產的有機化工原料就地深加工（《產業結構調整指導目錄》限制類、淘汰類項目除外）

2. 節水器材、種子加工、肥料、高效低毒農藥、農牧機械加工組裝等農業資料和農用化工產品製造

3. 高壓法三聚氰胺技術及其下游產品的開發和應用

4. 30萬噸及以上羰基法合成醋酸技術及其下游產品的研發及應用

5. 碳酸二甲酯（DMC）開發與生產

6. 聚己二酸對苯二甲酸丁二醇酯（PBAT）生物可降解聚合物的生產及其可降解塑料製品、農用地膜的研發及應用

7. 1萬噸/年及以上甲硫醇硫化法生產二甲基二硫及其深加工產業

8. 煤制聚甲醛及煤經甲醇制烯烴(《產業結構調整指導目錄》限制類、淘汰類項目除外)

內蒙古自治區

1. 有機—無機復混肥料生產

2. 6萬噸/年及以上聚甲醛、6 萬噸/年及以上二甲基甲酰胺、6萬噸/年及以上聚乙烯醇等精細化工產品生產

3. 15 萬噸/年及以上單套無水煤焦油深加工

4. 無汞催化劑聚氯乙烯生產技術開發及應用

5. 六氟乙烷等精細氟化工產品生產（《產業結構調整指導目錄》限制類、淘汰類項目除外）

6. 石墨烯和納米碳材料、細結構石墨、生物炭、鋰電池負極等新型碳材料開發及生產

7. 新建 120 萬噸/年及以上天然鹼綜合開發利用項目

廣西壯族自治區

1. 生物乙醇制乙烯（以糧食為原料的除外）

2. 精細化工產品開發及生產（《產業結構調整指導目錄》中限制類、淘汰類項目除外）

3. 以錳礦為原料生產高純硫酸錳、碳酸錳、四氧化三錳以及電解金屬錳為原料的電器元件、無汞鹼錳電池、鋰錳電池、不鏽鋼製品的先進生產技術開發及生產

4. 乙烯、丙烯、烷烴等有機化工原料生產及下游精深加工；鹽化工（《產業結構調整指導目錄》限制類、淘汰類項目除外）

5. 超細重鈣、改性重鈣、超細輕鈣、改性輕鈣、納米鈣等高端碳酸鈣生產、下游精深加工及配套產業（《產業結構調整指導目錄》限制類、淘汰類項目除外）

6. 納米碳材料、細結構石墨、生物炭、鋰電池負極等新型碳材料開發及生產
