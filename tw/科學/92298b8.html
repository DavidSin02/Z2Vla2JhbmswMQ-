春來“堇”色秀-江南8種紫堇辨識

2020-10-29
| 科學

東風吹拂，次第花開。

春天的大自然，一派欣欣向榮景象。

在江南野花之中，罌粟科紫堇屬的小仙女，恐怕是草花植物中最引人注目的焦點之一。

它們數量眾多，氣勢頗盛，常常成片開放，像是給大地鋪上了一層美麗的花地毯。

它們顏色豐富，紫色、粉色、玫瑰紅、黃色以及各種過渡色，給人以賞心悅目之感。

它們造型奇特，多數具有長長的花距，或似一隻只自在飛翔的小鳥，又如一隻只昂首高歌的小天鵝，是山林泉野裡的花之精靈。

不過，紫堇好看花難辨。它們有的長相非常類似，差別非常細微，就像人家家裡的雙胞胎，如果不是他們的親人，要想分清大大或者小小，還是比較困難的，因為無從下手。

本篇以華東地區常見的八種紫堇為標本，通過實地觀察和爬梳檢索表，對它們進行詳細比對和一一辨析，期待能為花友們野外識花賞花助上一臂之力。紫堇的辨識是個難題，能夠一一遇見這些小仙女，也頗不容易，直到2019年拍到紫堇、延胡索之後，才算8種全部收齊。倘若本文有錯誤之處，還請方家指正。

1、紫堇

Corydalis edulis

罌粟科 Papaveraceae >> 紫堇屬

紫堇屬第一個繞不過去的，是屬長紫堇。這種植物花色淡紫，帶點透明，氣質比較獨特，一眼即可認出。它們喜歡生於城牆、寺院或者老宅的牆縫之中，似乎挺有歷史底蘊。在江浙地區，南京城牆之上的紫堇最為著名，杭州一些古橋、老宅及山野也略有分佈。但在寧波，卻非常稀見，小山找了兩年，也無緣得見，請教關注寧波植物20多年的林海倫老師，他說也只是在鄞州北侖交界之地見過一次。去年春天，遊蘇州木瀆古鎮，偶遇不少正當花季的紫堇，才算一解多年的相思之苦。

2、刻葉紫堇

Corydalis incise

罌粟科 Papaveraceae >> 紫堇屬

刻葉紫堇因葉子的裂片具缺刻狀齒而得名，這是在寧波分佈最廣泛的紫堇。它有個別名是紫花魚燈草，典型的刻葉紫堇，花冠口深紫色，整體看起來像只靈動的小魚，故有是名。刻葉紫堇顏色變化多端，小山曾在野外拍到五六種以上顏色的刻葉紫堇。

3、夏天無

Corydalis decumbens

罌粟科 Papaveraceae >> 紫堇屬

夏天無，是名字取得最好、最具少女特色的一種紫堇，在寧波分佈第二多。夏天無彷彿一句警語，提示大家不負春光及時賞花。因為這些草花植物，要在春天完成開花結子的生命輪迴，夏天一到，其地上部分即消失的無影無蹤。夏天無的顏色比較穩定，一身淡粉，是小姑娘們最喜歡的色彩。夏天無在浙江植物誌之中的名字是伏生紫堇。

4、珠芽尖距紫堇（珠芽地錦苗）

Corydalis sheareri f. bulbillifera

罌粟科 Papaveraceae >> 紫堇屬

它的名字非常精準地說出了其辨識特點，珠芽是指其莖上部葉腋具易脫落的珠芽，尖距是指它們的尾巴又長又尖，掌握了這兩個特色，珠芽尖距紫堇辨識起來就毫無難度了。有些珠芽尖距紫堇的葉子邊緣，常會有暗紫色斑，像是給葉子鑲上了花邊，非常好看。需要提及的是，珠芽尖距紫堇是浙江植物誌的名字，在中國植物誌之中，此物名為珠芽地錦苗，在寧波分佈也比較廣泛，保國寺、福泉山等地就有很多。

5、延胡索

Corydalis yanhusuo

罌粟科 Papaveraceae >> 紫堇屬

浙江道地藥材很多，因質量好、應用範圍廣及療效佳而為歷代醫家所推崇，其中以“浙八味”最為有名，延胡索即為八味之一。延胡索在外形上和夏天無有點相似，夏天無有個別名就是野元胡，但是延胡索的外花瓣更加寬展，而且邊緣具齒，其小葉三深裂，裂片有點像竹柏的葉子。這種植物在寧波野外，和紫堇一樣稀見，去年3月份，根據林海倫老師的指點，在奉化溪口一片丹霞地貌區域，拍下了正當花期的延胡索。

6、黃堇

Corydalis pallida

罌粟科 Papaveraceae >> 紫堇屬

開黃花的紫堇在寧波有三種，辨識起來頗費腦子。不過黃堇還好，“蒴果線形，念珠狀”是一招斃命的絕招，只看念珠有無，其他不論，即可定種。

7、北越紫堇

Corydalis balansae

罌粟科 Papaveraceae >> 紫堇屬

光從圖片來看，最搞不清楚的是北越紫堇和小花黃堇。北越紫堇的特點是外花瓣勺狀，具龍骨狀突起，其上還有不伸達頂端的雞冠狀突起。當蒴果都是線狀長圓形之時，就要去外花瓣上面找雞冠狀突起，如有就是北越紫堇。在浙江植物誌之中，此物名為臺灣黃堇。老勿大老師還提供一個辨識要點，北越紫堇的花枝和葉枝對生，而黃堇的花枝生於葉枝之腋。

8、小花黃堇

Corydalis racemosa

罌粟科 Papaveraceae >> 紫堇屬

和北越紫堇相比，小花黃堇外花瓣不寬展，且無雞冠狀突起，頂端通常近圓，具寬短尖，看起來就像是綠色的小海豚腦袋，如果在外花瓣上看到了這個可愛的小腦袋，即可初步確定。另外，老勿大老師還指出了小花黃堇另外一個需要靠鼻子來辨識的特徵，就是其葉子揉碎了有臭臭的味道，麗水老百姓叫它們“糞缸草”或者“雞汙爛點草”，可見其味道之不招人待見，有興趣的可以體驗一下。
