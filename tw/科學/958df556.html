保險到底保什麼，你知道嗎？

2020-11-14
| 科學

大家好我是樸鬆，這篇主要聊一下人身保險所保的東西，也就是“保險標的(dì)”：人的健康與生命。

保險分兩種，財產險和人身險。財產險保障的是財物的完好，而人身險保障的是人的健康與生命。

前者大家都有具體的認識，財物是有形的、有價的，而對後者的認識就比較模糊，因為人們只有生病的時候才會意識到健康可貴。


看完這篇文章，相信你對人體的結構、系統會有初步認識，從而知道人身險保的到底是什麼。

先上一張圖[1]感受下：這張圖表現人體結構的層次。人體的基本單位是細胞，細胞構成組織，組織構成器官，以此類推，最終構成人體。


說到人的健康與生命，就不得不提醫學。不同於其他學科專注於外在世界，醫學專門探索人體內的世界，因而有獨特的視角和魅力。



保險有兩大重要內容：核保、理賠，其中有很多西醫知識，要了解醫學就必須瞭解形態學。西醫在全世界普及的重要原因之一，就是立足於形態學的基礎。

什麼是形態學？簡單講，就是一個東西，它長什麼樣，長的像什麼，乍一聽有點好笑，一個東西長什麼樣，還能成為一門學問？別說，它還真是一門學問，從公元前300多年，西醫奠基人、古希臘名醫希波克拉底就開始研究和描述心臟的形態，邁出了人類醫學的一大步。


如果你還是覺得“形態學”太抽象，就記住那張層次圖，那就是形態學。

形態學這方面，大家只需先記住人體有八大系統：運動系統、消化系統、呼吸系統、泌尿系統、生殖系統、脈管系統、神經系統、內分泌系統。


這麼多系統，是不是很難記？教你一招助記：

脈管，運動，消化，內分泌，生殖，呼吸，神經，泌尿

簡化：脈動化泌，殖呼神泌

再加個諧音：“脈動”花蜜，直呼神祕

——脈動哪天要是出了蜂蜜口味的，我直呼內行，第一個買來嚐嚐。怎麼樣，是不是很好記了？


八大系統比較抽象，中國人講五臟六腑。五臟我們再熟悉不過了：心肝脾肺腎。心、肝、肺、腎分別是脈管系統、消化系統、呼吸系統和泌尿系統的重要器官。其他器官有機會給大家一一介紹。

回到最初的問題，人身險保的是什麼，或者說保險責任是什麼，就十分清楚了：人體系統在某個層次出現的問題。

舉幾個例子，層次從小到大，這些問題可能發生在分子（基因）層次，造成所謂的“分子病”，最典型的是地中海貧血（ICD-10 code D56.151）；也可能發生在生殖系統的組織層次，比如宮頸原位癌（ICD-10 code D06.901），也可能發生在消化系統的器官層次，比如胃潰瘍（ICD-10 code K25）；還可能發生在泌尿系統層次，比如泌尿繫結石（ICD-10 code N20.901）。

有同學問括號裡的英文是幹嘛的，ICD-10就是保險公司用來審核保險責任的、由世界衛生組織出版的、全球通用的疾病分類手冊（第10版），後面的字母+數字就是具體疾病在這本手冊中的編碼，以後只要談到具體的疾病，我都會附上編碼。至於ICD編碼對於保險的重要性，以後跟大家好好聊聊。

總之，無論問題發生在什麼層面，最終導致健康和醫療費用的損失，甚至死亡，這些即屬於人身險的保障範圍。


當然，光談保險責任不談責任免除是不負責任的，關於責任免除，今後會單獨拿出來講。

人們往往過於關注外界事物，反而對自身缺乏認識，雅典神廟上鐫刻的一句箴言迴響千年：“認識你自己”。



現在知道保險到底保什麼了，以後跟人聊保險，除了說保“生老病死”，你也能從醫學的角度哈拉兩句。

我是樸鬆，點個關注，一起學習更多保險醫學知識
