File Cabinet Pro Mac版(菜單欄文件快捷管理)

2020-10-29
| 科學

File Cabinet Pro 這款文件快捷管理軟件看起來是不是非常不起眼？您可以將它看作是用來做文件臨時存放的平臺。與 Unclutter 相比，她雖然沒有精緻的 UI 和炫酷的交互設計，但功能非常全面，可以真正改變文件的存放位置（File Cabinet 獨有的目錄），並且可容納文本、圖片、視頻、文件夾以及各種格式的文件，軟件的 Menubar 菜單可在一定範圍內跳轉尺寸，從而給了用戶很大餘地去管理桌面的文件（或常用的文件/程序）。



File Cabinet Pro for Mac版官方介紹

File Cabinet Pro是macOS菜單欄的文件管理器。快速導航到文件，而不會混淆桌面。打開，移動，重命名，壓縮，標記，垃圾，複製和粘貼文件，全部來自菜單欄。您可以在File Cabinet Pro中本地打開許多文件類型，或通過啟動其他應用程序打開文件。在圖標視圖中點擊目錄，在列表視圖中鑽取目錄或在列視圖中導航文件樹將非常容易。



File Cabinet Pro for Mac軟件特點

File Cabinet Pro也是一個基於文檔的應用程序，附帶內置的文本編輯器，圖像查看器，PDF查看器和媒體播放器。File Cabinet Pro的某些文件類型可以打開進行編輯/查看，包括：

File Cabinet Pro for Mac(文件管理軟件) 軟件功能

∙創建子目錄。

∙點擊***子目錄並打開文檔。

∙在新標籤中打開目錄。

∙輕鬆標記文件。在File Cabinet Pro中選擇文件，右鍵單擊，然後在上下文菜單中簡單地添加或刪除控件中的文件標籤。

∙在Finder中顯示選定的文件。

∙重命名文件。

∙複製和粘貼文件。

∙顯示隱藏的文件。

∙顯示包裹內容。

∙壓縮文件。

∙垃圾文件。

∙將項目視為圖標，列表，列或封面流視圖。

∙登錄時啟動應用程序。當您登錄到Mac時，您可以自動啟動File Cabinet Pro（可選功能，默認情況下禁用）。

∙在側欄中創建所有您最喜歡的目錄列表。

∙使用內置的輕量級文本編輯器創建和編輯txt，rtf，rtfd和.md文件。

∙從剪貼板上的文本數據創建txt，rtf，rtfd和.md文件。

∙使用File Cabinet Pro的Document Creator Finder擴展名從Finder創建文本文檔（注意：要使用此功能，您必須在“擴展”選項卡下的系統偏好設置中啟用Finder擴展名）。

∙使用內置的輕量級圖像編輯器旋轉圖像，裁切圖像並將濾鏡應用於圖像。

∙使用內置的媒體播放器觀看視頻並播放音頻文件。

∙使用內置的pdf查看器查看pdf文件。

為受支持的可編輯文檔類型恢復文檔版本。

∙設置全局熱鍵顯示和關閉File Cabinet Pro窗口（可選功能，默認情況下禁用）。

∙通過按空格鍵預覽Quicklook中的選定文件。

∙在支持共享擴展程序（如郵件和郵件）的應用程序***享文件。

∙使用“File Cabinet Pro中的查看校驗和”服務驗證文件的SHA256，SHA1或MD5校驗和。
