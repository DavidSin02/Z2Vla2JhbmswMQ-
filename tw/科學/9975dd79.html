核心突破一　給文言材料斷句

——緊抓標誌，巧破難點


古時稱文詞停頓的地方叫作句或讀，而古文在書寫上原本是沒有標點符號的設計的，閱讀時為求語氣的順暢和正確傳達意思，讀書人便會在文章中自行加註記號。像這樣加註句讀的動作，被稱為“斷句”。現代用的標點符號是“句讀”的發展。《三字經》有言：“凡訓蒙，須講究。詳訓詁，明句讀。”意思是，凡是教導剛入學兒童的老師，必須把每個字都講清楚，每句話都要解釋明白，並且使學童讀書時懂得標點斷句。可見斷句是古代啟蒙教育的第一步，是讀書人的一項基本功。古人之所以重視斷句，是因為斷句正確與否，直接影響對文意的理解，如果不懂句讀，往往會造成誤讀、誤解原意。古代有些文獻呈現出不同的解釋，造成不少混亂，就是因為後人在句讀上產生了分歧。

文言文斷句，旨在考查文言文閱讀理解綜合能力，絕非一朝一夕之功。閱讀文言文能力的提高需要在平時的學習中加強課內外朗讀，培養文言文語感，積累古代漢語知識、熟知文言文用詞用句的基本特點及規律，熟悉古代歷史文化常識。不過，首先要掌握斷句的基本方法。

一、據語境，顧前後

斷句正確與否，直接反映了考生對文意理解的正確與否，而句不離段、段不離文，因此要正確斷句，一定要建立在對文章全面理解、透徹分析的基礎上。我們在做文言文斷句題時，首先要對文章內容進行整體理解，通讀全文，熟讀精思；然後運用我們所學過的知識，顧及前後文，弄清寫了什麼內容，想表達什麼意思；最後認真推敲，確定在適當的地方斷句。

知識運用1

　下列對文中畫波浪線部分的斷句，正確的一項是( )

五年，仁贍廉得近臣戚里遣人市竹木秦、隴間，聯巨筏至京師，所過關渡，矯稱制免算；既至厚結有司悉官市之倍收其直仁贍密奏之帝怒以三司副使範旻戶部判官杜載開封府判官屬吏旻、載具伏罔上為市竹木入官。貶旻房州司戶，載均州司戶。

(節選自《宋史·王仁贍傳》，有刪改)

B．既至/厚結有司/悉官市之/倍收其直/仁贍密奏之/帝怒/以三司副使範旻/戶部判官杜載/開封府判官屬吏/

C．既至/厚結有司/悉官市之倍/收其直/仁贍密奏之帝/怒/以三司副使範旻/戶部判官杜載/開封府判官屬吏/

D．既至/厚結有司/悉官市之/倍收其直/仁贍密奏之帝/怒/以三司副使範旻/戶部判官杜載/開封府判官屬吏/

答案　B

解析　要選準答案，首先要看畫波浪線部分的前後文字。前文敘述了事件發生的背景，後文是事件的結果，畫線文字是事件的經過。其次，看標誌：一是“仁贍”“帝”“範旻”“杜載”等人名或專用名詞；二是“悉”“倍”“以”等虛詞，如“倍”修飾動詞“收”，其後不能斷開；三是句子結構，如“厚結有司”就是一個省略主語的主謂賓句。

參考譯文

太平興國五年，王仁贍查知皇帝身邊的近臣及皇親國戚派人在秦、隴間買竹木，紮成巨筏運至京師，所經關卡渡口，都假稱詔令免稅；竹木運到京師後，重賄有關官吏，全部出售給官府，加倍收其價格。王仁贍密奏太祖，太祖大怒，把三司副使範旻、戶部判官杜載、開封府判官送給獄吏審問。範旻、杜載供認欺騙皇上賣竹木給官府的罪過。貶範旻為房州司戶，杜載為均州司戶。

二、析詞性，明語法

文言斷句，斷開的每個句子基本上都是完整的一個句子。除了倒裝句，古代漢語的語序大致與現代漢語的語序相同，一般是主語在前，謂語在後；謂語在前，賓語在後；定語在前，中心詞在後；狀語在前，謂語在後；謂語在前，補語在後。在給文言文斷句時，首先要弄清每個詞在具體語境中的詞義和詞性，然後弄清詞與詞之間的語法關係。先抓住句子的核心部分——謂語，因為一個句子要獨立成句一般都要有謂語。一般來說，動詞(短語)或形容詞(短語)作謂語。再通過它找主語，查賓語，根據語序逐層理清關係，正確對文言文進行斷句。名詞、代詞一般為陳述、描寫、說明或議論的對象，作句子的主語和賓語，在它們之前或之後往往要進行斷句。在閱讀中要隨時標出文段中的名詞和代詞，如人名、地名、官名、族名、器物名、動物名、植物名、時間名等。常見代詞有：吾、餘(表示“我”)，予、爾、汝(女)、子、公、卿、君、若(表示“你”)，彼、此、其、之(表示“他”)等。

宇文之邵，字公南，漢州綿竹人。舉進士，為文州曲水令。轉運以輕縑高其價，使縣鬻於民。之邵言縣下江上山地狹人貧耕者亡幾方歲儉飢羌夷數入寇不可復困之以求利運使怒。

(節選自《宋史·宇文之邵傳》)

B．之邵言/縣下江上山/地狹人貧/耕者亡幾/方歲儉飢/羌夷數入寇/不可復困之以求利/運使怒

C．之邵言縣下/江上山地狹/人貧耕者亡/幾方歲儉飢/羌夷數入/寇不可復/困之以求/利運使怒

D．之邵言縣下/江上山地狹/人貧耕者亡/幾方歲儉飢/羌夷數入寇/不可復困之/以求利運使怒

答案　B

解析　原文標點：之邵言：“縣下江上山，地狹人貧，耕者亡幾，方歲儉飢，羌夷數入寇，不可復困之以求利。”運使怒。

參考譯文

宇文之邵，字公南，是漢州綿竹人。考中進士後，擔任文州曲水縣令。轉運使把輕縑的價錢抬高，讓縣衙強賣給百姓謀取利益。宇文之邵說：“曲水縣下臨長江，上接高山，地勢狹窄，百姓生活貧困，種地的人沒有多少，正趕上收成不好，外敵多次入侵，不可以再使百姓困苦來謀求私利。”轉運使大怒。

三、識虛詞，辨句式

1．識虛詞

古代漢語中有些虛詞在文句中出現的位置相對比較固定，或在句末，或在句首，或與特定詞語組合在一起，形成固定結構。給文言文斷句時，要用慧眼找出句子中的虛詞，再根據其位置的特定性，從而準確斷句。

虛詞類別

常見典型虛詞及斷句方法

句首發語詞——常居句首，其前一般斷開

夫、蓋、至若、若夫、初、唯、斯、今、凡、且、竊、請、敬

句首時間詞——常居句首，其前一般斷開

頃之、向之、未幾、已而、斯須、既而、俄而

句末語氣詞——其後一般斷開

陳述句末尾——也、矣、焉、耳

疑問句末尾——耶、與(歟)、邪(耶)、乎

感嘆句末尾——哉、夫

疑問語氣詞——其後一般構成疑問句，其前一般斷開

何、胡、安、曷、奚、盍、焉、孰、孰與、何如、奈何

複句關聯詞語——其前一般斷開

雖、雖然、縱、縱使、向使、假使、苟、故、是故、則、然則、或、況、而況、且、若夫、至於、至若、已而、於是、豈、豈非

複音虛詞——複音虛詞需保持完整，不斷點斷

有所、無所、有以、無以、以為、何所、孰若、至於、足以、何以、然則

①“以、於、為、則、而”往往用於句中，在它們的前後一般不斷句。

②“乎”用在句中相當於“於”時，不能點斷。如：“生乎吾前，其聞道也固先乎吾。”

③“也”用在句中起舒緩語氣的作用，可點斷也可不點斷。如：“師道之不傳也久矣！欲人之無惑也難矣！”

④連詞“而”有時用於詞或短語之間起連接作用，所連接的內容共同作句子的成分，前後聯繫很緊密，一般不在其前斷開。但如果“而”是連接兩個句子，則往往要在其前斷開。如：“其下平曠，有泉側出，而記遊者甚眾，所謂前洞也。”

⑤“夫”有時可作句末語氣詞，表感嘆語氣，有時也作指示代詞(兼有舒緩語氣的作用)用在句中，這幾種情況都不能點斷。如：“故為之說，以俟夫觀人風者得焉。”

特殊句式

斷句要求

……者……(判斷句)

“者”後可斷開

為……所……，受……於……，見……於……(被動句)

前後可斷，中間不斷

奈……何，如……何，無乃……乎，不亦……乎，何……之有，何以……為，唯……為……，孰與……乎(固定句式)

知識運用3

　下列對文中畫波浪線部分的斷句，正確的一項是( )

劉弘，字和季，沛國相人也。朝廷以前東平太守夏侯陟為襄陽太守。陟，弘之婿也。弘下教曰：“夫統天下者宜與天下一心化一國者宜與一國為任若必姻親然後可用則荊州十郡安得十女婿然後為政哉乃表“陟姻親，舊制不得相監”。詔聽之。(節選自《晉書·劉弘傳》)

B．夫統天下者/宜與天下一心/化一國者/宜與一國為任/若必姻親然後可用/則荊州十郡/安得十女婿然後為政哉/

C．夫統天下者/宜與天下一心/化一國者/宜與一國為任/若必姻親然後可用/則荊州十郡/安得十女婿/然後為政哉/

D．夫統天下者/宜與天下一/心化一國者/宜與一國為任/若必姻親然後可用/則荊州十郡/安得十女婿/然後為政哉/

答案　C

參考譯文

劉弘，字和季，是沛國相縣人。朝廷讓前東平太守夏侯陟任襄陽太守。夏侯陟，是劉弘的女婿。劉弘對下屬說：“統率天下的人，應和天下人一條心，教化一國的人，應以一國為己任。要是非要任用自己的親族，那麼荊州有十郡，非得有十個女婿才能管理好嗎？”於是上表說“夏侯陟是我的姻親，按制度不能互相監督”。朝廷下詔予以批准。

虛詞斷句口訣

“曰”後冒(：)，“哉”後嘆(！)，“夫”“蓋”大都在句前。“於”“而”一般在句中，“矣”“耳”後面畫圈圈(。)，“邪”“乎”經常表疑問(？)。“也”“者”作用要停頓，或句(。)或逗(，)表情形，各自位置要記牢。

四、據修辭，巧判斷

古人注重文氣，講究語句整齊對仗、音韻和諧、意思對舉，對偶、排比、頂真、反覆是文言文常見的修辭手法。古文中兩個相同詞語連用，如果不屬於形容詞、名詞的重疊形式，一般來說，它們分屬兩句話，應當從中間斷開。如《愚公移山》：“汝心之固，固不可徹，曾不若孀妻弱子。雖我之死，有子存焉；子又生孫，孫又生子，子又有子，子又有孫，子子孫孫，無窮匱也。”把握古人用句的基本規律，利用結構的對稱性推斷詞性及短語結構，能比較容易找到斷點。

知識運用4

　下列對文中畫波浪線部分的斷句，正確的一項是( )

劉馥子靖，遷尚書，賜爵關內侯，出為河南尹。散騎常侍應璩書與靖曰：“入作納言，出臨京任。富民之術日引月長藩落高峻絕穿窬之心五種別出遠水火之災農器必具無失時之闕蠶麥有苫備之用無雨溼之虞封符指期，無流連之吏。鰥寡孤獨，蒙廩振之實。加之以明擿幽微，重之以秉憲不撓；有司供承王命，百里垂拱仰辦。雖昔趙、張、三王之治，未足以方也。”靖為政類如此。(摘編自《三國志·魏書·劉司馬樑張溫賈傳》)

B．富民之術/日引月長/藩落高峻/絕穿窬之心/五種別出/遠水火之災/農器必具/無失時之闕/蠶麥有苫備之用/無雨溼之虞/

C．富民之術日引/月長藩落高峻/絕穿窬之心五種/別出遠水火之災農器/必具無失時之闕蠶麥/有苫備之用/無雨溼之虞/

D．富民之術日引/月長藩落高峻/絕穿窬之心五種/別出遠水火之災/農器必具無失時之闕/蠶麥有苫備之用/無雨溼之虞/

答案　B

解析　根據句式的對稱可知，“絕穿窬之心”“遠水火之災”“無失時之闕”“有苫備之用”“無雨溼之虞”有對偶在裡面。從句子來看，“富民之術”“藩落”“五種”“農器”“蠶麥”作主語，在它們前面應該斷句。

參考譯文

劉馥的兒子劉靖，升為尚書，朝廷賜予他關內侯的爵位，又出任河南尹。散騎常侍應璩給劉靖寫信說：“你由外邊來擔任納言的職位，在京城做官。使百姓富足的政策，在於日積月累，循序漸進。院牆籬笆要造得又高又陡，來杜絕那些偷盜的企圖。播種的時候，一定要選好搭配黍、稷、麻、麥、豆五類種子，以避免水旱災害。農具一定要完備充足，不能耽誤農時。養蠶、種麥要有苫蓋的準備，才能保證不受雨淋。(吏員)要按期升遷，不要讓他們滯留原位。鰥寡孤獨的人，讓他們享受到官倉的賑濟。再加上明察秋毫，執法如山，不為權勢所屈；各級官吏遵奉朝廷的旨意，舉國上下都能垂手聽命。那麼，即使是前朝治理京兆的能臣趙廣漢、張敞、王尊、王章、王駿，也不能同您相比。”而劉靖施政大多做到了(應璩所說的)那樣。

五、熟典章，諳文常

古代的很多專用名詞，如官名、地名、族名、書名、物名、制度名、年號、諡號、帝號、時間等，還有天文、地理、科舉、宗法、稱謂、避諱等古代文化常識，對於今人來說是比較陌生的，這些也成為考生閱讀文言文的重大障礙之一，是文言文斷句的一個難點。這就要求考生平時注意熟悉和積累古代文化常識，在斷句的時候，如遇到這類專用名詞，應特別留心。

D．還/糾察在京刑獄/以兵部郎中知制誥/大雅初名侃/至是/避真宗藩邸諱/詔改之居/二歲拜右諫議大夫/集賢院學士/

答案　C

解析　史傳類文章中，常會有傳主“以……(官職)知……”的句式，“避……諱”也是固定說法，據此可排除A、B兩項。對比C、D兩項，區別在於“詔改之居二歲”的斷法，“詔改之”的動作行為的發出者是皇帝，“居二歲”作狀語，二者中間應斷開，排除D項。故選C。

參考譯文

(楊大雅)調任常州知州，兼任三司都磨勘司、戶部勾院。後升任集賢殿修撰、應天府知府。還朝後，任職於糾察在京刑獄，以兵部郎中的身份掌管制誥。楊大雅原名楊侃，到這時，為了迴避真宗在藩王第宅時的名諱，皇上下詔讓他改掉名字。任職二年，被授予右諫議大夫、集賢院學士、亳州知州等官職，後去世。

靈活使用標誌法斷句

上面介紹的五種方法，大都是標誌斷句法。使用此法既快又準，很好使用，但不可絕對使用，而要靈活使用。因為斷句斷句，斷的是意，關鍵是明確文意，“標誌”只是路標而已，況且不少“標誌”也很靈活。像“且夫”“蓋”這類句首虛詞，“矣”“哉”“耶”這類句尾虛詞，標誌基本沒有問題。但像下列虛詞相對靈活，必須輔以語意理解。

(1)“焉”“者”“而”“為”“爾”“乎”“也”這類虛詞，有的句首、句中皆可用，有的句首、句尾皆可用，有的句中、句尾皆可用，必須具體問題具體分析，依據具體文意、語氣來判斷。

(2)對話標誌詞如“雲”“謂”“曰”“言”一般情況下可以斷開，但也有例外。“曰”有兩種情況，一是表“叫作”的意思。例如明代魏學洢《核舟記》：“明有奇巧人曰王叔遠。”二是在句首、句中作助詞，無實義。例如《詩經·小雅·采薇》：“曰歸曰歸，歲亦莫止。”

“雲”有兩種情況，一是表“有”的意思。例如《荀子·法行》：“其雲益乎？”二是表“為、是”的意思。例如《後漢書》：“雖雲匹夫，霸王可也。”

“言”有七種情況，或表七種意思時，後面不可斷開：①談論；②記載；③告訴；④問；⑤陳述；⑥說明；⑦解釋引文、詞語或某種現象的發端詞，相當於“就是說”或者“意思是”。

另外，像固定句型也不全是固定的。有的固定句型恰恰是前後兩個小句子，中間要斷開。①……者，……也；②與其……，孰若……；③……，抑……；④其……，其……也；⑤非惟……，抑亦……。

全國卷斷句題採用的是“四選一”式選擇題，設題時一般先預設考生斷的整段文字中的兩處斷句錯誤，有兩個選項各安排一處錯誤，一個選項安排兩處錯誤，另一個選項無錯誤。對於這種較穩定成熟的題型，突破之道是：

1．排“同”聚“異”

四個選項中，先排查出來哪些停頓是相同的，對此不必關注；再聚焦停頓不同的，重點思考。

2．突破難點

那些停頓不一樣的地方，可能是難點，且難點就是那麼一兩處。對此，要有足夠的突破方法。先看看能否找到標誌，如名詞、虛詞、句式的標誌，能找到這些“路標”固然好，但這類題中真正的難點往往不是靠尋找標誌就能應付的，關鍵是揣摩語意，看看句子的陳述對象是誰，句子之間、詞語之間有什麼樣的意義關聯，每個詞語是什麼意思，這其中如有小技巧的話，就是會抓謂語動詞。

在文言文中，動詞是常見的。它的主要功能是充當謂語，而謂語又是構成句子的一種主要成分。因此，可以用抓動詞找主幹的方法來斷句。斷句時，可在瞭解文意的基礎上，先找出動詞謂語，然後前追主語，後問賓語、補語，即可確定句子的標點。如果句子裡出現了幾個動詞，就應根據具體語言環境，先弄清動詞間的關係，再找主幹。如它們是個古今同形異義詞，當斷開。如2018年全國卷Ⅱ斷句題中“敦儒學習《尚書》”中的“學習”是個古今同形異義詞，當斷開。如兩個動詞是連動式，則不需要斷開。如“於是相如前進缶”中的“前進”就是連動式，意謂“走上前進獻”，不可斷開。也有少數句子沒有動詞謂語，而是名詞、形容詞、數詞活用作動詞謂語的，斷句時要特別注意。最後，排除選定。

下面以2018年全國卷Ⅱ第10題為例。

C．王渙字稚子/廣漢郪人也/父順/安定太守/渙少/好俠尚氣力/數通剽輕少年/晚而改節敦/儒學習《尚書》/讀律令/略舉大義/

D．王渙字稚子/廣漢郪人也/父順/安定太守/渙少/好俠尚氣力/數通剽輕少年/晚而改節/敦儒學/習《尚書》/讀律令/略舉大義/

第一步，排“同”聚“異”。整個句子中，前半句“王渙字稚子/廣漢郪人也/父順/安定太守”相同，後半句中“《尚書》/讀律令/略舉大義”相同，這些可以忽略了。重點看句子中間的不同。不同之處有三：一是“少”與“好”的連與斷，二是“節”與“敦”的連與斷，三是“學”與“習”的連與斷。

第二步，突破難點。先看第二處“異”。憑語感可知“節”與“敦”兩個動詞不能連，於是排除掉A、C兩項。再看“少”與“好”的連與斷。“少”是指王渙年少的時候，這句話未說完，不該斷。於是排除掉C、D兩項。最後看“學”與“習”的連與斷。“敦”與“習”是兩個動詞，“儒學”與“《尚書》”分別是它們的賓語，且結構相同，“學”與“習”兩個看似同形，實則一為名詞一為動詞，就可以分開了。於是排除掉A、C兩項。綜合分析，B項是正確的。
