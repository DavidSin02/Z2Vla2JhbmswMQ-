一、方程組

f (x)含三角函數、指數函數、或其他超越函數時，就是超越方程。

二、點迭代的步驟與問題

可以通過函數圖像來確定函數實根的個數。

迭代步驟：

方 程 ： f (x) = 0

構造迭代函數：x = jФ (x) 經過簡單變形產生迭代序列：xn+1 = jФj (xn)，n =0，1，… 給定迭代初值 x0

思考問題：2個

1.迭代表達式x = jФ (x)是否唯一？

2.迭代產生的序列是否一定會收斂？

三、點迭代舉例-函數構造

例：用點迭代方法求解方程 x3 -x2 -x-1 = 0

解： 第一步 構造迭代函數： x=jФ (x)

迭代舉例-Matlab實現程序第二 / 三步 迭代+初始值設定初值 x0=1，xn+1 = jФ (xn)，n =0，1，… 用 MATLAB 編程 x(1)=1;y(1)=1;z(1)=1; %初始點for k=1:20 x(k+1)=x(k)^3-x(k)^2-1; % j 1 (x) y(k+1)=(y(k)^2+y(k)+1)^(1/3); % j 2 (y) z(k+1)=1+1/z(k)+1/z(k)^2; % j 3 (z)endx,y,z


四、加速迭代函數

加速迭代收斂

若 x= jФ (x) 迭代不收斂，則不直接使用j (x)迭代，

而用由jФ (x)與x的加權平均：

h(x) = lλ Ф j (x)+(1- λ )l x
