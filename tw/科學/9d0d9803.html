「心血管專題」變異型心絞痛發作時心電圖酷似deWinterST-T改變1例

2020-11-14
| 科學

字數1667 · 閱讀需要8分鐘 · 歡迎在文末留言

何蓉 李高森

陸軍軍醫大學第二附屬醫院內分泌科

//

1 病例簡介

//

患者男性，71歲，以“胸悶、心悸”入院，查體未見明顯異常。既往高血壓病史9年。入院後行心電圖，各項常規實驗室指標均未見明顯異常。超聲心動圖示：(1)心臟各腔室大小正常，主動脈升部增寬；(2)室間隔增厚；(3)主動脈瓣增厚並鈣化，輕度狹窄；(4)左室舒張功能減退。診斷為高血壓3級(極高危)、冠狀動脈粥樣硬化性心臟病。患者入院第2天上午突發胸悶、心悸，予硝酸甘油含化服，臥床休息，並急查心電圖顯示竇性心動過緩；ST-T顯著改變(見圖1)。15min後患者訴症狀緩解，予複查心電圖示竇性心律，ST-T改變(見圖2)。於次日行冠狀動脈造影，見左主幹(LM)末段狹窄約50%，左前降支(LAD)開口狹窄約40%，近中段瀰漫性斑塊、重度鈣化並狹窄約50%～85%；迴旋支(LCX)近段斑塊並狹窄約50%，遠段瀰漫性斑塊、鈣化並狹窄約60%～80%；右冠狀動脈(RCA)全程瀰漫性鈣化影，近中段侷限性狹窄約40%，遠段近“後三叉”處侷限性狹窄約85%(見圖3)。患者擇期行經皮冠狀動脈腔內血管成形術(PTCA)、支架置入術後，好轉出院。

圖1為患者胸悶、心悸發作時的心電圖，診斷為竇性心動過緩，ST-T顯著改變。特點為P波規律出現，P-R間期固定，Ⅱ、Ⅲ、aVF導聯R波較圖2增高，ST段呈下斜型抬高，ST-T幾乎完全與R波降支融合成一斜線，類似“巨R波”，Ⅱ導聯R波降支可見一小切跡，Ⅱ導聯ST抬高16 mm，Ⅲ導聯ST 抬高18mm；V1～V5導聯r波遞增，S波遞減，J點下移，ST段呈上斜型壓低6～14mm，並與直立、對稱的T波相延續；V6導聯ST段水平抬高2.5mm，T波倒置；Ⅰ、aVL、aVR呈上斜型分別壓低1.5mm、9mm、5.5mm，T波直立。15min後複查心電圖特點為Ⅱ、Ⅲ、aVF導聯R 波較前降低，ST 水平抬高0.5～1mm，T波淺倒置，其餘導聯未見特殊改變。

//

2 討論

//

1993年Madias首次提出巨R波型ST 段抬高。巨R波提示較大冠狀動脈急性閉塞或痙攣，引起大面積急性缺血早期的心肌損傷及損傷阻滯。其機制為缺血致心肌缺氧，使心肌細胞無氧代謝增加，堆積的酸性代謝產物使細胞膜功能受損，受損的細胞膜對K＋的通透性增加，使細胞內K＋外流，細胞內、外K＋濃度比值下降，靜息膜電位減小，0相除極Na＋通道開放速度和幅度減小，傳導減慢，但進一步缺血會導致心肌失去傳導能力，或缺血改善，心肌細胞恢復傳導能力，該心電圖表現會隨之消失。

V1～V5導聯J點下移，ST 段呈上斜型壓低6～14mm，並與直立、對稱的T 波相延續，這種改變與deWinterST-T相似。deWinterST-T的心電圖特徵是V1～V6導聯J點下移，ST段呈上斜型壓低≥1mm，並與直立高尖對稱的T 波相延續；QRS波群寬度正常或輕度增寬；某些患者可見胸前導聯的R波遞增不良；絕大多數患者aVR導聯ST段抬高1～2mm；這種心電圖改變不發展為ST段抬高，而是維持到冠脈介入治療後；且常見於年輕男性患者。提示LAD近段急性閉塞或次全閉塞。

因此，我們根據圖1下壁導聯或者胸前導聯ST-T的分析將罪犯血管分別指向了RCA(LCX待排)與LAD，但是結合12導聯心電圖整體ST-T改變的特點(即缺血損傷區的導聯ST段明顯抬高，背離損傷區的導聯ST段壓低)，Ⅱ、Ⅲ、aVF導聯呈巨R波提示缺血損傷區在下壁，則背離損傷區(Ⅰ、aVL導聯)ST段壓低；同理V1～V5導聯ST段壓低且程度與Ⅱ、Ⅲ、aVF 導聯ST段抬高程度相當，推測後壁發生透壁性缺血。另外，L導聯諸多ST段特徵都提示下壁及後壁心肌發生透壁性缺血。

我們推測此次“胸悶、心悸”的犯罪血管為RCA。造影顯示LAD、LCX、RCA 及LM 均有不同程度病變，且呈RCA優勢型，RCA病變位置以下的血管分佈與心電圖缺血區域相符。

在臨床工作中，應根據P-QRS-T改變的機制及動態變化對同步心電圖各導聯整體綜合分析，為診治提供更精準的依據。

來源｜醫學空間戰略合作夥伴《國際心血管病雜誌》2019年第46卷第3期，轉載請標明出處。

END
