解決由無線干擾引起的 Wi-Fi 和藍牙問題

2020-11-14
| 科學

無線干擾可能會導致 Wi-Fi 和藍牙設備斷開連接或性能欠佳，但您可以採取步驟來降低或消除這樣的干擾。

無線干擾症狀

以下任一症狀都可能是由影響 Wi-Fi 或藍牙信號的干擾所導致的：

  • 設備無法連接或無法保持連接狀態
  • 連接速度較慢以及信號強度較弱
  • 藍牙音頻往前跳躍、時斷時續、中斷，或者有靜電噪聲或嗡嗡聲
  • 指針移動不穩定或出現“跳躍”

如何降低無線干擾

以下常規步驟有助於實現更清晰、更強的無線信號：

  • 將 Wi-Fi 設備靠近 Wi-Fi 路由器。將相互連接的藍牙設備靠近些。
  • 避免在常見的干擾源附近使用無線設備，此類干擾源包括電源線、微波爐、熒光燈、無線攝像機和無繩電話等。
  • 減少使用同一無線頻段並且正在使用中的設備的數量。藍牙和 Wi-Fi 設備都使用 2.4 GHz 頻段，但實際上許多 Wi-Fi 設備也可以使用 5 GHz 頻段。如果您的 Wi-Fi 路由器支持這兩個頻段，則將更多 Wi-Fi 設備連接到 5 GHz 頻段或許會有所幫助。某些雙頻路由器能夠在這方面實現自動管理。
  • 將您的 Wi-Fi 路由器配置為使用另一個 Wi-Fi 頻道，或讓它掃描干擾最小的頻道。大多數路由器都會在啟動或重置時自動執行這項掃描。

如果您將 USB 3 或雷靂 3 設備與您的電腦搭配使用，則您可以限制此類設備對附近無線設備造成干擾的可能性：

  • 將一根具有屏蔽功能的高質量 USB 或雷靂 3 線纜與每臺設備搭配使用。
  • 將您的 USB 3 或雷靂 3 設備（包括任何 USB 集線器）遠離無線設備。
  • 避免將 USB 3 或雷靂 3 設備放在 Mac Pro、Mac 筆記本電腦或 Mac mini 上面。
  • 關閉所有未使用的 USB 3 設備。

避免無線信號路徑中出現實體障礙物。例如，藍牙鼠標與電腦之間的金屬表面可能導致鼠標性能欠佳，而 Wi-Fi 路由器與 Wi-Fi 設備之間由金屬加固的混凝土地面可能會導致 Wi-Fi 性能較差。

  • 潛在干擾程度低：木頭、玻璃和許多合成材料
  • 潛在干擾程度中等：水、磚塊、大理石
  • 潛在干擾程度高：灰泥、混凝土、防彈玻璃
  • 潛在干擾程度極高：金屬
