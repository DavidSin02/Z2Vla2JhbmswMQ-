南昌起義 中共前敵委員會成員和軍、師黨代表都有誰？

2020-10-29
| 科學

​​

南昌起義我黨的主要領導人是 :

南昌起義主要領導人——中國共產黨前敵委員會書記周恩來 ;

南昌起義中國共產黨前敵委員會成員——李立三 ;

​

南昌起義中國共產黨前敵委員會成員——彭湃 ;

​

南昌起義中國共產黨前敵委員會成員——惲代英

​

第九軍黨代表朱克靖 ; 第十一軍黨代表聶榮臻 ; 第二十四師黨代表顏昌頤 ; 第二十五師黨代表李碩勳 ; 第二十軍黨代表廖乾吾 ; 第一師黨代表方維夏 ; 第二師黨代表陳　恭 ; 第三師黨代表徐特立 。

時任參謀團祕書長李一氓還回憶說:

南昌起義，師黨代表還應加陽翰笙（即歐陽繼修）和梅龔彬，當時他們都是師黨代表 ;我南昌起義時當參謀團祕書長（參謀團參謀長是劉伯承）。

​

關於參謀團，周恩來曾回憶說：“參謀團主任不是我，是伯承同志。開始，參謀團沒有人任參謀長，我就指定劉伯承同志做，起初他謙虛不肯答應，後來我說，一定要你做，他才擔任這個職務。”與李一氓同志的回憶是一致的。

李一氓同志(1903年-1990年)，是老一輩無產階級革命家，馬克思主義理論家。他出生於四川省彭州市東大街，1925年加入中國共產黨。早年赴法國勤工儉學，追求革命真理。在國內革命戰爭時期，李一氓同志曾任國民革命軍總政治部宣傳部科長，南昌起義參謀團祕書長。他參加過紅軍二萬五千里長徵，並先後任陝甘寧省委宣傳部長，新四軍祕書長。抗戰勝利後，李一氓同志先後任蘇北區黨委書記，華中分局宣傳部長，大連大學校長等職。李一氓同志是馬克思主義理論家，二、三十年代他翻譯出版了《馬克思與恩格斯合傳》、《馬克思論文選譯》及馬克思的《哲學之貧困》等著作。

​

同時又是中國新文化運動先驅者，他是中國左翼文化界總同盟的負責人之一，1928年3月同陽翰笙共同編輯《流沙》。

​

新中國成立後，曾任中國駐緬甸大使，國務院外事辦副主任，中聯部副部長，中紀委副書記，中顧委常委等。
