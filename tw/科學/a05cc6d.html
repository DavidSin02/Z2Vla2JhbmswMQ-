SolidWorks文件批量轉換_低版轉高版

2020-10-29
| 科學

1、文件轉換

當我們在升級SoildWorks到高版本的時候，其文件的內部結構也隨著更新了，即它可以正確打開以前的低版本文件（向下兼容），但不支持低版本打開高版本的文件（向上兼容）。同時，如果我們的設計文件有大量的低版本文件的話，那在用高版本軟件打開時，SolidWorks將在內存中把文件的結構和版本轉換為當前版本，並在保存時寫入硬盤，如果須打開的低版本文件比較多或打開老版大型裝配體的話，這將大大增加加載時間。為此，SolidWorks提供了SolidWorks Task Scheduler來對文件和任務進行批量操作。

SolidWorks Task Scheduler——它獨立於Solidworks運行，可以按指定的方式在指定時間完成批量任務。

利用任何 SolidWorks 許可，您都可以排定轉換文件和分解文件任務。 要排定大多數其它任務或排定已分發的文件轉換，您必須具有 SolidWorks Professional、SolidWorks Premium 或 SolidWorks Office 許可。

2、文件批量轉換實例

SolidWorks Task Scheduler中的文件轉換隻是一個代替人工重複操作的批量處理工具，本質上還是會通過啟動solidworks依次打開、保存、關閉文件，然後關閉solidworks。所以，這是一個非常原始的方式，減少人工的重複勞動。

步驟1 打開一個低版本文件並關閉所有插件

隨意打開一箇舊版本文件，可在標準工具欄上看到舊版本文件的提示，然後單擊【工具】/【插件】，關閉所有插件以減少加載時間，如下圖所示。

不保存，關閉零件和SolidWorks。

步驟2 打開SolidWorks Task Scheduler

從Windows開始菜單中，單擊【所有程序】/【SolidWorks<版本>】/【SolidWorks工具】/【SolidWorks Task Scheduler】；對於win10可能是單擊【開始】/【SolidWorks<版本>】/【SolidWorks Task Scheduler】，如下圖所示。本例採用2013sp5。

步驟3 打開轉換文件窗口設置轉換選項

在【SolidWorks Task Scheduler】對話框中單擊左上角的【轉換文件】或菜單單擊【任務】/【轉換文件】，然後單擊【添加文件夾】，選擇要轉換文件的目錄和相關選項，如下圖所示。

步驟4 排定任務並轉換

採用當前時間，單擊【轉換】窗口下方的【完成】開始排定任務，如果當前沒有其他任務，將立即開始文件轉換，如下圖所示。

在轉換過程中SolidWorks會以縮小窗口的形式多次打開和關閉，如下圖所示。

注：1、SolidWorks Task Scheduler中的其他任務的設置與文件轉換類似，如任務不能執行，可嘗試重裝軟件或更換系統。

2、關於SolidWorks文件版本間的兼容問題，從2012版開始，SP5版本可打開後續一個主版本的文件，但打開後詳細特徵設計樹，只有一個輸入特徵。如2012sp5可打開2013SP0-SP5，不能打開2014及以後的，兩個主版本必須連續。
