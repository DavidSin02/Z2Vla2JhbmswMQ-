C語言之輔助循環break語句

2020-10-29
| 科學

break語句一般出現在循環的代碼塊中，或者出現在switch(多重選擇)裡面，用於終止循環或選擇，如果是嵌套循環，break語句包含在內循環裡，當執行到break語句的時候，終止的是內循環。

#include <stdio.h>int main(void){    char ch = 'A';   //字符A在   ASCII  碼錶  裡面對應的是65.    int i;    printf("請輸入一個整數，(輸入q退出)\n");    while (scanf("%d", &i) == 1)   //scanf如果成功讀取1個整數，則返回1，否則返回0，'q'不是整數，是字符。    {        while (i < 'A')  //如果i小於65會進入此循環        {            printf("哈哈\n");//打印。。。哈哈            break;      //break會跳出當前循環，去執行打印。。。嘻嘻那個語句            printf("嘿嘿\n");   //這個語句永遠都不會打印出來        }        printf("嘻嘻\n");//執行完這個語句，繼續執行第一循環，等待用戶輸入數字，除非輸入非整數則退出循環    }    return 0;}

把代碼複製下來，自己編譯一下，（看著結果，對比流程在腦裡過一下）應該就懂了。
