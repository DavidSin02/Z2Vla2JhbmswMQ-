八種幾乎絕跡的野生文玩核桃，個個有故事

2020-10-29
| 科學

玩文玩核桃的朋友們，都希望能收藏到一對品相絕佳的野生核桃，在和友人交流的時候還能稍稍顯擺一下。然而野生核桃種類繁多，能被大家熟知的卻並不多，有些小眾品類也幾近絕跡，咱們今天來說的就是基本上已經絕跡的九種核桃，快看看您手裡是否有一對兒？

一、通天塔

品種：楸子 / 產地：黑龍江 / 稱號：通天塔

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：原名粗筋楸子，2013年被毀。

二、金蟾繡花鞋

品種：楸子 / 產地：遼寧丹東 / 稱號：金蟾繡花鞋

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：遼寧省楸子名品稀少，能被野生核桃愛好者認可的，其中之一就是被核桃發燒友所津津樂道的瀋陽雞心。而能被楸子愛好者認可並且奉為上品的僅僅就兩種，其中一款就是丹東的繡花鞋絕品。

2013年遼寧丹東一位楸子愛好者無意間得到此類單隻，當時命名為佛光普照。後經這位丹東核友的尋找，終於收集到了一對，並且發在網上。此款楸子外形兼顧繡花鞋和金蟬的神態，浮雕凸紋，沉水，是一款值得收藏的楸子品種。

三、四座樓

品種：野生麻核桃 / 產地：北京平谷熊耳寨鄉 / 稱號：四座樓

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：四座樓獅子頭老樹生長在一個叫牛峪的深山溝裡，平谷僅此一顆老樹。隨著麻核桃價格一路飆升，據傳被幾戶村民所毀。

四座樓不同於南疆之處就是大氣且周正，這是野生麻核桃較少見的，四座樓屬於百年老樹，進入衰弱期後產量極低，愈發顯得彌足珍貴。隨著這三五年來，嫁接四座樓的大面積種植、上市，四座樓已然成為近幾年麻核桃市場領跌的急先鋒！

四、鳳頭燈籠

品種：燈籠 / 產地：秦嶺 / 稱號：鳳頭燈籠

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：秦嶺垂垂老矣的神樹，2015年是自12年後的再次結果。9月初下樹，被人拾取，數天後知道此樹的一行四人來到樹前，發現果子已經被人拾走，憤然毀樹。

五、螞蟻紋

品種：秋子 / 產地：吉林 / 稱號：螞蟻紋

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：螞蟻紋核桃成名較早，此品種被帶至京城，因為筋脈不明顯，花紋細碎接近傳統的麒麟紋，是首先被麻核桃屆廣泛所認可的楸子品種。

自赴京一炮而紅後，就傳說不斷。一說，螞蟻紋樹為吉林農村一兄弟所有，但是禍起蕭牆，先被砍斷一支，近年又報僅剩的一枝又被砍。據說，此樹還存在發新芽枝的可能性。

六、門墩獅子頭

品種：野生麻核桃 / 產地：不明，據說為天津薊縣 / 稱號：（麒麟紋）門墩獅子頭

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：傳言野生老樹生於天津薊縣，12年後就再少見。

七、龍骨

品種：楸子 / 產地：黑龍江 / 稱號：龍骨

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：據稱產自黑龍江和吉林交界的五常市沙河子林業局，其名字因酷似龍的骨架而來。一般長度在50-60毫米左右，2013年龍骨長度歷史性的超過70毫米，傳因為招哄搶引起爭執，導致樹被砍。

2013秋季的龍骨，大多被北京核商收購，市場已經難覓蹤影。龍骨主筋瘦骨嶙峋，伴隨凸刺紋，像蛟龍出水，又像飛龍盤山，上色快，是款值得楸子愛好者收藏的品種。龍骨分長樁、矮樁和圓形，以圖中長樁為代表，分別為不同樹的品種，其中圖示長樁歪尖的也有稱為魚骨。

八、龍紋柳葉

品種：楸子 / 產地：黑龍江 / 稱號：龍紋柳葉

（此處已添加小程序，請到今日頭條客戶端查看）

江湖傳聞：柳葉類楸子中紋路最為粗壯、深邃、複雜的柳葉上品，也是高樁楸子的代表作。據聞於2015年，此樹被砍。
