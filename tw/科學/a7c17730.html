三分鐘讀通10位經濟學大師，瞭解馬克思、凱恩斯、哈耶克的精華

| 科學



亞當·斯密：現代經濟學之父

我們期望的晚餐並非來自屠夫、釀酒師或是麵包師的恩惠，而是來自他們對自身利益的特別關注。——亞當·斯密

他那本可能排名第一的經濟學暢銷書《國富論》中，最核心的觀點就是：讓人們追求個人利益將有助於整個社會。

亞當·斯密花了十年時間完成了《國富論》，並在1776年出版了該書。《國富論》被視為全面闡述世界經濟理論的第一本書。這是一本為決策者準備的手冊，旨在說明社會如何能取得商業成功。全書分五卷，第一卷寫勞動生產力，以及各個不同社會階級之間的勞動力如何分配的問題。第二卷寫的是財富的積累。第三卷寫的是不同國家如何按著不同的速度發展的問題。第四卷討論了農業和商業體制的問題。第五卷，寫的是國家如何通過稅收來增加收入 和如何支出的問題。書中主要涉及六個方面的思想。

  1. “無形之手”其中包括個人自利的選擇，形成了最好的經濟成果，替代了計劃經濟。
  2. 勞動分工，根據人們的技能和裝備，把任務分配給個人或組織，達到最高的產出水平。
  3. 如果任由企業主實施他們的計劃，那麼就會形成壟斷，也就會抬高市場價格。
  4. 稅收制度應該公平、可靠、方便、有效的。
  5. 政府應該用稅收來保持經濟良好運作，但基礎設施的使用者，比如道路使用者，應該支付一定的費用。
  6. 禁止壟斷的形成，因為那樣會導致更高的價格。


大衛·李嘉圖：從移民到紳士

李嘉圖是亞當·斯密的迷弟，也是經濟學家中的富翁。他去世時留下了4500萬英鎊的遺產。他聞名於世的經濟理論主要是關於價值、租金、貿易及國家債券。雖然有些理論已經過時，但是200年前提出這些理論也是一種重要貢獻。

  1. 比較優勢：各國應專注於它們最有效的生產，並進口自己不擅長製造的產品。
  2. 李嘉圖效應：隨工資增長，資本密集型生產的產品，相對於勞動密集型的產品更便宜，因此，對前者的需求量會越來越大。
  3. 李嘉圖等價理論：政府藉此刺激經濟增長是不會成功的，因為人們會意識到，為了償還國債，他們不得不將來支付更高的關稅。



卡爾·馬克思：不僅是一位經濟學家

各盡所能，按需分配。——卡爾·馬克思

我們似乎都很熟悉馬克思，他是共產主義的創始人。他不但是經濟學家，更是哲學家、革命家和政治活動家。

馬克思的三個關鍵思想：對勞動的剝削是資本主義的基礎，資本主義播下了自我毀滅的種子，以及社會主義、共產主義制度最終將取代它。主要觀點：

  1. 工人的勞動與勞動權的區別。
  2. 僱傭者通過利用勞動者的工資和他們的勞動價值之間的差距來累積剩餘價值的觀點。
  3. 資本家保持工資最低化以最大化自己的利潤。
  4. 為了更為高效，資本家們不斷增加能節省勞動力的機器的投資，以增加剩餘價值。
  5. 這導致了“失業後備大軍”的產生，其貧困水平不斷增加將挑起階級衝突。
  6. 最終資本主義將會播下自我毀滅的種子。


阿爾弗雷德·馬歇爾：劃時代的《經濟學原理》

馬歇爾是經濟學家也是數學家，他寫的《經濟學原理》成了經濟學專業學生的標準教材。書中清晰地解釋了微觀經濟學的基本原理，包括供給和需求、邊際效用、需求彈性等概念。

  1. 將數學引入經濟學，並使其得到進一步發展。
  2. 供給和需求作為剪刀的兩個刃，決定了價格。
  3. 圖形可以作為經濟分析工具和解釋工具。
  4. 一個經濟互動的局部平衡分析，取決於經濟的其他方面處於其他條件不變的情況下。
  5. 給出了邊際主義的概念，並對成本或效用對行為的影響進行分析。
  6. 提出了工業區、內部經濟和外部經濟，這促進了公司發展。



約翰·梅納德·凱恩斯：升、跌、升、跌

凱恩斯提出在經濟衰退時期，國家有必要進行干涉的理論。凱恩斯算是富二代，其父親曾擔任劍橋大學教務長。自己也曾擔任過英國財政部駐巴黎和會代表，出版了暢銷書《和平的經濟成果》，幾乎預言了希特勒的崛起和第二次世界大戰。凱恩斯引入了“消費傾向”“投資誘因”和“資本的邊際效率”等新概念。以下是我們需要關注的要點：

  1. 儲蓄和投資並不總是趨於相等，因為它們由不同因素驅動，時間範圍也不同。
  2. 產出可能急劇下降，導致許多人失去工作，降低薪酬無法解決問題。
  3. 經濟中的總需求在決定總產出時起到了關鍵作用，所以政府可以採用稅收和支出方式，增加需求，改善經濟衰退時期的就業。
  4. 人們的不確定性和對未來的預期是經濟週期的主要驅動力。



弗里德里希·哈耶克：典型的自由主義者

哈耶克是凱恩斯的反對者，並因此獲得了諾貝爾經濟學獎。他將開放和自由市場視為組織經濟活動的最有效的方式，而且視其為個人自由的護衛者。

  1. 個體是社會根本，產生的社會結果是中央計劃者不能複製的。
  2. 政府裡的人，儘管意願很好或者很聰明，但永遠不能夠得到所有正確的信息，管理好經濟這類複雜系統。
  3. 社會最佳結果是被允許做決定的人基於他們的知識所做出的。
  4. 試圖讓所有人平等，而不是平等地對待他們，這是一種新的奴役形式。


米爾頓·弗裡德曼：貨幣主義之父

和哈耶克一樣，弗裡德曼熱衷於對自由市場的解釋，推崇小政府、少監管、純市場經濟的理論。他是理查德·尼克松和羅納德·里根的顧問。1976年，他因在消費分析、貨幣供應理論和歷史，以及政策穩定的複雜性論證等領域的成就而獲得諾貝爾經濟學獎。


弗裡德曼認為有必要限制政府的角色和範圍。他反對歧視同性戀，因為他認為這種偏見會限制一個人作為個體參與到市場的能力。他還認為有偏見的人強加了成本給他或她自己。他支持販毒和賣淫合法化的自由論者的思想，他也反對美國的兵役。最後，他提出了負所得稅的概念，即那些收入水平低於一定標準的人們能從州政府那裡收到錢，而那些賺得多的人則需要繳更多的稅。



保羅·薩繆爾森：新古典綜合經濟學

薩繆爾森的《經濟學導論》是賣得最好的教材，出售了上千萬冊，重印了19版，翻譯成41種語言。

他發現，在競爭市場內，如果參與者充分獲知了所有信息，價格最終呈現出無趨勢的隨機波動。

  1. 薩繆爾森區分了私人物品和公共物品的差異。公共物品是指某個人的消費不意味著其他人消費的減少。
  2. 世代疊加模型：每個人會生活在三個特殊的時間段：在前兩個段，他們工作、掙錢和存錢，最後一個階段是他們退休後，花錢且無錢可掙。
  3. 經濟學家應去找出消費者更偏好這個物品而不是其他物品的本質。


加里·貝克爾：現實世界的經濟學

貝克爾將新古典經濟理論運用到一系列社會問題的研究中，他的分析基於人類的行為是理性的，人們的行為動機是最大化其快樂感。

  1. 歧視會產生貨幣影響。
  2. 貝克爾假定犯罪行為是一個理性決定。
  3. 提供讓人上癮的商品的成本—對合法商品抽稅，對使用非法物品者加大懲處力度，可能會對減少上癮者的行為有好的影響。
  4. 家庭經濟學是加里·貝克爾的創新。



丹尼爾·卡尼曼：經濟心理學家

經濟學家考慮人們應該做什麼，而心理學家觀察人們實際上是怎樣做的。——丹尼爾·卡尼曼

卡尼曼作為心理學家卻獲得了2002年的諾貝爾經濟學獎。諾獎對其評價是：“他將心理學研究的觀點整合進經濟學中，特別是有關人類判斷和不確定性決策—由此奠定一個全新研究領域的基礎。”他是行為經濟學的主要推動者。

他認為：人們的行為經常表現出非理性。提出觀點：人們大腦中有兩個系統來做決策，系統1藉助直覺做出快速、自動和不費力的決定；系統2要運用複雜的推理來做出困難的決定。（《思考：快與慢》）


  1. 不僅僅是一名心理學家，卡尼曼（和特沃斯基）在行為經濟學領域也頗有建樹。
  2. 他的研究成果顛覆了作為經濟學基礎的理性行為主流思想的大部分假設。
  3. 他關注人們如何制定決策，現在已經融入政府制定金融決策和規章的過程中。
  4. 創立一個系統性的對啟發法和認知偏差的理解，極大地提高了策者理解人們如何做決策，以及避免做出糟糕決定的能力。
  5. 對於良好結果和決策者自身能力的過度自信，引發了金融繁榮泡沫破裂。
  6. 人們往往無法充分考慮他們所承擔的風險。
