熱拉提&熱瑪吉，傻傻分不清楚？

2020-11-14
| 科學

古往今來，廣大女性的美容需求總是層出不窮，有人追求白皙無暇，有人追求紅潤細膩，但在一點上大家都達成了空前一致：那就是 —— 緊緻飽滿、沒有皺紋的不老容顏。

先來說說皮膚是怎麼衰老的：

  • 皺紋

皺紋有大有小，有深有淺，淺層的小幹紋一般是角質層乾燥導致的，通常做好保溼工作就能緩解。

而較深、固定的皺紋則是因為真皮層的老化。當對皮膚起到支撐作用的膠原纖維和彈性纖維遭到破壞，就會出現肉眼可見的皺紋。

  • 鬆弛

當皮膚的衰老再發展，膠原蛋白、透明質酸的生產機器 —— 成纖維母細胞，就會因為老化而產能下降。

同時，因為紫外線照射、環境影響、不良作息等因素，最終會導致真皮層膠原蛋白、透明質酸加速流，含量大降。

再加上對錶皮有“拽著”作用的膠原纖維、彈性纖維持續遭到破壞，就會體現為皮膚外層的鬆弛。

  • 下垂

這時候必須提及一個名詞：

SMAS

Superficial Musculo-Aponeurotic System

表淺肌肉腱膜系統

這是一層纖維鞘包繞的肌肉筋膜，延伸至真皮纖維連接皮膚，通俗點就是骨肉相連之間的那層筋膜，覆蓋了整個面部組織。

筋膜層的衰老，比真皮層的衰老層次更深，明顯的表現是整個面部的軟組織下滑，就是感覺整個臉都往下垂，出現臉垮的感覺，比皺紋更顯老態。

皮膚的老化還有聯動性，淺層的衰老會升級成深層的衰老，淺層組織的鬆弛、位移也會拖著深層組織的更快下垂。

提升、緊緻......這些抗衰涉及的名詞，實則都是SMAS筋膜層的事兒。真正意義上的醫美手段提拉，其實是讓皮膚重新被SMAS筋膜層拽住的過程。也就是說提拉緊緻效果好不好，取決於有沒有作用到SMAS筋膜層。

護膚品是很難觸及到這樣深層衰老的，好在現在醫美比較給力，熱拉提、熱瑪吉這些黃金項目，花費不菲，究竟值不值得做呢？你又適合哪個呢？

熱瑪吉

// 原理

熱瑪吉採用的是射頻生熱激活膠原纖維再生，由外向內層層遞減式加熱皮膚。

// 深度

熱瑪吉作用方式主要在真皮層，這也是為什麼它收緊皮膚效果肉眼可見，但對深部組織提拉的效果不好。

// 熱量

表皮60℃左右，皮膚深層溫度大約在50℃。

// 作用方式

全層容積式加熱

// 舒適度

有明顯疼痛感，需要敷麻藥

// 治療部位

熱瑪吉不同的探頭可以針對不同的治療區域，大到身體、面部、頸部，小到獨有的眼周探頭。

// 效果體現

理論而言，治療後膠原會在3-6個月內逐漸增生，皮膚皺紋得到改善並且收緊。

// 維持時間

官宣熱瑪吉的作用會保持在2-3年不等，事實上時間和儀器優劣、醫生手法密切相關，事實上能保持1-2年左右。

// 副作用

熱瑪吉治療頭真假難辨，如果真的使用了假冒的治療頭，輸出不穩定，可能會造成燙傷、脂肪被破壞、局部凹陷等情況。


熱拉提

// 原理

熱拉提通過獨有的Focused RF聚焦射頻技術實現對皮膚分層加熱；淺層、中層、深層精準加熱。這個技術的厲害之處在於把加熱從盲探升級到雷達式定位導航，可以將射頻能量準確聚焦，精準加熱。

// 深度

淺層促進膠原收縮；中層激活膠原再生；深層可觸及到SMAS筋膜層和脂肪層；重現構建彈性纖維網，縮小並減少脂肪細胞。也就是說，熱拉提對於皺紋、鬆弛、下垂都有明顯的改善效果。

// 溫度

深層溫度在60℃-65℃，是刺激膠原的最佳溫度，表皮保持在40℃，不宜燙傷。

// 作用方式

分層聚焦加熱

// 舒適感

熱拉提在舒適度上甩了其他抗衰治療好幾條街，不需要麻藥，整體熱燙感在承受範圍之內。

// 治療部位

熱拉提針對不同治療部位有對應的手具，身體、頸部、面部、眼周，廣泛且不失精準。

// 效果體現

熱拉提治療後即刻就能看到明顯的皮膚收緊，之後隨著膠原不斷增生，提拉緊緻效果越明顯，輪廓線條感強烈。

熱拉提治療前後對比

// 維持時間

熱拉提需要按療程治療（3次/療程），療程治療後，可維持1-2年時間。

// 副作用

在FDA記錄數上，熱拉提的失誤率為零，安全性能優秀。但如果操作的手法不對或經驗不足，可能會引起燙傷問題，選擇正規醫院、正規醫生操作很重要。

  • 30秒拯救選擇困難症：

鬆垂hin厲害、怕疼的盆友選熱拉提

熱拉提是目前唯一可以用做到SMAS筋膜層的抗衰項目，再加上“隔空加熱”模式，實現了舒適提拉、溶脂小臉的功效。

PS:每個人的皮膚厚度和各部位的抗衰訴求是不同的，選擇專業醫生量身打造治療方案，選擇合適的擋位和深度很重要！不要貪圖便宜，那很有可能只是做了熱拉提維養版，預防衰老，對已經形成的鬆垮、下垂基本沒有效果。

普通鬆弛、耐受力強選擇熱瑪吉

熱瑪吉聚焦真皮層，除皺緊緻，一般也不需要療程治療，適合生活節奏非常快並且不怕疼的盆友。
