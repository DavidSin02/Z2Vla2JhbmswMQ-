1、尋找值得景仰的素紙

黑夜裡，有漩渦風

在某些裝置裡發出尖叫。

//

一匹瘦馬，揹負重要任務

它要尋找，值得景仰的素紙。

//

怪異之人，寂寞無奈

蝨子馱著他的鼾聲

剔出後腦勺他的抓狂

//

一隻鴻雁看到瘦馬，面臨危險

迅速砍斷他後腦勺的“抓狂”

//

而一匹瘦馬，沒心沒肺

繼續唱：“我要在奔跑中活出

不朽，我相信我能找到神祕的素紙”。

令人難以接受的是：這隻鴻雁

並沒有告訴瘦馬剛剛發生的事件。

此刻，它躲在樹冠中，看月光向

瘦馬方向移動，看神祕的一支筆

在樹下播種詩歌。而一匹瘦馬卻

在月光中看到自己，正和葉脈擁抱

青翠心靈。朝聖者的叮嚀送走

馬蹄，拽住叢林叫喊……

2、一個人把日子坐成傻瓜

一個人把日子坐成傻瓜

骨頭裡的山水一定有幾個朝代的瓦片與陶罐，

一些看不見的眼神如“霧中景象，虛幻如虛幻本身”

//

思想之外一定會看到

幽暗之地無數亡靈祕密的旅行

//

痛苦與磨難跟著肉體睡在一起

一開始自慰，然後自制堅強

到最後連同肉體不知去向

//

正如枝丫上的月亮

交出千萬銀針指令卻不留抓痕

正如夾在瓦片中的陶罐

交出五臟六腑堅硬

卻一聲不吭

//

一個人把日子坐成傻瓜

骨頭裡的枯枝一定有蜿蜒循環的生命

你一定會看見一個生命穿著時辰敲門

她在尋找什麼？難道是原野文字的靈魂？

抑或是灰燼中的火焰或火焰中的瓦片？

她的幻象裡總是出現沒有命題的U盤

//

她說：“這U盤是永遠壓不垮的陶罐的腳印”

她說：“陶罐在瓦片中間懷孕，生出的孩子

是故鄉田野裡的雲雀，是遠處山丘的銀杏”

//

墨色陶罐在佛祖那兒唸經的樣子被她確認

她確定那兒沒有騙子，沒有謊言

那瓦片與陶罐像潔淨的月牙，赤身裸體

掏出古典朦朧夙願

掏出江南靜默坦然

//

身體裡的密碼有還原的奇蹟

天空為之震顫：傾斜著

傾斜著，以悲憫的姿態

傾斜一地的瓦片與陶罐

3、在黃昏

在黃昏，你可以是蘿蔔、青菜、樹葉、或者一隻螞蟻

風高過屋簷，銀河的波濤也長出熟睡中嬰孩的口水。

河流上的潮汐，長出誦經的譜子。

通向萬物的大門，

被神祕的力量牽引。

//

雲朵向前回溯，惻隱之心落到綢緞處，

找到牧童的竹笛。

時光把人生裝點成，隨性踮腳的空間。

“天空齊腰深，雲在水面遊動"。電線杆子上沒有廣告“重金求子”

麻雀的胳膊，比蓮藕還粗

它們不嘰裡呱啦說詩人是瘋子。它們說詩人是有信仰的螞蟻。

//

在黃昏，不要聽噴嚏到處喊冤

低下頭，走向田野

到處都是夕陽

叼著的“火的星子”。

完整的天堂就站在

青菜蘿蔔的肩上

天空的骨子裡翻滾著葉脈的歡聲。

那是牧童的竹笛

邂逅琴絃的掌聲

//

在黃昏，自然的孩子

是你靈魂的一面鏡子

4、躲 藏

媽媽，其實你根本沒有上天堂

你的影子一直藏在熟悉的物件裡

為我託運冬天爐火營造的

人性力量。

手工縫製的棉布書包裡

依然藏著你種下的風景：

裡面有春天的暖風，也有

遼闊田野裡，翻滾的話題。

//

打開發黃的影集，彷彿又聽見

你對我嘮叨：“不要爭強好勝，

天外有天，人外有人……”

//

媽媽，你留下的物件就是

一面鏡子。鏡子裡有樹根

樹根裡有你翻唱的《風吹麥浪》
