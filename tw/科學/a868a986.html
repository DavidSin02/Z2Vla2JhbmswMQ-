6. P16

配製一定物質的量濃度的溶液（）。重點注意步驟、圖示、註解、注意事項（包括P18習題6）、學與問等。稱量固體時托盤天平只保留一位，量筒量取液體時也只保留一位。容量瓶使用的第一步操作：檢查是否漏水(簡稱“查漏”)。


“查漏”的方法：向容量瓶中加入適量水，蓋好瓶塞，左手食指頂住瓶塞，右手托住瓶底，將容量瓶倒轉過來看瓶口處是否有水滲出，若沒有，將容量瓶正立，將瓶塞旋轉180度，重複上述操作，如果瓶口處仍無水滲出，則此容量瓶不漏水。若漏水，可以在瓶塞處塗凡士林。常見容量瓶的規格有50 mL、100 mL、250 mL、500 mL、1000 mL 幾種。如配製溶液時明確知道所需容量瓶規格，則需將容量瓶規格一併答上。如圖所示：用玻璃棒引流時，玻璃棒末端應插入到刻度線以下，且玻璃棒靠近容量瓶口處且不能接觸瓶口。定容時，膠頭滴管不能伸入容量瓶。配製一定物質的量濃度的溶液所需要的實驗儀器：托盤天平、量筒、玻璃棒、容量瓶（容量一定要指明）、膠頭滴管、燒杯、藥匙。重要的實驗步驟：計算→稱量(量取)→溶解(稀釋)→轉移（輕搖）→定容→搖勻→倒出裝瓶。定容時視線與凹液麵最低處相平，直到液麵與刻度線相切。

7. P26－27

認真閱讀科學探求、科學史話、膠體的定義、膠體的性質、Fe(OH)膠體制備、區別膠體和溶液的方法等知識：FeCl溶液呈棕黃色，製備Fe(OH)膠體的操作方法是：在沸水中滴加飽和FeCl溶液，繼續煮沸至紅褐色，停止加熱。將0.1mol FeCl製成膠體，所得的膠粒數小於0.1N，Fe(OH)膠體不帶電，Fe(OH)膠粒帶正電。 FeCl溶液和Fe(OH)膠體最本質的區別是膠體粒子大小在1nm-100nm之間，區別這兩種分散系最簡單的方法是丁達爾效應。膠體粒子不能透過半透膜，能透過濾紙。氯化鐵溶液與氫氧化鐵膠體具有的共同性質是：加熱蒸乾、灼燒後都有氧化鐵生成。常見的膠體有：Fe(OH)膠體、Al(OH)膠體、硅酸膠體、澱粉溶液、蛋白質溶液、血液。工廠中常用的靜電除塵裝置就是根據膠粒帶電的性質設計的。

8.P24-25

分類的方法及物質的分類及實踐活動：特別注意：NO、CO為不成鹽氧化物，NO溶於水生成HNO，但HNO的酸酐為NO。酸酐不一定都是氧化物，如醋酸酐。酸性氧化物、鹼性氧化物不一定都與水反應生成對應的酸和鹼。SiO能和強鹼反應，也能和HF反應，但不是兩性氧化物。1mol HPO最多和3molNaOH，說明HPO為三元酸；1mol HPO最多和2molNaOH反應，說明HPO為二元酸，NaHPO為正鹽；1mol HPO最多和1molNaOH反應，說明HPO為一元酸，NaHPO為正鹽。


9.P30

電解質和非電解質：純淨的酸、鹼、鹽、金屬氧化物、水是電解質；其它純淨的化合物一般是非電解質；澱粉、鹽酸、氨水、單質銅既不是電解質，也不是非電解質（前三者為混和物、後者不是化合物）。BaSO的水溶液不易導電，但BaSO是強電解質。一水合氨是弱電解質。NaHSO在熔化狀態下的電離方程式為NaHSO=Na+ HSO。證明某化合物為離子化合物最簡單的方法是：在熔化狀態下是否導電，若導電則為離子化合物

10. 物理變化和化學變化：化學變化過程中有舊化學鍵的斷裂，同時又有新化學鍵的形成，但有化學鍵斷裂的變化未必就是化學變化，如NaCl晶體熔化、SiO晶體的熔化，分別破壞了離子鍵、共價鍵，但屬於物理變化。一種原子變為另一種原子涉及原子核的變化，不屬於化學變化。CuSO等電解質溶液的導電屬於化學變化。煤的氣化、液化、煤的乾餾、裂化、裂解為化學變化。石油的分餾為物理變化。

13.鎂及其化合物

鎂可以分別在N、CO中燃燒，化學方程式分別為3Mg + N MgN、

2Mg+2CO2MgO+2C。將鎂和N燃燒產物溶於水，有沉澱和氣體生成:

MgN+8 HO=3Mg（OH）↓+2NH↑。

從海水中來提取MgCl時的流程：加氫氧化鈣，過慮洗滌得氫氧化鎂沉澱，用鹽酸溶解沉澱，通過蒸發濃縮、冷卻結晶得到MgCl·6HO晶體，再在HCl氣體環境中加熱MgCl·6HO晶體得到無水MgCl，HCl氣流主要是為了抑制Mg水解；電解熔融的MgCl，不用MgO電解來製備Mg是因為MgO熔點很高，需要能量高。


14.鋁及其化合物

鋁是地殼中含量最多的金屬元素，工業上不用電解AlCl來製備鋁的原因是AlCl是分子晶體在熔融的狀態下不導電（必修二P93第6題從鋁土礦中提取鋁的有關反應）。


P58

氧化鋁的性質用途：氧化鋁為兩性氧化物，是電解質。


P58

實驗3－7、實驗3－8

氫氧化鋁是兩性氫氧化物，為弱電解質。實驗室製備Al(OH)的離子方程式為：Al + 3NH·HO == Al(OH)↓+ 3NH，不用NaOH等強鹼原因是Al(OH) + OH == AlO+ 2HO，以下幾種方法也可以得到氫氧化鋁。


取0.25ag鋁（鋁的質量為a g）溶於適量的鹽酸中，再取0.75a g鋁溶於適量的強鹼溶液中，將兩溶液混合即得白色沉澱，3AlO + Al + 6HO == 4Al(OH)↓；


偏鋁酸鈉溶液中通CO。CO少量與過量時也可以得到氫氧化鋁。（CO少量）CO + 3HO + 2NaAlO == 2Al(OH)↓+ NaCO、（CO過量）CO + 2HO + NaAlO == Al(OH)↓+ NaHCO。


Al和AlO的轉化：Al + 4OH == AlO + 2HO、AlO + 4H == Al + 2HO

AlO和HCO不能共存。不是雙水解，是得質子的能力不同引起的：

AlO+HCO+HO= Al(OH)↓+ CO


明礬的淨水：明礬、FeCl·6HO被稱作淨水劑，原因是Al、Fe水解形成膠體（Al + 3HOAl(OH) (膠體)+ 3H），吸附水中的懸浮物，使之沉降已達淨水目的，只有淨水作用，無殺菌、消毒作用。


嚮明礬溶液是加入Ba(OH)溶液，沉澱的質量最大和沉澱的物質的量最大的離子方程式分別為：Al+2SO+2Ba+4OH=AlO+2BaSO↓+2HO 、2Al+3SO+3Ba+6OH= 2Al（OH）↓+3BaSO↓


泡沫滅火器Al(SO)溶液不能裝在鐵桶中是因為Al水解顯酸性，NaHCO溶液不能裝在玻璃桶中是因為HCO水解呈鹼性。泡沫滅火器反應原理：Al +3 HCO == Al(OH)↓+CO↑。

15.鐵及其化合物


P59

鐵中含有碳等雜質，使鐵的熔點降低，在常溫下濃硫酸和濃硝酸使鐵鈍化。鐵的氧化物中，赤鐵礦（FeO）紅棕色粉末，俗稱鐵紅，常用著紅色油漆和塗料。也是鍊鐵的原料。磁鐵礦（FeO）具有磁性，俗稱磁性氧化鐵，是黑色晶體。FeO是一種黑色粉末，在空氣里加熱就迅速氧化成FeO.鐵在Cl中燃燒無論Fe或Cl過量均生成FeCl，但可以用化合反應生成FeCl，相關反應為：2Fe + 3Cl 2FeCl，Fe +2 FeCl == 3FeCl。鐵的氧化物與非氧化性酸，強氧化性酸，還原性酸反應的特殊性。FeO+8H= 2Fe+Fe+4HO，FeO+6H+2I=2Fe+I+3HO，3FeO+10H+NO=3Fe+NO↑+5HO。Fe被氧化，Fe被I還原。


P60

鐵的氫氧化物：在製備Fe(OH)時可以加熱到沸騰除水中的氧，冷卻後再配溶液，也可以加比水輕，不溶於水的有機溶劑（苯）封住液麵，加NaOH溶液時膠頭滴管要伸入到溶液中接近試管底，防止Fe被氧化，可以加入鐵粉，Fe(OH)氧化成Fe(OH)的現象為白色絮狀沉澱迅速變成灰綠色，最後紅褐色，化學方程式4Fe(OH) + O + 2HO == 4Fe(OH)。加熱FeCl溶液，最終得到的是FeO。氧氧化鐵、氫氧化亞鐵分別與強氧化性酸，還原性酸反應的：(OH)+6H+2I=2Fe+I+6HO，

3Fe（OH）+10H+NO=3Fe+NO↑+8HO。

Fe、Fe的性質及其檢驗。檢驗Fe通常有以下幾種方法：

①加KSCN溶液，無明顯變化，再加氯水，溶液變血紅色。Fe+3SCN≒Fe（SCN）。

②加氫氧化鈉溶液，出現白色絮狀沉澱迅速變成灰綠色，最後紅褐色。4Fe(OH) + O + 2HO == 4Fe(OH)。

③加溶KFe(CN)溶液，生成藍色沉澱。3Fe+2[ Fe(CN)]=Fe[ Fe(CN)]↓


檢驗Fe通常有以下幾種方法：

接觀察溶液是棕黃色。滴加氫氧化鈉溶液，出現紅褐色沉澱；滴加KSCN溶液，有血紅色溶液出現；加入苯酚溶液，呈紫色。

16.硅及其化合物

P74

碳是構成有機物的主要元素，而硅是構成岩石與許多礦物的基本元素。硅是一種親氧元素，在自然界中它總是與氧相互化合的，在自然界中主要以熔點很高的氧化物SiO及硅酸鹽的形式存在。結晶的SiO是石英，其中無色透明的是水晶，具有彩色環帶或層狀的稱為瑪瑙。沙子中含有小粒的石英晶體。純淨的SiO是現代光學及光纖製品的基本原料。可以用HF刻蝕玻璃，是因為SiO可與HF酸反應（SiO+4HF=SiF↑+2HO），但SiO不是兩性氧化物。SiO為酸性氧化物，但不溶於水生成硅酸。盛鹼溶液的試劑瓶一般用橡膠塞（P76圖4-6）因為SiO易與強鹼溶液反應，生成硅酸鈉使試劑瓶受腐蝕。SiO為原子晶體，不存在單個的SiO分子，1mol的SiO中含有4mol的Si-O鍵。


P76

HSiO↓。若CO過量，則反應為：NaSiO+2HO+2CO=2NaHCO+HSiO↓

硅酸鈉(NaSiO) 的水溶液俗稱水玻璃，不能燃燒，不易被腐蝕，熱穩定性強，是製備硅膠和木材防火劑的原料。NaSiO寫成氧物的形式可以表示為NaO·SiO。
