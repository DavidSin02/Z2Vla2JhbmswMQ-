清華大學藝術博物館傢俱圖賞

2020-11-14
| 科學


| 明清傢俱研習社 |

研習君語

清華藏珍，隨方制象

作為高校博物館，清華大學藝術博物館的藏品絕大多數來自美術學院的前身——中央工藝美院自1956年建校開始的歷年收藏，部分來自校友及各界人士的捐贈。

其中“清華藏珍·隨方制象”——清華大學藝術博物館藏品展/傢俱部分，作為博物館內的常設展，常年舉辦。

傢俱廳中的這批明式傢俱共有41（組）件，以黃花梨、紫檀、鸂鶒木等優質硬木為主，均為傳世明式傢俱中的罕見佳品。

本期研習君整理了一批高清精美大圖，與各位共賞（記得收藏哦）：



明 黃花梨草龍紋玫瑰椅（一對）

尺寸 / 長52cm、寬46.8cm、高90cm、座高46cm

椅子搭腦兩端與後立柱連接處，扶手與前立柱相接處都使用挖菸袋榫做法，工藝講究，後背和兩側草龍紋角牙與卷草紋和雷紋渾然一體又富有變化。

靠背牙板及兩端牙板較寬，增加了牢固性，搭腦和扶手處均做相同造型，靠背內安裝門券口牙子，座面下安裝壺門牙板，淺雕卷草花紋。

扶手平出直豎，通體方正嚴明，自然流暢。兩側與後部面板之上均做羅鍋棖，增強結構堅固性，與面板以草龍造型連接，增強堅固性的同時，與整體裝飾合，雕飾均重點式點綴，起點綴或烘托作用。

明 黃花梨壼門靠背玫瑰椅

尺寸 / 長57.5cm、寬45.5cm、高87cm、座高46cm

在明式傢俱中，這是典型的玫瑰椅做法。

四立柱均為“一木連做”。搭腦直接落在後腿上端，靠背內安裝門券口牙子，直接落在橫直棖上。壼門牙板淺雕草花，造型與口沿同形一體。

座面下依然安裝壺門券口牙板，落在前橫棖上。扶手平出直豎。通體風格方正嚴明，自然流暢，是為明代玫瑰椅之典型。

清 黃花梨群龍捧壽紋玫瑰椅(一對)

尺寸 / 長58cm、寬45.5cm、高88.5cm、座高51cm


明清時期玫瑰椅基本形態有四個特點：

1.所有主要受力杆體截面皆為圓形；

2.搭腦直靠背矮；

3.靠背與座屜夾角為90度；

4.靠背扶手下面有矮欄。

這對椅子具備以上四個特點，但又在靠背、扶手以及牙板上做了大面積的透雕、浮雕紋飾。造型手法摻入了一些清代因素，使得這對椅子具有了明風清韻的特點。

明 黃花梨高扶手南官帽椅

尺寸 / 長52cm、寬46.8cm、高90cm、座高46cm

明代南官帽椅類中有一種扶手傾斜度比較大的品種，扶手與後足連接處比一般扶手椅高，與搭腦相距較近，故呈現出向前傾斜之勢。

這類形制在江南各地廣為流行，所見用材各有不同，工藝也有簡有繁，但特徵相同。

這對椅子靠背為打槽裝板三段式做法，上段為浮雕草龍，下有亮腳，中段則選用紋理優美的板材鑲嵌。座屜也抹為混圓壓邊線，使得整體氣度非凡，雍容質樸，是一對明代南官帽椅的精品。

明 櫸木矮南官帽椅

尺寸 / 長77.8cm、寬75cm、高46cm、座高30cm

這件櫸木南官帽椅因為座面要比一般官帽椅矮20釐米，因此可稱為“矮南官帽椅”。

兩扶手向外彎曲，扶手下的鐮刀把和鵝脖也做成兩彎形式，一方面增大了坐具的使用空間，一方面也使得整個造型線呈現一種曲線美。

靠背板以方形鏟地浮雕為裝飾，較之一般的圓形雕花，則更容易使得這個空間充盈起來。

搭腦向後彎曲幅度較大，兩端與後立柱相接處加裝透雕草花角牙，扶手與前立柱相接處也裝角牙。座面以下，四面直券口牙子。椅盤為軟屜原裝，未經改換，下有穿帶支撐，並披硃砂灰。

當然，這件椅子最值得稱道的還是結構上的特點，椅盤高度只有一般明式椅的一半，加上墊子彷彿就是一件現代傢俱。明式官帽椅承載有禮制的意義，此椅據王世襄先生考證“很可能是寺院禪椅，專供趺坐而制”，而禪椅到了明代已逐漸進入文人生活。


明 黃花梨四出頭官帽椅

尺寸 / 長59cm、寬47.2cm、高115.7cm、座高50cm

這對官帽椅美在靠背採用了紋理十分漂亮的一塊獨板做成。因為紋理自然天成，就不必要再進行雕斫，這是古代工匠技藝高明之處。

搭腦中寬形成靠枕，兩端微微翹起，曲線既富有彈性，造型又十分流暢舒適。同樣的手法又見於扶手、以及腿部的細部處理上，彼此呼應，相互襯托，使得此椅大氣而富有匠意。

明 黃花梨透雕背板圈椅

尺寸 / 長58.5cm、寬45.9cm、高99cm、座高50cm

此椅獨具特色之處在於背板，一般圈椅的靠背板為打槽裝板或獨板，板上或全素、或雕刻、或鑲嵌。但這件圈椅的靠背板作法卻不同於此，它介於二者之間，或者說二者兼有。

靠背板兩邊立柱打槽，中嵌一塊從上到下的獨板，下有亮腳，上有團花。團花做法也很特別，是在靠背板上挖一如意頭形洞，另雕一與之相應形狀的蟠龍從前面嵌上去的。

明 黃花梨裹腿做雙絛環卡子花方杌凳

尺寸 / 長50.7cm、寬50.5cm、高46.6cm

明清時代硬木裹腿做法的傢俱，皆是仿竹製傢俱而創造出來的一種式樣。

圓材直腿直棖是裹腿構造傢俱的特點。凳面為格角攢邊框，草蓆硬屜，邊、抹加一條垛邊，為劈料做法，看上去彷彿凳面使用厚材製成。

明 黃花梨有束腰內翻馬蹄長方杌凳(一對)

尺寸 / 長48.2cm、寬42.7cm、高50.6cm

明、清杌凳，座屜大多為棕藤敷屜，也就是上層為藤屜，下層為棕屜，再下有弧型託棖。

但棕繩、藤席使用期限有限，特別是在北方，天氣乾燥，同時北方工匠不諳編棕編藤工藝，於是改用木板上鋪臺灣草蓆，這時間應從清後期開始。這對杌凳座屜即是這一時期改的硬板草蓆面子。

束腰牙板一木連做，牙板起陽線與腿子陽線相連，腿端馬蹄內翻有力，通體簡素而有品味，為明代杌凳一種典型式樣。

明 黃花梨帶束腰羅鍋棖內翻馬蹄長條桌

尺寸 / 長91.5cm、寬85cm、高85cm

此桌腿間安羅鍋棖，冰盤沿下帶束腰，面板中採用攢邊做法，其方法是把“芯板”裝入採用45度格角榫構合的帶有通槽的邊框內。

一方面使薄板能當厚板使用，以達到節約材料的目的；另一方面因乾溼髮生伸縮時，通槽留有充分餘地，不至於發生脹裂變形或收縮透縫等現象而造成全體結構的鬆動和傢俱體形的走樣。

此外將“芯板”裝入通槽後可以隱藏木材粗糙的斷面，使傢俱各部分都顯露出光潔平滑的縱直斷面來，充分顯示木材天然的紋理和色澤。

明 黃花梨四面平條桌

尺寸 / 長143.6cm、寬42cm、高79.2cm

四面平條桌也叫“四面齊條桌”，是明式傢俱中的經典樣式。其做法是腿子與牙條格相交，也就是先做成一具骨架，再與攢邊的桌面結合在一起，增強牢度，又因為邊抹和牙條重疊，增加體量感，避免單薄。

此類做法只見於明式傢俱，入清後少見。

古斯塔夫.艾克在1944年出版的明式傢俱研究專著《中國花梨傢俱圖考》中著錄了這件琴桌，同時還附上楊耀先生協助繪製的精美而準確的測繪圖。

明 黃花梨有束腰馬蹄足霸王穿條桌

尺寸 / 長96.2cm、寬48cm、高83.5cm

較於一般霸王棖傢俱，此桌的棖與腿足連接處稍微靠下，約在距桌面三分之一處，因此尤顯舒展，十分耐看。

清 紫檀夾頭榫素牙子平頭案

尺寸 / 長96.2cm、寬48cm、高83.5cm

此案雖為清制，卻有明風，通體全無雕飾，形體雖小卻沿襲了明式傢俱簡樸大氣的風範。

與一般平頭案相比，主要構件集中在上部，四足平豎直伸，光潔簡練，尤顯挺拔俏麗。



明 黃花梨帶束腰三彎腿虎爪足炕桌

尺寸 / 長90.1cm、寬56.6cm、高28cm

這是一件帶束腰齊牙條的三彎腿炕桌。

一般炕桌的牙條與腿足之間的相接，均成45度斜角，稱作“格肩”。這件炕桌卻不然，齊牙條的目的在於維護腿足肩上浮雕獸頭的完整，避免因格肩相接而使獸頭分為兩半。

由於上面有獸頭，所以足底通常雕作獸爪(也叫虎爪)，與獸頭相呼應。這種造型寓意驅避妖邪和護衛祥瑞。

牙條以陽紋淺浮雕刻畫兩條柺子龍，相向奔騰，中間又飾以卷草靈芝，寓意“謙和”“長壽”。

三彎腿自上而下的第一彎向外彎曲較長，第二彎向內較短，第三彎則表現為獸爪的內扣。浮雕紋樣與造型變化有機統一，體現著完整而豐富的裝飾風格。

明 黃花梨無束腰竹節紋炕桌

尺寸 / 長80cm、寬80cm、高24.2cm

這件炕桌基本結構是直棖加矮老做法，但在直棖兩端加做兩彎斜棖與桌面相連。


最突出的特點是以木材仿竹材，無論桌面冰盤、四足、直棖，還是矮老、斜棖，一律雕出竹節，四足略做彎曲，著意模仿竹根那種竹節密集、久經風霜的形態，愈顯逼真。

混面由一窄一寬的兩根“竹材”構成，上邊突出板面， 形成攔水線。這種做法，既維護了炕桌的基本形式與功能，又在保障結構的基礎上生成了有趣的變化，使其獨具特色。

明 黃花梨夾頭榫撇腿雲頭牙子翹頭炕案

尺寸 / 長135cm、寬41.6cm、高42.9cm

此案是傳世矮型傢俱中的佼佼者，其外撇的腿足不僅造型優美，亦且增加了穩定感。腿間透雕如意雲頭，腿與面板交界處，嵌鏤空雲頭角牙，與兩腿間的大雲頭相呼應。


此類造型尤其少見，《明式傢俱珍賞》中另收藏一件“明鸂鶒撇腿翹頭炕案”，外形與之相同，但其案面為打槽裝板做法，是上世紀40年代北京魯班館工匠依此件而複製的，而這件卻是難得的明代傳世精品，尤其珍貴。

明 黃花梨龍鳳紋翹頭案

尺寸 / 長129cm、寬40.6cm、高85.5cm

這件翹頭案面子用攢邊做法，面芯為獨板，小翹頭。腿截面長方形，內方外弧“兩炷香”做法。

牙板邊起陽線，牙頭為變形鳳紋，局部雕空與腿間花板相呼應，紋飾都很成熟，雖然程式化，但又十分生動。擋板下有託泥，腿間為透雕雙龍靈芝紋，疏密得體。

明 黃花梨龍鳳紋翹頭案

尺寸 / 長129cm、寬40.6cm、高85.5cm

這件大翹頭案，長近2.5米，形體巨大，在明代傢俱中的傳世品中極為少見。

檔板透雕“如意雲頭”，雲頭向上，頎秀挺拔，輪廓處均起陽線。雲頭外圍的檔板其實已經構成了四環牙板，由此固定了兩腿的牢度。

如此巨大的條案，若兩腿鬆動，後果將不堪設想，所以，兩檔板的作用不可低估。

明 黃花梨夾頭榫雲紋牙頭帶託泥翹頭案

尺寸 / 長198cm、寬85.5cm、高49.5cm

這件翹頭案的牙板造型，極盡變化之能事，在兩足上端鏤雕為對稱雲頭，上部又加小云勾，亦對稱。


兩側檔板也鏤雕深卷的雲頭，大小云勾間、雲勾與牙板間均加施“支釘”般的聯結。雲頭中央與下牙板相接處，本應漸寬漸放，形成外放的斜坡，今卻不然，偏偏向內收進，形成圓底相接。

這種富於變化的鏤雕裝飾，提高了傢俱堅牢耐用的性能。

清 鸂鶒木小圓角

尺寸 / 長76cm、寬46cm、高133cm

這件鸂鶒木圓角櫃形制規範，很有代表性。兩門中間無“閂杆”，名為“硬擠門”。櫃底無櫃膛，底棖下面開槽安裝“窪堂肚”券口牙子，兩側鏤成內捲雲勾。

此櫃製作上一絲不苟，選材不吝成本，四面獨板，結構嚴謹，通體明朗光潔，於簡潔中蘊涵著秀麗。



明 花梨包鑲框癭木門芯大四件櫃

尺寸 / 頂櫃: 113釐米、57.5釐米/高 77釐米底櫃: 113釐米、57.5釐米/高

此櫃用花梨木做鑲框，樺木癭做面板，松木做後背板。

櫃內由上而下，第一層為屜板，第二層為抽屜架和抽屜兩具，第三層為櫃膛，頂箱則只有一層屜板。

櫃採用四面平形式，整體方正大氣，在裝飾構件上十分用心。兩立櫃面芯上部各鑲嵌四枚黃花梨與黃楊木拼嵌而成的博古紋樣。

銅活面葉、合頁皆做蓮瓣式，面葉銅活蓮瓣為如意雲頭，並墜雙龍形吊飾，非常精美。
