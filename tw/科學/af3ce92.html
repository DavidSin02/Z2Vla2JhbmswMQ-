互助問答第20期：PSM檢驗問題、弱工具變量問題

2020-10-29
| 科學

問題1：在做完PSM後，用pstest來檢驗匹配結果是否較好地平衡了數據，但是出現r(2000)錯誤。請教各位老師這個問題該如何解決？

所用Stata命令如下：

psmatch2 POSTAH size lev ROE loss MB1 SALESGROWTH SOE CAPE first PC ind1, outcome(disp1) n(1) ate ties logit common noreplacement

pstest, both graph

Stata命令運行結果如下：

答案1：

r(2000)一般是指no observation r(2000)。出現該錯誤原因主要是pstest後面沒有加進行平衡性檢驗的變量。正確的命令為：

pstest size lev ROE loss MB1 SALESGROWTH SOE CAPE first PC ind1, both graph

問題2：（1）很多資料提到“檢驗弱工具變量的一個經驗規則是，如果在第一階段迴歸中，F統計量大於10，則可不必擔心弱工具變量問題。”如果2SLS第一階段迴歸的F值大於10，但是弱工具變量檢驗不通過，是否仍需考慮弱工具變量的問題？

（2）有論文提到，使用LIML（信息極大似然估計方法）迴歸可以解決弱工具變量問題，但我用LIML迴歸的結果與2SLS結果基本一致，是否在大樣本條件下LIML不能解決弱工具變量問題呢？或者是說在大樣本條件下，LIML已經不能避免弱工具變量問題引起的偏差？

（3）採用該工具變量回歸後，內生變量回歸係數增加了100倍，是由於弱工具變量的問題嗎？

（4）除了換其他工具變量，老師有其他解決弱工具變量問題的方法或建議嗎？

答案2：

（1）當有多個內生變量的情況下，存在多個第一階段迴歸，會有多個F統計量。此時，要用最小特徵值統計量進行檢驗。對只有一個內生解釋變量的情況，為了更加嚴謹，即便是2SLS第一階段迴歸的F值大於10，但是弱工具變量檢驗不通過，就仍需考慮弱工具變量的問題。

（2）這個理解是對的。在大樣本下，LIML與2SLS是漸近等價的。當存在弱工具變量時，LIML的小樣本表現要優於2SLS。

（3）內生變量回歸係數增加了100倍，說明這個工具變量不靠譜。

（4）尋找更強的工具變量可能是最好的選擇。
