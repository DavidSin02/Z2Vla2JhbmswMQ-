FileMenu Tools v7.6.2 右鍵菜單管理軟件

2020-10-29
| 科學

FileMenu Tools是一款強大的 Windows右鍵菜單增強工具，它允許您自定義 Windows 文件資源管理器的右鍵菜單，並可將一些工具添加到 Windows 文件資源管理器的右鍵菜單。應用程序添加了內置工具來執行文件和文件夾的操作，並添加了允許您運行外部應用程序的自定義命令、複製/移動到特定文件夾或刪除特定的文件類型。使用 FileMenu Tools 您還可以自定義“發送到…”子菜單，並啟用或禁用由其它應用程序添加的右鍵菜單命令。

軟件特色

FileMenu Tools允許您添加自定義命令和一些內置實用程序，以便從Windows資源管理器對文件和文件夾執行操作

從配置窗口的FileMenu Tools選項卡的命令，可以配置定製命令和內置命令，這些命令將顯示在Windows資源管理器的上下文菜單中。在每個命令的左側存在一個複選框。如果選中此複選框，命令將顯示在上下文菜單中，否則命令將不顯示

在“動作”面板和工具欄中，存在以下操作：

·添加命令。添加一個新的自定義命令。屬性是從“屬性”面板配置的。

·添加子菜單。將新的子菜單添加到上下文菜單中。到子菜單可以添加新元素或現有元素。為了將現有元素移動到子菜單，您必須用鼠標指針拖放到子菜單上。必須在“屬性”面板中配置要在上下文菜單中顯示的文本和圖標。此外，可以配置子菜單，因此當只有一個命令顯示在子菜單中時，則顯示該命令代替子菜單。這可以使用“替換單個命令”選項進行配置。

·添加分隔符。添加新的分隔符。

·刪除。刪除所選項目。只能刪除自定義命令，子菜單和分隔符。內置命令無法刪除;如果您希望從上下文菜單中刪除它們，則必須取消選中命令左側的複選框。

您可以通過單擊添加命令操作添加新元素。

要刪除現有元素，請單擊刪除操作。

如果要隱藏現有元素，請取消選中元素的左側複選框。

為了更改元素的屬性，請選擇元素並從屬性面板更改其屬性。您可以更改以下屬性：

· 名稱。將顯示在菜單中的文本。

·目標。必須發送文件的目標。它可能是可執行文件或文件夾。

·參數。將給予程序的論據。

·工作文件夾。將運行程序的工作文件夾。

·圖標。將顯示在菜單中的圖標。如果單擊右側按鈕，將顯示一個對話框以選擇文件系統中的圖標。您可以選擇帶有.ico擴展名或圖標的圖標進入圖標庫（.exe，.dll，.icl等）
