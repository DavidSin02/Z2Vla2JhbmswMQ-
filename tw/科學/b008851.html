KEEN戶外運動涼鞋，該如何挑選最適合自己的涼鞋呢？

2020-10-29
| 科學

說到KEEN的涼鞋大概無人不知，不管走到哪裡都會看到KEEN的身影，這是因為KEEN是可以做到不為迎合流行而犧牲專業表現的鞋子，而這一切都是因為KEEN創始人本身是戶外運動高手，為了在水上運動時享受穿著防水透氣的涼鞋，同時又不想讓自己脆弱的腳趾頭在激烈的賽事中受傷，於是他設計出全球第一雙可以保護腳趾的涼鞋！一雙革命性的戶外涼鞋奠定了KEEN這個不斷創新且融入生活的專業品牌，不過那麼多款涼鞋該如何挑選最適合自己的涼鞋呢？身為 KEEN 的多年用戶，這次整理了四雙 KEEN女款經典涼鞋的使用心得來分享給女孩們，順便介紹今年推出的超棒新款 Ravine 和 SOLR！

夏季戶外活動的開始，溪邊露營、步道輕徒步、海邊戲水、戶外攀巖等活動，少不了一雙能下水還能保持舒適的戶外涼鞋，而一般涼鞋和戶外涼鞋最大的差別在於戶外涼鞋擁有包覆、快乾、支撐等等機能，才能讓使用者在戶外活動時更加安全、自在。以我這個戶外活動重度成癮者，在還沒有成為KEEN的愛用者之前，都直接穿夾腳拖走，苦主當然就是常常踢到東西還有輕微外翻的大拇趾，穿了KEEN之後，我最喜歡KEEN能夠保護腳趾和貼合腳部這兩大優點，為了活到老玩到老，一定要穿好鞋保護能帶你到處去的腳。

先介紹四雙鞋的幾個共同點：

1. 全球首創護趾設計

2. 可調式快速鎖定彈性鞋帶，方便不滑脫

3. 可機洗，使用少量中性清潔劑洗滌、風乾

4. 不含PFC全氟化合物的防水塗層織帶

5. 鞋舌的織帶皆有車上環帶，方便讓使用者穿脫或是吊掛涼鞋時運用

6. ECO ANTI-ODOR Cleansport NXT 抑菌材質鞋墊

特別說明一下抑菌材質鞋墊，是採用具有控味機能的天然微生物，附著於細小纖維表面，強調不使用重金屬（大部分是用銀粒子），當運動過程中產生汗水或碰到髒水，其中的細菌會被聚合物的正電荷吸附，此原理可消除細菌滋生造成的異味，但大家不要誤會，抗菌防臭功能的織品主要在抑制細菌增殖，並不代表它可消除汗水本身散發之臭味，因為抗菌功能的要求很嚴格，所以通常品牌行銷用字不會用到 Bacteria 細菌等字眼，皆以Anti-odor控味來代替，不過控衛控味功能還是因人而異，勤勞清洗鞋子比較實在！（鞋子浸過海水的話，最好再泡肥皂水清洗，因為海水乾掉的味道實在是不好聞)，接下來就讓我以我各個行程中的實際體驗心得來介紹每雙鞋子的不同之處，以及我個人的推薦點評。

玩水必備 SOLR

2020 春夏全新鞋款 SOLR 的名稱是以戶外水性元素組成：Sea、Ocean、Lake、River，是一雙高性能的玩水鞋款，全新的 Aqua Stop 區域加強在水中或潮溼環境中的防滑性能，速幹襯裡特別適合戶外水上活動，注入 TPU 的防水耐磨鞋面和 Aquagrip 橡膠大底讓 SOLR 具有不錯的兩棲抓地力，可在潮溼的表面提供良好的抓地力，特別設計的織帶是由回收塑料製成，緩震輕量的 EVA 中底搭配足弓支撐鞋墊，讓雙腳全天穿著都舒適，薄型設計讓雙腳在水中也能保持敏銳路感，感受踩在腳底的路面，靈活移動。

SOLR 的 Aquagrip 橡膠大底可在潮溼的表面提供良好的抓地力

近年來 PFC 被證實是人造有害化學物質，透過大氣與空氣散佈，在生物體中累積，對人體、動物會造成長期的傷害，KEEN 在製鞋過程中，使用不含 PFC 全氟化合物的織帶防水層，新款 SOLR 還進一步使用回收塑膠來製造防水織帶，KEEN 展現了努力在創新與環境保護之間找到最佳平衡的決心！

SOLR 的 TPU 鞋面防水耐磨，亮色系的環保織帶由回收塑料製成

穿上 SOLR，帶著板子去海邊吧！

穿著 SOLR 也可以穩穩站在板子上

SOLR 具有不錯的止滑效果，但是在佈滿海藻的溼滑石面還是要非常小心行走

整雙鞋面內側為潛水衣軟布材質 Neoprene，免去沾溼後會磨腳的問題

趕快穿著 SOLR 去玩水吧！

我個人會把 SOLR 定位為水域活動專用，因為 AQUAGRIP 橡膠大底的材質黏性比較強，所以在潮溼的平面上，抓地力更好，比如像是船的甲板上或光滑平面上，而 AQUASTOP 鞋底小顆粒的刻紋設計也和 NEWPORT 稍有不同，可以讓水加快從鞋底排出。加上同樣也有EVA足弓支撐功能，整天穿著在外是沒問題的！

創新突破: 可拆式後腳踝織帶 Ravine

基本上我個人覺得 Ravine 和 Newport 的定位比較類似，大底的設計與支撐度是與 Newport 幾乎差不多的，也是護指系列中較好的，但是 Ravine 的可拆式織帶設計讓一雙涼鞋馬上有了拖鞋的雙重身份，且因為移除部分織帶包覆，比 Newport 來得更輕量、更透氣，後跟調節讓不同腳型也能穿的舒適，這個貼心設計非常適合結合露營和輕徒步的行程，例如簡單的野溪行程、露營行程，當你需要走路又會需要常常穿脫鞋進出帳篷時，可拆式織帶真的幫了一個大忙，而對我來說，Ravine也很適合穿去攀巖，因為要攀巖的時候需要換巖鞋、要幫繩伴確保時需要穿上護腳的鞋子，或是要移動到其他地點時又需要可以走山徑的鞋子，這時可拆式織帶就可以兼顧這些需求！

Ravine 和 Newport H2 定位類似，都很實用

Ravine 超適合穿去攀巖，可調式魔術貼織帶讓穿脫更快速

穿 Ravine 去泡溫泉當然也非常可以，後踝織帶拆掉直接當拖鞋

適合亞洲人的舒適寬楦頭拖鞋與涼鞋，一次擁有！

雖然織帶可拆很方便，但是大家一定要特別小心，不要在有風險的路面把織帶隨便拆掉（例如龍洞巖場），不然就會像我一樣踩到滾動的石頭，拖鞋mode Ravine直接噴飛，腳脫離了KEEN的保護傘就受傷了～ 所以說，好的裝備還需要對的使用方法，才能100%發揮好裝備的功能，不要因為穿著涼鞋之王就掉以輕心！在行走時，一定要把後踝織帶固定好，這樣KEEN才能把你保護得好好的。

在戶外活動，鞋子一定要穿好，不然容易受傷

夏天一起穿著 KEEN 去海邊玩水吧！

KEEN力求突破的精神，從2020春夏創新鞋款就能感受到，除了這些新款以外，當然也要來聊聊每年都大賣、每年都出好多好看顏色讓人想剁手的 Newport H2 和 WHISPER 。

經典中的經典 Newport H2

Newport是 KEEN 最經典的水陸兩用護趾鞋款，是我看過定位最全面的涼鞋款式，首創的護趾楦頭設計，以及織帶下方柔軟服貼的類潛水衣材質內襯，展現了KEEN著重安全的體貼，單隻重量390g，以涼鞋來說算是有點重，但耐磨的橡膠大底有如登山鞋般有較深的刻痕，能夠在更多不同的戶外地形下，抓住地面，人體工學足弓支撐鞋床與厚實的中底設計也是四雙鞋中最具支撐度的，多重密度EVA材質中底，吸震減壓，讓長時間戶外活動的腳底保持舒適感，比起其他涼鞋，更適合從事戶外活動，說他萬用一點也不為過，以我個人來說，最常穿的就是Newport，因為他的各項戰鬥值都很高，而且對我來說，最明顯的需求是我常常需要揹著5-10公斤行走，鞋底的支撐度一定要夠，這一個月以來，我穿著 Newport 去了許多地方，幾個野溪溫泉、徒步步道、還有我最愛的戶外攀巖行程，不得不說，四雙鞋裡面，我最喜歡這雙了，因為穿著他真的各種路面都可以穩穩通過，所以如果你的戶外活動偏多元，那麼你腳上那雙涼鞋非 Newport 系列莫屬！

Newport H2 的鞋底設計適合多元戶外活動使用

從鞋底一看就略知一二，相較其他鞋款，Newport H2 的多向剃刀式斜紋設計抓地力更高，穿越一般山徑的不同路面絕不是問題，在沙灘、礁岩、河床也能安心行走，但是還是要提醒大家，不管鞋子的抓地力再好，還是要注意步伐，跨出下一步之前先看好踩點，要留意會滾動或不穩的石頭，最好避開厚厚的青苔、石磚。

Newport H2 可以勝任各種步道山徑路面

非常適合簡單的單日郊山行程

也很適合爬山和玩水的綜合行程（北得拉曼步道會經過一個小型瀑布）

我也喜歡穿KEEN去攀巖，因為可以保持腳的乾燥涼爽，走去巖場的路上，抓地力良好的大底也能讓我輕鬆通過一些大石頭或是危險地形，最重要的是，KEEN的涼鞋可以保護我的腳趾頭！

穿著 Newport 讓我安心無虞通過亂石

即使穿著 Newport ，也要步步小心謹慎，才能安全通過危險地形

戶外巖場隨處可見KEEN，數數看有幾雙？

另外我個人認為穿著KEEN去巖場，除了透氣之外還有一個最大的優點，也是KEEN最為人知的優點：保護腳趾，因為當確保者在制動時有可能產生位移或飛起來， 穿著合適的鞋，可以讓確保者免於腳部受傷導致確保失效，且當給繩、收繩時，若需前後走動，也可能踩到鬆動的石塊，這時若腳趾受傷、跌倒，可能會把先鋒者拉下來。

穿著KEEN在巖場讓人很安心

我必須說，今年出的花色我超喜歡，織帶是由三原色：紅、綠、藍組成的熱帶感花色，讓穿搭上有很大的自由度，只要是其中一種色系都可以搭配這雙鞋，穿著這雙鞋走路就覺得腳步也跟著有活力了起來！KEEN 的涼鞋之王稱號能在戶外界屹立不搖的原因，就是在花色的設計上一點也不馬虎，每年都會推出不同的花色和鞋面設計。

Newport 鮮豔新色調，陪我迎接熱情的夏日

Newport H2 怎麼搭都好看，去哪裡都可以

輕量化的 WHISPER

WHISPER 雖輕量（單隻 280g），但在足弓的支撐上還是維持不錯的感受，稍微走久一點，腳掌也不會有不適感，多向式虎斑大底是與 KEEN 其他鞋款不同之處，各種類型的地面也都能輕鬆克服。護趾部分的橡膠著重於最前端，也讓 KEEN WHISPER 專為女性設計的立體交叉線條更顯優雅，是護趾系列中最輕量化的鞋款。

多向式虎斑大底能適應各種戶外地形

Whisper 也是一款可上山下海的鞋款

穿著 Whisper 去世界各個角落探險！

米色的 Whisper 怎麼搭都可以

剛柔並濟的 Whisper 搭配性多元

番外篇 EVOFIT ONE

除了WHISPER之外另外介紹 EVOFIT ONE，EVOFIT ONE 的鞋面是彈性編織技術製作的織面，會比織帶款式來得貼合腳步，看起來更偏向陸上運動一些，在足部運動過程中，鞋面可提供高張力和靈活性，緊密貼合腳型，穿著它就像第二層提供保護的肌膚一般，且一樣有透氣速乾的特性，而織帶系列因為比較沒那麼貼腳，洞又比較多，當然更透氣一些。除此之外，EVOFIT ONE 還有一個最大的不同是他的鞋床是可拆式設計，方便清洗，其他機能特點一樣兼具，EVOFIT ONE 一推出就囊括許多獎項，是適合各類型的戶外活動、露營的鞋款，我最常用於輕鬆的一日徒步，一試成主顧！

EVOFIT ONE獨特彈性編織技術製作的織面，能緊密貼合腳型

EVOFIT ONE 當然也是一雙可以陪你上山下海的好鞋

涼鞋之王的稱號一點也不為過，出遊時都是人腳一雙KEEN

介紹了這麼多雙 KEEN 的護趾鞋款，我只能說KEEN能如此受歡迎，不僅是因為使用場合的多元性，更是高科技材質與穿搭外型的完美結合，離開屋，就是你的戶外，希望大家都能找到最適合自己的戶外涼鞋，帶她上山下海，享受大自然的美好！
