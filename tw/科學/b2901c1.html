AACR速遞 | 新型免疫激動劑聯合PD-1單抗在三陰性乳腺癌中顯奇效

2020-10-29
| 科學

受全球COVID-19疫情影響，美國癌症研究協會(AACR)取消了原定於8月下旬召開的年會，決定分別於2020年4月27日-28日和6月22日-24日舉辦線上會議。

本次會議重點聚焦前沿臨床研究、基礎科學以及轉化研究，同時設立“新藥”專題討論。

在AACR在線年會I期間，乳腺癌專場公佈了

一項在三陰性乳腺癌中進行的新型免疫聯合療法，詳情如下：

研究背景

免疫檢查點抑制劑（ICI）單藥療法在轉移性三陰性乳腺癌（mTNBC）患者中獲益有限，特別是在二線+治療中（ORR〜5-6％，mOS 9個月）。在最近的Keynote-119 3期研究中，對2或3線化學難治性mTNBC患者進行了帕博利珠單抗單藥治療，顯示帕博利珠單抗mOS 9.9個月，ORR 9.6％，而化療組mOS為10.8，ORR 為10.6％。

Imprime PGG（Imprime）是一種新型的免疫激動劑，可激活先天免疫系統以重新調節免疫抑制性腫瘤微環境，激活抗原呈遞細胞並刺激抗原特異性T細胞活化。在臨床前模型中，Imprime可顯著增強ICI的抗腫瘤療效。因此，研究人員試圖探討在2L + mTNBC患者中Imprime是否可以增強帕博利珠單抗的療效。

研究方法

IMPRIME 1是一項開放標籤的II期單臂試驗，初步於2019年12月完成。患者接受Imprime（第1、8、15天4 mg / kg IV， 3周為一週期）+帕博利珠單抗200 mg（在每個週期的D1）。在基線開始進行CT掃描，此後每6周進行一次，直至進展。

主要終點是ORR（RECIST v1.1）和安全性。次要終點包括PFS，OS。收集活檢和血液樣本以評估免疫激活。從機理上講，Imprime介導的先天免疫細胞的激活需要抗β葡聚糖抗體（ABA），因此入組僅限於具有足夠的ABA IgG（≥20mcg/ ml）的患者。

研究結果

研究共入組44例患者，與帕博利珠單抗單藥相比，Imprime+帕博利珠單抗：

緩解和疾病控制率（Imprime 1 [95％CI]）：

• 總緩解率：15.9％（7.9- 29.4）

• 疾病穩定：38.6％（25.7-53.4）

• 疾病進展：40.9％（27.7-55.6）

• 疾病控制（CR + PR + SD）：54.5％（40.1-68.3）

• 疾病控制（CR + PR + SD≥24w）：25.0％（14.6- 39.4）

生存

• 中位總生存（月）：16.4（11.1- 23.9）

• 12個月生存率：57.6％（42.4-72.8）

• 18個月生存率：36.7％（21.3-52.1）

在先前接受內分泌治療的mTNBC患者（最初為ER / PR +患者）中，臨床獲益顯著。最早在治療3周時，外周血顯示出先天性免疫激活（例如循環單核細胞上CD86增加）和CD8 T細胞激活（PD1 + / Ki67 + / HLA-DR +）。這些變化與增強PFS和OS獲益相關。對應的腫瘤活檢顯示腫瘤組織被活化的髓樣細胞（Imprime + / PD-L1 + / CD80 +）和活化的CD4，CD8 T細胞（Ki67 + /粒酶B +）強烈浸潤。

Imprime+帕博利珠單抗耐受良好。最常見的不良反應是噁心，背痛，發冷，疲勞，腹瀉，關節痛和頭痛。在3/44例中出現了相關的Gr 3/4 AE。最常見的輸液相關反應大多為Gr1 / 2。

研究結論

IMPRIME 1為Imprime聯合帕博利珠單抗提供了臨床和分子生物學證明。同時為這種新型的免疫腫瘤學組合在mTNBC pts中進行更多臨床研究提供了支持。

觀點

該研究值得關注的一個亞組是12例最初患有激素受體陽性疾病，但在內分泌治療後轉變為TNBC的患者。這些患者在進行新的活檢時均符合ASCO TNBC指南標準。既往曾接受過他莫昔芬，芳香酶抑制劑和/或CDK4 / 6抑制劑治療；並在轉移性環境中接受了超過1種化療方案治療。

在該亞組中，ORR為50％，SD率為33％。在6個月時，DCR為50％。中位OS為17.1個月。

作者表示，目前尚不清楚激素抵抗是否會導致繼發性三陰性患者的反應增強，但這值得進一步研究。
