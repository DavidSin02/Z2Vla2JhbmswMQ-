南寧撩螺傳說，沒有店面的她在北大橋下守候了32年

2020-11-14
| 科學

“對於南寧人來說，撩螺就是夜宵時間永恆的課題”

舊時北大橋下南寧港的碼頭工人，卸下沉重的貨物，總會到附近找個小攤嗦嗦螺卸去一天的疲憊。似乎只要胃滿足了，一切溫暖和幸福就會包圍住全身。

時光荏苒，無數的記憶伴隨著歲月消逝，但在南寧北大橋底有一家名叫“螺婆”的小攤，32年裡連位置都未曾換過。

不會刻意張揚，就這樣安靜地在路邊守護著屬於南寧撩螺界的一枚傳說。

神龍見首不見尾

沒有店面的32年老字號——螺婆

//

擁有這麼多年的歷史，螺婆當仁不讓是南寧老字號，可它一直以來卻連個鋪面都沒有。

每次去吃，都需要手機定位一旁的北大江北路口公交車站，然後還得注意是橋下靠近朝陽這邊的方向，最後再在人群當中認著螺婆的相貌找過去。

這段尋覓過程，生怕出一丁點兒差錯。

對於螺婆而言，她這輩子大概都和螺分不開了。每天洗螺、泡乾淨、加料、煮螺……日復一日，年復一年。

經過我和朋友多次覓食，發現了一個規律，每每到了傍晚六點多，螺婆就會擺出一鍋已經煮好的螺和滷好的鴨腳鴨翅，用煤爐微弱的火保溫，等待一波又一波客人們的到來。

老南寧的回憶一直都在從未離開

//

住在附近的食客和我說起，他第一次和小夥伴們來這棵大榕樹下撩螺的時候還在上住小學，那時候螺雖然只要幾毛錢，但一高興吃嗨了還是會捂捂荷包，怕回家被爸爸媽媽打。

過去在螺婆這裡是用小別針折成的鐵籤來吃螺的，然後吃完插在生薑裡也就當是消毒，生薑也因此而變得千瘡百孔。

現如今為了乾淨，小別針也就變成了一次性的竹質牙籤，裝螺用的瓷碗也變成了不鏽鋼。雖然和回憶裡吃法有了些差別，但也還能適應，畢竟在南寧其他地方也都是用牙籤和不鏽鋼。

4元一碗的螺也有好味道

//

到了物價漲得飛快的現在，看到4元一小碗、5元一大碗的價格，怎麼感覺都好像時光在悄悄倒轉。不過，和小時候不太一樣的是，現在一連來個十碗也不會心疼，這大概就是長大的區別吧。

螺婆這裡一碗螺的價錢雖然便宜，但紫蘇的量卻一點都不吝嗇，清新的香氣瀰漫在每一次咀嚼裡，粒粒酸筍的點綴也讓石螺原本的土腥消失得差不多。

這使得本身不太入味的螺肉，擁有了一抹獨到的鮮美。這也是南寧人愛撩螺的原因之一，獨獨迷戀這特殊的味道！

吃完一碗螺，再舉起碗喝上一口湯，大量石螺化作的螺湯，濃郁的鮮直接充溢到鼻樑。螺婆對於自家的螺湯，也是萬般自信：“想喝湯，免費隨便加，喝到飽都行。”

嗦完一碗不夠盡興，想再加螺，就直接拿著自己吃完的碗喊句：“阿姆，加螺喂。”

螺婆滷出的鴨腳、鴨翅也如同這裡的石螺一般誘惑，淡淡的滷香，微微的鹹，吃的就是屬於上個世紀未完的古早味。

用牙齒輕輕咬上一口，滑滑的有些韌勁的皮就會接連被撕下，舌頭一轉，柔軟的肉完全被攆開的那刻，醬香和滷香就會在口腔裡散開。

旁邊的窗臺上，螺婆早已為你準備好了幾種蘸料，辣椒醬、檸檬醬、黃皮醬、酸醋、醬油一應俱全。

只是簡簡單單一小碗螺卻可以因自己不同而適當的調味而變得豐富多彩。對了，撩過了再多的螺，我也還是覺得再加一點檸檬醬，才會有南寧小時候的滋味。

北大橋底，總有人願為你不斷守候

//

當夜幕完全落下，周圍的攤子也紛紛出來，北大橋底的夜宵店們開始熱鬧非凡。

有人喝著酒猜著碼，幾個朋友嬉笑吹水，快樂的聲音在耳畔此起彼伏，匯成未來對於老南寧的記憶。

看著，螺婆周圍一家家店開了又關關了又開，招牌換了一輪又一輪。

可連攤位都沒有的她，就在這裡對著一鍋鍋石螺煮上了三十多年的光景。

以前，身邊的朋友總說南寧老味道難尋，其實，在不經意之間就有人在街道的一旁，為你留住關於南寧過去的味道。

“友仔，改天去撩螺嘛？”

“走喂！”

螺婆

  • 人均：4元
  • 地址：西鄉塘區北大路2號旁（靠近肥佐燒烤）
  • 營業時間：傍晚18:00-凌晨1:30左右
