附子類方

□附子應用範圍

1. 病程中出現多汗，四肢逆冷，惡寒，沈脈，心音微弱，血壓下降的場合。

2. 嘔吐、下痢伴見手足冷，腹痛、腹滿，脈沈弱，舌苔白滑的場合。

3. 關節的劇烈疼痛、局部腫脹，肌肉拘攣，運動受限，四肢逆冷者。

4. 患慢性疾患，或年老體弱，有下半身冷，腰痠膝軟而無力、或冷痛，足背浮腫，夜尿，頻尿，畏寒，脈沈弱等症者。

5. 慢性腎炎，心功能不全而伴全身機能衰弱症狀，並有浮腫者。

□附子證

1. 精神萎靡，倦臥欲寐。

2. 畏寒感，四肢厥冷，尤其下半身、膝以下清冷。

3. 脈微弱(脈形極細，按之若有若無)，沈浮(重按至骨方能按到)，細弱(脈細如絲，無力)，或脈突然浮大而空軟無力。(附子脈)

1. 患者神識雖然清楚但精神非常疲憊，經常想睡覺，但睡不熟，呼之即醒，狀如朦朧。也有些患者但見疲憊貌，訴說病情時有氣無力，或思維較慢。

2. 比較：以下為實熱證石膏證：身熱、大汗、煩渴、舌幹。黃連證：身熱煩躁、不眠、舌紅苔黃膩。大黃證：身熱便祕、舌乾焦。

□四逆湯

1. 脈微弱、沈浮、細軟、或脈突然浮大而中空軟弱無力(附子脈)。

2. 畏寒，四肢逆冷，尤其是下半身及膝以下清冷不溫。

3. 精神萎靡，倦臥欲寐。

4. 舌質淡或淡紅、暗淡，舌體多胖嫩，舌苔白膩或灰膩，或幹膩、或白滑(乾薑舌)

1. 四逆湯方證的識別以附子脈、四肢厥冷、乾薑舌為重點。

□陰寒體質

外觀特徵：形體偏胖，肌肉偏鬆，皮膚乾燥，面色晦暗或暗黃、缺乏光澤，或有浮腫貌，目精無神，精神萎靡，面帶倦容，脣色暗淡乾枯，舌質胖淡而暗，苔白潤，腹飢鬆軟，按之無力。

好發症狀：平時畏寒喜暖，四肢常冷，尤其是下半身冷，易疲倦，好靜惡動，大便常稀溏不成形，口不乾渴或渴不多飲而喜熱飲，小便清長。

□四逆加人蔘湯

1. 惡寒、四肢厥冷、精神萎靡。

2. 腹瀉或便溏、腹脹或心下痞硬、食慾不振。

3. 脈微弱沈遲，苔幹無津。

《傷寒論》加人蔘均在大汗、大下之後，患者出現脈沈遲、微弱，身體疼痛，舌上乾燥、渴感明顯等症狀。

□真武湯

1. 眩暈，甚至身體站立不穩。

2. 心悸，氣短，肌肉跳動。

3. 身體困重，嗜睡，四肢沉重疼痛，腰痛，關節活動不靈活。

4. 小便少，大便溏，腹脹或腹痛，或噁心嘔吐。

5. 下肢浮腫，面目浮腫。

6. 或咳或喘，痰清稀。

7. 面熱黃或暗，舌淡胖，苔白滑或黑潤。

臨床上只要有以上二、三項症狀，並見以下指徵者，可以使用真武湯。

1. 精神萎靡，倦臥欲寐，畏寒感，四肢冷，尤其是下半身，膝以下清冷。

2. 脈微弱、沈伏、細弱。

1. 茯苓、白朮是治療小便不利、浮腫的必用藥物。

2. 芍藥是治療腹痛、腳攣急的藥物，常與甘草同用。

3. 真武湯證是附子證加小便不利、浮腫、腹痛、腹滿等症。

4. 比較

相同

相異

附子證辦水飲證，眩暈、心悸的同時有惡寒、精神萎靡、脈沈微弱、腹滿腹痛、四肢沉重疼痛等症。

桂枝證伴水飲證，眩暈、心悸的同時有氣上衝胸、心下逆滿等症，且發病甚急，常因為精神刺激誘發，過則相安無事。

□附子瀉心湯

1. 心下痞，或上腹部脹痛，大便不通或裡急後重，心煩亂甚或昏厥，舌苔黃膩。

2. 惡寒汗出或額上冷汗，肢冷，脈沈微弱或血壓下降。

本方證一方面是腹滿、上腹部痞滿脹痛、大便不通或裡急後重、心煩亂、舌苔黃膩的大黃黃連證；一方面則是惡寒四肢冷，出冷汗，脈沈微弱的附子證。

乾薑類方

□乾薑證

1. 嘔吐物、唾液、痰液、大便、尿液清稀，無惡臭。

2. 腹脹、腹痛，噁心，嘔吐，或咳喘。

3. 口不乾渴，惡寒喜熱，精神萎靡。

4. 舌質淡或淡紅，舌上有膩苔，苔多白膩，或灰黑膩，或白滑(乾薑舌)。

1. 屬於裡寒證。裡只臨床表現以胃腸症狀為主，寒指機體新陳代謝低下，臨床表現為嘔吐物、瀉下物清稀無臭，痰液如水樣，患者惡寒喜熱，面色青黃，肢冷身寒，聲低氣微，舌淡白，苔白膩。

□理中湯

1. 腹滿腹脹，嘔吐下利，大便稀溏，食慾不振，心下痞硬，或涎唾多而清稀。

2. 畏寒喜溫，精神萎靡，口不乾渴或口乾而不思飲。

3. 舌質淡紅，苔白、或厚、或膩、或滑。

1. 甘草乾薑湯：治療裡寒證的煩躁、吐逆、及肺中冷、眩暈、多涎唾。

2. 附子理中湯：加附子，治理中湯證兼見四肢厥冷、精神萎靡、脈為弱者。

3. 連理湯：加黃連，治理中湯證兼見煩躁、心下痞痛、舌紅苔黃膩、脣暗紅者。

4. 桂枝人蔘湯：加桂枝，治理中湯證兼見自汗、身痛、惡風者。

5. 枳實理中丸：加枳實、茯苓，治理中湯證伴見腹脹滿拒按者。

6. 治中湯：加青皮、陳皮，治理中湯證兼見胃脘脹氣者。

□桂枝人蔘湯

1. 理中湯證。

2. 發熱自汗，或腹痛，心下痞硬。

3. 舌質淡紅或嫩紅，偏暗，脈浮大。

1. 桂枝體質患者的慢性胃腸疾病多見此證。

2. 理中湯證伴見自汗、惡風是常見的，自汗中有一動即汗出者，也有入夜汗出者，還有僅見惡風膚潤而無明顯出汗者；自汗者大多惡風，對風冷過敏。

黃耆類方

□黃耆證

1. 自汗，盜汗，惡風，身體重，肢體麻木不仁。

2. 浮腫，小便不利。

3. 潰瘍久不收斂，膿水清稀。

1. 自汗、盜汗而浮腫是黃耆證的特徵。

2. 大部分患者伴見四肢發冷、畏風、易感冒、下肢浮腫、小便少，自覺身體沉重，動作不靈活。

3. 有時並不表現為明顯的自汗，但通過問診，可以瞭解到患者平素出汗比較多，稍加勞動，即易出汗，或曾有自汗、盜汗史。

4. 比較：桂枝證自汗的同時，常伴有發熱或身熱感、失眠、頭痛、動悸等。桂枝體質多體瘦而肌肉堅緊，而黃耆體質多肌肉鬆軟而浮腫。

□黃耆體質

外觀特徵：面色黃白或黃紅隱隱、或黃暗，缺乏光澤。肌肉鬆軟，浮腫貌，目少精彩，面色無華。腹壁軟弱無力。舌質淡胖，苔潤。

好發症狀：平素易出汗，畏風，遇風冷易過敏，或咳嗽氣喘，或鼻塞，易感冒。大便稀溏、不成形、或先幹後溏。食慾不振，易腹脹滿。易浮腫、特別是足腫，手足易麻木。

體質傾向：多風，多溼。

□玉屏風散

1. 易於感冒，畏風，自汗，打噴嚏，身痛或頭痛。

2. 浮腫或易於浮腫，小便少，大便溏薄。

1. 本方是治黃耆體質的多汗、易感冒方劑。

2. 黃耆主浮腫而自汗，白朮主浮腫而小便不利，防風主畏風而身痛、頭痛等。

麻黃類方

□麻黃證

1. 發熱惡寒，頭痛，骨節痛，身痛。

2. 無汗，咳喘，鼻塞。

3. 浮腫，小便不利。

1. 麻黃證中，無汗一證最為重要。

2. 麻黃證的浮腫、小便不利是並見的。

□麻黃體質

外觀特徵：體型略胖，肌肉比較發達或偏鬆，膚色黃或淡黃色、或淺黑色，皮膚較粗、較乾燥，平時不易出汗，身體比較壯實。血壓不高。脣暗或紫紅、舌體偏大，舌質淡紅，口不幹、苔白。

好發症狀：易閉汗或汗出不暢，易受寒，易喘，易鼻塞流清涕，肌肉酸重感，全身倦怠感，感覺不敏感，心下部重壓感或腹脹，頭重感，有浮腫傾向。

體質傾向：多寒、多溼。

黃胖或黑胖者，多可以使用麻黃及麻黃劑。

比較：若體型消瘦、肌肉堅緊、面紅赤、身熱多汗、舌質紅，或血壓高、心動過速者，麻黃及麻黃劑要慎用。

□麻黃湯

1. 惡寒發熱，頭痛身痛。

2. 無汗而喘。

3. 脈浮緊，舌暗淡。

1. 惡風是風冷過敏、遇風則自覺怕冷，若覆被或移至溫暖處則沒有這種感覺。惡寒是體溫升高前的寒冷感，雖覆被加衣不解，而且通常伴有發熱。

2. 桂枝湯證自汗，患者皮膚溼潤；麻黃湯證無汗，患者皮膚乾燥。

3. 麻黃湯證患者體溫一般不超過39度C。

4. 頭痛身痛：麻黃湯證患者在惡寒發熱同時，往往訴說周身肌肉痠痛或關節疼痛，有的尚感到腰部特別痠痛。

5. 類方：麻黃加朮湯：治麻黃湯證兼見浮腫及肌肉酸重的關節炎、腎炎、感冒。但對體型偏胖、皮膚較白的中老年婦女，常因關節痛、浮腫、肌肉酸重而誤診為本方證，臨床需注意。(可參見黃耆類方)

6. 禁忌：對瘡家、淋家、衂家、亡血家、表虛自汗、脈尺中遲、誤下而見身重心悸者禁用麻黃湯。

□麻黃附子細辛湯

1. 無汗，惡寒感明顯，發熱或不發熱。

2. 精神萎靡，倦怠感明顯，面色晦滯而缺乏光澤，手足冷。

3. 脈沈弱、或沈細、或沈遲。

4. 舌質淡苔白潤。

1. 麻黃證伴見脈沈遲(附子證)。表實裡虛。

2. 本方對體質素弱的老年人及病後體弱者的感冒，以整個背部自覺難以名狀的畏寒為主要特徵的患者療效頗佳。

□小青龍湯

1. 咳喘，痰液呈水樣或黏液性，量較多，或鼻塞、打噴嚏、流清水樣鼻涕。

2. 惡寒，特別是背部有顯著的冷感，發熱或不發熱，平時無汗，咳喘時可有汗出。

3. 苔白滑。

1. 痰與鼻涕的性狀在本方證的診斷中十分重要。其痰液、鼻涕等分泌勿必須是量多清稀，成黏液性的，或如水樣。同時舌苔白滑，黏液滿布。

2. 惡寒是必見症狀。

□麻杏甘石湯

1. 發熱，汗出時多時少，體溫或升或降，口渴。

2. 咳喘，甚而氣急鼻扇，胸悶。

3. 脈滑數，苔薄膩較幹。

1. 熱喘：咳嗽氣喘、身熱汗出、痰液黏稠色黃、口乾渴、舌紅苔黃。

2. 石膏藥證為身熱、口渴、汗出。

3. 比較麻杏甘石湯麻黃湯相同喘喘相異

1. 汗出而喘。

2. 患者面色紅潤。

3. 雖無大汗，但撫之皮膚溼潤，故捫之不灼手。

4. 口渴能飲水，苔幹。

1. 無汗而喘。

2. 患者面色時而發赤，時而蒼白。

3. 肌膚粟起，撫之乾燥，絕無汗意。

4. 口不渴苔潤。

□陽和湯

1. 局部：皮膚無熱感，色蒼白、或紫暗，漫腫無頭，按之或堅硬如石、或空軟如絮，自覺不甚疼痛或不痛、或麻木、或酸楚。

2. 全身：精神萎靡，或畏寒喜溫，或腰脊空痛，或氣短喘息，或陽痿陰冷，皮膚蒼白松弛憔悴，枯瘦，貧血，不口渴，小便清長，脈沈遲細，舌質淡嫩。

陰疽臨床表現為局部漫腫無頭、皮色不變、不熱，患者面色無華，喜溫，口不渴，小便清長，舌苔白，脈沈緩或遲或細。

桂枝類方

□桂枝證

發熱或自覺熱感，易出汗，甚或自汗，惡風，對寒冷感覺敏感，關節痛。自覺腹部有上衝感或搏動感，動悸，易驚，烘熱，失眠。

1. 發熱大多是低熱，或僅僅有自覺的熱感，同時伴有出汗、惡風、怕冷。

2. 常見患者的腹部皮膚及手心比較溼潤。

□桂枝體質

外觀特徵：體型偏瘦；皮膚比較白，紋理比較細；肌表溼潤，肌肉較硬；腹部多扁平，腹肌比較緊張；目有神氣，脣淡紅或暗；脈象常浮大，輕按即得：舌體柔軟淡紅或暗淡，舌面潤，苔薄白。

好發症狀：易出汗，或自汗，或盜汗，或手足出汗；對寒冷、疼痛及心理刺激敏感；易傷風感冒，易腹痛，易心動悸，睡眠淺或多夢，易便祕，易肌肉痙攣。

體質特徵：多風、多熱。

桂枝舌：桂枝體質舌象大多表現為舌質暗淡、暗紅，甚或紫暗，但質地柔嫩而潤澤。

□桂枝湯

1.自汗，惡風，發熱或自覺熱感。

2.上衝感，動悸，肌肉痙攣拘急。

3.脈浮，或虛、或緩、或數、或大而無力，舌質淡紅或暗淡，苔薄白。

1. 自汗：是指自動汗出，雖天氣不炎熱，也不運動，也未服發汗藥物而依然汗出者。

2. 惡風：只對風冷過敏，在溫暖的居室或多加衣服可以緩解。

3. 發熱：可以是體溫升高，也可以是自我感覺周身發熱。

4. 上衝感：包括頭暈、烘熱、面紅、失眠、多夢、胸腹有氣上衝感、臍腹部的搏動感。

5. 桂枝湯證的脈象不僅浮，而且常並見緩象，即和緩不數，或並見虛象，即按之少力或無力，有時也可見虛數脈象。

類方

1. 加附子：桂枝加附子湯，治桂枝湯證見關節劇痛，冷汗多者。

2. 加厚朴、杏仁：桂枝加厚朴杏仁湯，治桂枝湯證見胸滿腹脹、咳嗽、喘息、痰多者。

3. 加大黃：桂枝加大黃湯，治桂枝體質便祕。

4. 加黃耆：桂枝加黃耆湯，治自汗、盜汗、黃汗，見浮腫、小便不利者。

5. 加人蔘：桂枝新加湯，治汗多、心悸、頭昏、食慾不振者。

6. 加葛根：桂枝加葛根湯，治桂枝湯證見項背拘急或下利者。

□桂枝加附子湯

1. 桂枝湯證見冷汗、皮膚溼冷、過汗，脈弱浮大，舌質淡。

2. 桂枝湯證伴見關節疼痛較劇、四肢居攣者。

3. 寒疝、腹痛、體痛、手足冷等見脈沈遲，舌質淡者。

桂枝湯證若虛的程度更重，出現汗出不止、皮膚溼冷、惡寒、手足拘攣疼痛、舌淡脈遲時，用之。

類方

加白朮：桂枝加朮附湯，治關節腫痛；若關節腫痛伴浮腫者，可以選用桂枝芍藥知母湯。

□小建中湯

1. 慢性腹痛伴動悸、煩熱、虛弱，腹部扁平而肌緊張。

2. 舌質嫩苔少。

1. 虛勞急痛：疼痛多為陣發性得痙攣性疼痛，但程度不劇烈而經常發作，同時伴有心動悸、煩熱、多夢、鼻衂等症狀。

2. 腹證：腹部可以見腹壁薄，腹直肌緊張。

3. 舌質嫩，是指舌質柔軟而有光澤。比較：舌質堅老而苔厚者，多表示體質充實，內有實熱、瘀血。

方根

芍藥甘草湯：解痙攣、止痛作用，對胃腸道平滑肌痙攣和腓長肌痙攣有效。疼痛劇烈時，白芍可用至30-45克。

類方

1. 加黃耆：黃耆建中湯，適用小建中湯證見貧血、自汗、易感冒者。

2. 加當歸：當歸建中湯，適用於女性產後體痛、腹痛及月經痛等。

3. 去飴糖：桂枝加芍藥湯，對病情不宜甜食或呈現急性化傾向、腹痛程度較重的患者較適宜。

□桂枝加大黃湯

1. 發熱或自覺熱感，惡風，自汗。

2. 便祕，持續性腹痛、拒按，舌苔厚幹。

1. 桂枝體質表易虛而裡易實，表虛自汗、多汗，裡實則便祕、腹痛。

2. 治桂枝體質的習慣性便祕、腹部術後的排便不暢、或伴有裡急後重的痢疾等。

3. 對桂枝體質患者的胃痛、咳喘、發熱、頭痛等病症見便祕燥結、腹痛、舌苔幹厚者。

□桂枝加龍骨牡蠣湯

1. 胸腹動悸，易驚，自汗盜汗，失眠多夢。

2. 脈浮大而無力，舌質嫩紅，苔少。

1. 臨床常見那些面白體瘦的兒童，目睛雖然有神而多易驚、夜啼不安、且多汗；那些白瘦體弱的青年男女，也多見心腹動悸、易失眠、多夢、易盜汗。可以考慮本方證是否存在。

2. 決定本方能否應用的關鍵是脈象與舌象：脈必見浮露、大而無力；若沈細、沈實或大而有力，均非本方脈象。舌質嫩紅、溼潤、舌苔薄者可用。這表示正氣虛而內無邪。比較：舌質暗紅堅老者，為裡有鬱熱；舌質淡白胖大，為裡有寒溼水飲。舌苔黃膩，表痰熱；舌苔焦乾，表積熱：舌苔厚膩，表溼濁。

□當歸四逆湯

1. 手足厥寒、麻木、冷痛，甚至青紫。

2. 脈細。

3. 腹痛、頭痛，或腰痛、腿痛、腳痛。

4. 舌淡苔白。

1. 手足冷：以指尖為甚，雖夏天亦陰冷異常。

2. 脈細：為血管收縮的緣故，並非心臟機能衰弱，故全身狀況比較好。

3. 疼痛：勢必見症狀，大多欲冷更劇。女性可見月經期間症狀加重。

4. 舌淡苔白：表示寒證存在，是臨床重要鑑別點。

5. 類方加吳茱萸、生薑：當歸四逆加吳茱萸、生薑湯，治療當歸四逆湯證疼痛較甚，伴見嘔吐、噁心、頭痛，尤其是巔頂痛者合適。

□ 溫經湯

1. 月經不調，血色暗有血塊。

2. 自覺手心熱而又惡風、自汗，午後有發熱感，或有頭痛、噁心。

3. 腹壁薄而無力，小腹拘急、疼痛或腹脹感。

4. 口脣乾燥，舌質暗淡。

1. 桂枝體質使用率較高。

2. 月經不調：所指病症有月經先後無定；或逾期不止，漏下不斷；或一月二行，也有更年期月經不調者。

3. 關鍵是問血色、血質：多見血色暗淡，有血塊。比較：若血色鮮紅或紫紅、質黏稠者，屬內熱甚。

4. 本證自覺發熱，手足心煩熱，口脣乾燥，似為熱證，但惡風、自汗、舌質暗淡、小腹拘急疼痛，顯然是寒證。

□炙甘草湯

1. 脈虛無力、三五不調。

2. 消瘦、面色憔悴，舌苔薄或無苔。

3. 倦怠、動悸、虛煩、頭昏、多夢或失眠、便祕。

1. 桂枝體質患者的心臟血管疾病，通常可以考慮炙甘草湯。

2. 人體的骨肉形體，為有形之物，故消瘦、枯燥、乾癟、細小為陰虛。

3. 適合陰虛證的心率失常，多見於桂枝體質，形體消瘦且面容憔悴。

4. 脈律不齊，是炙甘草湯的代表脈證，但脈虛無力是其特徵。如果脈象滑大，多表示裡有痰熱，當用溫膽湯類方：脈象鬱澀不暢者，則表示裡有瘀血，當用血府逐瘀湯。

5. 炙甘草湯證多見於”桂枝體質”，形體消瘦且面容憔悴。比較：體型肥胖、面色暗赤或油膩者，多為痰熱證或痰溼證。

□苓桂朮甘湯

1. 心下動悸，或氣上衝胸，或眩暈。

2. 腹部軟弱而胸脅部脹滿，胃內有振水音。

3. 小便不利，浮腫傾向。

1. 水液停留不化的主要表現：眩暈、動悸、胃內振水音、背中冷、咳嗽痰多清稀、胸脅支滿、小便不利、苔滑等。

2. 常可見患者惡寒、小便不利、大便溏薄、口但無味、常吐清唾、苔白滑。

3. 本方證有發作無定時、時好時壞的臨床特徵，即發作時各種症狀甚劇，來勢頗猛，但去後則相安無事，而精神刺激、身心疲勞常是引發本證的誘因。

4. 比較

茯桂甘棗湯

因大棗有緩急安神的作用，故本方對動悸、衝逆等症最適合。

五苓散

利水作用較苓桂朮甘湯強，臨床見浮腫、小便不利、嘔吐、眩暈、動悸等症。

桂苓甘露飲

多用於發熱性疾病伴見小便不利、嘔吐、頭痛、發熱、煩躁口渴。

□桂枝茯苓丸

1. 下腹痛，按壓則痛甚，或有包塊。

2. 頭痛昏暈，睡眠不良，煩躁，動悸。

3. 面紅或紫紅，舌質暗、或有紫點。

1. 瘀血臨床表現：疼痛部位固定，出血易凝固、紫黑，精神不安、煩躁、甚至發狂，舌質紫暗、面色晦暗。

2. 下腹部特別是臍的左腹部(少腹部)得壓痛與包塊，是瘀血特徵之一。

3. 臨床若見患者面色暗黃、舌質紫暗時可採用腹診，有時雖然疼痛不明顯或無明顯包塊，但有明顯的抵抗感疾患者不適感，在左少腹稍重按壓的話，患者可有疼痛感。

4. 除腹證外，患者的面色暗紅與暗黃、脣色暗紅、舌質暗或有紫點，及目睛充血對診斷本方證意義較大。

5. 臨床對女性患者應詢問其月經的色質及有關症狀。若月經色暗或黑、易凝固有血塊，並伴腹痛者；或月經前頭痛、乳脹、煩躁、易怒者，均應考慮瘀血證的存在。

2. 便祕或大便乾燥難解。

3. 舌苔厚、舌面幹膩、舌質暗或有瘀點。
