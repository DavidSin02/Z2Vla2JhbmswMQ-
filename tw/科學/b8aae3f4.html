八月上映的電影，依舊很爛，但能看的比七月多了兩部。

2020-11-14
| 科學

本文為微信公號電影天堂原創，轉載請聯繫授權

8月還處於保護月的保護之中，雖然院線依舊擁堵，爛片依舊扎堆。

但畢竟，能看的比七月多了幾部。

所以，一起來看下八月影市，小心別讓爛片辣了你的眼

﹁

八月院線前瞻

﹂

《愛寵大機密》

上映日期：8月2日

導演: 克里斯·雷納德 / 雅候·遲內

主演: 路易·C·K / 艾瑞克·斯通斯崔特 / 凱文·哈特 / 艾麗·坎伯爾 / 史蒂夫·庫根

類型: 喜劇 / 動畫

小黃人團隊原班人馬製作，選的愛寵都是胖乎乎圓滾滾的，所以不必擔心萌不到你。

電影在7月8號北美上映，兩天累計一億票房，打破了由《頭腦特工隊》創造的票房紀錄，成為原創動畫最高首周。

媒體影評人放出的評價也尚可， 不求內涵豐富，只圖歡樂氛圍的話，真的值得一看！

另外，不小心選到國配版也不用過於沮喪，《愛寵》這次的國語配音版除了何炅之外，還請來了陳佩斯老師位反派小白兔配音~

《龍拳小子》

上映日期：8月2日

導演：陶聲 王瑋主演：劉芮麟 童菲 林秋楠 樑超 吳永倫 餘薇薇 王皓禎 陳浩民 金巧巧

類型: 喜劇 / 動作

一個功夫小子拯救了世界的故事。

《我最好朋友的婚禮》

上映日期：8月5日

導演：陳飛宏

主演：舒淇 馮紹峰 宋茜

類型: 喜劇 /愛情

翻拍自好萊塢同名經典喜劇，舒淇接演大嘴朱莉婭·羅伯茨那個角色，宋茜演卡梅隆·迪亞茲演得那個白富美。

珠玉在前，總少不了對比，並且這幾年我們翻拍的《新娘大作戰》《命中註定》等作品質量都不行。觀影需謹慎啊。

《盜墓筆記》

上映日期：8月5日

導演：李仁港

主演：鹿晗 井柏然 馬思純

類型: 奇幻 / 冒險

兩大熱門IP，去年火的是《鬼吹燈》，今年輪到了《盜墓筆記》。

雖然《盜筆》網劇撲街，但這次影版來自南派三叔徐磊親自捉刀編劇，讓不少書粉稍稍安心。

演員選用鹿晗、井柏然兩大人氣偶像，票房的號召力不必擔心。

質量的話，本片導演為李仁港，他的作品有：鴻門宴傳奇、錦衣衛、三國之見龍卸甲。

《謊言西西里》

上映日期：8月5日

中韓合拍片。

導演是憑《翻滾吧，阿信》一片聞名華語影壇的臺灣導演林育賢，監製為拍過《藍宇》的香港著名導演關錦鵬。

幕後大咖來自韓國，《海雲臺》的攝影、《黃海》的燈光師。

主演周冬雨、阮經天、李準基也都算有演技的。

但看預告片似乎還是“車禍、癌症、治不好”這種韓劇老三寶的梗啊

《古曼》

上映日期：8月5日

導演: 張江南

主演: 韓雪 / 唐宸禹 / 吳藝璇 / 朱聖禕

類型: 驚悚

看海報，看主演。

看導演，

上映日期：8月9日

導演: 陳偉強 / 王惜今

主演: 胡夏 / 黃聖依 / 王惜今 / 陳籽熹 / 朱時茂 /保劍鋒

類型: 喜劇 / 愛情

《使徒行者》

上映日期：8月11日

導演: 文偉鴻

主演: 張家輝 / 古天樂 / 吳鎮宇 / 佘詩曼 / 許紹雄

類型: 喜劇 / 犯罪

熱播電視劇《使徒行者》的電影版本，劇版的評價不錯，豆瓣評分8.1

影版換了主角大換血，陣容倒是更強大了，導演還是文偉鴻，如果能保持風格的話還好，不然可能會跟影版《衝上雲霄》一樣慘淡。

類型: 懸疑 / 驚悚

《微微一笑很傾城》

上映日期：8月12日

導演：趙天宇

主演：井柏然 楊穎 李沁 王思聰 劉循子墨 譚鬆韻

類型: 愛情

也有《鍾馗伏魔》

不過看到張一白監製可能就更放心一點了是吧~

《危城》

上映日期：8月12日

導演: 陳木勝

主演: 劉青雲 / 古天樂 / 彭于晏 / 吳京 / 袁泉

類型: 動作 / 武俠 / 古裝

講述軍閥割據的亂世時代，發生在一個小城裡的恩怨情仇。

演員上齊聚兩岸三地多位大牌，槍戰和打戲也是一大看點，看過的盆友反應武戲尚可，文戲略生硬，總體還算差強人意。

Whatever，“當彭于晏顫著大胸騎著馬兒向你走來，你還能拒絕什麼....”

《我們誕生在中國》

上映日期：8月12日

導演：陸川

解說：周迅

類型: 劇情 / 紀錄片

迪士尼跨國團隊製作的野生動物題材的紀錄片，由陸川執導。

秀美山河，可愛動物，值得一看。

《海洋之歌》

上映日期：8月12日

導演: 湯姆·摩爾

類型: 劇情 / 動畫 / 奇幻

主演: 大衛·羅爾 / 布萊丹·格里森 / 麗莎·漢尼根 / 菲奧紐拉·弗拉納根

奧斯卡最佳動畫電影提名。歐州上映以來好評如潮！

豆瓣評分8.8，,好於94%動畫片，97%奇幻片！

手繪畫風美膩，任意一個截圖都可以當屏保。

隔了兩年，終於有機會在電影院的銀幕上一覽風采，我的本月必看。

《美好意外》

上映日期：8月12日

導演：何蔚庭主演：桂綸鎂/王景春/歐陽娜娜/王元也/陳坤

類型: 喜劇 / 奇幻

翻拍自韓國電影《妻子小姐》，講述一位都市白領一夜之間變成普通家庭主婦的故事。

奇幻設定，差不多跟《重返20歲》是同樣的路線，電影雖然有桂綸鎂和陳坤，但剛被歐陽娜娜的“鹿小葵”荼毒過的觀眾，估計不想在電影院裡再見她一次了。

《勇士之門》

上映日期：8月12日

導演: 馬蒂亞斯·霍恩

主演: 趙又廷 / 倪妮 / 尤賴亞·謝爾頓 / 吳鎮宇 /戴夫·巴蒂斯塔

類型: 奇幻 / 冒險

中外合拍的奇幻動作冒險電影。

由呂克·貝鬆編劇及監製。

這個劇情設置，演員搭配，欸？好熟悉，不是《功夫之王》嗎？

  • 《勇士之門》：廢柴男孩（尤賴亞·謝爾頓 Uriah Shelton 飾）機緣巧合地開啟了異世界的大門，捲入了一場驚心動魄的種族鬥爭。

  • 《功夫之王》：弱雞少年傑森（Michael Angarano 飾）從某中國古董店得到一根如意金箍棒，藉著金箍棒的力量，他竟然回到了幾千年前的中國。

《精靈王座》

上映日期：8月12日

導演: 宋嶽峰

類型: 動畫

人氣網遊《龍之谷》改編的動畫大電影，2014年的首部《龍之谷：破曉奇兵》口碑爆棚，當時被譽為國產3D動畫的巔峰之作。

兩年之後，原班人馬重新打造的續集《精靈王座》，你會去看嗎？
