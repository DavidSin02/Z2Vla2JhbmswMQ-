《DK博物大百科》

作者：英國DK公司 [編著]

出版社：科學普及出版社

原作名：Natural History

譯者：王敏潔 / 朱坤 / 吳海峰 / 何長歡 / 張興春 / 張勁碩 / 林然 / 鄭浩然 / 姜景一

簡介

我們的雜誌就叫《博物》，可能很多人都會有這個疑問：博物是什麼意思呢？一般來說，博物的範圍主要包括對動物、植物和礦物的基本認識和了解，“博物館”則是收集、展示“博物”的場地，而這本書就像是一本紙上的博物館，向我們展示了多樣的大千世界。

在這本書的譯者團隊中，有三位都是我們《博物》的成員，他們是吳海峰、何長歡和特約校對張勁碩。在與他們聊到參與翻譯工作的心得時，編輯得知這本書早在2012年就已經開始了翻譯工作，而到了2018年年底才上市。

為什麼會翻譯這麼久呢？因為翻譯只是其中一個環節，編輯們還需要對所有文字進行科學審校，讓譯文既能傳達原意又不失科學性。舉個例子：原文中有一種鸚鵡的圖注是“glossy parrot”，如果直譯就會變成“光滑的鸚鵡”，但這肯定是不對的。因此要結合這種鸚鵡獨特的外觀，翻譯為“體羽具金屬光澤的鸚鵡”才更為合適。

這本書足有7公斤重，當屬本月甚至是本年度書單中的NO.1，而它能帶給大家的知識量絕對會遠超這個數字。

《世界野生貓科動物》

作者： [澳] 盧克·亨特 / [英] 普瑞西拉·巴雷特出版社：湖南科學技術出版社

出品方：中國國家地理譯者：貓盟

簡介

每每看到“貓”的臉，就覺得造物主真是個大藝術家。有人說，聽說自然與人類之間需要一個信使，所以貓科動物就出現了。這當然只是一個美好的想法，但卻剛好體現了貓科動物的不同之處。

貓科動物的自由和美，成全了我們作為野生“貓奴”的本命。世界共有野生貓科動物40種，其野性和美令人目眩。40種野生貓科動物，40種荒野家園，這本書會讓你更瞭解這些人類的最美鄰居，過足貓奴的癮。

《奇趣動物寶典》

作者：史蒂夫·詹金斯

科學審校：吳海峰

出版社：北京少年兒童出版社

簡介

在本月的書單中，這本可能是最適合小朋友閱讀的。首先，它內容的編排方式很特別，不是按照動物類群或者生態環境給動物分類，而是按照小朋友提問的方式來編排的。例如，在動物如何感知這個世界這個部分講了許多種動物的視覺、聽覺、嗅覺等等。

其次，很多這類書中要麼是手繪圖，要麼是攝影作品，而這本書中則是拼貼畫。作者用不同材料、質地、顏色的紙張剪出動物的不同部分，之後再把它們一塊一塊、一層一層地拼貼起來，最後再拍照印在書上。因此，這本書不單單是一本動物圖書，更是一本手工書，讓小朋友們在學習知識的同時可以收穫快樂。

《國家地理珍稀鳥類全書》

作者：[美] 喬爾·薩託 / [美] 諾亞·斯特瑞克

出版社：江蘇鳳凰科學技術出版社

譯者：胡晗 / 王維

簡介

在這本書的封面上，最吸引人眼球的恐怕是這隻五彩斑斕的鳥，而很多人會忽略了PHOTO ARK 這幾個字。

“影像方舟”（PHOTO ARK）是由攝影師喬爾·薩託在十多年前發起的項目，他的目標不是拍攝那些生活在野生環境中的動物，而是拍攝“圈養動物”——生活在動物園和水族館裡的動物。該計劃已經在世界各地的動物園和水族館中拍攝了5400種動物的肖像照，喬爾·薩託表示，在拍完12000種圈養動物之前他不會罷手。

為什麼會選擇這樣的一群動物呢？在喬爾·薩託看來，很多物種已在野外滅絕，而動物園和水族館成了它們真正的賴以生息繁衍的“方舟”。

《藍色星球II》

作者：[英]詹姆斯·霍尼伯內/ [英]馬克·布朗羅出版社：江蘇鳳凰科學技術出版社

譯者：李蕊 / 印思玫

簡介

當年《藍色星球2》的紀錄片剛一推出，就立刻登上了各個播放平臺的榜單。它有什麼厲害之處呢？在長達4年的拍攝過程中, 紀錄片的製作團隊共執行了125次的探險，水下拍攝時數長達到6000多個小時，以全新的拍攝技術，呈現出前所未有的畫面。如果你是一個痴迷于海洋的人，我覺得你不應該錯過這部優秀的作品；而如果你已經看過這部紀錄片，這本基於片子的書也是值得一讀的。

書中不僅有200多幅來自紀錄片的4K高清圖片，還展示了眾多海洋居民的棲息地原貌，通過講述令人動容的海洋生命故事，系統地介紹了海洋知識，並獨家披露鏡頭之後的攝製過程。

《瀕危》

作者： [英] 蒂姆·弗拉克 攝影 / 喬納森·貝利 薩姆·威爾斯 撰文

出版社：天津人民出版社

出品方： 後浪

原作名：ENDANGERED

譯者： [美] 林肯

簡介

準確來說，這本書更像是一本攝影集。蒂姆·弗拉克是一位動物攝影大師，在他的鏡頭下，動物們或沉默，或靜思，或優雅，或深沉，具有一種奇特的美感，而我們也彷彿可以從它們的眼神感受到情感，和它們對話。

人和動物之間到底是怎樣的一種關係呢？也許很多科普作者都在用文字向我們講述他們各自的理解，但有時候最直觀的感受來自於視覺。攝影師蒂姆·弗拉克通過記錄瀕危物種和其生存境遇的特別項目，探索了我們這個時代最緊迫的問題之一。

《BBC科普三部曲》

作者：[英] 保爾·羅斯 安妮·萊金

出版社：重慶出版社

譯者：叢言 / 胡嫻娟 / 陳瑤

簡介

這套書有三本，分別是《地球：行星的力量》、《海洋：深水探祕》和《生命：非常的世界》。先拿《海洋》這本來說，雖然上面也有一本BBC講海洋的書，但其實它們完全不是從一個角度來解讀的，《藍色星球2》更偏重介紹海洋動物，而《海洋》則是帶你探索世界七大洋各自的祕密。

《地球》這本是探索地球如何運轉，生命如何孕育，逐一介紹塑造地球的四種力量：火山、大氣層、海洋與冰川；《生命》則是為大家展現了世界每個大陸上的動物和植物為了生存而使用的各種策略，看看它們有多厲害吧！
