AI也會相面？顏值是如何影響命運的？

2020-11-14
| 科學

“看面識相” 乍聽上去，給人一種不太科學的感覺，但自古以來，中西方各個時代對於“看面識相”都有所討論，且所持態度褒貶不一。從古希臘的相貌學家到意大利犯罪學家、精神病學家龍勃羅梭（Cesare Lombroso），人們都曾試圖揭開人的面部外貌與性格之間的關聯性，但大多數想法都沒能經得起現代科學的檢驗。


近日，來自俄羅斯國家研究型高等經濟大學和人文與經濟開放大學的研究人員聯合BestFitMe ，發佈了一項關於“看面識相”的最新研究成果，這項研究提供了新的證據，進一步證實了人類的個性與個人的面部特徵有關。而且人工智能能夠比 “人類 ”評估者更好地根據 “自拍照” 照片作出人物性格推斷。



這項研究成果於 5 月 22 日發表在 Scientific Reports 上，論文題為 “用真實的靜態面部圖像評估五大人格特徵（ Assessing the Big Five personality traits using real-life static facial images ）” 。


研究人員表示，基於這一研究發現，大量的潛在應用有待開發。在高速度和低成本和高準確度的情況下，從真實照片中識別人物性格可以補充傳統的人格評估方法。人工智能的這一應用可以用來尋找最適合客戶個性的產品，比如客服、約會或在線家教等活動中，為客戶選擇 "最佳匹配者"。



科學“看臉”算命的依據


看面相算命，相信大家都不陌生了。但是在這個“看臉”的時代，隨著人工智能的發展，又催生了一個新生業態 —— “ AI 看面識相 ”，AI 性格分析，AI 性格匹配 .... 各種相關應用也大受年輕人的追捧。


在人格科學研究領域，人格結構中的五個因素被稱為 “大五”（Big Five），五種人格特質分別是：

  • 神經質（N）——情緒穩定性：焦慮、敵對、壓抑、自我意識、衝動、脆弱
  • 外傾性（E）——外向性：熱情、社交、果斷、活躍、冒險、樂觀
  • 經驗開放性（O）——開放性：想象、審美、情感豐富、求異、智能
  • 宜人性（A）——隨和性：信任、直率、利他、依從、謙虛、移情
  • 認真性（C）——謹慎性：勝任、條理、盡職、成就、自律、謹慎



強有力的理論和進化論證表明，一些關於人的個性特徵的信息，尤其是那些對社交至關重要的信息，可能是通過人的面部傳達出來。畢竟，人的面容和行為都是由基因和荷爾蒙塑造的，而由人的外貌所產生的社會經驗可能會影響到人的性格發展。


現有的研究也已經揭示了基於五因素模型或大五 ( BF ) 人格模型的客觀面部圖像線索和一般人格特徵之間的聯繫。


面部表情分析是一個多特徵複雜分析問題，來自神經科學領域的研究證據表明，人的大腦在處理面部圖像時，並非只關注特定的面部特徵細節，而是以一種全局處理的方式，對所看到的面部圖像進行分析。


而機器學習領域的人工神經網絡（ANNs）恰恰具有這一優點，ANNs 可以以整體的方式提取和處理重要的特徵，且最近 ANNs 在分析人類面部、身體姿勢和行為以推斷明顯的個性特徵方面的應用表明，這種方法比單獨的人類評分者具有更高的預測準確性。

AI人物性格分析

在這項最新研究中，研究人員選取 1.2 萬名志願者為研究樣本，根據 “人格結構五因素模型” 完成自我報告問卷，測量他們的性格特徵，共上傳了 3.1 萬張 “自拍照片”。
在研究中，被調查者被隨機分為訓練組和測試組。通過一系列的神經網絡對圖像進行預處理，確保圖像的質量和特徵一致，並排除了帶有情緒化表情的人臉，以及名人和貓的照片。
然後研究人員開發了一個計算機視覺神經網絡 (NNCV) ，並在一個標有“自我報告五大特徵”的大數據集上對其進行訓練，從而繞過了人類評分員的可靠性限制，揭示了基於人臉 “ 自拍 ” 照片進行人物性格分析的能力。
接下來，研究人員訓練了一個人格診斷神經網絡 (NNPD) ，將每張圖片分解成 128 維空間中的向量，然後再訓練一個多層感知器，NNPD 以從 NNCV 得到的不變量向量作為輸入，預測大五人格特質作為輸出，利用圖像特徵向量來預測性格特徵。該網絡使用相同的硬件進行訓練，訓練過程耗時 9 天，整個過程分別對男性和女性面部進行看分析。


計算機視覺神經網絡 ( NNCV ) 和人格診斷神經網絡 ( NNPD ) 的層結構


研究結果表明，該模型的性能高於先前使用機器學習或人類評分者的研究發現。根據志願者上傳至網上的 “ 自拍照 ” ，人工智能模型能夠對責任心、情緒穩定性、外向性、親和性和開放性做出高於偶然性的判斷，性格評價性能優於之前使用機器學習或人類評判員所作出的分析性能。由此得出的性格判斷方法在同一個人的不同照片上的評價性能是一致的。
在 58％ 的案例中，AI 可以正確猜出兩個隨機選擇的個體在人格維度上的相對地位，而不是偶然預期的 50％。與自我報告的和觀察者的人格特質等級之間的相關性的薈萃分析估計相比，這表明依賴靜態面部圖像的人工神經網絡的性能要優於普通人，後者無需事先相識即可親自達到目標。相比其他四個特徵，責任心比其他四個特徵更容易被識別；基於女性面孔的性格預測似乎比男性面孔的預測更可靠。


根據大五人格特質合成的人臉特徵圖像，拿出你的自拍照來看下你是哪種性格

這一研究結果也表明，哪怕是在不受控制條件下拍攝的真實照片，也可以使用複雜的計算機視覺算法來預測人格特徵，這一發現與之前主要依賴於在受控環境下拍攝的高質量面部圖像的研究形成了對比。
雖然該研究表現出了較強的性格預測能力，但研究人員也表示，該研究目前還存在一定的侷限性。


首先，現實生活中的照片帶有各種微妙的線索，比如化妝、角度、面部表情，以及人們在拍攝和分享自己的照片時做出的所有其他選擇的相關信息。這些額外的線索可以說明他們的性格，而所有這些變量的影響與靜態面部特徵是不可分割的，因此很難從研究結果中得出重要結論。


其次，涉及到地理上有界限的個體樣本，該研究的參與者大部分是白種人，代表一個文化和年齡群體。未來的研究可以使用代表更多樣化的種族、文化和年齡群體的人口來擴展這一研究。


不管怎麼說，有科學依據的 AI“看臉算命”，還是比時不時刷屏的朋友圈算命，要靠譜許多。
