小心三類能引起耳鳴耳聾的藥物

2020-11-14
| 科學

一、解熱鎮痛藥、消炎藥

1.水楊酸鹽 :為常用的解熱鎮痛藥 ,長期大量應用時 ,可引起耳鳴、耳聾、聽力減退。一般是可逆的 ,停藥後即可恢復。但也有引起永久性耳聾的報道。在臨床上常用的藥物有乙酰水楊酸 (阿司匹林 )片、散 ,複方阿司匹林 (ＡＰＣ )片、散 ,賴氨匹林注射液 ,水楊酸咪唑 (施力靈、2-羥基苯甲酸咪唑 ) ,複方撲爾敏 (撲爾感冒片 )片。

2.萘普生 :偶有引起耳鳴、耳聾的報道 ,多數停藥後恢復正常 ,但極少數病人 ,停藥後聽力也不見改善。據報道發病率約 4%左右。

3.託美汀 (痛滅定、託麥汀 ) :為一種解熱鎮痛消炎新藥。在臨床上 ,偶見耳鳴、耳聾等耳毒性反應 ,停藥後症狀可消失。

4.舒林酸 (奇諾力 ) :為一種新的抗風溼病藥。偶可引起耳鳴、耳聾 ,可適當地減量或對症治療 ,一般不影響療效。

5.炎痛喜康 (吡羅昔康 ) :少數病人服藥後 ,偶見耳鳴 ,個別病例出現聽力減退或耳聾。

二、鎮痛藥

右旋丙氧芬 (達而豐 ) :為鎮痛新藥。偶見報道 ,此藥過量時可致嚴重感覺神經性耳聾 ,最後發展成全聾。

三、鎮咳藥、平喘藥

吡布特羅 (丁氨特羅、吡丁醇、吡丁舒喘寧 ) :主要用於支氣管哮喘、慢性支氣管炎、肺氣腫等。在臨床上 ,偶見有引起耳鳴、耳聾的報道 ,此時應減量或停藥。
