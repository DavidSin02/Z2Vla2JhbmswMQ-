1. 需要構建的TensorFlow網絡結構

下面要構建的模擬正弦函數的神經網絡，包含3個隱藏層，每個隱藏層16個隱藏節點，單變量輸入，單變量輸出，各層的激活函數都採用 sigmoid函數，需要構建的網絡結構如下圖所示。

2. 繪製標準的sin曲線

下面實現繪製標準的正弦函數，之後標準的正弦函數和模擬結果都採用 pylab.plot 畫到圖上，方便做對比。

具體代碼：

導入相應的Python包和模塊。

import tensorflow as tf

import math

import numpy as np

import matplotlib.pyplot as plt

import pylab

定義draw_sin_line() 函數，該函數用來繪製標準的sin曲線。

def draw_sin_line():

'''

繪製標準的sin曲線

'''

x = np.arange(0, 2*np.pi, 0.01)

x = x.reshape((len(x), 1))

y = np.sin(x)

pylab.plot(x, y, label='標準的sin曲線')

plt.axhline(linewidth=1, color='r')

plt.axvline(x=np.pi, linestyle='--', linewidth=1, color='g')

pylab將所有的功能函數（pyplot狀態機函數，大部分是 numpy裡面的函數）全部導入其單獨的命名空間內。為什麼要這樣做，是因為這樣可以很好地與ipython（或者類似的IDE，比如pycharm）實現很好的交互模式。

plt.axhline()函數，繪製平行於x軸的水平參考線。

plt.axvline()函數，繪製平行於y軸的垂直參考線。

3. 創建訓練樣本

定義 get_train_data() 函數，返回一個訓練樣本(train_x, train_y)，其中 train_x 是隨機的自變量，train_y 是train_x 的sin函數值。

具體代碼如下：

def get_train_data():

'''

返回一個訓練樣本(train_x, train_y)

其中 train_x 是隨機的自變量，train_y 是train_x 的sin函數值

'''

train_x = np.random.uniform(0.0, 2*np.pi, (1))

train_y = np.sin(train_x)

return train_x, train_y

函數原型： numpy.random.uniform(low,high,size)

功能：從一個均勻分佈[low,high)中隨機採樣，注意定義域是左閉右開，即包含low，不包含high.

參數介紹:

low: 採樣下界，float類型，默認值為0；

high: 採樣上界，float類型，默認值為1；

size: 輸出樣本數目，為int或元組(tuple)類型，例如，size=(m,n,k), 則輸出m*n*k個樣本，缺省時輸出1個值。

返回值：ndarray類型，其形狀和參數size中描述一致。

4. 定義推理函數inference

構建TensorFlow 網絡結構的邏輯在 inference() 函數中實現。

其中，構建了3個隱藏層，每個隱藏層16個節點，連接節點的參數 weight 和 bias的初始化是均值為0、方差為1的隨機初始化，每個隱藏層的單位採用 tf.sigmoid() 作為激活函數，輸出層中沒有增加 sigmoid函數，這是因為前面的幾層非線性變換已經提取好了足夠充分的特徵，使用這些特徵已經可以讓模型用最後一個線性分類函數來分類。

具體代碼如下：

def inference(input_data):

'''

定義前向計算的網絡結構

args: 輸入x的值，單個值

'''

with tf.variable_scope('hidden1'):

# 第一個隱藏層，採用16個隱藏節點

weights = tf.get_variable("weight", [1, 16], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

biases = tf.get_variable("bias", [1, 16], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

hidden1 = tf.sigmoid(tf.multiply(input_data, weights) + biases)

with tf.variable_scope('hidden2'):

# 第二個隱藏層，採用16個隱藏節點

weights = tf.get_varaible("weight", [16, 16], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

biases = tf.get_variable("bias", [16], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

mul = tf.matmul(hidden1, weights)

hidden2 = tf.sigmoid(mul + biases)

with tf.variable_scope('hidden3'):

# 第三個隱藏層，採用16個隱藏節點

weights = tf.get_varaible("weight", [16, 16], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

biases = tf.get_variable("bias", [16], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

mul = tf.matmul(hidden2, weights)

hidden3 = tf.sigmoid(mul + biases)

with tf.variable_scope('output_layer'):

# 輸出層

weights = tf.get_varaible("weight", [16, 1], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

biases = tf.get_variable("bias", [1], tf.float32, initializer=tf.random_normal_initializer(0.0, 1))

output = tf.matmul(hidden3, weights) + biases

return output

如果變量存在，函數tf.get_variable( ) 會返回現有的變量。如果變量不存在，會根據給定形狀和初始值創建變量。

初始器（initializer）= tf.random_normal_initializer(0.0, 1) 是其中一種內置的初始器。

5. 定義訓練函數

使用TensorFlow 實現神經網絡時，需要定義網絡結構、參數、數據的輸入和輸出、採用的損失函數和優化方法。

最繁瑣的訓練中的反向傳播、自動求導和參數更新等操作由TensorFlow 負責實現。

具體代碼如下，代碼中有詳細的註釋：

# 通過梯度下降將損失最小化

def train():

# 學習率

learning_rate = 0.01

x = tf.placeholder(tf.float32)

y = tf.placeholder(tf.float32)

# 基於訓練好的模型推理，獲取推理結果

net_out = inference(x)

# 定義損失函數的op

loss_op = tf.square(net_out - y)

# 採用隨機梯度下降的優化函數

opt = tf.train.GradientDescentOptimizer(learning_rate)

# 定義訓練操作

train_op = opt.minimize(loss_op)

# 變量初始化

init = tf.global_variables_initializer()

with tf.Session() as sess:

# 執行變量的初始化操作

sess.run(init)

print("開始訓練 ...")

for i in range(100001):

# 獲取訓練數據

train_x, train_y = get_train_data()

sess.run(train_op, feed_dict={x: train_x, y: train_y})

# 定時輸出當前的狀態

if i % 10000 == 0:

times = int(i/10000)

# 每執行10000次訓練後，測試一下結果，測試結果用 pylab.plot()函數在界面上繪製出來

test_x_ndarray = np.arange(0, 2 * np.pi, 0.01)

test_y_ndarray = np.zeros([len(test_x_ndarray)])

ind = 0

for test_x in test_x_ndarray:

test_y = sess.run(net_out, feed_dict={x: test_x, y: 1})

# 對數組中指定的索引值指向的元素替換成指定的值

np.put(test_y_ndarray, ind, test_y)

ind += 1

# 先繪製標準的正弦函數的曲線

# 再用虛線繪製出模擬正弦函數的曲線

draw_sin_line()

pylab.plot(test_x_ndarray, test_y_ndarray, '--', label = str(times) + ' times')

pylab.legend(loc='upper right')

pylab.show()

print("=== DONE ===")

從輸入數據到神經網絡，到輸出預測值，採用預測值和標準值的差的平方作為損失函數，然後將得到的損失函數的操作 loss_op 傳給隨機梯度下降優化方法 tf.train.GradientDescentOptimizer，從而得到最後訓練操作 train_op。

在會話的run 方法中，傳入訓練數據，每一次執行 train_op，就會根據輸入的訓練樣本做一次前向計算、一次反向傳播和一次參數更新。

每訓練10000個樣本，就將標準的正弦函數和模擬結果採用 pylab.plot()函數繪製出來做對比，其中用實線表示標準的正弦函數，虛線表示模擬的正弦函數（也就是基於神經網絡推理的結果）。
