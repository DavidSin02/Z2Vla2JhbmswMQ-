常見壓力容器設計軟件介紹，我收藏了！

2020-10-29
| 科學


由於壓力容器在各行各業中應用越來越廣泛，作用越來越突出，為了滿足壓力容器的製造和使用要求，對設計的要求也越來越高。作為壓力容器的設計，不但要掌握相關專業知識，根據有關標準和要求進行結構設計和強度、剛度等方面的計算，而且需要對壓力容器的受力、結構尺寸、材料等進行優化處理，以滿足壓力容器設計的合理性和經濟性。


而壓力容器的強度計算是一項極為繁瑣的工作，需要在相關標準的基礎上對壓力容器的應力進行計算、判斷並規範，最終形成壓力容器的存檔資料。這往往會佔用大量的人力、物力及時間。因此.開發並應用一些設計軟件，對提高壓力容器設計工作效率和設計質量，研製性能更優越的壓力容器是一件非常有意義的工作。


幾十年來，國內外許多公司和機構(國內：全國化工設備設計技術中心站、甘肅藍科石化高新裝備股份有限公司、大連理工大學、北京大學等。國外：德國的ASKA、英國的PAFEC、法國的SYSTUS、美國的ABAQUS、ADINA、ANSYS、BERSAFE、BOSOR、COS MOS、ELAS、MARC、CODEWARE、Bentley和STARDYNE 等公司)進行了壓力容器軟件的開發。



經學者專家們的努力，目前已開發出許多優秀的設計軟件，如SW6 - 2011 過程設備強度計算軟件包、LANSYS壓力容器強度設計軟件、TANK 儲罐設計軟件、CAESARII管道應力分析軟件， Pvelite &.Code Calc、COMPRESS、AutoPipe Vessel(MicoroProtol)、VVD壓力容器整體及零部件分析設計軟件、MT-EXC H 管殼式換熱器分析設計軟件、NozzlePro 管嘴有限元分析軟件、FE/ PIPE 有限元方法壓力容器及管道應力分析軟件以及FEPG、JlGFFX、HAJIF、ANSYS、ABAQUS、ADINA、DYNA3D、NASTRAN、ASKA、ALGOR、BEASY、AFEMS、PAFEC、SYSTUS 等有限元分析設計軟件。


下面就國內常用的幾種軟件進行簡單的介紹。


SW6-2011 強度計算軟件包


SW6-2011 是全國化工設備設計技術中心站開發的化工設備設計軟件，該軟件是基於Windows、WORD中文版的壓力容器設計工具軟件。同LANSYS 相比， SW6-2011 提供了用戶自定義數據庫和自動形成規範化格式、加入封面、頁碼及圖形等的中英文計算書的功能。


它不僅可以進行設備整體的計算，也可以進行單個零部件的計算，另外，它還能計算一些GB/T 150、GB/T 151 未列入的壓力元件。它主要包含了GB/T 150.1 ~ 150.4《壓力容器》、GB/T 151-2014 《熱交換器》、NB/T 47041-2014 《塔式容器》、NB/T 47042-2014 《臥式容器》、GB/T 12337-2014《鋼製球形儲罐》等，主要對如下幾方面進行計算。


①受壓元件：如筒體、封頭、法蘭和開孔補強等。

②臥式容器：包括筒體、封頭、設備法蘭、開孔補強及鞍座等。

③立式容器：包括筒體、封頭、夾套、開孔補強、設備法蘭及攪拌軸等。

④固定管板換熱器：包括殼程圓筒、管程圓筒、管箱法蘭和管板等。

⑤浮頭式及填料函式換熱器：包括浮頭蓋、鉤圈、筒體、前(後)端管箱、筒體法蘭、前(後)端管箱法蘭、管板及開孔補強等。

⑥U形管式換熱器：包括筒體、前(後)端管箱、筒體法蘭、前(後)端管箱法蘭、管板及開孔補強等。

⑦高壓設備：包括單層圓筒、多層包紮式圓筒、熱套筒體、高壓設備的封頭(包括球形、平蓋、鍛制緊縮口以及卡箍、金屬平墊、橢圓墊、八角墊和雙錐密封等) 。

③ 塔設備：包括板式塔、填料塔、塔板與填料混合內件的等截面或變截面塔以及基礎環板固定在框架結構上的塔等。

⑨ 球形儲罐：包括球殼板、支柱、地腳螺栓、拉桿、銷子、支柱底板、耳順及翼板等。

⑩非圓形容器：包括矩形、帶圓角矩形、長圓形、橢圓形截面容器的殼體、支撐板及加強件等。

⑾零部件：包括無墊片法蘭(或墊片反力可忽略不計)、卡箍連接密封、彎頭、三通、斜接管、帶法蘭凸形封頭等。

⑿局部應力：包含了美國焊接研究會WRC-107 公報1979 版和WRC-297 公報1987 版的全部內容，包括柱殼、球亮、橢圓封頭或碟形封頭等在外載荷作用下所產生的最大表面應力和最大薄膜應力等。


LANSYS 壓力容器強度設計軟件


LANSYS 是藍森石化技術有限公司開發的一套設計軟件，它按GB/T 150.1 ~ 150.4、GB/T 151 、GB/T 16749 、GB/T 12337、NB/T 47041、NB/T 47042、HG / T 20569 及有關基礎標準進行編制，在尊重原標準及行業習慣的原則下進行了合理的分類， 80 多種零件項(計算項)歸納為殼與封頭、平蓋與法蘭、換熱器零件、非圓形截面容器、筒體端部五類，設備類包括換熱器、臥式容器、直立容器、球形儲罐、攪拌器等幾大類。


LANSYS 以用戶為中心運作。用戶選擇需要的設備，並可添加零件，更可以由用戶組合零件建立自己的設備，甚至零件名稱也可由用戶命名。


LANSYS 強大的及時計算功能，加上數據輸入、圖形提示、計算結果同頁集成，用戶所見即所得，甚至重要的中間數據也能及時看到，直觀高效。


帶頁標籤的工作簿式設計，使計算項之間的切換準確快捷，如同翻看設計書一樣，而且計算項計算通過後頁標籤圖標將由紅色變為綠色。


輸入數據缺省值設置、標準元件數據套用功能(如法蘭、型鋼) ，不僅降低了用戶工作強度，也為用戶提供了參考數據。


LANSYS 自動產生規範化格式、加入封面、目錄及圖形公式甚至表格的設計計算書，方便用戶存檔。


LANSYS 具有強大的項目文件組織管理功能，如多文檔功能、複製粘貼功能、排序刪除功能等。項目文件包括計算內容、計算模式、輸入輸出數據及報告，這些內容均由LANSYS自主管理，無需第三方軟件支持，文件非常小，通常只有10KB 左右。LANSYS 合乎標準的專業糾錯提示，使用戶的疏漏減為最少，如管板管孔數太多，LANSYS 也能提示糾錯。


設備項目中各計算項的設計壓力、設計溫度、公稱直徑、壓力試驗類型四項數據相互關聯，一經修改，相關計算項全部動態產生計算結果，自動找出計算項中試驗壓力最小值作為設備試驗壓力。


詳實的LANSYS 幫助，儘可能引用標準原文原圖，用戶在得到準確幫助的同時，又能熟悉標準本身。


材料錄入與管理功能，使用戶可以方便地使用標準以外的材料。


LANSYS 於1999 年7 月順利通過了全國壓力容器標準化技術委員會計算機應用分技術委員會的測試評審，評審認為LANSYS 採用先進的軟件編程技術，在軟件界面的友好性、操作方便性、外部材料數據庫建立及標準零件數據庫和計算文檔管理等方面具有自己的特色，在國內同類軟件中具有先進水平。


有限元程序自動生成系統FEPG


有限元程序自動生成系統FEPG CFinite Element Program Generator) 為有限元分析和計算機輔助工程分析CCAE) 軟件平臺。


用戶只需輸入有限元方法所需的各種表達式和公式，即可由FEPG 自動產生所需的全部有限元計算的、源程序、包括單元子程序、算法程序等，免去了大量繁瑣的有限元編程工作，保證了程序的正確性和統一性。


FEPG 採用元件化的程序設計方法和人工智能技術，根據有限元方法統一的數學原理及其內在規律，以類似於數學公式推理的方式，由微分方程表達式和算法表達式自動產生有限元源程序。


FEPG 適用於求解各種領域的各種工程與學科的有限元問題，突破了目前通用有限元程序只能用於特定領域和特定問題的限制，廣泛應用於石油化工、機械製造、能源、汽車交通、鐵道、國防軍工、電子、土木工程、造船、生物醫學、輕工、地礦、水利、航空航天、日用家電等工業部門，尤其適合於各類學科的科學研究和高校的有限元教學。


FEPG 分為單機版和網絡版，單機版依次為FEPG l.0、FEPG2.0、FEPG2.2、F EPG3.0、FEPG4.0、FEPG4.2; 網絡版依次為IFEPG4.0、IFEPG4.2 0基於FEPG 開發的壓力容器分析系統VAS2.0 軟件。


目前包含的結構有軸對稱部件(有無中心孔接管的各類封頭、裙座與柱殼、夾套容器、膨脹節等) 和三維部件（有偏心開孔接管的各類封頭、柱殼開孔接管、球形儲罐(有託板結構、翻邊結構或U 形柱結構)、有無中心開孔結構的管版、帶法蘭的平封頭、橢圓封頭、三維膨脹節等），其進行計算的內容有線彈性材料的機械靜力計算、溫度場計算和熱應力計算、按應力分類進行應力評定、疲勞分析等。


ANSYS 分析設計軟件


ANSYS 軟件是第一個通過ISO 9001 質量認證的大型分析設計類軟件，是美國機械工程師協會(ASME)、美國核安全局(NQA) 及近20 種專業技術協會認證的標準分析軟件。


在國內第一個通過了中同壓力容器標準化技術委員會認證並在國務院17 個部委推廣使用。現在有多家壓力容器設計單位採用ANSYS 有限元分析軟件進行壓力容器的設計。


ANSYS 軟件是融結構、流體、電磁場、聲場和耦合場分析於一體的大型通用有限元分析軟件。由世界上最大的有限元分析軟件公司之一的美國ANSYS 公司開發，它能與多數CAD 軟件接口實現數據的共草和交換，如Pro / Engin eer、UG、SolidWorks、NASTRAN、A l ogor、CATTA J -DEAS、AutoCAD 等， 是現代產品設計中的高級CAE 工具之一。ANSYS 有限元分析軟件可進行設備的整體或零部件的分析計算。


ANSYS 軟件主要包括三個部分：前處理模塊、分析計算模塊和後處理模塊。


前處理模塊提供了一個強大的實體建模及網將劃分工具， 用戶可以方便地構造有限元模型。


分析計算模塊包括結構分析(可進行線性分析、非線性分析和高度非線性分析)、流體動力學分析、電磁場分析、聲場分析、壓電分析以及多物理場的耦合分析，可模擬多種物理介質的相互作用，具有靈敏度分析及優化分析能力。


後處理模塊可將計算結果以彩色等值線顯示、梯度顯示、矢量顯示、粒子流跡顯示、立體切片顯示、透明及半透明顯示(可看到結構內部) 等圖形方式顯示出來，也可將計算結果以圖表、曲線形式顯示或輸出。


軟件提供了100 種以上的單元類型，用來模擬工程中的各種結構和材料。該軟件有多種不同版本，可以運行在從個人機到大型機的多種計算機設備上。它不僅可以用來進行壓力容器設計的應力分析和疲勞分析，而且能夠進行壓力容器的優化設計，在各種壓力容器設計軟件中佔有極其重要的地位。


ABAQUS 分析設計軟件


ABAQUS 是一套功能強大的工程模擬的有限元軟件，其解決問題的範圍從相對簡單的線性分析到許多複雜的非線性問題。ABAQUS 包括一個豐富的、可模擬任意幾何形狀的單元庫。並擁有各種類型的材料模型庫，可以模擬典型工程材料的性能，其中包括金屬、橡膠、高分子材料、複合材料、鋼筋混凝土、可壓縮超彈性泡沫材料以及土壤和岩石等地質材料。作為通用的模擬工具， ABAQUS 除了能解決大量結構（應力 / 位移）問題，還可以模擬其他工程領域的許多問題，例如熱傳導、質量擴散、熱電耦合分析、聲學分析、岩土力學分析（流體滲透 / 應力耦合分析）及壓電介質分析。


ABAQUS 為用戶提供了廣泛的功能，且使用起來又非常簡單。大量的複雜問題可以通過選項塊的不同組合很容易的模擬出來。例如，對於複雜多構件問題的模擬是通過把定義每一構件的幾何尺寸的選項塊與相應的材料性質選項塊結合起來。在大部分模擬中，甚至高度非線性問題，用戶只需提供一些工程數據，像結構的幾何形狀、材料性質、邊界條件及載荷工況。在一個非線性分析中， ABAQUS 能自動選擇相應載荷增量和收斂限度。他不僅能夠選擇合適參數，而且能連續調節參數以保證在分析過程中有效地得到精確解。用戶通過準確的定義參數就能很好的控制數值計算結果。


ABAQUS 有兩個主求解器模塊— ABAQUS/Standard 和 ABAQUS/Explicit。ABAQUS 還包含一個全面支持求解器的圖形用戶界面，即人機交互前後處理模塊 — ABAQUS/CAE 。 ABAQUS 對某些特殊問題還提供了專用模塊來加以解決。


ABAQUS 被廣泛地認為是功能最強的有限元軟件，可以分析複雜的固體力學結構力學系統，特別是能夠駕馭非常龐大複雜的問題和模擬高度非線性問題。 ABAQUS 不但可以做單一零件的力學和多物理場的分析，同時還可以做系統級的分析和研究。


ABAQUS 的系統級分析的特點相對於其他的分析軟件來說是獨一無二的。由於 ABAQUS 優秀的分析能力和模擬複雜系統的可靠性使得 ABAQUS 被各國的工業和研究中所廣泛的採用。ABAQUS 產品在大量的高科技產品研究中都發揮著巨大的作用。


Pvelite & Code Calc


Pvelite & Code Calc 壓力容器整體及零部件分析設計軟件主要用來進行從最基本到非常複雜的臥式和立式壓力容器、換熱器的整體及零部件的設計。該軟件錄入了3600 多種材料的性能數據.井提供自選單位、自定義材料和動態跟蹤等功能。


其狀態欄可顯示指定元件的厚度、垂直高度、最大工作壓力等。它覆蓋了ASME 第VIII卷(ASME VIII - l、ASME VIII - 2)、BS5500、UBC、NBC、ANSI B16.5、TEMA、WRC-107 號公告等，具有強大的設計和分析能力。


Pvelite &.Code Calc 主要對如下幾方面進行計算。


① 臥式和立式壓力容器整體分析\'及設計。

② 臥式和立式壓力容器零部件的分析及設計。

③ 換熱器零部件：包括殼體、封頭、浮頭、開口、法蘭、管板、連接件、各種支撐方式、脹節分析及設計。

④矩形和其他非圓形容器等分析及設計。

⑤球形儲罐的整體分析及設計。


ASME VIII強度計算軟件COMPRESS


COMPRESS是CodeWare公司旗下主打產品，主要用於ASME VIII-1，VIII-2的強度計算。COMPRESS相比其它同類軟件其缺點是隻能按ASME標準計算，其優點是界面非常友好，模型效果非常逼真，這一點上不論是PVELITE,VVD,AUTOPIPE VESSEL都只能自嘆不如。
