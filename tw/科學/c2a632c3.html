什麼是天敵昆蟲？

天敵昆蟲是一類長期在農田、林區和牧場中控制著害蟲的發展和蔓延的昆蟲。分為捕食性天敵昆蟲和寄生性天敵昆蟲。


+ 寄 生 性 天 敵 +



寄生性天敵最重要的是寄生蜂和寄生蠅。

+姬 蜂+

Ichneumonidae



姬蜂看起來溫柔、善良，但是，它們全部成員無一不是靠寄生生活的，是一些小動物的致命死敵。




姬蜂寄生本領十分高強，即使在厚厚的樹皮底下躲藏的寄主也難逃其手。

而且不論哪一種姬蜂，它們在幼蟲時期都要在其它類昆蟲的幼蟲或蜘蛛等體內生活，以吸取這些寄主體內的營養，滿足自己生長髮育的需要。




正是由於姬蜂的寄生，寄主最終身體被掏空，而一命嗚呼。



姬蜂蜂蛹以及乾癟的毛蟲屍體


姬蜂的寄主包括：鱗翅目、鞘翅目、膜翅目等昆蟲，所幸當中的大多數種類是農、林害蟲。所以姬蜂是有重要經濟意義的一類昆蟲。




常見的例如：舞毒蛾瘤姬蜂 Pimpla disparis，寄主為舞毒蛾、金剛鑽、稻苞蟲、菜粉蝶、柑橘鳳蝶。螟蛉懸繭姬蜂Charops bicolor，寄主為棉小造橋蟲、鼎點金剛鑽、棉鈴蟲、粘蟲、稻苞蟲、稻螟蛉。

+ 繭 蜂 +

Braconidae





繭蜂個頭小巧，雌蜂大都有長長的產卵器。


繭蜂算是寄生蜂中的明星，主要寄生於鱗翅目、雙翅目、鞘翅目昆蟲；也有些種類寄生於膜翅目(葉蜂、蟻)和脈翅目昆蟲；還有少數種類寄生於半翅目和齧蟲目等不全變態昆蟲。


雌蜂在寄主體內產卵，之後寄生蜂寶寶在寄主體內放肆生長，待到幼蟲老熟時會在寄主體外結白色絲繭化蛹。

也有些雌蜂會在寄主的卵上產卵，但是這些蜂卵不會馬上孵化，要等到寄主先孵化之後，寄生蜂才會陸續孵化。彷彿繭蜂一直默默監視寄主的一舉一動，等到時機成熟才給予致命一擊。



千瘡百孔的可憐毛毛蟲


好在繭蜂主要寄生於許多重要的害蟲，尤其鱗翅目的害蟲，對害蟲的發生數量起一定的控制作用，在生物防治界鼎鼎有名。




我國常見的繭蜂有：麥蛾柔繭蜂Habrobracon hebetor，寄生於麥蛾、印度谷螟、粉斑螟、米蛾、地中海粉螟等倉庫害蟲幼蟲。



麥蛾柔繭蜂


螟蛉盤絨繭蜂Cotesia ruficrus，寄生粘蟲、勞氏粘蟲、禾灰翅夜蛾、條紋螟蛉、二化螟、三化螟、稻縱卷葉螟、稻苞蟲、稻眼蝶、棉鈴蟲、棉小造橋蟲等多種害蟲。

腰帶長體繭蜂 Macrocentrus cingulum，則是重要農業害蟲亞洲玉米螟幼蟲的寄生蜂。



腰帶長體繭蜂


還有整治了大壞蛋蚜蟲的菸蚜繭蜂Aphidius gifuensis，更是鼎鼎大名，它可以寄生於菸蚜、麥長管蚜、棉蚜、蘿蔔蚜等等。雌蜂在蚜蟲體內產卵，幼蟲取食蚜蟲“內臟”後化蛹。



菸蚜繭蜂

這猶如珍珠一般的僵蚜，就是菸蚜繭蜂消滅的蚜蟲。



+ 螯 蜂 +

Dryinidae



螯蜂屬於青蜂總科中的螯蜂科，至於為啥叫螯蜂呢？

因為大部分的雌性螯蜂，她們的前足會特化出螯，emmmmm，就是小鉗子，和螃蟹的差不多，可以用來捕捉獵物和抱握寄主，故名螯蜂。


螯蜂的寄主主要包括我們熟悉的飛蝨、葉蟬、蠟蟬等半翅目的昆蟲。

螯蜂不僅可以寄生這類害蟲的成蟲或若蟲，許多雌性螯蜂還可以捕食寄主。這種既可以捕食，又可以寄生的現象在所有寄生蜂中也是很少見的。



被螯蜂寄生的葉蟬 going to die


常見的比如：兩色食蝨螯蜂Echthrodelphax fairchildii寄生於灰飛蝨、白背飛蝨和褐飛蝨成蟲和若蟲。還有，黑尾葉蟬螯蜂是黑尾葉蟬的天敵，斑衣蠟蟬螯蜂則是斑衣蠟蟬的剋星。




圖片來源：中國國家地理BOOK

斑衣蠟蟬螯蜂

+ 小 蜂 +

Chalcididae



小蜂屬於小蜂總科的小蜂科，體長5 mm左右，後足股節極度膨大，其下緣多有一排齒，脛節彎曲，休息時翅不能縱向摺疊。


小蜂寄生於鱗翅目、雙翅目、鞘翅目等昆蟲。



常見的重要種類如廣大腿小蜂Brachymeria lasus，是粉蝶、松毛蟲苞蟲、舞毒蛾、稻縱卷葉螟、稻螟蛉、粘蟲等多種昆蟲的天敵；粉蝶大腿小蜂寄生於菜粉蝶和稻眼蝶的蛹。




+ 赤 眼 蜂 +

Trichogrammatidae



赤眼蜂更是寄生蜂中大名鼎鼎且深受歡迎，它們個頭很小，小到幾乎肉眼看不見，只能在顯微鏡下才能看清楚它的模樣，標誌性的紅色複眼十分可愛。


赤眼蜂是卵寄生蜂，雌蜂會尋覓理想產卵地，快速準確地找出最新鮮的蟲卵，然後用腹部末端的產卵器向寄主卵內探鑽，把卵產在其中。


之後孵化的幼蟲取食卵黃，並在其中化蛹，直到羽化成蟲之後才咬破寄主卵殼外出自由生活。


赤眼蜂是重要的天敵昆蟲。它們的寄主範圍很廣，有鱗翅目、鞘翅目、膜翅目、脈翅目、半翅目、直翅目。。。當然，最喜愛的口味還是鱗翅目昆蟲。


柞蠶卵培育的赤眼蜂


目前，國內人工飼養、釋放、利用最為廣泛的赤眼蜂種類，包括：稻螟赤眼蜂Trichogramma japonicum、松毛蟲赤眼蜂T.dendrolimi、擬澳洲赤眼蜂T.confusum、玉米螟赤眼蜂T.ostriniae等等。

玉米螟赤眼蜂 Trichogramma ostriniae



+ 寄 蠅 +

Tachinidae



寄蠅屬於雙翅目寄蠅科，雖然是蒼蠅的近親，外貌與其相似，卻有著蒼蠅難以企及的優勢——通過寄生消滅農林害蟲。


所有的寄蠅都是寄生的，它們的寄主很多，包括：鱗翅目、鞘翅目、膜翅目、雙翅目、半翅目、直翅目、革翅目、竹節蟲目等昆蟲，以及多足綱蜈蚣和蛛形綱蜘蛛均可被寄蠅寄生。

寄蠅是天敵昆蟲中寄生能力最強、活動能力最大、寄主種類多、分佈十分廣泛的類群。


而且寄蠅主要攻擊寄主的幼蟲階段，不僅可通過寄生殺死害蟲，且大多寄蠅都能保持30%-60%的寄生率。


比如：松毛蟲狹頰寄蠅Carcelia matsukarehae 和蠶飾腹寄蠅Blepharipa zebina 的寄主是松毛蟲；普通怯寄蠅Phryxe vulgaris的寄主是菜青蟲、黃地老虎、翠紋地老虎；稻苞蟲鞘寄蠅Thecocarcelia parnarae的寄主是稻苞蟲，稻縱卷葉螟。
