01 水黽屬於哪一目？

A 半翅目

B 蚤目

C 同翅目

D 鱗翅目

【答案】A

【解析】水黽是一類適宜在水面上生活的半翅目昆蟲。研究表明，水黽的浮力來自它們的3對特殊結構的足，在足的表面生有許多微毛，微毛表面有疏水性的油脂包裹，這種結構的足在水表面張力的作用下產生浮力，可以支撐起水黽瘦小的身體，從而能夠使它像船一下漂浮在水面。

02 水黽有幾對足？

A 2對

B 3對

C 8對

D 10對

【答案】B

【解析】水黽是一類適宜在水面上生活的半翅目昆蟲。研究表明，水黽的浮力來自它們的3對特殊結構的足，在足的表面生有許多微毛，微毛表面有疏水性的油脂包裹，這種結構的足在水表面張力的作用下產生浮力，可以支撐起水黽瘦小的身體，從而能夠使它像船一樣漂浮在水面。

03 水黽是哪種食性的動物？

A 肉食性

B 植食性

C 雜食性

D 只喝水

【答案】A

【解析】水黽是肉食性昆蟲，在水面上滑來滑去主要是覓食，取食在水面活動的其他昆蟲或小型動物。

04 龍蝨屬於昆蟲哪一目？

A 食毛目

B 雙翅目

C 膜翅目

D 鞘翅目

【答案】D

【解析】龍蝨是鞘翅目龍蝨科昆蟲的統稱，它們和其他陸地昆蟲一樣，也是用氣管來呼吸的。它們之所以能在水中呼吸是因為具有特殊結構的翅膀；鞘翅完全革制化，膜質的後翅隱藏於革質的鞘翅之下。

05 關於蜜蜂的“舞蹈語言”，下列敘述錯誤的是？

A 能傳達蜜源的遠近

B 能傳達蜜源所屬花的顏色

C 能傳達蜜源的性質

D 能傳達到達蜜源所需的能量

【答案】B

【解析】當一隻擔當偵查任務的蜜蜂發現蜜源後，它會返回巢穴，並在家門口用“舞蹈語言”告訴其他同伴蜜源植物所處的方向和距離。“舞蹈語言”傳達的信息包括到達蜜源所需的能量、蜜源的性質及相對太陽位置的方向等。領會含義的工蜂隨後會一起出發採蜜。蜜源越近，蜜蜂的舞蹈節奏就越快。而蜜蜂是色盲,它只對藍色、黃色和白色分得清楚,對紅色是盲區。因此，一般不能準確傳達蜜源所屬花的顏色。

06 如果食物在附近，蜜蜂一般會跳什麼樣的“舞”來傳達信息？

A 三角形舞

B 8字形“搖擺舞”

C “圓圈舞”

D 機械舞

【答案】C

【解析】當一隻擔當偵查任務的蜜蜂發現蜜源後，它會返回巢穴，並在家門口用“舞蹈語言”告訴其他同伴蜜源植物所處的方向和距離。如果附近有食物，它會跳相對簡單的“圓圈舞”；如果食物在遠處，它會跳更為複雜的8字形“搖擺舞”。

07 若食物距離蜜蜂很遠，它一般會跳什麼樣的“舞”來告知同伴？

A “圓圈舞”

B 古典舞

C 梯形舞

D 8字形“搖擺舞”

【答案】D

【解析】當一隻擔當偵查任務的蜜蜂發現蜜源後，它會返回巢穴，並在家門口用“舞蹈語言”告訴其他同伴蜜源植物所處的方向和距離。如果附近有食物，它會跳相對簡單的“圓圈舞”；如果食物在遠處，它會跳更為複雜的8字形“搖擺舞”。

08 以下哪種昆蟲一般不能幫助植物傳粉？

A 蜂類

B 蝶蛾類

C 甲蟲類

D 蝗蟲類

【答案】D

【解析】除了蜜蜂之外，還有其他蜂類，如熊蜂、黃蜂，螞蟻類，蝶蛾類，蠅類，甲蟲類和薊馬類等大量昆蟲也為植物提供傳粉服務。

09 哪種昆蟲是無花果的專一傳粉者？

A 熊蜂

B 榕小蜂

C 黃蜂

D 馬蜂

【答案】B

【解析】無花果是一種榕樹的隱頭果實。榕樹雖是有花植物，但它的花很小，肉質花序軸向內捲曲成球狀的花托，被稱為“隱頭花序”。深藏的小花，既沒有豔麗的色彩，也沒有芬芳的香氣，而且全部被包裹在了膨大的完全封閉的花托之中，形成隱頭果，無法吸引任何傳粉昆蟲進入其中。然而，這個看似天衣無縫的花托還是留了一個小孔與外界相通，這是為無花果的專一傳粉者——榕小蜂量身定製的。

10 如果黑客侵入了數據庫或網絡操作系統，他不能實現的是？

A 在服務器裡盜取密碼和文件

B 使計算機病毒迅速傳播

C 讓計算機表達人類的基因

D 造成整個網絡癱瘓崩潰

【答案】C

【解析】如果黑客侵入了數據庫或網絡操作系統，既可以在服務器裡盜取密碼和文件，又可以使計算機病毒迅速傳播，造成整個網絡癱瘓崩潰。
