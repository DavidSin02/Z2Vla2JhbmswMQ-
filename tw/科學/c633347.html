「音樂雜談28」Pro Tools 2018的常用快捷鍵整理

2020-10-29
| 科學

喜歡「音樂雜談」這個主題的朋友可以關注我的頭條號，將會在不定期發表一些音樂理論以外的音樂話題的文章或者是音樂知識的乾貨，另外「主觀談風格」主題也會不定期更新，推薦好聽的音樂給大家。（此文為覆聽音樂的天職老師 發佈於今日頭條的原創文章，轉載請告知並註明出處）

Pro tools 可以說是混音的行業標準宿主了，其功能的強大，完全不用說明，另外就是我們這種製作宿主的快捷鍵可以說是多到沒朋友，而且PT的話，還不像其他宿主，什麼都鼠標就可以解決。PT的很多功能都是要通過快捷鍵去實現，不然，你會非常非常痛苦。例如：沒有見到工具，只能用快捷鍵B。

下面我把我工作中會經常用到的快捷鍵和功能說明，全部整理出來。講真，太多了，我有時候都要自己翻出來看。所以我會整理出一個文檔，放在桌面上。有的說是常用，實際上，有的只是偶爾用，但是想要用卻不知道，要找半天，還是很痛苦的。所以整理出來，隨時可以查找。

打開文件--------CTRL-O

創建GROUP---------CTRL-G

暫時禁止編組------ SHIFT + CTRL + G

清除音頻空白區域----------CTRL-U

建立新音軌 CTRL-SHIFT-N

在當前指針的位置添加新的拍號------------CTRL-I

新增記憶點（就是插旗子）----------將播放指針定位在需要插旗子的地方，按下小鍵盤上的回車鍵即可 or win + 點擊標記條

音頻膠水----- ALT + SHIFT + 3 數字鍵盤

切換為循環LOOP播放--------CTRL-SHIFT-L或者小鍵盤上的4鍵

最大化當前所選擇的音頻或者MIDI區域------------ALT-R or E 快捷功能打開

放大縮小 軌道高度 ---------------------- win + ↑or ↓

橫向鋪滿窗口 ------------------- ALT + A

全屏鋪滿窗口 ------------------- CTRL + ALT + WIN + ↓

Mute ---------- shift + M

SOLO ---------- shift + S

錄音點亮 -------- shift + R

錄音監聽 ----------- alt + K

複製軌道 ---------- Shift + Alt + D

複製音頻片段 ------ Ctrl + D

重複粘貼----- ALT + R

導出選中音頻 ------ Shift + ctrl + K

片段窗口 選中沒用的片段 ------ Shift + ctrl + U 刪除沒用片段----- Shift + ctrl + B

打開調音臺 ------- Ctrl + = 調音臺軌道邊窄 ----- alt + ctrl + M

打開多個插件窗 ------ shift + 點擊插件 一次關閉多個插件窗 ------ alt + ctrl + W

指針跟蹤播放 ------- ctrl + N

保存插件預設 ----- ctrl + shift + S

音頻塊編組 ------ ctrl + alt + G

測速度： 確保 Tab至瞬態值 選項打開，，光標放到音頻重拍子開頭，點 Tab 鍵， 然後 點 ctrl + I ， 調整重拍位置。

彈力音頻

彈力音頻：Polyphonic 伴奏 和聲樂器

Rhythmic 節奏 打擊樂器

Monophonic 樂器SOLO 主人聲

Varispeed 變速變調

混音臺中對某些通道加入相同的效果器 --------- shift + alt + 插入效果器

混音臺發送通道小界面 ------ ctrl + 點擊發送通道 前面的小箭頭

混音臺通道寬度調整 ------ ctrl + alt + M

Shuffle模式---------F1 或者 alt 1 （音頻前後自動連接）

Slip模式----------F2 或者 alt 2 （自由模式）

Spot模式----------F3 或者 alt 3 （音頻區塊精準位置移動）

Grid模式----------F4 或者 alt 4 （絕對對齊，藍色。 相對對齊，紫色）

Zoomer工具---------F5

Trimmer工具--------F6

Selector工具---------F7

Grabber工具---------F8

Scrubber工具--------F9

Pencil工具-----------F10

SmartTool工具----------F6-F7或者F7-F8

激活---PT左上角的“A-Z”按鈕

PT左上角的“A-Z”按鈕

放大縮小 ----------------------- R鍵 和 T鍵

放大選擇區域 ------------------- E鍵

往上依次在各個音軌裡同一點切換光標-----------------P鍵

往下依次在各個音軌裡同一點切換光標-----------------；鍵

按音頻軌的音頻點與MIDI軌的音符位置來前後挪動指針------------L鍵與’鍵

在音頻軌裡以當前光標所在位置為基準，使這個位置之後的音頻為淡出------------G鍵

在音頻軌裡以當前光標所在位置為基準，使這個位置之後的音頻為淡入------------D鍵

選取區域超出了音頻塊範圍進行淡入淡出--------------------F鍵

在音頻軌裡以當前光標所在位置為基準，刪除這個位置之後的音頻-------------S鍵

在音頻軌裡以當前光標所在位置為基準，刪除這個位置之前的音頻-------------A鍵

撤銷上一步動作（可以撤銷N步）-----------Z鍵

將選區內的音頻剪切--------------X鍵

將選區內的音頻複製-------------C鍵

粘貼之前剪切或者複製下來的音頻--------------V鍵

將一段音頻齊光標所在位置切開成兩段---------------B鍵

需要注意的是，如果PT左上角的“A-Z”按鈕沒有被激活，

那麼快捷鍵表裡那些直接使用一個字母鍵的快捷鍵將不會起作用

=======================================================

在建立新音軌的對話框裡上下選擇音軌類型---------CTRL-上下按鈕

開始錄音-----CTRL-空格鍵或者F12或者小鍵盤的3鍵 注意如果輸入法切換的熱鍵為CTRL加空格，那麼就只能用F12和小鍵盤的3鍵了。

停止錄音----空格鍵

停止錄音並丟棄錄製的文件-------CTRL-句號鍵

錄音前先留出兩小節時間----------小鍵盤上的8鍵

MIDI混合錄音與覆蓋錄音切換----------小鍵盤上的9鍵

播放與停止-------空格鍵或者小鍵盤的0鍵

循環錄音開關--------小鍵盤上的5或者ALT-L

選擇多個MIDI音符------按住CTRL鍵然後用鼠標去劃取

選擇鼠標所點擊音區一下的所有MIDI音符------按住SHIFT鍵然後用鼠標點擊一下鋼琴捲簾鍵盤上某個音，那麼在這個音以下的音符全部被選中

在音軌裡將錄音按鈕變為SAFE模式 --------------CTRL-用鼠標單擊音軌上的錄音鍵

在音軌裡將SOLO變為SAFE模式------------------CTRL-用鼠標單擊音軌上的S鍵

定位PLAY/EDIT指針到下一個區域分界線------------TAB鍵

定位PLAY/EDIT指針到上一個區域分界線------------CTRL-TAB

將選擇區域擴展到下一個區域分界線----------SHIFT-TAB

將選擇區域擴展到上一個區域分界線----------CTRL-SHIFT-TAB

播放指針到最開始-----------向上按鈕或者回車鍵

播放指針到最末尾-----------CTRL-回車鍵

手動輸入指針位置---------小鍵盤上的*鍵，然後輸入數字，用左右方向鍵切換數字單位

快進與快倒--------小鍵盤上的1與2

當播放指針在音軌中間，將播放指針之前的區域全部選中---------向下按鈕

在播放的時候，讓播放指針跟隨播放條來定位-------向下按鈕

在播放的時候，使播放指針停放的位置到播放條當前的位置被選中--------向上按鈕

選擇單個音頻區域-----------------在該區域內雙擊鼠標

選擇整個音軌內的所有音頻區域---------CTRL-A

修改NUDGE 值-----------CTRL-ALT-加號或減號

橫向放大縮小音軌顯示-----------CTRL-[或] 或者是 ALT-鼠標滾輪 或者是R與T鍵

縱向放大縮小音頻軌顯示---------CTRL-ALT-[或]

縱向放大縮小MIDI軌顯示--------CTRL-SHIFT-[或]

快速往後翻頁---------SHIFT-鼠標滾輪

向上翻頁-------ALT-PAGE UP

向下翻頁-------ALT-PAGE DOWN

停止播放後播放條或者返回開頭或者停在停止的位置----------重複按N鍵來進行切換

以Nudge裡設置的音符精度來一步步使指針倒退或前進-----《鍵與》鍵或者小鍵盤上的減號鍵與加號鍵

兩倍於Nudge裡設置的音符精度來一步步使指針倒退或前進--------M鍵與、鍵

在眾多音軌裡立即選擇某個音軌------------CTRL-ALT-F

填上音軌號即可馬上找到

打開MIDI事件列表------------ALT-加號（等號）

打開Event Operations模板(在這模板裡可進行很多事件操作，比如量化等等)

ALT-T打開的是Transpose調製面板，

ALT-Y打開的是Select/Split Notes調製面板，

ALT-P打開的是Change Duration調製面板，

其實在裡面的選擇窗口裡可以調出許多其他的調製面板。

給插件的參數快速添加到自動控制窗口-----------CTRL-WIN-ALT-鼠標點擊該插件所需要自動化控制的旋鈕或者推子，在彈出的兩個選項裡選擇第一個即可快速添加進自動控制窗口，

選第二個則可以將其他可供自動控制的參數都可以添加進自動控制窗口

在資源管理器裡打開當前工程文件包，便於查看工程文件內容-------------ALT-O

打開與關閉Workspace-----------------ALT-；

打開與關閉Tasks--------------------ALT-‘

刪除工程文件裡沒有用到的素材------------在右側Regions窗口裡CTRL-SHIFT-U即可選中沒用到的文件，然後CTRL-SHIFT-B即可刪除

在彈出的窗口內選擇STRIP

依次切換工具----------ESC鍵或者CTRL-1 2 3 4 5

依次切換模式----------~鍵

切換音頻軌的波形顯示與音量包絡顯示和切換MIDI軌的音符顯示與REGIONS區域顯示-------------主鍵盤的減號鍵

快速進入某音軌的音量包絡顯示界面---------CTRL-WIN-用鼠標點擊I/O界面裡的VOL

快速進入某音軌的聲相包絡顯示界面---------CTRL-WIN-用鼠標點擊I/O界面裡的PAN

快速進入某音軌的MUTE包絡顯示界面---------CTRL-WIN-用鼠標點擊音軌名稱下的M按鈕

在顯示音量包絡或者別的包絡的時候快速回到波形（WAVEFORM）界面-----CTRL-WIN-用鼠標點擊音軌名稱

速度放慢一倍錄音-----CTRL-SHIFT-空格

放慢一倍速度播放-----SHIFT-空格
