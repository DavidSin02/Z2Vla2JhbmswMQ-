奇異值分解SVD在PACS圖像處理方面應用

2020-11-14
| 科學

這是我前年的一份論文，保存下。

[摘 要] 根據奇異值分解的基本原理及其特點，給出了運用奇異值分解進行PACS圖像壓縮的方法。通過示例詳細說明了該方法進行圖像壓縮的基本過程，給出了壓縮流程，並通過 R編程對實際圖像進行處理，表明了該方法的有效性。

[關鍵詞] 圖像處理；SVD；圖像壓縮；奇異值分解

1、 引言

在醫療信息化系統當中，PACS影像分析是醫生對患者病情一個非常重要的檢查檢驗方法，尤其是進行雙向轉診和遠程互診的時候，需要對圖像進行遠程傳輸，高質量的圖片固然清晰度較高，但對網絡傳輸成本較大，如果能夠有一套方案能夠對部分圖像進行壓縮，而且是可調節壓縮比進行，這樣就能夠確保需要優質圖片的時候進行很小壓縮或者不壓縮，而對於要求質量不高的可以進行深度壓縮。基於奇異值分解在理論和實際應用目前都得有較好的發展。本文基於SVD的理論對醫學影像圖片進行可調節壓縮。

2、 SVD理論簡介

奇異值分解定理：

設 A&isin;，則存在正交矩陣：

U = [] &isin; ，V = []&isin;，使得A=U=，其中=diag()，並且&ge;&ge;&hellip;&ge;>0，，，&hellip;，是矩陣A的奇異值，r=rank(A)是矩陣A的秩。

奇異值分解的神奇之處在於可以找到任意矩陣的奇異值分解，也即任意一個矩陣A都可以分解為正交矩陣U，對角矩陣，正交矩陣V，其中中的各元素按大小排序就是A矩陣的奇異值，它表示了對應向量的縮放信息，而U矩陣的各列就是矩陣A變換的主軸。

3、圖片處理

圖像格式包括jpg/jpeg/bmp等格式，圖像由像素點構成，每個像素點包含了RGB三原色的基本信息，也即從RGB角度來看，每個點的色彩信息是RGB組成的一個值，一般由16進制來表示，如FFFFFF表示白色，000000表示黑色，每兩位代表一個顏色值，如下圖1所示：

圖1

對原圖每個像素點進行分析，得出像素點的組成信息，然後分別形成R、G、B3個矩陣，分別對3個矩陣進行SVD操作，再進行選取合適的奇異值個數k，k值需要小於等r(秩)，k越接近於r，和原圖的匹配程度越高。

基於性能的考慮，採用R語言進行處理，R語言在向量處理方面具有先天優勢，能夠具有最少代碼工作量完成複雜向量工作，同時R在各個不同平臺都可以運行。為了更加方便，採用RStudio IDE進行開發。在進行開發之前需要安裝相關SVD和圖像處理的包，如：

install.packages("rARPACK")

install.packages("jpeg")

接下來按照先後順序進行處理

1） 加載需要的包

library(rARPACK);

library(jpeg);

2）讀取指定的影像圖片

rawimg = readJPEG("img1.jpg");

rawimg是一個3維數組，包括RGB的結構信息，下圖給出詳細信息，大小為162.7M

3）進行因式分解，主要用到svds方法，為了方便後面的壓縮比率，增加一個可調節參數k，方法詳細情況如下

factorize = function(m, k)

{

r = svds(m[, , 1], k);

g = svds(m[, , 2], k);

b = svds(m[, , 3], k);

return(list(r = r, g = g, b = b));

}

這樣獲得了3組奇異值分解的結果，封裝到list列表

4）之後進行數據還原，根據A=U來進行向量運算，函數如下：

recoverimg = function(lst, k){

recoverinner = function(fac, k){

dmat = diag(k);#構造矩陣對角線

diag(dmat) = fac$d[1:k];#對對角矩陣進行賦值

m = fac$u[, 1:k] %*% dmat %*% t(fac$v[, 1:k]); # A=U

m[m < 0] = 0;

m[m > 1] = 1;

return(m);

}

r = recoverinner (lst$r, k); #R分量還原

g = recoverinner (lst$g, k); #G分量還原

b = recoverinner (lst$b, k); #B分量還原

m = array(0, c(nrow(r), ncol(r), 3));

m[, , 1] = r;

m[, , 2] = g;

m[, , 3] = b;

return(m);

}

4、結論

本文討論了一種基於SVD的圖像壓縮方法，壓縮的基本原理是用較少的壓縮矩陣（）代替原始圖像矩陣，再使用該壓縮矩陣生成近似圖像，根據奇異值分解基本原理，可以得到U，V。通過調節不同的k取值來達到不同的壓縮比效果，實際工作中根據實際情況進行選擇，本文通過完整實驗進行了該方法的驗證，確實是一種過程簡單，行之有效的壓縮方法，具有一定的應用前景和擴展程度。
