如何在Python中將列表切成非零整數的連續組！只需一行代碼搞定

2020-10-29
| 科學

似乎無法在網上找到有關此問題的線索，也無法自己弄清楚它，因此：

我將如何對列表進行切片，以便返回連續非零整數切片的列表。即：

data = [3, 7, 4, 0, 1, 3, 7]

我想生產：

slices = [[3, 7, 4], [1, 3, 7]]

我嘗試了各種遍歷該列表的方法，一直傾向於使用一個生成器，該生成器通過測試前後是否存在0來讓我知道連續的組何時開始和停止，但是我有些困惑。

解決方案

import itertools[ list(x[1]) for x in itertools.groupby(data, lambda x: x == 0) if not x[0] ]
