蓄電池安裝有哪些要求？

2020-11-14
| 科學

蓄電池組的安裝應按已批准的設計圖紙及產品技術文件的要求進行施工。

蓄電池在運輸過程中，應輕搬輕放，不得有強烈衝擊和振動，不得倒置、重壓和日晒雨淋。

蓄電池到達現場後，應進行驗收檢查，並應符合下列規定：

1. 包裝及密封應良好。

2. 應開箱檢查清點，型號、規格應符合設計要求，附件應齊全，元件應無損壞。

3. 產品的技術文件應齊全。

4. 按本規範要求外觀檢查應合格。

蓄電池到達現場後，應在產品規定的有效保管期限內進行安裝和充電，不立即安裝時，其保管應符合下列規定：

1. 酸性和鹼性蓄電池不得存放在同一室內；

2. 蓄電池不得倒置，開箱後不得重疊存放；

3. 蓄電池應存放在清潔、乾燥、通風良好的室內，應避免陽光直射；存放中，嚴禁短路、受潮，並應定期清除灰塵。

4. 閥控式密閉鉛酸電池宜在5~40℃的環境溫度，相對溼度低於80%的環境下存放；鎘鎳鹼性蓄電池宜在-5~35℃的環境溫度，相對溼度低於75%的環境下存放。蓄電池從出廠之日起到安裝後的初始充電時間超過六個月時，應採取充電措施。

蓄電池施工應制定安全技術措施。

蓄電池安裝前，建築工程及其他輔助設施按設計要求全部完成，並驗收合格。

蓄電池室應採用防爆型燈具、通風電機，室內照明線應採用暗敷，室內不得裝設開關和插座。

閥控式密封鉛酸蓄電池組

蓄電池安裝前，應按下列規定進行外觀檢查：

1. 蓄電池外觀應無裂紋、無損傷；密封應良好，應無滲漏；安全排氣閥處於關閉狀態。

2. 蓄電池正、負端接線柱應極性正確，應無變形、無損傷。

3. 透明的蓄電池槽，應檢查極板無嚴重變形；槽內部件應齊全，無損傷。

4. 連接條、螺栓及螺母應齊全。

清除蓄電池表面汙垢時，對塑料製作的外殼應用清水或弱鹼性溶液擦拭，不得用有機溶劑清洗。

蓄電池組的安裝應符合下列規定：

1. 蓄電池放置的基架及間距應符合設計要求；蓄電池放置在基架後，基架不應有變形；基架宜接地。

2. 蓄電池在搬運過程中不應觸動極柱和安全排氣閥。

3. 蓄電池安裝應平穩，間距應均勻，單體蓄電池之間的間距不應小於5mm；同一排、列的蓄電池槽應高低一致，排列應整齊。

4. 連接蓄電池連接條時應使用絕緣工具，並應佩戴絕緣手套。

5. 連接條的接線應正確，連接部分應塗電力複合脂。螺栓緊固時，應用力矩扳手，力矩值應符合產品技術文件的要求。

6. 有抗震要求時，其抗震設施應符合設計要求，並應牢固可靠。

電纜的引出線正、負極的極性及標識應正確，且正極為赭色，負極為藍色。蓄電池電源引出電纜不應直接連接到極柱上，應採用過渡板連接。電纜接線端子處應有絕緣防護罩。

蓄電池組的每個蓄電池應在外表面用耐酸材料標識編號。

蓄電池組安裝完畢後，應按產品技術文件的要求進行充電，並應符合下列規定：

1. 充點前應檢查蓄電池組及其連接條的連接情況。

2. 充電前應檢查並記錄單體蓄電池的初始端電壓和整組電壓。

3. 充電期間，環境溫度為5~35℃，蓄電池表面溫度不應高於45℃。

4. 充電期間，充電電源應可靠，不得斷電。

5. 充電過程中，室內不得有明火；通風應良好。

蓄電池組安裝完畢投運前，應進行完全充電，並應進行開路電壓測試和容量測試。

達到下列條件之一時，可視為完全充電：

1. 蓄電池在環境溫度5~35℃條件下，以（2.4V±0.01V）/單體的恆定電壓、充電電流不大於2.5I10（10h率放電電流）（A）充電至電流值5h穩定不變時。

2. 充電後期充電電流小於0.005C10（10h率額定容量）（A）時。

3. 符合產品技術文件完全充電要求時。

完全充電的蓄電池組開路靜置24h後，應分別測量和記錄每隻蓄電池的開路電壓，測量點應在端子處，開路電壓最高值和最低值不得超過下表：

蓄電池容量測試應符合下列規定：

1. 蓄電池在環境溫度5~35℃的條件下應完全充電，然後應靜放1h~24h，當蓄電池表面溫度與環境溫度基本一致時，應進行10h率容量放電測試，應以0.1C10（A）恆定電流放電到其中一個蓄電池電壓為1.80V時終止放電，並應記錄放電期間蓄電池的表面溫度t及放電持續時間T。

2. 放電期間應每隔一個小時測試並記錄單體蓄電池的端電壓、表面溫度及整組蓄電池的端電壓。在放電末期應隨時測量。

3. 在放電過程中，放電電流的波動允許範圍為規定值的±1%。

4. 實測容量Ct（A*h）應用放電電流I（A）乘以放電持續時間T（h）計算。

5. 當放電期間蓄電池的表面溫度不為25℃，可按下式實測放電容量折算成25℃基準溫度時的容量：

1. 放電結束後，蓄電池應儘快進行完全充電。

2. 10h率容量測試第一次循環不應低於0.95C10，在第三次循環內應達到1.0C10，容量測試循環達到1.0C10可停止容量測量。

蓄電池組的開路電壓和10h率容量測試有一項數據不符合本規範的規定時，此組蓄電池應為不合格。

在整個充、放電期間，應按規定的時間記錄每個蓄電池的電壓、表面溫度和環境溫度及整組蓄電池的電壓、電流，並應繪製整組充、放電特性曲線。

蓄電池充好電後，應按產品技術文件的要求進行使用與維護。

鎘鎳鹼性蓄電池組

蓄電池安裝前應按下列規定進行外觀檢查：

1. 蓄電池外殼應無裂紋、損傷、漏液等現象。

2. 蓄電池正、負端接線柱應極性正確，殼內部件應齊全無損傷；有孔氣塞通氣性能應良好。

3. 連接條、螺栓及螺母應齊全，應無鏽蝕。

4. 帶電解液的蓄電池，其液麵高度應在兩液麵線之間；防漏運輸螺塞應無鬆動、脫落。

清除蓄電池表面汙垢時，對塑料製作的外殼應用清水或弱鹼性溶液擦拭，不得用有機溶劑清洗。

蓄電池組的安裝應符合下列規定：

1. 蓄電池放置的平臺，基架及間距應符合設計或產品技術文件的要求；蓄電池放置在基架後，基架不應有變形；基架宜接地。

2. 蓄電池安裝應平穩，間距應均勻，單體蓄電池之間的間距不應小於5mm；同一排、列的蓄電池應高低一致，排列應整齊。

3. 連接蓄電池連接條時應使用絕緣工具，並應佩戴絕緣手套。

4. 連接條的接線應正確，連接部分應塗以電力複合脂。螺栓緊固時，應用力矩扳手，力矩值應符合產品技術文件的要求。

5. 有抗震要求時，其抗震設施應符合設計規定，並應牢固可靠。

電纜引出線正、負極的極性及標識應正確，且正極應為赭色，負極應為藍色。蓄電池組電源引出電纜不應直接連接到極柱上，應採用過渡板連接。電纜接線端子處應有絕緣防護罩。

蓄電池組的每個蓄電池應在外表面用耐鹼材料標明編號。

配製電解液應採用化學純氫氧化鉀，配製電解液應用蒸餾水或去離子水。

電解液的密度應符合產品技術文件的要求。

配製和存放電解液應用鐵、鋼、陶瓷製成的耐鹼器具，不得使用配製過酸性電解液的容器。

配液時，應將鹼慢慢傾入水中，不得將水倒入鹼中。配製的電解液應加蓋存放並沉澱6h以上，應取澄清液或過濾液使用。對電解液有懷疑時應化驗。

注入蓄電池的電解液溫度不宜高於30℃；當室溫高於30℃時，應採取降溫措施。其液麵高度應在兩液麵線之間，注入電解液後靜置2~4h後再初充電。

配液工作應由具有施工經驗的技工操作，操作人員應戴專用保護用品，並應設專人監護。

工作場地應備有含量3~5%的硼酸溶液。

蓄電池的初充電應按產品技術文件的要求進行，並應符合下列規定：

1. 初充電期間，其充電電源應可靠，不得斷電。

2. 初充電期間，室內不得有明火；通風應良好。

3. 裝有催化栓的蓄電池應將催化栓旋下，待初充電完成後再重新裝上。

4. 帶有電解液並配有專用防漏運輸螺塞的蓄電池，初充電前應取下運輸螺塞換上有孔氣塞，並檢查液麵不應低於下液麵線。

5. 充電期間電解液的溫度範圍宜為20℃±10℃；當電解液的溫度低於5℃或高於35℃時，不宜進行充電。

蓄電池初充電達到產品技術文件所規定的時間，同時單體蓄電池電壓應符合產品技術文件的要求。

蓄電池初充電結束後，應按產品技術文件的規定做容量測試，其容量應達到產品使用說明書的要求，高倍率蓄電池還應進行倍率試驗，並應符合下列規定：

1. 在3次充、放電循環內，放電容量在20±5℃時不應低於額定容量。

2. 用於有衝擊負荷的高倍率蓄電池倍率放電，在電解液溫度為20±5℃條件下，應以0.5C5電流值先放電1h情況下繼以6C5電流值放電0.5s，其單體蓄電池的平均電壓，超過倍率蓄電池不得低於1.1V；gap倍率蓄電池不得低於1.05V。

3. 按0.2C5電流值放電終結時，單體蓄電池的電壓應符合產品技術文件的要求，電壓不足1.0V的電池數不應超過電池總數的5%，且最低不得低於0.9V。

充放電結束後，應用蒸餾水或去離子水調整液麵至上液麵線。

在製造廠已完成初充電的密封蓄電池，充電前應檢查並記錄單體蓄電池的初始端電壓和整組總電壓，並應進行補充充電和容量測試，補充充電及其充電電壓和容量測試的方法應按產品技術文件的要求進行，不得過充、過放。

放電結束後，蓄電池儘快進行完全充電。

在整個充放電期間，應按規定時間記錄每個蓄電池的電壓、電解液溫度和環境溫度及整組蓄電池的電壓、電流，並應繪製整組充、放電特性曲線。

蓄電池充好電後，應按產品技術文件的要求進行使用和維護。

蓄電池安裝後的驗收

1. 蓄電池的建築工程及其輔助設施應符合設計要求，照明燈具和開關的形式及裝設位置應符合設計要求。

2. 蓄電池安裝位置應符合設計要求，蓄電池組應排列整齊，間距應均勻，應平穩牢固。

3. 蓄電池間連接條應排列整齊，螺栓應緊固、齊全，極性標識應正確、清晰。

4. 蓄電池組每個蓄電池的順序編號應正確，外殼應清潔，液麵應正常。

5. 蓄電池組的充、放電結果應良好，絕緣電阻不應小於0.5MΩ。

其他規定

蓄電池組臺架安裝牢固，間距符合設計要求；同一排、列蓄電池高低一致，排列整齊。

蓄電池連接條與端子、電纜與蓄電池連接正確，螺栓緊固，接觸部位應塗有電力複合脂；有抗震要求時，其抗震措施應符合設計要求。

蓄電池電纜引出線極性標誌正確，正極為赭色，負極為藍色；電纜孔洞應用耐酸、耐鹼性材料封堵。

鉛酸蓄電池組在5次充放電循環內，當溫度在25℃時，放電容量不應低於10h率放電容量的95%。

充放電結束後，透明槽內鉛酸蓄電池外形應完好，無彎曲變形或活性物質剝落，電解液液麵在兩液麵之間。

鹼性蓄電池組充放電結果應符合下列規定：

1. 在5次充放電循環內，當電解液溫度在20±5℃時，放電容量不低於額定容量；當放電的電解液初始溫度低於15℃時，放電容量應按製造廠提供的修正係數進行修正；

2. 用於有衝擊負荷的高倍率蓄電池還應進行倍率放電校驗，在電解液為20±5℃條件下，以0.5C5電流值先放電1h情況下，繼續以6C5電流值放電0.5s，其單體蓄電池的平均電壓應為：高倍率蓄電池不低於1.05V，超高倍率蓄電池不低於1.1V；

3. 按0.1C5電流放電結束時，單體蓄電池的電壓應符合製造廠的規定，電壓不足1.0V的電池數不應超過電池總數的5%，且最低不得低於0.9V。

C5為鹼性蓄電池的額定容量值，指電池在環境溫度為20±5℃條件下，以5h率放電至終止電壓時應提供的電量，單位為Ah（安時）或mAh（毫安時）。

蓄電池組的絕緣應良好，絕緣電阻不應小於0.5MΩ。
