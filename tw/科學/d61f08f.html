編程與法律契合探索（1）——侵權行為因果關係的數據結構。

2020-10-29
| 科學



一、法律規定

我國《民法總則》第一百二十條規定：“民事權益受到侵害的，被侵權人有權請求侵權人承擔侵權責任。”

第一百八十六條規定：“因當事人一方的違約行為，損害對方人身權益、財產權益的，受損害方有權選擇請求其承擔違約責任或者侵權責任。”

二、法律概念：

在民法理論上，根據《民法總則》的有關規定並結合侵權行為的歸責原則理論，可將侵權行為分為一般侵權行為和特殊侵權行為兩大基本類型。一般侵權行為是指行為人基於過錯致人損害而應承擔民事責任的行為。

一般侵權行為的構成要件，是指在一般情況下，構成侵權行為所必須具備的因素。只有同時具備這些因素，侵權行為才能成立。一般侵權行為的構成要件包括：有加害行為、有損害事實的存在、加害行為與損害事實之間有因果關係、行為人主觀上有過錯四個方面。

三、一般侵權行為的數理邏輯表示：

A == 有加害行為

B == 有損害事實存在

C == 加害行為與損害事實之間有因果關係

D == 行為人主觀上有過錯

一般侵權行為的成立 == A ∧ B ∧ C ∧ D

四、契合探索思路：

人類活動是高度複雜的社會化活動，而法律規範是在人類政治、經濟、生活、社會中形成的高度科學、理性，符合社會發展趨勢和發展規律而形成的統一規範，法律規範必須要符合發展規律，也要符合倫理道德、公序良俗等方面。王澤鑑先生也認為：侵權行為責任法律規範，是在“行動自由”與“權益保護”之間如何平衡。立法、司法的目的也就在於此。因此，王老先生堅持“相當因果關係說”，因果關係的認定傾向於保護被害人。但是什麼是“相當”，怎麼量化“相當”？實踐中是個難題。因此，侵權責任特別是一般侵權責任的認定，不能單純割裂四種構成要件來判斷，而應該你中有我，我中有你，綜合衡量。

例如： A加害行為，不能單純從加害行為範疇考慮。而是應該從A、B、C、D四個範疇綜合考慮加害行為。同理，B有損害事實存在，C加害行為與損害事實之間有因果關係，D行為人主觀有過錯。每個問題的判斷，也應該從A、B、C、D四個範疇去考慮。這樣構成要件的判斷，就變成了三個子問題的判斷（判斷最終因果關係是目的）。當然中間可能有較多重複和假設，不同的人持不同觀點立場不同，可能得出的數據結論也有異同。這並不關鍵，因為即使不拆分為子問題，一樣是不同的人持不同觀點。通過算法來拆分的問題，並形成多種因果關係，目的是把法律問題的事實和因果關係契合的顆粒度更加細化，建立一種建築結構，來羅列所有的因果關係（包括假設和重複），通過論證和分辨，以取得最符合保護被害人利益的“相當因果關係”（其實這就是律師庭審前要做的準備功夫）。

四、實例探索

《施工造成傷害引發敗血症致人死亡案》，是最高人民法院頒佈的一宗典型民事侵權案例。案例如下：

１９８６年１０月，某建築工程公司將其承包的一項拆除工程轉包給個體工商戶張某經營的工人新村青年合作服務站，並簽訂了承包合同。１９８６年１１月１７日，張某委託丈夫徐某負責施工，指揮臨時工張某等人拆除混凝土大梁。在拆除過程中，張某因第６根大梁折斷而滑落墜地受傷，急送醫院檢查，診斷為左下踝關節內側血腫壓痛，活動障礙。１２月７日，張某因傷口感染化膿治療無效死亡。法醫鑑定結論為：張某系左內踝外傷後，引起局部組織感染、壞死，致膿毒敗血症死亡。後又經區醫療事故鑑定委員會鑑定認為，張某系因外傷所致膿毒敗血症死亡，與其他因素無關，醫院治療無誤。張某死亡後，其父親和姐弟向法院提起訴訟，請求人身損害賠償。被告以死者生前與其簽訂了“工傷概不負責任”的合同條款而拒絕承擔責任。
