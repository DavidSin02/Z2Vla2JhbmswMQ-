“資本老手”易賢忠攜百奧泰闖關科創板，長袖善舞引出背後“風雲人物”董正青

2020-10-29
| 科學

資本高手易賢忠攜未盈利的百奧泰捲土重來戰A股，而一切的背後不簡簡單單是急需補血的公司，還有斥資佈局醫院的戲碼，甚至早年間叱吒風雲的董正青。

近日上交所披露，百奧泰生物製藥股份有限公司(以下簡稱“百奧泰”)將於11月20日首發上會。今年10月末的格樂立（百奧泰研製的阿達木單抗）獲批為百奧泰其登陸資本市場造了不少聲勢。

然而這家創新藥企業是在易賢忠的裹挾下一步步前進，背後更是隱藏著早年間資本市場的風雲人物。

現在提及易賢忠可能很多人會覺得陌生，但要是提起分眾傳媒的江南春，想必大家肯定都是耳熟能詳。而兩者之間的淵源要從私有化的分眾傳媒在2015年借殼七喜控股講起。而彼時七喜控股的實控人就是易賢忠。

距今四年之久，此次闖關科創板。長袖善舞的資本大佬易賢忠意欲再次登上A股平臺“起舞”資本。

//

“資本老手”易賢忠

//

二十多年前，離開國企創業的易賢忠或許也不會想到有一天他會成為攪動資本市場的大佬。

2004年，易賢忠帶領七喜控股登陸深交所。隨著公司的3C產品在移動互聯網浪潮下日漸衰退，上市公司的業績也是一路下滑。機緣之下，易賢忠的七喜控股成為彼時分眾傳媒進入A股的踏板。

2013年5月，分眾傳媒從美股私有化。在對賭之下公司必須在2016年前在A股上市。可是萬萬沒想到的是，2015年分眾傳媒原本計劃借殼的宏達新材，但因後者遭到調查被迫取消了借殼。於是幸運的七喜控股成為分眾傳媒登上A股的殼。

2015年9月初，七喜控股公告，以截至擬置出資產及負債的方式與分眾傳媒全體股東持有的分眾傳媒100%股權的等值部分進行置換。根據評估報告，交易資產分眾傳媒100%股權的評估值為458.7億元。這對於業績頹勢盡顯的七喜控股無疑是天大的好事。

來源：東方財富

2016年3月8日，在聚光燈下作為第一支迴歸A股的中概股，江南春也將原七喜控股的名稱也將更名為分眾傳媒。借殼後，上市公司股價大漲，而原實控人易賢忠持有分眾傳媒2.36%股權，在此後兩年間易賢忠陸續高位減持套現十幾億元，掙的是盆滿缽滿。

然而完成借殼之後，易賢忠也將原七喜電腦資產置出，轉到七喜集團（由易賢忠、關玉嬋夫婦和兒子易良三人持股100%持股）。與此同時資本市場大獲全勝的易賢忠也並沒有選擇頤養天年，而是開始做大自己旗下其他產業，例如百奧泰。

百奧泰由李勝峰2003年7月28日成立，由於生物醫藥行業前期投入較大，李勝峰於在2009年末決定引入可以持續提供資金支持的易賢忠等人。與此同時，也包括以0元轉讓百奧泰60%股權。

易氏家族入主百奧泰9年後，在長袖善舞的易賢忠操盤下百奧泰衝刺科創板。易賢忠也再次踏入闊別幾年的A資本市場。

來源：天眼查

七喜集團目前由易賢忠、關玉嬋夫婦和兒子易良昱（持股98.53％）三人持股，根據七喜集團官網顯示，其目前擁有 IT、地產、大健康三大業務板塊，旗下包括兩家製藥企業、三家醫療器械企業和三家醫療投資運營企業。

而醫院建設似乎也變成了七喜集團的主流操作。2016年10月，易氏家族注資50億元，成立廣州聖德醫療管理有限責任公司，這揭開了其醫院版圖擴張的序幕。此後的2016年11月和12月，常德腫瘤、臨沂腫瘤、曲靖腫瘤和河南豫東腫瘤4家醫院相繼成立。2017年、2018年，易氏家族每年都新增了4家醫院。

這些易氏家族不斷通過七喜集團在背後擴張醫院版圖。不難猜出，再次起舞資本市場，易忠賢或將上演一邊A股市場融資，一邊斥資佈局興建醫院的戲碼。

市場沉浮幾十年，易賢忠對資本運作的老練也非常人能比。易賢忠家族為百奧泰的實控人，合計控制著69.31%股權。掌控一切的易賢忠在上市前突擊分紅3.5億。

來源：招股書

實際控制人不具備相關專業知識，公司經營高度依賴核心技術人員，所以便有了“為了吸引、保留和激勵人才”合情合理的藉口，百奧泰在易忠賢的操控下實施了股權激勵措施。這一操作百奧泰股權激勵獲益最大的也就是這一切的策劃者。股權激勵支付對象包括實際控制人之一的易賢忠，而且4.46%的股份支付比例在71位股權激勵對象中最高。

//

“戰友”董正青

//

本次百奧泰上市的聯席主承銷商為廣發證券，這個與易忠賢有著緊密聯繫的券商再次出現在大眾視野。而那個核心人物就是早年間攪動資本市場的風雲人物“董正青”。

1993年左右，董正青加入廣發證券，十年之後董正青升任廣發證券總裁。而在2004年，易賢忠的七喜控股登上資本市場，彼時廣發證券為其保薦機構和主承銷商。同為湖北潛江人的董正青和易賢忠的淵源從此開始。

2007年，廣發證券借殼上市前，董正青因涉內幕交易入獄。四年刑滿之後，董正青出獄，將匯天澤作為核心資本運作平臺。匯天澤的初始股東為易賢忠和易陽平，這家公司成立的最初目的就是替董正青承接所持廣發證券股權，董正青出獄前後，匯天澤的大股東已從易賢忠更換成董正青。此後董正青聯手老友舊部頻頻出入資本市場。

來源：天眼查

在易賢忠將百奧泰招股書遞交的那一刻，其與董正青在百奧泰詳細的資本往來也被掀開。

根據招股書顯示，2016年12月，七喜集團投資3.5億元，吉富啟恆、合肥啟興、匯智富聯合向百奧泰投資2.7億元。而其中吉富啟恆、合肥啟興、匯智富均由吉富創投控制。而後者正是昔日廣發證券員工持股平臺，董正青為吉富創投第一大股東和實際控制人。2018年12月，粵創三號、興昱投資、浥塵投資、匯天澤向百奧泰合計投資10億元。其中，董正青持有匯天澤98.88%股權。2019年3月，百奧泰再次增資，興昱投資、粵科金融集團、中科卓創、匯天澤、吉富啟恆、浥塵投資共計投資5億元。

來源：天眼查

除去易氏家族，百奧泰的主要投資方就是董正青控制的吉富創投和匯天澤。招股書顯示，吉富啟恆、合肥啟興、匯智富、匯天澤四者為一致行動人。董正青控制的企業合計投入了4.4億元，合計持股9.2631%，為百奧泰第三大股東。而董正青掌控的匯天澤最重要的醫藥產業就是投資了老朋友易賢忠的百奧泰。至此十幾年的你來我往，兩人之間的關係早已可以不言而喻。

而曾經把七喜控股賣給分眾傳媒借殼的易賢忠，如今再次大張旗鼓的出現在A股資本市場，或許這也將是這位多財善賈的大佬繼續利用上市公司來完成自己謀劃的開始。
