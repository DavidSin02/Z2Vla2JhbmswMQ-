實例分析｜沸石轉輪吸附濃縮+RTO處理傢俱廠噴漆廢氣

2020-10-29
| 科學

摘 要：針對傢俱噴漆過程VOCs成分複雜、風量大、濃度低的實際情況,本文重點介紹了沸石(分子篩)轉輪吸附濃縮+熱力燃燒VOCs治理技術,該技術特別適用於處理大風量、低濃度有機廢氣,可以有效提高處理效率、降低運行成本,VOCs去除效率可達到93%以上。

1 引言

噴漆是傢俱製造行業最主要工藝之一，在很大程度上決定了傢俱製品的外觀、色彩、質量、檔次與價值。就當前傢俱行業來說，VOCs主要來自於噴漆工序中塗料中有機溶劑和稀釋劑的揮發，而且不同種類的塗料在使用過程中所排放的VOCs種類和含量也不同，塗裝相同面積時，使用溶劑型塗料產生的v o c s量最多，水性塗料次之，粉末塗料最少。雖然目前國家要求傢俱行業大力推廣使用水性、紫外光固化塗料，到2 0 2 0 年底前，替代比例達到6 0 % 以上，但是鑑於傢俱行業噴漆程水性、紫外光固化塗料技術的不成熟，業內目前仍以溶劑型塗料為主，噴漆過程產生大量的漆塵和VOCs，因此對傢俱行業VOCs治理提出了很高的要求。

以無錫某大型家居企業為例，企業設置兩條噴漆線，進行木門和浴室櫃噴塗，噴塗廢氣具體包括:調漆廢氣、噴漆廢氣、烘乾廢氣，主要汙染物為醋酸乙酯、苯乙烯、二甲苯、醋酸丁酯、乙醇、甲基異丁酮、異氰酸酯單體、2 - 羥基- 2 - 甲基- 1 - 苯基- 1 - 丙酮、甲苯、2- 甲基- 2 - 丙烯酸- 2 - 羥乙基酯磷酸酯、丙烯酸、3- 氯- 1 , 2 - 環氧丙烷、乙苯、甲基乙基酮、環己酮、石油、1 ，2 , 4 -三甲基苯、乙二醇、顆粒物，該過程風量約為 215000m3/h ,噴塗、VOCs 產生濃度為 120 ~ 3 3 0 mg/m3，該廢氣具有風量大、濃度低、廢氣組分複雜、無回收價值等主要特點。

對於大風量、低濃度的有機廢氣經過沸石分子篩吸後，可轉換成小風量、中高濃度的有機廢氣，因此企業噴漆廢氣選用沸石轉輪吸附濃縮+ RTO 處理，具有淨化效率高、投資低等優點，可以確保噴漆廢氣達標排放，工藝技術合理、可行。

2 沸石轉輪濃縮+ RTO技術原理

本技術是沸石轉輪吸附同蓄熱式焚燒技術的組合工藝，淨化系統主要由三級乾式過濾裝置、沸石轉輪濃縮吸附裝置、RT0、風機、換熱器、PLC自動化控制系統組成。該組合技術通過沸石轉輪的吸附濃縮使大風量、低濃度有機廢氣濃縮為小風量、高濃度濃縮氣體，高濃度濃縮氣再經RTO高溫燃燒分解為(CO2和H2O等無機成分。沸石轉輪濃縮裝置是利用吸附-脫附-濃縮三項連續變溫的吸附、脫附程序，通過轉輪的旋轉，在轉輪(被分割成吸附區、脫附區、冷卻區)上同時完成VOCs的吸附、脫附再生。

組合技術工藝過程:經三級乾式過濾裝置去除粉塵、顆粒物後的有機廢氣流過濃縮轉輪時，其中的有機物在轉輪吸附區域會被吸附下來，經過吸附淨化後的廢氣(約佔處理風量的85% ~ 95% )排放到大氣中，一小部分廢氣(約佔處理風量的5% ~ 15% )對轉輪冷卻區降溫後經換熱器被加熱到180 ~ 220oC的脫附溫度後，流人脫附區，脫附區有機物從吸附劑一沸石上脫離到加熱的氣流中，轉輪得以再生，脫附後的高濃度VOCs被送人RT0高溫焚燒，反應後的高溫煙氣進人規整蜂窩陶瓷蓄熱體，95% 的熱量被蓄熱體吸收並“儲存”起來，溫度降低到接近RTO人口溫度，通常不超過50oC。蓄熱體溫度升高後，通過切換閥或旋轉裝置切換氣流流向，分別進行蓄熱和放熱，實現熱量的有效回收利用。




3 沸石轉輪濃縮+RTO簡介

案例所採用沸石轉輪濃縮+ RTO廢氣處理設備主要包括混合總管單元、三級乾式過濾器、升溫調溼單元、沸石轉輪濃縮單元、蓄熱氧化爐單元。

3.1 三級乾式過濾器

三級乾式漆霧過濾器能較完全地去除粉塵、漆霧，氣體中0.5um 以上的塵淨化效率大於等於98% 。它的原理是通過材料纖維改變漆霧顆粒的慣性力方向從而將其從廢氣中分離出來，材料逐漸加密的多重纖維經增加撞擊率，提高過濾效率。過濾時能有效通過不同過濾材料組合，利用材料空間容納漆霧，達到更高的過濾效率是乾式材料的特有性能。

當過濾系統壓力達到設定報警值時，報警系統發出報警信號，報警信號接人中央控制室，提醒操作人員更換濾材。

3.2 沸石轉輪




沸石濃縮轉輪被分為吸附區、脫附區、冷卻區三個功能區，沸石分子篩轉輪在各個功能區域內連續運轉。

在吸附區：廢氣通過前置的過濾器後，送至沸石分子篩轉輪的吸附區。在吸附區(吸附區面積為S1 )有機廢氣中的VOCs被沸石分子篩吸附，未被吸附的廢氣在吸附風機的帶動下，直接排人煙囪達標排放。在脫附區:沸石轉輪上吸附的VOCs,在脫附區(脫附區面積為S2 ) 被高溫逆向脫附、濃縮，脫附溫度約200oC：，濃縮倍數一般為5 ~ 25倍。濃縮倍數：n = (S1 x V1)/(S2 xV2 ) ，其中 S1/S2 = 10:1 ,V1/V2 = ( 0.5 ~ 2.5 )。脫附氣在脫附風機的帶動下進人RTO焚化系統。

在冷卻區：為保證高的吸附效率，需對高溫脫附後的轉輪進行冷卻。冷卻空氣冷卻轉輪吸附材後自身被預熱，作為脫附氣的源氣，再與來自RTO燃燒室來的高溫淨化氣換熱，溫度提升至180 ~ 200oC 後逆向進人轉輪脫附區進行高溫脫附。

3.3 蓄熱氧化爐

採用三床式RTO。RTO蓄熱式氧化爐，是在高溫下將可燃廢氣氧化成對應的氧化物和水，從而淨化廢氣，並回收廢氣分解時所釋放出來的熱量，廢氣分解效率達到99% 以上，熱回收效率達到95% 以上。主體結構由燃燒室、陶瓷填料床和切換閥等組成。根據客戶實際需求，選擇不同的熱能回收方式和切換閥方式。建設單位將調漆、噴塗廢氣收集後,經過濾除漆霧、吸附濃縮後，引人RTO進行焚燒處置。經焚燒處理後的廢氣通過換熱後，廢氣高空排放。RTO可實現廢氣安全、穩定地焚燒，在工作過程中需定期鼓人新鮮空氣用以維持燃燒，空氣中的氮在高溫條件下氧化會產生熱力型氮氧化物。根據文獻記載，熱力型氮氧化物的生成量主要取決於溫度，在相同條件下，氮氧化物的生成量隨溫度增高而增大，當溫度低於1350oC時，幾乎不生成熱力型氮氧化物，建設項目RTO工作溫度約為760 ~ 850oC,在此情況下，熱力氧化器氮氧化物的產生量極少，因此建設項目廢氣處理系統內不設氮氧化物處理裝置。

蓄熱式焚燒爐的工作原理:將有機廢氣預熱至760oC以上,在燃燒室加熱升溫至800oC左右，使廢氣中的碳、氫、氧元素氧化分解成為無害的CO2和H2O ;氧化時的高溫氣體的熱量被特製的陶瓷蓄熱體“貯存”起來，用於預熱新進人的有機廢氣。從而節省升溫所需要的料消耗，降低運行成本。

4 實際應用效果

無錫某家居企業沸石轉輪吸附濃縮+ RTO 設備已投人運行,VOCs 設計處理效率為95% ,調試運行中目前處理效率平均值為93.4% 。

沸石轉輪吸附濃縮+ RTO設備運行測試結果見表4。




表4顯示，經沸石轉輪吸附濃縮+ RTO設備處理後，VOCs濃度從進口的129 ~327.2 mg/m3 ,降到7.786 ~ 17.3mg/m3,平均去除效率93.4% ,均可以滿足《大氣汙染物綜合排放標準》(GB16297 - 1996)中“新汙染源大氣汙染物排放限值”標準。

5 結論與展望

沸石轉輪吸附濃縮+ RTO工藝先採用轉輪吸附濃縮，降低了 RTO設備廢氣處理量，使得RTO規模變小，降低了設備投資費用;儘管有機廢氣濃度比較低，但經濃縮後，有機廢氣濃度達到自持濃度以上，所以RTO除了首次開機預熱需要消耗天然氣(RTO輔助燃料)外，其他時間RTO輔助加熱系統無需開啟，同時脫附風熱源來自於有機廢氣氧化釋放的熱能，系統運行費用低。沸石轉輪濃縮一RTO淨化系統採用PLC全自動化控制並配備液晶觸摸顯示屏。控制程序監視所有控制器、驅動器、壓力開關、壓力變送器、熱電偶、限位開關、生產線連鎖信號和儀表連鎖信號。如發生任何異常，人機界面就會顯示故障信息，然後通過程序控制採取適當動作，確保系統的安全。

由上分析可知，沸石轉輪吸附濃縮+ RTO設備對於傢俱廠大風量、低濃度噴漆廢氣有良好的去除效果，VOCs總去除效率達到9 3 % ，且控制簡單，安全性能高，最終實現VOCs穩定達標排放。
