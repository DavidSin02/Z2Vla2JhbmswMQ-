2月去哪兒？最適合冬末春初前往的地方，絕不能錯過

2020-10-29
| 科學


​©Matteo Colombo

讓我們先暢想一下，2月剛被春節掏空的狀態，精疲力竭，兩眼無神……毋庸置疑，此時一定想繼續癱一下，或者想要外界給你一個刺激。那不如來看看，LP給你推薦18個2月驚喜目的地！

去廣東湛江吃著生蠔繼續癱，享受一下重慶被火鍋包圍的冬季，貴州隆裡的“花臉龍”可能會帶給你不一樣的舞龍體驗；再向外看看，法國芒通的檸檬節是不是夠酸爽刺激？還有布達佩斯的溫泉……不說了，LP先跳為敬！

- 中 國 -

# 去中國大陸最南端吃吃吃 #

現正是吃生蠔的好時候

地點：廣東，湛江

時間：2月

由圖蟲提供

2月，可能是湛江最舒服的時候了，剛經歷過“嚴寒”的摧殘即將迎來春天，沒有颱風的侵擾，氣溫20度上下。天氣雖好，但現在更是吃生蠔的好時候。

由圖蟲提供

湛江雞和湛江生蠔在整個廣東名氣甚響。這裡的海鮮種類之多，國內數一數二，你還可以嘗試當地盛產的對蝦，也可以挑戰奇異的沙蟲。湛江口碑好的餐館有不少，大多性價比高，記得帶上一個足夠大的胃。

由圖蟲提供

若想徹底領教湛江這座海鮮之城的厲害，一定要去巨大的霞山水產品批發市場看看從未見過的稀奇海產，這裡還是國內最大的對蝦交易中心。最地道的體驗方式就是來這邊親自挑，然後就地拿到市場內外的餐廳加工（加工費8元/斤起），絕對新鮮，性價比也高，吃起來當然更過癮。


# 去中國最北端體驗一場極寒 #

節後2月冰雪依舊，但遊人稀少

地點：黑龍江，漠河

時間：2月初

由圖蟲提供

趁著人少，東北的冬天還沒過去，再去漠河撒點野。中國版圖的最北端，年平均氣溫-5℃，極端最低氣溫-50℃，冬季長達8個月，夏至日照17個小時……一連串的數據都讓漠河顯得那麼與眾不同，再貼上亮眼的“極光”標籤，更是點燃了無數人對漠河的憧憬和想象，也給這個遼遠蒼茫的東北盡頭帶來了絡繹不絕的來客。

由圖蟲提供

對於匆忙的旅行者來說，這裡有清晰的“打卡”路線，但倘若放慢腳步，黑龍江水和莽莽林海勾勒出的壯闊風光比精算地理座標更能成全你的尋北之旅。

由圖蟲提供

繼續北上才是正道，如今，除了大肆開發的北極村，以4分緯度超越它的北紅村也漸漸被人們所關注。要是時間充裕，探訪一下鮮為人知的洛古河源頭，再沿著黑龍江遊覽到第一灣，你的行程路線恰好描繪了一段中國版圖的最北疆界。


# 隆裡元宵節 #

趕在正月十五，去貴州體驗一下花臉龍

地點：貴州，隆裡

時間：中國農曆正月十五

由圖蟲提供

貴州人好像天天都在過節，無論何時加入這場遍及全省的嘉年華，都不晚！今年二月，我們推薦你去隆裡，在這個黔東南古城，在最熱鬧的元宵節上，“花臉龍”狂歡節不容錯過。

由圖蟲提供

花臉龍也許是整個黔東南最有特色的漢族節慶活動。每年正月十五，扛龍人都畫著五顏六色的臉譜，按生旦淨末醜的角色成隊舞龍，一路用手裡的灰去抹路人，來不及躲的會變成花臉。而花臉龍的看點不只在舞龍本身，畫臉譜的手藝也是明朝傳下的絕學。

由圖蟲提供

沒趕上現場，可以去關聖宮街（木馬街）的開科第，參觀扎龍工藝坊；正陽門甕城內的濟陽第內有位老爺子是花臉龍的傳承人，也可找他聊聊；龍標書院裡也闢有相關展廳。


# 沒有什麼冷，是泡溫泉解決不了的 #

溫泉+滑雪，好不愜意

地點：內蒙古，阿爾山

時間：2月初

由圖蟲提供

“酒香不怕巷子深”，儘管阿爾山只是一座藏在大興安嶺腹地的中蒙邊境小城，卻依舊引得無數人為它遺世獨立的美遠道而來。冬日大雪紛飛，被大雪包裹著的密林，變成了一番讓人心動的景象。

由圖蟲提供

冬日前往阿爾山，除了滑雪之外，不得不泡口口相傳的溫泉，最為推薦的是中國溫泉博物館，相傳成吉思汗戰馬受傷，馬蹄浸入溫泉後神奇癒合，大汗驚呼“哈倫阿爾山”，意思是“熱的聖泉”，這便是阿爾山的由來。

由圖蟲提供

現在東山腳下溫泉群的37個泉眼都被收藏在興林路南端的中國溫泉博物館內，你可以用連環泡的方式來參觀。不同溫泉池對應不同疾病有療效，包括牛皮癬、風溼、運動傷等。

# 如夢似幻的雪景 #

九寨溝，一年四季都好看

地點：四川，九寨溝

時間：2月，看命

©Dulyanut Swdp/Getty Images

九寨溝的名氣讓周邊所有的“溝”都黯然失色，昂貴的門票雖然花得心疼，但“人間仙境”“童話世界”卻是九寨美景的真實寫照。“九寨歸來不看水”，光那108個充滿了夢幻色彩般不可思議的海子就足夠令人如痴如醉，更別說還有疊瀑、彩林和雪山風光。

由圖蟲提供

9月下旬至10月中旬，它的美到達極致，五彩斑斕的樹林倒映在晶瑩透亮的海子裡，水中靜靜躺著凝固時光的巨木殘枝。

© roman korzh,CC BY-NC-ND 2.0

九寨溝現已開放散客遊覽，二月前往九寨溝，如果運氣好下雪，那遠處巍峨的雪山，和近處飄落進海子的雪花，一定能讓你不停按下快門。


# 在雲南找一處溫暖地 #

看雲起雲落

地點：雲南，瀘沽湖

時間：2月底-4月

由圖蟲提供

當你翻山越嶺地來到川滇交界，站在晶瑩剔透、倒映著白雲蒼山的浩渺湖泊面前，當你圍著篝火和摩梭姑娘小夥牽手共跳甲搓舞，當你走進木楞房，沉下心聽祖母屋的故事……當了解取代了獵奇，你才能體會到瀘沽湖真正的與眾不同。

由圖蟲提供

有“高原明珠”之稱的瀘沽湖海拔2690米，被青山環繞。泛舟湖上，如同浮在一個大果凍上，湖水能見度有12米，最深處達93米，肉眼看不見任何雜質，除了偶爾在水底盪漾的水草。

由圖蟲提供

這裡年平均氣溫17℃，即使在炎夏也不會很熱。二月初此地氣溫較低，我們建議你二月底之後前往，不僅有可能看到草長鶯飛，還有可能邂逅正在開放的桃花。


# 嘉陽小火車 #

工業文明的活化石與山間田野的春天，相得映彰

地點：中國 四川省犍為縣

時間：2月末至4月

由圖蟲提供

火車吐著陣陣白煙，徐徐穿越金黃油菜花海，你可能在許多地方看過這樣一幅畫面，而它，每年都在犍為縣的嘉陽礦區真實上演。行駛於芭石鐵路的嘉陽小火車始建於20世紀50年代，原本用於運煤，後成了當地人的主要交通方式，而今，又加入了慕名搭乘的遊客。它是現存唯一仍運營的窄軌蒸汽火車，有“工業革命活化石”之稱。

由圖蟲提供

鐵路沿線共有8個站點，起於石溪，終於黃村井，往返一趟約2.5小時，其中別名三井或嘉陽的躍進站是多數人的出發點。蜜蜂巖和菜子壩是拍攝火車與油菜花合影的絕佳地點，前者還有詹天佑設計的“人字形”掉頭軌道。

由圖蟲提供

芭溝是線路上最熱鬧的小鎮，鎮內工業與“文革”風格的老建築頗多，適合懷舊，鎮外不遠處有隧道能拍攝火車駛出洞口的畫面；終點站黃村井有抗戰時留下的礦井遺址，可以下井體驗。


# 為了吃一次火鍋 #

去一趟3D重慶

地點：重慶

時間：1-3月

由圖蟲提供

你有過，為了一個食物說走就走的經歷嗎？如果沒有，那現在可以有。去重慶吧，入冬後的重慶並不寒冷，吃貨們可以盡情享受山城火鍋的溫暖。旅遊淡季帶來優惠的價格，但天氣陰霾，難見陽光。

由圖蟲提供

吃完火鍋幹嘛去？先去對重慶的網紅3D交通有個初步的瞭解。坐個索道纜車，體驗一把穿樓而過的輕軌。

由圖蟲提供

再去感受一下，別樣的重慶。羅漢寺——這座始建於北宋平治年間、有著“西川佛都”之稱的千年古剎一直隱身滾滾紅塵之中，卻因一部電影《瘋狂的石頭》而出名。進入雕飾繁複的大門後，走廊兩邊長約20米的古佛巖是宋代的摩崖石刻。

由圖蟲提供

還想感受一下不一樣的歷史，湖廣會館絕不容錯過，長江邊，東水門，湖廣會館依山而建，樓閣層疊，號稱中國最大的古代會館建築群。它其實由廣東公所（南華宮）、齊安公所（黃州會館）和禹王宮（湖廣會館）等組成，主要建築距今已有近300年曆史。

由圖蟲提供

# 巡遊尼羅河 #

趕在又一波埃及熱開始之前，好好逛逛尼羅河

地點：開羅至阿斯旺

時間：2月

©clu/Getty Images

這是一段最經典的遊河之旅。尼羅河兩岸搖曳的棕櫚樹、田地上的耕牛、泥磚建成的農房和古老的石建神廟——在幾百年、甚至幾千年的時間裡都幾無改變。而尼羅河也遠不僅是一條河，它是埃及的命脈，也是傳說中陽界（河水以東）和陰間（河水以西）之間的分界。

©Calin Stan/Getty Images

乘船遊覽尼羅河也是埃及最具代表性的體驗。2月，開羅的氣溫在18℃左右，阿斯旺在25℃左右，天氣最完美。

©Peter Seaward/Lonely Planet

2月的尼羅河水量也最豐沛，河岸灘塗鬱鬱蔥蔥，遊船也暢通無阻——如果在低水位期間來，泊船地點可能會受到限制，某些河段還需要遊客下船乘大巴，繞行之後再與遊船會合。這樣一來，哪裡還能體會到古埃及的那種浪漫味道呢？

©Anton_Ivanov/Shutterstock


# 檸檬節上享酸爽 #

"我酸了”

地點：法國裡維埃拉地區的芒通

時間：2月中旬至3月初

©cmfotoworks/Getty Images

檸檬在法國南部的地位可是舉足輕重的。小城芒通（Menton）就以可愛的檸檬知名。這座小城位於地中海岸邊，不遠處便是意大利。

© Cappuccino & Baguette，CC BY-NC-ND 2.0

芒通在亞熱帶微氣候影響之下，很利於這種富含精油、色澤鮮黃的水果成長，檸檬節誕生於20世紀30年代，最初的慶祝活動，不過是推著幾車果樹在小鎮街道上游行，現在則發展成了一個香氣撲鼻的狂歡節。在每年2月中旬至3月初為期3周的時間，大約會有25萬人蜂擁而至，向這種世界級水果表達虔誠的敬意。

©Laurent Carles，CC BY-SA 2.0

節日期間，會有花車遊行，花車就用檸檬裝點，主題每年都不同，造型各異，某些水果雕塑甚至高達10米，會用掉140噸的檸檬。遊客可以流連於氣味香甜的幻想花園，日夜觀看遊行表演，最後也要舉起一杯檸檬飲料，向這種美妙的水果致敬。

©Garsya/Shutterstock


#哇！盛裝亮相#

戴上面具，在水道縱橫的威尼斯狂歡

地點：意大利 威尼斯

時間：2月16日-3月5日

©Krzysztof Dydynski/Lonely Planet

威尼斯狂歡節（Carnevale Venizia）之華麗，不輸里約狂歡節之癲狂，是世界上最著名的巴洛克風情化妝舞會。

©Eleonora Tuveri/Shutterstock

狂歡節的歷史至少可以追溯到15世紀，節日的官方開始時間是週六16:00左右，頭戴面具的遊行隊伍從聖馬可廣場（Piazza San Marco）出發。

©Juliet Coombe/Lonely Planet

亮點活動包括大公舞會——其實你只要身著盛裝，就可以參加。大運河上還有裝飾一新的船和貢多拉遊行，主題活動之外也會有許多其他活動。全城個廣場上都會擠滿街頭藝人，聖保羅廣場有時還會設起溜冰場。買一張面具，沉浸在狂歡氛圍之中吧！

© jimmyweee，CC BY 2.0


# 去卡拉哈迪遊獵 #

雨水將於本月降臨

地點：博茨瓦納和南非

時間：2月

©Johan Swanepoel/500px

卡拉哈迪（Kgalagadi）意為“乾涸的土地”。這座跨境公園佔地52,000平方公里，條件艱苦，但野生動物卻很豐富。

©EcoPrint/Shutterstock

2月雨水一到，乾涸的土地會瞬間變成生機勃勃的綠洲，改變整片地區的景象。此時空氣中的那種興奮感很難用言語準確傳達：雄性羚羊會開始劃設地盤，並彼此角力，爭奪“後宮佳麗”的配偶權，非洲獵豹和黑鬃獅等獵捕者則潛伏於四周。

©Martin Mecnarowski/Shutterstock

最便利的遊覽方法是從公園位於南非境內的部分啟程。沿路分佈有水泵，由此產生的水坑周圍是觀察野生動物的絕佳地點。

©HannesThirion/Getty Images


# 欣賞黑脈金斑蝶 #

隨著陽光漸強，黑脈金斑蝶們也愈加活躍

地點：墨西哥，米卻肯州埃爾羅薩里奧蝴蝶保護區

時間：2月至3月

©JHVEPhoto/Shutterstock

墨西哥當地的村民把黑脈金斑蝶（monarch butterflies）叫作“las palomas”，意為“逝去的孩子的靈魂”——因為自打他們有記憶以來，這些蝴蝶每年都會在“墨西哥亡靈節”（11月1日）準時現身。這些村民不知道自己實際上已經解開了20世紀最令人費解的野生動物謎團之一：北美1.2億黑脈金斑蝶每年冬天都去哪裡了？但直到1975年，黑脈金斑蝶在墨西哥過冬的奇聞才在世界上傳開。

©Noradoa/Shutterstock

這種蝴蝶在4～5個月的時間裡幾乎會一動不動，但等到第二年2月天氣一轉暖，便會騷動起來。很快，這些蝴蝶便會在周圍地面上飛來飛去，尋找水和花蜜，數量之密集，彷彿一場遮天蔽日的橙色暴雪。

©Warren Price Photography/Shutterstock

可從墨西哥城（Mexico City）乘長途汽車抵達安甘格奧（Angangueo），再搭乘交通工具或是徒步前往埃爾羅薩里奧（El Rosario）。


# 日本小眾地，不小眾的雪祭 #

秋田可不止有秋田犬

地點：日本，秋田縣橫手

時間：2月15-16日

©Shibuya246, CC BY-NC 2.0

位於日本海與宏偉的奧羽山脈（Oū-sanmyaku）和出羽山脈之間的秋田縣，可能被大多數旅行者記住的只有，秋田犬和“此生必去”的乳頭溫泉。

© Shibuya246,CC BY-NC 2.0

殊不知，如果在二月前往秋田這個小眾地，還能趕上擁有400年曆史的橫手雪祭。橫手位於秋田縣東南面，溫泉資源豐富。

© Shibuya246,CC BY-NC 2.0

每年2月雪祭，更是會吸引日本國內大量遊客前往，雪祭期間特地搭建的雪洞（可在雪洞中用餐），以及在重要景點處設置的雪燈，都能吸引你駐足，2020年的雪祭日期為2月15-16日。

# 選一個海島，開始海島癱 #

就決定是你啦！

地點：泰國，素空島和裡邦島

時間：2月

©diegocardini/Getty Images

遊人罕至的素空島是大自然的天堂，這裡有黃褐色的海灘、淺綠色的大海和叢林密佈的黑巖海角。島上的高腳棚屋居住區內生活著2600多名以穆斯林為主的居民，他們的稻田、西瓜地和橡膠園分佈在狹窄的混凝土道路兩側。

©Sidelnikov Timur/500px

裡邦島是董裡群島中面積最大的島嶼，從大陸上的長灘（Hat Yao）乘長尾船過來只需要30分鐘。裡邦島的遊客比鄰近的島嶼更少，但它卻是一顆燦爛的明珠。

©Karntiphat Changrua/Shutterstock

島上群山蔥蘢，隨處可見橡膠樹，沿海分佈著茂密的紅樹林，島上極富魅力的動植物群（尤其是棲息於此的儒艮和遷徙的鳥類）比狹窄的金褐色海灘更加出名。總之，它們倆都非常適合你去海島癱。

©Petr Malyshev/Getty Images


# 與鯨鯊“戲水” #

和世界上最大的魚一同戲水

地點：菲律賓，呂宋島棟索爾

時間：11月至次年5月

©Fata Morgana by Andrew Marriott/Shutterstock

1998年以前，棟索爾（Donsol）不過是個安閒無名的小村。那一年，一位當地人在潛水時拍下了鯨鯊的視頻。幾天後，一家報紙以為發現了新物種，於是撰文報道，偷獵者聞風而至。幸好獵捕鯨鯊隨後遭禁——但這卻沒有阻止遊客蜂擁而至，觀賞這些“溫柔的巨人”。

©IRWIN ANG/500px

今天的棟索爾是世界上與這種最大的海洋哺乳動物共同戲水的頂級目的地之一。在它們旁邊游泳也好，浮潛也好（禁止深潛），絕對會讓人興奮不已。2月末至4月是這一活動的高峰期，在此期間，能否看到鯨鯊根本不是問題，問題是你到底能看到多少。

© Adrian Samiro See/Lonely Planet


# 東歐溫泉之旅 #

便宜，舒服又好玩的布達佩斯，誰不愛？

地點：匈牙利，布達佩斯

時間：1-3月

©Zsolt Hlinka/Getty Images

布達佩斯的魅力——無論是自然美景還是人造景觀，都令其卓爾不群。這座城市橫跨蜿蜒曲折的多瑙河兩岸，被西岸的布達山（Buda Hills）和東側的大平原起點所夾峙。

©Izabela23/Shutterstock

一趟沒有和當地人一起泡溫泉的布達佩斯之旅是不完整的，而在這座“溫泉之都”中最負盛名的莫過於具有悠久歷史的塞切尼溫泉了。

©vidalgo/Shutterstock

這座宮殿般華麗的浴場位於布達佩斯市區，龐大的場地中包含三個室外溫泉池，以及若干個市內池。室外池當中還有一個有50米的標準泳道，要求帶著泳帽使用。

©Li Kim Goh/Getty Images

冬天在熱氣氤氳的室外池裡和匈牙利大爺擺好棋盤對弈一局，邊品熱紅酒邊欣賞周遭飄落的雪花，讓從內到外的愜意帶走所有寒冷和疲憊。

©Heracles Kritikos/Shutterstock
