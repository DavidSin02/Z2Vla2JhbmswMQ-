LeetCode算法筆記74：搜索二維矩陣

2020-11-14
| 科學

題目描述

編寫一個高效的算法來判斷 m x n 矩陣中，是否存在一個目標值。該矩陣具有如下特性：

每行中的整數從左到右按升序排列。

每行的第一個整數大於前一行的最後一個整數。

就這？[看]

本文答案參考自 LeetCode 官方題解。

因為題目說了每行中的整數從左到右按升序排列。每行的第一個整數大於前一行的最後一個整數。

搞得這個矩陣像一條扭曲的繩子一樣，即 數組。

所以我們可以矩陣視作數組。

設 位於矩陣第 row 行，第 column 列的元素 對應 數組下標為 index 的元素

則 row = index // n ， column = index % n

然後就是一個經典的二分法啦~[耶]
