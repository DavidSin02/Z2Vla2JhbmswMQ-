一、選擇題(共6小題，每小題4分，共24分)

1．基因突變按其發生部位可分為體細胞的突變a和生殖細胞的突變b兩種。則

( )。

A．均易發生於有絲分裂的間期

B．a易發生於有絲分裂的間期，b易發生於減數第一次分裂前的間期

C．均只能發生於減數第一次分裂前

的間期

D．a只能發生於有絲分裂間期，b只能發生於減數第二次分裂前的間期

解析　基因突變是鹼基對排列順序的改變，由於DNA複製時結構不穩定所以易導致基因突變，DN

A複製發生在細胞分裂的間期，體細胞只能進行有絲分裂，生殖細胞的產生是減數分裂的結果。

答案　B

2．如果一個基因的中部缺失了1個核苷酸對，可能的後果是(多選) ( )。

A．

沒有蛋白質產物

B．翻譯為蛋白質時在缺失位置終止

C．所控制合成的蛋白質減少多個氨基酸

D．翻譯的蛋白質中，缺失部位以後的氨基酸序列發生變化

解析　缺失一個鹼基對，屬於移碼突變，自缺失處開始不再合成為原來的蛋白質，可有幾種情況：一是缺失後正好成為終止密碼子，可能翻譯過程到此為止；二是自缺失處翻譯一段後提前終止；三是一直翻譯下去，但自缺失處氨基酸序列發生很大變化。

答案　BCD

右圖為馬的生活史

，下列有關的敘述中，正確的是 ( )。

①有絲分裂發生在a、d

②基因重組發生在c

③基因突變可發生在a、b、d

④d過程有基因的複製、轉錄和翻譯

A．①②③ B．①④ C．①③④ D．②③④

解析　基因重組發生在有性生殖的減數分裂過程中，而減

數分裂發生在原始生殖細胞產生生殖細胞的過程中，即題圖中b。雌雄個體產生性原細胞是通過有絲分裂實現的，即題圖中a。c表示受精作用。由受精卵發育成生物體過程中有細胞分裂和分化。分裂間期有基因的複製和表達，細胞分化時也有基因的表達即轉錄和翻譯。

答案　C

4． 如圖為高等動物的細胞分裂示意圖，圖中不可能反映的是

( )。

A．發生了基因突變

B．發生了染色單體互換

C．該細胞為次級卵母細胞

D．該細胞為次級精母細胞

解析　圖示細胞處於減數第二次分裂後期，且細胞質均分，不可能是次級卵母細胞。

答案　C

5．下列關於基因重組的敘述，正確的是 ( )。

A．基因型為Aa的個體自交，因基因重組而導致子代性狀分離

B．基因A因替換、增添或缺失部分鹼基而形成它的等位基因a

，屬於基因重組

C．非姐妹染色單體間的互換可能導致基因重組

D．造成同卵雙生姐妹間性狀上差異的主要原因是基因重組

解析　基因型為Aa的個體自交，因等位基因分離導致子代性狀分離。B項屬於基因突變。D項中同卵雙生姐妹遺傳物質相同，造成性狀上差異的主要原因是環境影響。

答案　C

6．某基因的一個片段CGTAG α中的αDNA鏈在複製時一個鹼基由G→C，該基因複製3次後，發生突變的基因佔該基因總數的 ( )。

A．100% B．50% C．25% D．12.5%

解析　DNA的複製是半保留複製，以突變的α鏈為模板所得子代DNA全異常，以未突變的β鏈為模板所得子代DNA全正常。

答案　B
