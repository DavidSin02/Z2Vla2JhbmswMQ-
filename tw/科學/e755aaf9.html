上傳照片測面相、看運勢……真相太可怕

2020-11-14
| 科學

最近，一項“黑科技”刷遍朋友圈

只要在線掃碼後上傳個人照片

就可以在線進行“面相測試”

可是這樣的測試真的靠譜嗎？

有不少人還在微博和朋友圈

晒出了自己的測試結果

但也有不少網友轉發稱

“不要進行掃碼面相測試，

用戶面部識別信息可能會被盜用”

對此，小編決定親身試驗，一探究竟

//

面相測試，“運程報告”需付費

//

小編隨機在朋友圈裡找到一個可以進行面相測試的信息，根據信息

掃碼進入應用界面後可以看到，“面相測試”提示要獲得用戶的公開信息，包括暱稱、頭像、地區及性別，在選擇“允許”後才可進行下一步。

在上傳一張照片後，系統再次提示要確認照片是本人。點擊“是本人照片”後，系統有幾十秒的分析時間，隨後生成面相概述、五官評分和五官的介紹等。

儘管生成的是五官介紹，但小編髮現，除了眼睛和嘴巴等介紹外，其他部分的測試內容需要付錢才可以解鎖解讀。

而且除了第一次測試免費外，第二次之後測試則需要付費9.9元。其他測試比如“面相事業運程報告”和“面相情感運程報告”等，都需要付費19.8元。

//

一張照片測出不同結果

//

上傳的照片經過系統分析後

五官評分給出了65分的分數

但當小編用另一部手機

再次再次上傳同一個人的照片的時候

系統卻又出了85分

//

不同鏈接指向同一個平臺

//

除此之外小編還發現

儘管朋友圈中分享的面相測試的鏈接看似不同

但很多鏈接點開後

都打開的是由杭州算術科技有限公司開發的

同一個測試平臺

//

生物信息不能輕易洩露

//

那這種測試到底會不會洩露信息呢？

2月20號，北京青年報的記者聯繫到該公司推廣經理薛先生。薛先生對測試洩露信息予以否認。薛先生稱，用戶如果不放心數據安全，在進行測試後可以自行刪除報告，並稱公司在隱私保護方面做到的是歐盟通用數據保護條例標準，且有專業的運維工程師維護。

而DCCI互聯網數據中心創始人胡延平認為，用戶對這一類需要上傳照片的應用要高度警惕，原則上能不參加就不參加。胡延平介紹，涉及到用戶上傳照片的行為已經不只是隱私問題，而是用戶的網絡安全問題，“生物信息是個人信息安全的最後一道防線，最後一道防線不能輕易洩露，不能輕易放棄。”

胡延平進一步解釋了用戶要注意個人信息的原因：

其一，對於目前很多互聯網公司和開發者來說，他們的安全防護水平以及數據保護水準都還比較低，存在著信息洩露的風險。

其二，人的生物信息，比如指紋、視網虹膜和人臉識別，這三個方面的關鍵信息一定不能輕易提交。“如果僅僅是用戶名和密碼，可以隨時更改，但是指紋信息和麵部特徵等只有一次，不能隨意更改。”胡延平強調。

360信息安全部移動安全團隊負責人周燁也對記者表示，在整個測試過程中，多個環節會存在信息洩露風險；相關程序開發公司所在服務器的安全性如果不高，也容易被黑客攻擊。另外，相關公司或者團伙如果是灰黑色產業，本身就是為收集數據所開發的相關程序。
