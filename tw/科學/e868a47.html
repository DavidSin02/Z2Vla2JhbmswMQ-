什麼是滲透率曲線S-Curve模型？

在介紹S-Curve模型之前，先介紹一下產品生命週期理論(Life Circle)。

產品生命週期理論認為一個產品在市場上的需求，會經歷以下四個階段。



S-curve模型，就是用數學模型描述產品的前三階段：如下圖藍顏色標出部分。
其中虛線代表了峰值也就是市場飽和點。


S-curve模型的X軸，通常是時間；
Y軸，通常是產品的滲透率。

波士頓諮詢公司（BCG）曾對多種科技產品的迭代做過滲透率變化的研究，比如下圖中的上世紀50-90年代美國家庭彩電的滲透率變化，基本上符合S-curve模型。


模型的前提假設

S-curve模型的前提假設：
假設產品在不同的時期，增長率有所不同——增長率會經歷從慢變快再變快的過程。某些情況下，變化的增長率，會比勻速或者指數增長更貼近商業情況。

假設產品滲透率跟其剩餘的市場潛力的比率，會保持一個固定的變化速度。這裡如果理解有困難的話，可參考下面的數學模型中的解釋，會更清晰些。

假設有一個峰值或者飽和點，然後這個產品的市場佔有率或者市場表現會無限接近這個飽和點。

模型的數學原理

數學模型為：
Ln ( P/ (L-P) ) = a * T + b

其中，
Ln是以自然對數運算（當然可更換成其他底數如log10(X)，不影響模型預測結果）；
P是滲透率，L是峰值也就是飽和點，T是時間點；
a, b是待擬合的參數。

作圖表示模型：

如果將以上數學模型在X-Y座標系中作圖表示的話，
下面的左圖的y軸是P滲透率，右圖的y軸是自然對數運算後的P/ (L-P)；
x軸左右兩圖相同，都是T時間點。

可以看到，ln (P / (L-P) ) 跟T是線性關係，即斜率為常數。

對數處理後是T的線性關係，則代表對數處理之前的P / (L-P)，是T的指數關係，即維持固定的變化率。


關於對數(log)處理的解釋，可以參考我之前的一篇文章：數據可視化：使用對數刻度


對數學模型的商業理解：
P可以理解成產品已達成的市場佔有率；L-P可以理解成產品剩餘的市場潛力。

P/ (L-P) 可以理解成產品的滲透率/產品的剩餘市場潛力。

P/ (L-P)為指數變化，代表著產品的滲透率/產品的剩餘市場潛力，維持著固定的變化率。這一固定變化率，就是前文中模型假設中提到的第二點。

再進一步講，P的升高，可能是對全新市場的培育，也可能是對舊產品的擠出。

模型的應用場景

S-curve應用場景：

- 預測產品的滲透率和增長率

- 預測市場最佳進入市場時機

- 預測市場飽和點

- 預測市場拐點

初步商業結論：

新產品A的增長勢頭正勁，但是可以看到從19年開始，P增速開始下降，可以考慮市場是否已經進入成熟期。

三句話小結

S-curve假設產品滲透率跟剩餘市場潛力保持固定變化比率，同時存在一個滲透率上限值L，產品滲透率增速會由慢變快再變慢，而選擇選擇哪種底數做對數轉換不影響模型結果。

S-curve雖然是一個簡單的數學模型，但是很好地尊重和解釋了產品生命週期理論。

對S-curve的模型結果如何解讀，需要對商業事實有一個很好的理解和判斷。
