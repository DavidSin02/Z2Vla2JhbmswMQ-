髕骨軟化—膝關節疼痛不再是老年專屬病，它困擾著越來越多年輕人

2020-11-14
| 科學

膝關節，作為我們人體重要的關節，每天都承擔著大量勞力性工作，在大家的印象中，膝關節疼痛只會發生在老年人群裡，殊不知，有一種疾病—髕骨軟化，正逐漸困擾著越來越多的年輕人群。喜歡運動的年輕人，包括我本人在內，經常在運動以後出現膝關節前方的疼痛，很多朋友經常打電話問我：為什麼我一打完籃球膝關節就很痛，休息休息就能好了，我的膝關節到底怎麼了？今天我們來講一講常常引起膝關節疼痛的常見疾病—髕骨軟化症。

什麼是髕骨軟化症？

髕骨是全身最大的籽骨，上極與股四頭肌腱相連，下極由髕韌帶固定於脛骨結節。其關節面和股骨內、外髁相處形成髕股關節，膝關節屈伸活動時，髕骨在股骨內、外髁間由近到遠呈"S"形滑動。髕骨軟化症（chondromalacia patellae）是髕骨軟骨面因慢性損傷後，軟骨腫脹、侵蝕、龜裂、破碎、脫落，最後與之相對的股骨髁軟骨也發生相同的病理改變，而形成髕股關節的骨關節病，常常會造成髕股疼痛。髕股疼痛(patellofemoral pain, PFP)是常見的過度使用性疾病，累及髕股區，通常表現為膝前疼痛。髕股疼痛的特徵為髕骨周圍或髕骨後疼痛，會由一種或多種在屈膝負重時造成髕股關節負荷的活動加重，在初級保健醫生、骨科醫生及運動醫學專家接診的膝關節痛病例中，髕股疼痛是最常見的原因。

髕骨軟化症的病因有哪些？

1. 先天性髕骨發育障礙、位置異常及股骨髁大、小異常；或後天性膝關節內、外翻，脛骨外旋畸形等，均可使脛骨不穩定，在滑動過程中髕股關節面壓應力集中於某點，成為慢性損傷的基礎。

2. 膝關節長期、用力、快速屈伸，增加髕股關節的磨損，如自行車、滑冰運動員的訓練時本病的常見原因。

3. 髕骨軟骨的營養主要來自關節滑液，各種原因所導致的滑液成分異常，均可使髕骨軟骨營養不良，易受到輕微外力而產生退行性變。

髕骨軟化的臨床表現有哪些？

1. 青年運動員較多見。初期為髕骨下疼痛，開始訓練時明顯，稍加活動後緩解，過久訓練又加重，休息後膝關節疼痛逐漸消失。隨著病程的發展，疼痛時間多於緩解時間，以至於不能下蹲，上、下臺階困難或突然無力而摔倒。

2. 髕骨邊緣壓痛：伸直膝關節後，擠壓或推動髕骨可有摩擦感，伴有疼痛。單純髕骨軟化損害時，無關節積液，後期形成髕股關節骨關節炎時，可繼發滑膜炎而出現關節積液，此時浮髕實驗陽性（浮髕試驗是確定膝關節損傷時是否存在關節腔積液的方法。患腿膝關節伸直，放鬆股四頭肌，檢查者一手擠壓髕上囊，使關節液積聚於髕骨後方，另一手食指輕壓髕骨，如有浮動感覺，即能感到髕骨碰撞股骨髁的碰擊聲；鬆壓則髕骨又浮起，則為陽性。）。病程長者有股四頭肌的萎縮。

3. X線片早期無異常，晚期可見髕骨邊緣骨贅形成，髕股關節面不平整、光滑或間隙狹窄。X線片尚可發現部分病因，如小髕骨、高位髕骨或股骨外髁低平等先天性畸形。

4. 放射性核素骨顯像檢查時，側位顯示髕骨侷限性放射性核素濃聚，有早期診斷意義。

得了髕骨軟化症，我們可以採取哪些治療方案？

在治療髕骨軟化症時，我們通常採取的是保守治療。

1. 出現髕骨軟化的症狀後，首先制動膝關節1-2周，同時進行股四頭肌抗阻訓練，以增強膝關節穩定性。

2. 腫脹、疼痛突然加重時，應該進行冷敷，48小時後可改為溼熱敷和理療。

3. 氨基葡萄糖有助於軟骨中蛋白粘多糖的合成。即可止痛，又能有利於軟骨修復。

4. 關節腔內封閉治療雖然可以緩解症狀，但由於抑制糖蛋白、膠原的合成，對軟骨修復不利，所以要慎用該方法。

5. 關節腔內注射透明質酸有助於改善症狀。

6. 嚴格非手術治療無效且先天性畸形者，可以考慮手術治療。

手術的目的：1.增加髕骨在關節活動過程中的穩定性，如外側關節囊鬆解術等2.刮出髕骨關節軟骨上面較小的侵蝕病灶，促進修復。3.髕骨關節軟骨已經完全破壞者，可用髕骨切除方法減輕髕股關節骨關節炎的發展。

除了以上治療方法，日常生活中我們可以通過哪些鍛鍊治療呢？

其他康復治療包括進行某些鍛鍊來強健膝關節周圍的肌肉。醫生或護士會向您展示應進行哪些鍛鍊，或者會讓您與理療師(運動專家)合作。醫生或護士還可能推薦您：

1. 穿戴膝關節支具以支撐膝部

2. 用繃帶按特定方式纏繞膝部以提供支撐

3. 使用針對您足部定製的特殊鞋墊(以防您的足部過度內旋或外旋)

而且我們可以通過以下預防訓練減輕或避免髕骨軟化症的症狀，預防訓練由以下訓練組成：

1. 髖關節靠牆等距外展

2. 前弓步

3. 單腿下臺階

4. 單腿下蹲

5. 股四頭肌、髂脛束、膕繩肌及腓腸肌拉伸

什麼情況下需要患者及時到醫院就診？

若有以下情況，應前去就診：

1. 膝關節無法承重、"卡死"或"發軟"。

2. 膝關節嚴重腫脹和疼痛。

3. 出現發熱伴膝關節疼痛、腫脹和發紅。

4. 自行治療數日後膝關節疼痛加重或未見好轉。

日常生活中，我們如何預防髕骨軟化症？

關鍵要了解髕骨軟化症病因，採取科學的保健和治療方法並應及早進行。

1. 認識髕骨軟骨的生理性磨損規律：膝關節作為負重關節，伴隨著年齡的增長必然會帶來膝關節慢性磨損。15歲以前：膝關節處於發育階段，疼痛多發在膝關節附近，小孩的所謂"生長痛"多見。15歲～30歲：膝關節處於"如日中天"，不知疲倦地用。但能緩衝衝擊髕骨軟骨僅是3～5毫米厚，由於無神經分佈，不知痛或疼痛1～2天自己就緩解了，所以在15～50歲很長一段時間無"預警信號"。30歲～40歲：有輕度磨損，為脆弱期，可出現短期的膝關節周圍痠痛，常常我們不會引起重視。40歲～50歲：經常在走遠路後，膝關節就出現痠痛，應該開始保養關節了。50歲以上：膝關節疼痛常發而明顯，軟骨快全層磨損，關節炎開始發生（它是髕骨軟骨的"使用壽命"接近終點的信號）。人的膝關節軟骨退變，在30歲以後就開始了。45歲以下人群骨關節炎患病率僅為2%，而65歲以上人群患病率高達68%，可見自然磨損進展慢，不正常磨損才致病，所以對膝關節的保健應及早進行。有髕骨軟化症要及早矯正髕股關節關節兩部分對合不正常，不要症狀重了才治療。

2. 避免劇烈運動：避免不科學的持續性蹲位和劇烈的運動，如爬山、爬樓梯等膝關節屈曲位負重用力的鍛鍊。長時間保持膝關節半屈曲狀態，這絕對不適合髕骨軟化症患者，因為在屈膝35°～50°範圍內髕骨向外滑出力加大，髕骨軟骨所受到壓力和摩擦也最大，所以不少人進行這種體育鍛煉後出現症狀加重。人們要懂得如何避免不合理的膝蓋負重：躺下時，膝蓋的負重幾乎是0。站起來和走路時負重大約是上半身體重1～2倍。上下坡或上下階梯的時候，是3～4倍。跑步時，則是4倍。打球時，膝蓋的負重大約是6倍。蹲和跪時，膝蓋的負重大約是8倍。由此可見膝關節對於負重的重要性。

3. 保持健康的體重：保持健康的體重以降低作用於膝關節上的重力，研究表明肥胖會增加膝關節的退行性疾病的危險。所以減輕體重，就是減輕膝關節的負擔。

4. 股四頭肌對於膝關節穩定性的重要性：目前文獻報道關節周圍肌肉較強壯、發達時，關節的骨性關節炎發病率可降低80%，發生時間也較晚。股四頭肌保持膝關節穩定性方面起著重要的作用，它的強壯和發達有利於穩定膝關節，不但可治療髕骨軟化症還可減少膝關節內不正常的撞擊，可減少骨性關節炎發病率，使您的膝關節保持健康。股四頭肌鍛鍊如下圖：

●僅使用旨在提高膝關節、髖關節和核心肌群肌力的理療一般就可改善病情，訓練部位應包括髖外展肌和股四頭肌。
