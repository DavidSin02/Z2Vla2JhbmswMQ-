胃腸鏡檢查需注意哪些問題

2020-10-29
| 科學

來源： 中國醫學科學院腫瘤醫院服務號


一、什麼是電子內窺鏡？


它是一種可插入人體體腔進行直接觀察、診斷、治療的集光、機、電等高精尖技術於一體的醫用電子光學儀器。腫瘤醫院內鏡中心常規開展胃鏡、腸鏡、無痛胃腸鏡的診療工作。



二、胃鏡適應症：


1.存在上消化道症狀（如燒心、吞嚥困難、上腹痛、嘔吐等），懷疑有食管、胃、十二指腸病變，臨床需要確診者。

2.已確診的上消化道病變，如消化性潰瘍、食管癌、胃癌等疾病治療後需要隨訪或觀察療效者

3.消化道出血，病因及部位不明者

4.影像學檢查發現上消化道病變，需明確性質者

5.上消化道異物者

6.需內鏡下治療者，如食管胃底靜脈曲張行套扎或注射硬化劑治療，早癌行內鏡下黏膜剝離術（ESD）等

7.胃癌家族史及其他胃癌高危人群

8.存在幽門螺桿菌感染，需明確是否有胃黏膜病變者，或需要進行幽門螺桿菌培養及藥物敏感性試驗以指導治療者。


三、胃鏡禁忌症


絕對禁忌：


  1. 各種原因導致的休克
  2. 嚴重的心肺疾病、昏迷、癲癇發作、主動脈瘤、腦出血
  3. 懷疑食管、胃、十二指腸急性穿孔
  4. 急性重症咽喉部疾病


相對禁忌：


心肺功能不全，出血傾向伴血紅蛋白低於50g/L,高度脊柱畸形，食管或十二指腸巨大憩室等


四、胃鏡檢查前後的注意事項：


胃鏡檢查前需注意：


  1. 提前進行腔鏡前血清檢查的化驗
  2. 檢查前禁食水六小時，前一天少渣飲食(如稀粥、軟飯、爛麵條、肉末、菜泥等)，儘量不飲用牛奶或酸奶等影響檢查效果的食物，禁菸酒。
  3. 高血壓患者可正常服藥；如果長期服用阿司匹林、波立維、三七等抗凝成分的中西藥提前告知醫生。
  4. 就診時攜帶以往的就診病歷和檢查結果以便醫生參考
  5. 檢查前應取下義齒（假牙），眼鏡等物品，以免影響檢查。
  6. 注意調整好心情，保持良好情緒，如有不適要立即告知醫生
  7. 檢查要求有直系親屬陪同，以便交代病情及簽字


胃鏡檢查後注意事項：


  1. 檢查結束後兩個小時內不要喝水、進食，以免誤入氣管引起嗆咳或發生吸入性肺炎。
  2. 檢查中需要取病理活檢，術後兩小時宜進食溫涼流質飲食，如米湯、菜湯等，並禁止吸菸飲酒，喝濃茶，以減少對胃黏膜創傷面的刺激。
  3. 檢查後眼部可能會有疼痛及異物感或出現聲音嘶啞，一般3-5天內會好轉
  4. 結合身體情況適當休息，不宜做劇烈運動
  5. 檢查後三天內可留意大便顏色是否正常，如果出現柏油樣黑便，要及時就醫。
  6. 無痛胃鏡的患者術後不能駕車，不能從事高空、精細作業以防意外。


五、腸鏡檢查的適應症


  1. 原因不明的下消化道出血，包括便血和持續性便潛血陽性。
  2. 存在下消化道症狀，如腹瀉、腹痛、便祕、腹部包塊、大便習慣改變等可疑結直腸疾病。
  3. 鋇劑灌腸或影像學檢查發現可疑病變不能定性。
  4. 炎症性腸病做鑑別診斷或需要確定病變範圍、嚴重程度等以及治療後的複查。
  5. 結直腸腫瘤術後或息肉切除術後的定期隨訪。


六、腸鏡檢查的禁忌症


  1. 嚴重的心肺功能不全
  2. 休克
  3. 大的主動脈瘤
  4. 暴發性結腸炎
  5. 中毒性結腸
  6. 晚期妊娠


七、腸鏡檢查前後的注意事項：


腸鏡檢查前注意事項：

  1. 提前進行腔鏡前血清檢查的化驗
  2. 告知患者及家屬腸鏡檢查的相關知識，積極配合醫生。
  3. 高血壓患者可正常服藥；如果長期服用阿司匹林、波立維、三七等抗凝成分的中西藥提前告知醫生。
  4. 就診時攜帶以往的就診病歷和檢查結果以便醫生參考
  5. 腸道準備工作：

（1）檢查前2-3天開始少渣飲食，檢查前一晚無渣飲食，檢查當天禁食，但可喝糖水。

（2）腸鏡下午檢查：檢查當日早晨7：00開始服用和爽3袋，當日11:00服用西甲硅油15ml。

（3）腸鏡上午檢查：前晚晚餐後2小時服用和爽3袋，3小時服完，檢查當日6:00服用西甲硅油15ml。

（4）對於習慣性便祕的患者需在檢查前1～2天服用緩瀉劑或用開塞露塞肛，排出乾結的宿便。

（5）服用洗腸劑注意速度，避免過急導致嘔吐，並認真觀察糞便直至成清水樣。



6.檢查要求有直系親屬陪同，以便交代病情及簽字


腸鏡檢查後注意事項

  1. 檢查後會出現輕微腹脹、疼痛等不適，與術中注氣有關，一般會自行緩解
  2. 患者無腹痛、腹脹可安排進食，適當休息。
  3. 無痛腸鏡的患者術後不能駕車，不能從事高空、精細作業以防意外。
  4. 進行活檢的患者三天內勿劇烈活動。


八、無痛胃腸鏡檢查注意事項：


  1. 胃痛胃腸鏡術前要求同普通胃腸鏡檢查。
  2. 預約前需患者本人進行麻醉評估，可去止痛門診評估麻醉禁忌症，無禁忌症簽字蓋章後正常預約。
  3. 無痛胃腸鏡禁忌症包括（1）患有嚴重的呼吸系統疾病（2）張口障礙、頸項或下頜活動受限、矮胖體型者會影響呼吸道開放，不宜做無痛內鏡（3）非呼吸系統疾病如心力衰竭、心肌梗死、中風、昏迷、肝腎衰竭、嚴重貧血以及高齡衰弱等一般情況太差者（4）消化系統疾病如食管、賁門、幽門和腸梗阻、嘔吐嘔血者存在瀦留液反流誤吸者，應先進行胃腸減壓（插胃管）等預處理，排出瀦留液後進行無痛內鏡（5）孕婦不宜做無痛內鏡
  4. 麻醉方式採用的是靜脈麻醉，術後24小時不可開車，不可進行登高等運動，不能簽署重要文件等。


九、胃腸鏡檢查預約方式：


  1. 門診患者在門診開具申請並繳費後直接到診斷樓二樓與門診樓連接處腔鏡預約窗口進行預約
  2. 住院患者直接到預約窗口辦理。


十、胃腸鏡怎麼取結果


胃腸鏡報告三個工作日，病理結果五個工作日後會發往病案室，保存在患者病例中，門診患者可前往並複印結果，或醫生直接從HIS系統中調閱；住院患者直接發往病房，無需家屬或患者個人領取。要後續內鏡治療患者，可以預約內鏡門診號，由內鏡醫生進行後續診療評估及預約。
