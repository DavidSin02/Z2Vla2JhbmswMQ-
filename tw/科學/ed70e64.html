收藏！國家重點保護海洋生物名錄

2020-10-29
| 科學

李健的《深海之尋》，是世界海洋日的公益宣傳曲。今天，我們聊聊海洋生物，特別是那些國家重點保護對象。

《我國《刑法》第三百四十一條規定：非法獵捕、殺害國家重點保護的珍貴、瀕危野生動物的，或者非法收購、運輸、出售國家重點保護的珍貴、瀕危野生動物及其製品的，處五年以下有期徒刑或者拘役，並處罰金；情節嚴重的，處五年以上十年以下有期徒刑，並處罰金；情節特別嚴重的，處十年以上有期徒刑，並處罰金或者沒收財產。

法律是保護海洋生物的一種手段，從根本上保護生物的多樣性，更需要每一個人的自覺。

於是，我們參照《國家重點保護野生動物名錄》與《瀕危野生動植物種國際貿易公約》，整理出以下所有受到國家法律保護的海洋生物。希望藉此，能提高大家對涉保生物的認知，並提升我們的保護意識。

※注：

1.以下文字內容將以「序號+中文名+拉丁名（命名者及時間）+註釋」的形式陳列；

2.熊貓是國家一級保護，穿山甲是二級，此處一級保護的概念，相當於水中大熊貓；

3.圖片來源見水印，部分圖片來自網絡，請多指教。

《國家重點保護野生動物名錄》


01.儒艮 / Dugong dugon (P. L. S. Müller, 1776)

國家一級保護。儒艮（gèn），俗稱海牛。圖片中吸附在海牛背上的魚叫鮣(yìn)魚。


02.中華白海豚 / Sousa chinensis(Osbeck, 1765)

國家一級保護。剛出生的白海豚體呈深灰色，年青時呈灰色，到了成年則呈淡粉色。


03.中華鱘 / Acipenser sinensis(Gray, 1835)

國家一級保護。也是活化石。


04.紅珊瑚 / Corallium spp.

國家一級保護。


05.庫氏硨磲 / Tridacna cookiana

國家一級保護。下圖為硨磲的一種，非庫氏硨磲。



06.鸚鵡螺 / Nautilus pompilius

國家一級保護。


07.多鰓孔舌形蟲 / Glossobalanus polybranchioporus(Tchiang & Liang, 1965)

國家一級保護。圖A。

08.黃島長吻蟲 / Saccoglossus hwangtauensis(Si & Kwang-Chung, 1935)

國家一級保護。圖B。


09.鰭足目(所有種) / Pinnipedia

國家二級保護。鰭足目包含海獅科、海豹科及海象科，我國僅有前2科。圖為海狗，屬海獅科。


10其它鯨類 / Cetacea

國家二級保護。除一級保護外的鯨類都屬於二級保護，鯨主要分為鬚鯨和齒鯨，圖中的虎鯨就是齒鯨的一種。


11.蠵龜 / Caretta caretta (Linnaeus, 1758)

國家二級保護。蠵(xī)龜，又稱紅海龜，是現存最古老的爬行動物。


12.綠海龜 / Chelonia mydas (Linnaeus, 1758)

國家二級保護。綠海龜體色呈棕褐色，之所以叫綠海龜，是因為它是以海草、海藻等為主食的偏草食性的雜食生物，吃多了這些導致脂肪都綠了。


13.玳瑁 / Eretmochelys imbricata (Linnaeus, 1766)

國家二級保護。玳瑁是海洋中較大而凶猛的肉食性動物，經常出沒於珊瑚礁中，主要捕食魚類、蝦、蟹和軟體動物，偶爾也吃海藻。


14.太平洋麗龜 / Lepidochelys olivacea (Eschscholtz, 1829)

國家二級保護。太平洋麗龜是海生龜類中最小的一種。


15.稜皮龜 / Dermochelys coriacea (Vandelli, 1761)

國家二級保護。稜皮龜，是龜鱉目中體型最大者，四肢槳狀，無爪，前肢特別發達，游泳迅速且能力極強。


16.黃脣魚/ Bahaba taipingensis (Herre, 1932)

國家二級保護。經常上新聞頭版的「黃金魚」，新聞工作者總是一味地宣傳黃脣魚貴如黃金，卻無視對它的保護。


17.松江鱸魚 / Trachidermus fasciatus Heckel (1840)

國家二級保護。


18.花鰻鱺 / Anguilla marmorata(Quoy et Gaimard, 1824)

國家二級保護。目前花鰻鱺的野生資源量依然稀少，養殖個體均來源於捕撈的野生鰻苗。


19.克氏海馬 / Hippocampus kelloggi(Jordan & Snyder, 1901)

國家二級保護。


20.文昌魚 / Branchiostoma belcheri

國家二級保護。文昌魚，又稱蛞蝓魚，歷史上以廈門同安劉五店產量最多。


21.虎斑寶貝 / Cypraea tigris(Linnaeus,1758)

國家二級保護。學名黑星寶螺。


22.冠螺 / Cassis cornuta(Linnaeus,1758)

國家二級保護。學名唐冠螺，因其內脣擴張如帽緣，體形如唐朝僧人的帽子而得名。


23.大珠母貝 /Pinctada maxima (Jameson,1901)

國家二級保護。別名白螺珍珠貝，白碟貝，是我國最大的珍珠貝。


24.佛耳麗蚌 / Lamprotula mansuyi (Dautzenberg & Fischer, 1908)

國家二級保護。


《瀕危野生動植物種國際貿易公約》

｜根據《林業部關於核准部分瀕危野生動物為國家重點保護野生動物的通知》，列入《瀕危野生動植物種國際貿易公約》附錄一和附錄二的非原產我國的所有野生動物，分別核准為國家一級和國家二級保護野生動物。

01.露脊鯨屬(所有種) / Eubalaena Spp.

附錄一。圖中為黑露脊鯨，是唯一分佈於我國的露脊鯨品種。


02.小鰮鯨 / Balaenoptera acutorostrata (Lacépède, 1804)

附錄一。小鰮(wēn)鯨，又稱小鬚鯨、尖嘴鯨，為小型鬚鯨的一種。


03.塞鯨 / Balaenoptera Borealis

附錄一。別稱鰮(wēn)鯨，其頭部背面中央有1條隆起的縱脊，是分辨塞鯨的主要特徵。


04.布氏鯨 / Balaenoptera edeni (Anderson, 1879)

附錄一。別稱鯷(tí)鯨，與塞鯨極為相似，見頭部背面有3條縱脊，即可判定為鯷鯨。


05.藍鯨 / Balaenoptera musculus (Linnaeus, 1758)

附錄一。藍鯨不但是最大的鯨類，也是現存最大的動物，是迄今為止最大的哺乳動物。


06.角島鯨 / Balaenoptera omurai (Wada, Oishi & Yamada, 2003)

附錄一。大村鯨是其別稱，由於其模式標本是擱淺在日本山口縣下關市的角島，故俗名角島鯨。


07.長鬚鯨 / Balaenoptera physalus (Linnaeus, 1758)

附錄一。長鬚鯨是全球第二大的鯨，也是現存第二大的動物，僅次於藍鯨。


08.座頭鯨 / Megaptera novaeangliae (Borowski, 1781)

附錄一。座頭鯨的座頭二字源自於日文，意為琵琶，指鯨背部的形狀。


09.白海豚屬(所有種) / Sousa Spp.

附錄一。同《國家重點保護野生動物名錄》的02.。

10.灰鯨 / Eschrichtius robustus (Lilljeborg, 1861)

附錄一。灰鯨的沿後1/3背脊上有8-15個低的峰狀突，第一個峰狀突最大，越靠近尾部越小。


11.抹香鯨 / Physeter macrocephalus (Linnaeus, 1758)

附錄一。抹香鯨是體型最大的齒鯨，頭部可占身體的1/3；潛水能力極強，是潛水最深，潛水時間最長的哺乳動物。


12.儒艮 / Dugong dugon (P. L. S. Müller, 1776)

附錄一。同《國家重點保護野生動物名錄》的01.。

13.海龜科(所有種) / Cheloniidae Spp.

附錄一。同《國家重點保護野生動物名錄》的11/12/13/14.。

14.稜皮龜 / Dermochelys coriacea (Vandelli, 1761)

附錄一。同《國家重點保護野生動物名錄》的15.。

15.鋸鰩科(所有種) / Pristidae Spp.

附錄一。吻長而呈鋸齒狀故名。


16.鯨目(所有種) / CETACEA Spp.

附錄二。指除被列入附錄一外的其他鯨目。

17.鐮狀真鯊 / Carcharhinus falciformis

附錄二。


18.長鰭真鯊 / Carcharhinus longimanus

附錄二。


19.路氏雙髻鯊 / Sphyrna lewini

附錄二。


20.無溝雙髻鯊 / Sphyrna mokarran

附錄二。


21.錘頭雙髻鯊 / Sphyrna zygaena

附錄二。


22.前口蝠鱝屬(所有種) / Manta spp.

附錄二。圖中為雙吻前口蝠鱝 / Manta birostris (Walbaum，1792)。


23.蝠鱝屬(所有種) / Mobula spp.

附錄二。圖中為日本蝠魟 / Mobula japanica。


24.鯨鯊 / Rhincodon typus (Smith, 1828)

附錄二。鯨鯊是最大的鯊，是魚類中最大者，雖然擁有巨大的身軀，但卻是性格極為溫順的鯊魚。鯨鯊是濾食動物，以浮游生物、巨大的藻類、磷蝦等為食。


25.波紋脣魚 / Cheilinus undulatus (Rüppell, 1835)

附錄二。別稱蘇眉，為鱸形目隆頭魚科脣魚屬，一種熱帶地區近海海域生存的著名巨型魚類。


26.海馬屬(所有種) / Hippocampus Spp.

附錄二。同《國家重點保護野生動物名錄》的19.。

27.硨磲科(所有種) / Tridacnidae Spp.
