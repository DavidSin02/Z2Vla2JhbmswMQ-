Sklearn迴歸樹擬合正弦曲線

2020-11-14
| 科學


觀察決策樹是如何擬合一條曲線的，噪聲對曲線擬合的影響

1.導入需要的庫

#導入所需要的庫import numpy as npfrom sklearn.tree import DecisionTreeRegressorimport matplotlib.pyplot as plt%pylab inline

2.生成一個含有噪聲的正弦曲線

步驟生成隨機數0-5的x，將X值放到Sin(X)上生成正弦曲線，對正弦曲線添加噪聲

rng = np.random.RandomState(1)#生成隨機數種子#rand(80,1)生成0到1的隨機數80個#5 * rng.rand(80,1)生成0到1的隨機數80個乘以5（0-5）#sort 進行排序x = np.sort(5 * rng.rand(80,1),axis = 0) #生成0-5的隨機數X的取值y = np.sin(x).ravel()#生成正弦曲線#所有的行和列每5個取一個 5代表步長 y[::5] 行 列 步長#rng.rand(16) 生成16個隨機數 生成0到1之間得數據y[::5] +=3 *(0.5-rng.rand(16))#在正弦曲線上加噪聲

看看現在的圖片

plt.figure()plt.scatter(x,y,s=20,edgecolor="black",c= "darkorange",label = "date")

#建立兩個模型 看不同的模型深度對訓練集的擬合程度 最大深度設的小會限制模型隊訓練集的擬合regr_1 = DecisionTreeRegressor(max_depth=2)#最大深度為2regr_2 = DecisionTreeRegressor(max_depth=5)#最大深度為5regr_1.fit(x,y)regr_2.fit(x,y)
#導入測試集X_test = np.arange(0.0,5.0,0.01)[:,np.newaxis]#predict 導入測試集之後得到每個測試樣本點的迴歸或分類的結果y_1 = regr_1.predict(X_test)y_2 = regr_2.predict(X_test)
plt.figure()#畫散點圖 scatter 邊框的顏色 edgecolors 點的顏色 c 標籤labelplt.scatter(x,y, s=20,edgecolors="black",c="darkorange",label="data")#折線圖 plot 顏色 color plt.plot(X_test,y_1,color="cornflowerblue",label="max_depth=2",linewidth =2)plt.plot(X_test,y_2,color="yellowgreen",label="max_depth=5",linewidth =2)plt.xlabel=("data")#橫座標plt.ylabel=("target")#縱座標plt.title("Decision Tree Regression")#標題plt.legend()#顯示圖例plt.show()

不同剪枝深度對模型的擬合效果

限制樹的深度可以有效防止數據過擬合，此例中當樹的最大深度為5時，決策樹學習較好，包括噪聲，使模型偏離實際的正弦曲線，出現過擬合
