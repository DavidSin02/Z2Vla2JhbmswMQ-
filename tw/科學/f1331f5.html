先讓你看看有多“野”



/ 老董果 /

乍眼一看以為是酸多依，原來是“老董果”（不知道字對不對），富民人都喜歡拿來泡酒。查詢度娘，好像學名叫“刺玫果”。


/ 酸多依 /

這才是酸多依，是不是跟上面的很像呢？在富民，有人喜歡拿來泡酒。


說到酸多依，小編家裡喜歡把新鮮酸多依切絲切碎後，涼拌火燒的新鮮紅辣椒，那味道...啊呀呀...太！下！飯！啦！或者把酸多依用鹽水泡起來，把酸水當酸醋食用，“多依牌”天然酸醋，你值得擁有！


/ 卡卡果 /

這個又不知道字對不對了，對此比較陌生，據說也是泡酒搭檔。



/ 鹿心草 /

別名：鹿仙草。據說是一味藥材。可以煮雞燉湯，也可以泡酒。


/ 野壩子 /

嘿嘿，這個就再熟悉不過了，富民人也叫苦裡壩，居家必備涼茶飲品。



/ 八月瓜 /

小編表示對這個極其陌生，據說是一種野生水果，有誰吃過？


/ 野生葛根 /

比起種植的葛根，野生葛根雖然口感略苦，容易嚼渣，但是葛根粉含量高，清涼效果也更好。


/ 野生山藥 /

野生山藥也沒有吃過，不知道跟富民本地種植的山藥有什麼區別。賣主說非常非常難挖！


/ 老人頭 /（野生菌）

這個季節居然還會有野生菌？據賣主介紹說，也叫“山腦包”。好吃嗎？



/ 雞嗉子 /

雞嗉子是一種野生的果子，熟透了的雞嗉子果吃起來很香甜，老人說，把果核一起吞了可以消食？跟雞啄食沙粒石子同理？問了度娘：其有清熱解毒，利膽行水，消積殺蟲的藥用價值。富民人也是會用來泡酒。




唉！這一堆堆的，實在是不認識了......居然很多東西都不認識！

其實呢，小編認為，“野味”這樣的東西吧，不論是具有養生價值的藥材，還是山珍野味，大部分購買者都是經濟狀況穩定，尋求品質天然健康、養生意識高的人才懂享受的。像小編這樣的人，最多也就買點蛤蟆葉、蒲公英、薺菜（還不一定買到野生的呢）這一類了野菜。所以，不知道也是可以理解的。


上河院菜市場，一位大媽賣的野菜，有蒲公英、小鵝菜、薺菜，確定都是野生的。

以上就是今天拍到的“野”貨，當然了，這只是小編髮現的一部分而已。好了，接下來看看富民最近街上都還有些啥？



汁水充盈的大梨，糖心的紅蘋果......

號外！號外！富民山藥上線了......


lia了一圈，就拍到這些，重點的基本都拍到了，至於很多細化的，等著感興趣的你來慢慢發現！

看到這裡，也別再問小編這個多少錢、那個怎麼賣了，一句話：絕對新鮮，物有所值。自！己！來！逛！

為了拍照，被有的攤販懟“拍了照片買一點嘛！”拍了這麼多，逢拍必買的話，小編我買得完嗎？主要是沒錢！沒錢！沒錢！

也有的攤販，會很主動地拿開遮擋物讓小編拍；拍著這邊的，那邊的攤販看到還會主動說“來拍拍我的桃啊，我的桃很好的！”

總之，不管是什麼樣的攤販，都是為了生活，都祝大家生意興隆！也希望小編的這篇文章，會讓更多的昆明周邊乃至朋友來富民趕鄉街子！
