關於​Flexbox你需要了解的5種技術

2020-11-14
| 科學

Flexbox是一個用於設計用戶界面的css標準。使用各種flexbox屬性，我們可以構建頁面，從小積木中構建頁面，然後輕鬆地定位並調整我們想要的任何方式。這種方式提供的網站和應用程序響應性好，並且適合所有屏幕大小。

在本文中，我們將討論五種解決常見css佈局問題的方法。我們還包含了一些實際例子來展示這些技術應用於實際生活場景中。

1.。創建等高列

起初這似乎並不是一項艱鉅的任務，但是製作相同高度的列可能會非常煩人。簡單設置min-height不會工作，因為一旦列中內容的數量開始不同，其中一些將會增長，其他一些將保持較短。

使用flexbox來修復這個問題並不容易，因為創建這樣的列在默認情況下具有相等的高度。我們所要做的就是初始化flex模型，然後確保彎曲方向和對齊-項目屬性具有默認值。

css：.container { /* Initialize the flex model */display: flex; /* These are the default values but you can set them anyway */彎曲方向: row; /* Items inside the container will be positioned horizontally */對齊-項目: stretch; /* Items inside the container will take up it's entire height */}
HTML：<div class="container"><!-- Equal height columns --><div>...</div><div>...</div><div>...</div></div>

2.。重排元素

不久前，如果我們必須動態地更改某些元素的順序，我們可能會嘗試一些css代碼，退出時退出，然後用javascript進行操作。然而，使用flexbox，這個任務歸結為應用一個單一的css屬性。

它叫order就像它的名字一樣，它的一切都儘可能簡單明瞭。它允許我們交換任意數量的flex項目，並更改它們出現在屏幕上的順序。它唯一的參數是一個整數，確定那個元素的位置-低數意味著更大的優先級。

css：.container{ display: flex;}/* Reverse the order of elements */.blue{ order: 3;}.red{ order: 2;}.green{ order: 1;}
HTML：<div class="container"><!-- These items will appear in reverse order --><div class="blue">...</div><div class="red">...</div><div class="green">...</div></div>

3.。水平垂直定心

垂直中心在css是其中一個問題，讓我們問自己：如此瑣碎的事情怎麼會這麼複雜呢？什麼？而且真的是。如果你是谷歌垂直居中css，大量不同的技術將會出現，其中大多數都會涉及表格或轉換-那些不是用來製作佈局的東西。

Flexbox提供了一個更容易解決這個問題的方法。每個彈性佈局都有兩個方向(x軸和y軸)以及兩個不同的屬性來進行它們的對準。通過中心位置，我們可以將任何元素都定位在其父容器中間。

css：.container{ display: flex; /* Center according to the main axis */justify-content: center; /* Center according to the secondary axis */對齊-項目: center;}
HTML：<div class="container"><!-- Any element placed here will be centered both horizonally and vertically --><div>...</div></div>

4.。創建全響應網格

大多數開發人員在創建響應網格時依賴於css框架。自舉是最流行的一個，但是有數百庫可以幫助您完成這項任務。他們通常工作得很好，而且有很多選擇，但往往相當重。如果你是一個diy的人或者不想為網格實現一個完整的框架，那麼你就已經覆蓋了！

在網格中的一行只是一個容器。display:flex.。它內部的水平列可以是任意數量的元素，它的大小是通過撓曲.。flex模型適應了視窗大小，所以這個安裝在所有設備上都應該看起來很好。但是，如果我們決定屏幕上沒有足夠的空間，我們可以輕鬆地將佈局轉換成垂直的空間，並且使用媒體查詢。

css：.container{ display: flex;}/* Classes for each column size. */.col-1{ 撓曲: 1;}.col-2{ 撓曲: 2;}@media (max-width: 800px){ .container{ /* Turn the horizontal layout into a vertical one. */彎曲方向: column; }}
HTML：<div class="container"><!-- This column will be 25% wide --><div class="col-1">...</div><!-- This column will be 50% wide --><div class="col-2">...</div><!-- This column will be 25% wide --><div class="col-1">...</div></div>

5.。創建完美的粘性腳註

Flexbox也有一個容易解決這個問題的辦法。當構建包含粘性頁腳的頁面時，通過在flex元素中完成整件事情，我們可以確保我們的頁腳始終停留在頁面底部。

應用display: flex對於body標記，我們可以使用flex模式屬性構建整個頁面佈局。一旦完成了這個網站，網站的主要內容可以是一個flex項目和另一個腳註，這使得操作它們的定位非常容易，並將它們精確地放置在我們想要的地方。

css：HTML{ height: 100%;}body{ display: flex; 彎曲方向: column; height: 100%;}.main{ /* The main section will take up all the available free space on the page that is not taken up by the footer. */撓曲: 1 0 auto;}footer{ /* The footer will take up as much vertical space as it needs and not a pixel more. */撓曲: 0 0 auto;}
HTML：<body><!-- All the page content goes here --><div class="main">...</div><!-- Our sticky foooter --><footer>...</footer></body>
