陰性藥實戰：掌握這4大類方

2020-10-29
| 科學

黃連類方

□黃連證

1. 煩躁不安，或心悸，或失眠，或神志不清，身體自覺發熱。

2. 心下痞、胃痛、腹痛、下痢、噁心嘔吐等消化道症狀。

3. 舌質紅或暗紅，質堅老，舌苔黃膩或厚或薄，舌面較幹。(黃連舌)

1. 煩躁不安是黃連證的重要指徵。此外患者尚伴有心悸心慌、睡眠淺而易驚，甚或顛倒轉側，不得安眠。嚴重得精神症狀可以表現為精神錯亂。

2. 凡心下痞者，必黃連與黃芩同用。所謂心下痞是指胃脘部有一種脹悶的感覺，但按之並無肌衛現象，有的患者可以有輕度脹痛，按之更甚。

3. 腹診：用手作心窩部的觸診，患者往往會感覺不適或脹滿氣滯感，或疼痛感。

4. 黃連舌是黃連證的特異性指徵。

□黃連解毒湯

1. 煩躁，不安感，焦慮，或抑鬱。

2. 面紅赤，上火，脣暗紅，出血傾向。

3. 黃連舌。

4. 心下痞，不適感，按之隱痛。

1. 火熱症候：不安感、焦躁感、失眠、精神異常、面紅充血、口乾舌燥、頭痛頭重等。

2. 黃連解毒證的患者大多是體格強健、面色紅或紫赤的男性，口脣暗紅，舌質暗紅堅老、乾燥，舌苔黃而幹膩。

3. 問診可知有煩躁、不安感、失眠、焦慮或抑鬱及頭痛眩暈、記憶力減退、注意力不集中等精神症狀。

4. 切診可以發現心下有痞悶不適感、抵抗感、壓痛感。體檢可見血壓偏高、心率偏快，紅細胞計數及血紅淡白偏高。

5. 類方：溫清飲：合四物湯，是用黃連解毒湯兼見出血傾向、女性月經不調者。荊芥連翹湯：溫清飲加荊芥、連翹、薄荷、柴胡、桔梗、白芷、甘草，用於頭面部炎症。

□陽熱體質

外觀特徵：體格較強健，面色潮紅或紅黑，有油光，目睛充血多眵，口脣暗紅或紫紅，舌質紅或暗紅、質堅斂蒼老，舌苔薄黃或黃膩，腹部肌肉較緊張，按之有力或有不適感。

好發症狀：平時喜涼惡熱，喜涼飲，易煩躁、焦慮，好動，易失眠多夢，皮膚常有瘡癤，上腹部常痞悶不適，口乾口苦，常有口舌潰瘍，咽痛，小便黃短。

□黃連湯

1. 惡風，發熱，汗出。

2. 煩躁，心動悸。

3. 心下痞或嘔吐，或腹痛，或洩瀉。

4. 舌質紅或暗紅，舌苔膩較厚，或前半部苔薄，後半部有白厚苔。

□小陷胸湯

1. 上腹部、胸脅部痞痛、拒按。

2. 舌質紅，舌苔黃膩。

3. 或便祕，或噁心，或咳嗽氣急，痰黃黏膩。

黃連證伴見心下痞痛，按之更甚，便祕，或伴見胸脅苦滿、咳嗽喘息、痰黏而黃。

□三黃瀉心湯

1. 煩躁不安，面部潮紅，或為大黃體質。

2. 心下痞，便祕。

3. 舌質暗紅堅老，舌苔黃膩或乾燥(黃連舌)。

4. 脈實有力、數、滑。

5. 吐血、衂血或出血傾向。

以紅、腫、熱、痛為表現的感染性化膿性癤腫，多見本方證。

□黃連阿膠湯

1. 心中煩、不得眠。

2. 出血傾向或諸血證。

3. 面色蒼白、精神萎靡、口燥咽乾、手足心熱、耳鳴頭昏、小便短黃、口舌糜爛。

4. 心下痞、腹痛。

5. 舌質紅或深紅、苔黃或花剝、起裂，脈細數。

1. 本方適用於諸出血性疾患伴見心中煩、不得眠者。

2. 本方治療失眠，見煩躁不安同時也有精神萎靡、脈細數、舌紅少苔，且有入夜煩躁而白天稍安的特點。

3. 凡頭暈而眩，耳鳴心煩，口乾脣紅，脛酸懈怠，忽睡忽醒；或徹夜不眠，小便短赤，舌質紅，苔少，脈細數者。

4. 黃連阿膠湯證的舌象有明顯的特徵：舌質紅或深紅；苔或薄黃，或舌苔花剝、舌面起裂紋；口腔黏膜常破碎。

□香連丸

1. 腹痛、腹脹，裡急後重，痢下黏臭或膿血。

2. 舌苔黃膩，舌質暗紅。

1. 木香臨床用於腹痛腹脹。

2. 香連丸方證中裡急後重，痢下黏臭或膿血，舌苔黃膩，舌質暗紅提示黃連證的存在。

3. 臨床尚可以見到患者身熱有汗，煩躁等症。

4. 若腹痛劇烈，裡急後重舌苔呈焦黃膩，可以加酒制大黃。

□左金丸

1. 脘腹痛，脅脹痛，口苦，嘈雜，嘔吐酸苦水。

2. 舌質紅舌苔黃。

1. 吳茱萸用於止痛止嘔。

2. 左金丸證的特徵是腹痛、嘔吐酸苦水。

3. 比較：臨床也見一些慢性胃痛，常見疼痛綿綿，且常吐酸水，但細問後知疼痛並不劇烈，常在空腹時發作，其酸水不苦、量多、清稀，患者自覺胃脘冷，口內淡而無味，舌質多淡紅，舌苔白滑，此類腹痛多為小建中湯。

石膏類方

□石膏證

1. 煩渴喜飲。

2. 惡熱多汗。

3. 舌面乾燥。

4. 脈洪大、浮滑。

1. 以口渴、舌幹、身熱多汗、脈象洪大有力為主要症狀。

2. 舌面可以有舌苔，但舌面必定乾燥。

3. 大黃證是以便祕、腹痛腹脹、拒按、舌紅苔焦黃厚膩為特徵，表示胃腸道內有有形的宿垢燥屎充滿其中，謂之實熱證；石膏證以強烈渴感、身熱、多汗、脈洪大為特徵，表示體內有無形燥熱之氣瀰漫燔灼，故謂之燥熱證。

□白虎湯

1. 石膏證伴高熱者。

2. 石膏證伴煩躁、神志不清者。

3. 石膏證伴強烈渴感者。

1. 鑑別白虎湯證要特別注意舌象，舌苔要見乾燥無津；脈象要滑數有力，脈形要洪大。

2. 白虎湯治眼病經驗：眼睛局部紅腫較甚，刺激症狀較明顯。舌紅少津，或舌赤苔黃面燥。脈滑數、洪數或洪大有力。身體壯實，面色紅潤，鼻幹灼熱，口脣乾燥，煩渴，喜冷飲。

□白虎加人蔘湯

1. 屢經發汗，精神萎靡，口渴感明顯的白虎湯證。

2. 無大熱，無大汗，但以渴飲為主訴的慢性病。

3. 有白虎湯證而脈無力，全身狀態差者。

張仲景使用人蔘的指徵是大汗出後、口燥渴。此外患者當有心慌、胸悶、短氣、食慾不振、心下痞硬、消瘦等症。

□白虎加桂枝湯

1. 發熱，身無寒但熱，口渴。

2. 骨節煩疼，惡風，汗出不徹。

3. 舌質暗紅。

□竹葉石膏湯

1. 身熱，多汗，口渴，或咳嗽，或乾嘔。

2. 精神萎靡，消瘦憔悴。

3. 舌紅少苔，舌面乾燥，脈虛數。

本方是一張發熱性疾患恢復期的調理方。與白虎加人蔘湯相比，本方津液不足的程度比較嚴重，故表現為形體羸瘦，舌紅少苔而口乾舌燥，乾咳等。

□消風散

1. 皮膚搔癢，或發疹塊，搔之滲液，久發不愈。

2. 發熱感，煩躁，口渴。

3. 浮腫傾向，或小便不利。

大黃類方

□大黃證

1. 腹滿，腹痛，拒按或腹壁有抵抗感，便祕。

2. 精神不安，煩躁，易興奮，身熱有汗。

3. 舌質紅而堅老，苔焦黃、乾燥。(大黃舌)

1. 大黃證屬於裡實證與裡熱證。

2. 裡實是指以便祕、腹痛、腹脹、拒按抵抗感為臨床表現的腹內充實狀態。

3. 裡熱是指以面紅、身熱、煩躁、脣暗紅、舌紅苔黃、痰液黏稠、口乾燥為表現的代謝亢進狀態。

4. 大黃舌：大黃證中最具特徵的是舌象。

5. 慢性疾病患者中典型大黃舌可能不多，但舌苔一般多比較厚，且色黃，舌質也偏紅且堅老。

□大黃體質

外觀特徵：體格壯實，肌肉堅緊，面色紅有油光，或面膩滯，脣厚暗紅。舌苔厚而乾燥。

好發症狀：平素畏熱喜涼，食慾旺盛，易發頭暈頭昏，易便祕，汗少或汗出不暢，胸悶，口乾苦，痰液唾液黏稠，血脂、血壓偏高，腹部有壓痛感或抵抗感。

體質傾向：多熱、多火。

1. 多熱：主要指這種體質出現的症狀以熱性症狀為多，如畏熱喜涼、食慾旺盛、口乾苦、痰液唾液黏稠、便祕等。

2. 多火：是指大黃體質頭面部症狀比較多，如易發頭暈頭昏、或頭痛、甚或突然中風不語。

□ 大承氣湯

1. 劇烈腹痛、腹脹，堅滿拒按，便祕或痰液膿血便。

2. 潮熱或發熱，身熱汗出，脣舌焦乾。

3. 煩躁、譫語、神志失常。

4. 脈實有力，苔乾焦黃起紅刺。

1. 大承氣湯其證不出大滿大實大熱，其脈不外沈、實、滑。

2. 大滿指腹滿腹脹；大熱指身熱甚，揚手擲足，惡熱喜冷；大實則指不大便，按之臍周硬如卵石。

3. 小承氣湯：用於腹脹滿的大黃證。

4. 調胃承氣湯：適用於有明顯舌苔乾燥、黃厚、便祕的大黃證及有明顯身熱有汗、大便乾燥如慄的大黃證。除問診外，乾燥的舌苔及腹診見腹部硬滿、特別是臍周捫及糞塊也是診斷要點。

□桃核承氣湯

1. 大黃證。

2. 少腹部拘急疼痛，按之更甚。

3. 出血紫黑，易凝固結塊。

4. 精神不安，如狂。

5. 舌質暗紅或紫，舌面乾燥，脣暗紅，面紅。

1. 張仲景用於治療「熱結膀胱，其人如狂，血自下」「少腹急結」。

2. 熱結膀胱：膀胱是指少腹，熱結指裡實熱證。

3. 其人如狂：是指神經系統興奮性增高的煩躁不安、譫語，類似精神錯亂的狀態。

4. 少腹急結：為少腹部拘急疼痛，按之更甚。體徵是左下腹部劇烈疼痛，按壓時患者因痛劇而屈曲左下肢，有時按壓時可以觸及深部有壓痛及索狀柔軟抵抗物，有的患者左腹股溝也有顯著壓痛。

□大黃蟅蟲丸

1. 少腹部疼痛或有硬塊，腹滿感，腹脹感。

2. 形體消瘦，面色晦暗，肌膚乾燥如鱗甲，兩目黯黑。

3. 舌質紫暗，或見舌見瘀斑，脈細澀。

□茵陳蒿湯

1. 黃疸色澤鮮明，尿黃短少。

2. 胸悶，煩躁，身熱有汗。

3. 大黃證。

1. 黃疸病屬陽黃專方。

2. 陽黃：屬於熱證、實證的黃疸，黃疸色澤鮮明，並有口乾、身熱有汗、便祕、尿黃、舌質紅、苔黃。

3. 陰黃：屬於寒證、虛證的黃疸，黃疸色澤晦暗，口淡不渴、身寒、便溏、尿清白、舌質淡、苔白。

□防風通聖散

1. 發熱無汗或頭痛頭昏，或目赤腫痛，或皮膚癢疹、瘡瘍腫毒等。

2. 便祕，腹脹，胸隔滿悶。

3. 舌苔厚或幹或膩。

□大黃附子湯

1. 腹痛劇烈、拒按，便祕。

2. 精神萎靡，面色晦滯無華，汗出惡寒，手足冷，脈沈緊弦。

3. 舌苔白膩，舌面比較幹，舌質堅老。

柴胡類方

□柴胡證

1. 胸脅苦滿。

2. 寒熱往來，或休作有時。

1. 胸脅苦滿是柴胡證的必見指徵。胸脅部的脹痛、脹滿、硬滿、觸痛、壓痛；女性乳房脹痛、結塊；或膽囊痛、肋間神經痛都可包含在內。

2. 他覺胸脅苦滿：沿肋骨弓的下端向胸腔內按壓，醫生的指端有抵抗感，患者也訴說局部有脹痛不適的感覺。

3. 寒熱往來：除體溫以外，包括自覺的寒熱交替感；時而畏風發冷，時而面紅煩熱；或上半身發熱，下半身畏冷；或半身熱、半身冷；或心胸煩熱，而四肢冰冷；或覆被則煩躁發熱、似汗非汗，而去被又覺寒冷至骨、肌膚粟起；以及對溫度的自我感覺過敏等。

4. 往來與休作有時：指發病時間上有一定的規律，或有一定的週期，或是交替發作。

5. 柴胡帶：包括胸脅部、肩頸部、頭額部、腰胯極少腹部、腹股溝等，以身體的側面為主。這區域出現的脹痛、痠痛、牽掣感、感覺異常、腫塊、結核等，常可以考慮柴胡證及諸柴胡湯證存在的可能性。

□柴胡體質

外觀特徵：體型中等或偏瘦；面色微暗黃、或青黃色、或青白色，缺乏光澤；皮膚比較乾燥，肌肉比較堅緊；舌質堅老、暗而紫點、舌體不淡胖，舌苔正常或偏幹。脈多弦細。

好發症狀：主訴以自覺症狀為多，對氣溫變化的反應敏感，或時有寒熱感，情緒波動較大，食慾易受情緒的影響；胸脅部時有氣塞滿悶感、或有觸痛；肩頸部常有酸重感、拘攣感；四肢常冷，少腹易脹痛。女性月經週期不齊，經前多見胸悶乳脹、煩躁；經來腹痛、經血暗或有血塊。

體質傾向：多氣滯、多血瘀。

1. 面色暗是重要外觀特徵，暗是氣滯血瘀的外在表現；舌質亦可以見到暗或紫。由於有瘀血，皮膚多見乾燥。

2. 胸脅苦滿與四肢常冷也是重要特徵。

□小柴胡湯

1. 胸脅苦滿，或上腹部痞痛，或膽囊部明顯壓痛。

2. 發熱或低熱持續，呈寒熱往來樣。

3. 心煩喜嘔，或嘔吐、口苦，默默不欲飲食。

4. 脈弦、或弦細、或弦滑、或沈弦。苔黃或黃白相兼，或淡黃，或黃膩。

1. 凡用黃芩必見證為上腹部的痞痛或腹痛。

2. 傷寒論用人蔘的目的有三：一是救治汗吐下後，或失血後的虛脫及脫水的脈微、舌面乾燥；二是治療病後的虛弱少氣；三是用於心下痞硬。

3. 心下痞硬：指上腹部的似脹非脹，按之腹肌薄而緊張度比較高，一般多見體弱者或慢性胃腸病者。

4. 合方。

柴陷湯：合小陷胸湯，治咳瘦痰黏，伴胸脅苦滿及心下壓痛者。

柴樸湯：合半夏厚朴湯，治胸悶脅痛、咽喉、食道異物感、精神不安定、食慾不振、噁心嘔吐、苔白膩者。

柴苓湯：合五苓散，治小柴胡湯證伴見尿量減少、浮腫、口渴者。

柴平煎：合二陳湯，治小柴胡湯證見腹滿、苔白膩者。

□柴胡桂枝湯

1. 發熱惡風，往來寒熱，有汗，關節痠痛。。

2. 胸脅苦滿，或腹痛，食慾不振，心煩喜嘔。

3. 舌質暗紅或暗淡，苔薄白或薄黃膩。

1. 桂枝體質見胸脅苦滿、寒熱往來、嘔吐口苦者；柴胡體質見自汗、鼻塞、腹痛、關節痠痛、肌肉痙攣者；小柴胡湯與桂枝湯證互見者。

2. 舌質多偏暗。

□柴胡桂枝幹薑湯

1. 胸脅滿，或咳嗽，或胸骨痛、觸之更甚。

2. 寒熱往來感，或惡風、盜汗、自汗，頭頸以上多汗。

3. 心煩，胸腹動悸，不眠多夢，耳鳴。

1. 柴胡桂枝幹薑湯是柴胡類方中的安定劑及精神疲勞的恢復劑。

2. 女性最多此方證。例如有客人時或外出時，精神頗飽滿，應接熱情，但過後即感疲勞，精神頓失。

3. 柴胡體質患者，表現精神的波動幅度大，在過度的精神緊張，飲食不調，加上連續的肉體勞動或過多出汗的刺激下，容易形成柴胡桂枝幹薑湯證。

□柴胡加龍骨牡蠣湯

1. 柴胡證。

2. 精神神經症狀，尤其是臍腹動悸、易驚、譫語。

3. 舌質紅，苔厚黃膩。

1. 胸滿：可以看作胸脅苦滿為主的柴胡證。

2. 煩驚：可看作精神不安、不眠、胸脅動悸等精神症狀，也表現夜夢多、易醒、易心慌。

3. 譫語：可看作語言思維障礙，精神錯亂。

4. 一身盡重，不渴轉側：可看作神經肌肉病變。

5. 小便不利：可看作心身疾患的水腫、尿失禁、大便祕結等。

6. 臍腹動悸是張仲景使用龍骨牡蠣的重要指徵。

7. 醫生按其腹部扁平，腹主動脈搏動明顯。

□四逆散

1. 柴胡證；或對疼痛敏感，經常手冷，易緊張，肌肉易痙攣的柴胡體質。

2. 胸脅苦滿、疼痛，腹痛腹脹。

3. 脈弦，舌質堅老而暗，或舌有紫點。

1. 枳實主治胸腹脹滿、便祕、胃下垂、子宮下垂等。

2. 基本條件：胸脅腹疼痛，四肢不溫，柴胡體質；且腹痛頻發而精神尚飽滿，形體不憔悴，且無大黃證、桂枝證、附子證。

□血府逐瘀湯

瘀血表現：

1. 疼痛部位多固定。

2. 出血易凝固，色紫黑。

3. 精神不安，煩躁，甚至發狂。

4. 舌質紫暗，面色晦暗。

1. 柴胡體質出現瘀血的頻率是較高的。特別是那些頑固性的失眠、頭痛、腹痛、發熱等病症，常見血府逐瘀湯證。

2. 柴胡體質患者的慢性病調理中，要注意瘀血證的存在。

3. 本方是四逆散與桃紅四物湯的合方。

4. 臨床對以下情況要注意本方證存在：

一、病程比較長，常規治療不效，久治不愈，而不見衰弱者。

二、病症怪異、主述複雜，多見神經官能症或精神病。

三、患者肌肉堅緊。

四、有瘀血證。

□逍遙散

1. 胸脅苦滿或胸脅痛，腹痛、腹脹，女性月經痛，經前乳脹或頭痛。

2. 寒熱往來感，或月經週期參差不齊。

3. 食慾不振，浮腫。

4. 舌淡紅苔薄白。

1. 本方為四逆散與當歸芍藥散加減方。

2. 對象：貧血貌，面色萎黃或枯黃，或黃暗而月經不調的患者。

3. 類方

丹梔逍遙散：加丹皮、梔子，治療遙遙散證伴面色紅、目赤目糊、頭痛、發熱、心煩、出血、尿黃尿痛者。

□大柴胡湯

1. 發熱或往來寒熱。

2. 胸脅苦滿，上腹部拘急疼痛，局部肌緊張。

3. 便祕，尿黃，或下痢，或嘔吐，或黃疸，或頭痛。

4. 舌苔黃白且乾燥，脈滑數。
