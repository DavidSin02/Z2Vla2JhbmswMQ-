賦值運算符

賦值運算符是以符號“=”表示，它是一個二元運算符（對兩個操作數作處理），其功能是將右方操作數所含的值賦值給左方的操作數。其語法格式如下：

變量 = 表達式;

例如：int a = 100;

該表達式是將100賦值給變量a。左方的操作數必須是一個變量，而右邊的操作數則可以是任何表達式，包括變量（如a、number）、常量(如123、'book')、有效的表達式（如45*12）。例如：

例 使用賦值運算符為變量賦值，實例代碼如下：

int a = 10; //聲明int型變量a

int b = 5; //聲明int型變量b

int c = a+b; //將變量a與b進行運算後的結果賦值給c

遵循賦值運算符的運算規則，可知系統將先計算a+b的值，結果為15。然後將15賦值給變量c。因此“c=15”。

例 在項目中創建類Eval，在主方法中定義變量，使用賦值運算符為變量賦值。

public classEval { //創建類

publicstatic voidmain(String[] args) { //主方法

inta, b, c; //聲明int行變量a,b,c

a = 15; //將15賦值給變量a

c = b = a + 4; //將a與4的和賦值給變量b，然後再賦值給變量c

System.out.println("c值為：" + c); //將變量c的值輸出

System.out.println("b值為：" + b); //將變量b的值輸出

}

}

a+=b //相當於a=a+b

/＝

將運算符左邊的變量除以右邊表達式的值賦給左邊的變量

a/=b //相當於a=a/b

%＝

將運算符左邊的變量用右邊表達式的值求模，並將結果賦給左邊的變量

a%=b //相當於a=a%b

&=

將運算符左邊的變量與右邊表達式的值進行邏輯與運算，並將結果賦給左邊的變量

a&=b //相當於a=a&b

|=

將運算符左邊的變量與右邊表達式的值進行邏輯或運算，並將結果賦給左邊的變量

a|=b //相當於a=a|b

^=

將運算符左邊的變量與右邊表達式的值進行異或運算，並將結果賦給左邊的變量

a^=b //相當於a=a^b

算術運算符

Java中的算術運算符主要有+（加號）、—（減號）、*（乘號）、/（除號）、%（求餘），它們都是二元運算符。Java中算術運算符的功能及使用方式如表1所示。

其中“+”和“-”運算符還可以作為數據的正負符號，例如+5，-7。

下面通過一個小程序來介紹使用算術運算符的方法。

例 在項目中創建類Arith，在主方法中定義變量，使用算術運算符將變量的計算結果輸出。

public classArith { //創建類

publicstatic voidmain(String[] args) { //主方法

floatnumber1 = 45.56f; //聲明float型變量並賦值

intnumber2 = 152; //聲明int型變量並賦值

System.out.println("和為：" + number1 + number2); //將變量相加之和輸出

System.out.println("差為：" + (number2 - number1)); //將變量相減之差輸出

System.out.println("積為：" + number1 * number2); //將變量相乘的積輸出

System.out.println("商為：" + number1 / number2); //將變量相除的商輸出

}

}

在應用算術運算符時，可以分為對整型數據的運算和對浮點型數據的運算，下面進行詳細介紹。

1．整型數據的算術運算

整型數據的“+”、“-”、“*”、的運算與數學中的運算相同，這裡不再介紹，下面介紹整型數據間的“/”和“%”運算。

l 整數的除法（運算符“/”）運算

當整數間進行除法運算時，其結果將捨棄小數部分，並且忽略四捨五入，最終的結果為商的整數部分，所以通過計算機來計算“5/3*3”與“5*3/3”的結果是不同的，前者為3，而後者為5。

例 在項目中創建類Arith1，在主方法中定義變量，使用“/”運算符進行除法運算，並輸出計算結果。

publicclass Arith1 {

publicstaticvoid main(String[] args) { // 主方法

int i1 = 7 / 3; // 在數學計算中其結果約為2.33

int i2 = 11 / 3; // 在數學計算中其結果約為3.67

int i3 = -7 / 2; // 在數學計算中其結果為-3.5

int i4 = 0 / 9;

System.out.println("7/3=" + i1); // 輸出結果為：2

System.out.println("11/3=" + i2); // 輸出結果為：3

System.out.println("-7/2=" + i3); // 輸出結果為：-3

System.out.println("0/9=" + i4); // 輸出結果為：0

}

}

運行結果如圖2所示。

l 整數的求餘（運算符“%”）運算

當整數間進行求餘運算時，其運算結果為整數除法運算後的餘數。

例 在項目中創建類Arith2，在主方法中定義變量，使用“%”運算符進行求餘運算，並輸出計算結果。

publicclass Arith2 {

publicstaticvoid main(String[] args) { // 主方法

int i1=11%4; //結果為：3

int i2=8%4; //結果為：0

int i3=9%10; //結果為：9

int i4=-7%3; //結果為：-1

int i5=0%9; //結果為：0

System.out.println("11%4="+i1);

System.out.println("8%4="+i2);

System.out.println("9%10="+i3);

System.out.println("-7%3="+i4);

System.out.println("0%9="+i5);

}

}

l 將一個整數除以0或除以0求餘的運算

當進行一個整數除以0或除以0求餘運算時，雖然可通過編譯，但在運行時會拋出java.lang.ArithmeticException異常，例如：

publicstaticvoid main(String[] args) { // 主方法

int i1=11/0; //編譯成功，但運行時拋出異常

int i2=9%0; //編譯成功，但運行時拋出異常

System.out.println(i1);

System.out.println(i2);

}
