Java 語言中常用的運算符可分為如下幾種：

Ø 算術運算符

Ø 賦值運算符

Ø 比較運算符

Ø 邏輯運算符

Ø 條件運算符

Java中的條件運算符

條件運算符（ ? : ）也稱為 “三元運算符”。
語法形式：布爾表達式 ？ 表達式1 ：表達式2
variable x = (expression) ? value if true : value if false
運算過程：如果布爾表達式的值為 **true ，則返回 表達式1 的值，否則返回 表達式2
