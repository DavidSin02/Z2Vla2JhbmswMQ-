「兒科專題」不同手術方式治療小兒腹股溝嵌頓斜疝的療效對比研究

2020-11-14

趙 正 席紅衛

山西省兒童醫院普外科

基金項目：山西省衛生健康委員會一般項目(編號：2015062)

嵌頓性腹股溝斜疝是小兒普通外科常見的急腹症之一，典型臨床表現為腹股溝區可復性包塊，不能自行回納，患兒哭鬧、嘔吐，腹股溝或陰囊見質韌包塊，透光試驗陰性，彩超提示為腸管嵌頓。發病12h以內，患兒全身及局部情況良好、無明顯脫水及腹脹者可考慮先行手法復位。少數手法復位失敗或有手術指徵的小兒腹股溝嵌頓性斜疝患兒必須手術治療。經臍單部位腹腔鏡手術治療小兒嵌頓性腹股溝斜疝的優勢國內外已有報道。但是需要昂貴且較大的單部位特製Trocar或經臍縱向延長切口。朱真闖等已報道經臍雙孔腹腔鏡手術在小兒嵌頓疝中的應用。本研究在此基礎上進行了改良，近年來應用經臍單部位腹腔鏡手術治療小兒腹股溝嵌頓性斜疝，取得了滿意的臨床效果，現報道如下。

//

材料與方法

//

一、臨床資料

1.經臍單部位腹腔鏡組：2014年1月至2017年2月山西省兒童醫院經腹腔鏡手術治療小兒腹股溝嵌頓性斜疝58例，均為手法復位失敗或發病時間超過12h的患兒，其中男童54例，女童4例，年齡8個月至6歲；年齡＜1歲20例，年齡＞1歲38例；發現嵌頓至就診時間＜12 h 15例，＞12 h 43例；其中伴腹股溝區皮膚或陰囊皮膚潮紅20例，伴美克爾憩室1例，同時行腹腔鏡輔助下腸切除、腸吻合術。嵌頓內容物為小腸38例，回盲部15例，網膜2例，結腸1例，卵巢2例。嵌頓腸管漿肌層撕裂4例，予腹腔鏡下修補。

2.開腹手術組：2014年1月至2017年2月由山西省兒童醫院經傳統開放手術治療55例腹股溝嵌頓性斜疝患兒，其中男童54例，女童1例，年齡11個月至5歲。年齡＜1歲22例，年齡＞1歲33例；發現嵌頓至就診時間＜12h20例，＞12 h 35例；其中伴腹股溝或陰囊皮膚潮紅10例，伴鞘膜囊感染1例，腸壞死2例，行腸切除、腸吻合術。

3.倫理審查：本研究獲得山西省兒童醫院倫理委員會批准。

二、研究方法

1.經臍單部位腹腔鏡組手術方法：手術在氣管插管全麻下進行，分別取臍緣左、右切口約0.5 cm切開皮膚皮下，刺入氣腹針，注入CO2 氣體，建立氣腹，氣腹壓力9～11mmHg。於臍緣左、右切口分別置入5 mm Trocar，伸入腹腔鏡觀察。探查嵌頓組織(主要為腸管)血運情況和腹腔滲液的顏色、是否混濁及滲液量。腹腔鏡監視下行手法復位，復位後檢查腸管血運狀況，是否腸壁淤血水腫、點狀壞死、漿膜破裂及腸穿孔，女性應增加檢查輸卵管、卵巢的血運。若腸壁淤血水腫、血運欠佳，觀察15～30 min；4例漿肌層撕裂的患兒在腹腔鏡下5－0抗菌薇喬縫線間斷縫合行漿肌層修補；探查患側內環口，於患側腹股溝區刺入2－0帶針慕絲線，腹腔鏡監視下荷包縫合患側內環口，同時探查對側內環口，如未閉者，予同法處理。具體縫合方法、術中情況見圖1。

2.開腹手術組手術方法：採用全麻氣管插管，取平臥位，自嵌頓側外環口上方沿腹股溝管斜形切開皮膚約3.5cm，切開外環，鬆解腹股溝管前壁，將疝囊打開，探查疝內容物，瞭解其血運情況，如血運正常，復位嵌頓內容物，高位結紮疝囊，重建腹股溝管，加強腹股溝管前壁，重建外環，如嵌頓內容物壞死，行壞死腸管切除，腸切除、腸吻合術。

3.觀察指標：比較手術時間、切口長度、術中出血量、住院天數、併發症的發生率。

三、統計學處理

所有數據採用SPSS18.0軟件進行統計學處理，對於手術時間(min)、切口長度(cm)、術中出血量(mL)、術後排氣時間(h)和住院天數(d)等計量資料以均數±標準差(x±ｓ)表示，兩組間比較採用獨立樣本t檢驗；對於併發症等計數資料採用頻數分析，兩組間比較採用卡方檢驗。以P＜0.05為差異有統計學意義。

//

結 果

//

一 、基本情況

經臍單部位腹腔鏡組57例在腹腔鏡監視下均復位成功，並行內環口縫扎術，1例中轉傳統切開復位手術，成功率98.28％；4例因腸管充血腫脹明顯，疝囊頸束環卡壓太緊，復位後造成漿膜破裂，並在腹腔鏡下用5－0抗菌薇蕎線行漿膜層修補術；手術時間平均25.5min；發現對側內環口未閉29例，均同時予對側內環口荷包縫合。本組患兒嵌頓內容物復位後經觀察腸管或卵巢血運均好轉，未見腸壞死或卵巢壞死，伴美克爾憩室1例，同時行腹腔鏡輔助下美克爾憩室切除、腸切除腸吻合術。開腹手術組55例行嵌頓疝復位、疝修補術。術中疝囊質脆撕裂，行內荷包縫合2例，腸壞死2例，行腸切除、腸吻合術。

二、治療效果

經臍單部位腹腔鏡組手術時間平均為(25.5±10.5)min，開腹手術組平均為(37.0±15.2)min，差異有統計學意義(t＝－5.756，P＜0.05)；切口長度經臍單部位腹腔鏡組平均為(0.5±0.2)cm，開腹手術組平均為(3.5±1.5)cm，差異有統計學意義(t＝－18.487，P＜0.05)；術中出血量經臍單部位腹腔鏡組平均為(2.5±2)mL，開腹手術組平均為(16.5±10.5)mL，差異有統計學意義(t＝－11.772，P＜0.05)；術後排氣時間經臍單部位腹腔鏡組平均為(10.8±6.3)h，開腹手術組平均為(20±11.4)h，差異有統計學意義(t＝－6.547，P＜0.05)；住院天數經臍單部位腹腔鏡組平均為(2.5±1.5)d，開腹手術組平均為(4.5±1.5)d，差異有統計學意義(t＝－8.676，P＜0.05)；併發症發生率經臍單部位腹腔鏡組為1.7％，開腹手術組為9.09％，差異有統計學意義(X2＝0.081，P＜0.05)，見表1。

三、隨訪結果

經臍單部位腹腔鏡組術後隨訪2～6個月，無一例復發，58例均無切口感染。開腹手術對照組隨訪3～6個月，無一例復發，4例對側出現腹股溝斜疝，再次行對側疝修補術或腹腔鏡下疝修補術。

//

討 論

//

嵌頓性腹股溝斜疝是小兒普通外科常見急腹症之一，嵌頓疝處理不及時可造成腸壞死及腸穿孔等嚴重併發症，併發症發生率隨嵌頓時間的延長而明顯增加。男性和早產被認為是小兒嵌頓疝的危險因素。其手術治療方法包括傳統開放手術及腹腔鏡手術。

與傳統開放手術相比，經臍單部位腹腔鏡治療小兒嵌頓性斜疝有以下優勢：①直視下復位嵌頓內容物，觀察其血運情況。Lee報道腹腔鏡治療164例小兒嵌頓疝未見腸壞死病例。如腸管血運差或腸穿孔、壞死，可擴大臍部穿刺孔至2～3cm，將病變段腸管提出腹壁外，行腸切除、腸吻合術或腸造瘻術，酌情放置腹腔引流管；文獻報道有2％～33％ 女性嵌頓疝發生卵巢絞窄。對合並卵巢滑動疝的患兒，腹腔鏡下處理優勢明顯，可用絲線向腹腔內牽拉懸吊卵巢上緣腹膜，緊貼其邊緣內荷包縫合內環口，其操作較傳統手術處理滑動疝的情況更微創、簡便；②腹腔鏡的放大作用使開放的鞘狀突、腹壁下血管、輸精管和精索血管清晰可見，術中結紮內環口時，幾乎不可能對上述結構造成損傷；③經臍單部位腹腔鏡可以減少術後切口感染、陰囊血腫、睪丸萎縮等併發症的發生。有研究顯示傳統組術後併發症發生率為4.5％～33％，腹腔鏡組術後併發症的發生率及複發率較傳統手術明顯低。本研究顯示腹腔鏡組併發症發生率遠遠低於傳統組，由此可見腹腔鏡手術在避免術中誤損傷和減少術後併發症方面的優勢還是比較明顯的；④經臍單部位腹腔鏡，因切口及瘢痕隱匿在凹陷的臍部，美容效果更佳。這是傳統手術不可比擬的，且與國內外單部位特製Trocar相比，具有更加靈活及簡易、易於推廣等優勢；⑤同時對合並消化道畸形患兒可在一次手術中同時解決，減少患兒多次手術的痛苦。本組患兒中有1例伴美克爾憩室，同時行美克爾憩室切除、腸切除、腸吻合術；⑥能同時探查對側內環口情況，對於隱匿性疝患兒，可及時行修補術，避免對側疝的二次手術。有研究顯示傳統組術後出現對側隱匿疝的比例為2.86％，而腹腔鏡組術中發現對側隱匿性鞘狀突未閉的比例為49.52％；⑦避免醫源性隱睪的發生。傳統手術時把睪丸拉出陰囊，而術中未能及時確切地把睪丸放回陰囊正常位置，可造成醫源性隱睪；而腹腔鏡下行嵌頓疝修補術對睪丸影響較小，可很好避免醫源性隱睪的發生；⑧對術前與嵌頓性腹股溝斜疝難以明確鑑別的疾病有探查優勢，筆者曾收治1例有腹股溝斜疝病史，腹痛且右側腹股溝區腫物不可還納13h時的患兒，術前病史及彩超均提示右側嵌頓性腹股溝斜疝，但腹腔鏡手術時發現為急性化膿性闌尾炎，膿液沿未閉鞘狀突進入右側腹股溝區及陰囊形成鞘膜腔積膿。經腹腔鏡下及時發現並處理闌尾炎及鞘膜積膿情況，而傳統手術無此探查優勢。

腹腔鏡治療小兒嵌頓性斜疝也存在一定侷限性。本組1例因腹腔鏡手術中疝囊頸束環卡壓太緊，麻醉下亦手法復位困難，遂中轉傳統切開復位手術。傳統切開復位手術如遇到疝囊頸束環卡壓太緊可切開外環，鬆解腹股溝管前壁，將疝囊打開，解除對疝內容物的壓迫。腹腔鏡下疝囊頸束環卡壓太緊時，因同時多合併腹腔內腸管擴張，小嬰兒腹腔內空間有限，導致不能直視下觀察內環口外側嵌頓腸管情況，這種情況可用電刀或剪刀於腹腔內鬆解內環口。此法有損傷腸管風險，故有待進一步商榷和改進。另外，腹腔鏡手術不能探查同側睪丸血運情況，存在睪丸壞死漏診的可能，需結合彩超結果，對疑似有睪丸壞死病例，術中應通過彩超或陰囊切開探查。李忠等報道腹腔鏡輔助下小切口治療小兒腹股溝難復位嵌頓疝，但此方法需經疝囊建立氣腹，且需特殊手術器械，可探查腹腔或對側內環口，但對於腹腔鏡下對側隱匿性斜疝的處理或縫合等進一步操作優勢不明顯。

手術技巧方面，復位後如何實施單手操作內環縫扎方法是關鍵。嵌頓疝所致內環腹膜水腫、甚至嵌頓壞死後腹膜淤血使內環腹膜外重要神經、血管和輸精管難以辨認，我們的經驗認為雖然嵌頓後腹膜出現淤血、瘢痕，腹腔鏡直視下用2－0針尖輕輕挑起後仍可明確在直視下避過精索血管。部分病例輸精管處內環口腹膜瘢痕嚴重，則需仔細辨認，縫合右側內環口時，術者應站於患兒左側，第一針縫合時多自內環口外側進針，距離刺入腹腔進針點2cm以上，當前面一針縫合過後，第二針入針點位於緊貼第一針出針點的下方，同時以左手牽引針尾線將形成皺褶或瘢痕的腹膜拉開利於縫合，仔細辨認避過輸精管，如內環口內下方腹膜褶皺較多，可多次同法進針、牽引，儘量於瘢痕腹膜外側縫合，繞過瘢痕。縫合左側內環口時術者站於患兒頭端，進針仍自內環口外側，左側無法利用針尾線牽引，縫合時注意調整針尖角度，於瘢痕外側避過輸精管。縫合一週後打結時應將持針器鉗夾針尖並與縫針保持鈍角關係，以便於操作。縫線方面，我們主張2－0帶針慕絲線行內環口縫合，防止吸收線吸收後復發風險。

總之，經臍單部位腹腔鏡治療小兒嵌頓疝是一種安全有效的微創手術方法，對於大部分嵌頓性腹股溝斜疝患兒有優勢，是值得推廣和改進的好方法。
