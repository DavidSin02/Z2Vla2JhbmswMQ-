數據的讀取和預處理

現在有一個文件名稱為：ex6data1.mat，這個文件的格式不是python的格式，它是matlib的格式，但是python也可以進行相關的讀取操作。

import scipy.io as sio
mat = sio.loadmat('ex6data1.mat')#讀取mat數據的方式
print(mat.keys())#查看mat文件中的所有變量

可以使用scipy.io方式來讀取mat文件，使用loadmat方法來加載這個文件，我們輸出這個文件的所有的變量，結果為：

dict_keys(['__globals__', '__version__', 'y', 'X', '__header__'])

這個表示在這個數據文件中有5個變量，其中我們只是用兩個變量，分別是X和Y，X表示我們的樣本，而Y表示我們的樣本所對應的而標籤

既然我們訓練我們的svm的模型，那麼我們就只獲取到我們想要的樣本x和標籤y就可以了。

data = pd.DataFrame(mat.get('X'), columns=['X1', 'X2'])
data['y'] = mat.get('y')
print (data)

我們使用mat.get（X）為獲取到我們數據中的X變量，但是此時的變量是<class 'numpy.ndarray'>類型的我們輸出它為：

我們將其進行轉換轉成pandas的DataFrame類型，並且指定數據的列名分別為x1和x2，此時我們data為：

此時的data是DataFrame類型，這就是二者的不同之處，此時我們data['y'] = mat.get('y')獲取到y，直接賦值給data，其中mat.get('y')為ndarray，賦值給data['y']之後，我們的data['y']就會自動變成<class 'pandas.core.series.Series'>類型，其實我們可以把Series可以理解為帶索引的ndarray，DataFrame類型中的一列其實就是Serise類型，我們最終輸出data，最終的結果為:

這個就相當於預處理好了，就是我們想要的那個數據集，有兩個特徵x1和x2，還有一個標籤y，那麼下面我們。就可以使用svm進行訓練了

模型的訓練

再訓練之前我們先把我們的當前數據集的圖給畫出來

fig, ax = plt.subplots(figsize=(8,6))
ax.scatter(data['X1'], data['X2'], s=50, c=data['y'], cmap='Reds')#c表示顏色
ax.set_title('Raw data')
ax.set_xlabel('X1')
ax.set_ylabel('X2')
plt.show()

我們使用matplotlib畫出散點圖效果為:

我們可以看出這就是數據的大概的一個分佈的情況，下面我們使用sklearn庫封裝好的svm算法來對數據進行訓練。

svc1 = clf = svm.SVC(C=10,kernel='linear')

svc1.fit(data[['X1', 'X2']], data['y'])

這裡因為我們的數據基本線性可分，所以沒有必要使用核函數來處理這個數據，至於鬆弛變量C是多少，我們只能去一點一點的來嘗試。這裡我們使用C=1，然後分類器訓練時的損失函數是hinge，然後我們來擬合這個數據fit，最終得到的svc1就是我們想要的訓練的svm的模型

我們可以查看我們的訓練模型的訓練準確率是多少

accuracy=svc1.score(data[['X1', 'X2']], data['y'])#訓練集準確率

結果就是0.980392156863，這就是我們訓練數據的精確度

如果要是想要查看我們的決策函數值我們可以使用decision_function方法，查看我們每個樣本的正副值，正樣本越大於0，表示越正，負樣本越小於0表示越負，最終我們可以看到所有樣本的決策函數值為：

我們可以通過查看每個類別預測的置信水平來看出這一點，這是該點與超平面距離的函數。

每個樣本的決策函數值

我們下面再來畫一幅圖只不過這幅圖不再按照正負樣本進行畫圖，而是根據每個樣本的決策函數值進行畫圖，畫圖的代碼為：

fig, ax = plt.subplots(figsize=(8,6))

ax.scatter(data['X1'], data['X2'], s=50, c=data['SVM1 Confidence'], cmap='RdBu')

ax.set_title('SVM (C=1) Decision Confidence')

plt.show()

畫圖的效果為：

下面的任務是畫出分割超平面和支撐超平面，首先我們先來分析一下這個應該怎麼畫比較合適，就拿這個圖的數據可知，我們有兩個特徵x1和x2，我們的分割超平面的方程為w1x1+w2x2+b=0，我們完全可以把x1看著是自變量，x2看作是因變量，這樣我們就可以在這個圖中畫出一條直線，所以我們的方程變成了

所以我們只需要構造出這個方程來，我們的縱座標的x2就有了，x1我們可以隨機產生，然後使用畫圖工具畫圖就ok了，代碼為：

w=svc1.coef_[0]#獲取W
b=svc1.intercept_[0]#獲取b
w1=w[0]#獲取w1
w2=w[1]#獲取w2
x1=np.linspace(-5,5)#隨機生成1，這可以保證x1的連續性
xx2=-w1/w2*x1-b/w2#按照剛才的公式獲得x2

現在x1和x2有了，下面我們使用畫圖工具來畫出這個分割平面

plt.plot(x1,x2,'k-')

效果為：

下面我們畫上支撐超平面（這裡要註定一點的是支撐超平面其實就是y=w1x1+w2x2+b-1或者y=w1x1+w2x2+b+1），但是這裡我們使用x2作為因變量，所以現在就不能這樣思考這個問題了，因為我們現在畫圖沒有使用因變量y。

我們知道分割超平面和支撐超平面之間是平行的關係，所以首先我們的斜率-w1/w2現在已經有了，此時我們再求出截距b就可以了（截距相差不是1，這個不要和y=。。弄混）。我們可以設支撐超平面的方程為y=-w1/w2x+b。此時b=y+w1/w2x，所以只要找到這兩條支撐超平面上的任意一點帶入就可以得到b了，支撐超平面上的點很好找，因為支撐向量就再支撐超平面上。

supportb = svc1.support_vectors_[0]#第一個支撐向量

yy_down = a*x1 + (supportb[1] - a*supportb[0])

supportb = svc1.support_vectors_[-1]#最後一個支撐向量

yy_up = a*x1 + (supportb[1] - a*supportb[0])

之所以支撐向量選一個0和一個-1的目的就是為了讓我們選出的支撐向量恰好是兩條支撐超平面上的向量（這有時也會出一些問題，所以這裡要注意一下，這只是暫定的方法，這裡我實際上用的是-2，效果更好），我們帶入公式可以得出兩條支撐超平面的d了，下面我們畫圖：

plt.plot(x1,yy_down,'k-')

plt.plot(x1,yy_up,'k-')

顯示的效果為：

全部代碼為：

import numpy as np
import pandas as pd
from sklearn import svm
import seaborn as sns
import scipy.io as sio
import matplotlib.pyplot as plt
#數據預處理
mat = sio.loadmat('ex6data1.mat')#讀取mat數據的方式
print(mat.keys())#查看mat文件中的所有變量
data = pd.DataFrame(mat.get('X'), columns=['X1', 'X2'])
data['y'] = mat.get('y')
#畫出我們所有的數據，可以看出數據的大概的分佈
fig, ax = plt.subplots(figsize=(8,6))
ax.scatter(data['X1'], data['X2'], s=50, c=data['y'], cmap='Reds')
ax.set_title('Raw data')
ax.set_xlabel('X1')
ax.set_ylabel('X2')
#擬合我的數據svm
svc1= svm.SVC(C=10,kernel='linear')
svc1.fit(data[['X1', 'X2']], data['y'])
#訓練集準確度
accuracy=svc1.score(data[['X1', 'X2']], data['y'])#訓練集準確率
#構造切分超平面的直線
w=svc1.coef_[0]#獲取W
b=svc1.intercept_[0]#獲取b
w1=w[0]#獲取w1
w2=w[1]#獲取w2
x1=np.linspace(-5,5)#隨機生成1，這可以保證x1的連續性
x2=-w1/w2*x1-b/w2#按照剛才的公式獲得x2
#構造支撐超平面的直線
a = -w[0]/w[1]
supportb = svc1.support_vectors_[0]#第一個支撐向量
yy_down = a*x1 + (supportb[1] - a*supportb[0])
supportb = svc1.support_vectors_[-2]#最後一個支撐向量
yy_up = a*x1 + (supportb[1] - a*supportb[0])
#畫出這三條直線
plt.plot(x1,x2)
plt.plot(x1,yy_down)
plt.plot(x1,yy_up)
plt.show()
#決策值
data['SVM1 Confidence'] = svc1.decision_function(data[['X1', 'X2']])
#根據決策值畫圖
fig, ax = plt.subplots(figsize=(8,6))
ax.scatter(data['X1'], data['X2'], s=50, c=data['SVM1 Confidence'], cmap='RdBu')
ax.set_title('SVM (C=1) Decision Confidence')
w=svc1.coef_[0]
b=svc1.intercept_[0]
w1=w[0]
w2=w[1]
xx1=np.linspace(-5,5)
xx2=-w1/w2*xx1-b/w2
plt.plot(xx1,xx2)
plt.show()
