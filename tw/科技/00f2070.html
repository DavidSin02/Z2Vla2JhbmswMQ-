NaOH在無機化學中的作用

氫氧化鈉（俗名火鹼、燒鹼、苛性鈉）常溫下是一種白色晶體，具有強腐蝕性。易溶於水，溶解時放出大量熱，其水溶液呈強鹼性，具有鹼的通性，能使酚酞試液變紅、紫色石蕊試液變藍、甲基橙試液變黃。

1．酸性氧化物都可跟NaOH反應生成含氧酸鹽和水或生成酸式鹽：

CO2少量：CO2+ 2NaOH == Na2CO3，

CO2過量：CO2+ NaOH == NaHCO3

SO2少量：SO2+ 2NaOH == Na2SO3+ H2O，

SO2過量：SO2+ NaOH == NaHSO3

SO3+ 2NaOH == Na2SO4+ H2O，

P2O5+6 NaOH == 2Na3PO4+3H2O

SiO2+ 2NaOH ==Na2SiO3+ H2O

2．強酸或弱酸都可跟NaOH反應生成鹽和水；多元弱酸過量時，還會生成酸式鹽：

H2SO4+ 2NaOH ==Na2SO4+ 2H2O

HClO+ NaOH==NaClO+H2O

H2SO3+ 2NaOH ==Na2SO3+ 2H2O，

H2SO3+ NaOH ==NaHSO3+H2O

H2S + 2NaOH == Na2S+ 2H2O，

H2S + NaOH == NaHS+ H2O

H2SiO3+ 2NaOH ==Na2SiO3+ 2H2O

3．銨鹽能與NaOH溶液反應放出氣體：

NH4++ OH－NH3↑+H2O (NH4+的特性檢驗反應)

4．NaOH作為沉澱劑可與許多金屬離子發生反應生成各種顏色的沉澱：

Mg2++2OH－==Mg(OH)2↓(白色沉澱不溶於過量NaOH)

Al3++3OH－==Al(OH)3↓(白色沉澱可溶於過量NaOH)

Zn2++2OH－==Zn(OH)2↓(白色沉澱可溶於過量NaOH)

Fe2++2OH－==Fe(OH)2↓(白色沉澱不溶於過量NaOH，放置後轉為灰綠色，最後轉為紅褐色)

Fe3++3OH－==Fe(OH)3↓(紅褐色沉澱不溶於過量NaOH)

Cu2++2OH－==Cu(OH)2↓(藍色沉澱不溶於過量NaOH)

Ag++OH－== AgOH↓(白色沉澱不溶於過量NaOH，沉澱易轉為暗褐色)

5．酸式鹽都可跟 NaOH溶液反應：

(1)強酸酸式鹽如NaHSO4等，NaHSO4+NaOH==Na2SO4+H2O(H++OH－== H2O)

(2)弱酸酸式鹽如NaHCO3、NaHS、NaHSO3、NaH2PO4、Na2HPO4等，

NaHCO3+NaOH == Na2CO3+H2O (HCO3－+OH－==CO32－+H2O)

NaHS +NaOH == Na2S+H2O (HS－+OH－==S2－+H2O)

NaH2PO4+2NaOH ==Na3PO4+2H2O(H2PO4－+2OH－==PO43－+2H2O)

(3) 陰、陽離子均可反應的酸式鹽如Ca(HCO3)2、Mg(HCO3)2、NH4HSO3等，

NaOH過量：Ca(HCO3)2+2NaOH == Na2CO3+2H2O+CaCO3↓

Ca2++2HCO3－+2OH－== CO32－+2H2O+CaCO3↓

NaOH不足量：Ca(HCO3)2+NaOH == NaHCO3+H2O+CaCO3↓

Ca2++HCO3－+OH－== H2O+CaCO3↓

Mg(HCO3)2+4NaOH == Na2CO3+2H2O+ Mg(OH)2↓

NH4HSO3+2NaOH==（加熱）Na2SO3+2H2O+NH3↑

6．兩性化合物都可跟NaOH反應：Al2O3、ZnO、Al(OH)3、Zn(OH)2

Al2O3+2OH－+3H2O ==2[Al(OH)4]－，Al(OH)3+OH－== [Al(OH)4]－

ZnO+2OH－+H2O ==[Zn(OH)4]2－，Zn(OH)2+2OH－== [Zn(OH)4] 2－

7．鋁、硅單質能與NaOH反應放出氣體：

2Al+2OH－+6H2O ==2[Al(OH)4]－+3H2↑，Si+2OH－+H2O===SiO32－+2H2↑

8．非金屬單質與NaOH發生歧化反應：

Cl2等鹵素單質(吸收餘氯、除去溴苯中的溴單質)

Cl2+2NaOH==NaClO+NaCl+H2O，Br2+2NaOH==NaBrO+NaBr+H2O

S單質(洗滌粘在試管中的硫單質) 3S+6NaOH(熱)==Na2SO3+2Na2S+3H2O

9．非金屬氧化物與NaOH發生歧化反應或歸中反應：

2NO2+2NaOH==NaNO3+NaNO2+H2O

NO+NO2+2NaOH== 2NaNO2+H2O

10．某些複雜的反應NaOH作為鹼性溶液參與反應：

如：亞硫酸鈉鹼性溶液中滴加碘水，碘水褪色：

SO32－+I2+2OH-===SO42－+2I－+H2O

向鹽滷中加入NaClO和少量NaOH的方法可將Fe2+、Fe3+等除去：

2Fe2++ ClO－+4OH－+H2O===Cl－+2Fe(OH)3↓，Fe3++3OH－==Fe(OH)3↓
