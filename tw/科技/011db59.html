VBA過程代碼281：利用數組和字典，對字典嵌套的理解

Private Sub ComboBox1_Change()

ComboBox2.Clear

'二級下拉框對應的是第一級字典的鍵值為鍵的字典

ComboBox2.List = mydic(ComboBox1.Value).keys

End Sub

Private Sub UserForm_Activate()

'將數據裝入數組

myarr = Range("a1").CurrentRegion.Value

Set mydic = CreateObject("Scripting.Dictionary")

For i = 2 To UBound(myarr)

strF = myarr(i, 1)

strS = myarr(i, 2)

If Not mydic.Exists(strF) Then

'建立嵌套字典，第一重字典的鍵對應的鍵值為字典

Set mydicTemp = CreateObject("Scripting.Dictionary")

'注意下面的兩種寫法

'Set mydic.Item(strF) = mydicTemp

Set mydic(strF) = mydicTemp

End If

'注意下面的兩種寫法，嵌套字典的字典的鍵記入，鍵值為空

'mydic.Item(strF).Item(strS) = ""

mydic(strF)(strS) = ""

Next i

'一級下拉框對應的是第一級字典的鍵

ComboBox1.List = mydic.keys

End Sub

代碼講解：上述代碼利用了字典的嵌套，解決了兩級下拉菜單的製作問題。代碼中利用FOR NEXT在數組中建立循環,當Not mydic.Exists(strF),將數據存入字典,鍵是strF ,鍵值是一個字典:mydicTemp,如果能夠找到外層字典的鍵,那麼將建立內層字典的,即: mydic(strF)(strS) = "",此處大家要好好理解一下.一級下拉框對應的是第一級字典的鍵，二級下拉框對應的是第一級字典的鍵值為鍵的字典。
