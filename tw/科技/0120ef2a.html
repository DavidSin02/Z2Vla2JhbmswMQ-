算法：鏈表的兩數相加（JAVA）

2020-11-14
| 科技

對於兩數相加問題，假如您能靜下心來去牛客或者力扣做一些算法題，就會發現這類型的問題很多，解法幾乎大致相同，都是位數相加，然後進位，得出結果。

題目：給定兩個鏈表：鏈表1：1->2->3 鏈表2： 3->4->5 鏈表相加的結果輸出為：4->6->8

解題的大致思路就是：兩個鏈表的每一位數對應相加。大於10的部分進位，直到鏈表遍歷完成。

    /**     * Definition for singly-linked list.     * public class ListNode {     *     int val;     *     ListNode next;     *     ListNode(int x) { val = x; }     * }     */    class Solutiones {        public ListNode addTwoNumbers(ListNode l1, ListNode l2) {            int add = 0;   //進位數字            ListNode result = new ListNode(-1);  //新的鏈表            ListNode dummy = result;    //鏈表移動的起點            while(l1!=null || l2!=null || add!=0){                int x = l1==null?0:l1.val;                int y = l2==null?0:l2.val;                int sum = x + y + add;    //節點對於相加                dummy.next = new ListNode(sum%10);                dummy = dummy.next;  //新鏈表後移                add = sum /10;   //獲得進位樹                if(l1!=null){l1 = l1.next;}                if(l2!=null){l2 = l2.next;}            }            return result.next;        }    }
