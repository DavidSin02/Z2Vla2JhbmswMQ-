批量處理文件夾裡的文件

2020-10-29
| 科技

我們在工作中，一定遇到過需要對批量的excel文件進行同一種操作的情況。如果文件不多，我們當然可以一個一個文件打開，然後修改、保存、關閉。但是，如果有幾十個、幾百個文件，我們再這樣操作就相當的浪費時間了。

寫這篇文章就是因為有一個網友前幾天問了我一個這樣的問題，希望他能夠看到這篇文章。

今天我們就來學習一下使用VBA代碼來解決這個重複、枯燥的問題的方法。

既然要用到VBA，那我們就需要知道VBA編輯器在哪裡。在Excel中，是自帶了VBA編輯器的，只是通常情況下，在Excel的工具欄裡是看不到的，這需要我們進行一些設置。

調出【開發工具】選項卡

如下圖所示，我們依次點擊文件--選項--自定義功能區，在“自定義功能區”下勾選“開發工具”選項，然後點擊“確定”即可將“開發工具”選項卡顯示在工具欄上。

如下圖所示，VBA代碼編輯器的入口就在“開發工具”選項卡下面。點擊“開發工具”選項卡，我們可以看到工具欄的最左邊有一個“Visual Basic”按，點擊它就可以調出VBA代碼編輯器。

VBA代碼編輯器如下圖：

插入一個模塊

在VBA代碼編輯器中，雖然不一定要在“模塊”裡編輯代碼，但是我習慣了，所以我們還是插入一個模塊吧。

在編輯器左邊的“工程”窗口下找與我們打開的Excel文件同名的項目，點擊鼠標右鍵，在彈出的菜單欄裡選擇“插入”，在彈出的子菜單裡點擊“模塊”，即可在下面插入一個模塊。

好了，模塊插入成功後，我們“雙擊”剛才插入的“模塊”，就可以在右邊的代碼編輯區域裡編輯VBA代碼了。

代碼編輯

※ 第一步、獲取文件夾裡的所有Excel文件的文件名

獲取文件夾內所有Excel文件的文件名的代碼如下：

Sub dir_file() Dim path As String  '輸入文件夾路徑 path = InputBox("請輸入文件夾路徑：")  '讀取文件名後綴為 .xlsx 的文件的文件名 filetest = Dir(path & "\" & "*.xlsx") If filetest <> "獲取文件夾裡所有的文件名.xlsm" Then  '把文件名寫入表格 Cells(1, 1) = filetest end if i = 2  '循環讀取文件夾裡的文件名 Do While filetest <> "" filetest = Dir  '將文件名依次寫入本工作簿、本工作表的A列 If filetest <> "獲取文件夾裡所有的文件名.xlsm" Then Cells(i, 1) = filetest i = i + 1 End If Loop ' 在這裡插入後續代碼' 在這裡插入後續代碼   End Sub

※ 第二步、依次打開文件、編輯文件

將下面的代碼插入上面的代碼裡面，就可以實現打開Excel文件並對文件內容進行編輯的功能了。

 '依次打開文件然後進行操作 Dim opf As Workbook j = 1 Do While Cells(j, 1) <> "" '從表格中獲取文件名 file_name = Cells(j, 1).Value  '根據文件名打開文件 Set opf = Workbooks.Open(path & "\" & file_name)  '對打開的Excel文件進行編輯 opf.Worksheets(1).Cells(1, 2) = file_name   '保存文件 opf.Save '關閉文件 opf.Close  j = j + 1 Loop

在上面的這一段代碼中，如下圖所示的這一段代碼就是對打開的Excel文件的具體操作的代碼，如果你需要對Excel文件進行其他的操作，你可以錄製一段宏代碼，然後用這段宏代碼替換掉這一段代碼就可以了。

調用VBA代碼

我們可以通過插入按鈕來調用VBA代碼。

如下圖所示，點擊“開發工具”下的“插入”按鈕，在彈出的選擇列表下的“表單控件”下，選擇“按鈕（窗體控件）”，然後在下面的表格編輯區裡拖動鼠標繪製一個按鈕。

在彈出的“指定宏”窗口中，選擇你剛剛編輯的VBA代碼後點擊確定。

好了，現在你點擊這個按鈕就可以調用剛剛編輯的VBA代碼了。
