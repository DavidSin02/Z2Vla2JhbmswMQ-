ubuntu git生成ssh key (公鑰私鑰)配置github或者碼雲

2020-10-29
| 科技

Git是分佈式的代碼管理工具，遠程的代碼管理是基於SSH的，所以要使用遠程的Git則需要SSH的配置。

github的SSH配置如下：

設置Git的user name和email：

git config --global user.name "xx"

git config --global user.email "xx@gmail.com"

生成SSH密鑰過程：

1.查看是否已經有了ssh密鑰：cd ~/.ssh

如果沒有密鑰則使用下面命令生成

2.生存密鑰：

ssh-keygen -t rsa

按3個回車，密碼為空。

最後得到了兩個文件：id_rsa和id_rsa.pub

  • 查看id_rsa.pub公鑰

cat ~/.ssh/id_rsa.pub

  • 打開id_rsa.pub文件，並且複製全部內容。

4).打開GitHubn賬戶，打開設置找到SSH Keys:

點擊Add key將剛剛複製的內容添加到Key的文本域中

這樣就添加了一個SSH key。

5).完成上面的步驟之後就可以使用ssh來連接GitLab，並進行相應的操作了。
