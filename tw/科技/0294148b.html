PTT VBA｜統一或指定幻燈片進行文本和段落格式化

2020-11-14
| 科技

整個Office系列軟件都可以支持VBA二次開發，應當說，EXCEL對VBA支持最好，其次是Word，對於PPT來說，支持是較差的。

首先PPT不支持錄製宏操作，其次，提供的對象的屬性成員和成員方法也很欠缺。從其提供的幫助文檔就可以看出：https://docs.microsoft.com/zh-cn/office/vba/api/overview/powerpoint

現在的任務是，對全部幻燈片及每一張的全部對象統一進行文本和段落格式化。

代碼邏輯很簡單，就是遍歷全部幻燈片及每一個幻燈片的形狀對象。

For Each oSlide In oPres.Slides……NextFor Each oShape In oSlide.Shapes……Next

然後對每一個對象應用屬性。

只要熟悉相應對象語法即可。

運行效果：

當然，如果你只想設置其中的某一張幻燈片，或其中的第j個對象，可以把上面代碼的兩個循環去掉，具體指定幻燈片和第j個對象即可：

 i = oPres.Slides.Count j = 0'設置最後一個幻燈片或指定幻燈片的第一個對象的字體和段落格式 Set oSlide = oPres.Slides.Item(i) Set oShape = oSlide.Shapes.Item(1)

附代碼1：

Sub allSlideAllShapes()

On Error Resume Next

'對象和變量聲明

Dim oPres As Presentation

Set oPres = Application.ActivePresentation

Dim oSlide As Slide

Dim oShape As Shape

Dim tr As TextRange

Dim i As Long, j As Long

'遍歷全部幻燈片及每一個幻燈片的形狀對象

For Each oSlide In oPres.Slides

For Each oShape In oSlide.Shapes

With oShape '設置文本框的寬度和位置，適合只有一個文本框的，

'如果有多個，下面三行代碼要註釋掉，不然重疊到一起了

'.Left = 45

'.Top = 45

'.Width = 625

End With

If oShape.TextFrame.HasText = msoTrue Then

Set tr = oShape.TextFrame.TextRange

With tr.Font

.NameAscii = "宋體"

.NameFarEast = "宋體"

.Size = 28

.Color.SchemeColor = ppBackground

.Color.RGB = RGB(Red:=0, Green:=0, Blue:=0)

.Bold = msoFalse

End With

tr.ParagraphFormat.SpaceWithin = 1.1 '設置行距

Set tr = Nothing

End If

'oShape.Fill.Background '文本框背景色用幻燈片背景填充

'oShape.TextFrame.TextRange.IndentLevel = 1

Next

Next

'對象刪除

Set oShape = Nothing

Set tr = Nothing

Set oSlide = Nothing

Set oPres = Nothing

End Sub
