「按鍵精靈安卓版」找出不同顏色的文字（找圖方向參數的應用）

2020-11-14
| 科技




群裡有人問如何識別上圖得出紅色字體。（每次文字不一樣，字體個數不一樣，唯有字體顏色一樣）

這種識別一般是作為填空題的提示信息，提取出紅色文字回答題目，通常我們可以通過獲取界面的xml，然後根據題目中空缺前後的內容，進行截取法獲取紅色文字，但是如果是圖片的話，就需要做內容識別了。

當然，圖片識別我們可以做字庫，可以對接打碼平臺，我個人喜歡的的方式是對接百度雲識別。

對於此次案例中的文字，在識別之前我們先要把沒用的黑色文字去掉，這裡我用的是截圖的方式，直接截取紅色文字。

下面難題來了，怎麼精準截圖呢？文字的字數、位置都是不固定的，所以要想截圖，就要先定位確定圖色命令的範圍。


我們知道圖色命令範圍的表示方式是4個數字，前兩個是左上角的橫座標和縱座標，後兩個是右下角的橫座標和縱座標，就像上圖箭頭指向的位置。

那麼我們只需要找到這兩個點的座標就可以確定圖色命令的範圍。

找點的座標，這裡我用到找色命令FindColor。

左上角的座標

FindColor 0, 0, 0, 0, "0A0AFF-222222", 5, 0.9, intX, intY

右下角的座標

FindColor 0, 0, 0, 0, "0A0AFF-222222", 8, 0.9, x, y

兩次命令有兩處不一樣的地方，

一、用來存儲座標的變量名字不一樣，通過不重名可以把兩次座標都存儲起來。

二、找圖方向，分別5和8。



當你看這個找色命令的介紹的時候，會發現並沒有5和8這兩個找圖方向，只有0-4，如下圖。



原本我以為也是這樣的，但是當我測試發現這0-4的找色都無法滿足我想要的效果。

比如，找左上角點的座標，我希望的方向是從上往下，從左往右。如圖：



找右下角的座標，我希望的方向是從下往上，從右往左，如圖：



當我把鼠標放到命令上的時候，命令提示顯示（點看圖片查看）



其實還有四個方向，其中5和8正好是我想要的。算是“隱藏”功能了。

通過找色命令獲取到範圍，我就可以用截圖命令，把紅色文字單獨截取出來了。之後在通過百度雲識別獲取到具體的文字內容。

需要注意一點，我在找色命令使用的時候，加上了偏色-222222，增加了找色的容錯率。



找範圍這部分代碼是：

Dim x,y,intx,inty

FindColor 0, 0, 0, 0, "0A0AFF-222222", 5, 0.9, intX, intY

FindColor 0, 0, 0, 0, "0A0AFF-222222", 8, 0.9, x, y

If intx > -1 And x > -1 Then

TracePrint x,intx

SnapShot ("/sdcard/pictures/b1.png",intx-2,inty-2,x+2,y+2)

End If

百度雲識別代碼中涉及到我的apikey，我就不復制，在我之前的文章有專門的講解，可以自行查看。好啦，內容就這麼多了。
