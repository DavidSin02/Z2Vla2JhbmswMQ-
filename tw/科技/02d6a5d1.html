免費且支持 218 種格式的全能格式轉換工具，這款神器你值得收藏

2020-11-14
| 科技

在線工具擁有無需下載、即點即用的優勢，所以我們常會藉助在線工具進行臨時的格式轉換工作。但大部分的格式轉換工具僅支持少量格式，像支持圖片轉換的工具，就未必支持視頻或文檔的轉換。如果你正在尋找一個支持齊全的文件格式的轉換工具，不想每次再去尋找單獨類型的轉換工具，那麼不妨試試今天推薦的 CloudConvert。

CloudConvert 最引以為豪的就是它支持 218 種文件格式的轉換。無論是壓縮文件、音視頻文件還是文檔、網頁，CloudConvert 都能幫你完成轉換。你可以在 這裡 查看 CloudConvert 完整的格式支持列表。

支持的文件類別

在 CloudConvert 的網站上，你能看到它「Convert Anything to Anything」的標語。事實上，CloudConvert 不但支持轉換同類別的文件格式（比如將 JPG 轉換為 PNG），它甚至可以跨類別轉換，比如將 DOCX 文檔轉換成 PNG。

選擇文件格式

除了提供在線轉換服務外，CloudConvert 還提供了 API 接口讓我們擴展它的功能。你可以將它接入你編寫的程序，或是通過命令行完成批量轉換。如果你是自動化工具 Zapier 的用戶，你甚至可以直接將其與 Zapier 整合，而無需編寫任何代碼。

強大的 API 支持

ConvertCloud 在 GitHub 上公開了多種語言所編寫的官方程序包，你可以在 CloudConvert 的 API 頁面 查看第三方程序包、應用實例及官方文檔。

CloudConvert 不但支持從本地直接上傳文件，還可以通過鏈接獲取網絡文件。在註冊賬戶後，你還可以連接至 Dropbox、Google Drive 等雲盤，直接從雲盤獲取文件。免費用戶每天只能享有 25 分鐘的轉換時間，且文件不得大於 1GB。如果你有更多的需求，可以單獨購買轉換時間或進行訂閱，你可以在 這裡 查看收費標準。

除此之外，CloudConvert 還提供了相應的 iOS 客戶端，你可以在 App Store 免費下載使用。雖然 CloudConvert 目前還沒有開發桌面端應用，但是它也提供了一個 Chrome 應用，感興趣的朋友可以從 Chrome Web Store 下載。
