JDBC從Oracle中取出long字段類型注意事項

2020-10-29
| 科技

使用JDBC連接Oracle數據庫，當取出的字段類型中包含long時，請注意以下幾點

1.long類型的字段，在取的時候，只能獲取一次，如再次獲取否則就會報java.sql.SQLException: 流已被關閉，這個錯誤，代碼如下：

package com.hh.frame.dbmg.test.ora;import java.sql.Connection;import java.sql.DriverManager;import java.sql.PreparedStatement;import java.sql.ResultSet;import java.sql.SQLException;import org.junit.FixMethodOrder;import org.junit.Test;import org.junit.runners.MethodSorters;@FixMethodOrder(MethodSorters.NAME_ASCENDING)public class LongTest {	@Test	public void long_test() throws Exception {		Connection conn = null;		PreparedStatement ps = null;		try {			Class.forName("oracle.jdbc.driver.OracleDriver");			conn = DriverManager.getConnection("jdbc:oracle:thin:@192.168.1.123:1521:orcl11g", "test", "123456");			String createSql = "create table long_test1(id int,comments long,age int)";			String insertSql = "insert into long_test1 values(1,'測試測試測試測試測試測試測試測試',18)";			String selectSql = "select comments,age,id  from long_test1";			//建表			ps=conn.prepareStatement(createSql);			ps.execute();			//插入測試數據			ps = conn.prepareStatement(insertSql);			ps.execute();			//執行查詢操作			ps=conn.prepareStatement(selectSql);			ResultSet rs = ps.executeQuery();			while (rs.next()) {				//此處獲取long類型字段comments				String comments = rs.getString("comments");				//此處再次獲取long類型字段comments，則會報錯				comments= rs.getString("comments");				int id = rs.getInt("id");				int	age =  rs.getInt("age");				System.out.print("age: " + age);				System.out.print("id: " + id);				System.out.print("comments: " + comments);				ps=conn.prepareStatement("drop table long_test1");			ps.execute();			}		} catch (SQLException e) {		ps=conn.prepareStatement("drop table long_test1");			ps.execute();			e.printStackTrace();		} finally {			try {				if (conn != null)					conn.close();			} catch (SQLException e) {				e.printStackTrace();			}		}	}}

錯誤如下：

2.如果SQL中包含long字段，在提取的過程中，如果SQL中long字段後的字段在讀取long字段之前被讀取就會出現流的錯誤，否則就不會，假設sql語句是 select comments,id，age from long_test1 ，其中comments為long類型，在取出的時候因為id和age是排在comments後面的，所以要在comments之後讀取，否則就會報 java.sql.SQLException: 流已被關閉，這個錯誤。

package com.hh.frame.dbmg.test.ora;import java.sql.Connection;import java.sql.DriverManager;import java.sql.PreparedStatement;import java.sql.ResultSet;import java.sql.SQLException;import org.junit.FixMethodOrder;import org.junit.Test;import org.junit.runners.MethodSorters;@FixMethodOrder(MethodSorters.NAME_ASCENDING)public class LongTest {	@Test	public void long_test() throws Exception {		Connection conn = null;		PreparedStatement ps = null;		try {			Class.forName("oracle.jdbc.driver.OracleDriver");			conn = DriverManager.getConnection("jdbc:oracle:thin:@192.168.1.123:1521:orcl11g", "test", "123456");			String createSql = "create table long_test1(id int,comments long,age int)";			String insertSql = "insert into long_test1 values(1,'測試測試測試測試測試測試測試測試',18)";			String selectSql = "select comments,age,id  from long_test1";			//建表			ps=conn.prepareStatement(createSql);			ps.execute();			//插入測試數據			ps = conn.prepareStatement(insertSql);			ps.execute();			//執行查詢操作			ps=conn.prepareStatement(selectSql);			ResultSet rs = ps.executeQuery();			while (rs.next()) {				//此處先獲取排在comments字段後面的id字段，會報流的錯誤				int id = rs.getInt("id");				String comments = rs.getString("comments");				int	age =  rs.getInt("age");				System.out.print("age: " + age);				System.out.print("id: " + id);				System.out.print("comments: " + comments);			}			ps=conn.prepareStatement("drop table long_test1");			ps.execute();		} catch (SQLException e) {		ps=conn.prepareStatement("drop table long_test1");			ps.execute();			e.printStackTrace();		} finally {			try {				if (conn != null)					conn.close();			} catch (SQLException e) {				e.printStackTrace();			}		}	}}

當查詢語句為：select id，age，comments from long_test1的時候，此時取出順序則沒有要求，因為long類型字段comments後面沒有值，所以不需要按查詢順序取出。

當查詢語句為：select id，comments，photo from long_test1的時候，因為在long字段類型後面只有一個age字段，所以，在取出的時候，age字段必須要最後一個取出。
