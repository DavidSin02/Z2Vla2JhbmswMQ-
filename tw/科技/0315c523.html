各大船公司復工/停航/冷箱擁堵費/艙單新規/堆場復工最新信息彙總

2020-10-29
| 科技

2月17日以來，各大船公司陸續復工，近日不少船公司又公佈一批停航信息；同時由於疫情期間港口和拖車不能及時返崗復工，導致港口貨物積壓嚴重，特別是冷藏箱的積壓，已致多個港口冷插嚴重匱乏，近期多家船公司對出口至中國部分港口的冷藏箱貨物收取擁堵附加費，最高1250美元/每箱；有船公司通知國外海關新規。另附深圳和寧波堆場復工情況。下面做一下彙總。


MSK


根據馬士基大中華區各辦公室運營情況顯示：


除武漢辦公室還在關閉中，其他各地區辦公室都恢復正常運營！同時馬士基也指出，基於員工的健康和安全的考量，目前絕大部分馬士基員工保持在家遠程辦公。與此同時，馬士基的櫃檯業務仍正常運行，服務客戶無間斷。

現代商船


HMM在內的所有船公司，在上海、寧波和新港碼頭仍然面臨由於提箱速度緩慢而導致冷凍箱插頭供給不足的問題。碼頭正在超負荷作業。

HMM由衷地希望能幫助您解決這些問題，但同時我們也不得不勸說冷凍箱客戶儘量避免那些擁堵的港口，並試圖找到替代市場，直到這些碼頭恢復正常運作。對於在途貨物，若客戶同意，HMM將會其安排更改目的地。

HMM將繼續與意向將冷凍箱運輸到上海、寧波和新港的客戶合作，但客戶在向HMM訂艙的同時，須接受以下條件。


1. 對於目的地是上海、寧波和新港的，HMM無法提供保證運輸時間的服務。所有的航線都可能被最佳選擇所取代，並且在這些港口接受冷凍箱卸貨之前，都有可能會被安排轉運。


2.從2020年2月12日起，HMM將對所有前往上海、寧波和新港的冷凍箱訂艙收取每箱1000美元的擁堵附加費，以支付不可預估的轉運和在替代港口延長停留時間的額外費用。對於某些特定航線，此附加費將從2020年3月12日起生效。

海運抵達艙單 (Sea Arrival Manifest ”SAM”）

– 適用於所有進入印度的集裝箱：

  1. 海運抵達艙單 (SAM) 必須在船舶離開最後一個外國停靠港之前提交。例如: 韓國 / 中國 / 印度次大陸航線 (KCIS) 及中國 / 印度 / 越南航線 (CIVS) 西行航線就是巴生港， 而達累斯薩拉姆就是海灣 / 印度 / 非洲航線 (GIA) 北行航線最後一個外國停靠港。
  2. 裝貨港必須嚴格遵守裝運指示規定的截止時間 。
  3. 所有在印度港口作為 當地進口卸貨 或 中轉運卸貨 或 保留在船上的集裝箱。
  4. 強制性提供 6位 海關編碼 （HS CODE）。
  5. 艙單及提單必須強制性顯示進口商的進出口號碼（IEC），消費稅進口商的身份證號碼（GSTIN）和通知方（印度具名實體）的永久賬戶號碼（PAN）。
  6. 貨物發票金額，目前在海運抵達艙單(SAM)上是非強制的，亦不需要顯示在提單上。

海運出境貨物艙單 (Sea Departure Manifest ”SDM”)

– 適用於所有出口印度的集裝箱

  1. 強制性出境貨物艙單規定，船舶在印度裝貨港開航前，必須向海關提交出境貨物艙單。
  2. 適用於所有在印度港口裝載的集裝箱。
  3. 客戶必須在船舶到達裝貨港至少48小時之前提交裝運指示。
  4. 客戶必須在集裝箱入閘之前至少42小時提交貨物裝船準單。

批准艙單提交後的修改規定

海運抵達艙單 (SAM）

  1. 長途航行超過4天 -預計抵達時間48 小時前。
  2. 中途航行2至4天 -預計抵達時間24 小時前。
  3. 短途航行不到2天 -預計抵達時間6 小時前。
  4. 運載散裝貨物，不論其航程時間 – 直到貨物進口申報。

海運出境貨物艙單 (SDM)

  1. 一般不遲於船舶開航後24小時。
  2. 運載散裝貨物，不遲於船舶離境後72小時。

備註：以上時間表均為印度海關規定。所有有關修改均經由阿聯酋航運部門內部與外部評估及批准。

無船承運人提單（House BL “HBL”）申報安排

  1. 在印度的貨運代理人，必須在船舶離開最後一個外國停靠港之前72小時提交無船承運人提單（HBL）。
  2. DPD，DPD / CFS，的詳細轉運信息，應在當地港口的發運人或收貨人在船舶離開最後一個外國停靠港之前通知承運人。
