JavaScript中條件語句的使用

2020-10-29
| 科技

什麼是條件語句

JavaScript 語言中，條件語句（if 語句）常用於基於不同條件執行不同的動作。簡單來講就是判斷給出的某個條件是否是正確的，如果條件正確要如何做，條件錯誤要如何做。舉一個例子，例如現在有一個變量 age，給定一個條件語句為 “age是否大於18”，如果大於18 則可以玩遊戲，否則不可以玩遊戲。

var age = 20;if(age > 18){    console.log("你可以玩遊戲喲");}else{    console.log("未成年不可以玩遊戲");}

在 JavaScript 中，我們可以使用的 if 條件語句有如下幾種：

  • if 語句：當指定條件為 true 時，使用該語句來執行代碼。
  • if-else 語句：當指定條件為 true 時執行 if 後面的代碼，為false執行 else 後面的代碼。
  • else if 語句：當要指定多個條件時，可以在 if 語句後面加 else if 語句。

if 語句

if 語句是最基本的條件語句，規定假如條件為 true 時，則執行花括號 {} 中的代碼塊。

語法如下所示：

if (condition){    // 條件為 true 時要執行的代碼塊}

其中 condition 表示條件，並且 if 只能小寫，後面必須接英文的花括號{}，如果不按照語言要求寫代碼會報錯。

示例：

例如我們給定一個條件，當變量 num 大於10，輸出“嘻嘻嘻”，我們可以這樣寫：

var num = 15;if(num > 10){    console.log("嘻嘻嘻");}

if-else語句

if-else 語句規定假如條件為 true 時，執行 if 後面的花括號中的代碼塊，為 false 時則執行 else 後面花括號中的代碼塊。

語法如下所示：

if (condition) {    // 條件為真時要執行的代碼塊}else{    // 如果條件為false，則執行的代碼塊}

其中condition 是條件語句，else 後面不需要接條件語法。這也好理解，例如打個比方說，我今年大於18歲，那麼大於18歲就是一個條件if語句，只要不滿足這個條件，不管是等於還是小於18歲，都是 else。

示例：

同樣是一個關於時間的例子，根據當前時間是否符合標準來打招呼，12點之前是”Good morning“，12點之後是”Good afternoon“：

var hour = new Date().getHours();  // 獲取當前時間(小時)var greet;if(hour < 12) {    greet = "Good morning";} else {    greet = "Good afternoon";}console.log("現在時間為：" + hour);console.log("打個招呼吧：" + greet);

else if 語句

else if 語句和if 語句類似，後面也要接一個條件，例如if後面接的是條件1，else if 後面就接條件2，else 後面就是既不滿足條件1，又不滿足條件2 的其他情況。

語法如下：

if (condition1) {    // condition1 為 true 時，要執行的代碼塊} else if (condition2) {    // 當 condition1 為 false 而 condition2 為 true，則執行此代碼塊} else {    // 當 condition1 和 condition2 為 false，則執行這個代碼塊}

示例：

例如學生食堂早上8點吃早飯，中午12點吃午飯，晚上18點吃晚飯，其他時間不吃飯：

var hour = new Date().getHours();  // 獲取當前時間(小時)if(hour == 8) {    console.log("現在的時間為：" + hour + "點, 該吃早飯啦！");} else if( hour == 12) {    console.log("現在的時間為：" + hour + "點, 該吃午飯啦！");} else if( hour == 18){    console.log("現在的時間為：" + hour + "點, 該吃晚飯啦！");}else{    console.log("現在的時間為：" + hour + "點, 不是吃飯時間！");}
