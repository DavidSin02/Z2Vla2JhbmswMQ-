程序設計篇之選擇結構程序設計

2020-10-29
| 科技

1.關係表達式

①關係運算：是對兩個量進行“比較運算”；

②關係運算符：<、<=、>、>=、==、!= ；

③優先級別：算術運算符、關係運算符、賦值運算符（由高到低）；

④關係表達式：用關係運算符將兩個表達式（算術、字符表達式）連接起來的有意義的式子；

說明：數值量比較（數字的大小），字符量比較（ASCII的大小）；

⑤關係表達式的值：真（1）、假（0）；

⑥結合規則：左結合。

2.邏輯表達式

①邏輯運算符：&&(與)、||(或)、!(非)；

②優先級別：!(非)、算術運算符、關係運算符、&&、||、賦值運算符（由高到低）；

③邏輯量：當一個量（可以是基本類型的常量或變量）用作判時，

c編譯系統認為：非0--真、0--假。

④邏輯表達式：用邏輯運算符將關係表達式或邏輯量簡介起來的有意義的式子（單個邏輯量、關係表達式是邏輯表達式的特例）；

⑤邏輯表達式的值：真（1）、假（0）。

3.if語句

①if語句的三種形式：

1)條件執行 if(e) A;

2)分支選擇 if(e) A; else B;

當e為真值（非0）時執行A，否則執行B或後續語句。A或B都可以是單一語句，也可以是複合語句；else部分不能獨立存在，即else前一定有一個“;”，它必定是if語句的一部分。

注意：在c語言中，表達式e的值為非0時，系統均按“真值”處理。

3)else if形式（階梯式的if-else語句）

if(P1) S1

else if(P2) S2

...

else if(Pn) Sn

else Sn+1

系統從上到下的逐個判斷條件P，一旦發現某條件Pi滿足時，則執行與它有關的語句Si，並跳過其它剩餘的階梯。若所有條件均不滿足，執行最後一個else語句或後續語句。

注意：一般else與最近的if匹配。

②if語句的嵌套：else應與最近的if匹配；

由此可看出：通過加“{}”,可改變else的層位，從而改變程序的執行流程。

③條件運算符與條件表達式

條件運算符： ?和: /*唯一的三目運算符*/

條件表達式：e1 ? e2 : e3

執行過程：計算e1；e1值為非0（真），計算並返回e2的值否則計算並返回e3的值；

運算順序：高於賦值運算，低於算術、關係、邏輯運算；

條件運算符的結合方向是“右結合”。

4.switch語句

亦稱多分支選擇語句。它比用嵌套的if語句實現多路分支問題，其程序結構清晰、易讀。

①格式：

switch(e)

{

case c1: 語句1

case c2: 語句2

...

case cn: 語句n

[default: 語句n+1]

}

其中：表達式e可以是整型、字符型、枚舉型；

常量表達式ci必須與表達式類型一致（整型與字符型通用）；

②執行過程：計算表達式e的值；若與常量表達式ci值一致，則從語句i開始執行；直到遇到break語句或switch語句的"}"；若與任何常量表達式值均不一致時，則執行default語句，或執行後續語句。

注意：常量表達式ci僅起語句標號作用，不作求值判斷；常量表達式的值必須是唯一的，沒有先後次序；多個case語句可共用一組執行語句
