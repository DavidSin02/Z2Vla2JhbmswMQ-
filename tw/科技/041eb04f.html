阿里雲服務器快速建網站_安裝BT寶塔面板和wordpress

2020-11-14
| 科技


簡介

買了阿里云云服務器後，打算來折騰一下服務器。服務器都有哪些用途？

大概列一下常用的方法：

  1. 建網站、博客。就是這篇文章所說的內容。為甚麼要自己建立博客？因為博客在自己手上總是比較踏實。像社交網站興衰交替，以前寫的心得體會在舊的社交網站上都找不到了，而且自己博客限制也沒那麼大。
  2. 圖床，可以參考我此前寫的文章 七牛圖床添加阿里雲域名、阿里雲域名註冊與備案、服務器ECS購買與登錄
  3. 內網穿透、反向代理。比如說，遠程訪問內網NAS；遠程桌面；服務器深度學習（俗稱遠程煉丹）。這個我後續更新一下。
  4. 私人網盤。哈哈，以前都是用百度雲盤，自從pandownload掛了後，逐步想起這個念頭。
  5. 國外VPS上網。這個就不多說了。

下面就針對快速建網站來進行闡述。

概念通俗科普

解釋bt寶塔面板是什麼之前，先簡單說說一直以來建網站方法

  1. 比較久遠的方法，以前都是在Linux環境下，在黑洞洞的命令行窗口，安裝建立網站常用的軟件。建立網站肯定離不開三件套嘛——http服務器、數據庫、服務器運行語言。
  2. 安裝這三件套有各種各樣的組合。後來逐步形成LAMP（Linux+Apache+MySQL+PHP）和LNMP（Linux+Nginx+MySQL+PHP）幾個組合嘛。
  3. 這三件套一個個安裝需要逐個輸入命令，還有軟件版本對應和依賴問題，以及權限及相關配置，很麻煩。後來就有人就編寫了LAMP或者LNMP一鍵安裝命令腳本。比如以下是LAMP的一鍵安裝腳本。wget -c ftp://soft.vpser.net/lnmp/lnmp1.3-full.tar.gz && tar zxf lnmp1.3-full.tar.gz && cd lnmp1.3-full && ./install.sh lamp
  4. 上面只是解決了安裝的問題，關於網站後面還有很多維護工作需要——網站監控、數據庫備份、日誌記錄等等。如果有專門的服務器管理軟件就省事多了。因此服務器管理軟件（俗稱面板）就出現了。國外用Cpanel比較多，國內用BT寶塔面板。其中，Cpanel面板這個比較古老了，國外VPS廠商用的多。多數是英文版，並且操作起來不太符合操作習慣。BT面板是國產的，原生支持中文，官網介紹“提升運維效率的服務器管理軟件，支持一鍵LAMP/LNMP/集群/監控/網站/FTP/數據庫/JAVA等100多項服務器管理功能。”還有他是免費的，收費功能針對技術支持等業務。所以直接用bt面板來建網站和管理網站了。


為什麼用Wordpress

1、 像個人博客這種，hexo靜態網站也挺流行的。但是因為以前用wordpress，並且插件多，支持讀者評論這些動態內容。所以路徑依賴還是用回wordpress了。


快速建網站

  1. 命令行輸入一鍵安裝命令
yum install -y wget && wget -O install.sh [http://download.bt.cn/install/install.sh && sh install.sh](http://download.bt.cn/install/install.sh && sh install.sh)

輸入y確定安裝bt面板。

隨後等候一分鐘左右，出現成功安裝狀態，並且返回bt面板的用戶名和密碼。

  1. Bt面板登錄

用瀏覽器在地址欄輸入 命令行顯示的鏈接地址，一般為IP號碼+端口號格式——類似於http://144.144.144.144:8888格式，出現bt面板登錄界面。輸入命令行裡顯示的用戶名和密碼，進行登錄

  1. 修改BT面板登錄賬號和密碼。點擊左側導航欄“面板設置”，修改用戶名 和密碼。

  1. 選擇安裝LAMP或者LNMP。

選擇其中一樣進行安裝，我選擇LNMP。跳出安裝任務列表，等待所有安裝任務完成。

等待十幾二十分鐘後，安裝完成

  1. 利用面板添加網站

點擊左邊導航欄“網站”，再點擊添加站點，輸入域名、根目錄、FTP設置、數據庫設置等。

顯示成功建立相應資料

  1. 瀏覽站點創建頁面及主機目錄

瀏覽網站，在瀏覽器地址欄輸入域名，出現站點創建成功頁面。

  1. 上傳網站wordpress文件

下載wordpress壓縮包。然後瀏覽器中打開WordPress中文站點，下載程序壓縮包。

在面板左邊導航欄選擇“文件”，選擇網站站點文件夾，上傳wordpress.zip壓縮文件。

將上傳的wordpress.zip壓縮包解壓縮，解壓縮到wordpress文件夾。

進入該wordpress文件夾，將所有文件選擇，選擇剪切，將其粘貼至上一級文件夾。

最後網站目錄文件夾如下

  1. Wordpress配置

在瀏覽器地址欄輸入域名，跳出wordpress配置頁面

點擊“現在就開始”，跳轉到數據庫配置頁面。按照前面面板添加站點生成的數據庫配置信息填入。

數據庫配置完畢後，設置wordpress站點標題及賬戶信息。

添加完畢，登錄wordpress賬號及密碼，進入該網站頁面。

此時網站建立完畢。後續就是網站內容豐富和美化的事情。後續更新。
