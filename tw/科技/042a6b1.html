知識點理解：無雜散動態範圍(SFDR)

2020-10-29
| 科技

無雜散動態範圍(Spurious Free Dynamic range ，SFDR) 是衡量A/D和D/A轉換器(ADC/DAC)的指標，表示在雜散分量干擾基本信號或導致基本信號失真之前可用的動態範圍。

SFDR定義：正弦波信號(載波)均方根 (RMS)值與從0 Hz到二分之一數據轉換器採樣速率(Fs/2)範圍內測得的輸出峰值雜散信號均方根值之比。用公式表示如下：

數據轉換器的SFDR 常常受輸入信號的二次或三次諧波限制，峰值雜散分量可以是諧波關係，也可以是非諧波關係。通過精心設計濾波器和優化頻率分配，一般可避免二次諧波(HD2)和/或三次諧波(HD3)，從而大幅提高SFDR。

細心的你可能注意到上圖中出現了兩種單位：dBc和dBFS。其實，SFDR通常以dBc(相對於載波頻率幅度)或dBFS(相對於數據轉換器的滿量程範圍)表示。

在比較兩款數據轉換器的SFDR時，需給定基準電平以及工作和信號條件。我們來對比一下AD9625和AD9680的SFDR，打開它們的官方DataSheet，如下圖。其中紅色框中給出SFDR時都給出了對應的條件。

通常，SFDR是在預先定義的窗口或奈奎斯特頻率內觀測的。例如下圖，觀測窗口選擇中間的白色部分，其SFDR將大很多，這是因為整個Fs/2內的最大雜散分量並沒有落入定義的觀測窗口。

可以看出，限制SFDR性能的是最惡劣二次諧波或三次諧波，但如果使用正確信道濾波並採用理想採樣時鐘和良好頻率分配，可以輕鬆地濾除這樣的帶外諧波。

Matlab中有函數sfdr可以直接使用，具體你可以在Matlab中的幫助文檔中輸入sfdr查看。下面是利用該函數計算的SFDR，程序很簡單，僅供參考。

%% 測量SFDR

Fs = 200e6; % 採樣率

dt = 1/Fs;

T = 1e-5;

F0 = 10e6;

F2 = 20e6; % 假設二次諧波

F3 = 30e6; % 假設三次諧波

SNR = 60; % 信噪比

% 產生信號、雜散以及噪聲

t = 0:dt:T-dt;

st = cos(2*pi*F0*t)+0.001*cos(2*pi*F2*t)+0.002*cos(2*pi*F3*t);

st = awgn(st,SNR); 給信號加上噪聲

% Matlab函數測量SFDR

sfdr(st,Fs)
