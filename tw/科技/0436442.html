2017十大高分韓國電影排名 十部高人氣韓國電影推薦

2020-10-29
| 科技

說起韓國電影，人們最常提到的關鍵詞便是“感人”了，無論是《熔爐》還是《素媛》，不讓人掉下幾公升的淚水不罷休，但2017年的韓國高分電影卻刷新了鹿豹座小編的舊觀念，越來越多出色的類型電影讓人們眼前一亮，大呼過癮。接下來鹿豹座小編就為您整理出2017十大高分韓國電影，快來看看吧。

軍艦島

豆瓣評分：7.6

導演: 柳昇完

主演: 黃晸玟 / 蘇志燮 / 宋仲基 / 李貞賢 / 李璟榮

類型: 劇情 / 動作 / 歷史

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-07-26(韓國)

片長: 132分鐘 / 151分鐘（導演版）

影評：典型的人物、善惡結構，體現了商業電影的典型性。考慮到導演的地位，充其量也就是一般之作。 崔七星和吳末子之間的愛情線、李姜煜和李素姬之間的父女愛，算是照顧到了各個年齡層，而且考慮出演演員宋仲基光復軍所屬的特殊要員樸武英一角的人氣，票房會取得成功的。

之後

豆瓣評分：7.5

導演: 洪尚秀

編劇: 洪尚秀

主演: 金敏喜 / 權海驍 / 趙允熙 / 金璽碧 / 紀柱峰

類型: 劇情 / 愛情

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-05-22(戛納電影節) / 2017-07-06(韓國)

片長: 92分鐘

影評：洪尚秀電影中最具魅力的典型人物：不斷試圖逃避卻又不斷把自己推入淤泥的男性，突降凌辱卻尚能維持無辜本質的局外女孩，跳開時空的限制，記憶與現實以曖昧的方式相互交融，雖然免不了自我重複之嫌，但電影人完全追隨自我當下生活狀態的又何嘗不是一種勇氣和理智。

王者

豆瓣評分：7.4分

導演: 韓在林

編劇: 韓在林

主演: 趙寅成 / 鄭雨盛 / 裴晟祐 / 金雅中 / 柳俊烈

類型: 劇情 / 犯罪

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-01-18(韓國)

片長: 134分鐘 / 157分鐘(加長版)

影評：這是一部劇情連接最多韓國總統的韓影：《王者》的劇情故事表面上並不複雜，但細細咂摸之下，就會發現它的劇本野心很大——其時間軸長達30年（1983-2012），以電視新聞、條幅等多種形式，呈現出了7名、8任韓國總統的真實影像，他們分別是：全斗煥、盧泰愚、金泳三、金大中、盧武鉉、李明博、朴槿惠。

青年警察

豆瓣評分：7.4

導演: 金柱煥

編劇: 金柱煥

主演: 樸敘俊 / 姜河那

類型: 喜劇 / 動作

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-08-09(韓國)

片長: 109分鐘

影評：大陸也應該有這種劇本成熟的青春商業片，認真推敲每一個梗，該笑時笑，該緊張時緊張，該感動時感動——總之，讓觀眾完全融入，記起自己的熱血時光。雙男主就像TVB學警吳卓羲陳鍵鋒的設置，一個莽撞一個謹慎，超互補，各種互動也太有愛。

被操縱的城市

豆瓣評分：7.3分

導演: 樸光鉉

編劇: 樸光鉉

主演: 池昌旭 / 沈恩京 / 吳政世 / 金相浩 / 安在洪

類型: 動作

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-02-09(韓國)

片長: 126分鐘

影評：片中好比遊戲虛擬世界一樣的現實情況和主演們華麗的動作戲搶人眼球。受時長所限，真相似乎反轉得太快，影片反而給人另一種被操縱的影像的感覺，如果拍成電視劇，可能會讓觀眾更從容地審視，審視影像，也審視自己的情緒。

一天

豆瓣評分：7.1

導演: 趙善浩

編劇: 趙善浩

主演: 金明民 / 卞耀漢 / 申惠善 / 趙恩恆 / 劉在明

類型: 劇情 / 懸疑 / 驚悚

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-06-15(韓國)

片長: 90分鐘

影評：用輪迴、投胎、靈魂這些純粹東方的精神符號去解構純西方的時間循環的概念。一天既是循環的一天又是片中死去孩子的名字，而和解之後因罪活下來的兩個孩子一個承載了一天的肉體，一個承載了一天的靈魂，更多的是最後突出那一絲的人性。人生就是一場因果，每天都是現場直播輪迴。結尾就是點睛之筆，編劇實在是太強了！

獨自在夜晚的海邊

豆瓣評分：7.1

導演: 洪尚秀

編劇: 洪尚秀

主演: 金敏喜 / 徐永嬅 / 鄭在詠 / 文成根 / 權海驍

類型: 劇情

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-02-16(柏林電影節) / 2017-03-23(韓國)

片長: 101分鐘

影評：這部新片是一種考驗，也是一次坦率的問詢。大體上存在兩種觀影視角：一是將其帶入洪尚秀私生活，從臺詞中追索出導演心中的蛛絲馬跡；二是由作品本身而非作者角度出發去思考。這兩種立場其實互相牴觸。瞭解八卦的人在觀影之初，很可能會截取其中的個人化片斷，猜想導演在借臺詞為自己發聲。

不汗黨

豆瓣評分：6.9

導演: 卞成賢

編劇: 卞成賢

主演: 薛耿求 / 任時完

類型: 劇情 / 動作 / 犯罪

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-05-17(韓國)

片長: 120分鐘

影評：相比其他同類黑色電影，這部的攝影照明剪輯都好出彩，尤其是照明，光線的調度非常驚豔。演技當然也都在線。而故事方面，雖然反轉無數次但其實還是蠻俗套的，《監獄》加《新世界》，這樣的故事在韓國電影裡真的已經太多了。真心求各位編劇以後不要再重複了。

蚯蚓

豆瓣評分：6.8

導演: 尹學烈

編劇: 金嚴昊 / 尹學律

主演: 金正均 / 吳藝雪

類型: 劇情 / 犯罪

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-04-20(韓國)

片長: 102分鐘

影評：最黑暗的地方如果能被分享出來，那麼即是光明。想起前年的那部「韓公主」，看得我心都要碎了。昨晚睡前看了林奕含事件的新聞，校園暴力和性侵就像龍捲風，而受害者像身處的暴風眼，束手無策、無法逃脫。當這個世界真正變得好起來，從來都不是柔弱者開始變得強大， 而是因為強大者成功地控制了自己。

爸爸是女兒

豆瓣評分：6.7

導演: 金惠榮

編劇: 崔允美 / 金娜裡 / 喬晟佑 / 金吉森

主演: 尹宰文 / 鄭素敏 / 申久 / 李日花 / 李美度

類型: 劇情 / 喜劇 / 奇幻

製片國家/地區: 韓國

語言: 韓語

上映日期: 2017-04-12(韓國)

片長: 115分鐘

影評：個是改編自日本的電視劇《父女七日變》，電視劇沒看過。上次看的韓國翻拍日本版電影是《幸運鑰匙》（《偷鑰匙的人》），看完再看日版，覺得韓版比較好看。片頭樹葉飄下，一下讓人夢迴《阿甘正傳》。靈魂交換的梗其實真很老舊了，男女互換也很常見，不過父女互相印象中是第一次看到。搞笑的情節不多，橋段設計也沒什麼太大亮點，在韓國電影來說，這個片子有點平庸了，想起了《奇怪的她》。
