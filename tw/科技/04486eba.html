5.先看看這個框所在的位置，這個框框竟然在另一個html代碼裡面，也就是說在瀏覽器看的時候，出現了兩個html標籤，老實說，我是第一次看到這種情況的，奈何我的html也不好，連入門都算不上，沒辦法，我就去百度了，果然黃天不負有心人，說是因為iframe這個標籤可以再放html代碼，所以就是這種情況了

  1. 既然知道了是怎麼一回事之後，那就可以繼續操作了，首先我們先找到iframe這個標籤，然後獲取它的src屬性，這個鏈接就是這個框框登陸的鏈接了，如果不獲取這個iframe標籤的src屬性，那麼我們使用selenium是獲取不到這個框框的元素的。
# 點擊之後會彈出一個登陸框，這時候我們用顯示等待來等待這個登陸框加載出來WebDriverWait(browser, 1000).until( EC.presence_of_all_elements_located( (By.CSS_SELECTOR, '#loginWin iframe') ))print('登陸框已加載')# 登陸框加載之後，我們發現整個登陸框其實就是另一個網網頁# 如果在原網頁操作這個登陸框的話，是不能操作的# 所以我們只需要提取iframe標籤的src屬性，然後再去訪問這個url即可實現# 自動登陸# 找到iframe標籤並獲取srciframe_url = browser.find_element_by_css_selector('#loginWin iframe').get_attribute('src')# 再訪問這個urlbrowser.get(iframe_url)# 找到快捷登陸的頭像並點擊# 首先用顯示等待這個頭像已經加載完成WebDriverWait(browser, 1000).until( EC.presence_of_all_elements_located( (By.ID, 'qlogin_list') ))browser.find_element_by_css_selector('#qlogin_list a').click()print('登陸成功')

構建谷歌驅動器 browser = webdriver.Chrome() # 請求url browser.get(url) # 模擬登陸，首先找到登陸的id，並點擊 browser.find_element_by_css_selector('#headerInfo p a').click() # 點擊之後會彈出一個登陸框，這時候我們用顯示等待來等待這個登陸框加載出來 WebDriverWait(browser, 1000).until( EC.presence_of_all_elements_located( (By.CSS_SELECTOR, '#loginWin iframe') ) ) print('登陸框已加載') # 登陸框加載之後，我們發現整個登陸框其實就是另一個網網頁 # 如果在原網頁操作這個登陸框的話，是不能操作的 # 所以我們只需要提取iframe標籤的src屬性，然後再去訪問這個url即可實現 # 自動登陸 # 找到iframe標籤並獲取是如此熟悉 iframe_url = browser.find_element_by_css_selector('#loginWin iframe').get_attribute('src') # 再訪問這個url browser.get(iframe_url) # 找到快捷登陸的頭像並點擊 # 首先用顯示等待這個頭像已經加載完成 WebDriverWait(browser, 1000).until( EC.presence_of_all_elements_located( (By.ID, 'qlogin_list') ) ) browser.find_element_by_css_selector('#qlogin_list a').click() print('登陸成功') return browser# 切換句柄操作def switch_spider(browser): # 登陸成功之後，我們就找到群管理的標籤並點擊,首先等待這個元素加載完成 WebDriverWait(browser, 1000).until( EC.presence_of_all_elements_located( (By.XPATH, './/ul[@id="headerNav"]/li[4]') ) ) browser.find_element_by_xpath('.//ul[@id="headerNav"]/li[4]').click() # browser.switch_to.window這個是指定當前遊標切換到哪個窗口 # 其實也可以這麼寫 # all_window = browser.switch_to.window返回的是一個列表 # browser.switch_to.window(all_window[1]) # 效果是一樣的 return browser# 開始採集數據def start_spider(browser): #
