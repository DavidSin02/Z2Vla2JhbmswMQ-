arraycopy(Object src, int srcPos, Object dest, int destPos, int length);內存數組拷貝方法：參數1是要拷貝的數組對象（源數組）；參數2是從源數組的哪個下標位置開始拷貝；參數3是拷貝到那個數組對象（目標數組）；參數4是從目標數組的哪個位置開始放拷貝的數據；參數5是拷貝長度，拷貝多少個元素。

在運算小數點數值時，若按照一般的加減乘除方式操作會丟失精度且運算結果不正確：

當f1為int類型時，初始值為0，相加100次0.1，結果居然為0，而我們想要的結果是10。

當f1為double類型，初始值為0時，相加100次0.1的結果是9.99999999999998，這顯然不是我們想要的結果，理論結果是10，運算後精度丟失了：

如何避免這種誤差呢？

運算的時候想保留幾位數就乘於幾位數，還原時再除以幾位數就行了。想保留10位就乘於10，還原時再除以10轉換成double類型就不會丟失精度了。

這種計算就是把小數變成int或者是long類型， 計算完畢後可以還原float或者double類型(速度快，不消耗內存)。
