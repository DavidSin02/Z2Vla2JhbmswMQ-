幾個超級無聊的網頁，真的比你還無聊

2020-11-14
| 科技

在小狐上班時沒有思緒的時候，常常會對著電腦屏幕發呆！

那麼在發呆的時候，小狐就會想今天要寫什麼，才能引起你們的注意！後來。。。小狐想到了！就是我在發呆時對著屏幕的這些網頁。。。

想選題的時候

就會對著電腦屏幕，滑著鼠標滾輪，一直滑啊！滑啊！因為小狐知道這個網頁劃不到盡頭。

在滑動鼠標滾輪的過程中，能給予小狐部分解壓和靈感！總覺得手動一下才踏實！其實看似無底洞一般的無聊網頁，是一位小夥伴他為了試驗 CSS 在網頁上到底能有多大，就製作了一個世界上最長的網頁 —— The World Highest Website ！

這個網頁的全部長度，或者更確切的說是網頁上下翻轉的高度，大約是 18.939 公里，通過 CSS 的頁面壓縮功能，讓這個文檔存在於網頁之中，不得不說製作這個網頁的朋友個使用 CSS 很熟練，同時他也應該是世界上最無聊的人了。

而這個網頁，正如我上面所說的！它是有旅程終點的，只要按動鍵盤的 Ctrl + Home 和 Ctrl + End ,就能快速在網頁裡到達置頂和到達終點。

而起點和終點的彩蛋文字還是很勵志的。

吃午飯的時候

而一個上午過去了！小狐還是沒有找到選題，為了避免被狐友們發現小狐一早在划水，於是小狐只能設置一個快速而又有說服力的屏保，再去吃午飯。這時候小狐就用到這個網頁：

裡面有 10 款屏保，種類為 Windows 安裝升級的界面，和 OSX 的界面！而簡單的點開，按 F11 進入全屏和退出全屏即可，超級方便！

還有 B 格滿滿的 OSX ！

效果怎麼樣？是不是很真實！反正小狐就經常掛著 OSX ！應該誰也想不到一個人無聊到經常升級系統吧！

當然！還有殺手鐗 ，在各個系統的升級或者開機界面，按空格可觸發各個系統崩潰的界面，比如上面我們熟悉的新蓋中蓋 “ 藍瓶 ” ！還有 OSX 的也有！

午休的時候

吃完飯午休的時候，小狐會帶著耳機聽一些 Future Bass 的雨！這個時候請戴上耳機食用。

開玩笑開玩笑！其實這是一個聽下雨的網頁播放器！裡面播放的是下雨時的水滴聲和打雷的轟鳴聲！很舒服而又身臨其境的感覺，一定要去試試！午休聽著這個，小狐很容易進入睡眠狀態。恰巧你在上班思考時，也能給予你不錯的靈感。

繼續想選題的時候

怎麼辦？還沒想到！玩個音樂小遊戲解解壓放輕鬆就很容易想到了吧！

一款非常二次元的網站，界面很簡單！按下 START 直接可以進入遊戲界面開始， F11 全屏。

別看界面很炫酷！其實裡面的音樂更炫酷，按下畫面不同的區域有不同的人聲聲音，無意中就成了一首曲！這個一定要戴耳機食用呀。

快下班的時候

已經下班了！該怎麼辦？那小狐就以這個划水篇直接提交吧！
