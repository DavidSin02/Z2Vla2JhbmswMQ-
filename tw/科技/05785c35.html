學會這些命令行快捷鍵你將健步如飛

2020-11-14
| 科技

很多同學在Linux下工作幾年了，還是不清楚Linux Shell的快捷鍵。還在不斷的用“左右上下”方向鍵來操作命令行，今天本號就介紹一下命令行常見的快捷鍵，通過這些快捷鍵可以極大的提高使用命令行的效率。

正行移動

通過這組快捷鍵可以快速的將光標移動到行首或者行尾，對於一個很長的命令行，但只需要在行首或者行尾做很小改動的情況非常有用。

Ctrl + a ：移到命令行首

Ctrl + e ：移到命令行尾

Ctrl + xx：在命令行首和光標之間移動，也就是如果在命令行的某個非行首位置按該快捷鍵時光標將跳到行首，再次按該快捷鍵將跳回到原來光標的位置。

單詞移動

以整個單詞為單位進行光標的移動，移動後光標將處於單詞的首部。需要注意的是由於xshell等工具Alt+b也有快捷鍵，因此在xshell下回衝突。

Alt + f ：按單詞前移（右向）

Alt + b ：按單詞後移（左向）

字符移動

可以以字符為單位左右移動，其作用和方向鍵一樣，但好處是你的手不需要離開主鍵盤太遠。

Ctrl + f ：按字符前移（右向）

Ctrl + b ：按字符後移（左向）

行級刪除

可以實現跨字符的清理，將光標移動到某個位置之後，可以將其前或者其後的所有字符刪除。

Ctrl + u ：從光標處刪除至命令行首

Ctrl + k ：從光標處刪除至命令行尾

單詞級刪除

可以實現單詞級的刪除，將光標移動到單詞的某個位置之後，可以將光標前或者後的屬於該單詞的字符一次性刪除。避免使用退格鍵多次低效的方式。

Ctrl + w ：從光標處刪除至字首

Alt + d ：從光標處刪除至字尾

字符級刪除

可以快速刪除字符，避免使用較遠的退格鍵和delete鍵。

Ctrl + d ：刪除光標處的字符

Ctrl + h ：刪除光標前的字符

內容調整

可以快速修改命令行中的某些字符，對於命令中錯誤內容的修改非常有效。

Ctrl + y ：粘貼至光標後

Alt + c ：從光標處更改為首字母大寫的單詞

Alt + u ：從光標處更改為全部大寫的單詞

Alt + l ：從光標處更改為全部小寫的單詞

Ctrl + t ：交換光標處和之前的字符

Alt + t ：交換光標處和之前的單詞

Alt + Backspace：與 Ctrl + w 相同類似，分隔符有些差別

歷史命令

有的時候我們需要經常使用之前已經使用過的命令。

Ctrl + r：逆向搜索命令歷史，可以通過關鍵字符匹配最近的命令

Ctrl + g：從歷史搜索模式退出

Ctrl + p：歷史中的上一條命令

Ctrl + n：歷史中的下一條命令

Alt + .：使用上一條命令的最後一個參數
