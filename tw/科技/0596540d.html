新人也能輕鬆用的播客創作工具 Ultraschall

2020-11-14
| 科技

在收聽播客愈發流行的當下，創作播客也成為了一種越來越普遍的表達方式。但是在今天，播客的製作仍然面臨著一個有些尷尬的局面——現有的用於播客製作的軟件工具通常來自一些傳統的專業行業，例如音樂製作、音效製作、廣播電臺、電影工業，乃至企業音頻會議方案等，這些工具雖然功能全面，但是大多數並非針對個人用戶開發和銷售，因此普遍存在價格昂貴、學習曲線陡峭等問題。許多播客創作者可能用著盜版的昂貴的工業級數字音頻工作站（Digital Audio Workstation，簡稱 DAW），但實際上僅僅用到了一些簡單的剪輯功能，對於更加進階的聲音處理，可能既不瞭解，也不知道從何入手。

我作為兩檔獨立播客（告別攝影 和 洋涇浜）製作的參與者，在創作的過程中也踩過各種各樣的坑，幾乎嘗試過市面上所有的播客相關的軟件工具。最近，我試圖為我的播客添加章節標記，發現這個近年較為流行的功能卻嚴重缺少相關工具的支持，雖然章節標記的 ID3 標準是開源的，也已經推出了許多年，但是現在市面上幾乎所有的 DAW 都仍不支持此功能，只有 Podcast Chapters 和 Forecast 兩個成型的獨立軟件可用，而前者雖然功能完善，但 $19.99 的售價對於許多獨立播客製作者來說並不便宜，後者雖然目前免費，但是仍然處於 bug 較多的 beta 狀態，並且只支持導入 .wav 格式，導出幾種固定碼率的 .mp3 文件，對於希望對音質有更精確把控的創作者來說並不算友好。在這個搜尋的過程中，我發現了 Ultraschall。

Ultraschall 是什麼？

Ultraschall 是一個專門為播客創作而定製的 Reaper 插件。Reaper 是我目前製作播客的主力 DAW 軟件，它功能非常全面，非常輕量且價格低廉，並且提供事實上無限期的免費試用，應該是目前最適合個人音頻創作者的工具之一，Nick 在少數派寫的《播客製作入門指南》中對它也做過簡短的介紹。

但為什麼我覺得一個插件值得單獨來介紹？Ultraschall 不只是一個普通的插件，它對 Reaper 進行了深度的定製，完全改造了 Reaper 原有的界面和操作，增加了一些對於播客創作者來說非常實用的小工具，同時又保留了 Reaper 原有的強大的可定製性和可擴展性。可以說，安裝了 Ultraschall 之後的 Reaper 已經完全變成了另一個專門針對播客創作者的音頻工作站了。

很有趣的一點是，Ultraschall 是一個完全由德國的播客愛好者開發和維護的插件，在他的早期版本中甚至只提供德文的 changelog，並且至今也不提供官方的英文文檔。這導致它即使在英文世界中的知名度也並不高。但是不用擔心，Ultraschall 的軟件界面是 100% 英文的，而且操作也很簡單，只要是用過常見 DAW 的人，上手都不會有什麼障礙。Ultraschall 是完全免費和非盈利的，但製作之精良、功能之豐富讓我對德國的播客生態產生了一些好奇。

安裝與配置

雖然官方網站只有德文，但 Ultraschall 的安裝包裡提供了完整的英文安裝說明，簡單概括如下：

  • 安裝特定版本的 Reaper。由於定製程度非常深，所以 Ultraschall 推薦安裝特定版本的 Reaper 而不是當前的最新版作為軟件主體，參考安裝包中的目前最新的版本對應的是 Reaper 5.70，可以在 Reaper 官網的 歷史版本下載頁面 找到對應的 Windows 或者 Mac 版。
    • 如果你之前就已經安裝並且在使用 Reaper，建議對當前版本進行備份或將之 轉換為便攜版，因為 Ultaschall 很可能會跟既有的插件起衝突。好在 Reaper 只要是便攜安裝就可以支持多個版本同時存在，所以這個過程並不複雜。
  • 安裝 Ultraschall。最新的安裝包可以在 官網 獲得。
  • 安裝程序運行完畢之後，啟動 Reaper，這時候你會看到一個主題文件缺失的警告，沒有關係，點下一步進入程序主界面。
  • 進入主界面之後，將安裝包裡的 Ultraschall_3.1.ReaperConfigZip 拖入主界面。Reaper 會彈出提示詢問是否要導入配置，一路選是即可。
  • 到這一步安裝已經完成了，可以重新啟動軟件，如果沒有彈出警告並且出現下方的畫面就說明安裝成功了。

界面與基本邏輯——為播客而生的 DAW 應該是什麼樣？

Ultraschall 在 Reaper 基礎上，為了播客製作增加了許多貼心的設定，對於與常規 DAW 相同的剪輯功能，本文不再贅述，主要挑選幾點對於播客製作較為實用的進行介紹。

基本界面構成

貼心的菜單設計

Untraschall 的菜單欄

Reaper 原本的界面中附帶了非常複雜的按鈕設置，Ultraschall 則只留下了一組對於播客特別實用的，並且配上了定製的圖標。圖中的按鈕從上至下，從左至右依次是：

  • 縮放至顯示完整的項目。
  • 開啟/關閉音軌窗口跟隨播放進度滾動。
  • 開啟/關閉單音軌 ripple edit1 。
  • 開啟/關閉所有音軌 ripple edit。
  • 插入章節標記。
  • 插入章節標記並同時編輯內容。
  • 在光標處分割音頻。
  • 切換鼠標左鍵拖動行為：選取/移動。
  • 以 ripple edit 的方式刪去所選的內容。
  • 在當前音軌啟用靜音包絡線2 。

如果你曾經進行過播客剪輯，你會感受到這個按鈕的佈局一定是懂行的人做的，不多也不少。

強大而實用的工具

在 Reaper 原本的功能之外，Ultraschall 自己添加了一些針對播客製作的小功能，這些功能在其它的 DAW 軟件中通常都難以實現，或是需要第三方的插件。例如：

響度分析

響度分析窗口

在剪輯視圖中，在底部調音臺窗口的旁邊有一個 Loudness 的選項，可以方便的對項目中任意一個音軌（或是片段）進行響度分析，得到它的平均響度和動態範圍等。這對與多音軌錄音的播客剪輯有著極大的幫助，再也不用靠耳朵聽來估計不同嘉賓的音量大小了。在傳統 DAW 中，這個功能通常需要用第三方插件來實現，專業的音頻分析插件非常昂貴（例如售價 $199 的 iZotope Insight 2），而且其中大多數的功能可能播客創作者完全用不上，但這個小工具卻把這個問題一鍵解決了。

Project Bay

Project Bay 窗口

在審閱（故事板）視圖中，界面的右側會出現一個 Project Bay 窗口，可以在其中看到項目中所用到的所有源文件和所有音頻片段的列表。配合 Reaper 自帶的 Media Explorer 功能，讓錄音文件和音效的管理變得格外便捷。如果你在 DAW 之外對某條音軌進行了單獨的降噪和後期處理，你可以在這個窗口中方便地替換它。如果你使用了一些重複出現的音效，也可以直接從這個窗口中通過拖拽來將它放到音軌中。

Ultraschall Dynamics 插件

Ultraschall Dynamics 插件界面

Ultraschall 自己開發了一個叫作 Ultraschall Dynamics 的音頻插件。雖然看起來不起眼，但它同時提供了播客常用的三大音頻處理：限制器（limiter）、動態壓縮（dynamic compression），和噪聲門（noise gate）。這個插件只有三個參數，Target 規定了音軌的目標響度，Noisefloor 規定了噪音範圍的上限，Noisegate 則規定了噪音範圍的下限。簡而言之，它可以做到：音量高於 Noisefloor 的聲音會被壓縮至接近 Target，而低於 Noisefloor 的聲音會被壓低，低於 Noisegate 的聲音則會靜音。具體的曲線則是自動計算的。這個插件提供了一個非常簡潔但實用的音頻處理操作，對於錄音質量尚可的音頻，你可能只需要用它就可以完成後期處理的部分了。

導出章節標記一鍵完成

Ultraschall 最直擊我的痛點的功能當屬它完善的導出系統。作為一個 DAW 軟件，Reaper 自帶的渲染菜單已經是非常強大了，支持幾乎所有常見的音頻格式，並且支持分段、分音軌導出。然而，正如開頭所說，播客製作中的一些特有的需求，例如添加節目的單集封面和章節標記，目前仍然需要求助於第三方軟件來完成。而 Ultraschall 的導出功能將這些需求集成到了 Reaper 中，成為了目前唯二可以直接在 DAW 中實現章節標記的方案之一3 。

導出窗口

  • 點擊主界面左下角的 Export Assistant 就可以呼出播客導出窗口，只要按照他提供的幾個步驟，你就可以得到一個包含了完整的章節、封面、標題、作者名等元數據（metadata）的 .mp3 或 .m4a 文件。
  • 第一步是調用 Reaper 原生的渲染功能，你可以選擇你想要的碼率和格式，將節目導出成 .mp3/.m4a 文件（目前這兩種格式是唯二支持播客章節功能的音頻格式）。
  • Ultraschall 使用 Reaper 原生的 marker 功能來生成播客章節標記，因此如果你在剪輯時已經通過 marker 標記了章節，第二步就已經完成了，你可以點擊第二步中的 View Chapters 來查看章節列表4 。
  • 第三步是編輯節目的元數據，你可以在這裡填入節目的標題和作者。
  • 第四步是插入單集封面，你只需要將封面的圖片重命名為 cover.jpg(/jpeg/png)，再將其放到工程文件所在的文件夾中即可。
  • 在這些步驟都完成之後，點擊 Finalize MP3/m4a，選擇你在第一步中所導出的音頻文件，這些元數據就會被寫入音頻文件中。

總結

總而言之，對於播客創作者來說，Ultraschall 是一個難得的強大且趁手的工具，但由於語言的關係，成為了常見的推薦和選擇中的遺珠。如果你已經在使用 Reaper 製作播客，或者正在尋找一個理想的工具，我強烈推薦你嘗試 Ultraschall。它可以讓新手更快更好地著手播客創作，對於老手也是有許多功能上的驚喜，Reaper 本體 $60 的售價加上無限期試用也比大多數的 DAW 軟件要便宜。

Ultraschall 的官網上對自己的定位是 High-end Podcasting，我認為它的體驗可以擔得上這個名號。
