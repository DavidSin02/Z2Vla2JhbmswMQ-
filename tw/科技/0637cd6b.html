爬取B站視頻排名第一《祖國大好河山》1W+彈幕，得出一份詞頻詞雲

2020-10-29
| 科技

前言

2019 年 10 月 1 日，是一個值得被紀念的日子。轉眼間，10年已經過去了。上一次國慶歷歷在目，當年的我還參加了國慶 60 週年的翻花表演。

平時熱衷於 B 站的我，掃到了排行榜排名第一的視頻，數百位攝影師聯合制作，《10分鐘帶你看絕美祖國大好河山！》



感興趣的，可以複製下面地址看下：

https://www.bilibili.com/video/av69241910/​www.bilibili.com

可以看到上圖右側，有數千條彈幕。通過查看歷史記錄，發現是 9.28 才有的第一條彈幕，目測應該是 9.28 上傳的視頻。出於感興趣，於是打算來個詞頻統計。

文章寫到這裡，筆者並沒有開始代碼的編寫，也不知道彈幕哪個詞出現的頻率最高。（平時寫文章有個習慣，都是先寫前言，搭好文章架子再開始寫具體的代碼實現。）

在結果出現之前，來個盲猜吧！祖國，快樂。這兩個沒猜錯，應該是最高的頻率詞！

框架性思維

不知道大家還記不記得爬蟲的框架步驟。

首先，熟悉網站結構，找出我們想要的數據接口。

其次，分析提取數據。

最後，完成本次的詞頻統計。

所以，讓我們先來找接口吧！B 站的接口很好找喲！

B站彈幕接口

打開上述提到的視頻鏈接。

提供一個思路，打開 F12 後，一般默認先去找 XHR 裡是否有相似數據。因為 XHR 這欄，是異步請求。意味著，即使不刷新頁面，也可以像服務器異步發送請求，不會影響用戶操作。



總的來說，下面的異步請求並不多，一個個去看，也能全部瀏覽完。不過有個小技巧，就是根據名字挑著看，比如那些常見的單詞，list。



F12看數據，可以看到不是 json 的形式返回的，用瀏覽器訪問下試試。




它是以標籤的形式，將數據展示的。而這種形式，恰是之前在小課堂中介紹到的 XML 格式。

其中有個 maxlimit 的標籤，數字顯示 1500 。猜測應該是此頁數據最大 1500 條數據。複製到文本工具裡，刪除非數據以外的頭尾，驗證下，一共 1502 行。



由此得知，這並不是歷史數據。繼續來挖，點擊查看歷史，選擇9.28日，F12再次看請求。

當然，你可以先清除下之前的F12記錄，以免太亂，點擊F12的小垃圾桶即可。我這裡用的是 Safari 瀏覽器，谷歌瀏覽器類似操作。

出現了 history 的名字，無疑就是它了！明顯，請求地址發生了改變。

https://api.bilibili.com/x/v2/dm/history?type=1&oid=120004475&date=2019-09-28

這樣一來，就有了這幾天的歷史數據。模擬請求，修改下 url 中 date 字段的日期。接下來只需要將返回的 XML 進行數據提取就好啦！

簡單的說下代碼思路。

用 requests 庫發起模擬請求，請求頭需要攜帶自己的 cookie ，否則會提示請登錄的字眼。

爬取彈幕後，存儲到一個公有 list 中，通過 jieba 分詞的第三方庫，對其進行詞頻統計分析。

其中 jieba 使用的核心代碼是這段：

https://github.com/fxsjy/jieba​github.com



最後，使用 wordcloud 生成詞雲圖核心代碼：

def __draw_word(self):    """ 詞雲生成 """    d = os.path.dirname(__file__)    alice_mask = np.array(Image.open(os.path.join(d, "gq.jpg")))    windows_font_path = 'C:/Windows/Fonts/simsun.ttc'  # winodws字體    mac_font_path = '/System/Library/Fonts/PingFang.ttc'  # mac字體    wc = WordCloud(font_path=mac_font_path,  # 設置字體格式,系統自帶的中文字體                   mask=alice_mask,  # 設置背景圖                   background_color='white',                   max_words=400,  # 最多顯示詞數                   max_font_size=150  # 字體最大值)                   )    wc.generate_from_frequencies(self.word_dict)  # 從字典生成詞雲    image_colors = wordcloud.ImageColorGenerator(alice_mask)  # 從背景圖建立顏色方案    wc.recolor(color_func=image_colors)  # 將詞雲顏色設置為背景圖方案    plt.imshow(wc, interpolation='bilinear')  # 顯示詞雲    plt.axis('off')  # 關閉座標軸    plt.show()  # 顯示圖像
