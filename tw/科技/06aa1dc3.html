初識C++輸入輸出

2020-10-29
| 科技

C++沒有直接定義進行輸入輸出的任何語句，這種功能是由標準庫提供的。

本文的例子使用了處理格式化輸入輸出的iostream庫，iostream的基礎是istream 和ostream，分別是輸入流和輸出流。流是從某種IO設備上讀取或寫入的字符序列。

標準庫定義了4個IO對象，其中istream類型的對象有一個，命名為cin，這個對象被稱為標準輸入。ostream類型對象有三個，分別是cout、cerr、clog，cout是標準輸出，cerr是標準錯誤，clog用於產生程序執行的日誌。

一般情況下，從cin讀入時，數據從執行程序的窗口讀入，寫入到cout、cerr、clog時，會寫入同一窗口。運行程序時，Linux提供了重定向輸入輸出流的方法，可以通過重定向將這些流與所選擇的文件聯繫起來。

現在來看一個例子，從標準輸入讀入兩個數，然後輸出它們的和。

#include <iostream>

int main () {

std::cout << "Enter two numbers:" << std::endl;

int x, y;

std::cin >> x >> y;

std::cout << x << "+" << y << "=" << x+y << std::endl;

return 0;

}

程序先在用戶屏幕輸出：

Enter two numbers:

然後等待用戶輸入。如果用戶輸入

1 2

加一個換行符，程序輸出：

1+2=3

程序第一行是預處理指令

#include <iostream>

告訴編譯器要使用iostream庫。尖括號裡的名字是頭文件。程序使用庫時必須包含相關的頭文件。頭文件名必須和#include處於同一行。

main函數中第一條語句執行了一個表達式。C++中表達式由操作數和操作符組成。

std::cout << "Enter two numbers:" << std::endl;

這個語句用了兩次輸出操作符，每個操作符都接受兩個操作數，左操作數必須是ostream對象，在該句中為cout，右操作數是要輸出的值。操作符將其右操作數寫入左操作數的ostream對象。

C++每個表達式都會產生一個結果，std::cout << "Enter two numbers:" 的結果是std::cout，所以

std::cout << "Enter two numbers:" << std::endl;

等價於

std::cout << "Enter two numbers:";

std::cout << std::endl;

endl是個特殊值，將它寫入輸入流時，會換行，並刷新與設備相關聯的緩衝區，使用戶立即看到寫入到流中的輸出。

std是標準庫名字，std::表明cout 和endl是定義在namespace std 中的，使用命名空間，可以避免無意中使用了與庫中所定義的名字相同的名字而引發衝突。

int x, y;

定義了兩個未初始化的int類型變量。

這兩個變量在首次使用時會讀入一個值，故可以沒有初始值。

下一條語句讀取輸入

std::cin >> x >> y;

輸入操作符行為與輸出操作符類似。它從istream 操作數中讀取數據，保存到右操作數中。該語句等價於

std::cin >> x;

std::cin >> y;

效果是從標準輸入讀取兩個值，將第一個存入x，第二個存入y。

std::cout << x << "+" << y << "=" << x+y << std::endl;

用於輸出結果，表達式x+y是對x y的值相加，返回的是相加之後的和。
