CSS基礎選擇器與合併選擇器解析

2020-10-29
| 科技

本文創建於2020年9月，以下為正文：

1.通用選擇器，全局選擇器（*）

通用選擇器對整個頁面中所有HTML標籤樣式定義，這種功能類似“標籤選擇器”，覆蓋的對象更加廣泛，是整個HTML的所有標籤，功能是強大，但是這樣反而限制了它的靈活性。

通用選擇器可以與任意元素匹配，常用於設置一些默認樣式，但是它的優先級是最低的。

如使用代碼使HTML中任意標籤元素文本顏色全部設置為紅色：

*{color：red;}

2.元素選擇器

元素選擇器是最常見的CSS選擇器

HTML文檔中的元素 p、h1、b、div、a、img、body等甚至可以是html本身

如：html{color:red;}

h1{color:black;}

h2{color:biue;}

3.類選擇器

類選擇器以一個點號顯示。該選擇器可以單獨使用，也可以與其它元素結合使用。

.className{ }

如：.box{color:red;}

注意：類名不能以數字開頭，在一個頁面中class名字可以重複

如果只希望段落文本顯示為紅色則代碼為：

p.important{color:red;}

4.ID選擇器

ID選擇器類似於類選擇器，不過也有差別。ID選擇器前面有一個#符號

如：#idName{ } #box{color:red;}

一個頁面中id相同的id名只能出現一次

一般重複使用的樣式不使用ID選擇器，因為ID屬性是唯一的

類選擇器與ID選擇器相同之處與不同之處在哪兒呢？

相同之處在於：都可以應用於任何元素

區別在於：

1.ID選擇器只能在文檔中使用一次。與類選擇器不同，在一個html文檔中，ID選擇器只能使用一次，而且僅一次。而類選擇器可以使用多次

2.可以使用類選擇器詞列表方法為一個元素同時設置多個樣式。我們可以為一個元素同時設多個樣式，但只可以用類選擇器的方法實現，ID選擇器是不可以的（不能使用ID詞列表）。

5.合併選擇器

語法為：選擇器1，選擇器2.....{ }

作用是可以提取共同的樣式，減少重複代碼

例如：.header ,.footer{height:300px;} 中間由逗號隔開

代碼示例：

< head lang="en">

< meta charset="UTF-8">

< title>合併選擇器< /title>

< style>

p,div,h1,.xiang,#buxiang{

color:green;

}

< /style>

< body>

< p>果汁< /p>

< p>咖啡< /p>

< div>西紅柿< /div>

< div>胡蘿蔔< /div>

< div>大白菜< /div>

< h1>米飯< /h1>

< h1>麵條< /h1>

< i class="xiang"> 我也想變顏色< /i>

< i> 我不想變顏色< /i>

< i id="buxiang"> 我不想變顏色< /i>

< /body>

< /html>
