C語言的自定義輸入和輸出的三種方法

2020-10-29
| 科技

函數不一定非要使用C庫中的標準函數，如果無法使用這些函數或者不想用它們，完全可以在getchar()和putchar()的基礎上自定義所需的函數。假設你需要一個類似puts()但是不會自動添加換行符的函數。程序 put1() 給出了一個這樣的函數。

/* put1.c -- prints a string  without adding n */#include <stdio.h>void put1(const char * string) /* string not altered */{    while (*string != '0')        putchar(*string++);}

指向char的指針string最初指向傳入參數的首元素。因為該函數不會改變傳入的字符串，所以形參使用了const限定符。打印了首元素的內容後，指針遞增1，指向下一個元素。while循環重複這一過程，直到指針指向包含空字符的元素。記住，因為要從右向左運算，因此putchar(*string++)打印string指向的值，遞增的是string本身，而不是遞增它所指向的字符。

可以把put1.c程序作為編寫字符串處理函數的模型。因為每個字符串都以空字符結尾，所以不用給函數傳遞字符串的大小。函數依次處理每個字符，直至遇到空字符。

用數組表示法編寫這個函數稍微複雜些：

int i = 0;while (string[i]!= '0')        putchar(string[i++]);

要為數組索引創建一個額外的變量。許多C程序員會在while循環中使用下面的測試條件：

while (*string)

當string指向空字符時，*string的值是0，即測試條件為假，while循環結束。這種方法比上面兩種方法簡潔。但是，如果不熟悉C語言，可能覺察不出來。這種處理方法很普遍，作為C程序員應該熟悉這種寫法。

＊注意＊

為什麼程序put1()中的形式參數是const char * string，而不是const charstring[]？從技術方面看，兩者等價且都有效。使用帶方括號的寫法是為了提醒用戶：該函數處理的是數組。然而，如果要處理字符串，實際參數可以是數組名、用雙引號括起來的字符串，或聲明為char *類型的變量。用const char * string可以提醒用戶：實際參數不一定是數組。

假設要設計一個類似puts()的函數，而且該函數還給出待打印字符的個數。如程序 put2() 所示，添加一個功能很簡單。

/* put2.c -- prints a string and counts characters */#include <stdio.h>int put2(const char * string){    int count = 0;    while (*string)         /* common idiom       */    {        putchar(*string++);        count++;    }    putchar('n');          /* newline not counted */    return(count);}

下面的函數調用將打印字符串pizza：

put1("pizza");

下面的調用將返回統計的字符數，並將其賦給num（該例中，num的值是5）：

num = put2("pizza");

程序put_put.c使用一個簡單的驅動程序測試put1()和put2()，並演示了嵌套函數的調用。

//put_put.c -- user-defined output functions#include <stdio.h>void put1(const char *);int put2(const char *);int main(void){    put1("If I'd as much money");    put1(" as I could spend,n");    printf("I count %d characters.n",           put2("I never would cry old chairs to mend."));    return 0;}void put1(const char * string){    while (*string)  /* same as *string != '0' */        putchar(*string++);}int put2(const char * string){    int count = 0;    while (*string)    {        putchar(*string++);        count++;    }    putchar('n');    return(count);}

程序中使用printf()打印put2()的值，但是為了獲得put2()的返回值，計算機必須先執行put2()，因此在打印字符數之前先打印了傳遞給該函數的字符串。下面是該程序的輸出：

If I'd as much money as I could spend,

I never would cry old chairs to mend.

I count 37 characters.
