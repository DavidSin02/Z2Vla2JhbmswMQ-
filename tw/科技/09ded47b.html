ES6 字符串的擴展方法

2020-11-14
| 科技

基於ES6標準入門（第3版）這本書的筆記要學習angula，vue，react等新的前端框架，必須先熟悉ES6的語法

字符串Unicode表示方法

JavaScript允許採用\uxxx的形式表示一個字符，其中xxx表示字符的Unicode碼點。但是，這種表示方法只限於\u0000~\uFFF之間的字符。超出這個範圍的字符，必須用2個雙字節的形式表達。ES6對這一點做了改進，只要將碼點放入大括號，就能正確解讀該字符串。其實本質上還是在做ES6代碼編譯的時候轉換成了雙字節。

ES6代碼

let codeMsg = '\u{20BB7}';console.log(codeMsg);

編譯後的ES5代碼

var codeMsg = '\uD842\uDFB7';console.log(codeMsg);

字符串遍歷接口

ES6為字符串添加了便利接口，可以使字符串直接用for...of 循環

{    // 字符串循環遍歷    let str = 'forOfLoop';    for(let temp of str){        console.log(temp);    }    // for of 循環除了可以用於字符串循環，也可以循環map集合    var map = new Map();    map.put('k1','v1');    map.put('k2','v2');    map.put('k3','v3');    for(let [key,value] of map){        console.log(key);        console.log(value)    }}

at()方法

ES5對字符串對象提供了chartAt()方法，返回字符串給定位置的字符，但是該方法不能識別碼點大於0xFFFF的字符串。ES6提供的at()方法可以識別。

let str = 'forOfLoop';console.log(str.charAt(1));console.log(str.at(1));

這個方法需要通過墊片庫（github.com/es-shims/String.prototype.at）實現。

includes(),startsWith(),endsWith() 方法

ES5中只有indexOf方法來確認字符串是否包含在另一個字符串中，ES6提供了3個擴展方法

  • includes()：判斷是否包含
  • startsWith()：判斷是否已某一個字符串開頭
  • endsWith()：判斷是否已某一個字符串結尾
{    let str ='hello word';    console.log(str.includes('word'));  // true    console.log(str.startsWith('hell')); // true    console.log(str.endsWith('rd')); // true}

上面的方法還支持位數搜索，第二個參數n表示搜索的位數，includes和startsWith第2個參數n，表示從第n位到結尾的區間搜索，endsWith第二個參數n表示前n個字符區間搜索。

{    let str ='hello word';    console.log(str.includes('word',3));      console.log(str.startsWith('hell',4));     console.log(str.endsWith('rd',5)); }

repeat()方法

repeat方法返回一個新字符串，表示把原字符串重複n次

{    let str ='hello word';    console.log(str.repeat(2));}

padStart(),padEnd()方法

padStart()方法用於字符串補全頭部,padEnd()用於字符串補全尾部，如果原字符串長度大於等於指定的長度大小，則返回原字符串

{    let str ='hello word';    console.log(str.padStart(20,'字符串長度不足20位開頭補位字符串'));    console.log(str.padEnd(20,'字符串長度不足20位開頭補位字符串'));}

模版字符串

在ES5中我們做字符串拼接需要一段一段的用+號拼接單引號框起來的字符串，在ES6中可以直接用一個反引號（ ` ）,在反引號區域中，可以寫單行字符串，多行字符串，嵌套變量，調用函數，同時會保留格式，例如換行。

{    var str1 = `這是一個普通的單行字符串`;    console.log(str1);    var str2 = `這是第一行    這是第二行    `;    console.log(str2);    var msg = 'word';    var str3 = `hello ${msg}`;    console.log(str3)}


在模版中嵌入變量需要寫在${}中，嵌入的變量也可以是表達式，也可以是function方法。

{    let obj = {x : 1, y : 2};    var add = `${obj.x + obj.y}`;    console.log(add);    function hello() {        return 'hello word';    }    var meg = `say ${hello()}`;    console.log(meg);}

標籤模版

模版字符串還可以跟在一個函數名後面跟一個模版，就可以直接調用方法

用標籤模版方法的時候，如果有多個參數，第一個參數表示模版中未被替換的的值

var a = 5;var b = 10;function add(s,v1,v2){   console.log(s.toString());   console.log(v1);   console.log(v2);}add`hello ${a} ${b}`;


如果只有一個參數，從控制檯輸出可以看出，我們直接輸出第一個參數，也會是顯示的未被變量替換的值，哪怕我們沒有變量佔位，這個時候就需要用的函數的內置變量arguments，在函數體內可以通過arguments 對象來訪問參數數組，從而獲取傳遞給函數的每一個參數。

var msg ='hello word';function showMsg(msg){   console.log(msg);   // 在函數體內可以通過arguments 對象來訪問參數數組，從而獲取傳遞給函數的每一個參數   console.log(arguments.length);   console.log(arguments[0]);   console.log(arguments[1]);}showMsg`${msg}`;
