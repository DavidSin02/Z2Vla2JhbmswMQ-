程序員實戰調試阿里雲語音合成，自媒體語音合成軟件背後的技術

2020-10-29
| 科技

開篇

最近發現很多做自媒體的都是通過語音合成技術，用文字生成語音的。於是就好奇的研究了一下，發現這些軟件大都是調用的雲服務，例如騰訊雲或者阿里雲的語音合成服務。那麼今天就來調試一下阿里雲的語音合成接口。覺得有用，收藏轉發關注一下哦。如果在開發文檔找不到DEMO，請移步到文章末尾，有獲取方法。

前期準備

自己去在阿里雲上搜索語音合成功能，然後開通，步驟，這裡就不贅述了，自己去阿里雲官網看吧。


調試過程實戰

在阿里雲開發文檔中找到語音合成相關文檔，根據自己擅長的語言選擇合適的sdk進行開發，我這裡採用Java語言來調試。

1、如下圖找到Java SDK相關

從開發文檔中看到核心的jar包是如下面的SDK<dependency>    <groupId>com.alibaba.nls</groupId>    <artifactId>nls-sdk-tts</artifactId>    <version>2.1.6</version></dependency>

2、此時可以將官網的DEMO直接下載下來，主要看example，這裡的代碼可以直接拷貝到你的工程裡面，也可以直接調試使用。

A、300字符以內的語音合成，為：SpeechSynthesizerDemo

B、300字符以上的語音合成，為：SpeechSynthesizerLongTextDemo

C、多線程語音合成，為：SpeechSynthesizerMultiThreadDemo

代碼過於長了，這裡就不貼代碼了。

3、調試的時候在各個demo裡面修改你自己accessKeyId的和你在阿里雲申請的相關accessKeySecret值。然後測試即可。

 AccessToken tokenA = new AccessToken("****accessKeyId", "******accessKeySecret");        tokenA.apply();        String token = tokenA.getToken();        long expireTime = tokenA.getExpireTime();        System.out.println(expireTime);        String appKey = "你建立的項目的key值";

注意

語音合成demo裡面的"****accessKeyId", "******accessKeySecret" 和appKey 是關鍵，自己在阿里雲後臺申請建立設置。這裡不在累述。

結語：

語音合成作為自媒體語音合成軟件背後的技術，是很多程序員都需要的技術。如果你找不到DEMO,關注轉發一下哦，私信"20200807"即可。
