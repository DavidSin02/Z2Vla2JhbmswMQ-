Typora-編寫博客格式化文檔的最佳軟件

Typora 不僅是一款支持實時預覽的 Markdown 文本編輯器，而且還支持數學公式、代碼塊、思維導圖等功能。它有 OS X、Windows、Linux 三個平臺的版本，是完全免費的。

作為技術人員或者專業人員，使用Markdown語法進行寫作，可以提高文檔的編寫能力，而且格式也比較統一，支持編輯Markdown的軟件也比較多，任何電腦都很容易查看或者編輯。

偏好設置

通過打開文件-偏好設置。Typora 中提供了大量有關 Markdown 的設置功能，你可以設置幾乎完全適合自己的 Markdown 編輯器。下面我將介紹一些常用的功能亮點。

圖片插入

Typora 的圖片插入功能是廣受好評的，你可以在Typora中直接使用右鍵複製粘貼將網絡圖片、剪貼板圖片複製到文檔中，或者拖動本地圖片到文檔中，Typora 會自動幫你插入符合 Markdown 語法的圖片語句，並給它加上標題。

複製圖片

而且也完全可以使用圖床來保證文檔在複製分享後圖片仍能正常顯示。

Typora 支持在拖動或複製粘貼網絡圖片後自動將其保存到本地。你可以在"文件-偏好設置-圖像"中選擇複製到哪個路徑，什麼情況下需要複製。

網絡圖片

這一功能保證了即使網絡圖片源失效了，你還有本地的備份可用。同時也能使你的文件夾更合理、完整。

實時預覽

Typora提供實時預覽功能，編寫文字時會按照格式實時預覽樣式，而且Typora 保留了「源代碼模式」。你可以通過視圖-源代碼模式或左下角的</>按鈕進入。

側邊欄

Typora 會根據你 Markdown 標記的 H1、H2、H3…… 各級標題為你呈現大綱，類似導航。

空格與換行

空格：在輸入連續的空格後，Typora 會在編輯器視圖裡為你保留這些空格，但當你打印或導出時，這些空格會被省略成一個。可以在源代碼模式下，為每個空格前加一個\轉義符，或者直接使用HTML風格的&nbps來保持連續的空格。

智能標點

智能標點是比較喜歡的功能。它可以自動幫你將特定引號轉化為自己更喜歡的樣式的引號。 這樣就避免使用不同輸入法或者不同文檔引起的格式不一致的問題。
