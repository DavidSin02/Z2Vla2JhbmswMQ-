10kV 非晶合金變壓器提高抗短路能力的方法

蔣紅婭，邸俊俊

( 河南省森電電力股份有限公司 )


摘 要：隨著電力事業的發展及城農電網的建設與改造的不斷深入，廣大電力用戶對節能型變壓器的要求越來越高。為了加快產品的更新換代，提高節能水平，滿足市場的需求，通過採用新材料的方法，來降低配電變壓器的空載損耗值，是一條很有效的途徑。本文著重介紹了非晶合金變壓器提高抗短路能力的方法。

關鍵詞：非晶合金；變壓器；抗短路


1. 引言

變壓器是根椐電磁原理而製造的一種輸變電設備，導磁磁路系統是變壓器的一個主要部分，導磁材料的性能直接影響變壓器的技術經濟指標，非晶合金變壓器就是用非晶合金材料代替硅鋼片製造變壓器，加工成的非晶合金鐵芯帶材厚度僅為0.025mm 這種新材料用於變壓器鐵芯磁化過程相當容易，並且非晶合金的飽和磁密較低一般在 1.3 到I.35T 之間，從而大幅度降低變壓器的損耗，它比現行的S9，S11 系列變壓器空載損耗下降達75%，空載電流下降達45%，推廣應用非晶合金變壓器不僅有良好的節能效益，而且有環保效益。

3. 提高非晶合金變壓器抗短路能力的措施

變壓器的短路機械強度是影響變壓器可靠性的重要因數之一。變壓器的系統運行中，由於各種偶然因素的存在，要想完全杜絕短路故障的發生是不可能的，所以要求變壓器能夠承受標準規定的短路故障。也就是說，要求變壓器應該具備一定的短路能力。下面是在設計和製造工藝方面提高變壓器抗短路能力的措施：

3.1 設計方面

大家知道，短路機械力取決於漏磁密和短路電流的大小，所以控制或減小漏磁和降低短路電流是設計中應該首先考慮的。

（1）減小安匝不平衡度，軸向力是由輻向漏磁引起的，輻向漏磁大小取決於安匝不平衡程度，所以設計時應儘量減小安匝不平衡程度。對於非晶合金配電變壓器來說，低壓大部分採用銅箔繞制，高採用多層圓筒式，相對於大中型變壓器在設計時比較容易做到。

（2）適當提高變壓器的短路阻抗，可降低短路電流倍數，提高變壓器的抗短路能力。

（3）改善鐵心的截面形狀，設計好繞組內部的徑向支撐，內繞組採用強度好的硬紙簡，以及合理佈置主空道油道撐條的數量。

（4）設計合適的繞組端部絕緣結構，低壓繞組目前採用端絕緣與層絕緣一休的結構，高壓繞組的端絕緣採用紙板與菱格絕緣紙相結合的方式，有效的保證在器身乾燥之後使導體與絕緣材料緊密的結合，形成一個整體,有效加大了繞組的穩定性提高變壓器抗短路能力。低壓繞組除小容量 ( 160 kVA 以下 ) 採用銅導線以外，一般採用銅箔繞制的圓筒式結構；高壓繞組採用多層圓筒式結構，使繞組的安匝分佈平衡，漏磁小。

（5）採用合理的引線結構，保證引線具有足夠的機械強度，器身採用吊板定位，以保證其機械強度。

3.2 製造工藝方面

（1）改進繞組繞制設備和工藝，高、低壓繞組採用導線張力裝置—起繞製成矩形線圈，提高繞組的緊度，通過熱壓整形將線圈固化成一整體，以增強繞組的機械強度和抗短路的能力。

（2）器身壓緊採用合理的上、下夾件夾緊結構，繞組壓緊墊塊均勻分佈在繞組兩端，既保證在上緊夾件後整個器身受力均勻，緊實，又要保證鐵心不受力。

4. 關鍵材料的選擇

4.1 低壓內筒的選擇

低壓線圈與鐵心之間距離最近，極易在短路力作用下造成低壓線圈損壞事故，因此在低壓線圈內側加放絕緣硬筒是提高變壓器抗短路能力的有效措施之一。環氧玻璃絲纏繞筒介電常數在2.5 ~ 4，大於紙板而接近於變壓器油，通過浸油試驗後的機械強度試驗，其機械強度基本沒有變化。所以我們選用環氧玻璃絲纏繞筒作為低壓繞組與鐵心之間絕緣硬筒。

4.2 鐵心材料的選擇

非晶合金是一種新型節能材料，採用非晶合金帶材製造的變壓器的空載損耗和空載電流非常低。

4.3 導體材料的選擇

在繞組導線選用上，高壓線圈選用(QQ-2 系列漆包銅圓線，縮醛漆包線具有耐油性能好，絕緣性能好等特點，低壓繞組採用銅箔代替以前常用的紙包銅扁線，層間絕緣採用稜格點膠紙，和低壓銅箔熱固化後，形成一個整休，有效的提高了抗短路能力。（當電流流過變壓器繞組時，在漏磁場與電流的共同作用下，在繞組導休將產生電動力，尤其是在突然短路時，將遭受巨大的短路電流的作用，這時電動力達到很大數值。在設計時選用絕緣材料和整個的絕緣結構在電動力作用下必須有足夠的動穩定性和機械強度。)

4.4 絕緣浸漆材料的選杼

變壓器矩形繞組乾燥後進行整體浸漆。選用1032 改性醇酸樹酯浸漬漆，該漆屬於烘漆，具有良好的乾燥性，耐油性，熱彈性和電性能，能夠有效提升繞組的機械強度和絕緣強度。

5. 成品試驗結果

按照上述改進方法制作變壓器，對成品進行短路能力承受試驗：

（1）試驗方法按G B 1094.5-2003 標準中的進行，試驗溫度應在10℃ ~40℃之間，低壓繞組短接，高壓繞組施加電流。

（2）試驗電源頻率應為變壓器的額定頻率，試驗應進行9 次試驗，採用1.5 倍的相電流試驗方法，每相進行 3 次試 驗 。

A 相試驗時分級位置應放在1 分接；

B 相試驗時分接位置應在3 分接，C 相試驗時分接位置應在5 分接。試驗持續時間為0.5S 。

（3）試驗結果判定：

1 ) 試驗過程中觀察波形應無異常。

2 ) 試驗完之後，每相短路阻抗的歐姆值與原始數據比較偏差不大於2%;

3) 檢查繞組、鐵心、夾件、絕緣墊塊等結構件應無變形，以及影響變壓器運行的缺陷。

4 ) 執行標準：GB/T1094.1-2013、GB/T1094.5-2008、GB/T25446-2010。

6. 總 結

本文針對目前非晶合金變壓器存在抗短路能力不足的情況，在設設計方面和工藝方面給出可行的改進措施，掌握非晶合金變壓器的機理，設計與製造等關鍵技術，在工藝不復雜，成本沒有大的變化情況下，製造安全可靠抗短路能力更強的非晶合金變壓器，以適應現代化配網的需要。
