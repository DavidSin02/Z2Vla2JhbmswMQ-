Excel如何設置表格邊框樣式，記住這幾個關鍵屬性就懂了

2020-10-29
| 科技

Excel表格邊框的設置，對於製作一個漂亮的工作表，十分有用。

由於審美的差異，我們通常做的表格都是以黑色邊框白底為主，或者說，有些人根本不知道除了黑白表格還有帶顏色的。

如何設置表格邊框屬性呢，下面介紹一下，邊框線型設置也就是Border樣式。

比如粗細，單線還是雙線，點還是點線結合，等等。

不同線的形狀在應用中可以加強或減弱某些內容的突出顯示或隱藏效果。

也是對錶格視覺效果上的處理，進行一個最優化的組合。

最好學過一些簡單的圖形圖象設計方面的知識，更加有用。

下圖分別為單元格上、下、左、右設置不同的線型，舉了一個例子。

線條樣式

樣式用LineStyle屬性來設置

例如：

range("A1").Borders.LineStyle = xlContinuous'實線

把A1單元格邊框設置為實線，其它如顏色和粗細均為默認值。

可參考邊框線條樣式表來設置不同的樣式效果。

邊框粗細

range("A1").Borders.Weight = xlHairline '細線

邊框的組細用Weight屬性設置。

參數可查看下圖進行對比設置。

代碼

函數調用方式

代碼寫到一個OLEobject按鈕裡面了，也可以放到任何觸發的事件裡面進行調用。

Private Sub CommandButton1_Click()			Me.UsedRange.Clear'清除表裡的內容			setBordersStyle'調用設置邊框過程End Sub

邊框設置過程（設置邊框上下邊框）

Private Sub setBordersStyle()    Dim b As Border    Dim r As Range    Set r = Me.Range("A2:F2")    r.ClearFormats    Set b = r.Borders(xlEdgeTop)        With b            .Color = RGB(255, 0, 0)          '  .LineStyle = xlSolid            .LineStyle = xlDot          '            r.Value = "上線框_點劃"            r.Columns.AutoFit        End With    Set b = r.Offset(1, 0).Borders(xlEdgeBottom)        With b            .Color = RGB(0, 0, 0)            .LineStyle = xlDouble            .Weight = xlThick            .TintAndShade = 0.1            r.Offset(1, 0).Value = "下邊框_雙線"        End WithEnd Sub

總體來說比較簡單，但是通常在工具欄裡也可以找到相應設置。
