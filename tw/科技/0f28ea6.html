瞭解筆記本內存條的發展史，對未來分析主板有一定的意義

2020-10-29
| 科技

RAM: Random Access Memory

隨機存儲器,存儲單元的內容可按需隨意取出或存入，且存取的速度與存儲單元的位置無關的存儲器。這種存儲器在斷電時將丟失其存儲內容，故主要用於存儲短時間使用的程序。 按照存儲信息的不同，隨機存儲器又分為靜態隨機存儲器（Static RAM,SRAM)和動態隨機存儲器（Dynamic RAM,DRAM)。

SRAM： Static Random Access Memory

靜態隨機存儲器,曾經是一種主要的內存，SRAM速度很快而且不用刷新就能保存數據不丟失。它以雙穩態電路形式存儲數據，結構複雜，內部需要使用更多的晶體管構成寄存器以保存數據，所以它採用的硅片面積相當大，集成度低,製造成本也相當高，所以現在只能把SRAM用在比主內存小的多的高速緩存上,如CPU的CPU內部的L1／L2或外部的L2高速緩存 。通常比一般的動態隨機處理內存處理速度更快更穩定。所謂靜態的意義是指內存資料可以常駐而不須隨時存取。因為此種特性，靜態隨機處理內存通常被用來做高速緩存 ;

DRAM：Dynamic Random Access Memory (25MHz~50MHz )

顧名思義即動態RAM。DRAM的結構比起SRAM來說要簡單的多，基本結構是一隻MOS管和一個電容構成。具有結構簡單、集成度高、生產成本低等優點，適合製造大容量存儲器，所以現在我們用的內存大多是由DRAM構成的。所以下面主要介紹DRAM內存。在詳細說明DRAM存儲器前首先要說一下同步的概念，根據內存的訪問方式可分為兩種：同步內存和異步內存。區分的標準是看它們能不能和系統時鐘同步。內存控制電路（在主板的芯片組中，一般在北橋芯片組中）發出行地址選擇信號（RAS）和列地址選擇信號（CAS）來指定哪一塊存儲體將被訪問。在SDRAM之前的EDO (Extended Data Out DRAM )內存就採用這種方式。讀取數據所用的時間用納秒錶示。當系統的速度逐漸增加，特別是當66MHz頻率成為總線標準時，EDO內存的速度就顯得很慢了，CPU總要等待內存的數據，嚴重影響了性能，內存成了一個很大的瓶頸。因此出現了同步系統時鐘頻率的SDRAM。

SDRAM：Synchronous DRAM(66MHz~133MHz )

同步DRAM,它運用晶片內的clock使輸入及輸出能同步進行。所謂clock同步是指記憶體時脈與CPU的FSB (Front Side Bus )頻率相同,能同步存取資料。SDRAM節省執行指令及數據傳輸的時間，故可提升計算機效率

DDR SDRAM: Double Data Rate SDRAM (266MHz~333MHz~400MHz )

雙倍速率SDRAM,由於DDR在時鐘的上升及下降的邊緣都可以傳輸資料，從而使得實際帶寬增加兩倍，大幅提升了其性能／成本比。

DDR2（Double Data Rate 2） SDRAM(533MHz~667MHz~800MHz~1066MHz)

它與上一代DDR內存技術標準最大的不同就是，雖然同是採用了在時鐘的上升/下降延同時進行數據傳輸的基本方式，但DDR2內存卻擁有兩倍於上一代DDR內存預讀取能力（即：4bit數據預讀取）。換句話說，DDR2內存每個時鐘能夠以4倍外部總線的速度讀/寫數據，並且能夠以內部控制總線4倍的速度運行。

DDR3 （Double Data Rate 3） SDRAM(1066MHz~1333MHz~1600MHz)

相比起DDR2有更低的工作電壓,從DDR2的1.8V降落到1.5V，性能更好更為省電；DDR2的4bit預讀升級為8bit預讀，DDR3目前最高能夠可以達到1600Mhz的速度，由於目前最為快速的DDR2內存速度已經提升到800Mhz/1066Mhz的速度，因而首批DDR3內存模組將會從1333Mhz的起跳。

DDR4內存是新一代的內存規格。2011年1月4日，三星電子完成史上第一條DDR4內存。比DDR3最大的區別有三點：16bit預取機制（DDR3為8bit），同樣內核頻率下理論速度是DDR3的兩倍；更可靠的傳輸規範，數據可靠性進一步提升；工作電壓降為1.2V，更節能

DDR5內存要來了？性能高達DDR4的兩倍之多！

內存條，我們都知道它是電腦平臺裡不可缺少的一部分，最早開始到現在的DDR2、DDR3、DDR4一類的內存條。隨著科技的發展，DIY各類產品層出不窮，而鎂光成為內存的老牌廠家，為無數王佳提供縱橫遊戲的尖端裝備，據10月份的消息，Cadence和鎂光公佈了自己的DDR5內存研發進度，兩家廠商一起開始研發16GB DDR5的產品，並計劃在2019年年底之前實現量產目標，

DDR5的工作頻率高達4800以上，這已經是超越DDR4內存的極限頻率了，而且公佈的最高頻率達到6400MHz，達到DDR4的兩倍之多，這是一種什麼概念？如果DDR5在出世的那一刻我相信這對內存條來說絕對是具有改革換代的革命性意義，最重要還是DDR5的電壓降低了，理論講頻率越大功耗越高，這還是得得益於多少nm工藝。
