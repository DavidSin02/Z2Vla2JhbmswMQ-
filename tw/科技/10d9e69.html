【iOS 開發 – SwiftUI 基礎 2/12】運算符

2020-10-29
| 科技

這一節，我們學習一下SwiftUI中所使用的運算符，包括以下幾種類型：

  • 算術運算符
  • 比較運算符
  • 邏輯運算符
  • 位運算符
  • 賦值運算符
  • 其他運算符

接下來，我們通過例子來具體講解一下相關運算符：

注意⚠️：Swift中運算法的兩側需要同時使用空格，或者同時不使用空格，否則會報錯！

1: 算術運算符

// 算術運算符let a = 10let b = 100let c = a + b // 加法，c = 110let d = b - a // 減法，d = 90let e = a * b // 乘法，e = 1000let f = b / a // 除法，f = 10let g = a % b // 取餘，g = 10

2: 比較運算符

let a = 4let b = 6// == 用來判斷兩側的取值是否相等// != 用來判斷兩側的取值是否不等print(a == b) // 輸出 falseprint(a != b) // 輸出 true// >, < 分別判斷兩側取值是否大於或者小於右側取值print(a > b) //輸出 falseprint(a < b) //輸出 true// >=, <= 分別表示 大於等於 和 小於等於print(a >= b) //輸出 falseprint(a <= b) //輸出 true

3: 邏輯運算符

// 邏輯運算符包括 或 (||)、與 (&&)、非 (!)var a = truevar b = falseprint(a || b) // trueprint(a || a) // trueprint(b || b) // falseprint(a && b) // falseprint(a && a) // trueprint(b && b) // falseprint(!a) // falseprint(!b) // true

4: 位運算符

// 位運算符是將變量轉換成二進制格式，然後按照對應位進行操作，包括 位與、位或、取反、異或、左移、右移。 其中異或的計算法則為："同出零，異出一".var a = 24 // 二進制 0001 1000var b = 14 // 二進制 0000 1100print(a & b) // 8 對應二進制 0000 1000print(a | b) // 28 對應二進制 0001 1100print(a ^ b) // 20 對應二進制 0001 0100print(~b) // 243 對應二進制 1111 0011print(a << 1) // 48 對應二進制 0011 0000print(a >> 1) // 12 對應二進制 0000 1100

5: 賦值運算符

嚴格來說賦值運算符只有等號 （=），但是在實際使用中，為了簡化代碼書寫，Swift語言允許我們將上述提到的各種算術運算法、位運算符與等號相結合，構成新的賦值運算符，例如 +=，實際使用中 先操作 前面的加法，後賦值。詳見下面的例子：

var a = 3var b = 5b += a // b = a + b, b = 8b -= a // b = b - a, b = 2// 其他組合依次類推

6: 其他運算符

// 1: 區間符號 "...", "<..", "..<"，表示一個連續的區間for i in 1 ... 3 {  print(i)}// 上面這段代碼輸出： 1，2，3。for i in 1 ..< 3 {  print(i)}// 上面這段代碼輸出： 1，2。// 2. 取負 "-"var a = -5let b = -aprint(b) // 5// 3. 正號 "+"var c = 5let d = +cprint(d) // 5// 4. 三元運算符 ？：// 問號冒號運算符與C語言中的規則是一樣的，當問號前的變量為True的時候，輸出冒號前的變量，當問號前的變量為False的時候，輸出冒號後的變量，let a = 3let b = 5print((a > b) ? a : b) // 5

這一節，我們系統梳理了SwiftUI中所涉及到的運算符，這些運算符不是特別容易記憶，在後續的使用中，我們可以在本文或者官方文檔中查詢具體的使用方法。

此外，運算符的優先級中，大的原則是高優先級的先算，低優先級的後算，其中單目的運算符優先級最高（正負號），然後是先乘除後加減，先算術>移位>位運算>邏輯運算。 同等級，從左往右按結合順序計算。
