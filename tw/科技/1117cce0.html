分支語句

1、if分支語句

1.1、if單分支語句

語法： if(邏輯關係表達式){ 代碼段1 }

當邏輯關係表達式的值為true時，執行對應{}種的代碼段，為false不執行

語法： if(邏輯關係表達式）{ 代碼段1 }else{ 代碼段2 }

2、Switch語句

數值表達式的結果值，會和case後面的數值意義匹配，匹配到哪個case語句，則執行相對應的代碼段

在jdk1.7以前，數值表達式可以是字符串

注意1：數值表達式中的值的數據類型要和case後面的數據類型一致

注意2：在使用switch語句的時候要注意case滲透問題，即當case語句後面沒用跟上break關鍵字的時候，如果執行了此case語句，那麼當此case語句後面的代碼執行完了以後，代碼它會繼續往下執行，執行下一個case語句，直到遇到break為止。

語法： Switch(數值表達式){ case 值1: 代碼段1 break; case 值2: 代碼段2 break; case 值3: 代碼段3 break; default: 代碼段4 }示例1：Scanner sc = new Scanner(System.in);System.out.println("請輸入你要的分數");int score = sc.nextInt();Switch(score / 10){ case 9: System.out.println("A"); break; case 8: System.out.println("B"); break; case 7: System.out.println("C"); break; case 6: System.out.println("D"); break; default: System.out.println("不及格");}示例2：// 輸入月份，判斷是哪一個季節 System.out.println("請輸入一個月份"); int month = sc.nextInt(); switch (month) { case 1: case 2: case 3: System.out.println("春天"); break; case 4: case 5: case 6: System.out.println("夏天"); break; case 7: case 8: case 9: System.out.println("秋天"); break; case 10: case 11: case 12: System.out.println("冬天"); break;

3、for循環語句

注意：在使用for循環的時候要先對其值進行初始化

語法：for(1.初始化計數; 2.循環條件判斷語句； 3.再求值參數){ 循環體 }示例：//打印10個helloWorldfor(int i = 1; i <= 100; i++){ System.out.println("helloWorld"); }示例2：//將 123456789數字倒置 int j = 123456789; int temp = j % 10; int LastNum = 0; //用來存放取出來的最後一個數字 int num = 0; for(;j>0;j=j/10){ LastNum = j % 10; num = num * 10 + LastNum;  } System.out.println("num:"+num);

4、while循環

當我們知道循環次數的時候使用for循環，當我們不知道循環次數的時候使用while循環

語法： while(條件判斷語句){ 循環體 }即當滿足條件判斷語句時才執行while循環語句/* 系統隨機產生1~100之間的隨機數 用戶從控制檯輸入數字進行猜測，如果猜大了，給與提示猜大了；如果猜小了，給 與提示猜小了 猜對推出遊戲*/ Scanner sc = new Scanner(System.in);​ // 使用random隨機數工具類來幫助產生隨機數 Random ra = new Random(); // ra.nextInt(100)產生0-99之間的隨機數 int randomNum = ra.nextInt(100) + 1; System.out.println("遊戲開始，請輸入一個數字"); int inputNum = sc.nextInt();  while (inputNum != randomNum) { if (inputNum > randomNum) { System.out.println("猜大了，請繼續輸入"); } else if (inputNum < randomNum) { System.out.println("猜小了，請繼續輸入"); } inputNum = sc.nextInt(); } System.out.println("猜對了，遊戲結束"); }   for (; inputNum != randomNum;) { if (inputNum > randomNum) { System.out.println("猜大了，請繼續輸入"); } else if (inputNum < randomNum) { System.out.println("猜小了，請繼續輸入"); } inputNum = sc.nextInt(); } System.out.println("猜對了，遊戲結束");  //while循環也可以替代for循環

5、Do while 循環語句

無論給出的條件是否滿足do while語句的條件，它都會執行一次，

特點:無論是否滿足循環條件，循環體至少被執行一次

語法： do{ 循環體 }while(條件判斷語句)示例：int i = 19; do{ System.out.println("hello"); }while(i > 20); }此時我們給到i的值為19,它不滿足while中的條件判斷語句(i>20)，但是由於do while循環語句的特點，它是先執行循環，因此。它會輸出一次hello而後對該條件進行判斷，不滿足給出的條件語句，則停止循環。輸出一次hello

6、循環結束的關鍵字

1、break:

結束當前整個循環,即使還有剩餘的循環次數，也強制結束

int i = 0; while(i<5){ if(i == 3){ break; } System.out.println("i:"+i); i++; }在此處它表示當i = 3 的時候 結束當前整個循環，因此它的輸出結果為 0 1 2

2、continue

表示忽略當前這一次循環（即忽略當前這一次循環體中的內容），如果還有循環次數，繼續執行。

 for(int i = 0; i < 5; i++){ if(i == 3){ // break; continue; } System.out.println("i:"+i); }此處它表示跳過 i = 3 的這次循環，因此它的輸出結果為 0 1 2 4

7、Java中的方法

1、什麼是方法？

方法：是指對程序中的某個代碼段進行封裝，方便以後調用。一個方法儘量只執行一個功能。

2、如何定義方法

定義方法的語法：修飾詞(權限修飾詞，類型修飾詞..) + void/返回數據類型 + 方法名(參數列表){  方法體(具體的方法邏輯); }

8、return和void

1、什麼是return？

return關鍵字，一般用於方法體的末尾，在表示結束某個方法，在return後面不允許在出現代碼

2、什麼是void

void它緊跟方法名，表示此方法沒有返回值

3、什麼是返回數據類型

返回數據類型：返回數據類型它和void一樣都是緊跟方法名。表示此方法會有對應類型的數據返回，並且要通過return關鍵字將該對應類型的數據返回，否則編譯報錯示例：import java.util.Scanner;​public class Demo01 {​ //此方法用於找零錢操作 public static double getRish(double price,int num,double money){ double rush = money - price * num; return rush; }​ //此方法用於實現打印小票的操作 public static void print(double price,int num,double money){ System.out.println("商品名：xxx" + "商品單價:" + price + "商品數量：" + num + "商品總價：" + (num * price)); }​ //顯示找回的零錢 public static void showRish(double rush){ System.out.println("應找零錢:" + rush); } public static void main(String[] args) { //在此處模擬一個收銀臺操作，從控制檯輸入用戶需要的數量，商品價格，和用戶所給的錢 //計算機做出找零操作並顯示 Scanner sc = new Scanner(System.in); System.out.println("請輸入商品單價"); double price = sc.nextDouble(); System.out.println("請輸入商品數量"); int num = sc.nextInt(); System.out.println("請輸入您給收銀臺的錢"); double money = sc.nextDouble();​ //計算找零操作 double rush = getRish(price,num,money); showRish(rush); }}

10、形參和實參的區別

1、什麼是形式參數：

形式參數指的是函數調用時，用來接收實際參數的值，稱為形式參數

形參：全稱為形式參數，由於它不是實際存在的參數，所以它又稱之為虛擬變量，是在定義函數名和函數體的時候使用的參數，目的是用來接收調用該函數時所傳進來的參數。調用該函數時，實參將值賦給形參，因此必須要注意實參的個數，類型要於形參一一對應。並且實參要有確定的值。

2、什麼是實際參數

實際參數是指在調用是傳遞給函數的值，它可以是變量，常量，表達式，函數等，無論實際參數是什麼變量的值，在進行函數調用的時候，都必須要有一個確定值。以便於把這些值傳給形式參數，因此應該先對實際參數預先賦值，使實際參數擁有一個確定的值。

3、形式參數和實際參數的區別

形式參數通常出現在函數中定義，即寫在方法名後的括號中。在整個函數中可以使用，離開這個函數則不能使用。

實際參數通常出現在主函數(main方法)中，當實際參數進入到被調函數中，實際參數變量也不能被使用。

實際參數和形式參數無論是在類型上，數量上還是順序上都要嚴格一致，否則會出現"類型不匹配的錯誤"。

形式參數只要在被調用的時候才會分配內存空間，在調用結束後，立即釋放內存，因此形式參數值只在函數內部有效，當函數調用結束，返回主函數後，該形式參數變量則不能在被使用

實際參數可以是常量，變量，表達式，函數等，但是無論實際參數是什麼樣的值，在進行函數調用的時候，實際參數都必須要有一個確定的值。以便於把這些值傳給形式參數，因此應先對實際參數進行預先賦值。使實際參數擁有一個確定的值。

例子：形式參數：public void method(int a,int b){ 方法體; } 這裡的a和b就是形式參數，即形參是一個沒有賦值的變量，通常是在函數定義時使用的。​實際參數當進行函數調用的時候add(1,2) 這裡的1和2就是實際參數，即是一個有賦值的變量。通常是在函數調用時使用。
