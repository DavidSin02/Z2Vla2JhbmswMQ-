Fuse.js - 輕量級零依賴的JS模糊搜索插件

2020-11-14
| 科技

做了個數據的表格展示，還帶有前端搜索功能，你看著瀏覽器覺得很滿意，可不一會兒需求來了，說要改成模糊搜索。你在腦海中構思了好一陣子，暗想又要加班了。不！Fuse.js 來告訴你，前端模糊搜索只需幾行代碼就能搞定！


前端模糊搜索

簡介

Fuse.js，是 krisk 在 Github 上開源的 Javascript 模糊搜索工具，代碼倉庫在 https://github.com/krisk/Fuse，目前版本為 6.2.0。其功能強大，輕量，且沒有額外依賴，同時使用簡單，在想要實現前端模糊搜索，或在 Javascript 後端提供簡單模糊搜索服務時，Fuse.js 是你的最佳選擇。

Fuse.js

安裝

Fuse.js 沒有額外依賴，可以使用 npm 安裝

npm install --save fuse.js

或 yarn 安裝，

yarn add fuse.js

也可以使用 CDN 在瀏覽器上使用：

<script src="https://cdn.jsdelivr.net/npm/fuse.js@6.2.0/dist/fuse.min.js"></script>

示例

Fuse.js 使用相關性分數進行模糊搜索，分數決定於：模糊係數分數，關鍵權重和正規化的域長度。我們來看一些基本的使用例子。

const list = ["Old Man's War", 'The Lock Artist']const fuse = new Fuse(list)const result = fuse.search('od man')

首先定義了一個字符串列表作為待選的數據，然後實例化 Fuse，並接收想要搜索的文本執行 search。返回的輸出如下：

[  {    "item": "Old Man's War",    "refIndex": 0  }]

可以看到，“od man”的輸入並不是任何備選數據的子串，但通過 Fuse.js 的模糊搜索，得到了最適合的搜索結果。

Fuse.js 也可以對對象列表進行搜索，支持對於嵌套對象的搜索：

const list = [  {    title: "Old Man's War",    author: {      name: 'John Scalzi',      tags: [{ value: 'American' }]    }  },  {    title: 'The Lock Artist',    author: {      name: 'Steve Hamilton',      tags: [{ value: 'English' }]    }  }]const options = {  includeScore: true,  keys: ['author.tags.value']}const fuse = new Fuse(list, options)const result = fuse.search('engsh')

該代碼對於嵌套的 author.tags.value 域進行了模糊搜索。

Fuse.js 選項豐富，包括

  • isCaseSensitive：是否大小寫敏感；
  • includeScore：是否返回匹配分數；
  • shouldSort：是否對返回結果進行排序
  • ……

可以根據實際使用情況進行設置。

我們還可以對於不同域的權重進行定製。權重默認為1，必須大於0。權重越大，對搜索結果的影響越大。例如，我們可以對文章的標題賦予更高的權重，更符合搜索的需要。

const options = {  includeScore: true,  keys: [    {      name: 'title',      weight: 0.8    },    {      name: 'content',      weight: 0.4    }  ]}

另外，Fuse.js 還提供了擴展的搜索語言，來提供更為豐富的搜索功能，包括

  • abc：模糊搜索
  • 'abc：精準搜索，需要包括搜索的文本
  • !abc：精準反向搜索，需要不包括搜索的文本
  • ^abc：前綴搜索，前綴為目標文本
  • ……

總結

Fuse.js 使得模糊搜索功能的實現變得簡單，在前端搜索中應用廣泛。其所支持的功能豐富，能適應許多功能場景。

Fuse.js 十分輕量，沒有額外依賴，在前後端皆可使用；接口簡潔，使用簡單，方便上手，值得使用和學習。
