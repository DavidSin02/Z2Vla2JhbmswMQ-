90後文物修復師，化腐朽為神奇，贊！

2020-10-29
| 科技

（更多精彩，歡迎關注：知羹）

首先，請仔細看這兩張圖：

這是一件名為《鍾馗騎驢圖》古畫的修復前後對比圖，在看第一張圖時完全想象不到它原本破爛、滿是黴斑的樣子。

還有這些：

民國 /《虎》

原畫直接從中間裂開了一個大口子，老虎頭部也有許多缺失，修復後幾乎看不出裂開的痕跡，虎頭部分做到了完整的還原。

宋代 / 吉洲窯窯變釉茶盞

看完這兩張對比圖，不禁想到了那句古詩「粉骨碎身全不怕」。修復後茶盞不僅平整光滑，而且連釉面的光澤度也絲毫不差。

清代 / 字彙（亥部）

蟲蛀、汙漬、褶皺......這本古籍到底經歷了什麼？修復之後，邊頁變得平整，也保留了原本的歷史感。

眾多被修復的文物中最久遠的一件是下面這張，距今至少有1000年的歷史了。

▽

五代 / 越窯青釉水盂

這些被“救”活的文物，都出自南京藝術學院14級文物鑑賞與修復專業。

根據種類，又分為書畫、古籍和陶瓷三個方向，每個方向都有其神奇之處。（神奇在哪？往下看）

書畫 /////

《民初百子鬧春圖》

年代：清末民初 / 作者：曾博

邊緣充滿褶皺，還有不少已經凋落，汙漬的多年堆積也使整幅畫面模糊不清。

修復後，終於看得清畫面上的“百子”，就連放爆竹、打鬧的細節動作都十分清晰。

《猛虎圖》

年代：近代 / 作者：華文韜

透過破碎的畫也能感受到老虎的霸氣，但就是破碎太多，虎爪和虎頭都有明顯缺塊。（破成這樣，山大王不要面子的嘛？）

破碎處修復得毫無痕跡，就連老虎的鬍子都根根分明，霸氣！

《虎》

年代：民國 / 作者：李婕儀

又是一隻虎，但這隻虎好像是在尋找什麼。

整幅畫面變得乾淨了許多，最吸睛的是虎頭部分，本來缺一塊的虎眼也做到了毫無PS痕跡。

《大展宏圖》

年代：現代 / 作者：沐晶晶

這幅畫的整畫幅都充滿著褶皺，畫的右上部分還缺了很大一塊。

修復後不僅畫面變得平整了很多，而且還細心地將右上方的缺失部分做舊，以免與整幅畫不搭調。

《溪山行旅圖》

年代：現代 / 作者：邱駱圓

這幅山水畫的頂部直接被消失了，右側也有許多陳年的汙漬。

修復後畫面變得清楚了許多，前景的挑擔小人也十分可愛。

《鍾馗騎驢圖》

年代：清末 / 作者：水豪傑

這是最幸運的一幅畫了，主體部分還算完好，丟失的部分剛好避開了畫作的主體。

在修復後，畫作大量的留白給觀者十足的想象空間，畫中的鐘馗還有一些萌萌的。

《巧姐》

年代：現代 / 作者：陶穎

保存還算完好，但細節部分缺失較多，從圖片上根本無法分辨巧姐的五官，背景也模糊不清。

看完修復後的圖才知道原來是巧姐在勞作，裙襬上的吊穗以及花紋都躍然紙上。

《碩果累累》

年代：民初 / 作者：陶源

哪來的有心人，正正好好將畫作分成了十六份。畫布不僅泛黃，部分破損處還丟失了。

原來這幅畫作原本的樣子這麼白皙，修復後幾乎看不出原本的裂痕，好棒！

《策杖訪友圖》

年代：近代 / 作者：王銘鈺

老友、仙鶴、童子，在黴斑和破碎的痕跡下只能隱約透露出來...

可是在修復者的巧手下，畫作不僅完整了，身後童子、身下仙鶴，更體現出了原本老者訪友的意境。

《西施浣紗圖》

年代：清代 / 作者：熊沛霖

這幅畫作破損時的顏色本沒有這麼鮮豔，但在修復後，鮮豔的色彩和綠色的前後景將西施的美描繪了出來，一顰一笑皆為絕美～

《東風浩蕩春似錦》

年代：近代 / 作者：姚昊鵬

這彷彿是加上了「舊畫」濾鏡，原色被汙漬擋住了不少，也缺少了春的味道。

摘掉“濾鏡”後，畫作更加明朗鮮豔，兩隻可愛的小鳥也在花枝頭間嬉戲。

《張仲林書法》

年代：現代 / 作者：尹秋力

這實在是太有視覺衝擊力了！這一堆破爛的紙張到底是什麼？

“拼”完之後呈現出的是80年代的書法作品。

* 其實以上有許多的書畫作品原本都是這樣的，修復的每一個步驟都充滿著挑戰。就比如這裡的“拼圖挑戰”。

《常建樟書法》

年代：清末民初 / 作者：陳俊彥

這幅書法作品雖然能看清出字跡，但多年的色素沉積使作品不夠乾淨，也掩蓋了原本的字之美。

修復後的作品，比劃清晰可見的渾厚，就連印章也有邊有角。

《貴》

年代：現代 / 作者：潘思梵

最喜歡的就是這一幅了，因為它“貴”哈哈哈哈。紙張下端十分薄，還有許多零零散散的碎片。

修復之後，比劃間的藕斷絲連流暢，充滿線條美，“貴”字，活了！

古籍 /////

古籍的繁複，是在於一整本每一頁地修復。

細節處連古籍的託紙材質、書芯最大寬、PH值等都要一一測量，精準到毫釐。

《古本祭文書》

年代：清代乾隆 / 作者： 王澤鈺

就比如這本古籍的封面就缺失了很大一塊，角落處也有很多褶皺。（強迫症絕對忍不了！）

原本灰塵汙染嚴重，染色嚴重的古籍，在修復後還有一種通透的美感。

《花樣集》

年代：清代道光 / 作者：陳凱奕

這本古籍出現了一定程度上的脆化，每張書葉都存在蟲蛀，汙染面積也超過了1/3。

修補後補全了被蟲蛀的地方，也最大程度地降低了汙染面積。

《民歌集錦》

年代：清代 / 作者： 韓芳

這本古籍的書葉紙張厚度為0.064cm，稱作薄如蟬翼也不足為過了。

修復者用了宣紙作為託紙，為古籍增添了一份保護衣。

《花樣集錦》

年代：清代 / 作者： 李璐

這本古籍在保存過程中，染上了油漬、水漬、黴斑，在大部分書葉死角都存在著斑駁。

修復採用了“金鑲玉”的裝幀方式，用多出來的託紙（圖中滴白色部分）保護原本的古籍。

黃兆奎書《畫禪室隨筆》

年代：清代 / 作者：李盼

換了另一種裝幀方式，再加上修復者精心挑選的書套，是不是很炫酷？

《明代羅教寶卷》

年代：明代 / 作者： 彭晨

這本古籍字跡部分保存還算完整，但摺痕明顯，夾雜著很多汙漬。

修復後將所有的文字部分保存完整，在後面附上了一層宣紙保護原件。

《管韞山全集》

年代：清代 / 作者： 王芳

修復者正在拿毛筆修補蟲蛀處，左手是補紙，每一頁都要如此，想象一下......

《新刻異說南唐演義全傳》

年代：清代 / 作者： 王婷

這本線裝古籍的書芯最大長為22.6cm，書芯最大厚為0.5，書本的脆化程度也十分嚴重，看圖！

《禁扁 丁、戊》

年代：清代康熙 / 作者： 夏豔琳依

古籍的書衣有破損，書口也出現了斷裂。

修復後的內頁部分完好無損，並且書的切面十分平整，有一種新書做舊的感覺哈哈哈。

《字彙 （亥部）》

年代：清代乾隆 / 作者： 楊夢桐

汙漬太多，連書本上的字都快看不見了。

《四書六經義》

年代：清代光緒 / 作者： 張辰芳

經歷過時間的沖刷之後，產生了脆化，感覺隨便一翻就會“粉骨碎身”...

只能說：修得這麼好，太不容易了！

《澄衷蒙學堂字課圖說》

年代：民國 / 作者： 張曉楠

古籍的書葉一角被老鼠啃出了一個梯形，還有不少書葉掉落。

修復後還原了古籍的歷史感，在書角處還做了弧邊處理。

陶瓷 /////

作為三個方向裡最特殊的存在，陶瓷修復會根據陶器本身運用不同的技術手段，對比也是十分有視覺衝擊力。

《宋湖田窯印花花卉紋盤》

年代：宋代 / 作者：楊暘

陶瓷碎成了八塊，在第一步拼接是一定要先拼銳角，才能有利於修復的平整。

聽修復者說：在你沒有把它拼起來之前，你永遠不知道它是個啥！

《清景德鎮青花花卉紋盤》

年代：清代 / 作者：陳雨婕

由於青花產生不穩定的原因，修復青花瓷器的難度在陶瓷方向的同學心中堪比高考數學最後一題的難度。

《宋代青白釉花口瓜稜瓶》

年代：宋代 / 作者：郝夢

這個青白釉花口瓜稜瓶只剩下了底座和瓶身的部分，瓶口部分缺失。

瓶口在修復者的觀察走向以及參考歷史文獻後，做成了波浪形，靜態的瓶身有了動態的美感。

《宋湖田窯獸首三足爐》

年代：宋代 / 作者：李雅倩

小尺寸的香爐在古代往往用於焚香，爐口部分缺失近三分之二。

用石膏修補完之後，還原了三足爐的原樣，古器在修復後反倒有了一種反差萌的感覺？

《南宋龍泉窯青瓷鳳耳瓶》

年代：南宋 / 作者：李穎

青瓷瓶的瓶身上方一處脫釉，在修復後，完全看不出來，也太讚了。

《清康熙青花纏枝花卉紋碗》

年代：清代 / 作者：欒燕菲

這怎麼這麼像小時候家裡的青花碗？（別做夢了）

清代的青花碗在碗側丟失了一塊碗壁，確實的青花部分也需要修復者重新填色，我現在已經開始懷疑自己的視力了。（為啥我對照第一章都看不出第二個缺失了哪？）

《宋當陽峪窯綠釉刻花花卉紋瓷枕》

年代：宋代 / 作者：裴雪敏

數了一下，大大小小一共27塊，光看碎片完全看不出這是個啥。

拼完之後是個瓷枕，綠白的經典配色，加上藤枝的花紋，更增加一份東方神韻。

《清景德鎮窯白釉鋪首爐》

年代：清代 / 作者：屈婉婷

破碎的瓷器的邊緣都不平整，用石膏還原後的拼接感還有另一種特殊的美感。

《宋代吉州窯窯變釉茶盞》

年代：宋代 / 作者：戎悅

又是一個讓人懷疑視力的作品，光看第二張圖真是想象不出它原來的樣子。修復後的釉面質感很好的掩蓋了裂縫處痕跡。

《建窯黑釉瓷碗 》

年代：宋代 / 作者：阮博

這是修復中的樣子，破碎處已經用石膏進行填補。

碗內的金屬光澤也拿顏料做了補充，與原物做到儘量的協調補充。

《清代豆青釉青花五福捧壽紋盤》

年代：清代 / 作者：史雪楓

壽紋盤是一個對稱的圓盤，對照兩張圖，你能找出缺失的部分嗎？

《清雍正景德鎮窯青花過牆雲龍紋盤》

年代：清雍正 / 作者：孫含蕙

這個作品除了完整了整個盤子，龍爪處也做了一定的補充。

《清雍正景德鎮民窯青花雲龍三現紋盤》

年代：清雍正 / 作者：朱黎

這個瓷盤與上一個有一定的相似之處，但不同的是修復者運用了金箔和黑色的顏料，體現出了另一種美感。

《宋吉州窯窯變釉碗等》

年代：宋、元 / 作者：卞麗君

修復中....

五個瓷碗各有不同（最喜歡藍的那個），所缺的部分也用了最適合的修復手法，保存了原件的歷史感。

《宋吉州窯黑釉木葉紋盞》

年代：宋代 / 作者：孫悅

中間這一塊深淺不同的黑色就是碗身的碎片，你看出來了嗎？

從一塊碎片到一個碗的故事...... 太神奇了！

《民國粉彩福壽紋三多盤》

年代：民國 / 作者：陶思妤

瓷器中少見的彩色，這件粉彩盤的色彩清爽，是瓷器中少有的小清新了，完整後的效果更加驚豔。

《清末粉彩萬壽無疆紋蓋碗》

年代：清末 / 作者：王琰

這件紋蓋碗採用了偏現代化的修復手法，用金箔覆蓋裂痕處，既保存了原本的樣子，又給予了文物新生。

《明龍泉窯蓮瓣紋盤》

年代：明代 / 作者：王楊碩

可以從碎片中看出，盤底缺失了一塊，釉面基本保存完整。

修復後，盤內的花紋與原本做到了對齊。裂縫之間的空隙也填補到了最小。

《漢紅彩陶罐》

年代：漢代 / 作者：吳學翰

作為這些文物中較古老的存在，陶罐保存至今已經少之又少。

修復時更保存了罐子原本的歷史感，人工做出了斑駁和時間累積的汙漬，給予歷史最大的尊重，點贊！

《清光緒民窯青花山水人物紋》

年代：清光緒 / 作者：項影

這件瓷器雖然破損部位較少，但修復效果十分驚豔，告訴缺口你也找不出系列。

《宜興紫砂金繕鋦釘杯》

年代：現代 / 作者：楊曉熠

喜慶的橘紅色加上圓滾滾的杯型，看著還有些可愛，修復者還加了朵金色的梅花，俏皮中不失優雅。

《宋白釉菱紋花口尊》

年代：宋代 / 作者：周禺含

修復者有心的將所有的碎片編上號，以免拼接時出現差錯。

瓶身的花紋和光澤感都完好無缺（石膏的運用是用來保證文物的真實性），整個瓷瓶呈流線型，莊重感十足。

寫在最後

每一個文物都是經過了層層把關，PH值不均勻就一遍一遍地過濾，拼的不平整就重新來直至光滑，光揭背這一項程序能進行一個月......

正是因為他們的出現，那些原本破舊的文物，才有了新的生命和更高的藝術價值。

我只能說，文物鑑賞與修復專業，你們太太太太讚了！
