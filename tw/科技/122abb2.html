如何自定義座標刻度？掌握2個函數方法輕鬆搞定！

2020-10-29
| 科技

我們在使用Matplotlib時，雖然Matplotlib默認的座標軸定位器與格式生成器可以滿足大部分需求，但是有時候為了使圖片看起美觀，我們需要手動去調整XY軸的刻度。

主要刻度與次要刻度，通過什麼來設置？

每一個座標軸都有主要刻度線與次要刻度線。主要刻度往往更大或更顯著，而次要刻度往往更小。雖然一般情況下 Matplotlib 不會使用次要刻度。但是在處理對數的時候你會看到它們的身影。

import matplotlib.pyplot as pltplt.style.use('seaborn-whitegrid')import numpy as npax = plt.axes(xscale='log', yscale='log')

通過上面的例子，我們發現每個主要刻度都顯示為一個較大的刻度線和標籤，而次要刻度都顯示為一個較小的刻度線，且不顯示標籤。

但是我們可以通過設置每個座標軸的formatter與locator對象，自定義這些刻度屬性(包括刻度線的位置和標籤)。

用print打印一下圖形X軸的屬性；

print(ax.xaxis.get_major_locator())print(ax.xaxis.get_minor_locator())<matplotlib.ticker.LogLocator object at 0x11c64ac88><matplotlib.ticker.LogLocator object at 0x11c64a278>print(ax.xaxis.get_major_formatter())print(ax.xaxis.get_minor_formatter())<matplotlib.ticker.LogFormatterSciNotation object at 0x11c64a208><matplotlib.ticker.LogFormatterSciNotation object at 0x11c626390>

通過打印出來的對象，很清楚的看到主要刻度標籤和次要刻度標籤的位置都是通過一個LogLocator對象設置的。次要刻度有一個NullFormatter對象處理標籤，這樣標籤 就不會在圖上顯示了。

如何隱藏刻度與標籤？

可以通過plt.NullLocator()與plt.NullFormatter()這兩種方法實現。

ax = plt.axes()ax.plot(np.random.rand(50))ax.yaxis.set_major_locator(plt.NullLocator())ax.xaxis.set_major_formatter(plt.NullFormatter())

通過上面的圖表，我們發現，雖然移除了x軸的標籤(但是保留了刻度線和網格線)，以及y軸的刻度 (標籤也一併被移除)。

如何增減刻度數量？

處理數據時，默認的刻度標籤有一個問題，顯示較小的圖時，通常刻度看起來十分的擁擠。通過一個例子來展示一下。

fig, ax = plt.subplots(4, 4, sharex=True, sharey=True)

我們看到x軸數字幾乎都重疊在一起，辨識起來非常困難。我們可以用 plt.MaxNLocator()來解決這個問題，通過它可以設置最多需要顯示多少刻度。根據設置的最多刻度數量，Matplotlib 會自動為刻度安排恰當的位置。

# 為每個座標軸設置主要刻度定位器 for axi in ax.flat: axi.xaxis.set_major_locator(plt.MaxNLocator(3)) axi.yaxis.set_major_locator(plt.MaxNLocator(3))fig

調節了圖表的刻度後，對比上面的一張圖看起來是不是很美觀了不少？當我們數據可視化做的越多你會發現自定義刻度用起來比默認的要美觀，雖然沒有默認那麼快捷。
