這個方法，有效緩解玩遊戲出現的網絡延遲現象

2020-10-29
| 科技

大家知道，我們在電腦中玩遊戲固然可以獲得大屏視覺體驗，但是經常會出現網絡延遲較高的現象，網絡明明很好，也嘗試了幾種方法，還是不能解決問題，怎麼辦？小租詳細整理了兩種方法，可以幫助廣大玩家解決網絡延遲高的問題。

電腦玩遊戲網絡延遲高怎麼辦？

方法一

鼠標點擊網絡連接圖標，選擇【打開網絡和共享中心】，



進入界面，在左側菜單欄中找到【更改適配器設置】功能並點擊，如圖，



接著進入網絡連接界面，鼠標右擊【本地連接】，彈出菜單，找到【屬性】選項並點擊，



如圖，找到【Internet協議版本6】，取消勾選，然後點擊【確定】，此方法可以有效降低Windows 7系統的網絡延遲。



方法二

選擇關閉Multimedia Class Scheduler服務，這個服務主要是視頻播放的優先級服務，開啟之後，如果播放視頻就會影響遊戲延遲。

點擊開始菜單，然後點擊打開控制面板，如圖，選擇【管理工具】，



進入管理工具界面，找到並點擊【服務】功能，如圖，



進入服務界面，在右側找到【Multimedia Class Scheduler】選項並點擊，



打開Multimedia Class Scheduler服務的屬性界面，點擊【停止】功能，關閉，然後重啟計算機。



總體來說，以上兩種方法都可以有效解決“電腦玩遊戲網絡延遲高”的這一問題。
