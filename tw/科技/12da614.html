Linux這5款微型發行版，體積小+精簡，比win7運行還快，值得安裝

2020-10-29
| 科技


（此處已添加圈子卡片，請到今日頭條客戶端查看）


Seth Kenlon曾經說過：“這些微型 Linux 發行版可以讓你的老爺機復活，可以啟動一個損壞的系統，或者是確保在公共電腦上進行安全的操作。”眾所周知，能夠為我們所用的Linux發行版有很多，但大家是否聽說過Tiny Core、Mint等等發行版呢？



這些發行版體積小，使用一整套完整的操作系統驅動，只需一臺不到1GB和512MB內存的計算機即可，堪稱是終極的黑客作風！那麼接下來老毛桃就跟大家分享5款微型Linux發行版。



1、Linux Mint


Linux Mint是一種基於Ubuntu 或 Debian 開發出的 Linux發行版 操作系統，它繼承了Ubuntu 的眾多優點，同時也在 Ubuntu 的基礎上加入很多自己優秀的特性。為了追求易用性和良好的人機交互體驗，Linux Mint 集成了可以開箱即用的大量多媒體等軟件包。因此它的 Live ISO 體積超過了 ubuntu 的本地安裝鏡像，甚至大於 950MB。



此外，為了迎合不同用戶的口味，Linux Mint 官方提供了 4 種不同的桌面環境版本可以供你選擇下載，分別是Cinnamon、MATE、KDE和Xfce，感興趣的小夥伴可自行了解。如果你正在為臺式機、筆記本選用一款桌面級的 Linux 發行版，那麼 Linux Mint 可能會讓你感覺到相見恨晚！它簡單易用，桌面優雅舒適，速度流暢且非常穩定。


2、Tiny Core


Tiny Core Linux是一款體積極小且高度可擴展的微型Linux發行版，終端版本只有 11 MB，圖形界面版本只有 16 MB。由於設計得極端精簡，甚至安裝完整操作系統的應用程序都沒有被包含在內，安裝後我們需要通過以太網下載需要的應用程序。



3、SliTaz Linux


SliTaz Linux的鏡像文件大約在51MB左右，雖比Tiny Core的大了四倍，但比較於Windows系統也算是微型系統了。即便微型，但它還是包含了一整套出色的驅動程序和應用程序，能夠為大家提供的應用程序能夠滿足大多數基本的需求。



4、Porteus Linux


Porteus是微型 Linux 中鏡像文件最大的一個，最小的鏡像文件大約在270MB，最大的則有350MB，但是這些額外的空間都被用來確保一個非常順暢的 Linux 桌面環境的體驗。我們可以在MATE、LXQT、LXDE、OpenBox、XFCE、Cinnamon 或者 KDE 裡選擇自己喜歡的桌面環境。在此老毛桃給大家推薦MATE 或者是 KDE 桌面，它們可以為你提供熟悉的桌面環境體驗。



5、Bodhi Linux


Bodhi Linux系統以佔用系統資源小，速度快的Enlightenment窗口管理器為基礎，打造了一個優雅乾淨，來自未來的桌面環境。關於應用程序，大家可以在Bodhi Linux的庫中找到內置的應用程序，可供使用的可謂是數不勝數！



以上5款微型發行版體積小且精簡，如果你有臺舊電腦，不妨試試？
