Sql中 if else判斷語句

1. if...else 流程控制塊語句

語法：

if Boolean_expression begin.... endelse begin... end

注意：

--1.沒有bool值，只有條件表達式

--2.沒有{} 只有begin end

--3.可以多重嵌套

--4.如果包含的語句只有一句，也可以省略begin end

--5.if ...else 裡面必須有處理語句,如果沒有就報錯。

--6.不能在select語句中使用。

例如：

 if(1=1)	begin		print '劍雲鋒'		print 'Welcome!'	endelse		print 'You are wrong!'

while循環語句

語法：

WHILE Boolean_expression { sql_statement | statement_block } [ BREAK ] { sql_statement | statement_block } [ CONTINUE ]

解釋：

while 判斷表達式 begin  --循環體 endelse begin --輸出表達式 endend--循環結束標誌--continue 繼續--break 跳出循環

注意：

--1.可以嵌套

--2.沒有{} 只有begin end

--3.不能使用true /false ，必須使用條件表達式

--4.如果包含的語句只有一句，也可以省略begin end

--5.可以使用break / continue

--6.可用在select語句中。

例如：計算1-100的和，到50時停止跳出

declare @i int declare @sum int set @i = 1set @sum = 0while @i<10	begin 	if(@i!=5)		begin			set @sum = @sum+@i			set @i = @i + 1		end		else		break	end --循環結束標誌print @sumgo

3.邏輯運算符 and, or, between(),any(),in(),like(),some(),not() ,exists()

3.1 and, or, between..and,all()

--1.選擇KPS>90和Name='ee' 的結果集select * from TCourse where KPS>90 and Name='ee'--2.選擇KPS>90 或者 KPS<80 的結果集select * from TCourse where KPS>90 or KPS<80--3.選擇KPS在80-90之間的結果集select * from TCourse where KPS between 80 and 90--4.選擇滿足所有KPS>90的名字條件的結果集select * from TCourse where Name=all(select Name from TCourse where KPS>90)

3.2 any(),some(),exists(),not exists()

--5.選擇滿足任意一個KPS>90的名字條件的結果集select * from TCourse where Name=any(select Name from TCourse where KPS>90)--6.選擇滿足任意一個KPS!>90的名字條件的結果集select * from TCourse where Name=some(select Name from TCourse where KPS!>90)--7.存在Teacher.Name = TCourse.Name的TCourse表中所有的結果集select * from TCourse where exists (select * from Teacher where Teacher.Name = TCourse.Name )--8.不存在Teacher.Name = TCourse.Name的TCourse表中所有的結果集select * from TCourse where not exists (select * from Teacher where Teacher.Name = TCourse.Name )
