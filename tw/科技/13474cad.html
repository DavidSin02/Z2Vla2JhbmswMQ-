分享一個漂亮的查詢表，自帶切片圖表，查詢很給力！

2020-11-14
| 科技

工作中，經常會遇到多個表數據合併進行彙總的問題，一般都是先進行表格合併，再進行數據彙總統計。有時候數據彙總統計時不一定要對錶格進行合併，也可直接對多個表數據進行查找彙總，這樣就省去了多表合併的麻煩。今天給大家分享一個無需合併直接對多表數據進行彙總顯示的查詢表，通過幾個小技巧的運用，查詢起來非常高效便捷。

先看效果

除查詢表外，其他均為基礎數據表，每個表均由單位名稱和年度月份作為表單名，數據不需提前合併。在查詢表裡將表單名分解為單位和年度月份作為查詢條件，另外將文件里人員名單也作為查詢條件。

功能小技巧

1、首先通過下拉菜單選擇查詢單位，即所屬國家。

小技巧：上圖中查詢單位的下拉菜單選項直接由代碼生成，省去了輔助數據區的設置，表格更加乾淨精煉。重點是當選擇了查詢單位，比如選擇“蜀國”後，另外三處輸入框查詢員工、開始期號、結束期號均自動生成對應的二級下拉菜單，同時查詢員工框顯示第一個員工姓名，開始期號顯示第一期工資日期，結束期號顯示最後一期工資日期。於此同時激活查詢功能，將工資信息分期列到表中並在表末進行彙總合計。

主要代碼：

Private Sub Worksheet_Activate() Call 查找單位列表(Range("d3"))End SubSub 查找單位列表(rg As Range)Set d = CreateObject("Scripting.Dictionary")For i = 1 To Sheets.Count If Not IsError(Application.Find(201, Sheets(i).Name)) Then  If Mid(Sheets(i).Name, Application.Find(201, Sheets(i).Name), 3) = "201" Then nh = Left(Sheets(i).Name, Application.Find("201", Sheets(i).Name) - 1) d(nh) = "" End If End If Next s = Join(d.keys, ",") With rg.Validation .Delete .Add Type:=xlValidateList, AlertStyle:=xlValidAlertStop, Operator:= _ xlBetween, Formula1:=s End With Set s = NothingEnd Sub

2、選擇查詢員工輸入框可通過二級下拉菜單進行選擇輸入，當員工人數較多時，下拉菜單很長，且由於菜單間距較小，為避免選錯。在表單右側列出了當前單位有工資信息的所有員工姓名，直接點擊名字，查詢員工工資。

小技巧：為了直觀顯示當前查詢工資的員工姓名，右側列表中將當期員工姓名塗色進行標記，下拉輸入框的員工姓名和右側選擇姓名同步對應，即當從下拉菜單中選擇了某員工時，右側該員工名字隨之也被塗色標記。

主要代碼：

 arr = Split(s, ",") hzbb.[k7].Resize(100, 1).ClearContents hzbb.[k7].Resize(UBound(arr) + 1, 1) = Application.Transpose(arr) hzbb.[i3] = hzbb.[k7] Call 賦色(hzbb.[k7])

小技巧：當需要按順序依次查詢工資時，為更加便捷，設置了通過點擊右側的向上、向下圖標，依次選擇員工姓名進行查詢的功能。

主要代碼：

Sub 上一個員工() Set hzbb = Sheets("彙總報表") If hzbb.[k7] = "" Then Exit Sub For i = hzbb.[k60000].End(3).Row To 7 Step -1 If hzbb.[i3] = hzbb.Cells(i, 11) Then If i = 7 Then i = hzbb.[k60000].End(3).Row + 1 hzbb.[i3] = hzbb.Cells(i - 1, 11) Exit Sub End If NextEnd SubSub 下一個員工() Set hzbb = Sheets("彙總報表") If hzbb.[k7] = "" Then Exit Sub For i = 7 To hzbb.[k60000].End(3).Row If hzbb.[i3] = hzbb.Cells(i, 11) Then If i = hzbb.[k60000].End(3).Row Then i = 6 hzbb.[i3] = hzbb.Cells(i + 1, 11) Exit Sub End If NextEnd SubSub 賦色(rg As Range) Set hzbb = Sheets("彙總報表")  hzbb.Range("K7:k15").Interior.ThemeColor = xlThemeColorLight2 hzbb.Range("K7:k15").Interior.TintAndShade = 0.599993896298105 hzbb.Range("K7:k15").Font.ColorIndex = xlAutomatic rg.Interior.ThemeColor = xlThemeColorAccent6 rg.Interior.TintAndShade = -0.249977111117893 rg.Font.ThemeColor = xlThemeColorDark1End Sub

3、右側設置自動生成圖表，圖表標題、數據內容根據左側表格數據聯動，通過選擇不同的人名，左側彙總工資和右側圖表隨之更新，實現高版本excel裡圖表的切片器功能。

Sub 顯示圖表() Set hzbb = Sheets("彙總報表") Set ab = hzbb.Range("m3:w15") Set tb = ActiveSheet.ChartObjects With tb .Visible = True .Left = ab.Left .Top = ab.Top .Width = ab.Width .Height = ab.Height ' .Name = "圖表1" End With ActiveSheet.ChartObjects(1).Activate With ActiveChart .SetSourceData Source:=hzbb.Cells(8, 2).Resize(hzbb.[b30000].End(3).Row - 7, 8) .ChartType = xlColumnClustered '選擇柱形圖樣式 .ChartStyle = 29 '選擇綠色圖表類型 .SeriesCollection(1).XValues = "=彙總報表!$D$7:$I$7" '選擇X軸項目 .ApplyLayout (5) '選擇有頂標題和下面列表的圖表類型 .ChartTitle.text = hzbb.[i3].Value '設置頂部標題 .SetElement (msoElementPrimaryValueAxisTitleNone) '取消Y軸標題 .ChartTitle.Font.Name = "楷體"End WithEnd Sub

4、設置開關按鈕控制圖表顯示或隱藏。

Private Sub onoff_Click()If onoff.Value = False Thenonoff.Caption = "圖表隱藏"ActiveSheet.ChartObjects.Visible = FalseElseonoff.Caption = "圖表顯示"ActiveSheet.ChartObjects.Visible = TrueEnd IfEnd Sub

總的來說，本表看似簡單，但細節處理很到位，自動生成一、二級下拉菜單、自動生成圖表、圖表聯動、設置切片器等功能均為實際工作中有一定難度的功能。至於主功能方面，需根據公司的具體要求專門設置，難度相對較小。
