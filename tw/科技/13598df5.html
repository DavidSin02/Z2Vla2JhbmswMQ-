安卓APP開發 | 簡單學Java從編程入門開始-基本數據類型

2020-11-14
| 科技

新手在接觸Java編程時，除了要熟悉代碼中關鍵字的使用方法外，想要操作程序中的數據，必須要了解其代碼中基本的數據類型。通過對代碼中數據類型的熟悉，我們才能知道如何去傳值和改變數據內容。

數據類型能夠對程序中的數據進行歸類，讓我們開發者對數據的內容有更精確的操作。希望大家在使用的時候能夠分清楚傳值該用哪種數據類型。

基本數據類型的定義和變量

1.什麼是基本數據類型？

基本數據其實就是某件事物的最基本信息。比如描述一個人的時候，我們可以通過他的年齡、姓名、性別、是否結婚、收入等基本信息來確定一個人的最基本信息。

我們可以看到“人”的基本數據類型：

/** 人的基本數據類型 **/public class Person{      int 年齡;     char 姓名;     boolean 是否結婚;    double 收入;}

從上面的代碼，我們可以看到人的每個基本信息中，它對應的數據類型關鍵字的變量聲明是不一樣的，比如年齡使用的是int，姓名使用的是char，不同的數據內容採用不同的數據類型。

2.什麼是基本數據的變量？

基本數據變量主要分為全局變量和局部變量。

/**個人信息**/public class Person{     int 年齡 = 13;  //全局變量    public void addAge(){			//個人更改年齡的方法函數  	年齡 = 56；			//局部變量    char 姓名 = 'L'；//局部變量  }  }

所謂的全局變量，就是在一個類中的公有變量，在類中的所有方法都可以調用這個變量。

局部變量，則是在類中某一個方法或某一位置發生數據改變的變量，它可以是全局變量的局部變化，也可以在代碼的函數中去定義變量。

基本數據類型的分類

對於Java語言而言，它的基本數據類型如下：

Java語言的基本數據類型

1.整數類型：byte、short、int、long

2.浮點類型：float、double

3.字符類型：char

4.布爾類型：boolean

根據上面的基本數據類型的分類，我們可以看到Java語言的數據類型主要分為4種。

其中，整數類型的取值為整數，比如1，2，3，4，5等；

浮點類型的取值為小數，比如1.12，3.145，3.148等；

字符類型的取值為單字符，比如'X','Y','x','y';

布爾類型的取值為布爾值，布爾值即true或false，表示是和否。

/**個人信息**/public class Person{	  byte b = 23;				//字節整數類型，取值整數，默認值為0  	short s = 3344;			//短整型，取值整數，默認值為0   	int	i = 3333;				//整型，取值整數，默認值為0	long l = 32322;			//長整型，取值整數，默認值為0L	double d = 333.33;	//雙精度浮點型，取值為小數，默認值為0.0d	float	f = 333.33f;	//單精度浮點型，取值為小數，值得後面加小寫f,默認值為0.0f		boolean b ;					//布爾類型，取值為布爾值，有判斷兩面性，默認為false  }

注意：布爾類型的值一般在沒有賦值的情況下，默認值為false.

結語

在進行Java語言編程時，程序中很多代碼都是圍繞數據傳遞進行，這要求我們編程人員需要對代碼中的數據類型要非常熟悉才行，保證傳遞的數據內容能被正常傳遞。我們在後續的數據庫學習中，也要會定義表中字段的數據類型，保證數據庫服務端和程序中的數據類型必須一致。

Java語言的基本數據類型總共就8種，在使用時要加以區別，以免關鍵字的變量聲明錯誤，希望大家能夠認真瞭解Java語言的基本數據類型，這是數據傳遞的最核心部分。
