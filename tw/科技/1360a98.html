聖美歌EA400.4S四路功放 讓車友越來越燒得起的“純後級”高端貨

2020-10-29
| 科技

DSP作為音頻處理器以及DSP功放推動系統的玩法越來越盛行，音改市場上的整體玩法開始走向後級處理器設計或搭配上的簡化，也就是把前級處理部分全交由DSP處理。不少老江湖既明白車廂環境實現理想音頻的複雜化，同時也深知對於功放來說前級調控越少距離發燒就越近。市場上不少單路和二路功放早已出現純後級產品，而為了照顧通用性四路功放就少見這類產品。

為照顧車友或者愛好者對發燒汽車音響的追求，SOUNDMAGUS聖美歌早段時間就推出了M1純後級Hi-End單路放大器，音質更是不容置疑地得到認可。貫徹讓車友越來越燒得起的思路，聖美歌最近也推出旗下EA系列高端發燒音頻功放的改進版——全新的EA400.4S四路功放。

EA400.4S在輸入端和電路方面的設計做出了不少改進，不能說其直接以純後級方向發展，但對比“前者”EA400.4來說，精簡優化讓功放變得更為高效，也更貼近車友最理想的發燒功放趨向。功放的整體設計沒有大面積升級改進，保留了聖美歌EA系列的黑色拉絲面板設計，系列十分優秀高效的體積得以保留，接近一張A4紙大小的體積是EA功放一大亮點。

從輸入端看，我們會發現EA400.4S對比老款優化精簡了不少。左側是四聲道的RCA輸入端子，以及四/二聲道輸入轉換按鈕，按下功放只進行二聲道輸入，不按下就是一臺四路，原來在下方的輸入電平按鈕取消。功放設計為純以RCA低電平輸入信號，所以老款上的REM和Audio等自動信號開關也被取消。右側的1、2聲道調節上，可以選擇全通或者高通，頻響範圍在50Hz-500Hz可調以及GAIN增益可調。下側的3、4聲道調節上，可選高通、全通和帶通，高通全通頻響為0Hz-300Hz，帶通則在50Hz-500Hz之間可調，同樣提供GAIN增益可調，此外還預留了超低音線控插口。

對比老款EA400.4輸入端

EA400.4S連兩側的金屬倒流散熱格柵也與老款同出一徹，拆開底板會發現內部電路是用上雙面玻纖電路板，以SMD音頻專用貼片元件保證超低內阻內擾以及維持高保真輸出。功放在4Ω狀態下可提供100W每通道的輸出，2Ω下為每通道120W，進行橋接輸出則可達兩通道220W。大型的“牛”使用MCU控制數字電源平臺，保證大動態輸出也有足夠的電流穩定供應。附近分佈的則是大容量電解電容，以及MOSFET場效應管。應對AB類電路加上發燒音質處理上可能產生的高熱量，EA400.4S加入完善保護功能讓功放在過載、欠壓、過壓以及過熱等情況下也能穩定工作。

360汽車網點評：樂迷都懂後級處理越是減少前級各種調控和處理，越是“簡潔”的電路處理出來的效果越能接近理想的發燒音質。聖美歌EA400.4S設計上就為了更好推動四聲道的揚聲器，特別越發流行的全主動分頻玩法。為顧及玩法多樣，特別針對超低音橋接輸出設計，一臺功放完全玩轉2.0、2.1聲道系統，對應市場、對應車廂環境、對應音質訴求，三方面設計得出平衡。

SOUNDMAGUS聖美歌EA400.4S四路功放參數 / 輸出功率：100W×4@4Ω、120W×4@2Ω / 橋接功率：220W×2Ω / THD失真：0.08%@1kHz、4Ω / 頻率響應：20Hz-20kHz（-3dB） / 信噪比：＞98dB / 1、2通道濾波範圍：50Hz-500Hz（全通、高通） / 3、4通道濾波範圍：0Hz-300Hz（全通、高通）、50Hz-500Hz（帶通） / 阻尼係數：＞50@1kHz、4Ω / 負載阻抗：4聲道@2Ω、2聲道@4Ω / 最大音樂功率輸出電流：33A@14.4V / 尺寸：188mm×56mm×320mm
