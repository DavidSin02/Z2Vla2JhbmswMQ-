Tkinter-創建窗口實例

2020-10-29
| 科技

主要介紹了Tkinter各組件類之間繼承關係，能夠更好的瞭解Tkinter組件，同時列舉兩個Tkinter入門示例，簡單快速構建圖形界面程序!

Tkinter組件繼承關係

在創建Tkinter實例之前，先說一下Tkinter組件之間繼承關係


Tkinter組件繼承關係

說明：

  • Tkinter 有兩個根父類，它們直接繼承了object類
  • Misc：所有組件的根父類
  • Wm：提供了一些與窗口管理器通信的功能函數
  • 對於 Misc 和 Wm 兩個基類而言，它們是所有Tkinter組件的父類，所有組件都可以直接使用它們的方法
  • Tk：是Misc 和 Wm 派生的子類 ，它是應用程序的主窗口，Tkinter GUI編程通常都需要直接或間接使用Tk類
  • BaseWidget：是所有組件的基類
  • Widget：代表一個通用的 GUI 組件，它繼承於BaseWidget父類，所有組件都是 Widget 的子類
  • Pack、Place、Grid：是佈局管理器類，負責管理所包含的組件的大小和位置擺放，並派生出來Widget子類

組件介紹：

窗口、容器、組件關係

任何窗口可由若干個容器組成，而容器裡填充著各種基本組件，通過設置基本組件的屬性（大小、位置等），便可形成有交互能力用戶圖形界面

  • 一個窗口由一個或多個容器組成
  • 每個容器又由一個或多個組件組成
  • 窗口、容器、組件構成了一個完整的用戶圖形界面

創建Tk窗口示例

# -*- coding:utf-8 -*-from Tkinter import * if __name__ == "__main__": # 創建Tk對象，即主窗口 mw = Tk() # 設置窗口標題 mw.title('Tkinter入門示例') # 創建Label標籤，第一個參數意思是，將label標籤對象加入主窗口mw中 w = Label(mw, text="Hello World!Hello Tkinter!") # 佈局設置 w.pack() # 啟動主窗口循環 mw.mainloop()

示例代碼創建了兩個對象Tk 和 Label，Tk是主窗口，Label是文本標籤，將Label對象加入到Tk主窗口中

有關Tkinter包導入，在Python 2.x 使用from Tkinter import *在Python 3.x 使用from Tkinter import *

另一種方式是不直接使用Tk，只要創建Frame的子類，其子類會自動創建Tk對象作為窗口，示例如下：

# -*- coding:utf-8 -*-from Tkinter import *class App(Frame): def __init__(self, master=None): Frame.__init__(self, master) self.pack() # 調用initWidgets()初始化界面 self.initWidgets() def initWidgets(self): # 創建Label組件，第一個參數將該Label放入主窗口 w = Label(self) # 創建位圖 self.bm = PhotoImage(file = 'logo.png') # 設置Label背景圖片 w['image'] = self.bm w.pack() # 創建Button組件，第一個參數將該Button放入主窗口 okButton = Button(self, text="確定") # 設置Button的背景色 okButton['background'] = 'green' okButton.pack()if __name__ == "__main__": # 創建App對象 app = App() # Frame有個默認的master屬性，該屬性值是Tk對象（窗口） print(type(app.master)) # 設置窗口標題 app.master.title('Tkinter入門示例') # 啟動主窗口的消息循環 app.mainloop()

補充說明：

  • 初始化窗口分三個過程：創建組件、添加組件、配置組件
  • 配置組件，可以在初始化配置，也可以在創建完成後配置，如：
  • 初始化配置 okbtn = Button(self, text="確定")
  • 創建後配置 okbtn['text'] = "確定"，以字典形式配置，完全等價
  • 還可以這樣配置 okbtn.configure(text = "確定"), 完全等價，自由切換使用
  • 創建GUI組件很簡單，需要至少指定一個參數，用於指定GUI組件屬於哪個容器（Tk組件例外，因為該組件代表頂級窗口）
  • 程序在創建任意Widget組件時，沒有指定master屬性（即創建Widget組件時第一個參數傳入None），那麼程序會自動為該Widget組件創建一個Tk窗口
