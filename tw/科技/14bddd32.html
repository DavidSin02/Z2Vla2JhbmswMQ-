R&S羅德與施瓦茨信號發生器自檢自校準方法彙總-安泰維修中心

2020-11-14
| 科技

R&S羅德與施瓦茨信號發生器在信號質量、靈活性和可用性方面表現優異。因此定期就要對儀器進行自檢自檢準操作，安泰信號發生器維修中心分享羅德與施瓦茨信號發生器自檢自校準方法：


信號發生器
一、SMW200A/SMBV100B/SMA100B/SMB100B
<1>自校準
System Config > Setup > General > Internal Adjustments>Adjust All
<2>固件檢查、升級或修復
System Config > Setup > Instrument Assembly > Versions / Options

>Firmware>FW version顯示firmware版本。

存儲firmware的U盤插在信號源上開機啟動，即可自動安裝。

二、SMBV100A/SMA100A/SMB100A/SMF100A
<1>自校準
[Setup] >System > Internal Adjustments>Adjust All
<2>固件檢查、升級或修復
[Setup]>Software / Options顯示firmware版本。
存儲firmware的U盤插在信號源上開機啟動，即可自動安裝。

以上有關信號發生器自檢自校準操作方法由西安安泰信號發生器維修中心整理髮布，更多有關儀器維修知識歡迎訪問西安安泰儀器維修網，關注公眾號：安泰測試
