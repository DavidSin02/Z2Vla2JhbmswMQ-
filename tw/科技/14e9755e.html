大理石、花崗石及碎拼大理石地面 施工工藝標準

2020-10-29
| 科技

範圍

本工藝標準適用於民用建築的大理石、花崗石和碎拼大理石地面面層（大理石板材不得用於室外地面面層）。

施工準備

2.1 材料及主要機具；

2.1.1 天然大理石、花崗石的品種、規格應符合設計要求，技術等級、光澤度、外觀質量要求，應符合國家標準《天然大理石建築板材》、《花崗石建築板材》的規定，其允許偏差和外觀要求見表7-12。

2.2 作業條件：

2.2.1 大理石、花崗石板塊進場後，應側立堆放在室內光面相對、背面墊松木條，並在板下加墊水方。拆箱後詳細核對品種、規格、數量等是否符合設計要求，有裂紋、缺稜、掉角、翹曲和表面有缺陷時，應予剔除。

2.2.2 搭設好加工棚，安裝好臺鑽及砂輪鋸，並接通水電源。

2.2.3 室內抹灰（包括立門口）、地面墊層、預埋在墊層內的電管及穿通地面的管線均已完成。

2.2.4 房間內四周牆上彈好+50cm水平線。

2.2.5 施工操作前應畫出鋪設大理石地面的施工大樣圖。

2.2.6 冬期施工時操作溫度不得低於5℃。

操作工藝

3.1 工藝流程：

準備工作 → 試拼 → 彈線 → 試排 → 刷水泥漿及鋪砂漿結合層 →

鋪大理石板塊 (或花崗石板塊) → 灌縫、擦縫 → 打蠟

3.2 準備工作：

3.2.1 以施工大樣圖和加工單為依據，熟悉瞭解各部位尺寸和作法，弄清洞口、邊角等部位之間的關係。

3.2.2 基層處理：將地面墊層上的雜物清淨，用鋼絲刷刷掉粘結在墊層上的砂漿，並清掃乾淨。

3.3 試拼：在正式鋪設前，對每一房間的大理石（或花崗石）板塊，應技圖案、顏色、紋理試拼，將非整塊板對稱排放在房門靠牆部位，試拼後按兩個方向編號排列，然後按編號碼放整齊。

3.4 彈線：為了檢查和控制大理石（或花崗石）板塊的位置，在房間內拉十字控制線，彈在混凝土墊層上，並引至牆面底部，然後依據牆面+50cm標高線找出面層標高，在牆上彈出水平標高線，彈水平線時要注意室內與樓道面層標高要一致。

3.5 試排：在房間內的兩個相互垂直的方向鋪兩條幹砂，其寬度大於板塊寬度，厚度不小於3cm、結合施工大樣圖及房間實際尺寸，把大理石（或花崗石）板塊排好，以便檢查板塊之間的縫隙，核對板塊與牆面、柱、洞口等部位的相對位置。

3.6 刷水泥素漿及輔砂漿結合層：試鋪後將於砂和板塊移開，清掃乾淨，用噴壺灑水溼潤，刷一層素水泥漿（水灰比為0.4～0.5，不要刷的面積過大，隨鋪砂漿隨刷）。根據板面水平線確定結合層砂漿厚度，拉十字控制線，開始鋪結合層乾硬性水泥砂漿（一般採用1∶2～1∶3的乾硬性水泥砂漿，乾硬程度以手捏成團，落地即散為宜），厚度控制在放上大理石（或花崗石）板塊時宜高出面層水平線3～4mm。鋪好後用大槓刮平，再用抹子拍實找平（鋪攤面積不得過大）。

3.7 鋪砌大理石（或花崗石）板塊：

3.7.1 板塊應先用水浸溼，待擦乾或表面晾乾後方可鋪設。

3.7.2 根據房間拉的十字控制線，縱橫各鋪一行，做為大面積鋪砌標筋用。依據試拼時的編號、圖案及試排時的縫隙（板塊之間的縫隙寬度，當設計無規定時不應大於lmm），在十字控制線交點開始鋪砌。先試鋪即搬起板塊對好縱橫控制線鋪落在已鋪好的乾硬性砂漿結合層上，用橡皮錘敲擊木墊板（不得用橡皮錘或木錘直接敲擊板塊），振實砂漿至鋪設高度後，將板塊掀起移至一旁，檢查砂漿表面與板塊之間是否相吻合如發現有空虛之處，應用砂漿填補，然後正式鑲鋪，先在水泥砂漿結合層上滿澆一層水灰比為0.5的素水泥漿（用漿壺澆均勻），再鋪板塊，安放時四角同時往下落，用橡皮錘或木錘輕擊木墊板，根據水平線用鐵水平尺找平，鋪完第一塊，向兩側和後退方向順序鋪砌。鋪完縱，橫行之後有了標準，可分段分區依次鋪砌，一般房間直尤里後外進行，逐步退至門口，便於成品保護，但必須注意與樓道相呼應。也可從門口處往裡鋪砌，板塊與牆角、鑲邊和靠牆處應緊密砌合，不得有空隙。

3.8 灌縫、擦縫：在板塊鋪砌後l～2晝夜進行灌漿擦縫。根據大理石（或花崗石）顏色，選擇相同顏色礦物顏料和水泥（或白水泥）拌合均勻，調成1∶1稀水泥漿，用漿壺徐徐灌入板塊之間的縫隙中（可分幾次進行），並用長把刮極把流出的水泥漿刮向縫隙內，至基本灌滿為止。灌漿l～2h後，用棉紗團蘸原稀水泥漿擦縫與板面擦平，同時將板面上水泥漿擦淨，使大理石（或花崗石）面層的表面潔淨、平整、堅實，以上工序完成後，面層加以覆蓋。養護時間不應小於7d。

3.9 打蠟：當水泥砂漿結合層達到強度後（抗壓強度達到l.2MPa時），方可進行打蠟，打蠟方法詳見現制水磨正地面面層工藝標準。打蠟後面層達到光滑潔亮。

3.10 大理石（或花崗石）踢腳板工藝流程①：

3.10.1 粘貼法：

找標高水平線 → 水泥砂漿打底 → 貼大理石踢腳板 → 擦縫 → 打蠟

3.10.1.1 根據主牆+50cm標高線，測出踢腳板上口水平線，彈在牆上，再用線墜吊線確定出踢腳板的出牆厚度，一般8～10mm。

3.10.1.2 用1∶3水泥砂漿打底找平，並在面層劃紋。

3.10.1.3 找平層砂漿乾硬後，拉踢腳板上口的水平線，把浸水陰乾的大理石（或花崗岩）踢腳板的背面，刮抹一層2～3mm厚的素水泥漿（宜加10%左右的107膠）後，往底灰上粘貼，並用木錘敲實，根據水平線找直。

3.10.1.4 24h以後用同色水泥漿擦縫，用棉絲團將餘漿擦淨。

3.10.1.5 打蠟：詳見現制水磨石地面施工工藝標準。

3.10.2 灌漿法

找標高水平線 → 拉水平通線 → 安裝踢腳板 → 灌水泥砂漿 → 擦縫 → 打蠟

3.10.2.1 根據主牆+50cm標高線，測出踢腳板上口水平線，彈在牆上，再用線墜吊線，確定出踢腳板的出牆厚度，一般8～10mm。

3.10.2.2 拉踢腳板上口水平線，在牆兩端各安裝一塊踢腳板，其上楞高度在同一水平線內，出牆厚度要一致，然後逐快依順序安裝，隨時檢查踢腳板的水平度和垂直度。相鄰兩塊之間及踢腳板與地面、牆面之間用石膏穩牢。

3.10.2.3 灌1∶2稀水泥砂漿，並隨時把溢出的砂漿擦乾淨，待灌入的水泥砂漿終凝後，把石膏剷掉。

①踢腳線宜在地面完成後施工。

3.10.2.4 用棉絲團蘸與大理石踢腳板同顏色的稀水泥漿擦縫。

3.10.2.5 踢腳板的面層打蠟同地面一起進行，方法同現制水磨石地面施工工藝標準。

鑲貼踢腳板時，板縫宜與地面的大理石（或花崗石）板對縫鑲貼。

3.11 碎拼大理石面層施工工藝流程：

挑選碎塊大理石 → 彈線試拼 → 基層清理 → 掃水泥素漿 → 鋪砂漿結合層 →

鋪大理石碎塊 → 灌縫 → 磨光打蠟

3.11.1 根據設計要求的顏色、規格挑選碎塊大理石，要薄厚一致，不得有裂縫。

3.11.2 根據設計要求的圖案，結合房間尺寸，在基層上彈線並找出面層標高，然後進行試拼，確定縫隙的大小。

3.11.3 清理基層，必須將粘結在基層上的灰漿層、塵土清掃乾淨，然後灑水溼潤，掃水泥素漿（隨刷水泥漿隨鋪砂漿）。

3.11.4 彈水平標高線，開始鋪砂漿結合層，採用1∶3乾硬性水泥砂漿（手捏成團，一顛即散），鋪好後用大槓刮平，木抹子拍實抹平。

3.11.5 根據圖案和試拼的縫隙鋪砌大理石碎塊，其方法同大理石板塊地面。

3.11.6 鋪砌1～2晝夜後進行灌縫，根據設計要求，碎塊之內間隙灌水泥砂漿時，厚度與大理石塊上面層幹，並將其表面找平壓光。如果設計要求間隙灌水泥石渣漿時，灌漿厚度比大理石碎塊上面層高出2mm厚。養護時間不少於7d。

3.11.7 如果間隙灌水泥石渣漿時，養護後需進行磨光和打蠟，其操作工藝同現制水磨石地面施工工藝標準。

質量標準

4.1 保證項目：

4.1.1 面層所用板塊品種、規格、級別、形狀、光潔度、顏色和圖案必須符合設計要求。

4.1.2 面層與基層必須結合牢固，無空鼓。

4.2 基本項目：

4.2.1 面層

4.2.1.1 磨光大理五和花崗石板塊面層；板塊擠靠嚴密，無縫隙，接縫通直無錯縫，表面平整潔淨，圖案清晰無磨劃痕，周邊順直方正。

4.2.1.2 碎拼大理石面層：顏色協調，間隙適宜美觀，磨光一致，無裂縫和磨紋，表面平整光潔。

4.2.2 板塊鑲貼質量：任何一處獨立空間的石板顏色一致，花紋通順基本一致。石板縫痕與石板顏色一致，擦縫飽滿與石板齊平，潔淨、美觀。

4.2.3 踢腳板鋪設質量：排列有序，擠靠嚴密不顯縫隙，表面潔淨，顏色一致，結合牢固，出牆高度、厚度一致，上口平直。

4.2.4 地面鑲邊鋪設質量：

4.2.4.1 花崗石、大理石板面層：用料尺寸準確，邊角整齊，拼接嚴密，接縫順直。

4.2.4.2 碎拼大理石面層：尺寸正確，拼接嚴密，相鄰處不混色，分色線順直，邊角齊整光滑、清晰美觀。

4.2.5 地漏坡度符合設計要求，不倒泛水，無積水，與地漏結合處嚴密牢固，無滲漏（有坡度的面層應做潑水檢驗，並以能排除液體為合格）。

4.2.6 打蠟質量：大理石、花崗石和碎排大理石地面燙硬蠟、擦軟蠟，蠟灑布均勻不露底，色澤一致、厚薄均勻、圖紋清晰、表面潔淨。

成品保護

5.1 運輸大理石（或花崗石）板塊和水泥砂漿時，應採取措施防止碰撞已做完的牆面、門口等。

5.2 鋪砌大理石（或花崗石）板塊及碎拼大理石板塊過程中，操作入員應做到隨鋪隨用乾布揩淨大理石面上的水泥漿痕跡。

5.3 在大理石（或花崗石）地面或碎拼大理石地面上行走時，找平層水泥砂漿的抗壓強度不得低於1.2MPa。

5.4 大理石（或花崗石）地面或碎拼大理石地面完工後，房間應封閉或在其表面加以覆蓋保護。

應注意的質量問題

6.1 板面空鼓：由於混凝土墊層清理不淨或澆水溼潤不夠，刷素水泥漿不均勻或刷的面積過大、時間過長已風乾，乾硬性水泥砂漿任意加水，大理石板面有浮土末浸水溼潤等等因素，都易引起空鼓。因此必須嚴格遵守操作工藝要求，基層必須清理乾淨，結合層砂漿不得加水，隨鋪隨刷一層水泥漿，大理石板塊在鋪砌前必須浸水溼潤。

6.2 接縫高低木平、縫子寬窄不勻；主要原因是板塊本身有厚薄及寬窄不勻、竄角、翹曲等缺陷，鋪砌時未嚴格拉通線進行控制等因素，均易產生接縫高低不平、縫子不勻等缺陷。所以應預先嚴格挑選板塊，凡是翹曲、拱背、寬窄不方正等塊材剔除不予使用。鋪設標準塊後，應向兩側和後退方向順序鋪設，並隨時用水平尺和直尺找準，縫子必須拉通線不能有偏差。房間內的標高線要有專人負責引入，且各房間和樓道內的標高必須相通一致。

6.3 過門口處板塊易活動：一般鋪砌板塊時均從門框以內操作，而門框以外與樓道相接的空隙（即牆寬範圍內）面積均後鋪砌，由於過早上人，易造成此處活動。在進行板塊翻樣提加工定貨時，應同時考慮此處的板塊尺寸，並同時加工，以便鋪砌樓道地面板塊時同時操作。

6.4 踢腳板不順直，出牆厚度不一致：主要由於牆面平整度和垂直度不符合要求，鑲踢腳板時未吊線、未拉水平線，隨牆面鑲貼所造成。在鑲踢腳板前，必須先檢查牆面的垂直度、平整度，如超出偏差，應先進行處理後再鑲貼。
