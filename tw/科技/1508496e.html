2．維持阻塞D觸發器

工作原理：

1） CP = 0時， G3、G4被封鎖，觸發器維持原態不變。

2） CP的上升沿到來時：

① D = 0，則Q5 = 1， Q6= 0， Q3 = 0， Q4 = 1，觸發器置“0”， Q = 0，

Q 非 = 1； G3輸出端到G5輸入端的連線叫做“置0維持線”。

② D =1，則Q5 = 0， Q3 = 1，Q6= 1， Q4= 0，觸發器置“1”， Q = 1，

Q 非 = 0；G4輸出端到G6輸入端的連線叫做“置1維持線”。

維持阻塞觸發器不會有空翻現象

維持阻塞D觸發器就是D觸發器。由於是上升沿觸發，又稱為邊沿D觸發器。

4．測試電路

（1）對照測試線路圖和裝配圖進行檢查，仔細檢查電路中各電路是否安裝正確，導線、焊點是否符合要求，檢查有極性器件是否安裝並連接正確。

（2）用萬用表R×1擋測電源與地之間的電阻。發現短路，應先檢查，排除短路點。

（3）檢查無誤後，按集成電路標記口的方向插上集成電路，方可通電測試。
