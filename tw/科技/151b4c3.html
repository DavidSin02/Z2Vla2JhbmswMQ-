EXCEL打印預覽中如何做到自動精準分頁

2020-10-29
| 科技

很多excel工作者在打印預覽多頁內容時，發現往往有分頁線不精準的問題，於是只能手工調整。頁數少的話還可以，但如果頁數很多，手工調整起來費時費力。朋友們看下面表一的E列中分頁線沒有處於第13考場和第12考場之間。這時我們把鼠標放到分頁線上，按下鼠標左鍵分頁線拖到第12考場和第13考場之間，即把分頁線放到第32行的上面。

表一

這時Excel會隨之自動把下面的分頁線作智能改變，也會向下移動，但是大多數情況下，每個班的人數是不一樣的，當我們把滾動條往下拉時，發現第12考場和第11考場的分頁線並沒有處於我們理想的位置，而是如下表這樣的：

表二

怎麼辦？不用說，再作手工調整。然後滾動條再往下拉，發現第11考場和第10考場的分頁線也沒有正好處於兩考場中間：

表三

怎麼辦？好像沒有別的辦法。難以想象如果大點的企業報表，動則上百頁的打印，難道也這樣手工調整啊！費時費力不說，關鍵是眼花呀，一旦弄錯事情可大可小啊！

其實利用excel的VBA代碼可以輕易解決這個問題。下面跟小蕃一起來看看吧。什麼？你不懂VBA，沒有聽說過？沒有關係啦。如果不想去理解，會複製粘貼就能實現功能的了。如果是高手就繞過吧。

第一、進行使用VBA代碼前的準備操作：

1、 點擊“文件”—“選項”---自定義功能區，在右側窗口中找到“開發者工具”，在它前面小方格中打上“√”，點確定。

2、 點擊“文件“—”選項“---信任中心---信任中心設置，找到”宏設置“----選”啟用所有宏“。注意，一般Excel是禁用宏的，剛才這部是啟用宏，允許程序運行VBA代碼。為了讓剛才的設置起作用，關閉EXCEL程序，重啟程序並打開剛才的文件。

第二、複製代碼實現精準分頁：

點擊“開發工具“下的最左邊的”Visal Basic”按鈕，就打開了VBA代碼編輯器，如下圖：

VBA編輯器窗口

1、 點擊“代碼編輯窗口“左邊的“VBAProject(工作簿1)“，再在上面點擊鼠標右鍵，選擇”插入“---”模塊”，然後複製下面的20行代碼，粘貼到到“代碼編輯窗口“右邊的空白區域：

Sub 按考場自動精準分頁( )

'一、定義變量和初始值

Dim i, j, k, t1, t

j = 1

i = 4

k = 2

'二、取得佔有單元格的列數和行數

t1 = ActiveSheet.UsedRange.Columns.count '取得列數

t = ActiveSheet.UsedRange.Rows.count '取得行數

'三、先設置打印預覽

ActiveWindow.View = xlPageBreakPreview

'四、設置按考場調整分頁符，即同一考場打印在同一頁內

Do While Cells(i, 5) <> ""

If Cells(i, 5) <> Cells(i - 1, 5) Then

Set ActiveSheet.HPageBreaks(j).Location = Cells(i, 1)

j = j + 1

End If

i = i + 1

Loop

End Sub

3、【特別提醒】本例中的考試號在第5列，也就是E列，如果朋友們的工作表中，排序基準列不是E列，比如說在H列，那麼H列在EXCEL中是就是左邊數過來第8列。這時你要把上面代碼中的第四部分中的數字“5”，改成“8”。改好後，也別急著去執行代碼，還要做下面兩步。

第三、按考場號排序。小蕃這裡用的降序排序，所以大號的考場在前，號小的考場在後。

第四、點擊“視圖”菜單下的“分頁預覽”。你會發現分頁線沒有正好處於不同的考場之間。沒關係，接下來該讓VBA代碼出來工作啦。

第五、執行代碼。點擊“開發工具”菜單下，左邊第二個菜單---“宏”。在跳出的宏窗口，點擊“按考場自動精準分頁”，再點右邊的“執行”按鈕。再去觀察一下分頁線，每一條分頁線都準確地放在了兩個考場之間，不管是有兩頁，還是兩百頁，都一鍵搞定！
