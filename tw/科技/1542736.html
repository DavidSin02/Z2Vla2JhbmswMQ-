建築幕牆簡述

特點

1.1 建築幕牆它是外圍護結構，建築幕牆自身具有完整的結構系統，通常包封主體結構。

1.2 在自身平面內可承受較大的變形，相對主體結構有足夠的變形能力。

1.3 只承受本身的荷載且傳給主體結構，不分擔、也不傳遞主體結構的荷載作用。

1.4 重量輕，節省主體結構與基礎的費用。

1.5 保護主體結構溫度變化時應力小。

1.6 維修簡單可替換性強。

幕牆的種類

按建築幕牆的面板可分為玻璃幕牆（ 明框玻璃幕牆、隱框玻璃幕牆、半隱框玻璃幕牆、 全玻璃幕牆、點支承玻璃幕牆。）、金屬幕牆、石材幕牆、 陶板幕牆等。

在工廠組裝為幕牆單元，以幕牆單元形式在現場安裝施工的框支承玻璃幕牆。

構件式玻璃幕牆：在現場依次安裝立柱、橫樑和玻璃面板的框支承玻璃幕牆

構件式玻璃幕牆是玻璃幕牆的一種分類形式，指在工廠製作的是一根根元件( ( 立柱、橫樑) ) 和一塊

塊玻璃( ( 組件) ) ，再運往工地將立柱用連接件安裝在主體結構上，再在立柱上安裝橫樑，形成幕牆

框格後安裝固定玻璃( ( 組件) ) 。 一般分為構件式明框玻璃幕牆、構件式隱框玻璃幕牆、構件式半隱框玻璃幕牆等幾種形式。
