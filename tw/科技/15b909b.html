給定文字生成摘要

2020-10-29
| 科技

Writing-editing-Network，主要是通過編輯機制來生成論文摘要。

依賴：

•Pytorch 0.4

•Python 3.6

所用數據是•ACL_titles_abstracts_dataset

如何開始：

先進行預處理，把acl_titles_and_abstracts.txt放到此代碼的目錄下。

並通過runing split_data.py來隨機拆分訓練，驗證，測試的數據集。

訓練：可以在main.py中的config類來調整超參數

python main.py --cuda --mode 0

驗證：python main.py --cuda --mode 3

預測: python main.py --cuda --mode 1
