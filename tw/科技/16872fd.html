配電網架空線路施工檢修工藝

2020-10-29
| 科技

1、施工前現場檢查



（1）鋼筋混凝土電杆表面應光潔平整，壁厚均勻，無露筋、偏筋、漏漿、掉塊等現象。

（2）鋼筋混凝土電杆杆頂應封堵。

（3）鋼管杆及附件均應熱鍍鋅，鋅層均勻，無漏鋅、鋅渣、鋅刺。

（4）鐵塔塔材尺寸、螺栓孔允許偏差合格，鍍鋅表面應連續、完整，不應有過酸洗、起皮、漏鍍等使用上有害的缺陷。

a：混凝土電杆杆身應無縱向裂紋，橫向裂紋寬度不應大於0.1mm，長度不允許超過1/3周長，且1000mm內橫向裂紋不超過3處。

b ：鋼筋混凝土電杆杆身彎曲不超過杆長的1/1000。

c ：鋼管杆整根及各段的彎曲度不超過其長度的2/1000。

2、電杆組立

2.1

立杆機立杆法



（1）杆坑應位於立杆機內重心適當位置，支點堅固、支腿平穩。

（2）起吊繩應系在電杆重心偏上位置。

（3）操作手搖捲揚機時應用力均勻。

a：立杆機法只能用於豎立12m 及以下的拔梢杆。

b：用直徑約 25mm、長度超過杆長 1.5 倍的棕繩或具有足夠強度的線繩作為拉繩，綁紮應採用活結。

注意：電杆組立應儘量避免穿越樹林、魚塘、跨越或貼近居民住房，確需穿越林區的配電線路應採用獨立耐張段（不超過3檔）並設置防風拉。原則上應滿足線路兩側邊線向外延伸5米以外的淨空通道。

2.2

汽車起重機立杆法



（1）立杆時，汽車開到距基坑口適當位置；一般起吊時，吊臂和地

面的垂線成30°夾角。

（2）放下汽車起重機的液壓支撐腿時，應使汽車輪胎不受力；將吊

點置於電杆的重心偏上處，進行吊立電杆。

（3）電杆起立後，應及時調整杆位，使其符合立杆質量的要求，

然後進行回填土。

應在回填夯實並確保杆身穩固後鬆開起吊繩索。

2.3

固定人字抱杆立杆法



（1）兩抱杆根位於坑口兩側，前後錨樁與人字抱杆頂點杆坑中心在同一垂直面上。打好前後臨時拉線和絞磨的樁錨。

（2）起立過程中電杆重心應在基坑中心。

（3）電杆入坑後應校直電杆，並立即分層夯實回填土。

a：抱杆長度一般可取杆塔重 心高度加 1500～2000mm，臨時拉線樁到杆坑中心距離，可取杆塔高度 1.2～1.5 倍。抱杆的根開根據實際經驗在 2000～3000mm 範圍內。

b：滑輪組應根據被吊電杆重量決定。

c：抱杆起立到 70°左右應放 慢動作，調節好前後橫繩。起立 80°時，停止牽引，用臨時拉線調整杆塔。

2.4

電杆組立質量檢查



電杆立好後應正直，沿線電杆在一條直線上，位置偏差符合設計或規範要求。電杆回填、埋深符合要求。

a：直線杆的橫向位移不大於50mm。

b ：直線杆杆梢的位移不大於杆梢直徑的1/2。

c ：轉角杆的橫向位移不大於50mm。

d ：轉角杆組立後，杆根向內角的偏移不大於50mm、不能向外角

偏移。杆梢應向外角方向傾斜，但不得超過一個杆梢直徑，不允許

向內角方向傾斜。

e ：終端杆立好後，應向拉線側預偏，其預偏值不應大於杆梢直徑，緊線後不應向受力側傾斜。

f ：回填土每升高300mm夯實一次，防沉土臺高出地面300mm。

注：電杆組立後線路橫向邁步不超過5公分。 同杆架設和同線路走向的架空配電線路，嚴禁在同一通道內組立低於通道內兩側電杆等級的水泥杆架設低壓線路。如下圖， 如新建線路穿越或跨越其他配電線路應在交叉點組立適當的電杆做支撐且滿足線間最小垂直距離10千伏和10千伏最小垂直距離0.8米。10千伏和0.4千伏最小垂直距離1.2米的要求。




3、橫擔安裝




（1）橫擔安裝應平正，安裝偏差應符合設計或規範要求。

（2）架空線路所採用的鐵橫擔、鐵附件均應熱鍍鋅。檢修時，若有嚴重鏽蝕、變形應予更換。

（3）同杆架設線路橫擔間的最小垂直距離應滿足下表要求（單位：mm）。



備註：（）內為同杆架設的絕緣線路適用數據。

a：橫擔端部上下歪斜不大於20mm，左右扭斜不大於20mm；雙杆橫擔與電杆連接處的高差不大於連接距離的5/1000，左右扭斜不大於橫擔長度的1/100。

b：瓷橫擔絕緣子直立安裝時，頂端順線路歪斜不大於10mm；水平安裝時，頂端宜向上翹起5°～15°，頂端順線路歪斜不大於20mm。

c：當安裝於轉角杆時，頂端豎直安裝的瓷橫擔支架應安裝在轉角的內角側（瓷橫擔應裝在支架的外角側）。

d：對原有單側雙橫擔加強方式進行檢修，直線杆橫擔應裝於受電側，90°轉角杆及終端杆應裝在拉線側，轉角杆應裝在合力位置方向。新安裝橫擔應為水平加強橫擔方式。

4、拉線製作與安裝

4.1

一般拉線製作與安裝

4.1.1、拉線製作



當拉線連接金具採用楔形線夾、UT型線夾或鋼卡時：

（1）鋼絞線的尾線應在線夾舌板的凸肚側，尾線留取長度應為300～500mm。

（2）鋼絞線在舌板迴轉部分應留有縫隙，並不應有鬆股。

（3）鋼絞線的尾線在距線頭50mm處綁紮，綁紮長度應為50～80mm。

（4）鋼絞線端頭彎回後應用鍍鋅鐵線綁紮緊。

（6）嚴重腐蝕地區拉線棒應進行防腐處理。

a：拉線一般採用多股鍍鋅鋼絞線，其規格為GJ-35～100。

b ：鋼絞線剪斷前應用細鐵絲綁紮好。

c ：拉線時應明確主、副線方向。

d ：同組拉線使用兩個線夾時，則線夾尾線端的方向應統一。

e ：拉線上把和拉線抱箍連接處採用延長環連接。

f ：防腐處理，防護部位：自地下500mm至地上200mm處；防護措施：塗瀝青，纏麻袋片兩層，再刷防腐油。

4.1.2、拉線安裝




（1）拉線應採用專用的拉線抱箍。

（2）拉線抱箍一般裝設在相對應的橫擔下方，距橫擔中心線100mm處。

（3）拉線的收緊應採用緊線器進行。

（4）根據需要加裝拉線絕緣子。

（5）拉線底把應採用熱鍍鋅拉線棒，安全係數不小於3，最小直徑不應小於16mm。

（6）拉線宜加裝警示標識。

（7）拉線地錨必須安裝在地面或現澆混凝土構件上（樑、柱），安裝在牆上的必須做防鏽處理。

（8）同一方向多層拉線的拉錨應不共點，保證有兩個或兩個以上拉錨。

（9）拉線地錨應埋設端正，不得有偏斜，地錨的拉線盤與拉線垂直。

a ：楔形線夾的螺栓與延長環連接好後R型銷針的開口在30～60°。

b ：線夾舌板與拉線接觸應緊密，受力後無滑動現象，線夾的 凸肚應在尾線側，安裝時不應損傷線股。

c ：有墜線的拉線柱埋深為柱長1/6，墜線上端固定點距柱頂距 離應為250mm。

d ：當拉線裝設絕緣子時，斷拉線情況下絕緣子距地面不應小 於2500mm。

e ：UT型線夾應有不小於1/2螺桿絲扣長度可供調緊。調整後， UT線夾的雙螺母應並緊。

f ：若為拉線檢修更換拉線（整體或部件），在拆除舊拉線 （或部件）前應採取加裝臨時拉線措施，防止線路失去拉線保護導致線路跑偏、倒杆等。

4.2

預交式拉線製作與安裝

4.2.1、拉線製作



當拉線採用預絞式拉線時：

（1）拉線連接金具一般採用螺旋式預絞式耐張線夾。

（2）預絞絲表面應光潔，無裂紋、摺疊和結疤等缺陷。

（3）確認線夾與拉線直徑和種類相匹配。

（4）線夾與鋼絞線旋向一致，採用標準右旋方式。

a：拉線一般採用多股鍍鋅鋼絞線，其規格為 GJ-35～100。

b：鋼絞線剪斷前應綁紮好。

c：拉線上把和拉線抱箍連接處採用延長環連接。

d：防腐處理，防護部位：自地下500mm 至地上200mm 處；防護措施：塗瀝青，纏麻袋片兩層，再刷防腐油。

4.2.2、拉線安裝




（1）拉線應採用專用拉線抱箍。

（2）拉線抱箍一般裝設在相對應的橫擔下方，距橫擔中心線100mm 處。

（3）拉線的收緊應採用緊線器進行。

（4）拉線應根據需要加裝拉線絕緣子。

（5）城區或人口聚集地拉線宜加裝警示標識，標識高度不低於2000mm。

a：應用鉸鏈將拉線拉緊至合適位置，在拉線上從錨杆向上約 20mm 處，用細鐵絲綁紮，剪去多餘拉線。

b：拉線應放在離拉線耐張線夾絞環最近的鉸接標識處，拉線耐張線夾與錨杆心形環的槽對齊。

c：有墜線的拉線柱埋深為柱長 1/6，墜線上端固定點距柱頂距離應為 250mm。

d：安裝後拉線絕緣子應與上把拉線抱箍保持 3000mm 距離。

e：拉線安裝完成後應在地面以上部分安裝拉線警示保護管。

f：UT 型線夾應有不小於 1/2螺桿絲扣長度可供調緊。

g：預絞絲拉線線夾不得重複使用。

5、導線架設

5.1

放線前檢查




（1）導線型號、規格應符合設計要求。

（2）導線展放時，應清理線路走廊內的障礙物，滿足架線施工要求。

（3）跨越架與被跨越物、帶電體間的最小距離，應符合規定。

a：導線不應有鬆股、交叉、摺疊、斷裂及破損等缺陷。

b ：導線不應有嚴重腐蝕現象。

c ：鋼絞線、鍍鋅鐵線表面鍍鋅層應良好，無鏽蝕。

d ：絕緣線端部應有密封措施。

e ：絕緣線的絕緣層應緊密擠包，目測同心度應無較大偏差，表面平整圓滑，色澤均勻，無尖角、顆粒，無燒焦痕跡。

5.2

線盤佈置




（1）放線前應先制定放線計劃，合理分配放線段。

（2）根據地形，適當增加放線段內的放線長度。

（3）根據放線計劃，將導線線盤運到指定地點。

（4）應設專人看守，並具備有效制動措施。

（5）臨近帶電線路施工線盤應可靠接地。

a ：導線佈置在交通方便、地勢平坦處。地形有高低時，應將線盤佈置在地勢較高處，減輕放線牽引力。

b ：導線放線應考慮減少放線後的餘線，儘量將長度接近的線軸集中放在各耐張杆處。

6、放線操作

6.1

放線準備



（1）放線架應支架牢固，出線端應從線軸上方抽出，並應檢查放出導線的質量。

（2）在每基電杆上設置滑輪，把導線放在輪槽內。

（3）滿足GB 50173相 關要求。

a：線軸應轉動靈活，軸槓應水平，線軸應有制動裝置。

b：絕緣線應使用塑料滑輪或套有橡膠護套的鋁滑輪；滑輪應具有防止線繩脫落的閉鎖裝置；滑輪直徑不應小於絕緣線外徑的12倍，槽深不小於絕緣線外徑的1.25倍，槽底部半徑不小於0.75倍絕緣線外徑，輪槽槽傾角為15°。

c：絕緣線宜採用網套牽引。

6.2

人力放線



（1）人力牽引導線放線時，拉線人員之間應保持適當距離。

（2）領線人員應對準前方，隨時注意信號。

a：牽引過程中應保持牽引平穩。

b：導線不應拖地，各相導線之間不得交叉。

c：跨（穿）越障礙物時應採取相應措施。

d：牽引時應在首、末、中間派人觀察，及時發現導線掉槽、滑輪卡滯等故障，發現異常情況後及時用對講機聯繫。

6.3

機械牽引放線



（1）將牽引繩分段運至施工段內各處，使其依次通過放線滑車。

（2）牽引繩之間用旋轉連接器或抗彎連接器連接貫通。

（3）用機械捲回牽引繩，拖動架空導線展放。

a：固定機械牽引所用牽引繩，應為無捻或少捻鋼繩。

b：旋轉連接器不能進牽引機械捲筒。

c：牽引鋼繩與導線連接的接頭通過滑車時，牽引速度不宜超過20m/min。

d：牽引時應在首、末、中間派人觀察，及時發現導線掉槽、滑輪卡滯等故障，發現異常情況後及時用對講機聯繫。

7、緊線

7.1

緊線準備



（1）緊線施工應在全緊線段內的杆塔全部檢查合格後方可進行。

（2）緊線前應按要求裝設臨時拉線。

（3）放線工作結束後，應儘快緊線。

a：總牽引地錨與緊線操作杆塔之間的水平距離，應不小於掛線點高度的兩倍，且與被緊架空導線方向應一致。

b ：緊線應緊靠掛線點。

c ：緊線時，人員不準站在或跨在已受力的導線上或導線的內角側和展放的導線圈內以及架空線的垂直下方。

d ：跨越重要設施時應做好防導線跑線措施。

7.2

耐張杆塔補強



（1）當以耐張杆塔作為操作或錨線杆塔緊線時，應設置臨時拉線。

（2）臨時拉線裝設在耐張杆塔導線的反向延長線上。

（3）臨時拉線對地夾角宜小於45°。

a ：臨時拉線一般使用鋼絲繩或鋼絞線。

b ：臨時拉線一般採用一錨一線。

c ：臨時拉線不得固定在可能移動的物體上。

7.3

緊線



（1）絕緣子、拉緊線夾安裝前應進行外觀檢查，並確認符合要求。

（2）緊線時，應隨時查看地錨和拉線狀況。

（3）導線的弧垂值應符合設計數值。

（4）滿足SD 292 相關要求。

a ：安裝時應檢查碗頭、球頭與彈簧銷子之間的間隙；在安裝好彈簧銷子的情況下，球頭不得自碗頭中脫出。

b ：緊線順序：導線三角排列，宜先緊兩邊線，後緊中線；導線水平排列，宜先緊中導線，後緊兩邊導線；導線垂直排列時，宜先緊上導線，後緊中、下導線。

c ：絕緣線展放中不應損傷導線的絕緣層和出現扭、彎等現象，接頭應符合相關規定，破口處應進行絕緣處理。

d ：三相導線馳度誤差不得超過-5%或+10%，一般同一檔距內馳度相差不宜超過50mm。

8、導線固定

8.1

導線固定及附件安裝



（1）導線的固定應牢固、可靠。綁線綁紮應符合“前三後四雙十字”的工藝標準，絕緣子底部要加裝彈簧墊。

（2）緊線完成、弧垂值合格後，應及時進行附件安裝。

a ：直線轉角杆：對瓷質絕緣子，導線應固定在轉角外側的槽內；對瓷橫擔絕緣子導線應固定在第一裙內。

b ：直線跨越杆：導線應雙固定，導線本體不應在固定處出現角度。

c ：裸鋁導線在絕緣子或線夾上固定應纏繞鋁包帶，纏繞長度應超出接觸部分30mm。鋁包帶的纏繞方向應與外層線股的絞制方向一致。

d ：絕緣導線在絕緣子或線夾上固定應纏繞粘布帶，纏繞長度應超過接觸部分30mm,纏繞綁線應採用不小於2.5mm2的單股塑銅線，嚴禁使用裸導線綁紮絕緣導線。

8.2

導線連接



（1）鋁絞線及鋼芯鋁絞線在檔距內承力連接一般採用鉗壓接續管或採用預絞式接續條。

（2）10kV絕緣線及低壓絕緣線在檔距內承力連接一般採用液壓對接接續管。

（3）對於絕緣導線，接頭處應做好防水密封處理。

a ：10kV架空電力線路當採用跨徑線夾連接引流線時，線夾數量不應少於2個，並使用專用工具安裝，楔型線夾應與導線截面匹配。

b ：連接面應平整、光潔，導線及並溝線夾槽內應清除氧化膜，塗電力複合脂。

c ：銅絞線與鋁絞線的接頭，宜採用銅鋁過渡線夾、銅鋁過渡線，或採用銅線搪錫插接。

8.3

淨空距離



導線架設後，導線對地及交叉跨越距離，應符合國標50061、DL/T 602 及設計要求。

a ：3～10kV線路每相引流線，引下線與鄰相的引流線，引下線或導線之間，安裝後的淨空距離應不小於300mm；3kV以下電力線路應不小於150mm。

b ：架空線路的導線與拉線，電杆或構架之間安裝後的淨空距離，3～10kV時應不小於200mm；3kV以下時應不小於100mm。

c ：中壓絕緣線路每相過引線、引下線與鄰相的過引線、引下線及低壓絕緣線之間的淨空距離不應小於200mm；中壓絕緣線與拉線、電杆或構架間的淨空距離不應小於200mm。

d ：低壓絕緣線每相過引線、引下線與鄰相的過引線、引下線之間的淨空距離不應小於 100mm；低壓絕緣線與拉線、電杆或構架間的淨空距離不應小於 50mm。
