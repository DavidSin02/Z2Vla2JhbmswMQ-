彙編程序設計：LCD1602靜態顯示

2020-11-14
| 科技

嗨，大家好，昨天剛寫好一個彙編程序，之前有bug，不能正確顯示，經過一天的調試修改終於改好了，和大家分享一下。


我今天給大家介紹的是LCD1602的顯示，該程序是用彙編語言來寫的。我們先來看一下實物效果：


LCD1602實物效果圖

背面接線

偽指令定義，可以寫在ORG 0000H的前面。

LCD_RS      BIT  P1.3    ; 偽指令，定義LCD的RS引腳 LCD_RW    BIT  P1.5    ;偽指令，定義LCD的RW引腳LCD_E         BIT  P1.4     ;偽指令，定義LCD的E引腳LCD_Status EQU 20H   ;定義LCD的忙狀態

以下為程序中用到的量含義說明：在Keil中應該寫成註釋

LCD_Status：存放LCD忙狀態信息，=0，為不忙R6 存放LCD的命令字，LCD_Write_Command的入口參數R5 存放要顯示的數據，LCD_Write_Dat的入口參數R4 存放顯示位置，LCD_Position的入口參數R3 存放要顯示的數組長度30H 延時函數的入口參數，通過給30H單元賦值，可以實現不同的延時時間。 
 ORG 0000H LJMP MAIN;

/**************************

主程序：MAIN

功能：主函數

************************/

                   ORG 0030HMAIN:         MOV SP,#40H                   LCALL LCD_Init;                          MOV 30H,#5;                   LCALL Delay;                    LCALL LCD_Display;                    MOV 30H,#5;                   LCALL Delay;                    LJMP  MAIN;

/*********************************

子程序：Nop4

功 能：6個機器週期延時

*********************************/

Nop6:         NOP                   NOP                   NOP                   NOP                   NOP                   NOP                                     RET       // 6個空操作

/******************************************

子 程 序：LCD_Busy_Check

功 能：LCD忙檢測

出口參數：LCD_Status，LCD_Status=0，說明LCD不忙；

*****************************************/

LCD_Busy_Check:                   CLR  LCD_RS        ;LCD_RS=0                   SETB LCD_RW      ;LCD_RW=1                         SETB LCD_E         ;LCD_E=1                  LCALL Nop6                   ;延時4個機器週期                   ANL P2,#80H                ;LCD_Status=P0&0x80                   MOV LCD_Status,P2     ;LCD_Status=P0&0x80                                                                CLR LCD_E                   RET

/***************************

子 程 序：LCD_Write_Command

功 能：向LCD發佈命令

入口參數：R6，在調用該程序前應該對R6進行賦值，即LCD的命令字。

出口參數：無

*******************************/

LCD_Write_Command:                  NOPNEXT1:       LCALL LCD_Busy_Check ;                  MOV A,LCD_Status         ;                   JNZ NEXT1                    ;   A不等於0，說明LCD為忙狀態，繼續檢測                        CLR LCD_RS                   ;LCD_RS=0                   CLR LCD_RW                 ;LCD_RW=0                     CLR LCD_E                     ;LCD_E=0                                                 MOV P2,R6            ;將LCD命令字送入P2口                                     LCALL Nop6                                     SETB LCD_E;                   LCALL Nop6                   CLR LCD_E ;                   RET

/*********************

子 程 序：LCD_Write_Dat

功 能：向LCD發送數據

入口參數：R5，在調用該程序前應該對R5進行賦值，即向LCD寫入的數據。

出口參數：無

*******************/

LCD_Write_Dat:                  NOPNEXT2:       LCALL LCD_Busy_Check;                   MOV A,LCD_Status    ；LCD_Status                   JNZ NEXT2                 ;  A不等於0，說明LCD為忙狀態，繼續檢測;  A=0,說明LCD不忙了。                                   SETB LCD_RS              ;                   CLR LCD_RW              ;                   CLR LCD_E                  ;                   MOV P2,R5               ;將LCD數據送入P2口                   LCALL Nop6                    SETB LCD_E ;                   LCALL Nop6                   CLR LCD_E                  RET

/***************************

子 程 序：LCD_Position

功 能：向LCD發佈命令

入口參數：R4，在調用該程序前應該對R4進行賦值。

出口參數：R6，將或運算後的數據送入R6，作為LCD_Write_Command的入口參數

**************************/

LCD_Position:                   MOV A,R4            ;將位置數據送入A                   ORL A,#80H          ;設置LCD的當前顯示位置                   MOV R6,A            ;將位置存入R6；                   ACALL LCD_Write_Command;                   RET                        ;

/****************************

子 程 序：LCD_Display

功 能：顯示

入口參數：R4，在調用該程序前應該對R4進行賦值。

出口參數：R6，將或運算後的數據送入R6，作為LCD_Write_Command的入口參數

***************************/

 LCD_Display:                   MOV R4,#00;                   LCALL LCD_Position ；    設置第一行的顯示位置：00H開始                                  MOV DPTR,#TAB1;                MOV R3,#16 LOOP1:  CLR A                MOVC A,@A+DPTR                MOV R5,A;                LCALL LCD_Write_Dat;                INC DPTR;                DJNZ R3,LOOP1                MOV R4,#41H             ;      設置第二行的顯示位置：41H開始               LCALL LCD_Position               MOV DPTR,#TAB2;               MOV R3,#16LOOP2:   CLR A                MOVC A,@A+DPTR                MOV R5,A;                LCALL LCD_Write_Dat;                INC DPTR;                DJNZ R3,LOOP2                RET;

/****************************

子 程 序：LCD_Init

功 能：LCD初始化，

入口參數：無

出口參數：無

***************************/

LCD_Init:                   MOV R6,#38H;                   LCALL LCD_Write_Command;                   MOV 30H,#10                   LCALL DELAY                                     MOV R6,#0FH;                   LCALL LCD_Write_Command;                   MOV 30H,#10                   LCALL DELAY                                     MOV R6,#06H;                   LCALL LCD_Write_Command;                   MOV 30H,#10                   LCALL DELAY                                     MOV R6,#01H;                   LCALL LCD_Write_Command;                   MOV 30H,#10                   LCALL DELAY                    RET

/************************

子 程 序：Delay

功 能：延時函數

入口參數：30H

出口參數：無

***************************/

DELAY:   ;誤差 0us             MOV R2,30H;DL0:    MOV R1,#5FH             DJNZ R1,$             DJNZ R2,DL0             RET

要顯示的字符串，用DB進行定義

TAB1:        DB "I Love MCU    BY"TAB2:        DB "2020.5.27    LY"                  END
