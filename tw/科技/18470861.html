大揭祕！冠希連晒三次的 Jordan 新鞋！小編提前上腳

2020-11-14
| 科技

上個月初，冠希在個人 Instagram 突然晒出一雙帶有「000 23」字樣的神祕新鞋。「全新聯名」、「冷門鞋型」以及「瘋狂暗示」等一系列流言席捲網絡。

眾說紛紜之中，隨著整雙鞋的全貌釋出，謎題也正式揭曉：來自 Jordan Brand 打造的全新鞋型 Jordan Delta React。

作為一雙全新 Lifestyle 系列鞋款，能夠受到冠希如此青睞，甚至得到：“我認為它用設計證明了自己不凡的身份。” 這樣的超高評價。

今天，小編就將為你揭開這雙鞋的神祕面紗！究竟實物怎麼樣？上腳怎麼樣？想必看完你就能夠找到答案！

▼ 小編上腳實拍

01 挑戰全新造型

全新 Jordan Delta React 設計初衷在於打造一雙適合全天候舒適穿著的鞋款。

設計圍繞 “飛翔” 的理念進行創作，低幫造型設計能夠充分保障腳踝處的靈活需求。

鞋面借鑑航天科技和設計，採用各種高端材料結合打造。多種材質的拼接效果，呈現出豐富層次感。

搭載全掌 Nike React 泡棉，為雙腳帶來極佳的舒適感覺。

外底採用了籃球鞋經常使用的人字紋圖案，在致敬籃球運動的同時，為穿著者提供充分的防滑效果。

02 做工精細，細節出眾

作為一雙極具科幻感的全新鞋型，Jordan Delta React 在細節方面的刻畫同樣值得一看。首先便是整雙鞋的用料，鞋身大面積採用緻密的織物材質打造鞋面。

鞋頭外側部分採用被透明橡膠包裹，內裡採用類似牛皮紙一般的材質，呈現出復古做舊質感。

側邊點綴一小塊 3M 反光飾條，「000 23」字樣也與冠希當時晒出的實物保持一致。

▼ 「000 23」字樣

鞋身中部應用航天材質，不規則的褶皺很容易令人想到太空服手套的造型，這一點也與 Nike Mars Yard 3.0 相類似。

同時粗獷的線條縫線將不同材質拼接組合，淡藍色防撕裂鞋面也體現出十足科技感。

▼ 鞋身材質豐富

厚實的海綿鞋舌上佈滿孔洞設計，不僅復古質感出眾，同時也為透氣效果起到一定作用。頂端部分採用外露式設計，頗有幾分解構風格。

▼ 海綿鞋舌填充厚實

手繪風格的 Jumpman Logo 以及「000 23」字樣點綴鞋舌標籤，彰顯整雙鞋的主題。

▼ 鞋舌藍色 Jumpman Logo

後跟位置同樣採用透明橡膠材質包裹，圓形孔洞設計呼應鞋舌，藍色線條勾勒出經典飛翼 Logo 的輪廓。

▼ 後跟藍色線條飛翼 Logo

中底內側帶有「react」字樣，表明整雙鞋搭載 React 緩震科技，充分保障腳感。

▼ 中底 React 標籤

外底紋路格外清晰，後跟位置一個大號 Jumpman Logo 亮明品牌身份。

▼ 外底 Jumpman Logo

最驚喜的是，Jordan Brand 為全新 Delta React 配備了軟木塞鞋墊，這一點怕是要打動不少人的心！

▼ 軟木塞鞋墊

03 腳感軟彈，造型百搭

整雙鞋上腳的第一感受便是來自 React 緩震科技的軟彈感覺，不會有過度塌陷感，同時也能夠給雙腳提供不錯的反饋。

外底淺黃色橡膠環繞 React 緩震中底 ，同時在側邊部分還有向上延伸的設計，為整雙鞋提供支撐。

清晰的外底紋路為整雙鞋帶來充足的抓地效果，抬腳間露出的 Jumpman Logo，彰顯 Jordan 品牌身份。

海綿鞋舌填充厚實，同時圓形孔洞設計的存在也充分起到透氣效果。

鞋後跟故意做歪的拉環符合人體工學設計，更加體現了性能優先的設計考量。

04 本月即將發售

小編上腳親測整雙鞋尺碼不偏，按個人正常尺碼入手即可。

目前，Jordan Brand 官方已經確定，小編上腳的這雙 Jordan Delta React SP 將於 4 月 20 日正式在國內發售。

隨後，5 月 1 日還將推出火星鞋配色以及黑白配色等更多搶眼配色！

想要入手的朋友屆時一定不要錯過！
