PHP鏈接數據庫操作。

一、PHP鏈接本地數據庫創建表，增刪查改數據。

1、測試連接數據庫是否成功。

（1）源代碼：

<?php

header("Content-Type:text/html;charset=utf-8");

$conn=mysql_connect('localhost','root',"1");

if($conn){

echo "數據庫連接成功";

}else{

echo "數據庫連接失敗";

}

?>

（2）顯示效果

二、在php語法中運用sql語法

1、創建學生表，並創建字段

2、向數據庫插入數據

（1）源代碼：

<?php

header("Content-Type:text/html;charset=utf-8");

$conn=mysql_connect('localhost','root',"1") or die('數據庫連接失敗');

if($conn){

mysql_select_db('test',$conn) or die('數據庫連接失敗');

//test是數據庫名

$sql="insert into student(sno,name,sex) values('26373','小明','男')";

//stu是表名，sno,name,sex可替換，是字段；12456','黎明','男可替換，是字段值。

$result=mysql_query($sql);

if($result){

echo "數據庫操作成功！";

}else{

echo "數據庫操作失敗！";

}

}

else{

echo "連接服務器失敗！";

}

（2）效果圖：

2-1插入多條數據：

（1）源代碼：

將$sql="insert into student(sno,name,sex) values('26373','小明','男')";改為

$sql="insert into stu(sno,name,sex) values('26373','小明','男'), ('37849','小紅','女'), ('26373','小李','男')";
