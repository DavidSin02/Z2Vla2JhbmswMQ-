電梯轎廂動、靜平衡！

2020-10-29
| 科技

靜平衡轎箱

---- 所有電梯轎箱全部可以做到靜平衡。

---- 高速電梯要做動平衡。

---- 折中處理動平衡對高速電梯有利。

最重要的事例

1、 安全鉗的清潔、加油和調節工作

2、 安全鉗卡口和滾輪導靴的虎口必須中心對準。

3、 滾輪導靴必須與導軌垂直。

4、 滾輪導靴的最大壓力不得超過25kg，或則每個滾輪或靴襯上的壓力不得超過12。5kg。

5、 靜平衡可以用轎箱底部的平衡塊來調節。

6、 動平衡不用轎箱底部的平衡塊來調節。

7、 如果補償繩和隨行電纜在轎箱架的同一側，轎箱就不能平衡。

8、 但隨行電纜、曳引繩和補償繩不在同一垂直平面上時，轎箱就不能平衡。

9、 那一個更重要，是隨行電纜的正確運行軌跡，還是轎箱的動平衡？

Ⅱ轎廂靜平衡

下面介紹在工地來完成轎廂靜平衡。

開始之前必須保證轎廂沒有扭曲。保證沒有扭曲的最佳時間是開始拼轎廂時。如轎廂存在扭曲現象，接下去的靜平衡就不可能做。因此，在開始做轎箱平衡和滾輪導靴調整時，轎箱的龍門框必須自由狀態無扭曲。如扭曲現象確實存在，開始之前必須先調整好。

工作順序：

a） 電梯開到頂層。

b） 對準滾動導靴，使導軌位於導靴座的中心位置， 定位好導靴的止推件。

c） 從曳引輪的中間鋼絲繩掛下鉛垂，一直到轎頂繩頭板，如鋼絲繩為雙數，則應從中間相鄰的兩根鋼絲繩的中間掛下，如為單數，則沿著中間鋼絲繩的前面或後面掛下，但必須保持一定的距離。同時也應保證鉛垂與轎頂繩頭板的距離。由於轎頂繩頭板上的孔一般為兩排，所以應有足夠的空間掛鉛垂。

d） 接下去主要講述如何移動轎頂繩頭板。我們不能太強調移動轎頂繩頭板僅是讓轎頂繩頭板的中心對準曳引輪的中心。移動繩頭板的目的不是想來平衡在某一位置的轎箱。如果轎頂繩頭板為可移動式，則可移動繩頭板對準曳引輪。如果轎頂繩頭板為固定式且用螺栓固定在橫樑上，可拆去原螺栓，螺栓暫時可用1/2”加墊片來代替3/5”，沿著立柱的方向移動。注意這時在移動繩頭板時必須十分小心，因如螺栓掉出就會導致轎箱墜落。這時最好用4個“C”型夾夾住繩頭板和上樑，使得繩頭板不會前後移動，繩頭板可以用撬棒沿著立柱方向移動，記住完成後必須更換回3/5“的螺栓。這時可能需要在橫樑上開槽，開槽前必須用定位銷定好位，然後鑽一新孔，不要在前後方向移動繩頭板。

如果掛下鉛垂後發現有前後偏差，則只能移動主機。這也是為什麼不能把主機和承重樑焊牢的原因之一。

當工作結束，仔細測量導軌面與繩頭板中心的位置，記錄備用。

e） 下一步是平衡曳引繩張力。有許多方法，由於沒有統一的規定，不同的地方用不同的方法，下面介紹幾種方法：

方法一、在對重側平衡曳引繩。

繩頭板裝好彈簧後，彈簧的高低就可作為鋼絲繩張緊力大小的指示。通過檢查彈簧的高低，就可以馬上決定是哪根鋼絲繩需要調整。電梯開到井道2/3的位置，這樣在對重側的鋼絲繩就比較長。用彈簧秤鉤住鋼絲繩，拉向轎頂方向相同距離，記下彈簧秤的讀數（建議使用一木板，一頭頂牢井道壁，另一段標有標記，水平放置，每次拉時都到達這樣刻度）

電梯下開，根據經驗調整對重繩頭板上的垂套螺母，保證鋼絲繩的張力一致。

至少上下開動電梯兩次，重新測量拉動鋼絲繩的數據，如需要，重新調整好垂套螺母，直到彈簧秤上的讀數基本一致，則可認為鋼絲繩已調整好了。這時也可發現鋼絲繩的長度也是基本相同的。

當調整螺母時，必須保證鋼絲繩沒有轉動。

方法二、在轎箱側調整鋼絲繩張力。開始之前先測量彈簧長度，這樣也可提示哪根鋼絲繩需要調整。

此時的方法沒有方法一簡單，電梯停於井道的下部，這樣可以從至少是電梯兩層以上的某處廳門可以打開。這裡需要特殊的工具，方法與方法一時相同。

方法二一般只用在當電梯兩側的張力不平衡時使用。

方法③~⑤省略

a） 補償繩的調整也在靜平衡時完成。

b） 當所有的部件檢查調整完畢就可進入下一步工作。電梯下開至較低層樓。在此我們可以回顧一下剛才已完成的工作，以便開始正式調整靜平衡。

電梯轎箱已沒有扭曲，繩頭板的中心與曳引輪的中心已對準，曳引繩和補償繩的張力也已調均勻。

c） 下一步可以調整底坑補償輪，如果採用自由式的補償輪，其固定在轎底安全鉗地板上的繩頭板也是可以橫向移動的。

這樣的移動可以保證補償繩頭板與補償輪對準。

d） 注意考慮接線箱，轎底接線箱的位置是在圖紙設計時考慮了的，必須確認此位置。

隨行電纜也是應列入靜平衡工作，但在低樓層時，影響比較小，可以不考慮。

e） 下一步：鬆開上下導靴。

f） 在轎箱的幾何中心放入30%的負載。

g） 把平衡塊放入底坑，電梯下行至人可以接觸到轎底。把平衡塊放入轎底槽中，使轎箱平衡，固定平衡塊，靜平衡就完成了。

Ⅲ 轎箱動平衡

靜平衡完成後，就可以開始做動平衡。做這之前，必須先在低樓層時調整好導靴，保證電梯在上下運行時不會碰到井道內的其他設備。這時需讓導靴滾輪的彈簧保持自由狀態。把電梯開到中間和頂部，檢查一下電梯是否傾斜或向哪個方向傾斜。記住這些狀態，在下一步調節時要用到。由於在低層樓時靜平衡已調整好，上下運行電梯所出現的問題，只須在下一步調整曳引繩和隨行電纜的重量就可以了。

大家知道只有那些電梯在底層和高層時重量不一樣的部件會影響電梯的動平衡。確定了這些部件的固定位置就有可能實現動平衡。

因此，轎箱的重心、隨行電纜和曳引繩的固定板的中心必須在同一垂直平面內。

第二和第三個影響動平衡的因數是曳引繩到隨行電纜中心的力臂與曳引繩到中心到補償鏈中心的力臂。

注意動平衡只有在靜平衡完成後才可能到達。

當轎箱上行時，與電梯上行距離相等的補償鏈的重量就加到電梯上，同時1/2上行距離的隨行電纜也加入到電梯上。

大家明白，靜平衡可以用轎底的平衡塊來調整，但必須認識到動平衡是不能用轎底的平衡塊來完成的。

動平衡程序

a) 電梯下行至底層，如果在高層樓時電梯發現有極不平衡的現象出現，最好去掉上導靴的彈簧使得電梯能夠在自由狀態。如果在高層樓時電梯的不平衡的現象很小，只須退鬆彈簧就可以了。

b) 檢查一下是否有30%載荷放在轎箱的幾何中心。

c) 如是信號控制的電梯，則必須在電梯駕駛員的位置放上70kg（150磅）的重物。

d) 離開電梯，進入底坑按下面的方法一或二觀察下滾輪導靴的狀況。

方法一 檢查導軌是否位於導靴的中心，和在靜平衡時一樣。

方法二 拆除彈簧，讓轎箱自由擺動。

方法一由於導靴仍固定在轎箱上，因此會更實用。因為在檢查動平衡時和在底坑檢查各項部件時，更有可能使得電梯上行時無須移動下導靴。隨行電纜的位置會更準確。

e) 下一步是檢查底坑各部件的距離—參照曳引繩的中心為準。在轎頂測量曳引繩中心到導軌中心的距離，然後把此中心的位置標記在轎底的緩衝器頂板上。同時打好標記備用。

從補償鏈掛鉤中心和剛才的標記點連一根線，此線的另一端應穿過隨行電纜掛板的中心。如果隨行電纜的掛板中心沒有落在聯線上，則必須調整掛板。

只有這樣才能保證三者在同一平面內。

f) 接著需要確定隨行電纜的離曳引繩中心的正確位置，由於補償鏈掛鉤固定，所以可以計算得到其位置。

為了使得轎箱底的載荷平衡，必須保證補償鏈乘以力矩等於隨行電纜乘以力矩。

公式為：

TWR * MAR” = TWC * MAC”

TWR = 補償鏈總重

MAR = 補償鏈掛鉤中心與曳引繩中心的距離

TWC = 隨行電纜總重

MAC = 隨行電纜掛板與曳引繩中心的距離

TWR = R * N *W t + 1000磅/2

R = 電梯提升高度

N = 補償鏈根數

W t = 補償鏈單位長度的重量

1000磅/2 = 底坑中補償鏈張緊輪重量的一半

TWC = T1 + T2 + T3 +… * R/2

T1, T2, T3 …單根隨行電纜的重量

R = 電梯提升高度

TWR 和TWC都可通過計算得到，MAR和MAC可以在轎底通過測量得到，乘積必須相等。如不相等，則可調整MAC的數值直至相等，其數值的變化就是需要移動的距離。這樣隨行電纜的位置就確定了，動平衡調整完成。

g) 只有在電梯開到頂層時才可以校合動平衡調的是否正確。因為重量計算和距離測量都都可能出錯，因此結束後需要確認調整是否正確。

如果下導靴調整後，導靴底座槽的中心對準導軌，則沒有問題。上導靴的滾輪必須設置止推，使得轎頂、門機等不會與井道內的物件相撞。但是，上導靴必須是自由狀態，使得轎箱上部能夠自由移動。

如果轎箱後側較重，導致電梯在高層樓時轎頂向後傾斜，則轎底的隨行電纜應向前移動。如轎頂前傾，則隨行電纜因後移。但此時移動距離應很小，不至於引起距離的較大的變化。

注意在調整隨行電纜的距離時，保證與曳引繩和補償鏈等在同一垂直面內。

由於隨行電纜掛板位置的變化，可能導致隨行電纜的半徑的變化，這時應站在底坑內認真觀察電梯上行和下行時電纜半徑的變化，因此掛板的位置必須同時保證隨行電纜正常運行。

h) 動平衡調整結束後，上下導靴的滾輪彈簧施以最小的彈力，保證轎箱居中。這樣轎箱居中掛著，可以儘可能地減少由於導軌調整的誤差而導致的震動和碰撞。

i) 拿掉轎內30%的配重，用不同的載荷上下開動電梯，停靠不同的層樓，可以檢查一下動平衡和靜平衡的效果。

過去經常發生轎底的掛板無法移動或則無法移動動平衡調整時所需要的距離。許多工地就用開有2“長槽的掛板來代替。

轎頂接線箱的位置也影響隨行電纜的位置。
