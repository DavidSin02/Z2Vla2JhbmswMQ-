棧

說到棧，學了面向對象就很瞭解，面向對象說過的主函數進棧，構造函數進棧、構造函數彈棧，都是在對棧的操作，比如我們主函數先進棧，構造函數再 進棧，我們無法先將主函數彈棧，只能等構造函數彈棧後，再操作。由此就可以得到棧的定義：棧是限定僅在表尾進行插入和刪除操作的線性表。

棧這種後進先出數據結構的應用是非常普遍的，例如子彈夾，最後進入彈夾的子彈總是最先射出去。

我們把允許插入和刪除的一端稱為棧頂（top），另一端稱為棧底（bottom），棧又被稱為後進先出的線性表，也就是說棧元素有線性關係，只不過它是一種特殊的線性表而已。定義中說是在線性表的表尾進行插入和刪除操作，這裡的表尾是指棧頂，而不是棧底。

棧的插入操作，叫作進棧，也稱為壓棧，入棧。類似子彈入彈夾。

棧的刪除操作，叫作出棧，也叫作彈棧，如同彈夾中的子彈出來。

棧的抽象數據類型我們定義為接口Stack，用Java語言實現，基本內容如下：（其中的方法都是抽象的，有public abstract，這裡我們省略了）

public interface Queue<E> extends Iterable {

//獲取隊列中元素的個數

int getSize();

//判斷隊列是否為空

boolean isEmpty();

//入隊一個元素

void enqueue(E e);

//出隊一個元素

E dequeue();

//獲取隊頭

E getFront();

//獲取隊尾

E getRear();

//清空隊列

void clear();

}

接下來寫順序棧ArrayStack，讓它實現接口Stack，順序棧內部是由順序表實現的，之前我寫了順序表ArrayList，只需要創建ArrayList對象list，重複的方法直接調用list的方法實現。

import java.util.Iterator;

/*

順序棧內部是有順序表實現的

ArrayStack

ArrayList

*/

public class ArrayStack<E> implements Stack<E> {

private ArrayList<E> list;

//創建一個默認大小的棧

public ArrayStack(){

//創建順序表對象list

list = new ArrayList<>();

}

//創建一個容量由用戶指定的順序棧

public ArrayStack(int capacity){

list = new ArrayList<>(capacity);

}

//獲取有效元素個數這個方法，順序表中有，直接調用

@Override

public int getSize() {

return list.getSize();

}

@Override

public boolean isEmpty() {

return list.isEmpty();

}

@Override

public void push(E e) {

//進棧從表尾進，調用addLast

list.addLast(e);

}

@Override

public E pop() {

//出棧元素從表尾出，調用removeLast

return list.removeLast();

}

//查看當前棧頂

@Override

public E peek() {

return list.getLast();

}

@Override

public void clear() {

//清空棧和清空順序表一樣，直接調用

list.clear();

}

@Override

public Iterator<E> iterator() {

//迭代器也是直接調用

return list.iterator();

}

//順序棧的toString方法，重新寫

public String toString(){

//創建StringBuilder對象，不需要額外創字符串，在原來的上加字符

StringBuilder sb = new StringBuilder();

sb.append(String.format("ArrayStrck: %d/%d\n",getSize(),list.getCapacity()));

//用數組的形式輸出

sb.append('[');

if(isEmpty()){ //若為空，打印 []

sb.append(']');

}else{

for(int i=0;i<getSize();i++){

//在原來上加字符

sb.append(list.get(i));

if(i==list.getSize()-1){

sb.append(']'); //遍歷到最後一個元素打印']'

}else{

sb.append(','); //不然在每個元素中間打','

}

}

}

//調用對象的tostring方法返回對象

return sb.toString();

}

}
