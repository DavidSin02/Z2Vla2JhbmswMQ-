自己動手豐衣足食之string擴展輔助函數

2020-11-14
| 科技

在使用C/C++開發過程中，和PHP、Python等語言相比，真的是很多功能都需要我們自己來手動擼出來，很是不方便。

最近的幾個項目中用到了很多C++標準模板庫STL裡面的類/容器之類的，最常用的就是字符串 string，其基本的一些用法都還可以，但是遇到了幾個 string 沒有實現，但是還很常用的操作，比如：

字符串根據某個關鍵字符進行拆分，字符串的格式化輸入，字符串的 trim 等操作，這些 string 都沒有提供。每當此時就想起了 MFC 中的 CString 和 AtlString 的方便之處，但是有時候又不想轉來轉去的，而且部分工程還不支持MFC，沒辦法只能自己動手，豐衣足食了。這裡找到了一些不錯的關於以上幾個操作的 string 實現，加上自己的修改封裝了幾個小函數發出來供大家使用！

  1. string 的字符串分割：

void split(string str, string pattern, vector<string>& arrout)

{

arrout.clear();

size_t idx = 0, pos = 0;

size_t size = str.size();

for (idx = 0; idx < size; ++idx)

{

pos = str.find(pattern, idx);

if (pos != string::npos)

{

string s = str.substr(idx, pos - idx);

if (s.size() > 0) arrout.push_back(s);

idx = pos + pattern.size() - 1;

}

else

{

if (idx < size)

{

string s = str.substr(idx, str.size() - idx);

if (s.size() > 0) arrout.push_back(s);

break;

}

}

}

}

第一個參數 str，代表源字符串，即要被分割的字符串；

第二個參數 pattern，代表分割符；

第三個參數 arrout，是傳出參數，把分割好的子串放到這個 vector 容器中；

舉個栗子：

vector <string> arr_out;

string str = "123#abc#222";

split(str, "#", arr_out);

2. string 字符串的格式化輸入：

string string_format(const string fmt, ...)

{

string str;

int size = ((int)fmt.size()) * 2 + 50; // Use a rubric appropriate for your code

va_list ap;

while (1) { // Maximum two passes on a POSIX system...

str.resize(size);

va_start(ap, fmt);

int n = vsnprintf((char *)str.data(), size, fmt.c_str(), ap);

va_end(ap);

if (n > -1 && n < size) { // Everything worked

str.resize(n);

return str;

}

if (n > -1) // Needed size returned

size = n + 1; // For null char

else

size *= 2; // Guess at a larger size (OS specific)

}

return str;

}

舉個栗子：

string str_ret = string_format("%s = %d", "3+2=", 5);

字符串 str_ret 的內容為："3+2=5"

3. string 的 trim 操作：

inline string& string_ltrim(string &str)

{

if (str.empty()) return str;

str.erase(0, str.find_first_not_of(" \n\r\t"));

return str;

}

inline string& string_rtrim(string &str)

{

if (str.empty()) return str;

str.erase(str.find_last_not_of(" \n\r\t") + 1);

return str;

}

inline string& string_trim(string &str)

{

return string_ltrim(string_rtrim(str));

}

功能：可以去掉字符串開頭或者結尾的：空格（ ）、回車符（\r）、換行符（\n）、製表符（\t）等；

以上吧，暫時就這些，如果大家手頭還有其他比較不錯的 string 擴展函數也可以發出來啊。比如：string 字符串的大小寫轉換，string 字符串的 reverse 反轉 等等。
