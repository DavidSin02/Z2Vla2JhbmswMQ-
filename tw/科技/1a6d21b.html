2016年造價工程師《工程計價》真題與答案

一、單項選擇題(共72題，每題I分，每題的備選選頂中，只有I個是符合題意)

1.【題幹】關於我國建設項目投資，下列說法中正確的是(　)

【選項】

A.非生產性建設項目總投資由固定資產投資和鋪底流動資金組成

B.生產性建設項目總投資由工程費用、工程建設其他費用和預備費三部分組成

C.建設投資是為了完成工程項目建設，在建設期內投入且形成現金流出的全部費用

D.建設投資由固定資產投資和建設期利息組成

【答案】C

【解析】A錯誤，不包含流動資金;B錯誤，應該包含流動資金;D錯誤，固定資產投資包含建設投資和建設期利息。

【考點】我國建設項目投資及工程造價的構成。

2.【題幹】已知生產某非標準設備所需材料費、加工費、輔助材料費、專用工具費合計為30萬元，廢品損失率為10%，外購配套件費為5萬，包裝費率為2%，利潤率為10%，用成本估算法計算該設備的利潤值為(　)元。

【選項】

A.3.366

B.3.370

C.3.376

D.3.876

【答案】C

【解析】{[30×(1+10%)+5]×(1+2%)-5｝×10%=3.376

【考點】國產非標準設備原價的計算。

3.【題幹】進□設備的原價是指進口設備的(　)

【選項】

A.到岸價 B.抵岸價 C.離岸價 D.運費在內價 【答案】B

【解析】進口設備的原價就指進口設備的抵岸價。

【考點】

4.【題幹】某縣城建築公司承建市區某建築工程，2015年8月編制估算的稅前造價為，已知6000萬元，已知營業稅率為3%，市、縣城市維護建設稅稅率分別為7%、5%，教育附加率和地方費附加率分別為3%、2%。該工程的結算造價為(　)萬元

【選項】

A.6198.0 B.6201.2 C.6204.8 D.6208.6

【答案】D

【解析】6000/(1-3%-3%×7%-3%×3%-3%×2%)=6208.6

【考點】稅金的計算.

5.【題幹】根據我國現行建築安裝工程費用項目構成的規定，下列費用中屬於安全文明施工費的(　)

【選項】

A.夜間施工時，臨時可移動照明燈具的設置、拆除費用

B.工人安全防護用品的購置費用

C.地下室施工時採用的照明設施拆除費

D.建築物的臨時保護設施費

【答案】B

【解析】A屬於夜間施工增加費;C屬於非夜間施工照明費;D選項屬於建築物的臨時保護設施費。

【考點】措施費的構成。

6.【題幹】關於國外建築安裝工程費用的計算，下列說法正確的是(　)

【選項】

A.分包工程費不包括分包工程的管理費和利潤　B.材料的預漲費計入管理費

C.開辦費一般包括了工地清理費及完工後清理費 D.現場管理費屬於開辦費

【答案】C

【解析】A錯誤，分包工程費包括：分包報價、總包管理費和利潤。材料預漲費屬於材料費的一部分，B錯誤。

【考點】

7.【題幹】下列與建設用地有關的費用中，歸農村集體經濟組織所有的是(　)

【選項】

A.土地補償費 B.青苗補償費 C.拆遷補償費　D.新菜地開發建設基金

【答案】A

【解析】B選項青苗補償費歸青苗所有者;C拆遷補償費歸被拆遷人;D選項新菜地開發建設基金歸政府。

【考點】建設用地費的構成。

8.【題幹】關於工程建設其他費用，下列說法中正確的是(　)

【選項】

A.建設單位管理費一般按建築安裝工程費乘以相應費率計算

B.研究實驗費包括新產品試製費

C.改擴建項目的場地準備及臨時設施費一般只計拆除清理費

D.不擁有產權的專用通信設施投資，列入市政公用設施費

【答案】C

【解析】A選項建設單位管理費的計算基礎為工程費用;B選項研究實驗費不包括新產品試製費;D選項不擁有產權的專用通信設施投資列入專利及專有技術使用費。

【考點】工程建設其他費用。

9.【題幹】某建設項目靜態投資為10000萬元，項目建設前期年限為1年，建設期為2年，第一年完成投資40%，第二年完成投資60%。在年平均價格上漲率為6%的情況下，該項目價差預備費應為(　)萬元。

【選項】

A.666.3 B.981.6 C.1306.2　D.1640.5

【答案】C

【解析】10000×40%×[(1+6%)1+1-0.5-1]+10000×60%×[(1+6%)1+2-0.5-1]=1306.2

【考點】價差預備費的計算。

10.【題幹】某項目建設期為2年，第一年貸款3000萬元，第二年貸款2000萬元，貸款年均衡發放，年利率為8%，建設期內只計息不付息。則建設期利息為(　)

【選項】

A.366.4　B.449.6 C.572.8　D.659.2

【答案】B

【解析】第一年利息為3000/2×8%=120萬;第二年利息為(3000+120+2000/2)×8%=329.6 兩年合計利息為449.6萬

【考點】建設期利息計算。

11.【題幹】下列說法中，符合工程計價基本原理的是(　)

【選項】

A.工程計價的基本原理在於項目劃分與工程量計算

B.工程計價分為項目的分解與組合兩個階段

C.工程計價包括工程單價的確定和總價的計算

D.工程單價包括生產要素單價、工料單價和綜合單價

【答案】C

【解析】A錯誤，工程計價的基本原理在於項目的分解與組成;B錯誤，工程計價分為工程計量和工程計價兩個環節;D錯誤，工程單價包括工料單價和綜合單價。

【考點】工程計價的基本原理。

12.【題幹】關於工程定額，下列說法正確的是(　)

【選項】

A.勞動定額主要表現形式為時間定額，機械消耗定額主要表現形式為產量定額

B.企業定額是企業內部使用的預算定額

C.補充定額只能在指定範圍內使用

D.地區統一定額應依據地區工程技術特點、施工生產和管理水平獨立編制

【答案】C

【解析】A錯誤，勞動定額主要表現形式為時間定額，機械消耗定額主要表現形式為時間定額;B錯誤，企業定額是企業內部使用的施工定額;D錯誤，地區統一定額主要是考慮地區性特點和全國統一定額水平做適當調整和補充編制的。

【考點】工程定額體系。

13.【題幹】根據《建設工程工程量清單計價規範》(GGB505-2013)下列關於工程量清單項目編碼的說法中，正確的是(　)

【選項】

A.第三級編碼為分部工程順序碼，由三位數字表示

B.第五級編碼應根據擬建工程的工程量清單項目名稱設置，不得重碼

C.同一標段含有多個單位工程，不同單位工程中項目特徵相同的工程應採用相同編碼

D.補充項目編碼以"B"加上計量規範代碼後跟三位數字表示

【答案】B

【解析】A錯誤，第三級編碼由二位數字表示;C錯誤，特徵相同的工程不得有重碼;D錯誤，補充項目編碼以上計量規範代碼加"B"和三位數字表示。

【考點】分部分項工程量清單。

14.【題幹】對於不能計算工程量的措施項目，當按施工方案計算措施費時，若無"計算基礎"和"費率"數值，則(　)

【選項】

A.以定額計價為計算基礎，以國家、行業、地區定額中相應的費率計算金額

B.以"定額人工費+定額機械費"為計算基礎，以國家、行業、地區定額中相辦計算金額相應費率計算金額

C.只填寫"金額"數值，在備註中說明施工方案出處或計算方法

D.備註中說明中計算方法，補充填寫"計算基礎"和"費率"

【答案】C

【解析】對於不能計算工程量的措施項目，當按施工方案計算措施費時，若無"計算基礎"和"費率"數值，則只填寫"金額"數值，但應在備註中說明施工方案出處或計算方法。

【考點】措施項目清單的編制。

15.【題幹】採用工程量清單計價的總承包服務費計價表中，應甶投標人填寫的內容是(　)

【選項】

A.項目價值

B.服務內容

C.計算基礎

D.費率金額

【答案】D

【解析】投標時，總承包服務費的費率及金額由投標人自主報價，計入投標總價中。

【考點】總承包服務費。

16.【題幹】下列機械工作時間中，屬於有效工作時間的是(　)

【選項】

A.築路機在工作區末端的調頭時間

B.體積達標而未達到載重噸位的貨物汽車運輸時間

c.機械在工作地點之前的轉移時間

D.裝車數量不足而在低負荷下工作的時間

【答案】B

【解析】B選項為有根據的降低負荷下的工作時間。

【考點】工作時間分類。
