一閃一閃亮晶晶的，除了星星還有它們

2020-11-14
| 科技

陽光明媚的春夏季節，喜歡各種亮晶晶的玻璃食器。

晶瑩剔透的模樣足以讓人賞心悅目，撲面而來的是清爽的春夏氣息。微風習習的清晨，或是烈日當空的午後，一杯自己調配的果汁、飲品，一小碗雪糕、甜點，慢悠悠品嚐，應該是許多人關於夏天的記憶。


亮晶晶的玻璃食器在日本稱為「硝子」，和普通玻璃不同的是，硝子是用日本獨有的玻璃工藝製作出來的，它不僅摸起來更加溫潤細膩，而且不全是透明無色，有的還色彩斑斕，形態萬千，如同工藝品一般。

硝子在日本家庭中扮演著重要的角色，從早到晚，一日三餐，甚至是日常的生活用品都能見到它的身影。


而誕生於 1899 年的廣田硝子，是東京歷史最古老、也是最具代表性的玻璃製造商之一，製作出許多美觀且有質感的玻璃器皿。


:: 堅持手工製作的硝子品牌 ::


廣田硝子成立於明治維新時期，當時日本在東京品川創立了官營的「品川硝子製造所」，自此東京下町的玻璃製造廠商紛紛起家，將西方傳來的玻璃製造技術與日本美學意識相融合，製作日本獨特的玻璃設計，其中就包括廣田硝子。

創始人是當時年僅26歲的廣田金太先生，在東京市田町建立了一家專門批發玻璃製品的小商鋪，主要販賣玻璃食器，容器和玻璃燈罩等，打下了堅實基礎後於1915年在墨田區開了一家屬於自己玻璃工廠「江東硝子工廠」，也就是後來的廣田硝子。


到了第二代繼承人榮次郎，他吸取了大正與昭和時期的新技術，創造出與其他公司完全不同的產品，使得廣田硝子開闢出屬於自己的道路。

儘管經過動盪的戰亂時期，但廣田硝子依舊小心翼翼地延續下來，成為了一個見證過東京的動亂與繁榮，也見證過不同時代風貌的古老品牌。不管時代如何變更，廣田硝子120多年來始終貫徹了代代相承的「追求傳達人之感性，器之溫度」的理念，堅持手工製作玻璃製品，這些產品隨著廣田硝子的基調延續下來。



廣田硝子相信親手製作的器物會帶來機器生產所不能共鳴的溫度和情感，也期望以傳統技術為支持，創造出新產品。


:: 獨有的江戶切子 ::


在《日本人家必備的杯子，竟然是「吹」出來的》一文中，我們曾介紹過硝子的製作工藝：每一隻硝子誕生的背後，都經歷了燒、熔、吹、切、磨的工藝歷程，飽含了手工藝人繁複的心思和高超的水準。

但正如廣田硝子現任社長廣田達朗先生所說的「每家公司的生存之道，就在於它與眾不同的地方」，廣田硝子最引以為傲的，是其獨有的玻璃製作技能「江戶切子」，這是日本現存的唯一從江戶時代流傳下來玻璃切割技術。


江戶切子原本是指在江戶時代流行的一種手工玻璃，因工藝精美華麗而聞名，後演變為一種製作工藝的稱呼。


在固態的硝子冷卻之後才進行加工，由匠人全程手工在表面刻磨出墨線，寬深的紋路被砂輪雕刻完畢後再用細輪勾勒出纖細，修整後拋光打磨出精美的花樣和紋理。整個過程沒有草稿，一氣呵成，不允有誤。



而江戶切子最大的特徵就是彩色的表面玻璃非常輕薄，如果要成功地裁切，動作必須非常謹慎且細心，若稍有閃失，內層透明玻璃就會出現在表面上。

這是對手藝人是極高考驗，據廣田硝子的匠人說，要能成為獨當一面的專家，負責所有的工程至少需要10年的時間，這也是為什麼硝子在日本到處可見，但真正做出藝術的卻屈指可數。


獨特的技術造就了江戶切子與眾不同的美麗，不用上各種各樣的顏色，僅僅通過切割的反射就能讓玻璃綻放出如同鑽石的光芒，展現了日式寧靜、質樸通透的美學精神。



:: 各不相同的系列產品 ::


雖說最廣為人知的是江戶切子技藝，但實際上，廣田硝子的每個不同系列產品都有自己的獨特的工藝技巧，也展現出了不同的樣貌，各具特色。


// 昭和時代系列


廣田硝子目前一條重要的產品線是對品牌之前非常經典且優秀，卻由於各種各樣的問題而不得不停產的產品進行「復刻」，其中就包括昭和時代系列。

昭和時代系列是以昭和時期在喫茶店所使用的非常盛行的咖啡杯為靈感進行復刻的。

©️禾希有物


這款昭和時代的復古咖啡杯採用當時的手壓技術，由手工吹制而成，用耐熱玻璃製作，即便是熱飲也能放心使用。

它兼具了美觀和實用，復古的琥珀色低調而時髦，玻璃晶瑩剔透，底部還有精細的雕刻，質感十足，捧在手上時觸感溫潤。


©️禾希有物


無論是當作咖啡杯、酒杯或水杯都可以，優秀的品質也讓它成為送禮的佳選。


// 雪花系列


相比之下，雪花系列是看起來更加新潮和浪漫。採用昭和時期留下的模具製作，有杯子和甜品碗的組合，邊緣的乳白色是由人工烘烤而成，與透明的食器互相映襯，彷彿是片片雪花飄落而至，添加了一份柔和。

©️禾希有物


杯子線條分明，宛如一朵鬱金香般修長立體，透明的杯身就如夏天一般清涼怡人，最適宜承裝夏天的飲品、果汁；甜品碗小巧迷你，與小勺搭配使用，用來吃雪糕最好不過，也可以用來裝小點心或配菜。

©️禾希有物


©️禾希有物


// 雪月花系列


這個系列最引人注意的是器皿上浮現的一個個小圓點，當陽光照射進來，有種光彩四溢的迷離感。這是以日本經典的「霰紋」作為主題創作的，透過將高溫的玻璃流入模具中，並加壓制成形，也是廣田硝子獨特的技法之一，相比吹制玻璃更加堅固，花紋也清晰可見。


©️禾希有物


杯子和碗碟可搭配成套，會是你餐桌上兩眼的一員，即使是喝水也很好看，裝和式、洋式的小點心都非常適合。

©️禾希有物


// 酒器套組系列


酒器套組是廣田硝子一直製作的經典暢銷款，倒上清酒或者自己釀泡的果酒，與家人、好友小酌一杯。

「淺櫻」以柔軟而微微發光的櫻花為主題，酒壺酒杯呈淡粉色，下半部分有如霧霾妝的渲染，握在手中是磨砂質感，杯底還有花蕾的印刻，彷彿手中拿著一朵櫻花，置身花叢中一般。

©️禾希有物


瓶身透明可見酒色，浪漫的粉紅色裝飾春天空間，除了自用也適宜送禮，有著愛與希望的寓意。

©️禾希有物


「吹雪」則是以冬日飛雪作為主題，瓶身杯身均經過磨砂處理，瓶子是不規則的凹凸設計，造型別致外更是便於手握，杯口設計貼合嘴形。整體素雅簡潔卻不失優雅，在家也能感受日式料理店的氛圍。

©️禾希有物

©️禾希有物


// 醬油瓶系列

醬油瓶系列擁有「緩柔細出、急停立止、一滴不漏」三大令廣田硝子引以為傲的特質。

©️禾希有物


千筋透明款醬油瓶是橢圓的瓶肚，持握舒適，由層層線條設計組成。倒醬油時，扭緊瓶蓋，細緩流出，不多不少，也不會把醬油流出瓶身。除了醬油外，也可以放入醋、白酒或其他調味品。

©️禾希有物


廣田硝子的每一款玻璃器皿總能給人帶來第一眼的驚喜，光是放在桌上看著就覺得心情愉悅。陷於美貌，忠於質感，只有拿上手方能感受到質感的不同尋常，細心愛護，便能夠長久相伴。
