金屬的化學性質記憶點

2020-10-29
| 科技



1．金屬的物理通性有哪些？

（1）金屬在常溫下的狀態

除汞是液體外，其他在常溫下是固體。

（2）金屬的顏色、光澤

絕大多數金屬都是銀白色，具有金屬光澤，少數金屬是特殊顏色如銅是紫紅色，金是金黃色。

（3）良好的導電、導熱性。

（4）延展性

延性：拉成細絲的性質。展性：壓成薄片的性質。

2．化學通性有哪些？

（1）化合態金屬元素只有正化合價

（2）金屬單質易失電子，表現還原性

（3）易與氧氣反應，得到氧化物

（4）活動性排在氫前的金屬元素與酸反應得到鹽和氫氣

（5）與鹽反應，置換出活動性弱的金屬單質

3.金屬鈉的性質有哪些？

（1）物理性質有哪些？

鈉銀白色、質軟、熔點低、密度比水的小但比煤油的大。★

（2）化學性質有哪些？

①很活潑，常溫下：4Na + O2＝2Na2O ★（新切開的鈉放在空氣中容易變暗）

②加熱條件下：2Na+O2 Na2O2

★（先熔化成小球，後燃燒產生黃色火焰，生成淡黃色固體Na2O2。）

鈉在空氣中的變化過程：Na―→Na2O―→NaOH―→Na2CO3·10H2O（結晶）―→Na2CO3（風化），最終得到是一種白色粉末。一小塊鈉置露在空氣中的現象：銀白色的鈉很快變暗（生成Na2O），跟著變成白色固體(NaOH)，然後在固體表面出現小液滴（NaOH易潮解），最終變成白色粉未（最終產物是Na2CO3）。

③鈉與水的反應與H2O反應

2Na＋2H2O＝2NaOH＋H2↑ ★離子方程式：2Na＋＋2H2O＝2Na＋＋2OH－＋H2↑（注意配平）

實驗現象：鈉浮在水面上，熔成小球，在水面上遊動，有哧哧的聲音，最後消失，在反應後的溶液中滴加酚酞，溶液變紅。“浮——鈉密度比水小；遊——生成氫氣；響——反應劇烈；熔——鈉熔點低；紅——生成的NaOH遇酚酞變紅”。

知識拓展：

a：將鈉放入硫酸銅溶液中，能否置換出銅單質？

不能，2Na + 2H2O = 2NaOH + H2↑

2NaOH + CuSO4 = Cu(OH)2↓ + Na2SO4

實驗現象：鈉熔成小球，在液麵上四處遊動，有藍色沉澱生成，有氣泡放出

K、Ca、Na三種單質與鹽溶液反應時，先與水反應生成相應的鹼，鹼再和鹽溶液反應

b：將鈉放入鹽酸中，鈉將先和H2O反應，還是先和HCl反應？

2Na + 2HCl = 2NaCl + H2↑

鈉與酸反應時，如酸過量則鈉只與酸反應，如酸不足量則鈉先與酸反應再與水反應。

一般規律：

鈉與酸溶液反應時，鈉直接與溶液中的酸反應，但當鈉與其它溶液反應時，一般是鈉先與水反應生成氫氧化鈉和氫氣，然後再看生成的氫氧化鈉是否與原溶液中的溶質反應！

（3）鈉的保存方法是什麼？

①鈉必須保存在隔絕空氣環境中，實驗室將鈉保存在煤油或石蠟中。

②鈉著火的處理？用乾燥的砂土撲滅

3、鈉的存在：以化合態存在。

4、鈉的保存：保存在煤油或石蠟中。

5、工業制鈉：電解熔融的NaCl：2NaCl(熔融) 2Na + Cl2↑★

6、鈉的用途：① 在熔融的條件下鈉可以製取一些金屬，如鈦、鋯、鈮、鉭等；

② 鈉鉀合金是快中子反應堆的熱交換劑；

③ 鈉蒸氣可作高壓鈉燈，發出黃光，射程遠，透霧能力強。

4.鋁的性質

1、物理性質：銀白色金屬，質較軟，但比鎂要硬，熔點比鎂高。有良好的導電、導熱性和延展性。

2、化學性質：鋁是較活潑的金屬。

① 通常與氧氣易反應，生成緻密的氧化物起保護作用。4Al + 3O2 == 2Al2O3。

（Fe形成的氧化膜疏鬆，不能保護內層金屬。Mg和Al形成的氧化膜緻密，能保護內層金屬不被繼續氧化。)

加熱鋁箔的現象和解釋是什麼？

現象：鋁箔熔化，表面失去光澤，熔化的鋁並不滴落。

現象解釋：（因為鋁表面有一層氧化膜保護了鋁。即使打磨過的鋁箔，在空氣中也會很快生成新的氧化膜。鋁與空氣形成了一層緻密的氧化膜，阻止內層金屬繼續與氧氣發生反應。）

同時也容易與Cl2、S等非金屬單質反應。

② 鋁的兩性體現在哪裡？

①鋁與稀鹽酸的反應

化學方程式為：2Al + 6HCl == 2AlCl3 + 3H2↑

離子方程式為：2Al + 6H+ == 2Al3+ + 3H2↑

(與酸反應：強氧化性酸，如濃硫酸和濃硝酸在常溫下，使鋁發生鈍化現象；加熱時，能反應，但無氫氣放出；非強氧化性酸反應時放出氫氣。（ 2Al＋6H＋＝2Al3＋＋3H2↑ ）)

③②鋁與氫氧化鈉溶液的反應

化學方程式為：2Al + 2NaOH + 2H2O == 2NaAlO2 + 3H2↑

離子方程式為：2Al + 2OH- + 2H2O == 2AlO2- + 3H2↑

反應可以看成兩步進行：

第一步：2Al + 6H2O == 2Al(OH)3↓ + 3H2↑

第二步：Al(OH)3 + NaOH = NaAlO2 + 2H2O

所以在這個反應中Al做還原劑，H2O水做氧化劑。

鋁既能與強酸反應，又能與強鹼反應，是兩性金屬。

與強鹼溶液反應：2Al + 2NaOH + 2H2O == 2NaAlO2 + 3H2↑

（ 2Al＋2OH－＋2H2O＝2AlO2－＋3H2↑）

鋁既能與強酸反應，又能與強鹼反應，是兩性金屬。

④ 與某些鹽溶液反應：如能置換出CuSO4、AgNO3等溶液中的金屬。

⑤ 鋁熱反應：（瞭解就行）

鋁與某些金屬氧化物的反應（如V、Cr、Mn、Fe的氧化物）叫做鋁熱反應

2Al + Fe2O3 Al2O3 + 2Fe。Al 和 Fe2O3的混合物叫做鋁熱劑。利用鋁熱反應焊接鋼軌。

（1）Na2O 和Na2O2 有哪些性質？

①氧化鈉是白色固體，過氧化鈉是淡黃色固體。

②鹼性氧化物有哪些性質？

1、Na2O

白色固體，是鹼性氧化物，具有鹼性氧化物的通性： Na2O + H2O == 2NaOH, ★

Na2O + CO2 == Na2CO3, ★

Na2O + 2HCl == 2NaCl + H2O .★

另外：加熱時，2Na2O + O2 == 2Na2O2

鹼性氧化物+水→鹼 Na2O + H2O = 2NaOH

鹼性氧化物+酸性氧化物→含氧酸鹽 Na2O + CO2 = Na2CO3

鹼性氧化物+酸→鹽+水 Na2O + 2HCl = 2NaCl + H2O

Na2O屬於典型的鹼性氧化物。

③2、Na2O2：淡黃色固體是複雜氧化物，易與水和二氧化碳反應。

Na2O2與H2O的反應

2Na2O2 + 2H2O == 4NaOH + O2 ★；

離子反應方程式：2Na2O2 + 2H2O = 4Na+ + 4OH- + O2↑

實質：Na2O2 + 2H2O = 2NaOH + H2O2 2H2O2 = 2H2O + O2↑

實驗現象：產生氣泡，試管壁發燙，溶液(滴加酚酞)先變紅，後褪色

④Na2O2與CO2的反應

2Na2O2 + 2CO2 = 2Na2CO3 + O2

Na2O2與H2O、CO2的反應中，Na2O2既是氧化劑又是還原劑，這兩個反應是典型的歧化反應。Na2O2不屬於鹼性氧化物。

因此Na2O2常做生氧劑，供氧劑，同時，Na2O2還具有強氧化性，有漂白作用。如實驗：Na2O2和水反應後的溶液中滴加酚酞，變紅後又褪色。

⑤Na2O2的用途

可做漂白劑 （強氧化性漂白），呼吸面具或潛艇中的供氧劑。

（2）Na2CO3和NaHCO3的性質有哪些？

①碳酸鈉俗名純鹼，也叫蘇打，為白色粉末狀固體，碳酸氫鈉俗名小蘇打，為白色細小晶體。碳酸鈉晶體的化學式是Na2CO3·10H2O，碳酸鈉晶體在乾燥的空氣中容易失去結晶水變成碳酸鈉粉末。

②與酸反應(HCl)

a、Na2CO3 + 2HCl ==2NaCl + CO2 ↑ + H2O

CO32- + 2H+ == CO2 ↑ + H2O

b、NaHCO3 + HCl ==NaCl + CO2↑ + H2O

HCO3- + H+ == CO2 ↑ + H2O

結論：碳酸鈉和碳酸氫鈉都易與酸反應，但是碳酸氫鈉與酸反應比碳酸鈉更劇烈。

碳酸鈉與酸反應之所以緩慢是因為：

Na2CO3 + HCl =NaCl + NaHCO3（無氣泡產生）

NaHCO3 + HCl = NaCl + CO2↑ + H2O（有氣泡）

③與鹽反應(BaCl2和CaCl2)

Na2CO3與可溶性的鈣鹽、鋇鹽反應生成沉澱

Na2CO3 + CaCl2 = CaCO3↓ + 2NaCl CO32- + Ca2+ = CaCO3↓

Na2CO3 + BaCl2 = BaCO3↓ + 2NaCl CO32- + Ba2+ = BaCO3↓

NaHCO3可與NaHSO4反應

NaHCO3 + NaHSO4 = Na2SO4 + H2O + CO2 ↑

HCO3- + H+ = H2O + CO2 ↑

應用：利用能否與可溶性的鈣鹽、鋇鹽反應生成沉澱來來鑑別Na2CO3與NaHCO3

④與鹼反應(Ca(OH)2和Ba(OH)2 )

Na2CO3 + Ca(OH)2 = CaCO3↓ + 2NaOH CO32- + Ca2+ = CaCO3↓

Na2CO3 + Ba(OH)2 = BaCO3↓ + 2NaOH CO32- + Ba2+ = BaCO3↓

NaHCO3與 Ca(OH)2(或Ba(OH)2 )反應遵循“以少定多”的原則

當NaHCO3少量時，NaHCO3+ Ca(OH)2= CaCO3↓ + NaOH + H2O

HCO3- + Ca2+ + OH- = CaCO3↓ + H2O

當Ca(OH)2少量時，2NaHCO3 + Ca(OH)2= CaCO3↓ + Na2CO3 + 2H2O

2HCO3- + Ca2+ + 2OH- = CaCO3↓ + 2H2O + CO32-

Na2CO3易與Ba2+、Ca2+的鹼反應生成碳酸鹽沉澱和NaOH

NaHCO3能與所有可溶鹼反應生成碳酸正鹽和水

例如：NaHCO3 + NaOH = Na2CO3 + H2O

HCO3- + OH- = CO32- + H2O

⑤熱穩定性

碳酸鈉受熱不分解，碳酸氫鈉不穩定，受熱易分解

2NaHCO3 Na2CO3 + H2O + CO2↑

結論：加熱時，碳酸鈉不分解，而碳酸氫鈉則分解。即：碳酸鈉比碳酸氫鈉更穩定。

所以利用它們對熱的穩定性來鑑別它們。

2.焰色反應的概念和實驗步驟是什麼？

（1）概念：

金屬或它們的化合物在灼燒時使火焰呈現特殊的顏色，這在化學上叫焰色反應。

（2）焰色反應是物理變化，反映的是元素的性質。元素原子發射一定波長的光而產生各種顏色。

（3）鈉的焰色是黃色，鉀透過藍色的鈷玻璃焰色是紫色。

（4）實驗步驟

①將鉑絲（或光潔無鏽的鐵絲）用稀鹽酸洗滌後放在酒精燈外焰上灼燒至火焰顏色與原來相同；

②用鉑絲蘸取待測碳酸鈉溶液放在外焰上灼燒，觀察火焰顏色；

③用鹽酸洗淨鉑絲，在外焰上灼燒至火焰無色後，再蘸取碳酸鉀溶液在外焰上灼燒，透過藍色的鈷玻璃觀察火焰的顏色。

（5）節日燃放的五彩繽紛的煙花，就是鹼金屬、以及鍶、鋇等金屬化合物所呈現的各種豔麗的色彩。

3.鋁的重要化合物有哪些？

（1）氧化鋁（Al2O3）有哪些性質？

①物理性質：Al2O3是一種白色難熔的固體（熔點高（2054℃），沸點2980℃），可作耐火材料，不溶於水。

②我們常見到的寶石的主要成分是氧化鋁。有各種不同顏色的原因是在寶石中含有一些金屬氧化物的表現。如紅寶石因含有少量的鉻元素而顯紅色，藍寶石因含有少量的鐵和鈦元素而顯藍色。氧化鋁(天然產物稱作剛玉)，剛玉氧化鋁的天然晶體，硬度大小僅次於金剛石、金剛砂（SiC），工業生產中的礦石剛玉主要成分是α－氧化鋁，硬度僅次於金剛石

③化學性質——兩性氧化物

Al2O3是兩性氧化物。

兩性氧化物：既能與強酸反應又能與強鹼反應生成鹽和水的氧化物。

與鹽酸反應：Al2O3 + 6HCl = 2AlCl3 + 3H2O

Al2O3 + 6H+ = 2Al3+ + 3H2O

與氫氧化鈉反應：Al2O3 + 2NaOH = 2NaAlO2 + H2O

Al2O3 + 2OH¯= 2AlO2- + H2O

既能和強酸又能和強鹼反應生成鹽和水的氧化物。但不與弱鹼氨水（NH3·H2O）反應。

Al2O3是工業冶煉鋁的原料，由於氧化鋁的熔點高，電解時，難熔化，因此鋁的冶煉直到1886年美國科學家霍爾發現在氧化鋁中加入冰晶石(Na3AlF6 )，使氧化鋁的熔點降至1000度左右，鋁的冶煉才快速發展起來，鋁及其合金才被廣泛的應用。2Al2O3 4Al + 3O2↑

（2）氫氧化鋁(Al(OH)3 ) ——兩性氫氧化物的性質有哪些？

①物理性質：氫氧化鋁是幾乎不溶於水的白色固體。在水溶液中新生成的氫氧化鋁呈膠狀，能凝聚水中懸浮物，又有吸附色素的性能。

②用途：氫氧化鋁膠體可用作淨水劑、色素吸附劑等。胃舒平又名複方氫氧化鋁，

能治療胃酸過多。

③實驗室製備氫氧化鋁——氨水（NH3·H2O）與鋁鹽反應

現象：生成白色膠狀沉澱

Al2(SO4)3 + 6NH3·H2O＝2Al(OH)3↓ + 3(NH4)2SO4

Al3+ + 3NH3·H2O ＝ Al(OH)3↓ + 3NH4+

④AlCl3與NaOH的反應

AlCl3 + 3NaOH = Al(OH)3↓ + 3NaCl Al3+ + 3OH- = Al(OH)3↓

AlCl3 + 4NaOH = NaAlO2 + 3NaCl + 2H2O Al3+ + 4OH = AlO2- + 2H2O

⑤Al(OH)3的化學性質

a、不穩定性：加熱易分解

2Al(OH)3 Al2O3 + 3H2O

b、與鹽酸反應：Al(OH)3 + 3HCl = AlCl3 + 3H2O Al(OH)3 + 3H+ = Al3+ + 3H2O

c、與氫氧化鈉反應：Al(OH)3 + NaOH = NaAlO2 + 2H2O

Al(OH)3 + OH¯ = AlO2- + 2H2O

既能和強酸又能和強鹼反應生成鹽和水的氫氧化物。但不與弱鹼氨水（NH3·H2O）反應。是兩性氫氧化物。

（3）NaAlO2的化學性質有哪些？

①NaAlO2與鹽酸的反應

鹽酸少量時：NaAlO2 + HCl + H2O = Al(OH)3↓ + NaCl

AlO2- + H+ + H2O = Al(OH)3↓

鹽酸過量時：NaAlO2 + 4HCl = AlCl3 + NaCl + 2H2O

AlO2- + 4H+ = Al3+ + 2H2O

②NaAlO2與CO2的反應

CO2少量時：2NaAlO2 + CO2 + 3H2O = 2Al(OH)3↓ + Na2CO3

2AlO2- + CO2 + 3H2O = 2Al(OH)3↓ + CO32-

CO2過量時：NaAlO2 + CO2 + 2H2O = Al(OH)3↓ + NaHCO3

AlO2- + CO2 + 2H2O = Al(OH)3↓ + HCO3-

（4）硫酸鋁鉀{ KAl(SO4)2 }是如何淨水的？

KAl(SO4)2·12H2O俗名明礬或白礬，無色晶體、易溶於水，是一種複鹽，常用作淨水劑。電離方程式：KAl(SO4)2 == K+ + Al3+ + 2SO42-

鋁離子與水反應，生成氫氧化鋁膠體，具有很強的吸附能力，吸附水中的懸浮物，使之沉降已達淨水目的。

淨水原理：Al3+ + 3H2O == Al(OH)3 (膠體)+ 3H+ 。

四）鋁的冶煉（瞭解就行）

鋁是地殼中含量最多的金屬元素，自然界中主要是以氧化鋁的形式存在。工業生產的流程：鋁土礦（主要成分是氧化鋁）→用氫氧化鈉溶解過濾→向濾液中通入二氧化碳酸化，過濾→氫氧化鋁→氧化鋁→鋁。

主要反應：Al2O3 + 2NaOH == 2NaAlO2 + H2O ，

2Al(OH)3 2O3 +3 H2O ,2Al2O3 4Al + 3O2↑。

（五）鋁的用途：鋁有良好的導電、導熱性和延展性，主要用於導線、炊具等，鋁的最大用途是制合金，鋁合金強度高，密度小，易成型，有較好的耐腐蝕性。迅速風靡建築業。也是飛機製造業的主要原料。

知識整理：

①（Al(OH)3）的製備：在氯化鋁溶液中加足量氨水。AlCl3 + 3NH3·H2O == Al(OH)3↓+ 3NH4Cl ★

② 實驗：A、向氯化鋁溶液中滴加氫氧化鈉溶液，現象是先有沉澱，後溶解。

反應式：先Al3+ + 3OH- == Al(OH)3↓, 後Al3+ + 4OH- == AlO2- + 2H2O。

B、向氫氧化鈉溶液中滴加氯化鋁溶液，現象是開始無沉澱，後來有沉澱，且不溶解。

反應式：先Al3+ + 4OH- == AlO2- + 2H2O，後Al3+ + 3AlO2- + 6H2O == 4Al(OH)3↓。

③ 實驗：向偏鋁酸鈉溶液中通二氧化碳，有沉澱出現。CO2 + 3H2O + 2NaAlO2 == 2Al(OH)3↓+ Na2CO3。

④ 將氯化鋁溶液和偏鋁酸鈉溶液混和有沉澱出現。 Al3+ + 3AlO2- + 6H2O == 4Al(OH)3↓。

⑤ 實驗：A、向偏鋁酸鈉溶液中滴加稀鹽酸，先有沉定，後溶解。

反應的離子方程式：AlO2- + H+ + H2O == Al(OH)3 ,Al(OH)3 + 3H+ == Al3+ + 2H2O 。

B、向稀鹽酸中滴加偏鋁酸鈉溶液，先無沉澱，後有沉澱且不溶解。

反應的離子方程式：AlO2- + 4H+ == Al3+ + 2H2O ,3AlO2- + Al3+ + 6H2O == 4Al(OH)3↓。

4.鐵的重要化合物有哪些？（膠體）

（1）鐵的氧化物（FeO、Fe2O3、Fe3O4）的性質有哪些？

①物質性質

FeO是一種黑色粉末，不穩定，在空氣裡會迅速被氧化為Fe3O4。

Fe2O3是一種紅棕色粉末，俗稱鐵紅。用於制紅色油漆，塗料。是赤鐵礦的主要成分。

Fe3O4是一種複雜的化合物，是具有磁性的黑色晶體，俗稱磁性氧化鐵，是磁鐵礦的主要成分。

FeO、Fe2O3、Fe3O4的比較

②化學性質

a、不溶於水也不與水反應

b、與酸反應：

FeO + 2HCl = FeCl2 + H2O FeO + 2H+ = Fe2+ + H2O

Fe2O3 + 6HCl = 2FeCl3 + 3H2O Fe2O3 + 6H+ = 2Fe3++3H2O

氧化亞鐵，氧化鐵是鹼性氧化物

Fe3O4 + 8HCl = 2FeCl3 + FeCl2 + 4H2O Fe3O4 + 8H+ = 2Fe3+ + Fe2+ + 4H2O

c、FeO不穩定 6FeO + O2 = 2Fe3O4

d、均能被CO還原成Fe

Fe2O3 + 3CO ===高溫 2Fe + 3CO2

Fe3O4 + 4CO ===高溫 3Fe + 4CO2

（2）鐵的氫氧化物（Fe(OH)3和Fe(OH)2）的性質有哪些？
