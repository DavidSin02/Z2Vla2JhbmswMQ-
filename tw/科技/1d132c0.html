模塊二　物質構成的奧祕

命題點15 化學式的意義及其相關計算、化合價

(時間：30分鐘　分值：28分)

一、選擇題

1. (2016瀋陽)P2O5的化學名稱是( )

A. 二磷化五氧 B. 五氧化二磷

C. 氧五化磷二 D. 磷二化氧五

2. (2016蘇州)下列物質化學式書寫正確的是( )

A. 鋁：AL B. 氯化鉀： KCl2

C. 硫酸銅：CuSO4 D. 四氧化三鐵：O4Fe3

3. (2016合肥38中一模)NaNO2有劇毒，外貌酷似食鹽且有鹹味，NaNO2中N元素的化合價為( )

A. －3 B. ＋1 C. ＋2 D. ＋3

4. (2016阜陽市潁泉區二模)銅是生活、生產中重要的金屬，常見的黃銅礦中，主要成分是CuFeS2，已知銅元素在該物質中的化合價為＋2價，則該物質中鐵元素的化合價為( )

A. ＋2 B. 0 C. ＋3 D. ＋4

5. 鈦鐵礦的主要成分的化學式為FeTiOx，其中鐵元素和鈦元素的化合價均為＋3。則x為( )

A. 2 B. 3 C. 4 D. 6

6. (2016泉州)下列物質中，氯元素的化合價最高的是( )

A. ClO2 B. HClO

C. NaCl D. Cl2

7. (2016合肥45中一模)生日蛋糕中的奶油含有對人體有害的反式脂肪酸，其化學式為C18H34O2。下列說法正確的是( )

A. 反式脂肪酸是氧化物

B. 反式脂肪酸由碳、氫、氧三種元素組成

C. 反式脂肪酸由54個原子構成

D. 反式脂肪酸中碳、氧元素的質量比是9∶1

8. 風油精是家庭常備藥，常用於蚊蟲叮咬、暈車等引起的不適。它的有效成分之一是丁香酚(C10H12O2)，下列有關丁香酚的說法正確的是( )

A. 丁香酚的相對分子質量是164 g

B. 一定條件下，丁香酚可以燃燒生成CO2和H2O

C. 丁香酚中C、H、O元素質量比為10∶12∶2

D. 丁香酚的一個分子中含1個氧分子

9. (2016阜陽二模)安徽黃山盛產茶葉，茶葉中含有茶氨酸(C7H13O3N2)、鋅、硒等多種成分。茶樹適宜在pH為5～6的土壤中生長。下列說法不正確的是( )

A. 茶樹不宜在鹼性土壤中生長

B. 茶氨酸由碳、氫、氧、氮四種元素組成

C. 茶葉中所含茶氨酸屬於有機物，鋅、硒屬於單質

D. 一個茶氨酸分子中含有25個原子

10. (2016南昌)南昌漢代海昏侯墓出土的文物中，保存完好的玉器非常罕見。對玉石主要成分(化學式為NaAlSi2O6)的描述正確的是( )

A. 含有3個氧分子

B. 相對分子質量為94

C. 鈉元素的質量分數為13.4%

D. 鈉元素與鋁元素的質量比為23∶27

11. (2016黔南州)高鐵酸鉀(K2FeO4)是一種集吸附、凝聚、殺菌等功能為一體的新型高效水處理劑，可將水中的三氯乙烯(C2HCl3)除去85.6%，下列說法正確的是( )

A. K2FeO4是氧化物

B. C2HCl3中三種元素的質量比是2∶ 1∶3

C. K2FeO4中Fe元素的化合價為＋3價

D．三氯乙烯中碳元素的質量分數約為18.25%

12. (2016孝感)我國科學家用濾紙和二氧化鈦(TiO2)薄膜製作出一種新型"納米紙"，又在納米紙上"鋪"一層"萘胺"(C10H9N)染料，製成一種試紙，用於檢測食品中亞硝酸鹽濃度的高低。下列說法正確的是( )

A. 二氧化鈦中Ti的化合價為＋2價

B. 萘胺中氮元素的質量分數約為11.9%

C. 這種試紙是一種新型有機化合物

D. 萘胺中碳、氫、氮三種元素的質量比為120∶9∶14

二、填空和簡答題

14. (2016邵陽)按要求用化學用語填空：

(1)兩個硫原子____________。

(2)鋁離子____________。

(3)－1價的氯元素________。

(4)碘酒中的溶劑________。

1. B　【解析】由兩種元素組成的化合物的名稱，一般讀作某化某，有時還要讀出化學式中各種元素的原子個數，因此，P2O5讀作五氧化二磷。故選B。

2. C　【解析】鋁是金屬單質，用鋁的元素符號表示化學式，即Al，A錯誤；氯化鉀中鉀元素為＋1價，氯元素為－1價，化學式應為KCl，B錯誤；根據化合物中各元素正負化合價代數和為零，判斷硫酸銅化學式正確，C正確；四氧化三鐵中鐵是金屬元素，寫在前面，氧為非金屬元素，寫在後面，D錯誤。

3. D　【解析】亞硝酸鈉中鈉元素顯＋1價，氧元素顯－2價，設氮元素的化合價為x，根據化合物中正負化合價的代數和為零，有1＋x＋(－2)×2＝0，解得x＝＋3。

4. A　【解析】CuFeS2中銅元素顯＋2價，硫元素顯－2價，設鐵元素的化合價為x，根據化合物中正負化合價的代數和為零，有2＋x＋(－2)×2＝0，解得x＝＋2。

5. B　【解析】根據化合物中正負化合價的代數和為零，有3＋3－2x＝0，解得x＝3。

6. A　【解析】根據化合物中各元素正負化合價的代數和為0的原則，ClO2中O顯－2價，Cl顯＋4價；HClO中H顯＋1價，O是－2價，Cl顯＋1價；NaCl中Na顯＋1價，Cl顯－1價；Cl2中Cl顯0價；因此，A中氯元素的化合價最高，故選A。

7. B　【解析】氧化物是由兩種元素組成，且其中一種元素是氧元素的化合物，反式脂肪酸由碳、氫、氧三種元素組成，A錯誤，B正確；反式脂肪酸由分子構成，一個反式脂肪酸分子由54個原子構成，C錯誤；反式脂肪酸分子中碳原子與氧原子的個數比為9∶1，由於不同原子的質量不同，所以元素的質量比不等於原子的個數比，D錯誤。

8. B　【解析】相對分子質量的單位是"1"，常省略不寫，A錯誤；根據質量守恆定律和物質燃燒的規律可知，丁香酚充分燃燒生成水和二氧化碳，B正確；丁香酚分子中碳、氫、氧元素的原子個數比為10∶12∶2，由於不同原子的質量不同，所以元素的質量比不等於元素原子個數之比，C錯誤；丁香酚能用化學式表示，屬於純淨物，所以丁香酚中不可能含有氧分子，D錯誤。

9. C　【解析】pH小於7顯酸性，即茶樹適合在偏酸性土壤中生長，A正確；根據茶氨酸的化學式可知，茶氨酸由碳、氫、氧、氮四種元素組成，B正確；根據茶氨酸的組成可知，茶氨酸屬於有機物，但鋅和硒指的是元素，不是單質，C錯誤；一個茶氨酸分子中含7＋13＋3＋2＝25個原子，D正確。

11. D　【解析】氧化物是由兩種元素組成且其中一種元素是氧元素的化合物，K2FeO4由三種元素組成，不是氧化物，A錯誤；C2HCl3中三種元素的質量比為(12×2)∶1∶(35.5×3)＝24∶1∶106.5，B錯誤；K2FeO4中K顯＋1價，O顯－2價，設Fe的化合價為x，根據化合物中各元素的化合價代數和為0，可得(＋1)×2＋x＋(－2)×4＝0，x＝＋6價，C錯誤；三氯乙烯中碳元素的質量分數為eq \f(12×2,12×2＋1＋35.5×3)×100%≈18.25%，D正確。

12. D　【解析】氧元素顯－2價，設鈦元素的化合價是x，根據在化合物中元素正負化合價代數和為零，可得x＋(－2)×2＝0，則x＝＋4價，A錯誤；該"萘胺"中氮的質量分數為eq \f(14,12×10＋1×9＋14)×100%≈9.8%，B錯誤；二氧化鈦不含碳元素，不屬於有機物；"萘胺"(C10H9N)是含有碳元素的化合物，屬於有機物，這種試紙是由二者組成的混合物，C錯誤；萘胺中碳元素、氫元素和氮元素的質量比為(12×10)∶(1×9)∶(14×1)＝120∶9∶14，D正確。

13. C　【解析】根據乙酸分子的模型可以寫出乙酸的化學式，化學式為C2H4O2；由化學式可知，乙酸是由碳、氫、氧三種元素組成的化合物，乙酸的相對分子質量＝12×2＋1×4＋16×2＝60；乙酸中碳元素的質量分數＝eq \f(12×2,60)×100%＝40%，C錯誤；乙酸分子中碳、氫、氧的原子個數比為2∶4∶2＝1∶2∶1。

14. (1)2S　(2)Al3＋　(3)Ceq \o(l,\s\up6(－1))　(4)C2H5OH

【解析】(1)硫原子用符號"S"表示，2個硫原子在前面加上數字"2"。(2)每個鋁離子帶3個單位的正電荷。(3)氯的元素符號為"Cl"，再在元素符號正上方標上"－1"。(4)碘酒中的溶液是酒精，化學式為C2H5OH。
