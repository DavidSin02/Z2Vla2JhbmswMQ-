註冊消防工程師-〖技術實務〗氣體滅火系統模擬題

2020-10-29
| 科技

1、【單選題】二氧化碳氣體滅火系統不適用於撲救（ ）

A 電氣火災

B 固體表面火災

C 金屬氫化物火災

D 滅火能量切斷氣源的氣體火災

2、【單選題】某自備發電機房採用七氟丙烷滅火系統進行保護，系統的設計濃度為9.1%，則該系統的滅火濃度可以為（ ）

A 7%

B 9.1%

C 10.01%

D 11.83%

3、【單選題】某博物館文物的展覽部分，設計採用氣體滅火裝置進行保護。下列關於該區域的噴頭設置，說法錯誤的是（ ）

A 展廳噴頭的安裝高度為6m

B 噴頭的安裝高度距頂板0.4m

C 噴頭在防護區內均勻佈置

D 展櫃噴頭的安裝高度為1.3m，保護半徑為5m

4、【單選題】某電子計算機房設置了七氟丙烷滅火系統，七氟丙烷滅火劑噴放時間不應大於（ ）S；滅火浸漬時間應採用（ ）min

A 5， 5

B 6， 10

C 8， 5

D 10，10

5、【單選題】關於氣體滅火系統管道應滿足的要求，下列說法不正確的是（ ）

A 啟動氣體的管道宜採用銅管

B 輸送氣體滅火劑的管道應採用無縫鋼管，無縫鋼管應進行內外防腐處理

C 管道的公稱直徑為80mm時，宜採用法蘭連接

D 管道的公稱工作壓力，不應小於在最高環境溫度下所承受的工作壓力

6、【單選題】某大型城市綜合體中的變配電間，計算機主機房、通訊設備間等場所內設置了組合分配式七氟丙烷氣體滅火系統。下列關於該系統組件的說法中，錯誤的是（ ）

A 集流管應設置安全洩壓裝置

B 選擇閥的公稱直徑應和與其對應的防護區滅火系統的主管道的公稱直徑相同

C 輸送啟動氣體的管道應採用銅管

D 選擇閥應在容器閥動作之前或同時打開

7、【多選題】某住宅小區的變配電間，建築面積為300㎡，淨空高度為3m，劃分為一個防護區並採用無管網七氟丙烷氣體滅火裝置進行防護。委託消防檢查機構對該氣體滅火系統進行檢測，下列檢測結果中，符合現行國家消防技術標準要求的有（ ）

A 該防護區的裝置動作響應時間差為1s

B 防護區內設置8臺預製滅火裝置

C 防護區門口未設手動與自動控制的轉換裝置

D 儲存容器的充裝壓力為4.2MPa

E 該系統未設置備用量

8、【多選題】某市規劃建設大型數據中心，共4層，層高5m，每層建築面積2200㎡。由於該建築內平常有人值班，該信息機房內設計採用IG541組合分配式氣體滅火系統。下列有關該機房氣體滅火的設計不符合規範要求的是（ ）

A 該機房內滅火系統的啟動方式有自動啟動和手動啟動兩種啟動方式

B 信息機房內設有可開啟式甲級防火窗，但未設置機械排風裝置

C 該建築內共劃分12個防護區

D 該系統不可以設置為無延遲的噴射方式

E 系統儲存容器採用焊接容器或無縫容器

==============

答案

1 C

【解析】氣體滅火系統的適用範圍

適用於撲救：

（1）電氣火災

（2）液體表面或可融化固體火災

（3）固體表面火災

（4）滅火前可切斷氣源的氣體火災

不適用於撲救：

（1）含氧化劑的化學制品

（2）活潑金屬

（3）金屬氫化物

（4）能自行分解的化學物質

2 A

【解析】根據《氣體滅火系統設計規範》3.3.1七氟丙烷滅火系統的滅火設計濃度不應小於滅火濃度的1.3倍，惰化設計濃度不應小於惰化濃度的1.1倍。9.1%/1.3=7%，故選A。

3 D

【解析】噴頭的安裝高度和保護半徑

（1）最大保護高度不宜大於6.5m

（2）最小保護高度不應小於0.3m

（3）噴頭安裝高度小於1.5m時，保護半徑不宜大於4.5

（4）噴頭安裝高度不小於1.5m時，保護半徑不應大於7.5m

（5）噴頭宜貼近防護區頂面安裝，距頂面的最大距離不宜大於0.5m

4 C

【解析】教材P240，七氟丙烷滅火系統在通信機房和電子計算機房等防護區，設計噴放時間不應大於8s，在其他防護區，設計噴放時間不應大於10s；滅火浸漬時間的規定：木材、紙張、織物宜採用20min，通信機房、電子計算機房應採用5min，其他固體表面火災宜為10min，氣體和液體火災不應小於1min。

5 C

【解析】教材P246，公稱直徑小於等於80mm的管道，宜採用螺紋連接，公稱直徑大於80mm的管道宜採用法蘭連接。

6 C

【解析】

（1）為了防止儲存器發生誤噴射，在集流管末端設置安全洩壓裝置

（2）選擇閥的公稱直徑應和與其對應的防護區滅火系統的主管道的公稱直徑相同

（3）輸送啟動氣體的管道宜採用銅管

（4）輸送氣體滅火劑的管道可採用無縫鋼管、不鏽鋼管

（5）選擇閥應在容器閥動作之前或同時打開

7 BE

【解析】《氣體滅火系統設計規範》GB50370-2005 3.1.14、3.1.15

詳細【解析】根據題意可知，該系統為預製式氣體滅火系統。根據3.1.14、3.1.15“一個防護區設置的預製滅火系統，其裝置數量不宜超過10臺，同一防護區內必須能同時啟動，動作響應時間差不得大於2s”A項錯誤，B項正確；預製系統儲存容器的充裝壓力不應大於2.5mpa，D項錯誤；預製系統不需要設置備用量，E項正確。

8 ACE

【解析】根據《氣體滅火系統設計規範》5.0.2管網滅火系統應設自動控制、手動控制和機械應急操作三種啟動方式，故A項不符合規範要求；6.0.4滅火後的防護區應通風換氣，地下防護區和無窗或設固定窗扇的地上防護區，應設置機械排風裝置，排風口宜設在防護區的下部並直通室外，故B符合要求；根據3.2.4採用管網滅火系統時，一個防護區的面積不宜大於800㎡，且容積不宜大於3600m³，本題根據面積2400/800=3個，根據體積2400×5/3600=4個，每層劃分為4個防護區，該建築4層×4個=16個防護區；根據5.0.3對於平時無人工作的防護區，可設置為無延遲的噴射，故D項符合要求；根據4.3.1，IG541氣體滅火系統的儲存容器應採用無縫容器。
