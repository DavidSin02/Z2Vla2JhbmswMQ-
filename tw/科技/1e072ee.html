卸料臂種類

LNG卸料臂是一種裝卸設備，當LNG船抵達LNG接收站碼頭後，通過液相卸料臂和卸料管線，藉助船上卸料泵將LNG輸送至接收站的儲罐內。

根據平衡的方式，將卸料臂分為全平衡型、旋轉平衡型和雙平衡型三類。

1、卸料臂控制系統

  • 邏輯控制檯（Electro-logic console) ：

主要負責成套包及接收站DCS/ESD之間信息的交互傳送。

  • 就地控制盤 (local control panel)

用於四臺臂的操作控制。

  • 液壓動力單元（Hydraulic power pack )

液壓動力單元主要包括：液壓油罐和電動泵，備用泵（手泵）及相關管路和附件。主要為液壓系統提供動力。

  • 便攜式無線遠程控制單元（Portable radio remote control unit ）

用於遠程進行四臺臂的操作控制

  • 液壓操縱箱 (Hydraulic cabinets)

包含液壓蓄能器（hydraulic accumulator ），閥門，接線盒等液壓系統元件。以便在電力故障時，為ERC 的操作和臂的鎖定提供能量。

  • 卸料臂的位置監控系統（Constant position Monitoring System suitable for monitoring 4arms）

主要包含角度探測器等硬件及配套的軟件，用於監測各臂的位置。

試車前準備工作

確認卸料臂及其附屬設備、管道、儀表、電氣已安裝合格，安全閥已調校裝上，按流程圖進行檢查，確認完整無誤；

各附屬設備和管道的安裝經檢驗符合規範要求，試壓、吹掃、置換合格；

檢查風速，處於運行（操作/連接）狀態的卸料臂的最大許可風速為22m／s；

各支承、聯接螺栓已緊固，液壓泵的油位處於1/2～2/3處；

卸料臂及其附屬管線水壓、氣密試驗合格；

公用工程系統能連續提供水、電、氮氣、儀表空氣、工廠空氣等；

消防系統已測試完畢並處於備用狀態；

試車所需的工器具及記錄報表已備好；

操作人員培訓合格，具備上崗要求；

機械、電氣、儀表、工藝、製造廠家等技術人員在場。
