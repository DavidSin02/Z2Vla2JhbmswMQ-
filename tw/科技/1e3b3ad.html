01

排水管渠斷面形式

排水管渠對於斷面形式的要求，要滿足四個方面的需求。

  1. 靜力學：具備一定抗荷載能力；
  2. 水力學：需要排水能力好，不容易淤積；
  3. 經濟性：造價低；
  4. 養護性：要便於沖洗和清淤。

排水管渠一般根據管徑範圍確定其屬於小型、中型、大型或者特大型管，具體的劃分原則如下。

  1. 小型管：管徑＜600mm；
  2. 中型管：600mm≤管徑≤1000mm；
  3. 大型管：1000mm＜管徑≤1500mm；
  4. 特大型管：管徑＞1500mm。

排水管渠斷面形式，常用的形式有九種：圓形、半橢圓形、馬蹄形、拱頂矩形、弧形、矩形、弧形流槽的矩形、帶低流槽的矩形、梯形。具體斷面形式如下九幅簡圖所示。

這9種斷面形式中，需要重點注意圓形、半橢圓形、馬蹄形、蛋形、低流槽矩形和梯形斷面形式。這幾種斷面形式的特性及應用可以總結如下。

  1. 圓形：適用於指定坡度和斷面面積下，水力半徑最大，最常用的斷面形式；
  2. 半橢圓形：適用於土壓力及動荷載較大時，可以更好的分配管壁壓力，可減小壁厚，適用於流量無大變化及管渠直徑＞2m的情況；
  3. 馬蹄形：適用於受地下水位限制，需減小埋深以降低造價，且流量無大變化的大流量汙水的排除；
  4. 蛋形、低流槽矩形：適用於流量變化大的情況；
  5. 梯形：適用於明渠。

02

常用排水管渠材料

排水管渠對於材料有以下5點要求。

  1. 強度要求：能夠承受外部荷載和內部水壓；
  2. 耐沖刷、磨損、抗腐蝕要求：因為輸送的汙水大多具有腐蝕性，所以要求管渠材料必須耐腐蝕，其接口及附屬構築物必須採取相應的防腐蝕措施；
  3. 不透水性要求：要能夠防止汙水滲出管外或者地下水滲到管內；
  4. 內壁光滑性要求：汙水在管內流動的水流阻力要小；
  5. 能夠就地取材：要儘量降低管渠的造價和運輸及施工費用。

常用排水管渠的材料主要有以下5種。

①混凝土管和鋼筋混凝土管：其優點是便於就地取材，製造方便，且能夠根據抗壓的不同要求，製成無壓管、低壓管、預應力管等。鋼筋混凝土管及預應力鋼筋混凝土管亦可用作泵站的壓力管及倒虹管。其缺點就是抗酸鹼浸蝕和抗滲性能較差、管節短、接頭多、施工複雜。在地震烈度大於8度的地區及飽和鬆砂、淤泥土、衝填土、雜填土等地區不宜採用。

②陶土管：適用於排除酸性廢水，或管外有侵蝕性地下水的汙水管道。

③金屬管：價格昂貴，且鋼管扛酸鹼腐蝕及地下水侵蝕的能力差。

④漿砌石或鋼筋混凝土渠道：就地取材，便於建造敷設。

⑤新型材料管道：最常見的包括PVC-U、玻璃鋼管以及HDPE管等。

對於這幾類管材，需要知道的一點就是，哪些屬於剛性管道，哪些屬於柔性管道？其中剛性管道包括鋼筋混凝土管道、預（自）應力混凝土管道和預應力鋼桶混凝土管道。對於埋設於地下的矩形或拱形管道結構，均應屬於剛性管道。而柔性管道包括：鋼管、化學建材管等。

03

排水管渠接口

排水管渠接口一般可分為五種形式：剛性接口、柔性接口、半鋼半柔接口、塑料管道接口、頂管接口。其中經常考得是剛性接口和柔性接口，以及其使用條件。

①剛性接口包括水泥砂漿抹帶接口和鋼絲網水泥砂漿抹帶接口兩種。

②柔性接口包括石棉瀝青卷材接口、橡膠圈接口、鋼邊止水帶三種。尤其需要注意的是柔性接口的使用條件，《室外排水設計規範》4.3.4條規定，管道接口應根據管道材質和地質條件確定，可採用剛性接口或柔性接口，汙水及合流管道應選用柔性接口。當管道穿過粉砂、細砂層並在最高地下水位以下，或在地震設防烈度為7度及以上設防區時，必須採用柔性接口。

③半剛半柔性接口包括預製套環石棉水泥（或瀝青砂漿）接口。

④塑料管道的接口包括單密封圈承插連接、雙密封圈承插連接、套管承插連接、螺紋連接、膠黏劑承插粘結、電熱熔帶連接等。

⑤頂管接口包括混凝土（或鑄鐵）內套環石棉水泥接口，一般只用於汙水管道、瀝青油氈、石棉水泥接口，麻辮（或塑料圈）石棉水泥接口，一般只用於雨水管道。

對於接口部分內容，還應注意一點就是《室外排水設計規範》4.3.4A，當矩形鋼筋混凝土箱涵敷設在軟土地基或不均勻地層上時，宜採用鋼帶橡膠止水圈結合上下企口式接口的形式。

04

排水管渠基礎

管道基礎應根據管道材料、接口形式和地質條件確定，對地基鬆軟或不均勻沉降地段，管道基礎應採取加固措施。

對於排水管渠的基礎，有以下三種：砂土基礎、混凝土枕基、混凝土帶形基礎。其中砂土基礎適用於無地下水、原土能挖成弧形的乾燥土壤；混凝土枕基適用於乾燥土壤中的雨水管道及不太重要的汙水支管；混凝土帶形基礎適用於各種潮溼土壤以及地基軟硬不均勻的排水管道。

對於基礎部分內容，把握一個原則，就是剛性管道使用剛性基礎，柔性管道使用柔性基礎。假如在地震烈度較高地區或者地下水位較高地區，則不論剛性柔性管道，都使用混凝土帶形基礎。

05

本節總結

本節內容基本都是知識選擇題，基本不會涉及到案例考點，所以理解起來相對較簡單，總結起來有以下幾點：

  1. 管渠的9種斷面及其適用條件；
  2. 常用5種排水管材的材質；
  3. 常用5種管渠接口形式，尤其重視柔性接口的使用條件，汙水及合流管道應選用柔性接口。當管道穿過粉砂、細砂層並在最高地下水位以下，或在地震設防烈度為7度及以上設防區時，必須採用柔性接口；
  4. 常用的3種管渠基礎。
