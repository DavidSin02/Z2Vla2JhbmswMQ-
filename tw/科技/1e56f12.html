變頻器現場常見15個問題及解決方法

問題1：DCS模擬信號給定引起電流波動故障現象：

（1）變頻器在由DCS4-20mA信號控制，穩定運行時發現工頻輸入端電流波動太大，DCS系統監控該電流波形呈鋸齒狀，變化範圍在10A左右。

（2）變頻器在由DCS4-20mA信號控制時，報“模擬量斷線”故障，用萬用表實測該4-20mA直流信號，發現與DCS系統給定電流相同。

（3）變頻器在由DCS4-20mA信號控制，穩定運行時發現風機工頻輸入端電流波動太大，DCS系統監控該電流波形呈鋸齒狀，變化範圍在15A左右。

故障原因：

由於用戶信號源不穩定或者直流信號受外部信號干擾，導致變頻器給定頻率不穩，變頻器不斷在進行頻率調整，從而引起工頻端輸入電流不斷變化，電流波動較大。

解決方法：

（1）在DCS給定信號到主板信號採集迴路之間加裝一隻有源隔離變送器。

（2）把變頻器117號功能參數（給定頻率閥值）由0改為30，無須加裝隔離變送器。

問題2：DCS頻率給定變頻器不響應問題故障現象：

（1）遠方DCS給定一定頻率，變頻器觸摸屏接受到頻率後不進行轉速調節。

故障原因：

PLC判斷系統處於“遠控”方式時，主控才能接受到遠方4~20ma信號進行頻率調節。因此出現DCS給定頻率系統不調速的主要原因為1）主控接受的控制方式（功能號207）不對；2）面板控制方式下的頻率給定模式（功能號208）不對.

解決方法：

（1）旋動控制櫃門上的旋動按鈕，使功能號207為1，即遠控方式。

（2）選擇面板控制方式下的頻率給定模式，功能號為1，即模擬輸入AI頻率給定。

問題3：“請合高壓”問題

故障現象：

（1）變頻器在由“系統就緒”狀態變為“請合高壓”狀態，過程變化延時只設定了60S，在斷開高壓60S以後，“請合高壓”上傳到DCS，操作人員重新將高壓合上，導致16個模塊保險燒燬。

（2）變頻器由於用戶拒絕引入“請合高壓”狀態到DCS系統，在因其他設備故障而引起變頻器高壓斷掉後，操作人員沒按規程延時300S後再合高壓，而是緊急搶合變頻器高壓開關，導致其中16個模塊保險燒燬。

故障原因：

由於變頻器高壓掉電後，模塊中電容裡邊的電不能立即放掉，需要一定的時間，此時重新合上高壓，導致短路，將保險燒燬。

解決方法：

（1）將“系統就緒”狀態到“請合高壓”狀態時間設為300S。

（2）現場安裝調試一定要將“請合高壓”信號接入用戶DCS系統，並讓熱工在合閘迴路中做好聯鎖保護。

問題4：系統輸出過流、過載

故障現象：

（1）變頻器正常運行過程系統輸出過流或輸出過載故障，導致變頻器重故障停機。

（2）變頻器在升速過程中系統輸出過載或系統過流故障停機

（3）變頻器在啟動過程中報變頻器輸出過流。

故障原因：

（1 ) 變頻器在正常運行過程中突然輸出過載或過流可能的原因是母線電壓波動，突加大負載的啟動，或者變頻輸出電流採樣迴路故障引起變頻電流採集過大。

（2 ) 電流傳感器故障或者主板信號採集迴路故障，導致變頻器誤動作。

（3 ) 變頻器在升速過程中輸出過載或過流主要是因為升速時間過快。

（4 ) 由於變頻器啟動過程負載（主要是風機由於對側風機作用處於反轉）處於墮轉狀態或者電機負載處於堵轉。

解決方法：

（1）正常過程中變頻器突然過載過流主要是確認是什麼原因引起的跳閘，如果是電網變化或負載突變引起就重新啟動變頻器，如果是變頻器採集迴路本身故障引起就要檢查相應的連線和霍爾傳感器。

（2）變頻器在升速過程中系統輸出過載或系統過流引起故障停機就要修改相應的上、下升速時間，把時間儘可能設大。

（3）啟動時確保風機負載處於靜止狀態；修改風機的DCS啟動邏輯，在電機啟動之前關所有的入口和出口擋板，啟動後再打開，避免電流的衝擊；先工頻啟動後再啟動變頻器；在功能號裡設定轉矩提升，增加變頻器的啟動轉矩。

問題5：模塊直流過壓

故障現象

（1）變頻器在停機降速過程中，多次出現模塊直流過壓故障，導致將用戶高壓開關跳掉。

（2）用戶母線電壓過高，6KV電源實際母線達6.3KV以上，10KV電源實際母線達10.3KV以上，母線電壓加到變頻器上時模塊輸入電壓過高，模塊報直流母線過壓。

（3）變頻器在啟動過程中，大約到運行到4HZ左右，變頻器直流母線過壓。

故障原因：

（1 ) 變頻器在停機過程由於降速時間太快，使得電機處於發電機狀態，電機回饋能量到模塊的直流母線產生泵升電壓，從而使直流母線電壓過高。

（2 ) 由於現場變壓器出廠標準接線是10KV和6KV，母線電壓如果超過10.3KV或6.3KV，就會使變壓器輸出電壓過高，從而使模塊的母線電壓升高造成過壓。

（3) 同一位置的不同相模塊光纖接反（比如A4與B4光纖接反），造成其相電壓輸出過壓。

解決方法：

（1）將上降速時間和下降速時間適當的延長。

（2）將模塊內過壓保護點提高，現在全部是1150V。

（3）用戶電壓達到10.3KV（6KV）以上將變壓器短接端改為10.5KV(6.3KV)。

（4）檢查光纖是否插接錯誤，把接錯的光纖改正過來。

問題6：模塊通訊故障

故障現象：

1、變頻器運行過程重故障跳閘停機，觸摸屏報模塊通訊故障。

故障原因：

（1）模塊的輸入保險、整流橋、充電電阻燒斷導致模塊控制失電通訊無法進行

（2）主板上的光通子板本身故障或者通訊電路供電電源的保護二極管保護。

（3）連接光纖插錯位置或者光纖折斷或破損

（4）模塊裡的電源板輸出電壓不正常或無輸出，導致模塊通訊中斷

解決方法：

（1）打開模塊蓋板，更換模塊中已經損壞的保險、充電電阻等元器件。

（2）更換損壞的光通子板或者保護二極管

（3）光纖按照標示正常連接，光纖損壞的話進行更換。

（4）更換模塊電源板。

問題7：模塊驅動故障

故障現象：

變頻器運行過程系統輕故障，模塊旁通運行，觸摸屏報模塊驅動故障。

故障原因：

（1）模塊相應的驅動電路燒壞。

（2）控制板上的D25二極管等器件損壞

（3）旁通迴路損壞，柵極保護板及IGBT損壞，驅動板故障。

解決方法：

更換報驅動故障相應的模塊。

問題8：模塊過熱故障

故障現象：

變頻器運行過程系統輕故障，模塊旁通運行，觸摸屏報模塊過熱故障。

故障原因：

（1）變頻器頂部冷卻風機故障停止轉動導致模塊開關器件產生的熱量不能及時散出變頻器櫃內，大量熱量聚集模塊裡導致過熱。

（2）頂部風機反轉導致模塊開關器件產生的熱量不能及時散出變頻器櫃內，大量熱量聚集模塊裡導致過熱。

（3）變頻室的空調故障導致變頻器內部散出的熱量不能及時進行冷熱交換從而引起變頻器模塊過熱故障。

（4）變頻器頂部裝有專門的散熱風道，但整個變頻器室密閉，變頻器運行一段時間冷卻風機把變頻器室的空氣帶出室內，而變頻器室沒預留一定的進風口，空氣流入流出不平衡，導致室內負壓從而影響變頻器散熱。

解決方法：

（1）把故障的冷卻風機系統恢復正常；

（2）確保冷卻風機正轉；

（3）及時處理空調的故障讓空調正常製冷；

（4）保持室內進出風平衡，變頻器室預留一定進風口，讓室內保持正壓。

問題9：主板與PLC通訊

故障現象：故障

（1 ) 變頻器觸摸屏報主板與PLC通訊故障，PPI電纜接收燈RX燈滅或不閃爍。

主板各個電源指示燈熄滅。高壓電仍加在模塊輸入，模塊輸出封鎖。

（2）變頻器觸摸屏報主板與PLC通訊故障，PPI電纜發送燈TX燈滅或不閃爍。

3) 觸摸屏報主板與PLC通訊故障，PPI電纜電源燈POW不亮。

故障原因：

（1 )、變頻器在運行過程中由於主板的供電開關電源PW1或者主板上的直流電源部分故障，導致整個主板的電源失電，IGBT開關信號停止。因此報出主板與PLC通訊故障；模塊封鎖輸出；同時由於主板失電，故障跳閘信號無法發出，高壓電一直加在模塊上。

（2）主板接收不到PLC發送的通訊信號，PLC本身通訊部分有問題。

（3）、PPI電纜本身故障。

解決方法：

（1）、更換損壞的主板或開關電源。

（2）、更換PLC

（3）、更換PPI電纜。

問題10：觸摸屏故障信息顯示問題

故障現象：

查看觸摸屏故障信息時發現無故障信息或者故障信息要經過較長時間才能正常顯示。

故障原因：

（1）觸摸屏CF卡沒有安裝導致故障信息無法顯示。

（2）CF卡本身質量問題或者連接不正常就使故障信息在讀取時速度緩慢

解決方法：

（1）安裝備用的CF卡

（2）更換容量更大的CF卡。

問題11：“高壓掉電”後報輕、重故障不定的問題故障現象：

變頻器運行過程中直接跳進線開關或者高壓失電後，變頻器本應該報重故障“高壓掉電”，但是觸摸屏顯示輕故障，模塊欠壓。

故障原因：

由於高壓失電檢測延時1s，而欠壓可能低於1s，也可能長於1s，這要取決模塊放電時間的快慢，如果欠壓時間低於1s，那就會先報輕故障模塊欠壓。

問題12：觸摸屏顯示輸出電流偏大問題故障現象：

（1）變頻器通過端子短接相關狀態模擬運行時，觸摸屏顯示較大電流值

（2）變頻器高壓加電正常運行時輸出顯示電流比實際測量值大。

故障原因：

（1）霍爾電流傳感器的電源線與信號線接錯導致輸出顯示較大電流

（2）霍爾傳感器本身損壞

（3）功能號220輸出線電流量程或者320模擬輸出通道2調幅參數設置不準確

解決方法：

（1）更換錯誤的接線或已經損壞的霍爾傳感器

（2）重新修正功能號。

問題13：變頻器開關狀態量遠方（DCS）不顯示問題

故障現象：

（1）變頻器正常運行過程中一些相應的狀態量DCS不能正常顯示。

故障原因：

（1）端子排到DCS櫃連線錯誤或接觸不好

（2）控制櫃內輸出繼電器損壞或著繼電器線圈與底座接觸不好

解決方法：

（1）檢查DCS連線

（2）檢查PLC與輸出繼電器以及端子排的連線，同時檢查繼電器線圈與底座的連接。

問題14：帶瞬停功能變頻器在運行過程中報瞬停時間過長問題

故障現象：

（1）變頻器正常運行過程中突然報瞬停時間過長或者輸出過流導致變頻器故障跳閘。

故障原因：

（1）變頻器瞬停功能通過信號檢測板檢測變壓器三相電源和變頻器輸出殘壓信號，如果信號檢測板故障的話就可能採集傳送錯誤的信號。導致變頻瞬停失敗。因為母線電壓根本沒有瞬時停電。

解決方法：

（1）檢查三相電源信號的檢測線路是否連接良好。

（2）更換新的信號檢測板。

問題15：變頻器帶負載測試過程中風門全開運行不到50HZ

故障現象：

變頻器帶負載在風門全開的情況下進行調試，變頻器運行不到額定轉速（50HZ）就已經達到額定電流。

故障原因：

（1）由於風機在調試過程中機組處於停運狀態，吹的風處於冷態；而機組正常發電過程風是熱態，其風的空氣密度明顯小於冷態。因此在調試過程（冷態）下變頻器帶風機的工作電流比在同樣的風門開度下發電過程中（熱態）的電流要大。所以調試過程風門全開就可能運行不到額定頻率。

（2）由於輸入母線電壓偏低或者變壓器的輸出電壓偏低，導致變頻器在運行過程中比正常情況下的電流要偏大。這樣會使變頻器運行不到額定頻率（50HZ）就已經達到額定電流。

解決方法：

（1）保持風門與變頻運行一樣的開度工頻試轉，看是否電流同樣過大。如果同樣電流過大，那麼說明冷態下運行就比熱態運行電流過大。變頻器不存在任何問題。

（2）測試輸入電壓或變壓器的輸出電壓，如果偏低。更換變壓器檔位，抬高模塊的輸入電壓，從而增大輸出電壓，降低變頻器的輸出電流。
