劍指 Offer 51. 數組中的逆序對

2020-10-29
| 科技

在數組中的兩個數字，如果前面一個數字大於後面的數字，則這兩個數字組成一個逆序對。輸入一個數組，求出這個數組中的逆序對的總數。例如輸入數組[7,5,6,4]，則有逆序對[7,5]，[7,6]，[7,4]，[5,4]，[6,4]。一共6個逆序對。

這個是我一個曾經的面試題，當時只要求說思路，沒讓寫代碼，這次記錄一下代碼。

首先，看到逆序對，第一反應應該就是冒泡排序中數據交換的次數，但是冒泡排隊時間複雜度較高。再次觀察，我們可以思考到另一種排序方案，歸併排序。以數組[7,5,6,4]為例子，具體思路如下圖：

1、對數組的拆分，排序合併。

2、在每一個合併步驟時，兩邊數組都是有序的，當找到第一個逆序對時，右邊數組後面的數據與當前比較的左邊數組的數據全部可生成逆序對。

3、歸併排序完成，逆序對個數計算也完成。時間複雜度優於冒泡排序。

具體代碼如下：

def mergesort(list):    if len(list) <= 1:        return list, 0    mid = len(list)//2    left_li,l_cnt = mergesort(list[:mid])    right_li,r_cnt = mergesort(list[mid:])    #排序所需的輔助空間    temp = []    cnt = l_cnt + r_cnt		#the index of the left_list and the index of the right_list    lindx, rindx_p = 0, 0     #Merging the ordered arrays    while lindx < len(left_li) and rindx_p < len(right_li):        if left_li[lindx] > right_li[rindx_p]:         		temp.append(left_li[lindx])            cnt += len(right_li[rindx_p:])            lindx += 1        else:            temp.append(right_li[rindx_p])            rindx_p += 1    temp += left_li[lindx:]    temp += right_li[rindx_p:]    return temp, cntif __name__ == "__main__":    l=list(map(int,input().split()))    list,res=mergesort(l)    print(res)
