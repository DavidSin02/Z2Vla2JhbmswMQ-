今天的主題是黎曼重排定理。定理斷言，“條件收斂的實數項級數通過重排可以收斂到任意實數”。我們接下來將要對此詳細說明，暫時看不懂這個定理的人也請放心。


無窮級數絕對收斂是指，級數各項取絕對值也收斂。


就像“絕對收斂”這個名稱的字面意思那樣。


相對地，條件收斂是指無窮級數收斂但不是絕對收斂。


比如，平方數的倒數之和是絕對收斂。



自然數的倒數的交錯級數是條件收斂。



也許會有人提出疑問“為什麼要關心絕對收斂和條件收斂呢？”，這是有原因的。


絕對收斂級數，不論哪種求和順序都收斂到同一個值。總之，不需要關心求和順序。


另一方面，對於條件收斂級數，收斂的值隨著求和順序而改變。條件收斂也太頑皮了呢。

更有趣的是，根據本文開頭提到的黎曼重排定理，對於條件收斂級數，通過改變求和順序，可以使級數收斂到任意實數。



不管怎麼說，“任意的實數”給人很不顯然的感覺呢。


定理的完整內容和證明，參見網上的其他。


總之定理是可以證明的。在這裡沒有詳細證明，會讓人迷迷糊糊摸不著頭腦，感覺好像很難的樣子。


想要好好的理解定理，試著去看前面提到的證明，是可以比預想中更清晰地理解的。而且，如果仔細地閱讀證明，就會注意到證明之中包含了讓級數收斂到任意實數的方法。


不管怎麼說，我們能讓級數收斂到喜歡的實數值，這應該是很有趣的！

用於驗證的python代碼如下所示。有興趣的話請試著把玩一下。


r = 3.14159265 # 在這裡輸入收斂到的實數值
#r = 2 # 在這裡輸入收斂到的實數值


def a_pos(n_pos):
return 1/(2*n_pos+1)

def a_neg(n_neg):
return -1/(2*n_neg+2)


n_pos = 0
n_neg = 0
sum = 0

pos_neg_flag = 1 # 1: pos, -1: neg

for i in range(5):
print("(ans) ".format(2*n_pos+1), end='')
if pos_neg_flag > 0:
while sum < r:
sum += a_pos(n_pos)
#print(sum)
print("+ 1/{}".format(2*n_pos+1), end='')
n_pos += 1
else:
while sum > r:
sum += a_neg(n_neg)
#print(sum)
print("- 1/{}".format(2*n_neg+2), end='')
n_neg += 1
print(" =",sum)
pos_neg_flag *= -1 # pos, neg改變符號
