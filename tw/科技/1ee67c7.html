win10桌面鼠標右擊卡頓怎麼辦？刪除右鍵菜單中的圖形選項就行啦

是否有小夥伴們會遇到這樣的問題，當系統升級到win10後，一個很奇怪的現象——鼠標右擊桌面變卡頓了，還有一些其他小問題，比如網絡速度也慢了。頓時想說，這是在逗我玩嗎？新裝的固態硬盤，驅動是最新的，顯卡驅動卸載了重裝還是卡頓，下血本重買個新鼠標換上也不行。最後按照網上win10的優化教程進行了設置，還是不行。(┬＿┬)！

那麼小編想跟遇到這種情況的小夥伴們說，問題不在於你的配置，其實是代號為1709版的win10系統出現了一個小bug——intel核顯與獨顯切換的bug。

如何解決卡頓問題

當我們鼠標右擊桌面的時候，會彈出菜單，其中，有個“intel的圖形選項和圖形屬性”，有雙顯卡的電腦還會有一個“Nvidia的圖形控制面板選項”。就是他們引起了桌面右擊卡頓的問題，那麼我們只需要將其中的“intel圖形選項和圖形屬性”清理刪除掉即可解決卡頓的問題，Nvidia的圖形控制面板選項不用刪除。那麼小編接下來就教大家如何刪除右鍵菜單中的intel圖形屬性和圖形選項。

清理刪除圖形屬性與圖形選項

如何刪除右鍵菜單中的intel圖形屬性和圖形選項

第一步：我們首先要查看自己的win10系統版本代號是不是1709的，如果不是，這個方法不一定能解決卡頓問題，不過也可以值得一試。怎麼看？很簡單！按住組合鍵【Win + R】調出“運行”窗口，然後輸入“winver”命令並按“確定”，即可顯示win10的版本號，如圖：

查看win10的版本號

第二步：我們新建一個記事本，複製粘貼如下“刪除intel圖形選項和圖形屬性”的代碼

Windows Registry Editor Version 5.00

[-HKEY_CLASSES_ROOT\Directory\Background\shellex\ContextMenuHandlers\igfxDTCM]

@="{9B5F5829-A529-4B12-814A-E81BCB8D93FC}"

我們將記事本重命名為“刪除圖形屬性和圖形選項”，並將文件名擴展名後綴“.txt”重命名為“.reg”，變成一個註冊表文件，最後雙擊reg註冊表文件完成修改刪除。如圖：

刪除圖形屬性和圖形選項的註冊表值

注意事項

  1. 修改註冊表文件前，我們需要備份整個註冊表，以防萬一修改失敗，我們可以及時還原恢復。如何備份還原註冊表呢？請閱讀這篇文章《win10的註冊表文件怎麼備份？又該如何還原恢復呢？》

  2. 如果文件名擴展名後綴無法看到，我們需要設置“文件夾選項查看”，並將“已知文件類型的擴展名”顯示出來，取消隱藏。具體設置方法，請閱讀這篇文章《文件名後綴怎麼顯示？教你在win10中顯示文件擴展名的方法》

第三步：我們將電腦重啟一下直接進入win10系統桌面後，我們測試發現，在系統桌面空白處鼠標右鍵點擊，右鍵菜單中的“圖形屬性”和“圖形選項”菜單已經消失，而且延遲卡頓的問題也沒有了，但是可能並不完美，還會有那麼一丁點兒的延遲。習慣就好，習慣就好，^_^||

圖形屬性和圖形選項菜單已經消失

結語

萬一有小夥伴們測試不成功，依舊卡頓！想恢復顯示右鍵菜單中的“圖形屬性”和“圖形選項”，怎麼辦？很簡單，我們還是按照第二步的操作，新建一個記事本，複製粘貼如下“添加intel圖形選項和圖形屬性”的代碼

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\Directory\Background\shellex\ContextMenuHandlers\igfxDTCM]

@="{9B5F5829-A529-4B12-814A-E81BCB8D93FC}"

我們將記事本重命名為“添加圖形屬性和圖形選項”，並將文件名擴展名後綴“.txt”重命名為“.reg”，變成一個註冊表文件，最後雙擊reg註冊表文件完成修改添加。最後就可以恢復如初啦！

“刪除與添加intel圖形選項和圖形屬性”這2段代碼其實是有點細微的區別的，不仔細發現的小夥伴們還以為是一樣的代碼，區別效果圖如下：
