發射器、接收器和突發編碼器，美國CIA模塊化短波間諜無線電設備

2020-11-14
| 科技


六十年代美國中情局研發的模塊化短波間諜電臺RS-59

RS-59是一種模塊化短波（SW）間諜無線電設備，由美國中央情報局（CIA）的技術服務部門（TSD）於20世紀60年代末開發。該套裝包括一個RR-59接收器，一個RT-59發射器，一個CK-8脈衝編碼器和幾個接口和附件。儘管沒有關於這一組的詳細信息，但從Stasi 2檔案中可以看出，在冷戰期間，前DDR（東德）已經抓獲了一套完整裝備。

RS-59是一款低功率中程無線電臺，適用於長達600英里的距離。它與現有的遠程RS-49互補，後者具有更高的輸出功率。 RS-59由中央情報局內部完全開發，由中央情報局和其他人建造。

下圖顯示了RT-59發射器，它是目前保存在集合中的RS-59組的唯一部分。該裝置尺寸為105 x 38 x 63 mm，重量僅為328 g。所有連接（天線除外）均通過右前方的滑動連接器進行。

RS-59的開發和改進從1966年到1976年進行了10年。RR-59接收器是第一個發佈的單元，原型於1966年5月上市。那時，RT-59發射機和RS-59電臺的其餘部分“預計很快”。為了減少互連電纜的數量，同時由（未知）外部方開發了一種新型的高質量薄型滑動連接器。

在接下來的幾年中，由於電子部件的短缺，RS-59接收機組件的生產被推遲了幾次。此外，組件被改進了幾次，導致型號中的附加標識符，例如，RR / C-59和RT-59A。

1976年，推出該套裝10年後，RT-59發射器由改進的RT-59A繼承，提供10瓦的輸出功率。到1976年6月，前30個單位由中央情報局內部生產，還有50個單位正在進行中，另外還有專門建造的CC-59便攜箱。其他地方將生產一批另外75個單位。幾年前（1974年）開發了自動20瓦3-30 MHz版RT / B-59。






全套裝備

下圖概述了構成RS-59間諜無線電臺的所有組件。左半部分由無線電設備的實際部分組成，而右半部分則顯示用於以非常高的速度發送預編碼消息的CK-8突發發射機，以便通過手段最小化無線電測向（RDF）攔截和發現的風險。


完整的RS-59間諜電臺。

RS-59接收機的單元（左）通過每個模塊一側的滑動觸點相互連接。電池組和左前方的電源PSU清晰可見這些觸點。 RR / C-59接收器有時作為獨立接收器提供。


完整的RS-59套裝公文包。

上圖是在前斯塔西（MfS）的檔案中找到的。它顯示了一個普通的公文包，其中一個RS-59無線電臺配有CK-8脈衝編碼器（GRA-71）和電纜，存放在保護性泡沫嵌體的切口中。在冷戰期間，它在DDR中被沒收。根據手寫的斯塔西筆記，它來自美國特勤局INSCOM 1 。

發射器RT-59

發射器尺寸為105 x 38 x 63 mm，與接收器大致相同，重量為328克。它產生的最大輸出功率約為5瓦，覆蓋頻率範圍3-16 MHz，分為四個頻段，可通過BAND選擇器選擇。


上圖顯示了一個典型的RT-59裝置，因為它是在1966年設計的。左上方是兩個彈簧式端子，用於連接線天線（ANT）和合適的平衡器（GND）。天線端子右側的4位LOAD選擇器用於選擇天線電路的最佳匹配，使用小型儀表作為指示器。

合適的HC-25 / U晶體可安裝在右前方的2針插座中，而BAND選擇器則設置為相應的頻率範圍：3-5,5-8,8-12或12-16 MHz 。

右上角是MODE開關。在最低位置（RECEIVE），接收器被啟用（通過PSU）。在中心位置，發射器被啟用，但不直接產生信號。在此位置，發射器受突發發射器的控制。為了調節變送器的最大輸出功率，將開關推入（並保持）到KEY位置，並使用微型儀表作為指示器調節TUNE（在左下角）以獲得最大功率。

前表面右側的結構是專門設計的10向滑動連接器，可與PSU上的類似連接器配合使用。通過該連接器，向發射器提供電力，發射器控制接收器，並且突發編碼器控制傳輸。在兩個短邊的中心是圓形防潮壓力膜。更高版本的發射器提供10瓦（RT-59A）甚至20瓦（RT / B-59）。

接收機RR-59


接收器尺寸為100 x 45 x 22 mm。它支持多達10個晶振控制通道，這些通道預先安裝在控制面板左側的可拆卸蓋子下面。實際頻率寫在蓋子上的標籤上。此外，外部晶振可以安裝在左前方的插座上，在這種情況下，11位通道選擇器（插座右側）設置為0。

該設備有一個2針插座，用於連接附帶的耳機，兩個線端子用於連接天線和右上角的平衡器（接地）。接收器覆蓋3至24 MHz的頻率範圍，分為三個頻段。頻段選擇器（位於右前方）也用作電源開關。在最左邊的位置，接收器關閉。

右側的滾花旋鈕（TUNE）用於將RF電路調諧到所選頻率。其餘兩個旋鈕用於音量（GAIN）和Beat Frequency Oscillator（BFO）設置。

雖然這在右側圖像中不可見，但據信應該向設備後部的滑動觸點供電。接收器可以直接由電池或電源PSU（獨立）供電，也可以通過RS-59接口單元（作為整個站的一部分）供電。

突發編碼器CK-8

為了最大限度地降低敵方監測站攔截的風險，或通過無線電測向（RFD）發現，通常的做法是通過使用所謂的突發編碼器結合使用，儘可能短。突發發射器（也稱為鍵控器）。

RS-59配備了CK-8鍵控器/編碼器，這是眾所周知的GRA-71的CIA版本。它使用磁帶盒，通過此處顯示的CO / B-8字母編碼器在其上記錄預編碼（通常加密）的消息。

為此，將CA / A-3B盒式磁帶連接到編碼器的磁頭上。然後將字母表盤設置為消息的第一個字母，然後拉動手柄。然後將這封信記錄在磁帶上。對每個字母重複該過程，直到記錄整個消息。







一旦將信息記錄（編碼）到磁帶上，磁帶就連接到KE-8B鍵控器上，而鍵控器則通過小型AA-59接口（和RS-59單元）連接到RT-59發送器。在重繞磁帶並捲起鍵控器的彈簧機構後，按下一個按鈕以高速發送記錄的信息，稱為突發傳輸。

內部電路

發射器由一塊由實心鋁塊銑制而成的框架組成，其中前面板構成一個整體部件。該裝置採用鋁製外殼封裝，通過三個凹入式螺釘固定到位：一個位於底部，一個位於前部，一個位於右側。

一旦拆下這三個螺釘，就可以取下外殼。這可能需要一點力，因為殼體可以由於在裝置內部使用厚的保形塗層而被束縛。


移除外殼後，內部會露出，如上圖所示。左邊是印刷電路板（PCB），帶有大部分電子部件。右半部分包含控制器，功率放大器（PA）晶體管和一些無源元件。右前方是天線匹配器，由多抽頭環形線圈和可調電容器組成。







環形線圈的抽頭連接到控制面板上的4位帶選擇器，而電容器（TUNE）用於最大化調諧電路的效率，從而最大化發射器的功率輸出。右側是一個微型繼電器，用於在控制面板上的3位MODE開關的控制下在發送和接收之間切換。
