DIY｜裝修邊角料變身家居好物！省出一個億！

2020-11-14
| 科技

裝修邊角料別浪費，只需一些簡單的工具和非常容易買到的輔助材料，就能做出實用的家居好物呢！

1. 摺疊沙灘椅 難度係數 一星

材料和工具：

切出33 x 38釐米的膠合板 2塊

直徑4mm，長度1.5米的聚丙烯繩 1根

直徑4mm,長度1m的彈力繩2個

直徑4mm,長度0.5m的彈力繩 1根

直徑5mm的鑽頭和鑽機

磨料板

操作步驟：

在每塊板上鑽12個孔，直徑為5mm

將直徑4mm,長度0.5m的彈力繩插入板孔，製作椅子的鉸鏈。

在彈力繩的兩端系一個結。

將兩根直徑為4毫米，長度為1米的彈性繩插入板的孔中來製作兩個十字架。

翻轉兩塊板。

在兩根彈性繩的兩端系上一個結。

將直徑4毫米，長度1米50的聚丙烯繩插入板的孔中，製作成椅子的提手。

用繩子的兩端進行漁夫結。

完成！還可以在十字叉內裝雜誌、毛巾等小物品。走到哪裡隨時打開就可以坐啦，非常方便！

​

2. 碗碟架 難度係數 兩星

材料和工具：

塑料托盤 1個

21 x 21 mm, 長2.5 m的木條

S形鉤

不鏽鋼餐具筒 1個

手鋸

螺絲刀

3mm木鑽

5mm金屬鑽

尺子、鉛筆

使用手鋸和斜接箱切割木條：

- 2個長度為20釐米

-5個長度為32.5釐米

塗上顏色

在32.5釐米的木條上，用4毫米木鑽鑽5個孔，擰緊螺栓。

使用4 x 30 mm螺釘組裝成下圖樣式

托盤中鑽5mm的孔，一側3個，距離邊緣10cm，另一側5個，用於瀝水。

在不鏽鋼餐具筒上穿一個孔。

用4 x 35 mm的螺絲將鋼板擰到板條框架上，把餐具筒用S形鉤掛在上面。

完成！這款2合1濾水碗碟架既美觀又巧妙，十分節省空間，可謂廚房必備良品！

​

3. 雨傘架 難度係數 兩顆星

材料和工具：

L.80 X L.60釐米，厚18mm的木板1塊

長1m，直徑22mm的光滑山毛櫸細木棍18個

清漆、彩漆若干，螺釘若干

電鑽，尺子，鉛筆，磨砂紙等

在L.80 X L.60釐米的18mm板上切割出：

- 2個33 x 33釐米的正方形

- 2個33 x 6.5釐米的長方形

- 2個29.4 x 6.5釐米長方形

將長1M,22 X 22mm粗的山毛櫸細木棍切成兩半。

在兩個33 x 33釐米的正方形板上，留出邊緣4cm，每隔5釐cm打一個孔，一共36個。

作為底部的正方形板，用直徑4mm的鑽頭，作為頂部的正方形板，用直徑22mm的鑽頭。

組裝木板做成底座。

將底座塗成灰色，將16個山毛櫸細木棍塗成孔雀藍色，再塗一層無色清漆。

將36個細木棍擰入底座頂部的孔中，然後將它們從底座下方擰入預孔中。

完成！這個功能與裝飾兼備的雨傘架適用於辦公室等公共場所，用於家庭中不妨開發它的其他功能，去掉幾個細木棍也可以的。

​

4. 儲物小邊桌

材料和工具：

18毫米膠合板31.4 x 28.1釐米 4個

18毫米膠合板53.8 x 53.8釐米 1個

10毫米膠合板53.8 x 53.8釐米 1個

木製凳腳直徑50 x H.80 mm x W.50 mm 4個

油漆、密封膠、螺絲、木栓若干

鑽頭、刷子、砂紙、刀具等

切割32件28.1釐米的木條，

在兩側塗上喜歡的顏色。

切割膠合板，尺寸如下：

將4個31.4 x 28.1釐米的膠合板裝到側面標記處，用螺絲擰緊固定。

根據標記預先刺穿底部和頂部，將32個木條裝到側面，用釘子固定。

在上面一個面的邊框兩側中間位置各打一個3mm的孔，將最後一塊板蓋在上面，用木銷釘固定，木銷釘隨時可以拔出，方便邊桌儲物。

將桌腳擰在傢俱下面，還可以刷一層無色清漆或你喜歡的顏色。

完成！這款矮桌既可以作為小邊桌使用，還兼具儲物功能，甚至可以拿來充當臨時凳子，非常實用的小傢俱！
