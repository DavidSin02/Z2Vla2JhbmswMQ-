高考化學一輪提分練：考點6-附加條件的離子共存解答

2020-10-29
| 科技

一、題幹中有附加條件的離子共存判斷

1．(2017·四川雙流中學高三第一次月考)在pH＝1的無色透明溶液中，能大量共存的離子組是( )

A．Al3＋、Ag＋、NO3(－)、SO4(2－)

B．Mg2＋、NH4(＋)、NO3(－)、Cl－

C．Cu2＋、Cl－、NO3(－)、Cl－

D．K＋、Na＋、ClO－、S2－

2．(2017·衡陽八中高三第二次月考)某溶液能與鋅片作用產生H2，則該溶液中可以大量共存的離子組是( )

A．Na＋、NO3(－)、K＋、Fe2＋

B．Fe2＋、Na＋、SO4(2－)、K＋

C．K＋、I－、NO3(－)、H＋　

D．NH4(＋)、NO3(－)、Na＋、HCO3(－)

3．(2017·定州中學高三第一次月考)下列各組離子，在溶液中能大量共存，加入NaOH溶液後加熱既有氣體放出又有沉澱生成的一組是( )

A．Ba2＋、NO3(－)、NH4(＋)、Cl－

B．Ca2＋、HCO3(－)、NH4(＋)、OH－

C．K＋、Ba2＋、Cl－、HSO3(－)

D．Mg2＋、NH4(＋)、SO4(2－)、K＋

4．(2017·棗陽一中高三開學考試)由水電離出來的氫離子濃度為10－13 mol·L－1的溶液中，下列各組離子一定不能大量共存的是( )

A．Na＋、NO3(－)、Al3＋、K＋

B．Na＋、ClO3(－)、ClO－、ClO4(－)

C．K＋、Cr2O7(2－)、I－、Na＋

D．Na＋、S2O3(2－)、SO4(2－)、NH4(＋)

5．(2017·河北武邑中學高三上學期週考)在溶液中加入足量Na2O2後仍能大量共存的離子組是( )

A．NH4(＋)、Ba2＋、Cl－、NO3(－)

B．K＋、AlO2(－)、Cl－、SO4(2－)

C．Ca2＋、Mg2＋、NO3(－)、HCO3(－)

D．Na＋、Cl－、CO3(2－)、SO3(2－)

二、選項中有附加條件的離子共存判斷

6．(2017·凱里一中高三上學期月考)常溫下，下列各組離子一定能在指定溶液中大量共存的是( )

A．使pH試紙變藍色的溶液中：K＋、Al3＋、NO3(－)、SO4(2－)

B.c(H＋)(Kw)＝1×10－12 mol·L－1的溶液中：NH4(＋)、Cu2＋、Cl－、SO4(2－)

C．有較多Fe3＋的溶液中：Na＋、SCN－、K＋、Na＋、CO3(2－)

D．與Al反應能放出H2的溶液中：Fe2＋、K＋、NO3(－)、Cl－

7．(2017·葫蘆島市六校協作體高三期初)常溫下，下列各組離子在指定溶液中一定能大量共存的是( )

A．使甲基橙變紅的溶液中：Na＋、Cl－、CH3 COO－、Fe2＋

B．水電離的OH－濃度為10－12 mol·L－1的溶液中：Mg2＋、NH4(＋)、AlO2(－)、SO3(2－)

C．無色溶液中：Al3＋、NO3(－)、Cl－、SO4(2－)

D．c(K2C2O4)＝0.5 mol·L－1溶液中：Na＋、K＋、MnO4(－)、Cl－

8．(2017·揚州中學高三開學測試)25 ℃時，下列各組離子在指定溶液中一定能大量共存的是( )

A．0.1 mol·L－1 AlCl3溶液中：H＋、Na＋、Cl－、SO4(2－)

B．能使紫色石蕊溶液變藍的溶液：Ag＋、Fe3＋、Br－、SO4(2－)

C．能使澱粉碘化鉀試紙顯藍色的溶液：K＋、SO4(2－)、S2－、SO3(2－)

D．由水電離產生的c(H＋)＝10－12 mol·L－1的溶液中：NH4(＋)、SO4(2－)、HCO3(－)、Cl－

9．(2017·贛州十四校高三上學期期中聯考)在不同溫度下，水溶液中c(H＋)與c(OH－)有如圖所示關係。下列條件關於離子共存說法中正確的是( )


A．a點對應的溶液中大量存在：Fe3＋、Na＋、SCN－、SO4(2－)

B．b點對應的溶液中大量存在：Na＋、Fe3＋、HCO3(－)、I－

C．c點對應的溶液中大量存在：Na＋、Ca2＋、Cl－、CO3(2－)

D．d點對應的溶液中大量存在：Na＋、K＋、SO3(2－)、Cl－

10．(2016·河北高三入學考試)下列各組離子在指定溶液中一定能大量共存的是( )

A．25 ℃時，c(H＋) /c(OH－)＝1×10－12的溶液中：Na＋、SO3(2－)、ClO－、CO3(2－)

B．能與金屬鋁反應放出氫氣的溶液中：K＋、Fe2＋、Cl－、NO3(－)

C．0.1 mol·L －1Ba (OH)2溶液中：Al3＋、NH4(＋)、NO3(－)、HCO3(－)

D．0.1 mol·L－1 AlO2(－)溶液中：Na＋、K＋、SO3(2－)、CO3(2－)
