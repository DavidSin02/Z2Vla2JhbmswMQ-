理解數字通信系統中的IQ信號和正交調製技術

2020-10-29
| 科技

本文的目的：瞭解“ I / Q”信號，如何使用以及為什麼它們在RF系統中具有優勢。

如果沒有有關正交解調方面的文章，則本文將不完整。但是，在探討正交解調之前，我們至少需要簡要地討論一下正交調製。而在討論正交調製之前，我們需要了解I / Q信號。

同相和正交

術語“ I / Q”是“同相（“in-phase”）”和“正交”（“quadrature”）的縮寫。不幸的是，我們已經遇到了術語上的問題。首先，“同相”和“正交”本身沒有意義。相位是相對的，相對於另一個信號或已建立的參考點，某些東西只能是“同相”或“異相”。此外，我們現在將“正交”一詞應用於某種信號以及與該信號相關的調製/解調技術。

無論如何，“同相”和“正交”是指兩個具有相同頻率並且相差90°的正弦波。按照慣例，I信號是餘弦波形，而Q信號是正弦波形。如您所知，正弦波（沒有任何附加相位）相對於餘弦波偏移90°。另一種表達方式是正弦波和餘弦波相互正交。

瞭解I / Q信號的第一件事是它們始終是幅度調製的，而不是頻率或相位調製的。但是，I / Q幅度調製與之前我們在文章中討論的AM技術不同：在I / Q調製器中，調製I / Q正弦波的信號不會移位，因此它們始終為正。換句話說，I / Q調製涉及通過調製可能具有負電壓值的信號來倍增I / Q波形，因此“幅度”調製可能會導致180°相移。在本文的後面，我們將更詳細地探討此問題。

對兩個相位相差90°的正弦波進行幅度調製有什麼好處？為什麼I / Q調製和解調在數字通信系統中應用如此普遍？請繼續閱讀。

I/Q信號的求和

I和Q信號本身並不是很有趣，但是當I和Q波形相加時，就會發生有趣的事情。事實證明，只要改變I和Q信號的幅度，對的，沒錯，僅僅是幅度，然後將它們加在一起，就可以執行任何形式的調製。

如果取等幅的I和Q信號並將其相加，其結果則是一個正弦波，其相位恰好是在I信號的相位和Q信號的相位之間，如下圖所示：

等幅的I和Q信號相加之後的結果

換句話說，如果您將I波形的相位設為0°，將Q波形的相位設為90°，則求和信號的相位將變為45°。 如果要使用這些I和Q信號創建幅度調製的波形，只需對單個I和Q信號進行幅度調製。 顯然，如果信號是通過將兩個幅度都增大或都減小的信號相加而創建的，則合成之後的信號幅度將相應地增大或者減小。 但是，必須確保應用於I信號的幅度調製與應用於Q信號的幅度調製相同，因為如果它們不相同，則將發生相移。 這將為我們引入I / Q信號的下一個特性。

從幅度到相位

以相移鍵控形式出現的相位調製是現代RF系統中的一項重要技術，並且可以通過改變I / Q信號的幅度方便地實現相位調製。 考慮下面的圖示：

改變I / Q信號的幅度方便地實現相位調製

改變I / Q信號的幅度方便地實現相位調製

如您所見，增大其中一個波形相對於另一個波形的幅度會導致求和信號向更高幅度的波形偏移。這具有直觀的意義：例如，如果消除了Q波形，則求和將一直移到I波形的相位，因為（顯然）將I波形加到零將導致求和信號成為與I波形相同的信號。

從上面的討論中可以看出，I / Q信令只能用於將信號移位90°（即，每個方向上為45°）：如果Q信號的幅度減小到零，則求和信號直接到I相位;如果將I信號的幅值減小到零，則求和信號直接到Q相位。那麼，我們如何使用I / Q信號來創建（例如）正交相移鍵控（QPSK），同時讓它覆蓋270°範圍的相位值？我們將在下一部分中對此進行討論。

正交調製

術語“正交調製”是指基於正交的兩個信號之和的調製。換句話說，它是基於I / Q信號的調製。我們將以QPSK為例來說明正交調製的工作原理，在此過程中，我們將瞭解I / Q信號的幅度調製如何產生超過90°的相移。

正交調製

上圖是QPSK調製器的基本框圖。首先，對數字數據流進行處理，以使兩個串行的bits位變為兩個並行的bit位。這兩個bit位將同時發送。換句話說，如本文中所述，QPSK允許一個符號同時傳輸兩個bits位。本地振盪器生成載波正弦波。本地振盪器信號本身成為I載波，並且施加90°相移以創建Q載​​波。 I和Q載波與I和Q數據流相乘，並且將這些相乘所得的兩個信號相加，以生成QPSK調製波形。

I和Q數據流正在對I和Q載波進行幅度調製，並且如上所述，這些單獨的幅度調製可用於在最終信號中產生相位調製。如果I和Q數據流是從地電平延伸到某個正電壓的典型數字信號，則我們將對I和Q載波應用開關鍵控調製，並且我們在任一方向上的相移都將限制為45°。但是，如果I和Q數據流是雙極性信號（即，如果它們在負電壓和正電壓之間擺動），則只要輸入數據為邏輯低（由於負輸入電壓），我們的“幅度調製”實際上會使反相乘以載波會導致載波翻轉。這意味著我們將具有四個I / Q狀態：

I正常，Q正常

I正常，Q翻轉

I翻轉，Q正常

I翻轉，Q翻轉

在每種情況下，求和後會產生什麼信號呢？ （請注意，在下面的圖中，選擇了波形的頻率，使得x軸上的秒數與以度為單位的相移相同。）

I翻轉，Q正常

如您所見，上面圖中所示的這四種情況下的求和恰好產生了我們想要的QPSK信號：45°，135°，225°和315°的相移。

總結

  • I / Q信號是指使用兩個具有相同頻率和90°相對相移的正弦波。
  • 幅度，相位和頻率調製可以通過對幅度調製的I / Q信號求和來實現。
  • 正交調製是指涉及I / Q信號的調製。
  • 正交相移鍵控可以通過將根據輸入數字數據分別乘以+1或–1的I和Q載波相加來實現。
