AirCom壓差調節器 R03-J

描述：

具有正偏壓的先導式體積助力器，設計用於提供等於信號壓力的出口壓力，以及可調節的預設彈簧常數。具有非常高的正向和反向流動特性，並具有出色的靈敏度。典型的應用是平衡或調節儲罐中的壓力。


更多信息請聯繫漢達森靳叢偉 15810543160

介質：無油，5 µm過濾的壓縮空氣或非腐蝕性氣體

供給壓力：最大16巴

比例：1：1

先導壓力：

最高10 bar，在手動預壓設置下相應降低

先導油口G1 / 8

準確性：

供氣壓力從2 bar變為7 bar

= <6 mbar壓力偏差

流量從0 l / min更改為20 l / min

= <20 mbar壓力偏差

響應靈敏度= <2 mbar

耗氣量：

在P1 = 5 bar時為1.5 l / min

在P1 = 7 bar時為2 l / min，

P1 = 10 bar時為4 l / min，<體積流量的1％

緩解功能：緩解

洩壓能力：6 bar出口處700 l / min，高於設定點0.35 bar超壓

儀表端口：閥體兩側為G1 / 4，隨附一個螺塞

Mounting_position：任何

溫度範圍：

0°C至60°C / 32°F至140°F，適用於溫度低至-30°C / -22°F的經過適當調節的壓縮空氣

身體：鋅壓鑄

彈性體：NBR / Buna-N
