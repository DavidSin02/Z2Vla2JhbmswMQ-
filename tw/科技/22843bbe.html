VBA｜使用和定製Excel內置對話框

2020-11-14
| 科技

在Excel中，對話框可以實現應用程序與用戶的交互。在VBA中，Excel提供了調用這些內置對話框的接口，使開發者可充分利用內置對話框實現更多的操作。

1 用FindFile方法打開文件

Sub 打開工作簿()

Dim wb As Workbook

Set wb = ActiveWorkbook

If Application.FindFile Then '顯示“打開”對話框

MsgBox "已打開工作簿文件！"

wb.Activate

Else

MsgBox "打開工作簿失敗！"

End If

End Sub

'上面代碼運行後，顯示“打開”對話框，選擇一個工作簿，單擊“打開”後即可打開該工作簿

2 使用對話框獲取單個文件名

2.1 使用GetOpenFilename方法顯示“打開”對話框，獲取選擇的文件名

Sub 獲取單個文件名()

Dim strFilt As String

Dim strTitle As String

Dim strFname As Variant

strFilt = "文本文件(*.txt),*.txt," & _

"Excel文件(*.xls;*.xlsx;*.xlsm),*.xls;*.xlsx;*.xlsm," & _

"所有文件(*.*),*.*"

strTitle = "打開Excel文件"

strFname = Application.GetOpenFilename _

(fileFilter:=strFilt, _

Title:=strTitle)

If strFname = False Then

MsgBox "沒選擇文件！"

Else

MsgBox "選擇的文件是：" & strFname

End If

End Sub

2.2 使用GetSaveAsFilename方法打開“另存為”對話框獲取文件名

Sub 獲取保存文件名()

Dim fn As String

fn = Application.GetSaveAsFilename( _

fileFilter:="Text Files (*.txt), *.txt")

If fn <> "False" Then

'在此處編寫保存文件的代碼

MsgBox "保存文件為：" & fn

End If

End Sub

3 使用對話框獲取多個文件名

Sub 獲取多個文件名()

Dim strFilt As String

Dim strTitle As String

Dim strFname As Variant

Dim i As Integer

Dim strMsg As String

strFilt = "文本文件(*.txt),*.txt," & _

"Excel文件(*.xls;*.xlsx;*.xlsm),*.xls;*.xlsx;*.xlsm," & _

"所有文件(*.*),*.*"

strTitle = "打開Excel文件"

strFname = Application.GetOpenFilename _

(fileFilter:=strFilt, _

Title:=strTitle, _

MultiSelect:=True) '允許選擇多個文件

If Not IsArray(strFname) Then '返回值不是數組

MsgBox "沒選擇文件！"

Else

For i = LBound(strFname) To UBound(strFname)

strMsg = strMsg & strFname(i) & vbCrLf

Next

MsgBox "選擇的文件是：" & vbNewLine & strMsg

End If

End Sub

4 使用Dialogs集合可以調用所有Excel內置對話框

Excel2007中有1101個內置對話框，如下面代碼可以調用“對齊"標籤頁：

Sub 對齊對話框()

Application.Dialogs(xlDialogAlignment).Show

End Sub

5 設置內置對話框的初始值

在VBA中，可以通過更改對話框的選項參數，定製用戶需要的對話框：

Sub 設置字體1()

Application.Dialogs(xlDialogFontProperties).Show _

"黑體", "粗體", 20, True, True, False, , _

True, 5, 4, True, 3

End Sub

Sub 設置字體2()

Application.Dialogs(xlDialogFontProperties).Show _

arg1:="黑體", arg2:="粗體", arg3:=20, _

arg4:=True, arg5:=True, arg7:=True, arg8:=True, _

arg9:=5, arg10:=4, arg11:=True, arg12:=3

End Sub
