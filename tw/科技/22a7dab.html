發財不一定能發財，大生意又不會做，就靠這點經濟學常識維持生活

2020-10-29
| 科技

今天是8月8日，對於許多中國人而言，都是一個財運亨通的日子，你既可以說這是迷信，也不妨就看做一種美好的願景，就像那句有意無意寒暄開場白，

“最近在哪裡發財？”

既有恰到好處的關心，又不會讓人覺得你自找沒趣。

俗話說，天上不會掉餡餅。發財這等美事，也不會從天上掉下來。就算中大獎，也得先去買彩票。

可是，中大獎可遇不可求，工資又花得飛快。看一眼賬單，驚覺“我的錢雖然不是大風颳來的，但很像是大風颳走的”。難道就真的沒有辦法了嗎？

行為經濟學告訴你，當然有辦法，我們並非隨時保持理性，花錢和花錢是不一樣的。

No.1 你瞭解你的“心理賬戶”嗎？

先來做一個小測試吧！

你特別期待的一個電影上映了，準備去電影院一飽眼福，可是就在首映那天，出了一個不大不小的意外：

意外一：你剛到電影院門口準備買票時，發現你之前放在上衣口袋裡的 70 元不見了，你還會繼續花 70 元買票看電影嗎？

意外二：你剛到電影院門口，發現自己幾天前花70 元買的電影票不見了，你還會繼續花 70 元買票看電影嗎？

這兩個意外其實沒有什麼區別，你都損失了70塊。但是，大部分人卻呈現出了不一樣的測試結果：在意外一的情景下，大部分人選擇買票看電影；而在意外二的情景下，大部分人選擇打道回府，等待網絡放映。

為什麼同樣的損失，人們會做出不一樣的選擇？這可以用行為經濟學中“心理賬戶”的理論進行解釋。

// 心理賬戶

///

NOCITCE

在日常經濟生活中，人們並不是將所有的收入和支出放在一起進行統籌管理，而是會劃分一些不同的心理賬戶，將不同的收入和支出分門別類地放入不同的心理賬戶中進行管理。

心理賬戶有很多重要的機制。

比如，在價值認識上，人們會給不同的心理賬戶貼上不同的標籤；不同的標籤下，等量的錢在不同的心理賬戶裡的價值是不同的。

在收支管理上，心理賬戶傾向於“抓大放小”，即人們在收入方面，會存住大筆的收入，花掉小筆的收入；在消費方面，會控制大額消費，卻不在意小額消費。

心理賬戶是一種對個人財務進行管理的心理捷徑，有時會使我們趨於理性，比如為了一些長期目標攢錢等有利的經濟行為。

但有時，心理賬戶也會造成認識偏差，讓我們不能理性地管理自己的財富，比如花唄裡的錢就比銀行卡里的錢花起來不心疼，但是誰都知道下個月的還款日遲早會來。

那麼我們要如何克服心理賬戶帶來的不良影響，以免在年終看到賬單時一臉懵逼呢？

第一步就是要有“金錢無貴賤”的意識，什麼錢不是錢吶？就算是大風颳來的，只要是跑進我們戶頭裡的錢，都應該一視同仁。

此外，平時要多注意“小錢”。比如在大筆消費的時候，也應該時時注意那些附加的消費。並且養成存錢的好習慣，勿因錢“小”而不存。畢竟，“大錢”是難有“大錢”的，也就只能靠擺弄這些“小錢”實現財務自由了。

No.2 我想給房子開個窗，怎麼辦？

還是先來一個小測試。

A和B素不相識， A臨時需要向B借100元。直接說借100元不僅突兀，而且一般人可能嫌多，A擬採用以下兩種策略：

策略一：先說借50元，等對方同意之後託故再借50元；

策略二：先說借200元，對方估計不同意，假裝退一步說借100元。

結果顯示，不論在哪一種策略下，借錢成功的機會都會比直接借 100 元高。

策略一是得寸進尺，也就是說從低要求開始，逐漸提出更高的要求。心理學家查爾迪尼分析認為，對人們提出一個很簡單的要求時，人們很難拒絕。

當人們接受了簡單的要求後，再提出較高的要求，為了保持認識上的統一和給外界留下前後一致的印象，人們心理上就傾向於接受較高要求。

這就叫做“登門檻效應”，先誘導對方接受容易達成的小目標，然後順勢推波助瀾一步步接近大目標。

商家發現這種效應並加以利用，就能讓你花掉一些預算外的錢。比如在商場買了一雙靴子，這時候不應該去旁邊的女裝店為它搭一條靴褲嗎？咦，這件大衣好像也很配……

策略二是“先過分後讓步”。也就是當我們打算給房子開扇窗時，可以先要求把屋頂掀了，等到別人大驚失色的時候，做出開窗戶的讓步，打破心理預期的對方也許就“行吧行吧”地同意了。

一些商家先抬價再打折的做法，也就是利用了這種心理。畢竟，有時候看到打折真的走不動路，我越買老闆就越虧，衝鴨！

steam又打折？不買不是人！

消費者和廠商，也就是買者和賣者，永遠是市場裡博弈的對手。儘管買者都會自律和自我約束，但商家有的是辦法給買者一些理由去違反自律規則，讓你不自覺但是很情願地“上鉤”。這就是消費者的自控與商家的反自控。

如何在這些直擊人性弱點的營銷手段中保住自己的錢包呢？

這就需要我們制定嚴格的消費預算，提高自控力，尤其是對非必需品或者奢侈品的消費。

No.3 什麼在影響你的判斷？

啥也別說了，開始測試。

請按照人口數從多到少，給這四個國家排序：匈牙利、日本、以色列、尼日利亞。

點擊空白處查看答案

正確答案是：尼日利亞、日本、匈牙利、以色列

你排對了嗎？很多人都沒排對，他們的測試結果顯示，對其中一個國家瞭解得越多或聽到得越多，對這個國家人口的估計就越多。這種心理機制在行為經濟學中被稱作“可用性啟發法”。

// 可用性啟發法

///

NOCITCE

通常，人們依據記憶中易於使用的信息做出各種判斷。也就是說，信息越容易被記起，就更傾向於被作為判斷的依據。

顯然，通過可用性啟發法做出的判斷，往往是不精確的。所以，在做重要的判斷之前，有必要考察自己的信息來源，綜合儘可能多的信息——包括容易回憶起來的以及那些可能被忽視的，以求做出儘量準確的判斷。

那麼商家會如何利用這種心理機制呢？

這張動圖自帶BGM。

十幾年前，一句廣告詞強勢打破了廣告市場的平靜。

“今年過節不收禮，收禮只收腦白金”，很多人覺得，腦白金的這句廣告詞一無是處；業內人士更是直斥其毫無創意、“土得令人噁心”。十佳廣告的名單年年翻新，但最爛廣告的稱號它一直蟬聯，與恆源祥的羊羊羊不分伯仲。

志在成為青年人的腦白金的BOSS直聘

但是，銷量數字又顯示，人們嘴上說著噁心噁心，身體卻很誠實。這廣告詞怎麼就讓人一邊噁心一邊乖乖掏錢呢？難道“噁心也是一種生產力“？

當然不是，原因就在於這些廣告很好地利用了行為經濟學中可用性啟發法的原理。人們在選購相關商品時，最容易想到的就是它。

除了可用性啟發法，在日常進行判斷的時候，人們還會利用代表性啟發法、錨定啟發法和熟悉啟發法。

// 代表性啟發法

///

NOCITCE

人們在不確定性的情形下，會抓住問題的某個特徵直接推斷結果，而不考慮這種特徵出現的真實概率以及與特徵有關的其他原因。

比如堅信這世界上哪有隻跌不漲的股票，覺得“跌跌跌跌漲”肯定比“跌跌跌跌跌”的概率要大，結果很可能就被套牢了。

// 錨定啟發法

///

NOCITCE

當人們對某個不太熟悉的事物進行判斷時，一個通常採用的方法是首先確認一個初始值——該初始值與所要判斷的對象相關與否都可以，通過對這個初始值進行不充分的調整，從而得出自己的答案。另一方面，由於人們所做的調整是不充分的，判斷的結果會過分拘泥於這個初始值，就像船拋錨之後不會隨波逐流一樣。

比如買烤冷麵的時候，老闆問你加烤腸還是加雞腿排，你就很可能在這兩個選項中選一，而不是選擇都不加。

// 熟悉性啟發法

///

NOCITCE

行為經濟學家認為人們有時會更加喜歡自己所熟悉的事物，而相對不喜歡自己不熟悉的事物。也就是說，人們往往以自己熟悉與否來作為自己判斷的依據。

這也被叫做“曝光效應”，也就是“越熟悉、越喜歡”。熟悉性啟發法和可用性啟發法有點相似，不過熟悉性啟發法更偏向於無意識的選擇。

這幾種啟發法是人類在長期的自然進化和社會實踐中慢慢形成的，因此，在大多數情況下都可以視作思維的捷徑，並能夠藉以做出有 效而準確的判斷。但是，這些思維捷徑都有著不同程度的缺陷，也有可能導致錯誤的發生。

所以，在做出消費判斷的時候，應該儘可能該冷靜而理性地檢查自己的整個思維過程，同時要參考儘可能多的信息，以保證自己的判斷能夠儘量做到準確。

很多平時看起來理所當然的選擇或行為，仔細思考就會發現其中的不理性和荒謬之處。這些心理機制在商家的精心策劃下，就會慢慢掏空我們的錢包。

今天是8月8號。8在中國人心中是一個有美好寄託和祝福的數字。那就在這裡祝大家能夠充分利用行為經濟學的知識，做錢包的掌控者！
