《英雄聯盟》賽娜技能疑曝光 光暗雙修，大招反射技能

2020-10-29
| 科技

《英雄聯盟（League of Legends）》新英雄賽娜預計會在9.23版本上線，日前有Reddit論壇用戶曝光了德文版的新英雄賽娜技能，光暗雙修，加奶加盾、大招反射技能。雖然不知道目前這份爆料的可靠性，但是大家可以先看看做一個參考。

被動技能1

*賽娜的槍每次普攻都會用掉一發彈藥。賽娜可以在任意時間選擇她的下一發子彈是“光彈”還是“暗彈”。在用盡所有的子彈之後，賽娜可以用幾秒的時間來重新填彈。隨著賽娜等級和攻擊速度的提升，填彈的時間會有所下降。

被動技能2

*如果在賽娜填彈期間附近有友方的盧錫安，賽娜會獲得額外的移動速度（受AP加成）。

Q技能 - 透體射擊

被動-光彈

*賽娜可以以友軍作為目標。每次對友軍普攻可以祝福友軍，使其獲得150 / 175 / 200 / 255 / 250（受AD加成）的治療同時提高其移動速度（受AP加成）。如果你在短時間內多次對同一目標進行治療，目標不會受到額外的治療但是可以享受到加速效果。

被動-暗彈

*賽娜的普攻會使敵人減速並且疊加一層“復仇”效果。

W技能 - 復仇

被動

*賽娜在脫離戰鬥後獲得20 / 35 / 50 / 65 / 80點額外移動速度。

主動-光彈

*在短時間內，賽娜的普攻射程翻倍，但是她的傷害也會降低64% / 48% / 32% / 16% / 0%

主動-暗彈

*每層復仇效果對敵人造成65 / 120 / 155 / 195 / 270點魔法傷害（受AP加成）。

*如果目標之前受到了盧錫安W的傷害，賽娜獲得額外的移動速度（受AP加成）。

E技能 - 跟上節奏

*0.25秒的延遲之後，賽娜向所選方向衝刺並且獲得一個可以吸收100 / 125 / 150 / 200 / 250魔法傷害的護盾。

R技能 - 在劫難逃

注：這個技能和你選擇什麼彈藥沒有關係

*賽娜朝友軍發射一個光束，在其身後產生一面鏡子。鏡子可以反射接下來第一個命中的敵方飛行技能。（技能表示哪來的就會飛回哪去）

*在鏡子碎裂最後，在友軍身下產生一塊暗影區域，減速其中的敵人和野怪80% / 90% / 99%，持續4秒。
