Java正則表達式詳解

2020-11-14
| 科技

親愛的小夥伴們，本期我們講解java中的正則表達式和它相關的一些類與方法。

概述

正則表達式非常強大，能夠解決我們開發時用於匹配一些特定的要求。並且java中還提供了相當強大的方法，擴展其功能性。下面我們就來介紹一下：

1.定義一個關於用戶名的正則表達式

String regex = "[\\u4E00-\\u9FEF[a-z][A-Z]{2}]\\w{6,20}";

2.定義一個關於密碼的正則表達式

String password = "[\\w\\.]{6,16}";

3. 定義一個關於QQ郵箱的正則表達式

String email = "[\\d@qq\\.com]{12,16}";

4.兩個用來處理正則表達式的工具類

compile是靜態方法,用於編譯規則返回Pattern對象

matcher用於匹配字符串

Pattern p = Pattern.compile(regex);

Matcher m = p.matcher(str);

5.find()：判斷字符串是否匹配

6.group()：將匹配到的字符串分組

7.start()和end()：分別表示第幾輪匹配到的字符串的下標，end()下標自動加1

8.appendReplacement()：用特定的字符串替代匹配到的字符串，並放入其第一個參數 StringBuffer中

9.appendTail()：將最後匹配到的字符串的剩餘串放入StringBuffer中
