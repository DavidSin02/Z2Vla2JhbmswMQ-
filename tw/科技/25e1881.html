C++程序設計之概述

2020-10-29
| 科技

C語言是一種面向過程的編程語言，不能滿足面向對象開發軟件的需要。為支持面向對象的程序設計，1980年由貝爾實驗室的Bjarne Stroustrup創建了C++程序設計語言，C++的重要目標就是面向對象的程序設計，因此在C++中引入了類的機制。1983年正式命名為C++（C Plus Plus）。以後經過不斷完善，形成了目前的C++。

當前運用得較為廣泛的C++有Microsoft公司的Visual C++（簡稱V C++）和Borland公司的Borland C++ （簡稱BC++）。本書以Microsoft Visual C++ 6.0集成環境為例介紹C++語言。

1.2 C++的特點C++語言的主要特點表現在兩個方面，一是全面兼容C語言，二是支持面向對象的程序設計方法。

（1）全面兼容C語言

C++是一個更好的C，它保持了C語言的優點，大多數的C程序代碼略作修改或不作修改就可在C++的集成環境下調試和運行。這對於繼承和開發當前已在廣泛使用的軟件是非常重要的，可以節省大量的人力和物力。

（2）支持面向對象

C++是一種面向對象的程序設計語言。它使得程序的各個模塊的獨立性更強，程序的可讀性和可移植性更強，程序代碼的結構更加合理，程序的擴充性更強。這對於設計、編制和調試一些大型的軟件尤為重要。

1.3 簡單C++程序C++集成環境支持C++程序與C程序的編譯和調試，C++集成環境約定：

（1）源程序文件擴展名為.c時，則為C程序；

（2）源程序文件擴展名為.cpp時，則為C++程序。

本書中，所有例題程序的文件擴展名均為.cpp。

【例1.1】 文本的原樣輸出。文件名為example1_1.cpp。

//文本原樣輸出程序

#include <iostream.h>

void main(void)

{ cout<<"Welcome to C++!/n";

}

該程序經編譯和連接後，運行可執行程序時，在顯示器上顯示：

Welcome to C++!

【例1.2】 求兩個整數的和。

/*求兩個整數的和程序*/

#include <iostream.h>

void main(void)

{ int x,y,sum; //說明變量x,y,sun為整型數

cout<<"Input first integer:"; //顯示提示信息

cin>>x; //從鍵盤上輸入變量x的值

cout<<"Input second integer:"; //顯示提示信息

cin>>y; //從鍵盤上輸入變量y的值

sum=x+y; //求和

cout<<"Sum is "<<sum<<endl; //輸出結果

}

該程序經編譯和連接後，運行可執行程序時，在顯示器上顯示：

Input first integer：5

Input second integer：4

Sum is 9

該程序中的語句：int x,y,sum; 用來說明變量x、y、sum為int（整型）變量。程序中的語句：sum=x+y; 是一個賦值語句，表示將x和y的值相加，其結果送給變量sum。在“/*”和“*/”之間的部分也表示註釋。“endl”是換行符，相當於“/n”。

【例1.3】 輸入兩個整數a和b，輸出其中較大的一個數。

#include <iostream.h>

void main(void)

{ int max(int x,int y);

int a,b,m;

cout<<"Input a,b:";

cin>>a>>b;

m=max(a,b);

cout<<"max="<<m<<endl;

}

int max(int x,int y)

{ int z;

if (x>y) z=x;

else z=y;

return(z);

}

該程序經編譯和連接後，運行可執行程序時，在顯示器上顯示：

Input a,b：3 5

max=5

該程序由兩個函數組成：主函數main()和被調用函數max()。函數max()的作用是找出x和y中的較大者，並通過return語句返回給主函數。主函數用來輸入兩個變量a和b的值，調用max函數找出其中的較大者，並輸出結果。

通過以上例題，可以看出C++程序的結構有以下特點：

（1）C++程序由一個或多個函數組成（即函數是C++程序的基本構成單位）

其中至少有一個主函數main()，程序從主函數開始執行，由主函數來調用其他函數。被調用的其他函數可以是系統提供的庫函數，也可以是用戶自定義的函數。例如，例1.3的C++程序就是由主函數main()和用戶自定義函數max()組成的。

（2）C++函數組成

① 函數說明部分：函數名、函數類型、函數參數（形式參數）及其類型

函數類型為函數返回值的類型。

② 函數體{}：變量定義和執行語句兩部分。

在某些情況下可以沒有變量定義，甚至可以既無變量定義又無執行語句（即空函數）。例如：void dump(void )

{ }

（3）語句和數據說明以分號結束

分號是C++語句的必要組成部分。例如：int z; 又例如：return(z);

（4）程序書寫規則

① 同一層語句同列書寫。同一層次的開花括號必須與對應的閉括號在同一列上。

② 內層語句縮進兩個字符的位置。

③ 函數定義第一列書寫。

（5）C++輸入/輸出通過流cin和cout來實現的

例如： cin>>a>>b;

用來輸入變量a和b的值。又例如：

cout<<"max="<<m<<endl;

用來輸出變量m的值。

（6）嚴格區分字母的大小寫

例如：int a,A; 表示定義兩個不同的變量a、A。

（7）註釋

① /* 註釋內容 */

② // 註釋內容

它表示從此開始到本行結束為註釋內容。例如：//說明變量x,y,sun為整型數

（8）編譯預處理命令

以“#”開頭的行稱為編譯預處理命令。例如：#include <iostream.h>

表示本程序包含有頭文件iostream.h。

以上所述的有關函數、輸入/輸出流等概念將在以後的章節中詳細介紹。

本章小結C++程序的基本結構1．C++程序通常由一個或多個函數組成，函數是構成C++程序的基本單位。C++程序中至少要包含一個主函數main()，一個C++程序總是從主函數開始執行的。

2．C++的函數由函數的說明部分和函數體兩部分組成。

（1）C++中每個語句和數據說明必須以分號結束，分號是C++語句的必要組成部分。

（2）C++語言沒有專門的輸入/輸出語句，輸入/輸出操作是通過輸入/輸出流cin和cout來實現的。

（3）程序的書寫必須規範，以便於程序的閱讀、修改和相互交流。

（4）在C++中，嚴格區分字母的大小寫。

（5）在C++程序的任何位置處都可以插入註釋信息。
