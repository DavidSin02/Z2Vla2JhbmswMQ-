C++中虛函數的缺省參數值

2020-11-14
| 科技


知識點：C++中虛函數是動態綁定的，但函數的缺省參數確實在編譯時就靜態綁定的。

若在虛函數中，指定了形參的缺省參數值，這意味著最終執行的函數是一個定義在派生類，但使用了基類的缺省參數值的虛函數。

比如下面這段代碼：

#include <iostream>using namespace std;class base {public:    virtual void Display1() {        cout << "Base: Display1" << endl;    }    virtual void Display2(const int x = 1) {        cout << "Base: Display2: " << x << endl;    }};class derived : public base {public:    virtual void Display1() {        cout << "Derived: Display1" << endl;    }    virtual void Display2(const int x = 2) {        cout << "Derived: Display2: " << x << endl;    }};int main(){    base *pBase = new derived;    pBase->Display1(); // 實現多態    pBase->Display2(); // 未實現多態      delete pBase;    pBase = nullptr;      return 0;}

運行結果如下：

可以看出，Display1()調用了派生類的函數，實現了多態；而Display2()雖然調用了派生類的函數，但卻使用了基類的缺省參數值，未能完全實現多態。
