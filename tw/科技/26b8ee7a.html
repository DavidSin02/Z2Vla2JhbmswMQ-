一、 基礎處理說明

由吊裝方案中計算可知要求地面的承載力達到90KN/M2，由於現場現有的地面無法滿足承載力的要求，需對現有的支撐點進行處理，處理方法是將支撐點處地面挖到原土層，然後填碎石來提高地基的承載力。

三、計算書

（一）型鋼支撐架子寬基礎計算

1、墊層地基承載力驗算

經處理後的地基，當考慮基礎寬度和深度對地基承載力特徵值進行修正時，一般寬度不作修正，即基礎

寬度的地基承載力修正係數取零，基礎深度的地基承載力修正係數取1.0。

基礎埋深，d=0m

fspk 為墊層的地基承載力特徵值，fspk=200.00kPa

fa = 200.00+18.00×(0.00-0.50)=191kPa

荷載效應標準組合時軸心荷載作用下

pk = (Fk+Gk)/A =(360.00+0)/4.00 = 90.00kPa pk<=fa,滿足要求

荷載效應標準組合時偏心荷載作用下

pkmin = 90.00kPa pkmin> 0,滿足要求

pkmax = 90.00kPa Pkmax<=1.2fa,滿足要求

2、墊層底面處地基承載力驗算基礎底面的附加壓力

基礎底面平均壓力為：pk= 90.00kPa

基礎底面自重壓力為： 0kPa

基礎底面的附加壓力為：p0= 90.00 - 0 = 90kPa

(1)計算墊層底面處的附加壓力

Pz =46.6kPa

(2)計算墊層底面處的自重壓力

pcz = 18.00×0.80 = 14.40 kPa

(3)墊層底面處經深度修正後複合地基承載力特徵值faz為

墊層底面處埋深，d=0.80m

faz = 60.00+18.00×(0.80-0.50)=65.40kPa

pz+pcz = 46.6 + 14.40 =61.00 kPa <= faz = 65.40 kPa

墊層底面處承載力滿足要求!

3、墊層尺寸驗算

（1）按照構造要求驗算墊層頂面寬度

基礎寬度B=2.00m

頂面寬度B1=2.6m

(B1-B)/2>=300mm,滿足構造要求：墊層頂面寬度超出基礎底邊寬度不小於300mm。

（2）按照構造要求驗算墊層頂面長度

基礎長度L=2.00m

頂面長度L1=2.6m

(L1-L)/2>=300mm,滿足構造要求：墊層頂面長度超出基礎底邊長度不小於300mm。

（3）按應力擴散角驗算墊層底面寬度

底面寬度B2=2.80m

基底壓應力傳遞到墊層底面處的應力分佈寬度B+2ztanθ =2.78m

墊層底面寬度B2>=B + 2ztanθ,墊層底面滿足應力擴散要求！

（4）按應力擴散角驗算墊層底面長度

底面長度L2=2.80m

基底壓應力傳遞到墊層底面處的應力分佈長度L+2ztanθ =2.78m

墊層底面長度L2>=L + 2ztanθ,墊層底面滿足應力擴散要求！
