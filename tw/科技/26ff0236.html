C語言重點——指針篇（一篇讓你完全搞懂指針）

2020-11-14
| 科技

一. 前言

C語言是比較偏底層的語言，為什麼它比較偏底層，就是因為他的很多操作都是直接針對內存操作的。

這篇我們就來講解C語言的一大特點，也是難點，指針和指針操作。

這篇文章我會先從基本類型的存儲過程和原理講起，然後再講解指針int *p，再舉一反三，搞懂int **p和int ***p,學會指針。

搞懂int *p，int **p和int ***p,完全學會指針！！！！

二. 理解一個變量的存儲過程和原理（必須清楚掌握）

2.1 直接'='賦值

int a = 5;printf("a = %d",a);

結果： a = 5

這一句話完成了兩個操作，我們先了解c語言在計算機內部幹了什麼?

兩個操作：

（1）int a;

在棧中定義了一個變量a，並且在內存中開闢了一個int類型大小的空間， 即4個字節 ，然後讓a指向這篇空間，也就是這篇空間，計算機分配給了a， a以後就有了一片屬於自己的空間；

(2) a = 5;

在a的自己的那片空間，裡面存放數值5 ，把5轉換成二進制，存到a的4個字節的空間 。

2.2 利用輸入流，手動賦值

scanf("%d",&a);

我們還有過輸入賦值操作，剛好可以證明上述觀點：

用戶輸入了一個int類型的數值，比如輸入5，然後&a，先找到a的那片地址空間，最後把5轉成二進制，存入a的那片地址空間，即完成了對a的賦值，也就是在a的那片4字節的空間填入了二進制的5；

2.3 總結

從上述講述我們可以瞭解，一個變量的存儲，先從內存開闢一個類型大小的空間（int類型4個字節大小），在讓變量指向這篇空間，即就是這片空間屬於這個變量，再在這片空間中存儲你要存儲的數值。

三. 指針類型（int *）的存儲過程和原理

3.1 指針類型的賦值規範

(1) 第一種先定義後賦值

int *p;p = &a	  //這種方式正確printf("p = %d\n",p);

結果：p = 6618636

變量p存放的a的地址

重點：

先了解，指針類型，int *p,雖然是*p在一起寫著，但是變量名叫p，類型為int *，也就是整型的指針類型，當你理清變量名和類型之後，你對指針的理解程度已經懂了大半了 。

（2）第二種定義賦值一步完成

int *r = &aprintf("r = %d\n",r);

結果：r = 6618636存放的是a的地址

還有一種常用的錯誤賦值方法：

// 	p = a;    //這種賦值方式錯誤

錯誤的操作,不能把一個具體的數字賦給指針（類型不匹配），

一個指針類型，一個int類型


3.2 指針存儲過程和原理

前面列舉了兩種常用的指針的賦值，下來具體講解計算機都幹了什麼？

  • 可以把存放一個int類型變量的地址賦給一個int *指針類型的變量
  • '='左邊是一個int *指針類型的變量 ，可以存放放置著int類型數值的地址
  • '='右邊是&a，a是int類型的變量數值5，&是取地址符，&a就是拿到int類型a的數值的地址

總的來說，就是把a的那片空間，給了p一個鑰匙，讓p也可以對a的那片空間操作，這個已經屬於指針操作了，後面我們會講到。

由上述可以證明，c語言的賦值，必須是類型對應

總結：int *p; 變量名叫p，類型為int *,可存放一個int數據的地址 。

注意：這塊的可存放一個int數據的地址，不是存放一個地址，是int類型

例如：

 int a = 5; int *p; p = &a
  • 這裡a是一個int類型的變量，存放的int類型的數值5

&a 取到了存放int類型a的地址

p = &a 把int類型a的地址賦給了int *類型的p

即就是int *類型的變量可存放一個int數據的地址

四. 指針類型（int **）的存儲過程和原理

int **q;q = &pprintf("q = %d\n",q);

結果：q = 6618624存放的p的地址

int *p明白了，那麼int **q呢？

首先： 先對數據類型和變量劃分開

int **q; 變量名為q，數據類型為int **

int *中存放的是int類型數據的地址

int **中存放的是int類型數據的地址的地址

上述我們明白了，一個*是指一個int數值的地址，

p中存放的是個int數值的地址，p = 6618636，為a的地址。

那麼我們可以推到，兩個*q就是存放p的地址。

p存放a的地址，p本身也是一個變量，他的值為a的地址 ，

而內存也給他自己開闢了一片空間，讓他存放而他的數值

q存放p的地址，q也是一個變量，


他們的指向關係如下 ：

a<----p<----q

五. 指針類型（int ***）的存儲過程和原理

int ***m;m = &qprintf("m = %d\n",m);

結果：m = 6618616存放的q的地址

既然，int *和int **都懂了，那麼int ***就迎刃而解了

同理，int*** 存放的是int **類型數據的地址

六. 指針操作（*操作）

這塊我們這說屬於指針自己的操作

        printf("p = %d\n",p);	printf("*p = %d\n",*p);	printf("q = %d\n",q);	printf("*q = %d\n",*q);	printf("**q = %d\n",**q);	printf("m = %d\n",m);	printf("*m = %d\n",*m);	printf("**m = %d\n",**m);	printf("***m = %d\n",***m);

結果：

p = 6618636*p = 5q = 6618624*q = 6618636**q = 5m = 6618616*m = 6618624**m = 6618636***m = 5

p、q和m都是上述例子中的變量

首先除了定義指針變量的時候，變量前面有*為定義類型，其他時候均為指針的取值操作，注意是取值，不是取址，拿的是指針變量中存放的值。

6.1 舉個現實中栗子

舉個現實中的例子，你比如說去銀行開保險櫃，其中*p操作比如開保險櫃這個操作，你得先拿著你的櫃子號在銀行找到保險櫃，然後拿著鑰匙再打開保險櫃取出裡面的錢；就像是p中存放的是一個地址，你先拿著p中存放的地址，在內存中找到那塊空間，然後再*p操作，取出那塊空間中存放的值。

所以 * 操作就是取值操作，即取出指針變量存放的地址中所存放的數據。


6.2 *操作怎麼去分析

上面幾個例子都屬於指針的取值操作，也就是也就是拿著指針變量中存的地址號去內存中找裡面存的東西。

所以看這種連著好幾個*後面跟個變量的表達式，需要從右往左依次拋開

即：***m ； 就是* ( * ( * m )))，看的時候需要從最裡層一層一層拋開。


  • *p ；
  • 先看成*（p）,再從裡向外看，首先他有一個變量p，所有直接可以先從內存中拿到p存放的數據6618636(p中的數據)，再找到內存中6618636那片內存，最後取出6618636中的存放數據5(具體數據)


  • **q = 5 ；
  • 先看成 *( * (q)),再從裡向外看
  • 先從內存中拿到q存放的數據6618624(q中的數據)，再從內存中找到6618624那片內存，取出存放的數據6618636(*q中存放的數據)，完成了 * (q)操作，再從內存中找到6618636那片內存，取出存放的數據5 ( *( *(q))中存放的數據 )，完成了 *(* (q)）操作，再中的存放數據5(具體數據) （幾個*查找幾層）


  • ***m = 5
  • 先看成 *(*( * (m))),再從裡向外看
  • 先從內存中拿到m存放的數據6618616(m中的數據)，再從內存中找到6618616那片內存，取出存放的數據6618624(*q中存放的數據)，完成了 * (m)操作，再從內存中找到6618624那片內存，取出存放的數據6618636 ( *( *(m)) 中存放的數據)，完成了 *(* (m)）操作，再從內存中找到6618636 那片內存，取出存放的數據5( *(*( *(m))) 中存放的數據)，完成了 *(*( *(m)))操作， （幾個*查找幾層)
