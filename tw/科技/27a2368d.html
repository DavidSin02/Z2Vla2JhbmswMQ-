多條件模糊查詢，除了VLOOKUP，你還有新的選擇！

2020-11-14
| 科技

本例是一個跨工作表的多條件模糊的查詢案例。

源數據如下：

目標工作簿及內容：

問題描述：

1：源數據，在名為“進價”的工作表內；

2：目標數據，即查詢的結構顯示工作表，在名為“查詢”的工作表內；

3：按照C2、C3 單元格的查詢條件，查詢出相應的數據；

解決的思路詳解：

1：把“進價”的工作表的數據，放入arr數組；

2：用for循環遍歷數組；

3：用if判斷是否滿足查詢的條件；

4：把滿足條件的數據，放入數組brr；

5：最後在指定的單元格區域內，放入brr；

代碼運行的結果如下：

代碼如下:

Sub test() Dim r%, i% Dim arr, brr With Worksheets("查詢") If .Range("c2") = "" Or .Range("c3") = "" Then MsgBox "查詢條件不能為空！" Exit Sub End If tj1 = .Range("c2") tj2 = .Range("c3") End With With Worksheets("進價") r = .Cells(.Rows.Count, 1).End(xlUp).Row arr = .Range("a3:k" & r) ReDim brr(1 To UBound(arr), 1 To UBound(arr, 2)) For i = 1 To UBound(arr) If arr(i, 3) = tj1 And arr(i, 4) Like tj2 & "*" Then m = m + 1 For j = 1 To UBound(arr, 2) brr(m, j) = arr(i, j) Next End If Next End With If m = 0 Then MsgBox "沒有查詢到符合條件數據！" Exit Sub End If With Worksheets("查詢") .UsedRange.Offset(5, 0).ClearContents .Range("a6").Resize(UBound(brr), UBound(brr, 2)) = brr End WithEnd Sub

代碼解析

1：2~3行 定義變量；

2：4~8行 判斷查詢條件是否為空；

3：9~10行 查詢條件賦值；

4：13行 獲取“進價”工作表的最後一行行號；

5：14行 把查詢數據的數據源區域放入arr數組；

6：15行 定義一個與arr區域或者說大小一樣的brr數組；

7：16~23行 遍歷arr數組；

8：17行 判斷相應的條件是否符合查詢條件（注意用了like進行了模糊的匹配）；

9：18行 計算器m；

10：20行 把查詢的結果，從arr數組賦值到brr數組；

11：25~27行 如果計數器的結果為0 ，則彈出對話框，沒有"沒有查詢到符合條件數據！"；

12：30行 清空指定區域的格式；

13：31行 把brr數組，賦值給指定的區域；

本例思考：

1：本例利用了Like運算符，使查詢的運用範圍，更加的廣泛。

小結：

解決本問題，需要用的知識點：

1：Like運算符
