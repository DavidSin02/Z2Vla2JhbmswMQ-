AB羅克韋爾自動化Micro850可編程邏輯控制器系統型號及功能介紹

2020-11-14
| 科技

產品介紹

我們的 Micro850® 可編程邏輯控制器系統適用於需要靈活通信和更強 I/O 功能的獨立設備控制應用。這些控制器支持最多 132 個 I/O 點，同時提供高性能 I/O、中斷和 PTO 運動控制以及嵌入式以太網和 2085 擴展 I/O。

Micro850可編程邏輯控制器系統產品

產品參數

提供 24 點和 48 點控制器

在 24V 直流型號上包含 100 kHz 的高速計數器 (HSC) 輸入

通過 USB 編程端口、非隔離串口（用於 RS-232 和 RS-485 通信）和以太網接口

最多支持 5 個 Micro800™ 插件式模塊

最多支持 4 個 Micro850® 擴展 I/O 模塊、132 個 I/O 點

支持多達 3 個脈衝序列輸出 (PTO)功能，

通過 EtherNet/IP™ 進行通信

在 -20 至 65 °C（-4 至 149 °F）溫度下工作

通過插件式模塊支持用於程序傳輸、數據記錄和配方管理的 microSD™ 卡

特性優勢

• 外形尺寸與 Micro830® 24 點 和48 點控制器相同，在支持功能性插件、指令/數據容量，運動控制方面也相同

• EtherNet/IPTM 可用於 Connected Components WorkbenchTM 編程、RTU 應用、連接人機界面。可通過客戶端報文方式連接變頻驅動器，並採用符號尋址與其它控制器進行通信。

• 適用於需要更高密度和更高精度的模擬量和數字量 I/O 的大型單機應用項目 (與 Micro830 控制器相比)

• 結合 Micro850 擴展 I/O 模塊，48 點控制器最大可擴展到 132 個數字量I/O 點

• 支持多達 4 個 Micro850 擴展 I/O模塊

• 採用可拆卸端子塊，靈活性更出色

• 可免費下載標準版 ConnectedComponents Workbench 軟件

功能介紹

對於尋求高靈活性、高定製性、高 I/O 性能以及低空間佔用的機器製造商和最終用戶而言，Micro850 可擴展控制器是最理想的解決方案。作為Micro800® 系列的旗艦級控制器，Micro850 還能搭載尺寸小巧的功能性插件和擴展 I/O 模塊，並採用可拆卸端子塊設計，這無疑將 Micro800 PLC系列的靈活性和可定製性推向了一個新的高度。此外，Micro850 控制器具備與 Micro830 24 點、48 點控制器相同的外形尺寸、功能性插件支持度、指令/數據容量以及嵌入式運動控制功能。嵌入式運動控制功能可通過TouchProbe 指令支持 3 軸運動，該指令能夠記錄軸的位置，比使用中斷更加精確。Micro850 擴展 I/O 模塊專為大型單機應用項目而設計，能夠根據需要幫助實現更高密度以及更高精度的模擬量和數字量 I/O。Micro85048 點控制器能夠支持多達 4 個擴展 I/O 模塊，最多可擴展至 132 個 I/O 點。

Connected Components Workbench 軟件能夠在 Micro800 全系列控制器以及 PanelView Component 人機界面和 PowerFlex 變頻器等其它組件產品之間共享。這種新型軟件基於羅克韋爾自動化和 Microsoft Visual Studio的成熟技術，為 PanelView Component 操作員產品提供了控制器編程、設備配置以及與人機界面編輯器共享數據的功能。此外，該軟件可支持 3 種標準 IEC 編程語言：梯形圖、功能塊圖和結構化文本。為增強安全性，所有Micro800 控制器均支持控制器密碼保護。
