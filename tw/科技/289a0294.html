VBA｜選區字符統計並顯示到狀態欄

2020-11-14
| 科技

選擇單元格中任意區域時，在狀態欄顯示選區中的字符個數，以及字母、數字和漢字個數。

Private Sub Worksheet_SelectionChange(ByVal Target As Range)

Dim 字符 As String, 字符數 As Long, 漢字 As Long, 字母 As Long, 數字 As Long, 其它符號

Dim rng As Range, Item As Integer, rngg As Range

'遍歷選區，為了防止用戶全選或者選列整列時造成的不必要的循環，使用Intersect方法讓程序在非空區循環

If Intersect(Selection, ActiveSheet.UsedRange) Is Nothing Then Application.StatusBar = "": End

For Each rng In Intersect(Selection, ActiveSheet.UsedRange)

'彙總所有單元格的字符數

字符數 = 字符數 + Len(rng)

'逐一提取每個單元格中的每個字符

For i = 1 To Len(rng)

字符 = Mid(rng, i, 1)

'一到龥之間的都是漢字（僅適用於簡體系統）

If 字符 Like "[一-龥]" = True Then

漢字 = 漢字 + 1 '漢字累加

ElseIf 字符 Like "[a-zA-Z]" = True Then

字母 = 字母 + 1 '字母累加

ElseIf 字符 Like "[0-9]" = True Then

數字 = 數字 + 1 '數字累加

Else

其它符號 = 其它符號 + 1

End If

Next

Next

Application.StatusBar = "字數：" & 字符數 + 0 & "個，其中：" & "漢字：" & 漢字 + 0 & "個，" & _

"字母：" & 字母 + 0 & "個，" & "數字：" & 數字 + 0 & "個，" & "其它符號：" & 其它符號 + 0 & "個"

End Sub
