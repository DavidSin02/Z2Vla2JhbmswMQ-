「通信與廣電實務簡答知識彙總五」：施工質量管理

2020-10-29
| 科技

質量監督機構到施工現場進行質量監督是否正確

質量監督機構是政府依據有關質量法規、規章行使質量監督職能的機構，具有典型的第三方檢驗的公正性。工業和信息化部通信工程質量監督機構負責全國通信工程質量監督工作，省、自治區、直轄市通信工程質量監督機構負責各行政區內的通信工程質量監督工作。施工單位應自覺接受質量監督機構的質量監督，為質量監督檢查提供方便，並與之積極配合。

施王單位質量行為的規範

1.依法承攬及分包工程

2.建立質量責任制

3.做好項目施工前質量策劃及設備器材檢驗工作

4.按標準及規範要求施工

5.自覺接受質量監督機構的質量監督

6.提供竣工資料及工程返修、保修

質量控制點的控制方法

1.制定相應的控制措施

2. 進行技術交底

3. 落實"三檢"制度

4. 確保測量設備處於良好狀態

5. 制定糾正或預防措施

設備安裝準備階段控制點

1.現場勘查:機房內部裝修質量;地槽、走線路由檢查;交流電源引人質量，機房照明情況檢查;機房的防靜電地板(如有) ，情況檢查;空調系統情況檢查，機房溫度、溼度情況檢查;機房的防雷接地排、保護接地排是否有空餘位置;機房內消防設施的完整性。

2. 器材檢驗:現場開箱檢驗設備、主要材料的品種、規格、數量、外觀檢查;檢查設備、主要材料的出廠合格證書、技術說明書是否齊全。

設備安裝階段控制點

1.走線槽(架)安裝:走線槽(架)安裝的位置、高度，水平度、垂直偏差;吊掛、

撐鐵的安裝。

2. 機架安裝質量:基座的位置、水平度;機架安裝的位置、垂直度，機架上加固

(如有)製作;機架的標誌製作。

3. 子架安裝質量:子架安裝的位置檢查;面板佈設;子架安裝、插接件的連接。

4. 室外設備的安裝:地線排的電阻率;塔上設備安裝的位置、方向、緊固;饋線的

接頭、固定、防水、彎曲度、接地等裝置的製作。

5. 光、電纜布放及成端:光纖的布放路由走向、衰耗、標誌;射頻電纜布放的路由

走向、電纜排序以及綁紮、成端、標誌的製作質量;電力電纜布放的路由走向、絕緣、極性、標誌。

6. 電纜頭的焊接:牢固度、光滑度、餘線長短的→致性。

設備測試階段控制點

1.設備通電檢查:電源線連接的極性是否正確，電源電壓;加電步驟，各種可聞、可視告警設施。

2. 電源設備安裝測試:輸入電壓、輸出電壓、充放電試驗、絕緣測試、保護地線電阻測試。

3. 交換設備測試:硬件測試(模塊單元硬件診斷測試、大話務量呼叫測試、外接設備電源告警測試、計費準確率測試、端子板用戶呼叫測試、錄音通知測試)、軟件測試。

4. 微波設備測試:天線場強、天線交叉極化鑑別度、饋線損耗、天饋線系統駐波比、發信機輸出功率、本振頻率及中頻頻率準確度、接收機噪聲係數、接收機門限電平、空間分集時延、中頻特性。

5. 基站設備測試:發送功率、時鐘準確度、天饋線系統駐波比。

6. 光傳輸設備測試:平均發送光功率、接收光功率、接收靈敏度、過載光功率、輸入抖動容限、輸出抖動、系統輸出抖動、誤碼率指標測試。

線路工程質量控制點

工程準備階段

1.施工現場摸底:核對圖紙，確定障礙點及重要點。

2. 線纜單盤檢測:對使用的光(電)纜進行光、電性能測試、外觀檢查。

3. 路由複測及光(電)纜配盤:線路路由的定位、劃線，路由適當調整;線纜配盤。

土石方施工

開挖纜溝:挖溝的位置、深度、溝寬、穿越障礙物、與其他管線的間距;路由上的障礙處理;掘路、頂管、截流。

杆路施工

1.立線杆:杆路定位、杆坑及拉線坑深度、杆距、拉線角度;電杆、拉撐設備及附

屬設備電器性能和物理性能檢測;電杆的埋設工藝;拉撐設備的安裝工藝。

2. 架設吊線:鋼絞線拉開、架設、調整吊線垂度;架設時空中障礙物、電力線交越的處置。

線纜敷設

1.埋式線纜敷設:線纜的敷設、穿越障礙點、埋深;線纜的 A/B 端，敷設中的通信聯絡、人員組織、線纜安全。

2. 埋式線纜回填及保護:線纜在溝底的狀況;分層回填及勞實，敷設排流線、鋪蓋板、紅磚等保護措施;線纜對地絕緣;全部回填。

3. 架空線纜敷設:光(電)纜卡勾間距、線纜架設、預留、保護。

4. 布放管道光纜:管孔佔用、子管穿放、子管佔用、管口封堵;人(手)孔內原有

設備的保護;線纜穿放、人孔內線纜預留。

5. 布放管道電纜:電纜的端別、電纜氣壓、絕緣;電纜配盤、敷設方向確定;全塑電纜彎曲餘長、尾巴電纜的位置、長度;電纜充氣、氣閉測試;芯線接續、接口封閉、屏敲層測試、接地測試。

6. 定向鑽敷設管線:機械調試，管道的口徑、光纜進出管道的位置。

7. 氣流吹放管道光纜:機械調試、吹放氣壓、纜線準備。

線路設備安裝

1.交接設備安裝:交接設備及附屬設備的電器性能檢測;安裝的環境因素和位置選擇;成端電纜的編把、綁紮工藝;電纜的標誌和安全防護措施;跳線穿放工藝。

2. 配線電纜布放:設備器材及附屬設備測試;電纜架設的工藝;絕緣電阻測試;芯

線接續工藝;屏蔽防護層連通測試;架空電纜接口封閉;引上電纜、分線設備安裝工藝;環阻測試。

3. 充氣設備安裝:充氣設備、管路及輔助材料電器性能檢測;充氣設備及各路管線的安裝;高壓罐、低壓罐及各管線密閉檢查;充氣設備的自動控制和告警檢查。

4. 局內成端電纜布放:主幹電纜的布放、電纜彎曲、餘長定位、電纜標誌、布放綁; 成端電纜上列，把線制、編、綁;保安排的安裝;成端豎接頭的安裝;成端接頭的屏蔽、接地、封閉、充氣。

線纜接續及測試

1.線纜接續及實時測試:線纜接續、光纖損耗的 OTDR 實時測試、電纜性能測試;

接頭預留纜的保護、接頭盒的安裝、接頭監測標石的安裝及編號。

2. 線纜成端及標識:線纜成端的衰耗測試、標識;佈線工藝、固定。

3. 光纜:每根光纖的接頭損耗、中繼段損耗、光功率、 PMD 值等性能測試。

4. 電纜:線路的絕緣、電阻、串音、電容等電氣性能測試;氣壓測試。

標識、保護

1.路由標識、繪圖控制點:標石的埋設、刷漆、編號;繪製竣工草圖;杆路編號、

測量間距、繪製竣工草圖(含杆面圖)。

2. 管線標識、繪圖控制點:人(於)孔編號、測量人孔段長、繪製竣工草圖(含管

孔佔用圖)。

3. 路由地面保護措施:護坡、護坎、堵塞、漫水壩的製作。

4. 人(於)孔內線纜的保護:根據設計要求，人(於)孔內裸露線纜採取保護措施，

綁紮固定、掛牌標識。

管道工程質量控制點

施工前路由及環境勘查及測量

瞭解路由內是否有其他交越、重合的管線，地面上的樹木、建築、電杆、拉線、道路及各種檢查井對施工的影響程度;地質條件、地下水情況。水準點的確定。

進場材料的清點檢查

對全部進場材料的型號、規格、質量、數量清點檢查;對材料的生產廠家、出廠合格證、人網證檢查核對。

管道坑槽

管道坑槽的土質、寬度、深度、放坡比例;槽底障礙的處理、管道與障礙物的淨距離;換土窮實情況。

管道基礎

管道基礎鋼筋製作與綁紮、管道基礎的寬度、厚度、位置偏移;基礎混凝土配合比及養護時間、強度;模板安裝、拆除。

水泥管道敷設

水泥管道的敷設工藝、管塊與管塊及管塊與基礎間縫隙、行間及層間縫隙;管帶管縫處理、底角八字抹灰、鋪管砂漿配合比、養護時間、強度。

鋼管敷設

鋼管排列、定位架、鋼管的防腐及接口處理、管間縫的填充、管道進入人(手)孔的排列。

塑料管敷設

塑料管排列、定位架安裝、接口處理、管間縫的填充、管道進入人(於)孔的排列。

包封加固鋼筋的製作與綁紮、模板安裝、拆除;混凝土配比、養護時間、強度。

回填土方

土質更換情況、管道兩側窮實、管頂回填高度、不同路面分層回填窮實。

人(手)孔、電纜通道施工

人(於)孔、電纜通道尺寸;基礎厚度及所用鋼筋的型號、泛水高度、積水罐安裝位置;內、外壁抹灰;窗口位置;人孔上覆、井口、口圈安裝;電纜支架、人孔專用器材的安裝。

管道試通

試通管孔的選擇、試通棒的規格。

質量控制依據的內容

施工規範、操作規程、設計圖紙、作業指導書、施工技術

施工質量控制對策

以人的工作質量確保工程質量

嚴格控制投入晶的質量

全面控制施工過程，重點控制工序質量

堅持預防為主的方針

重大質量事故

有下列情況之一者，為重大質量事故:

1.由於工程質量低劣，引起人身死亡或重傷 3 人以上(含 3 人) ;

2. 直接經濟損失在 50 萬元以上。

嚴重質量事故

有下列情況之一者，為嚴重質量事故:

1.由於工程質量低劣，造成重傷 1 至 2 人;

2. 直接經濟損失在 20 萬元至 50 萬元者;

3. 大中型項目由於發生工程質量問題，不能按期竣工投產。

一般質量事故

凡具備下列條件之一者，為一般質量事故:

1.直接經濟損失在 20 萬元以下;

2. 小型項目由於發生工程質量問題，不能按期竣工投產。

質量事故書面報告內容:

1.事故發生的時間、地點、工程項目名稱，建設、維護、設計、施工、監理和質量監督單位名稱;

2. 事故發生的簡要過程、傷亡人數和直接經濟損失的初步估算;

3. 事故發生原因的初步判斷;

4. 事故發生後採取的措施及事故控制情況;

5. 事故報告單位。

質量事故調查人員在調查過程中，應堅持做好以下工作:

1.查明事故發生的原因、過程、事故的嚴重程度和經濟損失情況

2. 查明事故的性質、責任單位和主要責任者

3. 組織技術鑑定

4. 提出事故處理意見，明確事故主要責任單位和次要責任單位承擔經濟損失的劃分原則

5. 提出技術處理意見及防止類似事故再次發生所應採取措施的建議;

6. 提出對事故責任者的處理建議;

7. 寫出事故調查報告。

直埋光纜線路工程質量事故的預防措施

1.直埋光纜埋深不符合要求的預防措施:纜溝開挖深度嚴格按照設計要求和有關驗收標準執行;放纜前應清理溝底雜物，全面檢查溝底深度，重點檢查溝、坎等特殊點溝底的深度，並保持溝底平整;光纜應順直地貼在溝底;回填土應滿足規範要求。

2. 光纜敷設時打結釦的預防措施:敷設光纜時，應嚴格按施工操作規程進行;配備

必要的放纜機具;按照規範要求盤好"∞"形;根據不同的施工環境配備足夠的敷纜人數;控制好放纜的速度;在關鍵點要安排有豐富經驗的人員具體負責;保證光纜在現場指揮人員的視線範圍以內。

3. 光纖接頭衰耗大、纖芯在接頭盒中擺放不整齊、接頭盒安裝工藝不符合要求的預防措施:對從事光纖接續的技術人員應進行技術培訓和示範，使其熟練掌握接續、安裝要領;對切割刀、光纖接續設備和光時域反射儀 (OTDR) 進行維護和校準，滿足施工需要;按接頭盒說明書的要求盤纖，並保證光纖的曲率半徑滿足規範要求;現場環境的溫度、潔淨度應滿足接續要求。

4. 光纜對地絕緣不合格的預防措施:在光纜敷設和回填士時應避免光纜外皮損傷，接頭盒必須按工藝要求封裝嚴密。

杆路工程質量事故的預防措施

1.電杆傾倒的預防措施:電杆進場要進行嚴格的質量檢查，符合出廠標準;直線杆

路中間杆位的左右偏移量不得超出驗收規範要求;杆根裝置應按設計和相關操作規程安裝，電杆埋深應符合驗收規範或設計要求;不同位置電杆的垂直度、拉線距高比、地錨的埋深和地錨出土點的左右偏移量均應滿足規範要求，不得隨意改功地錨位置;地錨應埋設牢固;電杆回填土要弄實;杆路的檔距和杆上負荷必須符合工程設計要求、收緊拉線時應鬆開吊線上的夾板。

2. 吊線垂度不符合要求的預防措施:吊線收緊時應根據不同地區、地形和掛設的光(電)纜程式分段進行，分檔檢查，每檔吊線垂度都要滿足規範要求;在吊線收緊時，不得使吊線過緊;吊線上的電纜負荷應滿足要求;線路的杆距不宜過大;終端拉線製作應滿足要求。

市話電纜工程質量事故的預防措施

1.芯線間絕緣電阻小的預防措施:在敷設電纜前應嚴格檢查線間絕緣電阻、耐壓和串音等項目，保證電纜的各項電氣性能良好;電纜應帶氣布放;施工完畢後，電纜端頭應及時封閉，及時進行氣壓維護;接頭盒應按說明書要求嚴密封裝。

2. 接頭接觸不良或棍線的預防措施:在刨電纜端頭分線時，要按規定色譜分編芯線;檢查接線子質量，壓接或焊接工具滿足施工要求;接續結束，應認真對線;電纜接頭應儘早密封;封裝接頭套管前應填寫接頭責任卡片;完成接續的電纜應及時保氣。

通信管道工程質量事故的預防措施

1.塑料管孔接頭斷開造成管孔不通的預防措施:如果是管孔溝槽地基土質鬆軟而容易下沉的原因，應採取措施芳實地基，並澆灌混凝土基礎。接頭塑膠使用時，應進行試驗，粘結不牢固時，要及時更換。

2. 管道漏水的預防措施:敷設的塑料管在接頭時，管孔接頭不得鬆弛，接頭兩端橡膠圈的質量應良好;接頭處應嚴密堵封;在人(手)孔內，管孔兩端應堵封嚴密。

3. 硅芯管漏氣的預防措施:硅芯管運到工地後，應作保氣試驗;敷設時禁止在地面上拖、磨、刮、蹭、壓;因填土時，不要使石頭等尖狀物體損壞硅芯管;用接頭套管接續時，一定要保持接頭嚴密;每段接續(兩人孔之間)完成後，可回填部分細土，應按要求作保氣試驗，確認不漏氣後，再回填土。

4. 人(於)孔漏水、滲水，體積不符合要求的預防措施:在建人(手)孔時，按照規定尺寸放樣，用測量工具經常檢查，發現問題，及時糾正;在建人(於)孔基礎時，要採取防止漏水、滲水的措施，在砌牆體時，裡外都要按要求抹一定厚度的水泥牆面;人孔上覆必須按圖紙要求製作;牆體與基礎之間、上覆與牆體之間、口圈與上覆之間均應使用規定強度等級的水泥抹"八"字。

室內設備安裝工程質量事故的預防措施

1.纜線布放不整齊的預防措施:布放線纜前應設計好纜線的截面;嚴格按施工操作規程和工程驗收規範要求放纜;線纜綁紮的鬆緊度應符合要求;做好纜線的整理工作。

2. 線纜端接的假焊和虛焊的預防措施:焊接前，組織人員進行示範和操作技術交底;選派有經驗和合格的操作人員與質量檢查人員進行操作和及時檢查;選用合格的焊接工具，並保證焊接工具工作時的溫度滿足要求;應防止烙鐵頭氧化不粘錫;使用適宜的焊劑;焊錫絲的質量應合格。

3. 機架、槽道、走線架安裝不整齊的預防措施:機架、槽道、走線架安裝的位置、垂度及機架之間縫隙必須滿足工程規範要求;安裝好的機架、槽道、走線架應加固牢固。

天、饋線工程質量事故的預防措施

1.天線隨風左右搖擺的預防措施:固定在天線支撐杆架上的天線要按說明書要求加固牢固;天線支架的螺釘及固定天線的螺絲均應緊固。

2. 室外饋線受潮、損耗變大的預防措施:要嚴格按操作規程和操作工藝對饋線接頭進行防水處理，如在接頭處纏防水膠帶時必須由下而上纏繞，使接頭防水可靠。

3. 饋線接口處斷裂的預防措施:饋線接口應自然對齊，不得強行受力使其扭曲。

4. 天線方位角(方向)誤差較大的預防措施:應用羅盤或場強儀測定，準確定位，

調整天線支撐架位置，固定牢固。

5. 防雷措施不到位的預防措施:天、饋線的避雷設施必須按工程設計要求安裝，接地裝置的電阻值必須經檢驗合格。

鐵塔工程質量事故的預防措施

1.拉線塔倒塌的預防措施:要注意拉錨工程質量，理設位置、深度應符合要求;各條拉線的拉力應均勻;塔身材料和鋼絞線繩索的規格、質量應符合要求。

2. 自立式鐵塔傾斜和倒塌的預防措施:鐵塔基礎施工時，每道工序必須經監理檢驗合格;開挖基礎坑時，必須挖到地基的持力層，以防止基礎不均勻下沉;基礎用的鋼筋材料要嚴格檢驗，必要時，取樣送檢驗機構檢驗其理化特性;澆灌基礎混凝土時，混凝土必須嚴格按照配合比的要求配製;鐵塔基礎的強度和保養時間應符合要求;塔身材料必須有出廠的化學成分檢驗報告，並符合質量要求;地腳螺栓澆灌應牢固，塔身連接螺栓應緊固，焊接點焊接應牢靠，避雷設施完好;在組裝塔身時，每安裝一層都要用經緯儀進行測量，保持塔身垂直度符合要求。

3. 防腐的預防措施:塔身材料必須採用熱鍍缽鋼材，所有焊接點及避雷設施都必須經過防腐處理。
