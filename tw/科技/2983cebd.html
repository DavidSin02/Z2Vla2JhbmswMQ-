心電圖常用術語解釋

2020-11-14
| 科技

心電圖是檢查心臟疾患常用的方法之一，目前也是健康體檢的必查項目之一。但很多人對心電的一些術語感到很費解，這些心電術語都代表了什麼意思呢？

  1. 1. 心電基礎知識


英文

中文

術語解釋

Resting ECG

靜息心電圖機

根據心電環境要求來分類，需要病人安靜躺在病床上，採集時間為10秒到300秒，適用大部分心臟疾病的初級篩查。

Holter / Dynamic ECG

動態心電圖機

作為靜息心電圖機的補充，病人可以進行一切正常生活行動，採集盒佩戴在病人身上，可以監護24/48/72小時，醫生通過強大的分析軟件進行全方位的數據分析；適用於長時間心臟監測，特別是一些慢性的、在安靜平躺的條件下激發不了的心臟病。

Stress ECG

運動心電圖機

作為前兩種心電圖儀的補充，病人通過跑步機或腳踏自行車進行逐漸加速運動，採集盒佩戴在病人身上，連接PC上的運動心電分析軟件，實時分析病人在運動條件下心臟工作情況；適用於只有在激烈運動下才能激發的心臟病檢查。

Lead

導聯

心臟除極，復極過程中產生的電生理信號，將正負電極安置於體表相隔一定距離的任意兩點，原則上均可測出心電的電位變化，此兩點構成一個導聯。

Channel

通道

對應 printing功能，指的是在打印時，最多可以同時打印多少個通道的波形；

心電圖機有12個導聯，心電報告需要把12個導聯全部打印出來，如果打印時：

一次最多打印一個導聯，分成十二組，完成12個導聯，稱為單道心電圖機；

一次最多打印三個導聯，分成四組，完成12個導聯，稱為3道心電圖機；

一次最多打印6個導聯，分成兩組，完成12個導聯，稱為6道心電圖機；

一次可以完成12個導聯打印，稱為12道心電圖機；

一次可以完成15個導聯打印，稱為15道心電圖機；

P wave

P 波

正常心臟的電激動從竇房結開始。由於竇房結位於右心房與上腔靜脈的交界處，所以竇房結的激動首先傳導到右心房，通過房間束傳到左心房，形成心電圖上的P 波。 P波代表了心房的激動， 前半部代表右心房激動， 後半部代表左心房的激動。 P 波時限為0.12秒，高度為0.25mv。當心房擴大，兩房間傳導出現異常時，P 波可表現為高尖或雙峰的P波。

QRS complex

QRS 波群

激動向下經希氏束、左右束支同步激動左右心室形成QRS波群。QRS波群代表了心室的除極，激動時限小於0.11秒。當出現心臟左右束支的傳導阻滯、心室擴大或肥厚等情況時，QRS波群出現增寬、變形和時限延長。

T wave

T 波

之後的T 波代表了心室的復極。在QRS波主波向上的導聯，T 波應與QRS主波方向相同。心電圖上T波的改變受多種因素的影響。例如心肌缺血時可表現為T波低平倒置。T 波的高聳可見於高血鉀、急性心肌梗死的超急期等

PR interval

PR 間期

激動沿前中後結間束傳導到房室結。由於房室結傳導速度緩慢，形成了心電圖上的PR 段，也稱PR間期。 正常PR 間期在0.12～0.20秒。當心房到心室的傳導出現阻滯，則表現為PR 間期的延長或P 波之後心室波消失

QT interval

QT 間期

代表了心室從除極到復極的時間。 正常QT 間期為0.44秒。QT間期的延長往往與惡性心律失常的發生相關。

PR segment

PR 段

代表心房復極過程及房室結、希氏束、束支的電活動。

ST segment

ST 段

心室肌全部除極完成，復極尚未開始的一段時間。此時各部位的心室肌都處於除極狀態，細胞之間並沒有電位差。因此正常情況下ST段應處於等電位線上。當某部位的心肌出現缺血或壞死的表現，心室在除極完畢後仍存在電位差，此時表現為心電圖上ST段發生偏移

Frank system

弗蘭克導聯繫統

1956年，Frank提出了一套校正的正交導聯繫統，該導聯繫統共有7個電極。胸部放置5個電極，另外兩個電極分別放在左足和頸部背面偏右1cm處。每個電極連有不同的電阻，在一定程度上校正了心臟在胸腔中偏左前和人體導電的不均勻性，由於其物理基礎健全、設計合理而廣泛用於心電向量圖技術中。為了標記空間心電向量而設想三個軸互相垂直相交於心臟中心，再由三個軸分別組成Frank導聯繫統的水平面、矢狀面和額面。

Wilson system

威爾遜導聯繫統

Wilson導聯體系是有威爾遜提出的中心端連接導聯體系，將探查電極放在標準導聯的任一肢體上，而將其餘二肢體上的引導電極分別與5000歐姆電阻串聯在一起作為無關電極。這種導聯記錄出的心電圖電壓比單極肢體導聯的電壓增加50％左右，故名加壓單極肢體導聯。根據探查電極放置的位置命名，如探查電極在右臂，即為加壓單極右上肢導聯（aVR），在左臂則為加壓單極左上肢導聯（aVL），在左腿則為加壓單極左下肢導聯。


VCG

心電向量

心肌細胞在除極和復極過程中形成的電位差，既有大小又有方向，稱為心電向量。心電向量圖與心電圖是相輔相成、相互補充、用來診斷心臟病的兩種手段。某一導聯上的心電圖是一條隨時間t展開的曲線,而某一面上的向量圖,是一條以t為參數的複雜平面曲線。向量圖有三個電軸X、Y、Z,兩兩構成一個面,稱橫面(H)、額面(F),右側面(s)。

AD converter

AD轉換

AD轉換就是模數轉換，就是把模擬信號轉換成數字信號，單位為位，位數用來衡量轉換精度的問題。常見的有12位，16位，24位。

Sampling rate

採樣率

即採樣頻率，定義了每秒從連續信號中提取並組成離散信號的採樣個數，單位赫茲（Hz）；頻率越高，採樣個數越多，精度越好。

CMRR

（Common mode rejection ratio）

共模抑制比

指心電圖機的差模信號（心電信號）放大倍數Ad與共模信號（干擾和噪聲）放大倍數Ac之比，表示抗干擾能力的大小。比值越大，此時差分放大電路抑制共模信號的能力越強，抗干擾能力越優良。

Noise level

噪聲電平

也叫內部噪聲。是指心電圖機內部元器件工作時，由於電子熱運動產生的噪聲，而不是因使用不當外來干擾形成的噪聲，這種噪聲使心電圖機沒有輸入信號時仍有微小的雜亂波輸出，這種噪聲如果過大，不但影響圖形美觀，而且還影響心電波的正常性，因此要求噪聲越小越好，在描記曲線中應看不到噪聲波形。

Frequency response

頻響範圍

人體心電波形並不是單一頻率的，而是可以分解成不同頻率、不同比例的正弦波成分，也就是說心電信號含有豐富的高次諧波。若心電圖機對不同頻率的信號有相同的增益，則描記出來的波形就不會失真。但是放大器對不同頻率的信號的放大能力並不一定完全一樣的。心電圖機輸入相同幅值、不同頻率的信號時，其輸出信號幅度隨頻率變化的關係稱為頻率響應特性。心電圖機的頻率響應特性主要取決於放大器和記錄器的頻率響應特性。頻率響應越寬越好。

Polarization voltage

極化電壓

皮膚和表面電極之間會因極化而產生極化電壓。這主要是由於心動電流流過後形成的電壓滯留現象，極化電壓對心電圖測量的影響很大，會產生基線漂移等現象。極化電壓最高時時可達數十毫伏乃至上百毫伏。處理不好極化電壓，產生的干擾將是很嚴重的。儘管心電圖機使用的電極已經採用了特殊材料，但是由於溫度的變化以及電場和磁場的影響，電極仍產生極化電壓，一般為200～300mV，這樣就要求心電圖機要有一個耐極化電壓的放大器和記錄裝置。要求大於300mV，國際上大於500mV。

Input impedance

輸入阻抗

即前級放大器的輸入電阻。輸入電阻越大，因電極接觸電阻不同而引起的波形失真越小，共模抑制比越高。一般要求大於2MΩ，國際上大於50MΩ。

Sensitivity

靈敏度

也叫增益；是指輸入1mV標準電壓時，記錄波形的幅度。通常用mm/mV表示，它反映了整機放大器放大倍數的大小。心電圖機標準靈敏度為10mm/mV。規定標準靈敏度的目的是為了便於對各種心電圖進行比較。

Paper speed

走紙速度

用來體現心電圖紙運行的快慢，單位為mm/s

術語解釋

ECG mode

心電模式

12導聯心電圖

標準12導聯：I，II, III, avR, avL,avF, V1, V2,V3,V4,V5,V6

15導聯心電圖

在標準12導聯的基礎上，多加三個導聯，根據成人、兒童這兩種類型，有不同的導聯可選：

成人一般臨床上建議選擇：V4R，V8，V9

兒童一般臨床上建議選擇：

iE15上此模式默認為成人15導聯；

15導聯的臨床意義：

1） 提高心梗診斷的敏感性

2）提高右室梗死的檢出率

3）對診斷迴旋支缺血引起的後壁損傷是一種有用且診斷較準確的手段

兒科心電圖

一鍵進入兒童15導聯模式

心電向量圖

進入心電向量工作模式，X,Y,Z的正交導聯分析模式

Lead mode

導聯模式

標準12導聯

標準12導聯，順序為：I，II, III, avR, avL,avF, V1, V2,V3,V4,V5,V6

Cabrera導聯

2000年歐洲心臟病學會和美國心臟病學會（ESC/ACC)指南推薦Cabrera導聯作為標準導聯，便於心肌缺血或心律失常的起源定位；對梗死部位的判斷和梗死麵積的估計具有重要的意義；

導聯的順序為：avL, I, -avR , II, avF, V1, V2,V3,V4,V5,V6

Nehb 導聯

一種胸前雙極導聯心電圖，連接方法為：將肢體導聯的右手電極接胸骨右緣第2肋間，左下肢電極接在心尖搏動點上，左手電極接心尖水平的腋後線，右下肢地線電極不變。描記得I導聯圖為Dorsalis導聯，II導聯圖為Anterior導聯，III導聯圖為Inferior導聯。

臨床意義：心房波的辨別在心電診斷，尤其在心律失常診斷中有重要意義。常規心電圖有時對一些低小、不能明視的心房波難以確定。Nehb導聯的探查電極靠近心臟表面，且其3個導聯軸構成的三角形接近心電偶軸；當電極距心電偶中心越近，越靠近心電偶軸，其電位絕對值就越大，這就是Nehb導聯心電圖各波振幅高、ST-T改變敏感之故。P波顯示清楚，對於診斷以P波及P-R段為主要診斷依據的心律失常優於常規導聯；R波振幅高且ST-T改變陽性率高、出現時間早、改變程度明顯，且D導聯優於A導聯，這對於診斷冠心病和左室肥大具有早期診斷的價值；U波振幅高，對診斷低血鉀症等亦有較大的幫助。在可疑冠心病患者尤其是年老體弱者，當Nehb導聯出現缺血性ST-T改變而常規導聯正常時，結合臨床可以診斷冠心病，從而免去運動試驗及可能發生不良反應的藥物試驗。

Low-pass filter

(EMG filter)

低通濾波器

25Hz、35Hz、75Hz、100Hz、150Hz、250Hz

由於靜態心電圖要求病人安靜躺在床上，但有些病人，如骨折病人、調皮的小孩，出現抽搐的病人等，這些病人沒有辦法很好地控制自己的身體，出現挪動或顫動，對心電圖形產生干擾，此濾波器用來消除這部分的干擾。

Baseline wander filter

(ADS filter)

(high-pass filter)

基線漂移濾波器

0.01 Hz, 0.02 Hz, 0.05 Hz、0.35 Hz, 0.5 Hz、0.8 Hz

由於受到使用環境的影響，如其它設備正在使用，或者機器本身某些元器件工作時產生的干擾，引起心電波形的基線不穩定，出現漂移情況，此濾波器的不同選項可以消除不同的干擾信號

AC filter

交流濾波器

50Hz，60Hz

連接交流電，交流電產生的信號干擾，此濾波器用來消除這部分的干擾

Freeze

波形凍結

30~300秒

在臨床中適合突然發生的心律失常或一些不配合做心電圖的小孩或者皮膚乾燥或心電電極很難固定的老人和骨折病人，常規心電圖裡存在比較多的干擾波等。凍結後，就可以選擇最有價值的10秒的波形進行打印記錄，所以在臨床的意義重大

Rhythm mode:

Single rhythm mode

Three-rhythm mode

節律模式

單節律

由於普通的心電模式一般只採集10秒的數據進行分析和打印，有時候不足以用來判斷某個心臟疾病。運用節律模式，可以長時間對一個導聯或三個導聯進行最多300秒的波形採集和數據分析，用於醫生的進一步細緻觀察。

單節律模式，可以從12個導聯中任選一個導聯進行長達300秒的分析；

三節律模式，可以從12個導聯中任選三個導聯，每個導聯可以進行100秒的分析；

三節律

Extra lead

額外導聯

應用於15導心電中，有三個選項：標準成人:V4r, V8, V9; 標準兒童:V3r, V4r, V7; 後壁:V7, V8, V9；

選擇一個後，選擇額外三導模式時，自動切換成所選項；這三個選項的算法是分別對應成人，兒童和心臟後壁分析的。

Pacemaker detection

起搏檢測

如果病人佩戴了心臟起搏器，醫生可以選擇不同的起搏強度，分為正常，較弱和較強，以便更好地分析和診斷起搏器的工作狀態

Arrhythmia mode

心律不齊檢測

其他廠家也稱為：延長打印功能；

如果開啟，檢查過程中發現了心律失常，打印結束後，界面會彈出提示框：檢查到心律失常，是否打印節律波形；醫生可以自行選擇；

如果關閉，不會彈出心律失常的提示框。

Cycle print time

週期打印長度

如選擇週期打印模式，需要選擇週期打印的時長，如2小時，2小時後自動退出週期檢查模式；類似於監護裡的週期血壓檢查；

Cycle print duration

週期打印間隔

如選擇週期打印模式，選擇完時長後，需要選擇每隔多長時間進行一次心電波形打印和分析，如10分鐘；

Pre-acquisition

預採樣

如開啟，按下打印鍵時，分析和打印的是之前10秒鐘的數據；

優勢是更於快速做圖，適用於體檢科，因為大部分都是正常心電；

劣勢是可能會剛好漏掉一些疾病信息，如偶發性室早，可能剛好那個波形沒有出現在前10秒，但出現在後10秒了，就會錯過信息捕捉。

打印網格

適用於空白熱敏打印紙或A4紙（連接激光打印時），如開啟，機器將自動在空白紙上打印 1：1的背景網格；如關閉，打印時不會出現背景網格

Paper type

紙張類型

捲紙 Roll paper

一般需要配合紙軸使用，用於紙張固定，避免打印歪斜

摺紙 Z-fold paper

摺紙有黑點，打印時需進行黑點檢測，以便報告是從黑點摺疊處開始打印，方便撕開和保存；

Print mode

打印數據類型

同步數據打印Synchronous Data Printing

每組波形的數據打印起始點是一致的，即按下打印鍵的那一刻數據

實時數據打印Real-time data printing

每組波形都是實時數據打印，每組波形的打印起始點不一致

Record format

記錄模式

省紙 Economic

病人信息、測量數據、分析結果、波形集合在一張類似A4紙大小的熱敏紙上；

自動 Auto

按照機器當前的參數設置，按下打印鍵後，自動完成波形、病人信息、測量數據、分析結果等的全部打印或部分打印。

手動 Manual

按照機器當前的參數設置，按下打印鍵後，開始打印波形，直至再次手動按下打印鍵後，方會停止；此種模式下，只打印波形，沒有病人信息、測量數據和分析結果。

自動觸發 Trigger

在檢查過程中，如果出現心律失常，機器自動觸發打印，把該段心律失常的波形打印出來，隨後屏幕上出現測量結果和分析結果的預覽，醫生可選擇是否打印。

週期 Cycle

在已設置的週期打印間隔和週期打印時間後，每隔規定時間，自動打印波形、病人信息、數據測量和分析結果，直至週期打印時間結束。

上傳 Upload

檢查結束後，報告自動上傳至ECG-1000工作站管理軟件

Printer switch

打印機開關

打印機為“開”，按下屏幕或鍵盤區的打印鍵時，打印機可正常打印；

打印機為“關”，按下屏幕或鍵盤區的打印鍵時，打印機不可使得，同時屏幕上出現提示：打印功能已關閉，如需要，請開啟

ID number

ID編碼方式

手動

手動輸入ID數字

自動

掃描槍

General settings

常用設置

保存機器當前所有的參數設置，可保存多個常用設置；在實際中，主要應用場景：

1） 對於同種採集條件，同種病人類型，醫生覺得機器設置可以一樣時；

2） 一個醫院配了多臺心電，要求每臺心電設置一樣，特別是在與網絡工作站連接的情況下，只需要一個機器上保存一個常用設置，通過導出/導入功能，即可輕鬆完成多臺機器的設置，無需逐臺機器逐個參數進行設置。

Settings import

導入設置

將保存在U盤或SD卡的設置導入至心電圖機內，只需一鍵操作，免除逐步設置每個參數，優化工作流；

Settings export

導出設置

將常用設置保存在心電圖機後，導出至U盤或SD卡，需要時進行導入；
