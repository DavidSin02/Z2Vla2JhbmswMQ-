面試寶典之LeetCode算法題 - 2.兩數相加

2020-11-14
| 科技

題目描述：

給定兩個非空鏈表來表示兩個非負整數，其中的數字是按逆序方式存儲的，並且每個節點只存儲一位數字（0 <= Node.val <= 9）。例如數字123表示為3->2->1。假定除了數字0之外，這兩個數不會以0開頭；鏈表的長度在1到100之間。

將這兩個整數加起來，可以得到它們的和，請用上述鏈表的方式來輸出它們的和。


例子1：

輸入：l1 = [2, 4, 3], l2 = [5, 6, 4]

輸出：[7, 0, 8]

342 + 465 = 807

注意鏈表是逆序存儲的，這兩個數字是342和465，而342 + 465 = 807，所以輸出7->0->8。


例子2：

輸入：l1 = [0], l2 = [0]

輸出：[0]


例子3：

輸入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]

輸出：[8,9,9,9,0,0,0,1]


思路：

鏈表長度可能到100，即可能有100位數，所以不要轉換為數字再相加（會溢出或丟失精度）。

參照例子1中的圖，可以用列豎式的方式計算兩個數字的和，用代碼來模擬這個過程。

由於是逆序存儲的，列表的頭節點存儲的是個位數字，豎式計算也是從個位數開始的。只需將對應位上的2個數字相加，即可得到結果中相應位的數字。如果有進位，計算下一位的時候把進位也算上。

兩個數的位數可能不一樣長，計算到後面如果只有一個數字了，只需把另一個數字看成0來計算即可。


JavaScript代碼實現：

/** * Definition for singly-linked list. * function ListNode(val, next) { *     this.val = (val===undefined ? 0 : val) *     this.next = (next===undefined ? null : next) * } *//** * @param {ListNode} l1 * @param {ListNode} l2 * @return {ListNode} */var addTwoNumbers = function(l1, l2) {    var n1 = l1; // 當前的數字    var n2 = l2;        var result; // 結果鏈表的頭節點    var previous; // 結果鏈表中當前位的前一個節點    var carry = 0; // 進位        while (n1 != null || n2 != null) {        var val1 = n1 != null ? n1.val : 0;        var val2 = n2 != null ? n2.val : 0;        var sum = val1 + val2 + carry; // 相應位置的數字相加，不要忘記加上進位        var val = sum % 10; // 得到對應位上和的數字        carry = sum >= 10 ? 1 : 0; // 更新進位        var current = new ListNode(val);        if (result == null) {            // 鏈表的頭節點            result = current;        }        if (previous != null) {            // 把當前節點加入鏈表            previous.next = current;        }        previous = current;        n1 = n1 != null ? n1.next : null; // 指向下一位的節點        n2 = n2 != null ? n2.next : null;    }        if (carry > 0) {        // 最後可能還有一個進位        previous.next = new ListNode(carry);    }        return result;};
