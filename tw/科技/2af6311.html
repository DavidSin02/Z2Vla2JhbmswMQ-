開始使用 Budgie 吧，一款 Linux 桌面環境

2020-10-29
| 科技

-- Kevin Sonney

使用 Budgie 按需配置你的桌面，這是我們開源工具系列中的第 18 個工具，它將在 2019 年提高你的工作效率。

每年年初似乎都有瘋狂的衝動想提高工作效率。新年的決心，渴望開啟新的一年，當然，“拋棄舊的，擁抱新的”的態度促成了這一切。通常這時的建議嚴重偏向閉源和專有軟件，但事實上並不用這樣。

這是我挑選出的 19 個新的（或者對你而言新的）開源工具中的第 18 個工具來幫助你在 2019 年更有效率。

Budgie 桌面

Linux 中有許多桌面環境。從易於使用並有令人驚歎圖形界面的 GNOME 桌面 （在大多數主要 Linux 發行版上是默認桌面）和 KDE ，到極簡主義的 Openbox ，再到高度可配置的平鋪化的 i3 ，有很多選擇。我要尋找的桌面環境需要速度、不引人注目和乾淨的用戶體驗。當桌面不適合你時，很難會有高效率。

Budgie 桌面 是 Solus Linux 發行版的默認桌面，它在大多數主要 Linux 發行版的附加軟件包中提供。它基於 GNOME，並使用了許多你可能已經在計算機上使用的相同工具和庫。

其默認桌面非常簡約，只有面板和空白桌面。Budgie 包含一個集成的側邊欄（稱為 Raven），通過它可以快速訪問日曆、音頻控件和設置菜單。Raven 還包含一個集成的通知區域，其中包含與 MacOS 類似的統一系統消息顯示。



點擊 Raven 中的齒輪圖標會顯示 Budgie 的控制面板及其配置。由於 Budgie 仍處於開發階段，與 GNOME 或 KDE 相比，它的選項有點少，我希望隨著時間的推移它會有更多的選項。頂部面板選項允許用戶配置頂部面板的排序、位置和內容，這很不錯。



Budgie 的 Welcome 應用（首次登錄時展示）包含安裝其他軟件、面板小程序、截圖和 Flatpack 軟件包的選項。這些小程序有處理網絡、截圖、額外的時鐘和計時器等等。



Budgie 提供乾淨穩定的桌面。它響應迅速，有許多選項，允許你根據需要自定義它。
