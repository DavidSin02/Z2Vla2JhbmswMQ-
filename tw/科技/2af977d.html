只會用微信傳文件？這 9 個在線傳輸神器，什麼設備都能用！

2020-10-29
| 科技

經常傳輸文件已經成為我們生活中的一部分，TNW前段時間曾表示，即使我們面前已經有一個屏幕，我們可能還會開啟另一個設備。擁有多個智能設備，並且有許多協同操作，是我們所處的信息時代其中一個顯著特徵。

但 TNW 同時表示，如果我們長時間對著多個設備，耐心和注意力程度會下降。所以，如果能快速解決不得不面對的多設備操作問題，這能給我們的生活帶來不小的改變。

跨設備傳文件應該怎麼操作？不用數據線，你知道什麼方法最適合你嗎？今天的文章將介紹值得一試的方法。

懶人目錄

  • 蘋果自帶同步工具
  • 從 Windows PC 轉移至 Mac
  • 不分系統，手機和 PC 間互傳
  • 網盤

蘋果自帶同步工具

其實蘋果自帶就有好用的文件傳輸工具，隔空投送和 iCloud。前者不花一點流量，後者自動同步，省事程度滿分。

隔空投送（AirDrop）

隔空投送可以在 Mac 和 iPhone、iPad、iPod touch 之間傳遞文稿、照片、視頻、網站、地圖位置等。

使用前提是確保兩臺設備均已開啟 Wi-Fi 和藍牙，並且彼此之間的距離不超過 9 米（30 英尺）。兩臺設備不需要連接同一個 Wi-Fi，但對設備有要求。

你所使用的設備需要 OS X Yosemite 或更高版本的 Mac（2012 年或更新機型），不包括 2012 年中的 Mac Pro 機型和運行 iOS 7 或更高版本的 iPhone、iPad 或 iPod touch。

如果是在 Mac 電腦間傳遞文件，兩臺電腦必須是以下機型：

• 2008 年末推出的 MacBook Pro 或更新機型，不包括 MacBook Pro（17 英寸，2008 年末）

• 2010 年末推出的 MacBook Air 或更新機型

• 2008 年末推出的 MacBook 或更新機型，不包括白色 MacBook（2008 年末）

• 2009 年初推出的 iMac 或更新機型

• 2010 年中推出的 Mac mini 或更新機型

• 2009 年初推出的 Mac Pro（配備 AirPort Extreme 卡的機型）或 2010 年中推出的 Mac Pro

• iMac Pro（所有機型）

雖然要求看起來很多，但只要你的機型不是過老、系統非陳舊，便可以暢快使用隔空投送。

iCloud

iCloud 能為蘋果所有設備設備上同步照片、通訊錄、日曆。你只需要在 Apple ID 登錄，剩下的同步操作交由系統自動執行。

設置方法：

檢查 iCloud 設置，決定對哪些應用啟動 iCloud 同步。

蘋果建議用戶更新到最新系統，也就是 iOS 12 和 macOS Mojave，iPhone、iPad、iPod touch 的 iWork 套件則是 2.5 或更高版本，Mac 的 Pages 文稿 5.5 或更高版本、Numbers 表格 3.5 或更高版本、Keynote 講演 6.5 或更高版本。

Safari 瀏覽器要求 9.1 或更高版本，Firefox 45 或更高版本，或者 Google Chrome 54 或更高版本。

規定這些，是因為文件在不同系統及版本中會有功能和細節上的不同，這麼做可以防止文件內容丟失和破損。

值得一提的是，iCloud 也可以在 Windows 上使用，同樣可以同步照片、文稿、書籤等。只需你在 Windows 電腦上下載 iCloud 客戶端即可使用。

此外，你也可以登錄 https://www.icloud.com/ 查看你已經保存在雲端的內容。

從 Windows PC 轉移至 Mac

如果你正打算從 Windows PC 轉移至 Mac，Windows 遷移助理這款應用能幫到你。

Windows 遷移助理是蘋果開發的一款文件轉移工具。需要兩臺電腦同處一個 Wi-Fi，並且在兩臺電腦上下載應用。遷移時一旦 Mac 顯示驗證密碼，點擊「繼續」則可以開始遷移過程。

Mac 會先掃描 Windows PC，電子郵件、通訊錄、日曆信息、圖片、書籤、iTunes 內容、系統設置等都可以同步到 Mac 上。再次點擊「繼續」，可以完成遷移過程。

有一點值得注意，Windows 系統最好更新到最新版本，遷移助理可與 Windows 7 及更高版本搭配使用。

不分系統，手機和 PC 間互傳

Windows 用戶、Android 用戶有什麼好用的傳文件大法？

得益於 app 的全平臺性，令文件可以在不同系統、不同設備中流通。我們最常見的就是發郵件給自己以及 QQ、微信。

發郵件的方法已經非常「古老」，適用於依賴郵件這種通訊工具的人群。只要你在其他設備上登錄郵箱，便可以查收自己的這份郵件，並獲取郵件內的文件。

在移動端，QQ 的「我的文件」功能，包括傳文件到我的另一臺設備、面對面快傳。

在電腦端，則直接顯示我的設備，方便向指定設備傳輸文件。

在 QQ 剛一推出傳文件功能的時候，不僅方便了給自己不少沒法跨設備傳文件的小夥伴，還方便了群文件和圖片的管理。

這一功能的便利性已經由千萬人驗證，所以 QQ 至今仍有大批離不開的用戶。

如果文件過大，你可能還記得 QQ 有個文件中轉站功能，不過注意，需要在文件到期前下載下來，以防丟失。

「文件傳輸助手」和「收藏」則是微信的跨設備傳輸工具。

由於微信的用戶群體更多，所以這兩個實用功能我們已經無數次提及。「文件傳輸助手」是微信放在你通訊錄裡的一個隱形的助手。需要時只需從通訊錄中找到它，將文件在對話框中發出，便可在同時登錄的設備上互傳文件。傳輸文件大小不可超過 10M 是它唯一的限制。

如果說文件傳輸助手適合按照時間線搜尋文件的你，那麼「收藏」功能更適合分門別類傳輸文件的用戶。

圖片與視頻/鏈接/文件/音樂/聊天記錄/語音/筆記/位置 可以按分類查找，即使你收藏的東西再多，也可以快速定位找回。

而「收藏」中的「筆記」功能更是一個萬花筒一樣的存在，可以在裡面添加待辦、項目分割線，還可以加入多媒體文件和定位，用來作為項目整理最合適不過了。

收藏內容會存在於你的賬號信息內，且不需要你同時登錄多個設備，只要你在其他設備登錄這個賬號，收藏內容便會出現。

需要注意，收藏功能可容納 2M 以內的文件。

說完最常見的 QQ 和微信，讓我們再來看看還有哪些工具專為跨設備文件而來。

A 君推薦 Airdroid 和 Telegram 這兩款。

Airdroid 本是為 Android 平臺打造的手機管理工具，如今已經發展為一個完善的全平臺服務，可以讓多屏幕操作更精彩。如果你想傳輸文件，用它便可以在 Android、iOS、Windows、macOS 、web 完成，前提只是擁有一個平臺賬號並登錄。

無需數據線，文件、圖片、文檔、音樂、APK 都可以隨心所欲、自由傳輸。解鎖高級用戶，還可以傳輸文件夾。

在電腦上，Airdroid 的網頁版給人的感覺是更好用，它呈現出的狀態類似於電腦桌面，還能幫你查收手機信息和來電。Airdroid 幫你實現了用電腦控制手機、用手機控制電腦，讓設備之間無差別。

Telegram 也是一款即時傳輸工具，類似於 QQ 和微信，但它的傳輸更快，而且單次可傳送 1.5M 大小的文件。

和文件傳輸助手不同的是，Telegram 這款應用會記錄你的聊天記錄。只要在另一臺設備上登錄，你發送的文件都可以回溯。免去了傳輸文件時的多設備同時登錄操作。

而想要在 Telegram 建立一個純粹的文件傳輸工具，只需要成立一個只有自己的頻道即可。

添加方法是點擊 New Channel，添加頻道名稱，並給這個頻道添加 Link 的後綴。名稱不受限，但 Link 後綴會被檢測是否已經被佔用。當然如果你設置這個頻道僅為私用，Telegram 會自動為你生成一個 Link。

需要注意的是 Telegram 需要具備科學上網環境，首次使用需要在移動端註冊賬號才可使用。

網盤

說完了我們上面的文件傳輸方法，下面就不得不說說網盤了。

百度雲 2T 的免費大容量讓不少人入坑，保存大文件、備份資料是很多人常用的功能。它允許批量上傳文件，最多可以 500 個文件同時傳送。傳輸文件最大可以為 4GB。

如果你開通了百度雲的會員，它還可以幫你備份手機裡的視頻、實況照片、歷史版本等，兼顧了多類文件，考慮了我們誤操作的可能。

使用堅果雲的好處是，在電腦本地操作，它的兼容性更強，文件本地化，更易搜尋。

而且在進行多設備數據遷移時，還可以設置應用同步，能給你把 WPS、PDF Expert 等多個應用上的數據同步過來。時光機還可以幫你恢復文件操作過程，讓你的文件更有保障。

如果堅果雲賬號在另一臺設備登錄，也可以完成協同操作，成全了多人的跨地協作。

寫在最後

如果要對上面的幾種跨設備文件傳輸做一個分類，可以按照文件大小、是否藉助第三方工具及是否支持「離線」操作劃分。

QQ、微信自帶的功能不能傳輸過大的文件，但由於使用人群廣泛，成為最普遍使用的一種方法。

AirDrop、iCloud 是蘋果用戶自帶的傳輸工具，快速安全。

AirDriod、Telegram 在傳輸文件大小上有優勢，如果你需要對自己的多臺設備進行傳輸，可以考慮在設備上安裝這兩款應用。前者需要賬戶同時在線，後者支持「離線」操作。

網盤則更適合有大文件傳輸需求的用戶，無論如何，擁有一個網盤，總是有備無患。

這些方法都是我們可能會用到的，一技傍身，以後別再為無線傳輸文件煩惱了！
