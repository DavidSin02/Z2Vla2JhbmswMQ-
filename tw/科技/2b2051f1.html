「學員筆記」LINUX隨堂筆記（一）

| 科技

好的筆記是成績提升的開始，試想一下，如果在你開始複習功課時看到的是連自己都看不懂的筆記，會不會更加沒有學習的激情？

今天就給大家分享一下知了匯智學員乾淨整潔的LINUX隨堂筆記，大家一起學習一起進步!

第1章 基本命令

一. 系統開關機、重啟命令

1.1 shutdown

格式 ： shutdown [選項]

常用命令選項：

-r： 重新啟動系統

-h： 關閉系統

-c： 取消運行shutdown

1.2 init 0 // 關閉系統

1.3 init 6 // 重啟系統

1.4 reboot // 重啟系統

二. 查看類命令

2.1 type

格式：type [命令] // 查看命令類型

實例：

[root@HH 桌面]#type who

who is /usr/bin/who

[root@localhost ~]# type service

service is hashed (/sbin/service)

2.2 pwd

格式：pwd [選項（可為空）]

常用選項

-P // 輸出當前工作目錄的實際物理路徑

-L // 輸出當前工作目錄的鏈接路徑

實例：[root@rhel init.d]#pwd

/etc/init.d // 用戶當前的工作目錄路徑是/etc/init.d

[root@rhel init.d]#pwd -P

/etc/rc.d/init.d

[root@rhel init.d]#pwd –L

/etc/init.d

2.3 cd

格式 cd [選項]

常用選項：

cd ~ // 進入root目錄

cd – // 返回上一次所在目錄

cd . . // 返回上一級目錄

cd . ./ . . // 返回上兩級目錄

2.4 ls

格式 ls [選項] //查看目錄內容

常用命令選項

- l ：以長格顯示（簡寫“ll”）目錄的詳細資料

- a ：顯示所有目錄和文件信息，包括隱藏文件

- A ：類似於“-a”，但不顯示“.”和“..”目錄的信息

- d ：顯示目錄本身的屬性

- h ：以更易讀的字節單位（K、M）等顯示信息

- R ：遞歸顯示內容

- color ：以顏色區分不同文件

實例：

[root@localhost ~]# ls -l

總用量 132

-rw------. 1 root root 2158 3月 9 17:58 anaconda-ks.cfg

drwxr-xr-x 14 root root 4096 3月 10 14:38 cmake-2.8.6

drwxr-xr-x 4 root root 4096 3月 10 15:01 DiscuzPkg

-rw-r--r--. 1 root root 64664 3月 9 17:58 install.log

2.5 which

格式：which [命令] // 查看命令地址

實例：

[root@localhost ~]# which service

/sbin/service

2.6 who

格式： who //顯示已登錄當前計算機用戶的信息

實例：

[root@rhel ~]#who

zhngsan tty2 2017-03-20 02:11

root tty1 2017-03-11 18:00

2.7 du

格式：du [選項]……[目錄名] //統計目錄及文件的空間佔用情況

常用命令選項：

- a ：統計時包括所有的文件，而不僅僅只統計目錄

- h ：以更易讀的字節單位（K、M等）顯示信息

- s ：只統計每個參數所佔用空間總的大小

實例：

[root@localhost ~]# du -ah /root/cmake-2.8.6/

4.0K /root/cmake-2.8.6/Source/cmVersionMacros.h

76M /root/cmake-2.8.6/Source

4.0K /root/cmake-2.8.6/CMakeGraphVizOptions.cmake

32K /root/cmake-2.8.6/Makefile

52K /root/cmake-2.8.6/bootstrap

4.0K /root/cmake-2.8.6/cmake_uninstall.cmake

4.0K /root/cmake-2.8.6/http://CTestCustom.cmake.in

4.0K /root/cmake-2.8.6/Copyright.txt

52K /root/cmake-2.8.6/CMakeCache.txt

170M /root/cmake-2.8.6/

三. 顯示文本文件內容

3.1 cat

格式：cat [文件名] //查看文件內容

實例：

[root@localhost hh]# cat ss

hspowudoabc

skajcwu

jaicboa

3.2 less

格式 ：cat [文件名]| less //可翻頁查看文件所有內容

3.3 more

格式：cat [文件名] | more //查看文件所有內容,不能翻頁

3.4 tail

格式： tail [選項] [文件名] //查看文件內容

常用選項

tail -# file //查看文件內容末#行

tail –f file //實時查看添加到文件裡的內容

head -# file //查看文件內容的前#行

例：

[root@localhost hh]# cat ss

hspowudoabc

skajcwu

jaicboa

[root@localhost hh]# tail -1 ss

jaicboa

四. 目錄和文件操作

4.1 wc

格式 ： wc [命令][文件絕對路徑] //對文件進行統計

常用命令：

wc [絕對路徑] //統計行數、大小、字符數

wc –l [絕對路徑] //統計行數

wc –w [絕對路徑] //統計字符數

實例：

[root@localhost hh]# cat ss

hspowudoabc

skajcwu

jaicboa

[root@localhost hh]# wc ss

4 3 29 ss

4.2 find

格式： find [路徑][選項]

常用命令選項

-name：[文件名] //按照文件名查找文件

-user：[用戶名] //按照文件的用戶所有者來查找文件

-group：[組名] //按照文件的組群所有者來查找文件

-type：[文件類型] //按照文件類型來查找文件

-size： n[ckMG] //查找大小為n的文件，n為數字，c為字節，k代KB，M代表MB，G代表GB

實例：

[root@localhost ~]# find / -name w

/var/lib/yum/yumdb/w

/usr/share/terminfo/w

/usr/bin/w

4.3 mkdir

格式 mkdir [目錄名] //創建目錄

常用命令選項

mkdir dir1 //創建一個叫做 'dir1' 的目錄 mkdir dir1 dir2 //同時創建兩個目錄 mkdir -p /tmp/dir1/dir2 //創建一個目錄樹

4.4 touch

格式 touch [文件名] //創建一個文件

4.5 cp

格式：cp[選項] [源路徑] [目的路徑] //複製源路徑內容到目的路徑內

常用命令選項

-r： 遞歸複製

-p： 連同屬性一起復制

-a： 目錄所有複製

實例：

[root@localhost ~]#cp –r /root/etc /root/ww //複製目錄root下的etc目錄所有內容到root目錄下ww目錄內

[root@localhost ~]#cp –r /root/* . //複製root目錄下的所有文件到當前目錄

4.6 ln

格式：ln –s [目錄或文件][鏈接名] //創建一個指向目錄或文件的軟鏈接,源文件更改，鏈接即失效

ln –l [目錄或文件][鏈接名] //創建一個指向文件的硬鏈接，源文件更改不影響鏈接

實例：

[root@localhost ~]#ln –s /root/media/Centos /root/cdrom //在root目錄下創建指向Centos鏡像目錄的軟連接cdrom

[root@localhost ~]#ln –l /root/media/Centos /root/cdrom //在root目錄下創建指向Centos鏡像目錄的軟連接cdrom

4.7 rm

格式 ： rm [選項] [文件或目錄] //刪除文件或目錄

常用命令選項

-f file1 //強制刪除一個叫做‘file1’的文件

-rf dir1 //強制刪除一個叫做‘dir1’的目錄並同時刪除其內容

-r dir1 //遞歸刪除目錄和目錄裡的內容（需要依次確認）

4.8 mv

格式： mv [源文件或目錄] [目的文件或目錄] //移動當前工作目錄下的文件或目錄

實例：

[root@localhost ~]# ls

a b hh

[root@localhost ~]# mv a b

[root@localhost ~]# ls

hh b

[root@localhost ~]# cd b

[root@localhost b]# ls

a

4.9 grep

格式：grep [選項][查找模式][文件] //在文件中查找並顯示包含指定字符串的行

常用命令選項

-i： 查找忽略大小寫

-v： 反轉查找，輸出與條件不相符的行

查找條件設置

1、要查找的字符串以雙引號括起來；

2、“^……” //表示以……開頭

3、“……$” //表示以……結尾

4、“^$” //表示空行

4.10 gzip、bzip2

格式 ： gzip [選項][文件] //製作、解壓gzip方式壓縮文檔

bzip [選項][文件] //製作、解壓bzip方式壓縮文檔

常用命令選項

-q： 表示高壓比，多在創建時

-d： 用於解開已經壓縮過的文件

實例：

[root@localhost hh]# gzip kk

[root@localhost hh]#ls

kk.gz

4.11 tar歸檔

格式： tar [選項][壓縮包名.tar] 創建壓縮包

常用命令選項

-c ：創建tar的包文件

-x ：解開tar格式的包文件

-f ：使用歸檔文件

-z ：用gzip壓縮

-j ：用bzip2壓縮

實例：

tar -cvf [壓縮包名.tar] file1 //創建一個非壓縮的 tarball

tar -cvf [壓縮包名.tar] file1 file2 dir1 //創建一個包含了 'file1', 'file2' 以及 'dir1'的檔案文件

tar -tf [壓縮包名.tar] //顯示一個包中的內容

tar -xvf [壓縮包名.tar] //釋放一個包

tar -xvf [壓縮包名.tar] -C /tmp //將壓縮包釋放到 /tmp目錄下

tar -cvfj [壓縮包名.tar].bz2 dir1 //創建一個bzip2格式的壓縮包

tar -xvfj [壓縮包名.tar].bz2 //解壓一個bzip2格式的壓縮包

tar -cvfz [壓縮包名.tar].gz dir1 //創建一個gzip格式的壓縮包

tar -xvfz [壓縮包名.tar].gz //解壓一個gzip格式的壓縮包

4.12 chmod

格式： chmod [選項][模式][文件] //更改文件或目錄的模式

chmod [選項][文件]

常用選項

-v： 為每一個文件處理輸出診斷

-R： 遞歸更改文件和目錄

操作對象

u： 用戶所有者

g： 組群所有者

o： 其他用戶

a： 所有用戶

操作符號

+： 添加某個權限

-： 取消某個權限

權限

r： 讀取權限

w： 寫入權限

x： 可執行權限

實例：

chmod u+w hh // 添加用戶所有者對hh文件的寫入權限

4.13 sed

格式：sed [選項] [命令] 文件名 // 非交互式編輯器

常用選項

-n： 取消默認輸入

-e： 多項編輯

-i： 直接修改讀取的檔案內容，不顯示

常用命令

a： 當前行後面添加行

i： 當前行前面添加行

c： 用字符串替代行

s： 用一個字符替代另一個字符

p： 將選擇的數據印出，通常是和sed –n 一起使用的

例：

sed –n “2 s/root/hh/g” passwd //將passwd文件的第二行裡所有root替換成hh

五. 獲取幫助

5.1 man

格式： man [選項（或為空）][命令]

常用選項：

-M [路徑] //指定man手冊頁的搜索路徑

-a //顯示所有的手冊頁

實例：

[root@localhost hh]#man pwd

5.2 help

格式： help [選項（或為空）][命令]

常用選項：

-d //顯示命令簡短的主題描述

-s //顯示命令簡短的語法描述

實例：

[root@localhost hh]#help pwd
