平遙第三年，片單太硬了

2020-10-29
| 科技

10月10日，第三屆平遙國際電影展即將拉開帷幕。

回想兩年前，這個深植於中國內地的國際影展，剛剛帶著西北的煙火氣在全球電影人面前亮相。對故鄉山西飽含無限深情的賈樟柯，給了平遙電影展一個定位：“這將是一個尊重電影、分享文化、尊重影迷、尊重電影人勞動成果的電影展。”

在過去的兩年裡，這個目標的完成度可以說是有目共睹的。

創始人賈樟柯與影展藝術總監馬可·穆勒

對觀眾來說，這裡可以享受到國際電影節帶來的視覺饕餮盛宴；而對一些青年電影人來說，這是一次向大眾介紹自己的寶貴機會。每年10月的平遙，正成為越來越多國內電影人和影迷的必選行程。

1

發出來自中國的聲音

說平遙，就不能不說賈樟柯。這個國際影展的從無到有，離不開這位國際名導的促成。

賈樟柯曾說，自己人生中的重要時刻，都是各式各樣的電影節賦予的。憑藉《小武》，賈樟柯在柏林電影節一鳴驚人；憑藉《三峽好人》，他在威尼斯擒獲金獅；憑藉《天註定》，他拿下戛納電影節最佳編劇獎。其作品更曾五次入圍戛納主競賽單元。


如今舉辦平遙國際電影展，賈樟柯為最初的理想找到了更廣闊的場景,並讓世界電影聽見了更多來自中國的聲音。


影展獲得了導演李安的授權，將兩個競賽單元分別命名為“臥虎”和“藏龍”。其中，“臥虎”單元致力發掘優秀的新人新作，在全球範圍選擇新導演的處女作或第二部電影；而“藏龍”單元關注華語新人，放映華語新導演的新銳作品。

過去兩年，從平遙走出來的獲獎電影如《騎士》《大世界》《嘉年華》《過昭關》和《過春天》等，都成為備受影迷群體推崇的口碑之作。

78歲的楊太義憑《過昭關》獲頒費穆榮譽-最佳男演員

新人輩出的華語電影界，正在平遙發出自己的聲音。這樣的聲音，我們不希望錯過。

2

“大家和大家”

第三屆平遙國際電影展官方海報“電影小星球”，寓意電影是一個獨特神奇的星球，所有電影創作者、電影觀眾都是這個星球上的居民。

“這個星球裡有遊蕩的年輕人，有閃閃發光的創意。”賈樟柯說，“因為十天的電影展，平遙也變成了一個星球。”

而本屆電影展主題“大家和大家”，則是電影展的另一重含義。如果你曾經來過平遙影展，一定明白這其中的意味。

從第一年的“平遙元年”，到第二年的“從電影到市集”，每年平遙影展的主題都別有趣味。兩千七百年的平遙古城，與藝術家族的幼子不期而遇，彷彿一場火星撞地球的奇妙體驗。

平遙影展根植於一個北方古城，“大格局、小身段”是它的特色之一。影展主要集中在一個內部空間，由平遙柴油機廠舊廠房改造的電影宮，很有復古工業感。

電影宮內景

各個影廳幾乎只有上下臺階的路程，幾天下來，不但能認識一幫影迷朋友，更可以邂逅不少電影主創和業界大佬。和他們在同一影廳看電影。



可容納近500人的“小城之春”影廳

“大家”，是引領一個行業前行的先行者；“大家”，則是讓電影走進千家萬戶的傳播者。“大家與大家”，缺一不可。而他們的相遇，正是在這座神奇的北方古城。

3

這個片單有點硬

本屆平遙影展的首批片單公佈之後，吸引了不少關注，共發佈來自26個國家和地區的52部影片信息。影展開幕前，還將公佈最後一批官方單元入圍片單。

在官方單元的49部展映影片中，28部為全球首映，36部為亞洲首映，中國首映率則達到100%。在這份目前的展映片單裡，還能看到許多今年歐洲三大的口碑之作。

其中“臥虎”單元的入圍影片，包括今年戛納“金攝影機獎”得主《我們的母親》，在威尼斯首映的、小田切讓自編自導的首部長片《一個船伕的故事》；新加坡導演陳哲藝，也帶著多倫多歸來的《熱帶雨》亮相平遙。

《一個船伕的故事》

而關注華語新人新作的“藏龍”單元，更有演員祖峰執導、此前入圍戛納的《六慾天》，入圍莫斯科、慕尼黑電影節的《蕃薯澆米》，都將在平遙進行亞洲首映。

《蕃薯澆米》

此外，由李仁港執導的《攀登者》將亮相“影迷嘉年華”。本屆影展青年導師謝飛的《黑駿馬》2K修復版、張藝謀《紅高粱》4K修復版將在電影展進行展映。

其中值得關注的幾部電影，這就為您做個簡單的介紹，準備去平遙的朋友一定別錯過這份片單，這些很可能是今明兩年的必看片哦~



《我們的母親》

Nuestras Madres

入圍單元：臥虎

導演 / 編劇: 塞薩爾·迪亞茲

主演:Armando Espitia / Emma Dib / Aurelia Caal

製片國家/地區: 危地馬拉

電影獲得今年戛納的金攝影機獎（最佳處女作）。故事以2013年的危地馬拉內戰期間的大屠殺為背景，法醫中心的年輕人類學家伊涅斯托的工作，是確認在戰爭中失蹤的受害者的身份。


在跟一位老婦人的談話中，伊涅斯托意外發現了一條引向他父親行蹤的線索。而後者是在內戰中失蹤的一名游擊隊員，從而引起了一系列關於身份、家庭、群體的問題。

女導演塞薩爾·迪亞茲長期在法國學習電影，曾拍了很多紀錄短片，這種強紀錄的平實質感也表現在電影中。


《一個船伕的故事》

ある船頭の話

入圍單元：臥虎

導演 / 編劇: 小田切讓

主演: 柄本明 / 村上虹郎 / 川島鈴遙 / 伊原剛志

類型: 劇情

製片國家/地區: 日本

小田切讓的導演處女作，這個名頭已經讓《一個船伕的故事》在今年威尼斯廣受關注。

《一個船伕的故事》主要講述在日趨商業化的世界裡，一些傳統古老的事物逐漸消失。主人公是一位載人渡河的船伕，在他平淡的生活中總會遇到各式各樣的人，帶來各種故事。一部探討人性的典型日式影片。

電影的陣容也讓不少人感嘆小田切讓的圈內人緣。參演電影的卡司還包括村上虹郎、蒼井優、淺野忠信、永瀨正敏、橋爪工等眾演員，而這些人或先或後都和小田切有過合作。攝影為杜可風，配樂則是日本音樂大師細野晴臣。

一直被譽為“圈中宅男”的小田切讓，此前在接受採訪時的一段話，直接在影視圈裡投放了一顆炸彈：“我之前的健康檢查結果並不好，誇張地說，我要重新考慮自己剩下的時間了，果然還是很想拍電影。”

儘管讓叔語氣平和，表示並不很嚴重，但不管是粉絲，還是相關影視人，對此還是震驚了一番，紛紛擔憂地表示，“讓叔，請保重身體啊！”

《熱帶雨》

入圍單元：臥虎

導演 / 編劇: 陳哲藝

主演: 許家樂 / 楊雁雁

類型: 劇情

製片國家/地區: 新加坡 / 中國臺灣

在今年的多倫多電影節上，新加坡導演陳哲藝執導的最新電影《熱帶雨》，是電影節“站臺單元”中唯一入圍的華語片。雖然沒有成為最後贏家，但依然得到了場刊3分以上的好成績，妥妥的年度華語佳作。

陳哲藝在2013年的金馬獎上，以處女作《爸媽不在家》一舉拿下最佳影片，還在戛納電影節斬獲了金攝影機獎。以這樣的起點再度出發，《熱帶雨》天然吸引了無數注意力。

《熱帶雨》的故事依然發生在新加坡。正值雨季，身為中學國文老師的女主人公，生活上也在經歷“雨水”沖刷：與丈夫因為無子關係愈發疏冷、照顧癱瘓在床的公公、學生對國文課的不屑一顧、以及她和一位男學生的曖昧關係……

所有這些，隨著雨季的到來，讓女主人公的心裡也越發潮溼。雖然線索很多，但整部電影並不鬆散顯得渙散，為主人公心事的展開提供了更多側面。觀眾會看到的是整個華人世界的類似問題，不禁回味自己身邊種種。

《蕃薯澆米》

入圍單元：藏龍

導演: 葉謙

主演: 歸亞蕾 / 楊貴媚 / 班鐵翔

類型: 劇情

製片國家/地區: 中國大陸

《蕃薯澆米》去年在平遙拿下發展中電影計劃的“無限自在”榮譽，可謂土生土長，今年重回平遙，儼然進入了收獲的季節。

電影由歸亞蕾和楊貴媚主演，寡婦林秀妹撫養兩兒成家後，發現自己變得“無關緊要”；而小她十歲的唯一老姐妹青娥突然離世。給了她巨大觸動。已過七旬的她想要再次證明自己價值與存在感。

該片使用全閩南語拍攝，展示泉州特有的地域文化和風土人情。片名“蕃薯澆米”指的是是閩南百姓吃的地瓜稀飯，寓意“生命的真諦就藏在平凡生活之中”。



《追凶十九年》

入圍單元：藏龍

導演: 徐翔雲

主演: 王瀧正 / 宋寧峰 / 黃璐 / 國永振

類型: 劇情 / 懸疑 / 犯罪

製片國家/地區: 中國大陸

一部硬核犯罪動作片，張猛監製，導演徐翔雲此前曾經是爆款犯罪網劇《法醫秦明》《十宗罪》的編劇，這是他的導演處女作。

1999年起的數年內，一向安寧平靜的弘州市先後發生多起針對年輕女性的強姦命案，受害者的皮膚均遭凶手割下奇異的血雛菊形狀。警察劉一波和搭檔何晨奉命偵查，這場鬥智鬥勇的追凶之旅，一追就是十九年。

一場十九年的追凶之旅，將會雜糅怎樣複雜的情緒和仇恨，著實令人期待。除了文藝片女神黃璐，在網劇領域樹立了良好口碑的王瀧正、宋寧峰，表現也相當值得期待。

《六慾天》

入圍單元：藏龍

導演: 祖峰

主演: 祖峰 / 黃璐 / 陳明昊 / 田雨

類型: 劇情 / 犯罪

製片國家/地區: 中國大陸


今年入圍戛納一種關注單元的《六慾天》，是演員祖峰的導演處女作。劇本原名為《熱》，祖峰看到之後深為打動決定自導自演。故事也同樣為犯罪題材，關於一次警察查案背後的背叛與救贖。



因女友自殺而患上抑鬱症的警察阿斌（祖峰飾），在查案時與同樣患上抑鬱症的李雪（黃璐飾）相識，彼此憐憫，漸漸親近。當在城市漂泊的少女婷婷（張倩如飾）也和阿斌產生說不清的牽絆。 他們期待在這個世界找到光明。

該片作為聚焦抑鬱症的電影，是現代都市快節奏下人所經歷的環境和內心壓力變化。人與人之間的情感糾葛，在追尋案件真相的過程中映射個體與社會的關係。

《灼人祕密》

入圍單元：首映

導演: 趙德胤

主演: 吳可熙 / 宋芸樺 / 夏于喬

類型: 劇情

製片國家/地區: 中國臺灣 / 德國 / 馬來西亞

曾經執導《再見瓦城》的趙德胤，是臺灣較受關注的新生代導演憑該片於今年首度入圍戛納的電影；而女主角吳可熙已經跟她合作了4部影片，此前還主演過大熱的《血觀音》，她和趙德胤聯名作為本片的編劇。

這是一個鄉下女孩到城市追夢的故事。女孩妮娜（吳可熙飾）為了追求演員夢獨自北上，雖如願獲得電影《諜戀》演出，卻因重重行業怪象幾度瀕臨崩潰。


當她因家中變故不得不回到家鄉，她遇上昔日好友Kiki（宋芸樺飾），發生了一連串奇異事件，而這似乎都和當初與她競爭角色後突然消失的女孩有關。


電影據說是以韋恩斯坦案為原型。相比之前的趙德胤，這部明顯更戲劇化了，風格也偏心理驚悚片，其中的長鏡頭使用頗為驚豔，對於娛樂圈的許多呈現諷刺而有力量。

《我失去了身體》

I Lost My Body

入圍單元：影展之最

導演: 傑赫米·克拉潘

主演: 維克圖瓦爾·杜·布瓦 / 帕特里克·德阿薩姆曹

類型: 劇情 / 動畫

製片國家/地區: 法國


今年戛納影評人週單元大獎得主，這是戛納推出全球新銳導演的長片處女作或第二部作品的一個單元。這部傑赫米·克拉潘導演的長片處女作，被許多影評人評為今年“戛納最佳”。

電影呈現出非常奇瑰的想象力。影片分兩條線索，一條線頗具cult片氣質，講述一隻斷手從警察局或者醫院的冰箱裡逃脫，一路經歷千難萬險穿越半個巴黎、回到主人身邊。

另一條線則是主人的年少回憶、父母車禍去世後，自己在巴黎的生存掙扎和美好而痛苦的初戀故事。

影片的畫風簡約卻有氣質，音軌和配樂非常出彩，尤其是很多細節的想象力令人驚歎，導演被認為有著令人驚歎的才華。那隻遊蕩的動畫手，情感之豐富真實過大部分真人電影。



《大西洋》

入圍單元：影展之最

導演 / 編劇: 瑪緹·迪歐普

類型: 劇情

製片國家/地區: 塞內加爾 / 法國 / 比利時


塞內加爾裔法國導演瑪緹·迪歐普電影《大西洋》，在今年戛納拿下象徵二獎的評委會大獎，算得上爆冷了，畢竟這不是一部廣受關注的作品。但場刊2.8的高分，已經足夠說明質量。


該片翻拍自同名短片，講述的是一個非洲非法移民的故事，有趣的是，電影裡使用了極為奇妙的奇幻設定，非洲風情的神祕主義氣息非常迷人，甚至被拿來對比《幸福的拉扎羅》。

雖然被認為非洲的元素有些過剩，但電影本身對於階級和社會正義的探討有著非常有趣的視角。女主角的情感線索處理得也極為細膩和剋制，足夠深刻，卻也足夠好看。




《氣球》

入圍單元：影展之最

導演：萬瑪才旦

主演：索朗旺姆 / 金巴 / 楊秀措

類型：劇情

製片國家/地區：中國大陸

萬瑪才旦的《氣球》入圍了今年威尼斯電影節的一種關注單元，電影被認為像是以往的萬瑪才旦和《撞死了一隻羊》的一次融合。

比較《撞死了一隻羊》，《氣球》更具有故事性，主人公達傑一家因一隻避孕套，捲入了一系列尷尬而又難以抉擇的事件當中。通俗易懂的同時又不失輪迴和夢境等超現實筆觸，讓整個故事都顯得更有靈性。

據說在威尼斯，電影讓不少女性觀眾都流淚了，看來故事倒是頗具普世性。萬瑪才旦的作品，當然期待。


以上這份片單，雖然部部都優中選優，還是不可能把今年平遙的好作品說完。首批公佈的這些片單，幾乎每一部的介紹都讓人心癢癢的。

包括今年的 “回顧”單元，以“印度新電影”為主題推出的十部印度片，喜歡的觀眾也絕對不容錯過。
