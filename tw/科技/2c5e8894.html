C++知識點 32：左移運算符重載

2020-11-14
| 科技

#define _CRT_SECURE_NO_WARNINGS
#include<iostream>
using namespace std;
class Person
{
friend ostream& operator<< (ostream& cout, Person& p); //全局函數友元化，使函數可以訪問私有屬性
public:
Person() {};
Person(int a, int b) :m_A(a), m_B(b){}
private:
int m_A;
int m_B;
};
/*
問題：輸出對象的元素，只能 cout << p1.m_A << p1.m_B << endl;
需求：直接輸出 cout << p1 << endl; (編譯器不認這種運算)
解決：使用<<運算符重載
*/

// 使用全局函數實現
//說明： << 是一個雙目運算符，兩邊都是類，使用全局函數需要傳2個參數
cout屬於系統標準輸出類 ostream，系統類屬性要求唯一，因此必須傳本體，即 ostream&
p 傳 Person（先拷貝，在使用）或者 Person& （直接傳本體）都可以，但使用 Person& 效率更高
ostream& operator<<(ostream& cout, Person& p)
{
cout << "m_A = " << p.m_A << " m_B = " << p.m_B; //屬性私有化，不能訪問，因此設置成友元函數
return cout; // 與傳參相同，返回值也要是本體
}
int main()
{
Person p1(10, 20);
cout << p1 << endl;
}
// 說明：採用成員函數實現
// 寫法： Person& operator<< (ostream& cout); 使用時候寫法： p1 << cout
// 不符合使用習慣，因此只能採用全局函數實現
