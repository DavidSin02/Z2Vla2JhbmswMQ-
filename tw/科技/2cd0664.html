圖表詳解乾式真空泵性能特點

2020-10-29
| 科技

無油乾式機械真空泵 ( 以下簡稱乾式機械泵 ) 是指泵能從大氣壓力下開始抽氣，又能將被抽氣體直接排到大氣中去，泵腔內無油或其他工作介質，而且泵的極限壓力與油封式機械真空泵同等量級或者接近的機械真空泵。下面臺冠真空泵技術團隊工程師們，給大家分享詳解乾式真空泵性能特點參考表.希望能夠給大家帶來幫助.

乾式機械真空泵還分為接觸型及非接觸型。

1.接觸型的乾式泵有葉片式、凸輪式、往復活塞式、膜片式等，這類泵的速度較低，適用於小容量高壓縮比 ( 單級壓縮比 ) 。

2.非接觸型的乾式泵有羅茨型、爪型、螺桿型、渦旋型等，其速度較高，適用大容量，低壓縮比 ( 指單級壓縮比 ) 。不同類型的乾式泵具有各自的特點。
