| 科技

心血來潮在知乎裡面搜索一下 Excel，想學習一些高點贊文章的寫作方法。



看到這些標題，完了，一下子勾起下載和收藏的慾望啦！


怎麼樣把所有高點讚的文章全部都抓下來呢？


開始的時候我想的是用 Python。


後來想一想，用 Power query 好像也可以實現，於是就做出了下面這個效果。



在表格裡面輸入搜索詞，然後右鍵刷新，就可以得到搜索結果。



你能明白我一定要抓到表格裡嗎？


因為 Excel 裡可以根據「點贊數量」直接排序啊！


那種感覺就跟排隊一樣，到哪裡排隊，我都是第 1 個，挑最好的！



好了，閒話少說，我們來看一看這個表格是怎麼做出來的。


大致可以分為 4 個步驟：

❶ 獲取 JSON 數據連接；

❷ Power query 處理數據；

❸ 配置搜索地址；

❹ 添加超鏈接。


01

操作步驟


❶ 獲取 JSON 數據連接


平常在瀏覽網頁的時候，是一個單純的網頁地址。



而網頁裡所看到的數據，實際上也有一個單獨的數據鏈接，這個可以在瀏覽器裡面查找到。



我們需要的數據鏈接，對應的通常是 JSON 格式的數據，就像下面這樣。



查找的方法，需要進入到開發者模式，然後查看數據的 Network 變化，找到 xhr 類型的鏈接，其中一個就是數據的傳輸連接。



把這個鏈接複製下來，這就是 Power query 要抓取數據的鏈接。


❷ Power query 處理


你可能不知道，Power Query 除了可以抓取 Excel 當中的數據之外，


還可以抓取 SQL、Access 等多個類型的數據：



網站數據也是其中一個：



把前面我們獲取的鏈接，粘貼到 PQ 裡面，鏈接就可以抓取數據了。



然後得到的是網頁的數據格式，怎麼把具體的文章數據獲取到呢？


Power Query 強大的地方就在於，它可以自動識別 json 的數據格式，並解析提取具體的內容。


整個過程，我們不需要做任何的操作，只是鼠標點點就可以完成。



這個時候我們獲得的數據，會有一些不需要的多餘的數據。


比如說：thumbnail_info（縮略圖信息），relationship，question，id.1 等等。



把它們刪掉，只保留可需要的文章的標題、作者、超鏈接等等就可以了。



數據處理完成後，在開始選卡，點擊「關閉並上載」，就完成了數據的抓取，非常簡單。



❸ 配置搜索地址


不過這個時候，我們所抓取到的數據是固定的，沒辦法根據我們輸入的關鍵詞來更新。


這是因為數據超鏈接當中所包含的搜索詞沒有更新。



所以這一步呢，我們需要配置一下這個數據鏈接，實現根據搜索詞動態更新。


在表格裡面新創建一個數據，然後加載到 Power query 裡面。



再獲取這個搜索詞，以變量的形式放在搜索地址裡面，就完成了搜索地址的配置。



修改後的地址代碼如下：


  getdata = (page)=>  let      keywords = 搜索詞[ 搜索詞]{0},      源 = Json.Document(Web.Contents("https://www.zhihu.com/api/v4/search_v3?t=general&q="& keywords & "&correction=1&offset="& Text.From(page*20) &"&limit=20&random=" & Text.From(Number.Random()))),      data = 源[data],      jsondata = Table.FromList(data, Splitter.SplitByNothing(), null, null, ExtraValues.Error)  in      jsondata,  轉換為表 = Table.Combine(List.Transform({1..10}, getdata)),

❹ 添加超鏈接


到這一步所有的數據都已經處理完成了，但是如果想要查看原始的知乎頁面，需要複製這個超鏈接，然後在瀏覽器裡打開。



每次要點擊好幾次鼠標比較麻煩；


這裡我們藉助 HYPERLINK 這一個函數，生成一個可以點擊的超鏈接，這樣訪問起來就簡單很多了。



❺ 最後效果


最後的效果就是：



❶ 輸入搜索詞；

❷ 點擊右鍵刷新；

❸ 找點贊最高的；

❹ 點擊【點擊查看】，享受插隊的感覺！



02

總結


知道在表格裡面搜索的好處了嗎？


❶ 按照「點贊數」排序，「評論數」排序；

❷ 看過的文章，可以加一列寫備註；

❸ 可以篩選自己喜歡的「作者」等等。


明白為什麼，精英都是 Excel 控了吧？


現在大部分表格的使用者，還是把 Excel 當做一個報表工具，畫畫表格、寫寫公式而已。


請你記住下面幾個 Excel 新功能，這些功能已經讓 Excel 成長為了，一個強大的數據統計、數據分析軟件，不再是你印象中的報表而已。


❶ Power query：數據整理、清洗工具，搭載M強大的M語言，可以實現多表合併，也是本文的主要技術。


❷ Power Pivot：數據統計工具，可以自定義統計方法，實現透視表多字段計算、自定義DAX數據計算方式。


❸ Power BI：強大、易用的可視化工具，實現交互式數呈現，是企業商務數據報告優質解決方案。
