七款常見手機輸入法測評

2020-11-14
| 科技

手機輸入法的好用與否可能有時候會對我們使用手機帶來不同的體驗。

今天想給大家來一個七款常見手機輸入法的簡短測評，希望可以幫助朋友找到屬於自己的輸入法。

百度輸入法

鼎鼎大名的百毒開發出來的輸入法

分類：全拼輸入法/雙拼輸入法/手寫輸入法/筆劃輸入法

優點：

  1. 詞庫較為豐富

  2. 皮膚自定義性高，預製皮膚多

  3. 聯想方面做得還可以

  4. 彈窗算是國產比較好的

缺點：

  1. 安裝包過於臃腫，有很多多餘的功能

  2. 有較多多的廣告

  3. 糾錯不是很好

  4. 動畫效果有點生硬

  5. 請求了過多多餘的權限

訊飛輸入法

這款輸入法應該是國產輸入法裡面評價最好的吧，老羅的發佈會火了一款輸入法。

分類：全拼輸入法/雙拼輸入法/手寫輸入法/筆劃輸入法

優點：

  1. 語音輸入非常好用，現在更是加如了多種方言。

  2. 鍵盤手寫

  3. 表情包非常豐富

  4. 可以更改各家鍵盤的佈局

缺點：

  1. 主頁不美觀

  2. 滑動輸入木有了

  3. 糾錯智能性一般

谷歌拼音輸入法

谷歌大法就是好，這句話也不是沒有道理。相信很多人都用過這款輸入法，簡約不臃腫。

優點：

  1. 純淨，無廣告

  2. 滑動輸入功能強大

  3. 外觀簡潔

  4. 糾錯功能強大

  5. 較為全面的支持了多種雙拼方案

  6. 手寫識別較強

缺點：

  1. 國內無雲詞庫

  2. 國內無語音輸入

  3. emoji和顏文字不支持自定義，無剪切板

  4. 預製皮膚較少

QQ輸入法

分類：全拼輸入法/雙拼輸入法/筆劃輸入法/五筆輸入法/手寫輸入法

優點：

  1. 浮空鍵盤

  2. 皮膚非常全面，而且預製皮膚多

  3. 符號部分分類很好

  4. 滑動鍵盤移動光標的功能不錯

  5. 雲聯想也可以

缺點：

  1. 糾錯一般

  2. 依然不支持滑動

  3. 安裝包還是大了點

  4. 符號界面存在一定問題，會導致有的符號變形

手心輸入法

一款不錯的國產輸入法

分類：全拼輸入法/雙拼輸入法/手寫輸入法/筆畫輸入法/五筆輸入法

優點：

  1. 有較強的皮膚自定義功能

  2. 純淨無廣告

  3. 糾錯功能較為強大

  4. 雲同步詞庫

  5. 剪貼板功能較為強大

缺點：

  1. 不支持按鍵全時彈窗

  2. 不支持單手模式

  3. 動畫效果略微生硬

搜狗輸入法

可能說是用戶最多的手機輸入法吧

分類：全拼輸入法/雙拼輸入法/手寫輸入法/筆劃輸入法

優點：

  1. 詞庫功能非常強大

  2. 擁有較為全面的皮膚

  3. 手感做的還算不錯

  4. 擁有較好的語音輸入功能

缺點：

  1. 過於臃腫，安裝包體積過大

  2. 有廣告

  3. 請求了過多不合理的權限

  4. 較為容易卡頓

必應輸入法

微軟開發的輸入法

分類：全拼輸入法/雙拼輸入法

優點：

  1. 滑動輸入較為強大

  2. 糾錯功能較為強大

  3. 有較強的光標移動控制器功能

  4. 純淨無廣告

缺點：

  1. 容易卡頓

  2. 鍵盤高度不支持自定義

  3. 不包含手寫和筆劃輸入

  4. 鍵盤可自定義性極差

  5. 不支持單手模式
