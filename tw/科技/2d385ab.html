Python編程快速上手讓繁瑣工作自動化-生成隨機的測驗試卷文件

2020-10-29
| 科技

需求描述

假如你是一位地理老師，班上有 35 名學生，你希望進行全國的一個小測驗。不妙的是，班裡有幾個壞蛋，你無法確信學生不會作弊。你希望隨機調整問題的次序，這樣每份試卷都是獨一無二的，這讓任何人都不能從其他人那裡抄襲答案。當然，手工完成這件事又費時又無聊。好在，你懂一些 Python。

需求分析

程序用到了文件的操作：打開、寫入、關閉，和隨機數模塊以及隨機打亂的方法。

  1. 首先要有一個字典，包含所有省份和對應的省會。
  2. 創建 35 份不同的測驗試卷。
  3. 為每份試卷創建所有包含所有省份的多重選擇題，次序隨機。
  4. 為每個問題提供一個正確答案和 3 個隨機的錯誤答案，次序隨機。
  5. 將測驗試卷寫到 35 個文本文件中。
  6. 將答案寫到 35 個文本文件中。

代碼展示

import random#創建35份試卷，問下列省份的省會是哪裡，每份試卷的問題的順序都不一樣，防止抄襲。#包含省份和省會的字典capitals = {'湖南':'長沙', '湖北':'武漢', '廣東':'廣州', '廣西':'南寧', '河北':'石家莊', '河南':'鄭州', '山東':'濟南', '山西':'太原', '江蘇':'南京', '浙江':'杭州', '江西':'南昌', '黑龍江':'哈爾濱', '新疆':'烏魯木齊', '內蒙古':'呼和浩特', '雲南':'昆明', '貴州':'貴陽', '福建':'福州', '吉林':'長春', '安徽':'合肥', '四川':'成都', '西藏':'拉薩', '寧夏':'銀川', '遼寧':'長春', '青海':'西寧', '甘肅':'蘭州', '陝西':'西安', '臺灣':'臺北', '北京':'北京', '上海':'上海', '天津':'天津', '重慶':'重慶', '香港':'香港', '澳門':'澳門'}#創建35份試卷,每次循環就會生成一份試卷。for quizNum in range(35): # 創建問卷文件和答案文件 quizFile = open('試卷%s.txt' % (quizNum + 1), 'w', encoding='utf-8') answerFile = open('答案%s.txt' % (quizNum + 1), 'w') #寫入試卷的頭部 quizFile.write('姓名:\t\t\t\t日期:\n\n') quizFile.write('\n\n') #隨機打亂省份的順序,使每份試卷的問題順序都不一樣 allProvinces = list(capitals.keys()) random.shuffle(allProvinces) #開始循環設置答案選項，多少個省份就有多少道問題。 for questionNum in range(len(capitals)): #獲取正確答案和其他三個錯誤答案 #這裡的正確答案就是此次循環到的省份對應的省會。 correctAnswer = capitals[allProvinces[questionNum]] #這裡的錯誤答案就是隨機獲取除此次循環對應省份的省會之外的3個省會即可。 allCapitalList = list(capitals.values()) #刪除本次循環省份的省會 del allCapitalList[allCapitalList.index(correctAnswer)] #在已經刪除掉正確省會的列表中隨機選取三個省會 wrongAnswers = random.sample(allCapitalList, 3) #將錯誤答案和正確答案的列表相加獲得本次循環（本題）的4個回答選項的一個列表。 answerOptions = wrongAnswers + [correctAnswer] #隨機打亂生成的回答選項的順序,否則正確答案都是第四個選項（D） random.shuffle(answerOptions) #寫入問題到試卷文件 quizFile.write('{}. {}的省會是哪裡？\n'.format(questionNum + 1, allProvinces[questionNum])) #開始寫入每題的四個選項,每題循環4次 for i in range(4): quizFile.write(' {}. {}\n'.format('ABCD'[i], answerOptions[i])) quizFile.write('\n') # 將正確回答寫入答案文件 answerFile.write('{}. {}\n'.format(questionNum + 1, 'ABCD'[answerOptions.index(correctAnswer)])) #設置為每個試卷的問題和答案就可以關閉這個試卷文件和答案文件 quizFile.close() answerFile.close()
