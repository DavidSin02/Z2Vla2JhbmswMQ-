2020-10-29
| 科技

之前我分享過用JavaScript來寫腳本實現電腦自動控制手機的功能。但是很多老鐵只會易語言、java、python等其他語言怎麼辦？其實很簡單，用 Total Control 提供的REST API 就可以啦！Total Control 的REST API 支持任意腳本語言。今天我又來說說，怎樣用易語言實現自動打開、關閉手機上的軟件。易語言實現自動安裝軟件（installApp）

http://IP:8090/TotalControl/v1/devices/:device/apps?token=:token&file_name=:file_name
