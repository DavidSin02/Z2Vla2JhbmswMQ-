作為EMUI10的親兄弟，誰說Magic UI 3.0沒有遊戲小窗口？

2020-10-29
| 科技

榮耀V30 Pro升級Magic UI 3.0以後，真真體驗到了智慧生活的便捷，比如多屏協同，全屏滑動翻譯等功能，大大方便了小編的生活，但Magic UI 3.0還有些隱藏的功能。

雖說官方沒有宣傳，但並不代表手機沒有，比如說，升級到EMUI10或者Magic UI 3.0的用戶，都應該用上了遊戲空間，對於它增加的屏蔽指關節，以及優化外設連接，都是明面上的功能。
很多喜歡手遊的用戶，都希望有聊天窗口能懸浮在遊戲界面的功能，這樣即能不錯過遊戲，又能不錯過信息。

但不可否認，很多已經使用EMUI10或者Magic UI 3.0系統的用戶，到現在的都沒發現有這個功能，其實，這個功能已經添加，只是你沒發現而已。
假如現在你玩遊戲，並且沒有開啟防打擾，那麼QQ消息彈出來，就會顯示上圖這種圖像，這裡其實末尾多了一個像窗口的東西。這個時候，你點擊一下，QQ消息窗口就會懸浮在遊戲界面，如下圖顯示一般：

當然，開始彈出的QQ消息大概佔據了遊戲大半個屏幕，這樣肯定會有點大是不是，沒關係，EMUI10以及Magic UI 3.0系統，也給我們提供了壓縮窗口變小的功能。比如，你窗口在左邊，那麼按住窗口右側邊緣，推向左側即可縮小窗口。

你也許不想把聊天窗口放在邊緣，想按這個大小居中顯示，那麼調整好大小之後，按住窗口中心，移動即可，非常方便。

以上這些其實都不是最重要的功能，畢竟，你聊天窗口懸浮，是用來聊天的，必定會輸入文字，所以，當你點擊輸入框時，也會如下圖所示，彈出輸入法：

需求更多的花粉，肯定還會關注這樣懸浮後，如果關閉窗口，或者我想跳轉到聊天界面，不想懸浮聊天了。

很簡單，懸浮的窗口，不管是QQ，亦或是微信，右上角都有兩個按鈕，一個正方形的用於跳轉到聊天界面，一個用於關閉懸浮窗口。

雖然說，官方從來沒有宣傳過這種懸浮窗口，但功能已經應用到EMUI10或者Magic UI 3.0，但有人要說連這都沒有，恐怕又要被啪啪打臉了，畢竟華為隱藏的技術太多，稍不注意，可能掉入陷阱哦！
