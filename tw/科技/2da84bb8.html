紙箱廠心態要崩！停機潮提前

2020-11-14
| 科技

理文提前發佈春節停機計劃，

2020年第一波漲價潮預定？

三級廠心態要崩！

//

停機臺數多，發佈時間早，理文CEO李文斌簽發2020春節停機計劃

//

11月5日，理文造紙CEO李文斌簽發了2020年春節理文集團停機計劃表：從2020年1月12日起到1月31日止，在春節前後，理文國內外六大生產基地共有14臺紙機將錯時停機，停機時間在6-20天不等（詳情請點擊：【頭條】理文六大基地14臺紙機陸續停機，紙價再漲100元！幾乎全國紙廠都在漲）。

理文停機計劃會不會發布太早了？

有業內人士提出疑問，離2019年結束還有將近兩個月，理文在這時候公佈春節停機計劃，會不會太早了？

對比2019年的停機計劃，理文造紙今年發佈停機計劃的時間很有意思，2018年理文公佈停機計劃的時間是11月29日（玖龍是11月18日），即使把今年春節提前的時間算上，理文此舉也顯得過於著急。

另一方面，今年停機臺數也比去年多，2020年停機臺數高達14臺紙機，其中單廣東理文就有5臺紙機停機：一臺停機長達20天，三臺停機15天，只有一臺停機較短，停機6天。

//

理文如此早、大規模的停機計劃或引起突發性暴漲？紙箱廠心態要崩

//

新一年漲價已預訂？

而對於三級廠來說最大的疑慮是，廣東理文如此早、如此大規模的停機計劃，會不會提前烘托出原紙庫存不足的氣氛？更嚴重的是，會不會引起突發性的暴漲？這種突發性的漲價，最受傷害的可能就是夾在造紙廠和終端客戶間的三級廠了。

值得一提的是，2018年，由於玖龍理文過早宣佈停機計劃，有媒體評價認為此舉透露出了紙業環境的寒冷，那麼今年又是什麼情況？冰川時代嗎？（瑟瑟發抖）

或許正是如此吧。

有數據顯示，截至2019年10月，全國箱板瓦楞紙總產量不足，3100萬噸，比去年同期減少39萬噸；總銷量不足3200萬噸，同比跌幅超過3.5個百分點。

而形成對比的是庫存天數，今年10月箱板瓦楞紙平均庫存天數為11天，對比2018年的庫存天數是8天，三個指標的表現均差於2018年同期。

基於這些數據，容小編以小人之心度君子之腹一下：這次的停機計劃有沒有可能人為製造一起短暫的原紙缺貨？

當然，這得看其他造紙廠肯不肯跟，如果其他紙企大力度跟停，那麼在紙企恢復開機到庫存重新建立的空窗期內，可能真的就會造成原紙短缺，繼而進一步引起階段性的大規模漲價吧。

原紙一漲，紙板是不是也會漲？ 2020年新春開工，“全國人民”就要“喜迎一波漲價潮”，估計心態都要崩了吧…

目前來看，其它規模紙廠並沒有公佈相應春節停機計劃，但考慮到規模紙企素來有春節停機計劃的傳統……我們持續關注。
