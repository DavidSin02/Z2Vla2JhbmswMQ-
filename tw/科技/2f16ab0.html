Office 自定義功能區開發中自定義按鈕的圖標

2020-10-29
| 科技

在office開發中，我們經常會通過自定義功能區的方式進行二次開發，比如請看一下筆者開發的一個日常使用的公文格式處理小功能如下：

今天，我們就來給大家講解一個如何給所開發的按鈕設置圖標喲。大家應該都知道，office內置了非常多的圖標庫，我們在二次開發的時候可以直接調用內置的圖標，使用imageMso就可以調用office內置的圖標，例如批量刪空段落的定義如下。

<button id="btn5" label="批量刪空段落" size="large" imageMso="Delete" onAction="DelPar"/>

但在某些時候，我們希望自己定義開發按鈕的圖標，或者希望插件用在wps上（因為內置圖標的不同，使用內置圖標在wps上不能正確顯示，如下圖），此時就需要用到自定義圖標的開發了喲，下面我就以C# VSTO開發為例給大家介紹下開發步驟。

一、添加自定義的圖片到項目中

雙擊Properties下的Resources.resx，打開添加資源截面。

在添加資源中選擇【添加現有文件（E）】，找到需要的圖標。大家可以到各種圖標庫中尋找自己喜歡的圖標，這裡給大家推薦個本人使用的Iconfont-阿里巴巴矢量圖標庫，使用起來還是比較方便。

以下我們就以剛才添加的【safe】圖標為例演示。

二、通過getImage回調方法添加圖標

在需要自定義圖標的按鈕上使用getImage回調方法添加圖標，此時我們需要在Ribbon.cs文件中為getImage定義回調方法。

Ribbon.xml文件中的定義：

<button id="btn7" label="批量刪空段落" size="large" onAction="DelPar" getImage="GetImage"/>

Ribbon.cs文件中的定義：

public Bitmap GetImage(Office.IRibbonControl Control)

{

switch (Control.Id)

{

case "btn6":

return new Bitmap(Properties.Resources._2);

case "btn7":

return new Bitmap(Properties.Resources.safe);

default:

return null;

}

}


通過以上的步驟，我們就完成了為按鈕定義自己喜歡的圖標了喲，大家學會了嗎？
