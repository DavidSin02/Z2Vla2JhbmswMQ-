字符串去除重複字符

2020-10-29
| 科技

現有一串字符中有多個重複的字符，需要將重複的字符去除，只保留一個不重複的字符。

​代碼：

Public Function MoveSame(strSame) As String

Dim s As String, i As Long

s = strSame

i = 1

Do While i < Len(s)

s = Left(s, i) & Replace(s, Mid(s, i, 1), "", i + 1)

i = i + 1

Loop

MoveSame = s

End Function
