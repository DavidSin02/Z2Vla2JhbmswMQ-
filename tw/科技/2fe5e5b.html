各大電影節青睞的佳片，每一部都不容錯過

2020-10-29
| 科技

又到了一年一度的頒獎季，除了歐洲“三大電影節”的精選，還有許多年底的頒獎大熱門都將陸續登場。鳳凰衛視電影臺特別精選了近年入圍各大電影節甚至獲獎的經典佳作，不僅有意大利國寶級女神莫妮卡·貝魯奇、鬼才導演昆汀·塔倫蒂諾的驚世神作，下面這些好片也不容錯過！

01

西西里島的美麗傳說



導演: 朱塞佩·託納多雷

主演: 莫妮卡·貝魯奇 / 朱塞佩·蘇爾法羅

類型: 劇情 / 戰爭 / 愛情

地區: 意大利 / 美國

播映日期: 10月27日 21:15 / 10月28日 16:00




該片獲得柏林國際電影節金熊獎最佳影片提名，講述了二戰時期意大利西西里島上的美麗少婦瑪蓮娜的故事。1941年春末，13歲的少年雷納多初次見到瑪蓮娜。她的美麗讓男人心醉、女人羨妒，徹底征服了西西里島。雷納多也對瑪蓮娜情難自已，他不僅跟其他青少年們一起搜尋瑪蓮娜的誘人丰姿與萬種風情，還悄悄地成為她不知情的小跟班，如影隨形地窺視她的生活。她搖曳的倩影，她聆聽的音樂，她貼身的衣物都成為這個被荷爾蒙淹沒的少年，最真實、最美好的情慾幻想。然而，透過雷納多的眼，人們也看到了瑪蓮娜掉進了越來越黑暗的深淵之中。

02

非常背叛


導演: 保羅·馬祖斯基

主演: 雪兒 / 查茲·帕爾明特瑞 / 裡安·奧尼爾

類型: 劇情 / 喜劇

地區: 美國

播映日期: 10月2日 21:15 / 10月3日 18:00




結婚二十年來，瑪格麗特因為婚姻不幸福與自己的助理外遇了。發現自己妻子外遇的傑克僱用的殺手東尼闖進瑪格麗特家將她全身捆綁，並斥責她不該背叛丈夫。而後兩人竟不知不覺地談論起“忠貞”的意義，後續情勢發展更趨古怪。殺手東尼竟被瑪格麗特迷住並陷入荒淫之中，究竟是誰在勾引誰？東尼最終又是否能完成任務？這一段古怪脫序的謀殺情事，最後會如何完結呢？

03

四個房間



導演: 阿利森·安德斯 / 亞歷山大·洛克威爾 / 羅伯特·羅德里格茲 / 昆汀·塔倫蒂諾

主演: 安東尼奧·班德拉斯 / 薩爾瑪·海耶克 / 麥當娜 / 昆汀·塔倫蒂諾 / 布魯斯·威利斯

類型: 喜劇

地區: 美國

播映日期: 10月31日 21:15




影片講述了酒店服務生特德服務的四個房間裡發生的一系列怪事：一群女巫準備在當晚復活他們的女神，而她們卻弄丟了至關重要的精液，小侍應生臨時客串，勇敢獻身；善妒的丈夫懷疑妻子與他人有染，遂將妻子綁在椅子上，倒黴的侍應生則被當作姦夫，手槍抵頭，命懸一線；富翁夫妻參加新年宴會，將子女留在房間中。兩個淘氣的小傢伙展開了一場大戰，卻看到了恐怖的一幕；幾名好萊塢演員湊在房間中玩打火機遊戲，每人打十下，十下連續打著就贏得一部跑車，反之便要切掉一根手指。侍應生被臨時找來充當劊子手。


04

危險關係



導演: 昆汀·塔倫蒂諾

主演: 帕姆·格里爾 / 塞繆爾·傑克遜 / 羅伯特·福斯特

類型: 劇情 / 驚悚 / 犯罪

地區: 美國

播映日期: 10月25日 11:20 / 10月26日 04:10




該片改編自美國小說家埃爾穆爾·倫納德的小說《神奇之力》，獲得柏林國際電影節金熊獎最佳影片提名，塞繆爾·傑克遜憑藉此片獲得銀熊獎最佳男演員。影片講述了空中小姐傑奇暗地裡一直利用職務之便，幫助軍火商奧德爾運送贓款，並從中獲取暴利。一次意外中，傑奇的“生意”曝光了，她本人亦遭到逮捕。奧德爾知道後派馬克斯殺人滅口。讓奧德爾沒有想到的是，馬克斯竟然愛上了傑奇，兩人一拍即合，策劃了一場陰謀將奧德爾耍得團團轉。當奧德爾發現自己受騙時，事態的發展早已經脫離了他的掌控。

05

尋找費里尼


導演: 塔倫·萊克斯頓

主演: 瑪麗亞·貝羅 / 克塞尼亞·索羅

類型: 劇情 / 冒險

地區: 美國 / 意大利

播映日期: 11月13日 21:15 / 11月14日 15:25


本片改編自替卡通《辛普森一家》巴特·辛普森配音的美國知名配音演員南希·卡特賴特年輕時的尋夢故事。影片講述了露西從小在母親的保護下長大，母女倆最大的休閒娛樂是一起看通俗感人的愛情電影。在某次失敗的工作面試後，露西走進電影院，費里尼的電影正在播映。費里尼的電影與過去她的世界截然不同，一個她未曾感受過的世界充滿奇想與慾望。於是，她決定離開美國，前往意大利，踏上尋找費里尼的旅程……

06

嗅探員



導演: 阿穆爾·古普特

主演: 帕什瓦·達銳沃 / 庫什米特·吉爾

類型: 喜劇 / 家庭

地區: 印度

播映日期: 11月14日 21:15 / 11月15日 17:45




影片講述了八歲的小男孩桑尼因為天生沒有嗅覺而感到非常落寞和自卑。在一個平常的上學日，桑尼在探索生物教室的過程中，意外打翻了神祕的藥水。吸入藥水釋放的氣體之後，桑尼意外獲得了異常靈敏的嗅覺。他不但可以聞出味道，而且能聞出人們的情緒以及感情，比如悲傷，喜悅等；更能分辨出人們剛剛去過哪裡，將要去哪裡。桑尼憑藉比狗更靈敏的嗅覺，成為了小鎮上的民間偵探。這時小區裡出現了一個偷車賊，桑尼幫助大家找尋失蹤的汽車，沒想到卻揭開了一件更大的盜車案。

07

傑出公民


導演: 加斯頓·杜帕拉特 / 馬里亞諾·寇恩

主演: 奧斯卡·馬丁內茲 / 達迪·佈列瓦

類型: 劇情 / 喜劇

地區: 阿根廷 / 西班牙

播映日期: 11月24日 21:15 / 11月25日 15:35



本片獲得威尼斯國際電影節主競賽單元金獅獎提名，主角奧斯卡·馬丁內茲憑藉此片榮獲沃爾皮杯最佳男演員。影片講述了小說家丹尼爾榮獲諾貝爾文學獎後，登上了事業的頂峰。成功給他帶來了巨大的財富，卻讓他擔憂名聲和財富會奪走他文學創作上的靈感與鋒芒。五年後，這位定居巴塞羅那的作家依然炙手可熱。他讓助手禮貌地回絕一堆盛情邀請，直到收到了一封來自家鄉薩拉斯的信。丹尼爾所有創作的靈感全部來自這片故土，卻已經四十年沒有回鄉。令人意外的是，他接受了小鎮的邀請，踏上前往接受“傑出公民”獎的旅途。


08

脆弱地帶


導演: 羅賓·史威考德

主演: 布萊恩·科蘭斯頓 / 詹妮弗·加納 / 比佛莉·德安姬羅

類型: 劇情

地區: 美國

播映日期: 11月8日 11:50 / 11月9日 06:20




在曼哈頓工作的律師霍華德擁有美麗的妻子和一對可愛的雙胞胎女兒。他們住在郊外的獨棟別墅，過著美國傳統精英階層的典型生活。然而霍華德厭煩了這樣朝九晚五的生活。直到某天，下班回家的火車突然出軌，也誘發了霍華德開啟他的脫軌人生。於是，他決定躲在家對面的閣樓，給家人造成消失的假像。可是他不知這個作死的行為將帶給他嚴重的後果。


09

希望的另一面


導演: 阿基·考里斯馬基

主演: 韋勒·維坦恩 / 卡蒂·奧廷寧

類型: 劇情 / 喜劇

地區: 芬蘭 / 德國

播映日期: 12月6日 12:40 / 12月7日 07:25


該片獲得柏林國際電影節金熊獎最佳影片提名，導演阿基·考里斯馬基憑藉此片獲得銀熊獎最佳導演。影片講述了敘利亞難民哈立德偷渡到芬蘭赫爾辛基請求庇護的故事。因家鄉戰亂、房屋被炸，敘利亞人哈立德逃難到其他國家，途中遭遇驅逐、毆打，還和妹妹走散。但移民中心並沒有打算收留他，下令將他遣送回國。哈立德不得不逃出警察視線，開始了在芬蘭東躲西藏的生活。


10

狂鼠



導演: 約瑟夫·哈德

主演: 約瑟夫·哈德 / 皮婭·希爾澤伽 / 耶爾格·哈特曼

類型: 劇情 / 喜劇 / 犯罪

地區: 奧地利 / 德國

播映日期: 12月27日 12:55 / 12月28日 07:45


本片榮獲柏林國際電影金熊獎最佳影片提名。影片講述了維也納一家報社的樂評編輯格奧爾格對工作一向遊刃有餘，受到同事們的愛戴和敬仰。可是報社突然裁員，格奧爾格被解僱了。格奧爾格決定向他的前上司復仇。起初他只是肆意破壞他的私人物品，之後愈演愈烈。有一天，格奧爾格遇到了他兒時的同學埃裡希。埃裡希曾經是學校的狠角色，現在也是無業遊民。兩人一拍即合，決定修復維也納普拉特主題公園內年久失修的過山車“狂鼠”。

11

重返蒙托克




導演: 沃爾克·施隆多夫

主演: 斯特蘭·斯卡斯加德 / 尼娜·霍斯

類型: 劇情

地區: 德國 / 愛爾蘭 / 法國

播映日期: 12月11日 21:15 / 12月12日 17:50


該片榮獲柏林國際電影節提名，講述了作家馬克思·左恩來到紐約參加自己新書的發佈會。書中描寫了一段17年前在這座城市發生和終結的感情。機緣巧合，他遇到了這位被他寫進書裡的女人麗貝卡。她來自東德，如今已是一位成功的律師，在紐約生活了20年。二人決定在一起度一個週末。冬天長島盡頭蒙托克的海灘上，兩把躺椅架在寒風中，兩個在時光裡錯過的人再度回到這裡，帶著無盡的希望和失意。

12

兩張彩票


導演: 保羅·內戈埃斯庫

主演: 多裡安·波古策 / 德拉戈什·布庫爾

類型: 劇情 / 喜劇

地區: 羅馬尼亞

播映日期: 12月8日 21:15 / 12月9日 18:10


該片獲得北京國際電影節天壇獎提名，講述了一個小城鎮的三名男子尋找遺失的彩票的過程中發生的有趣故事。迪奈勒是一名修車工人，幸運之神似乎從未光顧過他。客戶總是對他的工作不滿意，他窮的連酒都買不起，妻子也因此離開了他。然而，有一天，迪奈勒買的彩票意外中了大獎，這讓他重新燃起了生活的希望。但他似乎高興過了頭，弄丟了那張彩票。於是迪奈勒和他的朋友思萊、帕皮裡優一起踏上了尋找彩票的旅程。
