文件操作

  • 文件操作的作用把一些內容(數據)存儲存放起來，可以讓程序下一次執行的時候直接使用，而不必重新制作一份,省時省力
  • 文件的基本操作：可以只打開和關閉文件,不進行任何讀寫操作
  • 打開
  • 使用open函數, 可以打開一個已經存放的文件, 或者創建一個新的文件,
  • 語法:
  • open(name, mode)
  • name: 是要打開的目錄文件名的字符串(可以包含文件所在的具體位置)
  • mode: 設置文件的模式(訪問模式)：只讀、寫入、追加等
  • 例如：
  • f = open('test.txt', 'w') # 注意: 此時的f是open函數的文件對象
  • 訪問模式:
  • 模式
  • 描述
  • r
  • 以只讀方式打開文件。文件的指針將會放在文件的開頭。這是默認模式。
  • rb
  • 以二進制格式打開一個文件用於只讀。文件指針將會放在文件的開頭。這是默認模式。
  • r+
  • 打開一個文件用於讀寫。文件指針將會放在文件的開頭。
  • rb+
  • 以二進制格式打開一個文件用於讀寫。文件指針將會放在文件的開頭。
  • w
  • 打開一個文件只用於寫入。如果該文件已存在則打開文件，並從開頭開始編輯，即原有內容會被刪除。如果該文件不存在，創建新文件。
  • wb
  • 以二進制格式打開一個文件只用於寫入。如果該文件已存在則打開文件，並從開頭開始編輯，即原有內容會被刪除。如果該文件不存在，創建新文件。
  • w+
  • 打開一個文件用於讀寫。如果該文件已存在則打開文件，並從開頭開始編輯，即原有內容會被刪除。如果該文件不存在，創建新文件。
  • wb+
  • 以二進制格式打開一個文件用於讀寫。如果該文件已存在則打開文件，並從開頭開始編輯，即原有內容會被刪除。如果該文件不存在，創建新文件。
  • a
  • 打開一個文件用於追加。如果該文件已存在，文件指針將會放在文件的結尾。也就是說，新的內容將會被寫入到已有內容之後。如果該文件不存在，創建新文件進行寫入。
  • a+
  • 打開一個文件用於讀寫。如果該文件已存在，文件指針將會放在文件的結尾。文件打開時會是追加模式。如果該文件不存在，創建新文件用於讀寫。
  • ab
  • 以二進制格式打開一個文件用於追加。如果該文件已存在，文件指針將會放在文件的結尾。也就是說，新的內容將會被寫入到已有內容之後。如果該文件不存在，創建新文件進行寫入。
  • ab+
  • 以二進制格式打開一個文件用於追加。如果該文件已存在，文件指針將會放在文件的結尾。如果該文件不存在，創建新文件用於讀寫。
  • 主訪問模式的特點:
  • # r: 如果文件不存在,報錯; 不支持寫入操作,表示只讀
  • f = open('test1.txt','r')
  • f = open('test.txt','r')
  • f.write('aa')
  • f.close()


  • # w：只寫,如果文件不存在,新建文件; 如果執行寫入會覆蓋原有內容
  • f = open('1.txt','w')
  • f.write('bbb')
  • f.close()


  • # a：追加,如果文件不存在,新建文件; 在原有內容基礎上追加新內容
  • f = open('2.txt','a')
  • f.write('xyz')
  • f.close()

  • # 訪問模式參數是否可以省略, 如果省略表示訪問模式為r
  • f = open('1.txt')
  • f.close()


  • 訪問模式特點2：
  • # r+: r+沒有該文件則報錯; r特點:文件指針在開頭,所以能讀取出來數據
  • f = open('test1.txt', 'r+')
  • f = open('test.txt', 'r+')

  • # w+: 沒有該文件會新建文件; w特點: 文件指針在開頭,用新內容覆蓋原內容.
  • f = open('test1.txt', 'w+')
  • f = open('test.txt', 'w+')

  • # a+: 沒有該文件會新建文件; a特點: 文件指針在結尾,無法讀取數據(文件指針後面沒有數據)
  • f = open('test100.txt', 'a+')
  • f = open('test.txt', 'a+')
  • con = f.read()
  • print(con)
  • f.close()

  • 寫
  • 語法：
  • 文件對象. write('內容')
  • 例如：
  • # 1. 打開文件
  • f = open('test.txt', 'w')
  • # 2. 文件寫入
  • f.write('hello world')
  • # 3. 關閉文件
  • f.close()
  • # 注意:
  • # 1. w和a模式: 如果文件不存在則創建文件;如果文件存在，w模式先清空再寫入, a模式直接末尾追加
  • # 2. r模式: 如果文件不存在則報錯

  • 讀
  • 語法一:
  • 文件對象.read(num)
  • num表示要從這個文件中讀取的數據的長度(單位是字節), 如果沒有傳入num，那麼就表示讀取文件中的所有的數據
  • 例如:
  • f = open('test.txt','r')
  • # 文件內容如果換行：底層有\n,會有字節的佔位,導致read書寫參數讀取出來的眼睛看到的個數和參數值不匹配
  • # read不寫參數表示讀取所有的;
  • # print(f.read())
  • print(f.read(10)) f.close()

  • 語法二：
  • 文件對象.readlines()
  • readlines可以按照行的方式把整個文件中的內容進行一次性讀取,並返回的是一個列表，其中一行的數據為一個元素
  • 例如：
  • f = open('test.txt')
  • content = f.readlines()
  • # ['hello world\n', 'abcdefg\n', 'aaa\n', 'bbb\n', 'ccc']
  • print(content)

  • 語法三:
  • 文件對象.readline()
  • readline()一次讀取一行內容
  • f = open('test.txt','r')
  • content = f.readline()
  • print(f'第一行:{content}')
  • content = f.readlin()
  • pint(f'第二行:{content}')
  • # 關閉文件
  • f.close()


  • 語法四：
  • 文件對象.seek(偏移量, 起始位置)
  • 起始位置：0：文件開頭 1：當前位置 2：文件結尾
  • # f = open('test.txt', 'r+')
  • f = open('test.txt', 'a+')
  • # 1. 改變讀取數據開始位置
  • # f.seek(2,0)
  • # 1. 文件指針放在結尾(無法讀取數據)
  • # f.seek(0,2)
  • 2. a 改變文件指針的位置,做到可以讀取出來數據
  • # f.seek(0,0)
  • f.seek(0)
  • con = f.read()
  • print(con)
  • f.close()
  • 關閉語法：文件對象.close()

with語句和上下文管理器

  • with語句的作用
  • python提供了with語句的寫法, 即簡單又安全
  • 文件操作的時候使用with語句可以自動調用關閉文件操作，即使出現了異常也會自動調用關閉
  • 文件操作文件使用完後必須關閉，因為文件對象會佔用操作系統的資源，並且操作系統同一時間能打開的文件數量也是有限的。
  • 上下文管理器
  • with語句之所以這麼強大，背後是由上下文管理器做支撐的
  • 也就是說使用f = open('xxx')中的open函數創建的f文件對象就是一個上下文管理器對象
  • 什麼是上下文管理器？
  • 一個類只要實現了__enter__()和__exit__()這兩個方法
  • 通過該類創建的對象我們就稱之為上下文管理器
  • class File(object):
  • # 初始化方法
  • def __init__(self, file_name, file_mode1):
  • # 定義變量保存文件名和打開模式
  • self.file_name = file_name
  • self.file_mode1 = file_mode1
  • # 上文方法 def __enter__(self):
  • print('進入上文的方法')
  • # 返回文件資源
  • self.file = open(self.file_name, self.file_mode1)
  • return self.file
  • # 下文方法 def __exit__(self, exc_type, exc_val, exc_tb):
  • print('進入下文的方法')
  • self.colose()
  • __enter__表示上文方法，需要返回一個操作文件的對象
  • __exit__表示下文方法，with語句執行完成後會自動執行，即使出現異常也
