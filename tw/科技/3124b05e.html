OpenCV 中的輪廓

✏️問：什麼是輪廓？

️答：輪廓是一系列相連的點組成的曲線，代表了物體的基本外形，相對於邊緣，輪廓是連續的，邊緣並不全部連續。

✏️問：如何尋找輪廓？

️答：尋找輪廓的操作一般用於二值化圖，所以通常會使用閾值分割或Canny邊緣檢測先得到二值圖

PS：尋找輪廓是針對白色物體的，一定要保證物體是白色，而背景是黑色，不然很多人在尋找輪廓時會找到圖片最外面的一個框。

尋找輪廓

❣️調用 cv2.findContours() 函數:

import cv2img = cv2.imread('handwriting.jpg')img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)ret, thresh = cv2.threshold(img_gray, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)# 尋找二值化圖中的輪廓image, contours, hierarchy = cv2.findContours(    thresh, cv2.RETR_TREE, cv2.CHAIN_APPROX_SIMPLE)print(len(contours))  # 結果應該為2
  • 參數1：二值化原圖
  • 參數2：輪廓的查找方式，一般使用cv2.RETR_TREE，表示提取所有的輪廓並建立輪廓間的層級。
  • 參數3：輪廓的近似方法。比如對於一條直線，我們可以存儲該直線的所有像素點（cv2.CHAIN_APPOX_NONE），也可以只存儲起點和終點。使用 cv2.CHAIN_APPROX_SIMPLE 就表示用儘可能少的像素點表示輪廓.

繪製輪廓

❣️調用 cv2.drawContours() 函數：

cv2.drawContours(img, contours, -1, (0,0,255),2)

️其中參數2就是得到的contours，參數3表示要繪製哪一條輪廓，-1表示繪製所有輪廓，參數4是顏色（B/G/R通道，所以(0,0,255)表示紅色），參數5是線寬.

️一般情況下，我們會首先獲得要操作的輪廓，再進行輪廓繪製及分析：

cnt = contours[1]cv2.drawContours(img, [cnt], 0, (0, 0, 255), 2)

輪廓層級


輪廓層級

圖中總共有8條輪廓，2和2a分別表示外層和裡層的輪廓，3和3a也是一樣。從圖中看得出來：

  • 輪廓0/1/2是最外層的輪廓，我們可以說它們處於同一輪廓等級：0級
  • 輪廓2a是輪廓2的子輪廓，反過來說2是2a的父輪廓，輪廓2a算一個等級：1級
  • 同樣3是2a的子輪廓，輪廓3處於一個等級：2級
  • 類似的，3a是3的子輪廓，等等…………

️ OpenCV中輪廓等級的表示:如果我們打印出cv2.findContours()函數的返回值hierarchy，會發現它是一個包含4個值的數組：[Next, Previous, First Child, Parent]

  • Next: 與當前輪廓處於同一層級的下一條輪廓，沒有為-1。
  • Previous: 與當前輪廓處於同一層級的上一條輪廓，沒有為-1。
  • Firtst Child: 當前輪廓的第一條子輪廓，沒有為-1。
  • Parent: 當前輪廓的父輪廓，沒有為-1。

️ 輪廓的四種尋找方式：

  • RETR_LIST：所有輪廓屬於同一層級
  • RETR_TREE: 完整建立輪廓的各屬性
  • RETR_EXTERNAL: 只尋找最高層級的輪廓
  • RETR_CCOMP: 所有輪廓分2個層級，不是外界就是最裡層

✏️問：如何把下圖的三個內圈填滿灰色？

灰色填充

️代碼如下：

import cv2import numpy as npimg = cv2.imread('circle_ring.jpg')img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)_,th = cv2.threshold(img_gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)# 尋找輪廓，使用cv2.RETR_CCOMP尋找內外輪廓image, contours, hierarch = cv2.findContours(th, cv2.RETR_CCOMP, 2)# 找到內層輪廓並填充# hierarchy的形狀為(1,6,4)，使用np.squeeze壓縮一維數據，變成(6,4)hierarchy = np.squeeze(hierarchy)for i in range(len(contours)):    # 存在父輪廓，說明是裡層    if (hierarchy[i][3] != -1):        cv2.drawContours(img, contours, i, (180, 215, 215), -1)cv2.imwrite('result.jpg', img)

輪廓的特徵

  • 計算物體的周長、面積、質心、最小外接矩形等 ⛳️
  • OpenCV函數：cv2.contourArea(), cv2.arcLength(), cv2.approxPolyDP()等 ⛳️

圖像矩

  • 圖像矩可以幫助我們計算圖像的質心，面積等；
  • 函數 cv2.moments() 會將計算得到的矩以字典形式返回。

圖像輪廓

import cv2import numpy as npimg = cv2.imread('handwriting.jpg', 0)_, thresh = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)image, contours, hierarchy = cv2.findContours(thresh, 3, 2)# 以數字3的輪廓為例cnt = contours[0]img_color1 = cv2.cvtColor(image, cv2.COLOR_GRAY2BGR)img_color2 = np.copy(img_color1)cv2.drawContours(img_color1, [cnt], 0, (0, 0, 255), 2)cv2.imshow('img',img_color1)cv2.waitKey(0)

️採用圖像矩

M = cv2.moments(cnt)# 對象的質心cx = int(M['m10'] / M['m00'])cy = int(M['m01'] / M['m00'])

✔️M中包含了很多輪廓的特徵信息，比如M[‘m00’]表示輪廓面積，與cv2.contourArea() 計算結果是一樣的.

輪廓面積

area = cv2.contourArea(cnt)

✔️注意輪廓特徵計算的結果並不等同於像素點的個數，而是根據幾何方法算出來的，所以有小數。

如果統計二值圖中像素點個數，應儘量避免循環，可以使用cv2.countNonZero()，更加高效。

輪廓周長

perimeter = cv2.arcLength(cnt, True)

✔️ 參數2表示輪廓是否封閉，顯然我們的輪廓是封閉的，所以是True。

外接矩形

  • 形狀的外接矩形有兩種，如下圖，綠色的叫外接矩形，表示不考慮旋轉並且能包含整個輪廓的矩形。藍色的叫最小外接矩，考慮了旋轉：

外接矩形

1️⃣ 外接矩形：

x, y, w, h = cv2.boundingRect(cnt)

2️⃣ 最小外接矩形：

rect = cv2.minAreaRect(cnt)# 矩形四個角點取整box = np.int0(cv2.boxPoints(rect))cv2.drawContours(img_color1, [box], 0, (255, 0, 0), 2)

✔️ np.int0(x) 是把x取整的操作，比如377.93就會變成377，也可以用x.astype(np.int)

最小外接圓

(x, y), radius = cv2.minEnclosingCircle(cnt)(x, y, radius) = cv2.int0((x, y, radius))cv2.circle(img_color2, (x,y), radius, (0, 0, 255), 2)

形狀匹配

  • cv2.matchShapes()可以檢測兩個形狀之間的相似度，返回值越小，越相似 ⛳️

形狀匹配

img = cv2.imread('shapes.jpg', 0)_, thresh = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)image, contours, hierarchy = cv2.findContours(thresh, 3, 2)img_color = cv2.cvtColor(thresh, cv2.COLOR_GRAY2BGR)  # 用於繪製的彩色圖

圖中有3條輪廓，我們用A/B/C表示：

cnt_a, cnt_b, cnt_c = contours[0], contours[1], contours[2]print(cv2.matchShapes(cnt_b, cnt_b, 1, 0.0))  # 0.0print(cv2.matchShapes(cnt_b, cnt_c, 1, 0.0))  # 2.17e-05print(cv2.matchShapes(cnt_b, cnt_a, 1, 0.0))  # 0.418

✔️ 可以看到BC相似程度比AB高很多，並且圖形的旋轉或縮放並沒有影響。其中，參數3是匹配方法，參數4是OpenCV的預留參數，暫時沒有實現，可以不用理會。

輪廓近似

  • 將輪廓形狀近似到另外一種由更少點組成的輪廓形狀，新輪廓的點的數目由我們設定的準確度來決定，用的Douglas-Peucker算法。⛳️
import cv2import numpy as np# 1.先找到輪廓img = cv2.imread('unregular.jpg', 0)_, thresh = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)image, conturs, hierarchy = cv2.findContours(thresh, 3, 2)cnt = contours[0]# 2.進行多邊形逼近，得到多邊形的角點approx = cv2.approxPolyDP(cnt, 3, True)# 3.畫出多邊形image = cv2.cvtColor(image, cv2.COLOR_GRAY2BGR)cv2.polylines(image, [approx], True, (0, 255, 0), 2)

其中cv2.approxPolyDP() 的參數2(epsilonepsilon)是一個距離值，表示多邊形的輪廓接近實際輪廓的程度，值越小，越精確；參數3表示是否閉合。

輪廓近似

凸包

  • 凸包跟多邊形逼近很像，只不過它是物體最外層的”凸”多邊形：集合A內連接任意兩個點的直線都在A的內部，則稱集合A是凸形的。如下圖，紅色的部分為手掌的凸包，雙箭頭部分表示凸缺陷(Convexity Defects)，凸缺陷常用來進行手勢識別等。

import cv2img = cv2.imread('convex.jpg', 0)_, th = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY+cv2.THRESH_OTSU)image, contours, hierarchy = cv2.findContours(th, 3, 2)cnt = contours[0]image = cv2.cvtColor(img, cv2.COLOR_GRAY2BGR)cv2.drawContours(image, contours, -1, (0, 0 , 255), 2)# 尋找凸包，得到凸包的角點hull = cv2.convexHull(cnt)# 繪製凸包cv2.polylines(image, [hull], True, (0, 255, 0), 2)

✔️ 其中函數cv2.convexHull()有個可選參數returnPoints，默認是True，代表返回角點的x/y座標；如果為False的話，表示返回輪廓中是凸包角點的索引，比如說：

print(hull[0])  # [[362 184]]（座標）hull2 = cv2.convexHull(cnt, returnPoints=False)print(hull2[0])  # [510]（cnt中的索引）print(cnt[510])  # [[362 184]]

✔️ 當使用cv2.convexityDefects()計算凸包缺陷時，returnPoints需為False

✔️ 另外可以用下面的語句來判斷輪廓是否是凸形的：

print(cv2.isContourConvex(hull))  # True

凸面缺陷

  • OpenCV提供了現成的函數來做這個，cv2.convexityDefects().
  • 注意：我們要傳returnPointsFalse來找凸形外殼。
  • 它返回了一個數組，每行包含這些值：[start point, end point, farthest point, approximate distance to farthest point].我們可以用圖像來顯示他們。我們畫根線把start point和end point連起來。然後畫一個圓在最遠點。記住最前三個返回值是 cnt 的索引，所以我們我們得從 cnt 裡拿出這些值.
import cv2import numpy as npimg = cv2.imread('star.jpg')img_gray = cv2.cvtColor(img,cv2.COLOR_BGR2GRAY)ret, thresh = cv2.threshold(img_gray, 127, 255,0)_,contours,hierarchy = cv2.findContours(thresh,2,1)cnt = contours[2]# 返回凸包角點的索引hull = cv2.convexHull(cnt,returnPoints = False)# 檢測凸凹陷defects = cv2.convexityDefects(cnt,hull)# 可視化for i in range(defects.shape[0]):    s,e,f,d = defects[i,0]    start = tuple(cnt[s][0])    end = tuple(cnt[e][0])    far = tuple(cnt[f][0])    cv2.line(img,start,end,[0,255,0],2)    cv2.circle(img,far,5,[0,0,255],-1)

點到輪廓距離

  • cv2.pointPolygonTest() 函數計算點到輪廓的最短距離（也就是垂線），又稱多邊形測試：
dist = cv2.pointPolygonTest(cnt, (100, 100), True)  # -3.53

✔️ 其中參數3為True時表示計算距離值：點在輪廓外面值為負，點在輪廓上值為0，點在輪廓裡面值為正；參數3為False時，只返回-1/0/1表示點相對輪廓的位置，不計算距離。
