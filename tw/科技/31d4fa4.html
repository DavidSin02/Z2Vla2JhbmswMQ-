帶隙可調的Se-Te薄膜，用於短波紅外光電探測

2020-10-29
| 科技

研究背景

紅外光電探測器和成像傳感器陣列（波長範圍為1-14 μm）在包括夜視、遙感、光通信和新興醫學成像在內的廣泛應用中至關重要。短波紅外（SWIR，1-3 μm）器件和陣列中的半導體主要由單晶鍺（Ge）和III-V半導體（例如InGaAs和InGaAsP）主導。但是，這些材料通常通過複雜的方法來生長，包括MBE和MOCVD，從而導致複雜的製備過程並隨之產生高成本。因此，開發能夠結合簡單製備過程和高器件性能的新材料體系仍然具有挑戰性。可以滿足上述要求的一種潛在材料體系是碲（Te），一種具有一維晶體結構的材料，其間接帶隙~0.31 eV。最近的研究表明，Te可製備用於紅外光探測的光電導體，具有良好的響應率和比探測率。然而，Te基光電導體的性能受到高暗電流的限制。

成果介紹

有鑑於此，近日，美國加州大學伯克利分校Ali Javey教授等報道了帶隙可調的熱蒸發SexTe1-x合金薄膜，用於製備高性能SWIR光電探測器。從吸收測量結果可以看出，SexTe1-x薄膜的帶隙可以連續地從0.31 eV（Te）調諧到1.87 eV（Se）。由於其可調的帶隙，可以在SWIR方案中調控 Se x Te 1-x薄膜基光電導體的峰值響應位置和光響應邊緣。通過使用由Au/Al2O3組成的光學腔體襯底來增強其在帶隙邊緣附近的吸收，基於Se0.32Te0.68薄膜（光學帶隙≈0.8 eV）的光電導體在1.7 μm處具有截止波長，並且室溫下在1.55 μm時的響應率為1.5 AW-1，隱含探測率為6.5×1010 cm Hz1/2 W-1。重要的是，熱蒸發過程的性質使得能夠製備具有良好像素均勻性的 Se 0.32 Te 0.68基42×42焦平面陣列，從而證明了這種用於紅外成像傳感器系統的獨特材料體系的潛力。文章以“Evaporated Se x Te 1-x Thin Films with Tunable Bandgaps for Short-Wave Infrared Photodetectors”為題發表在著名期刊Advanced Materials上。

圖文導讀

圖1. Se x Te 1-x薄膜的晶體結構和材料表徵。（a）從不同方向觀察 Se x Te 1-x晶體的晶體結構。（b-d）不同組分 Se x Te 1-x薄膜的RBS光譜、拉曼光譜和XRD圖。（e）25 nm厚Se0.32Te0.68薄膜的HRTEM圖像。

Se x Te 1-x合金與Te具有相同的晶體結構，其中Te被Se部分取代，如圖1a所示。可以清楚地看到，三角形螺旋Se/Te原子鏈通過範德華力以六邊形陣列堆垛在一起。這種材料的獨特優勢之一是，Se/Te原子僅通過每個單螺旋鏈中的共價鍵連接到兩個最近的Se/Te原子，這表明SexTe1-x具有真正的一維晶體結構。因此，懸掛鍵僅存在於該晶體的分子鏈末端，表面複合較小。將準備好的 Se x Te 1-x合金晶體用作源，通過熱蒸發工藝在大約-110 ℃的襯底溫度下沉積厚度不同的 Se x Te 1-x薄膜。 Se x Te 1-x薄膜的組分通過盧瑟福背散射光譜（RBS）進行表徵，如圖1b所示。 Se x Te 1-x薄膜還具有拉曼光譜特徵，進一步證明了從Te到Se的逐漸轉變（圖1c）。從XRD表徵可以看出，只有少數高Te含量樣品是多晶的，而其他樣品是非晶的（圖1d）。非晶態的SexTe1-x（x> 0.3）薄膜可以在幾分鐘內通過在溫和的溫度（100-140 ℃）下進行簡單的退火工藝輕易地轉變為多晶薄膜。對於諸如Te之類的可移動原子，襯底溫度是影響薄膜質量的關鍵因素，在低溫下可以獲得最大晶粒尺寸。HRTEM圖像清楚地顯示了退火後Se0.32Te0.68薄膜的多晶結構（圖1e）。

圖2. 組分依賴的光學性能。（a）各種組分 Se x Te 1-x薄膜的吸收光譜。（b）吸收光譜中提取的光學帶隙。

接下來，對在石英（x>0.3）或CaF2（x<0.3）襯底上蒸發的 Se x Te 1-x薄膜進行系統的光學表徵。如圖2a所示， Se x Te 1-x薄膜的吸收光譜覆蓋了中波紅外（純Te）到可見光（純Se）的較寬波長範圍。純Te和Se薄膜分別具有約4000 nm和650 nm的吸收邊緣，並且當Se含量增加時，吸收邊緣會從4000 nm逐漸移至較短波長。圖2b清楚地表明，通過增加Se含量，可以將 Se x Te 1-x薄膜的帶隙從0.31 eV（純Te）調諧為1.87 eV（純Se）。請注意，所有樣品均為間接帶隙半導體。

圖3. 組分依賴的電學性能。（a） Se x Te 1-x薄膜基FET的光學顯微鏡圖像和器件結構示意圖。（b）不同組分 Se x Te 1-x FET的Id-Vg曲線。（c）不同組分 Se x Te 1-x薄膜基FET的有效遷移率和電流開/關比。

接下來，探討 Se x Te 1-x薄膜組分依賴的電學特性。進一步增加SexTe1-x薄膜中的Se含量會進一步降低霍爾效應遷移率和空穴濃度。 Se x Te 1-x薄膜基FET的光學顯微鏡圖像和器件結構示意圖如圖3a所示。Id-Vg曲線清楚地顯示出所有 Se x Te 1-x FET的p型輸運特性（圖3b）。如圖3c所示，基於純Te且厚度為8.5 nm的FET具有≈46 cm2 V-1 s-1的有效遷移率和≈103的開/關比。當 Se x Te 1-x溝道中的Se含量從0％增加到32％時，由於結晶相到非晶相的轉變，有效遷移率從≈46降低到≈1.2 cm 2 V -1 s -1，與霍爾效應測量顯示出相同的趨勢。由於帶隙隨硒含量的增加而增加，開/關比從≈103增加到大於104。

圖4. Se x Te 1-x薄膜基光電導體。（a）在Au/Al2O3襯底上 Se x Te 1-x薄膜基光電導體的示意圖。（b）不同組分 Se x Te 1-x薄膜基光電導體的歸一化光譜響應率。（c）28 nm厚Se0.32Te0.68光電導體的Id-Vg特性。（d）Se0.32Te0.68光電導體的每瓦光譜響應率。（e）隱含探測率。（f）Se0.32Te0.68光電導體的頻率響應，顯示3 dB頻率為2.4 MHz（λ=1.55 μm）。

接下來，使用Au/Al2O3作為襯底來製備 Se x Te 1-x光電導體，器件結構如圖4a所示。Au叉指接觸用於促進有效的電子/空穴提取。隨後，探討 Se x Te 1-x薄膜基光電導體組分依賴的光響應。如圖4b所示，它們具有可定製的光譜光響應，可通過改變 Se x Te 1-x的組分進而調控帶隙來實現。通過降低Se含量，可以將光響應邊緣調諧到更長的波長，Se含量分別為43％、32％和23％時，其值分別為1.5、1.8和2.5 μm。因此， Se x Te 1-x薄膜具有可廣泛調節的帶隙，能夠選擇適合給定器件應用的光響應邊緣。為此，選擇光學帶隙為0.8 eV的Se0.32Te0.68薄膜用於光電探測器的製備，以截止波長約為1.7 μm（單晶Ge SWIR光電探測器的典型值）的SWIR為目標。值得一提的是，寬的子帶隙吸收可歸因於Se0.32Te0.68薄膜中的缺陷。

為了獲得最佳性能，需要在吸收和暗電流之間取得平衡，因此，在接下來的器件中採用厚度約28 nm的 Se 0.32 Te 0.68薄膜。297和78 K時，在黑暗和白光照射下從典型的 Se 0.32 Te 0.68光電導體的Id-Vg曲線中可以觀察到光電流（圖4c）。器件的全光譜響應率如圖4d所示，對於Vd=2 V和Vg=10 V，在78 K和297 K時，探測器在λ=1.2 μm處分別顯示出1.5和1.8 AW-1的峰值響應率。圖4e給出了78 K和297 K時隱含探測率與波長的關係，該器件在室溫下的峰值D*為6.5×1010 cm Hz1/2 W-1，在低溫下，此值會增加到3.6×1011 cm Hz1/2 W-1的峰值，因為噪聲電流會大大降低，而響應率僅略有降低。為了表徵頻率響應，通過在較寬的頻率範圍（1~8 MHz）內以正弦方式調製激光二極管來測量光響應。如圖4f所示，在此頻譜範圍內，可以看到≈2.4 MHz的清晰截止頻率。

圖5. 基於Se0.32Te0.68薄膜的焦平面陣列。（a）不同放大倍數下的光學顯微鏡圖像。（b）像素的器件結構示意圖。（c）無源矩陣尋址的構型。（d）FPA所有1764像素在1.4 μm處的響應率直方圖。（e）FPA的暗電流圖像。（f）FPA拍攝的FTIR白光源圖像。

通過簡單的熱蒸發工藝， Se 0.32 Te 0.68薄膜可在各種襯底上進行晶圓級的均勻薄膜沉積。結合出色的光電探測器性能， Se 0.32 Te 0.68薄膜成為製備紅外圖像傳感器（如焦平面陣列（FPA））的有力候選者。圖5a顯示了在不同放大倍數下獲得的42×42 FPA的光學顯微鏡圖像。每個像素的範圍為80 μm×90 μm，其結構如圖5b所示。為了確認FPA中像素到像素均勻性，使用串行無源矩陣構型（圖5c）來測量所有1764個像素的光響應和暗電流。為了量化FPA均勻性，給出了在1.4 μm波長處像素響應率的直方圖，如圖5d所示。可以看出，這些響應率具有相對較窄的分佈，表明FPA具有良好的均勻性。此外，FPA在暗電流方面也顯示出良好的均勻性，1764個像素顯示出170-220 nA範圍內的暗電流（圖5e）。為了演示成像，還使用相機鏡頭將FTIR白光源成像到FPA上（圖5f）。

總結與展望

本文對熱蒸發的 Se x Te 1-x薄膜的電子和光學性質進行了系統的研究。帶隙可調的 Se x Te 1-x合金是用於製備高性能和低成本SWIR光電探測器的有前途材料。通過熱蒸發對均勻的薄膜進行簡單的低溫沉積（-110 ℃），也可以製備用於SWIR成像的FPA。將來，可以使用結晶度和純度的進一步改進來優化光電探測器的性能。憑藉簡單的沉積方法和低成本，將SexTe1-x薄膜與Si CMOS讀取芯片集成在一起，在構建高性能、低成本和高分辨紅外圖像傳感器系統方面具有巨大潛力。
