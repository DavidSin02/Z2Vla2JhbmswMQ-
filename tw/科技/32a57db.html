極致性能、無限擴展，說的就是 ALIENWARE Area-51m

2020-10-29
| 科技

來自數字尾巴

熱愛外星人的玩家可能不會對 Area-51m 感到陌生，17 年前的外星人正是憑藉著它闖入了遊戲筆記本領域，從此開啟了遊戲筆電的探索之路。如今回過頭來看這款遊戲本，無論是外形設計還是內部配置，處處都書寫著 ALIENWARE 筆電另類、張狂、強悍的風格。

而在今年的 CES 2019 展會上，ALIENWARE 將最新的旗艦遊戲本命名為 Area-51m，這也不僅讓人浮想聯翩：除了濃濃的致敬意味，背後的含義到底是什麼？如今，隨著 ALIENWARE Area-51m 到達尾巴編輯部，我也希望能從中找到答案。

充滿未來感的設計構成了 Area-51m 的第一印象

「未來感」是我第一眼見到 Area-51m 時，腦海中即刻浮現的字眼。不同於此前的扁平設計，Area-51m 從 ALIENWARE 早期的工業設計和科幻文化中汲取靈感，引入了全新的「Legend」設計風格，再加上常用於航天科技領域的鎂合金機身，最終造就了 Area-51m 更具現代化、也更具科幻色彩的外觀。

就拿機身後方的凸出式底座來說，Area-51m 在此處加入支持 ALIENFX 燈效系統的環形流光燈帶，配合上蜂窩狀的通風孔，從側面看的話，和科幻電影中經常出現的飛行推進器頗有幾分相似，這也是 Area-51m 的外觀設計中我認為最吸睛的一點。

而為了迎合新的設計語言，Area-51m 的 A 面也變得更加簡潔，除了標誌性的可發光外星人 LOGO 帶來的超高辨識度之外，右下角還能看到代表 Area-51m 的 A51 字樣，這同樣讓我想起了科幻電影中各類行動代號。另一方面，前面提到的圓環狀燈帶可以實現和 LOGO 的聯動，你可以在 ALIENWARE COMMAND CENTER 中自定義燈效 ，打造屬於你自己的外星人遊戲本。

ALIENWARE Area-51m 的 B 面的屏幕由於引入了窄邊框的設計，所以帶來了不同以往的視覺效果。個人覺得在如今的遊戲本領域，窄邊框設計雖然越來越普遍了，但如何協調好窄邊框和其他的設計就顯得尤為重要，像有些筆電為了實現所謂的三邊窄邊框，不惜將前置攝像頭挪到了底部或者採用外置的方式，這樣的窄邊框對於用戶來說到底值不值可能就要打個大大的問號了。

反觀 Area-51m 將三邊框控制在不影響前置攝像頭正常使用的寬度上，即兼顧了美觀同時還不影響前置攝像頭的正常使用；同時，底部邊框也壓縮到幾乎只留有「ALIENWARE」字樣，這也讓它的屏佔比幾乎壓縮到了極致，加上屏幕與邊框間的銜接過渡非常自然，遠遠望去，會由衷地發出驚歎聲！

要說 Area-51m 變化最小的地方可能就是 C 面了，依舊配備了快捷按鍵和全鍵盤，而由於有類膚塗層的覆蓋，所以擔心遊戲過程中手汗打滑的玩家可以安心了，整個 C 面摸起來不算特別細膩，但在鎂合金機身的作用下，有些金屬獨有的冰冷觸覺。

同時 Area-51m 的 C 面也貫徹了「燈」的主旨，包括鍵盤、觸控板以及位於頂部的電源 LOGO 二合一鍵都支持 ALIENFX 情緒同步燈效。熟悉外星人遊戲本的玩家可能都知道其燈效的優點是比較柔和，亮度分佈也比較均勻，基本不會出現明顯的斷層，配合上 ALIENWARE COMMAND CENTRE 的話能夠實現多種自定義效果，但從我的經驗來看，要在百萬中色彩中調出好看的搭配的話，還是需要有一定的色彩審美的。所以相比之下，我個人更喜歡統一的純藍色、純黃色、或者純紅色燈效，一來不用費心的搭配色彩，二來白色機身也能夠將燈效更好地呈現出來。

便攜性方面，像 Area-51m 這類遊戲本幾乎只能得個及格分，但轉念一想，其實 Area-51m 的目標用戶可能本來就不會是那些追求極致便攜的群體，他們想要的可能更加完整的遊戲體驗，哪怕犧牲一定的便攜性，但 ALIENWARE 其實也給了玩家一定的選擇空間。為什麼這麼說呢？因為需要完全發揮 Area-51m 的全部性能的話，需要兩個電源適配器（330W+180W）同時上陣，但你也可以只接入一個體積稍小的 180W 適配器來獲得基本的性能，這種情況下完成一些基礎的操作完全不是問題。

強悍性能和無限擴展才是 Area-51m 的大殺器

除了以上提到的外觀變化，ALIENWARE Area-51m 真正讓玩家感到興奮的地方可能在於發生在內部的「質變」，可以說這是一臺從誕生之初就含著金鑰匙的遊戲本，之所以這麼說不僅因為 Area-51m 是市面上第一臺搭載 i9-9900K 桌面級 CPU 的遊戲本，更是因為它支持完全擴展的特性能夠讓玩家自主更換 CPU 和 GPU，拒絕處理器和顯卡迭代造成的性能落伍。

不同於其他搭載移動級處理器的遊戲本，手上的這臺 Area-51m 搭載了英特爾 i7-9700K 桌面級處理器，在性能上有著一定的先天優勢，例如採用 14 納米制程、TDP 達 95W 等。和前一代的 i7-8700K 相比，i7-9700K 多了兩個核心數，並且基礎頻率和最高睿頻也更高一些，而這種變化對於遊戲體驗來說也更加有利，從 CINBENCH R15 的多次跑分來看，ALIENWARE Area-51m 第一次、第五次和第十次的數據差距不大，這也在一定程度上說明整機運行的穩定性：

  • 多核 1459，單核 205（第一次）
  • 多核 1426，單核 202（第五次）
  • 多核 1416，單核 199（第十次）

當然了，作為一顆帶著 K 後綴的桌面處理器，i7-9700K 允許玩家對其進行超頻，經過 ALIENWARE COMMAND CENTER 的超頻測試之後，ALIENWARE Area-51m 大致能夠維持在 5.0GHz 左右，而此時隨著 CPU 溫度的提升，機身風扇的轉動也慢慢快了起來。但讓我感到滿意的是，即使是在風扇全速運轉的情況下，ALIENWARE Area-51m 機身的噪音依舊保持在一個很小的範圍，這對於那些像我一樣，打遊戲不喜歡戴耳機的群體來說十分地友好。

另一方面，手上的這臺 ALIENWARE Area-51m 還搭載了 RTX 2080 顯卡，新一代 R 卡除了帶來比桌面版 GTX 1080 更強的性能之外，最大的亮點可能就是新加入了實時光線追蹤 RTX 和 DLSS 深度學習超級採樣技術。從 3DMark 的跑分來看，ALIENWARE Area-51m 上的 RTX 2080 的表現還算不錯，總分和桌面級 RTX 2080 咬得很緊：

而在遊戲實戰中，我也試著用 ALIENWARE Area-51m 跑了幾款支持實時光線追蹤和 DLSS 的遊戲，具體幀數如下：

▎《戰地 Ⅴ》

  • 開啟 DXR，平均幀數為 94
  • 開啟 DXR、DLSS，平均幀數為 119
  • 關閉 DXR、DLSS，平均幀數為 134

▎《地鐵：離去》（Extreme 畫質下）

  • 關閉 RTX，平均幀數為 124
  • 光追效果高，關閉 DLSS，平均幀數為 87
  • 光追效果最高，關閉 DLSS，平均幀數為 81
  • 光追效果最高，開啟 DLSS，平均幀數為 104

▎《Apex 英雄》

  • 默認分辨率、最高畫質下且關閉垂直同步，平均幀數為 198

可能是為了緩解機身散熱的負擔，這一次 ALIENWARE Area-51m 的屏幕支持 1080P 分辨率，但刷新率提高到了 144Hz，並同時支持 G-Sync 技術，從上面測試的結果來看，這種改變帶來的變化是十分顯著：一方面，如果同時支持高分辨率和高刷新率，那麼對於顯卡的壓力將十分巨大，而牽一髮動全身地也會對機身的散熱提出更高的要求。

但降低一定的分辨率並提高玩家感知更加明顯的刷新率將明顯降低顯卡的壓力；另一方面，在 ALIENWARE Cryo-Tech v2.0 散熱技術的加持下， ALIENWARE Area-51m 通過雙進氣口和雙排氣口通風、高壓驅動風扇、銅質散熱翅片組、導熱管等散熱設計，進一步帶出機身內的熱量，實測在玩家經常接觸的 WASD 區域，溫度僅在 45℃ 左右，而掌託的位置由於金屬的特性，甚至能夠感知到冰涼的觸覺。

最後，別忘了 ALIENWARE Area-51m 還是一臺支持完全自主升級的遊戲本，通過戴爾的 DGFF 技術，ALIENWARE Area-51m 將雙芯片與主板獨立開來，支持 CPU 和 GPU 的自定義升級，例如你可以自行更換至其他支持 Z390 芯片組的 CPU，也可以通過板載顯卡模塊更換功能來升級 GPU，而這也意味著你的遊戲本將不再和「落伍」掛鉤，通過更換最新的 CPU 和顯卡，你的 Area-51m 將具備與時俱進的性能，這也解決了遊戲本最大的槽點——不夠完善的擴展性。

整合一體機，一直是筆記本既定的方式，其實是一定程度上保障了其穩定性，而以往我們在絕大多數品牌產品上最多也只能是「升級內存」「加裝硬盤」，為了防止人為拆解，保修政策依然還是自主拆機之後不再享受官方的保修政策。而在ALIENWARE Area-51m 見到這樣連CPU和GPU均可更換的理念，確實很大膽很有前瞻性，讓你真正意義上實現自由DIY筆記本。

當然，還少不了這些秀肌肉的「黑科技」

之前在體驗 ALIENWARE 17R5 和 ALIENWARE m15 時，我就對外星人自帶的 ALIENWARE COMMAND CENTER 積累了不少好感，主要是因為這個軟件無論是界面的 UI 設計還是豐富的功能都做得十分精緻，特別是針對超頻部分的功能，以往玩家要對電腦進行超頻的話，需要跑到 BIOS 裡面設置，對於不懂基本知識的玩家來說難免存在過高的門檻。而 ALIENWARE COMMAND CENTER 通過可視化的 UI 和簡易的表現方式，將複雜的超頻步驟儘可能簡化，普通玩家也能一鍵上手，實在是贊！

另一方面，ALIENWARE COMMAND CENTER 還支持更改 ALIENFX 情緒同步光效，而除了實現常規的多燈效聯動和特定遊戲特定燈效之外，ALIENWARE Area-51m 還藏著另外一個大殺器。

沒錯，它就是「情緒同步反饋」：通過感知遊戲過程中的關鍵情節或者燈光效果，ALIENWARE Area-51m 機身的燈效能夠產生同步的效果，讓玩家感官與遊戲達到 100% 的同步率。舉個例子，在玩《地鐵：離去》的時候，當我打開武器的燈光時，鍵盤的燈效會伴隨著一同亮起，而當我的血條低至一定數值，或者遊戲角色面臨危險時，Area-51m 機身的燈效會相應地泛起閃動的紅色光效，就好像你和遊戲角色有著一樣的感知一樣，十分炫酷，通過下面這張 GIF 圖來感受一下！

此外，ALIENWARE Area-51m 還繼承了前幾代的眼球追蹤技術，藉助 Tobii 眼球追蹤裝置，Area-51m 衍生出了許多玩法，其中基礎的比如它會檢測你在不在電腦前，不在的話自動調暗屏幕亮度節省電量，如果檢測到你在電腦前的話，又會將屏幕自動調亮；此外，還可以利用「眼神」來「觸控電腦」。簡單來說就是依靠深根在系統層面的 Tobii 眼球追蹤裝置，你可以利用眼神來切換多任務頁面，當你按下 ALT 並同時看著某一個任務頁面時，它將自動切換到相應的頁面，省去了手動操作鼠標的步驟。更甚者，在 Tobii Experience 應用中，列出了支持眼球追蹤的遊戲，例如《刺客信條：奧德賽》《孤島驚魂》等等。

既是致敬過去，也是瞻望未來

17 年前，ALIENWARE 憑藉著一臺 Area-51m 叩開了「遊戲本的大門」，而如今全新的 Area-51m 將外星人最新的科技貫徹地淋漓盡致，稱之為擁有強悍臺式機性能的未來 PC 也不為過。

在這幾天的體驗之後，我漸漸明白了 ALIENWARE Area-51m 之所以取了這樣一個名字的原因：在它身上，諸如科幻的外觀設計、散熱設計、史無前例的自由擴展性等都讓我看到了外星人對於為玩家創造極致體驗的執著追求，而這些特性不正是那些追求極致體驗的玩家們一直嚮往的嗎？

另外，ALIENWARE 還為玩家提供了多款專業外設：AW959 專業級電競鼠標、擁有 12000 DPI 與 5 檔飛敏設置、全鍵無衝等精緻設計。AW988 專業電競耳機，其 40mm 驅動單元的設計與 20Hz~20KHz 頻響範圍，確保音質得到最佳還原。AW3418DW 遊戲顯示器，支持NVIDIA® G-Sync™技術。眾多的配件加持，能讓你更加全情地投入遊戲世界。
