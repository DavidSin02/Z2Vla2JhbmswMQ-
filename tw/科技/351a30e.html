德生PL-330收音機：初步印象，功能概述和操作


文：Jaap de Goede


作者在沒有英語手冊的情況下操作德生 PL-330


介紹


受父親的啟發，我喜歡聽無線電廣播已經有60個年頭了。這包括收聽短波傳輸。在我的收藏中，有很多收音機，包括基於SDR的接收器。德生PL-330是具有SSB和數字讀數功能的短波便攜式收音機。


我想分享一下我在PL-330上的操作經驗，並且在整個過程中，我將把這個收音機與我的收藏中的類似收音機進行比較：Eton Satellit，XHDATA D-808和CountyComm GP5-SSB（又名Tecsun PL-365） 。


需要明確的是，此處使用的PL-330是中國國產版本，可能於2020年7月生產。在撰寫本文時，尚無英文手冊。同時，德生PL-990的英文手冊有助於弄清楚PL-330的許多功能。較新的製造批次可能包含其他固件，這可能會更改操作方式。


大小和聲音


PL-330的體積約為18立方英寸，小於20立方英寸。它很容易適合我的牛仔褲的口袋。20立方英寸（非正式）將其歸類為Ultralight DX收音機。四個中最小的收音機是具有14立方英寸的GP5-SSB。Eton Satellit是這四臺收音機中最大的一個，這些收音機的大小可以由眼睛決定，但也可以由耳朵決定。


PL-330具有數字音量控制，顯示屏上會顯示音量。矛盾的是，其他三個收音機具有模擬音量控制。所有收音機均具有用於立體聲耳機的3.5毫米插座。PL-330的一個不錯的功能是FM立體聲解碼器只能與插入式耳機配合使用。其他FM仍然處於單聲道狀態。我認為這是一個非常聰明的功能，因為如果FM立體聲接收效果不好，為什麼立體聲解碼器會降低揚聲器的單聲道聲音？


個人音頻質量排名（從最佳到最差）：

1. Satellit

2. PL-330和D-808

3. GP5-SSB


電源和電池


PL-330配有1000 mAh的BL-5C電池。電池可以通過微型USB端口在收音機中充電。無論收音機是否打開，顯示屏都會永久顯示電池狀態。USB使從任何外部USB充電器或外部USB電池充電非常容易。這裡是四臺收音機的功率的快速比較：



我沒有運行時數據。我只是在收音機的顯示屏上指示時充電，並攜帶備用USB電池為收音機充電。



天線和背光


所有四臺收音機都具有可擴展的鞭狀天線（您還會期望什麼）。另外，所有設備都有一個3.5毫米的插座，用於外接天線。唯一帶有天線衰減器開關的收音機是Satellit。


所有四個收音機都有背光。但是其他收音機都沒有的功能是PL-330能夠在“始終打開”或“自動關閉”之間切換背光。只需按住5按鈕即可在兩種模式之間切換。為了進行比較，Satellit具有所有四個中最好的照明顯示，而D-808太亮了。


支持的無線電頻段


PL-330支持以下無線電頻帶：

  • 長波（153-513 kHz）
  • 中波（520-1710 kHz或522-1620 kHz）
  • 短波（1711-29999 kHz）
  • FM廣播（64-108 MHz）


無線電頻段和解調選擇


要啟用或禁用LW頻段，必須在無線電關閉電源時長按2按鈕。當無線電打開電源並且啟用了LW時，您可以通過短按MW / LW按鈕在LW和MW之間進行選擇。


關閉電源後，長按3按鈕可以切換美國或世界其他地區的MW頻段計劃。無線電開啟後，您可以通過短按MW / LW按鈕在MW之間進行選擇。


要選擇區域FM頻段計劃，您應該在收音機關閉時長按0按鈕。收音機打開後，可以通過短按FM / ST選擇FM波段按鈕。


收音機打開後，可以通過短按<或>按鈕選擇SW波段。


在LW中，MW和SW頻段的解調可以是AM，SSB和AM同步（SYNC）。短按SSB按鈕可在AM和SSB之間切換。長按SSB按鈕可在AM / SSB和AM同步模式之間切換。


SW頻段內的一個不錯的功能是您可以快速跳到預定的廣播或HAM頻段。在AM和SSB模式下，您可以按<或>按鈕跳至所需的頻段。可以選擇以下頻段並在顯示屏中指示：


•AM（廣播）波段：120m，90m，75m，60m，49m，41m，31m25m，22m，19m，16m，13m，11m


•SSB（HAM）頻段：160m LSB，80m LSB，60m，LSB，40m LSB，30m USB，24m USB，20m USB，17m USB，15m USB，12m USB，10m USB


手動調整和步進


一個（數字）TUNING撥盤可用於任何收音機波段中的所有調諧操作。它僅在頻段上調和調低，沒有其他功能。（與其他三個帶多功能撥盤的收音機相比，使用PL-330進行調諧非常容易。）根據所選頻段和解調情況，可以通過“ STEP”按鈕更改調諧步進。


在用於AM解調的LW和MW頻段中，可以通過短按STEP按鈕在1 kHz和10/9 kHz之間切換步進。


在用於AM解調的SW中，可以通過短按STEP按鈕在1 kHz和5 kHz之間切換步進。


在用於SSB和同步模式的SW波段中，可以通過短按STEP按鈕在10 Hz，1 kHz和5 kHz之間切換步進。我的其他三臺便攜式收音機，但僅實際頻率的正負1 kHz（以10Hz為步長）。PL-330在SSB中具有連續微調功能；您可以以10 Hz為步長上下調諧整個SW頻段。但是，我注意到如果您快速轉動TUNING撥盤，步長可以從10 Hz更改為50 Hz。


在FM頻段中，可以通過短按STEP按鈕在100 kHz和10 kHz之間切換步進。


帶寬設定


對於AM和SSB，您可以通過按AM BW按鈕，將VOLUME撥盤轉到所需的帶寬並再次按AM BW按鈕來更改帶寬。


•LW / MW頻段，AM帶寬選擇：2.5 kHz，3.5 kHz，5.0、9.0 kHz kHz

•SW波段，AM帶寬選擇：2.5 kHz，3.5 kHz，5.0 kHz

•SSB帶寬選擇：0.5 kHz，1.2 kHz，2.2 kHz，3.0 kHz，4.0 kHz


輕鬆調整模式或ETM和內存


我決定購買PL-330是因為我在GP5-SSB（PL-365）方面的良好經驗和不良經驗。PL-365超級便攜，我認為ETM非常適合偶爾短波收聽。但是，如果沒有直接輸入頻率，GP5-SSAB就是不時操作的噩夢。現在，PL-330具有增強版的ETM：ETM +。沒有其他電臺具有這種功能，我認為ETM +很棒。


ETM +提供24個ETM庫，這些庫是根據小時自動選擇的。您可以每小時按住ETM按鈕來啟動（新的或更新的）短波掃描。它將顯示“ E”以及24小時格式的小時，例如E00到E23，具體取決於時間。當您快速按下ETM按鈕時，它會根據時間調用存儲庫。例如，當您在15：24h快速按下ETM按鈕時，它將調出庫E15。然後，您可以通過轉動TUNING撥盤來選擇存儲在存儲區中的存儲位置。



要在ETM和調整模式之間切換，請按ETM按鈕。


除了ETM存儲器外，PL-330還具有很多存儲器，但缺少alpha標籤。我真的很難記住我在那些記憶中的記憶。我幾乎不使用它們。四個帶有alpha標籤的唯一一個是Eton Satellit，這使事情變得容易得多。


通過使用自動調整和存儲（ATS）功能，可以將內存自動填充到與ETM庫分開的庫中。全部四個收音機都有ATS，我認為這僅適用於FM。它的工作原理如下：


1.按FM / ST。選擇FM波段

2.按住FM / ST。啟動ATS

3.使用TUNING撥盤從存儲的FM電臺中選擇


要在內存和頻率調整模式之間切換，只需按VM / VF按鈕。


BUG或功能所限


使用“顯示”按鈕，您可以在以下之間更改顯示的右上部分：


•信號強度

•時鐘

•預設（僅在VM模式下）

•警報時間


幾秒鐘後，顯示屏始終返回信號強度。除非長按DISPLAY按鈕選擇了時間。但是，在進行任何操作（包括音量更改）後，顯示屏將恢復信號強度。我不確定這是功能還是錯誤。


使用耳機並更改音量時，有時聲音會被切斷。通過再次更改音量，它又回來了。似乎是一個BUG，而不是功能。


我嘗試了AM同步模式。就像在衛星上一樣：可憐。我建議隱藏此模式或使其正常工作。


通用串行總線


用USB電纜將收音機連接到PC時，其操作系統不會顯示任何連接信息。如果收音機除了充電之外還具有USB邏輯，那麼連接信息就會顯示出來。我認為沒有辦法通過USB更新固件。


隱藏功能


沒有英文手冊，甚至沒有中文手冊，我也聽不懂所有功能。


收音機電源關閉時：

•按住VF / VM將顯示所有項目。

•按住8可以在“ HH：MM”和“ HH：MM：SS”之間切換顯示時鐘

•按住MW / LW波段中的3，在內部鐵氧體和外部鞭狀天線之間切換

•按住Enter鍵顯示“ dEL ALL”，可能刪除所有（全部除外）？

•按住M顯示數字，不知道。

•按住0表示“ PO []”，不知道。


似乎可以組合顯示固件版本。


缺少功能


歡迎使用RDS顯示。即使PL-330與D-808和Satellit共享無線電芯片，PL-330也不在其顯示器上顯示FM電臺數據系統（RDS）。背光按鈕非常適合在黑暗中進行操作。（儘管您很快就會習慣按鈕的佈局。）


我沒有錯過的功能


四臺收音機都沒有DAB +或HD功能。對於DAB +接收，我使用Sony XDR-S41D，但在歐洲無法接收高清廣播。DAB +和高清廣播可能會使廣播方式變得更昂貴，而且無論如何，我還是希望擁有良好的軟件性能。


升級固件的可能性將很方便。但是我瞭解成本的限制以及無線電廣播的負擔。


Satellit和D-808支持空中頻段（108-135 MHz）。偶爾聽一個頻道沒關係，但是忘了頻道掃描。為此，我有Bearcat UBC XLT125 VHF / UHF掃頻儀。


結論


就像我剛開始時一樣，我偶爾也是短波聽眾。我真的無法說出這四臺收音機之間收音機性能的差異。當然，最好的收音機是隨身攜帶的收音機。由於其尺寸大，使用單個調諧轉盤易於操作以及具有ETM +之類的功能，PL-330可能是目前最好的便攜式短波接收器（對我而言？）。感謝德生！


Jaap de Goede


2020年10月
