在Android中的工具欄中添加菜單

2020-11-14
| 科技

菜單對於將多個動作放入其中非常有用，尤其是那些使用頻率較低並且不需要自己的按鈕的動作。 菜單顯示在Android的工具欄上。 通常使用支持工具欄，但如果沒有相應的類庫，下面的方法也可以實現。

1.製作菜單xml

  • 右鍵單擊res文件夾，然後選擇“新建”>“Android資源文件”。
  • 鍵入main_menu作為文件名。
  • 選擇資源類型的菜單。

粘貼以下內容作為啟動器。

<?xml version="1.0" encoding="utf-8"?><menu xmlns:android="http://schemas.android.com/apk/res/android" xmlns:app="http://schemas.android.com/apk/res-auto"> <item android:id="@+id/action_add" android:icon="@drawable/ic_add" app:showAsAction="ifRoom" android:title="Add"> </item> <item android:id="@+id/action_settings" app:showAsAction="never" android:title="Settings"> </item></menu>

您可以右鍵單擊res並選擇New image asset以創建ic_add圖標。

2. 初始化菜單

在您的activity中添加以下方法

@Overridepublic boolean onCreateOptionsMenu(Menu menu) { MenuInflater inflater = getMenuInflater(); inflater.inflate(R.menu.main_menu, menu); return true;}

3.處理菜單點擊

同樣在您的Activity中，添加以下方法：

@Overridepublic boolean onOptionsItemSelected(MenuItem item) { // Handle item selection switch (item.getItemId()) { case R.id.action_add: addSomething(); return true; case R.id.action_settings: startSettings(); return true; default: return super.onOptionsItemSelected(item); }}
