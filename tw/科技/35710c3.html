5種方法教你如何跨多個PDF文件查找文本內容

2020-10-29
| 科技

PDF是絕大多數人共享文檔最喜歡的格式。主要是因為它保留了其中所有的文本格式和圖形。但PDF格式的唯一缺點是，您無法像使用其他文件一樣輕鬆地在所有這些文件中搜索文本。

但是，的確有幾種方法可以讓您執行PDF搜索操作，以便您可以在計算機上一次在多個PDF文件中搜索特定關鍵字。

以下指南教您如何做到這一點。

如何使用 Acrobat Reader 搜索多個PDF文件

搜索PDF或多個PDF文件的最流行的方法是使用計算機上的 Acrobat Reader 軟件。以下是執行方式。

  1. 在計算機上打開 Acrobat Reader。
  2. 單擊編輯菜單，然後選擇高級搜索。
  3. 選擇“所有PDF文檔”，然後選擇您的PDF文件夾。
  4. 在輸入框中輸入搜索詞。
  5. 點擊搜索按鈕。
  6. 在屏幕上的提示中點擊允許。

該軟件將在指定文件夾中的所有PDF文件中搜索給定術語。您將在搜索窗格中直接看到結果。

在多個PDF文件中搜索特定的搜索詞

大多數PDF閱讀器可讓您搜索PDF文件中所需的任何文本。

這是因為PDF是可掃描的文檔，並且與 Microsoft Word 文件一樣，它們的字符也可以由您的系統識別。

不過，使用PDF文件不容易做到的是，您無法一次在多個文件中搜索特定術語。

按下 Control + F 鍵只會搜索在軟件中打開的當前文檔，而不會搜索任何其他PDF文件。

如果您有很多PDF文件，並且正在尋找特定的文件，最好的選擇是使用 Acrobat Reader 的高級搜索功能。它使您可以在計算機上一個位置的所有PDF文件中查找特定的搜索詞。

使用 Acrobat Reader 的整個過程如下：

如果您尚未在計算機上安裝 Acrobat Reader，請下載並安裝它。

啟動軟件，您會看到最近的文件列表。您要查找的選項位於“編輯”菜單中。單擊頂部的“編輯”菜單，然後選擇“高級搜索”選項。或者，您可以按 Shift + Ctrl +F。

在接下來的屏幕上，將選項設置為以下內容：

您想在哪裡搜尋？ —— 選擇“所有PDF文檔”，然後使用下拉菜單選擇保存多個PDF的文件夾。

您想搜索什麼單詞或短語？ —— 在PDF文件中輸入要搜索的單詞。

您可以使用其他選項來自定義單詞的搜索方式，例如在 “區分大小寫” 複選框上打勾，以使搜索查詢區分大小寫，依此類推。

使用 Windows 搜索在PDF文件中進行搜索

也可以使用 Windows 計算機上的默認搜索選項來搜索PDF文件。

轉到控制面板>索引選項，然後單擊高級。

在以下屏幕上選擇“文件類型”選項卡，然後在列表中查找 pdf。勾選方框為 pdf。然後啟用 “索引屬性和文件內容” 選項，然後單擊 “確定”。

現在是時候將PDF文件夾添加到索引列表中了。為此，打開相同的“ 索引選項” 對話框，然後單擊 “修改”。然後選擇您的PDF所在的文件夾，然後單擊“確定”以添加為PDF編制索引的文件夾。



Windows 完成對PDF及其內容的索引後，您就可以一次在多個PDF文件中搜索文本。


使用 SeekFast 搜索PDF文件

SeekFast 還使您可以輕鬆搜索各種文件類型（包括PDF）中的關鍵字。運作方式如下。

  • 在計算機上下載並安裝該軟件。
  • 啟動軟件，在頂部的搜索框中輸入搜索詞，單擊瀏覽按鈕以選擇您的PDF文件夾，最後點擊搜索。


它將在所選目錄中的所有PDF文件中搜索關鍵字。

使用 Foxit Reader 執行PDF搜索

Foxit Reader 還具有高級搜索功能，您可以使用它在多個PDF文件中查找所需內容。

獲取計算機上安裝的軟件，然後啟動它。

單擊右上角菜單中搜索框旁邊的搜索圖標。它會打開高級搜索選項。

在以下屏幕上，從第一個下拉菜單中選擇您的PDF文件夾，在搜索框中輸入搜索詞，如果要應用其他過濾器，請選中該標記，最後單擊該“搜索”按鈕。

您的搜索結果現在應該在屏幕上。

如何使用 UltraFinder 搜索PDF文件

UltraFinder 是Windows的高級搜索工具，也可用於在PDF文件中搜索文本。

在 Windows 計算機上安裝並啟動該工具。

如下設置選項，以便搜索PDF文件的內容。然後點擊綠色的按鈕開始搜索。

Find matches by — 文件內容

Find what — 在這裡輸入您的搜索詞

Find where — 添加PDF所在的目錄

In files/types — *.pdf

它將列出在其中找到您的搜索詞的所有PDF文件。
