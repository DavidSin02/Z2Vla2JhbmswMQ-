鍵帽有點甜哦 Ducky 3108S機械鍵盤圖賞簡評

2020-10-29
| 科技

上個月外設天下為大家帶來了Akko X Ducky ZERO 3108機械鍵盤的評測，這款鍵盤在鍵帽工藝上有很大進步，在外觀上使用了更具質感和美感的金砂黑配色，辨識度頗高，可謂是“黑出新意”。在這之後，Akko X Ducky又推出了3108S機械鍵盤，也就是今天的主角。3108S與3108相比增加了單色藍光，並且配備了極具特色的PBT鍵帽，從外觀上很容易讓人聯想到經典的三明治餅乾奧利奧。我們現在就為大家帶來這款鍵盤的圖賞簡評。

Akko X Ducky ZERO 3108S機械鍵盤包裝

Akko X Ducky ZERO 3108S機械鍵盤包裝

Akko X Ducky ZERO 3108S機械鍵盤的包裝與3108完全一致，都是Ducky最新的家族式包裝，就連包裝正面的產品型號都是3108，所以如果把3108和3108S放在一起，不看銘牌貼紙完全無法區分。

Akko X Ducky ZERO 3108S機械鍵盤包裝

Akko X Ducky ZERO 3108S機械鍵盤附件

Akko X Ducky Zero 3108S機械鍵盤附送了防塵罩、備用鍵帽、鴨頭拔鍵器、說明書和質保卡，由於3108S的外包裝和3108沒有區別，所以筆者特別檢查了說明書，好在說明書是明顯不同的，對3108S的燈光調節進行了介紹，兩者之間的一大差別就是燈光方面。

Akko X Ducky ZERO 3108S機械鍵盤外觀

Akko X Ducky ZERO 3108S機械鍵盤外觀

Akko X Ducky Zero 3108S機械鍵盤的外殼採用了跟3108同樣的金砂黑配色，它是以黑色為底，摻雜金砂進行噴漆，使得鍵盤表面有細目顆粒反光狀態。這款金砂黑明顯降低了產品的塑料感，比純黑色更具質感以及美感。

Akko X Ducky ZERO 3108S機械鍵盤外觀

Akko X Ducky Zero 3108S機械鍵盤與3108的最大外觀特點就是鍵帽了，它採用了特製的PBT二色熱昇華鍵帽，黑白配色。其中鍵帽的頂面是白色透光的，側面均為黑色不透光的，它的字符通過熱昇華工藝印製鍵帽上，非常有特點，是跟大家喜歡吃的三明治餅乾奧利奧神似，從視覺效果上給人一種甜甜的感覺，好想扭一扭，舔一舔，再泡一泡。因此它被大家戲稱鍵盤中的“奧利奧”也算是名至實歸了。

Akko X Ducky ZERO 3108S機械鍵盤外觀

Akko X Ducky ZERO 3108S機械鍵盤外觀

從側面看這款這鍵盤觀賞性極佳，它的鍵帽給人一種將鍵盤一刀切開的感覺，對於看慣了塗層、雙色注塑和二色成型鍵帽的用戶來說，這款鍵帽還是很有新意的，筆者在看到這款鍵盤的新聞時就被它深深地迷住了，想趕快見到實物。

Akko X Ducky ZERO 3108S機械鍵盤底面

Akko X Ducky ZERO 3108S機械鍵盤撐腳

Akko X Ducky ZERO 3108S機械鍵盤撐腳

Akko X Ducky ZERO 3108S機械鍵盤撐腳

Akko X Ducky ZERO 3108S機械鍵盤線材接口

Akko X Ducky Zero 3108S機械鍵盤在防滑腳墊、3向出線口、二段式撐腳和線材接口方面與3108是一致的，它的防滑性不錯，線材長約1.8m，材質為橡膠線，粗細適中，比較柔軟，它配置了屏蔽磁環但接口並沒有採用鍍金工藝。

Akko X Ducky ZERO 3108S機械鍵盤外殼

Akko X Ducky ZERO 3108S機械鍵盤外殼

Akko X Ducky ZERO 3108S機械鍵盤圖案

Akko X Ducky Zero 3108S機械鍵盤的外殼右下角印有Akko以及Ducky的logo，與3108不同的是空格鍵側面的ZERO圖案，不光是在圖案的樣式上有所區別，3108S更是把圖案的顏色從白色更改為更貼近金砂黑的金色，跟產品的整體風格更加貼合。

Akko X Ducky ZERO 3108S機械鍵盤鍵帽

Akko X Ducky ZERO 3108S機械鍵盤鍵帽

Akko X Ducky ZERO 3108S機械鍵盤鍵帽

Akko X Ducky Zero 3108S機械鍵盤的鍵帽字符采用了熱昇華工藝，它的字體比較美觀，黑色與白色的鍵帽表面對比十分明顯，看起來非常的清晰。不少玩家很喜歡熱昇華工藝，因為它能提供美觀的字符、平整的手感和更長久的使用壽命，當然熱昇華鍵帽的成本也是比較高的，而這款酷似甜品的熱昇華PBT鍵帽，製作工藝要更復雜，成本自然也更高。

Akko X Ducky ZERO 3108S機械鍵盤鍵帽

Akko X Ducky ZERO 3108S機械鍵盤鍵帽

這款鍵帽的表面細膩平整，很有質感，手感出眾，但是因為平整的關係，它的摩擦力不大，觸感比較光滑。從背面看，這款鍵帽做工紮實，內部乾淨整潔，邊角也沒有毛邊和粗糙的現象，它的加強筋圖案也非常有特點，從透光性來說，鍵帽的整個頂部都是透光的，但實際上並沒有特別透光，相信也是考慮到熱昇華字符的背光效果才這麼設計的。

Akko X Ducky ZERO 3108S機械鍵盤軸體

Akko X Ducky ZERO 3108S機械鍵盤軸體

Akko X Ducky Zero 3108S機械鍵盤在3108的基礎上增加了藍色的背光，因此採用了軸體倒裝的方式，筆者手中的這款為cherry青軸，大鍵位為衛星軸設計，在手感上與3108一致，按鍵穩定，沒有明顯的晃動，大小鍵位的手感很接近，一致性較好。

Akko X Ducky ZERO 3108S機械鍵盤燈光

Akko X Ducky ZERO 3108S機械鍵盤燈光

Akko X Ducky ZERO 3108S機械鍵盤燈光

Akko X Ducky ZERO 3108S機械鍵盤燈光

Akko X Ducky ZERO 3108S機械鍵盤採用了藍色背光，因為它的鍵帽頂面是全透的，所以視覺效果上要比單純的字符透光更加誇張，在白天和燈光環境中，它的亮度不錯，但是受到相機白平衡的影響，拍攝樣張有些偏紫。

Akko X Ducky ZERO 3108S機械鍵盤燈光

Akko X Ducky ZERO 3108S機械鍵盤燈光

在黑暗環境中拍攝就沒有白平衡問題，它的視覺感受很奇妙，有些滿天星辰的感覺，彷彿置身在夜空中。

Akko X Ducky ZERO 3108機械鍵盤有常亮、呼吸、波浪、跑馬燈、雨滴、單點亮、爆炸和極光這幾種燈光模式，燈光亮度為7階調節，從燈光效果來說，它的亮度不算均勻，能夠看出LED燈的位置，好在它的燈光流暢度還是不錯的。

Akko X Ducky ZERO 3108S機械鍵盤燈效

Akko X Ducky ZERO 3108S機械鍵盤燈效

Akko X Ducky ZERO 3108S機械鍵盤燈效

Akko X Ducky ZERO 3108S機械鍵盤燈效

從整體視覺效果來講這款鍵盤是非常有特點的，筆者是非常喜歡它酷似奧利奧的外觀，但是打開燈光後，這款鍵盤原來甜甜的感覺就被很像滿天星辰的燈光所取代，別有一番風味，對於追求完美的用戶來說，鍵帽的透光性和燈光的均勻度還不夠完美，這也是受到PBT材質的影響。當然鍵帽在美觀度和手感方面是非常出色的，相信對鍵帽的顏值和手感都有追求的玩家們一定會喜歡這款產品的。

目前Akko X Ducky ZERO 3108機械鍵盤的售價為599-649元，喜歡這款鍵盤的玩家可以多多關注一下哦~
