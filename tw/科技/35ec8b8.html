佳能正式宣佈1dx3開發，發佈RF85/1.2DS、70-200/2.8

2020-10-29
| 科技

佳能宣佈EOS-1D X Mark III開發，將會在10月24日至26日的PHOTOPLUS上進行展示。

新品特性：

1）全新CMOS，更高感光度；

2）可以拍攝10bit HEIF照片；

3）4K / 60p / YCbCr 4:2:2 / 10bit / Canon Log以及RAW格式進行內錄；

4）AF/AE追蹤功能時，16fps（光學取景方式）20fps（實時取景方式）連拍；

5）使用RAW格式進行連拍時，新機型的一次最多連拍張數將提升5倍；

5）新的自動對焦感應器，其中心像素區域的有效像素數量比前代機型增加約28倍；

6）深度學習技術，將進一步提升對焦追蹤能力，在實時取景拍攝方式下，自動對焦區域將擴展至圖像感應器約100%（垂直方向）以及約90%（水平方向），525個對焦框。

7）無線文件傳輸器WFT-E9速度提高2倍；

8）按鈕背光；

9）相機存儲卡將由CFast2.0改為CFexpress

10）電池效能更高，能拍攝更多照片。

佳能正式發佈RF70-200mm F2.8 L IS USM，與EF 70-200mm f/2.8 L IS III USM相比在長度和重量上分別大幅縮減約27%和28%，1片玻璃模鑄非球面，3片UD鏡片，1片超級UD鏡片，雙Nano USM對焦馬達。最近對焦距離約0.7m，放大倍率在70mm端以及200mm端分別約為0.11X和0.23X，5檔防抖效果，濾鏡規格為77mm。鏡頭最大直徑及長度分別為約89.9mm及約146.0mm，重量約1070g。

在防護性方面，由於採用外變焦類型，該鏡頭為此特別設計了通風路線，抑制水汽吸入，並採用特殊防塵材料，以阻隔灰塵滲入。11月上市。

佳能正式發佈RF85mm F1.2 L USM DS，採用變跡濾鏡設計，中心的透射率較高，邊緣的透射率較低，因此可實現平滑的焦外虛化效果。鏡頭光學結構為9組13片。採用了BR鏡片及UD（超低色散）鏡片的組合有效抑制色像差。

9片圓形光圈葉片設計，最近對焦距離為約85cm，最大放大倍率為0.12X，濾鏡規格為82mm。鏡頭最大直徑及長度分別為約103.2 mm及117.3 mm，重量約1195g。12月上市。

1989年佳能EOS-1相機誕生。今年佳能則迎來了旗艦級機型EOS-1系列誕生30週年紀念。EOS-1基於當時最初的EOS產品理念，即根據用戶的需求實現自動化，其設計以高水準的技術、一流的功能、耐久性、堅固性和可靠性，迴應了專業用戶的嚴苛需求。

2001年，佳能發佈了旗下首款旗艦級數碼單反相機EOS-1D。從那時起，該相機及其後續型號一如既往地提供了高圖像畫質、高速連拍能力，以及藉助佳能高水準技術所實現的諸多功能，贏得了新聞、體育、商業和生態攝影等領域一線專業人士的廣泛讚譽。

在過去幾十年裡，佳能還持續為報道體育賽事的記者提供幕後支持。秉承“捕捉並保存來自優秀運動員和激烈競爭中那些決定性的、激動人心的時刻”的美好願景，佳能在多項體育賽事中都設有專業服務中心，以保證攝影師能夠隨時準備捕捉賽場上的珍貴瞬間。更重要的是，這些專業人士的經驗和反饋也有助於為後續佳能產品的開發帶來積極影響，使佳能能夠持續提供更為可靠的相機技術。

隨著EOS-1的推出，一系列高規格技術和一流功能得到應用，並且這些功能還陸續應用到甚至包括入門級型號的更多產品線中，佳能也因此滿足了不同類型用戶的需求。展望未來，佳能將繼續推進其核心光學技術的發展，不斷突破具有競爭力和可靠性的相機以及鏡頭的製造極限，以支持攝影和視覺表現領域的進一步發展。

正在開發中的EOS-1D X Mark III將在美國大型攝影展會PHOTOPLUS上的佳能展臺進行參考展示。該展會將於2019年10月24日至26日，在美國紐約舉辦。

正在開發中的EOS-1D X Mark III預計將具備以下功能：

1. 新開發的CMOS圖像感應器以及影像處理器，可實現更高畫質和更高的連拍速度

（1）佳能正在開發中的CMOS圖像感應器以及影像處理器將實現比前代機型更高的靜態照片拍攝感光度。新機型將支持以10bit的HEIF格式2保存靜態照片，同時還將支持以4K / 60p / YCbCr 4:2:2 / 10bit / Canon Log以及RAW格式進行視頻機內錄製。

（2）無論採用光學取景還是實時取景方式拍攝，都可實現AF/AE追蹤功能，並實現比前代機型更高的連拍速度。當啟動AF/AE追蹤功能時，該相機將實現最高約16張/秒（光學取景方式）以及20張/秒（實時取景方式）連拍表現。此外，當使用實時取景方式拍攝時，拍攝者能夠選擇使用機械快門或電子快門進行拍攝。

（3）相對前代機型，當使用RAW格式進行連拍時，新機型的一次最多連拍張數將提升5倍。

2. HEIF（High Efficiency Image File高效率圖像文件）格式。相對JPEG格式，可實現更高的動態範圍和色彩還原表現。

2. 藉助新開發的自動對焦感應器以及自動對焦算法，將實現更精準的自動對焦表現

（1）將採用新型的自動對焦感應器，其中心像素區域的有效像素數量比前代機型增加約28倍。當通過光學取景器拍攝時，自動對焦感應器獲得的高分辨率信號將使高對焦精度成為可能。此外，進一步擴展的自動對焦亮度範圍將適用於光學取景拍攝以及實時取景拍攝。

（2）當使用光學取景或實時取景方式拍攝時，新的自動對焦算法將帶來更穩定的對焦表現，同時由於新算法應用了深度學習技術，將進一步提升對焦追蹤能力。

（3）在實時取景拍攝方式下，自動對焦區域將擴展至圖像感應器約100%（垂直方向）以及約90%（水平方向）。此外，當相機自動選擇時，最多525個對焦框可以用於自動對焦。

3. 提升數據傳輸能力以滿足專業用戶需求

（1）正在開發中的無線文件傳輸器WFT-E9（無線認可審批中）配合EOS-1D X Mark III使用時，將實現比前代機型WFT-E8（2016年4月上市）快2倍以上的無線圖像數據傳輸速度3。此外，WFT-E9還可配合EOS C500 Mark II電影攝影機（將於2019年12月底上市）使用。

2019年10月24日，佳能（中國）有限公司宣佈推出兩款RF系列鏡頭新品，分別為RF大三元遠攝變焦鏡頭RF70-200mm F2.8 L IS USM和具有平滑散焦的專業人像鏡頭RF85mm F1.2 L USM DS。作為兩款擁有大光圈的佳能L級鏡頭，RF70-200mm F2.8 L IS USM及RF85mm F1.2 L USM DS充分發揮EOS R系統的大卡口直徑以及短後對焦距離優勢，達到令人期待的高光學素質。RF70-200mm F2.8 L IS USM顯著實現了小型化，鏡身與EF 70-200mm f/2.8 L IS III USM相比在長度和重量上分別大幅縮減約27%和28%，進一步提升便攜性和機動性，並在佳能鏡頭中首次應用雙Nano USM對焦馬達，實現安靜、高速的對焦表現。該鏡頭的推出也標誌佳能1RF卡口大三元鏡頭系列已構築完整。RF85mm F1.2 L USM DS則是RF系列中首款支持平滑散焦的鏡頭，其柔和的焦外虛化效果令人像作品擁有迷人的視覺表現力。

EOS R系統大卡口直徑和短後對焦距離特性令鏡頭設計擁有更多可能，這在佳能RF70-200mm F2.8 L IS USM上得到充分體現。該鏡頭擁有EF卡口同規格鏡頭難以實現的小型化設計，與EF 70-200mm f/2.8 L IS III USM相比在長度和重量上分別大幅縮減約27%和28%，不僅方便放入攝影包中，而且也顯著減輕了用戶在長時間攜帶和拍攝時的負擔。該鏡頭的大部分配重位於相機機身側，重心更接近攝影師，即使在延伸鏡頭筒的遠攝端，也能保持良好的握持平衡。此外，其結構緊湊也能減輕給周圍環境和拍攝對象帶來壓迫感。

佳能RF70-200mm F2.8 L IS USM所擁有的70mm至200mm焦段，可以兼顧體育題材、生態攝影、人像、風光等多種題材影像創作。同時全部焦段內的F2.8恆定最大光圈以及9片葉片圓形光圈結構，也能獲得美麗柔滑的大幅虛化效果，同時能使拍攝獲得更高的快門速度，特別是在暗光線環境下能進一步提升拍攝成功率。

作為佳能配備F2.8恆定最大光圈鏡頭的新一代產品，佳能RF70-200mm F2.8 L IS USM鏡頭得益於EOS R系統的大口徑卡口和短後對焦距離優勢，並通過應用多種佳能先進的光學技術，從而在整個變焦範圍內，實現了從畫面中心到邊緣畫質的再一次提升。該鏡頭採用13組17片光學結構，藉助1片玻璃模鑄非球面鏡片和1片UD（超低色散）非球面鏡片，實現了球面像差校正和變焦鏡頭小型化。高效排列的3片UD鏡片和1片超級UD鏡片，以大幅校正色像差，減少從畫面中心到邊緣的色暈，實現了高畫質。此外，該鏡頭還應用了佳能SWC亞波長結構鍍膜。SWC鍍膜具有出色的抗反射性，特別是對於入射角較大的光，可以減少在具有大麴率鏡片的邊緣發生眩光和鬼影的可能性。

佳能RF70-200mm F2.8 L IS USM鏡頭的另一突破是成為佳能首款採用雙Nano USM對焦馬達的鏡頭。該鏡頭通過兩個Nano USM對焦馬達分別驅動浮動鏡片組和對焦鏡片，實現電子浮動對焦控制，在不犧牲緊湊外形的前提下，實現安靜、節能和高速的操作，同時有效抑制了對焦的呼吸效應，在進行短片拍攝時，可在進行焦點偏移操作時，獲得更穩定的構圖。此外，電子浮動對焦控制的應用還顯著縮短了最近對焦距離。RF70-200mm F2.8 L IS USM在全部焦段內的最近對焦距離僅為約0.7m，放大倍率在70mm端以及200mm端分別約為0.11X和0.23X，能夠獲得更強大、更突出的特寫攝影表現力。

在拍攝靜態照片時，RF70-200mm F2.8 L IS USM鏡頭與EOS R或EOS RP相機協作，可實現雙重檢測IS，在鏡頭IS功能工作同時，還可以通過相機CMOS圖像感應器的圖像信息中檢測抖動量。通過比較和分析這兩類信息，可以高精度地檢測和校正單靠鏡頭難以檢測到的低頻抖動（由於呼吸等引起的機身慢速抖動），從而實現最大相當於約5級2快門速度的手抖動補償效果。而在進行短片創作時，同樣可通過相機的圖像信息來檢測鏡頭IS功能未進行校正的模糊量，同時通過RF卡口更強大的通信能力，可以對相機上的短片五軸數碼IS和具備IS功能的RF鏡頭進行協同控制，實現組合IS功能，獲得更強大的影像穩定效果。

RF70-200mm F2.8 L IS USM鏡頭使用的濾鏡規格為77mm。鏡頭最大直徑及長度分別為約89.9mm及約146.0mm，重量約1070g（僅鏡身）。

RF85mm F1.2 L USM DS，

平滑散焦呈現柔美的焦外虛化效果

焦外虛化效果是拍攝人像的關鍵，但用戶對虛化形態的喜好各不相同，沒有正確的答案。因此，在RF85mm F1.2 L USM基礎上，佳能又推出支持平滑散焦的RF85mm F1.2 L USM DS。這一新鏡頭繼承了RF85mm F1.2 L USM出色光學設計的同時，追求平滑的焦外虛化效果，為追求焦外虛化之美的專業攝影師以及熱愛人像創作的攝影愛好者帶來新選擇。

佳能RF85mm F1.2 L USM DS

佳能RF85mm F1.2 L USM DS鏡頭樣片

為實現柔美的焦外虛化效果，RF85mm F1.2 L USM DS採用了DS鍍膜設計。DS鍍膜是佳能的一種蒸鍍多層鍍膜技術。該結構中心的透射率較高，邊緣的透射率較低，因此可實現平滑的焦外虛化效果。同時，由於該鏡頭採用將DS鍍膜應用於鏡片的前表面和後表面的雙面覆蓋設計，可以使整個邊緣的透射率降低，從而柔和地虛化整個焦外虛化圖像的輪廓，以更大限度地發揮DS鍍膜的效果。同時，通過使用F1.2大光圈和DS鍍膜，即使在畫面的邊緣也能獲得顯著而平滑的焦外虛化效果，因此十分適用於對前景和背景進行柔和的焦外虛化拍攝創作。此外，DS鍍膜帶來的焦外虛化平滑散焦效果會隨光圈值的變化而產生變化，在F1.2大光圈下，DS鍍膜的平滑散焦效果會更明顯。鏡頭由於DS鍍膜邊緣透光率下降，全開光圈時鏡頭亮度比F1.2低約1⅓級。相機通過升級至新版固件可實現合適的自動曝光控制。

RF85mm F1.2 L USM DS非常適合在裝飾性照明等點光源背景下拍攝，可以柔和平滑地虛化點光源的光斑輪廓以創造迷人的焦外虛化效果，或者在複雜背景下突出拍攝對象的拍攝，例如柔和地虛化重疊的樹木，降低背景和前景的複雜度，使拍攝對象更加突出。

RF85mm F1.2 L USM DS繼承了RF85mm F1.2 L USM出色的光學設計，除了通過DS鍍膜獲得平滑散焦的焦外虛化效果外，還具有高分辨率、高對比度和色像差抑制表現，實現了從畫面中央到邊緣的高畫質。基於佳能EOS R系統大卡口直徑以及短後對焦距離的先天優勢，鏡頭得以在光學素質以及設計空間上實現更大提升。鏡頭光學結構為9組13片。採用了BR鏡片及UD（超低色散）鏡片的組合有效抑制色像差，大幅減少了大光圈鏡頭易產生的色暈。同時，還在光圈前方配置了1片研磨非球面鏡片，並通過鏡片位置的優化與合理配置，實現了F1.2大光圈下畫面中心到邊緣整體的高畫質表現。此外，鏡頭還採用了防反射效果好的ASC鍍膜，可提高鏡片的透射率，有效抑制畫面內光源造成的眩光與鬼影，降低逆光對成像的影響，帶來清晰通透的照片效果。

RF85mm F1.2 L USM DS採用9片圓形光圈葉片設計，最近對焦距離僅為約85cm，最大放大倍率為0.12X，因此可以實現更近距離的人像或靜物特寫捕捉，進一步提升拍攝的靈活度。該鏡頭使用濾鏡規格為82mm。鏡頭最大直徑及長度分別為約103.2 mm及117.3 mm，重量約1195g。

更多共同特性

兩款鏡頭同樣配有RF鏡頭上新增的控制環，通過控制環，可以根據使用需求，將該控制環設定為曝光補償（Ev）、感光度數值（ISO）、光圈值（Av）或快門速度（Tv）等常用拍攝參數的調整。藉助這一設計，用戶可以將視線更專注地聚焦在取景器內，無需轉移視線就能方便、快速地完成設定拍攝參數的調整，進一步提升了拍攝體驗。

RF70-200mm F2.8 L IS USM及RF85mm F1.2 L USM DS鏡頭均採用了防水滴防塵結構3，從而實現在嚴苛拍攝環境下的高可靠性。特別是RF70-200mm F2.8 L IS USM鏡頭，由於採用外變焦類型，該鏡頭為此特別設計了通風路線，抑制水汽吸入，並採用特殊防塵材料，以阻隔灰塵滲入。同時，兩款鏡頭也加入了具有更好憎油憎水性的防汙氟鍍膜，在帶來良好透光率的同時，可更加輕鬆地擦除附著於鏡片上的水滴或油漬4。
