液晶顯示屏應用示例以及程序詳解

2020-10-29
| 科技


在本應用中，我們將向您展示如何在Arduino上使用1.8 TFT顯示屏。您將學習如何連接顯示屏幕，書寫文本，繪製形狀以及在屏幕上顯示圖像。

使用1.8 TFT顯示屏

1.8 TFT是具有128 x 160彩色像素的彩色顯示器。顯示器可以從SD卡加載圖像-它的背面有一個SD卡插槽。下圖顯示了屏幕的前視圖和後視圖。

該模塊使用SPI通信–請參見下面的接線。為了控制顯示，我們將使用TFT庫，該庫已包含在Arduino IDE 1.0.5和更高版本中。

初始化顯示

TFT顯示屏通過SPI通信與Arduino通信，因此您需要在代碼中包括SPI庫。我們還使用TFT庫在顯示器上書寫和繪圖。

#include <TFT.h>

#include <SPI.h>

然後，您需要定義CS，A0（或DC）和RST引腳：

＃定義cs 10＃定義

dc 9＃定義

創建一個名為TFTscreen的庫的實例：

TFT TFT屏幕= TFT（cs，dc，rst）;

最後，在setup（）中，您需要初始化庫：

TFTscreen.begin（）;

顯示文字

要在顯示屏上寫文本，您可以自定義屏幕背景顏色，字體大小和顏色。

要設置背景顏色，請使用：

TFTscreen.background（r，g，b）;

其中，r， g和b是給定顏色的RGB值。選擇字體顏色：

TFTscreen.stroke（r，g，b）;


設置字體大小：

TFTscreen.setTextSize（2）;

您可以增加或減少作為參數給出的數字，以增加或減少字體大小。

最後，使用以下行在顯示器上繪製文本：

TFTscreen.text（" Hello，World！ "，x，y）;

在其中" 世界，您好！"是您要顯示的文本，（x，y）座標是您要開始在屏幕上顯示文本的位置。

程序代碼

下面的示例顯示" Hello，World！" 在屏幕中間，每200毫秒更改一次字體顏色。

將以下代碼複製到Arduino IDE，並將其上傳到Arduino開發板。


#include <TFT.h>

#include <SPI.h>

#define cs 10

#define dc 9

#define rst 8

TFT TFTscreen = TFT(cs, dc, rst);

void setup() {

TFTscreen.begin();

TFTscreen.background(0, 0, 0);

TFTscreen.setTextSize(2);

}

void loop() {

int redRandom = random(0, 255);

int greenRandom = random (0, 255);

int blueRandom = random (0, 255);

TFTscreen.stroke(redRandom, greenRandom, blueRandom);

TFTscreen.text("Hello, World!", 6, 57);

delay(200);

}


這是您的"世界，您好！" 在1.8 TFT顯示屏上顯示文本。

顯示形狀

TFT庫提供有用的功能來在顯示器上繪製形狀：

· TFT屏幕。point（x，y） – 在（x，y）座標上顯示一個點

· TFT屏幕。線（XSTART，yStart，XEND，YEND） -繪製線，在（XSTART，yStart）的開始和結束處（XEND，YEND）

· TFT屏幕。rect（xStart，yStart，width，height） –繪製一個矩形，其左上角位於（xStart，yStart），具有定義的寬度和高度

· TFTscreen .circle（x，y，radius） –以指定的半徑繪製一個以（x，y）為中心的圓

碼

顯示影像

1.8 TFT顯示屏可以從SD卡加載圖像。要從SD卡讀取數據，請使用Arduino IDE軟件中已包含的SD庫。請按照以下步驟在顯示屏上顯示圖像：

1） SD卡的焊接頭引腳。與顯示引腳相對的有四個引腳，如下圖所示。

2） 顯示器可以加載大於或小於顯示器尺寸（160 x 128 px）的圖像，但是為了獲得更好的效果，請將圖像尺寸編輯為160 x 128 px。

3） 圖片應為 .bmp格式。為此，您可以使用照片編輯軟件並將圖像保存為 .bmp格式。

4） 將圖像複製到SD卡上，然後將其插入顯示屏背面的SD卡插槽中。

5） 按照下表將SD卡的引腳連接到Arduino：

顯示屏和SD卡都可以通過SPI通信工作，因此您將在Arduino上使用兩個連接來連接引腳。

6） 在Arduino IDE中，轉到 文件>示例> TFT> Arduino> TFTBitmaLogo。

7） 編輯代碼，以便它搜索您的圖像。將" arduino.bmp " 替換為您的圖像名稱：

徽標= TFTscreen.loadImage（" arduino.bmp "）;

8） 將代碼上傳到您的Arduino。
