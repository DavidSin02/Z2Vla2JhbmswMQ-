傳奇技能觸發腳本設置教程，傳奇套裝屬性在哪裡設置？

2020-10-29
| 科技

今天給大家分享一下傳奇堆疊物品設置方法，傳奇技能觸發腳本設置教程


（一）堆疊物品設置

打開桌面的DBC2000，點擊StdItems.DB

找到強效金瘡藥，把Source值改為5，把DuraMax值改為999
這時從藥店購買的一個強效金瘡藥就可以使用999次
把強效金瘡藥Ac的值從90改為900，就能改變強效金瘡藥的恢復數值，一個強效金瘡藥就可以恢復900的HP
強效魔法藥同理，其他藥品也同理


（二）傳奇技能觸發腳本設置教程

打開D:\Mirserver\Mir200\Envir\Market_Def文件夾下的QFunction-0.txt文件

[@PlayLevelUp]

戰士技能觸發腳本

#if
checkjob warrior
CHECKLEVELEX = 7
#act
ADDSKILL 基本劍術 3
break
腳本是這樣的
上面的意思是
當戰士等級達到7級時，自動學習3級基本劍術
以此類推
道士技能觸發腳本
#if
checkjob taoist
checklevelex = 7
#act
ADDSKILL 治癒術 3
break
法師技能觸發腳本
#if
checkjob wizard
checklevelex = 7
#act
ADDSKILL 火球術 3
break
當然你也可以一次性學習所有技能，腳本如下
#if
checkjob warrior
CHECKLEVELEX = 7
#act
ADDSKILL 基本劍術 3
ADDSKILL 攻殺劍術 3
ADDSKILL 刺殺劍術 3
ADDSKILL 半月彎刀 3
ADDSKILL 野蠻衝撞 3
ADDSKILL 烈火劍法 3
break
如果你是武士，想學習道士的技能，腳本如下

#if
checkjob warrior
CHECKLEVELEX = 7
#act
ADDSKILL 基本劍術 3
ADDSKILL 攻殺劍術 3
ADDSKILL 刺殺劍術 3
ADDSKILL 半月彎刀 3
ADDSKILL 野蠻衝撞 3
ADDSKILL 烈火劍法 3
ADDSKILL 召喚神獸 3
ADDSKILL 神聖戰甲術 3
ADDSKILL 幽靈盾 3
break

（三）套裝屬性設置

打開桌面的DBC2000，點擊StdItems.DB


找到綠色項鍊，力量戒指，騎士手鐲，並他他們的Reserved值改為100
打開M2控制檯-->查看-->列表信息-->套裝功能
全套數量=5，代表5件觸發，你也可以寫1,2,3件觸發
套裝編號=100，就是上面的Reserved值
套裝名稱=戰士祖瑪5件套，只是一個說明，可寫可不寫
描述，描述裡可隨便填寫，穿上套裝後，聊天框會顯示描述裡的字
然後點添加，點保存，這樣一個套裝就完成了
攻擊倍數，魔法倍數和道術倍數是除以10的值
比如你填10，那就是10/10，增加1倍攻擊力
