試題編號 : 01

知識點 : 送電線路

使用飛車應注意哪些事項?

答：應注意以下事項：

(1)使用前應對飛車進行全面檢查，以保證使用安全。

(2)使用中行駛速度不宜過快，以免剎車困難。

(3)使用後平時注意保養。

試題編號 : 02

知識點 : 送電線路

架空線連接前後應做哪些檢查?

答：應做如下檢查：

(1)被連接的架空線絞向是否一致。

(2)連接部位有無線股絞制不良、斷股、缺股現象。

(3)切割鋁股時嚴禁傷及鋼芯。

(4)連接後管口附近不得有明顯鬆股現象。

試題編號 : 03

知識點 : 送電線路

維修線路工作的內容包括哪些?

答：包括以下內容：

(1)杆塔、拉線基礎培土。

(2)修理巡線小道，砍伐影響線路安全運行的樹、竹。

(3)添補少量螺栓、腳釘、塔材。

(4)緊固螺栓，調整拉線、塗刷設備標誌。(5)消除杆塔上的鳥巢及其他雜物。

整體起立杆塔有何優、缺點?

答：(1)優點：

①高空作業量小。

②施工比較方便。

③適合流水作業。

(2)缺點：

①整體份量重，工器具需相應配備。

②施工佔地面積大。

試題編號 : 05

知識點 : 送電線路

緊線時耐張(轉角)塔均需打臨時拉線，臨時拉線的作用及要求各是什麼?

答：(1)作用：平衡單邊掛線後架空線的張力。

(2)要求：

①對地夾角為30°～45°。

②每根架空線必須在沿線路緊線的反方向打一根臨時拉線。

③轉角杆在內側多增設一根臨時拉線。

④臨時拉線上端在不影響掛線的情況下，固定位置離掛線點越近越好。



試題編號 : 06

知識點 : 送電線路

鋼筋混凝土杆地面組裝工藝包括哪些方面?

答：包括以下內容：

(1)組裝橫擔：可將橫擔兩端稍微翹起10～20mm，以便懸掛後保持水平。

(2)組裝叉樑：先安裝四個叉樑抱箍，將叉樑交叉點墊高，其中心與叉樑抱箍保持水平，再裝上、下叉樑。

(3)部分螺栓連接構件不宜過緊，以防起吊時損壞構件。

(4)絕緣子組裝後在杆頂離地時再掛在橫擔上，以防絕緣子碰傷。

(5)金具、拉線、爬梯等儘量地面組裝，減少高空作業。安裝完畢後，進行各部分全面檢查。

試題編號 : 07

知識點 : 送電線路

怎樣用異長法觀測並計算運行線路的弧垂？

答：（1）在觀測檔的任一杆塔的導線懸垂線夾下方選取一垂直距離a，綁一弛度板。

（2）另一人持另一塊弛度板登觀測檔的另一杆塔，觀測該檔弧垂。

（3）上下試湊，使兩弛度板與弧垂點恰成一直線，此時綁好第二塊弛度板，量取至該導線懸垂線夾的距離b。

（4）由弧垂計算公式計算出導線的弧垂f。

試題編號 : 08

知識點 : 送電線路

使用線材時應做哪些檢查？

答：線材的技術特徵、製造規範而應符合國家標準，在取得合格證和出廠證明後尚需進行下列外觀檢查：

（1）不得有鬆股、交叉、摺疊、硬彎、斷裂及破損等缺陷。

（2）鋁線不得有嚴重腐蝕現象。

（3）鋼線表面應鍍鋅良好，不得鏽蝕。

（4）絞股線的絞制方向是否一致。

試題編號 : 09

知識點 : 送電線路

杆塔編號一般有何規定？

答：（1）線路杆塔的編號一般以送電端出線杆塔為1號，從送電端依次編至受電端杆塔。

（2）完整的杆塔編號應包括表明電壓等級、線路名稱、杆塔號碼。耐張轉角換位杆塔應塗相色漆。

（3）用油漆在杆塔上的編號或掛牌，以離地面2.5-3.0m為宜，字體不宜過小，面向線路附近巡線通道或大路。

（4）雙迴路的雙杆塔應在每一基杆塔上分別刷印杆上相應線路的名稱和杆號。若雙迴路為單杆或鐵塔，則應將線路名稱和杆號分別印在同一杆塔的兩邊外側或鐵塔的兩側塔腿上。

試題編號 : 10

知識點 : 送電線路

架空線路為什麼會覆冰？

答：形成覆冰層的原因是由於在架空線路上附著水滴，當氣溫下降時，這些水滴便凝結成冰，而且越結越厚。

試題編號 : 11

知識點 : 送電線路

覆冰有什麼危害？

答：(1)引起閃絡。

(2)引起斷線、斷股。

(3)杆塔倒塌、變形及橫擔損壞。

(4)絕緣子及金具損壞。

(5)對跨越物的危害。

(6)對電力系統載波通信的影響。

試題編號 : 12

知識點 : 送電線路

為什麼要儘可能地降低接地電阻的數值？

答：因為接地電阻愈小，雷擊時引起的過電壓愈小，防雷效果就愈好，所以要儘可能地降低接地電阻數值。

試題編號 : 13

知識點 : 送電線路

弧垂的大小對線路的安全運行有何影響？

答：弧垂的大小直接影響到線路的安全運行：

（1）弧垂過小，導線受力增大，當張力超過導線許可應力時，會造成斷線。

（2）弧垂過大，導線對地距離過小而不符合要求，在有劇烈擺動時，可能引起線路短路。

試題編號 : 14

知識點 : 送電線路

導線為什麼要進行換位？

答：(1) 架空線路三相導線在空間排列往往是不對稱的，引起三相系統電磁特性不對稱。

(2) 三相導線電磁特性不對稱引起各相電抗不平衡，從而影響三相系統的對稱運行。

(3) 為保證三相系統能始終保持對稱運行，三相導線必須進行換位。

試題編號 : 15

知識點 : 送電線路

試述不帶拉線的直線單杆的分坑的步驟。

答： （1）定位：用經緯儀定出直線單杆的杆位。將經緯儀擺在主樁位上，沿線路方向定出前、後的副樁。

（2）分坑：以主樁為中心，根據杆坑的大小畫一圓形或方形的坑口線。

填寫接地電阻測量記錄主要包括哪些內容？

答：接地電阻的測量記錄一般須填寫時間、線路名稱、杆號、接地電阻值（雙杆塔分別測兩側接地電阻值）、測量人的姓名等內容。

試題編號 : 17

知識點 : 送電線路

制動鋼絲繩受力情況是怎樣變化的?如何有效防止制動鋼絲繩受力過大?

答： (1) 制動鋼繩在杆塔起立開始時受力收緊。

(2) 隨著杆塔起立角度增加，抱杆的支撐力逐漸減小，制動繩索的受力逐漸增大。

(3) 當抱杆失效時，抱杆的支撐力消失，制動繩的受力最大。

(4) 為防止制動鋼繩受力過大，在抱杆失效前，應調整制動繩長度，使杆根坐入底盤。

試題編號 : 18

知識點 : 送電線路

使用倒落式抱杆整體組立杆塔時如何控制反向臨時拉線?

答：(1) 隨著杆塔起立角度的增大，抱杆受力漸漸減小。

(2) 在抱杆失效前，必須帶上反面臨時拉線。

(3) 反面臨時拉線隨杆塔起立角度增加進行長度控制。

(4) 當杆塔起立到70°時，應放慢牽引速度。

(5) 當杆塔起立到80°時，停止牽引，用臨時拉線調直杆身。

試題編號 : 19

知識點 : 送電線路

拉線在水泥杆上的固定應符合哪些規定？

答：拉線在水泥杆上的固定必須符合以下規定：

（1）楔形線夾和UT線夾卡其舌板與拉線應接觸緊密，在拉線正常受力情況下應無滑動現象，鋼絞線的端頭應裝在受力面的一側。

（2）鋼絞線斷頭露出規定：上端頭應為300mm，可用不小於2.0mm的單股鍍鋅鐵線綁紮。下端頭露出應為500mm，且用不小於3.2mm的單股鍍鋅鐵線綁紮。上下端綁紮一般為50mm。

（3）拉線的交叉處不得互相摩擦。

（4）拉線調好後，楔形線夾和UT線夾露出的絲扣長度應為10-30mm。

（5）拉線包箍距杆頂200mm。

（6）水平拉線在跨越公路及其他交通道路時，其對地距離一般不小於6m。

（7）在居民區若拉線高於帶電導線時，應加裝拉線絕緣子，且必須使拉線絕緣子位於帶電部位以下。一般情況下，拉線絕緣子安裝在距拉線上端1-3m處。

試題編號 : 21

知識點 : 送電線路

架空線弧垂觀測檔選擇的原則是什麼?

答：其選擇原則如下：

(1)緊線段在5檔及以下時靠近中間選擇一檔。

(2)緊線段在6～12檔時靠近兩端各選擇一檔。

(3)緊線段在12檔以上時靠近兩端及中間各選擇一檔。

(4)觀測檔宜選擇檔距較大、懸點高差較小及接近代表檔距的線檔。

(5)緊鄰耐張杆的兩檔不宜選為觀測檔。

在緊線施工中，對工作人員的要求有哪些?

答：對其要求如下：

知識點 : 送電線路

導、地線線軸佈置的原則是什麼?

答：其佈置原則如下。

(1)儘量將長度或重量相同的線軸集中放在各段耐張杆處。

(2)架空線的接頭儘量靠近導線最低點。

(3)導線接頭避免在不允許有導線接頭的檔距內出現。

(4)儘量考慮減少放線後的餘線。

(5)考慮施工方便，為運輸、放線、連接及緊線創造有利條件。

試題編號 : 24

知識點 : 送電線路

簡要解釋檔距、水平檔距、垂直檔距、代表檔距和臨界檔距的含義。

答：(1)檔距：是指相鄰兩杆塔中心點之間的水平距離。

(2)水平檔距：是指某杆塔兩側檔距的算術平均值。

(3)垂直檔距：是指某杆塔兩側導線最低點間的水平距離。

(4)把長短不等的一個多檔耐張段，用一個等效的孤立檔來代替，達到簡化設計的目的。這個能夠表達整個耐張段力學規律的假想檔距，我們稱之為代表檔距。

(5)臨界檔距：是指由一種導線應力控制氣象條件過渡到另一種控制氣象條件臨界點的檔距大小。

知識點 : 送電線路

什麼是混凝土的和易性?和易性對混凝土構件質量有何影響?

答：(1)定義：混凝土的和易性是指混凝土經攪拌後，在施工過程中幹稀均勻的合適程度。

(2)質量影響：

①和易性差的混凝土影響構件內部的密實性。

②和易性差的混凝土影響構件表面的質量。

③和易性差的混凝土影響構件稜角的質量。

什麼是混凝土的坍落度?坍落度主要評價混凝土的什麼指標?

答：坍落度及其評價分述如下：

(1)混凝土的坍落度是指將拌和好的混凝土按要求裝入測試容器後，容器取出時混凝土自行坍落下的高度。

(2)坍落度是評價混凝土和易性及混凝土稀稠程度的指標。

答：交叉跨越的測量記錄一般須填寫時間、線路名稱、杆號、測量溫度、測量人的姓名、測量值等內容，必要時測量值換算到最高溫度下的數值。

試題編號 : 31

知識點 : 送電線路

放線前應進行哪些準備工作？

答：（1）根據架線方向及長度佈置線軸的安裝位置。

（2）將放線支架裝設穩固。

（3）消除放線通道上的障礙物或採用可靠的護線措施。

（4）跨越電力線、通訊線、鐵路及公路時，應搭設安全穩固的跨越架或採取可靠的跨越措施。

（5）根據具體情況合理佈置護線人員，並檢查工作地段的通訊或聯繫信號是否統一、及時和可靠。

試題編號 : 32

知識點 : 送電線路

現澆基礎施工時，如遇特殊情況中途中斷混凝土澆灌，應如何處理?

答：原則上現澆基礎的混凝土應一次連續澆成，不得中斷。特殊情況下在澆製同一基礎混凝土時遇特殊情況中斷超過2h，不得繼續澆灌。若要繼續澆灌必須待混凝土的強度達到1.18MPa後，將連接面打毛，並用水清洗，然後澆一層厚為10～15mm與原混凝土同樣成分的水泥砂漿，再繼續澆灌。

試題編號 : 33

知識點 : 送電線路

從哪些方面檢查現澆混凝土的質量?

答：從以下幾方面進行檢查：

(1)坍落度不得大於配合比設計的規定值。

(2)配比材料誤差應控制在施工措施規定範圍內，並嚴格控制水灰比。

(3)混凝土強度檢查以現場試塊為依據。

試題編號 : 34

知識點 : 送電線路

現場澆築基礎混凝土的養護應符合哪些規定?

答：應符合以下規定：

(1)澆築後應在12h內開始澆水養護，當天氣炎熱、乾燥有風時，應在3h內澆水養護。澆水次數以保持混凝土表面始終溼潤為準。

(2)混凝土養護日期不少於5晝夜。

(3)基礎拆模經表面檢查合格後應立即回填土，並繼續澆水養護。

(4)採用養護劑養護時，應在拆模並經表面檢查合格後塗刷。塗刷後不再澆水。

(5)日平均氣溫低於5℃時不得澆水養護。

試題編號 : 35

知識點 : 送電線路

基礎分坑前杆塔中心樁位置在複測中出現哪些情況應予以糾正?

答：(1)橫線路方向偏差大於50mm。

(2)順線路方向偏差大於設計檔距的1%。

(3)轉角杆的角度偏差超過1′30″。

(4)標高與設計值相比偏差超過0.5m。

試題編號 : 36

知識點 : 送電線路

架空線路耐張段內交叉跨越檔鄰檔斷線對被交叉跨越物有何影響

答：其影響如下：

(1)交叉跨越檔鄰檔斷線時，交叉跨越檔的應力衰減，弧垂增大。

(2)交叉跨越檔因弧垂增大，使導線對被交叉跨越物距離減小，影響被交叉跨越物及線路安全。

(3)對重點交叉跨越物必須進行鄰檔斷線交叉跨越距離校驗。

試題編號 : 37

知識點 : 送電線路

架空線路的垂直檔距大小受哪些因素的影響?其大小影響杆塔的哪種荷載?

答：(1)影響因素：

①杆塔兩側檔距大小。

②氣象條件。

③導線應力。

④懸點高差。

(2)垂直檔距大小影響杆塔的垂直荷載。

試題編號 : 38

知識點 : 送電線路

何為避雷線的保護角?其大小對線路防雷效果有何影響?

答：(1)避雷線的保護角：是指導線懸掛點與避雷線懸掛點點連線同鉛垂線間的夾角。。

(2)影響：保護角越小，避雷線對導線的保護效果越好，通常根據線路電壓等級取20°～30°。

試題編號 : 39

知識點 : 送電線路

何為避雷針的逆閃絡?防止逆閃絡的措施是什麼?

答：(1)逆閃絡：是指受雷擊的避雷針對受其保護設備的放電閃絡。

(2)防措：

①增大避雷針與被保護設備間的空間距離。

②增大避雷針與被保護設備接地體間的距離。

③降低避雷針的接地電阻。

試題編號 : 40

知識點 : 送電線路

何為接地裝置的接地電阻?其大小由哪些部分組成?

答：(1)接地裝置的接地電阻：是指加在接地裝置上的電壓與流入接地裝置的電流之比值。

(2)接地電阻的構成：

①接地線電阻。

②接地體電阻。

③接地體與土壤的接觸電阻。

④地電阻。




試題編號 : 41

知識點 : 送電線路

杆塔整體組立時，人字抱杆的初始角設置多少為好?為什麼?

答：(1)人字抱杆的初始角設置為60°～65°最佳。

(2)原因：

①初始角設置過大，抱杆受力雖可減小，但此時抱杆失效過早，對立杆不利。

②初始角設置過小，抱杆受力增大，且杆塔起立到足夠角度不易脫帽，同樣對立杆不利。

試題編號 : 42

知識點 : 送電線路

架空線路杆塔有何作用?

答：架空線路杆塔的作用是：

(1)用於支承導線、避雷線及其附件。

(2)使導線相間、導線與避雷線、導線與地面或與被交叉跨越物之間保持足夠的安全距離。

試題編號 : 43

知識點 : 送電線路

塔身的組成材料有哪幾種?

答：有以下幾種：

(1)主材。

(2)斜材。

(3)水平材。

(4)橫隔材。

(5)輔助材。

試題編號 : 44

知識點 : 送電線路

常用的複合多股導線有哪幾種？

答：(1)普通鋼芯鋁絞線。

(2)輕型鋼芯鋁絞線。

(3)加強型鋼芯鋁絞線。

(4)鋁合金絞線。

(5)稀土鋁合金絞線。

試題編號 : 45

知識點 : 送電線路

輸電線路檢修的目的是什麼?

答：其目的如下：

(1)消除線路巡視與檢查中發現的各種缺陷。

(2)預防事故發生，確保線路安全供電。

試題編號 : 46

知識點 : 送電線路

混凝土中水泥的用量是否越多越好?為什麼？

答：(1)混凝土水泥用量不能過量，應符合配合比設計要求。

(2)混凝土中水泥用量過多會使混凝土在硬化過程中因水分蒸發引起水泥體積過量收縮，造成混凝土開裂、露筋等缺陷，從而影響混凝土的強度。

試題編號 : 47

知識點 : 送電線路

為何拌制混凝土儘可能選用較粗的砂

答：其原因如下：

(1)較粗的砂粒，其單位體積的表面積小，易與水泥漿完全膠合。

(2)砂粒表面膠合的水泥漿充分，有利於提高混凝土強度。

(3)有利於降低水泥的用量，減小水灰比。

試題編號 : 48

知識點 : 送電線路

如何更換直線杆塔?

答：(1)為了減少停電時間，通常在原直線杆前後1～2m處先豎立新杆，利用舊杆掛起滑輪，即可吊立新杆。

(2)在立新杆前，先將舊杆上懸垂線夾拆去，使導線懸空並用粽繩將導線吊起。架空地線暫不撤除起支持作用。

(3)新杆立好後填土夯實，將導地線接於新杆上，調整兩檔弧垂相等即可固定導線。

(4)利用新杆掛滑輪拔去舊杆，拔杆前先將舊杆前方泥土挖成斜槽，挖至杆根，然後將鋼絲繩端結紮舊杆橫擔附近，牽引絞磨後慢慢鬆出牽引繩使電杆慢慢落地。

(5)若新杆不能利用，則可用人字抱杆進行施工。

試題編號 : 49

知識點 : 送電線路

為什麼導線截面積要按機械強度的要求校驗其截面？

答：由於架空線路架設在大氣中，導線要經受各種外界不利條件的影響，如果選用導線截面小，機械和電氣過載能力小，易發生事故，為保證安全運行，導線應有一定的機械強度，即導線的截面不宜過小。概括起來說，在高壓線路上：

（1）不許使用單股導線。

（2）鋁及鋁合金、鋼芯鋁絞線的最小截面為35平方毫米以上。

（3）其他材料導線的最小截面積為16平方毫米。

試題編號 : 50

知識點 : 送電線路

簡要說明導線機械特性曲線計算繪製步驟。

答：其計算繪製步驟如下：

(1)根據導線型號查出機械物理特性參數，確定最大使用應力，結合防振要求確定年平均運行應力。

(2)根據導線型號及線路所在氣象區，查出所需比載及各氣象條件的設計氣象條件三要素。

(3)計算臨界檔距並進行有效臨界檔距的判定。

(4)根據有效臨界檔距，利用狀態方程式分別計算各氣象條件在不同代表檔距時的應力和部分氣象條件的弧垂。

(5)根據計算結果，逐一描點繪製每種氣象條件的應力隨代表檔距變化的曲線和部分氣象條件的弧垂曲線。
