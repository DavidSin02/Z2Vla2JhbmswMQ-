如何在浩瀚的星空下設置最佳ISO拍攝：動態範圍和噪聲

2020-10-29
| 科技

各位的大方關注，是俺不斷更新的動力喔！

ISO是數碼相機曝光三角形中的三個主要曝光設置之一。在三個時間中：快門時間，f /數字和ISO，最容易被誤解的是ISO。比f / number還要大。實際上，常見的誤解是較高的ISO設置會導致圖像噪聲較大。實際上，情況恰恰相反。

等一下沒錯，僅較高的ISO設置並不會增加圖像噪點，較高的ISO甚至可以對弱光攝影有所幫助。在本文中，將討論有關ISO設置的問題，ISO實際如何影響曝光以及如何在相機上為天文攝影找到最佳ISO設置。

ISO標準

在數碼攝影中，需要了解ISO的曝光亮度制定的標準（特別是ISO 12232：2006）。不同的相機傳感器型號具有不同的靈敏度。

信號

信號是我們想要的照片的一部分。光是信號。信號就是圖像。沒有信號（沒有光），我們將沒有圖像。我們可以收集的光越多，我們收到的信號就越多。通常，信號越多，照片的質量越高。

噪聲

噪點是照片中我們不想要的部分。噪點是干擾，表現為斑點狀的顆粒，會遮蓋信號，從而掩蓋照片的細節。噪聲通常是由熱量或數碼相機電子設備的行為不完善所產生的。每次拍攝時某些噪聲是隨機的（拍攝噪聲），並且相機的傳感器會始終產生某些噪聲（上游讀取噪聲），或者傳感器信號被放大後，電子設備會產生一些噪聲（下游讀取噪聲）。通常，噪點越多，照片的質量越低。

信噪比（SNR）

圖像中信噪比。信噪比越高，圖像質量越高。更多的光=更多的信號=良好。更多噪音=不好。收集更多的光是增加信噪比的最佳方法。

較高的信噪比是改善圖像質量的最佳方法。索尼a7S，55mm f / 1.8 @ f / 2.8、48x5s，PP7，ISO 12800

動態範圍

場景的全部光線，從最暗的黑暗到最亮的明亮。高動態範圍場景具有極亮的高光（例如太陽）和極暗的黑暗（例如完全陰影的黑色岩石）。低動態範圍場景在整個場景中具有相對均勻的光線，在該場景中，圖像的最亮部分不會比圖像的最暗部分亮得多。攝像機只能捕獲有限的動態範圍的光線。如果場景的動態範圍足夠高，則相機傳感器範圍之外的任何東西都將被吹成純白色（在非常明亮的區域中）或壓碎為純黑色（在非常暗的區域中） 。通常，更需要一種能夠捕獲更高動態範圍的光的相機傳感器。

此外，本文中提出的所有觀點都適用於RAW圖像文件。以RAW格式拍攝天文照片以保持傳感器收集的最佳數據非常重要。在JPEG上嘗試本文中的任何測試時，請不要抱怨。同樣，優化ISO選擇的許多好處主要適用於低照度拍攝（如天文攝影），在這種情況下，我們有相對少量的信號與侵蝕我們照片的各種噪聲源競爭。

ISO是放大還是增益

（非常）常見的誤解是，提高ISO可以提高相機傳感器的靈敏度。ISO不會改變感光度。增加ISO只是通過放大傳感器信號來增加照片的亮度。在電子世界中，放大有時稱為“增益”。就像我們增加飲食可以“增加”體重，增加ISO可以“增加”亮度。

ISO不會改變感光度

ISO絕對不會影響相機可以收集多少信號（光線）。如果我們實際上想用相機提高感光度，則需要增加快門時間或光圈大小（降低f / 函數）。

較高的ISO不會增加噪音

好的，要點是：較高的ISO不會增加照片中的可見噪聲。

再讀一遍，意識到它與您可能對ISO的瞭解相矛盾，然後讓我詳細說明：

在所有其他條件相同的情況下，較高的ISO將執行以下操作：

  • 較高的ISO將增加圖像的亮度
  • 較高的ISO會減少圖像的總動態範圍
  • 而且，在許多情況下（例如天文攝影），較高的ISO值實際上會減少可見噪聲

好的，我知道您在想什麼：“當我使用更高的ISO時，會得到更多的噪音嗎？！” 原因如下：

對於大多數成像情況，攝影師通常會在相機上使用P（程序），A / Av（光圈優先/光圈變量）或S / Tv（快門優先/時間變量）模式。在這些曝光模式下，使用較高的ISO設置將導致圖像具有更多的相對噪點。大多數人沒有意識到的是，噪聲的增加並不是由於ISO的增加。在自動曝光模式下（例如P，A / Av或S / Tv）使用較高的ISO時，相對噪聲的增加實際上是由於使用自動曝光導致快門時間或光圈尺寸的倒數減少曝光模式。當噪聲的增加實際上是由於快門或光圈引起的信噪比降低而引起的時，大多數人將噪聲的增加歸因於ISO。

在這些自動曝光模式之一上設置較高的ISO時，照相機會嘗試實現中性曝光，並通過減少進入照相機的光量來補償ISO的增加。相機會通過減少快門打開時間（在A / Av模式下）或通過使用較高的f / number來自動減少光線，從而減小鏡頭光圈的大小並在時間（在S / Tv模式下），或兩者結合（在P模式下）。

因此，快門或光圈導致的光線減少是圖像顯得噪點更大的原因。由於較高的ISO，因此噪音較小。光的減少是信號的減少，信號的減少產生總體較低的信噪比，因此產生了噪聲較大的照片。

快門，光圈和ISO如何影響噪音？

一個簡單的比較測試可以表明，相對噪聲水平主要受快門和光圈的影響，而不受ISO影響的程度則差不多。在這些測試中，除我們要測試的設置（通過兩個檔位進行調整）外，所有設置均保持相同。然後，在後期處理中，圖像被均衡亮度並進行比較。

快門時間如何影響噪音

  • 8秒，f / 2.8，ISO 3200
  • 4s，f / 2.8，ISO 3200（開機自檢+1）
  • 2秒，f / 2.8，ISO 3200（開機後停止2個）

快門時間如何影響噪音：55mm，f / 2.8，ISO 3200

結論：更短的快門時間=更少的信噪比=更嘈雜的照片

光圈（F /數字）如何影響噪聲

  • 8秒，f / 2.8，ISO 3200
  • 8秒，f / 4.0，ISO 3200（開機自檢+1）
  • 8秒，f / 5.6，ISO 3200（開機後停止2個）

光圈（f /數字）如何影響噪聲–：55mm，8s，ISO 3200

結論：較高的f /數字=較小的信噪比=噪聲較大的結論：較高的f /數字=較小的信噪比=噪聲較大的噪聲

ISO如何影響噪音

  • 8秒，f / 2.8，ISO 3200
  • 8秒，f / 2.8，ISO 6400（開機自檢中-1停止）
  • 8秒，f / 2.8，ISO 12800（後置2檔）

ISO如何影響噪音：55mm，f / 2.8，8s

結論：更高的ISO≠更多的相對噪聲

ISO如何影響低照度圖像，從傳感器到傳感器，從攝像機型號到攝像機型號都有不同。瞭解相機傳感器的行為方式可以幫助您找到適合天文攝影的最佳ISO設置。在大多數現代數碼相機中，我們看到兩種相當常見的配置，因此我們可以將大多數相機分為兩個陣營之一：ISO變量和ISO不變量。

ISO變式相機

相機使用各種級別的模擬放大來調整ISO。為了簡化這種情況，對於每個ISO：100、200、400、800、1600等，放大器將傳感器的電子電壓讀數提高2倍。較高的ISO意味著更多的傳感器輸出數據放大。

傳感器數據通過ISO放大後，通過一些（下游）電子設備（例如模數轉換器）發送，以最終將我們的數據從電壓更改為計算機可以讀取的數字文件。ISO可變攝像機的獨特特徵之一是來自這些下游電子設備的噪聲貢獻更大。

如果開始時信號相對較少（例如在弱光條件下），則較低的ISO設置可能無法對傳感器數據的電壓施加足夠的放大率，以克服下游電子設備產生的電子噪聲的影響。這意味著，在諸如天文攝影的低照度情況下，ISO變量相機實際上在低ISO設置下顯示更多的噪點，而在較高ISO設置下顯示更少的噪點。佳能EOS 6D仍然是天文攝影最佳選擇之一，是高ISO變，實際上顯示了在ISO 6400和更高最佳的低光噪聲性能！

佳能EOS 6D具有很高的ISO可變性，並在ISO 6400及更高版本下實現了最佳的微光噪點性能。

ISO不變式相機

ISO-Invariant相機的下游讀取噪聲較低，因此在弱光拍攝條件下，隨著ISO設置的變化，信噪比將保持更恆定。在這種情況下，在將傳感器數據轉換為數字信號之前，已經將其放大到下游讀取噪聲源的最小貢獻量以上。結果是具有低ISO的相機趨向於具有較少的陰影噪點和較少的ISO設置差異。這些類型的相機大多數都被認為相對沒有ISO或ISO不變。

由SONY和Fujifilm製造的現代數碼相機往往相對ISO不變

註釋和例外

好的，並不僅是黑白的：在某些較高的ISO設置之上，許多ISO可變攝像機最終都會像ISO不變攝像機那樣工作。並且在較高ISO的情況下，其信噪比差異很小。多數佳能相機在ISO1600左右都採用這種方式。瞭解ISO閾值設置可以幫助我們獲得最佳的弱光性能。

ISO不變性測試– Fujifilm X-T1

最終，兩種配置都達到了使照片變亮以與特定的ISO設置相對應的相同目標，但是最終結果可能會大不相同，尤其是在弱光環境下拍攝時。

結論

與普遍的看法相反，較高的ISO不會產生更多的噪點，而在弱光環境下拍攝時，尤其是在具有ISO變量傳感器的相機上拍攝時，使用較高的ISO實際上是有益的。在相機上運行ISO不變性測試，以確定拍攝天文攝影時使用的最佳ISO設置。不同相機型號的ISO行為各不相同，測試每個ISO設置可以幫助確定用於天文攝影中最佳噪點性能的最佳ISO。

重要的是要了解，ISO差異或不變性並不一定會使相機在弱光條件下變得更好或更差，而只是有所不同。瞭解相機的行為是獲得最佳圖像質量的重要一步。
