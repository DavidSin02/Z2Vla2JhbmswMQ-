GitHub 通知提醒開源小工具-Gitify

2020-10-29
| 科技

Gitify是Mac OS下的GitHub 通知提醒開源小工具，基於Electron編寫。運行Gitify後用 GitHub 進行帳號授權，之後Gitify會在菜單欄上通過變換圖標顏色提醒新消息。點開圖標查看未讀消息，可區分已讀未讀。支持企業版和多賬戶。Gitify還有IOS和Andriod手機版。軟件遵守MIT開源協議。
