李新勇：鑑定N摻雜碳材料電催化脫氯反應活性位點

2020-11-14
| 科技

▲第一作者：甘國強；通訊作者：李新勇，陳愛成
通訊單位：大連理工大學論文DOI：10.1021/acscatal.9b02853


全文速覽本課題利用不同類型 pyridinic N, pyrrolic N, graphitic N 和 oxidized N 的含量與材料電催化脫氯活性的相關性分析發現，Oxidized N 可能是氮摻雜碳材料電催化 1,2-二氯乙烷（DCE）脫氯產乙烯的主要活性位點，經 DFT 理論計算驗證後發現，Oxidized N 位點產乙烯的能壘最小為 1.86 eV，pyrrolic N 位點脫氯產氯乙烯的能壘最小為 2.30 eV，證明 Oxidized N 是產乙烯的活性位點，並且反應中乙烯的選擇性遠大於氯乙烯的選擇性。


背景介紹DCE 的電催化脫氯主要是通過β消除機制反應，產物包括乙烯、氯乙烯、氯化氫以及氫氣這些經濟附加值較高的物質。這一反應具有“一石二鳥”的效果，既消除了 DCE 對環境的汙染，又得到了經濟能源物質，因此，其在化工和環境領域均表現出較高的前景。然而，這一反應目前在時間應用中比較難實現，因為目前對脫氯反應的機制認識比較薄弱，同時，反應中對產物的選擇性較低。


近些年，研究者開發出多種多樣的金屬基電催化劑（Pd, Pt, Au, Ag, Cu, Fe, Ni, Pb 等）及碳基催化劑（如石墨電極、玻碳電極等），考察了其 DCE 脫氯的性能。氮摻雜碳，N 摻雜碳材料，如 N 摻雜石墨烯、N 摻雜碳納米管、N 摻雜碳纖維、N 摻雜納米金剛石以及 N 摻雜導電聚合物等，作為一種最有可能替代貴金屬催化劑的材料，被廣泛應用於氧還原，電解水、產氫、二氧化碳還原以及氮還原的反應。受這種普適活性的啟發，作者將其應用於 DCE 的電催化還原脫氯反應，同樣獲得了滿意的結果。


通常，為了更進一步提高氮摻雜碳材料的催化活性，需要儘可能多的暴露活性位點，因此，鑑定氮摻雜碳材料用於 DCE 電催化還原脫氯活性位點至關重要。雖然其用於氧還原，電解水、產氫、二氧化碳還原等反應的活性位點已初步研究並獲得結果，但是，由於脫氯反應的複雜性，導致活性位點的確定仍然具有相當的難度。


研究出發點本研究利用多巴胺（DA）原位聚合沉積反應，製備了一系列表面氮摻雜碳材料（SNPC），將其用於 DCE 的電催化還原脫氯。考察了所得的 SNPC 材料的 DCE 電催化脫氯活性及產物選擇性，分析了材料類型及電勢對活性和產物選擇性的影響。在此基礎上，通過 H2-TPR 後處理及 XPS 分析，考察了催化活性及產物選擇性與 pyridinic N, pyrrolic N, graphitic N, 和oxidized N 含量的顯著性關係，從實驗上確定 DCE 在 SNPC 上的電催化脫氯反應活性位點，同時，結合理論計算，分析 DCE 在四種類型 N 位上反應的能壘，進而從理論上證明實驗所得結論。


圖文解析作者以高比表面積的活性炭為多孔基體，以多巴胺為氮源，利用表面N摻雜策略，將多巴胺在活性炭表面原位聚合沉積，經高溫熱解後，得到了表面 N 摻雜的多孔碳。活性炭具有較高的比表面積，因此能夠儘可能的暴露催化劑的活性位點，同時這種表面氮摻雜策略能減少多巴胺的用量，達到降低成本的目的。

為了探究 SNPC-x 材料用於 DCE 脫氯的催化活性位點，分析了材料的脫氯活性與吡啶 N、吡咯 N、石墨 N 以及氧化 N 含量的關係。作者將 SNPC-10 材料在 200℃、600℃ 和 900℃ 下 H2 還原，得到的材料隨著還原程度的不同，各類型的 N 含量不同，然後以比表面積歸一化的乙烯分電流（J BET surface area）與不同類型的 N 含量進行相關性分析，發現所有類型的 N 含量與乙烯分電流均呈正相關，但是氧化 N 含量與材料的脫氯活性的線性關係最好，說明氧化 N 有可能是 DCE 脫氯產乙烯的主要活性位點。 ▲Figure 6. 為了進一步驗證上述實驗結果，利用 DFT 計算對 DCE 在不同位點上反應過程中的能壘和態密度進行計算。結果發現，DCE 脫氯生成 CH2CH2 反應中第二脫氯 (*CH2ClCH2·+ H+ + e- → *CH2CH2 + HCl)是反應的速率決定步驟。氧化 N 位點反應的能壘最低為 1.86 eV，說明氧化 N 位點是 DCE 脫氯成 CH2CH2 的催化活性中心。


同時，第二步脫氫步驟(*CH2ClCH2·+ Cl·→*CHClCH2 + HCl)為 CH2ClCH2Cl 脫氯成 CHClCH2 的速率決定步驟。吡啶 N 位點的能壘最小為 2.30 eV，說明吡啶 N 是產氯乙烯的活性位點。產乙烯的控速步驟的能壘低於產氯乙烯的控速步驟的能壘，也進一步證明了 CH2CH2 選擇性較好。有趣的是，計算得到的 PDOS 顯示，石墨 N 和氧化 N 的 N 2p 態比吡啶 N 和吡咯 N 的 N 2p 態更傾向於定位在一個更正的區域，這說明越正的 N 2p 位點越有利於 CH2CH2 的生成。


DCE 具有較強的電子捕獲能力，所以 DCE 在被吸附到催化劑表面後，快速從電極上得一電子，生成活躍的帶負電的離子 *-CH2ClCH2Cl。然後 *-CH2ClCH2Cl 快速脫去一個 Cl，生成氯乙基自由基，氯乙基自由基進一步脫 Cl 或脫 H 生成吸附態的乙烯和氯乙烯。最後，乙烯和氯乙烯共電極材料上解析，成為氣態產物。
