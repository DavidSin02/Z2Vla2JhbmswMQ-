Akko Ducky 3108地平線鍵盤評測：永無止境

2020-10-29
| 科技

“海的思念綿延不絕 終於和天 在地平線交會”這是筆者在上學時聽的歌曲中的一句歌詞，好像早已忘記的歌和歌詞，當我在拿到並看到手中的這款Akko X Ducky 3108寶石藍Horizon地平線機械鍵盤時，腦中浮現的除了有地平線的樣子外，就是這首歌了。

相信大家對Akko（艾酷）X Ducky推出的3108系列機械鍵盤早已不再陌生，因為這個系列早已推出了不少產品，比如：3108金砂黑機械鍵盤、3108S機械鍵盤、3108S Newbee戰隊限定版機械鍵盤、3108 PBT側刻機械鍵盤等等。而這次，它又以Ducky今年的主題色“寶石藍”為基色，並搭配上Hardcap全新的配色“地平線”PBT二色鍵帽，打造出了一款既沉穩又不失活潑的機械鍵盤。

本次，外設天下除了收到了Akko X Ducky 3108 Horizon地平線機械鍵盤外，還收到了Horizon地平線配色的108鍵PBT二色鍵帽，對於兩種風格略有不同的“地平線”，你更喜歡哪種呢？趕緊來看下吧~

Akko X Ducky 3108地平線鍵盤的包裝展示

Akko X Ducky 3108地平線鍵盤的包裝展示

　　Akko X Ducky 3108 Horizon地平線鍵盤的包裝依然採用了Ducky最新的家族式包裝，整體的像素塊設計使它看起來很有時尚感。包裝的正反面都有著銀色亮面的大大LOGO及型號名稱，反而對鍵盤的特點介紹就少的可憐了。

Akko X Ducky 3108地平線鍵盤的包裝展示

Akko X Ducky 3108地平線鍵盤的配件展示

　　它的內包裝還算不錯，保護比較到位，附件也非常全面、實用，除了鍵盤本體外，還有著1個全尺寸防塵罩、1本說明書、1張保障卡、8顆PBT二色鍵帽，以及1個鴨頭拔鍵器。

Horizon地平線配色的108鍵PBT二色鍵帽的包裝展示

Horizon地平線配色的108鍵PBT二色鍵帽的包裝展示

　　Horizon地平線配色的108鍵PBT二色鍵帽的包裝看起來很簡單，只是透明的塑料外殼，也沒有贈送額外的拔鍵器。

Akko X Ducky 3108地平線鍵盤的外觀展示

　　3108系列是Akko與Ducky合作後推出的全新入門級機械鍵盤，是為代替原有的2108系列，3108延續了窄邊框的經典設計，比2108更窄、更精緻、也更美觀。該鍵盤的尺寸約為：440mm×140 mm×41mm，重量約為：1.2KG。

Akko X Ducky 3108地平線鍵盤的外觀展示

Akko X Ducky 3108地平線鍵盤的外觀展示

　　鍵盤整體配色為純藍色時的照片，就是那套Horizon地平線配色的108鍵PBT二色鍵帽的樣子。帶有一顆金黃色鍵帽的照片，就是這款3108地平線剛被取出時的樣子。

Akko X Ducky 3108地平線鍵盤的外觀展示

Akko X Ducky 3108地平線鍵盤的官方宣傳圖

　　不過，筆者還是更喜歡現在這樣將附贈的鍵帽全部都搭配好的樣子。因為它的外殼為寶石藍色，鍵帽又為深藍+寶石藍的配色，再加上那些金橘色與晴空藍的混色搭配字體，以及能起到畫龍點睛的金黃色鍵帽，就使它很形象地還原了太陽初升，海平面的狀態，故命名為“地平線”。

Akko X Ducky 3108地平線鍵盤的細節展示

Akko X Ducky 3108地平線鍵盤的細節展示

　　Akko X Ducky 3108 Horizon地平線鍵盤的外殼很漂亮，寶藍色的配色，再加上那星星點點的珠光感覺，很有那種大海在陽光照耀下泛著波光的感覺。而且為了再次提升鍵盤的辨識度，在鍵盤下邊框的右側還有著Akko X Ducky的LOGO。

Akko X Ducky 3108地平線鍵盤的細節展示

　　該鍵盤為108鍵配列，所以在右上方還有著4顆多功能按鍵，分別為：計算器按鍵、靜音按鍵、聲音減小按鍵、聲音增大按鍵。

Akko X Ducky 3108地平線鍵盤的細節展示

Akko X Ducky 3108地平線鍵盤的細節展示

　　鍵盤的背面雖然沒有過多的繁瑣設計，但是很人性化。它的背面一共有4小條防滑橡膠，並且即便不將撐腳打開，它也有著一定的高度設計。

Akko X Ducky 3108地平線鍵盤的細節展示

Akko X Ducky 3108地平線鍵盤的細節展示

　　同時，它的撐腳還為兩段式設計，再加上它的三向導線槽設計，更加突顯它人性化的一面了。

Akko X Ducky 3108地平線鍵盤的細節展示

　　由於鍵盤本身就帶有一定坡度，所以只將最小的撐腳打開即可。對比筆者來說，總感覺最高的撐腳實在是有些高，坡度太明顯了，不過這也得看每個人的使用習慣和鍵盤的擺放了。

Akko X Ducky 3108地平線鍵盤的細節展示

　　它的線材為黑色的橡膠材質，柔軟度還不錯，長度為1.8m，並且像防拉扯設計、屏蔽環、防塵罩、鍍金接口、綁線帶等均有，配置還是很齊全的。

Akko X Ducky 3108地平線鍵盤的鍵帽展示

　　Akko X Ducky 3108地平線鍵盤的鍵帽是在與“鎧硯Hardcap”合作突破PBT二色完整字形（就是O、D、B、P等字符為全閉口設計）技術後，推出的首款配色產品。當然，這種配色除了用在這款3108上以外，鍵帽也同時提供零售版本，比如與鍵盤同時寄來的那款Horizon地平線配色的108鍵PBT二色鍵帽，而它除了108鍵配列外，還單獨又推出了加入雞年11Key的大全套套裝。

Akko X Ducky 3108地平線鍵盤的鍵帽展示

Akko X Ducky 3108地平線鍵盤的鍵帽展示

　　據官方介紹說，這款Horizon地平線配色的鍵帽是經過第三方檢測機構測試的，它擁有高達85%的PBT含量，能保證它不易打油的特性。這款鍵帽雖然看起來會有些類似磨砂質感的晒紋，但它的整體手感還是很順滑、很舒適的，而且鍵帽的整體也很乾燥，不是那種黏膩膩的手感。

Akko X Ducky 3108地平線鍵盤的鍵帽展示

　　鍵帽顏色的主體為深邃的深海藍底色，而它的字體卻為稜角分明的金橘色與安靜的晴空藍混色搭配，再加上細字體及全閉口式設計，使它的整體觀感很漂亮。

Akko X Ducky 3108地平線鍵盤的軸體展示

　　它的軸體採用了極具信仰的CHERRY原廠軸體，筆者手中的這款為CHERRY紅軸，而它還有著青、茶、黑三種軸體。關於CHERRY紅軸的手感，相信也不用筆者再多說什麼了吧，直上直下的輕盈手感，還是很適合玩遊戲使用的，不過由於這款鍵盤帶有鋼板，所以在與無鋼板的CHERRY紅軸相比時，它的手感就要略硬一些了。

這款鍵盤沒有背光燈，所以在軸體上方也並沒採用LED燈珠，但它卻預留了燈位，可支持玩家們自己進行DIY改燈。

Akko X Ducky 3108地平線鍵盤的大鍵位展示

　　它的大鍵位採用了衛星軸設計，整體的調校非常不錯，一致性也很好，但可惜它的聲音會略微有一點點大。

Akko X Ducky 3108地平線鍵盤的無衝測試

　　經過測試，這款Akko X Ducky 3108地平線鍵盤支持全鍵無衝，所以以後喜歡打拳皇類型遊戲的玩家們就可以放心使用了。

Akko X Ducky 3108地平線鍵盤的遊戲測試

　　在遊戲測試中，這款鍵盤的表現還是非常不錯的，鍵帽的觸感平滑、舒適，軸體的回彈速度快、力度輕，能夠迅速按出技能對應的按鍵。在打字方面，它的手感也同樣很好，筆者的這篇評測就是用它敲出來的，不過對於帶鋼板的紅軸來說，其實筆者更喜歡無鋼板的紅軸，畢竟那樣會感覺更綿軟一些，不過筆者猜肯定會有不少人喜歡這種帶鋼板的感覺。

型號：3108（非背光）

顏色：寶石藍/地平線配色

按鍵數：108鍵

鍵帽：PBT二色（85%含量）

軸：Cherry MX

LED燈光：無，預留燈位，支持DIY改燈

全鍵編程：支持，Ducky Macro V2.0

多媒體按鍵：支持

無衝：Ducky N-Key Rollover，支持全鍵無衝切換

Win鍵屏蔽：支持

尺寸：440mm×140 mm×41mm

重量：1.2KG

產品優點：

+ 鍵盤帶有兩段式撐腳及三向導線槽；

+ PBT鍵帽手感舒適，做工精細，不易打油；

+ 鍵帽字體為工藝難度較高的閉口式設計；

產品缺點：

- 空格鍵聲音略大。

Akko X Ducky 3108地平線鍵盤擁有全新且漂亮的配色，在創意方面也是很有新意的，而且不得不說的是Akko X Ducky所採用的鍵帽做工確實是越做越好，哪怕只是和半年前的產品做對比，也能看出它的變化之處。

就像“地平線”一樣，是個神祕、永遠讓人嚮往，但卻始終沒有盡頭的路，而好的設計與好的工藝亦是如此，希望Akko X Ducky能在未來創造出更多工藝精良、賞心悅目的好產品。

　　這款地平線鍵盤的價格為599元；Horizon地平線配色的基礎版108鍵為169元（不含黃色按鍵部分）；Horizon地平線配色的完整版加入了雞年11Key套裝，大全套零售價199元，對這款鍵盤或是對鍵帽感興趣的玩家們可以多多關注下。
