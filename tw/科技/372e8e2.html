「二十八」MATLAB數字圖像運算之圖像配準

2020-10-29
| 科技

在MATLAB數字圖像處理工具箱中，提供了實現圖像空間變換的函數。空間變換包括前面所介紹的可用數學函數表達的簡單變換（如平移、拉伸等仿射變換）和依賴實際圖像而不易用函數形式描述的複雜變換（如對畸變圖像作校正）。

其中，仿射變換可以理解為：

  • 對座標進行縮放、旋轉、平移後取得新座標的值；

  • 經過對座標的縮放、旋轉、平移後原座標在新座標領域中的值。

投影變換則是表示將原圖像像素點的座標變換為另一種圖像像素點的座標的過程。

在對圖像進行空間變換前，首先得創建一個空間變換結構。MATLAB提供了兩種構建空間變換結構的函數，即cp2tform()和makecform()函數。

makecform()函數的調用格式如下：

  • T = maketform(transformtype,...)

  • T = maketform('affine',A)

  • T = maketform('affine',U,X)

  • T = maketform('projective',A)

  • T = maketform('projective',U,X)

  • T = maketform('custom', NDIMS_IN, NDIMS_OUT, FORWARD_FCN, INVERSE_FCN, TDATA)

  • T = maketform('box',tsize,LOW,HIGH)

  • T = maketform('box',INBOUNDS, OUTBOUNDS)

  • T = maketform('composite',T1,T2,...,TL)

  • T = maketform('composite', [T1 T2 ... TL])

參數transformtype的對應值可以為：

  • 'affine'：仿射變換形式，至少需要3對匹配點；

  • 'projective'：至少需要4對匹配點；

  • 'cusyom'：自定義函數變換；

  • 'box'：利用函數中的另外參數產生仿射變換結構；

  • 'composite'：該參數是實現多次調用tformfwd功能；

cp2tform()函數的調用格式如下：

  • T = cp2tform(movingPoints,fixedPoints, transformtype)：參數movingPoints和fixedPoints對應元素構成一組匹配點；參數transformtype如上。

在完成變換空間的構建後，可以利用MATLAB提供的imtransform()函數實現圖像的空間變換。該函數的調用格式如下：

  • B = imtransform(A,T)：按照指定的二維變換結構T對圖像A進行空間變換處理；

  • B = imtransform(A,T,interp)：參數表示interp插值方法，例如最近鄰插值（'nearest'）、雙線性插值（'bilinear'）、立方核插值（'cubic'）等；

  • [B,xdata,ydata] = imtransform(...)：返回X-Y空間上的輸出圖像B的位置；

  • [B,xdata,ydata] = imtransform(...,Name,Value)：指定影響空間變換的參數。

上述空間變換的應用之一就是實現圖像配準。圖像配準指的是將同一場景的兩幅或多幅圖像進行對準。

這些圖像的拍攝角度、拍攝方式或拍攝距離等有差異，導致這些圖像的匹配點不在圖像中的同一位置。這時，這些圖像簡單的疊加就不能夠實現圖像配準，所以需要找到這些圖像座標系之間的關係，從而實現圖像配準。一般來說， 我們以基準圖像為參照， 並通過一些基準點找到適當的空間變換關係，對輸入圖像進行相應的幾何變換，從而實現它與基準圖像在這些基準點位置上的對齊。

圖像配準的基本過程為：讀入圖像數據——在兩副圖像上選擇足夠匹配點——選擇配准算法，計算變換參數——變換圖像。
