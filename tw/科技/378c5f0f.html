強大而好用的選擇器：focus-within

2020-10-29
| 科技

偽類和偽元素

在開發網頁樣式中，選擇器必不可少，而且選擇器也是在開發css中非常重要的內容，包括常用的類選擇器，id選擇，同時還有偽類，偽類選擇器最大的特點就是冒號開頭。平時也經常會有小夥伴問到，在使用的時候有偽類，也有偽元素，有些添加單冒號，有些添加雙冒號，有沒有什麼區別？

在網上總結的比較多的一個特點就是，其實在使用的時候，無論是偽類還是偽元素都是可以用單冒號的。但是偽元素可以用雙冒號，而偽類只能用單冒號。符合標準而言，單冒號(:)用於 CSS3 偽類，雙冒號(::)用於 CSS3 偽元素。

:focus-within偽類

上面簡單講解一下偽元素選擇器和偽類選擇器，本文的主角是一個新的偽類選擇器：:focus-within。

:focus-within選擇器其實跟:focus是很相似的，但是也有很大的區別，:focus設置之後，是當前元素觸發焦點狀態後的樣式。而:focus-within，通過名字可以猜到，它是當前元素或者後的元素獲取焦點後的樣式。

舉個例子：

<style>	form{		width: 500px;		min-height: 300px;		padding: 10px;	}	/* 第一種情況 */	input:focus{		outline: solid red;	}	/* 第二種情況 */	form:focus-within{		outline: solid red;	}</style><form action="">	<input type="text" id="txt"></form>12345678910111213141516171819

第一種情況是input元素獲取焦點的時候觸發，讓input自身設置outline的樣式；第二種情況，當input獲取焦點後，form設置outline的樣式。

通過這一點我們可以製造出一些常用的交互樣式，簡化JavaScript的代碼。

兼容性

這個屬性有點類似 Javascript 的事件冒泡，從可獲焦元素開始一直冒泡到根元素 html，都可以接收觸發 :focus-within 事件，在上面的示例中就是如此。
