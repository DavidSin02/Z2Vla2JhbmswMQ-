Windows Core OS 最新消息：移植 Win 10 操作中心

2020-10-29
| 科技

微軟 Windows Core OS 又有新消息。

關於微軟正在打造的新一代操作系統 Windows Core OS（WCOS）的消息沒有停過，但是官方又沒有正面給出迴應，外界一直通過各種渠道在瞭解該操作系統。

去年 1 月份，微軟正在開發 WCOS 的消息被披露，從披露的消息來看，WCOS 是一個跨 PC、平板、智能設備與移動等各種物理端的操作系統，它採用模塊化設計，使用 Composable Shell（C-Shell/CShell）模塊化交互界面，以實現跨平臺界面。圍繞 CShell，WCOS 衍生出了眾多不同的版本（子系統/模塊），最早為人熟知的是：

  • AndromedaOS，它支持雙屏，用於摺疊屏的 Surface 設備中；
  • Polaris，它完全基於 UWP 和現代硬件定製，用於 PC 端；
  • Centaurus，用於更大的雙屏移動設備。

除此之外，有開發者發現了其它相關版本代碼以及其適用場景：

  • Oasis，適用於 HoloLens 等混合現實設備；
  • Aruba，適用於 Surface Hub 等巨屏設備；
  • Lite（Santorini），輕量型精簡版，類似 Windows 10 S，只能運行 Microsoft Store 的 UWP/PWA 應用。
  • ……

值得一提的是其中的 Windows Lite OS，因為這個版本已經被曝光得很詳實，我們之前就報導過：Windows Lite OS 代號為“Santorini”，基於 Windows Core OS，它的界面與 Chrome OS 很相似。

這些消息讓大眾對 Windows Core OS 有了一定的瞭解，但微軟官方從來也沒有鬆過口，沒有正面介紹過 WCOS。

到了去年 8 月份，在微軟招募的軟件工程經理崗位說明中明確寫到：SiGMa Device Platform (DPlat)團隊擁有 Windows Core OS（WCOS）的定義和交付權利，而該系統將適用於所有新設備。所招募工程師主要工作是在新硬件上整合軟件體驗，這就相當於證實了 WCOS 的存在，同時微軟團隊正在進行積極的研發。

今年 1 月份的時候，又有消息指出 Windows Core OS 包含了開源組件，這讓其模塊化（子系統化）的猜想進一步得到證實。

而近日，有網友發現微軟某工程師的 Linkedin 檔案上有關於開發 WCOS 的記錄：開發 Action Center 的 Windows Core OS（WCOS）變體（Develop the Windows Core OS (WCOS) variation of Action Center）。

Action Center 其實是 Windows 10 中的操作中心，它是一個快速操作系統通用功能的便捷入口，同時也是一個系統消息通知中心。該工程師的一個職責應該正是將 Windows 10 的這一功能移植到 Windows Core OS 中。

披露出來的種種信息在不斷豐富我們對於 WCOS 的想象，就像文章開頭說的，WCOS 應該是一個集合了眾多“版本”Windows 子系統（模塊）的操作系統，並且其中還有部分子模塊是開源的。

但是必須強調的是，這一切都是外界的小道消息，官方還沒有明確的公告，所以 Windows Core OS 究竟存不存在，是不是這些消息中所描述的樣子？暫時不得而知。另外，據傳言，該操作系統的子系統 Polaris 將會在今年發佈，具體情況如何，我們拭目以待。
