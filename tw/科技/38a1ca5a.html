Java正則表達式組

2020-11-14
| 科技

我們可以通過括號將多個字符組合為一個單位。例如，(ab)。

正則表達式中的每個組都有一個組號，從1開始。

Matcher類中的方法groupCount()返回與Matcher實例相關聯的模式中的組數。

組0引用整個正則表達式和不由groupCount()方法報告。

正則表達式中的每個左括號標記新組的開始。

我們可以在正則表達式中返回引用組號。

假設我們要匹配以“abc"開頭，後跟“xyz"的文本，後跟“abc"。

我們可以寫一個正則表達式為“abcxyzabc"。

我們可以使用反向引用將正則表達式重寫為“(abc)xyz \\ 1"。 \1 指第1組，即(abc)。

\2 引用組2， \3 引用組3，依此類推。

以下代碼顯示如何顯示格式化的電話號碼。在正則表達式 \b(\d{3})(\d{3})(\d{4})\b \b 表示我們感興趣的是僅在字邊界匹配十個數字。

import java.util.regex.Matcher;import java.util.regex.Pattern;public class Main { public static void main(String[] args) { String regex = "\(\\d{3})(\\d{3})(\\d{4})\"; Pattern p = Pattern.compile(regex); String source = "1234567890, 12345, and 9876543210"; Matcher m = p.matcher(source); while (m.find()) { System.out.println("Phone: " + m.group() + ", Formatted Phone: (" + m.group(1) + ") " + m.group(2) + "-" + m.group(3)); } }}

例子

以下代碼顯示如何引用替換文本中的組。

$n ，其中 n 是組編號，替換文本內部是指組 n 的匹配文本。

例如， $1 是指第一個匹配的組。要重新格式化電話號碼，我們將使用($1) $2- $3 。

import java.util.regex.Matcher;import java.util.regex.Pattern;public class Main { public static void main(String[] args) { String regex = "\(\\d{3})(\\d{3})(\\d{4})\"; String replacementText = "($1) $2-$3"; String source = "1234567890, 12345, and 9876543210"; Pattern p = Pattern.compile(regex); Matcher m = p.matcher(source); String formattedSource = m.replaceAll(replacementText); System.out.println("Text: " + source); System.out.println("Formatted Text: " + formattedSource); }}

命名組

我們可以在正則表達式中使用命名組。

我們可以命名一個組，然後使用他們的名字來引用參考組。

我們可以在替換文本中引用組名稱，並使用組名稱獲取匹配的文本。

定義命名組的格式為

(?<groupName>pattern)

一對括號標記一個組。開始括號後面跟著一個?和放在尖括號中的組名稱。

組名稱只能包含字母和數字，且只能以字母開頭。

以下正則表達式具有三個命名組。

  • areaCode
  • prefix
  • postPhoneNumber

正則表達式匹配10位數的電話號碼。

(?<areaCode>\d{3})(?<prefix>\d{3})(?<postPhoneNumber>\d{4})

以下代碼顯示如何使用命名組。

String replacementText = "(${areaCode}) ${prefix}-${postPhoneNumber}";

我們可以混合組號和組名。

上述正則表達式可以重寫如下。

String replacementText = "(${areaCode}) ${prefix}-$3";

以下代碼顯示如何在正則表達式中使用組名稱以及如何在替換文本中使用名稱。

import java.util.regex.Matcher;import java.util.regex.Pattern;public class Main { public static void main(String[] args) { String regex = "\(?<areaCode>\\d{3})(?<prefix>\\d{3})(?<postPhoneNumber>\\d{4})\"; String replacementText = "(${areaCode}) ${prefix}-$3"; String source = "1234567890 and 9876543210"; Pattern p = Pattern.compile(regex); Matcher m = p.matcher(source); String formattedSource = m.replaceAll(replacementText); System.out.println("Text: " + source); System.out.println("Formatted Text: " + formattedSource); }}

組邊界

我們可以使用 start()和 end()方法來獲取組的匹配邊界。 這些方法重載:

int start()int start(int groupNumber)int start(String groupName)int end()int end(int groupNumber)int end(String groupName)

方法返回上一次匹配的開始和結束。

以下代碼顯示如何匹配10位電話號碼，併為每個成功匹配打印每個組的開始。

import java.util.regex.Matcher;import java.util.regex.Pattern;public class Main { public static void main(String[] args) { String regex = "\(?<areaCode>\\d{3})(?<prefix>\\d{3})(?<postPhoneNumber>\\d{4})\"; String source = "1234567890, 12345, and 9876543210"; Pattern p = Pattern.compile(regex); Matcher m = p.matcher(source); while (m.find()) { String matchedText = m.group(); int start1 = m.start("areaCode"); int start2 = m.start("prefix"); int start3 = m.start("postPhoneNumber"); System.out.println("Matched Text:" + matchedText); System.out.println("Area code start:" + start1); System.out.println("Prefix start:" + start2); System.out.println("Line Number start:" + start3); } }}
