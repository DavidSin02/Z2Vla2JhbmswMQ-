企業風險評價，（SCL）安全檢查表，Excel工作表

2020-10-29
| 科技

說實在的，在很多中小企業裡，風險評價在企業運行過程中，還是一項比較概念化的工作，明確地說就是不被重視，似乎有沒有都無所謂。

初次接觸風險評價，感覺十分陌生，不知道從何入手，什麼都不知道，由於要做企業標準建設，不可避免要做風險評價管理工作，包括風險評價制度建立，表格建立，會議記錄，評價結果等等一系列內容。

其實風險評價是建立在對整個企業特別熟悉和了解的基礎之上來做的一項工作，不然真的不知道如何來評價，在第三方的幫助之下，這件工作就比較順利展開了。

這裡表格是以安全檢查表SCL方式進行評價，內容基本涵蓋了所有工作項目，其實這個工作依然停留在紙面，要真正落實到工作當中，我想還是一個漫長的過程。

評價要很多內容，在評價管理制度裡面明確了，這裡不做分析，最終呈現的結果就是這幾張表格，往往也就是這幾張表格，真正反映企業評價後是一個現狀。

最終要做出一個綜合性評價結果，包括一些可能存在的風險，和預防解決辦法等等。

總之風險評價就是要一些未雨綢繆的工作，通常是看不到實際效果的。確切地說，如此評價出來的結果，往往作為一種企業規避問題的一種參考，很有必要。

本來做一個窗體來，對評價過程進行描述，沒有實現，下一次再做。窗體式管理和顯示對於Excel來說是一個很簡單的事情，也相對容易實現。

這些代碼是對評價結果進行顏色標識，以不同顏色對結果分別顯示，看圖可以理解，很容易實現。​

顏色代碼

Private Sub Worksheet_Activate()Dim varr()varr = Array("低風險", "可接受", "一般風險", "較大風險", "重大風險")Dim R As Integer, g As Integer, b As IntegerDim Ro As Range, Rr As RangeSet Rr = Range("K4:K" & Range("K65535").End(xlUp).Row)For Each Ro In Rr    With Ro        Select Case .Value                Case varr(0)                        R = 201                        g = 252                        b = 211                Case varr(1)                        R = 121                        g = 212                        b = 111                Case varr(2)                        R = 211                        g = 182                        b = 181                Case varr(3)                        R = 211                        g = 112                        b = 111                Case varr(4)                        R = 251                        g = 202                        b = 11                Case Else                   Exit Sub        End Select                .Interior.Color = RGB(R, g, b)    End WithNext RoEnd Sub
