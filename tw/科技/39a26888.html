13.56MHz、125KHz、915M、2.4G門禁讀卡器

2020-10-29
| 科技

DAIC-MJ-RW是 13.56MHz 非接觸式射頻 IC 卡專用讀卡器，採用高集成度的射頻芯片、及自帶多奧看門狗電路 CPU 芯片設計。讀卡器性能穩定，讀卡種類多，抗干擾能力強。可支持 ISO14443A/B、ISO15693 國際協議標準，可讀取 MF 卡、二代證、CPU 卡等。可提供韋根 26、韋根 34 通訊接口，適用於多奧門禁、考勤、收費、防盜、巡更等各種射頻識別應用領域。

非接觸式射頻 ID 卡專用用遠距離讀頭，釆用多奧自主研發的 125KHz 解碼電路和譯碼算法，具有卡號識別準確﹑抗干擾能力強、性價比高等特點。讀 ID 遠距離能夠達到 90~120CM 的距離。可應用於門禁，考勤，停車場等領域。

DAID-MJ-RW是 125KHz 非接觸式射頻 ID 卡韋根門禁讀頭器，釆用先進的射頻接收線路設計，通過嵌入式微控制器結合高效譯碼算法，完成對 64bits Read-Only uem4100 兼容式 ID 卡的接收。本門禁讀頭具有工作電流小、讀卡穩定性高、讀卡速度快、性價比高，卡號識別的準確率 100% 等特點。適用於DUOAO門禁、考勤、收費、防盜、梯控等各種射頻識別應用領域。

多奧LEGIC 13.56MHZ 非接觸式射頻 IC 卡專用讀卡器，是目前國際上讀卡種類最多的讀卡器之一。它的讀卡芯片採用了 LEGIC 公司設計的高集成度讀卡專用芯片 SM-4200，可支持國際上所有 13.56MHZ 非接觸卡協議。並且模塊的控制芯片自帶DUOAO看門狗和電壓監控電路，使得該讀卡器具有支持多種卡協議、卡安全性高、性能穩定、功耗低等特點。讀卡器同時擁有 W26/34,RS485 多種接口可選，非常適用於門禁、考勤、收費、酒店梯控等各種射頻識別應用領域。
