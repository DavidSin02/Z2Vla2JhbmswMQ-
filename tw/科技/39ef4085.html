VBA｜與用戶交互的輸入、輸出對話框操作

2020-10-29
| 科技

對於Excel VBA來說，用戶與應用程序的交互主要是通過工作表的數據區來進行的，除此之外，交互的方式還有：

I Excel內置對話框，可以通過VBA代碼調用；

II 可以在VBE中編寫用戶窗體，用戶與應用程序的輸入、輸出交互可以在自定義窗體中進行；

III 調用專門的輸入、輸出對話框；

1 在VBA中，可以通過VBA函數、或對象方法調用對話框輸入數據。

1.1 通過VBA函數InputBox輸入

Sub 使用InputBox函數()

Dim sPrompt As String

Dim sTitle As String

Dim sDefault As String

Dim sReturn As String

sPrompt = "請輸入用戶姓名："

sTitle = "輸入姓名"

sDefault = "伍遠高"

sReturn = InputBox(sPrompt, sTitle, sDefault)

Debug.Print sReturn

End Sub

1.2 通過對象的InputBox方法調用對話框輸入

Sub 使用InputBox方法()

Worksheets("Sheet1").Activate

Set myCell = Application.InputBox( _

prompt:="選擇一個單元格", Type:=8)

End Sub

2 使用MsgBox函數顯示信息對話框

Sub 使用msgbox函數退出系統()

Dim intReturn As Integer

intReturn = MsgBox("真的退出系統碼？", vbYesNo + vbQuestion, "提示")

If intReturn = vbYes Then Application.Quit

End Sub
