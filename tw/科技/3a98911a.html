1

智能化

汽車智能化的終極目標是實現所有條件下都能完成所有的駕駛任務，即完全的自動化。在SAE關於自動駕駛的五級分層定義裡，L3以下稱為輔助駕駛（ADAS）。目前奔馳、寶馬、Tesla上量產應用如ACC、AEB、LKA等均屬此類，典型配置包含：1個77GHz雷達+2個24GHz雷達+ME前視+12個超聲波 + 全景環視 + 域控制器。

儘管在自動駕駛傳感器的三大玩家（視覺、毫米波和激光雷達）裡，毫米波雷達仍因其全天候、結構緊湊及長距離檢測等優勢備受關注。其能夠實現對被測物體相對距離、現對速度、方位的高精度測量。其中24GHz多用於短/中距測量，77GHz用於中/長距離測量。

汽車毫米波雷達種類

當下國內汽車毫米波雷達還處於產業發展初期，與國外差距較大，在設計仿真、開發驗證及生產測試方面面臨諸多挑戰。Keysight提供的軟硬件組成的系統級解決方案，在雷達基帶算法、系統架構設計、雷達信號的生成和分析、目標的仿真、天線測試方面為產業提供全方位的技術服務。

// 設計模擬及評估 //

Keysight 的 SystemVue 軟件包含豐富的汽車雷達庫，結合毫米波汽車雷達系統仿真模板，能夠快速搭建用戶自定義的汽車雷達系統，並可通過場景仿真模型構建真實汽車雷達工作場景。

在雷達射頻微波電路上，Keysight 的 ADS能夠進行精確仿真，將 MMIC、LTCC、鍵合線、封裝、接頭、天線等進行有機集成。

使用 Keysight 的 EMPro，能夠方便的將波導功分網絡、波導至同軸轉換、同軸至微帶轉換以及微帶功分網絡、微帶輻射單元等在一個環境中進行整體仿真、加快仿真效率。

Keysight仿真設計軟件在汽車毫米波雷達中的應用方案

// 產品測試及驗證 //

通過使用信號產生套件（Signal Studio）搭配任意波形發生器（AWG）及高性能信號源（PSG），可以輕鬆產生FMCW、Pulse等雷達測試信號。利用高性能信號分析儀（UXA）搭配矢量信號分析軟件（VSA）可以完成精確的波形分析。

對於汽車防撞雷達這樣工作環境複雜的系統，任何單一的測試項目都無法代替系統性的測試驗證。通過Keysight的雷達目標模擬器，可以輕鬆在實驗室環境下對所探測的目標進行距離、目標RCS以及相對速度的模擬，避免傳統的外場易受天氣及不確定性等因素，以保證測試驗證的一致性。最終可以輕鬆架設24GHz、77GHz、79GHz及更高頻段的雷達測試環境。

Keysight毫米波雷達綜合測試系統

通過以上設計評估、信號產生與分析和系統功能驗證（目標模擬）等一系列的開發和驗證工作之後，將為我們的自動駕駛帶來更多的安全性，並確保其往更高的層面升級。

2

網聯化

對於更高級別的自動駕駛，SAE所定義的L4-L5自動駕駛，則離不開汽車的網聯化。V2X，全名vehicle to everything，包含V2I（車路）、V2P（車人）、V2N（車網）和V2V（車車）四大部分，涵蓋汽車與其他幾乎所有目標進行通信的協議，是實現輔助駕駛甚至無人駕駛的核心技術。

現在已被業內公認的未來提高用戶駕駛感受的一個必要實現途徑，同時也逐漸成為汽車在市場上的一大賣點。V2X早期主要是基於DSRC，專用短距離通信技術，後期隨著蜂窩移動通信技術發展又出現了C-V2X。從技術持續迭代、可靠性和信號傳輸時延等來說，C-V2X有較大優勢。

然而汽車的應用環境極其複雜，無論是DSRC還是C-V2X其信息交互的速率和可靠性的要求與手機等產品不可同日而語。Keysight的UXM網絡模擬器、信道仿真及虛擬路測系統可以更完整更可靠的驗證車聯網模塊，從測試測量維度加速V2X相關技術的發展。

Keysight 智能汽車應用複雜電磁環境模擬系統

在智能化、網聯化的浪潮裡，ADAS甚至自動駕駛技術的不斷革新、高品質車載娛樂影音、OTA升級、V2X、雲計算等一系列技術的發展，已經遠遠超出了CAN或FlexRay等傳統車載總線的數據傳輸能力。

汽車行業需要的高可靠性、高傳輸帶寬、低成本、低電磁輻射和時延等車載網絡系統的需求，極大推進了汽車以太網的發展。而為了確保以上的特性，需針對物理層的一致性、兼容性和功能的測試，對網絡進行壓力測試並針對高低溫、高低溼度、噪音或低電壓等惡劣條件進行測試。

Keysight 可以為您提供完整的L1-L7測試解決方案，100% 覆蓋使用 BroadR-Reach 和 100Base-T1。讓您輕鬆應對產業發展帶來的挑戰，推動自動駕駛和網聯化早日實現。

3

新能源化

在汽車工業智能化、網聯化革命的同時，越來越多的傳統汽車廠宣佈停止或計劃停止內燃機研發、使用更加綠色環保的電驅動系統取而代之，並從簡單的電池發展成為完整的獨立電網。無論您正在進行何種創新，如研發新的智能電網技術、測試電動汽車、製造電池或用於電源管理，能源效率和安全都決定著您的創新能否成功。

Keysight提供全面的電池、BMS、充電樁、車載充電機和逆變器等測試方案，嚴苛模擬驗證電驅動系統的工作環境、狀態，將效率和安全放在第一位，推動綠色能源的健康發展。同時為新能源汽車客戶提供全面可定製化的方案。

從智能電網到HEV/EV的功率傳輸路徑及Keysight全方位解決方案
