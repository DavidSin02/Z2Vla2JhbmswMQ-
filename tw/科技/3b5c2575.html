文本編輯器EmEditor快捷鍵指令

2020-11-14
| 科技

EmEditor是一款非常實用且專業的文本編輯器，該軟件內含眾多便於文本編輯的功能。幫助用戶以輕鬆、便捷、高效的方式處理文本文件，受到眾多用戶的喜愛。

對於一名快節奏的用戶而言，深度運用EmEditor快捷鍵指令是必備的工作方式之一。相對於通過鼠標一步步操作而言，運用EmEditor快捷鍵指令工作將會更加省時省力，快捷方便。

接下來，軟髮網將詳細為大家彙總文本編輯器EmEditor快捷鍵指令。

新建文本：Ctrl+N - 創建一個新的文本文件

打開：Ctrl+O - 打開一個已存在的文件

保存：Ctrl+S - 保存當前文件

打印：Ctrl+P - 打印當前文件

還原默認工作區：Alt+0 - 還原一個已保存的默認工作區狀態

保存默認工作區：Ctrl+Shift+0 - 保存默認的工作區

保存並關閉：Ctrl+E - 保存並關閉當前文件

關閉：Ctrl+F4, Ctrl+W - 關閉當前文件

保存並全部關閉：Ctrl+Shift+E - 保存並關閉所有打開的文件

全部關閉：Alt+Shift+X - 關閉所有打開的文件

撤消：Alt+Backspace, Ctrl+Z - 撤消上一次的操作

重做：Ctrl+Y - 重做上一次撤消的操作

剪切：Shift+Delete, Ctrl+X - 剪切所選內容或當前行至剪貼板

剪切行：Ctrl+L - 剪切選取的行或當前行至剪貼板

複製：Ctrl+C, Ctrl+Insert - 複製所選內容或當前行到剪貼板

複製引用並取消選擇：Ctrl+Shift+Q - 將選區複製引用,粘貼至剪貼板並取消選擇

粘貼：Ctrl+V, Shift+Insert - 在光標位置插入剪貼板中的內容

粘貼為系統默認編碼：Alt+Ctrl+V - 用系統默認編碼粘貼剪貼板上的內容

粘貼為引用文本：Ctrl+B - 插入剪貼板中的內容為引用文本

粘貼包含換行符：Ctrl+J - 插入剪貼板內容以及換行符

循環粘貼：Ctrl+Shift+V, Ctrl+Shift+Insert - 在光標位置處插入剪貼板中的歷史內容

刪除：Shift+Backspace, Delete - 刪除所選內容或刪除當前光標右側的一個字符

刪除左側字符：Backspace - 刪除所選內容或刪除當前光標左側的一個字符

刪除行：Ctrl+Shift+L - 刪除選取的行或當前行

刪除單詞：Ctrl+Shift+Delete - 刪除當前光標所在位置的單詞

刪除右側單詞：Ctrl+Delete - 刪除當前光標右邊的單詞

刪除左側單詞：Ctrl+Backspace - 刪除當前光標左側的單詞

右移一個字符：Right - 將光標向右移動一個字符

左移一個字符：Left - 將光標向左移動一個字符

右移一個單詞：Ctrl+Right - 將光標向右移動一個單詞

左移一個單詞：Ctrl+Left - 將光標向左移動一個單詞

移至行首：Alt+Ctrl+Home - 移動光標到當前行的行首

移至行首或文本起始位置：Home - 移動光標到當前行首個非空的字符位置

移至邏輯行行首：Alt+Home - 移動光標到當前邏輯行的行首

移至行末：End - 移動光標到當前行的行末

移至邏輯行行末：Alt+End - 移動光標到當前邏輯行的行末

查找配對的括號：Ctrl+] - 移動光標到配對的圓括號/方括號

查找匹配標記：Ctrl+. - 移動光標至匹配的標記

製表符或增加行縮進：Tab - 插入製表符或增加當前行縮進

左移製表符或減少行縮進：Shift+Tab - 將光標向左移動一個製表符或減少行縮進

上一行：Up - 移動光標到上一行

下一行：Down - 移動光標到下一行

上一頁：Page Up - 移動光標到上一頁

下一頁：Page Down - 移動光標到下一頁

文檔頂部：Ctrl+Home - 移動光標到當前文檔的頂部

文檔底部：Ctrl+End - 移動光標到當前文檔的底部

上一邏輯行：Ctrl+Page Up - 移動光標到上一個邏輯行行首

下一邏輯行：Ctrl+Page Down - 移動光標到下一個邏輯行行首

窗口頂部：Alt+Page Up - 移動光標到當前窗口頂部

窗口底部：Alt+Page Down - 移動光標到當前窗口底部

交換開始位置和結束位置：Shift+F8 - 交換選區中的開始位置和結束位置

全選：Ctrl+A - 全選

選擇字符：F8 - 切換字符選擇模式

選擇行：Ctrl+F8 - 切換行選擇模式

垂直選擇：Ctrl+Shift+F8 - 切換垂直選擇模式

選擇單詞：Alt+F8 - 選擇當前光標所在位置右側的單詞

往右擴展一個字符：Shift+Right - 把選區往右擴展一個字符

往左擴展一個字符：Shift+Left - 把選區往左擴展一個字符

往右擴展一個單詞：Ctrl+Shift+Right - 把選區往右擴展一個單詞

往左擴展一個單詞：Ctrl+Shift+Left - 把選區往左擴展一個單詞

擴展選區到上一行：Shift+Up - 把選區向上擴展一行

擴展選區到下一行：Shift+Down - 把選區向下擴展一行

擴展選區到上一頁：Shift+Page Up - 把選區向上擴展一頁

擴展選區到下一頁：Shift+Page Down - 把選區向下擴展一頁

擴展選區到行首或文本起始位置：Shift+Home - 把選區擴展到當前行的行首或文本起始位置

擴展選區到邏輯行行首：Alt+Shift+Home - 把選區擴展到當前邏輯行的行首

擴展選區到行末：Shift+End - 把選區擴展到當前行的行末

擴展選區到邏輯行行末：Alt+Shift+End - 把選區擴展到邏輯行的行末

擴展選區到文檔頂部：Ctrl+Shift+Home - 把選區擴展到文檔頂部

擴展選區到文檔底部：Ctrl+Shift+End - 把選區擴展到文檔底部

擴展選區到配對的括號：Ctrl+Shift+] - 把選區擴展到配對的圓括號/方括號

擴展選區到配對的標記：Ctrl+Shift+. - 把選區擴展到配對的標記

取消：Esc - 取消所選內容

大寫：Ctrl+Shift+U - 把選定文本全部轉換為大寫字符

小寫：Ctrl+U - 把選定文本全部轉換為小寫字符

日期與時間：Shift+F5 - 插入日期與時間

插入抑音符：Ctrl+` - 在輸入字符 a、e、i、o、u、A、E、I、O、U、` 或 " 時在該字符上插入抑音符

插入銳音符：Ctrl+' - 在輸入字符 a、d、e、i、o、u、y、A、D、E、I、O、U、Y、' 或 " 時在字符插入銳音符

插入揚抑符：Ctrl+Shift+6 - 在輸入字符 a、e、i、o、u、A、E、I、O 或 U 時在該字符上插入揚抑符

插入波形符：Ctrl+Shift+` - 在輸入字符 a、n、o、A、N或O 時在該字符上插入波形符

插入分音符：Ctrl+Shift+; - 在輸入字符 a、e、i、o、u、y、A、E、I、O、U 或 Y 時在該字符上插入分音符

插入上圓圈符：Ctrl+Shift+2 - 在輸入字符 a、A 或空格時在該字符上插入上圓圈符

插入連字符：Ctrl+Shift+7 - 在輸入字符 a、o、s、A 或 O 時在該字符上插入連字符

插入下加符：Ctrl+, - 在輸入字符 c 或 C 時在該字符上插入下加符

插入劃線符：Ctrl+/ - 在輸入字符 o、O 或 c 時在該字符上插入劃線符

插入變音符：Alt+Shift+6 - 在輸入字符 a、i、o、u、A、I、O 或 U 時在該字符上插入一個變音符

插入長音符：Alt+- - 以鍵入字母的方式插入一個長音符

插入倒轉的問號：Alt+Ctrl+Shift+/ - 插入一個倒轉的問號

插入倒轉的感嘆號：Alt+Ctrl+Shift+1 - 插入一個倒轉的感嘆號

插入版權符：Alt+Ctrl+C - 插入版權符號

插入註冊符：Alt+Ctrl+R - 插入註冊符號

插入商標符：Alt+Ctrl+T - 插入商標符號

插入歐元符：Alt+Ctrl+E - 插入歐元符號

插入特殊字符：Ctrl+Shift+I - 插入一個特殊字符

在上方插入一行：Shift+Enter - 在當前光標位置上方插入新的一行

在下方插入一行：Ctrl+Shift+Enter - 在當前光標位置下方插入新的一行

重複行：Ctrl+Shift+Y - 重複當前邏輯行

插入/改寫：Insert - 切換插入/改寫模式

向上移動行：Ctrl+Shift+Up - 將已選中的行或當前行上移一行

向下移動行：Ctrl+Shift+Down - 將已選中的行或當前行下移一行

編號：Alt+N - 在當前光標或垂直選區中插入編號

半角/全角轉換：Ctrl+T - 轉換選定文本中的半角和全角字符

將所有制表符轉換為空格：Ctrl+Shift+T - 轉換整個文檔中製表符為空格

移動到上次編輯位置：Ctrl+Shift+Z - 移動光標到上次編輯位置

跳轉至標籤：F10 - 跳轉至標籤

跳轉：Ctrl+G - 跳轉到指定行

切換輸入法：Ctrl+F11 - 為遠東語言啟用或禁用輸入法編輯器

跳轉到下一個：F4 - 跳轉到輸出欄或語法檢查器中的下一個位置

跳轉到上一個：Shift+F4 - 跳轉到輸出欄或語法檢查器中的上一個位置

後退：Alt+Left - 移動光標至上一位置

前進：Alt+Right - 移動光標至下一位置

設置或清除書籤：Ctrl+F2 - 為當前行設置書籤,或清除當前行的書籤

清除當前文檔中的所有書籤：Ctrl+Shift+F2 - 清除當前文檔中的所有書籤

轉到本組文檔的下一個書籤：F2 - 轉到當前群組中的下一個書籤

轉到本組文檔的上一個書籤：Shift+F2 - 轉到當前群組中的上一個書籤

下一個節點：Alt+Down - 當顯示大綱視圖時移動光標到下一個節點

上一個節點：Alt+Up - 當顯示大綱視圖時移動光標到上一個節點

擴展到下一個節點：Alt+Shift+Down - 當顯示大綱視圖時擴展選區到下一個節點

擴展到上一個節點：Alt+Shift+Up - 當顯示大綱視圖時擴展選區到上一個節點

下一單元格：Alt+Shift+Right - CSV、TSV或DSV模式下跳轉至下一單元格

上一單元格：Alt+Shift+Left - CSV、TSV或DSV模式下跳轉至上一單元格

下一拼寫錯誤：Ctrl+M - 跳轉至文檔內的下一拼寫錯誤

上一拼寫錯誤：Ctrl+Shift+M - 跳轉至文檔內的上一拼寫錯誤

查找：Ctrl+F - 查找一個字符串

查找下一個：F3 - 查找下一個匹配

查找上一個：Shift+F3 - 查找上一個匹配

查找下一個字：Ctrl+F3 - 查找指定文本的下一個匹配結果

查找上一個字：Ctrl+Shift+F3 - 查找指定文本的前一個匹配結果

添加下一個匹配結果：Ctrl+R - 添加選取指定文本的下一個匹配結果

選擇所有匹配結果：Ctrl+Shift+A - 選擇指定文本的所有匹配結果

替換：Ctrl+H - 替換一個字符串

取消查找高亮：Alt+F3 - 清除搜索字符串的高亮

查找下一個Unicode字符：F9 - 查找下一個無法轉換為要保存的編碼的Unicode字符

查找上一個Unicode字符：Shift+F9 - 查找上一個無法轉換為要保存的編碼的Unicode字符

取消Unicode高亮：Alt+F9 - 取消高亮顯示無法轉換為要保存的編碼的Unicode字符

在文件中查找：Ctrl+Shift+F - 在多個文件中搜索匹配的字符串

將焦點設置在篩選工具欄：Ctrl+Shift+D - 把焦點設到篩選工具欄上

將焦點移至查找工具欄：Ctrl+D - 將焦點移至查找工具欄

不換行：Ctrl+1 - 以不換行的方式呈現每一行

按字符換行：Ctrl+2 - 按指定的字符換行

按窗口換行：Ctrl+3 - 按窗口大小換行

按頁面換行：Ctrl+4 - 按頁面寬度換行

切換輸出欄：Alt+2 - 顯示或隱藏輸出欄

全屏：F11 - 切換全屏模式

重設字體大小：Ctrl+0 - 重設顯示EmEditor字體大小

字符代碼值：Ctrl+I - 顯示 Unicode 字符值

縮小：Alt+O - 縮小顯示一個較大的區域(打印預覽)

放大：Alt+I - 放大顯示一個較小的區域(打印預覽)

重新掃描：F5 - 重新掃描並刷新所比較的文檔

上一更改：Shift+F7 - 跳轉至上次更改

下一更改：F7 - 跳轉至下一更改

上一更改的行：Ctrl+Shift+F7 - 跳轉至上一更改的行

下一更改的行：Ctrl+F7 - 跳轉至下一更改的行

開始/停止宏錄製：Ctrl+Shift+R - 開始或停止宏錄製

運行宏：Ctrl+Shift+P - 運行宏

查找宏關鍵字：F1 - 查找選定宏關鍵字的幫助主題

當前配置屬性：Alt+Enter - 更改當前選定配置的屬性

選擇配置：Shift+F11 - 顯示一個彈出菜單來選擇配置

快速啟動：Ctrl+Q - 快速啟動一個指定的命令

快速啟動 - 命令：Ctrl+Shift+C - 顯示快速啟動窗口並選擇命令模式

快速啟動 - 選項：Ctrl+Shift+O - 顯示快速啟動窗口並選擇選項模式

快速啟動 - 符號：Ctrl+Shift+S - 顯示快速啟動窗口並選擇符號模式

重複上一命令：Ctrl+Shift+G - 重複上一個已執行的命令

用戶菜單(1)：Ctrl+K - 顯示用戶菜單(1),被用作兩次按鍵快捷方式的第一個快捷鍵

終止工具任務：Ctrl+Break - 終止當前正在運行的工具任務

還原窗口位置：Ctrl+9 - 將窗口還原到上次保存的位置

分割：Ctrl+F12 - 分割當前窗口到窗格中

移除分割：Shift+F12 - 移除分割

切換水平分割：Ctrl+- 切換水平窗口分割

下一個窗格：F6 - 切換到下一個窗格

上一個窗格：Shift+F6 - 切換到上一個窗格

向上滾動一行：Ctrl+Up - 將文檔向上滾動一行

向下滾動一行：Ctrl+Down - 將文檔向下滾動一行

下一個：Ctrl+Tab - 切換到下一個文檔

上一個：Ctrl+Shift+Tab - 切換到上一個文檔

右邊的文檔：Ctrl+F6 - 切換至標籤頁右邊的文檔

左邊的文檔：Ctrl+Shift+F6 - 切換至標籤頁左邊的文檔

搜索幫助主題：Ctrl+F1 - 顯示EmEditor幫助主題

打開EmEditor軟件Windows資源管理器：Alt+Shift+E - 使用該文件夾作為起始路徑打開 Windows 資源管理器

以上即為常用EmEditor快捷鍵指令，希望對大家學習或使用EmEditor軟件有所幫助。
