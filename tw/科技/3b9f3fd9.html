2021年雷電4將普及，盤點六款intel 10nm筆記本

2020-10-29
| 科技

2020年深秋搭載intel 11代Tiger Lake-U處理器的筆記本開始扎堆上市，11代處理器率先支持雷電4超高速接口標準，採用10nm製程優勢碾壓14nm牙膏那是毫無疑問的，先進的製程帶來更低功耗更強的性能。在11代i5/i7型號上集成了Iris Xe核心顯卡，性能媲美入門級獨顯，在外圍性能上，雷電4與USB4全面兼容，也向下兼容雷電3，最高速率保持40Gbps，讓用戶一根線享受超高速外鏈與大功率快充。

從2020年開始，基於英特爾處理器的筆記本電腦和臺式機將提供對雷電4技術的集成支持，從而使連接現代工作區比以往任何時候都更加靈活和簡單。

在雷電3規範的基礎上，雷電4技術提高了最低性能，擴展了端到端解決方案功能，支持兩臺4K顯示器、支持擴展4個雷電設備，並增加了新的通用電纜最長可達2m。

根據部分公開信息總結的雷電4、雷電3、USB4、USB3區別表格。

問：intel 11代處理器為什麼只有在筆記本上看到？
答：intel每一次處理器更換製程大換代都會先推出移動版，11代桌面PC處理器要2021年才會上市。
問：intel 11代處理器筆記本都支持雷電4嗎？
答：是否支持要看筆記本廠商具體型號而定，廠家對產品的市場定位、成本等都會做出配置變動，目前開售的大部分11代處理器筆記本都是支持雷電4、雷電3接口的，只有少部分不支持。
問：雷電4接口可以充電嗎？
答：用戶使用PD充電器、PD移動電源連接筆記本的雷電4接口就可以為筆記本提供PD快充了。筆記本的雷電4、雷電3接口也支持反向對外輸出5V3A電力。
問：雷電4跟雷電3有那些區別？
雷電4最高傳輸速率與雷電3一樣均為40Gbps，區別是雷電4對雷電3的某些部分進行“打補丁完善”，例如雷電4接口要求支持連接4個雷電配件、支持2米40Gbps線纜、輕薄型筆記本強制要求最少一個雷電接口支持輸入充電、支持基於VT-d的DMA保護提升安全性等等。
下面是小編蒐集了幾款已上市雷電4接口intel 10nm 11代處理器筆記本，逐一為大家做個介紹。
一、DELL戴爾XPS13-9310

DELL戴爾XPS13 9310搭載全新11代酷睿處理器，i5-1135G7 / i7-1165G7兩種處理器配置可選，內存方面全系列採用16GB DDR4 4267MHz，存儲方面512GB / 1TB兩種容量PCIe固態硬盤可選。集成強大的銳炬Xe顯卡，相比起以往性能低下的HD Graphics核顯，Iris Xe擁有接近獨顯的性能可以讓用戶輕鬆吃雞。

XPS13-9310配備了一塊高達91.5%屏佔比的超窄邊框全面屏，4K分辨率支持觸控操作，16:10的黃金比例讓Excel、Word等辦公工具使用起來更高效，結合Eyesafe物理防藍光技術與90% DCP-P3電影級廣色域，給用戶一塊養眼不偏色的高清屏。

DELL戴爾XPS13-9310機身接口包含了兩個滿血40Gbps雷電3接口，可同時外接2個4K顯示器或顯卡塢，只需要一根線纜輕鬆讓XPS13變身成高性能娛樂主機或工作站，並支持反向充電技術。另還帶有MicroSD卡槽、3.5mm耳機插孔。
XPS13-9310機身採用一整塊6000系航空鋁CNC而成，在保證硬度的同時進一步偷輕重量，色彩有銀色/白色可選，售價方面i5-1135G7 16+512G 4K屏13999元，i7-1165G7 16+512G 4K屏15999元，i7-1165G7 16+1TB 4K屏16999元，均配置了一個45W充電器，可額外購買65W充電器獲得更快的充電體驗。

二、DELL戴爾XPS13-9310 2合1版

DELL戴爾XPS13-9310 2合1版與 XPS13-9310是兄弟款產品，區別是2合1版本採用了特殊屏幕轉軸，搭配同樣高素質的4K 16：10窄邊觸控屏，支持360度翻轉秒變平板模式，更好地進行文檔演示與直觀的交互。

XPS13-9310 2合1採用6000系航空鋁CNC而成，掌託採用輕量而強韌的碳纖維材質製作，並配備4096級壓感Active Pen觸控筆，深淺線條明暗暈染皆可完美書寫。

DELL戴爾XPS13-9310 2和1版本機身接口包含了兩個滿血40Gbps雷電3接口，可同時外接2個4K顯示器或顯卡塢，只需要一根線纜輕鬆讓XPS13變身成高性能娛樂主機或工作站，並支持反向充電技術。另還帶有MicroSD卡槽、3.5mm耳機插孔。
DELL戴爾XPS13-9310 2和1版本黑色/白色兩種顏色可選，定價方面，i7-1165G7 16+512G 1080p屏14999元，i7-1165G7 16+512G 1080p屏16999元。
三、ASUS華碩 靈耀X縱橫

華碩靈耀X縱橫是華碩用於主打輕薄的高性能觸屏筆記本，隸屬於靈耀系列，分為靈耀X縱橫、靈耀X逍遙兩個產品線。華碩靈耀X縱橫配備了一塊13.9英寸92%屏佔比觸摸屏，3300 x 2200分辨率3:2比例，相比起常見的16:9顯示屏，相同對角線長度下顯示面積大了8%，擁有更佳的商務辦公體驗，

華碩靈耀X縱橫擁有一塊5.7英寸超大觸控板，創新的NumberPad將數字小鍵盤融入其中，能快速地輸入大量數字，高效處理工作。採用1.35mm大鍵程與全尺寸鍵距的背光鍵帽，帶來出眾的文本輸入手感。

接口規格方面，華碩靈耀X縱橫擁有兩個滿血雷電4接口，傳輸速率達40Gbps，均支持PD快充。另還帶有一個HDMI接口、一個USB3.2 Type-A接口，一個MicroSD讀卡器，出門使用無需擴展塢也擁有寬容度更高的接口。
華碩靈耀X縱橫採用墨玉黑玉琥珀紅鑽石切邊，鎂合金機身兼顧高強度與輕量化。定價方面，i7-1165G7 16GB+512GB 3.3K觸屏售價為8999元，i7-1165G7 16GB+1TB 3.3K觸屏售價為9999元。
四、ASUS華碩 靈耀X逍遙

華碩靈耀X逍遙與靈耀X縱橫都隸屬於靈耀系列，靈耀X逍遙最大的區別是採用了360度翻轉屏結構，快速方便地從筆記本形態切換成平板創作模式，靈活應對不同使用場合。

屏幕素質方面配備了一塊13.3英寸4K OLED HDR超感屏，分辨率高達3840×2160，擁有100% DCI-P3影院級廣色域，輕鬆勝任專業色彩工作。並支持10點觸控工作，配備了4096級壓感筆，讓你揮灑自如。

接口規格方面，華碩靈耀X逍遙擁有兩個滿血雷電4接口，傳輸速率達40Gbps，均支持PD快速充電。另還帶有一個HDMI接口、一個USB3.2 Type-A接口，一個MicroSD讀卡器，用戶出門也擁有寬容度更高的接口兼容性。
華碩靈耀X逍遙採用墨玉黑玉琥珀紅鑽石切邊，鎂合金機身兼顧高強度與輕量化。定價方面目前只有一個版本開售，i7-1165G7 16GB+512GB 4K OLED觸屏售價8999元。
五、HP Envy 13

日前HP美國官網發佈了新一代搭載11代intel處理器的HP Envy 13，為用戶提供性能強大的便攜辦公體驗，HP Envy 13在官網採用靈活配置的方式開售，用戶可根據自身需求調整大部分配置。

HP Envy 13配備一塊13.3英寸屏幕，從1080p普通屏至4K觸摸屏可選，兩種11代處理器可選i7-1165G7 / i5-1135G7，存儲容量方面分為8GB / 16GB兩種，固態硬盤256GB / 512GB / 1TB三種可選。

接口配置方面，HP Envy 13帶有一個40Gbps帶寬並支持65W輸入充電的滿血雷電4接口，兩個USB3.2 Gen1 USB-A接口，一個HDMI接口，一個3.5mm接口。
HP Envy 13擁有豐富的定製系統，消費者可在官網隨意調整需求的配置，以下只列出三種常見的配置作為參考。i5-1135G7 8GB+256GB 1080p普通屏售價為899美元，i7-1165G7 16GB+512GB 1080p觸屏售價為1209美元，i7-1165G7 16GB+1TB MX450獨顯4K觸屏售價為1609美元。
六、Lenovo聯想 ThinkPad X1 Nano

日前聯想美國官網發佈了最新一代ThinkPad X1 Nano，在僅有907克重量的身體內搭載性能強大的11代intel處理器，並罕見地在ThinkPad X1 Nano上配備了一塊16:10 2K屏，分辨率2160×1350支持100% sRGB色域，不到1Kg的重量讓外出辦公用戶輕鬆無壓力。

ThinkPad X1 Nano在整體設計上保留了ThinkPad元素，經典的小紅點與觸摸板按鍵，繼承優秀的鍵盤打字手感，並支持180度展開屏幕，方便地為客戶進行商業演示，

接口方面，ThinkPad X1 Nano擁有兩個40Gbps雷電4接口，一個3.5mm耳機/麥克風接口，支持65W快速充電技術。定價方面暫未公佈，從洩露的消息來看，i7-1165G7 16GB+1TB 2K觸屏配置售價或為1599美元。

充電頭網總結
以上就是六款支持雷電3接口的11代筆記本處理器介紹，售價從8999元跨至16999元，從性能來看他們全部採用了i5-1135G7 / i7-1165G7這兩種滿血11代處理器，相比起只配備普通核顯的11代i3，11代i5-1135G7 / i7-1165G7帶有性能強大的Iris Xe顯卡，讓輕薄超級本也能擁有入門級獨顯性能。
難能可貴的是他們都配備了雷電3甚至雷電4接口，數量最高的還擁有兩個雷電4接口。雷電3、雷電4接口支持40Gbps超高速數據傳輸與快速充電，全功能可連接各種高速設備、顯卡塢、顯示器、充電等。從這六款先行上市的產品來看，2021年雷電3、雷電4接口將會全面普及，價格將會更加親民。
