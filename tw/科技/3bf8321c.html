LeetCode 2. 兩數相加

2020-11-14
| 科技

題目描述

給出兩個非空的鏈表用來表示兩個非負的整數。其中，它們各自的位數是按照逆序的方式存儲的，並且它們的每個節點只能存儲 一位 數字。

如果，我們將這兩個數相加起來，則會返回一個新的鏈表來表示它們的和。

您可以假設除了數字 0 之外，這兩個數都不會以 0 開頭。

示例：

輸入：(2 -> 4 -> 3) + (5 -> 6 -> 4)輸出：7 -> 0 -> 8原因：342 + 465 = 807

題解

備註:沒有進行輸入檢查,默認是有效輸入數據.

按位依次累加

首先觀察數字和鏈表的對應方式:342--(2->4->3),在鏈表中,最低位是頭節點,那麼明顯的,我們可以將兩個鏈表按照順序累加,同時需要注意會產生進位.

考慮到進位:

  1. 獲取鏈表上相同位數的元素並相加得到x如果之前上一個有進位標誌,則x++,設置進位標誌為false

  2. 判斷x>=10,如果大於等於10,則需要x-=10,並且將進位標誌重新設置成tre

  3. 創建節點等

  4. 在所有的值都處理完成後,如果進位標誌是true,說明需要單獨進位,如99+1 = 100.

鏈表相加:

  1. 兩個鏈表從頭開始,設置p1和p2來依次遍歷

  2. 當p1和p2都不為nullptr時,可以同時遍歷兩個鏈表:p1 != nullptr && p2 != nullptr

  3. 如果p1和p2有一個走到了鏈表終點,即指向了nullptr,那麼我們只需要遍歷另外一個鏈表的剩餘元素(需要考慮進位信息)

/** * Definition for singly-linked list. * struct ListNode { *     int val; *     ListNode *next; *     ListNode(int x) : val(x), next(NULL) {} * }; */class Solution { public:  ListNode *addTwoNumbers(ListNode *l1, ListNode *l2) {    bool isOut = false;  // 進位標誌    ListNode *p1 = l1;    ListNode *p2 = l2;    ListNode *head = new ListNode(0);  // 臨時的頭節點,用來懸掛後續節點    ListNode *cur = head;              // 當前節點    while (p1 != nullptr && p2 != nullptr) {      int x = p1->val + p2->val;  // 計算和      if (isOut)  // 如果上一個有進位,則x++,取消進位標誌      {        x++;        isOut = false;      }      if (x >= 10)  // 判斷當前值是否溢出      {        x -= 10;  // 溢出則-10,並且設置進位標誌        isOut = true;      }      ListNode *tmp = new ListNode(x);  // 構建節點      cur->next = tmp;                  // 懸掛在當前節點後面      cur = tmp;                        // 重設當前節點      p1 = p1->next;      p2 = p2->next;    }    // 獲取到沒有遍歷完的list    ListNode *s = nullptr;    if (p1 != nullptr) s = p1;    if (p2 != nullptr) s = p2;    while (s != nullptr) {      int x = s->val;  // 獲取值      if (isOut)       // 還需要結合進位標誌判斷      {        x++;        isOut = false;      }      if (x >= 10) {        x -= 10;        isOut = true;      }      ListNode *tmp = new ListNode(x);      cur->next = tmp;      cur = tmp;      s = s->next;    }    if (isOut)  // 最後判斷進位標誌,如果還有,說明得單獨創建一個節點    {      ListNode *tmp = new ListNode(1);      cur->next = tmp;    }    ListNode *r = head->next;    delete head;    return r;  }};

執行用時：32 ms, 在所有 C++ 提交中擊敗了81.51%的用戶

內存消耗：9.1 MB, 在所有 C++ 提交中擊敗了75.31%的用戶

常見錯誤

  1. 將鏈表轉換為數字,然後數字相加,最後將數字轉為鏈表.在第一步轉換數字的時候,鏈表可以存儲超大數,但是數字類型long long等不一定能夠存放下.

  2. 忘記考慮最後的一個進位標誌.
