原來將 Python 程序打包成exe文件這麼簡單！只需三步

2020-11-14
| 科技

一、在cmd命令行執行安裝pyinstaller工具

pip install pyinstaller


二、下載安裝pyinstaler運行時所需windows擴展：pywin32

https://github.com/mhammond/pywin32/releases選擇最新版的下載，注意選擇與python版本和位數對應的如32位python3.7.4對應下載：【pywin32-223.win32-py3.7.exe】如64位python3.7.4對應下載：【pywin32-223.win-amd64-py3.7.exe】

三、在命令行中執行打包命令即可

格式如下：pyinstaller [opts] yourprogram.py參數含義-F 指定打包後只生成一個exe格式的文件(建議寫上這個參數)-D –onedir 創建一個目錄，包含exe文件，但會依賴很多文件（默認選項）-c –console, –nowindowed 使用控制檯，無界面(默認)-w –windowed, –noconsole 使用窗口，無控制檯-p 添加搜索路徑，讓其找到對應的庫。-i 改變生成程序的icon圖標例1.指定文件路徑打包：pyinstaller -F D:\project\tanchishe.py例2.不帶窗口，後臺運行打包：pyinstaller -F -w D:\project\tanchishe.py例3.更換程序圖標打包：pyinstaller -F -w -i D:\project\test.ico D:\project\tanchishe.py
這裡咱們執行開始打包：(venv) C:\Users\www_9\PycharmProjects\python-tanchishe>pyinstaller -F tanchishe.py
