一分鐘掌握異步操作(Ajax請求）

2020-10-29
| 科技

1、所有瀏覽器均支持“jQuery中Ajax”，出錯的原因大部分是：

(1)有些jQuery中的內置函數，在某些瀏覽器中不支持==>在js代碼中儘量少使用jQuery的內置函數，儘量放到PHP語言中去處理！

(2)異步請求的地址錯誤：地址不存在

2、jQuery“鍵盤事件”的注意事項：

一定要加上“event”事件，即：

$('#keywords').keyup(function(event){

showKeywordsList(event);

})

三種常用異步方式：get、post、ajax

調用格式：

$.get(url,[data],[callback],[type]);

$.post(url, [data], [callback],[type]);

參數：

url(String)：待載入頁面的URL地址

data(可選)：待發送 Key/value 參數。

callback(可選) Function：載入成功時回調函數。

type(String)(可選)：返回內容格式，xml, html, script, json, text, _default。

url地址的兩種傳遞方式：

(1)第一種

<script>

var MyformUrl='/PHPxueba15/App/Modules/Admin/Tpl/Public/';

var AjaxUrl = '/PHPxueba15/Admin/Article';

</script>

(2)第二種

<a href="" url="{:U('Member/Cart/addCart',array('gid'=>$goods['gid']))}" id="addCart"></a>

var url = $('#addCart').attr('url');

第一種方式：get方式

$.get("{:U(GROUP_NAME.'/List/keywordsList')}",

{keyword:keyword},

function(data){

$('#message_tips').html(data.data['message']);

$('#task_tips').html(data.data['task']);

$('#event_tips').html(data.data['event'] );

$('#contract_tips').html(data.data['url'] );

},'json');

第二種方式：post方式

$.post('{:U(GROUP_NAME.'/List/keywordsList')}',

{ uid:uid,

passwd:passwd,

phone:phone

},

function(data){

$('#message_tips').html(data.data['message']);

$('#task_tips').html(data.data['task']);

$('#event_tips').html(data.data['event'] );

$('#contract_tips').html(data.data['url'] );

},

'json');

服務器端：

3.1版本以後，ajaxReturn方法可以更加靈活的進行ajax傳值，能夠完全定義傳值的數組和類型，例如：

$data['message'] = '信息';

$data['task'] = 'info';

$data['event'] = '事件';

$data['url'] = $url;

$this->ajaxReturn($data,'JSON');

第三種方式：ajax方式

$.ajax({

type:'GET',

url:'{:U(GROUP_NAME."/List/keywordsList/id/")}'+department_id,

data:'id='+department_id+'&num='+goodsNum,

async:false,

success:function(data){

options = '';

$.each(data.data, function(k, v){

options += '<option value="'+v.position_id+'">'+v.name+'</option>';

});

$("#role").html(options);

},

dataType:'json'

});

服務器端：

public function getPositionList() {

if($_GET[id]){

$m_position = M('position');

$where['department_id'] = $_GET[id];

$position_list = $m_position->where($where)->select();

$this->ajaxReturn($position_list, "獲取成功！", 1);

}else{

$this->ajaxReturn($position_list, "請選擇部門！", 0);

}

}

1、注：async(Boolean)

(默認: true) 默認設置下，所有請求均為異步請求;

如果需要發送同步請求,請將此選項設置為 false;

注意,同步請求將鎖住瀏覽器，用戶其它操作必須等待請求完成才可以執行;

2、有關dataType的相關說明：

dataType的默認數據格式是：html(默認輸出的是html標籤形式的數據)

json格式的數據:輸出的是一個“對象”，按對象的方式調用數據

3、Ajax使用技巧

(1)如果前端只是想知道，服務端是否成功響應了

前端：dataType不用設置

服務器端:echo 0;-->表示失敗

echo 1;-->表示成功

(2)如果前端想獲得服務器端返回的數據

前端：dataType設置為“json”

服務器端：

$this->ajaxReturn($position_list, "獲取成功！", 1);或者

$this->ajaxReturn($result,'JSON');

exit(json_encode($result));

例1：

服務器端：

$data['message'] = '信息';

$data['task'] = 'info';

$data['event'] = '事件';

$data['url'] = $url;

$this->ajaxReturn($data,'JSON');

前端：

$('#message_tips').html(data.data['message']);

例2：

服務器端：

$result = array(

'status'=>true,

'num'=>$num

);

exit(json_encode($result));

前端：

oGoodsNum.val(result.num);

例3：

前端：dataType設置為“html”

$this->ajaxReturn('<div>測試</div>', "獲取成功！", 1);

服務器端：

/**

* 異步驗證管理員帳號

*/

Public function checkAname () {

if (!$this->isAjax()) halt('頁面不存在');

$aname = $this->_post('aname');

$where = array('aname' => $aname);

if (M('admin')->where($where)->find()) {

echo 1;

} else {

echo 0;

}

}

/**

* 異步驗證密碼

*/

Public function checkApwd () {

if (!$this->isAjax()) halt('頁面不存在');

$aname = $this->_post('aname');

$apwd = $this->_post('apwd', 'md5');

$where = array('aname' => $aname);

$password = M('admin')->where($where)->getField('apwd');

if ($password && $password == $apwd) {

echo 1;

} else {

echo 0;

}

}

/**

* 異步驗證驗證碼

*/

Public function checkVerify () {

if (!$this->isAjax()) halt('頁面不存在');

if ($_SESSION['verify'] == $this->_post('verify', 'strtolower')) {

echo 1;

} else {

echo 0;

}

}

/* Ajax方法 */

function treemodelajax(data,type) {

$.ajax({

url:url+'/Classification/'+type,

type:"post",

data:data,

dataType:"json",

success:function(data) {

if(data == 1) {

//alert("操作成功！");

$("#treemodelprompt .img").html("✔");

$("#treemodelprompt .text").html("操作成功！");

$("#treemodelprompt .img").css("color", "#5EBB29");

$("#treemodelprompt .text").css("color", "#5EBB29");

$("#treemodelprompt").slideToggle(500);

treemodelsetintervalset();

} else {

//alert("操作失敗！");

$("#treemodelprompt .img").html("✘");

$("#treemodelprompt .text").html("操作失敗！");

$("#treemodelprompt .img").css("color", "#F00");

$("#treemodelprompt .text").css("color", "#F00");

$("#treemodelprompt").slideToggle(500);

treemodelsetintervalset();

}

}

});

}

倒計時關窗：

/* 倒計時方法 */

var i = 5;

function treemodelsetintervalset() {

i = 5;

setInterval("treemodelsetinterval()", 1000);

}

function treemodelsetinterval() { //倒計時關閉跳轉窗口

if(i == 1) {

$(document).ready(function() {

$("#treemodelprompt").slideUp(500);

});

} else {

i--;

}

}
