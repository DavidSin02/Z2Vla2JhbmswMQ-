應用程序編程接口

請參見 SOLIDWORKS API 幫助： 版本說明了解最新更新。

SOLIDWORKS 2020 API 包括以下功能：

  • 將新的選擇列表附加到之前暫停的選擇列表。
  • 創建和編輯：鏡向零部件、使用新方法的圓角和倒角、多實體零件爆炸視圖。
  • 查找圓角和倒角中的缺失參考，並將它們重新分配給新的邊線。
  • 獲取或設置：派生零件和陣列特徵數據對象中的更多屬性、草圖是否可編輯、導入到零件草圖時是否將寬度折線轉換為實體填充剖面線。
  • 將模型切割為 2D 草圖剖面。
  • 對實體切割清單文件夾排序。
  • 使用 3D Interconnect 功能。
  • 使用改進的質量屬性報告。

系統選項和文檔屬性更改

將在軟件中添加、更改或移除以下選項。

系統選項

可通過工具 > 選項 > 系統選項訪問這些系統選項。訪問列列示選項所在系統選項的子區域。

文檔屬性

可通過工具 > 選項 > 文檔屬性訪問這些文檔屬性。訪問列列示選項所在文檔屬性的子區域。
