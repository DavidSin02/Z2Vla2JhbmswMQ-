3. 工作界面

從 Excel 2007 版開始，Excel 的工作界面風格就一直延續至今，其中設置了一些便捷的工具欄和按鈕，如【快速訪問工具欄】、【拼寫檢查】按 鈕、【分頁瀏覽】按鈕和【顯示比例】滑動條等。

Excel 2016工作界面

啟動 Excel 2016 後，將打開圖 1-7 所示的啟動界面。該界面的左側為最近使用的文檔列表和【打開工作簿】選項，用戶可以通過其打開最近打開的 Excel 文檔或電腦中保存的其他 Excel 文件。

Excel 2016的工作界面

在 Excel 工作界面中，功能區選項卡是 Excel 窗口界面中最重要元素，通常位於標題欄的下方。由一組選項卡面板所組成，單擊選項卡標籤可以切換到不同的選項卡功能板。

3.1 功能區選項卡

在 Excel 的功能區選項卡中，當前選中的選項卡也稱為“活動選項卡”。每個選項卡中包含了多個命令組(例如“剪貼板”命令組、“字體”命令 組、“對齊方式”命令組)，每個命令組通常由一些相關命令組成。

例如，下圖所示的【字體】命令組。

【開始】選項卡【字體】命令組中的各種命令控件

通過執行命令組中的各種命令控件，用戶可以執行 Excel 所提供的各種功能(參見01-Excel概中關於Execl主要功能的介紹)。

在 Excel 功能區選項卡中主要選項卡的功能說明如下。

【文件】選項卡

該選項卡是一個比較特殊的功能區選項卡，其由一組命令列表及其相關的選項區域組成，包含【信息】、【新建】、【打開】、【保存】、【另存為】、【歷史記錄】等選項。

選擇【文件】選項卡後顯示的菜單

選擇命令列表中的不同選項，將顯示不同的選項區域，例如：

  • 選擇【信息】選項，在顯示的選項區域中，可以設置保護、檢查、管理工作簿，並設置瀏覽器視圖選項(即用瀏覽器查看工作簿時，可以看到的內容)。

【打開】選項區域

  • 選擇【新建】選項卡，在顯示的選項區域中用戶可以新建空白工作簿，或使用 Excel 工作簿模板創建新工作簿。

【新建】選項區域

  • 選擇【打開】選項，在顯示的選項區域中，用戶可以選擇打開 OneDrive(微軟公司提供的雲存儲服務)或當前電腦中保存的工作簿文件。此 外，單擊【瀏覽】選項，還可以通過打開的打 開【打開】對話框，打開電腦或局域網中的Excel 工作簿文件。

【打開】選項區域

  • 選擇【保存】選項，將立即保存當前打開的工作簿文件(若當前工作簿從未執行過“保存”操作，將打開【另存為】選項區域)。
  • l選擇【另存為】選項，在打開的選項區域中， 用戶可以選擇工作簿文件的保存位置，單擊 【瀏覽】選項，在打開的【另存為】對話框中用戶可以設置工作簿文件的具體保存位置。

【另存為】選項區域

  • 選擇【打印】選項，在顯示的選項區域中用戶可以設置打印工作簿的打印機、打印份數、打印排序、打印方向、紙張、邊距和縮放等參數。在選項區域的右側，Excel 會顯示當前選中 工作簿的打印預覽效果，在其中確認打印效果後，單擊【打印】按鈕即可打印工作簿。

【打印】選項區域

  • 選擇【共享】選項，在顯示的選項區域中用戶可以設置將當前工作簿保存到“雲”(OneDrive)，從而實現與網絡用戶的共享;或者通過發送“電子郵件”，將當前工作簿作為郵 件的附件（或鏈接）發送給指定的郵件聯繫人。

【共享】選項區域

  • 選擇【導出】選項，在打開的選項區域中用戶可以將當前打開的工作簿導出為 PDF、XPS 格式的文檔，或將工作簿的文件格式類型修改為模板、OpenDocument 電子表格(.dos 文件)、啟用宏的工作簿或二進制工作簿。

【導出】選項區域

  • 選擇【發佈】選項，在顯示的選項區域中，用戶可以將當前工作簿文件發佈到 Power BI。

【發佈】選項區域

  • 選擇【關閉】選項，將立即關閉當前打開的工作簿(若當前工作簿尚未保存，Excel 將打開提示對話框，提示用戶保存工作簿)。
  • 選擇【賬戶】選項，在打開打開的選項區域 中，用戶可以選擇登錄 Office，查看關於Excel 軟件的許可條款、支持、產品 ID，或者更改當前軟件的產品密鑰。

【賬戶】選項區域

選擇【選項】選項，將打開【Excel 選項】對話框(該對話框的作用將在後面詳細介紹)。

【開始】選項卡

【開始】選項卡包含 Excel 中最常用的命令，包括【剪貼板】、【字體】、【對齊方式】、【數字】、【樣式】、【單元格】和【編輯】等 7 命令組，用於基本的剪貼數據、字體格式化、單元格對齊、單元格格式和樣式設置、條件格式、單元格和行列的插入刪除以及數據編輯。

【開始】選項卡

3.3 快速訪問工具欄

Excel 各個版本工作界面中的快速訪問工具欄位於界面的左上角(如下圖所示)，它包含一組常用的快捷命令按鈕，並支持自定義其中的命令，用戶可以根據工作需要添加或刪除其所包含的命令按鈕。

Excel快速訪問工具欄中默認顯示的命令按鈕

快速訪問工具欄默認包含【保存】按鈕、【撤銷】按鈕 、【恢復】按鈕 等 3 個快捷命令按鈕。單擊工具欄右側的【自定義快速訪問工具 欄】按鈕 ，可以在彈出的列表中選擇在快速訪問工具欄中顯示更多的內置命令按鈕，例如【快速打印】、【拼寫檢查】和【新建】等。

3.4 各種命令控件

Excel 工作界面中包含了多種命令，這些命令通過多種不同類型的控件顯示或隱藏在界面窗口中。下面將簡要介紹命令組中各種命令控件的類型及其功能說明。

按鈕

按鈕(Button)可以通過單擊執行一項命令或一項操作。以工作界面底部狀態欄為例，其中包括的 【普通】、【頁面佈局】和【分頁預覽】按鈕，用於切換當前 Excel 窗口的視圖模式。

Excel狀態欄上用於切換視圖模式的三個按鈕

切換按鈕

切換按鈕(ToggleButton)可以通過單擊按鈕在“激活”和“未激活”兩種狀態之間來回切換。 例如【審閱】選項卡中的【顯示所有批註】和【顯示墨跡】切換按鈕。

  • 激活狀態下的切換按鈕

激活切換按鈕，顯示工作表中的所有批註

  • 未激活狀態下的切換按鈕

未激活的【顯示所有批註】切換按鈕

下拉按鈕

下拉按鈕(DropDown)包含一個黑色倒三角標識符號，通過單擊下拉按鈕可以顯示詳細的命令列表或圖標庫，或顯示多級擴展菜單。例如單擊【公式】選項卡中的【自動求和】下拉按鈕，將顯示下圖所示的命令列表。

單擊【自動求和】下拉按鈕彈出的命令列表

又如，單擊【插入】選項卡中的【形狀】按鈕將顯示如下圖所示的圖標庫。

單擊【形狀】下拉按鈕後彈出的圖標庫

拆分按鈕

拆分按鈕(SplitButton)是一種由按鈕和下拉按鈕組成而成的按鈕形式。單擊拆分按鈕可以執行特 定的命令，而單擊其下方的下拉按鈕則可以在彈出的下拉列表中選擇其他相近或相關的命令。例如【開始】選項卡中的【粘貼】拆分按鈕。

單擊【粘貼】拆分按鈕後彈出的下拉列表

文本框

文本框(TextBox)可以顯示文本，並且允許用戶對其中的內容進行編輯。例如功能區下方的名稱框。

功能區下方的名稱框

庫

庫(Gallery)包含了一個圖標容器，在其中顯示一組可供用戶選擇的命令或方案圖標。例如【圖標工具】|【設計】選項卡中的【圖表樣式】庫，單擊右側的上/下三角箭頭按按鈕，可以切換顯示不同行中的圖標項。

【圖表樣式】庫

單擊上圖中【圖表樣式】庫右下角的擴展按鈕，可以 顯示庫內容。

組合框

組合框(ComboBox)由文本框、下拉按鈕和列表框組合而成，用於多種屬性選項的設置。通過單擊其右側的倒三角按鈕 ，可以在彈出的下拉列表框中選取列表項，被選中的列表項會顯示在組合框的文本框中。另外，用戶也可以直接在組合框的文本框中輸入具體的選項名稱後，按下 Enter 鍵確認，使選擇生效。常見的組合框如【開始】選項卡中的【字體】組合框。

【字體】組合框既可以輸入文本也可以顯示下拉列表

微調按鈕

微調按鈕(SpinButton)包含一對方向相反的三角箭頭按鈕，通過單擊箭頭按鈕，可以對文本框中的數值大小進行調節。例如【頁面佈局】選項卡中的【縮放比例】微調按鈕。

複選框

複選框(CheckBox)與切換按鈕的功能類似，通過單擊複選框可以在【選中】和【取消選中】兩個選項狀態之間來回切換。例如上圖中【頁面布 局】選項卡中的【查看】、【打印】複選框。

對話框啟動器

對話框啟動器(Dialog Box Launcher)是一種特殊的按鈕，它位於功能區選項卡中命令組的右下 角，並與其所在的命令組功能相關聯。對話框啟動器顯示為斜角箭頭按鈕：

【頁面設置】命令組右下角的對話框啟動器

單擊該按鈕可以打 開與特定命令組相關的對話框，例如單擊【頁面佈局】選項卡【工作表選項】命令組中的對話框啟動器，將打開如下圖所示的【頁面設置】對話框。

單擊對話框啟動器打開相應的對話框

選項按鈕

選項按鈕(OptionButton)也稱為“單選按鈕”通常由兩個以上的選項按鈕成組出現。單擊選中其中一個選項按鈕後，將自動取消其他選項按鈕的選中狀態。例如上圖對話框中的【打印程序】組中【先列後行】和【先行後列】單選按鈕。

編輯框

編輯框(EditBox)由文本框和文本框右側的摺疊按鈕組成，文本框內可以直接輸入或編輯文本，單擊摺疊按鈕可以在工作表中直接框選目標區域， 目標區域的單元格地址會自動填寫在文本框中， 例如在【插入】選項卡中單擊【表格】按鈕，打開【創建表格】對話框中的【表數據的來源】編輯框。

3.5 快捷菜單和快捷鍵

快捷菜單

Excel 中許多常用操作除了可以通過功能區選項卡上選擇對應的命令控件執行以外，還可以在快捷菜單中選定執行。在 Excel 中，單擊鼠標右鍵就可以顯示快捷菜單，菜單中顯示的內容取決於當前選中的對象。

右擊Excel單元格時，彈出的快捷菜單

在使用 Excel 時，使用快捷菜單可以使命令的選擇更加快速有效。例如，在選定一個單元格後右擊鼠標，在彈出的菜單中將顯示包含單元格格式操作等命令的快捷操作。

上圖中顯示在菜單上方的工具欄稱為“浮動工具欄”，主要包含了單元格格式的一些基本命令，例 如【字體】、【字號】、【字體顏色】、【格式刷】等。

快捷鍵

除了快捷菜單以外，用戶還可以在 Excel 中藉助快捷鍵來執行命令，按下 Alt 鍵將在當前功能區選項卡上可執行的快捷鍵提示，如下圖所示，根據提示鍵可以執行許多快捷操作。

按下Alt鍵在功能區選項卡上顯示可執行的快捷鍵

例如，在上圖所示的提示下

  • 按下“1”鍵可以打開【另存為】界面；
  • 按下“2”鍵可以打開【撤銷】列表；
  • 按下“3”鍵可以執行【恢復】命令；
  • 按下F 鍵可以打開【文件】選項卡；
  • 按下 H 鍵可以打開【開始】選項卡；
  • 按下 N 鍵可以打開【插入】選項卡；
  • 按下 M 鍵可以打開【公式】選項卡；
  • 按下 F 鍵可以打開【頁面佈局】選項卡；
  • 按下 A 鍵可以打開【數據】選項卡；
  • 按下 R 鍵可以打開【審閱】選項卡；
  • 按下 W 鍵可以打開【視圖】選項卡；
  • 按下 L 鍵則可以打開【開發工具】選項卡。.

以執行【公式】選項卡中【定義名稱】命令組內【名稱管理器】命令按鈕為例，依次按下 Alt 鍵、M 鍵，打開下圖所示的【公式】選項卡，然後按下 N 鍵即可。

掌握快捷鍵的應用方法後，在日常工作中反覆使用，就可以逐漸脫離鼠標使用鍵盤完成對 Excel 報表的操作，從而大大提高表格處理效率。

此外，Excel工作界面中處理表格時，還有一些可以快速執行某些操作或切換的快捷鍵，這些快捷鍵通常與Ctrl、Alt或Shift鍵組合使用，理解這些快捷鍵的作用，找到與自己工作相關的快捷鍵，並通過刻意地練習掌握它們的使用方法，對於提高Excel軟件的使用效率，非常重要。

  • Ctrl：control控制，用於選擇時按住Ctrl常用於選擇不連續的單元格或表格。
  • Alt ：alternative替代，
  • Shift：shift轉換，用於選擇時按住Shift常用於選擇連續的單元格區域或表格
