撬動offer：兩個長字符串數字相加

2020-11-14
| 科技

算法題千奇百怪，今天分享的這樣一道算法題。


0x01：題目

題目一：有兩個字符串分別如下：

String a = "10101001101010100110";

String b = "1011011101100";

字符串裡的數字或者是0，或者是1。編寫一個函數實現這兩個數的二進制字符串相加。考慮都是無符號數字相加。

題目變種

有兩個字符串分別如下：

String a = "1879987637894818799876378948";

String b = "176898";

字符串裡的數字是0到9的任意一個數字字符。編寫一個函數實現這兩個十進制字符串的相加。考慮都是無符號數字相加。


0x02：分析

如果a和b這兩個字符串的數字位數不是很多大，確實直接用Integer或者Long類型轉換就可以求出a+b了，但是如果a和b的位數都是幾百位呢？那麼就不可以使用簡單的數字相加的方法了，因為a或者b早就溢出了，Integer或者Long已經無法表示a或b這兩個數值了。這時候要怎麼計算呢？這種情況下，可以想到如下一些解決方案

  • 用字符串來存儲a和b的值；
  • 用數組來存儲a和b的值；
  • 用鏈表來存儲a和b的值。

可能有其他解決方案，請大家想想。以上這三種方案都有一個共同點，就是無論哪一種方案，都可以不需要擔心因為數字太大而造成數值溢出出錯。除了內存溢出的情況。

回想一下老師教的數字是怎麼相加，其實就是從數字的最右邊的最後一位開始（就是個位開始計算），將兩個對應的位進行相加，對於十進制，如果相加的數字大於等於10，就往前進一位。就這樣教計算機進行計算即可。（如果是二進制就是逢2進1即可，原理一樣）


0x03：代碼實現

Java實現如下，可能不是最優解，其他時間複雜度和空間複雜度的大家開動腦筋，發散思維想想。

public class TestAdd {    public String add(String a, String b){        a = getStr(a);        b = getStr(b);        if(a.equals("") && b.equals("")){            return "";        }        String res = "";        int addOne = 0;//用於判斷是否需要進位        char[] aCharArray = a.toCharArray();        char[] bCharArray = b.toCharArray();        int aIndex = a.length();        int bIndex = b.length();        while(aIndex>0 || bIndex>0){            int aTemp = getNumber(aCharArray, aIndex);            int bTemp = getNumber(bCharArray, bIndex);            System.out.println("aTemp = " + aTemp + ", bTemp = " + bTemp);            int sum = aTemp + bTemp + addOne;            if(sum > 9){                addOne = 1;                sum = sum % 10;            }else{                addOne = 0;            }            res  = sum + res;            System.out.println("res = " + res);            aIndex--;            bIndex--;        }         // 當a和b都全部加完了之後，再最後一次判斷是否需要進位        if(addOne==1){            res = "1" + res;        }        return res;    }    public int getNumber(char[] cs, int index){        try{            return Integer.parseInt(String.valueOf(cs[index-1]));        }catch (Exception e) {            //如果出現異常說明數組越界返回0， 加0等於沒加        }        return 0;    }    public String getStr(String a){        if(a==null){            return "";        }        return a.trim();    }    public static void main(String[] args) {        TestAdd ta = new TestAdd();        System.out.println(ta.add("19", "9111"));    }}
