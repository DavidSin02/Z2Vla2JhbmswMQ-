第一作者簡介：楊必勝(1974-), 男, 教授, 博士生導師, 研究方向為三維地理信息獲取與分析。E-mail:bshyang@whu.edu.cn

摘要：三維激光掃描直接對地球表面進行三維密集採樣，可快速獲取具有三維座標（X，Y，Z）和一定屬性（反射強度等）的海量、不規則空間分佈三維點雲，成為數字化時代下刻畫複雜現實世界最為直接和重要的三維地理空間數據獲取手段，在全球變化、智慧城市、全球製圖等國家重大需求和地球系統科學研究中起到十分重要的作用。目前，在傳感器技術和國家需求的雙重驅動下，三維激光掃描在硬件裝備、三維點雲數據處理以及應用3個方面取得了巨大的進步，同時也面臨新的挑戰。本文以三維激光掃描的發展歷史為線索，總結了三維激光掃描系統的現狀、三維點雲數據處理的關鍵進展以及在測繪地理信息等領域的典型應用，並分析了三維點雲數據處理面臨的挑戰，最後展望了三維激光掃描與點雲處理的發展趨勢。

Progress, Challenges and Perspectives of 3D LiDAR Point Cloud Processing

YANG Bisheng, LIANG Fuxun, HUANG Ronggang

Abstract: 3D LiDAR can perform an intensive sampling of the earth surface in a direct way, and yield the 3D point cloud that contains numerous and scattered points with the coordinates (X, Y, Z) and attributes (e.g., intensity). As the vital 3D geospatial data for description of the world in the digital era, 3D point cloud plays an important role not only in earth science researches but also in national requirements (e.g., global change analysis, global mapping, and smart city). Inspired by sensor technologies and national requirements, 3D LiDAR has got great progresses in hardware, data processing and applications, and is facing new challenges. Following the history of 3D LiDAR, this paper first reviews the status of 3D LiDAR system, and introduces the development of key technologies in data processing. Then the typical applications of 3D LiDAR in surveying and other related fields are listed, and current challenges in point cloud processing are concluded. Finally, some future perspectives are presented.

Key words: 3D LiDAR point cloud point cloud fusion object extraction 3D representation ubiquitous point cloud

激光掃描/激光雷達(light detection and ranging，LiDAR)是一種非接觸主動式快速獲取物體表面三維密集點雲的技術，已成為高時空分辨率三維對地觀測的一種主要手段[1]。該技術可直接獲取具有三維座標(X,Y,Z)和一定屬性(反射強度等)的海量、不規則空間分佈的三維點雲，而且受天氣影響小，同時具有一定的穿透性，在全球變化、智慧城市、資源調查、環境監測、基礎測繪等領域發揮越來越重要的作用[2]。

經過20多年的發展，三維激光掃描硬件在穩定性、精度、易操作性等方面取得了長足的進步，尤其是在機載/車載/地面三維激光掃描方面進展顯著，其中具有代表性的三維激光掃描硬件研製廠商包括：Riegl、Leica、Optech、Velodyne、北科天繪、海達數雲等。此外還有美國NASA的ICESat和中國資源三號02星搭載的激光雷達。

在三維點雲數據處理軟件方面，商業化的軟件主要有TerraSolid公司的TerraSolid、Trimble公司的RealWorks、Leica公司的Cyclone、Bentley公司的Pointools、Orbit GT公司的Orbit Mobile Mapping等，以及國內科研院所和公司開發的一些工具軟件。其重點主要集中在點雲數據的管理、面向DEM生產的濾波[3-4]、三維建築物提取及重建[5-6]、森林垂直結構參數提取[7]等方面。相對三維激光掃描硬件設備的快速發展而言，三維點雲的智能化處理髮展較為落後，點雲處理的智能化水平、軟件界面友好性、專業化應用數據接口等方面還有待提高。

在回顧三維激光掃描發展歷史的基礎上，本文分析總結了三維激光掃描硬件裝備、三維點雲數據處理以及工程化實踐方面取得的成就和麵臨的挑戰，最後展望三維激光掃描與點雲處理的發展趨勢。

1 三維激光掃描硬件系統的現狀

近20多年來，國內外廠商在三維激光掃描硬件系統方面進行了各種努力和創新。表 1列舉了目前以星/機/車/地為搭載平臺的幾類典型激光掃描硬件系統及其參數指標。從技術發展趨勢來看，三維激光掃描正在從低精度(釐米級)獲取向高精度(毫米級)獲取，從幾何與強度的採集走向幾何與多/高光譜協同採集。相比國外，國內激光掃描硬件起步晚且仍有較大差距。此外，便攜式/揹包式/無人機為平臺的輕小型三維激光掃描裝備正蓬勃發展。例如：Riegl公司研製的集成VUX的低空無人旋翼機激光掃描系統，武漢大學測繪遙感信息工程國家重點實驗室研製的低空無人直升機激光掃描系統Heli-Mapping，北京拓維思科技有限公司研製的“巡線鷹”等。隨著SLAM(simultaneous location and mapping)等技術的發展，三維激光掃描通過與機器人等平臺集成，可實現自主測量[8]。

三維激光掃描系統的幾何測量原理基本相同，通過激光發射點與目標之間的距離結合激光發射器的位置和姿態信息進行聯合解算，得到目標區域的三維點雲(x，y，z)。其中，距離的觀測方式主要採用脈衝式測距。根據觀測目標的幾何結構不同，單條激光束可能有多個回波[10]，準確分離和探測每個回波是距離觀測的基礎。對於多回波和全波形兩種回波記錄方式，其回波探測方法有所不同。多回波方式在數據獲取過程中利用簡單的回波探測方法(constant fraction discriminator，CFD)實時檢測回波[11]，但存在一些問題：算法對用戶保密；算法過於簡單，對複雜波形處理效果較差，容易造成部分回波漏提取。全波形方式則以一定間隔不斷記錄後向散射信號，並提供給用戶進行後續波形處理，以得到回波的位置及形狀參數，常用的處理方法有：高斯分解[12]、去卷積方法[13]等。

2 三維點雲數據處理的重要進展

與傳統的測量方式相比，三維激光掃描數據採集速度快、採樣頻率高等優勢，導致點雲數據具有高冗餘、誤差分佈非線性、不完整等特點，給海量三維點雲的智能化處理帶來了極大的困難：① 多視角、多平臺、多源的點雲數據難以有效整合，限制了數據間的優勢互補，導致複雜場景描述不完整；② 複雜對象模型結構和語義特徵表達困難，模型可用性嚴重受限，極大地限制了複雜場景的準確感知與認知。近年來，國內外學者在點雲處理理論以及數據質量改善、自動化融合、點雲分類和目標提取、按需多層次表達等方法方面進行了深入研究，取得的主要進展如下。

2.1 廣義點雲模型理論方法

針對多源多平臺點雲數據的融合難、目標提取難和三維自適應表達難的嚴重缺陷，文獻[14]提出了廣義點雲的科學概念與理論研究框架體系。廣義點雲是指彙集激光掃描、攝影測量、眾源採集等多源多平臺空間數據，通過清洗、配準與集成，實現從多角度、視相關到全方位、視無關，建立以點云為基礎，基準統一，且數據、結構、功能為一體的複合模型(如圖 1)。

2.2 三維點雲數據質量改善

點雲數據質量改善包括幾何改正和強度校正。一方面，由於測距系統、環境及定位定姿等因素的影響，點雲的幾何位置存在誤差，且其分佈存在不確定性。利用標定場、已知控制點進行點雲幾何位置改正，能夠提高掃描點雲的位置精度和可用性。另一方面，激光點雲的反射強度一定程度上反映了地物的物理特性，對於地物的精細分類起到關鍵支撐作用，然而點雲的反射強度不僅與地物表面的物理特性有關，還受到掃描距離、入射角度等因素的影響。因此，需要建立點雲強度校正模型進行校正，以修正激光入射角度、地物距離激光掃描儀的距離等因素對點雲反射強度的影響[10, 15]。

2.3 多源、多平臺三維點雲融合

由於單一視角、單一平臺的觀測範圍有限且空間基準不一致，為了獲取目標區域全方位的空間信息，不僅需要進行站間/條帶間的點雲融合，還需要進行多平臺(如機載、車載、地面站等)的點雲融合，以彌補單一視角、單一平臺帶來的數據缺失，實現大範圍場景完整、精細的數字現實描述[16-18]。此外，由於激光點雲及其強度信息對目標的刻畫能力有限，需要將激光點雲和影像數據進行融合，使得點雲不僅有高精度的三維座標信息，也具有了更加豐富的光譜信息[19-20]。

不同數據(如不同站點/條帶的激光點雲、不同平臺激光點雲、激光點雲與影像)之間的融合，需要同名特徵進行關聯。針對傳統人工配準法效率低、成本高的缺陷，國內外學者研究基於幾何或紋理特徵相關性的統計分析方法[21-22]，但是由於不同平臺、不同傳感器數據之間的成像機理、維數、尺度、精度、視角等各有不同，其普適性和穩健性還存在問題，還需要突破以下瓶頸：魯棒、區分性強的同名特徵提取，全局優化配準模型的建立及抗差求解。

2.4 三維點雲的精細分類與目標提取

三維點雲的精細分類是從雜亂無序的點雲中識別與提取人工與自然地物要素的過程[4, 23-26]，是數字地面模型生成、複雜場景三維重建等後續應用的基礎。然而，不同平臺激光點雲分類關注的主題有所不同。機載激光點雲分類主要關注大範圍地面、建築物頂面、植被、道路等目標[3-4, 23, 27-29]，車載激光點雲分類關注道路及兩側道路設施、植被、建築物立面等目標[25, 30-32]，而地面站激光點雲分類則側重特定目標區域的精細化解譯[33]。其中，點雲場景存在目標多樣、形態結構複雜、目標遮擋和重疊以及空間密度差別迥異等現象，是三維點雲自動精細分類的共同難題。據此，國內外許多學者進行了深入研究並取得了一定的進展，在特徵計算基礎上，利用逐點分類方法[34-35]或分割聚類分類方法[25, 30-31]對點雲標識，並對目標進行提取。但是由於特徵描述能力不足，分類和目標提取質量無法滿足應用需求，極大地限制了三維點雲的使用價值。目前，模擬人腦的深度學習方法突破了傳統分類方法中過度依賴人工定義特徵的困難，已在二維場景分類解譯方面表現出極大潛力[36]，但是在三維點雲場景的精細分類方面，還面臨許多難題：海量三維數據集樣本庫的建立，適用於三維結構特徵學習的神經網絡模型的構建及其在大場景三維數據解譯中的應用。綜上，顧及目標及其結構的語義理解，三維目標多尺度全局與局部特徵的學習，先驗知識或第三方輔助數據引導下的多目標分類與提取方法，是未來的重要研究方向。

2.5 三維場景的按需多層次表達

在大範圍點雲場景分類和目標提取後，目標點雲依然離散無序且高度冗餘，不能顯式地表達目標結構以及結構之間的空間拓撲關係，難以有效滿足三維場景的應用需求。因此，需要通過場景三維表達，將離散無序的點雲轉換成具有拓撲關係的幾何基元組合模型，常用的有數據驅動和模型驅動兩類方法[5-6]，其中存在的主要問題和挑戰包括：三維模型的自動修復，以克服局部數據缺失對模型不完整的影響[37]；形狀、結構複雜地物目標的自動化穩健重構；從可視化為主的三維重建發展到可計算分析為核心的三維重建，以提高結果的可用性和好用性。此外，不同的應用主題對場景內不同類型目標的細節層次要求不同[38]，場景三維表達需要加強各類三維目標自適應的多尺度三維重建方法[23, 39-40]，建立語義與結構正確映射的場景—目標—要素多級表達模型。

2.6 典型行業和重大工程應用

三維激光掃描已在許多重大工程和典型領域裡得到了廣泛的應用。從深空到地球表面，從全球範圍製圖到小區域監測，從基礎科學研究到大眾服務，三維激光掃描都展現出了與眾不同的優勢。

2.6.1 深空探測

深空探測的目的是促進人類對月球以及更遠天體的科學認識，而距離測量是其中的關鍵技術，許多國家採用激光測高儀進行星體地形表面的測量，如1971年美國阿波羅-15首次進行的月球形狀測量等[41]。

2.6.2 地球科學應用研究

數字地面模型是各種地學過程研究的基礎，利用三維激光掃描系統觀測地表形態及其變化，已被廣泛用於各類地學應用，例如：全球冰川物質平衡[42-43]，地質災害區域時空變化監測[44-45]，海岸線提取和海岸侵蝕監測[46-47]，海底測繪及水下目標探測[48]等。

2.6.3 森林資源調查

及時準確瞭解林區的植被動態變化是林業科學研究的基礎。相比光學遙感，激光雷達能夠獲取植被冠層的三維結構：地面激光雷達用於精細地獲取單株樹木的垂直結構[7]，機載激光雷達用於大範圍森林的蓄積量和生物量等生態參數的反演[49-51]，星載激光雷達還可以進行全球植被覆蓋及其生態參數製圖[52-53]。

2.6.4 城市形態分析

城市形態分析對城市規劃設計與管理具有重要意義，傳統手段難以監測城市形態的垂直結構及其演化，激光雷達可以快速獲取城市三維形態，為更精細的城市形態分析提供基礎，滿足基於城市形態的各種應用需求，如基礎設施管理、太陽能潛力估計等[54-55]。

2.6.5 無人駕駛高精度地圖

高精度地圖是實現自動化駕駛的關鍵因素[25]，車載三維激光掃描系統可以高效、快速的獲得道路以及周邊高精度、高密度的三維幾何信息和紋理信息，為高精度地圖的自動化生產提供高質量的數據支撐[56]。

2.6.6 電力走廊安全巡檢

我國電力資源分佈和經濟建設中心不一致，高壓輸電線路區域地理環境複雜，傳統的人工巡檢手段難以適應。機載三維激光掃描可以直接獲取電力線及其附屬設備的幾何形態參數[57]，為電力巡檢提供了新的手段[58]。

2.6.7 海島礁測繪

精確的海島礁基礎地理空間信息是海洋管理、經濟開發、海防安全的重要依據，傳統測繪手段作業週期長、成本高，三維激光掃描可以直接觀測目標的三維空間信息，直接生產數字測繪產品，是島礁高精度測圖的重要手段。

2.6.8 文化遺產數字化保護

文化遺產數字化保護是人類共同的歷史責任。三維激光掃描可以快速繪製物質文化遺產的結構圖和精細的三維模型，大幅提高文化遺產保護的工作效率，並豐富文化遺產成果表現形式[59-60]。

3 點雲處理面臨的挑戰

近年來，星、空、地掃描以及便攜式泛在傳感器(如：RGB-D深度相機)廣泛運用，不但提高了點雲獲取的時效性、顆粒度和覆蓋面，而且帶來了點雲的多時相、流形(streaming)和多樣屬性的新特性，從而產生了多維點雲數據。多維點雲本質上是對物理世界中地理對象/現象的三維幾何、物理乃至生化特性的多維密集採樣，其不但記錄了地物的三維空間結構特徵，同時也記錄了地物目標的物理特性(如波形、反射強度等)。深入挖掘多維點雲的內在特徵對提升多維點雲處理的智能化程度，揭示覆雜動態三維場景的變化規律至關重要。儘管點雲處理方面已經取得了較好的研究成果，但是多維點雲的智能化處理方面仍然面臨如下的巨大挑戰：

3.1 多維點雲幾何與屬性協同的尺度轉換

探索不同平臺獲取點雲的誤差分佈規律，建立比例尺依賴的特徵點質量評估模型；研究融合點雲物理特性的特徵點簇聚合與分層方法；建立基於特徵分層的多維點雲多尺度整合方法，實現多維點雲的時空基準自動統一。

3.2 多維點雲變化發現與分類

建立統一時空參考框架下多維點雲的變化發現與提取方法，研究基於時間窗口的多維點雲與地物三維模型的關聯方法，提取地物空間要素的幾何和屬性變化，研究面向地物空間結構變化的可視化分析方法，為揭示空間要素的變化規律提供科學工具。

3.3 複雜三維動態場景的精準理解

基於機器學習、人工智能等先進理論方法探索多維點雲結構化建模與分析的理論與方法，研究建立複雜三維動態場景中多態目標的準確定位、分類以及語義化模型的建立，建立面向多維點雲的三維動態場景中各類要素的特徵描述、分類與建模方法，架設多維點雲與地理計算模型的橋樑。

上述關鍵挑戰問題的突破將形成完備的廣義點雲全三維(覆蓋全、要素全、關係全)建模的理論與方法體系，從而實現點雲處理由“靜態、可視、量算”到“動態、模擬、分析”的跨越。

4 三維激光掃描與點雲處理發展趨勢與展望

近年來，傳感器、通信和定位定姿技術的發展，人工智能、深度學習、虛擬/增強現實等領域先進技術的重要進展有力推動了數字現實(digital reality)時代的來臨。激光掃描與點雲智能化處理將順應數字現實時代的需求朝以下幾個方面發展。

(1) 三維激光掃描裝備將由現在的單波形、多波形走向單光子乃至量子雷達，在數據的採集方面由現在已幾何數據為主走向幾何、物理，乃至生化特性的集成化採集。

(2) 三維激光掃描的搭載平臺也將以單一平臺為主轉變為以多源化、眾包式為主的空地柔性平臺，從而對目標進行全方位數據獲取，當前國家重點研發計劃重點專項項目：國產空地全息三維遙感系統及產業化(編號：2016YFF0103500) 已支持相關研究。

(3) 點雲的特徵描述、語義理解、關係表達、目標語義模型、多維可視化等關鍵問題將在人工智能、深度學習等先進技術的驅動下朝著自動化、智能化的方向快速發展，點雲將成為測繪地理信息中繼傳統矢量模型、柵格模型之後的一類新型模型，將有力提升地物目標認知與提取自動化程度和知識化服務的能力。

(4) 虛擬/增強現實、互/物聯網+的發展將促使三維激光掃描產品由專業化應用擴展到大眾化、消費級應用，滿足網絡化多維動態地理信息服務的需求。
