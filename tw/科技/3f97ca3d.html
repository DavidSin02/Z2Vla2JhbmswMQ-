四核APU加持2GB獨顯！ThinkPad E475評測

2020-10-29
| 科技

作為 ThinkPad 品牌銷量擔當的 E 系列一直有著出色的性價比，今天的主角 ThinkPad E475 就是一臺採用 AMD 四核處理器與 AMD Radeon 獨立顯卡組合的 14 英寸筆記本，標準的小黑外觀設計配合具有強大性能的硬件組合給 E475 帶來了更大的想象空間，伴隨著 ThinkPad 25 週年的紀念，我們一起來看看這臺配備了 APU 的 ThinkPad 有什麼與眾不同。

純黑色商務風外觀設計

在外觀上，ThinkPad E475 繼承了 ThinkPad 一貫沉穩低調的設計風格，通體的黑色設計充滿了商務範，頂蓋角落裡銀色的 ThinkPad 標識凸顯了它的高貴血統，可發光的紅色小紅點讓人輕鬆地瞭解它的實時運行狀態。Lenovo 的標識則鑲刻在機身左下角。

ThinkPad E475 採用了一塊 14.0 英寸 1920×1080 分辨率屏幕，屏幕上方配有一枚 720P 攝像頭和一組雙陣列麥克風，方便在視頻會議或是日常視頻聊天中使用。

ThinkPad E475 延續了 ThinkPad 一貫舒適的小紅點鍵盤，按鍵的間距較長，反饋速度很快，日常打字不會有任何壓力，配合小紅點在一定程度上完全可以替代鼠標。

ThinkPad E475 的機身右側配有一個 USB 2.0 接口，一個 USB 3.0 接口，一個耳機/麥克風二合一接口，一個 SD 讀卡器。

它的機身左側配有一個電源接口，一個 RJ45 網絡接口，一個 HDMI 接口，一個 USB 3.0 接口。

在輕薄本陣營中，ThinkPad E475 的接口配置絕對算得上是豐富，尤其是 RJ45 網絡接口和 SD 讀卡器的配置能夠秒殺眾多輕薄本。

APU+Radeon 獨顯帶來更多想象

ThinkPad E475 有多種配置可選，我們拿到的機器搭載了 AMD A10-9600P 四核處理器，16GB DDR4 2133MHz 內存，256GB SSD，顯卡為具有 2GB 顯存的 AMD Radeon R5 M430 獨立顯卡。下面我們將進行一系列的測試。

CPU-Z 性能測試

AMD A10-9600P處理器採用四核心架構設計，主頻 2.4GHz-3.3GHz，在 CPU-Z 的性能測試中，它的單線程分數為 157，多線程分數為 481。

顯卡規格介紹

ThinkPad E475 也採用了雙顯卡的組合，A10-9600P 中集成了 Radeon R5 GPU，GPU 的最高頻率為 720MHz。此外，ThinkPad E475 還配備了同樣基於 AMD GCN 架構設計的R5 M430 獨立顯卡，與APU內置R5核顯組成雙顯卡交火，交火後的顯卡型號為Radeon R6 M435DX，擁有 320 個計算單元，核心頻率 1030MHz，顯存頻率1000MHz，顯存容量 2GB DDR3。

GPU 性能基準測試

性能方面使用 3DMark Cloud Gate 進行測試，其最終圖形成績為 8991，可以充分滿足日常遊戲需要。

魯大師整機性能測試

在魯大師的整機性能測試中，ThinkPad E475 的成績為 79737，其中處理器性能為 23236 分，顯卡性能為 15864 分，內存性能為 6744 分，磁盤性能為 10650 分，表現非常不錯。

PCMark 8 家庭測試

在注重多媒體的家庭娛樂測試中，ThinkPad E475 得到了 2451 分。

PCMark 8 工作測試

在更偏向日常工作應用的 Work 測試中，ThinkPad E475 得到了 3371 分。兩次測試充分體現了 ThinkPad E475 足以應對日常的應用操作。

遊戲測試

本次遊戲測試我們選擇了主流的《英雄聯盟》，將畫質調至中等，關閉垂直同步功能，使用 Fraps 記錄運行幀數。

在整個遊戲運行過程中，ThinkPad E475 維持在 40 幀左右，最終的平均幀數為 47，即便是大規模團戰場景也非常穩定，可以說是輕鬆應對。

除了性能，AMD 還有這兩件黑武器

低負載的 4K 視頻播放

APU 的一大優勢在於它在解碼 4K 視頻時的超高效率，A10-9600P 支持 HEVC 硬件加速、AMD 完美圖像和 AMD 視頻穩定技術，在播放 4K 超清視頻時 CPU 的使用率非常低，在保證流暢播放的同時還可以降低功耗，保證整機的續航時間。

我們使用迅雷影音來進行測試，開啟內置的硬件加速功能播放著名的《美女吃烤鴨》4K 視頻片段，迅雷影音的 CPU 佔用率僅為 5.1%。

在播放《臺北101》4K 視頻時，迅雷影音的 CPU 佔用率僅為 5.8%！在高清視頻硬解碼的測試中，得益於 AMD A10-9600P 出色的性能表現，ThinkPad E475 可以說是毫無壓力。

VSR 帶來更大視野

近年來隨著 4K 技術的不斷崛起，越來越多的消費者在購買電腦時也會考慮 2K 甚至 4K 屏幕的配置，但是超清屏幕筆記本的價格一直居高不下，這給預算有限的消費者帶來了很多困擾。ThinkPad E475 雖然只配備了 FHD 分辨率的屏幕，但是用戶可以通過內置的 VSR 功能一鍵將屏幕分辨率“升級為 2K”。

操作方法很簡單，在 AMD Radeon 設置中點擊“顯示器”選項卡，接著將“超虛擬分辨率”功能打開，屏幕閃爍幾次後即可在 Windows 的設置中找到更高的分辨率選項。

其原理也不難理解，此舉並沒有改變屏幕的物理分辨率，而是讓系統將 4K 的圖像以 4K 的標準輸出到 FHD 屏幕當中，由於圖像本身的清晰度變高了，所以在人眼視覺中圖像就會變得更加清晰了。

4K 屏幕的好處有很多，比如顯示的圖像更加細膩，同一窗口下可以顯示更多的內容、平鋪更多的應用程序，同樣的 Excel 表格可以一次性顯示更多的行列內容等等，這都是 FHD 分辨率很難達到的效果。AMD 通過這一技術讓普通分辨率的屏幕也能有高分屏的待遇，著實是“小錢辦大事”的典範。

綜合體驗出色，同價位中值得考慮

作為面向主流價位的產品，ThinkPad E475 保持了 ThinkPad 品牌一貫沉穩的設計基調，純黑色的外觀是它的外表，但絕對不是它的全部，出色的鍵盤手感、豐富的接口配置共同奠定了它優質的使用感受。

AMD 四核 APU 與 Radeon 獨顯的加持更為 ThinkPad E475 帶來了全新的性能感受，無論是日常遊戲娛樂還是辦公作業都遊刃有餘，AMD 出色的 4K 硬解碼功能讓流暢且節能的超清視頻播放成為可能，讓你能夠更輕鬆、更簡單地播放 4K 大片；VSR 更是以獨特的思路解決了屏幕物理分辨率有限的問題，在 FHD 屏幕上就能夠體驗到媲美 4K 屏幕的使用體驗，大幅度提高辦公效率。總而言之，ThinkPad E475 是同價位中值得考慮的一款產品。■
