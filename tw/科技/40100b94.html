從基礎學起，C輸入輸出。

| 科技

一，

理解帶格式的輸入/輸出函數

- scanf() 和 printf()

使用字符輸入/輸出函數

- getchar() 和 putchar()

C 語言中的標準庫提供了輸入和輸出函數

標準庫提供： 輸入/輸出函數來處理輸入、輸出、字符 和字符串操作

標準輸入通常是指鍵盤

標準輸出通常是指顯示器（也稱控制檯）

可以用文件代替標準輸入和輸出設備

二，

頭文件 <stdio.h>

#include <stdio.h>

這是一個預處理命令

stdio.h 是一個文件，稱為頭文件

包括對應於 C 語言程序中使用的輸入/輸出函數的宏

所設計的 printf()、scanf()、putchar()、getchar() 等函數需要使用頭文件中的這些宏才能正確執行

帶格式的輸入/輸出

printf() – 用於帶格式的輸出

scanf() – 用於帶格式的輸入

能以用戶控制下的各種預指定格式讀入和輸出數據

格式說明符指定了變量值輸入和輸出的格式

printf ()-2

控制字符串由三類項目中的一項或多項組成：

文本字符：由可打印字符組成

格式命令： 以 % 符號開始，隨後緊跟與數據項相應的格式代碼

非打印字符：包括製表符、空格和新行

printf() 示例

#include <stdio.h>

void main()

{

int a = 10;

float b = 24.67892345;

char ch = 'A';

printf("Integer data = %d\n", a);

printf("Float Data = %f\n",b);

printf("Character = %c\n",ch);

printf("This prints the string\n");

printf("%s","This also prints a string\n");

}

printf() 中的修飾符-1

“-” 修飾符

數據項在範圍內向左對齊，從範圍最左側的位置開始輸出數據項

字符寬度修飾符

可以用於浮點型、雙精度型和字符數組型（字符串）。

範圍寬度修飾符是一個整數，定義了數據項的 最大範圍寬度。

printf() 中的修飾符-2

精度修飾符

可以用在浮點型、雙精度型或字符數組型（字符串）中。如果用在浮點型或雙精度型數據中，數字字符串表示小數右側最多可輸出的數字位數

“0” 修飾符

字段的默認填充符是空格。如果用戶希望用零作為填充符，需要使用此修飾符

“l” 修飾符

可以藉助於此修飾符將整數顯示為長整型或雙精度型參數。相應的格式代碼為 %ld

printf() 中的修飾符-3

“h” 修飾符

此修飾符用於顯示短整數。

相應的格式代碼為 %hd

“*” 修飾符

如果用戶不希望預先指定字段寬度，

而希望程序來指定，則需要用到此修飾符

*是表示輸出的寬度需再讀入一個變數

printf("%*d",5,5) 等於 printf("%5d",5)

printf("%*d",9,3) 等於 printf("%9d",3)

修飾符示例

#include <stdio.h>

void main()

{

printf("The number 555 in various forms:\n");

printf("Without any modifier: \n");

printf("[%d]\n",555);

printf("With – modifier :\n");

printf("[%-d]\n”,555);

printf("With digit string 10 as modifier :\n");

printf("[%10d]\n”,555);

printf("With 0 as modifier : \n");

printf("[%0d]\n”,555);

printf("With 0 and digit string 10 as modifiers :\n");

printf("[%010d]\n”,555);

printf("With -, 0 and digit string 10 as modifiers: \n");

printf("[%-010d]\n",555);

}

scanf()

printf()語句中的使用的格式極其語法同

樣適用與scanf()語句

printf() 和 scanf() 參數列表之間的區別

printf() 使用變量名、常量、符號常量和表達式

scanf()使用變量指針

使用 scanf()時，對於參數列表應遵循以下規則：

如果希望讀取基本數據類型變量的值，應在變量名的前面加上符號&

當讀取指針變量指向的值時，在變量名前不能使用 &

printf() 和 scanf() 格式命令之間的區別

沒有 %g 選項

格式代碼 %f 和 %e 在效果上是一樣的

scanf() 示例

#include <stdio.h>

void main()

{

int a;

float d;

char ch, name[40];

printf("Please enter the data\n ");

scanf("%d %f %c %s", &a, &d, &ch, name);

printf("\nThe values accepted are :

%d, %f, %c, %s", a, d, ch, name);

}

scanf()使用非打印字符來判斷輸入數據是什麼時候開始和結束的

可以忽略空白區域和行邊界來獲取數據

控制檯輸入/輸出

控制檯輸入/輸出函數將其操作指向系統的標準輸入和輸出

C 中最簡單的控制檯輸入/輸出是：

getchar() – 從鍵盤讀取一個（且僅一個）字符

putchar() – 在屏幕上輸出一個字符

getchar()

用於從鍵盤讀取輸入數據，方式為一次讀一個字符

緩衝字符，直到用戶按回車鍵

getchar() 函數沒有參數，但仍然必須使用括號

getchar() 示例

#include <stdio.h>

void main()

{

char letter;

printf("\nPlease enter any character: ");

letter = getchar();

printf("\nThe character entered by you is %c", letter);

}

putchar()

#include <stdio.h>

void main()

{

putchar('H'); putchar('\n');

putchar('\t');

putchar('E'); putchar('\n');

putchar('\t'); putchar('\t');

putchar('L'); putchar('\n');

putchar('\t'); putchar('\t'); putchar('\t');

putchar('L'); putchar('\n');

putchar('\t'); putchar('\t'); putchar('\t');

putchar('\t');

putchar('O');

}

getch和putch

getch和putch不屬於C語言標準，但在windows和其他操作系統的補充庫提供。

Windows包含於<conio.h>,linux包含與於<curses.h>中。

與getchar和putchar的區別是不需要回車，也不回顯。
