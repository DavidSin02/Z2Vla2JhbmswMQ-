MATLAB圖像配準各方法介紹和對比

圖像配準是對同一場景的兩個或多個圖像進行對齊的過程。此過程包括將一個圖像指定為參考圖像(也稱為固定圖像)，並將幾何變換或局部位移應用於其他圖像，使其與參照對齊。由於各種原因，圖像可能會未對齊。通常，圖像是在可變條件下拍攝的，這將影響攝像頭的視角或者場景的內容。鏡頭和傳感器失真或捕獲設備之間的差異也會導致未對齊。

在其他圖像處理應用中，圖像配準通常被用作初步步驟。例如，可以使用圖像配准將衛星圖像或使用不同診斷模式(如MRI和SPECT)捕獲的醫療圖像配準。通過圖像配準，可以比較不同圖像中的常見特徵。藉此，可能會發現河流是如何遷移的，一個區域是如何被淹沒的，或者在MRI或SPECT圖像中是否可見腫瘤。

圖像處理工具箱™提供三種圖像配準方法：交互式配準估計應用程序、基於強度的自動圖像配准以及使用控制點的圖像配準。計算機視覺工具箱™提供自動特徵檢測和匹配。

交互式配準估計應用程序 Registration Estimator App

交互式配準估計應用程序使您能夠以交互方式配準二維圖像。您可以比較不同的配準方法、調整設置和可視化已配準的圖像。該應用不僅提供了檢查配準質量的定量度量，而且它會反饋配準後圖像和轉換矩陣。該應用程序還能夠使用所選的配準方法和設置自動生成代碼,因此您之後可對多個圖像使用相同的轉換。

交互式配準估計應用程序提供了六種基於特徵的方法、三種基於強度的方法和一種非剛性配準方法。有關可用技術的更詳細比較，請參閱交互式配準估計應用程序支持的方法 Techniques Supported by Registration Estimator App

基於強度的自動圖像配準 Intensity-Based Automatic Image Registration

基於強度的自動圖像配準根據相對強度模式映射每個圖像中的像素。您可以同時配準單模和多模圖像對，也可以配準二維和三維圖像。此方法可用於:

  •  配準大量圖像
  •  自動配準

若要使用基於強度的技術配準圖像，請使用 imregister 並指定要應用於源圖像的幾何轉換類型。imregister 迭代地調整變換，優化兩個圖像的相似性。

或者，您可以估計局部位移場，並使用 imregdemons 對源圖像應用非剛性變換。

使用控制點的圖像配準 Control Point Registration

使用控制點的圖像配準 Control Point Registration ，您可以手動選擇每個圖像中的通用特徵。使用控制點的圖像配準在以下情況下很有用:

  •  您希望確定特定特徵的對齊順序，而不是使用自動特徵檢測匹配檢測到的整個特徵集。例如，在配準兩個醫學圖像時，您可以集中校準所需的解剖特徵，而忽略信息較少的解剖結構相對應的匹配特徵。
  •  圖像具有重複的模式，可使用自動特徵匹配提供模糊的映射。例如，有許多窗戶的建築物的照片，或網格化城市街道的航拍照片，都有許多類似的特徵，很難自動映射。在這種情況下，手動選擇控制點對可以提供更清晰的特徵映射，從而更好地轉換以對齊特徵點。

控制點配準可以對源圖像應用多種類型的變換。全局變換對整個圖像均勻作用，包括仿射、投影和多項式幾何變換。非剛性變換作用於圖像的局部區域，包括分段線性變換和局部加權均值變換。

使用控制點選擇工具選擇控制點。啟動該工具使用 cpselect 命令。

自動特徵檢測和匹配 Automated Feature Detection and Matching

自動特徵檢測和提取 Feature Detection and Extraction (基於計算機視覺工具箱) 可檢測角和 blobs 等特徵，匹配源圖像和目標圖像中的相應特徵，並估計幾何變換以匹配特徵。

有關示例，請參閱使用自動特徵匹配尋找合適圖像旋轉和縮放 Find Image Rotation and Scale Using Automated Feature Matching 。您必須藉助計算機視覺工具箱才能使用此方法。

注意：交互式配準估計應用程序提供了六種基於特徵的方法來配準一對圖像。但是，該應用不提供用於配準多個圖像的自動工作流。

基於特徵的圖像配準 Feature-Based Registration

基於特徵的配準技術可自動檢測不同的圖像特徵，如尖角、斑點或均勻強度的區域。源圖像經過單一的全局轉換，使得特徵同目標圖像的相應特徵最佳對齊。

  1. FAST 可檢測角特徵，尤其是在人源場景中（如街道和室內客房）。FAST 支持單尺度圖像和點追蹤。
  2. MinEigen 也檢測角特徵。MinEigen 支持單尺度圖像和點追蹤。
  3. Harris 也檢測角特徵，使用比 MinEigen 更有效的算法。Harris 支持單尺度圖像和點追蹤。
  4. BRISK 也檢測角特徵。與前面的算法不同, BRISK 支持放縮和旋轉以及點追蹤方面的更改。
  5. SURF檢測圖像斑點。並支持放縮和旋轉方面的更改。
  6. MSER檢測均勻強度的區域。 MSER 支持放縮和旋轉的變化，與其他基於特徵的算法相比，具有更魯棒性的仿射變換。

注意：計算機視覺工具箱™中存在基於特徵的自動化工作流。此工作流包括特徵檢測、提取和匹配，然後進行轉變換估計。有關詳細信息, 請參閱使用自動特徵匹配尋找圖像旋轉和縮放.

基於強度的圖像配準 Intensity-Based Registration

基於強度的圖像配準方法評估圖像在空間域或頻率域中的強度相關性。源圖像經歷了一個單一的全局變換，以最大限度地提高其強度與目標圖像強度的相關性。

  1. 單模態配準被應用在從同一類型的掃描儀或傳感器上捕獲的亮度和對比度相似的圖像。例如，使用單模強度來配準使用 MRI 掃描類似對象得到的圖像序列。
  2. 多模態強度配準不同亮度和對比度的圖像。這些圖像可以來自兩種不同類型的設備，如兩個型號的相機或兩種類型的醫學成像系統(如 CT 和 MRI)。這些圖像也可以來自一個設備。例如，使用多模態強度配準同一相機使用不同曝光設置後拍攝的圖像，或配準在單個階段 MRI 採集不同成像序列的圖像。
  3. 基於相位相關性的配準，根據圖像在頻率域的特性配準圖像。與多模態強度一樣,基於相位相關性的配準與圖像亮度也是無關的。相位相關性與其他基於強度的配方法相比，對噪聲更具魯棒性。

注意：當每個圖像為正方形時,基於相位相關性的配準可提供更好的結果。

Nonrigid Registration非剛性配準

非剛性配准將非全局變換應用於源圖像。非剛性變換生成一個向量場，在該場中，目標圖像中的每個像素都將映射到源圖像中的相應位置。然後根據位移場對源圖像的變形，並利用線性插值對其進行重新採樣。有關估計非剛性變換的向量場的詳細信息，請參見 imregdemons 。
