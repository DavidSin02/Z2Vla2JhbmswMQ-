基於LNMP架構部署NextCloud私有云盤

2020-11-14
| 科技

一、NextCloud介紹

1、簡介

Nextcloud是一款開源免費的私有云存儲網盤項目，可以讓你快速便捷地搭建一套屬於自己或團隊的雲同步網盤，從而實現跨平臺跨設備文件同步、共享、版本控制、團隊協作等功能。它的客戶端覆蓋了Windows、Mac、Android、iOS、Linux 等各種平臺，也提供了網頁端以及 WebDAV接口，所以你幾乎可以在各種設備上方便地訪問你的雲盤。

Nextcloud也提供了許多應用安裝，包括但不限於Markdown在線編輯、OnlyOffice（需另外部署服務端）、思維導圖、日曆等，你可以自行選擇以豐富個人網盤的功能。

Nextcloud的服務端部署方式有很多種，本文采用基於LNMP環境源碼安裝。

2、Nextcloud部署

#本文采用的LNMP環境可參考我另一篇文章部署（LNMP環境部署）

1)下載Nextcloud安裝包

[root@localhost ~]# wget https://download.nextcloud.com/server/releases/nextcloud-18.0.1.tar.bz2

2）解壓Nextcloud安裝包

[root@localhost ~]# tar xf nextcloud-18.0.1.tar.bz2

3）拷貝Nextcloud到發佈目錄

[root@localhost ~]# mv nextcloud /usr/local/nginx/html

6）重啟Nginx服務

[root@localhost ~]# /usr/local/nginx/sbin/nginx -t

[root@localhost ~]# /usr/local/nginx/sbin/nginx -s reload

7）創建Nextcloud數據庫

[root@localhost ~]# mysql

mysql> create database nextcloud charset=utf8;

mysql> show grant on nextcloud.* to nextcloud@’localhost’ identified by‘123456’;

mysql> flush privileges;

8）創建NextCloud 數據存儲目錄、併為其設置權限

[root@localhost ~]# mkdir -p /usr/local/nginx/html/nextcloud/data

[root@localhost ~]# chown -R www.www /usr/local/nginx/html/nextcloud
