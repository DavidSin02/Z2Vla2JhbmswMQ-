遍歷

通過for ... in ...:的語法結構，我們可以遍歷字符串、列表、元組、字典等數據結構。

1.1 遍歷字典的所有key

>>> info = {'name': 'xiaoming', 'age': 20, 'hometown': '廣州'}>>> for i in info.keys():... print(i)... nameagehometown>>>

1.2 遍歷字典的左右value

>>> info = {'name': 'xiaoming', 'age': 20, 'hometown': '廣州'}>>> for i in info.values():... print(i)... xiaoming20廣州>>>

1.3 遍歷字典元素

>>> info = {'name': 'xiaoming', 'age': 20, 'hometown': '廣州'}>>> for i in info.items():... print(i)... ('name', 'xiaoming')('age', 20)('hometown', '廣州')

1.4 遍歷字典的key-value

>>> for key,value in info.items():... print(key,value)... name xiaomingage 20hometown 廣州>>>

共有操作

2.1. 合併操作 +，

適用於字符串，列表，元祖

>>> str1 = 'hello'>>> str2 = 'world' >>> str1 + str2 # 字符串'helloworld'​>>> [1,2,3] + [4,5,6] # 列表[1, 2, 3, 4, 5, 6]​>>> ('a','b')+('c','d') # 元祖('a', 'b', 'c', 'd')

2.2. * 複製

表示倍數，適用於字符串、列表、元組

>>> str1 = 'hello'>>> str1*3'hellohellohello' # 字符串>>> [1,2,3]*3[1, 2, 3, 1, 2, 3, 1, 2, 3] # 列表>>> (1,2,3)*3(1, 2, 3, 1, 2, 3, 1, 2, 3) # 元祖

2.3. in 判斷元素是否存在

適用於字符串、列表、元組字典

>>> 'my' in 'my name is lilei'True>>> 1 in [1,2,3]True>>> 'age' in {'name':'xiaoming','age':18}True
