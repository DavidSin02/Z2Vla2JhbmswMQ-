中低端交換機典型配置實例：應用Xmodem方式升級app軟件

2020-10-29
| 科技

一 組網需求：

利用PC機本地的版本文件，在BOOT菜單中採用Xmodem方式對交換機的app主機軟件進行升級。

二 組網圖：


三 配置步驟：

全系列交換機應用xmodem方式升級主機軟件

1．按照安裝手冊中交換機初次上電啟動部分，啟動並配置超級終端。啟動交換機，根據超級終端顯示的提示按Ctrl+B，進入BOOTROM主菜單

BOOT MENU

1. Download application file to flash 下載應用程序到Flash中

2. Select application file to boot 選擇啟動文件

3. Display all files in flash 顯示Flash中所有文件

4. Delete file from flash 刪除Flash中的文件

5. Modify bootrom password 修改BootROM密碼

6. Enter bootrom upgrade menu 進入BootROM升級菜單

7. Skip current configuration file 設置重啟不運行當前配置文件

8. Set bootrom password recovery 恢復BootROM密碼

9. Set switch startup mode 設置交換機啟動模式

0. Reboot 重新啟動交換機

Enter your choice(0-9):

2．鍵入1然後回車，進入加載app軟件的菜單

1. Set TFTP protocol parameter  設置TFTP協議參數

2. Set FTP protocol parameter  設置FTP協議參數

3. Set XMODEM protocol parameter 設置XMODEM協議參數

0. Return to boot menu 返回BOOT菜單

Enter your choice(0-3):

3．鍵入3，選擇採用XModem協議完成BOOTROM的加載，回車後，系統進入下載速率設置菜單：

Please select your download baudrate:

1.* 9600

2. 19200

3. 38400

4. 57600

5. 115200

0. Return ¬返回下載程序菜單

Enter your choice (0-5):

4．根據實際情況，選擇合適的下載速率，若如上所示鍵入5，即選擇115200bit/s的下載速率，回車後終端顯示如下信息：

Download baudrate is 115200 bps

Please change the terminal's baudrate to 115200 bps and select XMODEM protocol

Press enter key when ready ¬波特率更改完成後，請鍵入回車

如果下載波特率選擇為9600bit/s，用戶不用修改超級終端的波

特率，也不用進行下面的第五步操作，直接進入第六步的操作。此時系統不給出上面的提示。

5．單擊超級終端的<斷開>按鈕，如圖1-1所示，斷開超級終端連接。進入超級終端軟件的[文件/屬性]菜單項，如圖1-2所示，在彈出的對話框單擊<配置>按鈕，進入Console口配置對話框，如圖1-3所示，將速率配置115200bit/s。單擊<確認>按鈕完成波特率設定。


圖1-1 連接按鈕和斷開按鈕


圖1-2進入屬性對話框


圖1-3 Console口配置對話框

要做一次終端仿真程序的斷開操作，系統才允許修改終端的波特率。

6．然後鍵入回車即可開始程序的下載，終端顯示如下信息：

Now please start transfer file with XMODEM protocol.

If you want to exit, Press <Ctrl+X>.¬若想退出程序下載，請鍵入<Ctrl+X>

Loading ...CCCCCCCCCC

7．從終端窗口選擇[傳送/發送文件]菜單項，在彈出的對話框，如圖1-4所示，單擊<瀏覽>按扭，選擇需要下載的軟件，並將下載使用的協議改為XModem


圖1-4發送文件對話框


7．選擇完成後，單擊<發送>按鈕，系統彈出如圖1-5所示的界面


圖1-5正在發送文件界面

8．程序下載完成後，出現下面的提示，要求輸入文件名

The file you download is not a app file, please name the file!

Load File name : default_file

這裡需要輸入的文件名不能帶擴展名。

9．這裡應用默認的文件名，直接回車，出現以下提示。

Free flash Space: 5724160 bytes

Writing flash..................................................................!

Your baudrate should be set to 9600 bps again!

Press enter key when ready

如果下載的速率選擇為9600bit/s，用戶不用重新調整超級終端的速率，此步驟沒有必要。此時系統不給出“Your baudrate should be set to 9600 bps again! Press enter key when ready”的提示。

10．此時重新將超級終端的波特率調整為9600bit/s，過程請參考前面的第五步，然後根據提示按回車鍵，系統將進入BOOT菜單，鍵入0，即選擇重啟交換機，回車後系統重啟，至此完成了主機軟件的升級。
