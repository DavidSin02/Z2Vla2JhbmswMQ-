前端筆記-css選擇器常識

2020-10-29
| 科技

1.按ID查找

2.按tag查找

3.按class查找

4.按屬性查找

a.'[name=email]'

b.'[type=password]'

c.[class^="icon-"]

5.組合查找

a.input[name=email]

b.tr.red

c.div[title~="warning"] // 選擇title中包含warning字符串的div

e.div[title^="intro"] // 選擇title中以intro開頭的div
f.div[title$="intro"] // 選擇title中以intro結尾的div

g.div[title*="intro"] // 選擇title中包含intro字符串的div 與c一樣

5.多項選擇器

$('p.red,p.green'); // 把<p class="red">和<p class="green">都選出來

要注意的是，選出來的元素是按照它們在HTML中出現的順序排列的，而且不會有重複元素。例如，<p class="red green">不會被上面的$('p.red,p.green')選擇兩次。

6.派生選擇器

a.子代選擇器：div>a

b.後代選擇器：div a

c.相連選擇器：li+li

<style type="text/css">		li + li {			color: red;		}	</style><ul>		<li>1</li>		<li>1</li>		<li>			<ul>				<li>2</li>				<li>2</li>				<li>2</li>			</ul>		</li>	</ul>

7.偽類選擇器

a.未訪問的鏈接a:link { }

b.已訪問的鏈接a:visited { }

c.鼠標移動到鏈接上a:hover { }

d.選定的鏈接a:active { }

注意：a:hover 必須被置於 a:link 和 a:visited 之後。a:active 必須被置於 a:hover 之後。

1).:focus獲取焦點，當鼠標點擊文本框時，對應狀態改變。

2).::selection為選中的文字設置樣式。

3).:empty用來選擇沒有任何內容的元素

4).:root選擇器等同於<html>元素

5).:target選擇器稱為目標選擇器，用來匹配文檔(頁面)的url的某個標誌符的目標元素。

6).:read-only和:read-write選中設置了只讀和只寫

7).:enabled和:disabled選中（“:enabled”）和不可用（“:disabled”）狀態

8).first-letter:為元素第一個字設置樣式

9).first-line:為元素第一行設置樣式

10).:before

11).:after

12).p:first-child，選中第一個子元素

13).p:last-child，選擇最後一個子元素

14).p:nth-child(數字/even/odd)：選中任意位置/偶數位/奇數位

15).p:not(.hello){ }為p段落中的除了class=“hello”的添加樣式

選擇器的優先級

內聯樣式(1000)>id選擇器(100)>類和偽類(10)>元素選擇器(1)>全局選擇器>繼承的樣式(0)
注意：當選擇器中包含多種選擇器時（p#p2），需要將多種選擇器的優先級相加再比較.選擇器的優先級不會超過他的最大數量級，比如多個id選擇器相加不會超過999.
如果選擇器的優先級一樣，則使用位置靠後的樣式。class="p1 p2"
並集選擇器h1，p，div{}的優先級是單獨計算的，會各自運用到自己的內容中。在樣式的最後添加一個!inportant會獲得最高的優先級，優先顯示。
