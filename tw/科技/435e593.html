橡膠硫化中硫磺的作用

2020-10-29
| 科技

1839年和 1843年固特異 (Cood year) 和漢考克 (Hancook) 先後發現，將天然橡膠與硫黃共熱後， 就會變成堅實有彈性的物質。這一過程即是硫化。硫黃聚為硫化劑。從發現用硫黃硫化橡膠時，由於成本低廉，來源豐富，硫化橡膠質量較好，至今仍廣泛地應用於橡膠工業。它不僅用於天然橡膠， 而且可用於二烯類合成橡膠 ( 丁苯橡膠、順丁橡膠 、異戊橡膠、丁腈橡膠 ) 的硫化。低不飽和度的丁基橡膠和某些硫化速度快的三元丙橡膠有時也可用硫黃硫化。但是。在生產上早已不是單獨採用硫黃來硫化橡膠了。有罪是單純用黃硫化反應很慢， 硫化時間硫黃用量多， 且硫化膠結構複雜， 性能不好。但為便於比較和說明含有有機硫化會促進的硫化。反應體系的性質，先討論單用硫黃硫化的問題。一、硫黃的性質和品種天然硫黃分子是由八個硫原子的環狀結構，空間結構為冠型配位 ( 如圖3-3) 。它是淡黃色或黃色固體物質，不溶於水，易溶於二硫化碳 (CS2 ，室溫下為菱形晶體，熔點為 113℃。159℃時硫黃的八元環可以斷裂，當受到橡膠大分子雙鍵的極化作用時，其開環能量降低， 140℃即可開環。開環所生成的活潑雙基硫黃( ·S8·) 能夠與橡膠發生硫化反應。

橡膠工業常用的硫黃有如下品種

1 、粉末硫黃由塊狀硫黃經粉碎篩選而得。其細度在 200 目以上，特殊情況也有 600目以上者。

2、沉澱硫黃硫黃與氫氧化鈣共熱生成多硫化鈣， 再加入稀硫酸使硫黃沉澱出來。其粒徑為 1～5 微米。純度高，在膠料中分散性好，但成本高，僅於用高級製品。

3、膠體硫黃 ( 高分散硫黃 ) 粉末硫黃或沉澱硫黃加入保護膠體 ( 分散劑 ) ，經球磨或膠體磨研磨而成， 其粒徑為 1～3 微米，適用於膠乳製品。

4、不溶性硫黃由粉末硫黃加熱至沸點， 傾於冷水中急冷而成。它是透明的無定形彈性硫黃，不溶於二硫化碳。使用這種硫黃不會噴硫， 也不易產生早期硫化。很適於高溫短時間硫化。但價格較貴。一般用於特別重要的製品。5、表面處理硫黃在硫黃粒子表面上包覆一層油類和異丁烯等物質，有利於分散。採用 2.5%的油類與硫黃製成膏劑，分散效果也很好。

二、硫黃在膠料中的性狀為了獲得質量均一的橡膠製品， 硫黃在膠料中必須均勻分佈。瞭解硫黃在橡膠中的溶解、擴散和結晶作用，對於正確制訂工藝是非常重要的。硫黃能溶解於橡膠中，但溶解度不大。隨著溫度升高，溶解度增大，冷卻膠料時則下降， 超出飽和量的硫黃容易析出至膠料的表面， 並呈細微結晶， 這種現象稱為“噴硫”。

噴硫現象破壞了硫黃在膠料中分散均勻度， 致使硫化質量降低；同時也會降低膠料表面的粘著力， 給加工生產造成困難。如果膠料中的硫黃用量不超過室溫下硫黃在膠料中溶解度， 則在常溫下不會噴出。此外採用不溶性硫黃可有效地避免噴硫現象。三、橡膠與硫黃的化學反應硫黃主要用於天然橡膠和二烯類通用合成橡膠 ( 丁苯橡膠、丁腈橡膠、順丁橡膠、異戊橡膠 ) ，也可用於三元乙丙橡膠和丁基橡膠的硫化。

硫黃在常溫下不能與橡膠反應。為使硫黃易於進行反應，必須使 S8 環裂解，而裂解所需能量為 268.0kj/mo1 。為此，用硫黃硫化橡膠，就必須提高溫度。在較高的溫度下，八元環硫黃開環均裂成雙基活性硫黃分子。

這些活性硫黃分子又可分裂為含有不同硫原子數目的較短一些的多硫雙基·Sx ·，如· S 6·、·S5 ·、·S4 ·、·S3 ·、·S2 ·，但一般說來，由於反應

所需要的能量太高， 故生成單硫雙基的或然率很小。多硫雙基形成後，將與橡膠分子發生反應。反應的位置將發生在橡膠分子的雙鍵和雙鍵附近 α- 碳原子上。這是因為雙鍵的不飽和性而易於進行自由基的加成反應，而超共軛效應使 α- 碳原子上的氫原子變得活潑，而易於進行自由基的取代反應。橡膠的單純硫黃硫化反應中， 除在橡膠的雙鍵位置和α- 碳位置上產生多硫交聯外， 還可能產生鄰位交聯、 分子內硫環化合物。此外，還存在共軛三烯的生成反應。

隨著硫化時間的延長， 多硫交聯鍵的分解重排增加， 而分解出的硫又會生成硫環化合物。因此，硫環化合物的生成量增多。大量硫環化合物的生成，消耗了相當數量的活性硫，降低了交聯效率。實際上，硫黃交聯橡膠的反應極為複雜， 除以上所述反應外， 同時還可以發生許多其它反應，一般來說，分子內硫環化合物的生成， 會不利於改善性能， 且降低硫化膠的彈性，而主鏈上的共軛二烯結構化學上不夠穩定， 易產生老化等化學反應， 使性能下降。總之，單純用硫黃硫化橡膠，硫化時間長，硫化交聯效率低，硫化膠的性能差，故硫黃鬚與硫化促進劑，活性劑並用。
