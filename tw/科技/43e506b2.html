Python文件讀寫操作詳細整理（一）

open函數參數說明：

file: 文件的路徑

mode: 讀寫模式

encoding: 編碼格式

（1）讀寫字符

r --- 從開始就是讀

w ----從開始寫

a ----從末尾寫

（2）讀寫字節

rb----從開始就是讀

wb----從開始寫

ab----從末尾寫

（3）w+

打開一個文件用於讀寫。如果該文件已存在則打開文件，並從開頭開始編輯，

即原有內容會被刪除。如果該文件不存在，創建新文件

（4）wb+

以二進制格式打開一個文件用於讀寫。如果該文件已存在則打開文件，並從開頭開始編輯，

即原有內容會被刪除。如果該文件不存在，創建新文件。一般用於非文本文件如圖片等

（5）a+

打開一個文件用於讀寫。如果該文件已存在，文件指針將會放在文件的結尾。

文件打開時會是追加模式。如果該文件不存在，創建新文件用於讀寫

（6）ab+

以二進制格式打開一個文件用於追加。如果該文件已存在，文件指針將會放在文件的結尾。

如果該文件不存在，創建新文件用於讀寫

示例-1：

#讀取文件，獲取文件流對象

file = open(file='d:/test/hello.txt',mode='r',encoding='utf-8')

#讀取文件所有內容

content = file.read()

print(content)

#關閉文件

file.close()

示例-2：使用with open

with open(file='d:/test/hello.txt',mode='r',encoding='utf-8') as file:

content = file.read()

print(content)

#關閉文件

file.closed

示例-3：讀取文件的每一行

with open(file='d:/test/hello.txt',mode='r',encoding='utf-8') as file:

line = file.readline()

print(line)

line = file.readline()

print(line)

line = file.readline()

示例-4：讀取文件內容，返回值為List

with open(file='d:/test/hello.txt',mode='r',encoding='utf-8') as file:

#返回值為List

list = file.readlines()

print(list)

結果：['hello you\n', 'hello me']

示例-5：

file = open(file='d:/test/hello.txt',mode='w',encoding='utf-8')

#寫並換行

file.write('hello everbody\n')

file.write('hello world')

file.close()

Python文件讀寫操作詳細整理（三）

示例-1：

from io import StringIO

#創建對象

f = StringIO()

f.write('hello')

#獲取值

v = f.getvalue()

print(v)

#關閉

f.close()

結果：hello

示例-2：讀取5個字符

#coding=utf-8

from io import StringIO

f = StringIO('hello you')

s1 = f.read(5)

結果：hello

示例-2：讀取1行字符，注意有個換行\n

#coding=utf-8

from io import StringIO

f = StringIO('hello you\nhello me')

s2 = f.readline()

print(s2)

結果：hello you

示例-3：讀取所有行字符，注意有個換行\n

#coding=utf-8

from io import StringIO

f = StringIO('hello you\nhello me')

s3 = f.readlines()

print(s3)

f.close()

結果：['hello you\n', ' hello me']

示例-4：

from io import BytesIO

f = BytesIO()

f.write('漢字'.encode('utf-8'))

v = f.getvalue()

print(v)

f.close()

結果：b'e6b189e5ad97'


示例-5：

#coding=utf-8

import pickle

list = [1,2,3]

#將List序列化為字節對象

obj = pickle.dumps(list)

print(obj)

#將字節轉對象

new_list = pickle.loads(obj)

print(new_list)

結果：

b'8003]q00(K01K02K03e.'

[1, 2, 3]

示例-6：

#coding=utf-8

import pickle

list = [1,2,3]

#將List序列化為字節對象

obj = pickle.dumps(list)

file = open('d:/test/hello.txt','wb')

#將對象寫到文件中

file.write(obj)

file.close()

file = open('d:/test/hello.txt','rb')

content = file.read()

print(content)

new_obj = pickle.loads(content)

print(new_obj)

示例-7：使用with open

#coding=utf-8

import pickle

list = [1,2,3]

#將List對象序列化後寫入文件

with open('d:/test/hello.txt','wb') as file:

pickle.dump(list,file)

#從文件中讀取對象

with open('d:/test/hello.txt','rb') as file:

obj = pickle.load(file)

print(obj)

示例-8：

#coding=utf-8

import json

students={

"sid" : "s001",

"sname": "lily",

'age':21

}

#將字典對象編程json字符串對象

s1 = json.dumps(students)

print(s1)

#查看轉換後的類型（字符串類型）

print(type(s1))

#將s1對象轉化為字典對象

s2 = json.loads(s1)

print(s2)

#查看轉換後的類型（字典類型）

print(type(s2))

結果：

{"sid": "s001", "sname": "lily", "age": 21}

<class 'str'>

{'sid': 's001', 'sname': 'lily', 'age': 21}

<class 'dict'>
