關於搬家，這些小技巧你知道嗎？

2020-11-14
| 科技

3 年 3 個城市，從廣州到上海，再到北京，經歷了兩次跨省搬家、一次同城換房後，對於「搬家」這個詞，我已經從不知所措無處下手，到現在遊刃有餘不慌不忙。可以說是用親身經歷踩過了一些坑，也學到了一些經驗。

我的形象很簡單，大城市打拼的社畜一枚，普通的租房人群，房間的物品很符合女生該有的數量，但也沒有多到誇張（至少化妝品一類真的沒有很多），也不是極簡生活實踐者（因為還有點收集癖），所以本文的適用對象面對的是大部分租房青年。

我認為「搬家」這件事從開始準備到最後完成，就像打王者一樣，1 塔 2 塔逐步攻下，最後水晶前火力全開，VICTORY！

勝利✌️

（彙報一下自己的近況：國慶假期利用一天的時間，完成了看房到搬家收拾，忙完自己都不信原來搬家全過程可以這麼快。）

拿下 1 塔——前期準備

搬到哪裡去？

一般不管是跨省搬家還是同城，基本都是會先把落腳點找好後，再開始準備搬家事項。畢竟有了目標以後，會更有方向性的整理。

像我之前去廣州和上海，都是在家裡通過網上資源（豆瓣 / zuber）來找個人房東和合租室友。缺點就是沒法去現場看房，有些細節沒法核對。

網絡房源平臺

現在基本都是住的自如，雖然房租會貴一點，但是會比較省心吧。不用自己單獨再去辦寬帶什麼的，維修也可以直接線上保修（或者貝殼找房）。

部分物品的取捨

終於有自己的房間了，覺得新鮮好玩的都會買回來，甭管用不用得上，「新房間就該配新的」。然而搬家整理的時候你就會發現，好多東西買來就是佔地方的，基本沒用過。

像以前我買過的：放在床上的小桌板 / 可夾在床板的看平板神器，雖然可能花費不怎麼多，但是使用過後自己覺得利用率不高，屬於沒必要帶走的東西，遂搬家時棄之。

不過衣服鞋子這種就還好，一般一年週期內買的都不會嫌棄，但凡開始嫌棄的 / 突然覺得醜，可以放在閒魚 / 捐出去（捐出去的效率會更高一點，閒魚你需要預估時間成本）。

線上衣服捐贈平臺比如：飛螞蟻、白鯨魚等，都可以線上提交後上門回收，這些平臺在支付寶裡「舊衣物回收」也可以便捷使用。

另外一些物品，比如小桌子 / 檯燈 / 部分碗筷廚具等，像我這個宜家的小桌子，如果快遞的話，打包裝箱難度挺大，還得考慮半路磕碰等，不如忍痛割愛留給下一個住戶或者送人。但是同城搬家的話，還是會帶上這個小東西的。

現在放在床邊繼續利用

這種會額外增加運輸途中的風險和難度的物品，不如用物流省下的錢，去購買一份新的，何樂而不為呢。

打包工具 & 方法

準備

打包紙箱

首先搬家必備——淘寶搜索「打包紙箱」即可，會有各種尺寸 / 有無把手扣等規格的紙箱。

尺寸的話，針對個人搬家我比較推薦「60x40x50」或者「50x50x50」這兩種規模的紙箱。比他們大的不方便拿，比他們小的隨時都可以找到替代品。有無把手扣主要是方便搬運，有放手的地方總比一個箱子不知道抓哪發力更好一些。

手扣和尺寸參考

數量建議 3-4 個。其中一個用來裝鞋子，鞋子多的也可以用兩個紙箱。尤其是籃球鞋、商務通勤鞋都非常怕壓。買鞋的鞋盒我都不會扔，就是為了搬家時保護鞋子。所以這種時候把他們都碼到合適的鞋盒裡，再通過形狀穿插，最大程度的利用好紙箱空間。其餘的見縫插針填補好即可。

還可以再拿出來一個放書本類等規則型物品 / 收藏品等。需要注意的是，尤其是書本這種怕水或者部分怕碎的物品，都需要再進行額外的單獨包裝。

其次，小體積的紙箱，完全可以利用一些平時收完快遞沒扔的快遞盒子，在確定搬家日期前，刻意收集一些形狀完好 / 乾淨的。用來裝一些桌子上的物品 / 廚房裡的調味品 / 衛生間的洗浴用品等中小型零散類，相比一整個大紙箱，這種會更方便搬過去後快速分類物品。

分裝說明書 / 各類襪子的箱子

需要注意的是，洗浴用品尤其是已經開封用過的，一定要在封口處用寬膠帶做好嚴絲合縫的粘貼準備，防止流體的意外傾倒（膠帶就很隨意了，但是儘量用寬膠帶，窄膠帶用量很多還很費時間）。

其實我平時生活裡，就會利用一些好看的包裝盒來裝一些我的床邊小物件 / 首飾等等，搬家的時候，這種只要包裝一下封個口，到了新家直接拿出來立刻用，節省時間還不凌亂。

一般這種可以合蓋的 / 或者是一些零食帶有分隔空間的包裝盒，都可以利用起來。

日常用包裝盒裝一些小零碎

打包袋

袋子類準備

首先外購打包袋必備，一般用來裝一些床上用品和不太重要的衣物（不怕壓），所以一定要買防水材質的。

尺寸方面，購買 2-3 個圖下這種 105 升 / 126 升即可，其中一個用來裝床上用品。

尺寸參考

如果想要更精緻地打包，可以再購置一些真空打包袋，對床品和衣服進行密閉封裝後，再放入打包袋中，最大程度的利用打包袋「無形狀包裝」，可以塞更多東西。

真空袋壓縮效果

同時這種拉鍊口的，不用太擔心會散落。因為在物流打包時，師傅會專門在外邊再套一層編織袋的，然後再用膠帶固定。而且這種拉鍊款，對於後期我們拆開收拾也會更簡單一點。

如果在打包後期有一些隨身現在用著的物品，不想放進各種封口箱子等，你也可以放置在自己的包裡 / 日常購物用來裝衣服的手提袋等，

額外準備：防震泡沫墊

防震泡沫

防震泡沫墊的話，如果個人搬家，倒也沒必要買那麼大一卷，可以買這種方形的防震泡沫一打，然後塞在各個需要的角落，或者是套在外面。如果需要防震的物品很多且中型以上的偏多，那麼可以考慮買一卷。

推薦用途：隔離廚房用品 / 洗浴用品、包裹書本 / 中型擺件等易碎品，或者在每個打包盒底層放置，做好充足的減震措施。

二塔快速進擊——中期運輸準備

跨省：德邦快遞 3.60

跨省的話，推薦用物流。這個物流不是說畢業時學校樓下那種物流，哪個便宜用哪個。自己搬家的話，還是找點運輸站點都比較多的牌子。

我跨省搬家一直用的是德邦物流。德邦有個專門運大件的業務 ------ 大件快遞 3.60。具體的服務可以看下圖的官網介紹，選擇這個服務就是因為可以直接運送到家門，就很 nice。

直接運送到家門就很 nice

成本預估

參考距離：上海到山東 參考數量：三個紙箱（505050）+ 三個打包袋（105 升 * 3）全部裝滿 參考價格：430 元

⚠️額外注意事項

在叫師傅上門收件之前，提前電話溝通一下，大概描述一下你的物品數量（有幾個箱子 / 幾個袋子）這樣師傅就會準備好車的空間，不至於來了以後一看東西太多而車子空間不允許，導致的時間耽擱。

同時也叫師傅準備好多個膠帶，對於有多個打包袋的情況，讓他用膠帶將幾個打包袋多繞幾圈固定在一起，這樣就會減少出現快件丟失的概率。

同城：貨拉拉 / 自如搬家

同城的話就比較省心一點了，自己有車並可以搬運的話，當然可以自己搞定。對於租房一族而言，還是從網上找個搬家師傅來更划算一些（如果是這種個人租房，沒必要叫搬家公司，我們只有運輸的需求），推薦用「貨拉拉」，比較實惠。

如果你是租的自如的房子，也可以直接在自如 APP 裡用「自如搬家」的業務，不管是價格還是車型規模，跟貨拉拉也差不多（我這次搬家就是體驗的自如小麵包車，20 公里花了 135 元，其中用了 20 元優惠券。師傅全程幫忙搬運的，都很放心）。

日常租房（合租）搬家的話，1～2 人一輛小麵包車完全可以搞定。如果是整租的話，可以根據物品數量以及常住人口數，選擇麵包車 / 大一點的貨車進行搬運。3 人以上就可以考慮中型麵包車。

參考價格

成本預估

一般北上廣的話，同城基本都是 100-200 間就差不多可以搞定。可以提前在軟件上輸入距離和部分需求進行金額的預估。

⚠️額外注意事項

不管住的地方有沒有電梯，都要跟師傅提前溝通好搬運時能否上樓幫忙（搬走和搬入都需要），加多少錢可上樓搬運，除非你想借此鍛鍊身體，其他情況下真的建議能幫忙就幫忙吧。血的教訓，太累了。

有錢人的選擇：無需操心的一站式搬家服務

當然也有那種「一站式搬家服務」，會有專門的人來上門幫你收拾整理，然後按照日式風格分箱打包，後期再幫你佈置還原。

之前在微博看鐘文澤老師就使用的這種搬家方式，搬一趟人力花費大概是 3000-6000 之間。國外 YouTube 上也有類似服務的實拍視頻，如果你是那種害怕整理和恐懼搬家的人兒，可以去油管上看一下。

亦凡微博截圖

挺進高地——後期收拾整理

當你確定了「快遞師傅會把行李送到家門」「搬家師傅幫你抬行李到家裡」的話，後期收拾的過程你會輕鬆一些。

其實我個人而言，對於搬家前的整理，我會更頭疼搬家後的整理。因為看到那滿屋子的東西，我就覺得跟無底洞一樣。

在真正擼起袖子開幹前，我們需要先對自己的新住處有個大概規劃，提前規劃好各種物料的擺放佈局。你可以根據之前的房間擺放去還原位置，也可以根據新環境重新佈置。

我建議是每次都重新規劃一次，就像重置新鮮感一樣。有時候同一件物品，因為擺放角度不同，都會讓你覺得耳目一新的波動。

關於家居收納方法，可以參考 《麻理惠的整理魔法》。這本書中有對家居收納提出的六大法則，以及針對多個場景「衣櫃」「書架」「小物品」「廚房」「紀念品」進行的詳細剖析（麻理惠本人看起來就很溫柔 555）

節目截圖

如果你不喜歡看長篇文字，奈飛上也有麻理惠的真人秀節目 《怦然心動的人生整理魔法》，類似於《粉雄救兵》（Queer Eye）的展現方式，每一集都會去到一個家庭，幫助他們重塑和改變。這個會更有趣味性和實用性，B 站上也有搬運的 中字視頻，看到下邊大家的評論，覺得是一個看了不會後悔的節目。

B 站網友評論

水晶前的勝利——享受過程

像我們這種自己參與全部搬家過程的，雖然說有點費力，但是可以在收拾東西過程中，梳理物品的使用價值和使用頻率，矯情一點還能回憶物品的來源（不管是開心的還是不開心的，但都是你經歷過的）。當然，也有可能會在某個角落收穫意外錢財。

我在這幾次的搬家整理時，會慢慢發現自己的一些整理短板。我有很多物品都處於「存在好像也沒什麼用，丟了又覺得挺可惜的」狀態，然後我的處理方法就是：全都留下來，早晚有一天會用到。

就像我們平時收藏網站 / 乾貨視頻等等，馬了就等於做了。放在收藏夾裡，它就沒有展示出來自己的使用價值，哪怕你不用但是你分享出去了，被有需要的人是用了，那麼你就完成了對這個「物品」的使用價值傳遞。

所以當這個東西處於「存在好像沒什麼用，丟了又覺得可惜」時，要麼自己用起來，要麼分享給需要的人。

學會捨棄一些當下「不需要的物品」，才會有更多的空間去容納真正讓自己提高效率和幸福的東西。

搬家大概就是這樣吧。
