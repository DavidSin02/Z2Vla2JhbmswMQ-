分身變永動的劫？拉克絲傷害變十倍！雲頂之弈 10.5 版本前瞻

2020-10-29
| 科技

3 月 2 日，雲頂之弈的設計師在外網上傳了雲頂之弈 10.5 版本的前瞻視頻，我們遊犀社區的遊犀攻略君對其做了簡單的翻譯，小犀牛也在這裡分享給大家～

以下內容由遊犀攻略君翻譯自雲頂之弈設計師視頻：

羈絆

【劍士】額外攻擊觸發機率：35% -> 40%

【雷霆】暴擊或受到暴擊時對敵方造成的傷害：80/250/500 -> 80/250/550

【極地】眩暈時造成的額外魔法傷害：75/175/350 -> 75/175/475

另外，雲頂官方在透露版本前瞻時表示會對狂戰士進行調整，但是設計師的視頻並未提及，最終調整還是以實際官方公告為準。

英雄平衡性

1 階

【黛安娜】 護盾持續時間：3s -> 4s，創造烈焰法球數： 3/4/5 -> 3/4/6

【蕾歐娜】 屏障持續時間：5s -> 4s

【婕拉】 初始 / 最大法力值： 0/75 -> 0/65 ， 召喚火焰噴射者的數目 2/3/4 -> 2/3/5

2 階

【雷克塞】 狂怒之咬真實傷害： 250/550/850 -> 250/550/1000

【辛德拉】 法球傷害： 175/350/600 -> 225/375/650

【亞索】 初始 / 最大法力值：0/85 -> 0/90

3 階

【蒙多醫生】 毒雲造成傷害： 50/100/150 -> 50/100/200

【伊澤瑞爾】 寒冰射擊傷害： 225/250/900 -> 250/450/900

【卡爾瑪】 鼓舞攻速： 40%/60%/80% -> 35%/50%/100%

【魔騰】 治療生命值： 40%/45%/50% -> 50%/60%/70%

4 階

【迦娜】 季風治療值： 20%/30%/100% -> 25%/35%/200%

【圖奇】 物理攻擊傷害： 65 -> 70

5 階

【劫】 分身法力消耗提高： 50/25/0 -> 50/25/-125

7 階

【拉克絲】 終極閃光傷害： 550/900/9999 -> 550/900/99999

以上便是小犀牛為各位帶來的雲頂之弈 10.5 的版本前瞻了，不知道各位對於這個 10.5 版本是否滿意呢？

歡迎在評論區給小犀牛留言，小犀牛也會更快更及時地為大家帶來第一手的自走棋攻略與資訊，我們下期再見～
