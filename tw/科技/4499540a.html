一鍵輕鬆搞定合併和拆分單元格

2020-10-29
| 科技

用Excel做統計是很多職場夥伴的喜好，因為劃分清晰明瞭，Excel的行列線像經緯線一樣將內容分的清楚明白。但是絕大多數盆友都被合併和拆分單元格整的苦不堪言吧，有的請舉爪。。。老闆們喜歡將相同內容合併著看，而做統計的時候合併單元格帶來的痛苦是誰用誰知道哇。今天我們一起脫離苦海，走上人生巔峰。

總體思路是：合併時從下至上，即從最後一行開始判斷單元格內容是不是相同，相同則合併，不相同跳過；

拆分時是從上到下，即從第一行開始判斷單元格是不是合併了，合併了就拆開，沒合併跳過 1. 打開VBA編輯區，插入模塊，輸入以下代碼

Sub Button1_Click()

Dim k As Integer

Application.DisplayAlerts = False

For k = Cells(Rows.Count, 1).End(xlUp).Row To 2 Step -1

If Cells(k, 1) = Cells(k, 1).Offset(-1, 0).Value Then

Range(Cells(k, 1), Cells(k - 1, 1)).Merge

End If

Next

Application.DisplayAlerts = True

End Sub

Sub Button2_Click()

Dim m, n As Integer

Application.DisplayAlerts = False

For m = 1 To [A1048576].End(xlUp).Row '最後一行數據的行數

If Cells(m, 1).MergeCells = True Then

n = Cells(m, 1).MergeArea.Count '合併的單元格個數

Range(Cells(m, 1), Cells(m + n - 1, 1)).UnMerge '拆分單元格

Range(Cells(m, 1), Cells(m + n - 1, 1)).FillDown '給每個單元格填充數據

m = m + n - 1

End If

Next

End sub

開發工具→插入→表單控件→按鈕→指定宏（Button1_Click()/Button2_Click()）→按鈕右鍵修改控件信息
