如何畫函數圖像——「動態數學軟件GeoGebra」

2020-10-29
| 科技

在GeoGebra裡，一輸入表達式，函數圖像瞬間出來。

下面，我們來看看常見函數是如何輸入的。

分類

普通函數

例：f(x) = sin(2x)，g(x) = x² + 2

【直接輸入“=”右邊的式子即可】

分別輸入sin(2x) ， x^2 + 2


係數待定的函數

【法一】輸入表達式，再點“創建滑動條”

例：f(x) = k x + b

輸入k x + b，再“創建滑動條”

【法二】先創建滑動條，再輸入表達式

例：f(x) =a / x

創建滑動條a 後，輸入a / x


限定範圍的函數

1.限定自變量

例：f(x) =2x,x>0

【輸入】如果(x>0,2x)

【也可輸入】2x / (x>0)

理解：前者是“如果……，就……”；後者是運用布爾運算，具體見下圖。

輸入：2x / (x>0)

2.限定因變量

例：g(x) =sin x，且g(x) >0

【輸入】如果(sin(x) > 0, sin(x))


分段函數

例：x<0時，f(x)=-x；否則，f(x)=x

【輸入】如果(x < 0, -x, x)

理解：如果( <條件>, <結果>, <否則> )

例：x≤-1時，f(x)=-x；-1<x<1時，f(x)=x；x>2時，f(x)=2x

【輸入】如果(x ≤ -1, -x, x < 1, x², x > 2, x)

理解：如果( <條件1>, <結果1>, <條件2>, <結果2>,…… )
