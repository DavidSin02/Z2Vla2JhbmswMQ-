QT開發（五）——窗口組件和窗口類型

2020-10-29
| 科技

一、窗口組件

圖形用戶界面由不同的窗口和窗口組件構成，<QtGui>頭文件包含窗口組件，對應QT中的GUI模塊，QT以組件對象的方式構建GUI。

組件的類型：

A、容器類（父組件）用來包含其他的界面組件

B、功能類（子組件）用於實現特定的交互功能

QT中沒有父組件的頂級組件叫窗口。

QWidget是容器組件，繼承自QObject類和QPaintDevice類，QObject類是所有支持QT對象模型的基類，QPaintDevice類是QT中所有可繪製組件的基類。

QWidget組件

QWidget能夠繪製自己和處理用戶輸入，是QT中所有窗口組件類的父類，是所有窗口組件的抽象，每個窗口組件都是一個QWidget，QWidget類對象常用作父組件或頂級組件使用。

QLabel組件

QLabel組件用於顯示一個提示性字符串，是功能性組件，一般需要父組件作為容器，QLabel作為窗口存在沒有實際意義。

二、窗口類型

QT中可以根據需要定製窗口樣式，不同的窗口類型可以使窗口具有不同的樣式特性。

窗口類型：

Qt::Widget

Qt::Window

Qt::Dialog

Qt::Sheet

Qt::Drawer

Qt::Popup

Qt::Tool

Qt::ToolTip

Qt::SplashScreen

Qt::Desktop

Qt::SubWindow

窗口標識：

Qt::WindowStaysOnTopHint

三、座標系統

QT中的座標系統以左上角為原點，水平為x軸，從左向右為正向，垂直為y軸，從上到下為正向。

1、窗口部件的定位

GUI程序在座標系統中進行窗口和部件的定位

定位類型分為：

A、頂級窗口部件的定位

B、窗口內部件的定位

C、窗口部件的大小設置

QT使用統一的座標系統定位窗口部件的位置和大小

QT部件類提供成員函數在座標系統中進行定位

QWidget類提供了窗口部件所需的座標系統成員函數

intx() const;

inty() const;

int width() const;

int height() const;

以窗口左上角為座標系原點，但width()和height()分別為客戶區的寬和高

const QRect &geometry() const;

intx() const;

inty() const;

int width() const;

int height() const;

以客戶區左上角為座標系原點，width()和height()分別為客戶區的寬和高

QRectframeGeometry() const;

intx() const;

inty() const;

int width() const;

int height() const;

以窗口左上角為座標系原點，width()和height()分別為窗口的寬和高

使用實例：

#include <QtGui/QApplication>#include "Widget.h"#include <QDebug>int main(int argc, char *argv[]){ QApplication a(argc, argv); QWidget w(NULL,Qt::Window|Qt::WindowSystemMenuHint); w.setWindowTitle("HelloWorld"); w.show(); qDebug() << w.x(); qDebug() << w.y(); qDebug() << w.width(); qDebug() << w.height(); qDebug() << w.geometry().x(); qDebug() << w.geometry().y(); qDebug() << w.geometry().width(); qDebug() << w.geometry().height(); qDebug() << w.frameGeometry().x(); qDebug() << w.frameGeometry().y(); qDebug() << w.frameGeometry().width(); qDebug() << w.frameGeometry().height(); return a.exec();}

geometry()、frameGeometry()函數需要在show()函數後調用。QT平臺在窗口顯示出來之前無法確定窗口的大小。

2、窗口部件的設置

QWidget類提供了設置窗口部件大小的成員函數

A、改變窗口部件的大小

void resize(const QSize &);

void resize(int w, int h);

B、改變窗口部件的位置

void move(int x, int y);

void move(const QPoint &);
