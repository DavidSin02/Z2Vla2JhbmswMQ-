461. 兩兩交換鏈表中的節點

2020-10-29
| 科技

給定一個鏈表，兩兩交換其中相鄰的節點，並返回交換後的鏈表。


示例 1：

輸入：head = [1,2,3,4]

輸出：[2,1,4,3]

示例 2：

輸入：head = []

輸出：[]

示例 3：

輸入：head = [1]

輸出：[1]


提示：

  • 鏈表中節點的數目在範圍 [0, 100] 內
  • 0 <= Node.val <= 100


非遞歸解決

這題主要考察的是鏈表節點的交換，要交換鏈表的節點首先要搞懂鏈表節點的斷開和連接，之前寫過一篇文章，《352，數據結構-2,鏈表》，圖文詳細介紹了單向鏈表，雙向鏈表，還有環形鏈表的斷開和鏈接。


這道題我們可以人為的把鏈表分為兩組，前面兩個節點為一組，後面的為一組，我們首先交換第一組的兩個節點，交換完之後把後面的節點再分為兩組，繼續這樣交換下去……，直到不能交換為止。注意，如果能交換，那麼原來鏈表的第二個節點就是我們要返回新鏈表的頭結點。

這裡就以示例為例畫個圖來看下

再來看下代碼

public ListNode swapPairs(ListNode head) {    //鏈表至少有2個節點才能交換，否則就不要交換    if (head == null || head.next == null)        return head;    //這裡的first，second，third可以參照圖中的標註    ListNode first = head;    ListNode second;    ListNode third;    //這個是交換鏈表之後的尾結點，他的next要指向新交換的鏈表    ListNode preLast = null;    //這個只賦值一次，它是要返回的新鏈表的頭結點    ListNode newHead = head.next;    //如果能交換就繼續操作    while (first != null && first.next != null) {        //給second，third賦值        second = first.next;        third = second.next;        //first和second這兩個節點交換        first.next = third;        second.next = first;        //這個時候second就是交換之後新鏈表的頭結點，        //如果preLast不為空，說明前面還有交換完成的鏈表        //，要讓preLast的next指向新鏈表的頭結點        if (preLast != null) {            preLast.next = second;        }        //因為first和second交換之後，first就變成新鏈表        //的尾結點了，把它保存在preLast中        preLast = first;        //前兩個交換了，然後從第3個在繼續操作        first = third;    }    //返回新鏈表    return newHead;}


遞歸解決

之前講過《426，什麼是遞歸，通過這篇文章，讓你徹底搞懂遞歸》，遞歸有2個條件，一個是終止條件，一個是調用自己，並且要同時具備，所以這題的遞歸模板很容易寫出來。

public ListNode swapPairs(ListNode head) {    //終止條件，如果節點為空或者沒有後繼節點，就不能交換    if (head == null || head.next == null)        return head;    //一些邏輯運算，可有可無，視情況而定    //前面兩個節點我們保留不要動，從第3個節點往後開始    //兩兩交換    ListNode newListNode = swapPairs(head.next.next);    //一些邏輯運算，可有可無，視情況而定    return 交換之後的鏈表;}

假如使用遞歸從第3個節點往後的節點全部兩兩交換了，這個時候我們可以把鏈表分為3部分，第一個節點，第二個節點和後面交換完成的鏈表，就是1→2→3，這種形式，我們只要再把1和2位置交換了，那麼這道題就完成了。來看下代碼

public ListNode swapPairs(ListNode head) {    //邊界條件判斷    if (head == null || head.next == null)        return head;    //從第3個鏈表往後進行交換    ListNode third = swapPairs(head.next.next);    //從第3個鏈表往後都交換完了，我們只需要交換前兩個鏈表即可,    //這裡我們把鏈表分為3組，分別是第1個節點，第2個節點，後面    //的所有節點，也就是1→2→3，我們要把它變為2→1→3    ListNode second = head.next;    head.next = third;    second.next = head;    return second;}


交換節點的值

還可以換種思路，因為交換鏈表的結點需要不停的斷開和連接，比較麻煩。所以我們還可以不交換鏈表的節點，直接交換鏈表節點的值即可，這樣就非常簡單了

public ListNode swapPairs(ListNode head) {    int first;    ListNode temp = head;    while (temp != null && temp.next != null) {        //直接交換鏈表節點的值        first = temp.val;        temp.val = temp.next.val;        temp.next.val = first;        temp = temp.next.next;    }    return head;}


總結

鏈表節點的交換要注意，在交換前兩個節點之前，要把第3個節點給先保存起來，否則交換完了就找不到第3個節點了。
