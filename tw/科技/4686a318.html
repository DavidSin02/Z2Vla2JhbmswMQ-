Excel批量顯示和隱藏工作表，第一種方法相當實用！

嗨，大家好，歡迎來到Excel倫特吧，今天要給大家分享的是一個超級簡單的高手技巧，批量隱藏和顯示工作表。我們來看看具體操作吧↓↓↓

Excel skills

隱藏工作表，我們平時一般是怎麼做的呢？見下圖：

隱藏工作表

大家都知道，要想將顯示隱藏的工作表，只能將隱藏的工作表逐個顯示出來，這樣就比較麻煩，有沒有辦法一次性讓所有隱藏的工作表全部顯示出來呢？我們來看看怎麼做：

操作方法：

  • 隱藏工作表之前我們在視圖菜單中自定義視圖為“All”
  • 隱藏了部分工作表之後，我們自定義視圖為“封面”
  • 這樣就可以在自定義視圖中來回切換，達到批量隱藏和顯示工作表啦！

是不是很方便？

批量隱藏和顯示工作表

除了這種方法之外，我們還有另外一種方式隱藏工作表的方式，可以通過設置達到隱藏工作表標籤的目的。來看看怎麼操作吧！

  • 文件——選項——高級
  • 取消勾選顯示工作表標籤前面的方框
  • 確定後，就不會顯示工作表標籤啦！

同理，如果要顯示出來，直接勾選即可。

批量隱藏和顯示工作表

今天我們要講的第三種方法是VBA實現，主要是雖然本文第一種方法已經完勝了，但是為了讓大家更全面的學習技巧，故還是將此法分享給大家。代碼如下：

Sub 隱藏工作表()

'以下由Excel倫特吧製作

'=========我是華麗分割線=========

Dim sht As Worksheet '定義工作表對象

For Each sht In Worksheets 'For Each循環語句，在工作簿中所有表格循環執行以下命令

If sht.Name <> "封面" Then '判定工作表名稱是否為封面

sht.Visible = xlSheetHidden '工作表名稱不等於“封面”的工作表隱藏起來

End If

Next sht

'=========我是華麗分割線=========

End Sub

Sub 顯示工作表()

'以下由Excel倫特吧製作

'=========我是華麗分割線=========

Dim sht As Worksheet '定義工作表對象

For Each sht In Worksheets 'For Each循環語句，在工作簿中所有表格循環執行以下命令

If sht.Name <> "封面" Then '判定工作表名稱是否為封面

sht.Visible = xlSheetVisible '工作表名稱不等於“封面”的工作表顯示出來

End If

Next sht

'=========我是華麗分割線=========

End Sub

具體操作如下:


這裡我們重點講下sht.Visible = xlSheetVisible 這句代碼哈，其實我們的表格顯示的狀態有三種，第一種是visible（可見），第二種是Hidden(隱藏)，第三種是veryHidden(深度隱藏)，對應的值分別為-1,0,2,所以sht.Visible =-1即等價sht.Visible = xlSheetVisible 。

另外細心的童鞋發現了，除了顯示和隱藏，還有第三種狀態——深度隱藏。

深度隱藏和一般隱藏的區別是啥？簡單的說前者在工作表標籤中是無法取消隱藏的，後者是可以取消隱藏的。我們來看看區別：

深度隱藏和一般隱藏的區別

所以我們當我們不想讓別人在取消隱藏中顯示我們隱藏的表格時，可以按上述方法將對應的表格的visible屬性設置成2，別人就沒法通過取消隱藏看到自己的表格啦，當然這種對Excel高手作用不大哈。
