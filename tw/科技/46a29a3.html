體驗最經典的 Windows 系統，裝上這個 App 就可以

2020-10-29
| 科技

Windows 95 是微軟操作系統上的一塊里程碑，經典的右下角「開始」菜單、《空當接龍》遊戲都是從這一代開始加入 Windows 系統的。現在，聊天軟件 Slack 的開發者 Felix Rieseberg 把 Windows 95 變成了一個可以在 macOS、Windows、Linux 上運行的應用程序，只要 下載安裝，你就能在電腦上重溫這款經典的操作系統。

打開應用，點擊「Start Windows 95」就會進入 Windows 95 的操作界面。雖然這個圖形用戶界面現在看起來可能有點樸素單調，但在 1995 年還是很先進的。使用鼠標操作起來就和 Windows 10 一樣，只不過點擊圖標後程序打開得有點慢，沒有最新的操作系統那麼流暢。

macOS 版的安裝包只有 129 MB 大小，卻能完整運行 Windows 95 的各類程序，並且內存佔用也僅有 200 MB 左右。測試下來，Windows 95 的寫字板、掃雷、電話撥號器、MS Paint 等內置程序都能正常運行，就是 Internet Explorer 打開後顯示頁面無法加載，開發者甚至還加入了「插入軟盤（Insert Floppy Disk）」功能，可以加在 .img 格式的文件。

由於微軟在 Windows 8 之後移除了系統內置的幾款經典遊戲，所以除了懷舊一下 Windows 95 熟悉的操作界面，你還可以在系統中重溫一下陪伴我們多年的經典小遊戲。

目前，開發者已經在 Github 上發佈了這個項目的源代碼和安裝包。想要體驗經典的 Windows 95 操作系統的可以前往下載。
