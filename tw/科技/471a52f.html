洗衣服再也不用心煩，小汙漬也能迅速祛除，超全去汙大法拿走不謝~

2020-10-29
| 科技

秋冬最煩的事情，洗衣服算一個，特別是小塊的汙漬，丟進洗衣機又怕洗不乾淨，手洗又太累，到底怎麼辦才好呢？

所以今天這篇文章給大家整理了超全去汙大法，各種各樣的汙漬都不怕，隨手就能高效去汙，這份清潔乾貨請收好！

油漬汙染

/ 土豆＋食鹽 /

油 漬

前兩天有條牛仔褲報廢了，因為吃燒烤的時候被油濺了一褲腿，用了各種方法都沒洗乾淨，最後居然被朋友拿了一個土豆輕輕鬆鬆就擦乾淨了。

先在汙漬區域上撒上一些食鹽，再拿一個土豆，切開土豆以後，用新鮮土豆的那一面擦拭汙漬區。

最後用溼毛巾把汙漬區域擦乾淨，晾乾，油汙就消失不見啦！土豆中含有澱粉，具有很強的汙漬吸附能力，而且也不會對衣服材料有傷害，處理容易掉色或者面料比較嬌貴的衣服的時候，可以使用這個方法哦！

視頻來源：生活好辦法

/ 爽身粉 /

油 漬

當然還有一種超簡單的方法，就是寶寶用的爽身粉啦！用爽身粉覆蓋住油汙區域，再用洗衣液搓洗，也可以消除汙漬哦，不過這種方法比較適合剛蹭上的新鮮油汙，那種很難祛除的油漬還是用第一種方法吧！

食物殘渣

除了油汙之外，“漏嘴巴”對掉落在身上的各種食物殘渣如何處理也很有心得，這幾個小妙招，必須要學會！

/ 小蘇打 + 醋 + 肥皂液 /

醬料漬、果汁漬

平常愛吃各類炸醬麵拌麵的你，是不是偶爾也會出現這種不小心把醬料蹭到衣服上的尷尬情況呢？

這個時候用小蘇打+醋就能解決，將小蘇打完全覆蓋汙漬區域，靜置5分鐘左右。

最後用肥皂液和醋小心的擦拭汙漬區域，一般小塊的汙漬用牙刷蘸點醋直接刷兩遍就能搞定。

再按照正常程序洗一遍衣服就乾淨了，是不是跟新的一樣呢！

/ 醋 + 洗潔精 /

番茄醬漬

吃薯條時番茄醬沾到衣服的人都知道，番茄醬特別難洗，很容易留下紅紅黃黃的印子，然後一件衣服又不能穿了。

其實番茄醬有個非常簡單的處理方法，就是：醋+洗潔精。

用醋沾溼衣服上的汙漬區，浸透幾分鐘以後塗上洗潔精，然後用小刷子輕輕擦拭一下，再按照正常的方式清洗就不會留下痕跡啦！

/ 防鏽劑 /

醬料漬、油漬、鏽漬

對於難洗的醬汁漬、染色的鏽漬，推薦大家不妨試試專業防鏽劑。一些電器設備的除鏽劑，如WD40，也具有神奇的清潔效果哦~

不管是番茄醬還是甜麵醬，在牛仔褲這種比較容易褪色的衣服上都不太適合酸洗，所以家裡有這種防鏽劑的可以拿出來用了，直接噴到汙漬處，等待5分鐘以後再照常丟進洗衣機清洗。

完全不會有褪色的痕跡，而且汙漬也消失的乾乾淨淨。WD40非常適合用在纖維、尼龍等材質上，所以不如在家裡備上一瓶試試看吧！

酒水汙漬

/ 食鹽 + 開水 /

紅 酒 漬

對付酒漬，有一個超級好用的辦法，3秒搞定，連洗都不用洗。

拿一個大一點碗，或者找一個能架起衣服的洗手檯，在酒漬區域上撒點鹽，然後在汙漬處倒上開水，汙漬就順著水直接流下去了。

這個真的非常實用，不用整件衣服都過水，最後只用把浸溼的地方晒乾，或者用吹風機吹乾就可以再上身了。

/ 過氧化氫 + 肥皂液 /

紅 酒 漬

家裡有桌墊那種比較厚布料蹭到酒漬或者飲料漬好像也很難洗，布料本身就非常難揉搓，這個時候可以用過氧化氫+肥皂液試試。

蘸取過氧化氫和肥皂液混合的清潔劑，直接浸溼桌布上髒汙的地方，等待五六分鐘左右。

最後再丟到洗衣機裡過水清洗，最後晾乾就跟新的一樣啦！

/ 白 醋 /

咖 啡 漬

喝咖啡的時候沒拿穩，直接灑在了衣服上，最後把那件衣服手洗了兩遍才弄乾淨，特別麻煩。

咖啡漬的剋星就是白醋，直接用白醋浸泡一下汙漬區域，然後過水清洗就可以了，這個方法真的減少了不少洗衣工作量！

/ 小蘇打＋白醋 /

除 油 去 黃

對於顏色更深的汙漬，還可以用小蘇打+醋混合的方式，先把小蘇打灑在汙漬表面，然後再用小刷子蘸取白醋輕輕擦拭。

最後直接照常過水，一遍就能洗出原本顏色啦！

生活汙漬

/ 阿司匹林 + 水 /

汗 漬

愛出汗的人，腋下也會經常殘留微微泛黃的汗漬，別的顏色有時候也看不出來，但是如果愛穿淺色衣服的話就比較明顯。

其實這種汗漬是最好解決的，把家裡的阿司匹林碾碎了以後加水。

直接塗抹在洗不乾淨的汗漬區域上，靜置5分鐘左右，再拿去清洗。

汗漬毫無殘留，白衣服也能跟新的一樣啦！

/ 白 砂 糖 /

甲 油 汙 漬

不管是什麼年齡段的女人，都愛塗指甲油，但是指甲油瓶子太小了，不小心碰到就是災難。

對付這種地板上斑斑點點的指甲油，其實有個很有用的江湖救急法，拿一勺白砂糖，覆蓋住汙漬區，等幾分鐘直接掃掉。

血淚經驗告訴你們，千萬不要試圖用溼紙巾直接擦地板！會留下非常難清理的汙漬，怎麼都扣不掉的那種！

/ 醋 + 卸甲水 /

化 妝 品 汙 漬

對於經常穿白襯衣的女生來說，衣領上總是會沾到很多汙漬，特別是粉底液或者防晒霜這種不太好洗還防水的汙漬，這個時候可以用卸甲油+稀釋過的醋擦一擦。

基本上能夠恢復如新，對化妝品汙漬這個方法更有效，不用使勁揉搓就能洗的乾乾淨淨。

/ 髮膠噴霧 /

口 紅 汙 漬

當然如果家裡有頭髮定型噴霧的話，搞定這類化妝品汙漬更簡單，連清洗都不用。

把定型噴霧直接噴到汙漬上，然後用溼抹布擦一擦，分分鐘就乾淨了，這個救急也非常好用，特別是小範圍的汙漬，清潔效果立竿見影。
