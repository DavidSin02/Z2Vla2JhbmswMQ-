第一個MFC程序

2019年7月，MFC入門，開始編寫第一個MFC程序。

一、編寫前的準備。

安裝vs2019；（注：安裝時勾選C++ MFC 生成工具）

VS2019好看多了。

二、正式編寫開始：

1、創建新項目

2、選擇空項目

3、輸入項目名稱HELLOMFC

4、點擊創建

5、修改項目屬性（修改"高級"-"MFC的使用"為"在靜態庫中使用 MFC"，修改"連接器"-"子系統"為"窗口 (/SUBSYSTEM:WINDOWS)"

6、添加一個源文件main.cpp

代碼如下:

#include <afxwin.h>

class MainApp :public CWinApp

{

public:

BOOL InitInstance();

};

class MyMainWnd :public CFrameWnd

{

public:

MyMainWnd();

};

MainApp a;

BOOL MainApp::InitInstance()

{

m_pMainWnd = new MyMainWnd();

m_pMainWnd->ShowWindow(m_nCmdShow);

m_pMainWnd->UpdateWindow();

return TRUE;

}

MyMainWnd::MyMainWnd()

{

Create(NULL, _T("第一個MFC程序"));

}

編譯：代碼成功運行了。第一個窗口出現了。
