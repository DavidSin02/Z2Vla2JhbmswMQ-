高速公路快充網絡雖完善，但節假日還是別讓純電動車跑高速

2020-11-14
| 科技

決定續航里程的除了產品本身的性能技術之外，還有低溫和高速行駛兩大用車環境因素。氣溫過低的情況下，電池活性受影響，加上開暖氣，特別費電，續航里程大幅度折損。

至於高速行駛，電池系統提供輸出以“高放電”模式工作，續航里程大幅度折損。姑勿論電動車高速續航里程的折損，如果高速公路充電足夠方便，純電動車長途奔襲不是什麼問題。

關鍵就在於高速公路充電設備的建設程度，據悉，為服務電動汽車用戶城際出行， “九縱九橫兩環”高速公路為骨幹網架的高速公路快充網絡已建成。


加上以廣東為基地的南方電網，純電動汽車主銷區域的高速公路快充網絡已經相對完善了。近日，曉電籍著國慶假期上高速公路溜達了一圈，發現好像不是那回事兒。


//

電樁是在的，多半是壞的

//

其實，網絡上關於高速公路快充的內容已經有很多，還整理為“充電攻略”，不少還是“親身經歷”。問題是，曉電看了這些“攻略”後發現，落差還是有的。

十一小長假從廣州出發，向鄰省福建行駛。第一個充電站正建設中，圍蔽起來暫不能使用。第二個充電站，4個電樁3個故障。第三個充電站稍微好點，同樣4個電樁，2個故障。

後面還有幾個充電站，已建成但未投入使用。充電站之間平均相隔50公里左右，總共約200公里，如果急需充電補給的電動車早已趴窩在高速公路上等待拖車救援了。

還有就是，曉電進入福建省後不久，碰巧遇到一個蔚來汽車的換電站，有趣的是工作人員開著江淮電動車過去進行設備檢修。


//

沒有停車位談何充電？

//

多個高速公路服務區的電動車充電站，要麼無法使用（設備故障），要麼還是無法使用（傳統燃油車佔用停車位）。前者可以加強人手對設備維修維護，後者則需要更多合理的指引和規範。

燃油車佔位問題十分突出，讓新能源車車主傷透腦筋。日常生活當中，由於燃油車“霸佔”充電車位，新能源車主只能“乾瞪眼、直跺腳”。高速公路充電站就更不用說了。

清明節假期，剛結束的五一小長假，十一假期進行時，高速公路免費通行，不少市民自駕出遊。導致高速公路服務區成了大型停車場。

只見整個服務區的所有充電樁密密麻麻的被傳統燃油車佔滿了，而且還有許多車子排著長龍等進服務區。雖然，急需充電的電動車車主有權讓燃油車挪位騰出充電車位，問題是燃油車能挪到哪去呢？動都動不了！


//

最後說一句

//

隨著的科技技術的發展，現在電動車的續航里程平均每半年可能會以50公里的速度在增長，國內NEDC綜合續航里程超過500km的產品比比皆是。

NEDC綜合續航里程達到600km，甚至700km，只是時間的問題，電動車六七百的續航能力已經達到傳統燃油車的續航水平了。

既要馬兒跑，又要馬兒不吃草，這是不現實的。電動車的續航里程不斷突破，充電設備的普及應用是至關重要的使用保障。

哪怕電動車續航里程不高，或者高速和低溫的里程折損太大，只要充電足夠便利，電動出行依然是最好的體驗。
