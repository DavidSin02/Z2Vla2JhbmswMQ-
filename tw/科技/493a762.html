「歡迎評論」什麼叫穩鉤？老司機上演教科書式示範

2020-10-29
| 科技

今天小編給大家介紹下操作吊車作業進行地面翻轉物件的方法：

//

物件的兜翻操作

//

兜翻又稱兜底翻。這種兜翻操作適用於一些不怕碰撞的鑄鍛毛坯件，其翻轉操作要領是：

(1)被翻轉的物件兜掛方法要正確，繩索應兜掛在被翻轉物件的底部或下部。一般死圈釦的鎖點放在被翻轉物件遠離重心的下部。

(2)繩釦繫牢後，推動起升控制器手柄，使吊鉤逐步提升。



//

物件的遊翻操作

//

遊翻的方法對於一些不怕碰撞的盤狀、扁形工作的翻轉尤為適合。其操作要領是：


根據被翻物件的尺寸大小和形狀，先把已吊掛穩妥的被翻物件提升至稍離地面時，快速開動大車或小車，人為地使吊物開始遊擺至最大擺角的瞬間，立即開動起升機構，以最快的下降速度將物件快速降落。


當被翻轉物件的下角部位與地面接觸後，吊鉤繼續下降，物件在重力矩的作用下自行傾倒，在鋼絲繩的鬆弛度足夠時，即停止下降，同時向回迅速開動大車或小車，用以調整車體位置，以達到當被翻物件翻轉完成後，鋼絲繩處於鉛垂位置。



△遊翻操作示意圖

遊翻操作時要防止物件與周圍設備碰撞

要掌握好翻轉時機

動作要快、要準


//

物件的帶翻操作

//

對於某些怕碰撞的物件，如已加工好的齒輪、液壓操縱板等精密件。一般都採用帶翻操作來完成，如下圖所示。


帶翻的具體操作方法是：帶翻操作時首先把被翻轉的物件吊離地面，再立著慢慢降落，降到被翻物件與地面剛剛接觸時，迅速開動大車或小車，通過傾斜繃緊的鋼絲繩的水平分力，使物件以支點A為中心作傾翻運動。


當吊物重心G超過支點A時，物件在重力矩作用下。就會自行傾倒。在被翻物體向自行傾倒時，要順勢開動起升機構落鉤並控制其下降速度，落鉤時要使吊鉤保持垂直



帶翻操作示意圖

a)起鉤繃繩b)開車帶物件c)落鉤翻轉


值得注意的是：帶翻這種操作，被翻轉的物件必須是扁形或盤形物件，吊起後的重心位置必須較高，底部基面較窄。


另外在操作過程中，要使吊鉤保持垂直，在帶翻拉緊鋼絲繩成一定角度以後，不允許起升捲揚，以免鋼絲繩亂繞在捲筒上或從捲筒上脫落而繞在轉軸上絞斷鋼絲繩。再有，物件翻轉時，車體的橫向運動和吊鉤的迅速下降等彼此要配合協調。
