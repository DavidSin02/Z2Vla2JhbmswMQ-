產品接貨流程

一、 接貨和儲存

——製作高質量產品的第一步

接貨和儲存工作是為顧客提供高質量成品的重要環節。正確地處理和儲存貨品是提供安全、高質量產品的第一步。

(一) 接貨

接貨流程

1. 事前準備

1) 查看員工班表，合理安排接貨人員。

2) 在接貨前安排整庫：在接貨前幾小時，應先對庫中的所有貨品進行輪替及整理，以確保最臨近保存期限的貨品最先使用。

3) 準備訂貨單據在手中，以便核對數量。

2. 核對數量

進量 = 訂量

1) 清點實際到貨數量是否與訂貨數量、送貨單數量都一致。

2) 隨機抽查庫中貨物溫度、數量及重量。

3. 檢查品質

接貨時，對於未達到正確接貨溫度的貨品，不得接收。

接貨時按制定的原料驗收標準對貨品進行全面檢查，檢驗出進貨批次原料的任何品質問題，可直接與食品加工公司聯繫，要求退貨。

接貨品質檢查要點：

1) 有效期：不接收到期或即將到期的原材料。

2) 貨品中沒有出現有雜物、變色、變質等質量問題。

3) 外觀：箱子及內包裝的密封性保存良好，無明顯破損現象；味道：所有原料無異味。

4) 產品規格符合標準要求。

5) 發生其他以上未列出的產品品質問題

4. 安全搬運

先搬運溫度敏感貨品，並注意小心搬運，不擠破、不損壞。所有貨物從下貨到進入庫過程中不得直接離地放置，需要墊置專用接貨工具。

1） 進貨順序：溼貨 乾貨

2） 搬運時一定遵循的既定安全程序

n 使用梯子拿取高於頭頂的貨品。

n 在搬起、運送和放下貨品時，要挺直背部，轉動背部會引起腰部受傷。

5. 擺放

預先整理好庫房。安全保存，預防老鼠及偷竊。

擺放原則：

n 按時間順序依次擺放。

n 從裡到外，從右到左，從下到上擺放。

n 溼貨的貨品擺放應離地15CM,離牆5CM,間距2.5CM。

n 貨品的堆放高度不要超過建議的箱數，應在冷庫頂和貨物之間留出30釐米的空間。防止貨箱壓損及倒塌，並保持空氣的正常循環。

n 擺放標籤朝外，有利於檢查生產日期。
