讓你從根本上理解C語言指針，真正明白C語言指針

2020-11-14
| 科技

C語言指針有些同學理解指針很難，因為老師每次講解指針根本沒有講解指針的本質東西，而是將一些相互混淆的東西講解在一起，今天看完這個文章你絕對能從根本理解c語言指針而且理解深刻，隨意使用指針。

首先什麼是指針：

1，C語言指針就是一個32位的數字

2，C標準規定帶*的就是定義指針，就像C標準規定int就是整形數據一樣，例如 int * p (其中int是內型，它標明指針操作的步長，例如int內型的指針每次+1就是跳四個字節，而char類型的指針每次加一是跳1個子節，其中*是指針的標號，就是制定C標準的人規定的，人為的， p就是這個變量的名字

所以我們每次在定義指針時候，先定義內型再加一個*號再加一個名字。

下邊我們用實際的例子：

#include <QCoreApplication>

#include <QDebug>

int main(int argc, char *argv[])

{

int a = 100;

int b[1] = {10};

int *c[1] = {&a};

int d[1][3] = {{1,2,3}};

/*

普通指針定義

1，先定義指針類型 int

2，定義*

3，定義名字 p_a

*/

int * p_a = &a;

/*

這裡定義一個數組指針，這個指針是代表一個數組的地址

1, 定義指針內型，由於是數組內型指針還是int類型

2，定義一個指針標號 *[1]

3，定義指針名字p_b

*/

int (*p_b)[1] = &b; //所以p_b是一個指向數組的指針

/*

這裡定義一個數組指針，這個指針是代表一個數組的地址，只是數組存放的是指針

1, 定義指針內型，由於是數組內型是int *，所以指針內型定義位int *

2，定義一個指針*[1]

3，定義指針名字p_c

*/

int * (*p_c)[1] = &c; //所以p_c是一個指向數組的指針

/*

這裡定義一個二維數組指針，這個指針是代表一個二維數組的地址

1, 定義指針內型，由於是數組內型是int，所以指針內型定義位int

2，定義一個指針*[1][3]

3，定義指針名字p_d

*/

int (*p_d)[1][3] = &d; //所以p_d是一個指向二維數組的指針

//相信大家都明白*和[]都是取地址的值

//p_b[0]就是取地址p_b的值，所以p_b[0]就相當於數組a， 所以pb[0][0]就是a[0]

//p_c[0]就是取地址p_c的值，所以p_c[0]相當於數組c,所以p_c[0][0]就是c[0]

//p_d[0]就是取地址p_d的值,所以p_d[0]相當於數組d,所以p_d[0][0][2]就是d[0][2]

qDebug()<<p_a[0] << "" << p_b[0][0] << "" << p_c[0][0][0] << "" <<p_d[0][0][2];

//輸出結果是100 10 100 3

QCoreApplication k(argc, argv);

return k.exec();

}
