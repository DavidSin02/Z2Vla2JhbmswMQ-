詳解Oracle12c中多租戶（容器）創建PDB--克隆本地PDB

2020-11-14
| 科技

概述

在容器數據庫CDB裡面如何創建PDB，官方文檔給出了一張創建PDB技術選項圖，圖片如下：

以下基於Oracle19c做演示。

1、克隆本地PDB概述

克隆本地PDB的意思是指：源PDB位於本地CDB中，在當前CDB內，通過某個PDB（非PDB$SEED種子）來克隆新的可插入數據庫PDB。

整體過程為：

1）設置相關參數或子句

2）連接到根後將被克隆的PDB置為只讀模式

3）開始克隆

4）打開被克隆的PDB和新的PDB


2、設置相關參數

這裡可以設置 OMF 參數 DB_CREATE_FILE_DEST='具體路徑';

或者設置 PDB_FILE_NAME_CONVERT='本地PDB路徑','新的PBD路徑';

或者在創建語句中使用 FILE_NAME_CONVERT=('本地PDB路徑','新的PDB路徑'); 子句

或者在創建語句中使用 CREATE_FILE_DEST='新的PDB路徑';子句

--以設置DB_CREATE_FILE_DEST='具體路徑'; 為例!mkdir /oracle/oradata/SRM/pdb4show parameter db_create_file_dest;alter system set db_create_file_dest='/oracle/oradata/SRM/pdb4';



3、設置被克隆的PDB為只讀模式

show pdbs;alter pluggable database pdb close immediate; alter pluggable database pdb open read only;


4、克隆新的PDB

從PDB1克隆PDB4

create pluggable database pdb4 from pdb;show pdbs



5、打開新的PDB

alter pluggable database pdb4 open read write;alter pluggable database pdb close;alter pluggable database pdb open read write;show pdbs
