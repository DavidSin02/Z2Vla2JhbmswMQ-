高中化學離子反應

一、離子反應與離子方程式


1．離子反應

(1)電解質在溶液中反應的實驗探究

按下圖所示完成實驗，並回答問題：

①圖甲實驗的現象是無明顯變化，因為溶液K＋、Cl－、Na＋、SO42—相互之間不發生化學反應。

②圖乙實驗的現象是有白色沉澱生成，其原因是溶液中Ba2＋與SO42—結合生成硫酸鋇。

③結論：電解質在溶液中的反應實質是離子之間的反應。

(2)離子反應：有離子參加或有離子生成的化學反應。


2．離子反應方程式

(1)離子反應常用離子方程式來表示。離子方程式是用實際參加反應的離子符號來表示反應的式子。

(2)離子方程式的書寫步驟

以氯化鋇和硫酸鈉在溶液中反應為例：

①寫：化學方程式(必須根據實驗事實)

BaCl2＋Na2SO4===BaSO4↓＋2NaCl；

②拆：把易溶於水、易電離的物質拆成離子的形式

2Na＋＋SO42—＋Ba2＋＋2Cl－===BaSO4↓＋2Na＋＋2Cl－；

③刪：刪去方程式兩邊不參加反應的離子

SO42—＋Ba2＋===BaSO4↓；

④查：檢查方程式兩邊各元素的原子個數和電荷總數是否相等、反應條件、沉澱符號、氣體符號等。

(3)書寫下列反應的離子方程式：

①NaOH溶液與稀硫酸：OH－＋H＋===H2O；

②KOH溶液與稀鹽酸：OH－＋H＋===H2O；

③BaCl2溶液與稀硫酸溶液：Ba2＋＋SO42—===BaSO4↓；

④Ba(NO3)2溶液與硫酸鈉溶液：Ba2＋＋SO42—===BaSO4↓；

⑤碳酸鈉與稀硫酸：CO32—＋2H＋===CO2↑＋H2O

⑥碳酸鈣與稀鹽酸：CaCO3＋2H＋===Ca2＋＋CO2↑＋H2O。


【歸納總結】

(1)書寫步驟：

“寫(基礎)→拆(關鍵)→刪(途徑)→查(保證)”。

(2)書寫關鍵：

①要將易溶於水、易電離的電解質寫成離子的形式；

②單質、沉澱、氣體、難電離的物質(如水、弱酸、弱鹼)等寫成化學式。


【深度思考】

(1)以H＋＋OH－===H2O為例說明離子方程式表示的意義是什麼？

提示　①表示某一個具體的化學反應，如NaOH＋HCl===NaCl＋H2O。

②表示同一類反應，如：強酸與可溶性強鹼生成水和可溶性鹽的反應。

(2)①酸鹼中和反應的離子方程式都是H＋＋OH－===H2O嗎？

②H＋＋OH－===H2O僅表示酸鹼中和反應嗎？

提示　①硫酸與氫氧化鋇溶液反應的離子方程式為：2H＋＋SO42—＋Ba2＋＋2OH－===BaSO4↓＋2H2O。

②硫酸氫鈉(NaHSO4)溶液與氫氧化鈉溶液反應的離子方程式為：H＋＋OH－===H2O。

(3)判斷下列反應的離子方程式正誤。

①稀硫酸滴在銅片上：Cu＋2H＋===Cu2＋＋H2↑(　　)

②稀硫酸與氫氧化鋇溶液混合：SO42—＋Ba2＋===BaSO4↓(　　)

③稀硝酸滴在大理石上：CaCO3＋2H＋===Ca2＋＋H2CO3(　　)

④氧化鐵與稀鹽酸混合：Fe2O3＋6H＋===2Fe3＋＋3H2O(　　)

⑤氫氧化銅和稀鹽酸反應：OH－＋H＋===H2O(　　)

⑥鎂和硝酸銀溶液反應：Mg＋Ag＋===Mg2＋＋Ag(　　)

提示　①×　②×　③×　④√　⑤×　⑥×

解析　①Cu的活潑性小於H2的活潑性，Cu不能和稀硫酸反應生成H2，故①錯誤；②稀H2SO4和Ba(OH)2溶液混合，除SO42—和Ba2＋反應，還有H＋和OH－反應，離子方程式為：2H＋＋SO42—＋Ba2＋＋2OH－===BaSO4↓＋2H2O，故②錯誤；③反應生成的H2CO3不穩定，分解成CO2和水，故③錯誤；④Fe2O3為氧化物，在離子方程式中不能拆分為離子，因此④正確；⑤Cu(OH)2在離子反應中，不能拆分為離子，應為Cu(OH)2＋2H＋===Cu2＋＋2H2O；⑥中離子反應，電荷不守恆，應為Mg＋2Ag＋===2Ag＋Mg2＋。


【歸納總結】

離子方程式正誤判斷——“五看”

(1)看是否符合客觀事實。

(2)看拆分是否正確。

(3)看是否符合原子守恆和電荷守恆。

(4)看是否漏寫離子反應。

(5)看是否符合題設條件。


二、離子反應發生的條件


1．實驗探究

(1)上述三個實驗的現象：①生成藍色沉澱；

②溶液紅色消失；③產生氣泡。

(2)寫出上述三個實驗中反應的離子方程式：

①Cu2＋＋2OH－===Cu(OH)2↓；

②H＋＋OH－===H2O；

③CO32—＋2H＋===CO2↑＋H2O。


2．複分解型離子反應發生的條件


3．離子共存

(1)幾種離子在溶液中能否大量共存，取決於它們之間是否發生反應。

(2)在溶液中若離子之間不發生反應，則能大量共存；否則不能大量共存。

(3)溶液中離子不能大量共存的判斷

①離子間反應生成沉澱：如Ca2＋與CO32—、Ag＋與Cl－、Mg2＋與OH－等。

②離子間反應生成氣體：如H＋與CO32—、H＋與HCO3—等。

③離子間反應生成難電離的物質：如H＋與OH－、H＋與CH3COO－、OH－與NH4+等。


【深度思考】

(1)離子反應有複分解反應和置換反應等多種類型。判斷下列置換型離子反應能否發生，若能，寫出相應的離子方程式。

①Zn和稀硫酸：_______________________________________________________________。

②Cu和稀鹽酸：________________________________________________________________。

③Fe和硫酸銅溶液：____________________________________________________________。

提示　(1)①能；Zn＋2H＋===Zn2＋＋H2↑

②否　③能；Fe＋Cu2＋===Fe2＋＋Cu


(2)判斷正誤

①在強酸性溶液中，CH3COO－可以大量共存(　　)

②含大量Ca2＋的溶液中，SO42—可以大量共存(　　)

③在溶液中與Ag＋能共存的常見陰離子只有NO3—(　　)

④無色透明溶液中，一定不存在Cu2＋、Fe3＋、MnO4—等(　　)

⑤強酸性溶液中，一定不能大量存在CO32—、HCO3—、S2－、OH－等(　　)

⑥滴入酚酞溶液顯紅色的溶液中K＋、Na＋、Cu2＋、SO42—不能大量共存(　　)
