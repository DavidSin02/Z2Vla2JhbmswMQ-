初中化學：置換反應化學式總結

2020-10-29
| 科技

金屬單質 + 酸 ---鹽 + 氫氣

1、鋅和稀硫酸反應：Zn + H2SO4 === ZnSO4 + H2↑

2、鎂和稀硫酸反應：Mg + H2SO4 === MgSO4 + H2↑

3、鋁和稀硫酸反應：2Al + 3H2SO4 === Al2(SO4)3 + 3H2↑

4、鋅和稀鹽酸反應：Zn + 2HCl === ZnCl2 + H2↑

5、鎂和稀鹽酸反應：Mg+ 2HCl === MgCl2 + H2↑

6、鋁和稀鹽酸反應：2Al + 6HCl === 2AlCl3 + 3H2↑

1－6的現象：有氣泡產生。

7、鐵和稀鹽酸反應：Fe + 2HCl === FeCl2 + H2↑

8、鐵和稀硫酸反應：Fe + H2SO4 === FeSO4 + H2↑

7－8的現象：有氣泡產生，溶液由無色變成淺綠色。



金屬單質 + 鹽—另一種金屬 + 另一種鹽

9、鐵與硫酸銅反應：Fe+CuSO4 == Cu+FeSO4

現象：鐵條表面覆蓋一層紅色的物質，溶液由藍色變成淺綠色。

(古代溼法制銅及"曾青得鐵則化銅"指的是此反應)

10、鋅片放入硫酸銅溶液中：CuSO4 + Zn == ZnSO4 + Cu

現象：鋅片表面覆蓋一層紅色的物質，溶液由藍色變成無色。

11、銅片放入硝酸銀溶液中：2AgNO3+Cu==Cu(NO3)2+2Ag

現象：銅片表面覆蓋一層銀白色的物質，溶液由無色變成藍色。金屬氧化物＋木炭或氫氣→金屬＋二氧化碳或水

12、焦炭還原氧化鐵：3C+ 2Fe2O3 ==4Fe + 3CO2↑冶煉金屬

13、木炭還原氧化銅：C+ 2CuO==2Cu + CO2↑冶煉金屬

現象：黑色粉未變成紅色，澄清石灰水變渾濁。

14、氫氣還原氧化銅：H2 + CuO==Cu + H2O

現象：黑色粉末變成紅色，試管內壁有水珠生成。

15、氫氣與氧化鐵反應：Fe2O3+3H2==2Fe+3H2O
