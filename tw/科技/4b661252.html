dos快捷鍵教程，windows控制檯程序，cmd命令行窗口命令快捷鍵

2020-10-29
| 科技

大家好，我是老蓋，首先感謝觀看本文，本篇文章做的有視頻，視頻講述的比較詳細，也可以看我發佈的視頻。

今天我們學習dos裡邊比較常用的幾個快捷鍵，F3 F7 ↑↓ 鍵 Tab 鍵 Ctrl + C。

先介紹一下F7這個命令，F7能調出以往輸入的命令，我們先隨便輸入幾個命令dir cls cd..，然後按f7，就會出現以前輸入的命令菜單，就可以方便的根據菜單中的命令直接選擇後回車運行。

F3，這個快捷鍵是調出上一個輸入的命令。

上下箭頭 ↑↓ 鍵快捷鍵，可以上下調出以前的命令。

我們再看一下ctrl + c這個快捷鍵，它可以中斷運行的DOS命令，比如我用ping 192.168.1.1 -t，它會一直在執行這個命令，如果想中斷的話，可以使用ctrl + c 就可以中斷這個命令。

在介紹一下tab這個快捷鍵，它可以補全單詞，在當前目錄中有個文件夾叫Administrator。

我輸入cd admin，然後按tab鍵，它會自動補全這個單詞英文文件夾。

再舉一個tab自動切換補全例子，比如當前目錄有三個文件夾。

PerfLogs

Program Files

Program Files (x86)

我輸入cd p。

按一個tab鍵，第一次會自動匹配第一個PerfLogs文件夾。

再次按tab會自動匹配第二個Program Files文件夾。

第三次按tab鍵會匹配第三個Program Files (x86)文件夾。

我們再按一下f7，這樣能看到剛才輸入的幾個命令全部都在菜單中方便使用。

今天學的這幾個dos快捷鍵我們經常會用到，大家可以多練習一下。
