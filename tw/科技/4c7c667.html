「工程師筆記」你的鑽孔表格，就讓我來守護

2020-10-29
| 科技

任何pcb設計文件設計完成後，都需要通過相關功能輸出生成所需要的文件資料，allegro軟件的功能確保了pcb的設計數據與成產文件完全一致，從而可以直接在設計界面中輸出所有pcb生產、測試、裝配所需的文件，以下要講的是pcb鑽孔表格的設置與生成。

1.首先是執行鑽孔符號的優化命令，

執行菜單命令Manufacture→Nc→Dill Customization，如圖所示。

2.彈出鑽孔符號優化對話框，其中包含項目中出現的所有通孔信息參數，如圖所示。

【#】表示通孔的編號

【Type】表示通孔的形狀

【Size X、Size Y】表示通孔的尺寸

【+/-Tolerance】表示孔徑的精度

【Symbol Figure】表示通孔鑽孔的圖標形狀

【Symbol Characters】表示通孔鑽孔的符號

【Symbol Size X、Symbol Size Y】表示鑽孔符號的尺寸

【Plating】表示通孔是否金屬化

【Quantity】表示通孔的數量

3.鑽孔符號優化的要求：

（1）不同鑽孔尺寸符號和尺寸不同

（2）相同鑽孔尺寸符號不能對應不同發熱鑽孔符號

（3）金屬化孔與非金屬化孔尺寸相同時，鑽孔符號不能合併

4.單擊2圖中的Auto generate symbol按鈕，自動生成所有鑽孔的鑽孔符號及符號大小。

5.為了減少生產時和核對鑽孔參數的數量，需將同一孔徑的過孔合併，使其使用同一個鑽孔符號。將同一孔徑同一形狀的金屬化鑽孔選擇同一種Symbol Figure和Symbol Characters，並將Symbol Size X、Symbol Size Y設置為相同的數值，再單擊Merge按鈕，將原來的兩項合併一項。

6.所有的鑽孔符號合併完成後，再單擊下圖按鈕，重新生成所有鑽孔符號

7.當優化完畢後，單擊ok按鈕保存，鑽孔符號優化完成。

8.完成符號優化後要將鑽孔表格在pcb編輯界面中提取出來，放在pcb板框外。

9.執行菜單欄命令Manufacture→Nc→Drill Legend，如圖。

10.單擊Drill Legend命令後出現下面對話框，候選相應的選項，如圖。

Template file：鑽孔表格的模板文件，默認使用軟件自帶的格式default_mil.dlt

Drill legend title：鑽孔表格表頭的名稱，默認即可

Output until：默認選擇與pcb設計文件一致的單位

Hole sorting method：鑽孔表格中鑽孔符號的排列順序

Legend：一般默認選項即可，如果pcb要使用背鑽技術，則勾選include backdrill複選框

11.設置完成後，單擊ok，鑽孔表格附在光標上，在pcb板框外合適的位置單擊鼠標左鍵，將鑽孔表放置出出來，當再次提取鑽孔表格時，默認表格放置在上一次的位置上，如果需要移動鑽孔表格，可用move命令，且在find面板中勾選groups，再選擇鑽孔表格進行移動。

在生成鑽孔文件時需要注意在鑽孔符號優化時要相同孔徑相同形狀且電鍍的選擇也要一樣才能進行合併，在進行符號提取時，output unit的單位要與pcb設計文件的單位一致。
