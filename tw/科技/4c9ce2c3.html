SUM/SUMIF/SUMIFS函數多條件求和教程、使用技巧及常見出錯問題

2020-10-29
| 科技

在日常工作中，經常需要運用Excel對一些數值進行求和、對滿足單個條件的值相加求和以及對滿足多個條件的值相加求和，而大家往往會混淆對這些函數的用法:SUM/SUMIF/SUMIFS,尤其是SUMIF/SUMIFS.現分別對這三個函數的語法、參數及常見問題做講解。

一、SUM函數

SUM函數是一個常見的數學及三角函數，可將值相加。你可以將單個值、單元格引用或是區域相加，或者將三者的組合相加。

1.1 SUM函數語法及參數

SUM(number1,[number2],...)

Number1, Number2，Number2………分別求和的數值

1.2實例講解

1:求出每個學生的總成績

方法一:直接在彙總單元格中輸入=SUM(C2:E2),然後按回車，之後下來填充。

方法二(快捷鍵):選中彙總單元格即選中總成績這一列(F2:F7),按下組合鍵："ALT+="，就可以求出所有學生的總成績了。

1.2常見問題彙總

常見問題1：SUM函數顯示的是 #####，而不是結果。

出錯原因:檢查列，##### 通常表示列太窄，無法顯示公式的結果。

常見問題2：SUM 函數以文本形式顯示公式本身，而不是結果。

出錯原因:檢查確定單元格未設置為文本格式。選擇出問題的單元格或區域並使用 Ctrl+1 調出“設置單元格格式”對話框，然後單擊“數字”選項卡，選擇所需的格式。如果單元格設置為文本格式，並且在你更改格式後仍未更改，則可能需要使用 F2 > Enter 強制更改格式。

常見問題3：SUM 函數不會更新。

出錯原因:請確保將“計算”設置為“自動”。在“公式”選項卡上轉到“計算”選項。也可以使用 F9 強制工作表進行計算。

二、SUMIF函數

SUMIF 函數僅對滿足單個條件的值相加求和即對範圍中符合指定條件的值求和

2.1 SUMIF函數語法及參數

SUMIF(range, criteria, [sum_range])

參數1:range必需。根據條件進行計算的單元格的區域。每個區域中的單元格必須是數字或名稱、數組或包含數字的引用,空值和文本值將被忽略

參數2：criteria必需。用於確定對哪些單元格求和的條件，其形式可以為數字、表達式、單元格引用、文本或函數。例如，條件可以表示為 32、">32"、B5、"32"、"蘋果" 或 TODAY()。

參數3:sum_range 可選。要求和的實際單元格（如果要對未在 range 參數中指定的單元格求和）。如果省略 sum_range 參數，Excel 會對在 range 參數中指定的單元格（即應用條件的單元格）求和。

2.3常見問題

使用 SUMIF 函數匹配超過 255 個字符的字符串或字符串 #VALUE! 時，將返回不正確的結果。

三、SUMIFS函數

SUMIFS函數僅對滿足多個條件的值相加求和即對範圍中符合指定條件的值求和

3.1 SUMIFS函數語法及參數

SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)

參數1：Sum_range (必需) 要求和的單元格區域。

參數2：Criteria_range1(必需)使用 Criteria1 測試的區域;Criteria_range1 和 Criteria1 設置用於搜索某個區域是否符合特定條件的搜索對。 一旦在該區域中找到了項，將計算 Sum_range 中的相應值的和。

參數3：Criteria1 (必需)定義將計算 Criteria_range1 中的哪些單元格的和的條件。 例如，可以將條件輸入為 32、">32"、B4、"蘋果" 或 "32"。

Criteria_range2, criteria2, … (可選) 附加的區域及其關聯條件。 最多可以輸入 127 個

3.3常見問題

常見問題1：不顯示預期結果，而是顯示 0（零）。

錯誤原因:如果要測試文本值，如姓名，請確保將 Criteria1,2 用引號引起來。

常見問題2：Sum_range 有值 TRUE 或 FALSE 時，結果不正確。

錯誤原因：Sum_range 的值 TRUE 和 FALSE 的求值方式不同，可能會在將其相加時導致意外結果。Sum_range 中包含 TRUE 的單元格的求值結果為 1。包含 FALSE 的單元格的求值結果為 0（零）。
