最新Linux 除了能夠偽裝成Win 10，還能讓手機變桌上電腦？

2020-10-29
| 科技

Kali Linux官網宣佈了2019年第四個也是最後一個版本——Kali Linux 2019.4。相比之前有較大的變動，增加了很多全新的模式，例如全新的桌面環境Xfce、Kali Undercover模式（能夠偽裝成win10）、NetHunter Kex 。


對Gnome說再見，迎接 Xfce

在最新的版本中， Kali完全捨棄了Gnome桌面而轉向了Xfce。該最新迭代用 Xfce 替換了其主要桌面環境 Gnome。官方認為，Gnome雖然功能很強大，有很多非常棒的特性，但對於大部分Kali用戶而言，要想的只是能夠一次運行多個終端窗口的窗口管理器和一個網頁瀏覽器。再見 Gnome，你好 Xfce

其次，Kali需要支持從低端到高端不同的硬件設備，包括樹莓派。Kali致力於輕量級的解決方案，希望能夠在不同的設備上都儘可能提供一致的使用體驗。基於這些考慮，Gnome則顯得有些臃腫，故而選擇更為輕量的Xfce作為默認桌面環境。

但同時Kali Linux團隊也表示：

“我們對用戶界面的更新感到非常興奮，我們認為您會喜歡它的。但是，由於用戶界面變化巨大，如果您不想離開Gnome，請不要擔心。我們仍然有一個Gnome為您量身打造的產品，已經進行了一些更改。隨著時間的流逝，我們將對所有桌面環境進行更改。”


新的 Xfce 用戶界面 (UI) 是可運行於各層級 Kali 安裝上的輕量級 UI。其功用在於無需變動即可處理普通用戶的各種需求。該界面採用大眾熟悉的標準 UI 概念以確保毫無學習曲線。且其現代 UI 元素充分運用屏幕空間，外觀十分精美。


將Linux 變成成Win 10

官方製作了一個“ Kali Undercover”模式，能夠快速將默認主題切換為類似Windows 10桌面環境，非常逼真，這樣可以放心的“工作”。再次運行腳本就可以重新切換回之前的主題。

該模式主要是為了將Xfce桌面環境轉變為Windows外觀的桌面而設計的，其目的是為那些在公共場所和不想讓人們說他們使用的是Linux發行版，而是Windows操作系統的人們設計。由於多數人經常看到的是 Windows 和 macOS 設備，因此如果在辦公室走廊或其它公共場所看到有人在使用帶有標誌性的龍圖標的 Kali Linux 可能會引起猜疑。


裝進口袋的全功能ARM 臺式機

Kali Linux推出了一個重磅功能：Android手機上完整的Kali桌面——NetHunter Kex
名為 NetHunter Kex 的新功能使用戶可在 HDMI 輸出上連接安卓設備，還可使用藍牙鍵盤和鼠標，在安卓智能手機上獲得 “完整無刪減的 Kali 桌面”。這其實有點類似現在智能手機上內置的桌面模式，提供輕量級的電腦工作模式。

Kali Linux 團隊稱：只要手機性能夠強大，這就好像一臺恰好能裝進口袋的全功能便攜式 ARM 臺式機。


專注在“容錯”的BTRFS

Kali Linux 文檔中新增將 BTRFS 用作根文件系統的內容，賦予用戶在升級後回滾文件系統的能力。Kali Linux允許用戶在安裝過程中將Btrfs文件系統用於根分區。

BTRFS 是個寫入時複製 (CoW) 的文件系統，旨在解決 Linux 文件系統中缺乏緩衝、快照、校驗和及集成多設備跨越的問題。各大公司均對其代碼庫有所貢獻，包括 Facebook、Red Hat、Oracle SUSE 等。

在虛擬機裡嘗試新東西的時候，往往會做個快照，萬一問題時可以簡單回滾到已知良好狀態。但在裸機上運行 Kali 就沒那麼容易做到這一點了。你要麼超級小心謹慎，要麼一旦出問題就陷入無盡的手動清理中。但有了 BTRFS，即便在裸機安裝上都可享有同樣的快照功能！

可安裝Powershell

PowerShell 是跨平臺自動化與配置工具，優化處理結構化數據（如 JSON、CSV、XML 等）、REST API 和對象模型，包含一個命令行 shell、一個關聯腳本語言和一個處理 cmdlet 的框架。Kali 官方 git 庫已於 2019 年 6 月置於 GitLab 上。

在今年11月份的更新中，powershell已添加到主存儲庫中，現在只需要運行：
apt update && apt -y install powershell
就能讓你的系統擁有強大的PowerShell。


​

Kali Linux是一個高級滲透測試和安全審計Linux發行版。作為使用者，你可以把它理解為一個特殊的Linux發行版，集成了精心挑選的滲透測試和安全審計的工具，供滲透測試和安全設計人員使用，也可稱之為平臺或者框架。

Kali Linux自帶安全工具集，它將所帶的工具集劃分為十四個大類，這些大類中，很多工具會重複出現，因為這些工具同時具有多種功能，比如nmap既能作為信息蒐集工具也能作為漏洞探測工具。另外，除了這些系統推薦的工具，我們也可以自行添加新的工具源，豐富工具集。

Kali linux可以說是網絡安全人員的專用系統，它是BackTrack系列的升級版。

聽過一個笑話：當別人問你為什麼要使用Linux時，一個很好的理由是當有人要求你去他家裡修理電腦並且你不知道問題的原因時，這時候你就藉口說你不使用Windows。你不瞭解Windows而無法修復它。
