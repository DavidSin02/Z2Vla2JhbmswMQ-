1.CLO 3D

1.1關於CLO的歷史

CLO 軟件的雛形源於一位對面料模擬技術充滿熱情的計算機博士Jaden。他在校期間開發了這款軟件的第一個原型。當看到學生們都開心地通過該程序製作3D服裝時，Jaden決定將其公開。


軟件公開後，在最早一批軟件使用者中，有一位cosplay藝術家敏銳地捕捉到了這款軟件的潛力，認為這款軟件會給虛擬cosplay藝術帶來巨大的改變，並建議為此建立一個社群。很快，這款軟件便從這個由超過200萬虛擬cosplay藝術家的社群中收穫到了巨大的肯定。2009年，針對這個群體的軟件應運而生，這就是第一代的Marvelous Designer。隨後在2011年，由Marvelous Designer製作了第一部電影《丁丁歷險記》。


在CG和遊戲行業不斷髮展的同時，這家公司也並沒有忘記自己最初的目標——開發一個專門針對服裝行業的程序。2010年，在結合了無數設計師和打版師的意見之後，第一代CLO正式發佈。CLO的開發是為了更加精準地實現虛擬服裝在外觀和功能上與物理服裝的無縫轉換。


經過10年的發展，這兩個程序都越來越成熟，Marvelous Designer已經更新到了版本8.0，而CLO也更新到了版本5.0。如今CLO更是推出了CLO-SET和Benefit by CLO這兩個平臺，來進一步完善虛擬服裝這一市場。


1.2 CLO的功能

說到CLO和Marvelous Designer的區別，CLO如是答道：“雖然兩者的核心技術相同，但Marvelous Designer主要用於CG、動畫和遊戲行業。而CLO更傾向於時尚和服裝行業，它可以應用於服裝設計開發過程的各個階段。這兩者的主要區別在於CLO導入和導出的DXF-AAMA或DXF-ASTM 文件可在生產時發送給生產廠商。”


AAMA和ASTM其實都是DXF文件，只不過是適配不同軟件的DXF文件。我們也許知道，只要是DXF文件就可以導入修改。我最開始接觸的打版軟件是PAD，當時並不知道如何在CLO中打版，在這兩個軟件中不斷使用切換的過程中我才發現，原來DXF文檔可以直接在適配軟件中通用，至少和PAD的轉換中是非常成功的，其他的打版軟件我沒有實戰經歷，但應該也問題不大。


CLO的一大特點就是可以流暢地完成3D打版和2D打版之間的轉換。設計師，打版師既可以導入已有的DXF文檔對其進行修改，也可以直接在CLO中從零起草新的平面版型，還可以在虛擬模特上完成立體裁剪，而3D和2D的同步呈現也使得整個過程更為高效。如果真的要從操作上來講，我還是更喜歡PAD，所以一開始有點不願意放棄，很執著地使用PAD。後來學習CLO打版之後發現，雖然從打版上來說有一點點遜色（也可能是我個人的使用習慣造成的），但是它能2D3D實時同步這一點真的很方便。


CLO的特點包括無限設計，實時呈現，更加精準的屬性以及更加簡便的流程。


2.Style 3D

Style 3D是浙江凌迪數字科技有限公司（原上海凌笛數碼科技有限公司）開發的數字化服務平臺，可以連接面料供應商、設計部門和合作客戶。通過面料數字化、服裝建模和遠程推款，實現快速開發打樣和審版。

圖片來源：Style 3D微信公眾號


1.面料數字化


下面介紹三種獲得面料貼圖的方法：


Step1:

選擇合適大小的面料樣品，保證面料掃描之後的花型紋理。


Step2：

A.使用專業的面料掃描儀，可以完整獲得面料的紋理貼圖和法線貼圖，效果逼真，但儀器價格昂貴，不易養護（參考官方面料掃描介紹）。

B.使用日常辦公的彩色平板掃描儀（要求掃描精度不低於300dpi）獲得面料的表面紋理，成本低，但面料仿真效果不好（參考官方面料掃描介紹）。

C.手機拍照，獲得面料表面紋理。使用方便，但效果差，容易產生色差且面料絲縷方向不正，有一定透明度的面料無法塑造透明效果；


這個掃描過程很快，整體不會花很多時間，上傳平臺後大家就可以一起使用了。在第二步，我們公司主要用A方法，前幾天我去我們公司設計部體驗過面料掃描儀，掃描出來的效果的確清晰，但在掃描的時候，面料不能拉伸，容易造成紋理變形，絲縷不正。


而C方法手機拍照，在我之前做作業時，就遇到手邊剛好有一塊麵料可以用於我正在做的模型，就自己用手機拍照試了一下。用這種方法，如果把方向找對，紋理拍清楚，效果也是不錯的。


2.服裝建模


服裝建模主要分為三個步驟：

Step1：在傳統平面製版軟件中繪製紙樣，導出DXF格式文件，紙樣導出時去除縫頭，只留淨紙樣。Style 3D不能直接打版，需要先用其他打版軟件做好，然後導出DXF。導出的時候一定要去掉縫頭。


Step2：在Style 3D建模軟件中打開DXF格式文件，選擇與待建模服裝尺寸匹配的模特，縫製紙樣，通過模擬狀態下的拉扯調整褶皺效果，然後添加面料、印花圖案、明線、鈕釦、拉鍊等。添加縫線後，為了拉出符合實際效果的褶皺，對模型做各種拉扯的過程很花時間。在這一步，我個人經驗是用好軟件的冷凍、硬化、失效還有固定針等功能。


部件模型做好以後，如果還要修改其他地方，一定要先冷凍再繼續模擬，不然後果很崩潰。譬如我們剛開始學習軟件時，做的一件戴帽衛衣，當時用了固定針做帽子的造型。剛開始我們經常會忘記冷凍，忘記退出模擬，就把固定針取了，結果就是好不容易做好的帽子造型全塌。


Step3 保存文件，並上傳至Style 3D網絡平臺，備註好內部編號、服裝分類、創建人等信息。


注：這裡也可以根據設計部門的人員職能，選擇在Style 3D中製作白坯上傳至雲端，設計者或客人從網絡平臺進入，添加面料、印花圖案等細節。

3.遠程推款

遠程推款可以根據客人是否有平臺賬號來選擇不同方法：


A.客人有Style 3D網絡平臺賬號，可直接進入平臺查看建模，並自行改變顏色、面料、印花等，查看不同的效果。查看後若有問題可直接在平臺上留言，提出修改要求，由建模人員線下修改後再上傳。


B.客人沒有Style 3D網絡平臺賬號，可由建模人員做好模型以後，導出不同角度的圖片或者gif格式的動圖，發送給客人查看。動圖會360度旋轉，旋轉速度在導出時可以自己調整。

二、3D軟件帶來的便利

1.新品開發

傳統的新品開發，在大貨開裁之前需要經過頭樣到船樣的多個打樣修改的環節。中間涉及面輔料、印繡花的跟進，會出現某個環節跟不上的情況。外貿企業打樣中須向客人寄樣衣，等待客人收到樣衣後給出修改意見，耗時久。


對於印花，在模擬時因為可以調整大小和位置，所以可以幫助用戶更好地判斷自己想要的比例。對於刺繡類更加昂貴的工藝，這樣的模擬更可以減少因不確定而引發的浪費。我們部門打樣的服裝有很多印繡花，經常聽到業務員吐槽印花的花型開版費用，而且有時候印繡花還會出問題，譬如可能顏色印錯、方向印錯，從而影響之後的打樣進度。前一段時間我們部門接了一個幾百件的小訂單，花片裁下以後送去印花，印回來發現方向不對，跟單要求全部再裁，重印。但是這些問題，使用軟件都可以避免，因為在軟件裡我們就可以改變印花的效果。

圖片來源：莊主使用課程資料自制


以上三張圖的圖案效果從上至下，分別是消光、絲綢和金屬。


相比傳統樣品開發，3D軟件速度更快，成本更低。紙樣做好後，不需等待面輔料，不需裁剪，就可直接在虛擬模特上模擬。在紙樣和數字化面輔料完成的情況下，簡單款式半小時甚至十幾分鍾就可以完成；複雜樣衣（例如羽絨服）熟練工半天可完成，新手兩天左右。建模完成後不需寄快件，直接向客人發送3D模型，客人和設計師、版師可實現實時溝通，有問題直接修改，不需多次做修改樣，省時省力。

2.線上批版（有Style 3D的賬號）

上面提到過Style 3D將製作好的模型上傳後，客人可以根據自己的喜好，在面料庫和圖案庫裡下載喜歡的面料或圖案，然後填充在模型上做並查看效果。並且可以直接在平臺上做查看後的留言，對局部不滿意，也可以添加熱點後留言。對比傳統的寄樣衣，等修改意見，互動的速度加快很多。


3.一版多用

我們在做樣品開發時，經常會遇到一個款式需要做不同顏色、不同面料的多個樣衣，供客人選擇。我目前有三個大批推銷樣的單子，都是一套紙樣，做多個顏色的樣衣，每個顏色的數量幾件到十幾件不等（同尺碼）。而使用3D數字樣衣，只需要建模一次，在做添加面料時，修改顏色，即可看到不同顏色的效果。這裡還可以添加熱點，直接指出具體哪個地方有什麼問題，做什麼修改。（點擊3D樣衣建模查看）


這個鏈接是在Style3D的微信公眾號找到的。通過這個鏈接，大家可以體驗一下在同一個版型上進行不同的面料切換。


除了製作整件樣衣，3D軟件也可以把一件衣服拆分成領、袖、大身等幾個部分，做部件的建模。將這些部件拼合在一起，就是一件完整的樣衣。

1.虛擬模特

1.1 Style 3D

目前根據我自己的使用體驗來看，Style 3D的虛擬模特還存在以下一些亟待改善的問題：

A.軟件自帶模特數量有限，只有女165、男175、130男女大童和90男女小童，若沒有定製新模特，所能做的數字樣衣尺碼有限。對於大童的樣衣，在最初打樣時，我們公司按照140的尺碼製版做樣衣，做3D建模時，打樣的尺碼和模特尺寸不符，建模就沒有用。


B.軟件自帶模特尺寸與實際人體尺寸有差異。在最初我用公司自己的樣板做模型時，發現上衣的袖子袖口和褲子腳踝處總會堆積很多衣褶，之後點開虛擬模特的尺寸詳情，臂長等一些長度尺寸對比實際人體尺寸都短几公分，造成建模效果差。


關於這個問題，剛開始的時候，我以為是我們尺寸的問題。後來專門上天貓找同類型的服裝尺寸，對比的結果是，模特尺寸與實際人體尺寸存在差異。針對以上兩個問題，我們曾諮詢過凌迪的老師，得到的回覆是，要麼按照軟件已有模特尺寸製版，要麼出錢做新的虛擬模特。但虛擬模特的建模成本高，據說3000元一個。Style 3D的網絡平臺上也有人臺庫， 但目前官方人臺庫的模特無法查看詳細的尺寸信息。


對於以上兩個問題，我們公司目前的應對辦法如下：


A.大童樣衣打樣時按照140打樣，然後推碼，得出130碼的紙樣，然後再做數字樣衣；

B.明確做數字化樣衣的目的是追求3D模型的效果，在不改變結構的情況下，可以調整紙樣尺寸，保證數字樣衣的穿著效果。


如此一來，我們所看到的數字樣衣並不是尺寸正確、符合客人標準的樣衣。客人看中樣衣下單後，還需調整紙樣至符合目標消費人群的尺寸。個人覺得，這種做法不僅未達到3D建模改變傳統生產流程、加快生產速度的目的，反而多增加了一個環節，給自己增加工作量。


1.2 CLO的虛擬模特

而上述提到的問題可以在CLO軟件中很好的避免。

在CLO中，用戶可以根據自己的需求調整虛擬模特，下圖是來自於CLO編輯虛擬模特的截圖。我們可以看到這上面的參數非常詳細，基本上製衣需要的數據都有。不過CLO最開始默認的模特設置和真人不太符合，除了自己調整之外，CLO SET裡也有更接近現實比例的人臺出售，價格是30美金一個。

圖片來源：莊主自制


CLO有一個很有用的功能是壓力模擬，當一個版製作好穿在虛擬模特身上之後，該功能會提醒用戶哪些地方太緊或者根本不能穿。


下面是我之前學習3D立體裁剪時做的一個小練習：當這件緊身小背心選擇較硬的牛仔面料時，大家可以看到紅色部分是“不能穿”需要調整的部分，以及黃色部分為“過緊部分”。我做的一個簡單調整是把面料換成了針織的，可以看到換了面料之後，因為針織面料的彈性，不可穿部分也隨機消失。小範圍是可以用這種方法調整模特尺寸，但調整數值太大會造成模特走形。但從我知道的例子來看，在CLO上修改尺寸後，暫時還沒遇見過這樣的問題。CLO上還可以直接改變面料類型，改變面料後壓力也隨之改變。

2.面輔料的仿真效果

2.1 Style 3D的數字面料

面料的掃描結果是一張圖片（紋理貼圖和法線貼圖），格式要求jpg、png或jpeg。給數字樣衣添加面料，其實就是把圖片附到紙樣表面。因此，在做模型時，軟件裡的版片是沒有模擬面料性能的。雖然有經、緯向強度等參數調節，但這些參數的調節依賴個人對面料性能的掌握。


我們公司主要做針織類服裝，休閒類還好，但合體的服裝，例如貼體的內衣，在打樣時是不收省的，依靠針織面料的自身彈性達到上身的合體效果。因此，在軟件中建模時，如果用未經修改的紙樣，袖窿處就會有多餘的量，與實際樣衣不符。

圖片來源：莊主自制


這件吊帶是我之前做的模型，原樣是針織的面料，緊身型，穿在身上的時候，因為針織面料的彈性，胸部會很服帖。但在3D軟件中，如果沒有修改紙樣，是達不到貼體的效果的。針對這個問題，做樣板的時候注意放量即可。


在軟件中，面料除了紋理貼圖，還有法線貼圖。法線的作用是加深面料的模擬紋路，使得模擬效果更逼真。但是法線貼圖目前只能用專業的面料掃描儀才能獲得，我們公司用的儀器是愛普生。

3.虛擬市場和現實的結合

在CLO的例子中，它一直在完善著自己的虛擬市場，雖然呈現的效果和未來潛力都令人激動，不過目前不管是CLO SET還是Benefit by CLO都還有待更多的商品進行填充。目前CLO SET能買到的虛擬面料，雖然從參數和功能效果上也一定是符合現實的，但如果真的要使用，卻很難獲得百分百相同的面料。而購買來的面料，也會因為缺乏具體的參數信息，從而導致可能無法完全在軟件中重現。如果能打開虛擬和現實面料市場的通道，對於用戶來說是極大的福利。


4.關於學習

最後想分享的這點雖然算不上面臨的問題，但也是客觀存在的事實。雖然3D軟件的操作已經儘量簡化，在一些細微的調整上是對所有人都很友好的，但是要從零開始製作出好的設計和版型，也還是需要非常強的打版知識儲備，以及對於電腦知識的運用。


就CLO的例子來講，如果想要開發設計一個全新的無限接近實物的樣品，整個過程也是需要投入大量的時間（但卻是非常節約材料和空間），對電腦配置也是有相應的要求。另外，雖然在CLO中可以模擬出很多輔料，但它也不是什麼都可以做，比如一些特別獨特的輔料細節也需要運用到其他軟件來配合。此外，style3D目前不支持A卡，想要學習的朋友們請注意。

一、服裝行業現有3D軟件概況介紹

1.CLO 3D

（1）關於CLO的歷史

Clo 軟件的雛形源自於一位對面料模擬技術充滿熱情的計算機博士Jaden。

（2） CLO的功能

無限設計，實時呈現，更加精準的屬性以及更加簡便的流程。

2.Style 3D

Style 3D是浙江凌迪數字科技有限公司開發的數字化數字化服務平臺，連接面料供應商、設計部門和合作客戶，通過面料數字化、服裝建模和遠程推款，實現快速開發打樣和審版。


二、3D軟件帶來的便利

1.新品開發

相比傳統樣品開發，3D軟件速度更快，成本更低。紙樣做好後，不需等待面輔料，不需裁剪，可直接在虛擬模特上模擬。

2.線上批版

Style 3D將製作好的模型上傳以後，客人可以根據自己的喜好在面料庫和圖案庫裡下載喜歡的面料或圖案在模型上做填充。

3.一版多用

使用3D數字樣衣，只需要建模一次，在做添加面料時，修改顏色，即可看到不同顏色的效果。

4. CLO相關實例

（1）CLO SET

CLO SET是一個實現全供應鏈溝通的平臺，全面連接產業鏈中各種角色，用戶可以在上面完成數字採購，同時在服務器上生成3D圖像。

（2） Benefit by CLO

Benefit by CLO 是一個在線的虛擬試衣平臺。


三、3D在現階段的應用中存在什麼問題

1.虛擬模特

（1）Style 3D的侷限

A.軟件自帶模特數量有限，只有女165、男175、130男女大童和90男女小童。

B.軟件自帶模特尺寸與實際人體尺寸有差異。

（2）CLO 3D

在CLO中，用戶可以根據自己的需求調整虛擬模特，除了自己調整之外，CLO SET裡也有更接近現實比例的人臺出售，價格是30美金一個。

（1）Style 3D數字面料

面料的掃描結果是一張圖片（紋理貼圖和法線貼圖），格式要求jpg、png或jpeg，給數字樣衣添加面料，其實就是把圖片附到版片表面。在軟件中，面料除了紋理貼圖，還有法線貼圖。法線的作用是加深面料的模擬紋路，使得模擬效果更逼真。
