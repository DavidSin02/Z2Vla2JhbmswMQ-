IntelliJ IDEA 常用快捷鍵講解（Win + Linux）

2020-11-14
| 科技

說明

IntelliJ IDEA 的便捷操作性，快捷鍵的功勞佔了一大半，對於各個快捷鍵組合請認真對待。IntelliJ IDEA 本身的設計思維是提倡鍵盤優先於鼠標的，所以各種快捷鍵組合層出不窮，對於快捷鍵設置也有各種支持，對於其他 IDE 的快捷鍵組合也有預設模板進行支持。

關於各個快捷鍵的頻率分類上可能每個人都有各自的看法，下面的整理也只是以我個人的使用習慣來劃分的，而我應該是可以代表某一部分小眾人員。但是我個人還是強烈建議你可以在我的基礎上整理一份屬於你的快捷鍵目錄（刪除掉多餘的字眼，只保留快捷鍵內容），本篇文章也只是起到一個工具和引子的作用。

對於下面各個快捷鍵的使介紹描述也許用我個人語言翻譯起來不夠準確或是不全面，且在不同的文件類型上按出來的效果也可能結果不太一樣,所以 強烈建議 你自己把各個快捷鍵都親自操作下體會下各個快捷鍵的實際用法。

前提

由於文化的不同，我們使用的電腦必備一個軟件就是中文輸入法，而目前大多數人都使用搜狗拼音輸入法或是其他類似的。而這些輸入法跟 IntelliJ IDEA 有一個萬惡的衝突永恆不變：快捷鍵衝突。所以為了配合 IntelliJ IDEA，我們要去掉這些輸入法下的所有快捷鍵。

Ctrl

快捷鍵

介紹

Ctrl + F

在當前文件進行文本查找 （必備）

Ctrl + R</kdb>

在當前文件進行文本替換 （必備）

Ctrl + Z</kdb>

撤銷 （必備）

Ctrl + Y</kdb>

刪除光標所在行 或 刪除選中的行 （必備）

Ctrl + X</kdb>

剪切光標所在行 或 剪切選擇內容

Ctrl + C</kdb>

複製光標所在行 或 複製選擇內容

Ctrl + D</kdb>

複製光標所在行 或 複製選擇內容，並把複製內容插入光標位置下面 （必備）

Ctrl + W</kdb>

遞進式選擇代碼塊。可選中光標所在的單詞或段落，連續按會在原有選中的基礎上再擴展選中範圍 （必備）

Ctrl + E</kdb>

顯示最近打開的文件記錄列表 （必備）

Ctrl + N</kdb>

根據輸入的 類名 查找類文件 （必備）

Ctrl + G</kdb>

在當前文件跳轉到指定行處

Ctrl + J</kdb>

插入自定義動態代碼模板 （必備）

Ctrl + P</kdb>

方法參數提示顯示 （必備）

Ctrl + Q</kdb>

光標所在的變量 / 類名 / 方法名等上面（也可以在提示補充的時候按），顯示文檔內容

Ctrl + U</kdb>

前往當前光標所在的方法的父類的方法 / 接口定義 （必備）

Ctrl + B</kdb>

進入光標所在的方法/變量的接口或是定義處，等效於 Ctrl + 左鍵單擊 （必備）

Ctrl + K</kdb>

版本控制提交項目，需要此項目有加入到版本控制才可用

Ctrl + T</kdb>

版本控制更新項目，需要此項目有加入到版本控制才可用

Ctrl + H</kdb>

顯示當前類的層次結構

Ctrl + O</kdb>

Ctrl + I</kdb>

選擇可繼承的方法

Ctrl + +</kdb>

展開代碼

Ctrl + -</kdb>

摺疊代碼

Ctrl + /</kdb>

註釋光標所在行代碼，會根據當前不同文件類型使用不同的註釋符號 （必備）

Ctrl + [</kdb>

移動光標到當前所在代碼的花括號開始位置

Ctrl + ]</kdb>

移動光標到當前所在代碼的花括號結束位置

Ctrl + F1</kdb>

在光標所在的錯誤代碼處顯示錯誤信息 （必備）

Ctrl + F3</kdb>

調轉到所選中的詞的下一個引用位置 （必備）

Ctrl + F4</kdb>

關閉當前編輯文件

Ctrl + F8</kdb>

在 Debug 模式下，設置光標當前行為斷點，如果當前已經是斷點則去掉斷點

Ctrl + F9</kdb>

執行 Make Project 操作

Ctrl + F11</kdb>

選中文件 / 文件夾，使用助記符設定 / 取消書籤 （必備）

Ctrl + F12</kdb>

彈出當前文件結構層，可以在彈出的層上直接輸入，進行篩選

Ctrl + Tab</kdb>

編輯窗口切換，如果在切換的過程又加按上delete，則是關閉對應選中的窗口

Ctrl + End</kdb>

跳到文件尾

Ctrl + Home</kdb>

跳到文件頭

Ctrl + Space</kdb>

基礎代碼補全，默認在 Windows 系統上被輸入法佔用，需要進行修改，建議修改為 Ctrl + 逗號 （必備）

Ctrl + Delete</kdb>

Ctrl + BackSpace</kdb>

在打開的文件標題上，彈出該文件路徑 （必備）

Ctrl + 光標定位</kdb>

按 Ctrl 不要鬆開，會顯示光標所在的類信息摘要

光標跳轉到當前單詞 / 中文句的左側開頭位置 （必備）
