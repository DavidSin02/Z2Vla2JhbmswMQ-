深入理解C語言的指針

2020-11-14
| 科技

指針是C語言的一個核心特色，它以一種統一方式對不同數據結構中的元素產生引用。對於新手來說，指針總是會帶來很多困惑，但其實指針的基本概念非常簡單。下面是一些指針和它們映射到機器代碼的關鍵原則。

1、每個指針都對應一個類型

指針類型表明指針指向的是哪一類對象。比如：

int *ip;

char **cpp;

變量ip是一個指向int類型對象的指針，而cpp指針指向的對象自身就是一個指向char類型對象的指針。

通常，如果對象類型是T，那麼指針的類型為T*。特殊的void *類型代表通用指針。比如malloc函數返回一個通用指針，然後通過顯式強制類型轉換或賦值操作的隱式強制類型轉換，將它轉換成一個有類型的指針。

指針類型不是機器代碼中的一部分，它是C語言提供的一種抽象，幫助程序員避免尋址錯誤。

2、每個指針都有一個值

指針的值是某個指定類型的對象的地址。特殊的NULL（0）值表示該指針沒有指向任何地方。

3、用“&”創建指針

“&”運算符可以應用到任何value類的C表達式上，value類指可以出現在賦值語句左邊的表達式，包括變量、結構和數組的元素。

4、用“*”間接引用指針

“*”運算符用於間接引用指針，其結果是一個值，它的類型與該指針的類型一致。間接引用是用內存引用來實現的，要麼是存儲到一個指定的地址，要麼是從指定的地址讀取。

5、數組與指針緊密聯繫

一個數組的名字可以像一個指針變量一樣引用（但不能修改）。數組引用（a[3]）與指針間接引用（*(a+3)）有一樣的效果。

數組引用和指針運算都需要用對象大小對偏移量進行伸縮。如表達式 p+i，這裡的指針p的值為p，得到的地址計算為 p+L*i，這裡的L是與p相關聯的數據類型的大小。

6、指針的類型轉換改變指針運算的伸縮

將指針從一種類型強制轉換成另一種類型，只改變它的類型，而不改變它的值。強制類型轉換的一個效果是改變指針運算的伸縮。

比如p是一個char *類型的指針，它的值為p，那麼（int *）（p+7）的結果為 p+7，而（int *）p+7的計算結果為p+28。

PS：強制類型轉換的優先級高於加法。

7、指針也可以指向函數

這裡指針提供了一個強大的存儲和向代碼傳遞引用的功能，這些引用可以被程序的其他部分調用。

比如有一個函數定義如下：

int fun（int x, int*p）;

然後可以聲明一個指針fp，將它賦值給這個函數，

int （*fp）（int，int *）;

fp = fun;

然後就可以用這個指針來調用這個函數：

int y = 1;

int result = fp(3, &y);

函數指針的值是該函數機器代碼表示中第一條指令的地址。

函數指針聲明的語法對新手程序員比較難以理解，對於以下聲明：

int (*f)(int*);

要從裡往外讀，即從(*f)開始，f是一個指針，而(*f)(int *)表明f是一個指向函數的指針，這個函數以一個 int *作為參數。最後我們看到，它是指向以 int *為參數並返回int的函數的指針。
