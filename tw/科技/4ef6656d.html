武義90後女孩成功出道，帶火了家裡的炸串店！網友：這周就去吃

2020-11-14
| 科技

以前，老沈看到甜甜的女孩子都會說：“你吃可愛多長大的吧”。

但是現在，老沈覺得甜甜的女孩子都像小彩虹徐夢潔一樣，是吃根建燒烤長大的！

徐夢潔爸爸媽媽經營了十多年的根建燒烤冷飲，在武義當地早就小有名氣。

但是創造101播出後，武義吃貨們私藏這家的燒烤小店再次人氣暴漲。微博、豆瓣......全是網友們探店打開後的repo。最關鍵的是，大家都表示：根建燒烤真的很好吃啊！

那老沈可就坐不住了，連忙買票出發去武義！下面吃貨老沈就放上新鮮出爐的探店攻略，前排說一句，小彩虹的爸爸媽媽真的是hin可愛沒錯了~

交通指南

根建燒烤所在的武義縣位於浙江金華，從杭州出發有高鐵直達。票價77元，一個多小時就能到。

老沈想了想，從濱江出發到城西也要這麼久，對杭州上班族來說，用週末的時間出游完全足夠。

武義北站下車後，建議直接打車去根建燒烤。根建燒烤開在熟溪北路上，是武義縣比較繁華的地段。

送我們過去的出租車司機是武義當地人，聽說我們是去根建燒烤，告訴我們一定要點藕餅（就是徐夢潔吐槽包起來很難受的那個藕餅）。

司機還讓我們問問老闆是不是要漲價。在這裡老沈先替彩虹爸帶句話，根建燒烤不會漲價，大家放心大膽地點！

我們12點45分從杭州出發，到根建燒烤才2點多。但是店裡已經陸續來了客人，不出半個小時，就沒什麼座位了。

彩虹爸說，以前都是下午4點多營業，現在好多客人都是從上海、杭州過來的，得趕下午的車回去，所以就改成2點多營業，不能讓人家白跑。

下午三點左右，店裡就坐滿了。

推薦菜品

走進店裡，彩虹爸媽已經忙開了。根建燒烤更像是炸串，一口大鍋熱著油，彩虹爸一個人負責炸，忙得不可開交。

問他為什麼不找個人來幫忙，他說自家的炸串好吃是因為火候掌握得比較好，哪個串先下，哪個串後下都有講究。這個技術沒有十幾年練不出來，只能自己上。

店裡炸串的品種不多，幾乎沒有素菜。點菜的方式也很復古，所有食材就放在檯面上，要吃什麼直接跟老闆說。碰到人多到時候，就要比誰嗓門大了……

老沈點了兩份雞爪、兩串藕餅、兩份香腸和一串芋頭。炸中翅、臭豆腐、螺螄肉等沒點上，留著下次再吃。

/ 雞爪 /

小彩虹在節目裡說過，她們家的炸雞爪是先滷後炸。雞爪上桌後，油光鋥亮的色澤就讓人食慾大增。

雞爪完全入味，口感酥爛，不用動手就能剔下肉。醬料是祕製的，店裡大部分的炸串都用同一種醬料，甜辣鮮香。覺得味道不夠重的話，還可以蘸一下大蒜葉做的綠色調味醬。

/ 藕餅 /

剛到店裡的時候老沈就看到邊上坐著三個女孩子在包藕餅（往藕片裡填肉餡）。彩虹爸爆料，徐夢潔十多歲就開始幫店裡包藕餅，每次包藕餅都要包一大盆，所以才“包得難受”。

藕餅表皮炸得酥脆，內裡又很軟糯，肉餡填得滿滿當當。雖然炸串，但一點都不油。只是用料太大方了，暴風式掃完一整盤藕餅，竟然有點飽！

/ 香腸 /

不知道是不是童年濾鏡加成，這盤炸香腸真的深得我心。香腸炸完後酥香可口，刷了一層薄薄的甜麵醬，完全就是以前放學後最愛吃的那種油炸點心！

/ 炸翅中 /

炸翅中是坐在旁邊的小姐姐推薦的，她專程從上海過來打卡。翅中炸得酥軟，沒有撒很多調味料，一口咬破錶皮，帶著鮮香的熱浪衝到鼻尖，很過癮了。

必點冷飲

彩虹爸一邊忙著炸串一邊招呼我們坐下來喝點冷飲。根建燒烤的全名是根建冷飲燒烤。別看炸串的品類不多，冷飲的種類可是多到挑花眼！

有銀耳湯、綠豆湯、橘子糖水這些常見的，也有清涼糕、青草腐之類比較有地方特色的。

老沈數了數大概有16種可以選，價格在7元到8元一碗不等。老沈橫掃9碗，最後Pick了下面幾種。

/ 清涼糕 /

清涼糕是金華地區的特色冷飲，衝著“清涼糕”這個名字，老沈立馬就要了一碗。

一碗清涼糕裡，糕體軟糯Q彈，入口甘甜，糖水部分好像加入了薄荷，入口絲絲清涼。老沈原本想嘗一下拍個照的，結果一口氣全吃完了……

後來有朋友告訴我，金華本地人吃清涼糕喜歡往裡面加一點點醋，口感和降暑效果都會更好。

/ 青草腐 /

青草腐用仙草做底，撒了芝麻糖粉和薄荷。

整碗冷飲的加分點就在芝麻糖粉上，芝麻香混著青草香，和甜品店裡燒仙草的口感完全不同，清爽不膩。

/ 白藥酒 /

彩虹媽媽傾力推薦的白藥酒，其實就是冰酒釀。濃稠潤口，甜而不燥，發酵後的酒味比較濃郁，平時喜歡吃醪糟、酒釀的可以試試。

/ 橘子糖水 /

橘子糖水和黃桃糖水也是人氣選手，老沈要了一碗，黃澄澄的，拍照很好看。味道就像大家小時候都吃過的那種糖水味道，沒有什麼複雜的調味劑，簡簡單單一碗糖水，卻讓人覺得超級親切。

因為店裡沒有菜單，所以菜品沒有標出價格。但是老沈吃了那麼多東西，總共才花了90塊左右，其中還包括了9碗冷飲，算下來真的不貴了。

除了推薦好吃的，老沈還總結了幾點注意事項，給準備去打卡的老鐵們參考。

溫 馨 提 示

下午2:00——4:00之間店裡的人會比較少。

點單的時候告訴老闆要吃什麼就行，不要自己拿。

調味是固定的，不能吃辣的話要主動告訴老闆少放點辣。

因為沒有菜單，所以點完單要記一下自己點了什麼，方便結賬買單。

住宿推薦武義唐風酒店，是離根建燒烤比較近的溫泉酒店。

攻略都放到這兒啦。最後給大家爆個料，在參加創造101前，小彩虹徐夢潔還參加過《熱血街舞團》。

彩虹爸說，徐夢潔是個很聽話的孩子，同時也很有自己的想法。原本希望她接燒烤店的班，現在家裡人都支持她去做自己喜歡的事。

老沈也祝這個努力又可愛的姑娘，花路越走越順啦~
