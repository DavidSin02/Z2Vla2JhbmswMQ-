日語動詞的て形（te形）變化規則

2020-10-29
| 科技

日語算是一門比較好入門的語言，而且對作為漢語使用者的中國人來說，更是有各種天然的學習優勢。不過好多初學者一路高歌猛進，往往都會在動詞的て形（te形）變化這裡一下子被難住了。而て形的變化偏偏又是日語裡面基礎中的基礎，而且在日語裡面用途非常非常廣泛。て形的日常用法少說也有幾十種，可以說如果你不熟練掌握動詞的て形的話，那麼甚至就連話都不會說了。

並且動詞的て形變化和た形（ta形）變化規則是相通的，掌握了其中一個，另一個也就自然而然會了。而た形是日語中兩大時態，現在時和過去時之一，其重要性不言而喻。而這兩者的變化規律是所有詞語變形中最複雜的，所以是不得不下點功夫以求牢固掌握的語法內容。

日語中的所有動詞，結尾的最後一個假名都在う段，根據結尾的不同，可以分為三大類動詞。把這三類動詞的變化規則分門別類整理出來，其規律就會比較清晰，掌握て形的變化也就不難了。

一類動詞（五段活用動詞）

1、動詞原形（即基本形，下同）詞尾以“う”、“つ”、“る”結尾的作促音變，即詞尾最後一個音變為“って”

買う（かう） → 買って（かって）

待つ（まつ） → 待って（まって）

降る（ふる） → 降って（ふって）

2、動詞原形詞尾以“む”、“ぶ”、“ぬ”結尾的作撥音變，即詞尾最後一個音變為“んで”

休む（やすむ） → 休んで（やすんで）

遊ぶ（あそぶ） → 遊んで（あそんで）

死ぬ（しぬ） → 死んで（しんで）

3、動詞原形以“く”結尾的則變“く”為“いて”，以“ぐ”結尾的 ，則變“ぐ”為“いで”

歩く（あるく） → 歩いて（あるいて）

書く（かく） → 書いて（かいて）

泳ぐ（およぐ） → 泳いで（およいで）

注意！行く（いく）是特例 ，它的變化為行って（いって），單獨記住即可。

4、動詞原形以“す”結尾的， 則變“す”為“して”

例：話す（はなす） → 話して（はなして）

二類動詞（上一段活用動詞、下一段活用動詞）

這是以る結尾並且倒數第二個假名在い段上或者え段上的詞，存在個別特例符合以上規則卻不屬於二類動詞的比如帰る（かえる）、切る（きる）等。這些個別的單詞看上去形式和二類動詞一樣，但其實是一類動詞。特例數量很少，只能靠自己遇到的時候就單獨記住了。二類動詞的變化是最簡單的，將詞尾“る”變為“て”就可以了。

食べる → 食べて

寢る → 寢て

三類動詞

1、カ變動詞，這種動詞只有一個，就是來る（くる），它的變化規則是特別的，注意發音會變

來る→來て（きて）

2、サ變動詞，包括する及後面帶有する的其它三類動詞，規則是把“する”變為“して”

勉強する → 勉強して

整理する → 整理して

這些規則一下子看起來好像會令人眼花繚亂，畢竟這是日語基礎的動詞活用中最複雜的一部分，在你熟練掌握之前可能還要反覆回來看幾次。但其實當你理解了這些規律之後，就會發現這些內容並不難，並且在把這一關過了之後，其他的詞語變形對你來說也是小菜一碟了。

到了這裡，相信你已經大致掌握了日語動詞て形的活用規則了，如果你想要測試一下自己是否已經融會貫通，不妨來做幾個練習驗證一下吧。
