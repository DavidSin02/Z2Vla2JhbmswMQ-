「浩博電池資訊」系統介紹最常見的兩種高能綠色電池

2020-11-14
| 科技

本篇將對現在市面上最為常見的兩種高能綠色 充電 電池 鋰離子 電池 和 鎳氫 蓄 電池 做一個體系的知識說明，以供我們對這兩種 電池 有個全面的瞭解以及在選購 電池 對比角度與 電池 運用留神方面做個參看。

一、界說

鋰離子 電池 （Li-ion Batteries）是以含鋰的化合物作正極，以炭資料為負極的鹼性 蓄 電池 。

鎳氫 蓄 電池 （Nickel-Metal Hydride battery）正極活性物質首要由鎳製成，負極活性物質首要由貯氫合金製成的一種鹼性 蓄 電池 。

二、原理

鋰離子 電池 首要依託鋰離子在正極和負極之間移動來作業。在充放電進程中，Li+ 在兩個電極之間往復嵌入(Intercalation)和脫嵌：充 電池 時，Li+從正極脫嵌，經過電解質嵌入負極，負極處於富鋰狀況；放電時則相反。

1、正極反應：放電時鋰離子嵌入，充電時鋰離子脫嵌。

充電時：LiFePO4 → Li1-xFePO4 + xLi+ + xe−

放電時：Li1-xFePO4 + xLi+ + xe− → LiFePO4

2、負極反應：放電時鋰離子脫插，充電時鋰離子刺進。

充電時：xLi+ + xe− + 6C → LixC6

放電時：LixC6 → xLi+ + xe− + 6C

鎳氫 蓄 電池 在 電池 充電時，氫氧化鉀（KOH）電解液中的氫離子（H+）會被釋放出來，由這些化合物將它吸收，避免構成氫氣（H2），以堅持 電池 內部的壓力和體積。當 電池 放電時，這些氫離子便會經由相反的進程而回到原本的當地。 http://www.dghoppt.cn/ ​

1、充電時

正極反應：Ni(OH)2 + OH- → NiOOH + H2O + e-

負極反應：M + H2O + e- → MH + OH-

總反應：M + Ni(OH)2 → MH + NiOOH

2、放電時

正極：NiOOH + H2O + e- → Ni(OH)2 + OH-

負極：MH + OH- → M + H2O + e-

總反應：MH + NiOOH → M + Ni(OH)2

三、特性

1、電壓：鋰 電池 的電壓是 鎳氫 蓄 電池 電壓的3倍。

鋰離子 電池 電壓 = 3.6V

鎳氫 蓄 電池 電壓 = 1.2V

2、按質量核算， 鋰離子 電池 有更高能量密度。

鋰離子 電池 能量∕重量 = 150～200Wh/kg(540～720kJ/kg)

鎳氫 蓄 電池 能量∕重量 = 60-120 Wh/kg（瓦特小時/千克）

3、按體積核算， 鋰離子 電池 有更高能量密度。

鋰離子 電池 能量∕體積 = 250～530Wh/L(0.9～1.9kJ/cm3)

鎳氫 蓄 電池 能量∕體積 = 140-300 Wh/L（瓦特小時/升）即 504-1188kJ/kg（千焦耳/千克）

4、輸出功率： 鋰離子 電池 具有更大輸出功率。

鋰離子 電池 功率∕重量 = 300～1500W/kg(@20秒)

鎳氫 蓄 電池 功率∕重量 = 250-1000 W/kg

5、自放電率： 鋰離子 電池 具有更低自放電率

鋰離子 電池 自放電率 < 5％～10％/月，一般每月在2%以下（可恢復）

鎳氫 蓄 電池 自放電率 = 一般每月 2-30%，見溫度而定，低自放電型號為每年10-30%

6、充放電功率： 鋰離子 電池 充放電率更高。

鋰離子 電池 充放電功率 = 100%

鎳氫 蓄 電池 充放電功率 = 66%

7、循環運用次數：兩種 電池 循環次數靠近，循環運用壽命都比較長。

鋰離子 電池 充放電循環次數 = 500 -2000次（磷酸鐵鋰 電池 能夠抵達2000次以上）

鎳氫 蓄 電池 充放電循環次數 = 500 -1800次

8、作業溫度規劃：兩種高功用 電池 的溫度規劃都很寬，相較而言在 低溫 環境中， 鎳氫 蓄 電池 的放電特性更佳。

鋰離子 電池 作業溫度規劃 -20℃～60℃

鎳氫 蓄 電池 作業溫度規劃 -30℃～50℃（ 低溫 鎳氫 電池 可在 低溫 -40℃中正常運用） ​

​

四、利益

1、比能量高：一次充電可行使的距離長，兩者的比能量都十分高，相對而言， 鋰離子 電池 （當時比能量最高的 蓄 電池 ）＞ 鎳氫 蓄 電池 ；

2、比功率高：在大電流作業時能平穩放電，加快爬坡才華好，兩者的比功率都十分高，鋰離子優勢顯著一點；

3、自放電小：二者都具有此利益， 鋰離子 電池 略勝一籌；

4、重量輕、體積小：同型 電池 鎳氫 蓄 電池 重量更輕、體積更小，只是在輸出平等電壓的狀況下（ 鋰離子 電池 電壓為 鎳氫 蓄 電池 的3倍），使得單個 電池 組合時數目 鋰離子 電池 可減少1/3，因而使成型後的 電池 重量和體積 鋰離子 電池 更小、更輕。

5、回想效應低： 鋰離子 電池 無回想效應因此在這個方面具有更大的優勢；

6、 低溫 放電功用好：在 低溫 狀況下 鎳氫 蓄 電池 更強悍；

7、循環壽命長：二者循環運用次數都十分多，很合算；

8、無環境汙染：兩者都是綠色環保 電池 ， 鎳氫 蓄 電池 被稱作是最環保的 電池 ，回收再用的功率 鋰離子 電池 略勝一籌；

9、標明容量高： 鎳氫 蓄 電池 標明容量可達2900mAh（毫安培－小時/毫安時），但仍不及 鋰離子 電池 高；

10、可快速放電：二者都具有這種利益， 鋰離子 電池 才華較強一些；

11、充電時間短：二者都有此利益。

別的， 鎳氫 蓄 電池 還具有安全可靠，免維護的特色，被稱作是最安全的 電池 ，而 鋰離子 電池 的安全功用還有待大大提高；而另一方面 鋰離子 電池 因為完全沒有回想效應，在運用上更為便利簡略，而 鎳氫 蓄 電池 卻因為剩下電壓的原因而無法避免進行守時放電處理操作。

五、運用

鋰離子 電池 在充電進程中容易發生短路狀況，此外留神不要過充，這點比較要害。

鎳氫 蓄 電池 需求進行守時放電處理，

六、問題

兩者有個一同的缺陷，那就是製作工藝要求嚴峻，導致本錢比較高，價格不便宜。相對而言，鋰 電池 仍是更合算一點。

七、運用

鎳氫 蓄 電池 在筆記本電腦、便攜式攝像機、數碼相機及電動自行車等領域得到了廣泛運用。

鋰離子 電池 廣泛運用於消費電子產品、軍工產品、航空產品等。

八、總述

鋰離子 電池 和 鎳氫 蓄 電池 都是綠色 電池 ，不會因扔掉而形成二次汙染，這契合環保理念，一同它們都可再生運用，契合持續發展的觀點，此外，我們在 電池 職業技術現已處於國際領先水平，而且這些綠色 電池 出口前景十分好，雖然現在價格有點高，有些小問題還需求更進一步的得到解決與改善，不過跟著我們國家科研人員以及、企業研發部門人員的一同極力，很快就能抵達優質廉價、高效節能、安全安穩、綠色環保這些要求，而像這種綠色高能且極具開發潛力的 電池 值得我們大力提倡與推行。
