Python零基礎入門之Python變量與字符串

2020-11-14
| 科技

函數是什麼？

  • 函數就是Python程序題前準備好的功能
  • 每個函數都要對應的功能
  • 函數的使用方式為：函數名(參數)

print函數

  • print用於向控制檯輸出字符串
  • 示例：print("鋤禾日當午") 、 print(3)
  • 在輸出文本時增加 \n 對文本換行
print("鋤禾日當午\n汗滴禾下土")Run:鋤禾日當午汗滴禾下土

常見錯誤

  • Python請使用半角字符（英文模式下的字符）
  • 大小寫錯誤，Python大小寫敏感（區分大小寫）
  • 英文單詞拼寫錯誤

註釋的作用

  • 註釋就是我們自己的語言，對代碼進行標註，增加可讀性。

Python的兩種註釋方式

  • 單行註釋： #我是人見人愛的單行註釋
  • 塊註釋：""" 註釋內容 """
 print("鋤禾日當午\n汗滴禾下土") #這是prtint函數""" 這是塊註釋註釋內容在這裡"""Run:鋤禾日當午汗滴禾下土

變量(variable)的作用

  • 程序的作用就是用來處理數據
  • 編程語言中數據使用變量的形式保存
  • 為變量設置“值”的過程，稱為“賦值”

定義變量

  • 變量的語法：變量名 = 值
  • 等號左邊是變量的名稱
  • 等號右邊是變量要保存的數據
name = "學了一點"

變量的命名要求

  • 變量名有意義，見名知意
  • 變量名只能包含字母、下劃線與數字，不能數字開頭
  • 不能與python關鍵字重名

如何給變量起一個好名字

  • 所有單詞小寫，多個單詞之間使用 _ 連接
  • 最好使用英文單詞，不建議使用拼音
  • 長度最好不要超過20個字符，過長可是使用縮寫

變量的數據類型

  • 變量在賦值時會自動判斷數據的類型

Python最常用有四種數據類型

type函數

  • type函數用於得到變量的數據類型
  • 語法： 變量 = type(變量名)
  • 輸出： str | int | float | bool
str = "學了一點"print(type(str))Run:<class 'str'>

基本運算符

  • 基本運算符是指python中使用的基本數學計算符號

基本運算符

接收用戶輸入

  • 使用input函數將用戶輸入的字符串保存到變量
  • 語法格式：變量 = input("提示信息")
name = input("請輸入您的姓名：")print("name")Run:請輸入您的姓名：學了一點學了一點

字符串與數字互相轉換

  • 字符串->數字：int(字符串) 、float(字符串)
  • 數字->字符串：str(數字)

Python字符串

  • 字符串就是一系列字符
  • 字符串可以使用單引號，也可以使用雙引號
str1 = "這是雙引號"str2 = '這是單引號'str3 = "這是雙引號和 '單引號' 混合使用"

拼接(合併)字符串

  • 字符串拼接是指將多個字符串合併，形成一個新的字符串
  • 字符串拼接使用 + 號處理
str4 = "您乘坐的" + 'G12345' + "次列車正在檢票"

字符串的大小寫轉換

  • 字符串提供了大量使用函數，允許讓我們對字符串進行加工

Python 3 中的五個大小寫函數

print("bmw".upper())#轉換為大寫print("BMW".lower())#轉換為小寫print("how are you".capitalize()) #首字母大寫print("michael jackson".title())#title()設置每個單詞首字母大寫print("abcDEF".swapcase())#大小寫互換Run:BMWbmwHow are youMichael JacksonABCdef

格式化字符串

  • Python2.6 開始，新增了一種格式化字符串的函數 str.format()
"{} {} you".format("I", "love") "{3}.{2}.{1}.{0}".format("點", "一","了","學")  # {} 為佔位符；{1} 數字1為索引，從0開始Run:I love you學了一點

格式化數字

  • format()函數同樣支持數字格式化（返回的都是字符串）
  • 示例：format(1234.567, '0.2f') #小數保留2位 1234.57
  • 示例：format(1234.567, ',') #千分位分隔符 1,234.567
num = 1234567.89555str5 = format(num,'0.2f')print(str5)amt = 123456789str6 = format(amt,"0,.2f")print(str6)Run:1234567.90123,456,789.00

製表符與換行符

  • 製表符是指增加字符的縮進，在字符串中使用\t
  • 換行符是指為字符串換行輸出，在字符串中使用\n
print("姓名\t性別\t年齡\n\t趙四\t男士\t42")Run:姓名	性別	年齡趙四	男士	42

刪除空白

  • 在' python'與'python'是不同的字符串

三個函數來刪除左右的空白

space_str = " hello world "print(space_str)nospace_str = space_str.strip()print(nospace_str)Run: hello world hello world

查找字符串

  • str.find()函數用於獲取子字符串出現的位置
  • 語法：str.find(目標串，[開始位置]，[結束位置])
source = "這裡是學了一點，關注我，教你Python零基礎入門"index = source.find("學了一點",0,22)print(index)isexist = "學了一點" in source #判斷是否存在“學了一點”，存在返回True，反之返回Falseprint(isexist)Run:3True

字符串替換

  • str.replace()函數用於字符串替換
  • 語法：str.replace(原始串，目標串，[替換次數])
  • 示例："aaabbbccc".replace("b", "d" ,2) 輸出 aaaddbccc
str7 = "this is string example. This is really string"pstr7 = str7.replace("is","was")print(pstr7)pstr7 = str7.replace("is","was",3)print(pstr7)Run:thwas was string example. Thwas was really stringthwas was string example. Thwas is really string
