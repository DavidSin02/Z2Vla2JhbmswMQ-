榮耀小米OPPO變焦拍照對比，誰能拍的更遠更穩更好

2020-10-29
| 科技

【手機中國評測】人類對於“看的更遠”有著執著的追求，好奇大洋彼岸，宇宙星空的深處有何奇妙之境。過去如此，現在亦如此，只不過現在是從望遠鏡換成了幾乎人手一部的智能手機，變焦鏡頭的存在讓我們通過幾英寸的屏幕就能看到千里之外的風光。

當然，變焦鏡頭的作用不只是拍到更遠的景緻，同時還能發現一個觀察事物的新角度。面對高聳入雲的奇觀，我們無法飛行，只能仰望，但是手機上的變焦鏡頭可以讓我們湊近一點，再湊近一點，距離上的變化可以直接改變我們看待事物的角度，從仰望到對視，都在指尖上實現。

正是因為變焦鏡頭擁有如此意義和魅力，使其成為諸多智能手機，尤其是旗艦級產品的標配，比如今年的榮耀30 Pro+、OPPO Find X2 Pro、小米10 Pro等，都在這方面有著非常優秀的表現。那麼這時候問題來了，如果將三款手機的變焦樣張擺在一起會碰出何種火花呢？

變焦鏡頭的軍備競賽

榮耀30 Pro+、OPPO Find X2 Pro、小米10 Pro三款手機雖然都搭載了高素質的變焦鏡頭，但是在具體配置上還是有著十分明顯的區分。

作為榮耀最新旗艦產品，榮耀30 Pro+後置三攝組合，包括5000萬像素超感光攝像頭、1600萬像素超廣角攝像頭、800萬像素潛望式長焦攝像頭。該機支持5倍光學變焦、10倍混合光學變焦和50倍數碼變焦，支持光學防抖；

榮耀30 Pro+

OPPO Find X2 Pro同樣也後置三攝，涵蓋4800萬像素廣角攝像頭、4800萬像素超廣角攝像頭、1300萬像素潛望式長焦攝像頭，三顆攝像頭通過接力的方式實現了10倍混合光學變焦，同時支持最高60倍數碼變焦，支持光學防抖；

小米10 Pro這次後置四攝組合，由1億像素主攝、2000萬像素超廣角攝像頭、1200萬像素人像攝像頭、800萬像素長焦攝像頭組成。注意，這款手機的長焦攝像頭雖然沒有潛望式設計，但是依然實現了10倍混合光學變焦和最高50倍數碼變焦，支持光學防抖。

僅從三款旗艦手機的拍照配置來看，都達到目前智能手機的旗艦水準，為清晰的成像效果奠定了堅實的硬件基矗當然，看這些參數可能沒有什麼感覺，不如我們直接看效果。

榮耀30 Pro+（15倍及以上激活畫中畫功能）

小米10 Pro

OPPO Find X2 Pro

通過上述動圖可以看到，三款手機的最高倍數變焦基本上貫穿整個街道。另外，榮耀30 Pro+的取景器畫面在變焦過程中較為穩定，在50倍變焦的前提下還能保證一定的畫面穩定性，可見其配備的雙結構光學防抖和AIS長焦防抖確實發揮了作用。這樣的拍攝效果是非常適合我們進行長焦拍攝的，你會發現手抖不是問題，“麒麟臂”也不是必需品。

榮耀30 Pro+

另外，如果再加上1倍變焦和0.6倍變焦，也就是我們常說的普通模式和超廣角模式，你會發現榮耀30 Pro+的變焦距離實際是非常遠的。

畫中畫

值得注意的是，當榮耀30 Pro+的變焦倍數超過15倍及其以上時，會自動激活畫中畫功能，方便用戶確定拍攝對象，避免因變焦倍數拉大而找不到拍攝目標。目前，三款手機中僅榮耀30 Pro+具備該功能。

當然，拍得遠拍得穩是一回事，拍得好不好，清不清楚又是另外一回事了。

長焦成像對比

正如我在開篇提到的，長焦拍照質量是這次對比的主角，我將通過2倍、5倍、10倍以及最高變焦倍數等四種變焦倍數進行對比，來看看三者的表現。

這組樣張拍攝於下午4點，光線充足，在2倍變焦的前提下拍攝完成的。由於2倍變焦是日常拍照中常用的變焦倍數選擇，既可以拉近距離，又能避免離拍攝主體太近，因此頗受普通用戶歡迎。

榮耀30 Pro+（2倍變焦）

小米10 Pro（2倍變焦）

OPPO Find X2 Pro（2倍變焦）

我們可以看到在這種樣張中，三款機型都有著非常不錯的整體觀感。榮耀30 Pro+畫面的清晰度得到了很好地保證，同時顏色還原準確，接近我當時肉眼看到的色彩，細節內容豐富；小米10 Pro在2倍變焦前提下同樣保留了豐富的畫面細節，不過左下角的紅樹林出現了一點偏色現象，但好在沒有影響到整張照片的高素質和視覺感受；OPPO Find X2 Pro在保證基本清晰度的同時，色彩有些偏暖，但也更討好眼球，這樣的色彩表現是很討人喜歡的。

順帶一提，榮耀30 Pro+默認變焦標籤是沒有2倍變焦的，因此每次都需要用戶手動調節，但好在不影響使用。當然，這也代表了不同廠商對於使用體驗的不同理解。

可見，三者在2倍變焦中都有著出色的成像表現，那如果繼續拉大變焦倍數呢？

榮耀30 Pro+（5倍變焦）

小米10 Pro（5倍變焦）

OPPO Find X2 Pro（5倍變焦）

在這組5倍變焦樣張中，榮耀的優勢較為明顯，當然這也與其本身的功能有關。由於榮耀30 Pro+支持5倍光學變焦，因此畫面有著非常高的清晰度和精細度，比如建築物上的“朝外SOHO”字樣和樓體細節，同時顏色還原也最為準確。值得注意的是，這款手機壓光效果明顯，樓宇上的玻璃幕牆沒有出現明顯的過曝。

榮耀30 Pro+（10倍變焦）

小米10 Pro（10倍變焦）

OPPO Find X2 Pro（10倍變焦）

注意，這三款手機在變焦上有一個共同點，即均支持10倍混合光學變焦。因此在這一環節中，榮耀30 Pro+依舊保持著較高的銳度，因此看上去也更為清晰，觀感也更好，而且和OPPO Find X2 Pro相同，曝光控制得較好，沒有出現明顯的過曝。同時，榮耀30 Pro+和小米10 Pro噪點控制都比較明顯，這一點在天空部分中會表現得較為顯著，畫面乾淨。

可以看出，三款手機在變焦拍照上都有著很好的表現，展示了作為旗艦手機應有的拍照水準。但值得注意的是，這還不是變焦的極限。

我們將這三款手機的變焦倍數均提到最高，榮耀30 Pro+和小米10 Pro最高均為50倍數碼變焦，而OPPO Find X2 Pro最高可達60倍數碼變焦，拍攝效果如下。

榮耀30 Pro+（50倍變焦）

小米10 Pro（50倍變焦）

OPPO Find X2 Pro（50倍變焦）

榮耀30 Pro+在三款機型中效果最好，不僅壓光效果明顯，而且畫面乾淨，清晰度良好；小米10 Pro則出現了一定的過曝，同時字體邊緣出現了紫邊現象，但是清晰度還不錯，效果還是很震撼的；OPPO Find X2 Pro由於倍數更高一些，因此字體會顯得更大，同時也擁有較好的清晰度和色彩表現，但是畫面上的噪點相比另外兩款機型會更為明顯一些，畢竟放的更大了嘛。

要不極限一點？

上述變焦樣張都是在白天光線好的時候拍攝的，那麼在晚上光線不佳的前提下又表現如何？不過，從測試環境的角度來說可能有點極端，但越是這樣越是能看出手機的真實水平。而且鑑於夜晚也是生活中常見的拍照場景，因此也加入到了本篇對比評測中，而夜景最考驗手機對於噪點的控制以及畫面純淨度，那麼這三款機型表現如何？

榮耀30 Pro+（2倍變焦）

小米10 Pro（2倍變焦）

OPPO Find X2 Pro（2倍變焦）

在2倍變焦的前提下，OPPO Find X2 Pro的夜景表現最好，不僅保證了畫面的清晰度和色彩的還原，同時畫面更為乾淨，尤其是夜空的噪點控制得很好；榮耀30 Pro+儘管出現了一定的偏色，而且夜空中出現了一定的噪點，但是清晰度是能夠保證的；小米10 Pro畫面中的建築物出現了一定的塗抹和偏色現象，但是依靠較好的亮度控制，還是獲得了不錯的觀感。另外，三款機型的廣告牌燈箱部分均出現了一定的過曝。

榮耀30 Pro+（5倍變焦）

小米10 Pro（5倍變焦）

OPPO Find X2 Pro（5倍變焦）

而在5倍變焦下，三者同樣出現過曝現象，不過OPPO Find X2 Pro噪點控制得較為明顯，同時亮度和色彩更討好眼球；而榮耀30 Pro+的樣張更貼近當時的實際環境亮度，清晰度和噪點控制得較好，同時壓光效果明顯，整體觀感尚可；小米10 Pro樣張最為明亮，觀感較好，畫面也較為乾淨。

榮耀30 Pro+（10倍變焦）

小米10 Pro（10倍變焦）

OPPO Find X2 Pro（10倍變焦）

但是榮耀30 Pro+的10倍變焦拍照效果最好，畫面純淨，噪點較少，值得注意的是，該機不僅曝光效果控制得較好，而且還保證了畫面的亮度；OPPO Find X2 Pro和小米10 Pro均出現了一定的過曝現象，不過好在畫面乾淨，而且色彩也更為討喜。值得注意的是，小米10 Pro還保留了更多的畫面暗部細節。

注意，這個場景的光源較暗，三款手機通過長焦攝像頭還能獲得如此效果，也足以看出旗艦手機的優化水平。正如我在開篇中提到的那樣，變焦攝像頭除了可以拍的更遠，還能找到一個新的視角，除了上述遠景，在近景中也能拍出不錯的效果。這裡我們以榮耀30 Pro+為例。

榮耀30 Pro+樣張（2倍變焦）

榮耀30 Pro+樣張（1倍變焦）

榮耀30 Pro+樣張（2倍變焦）

榮耀30 Pro+樣張（5倍變焦）

榮耀30 Pro+樣張（10倍變焦）

從遠到近，從小到大，變焦攝像頭為我們拍照提供了很多的新視角，彷彿為我們插上翅膀一般，觸及到我們看不到的“月球背面”。

寫在最後

作為各自品牌的旗艦產品，榮耀30 Pro+、OPPO Find X2 Pro、小米10 Pro都在變焦拍照上展現出自己應有的水準，而且都有自己的特點，比如榮耀的5倍光學變焦、OPPO的AI炫彩、小米的清晰度等。不過，榮耀30 Pro+在保證自身成像穩定的基礎上，讓顏色還原也更貼近現實，而且5倍光學變焦的加持也使其在同價位手機中擁有更高的競爭力。

對於消費者而言，在擁有防抖穩定成像的同時，還能夠擁有更好的使用體驗也是十分重要的。榮耀30 Pro+在15倍及以上高倍變焦的前提下，能激活“畫中畫”功能，幫助用戶更好的尋找拍攝對象，解決用戶在高倍變焦拍攝中“找不到拍攝對象”“畫面容易抖動”等痛點問題，讓我們可以拍得更穩更遠，用這雙特別的“眼睛”探索未知的世界。大概，這就是我們熱愛變焦拍照的意義吧。
