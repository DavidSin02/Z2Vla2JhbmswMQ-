Java字節類型

最小的整數類型為 byte 。 byte 類型變量在使用來自網絡或文件的數據流時非常有用。

字節變量通過使用byte關鍵字聲明。下面聲明兩個字節變量b和c：

byte b, c;

byte 是有符號的8位類型，範圍從-128到127。

以下代碼創建兩個字節類型變量並分配值。

public class Main { public static void main(String[] args) { byte b1 = 100; byte b2 = 20; System.out.println("Value of byte variable b1 is :" + b1); System.out.println("Value of byte variable b1 is :" + b2); }}

字節類在對象中封裝了原始類型字節的值。字節類提供了將字節轉換為字符串和字符串轉換為字節的幾種方法

Java short類型

Java short類型的大小在字節和整數之間。

short是一個有符號的16位類型。短類型變量的範圍為-32,768到32,767。

下面是一些簡短變量聲明的例子：

short s; short t;

Java int類型

當在表達式中使用字節和短型值時，在求值表達式時將它們提升為int。

int 是有符號的32位類型，範圍為 -2,147,483,648 到 2,147,483,647 。

Java long類型

當int類型不夠大時，使用Java long類型。

long是一個有符號的64位類型和。長型的範圍是-9,223,372,036,854,775,808至9,223,372,036,854,775,807。

要指定一個長文字，你需要告訴編譯器字面值的類型為 long ，通過在文本上附加一個大寫或小寫的 L 。例如， 0x7ffffffffffffffL 或123123123123L 。

以下代碼創建一個 long 類型文字，並將值分配給a long 類型變量。

public class Main { public static void main(String args[]) { long l = 0x7ffffffffffffffL; System.out.println("l is " + l); }}

八進制整數(基數8)

八進制值在Java中由前導零表示。有效值09將從編譯器產生錯誤，因為9在八進制的0到7範圍之外。

public class Main { public static void main(String[] args) { int i = 010; System.out.println(i); }}

十六進制整數(基數16)

十六進制與模8字大小(例如8,16,32和64位)匹配。您表示具有前導零-x(0x或0X)的十六進制常數。

十六進制數字的範圍為0到15，因此A到F(或a到f)代替10到15。

整數常量可以始終分配給一個長變量。也可以為char分配整數，只要它在範圍內。

public class Main{ public static void main(String[] argv){ int f = 0XFFFFF; System.out.println(f);//1048575  }}
