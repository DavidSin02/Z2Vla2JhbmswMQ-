24 個值得嘗試的 Linux 桌面環境 | Linux 中國

2020-10-29
| 科技

我全要！

• 來源：linux.cn • 作者：Seth Kenlon • 譯者：鄭 •

（本文字數：2014，閱讀時長大約：3 分鐘）


Linux 桌面的最大優點之一是它提供給用戶選擇的權利。如果你不喜歡你屏幕左下角的應用程序菜單，你可以移除它。如果你不喜歡你的文件管理器組織文件的方式，你可以使用另外一個文件管理器。不過，不可否認的是，這對那些不習慣自己選擇如何使用計算機的用戶來說可能會感到困惑。如果你正在考慮安裝 Linux，你要做的選擇之一就是你想使用哪一個桌面，最好的方法就是嘗試幾個不同的桌面，直到你找到感覺適合你的桌面。

默認桌面

作為在 Fedora、Ubuntu、Debian 和一些其它操作系統上的默認桌面，GNOME 桌面可能是最流行的 Linux 桌面。它有一個易於使用和現代化的界面，並帶有為手機環境設計的小點綴，因此即使在觸摸屏上也感覺很自然。

不過，一些 Linux 發行版選擇了不同的桌面環境，如 KDE Plasma、Pantheon、Cinnamon 和 Mate。

  • GNOME : 現代化和易於使用，默認桌面
  • Mate : 傳統的 GNOME
  • KDE : 功能強大、可配置的 Plasma 桌面
  • Cinnamon : 經典的 GNOME
  • Pantheon : ElementaryOS 的默認桌面
  • Budgie : 一個柔和的桌面
  • Trinity : 傳統的 KDE

傳統的 Unix

Window Maker running on Fedora

Unix 是現代操作系統的鼻祖，也是 Linux 的直接靈感來源，擁有令人吃驚的豐富的桌面歷史。大多數人都不熟悉 Unix 桌面設計，因為 Unix 計算機被認為是學術界、科學界和電影界常見的專業機器，而不是家用計算機。如果你是一名 Unix、IRIX 或 NeXT 的用戶，那麼你可能會對類似於 CDE 或 NeXTStep 的桌面有著美好的回憶。你很幸運，因為這一傳統延續至今：

  • Windowmaker : NeXT 界面的復興
  • Enlightenment : 一個現代化的改進型 Windowmaker 風格的桌面
  • Fluxbox : 跳出 Blackbox 思維定式
  • Openbox : 改善 Blackbox 的可用性
  • TWM : 與 X11 綁定至死的示例桌面
  • Fvwm : 改進的 TWM

輕量級桌面

XFCE on Mageia Linux in 2019

你可能想知道為什麼 Linux 有那麼多的桌面可供選擇。你可以把它歸結為個人喜好和對效率低下的低容忍度，但優化用戶界面也有技術上的好處。例如，老電腦難以跟上用戶界面的更新，可以使用輕量級桌面 賦予它新的生命 。另外，你可能只是想把 CPU 處理能力分配到除桌面以外的其它東西上，所以保持你的主用戶界面極簡是有意義的。不管你的動機是什麼，這裡有一些你可以嘗試的桌面：

  • XFCE : 熟悉的桌面，極少的性能損失
  • LXQT 和 LXDE : 熟悉的桌面，較少的性能損失
  • PekWM : 一個 Blackbox 式的窗口管理器
  • Joe's Window Manager : 另一個 Blackbox 式的窗口管理器
  • Ratpoison : 不使用鼠標
  • Herbstluftwm : 針對 Vim 用戶 的Ratpoison
  • Lumina : 一個為 PC-BSD 開發的桌面

實驗性桌面

當創建一個桌面並將其集成到一個系統中相對容易的時候，會發生的事情之一就是你會得到一些有趣的概念驗證項目和試驗性項目。有一些會比其它的更精緻，有一些則不是很容易安裝。這些很可能註定不會成為你的永久桌面，但是它們可以是很有趣的體驗:

  • Unix Desktop Environment : 該桌面的重塑版
  • Rox : 在它那個時代是先進和現代的桌面
  • MLVWM : 要是 Apple IIe 運行 Unix 怎麼樣？
  • Equinox : 只是個最基礎的東西

選擇你的桌面

如果太多的選擇讓你不知所措，那麼記住：桌面意味著 可選。你沒有義務來嘗試發行版附帶的多個桌面。

許多 Linux 的高級用戶都會滿足於他們的發行版所提供的某個桌面。最重要的是得到一個可以與你的計算機一起工作的 Linux 系統系統，並花費時間來學習它是如何工作的。歸根結底，所有的桌面都只是為了做同一件事：幫助你組織和管理你的重要數據。讓你瞭解你的桌面是如何工作的，這才是最重要的。但是如果你已經瞭解了你的默認桌面的各個方面，那你現在有了可以在週末嘗試的大量選擇。祝你玩得開心！
