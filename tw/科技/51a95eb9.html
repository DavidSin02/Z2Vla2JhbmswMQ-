掌握這條命令行，讓你一鍵搞定圖片、音頻、視頻、文本格式轉換

2020-11-14
| 科技

工作時我常常需要把一個圖片、視頻或文本文件轉換成其他格式，或者需要把一大堆同樣後綴名的文件轉換成其他格式，這時，再打開一個軟件要麼有點得不償失，要麼一個個點按非常繁瑣。幾番尋找，我發現了下面這些全平臺命令行工具，安裝後，只要一行命令，就能搞定簡單的文件格式轉換。

圖片：ImageMagick

ImageMagick 是一款幾乎全能的圖片處理套件，它能完成圖片的創建、編輯、對比工作，當然，轉換格式也不在話下。軟件開源、免費，並且全平臺覆蓋，在官方網站下載安裝對應版本後，在終端輸入 magick 有返回值說明安裝成功。推薦使用軟件包管理器安裝：

 # macOS 需要安裝 Ghostscript fonts $ brew install ghostscript $ brew install imagemagick # Windows 兩條命令二選一，需要安裝對應的包管理器 $ scoop install imagemagick $ choco install imagemagick

該工具可以轉換所有常用圖片格式，例如 PNG、JPG、ICO、SVG。命令行使用非常簡單，格式是 magick 待轉換圖片路徑 輸出圖片路徑.後綴名，舉例來說，將 A.png，轉換成後綴為 jpg 的文件，在文件所在目錄打開終端，命令為：

 $ magick A.png A.jpg

對於 Windows 用戶，可以使用 clipboard: 變量表示剪貼版，下面這個命令表示將剪貼板中的圖片輸出為 B.jpg：

 $ magick clipboard: B.jpg #執行此命令請確認剪貼板首位是圖片

你也可以使用一些參數對圖片進行處理，例如，-resize 表示縮放，-quality 表示輸出質量，下面這行命令將 A.jpg 縮小 50%，並以 50% 的質量輸出，對圖片進行了壓縮；

 $ magick A.jpg -resize 50% -quality 50 B.jpg

音頻、視頻：FFmpeg

鼎鼎大名的 FFmpeg 是許多視頻播放、剪輯軟件的基石。軟件本身也能完成視頻的錄製、轉換、串流等一系列工作，你可以從 官方網站 下載安裝包，但同樣推薦使用軟件包管理器安裝，只需要把前文 install 後面的參數改成 ffmpeg 即可。

下面這行命令把 A.mkv 轉換成為了 A.mp4，這樣才能拖進某些剪輯軟件的媒體庫進行剪輯：

 $ ffmpeg -i A.mkv A.mp4

「-i」後面輸入待轉換文件，最後填上輸出文件名稱、格式即可。除了視頻格式以及音頻格式的的轉換，你還可以將視頻文件轉換為音頻文件，這樣就可以不用在剪輯軟件中「分離音頻軌」然後單獨保存了，在處理一些 MV 或者語言類節目時非常實用:

 $ ffmpeg -i .\Revenge.webm demo.mp3 #將 Revenge.webm 這個 MV 轉換成了 mp3 歌曲

文本：Pandoc

想必你可能在其他文本處理軟件中看到過它的身影，這是一款專門轉換文本格式的命令行工具。支持的格式非常之多，常用的有 txt、docx、html、pdf、md、pptx，官方還畫了一個巨大網狀圖詳細展示了其所有支持的格式。下載地址在這裡。

轉換命令為 pandoc 待轉換文件 -o 輸出文件.後綴名 , 下面兩行命令，將 sspai.md 轉換成了 word 文檔，又把 word 文檔轉換成了網頁文件：

 $ pandoc sspai.md -o sspai.docx $ pandoc sspai.docx -o sspai.html

軟件還支持讀取網頁並轉換成其他格式，例如，將少數派編輯器使用指南這篇文章轉換為本地的 sspai.md，會保留頁眉頁腳等信息，對於下載一些結構簡單的博客十分有用：

 $ pandoc https://sspai.com/post/37790 -o author.md

PDF 格式處理需要 LaTeX 輔助，可以按需求下載。但不建議使用 Pandoc 進行 PDF 轉換，體驗不太好。簡單需求使用電腦自帶的打印功能即可。

結語

此外，這些工具都支持批量文件處理，可以使用一些通配符（*，% d 等）表示待轉換文件，這樣就能快速完成一些重複的工作。

文本介紹的是這些軟件初級的用法，對於輕量使用已經足夠了，複雜需求依靠有 UI 的專業軟件會更好，他們可調試，可實時預覽效果，體驗更佳。如果想要學習進階內容，可以翻看官方文檔。
