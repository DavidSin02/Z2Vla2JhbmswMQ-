JAVA深入學習（棧和隊列）棧應用之單詞逆序

2020-10-29
| 科技

前面的文章我們講了JAVA深入學習（棧和隊列）之棧 ，並且用數組實現了棧的先進後出。現在我們做一個棧的示例應用-單詞逆序。

題目

棧的第一個例子是做一件非常簡單的事情：單詞逆序。運行程序時，提示輸入一個單詞，回車鍵【Enter】後，便會顯示字幕順序倒置後的詞。

用棧進行單詞逆序：首先，字幕從輸入的字符串中一個接一個地讀取出來並壓入棧中。接著它們依次彈出棧，並顯示出來。因為棧的先進後出的特性，所以字母的順序就顛倒過來了。

代碼

棧代碼：

/** * Created by Bruce on 2020/8/17 **/public class StackOfChar_02 {    private int maxSize;    private char[] stackArray;    private int top;    public StackOfChar_02(int maxSize) {        this.maxSize = maxSize;        stackArray = new char[maxSize];        top = -1;    }    /**     * 是否為空     * @return     */    public boolean isEmpty(){        return top == -1;//棧頂指針為默認值-1，則代表為空    }    /**     * 棧是否已滿     * @return     */    public boolean isFull(){        return top == maxSize - 1;//棧頂指針為限制數量-1    }    /**     * 向棧中壓入元素     * @param value     * @return     */    public boolean push(char value){        if (isFull()){//已滿壓入失敗//            return false;//實際使用中-可拋出異常處理            throw new RuntimeException("stack is full");        }        stackArray[++top] = value;//棧頂壓入-top是在插入數據項之前遞增的        return true;    }    /**     * 彈棧     * @return     */    public char pop(){        if(isEmpty()){//棧為空//            return -1;//實際使用中-可拋出異常處理            throw new RuntimeException("stack is empty");        }        return stackArray[top--];//先返回再遞減    }    /**     * 查看棧頂元素     * @return     */    public char peek(){        if(isEmpty()){//棧為空返回-1；//            return -1L;//實際使用中-可拋出異常處理            throw new RuntimeException("stack is empty");        }        return stackArray[top];    }}

逆序操作代碼：

import java.io.BufferedReader;import java.io.IOException;import java.io.InputStreamReader;/** * Created by Bruce on 2020/8/17 **/public class Reverser {    private String inputStr;    private String outputStr;    public Reverser(String inputStr) {        this.inputStr = inputStr;    }    public String doRev(){        if(inputStr == null || inputStr.length() <= 0){            return null;        }        int length = inputStr.length();//獲取輸入字符串長度        StackOfChar_02 stack = new StackOfChar_02(length);//按照字符串長度初始化棧空間大小        for(int i = 0; i < length; i++){            stack.push(inputStr.charAt(i));//入棧        }        outputStr = "";        while (!stack.isEmpty()){//棧不為空時候，循環彈棧            outputStr = outputStr + stack.pop();        }        return outputStr;    }    /**     * 獲取鍵盤輸入流     * @return     * @throws IOException     */    public static String getSystemInputString() throws IOException {        InputStreamReader in = new InputStreamReader(System.in);        BufferedReader bufferedReader = new BufferedReader(in);        return bufferedReader.readLine();//只獲取一行輸入//        String finalStr = "";//        String temp;//        while ((temp = bufferedReader.readLine()) != null){//            finalStr = finalStr + temp;//        }//        return finalStr;    }    public static void main(String[] args) throws IOException {        String input,output;        while (true){            System.out.println("Enter a String:");            System.out.flush();            input = getSystemInputString();            System.out.println("inputStr:" + input);            if(input == null || input.length() <= 0 || input.trim().equals("")){//輸入數據不能為空，為空則中斷                break;            }            Reverser reverser = new Reverser(input);            output = reverser.doRev();            System.out.println("Reverser outputStr:" + output);        }    }}

示例輸出：

bruce

inputStr:bruce

Reverser outputStr:ecurb

Enter a String:

bruce and tom

inputStr:bruce and tom

Reverser outputStr: mot dna ecurb

Enter a String:

inputStr:

Process finished with exit code 0

代碼解析

建立Reverse類來處理輸入字符串的逆序工作。該類的關鍵組成部分是“doRev（）”方法，該方法利用棧實現逆置操作。在“doRev()”方法中創建了一個棧，它根據輸入字符串的長度確定棧的大小。

“main()”方法中由用戶輸入一個字符串，創建Reverser對象，字符串作為參數傳給構造方法，接著調用這個對象的“doRev()”方法，並顯示返回值，這個返回值是逆序的字符串。
