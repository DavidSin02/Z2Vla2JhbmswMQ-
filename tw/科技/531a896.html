CivilFEM薄殼結構五種不同網格劃分實例

2020-10-29
| 科技

幾何結構

CivilFEM幾何模型


結構元素： 薄殼結構


網格劃分一：Edge Length with Mesh Transition Parmeter: 1

網格定義參數：

Mesh control: Edge length

Element Type: Triangle

Type of algorithm: Advancing front

Mesh transition parameter: 1

Edge size: 1m

網格劃分結果：

網格劃分二: Edge Length with Mesh Transition Parameter: 2

網格定義參數：

Mesh control: Edge length

Element Type: Triangle

Type of algorithm: Advancing front

Mesh transition parameter: 2

Edge size: 0.3m

網格劃分結果：


Mesh Transition Parameter值表示從中部網格尺寸和邊緣尺寸的比值，大於1，中部網格尺寸大於邊緣網格尺寸；小於1，中部網格尺寸小於邊緣網格尺寸。


網格劃分三: No. of Division, Overlay Meshing with no bias

網格定義參數：

Mesh control: Number of division

Element Type: Quadrilateral

Type of algorithm: Overlay quad

Divisions U direction: 40

Divisions V directin: 20

Bias Factor U: 0

Bias Factor V: 0

網格劃分結果：


網格劃分四: No. of Division, Overlay Meshing with bias

網格定義參數：

Mesh control: Number of division

Element Type: Quadrilateral

Type of algorithm: Overlay quad

Divisions U direction: 40

Divisions V directin: 20

Bias Factor U: 0.5

Bias Factor V: 0

網格劃分結果：


Bias Factor為網格細分偏差係數，在-0.5到0.5之間，正值表示正方向上最末端網格越密，0值表示網格在該方向上均布。

Bias Factor V: 0.5

Bias Factor V: -0.5

Curvature Check

Max deviation: 0.1

Min element size: 0.2
