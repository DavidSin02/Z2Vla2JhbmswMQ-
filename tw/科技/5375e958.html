「C語言」格式化輸入與輸出

本文通過函數原型--參數--返回值--實例的順序一一解析C語言中常用輸入輸出函數。首先對在格式化輸入輸出中經常出現的控制格式以表格形式列出。

格式控制字符

以printf為例： printf（“%[falgs][width][.prec ][hIL]type”）

flags

width或.prec

hIL

type

轉義字符

格式化輸出

1.printf()函數

函數原型：

int printf(const char *format, ...);

函數說明：

printf函數的調用格式為：printf("<格式化字符串>", <參量表>);

格式化字符串(format)中包含兩部分：一是正常字符，按照原樣輸出；二是以%開始的可控字符，由參量表中的內容決定。參量表就是一系列需要輸出的參數，可以有多個參數，各個參數之間用","分開，並且與格式化字符串中的可控字符的個數一一對應。

返回值：對於絕大部分初學者來說幾乎沒人去關心printf的返回值，甚至說有一部分人可能都不知道它有返回值，但是從函數原型中我們可以看到它確實是有的，printf函數的返回值就是輸出的字符數量，該數量包括數字，字母，標點符號，空格和換行等。

下面通過一個例子來說明，想必對printf函數就會很清楚了，同時也會對同時也會對上面的控制字符有清晰地認識。

#include <stdio.h>

​

int main(void)

{

printf("%3d\n",12345);

printf("%9d\n", 123);

printf("%09d\n", 123);

printf("%-9d\n", 123);

printf("%+9d\n", 123);

printf("%-+9d\n", 123);

printf("%+-9d\n", 123);

printf("%9.2f\n", 123.0);

printf("%*d\n", 6, 123);

//測試printf的返回值

int num = 12345;

int i1 = printf("%d\n", num);

int i2 = printf("%d", num);

printf("\ni1 = %d\ti2 = %d\n", i1, i2);

​

return 0;

}

運行結果：


結果分析：

通過前面的表可以知道，width中的number是最小的字符數，它的意思是：如果實際輸出的位數不夠，則會補齊位數，如果實際輸出位數比它大，則原樣輸出，如運行結果中的第一二行。其它的大家應該都能看懂，我來解釋一下printf("%*d\n", 6, 123); * 表示下一個參數是字符數，於是這個語句其實等價於printf("%6d\n", 123);通過運行結果我們也可以看出來，最後確實是輸出了6個字符。那麼這樣做有什麼好處呢？這樣做的話，我們就可以通過使這個參數為變量，從而讓整個程序更為靈活，而且這個變量也可以讓用戶自己輸入。即代碼變成printf("%*d\n", number, 123);。最後我們來看printf的返回值，我們知道printf返回的是輸出的字符個數，i1=6,i2=5，是因為i1多打印了一個換行符，所以如果你想使用printf的返回值必須對換行符製表符加以注意，因為那也是一個字符。

2.sprintf函數

函數原型：

int sprintf(char *str, const char *format, ...);

函數說明：

sprintf函數的主要功能是把格式化的數據(format)寫入某個字符串中，即發送格式化輸出到string所指向的字符串。

函數調用形式：sprintf(str, "<格式字符串>"， <參量表>)；

從函數原型中可以看出sprintf和printf十分相似，對於相同的部分，在此不再贅述。不同之處在於多了一個指向char類型的指針str，通常情況下我們會定義一個字符數組，它的作用是存儲字符串。

返回值：函數調用成功則返回字符串長度（這個長度並不包括字符串末尾的空字符），如果失敗則返回一個負數。注意：使用該函數要避免堆棧溢出，或選擇使用更為安全的snprintf函數。

下面進行代碼分析：

#include <stdio.h>

​

int main(void)

{

printf("%3d\n",12345);

printf("%9d\n", 123);

printf("%09d\n", 123);

printf("%-9d\n", 123);

printf("%+9d\n", 123);

printf("%-+9d\n", 123);

printf("%+-9d\n", 123);

printf("%9.2f\n", 123.0);

printf("%*d\n", 6, 123);

//測試printf的返回值

int num = 12345;

int i1 = printf("%d\n", num);

int i2 = printf("%d", num);

printf("\ni1 = %d\ti2 = %d\n", i1, i2);

​

return 0;

}

結果分析：

運行的結果如我們所設想的那樣，123被存放到str的數組中，並且函數的返回值也正是字符個數。對於不同的打印格式這裡就不再演示，可以參照printf函數部分的代碼。

注意，請讀者試著編譯運行下方代碼，觀察運行結果是否如你所想

sprintf(str, "%.2f", 123);

printf("str = %s", str);

3.fprintf函數

函數原型：

int sprintf(char *str, const char *format, ...);

函數說明：

fprintf函數主要作用是格式化輸出到一個流文件中，即將指定格式(format)，輸入到stream中去。

函數調用形式：sprintf(str, "<格式字符串>"， <參量表>)；

通過函數原型我們可以知道fprintf函數與sprintf函數很相似，區別就是sprintf是向字符數組寫入數據，fprintf則是向文件寫入數據，具體的實現方法，則是通過向指向FILE的指針，寫入數據。

返回值：成功返回輸出的字節數，失敗返回EOF(end of file)。

下面進行代碼分析：

#include <stdio.h>

​

int main(void)

{

int x = 123;

int i = 0;

char str[10] = {0};

i = sprintf(str, "%d", 123);

​

printf("str = %s\ti = %d\n", str, i);

​

return 0;

}

//運行結果：

str = 123 i = 3

運行結果：

結果分析：

123456被成功的輸入到123.txt文件中了, 函數的返回值i=6。對於fopen函數可能有些人不是很瞭解，由於本文主要說的是格式化輸入輸出函數，所以不會在此展開，後面會專門寫一篇關於文件操作的函數。

格式化輸入函數

1.scanf函數

函數原型：

int fprintf(FILE *stream, const *format, ...);

函數說明：

scanf函數是從標準輸入流中讀取內容，並將其保存在對應的地址中。

函數調用形式：scanf("<格式化字符串>"，<地址表>);

scanf函數和printf函數是很相似的，對於初學者來說，它們也是使用率極高的兩個函數，所以他們的使用方法都大致相同，在此提一下，在寫地址標的變量時，一定不要忘記取址符&，同時它還有一個比較特殊但是很有用的格式控制字符[]，它的作用是掃描字符集合，具體後面會上代碼進行解釋。

返回值：返回成功讀入的數據個數，如在讀入數據時遇到“文件結束”則返回EOF。

其實關於scanf函數可以說的有很多，但是由於篇幅原因，本文會選擇經常容易出現的錯誤來舉例，不會說得十分詳細，有興趣的小夥伴可以自行查閱資料，或者在下方進行評論。

下面進行代碼分析_1：

#include <stdio.h>

​

int main(void)

{

int i;

FILE *stream = fopen("123.txt", "w");

i = fprintf(stream, "%d", 123456);

printf("i = %d\n", i);

return 0;

}

//運行結果

i = 6

結果分析：

scanf函數會在操作中略去一個或多個空白符，直至出現第一個非空白符為止，所以以上輸入方式都是可以的

此外你可能會有所疑問，如果我想輸入一段英文文字，那麼它是如何判定我這個是一句話還是一段話呢？答案是無法判定，scanf函數會自動略去空白符，所以它就只能讀取你的第一個單詞，那麼這樣一來我們是不是就無法輸入一段文字呢？答案當然是可以輸入的，這是就需要用到前面說的一種控制字符[]，具體的效果及使用方法看代碼。

2.sscanf函數

函數原型：

int sscanf(const char *buffer, const char *format,...);

函數說明：

sscanf與scanf函數類似，區別在於前者是以指定字符串作為輸入，後者是以鍵盤作為輸入。

函數調用形式：sscanf(<存儲的數據>, "<格式字符串>"， <存儲的位置>)；

返回值：返回成功賦值的個數，如果在第一次讀取之前到達字符串結尾則返回EOF，失敗返回-1。

演示代碼：

#include <stdio.h>

int main(void)

{

char str[100];

sscanf("123abc", "%5s", str);

printf("%s\n", str);

return 0;

}

//運行結果：

123ab //%5s限定了字符串的長度。

3.fscanf函數

函數原型：

int fscanf(FILE *stream, const char *format,...);

函數說明：

fscanf與sscanf函數類似，區別在於前者是以文件作為輸入，後者是以指定字符串作為輸入。

函數調用形式：fscanf(<輸入流>, "<格式字符串>"， <存儲的位置>)；

返回值：返回成功讀入參數的個數，失敗返回EOF(-1)。

演示代碼：

#include <stdio.h>

int main(void)

{

char str[30];

int x;

FILE *fp = fopen("1234.txt", "r");

fscanf(fp, "%s %d", str, &x);

printf("str = %s\tx = %d\n", str, x);

return 0;

}

//運行結果：

預先在1234.txt文件中輸入瞭如下內容：123456 789

程序運行的結果;

str = 123456 x = 789

注：本文的六個函數頭文件均為<stdio.h>。
