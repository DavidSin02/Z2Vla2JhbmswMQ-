Linux Kernel 5.8 發佈，華為在內核代碼貢獻上排名第二

2020-10-29
| 科技

IT之家 9 月 1 日消息 近日，Linux Kernel 5.8 版本正式發佈，Linus 表示 Linux Kernel 5.8 是 “有史以來最大的發行版之一”。Linux Kernel 5.8 在 ARM64 架構特性方面有不少的更新。

openEuler 社區發文表示，華為 92 個社區內核工程師貢獻包括：ARM64 SPE perf event、ACPI CPPC 支持 ARM64 CPU 超頻，以及虛擬化熱遷移頁標髒優化（128G 4K 頁標髒從 650ms 優化到 1.8 ms），CPU 休眠調控器默認可根據場景調整等等特性，與此同時，華為在 Linux Kernel 5.8 中的代碼貢獻（changesets）、代碼修改行（line changed）和內核缺陷發現方面，都交出了答卷。

從公司貢獻角度來說，華為提交的補丁數量位列第二名，佔比 8.6%，代碼修改行位列第一，佔比 27.8%。

在華為貢獻的 1399 個補丁中，除前文中提及的系統特性外，還有包括網絡、文件系統、perf 調測、安全等關鍵子系統的 200+ 的 bugfix 補丁，這些都是 Linux 能夠正常運行所需的基礎軟件系統，對於 Linux 是非常重要的。

openEuler 社區表示，在內核測試和 Bug 提交方面，華為自動內核缺陷發現機器人 HULK Robot（Huawei Unified Linux Kernel Robot）表現出色，在 Linux Kernel 5.8 版本中繼續霸榜。

HULK Robot 融合了大數據機器學習和語義分析技術，它包含了海量的測試數據，供 HULK Robot 進行學習，同時集成基於場景語意的模糊測試技術、全系統函數級故障注入配合精準的單元測試，使 HULK Robot 成為一個高效，精準的可擴展測試系統，大幅提高測試效率和問題檢出效率。

另外，IT之家瞭解到，華為工程師 Mauro Carvalho Chehab 作為 Media，EDAC Maintainer ，雙貢獻排名全球第一。

其中第一名是來自華為的 Mauro Carvalho Chehab，Mauro 無論在提交的補丁還是代碼修改行，都位列第一名，佔整個補丁的 3.4%，佔整個代碼修改行的 25.8%。
