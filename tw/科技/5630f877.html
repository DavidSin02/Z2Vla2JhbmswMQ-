這種對講機不要操作證也不用執照，普通人也能用的公眾對講機介紹

2020-11-14
| 科技

最近很多朋友諮詢小汪，玩對講機都需要考操作證辦理執照嗎？有沒有可以不需要操作證和執照的對講機呢？你別說，還真有，那就是公眾對講機。所謂公眾對講機是指，發射功率不大於0.5W，工作於指定頻率的無線對講機。

（小米米家對講機包含公眾頻率）

目前，在我們國家設置和使用公眾對講機，不需領取電臺執照 ， 免收頻率佔用費。但是公眾對講機是禁止在機場和飛行器上使用的 。 還有就是公眾對講機只能和公眾對講機進行通訊，公眾對講機是不能和業餘無線電臺以及其他非公眾對講機頻率的電臺進行通訊的。就像我們業餘無線電臺不能和非業餘無線電臺進行通聯是一個道理。

業餘無線電呼號

目前公眾對講機的工作頻率為（單位：MHz）: 409.7500 ; 409.7625 ; 409.7750 ; 409.7875 ; 409.8000 ; 409.8125 ; 409.8250 ; 409.8375 ; 409.8500 ; 409.8625 ; 409.8750 ; 409.8875 ; 409.9000 ; 409.9125 ; 409.9250 ; 409.9375 ; 409.9500 ; 409.9625 ; 409.9750 ; 409.9875 （共20個頻點）。

參數方面要求：調製方式：F3E；有效發射功率（EIRP ）：< 0.5W；發射頻率容限：<+5ppm；發射機雜散輻射：<50uW；接收機雜散輻射：<20nW；信道間隔：12.50kHz

使用人或使用單位只有在購買了符合規定的公眾對講機時，才可以不必領取執照、考取操作證或者交納頻率使用費，但此處的公眾對講機是不能和非公眾頻率進行通聯的，大家務必要注意。

小米米家對講機和泉盛UV2對講機

其他一些需要注意的地方：

首先就是大家在選購公眾對講機的時候一定要選擇有核准碼的對講機，購買之前一定要諮詢商家。其次就是已經國家無線電管理機構型號核准的公眾對講機，任何單位和個人不得擅自更改使用頻率、加大發射功率(包括額外加裝射頻功率放大器)，不得擅自外接天線或改用其它發射天線，或改變原設計特性及功能。最後就是針對0.5瓦的功率，有的朋友會質疑會不會太小，在這裡小汪要告訴大家，對講機的通聯距離跟功率確實是有一定的關係，但是隻要環境好，通訊電臺之間沒有阻擋，0.5瓦也是可以通聯很遠的距離的！最後祝大家通聯愉快，Happy New Year！73！
