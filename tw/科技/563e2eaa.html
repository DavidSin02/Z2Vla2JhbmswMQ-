什麼是平均收益？平均收益率與幾何平均值之間的差異

2020-11-14
| 科技

什麼是平均收益?

平均收益是一段時間內產生的一系列收益的簡單數學平均值。平均收益的計算方法與為任何一組數字計算簡單平均的方法相同。將數字加在一起成為一個總和，然後將總和除以集合中數字的計數。

平均收益公式

\ text {平均退貨} = \ dfrac {\ text {退貨總額}} {\ text {退貨數量}}平均回報=退貨數量返回的總和﻿

如何計算平均收益

有幾種回報方法和計算方法，但對於算術平均回報，一種方法是將回報之和除以回報位數。

平均收益告訴你什麼?

平均收益告訴投資者或分析師過去股票或證券的收益是多少，或者公司投資組合的收益是什麼。這與年化收益率不同。平均收益忽略了複利。



重要要點

  1. 平均回報是一系列回報的簡單數學平均值。
  2. 它可以幫助評估證券的過去表現或投資組合的表現。
  3. 幾何平均值始終低於平均收益。

如何使用平均收益示例

平均收益率的一個例子是簡單的算術平均值。例如，假設一項投資在整個五年的時間內每年獲得以下收益：10%，15%，10%，0%和5%。要計算這五年期間的平均投資回報率，將五個年度回報率相加，然後除以5。這將產生8%的年平均回報率。

或考慮沃爾瑪(紐約證券交易所：WMT)。沃爾瑪股價在2014年回報9.1%，在2015年下跌28.6%，在2016年上漲12.8%，在2017年上漲42.9%，在2018年下跌5.7%。這五年間，沃爾瑪的平均回報率為6.1%或30.5%除以5年。

計算增長回報

簡單增長率是起點和終點或餘額的函數。它是通過從起始值中減去終止值，然後除以起始值來計算的。計算公式如下：

\ begin {aligned}&\ text {增長率} = \ dfrac {\ text {BV}-\ text {EV}} {\ text {BV}} \\&\ textbf {其中：} \\&\ text { BV} = \ text {開始值} \\&\ text {EV} = \ text {結束值} \\ \ end {aligned}​增長率=BVBV-EV哪裡：BV=初始價值電動車=結束值﻿

例如，如果您向一家公司投資10,000美元，並且股票價格從50美元增加到100美元，則可以通過將100美元和50美元之間的差額除以50美元來計算回報。答案是100%，這意味著您現在有20,000美元。

平均收益率與幾何平均值之間的差異

當查看平均歷史收益時，幾何平均值是一種更精確的計算。幾何平均值始終低於平均收益。使用幾何平均值的好處之一是不需要知道實際的投資額。該計算完全著眼於收益數據本身，並且在考察兩個或更多個投資在不同時間段內的表現時，呈現出“從蘋果到蘋果”的比較。

幾何平均收益率有時稱為時間加權收益率(TWRR)，因為它消除了隨著時間的流逝各種資金流入和流出而對增長率產生的扭曲影響。

另外，貨幣加權收益率(MWRR)結合了現金流量的大小和時間安排，因此它是衡量已收到存款，股利再投資，利息支付或已提款的投資組合收益的有效度量。貨幣加權收益等於內部收益率，其中淨現值為零。

使用平均收益率的侷限性

簡單的收益平均值很容易計算，但不是很準確。為了更準確地計算收益，分析師和投資者還經常使用幾何均值或貨幣加權收益。
