Python實用案例編程入門：第八章 如何自動連接WIFI

2020-10-29
| 科技

本章的主題為介紹如何自動連接WIFI。如果你們公司也是使用WIFI網絡環境辦公，那麼也有可能遇到同樣的問題。在介紹具體實現之前，會先介紹一些常用的基礎庫，比如sys，time等。

8.1 要解決什麼問題

如果公司的網絡環境是WIFI，那麼隨著筆記本位置的移動，就需要接入就近的無線路由器。正常情況下使用是沒問題的，但是有段時間網絡有點問題，不能很快的正常接入網絡。需要手動重新連接。即使電腦位置不變，也總是會斷網，只有手動重新連接。所以，既然手動能解決的問題，那麼我們為何不寫個程序幫助我們默默的把這件事做了呢？

8.2 實現思路

現在我們需要思考一下如何去實現。一個最簡單的思路就是讓程序模擬我們自己的動作。

我們會去先ping一下百度，如果能ping通，那就表明網絡是通的，否則就表明網絡斷了。一旦檢測到網絡斷了，那麼就需要去進行連接。一個最簡單的方法就是利用dos命令種的netsh命令來管理網絡。

如果覺得上面的辦法太簡易且不專業，那我們可以再用更專業的思路去實現我們的想法。那就是pywifi模塊，不需要去通過ping百度來確定網絡是否正常了。而且代碼也會更靈活，可以配置WIFI名稱密碼等。

用上面思路實現的程序都需要我們手動去執行一次。我們還有辦法，那就是將上面的程序實現為服務，這樣就不用每次啟動都執行一次了。

我們這裡有點過設計的嫌疑，也有點類似茴香豆的茴字有多少中寫法的味道。但從做技術的角度來講，我們需要也應該知道不同方案的優缺點，以便於我們做出更適合我們需求的方案選擇。

8.3 相關模塊的安裝及介紹

本節將會涉及time模塊，sys模塊，subprocess模塊和pywifi模塊。接下來會逐個進行詳細介紹。

8.3.1 time模塊

time模塊提供了時間相關的各種函數。

time.asctime()函數可以將結構體struct_time所代表的時間轉換為這樣的字符串'Sun Jun 19 13:31:15 1994'。我們可以通過time.localtime()函數得到結構體struct_time。

time.sleep()函數的入參單位是秒，如果線程需要被掛起，可以通過調用此函數達到該目的。這裡的入參也可以是小數，表示更精確的睡眠時間。我們將會使用該函數進行必要的等待以確保另一件事情結束。

time.strftime()函數也用於格式化時間。

>>> import time>>> time.strftime('%Y%m%d',time.localtime(time.time()))'20190714'>>>>>> time.strftime('%Y-%m-%d')'2019-07-14'>>>

8.3.2 sys模塊

sys模塊提供了對Python解釋器使用的一些變量的訪問，並可以進行一些修改，例如對環境變量PATH的讀取和修改，並提供了某些和解釋器進行交互的函數以使我們的程序能夠和解釋器進行交互。

例如，sys.argv會將命令行參數以list的形式傳遞給Python腳本，sys.argv[0]是腳本的名字，sys.argv[1]是第一個參數，以此類推。

sys.exit()表示退出程序，也可以帶參數表示退出碼，如果有其他程序調用該程序，即可以通過返回的數字來確定被調用程序的退出原因。

sys.implementation查看當前正在運行的Python解釋器的版本信息。

>>> sys.implementationnamespace(cache_tag='cpython-36', hexversion=50726384, name='cpython', version=sys.version_info(major=3, minor=6, micro=5, releaselevel='final', serial=0))>>>

sys.stdin，sys.stddout，sys.stderr解釋器用於標準輸入，標準輸出和錯誤。

8.3.3 subprocess模塊

subprocess模塊，我們可以利用該模塊創建一個新的進程。如果需要的話，可以通過入口連接到程序的輸入，輸出和錯誤管道，並通過相應的管道獲取程序的輸入，輸出和錯誤返回。

subprocess.run()函數可以運行由參數組成的命令的字符串，並且直到命令完成，在完成後返回一個完成實例。

>>> import subprocess>>> subprocess.run("ping -n 1 www.baidu.com", shell=True)正在 Ping www.a.shifen.com [180.97.33.108] 具有 32 字節的數據:來自 180.97.33.108 的回覆: 字節=32 時間=33ms TTL=55180.97.33.108 的 Ping 統計信息:數據包: 已發送 = 1，已接收 = 1，丟失 = 0 (0% 丟失)，往返行程的估計時間(以毫秒為單位):最短 = 33ms，最長 = 33ms，平均 = 33msCompletedProcess(args='ping -n 1 www.baidu.com', returncode=0)>>>

subprocess.PIPE代表管道是一個特殊的值，使用該值可以將標準流的管道打開。打開後可以通過sys.stdin獲取輸入，通過sys.stdout讀出輸出結果，同時也可以通過sys.stderr得到錯誤輸出。

subprocess.Popen()在新進程中執行子程序，在本書中我們將會多次使用該接口，某些場景下我們不需要等待被調用程序執行完成或返回，只需要將其啟動即可。在預訂測試線章節的例子中，我們將使用該接口去啟動升級切換程序，啟動後無需等待其返回。如果需要等待執行完成，則只需要調用wait()接口等待其完成即可。

import subprocess, osp = subprocess.Popen("dir", cwd=os.getcwd(), shell=True,stdin=subprocess.PIPE, stdout=subprocess.PIPE, stderr=subprocess.PIPE)p.wait()

如上面代碼中使用的一樣，wait()等待子程序結束。同時可以設置等待超時時間。

8.4.2 Python代碼

基於偽碼的思想，我們編寫的第一個簡易版本的程序如下。如我們最初所想的，就是用ping百度的方法來確認網絡是否正常。雖然簡易，但功能沒問題，能解決我們的問題，所以不管黑貓白貓能抓住老鼠的就是好貓。

# -*- coding: utf-8 -*-"""該程序用於重連wifi，避免人手動去重連我們利用dos命令netsh進行重連操作，假設我的網絡名為GGangnetsh wlan connect name=GGang"""from time import sleepfrom subprocess import runwhile True:sleep(3)p = run("ping -n 1 www.baidu.com", shell=True)print(p)if 1 == p.returncode:run("netsh wlan connect name=GGang", shell=True)sleep(5)
