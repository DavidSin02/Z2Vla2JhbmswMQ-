第14節 Javascript函數-零點程序員-王唯

2020-11-14
| 科技

函數對任何語言來說都是一個核心的概念；

函數是完成特定任務的可重複調用的代碼段，是JavaScript組織代碼的部分，其只定義一次，就可以被執行或調用任意次；

可以使用JavaScript內置函數,也可以自定義函數;

函數的功能：

函數的主要功能是可以封裝任意多條語句，而且可以在任何地方任何時候調用執行；其本質就是將代碼組織為可重複使用的部分，可以完成特定的任務並返回數據；其不會主動的運行，需要在程序中手動的調用函數（比如：用戶點擊的時候），並且可以多次調用；

好處：把程序劃分為一些相對獨立的部分,各部分充分獨立，任務單一，易於維護;

定義函數：

在使用函數之前,必須先定義函數,函數定義的位置可以在任何位置；定義函數的方式有三種方式；

第一種：函數聲明式，使用關鍵字function定義，語法:

function funcname([arg0,arg1,…,argn]) {

statements; // 就是被封裝的代碼塊，稱為函數體

[return 表達式;]

}

說明：funcname是函數名，圓括號中的arg是參數列表，參數之間使用逗號分隔；函數體使用花括號；定義的函數不會自動執行，必須經過調用，如：

function showMsg(str){    console.log("零點程序員" + str);}showMsg("開新課了");

函數的命名規則：不能使用保留字，要使用有意義的名稱；

function square(x,y){    return Math.sqrt(x*x + y*y);}console.log(square(3,4));  // 5

第二種函數定義方式：

函數本質上是一個對象，該對象是Function類型的實例，因此可以使用Function對象的構造函數來創建一個函數，並被保存在一個變量（函數名）中；其和其他引用類型一樣具有屬性和方法；由於函數是對象，所以函數名實際上也是一個指向函數對象的指針，不會與某個函數綁定。

除了可以被存儲在變量中，還可被保存在數組以及對象的屬性中,甚至可以作為參數進行傳遞。

語法:

var 變量名 = new Function(“[形參1]”, “[形參2]”,…”代碼體”);

所有的參數必須是字符串;最後一個參數是程序代碼語句;

var myFn = new Function("x","y","var sum;sum=x+y;return sum;");alert(myFn(2,3));

注：從技術角度來說，這就是以下要所說的函數表達式定義方法；不推薦這種方法定義，因為這種方法會導致解析兩次代碼（第一次是解析常規的ECMAScript代碼，第二次是解析傳入構造函數中的字符串），從而影響性能；但這種方式有助於理解“函數是對象，函數名是指針(變量)”的概念；

注：此種定義方法的作用是可以動態創建函數；

第三種：函數表達式：

可以直接把定義函數保存在變量中;該變量名實際上就是函數名；

語法:

var myFun=function([參數1,[參數2],…]){

[語句組;]

}

var sum = function(num1,num2){return num1 + num2;};alert(sum(2,3));

說明：function後面沒有函數名，因為沒有必須使用函數名，因為myFun變量就要以引用該函數；另外，這種定義函數的方式的末尾有一個分號；

由於函數名僅僅是指向函數的指針，所以可以把函數賦值給多個變量，也就是說，一個函數可能會有多個名字；如：

function sum(num1,num2){return num1+num2;}var anotherSum = sum;alert(anotherSum(10,20));  // 30sum=null;alert(anotherSum(10,20));  // 30

說明，不能帶括號，帶括號就代表執行了；此時anotherSum和sum就都指向了同一個函數，因此，anotherSum也可以被調用；即使sum設為null，讓它與函數“斷絕關係”，anotherSum仍然可以正常調用。

var condition = false;if(condition){     function sayHi(){alert("Hi");};}else{    function sayHi(){alert("OK");};}sayHi();

說明：這種寫法在以前會出現許多意想不到的效果；

var sayHi;var condition = true;if(condition){    sayHi = function(){alert("Hi");};}else{    sayHi = function(){alert("OK");};}sayHi();

函數聲明與函數表達式的區別：

函數聲明與函數表達式的區別的就在於聲明的位置上不同；

解析器在執行環境中加載數據時，會率先讀取函數聲明，並使其在執行任何代碼之前可用（可以訪問）；至於函數表達式，則必須等解析器執行到它所在的代碼行，才會真正被解釋執行，如：

alert(sum(2,3));function sum(num1,num2){return num1+num2;}

說明：以上代碼正常運行；因為在代碼開始執行之前，解析器就已經通過一個名為函數聲明提升（function declaration hoisting）的過程，讀取並將函數聲明添加到執行環境中；對代碼求值時，Javascript引擎在第一遍會聲明函數並將它們放到源代碼的頂部；所以，聲明函數的代碼在調用它的代碼後面也沒有問題；

但函數表達式，就不可以；

alert(sum(2,3));var sum = function(num1,num2){return num1+num2;}

說明：會產生錯誤，原因是在於函數位於一個初始化語句中，而不是一個函數聲明；換句話說，在執行到函數所在的語句之前，變量sum中不會保存有對函數的引用；而且，由於第一行代碼就會導致錯誤，實際上也不會執行到下一行；

注：兩者除了以上的區別，其他都是等價的。

注：也可以同時使用函數聲明和函表達式，如：

var sum = function othersum(num1,num2){return num1+num2;}alert(sum(2,3));

注：沒有意義，並且有些瀏覽器會導致錯誤；

定義有參函數：

函數是用來完成一個功能的，但是有時需要有其他數據的參與才能完成某些功能，比如，需要計算兩個數的和，就需要把這兩個數作為參數傳遞到函數中去。如：

function sum(num1,num2){return num1+num2;}alert(sum(2,3));

說明：函數的參數是函數與外界交換數據的接口。外部的數據通過參數傳入函數內部進行處理，同時函數內部的數據也可以通過參數傳到外界;

形式參數：

寫在函數定義中，出現在函數名的圓括號（）裡，就是形參，如：

function 函數名字（形式參數1，形式參數2…）{

// 處理代碼

}

function showMsg(str){    document.write(str);}

說明：如果僅僅是這樣的話，沒有什麼意思，可以改寫一下：

使用函數參數的目的是很多的，比如，根據參數輸出多行多列的數據：

function printStar(row,col){    for(var i=1;i<=row;i++){        for(var j=1;j<=col;j++){            document.write("*");        }        document.write("<br/>");    }}printStar(3,5);

函數調用：

函數定義後,就可以調用;

可以在程序代碼中調用,也可以在事件響應中調用,還可以通過鏈接調用;

語法:

函數名([實參1,實參2,…,實參n])

說明:函數名要與定義函數名相同; 實參的類型,個數,意義及次序原則要與形參相同; 無參數時( )不能省略;

showMsg();  //無返回值調用showMsg("你好");  //無返回值，有參數調用var str=showMsg();  //有返回值；

實參的個數多於形參,則忽略多餘的實參;

實參的個數小於形參,則將沒有接收傳遞值的參數賦undefined；

對於函數的調用，如果該函數成為一個對象的成員屬性，那麼就稱之為方法的調用；如果使用關鍵字new的形式調用，稱為構造函數調用；另外，也可以間接調用；這些內容今後的課程會講到。

函數返回值：

如果函數執行的結果需要被其他語句使用，就需要函數的返回值,可以使用return讓函數返回一個值;

function 函數名(形參1，形參2){

// 處理語句；

return 返回值;

}

function sum(num1,num2){    return num1 + num2;}function sum(num1,num2){    return num1 + num2;    alert("zero"); // 永遠不會執行；}var result = sum(5,10);function diff(num1,num2){  // 一個函數中也可以包含多個return語句    if(num1 > num2){        return num1 - num2;    }else{        return num2 - num1;    }}

注：return只能使用在函數中;

函數在執行完return語句之後停止並立即退出；

函數返回值既可以賦值給變量,也可以參與表達式運算;

返回值類型：一般與return返回表達式類型一致，如果不一致,以函數類型為準.

注：return 語句也可以不帶有任何返回值；在這種情況下，函數在停止執行並返回undefined值；這種用法一般用在需要提前停止函數執行而又不需要返回值的情況下，如：

function say(){    return ;    alert("hi");}

練習：

// 輸出對象的屬性和值function printPro(o){    for(var p in o)        console.log(p + ":" + o[p] + "\n");}var person = {name:"wangwei",age:18,sex:"男",say:function(){}};printPro(person);// 計算兩個笛卡爾座標(x1,y1)和(x2,y2)之間的距離function distance(x1,y1,x2,y2){    var dx = x2 - x1;    var dy = y2 - y1;    return Math.sqrt(dx * dx + dy * dy);}console.log(distance(0,0,3,4));//  封裝一個表格函數function getTable(n,m){    var table = document.createElement("table");    for(var i=1; i<=n; i++){  // 行        var tr = document.createElement("tr");        for(var j = 1; j<=m; j++){            var td = document.createElement("td");            td.innerHTML = " ";            tr.appendChild(td);        }        table.appendChild(tr);    }    return table;}document.body.appendChild(getTable(4,5));

嵌套函數：

在Javascript中，函數可以嵌套在其他函數中，即要函數體內定義新的函數，如：

function outerFunc(){    var i = 0;    function innerFun(){        console.log(i);    }    innerFun();}outerFunc();

嵌套函數只能在內部調用，外部無法使用；內部函數可以訪問嵌套它們的函數的參數和變量，即變量的作用域，後面的課程會專門深入。
