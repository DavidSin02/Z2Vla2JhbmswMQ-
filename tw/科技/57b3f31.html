Configuring Kali Linux

我們將使用官方的卡利 Linux 官方 Iso 提供的進攻安全安裝和配置不同的桌面環境。

Getting ready

首先, 我們將使用攻擊性安全網站上列出的 64位 Kali Linux ISO: https://www.kali.org/downloads/。

對於希望為 VMware 和 Veritalbox 等虛擬機配置 Kali 的用戶, 可以從以下 URL 下載 Linux 的預構建映像: https://www.offensive-security.com/kali-linux-vmware-virtualbox-image-download/。

我們將在本章中使用虛擬映像, 並使用一些其他工具對其進行自定義。我們可以從網站下載, 如下面的屏幕截圖所示:

How to do it...

  1. 雙擊 "虛擬框" 圖像;它應該打開與虛擬框。
  2. 單擊 "導入"。
  3. 啟動計算機並輸入密碼。
  4. 現在, Kali 默認配置為 Gnome 桌面環境。

How it works...

使用預構建的映像, 您無需擔心安裝過程。您可以將其視為現成的解決方案。只需單擊 "運行", 虛擬機就會像普通機器一樣啟動 Linux。

Configuring the Xfce environment

Xfce 是 Unix 和 unix 類平臺的免費、快速和輕量級桌面環境。它是由 Olivier Fourdan 於1996年創辦的。Xfce 最初代表 XForms 公共環境, 但自那時以來, Xfce 已被重寫兩次, 不再使用 XForms 工具包。

Now let's have a look at the configuration of MATE environment.

Configuring the MATE environment

MATE 桌面環境是 GNOME 2 的延續。它使用 Linux 和其他類似 unix 的操作系統的傳統隱喻, 提供了一個直觀而有吸引力的桌面環境。最新版本的 MATE (1.20) 於07-02-2018年底發佈, 增加了很多修復程序, 並升級了主題。

完整的功能列表可以在這裡查看: https://mate-desktop.org/blog/2018-02-07-mate-1-20-released/。

在這個食譜中, 我們將學習如何在卡利 Linux 上安裝 MATE.

How to do it...

  • 我們首先使用以下命令來安裝 MATE 環境:
apt-get install desktop-base mate-desktop-environment
  • 當它要求確認額外的空間要求時, 鍵入 Y。
  • 安裝完成後, 我們將使用以下命令將 MATE 設置為默認環境:
update-alternatives --config x-session-manager

  • 註銷並再次登錄, 我們將看到 MATE 環境:

現在, 讓我們來看看 LXDE 環境的配置。

Configuring the LXDE environment

LXDE is a free open source environment written in C using the GTK+ toolkit for Unix and other POSIX platforms. LXDE stands for Lightweight X11 Desktop Environment.

LXDE is the default environment for many operating systems, such as Knoppix, Raspbian, and Lubuntu.

How to do it...

  • 我們開始使用下面的命令來安裝 LXDE:
apt-get install lxde-core lxde
  • 當它要求確認額外的空間要求時, 鍵入 Y。
  • 安裝完成後, 打開 "終端" 窗口並鍵入以下命令:
update-alternatives --config x-session-manager

Configuring the E17 environment

啟蒙運動, 也稱為 E, 是 X Windows 系統的窗口管理器。它最早於1997年發佈。它有很多功能, 如參與、虛擬桌面和平鋪。

How to do it...

  • 由於兼容性問題和有關依賴關係的麻煩, 最好直接從以下 URL 下載 Kali 與 E17 環境: https://www.kali.org/downloads/。
  • 設置它的步驟很簡單: 我們只需雙擊並在 Ververalbox 或 VMware 中啟動 VM 即可。

Configuring the KDE environment

K 桌面環境 (KDE) 是 UNIX 工作站的開源圖形桌面環境。它最初被稱為 Kool 桌面環境。Matthias Ettrich 於1996年首次啟動 KDE 項目, 目標是使 UNIX 平臺更具吸引力和更易於使用。在這個菜譜中, 我們將學習如何在卡利上設置 KDE。

How to do it...

  • 我們使用以下命令來安裝 KDE:
apt-get install kali-defaults kali-root-login desktop-base kde-plasma-desktop
  • Type Y when it asks for confirmation on additional space requirements.
  • Click OK on both the windows that pop up.
  • When the installation is complete, we open a Terminal window and type the following command:
update-alternatives --config x-session-manager

  • Log out and log in again, and we will see the KDE environment:
