如何設置用戶只能通過網頁只讀訪問 ProjectWise？

2020-11-14
| 科技

在平常的項目中，往往有一些項目或文件的安全級別比較高，只允許外部用戶通過只讀權限進行訪問，而不能在 ProjectWise 中進行創建、修改或刪除操作，如果通過設置權限的方式進行設置或歸類，工作量又比較大，還可能出現誤操作。因此，ProjectWise 提供了一個設置，即針對某些特定用戶，只允許其通過 ProjectWise 的 Web View Server 進行登錄，而不能通過客戶端或 Web Server 登錄。


Web View Server 的功能與 Web Server 有什麼不一樣的呢？

它們都是網頁端，都是通過瀏覽器就可以訪問 ProjectWise 數據。但使用 Web View Server 登錄的用戶，對於 ProjectWise 中的文件只有只讀功能。


想要使用 Web View Server，需要我們的管理員先在服務器IIS中部署 ProjectWise Web View Server。


部署完成後，當用戶通過 Web View Server 網頁訪問時，權限會自動變成只讀權限。


在 ProjectWise 當中，用戶是可以通過多種終端訪問到 ProjectWise。如果只需要某個特定用戶只能通過 Web View Server 網頁訪問到到 ProjectWise，就還需要在 ProjectWise 的管理員端，對其賬號進行設置。


設置完成後，該用戶就無法通過 ProjectWise 客戶端和 Web 端進行登錄了，而只能通過 ProjectWise 的 Web View 端進行訪問。
