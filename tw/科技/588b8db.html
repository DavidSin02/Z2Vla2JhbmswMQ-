一文讀懂國產操作系統核心邏輯，3大核心廠商獨享千億成長紅利

2020-10-29
| 科技

操作系統是計算機的靈魂，目前國外操作系統品牌幾乎壟斷了巨大的中國市場，其中在桌面端、移動端的市佔率分別超過94.75%、98.86%。在開源操作系統生態不斷成熟的背景下，中國的國產操作系統依託開源生態和政策東風正快速崛起，市場潛力巨大，未來發展前景值得期待。

操作系統三十年格局演進。操作系統因計算機而生，並隨著計算技術、網絡技術與通信技術的發展而不斷迭代更新。從硬件技術和網絡通信變革角度，可將操作系統進化史劃分為四個階段：第一階段，政企軍工；第二階段，微機桌面；第三階段，移動互聯，受益於3G/4G的快速發展，手機與平板等移動設備迅速普及，由此催生了Android與iOS等優秀的操作系統；以及隨著5G商用落地，信息時代正在迎來的第四個階段，萬物互聯（A-IoT）。

屢敗屢戰的本土化操作系統

隨著數字與信息化推進，網絡信息安全已經成為大國角逐的核心競爭要素。操作系統作為軟硬件紐帶，在安全領域扮演著核心地位，發展本土化操作系統，是國家防範網絡攻擊與威脅需要直接面對的問題。典型如2008年微軟黑屏事件、2010年Stuxnet震網病毒事件、2013年稜鏡門與2017年Intel芯片Minix洩漏風險。

政策支持。當前，無論是服務器，還是桌面與移動端，中國自主研發的操作系統都只佔有著非常低的水平，亦主要侷限於政府等領域。2011年以來，國家各個層面出臺一系列政策，支持推進國產自主化建設應用，鼓勵在操作系統、數據庫、中間件和辦公軟件等領域的研發，為操作系統國產本土化提供了政策基礎。

二十年操作系統國產本土化之路

中國操作系統本土化始於20世紀末，並多以 Unix/Linux 為基礎二次開發為主，過去20年，曾誕生超過20多個不同版本，較為市場熟知的有紅旗 Red Flag、深度 Deepin、優麒麟 Ubuntu Kylin、中標麒麟 Neokylin、銀河麒麟Kylin與中科方德等。但是，由於中國軟件市場的開放、微軟系統生態的攻勢與知識產權等問題，本土化操作系統在市場倖存下來者寥寥無幾，且多數系統僅限於政府領域，並以政府補貼為主，支持研發與應用，國產化之路依然坎坷。

Xteam Linux

1999年4月8日，中國第一款基於Linux/Fedora的國產操作系統Xteam Linux 1.0 發佈，開啟操作系統國產化之路，系統發行售價48元，受到市場廣泛關注。次年底，發行主體北京衝浪軟件在港交所掛牌上市，並在24個交易日內上漲265.79%。但是，由於沒有形成應用生態，支持並佔據市場，Xteam Linux系統於2003年5-pre版本後宣佈停更，股價也應聲隨之下跌，達90%，目前公司已經退出操作系統領域，且股價長期低於0.10元，第一次嘗試以失敗告終。

藍點 Blue Point

1999年9月7日，深圳信科思公司（藍點）成立，起源於一個著名的內核黑客小組OpenUnix Network Studio，該小組包含了一些在中國最為著名的Unix/Linux優秀人物。10月1日，公司發佈了基於紅帽Red Hat的BLPT Linux 1.0，成為深圳市高交會11大推薦項目之一。12月8日，藍點與TCL建立合作，在主打產品全部預裝藍點Linux，月出貨量達15萬套，成為中文Linux最大供貨商，佔國內市場80%以上。

2000年3月7日，公司 BluePoint Linux Software（BLPT） 借殼美股上市，上市第一天即從4美元暴漲400%以上至22美元，同年8月，Linux產品榮獲 “Linux操作系統市場佔有率第一品牌獎”。遺憾的是，2000年，互聯網金融泡沫破滅，在資本市場僅僅風雲兩年後，系統停更，從美國市場黯然退出。

紅旗 Red Flag

2000年6月，中科院軟件研究所和上海聯創共同組建北京中科紅旗，注資96萬美元。併發布桌面版、工作站版、數據中心服務器版、HA集群版和紅旗嵌入式Linux等產品。然而，2013年4月，公司爆出資金鍊斷裂問題，2014年解散清算。8月，五甲萬京收購中科紅旗，使紅旗品牌得以延續。

中標麒麟/銀河麒麟/優麒麟/湖南麒麟 Neokylin/Kylin/Ubuntu Kylin/Kylinsec

麒麟家族中的中標麒麟、銀河麒麟、優麒麟與湖南麒麟皆脫胎於國防科技大學，具有相同的歷史淵源，前二者主要針對我國政府機構設計研發；優麒麟則是2013年由天津麒麟主導開發，面向全球的開源項目；湖南麒麟成立於2007年，依託國防科大計算機研究院，於2014年後獨立開發Kylinsec品牌。

中標麒麟Neokylin，是在中標軟件和國防科技大學的支持下，於2010年12月由民用“中標Linux”與“銀河麒麟”正式在上海合併而成，並共同以“中標麒麟”的新品牌統一出現在市場，開發軍民兩用的操作系統。由於其技術積累和背景特殊，因而中標麒麟系統在我國國防、航天、電力、能源、政務等眾多重要行業得到廣泛的推廣和應用，並多年成為我國Linux市場佔有率第一的操作系統。

銀河麒麟Kylin，是由國防科技大學研製的開源服務器操作系統，始於2001年，是863計劃重大攻關科研項目，主要面向軍用服務器領域。2010年與“中標Linux”合併，後2014年 12月，濱海新區人民政府與國防科技大學在中國電子信息集團（CEC）支持下，成立天津麒麟信息技術有限公司，將湖南麒麟旗下的“麒麟”與“銀河麒麟”等無形資產注入天津麒麟，原有團隊整體轉入，銀河麒麟主要優勢仍在服務器領域，兼顧雲計算與雲桌面操作系統。

優麒麟 UbuntuKylin本身與麒麟系統沒有直接關係，是Ubuntu社區中面向中國用戶開發的Ubuntu衍生版本。項目首先是Ubuntu的一個官方版本，其次，系統也背靠國防科大和天津麒麟，銀河麒麟的社區版。

湖南麒麟 Kylinsec是湖南麒麟信息工程技術有限公司2007年依託國防科技大學計算機學院成立設計的操作系統。公司長期致力於信息安全研發，在集中管控和機要密碼等領域有一定的影響力。2014年天津麒麟成立，操作系統部分整體遷入銀河麒麟。目前，作為民企，湖南麒麟獨立研發操作系統Kylinsec，在政務、能源、金融等國家關鍵行業和領域有一定部署與應用。

中科方德 Delix，是由中科院軟件所研製，重點應用於電子政務、國防軍工、教育、能源交通等重點行業的操作系統。中科方德軟件有限公司成立於2006年，是“基礎軟件國家工程研究中心”項目的法人單位。公司依託大股東中科院軟件所在操作系統方向的技術積累，在“核高基”重大專項等支持下，積極突破系統安全性、虛擬化技術、大數據處理等方向技術，並建立成熟的服務器、安全及桌面操作系統。在政府體系中具有較高市場份額。

深之度 Deepin，是武漢深之度科技有限公司開發的一款基於Linux/Debian的本土開源操作系統。其前身Hiweed（曾用名）發行於2004年7月22日，且在過去16年中，平均每年更新一個版本。

據官方數據顯示，截至2018年，深度操作系統累計下載超過8000萬次，提供32種不同的語言版本，以及遍佈六大洲33個國家105個鏡像站點的升級服務。並且在全球開源操作系統排行榜（DistroWatch）上，長期保持前12名，是中國民用市場最為成功的本土化桌面操作系統之一。

大變局：國產操作系統行業生態展望

政府示範效應，民用替代空間巨大

從伊朗震網事件，到斯諾登稜鏡門事件，再到中美貿易摩擦對高端技術的鉗制，加強信息領域安全自主發展已成為中國發展毋庸置疑的選擇。如果從用戶群體作了劃分，可使用對象分為政府（2G）、企業（2B）和消費者（2C）業務。其中，政府業務又首先涉及安全與保密領域，尤為關鍵，也是打開本土化操作系統的首要切入口，因此，我們就三個領域對市場空間作一預測與分析。

央企國企，跟進第二輪替換

除了政府事業單位，中央企業與地方國企同樣存在安全方面考量與替換空間，根據Wind數據，截至2013年，全國中央與地方國有企業從業人員分別達到1834.90萬和2084.4萬人，且2005年至2013年，人員數量呈現穩步增長趨勢。假設現階段全國央地國企員工數量為4000萬，按1.5比例計算計算機臺數，則桌面臺式計算機數量為6000萬臺，服務器為600萬臺，假設替換其中50%數量，5年內替換完成，則桌面操作系統空間為150億，服務器操作系統為240億元。

民用市場，本土化系統大發展

根據目前PC機21.8% 滲透率，中國存量計算機市場規模約為3億臺，微軟Windows與蘋果OS 市佔率為94.75%，假設未來5年消費市場替換量可以提升5%，以14億人口計算，廠商OEM國產操作系統價格為300元，則國產桌面操作系統市場空間容量為45億元。

另一方面，隨著國民經濟發展，即使全球已經進入移動互聯網時代，與發達國家相比，中國仍在著相當大的滲透率提升空間，假設未來5年滲透率提升5%，其中，新增PC 20%採用國產化操作系統，則國產化空間為42億元。

上游芯片成熟，下游軟件應用豐富

計算技術領域自主安全的發展，不僅對操作系統、應用軟件提出了很高要求，對硬件的可控也是必不可少的條件。隨著國產芯片技術的不斷迭代更新，操作系統上游硬件逐漸成熟，為適配國產操作系統，提供了演進的土壤。

目前，在中科院、CEC/CETC、地方國資以及華為的支持下，國內已經有一系列具備生產能力的芯片廠商，包括龍芯、飛騰、鯤鵬和兆芯等，其基礎架構也幾乎涵蓋了從MIPS、ARM到x86和Alpha所有世界通行版本。

下游應用方面，鉗制操作系統普及的軟件生態也逐漸豐富，並得到開發者重視。以深之度Deepin為例，系統共研發提供28款原生應用，包括但不僅限於深度商店、啟動盤製作、截圖軟件、音樂、影院、看圖、日曆和錄音等日常使用工具應用。

此外，公司還同網易合作開發了網易雲音樂與有道雲筆記Deepin版本；辦公軟件方面，跨平臺，支持Linux環境的金山WPS也已經基本具備替代微軟Office的文字處理能力；其他軟件方面，深度科技通過deepin-wine虛擬環境技術順利安裝了諸如微信、騰訊QQ、同花順和百度網盤等應用。

在政府業務中持續多年拿下市場第一市場份額的中標麒麟，也在加速與軟件應用商進行適配，完善包括人大金倉、武漢達夢和神舟通用在內的國產數據庫，金蝶、中創和普元等中間件以及各類辦公軟件等。

投資建議

目前，在計算機上市公司中，與操作系統密切相關的上市公司主要有控股中標軟件、銀河麒麟的中國軟件，以及分別持有深之度26.4%和8.8%股權的三六零和綠盟科技，建議重點關注。

中國軟件

公司是目前涉及主流國產操作系統的上市公司，其主營業務為自主軟件產品、行業解決方案和服務。上市公司分別持有中標軟件（中標麒麟）50%、天津麒麟（銀河麒麟）40%的股權；此外，公司還持有核心國產數據庫公司武漢達夢33.42%股權。

中國軟件旗下的操作系統與數據庫在國產化市場中的市佔率均佔據頭部位置。據2019年賽迪生態夥伴大會披露，中標麒麟操作系統蟬聯中國Linux市場佔有率第一，中標麒麟高級服務器操作系統(ARM64版)V7.0與中標麒麟可視化單向光閘分別獲得中國軟件市場和信息安全市場年度創新產品。另外，據2018年賽迪數據顯示，武漢達夢公司獲得2017—2018中國國產事務性數據庫市場“年度佔有率第一”。

未來5年內實現政府單位全部體量替換，則政府、事業單位桌面操作系統空間為100億，服務器操作系統市場空間為160億規模，假設數據庫單價20000元/臺，則數據庫市場空間為80億，中標軟件與達夢數據庫具備廣闊的發展前景。

深之度

據Distrowatch顯示，公司是為數不多在全球開源Linux市場排名進入前15的中國本土化操作系統，且主要以民營資本為背景，深耕操作系統市場16年。Deepin前身是基於Ubuntu社區更新的Hiweed Linux，後改為商業公司進行運作，並將源系統向上至Debian。公司經過多年努力，不僅本土化開發了28款實用工具應用外，還自主研發了Deepin Desktop Environment（DDE）桌面環境，全部開源為世界所用。

目前公司實控人為創始人兼董事長劉聞歡，共計持有公司54.85%股份，上市公司中，綠盟科技持有公司8.8%股權，三六零通過全資子公司北京世界星輝持有公司26.4%股權。

另外，2019年9月15日，華為榮耀Magicbook pro 銳龍版宣佈搭載深度操作系統桌面版，Deepin首次進入民用市場，成為進入消費者視野的國產化操作系統。

中國民用操作市場未來5年替換空間容量為45億元，新增市場空間容量為42億元；結合政府事業單位桌面操作系統100億空間，央地國企桌面操作系統150億空間，國產操作系統深度Deepin未來成長潛力較大。

風險提示：政策紅利後續釋放存在放緩可能；國產化操作系統可用性不達預期；更多潛在競爭者介入導致競爭加劇。
