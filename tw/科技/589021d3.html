python之函數的參數詳解

2020-11-14
| 科技

參數傳遞：

1.不可變數據類型：數字number(含int、float、bool、complex)、字符串string、元組tuple。當不可變數據類型被當作函數的參數，傳遞的是值，函數體內改變值時實際是生成新的對象，不會影響函數體外原來的值。例如：

x=10def print_add_ten(x):    x+=10    print(x)print_add_ten(x)print(x)out:2010

2.可變數據類型：列表list、字典dict、集合set。當可變數據類型被當作函數的參數，傳遞的是引用，函數體內改變值時會影響函數體外的值。例如：

x=[1,2,3,4,5]def print_change_list(x):    x[0]=10    print(x)print_change_list(x)print(x)out:[10, 2, 3, 4, 5][10, 2, 3, 4, 5]

上述這個函數看起來沒有問題，但在特殊場合會導致不可預料的bug。推薦的方式是對可變數據類型參數做copy(注意若存在嵌套結構需要用deepcopy)，然後做處理，並將處理結果返回給調用者。例如：

x=[1,2,3,4,5]def print_change_list(x):    x=x.copy()    x[0]=10    return xprint(print_change_list(x))print(x)out:[10, 2, 3, 4, 5][1, 2, 3, 4, 5]

參數類型

1.位置參數：即按照位置傳入參數，參數的個數、位置、類型必須匹配。如：

def power(x, n):    s = 1    while n > 0:        n = n - 1        s = s * x    return sprint(power(2,3))out:8

2.默認值參數：是指該參數有默認值，默認值必須是不可變數據類型。如：

def power(x, n=2):    s = 1    while n > 0:        n = n - 1        s = s * x    return sprint(power(2))print(power(2,3))out:48

3.可變參數：可變參數允許傳入0個或任意個參數，這些參數在函數調用時被封裝到一個元組，並將這個元組賦值給了args。如：

def calc(*args): # args是約定俗成的名字，可以自定義，但建議用args    sum = 0    for n in args:        sum = sum + n * n    return sumprint(calc(1,2,3))out:14

4.命名關鍵字參數：必須傳入參數名，不指定名字傳參會報錯。另外命名關鍵字參數也可指定默認值，有默認值的可以不傳參。如：

def person(*, name, age):    print(name, age)person(name='Mike', age=20)#person('Mike',20) # 不指定名字傳參會報錯out:Mike 20

5.關鍵字參數：允許傳入0個或任意個含參數名的參數，這些關鍵字參數在函數調用時被封裝到一個字典中，並將這個字典賦值給了kwargs。但關鍵字參數不能指定哪些關鍵字必須有。如：

def person(**kwargs): # kwargs是約定俗成的名字，可以自定義，但建議用kwargs    print(kwargs)person(name='Mike',age=20)person(gender='men',height=180)out:{'name': 'Mike', 'age': 20}{'gender': 'men', 'height': 180}

補充幾點：

  • 默認值參數是一種特殊的位置參數或者命名關鍵字參數，即該參數有默認值。如果不傳參，那麼該參數取值為默認值(必須是不可變數據類型)；如果傳了參那麼取值為傳的參。
  • 關鍵字參數實際是將參數名和參數值封裝成字典傳給函數。但是關鍵字參數的key不固定會導致寫代碼不方便。所以又有命名關鍵字參數，命名關鍵字參數是key固定而value不定的鍵值對，方便編寫代碼。
  • 位置參數和命名關鍵字參數的區別：位置參數可按位置傳參，也可用名字傳參，位置參數必須定義在可變參數前面。命名關鍵字參數必須指定名字傳參，命名關鍵字參數必須定義在可變參數後(如果沒有可變參數時用*代替)，若同時存在關鍵字參數時命名關鍵字參數必須定義在關鍵字參數之前。
  • *和**在函數定義時表示封裝，在函數調用時代表拆包。
def func(*args, **kwargs):    print(args)    print(kwargs)    print('-'*30)func([1, 2, 3], [22, 33])  # 不拆包時函數接收的是2個列表func(*[1, 2, 3], *[22, 33])  # 可迭代對象均可拆包，拆包後函數接收到的是一個元組。func({'name': '張三'}, {'age': 20})  # 這樣傳參的2個字典被args接收,kwargs接收的是空字典func(**{'name': '張三'}, **{'age': 20})  # 這樣傳參的2個字典被kwargs接收,args接收的是空元組out:([1, 2, 3], [22, 33]){}------------------------------(1, 2, 3, 22, 33){}------------------------------({'name': '張三'}, {'age': 20}){}------------------------------(){'name': '張三', 'age': 20}------------------------------
