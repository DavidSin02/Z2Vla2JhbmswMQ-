python筆記49：兩種方式來遍歷目錄，想要什麼自己來寫

2020-11-14
| 科技

主要內容：

  • 小目標：遍歷目錄
  • 主要內容：遞歸應用，os.walk方法；

上需求：如何在成百上千目錄與文件中，找到所有的圖片文件？

分析：

遍歷制定目錄下的子目錄與文件；

根據文件名後綴判斷是否是圖片(png, jpg);

如何區分目錄與文件？如何查找子目錄；

上才藝：準備一個目錄，裡面放一些子目錄與圖片文件，然後開造。

1. os.walk方法

os.walk方法：

#top為指定路徑#topdown默認自上而下os.walk(top, topdown=True, onerror=None, followlinks=False)

返回迭代器，迭代器內容：根目錄，子目錄，文件組元組;

上操作：

fpath = 'E:\imgs'items = os.walk(fpath)item = next(items)print(item)

結果：

('E:\\imgs', ['2020_05', '2020_06', '2020_07', '2020_08'], ['readme.txt'])

分析：

第一個元素：根目錄，E:\imgs；

第二個元素：E:\imgs下的所有子目錄，類型為列表；

第三個元素：E:\imgs所有文件，類型為列表

查找圖片功能實現：

遍歷目錄；

通過文件後綴(.png/.png)判斷文件是否為圖片，

將文件的完成路徑保存到列表中；

代碼實現：

import osfpath = 'E:\imgs'#遍歷目錄items = os.walk(fpath)imglist = []#遍歷items，獲取主目錄，子目錄，文件列表for mdir, subdir, files in items:    #遍歷文件列表    for fname in files:        #判斷是否為指定格式圖片        if fname.endswith('.png') or fname.endswith('.jpg'):            #圖片完整路徑拼接            imgpath = os.path.join(mdir, fname)            #添加到imglist            imglist.append(imgpath)#遍歷imglistfor img in imglist:    print(img)

結果：

E:\imgs\2020_05\level1.pngE:\imgs\2020_05\level2.png...E:\imgs\2020_08\5c4bda15af2de.jpg

主要知識點：

掌握os.walk方法；

文件判斷與路徑拼接；

字符串處理；

2. 遞歸方式實現

基本思路：

遍歷當前目錄，獲取子目錄與文件；

判斷文件或者目錄；

若是文件，通過判斷是否是圖片，並添加到列表中；

若是目錄，重複1~3操作；

代碼實現：

def scandir(topdir, listf):    #獲取當前目錄內容    flist = os.listdir(topdir)    #遍歷列表    for fname in flist:        #路徑拼接        path = os.path.join(topdir, fname)        if os.path.isfile(path):            #文件處理            if path.endswith('.png') or path.endswith('.jpg'):                listf.append(path)        elif os.path.isdir(path):            #目錄處理            scandir(path, listf)fpath = 'E:\imgs'           flist = []scandir(fpath, flist)for f in flist:    print(f)

結果：

E:\imgs\2020_05\level1.pngE:\imgs\2020_05\level2.png...E:\imgs\2020_08\5c4bda15af2de.jpg
