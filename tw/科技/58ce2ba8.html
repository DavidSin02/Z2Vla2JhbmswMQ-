新版Kali 2020 詳細圖文安裝教程，手把手教你搭建黑客攻擊平臺

2020-10-29
| 科技

Kali Linux 2020.1是目前最新的版本，相對於之前的2019版進行了大量的改動，除了組件的常規更新外，還包括默認使用標準的非特權用戶（非root用戶），官方期望以此來增加系統額外的安全性

新版的安裝可能會遇到一些列的問題，比如中文亂碼、無圖形化操作界面等

除此之外，有些同學還覺得不默認用root用戶，覺著不舒服，就像失去了靈魂

今天我就給大家帶來一篇詳細圖文安裝教程，並附上解決這些問題的辦法

kali是什麼？

Kali Linux是基於Debian的Linux發行版， 設計用於數字取證操作系統，是黑客常用的攻擊平臺

Kali Linux預裝了超過300個滲透攻擊工具，包括常用的信息收集工具nmap 、數據包修改工具Burpsuite 、系統攻擊工具metassploit，以及無線攻擊工具Aircrack-ng等

Kali Linux可直接在官網下載，該系統是永久免費的

點進去後，有很多選擇項，可以根據自己需要進行選擇

一般情況下，我們選擇第一條64位的ISO系統鏡像即可

虛擬機配置

一般情況下，我們不會將Kali安裝到物理機上，而是安裝在虛擬機裡，方便管理也不影響正常使用

虛擬機管理軟件常見的有VMware、Parallels Desktop（專用於MAC）、Virtual Box

使用哪一個虛擬機管理軟件都不影響Kali的安裝

這裡我以VMware下的安裝為例

網絡適配器我們先設定為橋接模式，這裡很重要，我們需要在安裝時聯網

順便提一下，虛擬機管理軟件提供了三個網絡模式

橋接模式，相當於虛擬機作為一臺單獨的設備直接聯網

NAT模式，共享你物理主機的IP，分配了個子網IP給虛擬機，能否上網取決於物理機

僅主機模式，只能和你的物理機通訊，無法上網

優化Kali

  • 更換為root用戶

在空白屏幕處點擊右鍵，根據圖標就可以知道哪個是新建終端命令

命令行執行

sudo passwd root

會先提示輸入當前普通用戶的密碼，執行sudo命令需要

這裡我需要輸入123456

然後再設置root密碼，緊接著輸入兩次相同的密碼

這個密碼就是你設置的root密碼

這裡我依次輸入test123和test123來作為我root密碼

設置完成後，繼續輸入

su - root你設置的密碼

即可進入root權限

  • 修改源地址

從國內的代理站點下載軟件要快許多

終端輸入指令

vim /etc/apt/sources.list

輸入i進入編輯模式

空白處添加阿里源

deb https://mirrors.aliyun.com/kali kali-rolling main non-free contrib


點擊esc，輸入:wq

退出並保存

# 替換好源後，更新一下源就可以了

sudo apt clean && sudo apt update -y

  • 中文亂碼

雖然剛才已經說了使用英文，系統已經可以正常使用

但為了以後方便使用，我們還是需要將該問題解決

在終端輸入指令，安裝中文字體

apt update -y && apt install xfonts-intl-chinese ttf-wqy-microhei

輸入reboot，重啟系統

可以看到我們的中文亂碼已經解決啦
