一、鋼鐵行業鋼渣處理現狀

目前，我國出渣量與鋼鐵產量的比例基本上維持在100kg/ t240kg/t 範圍內、就此計算，僅2017年，我國產鋼渣約1.3億噸左右。而我國實際鋼渣的綜合利用率不足30%, 導致我國鋼渣堆存量已累計超過10億噸 。鋼渣的處理與資源利用及零排放已成為困擾各個鋼鐵企業可待續發展和迫切需要解決的問題。

鋼渣中含有大量的金屬鐵和廢鋼，鋼渣中的含鐵物料約10% - 15%，通過磁選以後，可以提取回收各個粒級的廢鋼鐵 ，回收的廢鋼鐵產品根據粒度不同可分為渣鋼、粒鐵和磁選粉。

目前，由鋼渣回收的渣鋼由於品位高主要是直接返回轉爐鍊鋼，粒鐵和磁選粉採用返回高爐重複利用。

在實際生產中，鋼渣回收的廢鋼鐵一般根據不同規格大小和含鐵成分不同選擇不同的用途：

1、含鐵量在80%以上，規格>50mm的鋼渣渣鋼供應給鍊鋼當廢鋼消耗使用 ；

2、含鐵量在60%以上，規格在s5omm之間的鋼渣粒鐵供應給鍊鐵直接入爐使用；

3、含鐵量在60%以下，規格在1smm的鋼渣磁選粉供應給燒結當礦粉使用。

二、鋼渣處理工藝介紹

目前國內鋼渣主要處理工藝有：熱潑法、風埣法、滾筒法、熱悶法。其中熱潑法、滾筒法、熱悶法最為常用，在此對其工作原理和優缺點進行簡單介紹。

(1 ) 渣線熱潑法

將鋼渣傾翻 ，噴水冷卻3-4天后使鋼渣大部分自解破碎 ，運至磁選線處理。

此工藝的優點在於對渣的物理狀態無特殊要求、操作簡單、處理量大。

其缺點為佔地面積大、澆水時間長、耗水量大，處理後渣鐵分離不好、回收的渣鋼含鐵品位低、汙染環境、鋼渣穩定性不好、不利於尾渣的綜合利用。

( 2 ) 渣跨內箱式熱潑法

該工藝的翻渣場地為三面砌築並鑲有鋼蚽的儲渣槽，鋼渣罐直接從鍊鋼車間吊運至渣跨內，翻入槽式箱中，然後澆水冷卻。

此工藝的優點在於佔地面積比渣線熱潑小、對渣的物理狀態無特殊要求、處理最大、操作簡單、建設費用比熱悶裝置少。

其缺點為澆水時間24h以上、耗水量大、汙染渣跨和鍊鋼作業區、廠房內蒸汽大、影響作業安全。鋼渣穩定性不好、不利於尾渣綜合利用。

( 3 ) 滾筒法

高溫液態鋼渣從溜槽流淌下降時，被高壓空氣擊碎，噴至周圍的鋼擋板後落入下面水池中。

此工藝的優點在於流程短、設備體積小、佔地少、鋼渣穩定性好、渣呈顆粒狀、渣鐵分離好、渣中f—CaO含量小於4% (質量分數，下同）、便於尾渣在建材行業的應用。

其缺點為對渣的流動性要求較高、必須是液態稀渣、渣處理率較低、仍有大量的幹渣排放、處理時操作不當易產生爆炸現象。

( 4 ) 熱悶法

待熔渣溫度自然冷卻至300- 800°C 時，將熱態鋼渣傾翻至熱悶罐中 ，蓋上罐蓋密封，待其均熱半小時後對鋼渣進行間歇式噴水。急冷產生的熱應力使鋼渣龜裂破碎，同時大量的飽和蒸汽滲入渣中與f —CaO、 f —MgO發生水化反應使鋼渣局部體積增大從而令其自解粉化。

此工藝的優點在於渣平均溫度大於3oo·c均適用，處理時間短（10-12h）,粉化率高（粒徑20mm以下者達851Jfi-), 渣鐵分離好，渣性能穩定，f—Cao、f—MgO含量小於 2%, 可用於建材和道路基層材料。

其缺點為需要建固定的封閉式內嵌鋼蚽的熱悶箱及天車廠房、建設投入大、操作程序要求較嚴格、冬季廠房內會產生少量蒸汽。

三：鋼渣處理工藝共性問題

1、處理過程中造成二次汙染，主要是水汙染。

2、由於工藝原因造成回收產品的品味相對較，經濟效益差，再次利用率相對較低。

3、尾渣含鐵率偏高，不僅造成資源浪費，更造成尾渣很難被高附加值利用。

四、鋼渣處理方面政策解讀

國家《環境保護稅法》2018年1月1日正式啟用，其中對鋼鐵企業產生且未處置的鋼渣每噸徵收25元的環保稅，對露天堆存的鋼渣雨季所造成的鋼渣滲濾液每噸徵收300元的環保稅。

這對於鋼鐵企業來說，既是機遇更是挑戰，高效處理鋼渣有助於降低企業生產成本，延伸資源利用的深度，真正實現綠色環保型循環經濟。
