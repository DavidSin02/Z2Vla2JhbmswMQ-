函數

一、函數基礎

1.什麼是函數？

在程序中，函數就是具備某一功能的工具，事先將工具準備好就是函數的定義，遇到應用場景拿來就用就是函數的調用。

函數只是一種思緒，幹活的具體的實現都是靠if/while/for+變量，函數只是讓幹活更輕鬆。

2.為何用函數？

如果不使用函數，寫程序時就會遇到這三個問題：

  1. 程序冗長
  2. 程序的擴展性差
  3. 程序的可讀性差

函數可以把重複代碼寫成一個通用模板，可以把冗餘的代碼獨立開。

3.如何用函數？

先定義函數，後調用。

  • 定義函數：
def 函數名(param1、param2....):    ''''''    函數功能的描述信息    :param1:描述    :param2:描述    :return:返回值    '''''''    code 1    code 2    code 3    ...        return 返回值
  • 調用函數

二、定義函數的三種形式

定義函數時參數是函數體接收外部傳值的一種媒介，其實就是一個變量名

比較兩個數的大小，max是內置關鍵字

1.無參函數

在函數階段括號內沒有參數，稱為無參函數。需要注意的是：定義時無參，意味著調用時也無需傳入參數。

def self_max():    x, y = 10, 20    if x > y:        print(x)    else:        print(y)self_max()

2.有參函數

在函數定義階段括號內有參數，稱為有參函數。定義時有參，意味著調用時也必須傳入參數。有幾個參數，就必須傳入幾個參數

def self_max(x, y):    if x > y:        print(x)    else:        print(y)self_max(100,500)

3.空參數

當你只知道你需要實現某個功能，但不知道該如何用代碼實現時，你可以暫時寫個空函數，然後先實現其他的功能。

def self_max():    passself_max()

三、函數的返回值

1.什麼是返回值？

函數內部代碼經過一些列邏輯處理獲得的結果

def self_max(x,y):      #x=1000,y=100    if x>y:        return x        #返回一個值，給函數名       #return 任意數據類型的數據    else:        return y
# 變量名用來接收變量值# 函數名用來接收函數的返回值# print(self_max(10,100))  # 返回值默認為None

2.為什麼要有返回值？

現在有一個需求，比較兩個人的月薪，然後想獲取月薪較大人的年薪。

如果需要在程序中拿到函數的處理結果做進一步的處理，則需要函數必須要有返回值。

需要注意的是：

  • return是一個函數結束的標誌，函數內可以有多個return，只要執行到return，函數就會停止。
  • return的返回值可以返回任意數據類型
  • return的返回值無個數限制，即可以使用逗號隔開返回多個值
# 0. return 默認返回None# 1. return可以返回任意數據類型的數據# 2. 當return返回多個值的時候,會以元祖的形式把多個值包在一起# 3. return 會終止函數

四、函數的調用

1.什麼是函數調用？

第一次將函數其實就講了函數的調用，但是你不得不再次更新你對函數調用的印象。函數名(…)即調用函數，會執行函數體代碼，直到碰到return或者執行完函數體內所有代碼結束。

函數運行完畢所有代碼，如果函數體不寫return，則會返回None。

def foo():    passprint(foo())
none

五、函數參數的應用

１.形參和實參

  • 形參
  • 實參
##  函數分為兩個階段# # 1. 定義階段:定義階段的x和y被稱為形參,形參具有描述意義,類似於變量名,不為具體的數據類型# # 2. 調用階段:調用階段的x和y被稱為實參,實參必須是具體的數據類型### # 默認形參:在函數定義階段,提前賦值給形參,當你實參傳值的時候,實參傳的值會覆蓋默認值

2.位置參數

  • 位置形參在函數定義階段，按照從左到右的順序依次定義的形參，稱之為位置形參特點：按照位置定義的形參，都必須被傳值，多一個不行，少一個也不行。
  • 位置實參在函數調用階段，按照從左到右的順序依次定義的實參，稱之為位置實參特點：按照位置為對應的形參依次傳值

3.關鍵字實參

在調用函數時，按照key=value的形式為指定的參數傳值，稱為關鍵字實參。

用在函數調用階段,找到與形參的對應關係,並賦值給對應的形參,形參名為什麼則關鍵字實參名也必須是什麼

特點：可以打破位置的限制，但仍能為指定的形參賦值。

注意：

  1. 可以混用位置實參和關鍵字實參，但是位置實參必須在關鍵字實參的左邊。
  2. 可以混用位置實參和關鍵字實參，但不能對一個形參重複賦值。

4.默認實參

在定義階段，就已經被賦值

def func(x, y=10):    print(x)    print(y)    func(2)

特點：在定義階段就已經被賦值，意味著在調用時可以不用為其賦值

注意：

  1. 位置形參必須放在默認形參的左邊。
  2. 默認形參的值只在定義階段賦值一次，也就是說默認參數的值在函數定義階段就已經固定了。

默認參數的值通常應該是不可變類型

5.總結

實參的應用：取決於個人習慣

形參的應用：

  1. 大多數情況的調用值一樣，就應該將該參數定義成位置形參
  2. 大多數情況的調用值一樣，就應該將該參數定義成默認形參

六、可變長參數

可變長參數：指的是在調用函數時，傳入的參數個數可以不固定

調用函數時，傳值的方式無非兩種，一種是位置實參，另一種是關鍵字實參，因此形參也必須得有兩種解決方法，以此來分別接收溢出的位置實參(*)與關鍵字實參(**)

def self_max(a,b,c,d,e,f,g,h,k,x=1,y=3,z=4):  # 默認參數    print(a,b,c,d,e,f,g,h,k,x,y,z)# a= (1,2,3,4,5,6,7,8,9)a= [1,2,3,4,5,6,7,8,9]  # *把這個列表內的元素一個一個取出來,然後一個一個傳給這個函數b = {'x':10,'y':12}  # **會把這個字典打散成鍵值對的形式作為實參傳給函數self_max(*a,**b)  # 可變長實參 # (1,2,3,4,5,6,7,8,9,x=10,y=12)# self_max(1,2,3,4,5,6,7,8,9) # *a的作用# self_max(x=10,y=12)   # **b的作用

1.可變長形參之*

形參中的 * 會 將溢出的位置實參全部接收，然後存儲元組的形式，然後把元組賦值給 * 後的參數。需要注意的是：*後的參數名約定俗成為args。

def sum_self(*args):    res = 0    for num in args:        res += num    return resres = sum_self(1, 2, 3, 4)print(res)102.可變長實參之*實參中的，會將*後參數的值循環取出，打散成位置實參。以後但凡碰到實參中帶 *的，它就是位置實參，應該馬上打散成位置實參去看

10

2.可變長實參之*

實參中的，會將*後參數的值循環取出，打散成位置實參。以後但凡碰到實參中帶 *的，它就是位置實參，應該馬上打散成位置實參去看

def func(x, y, z, *args):    print(x, y, z, args)func(1, *(1, 2), 3, 4)

1 1 2 (3, 4)

3.可變長形參之**

形參中的 ** 會將溢出的關鍵字實參全部接收，然後存儲字典的形式，然後把字典賦值給 ** 後的參數。需要注意的是：**後的參數名約定俗成為kwargs

def func(**kwargw):    print(kwargw)func(a=5)

{'a': 5}

4.可變長實參之**

實參中的** ，** 會將 ** 後參數的值循環取出，打散成關鍵字實參。以後但凡碰到實參中帶** 的，它就是關鍵字實參，應該馬上打散成關鍵字實參去看

def func(x, y, z, **kwargs):    print(x, y, z, kwargs)func(1, 3, 4, **{'a': 1, 'b': 2})

1 3 4 {'a': 1, 'b': 2}

5.可變長參數應用

# 可變長形參def self_max(*args,**kwargs):    print(args)  #*a接受了所有位置實參,然後以元祖的形式保存下來,只接受位置參數,不接受關鍵字實參    print(kwargs)  # **b接收所有的關鍵字實參,然後以字典的形式保存下來self_max(1,2,3,4,5,6,7,x=6,y=8,z=20,a=10,c=1000)# *後面必須得寫args,**後面必須得寫kwargs,約定俗成的一種規範

6.命名關鍵字形參(瞭解)

命名關鍵字形參：在函數定義階段，*後面的參數都是命名關鍵字參數。

特點：在傳值時，必須按照key=value的方式傳值，並且key必須命名關鍵字參數的指定的參數名
