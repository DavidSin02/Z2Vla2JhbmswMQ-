Windows Core OS再遭曝光：可輕鬆移植到Xbox

2020-10-29
| 科技

全新設計的Windows Core OS系統已經不止一次地被曝光，甚至出現在過微軟官方的文件代碼裡，但是微軟從未正面評論這一系統，尤其是什麼時候發佈？用於什麼設備？

事實上，有關於Windows Core OS系統的消息2017年就出現了，據稱這是一套UI界面風格更現代化、功能模塊較為簡單的系統平臺，面向雙屏摺疊Surface、Xbox、Surface Hub、Hololens等終端設備，Windows Lite精簡系統也是其衍生品。

現在有人發現，微軟產品經理Naim Ayat的領英檔案裡赫然出現了Windows Core OS的身影，他正是這一團隊的成員。

按照他的描述，Windows Core OS系統是Windows 10、Windows MR HoloLens、Xbox軟件系統未來迭代的基石，但是很快，這句話就被刪除了，等於坐實。

Windows 10、Windows MR、Xbox等字樣已被刪除

Xbox Series X現在運行的系統叫Xbox OS，內核與Windows 10完全相同，但是既然Windows Core OS本身就是模塊化設計，不再依賴於硬件，那麼移植到未來的Xbox上也很輕鬆。

同時，Windows Core OS既然也是Windows 10今後演進的基礎，那就意味著很長時間裡都不會有Windows 11。

另外，Naim Ayat還透露他目前正在從事Windows 10X相關工作，這套針對Surface Neo等雙屏筆記本設計的系統，將在今年年底的購物季面世。
