手把手教你批量處理WORD文檔，菜鳥瞬間變大蝦

2020-10-29
| 科技

在日常辦公中，我們經常需要處理大量的WORD文檔，很多情況下這些文檔需要處理的操作是相同的，比如批量進行頁面格式設置，批量替換某些文字等等，如果能夠實現對這些文檔的批量處理效率豈不高多了！

下面教大家怎麼實現，整個過程用到了Word的VBA編程，但是你根本不需要懂一句代碼，通過四步就可以搞定。

第一步：打開一個要處理的Word文檔

第二步：調出Word開發工具選項卡


第三步：錄製操作過程

勾選後確定，會發現Word多了一個開發工具選項卡，點擊該選項卡，點錄製宏按鈕。Word會記錄你的每一步操作。比如簡單輸入幾個字ABCDEF

對Word操作完成後，點停止錄製。然後再點 Visual Basic 按鈕，會出現下面的界面：


上面的宏1就是剛才錄製的代碼，將光標放到代碼區域，也就是 第一行到第五行的區域範圍，然後點運行按鈕，你會發現Word中多了一個ABCDEF，是不是很神奇？

第四步：實現批量處理

通過上面的方法我們把處理一個文檔的方法記錄了下來，如何批量處理多個文檔呢，也很簡單。把下面的代碼複製到剛才錄製的代碼前面， 設置第4行和第5行的內容，就大功告成了。

測試：將光標放到 開始批處理代碼塊中，按執行按鈕，程序會自動打開指定文件夾下的每一個Word文檔（包括子文件夾文檔），並在每一個文件頭插入ABCDEF六個字母，然後自動關閉Word文檔。所有文檔處理完畢後會彈出一個"批量處理完畢!“ 的對話框。

錄製不同的宏，可以進行各種各樣的批量處理，怎麼樣，是不是感覺功能太強大了？所有這些我們不用寫一行代碼就可以實現，當然你有一定的編程基礎的話，還可以修改錄製的宏或者編寫更加複雜的宏功能。我們也會不定期發佈一些常用的宏功能，供大家調用。所以請大家關注我們的頭條號，及時獲取最新資料。

說明：Word的錄製宏的功能，不能錄製所有操作，尤其一些鼠標操作錄製效果不好，如果想在這方面深入體會的話可以下載安裝Office 2003 ，錄製的效果是所有office版本中最好的。

Private myFun As String

Sub 開始批量處理() '

dim myDir as string

myFun = "宏1" '修改成錄製的宏名稱

myDir= "D:\我的文件夾\" '修改成要處理的Word文檔所在的文件夾

SearchFiles myDir, "*.doc '批量處理doc格式文檔

SearchFiles Mydir, "*.docx" '批量處理docx格式文檔

MsgBox "批量處理完畢！", vbInformation + vbApplicationModal

End Sub

'功能：這個函數可以遍歷指定一個目錄下的所有文件,包括子目錄.也可以指定文件類型遍歷。

'用法：SearchFiles "C:\Program Files\WinRAR\", "*" '查找所有文件

Function SearchFiles(Path As String, FileType As String)

Dim Files() As String '文件路徑

Dim Folder() As String '文件夾路徑

Dim a, b, c As Long

Dim sPath As String

If Right$(Path, 1) <> "\" Then Path = Path & "\"

sPath = Dir(Path & FileType) '查找第一個文件

Do While Len(sPath) '循環到沒有文件為止

a = a + 1

ReDim Preserve Files(1 To a)

Files(a) = Path & sPath '將文件目錄和文件名組合,並存放到數組中

Documents.Open Files(a)’打開文件

Application.Run myFun’執行自定義函數

ActiveDocument.Close‘關閉文件

sPath = Dir '查找下一個文件

DoEvents '讓出控制權

Loop

sPath = Dir(Path & "\", vbDirectory) '查找第一個文件夾

Do While Len(sPath) '循環到沒有文件夾為止

If Left$(sPath, 1) <> "." Then '為了防止重複查找

If GetAttr(Path & "\" & sPath) And vbDirectory Then '如果是文件夾則。。。。。。

b = b + 1

ReDim Preserve Folder(1 To b)

Folder(b) = Path & sPath & "\" '將目錄和文件夾名稱組合形成新的目錄,並存放到數組中

End If

End If

sPath = Dir '查找下一個文件夾

DoEvents '讓出控制權

Loop

For c = 1 To b '使用遞歸方法，遍歷所有目錄

SearchFiles Folder(c), FileType

Next

End Function
