在桌面 Linux 世界中，最常用的術語之一就是 桌面環境(Desktop Environment)（DE）。如果你是 Linux 的新手，你應該瞭解一下這個經常使用的術語。

什麼是 Linux 中的桌面環境？

桌面環境是一個組件的組合體，為你提供常見的 圖形用戶界面(graphical user interface)（GUI）元素組件，如圖標、工具欄、壁紙和桌面小部件。藉助桌面環境，你可以像在 Windows 中一樣使用鼠標和鍵盤使用 Linux。

有幾種不同的桌面環境，這些桌面環境決定了你的 Linux 系統的樣子以及你與它的交互方式。

大多數桌面環境都有自己的一套集成的應用程序和實用程序，這樣用戶在使用操作系統時就能得到統一的感受。所以，你會得到一個文件資源管理器、桌面搜索、應用程序菜單、壁紙和屏保實用程序、文本編輯器等。

如果沒有桌面環境，你的 Linux 系統就只有一個類似於終端的實用程序，你只能用命令與之交互。

Screenshot of GNOME Desktop Environment


Linux 中各種桌面環境

桌面環境有時也被簡稱為 DE。

如前所述，Linux 有 各種桌面環境可供選擇 。為什麼這麼說呢？

可以把桌面環境看成是衣服。衣服決定了你的樣子。如果你穿緊身牛仔褲和平底鞋，你會很好看，但穿著這些衣服跑步或登山就不舒服了。

GNOME 這樣桌面環境注重現代的外觀和用戶體驗，而像 Xfce 這樣的桌面環境更注重使用更少的計算資源，而不是花哨的圖形。

Screenshot of Xfce Desktop Environment


你的衣服取決於你的需要，決定了你的外觀，桌面環境也是如此。你必須決定你是想要一些好看的東西，還是讓你的系統運行得更快。

一些 流行的桌面環境 有：

  • GNOME - 使用大量的系統資源，但給你一個現代的、精緻的系統
  • Xfce - 外觀復古但佔用資源很少
  • KDE - 可高度定製的桌面，適度佔用系統資源
  • LXDE - 唯一的重點是儘可能少地使用資源
  • Budgie - 現代的外觀和適度佔用系統資源

Linux 發行版及其桌面環境變體

同樣的桌面環境可以在多個 Linux 發行版上使用，一個 Linux 發行版也可能提供多個桌面環境。

例如，Fedora 和 Ubuntu 都默認使用 GNOME 桌面，但 Fedora 和 Ubuntu 都提供了其他桌面環境。

Linux 的優點和靈活性在於，你可以自己在任何 Linux 發行版上安裝桌面環境。但大多數 Linux 發行版都為你省去了這個麻煩，併為不同的桌面環境提供了隨時安裝的 ISO 鏡像。

例如 Manjaro Linux 默認使用 Xfce，但如果你喜歡在 Manjaro 上使用 GNOME，也可以下載 GNOME 版本的 ISO。

最後...

桌面環境是 Linux 桌面計算機的重要組成部分，而 Linux 服務器通常依靠命令行界面。並不是說不能在 Linux 服務器上安裝桌面環境，但這是畫蛇添足，浪費了重要的系統資源，而這些資源可以被服務器上運行的應用程序所利用。
