「金猿產品展」ZoomEye：網絡空間資源搜索引擎

2020-10-29
| 科技


本產品由知道創宇投遞數據猿參與評選。

北京知道創宇信息技術股份有限公司自 2012 年底開始，針對全球約 42 億 IP 地址進行信息探測工作，發佈網絡空間資源搜索引擎公眾版“ZoomEye”。

ZoomEye 網絡空間資源搜索引擎系統，由 500 臺硬件服務器、60G 帶寬、平臺軟件系統、傳感器軟件系統、存儲系統、分析系統構成。探測上包含 200 臺基礎探測服務器（網絡空間衛星）和 60G 接入帶寬。每個服務器每天能對 42 億地址空間（包含整個互聯網網絡空間）的每個開放端口進行基礎信息精準探測。

應用場景／人群

對全球網絡空間基礎設施、網絡設備進行指紋特徵檢索，構建互聯網安全基礎態勢測繪地圖。在全球重大漏洞爆發時進行資產檢查、應急處置、安全保障、網絡空間安全應急響應等工作，保障國家網絡安全和企業信息資產安全。

產品功能

ZoomEye 網絡空間資源搜索引擎提供了強大的 Restful API接口，通過它能夠更好的與第三方系統相連接。

ZoomEye 支持公網設備指紋檢索和 Web 指紋檢索

網站指紋包括應用名、版本、前端框架、後端框架、服務端語言、服務器操作系統、網站容器、內容管理系統和數據庫等。設備指紋包括應用名、版本、開放端口、操作系統、服務名、地理位置等直接輸入關鍵詞即可開始檢索。

大數據採集

ZoomEye 平臺通過部署數百臺服務器，統一調度，有策略的週期性的向全球網絡空間的設備發送相關協議請求，採集設備對應端口和服務協議的響應數據，並通過自主積累的協議解析庫和組件指紋庫，智能解析響應數據，最終形成網絡設備的組件指紋大數據庫。

大數據存儲

全球網絡空間設備的組件指紋大數據庫，具備典型的大數據特徵：數據體量大、數據結構複雜、實時更新流動性強。ZoomEye 平臺使用優秀的存儲方案，進行大數據存儲；並在此基礎之上，針對關鍵字段進行索引，優化查詢能力，為大數據挖掘提供良好的基礎。

大數據挖掘

數據挖掘是非常重要的能力，沒有數據挖掘，大數據就無意義。ZoomEye 平臺，基於網絡設備的組件指紋大數據庫，利用多年積累的安全經驗，深入研究各個組件和各種協議的安全性，結合安全能力對大數據進行探索和關聯分析，提煉成可感知的安全知識。

大數據可視化

可視化是知識吸收的重要環節，在大數據深度挖掘之後，通過大可視化能力，能形象的感知到數據的意義所在。ZoomEye 平臺，具備大數據可視化功能，可使用列表展示、圖形展示、地圖展示（2D、3D）等方式形象描繪出數據的意義所在。

存活設備搜索

通過 PING 操作的 ICMP 回包和目標端口開放信息，判斷目標對象是否存活，以獲取搜索目標中存活的網絡設備。

目標基本信息搜索

通過查詢產品中內置的地理位置庫和基本信息庫，獲取目標所屬的國家、省份、城市、組織機構、服務提供商等基本信息。

目標設備信息搜索

通過自主研發的搜索引擎，獲取目標的設備類型、設備品牌、設備型號、開放端口、提供服務、使用組件及版本等設備信息。

目標信息搜索

根據用戶給定的關鍵詞進行目標詳情信息搜索，支持單字段精確搜索和多字段模糊搜索，並將目標的所有信息聚合進行展示。

目標信息統計

針對用戶搜索之後的目標信息，提供統計圖表，展示設備類型、設備品牌、開放端口、組件名稱及版本的佔比統計信息。

數據導出

用戶搜索之後的目標信息，可以 CSV、XML 等格式導出，供用戶查看處理。

產品優勢

全球領先的互聯網資源探測能力

ZoomEye 網絡空間資源搜索引擎全面覆蓋全球 IPV4 地址空間的資產探測，分別針對網絡空間中的設備及網站進行 24 小時不間斷的探測、識別，標識出互聯網設備及網站所使用的服務及組件，能夠準確識別 40,000+常見組件指紋，精準探測 24h 以內網絡資產變化情況。平臺現已累計發現全球互聯網設備 11.9 億臺，是全球兩大著名網絡資產探測平臺之一。

網絡空間高效快速的搜索能力

ZoomEye 網絡空間資源搜索引擎，長期針對全球網絡空間約 42 億 IP 地址進行週期性的資源搜索，在此過程中知道創宇安全研究團隊總結經驗教訓，針對性優化每處細節，持續打磨搜索引擎，大大提升搜索效率。

利用自主研發並經過實戰考驗的 ZoomEye 網絡空間資源搜索引擎，具備了高效且快速搜索能力，且兼顧了資源搜索的準確性，能夠對目標網絡進行全面快速的網絡資產摸底和梳理。

業界領先的識別能力

ZoomEye 是國內首個網絡空間資源搜索引擎系統，已成為業界資源識別能力的標杆。團隊多年來持續更新的搜索能力，形成了海量精準的資源搜索規則庫，積累了豐富的網絡掃描與組件識別經驗，能對 3 萬種餘種網絡資源，超過 15 萬個版本進行精準識別。依賴於自主研發的組件識別調度框架、高效的組件識別引擎和海量精準的組件指紋庫，並輔以不斷更新組件指紋庫的常態機制，保證系統的組件識別能力始終處於業界領先水平。

基於公司長期積累的資源搜索能力，產品的搜索範圍覆蓋了常見的網絡設備和組件，可搜索的設備包括工控設備、路由器、防火牆、打印機、代理服務器等，可搜索的組件包括操作系統、數據庫軟件、郵件服務軟件、Web 容器等。

獨有的 IP 拆分掃描機制

普通掃描系統在進行網絡掃描時，會依據 IP 地址進行按序掃描，而一些安全設備通常具有防掃描機制，會根據掃描的順序性來判斷，如果單一 IP 請求過多並帶有一定規則性，則很容易被判定為攻擊行為，後續的連接請求將會被阻斷。本產品進行掃描時，會將 IP 列表進行拆分，打亂順序進行掃描，掃描完成後將再次重新排列組合，避免掃描被安全設備的防禦機制所阻斷，大大提高了資源搜索的成功率。

服務客戶／使用人數

目前ZoomEye網絡空間資源搜索引擎已經成為全球兩大著名網絡資產探測平臺之一，公眾版ZoomEye憑藉其強大的網絡空間資源探測能力深得全球網絡安全從業者與研究員們喜愛。商業版ZoomEye BE也正在被廣大企業用戶採用，通過ZoomEye BE進行網絡空間資產安全管理，消除存在系統中的安全威脅。

市場價值

在網絡空間資源測繪方面，ZoomEye已經有了眾多的積累，在企業級信息資產管理領域，ZoomEye也不斷進化，衍生出了商業版ZoomEye BE，致力於協助企業構建綜合的信息資產安全管理平臺。

ZoomEye BE（商業版）將資產安全管理納入企業用戶的安全管理模型中，基於全生命週期的安全資產監控，從披露資產開始，持續監控資產變化，實時獲取資產情報，對資產掃描漏洞發現、分析、修復和審核過程進行持續跟蹤，對企業資產數據進行統一處理、分析，形成完善的信息資產管理體系，實現資產情況的實時監控和企業資產的高效管理。
