一、手語文法的結構

1.關於手語的文法：自然手語的文法與我們慣用的中文文法不同，熟習自然手與文法是一件很重要的，一如我們學習英文文法，一定樣遵照它的文法結構系統一樣。

一般手語文法的順序： 主詞→名詞→動詞→形容詞→副詞→助動詞→疑問詞

2.無論上面順序如何變動，均萬變不離其宗，活用為主。基本上手語不論句子的構成，名詞或主詞均先比出來，之後才比動詞及其他詞。

例一：

a. a. 我是鄭照斌---我 /鄭照斌 【助動詞可有可無】

b. b. 我就是鄭照斌---我 / 鄭照斌 / 是【強調】

c. c. A：鄭照斌是誰？---鄭照斌 / 誰？

B：鄭照斌是我。---鄭照斌 / 我。

d. d. A：你 / 鄭照斌？

B：我 / 鄭照斌 / 是（或不是）

e. e. A：你 / 手語名字 / ……？

B：不是，手語名字 / 鄭照斌 / 我

例二：

a. a. 那個男生長得高大英俊---那 / 男 / 高 / 英俊

b. b. 那個高大英俊的男生---那 / 高 / 英俊 / 男

例三：

a. a. 昨天我搬家了，今天整理傢俱---昨天 / 我 / 家 / 搬 / 完了，現在 / 家 / 東西 / 整理

3.疑問句的手語，均置於整個句子之後。

例一：

a. a. 有沒有鈛？---鈛 / 有沒有？

b. b. 有沒有鈛借我？---鈛 / 借我 // 鈛 / 有沒有？

c. c. 有沒有鈛借我買表？---我 / 表 / 買 / 鈛 / 借我 / 鈛 / 有沒有

例二：

a. a. 為什麼你要欺負我？(欺負我 / 為 / 什麼)

b. b. 週末放假，你想去那裡玩？(星期六 / 放假 ， 想 / 玩 / 那 / 處)

3.手語的方向及方位要清楚指明是何人何地，完全以空間指示來表達。

例一：

a. a. 我喜歡你---我 / 喜歡 / 你

b. b. 我喜歡你---我→你 / 喜歡

c. c. 你喜歡他嗎？---你→他 / 喜歡？

例二：

a. a. 他認識我，但他不知道我的名字

那 / 男 / 見 (右→我) / 認識 // 我 / 名字 / 那 / 一竅不通

b. b. 他認識我，但他知道我名字嗎？

那 / 男 / 見 (右→我) / 認識 // 我 / 名字 / 那 / 知道？

4.助動詞的使用，一般均用在句尾。

a. a. 你會打電腦？---電腦 / 會？

b. b. 我和我女朋友要去看電影---我 / 女 / 朋友 / 我倆 / 電影 / 看 / 要

c. c. 我沒空，不能去看你---我 / 時間 / 沒有 // 探訪 / 不能

5.動詞的方位。

a. a. 我請客，你出鈛？---請客 (你→我) / 出鈛 (你→右)

b. b. 他告訴我，我在告訴你---告訴 (他→我) / 完了 // 告訴 (我→你)

c. c. 你不要騙我哦---騙我 / 不

d. d. 我跟他說好了，一起出國去玩---我←→他 / 談 / 達成協議 // 我倆 / 出國 / 玩

6.手語具有同時性。【表情】

a. a. 明天我去他家。

明天 / 去(右手，我→他家)

家(左手，他處不動)

b. b. 你不認識他嗎：

你 (眼神)

不認識 (右手)

他 (左手)

嗎 (抬眉)

7.表達數量時，名詞 + 數量，單位詞省略。

a. a. 我家有兩隻狗，三條魚---我 / 房子 / 狗 / 二 / 魚 / 三 / 有

b. b. 我有六輛汽車，八部飛機---我 / 汽車 / 六 / 飛機 / 八

c. c. 他討了四個老婆---他 / 娶 / 四

8.在手語裡面，很注意表達手語的方位及動作方向及時間的次序，如此才能表達每一人物或事件的相互關係，也就是說，手語具有時間域和空間域。

a. a. 由臺北搭飛機到高雄，再由高雄搭船到香港

臺北（胸前）/ 飛機（至左）/ 高雄（左）/ 船（至右）/ 香港 （右）

b. b. 我坐飛機到日本再轉機到紐約

搭飛機 / 起飛（由胸前→右）/ 日本 / 轉到 / 起飛（右→左）/ 紐約

c. c. 我打電話給他，但沒人接聽

我 / 打電話（至左）// 人（左）/ 接電話（至胸前）/ 沒有

9.比較性手語

a. a. 他的朋友多，我的朋友較少

b. b. 去的人漸漸多了，來的人少了
