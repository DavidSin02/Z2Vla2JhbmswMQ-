2020高考化學關鍵題型 第十講 電解應用

2020-10-29
| 科技

一、解題策略

第一步：確定電解池→電解池構成條件（電極、電解質溶液、閉合迴路、直流電源）

第二步：判斷陰陽極→兩極判斷（①看電源的正、負極；②看電子流向；③看離子流向；④看兩極反應類型；⑤看現象）

第三步：書寫電極反應→放電順序

第四步：根據反應回答問題。

二、題型分析


【典例1】【2017·課標全國Ⅰ，11】支撐海港碼頭基礎的防腐技術，常用外加電流的陰極保護法進行防腐，工作原理如圖所示，其中高硅鑄鐵為惰性輔助陽極。下列有關表述不正確的是(　　)


A．通入保護電流使鋼管樁表面腐蝕電流接近於零

B．通電後外電路電子被強制從高硅鑄鐵流向鋼管樁

C．高硅鑄鐵的作用是作為損耗陽極材料和傳遞電流

D．通入的保護電流應該根據環境條件變化進行調整

【答案】C

【解析】鋼管樁接電源的負極，高硅鑄鐵接電源的正極，通電後，外電路中的電子從高硅鑄鐵(陽極)流向正極，從負極流向鋼管樁(陰極)，A、B正確；C項，題給信息高硅鑄鐵為“惰性輔助陽極”不損耗，錯誤。

【典例2】【2016·課標全國Ⅰ，11】三室式電滲析法處理含Na2SO4廢水的原理如圖所示，採用惰性電極，ab、cd均為離子交換膜，在直流電場的作用下，兩膜中間的Na＋和SO可通過離子交換膜，而兩端隔室中離子被阻擋不能進入中間隔室。


下列敘述正確的是(　　)

A．通電後中間隔室的SO離子向正極遷移，正極區溶液pH增大

B．該法在處理含Na2SO4廢水時可以得到NaOH和H2SO4產品

C．負極反應為2H2O－4e－===O2＋4H＋，負極區溶液pH降低

D．當電路中通過1 mol電子的電量時，會有0.5 mol的O2生成

【答案】B

【解析】電解池中陰離子向正極移動，陽離子向負極移動，即SO離子向正極區移動，Na＋ 向負極區移動，正極區水電離的OH－發生氧化反應生成氧氣，H＋留在正極區，該極得到H2SO4產品，溶液pH減小，負極區水電離的H＋發生還原反應生成氫氣，OH－留在負極區，該極得到NaOH產品，溶液pH增大，故A、C項錯誤，B正確；該電解池相當於電解水，根據電解水的方程式可計算出當電路中通過1 mol電子的電量時，會有0.25 mol的O2生成，錯誤。

A．石墨極上發生氧化反應

B．陽極的電極反應式為

C．除去 的反應：

D．隨著電解的進行，石墨極附近電解質溶液的pH會減小

【答案】C

【解析】由圖可知，Fe與電源正極相連，則Fe作陽極，Fe失電子，發生氧化反應生成亞鐵離子，石墨為陰極，溶液中氫離子得到電子發生還原反應，且處理酸性廢水中的Cr2O72-，發生Cr2O72-+6Fe2++14H+═2Cr3++6Fe3++7H2O。A．石墨與電源負極相連，為陰極，發生還原反應，錯誤；B．Fe作陽極，Fe失電子，發生氧化反應生成亞鐵離子，陽極反應為Fe-2e-═Fe2+，錯誤；C．發生Cr2O72-+6Fe2++14H+═2Cr3++6Fe3++7H2O，處理廢水，正確；D．陰極發生還原反應，溶液中的氫離子得到電子減少，同時生成氫氧根，pH增大，錯誤。

2．鐵片鍍銅實驗中（裝置如下圖所示），為提高電鍍效果，電鍍前鐵片需依次用NaOH溶液、稀鹽酸和水洗滌。下列說法不正確的是


A．稀鹽酸的主要作用是除去鐵鏽

B．a接電源正極，b接電源負極

C．電鍍過程中c(Cu2+)基本保持不變

D．陰極發生反應：Cu2+＋2e– = Cu

【答案】B

【解析】電鍍時，銅片作陽極，即b接電源正極，電極反應為Cu-2e-=Cu2+，鐵片作陰極，即a與電源負極相連，電極反應為Cu2++2e-=Cu。A、電鍍前，用NaOH溶液除去鐵片上的油漬，用稀鹽酸除去鐵鏽，正確；B、電鍍時，a接電源負極，b接電源正極，錯誤；C、根據兩個電極反應，可以推出電鍍過程中c(Cu2+)基本保持不變，正確；D、陰極的電極反應為Cu2++2e-=Cu，正確。

3．國家有色金屬工業“十三五”發展規劃要求再生鉛佔鉛總量比重到2020年升至45%，再生鉛行業在重視環境保護和充分利用有色金屬再生資源的情況下逐步發展起來。從廢舊鉛蓄電池中回收的工藝流程為：


A．陰極反應式為：2H＋＋2e－===H2↑

B．電解過程中陽極附近pH明顯增大

C．Na2PbCl4濃度下降後，在陰極區加入PbO，可實現電解質溶液的再生使用

D．電路中通過4 mol電子，陰極可得207 g鉛

【答案】C

【解析】根據工藝流程可知，電解Na2PbCl4溶液製備Pb，則陰極上PbCl發生還原反應，電極反應式為：PbCl＋2e－===Pb＋4Cl－，A項錯誤；電解過程中陽極反應式為：2H2O－4e－===4H＋＋O2↑，陽極附近c(H＋)增大，pH減小，B項錯誤；根據陰極反應式及電解原理圖可知，電解一段時間後陰極區為HCl、NaCl的混合溶液，結合工藝流程圖可知，加入PbO，可使電解質溶液再生使用，C項正確；根據陰極反應式可知，電路中通過4 mol電子時，陰極可得2 mol Pb，質量為414 g，D項錯誤。

4．利用電解法制備Ca(H2PO4)2並得到副產物NaOH和Cl2。下列說法正確的是(　　)


A．C膜可以為質子交換膜

B．陰極室的電極反應式為2H2O－4e－===O2↑＋4H＋

C．可用鐵電極替換陰極的石墨電極

D．每轉移2 mol e－，陽極室中c(Ca2＋)降低1 mol/L

【答案】C

【解析】通過圖示分析C膜只能是陽離子交換膜，故A項錯誤；B．陰極室的電極反應式為2H2O＋2e－===H2↑＋2OH－，故B項錯誤；C．陰極電極不參與反應，故可用鐵電極替換陰極的石墨電極，故C項正確；D．未給出溶液體積，無法計算濃度，故D項錯誤。

5．寶雞被譽為“青銅器之鄉”，出土了大盂鼎、毛公鼎、散氏盤等五萬餘件青銅器。研究青銅器(含Cu、Sn等)在潮溼環境中發生的腐蝕對於文物保護和修復有重要意義。下圖為青銅器在潮溼環境中發生電化學腐蝕的原理示意圖，下列說法不正確的是(　　)


A．青銅器發生電化學腐蝕，圖中c作負極，被氧化

B．正極發生的電極反應為O2＋ 4e－＋2H2O===4OH－

C．環境中的Cl－與正、負兩極反應的產物作用生成a的離子方程式為2Cu2＋＋3OH－＋Cl－===Cu2 (OH)3Cl↓

D．若生成0.2 mol Cu2(OH)3Cl，則理論上消耗的O2體積為4.48 L

【答案】D

【解析】根據圖知，氧氣得電子生成氫氧根離子、Cu失電子生成銅離子，發生吸氧腐蝕，則Cu作負極被氧化，腐蝕過程中，負極是c，發生氧化反應，A正確；B、氧氣在正極得電子生成氫氧根離子，電極反應式為：O2＋ 4e－＋2H2O===4OH－，B正確；C、多孔粉狀鏽 Cu2( OH )3Cl為固體，故生成 Cu2( OH )3Cl 的離子方程式為：2Cu2＋＋3OH－＋Cl－===Cu2(OH)3Cl↓，C正確；D、不確定氧氣是否在標準狀況下，則不能計算氧氣的體積，D錯誤，答案選D。

6．工業上，用電解Na2CO3溶液製備NaHCO3溶液的原理如圖所示。下列說法正確的是(　　)

A．如圖所示裝置中，c2＞c1

B．陽極的電極反應式為2H2O－4e－===4H＋＋O2↑

C．如果該裝置中電極反接電源，也可製備NaHCO3

D．若收集3 mol M和R的混合氣體，則理論上可製備3 mol NaHCO3

【答案】B

【解析】陰極的電極反應式為2H2O＋2e－===2OH－＋H2↑，Na＋通過陽離子交換膜進入陰極區，陰極區生成NaOH，c1＞c2，A項錯誤；陽極上水參與電極反應產生O2和H＋，B項正確；如果電極反接電源，M為H2，陰極生成氫氧化鈉，碳酸鈉與NaOH不能生成碳酸氫鈉，C項錯誤；理論上3 mol O2和H2的混合氣體中含1 mol O2和2 mol H2，反應過程中轉移4 mol e－，由陽極反應式2H2O－4e－===4H＋＋O2↑，可知轉移4 mol電子時生成4 mol H＋，4 mol H＋與碳酸鈉反應生成4 mol NaHCO3，D項錯誤。

7．硼酸(H3BO3)為一元弱酸，已知H3BO3 與足量NaOH溶液反應的離子方程式為：H3BO3＋OH－===B(OH)，H3BO3可以通過電解的方法制備。其工作原理如圖所示(陽膜和陰膜分別只允許陽離子、陰離子通過)。下列說法正確的是(　　)


A．當電路中通過1 mol電子時，可得到1 mol H3BO3

B．將電源的正、負極反接，工作原理不變

C．陰極室的電極反應式為：2H2O－4e－===O2↑＋4H＋

D．B(OH)穿過陰膜進入陰極室，Na＋穿過陽膜進入產品室

【答案】A

【解析】電解時，左側石墨電極為陽極，右側石墨電極為陰極，陽極上H2O失電子生成O2和H＋，即2H2O－4e－===O2↑＋4H＋，陰極上H2O得電子生成H2和OH－，即2H2O＋2e－===H2↑＋2OH－，當電路中通過1 mol電子時，陽極生成1 mol H＋，H＋通過陽膜進入產品室，與通過陰膜進入產品室的B(OH)反應生成1 mol H3BO3，A項正確；電源正、負極反接後，左側石墨電極為陰極，陰極反應式為：2H＋＋2e－===H2↑，H＋被消耗，無法移向產品室，不能生成H3BO3，B項錯誤；結合上述分析可知，陰極反應式為2H2O＋2e－===H2↑＋2OH－，C項錯誤；電解時原料室中Na＋穿過陽膜進入陰極室，B(OH)穿過陰膜進入產品室，D項錯誤。

8．以四甲基氯化銨[(CH3)4NCl]為原料，採用電滲析法合成四甲基氫氧化銨[(CH3)4NOH]，其工作原理如圖所示(a、b為石墨，c、d、e為離子交換膜)，下列說法正確的是(　　)


A．a是陽極，連接的M為正極

B．c、e均為陽離子交換膜，d是陰離子交換膜

C．b極的電極反應式：2H2O－4e－===O2↑＋4H＋

D．a、b兩極共產生0.5 mol 氣體時，可製備1 mol(CH3)4NOH

【答案】B

【解析】根據氯離子的移動方向可知，a為陰極，b為陽極，所以M為負極，A錯誤。又據圖分析，de區域間得到氯化鈉濃溶液，說明陽極上不是氯離子放電，則e是陽離子交換膜，陽極(b極)的電極反應式為4OH－－4e－===O2↑＋2H2O，據左池產物分析，c也是陽離子交換膜，a電極上水電離出的氫離子放電生成氫氣，根據以上分析可知c、e均為陽離子交換膜，氯離子能通過d，所以d是陰離子交換膜，B正確、C錯誤。a電極上水電離出的氫離子放電生成氫氣，b電極上氫氧根離子放電生成氧氣，製備1 mol(CH3)4NOH時轉移1 mol電子，根據電路中通過的電子數相等可知，a、b兩極共產生氣體0.5 mol＋0.25 mol＝0.75 mol，D錯誤。
