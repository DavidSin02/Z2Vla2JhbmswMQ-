4.CSS選擇器

2020-10-29
| 科技

1.CSS基礎選擇器

1）通配符選擇器

通配符選擇器用“*”號表示，他是所有選擇器中作用範圍最廣的，能匹配頁面中所有的元素。

<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        * {            margin: 0;            padding: 0;            font-size: 28px;        }        div {            color: red;            background-color: pink;        }</style></head><body>    <div>碼海無際</div>    <span>碼海無際</span></body></html>

2）標籤選擇器

標籤選擇器是指用HTML標籤名稱作為選擇器，按標籤名稱分類，為頁面中某一類標籤指定統一的CSS樣式。

<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        div{            color: pink;        }</style></head><body>    <div>碼海無際</div></body></html>

3）類選擇器

  1. 不要純數字、中文等命名，儘量使用英文字母來表示。
  2. 長名稱或詞組可以使用中橫線來為選擇器命名。
  3. 不建議使用“_”下劃線來命名CSS選擇器。
  4. 類選擇器使用“.”（英文點號）進行標識，後面緊跟類名。
<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        .color-blue{            color: blue;        }</style></head><body>    <div class="color-blue">碼海無際</div>    <span class="color-blue">志向高遠</span></body></html>

4）多類名選擇器

<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        .color-blue{            color: blue;        }        .font-size28{            font-size: 28px;        }</style></head><body>    <div class="color-blue font-size28">碼海無際</div>    <span class="color-blue">志向高遠</span></body></html>

5）id選擇器

  1. 用法基本和類選擇器相同。
  2. id選擇器使用“#”進行標識，後面緊跟id名。
  3. 該語法中，id名即為HTML元素的id屬性值，大多數HTML元素都可以定義id屬性，元素的id值是唯一的，只能對應於文檔中某一個具體的元素。

id選擇器和類選擇器區別：

  1. W3C標準規定，在同一個頁面內，不允許有相同名字的id對象出現，但是允許相同名字的class。
  2. 類選擇器(class)，好比人的名字，是可以多次重複使用的，比如：張偉、老王。
  3. id選擇器，好比人的身份證號碼，全中國是唯一的，不得重複，只能使用一次。
  4. id選擇器和類選擇器最大的不同在於使用次數上。
<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        #color-blue {            color: blue;        }</style></head><body>    <div id="color-blue">碼海無際</div></body></html>

6）選擇器的優先級

通配符選擇器<標籤選擇器<類選擇器<id選擇器

<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        * {            color: pink;        }        div {            color: red;        }        div {            color: blue;        }        .color-green {            color: green;        }        #color-blueviolet {            color: blueviolet;        }</style></head><body>    <div class="color-green" id="color-blueviolet">碼海無際</div></body></html>

2.CSS複合選擇器

1）交集選擇器

  1. 交集選擇器由兩個或兩個以上選擇器構成。
  2. 選擇器之間沒有任何的連接符號，選擇器之間沒有固定順序。
  3. 選擇器可以是：標籤選擇器、類選擇器、id選擇器。
<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        div.color-blue {            color: blue;        }        div#color-red.color-red{            color: red;        }</style></head><body>    <div class="color-blue">碼海無際</div>    <span class="color-blue">碼海無際</span>    <hr>    <div class="color-red" id="color-red">碼海無際</div></body></html>

2）並集選擇器

  1. 交集選擇器由兩個或兩個以上選擇器構成。
  2. 選擇器之間通過英文逗號連接，選擇器之間沒有固定順序。
  3. 選擇器可以是：標籤選擇器、類選擇器、id選擇器。
<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        /*span {*/        /*    color: blue;*/        /*    font-size: 28px;*/        /*}*/        /*div {*/        /*    color: blue;*/        /*    font-size: 28px;*/        /*}*/        /*倆種寫法一樣*/        span, div {            color: blue;            font-size: 28px;        }</style></head><body>    <div>碼海無際</div>    <span>碼海無際</span></body></html>

3）子元素選擇器

  1. 子元素選擇器只能選擇作為某元素子元素的元素。其寫法就是把父級標籤寫在前面，子級標籤寫在後面，中間跟一個 > 進行連接，注意，符號左右兩側各保留一個空格。
  2. 這裡的子，指的是親兒子，不包含孫子、重孫子之類。
<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        div > .color-pink > span{            color: pink;        }</style></head><body>    <div>        <div class="color-pink">            <span>碼海無際</span><!--有效果-->        </div>        <div>            <span>碼海無際</span><!--無效果-->        </div>        <span>            <div class="color-pink">               <span>碼海無際</span><!--無效果-->            </div>        </span>    </div></body></html>

4）後代選擇器

  1. 後代選擇器又稱為包含選擇器，用來選擇元素或元素組的後代，其寫法就是把外層標籤寫在前面，內層標籤寫在後面，中間用空格分隔。當標籤發生嵌套時，內層標籤就成為外層標籤的後代。
  2. 包含孫子、重孫子之類
<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        div h1{            color: blue;        }</style></head><body>    <div>        <h1>碼海無際</h1>        <span>            <h1>碼海無際</h1>        </span>    </div></body></html>

5）偽類選擇器

  1. 偽類選擇器用於向某些選擇器添加特殊的效果。
  2. 偽類對象要配合content屬性一起使用。
  3. 一個冒號是css2的寫法，兩個冒號是css3的寫法，表達的意思一樣。
  4. 冒號的前面和後面都不允許有空格。

<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <title>Title</title>    <style>        a:visited{            color: red;        }        p:before{            content:"P前面";            color: red;        }        p:after{            content:"P後面";            color: red;        }</style></head><body>    <a href="https://www.taobao.com">淘寶</a>    <a href="https://www.baidu.com">百度</a>    <a href="https://www.taobaoaa.com">淘寶aa</a>    <a href="https://www.taobaoaaa.com">淘寶aaa</a>    <hr>    <p>P元素</p></body></html>
