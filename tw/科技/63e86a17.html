混合級配砂定量配製與應用技術

2020-11-14
| 科技

摘要：通過理論計算結合試驗將不符合要求的兩種或兩種以上的砂，定量摻配成符合混凝土質量要求的混合砂。用此方法指導混合砂配製，應用於混凝土生產，在確保工程質量的前提下，降低了成本，節約了資源，改善了混凝土性能，對優質天然砂資源匱乏或價格高地區的混凝土生產具有一定的參考價值。

0 前言

在國民基礎設施建設過程中，混凝土材料佔到了相當大的比重。粗細集料作為混凝土的重要組成部分，在混凝土中佔到約70%的比例。在當前的情況下，混凝土中的細集料以天然砂為主。但是，隨著建設規模的不斷增加，天然砂資源日趨緊張，優質天然砂數量越來越少，價格越來越高，嚴重影響和制約著工程建設的進展。用兩種砂或兩種以上的砂，摻配組成混合砂代替天然中砂，配製混凝土應用於建設工程中，不但可以緩解優質天然砂資源短缺的問題，還可以改善混凝土的一些性能。

在混凝土細集料的技術指標中，砂的顆粒級配是反應砂各級粒徑顆粒分佈情況的指標，砂的細度模數是評價砂粗細程度的一種指標。根據《普通混凝土用砂、石質量及檢驗方法標準》（JGJ 52—2006)的規定，配製混凝土宜優先選用顆粒級配在II區的中砂。這樣的砂具有較高的密度和較小的比表面積，有利於混凝土的和易性和可泵性。通過試驗，我們對細度模數、顆粒級配等技術指標不符合要求和其中一種價格較高、貨源較緊張的兩種或兩種以上的砂進行了摻配。使摻配後混合砂的顆粒級配保持在II區，細度模數控制在中砂範圍內（2.3〜3.0)，其他性能均符合規範要求，且價格適中，供應有保障。再用摻配後的混合砂，進行混凝土試驗，並把試驗成果應用在工程建設中，起到了良好的社會效益和經濟效益。

表2 砂2篩分表

1.2 計算確定摻配比例

1.2.1 根據混合砂的細度模數宜控制在中砂範圍內（2.3〜3.0) ，上述砂1和砂2各自摻配比例可按以下公式計算：

μf(n)=μf(1)XS1+μf(2)X[1-S1]

式中μf——混合砂的細度模數；

μf(1)——砂1的細度模數；

μf(2)——砂2的細度模數；

S1——混合砂中砂1的摻配質量百分比。

列公式 2.3 = 3.5XS1下限+ 1.3X[1—S1下限]

3.0 = 3.5XS1上限+ 1.3X[1—S1上限]

解得S1下限=45%,S1上限=77%。

可大致確定砂1的摻配比例範圍在45%〜77%之間，砂2的摻配比例範圍在23%〜55%之間。

1.2.2將砂1、砂2按在1.2.1中確定的範圍內，用逼近法則按不同比例試算摻配，並進行校核， 結果見表3。

表3 不同摻配比例計算細度模數及級配情況表

1.2.3通過以上分析可確定細度模數和顆粒級配分佈均符合要求的砂1摻配比例為（62±5)%，砂2摻配比例為（38±5)%。

1.3摻配後實際篩分情況

把砂1按62%比例，砂2按38%比例摻配，用摻配後的砂做篩分試驗。試驗數據同理論計算數據吻合，具體見表4,此混合砂細度模數μf=2. 6,顆粒級配屬於II區。滿足要求。

2.2 摻配

2.2.1 摻配原因：

(1)3種砂的細度模數和顆粒級配分佈均不滿足要求。

(2)砂3、砂4分別和砂5摻配後，細度模數和顆粒級配均不滿足II區砂要求。

(3)砂3和砂4直接摻配後，細度模數和顆粒級配可以滿足II區砂要求，但砂3、砂4價格高，供應緊張。僅使用這兩種砂，供貨量不能滿足正常生產的需求，且混凝土單方成本較高。（注：砂3、砂4送到混凝土生產地的價格約為5。元/噸且貨源緊張，砂5到混凝土生產地的價格約為20元/ 噸且能滿足供應。）

2.2.2 步驟一：

根據混合砂的顆粒級配保持在II區，可按試算法確定組成混合砂中各砂的比例。其依據是：如有幾種砂，要配製成符合顆粒級配要求的混合砂。在決定各組成砂在混合砂中的比例時，可先假定混合砂中某區間粒徑的顆粒是由其中一種對這一粒徑佔優勢的砂組成，其他砂不含這種粒徑的顆粒。這樣根據各個主要粒徑去試算各種砂在混合砂中的大致比例。如比例不合適，則稍加調整，這樣逐步逼近，最終得出符合II區中砂要求的混合砂摻配比例。

2.2.3 步驟二 ：

(1)計算砂3在混合砂中的用量S3。

由表8可知，砂3中佔優勢的粒徑為2.50mm和1.25m的含量，故計算砂3的摻配比例時，假設混合砂中2.50mm和1.25m粒徑全部是由砂3組成。因此，mb(2.50+1.25)和mc(2.50+1.25)均等於0。

故有

mb(2.50+1.25)S3=Ma

S3= Ma(2.50+1.25)/ma(2.50+1.25) = (7.5十22.5)÷ (42.8十40.2) =36%

(2)計算砂5在混合砂中的用量S5。

由表8可知，砂5中佔優勢的粒徑為0.1 6mm的含量，故計算砂5的摻配比例時，假設混合砂 中0.16mm粒徑全部是由砂5組成。因此，ma(0.16)和mb(0.16)均等於。。

故有

mc(0.16)S5 =Mc(0.16)

S5 =Mc(0.16)/mc(0.16)=14÷45.6 = 31%

(3)計算砂4在混合砂中的用量S4。

S4 = 100-S3 -S5 = 100-36-31 = 33%

表8 各砂篩分情況及摻配後混合砂要求的顆粒級配範圍


2.2.4 步驟三：

校核。以摻配比例S3=36%，S4=33%，S5=31%，試算得出細度模數μf=2.6,級配情況見表9。

表9 砂3:砂4:砂5=36:33:31摻配計算校核結果


經校驗，細度模數和顆粒級配均符合要求範圍。如不符合應調整摻配比例，再進行試算經數次調整，直至達到要求。如經計算不能符合要求，貝IJ應調整或增加砂品種。

2.3 摻配後實際篩分情況

把砂3、砂4、砂5分別按36%、33%% 31%比例摻配，用摻配後的砂做篩分試驗。試驗數據同理論計算基本吻合，見表10，此混合砂的細度模數& = 2.5 (與計算所得的細度模數之差為0.1，符合平行試驗允許要求），顆粒級配屬於II區，滿足要求。

4 結論

通過試驗，可以得出以下結論：

(1)不符合細度模數和顆粒級配要求的兩種或兩種以上的砂，或者是有一種砂價格較高、貨源較緊張的情況下，可以按照理論計算的方法，定量計算出兩種或兩種以上砂的摻配比例，摻配成供應有保障、價格有優勢，顆粒級配符合II區的中砂。

(2)實際使用中，在原材料有小幅度波動時，可根據幾種砂的粗細程度、顆粒級配變化情況及預拌混凝土對砂粗細、顆粒級配的具體要求，對已確定摻配比例的砂，在±5%的範圍內進行摻配比例調整，以保證滿足混凝土性能要求。

(3)按上述方法摻配的混合砂，可拌製出滿足工作性要求和強度要求的混凝土。

(4)在實際生產中，每種砂要通過不同料倉分別計量，以確保混合砂摻配比例準確、質量均勻。

(5)在實際生產中，單種砂在混凝土中摻配比例不能太小，否則攪拌機不能稱量或稱量誤差會超出允許範圍。

混凝土中宜使用顆粒級配範圍在II區的中砂，通過理論計算等方法，把不符合使用要求的兩種或兩種以上的砂，摻配成顆粒級配合理、粗細適中、價格有優勢的混合砂，為混合砂的定量配製提供了一種切實可行的方法。
