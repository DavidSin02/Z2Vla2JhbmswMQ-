4-格式化輸出

普通輸出

print()

常用格式化輸出

# 重點掌握以下三個，剩下的作為了解name = 'Gary'print('My name is %s' % name) age = 18print("I'm %d year's old" % age) height = 176.5print("I'm %f cm tall" % height)  # ###########################################################3# %c 字符 http://ascii.911cha.com/print('%c' % 64) # %i 有符號十進制整數print('%i' % -50)# %u 無符號十進制整數print('%u' % -50) # %o 八進制整數print('%o' % 50) # %x 十六進制整數(小寫字母)# %X 十六進制整數(大寫字母)print('%x' % 60)print('%X' % 60) # %e 科學計數法(小寫'e')# %E 科學計數法(大寫'E')print('%e' % 50)print('%E' % 50) # %g％f和％e的簡寫# %G％f和％E的簡寫print('%g' % 50)print('%G' % 50)

5-格式化輸出（進階）

height = 176.5# 初始情況print("I'm %f cm tall" % height)print('*'*40) # %X.Yfprint("I'm %8.2f cm tall" % height)print('*'*40) # +代表保留正負號print("I'm %+8.2f cm tall" % height)# -代表左對齊print("I'm %-8.2f cm tall" % height)# 空格代表用空格填滿多餘位置print("I'm % 8.2f cm tall" % height)# 0代表用0填滿多餘位置print("I'm %08.2f cm tall" % height)

6-輸入

input()：把用戶輸入的全部數據作為一個字符串

name = input('請輸入您的姓名:') print(name)
