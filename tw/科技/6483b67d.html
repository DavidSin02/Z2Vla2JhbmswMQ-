SQL中利用LIKE實現模糊查詢的功能

2020-11-14
| 科技

大家好，今日繼續講解《VBA數據庫解決方案》，今日講解的內容是：利用ADO,實現模糊查詢。在上一講中，我們實現了利用ADO快速查找的功能，今日我們實現工作表中模糊查找的功能。我們仍是利用上一講的數據實現, 在"兩表查詢數據"的工作表中有如下數據：

我們注意到其中型號和生產廠家的數據有些是混的，這就給我們利用上講的內容造成了不便，那麼這個時候就要利用模糊查找了,如下的數據：

我們需要在"兩表查詢數據"中A列+B列的內容中找到上述表格中A列的對應的型號，怎麼處理呢？

我們看下面的代碼：

Sub mynzexcels_8()

'第39講，利用ADO,實現模糊查詢

Dim cnADO, rsADO As Object

Dim strPath, strTable, strSQL As String

Set cnADO = CreateObject("ADODB.Connection")

Sheets("Sheet5").Activate

'建立一個ADO的連接

strPath = ThisWorkbook.FullName

cnADO.Open "provider=Microsoft.ACE.OLEDB.12.0;extended properties='excel 8.0;hdr=no;imex=1';data source=" & strPath

i = 2

Do While Cells(i, 1) <> ""

Cells(i, 1).Select

Cells(i, 3) = "": Cells(i, 4) = "": Cells(i, 5) = ""

'這裡只是提出F3,F3,F5的數據

strSQL = "select F3,F4,F5 from [兩表查詢數據$] where F1&F2 Like '%" & Cells(i, 1) & "%'"

Set rsADO = New ADODB.Recordset

rsADO.Open strSQL, cnADO, 1, 3

If rsADO.RecordCount <= 0 Then

MsgBox ("第" & i & "行數據，沒有找到！")

Else

If rsADO.RecordCount = 1 Then

Cells(i, 3).CopyFromRecordset rsADO

Else

For TT = 1 To rsADO.RecordCount - 1

Rows(i + TT & ":" & i + TT).Select

Selection.Insert Shift:=xlDown, CopyOrigin:=xlFormatFromLeftOrAbove

Cells(i + TT, 1) = Cells(i + TT - 1, 1): Cells(i + TT, 2) = Cells(i + TT - 1, 2)

Next

Cells(i, 3).CopyFromRecordset rsADO

i = i + TT - 1

End If

End If

rsADO.Close

Set rsADO = Nothing

i = i + 1

Loop

Set cnADO = Nothing

End Sub

代碼截圖：

代碼講解：

1 strSQL = "select F3,F4,F5 from [兩表查詢數據$] where F1&F2 Like '%" & Cells(i, 1) & "%'" 這句SQL語句就是實現了從"兩表查詢數據"的工作表數據中找到類似於Cells(i, 1)內容的記錄，大家要注意的這種SQL語句的寫法，是like +% %的組合。

2 If rsADO.RecordCount = 1 Then

Cells(i, 3).CopyFromRecordset rsADO

如果僅為1條記錄，那麼就直接拷貝。

3 For TT = 1 To rsADO.RecordCount - 1

Rows(i + TT & ":" & i + TT).Select

Selection.Insert Shift:=xlDown, CopyOrigin:=xlFormatFromLeftOrAbove

Cells(i + TT, 1) = Cells(i + TT - 1, 1): Cells(i + TT, 2) = Cells(i + TT - 1, 2)

Next

Cells(i, 3).CopyFromRecordset rsADO

i = i + TT - 1

End If

如果是多條記錄，那麼需要在原查詢記錄的下面插入行來填入查詢的結果，上述代碼就是完成了這個目的。
