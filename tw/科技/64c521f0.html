字節流的輸入輸出，JAVA基礎知識筆記

2020-10-29
| 科技

字節流

  • 字節流是指流的內容是字節，字節流也稱為二進制流。
  • 所有的字節輸入流都繼承了InputStream抽象類；
  • 所有的字節輸出流都繼承了OutputStream抽象類。
  • InputStream和OutputStream抽象類定義了最基本的輸入和輸出方法。


InputStream的read() 方法


InputStream類的主要方法


InputStream類提供了以下用於讀取字節流的方法：

  • public abstract int read() throws IOException

該方法的作用是從來源輸入流中讀取數據的下一個字節到Java程序中

該方法的返回值是從0到255的int類型的值

如果沒有可用的字節，因為已經達到輸入流的末尾，則返回‐1

該方法調用的時候只讀取一個字節，因此往往放在while之後的條件判斷中，例如：

while ( (len = inputstream.read()) != ‐1)


InputStream還提供了用於批量讀取字節的read方法，包括：

public int read(byte[] b) throws IOExceptionpublic int read(byte[] b, int off, int len) throws IOException

第一個方法從來源輸入流中讀取一定數量的字節數，並存儲到緩衝區數組b中，返回值為實際讀取的字節數。

  • 該方法運行時讀取一定數量的字節數，Java會儘可能讀取的字節數是數組b的長度，但也有可能讀取少於b長度的字節數。
  • 如果沒有可用的字節，因為已經達到輸入流的末尾，則返回‐1

第二個方法多了兩個參數，off 和 len。

  • off 是存儲到數組b中時元素下標的偏移量，而len則是要讀取的字節數。

OutputStream的write方法


OutputStream類的主要方法

OutputStream類則提供了write方法用於將指定的數據寫入到輸出流中

最簡單的write方法如下：

  • public void write(int b);

該方法的作用是往輸出流中寫入一個字節b


OutputStream類還包括以下2個方法：

  • void write( byte b[ ] );

該方法的作用是往輸出流中寫一個字節數組b

  • void write( byte b[ ], int off, int len );

該方法的作用是把字節數組b中從下標off開始，長度為len的字節寫入到輸出流中


OutputStream中還包括了flush方法：

  • public void flush()

該方法是刷空輸出流，並輸出所有被緩存的字節。

由於某些流支持緩存功能，flush法將把緩存中所有內容強制輸出到流中。


OutputStream還提供了關閉輸出流的方法：

  • public void close();

該方法在輸出流操作完畢後必須關閉，否則可能會產生異常。


實際上，InputStream類也提供了close方法。

考慮到IO處理時可能會產生異常，一般在編程時，把close方法的調用放

在finally語句塊中，保證close方法確定會被調用執行。
