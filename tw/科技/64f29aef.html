基類和元類型

在Python源碼中，數據類型都是以類型來表示的，稱作類型對象。一個整形，浮點型，字符串，序列，元組，字典都是一個對象。比如說，一個int（類型） 是一個類型對象，類型int實例化之後生成對象（int （8））是一個實例對象。在這個對象體系中，既然都是對象，一定需要一個基類，其餘類繼承這個基類。就可能存在一個類型對象的元類型（type），一個實例對象的基類（object）。還需要某種關係把類型對象和實例對象聯繫起來。如圖1 基類和元類型。

變長對象與定長對象

我們可以感覺到，在python中，用一個整形變量無需知道整數的範圍，例 a= 8; a = 600000都可以，不會溢出。說明變長對象內部除了存儲數值內容之外，需要存儲一個頭來表示這些信息。從圖2 可以看出定長對象的頭部缺少一個字段ob_size。

圖2 定長對象與變長對象

實際代碼看一下，定長對象和變長對象的頭。

typedef struct _object {

Py_ssize_t ob_refcnt;

PyTypeObject *ob_type;

} PyObject; //定長對象頭

typedef struct {

PyObject ob_base;

Py_ssize_t ob_size; /* Number of items in variable part */

} PyVarObject; //變長對象的頭
