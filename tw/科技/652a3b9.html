幕牆常用術語，建議收藏

參考JGJ102-2003《玻璃幕牆工程技術規範》執行標準

（1）組合幕牆 composite curtain wall

由不同材料的面板（如玻璃、金屬、石材等）組成的建築幕牆

（2）玻璃幕牆 glass curtain wall

面板材料為玻璃的建築幕牆。

（3）斜玻璃幕牆 inclined building curtain wall

與水平面夾角大於75°且小於90°的玻璃幕牆。

（4）框支承玻璃幕牆 frame supported glass curtainwall

玻璃面板周邊由金屬框架支承的玻璃幕牆。主要包括下列類型：

（5）明框玻璃幕牆 exposed frame supported glasscurtain wall

金屬框架的構件顯露於面板外表面的框支承玻璃幕牆。

（6）隱框玻璃幕牆 hidden frame supported glasscurtain wall

金屬框架的構件完全不顯露於面板外表面的框支承玻璃幕牆。

（7）半隱框玻璃幕牆 semi-hidden frame supportedglass cur-tain wall

金屬框架的豎向或橫向構件顯露於面板外表面的框支承玻璃幕牆。

（8）單元式玻璃幕牆 frame supported glass curtainwall as-sembled in prefabricated units

將面板和金屬框架（橫樑、立柱）在工廠組裝為幕牆單元，以幕牆單元形式在現場完成安裝施工的框支承玻璃幕牆。

（9）構件式玻璃幕牆 frame supported glass curtainwall as-sembled in elements

在現場依次安裝立柱、橫樑和玻璃面板的框支承玻璃幕牆。

（10）全玻幕牆 full glass curtain wall

由玻璃肋和玻璃面板構成的玻璃幕牆。

（11）點支承玻璃幕牆 point-supported glass curtainwall

由玻璃面板、點支承裝置和支承結構構成的玻璃幕牆。

（12）支承裝置 supporting device

玻璃面板與支承結構之間的連接裝置。

（13）支承結構 supporting structure

點支承玻璃幕牆中，通過支承裝置支承玻璃面板的結構體系。

（14）鋼絞線 strand

由若干根鋼絲絞捻而成的螺旋狀鋼絲束。

（15）硅酮結構密封膠 structural silicone sealant

幕牆中用於板材與金屬構架、板材與板材、板材與玻璃肋之間的結構用硅酮粘接材料，簡稱硅酮結構膠。

（16）硅酮建築密封膠 weather proofing siliconesealant

幕牆嵌縫用的硅酮密封材料，又稱耐候膠。

（17）雙面膠帶 double-faced adhesive tape

幕牆中用於控制結構位置和截面尺寸的雙面塗膠的聚胺基甲酸乙酯或聚乙烯低泡材料。

（18）雙金屬腐蝕 bimetallic corrosion

由不同的金屬或其他電子導體作為電極而形成的電偶腐蝕。

（19）相容性 compatibility

粘接密封材料之間或粘接密封材料與其他材料相互接觸時，相互不產生有害物理、化學反應的性能。

（20）建築幕牆 building curtain wall

由金屬構架與板材組成的、不承擔主體結構荷載與作用的建築外圍護結構。

（21）金屬幕牆 metal curtain wall

板材為金屬板材的建築幕牆。

（22）石材幕牆 stone curtain wall

板材為建築石板的建築幕牆。

（23）單元建築幕牆 unit building curtain wall

由金屬構架、各種板材組裝成一層樓高單元板塊的建築幕牆。

（24）小單元建築幕牆 small unit building curtainwall

由金屬副框、各種單塊板材，採用金屬掛鉤與立柱、橫樑連接的可拆裝的建築幕牆。

（25）接觸腐蝕 contact corrosion

兩種不同的金屬接觸時發生的電化學腐蝕。

（26）永久荷載 permanent load

在結構使用期間，其值不隨時間變化，或其變化與平均值相比可以忽略不計，或其變化是單調的並能趨於限值的荷載。

（27）可變荷載 variable load

在結構使用期間，其值隨時間變化，且其變化與平均值相比不可以忽略不計的荷載。

（28）偶然荷載 accidental load

在結構使用期間不一定出現，一旦出現，其值很大且持續時間很短的荷載。

（29）荷載代表值 representative values of aload

設計中用以驗算極限狀態所採用的荷載量值，例如標準值、組合值、頻遇值和準永久值。

（30）設計基準值 design reference period

為確定可變荷載代表值而選用的時間參數。

（31）標準值 characteristic value/nominalvalue

荷載的基本代表值，為設計基準期內最大荷載統計分佈的特徵值（例如均值、眾值、中值或某個分位值）。

（32）組合值 combination value

對可變荷載，使組合後的荷載效應在設計基準期內的超越概率，能與該荷載單獨出現時的相應概率趨於一致的荷載值；或使組合後的結構具有統一規定的可靠指標的荷載值。

（33）頻遇值 frequent value

對可變荷載，在設計基準期間內，其超越的總時間為規定的較小比率或超越頻率為規定頻率的荷載值。

（34）準永久值 quasi-permanent value

對可變荷載，在設計基準期內，其超越的總時間約為設計基準期一半的荷載值。

（35）荷載設計值 design value of a load

荷載代表值與荷載分項係數的乘積。

（36）荷載效應 load effect

由荷載引起結構或結構構件的反應，例如內力、變形和裂縫等。

（37）荷載組合 load combination

按極限狀態設計時，為保證結構的可靠性而對同時出現的各種荷載設計值的規定。

（38）基本組合 fundamental combination

承載能力極限狀態計算時，永久作用和可變作用的組合。

（39）偶然組合 accidental combination

承載能力極限狀態計算時，永久作用、可變作用和一個偶然作用的組合。

（40）標準組合 characteristic/nominalcombination

正常使用極限狀態計算時，採用標準值或組合值為荷載代表值的組合。

（41）頻遇組合 frequent combinations

正常使用極限狀態計算時，對可變荷載採用頻遇值或準永久值為荷載代表值的組合。

（42）準永久組合 quasi-permanent combinations

正常使用極限狀態計算時，對可變荷載採用準永久值為荷載代表值的組合。

（43）等效均布荷載 equivalent uniform live load

結構設計時，樓面上不連續分佈的實際荷載，一般採用均布荷載代替；等效均布荷載係指其在結構上所得的荷載應能與實際的荷載效應保持一致的均布荷載。

（44）從屬面積 tributary area

從屬面積是在計算樑柱構件時採用，它是指所計算構件負荷的樓面面積，它應由樓板的剪力零線劃分，在實際應用中可作適當簡化。

（45）動力系數 dynamic coefficient

承受動力荷載的結構或構件，當按靜力設計時採用的係數，其值為結構或構件的最大動力效應與相應的靜力效應的比值。

（46）基本雪壓 reference snow pressure

雪荷載的基準壓力，一般按當地空曠平坦地面上積雪自重的觀測數據，經概率統計得出50年一遇最大值確定。

（47）基本風壓 reference wind pressure

風荷載的基準壓力，一般按當地空曠平坦地面上10ｍ高度處10min平均的風速觀測數據，經概率統計得出50年一遇最大值確定的風速，再考慮相應的空氣密度，按公式(D.2.2-4）確定的風壓。

（48）地面粗糙度 terrain roughness

風在到達結構物以前吹越過2km範圍內的地面時，描述該地面上不規則障礙物分佈狀況的等級。

（49）抗震設防烈度 seismic fortificationintensity

按國家規定的權限批准作為一個地區抗震設防依據的地震烈度。

（50）抗震設防標準 seismic fortificationcriterion

衡量抗震設防要求的尺度，由抗震設防烈度和建築使用功能的重要性確定。

（51）地震作用 earthquake action

由地震動引起的結構動態作用，包括水平地震作用和豎向地震作用。

（52）設計地震動參數 design parameters of groundmotion

抗震設計用的地震加速度（速度、位移）時程曲線、加速度反應譜和峰值加速度。

（53）設計基本地震加速度 design basic acceleration ofground motion

50年設計基準期超越概率10%的地震加速度的設計取值。

（54）設計特徵週期 design characteristic periodof ground motion

抗震設計用的地震影響係數曲線中，反映地震震級、震中距和場地類別等因素的下降段起始點對應的週期值。

（55）場地 site

工程群體所在地，具有相似的反應譜特徵，其範圍相當於廠區、居民小區和自然村或不小於1.0km2的平面面積。

（56）建築抗震概念設計 seismic concept design ofbuildings

根據地震災害和工程經驗等所形成的基本設計原則和設計思想，進行建築和結構總體佈置並確定細部構造的過程。

（57）抗震措施 seismic fortification measures

除地震作用計算和抗力計算以外的抗震設計內容，包括抗震構造措施。

（58）抗震構造措施 details of seismic design

根據抗震概念設計原則，一般不需計算而對結構和非結構各部分必須採取的各種細部要求。

（59）外部安裝 external installation

將玻璃的一面或兩面暴露在建築物外部的安裝方式。

（60）內部安裝 internal installation

玻璃的兩個表面都不暴露在建築物外部的安裝方式。

（61）玻璃框架 glass frame

由木材、金屬或其他耐久性材料或這些材料的組合所製成的用於安裝玻璃的結構。

（62）槽口 rabbet

是框架的一部分，其橫截面呈90º角，玻璃的邊部置於其內。

（63）凹槽 groove

是框架的一部分，其橫截面呈一凹形，玻璃的邊部置於其內。

（64）襯墊 bedding

位於槽內的安裝材料，在其內嵌入玻璃。

（65）壓條 beador installation bead

固定在槽口上夾固玻璃的木條、金屬條或其他剛性材料條。

（66）定位塊 location blocks

位於玻璃邊緣與槽之間，防止玻璃和槽產生相對運動的彈性材料塊。

（67）支承塊 setting blocks

位於玻璃的底邊與槽之間，起支承作用，並使玻璃位於槽內正中的彈性材料塊。

（68）彈性止動片 distance pieces

位於玻璃和槽豎直面之間，防止因荷載作用而引起玻璃運動的彈性材料片。

（69）密封墊 gasket

用於嵌入或固定玻璃的專門成型材料。

（70）前部擋邊 fronting

在槽口的平臺上，位於玻璃表面與槽前部邊緣之間的三角形的安裝材料條。

（71）縱橫比 aspect ratio

玻璃板的長邊與短邊之比。

（72）跨度 span

兩支撐部件之間的距離；對於四邊支撐的玻璃，它對應於透光尺寸中的較小值。

（73）可見線 sight line

玻璃在框架中裝配完畢，玻璃的透光部分與被玻璃安裝材料覆蓋的不透光部分的分界線。

（74）透光尺寸 sight size

玻璃裝配之後，透光部分的尺寸。

（75）固定門玻璃 side panels

安裝在門洞口處固定門框內的玻璃。

（76）安全玻璃 safe glass

指符合國家標準GB9962規定的夾層玻璃和符合國家標準GB9963規定的鋼化玻璃。

（77）有框架玻璃 framed panels

具有足夠剛度的支承部件連續地包住玻璃的所有邊。

（78）無框架玻璃 unframed panels

支承部件不符合有框架玻璃的規定時，該塊玻璃為無框架玻璃。

（79）普通退火玻璃 general annealing glass

由浮法、平拉法、有石曹垂直引上法、無石曹法等熔製成的，經熱處理消除或減小其內部應力至允許值的玻璃。

（80）暴露邊 exposed edges

無支承部件的玻璃邊緣為暴露邊，但兩塊玻璃相鄰並通過密封膠對接的情況除外。

（81）前部餘隙 front clearance

玻璃外側表面與壓條或凹槽前端豎直面之間的距離。

（82）後部餘隙 back clearance

玻璃內側表面與凹槽後端豎直面之間的距離。

（83）邊緣餘隙 edge clearancl

玻璃邊緣與凹槽底面之間距離。

（84）嵌入深度 edge cover

玻璃邊緣到可見線之間的距離。
