初中化學式、化學方程式和反應現象歸納大全

二、 化學方程式、反應現象、應用

1. 2Mg+O2點燃或Δ2MgO 劇烈燃燒.耀眼白光.生成白色固體.放熱.產生大量白煙 白色信號彈

2. 2Hg+O2點燃或Δ2HgO 銀白液體、生成紅色固體 拉瓦錫實驗

3. 4Al+3O2點燃或Δ2Al2O3 銀白金屬變為白色固體

4. 3Fe+2O2點燃Fe3O4 劇烈燃燒、火星四射、生成黑色固體、放出大量熱

5. C+O2 點燃CO2 劇烈燃燒、白光、放熱、使石灰水變渾濁

6. S+O2 點燃SO2 劇烈燃燒、放熱、刺激味氣體、空氣中淡藍色火焰.氧氣中藍紫色火焰

7. 2H2+O2 點燃2H2O 淡藍火焰、放熱、生成使無水CuSO4變藍的液體（水） 高能燃料

8. 4P+5O2 點燃2P2O5 劇烈燃燒、大量白煙、放熱、生成白色固體 證明空氣中氧氣含量

9. CH4+2O2點燃2H2O+CO2 藍色火焰、放熱、生成使石灰水變渾濁氣體和使無水CuSO4變藍的液體（水） 甲烷和天然氣的燃燒

10. 2KClO3 MnO2 Δ 2KCl +3O2↑ 生成使帶火星的木條復燃的氣體 實驗室製備氧氣

11. 2KMnO4Δ K2MnO4+MnO2+O2↑ 紫色變為黑色、生成使帶火星木條復燃的氣體 實驗室製備氧氣

12. 2HgOΔ2Hg+O2↑ 紅色變為銀白、生成使帶火星木條復燃的氣體 拉瓦錫實驗

13. 2H2O通電 2H2↑+O2↑ 水通電分解為氫氣和氧氣 電解水

14. Cu2(OH)2CO3Δ2CuO+H2O+CO2↑ 綠色變黑色、試管壁有液體、使石灰水變渾濁氣體 銅綠加熱

15. NH4HCO3ΔNH3↑+ H2O +CO2↑ 白色固體消失、管壁有液體、使石灰水變渾濁氣體 碳酸氫銨長期暴露空氣中會消失

16. Zn+H2SO4=ZnSO4+H2↑ 有大量氣泡產生、鋅粒逐漸溶解 實驗室製備氫氣

17. Fe+H2SO4=FeSO4+H2↑ 有大量氣泡產生、金屬顆粒逐漸溶解

18. Mg+H2SO4 =MgSO4+H2↑ 有大量氣泡產生、金屬顆粒逐漸溶解

19. 2Al+3H2SO4=Al2(SO4)3+3H2↑ 有大量氣泡產生、金屬顆粒逐漸溶解

20. Fe2O3+3H2 Δ 2Fe+3H2O 紅色逐漸變為銀白色、試管壁有液體 冶煉金屬、利用氫氣的還原性

21. 2Na+Cl2Δ或點燃2NaCl 劇烈燃燒、黃色火焰 離子化合物的形成、

22. H2+Cl2 點燃或光照 2HCl 點燃蒼白色火焰、瓶口白霧 共價化合物的形成、製備鹽酸

23. CuSO4+2NaOH=Cu(OH)2↓+Na2SO4 藍色沉澱生成、上部為澄清溶液 質量守恆定律實驗

24. 2C +O2點燃2CO 煤爐中常見反應、空氣汙染物之一、煤氣中毒原因

25. 2CO+O2點燃2CO2 藍色火焰 煤氣燃燒

26. C + CuO 高溫2Cu+ CO2↑ 黑色逐漸變為紅色、產生使澄清石灰水變渾濁的氣體 冶煉金屬

27. 2Fe2O3+3C 高溫4Fe+ 3CO2↑ 冶煉金屬Fe3O4+2C高溫3Fe + 2CO2↑ 冶煉金屬

28. C + CO2 高溫2CO

29. CO2 + H2O = H2CO3 碳酸使石蕊變紅 證明碳酸的酸性

30. H2CO3 ΔCO2↑+ H2O 石蕊紅色褪去

31. Ca(OH)2+CO2= CaCO3↓+ H2O 澄清石灰水變渾濁 應用CO2檢驗和石灰漿粉刷牆壁

32. CaCO3+H2O+CO2 = Ca(HCO3)2 白色沉澱逐漸溶解 溶洞的形成，石頭的風化

33. Ca(HCO3)2 Δ CaCO3↓+H2O+CO2↑ 白色沉澱、產生使澄清石灰水變渾濁的氣體 水垢形成.鐘乳石的形成

34. 2NaHCO3ΔNa2CO3+H2O+CO2↑ 產生使澄清石灰水變渾濁的氣體 小蘇打蒸饅頭

35. CaCO3 高溫 CaO+ CO2↑ 工業製備二氧化碳和生石灰

36. CaCO3+2HCl=CaCl2+ H2O+CO2↑ 固體逐漸溶解、有使澄清石灰水變渾濁的氣體 實驗室製備二氧化碳、除水垢

37. Na2CO3+H2SO4=Na2SO4+H2O+CO2↑ 固體逐漸溶解、有使澄清石灰水變渾濁的氣體 泡沫滅火器原理

38. Na2CO3+2HCl=2NaCl+ H2O+CO2↑固體逐漸溶解、有使澄清石灰水變渾濁的氣體 泡沫滅火器原理

39. CuO +CO Δ Cu + CO2 黑色逐漸變紅色，產生使澄清石灰水變渾濁的氣體 冶煉金屬

40. Fe2O3+3CO高溫 2Fe+3CO2 冶煉金屬原理

41. C2H5OH+3O2點燃2CO2+3H2O 藍色火焰、產生使石灰水變渾濁的氣體、放熱 酒精的燃燒

42. Fe+CuSO4=Cu+FeSO4 銀白色金屬表面覆蓋一層紅色物質 溼法鍊銅、鍍銅

43. Mg+FeSO4= Fe+ MgSO4 溶液由淺綠色變為無色

44. Cu+2AgNO3=2Ag+ Cu(NO3)2 紅色金屬表面覆蓋一層銀白色物質 鍍銀

45. Zn+CuSO4= Cu+ZnSO4 青白色金屬表面覆蓋一層紅色物質 鍍銅

46. Fe2O3+6HCl=2FeCl3+3H2O 鐵鏽溶解、溶液呈黃色 鐵器除鏽

47. Al2O3+6HCl=2AlCl3+3H2O 白色固體溶解

48. CuO+2HCl=CuCl2+H2O 黑色固體溶解、溶液呈藍色

49. MgO+2HCl=MgCl2+ H2O 白色固體溶解 CaO+2HCl=CaCl2+ H2O 白色固體溶解

50. NaOH+HCl=NaCl+ H2O 白色固體溶解

51. Cu(OH)2+2HCl=CuCl2+2H2O 藍色固體溶解

52. Mg(OH)2+2HCl=MgCl2+2H2O 白色固體溶解

53. Al(OH)3+3HCl=AlCl3+3H2O 白色固體溶解 胃舒平治療胃酸過多

54. Fe(OH)3+3HCl=FeCl3+3H2O 紅褐色沉澱溶解、溶液呈黃色

55. Ca(OH)2+2HCl=CaCl2+2H2O

56. HCl+AgNO3= AgCl↓+HNO3 生成白色沉澱、不溶解於稀硝酸 檢驗Cl—的原理

57. Fe2O3+3H2SO4= Fe2(SO4)3+3H2O 鐵鏽溶解、溶液呈黃色 鐵器除鏽

58. Al2O3+3H2SO4= Al2(SO4)3+3H2O 白色固體溶解

59. CuO+H2SO4=CuSO4+H2O 黑色固體溶解、溶液呈藍色

60. 2NaOH+H2SO4=Na2SO4+2H2O

61. Cu(OH)2+H2SO4=CuSO4+2H2O 藍色固體溶解

62. 2Al(OH)3+3H2SO4=Al2(SO4)3+3H2O 白色固體溶解

63. 2Fe(OH)3+3H2SO4=Fe2(SO4)3+3H2O 紅褐色沉澱溶解、溶液呈黃色

64. Ba(OH)2+ H2SO4=BaSO4↓+2H2O 生成白色沉澱、不溶解於稀硝酸 檢驗SO42—的原理

65. BaCl2+ H2SO4=BaSO4↓+2HCl 生成白色沉澱、不溶解於稀硝酸 檢驗SO42—的原理

66. Ba(NO3)2+H2SO4=BaSO4↓+2HNO3 生成白色沉澱、不溶解於稀硝酸 檢驗SO42—的原理

67. CuO+2HNO3=Cu(NO3)2+H2O 黑色固體溶解、溶液呈藍色

68. NaOH+HNO3=NaNO3+ H2O

69. Cu(OH)2+2HNO3=Cu(NO3)2+2H2O 藍色固體溶解

70. Fe(OH)3+3HNO3=Fe(NO3)3+3H2O 紅褐色沉澱溶解、溶液呈黃色

71. 2NaOH+CO2=Na2CO3+ H2O 吸收CO、O2、H2中的CO2、

72. 2NaOH+SO2=Na2SO3+ H2O 2NaOH+SO3=Na2SO4+ H2O 處理硫酸工廠的尾氣（SO2）

73. FeCl3+3NaOH=Fe(OH)3↓+3NaCl 溶液黃色褪去、有紅褐色沉澱生成

74. AlCl3+3NaOH=Al(OH)3↓+3NaCl 有白色沉澱生成

75. MgCl2+2NaOH = Mg(OH)2↓+2NaCl

76. CuCl2+2NaOH = Cu(OH)2↓+2NaCl 溶液藍色褪去、有藍色沉澱生成

77. CaO+ H2O = Ca(OH)2 白色塊狀固體變為粉末、 生石灰製備石灰漿

78. Ca(OH)2+Na2CO3=CaCO3↓+2NaOH 有白色沉澱生成 工業制燒鹼、實驗室制少量燒鹼

79. Ba(OH)2+Na2CO3=BaCO3↓+2NaOH 有白色沉澱生成

80. Ca(OH)2+K2CO3=CaCO3↓ +2KOH 有白色沉澱生成

81. CuSO4+5H2O= CuSO4·H2O 白色粉末變為藍色 檢驗物質中是否含有水

82. CuSO4·H2OΔ CuSO4+5H2O 藍色晶體變為白色粉末

83. AgNO3+NaCl = AgCl↓+NaNO3 白色不溶解於稀硝酸的沉澱（其他氯化物類似反應）

應用於檢驗溶液中的氯離子

84. BaCl2 + Na2SO4 = BaSO4↓+2NaCl 白色不溶解於稀硝酸的沉澱（其他硫酸鹽類似反應）

應用於檢驗硫酸根離子

85. CaCl2+Na2CO3= CaCO3↓+2NaCl 有白色沉澱生成

86. MgCl2+Ba(OH)2=BaCl2+Mg(OH)2↓ 有白色沉澱生成

87. CaCO3+2HCl=CaCl2+H2O+CO2↑

88. MgCO3+2HCl= MgCl2+H2O+ CO2↑
