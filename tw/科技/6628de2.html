一、命令描述

1、chattr：

作用：設置文件的擴展屬性，並且能夠提高系統的安全性。

2、lsattr：

作用：用於顯示chattr設置的屬性。

二、命令格式

1、chattr：

chattr [-RVf] [-+=aAcCdDeijsStTu] [-v version] files...

2、lsattr：

lsattr [-RVadlv] [files...]

三、命令幫助信息

1、chattr：

[root@localhost ~]# man chattr

-------------------------------------------------

NAME

chattr - change file attributes on a Linux file system

SYNOPSIS

chattr [ -RVf ] [ -v version ] [ mode ] files...

DESCRIPTION

chattr changes the file attributes on a Linux file system.

OPTIONS

-R Recursively change attributes of directories and their contents.

-V Be verbose with chattr's output and print the program version.

-f Suppress most error messages.

-v version

Set the file's version/generation number.

2、lsattr：

[root@localhost ~]# man lsattr

------------------------------------------

NAME

lsattr - list file attributes on a Linux second extended file system

SYNOPSIS

lsattr [ -RVadv ] [ files... ]

DESCRIPTION

lsattr lists the file attributes on a second extended file system. See chattr(1) for a

description of the attributes and what they mean.

OPTIONS

-R Recursively list attributes of directories and their contents.

-V Display the program version.

-a List all files in directories, including files that start with " . " .

-d List directories like other files, rather than listing their contents.

-v List the file's version/generation number.

四、命令選項

chattr：

  1. [ -R ]：遞歸設置指定目錄及其目錄下的所有文件和子目錄的屬性；
  2. [ -V ]：詳細顯示目錄和文件的屬性設置情況；
  3. [ + ]：增加某項屬性；
  4. [ - ]：取消某項屬性；
  5. [ = ]：設置文件或目錄只具有哪些屬性；
  6. [ a ]：只允許項文件追加數據，不允許任何進程覆蓋重寫文件的內容；
  7. [ A ]：設置不允許系統更新文件的最後訪問時間；
  8. [ i ]：不可修改屬性。具有該屬性的文件不能被刪除、更名或修改其內容；
  9. [ c ]：壓縮屬性，寫入或讀取時，系統會自動進行壓縮或解壓縮；
  10. [ s ]：安全刪除屬性。即在刪除一個文件時，會使用0填充源文件所在的存儲區，以使回覆工具無法恢復出文件的內容；
  11. [ u ]：可恢復性的刪除。對於具有該屬性的文件，在刪除時，系統會保留其數據塊一遍以後能恢復該文件的內容。

lsattr：

  1. [ -a ]：列出目錄中的所有文件（包括隱藏文件）；
  2. [ -d ]：以與文件相同的方式列出目錄的擴展屬性；
  3. [ -R ]：以遞歸的方式列出目錄的屬性及其內容。


五、練習

為了更好的掌握這些命令，接下來我們來一些簡單的練習

練習一：將/root/c.txt文件設置為具有不可修改屬性和不允許系統更新文件的最後訪問時間屬性，並查看設置後的屬性。

[root@localhost ~]# chattr =ia /root/c.txt // [ i ]選項代表不可修改；[ a ]選項代表不允許更新文件的最後訪問時間

[root@localhost ~]# lsattr /root/c.txt

----ia---------- /root/c.txt
