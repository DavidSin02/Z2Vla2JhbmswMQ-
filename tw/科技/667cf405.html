Python的輸入與輸出

一、輸出格式美化

Python常見的有兩種輸出值的方式：表達式語句和print()函數。而第三種方式是使用文件對象的write()方法，標準輸出文件可以用sys.stdout引用。

如果你希望輸出的格式更加多樣，可以使用str.format()函數來格式化輸出值。

如果你希望將輸出的值轉換成字符串，可以使用repr()函數或str()函數來實現。

其中:

  • str()函數返回一個用戶易讀的表達形式
  • repr()函數產生一個解釋器易讀的表達形式

例如：

s1 = '大家好，我正在學習Python！'print(str(s1)) # 大家好，我正在學習Python！print(repr(s1)) # '大家好，我正在學習Python！'# 三種方法輸出1~10的平方和立方# rjust()可以將字符串靠右，並在左邊填充空格 括號內為輸出值的位數for m in range(1,11):    print(repr(m).rjust(2), repr(m*m).rjust(3), repr(m*m*m).rjust(4))# 0，1，2分別對應format中的三個值，:2d表示兩個寬度的十進制，依次是三個和四個寬度for n in range(1,11):    print('{0:2d} {1:3d} {2:4d}'.format(n, n*n, n*n*n))# 使用f-string的方式格式化輸出for p in range(1,11):    print(f'{str(p).rjust(2)} {str(p*p).rjust(3)} {str(p*p*p).rjust(4)}')# zfill() 會在數字的左邊填充0  括號內為輸出值的位數for q in range(1,11):    print(repr(q).zfill(0), repr(q*q).zfill(3), repr(q*q*q).zfill(4))

二、format

括號及裡面的字符(稱為格式化字段)將會被format()中的參數替換。花括號{}中的數字用於指向傳入對象在format中的位置。

例子：

print('{0}和{1}是電影的男生角和女生角'.format('張國榮','毛舜筠')) # 張國榮和毛舜筠是電影的男生角和女生角print('英雄本色演員有{0},{1},和{2},其中，{2}是個內應'.format('張國榮','周潤發','狄龍')) # 電影英雄本色的主要演員有張國榮,周潤發,和狄龍,其中，狄龍是個內應# 也可以使用關鍵字參數print('{0}教會了{1}的{kongfu}'.format('風清揚','令狐沖', kongfu='獨孤九劍')) #風清揚教會了令狐沖的獨孤九劍[25]# :後傳入一個整數，可以保證該域至少有這麼多的寬度，用於美化格式很有用import math# 保留PI小數點後三位print('圓周率取三位小數：{:.3f}'.format(math.pi)) #圓周率取三位小數：3.142# 對字典中的數值，使用**輸出d1 = {'百度':100, '阿里':200,'拼多多':300, '今日頭條':400,'京東':'W'}print('百度:{百度:2d}; 阿里:{阿里:3d}'.format(**d1)) # 百度:100; 阿里:200

三、input

Python提供了input()內置函數從標準輸入讀入一行文本，默認的標準輸入是鍵盤, 其中，input可以接收一個Python表達式作為輸入，並將運算的結果返回

示例

print('=================代碼演示input====================')name = input('請輸入你的姓名：')print(f'你輸入的名字是：{name}')

四、文件的讀和寫

  • open(): 將會返回一個file對象，基本的語法格式如下：
open(filename, mode)# filename: 包含了你要訪問的文件名稱的字符串值# mode: 決定了打開文件的模式，有隻讀，寫入，追加等方式。默認方式是隻讀(r)

在Python中，打開文件的模式主要如下表所示：

文件讀寫mode的模式

  • 文件對象的常用方法

通過open(filename, mode, encoding)可以創建一個文件對象，那麼，創建好的文件對象有哪些常用的方法呢？

    • read():讀取一個文件的內容，調用read(size)，這將讀取一定數量的數據，然後作為字符串或字節對象返回， 其中，size是一個可選的數字類型的參數。當size被忽略時或者取負值時，那麼，該文件的所有內容都將被讀取並返回。

read()方法

    • readline(): 會從文件中讀取單獨的一行。換行符為'\n'。如果返回一個空的字符串，說明已經讀取到了最後一行。

readline()

    • readlines(): 將返回該文件包含的所有行,返回列表。

readlines()

    • write(string): 將string寫入到文件中，然後返回寫入的字符數量，如果寫入的不是字符串，則需要先進行轉換。
# 打開一個不存在的文件, 寫入字符串, 會先創建該文件with open('mynewfile.txt','w+') as f:    f.write('風清揚最喜歡令狐沖，所以教他獨孤九劍!')# 寫入一個非字符的時候，需要使用str()將其轉換成字符形式，否則會報錯with open('mynewfile.txt','a+', encoding='utf8') as f:    t1 = ('嶽不群', 100, '令狐沖', 200, '東方不敗', {'name':'鹿鼎記', 'author':'金庸'})    # f.write(t1) # TypeError: write() argument must be str, not tuple    f.write(str(t1))
    • tell(): 返回該文件對象當前所處的位置，它是從文件開頭開始算起的字節數。
    • seek(offset, from_what): 如果要改變文件的當前位置，可以使用該函數，其中，對於from_what: 0表示開頭(默認值)，1表示當前位置，2表示文件的結尾，例如：
seek(x,0): 從起始位置即文件首行字符開始移動x個字符；seek(x,1): 表示從當前位置向後移動x個字符seek(-x,2): 表示從文件結尾往前移動x個字符
    • close()：當你處理完一個文件後，調用close()方法來關閉文件並釋放系統的資源，如果嘗試再調用該文件，則會拋出異常。
