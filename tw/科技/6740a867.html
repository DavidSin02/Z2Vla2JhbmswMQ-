VC｜通俗理解編輯框中文本的讀寫、插入、剪切、查找、替換等操作

2020-10-29
| 科技

1 添加編輯框(Rich Edit)

1.1 編輯框ID：IDC_textbox

1.2 編輯框關連的control變量：m_textblock

2 變量聲明和定義

CString str;int nStart, nEnd;

3 得到編輯框文本的方式

有8種訪問對話框控件的方式：

GetDlgItem()->Get(Set)WindowText()

GetDlgItemText()/SetDlgItemText()

GetDlgItemInt()/SetDlgItemInt()

將控件和普通變量相關聯，用UpdateData();交換數據

將控件和控件變量相關聯

SendMessage()

SendDlgItemMessage()

GetSelText()

如：

3.1

GetDlgItem(IDC_textbox)->GetWindowText(str); //GetDlgItem()返回的是一個CEdit *指針

3.2

m_textblock.GetSel(nStart, nEnd);str = m_textblock.GetSelText();

3.3

對於普通編輯框關連的CString類型的變量，可通過UpdateData()函數直接與控件交換數據。

4 文本插入

4.1 在編輯框全部文本的前後插入文本

4.1.1

m_textblock.SetSel(0, 0);m_textblock.ReplaceSel("<pre>");m_textblock.SetSel(-1, -1);m_textblock.ReplaceSel("</pre>");

4.1.2

m_textblock.SetSel(0, -1);str = m_textblock.GetSelText();str = "<pre>" + str + "</pre>";GetDlgItem(IDC_textbox)->SetWindowText(str);

4.1.3

GetDlgItem(IDC_textbox)->GetWindowText(str);str = "<pre>" + str + "</pre>";GetDlgItem(IDC_textbox)->SetWindowText(str);

4.2 在編輯框文本的中間插入文本（或換行）

str = "\n\r";m_textblock.ReplaceSel(str);

5 文本替換

任務：如果編輯框沒有選擇文本，則在全部文本的前面加上<pre>，在全部文本的後面加上</pre>。如果編輯框有選擇文本，則在選擇的文本前後操作。

 m_textblock.GetSel(nStart, nEnd); if(nStart == nEnd)//如果nStart和nEnd相等，表明光標處於某個位置（直觀來看就是光標在閃動）； {		m_textblock.SetSel(0, -1);		str = m_textblock.GetSelText();		str = "<pre>" + str + "</pre>";		GetDlgItem(IDC_textbox)->SetWindowText(str); } else { 		str = m_textblock.GetSelText();		str = "<pre>" + str + "</pre>";		m_textblock.ReplaceSel(str);}

也可以利用CString類的Relace()方法做文本替換：

	CString strtxt,strold,strnew;	GetDlgItem(IDC_textbox)->GetWindowText(strtxt);	GetDlgItem(IDC_title)->GetWindowText(strold);	GetDlgItem(IDC_new)->GetWindowText(strnew);	strtxt.Replace(strold,strnew);	GetDlgItem(IDC_textbox)->SetWindowText(strtxt);

6 編輯框內文本的全選、複製、剪切、粘貼、清空、回退操作

m_textblock.SetSel(0, -1);m_textblock.Copy();m_textblock.Cut();m_textblock.Paste();m_textblock.Clear();CEdit* edit1 = (CEdit*)GetDlgItem(IDC_textbox);edit1->SetWindowText("");m_textblock.Undo();	m_textblock.SetFocus();

7 將光標停置於編輯框中默認文本的特定位置

如編輯框設置瞭如下默認文本：

hi, Very happy to write to you.     Attached please find! Any questions, please don't hesitate to let me know. You're early reply will be highly appreciated. Best Regards Yours Sincerely

想讓光標停靠在第三行空白處：

m_body.SetSel(45,45);

8 編輯框中文本的查找操作

	CString blocks,finds;	int start,end,lengths,lengths2, num;	start =0, end = 0, lengths = 0, lengths2 = 0, num =0;	GetDlgItem(IDC_textbox)->GetWindowText(blocks);	GetDlgItem(IDC_title)->GetWindowText(finds);	lengths = finds.GetLength();	start = blocks.Find(finds);		while (start >= 0)	{		lengths2 = blocks.GetLength();		start = blocks.Find(finds); //從第start +1開始，如是在第一個字符找到，則start = 0;		if((start+lengths) > (lengths2 - start+lengths))		{			break;		}		blocks = blocks.Mid(start+lengths, lengths2-start-lengths);		num++;	}		char str3[125];	itoa(num,str3,10);	CString textnum = str3;	textnum = "找到："+ textnum + "個！";	GetDlgItem(IDC_new)->SetWindowText(textnum);
