原來Word和Excel都自帶一鍵合併功能！早知道就不用複製粘貼了

2020-10-29
| 科技

我們在工作中經常需要將各種各樣的文檔合併，最常見的就是Word、Excel和PDF文件了，很多小夥伴都是用複製粘貼來合併文件的，這樣的效率實在是不高，那麼有什麼辦法可以快速合併多個文件呢？今天小編就來分享一些關於Word、Excel、PDF文件合併的方法，有需要的小夥伴可以學一學哦！

一、Word合併

1、插入對象

當我們需要合併多個Word文檔的時候，我們首先可以先打開一個空白的文檔，然後點擊工具欄的【插入】—【對象】—【從文件中選擇文字內容】。如下圖所示：

2、Ctrl多選

然後在彈出來的窗口中找到我們要合併的文件，在這裡我們可以按Ctrl鍵同時選中多個文件，然後點擊下方的【插入】按鈕即可！

二、Excel合併

1、VBA

有時候我們的Excel文件中有好幾個工作表，那麼我們要如何快速合併呢？首先，我們可以按下【Alt+F11】打開VBA界面，然後插入一個模塊。

2、代碼

然後將下面的代碼複製粘貼進去：

Sub 多表合併()

Dim arr()

For Each sh In Sheets

If sh.Name <> "彙總" Then

arr1 = sh.Range("a2:c" & sh.UsedRange.Rows.Count)

act = act + UBound(arr1)

ReDim Preserve arr(1 To 3, 1 To act) '

For j = 1 To UBound(arr1)

n = n + 1

arr(1, n) = arr1(j, 1)

arr(2, n) = arr1(j, 2)

arr(3, n) = arr1(j, 3)

Next

End If

Next

Sheets("彙總").[a2].Resize(n, 3) = Application.Transpose(arr)

End Sub

最後只要點一下這個按鈕即可合併所有工作表了！
