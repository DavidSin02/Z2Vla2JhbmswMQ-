文件的操作方法

文件只讀

  • 只讀模式
    • r: 以只讀方式打開文件。文件的指針將會放在文件的開頭。
    • rb: 以二進制格式打開一個文件進行只讀。指針將會放在文件的開頭。一般進行非文本文件如圖片等。
f1 = open('文件位置', mode='r', encoding='utf-8')  # 文件位置可以為絕對為位置,在根目錄下開始的位置,在與程序在相同目錄下的為相對位置# mode 填寫讀寫方式   r:文件只讀   rb:  f = open('文件位置', mode='rb'）  不需寫encoding,默認編碼方式是bytes類型content = f1.read()print(content)f1.close()# 可以定義讀的多少  以字符為單位f2 = open('sos.txt', mode='r', encoding='utf-8')content_2 = f2.read(2)  # 讀出來的都是字符print(content_2)f2.close()

文件只寫

  • 只寫模式
    • w: 打開一個文件只進行寫入。如果該文件已存在則打開文件,則把原文件的內容全部清除再寫。如果該文件不存在,則創建新文件。
    • wb: 以二進制形式打開一個文件只進行寫入。如果該文件已存在則打開文件,則把原文件的內容全部清除再寫。如果該文件不存在,則創建新文件。一般進行非文本文件如圖片。
# f = open('sos.txt', mode='w', encoding='utf-8')# f.write('3838438')  # .write:所需寫入的內容# f.close()f3 = open('sos.jpeg', mode='wb')f3.write('6666666'.encode('utf-8'))  # 默認編碼方式用.encode定義編碼方式f3.close()

文件追加

  • 追加模式
    • a: 打開一個文件進行追加。如果該文件已存在，則將指針放在文件的結尾。新的內容將會被寫入到已有內容之後。如果該文件不存在，則創建新文件進行寫入。
    • ab: 以二進制的形式打開一個文件進行追加，如果文件存在則在文件結尾處將新的內容寫入，如果不存在則創建新文件寫入。
f4 = open('sos.txt', mode='a', encoding='utf-8')f4.write('88888')f4.close()f5 = open('sos.jpeg', mode='ab')f5.write('999'.encode('utf-8'))f5.close()

文件讀寫

  • 讀寫模式
    • r+ : 打開一個文件進行讀寫。指針將會放在文件的開頭。
    • r+b: 以二進制格式打開一個文件進行讀寫。指針將會放在文件的開頭。一般進行非文本文件的讀寫。
f6 = open('sos.txt', mode='r+', encoding='utf-8')print(f6.read())f6.write('中國')f6.close()f7 = open('sos', mode='r+b')   # bytes類型 讀也會顯示為bytes類型 注意漢子print(f7.read())f7.write('38'.encode('utf-8'))f7.close()

文件寫讀

  • 寫讀模式
    • w+: 打開一個文件進行讀寫。如果該文件已存在則打開文件，並從開頭開始編輯，即原有內容會被刪除。如果該文件不存在，則創建新文件。
    • wb+: 以二進制格式打開一個文件進行讀寫。如果該文件已存在則打開文件，從開頭開始編輯，即原有內容會被刪除。如果該文件不存在，創建新文件。一般進行非文本文件處理等。
    • PS: 如果不加seek 則不會讀出東西
f8 = open('sos.txt', mode='w+', encoding='utf-8')f8.write('ss,dd')f8.seek(0)  # 移動光標到句頭位置print(f8.read())f8.close()

操作光標

  • seek:操作光標是按字節去找(注意中文1字符=3字節)
f9 = open('sos.text', mode='r+', encoding='utf-8')counter = f9.seek(3)  # 是按字節定光標的位置print(counter)print(f9.read())f9.close()

獲取光標位置

  • .tell:獲取光標位置
f0 = open('sos.txt', mode='r+', encoding='utf-8')counter = f0.seek(3)print(f0.tell())  # 告知光標的位置f0.close()

自動關閉close

  • 關鍵詞:with # 可以打開多個文件
with open('sos.txt', mode='r', encoding='utf-8') as fs,\    open('day_5.txt', mode='r', encoding='utf-8') as fs1:    print(fs.read(), fs1.read())
