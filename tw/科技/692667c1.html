Nature Highlight：信息碼存儲，靜態到動態的飛躍

2020-11-14
| 科技

信息碼利用其自身的形狀能夠存儲特定的信息，例如，一維條形碼和二維碼分別用線條和特定的幾何圖形帶代表存儲的信息，彩色三維碼是建立在傳統黑白二維碼基礎之上發展而來的一種全新圖像信息碼，它由紅藍綠黑四種顏色的矩陣而構成。以往的這些信息碼一般只能存儲特定的信息，是一種靜態存儲，其內容不能改變。

美國德克薩斯大學奧斯汀分校（University of Texas at Austin） Jonathan L. Sessler教授（通訊作者）和吉曉帆博士（論文第一作者），江蘇大學龍凌亮教授（共同通訊作者）和浙江大學黃飛鶴教授（共同通訊作者）合作，基於多重熒光的超分子聚合物水凝膠製備了響應型的熒光三維碼，分別用物理和化學的方式實現了其存儲信息內容的轉變。他們將聚丙烯酰胺通過N,N’-亞甲基雙丙烯酰胺（化學）和大環／陰離子複合物（物理）共同交聯形成水凝膠，引入不同的熒光基團形成了紅藍綠三種熒光顏色的超分子聚合物水凝膠（圖一）。由於大環／陰離子這一非共價鍵交聯不僅存在水凝膠的內部，而且也分佈在其表面，這三種熒光水凝膠能夠通過界面的非共價鍵作用力組裝在一起形成多重熒光的超分子聚合物水凝膠，再通過調整三種水凝膠的數量和位置分佈就可以在黑色基質上形成不同的熒光三維碼（圖二a，如Code A, Code B和 Code C)。

圖一、無熒光水凝膠G0, 藍色熒光水凝膠G1, 綠色熒光水凝膠G2, 非響應型紅色熒光水凝膠G3-1, 以及對氨氣具有響應的紅色熒光水凝膠G3-2的化學結構和卡通示意圖。

相對於傳統的信息碼，這些熒光三維碼有新的特點（圖二b），例如：（1）各種熒光顏色只在紫外燈照射下才顯示，其存儲的信息能夠得到保護； （2）非共價鍵交聯具有可逆性，熒光三維碼被破壞後可以進行自修復；（3）各種熒光會在添加特定的金屬離子後發生淬滅，其存儲的信息能夠在需要的時候得到刪除。

圖二、卡通示意圖：a) 熒光水凝膠G1, G2, 和 G3(G3-1和 G3-2)通過界面的非共價鍵作用力組裝熒光三維碼。b) 對Code A進行信息保護，信息自修復和信息刪除。c) 通過物理和化學方法，對信息碼存儲的信息進行轉換。

更有趣的是，其存儲的信息可以進行轉換（圖二c），例如，（1）這種熒光三維碼是三種熒光水凝膠通過界面可逆的非共價鍵作用力組裝在一起而形成，所以通過可逆的拆分／重組裝，即物理方法可以實現Code B到 Code A的轉換，其存儲信息也從Info B變為Info A；（2）引入一個具有對鹼性物質響應性的紅色熒光水凝膠，其熒光顏色能夠在接觸鹼性氣體（如氨氣）後從紅色變成藍色，這樣Code C能夠在氨氣條件下轉化成 Code A，即化學方法實現了存儲信息從Info C到Info A的轉換。因此，這項工作分別用物理和化學的方法實現了信息碼存儲信息的內容轉變。

另外，由於水凝膠材料自身的柔軟性以及水凝膠之間超分子組裝的穩定性，這種熒光三維碼可以作為一種具有信息存儲功能的可穿戴材料。

此項工作發表在Advanced Materials上， 此外，本文還入選Nature Highlight欄目。
