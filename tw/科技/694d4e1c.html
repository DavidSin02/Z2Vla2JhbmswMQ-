Word、Excel、PDF、PPT等格式居然能隨意轉換！方法也太簡單了

2020-11-14
| 科技

復工後大家的事情都比較多，身邊的朋友也經常抱怨工作做不完，天天都要處理各種文件，各種格式轉來轉去，實在麻煩。

其實對於小編來說，轉換文檔格式是一件很簡單的事情，根本花不了什麼時間，今天就把方法整理給大家！



一、文件轉換成PDF

想要將Word、Excel、PPT轉換成PDF格式其實很簡單，不用第三方工具或插件，直接另存為就能實現；

方法：打開文件 → 另存為 → 在保存類型下找到並選擇"PDF"就可以了；

二、PPT轉圖片

PPT轉換成圖片格式有很多小夥伴採用一張張保存圖片的方法，這樣太影響效率啦，不妨試試下面這個方法：

打開文件 → 導出 → 更改文件類型 → 在圖片文件類型下選擇要導出的圖片格式就可以啦！

三、Word轉Excel

Word想要轉換成Excel，如果使用複製黏貼的方法表格容易變形，所以不妨試試下面的方法吧：

打開Word，文件 → 另存為 → 在保存類型中選擇另存為網頁文件格式；

新建一個空白的Excel表格，點擊文件 → 打開 → 瀏覽 → 選擇剛剛保存的網頁格式的Word文檔並打開就可以啦~

四、Excel轉Word

打開Excel表格，複製全部內容，打開Word文檔，在鍵盤上按下快捷鍵【Ctrl+Alt+V】，然後在彈窗中點擊【Microsoft Excel 工作表對象】就好啦！

五、各種格式通用的轉換方法

如果你覺上面的方法還是不夠方便，想要一招解決所有文檔格式轉換的話，那就用下面的方法吧↓

藉助工具：迅捷PDF轉換器（可免費轉換小文件；不限於此工具，大家有其它的選擇也可以）

使用工具就很簡單了，只需選擇格式 → 添加文件 → 開始轉換就行了。
