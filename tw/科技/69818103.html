一、事務

1. 開始事務

start transaction

2. 事務開啟

set autocommit=1

3. 事務關閉

set autocommit=0

4. 提交事務

commit

5. 回滾事務

rollback

6. mysql默認不開始事務，需要手動開啟事務

7. 自動提交開啟，沒有事務；自動提交關閉，開始事務；

8. 查看事務是否開啟

show variables like 'autocommit';

9. 查看事務隔離級別

show variables like 'tx%';

10. 添加事務隔離級別

set tx_isolation = 'XXX';

(1) READ-UNCOMMITTED，髒讀：讀了未提交的數據

(2) READ-COMMITTED，幻讀：前後讀取的數據不一致

(3) repeatable-read，可重複讀：前後讀取數據一致

(4) serializable，串行化，事務依次執行，不能同時執行

二、視圖

1. 查看視圖

show tables;

desc 視圖名字;

show create view 視圖名字;

2. 創建索引

create index index_name on tb1(age)

3. 導出數據庫

mysqldump -uroot -p --default-character-set=utf8 數據庫名字 > 導出的位置

4. 導入數據庫

mysql -uroot -p --default-character-set=utf8 數據庫名字 < 導入的位置

5. 查看字符集和排序規則

show variables like 'character%';

show variables like 'collation_%';

6. 修改表中的列

alter table tb1 modify neirong;
