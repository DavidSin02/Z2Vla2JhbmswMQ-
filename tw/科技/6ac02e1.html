塑料性能大全，收藏起來慢慢看！

2020-10-29
| 科技

ABS（丙烯晴-丁二烯-苯乙烯共聚合物）

1. 屬乙烯類，乃一般普通料，較易成型。

2. 流動性、化學電鍍密著性及耐熱性佳。

3. 成形品性能安定。

4. 澆口部分之表面外觀及容易形成明顯之熔合線。

5. 乳白色、半透明。

6. 耐衝擊性比PS佳。

7. 由各聚合物組成比例之變化可得各種特性之ABS、機械強度與耐熱性在泛用塑料中極為優秀、表面光澤、表面噴塗及電鍍性佳、具吸溼性、底色為黃色，一般成品均須染色。

AS(SAN)（丙烯晴-苯乙烯共聚合物）

1. 屬乙烯類，乃一般普通料，較易成型。

2. 流動性佳，成形性佳，成形能率佳。

3. 容易缽裂。

4. 容易發生廢邊。

BT（BT樹脂）

1. 耐熱性與板面附著力佳。

2. 可滿足高速傳輸之電路板需求。

3. 可阻止漏電現象發生。

CA（醋酸纖維素）

1. 流動性佳，成形性佳。

2. 表面光滑美麗，尺寸精度佳、著色易。

3. 透明、可撓性、加工性佳。

CE（氰酸酯樹脂）

1. 純氰酸酯的介電常數為2.8。

2. 搭配不同之補強纖維可製成不同介質常數之基材，以石英纖維補強時，介質常數值可低至3.1。

EP（環氧樹脂【冷凝膠】）

1. 金屬接著性大、耐藥品性佳。

2. 機械強度大、絕緣性佳。

3. 傳統之環氧樹脂稱為雙官能基環氧樹脂，著火後會繼續燃燒，直至高分子中之碳、氫、氧燃燒完畢為止。

ETFE（乙烯-四氟乙烯共聚合物）

1. 為熱可塑性氟素樹脂的一種。

2. 良好的電氣特性：極佳的絕緣材料，有非常低的介電常數。

3. 耐熱性佳。

4. 耐燃性：Oxygen Index 95％，UL94, V-0,是一種耐燃性穩定的物質，可以用於不同的用途。

5. 極佳的抗化學性：幾乎對所有的化學品不反應。

6. 機械強度高。

7. 低摩擦，低非黏性。

8. 耐候性極佳：直接曝露於陽光、雨水或廢氣中沒有損耗或變形，長時間曝露於戶外特性並無改變。

9. 耐藥品性。

10. 加入GF、CF時可增加其振動性。

11. 收縮性大。

FR-PET（強化聚對苯二甲酸二乙酯）

1. 電氣特性優、耐候性佳、機械性能優異、Creep小。

2. 耐熱性佳。

3. 結晶速度快、成型性極佳、流動性良好。

4. 吸水性低，尺寸安定性良好。

5. 表面光澤佳，著色性良好。

6. 耐有機溶劑、油、弱酸。

7. 耐熱性優於FR-PBT。

MF（三聚胺甲硅樹脂【三聚胺甲硅樹脂科學瓷、三聚氰胺美耐皿】）

1. 與UF同，唯耐水性表面硬度大。

2. 耐燃性、無色、易於著色。

M-PPE（改性聚氧化二甲苯樹脂）

1. 加工安定性佳。

2. 機械性能優異。

3. 耐熱性、耐水性佳。

4. 具自熄性。

5. 電氣特性佳。

6. 成形收縮率小。

7. 抗化學藥品性差。

LCP I型（液晶高分子）

1. 耐熱性最高。

2. 耐藥品性、尺寸安定性及自己潤滑性佳。

3. 彎曲強度、高濃鹼蒸氣中會產生劣化。

4. 異方性大、須高溫成型、價格高。

LCP II型（液晶高分子）

1. 耐藥品性佳、高流動性、振動吸收特性及機械強度佳。

2. 縱膨脹係數小。

3. 彎曲強度差、異方性大。

4. 強度會受厚度影響。

5. 未充填的品級易產生表層剝落現象。

LCP III型（液晶高分子）

1. 耐藥品性佳、高流動性、振動吸收特性及成形性佳。

2. 彎曲強度差、異方性大。

PA（聚酰胺）

1. 溶液黏度低，流動性佳。

2. 收縮率之安全性不良、易形成廢邊。

3. 熔解溫度以外時之硬度高，對模具，螺桿等有損傷之虞。

4. 其黏度對加熱溫度敏感性高，亦是吸溼性大的塑料。

5. 高溫時流動性佳，本身不易熔解，熔解後又易冷卻凝固。

6. 強韌、自己油滑但耐磨、吸震性強、耐熱、耐寒、耐藥品性佳。

7. 機械強度佳、玻璃纖唯補強效果、氣體遮蔽性、吸溼性佳。

8. 低溫耐擊性強度不足。

9. 吸失溼後尺寸及物性變化大。

10. 尺寸安定性差。

PAI（聚醚酰亞胺）

1. 機械強度高、硬度高。

2. 難燃、UL94VO，耐候性佳。

3. 耐化學藥品、耐熱水俱優。

4. 電氣特性優、成型加工易、尺寸安定性佳。

5. 耐熱性佳，耐蠕變性優越。

6. 260℃以下彈性係數變化小。

7. 成型性、流動性、耐鹼性及吸溼性差。

8. 價格高。

PAR（聚芳酯）

1. 機械性佳，無色透明(未充填)。

2. 耐溫、HDT均為150℃以上，低溫耐衝擊。

3. 不易燃燒，氧氣指數36.8，UL94VO。

4. 耐候性、抗紫外線及電氣特性佳。

5. 衝擊強度大，厚度依存性小。

6. 高溫潛變性佳。

7. 耐溶劑性、耐酸鹼性及流動性差。

PBT（聚對苯二甲酸乙丁二醇酯）

1. 成型性、耐候性佳、耐熱性、熱老化性、電氣性、耐疲勞性、耐磨性、耐熱水性及耐藥品性佳。

2. 和PET同屬飽和性聚酯類，熔融度高、結晶性迅速、固化速度快。

3. 耐衝擊強度差。

PC（聚碳酸酯）

1. 熔液粘度高，須使用高壓、高溫成形。

2. 殘餘應力容易發生斷裂。

3. 由於硬度較高，易損傷模具。

4. 有廢料擠出。

5. 透明、強韌、低溫安定性優。

6. 耐候性、耐旋光性、耐衝擊性、耐熱性、耐Creep性佳。

7. 高低溫之機械性佳。

PCTFE（聚氯化三氟乙烯）

1. 熔液粘度極高，須高壓成形。

2. 易發生變色。

PDAP或DAP（聚鄰-苯二甲酸二丙烯酯）

1. 強度大、絕緣性、耐藥品性、耐熱性佳。

2. 尺寸安定性佳、溼潤時耐磨耗性佳。

3. 可低壓成形。

PE（聚乙烯）

1. 收縮大，容易發生彎曲及變形。

2. 須要冷卻時間，成形能率不佳。

3. 成形品有低陷，須要強制脫模。

4. 成形收縮率受模具溫度之影響大，安定性不良。

5. 流動性良好、熱安定性佳，但分子配性強容易變形。

6. 比水輕、柔軟。

7. 不耐熱、接著印刷差。

8. 耐藥品、耐水性、電氣絕緣性佳。

PEEK（聚苯醚醚酮【聚二醚酮】）

1. 耐溫、熱安定性佳、超高耐熱(較PPS優良)、HDT在315℃以上，UL連續使用溫度為250℃。

2. 耐衝擊、耐磨、耐潛變、耐疲勞特性等機械強度高。

3. 耐藥品性(濃硫酸除外)、耐熱水性最佳(PPS，PES以上)。

4. 耐γ射線、難燃、押出加工性皆佳。

5. 高溫高溼下，電氣特性極優。

6. 發煙量最小，耐放射線。

7. 價格高，須高溫成型，流動性差。

8. 韌性、強度及剛性均佳。

9. 耐燃性佳，不須添加耐燃劑。

10. 加工性不良。

PEI（聚酰胺酰亞胺）

1. 使用於射出成型加工，拉力強度最高。

2. 耐潛變性最佳(常溫~150℃)。

3. 難燃，UL94VO，耗氧指數42。

4. 耐磨、耐溫、耐油、耐化學品均佳。

5. 耐燃性及耐候性佳，發煙量少，透明性佳(未充填)。

6. 150℃以下彈性係數變化小。

7. 須高溫成型，流動性、耐鹼性及吸溼性差。

PES/PESF（聚醚）

1. -100~170℃，機械強度俱佳，耐溫。

2. 尺寸安定性佳、透明性佳(未充填)。

3. 耐高溫水蒸汽與熱水及酸鹼。

4. 耗氧指數高(UL94V0)，難燃。

5. 電氣特性優秀。

6. 發煙量少。

7. 高溫潛變性優越。

8. 180℃以下，彈性係數變化小。

9. 耐溶劑性及耐候性差，須高溫成型。

PF（酚醛樹脂【電木粉】）

1. 機械強度大、絕緣性、耐燃燒性、耐水性、耐酸性、耐油性及安定性佳。

2. 暗色不耐鹼、易變色、染色性有限。

PFA（四氟乙烯的共聚合物）

1. 為熱可塑性氟素樹脂的一種。

2. 良好的電氣特性：極佳的絕緣材料，有非常低的介電常數與非常低的介電消散因子。

3. 耐熱性：使用溫度範圍廣泛(-200℃至+280℃)。

4. 耐燃性：Oxygen Index 95％，是一種耐燃性穩定的物質，可以用於不同的用途。

5. 極佳的抗化學性：幾乎對所有的化學品不反應。

6. 機械強度高。

7. 表面能量低：低摩擦，低非黏性，並且對水及油的表面撥水性及撥油性高，可增加原料的可靠性。

8. 耐候性極佳：直接曝露於陽光、雨水或廢氣中沒有損耗或變形，長時間曝露於戶外特性並無改變。

9. 耐藥品性佳、低摩擦係數、非粘著性。

10. 成形性不良、收縮性大、價格高。

PI（聚酰亞胺）

1. 耐熱性最高。

2. 耐磨、耐潛變、耐放射性優。

3. 耐藥品性優、不耐強酸。

4. 射出、押出均困難。

5. 240℃下彈性係數變化小。

6. 耐藥品性佳(濃硝酸除外)。

7. 成型性、吸溼性差。

8. 結晶化溫度高。

9. 尺寸變化很小，以其X、Y方向之受熱膨脹很小，可避免細線路因受熱而造成銅箔附著力變差，Z方向之熱膨脹小，更可避免高層數多層板之孔壁銅層受熱斷裂。

PMMA（聚甲基丙烯酸甲酯【壓克力】）

1. 韌性強、無色透明。

2. 光學性、絕緣性、加工性及耐候性佳。

POM（聚甲醛）

1. 俗稱塑料鋼，易起熱分解，耐藥品性及耐Creep良好。

2. 耐磨性優異、自身具潤滑性、強韌、耐久力大、耐熱、耐藥品性佳。

3. 機械性佳、耐衝擊。

4. 無自熄性、抗酸性較差、成型收縮率大。

5. 類似尼龍、耐磨耗類似PC。

6. 高機械性質、彈性回覆性極佳、熱安定性較差、比重高(1.41)。

PP（聚丙烯）

1. 流動性差、充填不良有流痕形成，壓力不足，容易發生凹陷，需要高壓成形。

2. 光學用途時，透明度成問題，當異種材料混合時，須注意分解作用。

3. 成形性極佳、機械強度比PE高。

4. 耐水、耐藥品性、電氣絕緣性佳。

5. 容易發生變形，翹曲及低陷。

6. 有黏著性、為最輕的塑料。

7. 比重低(約0.90)、FDA級、透明性佳、不吸溼、價廉、耐熱及一般機械強度均不甚高、發泡倍率不高、印刷性較差。

PPO（聚氧化二苯）

1. 電氣特性、耐熱水性、耐燃性佳。

2. 耐藥品性及加工性差。

PPS（聚苯硫醚）

1. 耐熱性及耐燃性佳，HDT高，高溫下，機械強度仍高。

2. 結晶速度快，結晶度高達60~65％。

3. 尺寸安定性佳，電氣特性佳，熔融粘度低、成型性佳。

4. 耐化學性佳，耐藥品性極優(濃硝酸除外)，與TEFLON相當，且200℃下，無任何溶劑可溶解PPS。

5. 耐衝擊性、耐候性、韌性及磨擦磨耗特性較差。

6. 會有毛邊，機械性質受模具溫度影響。

PS（聚苯乙烯）

1. 屬乙烯類，乃一般普通料。

2. 流動性佳，成形性佳，成形能率佳。

3. 容易缽裂，不耐衝擊(GPPS)。

4. 容易發生廢邊。

5. 無色透明、易於染色、絕緣性佳。

6. 耐水性、耐藥品性佳。

PSF（聚風）

1. 在-100℃~150℃之機械強度高，電氣特性保持穩定。

2. 耐溫，UL746，HDT均為150℃以上。

3. 耐藥品性佳，耐熱水及水蒸氣最突出，在溫度變化下，均能保持優良之電氣特性。

4. 高溫下可耐氧及臭氧之劣化。

5. 無毒、FDA及衛生規格均可符合，琥珀色透明。

6. 耐酸鹼性佳、未充填時透明性佳。

7. 耐溶劑性、離型性及流動性差。

PTFE（聚四氟乙烯）

1. 高低溫電氣絕緣性佳。

2. 耐藥品性強度極大。

3. 耐熱性、耐磨性及安定性佳。

PU或PUR（聚氨基甲酸酯）

1. 有彈性、強韌。

2. 耐磨耗性、耐熱性、耐油性、耐老化性佳。

3. 稍不耐酸、鹼及熱水。

PVA（聚乙烯醇）

1. 無色透明彈性體。

2. 耐熱性佳。

3. 絕緣軟化點高。

PVAC（聚乙酸乙烯酯）

1. 無色透明。

2. 接著性、耐旋光性佳。

3. 耐熱性差。

4. 吸水性大、大部分之溶劑皆可溶。

PVDC（聚乙烯甲醛）

1. 比PVC耐藥品大。

2. 耐熱性佳。

3. 薄膜的透氣性小。

PVC（聚氯乙烯【硬質】）

1. 熱安定性不良，成形溫度與分解溫度範圍接近。

2. 流動性不良、外觀容易形成不良。

3. 會腐蝕模具。

4. 最易燒焦、產生酸性氣體。

5. 強度、電氣絕緣性、耐藥品性佳。

6. 加可塑劑會軟化。

7. 耐熱性不佳。

8. 比重高(1.4)、具耐燃性、印刷性佳、不同配方有不同特性、應用範圍廣泛、價廉、環保衝擊大、加工溫度超過190℃時易發生裂解，而放出HCI腐蝕性氣體。

SP或SI（硅酮塑料【或硅氧樹脂】）

1. 耐高低溫、絕緣佳。

2. 特有表面物理性、脫模性、消泡性、耐磨性優。

3. 耐日光、耐化學藥品、無毒。

TPE(TPR)（熱塑性彈性體）

1. 柔軟、有彈性、觸感佳。

2. 加工簡易、快速、可回收再使用。

TPU（TPU樹脂）

1. 機械性質優異、彈性佳。

2. 耐磨耗性、抗化學性質佳。

3. 與血液和細胞組織有很好的兼容性。

4. 加工容易。

UF（尿醛樹脂【電玉】）

1. 無色、著色自由。

2. 與PF性質類似而稍劣、耐水性和耐候性不好。

UP（不飽和聚酯【冷凝膠】）

1. 絕緣性佳。

2. 玻璃纖維補強之UP機械強度大。

3. 耐熱、耐藥品、可低壓成形、耐水。
