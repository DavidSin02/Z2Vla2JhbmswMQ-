如何為變頻器選擇合適的控制方法

2020-10-29
| 科技

電機至少佔美國能源消耗的一半。為應用選擇正確的控制方法可使電機最有效地運行，同時最大化扭矩和整體性能。高效運行的電機也可以減少能源消耗，減少停機時間，從而節省更多成本。

控制方法

業內許多人認為控制方法是控制VFD的排序方法，如2線和3線設置。這種2線和3線設置確定VFD的輸入控制端子是否與保持的觸點或瞬時按鈕接口以啟動和停止驅動器。本文所關注的控制方法可能更準確地稱為運動控制方法。他們確定VFD如何控制電機。

連接到VFD的感應電動機有四種主要類型的電動機控制方法：V / f（伏特/赫茲），帶編碼器的V / f，開環矢量和閉環矢量。這些方法都使用脈衝寬度調製（PWM），這種技術通過調製脈衝持續時間來改變固定信號的寬度，以產生可變模擬信號。

通過使用VFD直流母線電容器的固定直流電壓，將PWM應用於VFD。輸出側的一組絕緣柵雙極晶體管（IGBT）快速打開和關閉以產生脈衝。改變輸出電壓波形中的輸出脈衝寬度可以構建模擬的AC正弦波。即使驅動器的輸出電壓波形由DC脈衝引起的方波組成，電流波形也是正弦波，因為電機是電感式的。所有電機控制方法都依靠PWM電壓波形來控制電機。控制方法之間的區別在於它們如何在任何給定時刻計算電機的電壓需求。

通常使用脈衝寬度調製來控制AC電動機。在此過程中，載波頻率（以紅色顯示）是VFD輸出晶體管門控或調諧的速率。載波頻率通常可以是2至15kHz。頻率參考（藍色）是發送到電機的速度信號，通常為0到60 Hz。當覆蓋兩個波形時，工程師可以使用這兩條曲線之間的交點來調製輸出直流脈衝（黑色），以提供所需的速度控制。

V / f控制

不同的V / f模式使VFD可以控制多個不同的應用，同時保持每個應用的最佳性能。恆定扭矩模式是直線，這導致恆定的V / f比，在整個速度範圍內提供恆定的電動機扭矩。可變轉矩模式在較低速度下具有較低的電壓以防止電動機飽和。

伏特/赫茲，通常稱為V / f，是最簡單的電機控制方法。它經常被使用，因為它具有“即插即用”的簡單性以及驅動器所需的電機數據量。它不需要編碼器，也不需要將VFD調諧到電機（但推薦）。這意味著更低的成本和更少的佈線。V / f控制通常在需要運行時使用，可能超過1,000 Hz，因此通常用於機床和主軸應用。

V / f有一些限制。例如，使用V / f時，無法保證電機軸正在旋轉。此外，電機的啟動轉矩限制在3 Hz輸出的150％。對於大多數可變扭矩應用而言，有限的啟動扭矩綽綽有餘。事實上，現場幾乎所有可變扭矩風扇和泵應用都使用V / f控制。

V / f方法的相對簡單性部分歸因於其“更寬鬆”的規範。速度調節通常為最大頻率的2％至3％。速度響應額定值為3 Hz。速度響應定義為VFD響應參考頻率變化的程度。當參考頻率改變時，速度響應的增加導致更快的電動機響應。

控制方法還具有速度控制範圍（表示為比率）。V / f的速度控制範圍是1:40。將此比率乘以最大頻率可確定VFD控制電機的最小運行速度。例如，使用60 Hz最大頻率和1:40速度控制範圍，使用V / f控制的驅動器可以將電機控制在1.5 Hz。

AV / f模式定義了電機的電壓與頻率之比，VFD一次只能編程一個V / f模式。V / f模式或曲線根據給定的速度參考（頻率）確定輸出到電機的電壓。

操作員或技術人員可以使用單個參數在VFD編程中選擇預設的V / f模式。預設模式針對特定應用進行了優化。用戶可以更進一步，編程自定義V / f模式或配置文件，以將VFD調諧到特定應用和正在使用的電機。

同樣，恆轉矩應用，如輸送機，擠出機和提升機，應使用恆轉矩V / f模式。恆轉矩應用在所有速度下都需要全磁化電流。因此，在整個速度範圍內構建並遵循直線斜坡。

通常，VFD將根據電機設置的任何速度輸出電壓，同時參考V / f模式。

帶編碼器的V / f

如果應用需要更精確的速度調節，並且能夠以更高的參考頻率運行，則可以將編碼器添加到V / f控制。編碼器反饋將速度調節降至最大頻率的0.03％。輸出電壓仍由技術人員在VFD中編程的所選V / f模式決定。這允許高速控制而沒有高動態響應，因為電壓和頻率是預定的。

這種控制方法並不常見，因為它需要編碼器和反饋卡的成本，並且其優於標準V / f控制的優點是最小的。啟動轉矩，速度響應和速度控制範圍都與V / f控制相同。此外，較高的工作頻率受編碼器產生的每轉脈衝數的限制。

開環矢量控制

開環矢量（OLV）控制用於更大和更動態的電機控制。它獨立控制電機速度和扭矩，就像控制直流電機一樣。

根據電機方向（正向或反向）以及電機是電動還是再生，轉矩極限可分為四個象限。例如，瓶蓋將需要為象限1設置扭矩限制。或者，展開應用將需要正向電動機旋轉來供給線，但是由於線被拉動以產生張力而由於再生而產生負扭矩極限。因此扭矩限制將在象限4中設定。

運行OLV時，電機可以0.3 Hz的頻率產生200％的額定扭矩。較低速度下較高的啟動扭矩為各種應用打開了大門。該控制方法還允許四象限扭矩限制。

扭矩限制主要限制電機扭矩，以防止損壞設備，機器或產品。它們分為四個不同的象限，取決於電機方向（正向或反向），以及電機是電動還是再生。可以為每個象限單獨設置限制，或者用戶可以將整體扭矩限制編程到VFD中。

電動機狀態是指電動機的速度和轉矩都在同一方向。例如，前進速度和前進扭矩將使輸送機向前運動。再生是指負載對電機進行大修。在AC電動機上，當轉子比定子中的磁場旋轉得更快時，它起到發電機的作用。這導致再生能量流回VFD。

例如，瓶蓋可以使用象限1中的扭矩限制（正向旋轉和正向扭矩）以防止瓶蓋過度扭矩。它向前移動並使用正扭矩將蓋子放在瓶子上。涉及具有比空車重的配重的電梯的應用在象限2中具有限制（反向電動機旋轉和正扭矩）。如果將空車調到較高樓層，則扭矩與速度方向相反，以保持對配重的控制以及電梯在重力作用下的速度和位置。

從木塊中取出螺釘的機器鑽（反向電機方向和負扭矩）可以使用象限3中的限制。退繞應用可以使用象限4限制（正向電機旋轉和負扭矩）。電動機將旋轉電動機旋轉以供給線路，但是由於線路被拉動以產生張力而導致再生，因此還需要負的扭矩限制。

這些VFD中的電流反饋迴路允許用戶設置扭矩限制並在所有四個象限中運行。隨著電動機電流的增加，電動機轉矩也隨之增 如果應用需要更大的扭矩，則可以增加輸入到電機的輸出電壓;當達到扭矩限制時，可以減小輸出電壓。與V / f控制不同，這使得開環控制動態化。

除扭矩限制外，開環控制還具有10 Hz的更快速度響應，使其能夠處理對衝擊負載的更多動態響應。例如，岩石破碎機的負荷會不斷變化，這取決於正在處理的岩石的大小和數量。

OLV控制使用矢量算法來找到運行電機的最佳輸出電壓，而不是固定的V / f模式。矢量控制通過使用來自電機的電流反饋來實現此目的。通過VFD內部的電流互感器測量電流反饋。在VFD中執行的恆定電流讀數和快速計算確定當前的扭矩需求和通量。基本矢量數學將電機的磁化電流和產生轉矩的電流分解為矢量。OLV控制在很大程度上取決於電機動態，因此必須執行某種類型的電機自整定，以確保VFD具有儘可能多的電機數據。

藉助可靠的電機數據/參數，VFD可以將磁化電流（I d ）和轉矩產生電流（I q）計算為矢量。為了獲得最大效率和扭矩，VFD必須將這兩個矢量分開90°。90°是重要的，因為sin（90）= 1，值1表示最大電動機扭矩。

整體OLV控制導致更嚴格的控制。速度調節是最大頻率的+/- 0.2％，速度控制範圍跳至1：200，允許低速運行而不會犧牲扭矩。

閉環矢量控制

閉環矢量控制使用矢量算法來確定輸出電壓，非常類似於開環控制。關鍵的區別在於閉環矢量使用編碼器。編碼器反饋與矢量控制配合使用，意味著在0 rpm時可獲得200％的電機額定轉矩。這是需要在不移動的情況下保持負載的應用程序的賣點，例如電梯，起重機和升降機。

編碼器反饋允許速度響應超過50 Hz，速度控制範圍為1：1500，是所有控制方法中最高的。閉環控制還可以在轉矩控制模式下運行電機。轉矩控制使VFD控制電機轉矩而不是電機轉速。在扭矩比速度更重要的任何應用中都需要這樣做。卷繞機，復卷機，封蓋機和捲筒紙應用程序是使用扭矩控制的好例子。
