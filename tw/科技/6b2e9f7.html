Windows 10任務計劃程序為您提供了更多功能

2020-10-29
| 科技

Windows 10的任務計劃程序可自動執行任何應用程序，包括維護，鬧鐘等。在Windows 10中，省電模式會修改任務計劃程序以使用更少的能源。本文詳細介紹了最新版本的Task Scheduler與舊版本的區別。

什麼是任務計劃程序？



Windows 10 Task Scheduler 在特定 時間或某些事件後（我們將其稱為觸發器或條件）執行 腳本或程序。它作為維護或自動化工具很有用，但在Windows 10中，它對移動用戶進行了重大更改。

Windows 10 Task Scheduler中的新增功能？

Windows 10的實現與Windows Vista Scheduler幾乎相同，但實現方式卻有很大不同：省電模式會延遲某些任務的執行。打開省電模式後，如果滿足以下條件，則 計劃任務不會觸發：

  • 該任務設置為在計算機空閒時觸發。
  • 該任務設置為在自動維護期間運行。
  • 用戶登錄後，該任務未設置為運行。


    • 由於節電模式可以配置為以一定級別的電池電量（例如20％）打開，因此有助於大大降低功耗（如何配置節電模式）。以下是Windows 10如何修改任務計劃程序的詳細信息，並提供了示例。

    1.修改任務計劃程序的行為



    如果Windows檢測到用戶不在使用他們的計算機，它將認為系統處於空閒狀態。某些計劃的進程將不在此狀態下執行。例如，當計算機空閒時，磁盤優化將按設置的時間間隔運行。但是，使用電池供電時，在旋轉硬盤驅動器（HDD）上運行磁盤優化可能會給系統的正常運行時間帶來災難性的影響。從邏輯上講，電池模式會延遲由空閒狀態觸發的所有任務。

    那些擁有固態驅動器（什麼是SSD？）的人可能並不關心是否在運行磁盤優化（什麼是TRIM？），即使計算機使用電池模式也是如此。SSD優化需要幾秒鐘。如果要修改或禁用任何計劃的任務，請打開“任務計劃程序”。您只需在Windows搜索中鍵入“任務計劃程序”，它就會顯示出來。

    接下來，您將必須深入研究Task Scheduler庫的幾層。單擊右邊的V形符號（在條目的左側）以展開Task Scheduler Library的條目。然後（再次）單擊Microsoft的條目。然後，單擊Windows的向左V形。

    從出現的條目列表中，找到“ 碎片整理”選項，然後單擊它。在中央窗格中，雙擊ScheduleDefrag。請記住，不同的進程將位於任務計劃程序庫的不同部分。

    您將獲得一個彈出窗口，顯示控制Defrag實用程序行為的觸發器和條件。如果要讓該過程在不使用Battery Saver的情況下運行，則需要確保所有觸發器都不包含以下三個組件：

    • 它不能包含空閒狀態觸發器；
    • 無論用戶是否登錄，必須將其設置為運行。
    • 不能將任務設置為作為自動維護的一部分運行。



    但是，更好的選擇是在電池電量不足時關閉所有計劃的任務。不幸的是，在節電模式下，Windows 10不提供任何方法。用戶需要手動將其關閉-並且當他們重新插入電源時，他們需要手動將其重新打開。這確實是次優的

    2.關閉任務計劃程序

    與舊版Windows不同，Windows 10 Task Scheduler沒有關閉開關。幸運的是，可以在任務管理器中找到該項目並手動將其禁用-儘管用戶希望在退出電池模式後立即做出反應。該程序對長期系統性能至關重要。在電池模式下，禁用它會為您的筆記本電腦或平板電腦增加少量的正常運行時間。

    要關閉任務計劃程序，請按組合鍵CTRL + SHIFT + ESC進入任務管理器， 以打開任務管理器。接下來，找到“ 服務主機：本地系統”（16）的條目，然後單擊左側的V形符號，這將顯示許多子流程。該列表的底部是任務計劃程序。右鍵單擊“ 任務計劃程序”， 然後從上下文菜單中選擇“ 禁用 ”。要重新啟用它，只需右鍵單擊並從上下文菜單中選擇“ 啟用 ”。切記在退出電池模式後重新打開此服務。

    3. Windows 10的任務計劃程序摘要



    除了與Battery Saver配合進行的調整外，任務計劃程序還包括摘要功能-摘要包含所有活動任務。要訪問此文件，請在“任務計劃程序”中單擊“ 任務計劃程序（本地）” 。

    摘要顯示在中央窗格中“ 活動任務”下。在下面，您將看到一些與第三方軟件有關的條目。


    4.您真正不需要的預定的東西

    根據“任務計劃程序摘要”，有71個程序根據任務計劃程序中的條件觸發。儘管最重要的是重要的，但一些條目並不重要。這些是由第三方軟件安裝的，對計算機的運行並不是很重要。

    您可以通過導航到“ 任務計劃程序” 並單擊“ 任務計劃程序庫”來查看這些條目。在中心窗格中，將顯示一個應用程序列表。此處的大多數條目都與更新軟件有關。但是，許多潛在有害程序（PUP）在此區域中存儲條目。如果您看到許多無法識別的程序，則可能需要考慮運行惡意軟件掃描。

    您是否應該使用Windows 10任務計劃程序？

    如果您知道自己在做什麼，那麼一定可以。任務計劃程序提供的不僅僅是優化的電池性能。它可以自動執行無聊的過程，包括暫停和睡眠。對於想要節省工作時間的寶貴人員，任務計劃程序是必不可少的工具。
