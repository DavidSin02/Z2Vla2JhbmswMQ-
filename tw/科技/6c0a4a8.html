Ubuntu 20.10發佈：認證支持樹莓派 配GNOME 3.38桌面環境

2020-10-29
| 科技

Canonical 今天發佈了 Ubuntu 20.10 新版本。在本次更新中，最值得關注的就是為樹莓派桌面和服務器優化系統鏡像，這應該會受到“學習者、發明家、教育家和企業家”的歡迎。雖然很早就添加了對樹莓派的支持，但這是首個將樹莓派添加到認證設備列表中的 Ubuntu 版本。

Canonical 首席執行官 Mark Shuttleworth 表示：“在此版本中，我們慶祝樹莓派基金會對將開放式計算交付全世界的人們的承諾。我們很榮幸能夠通過在樹莓派上優化 Ubuntu 來支持該計劃，無論是出於個人用途，教育目的還是作為其下一次創業的基礎”。

現在，所有新款樹莓派均通過了 Ubuntu 的認證，只要內存上有 4GB 或者 8GB。Canonical 在新聞稿中寫道：“樹莓派2/3/4加入到了 Ubuntu 認證的 x86 和 ARM 設備長列表中。 Ubuntu 廣泛用於公共雲和桌面，用戶可以在戴爾、惠普和聯想等所有認證的 PC 上運行 Ubuntu Desktop，而該系統也被 AWS、微軟Azure、Google、IBM 和 Oracle 等雲服務廣泛使用”。

除了添加樹莓派桌面和服務器鏡像之外，Ubuntu 20.10 更新最重要的新功能就是 GNOME 3.38。該桌面環境引入了諸多改進，包括以網格組織應用，在註銷和關機界面中增加重啟選項等等。在本次 GNOME 更新中，用戶還可以通過本地生成的 QR 碼共享私人Wi-Fi熱點。

桌面版Ubuntu 20.10引入了對Ubuntu認證服務的支持，Canonical說正在添加生物識別功能，而完全支持帶有虛擬鍵盤的二合一設備。 Lenovo Yoga和DELLXPS 2合1是Ubuntu現已完全支持的兩種型號，您可以在其中本地使用觸摸屏。
