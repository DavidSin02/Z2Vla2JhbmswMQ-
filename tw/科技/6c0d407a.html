WIN32學習——對話框

2020-11-14
| 科技
  • 對話框的分類：

1）模式對話框 - 當對話框顯示時，會禁止其他窗口的輸入等用戶交互操作。

2）非模式對話框 - 在對話框顯示後，其他窗口同樣可以接收輸入等用戶交互操作。

  • 對話框的使用過程：

1）對話框窗口處理函數（參照MSDN - DialogProc）

2）添加窗口資源

3）創建對話框（模式對話框和非模式對話的創建不一樣）

4）關閉對話框（模式對話框和非模式對話的刪除也不一樣）

  • 對話框窗口處理函數（參照MSDN），模式對話框和非模式對話框，對話框窗口處理函數一樣

1）BOOL CALLBACK DialogProc(

HWND hwndDlg, //窗口句柄

UINT uMsg, //消息ID

WPARAM wParam, //消息參數

LPARAM lParam //消息參數

);

返回TRUE - 表示 DialogProc 函數中處理了這個消息，缺省處理函數不需要處理。

返回FALSE - 表示 DialogProc 函數未處理這個消息，交給缺省處理函數處理。

  • 模式對話框的創建和關閉

1）創建：

int DialogBox(

HINSTANCE hInstance, //應用程序實例句柄

LPCTSTR lpTemplate, //對話框模板資源ID

HWND hWndParent, //對話框父窗口

DLGPROC lpDialogFunc //對話框窗口處理函數

);

DialogBox 是一個阻塞函數，只有當對話框關閉後，才會返回，繼續執行後續代碼。返回值是通過 EndDialog 設置。

2）關閉：

BOOL EndDialog(

HWND hDlg, //關閉的對話框窗口句柄

int nResult //關閉的返回值

);

關閉模式對話框，只能使用 EndDialog ，不能使用 DestroyWindow 等函數。

nResult 是 DialogBox 函數退出時的返回值。在對話框退出的時候， nResult 作為返回值被 DialogBox 函數接收，可以用來判斷點擊了對話框中的哪個按鈕。

注意： EndDialog 函數可以解除 DialogBox 的阻塞。

  • 非模式對話框的創建和關閉：

1）創建：

HWND CreateDialog(

HINSTANCE hInstance, //應用程序實例句柄

LPCTSTR lpTemplate, //模板資源ID

HWND hWndParent, //父窗口句柄

DLGPROC lpDialogFunc //窗口處理函數

);

非阻塞函數，創建成功返回窗口句柄，需要使用 ShowWindow 函數顯示對話框。

2）關閉：

關閉時使用 DestroyWindow 銷燬窗口，不能使用 EndDialog 關閉對話框。

  • 對話框和窗口的對比：

1.創建

模式對話框 - DialogBox 函數，是一個阻塞函數

非模式對話框 - CreateDialog

普通窗口 - CreateWindow/CreateWindowEx

2.窗口處理函數

對話框 - DialogProc ，被缺省的對話框窗口處理函數調用

普通窗口 - WindowProc ，需要調用缺省的窗口處理函數（DefWindowProc）

3.窗口消息

對話框 - WM_INITDIALOG

普通窗口 - WM_CREATE

4.窗口關閉

模式對話框 - EndDialog

非模式對話框 - DestroyWindow
