自動化搭建微型linux系統ftp服務器

2020-10-29
| 科技

一、構思

在這裡我們是通過在一臺擁有兩個盤的linux上運行自動化腳本(一個系統所在盤sda,一個空盤sdb),在其系統所在盤創建一個類linux根目錄,和boot目錄,並通過母linux系統,將其所需的grub,軟件,服務等複製過去,然後將它掛載到sdb,最後新建一個linux用母系統的sdb盤作為系統盤.

二、所需文件

我的自動化腳本main.sh;另外我實現了用戶登錄驗證,這裡我是直接把login這個單獨弄了出來,方便複製;還有一個名叫txt的文件,這裡他是記錄我們所需的命令,例如ls等,如果有其他需要可以在這裡面繼續添加,防止在給微型linux系統加命令是有不必要的麻煩

四、建立微型linux系統

將上述已經做好的盤,放到準備建立的新的linux系統下

啟動服務/usr/local/sbin/vsftpd /etc/vsftpd/vsftpd.conf

bcp()

{

COMMAND=`which $1 | grep -o "/.*"`

CMDPATH=${COMMAND%/*}

[ -d $SYSROOT$CMDPATH ] || mkdir -p $SYSROOT$CMDPATHI

[ -e $SYSROOT$COMMAND ] || cp $COMMAND $SYSROOT$CMDPATH

for LIB in `ldd $COMMAND | grep -o "/.*lib[^[:space:]]*"`;do

LIBPATH=${LIB%/*}

[ -d $SYSROOT$LIBPATH ] || mkdir -p $SYSROOT$LIBPATH

[ -e $SYSROOT$LIB ] || cp $LIB $SYSROOT$LIBPATH

done

}
