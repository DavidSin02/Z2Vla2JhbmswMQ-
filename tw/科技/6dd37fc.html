Java 添加圖片和文本頁眉頁腳到Excel工作表

2020-10-29
| 科技

在前文中我們介紹過如何通過Java程序來給Word文檔添加頁眉頁腳，本文將演示在Excel文檔中如何來添加頁眉頁腳。主要包括：添加圖片到頁眉、添加文本到頁腳、為奇偶頁添加不同的頁眉頁腳。其中需要注意的是，Excel頁眉或頁腳在普通瀏覽模式下不可見，它們僅在頁面佈局模式或者打印預覽模式才可見。

使用工具：Free Spire.XLS for Java（免費版）

Jar文件獲取及導入：

方法1：通過E-iceblue中文官網下載獲取jar包。解壓後將lib文件夾下的Spire.Xls.jar文件導入Java程序。（如下圖）


方法2：通過maven倉庫安裝導入。具體安裝教程詳見E-iceblue中文官網。

【示例1】添加圖片到頁眉

import com.spire.xls.*;import javax.imageio.ImageIO;import java.awt.image.BufferedImage;import java.io.*;public class ImageHeader {    public static void main(String[] args) throws IOException {             String imageFile = "C:\\Users\\Test1\\Desktop\\Image.png";        //加載Excel示例文檔              Workbook workbook = new Workbook();      workbook.loadFromFile("C:\\Users\\Test1\\Desktop\\Sample.xlsx");        //獲取第一個工作表              Worksheet worksheet = workbook.getWorksheets().get(0);        //加載圖片              BufferedImage image = ImageIO.read( new File(imageFile));        //設置圖片頁眉              worksheet.getPageSetup().setLeftHeaderImage(image);      worksheet.getPageSetup().setLeftHeader("&G");        //設置顯示樣式              worksheet.setViewMode(ViewMode.Layout);        //保存文檔             workbook.saveToFile("output/ImageHeader.xlsx", ExcelVersion.Version2010);    }}

【示例2】添加文本到頁腳

import com.spire.xls.*;public class TextFooter {    public static void main(String[] args) {        //加載Excel示例文檔        Workbook workbook = new Workbook();        workbook.loadFromFile("C:\\Users\\Test1\\Desktop\\Sample.xlsx");        //獲取第一個工作表        Worksheet worksheet = workbook.getWorksheets().get(0);        //設置頁腳文字及位置        worksheet.getPageSetup().setCenterFooter("");        //設置顯示樣式        worksheet.setViewMode(ViewMode.Layout);        //保存文檔        workbook.saveToFile("output/TextFooter.xlsx", ExcelVersion.Version2010);    }}

添加效果：


【示例3】為奇偶頁添加不同的頁眉頁腳

import com.spire.xls.*;public class DifferentHeaderFooter {    public static void main(String[] args) {        //加載Excel示例文檔        Workbook workbook = new Workbook();        workbook.loadFromFile("C:\\Users\\Test1\\Desktop\\Sample.xlsx");        //獲取第一個工作表        Worksheet worksheet = workbook.getWorksheets().get(0);        //設置第一頁，第二頁        worksheet.getCellRange("A1").setText("Page 1");        worksheet.getCellRange("M1").setText("Page 2");        //將setDifferentOddEven的值設為1, 表示奇數頁和偶數頁可以分別設置頁眉頁腳        worksheet.getPageSetup().setDifferentOddEven((byte)1);        //設置奇數頁及偶數頁頁眉頁腳，樣式和顏色        worksheet.getPageSetup().setOddHeaderString( "&\"Arial\"&12&B&KFFC000 Odd_Header");        worksheet.getPageSetup().setOddFooterString ( "&\"Arial\"&12&B&KFFC000 Odd_Footer");        worksheet.getPageSetup().setEvenHeaderString ( "&\"Arial\"&12&B&KFF0000 Even_Header");        worksheet.getPageSetup().setEvenFooterString ( "&\"Arial\"&12&B&KFF0000 Even_Footer");        //設置顯示樣式        worksheet.setViewMode(ViewMode.Layout);        //保存文檔        workbook.saveToFile("output/Different.xlsx", ExcelVersion.Version2010);    }}
