零基礎自學js,數據變量

2020-11-14
| 科技

變量是存儲信息的容器,什麼數據型的變量就存放什麼數據,就好像家裡的籃子一樣,用來裝蔬菜的籃子肯定是裝不了水的,類型符合,一個整數數據類型當然不可以裝字符串數據,

在js中變量數據類型的關鍵字是"var",它是變體英文單詞的縮寫,變體變量是數據類型是所有沒被顯式聲明,它可以是整數,字符串,字節,日期,甚至是一個對像,具體要看用戶來放什麼.

我們來做一個例子:在桌面新建一個文本文檔,在裡寫入代碼

var str = "hello World!";

debug.printf(str.constructor);

var num = 123;

debug.printf(num.constructor);

var date = new Date();

debug.printf(date.constructor);

var bool = new Boolean();

debug.printf(bool.constructor);

var Arr = new Array(3);

debug.printf(Arr.constructor);

圖1

以上代碼中變量str="helloWorld"賦值的是一個字符串,進行賦值後,它就是一個字符串數據類型,num=123;它的值是一個整數,賦值後它就是一個整數型,date = new Date(),引用的是一個日期對像,所以它是一個日期數據型,bool = new Boolean引用的是布爾型,Arr = new Array(),是一個Array數組型,我們調用每個變量的constuctor()屬性看看,返回了什麼

圖2

運行後,在立即窗口中顯示了以下內容

function String() {

}

[native code]

[native code]

string(字符串),number(數字),date(日期),boolean(布爾),Array(數組)

每個數據變量都會有一個容量,例如一個短整數數據類型範圍是-32768到32767,存放的數字超出範圍,就會導致數據溢出,就像一個瓶子一樣的,這個瓶子只能裝500ml的水,卻倒入600ml的水,那麼多於的水就會溢出來浪費
