02、案例分析

試題編號 : 01

知識點 : 送電線路

某220kV線路開關B相故障跳閘，重合不成功，高頻距離、高頻方向保護動作跳閘，當時天氣為大風大雨並伴有打雷。隨後，強送成功。檢查發現#265杆B相（中相）導線在防震錘處附近有明顯放電痕跡，橫擔、接地引下線、避雷線及連接金具未發現明顯放電痕跡。#265杆型為ZLH1-20.45型直線杆，位於山頂，絕緣子串為13片LXY1-70型絕緣子，架空地線為雙地線，保護角為13.92o。接地電阻值為13歐。試分析線路跳閘原因，提出防範措施。

引起絕緣子閃絡的可能性是雷電過電壓、操作過電壓和汙閃。從當時的天氣情況看，線路跳閘是雷擊引起。水平排列的中相閃絡，可以肯定雷擊性質為反擊。重合不成功是由於重合閘時間短、重合閘時放電通道的空氣絕緣尚未恢復。該杆位於山頂，容易遭受雷擊，接地電阻不大，保護角也符合規程規定而且雷擊性質為反擊，因此應採用加強絕緣或加裝線路避雷器措施。

試題編號 : 02

知識點 : 送電線路

某500kV線路開關跳閘，重合成功，線路B相故障。天氣為陰天，無雷電大風。檢查發現#190塔B相（中相）導線、懸垂線夾、橫擔下層塔材有放電痕跡，在橫擔下層塔材面上還發現有鳥毛被燒成灰的痕跡，鐵塔拉線上把上有大量鳥糞痕跡，在橫擔上層塔材面上安裝有防鳥刺。橫擔下層塔材放電痕跡的投影與導線距離為60cm。#190塔為拉V塔，位於一小片甘蔗地內，地形為平原，四周空曠無樹，在距該塔減號側約500米處有一大水塘。試分析線路跳閘原因，提出防範措施。

根據天氣為陰天、無雷電大風，排除雷擊和大風引起外物短路的可能；根據天氣為陰天和放電通道為空氣的情況，排除汙閃的可能；從周圍環境情況看，該段落屬於鳥害區，從現場放電痕跡和鳥糞、被燒的鳥毛情況，可以認定此次跳閘為鳥糞所致，即鳥糞造成導線對鐵塔橫擔的綜合間隙距離不足引起放電短路跳閘。該塔已採取了防鳥害措施，即在橫擔面上安裝有防鳥刺，但沒有考慮到鳥仍可站立在中相橫擔的下層塔材上。對該線路鐵塔的中相橫擔的下層塔材上補裝防鳥刺；清查鳥害區的線路杆塔橫擔是否還存在類似絕緣子上方鳥仍可站立的情況，如有則補裝防鳥刺或加裝驅鳥裝置。

試題編號 : 03

知識點 : 送電線路

x月x日，當地天氣寒冷並伴有雨夾雪，某110kV線路開關跳閘，重合不成功，A相接地。護線員彙報：#65～#66杆間避雷線覆冰有5～6cm，造成右避雷線下沉對右導線的安全距離不足（約20cm）而對導線放電，並造成避雷線在距#65塔約150米處斷股4股；#65～#66左避雷線弧垂增大，與左相導線相平，空間距離約1.4m；電杆、鐵塔、絕緣子均嚴重覆冰。#65～#67為一個耐張段，均處於大山區，#65為耐張塔（JG2-15），#66為直線雙杆（ZL2-18），#65～#66檔距較大（619m），#66～#67檔距較小（237m），#66杆往#65傾斜約5°。該線路是某縣唯一的供電電源，停電對當地影響很大，需臨時恢復供電。請作出處理決定並說明理由。

首先線路覆冰已大大超過了設計標準，嚴禁上杆塔工作。處理方案是：採用射繩槍、小火箭拋繩設備、遙控飛艇、弓箭、彈弓、絕緣杆或其他方法把細繩拋過杆塔，拖引麻繩，再拖引鋼絲繩，對杆塔避雷線掛點處加臨時拉線加固；以前面同樣方法把細繩拋過右相導線，拖引麻繩，採取絕緣繩串接複合絕緣子後把導線拉到符合送電的安全距離後送電。

試題編號 : 04

知識點 : 送電線路

人們擔心：變電站使用玻璃絕緣子自爆時炸壞設備。請根據玻璃絕緣子的運行情況做出回答，並談一下玻璃絕緣子的特點。

這是對玻璃絕緣子的誤解。玻璃絕緣子的“自爆”又稱“自破”，並非“自爆炸”，是絕緣子玻璃體內部應力的平衡受到破壞而使玻璃體自行粉碎，不產生橫向的力，也沒有產生橫向力的能量。絕緣子玻璃體粉碎後，其碎片作為自由落體跌落地面，最大碎片約2cm，一般為1cm米下，碎片的重量和勢能能量很小，而且碎片剛度低、易碎，因此不會對設備造成危害。玻璃絕緣子已在變電站中使用，沒有出現玻璃絕緣子自爆時炸壞設備的情況。玻璃絕緣子有內部缺陷（例如玻璃體內殘存有雜質）或受大的外力衝擊時，絕緣子玻璃體外層壓應力和內部張應力的平衡受到破壞，內部的張應力得到“釋放”，使玻璃體破裂粉碎，這就是玻璃絕緣子的“零值自破”。零值自破是鋼化玻璃絕緣子一大特點，也是一大優點。玻璃絕緣子的自破，多發生在施工和運行初期，隨著時間的延長，自破率逐年下降並趨於穩定。只需憑藉目測就可方便地檢測出老化的絕緣子。就國產絕緣子而言，玻璃絕緣子的年自破率比瓷絕緣子的年零值率低。玻璃絕緣子的耐電弧能力比瓷絕緣子強。玻璃絕緣子是不可擊穿的，玻璃絕緣子零值自破的特點使它的外絕緣水平永遠小於內絕緣水平而不擊穿，具有零值絕緣子的玻璃絕緣子串，當發生汙閃或雷擊閃絡時，短路電流在零值絕緣子鐵帽和鋼腳的外部通過而不經過鐵帽內部（絕緣子自破後其鐵帽和鋼腳間的空間距離短），這就大大減少玻璃絕緣子掉串的概率。零值自破後的殘餘強度仍然很高，均大於額定負荷的標準值，因此不會造成導線落地事故。

試題編號 : 05

知識點 : 送電線路

某110kV線路開關跳閘，重合成功。當時天氣晴好。巡視發現#21鋼管塔C相（右下相）絕緣子有輕微放電痕跡（球頭上有放電痕跡，複合絕緣子傘群無閃絡痕跡），C相導線上有少量鳥糞痕跡，且發現C相的橫擔內有鳥窩及大量雜物、雜草，現場實測雜草有1米長。經詢問附近群眾得知，當時線路有放電引起的火光及聲響，且有鳥飛進橫擔內。請分析原因，提出反事故措施。

從當時的天氣情況、現場情況及附近群眾所觀察到的現象可以判斷：此次跳閘是由於鳥在杆塔橫擔內築巢，叼入雜物及雜草時對導線的安全距離不足引起。暴露問題：鋼管塔的橫擔是中空的，且在橫擔的杆塔側沒有封口，鳥類易在橫擔內築巢。防範對策：統計現有鋼管塔的線路名、數量及杆號，制定鋼管塔橫擔口封堵方案、對鋼管塔橫擔口實施封堵，或加裝驅鳥裝置；對於新建工程，制定鋼管塔的技術規範書時要求橫擔封口。

試題編號 : 06

知識點 : 送電線路

在對線路導線接頭進行測溫時，發現導線接頭溫度比環境溫度低。請作出判斷分析和工作指導。

測量數據明顯不對。造成這一原因可能是：測量距離太遠，超過了被測目標（導線接頭）相對應的有效測量距離；儀器的有關參數如輻射率的調整失當；儀器損壞。先就近測量其他物體的溫度以判斷儀器的好壞；正確調整測量參數；根據儀器距離係數計算出接頭的有效測量距離，把儀器拿到接頭的有效測量距離內對導線接頭進行測溫。

試題編號 : 07

知識點 : 送電線路

在對線路進行接地電阻測量時，接地搖表擺動不定。請作出判斷分析和工作指導。

接地搖表擺動不定，可能的原因是：地極打入深度不夠或因鬆動、土壤電阻率過高而與土壤接觸不好；儀器損壞；附近有變電站或其他電力設施在地中產生較大的地中電流。先對搖表進行開路和短路測量檢查以判斷儀器的好壞，如損壞，則更換好的儀器；把地極全打入地中並用水淋地極及其周圍。如這樣處理後，接地搖表仍然擺動不定，則是地中電流影響，需用大電流法或變頻接地電阻測試儀測量。

試題編號 : 08

知識點 : 送電線路

在對某線路的一基杆塔進行接地電阻測量時，測量值很小（小於1Ω），再反覆進行幾次測量，各測量值變化大。請作出分析判斷和工作指導。

測量值很小（小於1Ω），其原因有：杆塔位於土壤電阻率很小的地區；接地網在地下或基礎仍與鐵塔相連；接地極打對了接地網。各複測值間變化大，其原因有：電壓極和電流極與鐵塔間的距離太小，從而打在接地網的半徑範圍內，在每次複測時，電壓極和地網中心的距離與電流極和地網中心的距離的比例變化大；各個方向的土壤電阻率差別大；該接地網為引外式接地網，從而地網中心不在鐵塔處。綜合以上分析，出現題目中現象的原因是電壓極和電流極與鐵塔間的距離太小、不滿足DL/T887-2004《杆塔工頻接地電阻測量》要求。測量人員應按DL/T887-2004要求，把電壓極P和電流極C分別佈置在離杆塔基礎邊緣2.5L和4L處，L為杆塔接地裝置放射形接地極的最大長度。

試題編號 : 09

知識點 : 送電線路

有一鐵塔，採用重力式基礎，露出柱體為0.8mx0.8m，底寬度2.5mx2.5m，埋深3m，土質為可塑土（上拔角α為20°，摩擦角β為30°，容重γ0為16kN/立方米），陡坡40°，現距離基礎柱體邊緣1.8m處出現邊坡塌方，塌方長度8m、高度1.8m，已看見基礎。請提出基礎處理方案。

該基礎為重力式基礎，因此基礎邊坡塌方不影響基礎的上拔穩定性。基礎邊坡塌方的高度為1.8m，小於基礎埋深3m，因此基礎邊坡塌方不影響基礎的下壓穩定性。基礎處理的關鍵是控制基礎邊坡繼續塌方，當塌方高度大於基礎埋深時，將影響到基礎的下壓穩定性。故應採用以下方案：在基礎的上邊坡開設排水溝；回填土，夯實。


試題編號 : 10

知識點 : 送電線路

有一鐵塔，採用重力式基礎，露出柱體為0.8mx0.8m，底寬度2.5mx2.5m，埋深3m，土質為可塑土（上拔角α為20°，摩擦角β為30°，容重γ0為16kN/立方米），陡坡50°，現距離基礎柱體邊緣1.8m處出現邊坡塌方，塌方長度8m、高度3.5m，已看見基礎。請提出基礎處理方案。

該基礎為重力式基礎，因此基礎邊坡塌方不影響基礎的上拔穩定性。基礎邊坡塌方的高度為3.5m，大於基礎埋深3m，因此基礎邊坡塌方已影響基礎的下壓穩定性。基礎處理的關鍵是恢復已塌方的邊坡、控制基礎邊坡繼續塌方。故應採用以下方案：在已塌方的基礎邊坡砌築護坡；在基礎的上邊坡開設排水溝。

試題編號 : 11

知識點 : 送電線路

有一鐵塔，採用階梯式基礎，因處於邊坡上，為保護植被及增加基礎的穩定性，設計時已加大埋深（正常埋深2.5m，現埋深3.5m），露出柱體為0.8mx0.8m，底寬度2.5mx2.5m，土質為硬塑土（上拔角α為25°，摩擦角β為35°，容重γ0為17kN/立方米），陡坡52°，現距離基礎柱體邊緣1.5m處出現邊坡塌方，塌方長度10m、高度1m。請提出基礎處理方案。

該基礎為階梯式基礎，依賴基礎上方的土中來保證基礎的上拔穩定性。基礎邊坡塌方處已位於基礎的上方，基礎的抗拔力已有所減少，但由於基礎已加大埋深1m，而現塌方高度正好1m，因此基礎塌方後基礎的上拔穩定性仍然得到保證。基礎處理的關鍵是控制基礎邊坡繼續塌方。故應採用以下方案：在基礎的上邊坡開設排水溝；回填土，夯實。

試題編號 : 12

知識點 : 送電線路

在用火花間隙法對線路瓷絕緣子進行零值測量時，有一組人測量了3基杆絕緣子，每片絕緣子均沒有響聲，然後換一組人測量時，每片絕緣子均有響聲。請作出分析判斷和工作指導。

用火花間隙法測量零值瓷絕緣子時，為提高測量零值瓷絕緣子的準確率，火花間隙在帶到現場使用前應進行間隙放電試驗，調整好間隙，在每次上杆塔測量時，要用塞尺檢查間隙的變化情況。有一組人測量每片絕緣子均沒有響聲，可能是間隙發生了變化，間隙距離增大了；換一組人測量時每片絕緣子均有響聲，可能是間隙發生了變化，間隙距離減小了。應檢查間隙距離是否符合要求。另外，如果絕緣子增加較多，每片絕緣子的分佈電壓則減小較多，如仍用常規絕緣子串的間隙進行測量，也可能出現每片絕緣子均沒有響聲的情況，測量時需注意這一情況，根據電壓分佈調整相應的間隙距離。

試題編號 : 13

知識點 : 送電線路

某線路登杆塔檢查發現有幾基杆塔絕緣子串的U形環磨損嚴重（已磨損了直徑的一半）。請分析原因，提出防範措施。

原因分析：U形環磨損嚴重，外部原因是由於導線受到不規則的上下左右振盪，使金具和絕緣子串受到很大的衝擊拉力造成，這常發生在特殊地形地段，如風口；內部原因是金具的連接靈活性差，使金具受到較大的彎矩，久而久之，金具就嚴重磨損，甚至疲勞斷裂。防範措施：首先調查特殊地形杆塔，調查金具的磨損情況以找出磨損嚴重杆塔。然後對特殊地形和磨損嚴重杆塔，把金具更換為高強度的金具，或更換為雙掛點雙串，同時將絕緣子與橫擔的連接用兩個U形環連接以增加金具的連接靈活性。

有一耐張轉角塔JG2-18，運行若干年後內傾5‰。請提出處理方案。

運行若干年後出現內傾，說明了杆塔所在位置出現了地質變化，需進行持續監視，監視的週期取決於鐵塔傾斜的變化率，一般情況下每年測量一次，如每月都有可測量的增長，則可按每月測量一次，如每季度有可測量的增長，則可按每季度測量一次。如鐵塔傾斜的增長較快，說明該塔位處地質不穩定，需遷移該塔到地質穩定之處。如鐵塔傾斜的變化很小，連續3年無明顯變化，說明該處地質穩定，可以調整杆塔，確保線路不內傾。方法是：用千斤頂頂起塔基，墊高基礎，再進行基礎處理。

有一鐵塔，採用階梯式基礎，因處於邊坡上，為保護植被及增加基礎的穩定性，設計時已加大埋深（正常埋深2.5m，現埋深3.5m），露出柱體為0.8mx0.8m，底寬度2.5mx2.5m，土質為硬塑土（上拔角α為25°，摩擦角β為35°，容重γ0為17kN/立方米），陡坡52°，現距離基礎柱體邊緣1.5m處出現邊坡塌方，塌方長度10m、高度2m。請提出基礎處理方案。

在線路運行中，杆塔基礎塌方時有發生，當杆塔基礎塌方時需作出基礎處理方案。請問作出基礎處理方案需要了解哪些情況，並說明理由。

當因杆塔基礎塌方作出基礎處理方案前首先要了解基礎的受力穩定性情況。這需瞭解基礎的型式、尺寸和深度，土質，塌方的邊坡距離、長度、高度，周圍土質的穩定性。瞭解基礎的型式可判斷塌方是否影響基礎的穩定性及其影響程度，瞭解基礎的尺寸和深度、塌方的邊坡距離、高度、土質可判斷塌方對基礎穩定性的影響程度，瞭解塌方的長度可瞭解塌方的影響範圍，瞭解周圍土質的穩定性可瞭解塌方的可能發展趨勢。

試題編號 : 18

知識點 : 送電線路

有一耐張塔JG2-18，經竣工驗收檢查發現內傾3‰。請分析判斷其原因並提出處理方案。

造成鐵塔內傾的原因有：基礎預偏不夠，鋼材不滿足設計要求，鐵塔螺栓鬆。鐵塔內傾的原因分析判斷：先檢查鋼材是否滿足設計要求，如果規格偏小或強度偏低，則鐵塔內傾是鋼材不滿足設計要求造成的。再測量基礎預偏是否符合要求，如果基礎預偏符合要求，則鐵塔內傾是由於鐵塔螺栓鬆造成的；如果基礎預偏不符合要求，則比較基礎減少的預偏量與鐵塔的內傾量再判斷。如果基礎減少的預偏量大於鐵塔的內傾量，則鐵塔內傾是基礎預偏不夠造成的；如果基礎減少的預偏量小於鐵塔的內量，則鐵塔內傾是由於基礎預偏不夠及鐵塔螺栓鬆共同造成的。處理方案：如果鋼材不滿足設計要求，則更換鋼材。如果基礎預偏不夠，則用千斤頂頂起塔基，墊高基礎，再進行基礎處理。如果鐵塔螺栓鬆，則打好每根導線的反向臨時拉線，對地夾角不大於30°並儘量小，調節臨時拉線使鐵塔外傾，然後逐段鬆螺栓和緊螺栓。如果同時存在基礎預偏不夠及鐵塔螺栓鬆，則打好每根導線的反向臨時拉線，對地夾角不大於30°並儘量小，調節臨時拉線使鐵塔外傾，然後墊高基礎，並逐段鬆螺栓和緊螺栓。

利用人字抱杆起吊杆塔，抱杆座落位置及初始角的大小對起吊工作有何影響?

答：利用人字抱杆起吊杆塔時，若抱杆座落過前，初始角過大，在起吊過程中，易造成抱杆脫帽過早，增大主牽引繩的牽力，並且起吊過程中，穩定性差。抱杆座落過後，初始角小，起吊過程中易造成抱杆脫帽晚。這種情況初始時主牽引力和抱杆受垂直下壓力都會增大，特別是在抱杆強度不足的情況下，會造成抱杆變形破壞，對起吊工作是不利的。所以，在施工中抱杆的初始角和座落位置一定要按施工設計進行。

知識點 : 工器具知識

簡述鉗形電流表的用途和工作原理。

答：通常用普通電流表測量電流時，需要切斷電路才能將電流表或電流互感器一次繞組串接到被測電路中。而使用鉗形電流表進行測量時，則可在不切斷電路的情況下進行測量。其工作原理如下：鉗形電流表由電流互感器和電流表組成。互感器的鐵心有一活動部分，並與手柄相連，使用時按動手柄使活動鐵心張開。將被測電流的導線放入鉗口中，放開後使鐵心閉合。此時通過電流的導線相當於互感器一次繞組，二次繞組將出現感應電流，其大小由導線的工作電流和圈數比確定。電流表是接在二次繞組兩端的，因而它所指示的電流是二次繞組中的電流，此電流與工作中的電流成正比。由於將歸算好的電流標在電流表的刻度上，當導線中有工作電流通過時，和二次繞組相連的電流表指針便按比例發生偏轉，從而指示出被測電流的數值。
