目前還剩下，如何通知用戶是否購票成功。

之前我嘗試過使用qq郵箱的smtp服務，搶票成功後發送郵件通知，但是我覺得吧，並不好用，主要是我沒有打開郵箱的習慣，沒網也收不到，所以，並沒有採納這個方案。

加上之前我註冊過企業認證的公眾號，騰訊雲免費送了我1000條短信通知，而且短信也比較直觀，所以我這裡就安裝騰訊雲的SDK，部署了一套發短信的功能。

看下短信正文，{Number}這些裡面的數字是變量。


就是說短信的模板是固定的，但是裡面有{Number}的內容可以自定義

調用的時候，裡面的數字對應著傳過去的參數數組序號，{1}代表數組[0]參數，以此類推

定時任務

也聲明一個類，這裡我們用到的是schedule

// 定時任務class SetInter {  constructor({ timer, fn }) {    this.timer = timer // 每幾秒執行    this.fn = fn //執行的回調    this.rule = new schedule.RecurrenceRule(); //實例化一個對象    this.rule.second = this.setRule() // 調用原型方法，schedule的語法而已    this.init()  }  setRule() {    let rule = [];    let i = 1;    while (i < 60) {      rule.push(i)      i += this.timer    }    return rule //假設傳入的timer為5，則表示定時任務每5秒執行一次    // [1, 6, 11, 16, 21, 26, 31, 36, 41, 46, 51, 56]   }  init() {    schedule.scheduleJob(this.rule, () => {      this.fn() // 定時調用傳入的回調方法    });  }}複製代碼
