1、數據類型

a、字符類型是一種數據類型，它的名稱是char

字符類型裡包含256個不同的整數，每個整數可以代表一個字符數據(例如'a','^'等)

整數和字符僅僅是兩種不同的表示方式，ASCII碼錶中列出所有整數和字符之間的

對應關係

'a' 97

'A' 65

'0' 48

ASCII碼錶中所有小寫英文字母是連續排列的，

其中'a'的整數最小，'z'的整數最大

所有大寫英文字母和阿拉伯數字字符也都符合

這個規律

'd' - 'a' 等於 'D' - 'A'

'd' - 'a' 等於 '3' - '0' 等於 3 - 0

所有字符數據被分成兩組，每組包含128個

其中一組字符數據兩種表示方式之間的對應關係在所有計算機上都一樣，他們的整數範圍從0到127

另外一組字符數據兩種表示方式之間的對應關係在不同計算上不一樣，他們的整數範圍可能從-128到-1也可能從128到255

無符號字符類型也是一種數據類型它的名字是unsigned char

它同樣包含256個不同的字符數據，這些數據的整數範圍在所有計算機上都是從0到255

'\n' 換行

'\r' 回車

'\\' 代表字符\

'\'' 代表字符'

'\"' 代表字符"

b、短整數類型也是一種數據類型它的名稱是short它裡面包含65536個不同的整數，其中一半是非負數另外一半是負數。以數字0為中心向兩邊擴展。

無符號短整數類型也是一種數據類型它的名稱是unsigned short，它裡面也包含65536個不同的整數，從0開始向正數方向遞增，不包含負數

c、長整數類型也是一種數據類型它的名稱是long或者long int

它裡面包含2的32次方個不同的整數，其中一半是負數另外一半是非負數。以0為中心向兩邊擴展。

無符號長整數類型名稱是unsigned long或者unsigned long int

它裡面同樣包含2的32次方個不同的整數，從0開始向正數方向擴展，不包括負數

d、整數類型名稱是int，在我們的計算機上整數類型和長整數類型一樣

無符號整數類型名稱是unsigned int

在我們的計算機上無符號整數類型和無符號長整數類型一樣，所有隻包含整數的類型他們的數字範圍是重疊的並且逐級擴大在程序中不帶小數點的數字後加u表示這個數字是無符號整數類型

d、C語言程序中使用浮點類型表示帶小數點的數字。浮點類型分成單精度浮點類型和雙精度浮點類型

雙精度浮點類型可以記錄更多小數點後面的數位

單精度浮點類型的名稱是float

雙精度浮點類型的名稱是double

程序中帶小數點的數字計算機默認它的類型是雙精度浮點類型，如果在帶小數點數字後加f表示這個數字的類型是單精度浮點類型

2、C語言中允許程序員擴展新的數據類型，這些新的數據統稱為複合數據類型

複合數據類型需要先定義然後才可以使用

布爾類型是C99規範中引入的新的數據類型，布爾類型中只包含兩個整數，一個是真另外一個是假(計算機裡採用0代表假用1代表真)

任何一個整數都可以當作布爾值使用，0當作布爾值使用時是假，其他所有整數當布爾值使用時都是真

一般在程序中不需要使用這個數據類型，直接用整數代替就可以

數據類型和佔位符的對應關係

char和unsigned char %c

short %hd

unsigned short %hu

long %ld

unsigned long %lu

int %d

unsigned int %u

float %f或%g

double %lf或%lg

%f和%lf會保留小數點後面多餘的0

%g和%lg不會保留

練習：

1.編寫程序計算長7，寬3 長方形的面積，並把結果打印在屏幕上（用變量記錄）

3、不同數據類型的一個主要區別就是存儲區裡包含的字節個數不同

sizeof關鍵字可以用來計算一個存儲區或數據

類型中包含的字節個數

char和unsigned char 1個字節

short和unsigned short 2個字節

long和unsigned long 4個字節

int和unsigned int 4個字節

float 4個字節

double 8個字節

sizeof的小括號裡可以寫任何能當數字使用的內容

sizeof小括號裡如果修改了任何存儲區的內容，這個修改不會真正生效

這裡大家可以將代碼敲出來查看打出的具體數值，不明白的評論諮詢。。。謝謝~~

4、scanf標準函數可以把用戶在鍵盤上輸入的數字記錄在變量代表的存儲區裡

為了使用這個標準函數也需要包含stdio.h文件

在scanf函數調用語句中要使用存儲區的地址表示存儲區

雙引號裡使用佔位符表示存儲區的類型

在scanf函數調用語句的雙引號裡不要寫佔位符以外的其他內容

可以在一條scanf函數調用語句中得到多個數字

如果用戶輸入的格式和程序要求的格式不一致就可能不能得到正確數字
