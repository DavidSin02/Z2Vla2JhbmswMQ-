電力金具名詞術語

本標準規定了電力金具有關的專用名詞術語，主要供製訂標準、編制技術文 件、編寫和翻譯專業手冊、教材及書刊使用。

1 一般術語

1.1 電力金具

electric power fitting

連接和組合電力系統中各類裝置，以傳遞機械，電氣負荷及起到某種防護作 用的金屬附件。

1.2 線路金具

overhead power line fitting

架空電力線路上用的電力金具。

1.3 變電金具

electric substation fitting

變電所及發電廠配電裝置用的電力金具。

1.4 母線金具

bus-bar fitting

變電所及發電廠配電裝置母線用的電力金具。

1.5 破壞荷重

failure load

按規定試驗方法，在承受機械荷重時，試件任何部位產生破壞時的荷重值。

1.6 標稱破壞荷重

nominal failure load

設計規定的最低破壞荷重值。

1.7 最大使用張力

maximum working strength

導線允許承受的最大設計荷重值。

1.8 滑動

slip

試驗荷重不能繼續上升時，導線與線夾或接續管之間產生相對位移的現象。

1.9 握力

grip strength

導線固定在線夾或接續管中不產生滑動的最大荷重值。

1.10 可見電暈

visval corona

在指定條件下，憑肉眼能見到被試產品出現的電暈現象。

1.11 標稱值

nominal value

用以標識一個元件、器件或設備的合適的近似量值。

1.12 溫升

temperature rise

某一點的溫度與基準溫度之差。

2 產品名稱

2.1 懸垂線夾

2.1.1 懸垂線夾

suspension clamp

用於懸掛或支託導線，主要承受垂直荷重的金具。

2.1.2 懸垂線夾(鋼質)

suspension clamp(steel)

用鋼板或可鍛鑄鐵等材質製造的懸垂線夾。

2.1.3 懸垂線夾(鋁質)

suspension clamp (aluminium)

用鋁合金材料製造的懸垂線夾。

2.2 耐張線夾

2.2.1 耐張線夾

strain clamp

用於固定導線的端頭，並承受導線張力的金具。

2.2.2 耐張線夾(螺栓型)

strain clamp(bolted type)

用螺栓件固定導線的耐張線夾。

2.2.3 耐張線夾(壓縮型)

strain clamp(compression type)

用壓縮方法固定導線的耐張線夾。

2.2.4 耐張線夾(楔型)

strain clamp(wedge type)

用楔子固定導線的耐張線夾。

2.3 連接金具

2.3.1 掛環

link

兩端均為環形的連接件。

2.3.2 掛板

clevis & tongue

兩端均為螺栓組裝的板形連接件。

2.3.3 球頭掛環

ball-eye

由腳球與環所構成的一種連接件。

2.3.4 球頭掛鉤

ball-hook

由腳球與掛鉤所構成的一種連接件。

2.3.5 碗頭掛板

socket-clevis eye

由帽窩與雙片或單片掛板所構成的一種連接件。

2.3.6 球頭掛板

ball-clevis

由腳球與雙片或單片掛板所構成的一種連接件。

2.3.7 U形掛環

shackle

由U形環與雙片掛板所構成的一種連接件。

2.3.8 U形螺絲

U-bolt

由U形環與螺紋杆所構成的一種連接件。

2.3.9 聯板

yoke plate

將多個分支並聯組裝用的一種板形連接件。

2.3.10 牽引板

towing plate

帶有牽引孔的一種掛板。

2.3.11 掛鉤

hook

由圓環與掛鉤所構成的一種連接件。

2.3.12 吊架

hanger

在電杆上支持避雷線代替橫擔的組合件。

2.3.13 調整板

adjusting plate

可調節連接長度的板形連接件。

2.3.14 花籃螺絲

turn buckle

用左右旋螺紋來調節連接長度的組合件。

2.4 接續金具

2.4.1 接續管

splicing sleeve

用於接續導線，並能滿足機械、電氣性能要求的金屬管件。

2.4.2 接續管(橢圓形)

splicing sleeve(oval)

橫截面為橢圓形的接續管。

2.4.3 接續管(圓形)

splicing sleeve(circular)

橫截面為圓形的接續管。

2.4.4 補修管

repairsleeve

修補導線表面損傷用的金屬管件。

2.4.5 並溝線夾

parallel groove clamp

平行接續導線，以傳遞電氣負荷的線夾。

2.4.6 跳線線夾

jumper clamp

非直線杆塔跳線接續用的線夾。

2.5 防護金具

2.5.1 防振錘

damper

抑制導線受微風振動用的錘形組合件。

2.5.2 均壓環

grading ring

改善絕緣子串電壓分佈的環狀金具。

2.5.3 屏蔽環

shielding ring

使被屏蔽範圍內不出現電暈現象的環狀金具。

2.5.4 均壓屏蔽環

grading and shielding ring

兼有均壓和屏蔽作用的環狀金具。

2.5.5 招弧角

arcing horn

防止電弧沿絕緣子表面閃絡的角狀金具。

2.5.6 間隔棒

spacer

使分裂導線的子導線之間保持固定間隔的金具。

2.5.7 預絞式護線條

preformed armor rod

增加導線剛度及耐振能力的螺旋形線條。

2.5.8 鋁包帶

aluminium armor tape

包纏鋁質絞線使其表面不受磨損的鋁帶。

2.5.9 重錘

counter weight

掛在絕緣子串下端，以增大垂直荷重的重塊。

2.6 拉線金具

2.6.1 線卡子

guy clip

固定鋼絞線端部的馬鞍形組合件。

2.6.2 心形環

thimble

繞扎鋼絞線或鋼絲繩用的襯墊。

2.7 接觸金具

2.7.1 設備線夾

terminal connector

導線與電氣設備出線端子相連接用的金具。

2.7.2 T形線夾

T-connector

導線與分支線相連接的T形金具。

2.7.3 銅鋁過渡板

copper to aluminium adapter board

鋁質端子與銅質端子之間相連接，以防止電化腐蝕作用的過渡接觸板件。

2.8 母線金具

2.8.1 硬母線固定金具

bus-bar support

固定和懸掛硬母線附件的總稱。

2.8.2 母線間隔墊

bus-bar separator

保持硬母線片間一定間隔的支撐件。

2.8.3 母線伸縮節

bus-bar expansion joint

用於補償母線因受熱變形和振動變形的彈性元件。

2.8.4 管母線封端球

tubing bus-bar corona ball

防止管母線終端產生電暈用的球狀元件。

3 檢驗、試驗方法

3.1 外觀檢查

visual inspection

以目力觀察為主的對外表狀態的檢查。

3.2 型式試驗

type-tests

對按照一定技術條件設計製造的部件所進行的試驗，以表明這項設計符合一 定的標準。

3.3 抽樣試驗

sampling tests

在同一條件下所生產的產品中，隨機地提取一定的數量所進行的試驗。

3.4 出廠試驗

routine tests

對每個部件在製造中或完後所進行的試驗，以判明其是否符合某項標準。

3.5 機械破壞荷重試驗

mechanical failing load tests

在規定的試驗條件下作機械破壞試驗，試件達到最大負荷而破壞的荷重值。

3.6 握力試驗

grip strength tests

在規定的試驗條件下，對安裝導線的線夾所進行的握力值測定。

3.7 振動試驗

vibration tests

為測定防振器各部件，在規定條件下的振動情況所進行的試驗。

3.8 電阻試驗

resistance tests

以直流電測定導電元件電阻值的試驗。

3.9 溫升試驗

temperature-rise tests

在規定條件下，測定導電元件的溫升試驗。

3.10 老化試驗

ageing tests

在試件上施加規定的老化因子，從而能比實際使用期短得多的時間內，評定 出使用壽命的試驗。

3.11 低電壓大電流試驗

low-voltage high-current tests

試件在電壓為額定值的某一小額百分率，電流為額定值或大於額定值時進行 的試驗。

3.12 熱穩定試驗

thermal stability tests

為確定在規定的過負荷條件下，試件的溫度和損耗的熱穩定性的試驗。

3.13 無線電干擾試驗

radio interference tests

測定試件在運行電壓下，對無線電干擾電平的試驗。

3.14 局部放電起始電壓試驗

partial discharge inception voltage tests

測量導電體表面出現局部放電時的最低電壓試驗。

3.15 電暈試驗

corona tests

在導體附近發生不均勻的場強很高電場中的輝光放電試驗。

3.16 鋅層均勻性試驗

test for uniformity of galvanized coating

以金屬離子的置換，檢驗鍍層均勻程度的試驗。

3.17 衝擊動荷試驗

impulse load tests

在規定條件下，對元件進行瞬間的機械動荷載，以模擬部件在事故條件下的 承載能力的試驗。

3.18 耐腐試驗

corrosion resistance tests

暴露在一定溫度和鹽度的條件下，試驗元件耐腐程度的試驗。
