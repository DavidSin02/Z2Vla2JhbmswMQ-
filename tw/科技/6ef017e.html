題組一　物理變化與化學變化的判斷

[挑戰滿分](限時10分鐘)

1．下列物質的使用不涉及化學變化的是( )

A．明礬用作淨水劑 B．液氨用作製冷劑

C．氫氟酸刻蝕玻璃 D．生石灰作乾燥劑

答案　B

解析　明礬作為淨水劑的原因：溶於水電離出的鋁離子發生水解，生成的氫氧化鋁膠體具有強吸附性；氫氟酸能刻蝕玻璃的原因：氫氟酸與玻璃中的二氧化硅反應生成四氟化硅和水；生石灰作乾燥劑的原因：生石灰與水反應生成氫氧化鈣，以上物質的使用均涉及化學變化，A、C、D錯誤；液氨用作製冷劑的原因：液氨揮發吸熱使周圍環境溫度迅速降低，不涉及化學變化，B正確。

2．下列各組變化中，前者是物理變化，後者是化學變化的是( )

A．碘的昇華、石油的分餾

B．NH4Cl溶液除鐵鏽、食鹽水導電

C．蛋白質溶液中加飽和(NH4)2SO4溶液、藍色的膽礬常溫下變白

D．熱的飽和KNO3溶液降溫析出晶體、二硫化碳洗滌試管內壁上的硫

答案　C

解析　選項A、D都是物理變化；選項B，兩者都是化學變化，電解質溶液的導電過程就是電解的過程，電解必然會產生新物質，故食鹽水導電是化學變化；蛋白質的鹽析是物理變化，膽礬變白，說明膽礬失去結晶水，發生的是化學變化，選項C符合題意。

3．下列有關變化過程的敘述不正確的是( )

A．從海水中提取鎂必須通過化學反應才能實現

B．用侯氏制鹼法制純鹼必須通過化學反應才能實現

C．用"地溝油"純化生產"生物燃油"的過程發生的是化學變化

D．激光法蒸發石墨得C60發生的是化學變化

答案　C

解析　從海水中提取鎂的過程是Mg2＋―→Mg(OH)2―→MgCl2―→Mg，其中發生了化學變化，A項正確；侯氏制鹼法的過程是NaCl―→NaHCO3―→Na2CO3，其中發生了化學變化，B項正確；"地溝油"純化是分餾的過程，發生的是物理變化，C項錯誤；石墨與C60的結構不同，在轉化的過程中有舊化學鍵的斷裂與新化學鍵的形成，發生的是化學變化，D項正確。

4．下列有關物質變化的說法正確的是( )

A．熔融態的Al2O3導電、12C轉化為14C都屬於化學變化

B．根據反應中的能量變化，將化學反應分為"化合、分解、複分解、置換"四類

C．一定條件下石墨轉化為C60是化學變化

D．將盛有NO2氣體的密封燒瓶，放入熱水中氣體顏色加深的變化是物理變化

答案　C

解析　A項，熔融態的Al2O3導電的同時，Al3＋和O2－分別在陰陽兩極放電是化學變化，12C轉化為14C不是化學研究的範疇，不屬於化學變化，錯誤；B項，依據反應物和生成物的種類可將化學反應分為化合、分解、複分解、置換四種類型，錯誤；C項，同素異形體之間的相互轉化屬於化學變化，正確；D項，2NO2(紅棕色)N2O4(無色)　ΔH<0，放入熱水中，顏色變化的原因是平衡移動，屬於化學變化，錯誤。

5．下列過程中顏色變化與化學變化無關的是( )

A．銀白色的金屬鐵研磨成鐵粉後變成黑色

B．新切割開的金屬鈉的表面由銀白色迅速變成暗灰色

C．紅棕色的NO2通過裝有水的洗氣瓶後得到無色氣體

D．淺黃綠色的新制氯水久置後變為無色

答案　A

解析　A項中發生顏色變化是光學現象的變化，故鐵研磨成鐵粉屬於物理變化；B項中金屬鈉被氧化成了Na2O，屬於化學變化；C項中NO2通過裝有水的洗氣瓶後生成了NO，屬於化學變化；D項中新制氯水變為無色，是Cl2＋H2OHCl＋HClO平衡不斷右移的結果，屬於化學變化。

6．中華民族的發明創造為人類文明進步做出了巨大貢獻。下列我國古代發明中，不涉及化學反應的是( )

A．打磨磁石制指南針 B．燒結粘土製陶瓷

C．溼法鍊銅 D．鐵的冶煉

答案　A

解析　A項，打磨磁石制指南針沒有新物質生成。

7．中國漢語文化博大精深，很多成語體現了化學知識。下列成語中涉及到的主要變化不是化學反應的是( )

A．百鍊成鋼 B．飲鴆止渴

C．鐵杵磨針 D．火上澆油

答案　C

解析　A項，百鍊成鋼有新物質鐵生成；B項，飲鴆止渴使人中毒，發生蛋白質變性；C項，鐵杵磨針沒有新物質生成；D項，火上澆油發生燃燒有新物質生成。

8．我國古代文獻資料中記載了大量的化學研究過程與成果。下列選項不涉及化學變化的是( )

答案　D

解析　A項，水火既濟而土和，即燒製陶瓷過程，陶瓷燒製過程中有新物質生成，屬於化學變化；B項，"曾青"是指可溶性銅鹽，即用鐵將銅的化合物中的銅置換出來，有新物質生成，屬於化學變化；C項，丹砂為硫化汞，不穩定，加熱發生HgS分解生成Hg和S，溫度降低時，又可發生Hg＋S===HgS，該過程中有新物質生成，屬於化學變化；

D項，重結晶過程是將晶體溶於溶劑或熔融以後，又重新從溶液或熔體中結晶的過程，沒有新物質生成，屬於物理變化。

9．下列變化不能說明發生了化學變化的是( )

A．變化時有電子的得失或共用電子對的形成

B．變化過程中有化學鍵的斷裂和形成

C．變化時釋放或吸收能量

D．變化前後原子的種類和數目沒有改變，分子種類增加了

答案　C

解析　A項，電子的得失或共用電子對的形成過程中有新物質生成，屬於化學變化，故A錯誤；B項，化學鍵的斷裂和形成說明有新物質生成，屬於化學變化，故B錯誤；C項，物理變化時也能釋放出能量，不能說明生成新物質，故C正確；D項，分子是保持物質化學性質的最小粒子，分子的種類變了，一定發生了化學變化，故D錯誤。

題組二　常見無機反應類型的判斷

[挑戰滿分](限時10分鐘)

1．生物法淨化含硫廢氣時發生如下反應：

。下列有關說法正確的是( )

A．反應中CH3COOH是還原劑

B．反應中SO被氧化

C．H2S的酸性比H2CO3強

D．該反應屬於複分解反應

答案　A

解析　脫硫時硫元素的化合價由＋6價降低到－2價，則硫元素被還原，SO為氧化劑，CH3COOH為還原劑，A項正確；B、D兩項均錯誤；若H2S的酸性比H2CO3強，則應生成H2CO3和HS－(或S2－)，C項錯誤。

2．某學生以鐵絲和Cl2為原料進行下列三個實驗。從分類角度分析，下列選項正確的是( )

A．實驗①、③反應制得的物質均為純淨物

B．實驗②、③均未發生氧化還原反應

C．實驗②、③均為放熱反應

D．實驗①、②所涉及的物質均為電解質

答案　B

解析　實驗③製得的膠體為混合物，A選項錯誤；實驗②氯化鐵溶於水屬於物理變化，實驗③中發生的水解反應屬於複分解反應，B選項正確；實驗②屬於物理變化，該過程中放熱，實驗③為吸熱反應，C選項錯誤；實驗①所涉及的Fe、Cl2都不是電解質，D選項錯誤。

3．如圖是某燃煤發電廠處理廢氣的裝置示意圖。裝置內發生的主要反應中不含( )

A．化合反應 B．分解反應

C．置換反應 D．氧化還原反應

4．下列說法在一定條件下可以實現的是( )

①酸性氧化物與鹼發生反應　②弱酸與鹽溶液反應可以生成強酸　③沒有水生成，也沒有沉澱和氣體生成的複分解反應　④兩種酸溶液充分反應後的溶液為中性　⑤有單質參加的非氧化還原反應　⑥兩種氧化物反應的產物有氣體

5．下表中有關變化和反應的對應關係正確的是( )

答案　C

解析　乾餾是化學變化，A項錯誤；2FeCl3＋Cu===2FeCl2＋CuCl2不是置換反應，B項錯誤；木炭使品紅溶液褪色的原因是木炭的吸附作用，屬於物理變化，C項正確；在沸水中滴加飽和FeCl3溶液生成Fe(OH)3膠體，不是沉澱，D項錯誤。

6．科學家最近研究出一種環保、安全的儲氫方法，其原理可表示為

。下列有關說法正確的是( )

A．儲氫過程中，NaHCO3被氧化

B．釋氫過程的本質是鹽類的水解

C．儲氫、釋氫過程均無能量變化

D．HCOONa晶體中既含有離子鍵又含有共價鍵

解析　A項，儲氫過程中C元素的化合價由＋4降低為＋2；B項，鹽類水解是形成鹽的陽離子與水電離的氫氧根結合，陰離子與水電離的氫離子結合，生成酸與鹼的過程；C項，化學變化中一定伴隨能量變化；D項，HCOONa晶體是離子晶體，由Na＋與HCOO－構成，HCOO－中含有共價鍵。

7．中學化學中很多"規律"都有其使用範圍，下列根據有關"規律"推出的結論合理的是( )

A．根據氧化還原反應的規律，推出強氧化劑遇到強還原劑一定會發生氧化還原反應

B．根據化學反應的規律，推出氫氧化鐵可通過化合反應得到

C．根據元素的非金屬性較強，其單質也越活潑規律，推出磷單質比N2穩定得多

D．根據CO2通入Ba(NO3)2溶液中無沉澱的規律推測，SO2通入Ba(NO3)2溶液中也無沉澱產生

答案　B

解析　A項，一般強氧化劑遇到強還原劑可以發生氧化還原反應，但同種元素之間必須有中間價態才能發生；B項，氫氧化鐵可以由氫氧化亞鐵、氧氣和水發生氧化還原反應得到；C項，氮元素非金屬性大於磷，由於N2存在NN三鍵，N2很穩定；D項中SO2可被酸化的NO氧化為SO，會產生BaSO4沉澱。

答案　D

解析　A項，草木灰的有效成分為碳酸鉀(K2CO3)，K2CO3溶液顯鹼性，OH－可與銨鹽發生反應，導致氮元素損失，不是K2CO3和銨鹽直接發生複分解反應，原理分析錯誤；B項，遼寧艦主要由鋼材料製造，艦底鑲嵌一些比較活潑的金屬，如鎂、鋅等，在海水中構成原電池，較活潑金屬作負極被腐蝕，鋼材料作正極被保護，如果鑲嵌銅塊，銅為正極，鋼為負極，會加快鋼的腐蝕，錯誤；C項，相同狀況下，同體積的天然氣燃燒比液化氣燃燒消耗的氧氣少，液化氣灶具改天然氣灶具時需減小空氣進入孔的孔徑，避免天然氣隨空氣流失而降低熱量利用率，原理分析錯誤；D項，碳酸氫鈉與酸反應生成CO2，使麵糰產生氣孔，正確。

題組三　離子反應的綜合判斷

[解題指導]

1．命題時常用的設錯方式

(1)反應不符合客觀事實。

(2)反應不符合題設條件(如：滴加順序、反應物量的問題)。

(3)不配平(漏掉不起眼的離子，電子、電荷、質量不守恆)

2．離子方程式判斷題的答題流程

第一步：分析離子反應是否符合客觀事實

(1)只有實際發生的離子反應才能書寫離子方程式。

(2)離子方程式中的化學式、各種符號及反應產物必須符合實驗事實，不能隨意捏造。

第二步：判斷生成物對不對

(1)變價元素的生成物(高價還是低價)。

(2)量不同時的生成物(某反應物過量是否會與生成物再反應)。

(3)反應物滴加順序不同時的生成物。

逐滴加入試劑時，不同時段的生成物。

第三步：分析離子方程式平不平

(1)原子是否守恆。

(2)電荷是否守恆。

(3)氧化還原反應中電子得失是否守恆。

[挑戰滿分](限時15分鐘)

1．FeCl3溶液是實驗室中使用較為廣泛的一種試劑，下列有關FeCl3溶液的作用或實驗現象解釋合理的是( )

答案　B

解析　FeCl3溶液使藍色石蕊試紙變紅是因為Fe3＋的水解，水解反應為可逆反應，A項不合理；檢測FeCl3溶液中的Fe3＋，應用SCN－，二者發生反應生成紅色的Fe(SCN)3，B項合理；電荷不守恆，正確的離子方程式為2Fe3＋＋Cu===Cu2＋＋2Fe2＋，C項不合理；NH3·H2O為弱電解質，在離子方程式中不能拆寫，D項不合理。

2．下列反應的離子方程式書寫不正確的是( )

答案　C

解析　氨水不能溶解Al(OH)3，A項不符合題意；用氨水吸收過量的SO2，生成NH4HSO3，B項不符合題意；向偏鋁酸鈉溶液中加入過量稀鹽酸，發生反應AlO＋4H＋===Al3＋＋2H2O，C項符合題意；向次氯酸鈣溶液中通入過量的二氧化碳，生成HClO和Ca(HCO3)2，D項不符合題意。

3．下列離子方程式書寫正確的是( )

答案　A

解析　氯氣溶於水生成HCl和HClO(弱酸)，A項正確；向沸水中滴入飽和FeCl3溶液，繼續煮沸至溶液顯紅褐色，生成Fe(OH)3膠體，不是沉澱，B項錯誤；水解為可逆反應，C項錯誤；H2S為弱酸，不能拆寫，D項錯誤。

4．能正確解釋下列反應原理，並且書寫正確的離子方程式是( )

答案　D

解析　A項應生成Al(OH)3膠體，且不能寫等號，應寫可逆號；B項中小蘇打的HCO不能拆開；C項，電解飽和MgCl2溶液得到氫氧化鎂沉澱。

5．下列有關NaClO和NaCl混合溶液的敘述正確的是( )

答案　D

解析　NaClO和NaCl的混合溶液中含有ClO－，其在酸性條件下具有強氧化性，與H＋和Br－不能大量共存，A項錯誤；混合溶液中的Cl－與Ag＋反應生成氯化銀沉澱，不能大量共存，B項錯誤；溶液為鹼性，滴入少量硫酸亞鐵時，亞鐵離子被ClO－氧化，但離子方程式中不能寫H＋而應寫水，且生成Fe(OH)3，C項錯誤；向溶液中加入濃鹽酸時發生反應：ClO－＋Cl－＋2H＋===Cl2↑＋H2O，每產生1molCl2轉移1mol電子，約為6.02×1023個，D項正確。

6．下列反應的離子方程式書寫正確的是( )

答案　D

解析　氨水是弱電解質，書寫離子方程式時不能拆開，應寫化學式，A項錯誤；應是水參與反應而不是OH－，且生成的是錳離子而不是二氧化錳，B項錯誤；電解飽和MgCl2溶液，應生成氫氣、氯氣和氫氧化鎂沉澱，C項錯誤；次氯酸根離子將亞硫酸根離子氧化為硫酸根離子，自身被還原為氯離子，D項正確。

7．一定能在下列溶液中大量共存的離子組是( )

答案　A

解析　B項，SO、CO和Ca2＋生成沉澱不能大量共存；C項，SCN－和Fe3＋因發生絡合反應不能大量共存；D項，H＋、Fe2＋、NO因發生氧化還原反應不能大量共存。

8．下列各組離子或分子在溶液中能大量共存，且滿足相應要求的是( )

答案　C

解析　不符合溶液電中性原理，A項錯誤；Fe3＋與SO不能大量共存，B項錯誤；滴加氨水開始產生MgCO3沉澱，後產生Mg(OH)2沉澱，C項正確；滴加NaOH濃溶液會先和醋酸反應，不會立刻產生氣體，D項錯誤。

9．在給定條件下，下列離子組的離子共存判斷及反應的離子方程式均正確的是( )

答案　C

解析　滴加氨水，離子組中的Fe3＋與氨水反應生成Fe(OH)3沉澱，不能大量共存，氨水應寫化學式，離子方程式書寫錯誤，A項錯誤；因由水電離出的c(H＋)＝1×10－13mol·L－1的溶液可能呈酸性，也可能呈鹼性，在酸性溶液中AlO不能大量共存，在鹼性溶液中NH不能大量共存，B項錯誤；酸性溶液中Fe3＋與I－發生氧化還原反應而不能大量共存，C項正確；SO2與ClO－不能大量共存，二者會發生氧化還原反應生成SO，D項錯誤。

10．某溶液可能含有Na＋、NH、Ba2＋、SO、I－、S2－中的幾種離子。進行如下檢測：①用pH計檢測，溶液顯弱酸性；②加新制氯水和澱粉溶液無明顯現象。為確定該溶液的組成，還需檢驗的離子是( )

A．Na＋ B．Ba2＋

C．SO D．S2－

答案　A

解析　實驗①說明溶液中含有NH(水解使溶液顯弱酸性)；實驗②說明溶液中無I－、S2－，再根據電荷守恆可推出該溶液中含有SO，則Ba2＋不能存在；至於是否含有Na＋，需要進一步檢驗。

11．下列粒子在相應條件下可大量共存的是( )

答案　C

解析　A項，Fe3＋與AlO發生雙水解反應，生成Fe(OH)3和Al(OH)3沉澱；B項，條件實質是溶液呈中

性，Fe2＋、Al3＋在中性條件不能大量共存；C項，附加條件是澄清透明、酸性溶液，澄清透明是溶液的特徵，不能把澄清透明誤認為無色溶液，C項正確；D項，Ca2＋、Cu2＋與SiO都不能大量共存。

12．下列指定反應的離子方程式正確的是( )

答案　A

解析　B項得失電子不守恆，應該為3Ag＋NO＋4H＋===3Ag＋＋NO↑＋2H2O，錯誤；氨水為弱電解質，不能拆成離子形式，C項錯誤；忽略了Fe2＋與H2O2發生的氧化還原反應，D項錯誤。

13．下列反應的離子方程式書寫正確的是( )

答案　B

題組四　常見無機物之間轉化關係的綜合判斷

A．①②④ B．②③

C．①④⑤ D．③④

答案　D

解析　①Fe與O2反應可生成Fe2O3或Fe3O4；②若NaOH不足，只有HCO參與反應，生成Na2CO3和(NH4)2CO3，若NaOH過量，生成Na2CO3和NH3·H2O；⑤當碳少量時生成Si和CO，當碳足量時生成SiC和CO。

2．一定條件下，中學化學常見物質甲、乙之間存在如下轉化關係，則乙可能是( )

A．Al(OH)3 B．H2SiO3

C．FeCl2 D．CH3Cl

答案　C

解析　氫氧化鋁可由氯化鋁與氨水反應生成或由偏鋁酸鈉和氯化鋁反應生成，但兩者均不是化合反應，A項錯誤；硅酸可由硅酸鈉與鹽酸反應得到，但不能通過化合反應獲得，B項錯誤；氯化鐵與鐵單質發生化合反應可以生成氯化亞鐵，屬於化合反應，C項正確；甲烷與氯氣發生取代反應可生成CH3Cl，但不屬於化合反應，D項錯誤。

3．已知A、B、C、D為短週期元素構成的四種物質，它們有如下轉化關係，且D為強電解質(其他相關物質可能省略)。

下列說法不正確的是( )

A．若A為非金屬單質，則它與Mg反應的產物中陰、陽離子個數比可能為2∶3

B．若A為非金屬單質，則其組成元素在週期表中的位置可能處於第二週期第ⅣA族

C．不論A為單質還是化合物，D都有可能是同一種物質，該物質的濃溶液在常溫下都能使鐵和鋁發生鈍化

D．若A是共價化合物，0.1molA分子中含有的電子數可能為NA(NA為阿伏加德羅常數的值)

答案　B

解析　A項，當轉化關係為N2―→NO―→NO2―→HNO3時，A為N2，與Mg反應生成的Mg3N2中陰、陽離子個數比為2∶3，正確；B項，若A為碳，則對應的D為H2CO3，屬於弱電解質，錯誤；當轉化關係為H2S(或S)―→SO2―→SO3―→H2SO4時，C項正確；當轉化關係為NH3―→NO―→NO2―→HNO3時，D項正確。

4．下列物質的轉化在給定條件下能實現的是( )

答案　B

7．A、B、C三種物質中均含有同一種元素M，它們之間有如圖所示的轉化關係。A為一種金屬單質，其產量直接關係國民經濟命脈。反應①②③均為化合反應，反應④為置換反應。下列有關說法不正確的是( )

A．①中所加試劑可能是氯氣

B．③中需要加具有氧化性的試劑

C．A與稀硝酸反應一定能實現元素M在①中發生的價態變化

D．元素M在②中發生的價態變化也能通過加碘化鉀溶液實現

答案　C

解析　根據轉化關係可確定A為變價金屬，應為Fe，反應①加入氯氣能實現，B為FeCl3，C為FeCl2，A項正確；反應③是鐵元素由＋2價升高到＋3價，需要加具有氧化性的試劑，B項正確；過量的鐵與稀硝酸反應時，生成硝酸亞鐵，C項錯誤；Fe3＋能將I－氧化成I2，自身被還原成Fe2＋，D項正確。

8．A是一種常見的單質，B、C為中學常見的化合物，A、B、C均含有元素X，它們有如圖所示的轉化關係(部分產物及反應條件已略去)。下列判斷正確的是( )

A．X元素可能為Al

B．反應①和②互為可逆反應

C．X元素可能為Si

D．反應①和②一定為氧化還原反應

答案　D

解析　從框圖可以看出元素X應該有變價，反應①和②都屬於氧化還原反應；Al和Si均無變價；反應①和②分別在鹼性和酸性條件下進行，不是可逆反應，元素X可能為氯、溴、碘等元素。

9．海洋中有豐富的礦產、能源和水產資源，如圖所示為利用海水的部分流程。

C．在③④⑤步驟中，溴元素均被氧化

的溶解度比較小，A正確；B項中石灰水與NaHCO3和Na2CO3反應都能產生碳酸鈣沉澱，不能用於鑑別兩種物質，B錯誤；C項中④步驟中SO2＋2H2O＋Br2===H2SO4＋2HBr，溴元素被還原，C錯誤；D項中工業上製取金屬鎂採用電解熔融MgCl2，不是電解飽和MgCl2溶液，D錯誤。

B．檢驗D溶液中金屬陽離子的反應：Fe3＋＋3SCN－===Fe(SCN)3↓
