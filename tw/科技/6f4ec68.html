面試現場如何實現鏈表的逆序？

2020-10-29
| 科技

前幾天一位小夥伴去面試，被要求現場寫如何實現鏈表的逆序？寫完一種問還有沒有其他方式？


今天咱們就來聊聊到底如何實現鏈表的逆序以及有哪些方法？（文中的鏈表是單鏈表）


給定一個帶頭節點的單鏈表，請將其逆序。如下圖所示，head為頭部。


由於單鏈表與數組不相同，單鏈表中每個節點的地址都存儲在其前驅節點的指針域中，因此，對單鏈表中任何一個結點的訪問只能從鏈表的頭指針開始進行遍歷。在對鏈表的操作過程中，需要特別注意在修改結點指針域的時候，記錄下後繼結點的地址，否則會丟失後繼結點。

方法一：就地逆序

序主要思路：在遍歷鏈表時，修改當前結點的指針域的指向，讓其指向它的前驅結點。為此，需要用一個指針變量來保存前驅結點的地址。此外，為了在調整當前結點指針域的指向後還能找到後繼結點，還需要另外一個指針變量來保存後繼結點的地址，在所有的結點都被保存好以後就可以直接完成指針的逆序了。除此之外，還需要特別注意對鏈表首尾結點的特殊處理。具體實現方式如下圖所示。


在上圖中，假設當前已經遍歷到cur 結點，由於它所有的前驅結點都已經完成了逆序操作，因此，只需要使

cur.next=pre

即可完成逆序操作。在此之前，為了能夠記錄當前結點的後繼結點的地址，需要用一個額外的指針next來保存後繼結點的信息，通過上圖（1）～（4）四步把實線的指針調整為虛線的指針就可以完成當前結點的逆序；當前結點完成逆序後，通過向後移動指針來對後續的結點用同樣的方法進行逆序操作。


算法實現

//節點信息public class Node {   public int data;   public Node next;}public class NodeReversal {   public static void main(String[] args) {       Node head = new Node();       head.next = null;       Node temp = null;       Node cur = head;       for (int i = 0; i < 8; i++) {           temp = new Node();           temp.data = i;           temp.next = null;           cur.next = temp;           cur = temp;      }       System.out.println("逆序之前");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }       reversal(head);       System.out.println();       System.out.println("逆序之後");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }  }   public static void reversal(Node head) {       //如果頭結點為空或者只有一個頭結點，就不用反轉了       if (head == null || head.next == null) {           return;      }       //前驅節點       Node pre = null;       //當前節點       Node cur = null;       //後繼節點       Node next = null;       //把鏈表首節點變為尾節點       cur = head.next;       next = cur.next;       cur.next = null;       pre = cur;       cur = next;       //使當前遍歷到的結點cur指向前驅節點       while (cur.next != null) {           next = cur.next;           cur.next = pre;           pre = cur;           cur = cur.next;           cur = next;      }       //節點最後一個結點指向倒數第二個節點       cur.next = pre;       //鏈表的頭及誒單指向原來鏈表的尾節點       head.next = cur;  }}

運行結果

逆序之前0 1 2 3 4 5 6 7逆序之後7 6 5 4 3 2 1 0


算法性能分析

以上這種方法只需要對鏈表進行一次遍歷，因此，時間複雜度為O（N）。其中，N為鏈表的長度。但是需要常數個額外的變量來保存當前結點的前驅結點與後繼結點，因此，空間複雜度為O（1）。

方法二：遞歸法

假定原鏈表為1→2→3→4→5→6→7，

遞歸法的主要思路為：先逆序除第一個結點以外的子鏈表

（將1→2→3→4→5→6→7變為1→7→6→5→4→3→2），

接著把結點1添加到逆序的子鏈表的後面

（1→2→3→4→5→6→7 變為7→6→5→4→3→2→1）。

同理，在逆序鏈表2→3→4→5→6→7時，

也是先逆序子鏈表3→4→5→6→7（逆序為2→7→6→5→4→3），

接著實現鏈表的整體逆序（2→7→6→5→4→3轉換為7→6→5→4→3→2）。

算法實現
public class NodeReversal {   public static void main(String[] args) {       Node head = new Node();       head.next = null;       Node temp = null;       Node cur = head;       for (int i = 0; i < 8; i++) {           temp = new Node();           temp.data = i;           temp.next = null;           cur.next = temp;           cur = temp;      }       System.out.println("逆序之前");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }       reverse(head);       System.out.println();       System.out.println("逆序之後");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }  }   public static Node recursionReversal(Node head) {       if (head == null || head.next == null) {           return head;      }       //反轉後面的節點       Node newHead = recursionReversal(head.next);       //把當前遍歷的節點加到後面節點逆序後鏈表的尾部       head.next.next = head;       head.next = null;       return newHead;  }   /**    * 方法功能：對待頭節點的單鏈表進行逆序    *    * @param head head鏈表頭節點    */   public static void reverse(Node head) {       if (head == null) {           return;      }       //獲取鏈表的第一個節點       Node firstNode = head.next;       //對鏈表進行逆序       Node newHead = recursionReversal(firstNode);       //頭節點指向逆序後鏈表的第一個節點       head.next = newHead;  }}

運行輸出

逆序之前0 1 2 3 4 5 6 7逆序之後7 6 5 4 3 2 1 0

算法性能分析

由於遞歸法也只需要對鏈表進行一次遍歷，因此，算法的時間複雜度也為O（N）。其中， N 為鏈表的長度。遞歸法的主要優點是：思路比較直觀，容易理解，而且也不需要保存前驅結點的地址；缺點是：算法實現的難度較大。此外，由於遞歸法需要不斷地調用自己，需要額外的壓棧與彈棧操作，因此，與方法一相比性能會有所下降。

方法三：插入法

插入法的主要思路：從鏈表的第二個結點開始，把遍歷到的結點插入到頭結點的後面，直到遍歷結束。假定原鏈表為

head→1→2→3→4→5→6→7

在遍歷到2時，將其插入到頭結點後，鏈表變為

 head→2→1→3→4→5→6→7

同理將後序遍歷到的所有結點都插入到頭結點head後，就可以實現鏈表的逆序。


算法實現

public class NodeReversal {   public static void main(String[] args) {       Node head = new Node();       head.next = null;       Node temp = null;       Node cur = head;       for (int i = 0; i < 8; i++) {           temp = new Node();           temp.data = i;           temp.next = null;           cur.next = temp;           cur = temp;      }       System.out.println("逆序之前");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }       reversal(head);       System.out.println();       System.out.println("逆序之後");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }  }   public static void reversal(Node head) {       //判斷鏈表是否為空       if (head == null || head.next == null) {           return;      }       //當前節點       Node cur = null;       //後排節點       Node next = null;       cur = head.next.next;       //設置鏈表第一個節點為尾節點       head.next.next = null;       //把遍歷到節點插入到頭節點的後面       while (cur != null) {           next = cur.next;           cur.next = head.next;           head.next = cur;           cur = next;      }  }}

算法性能分析

以上這種方法也只需要對單鏈表進行一次遍歷，因此，時間複雜度為O（N）。其中，N為鏈表的長度。與方法一相比，這種方法不需要保存前驅結點的地址，與方法二相比，這種方法不需要遞歸地調用，效率更高。

引申

①對不帶頭結點的單鏈表進行逆序；

②從尾到頭輸出鏈表。

分析

對不帶頭結點的單鏈表的逆序，讀者可以自己練習（方法二已經實現了遞歸的方法），這裡主要介紹單鏈表逆向輸出的方法。

方法一：就地逆序+順序輸出

首先對鏈表進行逆序，然後順序輸出逆序後的鏈表。這種方法的缺點是改變了鏈表原來的結構。

方法二：逆序+順序輸出

申請新的存儲空間，對鏈表進行逆序，然後順序輸出逆序後的鏈表。逆序的主要思路為：每當遍歷到一個結點的時候，申請一塊新的存儲空間來存儲這個結點的數據域，同時把新結點插入到新的鏈表的頭結點後。這種方法的缺點是需要申請額外的存儲空間。

方法三：遞歸輸出

遞歸輸出的主要思路：先輸出除當前結點外的後繼子鏈表，然後輸出當前結點，假如鏈表為1→2→3→4→5→6→7，那麼先輸出2→3→4→5→6→7，再輸出1。同理，對於鏈表2→3→4→5→6→7，也是先輸出3→4→5→6→7，接著輸出2，直到遍歷到鏈表的最後一個結點7的時候會輸出結點7，然後遞歸地輸出6，5，…，1。實現代碼如下：

public class NodeReversal {   public static void main(String[] args) {       Node head = new Node();       head.next = null;       Node temp = null;       Node cur = head;       //製造鏈表0->1->2->3->4->5->6->7       for (int i = 0; i < 8; i++) {           temp = new Node();           temp.data = i;           temp.next = null;           cur.next = temp;           cur = temp;      }       System.out.println("逆序之前");       for (cur = head.next; cur != null; cur =cur.next) {           System.out.print(cur.data + " ");      }       System.out.println();       System.out.println("逆序之後");              reversal(head.next);  }   public static void reversal(Node firstNode) {       if (firstNode == null) {           return;      }       reversal(firstNode.next);       System.out.print(firstNode.data + " ");  }}

輸出

逆序之前0 1 2 3 4 5 6 7逆序之後7 6 5 4 3 2 1 0

算法性能分析：以上這種方法只需要對鏈表進行一次遍歷，因此，時間複雜度為O（N）。其中，N為鏈表的長度。
