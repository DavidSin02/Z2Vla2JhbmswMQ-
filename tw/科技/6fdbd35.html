1．在下列變化過程中，既有離子鍵被破壞又有共價鍵被破壞的是( )

A．將SO2通入水中 B．燒鹼溶於水 C．將HCl通入水中 D．硫酸氫鈉溶於水

【答案】D

解析：A、二氧化硫通入水中，二氧化硫和水反應生成亞硫酸，二氧化硫和水中都沒有離子鍵，錯誤，不選A；B、燒鹼溶於水，是氫氧化鈉中的離子鍵被破壞，沒有共價鍵的破壞，錯誤，不選B；C、氯化氫分子中含有共價鍵，溶於水中，破壞共價鍵，錯誤，不選C；D、硫酸氫鈉電離出鈉離子和氫離子和硫酸根離子，破壞了離子鍵和共價鍵，正確，選D。

考點： 離子化合物的結構特徵和性質，共價鍵的形成及共價鍵的主要類型

2. 純水在20℃和80℃時的pH( 　)

A. 前者大 B. 後者大 C. 相等 D. 無法確定

【答案】A

【解析】試題分析：溫度越高，水的電離程度越大，c（H+）增大，pH減小。選A。

考點：水的電離平衡、pH

3．設NA為阿佛加德羅常數，下列說法正確的是（ ）

①標準狀況下，11.2L以任意比例混合的氮氣和氧氣所含的原子數為NA

②同溫同壓下，體積相同的氫氣和氬氣所含的分子數相等

③1L 2mol/L的氯化鎂溶液中含氯離子為4NA

④標準狀況下22.4LH2O中分子數為NA

⑤32g O2和O3混合氣體中含有原子數為2NA

A．①②③④ B．③④ C．①②③⑤ D．①③④

【答案】C

解析：根據氣體摩爾體積和阿伏加德羅定律可知，選項①②都是正確的；1L2mol/L的氯化鎂的物質的量是2mol，根據化學式MgCl2可知，含有4mol氯離子，③正確；標準狀況下，水不是氣體，不能適用於氣體摩爾體積，④不正確；氧氣和臭氧都是由氧原子構成的不同單質，含有的氧原子的物質的量是2mol，⑤正確，所以答案選C。

考點：考查阿伏加德羅常數的有關計算和判斷

點評：考查阿伏加德羅常數的應用的題目，為高考必考題目，這是由於它既考查了學生對物質的量、粒子數、質量、體積等與阿伏加德羅常數關係的理解，又可以涵蓋多角度的化學知識內容。要準確解答好這類題目，一是要掌握好以物質的量為中心的各化學量與阿伏加德羅常數的關係；二是要準確弄清分子、原子、原子核內質子中子及核外電子的構成關係。

4. 下列各式中屬於正確的水解反應離子方程式的是( 　)

A. CH3COOH + OH- CH3COO- +H2O B. S2-+2H2OH2S+2OH-

C. CH3COOH+H2OCH3COO- +H3O D. NH4++H2ONH３· H2O+H+

【答案】D

解析：A是酸鹼中和反應；B中硫離子的水解要分兩步進行；C是電離過程；故選D。

考點：水解反應

點評：定義：水與另一化合物反應，該化合物分解為兩部分，水中氫原子加到其中的一部分，而羥基加到另一部分，因而得到兩種或兩種以上新的化合物的反應過程。水解反應是中和或酯化反應的逆反應。大多數有機化合物的水解，僅用水是很難順利進行的。根據被水解物的性質水解劑可以用氫氧化鈉水溶液、稀酸或濃酸，有時還可用氫氧化鉀、氫氧化鈣、亞硫酸氫鈉等的水溶液。這就是所謂的加鹼水解和加酸水解。

鹽類水解反應的定義：在溶液中鹽電離出的離子與水電離出的氫離子和氫氧根結合生成弱電解質的反應。無機物在水中分解通常是複分解過程，水分子也被分解，和被水解的物質殘片結合形成新物質，如氯氣在水中分解，一個氯原子和一個水被分解的氫原子結合成鹽酸，水分子的另一個氫原子和氧原子與另一個氯原子結合成次氯酸；碳酸鈉水解會產生碳酸氫鈉和氫氧化鈉；氯化銨水解會產生鹽酸和氨水等。

5．等體積的AlCl3和NaOH兩種溶液混合後，沉澱物中含鋁元素的質量與溶液中含鋁元素的質量相等，則AlCl3和NaOH兩種溶液物質的量濃度之比是( )

①l: 3 ②2: 3 ③1: 4 ④2: 7

A．①和② B．①和③ C．②和④ D．③和④

【答案】C

解析：等體積的AlCl3和NaOH兩種溶液混合後，沉澱物中含鋁元素的質量與溶液中含鋁元素的質量相等，如溶液中的Al元素以Al3+存在，則根據反應方程式：AlCl3+3NaOH=Al(OH)3↓+3NaCl可知反應的AlCl3與NaOH物質的量的比是1:3，還有1份AlCl3未參加反應，故AlCl3和NaOH兩種溶液物質的量的比是2:3，由於溶液的體積相等，所以二者的物質的量濃度的比是2:3；若溶液中的Al元素以AlO2-存在，根據AlCl3+3NaOH=Al(OH)3↓+3NaCl，Al(OH)3+ NaOH=NaAlO2+ 2H2O，總反應方程式是：AlCl3+4NaOH= NaCl+NaAlO2+2H2O，產生1mol NaAlO2需要消耗1mol AlCl3和4molNaOH，產生1mol Al(OH)3需要消耗1mol AlCl3和3molNaOH，故AlCl3和NaOH兩種溶液物質的量的比是2:7，由於溶液的體積相等，因此二者的物質的量濃度的比等於2:7。故②和④正確，選項C符合題意。

考點：考查AlCl3和NaOH兩種溶液混合反應的有關知識。

6. 在相同溫度下，等體積等物質的量濃度的4種稀溶液：①Na2SO4、②H2SO4、③NaHSO4、④Na2S中所含離子種類由多到少的順序是( 　)

A. ①＝④＞③＝② B. ④＝①＞③＞②

C. ①＞④＞③＞② D. ④＞①=③＞②

【答案】D

解析：①Na2SO4是強電解質，在水中完全電離成Na+、SO42－，包括水電離出的H+和OH－，一共4種離子；②H2SO4中含有SO42－、H+和OH－，一共3種離子；③NaHSO4中含有Na+、H+、SO42－、OH－，一共4種離子；④Na2S中含有Na+、S2－，以及水解產生的HS－，水電離出的H+和OH－，一共5種離子。所以①Na2SO4②H2SO4③NaHSO4④Na2S中含離子種類由多到少的順序是④＞①=③＞②，選D。

考點：離子種多少的判斷、鹽類水解

7．下列離子方程式書寫正確的是(　　)

A．過量的SO2通入NaOH溶液中：SO2＋2OH－=SO＋H2O

B．Fe(NO3)3溶液中加入過量的HI溶液：2Fe3＋＋2I－=2Fe2＋＋I2

C．NaNO2溶液中加入酸性KMnO4溶液：2MnO＋5NO＋6H＋=2Mn2＋＋5NO＋3H2O

D．NaHCO3溶液中加入過量的Ba(OH)2溶液：2HCO＋Ba2＋＋2OH－=BaCO3↓＋2H2O＋CO

【答案】C

【解析】過量的SO2與NaOH作用生成NaHSO3，A不符合反應的實際情況；在酸性條件下，NO的氧化性強於Fe3＋，方程式中應有NO；C選項電荷、質量、電子得失都守恆；NaHCO3與過量Ba(OH)2溶液反應的離子反應方程式為HCO＋OH－＋Ba2＋=BaCO3↓＋H2O。

8. 25 ℃時，水的電離達到平衡：H2OH++OH－；△H＞0，下列敘述正確的是( 　)

A. 向水中加入稀氨水，平衡逆向移動，c(OH－)降低

B. 向水中加入少量固體硫酸氫鈉，c(H+)增大，KW不變

C. 向水中加入少量固體CH3COONa，平衡逆向移動，c(H+)降低

D. 將水加熱，KW增大，pH不變

【答案】B

解析：向水中加入氨水，溶液由中性到鹼性，鹼對水的電離起抑制作用，所以平衡逆向移動，c（OH-）增大，故A錯誤；NaHSO4═Na++SO42-+H+，其中電離出的氫離子使c（H+）增大，但是溫度不變，KW不變，故B正確；CH3COONa中的醋酸根水解對水的電離起到促進作用，電離平衡右移，c（H+）降低，故C錯誤；因為水的電離是吸熱過程，所以溫度升高，水的離子積常數KW增大，則pH值減小，故D錯誤，所以本題選擇B。

考點：水的電離

9．將30mL0.5mol/LNaOH溶液加水稀釋到500mL，稀釋後溶液中NaOH的物質的量濃度為

A．0.3mol/L B．0.03 mol/L C．0.05 mol/L D．0.04 mol/L

【答案】B

解析：由於在稀釋過程中溶質不變，所以將30mL0.5mol/LNaOH溶液加水稀釋到500mL，稀釋後溶液中NaOH的物質的量濃度＝＝0.03mol/L，答案選B。

考點：考查氫氧化鈉溶液稀釋後濃度的有關計算

10. 下列能大量共存的離子組是( 　)

A. NH4+、Mg2+、OH－、Cl－ B. Ba2+、K+、SO42－、NO3－

C. Al3+、Cu2+、SO42－、Cl－ D. Na+、Al3+、Cl－、AlO2－

【答案】C

解析：A．NH4+、Mg2+與OH－均要發生複分解反應不能大量共存；B．Ba2+、SO42－生成沉澱不能大量共存；D．Al3+、AlO2－發生雙水解反應不能大量共存。

考點：離子共存

【名師點晴】離子共存題作為高考常規題，一方面考查學生對基本的離子間的反應掌握情況，另一方面考查了學生對已掌握知識的應用能力。離子共存近幾年出現的較難題型主要是附加條件型，“無色透明”、“酸性或鹼性”。特別注意題設條件：（1）“一定能大量共存（2）“一定不能大量共存” （3）“可能大量共存”等。解決這一類型的題目，首先要弄懂題幹中的隱藏條件，解題時要逐個條件分別考慮，可用排除法得出正確答案。離子不能大量共存的一般情況：能發生複分解反應的離子之間（包括生成氣體、難溶物、弱酸、弱鹼、水等弱電解質）；能發生雙水解反應的離子之間；能發生氧化還原反應的離子之間；能發生絡合反應的離子之間等。

因水解相互促進而不能大量共存的離子組合。

（1）Al3＋與HCO3-、CO32-、AlO2-、SiO32-、HS－、S2－、ClO－。

（2）Fe3＋與HCO3-、CO32-、AlO2-、SiO32-、ClO－。

（3）NH4+與SiO32-、AlO2-。

注意：NH4+與CH3COO－、HCO3-雖能發生水解相互促進的反應，但能大量共存。

11．下列有關實驗操作的敘述錯誤的是

A．過濾操作中，漏斗的尖端應接觸燒杯內壁

B．從滴瓶中取用試劑時，滴管的尖嘴可以接觸試管內壁

C．滴定接近終點時，滴定管的尖嘴可以接觸錐形瓶內壁

D．向容量瓶轉移液體時，導流用玻璃棒可以接觸容量瓶內壁

【答案】B

【解析】A．過濾操作中，漏斗的尖端應接觸燒杯內壁，使液體順利流下，正確；B．從滴瓶中取用試劑時，滴管的尖嘴不可以接觸試管內壁，否則容易造成試劑汙染，錯誤；C．滴定接近終點時，滴定管的尖嘴可以接觸錐形瓶內壁，可以使殘餘在滴定管的尖嘴的液體進入到錐形瓶，正確；D．向容量瓶轉移液體時，用玻璃棒引流可以接觸容量瓶內壁，正確。故選B。

考點：考查實驗基本操作

【名師點睛】化學是一門實驗性的學科，在進行化學學習的過程中常涉及物質的分離提純、氣體的製取、除雜、收集、尾氣處理、溶液的配製等操作。這就需要掌握物質的物理性質、化學性質、常見的分離混合物的方法、常見的儀器的名稱、使用、化學試劑的使用、分離方法及名稱、操作的先後順序等，這樣才可以得心應手，作出正確的分析與判斷。

12. 下列過程或現象與鹽類水解無關的是( 　)

A. 純鹼溶液去油汙 B. 鐵在潮溼的環境下生鏽

C. 加熱氯化鐵溶液顏色變深 D. 泡沫滅火器滅火

【答案】B

解析：A．純鹼水解可生成鹼液，可與油汙發生皂化反應而生成可溶於水的物質而達到去汙的目的；B．鐵在潮溼的環境下生鏽，是電化學腐蝕，與鹽類的水解無關；C．鹽類的水解是吸熱反應，加熱促進氯化鐵水解，使溶液顏色變深；D．泡沫滅火器主要成分是Al2（SO4）3和NaHCO3溶液，兩者發生雙水解反應：Al3+ ＋3HCO3-===3CO2↑ ＋Al（OH）3↓。

考點：鹽類水解的應用

13．下列指定反應的離子方程式正確的是

A．用FeCl3溶液腐蝕印刷電路板：Fe3+＋Cu=Fe2+＋Cu2+

B．用惰性電極電解氯化鈉溶液：4Cl－＋2H2O2Cl2↑＋2H2↑＋O2↑

C．用稀硝酸洗滌試管內壁的銀鏡： 3Ag＋4H+＋NO3-=3Ag+＋NO↑＋2H2O

D．NaHCO3溶液中與滴加少量Ca(OH)2溶液：CO32-＋Ca2+=CaCO3↓

【答案】C

解析：A．用FeCl3溶液腐蝕印刷電路板離子方程式為：2Fe3+＋Cu=2Fe2+＋Cu2+，故A錯誤；B．用惰性電極電解氯化鈉溶液：2Cl－＋2H2OCl2↑＋H2↑＋2OH-，故B錯誤；C．用稀硝酸洗滌試管內壁的銀鏡： 3Ag＋4H+＋NO3-=3Ag+＋NO↑＋2H2O，故C正確；D．NaHCO3溶液中與滴加少量Ca(OH)2溶液：2HCO3-＋Ca2+＋2OH-=CaCO3↓＋2H2O+ CO32-，故D錯誤。故選C。

考點：考查離子方程式書寫

14. 在蒸發皿中加熱蒸乾再灼燒下列物質的溶液，可以得到該物質固體的是( 　)

A. 氯化鋁 B. 碳酸氫鈉 C. 氯化亞鐵 D. 硫酸銅

【答案】D

解析：A．氯化鋁易水解生成氫氧化鋁和HCl，升高溫度HCl易揮發，蒸乾溶液得到的固體是氫氧化鋁，灼燒氫氧化鋁，氫氧化鋁分解生成氧化鋁，所以最終得到的固體是氧化鋁，故A不選；B．碳酸氫鈉受熱分解生成碳酸鈉、二氧化碳和水，故B不選；C．FeCl2都是揮發性酸鹽酸的鹽，所以能水解到底，生成氫氧化物Fe（OH）2，但是Fe（OH）2又很容易被氧化成Fe（OH）3，並受熱分解，所以最後的產物都是Fe2O3，故C不選；D．形成硫酸銅的酸是硫酸，硫酸屬於難揮發性酸，且硫酸銅較穩定，受熱不發生反應，所以加熱蒸乾灼燒硫酸銅溶液最後得到的仍然是硫酸銅，故D選；故選D。

考點：考查鹽類水解的應用

15．有15 g Na2O2、Na2O、NaOH的混合物與400 g質量分數為3．65%的鹽酸恰好反應，蒸乾溶液，最終得固體質量為　(　　)

A．23．4 g　　B．25．5 g　　C．31 g　　D．無法計算

【答案】A

解析： Na2O2、Na2O、NaOH的混合物與鹽酸反應所得溶液為氯化鈉溶液，根據氯元素守恆，鹽酸中的所有氯元素均進入氯化鈉中，即有n(NaCl)=n(HCl)。400 g質量分數為3．65%的鹽酸的物質的量是n(HCl)==0．4 mol，所以氯化鈉質量是m(NaCl)=0．4 mol×58．5 g·mol-1=23．4 g。

考點：金屬鈉的性質與應用

22. 判斷下列鹽溶液的酸鹼性，能發生水解的用離子方程式表示，不能發生水解的請寫上“不發生水解”字樣。

K2CO3溶液呈______________性，_______________________________________；

K2SO4溶液呈______________性，_______________________________________；

FeCl3溶液呈______________性，_______________________________________。

【答案】 (1). 鹼 (2). (3). 中 (4). 不發生水解 (5). 酸 (6).

解析：鹽類水解的實質是弱酸的陰離子或弱鹼的陽離子能夠與水電離的氫離子或者氫氧根離子結合生成弱電解質，從而使水的電離平衡發生移動，使鹽溶液顯酸鹼性。強酸弱鹼鹽顯酸性，強鹼弱酸鹽顯鹼性，強酸強鹼鹽不水解，顯中性。故K2CO3溶液呈鹼性，水解離子方程式為CO32－＋H2OHCO3－＋OH－；K2SO4溶液不發生水解，顯中性；FeCl3溶液顯酸性，水解離子方程式為Fe3+＋3H2OFe（OH）3＋3H+。

考點：鹽類水解

【名師點睛】鹽類水解的規律

（1）“誰弱誰水解，越弱越水解”。如酸性：HCN<CH3COOH，則相同條件下鹼性：NaCN>CH3COONa。

（2）強酸的酸式鹽只電離，不水解，溶液顯酸性。如NaHSO4在水溶液中：NaHSO4===Na＋＋H＋＋SO。

（3）弱酸的酸式鹽溶液的酸鹼性，取決於酸式酸根離子的電離程度和水解程度的相對大小。

①若電離程度小於水解程度，溶液呈鹼性。

如NaHCO3溶液中：HCO3-H＋＋CO32-（次要），HCO3-＋H2OH2CO3＋OH－（主要）。

②若電離程度大於水解程度，溶液顯酸性。

如NaHSO3溶液中：HSO3-H＋＋SO32-（主要），HSO3-＋H2OH2SO3＋OH－（次要）。

（4） 相同條件下的水解程度：正鹽>相應酸式鹽，如 CO32->HCO3-。

（5）相互促進水解的鹽>單獨水解的鹽>水解相互抑制的鹽。如NH4+的水解：（NH4）2CO3>（NH4）2SO4>（NH4）2Fe（SO4）2。

23．Ⅰ．某廢水中可能含有下列離子中的若干種：Fe3+、Al3+、Fe2+、Cu2+、Cl-、CO32-、NO3-、SO42-、SiO32-。現進行如下實驗：
