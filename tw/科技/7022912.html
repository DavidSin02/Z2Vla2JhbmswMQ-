什麼是操作系統，操作系統都有哪些？

2020-10-29
| 科技

對於什麼是操作系統可能很多人都沒認真考慮過，甚至可能都不知道自己其實用過很多種類的操作系統。今天我們就介紹介紹關於操作系統的基本知識。

首先我們先介紹一下具體什麼是操作系統。

什麼是操作系統

為了理解什麼是操作系統，我們將Linux操作系統的整體架構示意圖展示在這裡。從圖中可以看出，操作系統位於硬件之上，而處於應用程序之下。所以操作系統是介於應用軟件與硬件之間的一個軟件系統。

我們看一下維基百科給出的定義：

An operating system (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs.

大致翻譯一下就是，操作系統是用於管理計算機硬件、軟件資源，並對計算機程序提供通用服務的系統軟件。

下面我們從對操作系統的定義的不同的角度來介紹一下操作系統。

硬件管理

其實我們是無法直接對硬件進行操作的，對硬件的操作是需要軟件的。而操作系統就是用來管理硬件的軟件。常見的硬件，如CPU、內存、硬盤、網卡和鼠標鍵盤等都是需要操作系統的管理才能工作。而這就是我們所說的硬件管理。

軟件管理

軟件管理主要是對應用軟件的管理。其實我們最直接的認識就是應用軟件需要按照到操作系統上才可以運行。而操作系統對軟件的管理其實還要多得多，除了安裝刪除外，還有定時運行軟件和調度等等。

通用服務

通用服務是之操作系統提供給應用軟件的功能接口，也稱為API。例如最常見的提供文件訪問的接口，或者攝像頭訪問的接口等等。

操作系統的分類

操作系統的種類很多，從不同的角度進行劃分，則可以有不同的操作系統。為了便於大家理解，今天我們從支持設備的類型來分類來介紹一下常見的操作系統。如果從設備角度分類，操作系統分為桌面操作系統、服務器操作系統、手機操作系統和嵌入式操作系統等。

其中桌面操作系統應該是最常見的操作系統了。比如我們常用的Windows或者Mac OS就是桌面操作系統。

還有一類是運行在服務器上的操作系統，這類操作系統沒有美觀的界面，通常我們只能看到一個黑黑的窗口。如果大家看過《黑客帝國》類似的電影，可能對服務器上的操作系統有所瞭解。
