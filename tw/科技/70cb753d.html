8.5 傳遞任意數量的實參

有時候，你預先不知道函數需要接受多少個實參，好在Python允許函數從調用語句中收集任意數量的實參。

例如，來看一個製作比薩的函數，它需要接受很多配料，但你無法預先確定顧客要多少種配料。下面的函數只有一個形參*toppings，但不管調用語句提供了多少實參，這個形參都將它們統統收入囊中：

pizza.py

源文件1（測試正常）

#形參名*toppings中的星號讓Python創建一個名為toppings的空元組#並將收到的所有值都封裝到這個元組中def make_pizza(*toppings): """打印顧客點的所有配料""" print(toppings)make_pizza('pepperoni')make_pizza('mushrooms', 'green peppers', 'extra cheese')make_pizza('mushrooms2', 'green peppers3', 'extra cheese4')

形參名*toppings中的星號讓Python創建一個名為toppings的空元組，並將收到的所有值都封裝到這個元組中。函數體內的print語句通過生成輸出來證明Python能夠處理使用一個值調用函數的情形，也能處理使用三個值來調用函數的情形。它以類似的方式處理不同的調用，注意，Python將實參封裝到一個元組中，即便函數只收到一個值也如此：

('pepperoni',)('mushrooms', 'green peppers', 'extra cheese')('mushrooms2', 'green peppers3', 'extra cheese4')

現在，我們可以將這條print語句替換為一個循環，對配料列表進行遍歷，並對顧客點的比薩進行描述：

#形參名*toppings中的星號讓Python創建一個名為toppings的空元組#並將收到的所有值都封裝到這個元組中def make_pizza(*toppings): """概述要製作的比薩""" print("\n用下列配料做比薩餅:") for topping in toppings: print("- " + topping)make_pizza('pepperoni')make_pizza('mushrooms', 'green peppers', 'extra cheese')make_pizza('mushrooms34', 'green peppers44', 'extra cheese554')

不管收到的是一個值還是三個值，這個函數都能妥善地處理：

用下列配料做比薩餅:- pepperoni用下列配料做比薩餅:- mushrooms- green peppers- extra cheese用下列配料做比薩餅:- mushrooms34- green peppers44- extra cheese554

不管函數收到的實參是多少個，這種語法都管用。

8.5.1 結合使用位置實參和任意數量實參

如果要讓函數接受不同類型的實參，必須在函數定義中將接納任意數量實參的形參放在最後。 Python先匹配位置實參和關鍵字實參，再將餘下的實參都收集到最後一個形參中。

例如，如果前面的函數還需要一個表示比薩尺寸的實參，必須將該形參放在形參*toppings的前面：

代碼2（測試正常）

#在任意數量形參前面加一個size位置形參def make_pizza(size,*toppings): """概述要製作的比薩""" print("\n製作一個"+ str(size) +"尺寸的薩與下列配料:") #str() 可以將數字型變量或常量改變成字符型變量或常量 這樣才不出錯 for topping in toppings: print("- " + topping)make_pizza(15,'pepperoni')make_pizza(20,'mushrooms', 'green peppers', 'extra cheese')make_pizza(35,'mushrooms34', 'green peppers44', 'extra cheese554')

基於上述函數定義， Python將收到的第一個值存儲在形參size中，並將其他的所有值都存儲在元組toppings中。在函數調用中，首先指定表示比薩尺寸的實參，然後根據需要指定任意數量的配料。

現在，每個比薩都有了尺寸和一系列配料，這些信息按正確的順序打印出來了——首先是尺寸，然後是配料：

製作一個15尺寸的薩與下列配料:- pepperoni製作一個20尺寸的薩與下列配料:- mushrooms- green peppers- extra cheese製作一個35尺寸的薩與下列配料:- mushrooms34- green peppers44- extra cheese554

8.5.2 使用任意數量的關鍵字實參

有時候，需要接受任意數量的實參，但預先不知道傳遞給函數的會是什麼樣的信息。在這種情況下，可將函數編寫成能夠接受任意數量的鍵—值對——調用語句提供了多少就接受多少。一個這樣的示例是創建用戶簡介：你知道你將收到有關用戶的信息，但不確定會是什麼樣的信息。

在下面的示例中，函數build_profile()接受名和姓，同時還接受任意數量的關鍵字實參：

user_profile.py

#創建一個函數 裡面包含名和姓 形參 ，和任意數量形參def build_profile(first, last, **user_info): """創建一個字典，其中包含我們知道的有關用戶的一切""" #創建了一個名為profile的空字典 profile = {} #我們吧名和姓加入字典中 profile['名'] = first profile['姓'] = last  #我們遍歷字典user_info中的鍵—值對，並將每個鍵—值對都加入到字典profile中 for key, value in user_info.items():	 profile[key] = value return profile  user_profile = build_profile('艾伯特', '愛因斯坦',位置='普林斯頓',領域='物理學')print(user_profile)

函數build_profile()的定義要求提供名和姓，同時允許用戶根據需要提供任意數量的名稱—值對。形參**user_info中的兩個星號讓Python創建一個名為user_info的空字典，並將收到的所有名稱—值對都封裝到這個字典中。在這個函數中，可以像訪問其他字典那樣訪問user_info中的名稱—值對。

在build_profile()的函數體內，我們創建了一個名為profile的空字典，用於存儲用戶簡介。我們將名和姓加入到這個字典中，因為我們總是會從用戶那裡收到這兩項信息。我們遍歷字典user_info中的鍵—值對，並將每個鍵—值對都加入到字典profile中。最後，我們將字典profile返回給函數調用行。

我們調用build_profile()，向它傳遞名（'albert'）、姓（'einstein'）和兩個鍵—值對（location='princeton'和field='physics'），並將返回的profile存儲在變量user_profile中，再打印這個變量：

{'名': '艾伯特', '姓': '愛因斯坦', '位置': '普林斯頓', '領域': '物理學'}

在這裡，返回的字典包含用戶的名和姓，還有求學的地方和所學專業。調用這個函數時，不管額外提供了多少個鍵—值對，它都能正確地處理。編寫函數時，你可以以各種方式混合使用位置實參、關鍵字實參和任意數量的實參。知道這些實參類型大有裨益，因為閱讀別人編寫的代碼時經常會見到它們。要正確地使用這些類型的實參並知道它們的使用時機，需要經過一定的練習。就目前而言，牢記使用最簡單的方法來完成任務就好了。你繼續往下閱讀，就會知道在各種情況下哪種方法的效率是最高的。
