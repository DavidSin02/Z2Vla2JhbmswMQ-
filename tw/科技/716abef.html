PS1迷你主機遊戲棒 - PlayStation遊戲庫無限擴容神器實測！

2020-10-29
| 科技

1994年12月，日本發售的PlayStation初代主機，正式吹響了索尼進軍遊戲業界的號角。說起PlayStation，其實有一段頗為有趣的歷史呢：

最初，是任天堂請求索尼為其家用遊戲機開發一款CD-ROM光盤遊戲的附件，這個附件暫定名為Play Station或SNES-CD。為此兩家當時還成立了一家合資企業。然而在簽訂合作協議之後，索尼決定自行開發一款不僅能玩超任遊戲還能播放CD光盤的娛樂設備，說幹就幹，索尼1991年5月在CES上正式公佈了這款娛樂設備——Play Station！

時任任天堂社長的山內溥在索尼公佈Play Station原型機之後，意識到任天堂和索尼的合作協議實際上賦予了索尼對所有超任主機遊戲的所有權和控制權。於是，山內溥果斷取消SNES-CD的開發計劃，並中止了和索尼的合作。雙方中止合作之後，任天堂順利的找到了飛利浦來開發CD附件；而另一邊索尼邀請世嘉開發主機，最終被世嘉斷然拒絕，理由是索尼根本不懂如何製造遊戲機和遊戲。

把我忽悠來，卻不帶我玩了？！索尼一怒之下，決定自己幹了。推翻了超任主機的設計，完全刪除卡槽，採用3D多邊形圖形開發。最終PlayStation成功發售。自此索尼一發不可收拾，苦心耕耘，並在接下來的20多年，將PlayStation打造成為全世界最受歡迎的主機系列之一！

PlayStation系列的開山鼻祖！

而今，索尼將這款經典的PlayStation初代主機Mini化重置，命名為PlayStation Classic，年輕一代的玩家現在也能感到到原汁原味的PS1遊戲了。雖然外觀一樣，但是中那個機器蓋是不能打開的，機器內置了20款數字版的經典遊戲，包括老玩家耳熟能詳的《最終幻想7》、《山脊4》、《貼鐵拳3》……

MINI版比原版縮小了45%

PlayStation Classic主機採用了1GB RAM，比老任家Mini FC和Mini SFC的256MB大得多，但是遊戲卻僅僅提供了20款，這讓很多玩家望而卻步了，相對於700多的售價來言，20款遊戲有些誠意不足啊。

動手能力強的小夥伴可能已經自行研究灌遊戲的方法了，不過這種方法我們就不推薦了。畢竟不是正常手段嘛，而且我真的看不懂那些教程啊，看起來就很複雜。萬一再把主機折騰壞了呢？

是時候考慮下 TrueBlue MINI了。一款適用於PlayStation Classic主機的遊戲棒。簡單來說，TrueBlue MINI是一款內置PS1遊戲的U盤，只需要將其接入到PlayStation Classic主機上就能直接玩到多達101款經典PS1遊戲。重要的是，U盤內置系統，不需要任何操作，也不會對主機核心做任何修改。真正的安全綠色無汙染呢！

加上Hub之後，雙人遊戲也不受影響

這遊戲陣容，比起原版的20個可強太多了

TrueBlue MINI一共有三款，True Blue Mini - Fight Pack ( 32 GB-內置了64款經典遊戲 ) / True Blue Mini - Meth Pack ( 64 GB-內置了101款經典遊戲 ) / True Blue Mini - Crackhead Pack ( 64 GB-內置了101款經典遊戲 )，這其中兩款64G收錄的101款遊戲是完全不同的，可以根據自己的喜好來挑選。（遊戲列表可以在官網查看，點擊進入）

打開包裝之後，就能看到TrueBlue的真容了！

基於U盤打造的藍色TrueBlue，此外包裝中還附送了一個4口HUB線（底部還有一個）。

單人遊戲的話，安裝非常簡單，TrueBlue遊戲棒直接插2P口，然後開機玩就行。

如果需要雙人遊戲，也很方便，需要將Hub接入2P的USB接口，再將TrueBlue遊戲棒和2P插上Hub口上，然後開機玩就支持雙人了。

總結：TrueBlue遊戲棒，一款完美適配索尼PS1經典迷你主機的配件。即插即用，安全簡便，內置遊戲多。可以說是PlayStation情懷的完美護航者了。
