關於npm install升級失敗的問題

2020-11-14
| 科技

由於前一段時間，我們對客戶的ecshop4.0手機端進行二次開發使用到了vue打包命令，當初使用升級命令“npm install”還可以對npm進行升級了

但今天在開始使用這個命令的時候，發現已經不能使用了，每次升級都提示“Cannot download "https://github.com/sass/node-sass/releases/download/v3.13.1/win32-x64-57_binding.node":”。網上打開對應的鏈接，資源也不存在了。所以網上有的教程說，在瀏覽器中輸入對應的資源下載鏈接，把資源下載下來，然後放到對應的目錄下，修改資源路徑的方法已經不可能使用了。

我們是使用下面這種方法進行升級的：

使用淘寶鏡像

npm install -g cnpm --registry=https://registry.npm.taobao.org cnpm install node-sass（我是使用這種方法升級成功的。）

OR

npm set sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

第二種方法至於能不能用，還需要你們進行測試看看了

在此記錄一下解決方法，以方便遇到同樣問題的小夥伴們解決問題
