Excel中列出當前文件夾下所有文件目錄的2種方法，簡單高效！

2020-11-14
| 科技

前言：

Excel軟件中列出當前文件夾下所有文件目錄的2種方法，簡單高效！

廢話少說，先看要實現的最終效果演示：（列出當前目錄下所有文件名，並設置超鏈接，點擊後直接打開對應文件！）

列出指定文件夾下所有文件目錄

利用超鏈接函數法：

  • 將文件夾地址欄的絕對地址複製到瀏覽器（此處用的是chrome瀏覽器）地址欄中打開；

  • 將瀏覽器頁面得到的信息複製粘貼到工作表中；

藉助瀏覽器得到文件信息

  • 在B2單元格內寫入以下公式：

  • =HYPERLINK(LEFT(CELL("filename"),FIND("[",CELL("filename"))-1)&A2,A2)

編寫函數

利用VBA代碼方法實現：

1、在開發工具Visual Basic 中，插入模塊，寫入以下代碼：

Sub 建立文件目錄()

Set fso = CreateObject("scripting.filesystemobject")

Set 文件夾 = fso.getfolder(ThisWorkbook.Path)

'可以修改路徑

a = 2

For Each f In 文件夾.Files '遍歷指定路徑下所有文件

Cells(a, 2).Select

ActiveSheet.Hyperlinks.Add Anchor:=Selection, Address:= _

ThisWorkbook.Path & "\" & f.Name, TextToDisplay:=f.Name

a = a + 1

Next f

End Sub

插入模塊，寫入代碼

2、運行宏名：建立文件目錄，大功告成！
