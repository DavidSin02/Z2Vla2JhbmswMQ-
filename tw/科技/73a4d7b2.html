全國青少年機器人技術等級考試三四級知識點概要（精髓）

一、電子電路知識點

摩擦起電：用摩擦的方法使物體帶電。

強→得到電子→帶負電

摩擦起電 兩個物體摩擦時，哪個物體原子核束縛電子的本領

弱→失去電子→帶正電

摩擦起電並不是創造電荷，只是電荷從一個物體轉移到另一個物體，使正負電荷分開。（負電荷發生轉移）

正電荷：絲綢摩擦玻璃棒所帶的電荷

兩種電荷

電荷 負電荷：毛皮摩擦橡膠棒所帶的電荷

電荷量（電荷）表示：字母 Q → 單位 庫侖→ 簡稱 庫→ 符號 C。

電荷間的相互作用：同種電荷相互排斥，異種電荷相互吸引。

分子：一個原子或多個原子組成。

物質結構 質子（帶正電）

原子核（帶正電）

原子（不帶電） 中子（不帶電）

電子（帶負電）

導體：容易導電的物體（金屬、人體、大地、石墨、食鹽水）→金屬→自由電子導電

導體絕緣體

絕緣體：不容易導電的物體（橡膠、玻璃、塑料）→摩擦起電

電源：提供電能的裝置。

用電器（負載）：消耗電能的裝置。

電路組成作用 導線（電線）：將電路各部分連接起來形成迴路。

開關：控制整個電路連接還是中斷。

電路

通路：有電流通過。

電路三種狀態 斷路（開路）：電路中沒有電流。

短路：電源兩極直接連接起來。（不允許）

用電器被短接：用電器不會正常工作。

電流的形成：電荷的定向移動形成電流。

1.自由移動的電荷（金屬中自由電子）

電流產生條件 2.導體兩端存在電壓差→壓力→電源提供

3.電路中必須有閉合迴路

電源外部：電源正極→用電器→電源負極

電流 電流方向：正電荷移動的方向

電源內部：電源負極→電源正極

表示：字母 I→ 單位 安培→ 簡稱 安→ 符號 A

電流強弱（電流）

單位時間內通過導體橫截面的電荷量。I=Q/t。

電流測量：電流表，串聯在電路中

電壓提供：電源。電源的作用就是給用電器兩端提供電壓。

電壓與電流關係：要想電路中有電流，它的兩端就必須有電壓。

表示：字母 U→ 單位 伏特→ 簡稱 伏→ 符號 v

電壓

電壓測量：電壓表，並聯在被測物體兩端。

家庭電路 220v

乾電池 1.5v

常見電壓 電腦輸電壓出電壓 5v

人體安全電壓 36v

動力電路 380v

低壓 1000v以下 高壓 1000v以上

什麼是電阻：導體對電流阻礙的大小用電阻來表示，電阻越大對電流的阻礙越大。

表示：字母 R→ 單位 歐姆→ 簡稱 歐→ 符號Ω

導體的電阻是導體本身的一種性質，它的大小與導體的材料、長度和橫截面積有關。

電阻 影響電阻大小因素 材料：一般情況，材料不同電阻不同

因素 長度：同種材料，橫截面一定，長度越長電阻越大。

橫截面：同種材料，長度一定，橫截面積越大電阻越小。

定值電阻：實驗室常用的定值電阻是色環電阻（五色環）

分類 變阻器（電位器）：常用滑動變阻器，通過改變導體長度來改變電阻的大小。

特殊電阻：光敏、熱敏、壓敏。

定義：串聯就是把電路中的各個元件用導線依次連接起來

1.只有一個迴路，電流只有一條路徑。

串聯電路 2.電流處處相等。

（分壓） 3.總電壓等於各元件兩端電壓之和。

特點 4.總電阻等於各部分電阻之和。

5.各元件電壓和電阻成正比。

6.開關在任何位置都能控制整個電路。

7.電路一處斷開，所有元件都不能工作。

定義：並聯就是把電路中的各電子元件的兩端分別連接在一起

並聯電路 1.多個迴路。

（分流） 2.幹路總電流等於各支路電流之和。

3.各支路兩端電壓相等。

特點 4.總電阻小於各支路電阻，總電阻的倒數等於各支路電阻的倒數之和。

5.各支路電流與電阻成反比。電阻越大流過電流越小。

6.幹路開關控制整個電路，支路開關控制與其串聯部分。

7.支路一處斷開，其他支路正常，幹路一處斷開，則整個電路斷開。

歐姆定律：導體中的電流，跟導體兩端的電壓成正比，跟導體的電阻成反比。

歐姆定律 公式：I=U/R

電阻的測量：伏安法測電阻。電阻的大小可以通過電壓和電流來計算，但不是影響電阻的要素。電阻的大小是導體本身一種性質

半導體：導電能力介於導體和絕緣體之間

P型半導體（加入空穴帶正電）

PN結 合在一起中間形成PN結→內電場

N型半導體（加入電子帶負電）

正向導通

二極管 特性：單向導電性

反向截止

半導體二極管：把PN結兩端引出兩個電極封裝起來就形成了半導體二極管。

1、發光二極管

2、整流二極管

應用 3、開關二極管

4、檢波二極管

5、限幅二極管

6、穩壓二極管

1.半導體三極管（雙極型晶體管，晶體三極管）

2.半導體基本元件之一

NPN

三極管 3.兩個PN結

PNP

集電極、基極、發射極

4.結構 集電區、基區、發射區

集電結、發射結

5.應用：放大作用，開關作用（控制電路通斷）

1.容納電荷的元器件

2.表示：符號C→ 單位 法拉→ 簡稱 法→ 符號F

電容 3.通常體積大的電容量也大

4.分類：獨石電容，電解電容

濾波（通交流，阻直流）電容對於交流相當於導線，對於直流相當於斷開。

1.模擬量：在時間和數值上都連續的物理量。表示模擬量的信號就是模擬信號。

處理模擬信號的電路稱之模擬電路。

2.數字量：時間離散，數值也離散的信號就是數字信號→數字電路（處理數字信號電路）

3.在數字電路中，可以用0和1組成的二進制代表數量的大小，也可以用0和1表示兩

電子電路領域相關理論和相關人物

1.希臘哲學家泰勒斯 摩擦琥珀吸引羽毛 磁石吸引鐵片 這兩種物體對其他物體有吸引力

2.英國物理學家吉爾伯特 摩擦琥珀吸引羽毛的現象不僅存在於琥珀上，而且存在於硫磺、毛皮、陶瓷、紙、橡膠等摩擦起電的物質。證明電是地地道道的物質。

3.美國物理學家富蘭克林 發現電荷分為"正""負"之分，而且兩者的數量是守恆的。風箏實驗，證明雷電是由電力造成的

4.意大利帕維亞大學教授伏特 製造了起電盤，靜電計等儀器。 發明了伏打電堆。電動勢取名為伏特。

5.英國物理學家法拉第 發現電磁感應現象 發明了圓盤發電機 引入了電場和磁場的概念 冰桶實驗 證明了電荷守恆定律

6.德國物理學家歐姆 發現了電流公式 也叫歐姆定律

二、編程知識點

1.程序：程序就是為了實現特定目標或解決特定問題而用計算機語言編寫的一系列指令序列。

2.計算機語言：人與計算機之間交流的語言。

↓ 機器語言（0,1）→計算機能直接識別

發展 符號語言（彙編語言，低級語言） 編譯程序軟件

高級語言（C語言，C++） 源程序 目標程序

3.C語言用途：應用軟件、系統軟件、大型遊戲。

1.語言簡潔、緊湊，使用方便、靈活。

2.運算符豐富

3.數據類型豐富

4.具有結構化的控制語句。（if else while）

4.C語言特點 5.語法限制不太嚴謹，程序設計自由度大。

6.C語言允許直接訪問物理地址，能進行位（bit）操作。

7.用C語言編寫的程序可移植性好。

8.生成目標代碼質量高，程序執行效率高。

5.C語言程序舉例

#include<stdio.h> // #開頭預處理命令 包含一個標準的輸入輸出頭文件

int main() // 定義主函數

{ // 函數開始

printf("This is a C program.\n"); // 輸出一行信息

rerurn 0; // 函數執行完畢時返回函數值0

} // 函數結束

1.一個程序由一個或多個源程序文件組成。

2.函數是C程序的主要組成部分。

3.一個函數包括兩部分。函數首部和函數體。

4.函數總是從main函數開始執行的，不倫main函數在程序的位置如何。

6.C結構特點 5.程序中要求計算機完成的操作是由函數中的C語句完成的。書寫格式比較自由。一行內可以寫幾個語句，一個語句可以分寫在多行上，但為了清晰起見，習慣每行只寫一個語句。

6.在每個數據聲明和語句的最後必須有一個分號。

7.C語言本身不提供輸入輸出語句。

8.程序應當包含註釋。（//單行註釋 /*多行註釋*/）
