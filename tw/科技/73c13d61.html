QT系列之使用QT編輯多窗口程序

2020-10-29
| 科技

QT的功能是非常強大的，在很多桌面應用程序都有用到。這裡主要給大家介紹的是如何使用QT實現多窗口跳轉。所謂的多窗口程序簡單的理解一下，我們日常用的QQ，微信都有一個登錄界面，點擊登錄之後就會進行跳轉。今天主要給大家介紹的是如何使用QT進行多窗口程序，像密碼驗證，登錄界面的設計暫時先不介紹，有興趣的朋友歡迎進行留言，可以單獨進行分析普及。

程序實現功能：點擊運行會出現一個對話框，就像QQ的登錄窗口，按下登錄按鈕後登錄對話框退出，出現主窗口 ，如何點擊退出按鈕則不能進入主窗口，整個程序就會退出。在主窗口中，點擊對話框按鈕，會進入另一個對話框，點擊退出按鈕返回主窗口。

大致整個程序需要實現的功能就是上訴所描述的，下面我就實現的整個流程，進行操作演示。有不對的地方，希望各位志同道合的朋友給予批評指正，當然也更加歡迎大家一起討論學習。想要了解更多內容，歡迎點擊由上角的關注，更多精彩內容與您共享。

本次使用的操作環境：windaws10+QTcreator5.9.4

一、 新建主窗口

1. 首先打開QTcraztor,我這裡使用的是5.9.4版本，當然基本QT5系列都通用，不過每個版本穩定性可能有些差異，樓主使用5.9.4版本感覺還挺穩定的,新建Qt WidgetsApplication，項目名稱設置為windows，在類信息界面保持基類為QMainWindow、類名MainWindow，這此時一個主窗口界面就已經新建完成了。

ui界面

2. 主窗口新建完成後，點擊Forms文件下的mainwindows.ui，進入ui界面設計模式。在左側控件板塊中，選擇一個push button的控件，將其直接拖拽到主窗口界面，選擇合適的位置放置。雙擊此控件，進入名稱編輯模式，更改控件名稱為按鈕。MainWindow::MainWindow(QWidget *parent) :

二、 設置主窗口顯示文本

1. 使用Ctrl+2快捷鍵進入編輯模式，選擇mainwindows.cpp進入進行編輯，在maiinwindows構造函數裡面加入代碼，如下圖所示。

代碼塊

 MainWindow::MainWindow(QWidget *parent) : QMainWindow(parent), ui(new Ui::MainWindow){ ui->setupUi(this); ui->pushButton->setText("新窗口");}

這裡的ui對象就是界面文件對應的類的實例，在mainwindow.h文件中對其進行了定義，我們可以通過它來訪問設計模式添加到界面上的部件。前面添加的按鈕部件Push Button，在其屬性面板上可以看到它的objectName屬性的默認值為pushButton，這裡就是通過這個屬性來獲取部件對象的。

在這個地方，使用Push Button這個類的setText函數來實現顯示文本的功能點擊Ctrl+R按鈕運行程序，可以看見一下效果。觀察發現，通過程序直接編輯的顯示文本會覆蓋之前ui設計的文本。這是需要注意的一個地方。

主窗口

三、 添加登錄對話框（重點）

1. 首先添加一個新的對話框，需要添加新文件，在項目目錄下的Windows下右擊鼠標，選擇添加新文件

2. 模板選擇文件和類分類下的QT選項，選擇QT設計師界面類，點擊choose進入下一步。

3. 界面模板選擇Dialog withoutButtons，單擊下一步

4. 此時進入類的信息界面，這裡我們將類名修改為Login_interface，一般類名的首字母需要大寫，此時相關文件會自動進行修改。

5. 點擊完成 按鈕後會跳轉到登錄窗口的ui界面設計模式，這裡我們選擇一個push button控件，拖拽到登錄界面，選擇合適的位置，雙擊控件進入名稱編輯模式，修改名稱為進入主界面。我們的目的是為了實現點擊此按鈕，會關閉此對話款，並且進入顯示主窗口。這裡運用了QT的核心知識信號與槽的概念。這裡只演示，不做過多介紹，需要的朋友給以在後續的介紹中瞭解。

首先點擊設計模式上方的 按鈕，進入信號和槽的編輯模式，按著鼠標左鍵，從按鈕上拖向界面，如下圖所示。

6.進入配置連接中，在pushButton中選擇clicked選項,在login_interface中，選擇accept選項，完成後點擊ok按鈕。

設置好了信號與槽的關聯後，顯示界面如下圖。

6. 完成後，按F3鍵返回控件編輯模式

四、 設置自定義對話框類

1. 按下Ctrl+2返回代碼編輯模式，在這裡打開main.cpp文件，添加代碼：如下圖所示

此時即可測試一下程序。

#include "login_interface.h"int main(int argc, char *argv[]){ QApplication a(argc, argv); MainWindow w; Login_interface dlg;//建立Login_interface的實例dlg if(dlg.exec() == QDialog::Accepted)//利用Accepted返回值判斷按鈕是否被按下 { w.show();//如果按下，顯示主窗口 return a.exec();//程序一直執行，直到主程序關閉 } else return 0;//如果沒有按下，則不會繼續如主窗口，程序結束}

2. 上述方案以代碼形式實現，通過UI界面也可實現顯示對話框。

首先進入Forms文件下的mainwindow.ui界面，右擊按鈕這個控件在彈出的轉到槽對話框中選擇第一個clicked()信號並按下確定按鈕。這時會跳轉到編輯模式mainwindow.cpp文件的on_pushButton_clicked()函數處，這個就是自動生成的槽，已經自動在mainwindow.h文件中進行了聲明，這裡需要在windows.cpp下添加兩句代碼，代碼如下

void MainWindow::on_pushButton_clicked(){ QDialog *dlg = new QDialog(this); dlg ->show();}

需要注意，在cpp下需要添加頭文件，到此，整個過程就已經結束了，大家可以點擊運行觀察效果。
