懶豬編程實例六：Visual C# 實現外部文件的讀取和寫入

2020-10-29
| 科技

在Visual C#的環境下，用戶在打開外部文件時，有時需要讀取外部文件信息中的內容，有時也需要將特定的信息內容寫入到外部文件當中，那麼如何來實現這些功能呢？下面我們來看一下外部文件的讀取和寫入方法的實現

讀取外部文件信息

要想讀取外部文件信息內容，或者是以只讀方式打開文件，它是利用File類的OpenRead方法來實現的，我們先來看一下以只讀方式打開文件的例子，仍然使用我們上一期製作好的窗體界面，通過"文件"菜單下的"打開"子菜單項，來實現文件的讀取操作，並將信息顯示到窗體界面中的文本框內

雙擊"文件"菜單下的"打開"子菜單，這樣我們可以輕鬆的進入到代碼編輯窗口，進入代碼窗口後，寫入下列代碼，這些代碼就是實現讀取已有文件的信息內容的

下面對這些代碼進行解釋：

FileStream fs = File.OpenRead(openFileDialog1.FileName)：以只讀方式打開文件並實例化對象

using (StreamReader sr = File.OpenText(openFileDialog1.FileName))：創建StreamReader對象

string s = ""：定義變量s，並賦初始值為空，作用是存放每次循環讀取的行數據

string t = ""：定義變量t，並賦初始值為空，存放所有讀取的行數據，並以一定格式存放

while ((s =sr.ReadLine ())!=null)：當讀取文件內容不為空時，則繼續循環

t = t + s + "\r\n"：變量t以一定格式存放文件內容，這裡面"\r\n"為轉義字符，表示換行存儲

textBox1.Text = t：將變量t存放的字符顯示在textBox1文本框控件中

運行程序後，在彈出的對話框中點擊選擇文件

打開後，則將文件內容輸出到textBox1文本框中，可以看到內容格式與原始文件基本相同，此種方式是以只讀方式打開文件，我們並沒有對文件內容做任何更改

將信息寫入到外部文件

將信息寫入到打開的文件，這裡面使用了StreamWriter類，看下面這段代碼

using (StreamWriter file = new StreamWriter(openFileDialog1.FileName, true))：表示向文件裡面追加內容

file.WriteLine(textBox1.Text)：實現文本框的內容，追加到文件當中最末行

運行程序後，在程序界面中的文本框內，輸入我們需要追加到已有文件中的信息內容

輸入完想加追加到文件當中的信息內容後，點擊"文件"菜單下的"打開"子菜單，選擇我們需要為其追加新內容的的文件，選擇好路徑後，雙擊文件圖標即可

然後，打開我們剛剛選中的文件，發現最末一行已經追加了我們想要追加的信息，而這個信息正是通過我們輸入到窗體界面文本框中的信息所添加的，這樣就實現了已有文件的寫入功能
