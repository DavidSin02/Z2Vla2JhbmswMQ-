射頻電路PCB設計的一些技巧

2020-10-29
| 科技

利用電容的“零阻抗”特性實現射頻接地

在射頻電路板上，必須提供一個對輸入和輸出端口的任何射頻信號都能作為“零”或參考點的共用射頻接地點(這些點必須是等電位的)。在射頻電路板上的直流電源供電端和直流偏置端，直流電壓必須保持在直流電源供電的原始值上，但對可能疊加在直流電壓上的射頻信號，必須使它對地短路或將其抑制到所要求的程度。換句話說，在直流電源供電和直流偏置端，其阻抗對交流或射頻的電流或電壓信號必須接近於零，即達到射頻接地的目的 [7] 。

在一個射頻電路板上的每一個射頻模塊中，不管是無源還是有源電路，射頻接地都是不可缺少的部分。在進行電路測試時，不良的射頻接地將導致對各種參數的測量誤差。不良的射頻接地也會降低電路性能，如產生附加噪聲和寄生噪聲，不期望的耦合和干擾，模塊和器件之間的隔離變差，附加功率損失或輻射，附加相移，在極端情況下還會出現意想不到的功能錯誤。

然而，期望通過在射頻接地端與真實接地點之間連接一個容量無窮大的理想電容實現射頻接地是不現實的。一般來說，只要電容容量足夠大，能使射頻信號能夠接到一個足夠低的電平就可以了。例如，一個直流電源供電端的射頻接地，常用的設計方法 [89] 如圖12-40所示，即在直流電源供電端的導線上連接多個不同容量(10pF～10μF)的電容到地，以實現射頻接地。

圖12-40

但不幸的是，這種利用多個電容實現射頻接地的方式往往達不到所希望的效果。正如6.2.2節“電容(器)的射頻特性”中所介紹的那樣，電容的射頻等效電路是一個包含R、L、C的網絡，會對不同的工作頻率呈現出不同的阻抗特性。對於一個特定頻率的射頻信號而言，不同容值的電容可能呈現一個高阻抗的狀態，但並不能夠起到射頻接地的作用。

從串聯RLC電路的阻抗特性知道，當一個串聯RLC迴路產生串聯諧振時，感抗與容抗相等，迴路的阻抗為最小(純電阻R)。對於一個質量良好的電容而言，其R很小，阻抗趨向為零。

如圖12-41所示，在設計電路時，對於一個特定頻率的射頻信號，可以選擇一個特定容值的電容，使它對這個特定頻率的射頻信號產生串聯諧振，呈現一個低阻抗(零阻抗)的狀態，從而實現射頻接地。注意，串聯諧振的電容和電感包括PCB的分佈電容和分佈電感。

利用電感的“無窮大阻抗”特性輔助實現射頻接地

在實現射頻接地中，一個“無窮大阻抗”的電感對 “零阻抗”的電容來說是一個很好的輔助元件。理論上，理想電感的阻抗Z L 是

式中，L為電感的感抗;ω為工作角頻率。

從式(12-23)可知，電感的阻抗Z L 對於ω=0的直流電流或電壓來說為零。當ω≠0時，隨著ω的增加或電感值的增加，其阻抗也在增加。對於特定頻率的射頻信號來說，如果電感L的值足夠大，Z L 的值可以達到很高。

正如在6.2.3節“電感(器)的射頻特性”中所介紹的那樣，在射頻條件下，理想電感是永遠也得不到的，電感器的射頻等效電路是一個包含R、L、C的網絡，對不同的工作頻率呈現出不同的阻抗特性。當一個電感的自感與其附加電容工作在並聯諧振的頻率上時，阻抗會變得非常高，趨向開路狀態(無窮大)。

在設計電路時，對於一個特定頻率的射頻信號，可以選擇一個特定容值的電感，使它對這個特定頻率的射頻信號產生並聯諧振，呈現一個高阻抗(無窮大阻抗)的狀態，從而實現射頻信號的隔離。

利用“阻抗無窮大”電感輔助“零阻抗”電容實現射頻接地的連接形式 [89] 如圖12-42所示，在圖中，在點P 0 和P 1 之間插入了一個“阻抗無窮大”電感，而P 0 是插入“阻抗無窮大”電感之前的同一根傳輸線上與外部的連接點。該“阻抗無窮大”電感會阻止外部的射頻信號從P 0 點傳輸到P 1 點。在P 1 點，來自P 0 點的射頻信號電壓或功率可以顯著地降低到想要的值，而P 1 點和P 0 點則保持相同的直流電壓。

利用“零阻抗”電容實現複雜射頻系統的射頻接地

對於一個射頻系統來說，射頻電路的接地平面必須是一個等電位面，即在輸入、輸出和直流電源及其他控制端必須有等於“零”電位的相應接地端。當PCB的尺寸比50Ω信號線的1/4波長小得多時，由銅等高電導率材料製成的金屬表面所構成的接地平面可以看做是一個等電位面。然而，如果PCB尺寸和50Ω信號線的1/4波長相等或大得多時，該接地平面就可能不是一個等電位面。也就是說，在工作頻率範圍內，當PCB的尺寸大於或等於工作頻率的1/4波長時，該接地平面就可能不是一個等電位面。

在進行PCB的設計時，利用多個“零阻抗”電容可以實現複雜射頻系統的射頻接地，一個設計實例 [89] 如圖12-43所示。

端口的PCB連線設計

如圖12-48所示，在輸入、輸出和直流電源端口，需要保證在輸入或輸出連線與相鄰地的邊緣的間隔(即W 1 )必須足夠寬。根據經驗要求W 1 >3W 0 [89] ，使在輸入或輸出連線邊界的電容可以忽略。對於一個射頻電路，往往要求其輸入、輸出連接線的特性阻抗為50Ω。

如果輸入和輸出連接線設計為如圖12-49所示的共面波導形式，可以不要求W 1 >3W 0 。因此，在地表面可以添加更多的金屬區域。在共面波導設計中，輸入、輸出連接線寬度W 0 通常要比與之對應的50Ω特性阻抗微帶線窄得多。同樣，在連線和相鄰地邊緣之間的空隙也比3W 0 窄得多。因此，接地區域可擴展，其幾何形狀可更為簡單。

利用接地平面開縫減小電流回流耦合

在討論電路時，通常關注的是直流電源提供的前向電流，而經常忽略返回電流。返回電流是從電路接地點流向直流電源接地點而產生的。在實際的射頻PCB上，由於迴流在兩個模塊間的耦合，所以完全有可能會影響電路的性能 [89] 。

一個設計實例如圖12-53所示。在圖12-53(a)中，由兩個模塊組成的電路安裝在PCB上，PCB的底層敷滿了銅，並通過許多連接過孔連接到頂層接地部分。對直流電源而言，底層和頂層的銅覆蓋的部分為直流接地提供了一個良好的平面。在直流電源和接地點A、B之間連接“零阻抗”的電容到地，使直流電源和接地點A、B之間對射頻信號短路，不形成射頻電壓。只要PCB的尺寸比1/4波長小得多，整個接地部分就不會形成射頻電壓。雖然在這些模塊間存在耦合，但也可以保證電路性能優良。

但實際上，根據PCB上的元件放置和區域的安排，在PCB上的電流模型是非常複雜的。從直流電源流出的所有電流必須返回它們相鄰的接地點A和B。如圖12-53(a)所示，從直流電源流出的電流用實線表示;從模塊的接地點流向直流電源接地點A和B的迴流用虛線表示。在每一個模塊中的迴流大致可分為兩類，也就是圖中的i 1a 、i 1b 和i 2a 、i 2b 。電流i 1a 和i 2a 從模塊1和2的接地點返回，流經底層敷銅的中央部分。電流i 1b 和i 2b 從模塊1和2的接地點返回並分別流經底層敷銅的左側和右側。顯而易見，只要在兩組路徑間的距離足夠遠，i 1b 和i 2b 的返回電流就可以忽略。

由於i 1a 和i 2a 匯流在一起，故在i 1a 和i 2a 間的串擾是電磁耦合。當它們之間的耦合或串擾不可以忽略時，其相互作用就等同於從模塊2到模塊1之間存在某種程度的反饋，而這種反饋可能會使電路的性能變差。減少這種耦合的一種簡單方法如圖12-53(b)所示，可以在PCB底層中間敷銅部分切開一條縫，目的是儘可能消除或大大減少在i 1a 和i 2a 間的迴流。

隔離

在一個複雜的射頻系統中，存在三種類型的隔離，即RF模塊間的隔離、數字模塊間的隔離，以及RF模塊和數字模塊間的隔離。除了PA模塊必須特殊處理外，RF模塊間的隔離和數字模塊間的隔離有類似的難點，也有相同的解決方案。RF模塊和數字模塊間的隔離要稍微複雜一些。由於RF模塊的功率通常要遠遠大於數字模塊的功率，故一般RF模塊的電流是mA級，而數字模塊中的電流是μA級。從功率角度來看，RF模塊會對數字模塊產生干擾。從頻率觀點來看，數字模塊的脈衝信號是一個寬頻帶信號，包含多次諧波的高頻成分，其中也可能包含RF信號的頻率成分。數字模塊會對RF模塊產生干擾。因此，RF模塊和數字模塊間的隔離要比其他情況都要更困難一些。

在無線通信系統中，理想的隔離度級別大約是130dB，它比從天線到數據輸出端的有效信號總增益還要高10dB。

在複雜的射頻系統中，屏蔽是有效的隔離手段，屏蔽能有效地抑制通過空間傳播的各種電磁干擾。用電磁屏蔽的方法來解決電磁干擾問題的最大好處是不會影響電路的正常工作，因此不需要對電路做任何修改。

在射頻系統中，對大功率、高頻率的信號一定要進行屏蔽。屏蔽體的性能用屏蔽效能來衡量，即對給定外來源進行屏蔽時，在某一點上屏蔽體安放前後的電場強度或磁場強度的比值。

屏蔽體的屏蔽效能與很多因素有關，它不但與屏蔽體材料的電導率、磁導率及屏蔽體的結構、被屏蔽電磁場的頻率有關，而且在近場範圍內還與屏蔽體離場源的距離及場源的性質有密切關係。一般而言，屏蔽效果取決於反射和吸收的條件。但是當PCB採用金屬盒屏蔽時，在30MHz以上頻率範圍內，反射要比吸收的影響更重要。

為獲得有效的電屏蔽，在設計中必須考慮以下幾個方面。

① 屏蔽體必須良好接地，最好是屏蔽體直接接地。

② 發揮屏蔽效果的關鍵是設計合理的屏蔽體形狀，即如何設計屏蔽盒的開口和連接部分之間的間隙。必須增多屏蔽盒的連接部分，從而將開口和間隙的最長的邊減至最小。屏蔽盒的連接部分必須具有較低的阻抗，而且必須相互緊密結合，不許有間隙。應確保屏蔽盒的金屬表面沒有絕緣材料塗層。

使用金屬盒對測試板進行屏蔽，使用1m法對輻射噪聲進行測量(信號頻率為25MHz)。假設屏蔽盒的開口總面積約為2500mm 2 。屏蔽盒開口的影響如圖12-55～圖12-58 [91] 所示。從這些測量中可以看出，當開口面積被分割成小孔時可以獲得較為優越的屏蔽效果。但是當屏蔽盒具有單個矩形開口時，將會顯著地降低屏蔽效果。一般來說，盒形屏蔽比板狀或線狀屏蔽有更小的剩餘電容，全封閉的屏蔽體比帶有孔洞和縫隙的屏蔽體更為有效。

④ 對於電屏蔽來說，對電屏蔽體厚度沒有特殊要求，只要屏蔽體結構的剛性和強度滿足設計要求即可。

在一個複雜的射頻系統中，由於相互隔離的要求比較高，所以各射頻單元大多使用模塊單獨屏蔽封裝，工作地通過螺釘或直接和模塊連接。如圖12-59所示，為防止接地環路過大，接地點的間距應小於最高頻率的波長的1/100，至少小於最高頻率的波長的1/20。各模塊通過螺釘直接和機殼連接，機殼通過接地線和大地連接。PCB佈局應按信號處理的流程走直線，嚴禁輸出遷回到輸入端。因為射頻鏈路一般都有很高的增益，故輸出的信號要比輸入的信號高數十個dB，儘管有屏蔽隔離，但設計不當仍會引起自激，會影響電路的正常工作。

射頻電路PCB走線

在RF電路的設計中，電路通常是由器件和微帶線組成的。在射頻電路PCB上的信號走線是微帶線形式，它對電路性能的影響可能比電容、電感或電阻更大。認真處理走線是RF電路設計成功的保證。

有關微帶線的設計請參考有關的資料。在PCB的設計中，1/4波長是非常重要的參數。超過1/4波長的走線，對RF信號而言，可能會從短路狀態變到開路狀態，或者從零阻抗變成無限大阻抗。在進行PCB的設計時，走線要保持儘可能短，即要求

如果走線的長度與1/4波長相當或大於1/4波長，在進行電路仿真時必須將走線作為一個元件來對待。

在進行射頻PCB的設計時，要求走線儘可能短，如圖12-60所示。圖12-60(a)的走線人為拉長了，不是一個好的設計。

圖12-60　走線儘可能短

在進行射頻PCB的設計時，要求走線的拐角儘可能平滑。在射頻PCB中的拐角，特別是急拐彎的角度，會在電磁場中產生奇異點併產生相當大的輻射。在如圖12-61所示的實例中，圖(a)的拐角形式優於圖(b)和圖(c)，因為圖(a)是平滑的，是最短的連接線。

圖12-61　走線的拐角形式

在進行射頻PCB的設計時，要求將相鄰的走線儘可能畫成相互垂直形式，儘可能避免平行的走線。如果不能夠避免兩條相鄰的走線平行，兩條走線間的間距至少要3倍於走線寬度，使串擾可以被減小到能夠允許的程度。如果兩條相鄰的走線傳輸的是直流電壓或直流電流，則可以不用考慮這個問題。

在進行射頻PCB的設計時，不僅對走線的拐角要求是重要的，對整條走線的平滑要求也是很重要的。如圖12-62(a)所示，走線從A到B的寬度有一個突然的改變(P點)。微帶線的特徵阻抗主要取決於它的寬度。P點的特徵阻抗會從Z 1 跳變到Z 2 。因此，該走線實際上成為一個阻抗變換器。這個阻抗額外躍變，對電路性能來說可能導致災難性的後果：RF功率可能在P點來回反射。另外，在P點，RF信號也會輻射出去，因此要求走線寬度漸漸改變，如圖12-62(b)所示，即要求走線的阻抗的變化是平緩的，以使在這條走線上附加的反射和輻射減小。
