開發利器IDEA快捷鍵（Mac版）

工欲善其事，必先利其器。

剛入職場的時候，筆者曾經誤以為，個人對IDE的熟悉會隨著工作時間而慢慢沉澱。但是後來發現並不是！

對工具 的學習和熟悉其實和研發方向的知識獲取一樣，也需要主動鍛鍊。從使用IDE的角度出發，只有熟悉了快捷鍵並使用，才能更好的提高自己使用IDE的效率。

以下是筆者根據自己當時對IDE的key-map的翻譯和網上資料的整合。同時，文末會附上對應版本的IDEA的key-map獲取方式。

熟練以後你就可以把IDE耍得飛起啦！

Mac鍵盤符號和修飾鍵說明

下面有些功能，Mac小鍵盤空間不足，沒有單鍵實現，轉而使用組合鍵來實現。

  • ⎋ Escape (Esc)
  • ⌘ Command 命令 = Windows的Windows鍵
  • ⇧ Shift 切換
  • ⌥ Option 選項 = Windows 的Alt鍵
  • ⌃ Control 控制
  • ↩︎ Return/Enter 換行
  • ⌫ Delete 向左刪除
  • ⌦ 向右刪除 = Fn + Delete
  • ⇞ Page Up = Fn + ↑
  • ⇟ Page Down = Fn + ↓
  • Home = Fn + ←左方向鍵
  • End = Fn + →右方向鍵
  • ⇥ 右製表符（Tab鍵）
  • ⇤ 左製表符 = Shift + Tab


快速修復

option + enter對象賦值變量

TypeWildcard typeWildcard = new TypeWildcard();，在編輯器中輸入new TypeWildcard()後，然後按option + enter，就會生成TypeWildcard typeWildcard = new TypeWildcard();，再按enter完成對象的創建和賦值

光標移動

  1. 按頁上下移動：fn + ↑ / fn + ↓
  2. 光標移動到當前行代碼的首尾：cmd + ← / cmd + →
  3. 光標移動到當前單詞的首尾：option + ← / cmd + →
  4. 高亮文件中當前元素cmd + shift + F7，然後control + option + ↑ / control + option + ↓在高亮元素中快速跳轉
  5. 光標移動到代碼塊的開始或結束位置：cmd + option + [ / cmd + option + ]
  6. 光標跳轉到上次修改的地方：cmd + shift + delete，跳轉到下次修改的地方沒有快捷鍵，可以使用菜單-> Navigate-> Next Edit Location
  7. 光標跳轉到上/下次瀏覽(光標停留)的地方：cmd + option + ← / cmd + option + →
  8. 光標跳轉到指定的行列：cmd + L
  9. 光標跳轉到當前行的上一行或者下一行，開始新的一行：shift + enter / cmd + option + enter

代碼縮進

  1. 縮：tab，注意回到代碼行的開始，才會生效
  2. 進：shift + tab，這個沒有縮的限制

代碼提示

編碼時idea自帶提示，當光標移到其他地方時，或者按了esc後，代碼提示沒有了， 此時按option + /就有了

快捷鍵設置：Keymap -> 搜索basic(Completion)

自動結束代碼，行末自動添加分號

cmd + shift + enter，此功能鍵，還可以添加一些

String str = "abc";


// 可添加分號


if (str == "abc") {


}


// 輸入完if (str == "abc")，使用快捷鍵可添加花括號

制代碼

顯示方法的參數信息

cmd + P，加強版cmd + 鼠標左鍵可查看更多信息（所在類、返回值、參數信息）

快速查看文檔

control + J，查看光標所在元素的文檔

快速生成一些代碼

cmd + N，類似的cmd + O覆蓋方法（重寫父類方法）,cmd + I實現方法（實現接口中的方法）

Command+n 等同於 Command+Enter快速選擇覆蓋的setter/getter等方法。


環繞代碼

cmd + option + T（使用if..else, try..catch, for, synchronized等包圍選中的代碼） 或者生成包圍標籤


顯示意向動作和快速修復代碼

option + enter，移除只有一條語句的if的花括號等

格式化代碼

cmd + option + L

優化import

control + option + O

複製、剪切、粘貼、刪除

  1. cmd + X剪切當前行或選定的塊到剪貼板
  2. cmd + C複製當前行或選定的塊到剪貼板
  3. cmd + D複製當前行或選定的塊
  4. cmd + V從剪貼板粘貼
  5. cmd + delete刪除當前行或選定的塊的行
  6. option + delete刪除到單詞的開頭
  7. option + fn + delete刪除到單詞的結尾
  8. cmd + shift + delete從最近的緩衝區粘貼

摺疊展開代碼

  1. cmd + + / cmd + -展開 / 摺疊代碼塊
  2. cmd + shift + +展開所有的代碼塊
  3. cmd + shift + -摺疊所有代碼塊

關閉當前查看的tab編輯器選項卡

cmd + W

大小寫切換

cmd + shift + U

查找、替換

  1. 雙擊shift，查詢任何符號
  2. cmd + F：文件內查找
  3. cmd + R：文件內替換
  4. cmd + shift + F：全局查找（根據路徑）
  5. cmd + shift + R：全局替換（根據路徑）
  6. cmd + G：查找模式為向下查找
  7. cmd + shift + G：查找模式為向上查找
  8. 查找方法在何處調用，option + F7全局查找，cmd + F7文件查找

方法調用層次

control + option + H


入口：find action→call hierarchy(control+option+H)

查看類繼承結構

control + H

入口：find action→hierarchy actions→hierarchy(control + H)

查看類圖

普通的子類名右擊Diagrams→Show diagram(option+shift+command+U)即可出現類圖。

查看maven依賴

pom.xml中右擊選Maven→show dependencies可以看到所有的依賴關係。

可以按command+f進行搜索

可以右擊exclude進行排除

查看當前field、method大綱

navigate→File Structure（cmd + F12） 或者 cmd + 7

複製文件名

直接點擊文件名cmd + C在文本編輯區cmd + V即可

複製文件全名

cmd + shift + C 再cmd + V即可

複製多個文件名

多次cmd + C，再cmd + shift + V，從最近的緩衝區選擇要粘貼哪些文件名

快速賦值變量

cmd + option + V，將表達式賦值給變量

省略中間變量

cmd + option + N，將中間過渡的變量省略

輸入文件名稱，實現文件快速跳轉

cmd + shift + O，輸入文件名

更改變量名稱

shift + F6，更改當前文件中變量

Preferences | Editor | General | Smart Keys`中勾選**Surround selection on typing quote or brace**，選中內容，`shift + "

選擇更大區域範圍代碼

option + ↑

雙擊標識符，選中單詞，而不是整個標識符

Preferences | Editor | General | Smart Keys中勾選use CamelHumps words

查看某個類的所有子類

cmd + option + B

當前文件內查找

cmd + F12
