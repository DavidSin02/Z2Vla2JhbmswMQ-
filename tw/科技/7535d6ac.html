超實用快照功能，幫你快速打造完美照片

2020-11-14
| 科技


愛上修圖，從潑辣開始。潑辣修圖每日教程， 助你從小白成為大師。

潑辣修圖5.4版之後添加了許多有趣的新功能，包括「倒影」、「像素化」、「快照」等，為了讓大家對這些功能有快速瞭解，潑辣修圖編寫了系列教程，本篇為大家介紹「快照」功能。

熟悉Photoshop的朋友對「快照」這個詞並不陌生，不過與PS中建立圖像處理記憶點的功能不同，在潑辣修圖中，你可以把快照理解為「合併」功能，即將當前的所有操作，尤其是涉及圖層、局部調整等添加的各類元素，完全合併為一張圖片，就像你全新打開一張圖片一樣。

這樣做的好處是，允許你進一步作為一個整體，調整圖像效果，這為圖層創意、二次修改、和添加多重濾鏡提供了極大的便利！因為之前為了實現類似效果，你必須將圖片保存副本，然後再重新打開。

潑辣修圖中的「快照」工具，可以非常方便的在作品中疊加效果，今天大家就來跟潑辣熊學學這個功能如何使用吧！

//

工具檔案

//

潑辣修圖是一款全平臺的專業攝影后期軟件，支持手機和電腦多平臺系統，曾獲得蘋果AppStore年度最佳應用，還是微軟Windows 10中文系統的預裝軟件。本教程非特指均為手機版軟件截圖演示。

• 工具位置：按鈕：任意工具→歷史記錄→快照；

• 主要作用：將當前圖像效果合併，課直接繼續調整；

• 應用場景：拓展參數、疊加濾鏡效果；

• 出現版本：潑辣修圖V5.4版之後新增功能；

• 是否收費：免費工具，非會員可自由使用；

打開潑辣修圖，點擊按鈕：「任意調整」→「歷史記錄」→「快照」（如上圖所示），即可找到快照工具的位置。

//

適用場景

//

一、擴展參數範圍

在潑辣修圖中，所有參數的調節程度用數字表示，一般最大值為100。如果此時調整到了極限還沒有達到滿意效果，想繼續調整為150、200甚至更高時怎麼辦？

這時我們就可以利用快照功能，就像打開一個新的副本文件一樣，再次調整這個我們未調整好的參數啦！

二、疊加濾鏡效果

在加載濾鏡時，如果加載的濾鏡並沒有達到我們想要的效果，我們可以通過快照功能多次疊加濾鏡以增強效果。

原圖

添加一次濾鏡

快照後二次添加濾鏡

三、添加圖層後進行全局調整

在潑辣的圖層功能中，當我們添加圖層後，大家會發現這時我們對照片調整，我們所添加的圖層並不會產生變化，這時我們就可以通過快照功能對添加好圖層照片進行全局調整了！

//

使用教程

//

這裡我們使用潑辣修圖中的「HSL」來給大家演示一下快照的神奇之處吧，打開潑辣修圖，打開我們要調整的照片！

選擇全局調整-HSL-對畫面種不同的的顏色進行統一，紫色、紅色、橙色、黃色、都向洋紅色靠攏，綠色、青色、藍色都向綠色靠攏，這樣畫面就只剩下大面積的兩個色相！

然後我們找到歷史記錄-選擇快照功能-對畫面進行合併調整的操作！

然後我們再次打開基礎調整，找到HSL對畫面的色彩再次進行調整，讓綠色偏青色，洋紅偏橙色，其餘顏色向這兩個顏色靠攏！

這樣我們就可以得到非常和諧統一的青色與橙色了！

//

注意事項

//

使用快照工具後你將無法對圖層等內容進行單獨調整了，所以在使用之前，請確保你所有的調整不會再針對局部元素，當然你可以通過歷史工具無限撤消步驟。快照工具是高手進階的絕佳兵器，希望你能完全掌握。
