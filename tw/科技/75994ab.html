C語言快速入門——邏輯運算

2020-10-29
| 科技

if控制語句允許我們測試一個條件表達式對程序進行分支處理。但如果需要同時滿足多個條件時，則需要使用邏輯運算符，C語言提供了3種邏輯運算符。

&&（邏輯與運算）

||（邏輯或運算）

! （邏輯非運算）

加減(+-)運算需要左右兩個操作數，如a+b。與此類似，與運算符(&&)與或運算符(||)也需要左右兩個條件表達式，這被稱為雙目運算符，具有從左向右的結合性。如，a&&b，表示先計算a表達式再計算b表達式。賦值運算符(=)用於將右端值賦給左邊變量，這被稱為右結合性，非運算(!)與之類似，具有右結合性。!0表示非0，即為真。

與運算(&&)

與運算是一個並且關係，即當所有條件同時成立時結果為true，如果有一個條件不成立則整個判斷結束。

在下面的代碼中，如果玩家等級大於2並且等級小於5時，整個條件表達式為true。如果此時玩家等級是1，在第一個判斷處已經失敗，因此後面的判斷便不會在進行計算，整個表達式為false，將執行後面的語句。

你也已經注意到，我使用一個;作為成立時的語句，這是可行的。因為在C語言中;號代表語句的結果。這裡它是一條空語句，用於保證if語句的完整性。

if ( level > 2 && level < 5 ){ ;}

或運算(||)

或運算是一個或者關係，即當所有條件中有一個成立時整個判定結束，結果為true，如果一個條件不成立則判斷下一個。

假定一個表格有90列，編號為1-90，此處的代碼用於測試無效的列索引。如果此時pos值為0，則第一次判定時整個表達式已經成立，這時整個或運算結束且結果為true，表明索引無效。如果pos為91，此時第一次測試失敗，第二次測試成功，同樣表示索引無效。

if ( column < 1 || column > 90 ){ printf( "error column index" );}

非運算(!)

非運算相當於反向反義的意思。即，非真為假，非假為真。 當表達x+y*z結果為0時，測試結果為真；結果為一個數值時，取反代表為假。

if ( !( x +y + z ) ){ ;}

更多討論

注意下面這些例子的語法規則。

以 if ( level > 2 && level < 5) 來說，條件表達式相當於構造一個有效範圍，要求level大於2且小於5。 但在書寫上並不是 if ( 2 < level < 5 )， 這種表達方式在C語言中是錯誤的。

類似的，如果我們要判斷三個數值不相等，不能寫成 if ( a != b != c ) 正確的寫法是：if ( a != b && a != c && b != c )

if ( level > 2 && level < 5 ) // okif ( 2 < level < 5 ) // errorif ( a != b != c ) // errorif ( a != b && a != c && b != c ) // ok
