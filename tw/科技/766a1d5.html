基於MicroPython的自動網絡時間校準器

2020-10-29
| 科技

前言

之前使用TPYBoard v202（主控ESP8266模塊）開發板做過一款WIFI攻擊神器，一開始使用Arduino源碼做的，後來又接觸到micropython。覺得使用micropython真是方便，而且ESP8266模塊上面有很多可以挖掘出來的好玩的。後期用它又做過智能家庭控制平臺，說白了就是用v202開發板作為Web服務器。最近，接觸到了DS3231時鐘模塊，結合OLED顯示屏，做了一個簡單的時鐘。為了防止時間偏差，從網上找了一個免費的獲取時間的API，程序很簡單，分享出來，有需要的小夥伴可以直接用。

程序編寫

先說下程序的思路：

1、開機上連接家中的WIFI網絡，確保可以訪問網絡；

2、然後訪問API獲取到時間，更新DS3231的設置，並進行顯示屏顯示；

3、開啟定時器，每隔1秒鐘刷新下顯示屏時間信息；

4、設定每天獲取一次API更新下時間，用於時間的校正。

首先，編輯一個boot.py文件，裡面主要是連接WIFI的程序。然後編輯一個main.py，裡面主要是訪問API、操作DS3231、顯示屏等主要功能程序。v202開機會先運行boot.py再運行main.py文件。

boot.py

import networkimport timefrom machine import PinLED = Pin(2,Pin.OUT)#根據設定連接網絡count = 0wlan = network.WLAN(network.STA_IF)wlan.active(True)if not wlan.isconnected(): print('connecting to TurnipSmart...') wlan.connect('WIFI名稱','WIFI密碼') while not wlan.isconnected() and count < 20: LED.value(not LED.value()) count += 1 time.sleep(0.5)if wlan.isconnected():  LED.value(0) print('ip address:',wlan.ifconfig()[0])else: LED.value(1) print('TurnipSmart connect fail')

main.py

import urequestsimport jsonimport timefrom machine import SPI,Pin #導入SPI、Pin庫import ssd1306 #導入OLED顯示屏驅動庫from ds3231 import DS3231 #導入DS3231時鐘模塊庫from machine import Timer #導入Timer庫#-----------------------DS3231模塊-----------------------#ds=DS3231() #ds3231初始化NOW_DATE = ''#-----------------------OLED顯示屏-----------------------##SPI接口對應的引腳定義spi = SPI(baudrate=10000000, polarity=1, phase=0, sck=Pin(12,Pin.OUT), mosi=Pin(13,Pin.OUT), miso=Pin(2))#OLED顯示屏的設置，128寬 64高 spi對象 DC接的G15 RES接的G16 CS接的G5display = ssd1306.SSD1306_SPI(128, 64, spi, Pin(15),Pin(16), Pin(5))display.poweron() #打開顯示屏display.init_display() #初始化顯示display.text('Waiting.....',1,1) #顯示的內容，x座標，y座標display.show() #進行顯示def getNetTime(): url = 'http://quan.suning.com/getSysTime.do' res=urequests.get(url).text print(res) j=json.loads(res) t2_date = j['sysTime2'].split()[0] #日期 t2_time = j['sysTime2'].split()[1] #時間 display.fill(0) #清屏 display.text(t2_date,1,2) display.text(t2_time,1,15) display.show() #初始日期和時間，設置一次即可 ds.DATE([int(x) for x in t2_date[2:].split('-')]) #設置初始日期年、月、日 ds.TIME([int(x) for x in t2_time.split(':')]) #設置初始時間時、分、秒def updateTime(t): global NOW_DATE #讀取日期和時間，拼接成正常的時間格式 date = '20'+'-'.join(ds.DATE()) time = ':'.join(ds.TIME()) if NOW_DATE != date: getNetTime() #設定每天校對一下時間 NOW_DATE = date else: display.fill(0) #清屏 display.text(date,1,2) display.text(time,1,15) display.show()if __name__ == '__main__': getNetTime() time.sleep(1) #-----------------------Timer定時器-----------------------# tim = Timer(-1) #新建一個定時器 #每隔1秒執行一次updateTime函數調用，用於更新OLED顯示屏上的時間 tim.init(period=1000, mode=Timer.PERIODIC, callback=updateTime)

裡面還需要用到DS3231時鐘模塊庫，已經寫好了，直接拿來用就可以。記得也要下載到板子裡，否則會提示找不到。文章末會有下載。

程序下載與調試

v202通過micro USB數據線接入PC，即可進行程序下載和REPL操作。插入PC之前，先確保一下器件的電源線沒有接錯，防止出現器件或開發板燒壞的情況。

1、插上PC之後，設備管理器會出現一個虛擬串口，開發板上用的是CH340芯片（USB轉TTL串口），電腦沒驅動的話，需要手動安裝。

CH34X驅動下載

2、使用 MicroPython File Uploader 工具進行程序的下載，在這裡就不介紹具體下載的步驟，大家可去參考現有教程。 TPYBoard v202 快速入手教程 boot.py 、main.py 和 ds3231.py 都需要下載到v202板子裡。下載完畢後，點擊【RUN】或者按板子的RST鍵都可以運行程序。
