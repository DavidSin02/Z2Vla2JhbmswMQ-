小小科普 略帶專業——聊聊近炸引信的類型

2020-10-29
| 科技


引信，是炮彈、導彈上很重要的一個裝置，任務是在對目標毀傷效果最大的時機，引爆戰鬥部。近炸引信，又稱近感引信。顧名思義，因“近”而爆炸的引信。不過要注意：也有萬米高度上引爆的近炸引信。因為“近”不是“遠近”，而是“接近”。


那就是接近到一定距離就引爆唄。也不完全是。有的是接近到設定的瞄準線，或者某個角度，並非距離。這次我們就說說近炸引信依據什麼，判斷該不該炸。


先說一下近炸有什麼好處。一發配備碰炸引信（又稱觸發引信）的炮彈，碰地爆炸。可以看到，破片要麼向上飛，要麼被土地吸收。過去部隊裡有“新兵怕炮老兵怕機槍”的說法，因為老兵知道臥倒可以大幅減少傷亡。統計表明：炮彈殺傷兵員的 8 5 % 發生在炮擊的前 1 5 秒，此後新兵們也趴下了。


如果配近炸引信，凌空爆炸，趴在地面是什麼後果？


一般地講，一發炮彈近炸的殺傷力，相當於3 ~ 1 3 發碰炸，也就是替換比 3 ~ 1 3。越是臥倒、越是躲在塹壕中，替換比越高。


對空射擊時，近炸引信的效果更顯著。


比如用小口徑高炮迎頭攔截反艦導彈，如果配碰炸引信，或用不裝炸藥的動能彈，必須直接命中才能摧毀目標。而如果換用近炸引信，只要脫靶量（目標軌跡延伸線與彈道的距離）小於引信的作用半徑，脫靶的彈丸就還能通過爆炸破片毀傷目標。假設目標導彈半徑 0 . 3 米，引信作用半徑 3 米，可以輕易算出，一發近炸彈相當於1 0 0 發觸發彈。而且近炸引信的作用半徑一般都不止 3 米。





如果是側向攔截導彈，要比迎面攔截更難。在某次演習中，配碰炸引信的炮彈打 7 0 0 0 多發也很難命中一次，配近炸引信的卻只需要一兩百發。注意：這時的替換比可不是 7 0 0 0：2 0 0，而是 1 / 2 0 0：0 / 7 0 0 0，無窮大哦！


常有人提出這樣的疑問：如果一發彈丸本來可以直接命中目標，近炸引信讓它在 3 米外就炸，毀傷效果不是下降了嗎？


其實不會。有個術語叫“引戰配合”，就是說：為了取得最大的毀傷效果，引信要與戰鬥部匹配，不同的類型戰鬥部要配不同類型的近炸引信。近炸引信有多種分類方法，比如按引爆方式、按原理、按波長、按頻段等。需要與戰鬥部匹配的是按引爆方式分類，即引信根據什麼判斷起爆時機到了。


近炸引信的引爆方式有定距、定高、定角、定向、定線等幾種類型。


定距近炸，就是沿彈軸方向，彈丸和目標之間的距離達到預定值時，引爆。


定距近炸引信主要配用於破甲戰鬥部，比如反坦克導彈上。因為聚能破甲戰鬥部是靠金屬射流擊穿裝甲，而射流的形成需要一定距離。在最佳距離上起爆，可以取得最大破甲厚度。為此，破甲彈前面經常加一個空心頭錐，或者長鼻子探杆。


但頭部太長會影響飛行穩定性。而且現在很多坦克披掛爆炸反應裝甲，對付它得靠提前引爆的前置小戰鬥部。有時甚至要提前 5 ~ 1 0 米引爆前置戰鬥部，這時用長杆已不可能，必須用定距近炸引信。它雖然成本略高，但能在不改變導彈外形時，獲得最佳引爆距離。


反坦克彈藥的定距近炸引信，“定”的是軸向彈目距離。它一般是貼地飛行，彈藥側面和地面之間的距離，經常比預定的引爆距離小。而且坦克首裝甲經常是大角度傾斜，彈藥側面和裝甲的距離，也經常小於預定的引爆距離。但引信只“看”軸向，不理側面，因此這些側面距離不會讓引信動作。


定高近炸，在預定高度引爆。除了前面講過的炮彈凌空爆炸，很多空地彈藥上也需要，以便更好地對付地面目標。


子母彈要求在空中開艙拋撒子彈，而開艙高度直接影響子彈藥在地面的分佈密度。如果太低，會讓很多子彈藥“扎堆”浪費，總殺傷面積減少；如果太高，很容易漏掉目標。




雲爆彈，要求在地面上一定距離進行第一次引爆，撒開燃料，待燃料與空氣混合後第二次引爆。如果高了，到達地面的衝擊波會減弱；如果低了，影響燃料和空氣混合。


反輻射彈、煙幕彈等特種彈藥，也常有定高近炸的需求。


高度也是距離，因此定高近炸和定距近炸有點像。區別是“定高”的距離不是沿著炮彈、子母彈的軸線，而“定距”是沿軸線，不管兩邊的。因此它們在具體的測距技術上有些區別。測距、近炸的精度，也略有區別。雲爆彈和對付雷達、電臺天線的反輻射彈，希望炸高精準。子母彈對此要求略低。炮彈，則更低了。根據試驗和統計，1 5 5毫米榴彈對立姿步兵的最佳炸高，是 3 ~ 1 0 米；降低到 2 米或升高到 1 8 米，殺傷效果只下降 1 0 %；低到 0 . 5 米或高到 3 0 米，也只下降 3 0 %。因此炮彈的近炸引信允許有較大散佈，這也符合炮彈引信對低成本的要求。


即便是要求高的彈藥，一般也不是固定在某個炸高，而是多采用自適應定高引信，根據落角、目標等情況，自動調整炸高。


定角近炸，在離軸角達到預定值時引爆。離軸角，是目標和彈藥之間的連線，與彈藥軸線之間的夾角。


這大多用在防空導彈的戰鬥部上。要介紹這個，先得講講戰鬥部爆炸後的破片，是怎樣飛散的。


大多數炮彈、戰鬥部爆炸後，如果沿著炮彈軸線看，破片是一圈 3 6 0 °均布地分散。


但如果從側面剖一下，會發現 8 0 % 以上的破片集中在上下各 3 0 °左右的飛散角內，軸線前後方向的很少。


防空導彈，這個飛散角更小，特別是採用杆式預製破片戰鬥部時，飛散角只有兩、三度。如果是連續杆戰鬥部，就會一邊飛一邊散開成環，像“圓鋸”一樣“切斷”敵機。這時剖面上的飛散角就更小。


戰鬥部破片的飛散速度，和目標的相對速度合成，就會在空中形成一個圓錐形的毀傷面。相對速度不同，圓錐面的形狀也就不一樣。


定角近炸引信，就是負責預測出這個圓錐面，並在其“接觸”目標時起爆。起決定作用的是離軸角，和目標距離沒有太大的關係，只要在破片的有效殺傷範圍，也就是作用半徑內，離軸角達到預定值就起爆。炸點離目標近，殺傷效果當然好，但那隻能說明導彈打得準、脫靶量小，不是引信的功勞。


如果圓錐面離軸角計算錯誤，即便是飛機離得再近，那也可能打不中：炸早了，飛機從圓環裡穿過；炸晚了，圓環落在飛機後面。


不同交會條件下，“預定”離軸角的數值都不同，因此定角近炸引信要有很強的“自適應”能力，會自己計算。




定向近炸，也是防空導彈上常用的，依據脫靶方位選定起爆點。


這種近炸引信要配合定向戰鬥部使用，因為後者爆炸後，破片不是周向 3 6 0 °均勻飛散，而是某一個方向比較密集。實現這的方法很多，比如戰鬥部裡的炸藥偏心起爆，或者在某一方向預製 V 形槽。這需要導彈有控地旋轉，才能把殺傷力最強的側面對準目標。如果戰鬥部沿圓周設置多個偏心起爆點，定向引信選擇一個引爆，就不需要旋轉了。


定向近炸引信，大多是定角近炸引信的一種特例，增加了脫靶方位探測，能控制不同位置的雷管起爆。因此它在技術和結構上更復雜一些。


定線近炸，是在戰鬥部軸線對準目標時，就引爆。注意：戰鬥部軸線和彈藥的軸線、飛行線，可不一樣。比如下面這枚“比爾”反坦克導彈。


“比爾”能採用這種近炸引信，是因為它採用爆炸成型戰鬥部，產生一種自鍛破片攻擊坦克頂裝甲，距離遠近個十幾米都對殺傷效果影響不大。而且導彈是在距地面一定高度飛行，是一種掠飛擊頂反坦克導彈。


路旁式反坦克地雷，也喜歡採用這種定線近炸引信，這時就是它等著坦克自己撞到線上。末敏彈也用它。掛著降落傘下降，邊旋轉邊掃描，在地面掃出一個螺旋線。


定線引信的瞄準線與戰鬥部軸線基本重合，與飛行方向有一定夾角。它要求有很強的目標識別能力，能區分目標、地面，甚至真假目標、已被炸燬的目標。它視線、視野很窄，看到目標就“發火”。因此儘管這類彈藥沒有導引頭，但也能做到“打了就不管”，被列入精確制導彈藥。歸納一下：拋開引信探測方向圖、提前量、信號處理時間等複雜概念，可以概略地認為，近炸引信都有一個隨彈運動、不容觸碰、一觸即發的敏感區。




定距引信和定線引信的敏感區都在戰鬥部軸線上。但定距引信（上）只對彈軸上的一個點敏感；定線引信（下）對戰鬥部軸線上一個長線段敏感，而且這個線段與飛行方向不重合，並且只對特定目標，比如裝甲車輛敏感。


　 　定高引信，對一個以引信為球心的外球面敏感。如果是自適應定高引信，敏感面就是橢球面。　 　定角引信，對一個空心圓錐的內錐面敏感。圓錐角大小取決於彈藥和目標的相對速度，相對速度越高，圓錐角越小。敏感圓錐面內，彈丸的正前方是盲區。如果脫靶量很小或為零，目標始終處於盲區，直到碰炸了還未發現目標。這叫“觸發優先”，是定角引信的一種特性。有了這種特性就不用擔心明明能直接命中，還提前近炸了。



定向引信，只對偏向目標的一部分內錐面敏感。


現代彈藥，特別是巡飛彈等新型彈藥，一般配備多模戰鬥部，能以爆炸成型、聚能破甲、破片殺傷等不同爆炸方式攻擊不同目標。因此它們的近炸引信，一般也具有可選擇的不同作用方式，包括定線、定距、定高等，稱為多用近炸引信。如果近炸之外還可以選擇碰炸或定時模式，就稱為多模引信、多選擇引信。美國M782多選擇引信，用於炮彈，有無線電定高起爆、時間、觸發、延期等多種引爆方式。裝定它時不需要手工操作，靠電磁感應直接寫入數據。



　 　 德國PX581激光近炸引信，用於迫擊炮彈，定距起爆。長度只有9.5釐米，重約230克。內部有一個渦輪發電機充當電源。


　 　 DSU-33無線電近炸引信，用於航空炸彈，定高起爆。


　 　DM84多選擇引信，用於82、120 毫米迫擊炮彈，有無線電定高、時間、觸發、延期等選擇，也是電磁感應裝定的。

　 　美軍的幾種艦炮引信：MK417，無線電近炸，用於 76 毫米炮彈，長 15.2 釐米，重 894 克；MK418，無線電近炸，配 127 毫米炮彈，尺寸重量和前者一樣；MK404，紅外近炸，76 和 127毫米炮彈都能用，長 14.5 釐米，重 953 克。　 　


它們都是定角起爆，適合打擊空中和地面目標，甚至反導。


　 　看到引信還有這麼多名堂，很多讀者可能會感到意外，以為本文是“未來時”，或者僅適用於高價值彈藥。這可不對。近炸引信早在第二次世界大戰末期就已廣泛應用，產量達到過每年上千萬發。前面提到的一些數據，可都是真炮彈打出來的。　 　


我軍在抗美援朝、福建前線空戰和抗美援越時，都吃過近炸引信的虧。戰士們根據定高近炸炮彈的破片殺傷形式，還形象地稱其為“一把抓”。吃一塹長一智。目前，制導彈藥自不待言，我國各種大小口徑的炮彈、火箭彈也多配有近炸引信。　 　


在一些國家裡，三軍所有的曲射炮殺傷爆破彈，包括迫擊炮，都配備了包含近炸功能的多選擇引信，而且頻繁更新換代。目前已經可以將彈道修正裝置集成進多選擇引信，做成彈道修正引信，庫存老式彈藥換一個引信就變成了靈巧彈藥。
