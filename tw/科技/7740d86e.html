學習NLP的第10天——文章關鍵詞提取：詞頻統計

2020-10-29
| 科技

關鍵詞提取是詞語顆粒度的信息抽取的一種重要的需求，即提取文章中重要的詞語。

關鍵詞提取的常用方法包括詞頻統計、TF-IDF和TextRank等。

其中，詞頻和TextRank屬於單文檔算法，即只需一篇文章即可提取出其中的關鍵詞；而TF-IDF則屬於多文檔宣發，需要其他文檔的輔助來提取當前文章的關鍵詞。

詞頻統計的Python實現

詞頻統計的邏輯是：在一篇文章中，越重要的關鍵詞往往會在文章中反覆出現；因為為了解釋關鍵詞，作者經常會反覆地提及它們。所以通過統計文章中各個詞語的出現頻率，即可初步地獲得關鍵詞。

但是因為齊夫定律，文章中出現頻率最高的往往並不是長度較長的關鍵詞，而是標點符號和助詞等，因此在詞頻統計之前還需要先進行停用詞過濾。

齊夫定律：一個單詞的詞頻與它的詞頻排名成反比。

由此，詞頻統計的流程通常是中文分詞、停用詞過濾、詞頻統計。依據以上邏輯，我在Python中實現以下詞頻統計。（以《紅樓夢·桃花行》節選為例）

from pyhanlp import HanLPfrom pyhanlp import JClassdef load_from_words(*words):    """    從詞彙構造雙數組trie樹    :param words: 一系列詞語    :return:    """    map = JClass('java.util.TreeMap')()  # 創建TreeMap實例    for word in words:        map[word] = word    return JClass('com.hankcs.hanlp.collection.trie.DoubleArrayTrie')(map)def remove_stopwords_termlist(termlist, trie):    return [term.word for term in termlist if not trie.containsKey(term.word)]if __name__ == "__main__":    # 《紅樓夢·桃花行》節選    article = "桃花簾外東風軟，桃花簾內晨妝懶。簾外桃花簾內人，人與桃花隔不遠。"    # 停用詞表(詩中包含的哈工大停用詞表的停用詞)    trie = load_from_words("，", "。", "與")    # 中文分詞+停用詞過濾    termlist = HanLP.segment(article)    termlist = remove_stopwords_termlist(termlist, trie)  # 分詞結果去除停用詞    print("分詞結果:", termlist)    # 詞頻統計    word_frequency = dict()    for word in termlist:        if word not in word_frequency:            word_frequency[word] = 0        word_frequency[word] += 1    word_frequency_sorted = sorted(word_frequency.items(), key=lambda x: x[1], reverse=True)  # 詞頻排序    for i in range(5):        print(word_frequency_sorted[i][0], "詞頻:", word_frequency_sorted[i][1])

其中load_from_words和remove_stopwords_termlist在之前（第7天）的學習中已經掌握。

運行結果

分詞結果: ['桃花', '簾', '外', '東風', '軟', '桃花', '簾', '內', '晨妝', '懶', '簾', '外', '桃花', '簾', '內', '人', '人', '桃花', '隔', '不遠']桃花 詞頻: 4簾 詞頻: 4外 詞頻: 2內 詞頻: 2人 詞頻: 2

基於HanLP實現的詞頻統計

HanLP中封裝了TermFrequencyCounter類用來統計文檔的詞頻，接著我們使用這個類實現詞頻統計。

from pyhanlp import *TermFrequency = JClass('com.hankcs.hanlp.corpus.occurrence.TermFrequency')TermFrequencyCounter = JClass('com.hankcs.hanlp.mining.word.TermFrequencyCounter')if __name__ == '__main__':    counter = TermFrequencyCounter()    counter.add("桃花簾外東風軟，桃花簾內晨妝懶。簾外桃花簾內人，人與桃花隔不遠。")  # 第1個文檔    counter.add("東風有意揭簾櫳，花欲窺人簾不卷。桃花簾外開仍舊，簾中人比桃花瘦。")  # 第2個文檔    print("2篇文章的詞頻前5名:", counter.top(5))    #  根據詞頻提取關鍵詞    print("第1篇文章的詞頻前5名:", TermFrequencyCounter.getKeywordList("桃花簾外東風軟，桃花簾內晨妝懶。簾外桃花簾內人，人與桃花隔不遠。", 5))

運行結果

2篇文章的詞頻前5名: [簾=8, 桃花=6, 外=3, 東風=2, 隔=1]第1篇文章的詞頻前5名: [桃花, 簾, 外, 隔, 軟]

可以看到，整體結果是相近的，HanLP去除了更多的停用詞，包括“人”、“內”以及標點符號等。

用詞頻提取關鍵詞存在一個缺陷，就是即使使用過濾停用詞以後，高頻詞也並與關鍵詞完全等價。例如在分析一個明星的相關新聞時，明星名字的出現頻率可能是最高的，但是在我們希望找到每一篇文章各自的特點，而不是文章的共性，此時，我們就需要引入TF-IDF等關鍵詞提取方法。
