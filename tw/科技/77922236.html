2020-10-29
| 科技

選擇器是最長用到的屬性，一般都會用到也算比較熟。但是有都佈局或樣式需求比較特殊，可能需要JS計算才能完成，但是有好多可以用選擇器就可解決。本次整理出來這麼多選擇器也是一次踩坑了，為了避免再次掉進同一個坑就把所有的選擇器整理出來了；有需要的可以保存留著以解燃眉之急，或推薦給剛入行或基礎薄弱點的朋友，贈人玫瑰手留餘香；

屬性選擇器

1.屬性選擇器 [屬性 ~= value]

例：屬性中包含獨立的單詞為value

div [class ~= haha]{

width: 100px;

height: 100px;

background-color: cadetblue;

}

<div id="div1">

<p class="p1 haha"></p>//正確

<p class="p1"></p>

</div>

2. 屬性選擇器 [屬性 *= value]

例：屬性中做字符串拆分，只要能拆分出來value這個字符串就行；

div [class *= hello]{

font-size:16px;

color: red;

}

</div>

eg:[title*=flower] <img src="../../i/eg_tulip.jpg" title="ffffflowerrrrrr" />

3. 屬性選擇器 [屬性 |= value]

例：屬性中必須是完整且唯一的單詞，或者是 - 分割開

div [class |= ggg]{

font-size:20px;

color: blue;

}

<div id="div3">

<div class="rrr-ggg">1</div>//錯誤

<div class="rrr ggg">2</div>//錯誤

<div class="ggg">3</div>//正確

<div class="ggg-rrr">4</div>//正確

</div>

4. 屬性選擇器 [屬性 ^= value]

例：屬性中前幾個字符是value就行

div [class ^= en]{

width: 100px;

height: 40px;

background-color: coral;

margin-top:20px;

}

<div id="div4">

<div class="en"></div>//正確

<div class="enttt"></div>//正確

<div class="ensss"></div>//正確

</div>

e.g:[lang^=en] <p lang="ennn">

5. 屬性選擇器 [屬性 $= value]

例：屬性中後幾位字符是value就行

div [class $= yuan]{

width:200px;

height: 50px;

margin-top:10px;

background-color: cadetblue;

}

<div id="5">

<div class="yuanddd"></div>

<div class="ddddyuan"></div>//正確

<div class="jljlj-yuan"></div>//正確

</div>

eg:[src$=".pdf"] <img src="test.pdf">

21. ::before和::after

這兩個主要用來給元素的前面或後面插入內容，這兩個常和”content”配合使用，使用的場景最多的就是清除浮動。

偽元素選擇器

1、偽類 和 偽元素的區別

偽類：匹配的是元素(不同狀態或結構的)

偽元素：匹配的是元素中的一部分內容(首字符，首行文本)

2、語法

1、:first-letter 或 ::first-letter

匹配 某元素的 首字符

2、:first-line 或 ::first-line

匹配 某元素的 首行文本

3、::selection

匹配 被用戶選取的 部分

3、: VS ::

在 CSS3 之前，偽元素 使用的是 :

:first-letter

:hover

在 CSS3 之後，偽元素 一律使用 ::

::first-letter

:focus

語法：選擇器1+選擇器2
