瑞德西韋從毫克級別到百公斤級別合成工藝（含晶型和製劑研究）

2020-10-29
| 科技

瑞德西韋（Remdesivir，GS-5734）是一種在研的廣譜抗病毒藥物，屬於RNA依賴的RNA聚合酶（RdRp)抑制劑，可通過抑制病毒核酸合成發揮抗病毒作用。目前，多項臨床試驗正在調查該分子治療COVID-19的潛能。

該化合物的合成已經申請了多項世界範圍的專利，藥化方面的數據於2017年已經公開，發表在美國化學會ACS旗下著名的期刊J. Med. Chem.上（J. Med. Chem. 2017, 60, 1648−1661）

初期為了滿足細胞水平的活性測試，需要快速的拿到篩選活性化合物，吉利德的科學家團隊，以最簡單粗暴的方式，設計了一條路線合成瑞得西韋，路線如下：

該路線首先將化合物15的氨基（-NH2）使用TMS保護，再在丁基鋰的作用下與內脂14進行糖苷化反應然後將生成產物與化合物14反應再脫TMS保護生成化合物16（這裡面有一個化合物15的芳香伯胺的原位硅保護和脫保護過程）再將化合物16進行氰基化引入氰基（-CN）得到17，化合物17在BCl3作用下脫苄基得到化合物4，4與化合物19反應得到消旋化合物4a，最後通過手性HPLC拆分離純出4c與我們想要的終產物Remdesivir（4b）。

雖然該路線不適合放大生產，但是在做從成千上萬個化合物中篩選活性好的明星分子，這樣的合成方法屢見不鮮，百試不爽，總之策略就是快，效率高。

在做了細胞水平的活性測試以後，在細胞測試實驗中表現極好的活性，瑞得西韋脫穎而出，為了滿足藥理，藥代，動物實驗等方面的活性測試，吉利德的科學家，針對化合物合成中的問題，要想做大，必須重新設計合成方法，通過大量的實驗，他們有第一條路線進行了一些局部的優化和改進，合成路線如下：

針對化合物篩選時合成路線的問題，吉利德的科學家通過大量的文獻調研和實驗，優化出手性合成的方法，實現了百克級別制瑞得西韋的合成。他們使用碘代物20替換第一代路線的溴代物15，並改變反應條件使這一步的產率由第一代的25%提升至40%，再通過反應條件的優化，找到了不對稱氰基化的方法，改方法技巧性極強，合成設計極其巧妙、在順利得到化合物4後，通過對化合物4鄰位雙羥基（-OH）的保護，高產率（90%）地得到了化合物21。

此條路線中，最關鍵的步驟是單一構型化合物22b的合成。化合物18a與苯基二氯磷酸酯及對硝基酚反應得到消旋化合物22a（產率80%），經過多種溶劑、多種條件的篩選探索，藥物化學家發現將消旋化合物22a置於異丙醚中輕輕攪拌22h後，過濾後可得白色固體，此白色固體即為單一構型化合物22b，最重要的手性便這樣引入了， 化合物21與單一構型化合物22b反應（一定要高純度），再脫保護，最終得到單一構型的目標化合物Remdesivir（4b）。 這條路線可以合成百克級別的Remdesivir，用於後期研究中，但是問題很多，比如合成化合物16時，後處理需要使用柱層析，在合成幾十公斤的化合物16時，五元內酯環活性低、易於烯醇化和消除、滷代鹼基類似物脫滷等關鍵難題，需要使用很多的大層析柱，另外由於放大效應，反應比較雜，收率不太穩定，放大的時候仍然出現許多問題，曾經嘗試過在這一步反應中上大位阻的保護基，收率可以提高到50%~60%左右，但是大家想想，為了做幾十公斤的中間體，在車間裡同時並列使用幾十根大層析柱的場景，是比較頭疼的事情，且不安全，效率低下。

另外上氰基的時候，在酸性條件下加入如TMSCN，雖然是超低溫反應，但是體系中有HCN產生，很不安全，還有在酸性環境下，產物有一些分解，產生了雜質，降低了產品的收率

為了建立安全，可靠的合成方法，我突然發現吉利德公司負責工藝的科學家又開發了新的工藝，他們嘗試了一些新的工藝改進，方法如下：

在反應釜中加入無水氯化釹（III）（1.0當量），正四丁基氯化銨（1.0當量）和四氫呋喃（THF，9體積），並將所得混合物在環境壓力下在約66℃下濃縮至約4.5體積。加入THF（4.5體積），並重復濃縮兩次。將混合物冷卻至約20℃，化合物5（282kg，1.0當量）。約30分鐘後，將混合物冷卻至約-20℃並保存備用。

在另外一個反應容釜中，加入化合物4（1.10當量）和THF（5體積），並冷卻至約0℃。緩慢加入三甲基氯硅烷（1.10當量），保溫0℃30分鐘後，將混合物冷卻至約-10℃。緩慢滴加加苯基氯化鎂的THF溶液（2M/L，2.17當量），並且在約30分鐘之後，將反應混合物冷卻至約-20℃。緩慢滴加異丙基氯化鎂（2M的THF溶液，1.13當量），保溫約2小時後，將格氏反應混合液轉移至化合物5/ NdCl3 / n-Bu4NCl / THF混合物中。將所得混合物在約-20℃下攪拌，並且在約8小時後，添加乙酸（1體積）和水（4體積）的溶液。將所得混合物加熱至約20℃，並加入乙酸異丙酯（i-PrOAc）。分離各層，有機層依次用碳酸氫鉀水溶液和氯化鈉水溶液洗滌。將有機層在真空下濃縮至約4.5體積，並加入i-PrOAc。將有機層用水洗滌兩次，並在真空下濃縮至約4.5體積。加入i-PrOAc，並重復濃縮。將混合物過濾並將濾液在真空下濃縮至約3體積。加入甲基叔丁基醚（MTBE），並將體系降溫至約20℃。加入化合物1的晶種，然後加入正庚烷，並將體系冷卻至約0℃。通過過濾分離固體，並用MTBE /正庚烷混合液潤洗濾餅。將得到的固體在真空下乾燥，以69％的收率和100％的純度得到化合物1 。各種分析數據和先 前報道的數據完全一致。

本方法使用了 NdCl3 / n-Bu4NCl /的複合鹽調控親核性，以69％的收率和100％的純度得到化合物1，避開了柱層析，且做到了200過公斤的化合物1

另外就是上氰基的問題，1：是立體選擇性的問題，大量化合物1 / TFA與TMSOTf / TMSCN結合使用所需的工作時間增加，這可能會對非對映選擇性和收率產生負面影響，2：產生的HCN:，導致操作危險性很大，為了解決這兩問題，吉利德的工藝科學家嘗試了連續流的方法

他們篩選了大量的條件，就我感覺，摸索工藝做毫克級別的反應，是一個非常粗糙的數據，但是還是介紹一下，數據如下：

反應條件：將化合物1在DCM中的溶液冷卻至設定的溫度，並依次添加指定的Brønsted 酸，TMSOTf和TMSCN。a通過LC分析確定純度。 b反應條件：將化合物1的DCM溶液冷卻至設定溫度，並添加TMSCN，然後加入TFA和TMSOTf。 c分別在大約30分鐘內添加TMSOTf和TMSCN。 d反應條件：在-30°C下，將TFA添加至化合物1在DCM中的溶液中，然後添加預冷至-30°C的TMSCN和TMSOTf在DCM中的溶液。

首先觀察溫度升高的問題，從-78℃升高到-15℃，可以觀察到化合物2含量在降低，說明產物轉變成了雜質，將溫度降低到-30℃，產物增加。

通過這個表格發現TFA和TfOH的優勢相對明顯，在使用TFA時，當溫度升高到 30°C時，TMSOTf和TMSCN的用量都增加到6當量時，反應的選擇性增加，提供了93：7的非對映選擇性，而在使用TfOH時，溫度保持在-78 °C，TMSOTf和TMSCN的用量為2當量和3當量時，提供了96：4的高非對映選擇性，加料順序對非對映選擇性也有影響，在添加TFA和TMSOTf之前添加TMSCN會減弱選擇性。

這個表格總體設計不是科學，沒有時間參數，溫度低度變化幅度太大，不精細，另外使用TfOH時，數據太少了，需要補充，總之問題很多，估計是吉利德的科學家不想公開的太多吧。

結晶工藝優化：

反應條件：將TFA（3當量）加入到-30°C的DCM中的化合物1的溶液中，然後將TMSCN（6當量）和TMSOTf（6當量）在DCM中的溶液預冷至-30°C 添加。通過LC分析確定純度。 b未分析反應樣品，以最大程度減少暴露於危險反應混合物中。

在一次性投料化合物1,9。6公斤，22公斤，23.1公斤時，從表格上看，在混合溶劑乙酸乙酯和庚烷中結晶效果最好，以及在單一溶劑甲苯和異丙醇中結晶次之。

連續流化學條件的優化

使用連續流反應器，每次投料1g，做了一些的的小試平行實驗，我們發現

在TFA為3當量，TMSCN（6當量）和TMSOTf（6當量），整個體系預冷溫度控制在-40°C，化合物1溶解於DCM中，通過盤管循環預冷，和第2個同樣也通過盤管循環預冷的TFA和DCM的混合液混合，在第一個反應器混合盤管循環後，導入第3個通過盤管循環預冷的TMSOTf的DCM混合液，在第二個盤管反應器反應，接著導入第三個通過盤管循環預冷的TMSCN的DCM混合液，再在第三個盤管反應器反應中反應，後導入到含有KOH的接收瓶淬滅反應，控制了HCN的產生。

通過實驗數據，發現一個結果和第二個結果比較理想，特別是第二個，居然不用加入TFA也獲得了很好的結果，那為什麼還要加入TFA呢，是不是有一些矛盾，為了驗證這個結果，又進行了下面的實驗進行驗證。

使用SS連續流系統，對100克化合物1（不使用TFA）進行了實驗，以確認系統穩定性。儘管起始原料的殘留量略有增加（2.5％）以及非對映選擇性的降低（91：9），但所得結果仍在預期之內（表4，第1項數據）。當時，這些低於預期的結果歸因於實驗開始和結束時的擴散，因為使用SS反應器系統進行此較小規模的試驗運行時，收集到大量的非穩態反應混合物。在確認SS系統正常運行後，使用首選表3中條第12項數據表明的TFA為0當量，但是遇到了更為明顯的挑戰。觀察到原料的不完全轉化和非對映選擇性的顯著降低（表4，第2項數據）。使用先前使用的系統對設備的設置和執行進行調查（使用VapourtecTM系統進行小規模實驗，以及使用內部3/16英寸PFA反應器系統或SS反應器系統進行100 g實驗），並使用2.75 kg SS反應器系統中，當未向反應混合物中添加TFA時（表3的第11-12項數據和表4的第1項數據與表4的第2項數據相比），表明反應器的配置和結構材料不是改變結構的原因。反應性能。

尋找小規模試驗與2.75千克試驗運行之間差異的替代解釋。假設殘留的水可能通過TMSOTf的水解而產生微量的TfOH。然後，生成的TfOH可以充當催化劑，同時催化轉化的選擇性和轉化率。這種作用可能會對較小規模的反應（即1克甚至100克）產生更大的影響，而隨著規模的增加（即2.75千克）而影響減小。為了驗證該假設，使用SS反應器系統將TfOH引入到150 g反應的TMSOTf / DCM儲備溶液中，發現轉化率和非對映選擇性已完全恢復（表4，第3項數據）。進一步評估了TFA在反應混合物中的使用，並使用SS反應器系統進行了150 g規模的實驗（表4，第4~6項數據）。發現在工藝過程中0.5摩爾當量的TFA，得到可接受的轉化率和選擇性（表4，第5項數據）。為了適應三泵系統流量中的潛在波動，並提高過程的穩定性，在後續運行中選擇了1.0當量的TFA。在實施此更改過程中，TFA / TMSOTf進料管線的背壓增加了， 最終堵塞。在進行其他實驗以評估堵塞的原因後，發現TFA（熔點為-15.4°C）在-40°C的溫度下在TMSOTf / DCM混合物中的溶解度不好，並且可能在進料的預冷卻過程中析出結晶。為了避免此問題，工藝中已去掉了TFA / TMSOTf / DCM進料預冷卻迴路。此外，將反應溫度從-40°C調節到-30°C，以降低TFA從TFA / TMSOTf / DCM進料溶液中結晶的風險。這些變化對反應的性能沒有實質性影響，因為化合物2的溶液純度（和反應的非對映選擇性）是相似的，儘管反應溫度升高並且TFA / TMSOTf / DCM預冷迴路已移除（ 表4，第5,6項數據）。

在這些新的條件下，隨後的5次投料2.75 kg的化合物1進行實驗驗證，提供了高溶液純度和非對映選擇性表5，第2~6項數據表明TFA是有利於反應收率的提高。

合併來自多個批次的猝滅的流出物，並以66％的收率和99.7％的純度分離出13.0kg的化合物2。

在工廠使用連續流化學法合成化合物2

使用各種連續流反應器系統進行克級別到千克級別的結果，並將應用於每個系統的工藝參數用於設計工廠連續流反應器系統的基礎。以下主要因素用於工廠系統參考。

首先，必須妥善控制生產體系，以確保安全處理含氰化物的反應混合物。為了解決這個問題，流動反應器和輔助設備均由不鏽鋼製成，並在使用前進行了嚴格的壓力測試。

其次，生產體系需要具有在合理的時間內生產數百公斤化合物2的能力。表6列出了不同系統的工藝參數的比較。可以清楚地觀察到停留時間，反應器體積和產量（化合物1的消耗量）之間的相互作用。從該表可以看出，容積為2.5 L的不鏽鋼管反應器，可以實現約2 kg / h的通過量，從而能夠在約4天的時間內處理200 kg的化合物1。對於生產要求，該連續生產時間是可以接受的。為了始終如一地達到三種進料中每種進料的期望流速，選擇了三個由不鏽鋼和PTFE製成的隔膜泵，它們配有背壓調節器作為進料裝置，並在使用前進行了校準。

最後，生產體系需要具有足夠的冷卻能力和效率，以便在反應之前將試劑溶液調節到約-30°C，以及迅控制反應過程中產生的熱量，確保生成高質量的產品連續流生產。這些因素通過使用SS結構和管道式反應器系統控制，實現快速熱量傳遞，並使用大容量加熱/冷卻模塊。

在工廠大規模的連續流反應器系統的安裝和調試完成之後，在化合物1的投料量為86 kg的條件下進行了試驗運行，以驗證工廠的可操作性，用於合成化合物2的系統。如圖4所示，在短暫的誘導期後，即獲得系統的穩定操作，在整個流動反應過程中觀察到一致的反應曲線。數值結果可見於表7（第1項數據），通過表7的第1項數據和表5的第2-6項數據相比，表明與放大生產實驗系統相比，SS公斤級的系統提供了與化合物2相似的反應非對映選擇性和產品純度。用公斤級實驗批次相同的方式將淬滅的反應混合物進行水後處理，用於化合物2在DCM中在溶液儲存。在將溶劑置換成甲苯後，通過重結晶後過濾，獲得68kg的化合物2（表7，實驗數據1）。改工藝生產的瑞得西韋可滿足臨床的需要。

隨後使用相同的工藝，進行了另外兩次生產，投料量分別為280 kg和250 kg，分別得到204 kg和198 kg化合物2（表7，第2,3項數據）。

化合物2的製備

1：普通工藝:

投料化合物1（23 kg，1.0當量）在DCM（10體積）中的溶液中冷卻至-40°C，加入三氟乙酸（3.0當量）， 加入TMSOTf（6.0當量）和TMSCN（6.0當量）在DCM（5體積）中的預冷溶液（-30°C），同時確保反應釜體系內部溫度低於約-25°C。加畢， 將反應混合物在低於-30℃下攪拌不少於10分鐘後 ，取樣，通過HPLC監測反應完全，用預冷的KOH（35.8當量，14倍體積的水，預冷到-10℃）溶液淬滅反應。

將兩相混合物升溫至室溫。分離有機層，並用氯化鈉水溶液洗滌3次。有機相在真空下濃縮至約4體積，用DCM（8體積）稀釋並在真空下濃縮至約6倍體積。將混合物過濾，濾液濃縮至約4倍體積。將濃縮物用甲苯（19體積）稀釋，並在約50℃下真空濃縮至約14體積。將混合物加熱至約90℃，冷卻至約55℃，加入化合物2的晶種。將混合物在約55℃下攪拌養晶約1小時，並在約6小時內緩慢降溫至約0℃。過濾得到固體，用甲苯（3體積）洗滌濾餅。將固體在約50℃下真空乾燥，以71％的收率和99.2％的純度提供化合物2。

以上視頻是間歇式生產的工作視頻

2：連續流化學過程生成化合物2：

化合物1（250 kg，1.0當量）在DCM（15.0體積）（進料1），TMSOTf（6.0當量）和TFA（1.0當量）在DCM（4.4體積）中的儲存溶液（2） 在分開的反應器或進料容器中製備進料2）和DCM（4.5體積）中的TMSCN（6.0當量）（進料3）。將進料1以約504 mL / min的流速用計量泵輸送通過約-30°C的預冷卻迴路，並將進料2以約207 mL / min的流速泵送。將進料1和2在約-30℃的盤管反應器1中合併約30秒。然後將流出物與進料3（通過約-30°C的預冷卻迴路以約189 mL / min的速度泵輸送）在約-30°C的盤管反應器2中混合約2分鐘。合併的出料口的流出物，直接收集到裝有KOH（19.7當量）在水（8體積）中的溶液的容器中，保溫10°C。

將混合液升溫至約20℃，然後加入2-丙醇並分離各層。有機層用氯化鈉水溶液洗滌兩次，過濾，濃縮，所得溶液，加入甲苯到濾液中並 將混合物濃縮。 將混合物加熱至約55℃，然後冷卻至約0℃， 過濾所得漿液，用甲苯沖洗並在約60℃下乾燥，得到 化合物2，收率為78％，純度為99.9％。分析數據與文獻中先前報道的數據一致。

監測氰化過程（實驗室規模和工廠規模）的廢料中氰化物含量，並用次氯酸鈉進行處理，直到氰化物含量低於氰化物試紙的檢測極限為止。

Remdesivir的製劑

化合物Remdesivir在水性介質中相對不溶且化學上不穩定。在pH 約4至5之間的溶液中最穩定。在pH為4或以下時，化合物的溶解度增加。

(Remdesivir在不同pH時的穩定性)

注射用Remdesivir凍乾製劑是一種不含防腐劑的白色至灰白色或黃色凍幹固體。除活性成分外，凍乾製劑還包含以下非活性成分：注射用水，磺丁基倍他環糊精(SBECD）和鹽酸和/或氫氧化鈉。SBECD是陰離子型高水溶性環糊精衍生物，能很好地與藥物分子包合形成非共價複合物，從而提高藥物的穩定性、水溶性、、安全性、降低腎毒性等。

凍幹組合物製備方法：

將環糊精溶解在水中，向溶劑中加入酸調節pH小於2，將化合物Remdesivir加入到該酸化的混合物中形成複合物的懸浮液，向溶液中加入鹼調整pH至約3.5，形成過飽和溶液。根據需要向其中加入水調節化合物的濃度，並將混合物凍幹。
