原創： 王運永 朱宗宏 現代物理知識雜誌 8月30日

王運永1，2 朱宗宏21.中國科學院高能物理研究所；2.北京師範大學引力波與宇宙學實驗室

2.3 光量子噪聲

光量子噪聲源自光的量子性質。它直接產生於測量和讀出過程。在激光干涉儀引力波探測器探測頻帶內幾乎所有頻率上它都會對靈敏度加以限制。光量子噪聲通常表現為兩種形式：霰彈噪聲和輻射壓力噪聲。霰彈噪聲是光探測器中的強度量子噪聲，它在高頻區域占主導地位，輻射壓力噪聲是從測試質量反射的光子的動量轉移產生的，它在低頻區域占主導地位。

A. 霰彈噪聲

從統計物理可知，激光器發射的光子數目本身是有漲落的。也就是說在激光束中光子數並非在每個時間點都是相同的。激光束的強度是有起伏的。激光束強度的漲落在干涉儀輸出端引起的噪聲稱之為霰彈噪聲，有時也稱為散粒噪聲。引力波噪聲誤差h中的霰彈噪聲的大小可以用譜密度表示

hshot( f ) =（1/L）【(ℏcλ)/（2πPin）】1/2

其中L是干涉儀的臂長，c是光速，λ 是激光的波長，Pin 是輸入的激光功率。可以看出，增加激光功率能夠降低霰彈噪聲。

B. 輻射壓力噪聲

光子具有動量，在干涉儀臂中往返運動的光束中的光子，在撞擊到幾乎自由下垂的鏡子(即測試質量)表面之後，會向相反的方向折回，將自己的動量傳遞給鏡子。這種光子動量的轉移使鏡子受到一種壓力，稱為光輻射壓力。在該力的作用下，鏡子會向光子彈回方向的反方向反衝，其平衡位置發生變化。由於光子數目的統計漲落，到達鏡子表面的光子數並非在每個時間點都是相等的。也就是說，光輻射壓力不是常數，它是有統計漲落的。這種輻射壓力的漲落會直接引起測試質量位置的波動，形成噪聲。稱之為輻射壓力噪聲，這是光的量子特性產生的另一類噪聲，它導致測試質量位置的直接晃動。引力波噪聲誤差h中輻射壓力噪聲的大小也可以用譜密度表示為：

hrp( f ) =（2/L）X( f )

= （1/mLf 2）【（ℏPin ）/(2π3cλ )】1/2

在這裡m是測試質量的質量，L是干涉儀臂長，f 是輻射壓力噪聲的頻率，c 是光速，λ 是光的波長。可以看出輻射壓力噪聲的大小與輸入功率的平方根Pin 成正比，它不再是白噪聲，而是與頻率的平方f2成反比。這表明，輻射壓力噪聲在低頻區域顯得更為重要。光量子噪聲在低於20Hz的區域變得很大就是由這個效應引起的。增加鏡子的質量可以降低測試質量對力的機械易感性從而減小輻射壓力效應對測試質量運動的影響。在實際應用中，初級激光干涉儀引力波探測器測試質量為10kg，為了減小輻射壓力噪聲的影響，高級探測器的測試質量為40kg，而籌備中的第三代探測器ET的測試質量已增加到200kg。

C. 標準量子極限SQL

通過以上分析我們知道，在激光干涉儀引力波探測器中，有兩種與光的量子特性相關的噪聲源，它們與輸入功率Pin的關係是相反的。我們可以把這兩種噪聲看成一種噪聲的兩張面孔，它們都是由光的量子效應引起的，在無相互關聯(激光功率不是非常大時)的情況下，光量子噪聲的大小為兩種噪聲之和：

horn( f ) = h2sh ot( f ) + h2rp( f )

在低頻區域，輻射壓力噪聲占主導地位，在高頻區域，霰彈噪聲占主導地位，增加輸入光束的功率Pin可以改善干涉儀在高頻區域的靈敏度，但要以增加低頻區域的噪聲為代價。對於任何一個給定的工作頻率來說，都存在一個最佳激光功率，選擇這個最佳激光功率，霰彈噪聲和輻射壓力噪聲的幅度貢獻大小相等。使霰彈噪聲和輻射壓力噪聲的影響得到折中，這時激光干涉儀的位移靈敏度達到一個最佳值。

hshot( f ) = hrp( f )

這個最佳值是干涉儀位移靈敏度的一個基本極限，稱為標準量子極限，通常以無量綱振幅hSQL( f ) 來表示。對常規的激光干涉儀引力波探測器來說，標準量子極限hSQL( f ) 是其靈敏度提高的最後障礙。實驗表明，對於任何一個可以選用的激光功率，都會有一個霰彈噪聲曲線與輻射壓力噪聲曲線的交匯點，這些交點連接起來形成一條線，它就是激光干涉儀的標準量子極限曲線。

圖1 顯示了標準量子極限hSQL( f ) 與頻率及激光功率的關係。

標準量子極限SQL的大小與干涉儀的具體參數有關，例如，高級LIGO的標準量子極限hSQL( f )在頻率為f = Ω/2π = 100Hz 時，數量級為10-24 / Hz 1/2 。

D. 標準量子極限突破

標準量子極限SQL最初是作為激光干涉儀引力波探測器靈敏度不可逾越的最後極限提出來的。但是人們很快就意識到，標準量子極限僅僅適用於經典的激光干涉儀。但是有很多新技術，如失諧信號循環技術、壓縮態光場注入技術等，都可以用來建造結構更加複雜的干涉儀，俗稱非經典干涉儀。其靈敏度可以在一定頻帶內突破標準量子極限。

2.4 引力梯度噪聲

從萬有引力定律可知，測試質量周圍的所有物體都會與該測試質量相互吸引。局部質量分佈的變化(例如大氣密度的變化、人員來往、車輛移動和附近地區的風吹草動等)引起的局部牛頓引力場的漲落，產生噪聲，這種噪聲稱為引力梯度噪聲。引力梯度噪聲會使隔震系統”短路”，直接作用在鏡子上，它是無法迴避的。它是低頻段的主要噪聲源之一，對初級探測器來說(如第一代激光干涉儀)，其低頻靈敏度很差。牛頓噪聲的影響表現不出來，在高級探測器(如第二代干涉儀)中已經引起關注，而第三代激光干涉儀引力波探測器需要極大地提高低頻區域的靈敏度，引力梯度噪聲成為必須解決的問題。

為了降低引力梯度噪聲的影響，干涉儀要建在遠離局部質量密度漲落大的區域，例如把探測器建在地下。在測試質量周圍佈置一個監測器陣列，對質量密度的漲落進行實時監測並進行修正也是行之有效的。

2.5 殘餘氣體噪聲

激光干涉儀引力波探測器的真空室和真空管道中會有少量氣體殘留下來。這些殘留氣體除了引起單擺熱噪聲外，它的密度擾動會使折射率發生漲落，這種漲落會對激光束的傳播產生影響，形成噪聲，稱為殘餘氣體噪聲。另外，殘餘氣體分子對鏡面不同部位的撞擊是隨機的。撞擊產生的壓力在鏡面上的分佈是不均勻的。它會引起鏡子晃動，形成噪聲。降低殘餘氣體噪聲的主要方法是讓真空室和真空管道保持儘可能高的真空狀態，並減少材料的氣體釋放。

2.6 雜散光子噪聲

在光的傳輸過程中，少部分激光光子會被散射離開主光束。當它們隨後被反射回來時，會和干涉儀中攜帶引力波信號的光束耦合。這些散射光攜帶的是它們散射面上的信息，因此會汙染期望中的信號，形成噪聲，稱為雜散光子噪聲。在設計激光干涉儀引力波探測器的真空室和真空管道時，要採取必要的措施，如在適當的部位設置光闌，使該噪聲減至最小。

綜上所述我們可以看到，激光干涉儀引力波探測器的噪聲是由多種因素決定的。它們的大小和頻率特性也互不相同。降低的措施也不同。

03

激光干涉儀引力波探測器的靈敏度

靈敏度是激光干涉儀引力波探測器最重要的參數，由於引力波的強度非常弱，激光干涉儀的噪聲水平非常高，其靈敏度的定義和確定都具有鮮明的特色。

3.1 靈敏度的定義

為了表示干涉儀對引力波探測的靈敏程度，比較不同干涉儀之間的性能，需要定義一個通用的參數——“靈敏度”。激光干涉儀引力波探測器的靈敏度定義如圖2所示。

圖2 激光干涉儀引力波探測器的靈敏度

設干涉儀的臂長為L，當引力波到來時，根據引力波的特性，相互垂直的兩臂一個伸長另一個相應地縮短，設臂長的變化量為ΔL，則兩臂的長度分別變為L+ΔL，L-ΔL，干涉儀的應變靈敏度hd定義為：

hd = ΔL/L

3.2 干涉儀靈敏度的確定

為了確定激光干涉儀引力波探測器靈敏度的大小，需要用一個標準引力波信號源對其進行刻度。但是，迄今為止，人類還不可能在實驗室內建造這樣的信號源。因此，激光干涉儀引力波探測器靈敏度的高低尚不能用引力波信號的幅度來表示。需要用其他方法來確定。我們知道，引力波信號是極其稀有的。迄今為止，除了少數幾個真實事例之外，我們在引力波探測器上獲得的大量數據基本上都是噪聲。這就是說，對一臺引力波探測器來說，我們現在可以方便地知道它能探測到的最小噪聲水平是多少。探測器自身的靈敏度是由其噪聲水平決定的，對於一定幅度的信號來說，信噪比越大，探測到該信號的機率越大，該探測器的靈敏度越高。因此我們現在將激光干涉儀引力波探測器靈敏度的大小定義為：當干涉儀中引力波信號的幅度與其噪聲水平相等時，即信噪比等於1時的噪聲值。更明確地說，激光干涉儀引力波探測器的靈敏度曲線實際上是它在信噪比等於1時的噪聲值曲線。

為了更好地理解激光干涉儀引力波探測器的靈敏度曲線，我們首先要了解幾個基本概念，它們不但用來描述探測器的靈敏度，而且在分析探測器性能、處理噪聲和引力波信號的過程中也是經常用到的。

A. 譜密度和功率譜密度

在頻率域內，我們可以寫出一個探測器系統所探測到的能量隨頻率變化的函數，稱之為能量頻率譜，又稱能譜。在頻帶寬度為1Hz的單位頻帶中所含有的能量稱為能量頻率譜密度，簡稱譜密度，以符號D(w)來表示

單位時間內的能量譜密度稱為功率譜密度S(ω)。

S(ω) = D(w) T, （T是探測時間間隔）

現在我們來討論激光干涉儀引力波探測器的靈敏度和噪聲水平表示法。我們知道，探測器中的噪聲信息是時間的隨機函數，也就是說，噪聲信號何時出現，以多大幅度出現都是隨機的。在無限大的時間間隔內(從-∞到+∞)，噪聲幅度的平均值是零。因此用幅度平均值表示噪聲水平是不可取的。對於噪聲這一類的隨機函數，其大小要用功率譜密度來表示。功率譜密度有時簡稱為功率譜。時間函數S(t)的功率譜定義為其自相關函數的傅里葉變換Ps( f )

我們在討論功率譜Ps( f ) 時，通常只用正頻率而不用負頻率。故我們定義：

S2( f ) ≡2Ps( f ) f ≥0 , S2( f ) ≡ 0 f < 0

S2( f ) 稱為單邊功率譜。若表徵噪聲函數S(t) 的物理量是電壓V(t) ，那麼功率譜S2( f ) 的單位就是V2( f )/ Hz 。在討論像噪聲這樣的隨機時間序列時，通過帶寬為Δf 的過濾器的功率表示為：V2( f ) /Hz ∙ Δf 。這就是說，當把相鄰各噪聲合併在一起時，用的是功率(即幅度的平方)相加，而不是幅度相加。

在實際中，我們經常用從功率譜導出的一個物理量“幅度譜密度”S( f ) 。它定義為功率譜的平方根：

S( f ) ≡【 S2( f )】1/2

如果功率譜的單位為V2 /Hz ，則幅度譜密度的單位為V / Hz 1/2 。

功率譜密度和幅度譜密度是描述同一個噪聲水平的兩個物理量，當用頻譜分析儀進行頻譜測量時，用功率譜的單位比較方便。當用示波器或電錶進行測量時，用幅度譜密度的單位比較方便。符號Hz 沒有什麼物理意義，它的作用只是時刻提醒我們，即使使用幅度譜密度這一概念，也要牢記：在以時間函數表示的噪聲中，對各個獨立頻帶的噪聲值求和時，不是幅度的線性相加，而是幅度平方的線性相加。

引力波的強度通常以無量綱振幅h 表示，因此，在表示激光干涉儀引力波探測的靈敏度時，無量綱振幅h的單位為“ 1/ Hz 1/2 ”。

結 束 語

引力波的發現標誌著引力波天文學完成了從引力波尋找到天文學研究這一歷史性轉折。第三代激光干涉儀引力波探測器是引力波天文臺的核心設備，當前，世界各大引力波實驗室正在投入大量的人力物力對它進行前期研究，靈敏度直指10-24，這是一個非常艱鉅的任務，充滿了挑戰和機遇。分析主要的噪聲來源，研究降低不同噪聲的方法和技術是首先需要面對的難題。（全文完）
