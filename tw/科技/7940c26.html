Python網絡爬蟲與信息提取！爬蟲批量搜索音樂並下載

2020-10-29
| 科技

目標

將想要下載的歌曲名字存在列表中，批量搜索並下載代碼。

準備

因為webdriver打開網頁緩慢的原因，我考慮使用selenium控制已經打開的chrome網頁，畢竟直接用chrome搜索歌曲和打開網頁還是很快的。首先在pycharm中打開終端，輸入以下命令切換到谷歌瀏覽器目錄下：

cd \d C:\Program Files (x86)\Google\Chrome\Application1

執行命令打開谷歌瀏覽器並保存配置在本地，這裡路徑可以選擇和代碼同級目錄：

chrome.exe --remote-debugging-port=9222 --user-data-dir="e:\py_code\Reptile"1

執行後會打開谷歌瀏覽器 ，在地址欄輸入我們的網址：

http://www.gequdaquan.net/gqss/index.html

OK，準備工作完成！接下來交給腳本幹活。

代碼設計

網頁驅動的配置：

	chrome_options = Options()    # chrome_options.add_experimental_option("debuggerAddress", "127.0.0.1:9222")    chrome_options.debugger_address = "127.0.0.1:9222"    chrome_driver = "chromedriver.exe"    driver = webdriver.Chrome(chrome_driver, chrome_options=chrome_options)

這沒啥好講的，固定格式和端口號，具體參照谷歌中自動控制說明。

網頁靜音

因為前一篇文章說了下載歌曲需要點擊播放，所以靜音好點：

    try:        driver.find_element_by_xpath("//a[@class='player-btn btn-quiet']").click()    except:        pass

這裡用try except框架，因為如果用戶點擊了靜音，再一次點擊是靜音按鈕的class屬性會發生變化從而找不到對應的xpath會引發異常，所以如果異常直接跳過即可。

初始化

這裡初始化搜索的所有歌曲名字，本地保存路徑和歌曲下載地址的列表：

    search_name = ["萬有引力","苦笑"]    savaer_path = "D://music//"    list = []

批量獲取url

下面這段代碼調用我們自己寫的getMusicUrl函數獲取每個搜索名字對應歌曲的URL：

    print("開始獲取url")    for name in search_name:        music_url = getMusicUrl(driver,name)        list.append(music_url)

函數具體實現和上一篇博客基本相同，不做累述：

def getMusicUrl(driver, search_name):    print("打開搜素框")    driver.find_element_by_xpath("//span[@data-action = \"search\"]").click()  # 點擊按鈕    print("搜索音樂")    getXpath(driver, "//div[@class='search-group']/input[@id='search-wd']").clear()    getXpath(driver, "//div[@class='search-group']/input[@id='search-wd']").send_keys(search_name)    getXpath(driver, "//div[@class='search-group']/button[@class='search-submit']").submit()    time.sleep(5)    print("播放音樂")    # 因為搜索會重新加載界面，如果獲取不到按鈕控件則不能調用點擊函數會拋出異常    flag = True    while flag is True:        try:            flag = False            target = getXpath(driver,"//div[@class='list-item'][1]")            ActionChains(driver).move_to_element(target).perform()            getXpath(driver, "//div[@class='list-item'][1]/span[@class='music-name']/div[@class='list-menu']/span[@class='list-icon icon-play']").click()        except:            flag =True    print("獲取地址")    #music_url = str(driver.find_element_by_xpath("//audio").get_attribute("src"))    music_url = str(getXpath(driver, "//audio").get_attribute("src"))    print(music_url)    print("關閉音樂")    getXpath(driver, "//a[@class='player-btn btn-play btn-state-paused']").click()    return music_url

批量下載

下面這段代碼遍歷url列表，並批量下載：

    print("開始下載")    num = 0    for _url in list:        DownloadFile(_url, savaer_path, search_name[num] + ".mp3")        num+=1

下載代碼，上一篇博客已經詳細介紹不做累述：

def DownloadFile(mp3_url, save_url,file_name):    try:        if mp3_url is None or save_url is None or file_name is None:            print('參數錯誤')            return None        # 文件夾不存在，則創建文件夾        folder = os.path.exists(save_url)        if not folder:            os.makedirs(save_url)        # 讀取MP3資源        res = requests.get(mp3_url,stream=True)        # 獲取文件地址        file_path = os.path.join(save_url, file_name)        print('開始寫入文件：', file_path)        # 打開本地文件夾路徑file_path，以二進制流方式寫入，保存到本地        with open(file_path, 'wb') as fd:            for chunk in res.iter_content():                fd.write(chunk)        print(file_name+' 成功下載！')    except:        print("程序錯誤")
