windows系統常見快捷鍵的使用用法介紹

2020-11-14
| 科技

1、F功能快捷鍵

F1 顯示當前程序或者windows的幫助內容。

F2 當你選中一個文件的時候為“重命名”

F3 資源管理器的搜索功能

F4 顯示當前列表中的項目

F5 屏幕刷新

F10或ALT 激活當前程序的菜單欄

PrtSc（printscreen） 快速截屏（可以粘貼到QQ，畫圖，word等軟件中）

Backspace 資源管理器和瀏覽器中的後退

Ctrl shift Esc\Ctrl Alt Delete 任務管理器

Delete 刪除（放入回收箱）

shift delete 永久刪除（不放入回收箱）

2、Ctrl快捷鍵

Ctrl S 保存

Ctrl W 關閉程序

Ctrl N 新建文件夾

Ctrl O 打開

Ctrl Z 撤銷

Ctrl F 查找

Ctrl X 剪切

Ctrl C 複製

Ctrl V 粘貼

Ctrl A 全選

Ctrl [ 縮小文字

Ctrl ] 放大文字

Ctrl B 粗體

Ctrl I 斜體

Ctrl U 下劃線

Ctrl Shift 輸入法切換

Ctrl 空格 中英文切換

Ctrl 回車 QQ號中發送信息

Ctrl Home 光標快速移到文件頭

Ctrl End 光標快速移到文件尾

Ctrl Esc 顯示開始菜單（相當於win）

Ctrl Shift < 快速縮小文字

Ctrl Shift > 快速放大文字

Ctrl F5 強行刷新 （任何界面）

Ctrl 拖動文件 複製文件

Ctrl Backspace 啟動關閉輸入法

拖動文件時按住Ctrl Shift 創建快捷方式

3、win快捷鍵

win7新增的3D效果：

Win Tab 快速切換已打開的程序（和Alt tab一樣的效果）

Win Home 將所有使用中窗口以外的窗口最小化

Win Space 空格鍵將所有桌面上的窗口透明化

Win ↑ 上方向鍵 最大化使用中窗口

Win ↓ 下方向鍵 最小化窗口／還原先前最大化的使用中窗口

Win 左／右方向鍵 將窗口靠到屏幕的左右兩側

Win 1~9 開啟任務欄上相對應的程序

Win ( /-)　打開Windows放大、縮小功能

Win G 在屏幕上的Gadget 間切換

Win X 打開移動中心

Win D 顯示桌面，最小化所有窗口

Win E 打開資源管理器

Win F 打開資源管理器搜索功能

Win L 鎖定計算機，切換用戶，回到登陸窗口

Win M 最小化當前窗口

Win P 投影機輸出設定（僅屏幕、同步顯示、延伸、僅投影機）

Win R 打開運行窗口

Win T 任務欄的Alt Tab

Win U 打開控制面板輕鬆訪問中心

Win Break 打開控制面板系統屬性

￼4、Shift Win 左／右方向鍵 將窗口移到左、右屏幕（如果你有接雙屏幕的話）工作列快速列

Shift Win ↑ 上方向鍵 垂直最大化使用中窗口

Alt Win 1~9 開啟對應位置軟件的右鍵選單

5、Alt快捷鍵

Alt 空格 C 關閉窗口

Alt 空格 N 最小化當前窗口

Alt 空格 R 恢復最小化窗口

Alt 空格 X 最大化當前窗口

Alt 空格 M 移動窗口

Alt 空格 S 改變窗口大小

￼Alt Tab 兩個程序交換

Alt F 打開文件菜單

Alt V 打開視圖菜單

Alt E 打開編輯菜單

Alt I 打開插入菜單

Alt O 打開格式菜單

Alt T 打開工具菜單

Alt 左／右方向鍵 資源管理器中的前進/後退
