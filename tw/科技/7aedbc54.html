「音樂雜談56」混音選擇困難2，Waves均衡器全介紹與理論使用心得

喜歡「音樂雜談」這個主題的朋友可以關注我的頭條號，將會在不定期發表一些音樂理論以外的音樂話題的文章或者是音樂知識的乾貨 。（此文為混音師天職老師 發佈於今日頭條的原創文章，轉載請告知並註明出處）

通篇寫作整理下來差不多花了7個小時，不管怎樣，施捨點個贊吧。哈哈哈！

繼上一次「音樂雜談41」混音選擇困難第一期，給大家介紹了Waves全家桶的大部分壓縮器之後，本篇，我們將來看看，Waves全家桶的大部分均衡器的介紹。

這裡把第一期貼出來給大家方便一起閱讀。「音樂雜談41」混音選擇困難，Waves壓縮器全介紹與貼心使用心得

很多人都有一個疑問，其實，為啥會有這麼多的壓縮器還有均衡器？這個問題有點籠統對吧。我能理解他們想問的意思是，為啥同樣都是壓縮器，為啥有這麼多不同的，還有均衡器也是，其實原理不都是差不多嘛？

黑人？號 均衡器？？？哪個好？？？

壓縮器來來去去，主要也就是 閾值，壓縮比，attack，release，makeup gain，甚至有的沒有這麼多，就像LA-2A，只有一個 input和一個output。那為啥有這麼多不一樣的呢？如果你看過上一篇章，就應該瞭解，不同的壓縮器其實會針對不同的情況來應用的，或者說，同一個情況，用不同的壓縮器，也會得到有一些些不同的結果。當然，完全相同參數的情況下，其實壓縮的結果是差不多的，不過會有很多細節的差別。例如 插件的最後染色不同，純算法的可能會更乾淨，等等。

那麼均衡器的話，也是一樣，但是均衡器的話，主要是有圖示均衡器和參量均衡器的區別，另外，均衡器本身也有analog染色的問題，以及操作性的問題。所以，即便是均衡器，一樣也是根據自己的需要，來選擇適合的均衡器使用。

那麼本篇正式開始了

這一次，我也同樣挑出18個均衡器來講解，其中一個是壓縮器。為啥有個內奸？呆會講到了我們再說。那麼均衡器的原理什麼的估計也不用多囉嗦了吧，這個玩意兒也叫等化器，其實主要就是針對聲音的不同頻率的增益進行調整。

在混音中，均衡器 （EQ：equalizer）的應用，我個人認為：主要是有三個。

第一個是，修復性均衡，就是頻率成分太多的，影響聲音的正常狀態的，需要調整。這個一般以衰減為主，就是把多餘的弄小，也包括高低切濾波，去掉不要的成分。

第二個是，調節性均衡，就是已經聽起來差不多了，但是放到完整音樂中又覺得不足的，那這個就需要提升或者衰減來調整，以便適應音樂，以及準確定位該聲音。

第三個是，藝術處理均衡，這個就是基於上兩個都做好以後，自己主觀認為這個聲音的哪些頻率還要做些處理，讓他聽起來更接近自己主觀喜歡的感覺。通常也是提升衰減都可以。

那麼我們接下來看第一個。

第一個，Waves的元老 Q10 equalizer

Waves Audio由Gilad Keren和Meir Sha'ashua於1992年創立。同年晚些時候，Waves發佈了第一款產品Q10 Paragraphic Equalizer。Q10是音頻行業第一款商用音頻插件。在25週年的時候，給重新換了一個界面，相對更加友好。

元老級 Q10 的界面

Q10可以說是陪伴了我們這一代的混音師整整20年了。我第一次用waves就是特別喜歡Q10和C1(壓縮器)，功能不用多說了吧，曲線方式可選，±Gain，兩頭無極掃頻，頻點隨便跑，超大的Q值可調範圍。簡直就是EQ中的典範，最多可調節點10個，在過去可以說只有多餘，不會不夠用，即便現在也經常有這種感覺。Q10其實還帶有好幾個插件，包括Q2 Q4 Q6 Q8。其實就是通道數量，最多是Q10。Q4，那就是4個節點。

25週年紀念版界面

全新25週年紀念版還增添了這幾個功能：比例Q值濾波器、可選頻段控制、雙倍精度處理。可以說直接如虎添翼。

Q10是 mix非常全功能的EQ了。我現在主要是用來作為修復性EQ使用，非常精準。

第二個，GEQ，圖示均衡器的樣式典範

為啥說是樣式典範呢？因為30通道的圖示均衡器真的不管是軟硬件，都是那麼經典，需要的頻段全部都分好了。曲線你怎麼畫就怎麼得到結果。這個是圖示均衡器的特點，即便沒有示波器，看推子都大概知道均衡線的樣子。這個均衡有 classic 和 modern 兩個模式，下圖都有。

兩個模式的區別是，classic用的是比例型Q值濾波器，即在均衡量增大時濾波器的帶寬將會變窄，modern的是頻率無交疊的平頂濾波器，避免了頻段間重疊帶來的相互影響，從而能夠提供平直的階梯式的頻率響應。

除逐個設置各頻段的均衡量外，你也可直接在觸摸屏上或用鼠標畫出所需的均衡曲線。此外，GEQ內置的實時分析儀也可以方便地對兩個聲源進行差異比較。而GEQ實際上主要是用在現場比較適合，尤其是使用waves的服務器系統，可以觸屏直接調節。在現場調節簡直就是超級方便。不過呢，EQ生來平等，即便是在混音中，也是一樣出色。

第三個，API 系列的三兄弟

雙胞胎之一的API 550A均衡器基於60年代後期的API傳奇產品進行建模，在相互交叉重疊的三個區域中各提供了15個頻點，以5步遞進式進行增益調節。它的高頻和低頻區域可以根據使用需要選擇鍾型或者擱架式均衡，同時帶通濾波器也可以獨立應用到參數設定當中。

API 550A 和 API 550B

雙胞胎之二的API 550B以四段交叉頻點而聞名，也是解決問題和潤色聲音的利器。它的每段均衡中都有7個可選頻點，最大能跨越5個倍頻程。550B的“比例Q值”功能能夠自動在較小設定時以更寬的Q值濾波，或在大範圍提升或衰減時以更窄的Q值進行工作。它甚至還能夠讓你撤銷先前的處理，影響甚至逆轉對聲音的已有改變。

API-560

雙胞胎另外還有一個兄弟 API-560 基於1967年的經典硬件建模，跟550B一樣有“比例Q值”功能，作為圖示EQ，雖然沒有GEQ那麼多頻段，但是作為快速處理，絕對足夠。

我在處理要快速解決，並且要求並不是非常高的商業mix，就非常喜歡API的這種EQ，因為數值是定死的，不是無極擰，那麼其實更加容易得到想要的差不多的結果，不至於會一直糾結是100還是101這樣的問題。

第四個，Waves EQ中的心外科手術醫生---F6

F6可以說只有一點比不上Q10的就是少了4個通道。但是卻在功能上碾壓Q10。其操作以及調節的精準程度，如果說Q10是普通的外科手術，那F6做的就是心外科手術或者是腦科手術。它擁有6層參數完全可調的浮動濾波處理頻帶，每層頻帶內都帶有先進的均衡器和壓縮/擴展器控制，同時還包括中央/側向（Mid-Side）處理選項等等。可以說對於聲音的均衡調節非常非常細膩。

帶上了壓縮器和擴展器，意味著，某些頻段的聲音，除了增減增益，還可以讓其穩定或者根據閾值以及比值自動調節。還有MS制式的處理，如果一般處理算是平面的，那麼MS制式可以說是在三維空間中處理EQ了。

不用多說，我如果要處理細緻的聲音，或者要處理很棘手的某些頻段穩定卻又在某些頻段動態過分的聲音，一定選F6就沒錯了。

第五個，這個是奸細，他不是EQ，C4和C6多段壓縮器

這個其實在壓縮器篇章已經詳細講解過了。就不多講了。其實一定程度上來說，也可以作為一個4段EQ來用，而且還跟F6一樣，帶有特定頻段壓縮處理。可以說，是F6的一個閹割版吧。。反正對於頻段如果動態不對，那麼可以通過C4C6來處理。就像我老師說道的，哪裡不對，就調哪裡，壓住，就乖乖了。

第六個，文藝復興系列的 EQ

他跟之前壓縮講到的文藝復興的壓縮和人聲壓縮是一套的，這個系列主要的特點是聲音有老舊的味道，非常溫暖。作為算法效果器，模擬出物理建模的感覺，也算是相當牛逼的了。調節方面其實跟Q10非常相似，有2段，4段和6段選擇。

這個EQ可以說是中規中矩了。不過有能帶來溫暖染色，所以做老舊風格挺適合的。關鍵是跟其他文藝復興系列一樣，簡便。

第七個，調音臺經典之作 VEQ3 和 VEQ4

VEQ4是基於廣受歡迎的1081調音臺模塊建模的經典四頻帶均衡器，有著1081調音臺特別染色的感覺，我覺得對比文藝復興那種溫暖，這個才更加像機器的染色。調節方面掃頻有些限制，類似於API550的操作。也好，非常便利，畢竟選出來的頻點都是非常接近我們需要的，增益是無極擰。也有VU去看輸出量。

VEQ4

而V-EQ3，則是結合了兩款最讓世人渴求的調音臺模塊的所有最優特質——即1066和1073調音臺，這款三段均衡器，有著經典的可調截至頻率的高通濾波器，對混音和母帶都是非常理想。使用V-EQ3能以最輕鬆的方式獲得1960和1970年代那飽滿經典的聲音，有人說絕殺文藝復興的感覺哈。不過我覺得這個不能這麼比較，看音樂的風格需要吧。

VEQ3

總之，VEQ3和VEQ4，都是經典復刻，對於一些經典風格重現是很好的選擇。我也比較喜歡用於bass以及鋼琴的調節，或者是地鼓和軍鼓的調節。

第八個，兩個省吃儉用的，數字均衡 EMO

他們一個是均衡emo-q4，一個是濾波器emo-f2，主要是非常的省資源，非常不佔用CPU。作為現場或者直播，或者錄屏，都是很不錯的選擇，延遲超低。作為均衡的調節，其實無非就是頻率，增益，Q值，曲線。那麼EMO還能給你分開左右處理，可以區分開左右聲道的均衡。那還是相當厲害的了。

EMO-Q4

尤其是作為現場使用，非常的直觀得到我們要的效果，也很容易調節，尤其是使用觸屏系統的話，操作非常人性。像我同事說的，該有的都有了。

EMO-F2

第九個 Kramer HIS 英倫風味

這個跟Kramer PIE 是一套的，也是非常英倫風的經典之作，很好地再現了Helios調音臺的倫敦風味。並且帶有話放失真模擬，英國味的高頻臨場，肥厚的低頻增益。有點老滾石樂隊的味道。我的話主要是用這個來做一些比較俏皮風的或者英倫或者rnb類的作品用的多，當然其實也可以用的，看心情吧。

第十個，SSL調音臺中的現代經典 SSL-G EQ

建模自SSL G系列硬件EQ292簡單易用。沒有任何可視化的東西，純粹憑藉耳朵聽到的調。有一個低切，還有一個analog的開關，想要保留ssl的染色，就可以打開。

這個EQ我一般是用於藝術處理了。又要染色，又要調下EQ，就可以考慮，尤其是工程裡面其他地方也都用上SSL的東西的話。畢竟是G，那基本上如果工程中決定了用G的染色，那麼就可以考慮用這個。或者想單獨對某個東西進行染色也行，反正藝術處理嘛。

第十一個，基於 NEVE1073的建模 Scheps-73

基於經典的1073調音臺的均衡器及話筒前置放大器模塊建模而成，具有相互交叉重疊的頻段，忠於1073調音臺的原始參數並精確模擬它的運行狀態：固定的12kHz高頻段，可選並帶有衰減和增益控制的低、中頻段以及一個每倍頻程18dB衰減的高通濾波器。另外，在原始1073設計圖中發現的只在稀少的1078通道條上可操作的10kHz頻點，也有史以來第一次被建模並融合到插件之中。

一直以來，我都覺得NEVE的那種溫暖是一種失真感，用於搖滾或者比較強烈的風格特別搭配。尤其是1073這種經典臺子簡直就是為搖滾風格而生的感覺。所以失真吉他和bass，絕對可以考慮用他，聲音會變得更加飽滿以及有力量感。

第十二個 Abbey Road TG Mastering Chain

這個機子其實是一臺母帶處理器，以EMI TG12410母帶控制檯為原型建模——從1970年代初至今，它在Abbey Road錄音棚所有的母帶製作中都不可或缺。有著經典的 Abbey Road的染色，以及模塊化處理，其中包含有均衡器。

不多介紹，這個插件主要是用於bus處理，以及母帶處理。參數調節都是不用特別大動作就可以得到滿意的效果。

第十三個，PuigTec EQs

這個是包含兩個均衡器，有側重中頻的PuigTec MEQ-5和寬頻段的PuigTec EQP-1A，

當然，一看就知道，是基於硬件物理建模的插件，這套插件強力點是，可以在同一個頻點進行增益和衰減，從而產生一個共振擱架。如果你覺得聲音某個頻點想提升，但是提了又覺得滿，不提又不夠，而且還不是度的問題，那麼同頻點再衰減形成的共振擱架就能讓你得到想要的效果。很好使的東西。

第十四個，Manny Marroquin EQ

MMEQ是一個套裝裡面的東西，把 Manny Marroquin的個人對於不同設備使用習慣做出了集合，每一個頻率截止點的設置都源自於不同的設備，它們包括Quad 8、Motown、Neve 1073、API 550B、Avalon 2055以及SSL 9080 XL K。簡直就是插件火鍋，但是對於要處理一些快速工程，想立竿見影，是好東西。

第十五個，RS56 Passive EQ

基於傳奇的RS56 Universal Tone Control被動式均衡器建模的插件，它最早在1950年代初期被著名的Abbey Road錄音棚用於唱盤刻制，即現在我們所知的“母帶工程”的一部分。不久之後，Abby Road年輕的工程師們開始將它用於錄音室錄音，因為它具有非常強大的聲音操控能力，遠超當時調音臺上的僅有低頻和高頻控制器的兩段式均衡器。這一特性也讓RS56獲得了“頻響曲線塑形器”的綽號。

強大之處在於左右分開的均衡處理是當時的超先進技術，並且能夠非常容易得到想要的聲音，並且同樣帶有 Abby Road的染色。

第十六個，H-EQ Hybrid Equalizer

H-EQ可以說是 H系列中很厲害的傢伙，其實我覺得他有點類似於F6的物理化版本。它是物理建模一個英式均衡器，卻帶有美式均衡器的元素。吸血鬼也是混血最厲害，這個可以說真的很猛。染色方面跟整個H系列一樣，有多種選擇，真方便的。

功能上，跟F6有的一拼的就是同樣有MS制式調節，可以得到更加立體的聲音。

第十七個，一個萬能工具刀一般的EQ

AudioTrack是一個通道條，其實有點類似於SSL通道條，但是這個是超級省資源的貨，跟emo類似吧。這個是算法通道條，包含有壓縮和均衡和門限。主要是處理一些快速的內容特別好用。

第十八個，母帶上的均衡器 Linear Phase EQ

Linear Phase EQ是一款具有線性相位的均衡器，在保證原始音樂的平衡同時，以絕對零相位偏移對諧波頻譜提供精準的控制。母帶均衡器插件五個用戶自定義頻段，具有線性相位分頻器低頻段組件與全頻段組件最適用於原聲樂器、立體聲素材、高瞬態音頻軌道以及母帶製作九種線性相位濾波器類型。不過嘛，其實要做簡單母帶，有更多好的選擇。不過作為剛剛打開工程，臨時在總線做一個簡單的調節，還是很方面用的。

總結一下

那麼上面基本上已經把waves的獨立使用的均衡器插件都簡單的概括了一遍，那麼大家應該都大概瞭解每個均衡器都有自己的特點了吧。這個畢竟也只是一個概括講解，實際上真要知道是不是適合某個音頻處理，還是得自己慢慢多試試，這個是基本功呢，就跟學樂器練琴一樣。需要時間，也需要精力去做。包括對於不同廠牌的均衡器，也一樣是通過多使用來總結感覺。

其實，最終要用哪個均衡器去處理，都是基於自己要有明確的目的，就是，當你得到一個聲音，你想要把這個聲音做到什麼樣的變化，出來什麼樣的結果，要心裡有數。然後根據這個結果，來推導自己需要哪個均衡器來處理。那麼這樣，你就得知道，不同的均衡器，能做到什麼樣的效果，根據這個自己試驗過的經驗，來判斷哪個均衡器最接近自己要的效果，然後插上去調整。未必一次就中，但是起碼是要有方向去做，而不是胡亂去試。

特別似曾相識的總結對吧？對的，其實均衡器和壓縮器最終要怎麼用，都是一模一樣的，需要自己調試過，記住了效果以後，憑藉經驗去選擇使用。多鍛鍊耳朵吧。

希望新手和愛好者朋友們能夠有所收穫吧。祝大家樂運昌隆！！
