挖礦神器！AMD發佈Project 47超級服務器

2020-10-29
| 科技

近日，AMD Capsaicin 2017大會在美國洛杉磯盛大召開。今年AMD在CPU與GPU產品線可說是非常亮眼，不僅在低、中、高階等產品線都一應俱全，更高效能的產品亦往上進擊服務器產業、專業繪圖領域，2017年可以說是AMD大豐收的一年。

現在常會聽到業界爭辯，到底是CPU好、還是GPU好，兩派陷入對戰模式，AMD認為，兩者本來就各有所長，沒有誰取代誰的問題，應當是兩者合作，才能發揮最大效果。值得慶幸的是，AMD是一家同時擁有GPU和CPU產品的公司，大象的兩條腿已經站穩了。

在Capsaicin 2017大會上，AMD除了公佈大家熱切期待的Ryzen Threadripper處理器以及基於Vega架構一些列遊戲卡、專業卡以外，在最後，AMD CEO蘇姿豐博士親自上臺發佈了AMD Project 47超級服務器，拋出彩蛋引爆全場，因為這可是一臺可以提供1 PetaFLOPS算力的服務器。

AMD Project 47服務器是AMD聯合全球最大服務器製造商Inventec英業達公司開發的，使用了20顆AMD EPYC 7601服務處理器，80張Vega架構Radeon INSTINCT MI25計算卡，整個服務器單精度浮點性能可以達到1 PFLOPS之多。這可是個深度計算、機器學習的大殺器，可以顯著提高計算性能，節省時間。

為了提供高達1 PetaFLOPS的單精度計算能力，2 PetaFLOPS的半精度計算能力，AMD在窄小的2Ux20規模服務器機櫃中，使用了20顆最新發布的AMD EPYC 7601處理器，這也是AMD發佈的最強服務器處理器，32核64線程，默認頻率為2.2GHz,所有核心最高可以boost至2.7GHz,TDP僅僅是180W。

Project 47服務器內含有80張Radeon INSTINCT MI25顯卡，一顆EPYC帶4張顯卡還是蠻輕鬆的。Radeon Instinct MI25採用了新一代Vega架構，擁有64組NCU單元，換算過來就是4096個SP流處理器，配合16GB的HBM2顯存，顯存帶寬高達484 GB/s。每張卡可以提供12.3TFLOPS,80張算起來確實差不多是1PFLOPS的單精度性能，半精度性能直接翻倍，每瓦性能高達30GFLOPS。

Project 47超級服務器將會在今年第四季度由英業達與其它AMAX經銷商共同出售，不過AMD並沒有公佈一臺Project 47服務器的售價。想必也不便宜，光是一顆CPU就要4000美金......
