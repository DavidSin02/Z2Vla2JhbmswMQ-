一、指針的使用

C語言中提供了一個“間接訪問運算符”（也稱間接地址運算符）：“*”，如果指針變量中保存了確切的地址，就可以使用“*”來引用該地址的存儲單元。

1、通過指針來引用存儲單元

（1）指針的自加自減

Int *p, i=100, j;

p=&i;

j=*p;//等價於：j=i=100;

j=*&i

j=*p+1;

幾種常見的指針操作，見圖20-1。

圖20-1，指針的自加自減運算

在重申一下，運算的時候，先看運算符的優先級，順序是高優先級到低優先級，優先級相同的情況下，看結合性順序。

我們通過程序201.C驗證一下我們的分析是否正確！

printf("*p=%d\np=%d\n",*p,p);

（2）指針的間接訪問（**p指針的多層調用）

Int **p,*s, k=20

s=&k;

p=&s;

*(*p)=30;//該語句執行過程見圖20-2

**p執行那過程

例程202.C

 #include <stdio.h>int main(){  int **p,*s, k=20; printf("k=%d\n",k); s=&k; printf("執行s=&k;:\n*s=%d\ns=%d\n",*s,s); p=&s; printf("執行p=&s;:\n*p=%d\np=%d\n",*p,p); **p=30;// printf("執行**p=30;:\n**p=%d\np=%d\nK=%d\n&k=%d\n",**p,p,k,&k); getchar();}

2、指針的移動

指針的移動就是對指針變量加上或減去一個整數，或者通過賦值，使指針指向其他單元，因此只有指針指向一串連續的存儲單元時，指針的移動才有意義。

Int a[5] , *p,*q;

P=&a[0];

P=a;

q=p+2;

p++

q++
