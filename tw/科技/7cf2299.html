多表數據求和（適用於數據合併、報表彙總）

2020-10-29
| 科技

需求：將多張表相同地址（如每個表都是A2等）的單元格數據彙總到文件名為“彙總”的相應位置中（也是A2），同一工作簿有多少張表，彙總多少張。

要求：所有表格的格式必須一致（如行和列），僅數據不一致，否則會錯加。因為程序是加不同表格同址址數據。

PS：彙總的表格都是數值，沒有公式

===============================

Sub 多表求和()

[b2:n21].ClearContents ’清空範圍

Dim arr

For Each Rng In [b2:n21] ‘這是選擇需要求和的數據範圍

ReDim arr(1 To Sheets.Count)

n = 0

For sht = 2 To Sheets.Count

abc = Sheets(sht).Range(Rng.Address)

n = n + 1

arr(n) = abc

Next sht

adj = 0

For i = 1 To UBound(arr)

adj = arr(i) + adj

Next i

Sheets(1).Range(Rng.Address) = adj

Next Rng

End Sub
