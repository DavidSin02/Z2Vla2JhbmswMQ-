Excel中Dialog對象集合是寶庫，它能夠實現對內置對話框的調取

2020-11-14
| 科技

Excel 中的Dialog對象集合是一個值得探索的寶庫，它能夠實現對各種內置對話框的調取，並根據用戶喜好進行個性化的默認參數設置，為編程及前臺應用提供了極大的選擇空間。但長期以來，就此問題進行深入研究的人數寥寥，網絡上也基本搜索不到有價值的信息。根據實際需要，對一些常用的對話框設置進行了示例演示，相信其餘對話框設置大家也基本能夠舉一反三，觸類旁通。在探索過程中，我也有一些困惑和不解，例如參數的含義和參數值的設定等，還需要和大家共同探討解決。

另外要提醒的一點是，幫助中少量對話框參數說明是錯誤的，例如說明中是三個參數，實際是5個參數。幫助中的系統常量說明對理解對話框參數很有幫助，但一些系統常量在參數中也不能用，要自己一點一點摸索。

1、“高級篩選”對話框

application.Dialogs(xlDialogFilterAdvanced).Showarg1:=xlFilterInPlace,arg2:=activesheet.usedrange,arg3:=[criterior]

參數1：在原區域顯示篩選結果。參數2：列表區域。參數3：條件區域

application.Dialogs(xlDialogFilterAdvanced).Showarg1:=xlFilterCopy,arg2:=activesheet.usedrange,arg3:=[criterior],arg4:=sheets(sheets.Count).[a1],arg5:=true

參數1：將篩選出的數據複製到新位置 參數4：目標區域 參數5：選擇不重複記錄

2、“打開”對話框

Application.Dialogs(xlDialogOpen).Showarg1:="*t*.txt" 搜索當前文本夾中文件名中含有t關鍵字的文本文件

Application.Dialogs(xlDialogOpen).Showarg1:="d:\*.txt" 可以改變當前目錄，在D盤搜索文本文件

Application.Dialogs(xlDialogOpen).Showarg1:="d:\*.txt;*.doc;*.xls" 可以搜索D盤的文本文件、WORD文本和EXCEL文件，具體可以自己定義，注意中間用分號間隔。此語句非常實用，仍然記得自己在初次發現時的狂喜心情，隆重推薦！！！

Application.Dialogs(xlDialogOpen).Showarg1:="d:\*1*;*2*" 搜索D盤含有1或2關鍵字的文件，也很實用

application.Dialogs(1).Showarg1:=CreateObject("Wscript.Shell").SpecialFolders("Desktop")& "\*.lnk"

搜索桌面的快捷方式

application.Dialogs(1).Showarg1:=CreateObject("Wscript.Shell").SpecialFolders("favorites")& "\*.*"

搜索收藏夾文件夾

application.Dialogs(1).Showarg1:=CreateObject("Wscript.Shell").SpecialFolders("startmenu")& "\*.*"

搜索開始菜單所有文件

application.Dialogs(1).Showarg1:=CreateObject("Wscript.Shell").SpecialFolders("mydocuments")& "\*.xls;*.doc"

搜索“我的文檔”中的EXCEL及WORD文件

application.Dialogs(1).Show arg1:="d:\??學生*.xls"

搜索D盤中關鍵字為學生的EXCEL文件，這裡兩種通配符都可以使用

此外常用的特殊文件夾還有Startup、Fonts、Programs、Recent、SendTo、StartMenu、Templates

3、“標準字體”對話框

這個對話框更改的不是單元格格式，而是橫縱座標的字體字號等格式，太神奇了！！！，而且這種更改只對當前工作薄有效，即使關閉EXCEL和關機，設置仍然保留。

Application.Dialogs(xlDialogStandardFont).Showarg1:="華文彩雲",arg2:=20,arg3:=true

以上修改字體、字號和粗體，如果設置成各種西方字體，將使工作表畫面呈現千變萬化，充滿情趣的效果，如果用Wingdings等三種字體，簡直就成畫板了！，當然，在遊戲之餘，還是要用莊重醒目的字體比較好。

4、“對齊方式”對話框

application.Dialogs(xlDialogAlignment).Showarg1:=3,arg2:=1 水平居中\自動換行

application.Dialogs(xlDialogAlignment).Showarg1:=2 左對齊

application.Dialogs(xlDialogAlignment).Showarg6:=1 縮小字體填充

application.Dialogs(xlDialogAlignment).Showarg8:=1 合併單元格

根據幫助說明，“對齊方式”對話框共有五個參數，分別為horiz_align, wrap, vert_align, orientation,add_indent，但實際說明並不完整，或者是一個“bug”,“縮小字體填充”和“合併單元格”參數說明中均沒有，實際其分別位於參數6和參數8。這些都是不信邪大膽試的結果，大家在實踐中要引以為鑑。

5、“頁面設置”對話框

Application.Dialogs(7).Showarg9:=true,arg10:=true,arg11:=2方向橫向，水平垂直居中

Application.Dialogs(7).Showarg3:=0,arg4:=0,arg5:=0,arg6:=0,arg18:=0,arg19:=0 頁邊距的六個數值參數均設置為0

Application.Dialogs(7).Show arg12:=9 紙型為A4

Application.Dialogs(7).Show arg13:=80 縮放比例為80%

Application.Dialogs(7).Show arg2:="共" & (ActiveSheet.HPageBreaks.Count +1) * (ActiveSheet.VPageBreaks.Count + 1) & "頁" 頁腳設置為“共*頁”

Application.Dialogs(7).Show arg1:="第1頁"這個是設置頁眉，沒有什麼實際用途，原因是數字是死的，不會變動。無法模擬出自定義頁眉中的類似WORD域的效果。

注意：頁眉頁腳 打印標題行 打印標題等參數不能進行默認設置，參數中也沒有此項，不知道為什麼。

6、“打印”對話框

Application.Dialogs(xlDialogPrint).Showarg4:=5 打印4份

application.Dialogs(8).Show arg2:=1,arg3:=2 打印1－2頁

application.Dialogs(8).Show arg12:=1 打印選定區域。Arg12其他選項：2活動工作表 3整個工作薄

application.Dialogs(132).Show arg1:=2,arg2:=3定位常量－公式文本

application.Dialogs(132).Showarg1:=2,arg2:=23 定位常量－全部

application.Dialogs(132).Show arg1:=4 定位空值

application.Dialogs(132).Show arg1:=5 定位當前區域

application.Dialogs(132).Show arg1:=6 定位當前數組

application.Dialogs(132).Show arg1:=7 定位行內容差異單元格

application.Dialogs(132).Show arg1:=8 定位列內容差異單元格

application.Dialogs(132).Show arg1:=9 定位引用單元格

application.Dialogs(132).Show arg1:=10 定位從屬單元格

application.Dialogs(132).Showarg1:=10,arg3:=2 定位所有級別從屬單元格

application.Dialogs(132).Show arg1:=11 定位最後一個單元格

application.Dialogs(132).Show arg1:=12 定位可見單元格

application.Dialogs(132).Show arg1:=13 定位對象

application.Dialogs(132).Show arg1:=14 定位條件格式

application.Dialogs(132).Show arg1:=15 定位全部數據有效性，在幫助“Range.SpecialCells 方法”中，定位有效性的類型值為-4174，用在定位對話框參數中就會出錯。arg1:=15這種結果全是一點點摸索出來的，諸如此類的情況很多。大家用到其它對話框時往往要自己反覆嘗試，不要輕易放棄和否定。

application.Dialogs(132).Showarg1:=15,arg3:=2 定位相同數據有效性

8、“分列”對話框

10、選擇性粘貼對話框

application.Dialogs(53).Show arg1:=7 邊框除外

application.Dialogs(53).Show arg1:=8 粘貼複製的列寬

application.Dialogs(53).Show arg1:=11 公式和數字格式

application.Dialogs(53).Show arg1:=6 粘貼有效性

application.Dialogs(53).Showarg1:=2,arg4:=true 粘貼公式/轉置

arg1參數的其他選項值：3數值 4格式5批註 12粘貼值和數字格式

注意以上語句運行前，要有單元格處於被複制狀態，否則運行出錯。也可以用以下宏增強通用性，防止出錯。

Public Sub abc1()

If Application.CutCopyMode = xlCopy Then

Application.Dialogs(53).Show arg1:=2, arg4:=True

End If

End Sub

13、“另存為”對話框

application.Dialogs(xlDialogSaveAs).Showarg1:=CreateObject("Wscript.Shell").SpecialFolders("desktop")& "\" & [a1] & ".xls"

保存文件到桌面，默認文件名為A1單元格值。

還可以更智能一些，如果A1為空，則以A列第一個值為默認文件名

Public Sub abc()

Dim folder, name

folder =CreateObject("Wscript.Shell").SpecialFolders("desktop")& "\"

If [a1] <> "" Then name =[a1] Else name = [a:a].Find("*")

Application.Dialogs(xlDialogSaveAs).Showarg1:=folder & name & ".xls"

End Sub

14、“創建分類彙總”對話框

application.Dialogs(xlDialogSubtotalCreate).Showarg2:=2 分類彙總函數為計數，其他參數：平均值為3，最大值為4，最小值為5，數值計數為7標準偏差為9

application.Dialogs(xlDialogSubtotalCreate).Showarg4:=false 替換當前分類彙總為否

application.Dialogs(xlDialogSubtotalCreate).Showarg5:=true 每組數據分頁為是

application.Dialogs(xlDialogSubtotalCreate).Showarg6:=false 彙總結果顯示在下方選項為否，arg6:=true則是結果顯示在上方

application.Dialogs(xlDialogSubtotalCreate).Showarg2:=2,arg3:=1 對第一列進行計數

application.Dialogs(xlDialogSubtotalCreate).Showarg3:=array(2,3) 對第二三列進行求和

由於分類彙總對話框不能準確確定要進行計算的列，通過代碼可以彌補這一不足。以下的程序即可以實現以上功能。假定數據表第一列為字段名，第二列以上為數據記錄，通過判斷數據為數值型確定要計算的列數組。分類彙總對話框則根據數據自動標識要求和的列

Public Sub abc()

Dim ar(), i, m

For i = 2 ToRange("a1").CurrentRegion.Columns.Count

IfIsNumeric(Cells(2, i)) Then

m = m + 1

ReDim Preserve ar(1 To m)

ar(m) = i

EndIf

Next

Application.Dialogs(xlDialogSubtotalCreate).Showarg3:=ar

End Sub

15、“添加列表選項”對話框

application.Dialogs(xlDialogOptionsListsAdd).Show

EXCEL各版本進入自定義序列的路徑都比較繁瑣，用以上語句可以直接進入，相當於快捷方式了。

16、“計算”對話框

application.Dialogs(xlDialogCalculation).Showarg6:=true 以顯示精度為準（這個比較有用，對一個小數，設置以顯示精度為準，則減少小數位後就不可恢復了。相當於編輯小數值，而不是格式調整）

application.Dialogs(32).Show arg1:=3 設置計算模式為“手動重算”，參數值1為“自動重算”

application.Dialogs(32).Showarg1:=3,arg8:=false 保存前自動重算選項為否，這樣可以加快文件保存速度。注意arg8和arg1必須同時設置才有效

這個對話框參數不少，但以前面三種應用最為常用。

17、“單元格保護”對話框

application.Dialogs(xlDialogCellProtection).Showarg1:=true,arg2:=true 設置單元格保護，鎖定和隱藏兩個選項均為是

18、“清除”對話框

application.Dialogs(xlDialogClear).Showarg1:=2 將清除格式設為默認選項，因為這一選項是最常用的

application.Dialogs(xlDialogClear).Showarg1:=iif(not selection(1).Comment is nothing,4,2)

加入一種更“人性化”的判斷，如果第一個選定單元格中含有批註，則清除默認選項為“批註”，對有類似要求的用戶這是一個較好的參考，但也很可能畫蛇添足，多此一舉，自己靈活把握吧

Application.Dialogs(xlDialogClear).Showarg1:=5將清除超鏈接設為默認選項。對此還可以用一組宏語句實現更智能化的應用，如果單元格存在超鏈接，則將清除超鏈接設為默認選項，否則將清除格式設為默認選項。代碼如下：

Sub abc()

On Error Resume Next

Debug.PrintSelection(1).Hyperlinks(1).Address

If Err.Number > 0 Then

Application.Dialogs(xlDialogClear).Show arg1:=2

Else

Application.Dialogs(xlDialogClear).Show arg1:=5

End If

End Sub

19、合併計算

application.Dialogs(xlDialogConsolidate).Showarg1:=Array("Sheet1!R1C1:R4C2","Sheet2!R1C1:R5C2") 設定合併計算的所有引用位置，arg1的參數值是一個數組，包含各個參與合併的區域。如果自動準確添加所有引用位置，則需要用宏來實現。例如：

PublicSub abc()

Dimar(), i, m

For i= 1 To ActiveSheet.Index - 1

m = m+ 1

ReDim Preserve ar(1 To m)

ar(m) = Sheets(i).Name & "!"& Sheets(i).Range("a1").CurrentRegion.Address(ReferenceStyle:=xlR1C1)

Next

[a1].Select

Application.Dialogs(xlDialogConsolidate).Showarg1:=ar

EndSub

application.Dialogs(xlDialogConsolidate).Showarg2:=3 設置合併計算的函數為計數，其它參數值：1平均值，2數值計數，4最大值，5最小值，6乘積，9求和

application.Dialogs(xlDialogConsolidate).Showarg3:=true,arg4:=true 標籤位置設定為首行為最左列

application.Dialogs(xlDialogConsolidate).Showarg5:=true 選定“創建指向源數據的鏈接”選項
