配置通過Console口登錄設備示例

組網需求

當用戶無法進行遠程登錄設備時，可通過Console口進行本地登錄。使用Console用戶界面登錄設備時只需要密碼認證，為了防止非法用戶登錄設備，修改Console用戶界面的認證方式為AAA認證。

配置通過Console口登錄設備組網圖

配置思路

採用如下的思路配置通過Console口登錄設備：

1、使用終端仿真軟件通過Console口登錄設備。

2、配置Console用戶界面的認證方式。

說明：

如果PC使用系統自帶的終端仿真軟件（如Windows xp系統的超級終端），則無需另行準備；如果系統不帶終端仿真軟件，請您準備第三方終端仿真軟件，使用方法請參照該軟件的使用指導或聯機幫助。

操作步驟

使用終端仿真軟件通過Console口登錄設備。

請使用產品隨機附帶的Console通信電纜的DB9（孔）插頭插入PC機的9芯（針）串口插座，再將RJ-45插頭端插入設備的Console口中.

在PC上打開終端仿真軟件，新建連接，設置連接的接口以及通信參數。

說明：

因為PC端可能會存在多個連接接口，這裡需要選擇的是連接Console線纜的那個接口。一般情況下，選擇的接口是COM1。

若修改了設備的串口通信參數值，需要在PC端更換通信參數值與設備的串口通信參數值一致後，重新連接。

按Enter鍵，直到系統出現如下顯示，提示用戶輸入密碼。（AAA認證時，提示輸入用戶名和密碼，以下顯示信息僅為示意）

Login authentication

Password:

進入設備後，用戶可以鍵入命令，對設備進行配置，需要幫助可以隨時鍵入“?”。

配置Console用戶界面的認證方式

<HUAWEI> system-view

[HUAWEI] user-interface console 0

[HUAWEI-ui-console0] authentication-mode aaa

[HUAWEI-ui-console0] user privilege level 15

[HUAWEI-ui-console0] quit

[HUAWEI] aaa

[HUAWEI-aaa] local-user admin1234 password irreversible-cipher 023wg.com

[HUAWEI-aaa] local-user admin1234 privilege level 3

[HUAWEI-aaa] local-user admin1234 service-type terminal

執行以上操作後，用戶使用Console用戶界面重新登錄設備時，需要輸入用戶名admin1234，認證密碼023wg.com才能通過身份驗證，成功登錄設備。

配置文件

#

aaa

local-user admin1234 password irreversible-cipher %@%@HW=5%Mr;:2)/RX$FnU1HLO%-TBMp4wn%;~\#%iAut}_~O%0L%@%@

local-user admin1234 privilege level 3

local-user admin1234 service-type terminal

#

user-interface con 0

authentication-mode aaa

user privilege level 15

#

return
