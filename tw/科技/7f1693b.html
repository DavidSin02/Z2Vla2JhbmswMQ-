如何購買紫外/可見分光光度計

2020-10-29
| 科技

紫外/可見分光光度法是用於定性和定量分析的常用技術，適用於廣泛的應用。尋求新型紫外/可見分光光度計的實驗室有多種選擇，從最簡單的單波長儀器到高性能，多光譜分析儀。本指南將重點介紹購買UV / Vis分光光度計的重要注意事項。

1. 關鍵的問題

為了充分利用您的購買力，在與製造商交談之前，您應該問自己和您的同事以下問題：

1. 你的預算是多少？在現實的預算範圍內理解和工作非常重要。

2. 您將使用該儀器的哪些應用程序？考慮最適合您需求的硬件和軟件。不要忘記考慮您可能需要該儀器的任何未來應用，或實驗室中其他人可能希望使用該儀器的方式。

3. 是否可以升級系統或添加其他模塊？同樣，請考慮您可能需要此儀器的任何未來應用程序。購買可用於執行其他類型分析的系統是否有用？

4. 您是否正在更換儀器請與他人一起了解舊儀器的優點和侷限性。這將幫助您確定下次購買時哪些功能至關重要或可取。

5. 速度和效率有多重要？哪些功能或技術可以簡化您的工作流程併為用戶節省時間？系統如何集成到您當前的工作流程中？請記住考慮您未來的目標，例如引入液體處理機器人或增加樣品量。

6. 您需要什麼服務，維護和支持？找出儀器需要的維護類型和頻率。確定您需要哪種類型的維修包。

7. 該軟件用戶友好嗎？該儀器是否需要被許多非專家用戶或少數超級用戶使用？

8. 該軟件是否與您實驗室中的其他設備兼容？考慮要使用哪些平臺來查看數據。分光光度計的操作系統是否與Apple，Android或Microsoft設備兼容。

2. 基本概念

UV / Vis是指電磁波譜的紫外（UV）和可見（Vis）部分。應用光譜學命名聯合委員會將遠紫外區域設定在10-200nm，在200-380nm附近的紫外線和380-780nm處可見。現代UV / Vis分光光度計通常具有190nm至1100nm的波長範圍。當一束電磁輻射照射到物體上時，它可以被吸收，透射，散射，反射，或者它可以引起熒光。UV / Vis光譜學中涉及的過程是吸收和透射。當紫外線和可見光輻射與物質相互作用時，就會發生電子躍遷; 也就是說，處於基態的電子被促進到高能態。特別地，在UV / Vis區域中發生π至π*和n至π*躍遷。

3. 注意事項

紫外/可見分光光度法可用於許多學科的廣泛應用，包括生命科學，製藥，食品和飲料，環境和材料科學。在確定您需要的規格和功能時，必須仔細考慮您可能需要使用紫外/可見分光光度計的各種應用，而不僅僅是現在，而是在未來幾年。需要考慮的重要因素包括：

（1）分析物：您希望檢測和量化的分析物的最大吸收波長將決定分光光度計所需的波長範圍。許多紫外/可見分光光度計專為生命科學和藥物發現應用而設計，涵蓋一系列紫外/可見光譜，通常為190nm - 840nm。然而，一些專為材料和環境應用而設計的儀器具有更寬的波長範圍。

（2）待分析樣品的體積：那些使用少量珍貴樣品（如DNA或蛋白質）的人將受益於微量分光光度計的投資，該分光光度計可以分析低至0.5μL的樣品。一些分光光度計還可以容納大型比色皿，以促進其他吸光度和熒光方法。

（3）軟件和可用性：考慮用戶界面和包含軟件功能，如預定義方法，數據分析和汙染警報。這可以增強易用性，提高工作流程效率並減少培訓需求。

（4）受監管的行業：那些在受監管行業（如食品和飲料製造業或製藥行業）工作的人員將要求儀器符合行業標準，並且通常具有先進的質量控制和安全功能。

（5）未來的應用：您是否預見到任何未來的應用需求需要更大的光譜範圍，不同的樣品體積，額外的軟件功能或能夠執行不同的分光光度分析，如熒光？

3.1生命科學

在生命科學中，UV / Vis吸光度分光光度法通常用於快速且容易地確定核酸濃度並評估樣品純度。在蛋白質生物學中，它可用於直接或通過Bradford測定法定量蛋白質。UV / Vis也可用於監測細胞和微生物的生長，以及ELISA和動力學測定。您應該考慮的重要功能：

（1）微量樣品

（2）預先設計和可定製的應用程序 - 能夠選擇常用方法，如Bradford分析，酶動力學和核酸定量

（3）溫度控制

3.2受管制的工業和應用化學

紫外/可見分光光度法可應用於廣泛的應用，包括：

（1）藥物發現和藥物：藥物配方，質量控制，藥物開發和交付的定量分析。

（2）材料科學：分析一系列材料，包括玻璃，金屬和塗料，作為材料開發和質量控制的一部分。

（3）食品和飲料：成分的定量測定和飲用水分析。

（4）環境分析：廢物和環境水中痕量分析物的分析。

（5）臨床診斷和醫學研究：許多臨床分析，並越來越多地用於軟組織的非侵入性分析。

4. 儀器組件的注意事項

V / Vis分光光度計的不同組件有助於儀器的整體性能。紫外/可見分光光度計由五個基本組成部分組成：

· 光源 - 提供適當波長的輻射

· 樣品室 - 將樣品引入光束的區域

· 單色器 - 產生一束單色光; 在傳統的UV / Vis配置中，它包括入口狹縫，準直裝置，分散裝置，聚焦透鏡或鏡子，以及出口狹縫

· 探測器 - 探測並測量輻射強度

· 信號處理和測量系統 - 處理數據並控制儀器

4.1樣本格式

通過UV / Vis分析的大多數樣品處於液態或溶液態。傳統的樣品細胞包括比色皿，吸管（一種自動用樣品溶液填充細胞的附件）和微量滴定板，例如與Epoch™2兼容的那些。微量體積分光光度計，如DS-11 FX ，使用微量體積平臺，樣品溶液可以直接移液到其上。通常，樣品體積在0.5至2μL的範圍內。有些儀器配有光纖探頭，用於測量紫外/可見分光光度計樣品室外的樣品。這使得能夠原位分析樣品，這在不可能物理地移除樣品時尤其有用，例如，當監測工業生產線，血液或環境時。另一個考慮因素是光程。微量體積分光光度計具有非常短的光程，通常在0.1至1mm之間，這允許在不需要稀釋的情況下分析高濃度樣品。圖12顯示了SmartPath的視頻®採用DeNovix的BridgeTesting™技術，可以在0.74 ng /μL至37,500 ng /μLdsDNA範圍內精確分析樣品的微量液滴。在光譜的另一端，一些儀器適應非常長的光程。例如，除了微量樣品，一些儀器可以容納比色皿，提高靈敏度並允許檢測痕量水平的分析物。還應考慮通過將微量體積和比色皿吸光度與熒光測量結合來提供額外測定靈活性的儀器

提示：如果使用比色皿作為樣品池，重要的是要考慮比色皿材料。常見材料包括光學玻璃，石英和藍寶石。材料的傳輸範圍需要與分析物的波長範圍相匹配; 具有更寬透射範圍的材料，例如藍寶石，成本更高。

4.2光源

在測量期間光源應該是穩定的。也就是說，發射的輻射強度不應該波動，並且在儘可能大的波長區域上應該有足夠的強度。理想的光源可以在所有波長上產生恆定的強度，具有低噪聲和長期穩定性。表2列出了紫外/可見分光光度計的光源。不同的來源並不相同; 它們在光譜的不同部分提供光強度和噪聲。

許多紫外/可見分光光度計使用氘燈和滷鎢燈來覆蓋整個紫外線（氘燈）和可見（滷鎢燈）光譜。源選擇器用於在燈之間切換，或者來自兩個源的光被混合以產生單個寬帶源。這種類型的設置稱為掃描。氙氣閃光燈正成為生命科學應用中最常用的光源，因為它們覆蓋了整個紫外和可見光範圍，延長了使用壽命，不需要預熱時間，也不會升高樣品室的溫度。一些儀器中使用發光二極管（LED）作為簡單應用的低成本解決方案，因為燈的壽命幾乎是無限的。在這種情況下，設置稱為數組。

4.3軟件和數據管理

UV / Vis軟件是一個不斷髮展和擴展的領域。大多數分光光度計現在都包含自己的軟件，用於控制儀器，收集，分析和處理數據。許多軟件包現在都帶有預編程方法，使用戶只需點擊幾下即可執行常規分析並計算常用參數。直觀的用戶界面和軟件相結合，可以使偶爾的非專業用戶更容易訪問儀器，並加快常規分析。例如，許多適用於生命科學的儀器包括用於核酸定量的預編程應用，蛋白質和肽定量，動力學測定和光密度以及細胞/ mL計算。

示例包括：DeNovix Inc.的分光光度計，如DS-11 FX ，包括功能強大且直觀的EasyApps®軟件，使儀器易於學習，使用快捷，無需設置。該軟件由生命科學家設計，包括SmartQC™，用於評估汙染，監測光譜異常，並確保用戶在以前的用戶未能刪除以前的樣本時收到警報。

UV / VIS Excellence分光光度計可與LabX PC軟件配合使用。該軟件支持光譜數據管理，完全符合21 CFR part 11 / EU附件11.它還包括一個應用程序庫，可以集成到自動化多參數系統中。

標配Gen5酶標儀和成像軟件從BioTek的儀器公司提供的板載或PC上，並與所有BioTek的成像和檢測系統，如集成大紀元™2微孔板分光光度計。

Thermo Scientific™NanoDrop™One提供的Acclaro™樣品智能技術可以調用光譜庫來識別常見的DNA樣品汙染物，如苯酚和蛋白質，並對其進行預防。

有些儀器允許使用觸摸屏用戶界面進行獨立操作，如DeNovix的DS-11 FX ，操作簡單高效。像EasyApps軟件®，使用自定義的Android™操作系統易於使用的，圖標式界面，需要很多的已經很熟悉當今許多研究者的觸摸屏導航功能的優勢。查詢EasyApps的主要功能亮點®在這個軟件 應用筆記。

4.4連通性

易於數據導出對於獨立儀器非常重要。現在，許多型號都配備了USB，WiFi和以太網，在組織實驗室空間和更輕鬆地訪問數據時為您提供更多自由。某些平臺還具有應用程序，例如Denovix的數據應用程序，使用戶能夠將數據作為csv或jpeg文件發送，將數據保存到LIMS或網絡驅動器，或將數據打印到網絡打印機或標籤編寫者。考慮在儀器上設置用戶帳戶的能力也很重要，這些工具允許個人用戶保存他們選擇的預配置方法，提高結果的速度，甚至非專業用戶也能輕鬆使用。

4.5單色器

理想的單色器應該產生單色光。然而，在實踐中，輸出始終是帶，其形狀最佳對稱。單色儀中的分散裝置可以是稜鏡或衍射光柵。大多數現代分光光度計都包含全息光柵而不是稜鏡。

4.6探測器

理想情況下，探測器應在很寬的範圍內提供線性響應，噪聲低，靈敏度高。表3顯示了UV / Vis分光光度計中使用的不同類型的檢測器。光電倍增管（PMT）和光電二極管是單通道檢測器，是目前市場上最常用的儀器。光電二極管通常存在於低端儀器中，而PMT則用於高端儀器（研究級）。光電二極管陣列（PDA）和電荷耦合器件（CCD）是多通道檢測器。它們允許快速採集整個光譜，並且由於它們具有較少的移動部件，因此更加堅固。但是，它們不如PMT敏感。

5. 光學配置/光學設計的考慮因素

市場上有幾種用於紫外/可見分光光度計的光學配置，如表4所示。單光束配置是最早的設計，仍然普遍使用，特別是在低端儀器中。雙光束和雙光束分光光度計測量光強度的比率，因此對光源或檢測器的波動不敏感。分束光束類似於雙光束分光光度計，但使用分束器代替斬波器，並使用兩個獨立但相同的探測器。單光束，雙光束和分光束是傳統的紫外/可見分光光度計。在傳統系統中，來自光源的多色光聚焦在單色器的入口狹縫上，單色器選擇性地透射窄帶光。然後該光通過樣品區域到達檢測器。在多通道紫外/可見分光光度計中，來自光源的多色光通過樣品區域並聚焦在多色儀的入口狹縫上，將多個光分散到二極管陣列上，每個二極管測量一個窄帶的光譜。這種二極管陣列，稱為光電二極管陣列檢測器（PDA），通常由每個不同波長的1,024個光電管的線性陣列組成。

單光束

  • 一束光進行測量
  • 最簡單的配置
  • 示例：來自DeNovix的DS-11 FX 或來自Metter-Toledo的UV / VIS Excellence系列

雙光束

  • 通過斬波器（一個探測器）將光分成兩個不同的路徑
  • 更復雜的光學系統：成本更高，穩定性更好，靈敏度更低
  • 示例：水瓶™範圍從絲絲儀器使用雙光束光學系統，以提供一個完全對稱的頻帶，可實現高精度的測量。

分裂光束

  • 通過分離器將光分成兩個不同的路徑，一個通過樣品; 另一個用作參考
  • 更復雜的光學系統：成本更高，穩定性更好，靈敏度更低

基於多通道/陣列

  • 來自光源的所有波長都通過樣品; 穿過樣品的光被衍射光柵分散; 分離的波長落在陣列檢測器的不同像素上
  • 例如：卡里8454紫外-可見光二極管陣列系統由安捷倫科技

基於多通道/陣列的儀器的優點包括：

更高的堅固性 - 需要更少的移動部件，因此需要更少的維護

更快 - 全光譜掃描速度; 有些儀器可以在短短1秒內完成全譜掃描。

6. 績效標準的考慮因素

購買新的紫外/可見分光光度計時需要考慮幾個標準，但波長準確度，波長再現性和噪聲水平尤為重要。

6.1波長準確度

儀器在整個UV和可見光範圍內在一系列波長處產生明確定義的峰的能力。當您想要比較不同儀器之間的結果時，波長準確度是最重要的。波長的偏差會導致定量和定性結果的誤差。通常使用經過認證的參考標準來檢查波長精度。

提示：所需的波長精度水平取決於您的應用。UV / Vis儀器的典型波長精度約為±1nm，但有些儀器的精度達到±0.5nm。

6.2波長再現性

儀器在重複讀數上產生相同波長的能力，波長再現性是儀器在重複讀數上產生相同波長的能力。這很重要，因為它允許將一個讀數與另一個讀數準確比較，例如將樣品與空白或一個樣品與另一個樣品進行比較。

6.3光度（吸光度）精度

光度準確度定義為儀器測量吸光度的準確度。這由測量的吸光度與建立的標準值之間的差異確定。對於涉及儀器間消光係數比較，光譜鑑定和樣品純度分析的應用，它可被認為是最重要的參數。

提示：絕對光度精度對您的應用可能並不重要。在大多數定量應用中，只要測量在波長範圍內是可再現的和線性的，光度精度就不是關鍵。但是，在比較多種儀器的結果時，它確實成為一個關鍵參數。

6.4光度重現性

UV / Vis儀器可以進行重複測量的精度。它表示可以再現測量的吸光度值的程度。

6.5線性動態範圍

吸光度和濃度保持的濃度範圍彼此成正比。寬線性動態範圍允許分析各種樣品濃度（光密度）並減少樣品製備（稀釋）要求。

6.6噪音

源自光源和電子元件的UV / Vis光譜的波動。紫外/可見分光光度法中的噪聲主要來源於光源和電子元件。這會影響吸光度標度的低端和高端的準確度。來自光源的光子噪聲會影響低吸光度下的測量精度，而來自元件的電子噪聲會影響高吸光度測量精度。高噪聲水平將降低檢測限和儀器的靈敏度。

6.7光度範圍（工作吸光度範圍）

儀器處理強吸收樣品（低透射樣品）的能力。對於某些應用，特別是那些具有強吸收物質的應用，重要的是要考慮光度範圍。能夠檢測10％透射率的分光光度計具有1A的光度範圍，其中1％是2A，0.1％是3A。3.5A至4A的光度範圍意味著它可以處理吸收高達99.99％入射光的樣品。

提示：為紫外/可見分光光度計指定的光度範圍並不意味著它在該範圍內是線性的。並非所有儀器都提供線性動態範圍的規格，儘管它們總是給出光度範圍。因此，如果線性動態範圍對您很重要，請務必詢問。

6.8光度穩定性

儀器隨時間保持穩定狀態的能力，使得漂移對測量精度的影響變得微不足道。入射輻射（Io）和透射輻射（I）的測量之間的燈強度和電子輸出的變化導致儀器漂移。這些變化可能導致測量值的誤差，特別是在很長一段時間內。光度穩定性是儀器隨時間保持穩定狀態的能力，因此漂移對測量精度的影響變得微不足道。

6.9雜散光

到達檢測器的除了所需波長之外的不需要的光或波長的光。雜散光會導致吸光度降低並降低儀器的線性範圍。雜散光會嚴重影響高吸光度測量。

6.10頻譜帶寬和分辨率

指示離開單色器的波長範圍以及將光譜特徵和帶分解為單個組件的能力。頻譜帶寬和分辨率是相關的; 光譜帶寬越小，分辨率越精細。通常，分辨率差導致整個光譜中的消光係數降低，因此定量不準確。測量的靈敏度也受到影響。大多數紫外/可見分光光度計為常見應用提供足夠的分辨率。如果您的應用需要詳細的光譜信息，則需要帶寬非常小的儀器，以獲得更好的分辨率。

6.11基線平坦度

儀器能夠使燈的輸出和檢測器響應標準化。對於具有雙光源（用於UV範圍的氘燈和用於可見光範圍的鎢燈）的UV / Vis分光光度計，來自光源的輻射強度在整個UV /可見光範圍內不是恆定的。探測器的響應也在光譜範圍內變化。平坦的基線表明儀器能夠使燈的輸出和檢測器響應標準化。
