指針數組和數組指針

2020-11-14
| 科技

1、指針數組

定義：int * p[n] = { 0 };

[] 的優先級高於 * ，p先與 [] 結合，指明這是一個數組，然後再和 int* 結合，說明數組裡的每一個元素都是指針，每個元素都保存一個地址。

1.1 使用指針數組保存多個數據的地址

int main(){ int a[3] = { 1, 2, 3 }; int * p[3] = { 0 }; for (int i = 0; i < 3; ++i) { p[i] = &a[i]; } for (int i = 0; i < 3; ++i) { printf("p[%d] = %d\t *p[%d] = %d\n", i, p[i], i, *p[i]); } return 0;}

2、數組指針

定義：int * p; int (* p)[n]; int (* p)[n][m];

() 和 [] 的優先級相同，但是結合性從左往右，即 p 先於 * 結合，說明p是一個指針，然後再與 [] 結合，說明了這個指針指向了一個數組。

2.1

void func(int(*p)[3]);int main(){ int a[2][3] = { 10, 20, 30, 40, 50, 60 }; func(a);  return 0;}void func(int(*p)[3]) //明確列的數組指針{ printf("%d\n", p[1][2]); // 取出第2行的第3個元素：60 printf("%d\n", **p); //p是行指針，先是取出當前行的列地址，再取出這個列地址裡的值： 10 printf("%d\n", (*p + 1)[2]); // 先取出列地址，然後往後移動一個int(4個字節),再以這一列的下標為起點，取出後面第2個元素的值：40 printf("%d\n", *(*p + 1)); // 先取出列地址，往後移動1位(1個int)，再取出這個下標的值：20 printf("%d\n", *(p[1] + 2)); // 先取出p[1]的列下標地址，再往後移動2位(2個int),再取出這個下標的值：60 printf("%d\n", *(*(p + 1))); // 先將行指針p後移一位(3個int)，再取出這一行的這一列的下標地址，再根據這個下標取值：40 printf("%d\n", *((*p + 1) + 2)); //這個表達式相當於 *(*p + 3)，先取出列下標地址，然後往後移動3位（3個int），再取值：40 printf("%d\n", *(*p + 1) + 2); // 先取出列下標的地址，然後往後移動一位(1個int)，再進行取值得出20，再加2，結果是：22}

3、總結：

  • 指針數組：就是一個數組，這個數組裡的每個元素都是一個指針，這個數組在內存空間中佔用了n個指針的大小。
  • 數組指針：就是一個指針，這個指針指向一個數組，這個指針在內存空間中佔用一個指針的大小。
  • 二級指針p 和二維數組名p 的區別：
  • int ** p; 這個是一個整型的二級指針，p是一個可變句柄/鑰匙，我們可以讓這個指針指向任何我們希望它指向的地方.這個句柄不需要指定內存空間的大小。
  • int p[n][m]; 這個是一個整型的二維數組，p是引用了這塊內存空間的句柄/鑰匙，數組在定義的時候，就被固定指向某個內存空間了，這個空間大小是 sizeof(int) * n * m，而且不可修改p的指向，
  • 如果想確定某個一維數組的值： * 、 []
  • 如果想確定某個二維數組的值：** 、 [][] 、 *[]
