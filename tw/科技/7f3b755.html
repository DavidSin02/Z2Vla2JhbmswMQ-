獲取指定路徑下的文件夾名稱

2020-10-29
| 科技

說明：可以打開地址欄對話框，手選目錄。從A1單元格往下，依次列表所選文件夾裡面的1級文件夾名稱。默認起始地址為桌面，注意用戶名。

================================

Sub getfoldername() '獲取指定路徑下的文件夾名稱

Dim fd As Object

Set fd = Application.FileDialog(msoFileDialogFolderPicker)

With fd

.InitialFileName = "C:\Users\Administrator\Desktop"

If .Show = -1 Then fp = .SelectedItems(1) 'fp = 如果進行了選擇並確認,選中的文件夾地址

End With

Dim fs As Object

n = 1

Set fs = CreateObject("Scripting.FileSystemObject")

Set f = fs.getfolder(fp)

For Each fd In f.subfolders

Cells(n, 1) = fd.Name

n = n + 1

Next

Set f = Nothing

Set fs = Nothing

End Sub
