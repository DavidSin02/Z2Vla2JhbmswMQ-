2020烷基化工藝作業考試題庫及烷基化工藝考試試題

2020-10-29
| 科技

題庫來源：安全生產模擬考試一點通公眾號小程序

2020烷基化工藝作業考試題庫及烷基化工藝考試試題，包含烷基化工藝作業考試題庫答案解析及烷基化工藝考試試題練習。由安全生產模擬考試一點通公眾號結合國家烷基化工藝考試最新大綱及烷基化工藝考試真題出具，有助於烷基化工藝怎麼考考前練習。

1、【判斷題】負有危險化學品安全監督管理職責的部門的工作人員,在危險化學品安全監督管理工作中濫用職權、玩忽職守、徇私舞弊,構成犯罪的,依法追究刑事責任;尚不構成犯罪的,依法給予處分。（ √ ）

2、【判斷題】燃燒是一種同時伴有發光、發熱的激烈的氧化反應,具有發光、發熱和生成新物質三個特徵。（ √ ）

3、【判斷題】特種設備包括其所用的材料、附屬的安全附件、安全保護裝置和與安全保護裝置相關的設施。（ √ ）

4、【判斷題】自動信號聯鎖保護電路按主要構成原件不同可分為有觸點式和無觸點式,沒有混合式。（ × ）

5、【判斷題】現場處置方案的應急組織與職責主要包括應急自救組織機構、人員的具體職責,不應同單位或車間、班組人員工作職責緊密結合,明確相關崗位和人員的應急工作職責。（ × ）

6、【判斷題】轉動設備的軸封裝置其作用是防止介質從設備內部沿軸向外部洩漏。（ × ）

7、【判斷題】集散系統的核心裝置是集散採集單元。（ × ）

8、【判斷題】石化裝置的安全度等級的確定可以通過諸多因素經計算求得,也可通過工程經驗確定。（ √ ）

9、【判斷題】危險化學品的生產、貯存、使用單位,應當在生產、貯存、使用場所設置通訊、報警裝置,並保證在任何情況下處於正常使用狀態。（ √ ）

10、【判斷題】袋式除塵是目前控制微細粒子最有效的除塵設備。（ √ ）

11、【判斷題】乙炔氣瓶使用時應安裝阻火器。（ √ ）

12、【判斷題】氫氟酸烷基化裝置內的廢鹼主要來自含酸氣體中和器和酸溶性沒油(ASO)鹼洗罐。（ √ ）

13、【判斷題】使用、儲存、運輸氯氣的從業人員,必須接受有關預防和處置氯氣洩漏、中毒等知識培訓,考試後持證上崗。（ × ）

14、【判斷題】職業病防護設施所需費用不能納入建設項目工程預算。（ × ）

15、【判斷題】勞動者接受職業健康檢查應當視同正常出勤。（ √ ）

16、【判斷題】突發事件發生地的其他單位應當服從人民政府發佈的決定、命令,配合人民政府採取的應急處置措施,做好本單位的應急救援工作,並積極組織人員參加本單位的應急救援和處置工作。（ × ）

17、【判斷題】氟化鈉是非水溶性的,可以在水中析出。（ × ）

18、【判斷題】生產經營單位的應急預案由生產經營單位主要負責人簽署公佈後,再進行評審或者論證。（ × ）

19、【判斷題】氫氟酸烷基化裝置產生的酸溶性油經除酸後進入燃料油系統回收燃燒。（ √ ）

20、【判斷題】違反《中華人民共和國突發事件應對法》規定,構成犯罪的,依法追究民事責任。（ × ）

21、【判斷題】安全色規定有紅、黃、藍、綠等四種。（ √ ）

22、【判斷題】達到總量控制目標,應從降低汙染物濃度,減少汙染物排放著手。（ √ ）

23、【判斷題】石油化工企業中,無飛火的明火設備,應佈置在有散發可燃氣體的設備、建築物的主導風向下風頭。（ × ）

24、【判斷題】對環境空氣中可燃氣的監測,常用可燃氣環境爆炸危險度(%LEL)表示,其含義是環境中可燃氣的含量佔爆炸下限的百分數。（ √ ）

25、【判斷題】甲醇在空氣中的允許濃度為50mg/m³。（ √ ）

26、【單選題】在時間或空間上失去控制的燃燒稱為( )。（ B ）

A、著火

B、火災

C、閃燃

27、【單選題】在綜合應急演練前,通過安全( ),確認演練所需的工具、設備、設施、技術資料以及參演人員到位。（ B ）

A、會議

B、檢查

C、評審

28、【單選題】飽和液體線以下的區域稱為( )。（ B ）

A、氣相區

B、液相區

C、氣液共存區

29、【單選題】管道油漆施工需有要防火、防凍、防雨措施,且適宜的的環境溫度是( )。（ B ）

A、0℃-40℃

B、5℃-40℃

C、10℃-40℃

A、紅色

B、藍色

C、黃色

A、有組織

B、無組織

C、低點

A、2.1%-9.5%

B、2.3%-9%

C、1%-8%

A、泵的出口閥關死

B、入口壓力偏高

C、吸入管內有空氣

A、0℃-1300℃

B、0℃-400℃

C、-20℃-400℃

A、微分

B、積分

C、比例

A、長管式防毒面具

B、過濾式防毒面具

C、隔離式防毒面具

37、【單選題】酸儲罐充氮氣當壓力與沉降器( )時,可打開酸儲罐出口閥。（ C ）

A、壓力相等

B、高於0.1MPa

C、高於0.15MPa

38、【單選題】應當在其醒目位置,設置警示標識和中文警示說明的作業崗位為產生( )。（ B ）

A、職業病危害的

B、嚴重職業病危害的

C、較重和嚴重職業病危害

A、衛生部

B、勞動和社會保障部

C、安監總局

40、【單選題】從業人員為防禦物理、化學、生物等外界因素傷害所穿戴、配備和使用的各種護品的總稱稱為( )。（ A ）

A、個體防護裝備

B、個體防護用品

C、空氣呼吸器

41、【單選題】烷基化裝置流體輸送的設備有( )。（ A ）

A、離心泵,往復泵,齒輪泵

B、離心泵,往復泵

C、離心泵,噴射泵

42、【單選題】通常的爆炸極限是在常溫、常壓的標準條件下測定出來的,它隨( )的變化而變化。（ C ）

A、壓力、容積

B、溫度、容積

C、壓力、溫度

43、【單選題】如果原料中有四種組分(組分相互之間α>1),需用( )個連續精餾塔即可分離。（ C ）

A、1

B、2

C、3

44、【單選題】環境汙染事故是指由於違反環境保護法規的經濟、社會活動與行為,以及意外因素的影響或不可抗拒的自然災害等原因使環境受到汙染,國家重點保護的野生動植物、自然保護區受到破壞,人體健康受到危害,社會經濟與人民財產受到損失,造成不良社會影響的( )事件。（ C ）

A、計劃中

B、常規性

C、突發性

45、【單選題】由大氣汙染衍生出的環境效應主要有溫室效應、臭氧破壞和( )。（ C ）

A、風暴

B、颱風

C、酸雨

46、【單選題】( )應當包括危險性分析、可能發生的事故特徵、應急組織機構與職責、預防措施、應急處置程序和應急保障等內容。（ B ）

A、綜合應急預案

B、專項應急預案

C、現場處置方案

47、【單選題】下列安全洩放裝置中開啟排放後可自行關閉,容器和裝置可以繼續使用的是( )。（ B ）

A、爆破片

B、安全閥

C、易熔塞

48、【單選題】一般壓力容器的基本組成是( )。（ A ）

A、筒體、封頭、接管、法蘭

B、筒體、法蘭、支座、人孔

C、筒、法蘭、封頭、補強圈

49、【單選題】企業、事業單位和個體經營組織(統稱用人單位)的勞動者在職業活動中,因接觸粉塵、放射性物質和其他有毒有害物質等因素而引起的疾病稱為( )。（ C ）

A、職業病危害

B、職業禁忌症

C、職業病
