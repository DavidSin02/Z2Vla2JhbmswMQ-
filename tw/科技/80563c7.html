一天一個算法題

2020-10-29
| 科技

題目：

給你一個數N，拆成3部分，要求每一部分都不是3的整數。隨便輸出一個可能解即可。

分析：

題目很簡單，要求都不被3整除，不被3整除的數有以下特性：各位數字上面加起來的和不被3整除。可是又要一下子構建3個，看起來這個題目非常困難。

解

事實上，只要輸出一個可行解，我們不妨讓3個數字為 1,1,N-2, 看看這個是否滿足答案，也就是N-2是否滿足不被3整除。

如果N-2滿足被3整除，那麼N-3一定不被3整除，所以結果可為 1,2,N-3.

代碼

#include <iostream>using namespace std;int main(){ long n; cin >> n; cout << 1 << " "; if ((n - 2) % 3 == 0) { cout << 2 << " " << n - 3 << endl; }else { cout << 1 << " " << n - 2 << endl; }}
