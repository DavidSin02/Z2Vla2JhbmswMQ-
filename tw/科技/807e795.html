01、判斷題

試題編號 : 01

知識點 : 電力系統

在中性點不接地系統中發生單相接地故障時，允許短時運行，但不應超過4h。 ( × )

試題編號 : 02

知識點 : 電力系統

在中性點直接接地系統中，在發生單相接地故障時，非故障相對地電壓將不變。 ( √ )

試題編號 : 03

知識點 : 電力系統

電壓等級越高的設備,其耐雷水平也越高 ( √ )

試題編號 : 04

知識點 : 電工基礎

送電線路電壓一定時，輸送功率越大，輸送距離越小。 ( √ )

試題編號 : 05

知識點 : 電工基礎

導線電阻與溫度有關，溫度越高，電阻越大。 ( √ )



試題編號 : 06

知識點 : 電工基礎

電網中某點的實際電壓與電網額定電壓之差是電壓損失。 ( × )

試題編號 : 07

知識點 : 電工基礎

導線的電阻與導線溫度的關係是溫度升高，電阻減少。 ( × )

試題編號 : 08

知識點 : 電工基礎

電暈臨界電壓與導線間距離無關。 ( × )

試題編號 : 09

知識點 : 力學機械

當應力不超過某一限度時，縱向線應變與正應力成正比。 ( √ )

試題編號 :10

知識點 : 力學機械

材料的極限破壞強度與安全係數的比值稱為最大使用應力。 ( × )

試題編號 : 11

知識點 : 送電線路

在送電線路基礎現澆施工中，要求水泥的初凝時間不得早於60min。 ( × )

試題編號 : 12

知識點 : 送電線路

線路運行絕緣子發生閃絡的原因是表面汙溼。 ( √ )

試題編號 : 13

知識點 : 送電線路

混凝土杆卡盤安裝深度允許偏差不應超過—50mm。 ( × )

試題編號 : 14

知識點 : 送電線路

張力放線的多滑輪車，其輪槽寬應能順利通過接續管及其護套，且摩擦阻力系數不得大於1.015。 ( √ )

試題編號 : 15

知識點 : 送電線路

洩漏比距越大，越不容易發生汙閃事故。 ( × )



試題編號 : 16

知識點 : 送電線路

在軟土、堅土、砂、岩石四者中，電阻率最高的是岩石。 ( √ )

試題編號 : 17

知識點 : 送電線路

送電線路的垂直擋距決定杆塔承受風壓的擋距。 ( × )

試題編號 : 18

知識點 : 送電線路

張力放線時導線壓接後通過放線滑車的接續管必須加裝保護鋼套，以防止接續管彎曲。 ( √ )

試題編號 : 19

知識點 : 送電線路

架空線路的縱斷面圖反映沿線路中心線地形的起伏形狀及被交叉跨越物的標高。 ( √ )

試題編號 : 20

知識點 : 送電線路

架空導線截面在300平方毫米及以上時，其放線滑車輪底直徑與導線線徑之比應在18～24倍的範圍內。 ( √ )

試題編號 : 21

知識點 : 送電線路

導線的額定截面小於實際截面。 ( × )

試題編號 : 22

知識點 : 送電線路

架空擴徑導線的運行特點是傳輸功率大。 ( × )

試題編號 : 23

知識點 : 送電線路

水平接地體的地表面以下開挖深度一般為0.6～1.0m。 ( √ )

試題編號 : 24

知識點 : 送電線路

分裂導線子導線裝設間隔棒的主要作用是預防導線混線。 ( × )

試題編號 : 25

知識點 : 送電線路

送電線路的導線截面越大，導線的損耗越大。 ( × )

試題編號 : 26

知識點 : 送電線路

以抱箍連接的叉樑，其上端抱箍組裝尺寸的允許偏差應為±50mm。 ( √ )

試題編號 : 27

知識點 : 送電線路

送電線路絕緣子承受的大氣過電壓分為直擊雷過電壓和感應雷過電壓。 ( √ )

試題編號 : 28

知識點 : 送電線路

線路絕緣汙穢會降低絕緣子的絕緣性能，為防止汙閃事故發生，線路防汙工作必須在汙閃事故季節來臨前完成。 ( √ )

試題編號 : 29

知識點 : 送電線路

最容易引起架空線發生微風振動的風向是垂直線路方向。 ( √ )

試題編號 : 30

知識點 : 送電線路

送電線路中導線在懸點等高的情況下，杆塔的水平檔距與垂直檔距相等。 ( √ )


試題編號 : 31

知識點 : 送電線路

張力放線的區段最多為3~5km。 ( × )

試題編號 : 32

知識點 : 送電線路

全套屏蔽服最大電阻平均值不大於30Ω。 ( × )

試題編號 : 33

知識點 : 送電線路

直線杆塔絕緣子串所受的荷重同垂直檔距和水平檔距都無關。 ( × )

試題編號 : 34

知識點 : 送電線路

終勘工作應在初勘工作完成，初步設計定性後進行。 ( × )

試題編號 : 35

知識點 : 送電線路

重力基礎是指杆塔整體的自重力小於上拔力。 ( × )

試題編號 : 36

知識點 : 送電線路

阻尼線的防震原理相當於多個聯合的防震錘。 ( × )

試題編號 : 37

知識點 : 送電線路

線路防雷保護的重點是雲間放電。 ( × )

試題編號 : 38

知識點 : 送電線路

潮溼氣候條件是引起絕緣汙閃的必要條件。 ( √ )

試題編號 : 39

知識點 : 送電線路

汙穢等級的劃分，根據汙穢特徵，運行經驗，並結合鹽密值三個因素綜合考慮決定。 ( √ )

試題編號 : 40

知識點 : 送電線路

電力線路在同樣電壓下，經過同樣地區，單位爬距越大，則發生閃絡的可能性愈小。 ( √ )

試題編號 : 41

知識點 : 送電線路

張力放線長度不宜超過30個放線滑車的線路長度。 ( × )

試題編號 : 42

知識點 : 相關工種

因隔離開關無專門的滅弧裝置，因此不能通過操作隔離開關來斷開帶負荷的線路。 ( √ )

試題編號 : 43

知識點 : 相關工種

工業用紅外線熱像儀的測溫範圍一般為-20℃～2000℃。 ( × )

試題編號 : 44

知識點 : 相關工種

電纜溝內全長應裝設有連續的接地裝置，接地線的規格應符合規範要求。 ( √ )

試題編號 : 45

知識點 : 相關工種

連接兩種不同型號電纜的接頭稱為中間接頭 ( √ )

試題編號 : 46

知識點 : 規程和規範

在低壓配電盤、配電箱和電源幹線上的工作，應填用配電工作票。 ( × )

試題編號 : 47

知識點 : 規程和規範

直流高壓送電和交流高壓送電的線路走廊相比直流走廊要求高。 ( √ )
