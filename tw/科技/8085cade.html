機械行業對於超硬材料的分類及屬性，你知道多少？

2020-11-14
| 科技

根據維基百科的定義，Superhard Materials 超硬材料是維氏硬度超過40GPa的材料。在切磨拋行業，目前使用最廣泛的是金剛石和立方氮化硼。金剛石有天然金剛石和人造金剛石之分，立方氮化硼只能是人造。人造金剛石是加工業最硬的磨料，已經被廣泛應用於冶金、石油鑽探、建築工程、機械加工等領域。關於PCD（聚晶金剛石）和PCBN（聚晶立方氮化硼）是材料還是製品的問題，本文認為是材料，具體分類如下：

如圖1所示，超硬材料只有八個二級分類，沒有設置三級分類。

一、Natural Diamond 天然金剛石

天然金剛石作為天然磨料的一種，少量存在在自然界。與磨料一樣，也有分級標準和相關的粒度。

Grading System 分級標準：ANSI、FEPA、JIS、ISO、GB/T、Other；

Applications 用途：Resin Bonded Abrasives 樹脂磨具、Metal Bonded Abrasives 金屬磨具、Vitrified Abrasives 陶瓷磨具、Electroplated Products 電鍍製品、Lapping/Polishing 自由研磨/拋光、Sawing Tools 鋸切工具、 Drilling/Milling Tools 鑽探鑽進工具、Dressing Tools 修整工具、Wire Drawing Dies 拉絲模、Other 其他

二、Synthetic Diamond 人造金剛石

人造金剛石是超硬材料中使用最為廣泛的材料。本文沒有對人造金剛石進一步分類，全部按照屬性來區分金剛石。

Type 類型：Mesh 粗料、Micron 細料、Coated & Clad 鍍衣、Metal Encapsulated 金屬造粒

Particle Shape 顆粒形狀：Blocky 等積形、Angular 有稜角、Sharp 尖銳的、Irregular 不規則、Other 其他

Grading System 分級標準：參考天然金剛石；

Color 顏色：Light Brown淺棕色、Brown 棕色、Black 黑色、Aqua 淺綠、Green 綠色、Dark Green 墨綠、Light Yellow 淡黃色、Yellow 黃色、Gray 灰色、Colorless 透明、Other 其他

Metal Coated 金屬鍍衣：None 無、Copper 鍍銅、Nickel 鍍鎳、Silver 鍍銀、Titanium 鍍鈦、Combo 混合、Other 其他

Applications 用途：增加了一個PCD / PDC用途，其餘用途參考天然金剛石；

三、Cubic Boron Nitride 立方氮化硼

立方結構的氮化硼，分子式為BN,其晶體結構類似金剛石，硬度略低於金剛石，常用作磨料和刀具材料。立方氮化硼至今沒有發現天然的。針對CBN的分類和屬性，本文參考人造金剛石，其中Type 類型、Particle Shape 顆粒形狀、Grading System 分級標準、Metal Coated 金屬鍍衣、Applications 用途與金剛石一樣。顏色與金剛石不一樣，如下：

Color 顏色： Amber 琥珀色、Dark Amber 深琥珀色、、Black 黑色、Brown 棕色、Other 其他

四、PCD/TCP 金剛石聚晶

人造金剛石聚晶是一種把金剛石與結合劑在高溫、高壓下燒結而成的金剛石聚合物，它具有極高的耐熱性和很高的熱穩定性。廣泛應用於製造石油，地質鑽頭，鑽石加工和機械加工工具等。

Type 類型：PCD、TSP

Shape 形狀：Cube 立方體、Cylinder 圓柱體、Disc 圓盤、Rectangle 長方形、Triangle 三角形、Spheres 圓弧、Hexagon 六邊形、Other 其他

Application 應用：Gem Processing 寶石加工、Drilling Bits 鑽頭、Other 其他

五、PCBN 立方碳化硼聚晶

立方氮化硼聚晶是由立方氮化硼微粉在結合劑存在下，高溫高壓燒結而成的立方氮化硼多晶體。

Shape 形狀：Cube 立方體、Cylinder 圓柱體、Disc 圓盤、Rhombus 菱形、Triangle 三角形、Hexagon 六邊形、Other 其他

六、CVD Diamond 化學氣相沉積金剛石

在百度百科中，CVD（Chemical Vapor Deposition化學氣相沉澱）金剛石是指含碳氣體和氧氣的混合物在高溫和低於標準大氣壓的壓力下被激發分解，形成活性金剛石碳原子，並在基體上沉積交互生長成聚晶金剛石（或控制沉積生長條件沉積生長金剛石單晶或者準單晶）。

Shape 形狀：Triangle 三角形、Square 正方形、Round 圓形、 Bar 棒、Other 其他

七、Nano Diamond 納米金剛石

針對納米金剛石的分類及屬性，目前我們沒有設置，將在產品數量豐富時加入。

八、Other 其他超硬材料

我們強烈不建議發佈產品到這個分類，除非您確定您的產品的確是超硬材料，但在上述的七個分類中沒有找到。因為在產品列表中，我們會把此分類中的產品放在列表的最末端。
