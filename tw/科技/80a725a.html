前 言

根據住房和城鄉建設部《關於印發<2015年工程建設標準規範制訂、修訂計劃（第一批）>的通知》（建標[2014]189號）的要求，標準編制組經過廣泛調查研究，認真總結實踐經驗，參考有關國際標準和國外先進標準，並在廣泛徵求意見的基礎上，修訂了本標準。
本標準的主要技術內容是：1．總則；2．術語和符號；3．基本規定；4．極限狀態設計原則；5．結構上的作用和環境影響；6．材料和岩土的性能及幾何參數；7．結構分析和試驗輔助設計；8．分項係數設計方法。
本標準修訂的主要技術內容是：1．與《工程結構可靠性設計統一標準》GB 50153-2008進行了全面協調；2．調整了建築結構安全度的設置水平，提高了相關作用分項係數的取值，並對作用的基本組合，取消了原標準當永久荷載效應為主時起控制作用的組合式；3．增加了地震設計狀況，並對建築結構抗震設計，引入了“小震不壞、中震可修、大震不倒”設計理念；4．完善了既有結構可靠性評定的規定；5．新增了結構整體穩固性設計的相關規定；6．新增了結構耐久性極限狀態設計的相關規定等。
本標準中以黑體字標誌的條文為強制性條文，必須嚴格執行。
本標準由住房和城鄉建設部負責管理和對強制性條文的解釋，由中國建築科學研究院有限公司負責具體技術內容的解釋。執行過程中如有意見或建議，請寄送中國建築科學研究院有限公司（地址：北京市北三環東路30號；郵政編碼：100013）。
本標準主編單位：中國建築科學研究院有限公司。
本標準參編單位：中國建築東北設計研究院有限公司、重慶大學、中南建築設計院股份有限公司、中國建築西南設計研究院有限公司、大連理工大學、浙江大學、國家建築工程質量監督檢驗中心。

1 總則

1．0．1 為統一各種材料的建築結構可靠性設計的基本原則、基本要求和基本方法，使結構符合可持續發展的要求，並符合安全可靠、經濟合理、技術先進、確保質量的要求，制定本標準。
1．0．2 本標準適用於整個結構、組成結構的構件以及地基基礎的設計；適用於結構施工階段和使用階段的設計；適用於既有結構的可靠性評定。既有結構的可靠性評定，可根據本標準附錄A的規定進行。
1．0．3 本標準依據現行國家標準《工程結構可靠性設計統一標準》GB 50153的原則制定，是建築結構可靠性設計的基本要求。
1．0．4 建築結構設計宜採用以概率理論為基礎、以分項係數表達的極限狀態設計方法；當缺乏統計資料時，建築結構設計可根據可靠的工程經驗或必要的試驗研究進行，也可採用容許應力或單一安全係數等經驗方法進行。
1．0．5 制定建築結構荷載標準、各種材料的結構設計標準以及其他相關標準時，應符合本標準規定的基本準則，並應制定相應的具體規定。
1．0．6 建築結構設計除應符合本標準的規定外，尚應符合國家現行有關標準的規定。


2 術語和符號

2．1 術語

2．1．1 結構 structure
能承受作用並具有適當剛度的由各連接部件有機組合而成的系統。
2．1．2 結構構件 structural member
結構在物理上可以區分出的部件。
2．1．3 結構體系 structural system
結構中的所有承重構件及其共同工作的方式。
2．1．4 結構模型 structural model
用於結構分析、設計等的理想化的結構體系。
2．1．5 設計使用年限 design service life
設計規定的結構或結構構件不需進行大修即可按預定目的使用的年限。
2．1．6 設計狀況 design situations
表徵一定時段內實際情況的一組設計條件，設計應做到在該組條件下結構不超越有關的極限狀態。
2．1．7 持久設計狀況 persistent design situation
在結構使用過程中一定出現，且持續期很長的設計狀況，其持續期一般與設計使用年限為同一數量級。
2．1．8 短暫設計狀況 transient design situation
在結構施工和使用過程中出現概率較大，而與設計使用年限相比，其持續期很短的設計狀況。
2．1．9 偶然設計狀況 accidental design situation
在結構使用過程中出現概率很小，且持續期很短的設計狀況。
2．1．10 地震設計狀況 seismic design situation
結構遭受地震時的設計狀況。
2．1．11 荷載佈置 load arrangement
在結構設計中，對自由作用的位置、大小和方向的合理確定。
2．1．12 荷載工況 load case
為特定的驗證目的，一組同時考慮的固定可變作用、永久作用、自由作用的某種相容的荷載佈置以及變形和幾何偏差。
2．1．13 極限狀態 limit states
整個結構或結構的一部分超過某一特定狀態就不能滿足設計規定的某一功能要求，此特定狀態為該功能的極限狀態。
2．1．14 承載能力極限狀態 ultimate limit states
對應於結構或結構構件達到最大承載力或不適於繼續承載的變形的狀態。
2．1．15 正常使用極限狀態 serviceability limit states
對應於結構或結構構件達到正常使用的某項規定限值的狀態。
2．1．16 不可逆正常使用極限狀態 irreversible serviceability limit states
當產生超越正常使用要求的作用卸除後，該作用產生的後果不可恢復的正常使用極限狀態。
2．1．17 可逆正常使用極限狀態 reversible serviceability limit states
當產生超越正常使用要求的作用卸除後，該作用產生的後果可以恢復的正常使用極限狀態。
2．1．18 耐久性極限狀態 durability limit states
對應於結構或結構構件在環境影響下出現的劣化達到耐久性能的某項規定限值或標誌的狀態。
2．1．19 抗力 resistance
結構或結構構件承受作用效應和環境影響的能力。
2．1．20 結構整體穩固性 structural integrity;structural robustness
當發生火災、爆炸、撞擊或人為錯誤等偶然事件時，結構整體能保持穩固且不出現與起因不相稱的破壞後果的能力。
2．1．21 關鍵構件 key member;key element
結構承載能力極限狀態性能所依賴的結構構件。
2．1．22 連續倒塌 progressive collapse
初始的局部破壞，從構件到構件擴展，最終導致整個結構倒塌或與起因不相稱的一部分結構倒塌。
2．1．23 可靠性 reliability
結構在規定的時間內，在規定的條件下，完成預定功能的能力。
2．1．24 可靠度 degree of reliability;reliability
結構在規定的時間內，在規定的條件下，完成預定功能的概率。
2．1．25 失效概率Pf probability of failure Pf
結構不能完成預定功能的概率。
2．1．26 可靠指標β reliability index β
度量結構可靠度的數值指標，可靠指標β為失效概率Pf負的標準正態分佈函數的反函數。
2．1．27 基本變量 basic variable
代表物理量的一組規定的變量，用於表示作用和環境影響、材料和岩土的性能以及幾何參數的特徵。
2．1．28 功能函數 performance function
關於基本變量的函數，該函數表徵一種結構功能。
2．1．29 概率分佈 probability distribution
隨機變量取值的統計規律，一般採用概率密度函數或概率分佈函數表示。
2．1．30 統計參數 statistical parameter
在概率分佈中用來表示隨機變量取值的平均水平和離散程度的數字特徵。
2．1．31 分位值 fractile
與隨機變量概率分佈函數的某一概率相應的值。
2．1．32 名義值 nominal value
用非統計方法確定的值。
2．1．33 極限狀態法 limit state method
不使結構超越某種規定的極限狀態的設計方法。
2．1．34 容許應力法 permissible stress method,allowable stress method
使結構或地基在作用標準值下產生的應力不超過規定的容許應力的設計方法。
2．1．35 單一安全係數法 single safety factor method
使結構或地基的抗力標準值與作用標準值的效應之比不低於某一規定安全係數的設計方法。
2．1．36 作用 action
施加在結構上的集中力或分佈力和引起結構外加變形或約束變形的原因。前者為直接作用，也稱為荷載；後者為間接作用。
2．1．37 外加變形 imposed deformations
結構在地震、不均勻沉降等因素作用下，邊界條件發生變化而產生的位移和變形。
2．1．38 約束變形 constrained deformations
結構在溫度變化、溼度變化及混凝土收縮等因素作用下，由於存在外部約束而產生的內部變形。
2．1．39 作用效應 effect of action
由作用引起的結構或結構構件的反應。
2．1．40 單個作用 single action
可認為與結構上的任何其他作用之間在時間和空間上為統計獨立的作用。
2．1．41 永久作用 permanent action
在設計使用年限內始終存在且其量值變化與平均值相比可以忽略不計的作用；或其變化是單調的並趨於某個限值的作用。
2．1．42 可變作用 variable action
在設計使用年限內其量值隨時間變化，且其變化與平均值相比不可忽略不計的作用。
2．1．43 偶然作用 accidental action
在設計使用年限內不一定出現，而一旦出現其量值很大，且持續期很短的作用。
2．1．44 地震作用 seismic action
地震動對結構所產生的作用。
2．1．45 土工作用 geotechnical action
由岩土、填方或地下水傳遞到結構上的作用。
2．1．46 固定作用 fixed action
在結構上具有固定空間分佈的作用。當固定作用在結構某一點上的大小和方向確定後，該作用在整個結構上的作用即得以確定。
2．1．47 自由作用 free action
在結構上給定的範圍內具有任意空間分佈的作用。
2．1．48 靜態作用 static action
使結構產生的加速度可以忽略不計的作用。
2．1．49 動態作用 dynamic action
使結構產生的加速度不可忽略不計的作用。
2．1．50 有界作用 bounded action
具有不能被超越的且可確切或近似掌握界限值的作用。
2．1．51 無界作用 unbounded action
沒有明確界限值的作用。
2．1．52 作用的標準值 characteristic value of an action
作用的主要代表值。可根據對觀測數據的統計、作用的自然界限或工程經驗確定。
2．1．53 設計基準期 design reference period
為確定可變作用等取值而選用的時間參數。
2．1．54 可變作用的組合值 combination value of a variable action
使組合後的作用效應的超越概率與該作用單獨出現時其標準值作用效應的超越概率趨於一致的作用值；或組合後使結構具有規定可靠指標的作用值。可通過組合值係數對作用標準值的折減來表示。
2．1．55 可變作用的頻遇值 frequent value of a variable action
在設計基準期內被超越的總時間佔設計基準期的比率較小的作用值；或被超越的頻率限制在規定頻率內的作用值。可通過頻遇值係數對作用標準值的折減來表示。
2．1．56 可變作用的準永久值 quasi-permanent value of a variable action
在設計基準期內被超越的總時間佔設計基準期的比率較大的作用值。可通過準永久值係數對作用標準值的折減來表示。
2．1．57 可變作用的伴隨值 accompanying value of a variable action
在作用組合中，伴隨主導作用的可變作用值。可變作用的伴隨值可以是組合值、頻遇值或準永久值。
2．1．58 作用的代表值 representative value of an action
極限狀態設計所採用的作用值。它可以是作用的標準值或可變作用的伴隨值。
2．1．59 作用的設計值 design value of an action
作用的代表值與作用分項係數的乘積。
2．1．60 作用組合 combination of actions；荷載組合 load combination
在不同作用的同時影響下，為驗證某一極限狀態的結構可靠度而採用的一組作用設計值。
2．1．61 環境影響 environmental influence
環境對結構產生的各種機械的、物理的、化學的或生物的不利影響。環境影響會引起結構材料性能的劣化，降低結構的安全性或適用性，影響結構的耐久性。
2．1．62 材料性能的標準值 characteristic value of a material property
符合規定質量的材料性能概率分佈的某一分位值或材料性能的名義值。
2．1．63 材料性能的設計值 design value of a material property
材料性能的標準值除以材料性能分項係數所得的值。
2．1．64 幾何參數的標準值 characteristic value of a geometrical parameter
設計規定的幾何參數公稱值或幾何參數概率分佈的某一分位值。
2．1．65 幾何參數的設計值 design value of a geometrical parameter
幾何參數的標準值增加或減少一個幾何參數的附加量所得的值。
2．1．66 結構分析 structural analysis
確定結構上作用效應的過程或方法。
2．1．67 一階線彈性分析 first order linear-elastic analysis
基於線性應力-應變或彎矩-曲率關係，採用彈性理論分析方法對初始結構幾何形體進行的結構分析。
2．1．68 二階線彈性分析 second order linear-elastic analysis
基於線性應力-應變或彎矩-曲率關係，採用彈性理論分析方法對已變形結構幾何形體進行的結構分析。
2．1．69 有重分佈的一階或二階線彈性分析 first order or second order linear-elastic analysis with redistribution
結構設計中對內力進行調整的一階或二階線彈性分析，與給定的外部作用協調，不做明確的轉動能力計算的結構分析。
2．1．70 一階非線性分析 first order non-linear analysis
基於材料非線性變形特性對初始結構的幾何形體進行的結構分析。
2．1．71 二階非線性分析 second order non-linear analysis
基於材料非線性變形特性對已變形結構幾何形體進行的結構分析。
2．1．72 一階或二階彈塑性分析 first order or second elastoplastic analysis
基於線彈性階段和隨後的無硬化階段構成的彎矩-曲率關係的結構分析。
2．1．73 剛性-塑性分析 rigid plastic analysis
假定彎矩-曲率關係為無彈性變形和無硬化階段，採用極限分析理論對初始結構的幾何形體進行的直接確定其極限承載力的結構分析。
2．1．74 既有結構 existing structure
已經存在的各類建築結構。
2．1．75 評估使用年限 assessed working life
可靠性評定所預估的既有結構在規定條件下的使用年限。
2．1．76 荷載檢驗 load testing
通過施加荷載評定結構或結構構件的性能或預測其承載力的試驗。

2．2 符號

3 基本規定

3．1 基本要求

3．1．1 結構的設計、施工和維護應使結構在規定的設計使用年限內以規定的可靠度滿足規定的各項功能要求。
3．1．2 結構應滿足下列功能要求：
1 能承受在施工和使用期間可能出現的各種作用；
2 保持良好的使用性能；
3 具有足夠的耐久性能；
4 當發生火災時，在規定的時間內可保持足夠的承載力；
5 當發生爆炸、撞擊、人為錯誤等偶然事件時，結構能保持必要的整體穩固性，不出現與起因不相稱的破壞後果，防止出現結構的連續倒塌；結構的整體穩固性設計，可根據本標準附錄B的規定進行。
3．1．3 結構設計時，應根據下列要求採取適當的措施，使結構不出現或少出現可能的損壞：
1 避免、消除或減少結構可能受到的危害；
2 採用對可能受到的危害反應不敏感的結構類型；
3 採用當單個構件或結構的有限部分被意外移除或結構出現可接受的局部損壞時，結構的其他部分仍能保存的結構類型；
4 不宜採用無破壞預兆的結構體系；
5 使結構具有整體穩固性。
3．1．4 宜採取下列措施滿足對結構的基本要求:
1 採用適當的材料；
2 採用合理的設計和構造；
3 對結構的設計、製作、施工和使用等制定相應的控制措施。

3．2 安全等級和可靠度

3．2．1 建築結構設計時，應根據結構破壞可能產生的後果，即危及人的生命、造成經濟損失、對社會或環境產生影響等的嚴重性，採用不同的安全等級。建築結構安全等級的劃分應符合表3．2．1的規定。

3．2．2 建築結構中各類結構構件的安全等級，宜與結構的安全等級相同，對其中部分結構構件的安全等級可進行調整，但不得低於三級。
3．2．3 可靠度水平的設置應根據結構構件的安全等級、失效模式和經濟因素等確定。對結構的安全性、適用性和耐久性可採用不同的可靠度水平。
3．2．4 當有充分的統計數據時，結構構件的可靠度宜採用可靠指標β度量。結構構件設計時採用的可靠指標，可根據對現有結構構件的可靠度分析，並結合使用經驗和經濟因素等確定。
3．2．5 各類結構構件的安全等級每相差一級，其可靠指標的取值宜相差0．5。
3．2．6 結構構件持久設計狀況承載能力極限狀態設計的可靠指標，不應小於表3．2．6的規定。

3．2．7 結構構件持久設計狀況正常使用極限狀態設計的可靠指標，宜根據其可逆程度取0～1．5。
3．2．8 結構構件持久設計狀況耐久性極限狀態設計的可靠指標，宜根據其可逆程度取1．0～2．0。

3．3 設計使用年限和耐久性

3．3．1 建築結構的設計基準期應為50年。
3．3．2 建築結構設計時，應規定結構的設計使用年限。
3．3．3 建築結構的設計使用年限，應按表3．3．3採用。

3．3．4 建築結構設計時應對環境影響進行評估，當結構所處的環境對其耐久性有較大影響時，應根據不同的環境類別採用相應的結構材料、設計構造、防護措施、施工質量要求等，並應制定結構在使用期間的定期檢修和維護制度，使結構在設計使用年限內不致因材料的劣化而影響其安全或正常使用。
3．3．5 環境對結構耐久性的影響，可通過工程經驗、試驗研究、計算、檢驗或綜合分析等方法進行評估；耐久性極限狀態設計可根據本標準附錄C的規定進行。
3．3．6 環境類別的劃分和相應的設計、施工、使用及維護的要求等，應符合國家現行有關標準的規定。

3．4 可靠性管理

3．4．1 為保證建築結構具有規定的可靠性水平，除應進行設計計算外，還應對結構的材料性能、施工質量、使用和維護進行相應的控制。控制的具體措施，應符合本標準附錄D和有關的勘察、設計、施工及維護等標準的專門規定。
3．4．2 建築結構的設計必須由具有相應資格的技術人員承擔。
3．4．3 建築結構的設計應符合國家現行的有關荷載、抗震、地基基礎和各種材料結構設計標準的規定。
3．4．4 建築結構的設計應對結構可能受到的偶然作用、環境影響等採取必要的防護措施。
3．4．5 對建築結構所採用的材料及施工、製作過程應進行質量控制，並按國家現行有關標準的規定進行驗收。
3．4．6 建築結構應按設計規定的用途使用，並應定期檢查結構狀況，進行必要的維護和維修；當需變更使用用途時，應進行設計複核並採取相應的技術措施。


4 極限狀態設計原則

4．1 極限狀態

4．1．1 極限狀態可分為承載能力極限狀態、正常使用極限狀態和耐久性極限狀態。極限狀態應符合下列規定：
1 當結構或結構構件出現下列狀態之一時，應認定為超過了承載能力極限狀態：
1) 結構構件或連接因超過材料強度而破壞，或因過度變形而不適於繼續承載；
2) 整個結構或其一部分作為剛體失去平衡；
3) 結構轉變為機動體系；
4) 結構或結構構件喪失穩定；
5) 結構因局部破壞而發生連續倒塌；
6) 地基喪失承載力而破壞；
7) 結構或結構構件的疲勞破壞。
2 當結構或結構構件出現下列狀態之一時，應認定為超過了正常使用極限狀態：
1) 影響正常使用或外觀的變形；
2) 影響正常使用的局部損壞；
3) 影響正常使用的振動；
4) 影響正常使用的其他特定狀態。
3 當結構或結構構件出現下列狀態之一時，應認定為超過了耐久性極限狀態：
1) 影響承載能力和正常使用的材料性能劣化；
2) 影響耐久性能的裂縫、變形、缺口、外觀、材料削弱等；
3) 影響耐久性能的其他特定狀態。
4．1．2 對結構的各種極限狀態，均應規定明確的標誌或限值。
4．1．3 結構設計時應對結構的不同極限狀態分別進行計算或驗算；當某一極限狀態的計算或驗算起控制作用時，可僅對該極限狀態進行計算或驗算。

4．2 設計狀況

4．2．1 建築結構設計應區分下列設計狀況：
1 持久設計狀況，適用於結構使用時的正常情況；
2 短暫設計狀況，適用於結構出現的臨時情況，包括結構施工和維修時的情況等；
3 偶然設計狀況，適用於結構出現的異常情況，包括結構遭受火災、爆炸、撞擊時的情況等；
4 地震設計狀況，適用於結構遭受地震時的情況。
4．2．2 對不同的設計狀況，應採用相應的結構體系、可靠度水平、基本變量和作用組合等進行建築結構可靠性設計。

4．3 極限狀態設計

4．3．1 對本標準第4．2．1條規定的四種建築結構設計狀況，應分別進行下列極限狀態設計：
1 對四種設計狀況均應進行承載能力極限狀態設計；
2 對持久設計狀況尚應進行正常使用極限狀態設計，並宜進行耐久性極限狀態設計；
3 對短暫設計狀況和地震設計狀況可根據需要進行正常使用極限狀態設計；
4 對偶然設計狀況可不進行正常使用極限狀態和耐久性極限狀態設計。

4．3．2 進行承載能力極限狀態設計時，應根據不同的設計狀況採用下列作用組合：
1 對於持久設計狀況或短暫設計狀況，應採用作用的基本組合；
2 對於偶然設計狀況，應採用作用的偶然組合；
3 對於地震設計狀況，應採用作用的地震組合。

4．3．3 進行正常使用極限狀態設計時，宜採用下列作用組合：
1 對於不可逆正常使用極限狀態設計，宜採用作用的標準組合；
2 對於可逆正常使用極限狀態設計，宜採用作用的頻遇組合；
3 對於長期效應是決定性因素的正常使用極限狀態設計，宜採用作用的準永久組合。

4．3．4 對每一種作用組合，建築結構的設計均應採用其最不利的效應設計值進行。
4．3．5 結構的極限狀態可採用下列極限狀態方程描述：

g(X1,X2,…,Xn)＝0 (4．3．5)

式中：g(·)——結構的功能函數；
Xi(i＝1,2,…,n)——基本變量，指結構上的各種作用和環境影響、材料和岩土的性能及幾何參數等；在進行可靠度分析時，基本變量應作為隨機變量。

4．3．6 結構按極限狀態設計應符合下列規定：

g(X1,X2,…,Xn)≥0 (4．3．6)

4．3．7 當採用結構的作用效應和結構的抗力作為綜合基本變量時，結構按極限狀態設計應符合下列規定：

R－S≥0 (4．3．7)

式中：R——結構的抗力；
S——結構的作用效應。

4．3．8 結構構件的設計應以規定的可靠度滿足本標準第4．3．6或第4．3．7條的要求。
4．3．9 結構構件宜根據規定的可靠指標，採用由作用的代表值、材料性能的標準值、幾何參數的標準值和各相應的分項係數構成的極限狀態設計表達式進行設計；有條件時也可根據本標準附錄E的規定，直接採用基於可靠指標的方法進行設計。


5 結構上的作用和環境影響

5．1 一般規定

5．1．1 建築結構設計時，應考慮結構上可能出現的各種直接作用、間接作用和環境影響。

5．2 結構上的作用

5．2．1 結構上的各種作用，當在時間上和空間上可認為是相互獨立時，則每一種作用可分別作為單個作用；當某些作用密切相關且有可能同時以最大值出現時，也可將這些作用一起作為單個作用。
5．2．2 同時施加在結構上的各單個作用對結構的共同影響，應通過作用組合來考慮；對不可能同時出現的各種作用，不應考慮其組合。
5．2．3 結構上的作用可按下列性質分類：
1 按隨時間的變化分類：
1) 永久作用；
2) 可變作用；
3) 偶然作用。
2 按隨空間的變化分類：
1) 固定作用；
2) 自由作用。
3 按結構的反應特點分類：
1) 靜態作用；
2) 動態作用。
4 按有無限值分類：
1) 有界作用；
2) 無界作用。
5 其他分類。
5．2．4 結構上的作用隨時間變化的規律，宜採用隨機過程的概率模型進行描述，對不同的作用可採用不同的方法進行簡化，並應符合下列規定：
1 對永久作用，可採用隨機變量的概率模型。
2 對可變作用，在作用組合中可採用簡化的隨機過程概率模型。在確定可變作用的代表值時可採用將設計基準期內最大值作為隨機變量的概率模型。
5．2．5 當永久作用和可變作用作為隨機變量時，其統計參數和概率分佈類型，應以觀測數據為基礎，運用參數估計和概率分佈的假設檢驗方法確定，檢驗的顯著性水平可取0．05。
5．2．6 當有充分觀測數據時，作用的標準值應按在設計基準期內最不利作用概率分佈的某個統計特徵值確定；當有條件時，可對各種作用統一規定該統計特徵值的概率定義；當觀測數據不充分時，作用的標準值也可根據工程經驗通過分析判斷確定；對有明確界限值的有界作用，作用的標準值應取其界限值。
5．2．7 建築結構按不同極限狀態設計時，在相應的作用組合中對可能同時出現的各種作用，應採用不同的作用代表值。對可變作用，其代表值包括標準值、組合值、頻遇值和準永久值。組合值、頻遇值和準永久值可通過對可變作用的標準值分別乘以不大於1的組合值係數ψc、頻遇值係數ψf和準永久值係數ψq等折減係數表示。
5．2．8 對偶然作用，應採用偶然作用的設計值。偶然作用的設計值應根據具體工程情況和偶然作用可能出現的最大值確定，也可根據有關標準的專門規定確定。
5．2．9 對地震作用，應採用地震作用的標準值。地震作用的標準值應根據地震作用的重現期確定；地震作用的重現期可根據建築抗震設防目標，按有關標準的專門規定確定。

5．2．10 當結構上的作用比較複雜且不能直接描述時，可根據作用形成的機理，通過數學模型來表徵作用的大小、位置、方向和持續期等性質。結構上的作用F的大小可採用下列數學模型：

F＝

(F0,ω) (5．2．10)

式中：

(·)——所採用的函數；
F0——基本作用，通常具有隨時間和空間隨機的或非隨機的變異性，但與結構的性質無關；
ω——用以將F0轉化為F的隨機或非隨機變量，它與結構的性質有關。

5．2．11 當結構的動態性能比較明顯時，結構應採用動力模型描述。此時，結構的動力分析應考慮結構的剛度、阻尼及結構上各部分質量的慣性。當結構容許簡化分析時，可計算“擬靜態作用”響應，並乘以動力系數作為動態作用的響應。
5．2．12 對自由作用應考慮各種可能的荷載佈置，並與固定作用等一起作為驗證結構某特定極限狀態的荷載工況。

5．3 環境影響

5．3．1 環境影響可分為永久影響、可變影響和偶然影響。
5．3．2 對結構的環境影響應進行定量描述；當沒有條件進行定量描述時，可通過環境對結構的影響程度的分級等方法進行定性描述，並在設計中採取相應的技術措施。


6 材料和岩土的性能及幾何參數

6．1 材料和岩土的性能

6．1．1 材料和岩土的強度、彈性模量、變形模量、壓縮模量、內摩擦角、黏聚力等物理力學性能，應根據國家現行有關試驗方法標準經試驗確定。
6．1．2 當利用標準試件的試驗結果確定結構中實際的材料性能時，尚應考慮實際結構與標準試件、實際工作條件與標準試驗條件的差別。結構中的材料性能與標準試件材料性能的關係，應根據相應的對比試驗結果通過換算係數或函數來表示，或根據工程經驗判斷確定。結構中材料性能的不定性，應由標準試件材料性能的不定性和換算係數或函數的不定性兩部分組成。
6．1．3 材料性能宜採用隨機變量概率模型描述。材料性能的各種統計參數和概率分佈類型，應以試驗數據為基礎，運用參數估計和概率分佈的假設檢驗方法確定，檢驗的顯著性水平可取0．05。
6．1．4 材料強度的概率分佈宜採用正態分佈或對數正態分佈。
6．1．5 材料強度的標準值可按其概率分佈的0．05分位值確定。材料彈性模量、泊松比等物理性能的標準值可按其概率分佈的0．5分位值確定。
6．1．6 當試驗數據不充分時，材料性能的標準值可採用有關標準的規定值，也可根據工程經驗，經分析判斷確定。
6．1．7 岩土性能指標和地基承載力、樁基承載力等，應通過原位測試、室內試驗等直接或間接的方法測定，並應考慮由於鑽探取樣的擾動、室內外試驗條件與實際建築結構條件的差別以及所採用公式的誤差等因素的影響，結合工程經驗綜合確定。
6．1．8 岩土性能的標準值宜根據原位測試和室內試驗的結果，按有關標準的規定確定；當有條件時，岩土性能的標準值可按其概率分佈的某個分位值確定。

6．2 幾何參數

6．2．1 結構或結構構件的幾何參數宜採用隨機變量概率模型描述。幾何參數的各種統計參數和概率分佈類型，應以正常生產情況下對結構或結構構件幾何尺寸的觀測數據為基礎，運用參數估計和概率分佈的假設檢驗方法確定。
6．2．2 當觀測數據不充分時，幾何參數的統計參數可根據有關標準中規定的公差，經分析判斷確定。
6．2．3 當幾何參數的變異性對結構抗力及其他性能的影響很小時，幾何參數可作為確定性變量。
6．2．4 幾何參數的標準值可採用設計規定的公稱值，或根據幾何參數概率分佈的某個分位值確定。


7 結構分析和試驗輔助設計

7．1 一般規定

7．1．1 結構分析可採用計算、模型試驗或原型試驗等方法進行。

7．1．2 結構分析的精度，應能滿足結構設計要求，必要時宜進行試驗驗證。

7．1．3 在結構分析中，宜考慮環境對材料、構件和結構性能的影響。

7．2 結構模型

7．2．1 結構分析採用的基本假定和計算模型應能合理描述所考慮的極限狀態下的結構反應。

7．2．2 根據結構的具體情況，可採用一維、二維或三維的計算模型進行結構分析。

7．2．3 結構分析所採用的各種簡化或近似假定，應具有理論或試驗依據，或經工程驗證可行。

7．2．4 當結構的變形可能使作用的影響顯著增大時，應在結構分析中考慮結構變形的影響。

7．2．5 結構計算模型的不定性應在極限狀態方程中採用一個或幾個附加基本變量來考慮。附加基本變量的概率分佈類型和統計參數，可通過按計算模型的計算結果與按精確方法的計算結果或實際的觀測結果相比較，經統計分析確定，或根據工程經驗判斷確定。

7．3 作用模型

7．3．1 對與時間無關的或不計累積效應的靜力分析，可只考慮發生在設計基準期內作用的最大值和最小值；當動力性能起控制作用時，應有詳細的過程描述。

7．3．2 當不能準確確定作用參數時，應對作用參數給出上下限範圍並進行比較，以確定不利的作用效應。

7．3．3 當結構承受自由作用時，應根據每一自由作用可能出現的空間位置、大小和方向，分析確定對結構最不利的荷載佈置。

7．3．4 當考慮地基與結構相互作用時，土工作用可採用適當的等效彈簧或阻尼器來模擬。

7．3．5 當動力作用可被認為是擬靜力作用時，可通過把動力作用分析結果包括在靜力作用中的方法或將靜力作用乘以等效動力放大係數的方法等，來考慮動力作用效應。

7．3．6 當動力作用引起的振幅、速度、加速度使結構有可能超過正常使用極限狀態的限值時，應根據實際情況對結構進行正常使用極限狀態驗算。

7．4 分析方法

7．4．1 結構分析應根據結構類型、材料性能和受力特點等因素，採用線性、非線性或試驗分析方法；當結構性能始終處於彈性狀態時，可採用彈性理論進行結構分析，否則宜採用彈塑性理論進行結構分析。

7．4．2 當結構在達到極限狀態前能夠產生足夠的塑性變形，且所承受的不是多次重複的作用時，可採用塑性理論進行結構分析；當結構的承載力由脆性破壞或穩定控制時，不應採用塑性理論進行分析。

7．4．3 當動力作用使結構產生較大加速度時，應對結構進行動力響應分析。

7．5 試驗輔助設計

7．5．1 對沒有適當分析模型的特殊情況，可按本標準附錄F規定的方法，通過試驗輔助設計進行結構分析。

7．5．2 採用試驗輔助設計的結構，應達到相關設計狀況採用的可靠度水平，並應考慮試驗結果的數量對相關參數統計不定性的影響。

附錄A 既有結構的可靠性評定

A．1 一般規定

A．1．1 本附錄適用於按有關標準或行業規則建造既有結構的可靠性評定。
A．1．2 在下列情況下宜進行既有結構的可靠性評定：
1 結構的使用時間超過規定的年限；
2 結構的用途或使用要求發生改變；
3 結構的使用環境惡化；
4 出現構件損傷、材料性能劣化或其他不利狀態；
5 對既有結構的可靠性有懷疑或有異議。
A．1．3 既有結構的可靠性評定可分為承載能力評定、適用性評定、耐久性評定和抵抗偶然作用能力評定。
A．1．4 既有結構的可靠性評定應採取以現行結構標準的基本規定為基準，對建築結構能力的狀況或發展趨勢予以評價的方式。
A．1．5 既有結構宜採取保全結構，延長結構使用年限的處理措施。

A．2 承載能力評定

A．2．1 既有結構承載能力的評定可分成結構體系和構件佈置、構件的連接和構造、作用與作用效應的分析、構件與連接的承載力等評定分項。
A．2．2 既有結構的結構體系和構件佈置，應以現行結構設計標準的規定為依據對實際狀況進行評定。
A．2．3 既有結構的連接和與構件承載力相關的構造，應以現行結構設計標準的規定為依據對實際狀況進行評定。
A．2．4 結構構件的承載能力應以本標準規定的可靠指標為基準，對構件承載力與作用效應之間的關係予以評價。
A．2．5 結構構件和連接的承載力可採取下列方法進行評定：
1 基於結構良好狀態的評定方法；
2 基於材料強度係數的方法；
3 基於抗力系數的評定方法；
4 基於可靠指標的構件承載力分項係數的評定方法；
5 重力荷載檢驗的評定方法等。
A．2．6 同時滿足下列要求的既有建築結構，可依據結構的良好狀態評定結構構件的承載力是否符合現行設計標準的規定：
1 結構的體系符合大震不倒的設防規定；
2 結構不存在爆炸和碰撞等偶然作用的影響；
3 結構未出現影響結構適用性的變形、裂縫、位移、振動等；
4 在評估使用年限內，結構上的作用和環境不會發生顯著的變化。
A．2．7 基於係數的構件承載力的評定應符合下列規定：
1 構件承載力計算模型的選取和相關參數的取值應符合下列規定：
1) 構件承載力計算模型應符合構件承載能力極限狀態的破壞模式；
2) 計算模型的幾何參數宜取構件的實際值；
3) 在計算構件承載力時，應考慮構件不可恢復性損傷對構件承載力的不利影響。
2 採用現行標準中材料強度係數的評定應符合下列規定：
1) 構件材料強度的標準值，應以實測數據為依據，按現行結構檢測標準規定的方法確定；
2) 在計算承載力時，對標準公式中反映模型不定性的參數應予保留。
3 當採用現行標準抗力系數評定時，可採取下列措施：
1) 材料的強度可取現場測試的平均值或最小值；
2) 當對計算模型進行過相關的研究時，可以利用現行標準公式的模型不定性儲備；
3) 經過驗證後，在計算模型中可增補對抗力有利因素的實際作用。
A．2．8 當可確定某類構件承載力的變異係數δR,j時，可採用下列基於可靠指標的構件承載力分項係數的評定方法：
1 將可靠指標β分解為作用效應的可靠指標βS和構件承載力的可靠指標βR；
2 該類構件承載力的分項係數γR可按下式確定：

γR＝1/(1－βRδR) (A．2．8)

3 分析構件承載力變異係數δR的模型可以作為構件承載力的計算模型；
4 計算模型中的材料強度和幾何參數可以採用實測數值；
5 計算模型宜採用模型不定性的保守措施。

A．2．9 對具備檢驗條件的結構或結構構件，可採用基於荷載檢驗的評定方法，荷載檢驗應符合下列規定：
1 檢驗荷載的形式應與結構承受的主要作用的情況基本一致；
2 除與有關方專門協商之外，檢驗荷載不宜大於荷載的設計值；
3 構件係數或材料強度係數對應的檢驗荷載的檢驗結果，可通過分析的方法確定。
A．2．10 既有結構構件承受的荷載可按國家現行相關標準的規定確定，並宜按下列規定進行符合實際情況的調整：
1 建築構配件等的自重荷載宜以現場實測數據為依據分析確定；
2 當樓面均布活荷載出現過大於有關標準限定的標準值時，應採用曾出現的最大值與該類構件所屬面積的乘積作為評定樓面均布活荷載的代表荷載；
3 對於雪荷載敏感的結構，應取當地記錄到的最大地面雪壓和重現期100年的雪壓值中的較大值作為基本雪壓；
4 對於風荷載敏感的結構，應取瞬時風速換算的風壓和重現期100年的風壓中的較大值作為基本風壓。

A．2．11 構件的作用效應，應按下列規定計算確定：
1 結構構件的作用效應宜在荷載或作用組合後計算確定；
2 在計算分析既有結構的作用效應時宜考慮構件的軸線偏差、尺寸偏差、安裝偏差和不可恢復性變形等的不利影響；
3 在作用效應的計算後，應考慮由結構分析模型造成的作用效應的不定性；
4 當不能確定作用效應的不定性時，可採用所有可能出現不利組合效應的包絡作為作用效應的評定值。

A．3 適用性評定

A．3．1 既有結構的適用性應包括正常使用極限狀態和結構維繫建築功能的能力等分項。

A．3．2 結構構件正常使用極限狀態應以現行結構設計標準限定的變形和位移值為基準對結構構件的狀況進行評定。

A．3．3 結構構件的變形和位移等狀況可通過現場檢測確定；現場檢測時應區分施工偏差和構件的變形或位移。

A．3．4 當結構構件的變形或位移不能通過現場檢測確定時，應採用結構分析的方法計算確定。

A．3．5 當結構的位移或變形對建築的使用功能構成影響時，應評定為結構構件維繫建築功能的能力不足。

A．4 耐久性評定

A．4．1 既有結構的耐久性評定，應以判定結構相應耐久年限與評估使用年限之間的關係為目的。
A．4．2 既有結構耐久性極限狀態的標誌或限值應按有關標準的規定確定。
A．4．3 既有建築結構耐久性的評定應實施下列現場檢測：
1 確定已出現耐久性極限狀態標誌的構件和連接；
2 測定構件材料性能劣化的狀況；
3 測定有害物質的含量或侵入深度；
4 確定環境侵蝕性的變動情況。
A．4．4 對於已經出現耐久性極限標誌的構件或連接，應進行構件承載力的評定和適用性評定，在評定時應考慮不可恢復性損傷對承載力和適用性的實際影響。
A．4．5 對於未出現耐久性極限狀態標誌和未達到限值的構件和連接，可推定耐久年數。
A．4．6 結構構件的耐久年數可採取下列方法推定：
1 經驗的方法；
2 依據實際劣化情況驗證或校準已有劣化模型的方法；
3 基於快速檢驗的方法；
4 其他適用的方法等。

A．5 抵抗偶然作用能力的評定

A．5．1 既有建築結構的偶然作用包括其可能遭受的罕遇地震、洪水、爆炸、非正常撞擊、火災等。
A．5．2 既有結構抵抗偶然作用的能力，宜從結構體系與構件佈置、連接與構造、承載力、防災減災和防護措施等方面綜合評定。
A．5．3 對於罕遇地震可採取下列方法予以評定：
1 按現行標準對建築物的總高度、層數、高寬比等限制要求和結構構造措施進行抗倒塌能力的評定；
2 採取結構分析的方法對結構整體的變形限值和薄弱層變形限值予以評定。
A．5．4 對於可能受到洪水影響的既有結構，除應考慮洪水的衝擊作用和浸泡作用外，還應考慮洪水對地基的影響。
A．5．5 對於發生在內部的爆炸等偶然作用應進行下列三種境況的評定：
1 爆炸發生時和發生後，避免結構出現整體倒塌的能力或個別構件破壞後避免結構出現連續倒塌的能力；
2 爆炸發生時，避免建築內部人員受到嚴重傷害的防護措施；
3 減小爆炸對周邊建築影響的措施。
A．5．6 當既有建築結構周邊有爆炸源時，應評價避免爆炸造成人員受到傷害的防護措施和結構及圍護結構避免破壞的能力。
A．5．7 對於發生在建築內部和外部的撞擊，應進行下列評定：
1 防止撞擊發生的措施和減小撞擊作用效應的措施；
2 結構局部破壞或個別構件喪失承載力，避免結構出現局部坍塌或連續倒塌的能力。
A．5．8 對於發生在建築內部的火災，可進行下列評定：
1 對於未設置噴淋設施的建築，可評價可燃物全部燃燒的持續時間與結構構件耐火極限的關係；
2 對於設置噴淋設施的建築，應評價煙感和噴淋設施的有效性；
3 建築內的排煙措施和疏散措施。

A．5．9 在具有較多可燃物附近的建築結構，應進行下列評定：
1 建築的防火間距；
2 建築的結構和外圍護結構的可燃性和防火能力；
3 人員疏散的通道。


附錄B 結構整體穩固性

B．1 一般規定

B．1．1 本附錄適用於偶然荷載引起的結構整體穩固性的設計。對於與火災、極度腐蝕等非荷載相關的結構整體穩固性，可按相關標準的規定執行；對於設計、施工、使用中可能出現的錯誤和疏忽，應通過嚴格管理控制。
B．1．2 進行結構整體穩固性設計時，應區分與結構整體穩固性有關的偶然作用的類型。當幾個偶然作用同時出現或相繼出現時，應考慮這些偶然作用的聯合影響和後續影響。

B．1．3 影響結構整體穩固性的偶然作用類型可按下列形式劃分：
1 由自然或一般人類活動引起的危險；
2 蓄意破壞和恐怖襲擊等故意的或人為製造的危險；
3 錯誤和疏忽；
4 其他引起結構連續倒塌的作用。

B．2 設計原則

B．2．1 結構設計前應分析結構各種潛在的危險源。結構選址應避讓各種危險源。對於結構附近可能有危險源或結構使用中存在危險源的情況，設計中應考慮採取相應的防控措施，避免或控制偶然事件的發生，或減輕偶然作用的強度。
B．2．2 應對結構進行概念設計，選取對整體穩固性有利的結構形式，並採取有效的構造措施。
B．2．3 結構應具有較高的冗餘度和多條明確的荷載傳遞路徑，一條荷載傳遞路徑失效後，應具有將荷載傳遞到其他路徑的能力。
B．2．4 結構、結構構件或連接應具有保持結構整體穩定需要的變形能力和延性性能。
B．2．5 結構設計應明確關鍵構件和非關鍵構件，關鍵構件應能承受規定的偶然荷載或採取適當的保護措施。
B．2．6 對於允許發生局部破壞的結構，局部破壞應控制在不引起結構整體倒塌的程度和範圍內。

B．3 設計方法

B．3．1 結構整體穩固性設計應包括概念設計、構造處理和計算分析，並可採用下列方法：
1 控制事件法；
2 抵抗特定荷載法；
3 替代路徑法，包括提供拉桿等；
4 減輕後果法；
5 其他保持結構整體穩固性的方法。
B．3．2 採用抵抗特定荷載法進行設計時，應驗算偶然作用使結構關鍵構件失效後，受損的結構仍具有保持整體穩固性的能力，其中作用效應設計值應按本標準式(8．2．5)確定；偶然事件發生後受損結構整體穩固性驗算宜包括結構承載力和變形驗算，作用效應設計值可按本標準式(8．2．5)確定。
B．3．3 考慮材料性能的線性和非線性、結構幾何性能的線性和非線性時，結構整體穩固性可採用線性靜力方法、非線性靜力方法和非線性動力方法進行計算。採用線性靜力方法和非線性靜力方法進行計算時，應考慮動力效應的影響。結構材料性能可按動態性能考慮，針對不同的情況材料性能可採用設計值、標準值或平均值。
B．3．4 可按本標準表3．2．1規定的安全等級對結構進行整體穩固性設計。安全等級為三級的結構，可只進行概念設計和構造處理；安全等級為二級的結構，除應進行概念設計和構造處理外，可採用線性靜力方法進行計算；安全等級為一級的結構，除應進行概念設計和構造處理外，宜採用非線性靜力方法或非線性動力方法進行計算，也可採用線性靜力方法進行計算。

B．4 安全管理與評估

B．4．1 結構使用過程中應進行安全管理，控制和避免各種偶然事件的發生或減輕偶然事件對結構整體穩固性的影響。

B．4．2 結構維修、加固不應削弱已有的荷載傳遞路徑，結構用途變更應對結構的整體穩固性重新進行評估。

B．4．3 結構整體穩固性評估可根據不同的目的在結構設計、建造和不同的使用階段進行。結構整體穩固性評估應包括偶然事件評估和結構抗連續倒塌能力評估。

B．4．4 偶然事件評估應包括可能發生偶然事件的類型、偶然事件可能發生的位置及偶然作用可能的強度或等級，當有條件時應採用概率方法進行評估。

B．4．5 結構抗連續倒塌評估可根據假想的結構連續倒塌情景進行，包括針對所考慮偶然事件結構發生局部破壞的可能性、破壞的形式、破壞的範圍及造成的人員傷亡、經濟損失和社會影響。

B．4．6 根據結構不同階段的整體穩固性評估結果，應對結構採取相應的抗連續倒塌措施。


附錄C 耐久性極限狀態設計

C．1 一般規定

C．1．1 結構的設計使用年限應根據建築物的用途和環境的侵蝕性確定。

C．1．2 結構的耐久性極限狀態設計，應使結構構件出現耐久性極限狀態標誌或限值的年限不小於其設計使用年限。

C．1．3 結構構件的耐久性極限狀態設計，應包括保證構件質量的預防性處理措施、減小侵蝕作用的局部環境改善措施、延緩構件出現損傷的表面防護措施和延緩材料性能劣化速度的保護措施。

C．2 設計使用年限

C．2．1 結構的設計使用年限，宜按本標準表3．3．3的規定採用。

C．2．2 必須定期塗刷的防腐蝕塗層等結構的設計使用年限可為20年～30年。

C．2．3 預計使用時間較短的建築物，其結構的設計使用年限不宜小於30年。

C．3 環境影響種類

C．3．1 結構的環境影響可分成無侵蝕性的室內環境影響和侵蝕性環境影響等。
C．3．2 當把無侵蝕性的室內環境視為一個環境等級時，宜將該等級分為無高溫的室內乾燥環境和室內潮溼環境兩個層次。
C．3．3 根據環境侵蝕性的特點，宜按下列作用分類：
1 生物作用；
2 與氣候等相關的物理作用；
3 與建築物內外人類活動相關的物理作用；
4 介質的侵蝕作用；
5 物理與介質的共同作用。
C．3．4 當結構構件出現下列損傷時宜歸為生物作用：
1 木結構的蟲蛀和腐朽等；
2 植物根系造成的損傷；
3 動物糞便和細菌等造成的損傷。
C．3．5 結構構件出現下列損傷時宜歸為與氣候等相關的物理作用：
1 構件或材料出現凍融損傷；
2 出現因風沙造成的磨損和水的流動造成的損傷；
3 太陽輻射及相應的高溫造成聚合物材料的老化；
4 溫度、溼度等的變動使結構構件出現變形和開裂；
5 溫度、溼度等的變動使結構構件中的介質膨脹；
6 隨水分進入構件材料內部的介質結晶造成的損傷等。


C．3．6 結構構件出現的下列損傷時宜歸為與人類生產相關的物理作用：
1 高速氣流或水流造成的空蝕；
2 人員活動造成的磨損；
3 撞擊造成的損傷；
4 設備高溫、高溼等造成的損傷；
5 設備設施等造成的有機材料的老化等。
C．3．7 介質的侵蝕作用可分成下列幾種類型：
1 環境中或生產過程中的酸性介質或鹼性介質直接造成的損傷；
2 環境中或生產過程中的介質與構件出現化學不相容的現象；
3 環境中或生產過程中的介質加速高分子聚合物材料的老化或性能劣化等。

C．4 耐久性極限狀態

C．4．1 各類結構構件及其連接，應依據環境侵蝕和材料的特點確定耐久性極限狀態的標誌和限值。
C．4．2 對木結構宜以出現下列現象之一作為達到耐久性極限狀態的標誌：
1 出現黴菌造成的腐朽；
2 出現蟲蛀現象；
3 發現受到白蟻的侵害等；
4 膠合木結構防潮層喪失防護作用或出現脫膠現象；
5 木結構的金屬連接件出現鏽蝕；
6 構件出現翹曲、變形和節點區的幹縮裂縫。

C．4．3 對鋼結構、鋼管混凝土結構的外包鋼管和組合鋼結構的型鋼構件等，宜以出現下列現象之一作為達到耐久性極限狀態的標誌：
1 構件出現鏽蝕跡象；
2 防腐塗層喪失作用；
3 構件出現應力腐蝕裂紋；
4 特殊防腐保護措施失去作用。

C．4．4 對鋁合金、銅及銅合金等構件及連接，宜以出現下列現象之一作為達到耐久性極限狀態的標誌：
1 構件出現表觀的損傷；
2 出現應力腐蝕裂紋；
3 專用防護措施失去作用。

C．4．5 對混凝土結構的配筋和金屬連接件，宜以出現下列狀況之一作為達到耐久性極限狀態的標誌或限值：
1 預應力鋼筋和直徑較細的受力主筋具備鏽蝕條件；
2 構件的金屬連接件出現鏽蝕；
3 混凝土構件表面出現鏽蝕裂縫；
4 陰極或陽極保護措施失去作用。

C．4．6 對砌築和混凝土等無機非金屬材料的結構構件，宜以出現下列現象之一作為達到耐久性極限狀態的標誌或限值：
1 構件表面出現凍融損傷；
2 構件表面出現介質侵蝕造成的損傷；
3 構件表面出現風沙和人為作用造成的磨損；
4 表面出現高速氣流造成的空蝕損傷；
5 因撞擊等造成的表面損傷；
6 出現生物性作用損傷。

C．4．7 對聚合物材料及其結構構件，宜以出現下列現象之一作為達到耐久性極限狀態的標誌：
1 因光老化，出現色澤大幅度改變、開裂或性能的明顯劣化；
2 因高溫、高溼等，出現色澤大幅度改變、開裂或性能的明顯劣化；
3 因介質的作用等，出現色澤大幅度改變、開裂或性能的明顯劣化。
C．4．8 對具有透光性要求的玻璃構配件，宜以出現下列現象之一作為達到耐久性極限狀態的標誌：
1 結構構件出現裂紋；
2 透光性受到磨蝕的影響；
3 透光性受到鳥類糞便影響等。
C．4．9 結構構件耐久性極限狀態的標誌或限值及其損傷機理，應作為採取各種耐久性措施的依據。

C．5 耐久性極限狀態設計方法和措施

C．5．1 建築結構的耐久性可採用下列方法進行設計：
1 經驗的方法；
2 半定量的方法；
3 定量控制耐久性失效概率的方法。
C．5．2 對缺乏侵蝕作用或作用效應統計規律的結構或結構構件，宜採取經驗方法確定耐久性的系列措施。
C．5．3 採取經驗方法保障的結構構件耐久性宜包括下列技術措施：
1 保障結構構件質量的殺蟲、滅菌和乾燥等技術措施；
2 避免物理性作用的表面抹灰和塗層等技術措施；
3 避免雨水等衝淋和浸泡的遮擋及排水等技術措施；
4 保障結構構件處於乾燥狀態的通風和防潮等技術措施；
5 推遲電化學反應的鍍膜和防腐塗層等技術措施以及陰極保護等技術措施；
6 作出定期檢查規定的技術措施等。

C．5．4 具有一定侵蝕作用和作用效應統計規律的結構構件，可採取半定量的耐久性極限狀態設計方法。
C．5．5 半定量的耐久性極限狀態設計方法宜按下列步驟確定環境的侵蝕性：
1 環境等級宜按侵蝕性種類劃分；
2 環境等級之內，可按度量侵蝕性強度的指標分成若干個級別。
C．5．6 半定量設計方法的耐久性措施宜按下列方式確定：
1 結構構件抵抗環境影響能力的參數或指標，宜結合環境級別和設計使用年限確定；
2 結構構件抵抗環境影響能力的參數或指標，應考慮施工偏差等不定性的影響；
3 結構構件表面防護層對於構件抵抗環境影響能力的實際作用，可結合具體情況確定。
C．5．7 具有相對完善的侵蝕作用和作用效應相應統計規律的結構構件且具有快速檢驗方法予以驗證時，可採取定量的耐久性極限狀態設計方法。
C．5．8 當充分考慮了環境影響的不定性和結構抵抗環境影響能力的不定性時，定量的設計應使預期出現耐久性極限狀態標誌的時間不小於結構的設計使用年限。


附錄D 質量管理

D．1 質量控制要求

D．1．1 材料和構件的質量可採用一個或多個質量特徵表達。在各類材料的結構設計與施工標準中，應對材料和構件的力學性能、幾何參數等質量特徵提出明確的要求。
D．1．2 材料和構件的合格質量水平，應根據本標準規定的結構構件可靠指標確定。
D．1．3 材料宜根據統計資料，按不同質量水平劃分等級。等級劃分不宜過密。對不同等級的材料，設計時應採用不同的材料性能的標準值。
D．1．4 對建築結構應實施為保證結構可靠性所必需的質量控制。建築結構的各項質量控制要求應由有關標準作出規定。建築結構的質量控制應包括下列內容：
1 勘察與設計的質量控制；
2 材料和製品的質量控制；
3 施工的質量控制；
4 使用和維護的質量控制。

D．1．5 勘察與設計的質量控制應符合下列規定：
1 勘察資料應符合工程要求，數據準確，結論可靠；
2 設計方案，基本假定和計算模型合理，數據運用正確；
3 圖紙和其他設計文件符合有關規定。
D．1．6 為進行施工質量控制，在各工序內應實行質量自檢，各工序間應實行交接質量檢查。對工序操作和中間產品的質量，應採用統計方法進行抽查；在結構的關鍵部位應進行系統檢查。
D．1．7 材料和構件的質量控制應包括生產控制和合格控制，並應符合下列規定：
1 生產控制：在生產過程中，應根據規定的控制標準，對材料和構件的性能進行經常性檢驗，及時糾正偏差，保持生產過程中質量的穩定性；
2 合格控制：在交付使用前，應根據規定的質量驗收標準，對材料和構件進行合格性驗收，保證其質量符合規定。
D．1．8 合格控制可採用抽樣檢驗的方法進行。各類材料和構件應根據其特點制定具體的質量驗收標準，其中應明確規定驗收批量、抽樣方法和數量、驗收函數和驗收界限等。質量驗收標準宜在統計理論的基礎上制定。
D．1．9 對生產連續性較差或各批間質量特徵的統計參數差異較大的材料和構件，在制定質量驗收標準時，必須控制用戶方風險率。計算用戶方風險率時採用的極限質量水平，可按各類材料結構設計標準的有關規定和工程經驗確定；僅對連續生產的材料和構件，當產品質量穩定時，可按控制生產方風險率的條件制定質量驗收標準。
D．1．10 當一批材料或構件經抽樣檢驗判定為不合格時，應根據有關的質量驗收標準對該批產品進行復查或重新確定其質量等級，或採取其他措施處理。

D．2 設計審查及施工檢查

D．2．1 建築結構應進行設計審查與施工檢查，設計審查與施工檢查的要求應符合國家現行有關標準的規定。

6 對選定的所有代表性結構或結構構件、所有作用分項係數γG和γQ的範圍以0．1或0．05的級差，優化確定γR；選定一組使按分項係數表達式設計的結構或結構構件的可靠指標β與目標可靠指標βt最接近的分項係數γG、γQ和γR；
7 根據以往的工程經驗，對優化確定的分項係數γG、γQ和γR進行判斷，並進行調整。

E．5．3 對安全等級為二級的結構或結構構件，當永久作用起有利作用時，結構或結構構件分項係數表達式中的永久作用應取負號，根據第E．5．2條已經選定的永久作用起不利作用時的可變作用分項係數γQ和抗力分項係數γR，以0．1或0．05為級差優化確定永久作用分項係數γG。

E．6 可變作用組合值係數的確定方法

E．6．1 可變作用組合值係數的確定應符合下列原則:
對兩種或兩種以上可變作用參與組合的情況，基於已確定的可變作用分項係數γG、γQ和抗力分項係數γR，組合值係數的確定應使按分項係數表達式設計的結構或結構構件的可靠指標β與目標可靠指標βt具有最佳的一致性。
E．6．2 可變作用組合值係數可按下列步驟確定：
1 以安全等級為二級的結構或結構構件為基礎，選定代表性的結構或結構構件或破壞方式、由一個永久作用和兩個或兩個以上可變作用組成的組合和常用的主導可變作用標準值效應與永久作用標準值效應的比值、伴隨可變作用標準值效應與主導可變作用標準值效應的比值；
2 根據已經確定的作用分項係數γG、γQ，計算不同結構或結構構件、不同作用組合和常用作用效應比下的抗力設計值；
3 根據已經確定的抗力分項係數γR，計算不同結構或結構構件、不同作用組合和常用作用效應比下的抗力標準值；
4 計算不同結構或結構構件、不同作用組合和常用作用效應比下的可靠指標β；
5 對選定的所有代表性結構或結構構件、作用組合和常用的作用效應比，優化確定組合值係數ψc，使按分項係數表達式設計的結構或結構構件的可靠指標β與目標可靠指標βt具有最佳的一致性；
6 根據以往的工程經驗，對優化確定的組合值係數ψc進行判斷，並進行調整。


附錄F 試驗輔助設計

F．1 一般規定

F．1．1 試驗輔助設計應符合下列規定：
1 在試驗進行之前，應制定試驗方案。試驗方案應包括試驗目的、試件的選取和製作，還應包括試驗實施和評估等所有必要的說明。
2 制定試驗方案前，應預先進行定性分析，確定所考慮結構或結構構件性能的可能臨界區域和相應極限狀態標誌。
3 試件應採用與構件實際加工相同的工藝製作。
4 按試驗結果確定設計值時，應考慮試驗數量的影響。
F．1．2 應通過換算或修正係數考慮試驗條件與結構實際條件的不同。換算係數η應通過試驗或理論分析確定。影響換算係數η的主要因素應包括尺寸效應、時間效應、試件的邊界條件、環境條件、工藝條件等。
