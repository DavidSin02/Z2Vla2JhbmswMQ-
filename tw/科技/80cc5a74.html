國內開發者在 GitHub 上開源 LeetCode 刷題模板，火爆全網

2020-11-14
| 科技

許多開發者在初學算法時，都會跑到 LeetCode 網站上面刷題，也有一些開發者為了過微軟、Google、Facebook 等國際大企的面試，會選擇刷 LeetCode 來快速提升自己的編程能力與算法能力，以便順利通過面試。

於是，刷 LeetCode 的做法開始在技術圈傳播開來，程序員彼此見面聊天時也會時不時問候幾句，"你今天刷 LeetCode 了嗎

"。

因此，GitHub 上各類 LeetCode 項目開始層出不窮，此前我們曾經在公眾號上跟大家介紹過幾個比較實用的 LeetCode 項目，今天在這裡，小 G 再跟大家分享一個近期比較火的項目：algorithm-pattern。

今年 4 月的時候，國內一位工程師 greyireland 由於要找工作的原因，便開始刷 LeetCode 上的算法題，後面花了 6 周左右的時間，他斷斷續續刷完了 LeetCode 240 道題，最終也如願拿到了字節跳動 offer。

為了更好的與開發者分享自己的刷題技巧，greyireland 在 GitHub 上開源了一套 LeetCode 刷題模板：algorithm-pattern，主要記錄他通過各種刷題文章、專欄、視頻等內容總結出的刷題心得。

這份模板主要基於 Go 語言，編寫了入門篇、數據結構篇、基礎算法篇、算法思維等四大篇章。

通過這幾大篇章，你可以學到關於算法入門、二叉樹、鏈表、二分搜索、排序算法、遞歸思維、回溯法等算法知識。

在每篇文章下面，作者也都有留幾道題供讀者訓練學習：

項目發佈後到現在已經過去四五天時間了，目前其 GitHub Star 數已累積突破 1000，此刻的你如何還陷在 LeetCode 泥潭中，不妨試下這個項目吧。

最後，附上該項目的 GitHub 地址與在線閱讀地址，感興趣的同學可以開始學起來了。

GitHub：

https://github.com/greyireland/algorithm-pattern/

在線閱讀：

https://greyireland.gitbook.io/algorithm-pattern/
