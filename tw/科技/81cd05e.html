原創： 老表 簡說Python

今日問題

"""目標：寫一段程序，從無序鏈表中移除重複項例如：輸入-> 1->0->3->1->4->5->1->8輸出-> 1->0->3->4->5->8""""""Goal: write a program to remove duplicates from the unordered listFor example:Input - > 1 - > 0 - > 3 - > 1 - > 4 - > 5 - > 1 - > 8Output - > 1 - > 0 - > 3 - > 4 - > 5 - > 8"""

首先我們寫好鏈表的基本操作，在a_0_base.py文件中，目前包含對鏈表的定義類，初始化函數，遍歷函數。（帶頭結點）

# -*- coding: utf-8 -*-"""@author = 老表@date = 2019-10-19@個人微信公眾號 : 簡說Python"""# 鏈表數據結構定義class ListNode: def __init__(self, x): self.data = x self.next = Noneclass ListOperation: # 根據鏈表數據初始化鏈表 @staticmethod def init_list(n_list): # 初始化一個頭指針 head = ListNode("head") cur = head for i in n_list: node = ListNode(i) cur.next = node cur = node # 相當於 cur = cur.next，後移 return head # 遍歷鏈表 @staticmethod def ergodic_list(head): cur = head.next while cur: print(cur.data) cur = cur.next

解題

開始程序前需提前導入前面寫好的鏈表基本操作包a_0_base。

from Linked_list.a_0_base import ListOperation

方法一：遞歸去重

"""Method One : 遞歸去重核心思想：每次遍歷去重前，都先對子鏈表進行去重處理，這樣去重的子鏈表都會產生一個子鏈表，直到遞歸到尾結點，此時開始回溯，每次回溯，會完成一個子鏈的去重，直到回溯到頭結點，鏈表去重完成。時間複雜度：O(N^2)空間複雜度：O(1)"""

代碼

def remove_duplicates_one(head): """ :type head: ListNode :rtype: ListNode """ if not head.next: # 空鏈表或者遍歷到最後一個結點 return head # 返回頭結點或者開始回溯 cur_node = head # 記錄頭結點 head.next = remove_duplicates_one(head.next) # 遞歸，往後遍歷 # print("此時cur_node=", cur_node.data) # print("此時head=", head.data) pointer = head.next # 記錄後繼結點 # print("此時pointer=", pointer.data) # 遍歷查找 while pointer: # 遍歷當前子鏈表 if head.data == pointer.data: cur_node.next = pointer.next # 刪除相同結點 pointer = cur_node.next # 後移，遍歷子鏈表 else: pointer = pointer.next # 後移，遍歷子鏈表 cur_node = cur_node.next # 當前結點後移 # print("此時head=", head.data) # print("____________________________") return head # 返回當前子鏈表頭結點

方法二：順序刪除

"""Method Two : 順序刪除核心思想：你要是搞清楚方法一，這個就不難理解了。直接利用雙循環，一個慢指針，一個快指針，匹配到快慢指針的數據值相同時就刪除該快指針，然後繼續遍歷快指針，直至結束，然後慢指針後移，繼續遍歷。時間複雜度：O(N^2)空間複雜度：O(1)"""

代碼

def remove_duplicates_two(head): """ :type head: ListNode :rtype: ListNode """ if not head.next: # 空鏈表或者遍歷到最後一個結點 return head # 返回頭結點或者開始回溯 slow_node = head.next # 記錄慢指針（外層循環） while slow_node: # 外層遍歷 fast_node = slow_node.next # 記錄快指針（內層遍歷） cur_node = slow_node # 記錄當前結點，方便刪除結點 while fast_node: # 內層遍歷 if fast_node.data == slow_node.data: # 在子鏈中查找值與slow_node一樣的結點（值重複結點） cur_node.next = fast_node.next # 刪除重複結點（讓當前結點的next指向重複結點的next） else: cur_node = fast_node # 當前結點後移，繼續遍歷子鏈 fast_node = fast_node.next # 後移，繼續遍歷子鏈（快指針） slow_node = slow_node.next # 後移，繼續遍歷（慢指針） return head # 返回當前子鏈表頭結點

方法三：空間換時間

"""Method Three : 空間換時間核心思想：建一個輔助棧記錄鏈表數據項，遍歷鏈表，1、如果當前結點的數據項在輔助棧中，則刪除該結點2、如果當前結點數據項不在輔助棧中，則該結點數據項進行入棧操作3、繼續遍歷，直至遍歷結束。輔助棧可選數據類型：元祖，列表，字典，集合其中字典和集合的查找時間複雜度為O(1),字典內部是有哈希表（哈希函數+哈希衝突表）構成，故查找時間複雜度為O(1)，Python裡集合的內部實現是字典，故查找時間複雜度也為O(1)。時間複雜度：O(N)空間複雜度：O(N)"""

代碼

def remove_duplicates_three(head): """ :type head: ListNode :rtype: ListNode """ if not head.next: # 空鏈表或者遍歷到最後一個結點 return head # 返回頭結點或者開始回溯 run_node = head.next.next # 從第二個結點開始遍歷（因為第一個結點肯定不存在重複問題） pre_node = head.next # 記錄第一個結點，後面做前驅結點 dict_all = {pre_node.data: "node"} # 直接把第一個結點數據項加入字典，鍵值任意給 while run_node: # 遍歷鏈表 if run_node.data in dict_all: # 判斷當前遍歷結點值在不在字典中 pre_node.next = run_node.next # 刪除重複結點（讓當前結點的next指向重複結點的next） else: # 不在字典中，則入棧，並讓前驅結點後移 dict_all[run_node.data] = "node" pre_node = run_node run_node = run_node.next # 後移遍歷 # print(dict_all) return head # 返回當前子鏈表頭結點
