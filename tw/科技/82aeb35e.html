技嘉展示新款四路M.2轉接卡：PCIe 4.0 x16接口 自帶風扇和散熱裝甲

2020-11-14
| 科技

鑑於許多主板只提供 2~3 個 M.2 存儲插槽，一些廠商想到了通過 PCIe 插槽進行擴展的方案。對於高端玩家來說，他們不僅需要大量的 PCIe 通道，還希望支持最新的 PCIe 4.0 規範，以獲得最佳的理論性能帶寬。有趣的是，在本屆臺北電腦展（Computex 2019）上，我們就見到了來自技嘉的新款四路 M.2 轉接卡。它不僅採用了 PCIe 4.0 x16 插槽，還附帶了散熱風扇和厚重的散熱裝甲（帶背板）。

（題圖 via AnandTech）

乍一看，你或許會以為這就是一款採用公版渦扇設計的顯卡，直到發現它並沒有顯示輸出或外接的 PCIe 供電接口。

此前，許多主板 OEM 廠商僅提供了基於 PCIe 3.0 的雙路或四路 M.2 轉接卡，技嘉是我們見到第一家升級到 PCIe 4.0 的四路 M.2 轉接卡廠商。

升級到 PCIe 4.0 x16 插槽之後，該轉接卡可為每個 M.2 插槽分配 PCIe 4.0 x4 的帶寬。當然，該卡也向下兼容 PCIe 3.0 x16 。

鑑於四塊 M.2 SSD的功率不大可能超過 32W，主板自帶的 75W 功率輸出也足夠了。不過為了幫助 M.2 SSD 散熱，金屬外殼+全尺寸背板+渦扇的設計也很是亮眼。

目前市面上唯一支持 PCIe 4.0 的消費級平臺，只有 AMD 的銳龍 3000 系列處理器。儘管對於大部分消費者來說，都不大可能將常用於顯卡的 x16 插槽分享給額外的 M.2 存儲。

最後，我們預計 PCIe 4.0 規範可在新一代線程撕裂者（Threadripper）平臺上發揮更大的作用。技嘉表示，這款擴展卡專為 AMD 平臺打造，因為當前英特爾平臺還缺乏適當的功能支持。
