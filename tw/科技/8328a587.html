業餘無線電入門如何選購設備，你需要多大的功率

2020-11-14
| 科技

在業餘無線電裡，有兩個極端，有的朋友熱心於 QRO ，功率輸出最高可達法定限制的最高值 1000W ，有的熱衷於QRP，5W 的輸出或者更少。大多數業餘無線電愛好者的功率都介於 20~100W 之間，100W 功率是比較典型的收發機功率。

有的時候必須使用大功率才能保持穩定的通聯，大功率在某些波段的遠距離通信競賽中更有競爭力。多數情況下，100W 的功率足以提供非常好的通聯效果。有些業餘無線電愛好者認為廉價的低功率短波設備非常有吸引力，特別是那些以套件形式出現的，既鍛鍊動手能力，又可以獲得成就感。

雖然我一直說 QRP 很簡單，值得嘗試，確實值得嘗試。但是，對於剛剛開始嘗試短波的初學者，QRP 經常會帶來嚴重的致命性打擊。無論如何，在你嘗試 QRP 之前，我都推薦你首先積累一些 100W 功率通聯的經驗，這樣一來你就知道在短波頻段上會有哪些現象。它們和 UV 段的情況會有很大的不同，短波上功率因素對信號的影響比 UV 段要弱一些，或者說提升功率對於通聯效果的改善沒有 UV 段那麼明顯。

當你試圖在 UV 段呼叫幾百公里遠的電臺時，功率會比較重要，當使用性能優良的天線進行本地通信時，典型的 5W 手持設備或 25W 的車載電臺的功率完全夠用。

如果你的興趣點是 FM 語音、 APRS 或者與本地業餘無線電愛好者進行數據無線電通信，那麼一部 UV 雙段的收發機就可以滿足你的需求。FM 收發機有手持設備，也有車載移動設備。如果你有 12V 的電池或合適的電源，那麼在家也可以使用車載電臺作為固定臺。

在家裡使用的話，還可以選購性能更加強大，功能更豐富的固定式收發機。比如我目前使用的八重洲 FT-991 和 icom IC-7100 ，作為短波機它們應該還算便攜，不過我主用 UV 段全模式。手臺出於便攜性的考慮，天線不可能做的很大，車載電臺會好一些，家用設備天線就可以搞的很大，我目前在用 V 段 5 單元八木，U 段 2X10 單元八木配合雲臺進行水平，仰角旋轉。

手持式收發機俗稱手臺，對講機。手持式收發機的工作功率一般介於 0.05~5W，甚至更高。在此不建議使用很大的功率，滿足使用即可，低功率也會節省電池的電量，延長使用時間。大多數手持設備都提供高/低功率開關。好的二手手持收發機的價格通常低於 1000 元，而全新的手臺的起步價格通常為 1500。當然了，也有很多廉價設備可以選擇，比如寶鋒，歐訊，泉盛之類的，全新的設備價格在 100-1500 左右。如果你在汽車內部使用手持設備，你很肯定會對原裝天線的性能非常失望，此時在車外加裝一個移動式天線將會有好得多的性能。

UV 段車載電臺的功率一般介於 5- 50W 之間，請注意法律法規允許最大為 25W ，使用天線一般為卡邊或吸盤安裝移動天線。在使用中繼的情況下，任何大於 10W 的功率很可能都是沒必要的。二手的車載電臺價格通常在 500 以上，視具體情況而定。全新的車載電臺起步價格大約為 1300 元，功能更多的車載電臺價格高達四五千的也有。

固定電臺收發機，基地臺。使用電源或電池供電的固定式基站無線電臺。它也可以斷開連接，搬入汽車以便進行移動式通聯。利用電池對設備進行供電還有一個優點，在停電的時候可以繼續工作，我這裡常備一個 13.8V 40Ah 磷酸鐵鋰電池。此類設備多數是全波段全模式設備，包含短波，6 米，2 米和 70 釐米波段。如果你只打算在家進行短波通聯，這些可不是好的選擇。很多朋友不喜歡這些大而全的設備，我卻對它們情有獨鍾，我喜歡 UV 段的全模式操作。但是，國內的愛好者好像更喜歡短波多一些，對於 UV 段的操作只限於中繼，其它的內容涉及的比較少。所以我的全模式電臺，多數時候用於操作業餘衛星。

VHF 段數據包通信無線電。最常見的就是 APRS 自動分組數據交換系統，這可以實現短信息發送，當然幾乎所有的設備僅能發送字母和數字。你需要一部 2 米波段的電臺進行此項操作，除此之外你還需要一臺計算機或終端，還有一個終端節點控制器，也就是 TNC。有一些比較高端的手臺內置了 TNC ，可以直接連接電腦使用，有的手臺內置了 APRS 功能，在離開電腦的情況下可以直接進行 APRS 操作。

記住，高功率操作可能會產生射頻安全隱患。在車載電臺上，要把天線放置在遠離車廂的位置。對於更高功率的手臺來說，儘量使用手咪和天線延長線，以保證天線遠離你的雙眼和頭部。永遠都不要忽略這個問題。

短波設備

使用短波電臺，以及架設一個短波天線，比 UV 段更具有挑戰性。短波設備通常涉及多種模式和波段，設備價格也更加昂貴。短波頻段的天線，比手臺天線和車載天線大的多。和 UV 段不同，短波的天線更加多樣化，各種不同的電臺也更多。需要考慮的問題也更多，例如在業餘無線電方面的預算有多少，架設天線的尺寸和位置有沒有什麼限制。多數業餘無線電愛好者沒有辦法架設龐大的天線，他們使用室內天線或者隱藏的天線，工作起來困難重重。

在電臺選擇上，我不建議沒有任何動手維修能力的朋友購買古董電臺，雖然它們很便宜，一般花不到 1000 元，你不知道它經歷過什麼，不知道它何時會突然罷工。上面說過了，我的選擇是全波段的設備，多數朋友認為這樣的設備性能不好，認為設備需要專用。我不覺得全波段設備有什麼不好的，各種電臺都有它存在的價值和適用的人群。說實話我買了它們是玩 UV 段的，短波屬於附送的功能，而且我短波很少使用超過 5 W 的功率，雖然它們兩個都是 100 W 功率的電臺。請注意，不是 FT8 模式，不是 CW 模式，我使用的是 SSB 語音模式。

我的以前天線僅有一個簡易的單波段 DP ，去年使用 5W 功率在 20 米波段通聯了國內 1234567 區的電臺，還有一些日本電臺。今年我換上了一根 4 波段端饋天線，經常使用 2.5W ，1W 功率和國內以及周邊國家的 HAM 進行 SSB 語音通聯，使用 5W 還通聯了 10717 公里的美國亞利桑那電臺。有朋友說我電磁環境好，好啥啊，各波段常年 S9+。短波內容少一些，我對短波沒多大興趣，我覺得 UV 段的挑戰性更大。我喜歡玩不一樣的東西，短波玩的人太多了。
