世界各國核武器規模和核彈分類情況介紹

2020-11-14
| 科技

核武器，分幾個階段，原子彈，氫彈，中子彈（受控可調當量核效應彈）

彈頭有效負荷相同時氫彈威力是原子彈10倍

美國現役最大（民兵3）單彈頭100萬噸當量，700個運載工具，1500枚核彈頭

俄羅斯白楊M為100萬噸當量，700個運載工具，1500枚核彈頭

英國核力量在某種程度上是美國核力量的延伸，因為它的核彈是美國提供的，它有四艘彈道導彈核潛艇，一艘核潛艇有16枚核導彈，一枚導彈三個分彈頭，實際上它總共處於作戰狀況下的核彈頭應該是192枚，單彈頭30萬當量

法國擁有348枚核彈頭，包括288枚海基核彈（部署在四艘核潛艇上），50枚空基核巡航導彈，10枚空基核彈。單彈頭均不超過100萬噸。

中國東風5A為300萬噸(20枚)，東風31A為100萬當量（估計40枚），東方21為30萬當量（估計100枚）中國核彈頭數量估計350~400枚與法國相當。

印度核彈頭超過100枚，烈火5導彈攜帶1.5噸核彈頭，當量20萬噸，助爆型原子彈

巴基斯坦核彈頭100枚左右

以色列核彈頭100枚左右

朝鮮估計現在有造2~3枚小型化的能力

巴西既不承認有核武器，又不同意進行核查

伊朗不承認開發核武器，但又想保留核能力，並不與國際原子能機構完全合作

日本、德國等國家也具備核武器開發能力，但因政治原因而沒有相關項目

有效殺傷距離 = C * 爆炸當量^(1/3)...... (C為比例常數,^(1/3)為求立方根)

一般取比例常數為1.493885

當量為10萬噸時,

有效殺傷半徑 = 1.493885 * 10^(1/3) = 3.22千米

有效殺傷面積 = pi * 3.22 * 3.22 = 33平方千米

當量為30萬噸時

有效殺傷半徑 =4.7千米

有效殺傷面積 =70平方千米

當量為100萬噸時,

有效殺傷半徑 = 1.493885 * 100^(1/3) = 6.93千米

有效殺傷面積 = pi * 6.93 * 6.93 = 150平方千米

有效殺傷半徑 = 1.493885 * 300^(1/3) = 10.00千米

有效殺傷面積 = pi * 10.00* 10.00= 314平方千米

當量為1000萬噸時,

有效殺傷半徑 = 1.493885 * 1000^(1/3) = 14.93千米

有效殺傷面積 = pi * 14.93 * 14.93 = 700平方千米

朝鮮發展的是原子彈裡面最初級的槍式鈈彈，最大可能當量不超過4萬噸。印度那個就是這種，印度爆了好像有1,2萬噸。 但是這裡，實際爆炸的當量跟裝藥當量是不一樣的。比如設計裝藥當量是2萬噸，炸出來實測1萬噸，那就是50%的核燃料炸了，如果設計是2萬噸，只炸了1000噸，那就只有5%的核燃料炸了，這叫不成功，頂多算爆轟，不算核爆炸。是技術不成熟的表現。

高級一點的是鈾彈，這叫內爆式，複雜程度一下高了幾十倍，這個當量最大好像能到25萬噸，準確數字記不得了，但肯定不到50萬噸。但到10萬噸以上就會做得很大，飛機不好帶，火箭不好打的。中國自己上來就搞的這個，跳過了鈈彈階段。

這兩個階段都是原子核裂變，能量小，汙染大，所以叫原子彈，都是反應簡單，只求殺傷力和汙染，力求當量越大越好，殺傷範圍越大。是核武的初級階段。 這個階段不存在小型化，因為技術根本做不到。

第二個階段是氫彈，這個階段，是聚變彈，是用原子彈引爆氫彈，所以，這種蛋蛋，裡面有個原子彈，有個氫彈，很難做小，這才是小型化的難題，但只是體積小型化，而不是殺傷力小型化。

基本上，氫彈的威力是無上限的，從20萬噸到1億噸的都有過，老毛子就爆過設計一億，裁剪到五千萬的。 太陽實際就是個大氫彈。

再一步就是中子彈（或受控可調當量可調核效應彈）。這個階段，還是氫彈，但反應方向可能是爆炸衝擊波殺傷力，可能是放射線殺傷力，目標不同，對殺傷力的要求也不同，朝著可作戰術武器的方向去的。這個階段也要求小型化，低成本化，也會要求小殺傷力，但這個目標是用核彈大常規戰爭，低汙染。

現在各個核大國在搞的更高階的是非核彈引爆的氫彈，中子彈。這個能做得特別小，很考教技術。而原子彈，因為其裂變原理帶來的汙染，怎麼改小當量，都會被認為是戰略武器，不可能當戰術武器使用。
