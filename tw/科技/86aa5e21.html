「通信與廣電實務簡答知識彙總四」：施工安全管理

2020-10-29
| 科技

施工現場周圍環境的安全管理

1.對施工現場周圍人員、車輛的安全保護，主要涉及可能進入施工現場的人員、車輛的安全。其中包括:

(1)在敷設吊線時，路口、路邊等地段可能受到鋼絞線傷害的過往人員、車輛;

(2) 在開挖光纜溝、管道溝、接頭坑、人(於)孔坑、杆坑及拉線坑時，可能墜落摔傷的人員;

(3) 安裝路邊拉線可能掛傷的過往行人、掛壞的過往車輛。

2. 對其他設施的安全保護，主要涉及在用設備、設施的安全。其中包括:

(1)設備安裝工程施工過程中，在用設備的安全;

(2) 線路工程施工過程中，其他纜線、管線及設施的安全;

(3) 管道建設工程施工過程中，地下各種纜線、管線及設施的安全。

安全生產費的管理

1.提取方法

建設工程施工企業以建築安裝工程造價為計提依據，通信工程按1. 5%標準提取。建設工程施工企業提取的安全費用列入工程造價，在競標時，不得刪減，列人標外管理。總包單位應當將安全費用按比例直接支付分包單位並監督使用，分包單位不再重複提取。

2. 管理原則

施工企業的安全生產費用應當按照"企業提取、政府監管、確保需要、規範使用"的原則進行管理。

3. 管理要求

企業提取的安全費用應當專戶核算，按規定範圍安排使用，不得擠佔、挪用。年度結餘資金結轉下年度使用，當年計提安全費用不足時，超出部分按正常成本費用渠道列支。

不同施工環境應注意的安全問題

1.山區施工時，應採取措施保證施工人員的安全、保證車輛及機械設備的安全、保證工程材料的安全;應防止施工人員滑倒、摔傷，防止車輛側翻、制動裝置失靈，防止機械設備摔壞、損壞、丟失。臨時設施應遠離河道、峭壁、低窪地段等存在安全隱患的地方。

2. 高溫天氣施工時，應採取措施防止施工人員中暑，氣溫過高時應停止施工。

3. 低溫雨雪季節施工時，應採取措施保證施工人員及車輛的安全，應預防人員凍傷，防止施工人員在泥濘、冰雪路段摔傷，防止車輛凍壞或側翻。

4. 室內施工時，應採取措施保證施工人員的安全、機房建築的安全、機房內在用設備及新裝設備的安全，應防止施工人員發生觸電、物體打擊、高處墜落等安全事故，防止建築物的樓板、牆體損壞，防止在用設備短路，防止新裝設備受到撞擊或過壓、過流工作。禁止觸碰與工程施工無關的機房設備。

5. 市內施工時，應採取措施保證施工人員及周圍人員的安全、車輛的安全、周圍基礎設施及其他設施的安全，防止發生人員傷亡事故、車輛損壞事故、設施被破壞事故。

6. 郊外施工時，應注意的安全問題有:

(1)公路上施工時，應採取措施保證施工人員、材料及路上過往車輛的安全，防止發生交通事故，防止工程材料被車輛壓壞。

(2) 農田中施工時，應採取措施保證施工人員及農作物的安全，防止施工人員發生墜落及傳染病的危害，防止農作物被損壞。

(3) 鐵路附近施工時，應採取措施保證施工人員的安全、鐵路設施的安全和鐵路的通暢，防止發生鐵路交通事故，防止損壞鐵路路基及信號系統。

不同施工專業應注意的安全問題

1.設備安裝工程施工時，應注意防止施工人員墜落、電源短路、用電設備漏電、電摞線錯接、靜電損壞機盤、激光傷人等安全事故的發生。

2. 鐵塔上作業時，應防止發生人員墜落、鐵塔倒塌、物體墜落等安全事故。

3. 線路工程施工時，應防止發生施工人員觸電、杆上墜落、溺水、人(於)孔內毒氣使人窒息、跌人溝坑、激光傷人等安全事故。

項目準備階段，項目經理部應做好以下工作:

1.按照施工現場的具體特點識別危險源，評價出重大危險源，針對重大危險源制定安全控制措施，保證項目順利進行。

2. 確定安全控制目標，安全控制措施應寫人施工組織設計中。

3. 針對確定的重大危險源，應認真地進行安全技術交底工作。

4. 編制應急預案，並對其內容定期演練。

安全技術交底的依據

施工合同、施工圖設計、工程摸底報告、設計會審紀要、施工規範、施工組織設計，各項技術指標、管理體系要求、作業指導書、辨識的質量控制點和重大危險源、建設單位或監理工程師的其他書面要求等。

安全技術交底的內容

1.工程項目的施工作業特點和危險因素;

2. 針對危險因素制定的具體預防措施;

3. 相應的安全生產操作規程和標準;

4. 在施工生產中應注意的安全事項;

5.發生事故後應採取的應急措施。

安全技術交底工作的要求:

1.安全技術交底工作應由項目技術負責人負責，從安全施工的技術、操作、管理要求等方面向施工、管理人員進行詳細說明和講解。

2. 安全交底工作應符合規定要求

項目技術負責人應當對有關安全施工的技術要求向管理人員、施工人員進行詳細的講解和說明。安全交底的形式應根據工程規模確定，原則上要求逐級進行。安全交底的內容應滿足法律法規和安全生產的相關要求。安全技術交底工作應保存書面的簽字記錄。

通信線路工程的危險源

1.架空線路工程中的危險源:特殊的地形、地質、氣溫，路由附近的高壓電力線、

低壓裸露電力線及變壓器，有缺陷的夾槓、大繩、腳釦、安全帶、座板、緊線設備、梯子、試電筆等工具，有缺陷的鋼絞線、夾板、螺釘、螺母、地錨石、電杆等材料，固定不牢固的滑輪、線擔、夾板等高處重物，碼放過高的材料，車上固定不牢的重物，千斤頂上的光(電)纜盤，未做防護的杆坑、拉線坑，未立起的電杆，繃緊的鋼絞線，鋒利的工具，燃油，伙房的煤氣罐，雷電，有缺陷的標誌，激光，異常的電壓，行駛的車輛，傳染病等。

2. 直埋線路工程中的危險源:特殊的地形、地質、氣溫，地下電力線及其他各種管線，挖開的元警示標誌的光纜溝，車上固定不牢的重物，千斤頂上的光(電)纜盤，鋒利的工具，行駛的車輛，漏電的電動設備，使用不當的噴燈，激光，異常的電壓，雷電，有缺陷的標誌，炸藥，燃油，伙房的煤氣罐，傳染病等。

3. 管道光(電)纜工程中的危險源(長途管道光纜)特殊的地形，人(於)孔內的有毒氣體，落人人(於)孔的重物，車上固定不牢的重物，有缺陷的標誌，斷股的油絲

繩，固定不牢的滑輪，安裝不牢的拉力環，千斤頂上的光(電)纜盤，鋒利的工具，開鑿引上孔濺起的灰渣，使用不當的嗤燈，激光，異常的電壓，行駛的車輛，氣吹機噴出的高壓高溫氣體，打開的沒有圍欄的人(於)孔等。

通信管道工程的危險源

1.長途通信管道工程中的危險源:特殊的地形、地質、氣溫，地下電力線及其他各

種管線，挖開的無警示標誌的管道溝，車上固定不牢的重物，千斤頂上的硅芯管盤，行駛的車輛，鋒利的工具，雷電，有缺陷的標誌，炸藥，燃油，伙房的煤氣罐，傳染病等。

2. 市內通信管道工程中的危險源:特殊的氣溫，地下電力線、煤氣管道及其他管線，挖開的或無警示標誌的管道溝、人(於)孔坑，不牢固的擋土板，落人作業點的重物，車上固定不牢的重物，行駛的車輛，漏電的電動設備，有缺陷的標誌，炸藥，燃油，伙房的煤氣罐，傳染病等。

室內設備安裝工程的危險源

帶釘子或鐵皮的機箱板，有缺陷的電鑽、試電筆、萬用表、高凳、切割機、電焊機等工具，強度不夠的樓板，不合格的防雷系統，高處的重物，靜電，激光，異常的電壓，儲酸室、電油室中能夠產生電火花的裝置，有缺陷的標誌，割接時未作絕緣處理的工具，帶電裸露的電源線或端子，開關在外面的防火門，行駛的車輛，傳染病等。

室外設備安裝工程的危險源

特殊的環境，制動失靈的吊裝設備，有缺陷的安全帶、電鑽、絕緣鞋、電筆、切割機、電焊機等工具，高處的重物，附近的帶電體，斷股的繩索，微波輻射，雷電，不合格的防雷系統等。

各專業工程中常見的不安全行為

違章指揮，野蠻施工，違規操作，長時間作業，睡眠不足，身體不適，未進行安全技術培訓等。

施工單位的安全控制措施：

1. 建立健全安全生產管理制度；

2. 做好安全教育及施工前安全技術交底，使作業人員掌握安全操作規範；

3. 配備必要的防護裝備和救援裝備；

4. 做好應急預案及演練。

安全控制計劃

安全控制要求、安全檢查計劃、安全控制措施和應急預案等

施工現場的安全用電措施

1.施工現場用電應採用三相五線制的供電方式。用電應符合三級配電結構，即由總配電箱經分配電箱到開關箱。每臺用電設備應有各自專用的開關箱，實行"一機一箱"制。

2. 施工現場用電線路應採用絕緣護套導線。

3. 安裝、巡檢、維修、移動或拆除臨時用電設備和線路，應由持有電工證的人員完成，並應有人監護。

4. 檢修各類配電箱、開關箱、電氣設備和電力工具時，應切斷電源，並在總配電箱或者分配電箱一側懸掛"檢修設備，請勿合閘"的警示標牌，必要時設專人看管。

5. 使用照明燈應滿足以下要求:

(1)室外宜採用防水式燈具。在人孔內宜選用電壓 36V 以下(含 36V) 的工作燈照明。在潮溼的溝、坑內應選用電壓為 12V 以下(含 12V) 的工作燈照明。用蓄電池做照

明燈具的電源時，電瓶應放在人孔或溝坑以外。

(2) 在管道溝、坑沿線設置普通照明燈或安全警示燈時，燈具距地面的高度應大

於 2m。

(3) 使用燈泡照明時不得靠近可燃物。當用 150W 以上(含 150W) 的燈泡時，不得使用膠術燈具。

低溫雨期施工的控制措施

低溫雨期施工措施主要涉及室外鐵塔上作業、光(電)纜接續、車輛保養和行車等工作以及防雷、防滑、防潮等。對低溫雨期施工的安全要求如下:

1.低溫季節施工時，施工人員應儘量避免高處作業。必須進行高處作業時，應穿戴防凍、防滑的保溫服裝和鞋帽。在低溫下吊裝機具時，應考慮其安全係數。光纜熔接機和測試儀表工作時應採取保溫措施，以滿足其對溫度要求。車輛在冬季應加裝防凍液，雪天、冰路上行車應裝防滑鏈或使用防滑輪胎，注意防凍、防滑。

2. 雨期施工時，雷雨天氣禁止從事高空作業，空曠環境中施工人員避雨時應注意防雷。施工人員應注意道路狀況，防止滑倒摔傷。雨天及溼度過高的天氣施工時，作業人員在與電力設施接觸前，應檢查其是否受潮漏電。施工現場的儀表及接續機具在不使用時應及時放到專用箱中保管。下雨前，施工現場的材料應及時遮蓋;對於易受潮變質的材料應採取防水、防潮措施單獨存置。雨天行車應減速慢行。暫時不用的電纜應及時縮封端頭，及時充氣。

機具、儀表的保護措施

工程使用的機具、儀表的保管應注意防火、防盜、防潮，應嚴格按照其說明書要求進行保管和維護。在使用時，操作人員應持證上崗。儀表的使用應注意其所用電源的電壓情況，應注意避免電源問題導致儀表損毀。

公路上作業的安全防護措施

1.現場施工人員應嚴格按照批准的施工方案，在規定的區域內進行施工，作業人員應服從交警的管理和指揮，協助搞好交通安全工作，同時還要保護好公路設施。

2. 每個施工地點都應設置安全員，負責按公路管理部門的有關規定擺放安全標誌，觀察過往車輛並監督各項安全措施執行情況，安全標誌尚未全部擺放到位和收工撤離收取安全標誌時應特別注意，發現問題及時處理。在夜間、霧天或其他能見度較差的氣候條件下禁止施工。所有進入施工現場的人員必須穿戴符合規定的安全標誌服，施工車輛應裝設明顯標誌(如紅旗等)。

3. 施工車輛應按規定的線路和地點行駛、停放，嚴禁逆行。

4. 各施工地點的佔用場地應符合高速公路管理部門的規定。

5. 每個施工點在收工時，必須認真清理施工現場，保證路面上清潔。

高處作業時的安全防護措施

1.高處作業人員應當持證上崗。安全員必須嚴格按照安全控制措施和操作規程進行現場監督、檢查。

2. 作業人員應佩戴安全帽、安全帶，穿工作服、工作鞋，並認真檢查各種勞保用具是否安全可靠。高處作業人員情緒不穩定、不能保證精神集中地進行高處作業時不得上崗。高空作業前不準飲酒，前一天不準過量飲酒。

3. 高處作業應劃定安全禁區，設置警示牌。操作人員應統一指揮。需要上下塔時，人與人之間應保持一定距離，行進速度宜慢不宜快。高處作業用的各種工、器具要加保險繩、鉤、袋，防止失手散落傷人。作業過程中禁止無關人員進人安全禁區。嚴禁在杆、塔上拋擲物件。當地氣溫高於人體體溫、遇有 5 級以上(含 5 級)大風以及暴雨、打雷等惡劣天氣或能見度低時嚴禁高處作業。

4. 高處作業須確保踩踏物牢靠。作業人員應身體健康，並做好自我安全防護工作。操作過程中應防止墜落物傷害他人。

地下設施的安全防護措施

開挖土石方前，應及時通知管線產權單位，充分了解施工現場的具體情況，確定保護地下管線及其他設施的方案。開挖城市路面前，應與當地的規劃部門聯繫，必要時應使用儀器探明地下管線的深度和位置，應人工開挖，禁止使用大型機械對暴露的管線應及時採取措施保護。

防毒、防墜落、防原有線纜損壞的措施

1.在開挖光纜溝、管道溝、接頭坑、人(手)孔坑、杆坑及拉線坑時，如果當天不

能回填，應根據現場的實際特點，晚上在溝坑的周圍燃亮紅燈，以防人員跌落。

2. 施工過程中挖出有害物質時，應及時向有關部門報告，必要時啟動應急預案。

3. 在人(手)孔內工作時，井口處應設置警示標誌。施工人員打開人孔後，應先進行有害氣體測試和通風，確認元有害氣體後才可下去作業。在人孔內抽水時，抽水機或發電機的排氣管不得靠近人孔口，應放在人孔的下風方向。

4. 下人孔時必須使用梯子，不得蹬踩光(電)纜託極。在人孔內工作時，如感覺頭暈、呼吸困難，必須離開人孔，採取通風措施。嚴禁在人孔內吸菸。

在用通信設備、網絡安全的防護措施

1.機房內施工電源割接時，應注意所使用工具的絕緣防護;通電前應檢查新裝設備，在確保新設備電源系統元短路、接地、錯接等故障時，確認輸人電壓正常時，方可進行電源割接工作。

2. 在機房內施工時，應採取防塵措施，保持施工現場整潔。

3. 禁止觸動與施工無關的機房設備。需要用到機房原有設備時，應經機房負責人同意，以機房值班人員為主進行工作。

4. 拔插機盤時，應佩戴防靜電手環。

應急預案的編制原則

1.項目經理部應根據識別出的危險源評價潛在事故或緊急情況，識別應急響應需求(包括對應急設備的需求) ，制訂應急預案。

2. 由於經濟方面的原因，項目經理部並不需要對每一項風險都制訂應急預案，只需在風險評價的基礎上，對那些較大風險或可以分類的風險，制定應急方案。如火災應急預案，人員傷亡事故應急預案，電源短路事故應急預案等。

3. 項目施工現場的應急預案應在施工企業專項應急預案的基礎上編制。施工企業應確定應急期間的主要負責人、應急程序以及應急人員職責、權限和義務。項目經理部在此基礎上，根據工程的具體特點和項目經理部的人員構成情況編制施工現場的應急預案。

應急、預案應包括的內容

1.應急期間的負責人，決定採取的應急措施;

2. 起特定作用的人員和專家的職責、權限及義務;

3. 疏散的程序，包括疏散對象，負責人，疏散方法、步驟、路線，到達地點，疏散的組織和管理，工具需求，設備需求;

4. 危險材料的識別和放置以及所要求的應急措施;

5. 與外部應急服務機構的聯繫方式以及與鄰居和公眾的溝通方式、辦法;

6. 至關重要的設備和記錄的識別及保護方法;

7. 應急期間的必要信息及適用情況，如駐點佈置圖，庫房平面圖，危險原材料的資料及處理程序，工作指示和聯絡電話號碼等。

安全事故報告

(1)事故發生後，事故現場有關人員應當立即向本單位負責人報告;單位負責人接到報告後，應當於 1 小時內向事故發生地縣級以上人民政府安全生產監督管理部門和負有安全生產監督管理職責的有關部門報告。

(2) 情況緊急時，事故現場有關人員可以直接向事故發生地縣級以上人民政府安全生產監督管理部門和負有安全生產監督管理職責的有關部門報告。

(3) 事故報告應當及時、準確、完整，任何單位和個人對事故不得遲報、漏報、謊報或者瞞報。

事故調查報告應當包括的內容:

(1)事故發生單位的概況;

(2) 事故發生經過和事故救援情況;

(3) 事故造成的人員傷亡和直接經濟損失;

(4) 事故發生的原因和事故性質;

(5) 事故責任的認定以及對事故責任者的處理建議;

(6) 事故防範和整改措施。
