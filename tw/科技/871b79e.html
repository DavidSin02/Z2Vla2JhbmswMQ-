利用字典的嵌套，完成三級下拉菜單的製作

2020-10-29
| 科技

大家好，今日我們繼續講解VBA數組與字典解決方案，今日的內容是第56講，利用字典的嵌套，完成三級下拉菜單的製作。

在第54講中我講了利用字典的嵌套，完成二級下拉菜單的製作，很多朋友來信反饋說不過癮，可是三級下拉菜單自己卻調試不出來.今日我就給大家講解一下，其實這和二級菜單的製作是類似的，但是需要讀者充分理解字典嵌套的含義，也就是說ITEM值可以是字典，要充分理解這時字典的意義，在大腦中模擬這個過程。

實例，如下數據，我要建立一個自定義的窗體，實現省、市、縣三級菜單的互動相應，

首先我們要建立一個窗體：

下面看我給出的代碼：

Private Sub ComboBox1_Change()

ComboBox2.Clear

'二級下拉框對應的是第一級字典的鍵值為鍵的字典

If ComboBox1.Value <> "" Then ComboBox2.List = mydic(ComboBox1.Value).keys

End Sub

Private Sub ComboBox2_Change()

ComboBox3.Clear

'二級下拉框對應的是第一級字典的鍵值為鍵的字典

If ComboBox2.Value <> "" Then ComboBox3.List = mydic(ComboBox1.Value)(ComboBox2.Value).keys

End Sub

Private Sub UserForm_Activate() '第56講 利用字典的嵌套完成三級級下拉菜單的製作

'將數據裝入數組

myarr = Range("a1").CurrentRegion.Value

Set mydic = CreateObject("Scripting.Dictionary")

For i = 2 To UBound(myarr)

strF = myarr(i, 1)

strS = myarr(i, 2)

strT = myarr(i, 3)

If Not mydic.exists(strF) Then

'建立嵌套字典，第一重字典的鍵對應的鍵值為字典

Set mydicTemp = CreateObject("Scripting.Dictionary")

Set mydic(strF) = mydicTemp

End If

If Not mydic(strF).exists(strS) Then

Set mydicTemp2 = CreateObject("Scripting.Dictionary")

Set mydic(strF)(strS) = mydicTemp2

End If

mydic(strF)(strS)(strT) = ""

Next i

'一級下拉框對應的是第一級字典的鍵

ComboBox1.List = mydic.keys

End Sub

Sub mynzsz_56() '第56講 利用字典的嵌套，完成三級下拉菜單的製作

UserForm2.Show

End Sub

代碼截圖：

代碼解析：1 上述過程實現了三級菜單的響應，在"省"的菜單中實現省一級名稱菜單，在"市"一級菜單中出現對應於"省"的市級菜單，在點擊"縣"時會出現對應於"市"名稱的縣級菜單。

2 '將數據裝入數組

myarr = Range("a1").CurrentRegion.Value

上述代碼將所有備用數據放到數組中

3. For i = 2 To UBound(myarr)

strF = myarr(i, 1)

strS = myarr(i, 2)

strT = myarr(i, 3)

上述代碼先建立了一個FOR NEXT的循環 ，提取每一行的數據

4 If Not mydic.exists(strF) Then

'建立嵌套字典，第一重字典的鍵對應的鍵值為字典

Set mydicTemp = CreateObject("Scripting.Dictionary")

Set mydic(strF) = mydicTemp

End If

上述代碼將第一級數據放到字典中作為鍵，對應的鍵值建立字典對象

5 If Not mydic(strF).exists(strS) Then

Set mydicTemp2 = CreateObject("Scripting.Dictionary")

Set mydic(strF)(strS) = mydicTemp2

End If

上述代碼將第二級數據放到字典中作為鍵，對應的鍵值建立字典對象

6 mydic(strF)(strS)(strT) = ""

Next i

上述代碼完善第二級數據鍵值對應的字典，放入鍵，鍵值為空即可.

7 '一級下拉框對應的是第一級字典的鍵

ComboBox1.List = mydic.keys

上述代碼給一級菜單賦值、

8 Private Sub ComboBox1_Change()

ComboBox2.Clear

'二級下拉框對應的是第一級字典的鍵值為鍵的字典

If ComboBox1.Value <> "" Then ComboBox2.List = mydic(ComboBox1.Value).keys

End Sub

上述代碼給二級菜單賦值

9 Private Sub ComboBox2_Change()

ComboBox3.Clear

'二級下拉框對應的是第一級字典的鍵值為鍵的字典

If ComboBox2.Value <> "" Then ComboBox3.List = mydic(ComboBox1.Value)(ComboBox2.Value).keys

End Sub

上述代碼給三級菜單賦值
