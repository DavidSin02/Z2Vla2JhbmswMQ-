微軟：用戶安裝第三方殺毒軟件後Defender將自動關閉

2020-11-14
| 科技

IT之家 8 月 27 日消息 默認情況下，Windows Defender 安裝在 Windows 10 的所有版本和版本上。其總是在後臺運行，以檢測、隔離和刪除惡意軟件感染、勒索軟件攻擊和其他威脅。

一些用戶認為，當他們打算安裝另一個殺毒軟件應用時，他們需要禁用 Windows Defender。該理論顯然是錯誤的，Windows Defender 不會與另一個殺毒程序發生衝突。

微軟一直允許用戶進入 Windows 註冊表，啟用一個名為 “DisableAntiSpyware”的選項。一旦創建了這個數值為 “1”的條目，微軟就會關閉 Windows Defender。

▲註冊表編輯器

隨著本月對 Windows Defender 的更新，微軟表示，他們已經刪除了那個永久禁用 Defender 的註冊表選項，除非你正在運行 Windows Server 版本。

該公司現在已經更新了文檔，說明如果用戶安裝了殺毒軟件，Windows Defender 將自動關閉。如果你卸載了殺毒軟件，Windows Defender 將再次開啟。

“這一變化不會影響第三方殺毒軟件與 Windows 安全應用的連接。這些仍將按預期工作，”微軟指出。

雖然這個註冊表設置受到 Windows Security 應用的保護，但只有在 Windows 10 家庭版和專業版的新版本中，才有都有 Tamper Protection 篡改保護功能。IT之家瞭解到，此前曾有惡意軟件濫用編輯註冊表，並關閉 Windows 10 的防病毒保護來入侵計算機。

這是停用 DisableAntiSpyware 選項的另一個很好的理由，也是微軟防止惡意軟件對 Windows 的安全性進行不必要的更改的正確舉措。
