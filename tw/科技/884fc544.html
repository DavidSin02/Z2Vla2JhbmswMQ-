高一化學方程式 總結

硫酸根離子的檢驗: BaCl2 + Na2SO4 = BaSO4↓+ 2NaCl

碳酸根離子的檢驗: CaCl2 + Na2CO3 = CaCO3↓ + 2NaCl

碳酸鈉與鹽酸反應: Na2CO3 + 2HCl = 2NaCl + H2O + CO2↑

木炭還原氧化銅: 2CuO + C（高溫） 2Cu + CO2↑

氯化鈣與碳酸鈉溶液反應：CaCl2 + Na2CO3 = CaCO3↓+ 2NaCl

氧化物的反應

氧化鐵與鹽酸反應：Fe2O3 + 6HCl = 2FeCl3 + 3H2O

氧化鈣與水反應：CaO + H2O = Ca(OH)2

氧化鋁與鹽酸反應：Al2O3 + 6HCl = 2AlCl3 + 3H2O

氧化鋁與氫氧化鈉溶液反應：Al2O3 + 2NaOH = 2NaAlO2 + H2O

氯化鐵與氫氧化鈉溶液反應：FeCl3 + 3NaOH = Fe(OH)3↓+ 3NaCl

Na的化學反應方程式

鈉在空氣中燃燒：4Na + O2 = 2Na2O

鈉與氧氣反應： 2Na + O2 △ Na2O2 過氧化鈉

過氧化鈉與水反應：2Na2O2 + 2H2O = 4NaOH + O2↑

過氧化鈉與二氧化碳反應：2Na2O2 + 2CO2 = 2Na2CO3 + O2

鈉與水反應：2Na + 2H2O = 2NaOH + H2↑

Fe及化合物的化學反應方程式

鐵與水蒸氣反應：3Fe + 4H2O(g) = 高溫=F3O4 + 4H2↑

鐵片與硫酸銅溶液反應: Fe + CuSO4 = FeSO4 + Cu

氯化鐵與氫氧化鈉溶液反應：FeCl3 + 3NaOH = Fe(OH)3↓+ 3NaCl

硫酸亞鐵與氫氧化鈉溶液反應：FeSO4 + 2NaOH = Fe(OH)2↓+ Na2SO4

氫氧化亞鐵被氧化成氫氧化鐵：4Fe(OH)2 + 2H2O + O2 = 4Fe(OH)3

氯化亞鐵中通入氯氣：2FeCl2 + Cl2 = 2FeCl3

Al及其化合物的化學反應方程式

鋁與氫氧化鈉溶液反應：2Al + 2NaOH + 2H2O = 2NaAlO2 + 3H2↑

實驗室製取氫氧化鋁：Al2(SO4)3 + 6NH3·H2O = 2Al(OH)3↓ + 3(NH3)2SO4

氫氧化鋁與鹽酸反應：Al(OH)3 + 3HCl = AlCl3 + 3H2O

氫氧化鋁與氫氧化鈉溶液反應：Al(OH)3 + NaOH = NaAlO2 + 2H2O

氫氧化鋁加熱分解：2Al(OH)3 △ Al2O3 + 3H2O

二氧化硅與氫氟酸反應：SiO2 + 4HF = SiF4 + 2H2O

硅單質與氫氟酸反應：Si + 4HF = SiF4 + 2H2↑

二氧化硅與氧化鈣高溫反應：SiO2 + CaO 高溫 CaSiO3

二氧化硅與氫氧化鈉溶液反應：SiO2 + 2NaOH = Na2SiO3 + H2O

往硅酸鈉溶液中通入二氧化碳：Na2SiO3 + CO2 + H2O = Na2CO3 + H2SiO3↓

硅酸鈉與鹽酸反應：Na2SiO3 + 2HCl = 2NaCl + H2SiO3↓

Cl及其化合物的反應方程式

氯氣與金屬鐵反應：2Fe + 3Cl2 點燃 2FeCl3

氯氣與金屬銅反應：Cu + Cl2 點燃 CuCl2

鹽酸與硝酸銀溶液反應：HCl + AgNO3 = AgCl↓ + HNO3

漂白粉長期置露在空氣中：Ca(ClO)2 + H2O + CO2 = CaCO3↓ + 2HClO

N及其化合物的反應方程式

氮氣與氧氣在放電下反應：N2 + O2 放電 2NO

三氧化硫與水反應：SO3 + H2O = H2SO4

氯氣與氫氣反應：Cl2 + H2 點燃 2HCl

硫酸銨與氫氧化鈉反應：（NH4）2SO4 + 2NaOH △ 2NH3↑ + Na2SO4 + 2H2O

下列反應中氯元素只被氧化的是 （ ）

A. 5Cl2 + I2 + 6H2O = 10HCl + 2HIO3

選B, 逐一解釋下

A.氯元素 由0價變為-1價，被還原。

B. 氯元素 由-1價變為0價，被氧化。

C.氯元素由0價變為-1和+1價，既被還原，又被氧化。

D.氯元素 由+1價變為-1價，被還原。

鋁的化學反應方程式

鋁與鹽酸： 2AL+6HCL=2ALCL3+3H2 氣體
