鍍鋅石籠網A路橋鍍鋅石籠網A鍍鋅石籠網實體廠家

安平縣金賓絲網製造有限公司以先進生產技術創造合格格賓網產品，以誠信鑄造格賓網品牌。安平縣金賓絲網製造有限公司供應各種鍍鋅格賓網、包塑格賓網、鋅鋁合金格賓網、高爾凡格賓籠、稀土合金格賓網籠、PVC /PE格賓籠，廠家專業提供格賓籠參數、格賓籠規格、格賓網籠價格、格賓籠設計方案等，並對格賓籠施工做簡單指導。

石籠網（格賓網，綠格網）由高抗腐蝕、高強度、具有延展性的低碳鋼絲或者包覆PVC的以上鋼絲使用機械編織而成，使用該網製作而成的箱型結構就是石籠。根據ASTM和EN標準，所使用的低碳鋼絲直徑根據工程設計要求而不同。一般介於2.0-4.0mm之間，鋼絲的抗拉強度不少於380Mpa，金屬鍍層重量一般高於245g/㎡，石籠網片的邊緣線直徑一般要大於網線直徑。其雙線絞合部分的長度不得小於50mm.以保證絞合部分鋼絲的金屬鍍層和PVC鍍層不受破壞。

產品規格: 2m x 1m x 1m, 3m x 1m x 1m, 4m x 1m x 1m, 2m x 1m x 0.5m, 4m x 1m x 0.5m，還可按客戶要求進行生產。表面保護狀態有熱鍍鋅,鍍鋅鋁合金,塗PVC等。

孔徑：60*80mm、80*100mm、80*120mm、100*120mm、120*150mm、90*110mm其中雙線絞合部分的長度不得小於50mm.以保證絞合部分鋼絲的金屬鍍層和PVC鍍層不受破壞。

網絲的範圍在2mm-4mm

產品用途：可用於邊坡支護、基坑支護、山體巖面掛網噴漿、邊坡植生（綠化）、鐵路高速公路隔離護攔網，它還能製成箱籠、網墊，用於江河、堤壩及海塘的防沖刷保護，水庫、河流截流用網箱。河流最嚴重的災害是水流沖刷河岸使其破壞，引發的洪水氾濫，導致生命財產遭受大量損失和大量水土流失。因此在處理上述問題時侯，格該生態格網結構的應用成為最佳的解決方案之一，格賓網它能使河床河岸得到永久性保護。
