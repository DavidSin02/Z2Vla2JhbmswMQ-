mysql 創建表，查看錶，修改表名字，指定引擎和字符集

2020-11-14
| 科技

語法: create table 表名( 字段名1 類型[(寬度) 約束條件], 字段名2 類型[(寬度) 約束條件], )[存儲引擎 字符集];

==在同一張表中，字段名是不能相同

==寬度和約束條件可選

==字段名和類型是必須的

mysql> show tables;查看有哪些表

mysql> desc t1.查看錶結構：

select * from t1; select 字段，字段 from 表名查看錶裡面的記錄

show table status like '表名'\G 查看錶的狀態

rename table 舊錶名 to 新表名;修改表名稱

mysql> drop table 表名.刪除表 drop database 庫名.刪除庫

mysql> create table t1(id int) engine =innodb; 創建表時指定引擎

修改字符集 ：在創建表的最後面指定一下： default charset=utf8 #可以指定中文
