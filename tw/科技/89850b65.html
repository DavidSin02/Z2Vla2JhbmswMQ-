2020年一級造價工程師考試《水利計量》真題及答案解析

　　一、單選題

　　1、自穩時間很短，規模較大的各種變形和破壞都可能發生的水工隧洞圍巖類型是( ) 。

　　A. II

　　B. II

　　C. IV

　　D. V

　　答案: C

　　解析: P15

　　2、土的物理力學性質指標中，通過試驗測定指標換算獲得的是()。

　　A.土的天然密度

　　B.含水率

　　C.土的容重

　　D.幹密度

　　答案: D

　　解析: P4-5。試驗測定指標:土的天然密度(重度)、土的含水率、土粒比重:換算指標:土孔隙比、飽和度、飽和重度、乾重度、幹密度。

　　3、根據含水層的空隙性質，可將地下水分為孔隙水、裂隙水及()。

　　A.包氣帶水

　　B.潛水

　　C.岩溶水

　　D.承壓水

　　答案: C

　　解析: P23。 根據含水層的空隙性質，地下水又分為孔隙水、裂隙水和岩溶水三個亞類。

　　4、下列不屬於岩漿岩的( ) 。

　　A.花崗岩

　　B.玄武岩

　　C.安山岩

　　D.片麻岩

　　答案:D

　　解析: P10。 岩漿岩分侵入岩和噴出巖。侵入岩分深成岩(如花崗岩、正長巖、閃長巖、輝長岩等) :淺成巖(如花崗斑岩、閃長玢岩、輝綠岩、脈岩等) :噴出巖(流紋岩、粗面岩、安山岩、玄武岩、火山碎屑岩等)。

　　5、碳素鋼牌號Q235, Q代表( ) 。

　　A.抗拉強度

　　B.屈服強度

　　C.質量等級

　　D.含碳量

　　答案: B

　　解析: P70。Q235-B.F，表示屈服強度為235MPa,質量等級為B級的沸騰鋼。

　　6、一黏性土樣的液性指數為0.55，土樣狀態為( )。

　　A.堅硬

　　B.硬塑

　　C.可塑

　　D.軟塑

　　答案: C

　　解析: P7。

　　7、在堆石料原巖的質量技術指標中，一般要求軟化係數大於( )。

　　A 0.55

　　B.0.65

　　C 0.75

　　D 0.85

　　答案: C

　　解析: P111

　　8、工程中目前應用最廣泛的水泥類型()。

　　A.礦渣水泥

　　B.硅酸鹽水泥

　　C.鋁酸鹽水泥

　　D.鐵鋁酸鹽水泥

　　答案: B

　　解析: P62。水泥品種繁多，目前常用的是硅酸鹽水泥。

　　9、工程現場用標準貫入試驗測砂土密實度，標準錘擊數為13，砂土的密實度為( ) 。

　　A.鬆散

　　B.稍密

　　C.中密

　　D.密實

　　答案: B

　　解析: P6。

　　10、水利工程建築材料中絕熱材料是指常溫時導熱係數不大於( ) w/ (m*k)。

　　A.0.350

　　B. 0. 175

　　C.0.05

　　D. 0.075

　　答案: B

　　11.某水庫工程總庫容為1億m3,年引水量為1億m3,供水對象的重要性為比較重要，該水庫的工程等別為( ) 等。

　　A. I

　　B. II

　　C. III

　　D. IV

　　答案: B

　　解析:P125。根據下表的指標分屬機構不同等別時，整個樞紐的等級應以其中的最高等別為準。

　　12、混凝土中加入減水劑的作用是( ) 。

　　A.調節混凝土拌合的流變性

　　B.調節混凝土強度

　　C.調節混凝士凝結時間

　　D.調節混凝土水化熱

　　答案: A

　　解析: P92。 改變混凝土拌合物流變性能的外加劑，包括各種減水劑、引氣劑及泵送劑。

　　13、一般水工混凝土抗凍等級分為( ) 個等級。

　　A.4

　　B.5

　　C.6

　　D.7

　　答案: C

　　解析: P132.根據混凝土試件在28d齡期所能承受的最大凍融循環次數分為F50、F100、 F150、F200、F250 及F300六種等級。

　　14、硅酸鹽水泥的初凝時間不得早於( ) 。

　　A.45min

　　B.90min

　　C.180min

　　D.390min

　　答案: A

　　解析: P63。 硅酸鹽水泥的初凝時間不得早於45min,終凝時間不得遲於6.5h;普通硅酸鹽水泥初凝時間不得早於45min,終凝時間不得遲於10h。

　　15、載重汽車按噸位分，總重量為( ) t 屬於中性卡車。

　　A. <1.8t

　　B.1.8~6t

　　C. 6~14t

　　D.14~100t

　　答案: C

　　解析: P267。 按承載噸位分為微型卡車、輕型卡車、中型卡車、重型卡車、超重型卡車。微型卡車總質量小於1.8t，輕型卡車總總量1.8~6t,中型卡車總質量為6~14t，重型卡車總質量為14~ 100t，超重型卡車總質量為100t以上。

　　16、 水工建築物按使用期限分為( )

　　A.永久性建築物和臨時性建築物

　　B.永久建築物和次要建築物

　　C.主要建築物和次要建築物

　　D.主要建築物和臨時性建築物

　　答案: A

　　解析: P123。 水工建築物按使用期限分為永久性建築物和臨時性建築物。

　　17、蓄滯洪區的堤防級別宜為( )級。

　　A.1

　　B. 2

　　C.3

　　D.4

　　答案: B

　　解析: P127。 蓄滯洪區安全區的堤防永久性水工建築物級別宜為2級，對於安置人口大於10萬人的安置區，經論證後堤防永久性水工建築物級別可提高一級。

　　18、修建丁壩、順壩等，將水流挑離提岸，以防止水流、波浪或潮汐對堤岸的沖刷的是( )。

　　A.壩式護岸

　　B.牆式護岸

　　C.坡式護岸

　　D.平順護岸

　　答案: A

　　解析: P191。 壩式護岸是指修建丁壩、順壩等，將水流挑離提岸，以防止水流、波浪或潮汐對堤岸的沖刷，這種形式多用於遊蕩性河流的護岸。

　　19、壩體混凝土抗壓強度設計齡期- -般採用( ) 。

　　A.28d

　　B. 90d

　　C.180d

　　D.360d

　　答案: B

　　解析: P131。 壩體混凝土抗壓設計齡期- -般採用90d，最多不宜超過180d。

　　20、土質斜牆土石壩，斜牆上下游面應分別設置( )。

　　A.保護層、過濾層

　　B.墊層過、濾層

　　C.保護層、反濾層

　　D.墊層、反濾層

　　答案: C

　　解析: P142。

　　21、瀝青混凝土面板、砂礫土壩為調節壩體變形，面板下面應設置( ) 。

　　A.墊層和過渡層

　　B.墊層和反濾層

　　C.過濾層和反濾層

　　D.整平層和墊層

　　答案: A

　　解析: P144~ 145.瀝青混凝土斜牆鋪設在墊層上，墊層的作用是調節壩體的變形。過渡層主要對其兩側土料的變形起協調作用。

　　22、拱壩壩高大於( ) 米為高壩。

　　A. 30

　　B.50

　　C.70

　　D.90

　　答案: C

　　解析: PI51。 按壩高分類，大於70m為高壩，30 70m為中壩，小於30m為低壩。

　　23、.... 正槽溢洪道控制堰體軸線與( ) ( 題目不全)

　　A.壩軸線正交或近於正交

　　B.壩軸線平行或近於平行

　　C.洩槽軸線正交或近於正交

　　D.洩槽軸線平行或近於平行

　　答案: C

　　解析: P154。

　　24、習慣將用於軟土地層的隧道掘進機稱為( )。

　　A.敞開式TBM

　　B.雙護盾TBM

　　C.單護盾TBM

　　D.盾構機

　　答案: D

　　解析:習慣將用於軟土地層的隧道掘進機稱為盾構機，將用於岩石地層的稱為TBM。

　　25、根據工程習慣主廠房以發電層樓板面為界，分為上部結構和下部結構。下部結構除發電機層樓板外，均為大體積混凝土，包括( ) 風罩、蝸殼外圍混凝土、尾水管外圍混凝土、防水牆等。

　　A.引水鋼管鎮墩

　　B.廠房基礎

　　C.尾水邊牆

　　D.機墩

　　答案: D

　　解析: P169。下部結構除發電機層樓板外，均為大體積混凝土結構，包括機墩、風罩、蝸殼外混凝土、尾水管外圍混凝土、防水牆等。

　　26、將上下游水位分為若干級，形成梯形水位跌落的魚道形式分為( )。

　　A.斜槽式

　　B.水池式

　　C.隔板式

　　D.仿生態式

　　答案: C

　　解析:隔板式魚道是在水池式魚道的基礎上發展起來的，利用橫隔板將魚道上下游總水位差分為若干級，形成梯級水面跌落，故又稱梯級魚道。

　　27.平洞石方開挖是指水平夾角小於或等於( )的石方洞挖工程。

　　A. 5°

　　B. 6°

　　C. 7°

　　D. 8°

　　答案: B

　　解析: P386。 平洞石方開挖是指水平夾角小於或等於6°的石方洞挖工程。

　　28、將截流材料從龍口的一嘴向 另一端或從兩端向中間拋投進佔，逐漸束窄龍口，直到全部截斷的方法是( )

　　A.平堵法

　　B.下閘截流

　　C.立堵法

　　D.定向爆破截流

　　答案: C

　　解析: P290。立堵法是將截流材料從龍口的一端向另一端或從兩端向中間拋投進佔，逐漸東窄龍口，直到全部截斷。

　　29、正鏟挖掘機的剷土動作形式特點是( ) 。

　　A.向前向上，強制切土

　　B.向後向下，強制切土

　　C.向後向下，自重切土

　　D.直上直下，自重切土

　　答案: A

　　解析: P255. 正鏟挖掘機剷土動作形式特點是:前進向上， 強制切土。

　　30、格賓石籠、格賓墊是近些年發展起來的生態格網結構，其格網材料一般採用( ) 。

　　A.熱鍍鋅鋼筋

　　B.熱鍍鋅鉛絲

　　C.熱鍍鋅低碳鋼絲

　　D.熱鍍鋅高碳鋼絲

　　答案: C

　　解析: P194。 格賓石籠、格賓墊的網材稱為格網，一般採用PVC覆層或無PVC覆層的經熱鍍工藝進行抗腐處理的低碳鋼絲等編制。鋼絲宜為熱鍍鋅低碳鋼絲、鋁鋅混合稀土合金鍍層低碳鋼絲，以及經高抗腐處理的以上同質鋼絲等。

　　31、銅芯交聯聚乙烯絕緣聚氯乙烯護套，鋼帶鎧裝電力電纜表示為( )。

　　A. VV22

　　B. ZLL32

　　C. YJU22

　　D. VIV42

　　答案: C

　　解析: P233.交聯聚乙烯絕緣電力電纜: YJV 表示銅芯交聯聚乙烯絕緣聚氯乙烯護套電力電纜:YJILV22表示鋁芯交聯聚乙烯絕緣聚氣乙烯護套，鋼帶鎧裝電力電纜。

　　32、用漿液填充混凝土與圍巖或混凝士與鋼板之間的空隙和孔洞，以增強圍巖或結構的密實性。

　　A.固結灌漿

　　B.接觸濯漿

　　C.回填灌漿

　　D.接縫灌漿

　　答案: C

　　解析: P316。回填灌漿是用漿液填充混凝土與圍巖或混凝土與鋼板之間的空際和孔洞，以增強圍巖或結構的密實性。

　　33、以下( ) 不屬於水利水電工地的“綜合加工廠”。

　　A.混凝土預製件廠

　　B.鋼筋加工廠

　　C.木材加工廠

　　D.汽車修配廠

　　答案: D

　　解析: P362。水利水電工地的混凝土預製件廠、鋼筋加工廠和木材加工廠等,可視原材料供應、交通運輸，場地面積即施工管理等條件分開設置或聯合設置。在場地條件允許的情況下，宜聯合設置，簡稱為“綜合加工廠”。

　　34、下列不屬於二次設備安裝內容的是( ) 。

　　A.調試

　　B.防火封堵

　　C.調線

　　D.元器件整定

　　答案: B

　　解析: P239.二次設備安裝歸納起來有以下內容:立盤、配線、繼電器和元器件的整定、調試。

　　35、碾壓混凝土壩澆築方式( ) 。

　　A.平倉澆築

　　B.通倉厚層澆築

　　C.通倉薄層澆築

　　D.分塊澆築

　　答案: C

　　解析: P346。 碾壓混凝土採用通倉薄層澆築。

　　36、某水電站規劃裝機容量為1200W，,運行水頭範圍為16米，電站擬選用4臺相同容量的水輪發電機組，最合適的水輪機型式是( )。

　　A.混流式

　　B.軸流轉漿式

　　C.燈泡貫流式

　　D.軸伸貫流式

　　答案: C

　　解析: P200。 燈泡貫流水輪機適用水頭範圍為3-30m，單機容量最大為100MW。適用於低水頭大流量的水力發電工程。

　　37、水利工程中常用施工機械膠帶輸送機屬於( ) 。

　　A.土石方機械

　　B.混凝土機械

　　C.起重機械

　　D.運輸機械

　　答案: D

　　解析: P268。膠帶輸送機是以膠帶作為牽引機構和承載機構的連續型動作設備，又稱皮帶機或皮帶。屬於運輸機械。

　　38、為搬運和通風的需要，施工倉庫棚頂高度不宜低於()m。

　　A. 5

　　B.4.5

　　C.3

　　D.3.5

　　答案: B

　　解析: P366。 棚頂高度應不低於4.5m，以利於搬運和通風。

　　39、灌漿壓力在( ) MPa以上的灌漿為高壓灌漿。

　　A.5

　　B.3

　　C.4

　　D.1

　　答案: B

　　40、水輪發電機型號YFG423/2×10-40型式中，發電機的額定轉速為( ) 。

　　A.100

　　B. 200

　　C.150

　　D.300

　　解析: P206。 轉速= (我國電網頻率50HzX60) / (磁極個數40/2)=150(r/min)。

　　41.在地質條件較好的地下廠房中，項拱開挖採用中導洞先行掘進，兩側擴挖跟進的方法，一般中導洞超前( ) 。

　　A.10~15m

　　B.15~20m

　　C.20~30m

　　D.30~45m

　　42.為了防止漂浮物進入管內，進口段設有攔汙柵，同時應設( ) 以便可能對管道進行檢修。

　　43、開挖坡高小於5m的堅硬黏性土土質邊坡時可選容許坡度為( ) 。

　　A.1: 0.35~1; 0. 50

　　B. 1: 0.50~1: 0.75

　　C. 1: 0.75~1: 1.0

　　D.1: 10~1~1.25

　　45、根據水工混凝土施工規範，-般混凝土養護時間不容小於( ) 天。

　　A.7

　　B.14

　　C.28

　　D.90
