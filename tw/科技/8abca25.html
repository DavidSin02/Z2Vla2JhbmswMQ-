SQL語言例子彙總

CREATE DATABASE 學生檔案

創建一個名為“學生檔案”的數據庫

CREATE TABLE 學生成績（姓名 VARCHAR,語文 REAL，數學 REAL,英語REAL）

創建一張名為“學生成績”的數據表，VARCHAR,REAL為字段類型

SELECT 姓名，語文 FROM 學生成績

顯示“學生成績”數據表中所有記錄的姓名和語文信息

SELECT 姓名，語文，數學，英語，語文+數學+英語 AS 總分 FROM 學生成績

顯示“學生成績”數據表中所有記錄的語文，數學，英語和總分信息

SELECT * FROM 學生成績

顯示“學生成績”數據表的全部信息，這裡可以用“*”代表所有字段

www.zjdkdz.com浙江單考單招

SELECT * FROM 學生成績 WHERE 語文>=85

顯示“學生成績”數據表中語文成績大於等於85的記錄信息

SELECT * FROM 學生成績 WHERE 姓名 LIKE “李*”

顯示“學生成績”數據表中，姓名第一個字為李的記錄

SELECT * FROM 學生成績 WHERE 姓名 LIKE “李？”

顯示“學生成績”數據表中，姓名為兩個字，第一個字為李，第二個字為任意的記錄

SELECT * FROM 學生成績 WHERE 姓名 LIKE “*強”

顯示“學生成績”數據表中，姓名最後一個字為“強”的記錄

SELECT * FROM 學生成績 WHERE 姓名 LIKE “*強*”

顯示“學生成績”數據表中，姓名中含有“強”的記錄

SELECT * FROM 學生成績 WHERE 是否團員

顯示“學生成績”數據表中是否團員的記錄

SELECT * FROM 學生成績 WHERE NOT 是否團員

顯示“學生成績”數據表中所有不是團員的記錄

SELECT * FROM 學生成績 WHERE 出生日期=#8/20/1987#

顯示“學生成績”數據表中出生日期為1989年8月20日的記錄

SELECT * FROM 學生成績 WHERE 出生日期>#8/20/1987#

顯示“學生成績”數據表中出生日期為1989年8月20日以後的記錄

SELECT * FROM 學生成績 WHERE MONTH（出生日期）=8

顯示“學生成績”數據表中出生月份為8月的記錄

YEAR（）取年 MONTH（）取月 DAY（）取日

DATA（）取計算機系統即時日期 NOW（）取計算機系統即時日期和時間，精確到秒

SELECT 姓名，YEAR(NOW())-YEAR(出生日期) AS 年齡 FROM 學生檔案

顯示“學生檔案”數據表中的學生姓名和年齡信息

SELECT * FROM 學生成績 WHERE語文>=60 AND 語文<90

顯示“學生成績”數據表中語文成績大於等於60並且小於90的記錄

SELECT * FROM 學生成績 WHERE語文>=60 OR 數學>=60

顯示“學生成績”數據表中語文成績大於等於60或數學大於等於60的記錄

聚集函數

COUNT（）統計所選數據的行數 SUM（）計算數據列的 AVG（）計算數據列的平均值

MIN（）計算數據列的最小值 MAX（）計算數據列的最大值

SELECT COUNT(*) AS 總人數 FROM 學生成績

顯示“學生成績”數據表中總的記錄條數

SELECT AVG(語文),MAX（數學） FROM 學生成績

顯示“學生成績”數據表中語文的平均分和數學的最高分

SELECT AVG(語文),MAX（數學） FROM 學生成績 WHERE 英語>80

顯示“學生成績”數據表中英語成績大於80分同學的語文的平均分和數學的最高分

SELECT 班級，AVG(語文) FROM 年級成績 GROUP BY 班級

根據班級，分別統計各班的語文平均成績

SELECT 班級，AVG(語文) AS 語文平均分 FROM 年級成績 WHERE 語文>85 GROUP BY 班級

把“年級成績”數據表中語文大於85分的記錄，根據班級分別統計各班的語文平均成績

SELECT 班級，AVG(語文) AS 語文平均分 FROM 年級成績GROUP BY 班級 HAVING AVG（語文）>85

先根據班級分別統計各班的語文平均分，再把語文平均分大於85分的記錄顯示出來

SELECT * FROM 學生成績 ORDER BY 語文

按語文成績的升序顯示“年級成績”數據表的所有記錄

SELECT * FROM 學生成績 ORDER BY 班級，語文 DESC

對“年級成績”數據表按班級遞增，語文成績遞減排序

在VB中，隨機數表達式為 int(rnd*(上限-下限+1)+下限) int(rnd*(500-100+1)+100)
