《營造法式》與晚唐官式栱長制度比較

摘要

栱長是斗栱演進的過程中一個較為活躍的因素，對斗栱的整體造型有很大的影響。本文在前輩學者的基礎上深入解讀了《營造法式》栱長構成的特點，嘗試揭示出其精細分°值背後的設計智慧，並與《法式》成書之前影響甚廣的晚唐官式栱長制度作了對比。

斗栱是中國古代木構建築最具特色的標誌，其比例權衡和細部特點因不同時代、地域和匠作流派而有所不同，反映出審美價值取向的差異及營造技術的變遷。至《營造法式》成書之際（1100年），斗栱的分°值模數制已經極為成熟，其斗栱制度被其後歷代官式建築奉為經典。

從栱長的明承宋制、清承明制來看，可謂一脈相承。張十慶先生曾對《法式》栱長構成及其意義進行過解析①，探討了栱長設計的創意及其歷史演變規律。本文將在此基礎上，對《法式》的栱長制度作進一步的探討。

一 《法式》斗栱的比例模數

《營造法式》對鋪作中不同栱和斗的比例關係均有明確規定，但《法式》所載鋪作以重栱計心造為代表，制度的描述均側重於此。至於單栱造和偷心造只在註文中略加提及而易被忽視。加之孤立地考察栱長無法揭示其構成規律，須綜合考察鬥與栱的細部特徵及其組合關係。

1. 栱的分°值模數

關於鋪作中栱的比例權衡，據《法式》卷四可得以下簡表（表1）：

表1 《法式》造栱之制

其栱長分為三種：重栱造泥道栱和瓜子栱長62分°，花栱、令栱與單栱造泥道栱長72分°，慢栱長92分°。其中各栱的2分°尾數，是《法式》典型栱長的一大特點，頗令人費解。張十慶先生認為：“從實際操作角度而言，瓜子栱長62分°，華栱長72分°，慢栱長92分°，其中的2分°，並無實際意義，完全可以省去而不影響斗栱的組合，工匠在施工為圖省事，也常常略去這2分°零頭，以求整數。”如果其中的零頭並無實際意義，古人怎能畫蛇添足？那些為圖省事的工匠也並非北宋的梓人。如此精微的比例權衡顯然隱藏著創立者的某種巧思。

斗的種類基本分為大斗和小鬥兩種，大斗的尺寸約為小斗的兩倍。

根據《法式》卷四的造鬥之制，花栱跳頭既可施交互鬥也可施散鬥，故花栱長與栱頭小斗的種類密切相關。《法式》規定兩卷頭長72分°的前提是鋪作為計心造且出跳不減分°（花栱長 = 6+30+30+6= 72分°）。若鋪作偷心，其上小鬥改用散鬥，則花栱長 = 5+30+30+5 = 70分°。至於上昂造鋪作，其花栱長更是靈活多樣。因此，72分°只是典型的花栱長度，其2分°的尾數也有其存在的條件。

二 《法式》斗栱的模數構成法則

雖然《營造法式》的斗栱造型設計精美細膩，但應是在一個 “原始設計模型”的基礎上加工、修飾而來。若將斗栱的所有藝術加工和比例模數中的尾數都暫時略去，即可抽象出《法式》斗栱設計的“原始模型”，更便於揭示其初始的設計規律。此“原始模型”的斗栱種類和比例模數均大大簡化（圖1）：

圖1 《營造法式》斗栱原始模型圖

（作者自繪）

（1） 大斗：方30分°，高20分°；小鬥：方15分°，高20分°

（2）重栱下層栱長60分°，重栱上層栱長90分°；單栱長70分°

圖中可見無論重栱造還是單栱造的“原始模型”的構成均受到模數網格的嚴格約束（水平方向每格15分°，豎直方向15分°與6分°間隔），體現出簡潔、樸素的“以章栔為祖”的設計思想。

進一步考察《法式》各栱長度的相關性：

（1）花栱長 = 令栱長 = 單栱造泥道栱長 = 72分°

（2）瓜子栱長 = 重栱造泥道栱長 = 62分°

（3）慢栱長 = 瓜子栱長 + 兩材 = 92分°

值得注意的是，雖然瓜子栱與重栱造泥道栱長均為62分°，其栱頭卷殺並不相同，顯然與其下斗的長度有關。再看栱長與鬥長的關係：

（1）櫨鬥長 = 32分°

（2） 重栱造泥道栱長 = 櫨鬥長 + 兩材 = 62分°

可見重栱造泥道栱的2分°尾數可能源於櫨斗的2分°尾數，進而導致了慢栱的2分°尾數。需要追問的是：櫨斗的2分°尾數因何而來？栱長的尾數與櫨斗的尾數是否必然相關？

《法式》各種小斗的長廣均在15分°左右，高均為10分°。在斗栱發展的初期，小斗的規格簡單劃一尚未分化（如莫宗江先生1937年所測的北宋初山西榆次雨花宮的斗栱就無交互鬥、齊心鬥和散斗的分別）或雖有分化尚不夠細膩（如佛光寺東大殿的齊心鬥與散鬥尺寸相同②）。而《法式》小鬥微妙的2分°級差，顯然是經過北宋匠師們精心推敲的，既有構造的需求，又有美學上的考慮。如齊心鬥（正面16分°）居中，散鬥（正面14分°）在兩側，通過2分°的長度差體現了微妙的主次之別。又如交互鬥施於花栱出跳之上，受力較大且一般十字開口，故其正面尺寸放大至18分°。三種小鬥中，以齊心斗的模數（方16分°，高10分°）最具典型性和代表性，是所有斗的基準鬥。問題是齊心鬥之方為何不取標準一材高15分°呢？推測應與小斗的構造安全有關。三種小斗的側面寬度是統一的，均為16分°。說明小鬥側面開10分°的卯口，其所剩兩耳的厚度至少須各為3分°，若有隔口包耳，則能保證其厚度為1.5分°（1.5= 3×0.5）。而櫨斗的模數當是在齊心斗的基礎上倍增而來的，即32 = 16×2，20 = 10×2。至於角櫨鬥略大（方36分°）是基於構造考慮而作的微調。但栱長普遍的2分°尾數又如何解釋？櫨斗方32分°，泥道栱亦可以取60分°而不必是62分°。其中究竟隱藏著古人怎樣的匠心？

三 栱長尾數的玄機

筆者在參考《法式》卷三十大木作圖樣“栱鬥等卷殺第一”的表達方法，重新繪製五鋪作斗栱的過程中，發現前輩學者對“花頭子”細部的誤讀可能是問題的突破口。花頭子即外跳花栱的出頭，用來承託下昂，其設計的初衷是為了將昂上坐鬥歸平。《法式》卷四：“花頭子自鬥口外長九分°，將昂勢盡處勻分，刻作兩卷瓣，每瓣長四分°。”初看此文，似前後存在矛盾：既然花頭外長9分°，為何勻分的兩瓣又每瓣長4分°呢？劉敦楨先生認為原文有錯漏，將“每瓣長四分°”校訂為“每瓣長四分°半”③ 。樑思成先生亦認同此說，註釋本“大木作制度圖樣四”中的花頭子即是按此繪製的④。

其實原文不誤，花頭子自鬥口出，先平出1分°作為過渡，再捲起兩瓣，每瓣各4分°。這一細節恰體現出北宋建築極其微妙的藝術處理：不同造型輪廓的構件相交，其間至少要留出1分°的平段（圖2）。同樣的細節處理還出現在耍頭的設計上，也是從鬥口先平出1分°，再做斜抹。

圖2 《營造法式》斗栱細部權衡圖釋1

（作者自繪）

由此進一步考察各栱頭的藝術處理，遵循的都是同樣的設計法則：泥道栱（重栱造）自櫨鬥口出，先平出1分°，再做卷殺；泥道慢栱自散鬥口出，亦先平出1分°，再做卷殺——這樣做還能保證內側的折點不入鬥口，否則影響栱的受力（圖3）。可見栱長與栱頭分瓣卷殺設計也是同時進行的，二者相互關聯。進而又可以體會《法式》栱頭的卷殺為何會因栱而異，不做簡單的統一⑤。因此，無論是泥道栱（重栱造）62分°還是慢栱92分°的2分°零頭，都是經過宋代匠師深思熟慮的。由於重栱造的瓜子栱在立面上與泥道栱相對應，故也取62分°。

圖3 《營造法式》斗栱細部權衡圖釋2

（作者自繪）

至於令栱長72分°，當與鋪作的單栱造有關。與重栱造相比，單栱造是更為早期和原始的做法，即重栱造是在單栱造的基礎上發展演變而來的，至《法式》成書之際已成為官式建築斗栱的典型做法。《法式》卷四：“四曰令栱，或謂之‘單栱’。”可見令栱的本質是單栱。因此，分析令栱的栱長構成，應從單栱造入手。《法式》卷十七：“如單栱造者，不用慢栱，其瓜子栱並改作令栱。”即單栱造鋪作橫栱的長度只有一種。又《法式》卷四：“二曰泥道栱，其長六十二分°（若鬥口跳及鋪作全用單栱造者，只用令栱）。”單栱造改為泥道令栱，其長72分°，正與第一跳花栱長72分°相同。從構造上看，泥道令栱與第一跳花栱垂直正交於櫨鬥口內，第二鋪形成正十字栱，二者在視覺上需達到均衡，泥道栱長取與花栱相等——這一法則正是早期斗栱設計的基本規律。因此，令栱的尾數是由典型的花栱長72分°帶來的。

四 晚唐官式栱長制度

張十慶先生認為，《營造法式》成書以前的栱長制度並無一定之規，呈現出隨宜和多樣化的特點。但從現存唐遼宋建築實例看，仍有一定的規律可循，最遲至晚唐已形成了典型的官式做法，且對北方地區的斗栱設計有著深遠的影響。

我國現存最早的木構建築五臺山南禪寺大殿（782年）已有清晰的栱長制度（據祁英濤先生20世紀70年代實測）：

（1） 泥道栱長 = 2倍第一跳花栱頭長 （略小於令栱長）

（2） 慢栱長 = 2倍第二跳花栱頭長

由於此殿僅為五鋪作且第一跳偷心，故無瓜子栱和真正的慢栱。

從五臺山佛光寺東大殿（857年）的栱長⑥構成看，在南禪寺栱長制度的基礎上又有所發展變化，已形成了極為精嚴的法度，代表了京師地區的官式做法。鋪作的正樣和側樣之關係密切，除泥道栱外的各栱心長均受到490毫米（即一大跳心長的1/2或一跳的均值，合16.5寸）模數線的制約（圖4）。其栱長的構成規律特徵鮮明：

（1）泥道栱長 = 第一跳花栱長 = 令栱長 + 散鬥上下寬之差⑦

（2）慢栱長 = 2倍第二跳花栱頭長

（3）瓜子栱長 = 令栱長

（4） 慢栱心長 = 2倍瓜子栱心長 = 2倍令栱心長 = 2倍第一大跳心長

（作者自攝）

圖4-2 佛光寺東大殿柱頭鋪作理想模型

（作者自繪）

以上基本法則經過唐代匠師深思熟慮，自有其獨特的設計邏輯。其斗栱為晚唐典型的雙杪兩下昂七鋪作：外轉第一、三跳偷心，第二、四跳計心，形成兩個大跳。唐人將泥道栱設為與第一跳花栱等長，進而將泥道栱上的隱刻的慢栱取第二跳花栱外跳長的2倍，明顯是試圖使斗栱在造型上達到視覺的均衡（圖5）。泥道栱則因櫨鬥尺度較大，在瓜子栱的基礎上適當加長而成。瓜子栱和令栱的心長取慢栱的1/2，是為了使其上的散鬥分佈均勻。一大跳心長是東大殿鋪作設計的根本出發點，也是各栱長度設計的模數基準。可以肯定，早在《法式》成書以前，晚唐官式建築的栱長制度已經非常成熟且自成體系。

圖5 佛光寺東大殿斗栱模數構成

（作者自繪）

佛光寺東大殿的栱長制度於敦煌莫高窟第431窟的北宋窟簷（980年）和薊縣獨樂寺觀音閣和山門（984年）以及山西應縣木塔（1056年）中得到再現⑧（圖6）。至於平遙鎮國寺萬佛殿（963年）、太原晉祠聖母殿（984年）、義縣奉國寺大殿（1020年）等遼宋建築的栱長比例關係與東大殿相仿而又存在差異，可視為其制度的不同程度的變體。

圖6 獨樂寺觀音閣及山門栱長模數構成

（作者自繪）

《營造法式》栱長構成的原則與之有很大不同：典型的鋪作為重栱計心造，其出一跳為30分°（兩跳60分°）顯然較唐遼為大，故花栱變長，兩卷頭長（72分°）超過重栱造泥道栱（62分°）。但單栱造泥道栱（泥道令栱）仍按古制取與之等長，這直接導致了令栱的加長。因此，在栱長演變過程中，令栱的加長似未必如張十慶先生所言“主要出於視覺效果的需要”，而是與計心造的發展有密切的關係。至於慢栱比例的縮短，應與唐宋之際當心間尺度一般在2丈之內，而補間鋪作由唐遼的一朵變為北宋《法式》規定的兩朵的變化直接相關。

需要進一步討論的是，《營造法式》所載的材分°制始創於北宋抑或更早？佛光寺東大殿鋪作設計的背後，是否也隱藏著一套精密的材分°制？

根據前輩學者的實測統計，佛光寺東大殿花栱厚均值約210 毫米，假設晚唐鋪作設計已存在成熟的分°制（1分°= 1/10材厚）且材厚等於栱厚⑨，則1分°= 21 毫米，第一跳心長 = 26分°，第二跳心長 = 21分°⑩。由此假說可進一步得出表3：

五 結論

栱長制度至遲到晚唐已形成了成熟嚴謹的官式做法。斗栱立面的豎向模數控制線為一大跳心長的1/2，橫向聯繫栱的栱長由縱向出跳栱決定，慢栱比例較長（長同第二跳花栱），瓜子栱與令栱等長，其心長取慢栱心長之半。現存實例以佛光寺東大殿最為典型，斗栱比例舒朗，造型雄渾大氣，其制度堪稱“北派”譜系建築的經典。雖然其法度精嚴，但當時的鋪作設計尚未形成分°制。

北宋以來，藝術風格上“南秀”逐漸取代“北雄”，反映出統治階級審美價值取向的鉅變。至北宋晚期，源於江南營造技藝的《營造法式》一躍成為京師地區官式建築的主流派系，其斗栱制度實為“南派”譜系建築的代表。斗栱立面的原始豎向模數控制線為一材（15分°），橫栱的長度與縱向的出跳不再密切相關，慢栱的比例縮短而令栱加長，橫栱的組合顯得錯落有致。《法式》更多地反映出江南地區的審美趣味，斗栱比例緊湊，造型秀美精緻。其最大成就在於分°制的應用，通過分°值模數制來精確權衡斗栱的比例，較唐遼和北宋初期更為細膩，體現了高超的設計智慧。林徽因先生在《清式營造則例》緒論中指出：“美的大部分精神所在，卻蘊於其權衡中：長與短之比，平面上各大小部分之分配，立體上各體積、各部分之輕重均等，所謂增一分則太長，減一分則太短的玄妙。”以此來評價《營造法式》斗栱設計的分°值模數制是最恰當不過的。

⑤《法式》栱頭卷殺的典型做法是“四瓣卷殺”：栱頭上留6分°，下殺9分°；從栱頭順身量四瓣，每瓣長4分°。至於三瓣或五瓣以及分°數不同的做法，都是根據實際情況在以上“理想模型”的基礎上做出的微調。

⑥推測佛光寺東大殿單材高10寸、厚7寸，栔高4.5寸，足材高14.5寸。栱長及出跳尺寸：泥道栱長43.5寸（心長36寸 + 散鬥下寬7.5寸），慢栱長73.5寸（心長66寸+ 散鬥下寬7.5寸），瓜子栱、令栱長40.5寸（心長33寸+ 散鬥下寬7.5寸），一大跳心長33寸（第一跳18寸，第二跳15寸，第三、第四跳16.5寸）。櫨鬥上方21寸，下方16寸，高14.5寸；散鬥、齊心鬥上方10.5寸，下寬7.5寸，下深7寸，高7寸。昂制：平出33寸，抬高14.5寸。

⑦張十慶先生文中注3：“唐佛光寺大殿的令栱與泥道栱等長是已知最早之例”。但實際明顯不等，據實測泥道栱長約1290毫米，令栱長約1200毫米。

⑧ 觀音閣與應縣木塔的鋪作側樣設計不如佛光寺東大殿嚴謹（例如七鋪作兩個大跳的尺寸不再統一），模數控制線對側樣的控制僅基本限於第一大跳之內。

⑨材厚與栱厚有時並不一定相等，實例中存在縮減材厚的“偷料”做法。如北宋少林寺初祖庵，用六等材，材厚為4寸（1分°= 0.4寸），但實際栱厚偷減為3.7寸。又如明故宮英華殿，鬥口2.5寸，實際栱厚偷減為2.4寸。因此在古建築實測數據解讀中，不能簡單地將材厚等同於栱厚，須反覆校驗。
