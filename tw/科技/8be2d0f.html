Linux搜索文本數據的最佳工具

2020-10-29
| 科技

在bash中常用命令行工具：


1. Grep

首先是grep實用工具。使用grep，您可以執行多種功能，例如：

  • 搜索文件中的字符串或匹配模式。
  • 在壓縮文件中搜索字符串或匹配模式。
  • 計算字符串匹配數。
  • 打印包含字符串或模式的行號。
  • 遞歸搜索目錄中的字符串。
  • 執行反向搜索（即顯示不符合搜索條件的字符串的結果）。
  • 搜索字符串時忽略大小寫敏感。

使用grep命令的語法非常簡單：

$ grep pattern FILE

忽略大小寫的情況下在文件中搜索字符串“ Linux ”（例如a.txt),命令：

$ grep -i Linux a.txt



2. Ack

Ack是用Perl編寫的快速且可移植的命令行工具。Ack被認為是grep實用程序的友好替代品，並以視覺吸引人的方式輸出結果。

Ack命令在文件或目錄中搜索包含與搜索條件匹配的行。然後突出顯示各行中的匹配字符串。

Ack能夠根據文件擴展名以及在一定程度上文件中的內容來區分文件。


$ ack Linux a.txt

搜索工具非常智能，如果用戶未提供文件或目錄，它會在當前目錄和子目錄中搜索搜索模式。

在下面的示例中，沒有提供文件或目錄，但是ack自動檢測到了可用文件並搜索了提供的匹配模式。


$ ack Linux
