Excel根據公式說明計算數值

2020-10-29
| 科技

有時候，在報表中需要對一個數值的計算公式進行說明，以展示該數據是如何得來的。當然可以先設置數據的公式，然後在粘貼到說明單元格內。但是還有更加方便的做法，就是隻需修改說明單元格內的公式，數值就根據該公式相應改變。


壹


例如：

A1單元格顯示數值6，其公式為2*3，其公式說明在B1單元格內展示。那麼如何使得A1單元格內的數值根據B1單元格內書寫的公式說明變化呢？不用在A1重新設置公式。


1、首先選中A1單元格，必須要選中。

2、依次點擊菜單“公式”、“定義名稱”。

3、在彈出的窗口中“名稱”和“引用位置”內輸入以下內容，然後“確定”。

“名稱”這裡輸入的是JS，也可以輸入別的名稱，只是需要記住，以便後面引用。

“引用位置”只能手工輸入，單元格位置中沒有“$”符號。

4、在A1單元格中輸入“=JS”，這裡相當於我們定義了一個以JS為名稱的函數。

雖然在第3步，我們只輸入了B1，但是該公式是可以往下拖動的，加減乘除毫無問題。A2對應B2，A3對應B3。

但是如果在C1輸入“=JS”則會等於D1中公式的值。這就是一開始選中A1單元格的重要性。


貳


但是我們的報表常常像下面這樣，在C列金額均等於B2單元格的單價乘以B列相應的數量。公式這時用字母表示更清晰。


你會發現用剛才的辦法，先選中C5單元格，定義一個名稱依然有效。


叄


當然我們也發現了前面做法的缺陷就是有對應位置的關係，比如剛才的C1單元格輸入“=JS”時就不會等於B1單元格所列公式的值。

那麼，能不能像真正的函數一樣，我應用哪個單元格就計算該單元格公式的值呢？顯然，也是可以的。

1、首先“Alt+F11”，調用代碼編輯窗口。然後右鍵點擊VBAProject，選擇插入“模塊”。


2、雙擊新建的模塊（列如模塊1），在右邊的窗口中輸入以下內容：

Function JSSS(A)

A=A

JSSS = Evaluate(A)

End Function


如下圖：中“JSSS”就是我們真正自定義的函數了，可以像其他Excel函數一樣使用。其中"A=A”是千萬不能省略的。


3、B11單元格中輸入公式“=JSSS(D5)”，其中D5可以像其他函數一樣點擊D5單元格來引用，因為JSSS現在真的就是一個函數了。


肆


然而還有的時候，我們會這樣來寫一個數值的計算說明，公式中會帶有計量單位。


計算公式是：（71.1*5+100*10-100）*5/7，其中包含了括號、加減乘除。

這個稍微複雜一點，但是依然可以簡單搞定。

我們可以用“Alt+F11”調出代碼窗口，在新插入的模塊或原有的模塊中寫入下面的代碼：

Function ZJS(A)

Dim b, mhk, mh

Dim dy, dys


Set dy = CreateObject("vbscript.regexp")

With dy

.Global = True

.IgnoreCase = True

.Pattern = "(\d|[*+-.()（）]|[/]\d)"

End With

Set mh = dy.Execute(A)

For Each mhk In mh

dys = dys & mhk

Next


ZJS = Evaluate(dys)


End Function

如圖：

其中ZJS就是我們定義函數的名稱。

在B1單元格輸入公式“=ZJS(A1)”就會得到A1中數值運算的結果。


最後需要注意的是，在使用了模塊之後，Excel必須啟用宏。
