1.mysql自帶了不少函數，但是不一定都能滿足我們的需求，所以我們可以用function來自定義函數，完成一定的功能，並返回結果。

【案例操作】

1.￥￥創建一個計算斐波那契數列的函數￥￥

create function f_factorial (num int) returns int

begin

declare result int default 1;

while num>1 do

set result=result*num;

set num=num-1;

end while;

return result;

end

【存儲函數與存儲函數的區別】

1.存儲過程可以有多個in,out,inout參數，而存儲函數只有輸入參數類型，而且不能帶in.

2.存儲過程實現的功能要複雜一些；而存儲函數的單一功能性（針對性）更強。

3.存儲過程可以返回多個值；存儲函數只能有一個返回值。

4.存儲過程一般獨立的來執行；而存儲函數可以作為其它sql語句的組成部分來出現。

5.存儲過程可以調用存儲函數。函數不能調用存儲過程。
