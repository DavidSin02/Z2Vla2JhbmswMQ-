2020-11-14

01

循環流化床鍋爐空氣預熱器有何作用？

利用排煙熱量加熱鍋爐助燃所需空氣的受熱設備，叫做空氣預熱器。空氣預熱器的作用是：

1）強化燃燒。由於提高了鍋爐的助燃空氣的溫度，可以縮短燃料的乾燥時間和促使揮發分析出，從而使燃料迅速著火，加快

燃燒速度，增強燃燒的穩定性，提高燃燒的效率；

2）強化傳熱口由於使用了熱空氣並增強了燃燒，可以提高燃燒室的煙氣溫度，加強爐內輻射換熱；

3）提高鍋爐運行的經濟性，加裝了空氣預熱器可以有效的進一步降低排煙溫度，減少排煙損失，提高鍋爐效率。

02

循環流化床鍋爐空氣預熱器有哪幾種形式？

循環流化床鍋爐目前採用的空預器有三種，大多數循環流化床鍋爐使用管式空預器，管式空預器又分為立管式和臥管式；少數循環流化床鍋爐採用熱管空預器，它的優點是漏風係數較小；第三類是採用迴轉式空預器，它的優點是相對體積較小，適合大容量循環流化床鍋爐。如引進的白馬 300MW 循環流化床鍋爐。由於循環流化床鍋爐一次風壓較高，為避免漏風係數過大，用於循環床的迴轉空預器採用特殊分倉和密封方式。

03

為什麼循環流化床鍋爐不宜採用立式管式空預器？

由於循環流化床鍋爐風機壓頭比煤粉鍋爐高很多，如果採用立式管式空預器，空氣將從管外走，空預器護板的密封性不好，容易漏風。而採用臥式管式空預器，空氣從管內走，密封結構更易於處理，避免漏風。此外，採用臥式管式空預器，煙氣在管外橫向沖刷，空預器管子壁溫較高，不易腐蝕。

04

空氣預熱器的腐蝕與積灰是如何形成的？

由於空氣預熱器處於鍋爐內煙溫最低區，特別是未級空氣預熱器的冷端，空氣溫度最低、煙氣溫度也最低，受熱面壁溫最低，因而最易產生腐蝕和積灰。當燃用含硫量較高的燃料時，生成的 SO 2 和 SO 3 氣體，與煙氣中的水蒸氣生成亞硫酸或硫酸蒸汽。在排煙溫度低於酸蒸汽露點時，硫酸蒸汽便凝結在受熱面上，對金屬壁面產生嚴重腐蝕。同時，酸液體也會粘結煙氣中的灰分，越積越多，易產生堵灰。循環流化床鍋爐尾部煙道受熱面積灰，受熱面表面傳熱係數下降，使吸熱量下降，排煙溫度上升，鍋爐熱效率下降。如果積灰嚴重，則會增加煙道阻力，導致引風機負荷增大，廠用電率增加。長期腐蝕和積灰會造成受熱面的損壞和洩漏。當洩漏不嚴重時，可以維持運行，但使引風機負荷增加，限制了鍋爐出力，嚴重影響鍋爐運行的經濟性。

05

什麼是鍋爐的低溫腐蝕？

由於燃煤中含有 S，而 S 在燃燒過程中會產生 SO 2 ，進而部分 SO 2 會被氧化成 SO 3 ；另一方面，鍋爐煙氣中還含有 NOx 等酸性氣體，在煙氣溫度較低時，這些酸性氣體會與煙氣中的水蒸氣發生反應生成相應的酸，生成的酸附著在尾部受熱面以後，會對尾部受熱面的金屬產生腐蝕現象；或者在尾部換熱管壁溫度較低時，煙氣中的酸性氣體與管壁上的凝結水發生反應生成稀酸，腐蝕尾部受熱面的金屬，統稱為低溫腐蝕。
