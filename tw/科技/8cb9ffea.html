電力拖動的運動方程式

在電力拖動中涉及的相關物理量有轉矩，轉動慣量，角速度和角加速度。G*D的平方為飛輪矩，飛輪矩與轉動慣量有關。最簡單的電力拖動系統為單機單軸系統，多機多軸系統也存在但是較為複雜。

在多軸系統中還涉及到了折算：如負載轉矩的折算，飛輪矩的折算，平移部件質量m的折算。

生產機械地負載轉矩特性

1、恆轉矩負載特性

當轉速變化時，負載轉矩的大小保持不變，稱為恆轉矩負載。

（1）反抗性恆轉矩負載： Tr 的大小與 n 無關，但是 Tz 的方向始終與 n 相反。

（2）位能性恆轉矩負載： Tz 的大小及方向均與 n 無關，而且 Tz方向始終指向使位能部件的勢能減小的方向。

2. 恆功率負載特性

負載轉矩與轉速成反比。

3. 通風機負載特性

負載轉矩與轉速平方成正比。

他勵直流電機的制動

1.制動運行 其他不變，電磁轉矩與轉向相反

2. 能耗制動 保持If大小方向不變，將電樞迴路從電網脫離經制動電阻 Rz閉合。

3. 反接制動 保持 If 大小方向不變，令電樞經制動電阻 Rz 而反接於電網。

4. 回饋制動 保持 If大小方向不變，將電樞反接，從而使位能性負載以較高的轉速穩速下放。

他勵直流電機的過渡過程

1. 當電力拖動系統的負載、參數或運行方式等發生變化時，由於系統具有慣性（機械慣性、電磁慣性與熱慣性等），使之從一個穩定運行狀態變為另一個穩定運行狀態時，系統各物理量的改變不可能瞬時完成，必須經一個連續變化的過程，稱為過渡過程。在此過程中，各物理量隨時間的變化規律稱為拖動系統的動態特性 。
