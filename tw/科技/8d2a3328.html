軟網推薦：資源管理器直連Android文件夾

2020-11-14
| 科技

在電腦旁工作，有時我們需要訪問手機中的文件，這時在手機上通過軟件將文件傳輸到電腦的操作不夠方便。如果能在Windows環境下，在資源管理器中輸入一個簡單的單詞，便可以直接連接到手機文件夾，文件操作將變得非常方便。藉助於Tablacus Explorer資源管理器擴展和WebDAV服務，我們即可實現上述想法。

1. Android設備安裝並啟用WebDAV服務

首先在Android移動設備上的應用商店裡搜索“webdav server”並安裝WebDAV Server，啟動該服務軟件。

按下紅色按鈕後開啟WebDAV Server，之後按鈕變成綠色，屏幕上會出現WebDAV Server服務地址，例如“http://192.168.0.106:8080”。



2. 在Tablacus Explorer中安裝WebDAV插件

前往Total Commander - Plugins插件下載頁面中下載WebDAV插件（下載地址：http://www.ghisler.com/plugins.htm#filesys）。點擊WebDAV後的“32+64 bit”下載webdav.zip文件。解壓下載的webdav.zip文件到一個文件夾中。



接下來安裝Total Commander插件到Tablacus Explorer。先點擊“Help -> Get Add-ons”菜單命令，再點擊Total Commander file system plug-in後的Install按鈕安裝。




接著點擊左側欄中的Add-ons項，然後點擊右側欄中Total Commander file system plug-in後的Options按鈕。



隨後點擊“Browse...”按鈕，選擇剛才解包後文件夾中的davplug.uwfx，並單擊Open按鈕。成功時，將在Info信息列中顯示各種信息，單擊Add按鈕後點擊OK完成插件添加設置。



3. \\\WebDAV地址欄輸入Android訪問手機

啟動Tablacus Explorer資源管理器，在地址欄中輸入“\\WebDAV”（區分大小寫，不輸入引號），然後按下回車。



按下Ctrl並點擊K鍵打開文件夾建立屏幕（也可以按下F7）。給出適當的文件夾名字（如Android），並按下OK按鈕。



當設置屏幕出現之後，將先前記錄下的手機URL地址填寫到“Connect to”文本框中。然後選中“Send/Receive accents in URLs as UTF-8 Unicode”（如果不選中，非ASCII文件名將出現亂碼），點擊OK按鈕。



這樣就可以在Tablacus Explorer中，通過指定的文件夾名稱（這裡為Android），訪問Android手機文件夾了。我們可以轉到適當的文件夾，並可以提取和插入文件。



擴展閱讀：

另一種方法是在手機上安裝第三方軟件（如“軟件數據線”，或手機文件管理器APP內集成功能），啟動服務後生成一個局域網FTP訪問網址，在Windows文件資源管理器中輸入這個FTP地址，也可實現在PC資源管理器中訪問到Android的文件夾，對文件進行操作。
