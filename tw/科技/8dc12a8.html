2020高壓電工考試題及高壓電工實操考試視頻

2020-10-29
| 科技

題庫來源：安全生產模擬考試一點通公眾號小程序

2020高壓電工考試題及高壓電工實操考試視頻，包含高壓電工考試題答案解析及高壓電工實操考試視頻練習。由安全生產模擬考試一點通公眾號結合國家高壓電工考試最新大綱及高壓電工考試真題出具，有助於高壓電工理論考試考前練習。

1、【判斷題】雷電直接擊中建築物或其他物體,對其放電,強大的雷電流通過這些物體入地,產生破壞性很大的熱效應和機械效應,雷電的這種破壞形式稱為感應雷過電壓。（ × ）

2、【判斷題】 變壓器的效率η為輸出的有功功率與輸入的有功功率之比的百分數。（ √ ）

3、【判斷題】 瓷套管是否清潔,有無缺損、裂紋和放電現象,聲音是否正常,是電流互感器巡視檢查項目之一。（ √ ）

4、【判斷題】 在中性點不接地的電力系統中,單相接地故障運行時間一般不應超過2h。（ √ ）

5、【判斷題】 相線與中性線(或零線)間的電壓稱為相電壓。（ √ ）

6、【判斷題】 在磁體內部,磁力線的方向是由N極到達S極。（ × ）

7、【判斷題】 工作票簽發人不得兼任所簽發工作票的工作負責人。（ √ ）

8、【判斷題】 電壓互感器的高壓繞組與被測電路串聯,低壓繞組與測量儀表電壓線圈串聯。（ × ）

9、【判斷題】 絕緣鞋可作為防護跨步電壓的基本安全用具。（ √ ）

10、【判斷題】 當橫截面積相同時,同種材料導體的長度越長,導體的電阻越大。（ √ ）

11、【判斷題】 避雷器用來防護高壓雷電波侵入變、配電所或其他建築物內,損壞被保護設備。（ √ ）

12、【判斷題】 鋼筋混凝土杆又分普通型預應力杆和等徑預應力杆兩種。（ × ）

13、【判斷題】 交流電路中,電弧電流瞬時過零時電弧將消失,此後若觸頭間的介質擊穿電壓≤恢復電壓,則電弧將重燃。（ √ ）

14、【判斷題】 高壓熔斷器的熔絲一相或多相熔斷後,在熔斷器撞擊器的作用下使負荷開關跳閘,可防止由於缺相運行而造成的電氣設備損壞。（ √ ）

15、【判斷題】 更換成組的電壓互感器時,還應對並列運行的電壓互感器檢查其連接組別,並核對相位。（ √ ）

16、【判斷題】 當鐵損和銅損相等時,變壓器處於最經濟運行狀態,一般在其帶額定容量的50%~70%時。（ √ ）

17、【判斷題】 感應電動勢的方向與磁力線方向、導體運動方向相關。（ √ ）

18、【判斷題】 工作過程中,工作負責人和工作許可人任何一方不得擅自變更安全措施。（ √ ）

19、【判斷題】 撲滅火災時,滅火人員應站在下風側進行滅火。（ × ）

20、【判斷題】 雷雨天氣,需要巡視室外高壓設備時,應穿絕緣靴,並不得靠近避雷器和避雷針。（ √ ）

21、【判斷題】 交流高壓真空接觸器利用真空開關管熄滅電弧。（ √ ）

22、【判斷題】 對停電的注油設備應使用乾燥的沙子或泡沫滅火器等滅火。（ √ ）

23、【判斷題】 對帶電設備應使用不導電的滅火劑滅火。（ √ ）

24、【判斷題】 手車式開關櫃的斷路器,手車在接地開關合閘位置時可自由移動。（ × ）

25、【判斷題】 在中性點非直接接地的電力系統中廣泛採用兩相不完全星形接線方式來實現相間短路保護。（ √ ）

26、【判斷題】 運行過程中易發生過負荷和需要防止起動或自起動時間過長的電動機應裝設過負荷保護。（ √ ）

27、【判斷題】 絕緣安全用具分為基本安全用具及輔助安全用具。（ √ ）

28、【判斷題】 低壓電筆的測量範圍為500V以下。（ × ）

29、【判斷題】 臨時接地線的連接要使用專用的線夾固定,其接地端通常採用綁紮連接,各連接點必須要牢固。（ × ）

30、【判斷題】 變壓器的允許溫度主要決定於繞組的絕緣材料。（ √ ）

31、【判斷題】 人體距離高壓帶電體太近,造成對人體放電,這種情況的觸電稱之為間接接觸觸電。（ × ）

32、【單選題】 變、配電所一次主接線中所用的電氣設備,稱為()。（ C ）

A、遠動設備

B、二次設備

C、一次設備

33、【單選題】 手車式開關櫃,小車已推入,開關斷開,稱之為()。（ C ）

A、運行狀態

B、檢修狀態

C、備用狀態

34、【單選題】 電對人體的傷害,主要來自()。（ A ）

A、電流

B、電壓

C、電磁場

35、【單選題】 負荷開關與熔斷器配合使用時,由熔斷器起()作用。（ A ）

A、短路保護

B、切斷正常負荷電流

C、倒閘操作

36、【單選題】 運行過程中易發生過負荷和需要防止起動或自起動時間過長的電動機應裝設()。（ C ）

A、失步保護

B、低電壓保護

C、過負荷保護

37、【單選題】 倒閘操作票由()填寫。（ C ）

A、許可人

B、監護人

C、操作人

38、【單選題】 手車式開關櫃斷路器手車在試驗位置不能搖進的原因之一是()。（ A ）

A、接地開關在合閘位置好

B、接地開關在分閘位置

C、斷路器在分閘位置

39、【單選題】 所有斷路器,隔離開關均斷開,在有可能來電端掛好地線,說明設備處於()狀態。（ A ）

A、檢修

B、運行

C、備用

40、【單選題】 當電纜導體溫度等於電纜最高長期工作溫度,而電纜中的發熱與散熱達到平衡時的負載電流稱為()。（ A ）

A、電纜長期允許載流量

B、短時間允許載流量

C、短路允許載流量

41、【單選題】 拆除臨時接地線的順序是()。（ B ）

A、先拆除接地端,後拆除設備導體部分

B、先拆除設備導體部分,後拆除接地端

C、同時拆除接地端和設備導體部分

42、【單選題】 電流互感器的一次線圈()接入被測電路,二次線圈與測量儀表連接,一、二次線圈極性應正確。（ C ）

A、混聯

B、並聯

C、串聯

43、【單選題】 裝、拆接地線必須由()完成。（ B ）

A、一人

B、兩人

C、三人

44、【單選題】 導體在磁場中做切割磁力線運動時,導體內就會產生()。（ C ）

A、磁場

B、電流

C、感應電動勢

45、【單選題】 把設備編號和接線端子編號加在一起,每一個接線端子就有了唯一的()。（ C ）

A、設備文字符號

B、迴路編號

C、相對編號

46、【單選題】 在某一個時段內,電壓急劇變化而偏離額定值的現象,稱為()。（ A ）

A、電壓波動

B、電壓閃避

C、電壓偏移

47、【單選題】 可以不斷開線路測量電流的儀表是()。（ C ）

A、電流表

B、萬用表

C、鉗表

48、【單選題】 電力系統過電壓分成兩大類()。（ B ）

A、外部過電壓和大氣過電壓

B、外部過電壓和內部過電壓

C、操作過電壓和短路過電壓

49、【單選題】 電網諧波的產生,主要在於電力系統中存在各種()元件。（ C ）

A、電感元件

B、電容元件

C、非線性元件

50、【單選題】 變壓器()通常分為無勵磁調壓和有載調壓兩種方式。（ C ）

A、運行方式

B、連接方式

C、調壓方式
