VBA代碼如何對隨機文件進行讀取，並輸出一個準數據庫文件的記錄

2020-10-29
| 科技

大家好，我們今日繼續講解VBA代碼解決方案的第136講內容：如何操作隨機文件。在上一講中，我講了建立一個準數據庫文件的方法，今日我們就來講操作這個數據庫。我們還是利用上一講已經建立的文件,來完成我們的輸出過程。下面的過程中首先要以隨機的方式打開文件，然後確認文件的大小，已經存在的記錄數，最後把文件的內容輸出；

Option Explicit

Type MyDictionary

myen As String * 10

mysp As String * 20

End Type

Sub MyoutDictionary ()

Dim d As MyDictionary

Dim i As Integer

Dim recNr As Long

Dim records As String

Open "E:\nz\work\文章\MytypeDictionary.txt" For Random As #1 Len = Len(d) '打開隨機訪問文件

MsgBox "There are " & LOF(1) & " bytes in this file." ' 提示本文件的總字節數

recNr = LOF(1) / Len(d) '顯示總記錄數

MsgBox "Total number of records: " & recNr

For i = 1 To recNr

Seek #1, i ' 找到該隨機記錄

Get #1, i, d ' 讀取記錄

Sheets("sheet8").Cells(i, 1) = d.myen

Sheets("sheet8").Cells(i, 2) = d.mysp

Next

Close #1 ' 關閉文件

msgbox("OK!")

End Sub

代碼的截圖：

過程分析：過程MyoutDictionary先聲明變量，之後，打開一個隨機訪問文件"E:\nz\work\文章\MytypeDictionary.txt"，並且告訴VB每個記錄的長度Len = Len(d)，接下來，在彈出的窗口裡顯示文件的總字節數和總記錄數。字節數是由語句LOF(1)返回的。記錄數是總字節數（LOF）除以一個記錄的長度Len(d)。接下來，VBA執行循環裡的語句,指令Seek #1, i.在開啟的文件中移動光標到變量i記錄處，讀取記錄內容。要在打開的隨機訪問文件中讀取數據，你必須使用Get語句。

指令：Get #1, i, d

告訴VBA要讀取的記錄號碼I,以及要讀取數據的變量d。隨機文件中的第一個記錄在位置1，第二個記錄在位置2，依次類推。然後，用戶定義的類型字典的兩個成員都被輸出了出來。輸出的位置：

Sheets("sheet8").Cells(i, 1) = d.myen

Sheets("sheet8").Cells(i, 2) = d.mysp
