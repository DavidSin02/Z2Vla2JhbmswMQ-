Unison Unico CD Uno CD唱盤

2020-10-29
| 科技

意大利音響廠牌Unison Research不但以管機著稱，而且他們還喜歡將意大利最為擅長的木工運用在管機箱體上，因此Unison Research的管機運作時，其機箱流線型的金屬零件反射著真空管熾紅的光芒，加上以木工打造的流線型面板，帶來聽覺與視覺雙重享受，也賦予Unison Research管機名貴的印象。

不過，近幾年來，音響市場逐漸萎縮，年輕族群的聆聽習慣也與先前的世代大不相同，許多Hi-End廠牌為了吸引年輕族群的目光，紛紛推出轉型之作，Unison Research也不例外，自2000年起，他們邀請意大利帕多瓦（Padua）大學的Leopoldo Rossetto教授，參與「Unico」系列的開發計畫，除了推出多臺管晶混合設計的放大器，更考量到現在有多種音樂格式在市場流通的狀況，推出結合CD播放機、USB DAC與無線藍牙的數字訊源器材。這次試聽的主角——Unico CD Uno，便是其中一例。

Unico CD Uno：CD/USB/藍牙三合一

根據原廠的說明，這臺Unico CD Uno是延續 CD Duo設計推出的最新CD播放機。純鋁合金打造的箱體，中央可以顯示音檔取樣率的白色128 X 64 OLED顯示屏幕，兩旁的電源以及基本播放功能按鍵，還有抽屜式的CD承盤，各部外觀與尺寸，幾乎與CD Duo一模一樣。不過，Unico CD Uno比起Unico CD Duo少了AES/EBU XLR數字輸入端子，保留了較為常用的USB Type B與Toslink光纖數字音訊輸入端子，以及一個S/PDIF同軸端子，而且還具備連接手機十分便利的藍牙3.0無線連接功能。

Unico CD Uno內部電路，可以看出DAC電路與模擬電路位於不同的電路板上，減少互相干擾的機會發生。（圖片由垣慶提供）

這臺CD播放機採用奧地利公司StreamUnlimited生產的8829-CD承盤，我在使用時，發現此承盤進出順暢，不會有抖動的情形發生。Unico CD Uno外觀雖然看起來傳統，但是播放功能絲毫不落伍。其DAC電路搭載ESS Sabre32 9018 DAC芯片，並還有儲存自寫程式的Xilinx Spartan FPGA芯片，做為數字音訊的處理引擎，時脈則由石英震盪器負責，以減少Jitter的產生。

在此電路設計之下，Unico CD Uno的USB輸入可以支持至384 KHz/32 bits的PCM檔以及11.2 MHz/1 bit的DSD檔，而通過Toslink光纖輸入，也可支持至176.4kHz/24 bits的PCM檔，以及2.8MHz/1 bit的DSD檔，讓用戶得以聆聽市面上大多數的Hi-Res音檔。

Unico CD Uno的DAC電路搭載ESS Sabre32 9018 DAC芯片，並還有儲存自寫程式的Xilinx Spartan FPGA芯片，做為數字音訊的處理引擎。（圖片由垣慶提供）

此外，Unico CD Uno也提供三種數字低通濾波選擇，包含滾降斜率陡直、線性相位的「F1」、滾降斜率陡直、最小相位的「F2」、滾降斜率平緩、線性相位的「F3」，這三種數字濾波分別對時間一致性、相位一致性以及暫態細節有較為理想的效果。我在試聽時，試著切換這三種濾波器，發現其對聲音的影響並不大，很難說孰優孰劣，不過，若是覺得音樂略為刺耳，選用「F1」與「F2」可以達到較為舒適的聽感。

搭載ECC82真空管

由於代理商特別囑咐，我無法將Unico CD Uno的箱體打開，觀看內部電路佈局，不過根據原廠提供的資料，Unico CD Uno的DAC電路採用8組穩壓電源供應。在真空管電路上，搭載了1支12AU7/ECC82真空管，採圖騰柱式（totem pole，也就是臺灣俗稱的推輓式）設計，跟搭載4支12AX7/ECC83的CD Duo相比，換管更為容易，省去配對的煩惱。電路上也分別為真空管的陰極與柵極設立獨立的穩壓，維持電路的穩定。

最後，Unico CD Uno的緩衝及採用了德儀出產的NE5532AD OP芯片作為I/V轉換，並搭載威瑪（WIMA）電容以及威士（Vishay）的電阻，這些都是許多發燒友心儀的好聲元件。玩家還可通過Unico CD Uno的功能選單，選擇關閉真空管，讓Unico CD Uno以純電晶體方式發聲，就算真空管壽命已盡，來不及換管，也不用擔心這臺器材無法發聲運作。

Unico CD Uno 搭載了1支12AU7/ECC82真空管，並採用威瑪（WIMA）電容以及威士（Vishay）電阻等發燒元件。

由以上的介紹不難看出，Unico CD Uno表面上看起來雖然是傳統的CD播放機，但是具備了可以與手機直接連線的藍牙功能，U也能滿足現在大多數玩家播放數字流以及手機音樂的需求，價格上也比CD Duo少了5萬元之譜，是Unison Research貼近年輕、中階玩家的數字訊源力作。

Unico 90：可支持Bi-Wiring的管晶混合綜擴

我本來以為代理商只會單獨送來Unico CD Uno到我們辦公室進行評測，沒想到代理商還同時送來同系列的Unico 90真空管綜擴以及體積十分龐大的Max 2落地喇叭，與Unico CD Uno一起搭配試聽，陣仗之大，讓我感到受寵若驚。Unico 90廠商定價為195,000元，比起Unison Research先前推出的放大器相比，價格也是親民許多。

雖然Unico 90少了Unison Research管機向來慣有的木板裝飾，讓這臺真空管綜擴看起來樸素不少，不過Unico 90重達20公斤，機箱體積看起來也十分龐大，讓人覺得分量十足，再加上霧面處理的灰色鋁合金面板，平實中亦散發精緻的質感。

Unico 90中的電路可分成輸入級與輸出級兩大部分。輸入級為具有真空管的「圖騰柱式」推輓電路，其中在每個聲道中，搭載一枚ECC83三極管以及雙並聯ECC81真空管。輸出級各聲道搭載了三對由國際整流公司（International Rectifier）所生產的HEXFET MOSFET電晶體作為功率放大元件，並採用兼具音質與效率的A/B類混合設計，在小功率輸出時，以純A類擴大方式運作，若是輸出功率超過一定的瓦數，那麼就會以效率較高的A/B類混合方式運作，以傳達較高的輸出功率。如此設計下，讓Unico 90在8歐姆承阻下輸出功率可達100W，在4歐姆承阻下，輸出功率可達180W。

這臺放大器也與Unico CD Uno一樣，具有彈性極高的搭配性能，除了在背板上搭載3組非平衡輸入端子以及2組平衡輸入端子，每個聲道更具備2組喇叭輸出端子，可支持Bi-Wiring連接方式，提供喇叭絕佳的驅動力。為了讓Unico 90有著自然舒適的音色，原廠在裡面不採用整體負回授電路，只使用局部負回授電路。除此之外，Unico 90還具備前級電路bypass功能，搭配其它前級單純作為後級使用，更具備超低音輸出，為2.1聲道玩家提供便利的連接方式，可說是一臺設計精良、進退皆宜的綜合放大器。

Max 2：號角高音加布邊振盆低音，為管機而生

號角喇叭與真空管放大器為許多發燒玩家心目中的夢幻搭配，而Unison Research為了搭配自家的真空管放大器，便推出了兩款兩音路號角喇叭——Max 1與Max 2。這次送來與Unico CD Uno搭配的Max 2，採落地式式設計，Max 2高達101公分，再加上寬障板的設計，讓Max 2的寬度也達43.5公分，而箱體深度也達53公分，讓Max2看起來頗具份量，箱體容積高達130公升。

Max 2的中高音單元採用1.4寸鈦振膜壓縮式單元，以鋁質線圈推動，兼具質地輕盈、散熱性佳的特性，為單元帶來絕佳的反應速度，並減少熱能產生的失真。在單元前方，搭載有60度乘以50度的恆定角度橢圓形號角（Constant Coverage ESS Horn），以抑振效果絕佳的聚胺甲酸酯（polyurethane）鑄造而成，如此一來，壓縮式單元在發聲時，號角不易隨聲波產生共振，可以避免不必要的音染。

Max 2的高音單元搭載橢圓形號角，以聚胺甲酸酯鑄造而成，並以60度乘以50度的擴散角度傳導音波。

低音單元則採用1枚口徑達15寸的單元，其振膜以植物纖維以及碳纖維混合材質製程，而且表面具有多道同心圓皺摺，以增強單元振膜的韌度，提升單元的功率承受能力。此外，該低音單元採用有「Ｗ」三道皺褶形狀的布質懸編，以51%的天然棉以及49%的聚酯纖維製成。這樣的懸邊設計，除了提升單元的功率承受能力外，更重要的是還能減少單元的衝程距離，進而提升單元的反應速度，以重現更為清晰的細節。

至於衝程距離雖然減少，但該單元本身口徑高達15寸，能推動充足的空氣量，傳達的聲音能量不至於有所減損。除此之外，該低音單元的磁力系統以強力的釹磁鐵製成，並搭載直徑達10公分的線圈以及鋁質套筒，讓該單元可以產生800瓦的連續輸出功率。在該單元的磁鐵周圍，還搭載有兩個特製個金屬環，原廠聲稱這樣的設計可以調節磁力線的變化，減少諧波以及內調失真。

Max 2低音單元口徑達15寸的單元，其振膜以植物纖維以及碳纖維混合材質製程，而且表面具有多道同心圓皺摺，以增強單元振膜的韌度，提升單元的功率承受能力。此外，該低音單元採用有「Ｗ」三道皺褶形狀的布質懸編，以51%的天然棉以及49%的聚酯纖維製成。低音反射孔位於箱體前方，擺放時不用擔心後牆會影響到低音單元的運作。

與Opera聯手打造

此對喇叭分頻點設於800Hz，分音電路採取不對稱的設計，低通濾波的滾降速度為每八度12dB，高通濾波的上升速度為每八度6dB，分音電路採用意大利SET製造的印刷電路板，並採用無氧銅繞制的電感、Mundorf電容以及MOX電阻。在Max 2的後方，還有一個切換開關，可以改變高通濾波器的斜率，近而改變高音頻段的EQ。

Max 2的箱體與Max 1一樣同樣採用反射式設計，而且反射孔為於前障板下方， 如此一來讓反射孔不會與後牆過近，讓喇叭擺位更為自由。至於箱體本身以及前障板分別以25mm、30mm厚的MDF板製成，而箱體內部採取龍骨支架結構，以強化箱體的抑振能力，而箱體內壁則塗有開放式聚胺甲酸酯（open cell polyurethane）泡棉，減少箱體內音波反射產生不必要的共振。

Max 2箱體外觀以色澤華美的上漆原木貼皮處理，前障板則以合成皮革包覆，這樣的設計，不禁讓我聯想到一個意大利喇叭廠牌——Opera推出的喇叭，事實上，Unison Research與Opera早在1998年為「Audio Research in Audio」集團收購，Max 2身上會有Opera的影子，一點也不意外。除此之外，Max 2紮實的箱體設計，以及幾乎與低音單元直徑等寬的前障板，相信也都承襲自Opera的設計理念。

如同文藝復興繪畫般的高貴氣息

我一開始將Unico CD Uno與Unico 90器材，直接插上排插播放音樂。不知道是不是因為號角喇叭效率較高，對於前端器材較為敏感，或是當時排插還連接著數臺其它器材發現音質明顯較為粗糙、毛躁，不似這套系統該有的水準。後來我將Unico CD Uno、Unico 90器材接上Linear Acoustic Fuzzy電源，原有的毛躁聽感頓時減少許多，音質變為細緻柔和。由此可見，聆聽這套系統時，電源器材必須審慎選用，否則會容易覺得這套系統不耐聽，導致「誤殺忠良」的錯誤判斷。

試聽時，我以CD做為主要播放素材。我首先播放由英國知名女高音Emma Kirkby所演唱蒙特維爾地牧歌第七冊中的〈Con Che Soavita, Labra Odorate〉（Decca 4557182），Emma Kirkby的歌聲純淨無暇，控制力猶如長笛一般精準。通過這套系統播放，我可以立刻感受到這位女高音歌聲的圓潤高雅，以及大鍵琴如同珠玉般閃耀的光芒，加上人聲高雅細膩的強弱變化中，流露出雀躍興奮的喜悅之情，彷佛可以看到Emma Kirkby微笑唱著歌。

在音樂播放的過程中，我發現無論是人聲或是其他伴奏樂器，聲音直接，反應敏銳，而各種聲音出聲後綿延的殘響，更強烈地渲染試聽室中的氣氛，彷佛把視聽室重新打造成音樂廳一般，讓我如臨現場，直接感受音樂家的動人演出。這樣活靈活現的聲音表現，還有絕佳的擴散性，向來是號角喇叭的特色，不過聆聽許久，我發現人聲的結像過於突出，音樂情緒也略微興奮，這時我跑道喇叭後方一看，發現原來高音的EQ切換鈕停留在+檔的地方。我將EQ調至0檔，再把音量旋鈕調高至5度10度，會發現聲音更為乾淨，音場更為深邃，每個聲音不僅聽起來散發獨特的香味以及珍珠般溫潤的光澤色彩，整體音樂詮釋更為深情、高雅。

接著，我播放列夫席茲（Konstantin Lifschitz）彈奏的巴哈《郭德堡變奏曲》（Orfeo C864141A）。這張他於2015年的錄音，相較於他於1995年為Denon所錄製的同曲目演奏，可以用「從心所欲不踰矩」形容，除了節奏分配更加自由靈活，彈至弱音之處，探微賾奧；彈至強音之處，大鳴大放。

通過這套系統聆聽，我不但聽到琴音的色澤猶如珍珠般多彩潤澤，而且在播放至弱音樂段，可以清楚聽見鍵觸之間的強弱差異微妙精細，可以明顯感受到演奏家在樂曲上精雕細琢的細密心思，鍵觸起落細節歷歷在目，而且細節清晰，起落靈巧；播放至強音樂段，我更感受到毫無掩飾的情緒傳達，還有列夫席茲劇力萬鈞的驚人彈奏氣勢，如此可見這套系統非但音色高貴細膩，而且動態極為寬廣，情緒表現十分生動活潑。

情緒渲染力強烈

聆聽上面的古樂以及鋼琴獨奏，我可以確定這套系統可以完美呈現高貴的氣質，這讓我不由得好奇播放搖滾樂會呈現何種聽感。於是，我播放品客佛洛伊德《Wish You Were Here》（EMI– 50999 0 82613 2 8 ）中的〈Shine on your crazy diamond〉。通過這套系統聆聽，樂曲一開始，如同海鷗鳴叫般的電子音效儘管音量微弱，但是線條感特別凝聚、清晰，電風琴的聲音中微微的Vibration特效，亦能清楚呈現，隨後出現的電吉他彈奏，滑指效果也相當鮮明，讓整個音樂聽起來特別扣人心絃，將我帶入一個未知的想像世界。

這套系統亦能展現搖滾樂中該有的魄力，像是樂曲中腳踩鼓所營造的節奏，聽起來噗通有力，小鼓音色清脆，收放自如，中段電吉他亦能表現出暴力、粗獷的質感，並沒有因為這套系統採用真空管以及號角拉吧的古典配置，而有所減損。至於人聲演唱，通過這個系統播放，也能展現激昂的情緒，尤其David Gilmour唱至「You were caught on the crossfire of childhood and stardom」一句，發「caught」中「k」這個子音時，可以明顯感受歌者舌根用力頂住軟顎，製造強烈的氣流爆破聲，彷佛喇叭就是歌者的嘴巴，感覺有氣流以及從其噴出，能量直接而強烈。樂曲播放至11:30的地方，空心吉他的彈奏聲響出現在右側，管樂器吹奏出如同咆勃爵士樂，則位於中央前方，爵士鼓的打擊生則結像於中後方，營造出櫛比鱗次的透視感，讓我看見品客佛洛伊德刻意創造的超現實音樂景象。

最後，我播放普契尼歌劇《蝴蝶夫人》（Decca 4175772）中第三幕最後，由蝴蝶夫人自殺前所演唱的詠歎調〈Con amor muore chi non può serbar vita con onore〉。這首樂曲中，開場低音絃樂器與管樂器以極為沉重的重量，預示蝴蝶夫人即將悲劇性的結束自己的生命，儘管通過Max這對喇叭播放，表現眾多低音樂器齊奏絲毫不費吹灰之力，但是樂器聲音出手直接，而且節像在前，再加上背景變換的聲響還有其他演員的腳步聲結像位於喇叭的後方，讓試聽是充滿凝重的低氣壓，以及躁動不安的空氣分子，感染力時分強烈。女高音Mirella Freni以激昂的歌聲，唱出對男主角Pinkerton的不滿控訴，通過這套系統的播放，抖音如同電波一般觸動我心，聽起來慷慨激昂，情緒絲毫沒有任何保留。

4分16秒，蝴蝶夫人自殺後出現的大鑼聲，更是形體龐大，尾韻不絕，而且又能呈現出金屬打擊樂器迷人的光澤感。最後Pinkerton趕到現場，看見死去的蝴蝶夫人時，樂團演奏的能量高漲至頂峰，通過這套系統聆聽，更能感受到寬闊的音場以及深邃的舞臺透視感，氣勢滂礡。由此可見，這套系統表現大編制、大場面的音樂，不但舉重若輕，而且音樂中血肉靈魂絲毫沒有減損，不會讓人覺得只有靠蠻力發聲。

在試聽這套系統時，PS Audio Stellar DAC正好也在我們公司的試聽室，於是我將CD Uno與PS Audio的Stellar（音量開到全滿）兩相比較，通過USB播放上面CD所轉出的數字音檔。比較之中，我發現這Stellar在細節清晰度方面比CD Uno略勝一籌，但兩者之間實際並不會差距太遠，通過Unico CD Uno聆聽，同樣可以聽見小提琴擦弦以及歌唱流竄在脣齒間的氣流聲，只是Unico CD Uno對於細節的呈現方式更為柔和。在音色方面，相較於Stellar的中性，CD Uno或許因為有真空管的加持，音色總是帶有絲綢般華美、高貴的質感，其所帶來的情緒感染力也極為深刻，這種高貴細膩、舒適柔和的質感，實在讓人不忍換下。

由於Unico 90綜擴與Max 2喇叭上皆有兩對喇叭端子，我便嘗試看看以Bi-wiring接法耹聽。本來期待而來的是更為龐大、有力的低音，但迎面而來的是更乾淨的背景、更深邃的音場、更精準的控制力，音樂表現層次大為提升，捨不得換回一般喇叭接線方式了。

讓人大喊「Bravo」的意式絕美風格

這次試聽的Unico CD Uno，雖然是Unison Research的入門級訊源機種，樸實的鋁合金箱體，不同於Unison Research器材予人精雕細琢、工藝精湛的印象，但是他所帶來的深刻情緒、高貴氣質還有不拘一格的表現力，已在我心中留下無可抹滅的美好印象。將其搭配Unico 90綜擴以及Max 2，全套的意式風情，加上經典的管機與號角喇叭搭配，更為音樂帶來生動無比的氣氛，讓我彷佛坐在史卡拉歌劇院中，觀賞一場風味正宗的意大利歌劇演出，聆聽完畢，讓我忍不住起身鼓掌，為這些來自意大利的「聲樂家」高喊Bravo!
