ICPC--1060:逆序數字&1061:順序輸出各位數字

ICPC--1060: 逆序數字

題目描述

輸入一個正整數，將其逆序輸出，每個數字後有一個空格。

輸入

輸入一個正整數n，你可以假設n在int範圍內

輸出

將n按其逆序輸出，每個數字後有一個空格，輸出佔一行。例如，輸入12354，輸出4 5 3 2 1

樣例輸入

12354

樣例輸出

4 5 3 2 1

提示

整數n對10取模可得到個位上的數字，而利用整數商是整數這一特點，整數n除以10的商正好“扔掉”了個位上的數字

代碼

#include<stdio.h>int main(){    int m;    scanf_s("%d", &m);        while (m != 0) {        printf("%d ", m % 10);        m = m / 10;    }    return 0;}

ICPC--1061: 順序輸出各位數字

題目描述

輸入一個不大於10的9次方的正整數，從高位開始逐位分割並輸出各位數字。

輸入

輸入一個正整數n,n是int型數據

輸出

依次輸出各位上的數字，每一個數字後面有一個空格，輸出佔一行。例如，輸入 12345 ，輸出 1 2 3 4 5

樣例輸入

12345

樣例輸出

1 2 3 4 5

提示

注意整數運算避免使用double類型的函數如pow()。 本題可先用一個循環計算出最高位的位權h，然後再用一個循環，循環內容為： 輸出最高位(n/h)、扔掉最高位(n = n%h)、降低最高位位權（h = h/10)，直到位權h為0。

代碼

#include<stdio.h>int main(){    int m, h, temp;    scanf_s("%d", &m);    temp = m;    h = 1;    while (temp > 9) {        temp /= 10;        h *= 10;    }        while (h > 0) {        printf("%d ", m / h);        m %= h;        h /= 10;    }    return 0;}​
