Hornsea Project One海上風電場敷設首條輸出電纜

2020-10-29
| 科技

【電纜網訊】荷蘭海上解決方案供應商Tideway已為Ørsted擁有的1.2吉瓦Hornsea Project One海上風電場敷設了首條海上輸出電纜。

Hornsea Project One海上風電場敷設首條海上輸出電纜

Hornsea Project One風力發電場將位於約克郡海岸120公里外，將由三條高壓海底電力電纜從三個海上變電站向岸邊輸送電力。

Tideway通知稱，正在使用Ocean Yield公司的電纜敷設船“連接器”執行第一部分的電纜敷設。

“連接器”敷設船有兩個大轉盤，可容納6000噸和3000噸電纜。對於這項工作，Tideway公司為輸出電纜項目設計並安裝了一個定製的後甲板電纜敷設層，取代原有的敷設系統。

Tideway的工作範圍還包括電纜安裝、工程設計、除石、預挖溝、電纜接入變電站，交叉安裝、海上連接和電纜埋設。

Tideway項目經理Ken Swaegers表示：“這對我們來說是一項非常令人興奮的工作，同時也是開展海上風電最大型建設項目的絕佳機會。由於它的規模，我們有若干船隻在現場運行。並在夏季，我們自己的全新電纜敷設船Living Stone也會加入。Living Stone將配備創新的雙軌道電纜安裝系統，由兩條電纜高速公路組成，一條用於鋪設電纜，另一條可同時準備下一條電纜，顯著提高工作效率。”

Hornsea Project One董事Duncan Clark稱：“很高興Tideway能夠調動‘連接器’敷設船來滿足我們對這個項目的具體要求，同時期待Living Stone來到現場幫助建設這個非凡的風電場項目。 ”

2020年投入運營後，Hornsea Project One的容量將超過1吉瓦，並將能夠為超過100萬個家庭提供清潔電力。
