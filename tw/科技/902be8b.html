高爐幹法煤氣噴鹼塔脫氯效果評價的探討

2020-10-29
| 科技

【能源人都在看，點擊右上角加“關注”】

北極星大氣網訊:摘要：近年來,幹法除塵工藝得到普遍的應用。儘管在布袋除塵後增設了噴鹼塔,但仍出現煤氣與熱風管道先後出現腐蝕開裂的問題。研究提出了通過直接檢測噴鹼塔前後煤氣氯含量來評價噴鹼塔脫氯效果的設備和方法,比控制噴鹼液的pH值更直接、可靠。結果表明,幹法除塵-噴鹼塔工藝對煤氣中氯的脫除率為10%～70%,對下游工序的裝備存在腐蝕的風險。

高爐幹法除塵具有設備簡單、能耗低、操作靈活、結構緊湊和安全性好等優點， 己成為近年來鍊鐵技術發展的重要方向之一， 且發展迅速。新建或改造髙爐系統的除塵工藝普遍採用了幹法除塵技術鋼鐵企業採用幹法除塵後， 先後出現氯元素參與的設備腐蝕問題， 該類問題己在較多廠家出現並報道， 尤其是氣元素參與的腐蝕。為彌補幹法布袋除塵工藝在脫除腐蝕介質方面的不足， 均在布袋除塵器後增設了噴鹼塔以脫除腐蝕介質。生產上， 主要控制噴鹼塔循環水的PH 值呈鹼性， 然而， 噴鹼塔對腐蝕介質的脫除效果鮮有報道。

結合現場存在的氯腐蝕問題， 本文提出了用檢測噴鹼塔前後煤氣含氯量來推算脫氯效果的評價方法， 以及基於該方法對現場噴鹼塔脫氯效果進行了評價， 並指出了幹法布袋-噴鹼塔的煤氣處理工藝能獲得含塵量低的煤氣， 但其腐蝕介質含量仍較高， 仍存在煤氣用戶腐蝕的風險， 以便引起鍊鐵工作者對防腐工作引起重視。

1 高爐幹法煤氣攜帶走大部分腐蝕性氯元素

己有的生產實踐和己發表的文獻 ， 研究結果表明：

1 ) 氯元素主要來源於含鐵爐料、焦炭和煤粉， 帶入量分別約為63% ，31%和6% 。

2 ) 幹法除塵工藝中氯元素主要由爐渣、重力除塵灰、布袋灰和淨煤氣帶走， 支出量分別約為14% ，16% ，8% ， 62% 。

3 ) 溼法除塵工藝中氯離子主要由爐渣、重力除塵灰、汙泥、洗滌水和淨煤氣帶走， 支出量分別為10% ，12.9% ，1% ，76% ，0.1% 。

幹法除塵工藝條件下， 含鐵爐料是氯元素的主要帶入者， 煤氣是氯元素的帶出者， 兩者帶入與帶出的量基本相當， 分別為63%和62% 。溼法除塵工藝條件下， 煤氣帶出氣元素量僅為0.1% 。因此， 從煤氣所含腐蝕介質的含量來看， 溼法工藝所獲煤氣較幹法更“潔淨”。

也正是基於該原因， 在幹法布袋除塵後又增設了噴鹼塔， 通過噴鹼鹼液以吸收煤氣中的腐蝕介質。

2 增設噴鹼裝置後， 煤氣支管仍發生因腐蝕而壁厚度減薄的問題

Q 鋼熱風爐用煤氣經噴鹼塔脫除腐蝕介質， 但煤氣管道仍出現腐蝕問題。煤氣支管設計壁厚為8mm ， 運行4 年後出現壁厚減薄、漏氣現象， 其壁厚實測值見表1 。

由表1可見， Q銅熱風爐運行4年後各煤氣支管壁厚均有不同程度的減薄， 尤其是下部減薄嚴重， 最薄處厚度僅為3.36mm， 影響安全運行。

2.1 煤氣管道的腐蝕產物中發現氣元素的腐蝕介質

熱風爐煤氣支管內冷凝液、沉積物的物相分析結果見表2表。

由表2 可見， 熱風爐煤氣支管內主要存在以下3 種反應：

1 ) 由 煤氣帶來的氣離子參與的腐蝕過程。

2 ) 存在鋼材中的鐵元素的氧化反應。

3 ) 煤氣管道內存在的水蒸汽與溶有陰離子和陽離子的冷凝液形成的強電化學反應。

上述3 種反應的綜合作用使煤氣管壁厚減薄。

2.2 煤氣支管底部的冷凝液明顯酸性且導電率高

檢修期間從熱風爐煤氣支管中取得冷凝液， 檢測分析( 其pH值及主要腐蝕介質的分析)結果見表3 。

由表3 可見， 煤氣支管中的冷凝液具有酸性較強、氯離子含量高、導電性較強及陽離子含量較髙等特點。

綜上所述， 腐蝕性介質Cl-、( OH )-等與煤氣支管中的鐵元素反應是煤氣支管腐蝕變薄的原因之一; 煤氣中含有的水蒸汽結露冷凝析出液態水是促進煤氣支管腐蝕的關鍵因素; 較高含量的腐蝕介質溶於液態水而形成的強酸、髙含量的陰離子和陽離子與煤氣支管中的鐵元素形成的強電化學反應是煤氣支管腐蝕變薄的主要途徑。

然而， 生產現場對噴鹼塔運行管控若僅關注循環噴淋液的pH值與Cl-根離子， 來決定噴鹼液的加入量， 顯然是不夠的。若噴鹼塔脫除腐蝕介質的效率低， 噴淋液並未充分吸收煤氣中的酸根離子和氣離子， 而噴鹼液的pH 值與Cl-根離子仍然會低。因此， 有必要通過直接檢測煤氣中的氯含量來評價噴鹼塔的脫氯效果。

3 噴鹼塔的脫氯效率評價方法

該方法包括噴鹼塔前後煤氣含氯元素的測量裝置， 以及利用該測量裝置所獲得煤氣含氯量， 並推算噴鹼塔的脫除率， 具體所用設備和評價方法如下。

( 1 ) 如圖1所示， 為煤氣氯含量檢測裝置的的連接示意圖， 由溼式氣體流量計和依次串聯的3 個吸收瓶組成， 其中溼式氣體流量計要求額定流量為0.5m3 /h， 吸收瓶容積要求為0.5-1L。

( 2 ) 溼式氣體流量計用水為去離子水， 單個吸收瓶中放入200-300m l 鹼性溶液， 要求鹼性溶液由去離子水和純NaOH 化學試劑配製而成， 濃度控制在1.5mol/L 。

( 3 ) 實驗過程中， 要注意煤氣洩漏問題， 實驗人員要站在上風口並佩戴煤氣報警器，流量計出口排氣管要放於下風口處。

( 4 ) 實驗過程中通過煤氣閥開關程度控制合理的煤氣流量， 流量控制在3-5 L/min ;

( 5 ) 為了保證實驗數據可靠性， 要求每個取樣位置測量至少3組數據， 且每組數據測量時間控制在30min 左右。

( 6 ) 計算煤氣中Cl含量

利用上述脫氯效率評價方法， 對現場噴鹼塔前後的煤氣含氣量進行測定， 所獲噴鹼塔前後煤氣氯元素含量以及脫除率結果如下表4 所列。

噴鹼塔現場檢測煤氣含氯量來看， 噴鹼塔的脫氯效果並不理想， 最高脫氯率不足70% ，最低脫氯率不足10% ， 脫氣效果堪憂。噴鹼塔後煤氣含氯量在230-700 mg/m3 之間， 對煤氣管道和用戶產生腐蝕並危害設備的安全工作也就不足為奇了。

考慮到對裝備腐蝕影響其壽命和安全運行等問題， 如下幾個與腐蝕相關的問題需要鍊鐵工作者重視。

1 ) 近年來， 在非主流礦的應用中， 硫含量和氣含量對鍊鐵成本和對裝備腐蝕的負面影響並未得到充分考慮或受重視程度不夠。

2 ) 為 降低生產成本， 煤氣中腐蝕介質含量或入爐腐蝕介質負荷水平與加鹼量間仍未形成相關技術標準。

3 ) 管 道內壁防腐塗料的使用效果受工作溫度的影響較大， 髙溫下防腐塗料的防腐效果失效。

4 ) 為維持燒結礦入爐前檢測低溫還原粉化指標RDI處理較高水平， 而在燒結運輸皮帶上噴灑的氣化鈣的作用， 有必要重新審視?

5) 入爐原燃料的硫含量和氯含量應納入原燃料性價比評價的體系中。

5 結語

( 1 ) 與溼法除塵技術相比， 高爐幹法除塵技術獲得了低成本運行條件下煤氣含塵淨化的效果， 但是幹法除塵技術並沒有承接溼法除塵技術髙效脫除煤氣帶入腐蝕介質的功能。

( 2 ) 研宄提出了通過直接檢測噴鹼塔前後煤氣含氯量， 來評價噴鹼塔脫氯效果的設備和方法， 較控制噴鹼液的pH值更直接、可靠。

( 3 ) 利用該方法檢測結果表明： 幹法除塵-噴鹼塔工藝對煤氣中氯的脫除率在1 0%-70%之間， 仍對下游工序的裝備存在腐蝕的風險。
