機電安裝工程安全生產標準化施工彙總，值得學習借鑑！

2020-11-14
| 科技

1 加工車間及庫房

1. 1 加工車間及庫房臨時用電線路與照明

1.加工車間及庫房電纜敷設統一使用橋架，橋架分為垂直橋架和懸吊式水平橋架。照明線路橋架距離地面≥2500mm。

2. 垂直橋架固定方式:採用螺栓將橋架固定於牆面或圍擋上，間隔≤1000mm。懸吊式水平橋架固定方式:採用L25x3mm的型鋼托架和φ10mm全牙絲桿，間隔≤1500mm。

3. 開關箱進出線使用pvc管或鍍錚鋼管進行保護，電纜末端伸出pvc管或鍍錚鋼管500m血，使用波紋管進行保護。

4. 線管固定方式:垂直橋架採用馬鞍卡箍用螺栓和圍擋進行固定，懸吊式水平橋架採用吊杆將懸吊式線管進行固定。

5. 配電箱和設備要做好重複接地，採用4x40mm扁鋼及φ12mm圓鋼進行環繞接地，焊接採用搭接焊。

6. 加工車間照明變壓開關箱單獨設置，每8m².設置至少一盞36V18W的LED燈管。

1.2 加工車間通風排煙

1.樓層(地下室)內進行集中噴塗、焊接作業的場所， 空氣質量較差，提倡設置通風排煙設施。

2. 加工車間通風排煙系統施工前必須編制專項方案，經審核審批後實施。

3. 根據加工車間面積和所在樓層層高測算換氣風量和次數，選定風機型號和風管尺寸。風機及風管高度不得低於2800mm。

4. 風機必須設置按鈕式開關，按鈕開關中包含急停開關。

5. 風機為軸流風機，整個系統為負壓系統，排除加工車間內煙氣。風口處設置調節閥， 根據現場加工情況設置排風風量。

2 風管的加工與製作

1.加工區域使用高度為1200mm的網片式防護欄杆進行隔離。加工區域可按"U" 型或"一"字型進行佈置。

2. 流水線與各獨立設備應有明顯的作業區域標識加以區分，並有寬度不小於1000mm的安全通道。

3. 機械轉動部位及切割作業部位應加防護罩或使用圍擋進行隔離，轉動機械應使用帶急停開關的按鈕式開關。

4. 龍門架使用的腳輪應有可靠制動裝置。

3 風管與管道的安裝

1.風管組對的作業場所應鋪墊柔性材料，使用液壓鉚釘槍進行錨固，以達到成品保護、減小施工噪聲的目的。風管組對使用的手持電動工具應使用蓄電式III類手持電動工具，風管組對作業優先在加工車間完成。

2. 吊運豎向管道時，應編制專項施工方案，在管道不能焊置吊耳的情況下，可採用下圖所示的吊裝方式。

3. 風管或管道安裝作業時，所使用的電(手)動葫蘆、捲揚機、堆高車、登高車等施工機具，應符合本圖冊施工機具的有關規定。所使用的操作平臺應符合本圖冊高處作業的有關規定。

4 管道安裝階段防護

1.管道安裝時，原有安全防護應隨施工進度逐層拆除，嚴禁一次拆除。

2. 安裝階段安全防護應在作業面安全防護移交後，安裝開始前製作，管道安裝階段，做到隨安裝隨防護。

3. 安裝階段，應根據管道形狀用厚度不小於15mm的木模板對管道井口進行防護，防護效果如下圖所示。

4. 管道安裝結束後，應將安全防護移交給負責管道井封堵的單位。

5 豎向大直徑電纜敷設

1.堅向電纜敷設前，必須編制安全專項技術措施。

2. 電纜敷設必須設置專用的牽引裝置，牽引裝置的安全係數不得小於5 ，嚴禁使用人力牽引。

3. 電纜轉向點需有防摩擦滑輪，並每隔30m設置一組防晃滑輪。

4. 鋼絲繩牽引法:使用旋轉頭網套連接器(固定在電纜始端)和覆式側拉型中間網套連接器( 調整電纜位置) 作為主吊具; 每隔50m設置一副覆式側拉型中間網套連接器作為輔助吊具，分擔吊重。

5. 阻尼緩衝器法:採用阻尼緩衝器制約重力加速度， 把電纜由上往下垂直敷設，滑輪緩衝器在安裝前應進行阻尼試驗。

6. 電纜敷設完畢後，立即將電纜綁紮固定，檢查確認固定裝置可靠後，吊裝設備方可卸載。

6 設備整體吊裝安裝法

1.設備安裝前，必須編制安全專項施工方案，報上級單位審批後按方案實施。

2. 若設備無專用吊環或吊點時，應根據方案佈置吊耳，並進行外觀檢測。

3. 設備整體吊裝作業前應對作業人員進行安全技術交底，並對吊裝使用的設備及吊索具進行檢查。

4. 特種作業人員必須持有效特種作業操作證。

5. 設備整體吊裝作業時應設置警戒區域， 並由專人進行旁站監督。

6. 設備底部掛設溜繩，輔助設備就位。

7 聯合調試安全管理

1.項目部應成立聯合調試工作小組，依據項目聯合調試方案對相關責任人進行安全技術交底。

2. 調試前對調試現場、工具進行檢查，現場各點之間應保持實時聯絡，調試過程參數的變動有專人監控，並及時反饋。

3. 調試時要採取必要的防護措施，調試人員需穿戴安全帽、絕緣手套及絕緣膠鞋等。對所有帶電設備、線路設置明顯的"閃電符"及"有電危險"標示。

4. 停送電必須是"一人指揮"，且應分區、分段逐步送電。只有送電區域反饋正常後，再進行下一次送電。調試端維護時應"先驗電，後操作"。

5. 調試過程中，無關人員不得在危險區域逗留。

6. 調試結束後，對不需要運行的系統採取斷電、降壓、洩水等措施。
