避雷線的作用是什麼?

答：其作用如下：

(1)減少雷電直接擊於導線的機會。

(2)避雷線一般直接接地，它依靠低的接地電阻洩導雷電流，以降低雷擊過電壓。

(3)避雷線對導線的屏蔽及導線、避雷線間的耦合作用，降低雷擊過電壓。

(4)在導線斷線情況下，避雷線對杆塔起一定的支持作用。

(5)絕緣避雷線有些還用於通信，有時也用於融冰。

何為線路絕緣的洩漏比距?影響洩漏比距大小的因素有哪些?

答：(1)洩漏比距是指平均每千伏線電壓應具備的絕緣子最少洩漏距離值。

(2)影響洩漏比距大小的因素有地區汙穢等級及系統中性點的接地方式。

何謂輸電線路的耐雷水平?線路耐雷水平與哪些因素有關?

答：(1)線路耐雷水平是指不致於引起線路絕緣閃絡的最大雷電流幅值。

(2)影響因素：

①絕緣子串50%的衝擊放電電壓。

②耦合係數。

③接地電阻大小。

④避雷線的分流係數。

⑤杆塔高度。

⑥導線平均懸掛高度。

說出杆塔在各種運行情況時的荷載組合計算條件？

（3）最低氣溫、無風、無冰、未斷線。

採用倒落式人字抱杆整體起立分段混凝土杆排杆方法是怎樣的?

答：其排杆方法如下：

(1)排杆場地應先平整，並在每段杆下墊上道木。

(2)杆根距杆坑中心0.5m，杆身沿線路方向，轉角杆杆身應與內側角二分線成垂直排列。

(3)杆段鋼圈對口間隙隨鋼圈厚度而變，一般為幾毫米。

(4)整基杆塔焊接前各段應平直，並處於同一直線上。

(5)被排用的杆塔不得有明顯不合要求的缺陷存在。

降低線路跳閘率的措施有哪些？

答：降低線路跳閘率的一般措施有：

（1）提高線路的絕緣水平和機械強度。

（2）線路走廊一定要符合要求。

（3）改進線路杆型。

（4）在線路進出線、母線支柱瓷瓶上套裝塑料管帶等，以增大閃絡距離。

（5）加裝線路重合閘或重合式熔斷器以減少瞬間故障的跳閘停電事故。

（6）改進配電變壓器臺的設計。

接地裝置的安裝有何要求？

答：（1）接地裝置應按施工設計要求進行，若無設計時，可根據經驗施工。

（2）接地裝置埋入地中的深度不小於600mm。

（3）鐵帶必須立放，角鋼和鋼管應垂直打下並防止晃動。

（4）接地裝置的連接必須嚴密可靠，地下部分的連接必須焊接，焊前應清潔焊口，其焊接長度圓鋼為直徑的6倍，其周圏施焊。扁鋼為寬度的2倍並四面施焊。地下部分與地上部分的接頭可用藥包焊或鋼並溝線夾連接，其連接點居民區應離地面1.5m以下。

答：換位的方法是，可在每條線路上進行循環換位，即讓每一相導線在線路的總長中所處位置的距離相等。也可採用變換各回路相序的方法進行換位。

試述高壓驗電器的作用及使用要求?

答：驗電器的作用是驗證電氣設備或線路等是否有電壓。其實擁有以下要求：

(1)驗電器的額定電壓必須與被驗設備的電壓等級相適應。

(2)驗電器使用前必須在帶電設備上試驗，以檢查驗電器是否完好。

(3)對必須接地的指示驗電器在末端接地。

(4)進行驗電時必須帶絕緣手套，並設立監護人。

對短路接地線的構成有何規定？

答：（1）接地線應有接地和短路導線構成的成套接地線。

（2）成套接地線必須用多股軟銅線組成，其截面不得小於25平方毫米。

（3）如利用鐵塔接地時，允許每相個別接地，但鐵塔與接地線連接部分應清除油漆，接觸良好。

（4）嚴禁使用其他導線作接地線和短路線。

簡述繩索一般應力、最大使用應力及最大允許應力的區別與聯繫?

答：(1)區別：

①一般應力是指繩索受力時單位橫截面的內力。

②最大使用應力是指繩索在工作過程中出現的最大應力值。

③最大允許應力是指繩索按機械強度要求允許達到的最大應力值。

(2)聯繫：

①最大使用應力是應力在某一種特殊情況下的值。

②一般應力受最大允許應力的限制。

知識點 : 工器具知識

儀用互感器正確使用應注意哪些事項?

答：應注意以下事項：

(1)正確接線：電壓互感器應遵守“並列”的原則，而電流互感器應遵守“串聯”的原則。

(2)互感器二次繞組一端可靠接地，以防止一次側的高電壓串人二次側，危及人身和設備安全。

(3)接在互感器二次側的負載功率不應超過互感器的額定容量，以減小互感器的誤差。

(4)電流互感器的二次繞組在任何情況下都不允許開路。

(5)電壓互感器的一、二次側都應有保護短路的熔斷器。

鋼絲繩直徑與滑輪輪直徑如何配合？

答：纏繞鋼絲繩的滑輪，磨芯或滾筒，除特殊規定外，其最小直徑與鋼絲繩直徑之比應滿足下列要求：

（1）對起重滑輪的槽底應不小於10-11倍。

（2）對人推絞磨和機動絞磨的磨芯應不小於9-10倍。

鋼絲繩在什麼情況下應報廢或截除?

答：鋼絲繩有下列情況之一者應報廢或截除：

(1)在一個節距內(每股鋼絲繩繞捻一週的長度)斷絲根數超過有關規定的。

(5)斷絲數量雖不多，但斷絲增加很快者。

對安規的學習和考試有何規定？

答：（1）電力線路工作人員對本規程應每年考試一次。因故間斷電氣工作連續三個月以上者，必須重新溫習本規程，並經考試合格後，方能恢復工作。

（2）參加帶電作業人員，應經專門培訓，並經考試合格、領導批准後，方能參加工作。

（3）新參加電氣工作的人員、實習人員和臨時參加勞動的人員(幹部、臨時工等)，必須經過安全知識教育後，方可下現場隨同參加指定的工作，但不得單獨工作。

（4）對外單位派來支援的電氣工作人員，工作前應介紹現場電氣設備接線情況和有關安全措施。

答：應符合以下規定：

(1)保護層厚度允許偏差：－5mm。

(2)立柱及各底座斷面尺寸允許偏差：－1%。

(3)同組地腳螺栓中心對立柱中心偏移：10mm。

試題編號 : 138

知識點 : 規程和規範

根據安規規定，工作終結後，工作負責人應如何向工作許可人報告？

答：工作終結後，工作負責人應報告工作許可人，報告方法如下：

（1）從工作地點回來後，親自報告。

（2）用電話報告並經複誦無誤。電話報告又可分為直接電話報告或經由中間變電所轉達兩種。經中間變電所轉達報告，中間變電所值班員應將命令全文記入操作記錄簿，並向工作負責人直接傳達。電話傳達時，上述三方必須認真記錄，清楚明確，並複誦核對無誤。

試題編號 : 139

知識點 : 規程和規範

進行線路作業前，應作好哪些停電措施？

答：進行線路作業前，應作好下列停電措施：

（1）斷開發電廠、變電所(包括用戶)線路斷路器(開關)和隔離開關(刀閘)。

（2）斷開需要工作班操作的線路各端斷路器(開關)、隔離開關(刀閘)和熔斷器(保險)。

（3）斷開危及該線路停電作業，且不能採取安全措施的交叉跨越、平行和同杆線路的斷路器(開關)和隔離開關(刀閘)。

（4）斷開有可能返回低壓電源的斷路器(開關)和隔離開關(刀閘)。

試題編號 : 140

知識點 : 規程和規範

防止誤操作的“五防”內容是什麼?

答：(1)防止誤拉、誤合斷路器。

(2)防止帶負荷誤拉、誤合隔離開關。

(3)防止帶電合接地開關。

(4)防止帶接地線合閘。

(5)防止誤人帶電間隔。



試題編號 : 141

知識點 : 規程和規範

安規對單人巡線、夜間巡線、事故巡線工作有何規定?

答：(1)單人巡線時，禁止攀登電杆或鐵塔。

(2)夜間巡線時應沿線路外側進行。(3)事故巡線時應始終認為線路帶電，即使明知該線路已停電，亦應認為線路隨時有恢復送電的可能。

試題編號 : 142

知識點 : 規程和規範

杆塔上螺栓的穿嚮應符合哪些規定?

答：應符合如下規定：

(1)對立體結構：

①水平方向由內向外。

②垂直方向由下向上。

(2)對平面結構：

①順線路方向由送電側穿入或按統一方向穿入。

②橫線路方向兩側由內向外，中間由左向右(面向受電側)或按統一方向。

③垂直方向由下向上。

試題編號 : 143

知識點 : 規程和規範

安規對驗電有何規定？

答：（1）在停電線路工作地段裝接地線前，要先驗電，驗明線路確無電壓。驗電要用合格的相應電壓等級的專用驗電器。330kV及以上的線路，在沒有相應電壓等級的專用驗電器的情況下，可用合格的絕緣杆或專用的絕緣繩驗電。驗電時，絕緣棒的驗電部分應逐漸接近導線，聽其有無放電聲。確定線路是否確無電壓。驗電時，應戴絕緣手套，並有專人監護。

（2）線路的驗電應逐相進行。檢修聯絡用的斷路器(開關)或隔離開關(刀閘)時，應在其兩側驗電。對同杆塔架設的多層電力線路進行驗電時，先驗低壓、後驗高壓，先驗下層、後驗上層。

試題編號 : 144

知識點 : 規程和規範

上杆作業前應做哪些檢查？

答：（1）上木杆前，應先檢查杆根是否牢固。新立電杆在杆基未完全牢固以前，嚴禁攀登。遇有沖刷、起土、上拔的電杆，應先培土加固或支好杆架、或打臨時拉繩後，再行上杆。凡鬆動導、地線和拉線的電杆，應先檢查杆根，並打好臨時拉線或支好架杆後，再行上杆。

（2）上杆前，應先檢查登杆工具，如腳釦、升降板安全帶、梯子等是否完整牢靠。

（3）攀登杆塔腳釘時，應檢查腳釘是否牢固。

試題編號 : 145

知識點 : 規程和規範

炸藥和雷管的運輸、攜帶和存放有何規定？

答：（1）炸藥和雷管應分別運輸、攜帶和存放，嚴禁和易燃物放在一起，並應有專人保管。運輸中雷管應有防震措施。攜帶雷管時，必須將引線短路。電雷管與電池不得由同一人攜帶。雷雨天不應攜帶電雷管，並應停止爆破作業。在強電場附近不得使用電雷管。

（2）如在車輛不足的情況下，允許同車攜帶少量炸藥(不超過10kg)和雷管(不超過20個)。攜帶雷管人員應坐在駕駛室內，車上炸藥應有專人管理。

試題編號 : 146

知識點 : 規程和規範

帶電作業對天氣條件有何要求？

答：（1）帶電作業應在良好天氣下進行。如遇雷、雨、雪、霧不得進行帶電作業，風力大於5級時，一般不宜進行帶電作業。

（2）在特殊情況下，必須在惡劣天氣進行帶電搶修時，應組織有關人員充分討論並採取必要的安全措施，經廠(局)主管生產領導(總工程師)批准後方可進行。

試題編號 : 147

知識點 : 規程和規範

帶電作業時在什麼情況下應停用重合閘，並不得強送電？

答：帶電作業有下列情況之一者應停用重合閘，並不得強送電：

（1）中性點有效接地的系統中有可能引起單相接地的作業。

（2）中性點非有效接地的系統中有可能引起相間短路的作業。

（3）工作票簽發人或工作負責人認為需要停用重合閘的作業。

試題編號 : 148

知識點 : 規程和規範

簡述振搗對現澆混凝土的作用。過長時間對現澆混凝土進行機械振搗有何危害?

答：(1)對現澆混凝土進行振搗的作用在於增強混凝土的密實性、減小水灰比，以提高混凝土的強度。

(2)若過長時間進行振搗，易使混凝土內部出現分層的離析現象，反而影響混凝土的強度。

試題編號 : 149

知識點 : 規程和規範

杆塔及拉線中間驗收檢查項目包括哪些?

答：包括以下項目：

(1)混凝土電杆焊接後焊接彎曲及焊口焊接質量。

(2)混凝土電杆的根開偏差、邁步及整基對中心樁的位移。

(3)結構傾斜。

(4)雙立柱杆塔橫擔與主柱連接處的高差及立柱彎曲。

(5)各部件規格及組裝質量。

(6)螺栓緊固程度、穿入方向、打衝等。

(7)拉線方位、安裝質量及初應力情況。

(8)NUT線夾螺栓、花籃螺栓的可調範圍。

(9)保護帽澆築情況。

(10)回填土情況。

試題編號 : 150

知識點 : 規程和規範

根據南方電網公司《電氣工作票技術標準》規定，工作終結的報告應包括哪些內容？

答：工作終結的報告應簡明扼要，包括工作負責人姓名，工作地段，工作任務完成情況，設備及網絡變動情況，工作地點所做安全措施已全部拆除，線路上已無本工作班人員，工作地段具備送電條件。
