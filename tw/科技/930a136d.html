python read,readline,readlines吐血整理（一）

python 在open文件後，通常使用read，readline，readlines等函數讀取文件內容，它們的使用你掌握了嗎？讓我們一起來學習一下。

read():

無參數時，一次性讀取整個文件的內容，將讀取的字節（包括換行符等）保存到字符串變量中。

read(N)就是讀取N個字節。

readline():

無參數時，每次讀取一整行（包括行結束符）的內容，將讀取的字節（包括換行符等）保存到字符串變量中。

readline(N)：當N<當前行的字節數時，讀取N個字節，此時的效果同read(N)相同；當N>當前行的字節數時，只讀取當前行的全部內容，不再往下讀取。

由下圖可知，讀取20個字節時，read()與readline()讀出的內容相同，讀取80個字節時，read()與readline()讀出的內容不同。

readlines()：

無參數時，一次性讀取整個文件，將文件內容分析成一個行的列表：就是每行的內容（包括換行符等）作為一個字符串，不同的行轉換成的字符串集合成一個列表。

readlines(N) 的參數N官方定義如下

翻譯過來就是："hint確定要讀的行數：如果行的總大小（以字節計算）超過了hint的值，就不再繼續讀取"。

有時候大家不是很清楚到底指的是什麼意思。有些人說，為什麼我用readlines(2)沒有讀取第二行的內容呢？

這裡hint指的不是行數，而指的是需要讀取的字節數的總大小。假設hint=N，readlines()函數會確定第N個字節的數據在哪一行，然後一直讀取到該字節的數據所在的當前行為止。
