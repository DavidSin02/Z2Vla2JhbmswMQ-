VBA簡單入門09：End屬性獲取最後的單元格信息

2020-11-14
| 科技

1、End屬性是單元格對象Range的屬性，它返回工作表中（行列）的最後一個非空單元格，並常常用來構建動態的單元格數據範圍。

如，工作表中只有A1:C13是數據區域。此時該工作表的最後一個非空單元格的相關信息可以用End屬性來獲取。

A列最後一個非空單元格地址：返回 : "$A$13"

a = Range("a1048576").End(xlUp).Address

Range("a1048576")表示單元格 A1048576,而End是屬於Range單元格對象的屬性。

可以說白一點：Range("a1048576").End(xlUp)就是以單元格 A1048576 為基點，從 【A1048575 開始】 (A1048576即使非空也不在查找範圍內)在同一列（A列）往上查找，找到第一個非空單元格就停止，並返回該單元格。所以上面的代碼，返回的是A13單元格。

如將代碼改為：Range("a20").End(xlUp).Address，返回同樣結果。

原因只不過是，一個從A1048575開始，一個從A19開始，雖然基點不同，但往上查找到的第一個非空單元格都是A13。

2、既然可以往上查找，那就同樣可以往下、往左、往右查找。

在寫代碼的時候，就有提示：

xlup上； xldown下； xltoleft左； xltoright右

它們的相同點是，作為基點的單元格都不算。

在數據比較規範的時候，無論是xlup還是xldown，都能得出同樣的結果。

假如，A16單元格非空，那麼上圖兩句代碼返回的結果將不一樣。

xlup：向上找，找到第一個非空單元格就停止。

xldown：向下找，遇到空單元格就停止。

xltoleft和xltoright同理。

下列代碼返回規範數據情況下（只A1:C13有數據）當前活動工作表的數據範圍的相關信息。

假如，當前的數據表格需每天更新數據（添加行），該動態單元格範圍一般可以這樣表示：

Range("a1:c" & 數值變量) 或 Range("a1",單元格變量)，如下面代碼都可以表示單元格A1:C13

行數 = Range("a1048576").End(xlUp).Row

Range("a1:c" & 行數).Select

或

Range("a1", Range("c1048576").End(xlUp)).Select

當數據自動增加的時候，單元格範圍會自動增加。如新增3行數據。請看gif演示。

4、Excel版本問題造成的錯誤。

Excel03版只有65536行，Excel07版後有1048576行，所以Range("a1048576")在03版中會出錯。

所以通用的寫法是：Cells(Rows.Count, 列)或Cells(行, Columns.Count)來代替Rang對象。

下圖中，Rows.Count在03版是65536，在07版或以上是1048576，再也不擔心版本問題而出錯。

【小練習】提取唯一值（去重複）

要求：將A列的英雄姓名去除重複值，提取唯一值到D列。

此題目可以用菜單欄的刪除重複值操作(或字典）來實現更簡單，但這裡使用的是循環的方法。

Sub A()

Dim R As Long

Dim R1 As Long

Dim B As Boolean

Dim k As Long

R = Cells(Rows.Count, 1).End(xlUp).Row 'A列最後非空行行號

Range("d2:d10000") = "" '先將D列數據清空

[d2] = [a2] '第一個數據必然是唯一的，放入D2單元格

k = 2

For i = 3 To R '遍歷英雄姓名(第2個數據起)

B = False '此值改變用來標記是否有重複

R1 = Cells(Rows.Count, 4).End(xlUp).Row 'D列最後非空行行號

'............................................................................................................................................

'//注意R1數值的變化，隨著提取的唯一姓名個數增加而增加

For j = 2 To R1 'A列姓名每一個都遍歷D列的所有英雄姓名，查看是否在D列存在（重複）

If Cells(i, 1) = Cells(j, 4) Then '如果A列姓名在D列能找到，表示重複

B = True '布爾變量為TRUE表示重複

Exit For '確定重複後退出循環

End If

Next

'.............................................................................................................................................

'//判斷是否有重複，並輸出數據

If B = False Then '判斷B的值，如False，則是唯一值

k = k + 1 '計數

Cells(k, 4) = Cells(i, 1) '數據放到D列

End If

Next

End Sub
