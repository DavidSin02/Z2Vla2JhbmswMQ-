Apache Druid 0.19.0 發佈，實時分析數據庫

2020-11-14
| 科技

Apache Druid 0.19.0 發佈了。Druid 是一個分佈式的、支持實時多維 OLAP 分析的數據處理系統。它既支持高速的數據實時攝入處理，也支持實時且靈活的多維數據分析查詢。因此 Druid 最常用的場景就是大數據背景下、靈活快速的多維 OLAP 分析。 另外，Druid 還有一個關鍵的特點：它支持根據時間戳對數據進行預聚合攝入和聚合分析，因此也有用戶經常在有時序數據處理分析的場景中用到它。

此版本包含約 200 項新特性、bug 修復、性能增強、文檔改進和附加測試。主要的新特性和改進包括：

  • 默認啟用 GroupBy 和 Timeseries 矢量化查詢引擎
  • 支持 Apache Avro 對象容器文件的 Druid 本機批處理支持
  • 更新了對 SQL 數據庫的 Druid 本機批處理支持
  • 基於 Apache Ranger 的授權
  • 阿里巴巴對象存儲服務支持
  • Google Compute Engine 提取 worker 自動縮放

更新說明：https://github.com/apache/druid/releases/tag/druid-0.19.0
