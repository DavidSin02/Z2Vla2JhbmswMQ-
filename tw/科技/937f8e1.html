Python｜查找列表元素（線性查找法或二分查找法）

2020-10-29
| 科技

list類模塊已經實現有index方法來查找並返回匹配列表中某個元素的下標，也支持成員運算符in和not in來判斷一個元素是否在列表中。

lst = [2,9,5,4,8,1,6]

print(lst.index(8)) # output：4

print(8 not in lst) # output: False

當然，也可以自定義元素查找並返回元素下標，以下闡述的的是線性查找法和二分查找法。

1 線性查找法

線性查找法是指按順序挨個將需要查找的key與列表中的每一個元素比較，直到匹配到邏輯相等為止。

# The function for finding a key in the list

def linearSearch(lst, key):

....for i in range(len(lst)):

........if key == lst[i]:

............return i

....return -1

lst = [2,9,5,4,8,1,6]

print(linearSearch(lst, 8)) # putput:4

2 二分查找法

二分查找法的思路要複雜一點。對一個已經排序的列表，不斷將一個區間分為兩個子區間（包括匹配子區間），將需要查找的關鍵字與區間的中間元素比較，直到匹配到邏輯相等為止。

用一個簡單的實例手動模擬實際操作的過程：

lst = [2,4,7,10,11,45,50,59,60,66,69,70,79]

key = 11

附代碼：

# Use binary search to find the key in the list

def binarySearch(lst, key):

....low = 0.................... # 列表中當前要查找的第一個下標

....high = len(lst) - 1........ # 列表中當前要查找的最後第一個下標

....while high >= low:

........mid = (low + high) // 2 # 中間元素下標

........if key < lst[mid]:.... # 滿足條件,則定義前半段為子區間

............high = mid - 1

........elif key == lst[mid]: # key與中間元素相等,則返回中間元素的下標值

............return mid

........else: .... # 滿足條件,則定義後半段為子區間

............low = mid + 1

....return low - 1 .... # Now high < low, key not found

lst = [2,4,7,10,11,45,50,59,60,66,69,70,79]

print(binarySearch(lst, 11))....# output: 4
