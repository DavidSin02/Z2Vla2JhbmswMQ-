學習NLP的第7天——基於HanLP實現的停用詞過濾

2020-10-29
| 科技

這裡主要記錄我在學習過程中整理的知識、調試的代碼和心得理解，以供其他學習的朋友參考。

停用詞，是指文本中沒有多少實際意義的詞語，包括助詞、連詞、副詞、語氣詞等詞性，句子中去掉了停用詞並不影響語句的理解。

停用詞視具體情況在英雄聯盟的遊戲內聊天框中，敏感詞和低俗詞也都被視作了停用詞。

停用詞過濾是語言文本預處理的一個重要步驟，有兩種常用的情景：

  • 將分詞結果中的停用詞剔除
  • 直接將文本中的停用詞替換為*或移除

下面，我們使用HanLP提供的雙數組字典樹來實現這個功能。

首先，我們從HanLP實現停用詞典的加載。

def load_from_file(path):    """    從詞典文件加載DoubleArrayTrie    :param path: 詞典路徑    :return: 雙數組trie樹    """    map = JClass('java.util.TreeMap')()  # 創建TreeMap實例    with open(path) as src:        for word in src:            word = word.strip()  # 去掉Python讀入的\n            map[word] = word    return JClass('com.hankcs.hanlp.collection.trie.DoubleArrayTrie')(map)

停用詞詞典的格式為一個詞一行，每行之間用換行符\n分隔。

提出分詞結果中的停用詞

針對分詞結果，遍歷每個詞語，若它存在於停用詞字典樹中，則刪除該詞。這裡使用列表生成式實現。

def remove_stopwords_termlist(termlist, trie):    return [term.word for term in termlist if not trie.containsKey(term.word)]    if __name__ == '__main__':    HanLP.Config.ShowTermNature = False    trie = load_from_file(HanLP.Config.CoreStopWordDictionaryPath)    text = "停用詞的意義相對而言無關緊要的詞吧"    segment = DoubleArrayTrieSegment()    termlist = segment.seg(text)    print("分詞結果:", termlist)    print("分詞結果去除停用詞:", remove_stopwords_termlist(termlist, trie))

運行結果

分詞結果: [停用, 詞, 的, 意義, 相對而言, 無關緊要, 的, 詞, 吧]分詞結果去除停用詞: ['停用', '詞', '意義', '無關緊要', '詞']

直接處理文本中的停用詞

在過濾敏感詞時，需要將敏感詞替換為特定的字符串，例如**，這時使用不分詞，直接替換的方法效率更高。這裡使用HanLP的雙數組字典樹的getLongestSearcher方法實現。

def replace_stropwords_text(text, replacement, trie):    searcher = trie.getLongestSearcher(JString(text), 0)    offset = 0    result = ''    while searcher.next():        begin = searcher.begin        end = begin + searcher.length        if begin > offset:            result += text[offset: begin]        result += replacement        offset = end    if offset < len(text):        result += text[offset:]    return result    if __name__ == '__main__':    text = "停用詞的意義相對而言無關緊要的詞吧"    trie = load_from_words("的", "相對而言", "吧")    print("不分詞去掉停用詞:", replace_stropwords_text(text, "**", trie))

運行結果

不分詞去掉停用詞: 停用詞**意義**無關緊要**詞**
