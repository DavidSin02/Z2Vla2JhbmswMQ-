獨家祕訣：用YAKINDU StatechartTools進行Java開發的圖形狀態機

2020-10-29
| 科技

全文共2602字，預計學習時長8分鐘

來源：Pexels

你有沒有過看著自己的代碼，意識到它實際上是一個正嘗試使用的狀態機？

通常你會開始使用switch-case語句或採用狀態模式。這種方法對於較大的狀態機來說難以快速讀取和維護。


YAKINDU Statechart Tools附帶了一個圖形編輯器，可以為狀態圖建模並自動生成相應的Java代碼。人們還可以在圖形模型中直接使用Java代碼。本文將展示YAKINDU Statechart Tools，並演示操作步驟。


YAKINDUStatechart Tools中的深度Java集成可供人們直接訪問狀態圖中的Java APIs，因此有助於在Java開發過程中促進狀態機建模的集成！深度Java集成包含以下功能：


· 可在狀態圖中導入Java的類(classes)、接口(interfaces) 和枚舉 (enums) ，並將它們用於狀態圖變量。

· 可直接從狀態圖訪問變量、方法等所有公共 (public) 和靜態 (static) 的成員，這樣可以避免許多樣板代碼的出現。

· 狀態圖中使用的Java操作方法可在模擬期間調用。這避免了模擬過程中對返回值進行繁瑣的模擬操作。


以下視頻包含一千多字，錄製這個簡短的視頻是希望讀者能對本文描述的功能有一個印象。請享受觀看的樂趣，如果看完後仍存好奇，請閱讀下文。


對狀態圖變量使用Java類型


在YAKINDU中，狀態圖有一個域(domain)。為了使用本文中提到的功能，須採用Java域。可以在狀態圖的屬性中選擇Java域，也可以在創建新的狀態圖時選擇。



Java域狀態圖需使用面向Java的類型系統。可以在變量聲明中使用本機Java 類型（如int）或其包裝器類型（如Java.lang.Integer）。



此外，還可以使用import語句導入Java類、接口或枚舉。內容助手[Ctrl]+[Space] 列出了項目類路徑上的所有類。可以從JRE中導入類，也可以從自己的運行環境中導入。



在狀態圖中使用Javatypes還意味著可以使用泛型，就像是在Java代碼中使用它們一樣。



面向Java的類型系統能夠推斷泛型信息，並在類型不匹配時識別錯誤。這樣便可在生成代碼前避免編譯錯誤。



實例化與Java中的實例略有不同。構造函數被轉換為名為 new()的靜態工廠方法，而不是將關鍵詞new導入狀態圖語言。


訪問公共成員


一旦有了Java type的變量，就可以訪問方法、變量等公共成員。同樣，內容助手提供了所有可訪問成員的概述，其中包含所有公共實例和靜態類成員。



在模擬中調用Java APIs


YAKINDUStatechart Tools提供了一個內置模擬工具，用於在生成代碼前驗證系統行為。可以通過Run-> Run As -> Statechart Simulation 的步驟在上下文菜單中啟動模擬。


至於Java域狀態圖，該模擬直接調用了引用的Java方法。可在模擬視圖中查看效果，還可以在該視圖中手動更改可訪問變量的值，以查看系統反應。還可以在狀態圖和轉換處上設置斷點，就像調試Java代碼一樣。



代碼生成及客戶代碼集成


作者已經調整了YAKINDUStatechart Tools標準版附帶的Java代碼生成器，以便為Java域狀態圖生成正確的Java代碼。要在Java應用程序中使用生成的狀態機代碼，需執行以下基本步驟：

private void setup() {

// 1. Initialize the state machine

IPlayerStatemachine statemachine = new PlayerStatemachine();

statemachine.init();

// 2. Raise in-events on button clicks

playBtn.setOnAction((e) -> {

statemachine.getSCInterface().raisePlayPressed();

});

stopBtn.setOnAction((e) -> {

statemachine.getSCInterface().raiseStopPressed();

});

// 3. React on out-events

statemachine.getSCInterface().getListeners().

add(new SCInterfaceListener() {

@Override

public void onPlayBtnEnabledRaised(boolean value) {

playBtn.setDisable(!value);

}

@Override

public void onStopBtnEnabledRaised(boolean value) {

stopBtn.setDisable(!value);

}

});

// 4. Run the state machine

statemachine.enter();

}

行動起來


準備好在Java 項目中使用狀態機了嗎？以下幾點介紹瞭如何啟用狀態機：


1. 下載YAKINDU Statechart Tools專業版。

2. 解壓已下載的文件並打開應用程序。

3.首先會看到的是歡迎頁面。單擊Examples並選擇Music Player（Java）。

4. 如果按鈕右上角出現Install Dependencies 的字樣，單擊它並按照說明操作。可能需要重啟一次應用程序。

5. 點擊按鈕voilá完成操作，操作界面中會出現音樂播放器示例，就可以使用了。


最後一條是關於許可證的說明。下載YAKINDU Statechart Tools後，可以在沒有任何功能限制的情況下使用該應用程序一個月。一個月後，需要購買許可證。但如果不將其用於工作，可以申請學術或非商業許可證。
