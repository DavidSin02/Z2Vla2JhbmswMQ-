水磨石簡介

水磨石（也稱磨石）是將碎石、玻璃、石英石等骨料拌入水泥粘接料製成混凝製品後經表面研磨、拋光的製品。以水泥粘接料製成的水磨石叫無機磨石，用環氧粘接料製成的水磨石又叫環氧磨石或有機磨石，水磨石按施工製作工藝又分現場澆築水磨石和預製板材水磨石地面。

傳統的水磨石以其造價低廉、可任意調色拼花、施工方便等獨特的優勢，在中國有著巨大的市場，已在全國城鄉各類建築物中使用的水磨石地面有十億平方米之多。


水磨石優點

（1）高檔水磨石（又稱商業水磨石）表光高亮處理後，高亮度達到70~90度以上、防塵防滑達到大理石品質。

（2）耐磨性水磨石、表面硬度可達到6-8級。

（3）現制或預製水磨石、可隨意拼接花色、顏色均可自定義配製。

（4）新型水磨石不開裂、不怕重車碾軋、不怕重物拖拉、不收縮變形。

（5）不起塵、潔淨度高；潔淨度滿足製藥、芯片製造等高潔淨環境的要求。

（6）無機水磨石不燃、不發火、耐老化、耐汙損、耐腐蝕、無異味無任何汙染。

（7）色澤豔麗光潔，若需提高光亮打地板蠟或結晶（不影響其防靜電性能），硬化耐磨水磨廠石，無需打臘也可保持長期高麗。

（8）現制水磨石裝飾分格條橫平豎直，分格條間無需間隙，連接密實，整體美觀性好；商業水磨石可任意設計圖案，達到美觀裝飾效果。

（9）建築性能及機械性能與各種建築中使用的優質水磨石地面無異，符合國標GB50209-2002要求，符合JC507-1993建築水磨石製品的標準要求。

物理性能

（1）光澤度：60～95；

（2）抗壓（N/cm2）:﹥2744；

（3）抗折（N/cm2）:﹥490；

（4）強度：大於75～100#；

（5）耐磨性：500g時大於8000轉無明顯磨損；

（6）起塵性：<1mg/cm2。

水磨石概述

（1）新型水磨石的出現，讓傳統水磨石地板磚幾乎面臨倒閉。

（2）壽命週期：有機水磨石一般為20-40年；無機水磨石可同建築同壽。

（3）無縫拼接水磨石，避免了藏汙納垢。

（4）採用的AJS結晶或硬化，操作簡便，費用低，效果盡人意。

由於以上的原因，人們通常認為水磨石是一種中高檔的地面裝修材料。

然而，傳統的施工技術造成水磨石地面十分粗糙、藏汙納穢，甚至還未交付使用就已骯髒不堪，更給日常衛生保潔帶來嚴重阻礙。然而，隨著時間的推移，實際使用中卻遇到了十分突出的問題：汙跡斑斑、黯淡無光。如何使已經風化、磨蝕的水磨石恢復其原有的亮麗美感，是建築裝飾行業越來越迫切要解決的問題。

磨拋養護

面對水磨石的風化、磨蝕，國內外最通用便捷的辦法就是將石材表面的汙垢清洗後，打上一層蠟。這種辦法對新的水磨石表面可以起到一定的保護作用，延緩自然風化、磨蝕的速度。若干年後水磨石表面風化後，僅需建造成本的25/1、30/1的價格對地面簡單研磨即可恢復對水磨石翻新。清洗上蠟之所以不是真正意義上的石材地面處理，原因在於沒有真正解決水磨石處理的技術關鍵。水磨石之所以光亮如鏡、晶瑩剔透，主要是因為水磨石在機械加工之後表面形成了一層玻璃質的薄膜，這層薄膜不但具有鏡面光澤，而且可以透明顯示出石材的內在結構層次即人們所觀察到的花紋圖案，亮麗美感就在這裡。要使風化、磨蝕的水磨石恢復其亮麗美感，關鍵就在於要在水磨石表面重新形成玻璃質薄膜。水磨石地面磨拋機的技術原理是：將水磨石表面風化、磨蝕的老化層刨去露出新鮮層，然後經特殊機械方法在水磨石表面產生極其複雜的作用而形成新的玻璃質薄膜，從而恢復水磨石的自然亮麗的裝飾效果。整個工藝過程包括多道工序。從根本上解決問題，讓粗糙汙髒的水磨石變得平滑光亮，治標又治本，達到鏡面效果。翻新成本低。施工無汙染、無灰塵、無泥水飛濺或噪聲，如商場、賓館內施工可以邊施工邊營業。可以改變傳統地面施工工藝，如地面可以先鋪毛板或先拼花再磨拋。不僅可以降低成本，還可以的消除施工中的拼接縫。

新型技術

石材用相關化工藥劑翻新養護已經不是一個新鮮的話題，國內石材翻新養護已經完全淘汰了打蠟這門工藝。為什麼還有很多施工人員對水磨石翻新養護都還在用著古老又原始的打蠟技術？因為石材用化工藥劑翻新晶面操作最早是由西方國家引進，但卻未能研發出水磨石結晶處理的藥劑。雖然人們早已明白水磨石及地板使用打蠟都會出現：操作麻煩、成本高、地板滑、容易發黃，晶面效果差及不持久等等缺點。施工人員往往都很無奈，根本想不出更好的方案，也找不到更好的產品來代替打蠟。早在2005年初水磨石投入市場，此藥劑的優點是，容易操作，成本極低，晶面效果可以跟天然大理石媲美。而且在2005年已經施工的水磨石案例，都保持著原有的晶面及亮度，至今還無須護理。其實高亮水磨石這一產品在我國雲南、貴州、等地區已經家喻戶曉。相信此產品完全可以淘汰水磨石打蠟，在不久的將來水磨石水晶硅完全可以普及到各個地區及國家。

改性特點

（1）水磨石水晶處理表面光潔度高，可達到90度光澤度，最高可達到102度光澤度，相當於進口中高檔大理石表面的品質。

（2）表面硬高度達5-7級，接近高硬度花崗石的表面，耐磨性好。

（3）抗滲透，可以防水防汙(滲水率小於0.8)、耐油、耐弱酸、鹼、耐鹽霧，自然防護的綜合性能超過現行所有石材類產品。

（4）使用壽命高達30年，特殊的配方和結構設計確保了“高亮水晶水磨石”板在投入使用後的循環修復簡便，用於地面的保養清潔費用和衛生管理難度大大降低。

（5）“水磨石高亮處理劑”處理後的水磨石地坪因表面附著抗滲透材料，所以水磨石不反潮，不再有透水效果，不會造成地面潮溼、地面打滑等情況，工業廠房、學校教育體系等一系列工程優選使用。

工藝標準

範圍

本工藝標準適用於工業與民用建築普通與高級現制水磨石地面工程。

施工準備

1 材料及主要機具：

1.1 水泥：白色或淺色的水磨石面層，應採用白水泥，深色的水磨石面層，宜採用硅酸鹽水泥、普通硅酸鹽水泥或礦渣硅酸鹽水泥，其標號不應小於425號。同顏色的面層應使用同一批水泥。

1.2 礦物顏料：水泥中摻入的顏料應採用耐光、耐鹼的礦物顏料，不得使用酸性顏料。其摻入量宜為水泥重量的3%～6%，或由試驗確定。同一彩色面層應使用同廠，同批的顏料。

1.3 石粒：應採用堅硬可磨的白雲石、大理石等岩石加工而成，石料應潔淨無雜物，其粒徑除特殊要求外，宜為4～14mm。

1.4 分格條

1.4.1 玻璃條：平板普通玻璃裁製而成，3mm厚，一般10mm寬（根據面層厚度而定），長度以分塊尺寸而定。

1.4.2 銅條：l～2mm厚銅板裁成10mm寬（還要根據面層厚度而定），長度以分格尺寸而定，用前必須調直調平。

1.5 砂：中砂，過8mm孔徑的篩子，含泥量不得大於3%。

1.6 草酸：塊狀、粉狀均可，用前用水稀釋。

1.7 白蠟及22號鐵絲。

1.8 主要機具：水磨石機、滾筒（直徑一般200～250mm，長約 600～700mm，混凝土或鐵製）、木抹子、毛刷子、鐵簸箕、靠尺、手推車、平鍬、5mm孔徑篩子、油石（規格按粗、中、細）、膠皮水管、大小水桶、掃帚、鋼絲刷、鐵器等。

2 作業條件：

2.1 頂棚、牆面抹灰已完成並已驗收，屋面已做完防水層。

2.2 安裝好門框並加防護，與地面有關的水、電管線已安裝就位，穿過地面的管洞已堵嚴、堵實。

2.3 做完地面墊層，按標高留出磨石層厚度（至少3cm）。

2.4 石粒應分別過篩，並洗淨無雜物。[1]

操作工藝

3.1 工藝流程：

基層處理 → 找標高 → 彈水平線 → 鋪抹找平層砂漿 → 養 護 →

彈分格線 → 鑲分格條 → 拌制水磨石拌合料 → 塗刷水泥漿結合層 →

鋪水磨石拌合料 → 滾壓、抹平 → 試 磨 → 粗 磨 → 細 磨 →

磨 光 → 草酸清洗 → 打蠟上光

3.1.1 基層處理：將混凝土基層上的雜物清淨，不得有油汙、浮土。用鋼鏨子和鋼絲刷將沾在基層上的水泥漿皮鏨掉鏟淨。

3.1.2 找標高彈水平線：根據牆面上的+50cm標高線，往下量測出磨石面層的標高，彈在四周牆上，並考慮其它房間和通道面層的標高，要相一致。

3.1.3 抹找平層砂漿：

3.1.3.1 根據牆上彈出的水平線，留出面層厚度（約10～15mm厚），抹1∶ 3水泥砂漿找平層，為了保證找平層的平整度，先抹灰餅（縱橫方向間距1.5m左右），大小約8～10cm。

3.1.3.2 灰餅砂漿硬結後，以灰餅高度為標準，抹寬度為8～10cm的縱橫標筋。

3.1.3.3 在基層上灑水溼潤，刷一道水灰比為0.4～0.5的水泥漿，面積不得過大，隨刷漿隨鋪抹1∶3找平層砂漿，並用2m長刮槓以標筋為標準進行刮平，再用木抹子搓平。

3.1.4 養護：抹好找平層砂漿後養護24h，待抗壓強度達到l.2MPa，方可進行下道工序施工。

3.1.5 彈分格線：根據設計要求的分格尺寸，一般採用lm×lm。在房間中部彈十字線，計算好周邊的鑲邊寬度後，以十字線為準可彈分格線。如果設計有圖案要求時，應按設計要求彈出清晰的線條。

3.1.6 鑲分格條：用小鐵抹子抹稠水泥漿將分格條固定住（分格條安在分格線上），抹成30°八字形（圖7-1），高度應低於分格條條頂4～6mm，分格條應平直（上平必須一致）、牢固、接頭嚴密，不得有縫隙，作為鋪設面層的標誌。另外在粘貼分格條時，在分格條十字交叉接頭處，為了使拌合料填塞飽滿，在距交點40～50mm內不抹水泥漿（圖7-2）。

3.1.11 試磨：一般根據氣溫情況確定養護天數，溫度在20～30℃時2～3d即可開始機磨，過早開磨石粒易鬆動；過遲造成磨光困難。所以需進行試磨，以面層不掉石粒為準。

3.1.12 粗磨：第一遍用60～90號粗金剛石磨，使磨石機機頭在地面上走橫“8”字形，邊磨邊加水（如磨石面層養護時間太長，可加細砂，加快機磨速度），隨時清掃水泥漿，並用靠尺檢查平整度，直至表面磨平、磨勻，分格條和石粒全部露出（邊角處用人工磨成同樣效果），用水清洗晾乾，然後用較濃的水泥漿（如摻有顏料的面層，應用同樣摻有顏料配合比的水泥漿）擦一遍，特別是面層的洞眼小孔隙要填實抹平，脫落的石粒應補齊。澆水養護2～3d。

3.1.13 細磨：第二遍用90～120號金剛石磨，要求磨至表面光滑為止。然後用清水衝淨，滿擦第二遍水泥漿，仍注意小孔隙要細緻擦嚴密，然後養護2～3d。

3.1.14 磨光：第三遍用200號細金剛石磨，磨至表面石子顯露均勻，無缺石粒現象，平整、光滑，無孔隙為度。

普通水磨石面層磨光遍數不應少於三遍，高級水磨石面層的厚度和磨光遍數及油石規格應根據設計確定。

3.1.15 草酸擦洗：為了取得打蠟後顯著的效果，在打錯前磨石面層要進行一次適量限度的酸洗，一般均用草酸進行擦洗，使用時，先用水加草酸化成約10%濃度的溶液，用掃帚蘸後灑在地面上，再用油石輕輕磨一遍；磨出水泥及石粒本色，再用水沖洗軟布擦乾。此道操作必須在各工種完工後才能進行，經酸洗後的面層不得再受汙染。

3.1.16 打蠟上光：將蠟包在薄布內，在面層上薄薄塗一層，待幹後用釘有帆布或麻布的木塊代替油石，裝在磨石機上研磨，用同樣方法再打第二遍蠟，直到光滑潔亮為止。

3.1.17 冬期施工現制水磨石面層時，環境溫度應保持+5℃以上。

3.1.18 水磨石踢腳板：

3.1.18.1 抹底灰：與牆面抹灰厚度一致，在陰陽角處套方、量尺、拉線，確定踢腳板厚度，按底層灰的厚度衝筋，間距l～1.5m。然後裝檔用短槓刮平，木抹子搓成麻面並劃毛。

3.1.18.2 抹磨石踢腳板拌合料：先將底子灰用水溼潤，在陰陽角及上口，用靠尺按水平線找好規矩，貼好靠尺板，先塗刷一層薄水泥漿，緊跟著抹拌合料，抹平、壓實。刷水兩遍將水泥漿輕輕刷去，達到石子面上無浮漿。常溫下養護24h後，開始人工磨面。

第一遍用粗油石，先豎磨再橫磨，要求把石渣磨平，陰陽角倒圓，擦第一遍素灰，將孔隙填抹密實，養護l～2d，再用細油石磨第二遍，用同樣方法磨完第三遍，用油石出光打草酸，用清水擦洗乾淨。

3.1.18.3 人工塗蠟：擦二遍出光成活。

質量標準

4.1 保證項目：

4.1.1 選用材質：品種、強度（配合比）及顏色，應符合設計要求和施工規範的規定。

4.1.2 面層與基層的結合必須牢固，無空鼓、裂紋等缺陷。

4.2 基本項目：

4.2.1 面層：

4.2.1.1 普通水磨石：表面光滑、無裂紋、砂眼和磨紋，石粒密實，顯露均勻，圖案符合設計要求，顏色一致，不混色，分格條牢固、清晰、順直。

4.2.1.2 高級水磨石：表面平整光潔，無裂紋、砂眼和磨紋，石粒密實，顯露均勻一致，相鄰顏色不混色，分格條牢固、順直、清晰，陰陽角收邊方正。

4.2.2 踢腳板：

4.2.2.1 普通水磨石：高度一致，出牆厚度均勻，與牆面結合牢固，局部雖有空鼓，但其長度不大於200mm，且在一檢查範內不多於2處。

4.2.2.2 高級水磨石：表面平整光潔，與牆面結合牢固，出牆厚度一致，上口平直。

4.2.3 樓梯和臺階相鄰兩步的寬度和高差不超過10mm，楞角整齊，防滑條順直。

4.2.4 地面鑲邊：

4.2.4.1 普通水磨石：鑲邊的用料及尺寸應符合設計和施工規範的規定，邊角整齊光滑，不同面層顏色相鄰處不混色。

4.2.4.2 高級水磨石：尺寸正確，拼接嚴密，相鄰處不混色，分色線順直，邊角整齊光滑、清晰美觀。

4.2.5 打蠟質量：蠟灑布均勻不露底，色澤一致，厚薄均勻，光滑明亮，圖紋清晰，表面潔淨。


成品保護

5.1 鋪抹水泥砂漿找平層時，注意不得碰壞水、電管路及其它設備。

5.2 運輸材料時注意保護好門框。

5.3 進行機磨水磨石面層時，研磨的水泥廢漿應及時清除，不得流入下水口及地漏內，以防堵塞。

5.4 磨石機應設罩板，防止研磨時濺汙牆面及設施等，重要部位及設備應加覆蓋。

6.1 分格條折斷，顯露不清晰：主要原因是分格條鑲嵌不牢固（或未低於面層），液壓前未用鐵抹子拍打分格條兩側，在滾筒滾壓過程中，分格條被壓彎或壓碎。因此為防止此現象發生，必須在滾壓前將分格條兩邊的石子輕輕拍實。

6.2 分格條交接處四角無石粒：主要是粘結分格條時，稠水泥漿應粘成30°角，分格條頂距水泥漿4～6mm，同時在分格條交接處，粘結漿不得抹到瑞頭，要留有抹拌合料的孔隙。

6.3 水磨石面層有洞眼、孔隙：水磨石面層機磨後總有些洞孔發生，一般均用補漿方法，即磨光後用清水衝乾淨，用較濃的水泥漿（如彩色磨石面時，應用同顏色顏料加水泥擦抹）將洞眼擦抹密實，待硬化後磨光；普通水磨石面層用“二漿三磨”法，即整個過程磨光三次擦漿二次。如果為圖省事少擦抹一次，或用掃帚掃而不是擦抹或用稀漿等，都易造成面層有小孔洞（另外由於擦漿後未硬化就進行磨光，也易把洞孔中灰漿磨掉）。

6.4 面層石粒不勻、不顯露：主要是石子規格不好，石粒未清洗，鋪拌合料時用刮尺刮平時將石粒埋在灰漿內，導致石粒不勻等現象。
