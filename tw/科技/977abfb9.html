冒泡排序

思想：

1.相鄰元素倆倆比較，小的在前、大的在後。（這時按從小到大排序時，從大到小排序時相反）

2.重複這個步驟 n 次。（n為序列的長度，實際上比較的次數可能比n小，可以設置一個判斷（該輪沒有元素交換時排序完成）來優化冒泡排序）

3.倆倆比較時，不是位置 1 和 2、3和4 比較。 而是！！！，1 和 2 位置元素比較完後，比較 2和3位置的元素，一直到n - 1與 n 位置的元素比較。

如

{9、5、4、3、2、1}

第一次交換：

先判斷 9 與 5 ，9比5大，交換 {5、9、4、3、2、1}

判斷9與4，9比4大，交換{5、4、9、3、2、1}

重複，一直到最後兩個位置比較....

可以知道，第一次交換後，9被挪到了最後位置：{5、4、3、2、1、9}

第二次交換

5被挪到了1的位置。

{4、3、2、1、5、9}

....

像這樣重複n次後，將得到{1、2、3、4、5、9}

（從上面我們也可以知道，與我們需要的次序正好完全相反時，冒泡排序必須要執行n次，此時冒泡排序的效率最低）



快速排序

思想：

主要用到分治思想。

1.在該序列中，選擇一個數（處於中間的數（mid = （ l + r ）/ 2 ）。把序列中比它小的放在它左邊，把序列中比它大的放在右邊。

2.根據分治思想，我們知道，現在的步驟可以使該序列有序。那麼對它（mid）左邊的序列中也執行這個步驟，對它（mid）右邊的序列中也執行這個步驟。

3.這樣後整個序列都有序。

排序步驟：

要怎麼快速地執行上述（1）中的操作呢？

可以用一個遞歸函數 套 循環來實現。

現在排序（序列頭位置為l，序列尾位置為r的序列）{    找到中間位置元素mid    while（元素A的位置小於元素B的位置（相當於序列還未遍歷完））{                從左到右找到 第一個比mid大的元素A          從右到左找到 第一個比mid小的元素B                if（元素A的位置小於元素B的位置）{              交換元素A與元素B                        }    }  if（元素A的位置小於元素mid的位置） 現在排序（A的位置，mid的位置）  if（元素B的位置大於元素mid的位置） 現在排序（mid的位置，B的位置)

如序列{9、5、4、3、2、1}

中間位置元素是

4

9是第一個比4大的左邊元素

1是第一個比4小的右邊元素

交換

{1、5、4、3、2、9}

再繼續，5與2互換

{1、2、4、3、5、9}

再交換

{1、2、3、4、5、9}

排序完成



歸併排序

思想：

1.先拆分、後合併。

2.把倆個及以上的有序子表，二路合併到一個新的有序序列。



如

{5，9，6，3，4，2}


先拆分到最小：

{5} {9} {6} {3} {4} {2}

然後倆倆合併，並保證合併後有序

{5，9}、{3，6}、{2，4}

再倆倆合併（如果是奇數可以先保留最後一組不合並）

{3，5，6，9} {2，4}

再合併

{2，3，4，5，6，9}


排序過程：

最重要的就是合併過程：

如有兩個有序序列AB，如何把他們合併到序列C呢？

下面是偽代碼：

用 i 來遍歷 A，j 來遍歷 Bwhile( i 未遍歷完 A 且 j 未遍歷完 B){         if( A[ i ] 比 B [ j ] 小 ) {              把 A[ i ] 放入 序列 C中。              i++ // i 後移         }         else{              把 B[ j ] 放入 序列 C中。              j++ // j 後移         }}while(i 未遍歷完 A ){  把 A[ i ] 放入 序列 C中。  i++   }//當有一個已經遍歷完後，把另一個序列剩下的直接存入序列Cwhile(j 未遍歷完 B ){  把 B[ j ] 放入 序列 C中。   j++  }



基數排序

思路：

比較新奇，是按照數字每個位置的數來排序。（時候於）

並且需要有10個編號為0到9的不定長容器（容量大於n的數組也可以）

1.先取出每個數的個位數（為什麼不是從最高位開始？因為如果有某個數的位數遠遠超過其他數，那麼會從最高位開始會浪費很多時間）

2.從第一個數開始到最後一個數，把對應的數放到編號為個位數的容器裡。（同一個容器中時，按照先進的在前面的順序）

3.然後取出每個數的十位數，按照容器的編號0到9順序取出每個數，重新把對應的數放到編號為十位數的容器裡。

4.一直取，直到容器內元素無變化，或者取到最高位。

排序步驟：

如，序列{135，242，192，93，345，11，24，19}

先取個位，依次放入對應容器裡。（同一個容器中時，按照先進的在前面的順序）
