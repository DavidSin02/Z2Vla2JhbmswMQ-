微軟 Edge 瀏覽器開發版 87.0.664.8：新增大量新功能

2020-10-29
| 科技

IT之家10月16日消息 微軟昨日向 Dev 通道推出了 Edge 瀏覽器開發版的 87.0.664.8 版本，這是 v87 主版本的最後一個版本。意即從下週開始將推出 v88 版本的 Edge 開發版。

該版本除了 bug 修復外還重做了收藏夾菜單，引入了新的電商比價功能以及 PDF 的數據恢復等特性。

以下是新功能的完整列表：

現在將 “收藏夾”菜單更改為可固定的窗格，該窗格在單個垂直列表中列出了所有收藏夾，而不是多個彈出菜單。

添加了比價功能。

為 PDF 添加了數據恢復功能，以便在發生崩潰時不會丟失所編輯的內容或註釋。

改進了對 AV1 編碼視頻的支持。（請注意，仍然需要從此處安裝 Windows 擴展。）

添加了鍵盤快捷鍵以打開和關閉集錦。

如果尚未打開 “同步”，則向 “個人資料”彈出按鈕添加了一個按鈕以打開 “同步”。

添加了用於控制是否顯示用於在網頁上顯示密碼的按鈕的設置，以及 “啟用密碼顯示”的管理策略。請注意，仍然需要更新文檔和管理模板。

為 “地址欄編輯已啟用”添加了管理策略，但仍然需要更新文檔和管理模板。

為 “啟用 Web 捕獲”添加了管理策略。請注意，仍然需要更新文檔和管理模板。

為工作和學校用戶合併了新聞和 Office 新選項卡頁。請注意，仍然可以通過管理策略為用戶設置僅 Office 的新選項卡頁。

更廣泛地推出預渲染的新標籤頁。

棄用新選項卡 Feed 類型的管理策略。

當然，該版本同樣修復了部分程序 bug 和已知問題。以下是為提高可靠性而修復的問題：

修復了導航到某些網站會使瀏覽器崩潰的問題。

修復了 “朗讀”有時會使瀏覽器崩潰的問題。

修復了 PDF 上的 “朗讀”有時會使選項卡崩潰的問題。請注意，仍有一個已知的 “大聲朗讀”掛起，如果掛起時間過長，可能會導致該標籤最終崩潰。

修復了同步集合時崩潰的問題。

修復了將項目拖到 “收藏夾”有時會使 “收藏夾”窗格崩潰的問題。

修復了播放某些視頻時崩潰的問題。

修復了某些視頻無法播放的問題。

以下是更改行為的修復程序：

修復了更新 Edge 導致當前會話關閉而不是還原（如果設置為還原）的問題。

修復了更新後用戶退出 Edge 的問題。

修復了始終在 Edge 中打開下載的 PDF 而不是在默認 PDF 查看器中打開的問題。

修復了收藏夾列表有時無法打開的問題。

修復了一些網站權限無法從 “設置”中刪除的問題。

修復了無法從沉浸式閱讀器中將項目添加到集合中的問題。

修復了 Edge 在全屏狀態下斷開監視器有時會導致 Edge 在其餘監視器上不再可見的問題。

修復了 “密碼監視器”有時會在警報彈出窗口中報告與 “設置”中報告的數字不同的問題。

修復了固定在 “開始”菜單上的網站有時不顯示網站名稱的問題。

修復了任務欄快捷方式有時看起來像素化或粗糙的問題。

修復了啟用 Tracking Prevention 時 Amazon Assistant 無法工作的問題。

修復了有時無法使用 “全部打開”的問題。

修復了某些網頁表單在不期望的情況下會自動填充的問題。

修復了有時將網頁上的非密碼錶單檢測為密碼的問題。

解決了 Web Capture 預覽有時與所選區域不匹配或顯示所選區域超出窗口的問題。

最後，這是部分已知的問題：

使用某些廣告屏蔽擴展程序的用戶可能會在 YouTube 上遇到播放錯誤。解決方法是，暫時禁用擴展程序應允許繼續播放。有關更多詳細信息。

一些用戶仍然遇到一個問題，即所有選項卡和擴展名都立即崩潰，並顯示 STATUS_INVALID_IMAGE_HASH 錯誤。導致此錯誤的最常見原因是賽門鐵克等供應商提供的過時的安全性或防病毒軟件，在這種情況下，更新該軟件將對其進行修復。

安裝了相關擴展程序的 Kaspersky Internet Suite 用戶有時可能會看到 Gmail 等網頁無法加載。該失敗是由於主要的卡巴斯基軟件已過期，因此可以通過確保安裝了最新版本來解決。

在我們對該區域進行一些先前的修復後，一些用戶發現收藏夾被複制了。觸發此問題的最常見方式是安裝 Edge 的穩定通道，然後使用之前已經登錄 Edge 的帳戶登錄。現在可以使用重複數據刪除器工具來解決此問題。但是，我們還發現在多臺計算機上運行重複數據刪除器之前，任何一臺計算機都有機會完全同步其更改，因此會發生複製，因此，在等待一些我們已完成的修復工作中，請確保離開兩次運行重複數據刪除器之間有大量時間。

在對其進行初步修復之後，一些用戶仍然遇到 Edge 窗口變黑的情況。打開瀏覽器任務管理器（鍵盤快捷鍵為 shift + esc）並殺死 GPU 進程通常可以解決該問題。請注意，這似乎隻影響使用某些硬件的用戶，最容易通過調整 Edge 窗口的大小來觸發。對於具有獨立 GPU 的用戶，更新圖形驅動程序可能會有所幫助。

在使用觸控板手勢或觸摸屏滾動時，一些用戶看到 “搖擺”行為，其中在一個維度上滾動還會導致頁面在另一個維度上來回滾動。請注意，這隻會影響某些網站，並且在某些設備上似乎更糟。這很可能與我們正在進行的工作使滾動回到與 Edge Legacy 的行為同等有關，因此，如果這種行為是不受歡迎的，則可以通過禁用 edge：// flags /#edge-experimental-scrolling 標誌來暫時將其關閉。

存在一些問題，使用多個音頻輸出設備的用戶有時無法從 Edge 獲得任何聲音。在一種情況下，Edge 在 Windows Volume Mixer 中變為靜音，並且取消靜音可以對其進行修復。另一方面，重新啟動瀏覽器可以修復它。

IT之家知悉，該版本與往常一樣，用戶可以通過打開 Edge設置 ->關於微軟 Edge 手動進行更新，同時它也支持在後臺自動更新。
