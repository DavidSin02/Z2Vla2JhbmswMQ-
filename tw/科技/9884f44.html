變量係數對機器學習（ML）迴歸模型精度的影響分析

2020-10-29
| 科技

R平方（R²），均方誤差（MSE），平均絕對誤差（MAE）和均方根誤差（RMSE）是衡量連續變量準確度的最常用指標。在這篇文章中，我們將觀察變量係數（CoV）對MAE，MSE，R²和Accuracy 的影響。我們將相同的線性迴歸應用於4個不同的數據，這些數據具有不同係數的變量，以解釋MSE，MAE，R²和Accuracy 如何以及為何發生變化。首先，當我們保持MSE和MAE固定時，我們將通過變量係數的變化觀察R²和Accuracy 。其次，在保持R 2和Accuracy 固定的同時，我們將觀察MSE和MAE隨變量係數的變化。

導入Python庫：

import numpy as npimport pandas as pdimport matplotlib.pyplot as pltimport seaborn as snsfrom itertools import countfrom sklearn import metricsimport statsmodels.api as smfrom sklearn.linear_model import LinearRegression%matplotlib inline

1.恆定MSE和增加R²

我們將創建具有不同係數變量的4個不同數據（D1，D2，D3，D4）。首先，我們保持對D1，D2，D3和D4的MSE和MAE固定，而D1，D2，D3和D4的變量係數值分別為0.1,0.5,1,10。

Data1（D1） = 0.1 + 0.1 * X_1 + 0.1 * X_2 + 0.1 * X_3 +（Random Error）

雖然常數和變量係數（CoV）是0.1，但我們將添加具有誤差係數（CoE）= 0.1的隨機誤差。我們將看到CoE = 0.1 / CoV = 0.1速率對Accuracy 和R²的影響。

Data2（D2） = 0.5 + 0.5 * X_1 + 0.5 * X_2 + 0.5 * X_3 +（Random Error）

當常數和CoV為0.5時，我們將添加CoE = 0.1的隨機誤差。觀察Accuracy 和R²對CoE = 0.1 / CoV = 0.5的影響。

Data3（D3） = 1 + 1 * X_1 + 1 * X_2 + 1 * X_3 +（Random Error）

當常數和係數為1時，我們將添加CoE = 0.1的隨機誤差。

Data4（D4） = 10 + 10 * X1 + 10 * X2 + 10 * X3 +（RandomError）
