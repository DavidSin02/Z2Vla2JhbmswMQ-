馬鈴薯中耕季，注意做好4件工作

2020-11-14
| 科技

對於華北馬鈴薯區域而言，種地的朋友們剛剛從播種中鬆了口氣，又迎來了讓人心跳加速的第一次中耕，著實讓人有種喘不過氣來的感覺，尤其是大規模種植的朋友們，基本上是連軸轉。此番進入中耕季，關於中耕、除草和施肥方面，小編將近段時間發現的情況，並結合理論依據所形成的思路整理出來，僅供大家參考。

0 1中耕時間的確定

什麼時候開始第一次中耕，這個問題很關鍵，為什麼？根本原因就在於擔心“捂苗”，中耕太早，土層太厚，苗出不來；中耕時間太晚，苗太高，培土後壓苗。怎麼把握這個時間尺度，是個關鍵，當田間有零星幼苗破土以後，即可著手安排中耕，宜早不宜晚，如果用數字來標定的話，即每100個苗中有2~3個破土就可以中耕。時間上，通常是自播種後20天左右，溫度和土壤較適宜的情況下，也可提前至15天左右。


0 2成型壟的規範

中耕培土的目的主要是讓馬鈴薯塊莖有充足的生長空間，確保馬鈴薯在結2層，甚至3層薯的基礎上，不露頭，最終不出現青皮薯。通常一次中耕以後，單壟單行種植情況下，整個壟面的周長在110~120cm為宜，若低於110cm，培土效果較差，需要酌情矯正，同時，在培土過程中，需要定期檢查幼苗是否位於整個壟的中心位置，避免出現“偏壟”現象。

整個壟面的周長在110~120cm為宜

0 3封閉除草

伴隨著中耕，同時進行封閉除草，是最為省工省力、高效的一種除草方式。封閉除草最為恰當的時候是剛剛中耕的時候，此時馬鈴薯幼苗上部有5cm左右的溼土層保護幼苗不受除草劑的影響，同時雜草也正處於剛剛萌發或未萌發階段，噴施封閉除草劑，如二甲戊靈後，在地表2~3cm處形成毒土層，2葉期以前的雜草觸碰後即死亡。

一邊中耕，一邊噴施封閉除草劑（二甲戊靈）

0 4中耕施肥

注意中耕補鉀

以烏蘭察布地區為例，利用K+離子計，針對今年走訪的20餘家基地的土壤浸提液（土水比2.5:1）進行測定，發現土壤浸提液中的鉀含量普遍偏低，大部分遠低於適宜土壤的鉀含量標準。因此，建議中耕追肥時，施15~20公斤複合肥（12-18-15）或（12-19-16）+10~20公斤硫酸鉀，具體施用量視底肥施用量而定。
