優雅的製作一個溫溼度計圖表

2020-10-29
| 科技

1.有的和想要的

有數據：有溫度、有溼度

想要的：顯示在頁面某個位置

2.期望的優雅

期望有一個方法，只需給定溫度、溼度與位置，直接生成溫溼度計圖表。

3.來封裝一個

3.1 先睹風采

使用以下方法即可完成圖表顯示。

Gauge.Hygrothermograph('gauge1', 28, 60);

效果如下：

3.2 使用方式

引入echart 和封裝的 js ，即可使用 Gauge.Hygrothermograph 方法。

<script src="https://cdn.bootcss.com/echarts/4.2.1/echarts.min.js"></script><script src="http://www.timeddd.com/Demo/gauge.js"></script>

3.3 完整的 html 如下：

<!DOCTYPE html><html xmlns="http://www.w3.org/1999/xhtml"><head><meta http-equiv="Content-Type" content="text/html; charset=utf-8" /><title>儀表盤溫溼度示例</title> <style> .gauge { width: 230px; height: 230px; border: solid 1px gray; padding: 5px; }</style></head><body> <div id="gauge1" class="gauge"></div>   <script src="https://cdn.bootcss.com/echarts/4.2.1/echarts.min.js"></script> <script src="http://www.timeddd.com/Demo/gauge.js"></script>  <script> Gauge.Hygrothermograph('gauge1', 28, 60); </script> </body></html>

3.3 封裝思路

利用 echart 強大的圖表功能，設置好適合溫溼度呈現的 option，將數據部分參數化。為了支持更新時只修改 option 中的數據部分，定義一個列表來存儲元素渲染過後的對象。

var Gauge = {}; Gauge.ChartList = {};

具體方法定義如下（option屬性部分省略）：

/** * 生成溫溼計圖表  *  *@param{string}elementId 顯示溫溼計表的HTML元素id *@param{number}temperature 溫度 *@param{number}humidity 溼度 *@return */Gauge.Hygrothermograph = function(elementId, temperature, humidity){ var ele = document.getElementById(elementId); var myChart = this.ChartList[elementId];  if(!myChart){  myChart = echarts.init(ele);  this.ChartList[elementId] = myChart; }else{  var option = myChart.getOption(); option.series[0].data[0].value = humidity; option.series[1].data[0].value = temperature; myChart.setOption(option, true);  return; }  var option = { // ..... };  myChart.setOption(option, true);}
