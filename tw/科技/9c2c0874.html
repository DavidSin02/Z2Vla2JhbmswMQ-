計數循環

循環某一個範圍，例如從 0 循環到 10。

其中: 起始數字可省略，默認值為 0。

步長可省略，默認值為 1。 另外，循環範圍是起始數字~(結束數字-1)，即不包含結束數字。

for x in range([起始數字],結束數字,[步長]):    <循環內容>

列表循環

在循環列表時，in 後面是裝有列表的變量名稱。

循環次數等於列表的長度，x 是每次循環時列表對應位置的值。

books = [‘西遊記’,’水滸傳’,’紅樓夢’,’三國演義’]for x in books:    print(x)

文字的循環

在循環文字時，in 後面是文字的變量。 循環次數等於文字的字符數，x 是每次循環時對應位置的字符。

for x in ‘我愛中國’:    print(x)

while 循環

當判斷條件滿足時才會執行。

while <判斷條件>:    <循環內容>

break 和 continue

break:停止循環

continue:跳過本次循環

循環結構的嵌套

多個循環之間可以相互嵌套，從而形成更加複雜的邏輯結構。

如下為“99 乘法表”的實現程序:

try:<可能出現異常的語句>except ZeroDivisionError:<出現除 0 錯誤後才會執行的語句>except SyntaxError:<出現語法錯誤後才會執行的語句>...except :<出現異常後，但不是上述異常類別時才會執行的語句>for x in range(1,10):    for y in range(1,x+1):        print(“x”,”*”,y,”=”,x*y, end=””)    print()

try...except...

此語句進行異常處理，即出現異常時 except 語句塊將會執行。

try:    <可能出現異常的語句>except ZeroDivisionError:    <出現除 0 錯誤後才會執行的語句>except SyntaxError:    <出現語法錯誤後才會執行的語句>...except :    <出現異常後，但不是上述異常類別時才會執行的語句>

try...except...else...

此語句包含 else 語句塊，即沒有出現異常的時候將會執行。

try:    <可能出現異常的語句>except:    <出現異常後執行的語句>else:    <沒出現異常時執行的語句>

try...except...finally

此語句塊包含 finally，為無論是否出現異常均會執行的語句塊。

try:    <可能出現異常的語句>except:    <出現異常後執行的語句>finally:    <無論是否出現異常，均為執行的語句>

Turtle 繪圖

Turtle(海龜)繪圖為入門庫，用於進行簡單的編程繪圖，需要掌握的知識點如下:

# 導入海龜import turtle# 設置背景色 turtle.bgcolor('black') # 設置窗體大小 turtle.setup(800,600) # 獲取海龜的筆pen = turtle.Pen()# 更改筆的粗細 pen.width(10)# 更改筆的顏色 pen.color('red')# 更改繪畫速度 pen.speed(10)# 繪製直線 pen.forward(100)# 左/右轉彎 pen.left(90) pen.right(75)# 抬筆/落筆 pen.up() pen.down()# 設置筆的位置 pen.setposition(0,0) # 設置筆的方向 pen.setheading(90) # 設置填充顏色 pen.fill_color('pink') # 開始填充 pen.begin_fill()for x in range(4):     pen.forward(60)    pen.right(90) #結束填充pen.end_fill()# 註冊鼠標點擊事件 def click(x,y):    print('你點擊的位置是:',x,y) turtle.onscreenclick(click, btn=1) # 註冊鍵盤監聽事件def key():    print('你按下 a 鍵了') turtle.onkeypress(key, ‘a’) turtle.listen()# 隱藏筆的形狀 pen.hideturtle() #繪圖完成turtle.done()
