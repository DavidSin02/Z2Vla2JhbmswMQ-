初識C語言指針

2020-11-14
| 科技

一切皆為地址

C語言用變量來存儲數據，用函數來定義一段功能代碼，它們最終都要放到內存中供 CPU 調用。

數據和代碼都以二進制的形式存儲在內存中，計算機無法從格式上區分某塊內存存儲的是數據還是代碼。當程序被加載到內存後，操作系統會給不同的內存塊指定不同的權限，擁有讀取和執行權限的內存塊就是代碼，而擁有讀取和寫入權限（也可能只有讀取權限）的內存塊就是數據。

CPU訪問內存時需要的是地址，而不是變量名或函數名。程序在執行過程中會告知 CPU 要執行的代碼以及要讀寫的數據的地址。變量名和函數名只是地址的一種助記符，當源文件被編譯和鏈接成可執行程序後，它們都會被替換成地址。

何為指針

計算機中所有的數據都必須放在內存中，不同類型的數據佔用的字節數不一樣，例如 int 佔用 4 個字節，char 佔用 1 個字節。為了正確地訪問這些數據，必須為每個字節都編上號碼，就像門牌號、身份證號一樣，每個字節的編號是唯一的，根據編號可以準確地找到某個字節。這個編號就是指針（Pointer）。

也就是說，指針是一個變量，其值為另一個變量的地址（Address），即變量在內存中字節的編號。

指針變量的聲明

與其他變量或常量一樣，在使用指針存儲其他變量地址之前，需對其進行聲明。指針變量聲明的一般形式為：

type *var_name;

其中 type 是指針的類型，它必須是一個有效的 C 數據類型，var_name 是指針變量的名稱。

如下圖所示，x是一個整型變量，其值為8，p是一個整型指針，指向x。使用&操作符取得變量x的地址，將其賦給p，從而使p指向變量x的地址；使用*操作符即可訪問該地址存儲的變量值。

用來聲明指針的星號 * 與乘法所使用的星號是相同的。但在這裡，星號是用來聲明這個變量是指針。

char *pc; /* 一個字符型指針 */

int *pi; /* 一個整型指針 */

float *pf; /* 一個浮點型指針 */

double *pd; /* 一個 double 型指針 */

指針變量的大小

在32位計算機中，一個指針變量所佔的字節數通常是4，在64位系統中通常為8。
