PHP中開啟gzip壓縮的2種方法

2020-11-14
| 科技

網頁開啟gzip壓縮以後，其體積可以減小20%~90%，可以節省下大量的帶寬，從而減少頁面響應時間，提高用戶體驗。

php配置改法：

代碼如下:

zlib.output_compression = On

;開啟gzip功能zlib.output_compression_level = 4

;gzip壓縮級別，1~9，建議3~5

;zlib.output_handler =

;gzip壓縮方式，建議註釋

php文件改法：

建議在頁面開頭寫上，可以寫到頭部公共文件

代碼如下:

if( !headers_sent() && // 如果頁面頭部信息還沒有輸出

extension_loaded("zlib") && // 而且php已經加載了zlib擴展

strstr($_SERVER["HTTP_ACCEPT_ENCODING"],"gzip")) //而且瀏覽器接受GZIP

{

ini_set('zlib.output_compression', 'On');

ini_set('zlib.output_compression_level', '4');

}
