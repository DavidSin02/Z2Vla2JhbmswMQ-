西門子S120的存儲器有哪些，它們如何進行參數存儲訪問

2020-10-29
| 科技

新手在調試SINAMICS S120時，由於對SINAMICS S120的參數存儲結構不熟悉，執行了錯誤的上傳/下載操作，都會造成設置參數的丟失，對調試進度造成影響。比如：在線調試完成後卻又執行了下載操作，造成調試結果被離線數據覆蓋；又或調試完成後沒有執行copy RAM to ROM操作就斷電了，重新上電後設備還處於調試前狀態；這些操作都使剛剛完成的調試工作付之東流。因此瞭解SINAMICS S120的存儲結構以及每一個操作的意義是成功調試的前提。

對SINAMICS S120項目的操作分為兩種：

  • 在線模式（ONLINE Mode）：如果是在線進行操作，這是直接修改RAM裡的數值，而PG/PC本地項目裡的參數並沒有更改。
  • 離線模式（OFFLINE Mode）：如果是離線進行操作，就是對本地項目的參數進行修改，不會影響設備數據。

一、S120存儲器的類別

SINAMICS S120 存儲器分為兩個部分：

  • RAM：易失性存儲器，數據斷電即丟失，RAM位於控制單元內部，是設備自帶的。
  • ROM：即CF卡，數據可以永久保持在CF卡上，CF卡是單獨訂購的。

CF卡是S120的控制單元（CU，Control Unit）工作時必需的存儲設備，其中保存了固件（Firmware）、用戶數據、授權等信息，在斷電的情況下，數據也能夠永久保存。在STARTER軟件中通過菜單命令"Copy RAM to ROM"即可將數據存儲在CF卡上。如圖1所示，為CF卡的外觀及其版本信息。

上述提到的RAM和ROM都是S120上的存儲介質，在我們調試S120項目的時候，還要用到的存儲介質為PG/PC，它保存的是項目的離線參數信息。要正確的掌握S120的調試，首先要弄明白這三個存儲介質之間的關係和操作方式，如圖2所示。

二、PG/PC、RAM及ROM之間的操作

如圖2所示，PG/PC、RAM和ROM之間可進行的操作大致分為以下5種：

1、下載Download：PG/PC to RAM

下載操作是將PG/PC中保存的參數表下載到RAM中(Download CPU/ drive unit to target device)，就是在線執行下載操 作，將PG/PC的參數設置傳給控制單元CU，如圖3所示。

Starter工具欄上有兩個下載按鈕。左邊的黃色下載按鈕是對該項目裡所有在線設備的所有數據進行下載。右邊的下載按鈕是對所選在線設備的所有數據進行下載，一般選擇右邊的下載按鈕。

2、上裝Upload：RAM to PG/PC

Upload是Download的逆操作，它是將控制單元裡的參數設置傳到PG/PC，如圖3所示。

3、Copy RAM to ROM

從RAM到ROM(Copy RAM to ROM)，把參數保存到CF卡上。RAM裡的參數斷電後會丟失，因此雖然配置好的參數已經正確下載到控制單元，但還需要執行該操作來保證掉電後再上電，機器能正常工作，如圖4所示。

如果是通過BOP面板進行調試，調試參數也都存在RAM裡，也需要保存參數的設置。具體操作是當修改完參數面板上出現"S"字樣，按住P鍵保持3秒，面板出現閃爍，表示參數已經開始存儲。或者通過設置CU的參數：P0009=0，P0977=1來執行。

4、Power up：ROM to RAM

系統上電後，CU會自動將保存到CF卡上的參數裝載到RAM。

5、Load to file system：PG/PC to RAM

通過CF卡讀卡器，將參數表直接從PG/PC裝載到CF卡(Load to file system)，這個操作可以在沒有CU的情況下，將S120項目下載到CF卡 上，如圖5所示。

三、調試時應注意的事項

1、系統運行中，訪問的參數是RAM中的參數；

2、系統上電時，會自動的將ROM（即CF卡）中的參數拷貝到RAM中，因此，CF卡中應該存儲最新的、當前在用的參數表。也就是說，在調試完成，設備運行正常後，要執行"Copy RAM to ROM"命令，將最新的參數表寫入CF卡中。

3、Download命令是將PG/PC中保存的離線參數表寫入RAM中，如果在線的參數和離線參數不同，執行"Download"命令，在線參數表會被PG/PC中的離線參數表覆蓋。這是新手最容易犯錯的地方，調試完成後，執行"Download"命令，所有的參數都被離線參數表覆蓋，所有的調試工作都付諸東流。

4、良好的習慣：調試完成後，執行"Upload"命令，將最新的參數表保存於PG/PC中，此為"參數備份"；執行"Copy RAM to ROM"將最新的參數表保存在CF卡中，以便掉電時也不至於是參數丟失。
