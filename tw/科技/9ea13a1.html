各數據庫2種分頁語法支持

2020-10-29
| 科技

電腦小知識，值得收藏

Mycat後端通過jdbc接入oracle、db2、sqlserver、postgresql等通過以下2種方式支持分頁語法

以防找不到可以先收藏哦！

1數據庫原生分頁直接解析

應用端可以通過直接執行原生分頁sql到mycat，各數據庫原生分頁支持如下：

oracle支持三層嵌套和row_number兩種分頁，以及rownum控制最大條數

select * from ( select row_.*, rownum rownum_ from ( select sid

from test where sts<>'N' order by sid desc ) row_ where rownum

<= 15) where rownum_ > 5;

SELECT *

FROM (SELECT sid, ROW_NUMBER() OVER (ORDER BY sid ) AS ROWNUM1

FROM test t

WHERE sts <> 'N'

) XX

WHERE ROWNUM1 > 5

AND ROWNUM1 <= 15;

SELECT * FROM (SELECT * FROM test t) XX WHERE ROWNUM <= 5;

db2支持rownumber分頁和fetch first rows only語法

SELECT *

FROM (SELECT sid, ROW_NUMBER() OVER (ORDER BY sid ) AS ROWNUM

FROM test t

WHERE sts <> 'N'

) XX

WHERE ROWNUM > 5

AND ROWNUM <= 15;

SELECT sid

FROM test

ORDER BY sid desc

FETCH FIRST 15 ROWS ONLY;

Sqlserver支持row_number和row_number與top結合2種分頁以及top限制最大條數

SELECT *

FROM (SELECT sid, ROW_NUMBER() OVER (ORDER BY sid DESC) AS ROWNUM

FROM test

WHERE sts <> 'N'

) XX

WHERE ROWNUM > 10

AND ROWNUM <= 25


select * from ( select row_number()over(order by tempColumn)tempRowNumber,* from ( select top 30 tempColumn=0, sid from test where sts<>'N' order by sid )t )tt where tempRowNumber>20;

postgresql分頁

select sid from test order by sid desc limit 10 offset 5;

等價於mysql的

select sid from test order by sid desc limit 5,10;

其實mysql也兼容limit offset寫法

2Limit語法自動轉換原生分頁

select sid from test order by sid desc limit 5,10;

通過將標準的limit語法自動翻譯轉換為各數據庫的原生分頁，具體可以通過explain命令查看。
