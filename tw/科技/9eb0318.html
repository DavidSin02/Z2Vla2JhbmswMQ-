甘草提取物

光果甘草根提取物是從豆科植物光果甘草（學名：Glycyrrhiza Glabra 英文名：Lycoris）的根中提取得到的提取物。

光果甘草根提取物由於是天然成分，其成分組成會隨著國家、地域和時期的不同發生變化，主要由以下成分構成：

甘草亭酸

光甘草定

甘草苷

異甘草苷

甘草西定

甘草根糖苷

三萜系化合物

類黃酮

甘草，物如其名，甜度是砂糖的50倍，自古以來用於傳統醫學。

希臘自然科學家泰奧弗拉斯托斯曾將甘草根用於治療哮喘、乾咳和肺病，中醫也將甘草用於治療咽痛、哮喘、胃潰瘍和十二指腸潰瘍，在藥方中起協調作用。

在藥理作用方面，具有消炎、祛痰、緩和粘膜刺激的作用，近年來被發現具有抗病毒作用和免疫活化作用。

甘草的作用中心是三萜皁甙的甘草甜素，協同其他成分提高生物活性。

添加到化妝品中時，因其具有非常優秀的抗炎症作用，還有緩和刺激、促進創傷癒合、抗過敏、抗菌、抑制色素沉著作用，經常添加到皮膚護理化妝品、頭髮護理產品、身體護理產品、手部護理產品、防晒產品、潔面產品、面膜等產品中使用。

光果甘草根提取物的色素沉著抑制作用機制是阻礙酪氨酸酶活性，首先我們先了解一下色素沉著（黑色素黑化）的方式，如下圖。

經紫外線照射後的皮膚，首先紫外線信息到達基底層的黑素細胞，紫外線信息一到達黑素細胞，酪氨酸酶活性化，與黑素細胞內的酪氨酸結合後變化成多巴，然後活性化後的多巴與酪氨酸酶結合後變化成多巴醌，進而發展為黑化黑色素。

關於光果甘草根提取物的酪氨酸酶活性阻礙作用，1997年通過使用蘑菇酪氨酸酶後光果甘草根提取物的量比較檢測酪氨酸酶的抑制率，含量分別為0μg/mL、3.3μg/mL、333μg/mL的光果甘草根提取物使用蘑菇酪氨酸酶後驗證酪氨酸酶活性阻礙作用，3.3μg/mL和333μg/mL的阻礙率分別是25%和68%。

從結果看來，光果甘草根提取物與其他99種植物提取物相比，酪氨酸酶活性阻礙作用更強。


如前文報告所說（文獻1:2008），光果甘草根提取物是通過較高的酪氨酸酶活性阻礙作用來抑制色素沉著。

另外，光果甘草根提取物的美白作用是通過光甘草定進行的。

實際上使用光果甘草根提取物的產品種類、數量和添加量，請看以下2007-2008年海外調查結果報告。

複合植物提取物——光果甘草根提取物

一種名為プランテージ<ホワイト>的複合植物提取物由以下成分構成：

黃芩根提取物

光果甘草根提取物

棗果提取物

添加效果和目的：

1.抑制紫外線照射後加速黑色素生成的紫外線信息傳遞物質內皮素-1的產生

2.阻礙生物合成黑色素所必需的酪氨酸酶

3.促進已生成的黑色素的排出，防止皮膚黑色素的積累，促進新陳代謝

各種美白要點不同的植物提取物相輔相成，能夠達到抑制黑色素的產生、多方面預防皮膚暗沉和色素沉著的效果，可以從化妝品成分一覽中推測同時使用這些成分的就是プランテージ<ホワイト>。（文獻9:2006）。

一種名為プランテージ<ホワイト>EX的複合植物提取物由以下成分構成：

黃芩根提取物

茵陳篙花提取物

光果甘草根提取物

桑根提取物

棗果提取物

添加效果和目的：

1.抑制紫外線照射後加速黑色素生成的紫外線信息傳遞物質內皮素-1的產生

2.阻礙生物合成黑色素所必需的酪氨酸酶和抑制黑色素的產生

3.抑制已生成的黑色素的轉移

4.促進已生成的黑色素的排出，防止皮膚黑色素的積累，促進新陳代謝

各種美白要點不同的植物提取物相輔相成，能夠達到抑制黑色素的產生、多方面預防皮膚暗沉和色素沉著的效果，可以推測化妝品成分一覽中同時使用這些成分的就是プランテージ<ホワイト>EX。

光果甘草根提取物的安全性

（毒性 刺激性 過敏性）

目前就光果甘草根提取物的安全性來說，基本上沒有皮膚刺激性，有非刺激性和引起極其輕微的眼睛刺激性的可能，沒有出現皮膚過敏的報告，所以安全性上沒有問題。但是，有時會主觀性地感受到輕度刺激（刺痛感）和燃燒感，而且有極小引起接觸性皮炎的可能性，所以還是需要注意一下。

此項結論依據如下。

皮膚刺激性和皮膚過敏性

《Cosmetic Ingredient Review》的《Safety Assessment of Glycyrrhiza Glabra (Licorice) Rhizome/root, Glycyrrhiza Glabra (Licorice) Leaf Extract, Glycyrrhiza Glabra (Licorice) Root, Glycyrrhiza Glabra (Licorice) Root Extract, Glycyrrhiza Glabra (Licorice) Root Juice, Glycyrrhiza Glabra (Licorice) Root Powder, Glycyrrhiza Glabra (Licorice) Root Water, Glycyrrhiza Inflata Root Extract, and Glycyrrhiza Uralensis (Licorice) Root Extract》資料中記載瞭如下內容。

[人體試驗]

在9名受試者的背部塗抹0.02mL含光果甘草根提取物（～900ppm）的實驗物質，進行48小時封閉斑貼試驗，去除貼片30分鐘後對皮膚反應進行評分，觀察到3名受試者出現極其輕微的紅斑。

[人體試驗]

在106名日本人受試者的健康皮膚上用光果甘草根提取物含量為0.0001%的16種產品進行反覆傷害斑貼實驗（HRIPT），誘導階段進行48~72小時共計9次封閉斑貼試驗，經過12~24天未處置時間後，在未處理部位進行48小時激發斑貼實驗，48小時和96小時後對皮膚反應進行評分，結果有1人對1項產品出現接觸性過敏症狀的徵兆，除此之外其他受試者沒有出現任何刺激和過敏徵兆。出現接觸性過敏症狀徵兆的實驗者進行第二次試驗時，再次出現延遲過敏的症狀。

[人體試驗]

自我感覺為敏感肌的30名日本女性受試者和自我感覺為敏感肌的28名白種人、越南、韓國、中國等各色人種的受試者對含有0.0001%光果甘草根提取物的3種產品進行安全性實驗，將其中1種產品從第1天到第14天作為夜間一般護理塗抹在整張臉上，從第15天到第28天各色人種受試者使用第2種產品，日本人受試者使用第3種產品，第14天和第28天進行臨床性皮膚評分，結果客觀上和主觀上皮膚刺激分數都沒有增加。

[人體試驗]

68名日本女性受試者對含有0.0001%光果甘草根提取物的3種產品進行安全性實驗，第1天到第4天在受試者單個眼部區域使用第一種產品，1天2次，另一眼部區域使用第二種產品，1天1次。第5天到第18天，受試者全臉塗抹第一種產品，1天2次。第19天到第32天，受試者在早上全臉塗抹第一種產品，傍晚全臉塗抹第三種產品。第2、3、4、18、32天時進行臨床性皮膚評分，眼部區域經眼科檢查後觀察到臨床性、非常輕微的眼瞼充血，68人中有28名受試者（41%）在兩週內使用其中一種產品後出現輕微的刺痛和燃燒感。再次塗抹新增加的產品時，5名（7%）受試者出現輕度刺痛和燃燒感。通常使用其中一種產品時，臉部出現輕度的主觀感覺可能性較高，敏感皮膚和患有過敏性皮炎的皮膚可能會出現中度~重度的刺激感，客觀上沒有出現炎症。

[人體試驗]

36名女性受試者（敏感肌15人、色素沉著過多14人、雀斑7人）在臉部色素區域塗抹光果甘草根提取物含量為0.8%的淡斑製劑，第4、8、12周進行臨床性評分，結果在觀察期間均未觀察到出現紅斑和乾燥，客觀上和主觀上都沒有增加刺激性。

[人體試驗]

104名無皮膚疾病的健康受試者在誘導階段使用20μL光果甘草根提取物含量為0.0001%的產品進行斑貼試驗，每週3次，連續使用2周，去除各個貼片後直到進行下一個斑貼實驗期間對實驗部位進行評分。經過1~15的休息期間後，在實驗部位進行48小時激發斑貼試驗，去除貼片24小時後對皮膚反應進行評分，結果沒有出現刺激和過敏的徵兆。

[人體試驗]

106名受試者在誘導階段使用含有0.8%的光果甘草根提取物和其他5種植物提取物的載體溶液進行封閉Finn Chamber試驗，每週3次，連續使用3周，經過10~15天休息時間後進行激發斑貼實驗，24小時和48小時後觀察結果，沒有出現刺激和過敏的徵兆。


僅從實驗數據看來，光果甘草根提取物對健康的皮膚和敏感肌基本上都沒有皮膚刺激性和皮膚過敏性，一般情況下基本上沒有皮膚刺激性和皮膚過敏性。

但是，主觀上有時可能會感覺到輕度的刺激感和燃燒感，也有極小引起接觸性過敏皮炎的可能性，需要注意一下。

眼部刺激性

《Cosmetic Ingredient Review》的《Safety Assessment of Glycyrrhiza Glabra (Licorice) Rhizome/root, Glycyrrhiza Glabra (Licorice) Leaf Extract, Glycyrrhiza Glabra (Licorice) Root, Glycyrrhiza Glabra (Licorice) Root Extract, Glycyrrhiza Glabra (Licorice) Root Juice, Glycyrrhiza Glabra (Licorice) Root Powder, Glycyrrhiza Glabra (Licorice) Root Water,Glycyrrhiza Inflata Root Extract, and Glycyrrhiza Uralensis (Licorice) Root Extract》資料中記載瞭如下內容。

[動物試驗]

在6只兔子單眼內滴入0.1mL光果甘草根提取物含量為0.15%的玉米油，不洗眼，24小時、48小時和72小時時進行觀察，實驗物質對兔子眼睛來說沒有刺激性，所以該物質不是眼睛刺激劑。

[動物試驗]

在3只兔子單眼的結膜囊內滴入0.1mL光果甘草根提取物含量為1%的甘油，第3天沒有出現影響，直至觀察到第7天，第1天少量淚液流出後又恢復正常，1小時內觀察到輕微的眼睛腫脹，直到第2天才消除，所有的兔子在1一小時內都觀察到輕度角膜混濁，直到第1天才完全消除，所以該試驗物質有輕微的眼睛刺激性。

[in vitro試驗]

用培養的正常人體角質細胞中生成的角膜樣本，將100μL含量為10%的光果甘草根提取物進行不同時間的暴露，Draize分數都為0，所以光果甘草根提取物沒有刺激性。


僅從實驗數據看來，光果甘草根提取物為非刺激性和具有輕微眼睛刺激性，所以眼睛刺激性方面為非刺激性，有引起輕微眼睛刺激性的可能。

關於化妝品毒性判定百科的毒性判定


就已翻閱的化妝品毒性判斷百科辭典看來，光果甘草根提取物屬於△(∗1)類，安全性方面沒有問題。

∗1 毒性判斷百科辭典將毒性水平分為“無毒”、“△”、“■”、“■■”，2~3個“△”等於1個“■”，“■”是毒性很強的標識，產品的毒性成分“■”累計四個以上，則表示不可使用。

光果甘草根提取物也被分類在抗炎症成分、美白成分、抗菌成分中。
