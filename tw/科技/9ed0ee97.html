邏輯函數中IF函數判斷是與不是，NOT函數對參數值求反

2020-11-14
| 科技

今日和大家繼續分享邏輯函數中的IF和NOT，這兩個函數的應用非常廣泛，很多接觸EXCEL函數的朋友最初都是從IF函數開始的，這個函數在EXCEL中的地位是永遠撼不動的。下面給大家詳細地講解。這是邏輯函數的第三講。

一．IF 判斷是與不是。

函數定義：IF函數一般是指Excel中的IF函數，根據指定的條件來判斷其"真"（TRUE）、"假"（FALSE），根據邏輯計算的真假值，從而返回相應的內容。可以使用函數 IF 對數值和公式進行條件檢測。

使用格式：

IF(logical_test,value_if_true,value_if_false)

格式簡義：IF(條件,與條件一樣時運算這個,與條件不同時運算這個)

參數定義：logical_test條件必需，計算結果可能為 TRUE 或 FALSE 的任意值或表達式

value_if_true可選。logical_test參數的計算結果為TRUE時所要返回的值。

value_if_false可選。logical_test參數的計算結果為FALSE時所要返回的值。

注意點：

a).IF 函數語法具有下列參數(參數:為操作、事件、方法、屬性、函數或過程提供信息的值.) b).2003版最多嵌套7個IF函數.2007版最多嵌套64個IF函數。

c)在多條件情況時，可以多層次套用IF函數，即 IF（條件1，條件1為真的結果，（IF（條件2，條件2為真的結果），...條件n為真的結果，條件n為假的結果）））注意左括號的數量要和右括號的數量一致，否則無法計算。

二．NOT 對參數值求反

函數定義：對參數值求反.當要確保一個值不等於某一特定值時,可以使用NOT函數

使用格式：NOT(logical)

格式簡義：NOT(要判斷的單元格)

參數定義：Logical為一個可以計算出TRUE或FALSE的邏輯值或邏輯表達式.

注意事項：

a)如果邏輯值為FALSE,函數NOT返回TRUE；

b)如果邏輯值為TRUE,函數NOT返回FALSE.

c)NOT函數是用於對參數值求反的一種Excel函數。當要確保一個值不等於某一特定值時，可以使用NOT函數。簡言之，就是當參數值為TRUE時，NOT函數返回的結果恰與之相反，結果為FALSE。比如NOT(2+2=4)，由於2+2的結果的確為4，該參數結果為TRUE，由於是NOT函數，因此返回函數結果與之相反，為FALSE.

c) NOT函數的函數結構為：NOT(logical)，參數Logical 為一個可以計算出 TRUE 或 FALSE 結論的邏輯值或者邏輯表達式。如果A1=10、A2=15，那麼公式"=NOT(A1<A2)"的計算結果為FALSE。其使用方法也與AND函數基本相同.

三 實例

如果產品名稱為洗衣機,且顏色為白色，其調價後為原來單擊加50元,如果產品名稱為洗衣機，且顏色為白色以外的其餘色，其調價後為原來單擊加200元,其餘產品價格不變

在下面的公式中

=IF(NOT(LEFT(A5,3)="洗衣機"),C5,IF(AND(LEFT(A5,3)="洗衣機",NOT(B5="白色")),C5+200,C5+50))

用到了IF函數，NOT函數和AND函數以及IF函數的嵌套。來實現①如果產品名稱為洗衣機,且顏色為白色，其調價後為原來單擊加50元,②如果產品名稱為洗衣機，且顏色為白色以外的其餘色，其調價後為原來單擊加200元,③其餘產品價格不變。上面共講了三個條件：

①如果產品名稱為洗衣機,且顏色為白色，其調價後為原來單擊加50元,

②如果產品名稱為洗衣機，且顏色為白色以外的其餘色，其調價後為原來單擊加200元,

IF(AND(LEFT(A5,3)="洗衣機",NOT(B5="白色")),C5+200,C5+50)其中判斷條件是AND(LEFT(A5,3)="洗衣機",NOT(B5="白色")) 是一個AND函數，當為真時C5+200；不為真時C5+50，

③其餘產品價格不變IF(NOT(LEFT(A5,3)="洗衣機")，C5)條件NOT(LEFT(A5,3)="洗衣機")為真時，C5成立。

好了上面的公式已經非常詳細的講解了。
