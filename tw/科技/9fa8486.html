【溫馨提示】

　(1)向濾渣中加稀鹽酸，若有氣泡產生，說明濾渣中有活潑金屬；若無氣泡產生，說明濾渣中無活潑金屬。

(2)向濾液中加稀鹽酸，若有白色沉澱產生，說明濾液中有Ag＋；若無明顯現象，說明濾液中沒有Ag＋，Ag在濾渣中。

1．(2019·河北)將一定量的鐵粉加入含有Cu(NO3)2和AgNO3的廢液中，充分反應後過濾，向濾渣中加入稀鹽酸，無氣泡產生。下列判斷正確的是( ）

A．濾渣中一定含有Ag和Cu，濾液中一定含有Fe(NO3)2

B．濾渣中一定不含Fe，濾液中一定含有Fe(NO3)2和Cu(NO3)2

C．濾渣中一定含有Ag，濾液中可能含有Cu(NO3)2和AgNO3

D．濾渣中一定含有Ag和Cu，濾液中可能含有Cu(NO3)2和AgNO3

2．將鋅粉和鐵粉的混合物加入盛有一定量硝酸銀溶液的燒杯中，充分反應後過濾，將所得濾渣放入稀鹽酸中，有氣泡產生。下列說法錯誤的是( )

A．濾液中最多含有兩種溶質

B．濾液中一定含有Zn(NO3)2，可能含有Fe(NO3)2

C．濾渣中一定含有Fe和Ag，可能含有Zn

D．若反應後所得濾液呈無色，則濾渣中一定含有Fe和Ag，一定沒有Zn

3．(2019·齊齊哈爾)向硝酸銅、硝酸銀和硝酸鎂的混合溶液中加入一定質量的金屬鋅，充分反應後，過濾，得到濾渣和濾液，以下推斷正確的是( )

A．若濾液為藍色，則濾渣中一定有銀和銅

B．若向濾液中加入稀鹽酸無白色沉澱生成，則濾渣中一定有銀和銅

C．若向濾渣中加入稀鹽酸有氣泡生成，則濾液中只含有硝酸鎂和硝酸鋅

D．若反應前加入的金屬鋅的質量和反應後濾渣的質量相等，則濾渣中一定不含鋅和鎂

4．(2019·天水)向AgNO3和Mg(NO3)2溶液中加入一定量Zn粉和Cu粉，充分反應後過濾，得到濾渣和藍色濾液。關於該濾渣和濾液說法正確的是( )

A．向濾渣中加入稀鹽酸，可能有氣泡產生

B．濾液中一定有Cu(NO3)2、Mg(NO3)2和Zn(NO3)2

C．濾渣中一定有Ag，可能有Cu和Zn

D．濾液中一定無AgNO3，可能有Zn(NO3)2

5．(2019·武漢)將一定質量的硫酸和硫酸亞鐵的混合溶液加入盛有鋅和氧化銅的混合物的燒杯中，充分反應後過濾，得到濾渣和濾液。關於濾渣和濾液，下列說法正確的是( )

A．濾渣中最多有三種物質

B．若濾液呈無色，濾液中的溶質可能有兩種

D．若濾液中含有硫酸銅，濾渣中可能有兩種物質
