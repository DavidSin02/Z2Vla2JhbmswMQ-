6.18日問題及答案

1、呼吸系統的功能主要是 ( A )

A.進行氣體交換 B.進行水液代謝

C.呼出氧氣，吸入二氧化碳 D.進行血液循環

呼吸，是指機體與外界環境之間氣體交換的過程。人的呼吸過程包括三個互相聯繫的環節：外呼吸，包括肺通氣和肺換氣；氣體在血液中的運輸；內呼吸，指組織細胞與血液間的氣體交換。正常成人安靜時呼吸一次為6.4秒為最佳，一個呼吸分為三個部分：呼氣、屏息、吸氣。

2、氣管在4、5胸椎處分成 ( B )

A.上下主支氣管 B.左右主支氣管

C.前後主支氣管 D.大小主支氣管

3、鼻腔附近含有空氣的骨腔叫做 ( B )

A.鼻道 B.鼻旁竇 C.鼻甲 D.鼻前庭

鼻旁竇，又稱副鼻竇或鼻竇，為鼻腔周圍顱骨（額骨、蝶骨、上頜骨、篩骨）內的含氣空腔的總稱，均有竇口與鼻腔相通。對發音起共鳴作用。鼻竇左右成對，共四對，分別稱為額竇、上頜竇、蝶竇和篩竇。

4、牛奶浴下列那種食品能起到協調作用（ B ）

A 蘋果 B 雞蛋 C 菜花 D 橘子

雞蛋清可膨脹潤澤皮膚角質層，使毛孔擴張，促進面部血液循環，增強皮膚對營養物質及生物活性因子的吸收。

5、 俗稱“人造小太陽”的強光燈所採用的氣體是（ D ）

A 氬氣 B 氖氣 C 氦氣 D 氙氣

稀有氣體在電光源中有特殊的應用。在燈管裡充入氖氣的氖燈，通電時發出的紅色光，能透過濃霧，可用作航空、航海的指示燈。在燈管裡充入氬氣的氬燈，通電時發出藍紫色光；在燈管裡充入氦氣的氦燈，通電時發出粉紅色光；在不同材質的玻璃燈管裡充入不同含量的氦、氖、氬的混合氣體，就能製得五光十色的霓虹燈。在燈管裡充填少量的汞和氬氣，燈管的內壁塗上熒光物質，通電時就能發出近似日光的可見光，所以叫日光燈。充填氙氣的高壓長弧燈，通電時能發出比熒光燈強幾萬倍的強光，因此叫做“人造小太陽”，可用於廣場、體育場、飛機場等照明。

6、丁香、垂柳萎靡不振，出現“白斑病”說明空氣中有哪種氣體汙染？（D ）

A、SO2 B、CO2 C、NO2 D、臭氧

臭氧汙染會讓植物葉片壞死、脫落、長漂白斑、生長受抑制，從而造成農作物減產。

7、“桃花潭水深千尺，不及汪倫送我情”詩中的“我”指的是誰？（ B ）

A、杜甫 B、李白 C、白居易 D、杜牧

《贈汪倫》是唐代大詩人李白於涇縣遊歷桃花潭時寫給當地好友汪倫的一首留別詩。

贈汪倫

作者：李白

李白乘舟將欲行，忽聞岸上踏歌聲。桃花潭水深千尺，不及汪倫送我情。

8、下面哪句話出自《孟子》？（ D ）

A、水能載舟，亦能覆舟

出自《諫太宗十思疏》魏徵

B、先天下之憂而憂，後天下之樂而樂

范仲淹《岳陽樓記》

C、民惟邦本，本固邦寧

出自《尚書·五子之歌》

D、獨樂樂，與人樂樂，熟樂？

獨樂樂不如眾樂樂出自《孟子.梁惠王下》

9、文人對各種植物有自己的偏愛，陶淵明喜愛菊花，北宋文豪蘇東坡則說過：寧可食無肉，不可：（ B ）

A、居無鬆 B、居無竹 C、居無梅 D、居無菊

出自北宋蘇軾《於潛僧綠筠軒》，全詩為：

寧可食無肉，不可居無竹。無肉令人瘦，無竹令人俗。

人瘦尚可肥，士俗不可醫。旁人笑此言，似高還似痴。

若對此君仍大嚼，世間那有揚州鶴？

譯文：

寧可沒有肉吃，也不能讓居處沒有竹子。沒有肉吃不過人會瘦掉，但沒有竹子就會讓人變庸俗。原因是人瘦還可變肥，人俗就難以醫治了。旁人若果對此不解，笑問此言：“似高還似痴？”

那麼請問，如果面對此君（竹），仍然大嚼，既要想得清高之名，又要想獲甘味之樂，世上又哪來“揚州鶴”這等魚和熊掌兼得的美事呢？

10、兩頭牛的前面有一頭牛，兩頭牛的後面也有一頭牛，兩頭牛的中間又有一頭牛請問到底有多少頭牛。 答案：三頭

11、兩側聲帶之間的裂隙稱為( D )

A.咽腔 B.顎腔 C.前庭裂 D.聲門

解剖學結構：聲帶和聲門裂合稱聲門，聲門也是腸胃和氣管之間鏈接的重要過度器官。

12、氣管軟骨，具有彈性，使管腔保持開放，這種作用主要表現為（C）

A.呼吸作用 B.彈性作用 C.支架作用 D.固定作用

13、支氣管、血管、淋巴管、神經出入於肺的地方稱為 (B)

A.縱隔 B.肺門 C.肺尖 D.肺底

肺門位於肺縱隔面心壓跡後上方的凹陷。為支氣管、血管、神經、淋巴管的出入口。

14、下列哪個試劑可除去圓珠筆（A ） 。

A．丙酮 B、酒精 C、鹽水 D、醋

圓珠筆的墨水是油墨，難容於水，易溶於有機溶劑，丙酮是最簡單的酮類化合物常溫下，丙酮是無色透明的液體，易揮發，具有令人愉快的氣味，能與水、乙醇等互溶，是一種重要的有機溶劑和化工原料。

15、以下哪個選項中的糖按比例混合後甜度最高 （B）

A蔗糖、葡萄糖 B果糖、葡萄糖

C果糖、蔗糖 D果糖、乳糖

說明：果糖是甜度最高的，是葡萄糖的2.3倍，是蔗糖的1.7倍

16、醫療上用作收斂劑，可使機體組織收縮，減少腺體的分泌的物質是（B ）

A．明礬 B．皓礬 C．藍礬 D．綠礬

常溫下七水硫酸鋅為白色顆粒或粉末，正交晶體，有收斂性，是常用的收斂劑

17、為了達到“天地與我並生”、“萬物與我為一”這一超越時空限制的精神境界，追求絕對自由的精神滿足，歷代封建知識分子在（B ）中去獲取精神寄託。

A.《老子》 B.《莊子》 C.《周易》 D.《論語》
