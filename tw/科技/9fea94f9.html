強大腳本功能助您一臂之力-Total Control 通過POST執行 JS API

2020-10-29
| 科技

你還沒開始用手機群控軟件做推廣嗎？做客服系統嗎？做工作室嗎？做手機測試嗎？那你真該好好反省一下了。現在我來講講強大的群控軟件的腳本功能，讓你用起來更得心應手。

Total Control 是一款手機多控軟件，可以通過電腦控制一臺到100臺手機。擁有強大的腳本功能。

Total Control既提供 REST API，也提供了 JS API，用戶們可以根據自己需求選擇接口來實現功能，相比之下 Total Control 提供的JS API 更豐富，功能更強大，為了方便用戶，在6.9.0版本Total Control 可以通過POST執行 device.<function> 的 JS API。

如果用戶想通過 POST 調用 JS API, 可以使用下面介紹的 REST API 在 Total Control 上執行 JS API。

請求方式：POST

http://IP:8090/TotalControl/v1/devices/:device/scripts?cmd=:cmd&param=:parm&token=:token

參數說明：

參數名

類型

必選

描述

IP

string

Y

訪問 Total Control 服務器的 IP 地址

token

string

Y

Total Control TOKEN

cmd

string

Y

要執行的 JS API 命令對應的方法名稱

param

string

N

要執行的 JS API 命令對應的方法參數,多個參數使用’[]’,例如：[0.5, 0.25]）

返回示例：

成功，則返回:

{ "status": true, "value": "" }

字段

類型

描述

status

boolean

true：成功 false：失敗

value

string

JS API 的返回信息

示例：通過 POST執行JS API “send(sigmaConst.KEY_MENU)"

注意：下列示例是通過本機訪問 Total Control 服務器，所以 IP 都是用的 "localhost"。

RingoJS 請求示例：

可以複製下面的代碼，在 Total Control 的腳本終端中執行，也可以保存為後綴為 js 的文件, 例如:example.js，在Total Control 執行器中執行。

//導入一些 ringoJS 的包var {request} = require('ringo/httpclient');var base64 = require('ringo/base64');//將用戶名密碼編碼，'sigma:3D391497'是用戶名和密碼,在Total Control系統設置裡面可以看到var userpass = base64.encode('sigma:3D391497');//第一步：獲取 Total Control 腳本開發所用的開發 tokenvar gettoken = request({ method: 'GET', url: 'http://localhost:8090/TotalControl/v1/login', headers: {'Authorization': userpass} });//將獲取的響應內容轉換成json對象var ret_token = eval('(' + gettoken.content + ')'); var token = ret_token.value.token; print("//獲取 token 的值為：" + token);//第二步：獲取當前主控設備 ID 值var getdevice = request({ method: 'GET', url: 'http://localhost:8090/TotalControl/v1/devices/main?token=' + token });var device = eval('(' + getdevice.content + ')').id; print("//獲取到設備的 ID："+ device);//第三步：通過 POST 調用 JS APIvar ret = request({ method: 'POST', url: 'http://localhost:8090/TotalControl/v1/devices/'+ device+'/scripts', data: { "token": token, "cmd":"send", "param":"sigmaConst.KEY_MENU" } });print(ret.content);
