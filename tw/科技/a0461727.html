想要掙錢創收 那就用腳本操作手機群控軟件啊

2020-10-29
| 科技

很多人都在貼吧以及群裡吐槽Total Control的腳本難用，易用性太差。我覺得這點TC確實應該反省一下，畢竟軟件使用者很多都不是腳本掌握很好的開發人員。但是Total Control 真是一款神奇的軟件，有了它，不僅可以用腳本實現自動打開安裝卸載手機軟件等功能，還可以用腳本實現在多個手機上同時自動打開卸載手機上的軟件。


我今天就來分享一下用其他的腳本語言的使用實例。想用腳本實現安裝卸載手機軟件，是不是很高級，但是我不會JavaScript 怎麼辦？沒關係，Total Control 提供的 REST API 支持任意語言，易語言，python，java，都可以的哦！要實現在多個手機上同時打開、關閉指定應用程序，首先我們得用電腦連接多個手機，然後使用 Total Control 提供的 REST API 控制手機，使用Total Control REST API 的一般步驟：

1. 獲取 Total Control 腳本開發所用的開發 token

2. 獲取多個設備 ID 值，例如:{"ids":["device@33254183","device@230441652"]}

3. 通過第二步的多個設備 ID 值（多設備）,對獲取的設備進行操作

（一） Total Control 提供的REST API 獲取多臺設備 ID 集合的方法：

1. 獲取所有設備 ID 集合

請求方法：GET

請求 URL：

http://127.0.0.1:8090/TotalControl/v1/devices?q=all&token=270eq7lXQK8bXYsJ

2. 根據給定的組名稱獲取組內所有設備 ID 集合

請求方法：GET

請求 URL：

http://127.0.0.1:8090/TotalControl/v1/devices?q=group&name=group1&token=270eq7lXQK8bXYsJ

3. 自動連接設備，對當前準備好的設備自動連接

請求方法：POST

請求 URL：

http://127.0.0.1:8090/TotalControl/v1/devices/connections?token=270eq7lXQK8bXYsJ

上面三種獲取設備 ID 集合的方法返回值如下：

{

"ids":[

"device@795844152",

"device@795812215"

]

}
