讓在異地的兩臺電腦、手機組成局域網 實現文檔、圖片、視頻共享

2020-11-14
| 科技

蒲公英 是一款真正的 VPN 應用，用來快速建立一個虛擬局域網，將位於異地的電腦、手機、服務器、NAS 連接起來，實現共享文檔、圖片、視頻內容的工具。

不要談 VPN 色變，正經的 VPN 應用早已廣泛應用於各大企業，分支機構互聯互通、連鎖店信息共享、遠程移動辦公等等，而隨著個人數碼設備的增多，也開始在個人用戶間流行起來。

比起廣域網，局域網的好處在於共享內容的便利性，你可以直接通過 Windows 自帶的遠程控制軟件連接局域網中的 Windows 電腦，而如果在局域網以外，由於受到公網 IP、端口限制，是沒有辦法直接連接的，你需要儲備不少的網絡知識才能完成。

通過 蒲公英 將位於異地的設備連接之後，就和在家中使用的網絡一樣，是一個局域網了。

異地組建虛擬局域網只需兩步

蒲公英的各平臺客戶端都非常簡單應用，以 Windows 為例，在下載安裝後，先要在蒲公英官網註冊，然後去登錄：

登錄之後會自動虛擬網卡，這個時候如果長時間刷不出來，可能就是你的 DNS 問題了，蒲公英建議使用騰訊 dnspod 的公共 DNS 服務：

119.29.29.29

之後就能正常發現虛擬網卡了，而使用同一個賬號在其他設備登錄蒲公英以後，都會默認加入一個局域網：

蒲公英支持在後臺創建多個局域網。

不過對於免費用戶，創建多個網絡可能意義不大

在多臺設備都運行蒲公英客戶端，並且登錄成功之後，一個小巧的、跨區域的虛擬局域網就組建成功了。

文件共享與遠程訪問

你可以在蒲公英的客戶端上直接看到每臺設備的 IP 地址，進而實現局域網內的共享功能。雙擊客戶端上的設備名稱，就會自動彈出命令提示符進行 ping 操作。

其實2T在局域網中的需求，主要是互相間傳輸資料以及文件共享。對家裡有礦的朋友來說，將礦機接入局域網後，可以更方便的遠程管理於操作。

在手機端連接蒲公英後，可以直接通過微軟的 Remote Desktop 連接到 Windows 系統上，非常方便：

而局域網內的文件共享就太簡單了，在 Windows 下對著文件夾屬性共享就行了。

這之後，只需要跑到另外一臺機器上，在 開始 – > 運行 裡輸入：

\\172.10.54.36 （你要控制的電腦的蒲公英 IP 地址）

注意前面的兩個 “\”，是必須的。

然後就能在文件管理器中看到之前共享的 tools 文件夾啦。

注意如果蒲公英提示機器過多不能登錄，就需要前往網頁後臺刪除不需要的設備，免費賬號最多支持三臺設備。

而如果家中有 NAS，比如黑群暉之類，還可以購買一臺蒲公英的路由器，這樣就能將整個局域網都帶入蒲公英的虛擬局域網之中了。
