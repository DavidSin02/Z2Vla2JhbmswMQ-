基於ThinkPHP6和Element2的極簡後臺管理系統

2020-10-29
| 科技

簡介

yylAdmin 是一個極簡後臺管理系統，只有登錄退出、權限管理、日誌管理等基本功能，方便擴展；前後端分離，後端採用ThinkPHP6，前端採用Element2；項目由後端yylAdmin和前端yylAdminWeb組成。

開發

PHP部分

# 克隆項目git clone https://gitee.com/skyselang/yylAdmin.git# 進入項目目錄cd yylAdmin# 安裝依賴composer install# 可以通過composer鏡像解決速度慢的問題composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/# 配置環境（PhpStudy）# 導入數據庫數據庫文件：public/private/yyladmin.sql

WEB部分

# 克隆項目git clone https://gitee.com/skyselang/yylAdminWeb.git# 進入項目目錄cd yylAdminWeb# 安裝依賴npm install# 可以通過npm鏡像解決速度慢的問題npm install --registry=https://registry.npm.taobao.org# 啟動服務npm run dev

在 .env* 環境變量文件 修改接口地址

瀏覽器訪問 http://localhost:9527

賬號：yyladmin，密碼：123456

發佈

# 構建測試環境npm run build:stage# 構建生產環境npm run build:prod

其它

# 預覽發佈環境效果npm run preview# 預覽發佈環境效果 + 靜態資源分析npm run preview -- --report# 代碼格式檢查npm run lint# 代碼格式檢查並自動修復npm run lint -- --fix
