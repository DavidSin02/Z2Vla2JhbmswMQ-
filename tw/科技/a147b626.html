認識Python函數的兩個概念：形參與實參（16）

2020-11-14
| 科技

我們先用Python編寫一個比較兩個數字大小的函數pMax()。

def pMax(a,b): '''比較兩個數字的大小，並返回較大的數字''' if a < b: print("{0}與{1}的較大數字是{2}。".format(a,b,b)) else: print("{0}與{1}的較大數字是{2}。".format(a,b,a))

形參

從上面的函數定義中，我們把函數pMax小括號中的a和b當作函數的形式參數，簡稱形參。

函數封裝的代碼中通過IF...ELSE...選擇判斷條件來得到兩個形參a和b所代表的數字大小。

1、如果a < b，則b是較大值；

2、否則，a是較大值。

實參

定義函數的目的是使用，現在我們來看看調用函數pMax的效果。

首先是通過pMax(90,67)與pMax(50,67)類似方式來調用函數，從執行結果可以推導出：

1、調用函數前必須要用到函數名，也就是函數對象到內存中匹配到封裝的執行代碼；

2、如果函數設置了形參，調用時也必須要有對應形參的實際值，這就是實參；

3、根據實參的位置映射到形參，之後根據函數封裝的代碼邏輯判斷兩個實參大小，並輸出較大值的實參。

我們還需要問的一個問題是：實參與形參的個數是否要吻合？

通過調用函數實例pMax(50,67,98)可以看到Python打印出報錯信息：

pMax() takes 2 positional arguments but 3 were given

所以，形參與實參個數必須一致。

總結

1、形參只是在函數定義時使用；

2、實參是在函數調用時賦予的實際數值；

3、實參與形參是通過位置映射的；

4、形參與實參個數必須一致。
