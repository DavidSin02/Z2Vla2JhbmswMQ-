變壓器鐵芯的優點及影響質量的因素有哪些？

2020-11-14
| 科技

影響變壓器鐵芯質量的因素:

① 設計 ② 原材料：牌號、厚度不均、絕緣膜、波浪度。 ③ 剪切質量：毛刺、片寬、片長、角度。 ④ 疊裝質量：接縫、器身墊腳高度、心柱級支撐塊、夾緊、紮緊、心柱垂直度、地面水平度、磕碰摔打、片彎曲半徑。

主要設備工具：縱剪線、橫剪線、剪床、衝床、自動疊裝機、料料疊裝臺、千斤頂、定位棒盤、扳手、夾具、收緊帶電木板水平尺吸塵器、外卡鉗、盤升降架、 C 型夾具、液壓機、液壓缸等。

鐵心：鐵心擔負磁路重任，也是變壓器的骨架，產品各部件固定（繞組、絕緣件、引線、開關等）。

夾件：（對夾件製造的要求） ① 具有足夠的強度 ② 焊線強度 ③ 加工精度 ④ 焊接處不可形成“袋狀” ⑤ 月支板之間必須垂直 ⑥ 壓釘螺母焊後的中心線垂直肢板

⑦ 高低壓下夾件肢板必須在同一平面上，尤其是 A 、 C 端部 ⑧ 壓釘螺母要用三錐過一次 ⑨ 連接部位不塗漆，保證接觸良好 ⑩ 夾件倒角

鐵心選片的優點： ① 提高疊片工序的效率 ② 檢查廢片提前補充不足片，保證每級的厚度和層數

鐵心疊裝工藝要點準備：夾件、夾件絕緣、枕木、橫樑、拉板、壓釘、墊腳絕緣等。 ( 1 ) ．放置疊片：放在液壓升降臺上 ( 2 ) ．布臺：找平，使鐵心中心對準臺中心 ( 3 ) ．放千斤頂位置，留拉帶、夾緊空間 ( 4 ) ．將千斤頂固定在臺上 ( 5 ) ．墊腳與短臂距離

鐵心必須一點接地鐵心與金屬件必須相連接接地，使同一電位（零電位），鐵心兩點接地相當於鐵心兩端短路，會產生一定電流，導致損耗增加，嚴重時會熔斷接地片，鐵心懸浮絕不允許。
