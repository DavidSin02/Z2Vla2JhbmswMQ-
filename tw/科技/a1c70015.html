cpp函數的變量的存儲類別詳解

2020-11-14

變量的存儲類型

存儲類型是變量的屬性之一，C++定義了4種變量的存儲類型：auto變量、static變量、register變量、extern變量。

  1. 變量存儲方式不同會使變量的生存期不同，生存期表示了變量存在的時間。生存期和變量作用域是從時間和空間這兩個不同的角度來描述變量的特性。
  2. 靜態存儲變量通常是在變量定義時就分配固定的存儲單元並一直保持不變，直至整個程序結束。
  3. 如全局變量，他們存放在靜態存儲區中。
  4. 動態存儲變量是在程序執行過程中使用它時才分配存儲單元，使用完畢立即將該存儲單元釋放。例：形式參數，在函數定義時並不給形參分配存儲單元，只是在函數被調用時才予以分配的，調用函數完畢立即釋放，此類變量存放在動態存儲區中。

即：

靜態存儲變量是一直存在的，動態存儲變量則時而存在時而消失。

1.auto變量

函數內未加存儲類型說明的變量均視為自動變量，可省去關鍵字auto。

{int i ,j ,k ;auto int i,j,k ; //等價的。}

特點：

  1. 自動變量的作用域僅限定於定義該變量的個體內。在函數中定義的自動變量，只有在該函數內有效；在複合語句中定義的自動變量，只在該複合語句中有效。
  2. 自動變量屬於動態存儲方式。變量分配的內存是在棧中，當函數調用結束後，自動變量的值會被釋放。同樣，在複合語句中定義的自動變量，在退出複合語句後也不能再使用，否則將引起錯誤。
  3. 由於自動變量的作用域和生存期都侷限於定義它的個體內（函數或複合語句內），因此不同的個體中允許使用同名的變量而不會混淆。即使在函數內定義的自動變量也可以與該函數內部的複合語句中定義的自動變量同名。

實例： 輸出不同生存期的變量值。

#include<iostream>using namespace std;void main(){auto int i,j,k;cout <<"input the number :"<<endl;cin>>i>>j;k=i+j;if(i!=0&& j!=0){auto int k;k=i-j;cout<<"k:"<<k<<endl; //輸出變量 k的值}cout<<"k:"<<k<<endl; //輸出變量K的值}

2.static變量

在聲明變量時加關鍵字static，可以將變量聲明為靜態變量。靜態局部變量的值在函數調用結束後不消失，靜態全局變量只能在本源文件中使用。

static int a,b ;static float x,y;static int a[3] ={0,1,2};

靜態變量屬於靜態存儲方式。

特點：

  1. 靜態變量在函數內定義，在程序退出時釋放，在程序整個運行期間都不釋放，也就是說他的生存期為整個源程序。
  2. 靜態變量的作用域與自動變量相同，在函數內定義就在函數內使用，儘管該變量還繼續存在，但不能使用它，如再次調用定義它的函數，他又可以繼續使用。
  3. 編譯器會為靜態局部變量賦予0值。

實例：使用static變量實現累加。

#include<iostream>using namespace std ;int add(int x) {static int n=0; //靜態變量n=n+x;return n;}void main(){int i ,j ,sum;cout<<"input the number :" <<endl;cin>>i;cout << "the result is :"<<endl;for (j=1; j<=i;j++){sum = add(j);cout<<j<<":"<<sum<<endl;}system("pause");}

3.register變量

通常變量的值存放在內存中，當對一個變量頻繁讀寫時，需要反覆訪問內存存儲器，此時將花費大量的存取時間。為了提高效率，C++語言可以將變量聲明為寄存器變量，這種變量僵局不變來那個的值存放在CPU中的寄存器中，使用時不需要訪問內存，而直接從寄存器中讀寫。寄存器變量的聲明符是register。

注：

  • 寄存器變量屬於動態存儲方式。凡是需要採用靜態存儲方式的變量不能定義為寄存器變量。
  • 編譯程序會自動決定哪個變量使用寄存器存儲。
  • register可以起到程序優化的作用。

4.extern變量

在一個源文件中定義的變量和函數只能被本源文件中的函數調用，一個C++程序中會有許多源文件，可用extern變量使用非本源文件中的全局變量。

C++提供了extern 關鍵字來解決這個問題。在使用其他源文件中的全局變量時，只需要在本源文件中使用extern關鍵字來聲明這個變量即可。

實例：在app.cpp源文件中定義全局變量a、b、c代碼如下：

int a , b; //外部變量定義char c ; // 外部變量定義void main(){cout <<a<<endl;cout<<b<<endl;cout<<c<<endl;}

在app2.cpp源文件中要使用app.cpp中的全局變量 abc,代碼如下：

extern int a,b;//外部變量說明extern char c;//外部變量說明func(int x,y){cout<<a<<endl;cout<<b<<endl;cout<<c<<endl;}

注：

在app2.cpp源文件中，編譯系統不再為全局變量a、b、c分配內存空間，而是改變全局變量a、b、c的值，在app2.cpp源文件中輸出值也會發生變化。
