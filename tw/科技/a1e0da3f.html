Spire.Cloud.PDF 添加、提取PDF文本

2020-10-29
| 科技

Spire.Cloud.PDF提供了PdfTextApi接口來操作PDF文檔中的文本內容。本文將使用該接口來介紹如何給PDF文檔添加新的文本及提取文檔中已有的文本內容。具體操作步驟和代碼演示參見下文。

步驟一：創建一個maven程序，並通過maven倉庫安裝Spire.Cloud.SDK.jar包及其依賴。

步驟二：通過冰藍雲官網（https://cloud.e-iceblue.cn/）註冊賬號並登陸，在“我的應用” 版塊創建應用程序，以獲得App ID及App Key。

步驟三：上傳PDF示例文檔至冰藍雲官網的“文檔管理”版塊。

步驟四：在Maven程序中編寫代碼調用PdfTextApi接口來給PDF示例文檔添加新的文本及提取其已有文本內容。

添加新文本到PDF文檔

import spire.cloud.pdf.sdk.*;import spire.cloud.pdf.sdk.api.PdfTextApi;import spire.cloud.pdf.sdk.model.*;public class AddText {    private static String appId = "App ID";    private static String appKey = "App Key";    private static String baseUrl = "https://api.e-iceblue.cn";    public static void main(String[] args) throws ApiException {        //配置App ID和App Key        Configuration PdfConfiguration = new Configuration(appId, appKey, baseUrl);        //創建PdfTextApi實例        PdfTextApi PdfTextApi = new PdfTextApi(PdfConfiguration);        //PDF示例文檔名稱        String name = "Sample.pdf";        //指定結果文檔名稱        String outPath = "AddText_output.pdf";        //指定需添加文本的頁碼        int pageNumber =1;        //示例文檔所屬文件夾，若無，則null        String folder = null;        //示例文檔打開密碼，若無，則null        String password = null;        //示例文檔所屬內存，冰藍雲提供2G的免費存儲空間        String storage = null;        //指定需添加的文本內容        Text text = new Text();        text.setValue("Chapbook");        //設置文本字體格式        Font font = new Font();        font.setFontType(Font.FontTypeEnum.STANDARD);        font.setFontFamily("Helvetica");        font.setFontSize(15f);        font.setFontStyle(Font.FontStyleEnum.REGULAR);        text.setFont(font);        //指定文本所在位置        RectangleF rectangleF = new RectangleF();        rectangleF.setX(20f);        rectangleF.setY(20f);        rectangleF.setWidth(800f);        rectangleF.setHeight(800f);        text.setRect(rectangleF);        //設置文本字體顏色        Color backColor = new Color();        backColor.setA(1);        backColor.setR(120);        backColor.setG(182);        backColor.setB(193);        text.setBackgroundColor(backColor);        //調用addText方法給示例文檔添加新的文本        PdfTextApi.addText(name, outPath, pageNumber, text, folder, storage,password);    }}

提取PDF文本內容

import spire.cloud.pdf.sdk.*;import spire.cloud.pdf.sdk.api.PdfTextApi;public class ExtractText {    private static String appId = "App ID";    private static String appKey = "App Key";    private static String baseUrl = "https://api.e-iceblue.cn";    public static void main(String[] args) throws ApiException {        //配置App ID和App Key        Configuration PdfConfiguration = new Configuration(appId, appKey, baseUrl);        //創建PdfTextApi實例        PdfTextApi PdfTextApi = new PdfTextApi(PdfConfiguration);        //PDF示例文檔名稱        String name = "Sample.pdf";        //指定結果文檔名稱        String outPath = "ExtractText_output.txt";        //指定需提取文本的頁碼        int pageNumber = 1;        //示例文檔打開密碼，若無，則null        String password = null;        //示例文檔所屬內存，冰藍雲提供2G的免費存儲空間        String storage = null;        //示例文檔所屬文件夾，若無，則null        String folder = null;        //調用extractText方法來提取示例文檔中的文本信息        PdfTextApi.extractText(name, outPath, pageNumber, folder, storage, password);    }}
