DAY3-step4 Python字符串：替換，連接，拆分，反向，大寫和小寫

2020-11-14
| 科技

在Python中，一切都是對象，字符串也是對象。 只需將字符括在雙引號中即可創建Python字符串。

例如：

var = "Hello World!"

訪問字符串中的值

Python不支持字符類型，字符類型被視為長度為一的字符串。

我們使用方括號和一個或多個索引進行切片以獲得子字符串。

var1 = "Guru99!"var2 = "Software Testing"print ("var1[0]:",var1[0])print ("var2[1:5]:",var2[1:5])

Output

var1[0]: Gvar2[1:5]: oftw

各種字符串運算符

有各種各樣的字符串運算符可以以不同的方式使用，例如連接不同的字符串。

假設如果a = "guru"和b = "99"，則a + b =“ guru99”。 同樣，如果您使用a * 2，它將會是字符串“ GuruGuru”。 同樣，您可以在字符串中使用其他運算符。



更多例子

您可以通過將變量重新分配給另一個字符串來更新Python字符串。 新值可以與以前的值相關，也可以與完全不同的字符串相關。

x = "Hello World!"print(x[:6]) print(x[0:6] + "Guru99")

Output

HelloHello Guru99

注意：切片:6或0:6具有相同的效果

Python字符串replace()方法

方法replace（）返回字符串的副本，其中舊字符串的值已替換為新值。

oldstring = 'I like Guru99' newstring = oldstring.replace('like', 'love')print(newstring)

Output

I love Guru99

更改大小寫字符串

在Python中，您甚至可以將字符串更改為大寫或小寫。

string="python at guru99"print(string.upper())

Output

PYTHON AT GURU99

同樣，您也可以執行其他功能，例如capitalize,將字符串的第一個字母變成大寫,其他字母變小寫

string="python at guru99"		print(string.capitalize())

輸出

Python at guru99

您也可以將字符串轉換為小寫

string="PYTHON AT GURU99"print(string.lower())

Output

python at guru99

對字符串使用“ join”功能

join函數是連接字符串的更靈活的方法。 使用join功能，您可以將任何字符添加到字符串中。

例如，如果要在字符串“ Python”中的每個字符後添加一個冒號（:)，則可以使用以下代碼。

print(":".join("Python"))

Output

P:y:t:h:o:n

翻轉字符串

通過使用reverse函數，您可以反向字符串。 例如，如果我們有字符串“ 12345”，然後將代碼應用reverse函數，如下所示。

string="12345"		print(''.join(reversed(string)))

Output

54321

分割字符串

讓我們來看看字符串“ guru99 career guru99”。 首先在這裡，我們將使用命令word.split拆分字符串並獲得結果。

word="guru99 career guru99"		print(word.split(' '))

Output

['guru99', 'career', 'guru99']

為了更好地理解這一點，我們將再看一個split的示例，而不是空格（' '），我們將其替換為（'r'），並且它將在字符串中任何有'r'的地方拆分字符串

word="guru99 career guru99"		print(word.split('r'))

Output

['gu', 'u99 ca', 'ee', ' gu', 'u99']

重要的提示：

在Python中，字符串是不可變的。

考慮以下代碼

x = "Guru99"x.replace("Guru99","Python")print(x)

Output

Guru99

仍會返回Guru99。 這是因為x.replace（“ Guru99”，“ Python”）返回 X副本並已進行替換

您將需要使用以下代碼來看更改的結果

x = "Guru99"x = x.replace("Guru99","Python")print(x)

Output

Python

Python引入.format函數進行字符串的格式化，該函數比使用％d更容易。
