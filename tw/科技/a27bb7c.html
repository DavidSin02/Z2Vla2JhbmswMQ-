關於華為EMUI 10.0和Magic UI 3.0軟件：完整更改日誌和已知問題

2020-10-29
| 科技


華為發佈EMUI 10.0和Magic UI 3.0軟件

EMUI 10.0和Magic UI 3.0 beta程序已經在多款華為和榮耀智能手機上啟動，並且華為榮耀手機的頂級產品陣容正在獲得穩定的EMUI 10.0和Magic UI 3.0系統軟件更新。既然我們已經多次討論了EMUI 10.0和Magic UI 3.0軟件最新升級所帶來的功能和更改，那麼現在該向您展示完整的更改日誌以及EMUI 10.0和Magic UI 3.0所附帶的已知問題了。

EMUI 10.0 / Magic UI 3.0的已知問題

有些問題可能會影響您的日常使用。如果您經常使用相關的應用或功能，請不要安裝此更新。


華為EMUI 10.0和Magic UI 3.0軟件

1.某些銀行應用程序（banking app）與Android 10操作系統不兼容。只有發佈了兼容的應用程序版本後，才能解決此問題。

2.一些即時消息應用程序與Android 10操作系統不兼容，並且在發送圖像時可能會遇到問題。需要重新啟動應用程序以解決此問題。

3.如果在安裝此更新之前為即時消息應用程序禁用了聲音和振動警報，則默認情況下將為來自即時消息應用程序的通知啟用靜音模式。可以根據您的喜好在應用程序設置中修改警報模式。

4.對於啟用了“ App twin”功能的應用程序，此更新可能允許twin應用程序讀取“消息傳遞”內容。禁用並重新啟用App twin功能以解決此問題。

5.在播放PlayerUnknown的Battlegrounds（PUBG）時，在某些情況下屏幕可能會閃爍。設置非HDR分辨率或重新啟動遊戲即可解決此問題。

6.一段時間後，三星手錶可能會自動從您的手機斷開連接。此問題將在更高版本中解決。

7.用兩根手指捏住30倍變焦有時可能會失敗。拖動縮放滑塊或重新啟動相機即可解決此問題。

EMUI 10.0 / Magic UI 3.0更改日誌

EMUI 10.0 –此版本旨在讓您提前體驗新的軟件功能，從而為我們提供進一步改進軟件反饋意見的機會。以下是此版本的已知問題，可能會影響您的日常使用。如果您經常使用相關的應用程序或功能，請不要安裝此更新。

Magic UI 3.0 –旨在進一步簡化和增強用戶體驗的質量。它為用戶提供了全新的，經過微調的用戶界面，旨在促進更流暢，更直觀的用戶體驗，同時包括其增強的數據和隱私保護還帶給您更安全的體驗。

用戶界面：複雜而令人歎為觀止：

–雜誌設計：

結合了雜誌式的佈局，以提供更舒適和真實的閱讀體驗。

–莫蘭迪（Morandi）顏色：

低調的灰色調使顯示屏更柔和，更舒適。

–動畫：

動態自然的動畫，用於頁面過渡，應用程序啟動和其他情況。

性能：平穩；


華為榮耀手機

– GPU Turbo：

改善遊戲玩法時的流暢性和穩定性。

–手機克隆：

提高連接和克隆速度。

安全性：更安全的數據和隱私安全性：

–正式驗證的微內核

使用華為自己開發的微內核設計，該設計被認證為最安全的商用OS內核。

軟件更新安全性：更安全的數據和隱私經過正式驗證的微內核使用華為自己開發的微內核設計，該設計被認證為最安全的商業OS內核。

–分佈式技術：

提高與其他華為設備的連接以及在其他華為設備之間傳輸文件和存儲數據時的安全性

- 隱私保護

支持應用沙箱功能，以實現標準化的應用權限管理以及更安全的數據和隱私保護。

媒體和娛樂：更有趣

– AppAssistant dropzone

為AppAssistant添加一個放置區，使您可以快速訪問服務，例如遊戲加速和請勿打擾。

安全：

帶來安全性增強。

更新說明：

1.此更新不會刪除您的個人數據，但是我們建議您在更新之前備份所有重要數據。

2.更新後，某些與Android 10操作系統不兼容的第三方應用程序可能無法正常運行。建議您將這些應用更新為應用商店中的最新版本

3.如果在更新過程中遇到任何問題，請撥打華為客戶服務熱線或訪問華為客戶服務中心尋求幫助。

4.您的主屏幕布局可能會稍微重新排列。

5.此係統版本可以回滾到HiSuite中的EMUI 9.1的正式版本。
