二、關鍵對象/方法的分析

經過了 十 期教程，大家對 VBA 估計有了初步的認識。從這期開始，第二部分改為關鍵對象/方法的分析，會對一些比較常用，但又不容易掌握的對象/方法進行彙總和講解。

1. Dictionary

一種比較常用的對象，儲存數據的方式為：關鍵字(稱為 Key) - 條目(稱為 Item)。

屬於 Dictionary 常用的屬性、方法包括：

  • Add，添加一對相對應的關鍵字和條目到 Dictionary 對象。
  • Exits，如果在 Dictionary 對象中指定的關鍵字存在，返回 True，若不存在，返回 False。
  • RemoveAll，刪除所有關鍵字和條目對。
  • Count，返回 Dictionary 對象中的條目數。
  • Keys，返回一個數組，該數組包含一個 Dictionary 對象中的全部已有的關鍵字。
  • Key，重新設置一個 Key 的 Key 值。

2. Dir

返回文件名、目錄名或文件夾名稱。

常見的用法舉例：

  • Dir "C:\WINDOWS\WIN.ini" ——當 文件存在時，返回 "win.ini"，否則，返回""。
  • Dir "C:\WINDOWS\*.ini" ——返回第一個 ini 文件
  • Dir ——返回同樣查詢條件時的下一個文件，如果沒有，返回 ""。
  • Dir "c:\", vbDirectory ——以字母為順序，返回第一個 C 盤下面的第一個文件或目錄
  • Dir "c:\" ——以字母為順序，返回第一個 C 盤下面的第一個文件

3. For Each

  • 用於數組或者集合，針對一個數組或集合中的每個元素，重複執行一組語句。
  • 如果用於數組，數組元素必須是Variant 變量。
  • 更常用的用法是用於 VBA 定義的一些集合，比如 shapes，Worksheets，Workbooks，Range等。
  • 可以用 Exit For，強制退出循環。

4. Application

代表整個 Microsoft Excel 應用程序，非專業人員，不可能把 Application 的全部方法和屬性都掌握。日常運用，只需要知道幾張常用的方法/屬性就可以：

  • Application.WorksheetFunction，調用 Excel 的公式，這裡的 Application 或者 WorksheetFunction 可以省略其中一個，比如用 WorksheetFunction.Match 或者 Application.Match 都是可以的，但不能直接x寫 Match。
  • Application.InputBox，彈出窗口輸入，這裡的 Application 也可以省略，直接寫 InputBox 也可以。
  • Application.Transpose，行列轉置，這裡的 Application 不能省略，否則會出錯。

三、執行思路及過程分析

【1】獲取所有子目錄

第一個 If 函數，對 A1 路徑格式進行修正，以 \ 結尾。

接著創建一個字典 d，並且把 A1 的路徑作為 Key 寫入字典。後面對 d 僅用到它的 Key，所以，Item 取 “”就可以。

Zi，是否進行子目錄查找的標誌，取 1 表示查詢子目錄。

外層的 Do While 的循環，是對目標目錄以及所有子文件夾的循環。第一次循環，對 A1 的目標目錄執行 Dir，帶上參數 vbDirectory，表示匹配的範圍，包括該目錄下面的子目錄。

進入到內層的 Do While 循環。通過 fname = Dir 不斷獲取目標目錄下面的文件或者子目錄。

這兩個嵌套的 If 條件，其中外層的，涉及到一個小知識——在任何一個目錄（或者說文件夾）中，都有兩個隱含目錄："."表示當前目錄，".."表示當前目錄的上一級目錄，我們把這兩個目錄都忽略掉。對於獲取到的普通的文件或文件夾，會進入到內層的 If 條件，這裡用到了 GetAttr 這個函數——

GetAttr 返回的是一個整數，表示文件、目錄、或文件夾的屬性。

另外，還用到了 vbDirectory 這個常數，這個常數，其實就是 16 。

如果 GetAttr 返回的是 16，那就表示這是個子目錄，子目錄錄入到字典中去。

完成一次目錄的查找後，內層 Do While 循環跳出，i 自加 1 。

此時，如果在上次目錄的查找中，有找到新的子目錄，d 的元素就增加了，這樣雖然 i 自加 1，但 d.Count 也增加了，所以外層循環仍未能跳出，繼續對新找到的子目錄進行同樣的查找操作，直到所有的子目錄都完成查找為止。

【2】獲取所有符合條件的文件名

通過 InputBox，彈出窗口，進行獲取查找文件的後綴，參數 Default 表示默認值。

For Each 循環，它循環的數組是字典 d 的 Keys，也就是目標目錄和所有子目錄。

通過 Do While 循環，對目標目錄和子目錄下面的所有文件進行匹配。滿足查詢類型的所有文件的文件名，以字符串的形式，保存在 ms1 和 ms2 中。其中 ms1 只保存文件名，ms2 保存 路徑+文件名。

【3】結果形式轉換並且輸出

result1 和 result2 分別保存 文件名 和 路徑+文件名，這裡，用到了兩個函數——

Split，以“，”為識別符，將ms轉換為一維數組。

一維數組取值到 Excel 表格的行很容易，但取值到列比較麻煩，所以用 Transpose，將一維數組轉置為二維數組。

最後，把 result1 和 result2 填入到指定的區域，這個區域的定義，用了一個 Resize 函數。

Resize，VBA 的定義是 Range 的一種屬性，返回一個新的區域，它有兩個參數，Resize(RowSize, ColumnSize)，分別是新區域的行數和列數。
