新風設計基本要素

科學合理的風量設計：在保證室內空氣充分置換的前提下，應確保人體的舒適性，風速不宜過大；（推薦：0.1m/s ~0.15m/s ）

合理的氣流組織方式：儘量營造一個全面、均勻、穩定的氣流場，保證活動區域無死角，並且氣流流向汙濁出再排出。

全天候24小時的持續置換：確保空氣場始終處於動態運行中，無論主人是否在家，無論主人是睡眠還是活動狀態；

系統噪音低：不影響主人的工作生活和睡眠；

能耗低：設置必要的熱回收裝置，最好選用全熱回收；

系統易於操作、維護、清洗。

新風設計注意要點

為使環境空氣對流暢順，系統排風量=新風量。

建築空間體積大小：住宅室內人的呼吸活動帶為2.4m垂直高度；商業建築內人的呼吸活動帶為4.5m的垂直高度，呼吸帶佔整個空間的體積比約為0.26-0.47 。

住宅、辦公建築其新風量不小於30m3/h.p。綜合考慮換氣次數和最少新風量兩個因素，取兩者計算最大新風量作為選型依據。

體育場館、大會議廳、影院等，可根據上座率真結合換氣次數確定新風量選型，人停留3小時以下的，按50%上座率確定新風量。

對於大型商場等到中央空調系統空間，按中央空調系統總送風量的30%確定新風量進行選型。

工廠、車間等有毒、有害物散發場所、按稀釋濃度所需風量確定新風量，結合換氣次數進行選型。

維護-保養


維修保養須知

1、請不要自行檢查或維修機組；

2、每次維修前，停止運用機組，並務必關掉電源開關；

3、若發現異常現象（如著火等氣味），請立即切斷電源，與銷售商聯繫；

4、請務必使用過濾器；

5、請勿用手對機組進行操作；

6、請勿用水來清洗熱交換器元件；

7、請勿將全熱交換器用作其他用途；

8、請勿堵塞吸入吹出格柵。過濾網清洗拆卸說明(每三個月應對過濾網進行清洗)。



管道與風口安裝


一、管道安裝

1 ) 將主機與管道相連通向風口,採用紮帶固定管道與主機風口

2 ) 接口和管道的連接部分要用鋁箔膠帶包卷粘接，不可漏氣。

3 ) 使用管箍將管道固定在牆上或天花板上。管箍的間距如下：

Ø125-200mm Ø75-110mm 彎頭，縮節，三通等；

水平安裝 1600mm 1200mm；

垂直安裝 3000mm 2000mm；

離兩端200mm。

4 ) 室外的管道要向室外一側傾斜，防止室內進水。（坡度1/100~1/50）

二、迴風口安裝

1 ) 迴風口一般佈置在浴室、衛生間、儲藏室、吸菸區等空氣汙染較嚴重的地

方，儘量配合室內裝飾。

2 ) 將回風口連接到迴風管。如果採用複合軟管，必須用管箍或螺釘固定。

3 ) 迴風口和進風口間距保持在1.5 m以上，以防止排出氣體竄流。

4 ) 室外排風口應採用帶有防蟲網的不鏽鋼材料。

吸風口一般佈置在衛生間，走廊，雜物間，吸菸區等空氣汙染較嚴重的地方佈置位置應儘量配合室內裝飾。

窗式進風器，通常安裝在除廚房，衛生間以外的所有房間窗框上沿的位置。

送風口和迴風口相距必須在1.5m以上，防止排出氣體竄流，形成二次汙染；

室外必須採用帶有防蟲網，防淋雨的風口，室內可根據用戶選擇風形狀；

安裝時要預留檢修口空間，以便維修和清洗交換器芯體及過濾網。
