指針變量簡介

我想大家應該都知道，指針是C語言的靈魂，我的這種說法應該不是很過分，因為在C語言中，指針真的很重要。指針很強大，並且具有很強的靈活性。

指針變量

指針變量為什麼都是4個字節的？

因為我麼通常使用的都是32位的計算機，內存地址都是32位的整數，而指針變量的實質就是內存地址。

什麼是指針變量？

存放地址的變量成為指針變量。指針變量是一種特殊的變量，它不同於一般的變量，一般變量存放的是數據本身，而指針變量存放的是數據的地址。

示例

#includeint main(){	int a = 2; int *pa; char b = 'a'; char *pb; pa = &a; pb = &b;  return 0;}

如上圖所示，指針變量pa是一個char類型的指針變量，指針pb是一個int型的指針變量。熟悉C語言的朋友都知道char型變量佔一個字節，int型變量佔4個字節。指針變量pa指向變量a的地址時取的是存儲變量a在內存中的最小存儲地址，而所指向的佔用4個字節大小的內存區域。

所以我們從這裡可以看出，我們不應該簡單地將指針理解為地址，而是把指針理解為指向一塊內存區域的起始地址，指向區域的大小視所指變量的類型而定。
