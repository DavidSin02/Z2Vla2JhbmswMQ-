「消防術語“掃盲”」滅火設備專業術語

2020-10-29
| 科技

滅火設備Fire extinction equipments

01

滅火器

Fire extinguishers

由筒體、器頭、噴嘴等部件組成，藉助驅動壓力可將所充裝的滅火劑噴出滅火的器具。

1.1 滷代烷滅火器

Halon fire extinguisher

充裝滷代烷滅火劑的滅火器。

1.2 清水滅火器

Plain water fire extinguisher

充裝清水的滅火器。

1.3 二氧化碳滅火器

Carbon dioxide fire extinguisher

充裝液態二氧化碳滅火劑的滅火器。

1.4 酸鹼滅火器

Soda acid fire extinguisher

充裝酸鹼滅火劑的滅火器。

1.5 泡沫滅火器 Foam fire extinguisher

充裝泡沫滅火劑和水，能產生並噴射泡沫的滅火器。

1.6 乾粉滅火器

Powder fire extinguisher

充裝乾粉滅火劑的滅火器。

1.7 貯壓式滅火器

Stored pressure fire extinguisher

滅火劑與驅動氣體一起貯存在同一筒體內的滅火器。

1.8 貯氣瓶式滅火器

Gas cartridge fire extinguisher

滅火劑充裝在筒體內，而驅動氣體單獨貯存於貯氣瓶內的滅火器。

1.9化學反應式滅火器

Chemical reaction fire extinguisher

藉助分開貯存在筒體內的兩種或多種化學物質的反應壓力噴出滅火劑的滅火器。

1.10 固定式滅火器?

Fixed fire extinguisher

固定安裝在火災危險場所，可自動噴射滅火劑的滅火器。

1.11 移動式滅火器

Mobile fire extinguisher

當火災發生時，可將其從放置處移動到起火地點進行滅火的滅火器。

1.11.1 手提式滅火器

Portable fire extinguisher

可用手攜帶的移動式滅火器。

1.11.2 推車式滅火器

Transportable fire extinguisher

裝有輪子的移動式滅火器。

02

固定式滅火系統

Fixed extinguishing system

由固定安裝的滅火劑供應源、管路、噴放器件和控制裝置組成的滅火系統。

2.1 噴水滅火系統

Sprinkler system

按適當的間隔和高度，裝有噴頭的供水滅火系統。

2.1.1 溼式噴水滅火系統

Wet sprinkler system

由溼式報警裝置、噴頭、管路等組成，並在報警閥上、下管路始終充滿著水的滅火系統。

2.1.2 乾式噴水滅火系統

Dry sprinkler system

由於式報警裝置、噴頭、管路和充氣設備等組成，並在報警閥上部管路中充以有壓氣體。

2.1.3 預作用滅火系統

Pre-action system

由火災探測系統、閉式噴頭、預作用閥等組成，在噴頭打開之前管路內才充 水的滅火系統

2.1.4 雨淋滅火系統

Deluge system

由火災探測系統、開式噴頭、雨淋閥等組成的滅火系統。

2.1.5 水幕系統

Drencher system

由水幕噴頭、管路和控制閥等組成的一種阻火，隔火噴水系統。

2.1.6 水霧系統

Water spray projecter system

由水霧噴頭、管路、控制裝置等組成的滅火系統。

2.1.7 蒸氣噴射系統

Steam projection system

由蒸氣源、噴頭、管路系統和火災探測器等組成的滅火系統。

2.2 泡沫滅火系統

Foam extinguishing system

由水源、水泵、泡沫液供應源、空氣泡沫比例混合器、管路和泡沫產生器組 成的滅火系統。

2.2.1 液下噴射泡沫滅火系統

Base injection foam extinguishing system

是一種在可燃的液體表面下注入泡沫，泡沫上升到液體表面並擴散開，形成一個泡沫層的滅火系統。

2.3 乾粉滅火系統

Powder extinguishing system

由乾粉供應源通過輸送管路連接到固定的噴嘴上通過噴嘴噴放乾粉的滅火系統。

2.4 混合滅火系統

Combined agent extinguishing system

使用兩種以上滅火劑的滅火系統(例如泡沫和乾粉)。

2.5 二氧化碳滅火系統

Carbon dioxide extinguishing system

由二氧化碳供應源、噴嘴和管路組成的滅火系統。

2.6 滷代烷滅火系統

Halon extinguishing system

由滷代烷供應源、噴嘴和管路組成的滅火系統。

2.7 惰化系統

Inerting system

引入適當濃度的惰性氣體防止可燃的氣體、蒸氣、粉塵燃燒或爆炸的系統。

2.8 全淹沒 Total flooding

是一種將滅火劑(氣體、高倍泡沫等)以一定濃度(強度)充滿被保護的有限空間的滅火方式。

2.9 全淹沒滅火系統

Total flooding extinguishing system

以全淹沒方式滅火的固定系統。

2.10 局部應用滅火系統

Local application extinguishing system

主要由一個適當的滅火劑供應源組成，它能將滅火劑直接噴放到著火物上或認為危險的區域。
