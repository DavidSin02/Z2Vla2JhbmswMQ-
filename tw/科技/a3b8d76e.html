各種插口傻傻分不清 ？堂寶寶來給你理一理！

2020-11-14
| 科技

現在人手一兩部手機相對來說很普遍了，就拿我來說吧，一部 iPhone 肯定是不夠，因為續航實在堪憂。經常半天都不到，電量就不夠 20%了。備一個充電寶插著線充電玩手機實在是難受啊。

於是我又入了一部安卓手機，身上還要常備一個滿電充電寶，這樣才剛剛滿足要求。那麼大家身邊都是備用幾根線呢？

iPhone 的 Lightning 線，安卓的 USB Type-C 線和Micro USB 2.0 線。

這都已經有三種接口了。經常用的人還好，但是平時只用一種接口的人，經常會有傻傻分不清的情況。

3.5mm / 2.0mm 接口

這種帶著大屁股的電源適配器可以說是童年回憶了。這種接口雖然通用，但是考慮到不方便和不安全的因素，現在已經被淘汰掉了。

Mini USB

以前的 手機、MP3、照相機使用最多的就是這種接口了。但是很顯而易見一點是，你現在已經很難這種接口用在高端設備上了。

兩個原因，一是充電太慢，數據傳輸速度也是慢得不行。第二點是太厚了！

Micro USB 2.0

Micro USB 被拋棄的原因是因為太厚，那 2.0 版本要改變，第一條思路肯定是做薄。

USB Type - C

這種插口應該是目前安卓高端機上使用最多的接口了。對比 Micro USB 2.0 ，Type - c 有一種顯而易見的好處：正反插都行。

Lightning

我想肯定有小夥伴們會想問，為什麼蘋果和安卓的充電接口不能通用？一個原因：蘋果不想。蘋果不管是用系統上、硬件上，都不屑於與安卓“ 同流合汙 ”。

但是就設計上來說，Lightning 的接口還是很完美的，不管是用充電速度，傳輸速度上來說，都是可以個 Type - C 媲美的。

堂寶寶今天為大家講解了 5 種接口，不知道大家現在分清楚了沒有？
