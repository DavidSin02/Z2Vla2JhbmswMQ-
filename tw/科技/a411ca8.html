軟妹子帶你玩轉Python數據可視化，座標軸方向及範圍設置

2020-10-29
| 科技

在實際工作的過程中，針對不同的需求，我們需要對圖像的座標軸方向或者範圍進行自定義。比如，在一文中我們介紹瞭如何改變座標軸的位置，而當Y軸位於右側或者X軸位於頂部的時候，我們通常需要將X軸或Y軸的方向進行改變；我們也有可能會遇到數據量很大但是只需要顯示某個區間內的數據的情況，此時我們便需要對座標軸範圍進行設置。

座標軸方向設置

座標軸的方向設置通常有兩種方法：第一種為通過axes.invert_xaxis和axes.invert_yaxis方法對X軸和Y軸的方向進行改變；第二種為通過axes.set_xlim和axes.set_ylim方法對X軸和Y軸的方向進行改變。

1、axes.invert_xaxis和axes.invert_yaxis方法

axes.invert_xaxis()

axes.invert_yaxis()

axes.invert_xaxis和axes.invert_yaxis方法不需要參數，當需要改變座標軸方向時，直接調用該方法即可。

方法實例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

n_groups = 11

index= np.arange(1,n_groups)

means_men = np.random.randint(0,100,10)

means_women = np.random.randint(0,100,10)

fig, ax = plt.subplots(nrows=1,ncols=2,sharey=True)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,wspace=0.05)

ax[0].set_facecolor('None')

ax[0].yaxis.set_ticks_position('right')

ax[0].set_yticks(np.arange(1,n_groups))

ax[0].set_yticklabels(('A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J'))

ax[0].invert_xaxis()

ax[0].barh(index,means_women,color='lightgray')

ax[0].set_title('Women',color='w')

ax[1].set_facecolor('None')

ax[1].barh(index, means_men, color='lightgray')

ax[1].set_title('Men', color='w')

plt.show()

2、axes.set_xlim和axes.set_ylim方法

axes.set_xlim(left, right)

axes.set_ylim(bottom, top)

axes.set_xlim和axes.set_ylim方法的參數分別為X軸左端點值、右端點值和Y軸底部端點值、頂部端點值。

當需要改變X軸的方向的時候，只需將axes.set_xlim方法的left參數值設置為座標軸最大值、right參數值設置為座標軸最小值即可；想要改變Y軸的方向，只需將axes.set_ylim方法的bottom參數值設置為座標軸最大值、top參數值設置為座標軸最小值即可。

方法實例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

n_groups = 11

index= np.arange(1,n_groups)

means_men = np.random.randint(0,100,10)

means_women = np.random.randint(0,100,10)

fig, ax = plt.subplots(nrows=2,ncols=1,sharex=True)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,hspace=0.12)

ax[0].set_facecolor('None')

ax[0].set_xticks(np.arange(1,n_groups))

ax[0].set_xticklabels(('A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J'))

ax[0].bar(index,means_women,color='lightgray')

ax[1].set_facecolor('None')

ax[1].xaxis.set_ticks_position('top')

ax[1].set_ylim(100,0)

ax[1].bar(index,means_men,color='lightgray')

plt.show()

座標軸範圍設置

座標軸範圍的設置通常有兩種方法，第一種為通過axes.set_xbound和axes.set_ybound對X軸和Y軸進行分別設置，第二種為通過axes.set_xlim和axes.set_ylim方法對X軸和Y軸進行分別設置。

這兩種方法雖然都能改變座標軸刻度範圍，但是在使用的時候卻有差別。

1、axes.set_xbound和axes.set_ybound方法

axes.set_xbound(lower, upper)

axes.set_ybound(lower, upper)

axes.set_xbound和axes.set_ybound方法有兩個參數值，這兩個參數值不分先後，大的值代表座標軸的最大值，小的值為座標軸最小值。

方法實例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

x = np.linspace(0.0, 5.0)

y = np.cos(2 * np.pi * x) * np.exp(-x)

fig, ax = plt.subplots(nrows=1,ncols=2)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,wspace=0.12)

ax[0].set_facecolor('None')

ax[0].plot(x,y,'o-',color='gold')

ax[1].set_facecolor('None')

ax[1].plot(x,y,'o-',color='gold')

ax[1].set_ybound(-0.3,0.5)

plt.show()

2、axes.set_xlim和axes.set_ylim方法

axes.set_xlim(left, right)

axes.set_ylim(bottom, top)

axes.set_xlim和axes.set_ylim方法我們已經在座標軸方向中講過，他們的參數分別為X軸左端點值、右端點值和Y軸底部端點值、頂部端點值，所以只要給定參數值就設定好了座標軸範圍。

方法實例：

import numpy as np

import matplotlib.pyplot as plt

import matplotlib as mpl

import matplotlib.image as img

mpl.rcParams['ytick.color']='white'

mpl.rcParams['xtick.color']='white'

mpl.rcParams['ytick.labelsize']='large'

mpl.rcParams['xtick.labelsize']='large'

mpl.rcParams['axes.edgecolor']='white'

x = np.linspace(0.0, 5.0)

y = np.cos(2 * np.pi * x) * np.exp(-x)

fig, ax = plt.subplots(nrows=2,ncols=1)

bgimg = img.imread('picture.png')

fig.figimage(bgimg)

fig.subplots_adjust(left=0.05,right=0.95,top=0.95,bottom=0.05,hspace=0.12)

ax[0].set_facecolor('None')

ax[0].plot(x,y,'o-',color='gold')

ax[1].set_facecolor('None')

ax[1].plot(x,y,'o-',color='gold')

ax[1].set_xlim(1,4)

plt.show()

需要注意的是

  1. 當我們需要使用axes.set_xscale方法改變X軸比例尺時，axes.set_xbound方法和axes.set_xlim方法必須在axes.set_xscale方法之後使用才能正常顯示。

  2. 當我們需要使用axes.invert_xaxis方法對座標軸方向進行改變時，使用axes.set_xbound方法並不會對axes.invert_xaxis方法產生影響；而axes.set_xlim方法與axes.invert_xaxis方法互相影響，位置靠後的代碼效果會覆蓋位置靠前的代碼效果。

對比實例：

1、左圖axes.set_xbound方法在axes.set_xscale方法之後使用，右圖axes.set_xbound方法在axes.set_xscale方法之前使用

import numpy as np

ax[0].set_xscale('log',basex=2)

ax[0].set_xbound(0,4)

ax[1].set_facecolor('None')

ax[1].set_xbound(0,4)
