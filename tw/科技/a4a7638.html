springboot訪問圖片本地路徑並映射成url

2020-10-29
| 科技

springboot如何訪問本地路徑並獲取圖片等靜態資源文件的url？代碼寫法如下：

增加一個配置類：

@Configurationpublic class WebMvcConfigurer implements org.springframework.web.servlet.config.annotation.WebMvcConfigurer {    @Override    public void addResourceHandlers(ResourceHandlerRegistry registry) {        //和頁面有關的圖片放在項目的img目錄下        registry.addResourceHandler("/img/**").addResourceLocations("classpath:/img/");    }}

運行該工程：可以發現資源文件夾img也被放入了部署的target文件夾中；
