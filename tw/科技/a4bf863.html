4.混凝土結構工程

4.1基本原則

1 .同一標段內根據各樓棟進度， 在實測前隨機確定已拆完模板的2個樓層作為混凝土結構工程的實測層。

2 .根據選取樓層結構平面圖，實測實量選點考慮每層結構4個角和中間砼剪力牆、柱。當實測砼結構的截面尺寸、表面平整度、垂直度時， 每個實測層要選取1 0個實測區，2個實測層累計2 0個實測區。

3 .當實測同一樓層內頂板水平極差時， 每個實測層選取5個實測區，2個實測層累計1 0個實測區。每個實測區實測5個點，每個點均作為1個計算點。

4.2 截面尺寸偏差（砼結構）

指標說明：反映層高範圍內剪力牆、砼柱施工尺寸與設計圖尺寸的偏差。



以鋼捲尺測量同一面牆/柱截面尺寸，精確至毫米。同一牆/柱面作為1個實測區，累計實測實量2 0個實測區。每個實測區從地面向上3 0 0 m m 和1 5 0 0 m m各測量截面尺寸1次。



取其中與設計尺寸偏差最大的數，作為判斷該實測指標合格率的1個計算點。

4.4 垂直度（砼結構）

指標說明：反映層高範圍內剪力牆、砼柱表面垂直的程度。



砼柱： 任選砼柱四面中的兩面， 分別將靠尺頂端接觸到上部砼頂板和下部地面位時各測1次垂直度。



剪力牆：任取長邊牆的一面作為1個實測區。累計實測實量2 0個實測區。



當牆長度小於3米時， 同一面牆距兩端頭豎向陰陽角約3 0 c m位置，分別按以下原則實測2次：一是靠尺頂端接觸到上部砼頂板位置時測1次垂直度，二是靠尺底端接觸到下部地面位置時測1次垂直度。砼牆體洞口一側為垂直度必測部位。



當牆長度大於3米時， 同一面牆距兩端頭豎向陰陽角約3 0 c m和牆中間位置， 分別按以下原則實測3次： 一是靠尺頂端接觸到上部砼頂板位置時測1次垂直度，二是靠尺底端接觸到下部地面位置時測1次垂直度， 三是在牆長度中間位置靠尺基本在高度方向居中時測1次垂直度。砼牆體洞口一側為垂直度必測部位。這3個實測值分別作為判斷該實測指標合格率的3個計算點。

4.5頂板水平度極差(砼結構)

指標說明：反映同一房間砼頂板的平整程度。






使用激光掃平儀，在實測板跨內打出一條水平基準線。同一實測區距頂板天花線約3 0 c m處位置選取4個角點， 以及板跨幾何中心位（ 若板單側跨度較大可在中心部位增加1個測點） ， 分別測量砼頂板與水平基準線之間的5個垂直距離。

4.6 樓板厚度偏差（砼結構）

指標說明：反映同跨板的厚度施工尺寸與設計圖尺寸的偏差。




當採用破損法測量時，可用電鑽在板中鑽孔（需特別注意避開預埋電線管等） ， 以捲尺測量孔眼厚度。1個實測值作為判斷該實測指標合格率的1個計算點。



測量所抽查跨的樓板厚度，當採用非破損法測量時將測厚儀發射探頭與接收探頭分別置於被測樓板的上下兩側，儀器上顯示的值即為兩探頭之間的距離，移動接收探頭，當儀器顯示為最小值時，即為樓板的厚度；

4.7施工控制線設置（砼結構）

指標說明：反映砌築、抹灰、裝修尺寸前期控制的偏差，以便控制砌築、抹灰和裝修的尺寸精度，為砌築、裝修房集中加工等提供控制條件。




測量方法： 採用目測、尺量方法， 檢查同一個實測區是否設置二線，其尺寸是否符合設計要求。

數據記錄： 每一實測區未設置二線， 則該實測點不合格； 反之，則該實測點合格。不合格點均按“1記錄，合格點均按“0記錄。





4.8 軸線偏差控制（砼結構）

指標說明：反映現場軸線控制措施是否合理，現場放線是否滿足精度要求。

5.2表面平整度（砌築工程）

指標說明：反映層高範圍內砌體牆體表面平整程度。



（1） 每一面牆都可以作為1個實測區，優先選用有門窗、過道洞口的牆面。測量部位選擇正手牆面。累計實測實量1 0個實測區。

（2） 當牆面長度小於3米， 各牆面頂部和根部4個角中， 取左上及右下2個角。按4 5度角斜放靠尺分別測量2次， 其實測值作為判斷該實測指標合格率的2個計算點。



（3） 當牆面長度大於3米時， 還需在牆長度中間位置增加1次水平測量，3次測量值均作為判斷該實測指標合格率的3個計算點。

（4） 牆面有門窗、過道洞口的， 在各洞口4 5度斜交測一次， 作為新增實測指標合格率的1個計算點。所選2套房中牆面表面平整度的實測區不滿足1 0個時，需增加實測套房數。

指標說明：考慮實測的可操作性，選用同一房間內同一垂直面的砌體牆面與房間方正度控制線之間距離的偏差，作為實測指標，以綜合反映同一房間方正程度。

6.2 牆體表面平整度(抹灰工程)

指標說明：反映層高範圍內抹灰牆體表面平整程度。




當牆面長度小於4米， 在同一牆面頂部和根部4個角中， 選取左上、右下2個角按4 5度角斜放靠尺分別測量1次，在距離地面2 0 c m左右的位置水平測1次



當牆面長度大於4米， 在同一牆面4個角任選兩個方向各測量1次，在牆長度方向任意位置增加2次水平測量，在距離地面2 0 c m左右的位置水平測2次；



6.3 牆面垂直度(抹灰工程)

指標說明：牆面垂直度反映層高範圍抹灰牆體垂直的程度.




當牆長度小於3米時， 同一面牆距兩端頭豎向陰陽角約3 0 c m位置，分別按以下原則實測2次：一是靠尺頂端接觸到上部砼頂板位置時測1次垂直度，二是靠尺底端接觸到下部地面位置時測1次垂直度；具備實測條件的門洞口牆體垂直度為必測項.

6.4 室內淨高偏差(抹灰工程)

指標說明：綜合反映同一房間室內淨高實測值與理論值的偏差程度。





實測前， 所選套房必須完成地面找平層施工。同時還需瞭解所選套房的各房間結構樓板的設計厚度和建築構造做法厚度等；各房間地面的4個角部區域，距地腳邊線3 0 c m附近各選取1點（避開弔頂位），在地面幾何中心位選取1點，測量出找平層地面與天花頂板間的5個垂直距離，即當前施工階段5個室內淨高實測值。




6.5 頂板水平度極差(抹灰工程)

指標說明：考慮實際測量的可操作性，頂板膩子完成後，選取同一功能房間砼頂板內四個角點和一箇中點距離同一水平基準線之間5個實測值的極差值，綜合反映同一房間砼頂板的平整程度。



測量方法和數據記錄：使用激光掃平儀，在實測板跨內打出一條水平基準線。同一實測區距頂板天花線3 0 c m處位置選取4個角點， 以及板跨幾何中心位（ 若板單側跨度較大可在中心部位增加1個測點） ， 分別測量砼頂板與水平基準線之間的5個垂直距離。

測量方法和數據記錄：



以垂直距離的最低點為基準點， 計算另外四點與最低點之間的偏差，最大偏差值≤1 5 m m時，5個偏差值（基準點偏差值以0計）的實際值作為判斷該實測指標合格率的5個計算點。最大偏差值﹥1 5 m m時，5個偏差值均按最大偏差值計，作為判斷該實測指標合格率的5個計算點。

6.6 方正度（抹灰工程）

指標說明： 考慮實際測量的可操作性， 選用同一房間內同一垂直面的牆面與房間方正度控制線之間距離的偏差， 作為實測指標，以綜合反映同一房間方正程度。




測量方法和數據記錄：每套房同層內必須設置一條方正控制基準線（儘量通長設置，降低引測誤差），且同一套房同層內的各測區（即各房間）必須採用此方正控制基準線， 然後以此為基準， 引測至各測區（ 即各房間）；



在同一測區內，實測前需用5米捲尺或激光掃平儀對彈出的兩條方正度控制線，以短邊牆為基準進行校核， 無誤後採用激光掃平儀打出十字線或吊線方式，沿長邊牆方向分別測量3個位置（兩端和中間）與控制線之間的距離。



如果現場找不到控制線，可以一面帶窗牆面為基準，用儀器引出兩條輔助方正控制線。
