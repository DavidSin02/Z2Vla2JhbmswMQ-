VBA｜數據庫操作07：獲取數據庫的表名與字段信息

2020-10-29
| 科技

1 引用ADOX對象庫

Alt+F11（打開VBE）→工具→引用：

2 用Table對象列出所有表名

Sub 顯示錶名()

Dim cat As New ADOX.Catalog

Dim strdb As String, strcon As String, r As Long

strcon = "Provider=Microsoft.Jet.OLEDB.4.0;" _

& "Data Source=" & ThisWorkbook.Path & "\Northwind.mdb"

cat.ActiveConnection = strcon

r = 3 '工作表中顯示數據的行

With Worksheets("表名")

.Columns(1).Clear

.Cells(2, 1) = "表："

.Cells(2, 1).Font.Bold = True '加粗字體

For i = 0 To cat.Tables.Count - 1 '列出數據表名稱

If cat.Tables(i).Type = "TABLE" Then

.Cells(r, 1) = cat.Tables(i).Name

r = r + 1

End If

Next i

End With

Set cat.ActiveConnection = Nothing

End Sub

3 用Fields集合列出表的字段信息

Sub 表的字段信息()

Dim cnn As New Connection, rs As New Recordset

Dim strcon As String, strSql As String, r As Long

On Error Resume Next

strcon = "Provider=Microsoft.Jet.OLEDB.4.0;" _

& "Data Source=" & ThisWorkbook.Path & "\Northwind.mdb"

cnn.Open strcon '打開數據源連接

strSql = "客戶" 'SQL語句

rs.Open strSql, cnn, adOpenKeyset, adLockOptimistic

With Worksheets("字段信息")

.Cells.Clear '清除表中的數據

.Range("A1:C1") = Array("名稱", "類型", "大小") '填寫表頭數據

.Range("A1:C1").Font.Bold = True

For r = 0 To rs.Fields.Count - 1 '將字段信息填充到工作表

.Cells(r + 2, 1) = rs.Fields(r).Name '字段名

.Cells(r + 2, 2) = rs.Fields(r).Type '字段類型

.Cells(r + 2, 3) = rs.Fields(r).DefinedSize '字段大小

Next

End With

rs.Close

cnn.Close

Set rs = Nothing

Set cnn = Nothing

End Sub

使用Recordset對象的Open方法時，可直接給出一個表的名稱，這樣將在記錄集中返回該表中的所有數據。與下面的Select語句功能相同：

SELECT * FROM 客戶
