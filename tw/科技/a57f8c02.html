函數的形參和實參：

參數有形參（parameter）和實參（argument）的區別，形參相當於函數中定義的變量，實參是在運行時的函數調用時傳入的參數。

ECMAScript函數的參數與大多數其他語言中函數的參數有所不同。ECMAScript中的函數定義並未指定函數形參的類型，函數調用也未對傳入的實參值做任何類型檢查，甚至不檢查傳入形參的個數；也就是說，即便定義的函數只接收兩個參數，在調用 這個函數時也未必一定要傳遞兩個參數，可以傳遞一個、三個甚至不傳遞；在實際場景中，需要處理當調用函數時的實參個數和聲明的形參個數不匹配的情況，也會測試函數的實參的類型，以避免非法的實參傳入函數；

可選形參：

當實參比形參的個數要少時，剩下的形參都將設置為undefined值；為了讓實參與形參保持較好的適應性，有必要為可選形參設置一個合理的默認值；如：

function show(name,msg){    console.log("hello," + name + ",message: " + msg);}show(); // hello,undefined,message: undefinedshow("wangwe");  // hello,wangwe,message: undefinedshow("wangwei","18 year old");  // hello,wangwei,message: 18 year old// show函數改成function show(name,msg){    if(name == undefined) name = "wujing";    if(msg == undefined) msg = "28 girl";    console.log("hello," + name + ",message: " + msg);}// 不使用if，可以使用“||”，這是種慣用手法function show(name,msg){    name = name || "wujing";    msg = msg || "28 girl";    console.log("hello," + name + ",message: " + msg);}

// 將對象o中可枚舉的屬性名追加至數組a中，並返回這個數組a// 如果省略a，則創建一個新數組並返回這個新數組function getPropertyNames(o, /* optional */ a){    if(a == undefined) a = []; // 如果未定義，則創建新數組    //a = a || [];  // 使用||運算符    for(var property in o) a.push(property);    return a;}// 使用，先創建一個對象var o = {name:"wangwei",age:18,sex:"男"};// 調用這個函數可以傳入1或2個參數var arr = [1,2];// var arrPropertys = getPropertyNames(o);  // 使用一個參數var arrPropertys = getPropertyNames(o,arr);  // 使用兩個參數console.log(arrPropertys);

說明：當設置可選參數時，一定要把可選參數放到形參列表的最後，因為不能省略第一個參數（定參）並傳入第二個實參；如果第一個參數（定參）不存在或不使用，也需要顯式的傳入undefined，真實的場景是傳入一個無意義的佔位符null；

形參個數：

函數本身的length屬性，只讀，它代表函數的形參個數，即函數期望傳入的實參個數，如：

function sum(a,b,c){  console.log(sum.length);  // 3}sum();console.log(sum.length);  // 3

實參對象（arguments）：

ES函數的參數有自己獨特的特點，即當調用函數時傳入的實參個數超過函數定義的形參個數時，沒有辦法直接獲得超出的實參的值，但可以使用實參對象arguments來獲取；

在函數體內，arguments是指向實參對象有的引用，其是一個類數組對象（應該是一個集合）；函數接收的始終都是這個數組，而不關心數組中包含哪些參數；即使沒有傳遞參數，這個數組也存在，只不過是包含0個元素的數組；

function func(a,b){    console.log(arguments);}func();func(1,2,3,4,5);

arguments對象只是與數組類似，它並不是Array的實例；可以使用方括號語法訪問它的每一個元素（即第一個元素是arguments[0]，第二個元素是arguments[1]，以此類推），使用length屬性來確定傳遞進來多少個參數，即length屬性說明調用函數時實際傳遞的參數個數;；

如，不顯式地使用命名參數：

function sayHi(){    console.log("傳入了：" + arguments.length + "個參數");    console.log("Hello,"+arguments[0]+","+arguments[1]);}//sayHi();sayHi("wangwei","歡迎你來學習");

說明：在ECMAScript中，命名的參數只提供便利，但不是必需的；另外，在命名參數方面，其他語言可能需要事先創建一個函數簽名，而將來的調用必須與該簽名一致；但在ECMAScript中，沒有這些規定，解析器不會驗證命名參數，因此可以使用arguments對象來驗證所傳遞的參數是否符合函數要求；如：

// 實際場景中，如果對實參的個數有要求，需要進行驗證function func(x,y,z){    // 首先，驗證傳入實參的個數是否正確    if(arguments.length != 3){        throw new Error("函數func需要：" + arguments.length + "個參數");    }    // 再執行函數的其他邏輯    if(typeof arguments[0] !== "number"){        console.log("參數類型應為數字")    }    y = +arguments[1] ? y : 0;     return x + y + z;}console.log(func(1,"a",3));  // 4

說明：在某些時候，沒有必要檢查實參的個數，因為JS的默認行為是可以滿足需要的，如省略的實參都是undefined，多出的參數自動省略。

驗證形參與實參的個數，如：

function check(args){    var actual = args.length;  // 實參的真實個數    var expected = args.callee.length;  // 形參個數    if(actual !== expected)        throw Error("參數個數不對");}function f(x,y,z){  try{    check(arguments);    return x + y + z;  }  catch(e){    console.log(e);  }}console.log(f(1,2,3));

可以利用arguments對象，讓函數能夠接收任意個數並分別實現適當的功能，如：

function doAdd(){    if(arguments.length==1){        console.log(arguments[0]);    }else if(arguments.length==2){        console.log(arguments[0]+arguments[1]);    }}doAdd(10);doAdd(10,20);// 優化function doAdd(){    var sum = 0;    for(var i=0;i<arguments.length;i++)        sum += arguments[i];    console.log(sum);}

又如：

// 根據實參，返回最大值function max(/* ... */){    var max = Number.NEGATIVE_INFINITY;    for(var i = 0; i< arguments.length;i++)        if(arguments[i] > max) max = arguments[i];    return max;}var largest = max(1,10,100,2,3,1000,4,5,10000,6);console.log(largest);  // 10000

類似於這種可以接收任意個數的實參，這種函數也被稱為“不定實參函數”，這個術語來自C語言。

但真正的不定實參的個數不能為零，其應用的場景是：該函數包含固定個數的命名和必須的參數，以及隨後個數不定的可選實參，即arguments對象可以與命名參數一起使用；

function doAdd(num1,num2){   var result = parseInt(num1) + parseInt(num2);   if(arguments.length > 2){       for(var i = 2; i<arguments.length; i++){            result += parseInt(arguments[i]);       }   }   return result;}console.log(doAdd(1,2,3,4));

arguments並不是真正的數組，它是一個對象，其數組元素是函數實參的別名；arguments的值永遠與對應命名參數的值保持同步；

function func(num1){    // "use strict";  // 重寫arguments的值會失效    console.log(num1);  // 10    arguments[0] = 1;    console.log(num1);  // 1}func(10);

說明：這並不是說讀取這兩個值會訪問相同的內存空間，它們的內存空間是獨立的，但它們的值會同步；即兩者本質上是不同的，其使用的是映射的關係；

注：在嚴格模式下，重寫arguments的值會失效，但不會導致語法錯誤；

實參類型：

Javascript是一門弱類型語言，所以函數中的形參並沒有類型聲明，並且在傳入參數前也未做任何類型檢查，雖然JS可以進行數據類型的自動轉換，但在某些時候（類型不同，不會導致語法錯誤，但在程序運行時有可能導致錯誤），函數還是希望得到一個類型明確的參數值，此時，應當對所傳入的參數進行類型的檢查，如：

// 返回數組元素的累加和// 數組元素必須為數字，null和undefined的元素將被忽略function sum(arr){    if(Array.isArray(arr)){        var total = 0;        for(var i=0;i<arr.length; i++){            var element = arr[i];            if(element == null) continue;  // 跳過null和undefined            if(isFinite(element)) total += element;            else throw new Error("數組元素必須是有限數");        }        return total;    }else throw new Error("函數sum()的參數必須為數組");}console.log(sum([1,2,3,4,5]));

又如：擴大累加的數據類型：

function flexsum(a){    var total = 0;    for(var i = 0; i<arguments.length; i++){        var element = arguments[i], n;        if(element == null) continue;        if(Array.isArray(element))            n = flexsum.apply(this, element);        else if (typeof element === "function")            n = Number(element());        else            n = Number(element);        if(isNaN(n))            throw new Error("無法把：" + element + "轉換為數字");        total += n;    }    return total;}console.log(flexsum(1,2,3));console.log(flexsum(1,"",3));console.log(flexsum([1,[8,7],3]));console.log(flexsum(function(){return 18;}));

沒有重載：

ECMAScript函數不能像傳統意義上那樣實現重載；而在其他語言中，可以為一個函數編寫兩個定義，只要這兩個定義的簽名（參數的類型和數量）不同即可；ECMAScript函數沒有簽名，因為其參數是由包含零或多個值的數組來表示的；而沒有函數簽名，真正的重載是不可能做到的；

如果定義了兩個名字相同的函數，則只有後定義的有效，如：

function show(){alert("ok")};function show(){alert("wangwei")};show();

說明：即是第二個把第一個覆蓋了；

var addNum = function(num){return num + 100;};addNum = function(num){return num + 200;};

模擬重載：可以通過檢查傳入函數中參數的類型和數量並作出不同的反應，可以模擬方法的重載；

function overrideFunc(){    var argLen = arguments.length;  // 可以傳入整型，字符串    if(argLen == 0){        console.log("無參的overrideFunc()方法");    }else if(argLen == 1){        console.log("傳入的參數為：" + arguments[0]);    }else if(argLen == 2){        console.log("傳入了2個參數：");        if(!isNaN(arguments[0]) && !isNaN(arguments[1])){            console.log("和為：" + (arguments[0] + arguments[1]));        }else{            console.log("不是數字：" + arguments[0] + arguments[1]);        }    }else{        console.log("未知，其他處理，也可以拋出異常");    }}overrideFunc();overrideFunc("零點程序員");overrideFunc(3,4);overrideFunc(3,"a",33);

callee和caller屬性：

arguments對象除了擁有數組元素外，還定義了callee和caller屬性；

callee屬性指向當前正在執行的函數，在某些時候非常有用，如在一個匿名函數中使用遞歸；但在嚴格模式下會拋出錯誤，所以，儘量避免使用callee屬性，可以為函數明確定義一個名稱；

function func(num1){    // "use strict";    console.log(arguments.callee);}func(10);// 遞歸函數function factorial(x){    if(x <= 1) return 1;    // return x * factorial(x - 1);    return x * arguments.callee(x - 1);}console.log(factorial(4));// 或在匿名函數中使用function create(){    return function(n){        if(n <= 1) return 1;        return n * arguments.callee(n - 1);    };}console.log(create()(4));

caller是非標準的，但大多數瀏覽器都實現了這個屬性，它指向調用當前函數的函數，在ES5中被移棄了，其始終會返回undefined，不會報錯。

將對象用做實參：

可以採用將對象作為實參，該對象的屬性就是實際需要的實參值，並且參數的順序就無關緊要了，而且還可以方便的設置缺省參數的默認值；作為實參的對象，一般採用鍵/值對的方式進行定義；如：

function copyArr(args){    var j = 0;    args.from = args.from || [ ];    args.start = args.start || 0;    args.to = args.to || [];    args.end = args.end || args.from.length;    for(var i = args.start; i<args.end; i++)        args.to[j++] = args.from[i];    return args.to;}var a = [1,2,3,4,5], b = [];console.log(copyArr({from: a, to: b, start: 2, end: 5}));

函數回調：

回調 ：將一個函數對象a 傳遞給另一個函數對象 b，讓後者在適當的時候執行 a。這就是所謂的“回調”

適當的時候:當這個外部函數在一定條件下就會調用參數指定的函數，此函數就是回調函數。如:

function funA(){    // 代碼}function funB(fun){    fun();}funB(funA);function fun(num,callback){    if(num<0){        alert("分數不能為負");    }else if(num==0){        alert("可能未參加考試");    }else{        callback();    }}function test(){    var myp = document.getElementById("myp");    var str;    var score = document.getElementById("score").value;    fun(score, function(){        if(score<60){            str = "不及格";        }else if(score>=60 && score<80){            str = "及格";        }else{            str = "優秀";        }        myp.innerText = str;    });}</script><input type="text" id="score"><input type="button" value="檢測" onclick="test()"><p id="myp"></p>

匿名函數：

匿名函數即為沒有名字的函數，也稱為拉姆達函數；匿名函數功能強大，用途很廣；

形式：function(arg0,arg1){} // 其實函數表達式使用的就是匿名函數

一般將匿名函數作為參數傳入另一個函數，或者從一個函數中返回另一個函數，這是一種極為有用的技術；如：

function createFun(propertyName){    return function(object1,object2){        var value1=object1[propertyName];        var value2=object2[propertyName];        if(value1<value2){            return -1;        }else if(value1 > value2){            return 1;        }else{            return 0;        }    }   }var arr=[{name:"zhangsan",age:28}, {name:"lishi", age:29}];arr.sort(createFun("name"));alert(arr[0].name);arr.sort(createFun("age"));alert(arr[0].name);

說明：實質上就是嵌套了一個內部函數，並把它返回；這是一個比較函數；

作為值的函數：

在ES中函數不僅是一種語法，也是值；因為其本身就是變量，所以函數也可以作為值來使用；也就是說，可以將函數賦值給變量、存儲在對象的屬性或數組元素中，也可以當作一個參數進行傳遞；

函數可以賦值給其他的變量，如：

function func(x){return x * x};var s = func;console.log(func(4));  // 16console.log(s(4));  // 16

同樣可以將函數賦值給對象的屬性，此時應該稱為方法，如：

var o = {func: function(x){return x* x;}};var y = o.func(16);console.log(y);  // 256

賦值給數組元素，此時的函數可以不需要名字，即是匿名函數，如：

var arr = [function(x){return x*x},18];console.log(arr[0](arr[1]));  // 324 

函數作為參數傳遞，如：

function myFun(someFun,someArg){    return someFun(someArg);}function add(num){return num+10;}var result = myFun(add,10);alert(result); // 20function getGreeting(name){return "hello:"+name;}var result2 = myFun(getGreeting,"wangwei");  // wangweialert(result2);

說明：myFun接受兩個參數，第一個參數是函數，第二個參數是要傳遞給該函數的一個值；

函數作為值的示例：

// 定義一些簡單的函數function add(x,y){return x + y;}function subtract(x,y){return x - y;}function multiply(x,y){return x * y;}function divide(x,y){return x / y;}// operate函數將以上的函數作為參數，並傳入兩個操作數讓其調用function operate(operator, num1, num2){    return operator(num1, num2);}// 調用var result = operate(add, operate(add, 2, 3), operate(multiply, 4, 5));console.log(result);  // (2+3) + 4 * 5 = 25// 改寫// 使用函數直接量，定義在一個對象直接量中var operators = {    add: function(x,y){return x + y;},    subtract: function(x,y){return x - y;},    multiply: function(x,y){return x * y;},    divide: function(x,y){return x / y;},    pow: Math.pow  // 使用內置的Math對象的pow方法};// operate函數接受一個名字作為運算符，在operators對象中查找這個運算符// 然後將它作用於所提供的操作數function operate(operation, num1, num2){    if(typeof operators[operation] === "function")        return operators[operation](num1, num2);    else throw "unknown operator";}// 調用並計算var result = operate("add", "hello", operate("add", " ", "wangwei"));console.log(result);  // hello wangweiresult = operate("pow", 10,2);console.log(result); // 100

當函數作為值的一個典型的應用，就是在Array.sort()方法中使用，該方法用來對數組元素進行排序；因為排序的規則有很多，如：基於數值大小、字母表順序、日期大小、從小到大等，所以sort()方法接受一個自定義函數作為參數，該自定義函數用來處理具體的排序操作，其原理是：對於任意兩個值都返回一個值，以指定它們在在排序後的數組中的先後順序；如：

function compare(x,y){    return x - y;}var arr = [1,88,3,5,12,18,67];console.log(arr.sort(compare));
