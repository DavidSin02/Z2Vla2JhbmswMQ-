| 科技

css就如同一件化妝品，能夠使得單調的網頁變得豐富多彩。在這裝飾網頁的背後，本質上就是對一個一個html標籤進行修飾，如何選擇特定的html標籤，這就尤其關鍵。

  • 選擇器的意義？

選擇器的意義

如果想改變一張圖片的某個部分的顏色，例如將下面圖片左側頭髮改變為紅色，或者將左側的衣服改為黑色。

首先肯定是先選擇一個目標（頭髮，衣服），然後對這個目標進行樣式的修改。

對比圖


一個網頁裡，往往包含了眾多的html標籤，因為有選擇器存在，你才能夠選擇特定的目標（html標籤）來進行樣式的修改。


標籤選擇器

這種選擇器是css最基本的選擇器類型，以html標籤作為選擇器

優點：簡單明瞭，可以對全局標籤做統一樣式標準

缺點：所有同一個標籤都只能使用同一個樣式，無法對特定標籤做差異化

ID選擇器

id 選擇器可以為標有特定 id 的 HTML 元素指定特定的樣式，這個標識如同人的身份證，具有唯一性。

HTML元素以id屬性來設置id選擇器,CSS 中 id 選擇器以 "#" 來定義。

雖然瀏覽器現允許多個相同id出現，但十分不建議這樣，因為在js裡，它無法知道哪個是真的id 選擇器。

優點：可以對某個元素指定特定的樣式

缺點：無法對樣式複用，只能用於一個元素

Class選擇器

Class選擇器是css經常使用的選擇器類型，用於描述一組元素的樣式，class 選擇器有別於id選擇器，class最大的特點是可以在多個元素中使用。

class 選擇器在HTML中以class屬性表示, 在 CSS 中，類選擇器以一個點"."號顯示

優點：可以對某個元素指定特定的樣式；可以對多個元素使用同一個樣式，達到複用效果

缺點：無
