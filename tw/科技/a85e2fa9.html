數據結構系列：面試常問的二叉樹的遍歷和基本應用，不進來看看嗎

2020-11-14
| 科技

上篇數據結構系列：樹？二叉樹？有一說一，這倆貨你不會真的不行 ，我們簡單的分享了樹結構的一般特點和二叉樹結構相關的特性和實現，本篇將基於二叉樹分享相關的遍歷方法(前序，中序，後序，層序)，以及部分應用的算法。


瞭解二叉樹的遍歷

我們知道，線性表的遍歷，我們可以通過for循環或者迭代器。而樹的遍歷，明顯不太可能以相同的方式進行，畢竟樹結構抽象的是層級結構。

二叉樹的遍歷從宏觀的大角度看，分為深度優先遍歷(前序，中序，後序)，廣度優先遍歷(層級)，如圖

概括地說：深度優先遍歷主要包括前序遍歷，中序遍歷，以及後序遍歷，廣度優先遍歷主要是指層級遍歷(每遍歷完整層再遍歷下一層，這裡都是以從左往右的順序進行元素的遍歷)。

對於3種深度優先遍歷來講，最簡單的理解方法就是：將當前節點，當前節點的父節點，當前節點的子節點當作一個整體看，分別判斷左中右的順序，這樣其實就很好理解。

比如說：中序遍歷（左中右的順序），也就是先從最左邊的最左節點開始，然後此子樹的中間節點（當前樹的根節點），然後此子樹的右節點，此時這棵左子樹已經遍歷完成，把當前遍歷完成的左子樹看作一個整體，找到這棵樹是哪個節點的左子節點，然後遍歷此左子節點的中間節點，以此類推。

下面分享這幾種不同的遍歷方法及實現

前序遍歷(根->左->右)

前序理解

若二叉樹為空，則直接返回；否則按照根節點，然後左子樹，右子樹的順序依次訪問(解釋：每訪問一個節點，就把當前節點當作根節點，然後找左右子節點，如果有左節點，則把左節點當作當前樹的根結點，繼續找左右子節點……)

前序實現

# 遞歸實現def pre_order(tree):    if tree:        print(tree.get_root())        pre_order(tree.get_left_child())        pre_order(tree.get_right_child())        # 利用棧的思想def pre_order(tree_node):    s = Stack()    while tree_node or (not s.is_empty):        while tree_node != None:            print(tree_node.get_root())            s.push(tree_node)            tree_node = tree_node.get_left_child()        if not s.is_empty():            tree_node = s.pop()            tree_node = tree_node.get_right_child()

中序遍歷(左->根->右)

中序理解

樹為空，直接返回；否則就從根節點開始(但並不是以根節點開始訪問)，找到最最最左子節點開始，然後當前左子節點的父節點，然後右節點，結果如下

中序實現(基於遞歸)

def in_order(tree):    if tree != None:        in_order(tree.get_left_child())        print(tree.get_root())        in_order(tree.get_right_child())

後序遍歷

後序理解

若樹為空，則直接返回；否則先遍歷左子樹，然後右子樹，最後訪問根節點，結果如下

後序實現(基於遞歸)

def post_order(tree):    if tree != None:    	post_order(tree.get_left_child())        post_order(tree.get_right_child())        print(tree.get_root())

層序遍歷

層序理解

這個最簡單：從每一層開始，按照從左往右的順序遍歷結點

層序實現

# 利用隊列的數據結構def level_order(root):    q = queue()    q.enqueue(root)    while not q.is_empty():        current_root = q.dequeue()        print(current_root.get_root())        if current_root.left_child != None:            q.enqueue(current.left)        if current_root.right_child != None:            q.enqueue(current.right)

說到這，二叉樹的幾種遍歷就差不多結束了，接下來我們分享幾個重要的基於二叉樹的應用

堆(heap)

概念

堆排序是高效排序算法的一個解決方案，它的主要優點是，無論輸入數據如何，它的最壞情況運行時間都是O(n*logn)。

堆排序是優先隊列最主要的實現方法

特點

有這樣一個特點（堆次序）：任何一條路徑都是已經排好序的有序數列

  • 最小堆: 每個節點的數據項都小於或等於其兩個子節點數據，最小的項位於根節點
  • 最大堆: 每個節點的數據項都大於等於其兩個子節點的數據，最大的項位於根節點

堆結構一般有如下幾個接口

heap.is_empty() - 堆是否為空堆，返回布爾值

heap.heappush(item) - 增加數據到堆結構

heap.heappop() - 返回heap最頂部的數據，並刪除

heap.peek() - 返回heap最頂端的項

我們這裡都基於數組實現堆結構，因為對於數組(這裡引申為python的列表)來講，在一個列表中，父節點和子節點的索引存在著一定的關係(如果專門保存一個空元素給列表的第一個位置，那麼子節點的索引永遠等於父節點地板除2的值)

實現

# 最小堆實現（python標準庫也自帶heapq模塊，這裡為自己實現的邏輯）class BinHeap(object):    def __init__(self):        # 這裡給一個初始元素佔位，是為了下面的計算方便(parent_index = 2 * left_child_index = 2 * right_child_index + 1)        self._heap = [0]        self.current_size = 0    # 有序的添加數據    def heappush(self, data):        self._heap.append(data)        self.current_size += 1        # 避免current_size的變化,用新的變量引用        child_index = self.current_size        while child_index // 2 > 0:            if self._heap[child_index] < self._heap[child_index // 2]:                self._heap[child_index], self._heap[child_index // 2] = self._heap[child_index // 2], self._heap[child_index]            child_index = child_index // 2        # 刪除堆中最小元素並返回    def heappop(self):        # 將最小值拋出,並將當前最後的一個值填充給拋出的位置        rm_data = self._heap[1]        self._heap[1] = self._heap[self.current_size]        self.current_size -= 1        self._heap.pop(1)        current_index = 1        while current_index * 2 <= self.current_size:            min_index = self.min_child(current_index)            if self._heap[current_index] > self._heap[min_index]:                self._heap[current_index], self._heap[min_index] = self._heap[min_index], self._heap[current_index]                current_index = min_index        return rm_data    # 輔助函數,幫助pop_data選擇子結點的最小索引並返回最小索引    def min_child(self, current_index):        # 如果只有一個子節點的情況，因為是完全二叉樹，必然有左子節點的存在        if 2 * current_index + 1 > self.current_size:            return 2 * current_index        if self._heap[2 * current_index] > self._heap[2 * current_index + 1]:            return 2 * current_index + 1        return 2 * current_index

表達式樹

表達式樹是包含了表達式的運算數和運算符的二叉樹。這裡主要是將中綴表達式轉換為解析樹結構

大概的思路

  1. 將中綴轉換為全括號表達式來進行操作 ，exp: 3+5*3-2 -> ( ( 3 + ( 5 * 3 ) ) - 2 )
  2. 將所有數據存入列表，準備利用棧結構
  3. 從左到右掃描全括號表達式的每個單詞
  4. 創建一個空樹，當前結點就為根結點
  5. 如果是"("：為當前結點添加一個新的左結點，當前結點下降為這個新結點
  6. 如果是"±*/"，將當前結點的值賦值為此符號，同時為當前結點添加一個新結點作為其右子結點，當前結點下降為這個新結點
  7. 如果當前是操作數，將當前結點的值設為此數，當前結點上升到父結點
  8. 如果當前結點是")"，則當前結點上升到父結點

具體實現

# 將中綴表達式轉換為解析樹結構表示def build_parse_tree(tokens):    token_list = tokens.split(" ")    # 棧用來保存當前結點    s = Stack()    # 最開始構建一個空樹    root = BinaryTree("")    s.push(root)    current_root = root    for i in token_list():        if i == "(":            current_root.insert_left("")            s.push(current_root)            current_root = current_root.get_left_child()        elif isinstance(i, int):            current_root.set_root(int(i))            parent = s.pop()            current_root = parent        elif i in ["+", "-", "*", "/"]:            current_root.set_root(i)            current_root.insert_right("")            s.push(current_root)            current_root = current_root.get_right_child()        elif i == ")":            current_root = s.pop()        else:            raise ValueError("wrong")    return root
