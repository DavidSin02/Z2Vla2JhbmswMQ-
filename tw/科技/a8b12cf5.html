wxPython 對話框

對話窗口或對話框是大多數現代GUI應用程序中不可缺少的一部分。對話的定義是兩個或兩個以上的人之間的對話。在計算機應用程序中，對話框是一個窗口，用於與應用程序 "對話"。對話框用於輸入數據、修改數據、改變應用程序的設置等。對話框是用戶與計算機程序之間重要的交流手段。

我們可以使用預定義的對話框，如消息框、字體或顏色對話框，也可以創建自己的自定義對話框。

一個簡單的消息框

消息框為用戶提供簡短的信息。一個很好的例子是一個CD刻錄應用程序。當光盤刻錄完成後，會彈出一個消息框。

#message_box.pyimport wxclass Example(wx.Frame):    def __init__(self, *args, **kwargs):        super(Example, self).__init__(*args, **kwargs)        self.InitUI()    def InitUI(self):        wx.CallLater(3000, self.ShowMessage)        self.SetSize((300, 200))        self.SetTitle('Message box')        self.Centre()    def ShowMessage(self):        wx.MessageBox('Download completed', 'Info',            wx.OK | wx.ICON_INFORMATION)def main():    app = wx.App()    ex = Example(None)    ex.Show()    app.MainLoop()if __name__ == '__main__':    main()

這個例子顯示的是三秒鐘後的消息框

wx.CallLater(3000, self.ShowMessage)

wx.CallLater在三秒後調用一個方法。第一個參數是一個時間值，在這個時間值之後調用一個給定的方法。參數的單位是毫秒。第二個參數是一個要調用的方法。

def ShowMessage(self):    wx.MessageBox('Download completed', 'Info',         wx.OK | wx.ICON_INFORMATION)

wx.MessageBox顯示一個小的對話框窗口。我們提供三個參數：文本信息、標題信息和標誌。標誌用於顯示不同的按鈕和圖標。在我們的例子中，我們顯示一個OK按鈕和信息圖標。

預定義對話框

wxPython 有幾個預定義的對話框。這些對話框用於常見的編程任務，如顯示文本、接收輸入、加載和保存文件。

消息對話框

消息對話框用於向用戶顯示消息。它們比我們在前面的例子中看到的簡單的消息框更加靈活。它們是可定製的。我們可以改變對話框中顯示的圖標和按鈕。

flagmeaning
wx.OKshow OK button
wx.CANCELshow Cancel button
wx.YES_NOshow Yes, No buttons
wx.YES_DEFAULTmake Yes button the default
wx.NO_DEFAULTmake No button the default
wx.ICON_EXCLAMATIONshow an alert icon
wx.ICON_ERRORshow an error icon
wx.ICON_HANDsame as wx.ICON_ERROR
wx.ICON_INFORMATIONshow an info icon
wx.ICON_QUESTIONshow a question icon

這些是可以用於wx.MessageDialog類的flag

#message_dialogs.pyimport wxclass Example(wx.Frame):    def __init__(self, *args, **kwargs):        super(Example, self).__init__(*args, **kwargs)        self.InitUI()    def InitUI(self):        panel = wx.Panel(self)        hbox = wx.BoxSizer()        sizer = wx.GridSizer(2, 2, 2, 2)        btn1 = wx.Button(panel, label='Info')        btn2 = wx.Button(panel, label='Error')        btn3 = wx.Button(panel, label='Question')        btn4 = wx.Button(panel, label='Alert')        sizer.AddMany([btn1, btn2, btn3, btn4])        hbox.Add(sizer, 0, wx.ALL, 15)        panel.SetSizer(hbox)        btn1.Bind(wx.EVT_BUTTON, self.ShowMessage1)        btn2.Bind(wx.EVT_BUTTON, self.ShowMessage2)        btn3.Bind(wx.EVT_BUTTON, self.ShowMessage3)        btn4.Bind(wx.EVT_BUTTON, self.ShowMessage4)        self.SetSize((300, 200))        self.SetTitle('Messages')        self.Centre()    def ShowMessage1(self, event):        dial = wx.MessageDialog(None, 'Download completed', 'Info', wx.OK)        dial.ShowModal()    def ShowMessage2(self, event):        dial = wx.MessageDialog(None, 'Error loading file', 'Error',            wx.OK | wx.ICON_ERROR)        dial.ShowModal()    def ShowMessage3(self, event):        dial = wx.MessageDialog(None, 'Are you sure to quit?', 'Question',            wx.YES_NO | wx.NO_DEFAULT | wx.ICON_QUESTION)        dial.ShowModal()    def ShowMessage4(self, event):        dial = wx.MessageDialog(None, 'Unallowed operation', 'Exclamation',            wx.OK | wx.ICON_EXCLAMATION)        dial.ShowModal()def main():    app = wx.App()    ex = Example(None)    ex.Show()    app.MainLoop()if __name__ == '__main__':    main()

在我們的例子中，我們已經創建了四個按鈕，並將它們放在一個網格尺中。這些按鈕將顯示四個不同的對話窗口。我們通過指定不同的樣式標誌來創建它們。

def ShowMessage2(self, event):    dial = wx.MessageDialog(None, 'Error loading file', 'Error',         wx.OK | wx.ICON_ERROR)    dial.ShowModal()

消息對話框的創建很簡單。我們將對話框設置為一個toplevel窗口，提供None作為父窗口。這兩個字符串提供了消息文本和對話框的標題。我們通過指定wx.OK和wx.ICON_ERROR標誌來顯示一個OK按鈕和一個錯誤圖標。為了在屏幕上顯示對話框，我們調用ShowModal()方法。
