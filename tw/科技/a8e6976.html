教你如何自定義鼠標右擊菜單

2020-10-29
| 科技

你是否知道鼠標右鍵的菜單是可以自定義的呢？如果你有什麼常用的軟件的話可以添加到鼠標右鍵的菜單裡，這樣能提高你的工作效率的同時，還能省去你尋找軟件的時間，比如裝糊塗先森j就把谷歌瀏覽器放在了鼠標右鍵菜單，打開瀏覽器十分方便。

1. Win + R 彈出運行框，輸入“regedit ”。

2.在註冊表裡面，按照 “HKEY_CLASSES_ ROOT / Directory / background / shell ”依次打開目錄

鼠標右擊 shell 新建 -> 項，輸入添加右擊菜單的名字（理論上自己可隨意命名，但是為了好記最好與你想設置的軟件同名），本人以Chrome 為例。

點擊 Chrome ，在雙擊右邊的默認。在彈出框內，輸入右擊菜單你想顯示的名字。



3.鼠標右擊Chrome，【新建】-> 【項】，名字輸入command（必須為command）。



4.然後選擇command，雙擊右邊的【默認】。在彈出框內輸入軟件的安裝地址。如果你不知道軟件安裝地址的話，可以在桌面上右擊軟件圖標，點擊屬性，在快捷方式選項下的起始位置裡的數值就是軟件的安裝地址。



然後你在桌面右擊就發現已經有谷歌瀏覽器了
