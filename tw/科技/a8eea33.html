R語言作圖：分類變量與連續變量關係的圖形化表達

箱型圖

我們想看看連續變量在某一個分類變量的不同類別上的分佈，一個常用的方法就是畫箱型圖，如果有不清楚箱型圖是什麼的同學請看我之前的文章或者去問度娘：


箱型圖詳解

  • 實例操練

使用的數據集為自帶的diamonds數據集，我們用箱型圖畫出不同的cut上鑽石價格的分佈，代碼如下：

ggplot(data = diamonds, mapping = aes(x = cut, y = price)) +  geom_boxplot()


再畫一個：這次還是用自帶的mpg數據集，我們畫一下不同種類的汽車hwy的分佈（大家可以自行查看變量是啥意思哈，我就不寫了），代碼如下：

ggplot(data = mpg, mapping = aes(x = class, y = hwy)) +  geom_boxplot()


上圖中的箱圖有點亂，這個時候我們其實想能不能將我們圖中的箱圖按照某種順序排一排，比如按照hwy的中位數給所有箱圖排序：

ggplot(data = mpg) +  geom_boxplot(mapping = aes(x = reorder(class, hwy, FUN = median), y = hwy))


這一下就好多了，注意到要實現排序，我們用到了reorder方法，這個方法的第一個參數是要排序的變量，第二個參數是排序依照的標準變量，第三個參數則是排序依照的方法。

如果你的x軸的變量名比較長，可以考慮將圖像反轉90°，用coord_flip()就行，代碼如下：

ggplot(data = mpg) +  geom_boxplot(mapping = aes(x = reorder(class, hwy, FUN = median), y = hwy)) +  coord_flip()
