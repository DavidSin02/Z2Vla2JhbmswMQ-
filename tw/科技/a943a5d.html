問題：父組件異步獲取動態數據傳遞給子組件

例如，在一個index.vue組件裡面有一個content.vue子組件，需要在http請求數據返回後，將list數據傳遞給content.vue子組件。

利用 v-if 可在http請求返回後再顯示。這樣子組件可以返回的http請求數據。

<template>	<div class="indexWrap">		<content :list="list" v-if="isBack"></content>	</div></template><script type="text/javascript"> import content from '../components/content.vue'	export default{		data () {			return {				index:0, list:"", isBack:false			}		},		components : {			 content		},	}</script>
