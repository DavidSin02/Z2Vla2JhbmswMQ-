問題

上下文菜單是非常常見的一種交互界面，用來就近執行一些快捷操作。今天我們就來講一講如何在應用程序中添加上下文菜單的支持。

使用到的消息

Windows中使用WM_CONTEXTMENU消息來表示用戶單擊了鼠標右鍵，而根據一般Windows應用程序的慣例，應用程序通常會彈出一個上下文菜單。

以下是MSDN對此消息的解釋：

WM_CONTEXTMENU

Notifies a window that the user clicked the right mouse button (right-clicked) in the window.

A window can process this message by displaying a shortcut menu using the TrackPopupMenu or TrackPopupMenuEx functions.

根據MSDN的解釋，我們可以對WM_CONTEXTMENU消息建立消息映射，並在消息處理函數中添加對TrackPopupMenu或者TrackPopupMenuEx函數的調用，從而實現上下文菜單的彈出。
