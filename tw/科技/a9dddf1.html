將大於內存的csv文件過濾到Pandas數據幀

2020-10-29
| 科技

通常，我們需要解析大型csv文件，只選擇符合特定標準的行來加載dataframe。但是，對於大於內存的文件，我們不能簡單地在dataframe中加載它並選擇需要的文件。解決方案是用塊解析csv文件，只將需要的行追加到dataframe中。在本文中，我還提供了一些工具來了解內存的一般用法。

首先，一些方法來確定我們需要的不同大小(內存、csv、dataframe)。

查找可用的RAM：

為此，我們需要一個名為psutil的庫

pip install psutil

並使用以下方法獲取

import psutil

svmem = psutil.virtual_memory（）

print（svmem.available）

獲取csv大小:

為此，我們將使用os庫。我們還可以遍歷一個文件夾，如果你的數據集被分割成幾個文件，我們可以得到文件的累積大小:

import os

# only one file dataset:

os.path.getsize('./dataset.csv')

# same but for a folder containing several files:

total_filesize = 0

for filename in os.listdir('./dataset_folder/'):

total_filesize = total_filesize + os.path.getsize(filename)

獲取dataframe的大小：

以下命令返回數據幀df佔用的內存大小，包括其索引。

df.memory_usage(index=True).sum()

另一個選擇是使用sys庫如下：

import sys

sys.getsizeof(df)

處理csv文件

讓我們閱讀csv的前幾行，看看它需要多少內存。

df_sample = pd.read_csv(‘./dataset.csv’, nrows=10)

df_sample_size = df_sample.memory_usage(index=True).sum()

在下面的示例中，我將使用1Gb的RAM進行處理，如果我的結果dataframe變得太大而無法存儲在內存中，我將在處理過程中提醒我。

為此，我們使用chunksize和iterator參數，它決定了每次迭代讀取的行數:

# define a chunksize that would occupy a maximum of 1Gb

# we divide by 10 because we have selected 10 lines in our df_sample

# we then get the integer part of the result

my_chunk = (1000000000 / df_sample_size)/10

my_chunk = int(my_chunk//1)

# create the iterator

iter_csv = pd.read_csv(

‘./dataset.csv’,

iterator=True,

chunksize=my_chunk)

# concatenate according to a filter to our result dataframe

df_result = pd.concat(

[chunk[chunk['my_field']>10]

for chunk in iter_csv])

在連接中，我們傳遞我們想要應用於數據的過濾器，以僅保留與此過濾器匹配的行。在此示例中，我們希望保留my_field列的值大於10的行。但是您可以使用任何經典的pandas方式過濾數據。

下面的完整代碼，所以你不需要定義上面的Python代碼片段：

import psutil

import pandas as pd

import csv

svmem = psutil.virtual_memory()

print (svmem.available)

PATH = r”C:\tmp\dataset\tf_train.csv”

df_sample = pd.read_csv(PATH, nrows=10)

df_sample_size = df_sample.memory_usage(index=True).sum()

print (df_sample_size)

print (df_sample)

# define a chunksize that would occupy a maximum of 1Gb

# we divide by 10 because we have selected 10 lines in our df_sample

my_chunk = (1000000000 / df_sample_size)/10

my_chunk = int(my_chunk//1) # we get the integer part

print (my_chunk)

# create the iterator

iter_csv = pd.read_csv(

PATH,

iterator=True,

chunksize=my_chunk)

# concatenate according to a filter to our result dataframe

df_result = pd.concat(

[chunk[chunk[‘n3’]>0]

for chunk in iter_csv])

print (df_result)
