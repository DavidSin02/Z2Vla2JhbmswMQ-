TikTok視頻批量處理完全教程

2020-10-29
| 科技


之前分享過一篇TikTok無限郵箱註冊教程，今天我們再來看一下TikTok視頻批量處理怎麼操作，主要分四個大綱進行介紹

  1. -素材來源
  2. -批量下載
  3. -視頻處理
  4. -視頻上傳

一、素材來源

1.尋找素材

利用飛瓜等數據分析工具（針對熱門類目，及時獲取信息並第一時間搬運。PS: 很多朋友有疑問為什麼我同樣搬運的視頻，別人爆了我沒爆，有一個原因就是你搬運得太晚了），已抖音為例，點擊進入“熱門素材>熱門視頻>分類，找到自己關注的類目一段時間內比較火的視頻（6小時到30天均可）

這個軟件也支持主播監控功能（也可以在抖音關注某博主，開啟消息通知），目前飛瓜的會員還是比較貴，也可以到某魚或某寶花30-50體驗


2. 人肉搜索

如果是在意內容質量，做冷門方向，可以人肉搜索的方式，效率不會低太多，而且還能刷視頻的過程中培養網感


二、批量下載（注意關閉防毒和防火牆，這類軟件有一定程度的誤判）

1.某音分析師（免費）

支持作者批量解析和下載，可以看到作品的播放量，點贊量等。軟件壓縮包自帶教程，還是比較方面的


通過app端復製作者主頁地址，設置保存路徑，點擊”採集作品“就可以了，根據需要下載播放高的還是全部下載 即可。


2.快手下載

同樣是復製作者主頁的連接即可批量下載該up主的所有視頻，支持去重


快手比較麻煩的是要手動處理did，需要複製up主某個視頻的鏈接到瀏覽器打開，然後從瀏覽器地址上，拷貝did=”xxxx“中”xxxx"的部分，到軟件的根目錄，以“did.txt"保存


3.B站下載

JJDownload支持B站下載，網頁端將視頻鏈接（支持視頻專輯）複製，然後粘貼到“首頁”，軟件會自動彈出需要下載的內容


點擊進去，選擇需要的清晰度下載即可

如果是視頻集，會自動彈出批量下載任務。點擊下面的“批量下載”就會跳出視頻集的下載，

需要強調的是：文件名儘量用英文的，不然容易報錯！



三、視頻批量處理

1.文件名修改：

視頻下載之後，會有各種非常奇怪的命名，上面有些軟件是支持自動按序號命名的，但是大部分還是要我們按照自己的要求，建議大家在處理TikTok視頻之前，先處理好文件命名。

以下提供一段腳本，保存為bat後綴的文件

@echo off

set a=1

setlocal EnableDelayedExpansion

for %%n in (*.MP4) do (

set /A a+=1

ren "%%n" "Y!a!.MP4"

)

注：“MP4"是對應文件的格式（上下兩處），”Y"是對應的文件名


2.視頻字幕處理：

有字幕的內容會被認為更有價值，而且粉絲接受的程度會更高，所以字幕處理是非常有必要的！

安裝後，字幕編輯>新建>打開本地視頻>AI聽譯>一鍵壓制（需要AI聽譯完成，後

側出現字幕後才可以操作）,操作界面參加下圖紅圈：

AI聽譯需要用一點時間，根據視頻長短而定，需要等待一下


在壓制設置裡面，“編碼格式”有“GPU和ABR”兩種，官方的說明是“GPU”需要有獨立顯

卡支持，分辨率會低一些，但是速度特別快。

"ABR"集成顯卡就可以，速度比較慢，但是畫質較好


中文字幕，中文語音處理場景：

導入中文的視頻後，使用AI聽譯，先把中文的語音翻譯出英文的字幕，然後再用朗讀女軟件生成英文語音，最後再把原視頻的聲音關掉，導入生成的英文語音


3.視頻去重：

在“工具箱”裡，選擇“視頻消重”，然後選擇“視頻文件夾”，把視頻上傳，然後在界面裡可設置處理的各項參數




4.單視頻處理：

1): 加封面——可以通過快影App（快手的官方剪輯App）操作，添加視頻後，可以在右側

選擇添加封面

封面圖片設計——主要是統一封面風格，可以把圖片進行3、6、9分割，這樣上傳視頻

的時候，可以把原圖拼回來。主要通過網頁操作，把想分割的視頻上傳，選擇分割數量

操作即可

效果：

加片尾——通過將兩個小視頻的畫中畫的疊加，再把頭像通過畫中畫添加，在“混合

模式”裡選擇濾色模式，最後調整圖片位置即可。


分屏操作——這主要是為了突出對比，或者把某種效果加倍的一個操作。通過剪輯軟

件的畫中畫就可以實現


四、視頻上傳：

這裡採用腳本批量一鍵上傳，腳本的功能主要是模擬人工的操作，目前TikTok的檢查相對來說不太嚴謹
