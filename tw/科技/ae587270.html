PTT VBA｜幻燈片進行文本和段落格式化（一個或兩個文本框）

2020-11-14
| 科技

ppt和excel、word一樣都可以用vba操作。

正和excel中有Excel→Workbook→Worksheet→Range這樣的父子關係對象一樣。

ppt也有PowerPoint→Presentation→Slide→shape這樣的父子關係對象。

其中PowerPoint表示ppt應用程序，Presentation對象表示所有打開的ppt演示文稿、Slide對象表示演示文稿中的幻燈片。

所有形狀組成shapes對象。

所有的幻燈片組成了Slides集合對象。

所有的演示文稿組成了Presentations集合對象。

前面一篇《PTT VBA｜統一或指定幻燈片進行文本和段落格式化》的操作是不考慮幻燈片的標題與內容的關係，統一進行格式操作，不是很妥當，並且沒有位置的考量。

現在的任務是，只考慮一個幻燈片中有一個或兩個文本框的情形，做不同的文本與段落的格式化，並統一文本框的放置位置。

運行效果：

如果不想對每一張幻燈片進行操作，只想對最後一張或指定的一張進行操作的話，只需將遍歷的代碼去掉，並指定幻燈片即可：

For i = 1 To oPres.Slides.Count……Next

替換為：

i = oPres.Slides.CountSet oSlide = oPres.Slides.Item(i)

附代碼1：

Sub allSlides1or2shape()On Error Resume NextDim oPres As PresentationSet oPres = Application.ActivePresentationDim oSlide As SlideDim oShape As ShapeDim tr As TextRangeDim sText As StringDim i As Long, j As LongDim a As Long, b As LongDim fjd As IntegerDim high1 As Integer, high2 As Integer'循環每頁幻燈For i = 1 To oPres.Slides.CountSet oSlide = oPres.Slides.Item(i)Set oSlide = oPres.Slides.Item(i)fjd = oSlide.Shapes.Count'判斷每頁內是一個還是有2個文本框If fjd = 2 Then '兩個文本框設置為上下為標題+內容結構'把頂點值小的文本框仍然設置在上high1 = oSlide.Shapes.Item(1).Tophigh2 = oSlide.Shapes.Item(2).TopIf high1 < high2 ThenSet oShape1 = oSlide.Shapes.Item(1)Set oShape2 = oSlide.Shapes.Item(2)ElseSet oShape1 = oSlide.Shapes.Item(2)Set oShape2 = oSlide.Shapes.Item(1)End IfWith oShape1'設置文本框的寬度和位置.Left = 45.Top = 45.Width = 625End WithIf oShape1.TextFrame.HasText = msoTrue Then '字體設置Set tr = oShape1.TextFrame.TextRangeWith tr.Font.NameAscii = "宋體".NameFarEast = "宋體".Size = 32'.Color.SchemeColor = ppBackground.Color.RGB = RGB(Red:=165, Green:=42, Blue:=42).Bold = msoCTrueEnd WithSet tr = NothingEnd IfSet oShape1 = NothingWith oShape2.Left = 50.Top = 100.Width = 625.TextFrame.TextRange.ParagraphFormat.SpaceWithin = 1.1'設置行距End WithIf oShape2.TextFrame.HasText = msoTrue ThenSet tr = oShape2.TextFrame.TextRangeWith tr.Font.NameAscii = "宋體".NameFarEast = "宋體".Size = 28.Bold = msoCFalseEnd WithSet tr = NothingEnd IfSet oShape2 = NothingElse'單個文本框設置為內容結構Set oShape = oSlide.Shapes.Item(1)With oShape'設置文本框的寬度和位置.Left = 45.Top = 45.Width = 625.TextFrame.TextRange.ParagraphFormat.SpaceWithin = 1.1'設置行距End WithIf oShape.TextFrame.HasText = msoTrue Then'設置文本框的字體和大小Set tr = oShape.TextFrame.TextRangetr.Font.NameAscii = "宋體"tr.Font.NameFarEast = "宋體"tr.Font.Size = 28'tr.ParagraphFormat = 10Set tr = NothingEnd IfSet oShape = NothingEnd IfSet oSlide = NothingNextSet oPres = NothingEnd Sub
