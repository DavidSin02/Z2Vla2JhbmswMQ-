心電學相關檢查

2020-10-29
| 科技

來源：谷苟心電平臺

張芳芳，聶連濤，申繼紅，李中健

鄭州大學第二附屬醫院心電圖科

【關鍵詞】 心電描記術; 向量心電描記術; 胎兒; 運動試驗; 血壓監測，便攜式

自心電圖應用於臨床以來，已經不僅 僅侷限於常規體表心電圖，還衍生出眾多 心電學相關檢查項目，如胎兒心電圖、頭 胸導聯心電圖、Cabrera 導聯心電圖、心 電圖藥物試驗、心電圖運動負荷試驗、動 態心電圖、網絡心電圖、遠程心電圖、心 電向量圖、動態血壓、食管心房調搏、高 血壓分型診斷、直立傾斜試驗等，這些檢 查項目豐富了心電學的內容，更為心血管 疾病的診治提供了多種手段。現就其中臨 床常用的檢查方法進行介紹。

1 胎兒心電圖

1. 1 定義

將胎兒心臟活動時產生的生 物電繪製成圖形，即為胎兒心電圖; 經母 體腹壁記錄的胎兒心電圖稱間接胎兒心電 圖 ( FECG) 。

1. 2 臨床應用

胎兒心電圖是一門跨學 科的生物醫學工程科學，包括產科、心血 管科等; 胎兒心電圖應用於產前診斷和監 護，可以提高胎兒的監測質量，達到及早預防胎兒疾病以及優生優育的目的; 通過 對胎兒心電圖的分析，可以判斷胎兒在宮 內的生長髮育狀況，發現胎兒在妊娠期或 分娩期缺氧、其他病理情況及先天性心臟 病。此外，應用胎兒心電圖還可以協助確 定胎位。胎兒心電圖機與胎兒心電圖檢查 電極安放部位見圖 1 ～ 2。

1.3 正常胎兒心電圖

( 1) 胎兒心電 圖的辨認: 各導聯均出現有規律的、時限 為 0. 02 ～ 0. 05 s、振幅≥5 μV 的波，且 與母體心電圖無關，持續存在 15 s 者即 為正常胎兒心電圖。

( 2) 正常胎心率為 120 ～ 160 次/min，其變異範圍為 10 ～ 25 次/min。( 3) 胎兒 QＲS 波的振幅為 10 ～ 30 μV，時限 0. 03 ～ 0. 05 s ( 見圖 3 ～ 4) 。

1.4 常見的異常胎兒心電圖

( 1) 胎 兒心動過速: 胎心率 ＞ 160 次/min，持續 10 min 以上。

( 2) 胎兒心動過緩: 胎心率 ＜ 120 次/min，持續 10 min 以上。

( 3) 胎 兒心律不齊: 胎心率變化 幅 度 ＞ 30 次/ min。

( 4) 胎兒早搏: 可根據 QＲS 波形態、 時限及代償間歇明確室上性早搏或室性早 搏 ( 見圖 5) 。

( 5) fQＲS 波時限增寬: fQＲS 時限 ＞ 0. 06 s。

( 6) ST 段改變: ST 段抬高或壓低超過 5 μV。

( 7) 胎兒心臟 傳導阻 滯: 胎 心 率 緩 慢， ＜ 70 次/min， 可伴有 fQＲS 波增寬。

2 心電圖藥物試驗

2. 1 心得安藥物試驗

2. 1. 1 定義

應用 β 受體阻滯劑 ( 如心 得安等) 鑑別器質性與功能性 ST － T 改 變的方法稱為心得安試驗。

2. 1. 2 機制

交感神經活動佔優勢時可 使心率加快、復極異常而導致 ST － T 改 變。心得安可降低心肌耗氧量，減慢心 率，使功能性 ST － T 改變恢復正常。 2. 1. 3 方法

心率 ＜ 70 次/min 且無哮喘 病史者可進行檢查。服藥前先描記一份基 礎心 電 圖，然後一次性口服心得安 30 mg，分別於服藥後 30、60、90 min 各做 一次 12 導聯心電圖並與服藥前對比分析， 必要時可加做 120 min 心電圖。

2. 1. 4 判定標準

( 1) 陽性: 異常 ST－ T 改變全部恢復正常。

( 2) 改善: 至少 有 1 個 Ｒ 波佔優勢的導聯 ST － T 改變恢 復正常，或 T 波升高≥50% 。

( 3) 陰性: 無明顯改變。

2. 1. 5 評價

心得安試驗陽性者一般認 為可排除器質性心臟病。陰性者應進一步 檢查，如行運動負荷試驗、冠狀動脈造 影等。

2. 2 氯化鉀藥物試驗

2. 2. 1 定義

應用氯化鉀藥物鑑別器質 性與功能性 ST － T 改變的方法稱為氯化鉀藥物試驗。

2. 2. 2 方法

心率 ＜ 70 次/min 且無胃、 食管、腎病及房室傳導阻滯或高血鉀病史 者可於餐後 30 ～ 60 min 進行檢查。服藥 前先描記一份基礎心電圖，然後一次性口 服 10% 氯化鉀藥物溶液 100 ml，分別於 服藥後 30、60、90 min 各做一次 12 導聯 心電圖與服藥前對比分析，必要時可加做 120 min 心電圖。

2. 2. 3 判定標準

( 1) 陽性: 異常 ST － T 改變全部恢復正常。

( 2) 改善: 至少 有 1 個 Ｒ 波佔優勢的導聯 ST － T 改變恢 復正常，或 T 波升高≥50% 。

( 3) 陰性: 無明顯改變。

2. 2. 4 評價

氯化鉀藥物試驗陽性時一 般認為可排除器質性心臟病。陰性者應進 一步檢查，如行運動負荷試驗、冠狀動脈 造影等。

2. 3 阿托品藥物試驗

2. 3. 1 定義

應用阿托品藥物確定和鑑 別病態竇房結綜合徵、竇性心動過緩與竇 房阻滯的方法稱為阿托品藥物試驗。

2. 3. 2 機制

阿托品系迷走神經阻滯劑， 能降低迷走神經的張力，而使心率增快 ( 竇房結功能正常者) ，但當竇房結功能 有病變時，阿托品則不能使竇性心率增 快，故可用以診斷與鑑別病態竇房結綜 合徵。

2. 3. 3 方法

先記錄一份 12 導聯心電圖 作為初始圖進行對照。再按體質量將阿託 品 ( 0. 04 mg / kg) 加入 0. 9% 氯化鈉溶液 2 ～ 5 ml 直接快速靜脈注射後，測即刻、 1、2、3、5、7、10、15、20 min 的心電 圖，並與對照對比分析。

2. 3. 4 判定標準

( 1) 陽性: 注射後 心率 ＜ 90 次∕ min，或心率雖≥90 次∕ min，但系房室交界性心律，或出現竇性 停搏、竇房阻滯，均提示病態竇房結綜合 徵。

( 2) 陰性: 注射後心率≥90 次∕ min，或竇房阻滯、竇性停搏消失，說明 竇房結功能正常，提示迷走神經功能佔 優勢。

2. 3. 5 評價

本法用以診斷及鑑別病態 竇房結綜合徵。該試驗簡便、安全，但偶 可引起室性心動過速與室性顫動。可能出 現假陽性和假陰性的結果，如有條件可做 經食管心房調搏檢查確定診斷。

3 心電圖運動負荷試驗

3. 1 定義

心電圖運動負荷試驗是通過一定量的運動增加心臟負荷，觀察心電圖 變化，對已知或懷疑患有心血管疾病，尤 其是冠心病患者進行臨床評估的方法。與 冠狀動脈造影相比，雖然該試驗有一定比 例的假陽性與假陰性，但由於其簡便實 用、費用低廉、無創傷、符合生理情況等 優勢，被公認為是一項重要的臨床心血管 疾病檢查方法。

3. 2 適應證與禁忌證

3. 2. 1 適應證

用於瞭解冠心病患者的 病變程度、心肌梗死患者出院前的心功能 評估。 3. 2. 2 絕對禁忌證

急性心肌梗死 ( 2 d 內發病) 、高危不穩定型心絞痛、未控制 或伴有症狀及血流動力學障礙的心律失 常、有症狀的嚴重主動脈狹窄、未控制的 伴有症狀的心力衰竭、急性肺栓塞或肺梗 死、急性心肌炎或心包炎、急性主動脈 夾層。

3. 2. 3 相對禁忌證

左冠狀動脈主幹狹 窄、中度狹窄的瓣膜性心臟病、電解質異 常、嚴重高血壓、快速性或緩慢性心律失 常、肥厚型心肌病和其他形式的流出道梗 阻、高度房室阻滯、精神或身體異常不能 運動者。

3. 3 方法

有二階梯運動試驗、踏車運 動試驗和活動平板運動試驗 ( 見 圖 6 ～ 7) 。

3. 3. 1 運動前準備

( 1) 檢查者準備: 檢查儀器設備連接完好，確保儀器功能良 好，配以專業醫師監視患者運動過程中的 心電圖和血壓，同時逐次排除禁忌證、權 衡相對禁忌證。

( 2) 患者準備: 在檢查 12 h 內禁止做特殊運動，2 h 內無飲食， 穿寬鬆的衣服和舒適的運動鞋以保證達到 自身最佳的運動狀態。

( 3) 運動方式採 用運動負荷逐級增量法，患者保持運動速 度不變，運動過程中出現以下指標應停止 運動。

3. 3. 2 終止運動的指徵

絕對指徵: 試 驗中運動負荷增加，但 SBP 較基礎水平 下降 超 過 10 mm Hg ( 1 mm Hg = 0. 133 kPa) ，並伴隨其他心肌缺血徵象; 中重 度心絞痛; 增多的神經系統症狀 ( 如共 濟失調、眩暈、近似暈厥狀態) ; 低灌注 表現 ( 紫紺或蒼白) ; 由於技術上的困難 無法檢測心電圖或 SBP; 受試者要求終 止; 持續性室性心動過速; 在無診斷意義 Q 波的導聯上出現 ST 段 抬 高 ( ≥1. 0mm，非 V1 或 aVＲ 導聯) ; 出現 ST 段或 QＲS 波改變，如 ST 段過度壓低 ( 水平型 或下垂型 ST 段壓低 ＞ 2 mm) 或顯著的電 軸偏移; 除持續性室性心動過速之外的心 律失常，包括多源性室性早搏，室性早搏 三聯律，室上性心動過速，心臟傳導阻滯 或心動過緩; 勞累、氣促、哮喘、下肢痙 攣、跛行; 束支阻滯或室內阻滯與室性心 動過速無法鑑別; 胸痛增強; 高血壓反應 ( SBP ＞ 250 mm Hg 或/和 DBP ＞ 115 mm Hg) 。

3. 4 診斷標準

3. 4. 1 運動試驗陽性標準

( 1) ST 段 水平型或下斜型壓低，J 點後 80 ms 處壓 低≥0. 1 mV，持續時間≥2 min。

( 2) ST 段弓背抬高≥0. 1 mV。

( 3) 運動中出現 典型心絞痛。

3. 4. 2 可疑陽性標準

( 1) ST 段水平 型或下斜型壓低，J 點後 80 ms 處壓低為 0. 05 ～ 0. 10 mV，持續時間≥1 min。

( 2) ST 段上斜型壓低，J 點後 60 ms 處壓低為 ≥0. 15 mV，或 ST 段斜率 ＜ 1 mV / s，持 續時間至少 1 min。

( 3) 單純 U 波倒置。

( 4) 運動時 SBP 下降≥1. 33 kPa。

( 5) 運動中出現不典型心絞痛。

4 動態心電圖

4. 1 定義

動態心電圖又稱長程心電圖、 長時間心電圖及 Holter，通過讓受試者隨 身攜帶一個記錄儀，在不影響一般生活的 情況下長時間 ( 24 ～ 72 h) 動態記錄的心 電圖。其優點為: 可以對不明原因的、短暫的常規心電圖不易捕捉到的有心前區不 適症狀者進行追蹤，可以對心律失常、心 肌缺血、心肌梗死進行定性、定量、定位 的綜合分析，以判斷預後，為臨床提供診 治依據。目前推薦使用 12 導聯動態心電 圖，其記錄信息量大，時間長，獲取的心 電資料更豐富，且有利於建立數據庫。

4. 2 記錄器的安裝

4. 2. 1 患者準備

( 1) 預約 ( 提醒患 者合理安排時間，佩戴期間不能做 X 線 胸片、彩色超聲、CT、磁共振、理療等 影響記錄的檢查和治療) ;

( 2) 檢查前 1 天洗澡、穿不影響檢查的寬鬆衣服; 住院 患者儘量留院觀察，儘可能多地減少干擾 因素;

( 3) 根據檢查需要考慮是否停藥;

( 4) 如為門診患者應留下單位、住 址、 電話 號 碼、身份證等有效證件及真實 信息。

4. 2. 2 物品準備

記錄器、導聯線、鹼 性電池、一次性電極、膠布、酒精棉球、 砂紙、生活日誌記錄單。

4. 2. 3 皮 膚 處 理

( 1) 患 者 站 立 位、 坐位或平臥位;

( 2) 選擇各導聯的適宜 安放 部 位，儘量選擇肌肉最少的部位 ( 肋骨、胸骨柄及兩側) ;

( 3) 有胸毛者 先剔除，砂紙打磨，再清潔皮膚;

( 4) 牢固粘貼電極 ( 皮膚過敏者可選用抗過 敏電極) 。

4. 2. 4 連接電極與導聯線 12 導聯動態

心電圖電極安放部位: ＲA 在右鎖骨下窩 外 1 /3 處; LA 在左鎖骨下窩外 1 /3 處; LL 在左鎖骨中線第 7 肋緣; ＲL 在右鎖骨 中線第 7 肋緣; V1 在胸骨右緣第 4 肋間; V2 在胸骨左緣第 4 肋間; V3 在 V2 與 V4 導聯中點; V4 在左鎖骨中線第 5 肋間; V5 在左腋前線平 V4 導聯; V6 在左腋中線平 V4 導聯。

4. 2. 5 固定電極導線

4. 2. 6 檢查事項

裝電池、確認記錄器 是否運行正常。

4. 2. 7 患者注意事項

愛護記錄器，填 寫生活日誌，按時返回檢查室等。

4. 3 卸機與回放分析

( 1) 卸機時注 意事項: 觀察記錄器及導聯線連接是否良 好。

( 2) 分析人員要求: 具有豐富的臨 床心電圖工作經驗和分析複雜心律失常心 電圖的能力; 熟悉所使用的動態心電圖性 能及其基本操作、分析技術。

( 3) 瞭解 患者的臨床情況。

( 4) 分析時應注意的 問題: 採用自動分析和人工校正方式; 報 告總結和分頁報告必須與心電圖內容保持 一致; 提供心電圖證據: 各類心律失常或 不正常的診斷要有全圖為據; 對照日誌分 析動態心電圖。

4. 4 留圖時注意事項

( 1) 選擇波形 平穩無干擾的心搏進行分析;

( 2) 選擇 異常波段動態變化處留圖;

( 3) 特殊事 件留圖: 選取最具有代表性的特殊事件心 電圖，保留髮作前、中、後的動態變化趨 勢和時間分佈。

( 4) 起搏器: 保留自主 心搏圖，感知、起搏功能正常及不正常 圖，心房、心室起搏及房室順序起搏圖， 真假融合波圖等典型圖;

( 5) 總報告的 前半部分為數據留圖，後半部分為異常圖 例留 圖，最後為患者生活及病情日誌 記錄。

4. 5 動態心電圖報告模板( 來自本院心 電圖科)

( 1) 基礎心律性質 ( 竇性、 異位、起搏節律等) 、最小心率、最大心 率、平均心率、心搏總數、心率動態變化 等，起搏節律應註明起搏方式和起搏心搏 數。

( 2) 房性早搏及房性心律失常 ( 注 明數目、聯律、正常或異常) ，註明起源 部位，發 生 的 起 始、持續及終止過程。

( 3) 室性早搏及室性心律失常 ( 註明數 目、聯律、正 常 或 異 常) ，注 明 起 源 部 位，發生的起始、持續及終止過程。

( 4) 是否有 ST － T 段異常動態變化、異常程 度、起始、持續及結束過程。

( 5) 心率 變異性分析 ( 正常、降低、顯著降低) 。

( 6) 睡眠窒息危險分析結果 ( 正常、異 常) 。

( 7) 竇性心率振盪分析結果 〔正 常、異常，異常時註明震盪初始 ( TO) 、 震盪斜率 ( TS) 值〕。

( 8) T 波電交替 ( 有、無) ，有無 QT 間期延長。

( 9) 心率減速 力 及 連 續 心 率 減 速 力 分 析 結 果 ( 正常、異常) 。

( 10) 有無傳導阻滯及是 否與時相、迷走神經張力有關。

( 11) 長 Ｒ － Ｒ 間 期 分 析: 發 生 時 間、數 目、原 因、性 質、有無逸搏及逸搏心律等。

( 12) 患者自述不適時或按壓事件按鈕所 對應的心電圖表現。應注意，動態心電圖 系統及軟件不同，患者臨床情況不同，診 斷報告可適當增減條目。

4. 6 動態心電圖參考值

( 1) 24 h 心 搏數 8 ～ 12 萬，最小心率 40 ～ 60 次/min， 平均心率 60 ～ 80 次/min，最大心率 － 最 小心率 ＞ 40 次/min。停搏 ＜ 2. 0 s。ST 段 下移 ＜ 1 mm，ST 段抬高 ＜ 2 mm。以 Ｒ 波 為主導聯 T 波直立。房性早搏 ＜ 100 個/ 24 h。室性早搏 ＜ 100 個/24 h。

( 2) 心率 變異性 ( HＲV) : 瞭解自主神經受損情況 和心肌梗死後的預後評價，是目前認為能 定量分析自主神經功能的惟一方法。24 h 正常 Ｒ － Ｒ 間 期 的 標 準 差 ( SDNN － 24 h) : ≥100 ms ( 正 常) ， ＜ 100 ms ( 降 低) ， ＜ 50 ms ( 顯著降低，提示可能誘 發室性心律失常) 。

( 3) 睡眠窒息危險分 析: 通過睡眠呼吸監測，初步篩選阻塞 性、中樞性和混合性呼吸困難。

( 4) 竇 性心率振盪: 是竇房結對室性早搏的一種 雙向生理反應，表現為一次短暫的初期心 率加速和緊隨其後的心率減慢。可通過兩 個參數定量表達: TO 參考值 ＜ 0、TS 參 考值 ＞ 2. 5 ms/ＲＲI。危險指數: TO≥0、 TS≤2. 5 ms/ＲＲI。該指標是最新的一項 強有力的心肌梗死患者死亡的獨立預測 指標。

5 臨床心電向量圖

5. 1 定義

指按照時間順序，將心臟除 極與復極的各瞬間綜合向量的合力軌跡依 次相連所構成各自的向量環。心向量環是 一個佔有多維空間的立體圖形，也稱空間 向量環或立體向量環。心電向量圖從空間 角度分析心臟電活動，能真實反映心臟電 激動時瞬間綜合向量的方位、振幅、運行 方向、速度和時間 ( 見圖 8) 。

5. 2 心電向量圖的原理

向量是物理學 的一個名詞，其表示力的大小和方向。通 常用箭矢表示，箭矢的方向代表向量的方 向，長度代表向量的大小。心電向量圖採 用 Frank 導聯體系。

5. 3 心電向量電極安放部位

A 電極: 沿胸骨旁第 5 或第 4 肋間水平向左與左腋 中線相交點 ( 相當於 V6 位置) 。I 電極: 向右與右腋中線相交點 ( 相當於 V6Ｒ 位 置) 。E 電極: 與前正中線相交點。M 電 極: 與後正中線相交點。C 電極: A、E 兩點在體表連線的中點 ( 相當於 V4 位 置) 。F 電極: 與左下肢連接。H 電極: 與頸背部連接。

5. 4 臨床應用範圍

可用於診斷心房肥 大、心 室 肥 大、房 內 阻 滯、室 內 阻 滯 ( 右束 支、左 束 支、左 前 分 支、左 後 分 支、左間隔支、浦肯野纖維及心室肌阻 滯) 、預激綜合徵、心肌梗死、心肌缺血 等。作為互補，可以彌補心電圖在心血管 疾病診斷方面的不足。

5. 5 參考值

( 1) P 環: 很小，呈狹長 橢圓形，位於左下方稍偏前或偏後，常有 一個或幾個以上的小凹陷或突出。額面 ( F) : P 環最大，呈梨形或狹長形，略不 規則，P 環離心支常有一凹跡，P 環位於 左下方 30° ～ 90°，振幅 ＜ 0. 2 mV，呈逆 時針向。橫面 ( H) : P 環比 F 面 P 環小， 呈長圓形或 “8” 字形，呈逆時針向。P 環位於左側偏前或偏後 60° ～ － 60°，振 幅 ＜ 0. 1 mV。右側面 ( ＲS) : P 環呈狹長 形或梨形，位於下方 60° ～ 120°，呈順時 針向，振幅 ＜ 0. 18 mV。

( 2) QＲS 環: 總 時限 ＜ 100 ms，終未部光點密集時間 ＜ 30ms。F: 順時針向運行 59% ，逆時針向運 行 41% ; 最大向量位於 ( 44° ± 13°) ，振 幅 ( 1. 57 ± 0. 34) mV。H: 均為逆鐘向運 行; 最 大 向 量 位 為 ( 19° ± 4°) ，振 幅 ( 1. 23 ± 0. 3) mV; 前方面積≤QＲS 環總 面積的 40% ～ 60% 。ＲS: 除個別呈逆時 針向或 “8”字形運行外，均為順時針向 運行; 最 大 向 量 為 ( 84° ± 20°) ，振 幅 ( 1. 03 ± 0. 33) mV。3 個面 QＲS 環最長或 最寬向量振幅均 ＞ 1. 0 mV。

( 3) ST 向 量: 正常可出現位於左前下方的 ST 向量， 但振幅 ＜ 0. 1 mV。 ( 4) T 環: 橢圓或菱 形，至少在一個平面展開，離心支與歸心 支交角 ＜ 30°，長/寬 ＞ 2. 5，T/Ｒ ＞ 0. 25。 離心支起始部運行速度最慢，中間部稍 快，峰部又稍慢，歸心支較快。F: 99% 順時針向運行，1% 逆時針向運行，最大 T 向 量 位 於 ( 40. 0° ± 10. 3°) ， 振 幅 ( 0. 43 ± 0. 15) mV。H: 99% 逆時針向運 行，1% 重疊，最大 T 向量位於 ( 22. 0° ± 19. 7°) ，振幅 ( 0. 33 ± 0. 15) mV。ＲS: 99% 順時針向運行，1% 重疊，最大 T 向 量 為 ( 67. 0° ± 25. 1°) ， 振 幅 ( 0. 33 ± 0. 12) mV。

( 5) QＲS － T 夾角: F ＜ 40°、H ＜ 60°、ＲS ＜ 120°。

6 動態血壓

6. 1 定義

使用動態血壓記錄儀測定某 例患者 24 h 內每間隔一定時間的血壓值 稱為動態血壓，由血壓水平、血壓晝夜節 律和血壓變異性三部分組成。

6. 2 血壓水平

6. 2. 1 血壓水平

通常監測 24 h 平均血 壓、白天平均血壓、夜晚平均血壓、最高 血壓值和最低血壓值等指標。晝夜分為: 白天: 6: 00 ～ 22: 00，夜晚: 22: 00 ～ 6: 00。正常情況下: 白天平均血壓 ＞ 24 h 平均血壓 ＞ 夜晚平均血壓。

6. 2. 2 SBP、DBP 負荷

血壓負荷是指 測量所得的血壓值超過高血壓界限的比 例。如血壓負荷 21% 即代表有 21% 的血 壓數據超過高血壓界限，超過的比例越 高，高血壓的可能性越大。

6. 3 血壓 24 h 生理特點

通常呈明顯的 波動 性，即 “二 峰 一 谷”。第 1 峰: 8: 00 ～ 10: 00; 第 2 峰: 16: 00 ～ 18: 00; 低谷: 1: 00 ～ 3: 00。

6. 4 動態血壓臨床應用

( 1) 發現早 期高血壓患者;

( 2) 明確高血壓的診斷;

( 3) 確定高血壓類型;

( 4) 判斷高血壓 病情程度;

( 5) 評估高血壓患者的預後;

( 6) 指導及評價降壓治療。

6. 5 動態血壓參考值

( 1) 有效測壓 百分率 ( 有效測壓數/測量總次數) 參考 值 ＞ 80% 。

( 2) 白天 SBP 平均值 ＜ 140 mm Hg，DBP 平均值 ＜ 90 mm Hg。

( 3) 夜晚 SBP 平均值 ＜ 120 mm Hg，DBP 平均 值 ＜ 80 mm Hg。

( 4) 收縮期血壓負荷 ＜ 10% ，舒張期血壓負荷 ＜ 10% 。

( 5) 夜 晚血壓下降率 SBP ＞ 10% ，DBP ＞ 10% 。 夜晚血壓下降率: SBP = ( 白天收縮期血 壓平均值 － 夜晚收縮期血壓平均值) /白 天收縮期血壓平均值，DBP = ( 白天舒張 期血 壓 平 均 值 － 夜 晚 舒 張 期 血 壓 平 均 值) /白天舒張期血壓平均值。

6. 6 動態血壓報告格式舉例( 資料來自 本院心電圖科)

( 1) 有效測壓百分率 94. 9% ( 參考值 ＞ 80% ) 。

( 2) 白天 SBP 平均值 144 mm Hg ( 異常) ，DBP 平均值 83 mm Hg ( 正常) ( 參考值 ＜ 140 /90 mm Hg) 。

( 3) 夜間 SBP 平均值 134 mm Hg ( 異常) ，DBP 平均值 81 mm Hg ( 異常) ( 參考值 ＜ 120 /80 mm Hg) 。

( 4) 收縮期 血壓負荷 65. 5% 白天 ( 異常) ，75. 0% 晚 上 ( 異常) 。舒張期血壓負荷 20. 7% 白天 ( 異常) ，50. 0% 夜晚 ( 異常) ( 參考值 ＜ 10. 0% ) 。

( 5 ) 夜間血壓下降率 SBP 6. 9% ( 異常) ，DBP 3. 6% ( 異常) ( 參 考值 ＞ 10. 0% ) 。

7 高血壓分型診斷

7. 1 概述

我科自 2000 年至今採用常規 心電圖、動態血壓、動脈功能檢查 “三 合一”的方法分型診斷高血壓，取得了 明顯社會效益和經濟效益。

7. 2 心功能檢測儀

主要為心輸出量的 測定和各房室及大血管壓力的測定，還包 括時相、阻抗、血容量、血管彈性模量等 系列參數的測定。心功能檢測儀是同步採 錄心電波、動脈壓力波、靜脈壓力波、心 音四 個 信 號，達到檢測左右心功能的 目的。

7. 3 高血壓分型診斷與治療

心肌力能 指標: 左心搏指數 ( LSI) 、左心排指數 ( LCI) 、心臟總功率 ( CWT) 、左室噴血 壓 ( JP) 、左室總泵力 ( LTPF) 、心臟總 功率 ( LCWT) 。

7. 3. 1 高動力型高血壓平均 SBP ( PSM) 顯著升高，心肌力能顯著增高 ( JP↑，TPF↑，LCWT↑) 。

7. 3. 2 高排量型高血壓

左心搏指數顯 著增高，左心排指數顯著增高，心肌力能 可增高 ( LSI↑，LCI↑，CWT↑，JP↑， TPF↑，LCWT↑) 。

7. 3. 3 高阻抗型高血壓

血管阻抗指標: 總阻抗 ( ＲTM) 、動脈特 性 阻 抗 ( ＲS) 、 左室噴血阻抗 ( CＲ) 。平均 DBP ( PDM) 顯著增高，血管阻抗顯著增高 ( ＲTM↑， ＲS↑，CＲ↑) 。

7. 3. 4 高容量型高血壓

容量參數: 有 效循 環 容 量 ( GCV) 、左 室 舒 張 末 容 量 ( LVDV) 、每搏輸出量 ( SV) 、心排血量 ( CO) 。容量參數顯著增高 ( GCV ↑， VDV↑，SV↑，CO↑) 。

7. 3. 5 混合型高血壓

以上二項或三項 兼有之。

8 食管心房調搏術

8. 1 定義

食管心房調搏術是經食管電 極法對心臟進行心外調搏的技術，其具有 無創、簡便、安全、便於多次重複，無需 昂貴設備等優點。

8. 2 食管心電圖

將電極導管放在食管 中段靠近左心房部位描記心電圖稱為食管 心電圖。8. 3 食管電極

臨床多選用改進的食管 導聯 電 極: 4 極、6 極、8 極導管 ( 最 常用) 。8. 4 食管心電圖的描記方法

按照常規 心電圖記錄方法連接被檢查者的肢體導聯，採用利寧膠漿滴鼻及咽後壁 ( 本院 心電圖科採用利寧膠漿法取得較好檢查效 果) ，囑其吞嚥，潤滑鼻腔和食管後，輕 柔將頭部已浸沾利寧膠漿的食管電極插到 食管中段近左心房位置，將食管電極遠端 和導聯線相連，觀察食管心電圖 P 波形 態，確定電極位置，即可確定不同深度的 食管心電圖。

8. 5 食管心電圖圖形

根據電極位置由 深至淺可記錄到不同形態的圖形。

8. 6 食管心電圖應用價值

( 1) 用於 複雜心律失常的鑑別診斷;

( 2) 用於食 管心房調搏電極導管的定位;

( 3) 用於 心臟驟停患者的臨時起搏。

8. 7 食管心房調搏的應用

( 1) 評定 竇房結功能;

( 2) 評定房室傳導功能;

( 3) 心臟各部位不應期的檢測;

( 4) 診斷房室結雙徑路，揭示房室傳導的裂隙現 象;

( 5) 旁路電生理檢查;

( 6) 室上性 心動過速電生理檢查;

( 7) 室性心動過 速電生理檢查;

( 8) 對臨床藥理評價。

9 直立傾斜試驗

9. 1 定義

通過改變患者體位 ( 從平臥 位到傾斜位) 來評價血液重新分配後血 管與神經之間的反射，是血管迷走神經性 暈厥診斷和鑑別診斷的重要方法。

9. 2 被檢者準備

試驗前 3 天停用影響 自主神經的藥物，試驗前禁食至少 4 h， 在試驗的過程中保持放鬆，如出現不適立 刻向醫生說明。

9. 3 醫生準備

( 1) 詢問病史，排除 腦血管病變、高血壓、低血糖、精神障礙 等引起的暈厥;

( 2) 告知受試者檢查的 目的及過程，囑其放鬆並保持平靜呼吸， 密切觀察其身體狀態;

( 3) 連接心電監 護及血壓計;

( 4) 檢查開始，量取平臥 位血壓一次，啟動傾斜床開關，將其從平 臥位改成傾斜位 ( 60° ～ 80°) ，測量即刻 血壓一 次，接 著 每 隔 5 min 測 量 血 壓 1 次，20 min 為 1 個週期，密切觀察受試者 心電圖情況及身體反應 ( 有無視物模糊、 耳鳴、噁心、嘔吐、面色蒼白、出汗、心 慌、乏力、暈厥) ，若出現上述反應迅速 將其改平臥位，並監測心電和血壓情況。 若 20 min 內無上述反應，則需要藥物誘 導，常用藥物有異丙腎和硝酸甘油 ( 因 無需建立靜脈通道，且陽性率並不低於異丙腎，故 臨 床 多 使 用) ，含 化 硝 酸 甘 油 0. 3 mg，再重複上述操作一次。

9. 4 判斷標準

( 1) 血壓下降: SBP≤ 80 mm Hg 和/或 DBP≤50 mm Hg 或平均 動脈壓下降≥25% ;

( 2) 心率下降: 竇 性心動過緩 ( HＲ ＜ 50 次/min) ，竇性停 搏代以交界性逸搏心律，一過性Ⅰ度以上 房室阻滯或長達 3s 以上心臟停搏;

( 3) 接近暈厥: 試驗過程中出現面色蒼白、出 汗、胸悶、過度換氣、繼之黑朦、反應遲 鈍、但無意識喪失、恢復平臥後症狀消 失;

( 4) 暈厥: 突發短暫的意識喪失伴 不能維持自主體位，暈厥前可伴或不伴有 暈厥先兆，恢復平臥後，意識可在幾秒鐘 後恢復正常。具備以上 ( 1) 或 ( 2) 任 意一項，加上出現 ( 3) 或 ( 4) 即可判 為陽性。

9. 5 暈厥患者知識指導

試驗後對陽性 結果者進行健康宣教，指導患者平時可增 加水、鹽攝入量，靠牆站立訓練每次 30 min，2 次/ d，避免可能觸發暈厥發作的 各種誘因，如高溫、脫水、過度疲勞、長 時間站立等。部分暈厥患者的發病與疼 痛、見血、恐懼等情感因素有關，尤其是 年輕女性。情感因素可引起大腦高級中樞 的活動異常。針對不同患者實施個體化宣 教，暈厥多次發作者考慮藥物治療。
