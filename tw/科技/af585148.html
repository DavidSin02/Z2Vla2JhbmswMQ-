c語言入門24，三分鐘弄清函數指針和指針函數，C這麼愛咬文嚼字嗎

2020-11-14
| 科技

在第22節，我們弄清了數組指針和指針數組的區別和聯繫，現在趁熱打鐵，一起來看看函數指針和指針函數。

初學者看到這裡，可能會覺得 C 語言是一門喜歡咬文嚼字的編程語言，其實弄懂了，你自然也會這麼稱呼它們的。


函數指針

回想一下 22 節，我們只要把“數組”像“int”一樣看作是一種數據類型，數組指針和指針數組就具有很明顯的區別了。int 指針是指向 int 型數據，那數組指針就指向數組的指針。int 數組是一個存放 int 數據的數組，那指針數組就是存放指針的數組。在 C 語言中，函數也是一種類型，那函數指針就是指向函數的指針。

函數指針怎麼定義呢？請看下面這個例子：

#include <stdio.h>void hello(char* name){ printf("hello, %s\n", name);}int main(){ void (*f)(char *) = hello; f("Jim"); return 0;}


分析定義函數指針的表達式 void (* f )(char * ) 和第22節分析數組指針的定義方式是相似的，() 的優先級高，所以 f 先和 * 結合，因此 f 首先是一個指針，什麼類型呢？* f 外面是一個函數原型的格式，參數是 char* ，返回值是 void，所以 f 是一個函數類型的指針。而 hello 函數恰好參數是 char* 類型，返回值是 void 類型，因此可以讓函數指針 f 指向 hello。也能寫成：

void (*f)(char *) =&hello;

通過函數指針也能調用它指向的函數，例如上面的 f(“Jim”)，也可以以 (* f)(“Jim”) 的形式調用。應該注意到了，單獨寫 f 的時候，它是一個函數指針，並沒有函數調用。想調用函數，需要加上“()”符號。

可以把 “()”理解為函數調用的運算符，它的左側要求是函數指針。

初學者看到這裡，可能有些疑問，例如為什麼初始化 f 時，既可以把 hello 直接賦值給 f，也可以把 &hello 賦值給 f？再例如，為什麼通過 f 調用函數時，既可以直接 f(“Jim”)，也可以 (* f)(“Jim”)？其實做個試驗就明白了，我們把 &hello，hello，* hello 地址打印出來，請看：

printf("&hello: %p\n", &hello);printf(" hello: %p\n", hello);printf("*hello: %p\n", *hello);

編譯執行，會發現其實這三者是相等的，所以上面介紹的使用方式雖然略有不同，程序也可以正常工作。


不過，使用函數指針時，有些程序員更習慣下面這麼用，這樣一眼就能看出使用的是函數指針，當然，究竟使用哪種主要取決於個人習慣。

  • 把 hello 當作函數指針時，使用 &hello。
  • f 是指向函數的指針，通過 f 調用函數時，使用 (* f)(…)。

好了，現在知道怎麼使用函數指針了，只是，函數指針的定義方式有些繁瑣，如果我想定義多個同樣類型的函數，要寫好多重複代碼，這不是“不優雅”嗎？的確，應該儘量避免重複代碼，好在 C 語言有 typedef 關鍵字。

typedef unsigned char uchar;uchar i = 0;


typedef 關鍵字使得我們可以用其他符號代替較繁瑣的數據類型，例如上例代碼，我們使用 uchar 符號代替了 “unsigned char”類型，以後想定義 unsigned char 類型的變量，直接用 uchar 就可以了，在上例中，我們用 uchar 定義了一個 unsigned char 類型的變量 i。函數指針的定義方式也可以用 typedef 關鍵詞簡化：

typedef void (*FUN)(char *);FUN f = &hello;(*f)("Jim");

我們使用 FUN 符號代替了函數指針類型，這種函數的參數類型為 char* ，返回值為 void。以後遇到這種類型的函數，直接使用 FUN 符號就可以定義出對應的函數指針。

指針函數

似乎又是一個比較陌生的詞，不過我們對 int 函數還是挺熟悉的，int 函數是返回 int 數據的函數，對比一下，那指針函數就是返回指針的函數。就這麼簡單？是的，就這麼簡單。下面給出一個指針函數的使用實例，請看：

#include <stdio.h>char *str1 = "hello, i am Jim\n";char *str2 = "hello, i am Tom\n";char *get_str(){ static char i = 0; if((i ++)%2) return str1; else return str2;}int main(){ printf("%s", get_str()); printf("%s", get_str()); return 0;}

get_str 函數返回值是 char 指針類型的，所以 get_str 是一個指針函數。它的調用方式和 int 等其他類型函數的調用一樣，指針函數實在沒什麼特別的。


總結

指針函數是一個函數，它的返回值是指針類型。函數指針是一個指針，它指向函數，通過函數指針可以調用它指向的函數，通過函數指針，我們可以讓 C 語言仿 C++ 實現“類”的封裝，接下來會介紹。
