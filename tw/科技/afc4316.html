Python 快速設置 Excel 表格邊框

2020-10-29
| 科技

1. 安裝 xlwings

直接安裝用 pip install xlwings2. 導入 xlwingsimport xlwings as xw

3. 打開 Excel 文件

# 打開存好的 Excelapp = xw.App() # 設置應用wb = xw.Book('data/小蜜蜂超市銷售報表.xlsx') # 打開文件ws = wb.sheets['Sheet1']  # 選擇表格

4. 獲取表格行列

last_column = ws.range(1, 1).end('right').get_address(0, 0)[0] # 獲取最後一列last_row = ws.range(1, 1).end('down').row # 獲取最後一行a_range = f'A1:{last_column}{last_row}'  # 生成表格的數據範圍

這裡 a_range 輸出的範圍是 A1:D6，即數據表的整個內容。

5. 設置 a_range，也就是 A1:D6 的表格邊框

# 設置邊框ws.range(a_range).api.Borders(8).LineStyle = 1  # 上邊框ws.range(a_range).api.Borders(9).LineStyle = 1  # 下邊框ws.range(a_range).api.Borders(7).LineStyle = 1  # 左邊框ws.range(a_range).api.Borders(10).LineStyle = 1  # 右邊框ws.range(a_range).api.Borders(12).LineStyle = 1  # 內橫邊框ws.range(a_range).api.Borders(11).LineStyle = 1  # 內縱邊框

說明：

LineStyle = 1，這裡的 1 指的是邊框為實線。

到這裡還不算完，還要關閉 Excel，保存 xlsx 文件。

# 保存並關閉 Excelwb.save('data/小蜜蜂超市銷售報表.xlsx')wb.close()app.quit()
