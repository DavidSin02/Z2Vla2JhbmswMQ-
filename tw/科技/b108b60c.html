Python函數的形參與實參

1.形參與實參

形參：顧名思義，就是形式上的參數，沒有實際的意思，通過別人賦值後才有意義，相當於變量。在定義函數時，函數的參數列表中的變量就是形參。

實參：就是實際意義上的參數，是一個實際存在的參數，可以是字符串、數字等。在調用函數時，給函數傳遞的實際值就是實參。

3.值傳遞與引用傳遞

在Python中函數參數是引用傳遞(而不是值傳遞)。需要注意的是，對於不可變類型，比如說字符串，元組，數值和函數等，因為變量不能被修改，所以運算時不會影響到變量本身。而對於可變類型，比如說列表、字典、集合、自定義的對象等，函數體中的運算有可能會更改傳入的參數變量。

1.對“可變對象”進行“操作”，直接作用於可變對象本身；

2.對“不可變對象”進行修改操作，會產生一個新的對象空間，並用新的值填充這塊空間。

  • 不可變對象

def add(x): # 這裡的x是形參

print(f'我是形參時的x,地址是：{id(x)}')

x = x + 1 #這裡的x與形參裡的x不量同一個，是重新定義的(我們通過id來驗證))

print(f'我是內部的x,地址是：{id(x)}')

print(x)

# 調用

x = 10

print(f'函數add的地址：{id(add)}') # add在Python也是一個對象，有對應的地址

print(f'我是調用add方法前定義的x,地址是：{id(x)}')

# 調用函數

add(x)

print(f'我是調用完add方法後的x,地址是：{id(x)}')

不可變類型調用過程, 可以看出, x的指向10一直沒有變

  • 可變對象

對於可變類型，比如說列表，字典等，函數體中的運算有可能會更改傳入的參數變量，因為傳入的是地址，即對象的引用。在函數體中不創建新的對象的拷貝，而是直接修改所傳遞的對象。

mylist = [50,60]

def myappend(m):

print(f'list: {id(m)}') # mylist和m是指向同一個地址

m.append(100) # 由於m是可變的，不創建對象拷貝，直接修改

myappend(mylist)

print(f'mylist: {id(mylist)}')

print(mylist)
