PhotoShop常見的52個疑難問題及解答，值得收藏

2020-11-14
| 科技

（1）問：怎樣在圓角的圖形中設置圖形的陰影；

答：PS6裡有一個圖層效果，不管你的圖形是什麼形狀的，只要單獨做在一個層裡，就可以實行這個效果，裡面有所需要的圖形陰影設置。

（2）問： 用鋼筆工具鉤的路徑再用放射工具總是漸變放射，由粗到細，怎樣成一條直線；

答：1.筆工具鉤的路徑，再選取圖章及噴槍，然後在路徑面版中直接拖動工作路徑到描邊按鈕上，就能出現一條直線效果。2.筆工具的動力設置關閉（在鋼筆工具狀態行最後一項）或 reset tool。

（3）問： 用鋼筆工具鉤下一圖像後，怎樣摳到新建的文件中去；

答：用鋼筆工具鉤出圖像以後，把路徑變成選區，然後新建一個文件，使用複製、粘貼或者直接拖動選區到新鍵文件上都可。

（4）問： 用直線工具畫一個直線後，怎樣設置直線由淡到濃的漸變

答：用直線工具畫出直線後，1.把它變成選區，填充漸變色，選前景色到漸變透明。2.在直線上添加蒙版，用羽化噴槍把尾部噴淡，也可達到由淡到濃的漸變。

（5）問：做一個等腰梯形怎樣精確控制它的大小；

答：1.由變換裡的透視功能就能精確的控制它，快捷鍵 CTRL+T。2.TRL+T，再按住 CTRL+SHFIT+ALT ，用鼠標控制頂點，就會以等腰梯形控制。

（6）問： 怎樣把做好的方框拉成別的形狀；

答：把做好的方型變成選區，並把選區變成工作路徑，然後添加節點，就能變成其他形狀。

（7）問： 怎樣使一個圖片和另一個圖片很好地融合在一起(包括看不出圖像的邊緣)；

答：1.選中圖片，實行羽化，然後反選，再按 DELETE，這樣就可以把圖片邊緣羽化為了達到好的融合效果，你可以把羽化的像素設定的大點，同時還可以多按幾次 DELETE，那樣融合的效果更好。2.在圖片上添加蒙版，然後選羽化的噴槍對圖片進行羽化，同樣能達到融合的效果。最後別忘了把層的透明度降低，效果會更好。

（8）問： 在 PS 中寫入文字，怎樣選取文字的一部分；

答：把文字層轉換成圖層，然後在層面版上按住 CTRL，用鼠標點擊轉換成圖層的文字層就能選中全部文字，然後按住 ALT 鍵，就會出現+_的符號，然後選中不需要的文字，那麼留下的就是需要的文字。

（9）問： 怎樣使文字邊緣填充顏色或漸變色；

答：文字邊緣填充顏色，可描邊功能。如要給文字邊緣使用漸變色，先新建一個透明層，然後選中文字，然後在圖層中選中透明層，實行描邊功能，然後把描邊的層變成選區，填上漸變色即可。

（10）問：寫入一段文字，可否在同一圖層選取其中的一個文字填充漸變色；

答：把文字層轉換成圖層，然後在層面版上按住 CTRL，用鼠標點擊轉換成圖層的文字層就能選中全部文字，然後按住 ALT 鍵，就會出現+_的符號，然後選中不需要的文字，那麼留下的就是需要的文字，在用漸變色填充就可以了。

（11）問：我是新手，覺得鋼筆工具很不好控制，有什麼好的方法或者技巧呢？

答：鋼筆工具的關鍵是在節點，你的節點選擇好的位置，那麼你所畫出的弧度就會很漂亮，記得要多用，多練。

（12）問：用 PHOTOSHOP 輸出 AI 格式，在 MAX 裡輸不進?

答：PHOTOSHOP 輸出為 AI 格式的只能輸出路徑，你可以進入 MAX 裡面在添顏色。

（13）問：請問現在 PHOTSHOP 最高版本是多少了？（中文的）

（14）問：如何在 PHOTOSHOP6.0中顯示中文字體。

（15）問：各位大蝦，我敢說我不是個十足的菜鳥，但學了大半年的 ps，至今對通道還是不能靈活的應用，請問你們從前對通道的認識也是這樣嗎，現在經常用通道嗎，你們認為通道是 ps 最難的嗎？

答：通道應該分幾部分來說：

1.通道是保存信息的地方，通道是按照8位來存儲信息的，而 PS 的圖層是24的，這樣按照存儲格式 PSD 是很佔硬盤空間的，而用通道保存信息的話，那就可以少佔用硬盤空間。同時許多標準圖像式如：TIF、TGA 等均可以包含有通道信息，這樣就更大方便了不同應用程序來進行信息共享了。

2.通道層實際是圖像層選擇區域的映射，完全為黑色的區域表明完全沒有選擇，完全為白色的區域表明完全選擇，灰度的區域由灰度的深淺來決定選擇程度。現在網上流行的透明 GIF 圖像，實際上就是包含了一個通道，用來告訴應用程序（如 IE 瀏覽器）那些部分需要透明，那些部分不需要透明。3.通道還可以進行圖像層的計算合成，從而產生很多特效。另外真正的圖像處理專家通過通道來精確的對圖像進行調整。通道不管是在 R、G、B、還是在 C、M、Y、K 圖像中所顯示的都是黑、白、灰三種。所以可以單獨把通道分離出來對每張圖像進行精確的色彩調整。

（16）問：用 photoshop 設計出一個效果圖打印到 A4紙上,但是我不知道在 photoshop 新建這個文件的尺寸(也就是定義該文檔的長寬各是多少比較合適),打印時分辨率又該設為多少?

（30）問：請問在 PS 中參數設置裡的插值方式是什麼意思，主要作用體現在哪些地方？

它有鄰近，二次線性，二次立方三種選擇！書上說是對圖像做插值運算時用的，我想插值運算是怎麼回事？

答：插值是在離散數據之間補充一些數據，使這組離散數據能夠符合某個連續函數。插值是計算數學中最基本和最常用的手段，是函數*近理論中的重要方法。利用它可通過函數在有限個點處的取值狀況，估算該函數在別處的值，即通過有限的數據，以得出完整的數學描述。早在公元六世紀，中國的劉焯已將等距二次插值法用於天文計算。十七世紀，牛頓和格雷果黎建立了等距結點上的一般插值公式。十八世紀，拉格朗日給出了更一般的非等距結點上的插值公式。在近代，插值法是觀測數據處理和函數製表所常用的工具，又是導出其他許多數值方法（例如數值積分、非線性方程求根、微分方程數值解等）的依據。鄰近，二次線性，二次立方三種選擇主要是指插值的計算方法，與上面的意思相近。

（31）問：請問什麼是 overlay 方式？

答：這種模式以一種非藝術邏輯的方式把放置或應用到一個層上的顏色同背景色進行混合，然而，卻能得到有趣的效果。背景圖像中的純黑色或純白色區域無法在 Overlay 模式下顯示層上的 Overlay 著色或圖像區域。背景區域上落在黑色和白色之間的亮度值同0ver1ay材料的顏色混合在一起，產生最終的合成顏色。為了使背景圖像看上去好像是同設計或文本一起拍攝的，Overlay 可用來在背景圖像上畫上一個設計或文本。

（35）問：本人正在學 PS，看到書中有 dpi，ppi，lpi 的詞，dpi 是每英寸點數，指設備所具有的分辨率；ppi 是每英寸像素數，指圖像所具有的分辨率；但 lpi 具體指什麼我不是很清楚，書上寫是加網線數，不知它與什麼有關？

答：在商業印刷領域，分辨率以每英寸上等距離排列多少條網線即 LPI（Lines Per Inch）表示。在傳統商業印刷製版過程中，製版時要在原始圖像前加一個網屏，這一網屏由呈方格狀的透明與不透明部分相等的網線構成。這些網線也就是光柵，其作用是切割光線解剖圖像。由於光線具有衍射的物理特性，因此光線通過網線後，形成了反映原始圖像影像變化的大小不同的點，這些點就是半色調點。一個半色調點最大不會超過一個網格的面積，網線越多，表現圖像的層次越多，圖像質量也就越好。因此商業印刷行業中採用了 LPI 表示分辨率。

（42）問：用 photoshop 打印圖片時，為了更好的效果，是否要把圖片由 rgb 格式轉為 cmyk 格式？圖片的分辨率應設為多少？72的分辨率和300的分辨率分別適用在什麼情況下？謝謝！

答：用 PHOTOSHOP 打印圖片時，最好把圖片的模式改成 R G B，因為打印機的油墨是按照 R GB 的顏色來調配的。一般圖片打印分辨率只要72DPI 就足夠了。72分辨率的圖片一般是用在網上的，因為72分辨率的圖片是顯示圖片，它只要讓顯示器能顯示圖片顏色就可以了，而300分辨率的圖片使用在印刷的效果上。

（45）問：請問 PHOTOSHOP 怎樣批量把圖片改成一定尺寸？

答：用 ACTION 就可以了，很簡單的。

1、先打開一幅你要變尺寸的圖片，同時打開 ACTION 面板

2、按下 ACTION 的錄製鍵，然後對圖片進行操作

3、完成圖片操作後，按 CTRL+W，回車，保存修改結果，同時停止 ACTION 錄製

4、在 ACTION 面板裡對剛才錄製的每一步操作過程進行復制，比如你剛才操作步驟是：IMAGE SIZE、AUTO LEVEL、SAVE、COLSE，那麼每一步複製10個，按循序排好。

5、同時打開10個圖片，按下 ACTION 中剛才操作的播放鍵，一次10張圖片就 OK 了。

PS 應用常見問題

Q1.通道(channels)和圖層(layers)有什麼區別?

簡單地說,“圖層”猶如一張張相疊的透明紙，你可以單獨在每個圖層中作畫，可反覆更改，移動而不影響其它圖層，相疊一起形成一個完整的畫面。

原來帶的“通道”是控制每個原色在畫面中的分量（如：RGB 模式，就是分別控制紅、綠、藍三個顏色的份量，三色相疊形成了真彩圖像）而自己加上的“通道”就相當於把圈選線保存起來，(又稱遮擋)並可用各種繪畫工具對面積和深淺進行修改，在需要的時候再調出來用。（通道的用法，建議你到“點點生輝”“Photoshop Tips”內做一做第22條 Tips 按鈕的練習就會清楚了）

Q2.在製作網頁用圖形時什麼時候用 Gif,什麼時候用 Jpg?

通常講,顏色層次比較豐富細膩的圖片就用 Jpg,如寫實的照片.在存儲 Jpg 時會有壓縮的強度選擇,當然壓的越少文件越大,但失真也較少,反之…而顏色比較少,以平塗形式描繪的圖形通常就用 Gif,如一些文字及幾何圖形.Gif 是以顏色的數量來決定文件的大小的,在你把 RGB Color 轉成 Indexed Color 時就可以進行選擇適中的顏色數量.

Q3.為什麼我下載了 Actions 文件後,運行起來老報錯,達不到預期效果?

每個 Actions 都需要按各自的開始前的準備做簡單的準備才能開始運行,否則就不能達到預期效果.

Q4.能讓我知道 Actions 製作的效果的詳細製作步驟嗎?

如果你想知道 actions 每一步是怎樣做的,只要 在 Action 控制板上,actions 名字前的方框點上,你就可以一步一步地跟著幹了 ^-^

Q5.路徑(Path) 和通道(Channels)有什麼區別?

Path 和通道有各自的特點:

Path 可以勾畫出圓滑的曲線,並且在勾畫後可以方便地修改, 但轉成圈選範圍後,只能是一個由頭到尾虛實都沒有變化的輪廓. Channel 其實就是儲存圈選範圍的地方,並可以用各種的繪畫工具對圈選範圍進行修改, 包括各種灰階層次的微妙變化,也可以是一個圖形,這是 Path 所無法做到的. 可以把兩樣東西結合起來運用就可以發揮各自的優勢,如用 Path 勾畫出圓滑的輪廓,然後 轉成圈選範圍存入 Channel 來修飾就可以做出既流暢又有變化的圈選輪廓了.

Q6.如何在 PHOTOSHOP 裡做透明的 GIF?

首先請確認圖片模式為索引顏色 ( Image > Mode > Indexed Color…); 然後到 File > Export > Gif89a Export… 用吸管點取需要透明的顏色,最後按 OK 存盤就可以了.

Q7.如何安裝 KPT; Eye candy… 等 Photoshop 插件?

安裝 KPT;Eye candy… 等 Photoshop 的插件並非安裝在一個獨立的目錄內,而是在要求輸入安裝目錄的時候把目錄指向 Photoshop 的 Plug-Ins 目錄,(一般在:c:\Program Files\ Adobe\ Photoshop 5.5\ Plug-Ins)安裝完後打開 Photoshop 就可以在 Filter 菜單下找到新裝的插件了.

如果是一些沒有 Setup 的小插件就可以把 *.8bf 等文件拷到 Plug-Ins 目錄下便可以直接使用了.

Q8.掃描印刷品的時候如何去網(去除龜紋)

最直接簡單的方法是使用 Filter> Noise> Despeckle;但有時效果不太理想就可以按每個 Channel 的實際情況分開 Channel 來退,如果網紋實在明顯的話可以在結合 Filter> Noise> Median…

Q9.矢量圖形和點陣圖形有什麼區別,各有什麼特點?

點陣圖形是由一個個像素來組成的圖形,局部放大圖形所看到馬賽克就是像素,點陣圖形能紀錄下圖像無規律的微妙層次變化,通常是一些紀實的照片.

而矢量圖形的圖像是以曲線及節點來標示圖形,可以無限放大而不影響其精度,如 Coreldraw, Freehand 及 Flash 所畫出來的圖形均屬此類,照片要變成矢量圖形就必須經過處理. (矢量圖形並不適於紀錄真實,層次變化複雜的圖像的)

PS 數碼合成後期常見問題

針對 PHOTOSHOP 出現的一些常見問題，總結出以下解決方法：

（1）問：我裝了一個 KPT ，在 photoshop 上卻連 KPT 的影子都看不到？

答：把它裝在 PHOTOSHOP 的 PLUG-INS 的例徑下才可以用。

（2）問：建立調整層有是很方便，但怎麼能只讓調整層作用於同一圖層呢？

答：分別在做幾個層的效果，然後在把層合併就可以了。

（3）問：如何在 potoshop 將圖片淡化？

答：1. 改變圖層的透明度，100％為不透明。2. 減少對比度,增加亮度。3. 用層蒙板。4. 如果要將圖片的一部分淡化可用羽化效果。

（4）問：我在 photosho 裡的 新建了一個通道，為什麼我按 CTRL+點擊所建的通道不好使那？出現 任何象素都不大於50%，選區邊將不可見。

答：按 CTRL+點通道的時候，通道上必須有物體，空的通道是不行的。

（5）問：PHOTOSHOP 中如何將一幅圖分割為若干塊？用什麼工具？步驟如何？

答：1.任何一個 PS 版本里面都有裁切這個功能，你使用它就可以完成圖片的分割。 2.PS6.0 裡面有一個專用的裁切工具，你可以使用它來完成分割，並輸出為 HTML 或 JPG 或 GIF 等格式。

（6）問：PHOTOSHOP 下什麼方法可以很快畫出虛線?(包括曲線)

答：在筆刷的屬性裡，把筆刷的圓形壓扁，然後將筆刷的間隔距離拉大，這樣就可以畫出虛線。可以先做一個你要用的形狀的路徑，然後調整筆刷的 spacing 的值，描邊路徑就可以產生曲線。

（7）問：請問如何使用 adobe gamma

答：GAMMA 值是針對印刷來說的一種值。如果你的圖像與印刷有關的話，那麼就需要調整 GAMMA 值了。如果你的圖像只是用來打印或在網上發佈，使用默認值就可以了。

（8）問：ps 中，可不可以象 pagemake 裡一樣，使文字在一個不規則的框裡，並根據框的改動自動改動

答：不可以，但是 PS 有一個外掛濾鏡，可以使文字繞著路徑走，你改變路徑，文字也會跟著改變。

問題集二

（9）問：我是新手，覺得鋼筆工具很不好控制，有什麼好的方法或者技巧呢？

答：鋼筆工具的關鍵是在節點，你的節點選擇好的位置，那麼你所畫出的弧度就會很漂亮，記得要多用，多練。

（10）問：用 PHOTOSHOP 輸出 AI 格式，在 MAX 裡輸不進?

答：PHOTOSHOP 輸出為 AI 格式的只能輸出路徑，你可以進入 MAX 裡面在添顏色。

（11）問：如何在 PHOTOSHOP6.0中顯示中文字體。

答：在編輯裡點預置/常規，把字體用英文顯示前的勾去掉，在中文顯示前打勾。

（12）問：各位大蝦，我敢說我不是個十足的菜鳥，但學了大半年的 ps，至今對通道還是不能靈活的應用，請問你們從前對通道的認識也是這樣嗎，現在經常用通道嗎，你們認為通道是 ps 最難的嗎？

答：通道應該分幾部分來說： 1.通道是保存信息的地方，通道是按照8位來存儲信息的，而 PS 的圖層是24的，這樣按照存儲格式 PSD 是很佔硬盤空間的，而用通道保存信息的話，那就可以少佔用硬盤空間。同時許多標準圖像式如：TIF、TGA 等均可以包含有通道信息，這樣就更大方便了不同應用程序來進行信息共享了。 2.通道層實際是圖像層選擇區域的映射，完全為黑色的區域表明完全沒有選擇，完全為白色的區域表明完全選擇，灰度的區域由灰度的深淺來決定選擇程度。現在網上流行的透明 GIF 圖像，實際上就是包含了一個通道。
