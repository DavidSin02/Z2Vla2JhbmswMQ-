Go之配置文件

2020-10-29
| 科技

前言

我們都知道，不管用什麼語言做什麼項目，一般情況下配置文件是必不可少的，那麼對於go語言來說該怎麼處理配置文件呢？其實go語言的配置文件格式有很多種（比如：yaml、json、xml、ini、toml等等），但是在此我只分享yaml格式的配置文件，因為分享多了反而讓人覺得雲裡霧裡，不知該怎麼抉擇，所以其它的格式有需要的朋友可以自行去查找相關資料，這裡只當是拋磚引玉。


操作步聚

1、下載yaml包

go get gopkg.in/yaml.v2


2、新建yaml文件

比如我的是config.yaml，內容如下：

host: localhost:3306

user: ggb

pwd: 123456

dbname: test

注1：第一個冒號後面必須有空格，冒號前代表鍵K，冒號後代表值V

注2：這個配置文件該放到哪裡比較合適呢，可以將其放到工程目錄下即可，如下圖所示


3、實戰

首先，新建config目錄，然後新建go文件config.go，內容如下：

package configimport ( "fmt" "gopkg.in/yaml.v2" "io/ioutil" "path/filepath")//profile variablestype Conf struct { Host string `yaml:"host"` User string `yaml:"user"` Pwd string `yaml:"pwd"` Dbname string `yaml:"dbname"`}func (c *Conf) GetConf() *Conf { path, err := filepath.Abs("config.yaml") if err != nil { fmt.Printf("err: %v\n", err) return nil } fmt.Println("path: ", path) yamlFile, err1 := ioutil.ReadFile(path) if err1 != nil { fmt.Println(err1.Error()) return nil } err1 = yaml.Unmarshal(yamlFile, c) if err1 != nil { fmt.Println(err1.Error()) return nil } return c}

其次，在與config同目錄處新建go文件main.go，內容如下：

package mainimport ( . "config_test/config" "fmt")func main() { var c Conf conf := c.GetConf() fmt.Println(conf.Host)}

注：如果遇到自定義包無法加載的問題，記得點擊菜單File -> Settings，然後配置GOPATH，比如我的如下圖所示

最後，執行main，就可以看到取到的值了。
