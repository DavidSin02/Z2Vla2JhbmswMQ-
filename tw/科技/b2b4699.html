De、DN、D、d、Φ的含義

一、DN（Nominal Diameter）意思是管道的公稱直徑，是外徑與內徑的平均值。DN的值=De的值-0.5*管壁厚度。注意：這既不是外徑也不是內徑。


水、煤氣輸送鋼管（鍍鋅鋼管或非鍍鋅鋼管）、鑄鐵管、鋼塑複合管和聚氯乙烯（PVC）管等管材，應標註公稱直徑“DN”（如DN15、DN50）。


二、De（External Diameter）意思是管道外徑，PPR、PE管、聚丙烯管外徑，一般採用De標註的，均需要標註成外徑*壁厚的形式，例De25×3。


三、D一般指管道內徑。


四、d一般指混凝土管內直徑 。鋼筋混凝土（或混凝土）管、陶土管、耐酸陶瓷管、缸瓦管等管材，管徑宜以內徑d表示（如d230、d380等）。


五、Φ表示普通圓的直徑；也可表示管材的外徑，但此時應在其後乘以壁厚。如：Φ25×3，表示外徑25mm，壁厚為3mm的管材。對無縫鋼管或有色金屬管道，應標註“外徑×壁厚”。例如Φ108×4，Φ可省略。中國、ISO和日本部分鋼管標準採用壁厚尺寸表示鋼管壁厚系列。對這類鋼管規格的表示方法為管外徑×壁厚。例如Φ60.5×3.8。

管徑的表達方式

一、水、煤氣輸送鋼管（鍍鋅或非鍍鋅）、鑄鐵管和塑料管等管材，應標註公稱直徑“DN”（如DN15、DN50）。


二、無縫鋼管、焊接鋼管（直縫或螺旋縫）、銅管、不鏽鋼管等管材，管徑宜以外徑×壁厚表示（如De108×4、De159×4.5等）；對無縫鋼管或有色金屬管道，應標註“外徑×壁厚”。例如Φ108×4，ф可省略。


三、鋼筋混凝土（或混凝土）管、陶土管、耐酸陶瓷管、缸瓦管等管材，管徑宜以內徑d表示（如d230、d380等）。


四、塑料管材，管徑宜按產品標準的方法表示。


五、當設計均用公稱直徑DN表示管徑時，應有公稱直徑DN與相應產品規格對照表。


其它說明

公稱直徑(nominal diameter)，又稱平均外徑(mean outside diameter)。為了使管子、管件連接尺寸統一，採用DN表示其公稱直徑（也稱公稱口徑、公稱通徑)。


因為單位有公制(mm)及英制(inch)的區分，所以有下列的稱呼方法。

一. 以公制(mm)為基準，稱 DN (metric unit)


二. 以英制(inch)為基準，稱NB(inch unit)


三. DN (nominal diameter) NB (nominal bore) OD (outside diameter)


四. 【例】 鍍鋅鋼管DN50，sch 20 鍍鋅鋼管NB2”，sch 20


注：管子表號(Sch．)並不是壁厚，是壁厚系列。實際的壁厚，同一管徑，在不同的管子表號中其厚度各異。


1938年美國國家怔準協會ANSIB36.10(焊接和無縫鋼管)標準所規定：管子表號(Sch．)是設計壓力與設計溫度下材料的許用應力的比值乘以1000，並經圓整後的數值。


ISO公制標示 ( ISO metric dimension size ) 以Da表示 PVC管，ABS管之管外徑 。以De表示 PP管，PE管之管外徑 。


例如：外徑63mm的塑料管之公制標示 ( mm dimension size ) 為：Da63 ，是針對PVC管，ABS管而言的。


新型材料和塑料管用de，注意是小寫的哦。


Dn主要用在鑄鐵管、鍍鋅鋼管和鋼塑複合管上。另外Dn100等同於de110，Dn150等同於de160的管徑。


工字鋼和H型鋼的區別在哪?

在工作中，經常會有人問到，工字鋼和H型鋼的區別，正因為不瞭解這一點，所以，很難區分工字鋼和H型鋼，甚至，還有些人認為工字鋼和H型鋼就是一回事，其實這樣的想法是錯誤的，這兩種不是一個概念，所以，今天我們就來看看，工字鋼和H型鋼的區別在哪了，下面來看看詳細介紹。

工字鋼和H型鋼的區別在哪？

什麼是工字鋼：

1、工字鋼顧名思義，是一種“工”字形截面型鋼，上下翼緣內表面有傾斜度，一般為1：6，使得翼緣外薄而內厚，因此造成工字鋼在兩個主平面的截面特性相差巨大，在應用中難以發揮鋼材的強度特性，正如上圖所示。雖然工字鋼市場上也出現了加厚的工字鋼，但工字鋼的結構已經決定了其抗扭性能短板。

2、工字型鋼不論是普通型還是輕型的，由於截面尺寸均相對較高、較窄，故對截面兩個主袖的慣性矩相差較大，因此，一般僅能直接用於在其腹板平面內受彎的構件或將其組成格構式受力構件。對軸心受壓構件或在垂直於腹板平面還有彎曲的構件均不宜採用，這就使其在應用範圍上有著很大的侷限。

什麼是H型鋼：

1、H型鋼屬於高效經濟裁面型材(其它還有冷彎薄壁型鋼、壓型鋼板等)，由於截面形狀合理，它們能使鋼材更高地發揮效能，提高承裁能力。不同於普通工字型的是H型鋼的翼繡進行了加寬，且內、外表面通常是平行的，這樣可便於用高強度螺桂和其他構件連接。其尺寸構成合理系列，型號齊全，便於設計選用。H型鋼的軋製不同於普通工字鋼僅用一套水平軋輥，由於其翼絛較寬且無斜度(或斜度很小)，故須增設一組立式軋輥同時進行輥軋。因此，其軋製工藝和設備都比普通軋機複雜。

2、錫H型鋼的翼緣都是等厚度的，有軋製截面，也有由3塊板焊接組成的組合截面。工字鋼都是軋製截面，由於生產工藝差，翼緣內邊有1：10坡度。H型鋼的軋製不同於普通工字鋼僅用一套水平軋輥，由於其翼緣較寬且無斜度(或斜度很小)，故須增設一組立式軋輥同時進行輥軋，因此，其軋製工藝和設備都比普通軋機複雜。國內可生產的大軋製h型鋼高度為800mm，超過了只能是焊接組合截面。我國熱軋H型鋼國標(GB/T11263-1998)將H型鋼分為窄翼緣、寬翼緣和鋼樁三類，其代號分別為hz、hk和hu。窄翼緣H型鋼適用於樑或壓彎構件，而寬翼緣h型鋼和h型鋼樁則適用於軸心受壓構件或壓彎構件。工字鋼與H型鋼相比，等重量前提下，w、 ix、 iy都不如h型鋼。

工字鋼和H型鋼的區別：

H型鋼是當今鋼結構建築中應用廣泛的型材，它與工字鋼相比有很多區別。

1、H型鋼翼緣內表面沒有傾斜度，上下表面平行。

2、H型鋼的兩條外邊內側沒有斜度，是平直的。

3、H型鋼的截面特性要明顯優於傳統的工字鋼、槽鋼和角鋼。

4、H型鋼，是一種截面面積分配更加優化、強重比更加合理的經濟斷面高效型材，因其斷面與英文字母“H”相同而得名。這使得H型鋼的焊接拼接比工字鋼操作簡單，單位重量的力學性能更好，可以節省大量的材料和施工時間。

通過上文的介紹，相信大家都知道工字鋼和H型鋼有很大區別的，所以，在使用中，一定要正確區分工字鋼和H型鋼，這樣一來，才能更好的使用。
