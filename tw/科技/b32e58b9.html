拿了紅點獎的巖板長什麼樣？

2020-10-29
| 科技

近日，德國紅點獎公佈了2020年獲獎名單。陶城君統計發現，今年共有巖板、瓷磚、衛浴、馬桶和水龍頭等7款產品獲獎。

其中，意大利Florim和土耳其Kale的巖板產品獲得品牌與傳達大獎；中國Tsang Ching Pun設計公司的Clean Seat公共馬桶獲得產品設計概念大獎；Albarello公司Ignorance Is Bliss瓷磚、Duravit Viu / XViu衛浴、Aquno Select M81廚房水龍頭、Axia系列水龍頭等產品獲得產品設計大獎。

巖板設計獲得品牌與傳達大獎‍

紅點獎共有4大獎項，包括產品設計大獎、品牌與傳達設計大獎、設計概念大獎和當代好設計獎。其中，2020年品牌與傳達設計大獎共有6992件參賽作品，有854個品牌和傳達作品獲得了紅點獎，佔參賽作品的13%。

陶城君檢索發現，除了衛浴產品外，意大利Florim和土耳其Kale2家巖板企業的產品獲得該獎項。

Florim集團Policroma巖板

意大利Florim集團的新品Policroma系列瓷磚獲此殊榮。由設計師Cristina Celestino創作的Policroma系列巖板將古典建築經典與極具現代感的表達方式相結合，重新定義了瓷磚產品。Policroma巖板呈現了大理石和Marmorino風格的瓷磚表面。

仿大理石效果是Policroma系列內涵的關鍵，Policroma系列復興了一種越來越稀少的材料。Cristina Celestino表示，它並沒有提出“代替天然大理石”的口號，而是通過模仿，將大理石和Marmorino灰泥相結合，以一種對比強烈、具有現代感的方式再現了具有百年曆史的大理石。

設計所選用的大理石類型各具特色。比如，Verde Alpi大理石是經常出現在米蘭大教堂入口大廳的一種瓷磚；Breccia Capraia珍稀大理石則存在於意大利托斯卡納區的極少數地點；Cipollino大理石是特殊的綠紅色品種，帶有螺旋狀紋理；Rosa Valtoce大理石則來自阿爾卑斯山脈附近的皮埃蒙特的一個採石場。

Policroma系列的眾多元素突出了瓷磚加工工藝的重要性——通過各種模塊的自由組合，打造出各種半圓造型，或者通過小尺寸設計的大量複製，使最初只是用作點綴的細節化身為整個設計的“主角”。它以一種質樸但又並非極簡主義的方式來設計牆地面，簡單而不失智慧，低調的空間和前衛的宣言，充分體現了“米蘭style”。

土耳其Kale“無限”展館

意大利建築設計公司Paolo Cesaretti Arch為土耳其Kale陶瓷集團設計的臨時展館Bespoke Romance獲得了2020年紅點獎“品牌與傳達設計”獎項。

“無限”概念是該展館的設計靈感，兩個融匯的圓代表Kale集團文化的雙重魅力。展館的圓形外牆設計大膽、緊湊，同時採用鏤空設計，匯聚了國際化企業Kale集團的眾多品牌和特色。展館外牆由數百片豎立的Kalesinterflex超薄巖板建成。展廳內部的展示區也使用了這種巖板，用於室內家居裝飾，包括廚房、客廳和浴室，向參觀者充分展示了Kale集團產品的特性和功能。

燈光的巧妙運用凸顯了展廳的獨特形狀，讓整個展廳帶有一種抽象的美感。環形小路上的是家裝和商業產品展示區。由經過切割的瓷磚製成的裝飾面板豎立在天花板和地板之間。此外，展廳內寬敞的休息區也非常引人注目，其將優雅時尚的酒吧和可於集體工作的會議區空間融合在一起。

瓷磚、衛浴、馬桶、水龍頭獲得設計大獎‍

2020年紅點設計大獎共有超過6,500件參賽作品，僅76件作品獲得了紅點最佳設計獎，獲獎率僅為1.2%，小米、聯想等多箇中國企業品牌也在榜單中贏得一席。

陶瓷衛浴行業中，Albarello的Ignorance Is Bliss瓷磚、Duravit Viu / XViu衛浴、Aquno Select M81廚房水龍頭、Axia系列水龍頭等產品獲得該獎項。

Albarello的Ignorance Is Bliss瓷磚

金屬是造成全球汙染的原因之一，而且由於其不可再生性，資源是非常有限的。在這種背景下，“Ignorance Is Bliss”這款瓷磚百分百使用工業金屬廢料作為陶瓷顏料為陶瓷上色，促進了工業金屬廢物進行合理地循環利用。

Duravit Viu / XViu衛浴系列

Viu陶瓷系列、XViu傢俱系列以及配套的浴缸作為一個完整的浴室系列，整體設計散發著前瞻性的魅力。該系列作品被定位為提升“後工業化”的優雅，設計中體現出柔軟有機的形式與精確的幾何碰撞出的火花，在細節上也高度完善考究。

此外，該浴室系列由高級陶瓷、玻璃、漆面金屬和優質木材製成，各種不同原材料表面的組合和鮮明反差是該系列的一大特色。

洗漱區域是設計時的一大重點，該區域採用了杜拉維特(Duravit)的C粘合專利技術，為用戶提供友好舒適的使用體驗。

產品外形是嚴格的矩形，而內部則具有柔和的有機曲線，洗手盆寬敞而擱板收納區域寬闊。金屬控臺，梳妝檯，鏡子，鏡子櫃和獨立式浴缸等系列產品，通過C型粘合有機融合，並運用香檳色或黑色啞光的金屬材質被塑造成醒目的V型，外觀獨特。

通過不同材料的組合，打造出同色調和反差色的不同配色方案。高檔實木，清漆和裝飾飾面等可搭配出從“銀色”到“深色”的多種樣式。

Aquno Select M81廚房水龍頭

Aquno Select M81系列產品的設計中，將廚房冷熱水混合龍頭作為優化工作流程的重點。旨在用於大尺寸的水槽，該產品由拱形出水口和賞心悅目的幾何圖形組成。

這個水龍頭引人注目的外觀與廚房氛圍融為一體。其多功能性在於包含一個可拉出式的噴頭，以及一個精細的平板，在其下方有一個突出的邊緣導流，讓水以一種創新的方式流出。

使用者可以選擇花灑或噴射模式出水，也可以用底部的混合龍頭水平噴水，這樣水流更低也更接近臺盆。

根據不同的烹飪步驟，這種由單個超細微噴霧組成的水流可以更溫和地衝洗脆弱的食物，同時在清洗大物體時也不會濺水，另外洗手時也更節水。結合特製的多功能過濾器，這款水龍頭讓備置食物的過程變得簡單而優雅——以一種全新的方式來感受水流。

Axia系列水龍頭

Axia系列以其全新的現代極簡風格令人印象深刻。它的設計表達出設計師不斷探索突破輕薄和純粹的邊界。

Axia的外觀令人難以忘懷，一件式鑄造的6.5毫米高的超薄出水口，標榜著這一系列不容小覷。這種設計美感創造出富有表現力且功能強大的衛浴配件，完善起室內的整體現代設計。

該系列產品的一大亮點是洗臉池或浴缸所使用的壁面配件，該配件的非水平設計明顯打破了傳統佈局。牆式龍頭的非槓桿手柄位於噴嘴下方的斜對角位置。大轉盤和長長的出水管以精緻微妙的平衡感打造出輪廓精美鮮明的產品，給人以輕盈的感覺。

Axia龍頭具有防滑手柄和精刻凹槽，操作簡單直觀。先進的漸進式陶瓷濾筒可精確地輸送水。另外，牆式龍頭還具有精心設置的外環，只需一個動作就能完全控制水流，從關閉到水溫最熱。

中國馬桶獲得概念大獎

2020年紅點設計概念大獎獲獎共有參賽作品達4170件，僅42件作品獲獎，比例僅佔所有參賽作品的1%。其中，中國Tsang Ching Pun設計公司的Clean Seat公共馬桶獲獎。

Clean Seat的馬桶蓋會在第一個傳感器(位於馬桶蓋的前部)感應到有人靠近時自動打開。第二個傳感器感應到使用人離開後，馬桶蓋隨即關閉並自動上鎖。當馬桶蓋上鎖時，馬桶將開啟自潔功能。經過這一系列的清洗消毒，在下一位使用者使用時，馬桶座將再次達到了乾淨衛生的狀態。
