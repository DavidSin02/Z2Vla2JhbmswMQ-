“千年光盤”可刻錄！華碩SBW-06D5H-U-Ultra超便攜 刻錄藍光光驅

2020-11-14
| 科技

雖然光驅離我們越來越遠，但對於一些特殊需求，比如藍光愛好者，光驅依舊有用武之地，因此主打便攜的光驅依舊活躍在市場中。

近日，ASUS（華碩）發佈了一款超便攜藍光光驅——SBW-06D5H-U-Ultra，非常便於攜帶，支持USB 3.1 Gen 1高速連接，還可使用M-DISC光盤，也就是此前提到的可穩定保存1000年數據的光盤，對數據保存有高需求的用戶可關注。

這款光驅造型圓潤，設計很像便攜固態硬盤，在尾部還有一個便攜孔，通電運行時會有藍色指示燈亮起。尺寸為W149.4×D 144×H14.7mm，重約240克，非常便於攜帶，支持USB 3.1 TYPE-A/TYPE-C鏈接。

光盤支持方面，可讀寫120/80mm直徑光盤，支持BD-R x 6x，BD-R DL x 6x，BD-R TL x 4x，BD-R QL x 4x，BD-R（M -DISC）×4速，BD-RE×2速，BD-RE DL×2速，BD-RE TL×2速，DVD-R×8速，DVD-R DL×4速，DVD-RW×6雙倍速，DVD + R×8倍速，DVD + R DL×4倍速，DVD + RW×8倍速，DVD-RAM×5倍速，DVD + R（M-DISC）×4倍速，CD-R×40倍速CD-RW x 10速，兼容Windows和Mac OS X系統。

該光驅最大亮點是支持M-DISC儲存技術光盤（俗稱“千年光盤”），可保護用戶最有價值數據，如家庭寶貴的照片或視頻等。M-DISC是一種檔案級存儲解決方案，利用燃燒“燒洞"技術將數據寫入光盤表面的專利岩石塗層中，不會因太陽晒老化等因素導致數據損壞，即使在最惡劣的條件下也可以存儲和讀取、由於M-DISC光盤數據可穩定儲存1000年以上，所以業內稱其永久光盤或千年光盤。

此外，附贈CyberLink Power2Go 8和CyberLink PowerBackup 2.6以及Nero BackItUp專用工具，方便用戶刻錄等需求。

據悉，華碩SBW-06D5H-U刻錄光驅將於6月28日正式發售，售價18000日元（約1150元），感興趣的玩家可關注。
