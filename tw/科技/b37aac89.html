【學習筆記】Android開發之kotlin語言（一）

2020-11-14
| 科技

2011年JetBrains公佈了第一版的Kotlin語言，2012年將其開源，早期的Kotlin語言關注度不高，2016年JetBrains發佈了Kotlin的1.0正式版，2017年Google正式宣佈Kotlin成為Android開發的一級開發語言，Android Studio也加入了對Kotlin的支持。

相比較於Java語言，Kotlin語言的語法更簡單，對於同樣的功能使用kotlin語言，使用kotlin語言的代碼量會比Java減少50%甚至更多；相比於Java的老語法，kotlin還增加了許多新式語法；kotlin還在安全性上下了功夫，杜絕了空指針這個全球崩潰率最高的異常。

《第一行代碼Android》--郭霖


Kotlin Logo

來看一段Java代碼：

public class Test{  			 public static void main(String[] args)         {             System.out.println("Hello World!")；        } }

這個簡單的hello world程序用kotlin就是

fun main(){    println("Hello World!")}

是不是簡單多了

那麼，kotlin如何定義變量？

Java中 int x=a

Kotlin val m=123 和python一樣會根據你的輸入自動推導出來變量的類型，（也可以這樣寫 val m：String="abc" ）對了如果這樣寫程序：

fun main(){    val a=5    a = a+5    println(a)}

那麼這個一定會報錯的，提示“Val cannot be reassighed”，val定義的變量是不能被重新賦值的，相當於Java中的 final 關鍵字。需要定義可變變量則需要用var關鍵字。

關於函數：

看一下這個題：構造一個函數，計算兩個整數a,b的最大值。

用Java怎麼寫？

public TestJava{    public static void main(String[] args){          int a=5;         int b=6;          System.out.println(TestJava.add(a,b))；    }     public static double add(int a,int b){        return a>b?a:b    }}

用kotlin呢

fun main(){      val a=5      val b=6      println(add(a,b))}fun add(a:Int,b:Int) :Int =  if(a>b) a else b 

在kotlin語言中，當函數的代碼只有一行時候，就不必寫函數體了。


If 語句：

還是上面那個例子：

 switch(grade)      {         case 'A' :            System.out.println("優秀");             break;         case 'B' :           System.out.println("還可以");            break;         case 'C' :            System.out.println("良好");            break;         case 'D' :            System.out.println("及格");            break;         case 'F' :            System.out.println("你需要再努力努力");            break;         default :            System.out.println("未知等級");      }

在kotlin函數中，if的用法和Java是完全一樣的，但是kotlin中的if是可以有返回值的這個返回值是每一個條件的最後一行代碼的返回值舉個例子：

val max = if (a>b){        a    }    else{       b  }

when語句

kotlin中有個獨特的 when()語句，在功能上接近Java的switch語句，先用Java舉個栗子：

 switch(grade)      {            case 'A' : System.out.println("優秀");                  break;            case 'B' : System.out.println("還可以");                break;            case 'C' :  System.out.println("良好");                break;            case 'D' :  System.out.println("及格");                 break;            case 'F' :   System.out.println("你需要再努力努力");                 break;            default :   System.out.println("未知等級");      }

是不是嫌長了，而且每一個case都有一個break，有點煩，when語句是這樣用的：

when(grade){   'A' -> println("優秀")    'B' -> println("還可以")    'C' -> println("良好")   'D' -> println("及格")   'E' -> println("你需要再努力努力")    else -> println("未知等級")}

是不是簡單多了，匹配值-> { 執行代碼 }
