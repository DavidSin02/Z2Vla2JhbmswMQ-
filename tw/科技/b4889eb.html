JavaScript基礎之入口函數-2020版


1.入口函數

 window.onload = function(){     內部放js}

這個函數的意思就是說，當我們頁面加載完畢之後（就是說等頁面的結構 樣式 節點等加載完畢），採取執行函數體裡面的js部分。也就是說可以使用此方法，就不用考慮JS語句的位置了，放在獲取元素變量之前也行。

2.Js的書寫位置

Js 的書寫位置非常的自由。 也可以參照 css 的位置來分類。

2.1 行內式

一般情況，單雙引號是一樣 的 但是出現 了包裹的情況。

我們一般採取的是 外雙內單 的格式。

2.2 內嵌式

使用 <script> 標籤，在 HTML 頁面中插入 JavaScript，。

合理放置腳本位置，以優化加載體驗，js腳本放在 <body>標籤閉合之前。

那些老舊的實例可能會在 <script> 標籤中使用 type="text/javascript"。現在已經不必這樣做了。JavaScript 是所有現代瀏覽器以及 HTML5 中的默認腳本語言。

2.3 外鏈式

<script type=”text/javascript” src=”xx.js”></script>

注意：這對標記之間不能寫任何的東西。


如果涉及dom操作，且js在需要操作的dom元素之前取值，可以將相關代碼放到window.onload函數裡調用，這樣就不會報錯.

如果沒有dom操作，放到前面邏輯沒有問題，只是可能會影響性能和體驗
