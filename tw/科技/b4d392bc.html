國內非晶硅平板探測器的開創者奕瑞科技

2020-11-14
| 科技

今天科創板我們一起梳理一下奕瑞科技，公司是一家以全產業鏈技術發展趨勢為導向、技術水平與國際接軌的數字化X 線探測器生產商，主要從事數字化 X 線探測器研發、生產、銷售與服務，產品廣泛應用於醫學診斷與治療、工業無損檢測、安防檢查等領域。公司通過向全球知名客戶提供更安全、更先進的 X 線技術，助力其提升醫學診斷與治療的水平、工業無損檢測的精度或安防檢查的準確率，並提高客戶的生產效率、降低生產成本。



數字化 X 線探測器是典型的高科技產品，屬於“中國製造 2025”重點發展的高科技、高性能醫療器械的核心部件。公司是數字化 X 線探測器進口替代進程的主導者，是國內領先的通過自主研發成功實現產業化並在技術上具備較強國際競爭力的企業。公司成立後成功研製出國產非晶硅平板探測器，打破國外廠商對非晶硅平板探測器的技術壟斷，完成了產業鏈由發達國家向中國大陸的轉移。


公司在全球醫療和寵物醫療 X 線探測器市場佔有率從 8.09%增至 12.91%，2018 年位列國內第一。此外，公司在工業無損檢測、安防檢查等領域的產品可助力機械工業、高端裝備製造業、電子製造業等行業提高質量控制能力和檢測效率。


公司經過多年發展，已成為全球數字化 X 線探測器行業知名企業，產品遠銷亞洲、美洲、歐洲等地共計 70 餘個國家和地區，得到柯尼卡、銳珂、富士、西門子、飛利浦、安科銳、DRGEM、上海聯影、萬東醫療、普愛醫療、藍韻影像、東軟醫療及深圳安科等國內外知名影像設備廠商的認可。公司正逐步改變行業的競爭格局，公司在國內細分市場中始終排名第一，並在全球市場中形成局部領先地位，正加速趕超國際競爭對手。


數字化 X 線探測器是通過間接或直接的方式將 X 射線轉換為電荷，並最終數字化為醫學圖像。數字化 X 線探測器按照傳感器陣列形狀的不同，可分為平板探測器和線陣探測器；按照光子信號的轉換方式的不同，可分為積分式探測器和單光子計數式探測器；此外，光學傳感面板，一般都由光電轉化層和 TFT 陣列開關等尋址電路組成，按照這兩部分的組成材料，可分為非晶硅探測器、CMOS/單晶硅探測器、IGZO 探測器、非晶硒探測器和 CdTe/CZT（碲化鎘/碲鋅鎘）探測器等。此外，隨著柔性電子技術的發展，在消費電子領域中摺疊屏幕已經廣為人知。目前，傳統的 X 線探測器所用光學傳感面板工藝全部採用剛性襯底，但部分領先的 X 線探測器廠家已經開始應用這項技術生產柔性的光學傳感面板。



隨著數字化 X 攝影技術的進步，數字化 X 線探測器的成像質量不斷提高、成像速度不斷加快、輻射劑量不斷降低，得到世界各國的臨床機構和影像學專家認可，以探測器為核心部件的 X 線機廣泛應用於醫療和工業各個領域。據統計，2018 年全球數字化 X 線探測器的市場規模約為 17.3 億美金，其中醫療用產品市場份額約佔 74%，工業安防產品貢獻了餘下的市場。預計至 2024 年，全球數字化 X 線探測器的市場規模將達到 24.5 億美金。


2018 年，醫療用數字化 X 線探測器的市場規模約為 12.9 億美金，預計至 2024年將達到 16.7 億美金，作為 X 射線整機的核心部件，其發展趨勢需始終契合終端的臨床應用需求。


無損檢測也稱無損探傷，是在不損害或不影響被檢測對象使用性能的前提下，採用射線、超聲、紅外、電磁等原理技術並結合儀器對材料、零件、設備進行缺陷、化學、物理參數檢測的技術。其中，射線技術包括放射同位素及 X 射線兩大方向，由於放射同位素有很多應用限制，國家正逐步收緊相關政策，X 射線目前是主流的應用技術方向；相比於超聲、紅外、電磁等技術，X 射線較強的穿透力在終端應用中有更廣泛的需求。工業無損檢測目前廣泛應用於機械製造、汽車、電子、鐵路、航天航空、壓力容器、軍工等產業。


目前，全球工業數字化 X 線探測器佔整個市場份額相對較小，但在野外等工業現場等領域目前仍主要使用 X 線膠片，工業數字化 X 線探測器作為 X 線膠片的升級替代產品存在較大的市場上升空間。2018 年，全球工業數字化 X 線探測器的市場規模約為 1.9 億美金，預計至 2024 年將達到 3.1億美金，年複合增長率超過 9%。除了傳統的無損探傷外，動力電池檢測和半導體後段封裝檢測成為近年來 X線探測器在工業領域應用新的增長點。


隨著全球各國對公共安全問題的不斷重視，以及機場、鐵路、城市軌道交通等基礎設施的建設，X 線安檢設備需求保持快速增長。2018 年，全球安防檢查數字化 X 線探測器的市場規模約為 2.6 億美金，預計至 2024 年將達到 4.7 億美金，年複合增長率超過 10%。數字化 X 線探測器作為所有 X 線安防設備的核心部件，隨著安防檢查市場的擴張而擁有巨大的市場前景。隨著國家對基建持續的投入和一帶一路沿線國家的基礎建設，社會安檢需求將持續增長，公司的線陣探測器產品線在安防檢查領域應用前景廣闊。


公司發展勢頭良好，銷售數量及營業收入均實現大幅增長，增長速度均遠遠超出行業的發展速度。其中，銷售數量由 0.58 萬臺增至 1.13 萬臺，年複合增長率達到 39.7%；營業收入由 3.68 億元增至 5.46 億元，年複合增長率達到 23.9%。公司在全球醫療用數字化 X 線探測器行業市場佔有率穩步提升，根據 IHS Markit 統計，公司 2018 年全球醫療及寵物醫療探測器市場份額位列國內第一。



2019 年，公司醫療用數字化 X 線探測器全球市場佔有率達 12.91%，較 2018年增長超過 3 個百分點。公司當前主要量產產品為醫療靜態探測器，產品主要銷售給 DR 廠商用以整機配套，公司在國內市場份額始終處於領先地位，在全球市場中亦屬於領先集團，近年市場佔有率穩步提升；同時，公司亦著手於醫療動態領域進行佈局。2019 年，公司的動態數字化 X 線探測器銷售數量達到 842 臺，報告期內年複合增長率高達 268.5%，在國內廠家中處於領先地位。


一、國內非晶硅平板探測器的開創者


奕瑞科技成立於2011年，完成首款產品的測試與交付，打破國外品牌對國內市場的壟斷局面；2013年於西門子和飛利浦達成戰略合作協議；2014年於柯尼卡和萬東醫療達成戰略合作協議，同年發佈首款無限產品；2014年發佈首款動態產品，國內平板探測器出貨量第一；2016年與上海聯影達成戰略合作協議；2017年開始佈局醫療動態、工業、寵物、安防等新領域；2018年發佈首款IGZO平板，同年成功研發牙科產品；2019年具備柔性X線探測器製造生產能力；2020年科創板上市。



二、業務分析


2016-2019年，營業收入由2.56億元增長至5.46億元，複合增長率28.72%，19年同比增長24.37%，2020年上半年實現營收同比增長91.67%至3.85億元；歸母淨利潤由0.43億元增長至0.96億元，複合增長率30.70%，19年同比增長24.37%，2020年上半年實現歸母淨利潤同比增長330.28%至1.10億元；扣非歸母淨利潤由0.40億元增長至0.85億元，複合增長率28.56%，19年同比增長24.37%，2020年上半年實現扣非歸母淨利潤同比增長392.80%至1.02億元；經營活動現金流分別為0.4億元、1.02億元、0.19億元、0.31億元，2020年上半年實現經營活動現金流同比增長584.16%至0.60億元。



分產品來看，2019年普放有線系列實現營收2.01億元，佔比38.59%，毛利率增加4.16pp至50.44%；普放無線系列實現營收2.75億元，佔比52.91%，毛利率增加6.84pp至46.95%；乳腺系列實現營收405.86萬元，佔比0.78%，毛利率減少2.56pp至65.91%；放療系列實現營收1253.51萬元，佔比2.41%，毛利率增加2.13pp至79.08%；工業系列實現營收2762.37萬元，佔比5.31%，毛利率減少0.50pp至60.25%。



2019年前五大客戶實現營收2.46億元，佔比45.04%，其中第一大客戶柯尼卡實現營收1.31億元，佔比23.91%。



三、核心指標


2016年-2019年，毛利率由50.82%下降至18年階段低點45.99%，19年回升至49.27%；期間費用率17年升至階段高點19.65%，隨後逐年下降至15.71%，其中銷售費用率維持在7%左右，管理費用率17年升至階段高點12.34%，隨後逐年下降至7.09%，財務費用率由-1.1%上漲至0.63%，利潤率由16.72%下降至18年13.5%，19年回升至17.45%，加權ROE由27.36%下降至20.86%，19年回升至25.52%。

淨資產收益率=利潤率*資產週轉率*權益乘數


由圖和數據可知，權益乘數呈上漲趨勢，16-18年淨資產收益率的下降主要是由於利潤率和資產週轉率的下降，19年淨資產收益率的回升主要是由於利潤率的回升和權益乘數的提高。


看點：


目前，全球數字化 X 線探測器生產廠家不到 20 家，其中，通過自主研發掌握核心技術並達成穩定量產的不到 10 家。為數不多的行業競爭者中，以公司為代表的國內廠家成功打破國外廠商的技術壟斷並實現進口替代和產業化。
