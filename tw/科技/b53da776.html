數據結構和算法之二叉樹

2020-11-14
| 科技

二叉樹是樹型數據結構的一種，二叉就是每個節點最多有兩個子樹(度不超過兩個)，就是左節點、右節點。根據不同的特點又引申出完全二叉樹、滿二叉樹、平衡二叉樹、紅黑樹、二叉排序樹、一般二叉樹等。

節點是二叉樹的基礎，節點連接成複雜結構的單元；二叉樹是n個節點集合，有且僅有一個根節點，最多有連個子節點，每個子節點又可以看成子樹，所有子樹是不相交的。樹的定義本身就是一個遞歸的概念，所以樹的各種操作也都是遞歸來實現的。度是節點擁有的子樹的個數，二叉樹的度最大為2；節點的層，就是節點在樹的哪個深度，樹的深度樹樹最大的層數。

斜樹，只有左子樹的二叉樹叫左斜樹，只有右子樹的二叉樹叫右斜樹，統稱為斜樹。

滿二叉樹，所有非葉子節點的度都是2，所有葉子節點都在同一層

完全二叉樹，葉子節點在最下層和次下層，且最下層的葉子節點在左側，滿二叉樹是完全二叉樹的一種。

二叉樹的遍歷，前中後序遍歷，深度廣度優先遍歷。

前序遍歷從跟節點出發先遍歷左子樹的左節點一次從左到右訪問，ABDHIEJCFG

中序遍歷，從左中右訪問模式，HDIBJEAFCG

後續遍歷，左右中訪問模式，HIDJEBFGCA

//前序遍歷func PreOrderTraversal(tree *TreeNode)  {	if tree == nil {		return	}	fmt.Println("%d ->", tree.Val)	PreOrderTraversal(tree.Left)	PreOrderTraversal(tree.Right)}// 中序遍歷func MidOrderTraversal(tree *TreeNode) {	if tree == nil {		return	}	MidOrderTraversal(tree.Left)	fmt.Printf(" %d -> ", tree.Val)	MidOrderTraversal(tree.Right)}// 後序遍歷func PostOrderTraversal(tree *TreeNode) {	if tree == nil {		return	}	PostOrderTraversal(tree.Left)	PostOrderTraversal(tree.Right)	fmt.Printf(" %d -> ", tree.Val)}

按層遍歷，從上到下按層依次遍歷，ABCDEFGHIJ

// 按層遍歷func LevelOrderTraversal(tree *TreeNode) {	if tree == nil {		return	}	// 採用隊列實現	queue := make([]*TreeNode, 0)	queue = append(queue, tree) // queue push	for len(queue) > 0 {		tree = queue[0]		fmt.Printf(" %d -> ", tree.Val)		queue = queue[1:] // queue pop		if tree.Left != nil {			queue = append(queue, tree.Left)		}		if tree.Right != nil {			queue = append(queue, tree.Right)		}	}}
