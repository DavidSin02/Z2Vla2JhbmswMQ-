5.Vue.js

Vue.js是一套用於構建用戶界面的漸進式框架。與其它大型框架不同的是，Vue 被設計為可以自底向上逐層應用

Vue 的核心庫只關注視圖層，不僅易於上手，還便於與第三方庫或既有項目整合。另一方面，當與現代化的工具鏈以及各種支持類庫結合使用時，Vue 也完全能夠為複雜的單頁應用提供驅動

關於Vue的介紹及使用，可參考此專欄：淺談Vue.js

6.React

React

是一個用於構建用戶界面的 JAVASCRIPT 庫

主要用於構建UI，很多人認為 React 是 MVC 中的 V（視圖）

起源於 Facebook 的內部項目，用來架設 Instagram 的網站，並於 2013 年 5 月開源

擁有較高的性能，代碼邏輯非常簡單，越來越多的人已開始關注和使用它

  • Bulma是一個基於 Flexbox 構建的免費、開源的 CSS 框架
  • Bulma 框架最大的特點，就是簡單好用。所有樣式都基於class，只需為 HTML 元素指定class，樣式立刻生效

8.RAP2

RAP2是一個可視化接口管理工具，通過分析接口結構，動態生成模擬數據，校驗真實接口正確性， 圍繞接口定義，通過一系列自動化工具提升我們的協作效率。

其具有可視化編輯，完善的版本控制，各種格式的導入導出功能。讓前後端約定接口的工作變得十分的簡單

RAP2會自動根據接口文檔生成Mock接口，這些接口會自動生成模擬數據，支持複雜的生成邏輯。
