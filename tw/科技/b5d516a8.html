什麼是雷電三接口？電腦的這個接口能做什麼？

2020-11-14
| 科技

大家好，我是小匠。細心的朋友可能會發現部分筆記本電腦上，多 了一種“新”的接口，叫做“雷電3”，但其實你仔細看的話，就會發現，這不就是USB-Type C接口嗎？為什麼叫做雷電3接口呢？USB Type C接口和雷電3接口一樣嗎？今天就簡單的給大家講講雷電3接口。

首先，先來了解一下USB Type C。

USB Type C是USB接口的其中一種接口類型（外形），還有其他好幾種大家應該都清楚，Type C的優點在於正反兩面都一樣，可以隨便插，並且體積小，最高可達到10Gbps的傳輸速度，同時還能承載100W電力傳輸，當然視頻傳輸也可以，功能非常強大，不過現在普及程度還不高。

其次，再來了解USB 3.1。

USB Type C是接口類型，而USB 3.1是接口協議，這個大家要分清楚，雖然Type C和USB 3.1協議是一同推出的，但是這兩者不一定相互包含，簡單來說，就是Type C接口不一定支持USB 3.1協議，而使用USB 3.1標準的接口也不一定要是USB Type C接口，也可以是Type A，像現在的部分主板都支持USB 3.1的USB A口。

接下來就說到雷電3了。

雷電接口是由Intel開發制定的，蘋果有參與，接口為mini DP，此前因為雷電接口僅蘋果設備上使用，再加上會用雷電mini DP接口的外設的價格都非常高，所以推廣難度很大，直到Intel看見USB Type C逐漸普及之後，就宣佈將雷電3的原接口mini DP改為USB Type C。

因此，USB Type C只是雷電3的依附接口類型，外形看起來一樣，但是雷電3可以實現Type C的絕大多數功能，並且傳輸效率更高，而為了區分雷電3接口，該接口會在附近使用一個“閃電”標誌。

因此，現在的雷電3接口，可以看成是原有的雷電3拋棄mini DP接口改用USB Type C接口後，再配合USB 3.1標準而出現的產物，而這個接口的性能極其強大，傳輸速度高達雙向40Gbps，這也就意味著有足夠的帶寬來外接各類擴展塢，比如高性能獨立顯卡。你可以想象一下，買了一臺不帶獨顯的輕薄本，僅一個雷電3接口就能外接RTX2080Ti的感覺。

不過，目前使用到雷電3接口的筆記本價格還是偏高，普及難度還是比較大的，而市面上也很少有朋友需要雷電3接口的轉接設備與外設，但是雷電3對於輕薄本來說的確是一個很好的擴展接口，因為採用雷電3的話，輕薄本可以放棄很多其他接口，可以通過外接顯卡塢之類的設備來達到更高的性能。
