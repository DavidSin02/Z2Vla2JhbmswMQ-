多態定義

多態性是指允許不同類的對象對同一消息作出響應。多態性包括參數化多態性和包含多態性。多態性語言具有靈活、抽象、行為共享、代碼共享的優勢，很好的解決了應用程序函數同名問題。多態有兩種表現形式：重載和覆蓋。

重載（overload）

重載是發生在同一類中。與父類和子類、繼承毫無關係。

標識一個函數除了函數名外，還有函數的參數（個數和類型）。也就是說，一個類中可以有兩個或更多的函數叫同一個名字，而他們的參數不同。他們之間毫無關係，是不同的函數，只是可能他們的功能類似，所以才命名一樣，增加可讀性，僅此而已！舉個簡單的例子，我們想設計一個比較兩個數大小的方法，大家知道java中關於數值比較的類型分為int、 long、float 和double ，如果我們設計的方法想要支持上面四種類型，那麼就要分別設計四個方法：

equalInt(int p_a,int p_b)

equalLong(long p_a, long p_b)

equalFloat(float p_a, float p_b)

equalDouble (double p_a, double p_b)

這四個方法的本質都是對兩個數值的比較，我們完全可以根據重載原理，將他們統一命名為equal，然後通過不同的參數類型將其區分，具體代碼實現如下：

public class PolymDemo {		public void equal(int p_a,int p_b){				if (p_a==p_b)			System.out.println(p_a+"等於"+p_b);		else			System.out.println(p_a+"不等於"+p_b);					}	 public void equal(float p_a,float p_b){				if (p_a==p_b)			System.out.println(p_a+"等於"+p_b);		else			System.out.println(p_a+"不等於"+p_b);					}  public void equal(double p_a,double p_b){				if (p_a==p_b)			System.out.println(p_a+"等於"+p_b);		else			System.out.println(p_a+"不等於"+p_b);					}public void equal(long p_a,long p_b){				if (p_a==p_b)			System.out.println(p_a+"等於"+p_b);		else			System.out.println(p_a+"不等於"+p_b);					}}

覆蓋(override)

覆蓋是發生在子類中！也就是說必須有繼承的情況下才有覆蓋發生。

我們知道繼承一個類，就有了父類的全部方法，如果你感到哪個方法不符合需求，功能要變，那就把那個函數在子類中重新實現一遍。這樣再調用這個方法的時候，就是執行子類中的過程了。父類中的函數就被覆蓋了。注意：覆蓋的時候函數名和參數要和父類中完全一樣,不然你的方法對父類中的方法就不起任何作用，因為兩者是兩個函數，毫不關係！

示例代碼：

public class Car {public void drive(){ System.out.println("Car的drive方法。");}}public class Audi extends Car {public void drive(){ System.out.println("Audi的drive方法。");}}public class DasAuto extends Car {public void drive(){ System.out.println("Das Auto的drive方法。");}}public static void main(String[] args) { Car car1=new Audi();Car car2=new DasAuto();car1.drive();car2.drive();}輸出結果為：Audi的drive方法。Das Auto的drive方法。

覆蓋在實際編碼中應用相當廣泛，toString方法是Object裡面的方法，而所有類都是繼承Object，所以所有對象都有這個方法。在Integer、Long和Float三個類中都重寫了該方法以實現自身的返回字符串的需求！具體重寫代碼如下所示：
