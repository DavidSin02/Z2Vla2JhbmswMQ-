礦石結構構造不知，如何更好選礦

2020-11-14
| 科技

礦石構造是指組成礦石的礦物集合體的特點,即礦物集合體的形態、相對大小及其空間相互的結合關係等,它所反映的是礦物及其集合體的空間分佈特徵。礦物集合體是組成礦石構造的基本單位, 礦物集合體空間的相互結合關係組成各種形態的礦石構造 。

礦石結構是指礦石中礦物顆粒的特點,即礦物顆粒的形狀、相對大小、相互嵌布關係或礦物顆粒與礦物集合體的嵌布關係,它所反映的是礦物本身的形態特徵。礦物顆粒是組成礦石結構的基本單位, 礦物顆粒的大小和形態各有不同, 礦物顆粒間的相互嵌布關係組成了各種形態的礦石結構。

礦石的結構、 構造所反映的雖是礦石中礦物的外形特徵, 但卻與它們的生成條件密切相關,因而對於研究礦床成因具有重要意義。礦石的結構、構造特點,對於礦石的可選性同樣具有重要意義,而其中最重要的則是有用礦物顆粒形狀、大小和相互結合的關係,因為它們直接決定著破碎、 磨礦時有用礦物單體解離的難易程度以及連生體的特性。 因此, 礦石的結構和構造是礦石破碎、磨礦和選礦工藝選擇的主要依據,也是選礦工藝礦物學的重要內容。

　1 礦石的主要構造類型及其特性

礦石的構造在礦相學等地質書籍中已有詳細介紹, 這裡主要是對與選礦工藝有關的礦石構造進行列表分類,並對每一類構造進行簡單描述。礦石的主要構造類型見表2-3-1。

A 浸染狀

在描述浸染狀構造時, 要指出有用礦物是單一的還是幾種礦物集合體, 然後指出集合體粒度及含量(可以是估量)和分佈均勻度,並指出基質非金屬礦物的種類。浸染狀構造分為以下三類:

(1)浸染狀構造:屬於中粒嵌布,粒度大於0.1mm,有用礦物含量大於3%,嵌布較為均勻 。

(2)星點狀構造:屬於細粒嵌布,粒度小於0.1mm,有用礦物含量小於3%,嵌布不均勻。

(3)斑點狀構造:有用礦物粒度大小不均,分佈也不均勻,呈散染狀嵌布在礦石中。

浸染狀構造除星點狀構造外, 大多數是易於解離的 。 只要磨細到超過金屬礦物的粒度大小,就能與脈石解離。但當有用礦物中含有細小雜質包裹體時,便難於解離。本類構造以浸染狀為主, 常見於岩漿期後礦石中 。

B 延長狀

在描述延長狀構造時, 需要說明條帶 (或夾層) 的組成礦物種類及他們之間的關係、集合體的寬度、 條帶的對稱性、 重複出現或與脈石相間成條帶等。 還要指出條帶界線的規則程度以及是否再經過溶蝕作用等現象。本類構造分為條帶狀構造、層狀構造、脈狀構造和透鏡狀構造等。

延長狀構造適於選礦,但如條帶中構成礦物的結構複雜,溶蝕交代現象顯著或細小脈石礦物包裹體較多時, 同樣影響細磨造成解離上的困難。

C 渾圓狀

渾圓狀構造分為結核狀(包括腎狀、豆狀)構造、 鮞狀構造及環帶狀構造等。在描述渾圓狀構造時, 除了說明渾圓狀礦物顆粒的大小及組成成分外, 更主要的是要把環帶中心及相間環帶中含有何種雜質礦物及它們的顆粒大小、 形態等加以說明 。 本類構造以環帶及鮞狀構造為主,常見於淺成礦床及沉積礦床。若鮞粒核心大部分為一種有用礦物組成,另一部分鮞粒核心為脈石礦物組成,膠結物為脈石礦物,此時可在較粗的磨礦細度下(相當於鮞粒的粒度) ,得到粗精礦和最終尾礦。欲再進一步提高粗精礦的質量,常需要磨到鮞粒環帶的大小,此時磨礦粒度極細,造成礦石泥化,使分選指標急劇下降。因此,複雜的鮞狀構造礦石採用機械選礦的方法一般難以得到高質量的精礦。

D 不規則狀

不規則狀構造分為塊狀構造、角礫狀構造、交錯網脈狀構造、土狀、皮殼狀構造、蜂窩狀構造等 。

(l)塊狀構造。礦石礦物集合體成緻密排列,礦石內礦物的分佈沒有一定的規律,有用礦物呈結晶集合體作為礦石的基質部分, 而脈石礦物成細小顆粒或細脈存在於礦石中 。 在描述時要說明構成礦石基質的有用成分是單一金屬礦物還是多金屬礦物, 與非金屬礦物在塊狀礦石中存在的形態以及分佈的均勻程度 。 這些因素直接影響選礦工藝流程及精礦品位。

( 2 ) 角礫狀構造 。 角礫狀構造的特點是由兩種不同時代的礦物集合體結合而成的, 早期礦物形成角礫,後期礦物形成膠結物。若有用礦物組成角礫,脈石礦物為膠結物,則破碎到角礫粒度便可獲得最終尾礦。但如有用成分為膠結物,脈石礦物為角礫,則需在粗磨後,經選別再細磨。

(3)交錯網脈狀構造。交錯網脈狀構造的特點是在早期形成的礦物集合體中穿插有晩期生成礦物的不規則網脈。 從可選性的觀點來看, 它和角礫狀的礦石基本相同 。 如果網脈很細小,也影響有用礦物與脈石的解離。在描述時需特別注意網脈狀礦物集合體的結構及粒度 。

(4)土狀、皮殼狀、蜂窩狀構造。土狀、皮殼狀、蜂窩狀構造主要在氧化礦石中較常見。這種構造的礦物成分較複雜,有用礦物較細小,結構鬆軟,在選礦過程中易泥化,對富集有較大影響 。 具有這種構造的礦石需經特殊處理 。

　2 礦石的主要結構類型及其特性

礦石的結構類型很多, 在礦相學等地質書籍中已有較詳細的介紹 。 這裡主要對與選礦工藝有關的礦石結構進行列表分類,並對每一類結構進行簡單描述。礦石的主要結構類型見表2-3-2。

表 2 礦石主要結構類型

A 結晶作用形成的結構

結晶作用形成的結構是以結晶程度進行分類, 包括自形晶粒結構、 半自形晶粒結構、他形晶粒結構、 斑狀集晶結構和隱晶結構。

(l)自形晶粒結構。自形晶粒結構礦物結晶顆粒具有完好的結晶外形,如錫石、黃鐵礦、 毒砂等。 礦物顆粒間由於自形程度完整而接觸界線平滑, 礦石經碎磨後易於沿礦物間接觸面解離, 所以解離程度最好。

(2) 半自形晶粒結構。 半自形晶粒結構是由兩種或兩種以上礦物晶粒的關係而定。 其中有一種晶粒是各種不同自形程度的結晶顆粒, 較後形成的顆粒則往往是他形品並溶蝕交代先前形成的礦物顆粒。

(3) 他形晶粒結構。 礦物晶粒不具有完整晶面, 常位於自形晶顆粒的間隙或裂隙中, 因此礦物的外形是不定的 。 常由一種或幾種礦物的他形顆粒集合體組成, 常見於黃銅礦、閃鋅礦、 方鉛礦等。 幾種礦物集合體的接觸界線不平整, 有時溶蝕顯著, 呈彎曲狀的接觸線 。

(4)斑狀集晶結構。細粒礦物集合體的基質中,具有一定程度的自形礦物的巨晶,這種巨晶稱為斑狀集晶結構 。 這種結構一般溶蝕現象不顯著, 因而接觸界線較為平整 。 較常見的有黃鐵礦、 毒砂, 在閃鋅礦細粒基質中呈斑狀集晶結構 。

(5)隱晶結構。礦物結晶組分異常小,以至於用普通顯微鏡不能區別它們,如隱晶質的石墨等 。

呈自形、 半自形及斑狀集晶結構的礦物, 如果有用礦物粒度不是過細, 在磨礦過程中單體解離度是較高的。而呈他形晶粒結構,單體解離就比較困難。結晶的石墨、輝鉬礦是最易浮的。但具隱晶質結構的石墨、輝鉬礦其可選性很差。

B 交代作用形成的結構

交代作用形成的結構包括以下幾種:

(1) 溶蝕結構。 溶蝕結構是指早期生成礦物的晶粒被晩期生成礦物集合體所溶蝕交代, 因而兩種礦物的接觸界線很不規則, 常呈不平坦的鋸齒形的邊界線。 這種結構在硫化礦床的礦石中分佈極為普遍 。

(2)交代殘餘結構。交代殘餘結構的特徵是早期生成的礦物晶粒被晚期生成礦物集合體的溶蝕交代進一步加深, 晩期礦物的溶蝕交代作用更強烈, 使被溶蝕的早期礦物的晶粒僅剩殘餘物, 在晩期礦物集合體內這些殘餘物顆粒極不規則, 接觸界線也極為複雜 。

(3) 交錯結構 。 交錯結構是指較晚期的礦物侵入到早期礦物顆粒內而呈極不規則的樹枝狀、 網格狀和其他不規則狀的外形, 這種顆粒之間接觸界線大多是彎曲的, 此類結構在硫化礦石中較普遍 。

(4) 骸晶結構 。 骸晶結構是指較早結晶的礦物被晚期礦物從中心向外進行交代, 而早期礦物仍保持著原來一定的外形。這種結構是黃鐵礦和毒砂所特有的。

(5)鑲邊結構。鑲邊結構是指晩期礦物沿早期礦物邊緣交代,形似鑲邊。如銅藍、藍輝銅礦沿黃銅礦邊緣交代 。

呈交代作用結構的礦物的可選性,從溶蝕、交代殘餘到交錯結構, 一般礦物的解離程度逐漸變差 (在其他條件相同情況下) , 這類結構的礦石需細磨到遠遠小於有用礦物的粒度, 才能獲得單體解離, 選礦要徹底分離它們是比較困難的 。

C 固溶體分離作用形成的結構

固溶體分離作用形成的結構包括以下幾種:

(1) 乳濁狀結構。 乳濁狀結構是指一種礦物在另一種基質礦物中呈極細小的乳滴狀嵌布,乳滴狀顆粒分佈一般無規律性,但有時可見少數顆粒成平滑的接觸線。如閃鋅礦中乳滴狀黃銅礦、 方鉛礦中乳滴狀輝銀礦 。

(2) 格狀結構。 格狀結構是指一種礦物分佈在另一種礦物內的幾個不同結晶方向, 可通過光片在高倍鏡下觀察, 晶片在主要礦物中成三角形、 菱形、 矩形或由四、 五組平行排列的晶片而組成的格子。 如鈦鐵礦在磁鐵礦中呈格狀結構, 赤鐵礦一鈦鐵礦、 輝銅礦一斑銅礦等均屬格狀結構。 (3 ) 板狀結構。 板狀結構是指礦物呈板狀晶體分佈在基質礦物中, 有時具有定向排列 。 如磁鐵礦顆粒中的赤鐵礦板狀晶體, 黃銅礦中方黃銅礦板狀晶體等。

格狀等固溶體分離結構, 由於接觸邊界平滑, 也比較容易分離, 但對於呈細小乳滴狀的礦物顆粒,要解離出來就非常困難。

D 重結晶作用形成的結構

重結晶作用形成的結構包括以下幾種:

(1)放射狀、球粒狀結構。放射狀、球粒狀結構的特點是組成礦物的纖維晶體以放射狀排列組成放射狀結構 。 若放射狀結構的分泌物成為滾圓的堆積, 則稱為球粒狀結構 。 一般可通稱為放射狀、 球粒狀結構 。

(2) 環帶膠體結構 。 環帶膠體結構是指膠體物質經過重結晶作用形成的變晶顆粒常保留有凝膠沉澱物的同心環帶,常見的有膠狀黃鐵礦、針鐵礦、孔雀石等。

(3 ) 壓碎結構。 壓碎結構是指礦物受壓力作用而呈現裂縫和帶稜角的碎塊, 這種結構是堅硬而又性脆的礦物所特有的。如黃鐵礦、毒砂、錫石、鉻鐵礦等。壓碎結構一般有利於磨礦及單體解離。
