C語言學習|指針和數組之間的關係

2020-11-14
| 科技

在C語言中，指針和數組是相互關聯的。

首先，指針的算術運算和關係運算常常是針對指針數組中的元素而言的。由於數組在內存中是連續存放的，因此指向同一數組中不同元素的兩個指針的關係運算常用於比較它們所指元素在數組中的前後位置的關係。指針的算術運算則常用於移動指針的指向，使其指向數組中的其它元素。當然，僅當指針的算術運算結果仍然指向同一數組中的元素時才有意義。

其次，指針和數組的關係還表現在：在表達式中，數組名被自動轉化為指向數組中的第一個元素的常量指針（除用作sizeof運算的操作數外）。例如：

int a[100];

int *p;

p=a;

因為a被自動轉換為指向a[0]的一個int型常量指針，所以經p=a賦值後，p也指向了a[0]。

最後，數組的下標運算符[]實際上是以一個指針作為其操作數。例如，數組元素a[i]被編譯器解釋為表達式*（a+i），表示從數組首地址開始向後移動i個元素並取出其內容，而&a[i]表示取數組a的第i+1個元素的地址，它等價於指針表達式a+i。因此，數組元素可以通過指針來引用，指針也可以用下標形式來表示。可用數組實現的操作也可用指針來實現。只是指針實現比數組實現的效率高，而數組實現方法更容易理解。

一、一維數組的地址和指針

1.一維數組的指針概念

由於數組元素在內存中是連續存放的，即在內存中佔用一段連續的存儲單元，每個數組元素佔用其中的一個存儲單元。因此，數組元素實質上就是一個變量，數組元素的指針就是該元素在內存中所佔存儲單元的首地址，而一維數組的指針就是數組中各元素所佔連續存儲單元的起始地址，即第一個元素的指針。例如：

定義了一個有3個短整型元素的一維數組a，如圖1-1所示，因為數組a的每個元素在內存中佔2個字節的內存，所以有3個元素的數組a在內存中總計佔10個字節的連續的存儲單元，其起始地址可通過&a來得到。因為數組名a代表數組的首地址，即元素a[0]的地址（&a[0]），所以要定義一個指向一維短整型數組的指針變量p，並使其指向數組a的首地址，可用如下的變量語句：

short *p=a;

則相當於語句：

short *p=&a[0];

2.一維數組的指針引用一維數組元素

既然數組名a代表數組的首地址（即&a[0]），那麼表達式a+1就表示首地址後面的下一個元素的地址，即&a[1]。由此可見，表達式a+i代表數組中下標為i的元素a[i]的地址（&a[i]).。

不僅可以通過下標方式來引用數組中的元素，還可以通過使用間接尋址運算符*來引用數組的元素。例如，*a表示取出首地址a所指存儲單元中的內容，即元素a[0]，*(a+i)表示取出首地址元素後面第i個元素的內容，即下標為i的元素a[i]。

此外，還可以通過指向一維數組的指針變量p來引用數組a中的元素，如圖1-2所示。

當指針變量p指向數組的首地址&a[0]時，*p就表示取出p所指單元即元素a[0]的值，此外，由於p+i代表當前指針所指元素後面的第i個元素的地址，因此*（p+i）表示取出p+i所指向的存儲單元內容，即元素a[i]的值。*（p+i）也可以用它的下標形式p[i]來表示。

與通過*（a+i）來引用數組元素a[i]不同的是，由於指針變量p不一定永遠都指向數組的首地址&a[0]，因為僅當指針變量p指向數組的首地址&a[0]時，*（p+i）才是對a[i]的引用。否則，如果通過增1運算改變指針變量p的指向，使p直接指向a[i]，那麼就是對a[i]的引用了。

此外，需要注意的是，在傳統的算術中，1000+1的結果是1001，但是對於指針的算術就不是這樣了。當給指針加上一個整數或者從指針中減去一個整數時，並非簡單地增減這個整數值，而是這個整數值乘以指針所指向對象的字節長度。這個字節長度取決於指針所指向的對象的數據類型。不過，當對一個字符數組執行的指針算術運算時，運算的結果和普通的算術運算的結果是不一樣的，因為每個字符只佔1字節。

小貼士：

1）數組名a是一個地址常量，不能通過賦值操作改變它的值，指針變量p是一個變量，可以通過賦值運算改變它的值，從而使p指向數組中的其它元素

2）雖然p+1和p++都指向當前指針所指單元的下一個元素，但p+1並不改變當前指針的指向，p仍然指向原來所指向的元素，而p++相當於執行p=p+1，因此p++操作改變了指針p的指向，表示將指針變量p向前移動使其指向了下一個元素。

3）p+1並非將指針變量p的值簡單地增加1字節，事實上，它是增加了1*sizeof（指針的基類型）字節

4）指針的算術運算僅在作用於數組時才有意義。因為不能假設同一數據類型的兩個變量在內存中相鄰存儲的，除非它們是數組中的兩個相鄰的元素。同理，對並非指向同一數組的元素的兩個指針進行相減運算，也是無意義的。

如下有五種輸入/輸出數組元素的方法：

方法1 用下標法直接引用數組元素

int a[10],i;

for(i=0;i<10;i++)

{

scanf("%d",&a[i]);

}

for(i=0;i<10;i++)

{

printf("%d",a[i]);

}

方法2 通過數組名間接引用數組元素

int a[10],i;

for(i=0;i<10;i++)

{

scanf("%d",a+i);

}

for(i=0;i<10;i++)

{

printf("%d",*(a+i));

}

方法3 通過指針變量間接引用數組元素

int a[10],*p=NULL,i;

p=a;

for(i=0;i<10;i++)//循環開始前，確保指針p指向數組的首地址

{

scanf("%d",p+i);

}

p=a;

for(i=0;i<10;i++)//循環開始前，確保指針p指向數組的首地址

{

printf("%d",*(p+i));

}

方法4 通過指針變量增1運算間接引用數組元素

int a[10],*p=NULL;

for(p=a;p<a+10;p++)//循環開始前，確保指針p指向數組的首地址

{

scanf("%d",p);

}

for(p=a;p<a+10;p++)//循環開始前，確保指針p指向數組的首地址

{

printf("%d",*p);

}

方法5 用指針的下標表示法引用數組的元素

int a[10],*p=NULL,i;

p=a;

for(i=0;i<10;i++)//循環開始前，確保指針p指向數組的首地址

{

scanf("%d",&p[i]);

}

p=a;

for(i=0;i<10;i++)//循環開始前，確保指針p指向數組的首地址

{

printf("%d",p[i]);

}
