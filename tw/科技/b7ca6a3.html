適合在任何地方使用的 Linux：15 個小型 Linux 發行版

2020-10-29
| 科技

如果你有一臺老舊的 PC 或超小型設備，這些 Linux 發行版中的一個應該適合你。

  • 來源：https://linux.cn/article-12281-1.html
  • 作者：David Gewirtz
  • 譯者：Xingyu.Wang

如果你有一臺老舊的 PC，或者是基於單板計算機的小型系統，你可能希望運行的是一個佔用資源較少且易於下載的 Linux 版本。在這裡，我們來看看佔用資源非常小的 Linux 發行版。除了 Damn Small Linux 之外，這裡列出的所有 Linux 發行版都是最近才更新的。每個發行版都可以在內存不足 1GB 的機器上運行，而且大多數發行版都可以在更小資源內運行。以下排列以字母排序。

Alpine Linux

Alpine Linux - 130MB download


下載：130 MB

Alpine Linux 最初是針對虛擬服務器和設備的，它直接運行在內存中。它以安全為重點，面向最終用戶的功能較少，但如果你需要的話，也可以支持桌面功能。上一次公開發布是在 2019 年 10 月。

antiX Linux

antiX Linux - about 1GB download


下載：約 1GB

antiX 發行版聲稱完全不含 systemd —— 這是現代 Linux 發行版的基本構件。相反，antiX 提供了“antiX Magic”（我不確定那是什麼）。我會告訴你，這個安裝程序很複雜，有很多選項。這是一個基於 Debian 的構建版。最後一次更新是在去年 10 月份。

ArchBang

ArchBang - 600-700MB download


下載：600 - 700MB

ArchBang 是一款受 CrunchBang 啟發的發行版，其使用 Arch Linux 的 Openbox 窗口管理器。最近一次發佈是在 2019 年 11 月。

Bodhi Linux

Bodhi Linux - 700MB download


下載：700MB

Bodhi 是一款基於 Ubuntu 的輕量級發行版，這款發行版採用的是 Moksha（一種基於 Enlightenment 的桌面）。它的模塊化和可定製性很強，可以讓用戶構建非常簡單或合理的系統。它最近剛剛更新過。

BunsenLabs Helium

BunsenLabs Linux Helium - 600MB to 1.2GB download


下載： 600MB 至 1.2GB

前有 CrunchBang 和 CrunchBang++，今有 BunsenLabs Helium 和 ArchBang，這兩者都是在 CrunchBang++ 發行版的基礎上構建的。Helium 基於 Debian，採用 Openbox 窗口管理器。最新的版本是在 2019 年 7 月發佈的。

Damn Small Linux

Damn Small Linux - 50MB download


下載：50MB

Damn Small Linux 自 2008 年以來一直沒有正式發佈，雖然 2012 年曾經有過一個候選版本。如果你有舊的硬件，這應該沒什麼問題，但不要在互聯網上多用，因為較新的漏洞還沒有打補丁。其最輕量級的形式可以在 16MB 的內存中運行，完全加載也只需要 128MB 內存。

Elive

Elive - 2.4GB download


下載：2.4GB

Elive 是一款即用版 Enlightenment 發行版，其中還包含了 Elpanel 控制面板。按照真正的 Linux 風格，它的口號是：“雖然 Elive 並不適合所有人，但你是例外嗎？” 它只要 256MB 內存就能運行。Elive 最後一次更新是在 2019 年 10 月。

Linux Lite

Linux Lite - 1.4GB download


下載：1.4GB

Linux Lite 並不像一些微型發行版那樣小，但它也不會讓你感到多麼龐大。它配備了你需要的所有組件，基於 Ubuntu，運行著一個令人感到愉快的 Linux 桌面環境。它可以在 768MB 內存中運行，但 1GB 內存更好。它最近剛剛更新。

Lubuntu

Lubuntu - 1.6GB download


下載：1.6GB

Lubuntu 名字的意思是“輕量級 Ubuntu”，顧名思義，你就知道這個發行版是怎麼回事了。它是使用 LXQt 桌面環境構建的，你可以選擇你想安裝什麼典型的 Ubuntu 軟件包。它也是最近剛剛更新的。

LXLE

LXLE - 1.2GB download


下載：1.2GB

LXLE 基於 Ubuntu，有一個非常具體的口號：“重新喚醒那臺老舊的 PC。”它的設計理念是擁有豐富的功能和應用，可以讓你把一臺老舊的 PC 變成一臺可以工作的臺式機，基本的 PC 使用所需的一切都有。最後一次公開發布是在 2019 年 9 月。

Puppy Linux

Puppy Linux - around 300 MB download


下載：約 300MB

Linux 從來都不會讓事情變得簡單，Puppy Linux 也不例外。Puppy Linux 並不是一個發行版，而是有 11 個不同版本的 Puppy Linux。它不再像以前那樣小，1GB 的內存運行的比較好。儘管如此，它是最早的輕量級發行版之一。它最後一次更新是在 2019 年 3 月。

Raspberry Pi OS

Raspberry Pi OS - 400MB - 1.2GB


下載：400MB 至 1.2GB

Raspberry Pi OS（以前叫 Raspbian）是一個適用於樹莓派的 Debian 發行版。在這裡列出的所有發行版中，Raspberry Pi OS 是我日常使用的一個。我在三臺正在運行的樹莓派服務器上運行著它，另外還有幾臺暫時不活躍的服務器也加載著它。我不使用 Linux 圖形界面。相反，我連接到一個 OctoPi 網絡界面，可以讓我控制我的 3D 打印機。最近剛剛更新。

SliTaz

SliTaz - 50MB download


下載：50MB

SliTaz 是一款完全運行在內存中的 Linux，可以通過 CD-ROM 或 USB 盤啟動。它的體積很小，只有 50MB 左右，但具有強大的安全功能，可以只安裝基礎系統，或安裝一個精簡的 X 環境。最新的滾動發佈是在 2018 年。

SparkyLinux 基礎版

SparkyLinux base edition - 540MB download


下載：540MB

Sparky 是另一個基於 Debian 的發行版，它的設計是為了讓家庭用戶能有一個相對簡單的開箱即用的 Enlightenment 體驗，儘管它確實支持大約 20 個窗口管理器和桌面環境。它在 2019 年 11 月進行了更新。

Tiny Core Linux

Tiny Core Linux — 11MB download


下載：11MB

有三個版本，最小的是 11MB 的下載。在沒有圖形界面的情況下，Tiny Core Linux 的運行內存為 64MB。最後一次更新是在 2019 年 1 月。
