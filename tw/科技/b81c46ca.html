今天討論話題:::如何在String中插入新的一行字符？

2020-11-14
| 科技

大家請看這個代碼

String problemNotes = new String();problemNotes = "Date: "+childElement.getAttributeValue("date")+"\n"+"Submitter: "+childElement.getAttributeValue("submitter")+"\n"+"Notes: "+childElement.getText();

大家看到有換行 \n 如您所料。請參閱此修改後的代碼段。

import java.util.Date; public class Snippet { public static void main(String[] args) { String problemNotes = new String(); problemNotes = "Date: "+new Date() +"\n"+"Submitter: "+ "submitter" +"\n"+"Notes: "+"some notes"; System.out.println(problemNotes); }}

結果是這個

Date: Thu Jul 10 10:00:25 PDT 2018Submitter: submitterNotes: some notes

除非您將String寫入文本文件，並且您是Windows用戶。然後新行字符取決於您的操作系統（對於Unix為\ n，對於Windows為\ r \ n，對於舊Mac為\ r \ n），您應該使用：

String eol = System.getProperty("line.separator");problemNotes = "Date: " + childElement.getAttributeValue("date") + eol  + "Submitter: " + childElement.getAttributeValue("submitter") + eol  + "Notes: " + childElement.getText();

得出結論是這樣的

每個操作系統都有自己的“換行”字符。您應該使用

System.getProperty("line.separator")

自動獲取相應的行分隔符。
