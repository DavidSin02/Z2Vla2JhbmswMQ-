二、文件的使用

1、文件

I、文件是存儲在輔助存儲器上的一組數據序列，可 以包含任何數據內容。概念上，文件是數據的集合和抽象。文件包括兩種類型：文本文件和二進制文件。

2、文件的類型

I、文本文件一般由單一特定編碼的字符組成，如 UTF-8編碼，內容容易統一展示和閱讀。

II、二進制文件直接由比特0和比特1組成，文件內部數據的組織格式與文件用途有關。二進制是信息按照非字符但特定格式形成的文件，例如，png 格式的圖片文件、avi格式的視頻文件。

III、二進制文件和文本文件最主要的區別在於是否有統一的字符編碼。

IV、無論文件創建為文本文件或者二進制文件，都可以用 “文本文件方式”和“二進制文件方式”打開，但打開後的操作不同。

f = open("a.txt","rt") # t表示文本文件格式print(f.readline())f.close()

V、文本文件a.txt,採用二進制方式打開

f = open("a.txt","rb") # b表示二進制文件格式print(f.readline())f.close()

VI、採用文本方式讀入文件，文件經過編碼形式字符串，打印出有含義的字符；採用二進制方式打開文件，文件被解析為字節流。

3、文件的打開和關閉

I、Python對文本文件和二進制文件採用統一的操 作步驟，即“打開-操作-關閉”


II、Python通過open()函數打開一個文件，並返回一 個操作這個文件的變量，語法形式如下:

<變量名> = open(<文件路徑及文件名>,<打開模式>)


III、打開模式使用字符串方式表示，根據字符串定義， 單引號或者雙引號均可。上述打開模式中，'r'、 'w'、'x'、'b'可以和'b'、't'、'+'組合使用，形成既 表達讀寫又表達文件模式的方式。

IV、文件使用結束後要用close()方法關閉，釋放文件 的使用授權，語法形式如下:

<變量名>.close()

V、新建一個文本文件a.txt，其內容為“全國計算機等級考 試”，保存在目錄PATH中，假設此時路徑PATH是 Windows系統的D盤根目錄。打開並關閉該文件的操作 過程如下。

>>>PATH = "D:\\">>>f = open(PATH + "a.txt", "rt")>>>print(f.readline())國家計算機等級考試>>>f.close()

VI、根據打開方式不同，文件讀寫也會根據文本文件 或二進制打開方式有所不同。


①如果文件不大，可以一次性將文件內容讀入，保存到程 序內部變量中。f.read()是最常用的一次性讀入文件的函 數，其結果是一個字符串。

>>>f = open("D://b.txt", "r")>>>s = f.read()>>>print(s)新年都未有芳華，二月初驚見草芽。白雪卻嫌春色晚，故穿庭樹作飛花。>>>f.close()

②f.readlines()也是一次性讀入文件的函數，其結果是一個列表，每個元素是文件的一行。

>>>f = open("D://b.txt", "r")>>>ls = f.readlines()>>>print(ls)['新年都未有芳華，二月初驚見草芽。\n', '白雪卻嫌春色晚，故穿 庭樹作飛花。\n']>>>f.close()

③文件打開後，對文件的讀寫有一個讀取指針，當從文件 中讀入內容後，讀取指針將向前進，再次讀取的內容將 從指針的新位置開始。

>>>f = open("D://b.txt", "r")>>>s = f.read()>>>print(s)新年都未有芳華，二月初驚見草芽。 白雪卻嫌春色晚，故穿庭樹作飛花。>>>ls = f.readlines() >>>print(ls)[]>>>f.close()

結合讀取指針理解，上述代碼中ls返回值為空， 因為之前f.read()方法已經讀取了文件全部內容， 讀取指針在文件末尾，再次調用f.readlines()方 法已經無法從當前讀取指針讀入內容，因此返回 結果為空。

VII、f.seek()方法能夠移動讀取指針的位置，f.seek(0)將讀取 指針移動到文件開頭，f.seek(2)將讀取指針移動到文件 結尾。

>>>f = open("D://b.txt", "r")>>>s = f.read()>>>print(s)新年都未有芳華，二月初驚見草芽。 白雪卻嫌春色晚，故穿庭樹作飛花。>>>f.seek(0) # 將讀取指針重置到文件開頭>>>ls = f.readlines()>>>print(ls)['新年都未有芳華，二月初驚見草芽。\n', '白雪卻嫌春色晚，故穿庭樹作飛花。\n'] >>>f.close()

VIII、從文本文件中逐行讀入內容並進行處理是一個基本的文 件操作需求。文本文件可以看成是由行組成的組合類型， 因此，可以使用遍歷循環逐行遍歷文件，使用方法如下:




f.write(s)向文件寫入字符串s，每次寫入後，將會記錄一個寫入指 針。該方法可以反覆調用，將在寫入指針後分批寫入內容，直至文 件被關閉。

>>>f = open("D://c.txt", "w") >>>f.write('新年都未有芳華\n') >>>f.write('二月初驚見草芽\n') >>>f.write('白雪卻嫌春色晚\n') >>>f.write('故穿庭樹作飛花\n') >>>f.close()

上述語句運行後將在D盤目錄下生成一個文件c.txt，內 容如下

新年都未有芳華二月初驚見草芽白雪卻嫌春色晚故穿庭樹作飛花

使用f.write(s)時，要顯式的使用'\n'對寫入文本進行分 行，如果不進行分行，每次寫入的字符串會被連接起來。

f.writelines(lines)直接將列表類型的各元素連 接起來寫入文件f。

>>>ls = ['新年都未有芳華\n', '二月初驚見草芽\n','白雪卻嫌春色晚 \n','故穿庭樹作飛花\n']>>>f = open("D://c.txt", "w")>>>f.writelines(ls)>>>f.close()

三、數據組織的維度

1、數據組織的維度

一組數據在被計算機處理前需要進行一定的組織， 表明數據之間的基本關係和邏輯，進而形成“數 據的維度”。根據數據的關係不同，數據組織可 以分為:一維數據、二維數據和高維數據。

2、一維數據

一維數據由對等關係的有序或無序數據構成，採用線性方式組織，對應於數學中數組的概念。例 如:中國的直轄市列表即可表示為一維數據，一 維數據具有線性特點。


3、二維數據

I、二維數據，也稱表格數據，由關聯關係數據構成， 採用二維表格方式組織，對應於數學中的矩陣， 常見的表格都屬於二維數據

II、例如:國家統計局發佈的居民消費價格指數是二 維數據


4、高維數據

I、高維數據由鍵值對類型的數據構成，採用對象方式組織，可以多層嵌套。

II、高維數據在Web系統中十分常用，作為當今 Internet組織內容的主要方式，高位數據衍生出 HTML、XML、JSON等具體數據組織的語法結 構。



四、一維數據的處理

1、 一維數據的表示

I、一維數據是最簡單的數據組織類型，由於是線性結構，在Python語言中主要採用列表形式表示。 例如:中國的直轄市數據可以採用一個列表變量 表示。

>>>ls = ['北京', '上海', '天津', '重慶'] >>>print(ls)['北京', '上海', '天津', '重慶']

2、 一維數據的存儲

I、一維數據的文件存儲有多種方式，總體思路是採 用特殊字符分隔各數據。常用存儲方法包括4種。

(1)採用空格分隔元素，例如:

北京 上海 天津 重慶

(2)採用逗號分隔元素，例如:

北京,上海,天津,重慶

(3)採用換行分隔包括，例如:

北京

上海

天津

重慶

(4)其他特殊符號分隔，以分號分隔為例，例 如:

北京;上海;天津;重慶

II、逗號分割的存儲格式叫做CSV格式(Comma-Separated Values，即逗號分隔值)，它是一種通用的、相對簡單 的文件格式，在商業和科學上廣泛應用，大部分編輯器 都支持直接讀入或保存文件為CSV格式

III、一維數據保存成CSV格式後，各元素採用逗號分隔，形 成一行。從Python表示到數據存儲，需要將列表對象輸 出為CSV格式以及將CSV格式讀入成列表對象

ls = ['北京', '上海', '天津', '重慶'] f = open("city.csv", "w")f.write(",".join(ls)+ "\n")f.close()


3、 一維數據的處理

I、對一維數據進行處理首先需要從CSV格式文件讀 入一維數據，並將其表示為列表對象。f = open("city.csv", "r")ls = f.read().strip('\n').split(",") f.close()print(ls)>>>['北京', '上海', '天津', '重慶']

4、 二維數據的處理

二維數據由多條一維數據構成，可以看成是一維數據的組合形式。因此，二維數據可以採用二維 列表來表示，即列表的每個元素對應二維數據的一行，這個元素本身也是列表類型，其內部各元素對應這行中的各列值

ls = [['指標', '2014年', '2015年', '2016年'],['居民消費價格指數', '102', '101.4', '102'],['食品', '103.1', '102.3', '104.6'],['菸酒及用品', '994', '102.1', '101.5'],['衣著', '102.4', '102.7', '101.4'],['家庭設備用品', '101.2', '101', '100.5'],['醫療保健和個人用品', '101.3', '102', '101.1'],['交通和通信', '99.9', '98.3', '98.7'],['娛樂教育文化', '101.9', '101.4', '101.6'],['居住', '102', '100.7', '101.6'], ]

5、 二維數據的存儲

I、二維數據由一維數據組成，用CSV格式文件存儲。 CSV文件的每一行是一維數據，整個CSV文件是 一個二維數據。

II、二維列表對象輸出為CSV格式文件方法如下，採 用遍歷循環和字符串的join()方法相結合。

# ls代表二維列表，此處省略 f = open("cpi.csv", "w") for row in ls: f.write(",".join(row)+ "\n")f.close()

6、 二維數據的處理

對二維數據進行處理首先需要從CSV格式文件讀 入二維數據，並將其表示為二維列表對象。借鑑 一維數據讀取方法，從CSV文件讀入數據的方法 如下。

f = open("cpi.csv", "r")ls=[]for line in f: ls.append(line.strip('\n').split(","))f.close()print(ls)
