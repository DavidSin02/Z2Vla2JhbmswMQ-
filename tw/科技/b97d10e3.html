變量的輸出

變量如何輸入輸出呢？實際上，在這之前你已經使用過輸出語句（printf語句）了，我們可以使用printf來執行輸出。

printf語句的使用方法如下：

printf(格式控制字符串, 數據1, 數據2, 數據3.......);

格式控制字符串用特定的格式說明符代表輸出的數據類型，這裡的數據可以是變量、常量或者結果值具有合適類型的表達式。

之前看到的符號%d有什麼作用呢？如下這段代碼：

int a = 1;

int b = 2;

int c = 3;

printf("%d,%d,%d", a, b, c);

我們運行後在屏幕的輸出結果是： 1,2,3

你會發現，屏幕輸出的是a、b、c的值。哈！原來%d就是一個格式說明符，當這一行輸出時，其作用是指出輸出a、b、c的值。%d用於輸出int類型的值。事實上 每一種數據都有自己的說明符。後面在學到新的數據類型時，也會說到與之相應的說明符。

要特別注意的是，不要忘記給每一個想輸出的項目都使用一個格式說明符。

變量的輸入

scanf( )函數是C語言中最常用的一個輸入函數。scanf語句的使用方法：

scanf(格式說明字符串，變量地址1, 變量地址2, 變量地址3......);

scanf()函數接受你在鍵盤的輸入，把你在鍵盤輸入的文本轉成各種形式，如整數，浮點數，字符或者字符串。而printf( )是把各種形式的數據轉換成在屏幕上顯示的文本。也就是說，scanf()是printf( )的逆操作：

#include<stdio.h>  int main(void) {  int a; //定義一個int型變量a   scanf("%d", &a); //從鍵盤讀取一個數並賦值給a   printf("%d\n", a); //輸出變量a的值   return 0; }

在上面這段程序中，當編譯器運行到scanf語句時，程序會等待鍵盤的輸入，此時你可以輸入一個值，編譯器便會將其賦給變量a，然後通過printf語句在屏幕上輸出變量a的值。

scanf()函數用到指向變量的指針，我們會在第四章中學到關於指針的具體內容。現在我們只要知道，如果使用scanf()來讀取一個之前定義過的基本變量類型的值，記得在變量前加上&。
