python的常用擴展庫以及使用方式

2020-10-29
| 科技

一、常用庫（標準庫+擴展庫）

標準庫 說明

math 數學模塊

random 隨機數以及隨機化相關模塊

datetime 日期時間模塊

​collections 包含更多擴展性序列的模塊

functools 與函數以及函數式編程有關的模塊

tkinter 開發GUI程序的模塊

urllib 與網頁內容讀取以及網頁地址解析有關的模塊

擴展庫 場景

openpyxl 讀寫Excel文件

python-docx 讀寫Word文件

numpy 數組計算和矩陣計算

scipy 科學計算

pandas 數據分析

matplotlib 數據可視化或科學計算可視化

scrapy 爬蟲框架

shutil 系統運維

pyopengl 計算機圖形學編程

pygame 遊戲開發

sklearn 機器學習

tensorflow 深度學習

二、擴展庫安管理工具pip

pip命令實例 說明

pip freeze 列出已安裝模塊及其版本號

pip install package[==version] 在線安裝指定模塊（的指定版本）

pip install package.whl 離線安裝擴展模塊

pip install package1 package2 依次在線安裝模塊1、模塊2

pip install --upgrade package 升級模塊

pip uninstall package[==version] 卸載模塊

​如果計算上安裝多個版本的python或希望在虛擬環境中安裝模塊，最好切換至相應環境的scripts文件夾下再執行命令。

三、擴展庫導入

import 模塊名 [as 別名]

# 必須以“模塊名.對象名”（“別名.對象名”）的形式訪問對象。

import math #計算最大公約數。

math.gcd(2,4)

from 模塊名 import 對象名[as 別名]

#不需要使用模塊名作為前綴。

import math

math.gcd(2,4)

from 模塊名 import *

#一次性導入模塊所有對象，簡單粗暴，但一般不推薦使用。

from math import *

gcd(2,4)

sin(5)
