巧用力學分析解除一起飛控系統疑難故障

Relieving AProblem Of Flight Control System With Force Analysis


摘要：一直以來，波音公司都以其737NG飛機的飛控系統設計為榮，因多年的實踐證明其故障率極低，這也導致了相關領域的故障處理經驗較少。在面對一次飛控疑難故障時，發現了機隊中也存在不少類似的異常現象，最終，經過多方工程師的合力研討，巧用物理力學分析方法使故障得以排除。

關鍵詞：力學分析；飛控系統；故障

Keywords：ForceAnalysis；Flight Control System；Problem

0 背景

某日航後一架波音737-800飛機落地後，機組反映，機長側駕駛盤順時針旋轉至接近止動位時有異響。機務測試檢查發現，機長側駕駛盤轉動接近極限位時，有明顯的卡阻並伴有輕微類似鋼索拉緊的聲音；但是在操縱副駕側駕駛盤到左或右極限位過程中，在機長和副駕側座位處均未發現此異響。對比其他幾架飛機，都存在同樣的現象。

1 故障處理過程

波音737NG飛機飛控系統從設計上而言，可靠性非常高，多年以來涉及到飛控系統的故障極少，這也是波音公司對737NG飛機在經過多次改裝升級之後依然對飛控系統沒有較大改動的原因所在，所以，對於相關的故障在廠家提供的故障隔離手冊中並沒有給出相應的具體處理方案，工程師們只能按照經驗嘗試提出解決辦法。

工程師先檢查了機長側駕駛盤下部副翼控制鼓輪無異常，傳動機構正常無卡阻。在轉動駕駛盤過程中發現駕駛盤與駕駛杆上端連接處齒輪與襯套組件附近聲音較大，潤滑齒輪及花鍵；檢查駕駛盤導線纏繞正常，導線無磨損，無卡阻。恢復後，操作測試發現異常情況依舊存在。

檢查前電子艙內兩側駕駛杆同步鋼索正常無缺陷，張力正常。測量副翼機身段鋼索張力，溫度6℃，AA=135，AB=102，均在範圍值內。按手冊測試校裝孔：A/S-1、A/S-1A、A/S-15、A/S-3和A/S-4，均符合標準。檢查主輪艙內副翼控制機構，定中機構，未發現轉軸生鏽，卡滯等異常情況。

同時，在此期間檢查了多架波音737NG飛機，也有部分飛機存在同樣的問題，到此，故障還是沒有得以排除。

之後，將該情況向波音反映，尋求廠家支援。持續了一個星期，波音回覆未收到過類似故障報告，建議分解駕駛杆內部結構，進行詳細檢查檢查。

在此期間，工程師們組織了多方研討，結合駕駛盤的操作原理以及力的傳遞和力的作用，採用簡單的力學分析，最終發現了故障點，故障點恰好就是力的作用點，對故障點進行處理後，故障才得以排除。

2 故障處理分析

2.1 簡單的力學原理

牛頓力學提到，一切物體在沒有受到外力作用或所受合力為零時，物體將保持一個穩定的狀態，即總保持靜止狀態或勻速直線運動狀態；要使物體改變運動狀態，則必然要有外力或合力不為零的情況存在。

當外力作用在物體上時，不管物體形狀、大小如何，總有一個力的作用點，且所有作用在該物體上的力，包括正向力、摩擦阻力等等，均集中在這個力的作用點上並形成一個合力，當合力為零時，物體將保持穩定狀態；當合力不為零時，物體將以力的作用點為中心沿著合力的方向有一個相對運動的趨勢。

同時，力可以在介質中傳遞，可以從一個物體傳遞到另一個物體。

2.2 故障分析


波音737NG飛機位於駕駛艙用於控制橫滾和俯仰操作的控制機構主要由駕駛盤和駕駛杆組成，駕駛盤位於駕駛杆之上，機長側和副駕側各一套，如圖1所示。

駕駛員通過順時針和逆時針轉動駕駛盤控制副翼的作動，來實現飛機的橫滾操作；通過推拉駕駛杆操作升降舵，以實現飛機的俯仰操作。

當轉動機長側駕駛盤時，轉動駕駛盤的力會通過位於駕駛杆內的輸入軸傳到位於駕駛杆下部的副翼控制鼓輪（aileron bus drum）；轉動副駕側駕駛盤時，轉動駕駛盤的力會通過位於駕駛杆內的輸入軸傳到位於駕駛杆下部的副翼控制同步鼓輪（bus drum）和同步鋼索,副翼控制同步鼓輪轉動，同步鋼索受力，使得輸入的力通過副翼控制同步鼓輪和同步鋼索傳到機長側的副翼控制鼓輪。作用在副翼控制鼓輪上的力再讓副翼控制鼓輪轉動，將力傳遞到位於機身左側的副翼控制鋼索，通過鋼索和控制連桿將力傳送到副翼PCU（power control units），PCU再作動相應的機構使副翼作動，以實現橫滾控制。


安裝在駕駛杆底部的一套副翼控制同步鼓輪和同步鋼索，在轉動機長側駕駛盤實現對副翼的操作時，通過副翼控制同步鼓輪和同步鋼索將機長側駕駛盤的扭轉力傳到副駕側的副翼控制同步鼓輪，使得此時副駕側的駕駛盤可以隨動。當轉動副駕側駕駛盤時，在正常操作中，是通過副駕側副翼控制同步鼓輪和同步鋼索帶動機長側副翼控制同步鼓輪和副翼控制鼓輪轉動，實現對副翼操縱的同時，也使得機長側駕駛盤隨動。


當推拉機長側駕駛杆時，駕駛杆會移動升降舵前扇形盤和升降舵控制鋼索，將力通過鋼索和連桿傳遞到升降舵PCU（power control units），升降舵PCU再作動相應的部件以實現俯仰操作。與此同時，因機長側駕駛杆與副駕側駕駛杆底部通過前輸入扭力管（forward input torque tube）剛性連接，如圖3，則在推拉機長側駕駛杆時，副駕側駕駛杆也會隨動。反之，推拉副駕側駕駛杆時，效果亦然。

因此，通過以上分析，駕駛杆底部連接處在橫滾控制時可以實現水平方向的轉動以帶動相應的鼓輪轉動；在俯仰控制時，可以實現垂直方向的轉動，則必然存在一個萬向接頭機構，各方向的轉動均繞著這個萬向接頭機構為中心得以實現。同時，該萬向接頭機構，也是駕駛杆推拉力和駕駛盤扭轉力的集中點和傳播點，即力的作用點。


通過手冊分析對比，駕駛杆下部確實存在這樣一個萬向接頭機構，如圖4，主要由位於駕駛杆最下段扁平端的一個接頭平臺（PLATE）和叉型的接頭（FORK）以及相應的軸承將駕駛杆與控制鼓輪組件的軸連接，通過這些部件的配合實現多方向轉動。當轉動駕駛盤時，力在這裡集中並從這裡傳遞。下面來分析，當轉動駕駛盤時，位於機長側此處的受力情況。

當轉動機長側的駕駛盤時，轉動駕駛盤的外力通過駕駛杆內的輸入軸向下傳到該處，輸入軸轉動接頭平臺，接頭平臺帶動叉型接頭將扭轉力傳到副翼控制鼓輪和副翼控制同步鼓輪（兩個鼓輪同軸連接）並帶動鼓輪轉動。此時，接頭平臺和叉型接頭所組成的萬向接頭即為力的作用點和傳遞點，定義為O點。轉動機長側駕駛盤，從機長側駕駛盤傳遞到O點的力定義為F1；同時，副翼控制鼓輪的轉動帶動副翼控制鋼索和連桿實現副翼系統作動，則會有一個系統阻力作用在O點上，定義為F2；副翼控制同步鼓輪轉動，使得副駕側駕駛盤隨動，則有一個來自副駕側駕駛盤的阻力作用在O點上，定義為F3。機長側駕駛盤實現轉動，會有F1=F2+F3。

當轉動副駕側駕駛盤，副駕側副翼控制同步鼓輪帶動機長側副翼控制同步鼓輪和副翼控制鼓輪轉動，副翼控制同步鼓輪的轉動在O點處帶動機長側駕駛杆內的輸入軸轉動，以實現機長側駕駛盤的隨動，也就是說，此時O點需要克服機長側駕駛盤的自身阻力，即此時有一個來自機長側駕駛盤的阻力F4作用在O點上，只需要克服這個力，機長側的駕駛盤就會隨動，所以，在該狀態下，O點有且只有一個F4的存在。

簡單地說，當轉動副駕側駕駛盤時，只有F4一個力作用在機長側的O點上；而轉動機長側駕駛盤時，有F1、F2、F3三個來自三個方向的力作用在機長側的O點上。

通過以上分析，當轉動機長側駕駛盤和轉動副駕側駕駛盤時，位於機長側駕駛杆下部的萬向接頭機構受力情況不一樣。結合背景事件，轉動機長側駕駛盤到一定角度時有卡阻並伴有異響，而轉動副駕側駕駛盤時則不存在該異常現象。由此推論，是否該萬向接頭機構因不同的受力情況所產生的摩擦不同而導致了故障的出現。


工程師從飛機前電子設備艙接近機長側的萬向接頭機構，目視發現在接頭平臺和叉型接頭之間積聚了大量的灰塵，使如圖5所示。

初步判斷，正常情況轉動機長側駕駛盤時此處所受到的阻力較大，再加上此時灰塵的大量聚集，極大地增加了該點的活動阻力從而增加了摩擦，由此導致了卡阻和異響；而轉動副駕側駕駛盤時，此處只受到機長側駕駛盤的阻力，阻力相對較小，摩擦較小，沒有產生異響。

工程師使用潤滑劑對該部位清潔潤滑後，轉動機長側駕駛盤，卡阻和異響消失。同時，工程師對之前發現有類似現象的幾架飛機進行同樣的處理，操作測試，異常情況均消失，故障得以排除。

3 經驗總結

遇到傳送機構異響，凡是能從駕駛艙聽到異響時，一般首先考慮從駕駛盤及地板下部控制機構入手，主輪艙部件應放在後續再考慮，因主輪艙距離駕駛艙較遠，主輪艙部件卡阻產生的輕微異響不會如此明顯地傳到駕駛艙區域。

在排故中要把握好關鍵信息，對出現的差異信息進行對比，針對轉動機長和副駕側駕駛盤的現象不一致切中要害。

機械傳力機構的排故，在嚴格依據手冊施工的同時，可以借用物理力學分析方法，兩者巧妙地結合經常會收穫事半功倍的效果。

鑑於排故過程中發現該故障的普遍存在，且在波音的維修方案中並沒有對相關部件的檢查和潤滑方案；同時，在日常航線維護中該部件較難接近，只能採用噴霧的形式進行清潔和潤滑，此手段只是短時間的緩解措施。

查詢維護方案，未發現關於萬向節Plate和 Fork連接處的定期檢查或者相關維護程序，航司已經建議波音出臺相關標準文件去針對這個故障，並建議將清潔和潤滑工作加入到定期的檢查維護項目。

在檢查萬向節Plate和 Fork的連接處時，發現連接處存在積塵、汙染，而且位置難以接近，只能使用噴霧潤滑劑清潔，但是這樣的處理只能作為臨時處理方案。可靠性部門之後會就此類問題展開普查和調研，後期在必要時提出相關維護方案。
