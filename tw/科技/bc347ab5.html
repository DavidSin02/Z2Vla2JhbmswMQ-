建站經驗豐富設置是否顯示商務通對話框

2020-11-14
| 科技

企業網站的策劃運營者們經常會遇到這樣的問題，有時候需要將商務通彈出的對話框顯示，有時候需要把它隱藏

比方有活動時顯示對話框，活動過期時將對話框隱藏掉。這個需求比較棘手，需要有懂技術的人員長期手動維護js腳本才行

今天專門做網站的兼職小哥教您一招，您使用這個方法，即使是不懂技術的營銷策劃人員也可以輕易維護網站，設置對話框是否顯示了。

var swtID=setInterval("update_swt()", 100);

function update_swt() {

if (typeof(LR_UserInviteDiv) != "undefined") {

LR_UserInviteDiv="<div style=\"position:relative;\"><a href=\"javascript:void(0);\" onclick=\"openZoosUrl();LR_HideInvite();return false;\"><img src=\""+imagepath+"meirongSwt.png\" width=\"422\" height=\"220\" border=\"0\" /></a><a style=\"position:absolute;top:0px;right:0px;z-index:999;width:30px;height:30px;\" href=\"javascript:void(0);\" onclick=\"LR_RefuseChat();LR_HideInvite();return false;\"></a></div>";

clearInterval(swtID);

}

}

以上代碼設置是邀請框的顯示時間，間隔多少秒彈出來。最上面is_show變量我們加了一個變量，我們正是要用這個變量控制對話商務通對話框是否顯示彈出對話框。您一定看出來了，這是織夢後臺，現在80%以上的企業網站都是基於織夢開發的。用織夢後臺做網站的時候，建議選一個好的主機，阿里雲主機是不錯的選擇，然後注意一些安全選項，織夢安全建站方面的注意事項，我會在稍後分享給大家。

變量添加完之後，打開網站，底部的模板文件footer.htm，給js變量賦值

然後當你從網站後臺修改完是否顯示對話框的選項之後，需要在網站後臺重新批量生成一下靜態頁面，使變量生效。

找我做這個網站的朋友還有一個需求就是想防止別人複製粘貼他網站上的內容，禁用鼠標右鍵，禁止ctrl+c,ctrl+v的複製粘貼，這個需求用jquery實現很簡單，只需要以下幾行代碼
