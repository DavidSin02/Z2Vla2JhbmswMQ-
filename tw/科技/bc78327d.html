零基礎學C語言——運算符

2020-11-14
| 科技

這是一個C語言系列文章，如果是初學者的話，建議先行閱讀之前的文章。筆者也會按照章節順序發佈。

上一篇的最後，我們在說到強制類型轉換時，給出了一個例子：

int a = 5.0 / 2;

這裡其實涉及到了運算符，即 /。

在編程語言的世界裡並不是只有加減乘除的四則運算，還包含了很多其他運算，參見下圖：

運算符表

優先級

舉個簡單的例子：

1 + 2 * 3 = 7 而不會計算成9

這裡其實就是因為在我們小學數學中，乘除運算的優先級高於加減運算，因此優先計算乘除，然後計算加減。

同理，運算符表中，優先級從上到下依次遞減，換言之，越靠上的運算符，優先級越高。例如：

(1 + 2) * 3 = 9 而不會計算成7

因為1+2在括號中，而小括號在C語言中也是一種運算符，且其優先級最高。

結合方向

有的讀者會問，同級別的運算符如何計算呢？這就要看運算符結合方向了。例如：

2 * 10 / 4 % 2

其中%為求餘運算，即計算除法後的餘數，也稱其為取模運算。上面這個式子的結果是1。如何計算呢？

由於乘、除、求餘三個運算都是同級別，且他們的結合方向都是從左到右的，因此我們從左向右依次計算即可。

X目運算符

下面我們簡單介紹一下何為：單目運算符、雙目運算符、三目運算符。

其中，雙目運算符是最容易被理解的，例如加法：

1 + 2

在加號前後個需要一個值方可使用加法，即加法運算需要兩個數值，數值的個數也就是目數，換句話說，加法就是雙目運算符。

那麼，單目運算符就比較容易理解了，即運算符只需要一個數值即可完成運算，例如：

-1

其中-為負號運算符，它是單目運算符，需要一個數值來進行運算，在這個例子中，這個數值就是1。而運算符配合運算所需要的數據計算後的結果是-1。

而三目運算符，就是說運算需要三個數據來完成，後面我們會對每一個運算符舉出一個例子。

各個運算符舉例

下面展示各種運算符的使用方式，很多運算符初學者可能不理解其含義，暫且不要糾結，後面的文章還會涉及。

  • []數組下標
int a = arr[0]; //從數組arr中取其第一個元素（下標從0開始）
  • ()圓括號
int a = (1 + 2) * 3; //a = 9
  • . 成員選擇
int a = person.name; //從person結構體對象中取name成員的值
  • -> 成員選擇（指針）
int a = person->name; //從person結構體指針所指向的對象中取name成員的值
  • - 負號運算符
int a = -1;
  • (類型) 強制類型轉換
int a = (int)5.2; //a = 5
  • ++ 自增運算符
int a = 1;a++; // 變量a的值為2，但是a++表達式的值為1，表達式將在後面介紹，初學者可以先不糾結++a; // 變量a的值為2，且++a表達式的值為2
  • -- 自減運算符
int a = 1;a--; // a的值為0，但a--表達式的值為1++a；// a的值為0，且--a表達是的值為0
  • * 取值運算符
int a = 0, *p = &a;*p = 1; //向指針p所指向的內存地址中寫入整數1
  • & 取地址運算符
int a = 0;int *p = &a; //指針p等於變量a的內存地址
  • ! 邏輯非運算符
int a = !1; //a = 0，任何非0值的!運算後的值都是0
  • ~ 按位取反運算符
unsigned int a = ~0; //a = 0xFFFFFFFF 無符號整型佔4字節，即32位，～對每一位取反，0的每一位都是0，取反後每一位都是1
  • sizeof 長度運算符
int a = sizeof(int); //a = 4，sizeof將計算給定的類型或者對象所佔的字節數int b = sizeof(a); //b = 4，因為a是整型
  • / 除
int a = 10 / 2;
  • * 乘
int a = 3 * 2;
  • % 餘數（取模）
int a = 9 % 2
  • + 加
int a = 1 + 2
  • - 減
int a = 10 - 9
  • << 左移
int a = 1 << 1; //a = 2, 1默認為有符號整型，即其低字節比特形式為 0000 0001，整個32位向左移1位，低字節比特變為 0000 0010，移入部分默認補0
  • >> 右移
int a = 1 >> 1; //a = 0, 1默認為有符號整型，即其低字節比特形式為 0000 0001，整個32位向右移1位，低字節比特變為 0000 0000，移入部分默認補0
  • > 大於
int a = 1 > 2; //a = 0, 1 > 2是一個真值邏輯判斷，很顯然這個條件不成立，不成立時值為0
  • >= 大於等於
int a = 1 >= 2; //a = 0, 1 >= 2是一個真值邏輯判斷，很顯然這個條件不成立，不成立時值為0
  • < 小於
int a = 2 < 1; //a = 0, 2 < 1是一個真值邏輯判斷，很顯然這個條件不成立，不成立時值為0
  • <= 小於等於
int a = 2 <= 1; //a = 0, 2 <= 1是一個真值邏輯判斷，很顯然這個條件不成立，不成立時值為0
  • == 等於
int a = 2 == 1; //a = 0, 2 == 1是一個真值邏輯判斷，很顯然1不等於2，不成立時值為0
  • != 不等於
int a = 2 != 1; //a = 1, 2 == 1是一個真值邏輯判斷，很顯然1不等於2條件成立，成立時值為1
  • & 按位與
int a = 3 & 2; //a = 2， 3的低字節二進制為0000 0011，2的低字節二進制為0000 0010，與運算只有對應位都為1時結果為1，其餘為0，所以低字節與運算結果位 0000 0010
  • ^ 按位異或
int a = 3 ^ 2; //a = 1， 異或運算是兩數對應位數值不相同則運算後該位值為1，相同為0
  • | 按位或
int a = 1 | 2; //a = 3，或運算是兩數對應位數值只要有一個為1則運算後該位值為1，都為0則為0
  • && 邏輯與
int a = 0 && 1; //a = 0, &&前後均為真值判斷，且當左側真值判斷為假時，不再進行右側真值判斷，0在真值判斷時含義為假，而假對應的數值為0，因此a = 0
  • || 邏輯或
int a = 0, b = 2;int c = a || b; //c = 1, ||前後均為真值判斷，且當左側真值判斷為真時，不再進行右側真值判斷，2在真值判斷時含義為真，而真對應的數值為1，因此c = 1而不是2
  • ?: 條件運算符
int a = 1 > 2 ? -1 : 3; //a = 3//條件運算符的左右時先進行?左側的真值判斷，如果為真則計算?與:之間的值，如果為假則計算:右側的值int b = 1 > 2 ? 3 - 1 : 2 + 6; //b = 8
  • 賦值運算符
int a = 1; //其中的=就是賦值運算符
  • 除後賦值運算符
int a = 10;a /= 2; //a = 5, 運算符等價於 a = a / 2，原本a=10，除以2後等於5
  • 乘後賦值
int a = 2;a *= 3; //a = 6, 等價於a = a * 3
  • 取模後賦值
int a = 9;a %= 2; //a = 1, 等價於a = a % 2
  • 加後賦值
int a = 1;a += 1; //a = 2, 等價於 a = a + 1
  • 減後賦值
int a = 5;a -= 2; //a = 3, 等價於 a = a - 2
  • 左移後賦值
int a = 1;a <<= 1; //a = 2, 等價於 a = a << 1
  • 右移後賦值
int a = 3;a >>= 1; //a = 1, 等價於 a = a >> 1
  • 按位與後賦值
int a = 3;a &= 1; //a = 1, 等價於 a = a & 1
  • 按位異或後賦值
int a = 3;a ^= 1; //a = 2, 等價於 a = a ^ 1
  • 按位或後賦值
int a = 1a |= 2; //a = 3, 等價於 a = a | 2
  • 逗號運算符
int a = (1,2,3); //a = 3, 逗號運算符的值是最右側逗號右側的值//這裡之所以加()，是因為賦值表達式優先級高於逗號表達式，會導致a=1作為逗號運算符的一部分//我們將在表達式的文章中再次介紹這個運算符
