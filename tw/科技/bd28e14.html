榮耀30系列首搭Magic UI 4.0，可升鴻蒙2.0

2020-10-29
| 科技

在今日舉行的華為2020開發者大會上，華為消費者業務軟件部總裁王成錄在華為開發者大會上發表了《連接無限可能-全場景終端軟件發佈》主題演講，EMUI 11和Magic UI 4.0將同步開啟內測，Magic UI 4.0計劃將於9月中旬開啟內測招募活動，榮耀30系列和榮耀V30系列成為首批搭載機型，更多機型在後續升級計劃中，而支持EMUI 11和Magic UI 4.0的機型未來都可以升級到鴻蒙操作系統2.0。


技術共享同源，榮耀創新引領產業

眾所周知，脫胎於華為的榮耀，一直與華為保持著技術的共享與同步更新。特別是在軟件系統方面，從榮耀Magic UI發佈開始，榮耀每一代產品所搭載的系統都與華為最新系統同步更新。EMUI 11和Magic UI 4.0同步開啟內測正是如此。

值得一提的是，此次備受大家關注的Magic UI 4.0在分佈式技術、UX設計、隱私安全保護、AI智慧等方面都有重大升級。

最新發布的Magic UI 4.0，由於採用分佈式技術，打破了設備間、應用間的隔離，將帶來更多全場景業務體驗。具體表現在不同設備之間的無縫連接和體驗，像在多屏協同時，實現隱私投屏、跨屏拖拽等；在暢連通話中，實現與智慧屏進行擬人化交互，手機可以通過暢連消息對智慧屏進行互動和控制等。

在UX設計上，Magic UI 4.0不僅帶來更多藝術風格的主題和DIY的AOD，並且繼續將人因研究深入用到近百個用戶場景上，特別是在動效和多感官協同上，為用戶帶來全新視聽觸交互體驗。

而Magic UI 4.0還支持AI字幕（直出中文字幕+外語轉中文字幕）和系統級翻譯（瀏覽外文文章和圖片輕鬆讀懂內容）功能，大大提升了用戶的體驗。

除了軟件系統外，榮耀還與華為保持技術同源，在手機的研發生產上與華為使用同一套技術標準，堅持以自研為第一推動力，帶來各種極致黑科技。

以智能手機用戶的痛點需求拍照為例，從2014年榮耀6 Plus推出世界首個雙攝拍照系統，讓手機能拍出背景虛化的大光圈照片；到2017年底，榮耀率先將AI技術應用於榮耀V10、榮耀10上，針對不同場景中的不同元素進行最優的色彩和光影優化；再到2019年榮耀V30系列引入聯合SONY共同開發的RYYB顏色濾波器，以及通過榮耀30系列確立的超大底、超大像素、超感光、超長焦四位一體的系統級影像技術發展方向，榮耀已經完成從手機攝影發展的第一階段的拿來主義、第二階段的關鍵技術自研但仍需供應鏈技術反哺躍升到了對光學和算法全面系統的底層研究，旨在構築一體化攝影系統的第三階段，並將繼續引領手機攝影的技術與應用趨勢。

然而，除了手機的影像技術，榮耀早在 2016 年就推出一款主打人工智能體驗的手機——榮耀Magic，給現在的手機 AI 應用場景奠定了方向。在手機散熱方面，榮耀更是業界首創了“THE NINE 液冷散熱技術”，第一次在手機上實現了 PC 級的散熱水平。

當然，強大的技術實力還體現在方方面面，比如業內公認的自研麒麟芯片、GPU Turbo技術、方舟編譯器、超級藍牙、WiFi6+、40W超級快充、系統級影像技術等，使榮耀在市場極具競爭力，佔據行業第一梯隊席位，同時使得手機用戶享受到最新技術帶來的新鮮體驗。

種種過往的事實證明，榮耀在與華為技術同源共享的同時，已經打造出屬於自己的技術創新核心競爭力，並不斷引領行業的發展。

雙品牌戰略，榮耀持續領先市場

除了技術同源和共享外，華為、榮耀還堅持雙品牌戰略，其中互補性和差異性讓雙品牌穩步發展的同時，又彰顯出不同的個性和發展方式。例如榮耀的品牌定位是年輕人的科技潮牌，將“年輕”、“潮流”融入品牌和產品中；華為的品牌定位則是全球標誌性的科技品牌。

以具體的產品榮耀30系列為例，雖然其和華為P40系列，都是主打潮流、攝影，採用相同的麒麟芯片，硬件配置幾乎相同，但榮耀30的設計語言更加大膽，時尚、張揚和自信，除了引領潮流的鈦空銀和霓影紫配色，更是首次將榮耀HONOR大Logo放大，排列在相機矩陣的延長線上，辨識度極高。

此外，從榮耀30系列發佈會“潮流大秀”的形式，同樣體現出榮耀的科技潮牌定位。不同於業內多見的中規中矩的發佈形式，榮耀將30系列發佈會變成潮流秀，把手機的概念從工具延伸成時尚個性單品。

提及潮流，其實早在2017年，榮耀9就首創15層工藝3D全面激光玻璃，使得玻璃機身在光線下折射出金屬光澤版的炫光，搭配奶奶灰配色，並藉此引爆了年輕消費者。而2018年榮耀10首創的變色極光鍍膜，利用光程差的原理，當光線直射在機身，不同的角度，不同的光線，都會呈現出不同的獨特色彩，更是以引領潮流驚豔年輕用戶。

正是由於雙品牌戰略，讓榮耀和華為佔據國內市場近半份額，尤其是在國內，甚至國際形勢的影響，整個手機市場都處於一個相對低迷的環境之下，華為、榮耀反而是持續呈現逆勢增長，例如今年發佈的榮耀30系列，自發布以來一直是暢銷爆款，首銷線上就達到全平臺銷售額1分鐘破3億的好成績；6.18期間，榮耀30斬獲6.1-6.18京東平臺手機累計安卓手機銷售冠軍實現6連冠，助力榮耀手機連續7天蟬聯全平臺累計手機銷量冠軍。

總結：榮耀依託技術溢出和潮流設計在各檔位中佈局打造最高價值產品，憑藉與華為技術同源的先天優勢，逐步打造出屬於自己的技術創新核心競爭力， 同時又在做著“年輕人最感興趣的事兒”，成功吸引轉化大量其他手機品牌粉絲和大批追逐科技潮牌的年輕用戶，用出色實力不斷撬動年輕用戶市場，期待榮耀未來帶來的更多驚喜。
