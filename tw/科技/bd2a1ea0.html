21世紀最暢銷的F-16戰機 機背保形油箱是標配 戰力不輸歐洲四代半

2020-11-14
| 科技

對於很多中國的軍迷而言，F-16是一款經典第四代戰鬥機，這款戰機自從1978年裝備美國空軍後，就成為全球多個國家和地區空軍關注的焦點。從上世紀80年代初開始出口到歐洲多國，隨後幾十年在全球擴散。目前總產量接近5000架，裝備全球30多個國家和地區，今天我們重點聊聊其中一個型號。

進入21世紀後，美國推出了當時最強的F-16E/F戰機，也在就是大家常說的F-16E/F BlocK 60多用途戰鬥機。這款專為阿聯酋國空軍開發，與傳統型號的區別是：換裝了當時最新的AN/APG-80有源相控陣(AESA)雷達、全新航空電子設備、 保形油箱(CFT)，全新的通用電氣F110-GE-132發動機。 總數一共80架，花費64億美元。

以F-16E/F戰機的性能，與歐洲三款四代半戰機相比，其多用途作戰能力也不落後。外界很多人以為，2000年後，這款戰機也會像其他F-16戰機一樣大賣。然而最終這款戰機只生產了80架就停產。讓人想不到的是，F-16C/D的另外一款F-16C/D Block 52+卻成為近20年最暢銷的F-16戰機。本文所有配圖均為波蘭空軍F-16C/D Block 52戰機，所有圖片均為航空新視野原創攝影。

F-16C/D Block 52戰機作為新一代的戰鬥機與其他型號相比有很多特點。首先是裝備的國家多，例如希臘，波蘭，埃及，以色列，新加坡，伊拉克，摩洛哥，印度尼西亞，巴基斯坦等國均裝備大量的F-16C/D Block 52戰機，或者一些性能相當於F-16C/D Block 52戰機的F-16改進型號。

F-16C/D Block 52戰機，也是目前裝配機背保形油箱最多的F-16戰鬥機。大家有興趣注意觀察以上幾個國家，很多時候保形油箱都是標配。這一定程度上影響了戰機的機動性能。但是美國空軍自用的F-16C/D Block 52戰機並沒有見到安裝保形油箱的照片。這應該跟美國空軍自身強大的空中加油能力有關。與這款戰機性能相似的F-16C/D Block 50戰機也很少見到這種情況。

F-16C/D Block 52戰機均採用當時普惠公司最新的F100-PW-229IPE發動機。這臺發動機最大加力推力達到129千牛。在經濟性，可靠性以及壽命這些關鍵指標，至今依然是一個很高的水平。而傳統的F-16C/D Block 50戰機，採用的是通用電氣F110-GE-129發動機，量大加力推力達到131千牛，例如駐日美軍的F-16CM/DM戰機和自衛隊的F-2戰機均採用這一等級的發動機。

關於這兩型發動機的外觀區別，網絡上有很多對比的照片。筆者曾經發過很多F-16CM/DM和F-2戰機的高清照片，有興趣的看看那些戰鬥機的發動機噴口就明白了。這也是分辨這兩個批次F-16戰機最大的外觀區別。

在航電上，F-16C/D Block 52戰機最初使用的是美製的APG-8(V)5 多普勒雷達。近幾年很多F-16C/D Block 52戰機均升級到F-16C/D Block 52+戰機的標準。也就是雷達也升級到最新的APG-8(V)10 。據稱新的升級可以保證戰機的空對空搜索範圍提高到1.5倍以上。同時對地攻擊能力也有很大的提高。雖然這款雷達並非有源相控陣雷達，但是依然不可輕視。

通過F-16C/D Block 52戰機的改進，可以看到F-16戰機雖然是一款很老的第四代戰鬥機，但是在設計之初就採取模塊化的設計，特別是航電系統方面。另外戰機的機載武器，也經過很多的升級，以對空導彈為例，從早期的AIM-9L近距空空導彈，到現役最新的AIM-9X和AIM-120C空空導彈。其空戰能力都得到了顯著的提高。這些較新批次的F-16戰機，面對颱風，陣風單機性能，誰強誰弱，誰又說得清楚呢？

目前多個國家和地區，計劃把現役的F-16C/D戰機，甚至還有一些F-16A/B戰機均升級到F-16V的標準，主要的升級是採用全新的AN/APG-83有源相控陣雷達。新的任務計算機和電子戰套件。全新的玻璃化座艙......這些升級是否給我國數百架早期的殲-10A和殲-10S戰機帶來什麼啟示呢？我國殲-10B/C的未來是否又有更強的升級呢？我們拭目以待。
