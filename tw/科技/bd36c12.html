Android BroadcastReceiver的基本使用，撥號，短信，SD卡，開機

2020-10-29
| 科技

Android中四大組件的使用時重中之重，我這個階段也不奢望能把他所有的原理搞懂，但是最起碼的，我要把他的各種使用方法瞭如指掌才行

BroadcastReceiver

接收系統的廣播，比如電話，短信之類的

IP撥號器

我們在撥打電話的時候，我們系統也會事先發送一個廣播，所以我們可以用廣播接收者來接收到這個廣播撥打電話的時候在電話號碼前面加上一些優惠的長途短號，其邏輯就是入門使用廣播的一個小案例，那我們新建一個IPCall項目

我們撥打電話的時候系統發一個廣播，我們接受到這個廣播拿到號碼修改之後再放回去，就達到了我們IP撥號的目的，我們需要新建一個Receiver

CallReceiver

package com.lgl.ipcall;

import android.content.BroadcastReceiver;

import android.content.Context;

import android.content.Intent;

import android.util.Log;

/**

* 電話廣播

* Created by lgl on 16/4/10.

*/

public class CallReceiver extends BroadcastReceiver {

//接收到廣播時調用

@Override

public void onReceive(Context context, Intent intent) {

Log.i("CallReceiver", "打電話");

}

}

這個類暫時不需要做什麼。最重要的還是註冊，作為四大組建，他是需要在清單文件裡註冊的

<!--註冊廣播-->

<receiver android:name=".CallReceiver">

<intent-filter>

<!--定義接收的廣播-->

<action android:name="android.intent.action.NEW_OUTGOING_CALL" />

</intent-filter>

</receiver>

別忘了，你監聽了用戶的隱私，是需要權限的

```

<uses-permission android:name="android.permission.PROCESS_OUTGOING_CALLS" />

我們先運行一下，打個電話

好的，廣播我們已經抓住了，那我們可以首先獲取打出去的號碼

String iphone = getResultData();

Log.i("CallReceiver", "電話號碼："+iphone);

這樣我們就可以拿到String類型的電話號碼了

我們現在可以修改號碼重新設置IP段了

String newPhone = "+86" + iphone;

//把修改後的號碼放回去

這樣我們每次打電話他都會自動幫我們添加一個字段了

完整代碼

package com.lgl.ipcall;

import android.content.BroadcastReceiver;

import android.content.Context;

import android.content.Intent;

import android.util.Log;

/**

* 電話廣播

* Created by lgl on 16/4/10.

*/

public class CallReceiver extends BroadcastReceiver {

//接收到廣播時調用

@Override

public void onReceive(Context context, Intent intent) {

Log.i("CallReceiver", "打電話");

String iphone = getResultData();

Log.i("CallReceiver", "電話號碼：" + iphone);

//添加IP段

String newPhone = "+86" + iphone;

//把修改後的號碼放回去

setResultData(newPhone);

}

}

2.短信攔截器

系統受到了一條短信後，也是會發一條廣播的，所有我們可以在中間寫一個廣播接受者去進行我們的操作，這裡，我們繼續在IP撥號器這個項目中寫吧，就不新建項目了，不然等下上傳也麻煩

我們新建一個Class——SMSReceiver，重寫他的onReceive方法，然後我們先註冊

<!--註冊廣播-->

<receiver android:name=".SMSReceiver">

<intent-filter>

<!--定義接收的廣播,被Google隱藏的權限操作-->

<action android:name="android.provider.Telephony.SMS_RECEIVED" />

</intent-filter>

</receiver>

切記,權限

<uses-permission android:name="android.permission.RECEIVE_SMS" />

SMSReceiver

package com.lgl.ipcall;

import android.content.BroadcastReceiver;

import android.content.Context;

import android.content.Intent;

import android.os.Bundle;

import android.telephony.SmsMessage;

import android.util.Log;

/**

* 短信攔截器

* Created by lgl on 16/4/10.

*/

public class SMSReceiver extends BroadcastReceiver {

@Override

public void onReceive(Context context, Intent intent) {

System.out.println("收到短信");

//獲取短信內容

Bundle bundle = intent.getExtras();

//返回的是一個Object數組

Object[] objects = (Object[]) bundle.get("pdus");

//遍歷數組得到短信內容

for (Object object : objects) {

//把數組元素轉換成短信對象

SmsMessage sms = SmsMessage.createFromPdu((byte[]) object);

//獲取發件人號碼

String toPhone = sms.getOriginatingAddress();

//獲取短信內容

String smsContent = sms.getMessageBody();

Log.i("SMSReceiver", "發件人號碼:" + toPhone + "短信內容" + smsContent);

//判斷是否是攔截的號碼

if (toPhone.equals("12345678910")) {

//攔截廣播

abortBroadcast();

}

}

}

}

這樣我們運行之下，你會發現，雖然走到攔截這一步，但是並沒有阻止顯示在短信收件箱裡，這裡，我們要注意一個優勢，就是廣播接收者是有優先級定義的，我們只需要在清單註冊根節點的intent-filter標籤裡定義一個

android:priority="1000"

官方文檔是說數值在-1000到1000之間，但是最高支持int的最大值的權限，int最大值是多少？自己去看API

3.監聽SD卡

>我們在對SD卡進行讀寫的時候會用到，其實也就是鞏固一下對廣播的使用，做那種語音助手之類的輔助軟件，廣播和服務還是很有用的，我們還是定義一個SDReceiver並且在清單文件註冊

<!--註冊廣播-->

<receiver android:name=".SDReceiver">

<intent-filter>

<!--SD卡就緒廣播-->

<action android:name="android.intent.action.MEDIA_MOUNTED" />

<!--SD卡拔出廣播-->

<action android:name="android.intent.action.MEDIA_REMOVED" />

<!--SD卡卸載廣播-->

<action android:name="android.intent.action.MEDIA_UNMOUNTABLE" />

<data android:scheme="file"/>

</intent-filter>

</receiver>

我們現在可以監聽了

SDReceiver

package com.lgl.ipcall;

import android.content.BroadcastReceiver;

import android.content.Context;

import android.content.Intent;

import android.util.Log;

/**

* 監聽SD卡

* Created by lgl on 16/4/10.

*/

public class SDReceiver extends BroadcastReceiver {

@Override

public void onReceive(Context context, Intent intent) {

//判斷廣播

String action = intent.getAction();

if (action.equals(Intent.ACTION_MEDIA_MOUNTED)) {

Log.i("SDReceiver", "SD卡就緒");

} else if (action.equals(Intent.ACTION_MEDIA_REMOVED)) {

Log.i("SDReceiver", "SD卡拔出");

} else if (action.equals(Intent.ACTION_MEDIA_UNMOUNTABLE)) {

Log.i("SDReceiver", "SD卡卸載");

}

}

}

RebootReceiver

package com.lgl.ipcall;

import android.content.BroadcastReceiver;

import android.content.Context;

import android.content.Intent;

import android.util.Log;

/**

* 開機啟動

* Created by lgl on 16/4/10.

*/

public class RebootReceiver extends BroadcastReceiver {

@Override

public void onReceive(Context context, Intent intent) {

Log.i("RebootReceiver", "開機");

//啟動

Intent i = new Intent(context, MainActivity.class);

//在Activity之外啟動需要設置Flags

i.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK);

context.startActivity(i);

}

}

5.應用安裝卸載監聽

我們如果做手機助手或者應用的時候或許可以用得上這玩意，新建一個APPReceiver，然後去註冊

<!--註冊廣播-->

<receiver android:name=".APPReceiver">

<intent-filter>

<!--安裝應用-->

<action android:name="android.intent.action.PACKAGE_ADDED" />

<!--更新應用-->

<action android:name="android.intent.action.PACKAGE_REPLACED" />

<!--卸載應用-->

<action android:name="android.intent.action.PACKAGE_REMOVED" />

<!--攜帶包名-->

<data android:scheme="package"/>

</intent-filter>

</receiver>

然後我們來判斷

package com.lgl.ipcall;

import android.content.BroadcastReceiver;

import android.content.Context;

import android.content.Intent;

import android.net.Uri;

import android.util.Log;

/**

* Created by lgl on 16/4/10.

*/

public class APPReceiver extends BroadcastReceiver {

@Override

public void onReceive(Context context, Intent intent) {

//判斷廣播類型

String action = intent.getAction();

//獲取包名

Uri appName = intent.getData();

if (Intent.ACTION_PACKAGE_ADDED.equals(action)) {

Log.i("APPReceiver", "安裝" + appName);

} else if (Intent.ACTION_PACKAGE_REPLACED.equals(action)) {

Log.i("APPReceiver", "更新" + appName);

} else if (Intent.ACTION_PACKAGE_REMOVED.equals(action)) {

Log.i("APPReceiver", "卸載" + appName);

}

}

}
