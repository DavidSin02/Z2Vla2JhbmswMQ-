判斷一棵二叉樹是否為完全二叉樹

完全二叉樹

完全二叉樹，即每一層都是要按從左往右，依此填滿，最後一層可以不滿，但是所有節點都集中在最左邊，如圖，圖中1，2，3，4層節點的方向都是從左往右依此填充，每一層填充滿了才進入下一層，最後不滿，但是它也是從左往右依此填充的，節點都堆在最左邊。

如下圖，最後一層不是從左往右依此填充的，因此不為完全二叉樹。

節點8和節點9中間隔離一個節點的空位，9應該在8的右邊，整棵樹才是完全二叉樹。把黑色的節點9放到紅色的節點9處，整棵樹才是完全二叉樹。

樹的層次遍歷

一棵樹要從頂到底按層遍歷，要完成先進先出的操作，因此，可以使用一個隊列來完成，例如，按層遍歷如下的樹。

先把1進隊列，然後出隊，把1的左右節點2、3依此進隊，2出隊，把2的左右節點5、6依此進隊，3出隊，把3的左右節點7、8進隊......依此類推，直到隊列為空，整棵樹按層遍歷也就結束了。

判斷是否為完全二叉樹

一棵完全二叉樹有以下兩種情況。

1）節點有右孩子沒有左孩子的情況下，返回false

2）不違反1的條件下，左右不全的節點，則按層次遍歷的往後的節點都為葉子節點。

因此思路就是按層遍歷，然後判斷以上的兩種情況，最後返回結果。

代碼

public static boolean isCBT(Node head) { if (head == null) { return true; } Queue<Node> queue = new LinkedList<Node>(); boolean leaf = false; Node l = null; Node r = null; queue.offer(head); while (!queue.isEmpty()) { head = queue.poll(); l = head.left; r = head.right; if ((leaf && (l != null || r != null))  || (l == null && r != null)) { return false; } if (l != null) { queue.offer(l); } if (r != null) { queue.offer(r); } else { leaf = true; } } return true; }
