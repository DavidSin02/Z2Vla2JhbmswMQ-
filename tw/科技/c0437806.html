四、滑坡地段路基施工

2.滑坡防治的工程措施

滑坡防治的工程措施主要有排水、力學平衡和改變滑帶土三類。

在滑坡體未處治之前 ，嚴禁在滑坡體抗滑段減載、下滑段加載。滑坡整治不宜在雨期施工。

（1）環形截水溝

（2）樹枝狀排水溝

（3）平整夯實滑坡體表

（4）排除地下水

有截水滲溝、支撐滲溝、邊坡滲溝、暗溝、平孔等

2）力學平衡

當挖方路基上邊坡發生的滑坡不大時，可採用刷方（臺階）減重、打樁或修建擋土牆進行處理以達到路基邊坡穩定。牽引式滑坡、具有膨脹性質的滑坡不宜用滑坡減重法。

填方路堤發生的滑坡，可採用反壓土方或修建擋土牆等方法處理。

3）改變滑帶土

用物理化學方法改善滑坡帶土石性質。一般有焙燒法、電滲排水法和爆破灌漿法等。

林玉進老師【V:JZSLYJ】口訣：孫紅雷燒電報

(1)焙燒法：利用導洞焙燒滑坡腳部的滑帶，使之形成地下"擋牆"而穩定滑坡的一種措施。

(2)電滲排水：利用電場作用而把地下水排除，達到穩定滑坡的一種方法。

(3)爆破灌漿法：用炸藥爆破破壞滑動面，隨之把漿液灌入滑帶中以置換滑帶水並固結滑帶土，從而達到使滑坡穩定的一種治理方法。

3.滑坡地段路基的施工技術要點

1 ) 削坡減載施工應符合下列規定：

( 1 ) 應自上而下逐級開挖，嚴禁採用爆破法施工。

( 2 ) 開挖坡面不得超挖，開挖面上有裂縫時應予灌漿封閉或開挖夯填。

( 3 ) 支擋及排水工程在邊坡上分級實施時，宜開挖一級、實施一級。

3） 抗滑支擋工程施工應符合下列規定：

（3）抗滑樁與擋土牆共同支擋時，應先施作抗滑樁。擋土牆後有支撐滲溝及其他排水工程時應先施工。

（4） 抗滑樁、錨索施工應從兩端向滑坡主軸方向逐步推進。

（6）各種支擋結構的基底應置於滑動面以下，並應嵌入穩定地層。
