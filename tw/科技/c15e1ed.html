挑戰！一個數組和一個數值，在數組中找出兩個數的和等於這個數值

2020-10-29
| 科技

下面是來自leetcode的一個簡單的題目（小編能力有限只能挑戰下簡單的）：

給定一個整數數組 nums 和一個目標值 target，請你在該數組中找出和為目標值的那兩個整數，並返回他們的數組下標。

你可以假設每種輸入只會對應一個答案。但是，數組中同一個元素不能使用兩遍。

示例:

給定 nums = [2, 7, 11, 15], target = 9因為 nums[0] + nums[1] = 2 + 7 = 9所以返回 [0, 1]


解題開始

第一種解法[暴力解法]，也是最大眾化的解法，so easy。

解題思路:循環數組內的值，將當前的值和後面的值依次相加。

function twoSum($nums, $target) {    $nums_lenth = count($nums);    for($i=0; $i< $nums_lenth; $i++){        for($j=$i+1; $j< $nums_lenth; $j++){            if ($nums[$i] + $nums[$j] == $target) {                break ;            }         }    }    return [$i, $j];   }

這種解法的結果是沒問題的，但是效率可想而，肯定不高，而且違背的題目的限制條件（數組中同一個元素不能使用兩遍）。所以不能使用該方法，下面就是小編自己的另一種解法。

解題思路：

1.循環數組，用目標值減去數組中當前的值 （$target-$n）

2.判斷上一步的結果在不在數組中（用array_keys方法判斷），如果在數組中array_keys的返回值不為空。

3.循環array_keys後的結果，過濾當前值（因為數組可能為[3,3]，目標值為6，array_keys的結果即為一個數組）。


function twoSum($nums, $target) {        foreach($nums as $key=>$n)        {            $secnum = $target-$n;            $seckeys=array_keys($nums,$secnum);            foreach($seckeys as $seckey)            {                if( $seckey!=$key)                {                    return [$key,$seckey];                }            }        }    }
