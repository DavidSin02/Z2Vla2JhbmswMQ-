深入理解二叉樹01 二叉樹基礎

2020-11-14
| 科技

數據結構是所有的程序員就業過程中無法迴避的知識，最近在回顧數據結構的內容，因此會不定期的推出一些數據結構的文章，分享自己的筆記。

樹是數據結構中的重點，由於二叉樹又是樹中的重中之重。二叉樹的應用也非常的多，比如二叉排序樹、平衡二叉樹、紅黑樹、B樹、堆等。本篇文章是關於二叉樹的第一篇文章，主要介紹二叉樹的幾種遍歷方式以及代碼實現。

一、二叉樹的基本概念

二叉樹的定義：一棵二叉樹節點的有限集合，該集合或者為空，或者由一個根節點加上兩棵左子樹和右子樹組成，其左子樹和右子樹又分別是一棵二叉樹。

二叉樹的特點：

  • 每個節點最多隻有兩棵子樹，即二叉樹不存在度數大於2的節點
  • 二叉樹的子樹有左、右之分，其子樹的次序不能顛倒
  • 二叉樹的典型遍歷方式有四種：先序遍歷、中序遍歷、後序遍歷、層序遍歷

幾種特殊的二叉樹：

  • 滿二叉樹：在一棵二叉樹中，所有分支節點都存在左子樹和右子樹，並且所有的葉節點都在同一層上。如圖1。
  • 完全二叉樹：一棵具有N個節點的完全二叉樹的結構與滿二叉樹的前N個節點的結構相同。如圖2。

1、二叉樹的先序遍歷

二叉樹的先序遍歷：先訪問根節點，再訪問左子樹，最後訪問右子樹。

上述的二叉樹的先序遍歷結果為：10 6 4 8 14 12 16

遞歸方式實現：

void BinaryTreeOperate::preOrderTraverse(const BinaryTree* root){    if(root != nullptr)    {        cout << root->data << "\t";        BinaryTreeOperate::preOrderTraverse(root->leftChild);        BinaryTreeOperate::preOrderTraverse(root->rightChild);    }}

非遞歸方式實現：

思考一下如何實現：首先訪問根節點，然後訪問它的左子樹；當左子樹為空時，我們要退回到上一個訪問的節點(根節點)，訪問它的右子樹。如何實現退回到上一個訪問的節點的操作？有數據結構基礎的人很容易就會想到棧了。

過程：

  • 訪問根節點，然後將根節點入棧
  • 遍歷根節點的左子樹
  • 當左子樹為空，獲取棧頂元素並出棧，然後訪問棧頂元素的右子樹
void BinaryTreeOperate::preOrderTraverseNonRecurve(const BinaryTree* root){    stack<BinaryTreeNode*> nodes_stack;    //root用了const修飾，直接將const賦值為non-const是不可以的    //需要先使用const_cast去除root的const限定    BinaryTreeNode* p = const_cast<BinaryTree*>(root);    while(p != nullptr || !nodes_stack.empty())    {        if(p != nullptr)        {            cout << p->data << "\t";            nodes_stack.push(p);            p = p->leftChild;        }        else         {            p = nodes_stack.top();            nodes_stack.pop();            p = p->rightChild;        }    }}

2、二叉樹的中序遍歷

二叉樹的中序遍歷：先遍歷左子樹，再訪問根節點，最後訪問右子樹。

上述的二叉樹的中序遍歷結果為：4 6 8 10 12 14 16

遞歸方式實現：

void BinaryTreeOperate::inOrderTraverse(const BinaryTree* root){    if(root != nullptr)    {        BinaryTreeOperate::inOrderTraverse(root->leftChild);        cout << root->data << "\t";        BinaryTreeOperate::inOrderTraverse(root->rightChild);    }}

非遞歸方式實現：

中序遍歷的非遞歸實現和先序遍歷類似：

  • 先將根節點入棧
  • 遍歷根節點的左子樹
  • 當左子樹為空時，獲取棧定元素並出棧，訪問棧頂元素，然後遍歷棧頂元素的右子樹
void BinaryTreeOperate::inOrderTraverseNonRecurve(const BinaryTree* root){    stack<BinaryTreeNode*> nodes_stack;    BinaryTreeNode* p = const_cast<BinaryTreeNode*>(root);    while(p != nullptr || !nodes_stack.empty())    {        if(p != nullptr)        {            nodes_stack.push(p);            p = p->leftChild;        }        else         {            p = nodes_stack.top();            nodes_stack.pop();            cout << p->data << "\t";            p = p->rightChild;        }    }}

3、二叉樹的後序遍歷

二叉樹的後序遍歷：先遍歷左子樹，再遍歷右子樹，最後訪問根節點。

上述二叉樹的後序遍歷結果為：4 8 6 12 16 14 10

遞歸方式實現：

void BinaryTreeOperate::postOrderTraverse(const BinaryTree* root){    if(root != nullptr)    {        BinaryTreeOperate::postOrderTraverse(root->leftChild);        BinaryTreeOperate::postOrderTraverse(root->rightChild);        cout << root->data << "\t";    }}

非遞歸方式實現：

後序遍歷的非遞歸實現稍微複雜一點，先看一下下面的分析：

對於下圖所示的這一棵二叉樹：

後序遍歷的結果為：4 8 6 12 16 14 10

後序遍歷的逆序結果為：10 14 16 12 6 8 4

想一下後序遍歷的逆序結果是如何得到的：

  • 先遍歷根節點
  • 然後遍歷右子樹
  • 最後遍歷左子樹

思路出來了：

  1. 模仿先序遍歷的過程，但是把左右節點的訪問順序顛倒，同時把訪問的結果加到棧中（因為後序遍歷是它的一個逆序的過程）
  2. 遍歷棧中的所有節點，即為後序遍歷的結果
void BinaryTreeOperate::postOrderTraverseNonRecurve(const BinaryTree* root){    stack<BinaryTreeNode*> nodes_stack_1, nodes_stack_2;    BinaryTreeNode* p = const_cast<BinaryTreeNode*>(root);    while(p != nullptr || !nodes_stack_1.empty())    {        if(p != nullptr)        {            nodes_stack_2.push(p);            nodes_stack_1.push(p);            p = p->rightChild;        }        else         {            p = nodes_stack_1.top();            nodes_stack_1.pop();            p = p->leftChild;        }    }    while(!nodes_stack_2.empty())    {        p = nodes_stack_2.top();        nodes_stack_2.pop();        cout << p->data << "\t";    }}

4、二叉樹的層序遍歷

二叉樹的層序遍歷：層序遍歷也可以被稱為寬度優先遍歷，先訪問樹的第一層節點，再訪問樹的第二層節點······。再同一層節點中，以從左到右的順序依次訪問。

上述二叉樹的層序遍歷結果為：10 6 14 4 8 12 16

代碼實現（藉助隊列）：

void BinaryTreeOperate::widthFirstTraverse(const BinaryTree* root){    queue<BinaryTreeNode*> nodes_queue;    BinaryTreeNode* p = const_cast<BinaryTreeNode*>(root);    nodes_queue.push(p);    while(!nodes_queue.empty())    {        p = nodes_queue.front();        nodes_queue.pop();        if(p != nullptr)        {            cout << p->data << "\t";            if(p->leftChild != nullptr)                nodes_queue.push(p->leftChild);            if(p->rightChild != nullptr)                nodes_queue.push(p->rightChild);        }    }}

關於二叉樹的遍歷操作：遞歸實現簡單，但是效率就比較低，因為編譯器幫你做了很多的現場保護和現場恢復的工作；非遞歸實現都借用了棧，其實也是在模仿遞歸。兩種方式都很重要。
