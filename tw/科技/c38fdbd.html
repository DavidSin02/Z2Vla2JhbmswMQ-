Word批量化應用之：帶自動編號的簡易抽獎券

2020-10-29
| 科技

導讀：有一次小編在打字店裡看到店家用Word為一家KTV製作活動簡易抽獎券，因顧客等著獎券去救急，店家簡單設計好獎券模板得到顧客認可之後，複製模板並不斷重複粘貼 出近百份獎券；然後從頭開始去一張一張修改獎券編號，手工輸入編號容易出錯搞得手忙腳亂……其實稍微熟悉word操作的話，只需要利用域功能，幾分鐘就可以設計出上百份帶自動編號的簡易抽獎券（如下圖所示）。下面給大家分享操作方法。

  • 首先根據自己的需求自己設計一個簡易抽獎券，編號先不要填。可以利用word裡面自帶的圖形工具、藝術字等功能；下面的抽獎券是小編為了方便講解簡單做的一個。

  • 選中整個表格，打開“段落”對話框進入如下圖所示的設置，目的是避免獎券發生跨頁問題而成為無法使用的廢票。

  • 好了，現在到重點了。將光標定位到正券的“No：”之後，按 Ctrl + F9 插入域標識{ }，然後在其中輸入“seq z \#000”；再將光標定位到副券的“No：”之後，按 Ctrl + F9 插入域標識{ }，然後在其中輸入“seq f \#000”;如下圖所示。（注：\#000是數字格式開關，#後的3個0代表保留三位數，可根據實際需要設置0的個數。）

  • 按 Ctrl + A 全選文檔，按 F9鍵更新域即可看到自動生成的獎券編號，如下圖所示。

  • 在保持全選的情況下，按 Ctrl + C 複製做好的第1份抽獎券，接著按 Ctrl + End 將光標定位到文檔最後，然後按 Ctrl + V 不放來批量粘貼抽獎券模板，每隔幾秒鐘稍停一下觀察界面左下角的頁碼數，可大致估算已粘貼出多少抽獎券模板，如果覺得差不多夠了就按 Ctrl + A 全選再按 F9鍵更新域，即可看到最後1張抽獎券的編號；如果還不夠則繼續在文檔最後粘貼抽獎券模板，然後重複上述後續操作即可。

  • 完成之後，打印出來，可沿著虛線將正券與副券裁開；當然還需要橫向將獎券裁開成單張就可以了。
