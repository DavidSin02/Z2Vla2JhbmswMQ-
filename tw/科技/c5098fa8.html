函數

解釋

函數是為了達到某種目的而採取的行為，函數是可重複使用的，用來實現某個單一功能或者功能片段的代碼塊，簡單來說就是由一系列的程序語句組成的程序段落.

函數存在的意義:

1. 提高代碼的複用性

2. 將複雜的邏輯簡單化，分功能化

函數定義:

小知識 : []表示可選 <>表示必備

函數名字最好能直接表示該函數的功能，多個單詞之間用_鏈接

<def> <函數名>([參數列表])<:># 待執行語句# 如果有需要顯式返回<return 標識符>或<return>

定義一個簡單的函數:

def func():  print('我執行了')print('程序正常執行')

此時函數內的語句並沒有執行，是不是有點不符合程序從上到下的執行順序了?那肯定不是的，函數有個特點，只有調用了才能執行。

def func():    print('我執行了')print('程序正常執行')# 調用函數，函數名+()func()
# 函數先調用後定義# 拋出異常SyntaxError: invalid syntax# func()def func():  print('我執行了')func()

程序運行的時候先把函數內的代碼加載到內存中，有調用的地方直接執行，先調用的時候程序還沒有讀取函數到內存中，所以拋出異常。

def func():  print('嗨')  print(func)

直接打印函數就是該函數對應的內存地址。

函數的返回值

函數可以返回任何類型的數據，函數內執行到return後結束，後面代碼不在執行.

def func():    res = 1+1    print('我執行了')    return res    print('我不會執行')print('程序正常執行')# 函數內的語句確實是執行了，打印我執行了func()# 想要拿到函數return的值需要重新找個變量接收一下# 注意這裡再次打印了一次我執行了  因為函數再次調用了一次res = func()print(res)
# 不寫return默認返回Nonedef func():  print('我執行了')res = func()print(res)

參數

假如現在咱們要計算兩個數字的和。

def sue_for_peace():  res = 3+3  return resprint(sue_for_peace())

如果要計算不同數字的和就需要參數傳遞了

默認的按位置傳遞(位置一一對應)

def sue_for_peace(num1，num2):    res = num1+num2    return resprint(sue_for_peace(3，3))



關鍵字傳參(參數名對應)

def sue_for_peace(num1，num2):    print(num1，num2)    res = num1+num2    return resprint(sue_for_peace(num2=3，num1 = 4))

混合使用(先位置參數再關鍵字參數)

def sue_for_peace(num1，num2，num3):    print(num1，num2，num3)    res = num1+num2+num3    return resprint(sue_for_peace(3，num3=1，num2 = 2))

默認值(不傳遞參數就是用默認值，傳遞了就是用傳遞的值)

def sue_for_peace(num1=1，num2=2):    print(num1，num2)    res = num1+num2    return resprint(sue_for_peace())print(sue_for_peace(5，10))

混合使用時，先按位置傳遞參數，後按關鍵字傳遞參數，最後是默認值

錯誤示範:

def sue_for_peace(num1=1，num2，num3): # 第一行這裡就錯了，默認值只能再最後，編輯器報錯    print(num1，num2，num3)    res = num1+num2+num3    return resprint(sue_for_peace(2，3))
def sue_for_peace(num1， num2， num3=3):    print(num1， num2， num3)    res = num1+num2+num3    return resprint(sue_for_peace(num2=2，3)) # 先位置後關鍵字，編輯器報錯print(sue_for_peace(1，num1=2)) # 不允許給同一個形參傳兩個值

可變參數(多個不確定的參數)

*列表 **字典

通常都是這樣子的:*args，**kwargs

def sue_for_peace(*args):    res = 0    for i in args:        res += i    return res# 任意個 0個也沒錯print(sue_for_peace())print(sue_for_peace(1， 2， 3， 4， 5))# 假如已經有一個列表了li = [1， 2， 3， 4， 5]# 這樣不是把列表當作一個參數來傳遞了，會自動解釋為多個參數print(sue_for_peace(*li))
def func(**kwargs):    return kwargsprint(func())print(func(name='sb'，age=22))print(func(**{'name':'sb'，'age':22}))

一般都是這樣子的:

def func(x，y，z，*args，**kwargs):    print(x)    print(y)    print(z)    print(args)    print(kwargs)func(1， 2， 3， 4， 5， 6， a=11， b=22)

可變類型參數的傳遞

def func(li):    li[0] = 666li = [1， 2， 3， 4， 5]func(li)print(li)

可以發現實參傳遞給形參，形參發生改變，實參跟著發生改變

不可變類型的傳遞

def func(str1):    str1 = 'aaa'    print('函數內:'， str1)str1 = 'hell0'func(str1)print(str1)

不要被相同的變量名字迷惑了...


全局變量和局部變量

局部可以使用全局變量，全局變量可以再模塊(也就是這個文件)的任何地方使用，局部變量是在函數內部聲明並使用的數據量，隨函數的啟動而出生，隨函數的退出而消亡，作用域在函數內

局部可以使用全局變量

temp = 'hello'def test():  print(temp)

全局不能使用局部變量

def test():    temp = 'hello'    print(temp)# print(temp)# NameError: name 'temp' is not defined
temp = '你好'def test():    temp = 'hello'    print(temp)print(temp)
temp = '你好'def test():    temp = 'hello'    print(temp)test()print(temp)

看例子猜答案，皮一下，很開心。

num = 1def set_num(in_num):    num = in_num    passset_num(11)print(num)

聲明使用全局變量(global)

num = 1def set_num(in_num):  global num    num = in_num    passset_num(11)print(num)

全局變量，局部變量重名，仍需要用全局變量。

num = 11def test():    num = 22    # 這個是局部num    print(num)    # 打印全局num    print(globals()['num'])test()
