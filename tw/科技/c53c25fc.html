異常檢測看似是機器學習中一個有些難度的問題，但採用合適的算法也可以很好解決。本文介紹了孤立森林（isolation forest）算法，通過介紹原理和代碼教你揪出數據集中的那些異常值。


從銀行欺詐到預防性的機器維護，異常檢測是機器學習中非常有效且普遍的應用。在該任務中，孤立森林算法是簡單而有效的選擇。

異常檢測簡介

離群值是在給定數據集中，與其他數據點顯著不同的數據點。

異常檢測是找出數據中離群值（和大多數數據點顯著不同的數據點）的過程。

真實世界中的大型數據集的模式可能非常複雜，很難通過查看數據就發現其模式。這就是為什麼異常檢測的研究是機器學習中極其重要的應用。

本文要用孤立森林實現異常檢測。我們有一個簡單的工資數據集，其中一些工資是異常的。目標是要找到這些異常值。可以想象成，公司中的一些僱員掙了一大筆不同尋常的鉅額收入，這可能意味著存在不道德的行為。

在繼續實現之前，先討論一些異常檢測的用例。

異常檢測用例

異常檢測在業界中應用廣泛。下面介紹一場常見的用例：

銀行：發現不正常的高額存款。每個賬戶持有人通常都有固定的存款模式。如果這個模式出現了異常值，那麼銀行就要檢測並分析這種異常（比如洗錢）。

金融：發現欺詐性購買的模式。每個人通常都有固定的購買模式。如果這種模式出現了異常值，銀行需要檢測出這種異常，從而分析其潛在的欺詐行為。

衛生保健：檢測欺詐性保險的索賠和付款。

製造業：可以監測機器的異常行為，從而控制成本。許多公司持續監視著機器的輸入和輸出參數。眾所周知，在出現故障之前，機器的輸入或輸出參數會有異常。從預防性維護的角度出發，需要對機器進行持續監控。

網絡：檢測網絡入侵。任何對外開放的網絡都面臨這樣的威脅。監控網絡中的異常活動，可以及早防止入侵。

接著瞭解一下機器學習中的孤立森林算法。

什麼是孤立森林

孤立森林是用於異常檢測的機器學習算法。這是一種無監督學習算法，通過隔離數據中的離群值識別異常。

孤立森林是基於決策樹的算法。從給定的特徵集合中隨機選擇特徵，然後在特徵的最大值和最小值間隨機選擇一個分割值，來隔離離群值。這種特徵的隨機劃分會使異常數據點在樹中生成的路徑更短，從而將它們和其他數據分開。

一般而言，異常檢測的第一步是構造「正常」內容，然後報告任何不能視為正常的異常內容。但孤立森林算法不同於這一原理，首先它不會定義「正常」行為，而且也沒有計算基於點的距離。

一如其名，孤立森林不通過顯式地隔離異常，它隔離了數據集中的異常點。

孤立森林的原理是：異常值是少量且不同的觀測值，因此更易於識別。孤立森林集成了孤立樹，在給定的數據點中隔離異常值。

孤立森林通過隨機選擇特徵，然後隨機選擇特徵的分割值，遞歸地生成數據集的分區。和數據集中「正常」的點相比，要隔離的異常值所需的隨機分區更少，因此異常值是樹中路徑更短的點，路徑長度是從根節點經過的邊數。

用孤立森林，不僅可以更快地檢測異常，還需要更少的內存。

孤立森林隔離數據點中的異常值，而不是分析正常的數據點。和其他正常的數據點相比，異常數據點的樹路徑更短，因此在孤立森林中的樹不需要太大的深度，所以可以用更小的 max_depth 值，從而降低內存需求。

這一算法也適用於小數據集。

接著我們對數據做一些探索性分析，以瞭解給定數據的相關信息。

探索性數據分析

先導入所需的庫。導入 numpy、pandas、seaborn 和 matplotlib。此外還要從 sklearn.ensemble 中導入孤立森林（IsolationForest）。

import numpy as npimport pandas as pdimport seaborn as snsimport matplotlib.pyplot as pltfrom sklearn.ensemble import IsolationForest

導入庫後，要將 csv 數據讀取為 padas 數據框，檢查前十行數據。

本文所用數據是不同職業的人的年薪（美元）。數據中有一些異常值（比如工資太高或太低），目標是檢測這些異常值。

df = pd.read_csv('salary.csv')df.head(10)

數據集表頭。

為了更好地瞭解數據，將工資數據繪製成小提琴圖，如下圖所示。小提琴圖是一種繪製數值數據的方法。

通常，小提琴圖包含箱圖中所有數據——中位數的標記和四分位距的框或標記，如果樣本數量不太大，圖中可能還包括所有樣本點。

工資的小提琴圖。

為了更好地瞭解離群值，可能還會查看箱圖。箱圖一般也稱為箱線圖。箱圖中的箱子顯示了數據集的四分位數，線表示剩餘的分佈。線不表示確定為離群值的點。

我們通過 interquartile range, 的函數檢測離群值。在統計數據中，interquartile range，（也稱為 midspread 或 middle 50%）是度量統計學分散度的指標，等於第 75% 個數和第 25% 個數的差。

工資的箱圖，指示了右側的兩個離群值。

完成數據的探索性分析後，就可以定義並擬合模型了。

定義及擬合模型

我們要創建一個模型變量，並實例化 IsolationForest（孤立森林）類。將這四個參數的值傳遞到孤立森林方法中，如下所示。

  • 評估器數量：n_estimators 表示集成的基評估器或樹的數量，即孤立森林中樹的數量。這是一個可調的整數參數，默認值是 100；
  • 最大樣本：max_samples 是訓練每個基評估器的樣本的數量。如果 max_samples 比樣本量更大，那麼會用所用樣本訓練所有樹。max_samples 的默認值是『auto』。如果值為『auto』的話，那麼 max_samples=min(256, n_samples)；
  • 數據汙染問題：算法對這個參數非常敏感，它指的是數據集中離群值的期望比例，根據樣本得分擬合定義閾值時使用。默認值是『auto』。如果取『auto』值，則根據孤立森林的原始論文定義閾值；
  • 最大特徵：所有基評估器都不是用數據集中所有特徵訓練的。這是從所有特徵中提出的、用於訓練每個基評估器或樹的特徵數量。該參數的默認值是 1。
model=IsolationForest(n_estimators=50, max_samples='auto', contamination=float(0.1),max_features=1.0)model.fit(df[['salary']])


孤立森林模型訓練輸出。

模型定義完後，就要用給定的數據訓練模型了，這是用 fit() 方法實現的。這個方法要傳入一個參數——使用的數據（在本例中，是數據集中的工資列）。

正確訓練模型後，將會輸出孤立森林實例（如圖所示）。現在可以添加分數和數據集的異常列了。

添加分數和異常列

在定義和擬合完模型後，找到分數和異常列。對訓練後的模型調用 decision_function()，並傳入工資作為參數，找出分數列的值。

類似的，可以對訓練後的模型調用 predict() 函數，並傳入工資作為參數，找到異常列的值。

將這兩列添加到數據框 df 中。添加完這兩列後，查看數據框。如我們所料，數據框現在有三列：工資、分數和異常值。分數列中的負值和異常列中的 -1 表示出現異常。異常列中的 1 表示正常數據。

這個算法給訓練集中的每個數據點都分配了異常分數。可以定義閾值，根據異常分數，如果分數高於預定義的閾值，就可以將這個數據點標記為異常。

df['scores']=model.decision_function(df[['salary']])df['anomaly']=model.predict(df[['salary']])df.head(20)

給數據的每一行中都添加了分數和異常值後，就可以打印預測的異常了。

打印異常

為了打印數據中預測得到的異常，在添加分數列和異常列後要分析數據。如前文所述，預測的異常在預測列中的值為 -1，分數為負數。根據這一信息，將預測的異常（本例中是兩個數據點）打印如下。

anomaly=df.loc[df['anomaly']==-1]anomaly_index=list(anomaly.index)print(anomaly)

異常輸出。

注意，這樣不僅能打印異常值，還能打印異常值在數據集中的索引，這對於進一步處理是很有用的。

評估模型

為了評估模型，將閾值設置為工資>99999 的為離群值。用以下代碼找出數據中存在的離群值：

outliers_counter = len(df[df['salary'] > 99999])outliers_counter

計算模型找到的離群值數量除以數據中的離群值數量，得到模型的準確率。

print("Accuracy percentage:", 100*list(df['anomaly']).count(-1)/(outliers_counter))

準確率：100%
