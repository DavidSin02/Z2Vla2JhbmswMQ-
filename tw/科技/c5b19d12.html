C語言之變量

| 科技

變量

1.概念

計算機使用內存來記錄數據，內存中有很多的存儲位置，每個存儲位置中可以記錄數據。

變量在C語言中用來代表一個存儲位置，通過變量來訪問對應的存儲位置。

變量和存儲位置之間的關係由操作系統決定的。


2.變量的使用

（1）變量使用前必須先聲明

變量聲明語法：

		變量類型 變量名;

由於C語言中的數字是有類型的，所以代表存儲位置的變量也是有類型的。

變量名屬於標識符，需要遵循標識符的規範。


（2）訪問變量

訪問變量經常會用到賦值語句，賦值運算符(=)是賦值語句的核心

賦值運算符的作用是將"="右邊的數字保存到"="左邊的存儲位置上

比如： 	a = 10; 	b = 20;

賦值運算符的左邊必須是一個代表存儲位置的內容，稱之為左值。

賦值運算符的右邊必須是一個代表數字的內容，稱之為右值。

變量可以作為左值，也可以作為右值。數字只能作為右值。


（3）使用printf打印變量的值

printf函數不僅可以打印文字信息，也可以打印數字和變量

打印數字/變量使用佔位符，整數類型的佔位符是%d

用法如下：

int num; num = 10; printf("num = %d\n",num);

在打印的時候使用後面的 數字/變量 依次替換佔位符的內容

不同類型的 數字/變量 的佔位符不同


（4）變量的初始化和同時聲明多個同類型變量

在聲明變量的同時為變量賦值叫做變量的初始化

通常情況下所有的變量都應該初始化

沒有初始化的變量的值不確定

初始化語法：

變量類型 變量名 = 初始值;


可以在一行中聲明多個同類型的變量，語法如下：

變量類型 變量名1,變量名2，變量名3.....; 比如： int num, num_2, num_3, num_4, num_5;


（5）scanf標準函數

scanf標準函數可以在程序運行時把用戶在鍵盤上輸入的數字記錄到變量中

scanf調用語句中使用佔位符表示數字數字的類型

scanf使用變量的地址表示數字要存儲的位置(&變量名)

int num; scanf("%d", &num);


注意事項：

在雙引號中只能寫佔位符，不能添加其他的內容

一條scanf語句中可以讀取多個數字，輸入時多個數字之間用空格分開

使用scanf標準函數需要包含頭文件stdio.h的頭文件

scanf語句出錯會影響之後的scanf語句，在Linux中可以使用以下語句去掉這種影響(清除緩存)：

scanf("%*[^\n]"); scanf("%*c");


在C語言的代碼中可以直接使用 + - * /


練習：

編寫C語言的代碼，將用戶輸入的兩個整數的 和 差 積 商打印出來


3.變量的類型

整數類型：

（1）字符類型

關鍵字：char

字符類型代表一個數據類型，包含256個不同的數據，‘6’ ，'a' , '+'....

char類型的數據在我們的計算機上佔用1個字節

每個字符數據可以用數字表示，字符類型數據有兩種身份：字符身份和數字身份

字符身份和數字身份的對應關係在ASCII表中

'0' ---------------------- 48

'a' ---------------------- 97


char類型表示數字的範圍是 -128 ----- 127

ASCII表只規定了0-127對應的字符，-128 ----- -1對應的字符不確定


在ASCII表，所有的大寫字母，小寫字母，數字字符都是連續排列

'd' - 'a' 等於 'D' - 'A'

'd' - 'a' 等於 '3' - '0'


使用printf/scanf 輸入輸出字符類型數據時，使用%hhd表示數字身份，%c表示字符身份

越界處理：



幾個特殊的字符

'\n' ---------- 換行符

'\r' ----------- 回車符(將光標移動到本行開頭)

’\b‘ ---------- 退格符

'\t' ----------- 製表符

'\\' '\'' '\"' ------- 分別表示 \ ' " (轉義)

%% -------- 表示%


（2）無符號的字符類型

關鍵字：unsigned char

無符號的字符類型也包含256個字符數據，範圍0-255

大小也是1字節，其中0-127也對應ascii表

使用printf打印無符號字符類型時，使用佔位符%hhu打印數字身份，%c打印字符身份


（3）短整型

關鍵字：short (int)

短整型包含65536個不同的整數，範圍-32768 ----- 32767

在我們的計算機上佔2個字節

佔位符：%hd


（4）無符號短整型

關鍵字：unsigned short (int)

無符號短整型包含65536個不同的整數，範圍0 ----- 65535

在我們的計算機上佔2個字節

佔位符：%hu


（5）整型

關鍵字：int

整型包含2^32個不同的整數，範圍-2^31 ----- 2^31-1

在我們的計算機上佔4個字節

佔位符：%d


（6）無符號整型

關鍵字：unsigned (int)

整型包含2^32個不同的整數，範圍0 ----- 2^32-1

在我們的計算機上佔4個字節

佔位符：%u


注：可以使用sizeof來計算 類型 變量 數字的大小，單位是字節。

sizeof(int) sizeof(num) sizeof(-10)


（7）長整型

關鍵字：long (int)

32位系統：

長整型包含2^32個不同的整數，範圍-2^31 ----- 2^31-1

在我們的計算機上佔4個字節

64位系統：

長整型包含2^64個不同的整數，範圍-2^63 ----- 2^63-1

在我們的計算機上佔8個字節


佔位符：%ld


（8）無符號長整型

關鍵字：unsigned long (int)

32位系統：

整型包含2^32個不同的整數，範圍0 ----- 2^32-1

在我們的計算機上佔4個字節

64位系統：

長整型包含2^64個不同的整數，範圍0 ----- 2^64-1

在我們的計算機上佔8個字節


佔位符：%lu


（9）長長整型

關鍵字：long long (int)

長長整型包含2^64個不同的整數，範圍-2^63 ----- 2^63-1

在我們的計算機上佔8個字節

佔位符：%lld


（10）無符號長長整型

關鍵字：unsigned long long (int)

長長整型包含2^64個不同的整數，範圍0 ----- 2^64-1

在我們的計算機上佔8個字節

佔位符：%llu


浮點類型：

（11）單精度浮點型

關鍵字：float

在我們的計算機上佔4個字節

佔位符：%f %g(%f保留小數點後多餘的0，%g不保留) %e(科學計數法)


（12）雙精度浮點型

關鍵字：double

在我們的計算機上佔8個字節

佔位符：%lf %lg(%lf保留小數點後多餘的0，%lg不保留) %le(科學計數法)


注意事項：

C語言中，整數數字默認是int型，小數數字默認是double型。

如果要表示一個整數數字是無符號整型，可以在數字後面加u，比如8u

如果要表示一個小數數字是float類型，可以在數字後面加f，比如3.14f


（13）布爾類型 ----- C99加入

關鍵字：bool

在我們的計算機上佔1個字節

用來存放一個布爾值，布爾值只有兩種取值：真和假，本質上就是1和0

在C語言中使用關鍵字 true 和 false 分別表示真和假

除了數字0表示假以外，其他的數字都表示真

注：使用bool類型要添加頭文件stdbool.h

5.printf函數打印方法的補充

（1）可以使用%nd作為佔位符來打印整數數字，n是一個正整數，表示打印結果至少佔用n個位置，空位置在左邊補充空格字符


%nd打印整數


（2）可以使用%-nd作為佔位符來打印整數數字，n是一個正整數，表示打印結果至少佔用n個位置，空位置在右邊補充空格字符


%-nd打印整數


（3）可以使用%0nd作為佔位符來打印整數數字，n是一個正整數，表示打印結果至少佔用n個位置，空位置在左邊補充0字符


%0nd打印證書


（4）可以使用%n.mf來打印浮點數，表示整個數字至少佔n個位置，小數部分佔m個位置


%n.mf打印浮點數
