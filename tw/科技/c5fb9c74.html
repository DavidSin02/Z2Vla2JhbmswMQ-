Abaqus 2019 有限元分析軟件

2020-11-14
| 科技

Abaqus 是一款功能強大的用於有限元的分析軟件，其應用汽車、消費品、航空航天、能源和生命科學等多個領域。目前軟件的最新版已經發布，新版在諸多功能上進行了加強，Abaqus 提供了涵蓋工業應用的廣闊頻譜常規和複雜的工程問題的強大而完整的解決方案。Abaqus 的探索真實世界的物理行為，產品和材料，以提高性能，可靠性和安全性，同時減少開發時間和成本。是一款對未來具有戰略性的產品。


SIMULIA逼真的模擬解決方案，SIMULIA使虛擬測試成為一種標準的業務實踐，可以提高產品性能，減少物理原型並推動創新。搭載了3D體驗平臺，SIMULIA提供逼真的模擬應用程序，使用戶探索的產品，自然與生活真實世界的行為。SIMULIA應用程序在投入物理原型之前，加快了評估材料和產品的性能，可靠性和安全性的過程。SIMULIA提供先進的仿真產品組合，用於多物理場，流程集成和優化：Abaqus FEA，fe-safe，Isight，Tosca，Simpack，Simpoe和SIMULIA SLM。此外，SIMULIA V5和V6解決方案可為目前在CATIA V5或V6環境中工作的設計人員和設計工程師提供逼真的仿真。面向仿真用戶和企業的3DEXPERIENCE平臺提供了演化功能，以擴展當前的專家工具集。結合SIMULIA的技術Abaqus，fe-safe，Isight和Tosca來展示真實模擬的全部潛力。使用Extended Packaging可以訪問所有“產品組合的力量”技術。

軟件功能

靜態應力/位移分析：包括線性，材料和幾何非線性，以及結構斷裂分析等

動態分析粘彈性/粘塑性響應分析：粘塑性材料結構的響應分析

熱傳導分析：傳導，輻射和對流的瞬態或穩態分析

質量擴散分析：靜水壓力造成的質量擴散和滲流分析等

耦合分析：熱/力耦合，熱/電耦合，壓/電耦合，流/力耦合，聲/力耦合等

非線性動態應力/位移分析：可以模擬各種隨時間變化的大位移、接觸分析等

瞬態溫度/位移耦合分析：解決力學和熱響應及其耦合問題

準靜態分析：應用顯式積分方法求解靜態和衝壓等準靜態問題

退火成型過程分析：可以對材料退火熱處理過程進行模擬

海洋工程結構分析：

對海洋工程的特殊載荷如流載荷、浮力、慣性力等進行模擬

對海洋工程的特殊結構如錨鏈、管道、電纜等進行模擬

對海洋工程的特殊的連接，如土壤/管柱連接、錨鏈/海床摩擦、管道/管道相對滑動等進行模擬

水下衝擊分析：

對衝擊載荷作用下的水下結構進行分析

柔體多體動力學分析：對機構的運動情況進行分析，並和有限元功能結合進行結構和機械的耦合分析，並可以考慮機構運動中的接觸和摩擦

疲勞分析：根據結構和材料的受載情況統計進行生存力分析和疲勞壽命預估

設計靈敏度分析：對結構參數進行靈敏度分析並據此進行結構的優化設計

軟件除具有上述常規和特殊的分析功能外，在材料模型，單元，載荷、約束及連接等方面也功能強大並各具特點：

材料模型：定義了多種材料本構關係及失效準則模型，包括：

彈性：線彈性，可以定義材料的模量、泊松比等彈性特性

正交各向異性，具有多種典型失效理論，用於複合材料結構分析

多孔結構彈性，用於模擬土壤和可擠壓泡沫的彈性行為

亞彈性，可以考慮應變對模量的影響

超彈性，可以模擬橡膠類材料的大應變影響

粘彈性，時域和頻域的粘彈性材料模型


安裝步驟

1、準備好Abaqus 2019的安裝包，打開文件夾“_SolidSQUAD_”，解壓“_SolidSQUAD_.7z”；

2、打開解壓之後的“_SolidSQUAD_”，打開文件夾“Server”；

3、解壓“SSQ_UniversalLicenseServer_Core_20180127074300.zip”和“SSQ_UniversalLicenseServer

_Module_DSSimulia_20180127185300.zip”；

4、打開解壓之後的“SSQ_UniversalLicenseServer_Core_20180127074300”文件夾和“SSQ_UniversalLicenseServer_Module_DSSimulia_20180127185300”；

5、複製文件夾“Vendors”，粘貼到文件夾“SolidSQUAD_License_Servers”；

6、複製粘貼完成之後的文件夾“SolidSQUAD_License_Servers”，粘貼到C盤根目錄之下；

7、打開粘貼之後的文件夾“SolidSQUAD_License_Servers”，右擊“install_or_update.bat”，以管理員身份運行；

8、出現以下紅框內容，表示SolidSQUAD_License_Servers已經開啟，如果想要關閉服務，右擊“uninstall.bat”，以管理員身份運行，出現以下紅框內容，表明SolidSQUAD_License_Servers已經關閉；

9、需要對Abaqus 2019創建環境變量，跳過FlexNET License server的安裝，打開“此電腦”，點擊“屬性”；

10、點擊“高級系統設置”，點擊“環境變量”；

11、在“系統變量”點擊“新建”，變量名輸入“NOLICENSECHECK”，變量值輸入“true”，輸入完成之後點擊“確定”；

12、確認無誤，點擊“確定”；

13、返回Abaqus 2019安裝包，解壓“DS.SIMULIA.SUITE.2019.WIN64.iso；

14、打開文件“1”，右擊“setup.exe”，以管理員身份運行；

15、進入Abaqus 2019安裝環境，點擊“下一步”；

16、選擇Abaqus 2019的安裝組件，默認即可，FlexNET License server是不安裝的，點擊“下一步”；

17、確認Abaqus 2019的安裝組件信息，點擊“安裝”；

18、首先先安裝Abaqus Simulation Services，修改其安裝位置，點擊“下一步“；

19，確認Abaqus Simulation Services安裝目錄，確認無誤點擊“下一步”；

20、點擊“安裝”，Abaqus Simulation Services進入安裝；

21、Abaqus Simulation Services正在安裝；

22、Abaqus Simulation Services安裝完成，點擊“關閉“；

23、進入Abaqus Simulation Services CAAPI的安裝，

24、查看Abaqus Simulation Services CAAPI的安裝位置，點擊“下一步“；

25、確認Abaqus Simulation Services CAAPI安裝目錄，確認無誤點擊“下一步”；

26、點擊“安裝”，Abaqus Simulation Services CAAPI進入安裝；

27、Abaqus Simulation Services CAAPI正在安裝；

28、Abaqus Simulation Services CAAPI安裝完成，點擊“關閉“；

29、點擊“瀏覽”，將Abaqus 2019安裝包中的鏡像文件夾“DS.SIMULIA.SUITE.2019.WIN64”中的文件夾“2”進行添加，添加完成後，點擊“確定”；

30、進入Abaqus CAE 2019的安裝環境，修改其安裝位置，點擊“下一步”；

31、設置Abaqus CAE 2019的許可證類型，選擇“SIMULIAFLEXnet”，點擊“下一步”；

32、在Abaqus CAE 2019許可證界面中的【License Server 1】輸入“27800+@+計算機名稱”，點擊“下一步”；

33、進入Abaqus解算器的安裝，默認其安裝位置，點擊“下一步”；

34、設置Abaqus CAE 2019的文件保存位置，默認保存在C盤，可根據實際情況修改其文件默認保存位置，也可默認原設置路徑，點擊“下一步”；

35、設置Abaqus CAE 2019外部插件目錄路徑，可修改其目錄路徑，也可默認設置，點擊“下一步；

36、默認選擇“Connect to searchable documentation on help. 3ds. com”，點擊“下一步”；

37、設置Abaqus CAE 2019的數據保存地址，這裡修改其保存位置，基本上所有的數據保存地址都是可以默認的，點擊“下一步”；

38、確認Abaqus CAE 2019的安裝程序信息，點擊“安裝”；

39、Abaqus CAE 2019正在安裝；

40、Abaqus CAE 2019安裝完成，點擊“關閉”；

41、進入fe-safe2019的安裝環境，修改其安裝位置，點擊“下一步”；

42、選擇fe-safe2019的安裝部件，這裡默認設置，點擊“下一步”；

43、確認fe-safe2019的安裝部件信息，點擊“安裝”；

44、fe-safe2019正在安裝；

45、fe-safe2019安裝完成，點擊“關閉”；

46、進入Tosca 2019的安裝環境，修改其安裝位置，點擊“下一步”；

47、選擇文檔配置——“為本地安裝的文檔輸入文件路徑”，點擊“下一步”；

48、設置“SIMULIA文檔路徑”，默認保存到C盤，這裡設置為F（修改盤符級即可），點擊“下一步”；

49、點擊“繼續”；

50、選擇Tosca2019的安裝部件，這裡默認設置，點擊“下一步”；

51、可根據實際情況選擇接口類型，這裡默認選擇“ABAQUS接口”，點擊“下一步”；

52、這裡不做修改，點擊“下一步”；

53、確認Tosca 2019的安裝部件信息，點擊“安裝”；

54、Tosca 2019正在安裝；

55、Tosca 2019安裝完成，點擊“關閉”；

56、點擊“瀏覽”，將Abaqus 2019安裝包中的鏡像文件夾“DS.SIMULIA.SUITE.2019.WIN64”中的文件夾“”進行添加，添加完成後，點擊“確定”；

57、進入Isight 2019的安裝環境，修改其安裝位置，點擊“下一步”；

58、選擇Isight 2019的安裝部件，這裡默認設置，點擊“下一步”；

59、不勾選“Launch TomEE Configuration Ultility”，點擊“下一步”；

60、點擊“下一步”；

61、選擇“Enter a file path for locally installed documentation”，點擊“下一步”；

62、設置“SIMULIA文檔路徑”，默認保存到C盤，這裡設置為F（修改盤符級即可），點擊“下一步”；

63、點擊“Continue（繼續）”；

64、確認Isight 2019的安裝部件信息，點擊“安裝”；

65、Isight 2019正在安裝；

66、Isight 2019安裝完成，點擊關閉；

67、Abaqus 2019軟件全部安裝完成，點擊“關閉”；

68、打開開始菜單，點擊運行“Abaqus CAE”，查看其破解情況；

69、打開開始菜單，點擊運行“fe-safe2019”，查看其破解情況；

70、打開開始菜單，點擊運行“Design Gateway”，查看其破解情況；

71、打開開始菜單，點擊運行“Tosca Structure GUI”，查看其破解情況；
