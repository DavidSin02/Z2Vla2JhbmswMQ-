Excel VBA 學習 FillFormat 對象、ColorFormat 對象

2020-11-14
| 科技

代表形狀的填充格式。形狀可以有純色、漸變、紋理、圖案、圖片或半透明填充。

方法

OneColorGradient 方法：將指定填充設置為單色漸變。

Patterned 方法 ：將指定填充設置為一種圖案。

PresetGradient 方法：將指定填充設置為一個預設的漸變。

PresetTextured 方法 ：將指定填充格式設為預設紋理。

Solid 方法：將指定的填充格式設置為均勻的顏色。可用本方法將帶有漸變、紋理、圖案或背景的填充格式轉換為單色的填充格式。

TwoColorGradient 方法：將指定填充設置為雙色漸變。

UserPicture 方法 ：用圖像填充指定的形狀。

UserTextured 方法 ：用一張圖像的平鋪形式填充指定形狀。 如果您想要用一張大圖像填充形狀，請使用**UserPicture** 方法。

屬性

Application 屬性：在不使用對象識別符的情況下使用時, 此屬性返回代表 Microsoft Excel 應用程序的**Application** 對象。 如果與對象識別符一起使用，則此屬性返回代表指定對象的創建者的 Application 對象（可以將此屬性與 OLE 自動化對象一起使用以返回該對象的應用程序）。 只讀。

BackColor 屬性 ：返回或設置一個**ColorFormat** 對象，代表指定的填充背景色。

Creator屬性：返回一個 32 位整數，該整數指示在其中創建了此對象的應用程序。 只讀 Long。

ForeColor 屬性 ：返回或設置一個**ColorFormat** 對象，代表指定的前景填充色或純色。

GradientAngle 屬性：返回或設置指定填充格式的漸變填充的角度。 讀/寫

GradientColorType 屬性：返回指定填充的漸變顏色類型。 只讀的**MsoGradientColorType**。

GradientDegree 屬性 ：返回指定底紋單色填充的漸變程度的浮點值從 0.0 （暗） 到 1.0 （亮）。 只讀單個。

GradientStops 屬性 ：返回漸變填充的結束點。 只讀。

GradientStyle 屬性 ：返回指定填充的漸變樣式。 只讀**MsoGradientStyle**。

GradientVariant 屬性 ：從 1 到 4，作為整數值返回指定的填充格式的底紋變量。 此屬性的值對應於填充效果對話框中過渡選項卡上的漸變變量 （從左到右，從上到下編號）。

Parent 屬性：返回指定對象的父對象。

Pattern 屬性：返回或設置一個**MsoPatternType** 值，該值代表填充圖案。

PictureEffects 屬性：返回一個對象，該對象代表指定填充格式的圖片或紋理填充。 只讀

PresetGradientType 屬性 ：返回指定填充的預設漸變類型。 只讀**MsoPresetGradientType**。

PresetTexture 屬性 ：返回指定填充的預設紋理。 只讀**MsoPresetTexture**。

RotateWithObject 屬性 ：返回或設置與對象應旋轉的填充樣式。 讀/寫**MsoTriState**。

TextureAlignment 屬性 ：返回或設置指定FillFormat對象的文本對齊方式。 讀/寫。

TextureHorizontalScale 屬性 ：返回或設置為水平縮放FillFormat對象的文本值。 讀/寫 Single。

TextureName 屬性 ：返回指定填充的自定義紋理文件的名稱。 只讀 String。

TextureOffsetX 屬性 ：返回指定填充的偏移量 X 值。 讀/寫單個。

TextureOffsetY 屬性 ：返回指定填充的偏移量 Y 值。 讀/寫單個。

TextureTile 屬性 ：返回指定填充的紋理平鋪樣式。 讀/寫**MsoTriState**。

TextureType 屬性 ：返回指定填充的紋理類型。 只讀**MsoTextureType**。

TextureVerticalScale 屬性 ：返回指定填充的紋理垂直縮放。 讀/寫單個。

Transparency 屬性 ：返回或設置指定填充的透明度作為值從 0.0 （不透明） 到 1.0 （清除）。 讀/寫 Double。此屬性的值僅影響均一顏色填充和線條的外觀。對圖案線條及漸變、圖案、圖片和紋理填充的外觀無效。

Type屬性：返回一個**MsoFillType** 值, 該值代表填充類型。

Visible 屬性：返回或設置**MsoTriState** 值，它確定對象是否可見。 讀/寫。

ColorFormat 對象

代表單色對象的顏色、帶有漸變或圖案填充的對象的前景或背景色，或者指針的顏色。

Application 屬性：在不使用對象識別符的情況下使用時, 此屬性返回代表 Microsoft Excel 應用程序的**Application** 對象。 如果與對象識別符一起使用，則此屬性返回代表指定對象的創建者的 Application 對象（可以將此屬性與 OLE 自動化對象一起使用以返回該對象的應用程序）。 只讀。

Brightness 屬性：返回或設置指定對象的發光度。 讀/寫

Creator 屬性：返回一個 32 位整數，該整數指示在其中創建了此對象的應用程序。 只讀 Long。

ObjectThemeColor 屬性：返回或設置一種映射到主題配色方案的顏色。 讀/寫**MsoThemeColorIndex**。

Parent 屬性：返回指定對象的父對象。 只讀。

RGB 屬性：返回或設置一個Long值, 它代表指定顏色的紅-綠-藍值。

SchemeColor 屬性 ：返回或設置一個Integer值, 它代表顏色對象的顏色, 作為當前顏色中的一個索引。

TintAndShade 屬性：返回或設置一個變亮或加深顏色的Single 。

​Type 屬性：返回一個**MsoColorType** 值, 它代表顏色格式類型。
