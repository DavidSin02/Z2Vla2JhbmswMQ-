業餘無線電：新手入門如何選擇設備

2020-10-29
| 科技

很多愛好者剛開始玩無線電的時候不知道如何選擇設備，下面BG8BXM跟大家介紹如何合法使用業餘無線電，以及如何選擇手臺、車臺、天饋等等（個人見解，僅供參考。）

合法使用業餘無線電設備

根據《中華人民共和國無線電管理條例》第二十七條設置、使用無線電臺（站）應當向無線電管理機構申請取得無線電臺執照，第二十九條 申請設置、使用業餘無線電臺的，應當熟悉無線電管理規定，具有相應的操作技術能力，所使用的無線電發射設備應當符合國家標準和國家無線電管理的有關規定。

哪種條件的對講機不需要執照？

2001年12月，信息產業部無線電管理局發佈通知：低功率民用對講機市場全面放開。通知的具體內容是：設置和使用公眾對講機，不需領取電臺執照，免收頻率佔用費。

公眾對講機是指：發射功率不大於0．5W，工作於指定頻率的無線對講機，其無線電發射頻率、功率等射頻技術指標須符合通知要求。

工作頻率（單位：MHz）: 409.7500 ; 409.7625 ; 409.7750 ; 409.7875 ; 409.8000 ; 409.8125 ; 409.8250 ; 409.8375 ; 409.8500 ; 409.8625 ; 409.8750 ; 409.8875 ; 409.9000 ; 409.9125 ; 409.9250 ; 409.9375 ; 409.9500 ; 409.9625 ; 409.9750 ; 409.9875 （共20個頻點）

使用公眾對講機有以下幾點需要特別注意：

1、公眾對講機禁止在機場和飛行器上使用，禁止與公眾電話網、公眾移動通信網互聯。

2、不得擅自更改發射頻率、加大發射功率（包括額外加裝射頻功率放大器），不得擅自外接天線或改用其它發射天線。

3、使用時不得對各種合法無線電業務產生有害干擾，一旦發現有害干擾，應立即停止使用，並採取措施消除干擾後方可繼續使用。

如何選擇設備？

1：購買有核準代碼的電臺：因業餘無線電愛好者使用的設備均需要辦理電臺執照，所以在選購設備時需首選具備核准代碼的電臺。

核准碼查詢網址：www.srrc.org.cn/WP_Search.aspx

具有核準代碼的電臺，在機器串號區域有標註

2：根據自己的經濟能力選擇：有些愛好者一直猶豫是否加入這個愛好群體，是因為有個誤區，以為業餘無線電是“無線墊”，BG8BXM認為其實不是的，看你怎麼玩，想怎麼玩，如果經濟拮据，一個國產手臺1百多塊可以在U/V段中繼上CQ和通聯，也可以在此基礎上自制一個八木天線進行業餘衛星通信或接收空間站的SSTV(慢掃描電視)，覺得手臺功率小，家裡可以買一臺二手的GM300或1907一樣的可以坐檯，短波電臺也是，網上有很多經典的二手機器如725A，FT-80C等等成色好一點的也就1000塊錢，所以想象比現實可怕，只要你想做沒人能阻止你。

這位愛好者的通信車酷吧

如何選擇手臺（手持電臺：對講機）？

手持電臺的種類特別多，功率以2-7w居多，沒有阻擋的條件下直呼距離可以達到3-10公里，如果有一定的高度外接車載或玻璃鋼天線可以達到幾十公里，通過中轉檯（中繼臺）也能完成幾十上百公里的通聯，我覺得在功率上5w就可以了。

一、從置頻方式來講分為寫頻機和手置頻機；

1、寫頻機是固定頻道，以16頻道居多，很多機器沒有液晶屏幕，功能單一，需要通過電腦寫頻，適用於只用固定頻率的愛好者，不適合經常客席異地工作的愛好者，每次出差前先把該區域的頻率先寫一遍，確實不太方便，常規機型價格從100塊到幾千塊不等；

常規寫頻機的控制只有音量開關旋鈕和頻道旋鈕

2、手置頻機是指可以通過電臺自帶的液晶屏幕，對菜單進行頻率及其他功能的設置，可以根據自己的需要馬上進行調整，非常方便，常規機型價格從100塊到幾千塊不等；

最便宜的雙段手置頻機器之一，也有DMR數字版

二、從頻率範圍來講，分為單段機、雙段機或多段全段機；

1、單段機：指的是單一頻率範圍收發的機型，比如29MHz、50MHz、V段144MHz，U段430MHz等等，U、V段單段機在寫頻機中居多，也有手置頻的單段機；

2、雙段機：大多時候指的是支持在V段144MHz，U段430MHz等業餘頻率收發的機器，有單顯單待機的也有雙顯雙待的（雙顯雙待指：兩個頻率待機，監聽兩個頻率）。

FT-4X：不到600塊的雙段單顯雙段手臺

國產歐訊的KG-UV9Dplus在業餘無線電圈口碑不錯，UV雙段雙顯，多頻段接收（含航空頻段，有核準

三、從模式分類來講，分為模擬機和數字機；

1、模擬機：指的是收發模式為FM模式，由於模擬對講機抗干擾低，容易被竊聽，信道資源有限，根據國家的相關管理政策，模擬對講機講逐步被淘汰。由於數字模式各廠家不統一，而且數字機價格稍貴，所以目前市面上銷售的大多還是模擬機。

2、數字機：指的是用數字技術進行設計的數字對講機。數字對講機則是將語音信號數字化，要以數字編碼形式傳播，也就是說，對講機傳輸頻率上的全部調製均為數字。為了使模擬對講機能平穩過度，所以現在很多數字機都兼容模擬模式。但是數字模式各廠家沒有統一，目前市面上常用的數字機模式有：C4FM,D-Star,DMRPlus,dPMR和P25。所以在選購數字機的時候，根據自己地區的應用情況，如果你當地是C4FM的中繼，那麼久選擇C4FM的機型，如果是DMR的，就選擇DMR的機器。

ID-31E U段D-STAR防水GPS數模手臺，價格約1700左右（有核準碼）

P25數字機：XTS5000

四、如果要玩APRS，建議買帶GPS的機型；

FT2D觸摸屏C4FM數模手臺，多頻段接收，帶APRS（有核準碼）

TH-D74，APRS,D-STAR數模雙用手臺

如何選擇車臺（車載電臺）

車載臺U/V段機一般是25瓦~50瓦，車載短波臺5瓦~200瓦都有，價格從幾百塊到幾千塊不等，除了手持電臺的幾種類別，車載臺分為面板一體機和麵板分體機。

面板一體機：面板與機身不能分離，價格相對便宜一點，以單段機居多，在車上安裝不太美觀但適用於作為基地臺，不適合外觀控和強迫症的愛好者。

GM300只有二手貨，不到500塊就能買到25W或45W帶信令版本（無核準碼））

銷量最高的單U段面板一體機之一：FT1907，購買時注意區分仿版（有核準碼）

M8668i數模兩用車臺(有核準碼)

面板分體機：指面板可以和主機進行分離安裝的車載電臺，很多愛好者喜歡將主機裝在駕駛位或副駕駛位的座椅下面，也有些愛好者將主機裝在後備箱，面板通過吸盤或支架裝在中控臺不阻擋視線又方便操作的地方，通過一根數據線將主機與面板連接就行，手咪也可以通過延長線搞定。

如何選擇短波電臺

操作短波電臺必須持有B類或C類操作證書且在國家規定的頻率範圍內進行工作，A類操作證書設臺半年後可以申請參加B類能力驗證。

短波電臺分為基地臺和便攜（車載）臺，原身功率從5W~200W不等，價格也是從1000元至幾萬不等。

一、QRP短波電臺：有專業的QRP機，發射功率不超過5W，當然其他大功率機器可以通過調整功率作為QRP電臺使用。

千元級別的QRP短波電臺

二、車載（便攜）短波：除了前面介紹的IC-7100和FT-857以下機型也是不錯的選擇

協谷G90S 20瓦便攜短波電臺，面板可分離，國外愛好者評價也挺高的（有核準碼）

如何選擇天線

一、手臺天線，長一點的天線肯定比短的好，如果是定製天線，國內愛好者BG7IOO的產品挺不錯的，其他成品天線有拉桿的或鞭狀的，建議不超過40cm。

bg7ioo的手臺定製天線

要進行業餘衛星通信，需要自制或購買八木天線：

玩衛星通信及接收空間站SSTV的，建議使用定向八木天線

二、車載臺天線，根據你的電臺情況，如果是單段機建議購買單段天線效率更高，雙段機就購買雙段天線，如果是短波電臺可以配多波段的天線，不想太張揚也可以配2根單段的車載短波天線（14MHz和7MHz），icom的機器可以用BI7KJB的KB2，挺不錯的。

健博通的單段天線不到100塊

常規的雙段天線以老鷹和新名的居多，再貴一點的就是鑽石的。

性價比高的雙段天線：770，價格不到150

車載移動臺不建議使用U/V段玻璃鋼天線，效率不高。

ICOM車載短波天線價格要1萬多

BI7KJB生產的KB2車載短波天線，使用ICOM或柯頓電臺可以自動調諧

車載電臺天線配合卡片或吸盤使用，行李架安裝可以配自動放倒器，越重的天線對卡片和吸盤的質量要求越高。

三、基地臺U/V段天線：

可以使用車載臺天線替代，但是建議使用玻璃鋼全向天線，長度在2米~4米的最好，國產品牌健博通、華鴻以及任丘天悅的都不錯，也有很多愛好者DIY的。

四、便攜短波天線：

便攜短波天線比較多的選擇，長線天線、倒V天線只要有撐杆或樹幹就可以架設，如果是在空曠地pac-12這款GP天線以及便攜正V天線也是不錯的選擇。

Gipsy吉普賽5-50MHz卷繞式水平偶極天線 DIPOLE便攜短波天線

野外利用撐杆架設倒V天線

五、基地短波天線：

（不管是便攜還是固定臺的天線，很多國內愛好者研發的產品都很不錯，性價比高）

短波天線的種類比較多，常見的有垂直天線（GP），偶極天線（DP）以及八木天線（YAGI），基地短波天線的選擇根據自己的架設條件進行選擇，如果條件允許，建議架設八木天線。

GP天線是垂直的，適合DX通信，但同一架設環境比DP天線的底噪高1-2個點。

K4GP，四波段GP天線，我現在家裡就使用的這款天線，價格1千塊左右

DP天線有很多演變：比如倒V、正V、單極、偏饋等等，基地臺天線架設較多的是正V及倒V天線。

bi7kjb開發的KB2-V4四波段正V天線，價格約900左右，可以固定架設也可野外設臺使用

八木天線屬於定向天線，體積比較大，對架設場地要求高，有很好的方向性，較DP天線有更高的增益，常配合旋轉器使用，價格上也不一

BG7PNV開發的A5DX短波3波段5單元八木天線，價格3千多塊，相比進口八木實惠不少

如何選擇饋線

饋線在業餘無線電系統中相當的重要，選擇好饋線能提高你的收發效率，選擇適當的線徑可以避免駐波太大，導致機器發熱和長時間工作燒燬電路功放系統。

業餘無線電常用的同軸線有50歐、75歐、100歐三種標稱阻抗。 業餘通訊常用50歐，剩下的就是選擇饋線的粗細（-3 -5 -7等），所以在你購買饋線的時候經常看到的規格是50-3，,50-5或50-7等等。

饋線的直徑選擇與頻率、功率與饋線長度有關，比如100W短波設備如果20米內可以使用-3或-5的饋線，如果距離太長建議使用-7的純銅線

50瓦以內UV設備建議使用-5的饋線，距離太長以及中繼臺建議使用饋管。

車載電臺可以使用雙屏蔽的50-3的特氟龍饋線

野外設臺建議購買野外專用的超柔純銅饋線。

八哥電纜銷售的50-3純銅多芯超柔饋線，適合野外使用
