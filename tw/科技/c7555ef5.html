摘要：在自迴歸模型求解中，設計矩陣和觀測值均存在誤差，傳統的最小二乘法不能很好地解決這一問題。本文提出了一種顧及設計矩陣誤差的AR模型新解法，通過引入虛擬觀測值，使觀測向量與設計矩陣不僅同源而且帶誤差的元素個數相同，然後通過對觀測方程進行等價變換巧妙實現了在最小二乘框架下求解自迴歸問題。利用模擬數據及實測數據分別對新算法進行了內符合精度檢驗，並利用實測數據對新算法進行外符合精度檢驗，結果表明新算法得到的結果顯著優於奇異值分解（singular value decomposition，SVD）解法及傳統最小二乘解法，驗證了算法的精度和有效性。

A New Method to Solving AR Model Parameters Considering Random Errors of Design Matrix

YAO Yibin1,2,3, XIONG Zhaohui1, ZHANG Bao1, ZHANG Liang1, KONG Jian4

Abstract: The ordinary least square method could not solve the problem that the error exist both in design matrix and observation vector while compute parameter values of AR model. In this article, a new method is proposed which consider the random errors of design matrix. The source of design matrix and observation vector is same and the amount of parameters contain error can be equal by introducing virtual observation. Then, this problem could be solved under the framework of normal least square by equivalence transformation of observation equation. The result of this new method is superior to SVD method and normal least square method by simulation date and observation data which verify the feasibility and effectiveness of this method.

Key words: AR model design matrix error TLS virtual observations SVD method

平差的實際問題中常存在設計矩陣由含有誤差的觀測值構成，傳統最小二乘方法在求解此類模型的過程中，因忽略設計矩陣中實際存在的誤差而具有理論的不完備性。考慮設計矩陣誤差，將所有觀測值改正數平方和最小作為平差準則的求解方法最初由Adcock於1877年提出[1]。1980年，Golub和Van Loan提出了奇異值分解(SVD)算法，並將這類設計矩陣帶有誤差的問題命名為整體最小二乘方法(total least square, TLS)[2]。使用SVD解法雖能獲得穩定數值解，但計算複雜度高[3]，如果設計矩陣中含有誤差的元素重複出現，使用SVD解法，同一觀測值在不同位置改正數將不一樣。另外，使用SVD解法估計參數，難以給出以觀測值表達的未知參數平差值表達式，從而無法利用協方差傳播率進行精度評定[4]。近些年，文獻[4]提出一種顧及設計矩陣隨機誤差的最小二乘組合新解法，其將設計矩陣中所有含誤差的元素作為虛擬觀測值引入虛擬觀測方程求解模型，解決了TLS方法的精度評定問題。加權整體最小二乘亦有較大進展[5-10]。此外，文獻[11]研究了三維座標轉換的通用整體最小二乘解法；文獻[12]對部分變量誤差模型提出了整體抗差最小二乘估計方法；文獻[13]對Partial EIV模型提出了一種新解法；文獻[14]提出了一種基於中位數法的抗差整體最小二乘估計方法。

1.3 精度評定

在經典最小二乘中，權由觀測值精度確定，而觀測值出現的次數，則會在平差過程中，通過對係數矩陣的作用，進一步影響法方程進而影響平差結果。但是在整體最小二乘中，係數矩陣中的觀測值具有雙重身份，除了影響係數矩陣本身外，還作為觀測值參與平差，本文認為觀測值這部分影響與觀測值在係數矩陣中出現的次數也有關係。本文所定義的權陣P由兩部分構成，即由觀測值精度信息構建的權陣P′，以及一個與次數有關的因子陣K，其中P=P′K。為簡化問題，本文先行假定AR模型各期觀測值等精度，即P′為單位陣，P=K，若觀測值不等精度，則在按照精度信息得到的權陣P′的基礎上再乘以一個次數有關的因子陣K。

由於在AR模型中，設計矩陣中元素在不同位置中規律地重複出現，觀測值亦重複出現於設計矩陣中，在使用整體最小二乘方法進行求解未知參數時，實際上將同一觀測值當作多次相同觀測建模，為此，在各期觀測值等精度的條件下，模型各觀測值權可由觀測值出現的次數確定，考慮引入虛擬觀測值，該算法權陣P定義如下

3 結語

在AR模型中，觀測向量與設計矩陣的誤差同源，觀測值規律地重複出現在設計矩陣中，且設計矩陣中自身元素亦重複出現。傳統最小二乘方法難以解決係數矩陣及觀測值向量皆帶誤差的數據處理問題。本文提出了一種考慮設計矩陣誤差的AR模型整體最小二乘新解法，引入僅在設計矩陣中出現且含誤差的元素作為虛擬觀測值，使觀測向量與設計矩陣中帶誤差的元素個數相同。然後巧妙地對觀測方程進行等價變換，實現了最小二乘框架下求解整體最小二乘問題，有效地克服了傳統SVD方法的理論缺陷且能應用協方差傳播定律給出未知參數的精度評定公式。最後通過對模擬數據及實測數據的驗證，發現本文方法具有比SVD方法及經典最小二乘方法更高的精度及更優的外符合精度。
