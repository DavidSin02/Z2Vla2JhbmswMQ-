GitHub宕機兩小時，眾多軟件開發人員受影響

2020-11-14
| 科技

IT之家 6 月 29 日消息 今天晚上，GitHub 再次出現了宕機事故，剛剛已經恢復正常，宕機時間持續了 2 小時左右。

這家微軟旗下的服務通過 Git 提供版本控制，併為軟件開發提供託管服務，由於軟件開發與該服務緊密相連，GitHub 用戶今晚在登錄該服務時遇到錯誤，一些用戶甚至無法使用其集成開發環境（IDE）。

GitHub 官方在出現問題兩個多小時後才確定錯誤並進行修復，並表示正在調查此次故障。

GitHub 是一個大型代碼存儲庫，很多開發者和公司將整個項目和代碼託管在該服務上，非常受歡迎。蘋果、亞馬遜、谷歌、Facebook 和許多其他大型科技公司都在使用 GitHub。在 GitHub 上託管的倉庫超過 1 億個，有 4000 萬開發者為其做出貢獻。

IT之家瞭解到，微軟早在 2018 年就以 75 億美元的價格收購了 GitHub，並一直在逐步改善對開發者的服務。
