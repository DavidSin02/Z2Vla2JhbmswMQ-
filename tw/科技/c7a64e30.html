使用擴展運算符，讓你的代碼更加簡單優雅

2020-11-14
| 科技

擴展運算符是什麼

它是ES6中引入的一個運算符“...”，它可以把一個數組分割成一列由逗號組成的參數序列，可以被廣泛地用於函數調用，數組複製，合併等情形，下面就來看看使用擴展運算符有哪些更優雅的寫法吧：

更優雅的push操作

let a=[1,2,3];let b=[4];//舊寫法for(let i=0;i<a.length;i++){ b.push(a[i])}console.log(b); // [4,1,2,3]b=[4];// 使用擴展運算符b.push(...a);console.log(b); // [4,1,2,3]

更加方便的數組拷貝

let a=[1,2,3];let b=[];//舊寫法for(let i=0;i<a.length;i++){ b[i]=a[i];}console.log(b); // [1,2,3]// 使用擴展運算符b=[...a];console.log(b); //[1,2,3]

更方便的數組合並

let array1 = [10,20];let array2 = [50,60];let newarray = [25];//舊寫法newarray = newarray.concat(array1).concat(array2); //[ 25, 10, 20, 50, 60 ]console.log(newarray);//使用擴展運算符newarray = [20,...array1,...array2]; //[ 25, 10, 20, 50, 60 ]console.log(newarray);

代替字符串分割 split

let str = "盆盆兒WEB前端";// 使用 splitlet sp = str.split('');console.log(sp);//[ '盆', '盆', '兒', 'W', 'E', 'B', '前', '端' ]// 使用擴展運算符sp = [...str];console.log(sp);//[ '盆', '盆', '兒', 'W', 'E', 'B', '前', '端' ]

實現了 Iterator 接口的對象轉為數組

let nodes = document.querySelectorAll('div'); let array = [...nodes];

可變參數函數的調用

// ES5 的寫法 function f(x, y, z) {// ... }var args = [0, 1, 2];f.apply(null, args);// ES6 的寫法 function f(x, y, z) {// ... }var args = [0, 1, 2];f(...args);
