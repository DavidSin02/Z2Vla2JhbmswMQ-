《公路水運試驗檢測道路工程》第三章第四節第五節知識點

第四節 細集料試驗檢測

1.1篩分試驗

1. 試驗方法一：水泥混凝土用砂篩分（幹篩法）——通過試驗測定水泥混凝土用砂的顆粒級配，並確定砂的粗細程度。

2. 方法一標準套篩：孔徑（單位㎜）9.5/4.75/2.36/1.18/0.60/0.30/0.15/0.075方孔篩及配套底盤；天平稱量1000g，感量不大於0.1g。

3. 試驗方法二：瀝青混合料用砂的篩分（水篩法）——通過水洗方式，精確測出小於0.075㎜的顆粒含量，並測定砂的顆粒級配，以此確定用於瀝青混合料時砂的粒徑分佈。

4. 對水泥混凝土用砂可採用幹篩法，如果需要也可採用水篩法篩分。

1.2細集料的密度試驗——細集料表觀密度試驗（容量瓶法）

1. 試驗目的：通過測定單位砂表觀體積下物質的幹質量，為細集料空隙率和混合料配合比設計提供相關參數。

2. 儀器中天平稱量為1千克，感量不大於0.1克。

3. 烘箱：控溫要求在105±5℃。

4. 試驗中兩次平行試驗結果誤差應在0.010g/cm3的範圍內，若超出應重新取樣進行試驗。

5. 試驗時容量瓶中加入的水應為純淨水，所以需將自來水加熱煮沸後放涼再使用。

6. 試驗中採用的容量瓶也可替換廣口瓶，用於粒徑較大的粗集料表幹密度測定。

1.3細集料毛體積密度及吸水率試驗

1. 試驗目的：通過採用坍落筒法測定處細集料飽和麵幹質量和吸水率，進而得到細集料（包括天然砂、機制砂和石屑）毛體積（相對）密度及其他（相對）密度。

2. 本方法僅適用於小於2.36㎜以下的細集料，大於2.36㎜的顆粒採用粗集料吸水率試驗進行方法。

3. 飽和麵幹狀態的確定是整個試驗難點。

1.4細集料堆積密度及緊裝密度試驗

1. 測定砂在自然狀態下堆積密度和緊裝密度並以此計算出砂的空隙率。

2. 實驗儀器中天平稱量為5千克，感量為0.1克。

3. 堆積密度試驗進行裝填時，要控制好填砂高度，並且試驗過程中避免碰撞容量筒，以免影響砂的實際裝填效果。

4. 容量筒體積校正時要測量水溫，並進行水溫對密度的修正。

1.5細集料含泥量試驗（篩洗法）

1. 試驗目的：通過篩析的方法測定天然砂中粒徑小於0.075㎜的塵屑、淤泥和黏土的含量。

2. 試驗儀器中天平稱量為1千克，感量不大於0.1克。

1.6細集料亞甲藍試驗

1. 試驗目的：用於測定集料中所含膨脹性黏土礦物含量，以評定細集料的潔淨程度。適用於0-2.36㎜的細集料以及小於0.075㎜礦粉等材料的質量檢驗。

2. 當細集料中小於0.075㎜數量小於3%時，可不進行此項試驗，並判定該材料質量合格。

3. 試驗中的亞甲藍試劑（C16H18CN3S·3H2O）純度不小於98.5%。

4. 本試驗檢測依據的原理，是根據膨脹性黏土礦物能夠吸附亞甲藍試劑，從而使滴在濾紙上的懸濁液，在其邊緣無法產生亞甲藍試劑本身的淺藍色色暈。

5. 亞甲藍試驗適應較細的集料，集料粒徑不適宜於大於4.75㎜。

6. 由於膨脹性黏土吸附亞甲藍需要一定的時間才能完成，色暈可能在出現後又會消失。為此，需要每隔1min進行一次色暈試驗，直至出現的色暈可持續5min未消失為止。

1.7細集料砂當量試驗

1. 試驗目的：通過測定天然砂、人工砂、石屑等各種細集料中所含黏土或雜質的含量，以評價細集料的潔淨程度。

2. 砂當量越大，說明砂中小於0.75㎜的顆粒中黏性土所佔的數量越少，對應砂的潔淨度越高，則砂的品質越好。

第五節 礦質混合料組成設計

1.1級配類型和級配曲線

1. 集料即砂石材料，通常大多是與各種結合料（如水泥、瀝青等）組成混合料的形式用於土木工程，此時砂石材料往往要採用一定的級配形式，以保證混合料能夠更好地滿足實際工程需要。

2. 將幾種不同規格的集料按一定的方法進行摻配即級配設計，使其組成符合級配要求的礦質混合料（簡稱礦料）。

3. 粒徑粗細不同的礦料按照一定的比例組合搭配在一起，根據搭配組成的結果，可得到以下幾種不同級配形式：連續級配——連續級配是指礦料的顆粒由大到小連續分佈，每一級都佔有適當的比例；間斷級配——在礦料顆粒分佈的整個區間裡，從中間剔除一個或連續幾個粒級，形成一種不連續的級配，稱為間斷級配；開級配——整個礦料顆粒分佈範圍較窄，從最大粒徑到最小粒徑僅在數個粒級上（大多集中於偏粗的粒徑）以連續的形式出現，形成所謂的開級配。

1.2礦料級配理論

1. 實踐中，針對連續級配各級粒徑礦料數量的計算大多采用最大密度曲線理論，該理論認為當礦料的顆粒級配曲線愈接近拋物線，則其密實度愈大。

1.3礦料組成設計方法

1. 實際工程中往往要採用至少兩種以上規格的集料按一定比例配合起來使用。

2. 級配設計常用的方法有試算法和圖解法兩類。

3. 試算法：適用於採用三種不同規格集料進行級配設計的情形。首先對所使用的各集料進行篩分，並計算出各自的篩分結果。同時，明確設計級配要求的級配範圍，並計算出該級配範圍對應的中值。

4. 圖解法對級配設計時，所適用的原材規格樹木不限。
