吊索具的安全使用、檢驗和報廢標準

2020-10-29
| 科技


吊裝機械共享信息平臺 重機匯APP


吊索具種類

索具：鋼絲繩、吊帶（扁平吊帶、圓型吊帶）、吊鏈

索具配件：吊環（吊環螺釘、吊環螺母）、吊鉤、卸扣



吊索具的安全使用、檢驗和報廢標準

1. 無標識和檢驗證書或報告的吊索，嚴禁投入使用。

2. 使用前應進行日常檢查，發現鋼絲繩或金屬附件的損壞超過報廢標準時嚴禁投入使用。（鋼絲繩的報廢標準詳見下文）

3. 嚴禁超載荷使用。

4. 採用穿套式結索吊裝，繩套的自然夾角一般為120°。嚴禁重物吊起後以任何方式（如敲擊等）改變鋼絲繩的捆綁狀態。以外力強行改變繩套狀態，會使繩套夾角大於120°，將造成鋼絲繩局部過載。

5. 特殊裝置鏈條、吊鉤、吊環等金屬附件的吊索，相互間應用卸扣連接，其額定載荷應相等。

6. 起吊要平穩，並應避免衝擊載荷的作用。

7. 採取有效措施避免鋼絲繩在銳角處發生彎折或與載荷碰撞。

8. 禁止拖拽或從高空向下摔扔鋼絲繩吊索。

9. 不得在超過-40℃到100℃溫度範圍處使用。

10. 不得在腐蝕性的氣體、液體或蒸汽中使用。

11. 停用半年以上（包括半年）後又重新使用的鋼絲繩，使用前應進行檢驗，檢驗合格後方可使用，對貯存期超過二年的鋼絲繩索，必須按國家標準抽樣進行拉力試驗，合格後方可投入使用。



鋼絲繩吊索的安全檢查

1. 斷絲情況——斷絲根數及斷絲分佈狀況和程度。

2. 磨損程度——測量各磨損部位鋼絲繩直徑。

3. 腐蝕情況——是否生鏽，腐蝕狀況。

4. 潤滑狀態——油脂是否適當存在，油脂上有無附著雜物，保持表面清潔，每年要用鋼絲繩潤滑油潤滑一次。

5. 變形和其它異常現象——扭結痕跡、壓扁、損傷、鬆股或鬆捻的程度及位置。

6. 重型套環連接部位的緊股狀況——鋼絲繩錯動或合金壓頭是否發生變形、磨損、腐蝕及裂紋。

鋼絲繩吊索的報廢標準（出現以下之一均應進行報廢處理）

1. 重型套環連接部位斷絲為總絲數的5%；

2. 在一個控距內斷絲數為總絲數的13%；

3. 繩股斷裂。鋼絲繩相對公稱直徑減少10%以上；

4. 出現波浪形變形時，在鋼絲繩子長度不超過25d的範圍內，d1大於等於4d/3，式中：d—鋼絲繩公稱直徑，d1—鋼絲繩變形後包絡面的直徑；

5. 鋼絲繩被壓扁；

6. 扭結、彎折；

7. 鋼絲繩嚴重鏽蝕；

8. 重型套環連接部件，鋼絲繩鬆脫或錯動；

9. 壓制接頭損傷、變形、裂紋和嚴重腐蝕；

10. 鋼絲繩吊索的金屬附件報廢標準，按其國家標準的有關規定執行；

11. 吊裝作業使用頻繁疲勞的鋼絲繩索具，使用兩年後應強制報廢。



吊帶的安全使用

1. 吊帶在工作時，不準拖拉，以防損壞吊帶；

2. 吊帶不準確性打結使用，承載時不準確性轉動貨物使吊帶打擰；

3. 不要使用沒有護套的吊帶吊裝有尖角、稜邊的貨物，以防損傷吊帶；

4. 不允許長時間懸吊貨物；

5. 不要把吊帶存放在有明火或其它熱源附近，也應注意避光保存。

吊帶的檢查檢驗

1. 有無穿孔、切口、撕斷；

2. 有無接縫綻開、縫線磨斷；

3. 是否有軟化、老化彈性變小、強度減弱現象；

4. 纖維表面是否粗糙易於剝落；

5. 吊帶有無出現死結；

6. 吊帶表面是否存在點狀疏鬆、腐蝕、酸鹼燒損以及熱熔化或燒焦；

7. 帶有紅色警戒線吊帶的警戒線是否裸露。

吊帶的報廢標準

1. 穿孔切口、撕斷；

2. 承載接縫綻開、縫線磨斷；

3. 吊帶纖維軟化、老化、彈性變小、強度減弱；

4. 纖維表面粗糙易於剝落；

5. 吊帶出現死結；

6. 吊帶表面有過多的點狀疏鬆、腐蝕、酸鹼燒損以及熱熔化或燒焦；

7. 帶有紅色警戒線吊帶的警戒線裸露；

8. 經常疲勞使用無損1年需報廢。



吊鏈的安全使用

1. 操作吊鏈時，吊鏈的名義質量等級與所用鏈條等級一致；

2. 鏈式吊索使用前，應先用單肢極限工作載荷的2倍進行靜載試驗，試驗時懸掛10分鐘，無破損或個別鏈環沒有明顯伸長，合格後可投入使用；

3. 多肢吊鏈吊掛重物時，對重物加緊力隨吊鏈分肢間夾角的增大而增大，當重物強度和韌性較低而不能承受過大的加緊力時，應儘量減小吊鏈分肢間夾角，避免過大的加緊力對被吊物造成破環；

4. 吊裝輔助組件的級別不得低於圓環鏈級別；

5. 鏈條之間禁止採用非正規連接件連接；

6. 重物應掛在吊鉤鉤槽最底部，切不可掛在鉤尖出處，一般情況下起吊重物時鉤口應向外；

7. 承載圓環鏈禁止直接掛在起重機械吊鉤的鉤尖上或纏繞在吊鉤上；

8. 起吊重物時升、降、停要緩慢平穩，注意被吊物重心平衡，避免衝擊載荷，不得長時間將重物懸掛在吊鏈上；

9. 不要試圖從重物下強行拉出吊鏈，或讓重物在吊鏈上滾動；

10. 卸載後空載提升的吊鏈和多肢承載吊鏈的空載分肢，下端部應反掛到起重機吊鉤上或上端部的配件上；

11. 吊鏈要輕拿輕放，嚴禁摔打、扔碰和在地上拖拽；要防止從高處墜落，以免造成吊鏈變形、表面和內部損傷；

12. 吊鏈使用後應做清潔和相應檢測，無異常應將其放置在專用支架上，不得放在地面上；主環擱置在架託上，下端輔具反掛在吊鏈上端部輔具或支架上；起重吊裝圓環鏈不得堆壓存放在地上，應懸掛或放在貨架上。所有吊具存放要清潔。乾燥後塗防鏽油。存放地點要通風、乾燥、無腐蝕氣氛。

吊鏈的檢查檢驗

1. 鏈環是否發生塑性變形；

2. 吊索上任何部位不得有裂紋、裂縫、明顯鏽蝕等對使用有害的缺陷；

3. 鏈條、中間環、連接環應靈活轉動；

4. 鏈環之間隔以及鏈環與端部配件連接接觸部位磨損，其它部位磨損情況；

5. 檢驗必須由專業技術人員進行。

吊鏈的報廢標準

1. 鏈環發生塑性變形伸長達原長度的5%；

2. 鏈環之間以及鏈環與端部配件連接接觸部位磨損減小到原公稱直徑的60%，其它部位磨損減少到原公稱直徑的90%；

3. 裂紋或高拉應力區的深凹痕，銳利橫向凹痕；

4. 鏈環修復後，未能平滑過渡或直徑減少量大於原公稱直徑的10%；

5. 扭曲、嚴重鏽蝕以後積垢不能加以排除；

6. 端部配件的危險斷面磨損減少量達原尺寸10%；

7. 有開口度的端部配件，開口度比原尺寸增加10%；

8. 進行外觀檢驗、尺寸測量、拉伸試驗、NDT檢驗時發現超標缺陷。



吊索配件

一、吊環

吊環的安全使用

1. 用於組成鏈式吊索、鋼絲繩吊索、纖維繩吊索的主環宜採用長形環，其它場合可用圓形環；

2. 連接環必須採用長形環；

3. 出現傷痕或顯著鏽蝕的吊環再利用，必須按要求進行靜拉力試驗；

4. 不允許將有缺陷的吊環焊補後重新使用；

5. 起重索具組合部件上的吊環，按組合部件要定期檢查；

6. 與鏈條連接的連接環，其環材直徑必須大於鏈材直徑；

7. 與單隻吊環直接連接的鏈、索數不得多於三肢，四肢鏈、索與主環之間的連接應採滿足要求；

8. 不得采用錘擊的方法糾正已扭曲的吊環；

9. 禁止拋擲吊環；

10. 不要從重物下面拉拽吊環或讓重物在吊環上滾動；

11. 不準用卸扣代替連接環。

吊環的檢查檢驗

1. 吊環表面情況；

2. 吊環磨損情況；

3. 吊環扭轉變形的情況；

4. 起吊5噸以上大型索具使用的吊環每年要進行檢驗和無損探傷。

吊環的報廢標準

1. 從吊環不彎曲的平面算起，扭曲超過10%；

2. 長形環內長L（圓形環內徑D）變形率達5%以上；

3. 吊環直徑磨損或鏽蝕超過名義尺寸10%；

4. 吊環上出現裂紋、裂痕或凹槽。

二、吊鉤

吊鉤的安全使用

1. 起吊的貨物重量不得超過該吊鉤的額定載重量；

2. 起吊時應平穩，避免出現衝出載荷；

3. 載荷應置於吊鉤鉤座的中部；

4. 裝載腐蝕品後，應將殘留在吊鉤上的腐蝕品及時清洗掉，然後擦乾上油。

吊鉤的檢查檢驗

1. 吊鉤表面情況；

2. 吊鉤開口度增大情況；

3. 吊鉤扭轉變形的情況；

4. 吊鉤鉤座磨損情況；

5. 吊鉤頸部或危險斷面；

6. 帶有附件的吊鉤，附件的工作情況；

7. 起吊5噸以上大型索具使用的吊鉤每年要進行檢驗和無損探傷。

吊鉤的報廢標準

1. 吊鉤的開口度超過公稱尺寸的10%；

2. 吊鉤的扭轉彎形超過10%；

3. 危險斷面磨損達原尺寸的10%；

4. 吊鉤的頸部或危險截面發生塑性變形；

5. 吊鉤表面出現無法修理的裂紋、裂痕；

6. 吊鉤襯套磨損達原尺寸的50%，應報廢襯套；

7. 鉤心軸磨損達原尺寸的5%時，應報廢心軸。

三、卸扣

卸扣的安全使用

1. 嚴禁超負荷使用；

2. 應注意作用在卸扣的受力方向，不準橫向垂直受力；

3. 安裝橫銷軸螺紋旋足後，應迴轉扣螺距，不準敲擊螺紋部位；

4. 嚴禁用其它材料的螺栓取代卸扣配套螺栓。

卸扣的檢查檢驗

1. 外形尺寸：由檢驗員測量卸扣各種部件的外形尺寸，做好記錄；

2. 外觀：外觀檢測用倍率為5倍的放大鏡對卸扣表面進行目檢，檢查有無裂紋、變形、磨損現象，做好記錄；

3. 變形：通過測量和目檢檢查確定卸扣是否有變形情況存在，做好記錄；

4. 磨損：通過測量確定卸扣各個部件的磨損情況，尤其是卸扣頂部，做好記錄；

5. 危險斷面：檢查受力部件斷面的磨損及變形情況，做好記錄；

6. 探傷：對主要受力部件的危險斷面進行磁粉探傷，做好記錄；

7. 起吊5噸以上大型索具使用的卸扣每年要進行檢驗和無損探傷。

卸扣的報廢標準

1. 扣體長度：卸扣長度的實測尺寸超過公稱尺寸的0.25%或0.5mm兩者之較大者；

2. 變形：卸扣的扭轉變形超過10%；

3. 腐蝕：卸扣部件的磨損量超過公稱尺寸的5%；

4. 危險斷面：危險斷面發生塑性變形；

5. 表面：卸扣的表面出現無法修理的裂紋、裂痕；

6. 探傷：對主要受力部件的危險斷面進行磁粉探傷發現內部有缺陷；

7. 標識：沒有表明額定載荷的。
