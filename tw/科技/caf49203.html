免費開源資源管理器——RX 文件管理器

2020-11-14
| 科技


在前幾天給大家安利一款免費開源資源管理器 Files UWP，當時推薦這個的主要原因很簡單：

資源管理器是每天都會使用的高頻應用，而 Windows 自帶的常年沒有大的變化，所以一旦有新冒出來的第三方資源管理器我還是很願意嘗試的。

雖然有可圈可點之處，但是有小夥伴根據自己的需要指出了 2 個硬傷：不能 Win+E 快捷鍵打開和不能拖拽文件到其它文件夾。



於是有其他小夥伴非常熱心的貢獻了另一款更好用的同類工具：RX 文件管理器。



實話實說，這個管理器我確實之前也沒聽過，本來以為自己對各種新軟件瞭解得比較多了，但這件事讓我覺得哪怕你每天窮盡時間去找，依然還是會錯過一些優秀工具，誰都會有覆蓋不到的死角。

沒錯，這就是我試用體驗了大家推薦的這個 RX 文件管理器之後，收穫的最大感受。

如果我先瞭解到 RX 文件管理器的話，一定不會再介紹 Files uwp。

這個 RX 文件管理不僅在功能、顏值、穩定性上都超過了 Files uwp，而且上次小夥伴們提到的那些問題，在它這裡全都可以解決。

那麼，我們就再來挑戰一下你用了十幾年的「我的電腦」，看看這款軟件能不能挑戰成功。

RX 文件管理器

先一起來看看 Files UWP 的硬傷在它身上有沒有：

快捷鍵打開

Files UWP 不支持 Win+E 快捷鍵打開？RX 文件管理器就可以，當然這得事先在設置裡勾選才行。



支持拖拽

‍相比於 Files UWP 不能拖拽文件到其它文件夾，RX 文件管理器就能做到，不得不說，這樣確實方便不少。



顏值高

解決了硬傷，來看一下它的顏值，它可以是這樣的半透明，也可以是傳統的白色或黑色風格。



它還可以是這樣的自定義背景：



你只用在設置裡-模式選擇自定義進行設置就可以了，如果沒有自己心儀的圖片，你也可以選必應的每日壁紙作為背景。



多標籤功能

再來看看這個剛需的多標籤功能。

在 RX 文件管理器中選擇文件夾後點擊鼠標右鍵即可選擇在新標籤頁中打開、在新窗口中打開文件夾。

多標籤來回切換還是非常流暢的。

同時它還支持把文件夾固定主界面，和 Files UWP 把經常訪問的文件夾固定在側邊欄是一個道理。



右鍵功能

相比於 Files UWP 點擊文件夾右鍵沒有壓縮功能，在 RX 文件管理器中點擊文件夾右鍵是有壓縮功能的。



不過默認是 zip 格式的。



但是安裝了 Bandizip，還是同樣支持在文件內雙擊解壓文件。



右鍵點擊文件後出現的菜單項也比 Files UWP 要豐富不少。



這裡要說一下，RX 也支持右鍵顯示 MD5 值，上次文章對 MD5 的解釋有誤，感謝小夥伴在留言中指正。MD5 值確實不是隨機的，而是唯一的，也正是因為是唯一數值，才用來檢驗文件傳輸的一致性，可以理解為「文件指紋」。

大家常在一些網站的下載區見到提供的 MD5 值就是為了讓你檢驗你下載的文件是否和上傳者的文件一致。

此外，RX 文件管理器同樣支持 Quicklook 預覽，這一點自然也少不了。

快速啟動

而且 RX 文件管理器除了資源管理器的看家本領外，還可以快速啟動相關應用。比如微軟商店、設置什麼的。



如果這幾個滿足不了你的需要？那就點那個加號自己造吧。

點擊加號之後，手動添加快速啟動圖標，啟動協議（命令）、名稱。

其實這裡的快速啟動和直接使用 Win+R 調出使用命令打開 UWP 應用一個道理。

啟動協議不知道的話可以點擊進入那個箭頭所指的 UWP 應用啟動命令大全鏈接，裡面是比較全的。

只是主要就是微軟自身的應用和設置啟動協議。



記得啟動協議不能少了最後那個英文冒號。



設置好之後就能快速打開了。



眼神好的小夥伴肯定也看到了下面那個熱門網站的快捷設置。



還是剛才設置那個道理，網站後面記得加那個英文冒號。

這個網站什麼效果，你們肯定比我懂得多，具體我就不演示了，你們自己獨自享樂吧。



下載安裝

在微軟商店中直接搜索 RX 文件管理器就可下載安裝，它有一個免費試用版，還有一個收費版，15 元。



收費版只比免費試用版多了一個安全域加密文件的功能，其它功能都是一樣的。



而且免費試用版也沒有標註使用時間，也就是說，雖然說是試用版，但其實你可以一直用下去，開發者確實很良心，個人建議使用免費試用版就足夠了。

結語

說實話，挑戰大家用了好多年的「我的電腦」，也就是資源管理器的確是需要勇氣的，不管你是鼠標黨左鍵雙擊左上角，還是鍵盤黨 Win+E 快速呼出，這幾乎都成了下意識的肌肉反應。

所以第三方的資源管理器很少能流行開來，因為這確實是一件吃力不討好的事情，不過對於我們喜歡嚐鮮的軟件愛好者來說，面對那好多年如一日的界面，難免不會審美疲勞，也難免不會喜新厭舊。

始於顏值，陷於才華。初上手你會被它的高顏值和過場動畫所吸引，體驗後，那些伴隨你多年的使用習慣也能在上面找到，不至於無所適從。

如果你想嘗試一款新的文件管理器，那麼這款 RX 文件管理器的確比 Files UWP 是個更好的選擇。
