算法

算法(Algorithm)是一系列解決問題的清晰指令。通過算法我們可以高效的解決一些複雜的問題。算法對於程序員十分重要，如果你不懂算法，那麼你只能是半生都是普通程序員，而能夠熟練掌握算法的人才是晉升的對象，才是在編程世界裡遊刃有餘的人。李開復先生也說過：“算法是計算機科學領域最重要的基石之一，但卻受到了國內一些程序員的冷落。許多學生看到一些公司在招聘時要求的編程語言五花八門就產生了一種誤解，認為學計算機就是學各種編程語言，或者認為，學習最新的語言、技術、標準就是最好的鋪路方法。其實大家都被這些公司誤導了。編程語言雖然該學，但是學習計算機算法和理論更重要，因為計算機算法和理論更重要，因為計算機語言和開發平臺日新月異，但萬變不離其宗的是那些算法和理論，例如數據結構、算法、編譯原理、計算機體系結構、關係型數據庫原理等等。在“開復學生網”上，有位同學生動地把這些基礎課程比擬為“內功”，把新的語言、技術、標準比擬為“外功”。整天趕時髦的人最後只懂得招式，沒有功力，是不可能成為高手的。”所以我們可以看出算法，數據結構等等是有多麼重要，要想在計算機界吃上一口熱飯，就必須要掌握算法。至少我們要掌握最基本的算法，如分治法，貪婪算法，回溯法，動態規劃等等。現在我們就初步瞭解一下算法之一的分治策略。

學過編程的肯定聽說過查找中的二分法，排序中的歸併法，快速排序，這些都是蘊含有分治思想的算法。

分治法的基本思想

分治法的基本思想：

1.大問題分解為子問題，這些子問題互相獨立且與原問題相同。

2.分別求解子問題。如果子問題的規模不夠小，那就再繼續往下劃分，一直劃分下去，直到問題的規模足夠小，很容易求出問題的解為止。

3.合併解，自底向上求出原來問題的解。

分治法的設計思想就是：將一個難以解決的大問題，分割成一些規模較小的相同問題，以便各個擊破，分而治之。

分治法的適用條件：

1.問題的規模縮小到一定程度就可以很容易地解決；

2.問題具有最有子結構，當問題的最優解包含其子問題的最優解時，稱該問題就有最優子結構性質；

3.分解出來的問題能夠合併成該問題的解；

4.問題所分解出來的各個子問題是相互獨立的，即子問題之間不包含公共的子問題。

如果各個子問題不是獨立的，使用分治法要重複的解很多公共子問題，此時不如使用動態規劃。

分治法的應用

二分搜索算法：

在一個按升序排序的數組a1，a2，a3......，an，判定某個給定元素x是否在數組中出現，並找出位置。

對於這個問題最簡單明瞭的辦法就是將數組遍歷一遍，依次對比是否相等。這種方法是最常用的方法，我們可以直到這種方法的時間複雜度為O（n）；然而如果使用二分法，時間複雜度就會大大降低。

二分法的思路就是，先讓x和數組中間的數比較如果x比較大，那麼與之相等的數肯定在數組後半段，然後讓x再和後半段的中間數比較，進而可以確定x在更小的區間裡，再取此區間的中間數比較，這樣循環就可以找到與x相等的數。這樣我們可以思考一下此算法的時間複雜度，假設最壞情況下我們查找了k個數，數組總共有n個數，那麼可以知道2的k次方等於n，所以k等於log（n），以2為底，所以時間複雜度為log（n），其中log以2為底。

歸併排序：

歸併排序是將待排序的元素分成大小大致相等的2個子集合；分別對2個子集合進行排序（遞歸）；最終將排好序的子集合合併成為所要求的排好序的集合。

歸併排序的主要流程

歸併排序的算法描述：

#include<iostream>using namespace std;void merge_sort(int[],int ,int);void merge(int[],int ,int ,int );int main(){	int a[9]={9,8,7,6,5,4,3,2,1};	merge_sort(a,0,8); for(int i=0;i<9;i++) { 	cout<<a[i];	} }  void merge_sort(int data[],int left,int right) { 	if(left<right) 	{ 		int mid=(left+right)/2; 		merge_sort(data,left,mid);//遞歸  		merge_sort(data,mid+1,right);//遞歸  		merge(data,left,mid,right);//對數組進行排序 	 } } void merge(int array[],int p,int q,int r) { 	int i,k;int begin1,begin2,end1,end2;//begin1，end1，begin2，end2，分別為第一個序列的起始點，結束點，第二個序列的起始點，結束點  	int *temp=new int[r-p+1];//temp用來存放合併後的序列；  	begin1=p;end1=q;begin2=q+1;end2=r; 	k=0; 	while(begin1<=end1&&begin2<=end2) 	{ 		if(array[begin1]<array[begin2]) 		{ 			temp[k]=array[begin1];begin1++;		 }		 else {		 	temp[k]=array[begin2];begin2++;		 }		 k++;	 } 	while(begin1<=end1)//如果第一個序列有剩餘，則放在序列尾部  	{ 		temp[k]=array[begin1]; 		begin1++;k++;	 }	 while(begin2<=end2)//如果第二個序列有剩餘則放在尾部  	{ 		temp[k]=array[begin2]; 		begin2++;k++;	 }	 for(int i=0;i<r-p+1;i++)//將合併後的序列temp賦值給array 	 {	 	array[p+i]=temp[i];	 }	 delete [](temp);//釋放temp的內存

此程序輸出結果為123456789。可以看到其中使用了遞歸。

複雜度上為O（nlogn）。這種方法相比於堆排序和快速排序的最大特點是它是一種穩定的排序方法。

快速排序：

快速排序是對冒泡排序的改進算法。冒泡排序就是依次遍歷整個數組，找出最大的放在最後一位，然後再遍歷，把次大的放在倒數第二位，就這樣遍歷n-1遍才能把n個數排好序。其時間複雜度為（n*n）。而快速排序則要比冒泡排序更加高效快速。其基本思想是通過一趟排序將要排序的數據分割成獨立的兩部分，其中一部分的所有數據都要比另外一部分的所有數據小，然後再按此方法對這兩部分分別進行快速排序，整個排序過程可以遞歸進行，以此達到整個數據變成有序序列。

總結為一點就是挖坑填數。

快速排序步驟：

1.設置兩個變量left=0，right=N-1.

2.以第0個數為基準，作為一個坑。

3.從後向前找比坑小的數，right開始減小，找到後交換兩者的位置，此時坑就到了被找到的數的位置，

4.從前向後找比坑大的數，left開始增大，找到後並交換位置，此時坑又到了前邊。

5.重複3.4.步驟直到left和right相碰。

代碼：int main(){	int array[9]={9,8,7,6,5,4,3,2,1};	quicksort(array,0,8);	for(int i=0;i<9;i++)	{		cout<<array[i];	}}void quicksort(int number[],int left,int right){	int q;	if(left<right)	{		q=partition(number,left,right);//將數組分成兩部分，並返回作為基準的數的位置。 		quicksort(number,left,q-1);//對左半部分進行遞歸； 		quicksort(number,q+1,right);//對右半部分進行遞歸 	}}int partition(int a[],int left,int right){	int i=left;	int j=right;	int s=a[i];	while(i<j)	{		while(i<j&&a[j]>=s)j--;		if(i<j) a[i]=a[j];		while(i<j&&a[i]<=s)i++;		if(i<j) a[j]=a[i];	}	a[i]=s;	return i;}

快速排序的時間複雜度為O(n*logn)，要優於冒泡排序。快速排序是很重要的排序方法。

許多問題都可以通過分治法進行解決，通過分治和遞歸的結合，能夠大大簡化問題的思考量和複雜度。
