這些職業，短缺！人社部發布排行榜

2020-11-14
| 科技

從昨天開始到今早，

#100個短缺職業排行#一直佔據熱搜

引發熱議

據人社部官網發佈的消息，近日，中國就業培訓技術指導中心發佈102個定點監測城市公共就業服務機構人力資源市場《2019年第三季度全國招聘求職100個短缺職業排行》。

這是全國層面首次彙總發佈人力資源市場招聘求職短缺職業排行。

最短缺的100個職業

按類別來分

42個屬於社會生產服務和生活服務人員

36個生產製造及有關人員

17個屬於專業技術人員

4個屬於辦事人員和有關人員

1個屬於黨的機關、國家機關、群眾團體和社會組織、企事業單位負責人

具體排名如下

（“序號”代表排序關係，排在前面的意味著短缺程度高於排在後面的職業。）

序號 / 職業名稱 / 職業定義

1 / 營銷員 / 從事市場調查、商品與服務推銷工作的人員。

2 / 收銀員 / 在消費場所，從事顧客現金、支票收取或金融卡支付辦理等工作的人員。

3 / 餐廳服務員 / 在餐飲場所中，安排顧客座位，點配菜點，進行宴會設計、裝飾、佈置等就餐服務的人員。

4 / 保安員 / 為公民、法人和其他組織提供安全防範服務的人員。

5 / 保潔員 / 從事公共場所、公共區域、公共設施、公共水域的水面以及建築物內外清潔、維護工作的人員。

6 / 商品營業員 / 在零售實體店中，進行商品整理、銷售、管理，併為客戶提供服務的人員。

7 / 家政服務員 / 從事家務料理、家庭成員照護、家庭事務管理等工作的人員。

8 / 車工 / 操作車床，進行工件旋轉表面切削加工的人員。

9 / 焊工 / 操作焊機或焊接設備，焊接金屬工件的人員。

10 / 裝卸搬運工 / 使用工具、機具，進行車船貨物裝卸、堆垛、入庫、出庫等作業的人員。

11 / 包裝工 / 使用包裝設備或手工，使用包裝材料，將包裝物品裝入包裝容器和封緘、包紮包裝物品的人員。

12 / 房地產經紀人 / 在房地產交易活動中，提供房源、客源及房地產交易等信息，協助實地查看房地產，代擬交易合同、代辦貸款和不動產登記手續等服務工作的人員。

13 / 保險代理人 / 從事保險業務代理、保險產品推介，協助客戶辦理保險業務手續的人員。

14 / 快遞員 / 從事快件攬收、派送和客戶信息收集、關係維護及業務推廣工作的人員。

15 / 眼鏡驗光員 / 使用驗光儀器及輔助設備，檢查視力、眼睛屈光度及融像機能，開具眼鏡驗光處方並指導視覺康復訓練的人員。

16 / 保育員 / 在托幼園所、社會福利機構及其他保育機構中，從事兒童基本生活照料、保健、自理能力培養和輔助教育工作的人員。

17 / 電工 / 使用工具、量具和儀器、儀表，安裝、調試與維護、修理機械設備電氣部分和電氣系統線路及器件的人員。

18 / 機械設備安裝工 / 使用工具、機具、檢測器具等，安裝、調試機械設備、生產線和靜置設備，以及製作、安裝非標準設備的人員。

19 / 育嬰員 / 在0～3歲嬰幼兒家庭從事嬰幼兒日常生活照料、護理和輔助早期成長的人員。

20 / 計算機軟件工程技術人員 / 從事計算機軟件研究、需求分析、設計、測試、維護和管理的工程技術人員。

21 / 質檢員 / 使用設備、工具或運用感官，檢驗、檢查、確定原料、燃料、材料和半成品、成品等產品質量的人員。

22 / 養老護理員 / 從事老年人生活照料、護理服務工作的人員。

23 / 縫紉工 / 使用縫紉設備或工具，將紡織品、皮革等主、輔料裁片縫製成半成品或成品的人員。

24 / 汽車生產線操作工 / 操作、調整汽車塗裝、焊裝、衝壓、機加、熱處理、鍛造、鑄造等生產線設備、工裝，加工汽車零部件的人員。

25 / 客戶服務管理員 / 在企業中，從事售前、售中、售後客戶服務活動管理工作的人員。

26 / 客房服務員 / 在飯店、賓館、旅遊客船等住宿場所，清潔和整理客房，並提供賓客迎送、住宿等服務的人員。

27 / 企業經理 / 在企業中，經董事會或出資人聘任，或經職代會選舉、或經上級任命的企業負責人。

28 / 行政辦事員 / 在公共管理和社會組織機構中，從事具體行政業務辦理，以及基層人民政府和派出機構中，從事司法助理、民政助理等行政業務的人員。

29 / 會計專業人員 / 從事國家機關、社會團體、企事業單位和其他經濟組織會計核算和會計監督的專業人員。

30 / 多工序數控機床操作調整工 / 操作數控機加生產線、數控組合機床、複合機床和加工中心等，進行工件切削加工的人員。

31 / 水產品加工工 / 使用工具和設備，醃製、乾製、烘烤、蒸煮水產品的人員。

32 / 道路客運汽車駕駛員 / 駕駛客運機動車，運送乘客並提供服務的人員。

33 / 前廳服務員 / 為賓客提供諮詢、迎送、入住登記、結賬等前廳服務的人員。

34 / 市場營銷專業人員 / 從事市場分析、產品宣傳促銷、生產經營決策諮詢服務的專業人員。

35 / 中式烹調師 / 運用刀法與烹調技法，對原材料進行加工，製作中式菜餚的人員。

36 / 工程機械維修工 / 使用檢測儀器、檢修機具和診斷設備等，進行工程機械主機、總成件及主要零配件診斷、維修、試車和保養的人員。

37 / 祕書 / 從事文書、會議辦理，併為各類辦公事務處理提供服務的人員。

38 / 農產品購銷員 / 從事農產品的採購、初加工、運輸、保鮮儲藏、銷售及相關信息服務工作的人員。

39 / 倉儲管理員 / 從事倉儲物品出入庫、儲存、賬務等管理工作的人員。

40 / 廣告設計師 / 從事廣告創意、宣傳、形象設計的人員。

41 / 銀行客戶業務員 / 從事銀行客戶理財、轉賬、兌現及相關服務工作的人員。

42 / 物業管理員 / 在物業管理區域內，組織安排物業設施的維修養護、環境衛生的美化綠化、公共秩序的維護等工作的人員。

43 / 計算機程序設計員 / 從事計算機和移動終端應用程序設計、編制工作的人員。

44 / 機修鉗工 / 使用工具、量具和儀器、儀表，維護和修理設備機械部分的人員。

45 / 裝配鉗工 / 使用機械或工裝、工具，進行機械設備產品部件、組件或成品組合裝配與調試的人員。

46 / 保衛管理員 / 從事維護機關、團體、企業、事業單位內部治安秩序，開展治安防範，預防違法犯罪，保護單位內部人員和財產安全工作的人員。

47 / 後勤管理員 / 在國家機關、社會團體、企業和事業單位中，從事單位內部的水、電、氣、供暖等後勤事務管理工作的人員。

48 / 旅店服務員 / 在旅店、旅社、民居客棧、公寓、家庭旅館等場所，為賓客提供諮詢、住宿、客房、餐飲、結算、代辦等服務的人員。

49 / 衝壓工 / 操作衝壓、鈑金、折彎、卷板等設備，加工處理金屬板材、型材、管材及線材等的人員。

50 / 電子商務師 / 在互聯網及現代信息技術平臺上，從事商務活動的人員。

51 / 道路貨運汽車駕駛員 / 駕駛貨運機動車，運輸貨物並提供服務的人員。

52 / 美容師 / 從事顧客面部護理、身體護理和美化修飾容顏的人員。

53 / 機械加工材料切割工 / 使用切割工具或設備，進行機械加工材料切割成型的人員。

54 / 模具工 / 操作設備和使用工具，加工、裝配、調試和維修金屬或非金屬製件模具的人員。

55 / 理貨員 / 從事商品與貨物的整理、揀選、配貨、包裝、複核、置嘜和貨物交接、驗收、堆碼、計量等工作的人員。

56 / 化工單元操作工 / 操作過濾、換熱、蒸發、蒸餾、萃取等化工單元設備，加工處理物料的人員。

57 / 土木建築工程技術人員 / 從事工業與民用建築、市政基礎設施等建造施工、監督管理的人員。

58 / 裝飾裝修工 / 使用工具、機具或手工，進行建築物、構築物等表面和內部空間噴、塗、抹、鑲貼、藝術處理，以及製作、安裝裝飾構件的人員。

59 / 塑料製品成型製作工 / 操作捏合、煉塑、造粒等設備配製塑料製品成型用料，及操作專用塑料機械將其加工成型材或製作成塑料製品的人員。

60 / 信息通信營業員 / 從事信息通信業務諮詢、營銷與辦理，投訴處理、客戶維繫等業務工作的人員。

61 / 建築和市政設計工程技術人員 / 從事建築物、構築物和市政工程設計，工程項目管理、諮詢及技術服務的工程技術人員。

62 / 機械製造工程技術人員 / 從事機械製造加工工藝、工藝裝備研發和生產技術組織的工程技術人員。

63 / 銑工 / 操作銑床，進行工件平面、溝槽、曲面等型面切削加工的人員。

64 / 物業經營管理專業人員 / 從事物業管理項目市場拓展、資產經營和管理的專業人員。

65 / 織布工 / 操作織布機等設備，將經紗、緯紗交織成織物的人員。

66 / 食品工程技術人員 / 從事食品原輔材料、工業產品、工藝技術和專用裝備研發，生產流程管理和生產技術指導的工程技術人員。

67 / 液晶顯示器件製造工 / 操作鍍膜、曝光、顯影等設備，製作、裝配和調試液晶顯示器的人員。

68 / 電器接插件製造工 / 使用加工、鉚焊設備和工具，加工零配件，裝配低壓接插件的人員。

69 / 半導體芯片製造工 / 操作外延爐、高溫氧化擴散爐、光刻機、電擊臺等設備，製造半導體分立器件、集成電路、傳感器芯片的人員。

70 / 米麵主食製作工 / 操作和麵、熟化、壓延、切斷、烘乾設備或方便麵生產線，進行掛麵和方便麵等米麵主食品製作的人員。

71 / 公共遊覽場所服務員 / 在公園、影劇院、禮堂、博物館、展覽館等公共遊覽場所，為顧客提供綜合服務工作的人員。

72 / 通信系統設備製造工 / 使用頻譜儀、通信接口測試儀等儀器、設備和工具，裝配、調試通信系統設備的人員。

73 / 社會工作者 / 從事社會服務項目開發設計、個案服務、小組服務、社區建設等專門化社會服務的專業人員。

74 / 質量管理工程技術人員 / 從事產品、工程、服務質量管理和質量工程的工程技術人員。

75 / 化工生產工程技術人員 / 從事化工產品生產過程技術指導或生產技術組織的工程技術人員。

76 / 電工電器工程技術人員 / 從事發電、輸配電、用電電器設備和電氣功能材料研發、設計、檢驗、試驗、技術服務的工程技術人員。

77 / 紡紗工 / 操作細紗機、絡筒機、併線機、捻線機、線團機等設備，將粗紗或纖維條、管紗、多根紗分別加工製成細紗、筒子紗、線及線團的人員。

78 / 人力資源管理專業人員 / 從事企事業單位人力資源規劃設計、招聘配置、培訓開發、績效考核、薪酬福利、勞動關係管理的專業人員。

79 / 塗裝工 / 使用專用設備和輔助工具，進行工件表面噴塗、浸塗、滾塗、刷塗等處理，以及特種功能塗層塗覆的人員。

80 / 幼兒教育教師 / 在幼兒教育機構中，專門從事幼兒教育工作的專業人員。

81 / 下料工 / 操作剪切、鋸斷等下料設備，使用工卡夾具，分離加工板材、型材等原材料的人員。

82 / 築路工 / 操作專用機械和設備，進行路基、路面、橋隧、管涵等施工的人員。

83 / 電子元器件工程技術人員 / 從事電子元器件、電子封裝和電子元器件試驗與檢測研發、設計、生產指導的工程技術人員。

84 / 汽車維修工 / 使用工、夾、量具和儀器儀表、檢修設備，維護、修理和調試汽車及特種車輛的人員。

85 / 鍛造工 / 使用加熱、鍛造設備及輔助工具，進行金屬毛坯下料、加熱、鐓粗、拔長、預製坯、成形、衝孔、切邊、校正、熱處理、清理、檢驗等鍛件加工的人員。

86 / 開清棉工 / 操作抓棉、開棉、清棉、成卷等設備，混棉和去雜除疵，將不同規格的源棉製成棉卷和纖維生條的人員。

87 / 呼叫中心服務員 / 從事信息查詢、業務諮詢和受理、投訴處理、客戶回訪及話務管理等工作的人員。

88 / 租賃業務員 / 從事機械設備及耐用消費品租賃手續辦理並提供後續服務工作的人員。

89 / 機械設計工程技術人員 / 從事機械設計技術方法研究、產品和工廠設計、產品性能測試、設計流程管理的工程技術人員。

90 / 市場管理員 / 在商品集貿市場中，從事市場經營秩序組織、協調、管理工作的人員。

91 / 裁縫 / 依顧客個性化要求，進行人體測量、服裝款式草圖繪製、服裝樣版製作，裁剪、縫製、拆改服裝的人員。

92 / 鍋爐操作工 / 操作工業鍋爐、生活鍋爐及其附屬設備，生產蒸汽或者其他熱載體的人員。

93 / 金屬熱處理工 / 操作熱處理設備，進行金屬材料、坯件或工件退火、正火、調質、表面淬火、回火、滲碳、滲氮等熱處理加工的人員。

94 / 磨工 / 操作磨床，使用磨料、磨具及專用工具，進行工件、光學玻璃等柱面、平面、螺紋等型面磨削加工的人員。

95 / 化學合成製藥工 / 操作反應器、離心機、壓濾器等設備，控制化學反應和單元操作，生產原料藥及中間產品的人員。

96 / 康樂服務員 / 在健身和娛樂場所，為顧客提供服務的人員。

97 / 中式麵點師 / 運用中式麵點成型技術和成熟方法，進行麵點主料和輔料加工，製作中式麵食、小吃的人員。

98 / 西式烹調師 / 運用俄、法等西式加工切配技巧和烹調方法，進行烹飪原料、輔料、調料加工，製作西式菜餚的人員。

99 / 計算機網絡工程技術人員 / 從事互聯網等計算機網絡研究、設計、安裝、集成、調試、維護、管理和服務的工程技術人員。

100 / 快件處理員 / 從事快件及總包的接收、卸載、分撥、集包、裝載、發運等工作的人員。
