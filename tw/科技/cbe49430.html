Bash技巧：可以上下左右移動俄羅斯方塊的Shell腳本

| 科技

在 Linux bash 中，前面文章介紹過顯示彩色俄羅斯方塊的 shell 腳本。

下面繼續介紹如何通過 k、j、h、l 鍵來上下左右移動單個方塊的 shell 腳本。

假設有一個 moveblock.sh 腳本，具體的代碼內容如下所示。

在這個代碼中，幾乎每一行代碼都提供了詳細的註釋，方便閱讀。

#!/bin/bash# 實現一個可以上下左右移動的方塊,移動範圍限定在指定邊框內.# 下面幾個常量指定長方形邊框的上下左右邊界# 指定邊框左邊的列數FRAME_LEFT=3# 指定邊框右邊的列數FRAME_RIGHT=26# 指定邊框上邊的行數FRAME_TOP=2# 指定邊框下邊的行數FRAME_BOTTOM=18# 下面的 BLOCK 數組對應一個 Z 字形方塊,具體形狀為:# [][]#   [][]# 這裡使用行數、列數座標點的方式來表示每一個小方塊的位置.# 第一個小方塊的起始行數、列數都是 0,作為整個方塊的原點.# 第二個小方塊和第一個小方塊在同一行,行數也是0.每一個小方塊#   顯示兩個字符,所以第二個小方塊的起始列數是 2.# 第三個小方塊在第一個小方塊的下一行,行數是 1. 它的列數是 2.# 第四個小方塊和第三個小方塊在同一行,行數是 1. 它的列數是 4.# 使用這些行列數加上方塊的起始行列數,就能定位出每個小方塊要# 顯示在哪一行、哪一列.之後可以使用ANSI轉義碼設置光標的位置.BLOCK=(0 0 0 2 1 2 1 4)# 土字形方塊.第一個小方塊的行數是 0,列數是 2.其他小方塊類似.# 可以放開下面註釋來查看土字形方塊的移動效果.## BLOCK=(0 2 1 0 1 2 1 4)# 這個值加上 BLOCK 數組裡面的小方塊行數,# 會指定每一個小方塊要顯示在哪一行.# 其初始值是邊框上邊行數的下一行.blockLine=$((FRAME_TOP + 1))# blockColumn 指定整個方塊顯示的起始列.# 這個值加上 BLOCK 數組裡面的小方塊列數,# 會指定每一個小方塊要顯示在哪一列.# 其初始值是邊框左邊列數的下一列.blockColumn=$((FRAME_LEFT + 1))# 定義下面兩個常量來檢查方塊最右邊和最下邊的邊界.# Z 字形方塊有兩行. 邊框下邊的行數減去 2,就是方塊# 起始最大的行數. 大於這個行數就會超過邊框範圍.BLOCK_MAX_BASE_LINE=$((FRAME_BOTTOM - 2))# Z 字形方塊的長度是 6 個字符. 邊框右邊的列數減去 6,# 就是方塊起始最大的列數.大於這個列數就會超過邊框範圍.BLOCK_MAX_BASE_COLUMN=$((FRAME_RIGHT - 6))# 顯示一個長方形邊框,作為方塊移動的邊界範圍function showFrame(){    # 設置邊框字符的顯示屬性: 高亮反白顯示,綠色文本,綠色背景    printf "\e[1;7;32;42m"    local i    # 下面使用 "\e[line;columnH" ANSI 轉義碼移動    # 光標到指定的行和列,然後顯示對應的邊框邊界字符.    # 行數遞增,列數不變,豎向顯示邊框的左右邊界    for ((i = FRAME_TOP; i <= FRAME_BOTTOM; ++i)); do        printf "\e[${i};${FRAME_LEFT}H|"        printf "\e[${i};${FRAME_RIGHT}H|"    done    # 列數遞增,行數不變,橫向顯示邊框的上下邊界    for ((i = FRAME_LEFT + 1; i < FRAME_RIGHT; ++i)); do        printf "\e[${FRAME_TOP};${i}H="        printf "\e[${FRAME_BOTTOM};${i}H="    done    # 顯示邊框之後,重置終端的字符屬性為原來的狀態    printf "\e[0m"}# 顯示或者清除方塊.方塊的具體形狀由 BLOCK 數組指定.# 傳入的第一個參數為 1,會顯示方塊.# 傳入的第一個參數為 0,會清除方塊.function drawBlock(){    local i    # square 變量保存要顯示的小方塊內容.    # 如果內容為 "[]",會顯示具體的方塊.    # 如果內容為 "  ",也就是兩個空格,會清除方塊    local square    # line 變量指定某個小方塊顯示在哪一行    local line    # column 變量指定某個小方塊顯示在哪一列    local column    # 所給的第一個參數值為 1,表示要顯示具體的方塊    # 所給的第一個參數值為 0,表示要清除當前的方塊    # 方塊顯示的位置由 blockLine 和 blockColumn 指定    if [ $1 -eq 1 ]; then        square="[]"        # 顯示方塊時,把方塊的背景色設成紅色        printf "\e[41m"    else        square="  "        # 清除方塊時,背景色要顯示為原先的顏色        printf "\e[0m"    fi    for ((i = 0; i < 8; i += 2)); do        # 使用 blockLine 和 BLOCK 數組指定的小方塊行數        # 來獲取每一個小方塊要顯示在哪一行.        line=$((blockLine + ${BLOCK[i]}))        # 使用 blockLine 和 BLOCK 數組指定的小方塊列數        # 來獲取每一個小方塊要顯示在哪一列.        column=$((blockColumn + ${BLOCK[i + 1]}))        # 使用 "\e[line;columnH" 轉義碼移動光標到指定的        # 行和列,然後開始顯示對應的小方塊.        printf "\e[${line};${column}H${square}"    done}# 重置終端的顯示狀態為原先的狀態function resetDisplay(){    # 把光標顯示到邊框底部的下一行,    # 以便終端提示符顯示在邊框之後,避免錯亂    printf "\e[$((FRAME_BOTTOM + 1));0H"    # 顯示光標    printf "\e[?25h"    # 重置終端的字符屬性為原來的狀態    printf "\e[0m"}# 初始化顯示狀態.例如顯示邊框,隱藏光標,等等function initDisplay(){    # 由於方塊會顯示在指定的行和列,    # 為了避免已有內容的干擾,先清屏.    clear    # 隱藏光標    printf "\e[?25l"    # 顯示提示字符串    echo "Usage: k/j/h/l 鍵: 上/下/左/右移動方塊. q 鍵: 退出"    # 顯示邊框    showFrame}initDisplay# bash 的 : 命令什麼都不做,永遠返回 true,# 用在 while 命令中形成死循環.while :; do    # 基於 blockLine 和 blockColumn 的值顯示方塊    drawBlock 1    # 獲取用戶的按鍵, h/l/j/k 會 左/右/下/上 移動方塊    read -s -n 1 char    # 獲取用戶按鍵,要移動方塊.移動之前,先清除原先方塊    drawBlock 0    case "$char" in        # h 鍵要向左移,移動的距離間隔是一個小方塊的寬度,        # 由於每個小方塊佔據兩列,blockColumn 值要減去 2.        "h") ((blockColumn -= 2)) ;;        # l 鍵要向右移.類似的, blockColumn 值加上 2        "l") ((blockColumn += 2)) ;;        # j 鍵要下移一行, blockLine 值加上 1        "j") ((++blockLine)) ;;        # k 鍵要上移一行, blockLine 值減去 1        "k") ((--blockLine)) ;;        # q 鍵退出        "q") break ;;    esac    # 檢查方塊是否移動到邊框邊界,避免移到邊框外面    if [ $blockColumn -gt $BLOCK_MAX_BASE_COLUMN ]; then        # 當下次要顯示的列數大於方塊最大的起始列數時,        # 將列數設置為最大的起始列數.        blockColumn=$BLOCK_MAX_BASE_COLUMN    elif [ $blockColumn -le $FRAME_LEFT ]; then        # 當下次要顯示的列數小於或等於邊框左邊的列數時,        # 將列數設置成邊框左邊的列數加 1.        blockColumn=$((FRAME_LEFT + 1))    elif [ $blockLine -le $FRAME_TOP ]; then        # 當下次要顯示的行數小於或等於邊框上邊的行數時,        # 將行數設置成邊框上邊的行數加 1.        blockLine=$((FRAME_TOP + 1))    elif [ $blockLine -gt $BLOCK_MAX_BASE_LINE ]; then        # 當下次要顯示的行數大於方塊最大的起始行數時,        # 將行數設置成最大的起始行數.        blockLine=$BLOCK_MAX_BASE_LINE    fidoneresetDisplayexit

代碼關鍵點說明

如何移動光標到指定的行和列

一般來說，顯示字符之後，光標就會往前移動。新顯示的內容會在光標之後繼續顯示。

但是當需要向左移動方塊時，方塊要在當前光標之前顯示。

難點就在於如何移動光標到左邊的位置。

這裡使用了 \e[line;columnH 這個 ANSI 轉義碼來設置光標到指定的行和列。

所給的 line 指定行數。所給的 column 指定列數。

這個轉義碼要求以大寫的 H 結尾。

關於 ANSI 轉義碼的詳細說明，可以參考前面的文章。

如何數字化表示方塊形狀

由於俄羅斯方塊有多個不同形狀的方塊，如果硬編碼顯示每一個方塊，需要定義多個小函數來負責顯示。

當需要顯示某個形狀時，就調用對應的函數。這個調用關係很複雜，也不方便代碼複用。

為了避免這個問題，需要數字化表示方塊形狀。

在代碼中解析這些數字信息，就能顯示出對應的形狀。

在俄羅斯方塊中，任意一個方塊都是由四個小方塊組成。 所以可以用四個行列座標點來確定具體方塊的樣式。 以方塊自身左上角為座標原點。每一個行列座標點指定一個小方塊的行數和列數。

例如，在上面代碼註釋中，詳細說明了 BLOCK=(0 0 0 2 1 2 1 4) 這個數組對應 Z 字形方塊。

同時，也提供了另一個參考的 BLOCK=(0 2 1 0 1 2 1 4) 數組對應土字形方塊。其具體形狀如下：

  [][][][]

使用類似的表示方法，BLOCK=(0 0 1 0 1 2 1 4) 數組對應 L 字形的方塊。其具體形狀如下：

[][][][]

在 BLOCK=(0 0 1 0 1 2 1 4) 這個數組中：

  • 第一個小方塊的行列座標點是 (0,0)
  • 第二個小方塊的行列座標點是 (1,0)。相比於第一個小方塊，列數不變，行數加 1，所以第二個小方塊在第一個小方塊的下一行
  • 第三個小方塊的行列座標點是 (1,2)。相比於第二個小方塊，行數不變，列數加 2，所以第三個小方塊在第二小方塊的右邊一列。由於每一個小方塊顯示兩個字符，所以列數要加 2
  • 第四個小方塊的行列座標對是 (1,4)。類似可知，它顯示在第三個小方塊的右邊一列

參考上面說明，其他形狀的方塊可以用類似的數字化表示。

調用腳本里面的 drawBlock 1 語句就能顯示指定形狀的方塊。

如何清除原先的方塊

當移動方塊後，會在新的位置重新顯示方塊，並清除之前顯示的方塊。

清除原先方塊的方法非常簡單：重置終端字符屬性後，在原先方塊的位置重新輸出空格，就會顯示為空，起到清除原先內容的效果。
