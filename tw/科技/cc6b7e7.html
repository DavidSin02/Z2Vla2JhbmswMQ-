圖文詳解：快來創作你的第一幅NFT加密畫

2020-10-29
| 科技

免責聲明：本文旨在傳遞更多市場信息，不構成任何投資建議。文章僅代表作者觀點，不代表火星財經官方立場。

小編：記得關注哦

來源：區塊律動BlockBeats

得益於近日 NFT 熱點的火爆，越來越多人開始關注 NFT 與加密藝術領域。甚至有傳統領域的藝術家們開始詢問，如何將自己的作品做成一個 NFT 展示給加密愛好者們。也有許多普通人躍躍欲試，想要創作一幅 NFT 畫作。為此，律動 BlockBeats 就當前兩個較為熱門的網站為例，簡單的講解下如何在線創作 NFT 作品，並以加密藝術交易平臺 Rarible 為例，教大家將自己的作品上傳至平臺供大家查閱。

不過律動 BlockBeats 也提醒讀者，NFT 作品本身更帶有藝術性以及創意性，如果僅以炒作的心態看待 NFT 市場，會很難享受其中的樂趣。同時，由於 NFT 更多代表的是一種原創的精神，因此在作品的製作時，也建議不要過於偏激或包含暴力等不健康因素。以展示加密精神為主才更能展示 NFT 本身的魅力。

如何在線創作 NFT 加密畫

Nifty Ink

Nifty Ink 是一個在線 NFT 創作平臺，無需 Gas 費即可創建你的 NFT 畫作。

1、首先打開 Nifty Ink 官網。網站製作的十分可愛，在打開時會有一定的緩衝時間，顯示一個「加載中（Loading）」的動圖，耐心等待即可。

打開之後頁面十分簡潔，大概之後就是一個可以進行創作的空白格網站。默認畫筆顏色是黑色。

2、默認畫筆的顏色是黑色，不過大家可以根據自己的偏好選擇喜愛的顏色，且畫筆粗細程度也可以自行調整。之後便可以在下方畫布進行創作，還可以設置畫布與邊框顏色。

點擊「畫筆」圖標，會出現更詳細的顏色選擇頁面。

3、好了，選好顏色和畫筆粗細之後，開始進行繪畫吧（律動小編繪畫中.....）

4、繪製完成後，在「名稱（Name）」欄輸入一個名字，給你的畫作起個名字吧。然後在「限量（Limit）」中輸入一個數量，這個數量代表著你的這幅作品會被生成的可交易的代幣數量。

比如你僅希望作品生成一個代幣，那麼就選 1，這幅作品就是唯一的，針對這篇作品的 NFT 也就只有 1 個。如果你填 2，那麼這幅作品就會有兩個 NFT。確定「Limit」鑄幣數量，然後點擊「Ink！」，首個作品就已經做好了。

5、點擊「Ink！」之後，頁面最下方會出現下圖所示的內容。我們點擊「出售（Sell）」對畫作進行定價，這一步完全自由定價，沒有限制。

6、定價好後，點擊「發送（Send）」將作品發送到一個加密貨幣錢包地址（律動嘗試後發現，目前以太坊錢包、波場錢包都可以，大家也可以嘗試下其他錢包地址），發送作品到自己錢包地址不需要手續費。成功之後錢包會有一個提示：「發送成功（Sending successful）」。

不過雖然顯示是發送到你的地址了，但是錢包中還是無法查看到作品的。你所擁有的作品所屬權是通過在 Nifty Ink 網站中「持有（Holding）」這一欄來體現的。

當然，上述過程只是製作併發送到自己所有的地址中的過程，你之前的操作都是在 Nifty Ink 自己的鏈上進行操作所以無需手續費。

8、但如果想將作品上傳到以太坊網絡，需要選擇「升級至以太坊主網（Upgrade to Ethereum mainnet）」，這時候還是需要繳費的，價格標注在網站上，鏈接錢包後就可以繳費並上傳到以太坊網絡了。在 OpenSea 等平臺售賣也需要支付手續費，且手續費需要使用 xDAI 代幣支付。

PixelChain

相比於 Nifty Ink，在 PixelChain 上的創作是像素化的，相比於前者的靈活，後者雖然有些限制，但更適合繪畫新手。

1、首先打開 PixelChain 官網並開始創作

你可以使用顏色填充、畫筆、鏡像畫筆等工具，在右側還可以選擇不同的顏色，每種顏色還可以進行微調。

2、創作完成後可以點擊這個按鍵來一鍵上傳至以太坊網絡，並在 OpenSea 等平臺進行交易。

上傳是需要 ETH 作為手續費的，需要提前在錢包中預存。

上傳 NFT

以如今最火爆的交易平臺 Rarible 為例。

1、首先打開 Rarible 官網並點擊「鏈接錢包（Connect Wallet）」

2、鏈接錢包後點擊頁面右上角的「創建收藏品（Creat collectible）」

3、如果你希望你的作品被鑄造成僅此一枚 NFT 則選擇「Single」，如果你希望你的作品可以被鑄造成多枚 NFT 則選擇「Multiple」。

4、之後點擊「選擇圖片（Choose image）」並選擇一幅你想出售的作品

5、然後下拉網頁填寫

（1）作品名稱

（2）作品描述（選填）

（3）版稅（即該作品後續每次交易的分成比例），如果在之前選擇了「Multipie」則需要選擇發行數量

（4）作品特點（選填）

（5）設置價格（點擊右側按鈕）

（6）設置「彩蛋」（只有在購買後才會顯示的內容）

6、填寫完畢後點擊「創建（Create）」並按順序以此點擊即可

需要注意的是，上傳 NFT 需要 ETH 作為手續費，要提前在錢包中預存好。

7、等待完成後就可以在賬戶中看到了，接下來就等待你的作品被喜歡的人買走吧。
