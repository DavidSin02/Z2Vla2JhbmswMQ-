Oracle數據庫-查詢所有表名、字段名、字段名說明的簡介

2020-11-14
| 科技

1.查詢所有表名（table_name）：

select a.table_name from user_tables a;

2.查詢所有字段名（column_name）：

select b.column_name from user_col_comments b;

3.查詢指定表（XZ_TEST）的所有字段名（column_name）：

select c.column_name from user_col_comments c

where c.table_name = 'XZ_TEST';

其中‘XZ_TEST’為你需要查詢指定表的名稱

4.查詢所有表的表名（table_name）和表說明（comments）：

select t.table_name,t2.comments from user_tables t

inner join user_tab_comments t2

on t.table_name = t2.table_name;

5.查詢模糊表的表名（%XZ_TE%）：

select t.table_name from user_tables t

where t.table_name like '%XZ_TE%';

6.查詢模糊表的表名（%XZ_TE%）和表說明（comments）：

select t.table_name,y.comments from user_tables t

inner join user_tab_comments y

on t.table_name = y.table_name

where t.table_name like '%XZ_TE%';
