1、變量的基礎-掌握

變量在程序中用變量名錶示，變量名由用戶根據其用途任意命名變量名郵字母、數字、下劃線組成 不能以數字開頭，也不能和C關鍵字重名

2、變量的說明-掌握

變量在程序中使用時，必須先說明它們的存儲類型和數據類型變量說明的一般形式是：<存儲類型> <數據類型>  <變量名>； <存儲類型>是關鍵詞  auto  register static和extern  <數據類型> 可以是基本數據類型，也可以自定義數據類型

3、變量的存儲類型-掌握

auto說明的變量只能在某個程序範圍內使用，通常函數體內或函數中的複合語句裡在函數體的某程序段內說明auto存儲類型的變師可以省關鍵字auto 如下：auto int k;int j;double x;register稱為寄存器型，register變量是想將變量放入CPU的寄存器中，這樣可以加快程序的運行速度如果申請不到就使用一般內存，同autoregister int a;

4、總結與思考

C語言中，變量的存儲類型有哪些？修飾局部變量時，auto和register有什麼區別。

變量二

變量的存儲類型 static -掌握

static變量稱為靜態存儲類型的變量，即可以在函數體內使用，也可以在函數體外使用 默認為0在內存中以固定地址存放只要程序沒結束，就不會消失。#include <stdio>int main(){    int i=1;    whiel(i<5){       //int a =0;       static int a=5;        a++        printf("a++ %d\n",a)        i++;    }    return 0;}執行輸出：a=6a=7a=8a=9

變量的存儲類型 extern

當變量在一個文件中的函數外體說明，所有其他文件中的函數或程序都可以引用這個變量。extern 稱為外部參照引用型，使用extern說明是想引用在其他文件中的函數體外部聲明的變量 注： 如果使用static修飾的全部變量，其他文件無法使用extern int a; //可以使用其他文件的變量exern_static1.c #include <stdio> int global_a=10int main(){    int i=1;    return 0;} exern_static2.c #include <stdio> extern int global_a; int main(){       printf("global_a=d\n",global_a)       return 0;}
