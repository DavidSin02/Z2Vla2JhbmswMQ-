Linux的基本思想有兩點

（1）一切都是文件

（2）每個軟件都有確定的用途

FHS標準

/boot：啟動目錄，內核存放地

/etc：配置文件存放地

/home：用戶家目錄

/lib：庫文件目錄，程序額外的函數庫存放地

/bin：可執行文件和常用的Linux命令

/sbin：系統管理員的命令和工具

/usr：應用程序和文件安裝地

/mnt：掛載其他系統文件

/root：root賬戶的home目錄

/dev：存放Linux系統下的設備文件

VI

功能：編輯文檔

vi file_name

按i或a編輯

i：插入

a：追加

esc：切換模式

命令模式

可執行操作

（1）:w 保存文件

（2）:wq 保存並退出

（3）:wq! 保存並強制退出

（4）:q 退出

（5）:q! 強制退出

（6）dd 刪除一行文字

（7）x 刪除一個字符

（8）:n 光標移至文本第n行

（9）$ 光標移至文本的行尾

（10）G 光標移至文本的末尾

（11）/ 查找某個字符串 /sdb

Linux管理文件和目錄的命令

pwd：打印用戶當前所處的路徑

cd：改變用戶所在目錄

格式：cd <目錄名>

cd .. 返回上一級目錄

cd 返回當前用戶的主目錄

ls：顯示指定目錄下的內容

格式：ls [參數] [目錄或文件]

參數：-a 列舉目錄中的所有文件，包括隱藏文件

-l 列舉所有目錄詳細信息

-al 包括隱藏文件的詳細信息

-r 按順序排列顯示

-R 遞歸列取當前目錄下所有子目錄內的內容

-s 大小size:數據塊

ls -l 1.text 列舉1.text的詳細信息

cat：查看文本文件

創建多個文件：touch 文件名1 文件名2 文件名3

grep：在某個文件中查找一個特定的字符串

格式：grep 要查找的字符串 文件的名字【grep查找區分大小寫】

cp：複製文件（copy）

格式：cp [參數] [源文件] [重命名目標文件]

參數：-i 互動：如果文件將覆蓋目標中的文件，他會提示確認

-r 遞歸：這個選項會複製整個目錄，子目錄以及其他

-v 提示：打印移動信息

mv：可以對文件或者目錄進行移動

格式：mv [源對象] [目標對象]

參數：-i 互動：如果選擇的文件將覆蓋目標中的文件，他會提示確認

-f 強制：不提示的移動文件

-v 詳細：打印移動信息

mkdir：創建目錄

格式：mkdir [目錄1] [目錄2] ...

創建嵌套目錄：mkdir -p 目錄名1/目錄名2/目錄名3

rmdir：刪除空目錄

格式：rmdir [參數] [目錄名1] [目錄名2] ...

參數：-p：刪除指定目錄，如果上層目錄為空則一併刪除

rm：刪除文件

格式：rm [參數] [文件]

參數：-i：互動：提示確認刪除

-f：強制：代替互動模式，不提示刪除

-v：詳細：打印刪除信息

-r：遞歸：將刪除某個目錄以及其中所有的文件和子目錄

man：查詢命令說明

格式：man 命令名稱

退出按q

locate：定位文件和目錄【locate命令使用數據庫來定位帶有test這個詞的文件或目錄】

格式：locate 文件或目錄名字

whatis：用於查詢某個命令的含義

格式：whatis 命令名稱

文件備份和壓縮命令

bzip2：bzip2來壓縮文件

格式：bzip2 filename

文件壓縮後的後綴名為：.bz2

`bunzip2：解壓縮

格式：bunzip2 filename.bz2

gzip：gzip來壓縮文件

格式：gzip filename

-r：遞歸處理，指定目錄及子目錄一塊處理

gzip -r file1

上面命令把file1目錄中的內容壓縮起來（壓縮不了目錄）

gunzip：解壓縮

格式：gunzip filename.gz

tar：將文件或目錄進行打包或者解壓縮

格式：tar [參數] [打包後的文件名] [需要打包的文件或目錄]

壓縮：

tar cvf xxx.tar（要壓縮後的文件名）

解壓縮：

tar xvf xxx.tar

在不進行解壓的情況下，顯示壓縮文件裡的內容

tar tvf xxx.tar

使用tar和bzip2來壓縮/解壓縮文件

tar -cjvf xxx.tbz（要壓縮的文件名）

tar -xjvf xxx.tbz【解壓縮】

使用tar和gzip來壓縮/解壓縮文件

tar -czvf xxx.tgz（要壓縮的文件名）

tar -xzvf xxx.tgz【解壓縮】

參數：-c：創建壓縮文件

-x：展開歸檔文件

-t：顯示包括在tar文件中的文件列表

-z：壓縮/解壓縮文件（gz格式）

-v：寫入或讀取時顯示所有文件

-f：指定要展開的歸檔文件名

-j：壓縮或解壓縮文件（bz2格式）

文本閱讀命令

head：查看文件的開頭部分

格式：head 文件名【默認查看10行 head -n 文件名 查看n行】

tail：查看文件的結尾部分

格式：tail 文件名【默認查看10行 tail -n 文件名 查看n行】

more：按頁來查看文件的內容【使用空格鍵和b鍵來前後移動】

格式：more [選項] [filenames]

參數：-num：一次顯示行數

-f：計算行數

+num：從第num行開始顯示

+/查找的字符串 文件名：查找關鍵字

管道符的使用‘|’

ls -al/etc | more -5：把結果按一次五行的形式展現

定位查找文件的命令

which：在path環境變量所列的目錄中查找Linux指令的位置信息

格式：which 命令

whereis：查找符合條件的文件，whereis指令只能定位一些文件在文件系統的位置。一般用於程序名的搜索

格式：whereis [參數] 程序名

參數：-b：只找二進制文件

-m：man說明文件，查出說明文檔的路徑

管理使用者和設立權限的命令

chmod：用來改變許可權限

r：讀

w：寫

x：執行

還可以用數字表示權限：4-讀取 2-寫入 1-執行+

權限組成十位

-rw-r--r--

第一位：代表文件類型

如果第一位是 - 代表的是普通的文本文檔；

如果是 d 代表的是目錄

剩下九位分成三組，每三位一組

第一組：rw- 代表的是主人的權限 u

第二組：r-- 代表和文件主人同組成員的權限 g

第三組：r-- 代表其他人的權限 o

所有人的權限用a代表

用法：chmod u/g/o+/-/= 權限 文件或者目錄

chmod u+r b1

chmod 751 b1

useradd與userdel：添加刪除用戶

格式：useradd 用戶名

useradd [參數] 用戶名

參數：-r：不僅刪除用戶，還把用戶相關的目錄和文件一併清除

chgrp：用於變更文件或目錄所屬群組

格式：chgrp 組名 文件名

chown：將文件擁有者改為指定的用戶或組

格式：chown 文件擁有者或組 文件名

su：用來切換用戶

格式：su 用戶名

參數：-l 載入切換用戶的環境變量

passwd：修改用戶密碼

格式：passwd 用戶名

有關關機和查看系統信息的指令

退出：exit

文本模式下使用快捷鍵Ctrl+D

重新啟動

在文本模式下使用reboot

或者：shutdown -r <time>

列如：shutdown -r now 立即重啟

shutdown -r 11:30 在11:30重啟

關機

在文本模式下使用命令：halt

或者：shutdown -h <time>

列如：shutdown -h now 立即關機

shutdown -h 11:30 等到11:30關機

shutdown -t 2 在兩分鐘後關機

ps：查看進程

參數：u：列出使用者的名稱和使用時間

x：列出所有程序

列如：ps u 查看有關使用者的信息

ps x 列出所有程序分佈情況（查詢出所有的進程號pid）

top：和ps類似，查看進程，每幾秒鐘更新一次狀態

kill：終止進程

語法：kill 進程號

date：用來修改時間和日期

列如：date -u 使用格林尼治時間

date -r 1.txt 查看文件的最後一次修改時間

date -s 時間 設置時間

cal：查看日曆

列如：cal 2019 查看2019全年月份

磁盤空間命令

df：查看磁盤使用情況

參數：-a 可以列出全部文件和各分區的磁盤使用情況

du：顯示文件目錄的大小

語法：du 參數 文件名

參數：-ab 列出所有文件和目錄所佔的空間

網絡操作命令

ifconfig：顯示網絡設備狀態

ping：用於檢測主機

netstat：知道整個Linux系統的網絡情況或顯示網絡狀態

顯示詳細的網絡狀況

#netstat -a

顯示UDP端口號的使用情況

#netstat -apu

案例IP配置

Linux網絡配置文件存放地：

文件路徑/etc/sysconfig/network-scripts

第一種配置：網卡自動獲取IP地址

修改ifcfg-eth0文件

修改ONBOOT=yes 即可。 #開啟自動啟用網絡連接（BOOTPROTO=dhcp）

#service network restart#重啟網卡

第二種配置：網卡配置靜態IP

修改BOOTPROTO=static #啟用靜態IP地址

添加IPADDR=192.168.23.123IP地址

NETMASK=255.255.255.0 配置網關

GETWAY=192.168.23.1 配置默認網關

#service network restart#重啟網卡

其他命令

clear：清屏

echo：顯示回顯文字 echo $PATH 查看PATH環境變量的值

uptime：系統運行的時間

wc：將計算指定文件的行數、字數、以及字節數

語法：wc 參數 文件名

參數：-c 只顯示Bytes數

-l 只顯示行數

-w 只顯示字數
