利用信號調理提高測量質量（上）

2020-10-29
| 科技



簡介

測量現實世界中的物理現象時，信號調理是精確測量傳感器輸出信號、提高採集質量的先決條件。就好像生長在農場中的小麥，在包裝成雜貨店裡的一袋袋麵粉之前，需要經過大量的預處理，原始信號必須經過清理、轉換和適當的調整，才能成為人類或機器可以識別的信號。

針對不同的待測信號和測量儀器，應選用不同的信號調理方式。為了讓您更加熟悉信號調理的基礎知識，本文將討論數據採集領域最常見的信號調理技術：

1模擬輸入前端拓撲

2儀表放大器

3濾波

4衰減

5隔離

6線性化

7電路保護

每種技術都有相應的優勢和不足。本文力圖闡明其最佳使用方法和常見應用場合。電路圖和公式用於說明如何選取正確的電子元器件。理解這些信號調理技術的特點，將有助於您在相關應用中提高數據採集系統的測量精度。

1. 模擬輸入前端拓撲

數據採集系統架構

就某些方面而言，數據採集系統有別於單通道或多通道的測量儀器。它可以同步測量幾百個通道的輸入數據，並進行存儲，多數系統都具有8到32個通道，通道數一般是8的倍數。相比而言，一個可選量程的電壓表可以被認為是一個數據採集系統，但需要手動選擇量程，而且數據存儲能力的缺乏，限制了其應用範圍。



圖1：數據採集系統結構框圖。一個簡單的數據採集系統前置多路複用輸入段，後面跟一個儀表放大器（IA），將信號送入唯一的高精度、高成本的模數轉換器（ADC）。這種結構避免了使用多個ADC所需要的昂貴成本。

圖1說明了一個簡單的數據採集系統包括切換網絡（多路複用器）和模數轉換器（ADC），而我們要討論的對象，儀表放大器（IA），位於二者之間。每個電路模塊都有獨特的功能和使用限制，共同決定了系統的性能。

ADC是電路中模擬部分到數字部分路徑中的最後一級。在任何一個數據採樣系統中，例如多路複用數據採集系統，信號流進ADC之前，一個採樣-保持階段是非常必要的。ADC可能在數字化不斷變化的電壓模擬量時，無法充分利用其分辨率，除非電壓變化相對於ADC的採樣率而言較為緩慢。一些ADC具有內部採樣-保持電路，或在電路結構上模仿這一過程。本文中，我們假設電路中的ADC模塊包含採樣-保持電路（內置或外置），可使輸入信號在轉換週期中保持穩定。

數據採集系統中的ADC，兩個主要指標分別是分辨率和採樣率，通常情況是16位或24位的分辨率和20kS/s至1MS/s的採樣率，信號輸入類型有單極性和雙極性兩種模式。單極性輸入一般是0至某個正電壓值或0至某個負電壓值，雙極性輸入一般是從某個負電壓至同等大小的正電壓。許多數據採集系統在讀取單極性或雙極型信號時，都能夠充分利用分辨率，這需要進行電平轉換，使得雙極型信號適應單極性ADC，反之亦然。舉例來說，一個標準的16位分辨率，100kS/s採樣率，輸入範圍±5V的ADC，可將輸入信號量化為65536個單位，0V將對應的是名義上的第32768個單位。用10V的輸入範圍除以65536，得到的商代表1個最低有效位(LSB)，即153μV。



圖2：寄生的RC時間常數。信號源輸出電阻應該儘可能的小，以減小MUX的寄生電容C和串聯電阻R導致的RC時間常數，時間常數過大，可能對測量精度造成不利的影響。

信號源輸出電阻過高，可能導致多路複用系統出現問題。因此，在多路複用系統中，信號源輸出電阻必須足夠小，圖2中的RC電路很好的解釋了這一點。多路數據選擇器的輸入端和輸出端，都與地之間存在一個很小的寄生電容，這將在信號源輸出電阻過高或系統採樣率過高時，影響測量精度。RC電路可等效為直流電壓源、電阻、開關和電容的串聯，T = 0時，開關閉合，電容充電，通過10kΩ的電阻給100pF的電容充電時，RC時間常數為1µs，如果系統建立時間僅為2µs，電容只充有86%的電，誤差高達14%。將電阻降低為1kΩ，可使電容在20個時間常數內，完成充電，而且精度有所保證。



圖3A：輸入電阻和信號源輸出電阻。相對於輸入電阻Ri，傳感器的輸出電阻Rs應該儘量小，以最小化輸入到ADC的電壓誤差，對於mV級的信號來說，這也可以提高信噪比。



圖3B：MUX的電荷注入效應。模擬開關設備的輸入信號發生電平變化時，輸出端可能產生尖峰脈衝，即電荷注入效應。降低信號源輸出電阻可以減弱影響。

圖3A顯示了系統的輸入電阻和傳感器的輸出電阻是如何串聯成分壓電路，並將注入ADC的信號衰減掉一部分的。多數模擬輸入通道的輸入電阻大於等於1MΩ，在信號源輸出電阻較低的情況下，這並不是什麼問題。然而，有一些傳感器（例如壓電傳感器）的輸出電阻比較高，應對其進行特殊處理。最後，使用多路數據選擇器可有效提高數據採集系統的輸入阻抗。

圖3B演示了電荷注入效應。

運算放大器

許多傳感器輸出信號特別小，以至於不能直接接到低增益的多路複用數據採集系統的輸入端，因此，信號必須先進行放大處理。兩個例子分別是熱電偶和電橋式應變計，輸出信號通常低於50mV。

大多數數據採集系統使用不同結構的電路，在預處理中放大輸入信號。隨著現代模擬電路設計技術的發展，數據採集系統已經自帶了集成運放，可輕鬆配置用於放大信號或緩衝信號。集成運放內部包含許多電路模塊，在電路圖中通常表示為簡單的功能框圖，外圍電阻和電容的搭配，決定了其具體用途。集成運放的多功能特性使其特別適用於各種各樣的信號調理。



圖4：運算放大器。運算放大器的兩個基本組態被稱為反相組態和同相組態。放大係數等於反饋電阻和輸入電阻的比值。

運算放大器多被用於反相組態和同相組態（見圖4）。各種組態下，其理想增益都都反饋電抗和輸入電抗的函數。此外，在一些特殊的配置下，運放可以用於實現其他的基本功能，例如電壓跟隨器或差分放大器。

反相放大組態



圖5：反相放大組態。反相放大器的輸出信號極性與輸入信號相反，閉關增益等於兩個電阻的比值，即 -（Rf / Ri） = - 100kΩ / 10kΩ = -10。

反相組態是運放最基本的工作組態之一。它可以輸入參考自同一點的電壓信號，輸出放大並反相後的信號。運放的開環增益是非常高的，甚至達幾十萬，但理想運放的傳輸函數通常被假定為增益無窮大來簡化引入，使其在計算級聯增益時不會引入有效誤差。

儘管運放具有如此高的增益，但輸入信號看到的僅僅是由Rf和Ri組成分壓器。負號表示輸出信號與輸入信號極性相反。傳輸函數的具體推到不在此進行，我們只給出相關計算公式。

公式1：反相放大器

Vo = –Vin(Rf/Ri)

其中：

Vo = 輸出信號，單位V

Vin = 輸入信號，單位V

Rf = 反饋電阻，單位Ω

Ri = 輸入電阻，單位Ω

例，輸入信號500mV，需要放大為-5V

Vo = -Vin(Rf/Ri)

-(Vo/Vin)= Rf/Ri

-(-5/0.50) = Rf/Ri = 10

因此，反饋電阻應為輸入電阻的10倍，比如Ri選10kΩ，則Rf應選100kΩ（見圖5）。

運放可安全處理的最大輸入電壓大約比電源電壓低2V左右。例如，±15VDC供電的運放，輸出信號不應超過±13VDC。這是限制運放處理信號能力的最重要的因素之一。

同相放大組態



圖6：同相放大組態。工作於同相組態的放大器，輸入輸出信號極性相同，增益Acl = (Rf + Ri)/Ri = 11。

同相放大器和前面的反相放大器類似，但是輸出信號極性與輸入信號相同。同樣的，增益只與組成分壓器的兩個電阻Rf和Ri有關（見圖6）。

傳遞函數如下

公式2：同相放大器

Vo = Vin（Rf + Ri）/Ri

和上例一樣，輸入500mV信號時，

Rf = 100 kΩ, and Ri = 10 kΩ:

Vo/Vin = (Rf + Ri)/Ri,

Vo = Vi(Rf + Ri)/Ri

Vo = 0.50(100k + 10k)/10k

Vo = 0.50(110k/10k) = 0.50(11)

Vo = 5.5V

前面討論的反相放大器的輸入信號限制，同樣適用於同相放大器。

差分放大器



圖7：差分放大器。差分放大器的輸出信號與兩個輸入端信號的差值有關，即Acl = g(V1-V2)，g是增益。因為圖中所有電阻大小均相等，所以增益為1。如果需要10倍的增益，則反饋電阻應設計為輸入電阻的10倍，並且兩個反饋電阻大小相等，輸入電阻大小也相等。

相比而言，差分放大器具有比同相、反相放大器更多的優勢。如圖7所示，差分放大器實際上是同相放大器和反相放大器的組合。輸入信號加在差分放大器的正輸入端和負輸入端之間，與地和其他公共參考點隔離開來，可選擇的地，提升了差分放大器的靈活性。其輸出信號僅取決於兩個輸入端之間的電壓差，傳入函數如下：

公式3：差分放大器

輸入50mV的信號：

V1 = 1.050V and V2 = 1.000V

Vo = (Rf/Ri)(V1 – V2)

Vo = (100k/100k)(0.05 V)

Vo = 0.05 V

增益為10時，Rf = 100k，Ri = 10k：

Vo = (Rf/Ri)(V1 – V2)

Vo = (100k/10k)(0.05V)

Vo = 0.50V

差分放大器最大的優點在於它能夠抑制兩個輸入端上信號完全相同的部分，即共模信號（Vcm或CMV）。將兩個輸入端連接到同一個電壓信號時，即可看出共模電壓的抑制程度，儘管電壓出現在兩個輸入端上，但差分放大器僅僅響應兩個信號的差(本例中為0)，理想運放將輸出0(更多信息請參考儀表運放和高共模信號放大器兩部分)。

程控增益放大器



圖8：程控增益放大器。程控增益的同相放大器，可通過MCU輸出數字信號，以控制模擬開關。

程控增益放大器通常工作在同相組態，具有數字可控的模擬開關，來連接或斷開反饋迴路上的電阻。外部處理器或邏輯器件通過控制可尋址的模擬開關，來選擇適當的反饋電阻，最終達到控制增益的目的（見圖8），使得輸入信號達到可測量的範圍，並無失真地顯示。

2. 儀表放大器

一個最基本的問題

有些傳感器的輸出信號是微伏級別的，對其進行放大處理時，接地迴路問題和其他干擾頻繁出現。有些傳感器則從差分信號源輸出信號，以最小化接地迴路的問題，並降低共模干擾。這種放大器必須具有以下特性：

極低的輸入電流、漂移和失調電壓

穩定而精確的電壓增益

高輸入阻抗和共模抑制能力

雖然通用集成運放具有不止一級的放大器，並且使用超精密的電阻，但專用的儀表放大器（IAs）更適合這些應用。高性能運放仍然使用基本的電路結構，但是他們提供極高的共模抑制能力，並且在設定增益時，無需高精度的電阻。許多儀表運放被設計用於特定的應用中，並且具有獨特的功能，用來提升精確性和穩定性。



圖9：儀表放大器。儀表放大器（IAs）通常是具有高輸入阻抗的差分放大器。

例如，數據採集系統中，開關選擇網絡後級的功能模塊（見圖9），通常是具有關鍵作用的儀表放大器。它能夠抑制共模電壓，放大差分信號，並驅動ADC的輸入端。

高壓共模放大器



圖10：高壓共模信號放大器。將運放的兩個輸入端連接至同一個電壓源，來測試共模抑制能力。電位器用於調節放大器兩個輸入端的平衡，使共模影響達到最低。

共模電壓被定義為兩個輸入端信號中完全相同的部分（見表10），當輸入信號分別為4.10V和4.20V的時候，共模電壓Vcm為4.10V，差模電壓即0.10V。理想情況下，儀表放大器將忽略共模信號，只放大差模信號，放大器對共模信號的抑制能力可以用共模抑制比（CMRR）這一參數來衡量。儀表放大器的高壓共模信號抑制能力和高壓信號抑制能力是不同的，二者往往被混淆。

測量的信號電壓往往比ADC的輸入電壓範圍小得多，比如待測信號0-100mV，而ADC的典型輸入範圍是0-5V。因此需要對信號放大50倍，來充分利用ADC的分辨率。儀表放大器的增益一般能做到1至10000以上，但在多路複用系統中，增益一般控制在1至1000以內。

測量誤差來自於非理想的模擬開關和信號源帶來的電阻，但儀表放大器極高的輸入電阻可以把誤差降至最低，其輸入端前級是兩個電壓跟隨器，這種組態是通用放大器組態中輸入阻抗最高的。極高的輸入阻抗和極低的偏置電流，保證了模擬開關部分損耗的電壓達到最小，為儀放輸入端爭取了更精準的輸入信號。

另外，儀表放大器的低輸出阻抗，特別適合驅動ADC的輸入。一般的ADC並沒有很高的或固定的輸入電阻，因此其前級電路的輸出電阻必須足夠低。

失調電壓、增益誤差、帶寬和穩定時間在一定程度上限制了儀表放大器的應用。失調電壓和增益誤差可以在測量結果中進行校準和補償，但帶寬和穩定時間限制了放大器可處理信號的頻率以及多路複用系統中通道切換的速度。一連串穩定的直流電壓連續不斷地輸入至儀表放大器，放大後的信號將是複雜的混合信號。穩定時間是指從信號輸入到輸出放大後的信號至穩定值（誤差0.01%以內）所需要的最短時間。例如系統以100kHz的頻率掃描輸入端口，信號電壓的讀取要在10µs內完成，如果模數轉換耗時8µs，那麼留給輸入信號達到穩定的時間就只剩下2µs。

雖然對系統進行校準可以降低失調電壓和增益誤差，但並不需要經常這樣做。例如，放大器失調電壓0.5mV，增益為2時，若輸入信號為2V，理想輸出電壓應為4V，增益誤差將導致輸出電壓偏移1mV，即0.025%；而失調電壓不變，增益為50時，若輸入信號100mV，理想輸出電壓應為5V，增益誤差將導致輸出電壓偏移25mV，即0.5%。雖然失調電壓相同，但隨著增益的提升，0.25%的增益誤差將產生顯著的影響，在較高增益下將產生較大的絕對誤差，單位增益的誤差最小。系統軟件可以做到已知修訂係數的（mx+b）形式的修正，但有時並不值得這樣做。

集成儀表放大器



圖11：集成儀表放大器。儀表放大器具有極高的輸入阻抗，電阻Rm用於調節增益，輸出信號是V1和V2電壓差的函數。

集成儀表放大器屬於高質量的運算放大器，內部自帶精密反饋網絡，是放大微弱差分信號並在噪聲環境中精確測量小信號的理想方案。其廣泛適用於應變計、熱電偶、RTD、分流器和壓力傳感器等多種傳感器信號的直接測量，而無需外部信號調理電路。儀表運放通常配有三個放大器——兩個差分輸入放大器和一個差分輸出放大器（見圖11），增益通常由一個增益調整電阻控制，也有一些儀放具有內置程控增益功能。

程控增益儀表放大器

一類特殊的儀表放大器，程控增益儀表放大器（PGIAs），針對數據選擇器輸出的不同信號範圍，能夠在預置的幾種增益之間快速切換，用於選取輸入通道的數字電路即可同時選擇增益。其原理和上面描述的程控增益放大器一樣。
