軟件推薦丨Fluent Terminal ——基於UWP和Web技術的終端仿真器

2020-10-29
| 科技

Fluent Terminal 是一個基於 UWP 和 Web 技術的終端仿真器。

特性

  • PowerShell、CMD、WSL 和自定義 Shell 的終端
  • 支持 Tabs 和多窗口
  • 支持主題和外觀的配置
  • 支持主題的導入導出
  • 支持導入 iTerm 主題
  • 支持全屏模式
  • 可編輯的快捷鍵綁定
  • 提供搜索功能
  • 可配置 Shell 的配置以快速在不同的 Shell 間切換
  • 資源管理器上下文菜單集成
