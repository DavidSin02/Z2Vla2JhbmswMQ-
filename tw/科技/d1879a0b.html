華為交換機基本配置指令操作步驟

2020-11-14
| 科技

文章以X.X.X.X代替IP地址，以s5720為例

sys 提高權限

華為5720交換機配置流程：

一、設置管理VLAN IP信息

新建vlan，進入vlan管理，配置IP：

vlan 100

interface Vlanif100

ip address X.X.X.X 255.255.255.0

二、新建賬號

進入aaa視圖，新建賬號，設置密碼及權限等級、服務協議

aaa

local-user weiziidc password irreversible-cipher +密碼

local-user weiziidc privilege level 15

local-user weiziidc service-type telnet

三、設置訪問ACL

acl number 2000

step 1

rule 1 permit source X.X.X.X 0

rule 2 permit source X.X.X.X 0

rule 3 permit source X.X.X.X 0

rule 4 permit source X.X.X.X 0

rule 5 permit source X.X.X.X 0

rule 999 deny

四、開啟Telnet服務，修改Telnet默認端口

telnet server enable

telnet server acl 2000

telnet server port 33899

五、設置虛擬接口vty

user-interface vty 0 4

authentication-mode aaa

acl 2000 inbound

protocol inbound telnet

user-interface vty 16 20

authentication-mode aaa

acl 2000 inbound

protocol inbound telnet

六、設置上聯口

interface XGigabitEthernet0/0/1

port link-type trunk

port trunk allow-pass vlan 100 to 101

七、設置交換機回程路由

ip route-static 0.0.0.0 0.0.0.0 下一跳

八、設置各端口數據

interface GigabitEthernet0/0/1

port default vlan 100

qos lr outbound cir 102400 cbs 102400

qos lr inbound cir 102400 cbs 102400

...

其他端口按需依次設置。
