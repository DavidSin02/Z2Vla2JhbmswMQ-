圖文解析案例，如何實現西門子PLC與調速器之間的USS通訊控制

2020-10-29
| 科技

此次主要跟大家分享的是關於西門子S7-200PLC與6RA70裝置的連接，究竟如何來實現，現在就來詳細講解講解：

一、S7-200PLC與6RA70裝置連接

1．系統要求

安裝MicroWin software（V3.2以上）以及 USS協議庫（V2.0以上）

PC/PPI電纜、S7-200（如CPU 226XM）、電源模塊、通信電纜

6RA70驅動裝置及一臺PC機

2．在使用MicroWin software 創建項目之前，先檢查USS protocol協議是否被正確安裝，此協議庫需要單獨購買：

3．我們需要創建一個簡單的例程：

第一步：設置通訊接口

第二步：建立PC與S7-200PLC之間連接（注意：PC通過PC/PPI電纜與S7-200PORT1口連接）

第三步：用串口電纜將S7-200PORT0端口與6RA70的PMU上X300、CUD1端子X172或CUD2端子X162相連 （注意：端口連接的規則，X300是3對3、8對8；CUD1端子X172是3對X172:58，8對X172:59;CUD2端子X162是3對X162:63，8對X162:64）

第四步：使用USS協議的初始化模塊初始化S7-200的PORT0端口

二進制值2#1000 0000 0000表示要初始化USS地址11的6RA70驅動裝置，波特率為9600，此波特率值要與PC/PPI電纜的設置相同，詳細情況請參看USS_INIT的幫助文件；此外我們還需要在6RA70裝置上設置相應的參數。

第五步：使用USS_CTRL模塊來控制USS地址11的6RA70裝置，為了運行6RA70需要按照表一設置參數

第六步：在編譯程序之前，選擇 Program Block -> Library then right mouse click: select Library Memeory. 在點擊Suggested Address選擇V存儲區的地址後點擊 “OK”退出，如下圖所示：

第七步：編譯程序並下載到S7-200，將RUN位置1，並在狀態表中輸入速度給定，在閉合CUD1端子X172:37和X172:38之後，6RA70直流裝置就可以運行起來了，如下圖狀態表所示：（注意：由於MM440和6RA70的工作方式和控制字的定義並不完全相同，所以你需要將OFF2、OFF3、F_ACK命令位連接到6SE70相應的參數，它們才能起作用；此外，由於控制字的定義不相同，DIR方向控制位並不能起到方向控制的作用；詳細情況請參看USS_CTRL的幫助文件及6RA70控制字的定義，在本例中將RUN信號連接到6RA70脈衝使能控制參數P661，而將速度給定連接到主給定參數P644）

4. 讀寫O2類型參數：如讀寫參數P078，使用USS_RPM_W和USS_WPM_W（這兩個功能塊用來讀寫16位無符號整數）

讀參數P078，其數據類型為O2，表示16位無符號整數；其程序塊如下圖所示：

在運行此程序塊的情況下，只要給S7-200的I0.3置一個上升沿，就可以完成一次對參數P078的讀操作，讀入的值被保存到R_P078。需要特別注意的是USS_RPM_W的INDEX值必須置1，因為MM440默認的是PXXXX.0參數組,而6RA70默認的是PXXX.1參數組，寫參數P078，其程序塊如下圖所示

在運行此程序塊的情況下，只要給S7-200的I0.4置一個上升沿，就可以完成一次對參數P078的寫操作，將W_P078中保存的值寫入到參數P078。USS_WPM_W的EEPROM是邏輯“0”時，寫入的值只被保存到6RA70的RAM中，當EEPROM是邏輯“1”時，寫入的值同時被保存到6RA70的RAM和EEPROM中，但向EEPROM中寫數據是有次數限制，最多不要超過50000次。

5. 讀寫I2類型參數：如讀寫參數P320，使用USS_RPM_W和USS_WPM_W（這兩個功能塊用來讀寫16位無符號整數）

讀參數P320，其數據類型為I2，表示16位有符號整數；其程序塊如下圖所示：

在運行此程序塊的情況下，只要給S7-200的I1.1置一個上升沿，就可以完成一次對參數P320的讀操作，讀入的值被保存到R_P320。

寫參數P320，其程序塊如下圖所示：

在運行此程序塊的情況下，只要給S7-200的I1.2置一個上升沿，就可以完成一次對參數P320的寫操作，將W_P320中保存的值寫入到參數P320。

讀寫參數P078的操作請參照如下的狀態表所示

參數P320的數據類型是I2(16位有符號的整數)，而USS功能塊USS_RPM_W和USS_WPM_W是根據MM440參數數據類型的定義，用來讀寫16無符號的整數。因此用這兩個功能塊讀寫6RA70的I2型參數時會產生一定的問題，如參數值是正數是能夠正確讀寫，當參數值是負數時，讀寫操作就無法實現了

6. 此外S7-200的USS庫還包括USS_RPM_D和USS_WPM_D（這兩個功能塊用來讀寫32位無符號整數），以及USS_RPM_R和USS_WPM_R(這兩個功能塊用來讀寫浮點數)
