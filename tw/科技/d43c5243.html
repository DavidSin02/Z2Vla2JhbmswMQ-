變分法遙感影像人工地物自動檢測

胡翔雲 , 鞏曉雅 , 張覓

武漢大學遙感信息工程學院, 湖北 武漢 430079

收稿日期：2017-12-01；修回日期：2018-04-17

基金項目：國家重點研發計劃（2016YFB0501403）；國家自然科學基金（41771363）

第一作者簡介：胡翔雲(1973-), 男, 博士, 教授, 博士生導師, 研究方向為遙感數據的信息提取、計算機視覺、模式識別及其應用

摘要：人工地物（建築物、道路、橋樑等）檢測是目標識別的一個重要組成部分。本文將人工地物檢測轉換為能量泛函數最優化問題。首先對遙感影像進行超像素分割，綜合圖像的顏色、紋理、梯度等信息，以超像素為單元計算圖像的顯著度信息，然後構建一個包含顯著性約束、面積和邊界約束、紋理約束及灰度方差約束的能量泛函數，通過變分法迭代求解能量泛函最小值，獲取目標前景部分即為人工地物區域。本文以重慶和廣東某地的遙感影像數據為例對算法進行驗證，將其與常見的人工地物目標提取算法，如C-V模型、MRF模型，以及當下研究較為熱門的深度學習算法進行對比。試驗結果表明，該算法能有效地檢測出遙感影像中的人工地物區域，並保證較低的誤檢率及漏檢率。論文對該方法與深度學習方法進行了一定的分析對比。

A Variational Approach for Automatic Man-made Object Detection from Remote Sensing Images

HU Xiangyun , GONG Xiaoya , ZHANG Mi

Abstract: Man-made object detection is important for object detection from remote sensing images.In this paper we propose a variational approach for man-made object detection which formulates the man-made object detection problem as a problem of variational energy optimization.In this method, an image is firstly segmented into superpixels, and the saliency map by combining image features such as texture, color and gradient is computed.In second step, we construct an energy function with saliency, area, edge, texture and intensity variance constrains.The energy function is solved via variational method to obtain the foreground, which is the detected man-made objects.The proposed approach on several remote sensing images is evaluated and compared with the C-V model, MRF model and deep learning based semantic segmentation.Experimental results show that the proposed approach can effectively detect man-made objects on remote sensing images with low false alarm and false negatives rates.The comparison and analysis with deep learning based method are also presented.

Key words: variational method man-made object energy function deep learning semantic segmentation

目標提取是數字攝影測量中矢量地圖測繪的基本問題之一，人工地物(道路、建築物、橋樑、機場等)是興趣目標的重要組成部分，在遙感影像中對其進行提取有著廣泛的應用，如城市地圖製作與更新[1]、災害預警與評估[2]、變化檢測[3]及軍事目標檢測[4-5]等。隨著光學遙感影像空間分辨率不斷提高，影像細節更為豐富，場景與目標也更為複雜多樣，人工地物的提取與檢測有了更多可用信息，也面臨著更多的問題與挑戰。

傳統的人工選取特徵的檢測算法主要分為兩類：①基於特徵描述的影像分割算法；②基於模型構建的目標檢測算法。基於特徵描述的影像分割算法將遙感影像分為前景(人工地物)和背景兩部分，通過恰當的特徵描述和分割算法，將人工地物從背景中分割出來。文獻[6]採用一個基於分形誤差和紋理邊緣約束的簡化Mumford-Shah模型對航空影像進行分割得到人工地物區域。文獻[7]通過提取分形特徵及閾值分割的方法對紅外影像中的人工地物進行檢測，並對多種分形特徵進行試驗，最終得出多尺度分形特徵(multi-scale fractal feature related with K, MFFK)精度較高的結論。文獻[8]針對傳統水平集演化方法調參複雜、計算量大的問題，提出將傳統的平均曲率正則項替換成高斯核的改進水平集演化方法，並在高分辨率遙感影像中對人工地物進行提取，在保證精度的同時提高了運算速度。基於特徵的分割算法能較完整地提取遙感影像中的人工地物，但由於影像細節豐富，不可避免地會出現邊緣破碎的現象，使用超像素替代像素作為影像分割的基本單位可以有效改善此現象。文獻[9]就對此進行了嘗試，首先通過多尺度NSCT(nonsubsampled contourlet)變換獲取影像的特徵，然後採用一種改進的分水嶺算法對影像進行過分割，最後以分割的超像素為單位採用分形布朗運動(fractional Brownian motion，fBm)模型對影像進行分割得到人工地物區域。而基於模型構建的目標檢測算法則是對人工地物進行建模，然後進行全圖搜索尋找符合模型描述的人工地物區域。文獻[10]使用神經網絡對人工地物進行建模，結合人工地物的幾何特徵如形狀、鄰接關係、尺度及主方向等進行監督訓練，最終用SVM分類器對影像進行分類，得到包含人工地物的矩形區域。文獻[11]採用最小二乘SVM訓練器對包含人工地物樣本的SAR影像進行訓練，並通過交叉驗證和網格搜索的方法對訓練參數進行優化，得到人工地物的預測模型，最終使用該預測模型對影像進行分類得到人工地物區域。文獻[12]依據圖像的平滑性及4種目標特徵進行訓練得到人工地物的預測模型，然後使用滑動窗口遍歷圖像得到包含人造目標的矩形窗口。基於模型的檢測算法能較為緊湊地提取包含人工地物的矩形區域，適用於人工地物稀疏、特徵明顯的影像，但無法獲取人工地物的邊緣信息，且對於背景複雜、人工地物繁多的影像適用性較差。

近年來，深度學習理論知識得到不斷髮展，深度學習的概念最早由文獻[13]提出，是指通過深度網絡對複雜函數進行逼近的學習過程。深度學習方法通過深度網絡自動化學習特徵，減少了人為設計特徵的不完備性，在很多應用中具有優異的識別或分類性能。一些學者將深度學習方法應用到遙感影像人工地物檢測中，並取得了不錯的效果[14-15]。但深度學習方法需要大量的訓練數據和計算開銷，對樣本分佈也有一定的要求，因此在很多應用中受到限制。此外，使用深度學習生成的模型較為複雜，難以解釋和分析，也難以直接進行改進，這些都是有待解決的問題。

本文致力於從遙感影像中提取邊界光滑、區域完整的人工地物目標，提出了一種基於人工選定特徵的變分泛函優化方法。變分泛函優化方法是計算機視覺中的一類經典的方法，其中泛函數是以函數為自變量的一類函數，對泛函數求極值的問題稱為變分問題。本文算法受Chan-Vese圖像分割模型[15]的啟發，根據人工地物在影像中顯著度和紋理與背景差異較大的特點，構建了一個包含顯著性約束、面積和邊界約束、紋理約束以及灰度方差約束的能量泛函數。採用變分法迭代求取局部極值，得到前景部分，即為人工地物區域。該能量函數同時包含超像素優化項和像素優化項，一定程度上解決了傳統的圖像分割算法在遙感影像中易產生破碎邊緣的問題。

1 基於變分法的人工地物檢測

自文獻[16]提出活動輪廓模型(active contour model, ACM)以來，基於曲線演化的圖像分割算法得到了極大的發展[17-18]。該類算法的基本思想是在圖像感興趣區域內初始化一條曲線，同時構造能量泛函數驅動該曲線運動，在能量泛函數取得極小值時使得演化曲線逼近目標邊緣，從而得到分割結果[19]。本文算法首先基於超像素對遙感影像進行顯著性計算，將計算得到的顯著性圖進行閾值分割，並將其邊緣作為曲線演化的初值，然後構建驅動曲線演化的能量泛函，分別包含基於超像素的顯著性約束項和基於像素的邊界、面積、紋理和灰度方差約束項，最後通過變分法迭代求取曲線演化的結果，得到分割的前景即人工地物部分。算法流程如圖 1所示。

2 試驗結果與分析2.1 試驗數據選取

本文設計了兩組試驗，用來評估本文算法在高分辨率遙感影像中提取人工地物的性能。其中試驗1進行本文算法與傳統的人工地物提取算法的對比，試驗2進行本文算法與深度學習方法的對比。試驗1選取多樣的人工地物場景，包括城區結構複雜的高大建築、鄉村低矮建築、城區立交橋、鄉村土路等，用於驗證本文算法在多種複雜的場景中均有較為良好的表現。試驗2則選取高分辨率的樣本數據並對其進行降採樣，用於驗證本文算法在不同的分辨率下均有較好的普適性。兩組試驗均選取重慶和廣東某地的航拍影像為試驗數據，影像分辨率均在1 m以內，(試驗數據可從http://earthvisionlab.whu.edu.cn/MSOD/index.html下載)。

2.2 人工地物檢測結果與分析

2.2.1 與傳統算法的對比

3 結語

本文采用能量泛函變分求解的算法，通過提取特徵及構建合理的能量泛函數，實現了高分辨率遙感影像人工地物區域的分割提取。該算法以流形排序計算得到的顯著性圖為初值，綜合利用影像的顯著性、紋理及灰度等多項特徵構建能量泛函數，使得人工地物提取的結果更為準確可靠。試驗選取了多幅包含不同人工地物類型的高分辨率航拍影像作為試驗數據對本文算法進行驗證，並與其他傳統的人工選擇特徵的提取算法及深度學習方法進行對比，結果表明本文算法具有優異的表現。對人工地物這樣的有比較明顯的、易於用比較緊緻的特徵來進行描述的目標，總體來說並不比深度學習方法效果差，且無須進行樣本的學習。

本文算法仍需改進，如難以區分顯著性較高(比較規則)的裸地與人工地物，以及被陰影遮擋的人工地物區域等。對於難以人工設計特徵進行描述的高度複雜目標，深度學習方法(例如深度卷積網絡)往往可以通過大數據的訓練較好地抽取一些高層次的、抽象的特徵。人的先驗知識、人工設計的特徵和深度學習框架進行結合，從而克服深度學習對大數據樣本的依賴等問題，並運用於遙感影像的自動解譯，是繼續探索的方向。
