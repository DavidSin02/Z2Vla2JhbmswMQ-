R語言中使用while循環求斐波那契數列前n項及奇數偶數和

2020-10-29
| 科技

while循環概述

while循環也被稱為“當”型循環。即當條件為真時，執行循環體中，否則執行循環體後的其它語句（也可能沒有）。在R語言中，while循環的語法格式如下：

while(cond){expr}

其中，cond是一個邏輯條件表達式，其值應為TRUE或FALSE；expr是循環體中的語句，即當cond條件為TRUE時要執行的表達式或若干語句。

下面以幾個經典的例子來說明while循環的具體使用方法。

輸出向量中元素

x <- c(11, 12, 13, 14, 15)

i <- 1

n <- length(x) #獲得向量的長度

while( i <= n) #i<=n時循環

{

print(x[i]) #輸出x中第i個元素

i <- i + 1 #使i加1

}

這個例子中先定義了一個向量x，i為循環計數，n為向量的長度（元素的個數），循環體中輸出後再使i加1，當i<=n時，反覆執行循環體，當i>n時，則退出循環體。

輸出1~200之間的奇數和及偶數和

sum.odd<-0

sum.even<-0

i<-1

while(i<=100)

{

if(i%%2==0)

sum.even = sum.even + i

else

sum.odd = sum.odd + i

i = i + 1

}

print(paste("奇數和為：" , sum.odd))

print(paste("偶數和為：" , sum.even))

代碼中的%%運算符是求模運算，即求兩個數相除後的餘數。我們知道，偶數除以2後，餘數為0，奇數除以2後餘數不為0（餘數為1），則我們可以使用這個特點來判斷一個數是偶數還是奇數。是偶數的話，我們把其加到sum.even中，否則加到sum.odd中，最後將其輸出。

求1~100之間所有整數的和

i <- 1 #循環計數

sum <- 0 #保存和

while( i <= 100)

{

sum <- sum + i #將每個i加入到sum中

i <- i + 1 #循環計數量加1

}

print(sum) #輸出和

這裡先定義了一個計數變量i，又定義了一個保存和的變量sum，在循環體中，每次將i加入sum中，並使i加1。當i<=100時，執行循環體，否則跳出循環執行循環體後面的語句：print(sum)。

求Fibonacci數列的前N項

i <- 3

N <- 20

F <- NA

F[1] <- F[2] <- 1

while(i <= N)

{

F[i] <- F[i - 2] + F[i - 1]

i <- i + 1

}

print(F)

求斐波那契數列的前n項是編程語言中較經典的例子。這裡的i是表示第幾項，這裡從第3項開始。斐波那契數列的前2項都是1，從第三項開始，每項是前兩項的和，利用這一特點遞歸求其它項。
