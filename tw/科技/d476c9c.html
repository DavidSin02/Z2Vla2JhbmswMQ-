使用樹莓派+DHT22做一個溫溼度計

2020-10-29
| 科技

先上一些照片。

硬件：樹莓派+DHT22溫溼度傳感器（白色的那個就是）。

網頁，顯示當前時間、最新一條監測記錄。

傳感器記錄寫入DB程序。

一個簡單的查詢數據程序。

整個程序下來，涉及了：

樹莓派、DHT22溫溼度傳感器、Python3、SQLite3、Nginx、JQuery。

緣起：

因為疫情原因，在家的時間比較長，一直是在書房工作學習。偶爾去趟客廳，現在天熱，客廳朝南，又沒開空調，就比較悶熱潮溼。我就比較想知道客廳的溫溼度。

家裡倒是有個物理溫溼度計，就是這玩意兒：


不過作為一個IT男，數據不數碼化，總覺得很不爽，所以就動了心思買個電子溫度計，家裡小米的多，就想買這個：

但是現在窮的叮噹響，又恰好翻箱底的時候，翻出了我的老樹莓派——確實很老了，是很多年前，剛出來的時候買的，GPIO引腳還是26針的，配套的SD卡也找著了，因為這款沒有WIFI，恰好又有富餘的網線，一裝系統還能運行，所以乾脆就自己搞吧。

過程：不能放外鏈，大家想用就搜搜吧。

1、安裝系統：

1.1 格式化SD卡：去SD官方網站下載專門的格式化工具


1.2 安裝樹莓派系統：

下載燒錄程序

下載系統

分別是：桌面+命令行版、桌面版、命令行版。我選的是第一個，回頭可以在配置裡將啟動改為進入命令行界面。

燒錄：

這個沒啥了，選系統鏡像，選SD卡，寫入。

1.2 系統配置：

升級：

sudo apt-get updatesudo apt-get dist-upgrade

通過桌面首選項，可以打開VNC、SSH服務。

VNC是桌面連接，我們在電腦上用VNC Viewer可以訪問樹莓派桌面。

SSH是遠程命令行訪問，在終端上使用命令"ssh pi@樹莓派IP"即可。

安裝Python3.

安裝Nginx、安裝SQLite。

安裝ufw防火牆，打開端口：

sudo ufw allow 80#Nginx端口sudo ufw allow 8080#Python的HTTP服務接口sudo ufw allow 22#SSH端口sudo ufw allow 5900#VNC端口

2、硬件安裝：

去某寶買了DHT22溫溼度傳感器+杜邦線，20塊錢，從汕頭發過來的。

傳感器一共有三個針腳“+、-、out”

我的樹莓派比較老舊，只有26個針腳：


如圖，+接入3.3v，-接入Ground，out接入GPIO2。

正負極可不能接反了，會燒的。

3、程序：

3.1 創建數據庫、表：

#!/usr/bin/pythonimport sqlite3conn=sqlite3.connect('temper.db')print("Open temperDB")c = conn.cursor()c.execute('''CREATE TABLE temper       (ID INTEGER PRIMARY KEY   autoincrement,       create_time DEFAULT (datetime('now', 'localtime')),       temp            CHAR(50)     NOT NULL,       humi        CHAR(50) NOT NULL);''')print('Table created successfully')conn.commit()conn.close()

3.2 下載讀取傳感器相關驅動：

sudo apt-get install build-essential python-devgit clone https://github.com/adafruit/Adafruit_Python_DHT.gitcd Adafruit_Python_DHTsudo python3 setup.py install

3.3 每隔10分鐘，讀取傳感器記錄寫入到數據庫

#!/usr/bin/pythonimport Adafruit_DHTfrom datetime import datetimeimport timeimport sqlite3sensor = Adafruit_DHT.DHT22#pin = 'P8_11'pin = 2def timer(n):    while True:        humidity, temperature = Adafruit_DHT.read_retry(sensor, pin)        if humidity is not None and temperature is not None:            print(datetime.now().strftime("%Y-%m-%d %H:%M:%S"))            print('溫度={0:0.1f}*C  溼度={1:0.1f}%'.format(temperature, humidity))            conn = sqlite3.connect('temper.db')            c = conn.cursor()            c.execute("INSERT INTO temper (temp,humi) VALUES ("+'{0:0.1f}'.format(temperature)+","+'{0:0.1f}'.format(humidity)+")")            conn.commit()            conn.close()        else:            print('Failed to get reading. Try again!')        time.sleep(n)timer(600)

3.4 Python服務端，返回JSON格式的最新記錄：

#!/usr/bin/pythonfrom http.server import HTTPServer, BaseHTTPRequestHandlerimport jsonimport sqlite3host = ('0.0.0.0', 8080)#開8080端口class Resquest(BaseHTTPRequestHandler):    def do_GET(self):        self.send_response(200)        self.send_header('Content-type', 'application/json')        self.send_header('Accexx-Control-Allow-Origin','*')#解決跨域問題        self.end_headers()        conn = sqlite3.connect('temper.db')        c = conn.cursor()        cursor = c.execute("SELECT *  from temper order by id desc limit 0,1")        for row in cursor:            data = {'id':row[0],'time':row[1],'temp':row[2],'humi':row[3]}        result="returnTemp("+json.dumps(data)+")"        self.wfile.write(result.encode())if __name__ == '__main__':    server = HTTPServer(host, Resquest)    print("Starting server, listen at: %s:%s" % host)    server.serve_forever()
