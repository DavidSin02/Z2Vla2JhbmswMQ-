五 變量-在程序中保存修改信息

在上一講的"根號"類中,用了一個整數(int)變量來保存輸入值. "參數"是一個字符串(String)數組. Java中還有其他幾種基本變量: boolean, char, byte, short, long, float, double

boolean 年紀尚幼 = true; // true或false,真或假char 姓 = '好'; // 單個字符byte 年齡 = 27; // 字節: -128到127, 即-2^7到(2^7-1)short 認識人數 = 1234; // 短整數: -32768到32767, 即-2^15到(2^15-1)int 不認識人數 = 1234567890; // 整數: -2^31到(2^31-1)long 星球數 = 123456789000000000L; // 長整數: -2^63到(2^63-1)float 身高 = 1100000000f; // 單精度浮點數: 2^-126到(2-2^-23)*2^127double 體重 = 999999999999.9; // 雙精度浮點數: 2^-1074到(2-2^-52)*2^1023

它們的範圍逐漸增大,可以根據需要選擇. 長整數後如果不加'L'(大寫的l),會被默認為整數值.
