iterm for mac版軟件介紹

iTerm2是終端的替代品，也是iTerm的繼承者。它適用於使用macOS 10.10或更高版本的Mac。iTerm2將終端帶入了您從未想過的功能的現代時代。

iTerm2 for Mac功能介紹

1、拆分窗格
將選項卡劃分為多個窗格，每個窗格顯示不同的會話。您可以垂直和水平切片，並以任何可以想象的排列創建任意數量的窗格。

請注意非活動窗格如何略微變暗，因此很容易看出哪個窗格處於活動狀態。

2、熱鍵窗口
註冊一個熱鍵，當您在另一個應用程序中時，它會將iTerm2置於前臺。終端永遠是一個關鍵的壓力。您可以選擇讓熱鍵打開一個專用窗口。這為您提供了一個隨時可用的終端（如Visor，Guake或Yakuake）。

3、搜索
iTerm2具有強大的查找頁面功能。用戶界面不受影響。所有比賽立即突出顯示。甚至提供正規表達支持！

4、自動完成
只需鍵入窗口中出現的任何單詞的開頭，然後鍵入Cmd-; 將彈出一個帶有建議的窗口。您正在尋找的單詞通常位於列表的頂部！

5、無鼠標複製
使用“查找”功能開始搜索文本。按Tab鍵將選擇範圍擴展到右側，或按shift鍵將選擇範圍擴展到左側。Option-enter粘貼當前匹配。

6、粘貼歷史記錄
粘貼歷史記錄可讓您重新訪問最近複製或粘貼的文本。您甚至可以選擇將歷史記錄保存到磁盤，以免永遠丟失。

7、即時重播
即時重播可讓您及時返回。這就像你的終端的TiVo！

8、可配置
將任何鍵映射到任何函數。為每個選項鍵分配單獨的功能 - 甚至重新映射所有修改鍵。您可以自定義iTerm2的外觀以滿足您的需求：啟用透明度，背景模糊，背景圖像等等。

iTerm2 for Mac軟件特徵

1、256色（或更多！）
在256色模式下，Vim以照片級逼真的方式爆炸：終端是一種色彩混合，代碼變得活躍。在版本3中，支持24位顏色。

2、可讀性
當有許多不同顏色或程序顯示難以閱讀的顏色組合時，您是否會丟失光標？使用智能光標顏色和最小對比度功能，您可以確保這些問題永遠消失。

3、鼠標報告
您可以使用鼠標定位光標，突出顯示文本，並使用鼠標報告功能在Vim和Emacs等程序中執行其他功能。

4、咆哮支持
您可以選擇接收活動，鈴聲等的Growl通知。隨意讓長時間的工作在後臺運行，確保您知道什麼時候完成。

5、Exposé標籤
與macOS的Exposé功能一樣，iTerm2在一個屏幕上顯示所有選項卡。更好的是，您可以一次搜索所有內容。繼續打開任意數量的標籤 - 您可以隨時找到您要查找的內容。

6、標記的個人資料
您是否需要為許多不同的主機存儲單獨的配置？iTerm2提供可標記和可搜索的配置文件數據庫，因此您可以輕鬆找到您正在尋找的配置文件。

7、多種語言
iTerm2具有出色的國際化支持，包括支持Unicode組合標記，雙寬字符和所有Unicode平面。

iTerm2 for Mac快捷操作

標籤
1、新建標籤：command + t
2、關閉標籤：command + w
3、切換標籤：command + 數字 command + 左右方向鍵
4、切換全屏：command + enter
5、查找：command + f

分屏
1、垂直分屏：command + d
2、水平分屏：command + shift + d
3、切換屏幕：command + option + 方向鍵 command + [ 或 command + ]
4、查看歷史命令：command + ;
5、查看剪貼板歷史：command + shift + h
