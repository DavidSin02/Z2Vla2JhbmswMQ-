垂直鼠標好在哪？細品羅技 MX Vertical 人體工學鼠標獨到之處

2020-10-29
| 科技

一向以人體工學見長的羅技所推出的MX系列鼠標一直受到商務辦公人群的喜愛，旗下包括了MX Master系列、MX Anywhere系列以及MX Ergo軌跡球。我個人在用的就是MX Anywhere 2S，它基本成為了我日常出行辦公必備的工具。

前作中的MX Ergo軌跡球造型更加別緻，除了增加萬向軌跡球外，還可以實現20°的角度調節。

商務辦公群體應該都會注意到，長時間使用鼠標後，手腕部分關節疼痛、麻木，拇指肌肉感覺無力，需要一段時間的緩和按摩才能緩解。這是由於普通鼠標是平握，支點是手腕下部，壓迫了腕管中的正中神經，使神經傳導被阻斷。

為了應對“鼠標手”這一傳統鼠標使用痛點，羅技（Logitech）在去年8月推出了MX系列旗下第一款無線垂直鼠標 MX Vertical 。這款鼠標可以同桌面形成 57°的夾角，使其相比普通鼠標減少 10% 的手部肌肉活動，從而減輕用戶使用鼠標的手部疲勞。那麼這款主打人體工學的垂直鼠標——羅技 Logitech MX Vertical，是否能夠如期所宣傳的為使用者帶來更加健康的手部姿勢，減輕使用者的腕部壓力，敬請跟隨我的視角瞭解一番吧。

首先，我們來看看產品開箱，羅技 MX Vertical 採用了比較典型的羅技包裝，正面印有產品樣圖。

半開放式的開箱效果，可以讓用戶快速掌握產品的外觀，貼合的透明保護層能夠讓購買者上手感受一下尺寸。

包裝的背面是 羅技 MX Vertical 的產品信息，以及主打賣點等。

【全家福】：可以看到除了 羅技 MX Vertical 本體外，配備了一個無線發射器、USB-C充電數據線以及眾多的產品保修卡說明書。

羅技 MX Vertical 的充電數據線為USB-C接口，這點好評，之前我的MX Anywhere 2S也只是Microusb接口。同時在顏色上同鼠標保持了一致，這根數據線也可以將 羅技 MX Vertical 轉化為有線鼠標使用，在鼠標沒電時可以救急。

因為 羅技 MX Vertical 是有線、藍牙、2.4G三模連接可選，所以自然也配備了一個2.4G的無線發射器，不過比較可惜的是它不能存放在鼠標內部，而是單獨收納，所以存在一定的遺失風險，建議還是插在常用設備上。

終於到了外觀展示環節了，通過下圖360°的動圖展示，羅技 MX Vertical 之所以被稱作垂直鼠標，也是因為其獨特的造型。鼠標的中心並非如普通鼠標般平放在桌面上，而是像斜向上傾斜，讓手部自然的貼合。

從 羅技 MX Vertical 的正面視角可以看到左側內凹，右側圓潤，典型的右撇子設計，為手心預留了大量支撐區域，拇指則緊緊貼合內側，正面下方還有個USB-C接口，與充電數據線配合使用。

羅技 MX Vertical 正面的夾角是57°，我經過反覆的感受後才發現精妙之處，這個角度恰恰是我手部自然放鬆的區間。想像一下你手握一瓶脈動，垂直握住時手腕向外彎曲，當你在喝脈動時，手腕向內是不是最舒服的角度。

這應該也是羅技經過反覆測量設計後的結果，根據羅技的說法，MX Vertical 的設計旨在模擬“自然握手位置”。這有助於在長時間的計算過程中減少前臂張力，與普通鼠標相比，傾斜的 羅技MX Vertical 可以減少高達10%的肌肉活動。

右測視角，羅技 MX Vertical 有點像一個佇立的河蚌，右上角設計著鼠標的左右鍵位和滾輪。側面的手部貼合區域採用了橡膠材質，同時這裡採用的是稜角分明的紋理，相比於光滑的弧面有助於提升質感，增大摩擦。

羅技 MX Vertical 的頂部是一個長條形平面，左邊印有Logi標誌，右側則是一個可自定義的多功能按鍵，默認用以切換兩檔dpi（鼠標靈敏度）。

從 羅技 MX Vertical 的背面視角可以看到這裡設計了一個LED指示燈、兩個可自定義的多功能鍵。

羅技 MX Vertical 的底面有點像是達芬奇畫過的雞蛋，兩個弧形的腳貼實現了整個鼠標在平面上的划動。底面的頂部是電源開關，中部是傳感器，其下是設備切換鍵（3個LED），最下方是鼠標的認證及系列號等內容。

由此可見，羅技 MX Vertical 在按鍵和接口的分佈定位上是與MX Master系列、MX Anywhere系列基本保持一致的。

【上手感受】：羅技 MX Vertical 的上手感受主要表現為三個階段，初次上手時會有一種不適應感，畢竟以歐美人手型設計的鼠標始終會有些偏大，不過好在按鍵的位置均較為靠後，使用沒有問題。經過大概半小時的使用後，我已經能夠熟練掌握 羅技 MX Vertical 的使用技巧。

使用一天以後，才發現 羅技 MX Vertical 相比於普通的鼠標，長時間使用也不會有過多的負擔。我已經在讓師弟重點感受 羅技 MX Vertical 了，準備帶他跳坑入手一個。

羅技 MX Vertical 的滾輪部分和我的 羅技 MX Anywhere 2S 較為接近，力度適中，較為容易上手。

鼠標的定位感上，因為 羅技 MX Vertical 體積偏大，還需要我的不斷適應。讓我拿來玩遊戲，我肯定是拒絕的，但是當個生產力工具，我還是很歡迎的。

因為 羅技 MX Vertical 涉及到了多功能鍵的自定義、鼠標分辨率的調節、Logitech Flow技術，所以它也能夠適配 Logitech Options 軟件，通過官網下載安裝即可。

首先，Logitech Options 可以對鼠標的滾輪按鍵、頂部按鍵以及左側按鍵一共四個鍵位進行自定義。

頂部按鍵較為獨特，默認是指針速度（DPI）的兩檔切換，最大支持4000dpi。

左側兩個鍵位我傾向於設置為音量+-，畢竟翻頁有了滾輪，多媒體是我常用的場景。

其次，Logitech Options 也能夠調節 羅技 MX Vertical 的平滑滾動、滾輪方向等。

最後， 羅技 MX Vertical 支持自家的Logitech Flow技術，也就是在多個設備無線切換，這種功能要求每個設備上均安裝了Logitech Options，並同鼠標配對。

例如很多程序猿小哥哥們，經常擁有多個屏幕進行代碼編輯處理，此時擁有一個可以支持Logitech Flow跨屏幕操作的鼠標就算的上不錯的生產力工具了。

【鼠標操控】：為了測試不同分辨率（DPI）下的 羅技 MX Vertical ，只好將DPI換為最高、中等、最低三檔進行了下圖測試。實際的感受是中檔以下蚊香圖和井字都非常好畫，但是在最高

【充電續航】：羅技 MX Vertical 搭載了240mAh的可充電電池，完整充電一次能夠使用4個月，並且 羅技 MX Vertical 也支持快充功能，充電一分鐘可提供3小時使用時間。

實際體驗是我從開箱到使用了一週之後，Windows的藍牙面板上面數據顯示仍然是100%的電量標誌，由此可以判斷出 羅技 MX Vertical 的續航應該真的很強，這點非常貼合商務人群不喜充電的需求。

【辦公場景】：對於需要長時間使用鼠標的用戶而言，羅技 MX Vertical 就可以發揮巨大作用了，一方面幫助用戶減輕手部負擔，另一方面提升效率。

對於我們這些需要長時間端坐在電腦前研讀文獻的學生黨而言，羅技 MX Vertical 都算的上得心應手。無論是文獻的上下翻頁還是具體內容的標識都可以盡在掌握。唯一可能比較遺憾的就是，價格對於學生黨不夠友好，希望能搞點學生優惠吧。

經過本文的測評之後，我們最後聊聊 羅技 MX Vertical 的產品優勢和值得改進之處。

【產品優勢】：

1.羅技 MX Vertical 的人體工學設計，相比於普通鼠標，能夠有效減少使用者手部的負擔，保證長時間工作。

2.羅技 MX Vertical 的按鍵豐富，配合 Logitech Options 可以實現諸如多功能鍵、DPI兩檔切換、Logitech Flow等精細化的功能體驗。同時，羅技 MX Vertical 支持無線、藍牙、有線三種模式的連接，可以滿足多種使用場景。

3.羅技 MX Vertical 續航表現卓越，同時支持快充，對於在外旅行的用戶算的上利好的消息。

【值得改進】：

1.羅技 MX Vertical 的初期上手會有點難度，主要是定位的把控還需要一定的習慣過程。

2.羅技 MX Vertical 的體積稍大，可以考慮為手小的用戶如女性群體提供尺寸更加小巧的版本，也便於外出攜帶。

3.羅技 MX Vertical 的定價仍然高昂，建議下調定價空間，讓更多用戶有機會享用。
