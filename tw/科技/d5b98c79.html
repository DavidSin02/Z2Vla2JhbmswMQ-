GO語言學習資源整理

2020-11-14
| 科技

Go語言是谷歌發佈的一套編程語言，因為其簡單的部署、良好的併發性等特點受到很多企業的青睞，國外如Google、AWS、Cloudflare、CoreOS等，國內如七牛、阿里等都已經使用Golang開發其雲計算相關產品。

因此也有很多小夥伴選擇Go語言，其實也對，畢竟大眾化的語言Python、Java、C、C++等崗位競爭激烈，那麼為何不選擇一門競爭性相對而言比較小卻也有前景的一門語言呢。

那麼要學習Go語言就回到我一直說的問題啦“哪裡可以找到好的學習資源？”，其實Go語言還是可以在網上找到很多學習資源的，畢竟現在學習Go語言的、用的人還是蠻多的。

但是，我還是按照我的方式來總結一篇Go語言的學習資源吧，會從以下幾個方面整理：

  • 1）在線學習教程：比較系統全面，非常適合新手入門學習；

  • 2）書籍：畢竟很多人學習一門語言是從書籍開始的；

  • 3）其他Go語言的整理資源：很多比較好的Go語言資源總結整理，推薦看；

  • 4）Go語言中文社區：專門的Go語言中文社區，可以交流討論，感興趣的可以看看；

  • 5）其他：其他與Go語言相關的一些整理；

一、在線學習教程

主要整理一些比較系統的Go語言學習教程，適合新手學習；

1、官網：

要學習一門語言，當然首先就是推薦官網啦，官網上有各種想要的資源和文章，一定要看，不過門檻就是英文要過關： The Go Programming Language

官網裡面新手推薦看：

  • A Tour of Go（Go語言編程指南） （中文版本在這裡： Go 指南）：主要是帶你瞭解基礎的Go語言和使用方法，重點是有個在線編譯器，可以在瀏覽器裡面寫Go語言，直觀感受Go語言；

  • How to Write Go Code：也是帶你入門學習Go語言的，類似教程吧，英文的，新手同樣可以看；

  • https://golang.org/ref/spec：Go編程語言的參考手冊，英文的；

官網是要看，但是可以結合其他一些在線教程一起學習，下面推薦一些教程，分為文檔形式和視頻形式的教程，可以按照自己的喜好進行選擇。

2、在線教程——文檔：

  • Go by Example （中文版：GO - Go by Example 中文版）：純文檔；

  • 實驗樓Go語言編程入門教程：純文檔+在線開發環境；

  • 易百教程Go語言入門：純文檔；

  • 菜鳥教程Go 語言教程：純文檔；

以上4個教程都是純文檔形式的，但是都非常系統，非常適合新手入門學習；

3、在線教程——視頻：

  • 51CTO上 跟無聞學Go語言：Go編程基礎視頻教程

  • 慕課網上 Go語言第一課

  • 網易雲課堂Go編程基礎

以上3個教程都是視頻形式的，喜歡看視頻的可以選擇其中一個教程進行入門學習；

4、其他

  • Go語言體驗：免費的golang在線開發環境，可直接通過瀏覽器訪問進行Go語言的學習與編程。

  • 學習一門語言自然少不了動手實戰，那麼實戰項目教程可以去各大學習網站找，一般都會有，就不一一列出來了，需要的自己去看就行，推薦實驗樓上的實戰教程吧，畢竟文檔的話隨時都可以看，就像看文章一樣：Go語言實戰項目

二、GO語言相關書籍：

  • 《Go語言聖經 》（中文版：Go語言聖經 · GitBook）：推薦的人較多，可以看看；

  • 《深入解析Go》：gitbook上開源書籍 ，免費可以看的，作者主要分享對Go語言的內部實現的一些研究；

  • 《 Go Web 編程》：gitbook上開源書籍 ，主要是關於web編程方面的，同樣是免費的；

  • 一本學習 Go 語言的免費電子書：代碼託管在github上的，可以克隆項目，書籍主要是教你如何使用Go語言；

其實，書籍有很多很多，主要找到自己覺得不錯的，畢竟每個人對於每本書的看法不一樣，所以就推薦以上4本吧，如果你感興趣就可以看看，不感興趣就可以自己去找找；
