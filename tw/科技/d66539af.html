磷礦石的浮選工藝

2020-10-29
| 科技

不同礦石類型的選礦工藝 1.1我國磷礦石選礦近幾年的研究和發展較快，從技術上來說與國外較為接近，技術和經驗比較成熟。根據不同礦石性質通常採用如下的選礦方法：硅質磷礦採用Na SiO，等抑制硅酸鹽礦物而用陰離子捕收劑正浮磷酸鹽礦物的正浮選工藝，分選效果較好，如寧夏賀蘭山礦，工藝流程見圖2。圖2 硅質磷礦正浮選工藝流程沉積鈣質磷塊巖採用H sO 或H PO 等抑制磷酸鹽，陰離子捕收劑浮選白雲石、方解石等碳酸鹽礦物的單一反浮選工藝，工藝流程見圖3。對於含P O27.0% ，MgO 4.47% ，SiO，7.87%的原礦，用此單一反浮選工藝可以獲得磷精礦P O 32.89% ，MgO 1.01% ，磷回收率95.32% 的良好選礦指標。如想進一步提高品位，可採用正-反浮選工藝，即加Na CO 、Na SiO 等抑制硅酸鹽，陰離子捕收劑浮選磷酸鹽及含鈣鎂等碳酸鹽礦物，然後再用H sO 或H PO 將pH值調至5.5～ 6.0以抑制磷酸鹽，陰離子捕收劑反浮選碳酸鹽礦物，這樣可使磷精礦P O 含量提高到35.17% ，MgO降至0.78% ，R2O 31.97% ，磷回收率91.98% 的良好選礦指標，如貴州甕福磷礦，工藝流程見圖4。圖3 沉積鈣質磷礦單一反浮選工藝流程圖4 沉積鈣質磷礦正-反浮選工藝流程沉積變質型硅.鈣質磷灰巖屬易浮磷灰石型磷塊巖，採用Na CO 、Na SiO 等抑制硅、鈣礦物，陰離子捕收劑正浮選磷灰石的直接浮選工藝，對含P：O 8.0%的原礦，經此工藝可以獲得磷精礦P O 品位大於35% ，磷回收率83% 的良好指標，如湖北大悟縣黃麥嶺選礦廠。沉積硅.鈣質磷塊巖類磷礦石即膠磷礦是磷礦石中最難選的一種。它儲量很大，佔全國磷礦總儲量的85%以上。膠磷礦是一種結晶微細的與硅酸鹽、碳酸鹽膠結在一起的細晶磷灰石，晶格中的ca“可被Mg、Mn、Sr、Na、K、Sn等元素的離子所置換，磷酸根離子也可被其它陰離子基團所替代，造成表面性質發生變化。這種磷灰石嵌布粒度很細，與脈石礦物的單體解離較困難，同時，脈石礦物不僅含有硅酸鹽礦物，還含有白雲石、方解石等與磷灰石可浮性相近的雜質礦物，使其分選變得複雜而困難。該類型磷礦採用正-反浮選(工藝流程見圖4)、反.正浮選(先用H sO 或H PO 抑制磷礦物，陰離子捕收劑反浮選白雲石等碳酸鹽礦物，然後用石灰、Na CO 、Na SiO，等抑制硅酸鹽礦物而用陰離子捕收劑正浮選磷酸鹽礦物，工藝流程見圖5)或雙反浮選(先用H sO 或H PO 抑制磷礦物，陰離子捕收劑反浮選白雲石等碳酸鹽礦物，然後礦漿經脫泥後再用陽離子捕收劑反浮選硅酸鹽礦物，工藝流程見圖6)的選礦工藝進行分選，前兩種工藝都不容易得到良好的分離效果，後者的分選效果較好，但對選擇性好的高效陽離子捕收劑及選礦工藝尚需做進一步的研究，如湖北宜昌磷礦、荊襄磷礦等。圖5 膠磷礦反-正浮選工藝流程 圖6 膠磷礦雙反浮選工藝流程風化型或含泥質礦物多的磷礦石，如摩洛哥的胡卜加選礦廠、我國雲南滇池周圍的風化磷礦石，可將磷礦石進行擦洗脫泥。對於在採礦過程中混入較多圍巖或較貧的磷礦石，可先採用重介質旋流器一浮選的聯合工藝流程，即在粗粒級情況下通過調整介質密度來脫除密度輕的大粒級脈石，提高入浮礦石的P 0 品位，以降低選礦成本。上述這些規律是我國和世界磷礦選礦工作者經過幾十年的科學研究，近幾年來通過大量的選礦生產實踐得出的，也是近些年磷礦選礦通過科技進步所取得的進展。 2.2 湖北省磷礦選礦的基本狀況 鄂東地區大悟礦區和黃梅礦區，目前建有兩座採用單一正浮選工藝的選礦廠。由於該礦區磷礦石屬變質磷灰巖，選礦相對容易，選廠運行良好，精礦P 0 品位都在35% 以上。荊襄礦區主要有大峪口公司磷礦選礦廠。該選廠採用正一反浮選工藝，選礦指標較為理想，精礦P 0 品位大於31% ，磷回收率大於80% ，目前運轉正常，說明湖北省膠磷礦選礦在技術上和經濟上是可行的。 宜昌礦區目前有兩家重介質選礦廠，但其精礦產品不能達到生產磷酸二銨的要求。因此，在宜昌礦區採用正一反浮選或雙反浮選工藝建設浮選廠處理中低品位磷礦石是湖北省磷礦選礦必須面對的一個問題。 3 磷礦選礦中存在並需要解決的問題 3.1 推行“全層開採”。堅決制止“採富棄貧”我國的磷礦資源中富礦僅佔6.75%(約8.57億t)，中低品位的貧礦卻很多。2003年，全國生產磷礦2 488萬t，當年出口磷礦石750萬t，佔總量的32% ;2004年全國開採磷礦2 617萬t，當年出口磷礦石625萬t，佔總量的23.8% ;出口的磷礦石含P 0 都在32% ～34%之間，屬高品位的富礦，兩年中僅用於出口的富礦就分別佔當年全年磷礦生產量的32% ～23.8% ，如果再加上國內磷化工所用的富礦，那麼富礦的開採量佔全年的磷礦總開採量就更大。2006年，湖北省生產的磷礦石為1 200萬t，其中富礦為800萬t，400萬t是中低品位的貧礦，富礦佔總開採量的66% 。這些充分說明我國磷礦開採生產中“採富棄貧”的現象是多麼的普遍和嚴重。 以湖北磷礦山為例，宜昌磷礦山(儲量約8、61億t)呈兩貧礦帶(平均含P 0 15% ～18%)夾一富礦帶(平均含P 0 32.79%)的夾心餅乾礦層的結構，荊襄磷礦由下而上分4個礦層，第一層平均含P 0 24% ，第三層平均含P 0 17.32%。礦層厚度薄，只有幾米到十多米，而且貧礦層和富礦層相鄰緊靠，如果單獨採富礦，勢必要丟棄相當多的貧礦。如宜昌磷礦採1 t富礦就要丟棄6～8 t貧礦，富礦開採後形成的礦坑將造成貧礦難以開採而永久的被丟棄，資源浪費十分嚴重。如果推行全層開採，即在開採富礦的同時將相鄰的貧礦一起採出，富礦可直接使用，貧礦經過選礦生產出磷精礦(P 0 含量大於30%)供磷化工企業使用，這樣才能保持我國及湖北省磷化工業的長期持續高效發展，否則富礦開採十多年後將面臨無礦可採的窘迫局面，屆時再興建大批選礦廠就會措手不及，我國磷化工將陷入不可想象的困境當中。 3.2 加強選礦科學研究。開發出合理高效的貧礦選礦新藥劑、新工藝及新設備 3.2.1 新型浮選藥劑的研究不論是湖北磷礦選礦，或是中國磷礦選礦和世界磷礦選礦，其中存在的最大困難是缺乏選擇性高、專屬性強的高效浮選藥劑，如正浮選工藝中能耐低溫、溶解度高、選擇性好、效果顯著的陰離子捕收劑，也缺乏高效的抑制劑和在雙反浮選工藝中浮選脫硅的高效陽離子捕收劑。加強這些浮選藥劑的研究，側重性的開發出高效新藥劑將會解決當前磷礦選礦中的最關鍵問題，特別對於沉積型磷塊巖的選礦，將使磷礦選礦科技進步有著飛躍的發展。 3.2.2 工藝流程的研究 目前已開發出不少磷礦浮選的新工藝，而對於重一浮聯合流程要加強注意。重介質旋流器只是一個預選拋去粗粒尾礦的設備及工藝，只能用於特別貧礦或圍巖混入多的礦石中，其生產出的精礦還需要浮選進行精選才能用作磷化工的原料。如目前國內某些磷礦重介質選廠已建成單一重介質旋 流器工藝，生產實踐表明，精礦P 0 品位難以達到30% 以上，尾礦中P 0 含量則高達15% ，尾礦中磷損失嚴重，回收率低，生產與試驗、設計相差甚遠。原因是磷礦物密度為3.05～3.10 g/cm ，而脈石礦物密度為2.8～2.85 g/cm ，二者相差很小，選用的重介質密度應為2.9 g/em ，這在工業生產中難以精確保證重介質的密度穩定在2.9 g/em ，稍有偏差就會造成分選效果變差，選礦指標波動頻繁。因此，應該採用重一浮聯合工藝，使其既能使精礦達到質量要求，又能降低尾礦 P 0 的品位，減少資源的損失，與單純的浮選工藝相比還能使選礦成本有所降低。 3.2.3 浮選柱在浮選流程中的應用20世紀90年代後期，加拿大徹底解決了浮選柱的發泡裝置及其控制系統。目前，除國外已廣泛應用外，浮選柱在我國江西德興銅礦、湖南柿竹園多金屬礦、鞍鋼弓長嶺鐵礦等有色金屬、冶金行業已成功應用。 與傳統的浮選機相比，浮選柱在浮選中具有分選性能好，能耗低，佔地面積小等諸多優點，特別是浮選的選擇性高，預計能明顯提高磷礦浮選的效果。 磷礦開採　　磷礦開採有露天開採和掘井開採兩種採礦技術。美國佛羅里達、田納西、北卡羅來納等州及西部一些礦區多用露天採礦技術，而掘井採礦技術只在西部一些地區應用，東部沒有。　　多數礦區採出的礦石須經一定處理，以分離出原礦中的含磷酸鹽部分。有時還須將礦石粉碎，含磷酸鹽部分經過洗礦、篩礦、分礦、搖床附聚和浮選等一系列複雜工序而與雜質分離，並得到濃縮。乾燥後，便適用於磷肥生產。圖10-3所示為美國佛羅里達州篩選磷礦石的各個步驟。
