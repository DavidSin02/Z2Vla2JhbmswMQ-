防鏽術語(續)

2020-11-14
| 科技

1、主題內容和適用範圍

本標準規定了金屬製品防鏽專業在科研、生產中常見的專用術語和定義。本標準適用於金屬製品在加工過程、貯存運輸中的防鏽。本標準不包括非專用的術語。

2、防鏽處理

1） 清洗

cleaning

除去金屬製品表面汙物，包括油脂、機械雜質、鏽、氧化皮等的過程。是防鏽處理的第一道工序。

2） 塗抹

smearing of rust preventive oil

用手工塗敷防鏽油脂。

3）浸塗防鏽

applying preventives by dipping

製品在防鏽油或防鏽液中浸入後取出。

4 ）浸泡防鏽

immersion in liquid preventives

製品浸泡在防鏽油或防鏽水中以防鏽。

5）熱塗防鏽

rust prevention by applying hot prevention material

在加熱熔融的防鏽脂中進行浸塗。

6） 冷塗防鏽

rust prevention by applying cold prevention material

將製品在室溫下直接浸塗防鏽油或防鏽液。

7） 噴塗防鏽

rust prevention by applying liquid material

將具有流動性的防鏽材料，噴塗到金屬製品需防鏽的面上。

8） 噴淋防鏽

protection by spraying aqueous preventives

將防鏽水噴淋到金屬製品上以防鏽，一般用於大量工件中間庫防鏽。

9） 內包裝

interior package

直接或間接接觸產品的內層包裝，在流通過程中主要起保護產品、方便使用、促進銷售的作用。

10） 外包裝

exterior package

產品的外部包裝，在流通過程中主要起保護產品、方便運輸的作用。

11） 單元包裝

unit package

將若干個產品包裝在一起，作為一個銷售單元的包裝。

12）繭式包裝

cocoon package

在整臺產品周圍作網噴塑料似繭以封存防鏽。

13） 泡狀包裝

blister package

用透明膜作成與物品相似的堅固泡狀膜，邊緣作成法蘭，封入物品後，將法蘭固緊在底板上。

14）貼體包裝

skin package

包裝的透明膜緊貼在物品周圍，通過抽真空，將膜似皮膚一樣緊貼在製品表面，形成保護層。

15）防水包裝

water proof(water resistant)barrier packaging

使用防水材料製作的袋子、封套或容器包裝製品，然後封口。金屬製品需要時使用防鏽材料防鏽。

16） 防溼包裝

Water-vapor proof barrier packaging

使用防溼材料製作的袋子、封套或容器包裝製品，然後封口。包裝內一般使用計算數量的乾燥劑，以保持包裝內有低的相對溼度。

17） 環境封存

preserved in controlled atmosphere

除去包裝空間內致鏽的因素，以保證金屬製品不鏽蝕。

18） 乾燥空氣封存

preserved in dry atmosphere.

用降低包裝空間內相對溼度的方法，以保護金屬製品。

19） 充氮封存

preserved in nitrogen

用乾燥純淨的氮氣充於包裝空間內，以保護金屬製品。

20） 收縮包裝

shrink packaging

物品用透明膜包裹，通過加熱使膜收縮而緊貼於物品上的包裝。

3、實驗方法

1） 防鏽性能試驗

test for rust preventing ability

評價防鏽材料防鏽性能的試驗。

2） 人汗防止性試驗

synthetic perspiration prevention test

考察置換型防鏽油塗覆金屬製品上後，能抑制因裸手持取製品而致鏽蝕的能力的試驗。

3） 人汗置換性試驗

synthetic perspiration displacement test

考察置換型防鏽油置換金屬製品上手汗因而防止手汗鏽蝕的性能的試驗。

4） 人汗洗淨性試驗

synthetic perspiration cleaning test

考察置換型防鏽油對金屬製品上所粘手汗的清洗能力的試驗。

5） 防鏽材料的溼熱試驗

humidity cabinet test for rust preventives

在試驗室用加速方法評價防鏽材料在溼熱條件下防鏽性能的試驗。

6） 防鏽材料的鹽霧試驗

salt spray test for rust preventiyes

在試驗室用噴鹽水的方法模擬海洋環境評價防鏽材料防鏽性能的試驗。

7） 防鏽材料的老化試驗

accelerated weather'ing test for rust preventives

在試驗室用人工光源及淋水模擬日晒雨淋，強化氣候老化作用，以評價防鏽材料防鏽及耐候性能的試驗。

8） 防鏽材料的百葉箱試驗

shed storage exposure test for rust preventives

防鏽試樣或防鏽包裝試樣在百葉箱中較長時間放置，以評價其耐鏽蝕破壞性能的試驗。

9） 防鏽材料的現場暴露試驗

field exposure test for rust preventilves

防鏽試樣置於使用現場，以評價防鏽材料在使用條件下耐鏽蝕破壞性能的試驗。

10） 防鏽材料的室外暴露試驗

outdoor exposure test for rust preventives

防鏽試樣置於一定條件的大氣暴露站暴露，以評價防鏽材料的耐鏽蝕破壞性能的試驗。一般只 用於防鏽性很強的材料。

11） 加速凝露試驗

Accelerated condensation test

使防鏽試樣與環境溫度保持較大溫差，試樣表面凝露加強而使腐蝕加速進行的溼熱試驗。

12） 防鏽材料的間浸式試驗

altemate immersion test for rust preventives

將試樣在腐蝕溶液中浸漬，經規定的時間後，取出晾乾，反覆進行到規定的週期，考察試樣腐蝕、生鏽、變化等情況的試驗。

13） 防鏽材料的鹽水浸漬試驗

salt water immersion test for rust preventives

防鏽試樣浸泡在氯化鈉溶液中，考察防鏽膜層抗鹽水腐蝕能力的試驗。

14） 靜力水滴試驗

static water drop test

檢測防鏽油品抗靜力水滴腐蝕性的試驗，常用於油溶性緩蝕劑的篩選。

15） 水置換性試驗

water displacement test

考察防鏽油料置換金屬表面上附著水分，以防止鏽蝕的試驗。

16） 防鏽脂的低溫附著性試驗

low temperature adhesion test for petrolatum type rust preventive

考察防鏽脂在低溫時在金屬表面附著性能的試驗。

17） 防鏽脂的流失性試驗，

flow point test for petrolatum type rust preventive

考察防鏽脂在立面上塗覆時，在高溫下是否流失的試驗。

18） 防鏽材料的腐蝕性試驗

corrosiveness test for rust preventive

考察防鏽材料對金屬材料侵蝕性的試驗。

19） 氣相防鏽能力試驗

vapor inhibitability test

考察氣相防鏽材料的氣相防鏽效果的試驗。

20） 氣相防鏽材料與有色金屬的適應性試驗

test of compatibility with non-ferrus metals

考察氣相防鏽材料對有色金屬適應性的試驗。

21） 氣相防鏽材料與包裝材料的適應性試驗

test of compatibility with barrier materials

考察氣相防鏽材料與包裝材料（一般指塑料膜）的適應性的試驗。

22） 防鏽材料的篩選試驗

sieve test for rust preventives

從大量的防鏽材料中粗選較好的材料所使用的簡單的試驗。

23） 熱焊封試驗

heat-sealing test

對於可熱焊封的包裝材料，熱焊封后檢驗在一定負荷下焊縫是否分離的試驗。

24） 防鏽材料的貯存穩定性試驗

storage stability test for rust preventive

防鏽材料在規定的條件下貯存後，考察其質量性能變化的試驗。

25） 防鏽油脂的氧化安定性試驗

antioxidation test for petrolatum type preventives

防鏽油脂試樣耷規定氧氣壓力和溫度的氧彈中氧化，在規定的時間間隔，評定其氧化後穩定性的試驗。

26） 防鏽油脂的疊片腐蝕試驗

sandwich test for rust preventive oils

考察塗有防鏽油脂的試片重疊面的防護性能的試驗。

27） 防鏽試驗中的鏽蝕評級

the rust grade evalution in rust prevention test

對於防鏽試驗中試驗樣品鏽蝕情況的評價分級。

28） 漏汲試驗

leakage test for preservation package

考察防鏽包裝件密封性的試驗。

29） 防鏽包裝件的循環暴露試驗

cyclic exposure test

考察防鏽包裝件經過熱、冷、水淋等週期循環暴露試驗後，其中的製品是否受保護，不鏽蝕的試驗。
