​Python劍橋真題詞頻統計

提示：

最好還是要學以致用，小編自主蒐集了19年最近的14份劍橋真題之後，通過Python提供的jieba第三方庫，對所有的文章信息進行了詞頻統計，並選擇性地剔除了部分簡易詞彙，比如：數字，普通冠詞等，較懶，未清除乾淨。

Python代碼如下：

# 先進行分詞words = jieba.cut(text, cut_all = False, HMM = True)#cut_all:是否採用全模式#HMM：是否採用HMM模型
word_ = {}for word in words: if (word.strip() not in stwlist): if len(word) > 1: if word != '\t': if word != '\r\n': # 計算詞頻 if word in word_: word_[word] += 1 else: word_[word] = 1 # 將結果保存為元組word_freq = []for word, freq in word_.items(): word_freq.append((word, freq)) # 降序排列word_freq.sort(key = lambda x:x[1], reverse = True) #輸出前3500個詞彙for i in range(3500): word, freq = word_freq[i] print('{0:10}{1:5}'.format(word, freq))
