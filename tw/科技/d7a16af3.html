三星 One UI 3.0 現已推送，首批 3000 名額已無

2020-10-29
| 科技

IT之家 10 月 26 日消息 據網友投遞，三星 Galaxy S20 系列國行現已開啟 One UI 3.0 的首次內測，官方首批僅放出了 3000 個內測名額，目前已被爭搶一空。

據蓋樂世社區網友反饋，One UI 3.0 系統公測 Beta1 的鎖屏動畫相比之前更絲滑，指紋圖標有所不同，支持鎖屏指紋圖標常亮，鎖屏密碼鍵盤重做，整體風格感覺有點像 iOS 風格。

此外，主頁依然不支持第三方圖標包，但桌面圖標的 Shortcut 樣式發生了變化。

相比前代變化比較大的是通知欄，新版本的通知欄加入了高斯模糊（One UI 3.0 在很多地方都加入了高斯模糊），只能選擇 3×4 排列，設備和媒體的按鈕移到了快捷功能按鈕的上方，通知窗口支持詳細和簡潔兩種風格的選擇。

而新版本的撥號界面取消了直接選擇卡 1 或者卡 2 撥號的功能，只有輸入數字之後才能選擇，而且切換通話卡的動畫還偶爾會發生掉幀（可能由於測試版未完全優化），但新版本已經可以設置通話界面的背景，並對頭標進行了重新設計。

其他方面，三星輸入法加入了新的雙拼方案，相比 UI 2.5 增加了多種方案；圖庫的設置現在移到了界面右下角，底欄新增加縮略圖展示特性；內存管理界面新增基本功能——應用清理，可以卸載預裝應用；電池設置增加了一項 “增強處理”。
