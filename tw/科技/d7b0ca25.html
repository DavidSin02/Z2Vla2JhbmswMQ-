尚學堂百戰程序員：Java中的父類與子類

2020-10-29
| 科技

在java中規定：一個父類可以有多個子類，但是一個子類只能有一個父類。子類可以通過extends關鍵字來繼承父類。做個比較通俗的比喻，就像一個父親可以有多個親孩子，但是一個孩子只能有一個親生父親。

通過繼承，子類可以得到父類除構造函數以外所有的成員（包括成員變量和成員函數），但是要注意得到並不等於可以隨便使用。

子類繼承

假如定義了一個遊戲角色類，擁有血量，角色名，攻擊力等基礎屬性。對不同的角色類型都具有這樣的屬性，都歸屬為角色類，若想在以上基礎屬性上新加一個具有新屬性或者新方法的具體角色類（比如魯班），只需要在定義魯班類的時候讓魯班繼承遊戲角色類。

格式是：

public class Luban extends Player{}

這樣Luban就繼承到Player中定義的屬性和方法，如果子類中沒有定義新函數或者重寫函數。那麼在主函數中創建子類的時候就調用子類對象的方法時用到的就是父類的方法和屬性。

Java中的子類可以訪問父類中的非private修飾的類變量。

package two;

class Father {//同時需要注意class 之前沒有 public 關鍵字

protected static int i = 0;

//private static int i = 0; 父類中的private修飾的變量是不可以被訪問的

}

public class Son extends Father {//Son為這個文件的文件名

public static void main(String []ars){

System.out.println(i);//i have private access in two.Father

}

}
