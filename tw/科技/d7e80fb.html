題目：

利用指針進行兩個數字的交換。

實現代碼：

/* ============================================================================ Name        : TEST-25-20200604.c Author      : 愛折騰小大叔 Version     : Copyright   : Your copyright notice Description : 指針實現數據交換 ============================================================================ */#include <stdio.h>#include <stdlib.h>static void SWAP(int *ptr1,int *ptr2){	int temp;	temp = *ptr1;	*ptr1 = *ptr2;	*ptr2 = temp;}int main(void) {	int A;	int B;	int *ptr1,*ptr2;	printf("please input A and B:\n");	scanf("%d%d",&A,&B);	ptr1 = &A;	ptr2 = &B;	SWAP(ptr1,ptr2);	printf("The result of swap is:%d,%d\n",A,B);}

運行結果：

please input A and B:

1 2
