簡明數據科學 第六部分：模型選擇方法

2020-11-14
| 科技

作者丨Pradeep Menon

原文丨 https://towardsdatascience.com/data-science-simplified-part-6-model-selection-methods-2511cbdf7cb0

譯者丨TalkingData 張永超

在上一篇（本系列其他文章見文末）文章中，我們介紹了多元迴歸模型，費爾南多構建了一個多元線性迴歸模型，接受五個參數來預測汽車的價格。

從最後的結果來看，費爾南多的確構建了一個很好的模型，但是他還想選擇出最佳的輸入參數。那麼這篇文章我們將討論模型的選擇方法。

基本概念

模型選擇方法的思路其實比較直觀，只要回答了下面的問題即可：

如何為最優模型選擇正確的輸入變量？

開始之前，我們需要了解如何定義最佳模型？

最佳模型是適合評估指標最佳值的數據模型。

模型的評估有很好的指標，而調整R平方是多元迴歸模型的最好評價指標。

對於選擇最佳參數集來說，有三種方法，他們是：

最佳子集法 前向逐步選擇法 後向逐步選擇法

接下來分別對這三種方法進行介紹。

如上圖所示，假設我們有k個變量，那麼最佳子集法的過程如下：

  1. 從空模型開始，即沒有預測器的模型，我們稱之為M0；
  2. 用1個變量找出最佳模型，此時，該模型是一個簡單的單一變量回歸模型，只有一個獨立變量，稱之為M1；
  3. 用2個變量找出最佳模型，該模型是一個只有兩個獨立變量的迴歸模型，稱之為M2；
  4. 用2個變量找出最佳模型，該模型是一個有三個獨立變量的迴歸模型，稱之為M3；
  5. 重複這些步驟，只到找出最佳模型的所有預測變量的組合。 對於k個變量，我們需要從下面的模型中找出一組表現最佳的模型：
  • M1： 具有1個預測器的最優模型；
  • M2： 具有2個預測器的最優模型；
  • M3： 具有3個預測器的最優模型；
  • Mk： 具有k個預測器的最優模型。

然後選擇M1…Mk中最優的模型，也就是最適合的模型。 最佳子集法是一個複雜的處理過程，不僅需要梳理了預測變量的列表，而且還需要選擇最佳的變量組合。最佳子集法雖然看似簡單，但是它有著子集的困難和挑戰。

最佳子集法為每個預測變量及其組合創建一個模型，這意味著我們正在為每個變量組合創建模型。模型的數量可以是一個非常大的數字。

如果有2個變量，那麼有4個可能的模型，如果有3個變量，那麼有8個可能的模型。一般來說，如果有p個變量，那麼就有2 ^ p個可能的模型，這是相當多的模型可供選擇。想象一下，有100個變量（很常見）。將會有2 ^ 100個可能的模型。令人難以置信的數字。

在費爾南多的情況下，只有5個變量，他將不得不創造和選擇2 ^ 5個模型，即32個不同的模型。

向前逐步選擇法

至此，對最佳子集法已經有了一個瞭解了，但是它需要很多的計算能力，也相對比較耗時，而前向逐步選擇法試圖環節這種困難。

依然，假設我們有k個變量，則逐步前進選擇法的步驟如下：

  1. 從空模型開始，即沒有預測器的模型，我們稱之為M0。將預測變量添加到模型中，一次僅添加一個；
  2. 用1個變量找出最佳模型，此時，該模型是一個簡單的單一變量回歸模型，只有一個獨立變量，稱之為M1；
  3. 在M1的基礎上，增加一個變量，用2個變量找出最佳模型，稱之為M2；
  4. 在M2的基礎上，增加一個變量，用3個變量找出最佳模型，稱之為M3；
  5. 重複這些步驟，只到找出最佳模型Mk，即有k個變量的模型。

對於k個變量，我們需要從以下一組模型中選擇最優模型：

  • M1： 具有1個預測器的最優模型；
  • M2： 具有2個預測器的最優模型，該模型是M1 + 一個額外的變量；
  • M3： 具有3個預測器的最優模型，該模型是M2 + 一個額外的變量；
  • Mk： 具有k個預測器的最優模型，該模型是Mk-1 + 一個額外的變量。

再次選擇M1 …

Mk中的最佳模型，即具有最佳擬合的模型

與最佳子集法相比，前向逐步選擇法法創建更少的模型。如果有p個變量，那麼將會有大約p（p + 1）/ 2 + 1個模型可供選擇，遠低於最佳子集方法中的模型選擇。假設有100個變量，則基於前向逐步法創建的模型的數量是100×101/2 + 1，即5051個模型。

在費爾南多的情況下，只有5個變量，他將不得不創建和選擇5 6/2 + 1*模型，即16個不同的模型。

後向逐步選擇法

現在我們已經理解了模型選擇的前向逐步選擇過程，接下來看看後向逐步選擇法，這是前向逐步選擇法的反向。前向逐步選擇從沒有變量的模型即空模型開始，相比之下，後向逐步選擇是從所有變量開始的。後向逐步選擇法的過程如下：

假設我們有k個變量，則逐步前進選擇法的步驟如下：

  1. 從一個完整的模型開始，即包含所有預測變量的模型，我們稱這個模型為Mk。從完整模型中刪除預測變量。一次一個；
  2. 用k-1個變量找出最優模型，從Mk中刪除一個變量。計算所有可能組合模型的性能。選擇具有k-1個變量的最佳模型。我們稱這個模型為Mk-1；
  3. 用k-2個變量找到最優模型，從Mk-1中刪除一個變量。計算所有可能組合模型的性能。選擇具有k-2個變量的最佳模型。我們稱這個模型為Mk-2；
  4. 重複這些步驟，只到找出最佳模型M1，即有1個變量的模型。

對於k個變量，我們需要從以下一組模型中選擇最優模型：

  • Mk：具有k個預測值的最優模型；
  • Mk-1：具有k-1個預測變量的最優模型。這個模型是 MK - 一個額外的變量；
  • Mk-2：具有k-2個預測變量的最優模型。這個模型是 MK - 兩個額外的變量；
  • M1：具有1個預測器的最優模型。

模型構建

瞭解了模型選擇的方法之後，讓我們回到費爾南多的問題，回想一下前一篇文章，費爾南多有6個變量發動機大小，馬力，峰值RPM，長度，寬度和高度，他想通過創建一個多元迴歸模型來估算汽車價格。他想保持模型的平衡並選擇最佳的模式。

於是，費爾南多準備使用前向逐步選擇法，統計軟件包計算出了所有可能的模型以及輸出M1到M6的結果。

讓我們解釋上述結果：

  • Model 1：只有一個預測器，僅使用發動機的大小作為變量，調整的R平方為0.77；
  • Model 2：僅有兩個預測器，使用發動機大小和馬力作為變量，調整的R平方為0.79；
  • Model 3：僅有三個預測器，使用發動機大小、馬力和寬度作為變量，調整的R平方為0.82；
  • Model 4：僅有四個預測器，使用發動機大小、馬力、寬度和高度作為變量，調整的R平方為0.82；
  • Model 5：僅有五個預測器，使用發動機大小、馬力、寬度、高度和峰值RPM作為變量，調整的R平方為0.82；
  • Model 6：僅有六個預測器，使用發動機大小、馬力、寬度、高度、高度和峰值RPM作為變量，調整的R平方為0.82。

最終，費爾南多選擇了最好的、最簡單的模型。在上述結果中，Model 3 使用發動機大小、馬力和寬度作為變量，該模型能夠得到0.82的調整R平方，即說明該模型可以解釋82%的訓練數據變化，其他Model 4、5、6雖然使用了更多的變量，但是得到的調整R平方依然是0.82，相對於Model 3卻更加的複雜。

在費爾南多的統計軟件包中，同樣輸出瞭如下的係數結果：

根據發動機尺寸，馬力和寬度估算價格。

價格 = -55089.98 + 87.34 發動機大小 + 60.93 馬力+ 770.42 * 寬度

模型評估

費爾南多選擇了最好的模型，該模型將使用發動機尺寸，馬力和汽車寬度估算價格。他希望評估模型在訓練和測試數據上的表現。

之前，他已經將數據分成了訓練集和測試集，費爾南多使用訓練數據訓練模型，對於模型來說此時測試數據是不可見的數據。費爾南多評估模型在測試數據上的表現。

在訓練數據上，模型表現相當好,調整的R平方為0.815，說明該模型可以解釋81％的訓練數據變化。但是，對於該模型來說，在測試數據集上的表現更為重要。

費爾南多測試了模型在測試數據集上的表現性能，得到該模型在測試數據中的調整的R平方為0.7984。這意味著即使對於看不見的數據，模型也可以解釋79.84％的變化。

結語

現在，費爾南多有了一個簡單而且有效的模型來預測汽車的價格。但是，發動機大小，馬力和寬度的單位是不同的。於是他想到了如下的幾個問題：

如何使用常見的比較單位來估算價格變化？
