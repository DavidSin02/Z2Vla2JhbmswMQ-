一、設備的安裝

1、傳輸穩定性

傳輸穩定性選型，主要看周圍環境的無線干擾因素。一般來講，2.4G 的無線使用範圍十分廣泛，因 此 2.4G 頻段的干擾也相對比較大，若周邊已有較多的 2.4G 無線，可能會對無線網橋的傳輸穩定性帶來影 響。通常選擇原則如下：

（1）在居民區、商業區等密集且明顯有較多的無線使用環境，實施無線網橋工程，一般建議選擇 5G 頻段 的無線 AP 產品，可較好的減少干擾，提高穩定性。

（2）在偏僻的公園、農村、農場等區域實施無線網橋，則可以選擇成本更好的 2.4G 無線 AP 產品。


2、設備安裝高度

產品安裝高度的基本原則就是在產品覆蓋角度內兩點之間無障礙物阻擋，若兩點之間明顯有障礙物阻 擋，需要提高安裝高度達到穩定傳輸的目的。如下圖，AP 安裝高度需要滿足 H≥R+D。

二、設備的配置與調試

我們就以tp—link為例，我們將點對點兩端設備分別稱為主設備與副設備，對主設備與副設備我們的配置如下 ：


1、主設備的配置

（1）設置為 AP 模式。登錄 WEB 界面（默認 https://192.168.1.254），工作模式選擇 Access Point 模式，點擊“保存配置”。

（2）設置無線名稱。點擊“無線設置”——“AP 設置”——“SSID”設置 SSID 來命名的無線網絡（區分 大小寫）—— “加密算法” 選擇“WPA-PSK/WPA2-PSK”——“PSK 密鑰”至少 8 位數字、字符或 者 數字與字符的組合，區分大小寫 ——“確定”。

（3）配置信道與帶寬。信道帶寬設置：“無線設置”——“基本設置”——“信道帶寬 ”建議設置 20MHZ。信道設置建議通過 CPE 頻譜分析，來確定干擾最小的信道。頻譜分析功能在“無線設 置”——“基本設置”——“頻譜導航”如下圖:

系統將彈出如下圖所示界面，點擊 < 是 > 按鈕進入頻譜分析頁面:

如下圖所示，點擊右下角的 < 開始 > 按鈕開始分析，一段時間後，點擊 < 停止> 按鈕查看穩定 的圖像。

在選擇信道時應該儘量避開干擾較大的頻段。上圖中 5725~5755MHz 存在著較強的干擾，建議選 擇 149/5745MHz 信道外 的其他信道。關閉頻譜分析窗口即可返回無線設置頁面，根據上述結果，“信 道頻率”一欄建議選擇 149/5745MHz 信道外的其他信道。設置完之後點擊“確定”


（4）距離設置。“無線設置”——“無線高級設置”——“距離設置” 輸入 AP 與 Client 之間的距離——“確 定”。建議使用地圖軟件測距，如果實在在難以測量，建議輸入一個大於該距離的最小整數，但該項一定 不能設置為 0 或者過大。


（5）QoS 設置。使用 5G 產品建議開啟 QoS，使用 2.4G 產品建議禁用 QoS，操作步驟：打開 AP 管 理界面——“無線設置”——“無線高級設置”——“QoS ”（勾選則為啟用，反之禁用）——“確定” 。


（6）開啟 MAXtream 功能。MAXtream 配置：在點對點的使用場景模式下，不建議開啟 Maxtream 功能。

（7）保存配置。檢查是否點擊保存配置。若 WEB 頁面的上方還有提示，點擊保存配置。（一定要確認 配置保存工作做好）


2、副設備的配置

（1）Client 模式配置。登錄 WEB 界面（默認 https://192.168.1.254），，工作模式選擇 Client 模式，點 擊保存配置 。

（2）修改管理地址。“網絡參數”——“LAN 設置” 修改 LAN 地址為 192.168.1.2 （不要與其他設備衝突即 可）——“確定”

（3）WDS 設置。“無線設置”——“Client 設置”——“遠程 AP 的 SSID”點擊 < 掃描 > 按鈕，選擇 Access Point 的 SSID，點擊 < 連接 > 按鈕——“加密算法” 選擇“WPA-PSK/WPA2-PSK”——“PSK 密鑰” 輸入 Access Point 的 PSK 密鑰——“確定”。

（4）距離設置。（與 AP 端一致）：“無線設置”——“無線高級設置”——“距離設置” 輸入 AP 與 Client 之 間的距離——“確定”。建議使用地圖軟件測距，如果實在在難以測量，建議輸入一個大於該距離的最小 整數，但該項一定不能設置為 0 或者過大。


（5）QoS 設置。（與 AP 端一致）使用 5G 產品建議開啟 QoS，使用 2.4G 產品建議禁用 QoS，操作步 驟：打開 CLient 管理界面——“無線設置”——“無線高級設置”——“QoS ”（勾選則為啟用，反之禁用） ——“確定”

（6）保存配置。檢查是否點擊保存配置。若 WEB 頁面的上方還有提示，點擊保存配置。（一定要確認 配置保存工作做好）


三、網絡優化

1、判斷鏈路質量

在從設備（Client 模式的無線 AP）的管理界面中查看“系統狀態”——“無線信號質量”：

1、信號強度：數值大小越大越好（即數字越小越好），建議高於-80dBm（即數字小於 80）。

2、噪聲強度：數值越小越好（即數字越大越好）。

3、信噪比：越大越好

4、 CCQ 值 ：數值越大越好，一般高於 70 質量較好。


2、調整鏈路

若上述參數不是很理想，可以嘗試做如下調整：

（1） 檢查安裝角度是否對準中心安裝，安裝高度，是否中間依然有阻擋。（天線對準方法見常見問 題 3.1）

（2）檢查配置項，比如距離配置，QoS 配置，MAXtream 配置，是否有按照上文的提示正確配置。

（3）檢查無線環境，是否無線 AP 附近的干擾較大？可以更換 AP 信道嘗試，另外檢查信道帶寬是否工作在 20MHZ？
