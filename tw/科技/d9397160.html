65個excel操作教程大全，新手很容易上手，建議收藏備用

Excel表格在工作中常常要用到，如果你是Excel表格新手，不知道怎麼製作表格的話，可以看看本文提供的Excel表格操作技巧彙總詳細教程。

一、文件操作

1、為excel文件添加打開密碼

方法：文件 → 信息 → 保護工作簿 → 用密碼進行加密

2、為文件添加作者信息

方法：文件 → 信息 → 相關人員 → 在作者欄輸入相關信息

3、讓多人通過局域網共用excel文件

方法：審閱 → 共享工作簿 → 在打開的窗口上選中“允許多用戶同時編輯...”

4、同時打開多個excel文件

方法：按ctrl或shift鍵選取多個要打開的excel文件，右鍵菜單中點“打開”

5、同時關閉所有打開的excel文件

方法：按shift鍵同時點右上角關閉按鈕

6、設置文件自動保存時間

方法：文件 → 選項 → 保存 → 設置保存間隔

7、恢復未保護的excel文件

方法：文件 → 最近所用文件 → 點擊“恢復未保存的excel文件”

8、在excel文件中創建日曆

方法：文件 → 新建 → 日曆

9、設置新建excel文件的默認字體和字號

方法：文件 → 選項 → 常規 → 新建工作簿時：設置字號和字體

10、把A.xlsx文件圖標顯示為圖片形式

方法：把A.xlsx 修改為 A.Jpg

11、一鍵新建excel文件

方法：Ctrl + N

12、把工作表另存為excel文件

方法：在工作表標籤上右鍵 → 移動或複製 → 移動到”新工作簿”

二、窗口設置

1、同時打開對比兩個或多個excel文件

方法：視圖 → 全部重排 → 選排列的方向

2、同時查找一個工作簿中的兩個工作表

方法：視圖 → 新建窗口 → 全部重排 → 選排列方向

3、隱藏或顯示excel最上面的功能區

方法：Ctrl+F1

4、隱藏excel工作表界面

方法：選取要隱藏的工作表 → 視圖 → 隱藏

5、讓標題行始終顯示在最上面

方法：視圖 → 凍結窗格 → 凍結首行

6、把窗口拆分成上下兩部分，都可以上下翻看

方法：選取要拆分的位置 → 視圖 → 拆分

7、調整excel工作表顯示比例

方法：按ctrl+滾輪

8、快速切換到另一個excel窗口

方法：ctrl + tab鍵

9、全屏顯示excel界面

方法：視圖 → 全屏顯示

三、單元格顯示

1、隱藏0值

方法：文件 → 選項 → 高級 → 去掉“在具有零值.....”勾選

2、隱藏單元格內所有值

方法：ctrl+1打開單元格設置窗口 → 數字 → 自定義 → 右邊文框中輸入三個分號;;;

3、隱藏編輯欄、灰色表格線、列標和行號

方法：視圖 → 顯示 → 去掉 各項的勾選

四、單元格選取

1 、選取當前表格

方法：按ctrl+a全選當前數據區域

2 、選取表格中的值/公式

方法：按ctrl+g打開定位窗口 → 定位條件 → 值/公式

3 、查找值

方法：按ctrl+f打開查找窗口，輸入查找的值 → 查找

4、選取最下/最右邊的非空單元格

方法：按ctrl + 向下/向右箭頭

5、快速選取指定大小的區域

方法：在左上的名稱欄中輸入單元格地址，如a1:a10000，然後按回車

五、單元格編輯

1、設置單元格按回車鍵光標跳轉方向

方法：文件 → 選項→ 高級 → 最上面“按Enter鍵後移動所選內容” → 設置方向

2、設置下拉菜單，顯示A,B,C三個選項

方法：數據 → 數據有效性 → 允許 → 序列 → 輸入A,B,C

3、複製表格保存行高列寬不變

方法：整行選取複製 → 粘貼後點粘貼選項中的“保留列寬”

4、輸入到F列時，自動轉到下一行的首列

方法：選取A:F列，輸入後按回車即可自動跳轉

5、設置三欄表頭

方法：插入 → 形狀 → 直線 → 拖入文本框中輸入的字體並把邊框設置為無

6、同時編輯多個工作表

方法：按ctrl或shift鍵選取多個工作表，直接編輯當前工作表即可。

7、輸入身份證號或以0開始的數字

方法：把單元格格式設置成文本，然後再輸入

8、快速刪除空行

方法：選取表中某列 → ctrl+g定位 → 定位條件 → 空值 → 刪除整行

9、快速插入空行

方法：在表右側輸入序號1,2,3..N,然後再複製該序號到下面空行，最後按序號列排序即可。

10、快速合併多行數據

方法：插入批註 → 選取多列數據複製 → 粘貼到批註中，然後再從批註中複製粘至單元格區域中即可。

11、插入特殊符號

方法：插入→ 符號

12、查找重複值

方法：選取數據列 → 開始 → 條件格式 → 突出顯示單元格規則 → 重複值

13、刪除重複值

方法：選取區域 → 數據 → 刪除重複項

14、單元格分區域需要密碼才能編輯

方法：審閱 → 允許用戶編輯區域

15、用excel進行漢英互譯

方法：審閱 → 翻釋

16、不復制隱藏的行

方法：選取區域 → ctrl+g定位 → 定位條件 → 可見單元格 → 複製 → 粘貼

17、單元格強制換行

方法：在需要換行字符後按alt+enter鍵

六、單元格格式設置

1、萬元顯示

方法：自定義格式代碼：0!.0,萬

2、日期按年年年年-月月-日日顯示

方法：自定義格式代碼：yyyy-mm-dd

3、手機號分段顯示

方法：自定義格式代碼：000 0000 0000

4、數字1顯示成1月

方法：自定義格式代碼: 0”月”

七、打印設置

1、每頁都有標題行

有時候表格數據較多，正常打印的時候除了第一頁有標題外，其他頁都沒有，那怎麼能快速給每頁都加上標題呢？

方法：點擊頁面佈局 → 打印標題 → 頂端標題行 → 設置好需要打印的標題區域

2、打印頁眉頁腳

很多時候小夥伴們打印的表格頁數太多，容易搞亂，這時我們就需要添加頁碼來提高辨識度。

方法：點擊頁面佈局 → 頁面設置 → 頁眉/頁腳 → 自定義頁腳（或頁眉）→ 添加頁碼

3、打印表格數據合併到一頁

很多小夥伴在調整表格頁數的時候都會遇到一個問題：想把所有數據打印在一頁紙上，可偏偏會多出一兩行數據，導致需要兩頁紙去打印。

方法：點擊頁面佈局 → 頁面設置 → 頁面 → 縮放 → 調整為1頁寬1頁高

4、設置居中打印

打印表格時一般默認是居左，怎麼才能居中打印呢？

方法：點擊頁面佈局 → 頁面設置 → 頁邊距 → 居中方式 → 勾選水平、垂直

5、指定區域打印

我們在打印很多數據時，有時不想打印全部數據，只需要打印一部分，我們可以設置指定的打印區域。

方法：點擊頁面佈局 → 頁面設置 → 工作表 → 打印區域 → 設置好需要打印的區域

6、打印不顯示錯誤值

工作表中有時候難免會出現錯誤值，如何才能不顯示錯誤值，以便於打印出來後能靈活填寫呢？

方法：點擊頁面佈局 → 頁面設置 → 工作表 → 錯誤單元格打印為 → 選擇<空白>

7、單色打印

有時候為了突出重點數據，有些小夥伴會在單元格中填充底色，但是公司通常用的都是黑白打印機，打印出來的效果真是不咋地。那如何才能去除底色打印呢？

方法：點擊頁面佈局 → 頁面設置 → 工作表 → 打印 → 勾選單色打印

8、打印單元格批註

單元格中插入批註後，如何才能顯示批註打印呢？

方法：點擊頁面佈局 → 頁面設置 → 工作表 → 打印 → 批註 → 選擇“工作表末尾”

9、打印行號列標

方法：點擊頁面佈局 → 頁面設置 → 工作表 → 打印 → 勾選行號列標

10、設置橫向打印

當工作表裡列數據較多，而我們選擇的又是縱向打印時，就會發現列數據無法在一頁上顯示，那怎麼才能解決呢？這裡我們只需設置為橫向打印即可。

方法：點擊頁面佈局 → 頁面設置 → 頁面 → 方向 → 選擇橫向

11、按釐米設置行高

方法：視圖 → 頁面佈局，在頁面佈局下行高單位是釐米

12、插入分頁符

方法：選取要插入的位置 → 頁面佈局 → 分頁符 → 插入分頁符

八、圖片處理

1、刪除工作表中所有圖片

方法：ctrl+g定位 → 定位條件 → 對象 → 刪除

2、工作表插入背景圖片

方法：頁面佈局 → 背景 → 選擇插入圖片

3、插入可以打印的背景

方法：插入 → 頁眉頁腳 → 選取頁眉後點圖片 → 選取圖片並拖入
