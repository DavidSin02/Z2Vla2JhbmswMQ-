Pandas數據幀的6種創建方法

2020-10-29
| 科技

Pandas是Python的數據分析利器，DataFrame是Pandas進行數據分析的基本結構，可以把DataFrame視為一個二維數據表，每一行都表示一個數據記錄。本文將介紹創建Pandas DataFrame的6種方法。

創建Pandas數據幀的六種方法如下：

  • 創建空DataFrame
  • 手工創建DataFrame
  • 使用List創建DataFrame
  • 使用Dict創建DataFrme
  • 使用Excel文件創建DataFrame
  • 使用CSV文件創建DataFrame

1、創建空的Pandas DataFrame

學編程，上匯智網，在線編程環境，一對一助教指導。

首先我們看一下如何創建一個空的DataFrame（數據幀）：

pd.DataFrame(columns=['A', 'B', 'C'], index=[0,1,2])

columns參數用來定義列名，index參數用來定義行號。上面的代碼創建了一個3行3列的二維數據表，結果看起來是這樣：

嗯，所有數據項都是NaN。

2、手工創建Pandas DataFrame

接下來讓我們看看如何使用pd.DataFrame手工創建一個Pandas數據幀：

df = pd.DataFrame(data=['Apple','Banana','Cherry','Dates','Eggfruit'])

使用data參數來聲明數據，結果看起來是這樣：

由於我們沒有定義數據幀的列名，因此Pandas默認使用序號作為列名。和原來一樣，可以使用columns參數來聲明列名：

df = pd.DataFrame(data=['Apple','Banana','Cherry','Dates', 'Eggfruit'],  columns=['Fruits'])

結果如下：

最左側的列被稱為索引，默認從0開始，和原來一樣我們用index自行定義：

df = pd.DataFrame(data=['Apple','Banana','Cherry','Dates','Eggfruit'],  index = [1,2,3,4,5], columns=['Fruits'])

現在的數據幀看起來是這樣：

那麼如果要添加一個Quantity列來表示水果數量該怎麼做？

pd.DataFrame(data=[['Apple',5], ['Banana',10], ['Cherry',8], ['Dates',3], ['Eggfruit',7]], columns = ['Fruits','Quantity'], index=[1,2,3,4,5])

注意data參數和colums參數都進行了相應的調整。現在的DataFrame這樣：

3、使用列表創建Pandas DataFrame

學編程，上匯智網，在線編程環境，一對一助教指導。

假設我們有一個列表：

fruits_list = ['Apple','Banana','Cherry','Dates','Eggfruit']

要把列表轉換為DataFrame，直接將列表傳入pd.DataFrame即可：

pd.DataFrame(fruits_list)

得到的數據幀結構如下：

當然你可以用columns參數來定義列名，或者用index來聲明行號：

pd.DataFrame(fruits_list,  columns = ['Fruits'],  index = ['a','b','c','d','e'])

4、使用字典創建Pandas DataFrame

字典就是一組鍵/值對：

dict = {key1 : value1, key2 : value2, key3 : value3}

當我們將上述字典對象轉換為DataFrame時，看起來是這樣：

容易注意到，字段的鍵對應成為DataFrame的列，而所有的值對應數據。 記住這個對應關係。

現在假設我們要創建一個如下形狀的DataFrame：

由於列名為Fruits、Quantity和Color，因此對應的字典也應當 有這幾個鍵，而每一行的值則對應字典中的鍵值，字典應該是 如下的結構：

fruits_dict = { 'Fruits':['Apple','Banana','Cherry','Dates','Eggfruit'], 'Quantity': [5, 10, 8, 3, 7],  'Color': ['Red', 'Yellow', 'Red', 'Brown', 'Yellow']}

現在讓我們將這個字典轉換為DataFrame，一句話的事兒：

pd.DataFrame(fruits_dict)

結果如下：

5、將Excel文件轉換為Pandas DataFrame

如果你有一個excel文件，例如“fruits.xlsx“…

那麼可以使用下面的代碼將其轉換為Pandas DataFrame：

fruits = pd.read_excel('fruits.xlsx')

得到的數據幀看起來是這樣：

6、將CSV文件轉換為Pandas DataFrame

假設你有一個CSV文件，例如“fruits.csv“，可以使用如下的代碼 將其轉換為DataFrame：

fruits = pd.read_csv('fruits.csv')

也是一句話的事兒。
