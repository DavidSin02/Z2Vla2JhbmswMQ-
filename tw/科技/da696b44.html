Ansys中接觸單元並不是goodman 單元，類似於goodman單元 ansys裡面的接觸單元是是通用的，而goodman

是一種專業的單元。goodman單元假定兩片長為L的接觸面以無數微小的切向和法向彈簧所連接，接觸面單元與

13.面載荷轉化為等效節點力施加的方法

在進行分析時，有時候需要將已知的面載荷按照節點力來施加，比如載荷方向及大小不變的情況（ANSYS將面力

解釋為追隨力，而將節點力解釋為恆定力），那麼，在只知道面力的情況下，如何施加等效於該面力的等效節點

力呢？可以通過如下步驟給有限元模型施加與已知面載荷完全等效的節點力：

(1)在模型上施加與已知面力位置、大小相同但方向相反的面力。

Main Menu->Solution->Apply->Pressure->。（注意：所施加面力要與已知力反號）。

(2) 將模型的所有節點自由度全部約束。

Main Menu->Solution->Apply->Displacement->On Nodes

(3)求解模型。

Main Menu->Solution->Current LS（這一步會生成結果文件Jobname.rst）

(4)開始新的分析：

Main Menu->Solution->New Analysis

(5)刪除前兩步施加的面力和約束。

Main Menu->Solution->Delete->Pressure->

Main Menu->Solution->Delete-> Displacement->On Nodes

(6)從Jobname.rst中保存的支反力結果施加與已知面力完全等效的節點力。

Main Menu->Solution->Apply->Force/Moment->From Reaction

(7)施加其它必要的載荷和約束，然後求解。

16.如何得到徑向和周向的計算結果?

在圓周對稱結構中，如圓環結構承受圓周均布壓力。要得到周向及徑向位移，可在後處理/POST1中，通過菜單General Postproc>Options for Outp>Rsys>Global cylindric 或命令Rsys,1 將結果座標系轉為極座標，則X方向位移即為徑向位移，Y向位移即為周向位移。

26.隧道開挖模擬方法

小弟正在做一個大跨度隧道的施工模擬。現在主要採取兩種方法：

一、直接施加重力場進行計算。第一步，計算初始位移場。後續開挖計算則減去第一步的位移場，得到各階段的位移。

二、採用初始應力輸入的方法。首先計算初始應力場，寫出初始應力文件。後續開挖時，讀入初始應力，施加重力、相同的邊界和等效釋放荷載。直接計算各施工階段的應力場和位移場。

28.求塑性極限荷載時，結構的變形應該較大，建議把大變形打開。

因為小變形計算時不考慮結構的變形，因此計算是線性的，

而大變形打開時每個荷載步後都會對模型進行修正，其計算是非線性的因而計算結果更精確，如果結構變形比較小時兩種結算結果基本相同，但是大變形會消耗更多的資源.

29.如果在接觸分析中模型之間存在間隙，為了防止剛體位移，需要調整初始接觸條件

一般有三種方法：

1 通過自動計算contact surface offset來關閉小的間隙(keyopt(5)=1)

2 通過實常數ICONT指定目標面周圍環境的調整範圍

3 通過實常數PMIN和PMAX來調整目標面位置來調整初始穿透，如果目標面被約束，則PMIN和PMAX無效。

有間隙存在還能做接觸分析嗎?

1）. 建模時使接觸體處於剛好接觸的位置；

2）. 通過給定位移來將接觸體移到接觸位置；

3）. 通過軟彈簧把兩個分開的物體連起來，採用動態方法求解。

30. 【討論】結構計算完成後怎樣畫出某結點或單元的應力～應變關係曲線？

標準方法：

1， 定義變量：

拾取主菜單：Main Menu>Time Hist postproc>Define Variables>在隨之彈出的對話框中點擊Add鍵，定義第一個變量序號為2，選取第一個變量stress,確定與之對應的下一級選項（如Y-direction SY等）；返回定義變量對話框，再點擊add鍵，定義第二個變量序號為3，選取第二個變量strain-elastic及以及對贏得下一級選項（如Y-dir'n EPEL Y等，在應力-應變圖中，其向量的取向應相同）。同理再定義變量4，選取變量strain-plastic及與之對應的下一級選項如Y-dir'n EPEL Y等），在應力-應變圖中，應變是彈性應變和塑性應變累加的總應變。為使其實現相加，還需進行以下操作：拾取主菜單：Main Menu>Time Hist postproc>math operation>add,定義計算變量序號為5，同時在相應交互框內輸入3和4。點擊確認鍵，則由變量3，4代表的應變之和就存在變量5中。

2，繪製應力-應變曲線：

拾取主菜單：Main Menu>Time Hist postproc>setting>graph.設置x軸向變量為單變量，並將其變量序號定義為5。點擊確定鍵退出退化框。拾取應用菜單：Utility Menu>plot ctrls>styles>Graphs>Modify axis.將x,y座標軸分別命名為Y-strains,Y-stress,拾取主菜單：Main Menu>Time Hist postproc>graph variables. 在對話框上"the first variable"對應的交互框中輸入2。點擊確定鍵，則預想的應力-應變曲線就顯示在屏幕上。

34.ANSYS結果等值線輸出

對體和麵來說，ANSYS默認的結果輸出格式是雲圖格式，而這種彩色雲圖打印為黑白圖像時對比很不明顯，無法表達清楚，這對於發表文章來說是非常不便的。發文章所用的結果圖最好是等值線圖，並且最好是黑白的等值線圖。筆者原來進行這項工作時一般借用photoshop等第三方軟件，很麻煩，並且效果不好。現通過摸索，發現通過靈活運用ansys本身也能實現這項功能。現將步驟寫給大家，感謝simwe對我的幫助。

（1）將要輸出的結果調出，這時為彩色雲圖；

（2）將雲圖轉換為等值線圖的形式

GUI：plotCtrls->Device Options->[/DEVI]中的vector mode 選為on

命令：/DEVICE,VECTOR,1

這時結果為彩色等值線，若直接輸出，打印為黑白圖像時仍然不清晰，為此需進行以下幾步將圖像轉換為黑白形式；

（3） 將背景變為白色

命令：jpgprf,500,100,1

/rep

（4）對等值線中的等值線符號（圖中為A,B,C等）的疏密進行調整

GUI：plotCtrls->Style->Contours-> Contours Labeling 在Key Vector mode contour label 中選中on every Nth elem，然後在N= 輸入框中輸入合適的數值，例如5，多試幾次，直到疏密合適

命令：/clabel,1,5

（5）將彩色等值線變為黑色

GUI：plotCtrls->Style->Colors->Contours Colors 將Items Numbered 1，Items Numbered 2等複選框中的顏色均選為黑色，圖像即可變為黑白等值線圖像

命令：/color,cntr,whit,1 等等

（6）最後一步：出圖

GUI：plotCtrls->Capture Image

希望對大家能有所幫助。
