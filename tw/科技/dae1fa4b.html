新型鑄鐵檢查井與路面同步施工技術

2020-11-14
| 科技

摘要

為保證車輛和行人在公路和市政道路新改建、養護施工時可以正常通行，根據道路設計和施工作法，將檢查井產品的底層支座進一步分解為下支座和底支座，既能施工便捷又能正常通行車輛和行人。

關鍵詞: 道路; 檢查井; 多層支座; 分層; 同步施工

傳統和現行鑄鐵檢查井即單層支座檢查井、前期創新的雙層及多層支座鑄鐵檢查井的底層支座用水泥砂漿或水泥混凝土固定後，因為需要水泥砂漿或水泥混凝土養生而不能馬上鋪築同層瀝青混凝土，底層支座因上表面達到和將要鋪築的底層瀝青混凝土上表面平齊而高出基層 60mm 以上，嚴重影響道路上車輛和行人的正常安全通行，行車也會碾壓導致底層支座位移。現結合傳統和現行單層支座鑄鐵檢查井、雙層及多層支座鑄鐵檢查井作法進一步創新，將底層支座一分為二為下層支座和底座，從而適應道路施工和養護時車輛、行人的安全通行。

1 鑄鐵檢查井安裝傳統作法

1. 1 傳統和現行鑄鐵檢查井作法和缺陷

道路瀝青混凝土路面結構層都是按照設計作法分層施工，而傳統和現行的鑄鐵檢查井卻只有一層，無法與道路結構層分層同步施工，如雙層瀝青混凝土路面結構與傳統和現行的鑄鐵檢查井的施工工藝是: 將鑄鐵檢查井頂面與下面層瀝青混凝土路面調整水平後用水泥砂漿或水泥混凝土臨時固定鑄鐵檢查井→固定層達到可施工強度後，鋪築並壓實下面層瀝青混凝土→ 挖出鑄鐵檢查井→砌築井筒漲高井，調平固定鑄鐵檢查井，養生→鋪築並壓實表面層瀝青混凝土。

傳統和現行的單層支座鑄鐵檢查井作法有以下缺陷:

( 1) 鑄鐵檢查井與路面瀝青混凝土層分層施工不能保持同步;

( 2) 有技術間隙時間長，固定支座的砂漿或現澆水泥混凝土需等 3 ～ 5 天達到一定施工強度後才能進行面層瀝青混凝土鋪築;

( 3) 支座每次固定後鋪築瀝青混凝土前都高出一個即將鋪築的瀝青混凝土厚度，影響道路車輛和行人安全通行，支座高出後也容易被撞和碾壓產生高差和水平位移;

( 4) 反挖剛鋪築完的新料後又採用新材料回填修補，既浪費了材料又產生渣土垃圾，且只能小型機具配

合人工施工，有接茬整體性差還不美觀;

( 5) 支座用不易結合成整體的一層表面層瀝青混凝土和一層砂漿或水泥混凝土固定鑄鐵檢查井，在車輛荷載和熱脹冷縮作用下易發生開裂脫落病害。

1. 2 前期創新的鑄鐵檢查井的作法和缺陷

前期創新的新型道路鑄鐵檢查井具有了較大的創新和很大程度上方便了施工，但底層瀝青混凝土鋪筑前固定的底層支座高出基層不低於 60mm，嚴重影響車輛和行人在道路上安全通行，如圖 1 所示。

2 發明和改造

為完善和改進傳統和現行單層支座鑄鐵檢查井、前期創新的雙層和多層支座鑄鐵檢查井的缺陷和調整 鑄鐵檢查井的作法，特進一步優化新型鑄鐵檢查井的 產品作法和施工工藝。

2. 1作法

新型鑄鐵檢查井由上支座、若干中支座、下支座、和底座組成，各自的作法和組裝作法詳見圖 2 ～ 圖 6。

依據瀝青混凝土路面結構分層層數和每層厚度來增加或減少底座上中支座和上支座的數量來滿足道路的相應需求:

底座和井蓋組合為傳統和現行的鑄鐵檢查井;

上支座、底座和井蓋組合為單層支座鑄鐵檢查井， 適用於單層瀝青混凝土面層道路的檢查井;

上支座、1 層中底座和井蓋組合為雙層支座鑄鐵檢查井，適用於雙層瀝青混凝土面層道路的檢查井;

上支座、2 層中底座和井蓋組合為雙層支座鑄鐵檢查井，適用於三層瀝青混凝土面層道路的檢查井;

上支座、多層中底座和井蓋組合為雙層支座鑄鐵檢查井，適用於多層瀝青混凝土面層道路的檢查井。

三層瀝青混凝土路面的道路鑄鐵檢查井由上支座、中支座、下支座和底座組成，組裝後的平面圖和剖面見圖 6 所示。

井蓋和支座之間、支座與支座之間可以通過卡釘、卡槽相連，也可以採用絲扣相連，還可以採用銷釘、防盜螺栓等其它方式相連或混合連接方式相連，如圖 7所示。


上層支座伸入下層支座的部分稱為連接板或內伸 板，在滿足連接固定的基本功能後為減少材料使用節約 成本可將連接板做成有孔洞的片狀或柱狀作法，下端既 可以延伸到下層支座支撐臺上，又可以懸空，如圖8 所示。

2. 2與路面結構層同步施工工藝

新型鑄鐵檢查井與瀝青混凝土路面層分層同步施工的施工工藝如下所示:

( 1) 鋪築完基層或銑刨完底層瀝青混凝土後，在基層預製鋼筋混凝土模塊預留槽內用預埋筋和焊接件固定底座，使底座的上表面與基層頂面平齊，澆築固定材料，蓋上井蓋和鋼板養生，底座頂面與基層頂面平齊不影響道路基層上通行車輛和行人;

( 2) 養生期滿，取走鋼板，有封層時施工封層，然後取出井蓋，將下支座和井蓋安裝到底座的支撐臺上， 鋪設底層瀝青混凝土並碾壓密實;

( 3) 待底層瀝青混凝土溫度低於 50℃ 後，取出井蓋，連接固定中支座後蓋上井蓋，接著進行道路的上一層瀝青混凝土的鋪設和碾壓密實;

( 4) 根據需要重複步驟( 3 ) 直到鋪築完中層瀝青混凝土;

( 5) 待中層瀝青混凝土溫度低於 50℃ 後，取出井蓋，連接固定上層支座後安裝井蓋並固定，進行道路的表面層瀝青混凝土的鋪設和碾壓密實。

新型鑄鐵檢查井產品和施工工藝的核心是: 根據設計的道路瀝青混凝土面層層數和層厚選擇相應高度的各層支座，在滿足上支座的上表面與表面層瀝青混凝土上表面平齊後其餘支座的上表面不高於同層瀝青混凝土上頂面，以保證同步施工的實現。如 4cm 表面層瀝青混凝土 + 6cm 中層瀝青混凝土 + 8cm 底層瀝青混凝土 + 1cm 封層路面結構，如採用井蓋外沿厚 4cm 的常用井蓋則分層可為: 5. 2cm + 5. 8cm + 8cm + 底層支座，如採用井蓋外沿厚 3cm 的井蓋則分層可為:

4. 2cm + 6. 8cm + 8cm + 底層支座高度; 5cm 表面層瀝青混凝土 + 6cm 中層瀝青混凝土 + 8cm 底層瀝青混凝土 + 1cm 封層路面結構，採用井蓋外沿厚 4cm 的常用井蓋則分層可為: 5. 2cm + 6. 8cm + 8cm + 底層支座。

以三層瀝青混凝土的道路為例說明三層支座檢查井的施工工藝，詳見圖 9。

新型多層支座鑄鐵檢查井的施工工藝過程有以下積極和有益的效果:

( 1) 內優外美: 攤鋪機、壓路機一次性施工完成， 沒有二次作業和修補接縫，提高了道路壓實度、整體性 和美觀性;

( 2) 提前工期: 無漲井等待技術間歇，鑄鐵檢查井施工與路面結構層的分層同步施工，多層瀝青混凝土路面可緊跟連續鋪築;

( 3) 節材環保: 井周不再反挖和修補，既節省材料又不會產生建築垃圾;

( 4) 便捷施工: 在下次道路大修時銑鉋完路面後沒有漲井技術間歇，只要取下各層支座再分層安裝回去即可鋪築各層瀝青混凝土，特別適合快速開放交通的城鎮道路需求;

( 5) 安全平穩通行: 每層施工後均是平面，不影響交通並通行平穩安全。

2. 3原材料

新型檢查井不侷限於灰口鑄鐵、球墨鑄鐵生產加工，也可擴展到複合新型材料等原材料製作。

3 新型檢查井的創新點和優點

與道路同步施工的新型檢查井在具有提高質量、提前工期、節材環保的效果的同時，又有了新的有益創新:

( 1) 根據城鎮道路施工不中斷交通的實際需求優化鑄鐵檢查井底支座為下支座和底座，既方便了施工又保障正常安全通行;

( 2) 針對道路使用期內經常有日常養護修補和小中大修及新改建反覆拆裝的特性，鑄鐵檢查井井蓋、上下層支座間的採用了多種可方便安拆的連接方式;

( 3) 在滿足鑄鐵檢查井使用功能的情況下優化了結構作法，節約了原材料和費用;

( 4) 為增強鑄鐵檢查井的穩固性改進了底座錨固的作法。

4 適用範圍

改進後的新型鑄鐵檢查井針對不同層數的瀝青混凝土路面結構做了相應組合匹配，單層、雙層、三層機三層以上分別適用於新鋪築的單層、雙層、三層及三層以上瀝青混凝土面層的道路檢查井。

新型鑄鐵檢查井根據需要可以添加成防滑、防盜、防響、防墜落和防位移的五防檢查井。如井蓋上澆鑄 各專業花紋圖案或特製的藝術圖案，可以起到防滑、美 觀效果; 通過鎖具或連桿將井蓋固定或連接到支座上， 起到防盜效果; 將井蓋和支座通過卡釘和卡槽連接或安放橡膠墊圈，可以起到防響效果; 將底層支座的底板內伸進檢查井一定長度，可以防止子井蓋，起到防墜落 效果; 將底層支座通過預埋筋和焊筋固定到井筒上，能起到防位移效果。

5 結語

新型鑄鐵檢查井依據同步施工原理和一邊施工一邊通行的需求完善創新出了新的作法，依然具有提高質量、提前工期和節能環保、方便施工和安全通行的優點，具有推廣應用的價值。
