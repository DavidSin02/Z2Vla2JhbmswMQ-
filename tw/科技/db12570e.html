public class DataDemo2Action extends ActionSupport implements ModelDriven<User>{		//創建對象	//前提要求：表單輸入項name和實體類的屬性名稱一致	private User user=new User();		@Override	public User getModel() {		// TODO Auto-generated method stub				//返回創建的User對象		return user;	}		@Override	public String execute() throws Exception {		// TODO Auto-generated method stub		//return super.execute();				System.out.println("data>>"+user);		return NONE;	}
