◎ 佛山首家400㎡獨棟“ 製茶工廠” ◎

TA是佛山首家獨棟的【宋代美學茶館】，就開在嶺南天地！憑藉著獨特的新宋茶飲，俘獲了一大批忠實粉絲，最近還給大家帶來了夏日“解暑神器”！



400㎡的獨棟空間

足足有400㎡的茶館，分為上下2層，在當代設計中融入了宋代美學，極力打造極致的茶生活空間。

茶館的名字也蘊含著獨特意味，京時之“京”，取於宋朝都城“汴京”，京時之“時”，是汴京時間，天然古茶與現代茶飲的融合，賦予京時之茶以今時之味。



頂尖的製茶工藝

茶館對茶有著細緻的追求，特別引進國內頂尖的全自動炒茶機，炒制、烘焙、過篩、醒發、攤晾...每步工序都十分嚴謹，只為保留茶葉的原香。

① 炒制

② 烘焙

③ 過篩

④ 攤晾

正是這樣嚴格的製茶工序，加上對天然古茶和現代茶飲的潛心研究，才讓茶館的每一款茶飲都擁有自家獨特的味道。

◎ 店裡頭號“茶魁”，至佳解暑神器 ◎

京時觀音王檸檬茶

這杯讓人你從夏日中徹底解放的京時觀音王檸檬茶，穩坐茶館頭號“茶魁”的地位，一登場就就能感受到它的冰涼氣息~

// 現炒的濃香型觀音王 //

作為一家把茶葉做進骨子裡的茶館，京時採用全自動的炒茶機，用120°的精準溫度炒制80分鐘，將清香型的鐵觀音烘焙成濃香型~



制炒後的鐵觀音，油潤勻整香氣悠長，遠遠地都能聞到醇厚的茶香。



// 純手工捶打的檸檬 //

特別選用3種不同類型的檸檬，將香水檸檬、黃檸檬、青檸檬混合在一起，調配出獨特的檸檬香。


每片檸檬精切成0.3cm，然後純手工捶打，倒入新鮮煮好的茶底手搖4次，茶的微澀和檸檬的酸味完美融合




入口那刻，嘴巴就感受到冰爽的口感，醇厚香韻的茶香給混合著清新的檸檬香，越喝越暢快，脣齒間得到大大滿足~


這樣熱辣辣的夏天，想續命真的要靠這杯京時觀音王檸檬茶，趁著現在買一送一，大家趕緊安排上！



◎ 這些特色新式茶飲，必須安排上 ◎

陽山蜜桃烏龍冰茶


提起茶館裡的“顏王”，這款陽山蜜桃烏龍冰茶必須擁有名字！只要瞄它一眼，就被這桃紅色給治癒了！


陽山蜜桃可是中國國家地理標誌產品！現切的陽山水蜜桃肉粒，倒入水蜜桃果泥製成圓滾滾的冰球，在配上烏龍茶湯，這也真的太撩人了吧！


用力地吸上一口，滿嘴都是清新的烏龍茶香，還能嚼到水蜜桃冰沙果肉，口感超級特別！

臺灣鳳梨金鳳冰茶


採用臺灣進口的鳳梨，大大顆的鳳梨冰球加上清香的金鳳茶底，拿到手上就想立刻開喝~


第一口果斷要嘗這顆鳳梨冰球，入口冰冰涼很是刺激，香甜的鳳梨果肉和茶香瀰漫開來，滿嘴巴都是甘香～


傳統手工花茗茶


除了新式茶飲，這裡的手工花茗茶也是深客官喜愛！傳統的茶具和手工的沖泡，讓傳統茶添上幾分獨特味道~


選茶、鮮調、洗茶...每一步都是現場純手工操作，連泡茶的時間也分毫不差地精確到3分鐘，才能完美保留花茗茶的風味。




現泡的金盞菊烏龍花茶，喝起來清香甘甜，花香與茶香混合在一起，不濃不烈剛剛好~


另外，手工花茗茶還有各種不同的搭配哦，大家可以根據自己的喜好選擇~


冷泡水果茶


每天新鮮製作的冷泡水果茶，每一瓶都浸泡10個小時，有顏值更有內涵哦~


紅顏冰粉


手工製作的冰粉，面層鋪滿了玫瑰花瓣，入口冰涼Q彈帶著淡淡的玫瑰花香。

徑山綠茶冰淇淋

加入徑山綠茶製作的冰淇淋，入口濃濃的茶香味，大夏天的舔一口冰淇淋，真是幸福感爆棚。


茶館裡除了有現制的茶飲、小吃...還有各種關於茶的手工製品哦，客官們可以隨時把它們抱回家哦~



◎ 中式韻味的裝修，感受茶文化 ◎


中式風的裝修搭配木質的桌椅，相比一般的茶飲店，這裡更有一種中式茶文化的韻味~


桌子上整齊擺放著的茶葉和器皿，一點一滴都能感受到濃郁的中國茶的深邃。


坐在茶館的任意一個角落，點上一杯獨特的茶飲，這樣的體驗真的別具一番風味~


覓食小分隊探店小結：

1. 茶館的茶飲真的很特別，茶味比一般的店更點香濃醇厚。

2. 所有的茶葉都是現場的，進到茶館就會聞到濃濃茶香~

3. 個人很喜歡這樣的中式裝修，在這裡喝著新式茶飲蠻有有氛圍的。

4. 茶館還有各種手工製品，喜歡的客官可以來逛逛哦~
