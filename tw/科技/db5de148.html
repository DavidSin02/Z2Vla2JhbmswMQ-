高低壓力錶不會用？看這裡！

2020-11-14
| 科技

很多新手都不知道怎麼去看製冷劑的壓力錶，尤其是看到錶盤上密密麻麻的數字，很多人直接跳過不想看了，但是壓力錶在製冷系統中重要性幾乎涵蓋製冷系統調試、維護、檢修等過程，可謂製冷系統必不可少的一部分。今天我們就帶大家來認識這個壓力錶，希望對新手有用哦。

先來看圖片，首先是高壓表

其次是低壓表

一

製冷壓力錶分類

一般分為“高壓表” 和 “低壓表”

1、“低壓表”最高讀數一般為 1.5MPa 1.8MPa

2、“高壓表”最高讀數一般為 3.5MPa 3.8MPa

二

讀壓力值

1、低壓表最外一圈的數值為MPa的數值，如指針指到 .3 即當前的壓力為 0.3MPa 。

2、高壓表最外一圈黑色大字的數值為MPa的數值，如指針指到2，即當前的壓力為 2.0MPa

3、不同壓力單位的轉換 1MPa=10.2kg/cm^2=10bar=145Psi

三

讀溫度值

1、低壓表，由外往內數

第1圈 黑色 壓力值，單位 MPa (兆帕)

第2圈 綠色 R407C溫度值，單位 C （攝氏度）

第3圈 紅色 R134a溫度值，單位 C （攝氏度）

第4圈 綠色 R22 溫度值，單位 C （攝氏度）

第5圈 綠色 R404A溫度值，單位 C （攝氏度）

如一臺冷水機冷媒使用 R22, 運行時，低壓壓力 0.4MPa ，對應的蒸發溫度約為 0度 。

如一臺冷水機冷媒使用 R134a, 運行時，低壓壓力 0.3MPa 對應的冷凝溫度約為 8度 。

2、高壓表，由外往內數

第1圈 黑色 壓力值，單位 MPa (兆帕)

第2圈 綠色 R407C溫度值，單位 C （攝氏度）

第3圈 紅色 R134a溫度值，單位 C （攝氏度）

第4圈 綠色 R22 溫度值，單位 C （攝氏度）

第5圈 綠色 R404A溫度值，單位 C （攝氏度）

如一臺冷水機冷媒使用 R22, 運行時，高壓壓力 1.8MPa 對應的冷凝溫度約為 50度 。

如一臺冷水機冷媒使用 R404A, 運行時，高壓壓力 1.8MPa 對應的冷凝溫度約為 45度 。

四

補充說明：

1、 MPa 兆帕 kg/cm^2 公斤力/平方釐米

2、不同的壓力錶的壓力單位可能不一樣，讀壓力前要看前錶盤上的標識

3、冷水機，空調一般用 R22 R407C R134a ；

低溫機 R22 R404A ； 家用空調 R22或R410A ；

汽車空調 R134 R12

4、在壓縮機沒有運行時壓力為平衡壓力，即高低壓力相同，其壓力值與環境溫度或水箱水溫對應。

如 R22 環境溫度 30度時，壓力錶的平衡壓力約為1.1MPa

如 R22 環境溫度 10度時，壓力錶的平衡壓力約為0.6MPa
