在 Windows 安裝期間將 MBR 磁盤轉換為 GPT 磁盤

2020-11-14
| 科技

以 UEFI 啟動的 Windows 磁盤必須是 GPT 格式。本文將介紹如何在安裝 Windows 期間將磁盤從 MBR 轉換成 GPT。

特別注意：操作不慎可能丟失所有數據，如果你懂得安裝系統的一些基本概念，那麼可以繼續閱讀並嘗試實操；否則請交給專業人士操作。切記，切記，切記！！！

第一步：按下 Shift + F10 啟動命令提示符

在 Windows 的安裝界面其實是可以啟動命令提示符的，只需按下 Shift + F10 即可。

第二步：敲命令

啟動 Diskpart：

> diskpart

啟動後提示符的前面會出現“DISKPART”前綴：

DISKPART> list disk

這時命令提示符中會列出此計算機上所有的磁盤和其格式：

 磁盤 ### 狀態 大小 可用 Dyn Gpt -------- ------------- ------- ------- --- --- 磁盤 0 聯機 119 GB 118 GB * 磁盤 1 聯機 465 GB 1024 KB 磁盤 2 聯機 28 GB 0 B *

磁盤 0 是我準備裝系統的系統盤。

================ ！！！特別注意！！！ ================

  1. 注意這裡選擇的是磁盤，而不是分區！不是通常所說的 C 盤/D 盤，而是一塊 SSD，或一塊機械硬盤。
  2. 後續操作會清除選中磁盤中的所有數據，是所有數據，毫無保留！

================ ！！！特別注意！！！ ================

現在，我們選中“磁盤 0”：

DISKPART> select disk 0

這裡的序號取決於安裝 SSD 或機械硬盤的的連接線序號，所以一定要仔細查清楚，不要選錯了。

接著，敲入 clean 命令清除此磁盤上的所有內容，注意，這包括了所有的分區：

DISKPART> clean

等待清除結束，然後敲入 convert gpt 命令完成轉換。

DISKPART> convert gpt

第三步：平復激動的心情

操作結束之後直接按下 Alt + F4 切換到 Windows 安裝程序繼續安裝即可。如果你是強迫症重度患者，敲一個 exit 命令結束“Diskpart”程序也未嘗不可。

DISKPART> exit
