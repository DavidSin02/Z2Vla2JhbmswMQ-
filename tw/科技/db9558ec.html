指向字符串的指針（超詳解）

2020-11-14
| 科技

/*

1）什麼是字符串

用雙引號括起來的零個或多個字符"abcd"

2)字符串的結束符號：

\0,它是一個字符，而不是一個字符串

3）字符串的輸出；

printf("%s",數組名)

puts()

輸入:

scanf

gets

4)strlen:用來測量一個字符串中所包含的字符個數

5)如果一個指針p指向了一個字符串，那我們就說p就是一個指向字符串指針

定義格式：char *指針名 ，如char *p

賦值：char *p = "abcd" or char *p; p = "abcd";

可以把一個字符串直接賦給一個指針，但這並不說明把字符串中的所有字符都賦給了p，

實際上p永遠只能指向一個字符，指針p剛開始指向的是字符串的第一個字符

char a[10] = { "1234" }, *p = a;

a---->1<---p

2

3

4

\0

6）指向字符串指針的輸出

1：puts(指針名)

2：printf("%s",指針名)

輸出原理：以當前指針的位置開始，一直向下輸出，直到遇到\0為止

*/

#include<stdio.h>#include<string.h>void main(){	char a[100] = { "abcd123" }, *p = a+4;	printf("%s\n", p);//123	printf("%d", strlen(p));//3}

/*

將一個指向字符串的指針移動到一個數組的末尾

p = a + strlen(數組名) - 1;

*/

#include<stdio.h>#include<string.h>void main(){	char a[100] = { "abcd123" }, *p = a;	p = a + strlen(a) - 1;	printf("%s\n", p);	printf("%d", strlen(p));}

/*

*p代表的是字符串中的某一個字符，所以輸出參數應該是%c

只有輸出字符串才用%s

如printf("%s", p);printf("%c",*p)

*/

#include<stdio.h>#include<string.h>void main(){	char a[100] = { "abcd123" }, *p = a;	while (*p)	{		*p = *p + 1;		p++;	}	puts(a);//bcde234}
#include<stdio.h>#include<string.h>void main(){	char s[] = { "\n123\\" };	printf("%d\n", strlen(s));//5	printf("%d", sizeof(s));//6,其是用來返回一個數組的大小包含\0}


/*

1)一維字符型數組就是字符串

2）以數組作為函數的參數，型參數組就是實參數組，實參數組也就是型參數組，

因為數組名是首地址，實參把數組的地址傳給了實參數組

3）如果把一個數組的首地址賦給了某個指針，則這個指針就可以當數組使用

4）以字符串作為函數參數的幾種格式

A：型參和實參都用數組

型參：fun(char a[])

實參：fun(a)

B：型參用數組，實參用指針

型參：fun(char a[])

實參：fun(p)p是指針

C：型參用指針，實參都用數組

型參：fun(char *p)

實參：fun(a)

D：型參和實參都用指針

型參：fun(char *p)

實參：fun(p)

*/

#include<stdio.h>#include<string.h>void fun(char *a){	int i;	for (i = 0; a[i]; i++)	{		a[i] = a[i] + 1;	}}void main(){	char a[] = "abcd", *p = a;	fun(p);	puts(a);}
