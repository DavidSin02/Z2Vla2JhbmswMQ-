Windows下常用的遠程Linux連接工具

2020-10-29
| 科技

點擊上方藍字，關注↑“空橋”

✎__

在Windows系統上，想要通過遠程連接 Linux 服務器相互傳送文件，那麼你需要一個簡稱 SSH 的 Secure Shell 軟件。

SSH 使用公鑰加密來認證遠程的計算機。你可以有多種途徑使用 SSH，無論是自動連接，還是使用密碼認證登錄。

本篇文章介紹了幾種可以連接 Linux 服務器 SSH 客戶端。

Tera Term

TeraTerm 是一款類似超級終端的軟件，但功能更為強大，最大的特色是支持腳本。非常豐富的腳本命令擴展使TeraTerm能做許多事情，可以大大簡化了工作量，非常適合頻繁使用串口或者Telnet等方式調試的環境。

Bitvise SSH Client

Bitvise SSH 是一款支持 SSH 和 SFTP 的 Windows 客戶端。由 Bitvise 開發和提供專業支持。這款 SSH 客戶端性能強悍，易於安裝、便於使用。Bitvise SSH 客戶端擁有功能豐富的圖形界面，通過一個有自動重連功能的內置代理進行動態端口轉發。

Putty

是一個免費的、Windows 32平臺下的telnet、rlogin和SSH客戶端，但是功能絲毫不遜色於商業的telnet類工具。

WinSCP

是一個Windows環境下使用SSH的開源圖形化SFTP客戶端。同時支持SCP協議。它的主要功能就是在本地與遠程計算機間安全的複製文件

SSHSecureShellClient(ssh client)

SSH Secure Shell Client是一款連接遠程Linux系統的工具，簡稱ssh客戶端，操作簡單方便。特別適用其它系統下遠程操作Linux系統服務。

VNC (Virtual Network Computing)

是虛擬網絡計算機的縮寫。VNC是一款優秀的遠程控制工具軟件，由著名的AT&T的歐洲研究實驗室開發的。VNC是在基於UNIX和Linux操作系統的免費的開放源碼軟件，遠程控制能力強大，高效實用，其性能可以和Windows和MAC中的任何遠程控制軟件媲美

Xmanager

是全新標準的跨平臺集成解決方案。實現類似VNC和WinSCP的功能，功能比較齊全

SecureCRT

是一款支持ssh（ssh1和ssh2）的終端仿真程序，同時支持Telnet和rlogin協議。SecureCRT是一款用於連接運行包括Windows、UNIX和VMS的遠程系統的理想工具。

PieTTY

Windows 上發展的 Telnet/SSH 安全遠端連線程式，修正與完整支援亞洲語系字符，可切換多種 Unicode 字符顯示方式，並增加透明視窗、無邊框模式等視覺效果。特別是在輸入密碼的時候有星號提示。

Cygwin

Cygwin 是一款 GNU 和開源工具的大雜燴，以大量的可選包方式組織而成。在這些安裝包中，有高質量的編譯器和其他軟件開發工具、一個X11服務器、一套完整的X11開發套件、GNU emacs 編輯器、Tex 和 LaTeX、openSSH（客戶端和服務器），除此之外還有很多，包括在微軟 Windows 下需要編譯和使用 PhysioToolkit 軟件的每一樣東西。

DameWare SSH

這個免費工具是一個終端模擬器，可以讓你從一個易用的控制檯建立多個 telnet 和 SSH 連接。

MobaXterm

MobaXterm 是你的遠程計算的終極工具箱。在一個 Windows 應用裡，它為程序員、網管、IT 管理員及其它用戶提供了精心裁剪的一攬子功能，讓他們的遠程操作變得簡約時尚。

Xshell

Xshell 是一個強大的安全終端模擬軟件，它支持SSH1, SSH2, 以及Microsoft Windows 平臺的TELNET 協議。Xshell 通過互聯網到遠程主機的安全連接以及它創新性的設計和特色幫助用戶在複雜的網絡環境中享受他們的工作。
