《pythonweb接口開發》學習1-安裝擴展庫

2020-10-29
| 科技

Windows 命令提示符下操作：

pip

顯示pip的說明信息

pip install Model_Name[==版本號]

pip默認會安裝該庫的最新版本，如果我們知道該庫的版本號，也可以指定某個版本號進行安裝；

pip uninstall Model_Name

使用uninstall命令就可以將安裝的庫卸載了。

tar.gz文件安裝

並不是所有的擴展庫都支持pip命令安裝，對於個別庫可能只提供了壓縮包文件，這時候，就通過“python"命令安裝

例如：Django-1.10.2.tar.gz文件，下載完成後，進行解壓，進入解壓目錄，然後輸入如下命令：

...\Django-1.10.2>python setup.py install

whl文件安裝

wheel本質上是一個zip包格式，它使用.whl擴展名，用於python模塊的安裝。

例如，Django-1.10.2-py2.py3-none-any.whl文件，先下載，然後使用pip命令安裝：

pip install Django-1.10.2-py2.py3-none-any.whl
