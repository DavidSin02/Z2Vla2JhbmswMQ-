介紹一個軟件是很吃力不討好的事兒，思來想去很久，作為徐濤老師的鐵粉，旺仔男仔男孩，我「周又土 - 鐵旺」（以下簡稱「鐵旺」）決定以獨特的方式表達對老師的感激之情，以考研政治的學習為例子將所有 MarginNote 3 的使用重點串起來講。（準備了很久久 T T）

考研政治的 3 個要訣和 2 個題型

天才第一步，先要想清楚：我們首先對考研政治的考試有一個全面的認識：學政治 95% 的人都是為了在考試中取得個高分。那麼，為了應對考試，就要對不同的題目採取不同的學習方法。好在政治只有兩種題型，1 種是選擇題，另 1 種是分析題。

同時結合濤濤所說的「三個要訣」來進行操作：

要決一：聽

  • 聽重點（劃出濤濤說的重點）
  • 聽闡述（老師的解釋，各種形象生動的做包子作為例子的闡述）
  • 聽邏輯（時間線的串聯，每個運動的土地法的變化）

所以我們用 MarginNote 3 做筆記 + 思維導圖整理。

要決二：練

  • 強化班 → 真題集（講完一個模塊就可以開始練題，一個月就可以做完兩遍）→ 習題集
  • 試卷上的大部分題要你深度理解才能做出來
  • 一本真題，一本習題，各練兩遍，錯題集，只練選擇題，不練分析題

所以我們用 MarginNote 3 做錯題集。

要決三：背

  • 用心背書
  • 靈活運用，題目千變萬化，答案始終如一

所以我們用 MarginNote 3 + Anki 背誦。

其中，選擇題側重於聽和練，而分析題側重於背。

MarginNote 3 與「政治選擇題」

聽課，梳理，做筆記

在聽課的時候使用「文檔模式 / 學習模式」，對於能夠進行文字選擇的 PDF 文件，可以直接通過劃線、劃重點、高亮選中文字來創建卡片。

而對於不能進行文字選擇的 PDF 文件，用 iPad 和 Apple Pencil 通過「寫字筆」工具劃出重點，手寫筆記。

各種手寫、打字、錄音、截圖

可是有時候我們需要整段的書上的文字，同時也需要對一個知識點進行詳細的筆記，比如在講剩餘價值的時候，濤濤講了很長時間的做包子的例子 （很生動，很喜歡，聽完很餓），講了很多內容，但是樹上的筆記記不下該怎麼辦呢！Don’t worry，土土有好辦法！

我們可以先框選內容，點擊批註符號，就可跳出一個卡片的界面，然後進行文字輸入、手寫、錄音、截圖。

鐵旺這邊截了濤濤的考研視頻的圖片，同時手畫了做包子的時間圖、打字的筆記。（不好意思有點醜，給自己看，效率第一！）

說實話這段知識是很繞，聽都是能聽懂，和別人複述一遍的時候自己還是講不清楚，所以鐵旺我在這個知識點上花了很多時間。

「鏈接」和「合併」

細心地你一定會發現鐵旺這有個「筆記鏈接」。

當你的筆記想和另一個比價產生關聯的時候，通常的做法有兩種：1. 創立鏈接；2. 合併筆記。

「鏈接」適合距離比較遠的分散知識，建立完成後有「虛線的剪頭」和「右上角的數字標識」。

「合併筆記」適合距離比較近的相關知識，合併成一條筆記的示例如下：

構建思維導圖

進入「學習模式」後，屏幕會分成左側的提綱 \ 思維導圖區域以及右側的原文區域。

當我們做成了一張張卡片之後，所有的知識點還是分散在書中，我們打開思維導圖功能，就能將這一張張卡片構建出一個完美的思維導圖框架！

剋制的顏色，統一的規則

在思維導圖中，你選中文本的顏色就是卡片的顏色，MarginNote 3 在顏色方面給了用戶一些的自由度，但鐵旺我還是建議大家極其「剋制」地使用各種顏色，這樣不易造成不同顏色、工具的混用，最後屏幕上花花綠綠的一片，反倒讓找不到重點。

事不過三，所以我一般只使用五種顏色：飽和度低的黃、綠、紫、紅、藍。同時也定下了一些使用規則：

  1. 黃色用於濤濤強調的重點知識
  2. 綠色用於背誦的知識
  3. 紅色用於超重點，難點，疑問點
  4. 藍色用於書上老師整理的重點表格
  5. 紫色用於題外話

聰明的你可以和鐵旺我不一樣，但是一定要有自己統一的規則。

標籤也要剋制

標籤的功能也要很剋制地使用。我會在整理分類篩選的時候用標籤，有時候老師推薦了什麼電影或者書我都會用標籤標註一下。總的來說，顏色和導圖會更加地直觀。

「只顯示標題」讓腦圖更加清晰

為了觀看方便，可以將顯示調整成「只顯示標題」，讓腦圖更加清晰：

聽著濤濤的強化課慢慢地積累，我們最終能畫出整本政治書的思維導圖！而且是屬於我們每個人自己的思維導圖！

刷題

我們用濤濤的真題集和習題集來練習題目。

框圖梳理線索、做錯題集

錯題是我們最大的財富，錯題集必不可少，但是重新把題目和答案抄在本子上實在是費時費力，無紙化學習的好處就是能夠大大的節省了抄題目浪費的時間。我們可以將錯題框選，然後加入我們錯題集的框框中。

可以修改樹形，將屬性思維導圖修改成框架圖，這樣所有的錯題就被框在了一起。

MarginNote 中提供了「重組」功能，選擇多張卡片使用重組，就會將卡片放置在一個半透明的選框中，好似他們被歸置在某個區域中，可以視作一個集體，之後複習錯的題目就會巨方便！（PS：對於總分結構的文章，將主題相似的卡片重組成一個單獨的框框，再將多個組或卡片混合成樹狀結構的導圖，效果比直接用導圖好得多。）

政治題目因為有很多需要將零散的知識結合串聯起來的題目，比如各種土地法的梳理、各種毛爺爺的文章的整理、中共每一次會議的主題特點，濤濤都會要我們拿出筆記本來做筆記~ 我們就可以用 MarginNote 3 來做某個線索的梳理，讓散落一地的知識串聯起來！

多本教材一張腦圖！

根據課程的進度，需要添加很多新書，MarginNote 3 的思維導圖筆記本可以添加多個文檔，其中的卡片都會顯示在同一個思維導圖中。這樣我們可以不斷豐富一門課唯一思維導圖!

同時也把所有教材放在了一起，方便批量管理和主題閱讀。

新建自定義卡片

在整理的過程中，常常發現已有的卡片中沒有適合作為標題和關鍵節點的筆記，此時可以點擊思維導圖中的添加按鈕，手動添加一些卡片。

大綱也清晰

換一種有條理的方式查看，類似於幕布。

「源文件 + 腦圖 + 大綱」3 合 1

在學習模式下，MarginNote 3 支持通過左側邊緣滑動來拉出大綱，可以同時看到腦圖、大綱、源文件，讓你不用再像二代一樣來回在大綱和腦圖中點擊切換。

幫你 OCR 好了

MarginNote 3 還新增了 OCR 功能，可以將掃描版 PDF 中的圖片內容轉換為文字，免去了手動輸入的麻煩，同時便於搜索和進一步編輯

「搜索」眼裡出西施

最後在無數知識點的熱帶雨林，你可以用搜索來查找你的關鍵詞，有時往往會讓你有意想不到的效果 —— 沒想到這個知識點和這個知識點都有同樣的關鍵詞！

MarginNote 3 與「政治分析題」

分析題很簡單：分類，背誦。

分析題我們用到《考前預測必背 20 題》這本書。

小手一抖做卡片

一張張添加卡片太麻煩可以選擇「自動添加到卡片組」。

有時候有時候做卡片一個個手動輸入問題答案也很麻煩，框選小黃書的「問題部分」做成一張卡片，框選「答案部分」做成一張卡片，然後按順序選中「問題部分」+「答案部分」點擊「合併」，MarginNote 3 會默認第一張圖片作為問題，其他的所有部分為答案。

我通常會在卡片、腦圖、和文檔中來回切換，在腦圖中理解它的知識結構，在文檔中理解上下文，再進行背誦。

還是暫時導出 ANKI 背誦比較好

因為截止目前，MarginNote 3 的複習功能還未完善，所以我們選擇導出 Anki 來進行背誦複習。

Anki 是一個很強大的軟件，這裡就不介紹了，是最好的背誦類軟件了，用就完事兒了！安卓免費，iOS￥168，macOS 免費。

（PS: 在複習模式裡面才能導出到 Anki 背誦哦）

Anki 的工作原理就是按照艾賓浩斯背誦記憶曲線進行問答式（Q & A） 背誦學習。MarginNote 3 的背誦曲線目前不會發生變化，而 Anki 會更根據你反饋的答案難易程度來改變重複的天數。

注意哦1.政治不需要像我這樣花太多時間，我是因為愛徐濤老師，這是情懷2.這是一個MN3使用的範例，我所希望的是大家能舉一反三學習英語+數學+專業課~3.我的思路是根據你不同的考試的要求來制定學習流程，根據不同人的情況，大家要獨立判斷，自行取捨哦～
