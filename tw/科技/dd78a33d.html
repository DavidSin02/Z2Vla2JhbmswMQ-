Python機器人教學——控制數碼管顯示字符

2020-11-14
| 科技

今天的任務是使用Python通過micro:bit主控板來控制數碼管模塊實現字符顯示。

硬件準備：

TM1637數碼管模塊、micro:bit主控板、micro:bit擴展板。

硬件連接


電路連接圖

數碼管的VCC和GND連接到擴展板的VCC和GND，數碼管的DIO引腳連接至擴展板的Pin14口，數碼管的CLK引腳連接至擴展板的Pin13口。

工作原理

我們首先在MU環境中需要導入TM1637的庫文件，該庫文件是TM1637的底層驅動文件，裡面有TM1637類以及附屬很多實用的方法函數，比如shownum(self, num)顯示數字函數，showbit(self, num, bit = 0)顯示位數的函數等等。

TM1637是一種帶鍵盤掃描接口的LED (發光二極管顯示器)驅動控制專用電路，內部集成有MCU數字接口、數據鎖存器、LED高壓驅動、鍵盤掃描等電路。本產品性能優良，質量可靠。主要應用於電磁爐、微波爐及小家電產品的顯示屏驅動。


TM1637數碼管模塊

程序原理

由於程序以及先將TM1637庫文件導入，因此我們可以直接對其進行調用。比如我們想實現一個計數器的程序

from microbit import *

import TM1637

tm = TM1637.TM1637(dio=pin14, clk=pin13)#初始化TM1637模塊

n = 0

while True:

tm.shownum(n)#顯示數字

n += 1#數字累加

sleep(1000)

數碼管每秒鐘進行數據更新
