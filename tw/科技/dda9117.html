SQL注入——數據庫增刪改查語句

2020-10-29
| 科技

我們之前有講過數據庫是用來存儲數據的，那麼我們要如何才能從裡面去查詢到一些數據呢？如何才能把這些數據存儲進去呢？

那這就需要用到我們sql語句了。我們需要用這些語法去操作數據庫。


Sql，是一種特殊目的的編程語言，是一種數據庫查詢和程序設計語言，用於存取數據和查詢，更新還有管理數據庫，它分為三類：

1、 DDL數據定義語言 create，drop

2、 DML數據庫操作語言 select insert update

3、 DCL 數據控制語言 grant（授權）revoke（回收權限）


創建數據庫與刪庫

1） 創建數據庫

語法規則：create database 庫名；


2） 刪除數據庫

語法規則：drop database 庫名；



3） 創建指定編碼數據庫

語法規則：create database 庫名 character set utf8;



創建表


1） 語法規則create table 表名(

字段1數據類型,

字段2數據類型,（最後一個字段後面的逗號省略）

...

);



刪除表

語法規則：drop table 表名;


1、插入數據SQL語法

語法規則：

Insert into 表名（字段1，字段2，字段3，...）values （值1，值2，值3...）;

注：插入數據的時候，表後面跟的字段名字可以省略，但是後面插入的值需要與字段的數量對應。


2、插入數據sql語法

語法規則：insert into 表名 set 字段1=值1，字段2=值2.....;


4、 同時插入多條數據

語法規則：insert into 表名(字段1,字段2,字段3,...)values(值1,值2,值3...),(值1,值2,值3...),

(值1,值2,值3...);



刪除數據

語法規則：delete from 表名 where 字段=值



關係運算符

＞ = ，＞= ，＜ ，＜= ，！=（不等於）


查詢數據

語法規則：select 字段1，字段2，字段3...from 表名 where 條件;



查詢起別名

語法規則：select username a from 表名 where 條件;其中a就是username的別名



修改語句

語法語句：updata 表名 set 字段1=值1，字段2=值2...where條件



最後咱們的增（insert） 刪（delete）改（updata）查（select）就講完了，我們學習sql注入對於這幾個是一定要熟悉。
