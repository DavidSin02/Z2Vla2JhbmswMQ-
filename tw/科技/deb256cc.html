JAVA快速入門——輸入和輸出

2020-10-29
| 科技

輸出

如果你看過快速入門第一篇，就會注意到下面的代碼，向屏幕輸出 Hello World

System.out.println("Hello World")

println是print line的縮寫，輸出並換行。 如果輸出後不換行，可以用print()。

格式化輸出

有時候我們希望輸出的數據顯示成我們期望的格式，比如保留兩位小數；這時候就需要使用格式化輸出的功能。格式化輸出使用System.out.printf()，通過使用佔位符%?，printf()可以把後面的參數格式化成指定格式：

 double d = 3.1415926; System.out.printf("%.2f\n", d); // 顯示兩位小數3.14 System.out.printf("%.4f\n", d); // 顯示4位小數3.1416

Java的格式化 佔位符：

  • %d 格式化輸出整數
  • %x 格式化輸出十六進制整數
  • %f 格式化輸出浮點數
  • %e 格式化輸出科學計數法表示的浮點數
  • %s 格式化字符串

注意，由於%表示佔位符，因此，連續兩個%%表示一個%字符本身。

int n = 12345000;//把一個整數格式化成十六進制，並用0補足8位System.out.printf("n=%d, hex=%08x", n, n); // 注意，兩個%佔位符必須傳入兩個數

輸入

從控制檯讀取一個字符串和一個整數的例子：

import java.util.Scanner;public class LearnJava {  public static void main(String[] args) { 		Scanner scanner = new Scanner(System.in); // 創建Scanner對象 System.out.print("Input str: "); // 打印提示 String name = scanner.nextLine(); // 讀取一行輸入並獲取字符串 System.out.print("Input int nuumber"); // 打印提示 int age = scanner.nextInt(); // 讀取一行輸入並獲取整數 System.out.printf("Hi, %s, you are %d\n", name, age); // 格式化輸出 }}
  1. import語句導入java.util.Scanner，import是導入某個類的語句，必須放到Java源代碼的開頭
  2. 創建Scanner對象並傳入System.in。System.out代表標準輸出流，而System.in代表標準輸入流
  3. 使用scanner.nextLine()讀取用戶輸入的字符串
  4. 使用scanner.nextInt()讀取用戶輸入的整數

小結

  • Java提供的輸出包括：System.out.println() / print() / printf()，其中printf()可以格式化輸出；
  • Java提供Scanner對象來方便輸入，讀取對應的類型可以使用：scanner.nextLine() / nextInt() / nextDouble()
