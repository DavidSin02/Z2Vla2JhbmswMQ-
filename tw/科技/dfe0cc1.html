摘要：針對浮點模糊度精度較差時SEVB算法存在搜索耗時較大的問題，提出一種改進的SEVB算法。該算法通過限制初始搜索空間大小和優化計算過程，能夠有效減少模糊度搜索候選點個數和不必要的冗餘計算，進而提高搜索效率。試驗結果分析表明，當浮點模糊度解算精度較低時，改進算法的搜索效率比SEVB算法明顯提高，且其搜索耗時不易受模糊度維數及精度的影響，具有更好的穩定性。

A Modified SEVB Integer Ambiguity Search Algorithm

LIU Wanke1,2,3, MA Liye1, LU Liguo4, LI Jianlong1

Abstract: A modified integer ambiguity search algorithm named MSEVB is proposed to overcome the disadvantage that SEVB algorithm is time-consuming under low float ambiguity precision.The proposed algorithm, by restricting initial search space and optimizing the calculation procedure, can effectively reduce the number of ambiguity candidates and redundant computation, so that the search efficiency is improved significantly.The experiment results indicate that MSEVB algorithm has significant improvement in search efficiency, compared with SEVB algorithm, when the precision of ambiguity resolution is low.Moreover, MSEVB algorithm is more insensitive to ambiguity dimension and precision, thus it has better performance in stability.

Key words: ambiguity resolution integer search SEVB algorithm Bootstrap success rate search efficiency

相位整週模糊度的快速、準確解算是GNSS實時高精度定位的關鍵問題，也是GNSS研究領域中的熱點問題。在眾多整週模糊度解算方法中，基於整數最小二乘法(integer least squares，ILS)的搜索方法由於具有最高的成功率而被廣泛採用[1-5]。ILS搜索是指在給定空間內尋找出滿足模糊度殘差二次型最小的一組整數解，為快速獲得該組整數解，目前最為流行的是基於震盪式收縮策略的SEVB(Schnorr-Euchner and Viterbo-Biglieri)算法[6-9]。其中，震盪式搜索策略是指從Bootstrap估計點開始向兩側呈“Z字型”枚舉[10]；收縮策略是指每獲得一組二次型更小的解，則進行搜索空間的更新[11-12]。通過上述兩個策略，SEVB算法可以迅速減小搜索空間，從而減少搜索節點數，實現模糊度快速估計，因此該算法也被LAMBDA方法(V3.0)所採用。

採用SEVB算法處理時，首先將初始搜索空間設置為無窮大，以確保獲得的第一組整數候選解為Bootstrap估計解，然後基於該組整數解進行搜索空間的收縮和震盪式枚舉過程。但是當模糊度解算精度較差時，Bootstrap成功率偏低，此時獲得的Bootstrap估計解偏離ILS估計解較多，求得的搜索空間相對偏大，需進行多次枚舉才能實現搜索半徑的更新，導致搜索過程緩慢，耗時明顯增大。針對這一問題，本文通過限制初始搜索空間大小和簡化搜索算法實現過程兩個方面對SEVB算法進行改進，以期提高搜索效率。

由於搜索空間的大小可以不斷收縮，通常SEVB算法具有很高的搜索效率。但當模糊度解算的精度較差時，Bootstrap估計解與ILS估計解偏差較大，造成第1次的搜索空間很大，需要多次重複不必要的節點枚舉才能將搜索空間縮小到較小的範圍內，大大增加了搜索耗時。因此，本文從簡化算法實現和限制初始搜索空間大小兩個方面對現有的SEVB算法進行改進。

3 改進的SEVB搜索算法

針對現有SEVB算法存在的上述不足，本文在其基礎上進行了優化，增加了兩點改進策略，改進後的算法稱為MSEVB(Modified SEVB)。

3.1 改進策略1

初始搜索空間不再設為無窮大，而是首先計算出一個空間較小的初值，該初值既要儘可能的小，又要保證搜索空間內有整數解。初值計算方式分為兩種情形[21-22]：

(1) 如果ncands≤n+1(n為模糊度維數，ncands為待輸出的候選組個數)，構造出n+1個二次型值。第1個二次型值通過對模糊度浮點解序貫取整計算得到，然後仍按照序貫取整的方法，依次對n維模糊度中的一個分量取次接近整數，得到其餘n個二次型值。對這n+1個二次型值從小到大排序，取第ncands個二次型值作為初始空間大小χ02。

綜上所述，MSEVB算法的計算步驟為：

(1) 參數輸入及初始化。需輸入的參數包括模糊度浮點解ẑ及方差協方差陣Qẑẑ；初始化主要指對初始搜索空間大小χ02以及第1次Bootstrap估計時的初值進行設置，其中搜索過程從第n層處開始。

(2) 判斷當前局部空間大小newDist與χ02的關係。若滿足newDist < χ02則向下一層搜索，採用式(15)計算下一層的序貫條件估值，更新newDist並再次判斷其與χ02關係，若仍滿足newDist < χ02則繼續向下一層搜索，依此類推；若不滿足newDist < χ02則返回到上一層的下一個整數點，更新newDist並重復步驟(2)。

(3) 若能一直向下搜索直到第1層，則成功找到一個模糊度候選組，使用該候選組的目標函數值對χ02進行更新。之後在新的χ02內繼續搜索第1層的下一個整數點，若仍滿足newDist < χ02，則又找到一個候選組，再次對χ02進行更新，嘗試下一個整數點；當第1層沒有整數點時則返回到第2層的下一個整數點，更新newDist並回到步驟(2)。

(4) 當重新返回到第n層時則整個搜索過程結束，最後一次得到的候選組即為最優整數解。

5 結論

現有的SEVB算法在浮點模糊度精度較差時，由於初始空間設定為無窮大，所得到的模糊度初始估計候選組(Bootstrap估計解)偏離真值較大，導致模糊度搜索空間難以有效地收縮，且空間內包含較多的死節點造成冗餘計算較多(見式(13))，致使搜索耗時較大，影響了整週模糊度的整體解算效率。

針對這一問題，本文在SEVB算法基礎上，從限制初始空間大小和優化分層節點計算過程兩個方面開展了算法改進，提出了一種改進的SEVB搜索算法(MSEVB算法)。分別基於仿真數據和動態車載實測數據從不同角度對MSEVB算法進行了效果驗證與分析，得到以下結論：

(1) 對於確定的模糊度維數來說，隨著模糊度精度的降低，相應的成功率Ps_ib減小，SEVB算法的搜索耗時顯著增加，而改進的MSEVB算法耗時增長較小，因而具有更高的搜索效率，且Ps_ib越小，MSEVB算法的性能提升效果越明顯。

(2) 在相同的模糊度解算精度下，隨著模糊度維數的增加，MSEVB算法相比於SEVB算法的耗時增長更加平緩且波動更小，維數越高，改進算法的解算優勢越明顯。

綜上所述，本文的MSEVB算法是基於現有的SEVB算法發展而來的，是對SEVB算法的完善，不僅在低精度、高維模糊度的搜索方面具有更為明顯的效率優勢，而且可以整體上保證模糊度解算效率相對更加可靠穩定。
