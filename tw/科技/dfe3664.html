5G NSA 空口信令流程學習小結

2020-10-29
| 科技


RRC/X2AP/S1AP信令流程

（1）S1AP/InitialContextSetupRequest, MME >> MeNB



* 如果id-HandoverRestrictionList包含nRestriction參數，則表示LTE核心網禁止UE做NSA業務

（2）LTE-RRC_DCCH_DL (ueCapabilityEnquiry), MeNB >> UE



（3）LTE-RRC_DCCH_UL (ueCapabilityInformation), UE >> MeNB



（4）LTE-RRC_DCCH_DL (ueCapabilityEnquiry), MeNB >> UE



（5）LTE-RRC_DCCH_UL (ueCapabilityInformation), UE >> MeNB

（6）LTE-RRC_DCCH_DL (ueCapabilityEnquiry), MeNB >> UE


（7）LTE-RRC_DCCH_UL (ueCapabilityInformation), UE >> MeNB



* 如果UE未上報NR能力，則可能終端不具備5G能力

（8）S1AP/InitialContextSetupResponse, MeNB >> MME



（9）LTE-RRC_DCCH_DL (rrcConnectionReconfiguration), MeNB >> UE



（10）LTE-RRC_DCCH_UL (measurementReport), UE >> MeNB



（11）X2AP/SgNBAdditionRequest, MeNB >> SgNB


...


...


...


（12）X2AP/SgNBAdditionRequestAcknowledge, SgNB >> MeNB


...

* 如果SgNB添加失敗，SgNB則回覆為SgNBAdditionReject

（13）LTE-RRC_DCCH_DL (rrcConnectionReconfiguration), MeNB >> UE


（14）LTE-RRC_DCCH_UL (rrcConnectionReconfigurationComplete), UE >> MeNB

* 如果失敗則UE隨後立即回覆RRC SCG Failure Information NR消息給MeNB。

（15）X2AP/SgNBReconfigurationComplete, MeNB >> SgNB

MeNB向SgNB確認UE已完成重配


（16）X2AP/SgNBReleaseRequest, MeNB >> SgNB


（17）X2AP/SgNBReleaseRequestAcknowledge, SgNB >> MeNB

（18）LTE-RRC_DCCH_DL (rrcConnectionReconfiguration), MeNB >> UE
