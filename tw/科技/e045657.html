簡單、同步剪貼板歷史與常用語工具

2020-10-29
| 科技

Rainclip 是一款簡單的剪貼板工具，它可以幫你保存近期的剪貼板內容，以供再次使用，可以將常用的文字保存起來，可隨時調用，還能對剪貼板內容進行二次利用。

對於剪貼板應用，2T的要求是：夠快、夠穩定，然後如果漂亮點就最好了。

就是用上來說，Rainclip 可以通過 Ctrl + Shift + V 呼出，響應也算是非常快速了，此時可以直接輸入內容進行搜索：

並且可以多次按下此快捷鍵進行不同分類 Snippet 的切換。Rainclip 支持將常用短語、文本保存在 Snippet 中，供未來隨時使用。

不過 Rainclip 的設置頁面還是醜的可以。

設置頁面裡面的 Actions，是一個支持 js 腳本的自定義功能，目前列出了幾個：

  • 將字母轉換成全大寫
  • json 格式化
  • 解析 HTML 代碼
  • 插入當前時間
  • 生成隨機密碼
  • 創建 GUID
  • 獲取 Unix 時間戳

不過這個用起來2T還是有點疑問，因為每一個腳本都可以設置快捷鍵，但2T完全不知道如何激活這個快捷鍵。倒是每一個 Actions 都可以在剪貼板列表中啟用。

最後，無論如何，有一個歷史剪貼板工具，絕對可以讓效率翻倍。

關於同步，Rainclip 支持 Mac 與 Windows 電腦，需要 Google 賬號同步，這個只能會的同學上了。

Rainclip 可以在官網下載rainclip.net
