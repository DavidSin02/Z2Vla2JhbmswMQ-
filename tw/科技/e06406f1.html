Windows Server 2008 R2 主域控制器、文件服務器遷移、升級記

2020-11-14
| 科技

最近幾天沒發文章，因為有一客戶，6年前買的戴爾R720服務器，最近頻發藍屏、死機，每次只能硬重啟，有時候還不一定能順利啟動進入系統，幾經討論後，決定購買一臺新的戴爾R740XD服務器來替換，舊服務器待檢修後再另作他用。

實施計劃 Plan A：新服務器安裝vmware esxi，然後利用vmware converter，將舊服務器的Windows Server 2008 R2 直接轉換成虛擬硬盤，然後導入vmware，至於文件嘛，再慢慢往新服務器拷。

1、新的戴爾R740XD服務器貨到之後，立刻下載VMware-VMvisor-Installer-6.7.0.update03，並且完成安裝，這個就不多說了，前面文章就有。

2、舊服務器上安裝VMware-converter-en-6.2.0，然後開始轉換，注意，只能選擇C盤，即轉換系統盤就可以了，數據盤有8T，沒必要轉換，也沒條件轉換；操作過程略，可以參考本人於2019年9月18日發表的文章；

3、轉換了兩小時，終於完成了，生成兩個文件：一個是vmx文件，3KB；另一個是vmdk，20G；

4、登錄vmware esxi，創建虛擬機，選擇剛才轉換而來的兩個文件；

5、提示導入文件失敗，只支持OVF和vmdk導入，不能支持vmx和vmdk的導入！好吧，還得轉一次，我的筆記本電腦裡面有安裝好的VMware Workstation，裡面有個OVFTool能轉成OVF格式，但是如果用的我筆記本電腦來轉換，我得先下載這個20G的文件，轉換完成後，還得上傳到ESXI，我可等不了那麼長時間，於是把OVFTool複製到舊服務器，準備直接轉換，結果提示有一堆文件未註冊，好吧，上傳VMware Workstation，再安裝一遍，這些文件就應該被自動註冊了吧？嘿嘿，說幹就幹，可是，安裝到90%的時候，系統提示發生未知錯誤，安裝進度條開始回滾，靈光一閃，立刻使用任務管理器，強制結束回滾退出安裝程序，我又不需要完全安裝VMware Workstation，只要某些文件被註冊了就行啊，哈哈，不得不為自己點個贊！果然，OVFTool 能用了，開始轉換vmdk，生成名為disk1.vmdk文件，等等啊，一個小時，進度條走了一點點， 算了，時間不早了，先下班吧。。。

6、等等，還不能下班，萬一轉換不成功，還是得另想辦法啊，Plan B 立刻出現在腦海裡：新建虛擬機安裝Windwos Server 2019，配置成第二臺域控，然後再把舊服務器的共享文件夾拷過來，重新配置共享和網絡驅動器。說幹就幹，新建虛擬機、裝系統、安裝Active Directory 域服務、將此服務器提升為域控制器（注意：此處必須選擇 “將域控制器添加現有域”，然後輸入域名，以及域管理員的帳戶和密碼），這一步，可以參考本人2019年10月7日發表的文章；

7、乾等著沒什麼意義，先下班回家。。。

8、晚飯時間看了看，文件轉換完成，得到三個文件：.mf文件、.ovf文件、.vmdk文件，可是這個vmdk文件佔用空間竟然小了一半！什麼情況？ 不管它，開始用它來創建虛擬機，實際上就是物理機導入虛擬機的過程。。。

9、晚上休息之前，看了一眼，還是沒完成。。。

10、凌晨一點，上廁所的時候看了一眼筆記本，導入還是沒完成！我汗。。。

11、輾轉難眠，凌晨三點，再起來看筆記本——為了客戶真是嘔心瀝血啊——提示導入完成，BUT ！虛擬機在哪裡？怎麼憑空消失了？沒有任何錯誤報告！看日誌，只有一條消息： 虛擬機files已被移除！什麼意思，導入成功完成，然後又被自動移除了！ 都沒讓我看一眼啊，憑什麼？！

12、算了算了，半夜三更，天氣冷颼颼，腦子也是一團漿糊的狀態，無法多作思考。。。罷了罷了，不要舊系統了，反正第二臺Windwos Server 2019域控已經起來了；

13、新的戴爾R740XD服務器上面，有兩個RAID5陣列，第一個是3塊512G固態硬盤組成的，主要用於安裝操作系統；第二個是由6塊4T的SAS硬盤組成的，主要用於存儲，遂將這塊實際可用容量18T的硬盤掛載給了Windwos Server 2019，並且開始從舊服務器複製文件，鑑於舊服務器經常藍屏死機，決定採用fastcopy來複制數據，理由是，複製過程中出現任何錯誤，只要重新開始複製，軟件會快速地自動跳過已經複製的文件，此時凌晨3點半！

14、文章起草到這個時間節點（晚上21:05），已經接近42小時了，卻只複製了3.2T數據，中間藍屏一次，死機一次，中斷時間大概就40分鐘左右，我汗，總共將近9T的數據要複製，要多長時間才能完成啊？！清香三柱上給舊服務器，求您別再藍屏、死機了！

IT 外包服務，我真的是認真的！ 寫文章，我也是很認真的，都是自己的實際經驗！
