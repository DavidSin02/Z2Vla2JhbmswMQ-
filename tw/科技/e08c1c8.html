微信支付、微信提現、提現到銀行卡的RSA公鑰加密

2020-10-29
| 科技

由於微信支付最近用的挺多的，最近花了點時間把微信相關的接口封裝成了一個npm,直接安裝就能使。本模塊集成了大部分微信支付、提現等模塊的接口，採用async、await的方式調用，使用者不用再考慮參數加密發送，密鑰發送方式、xml怎麼解析、json怎麼轉成xml等一系列麻煩事。

注：

noncestr（可以自定義）、sigin不需要傳入，所必須的參數值請看微信文檔，sign_type支持MD5,HMAC-SHA256,不傳默認MD5

微信支付

安裝

yarn add node-wxpay3 或者 npm i node-wxpay3 --save

支付產品api

名稱（方法名）

統一下單（unifiedorder）

回調驗證（callback_check）

查詢訂單（orderquery）

關閉訂單（closeorder）

申請退款（refund）

查詢退款（refundquery）

下載交易賬單（downloadbill）

下載資金賬單（downloadfundflow）

交易保障（report）

拉取訂單評價數據（batchquerycomment）

付款碼支付（micropay）

撤銷訂單（reverse）

付款碼查詢openid（authcodetoopenid）

例子：

const { WxPay } = require('node-wxpay3');

const fs = require('fs');

const wxpay = new WxPay({

'appid': 'appid',

'mch_id': '商戶id',

'partner_key': '', // partner_key為商戶平臺設置的密鑰key

'pfx': fs.readFileSync('./apiclient_cert.p12'), // 證書

});
