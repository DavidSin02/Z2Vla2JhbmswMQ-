SpringAOP面向切面編程功能測試

2020-11-14
| 科技

AOP【動態代理】：指在程序運行期間動態的將某段代碼切入到指定方法的指定位置的編程方式。整個處理流程分以下7個步驟。

/** * AOP【動態代理】: * 指在程序運行期間動態的將某段代碼切入到指定方法的指定位置的編程方式。 * 1、需要導入AOP模塊：Spring-aspects * 2、創建一個業務邏輯類（MathCalculator），在業務邏輯運行的時候進行日誌打印（方法之前、方法之後、方法正常返回、方法出現異常） * 3、定義一個日誌切面類（LogAspects）：切面裡的方法需要動態感知MathCalculator.div運行到通知方法 * @Before：前置通知（logStart） * @After：後置通知（logEnd） * @AfterReturning：正常返回通知（logReturn） * @AfterThrowing：出現異常通知（logThrows） * @Around：環繞通知，動態代理，手動推進目標方法執行（ProceedingJoinPoint.proceed()） * 4、給切面類的目標方法標註何時何地進行（通知註解） * 5、告訴Spring哪個類是切面類@Aspect * 6、開啟基於註解的AOP模式@EnableAspectJAutoProxy * 7、將切面類和業務邏輯類（目標方法所在類）都加入到容器中 */

業務邏輯類：MathCalculator

public class MathCalculator{	public int div(int i ,int j) {		System.out.println("MathCalculator.div...");		return i/j;	}}

日誌切面類：LogAspects，@Aspect告訴容器該類是切面類，@Pointcut告訴容器切入點，通常用表達式來表示

@Aspectpublic class LogAspects{	@Pointcut("execution(public int com.ck.aopTest.MathCalculator.*(..))")	public void pointCut() {	}	@Before("pointCut()")	public void logStart(JoinPoint point) {		System.out.println(point.getSignature().getName() + "運行……@Before參數列表：{" + Arrays.asList(point.getArgs()) + "}");	}	@After("com.ck.aopTest.LogAspects.pointCut()")	public void logEnd(JoinPoint point) {		System.out.println(point.getSignature().getName() + "運行結束@After.");	}	@AfterReturning(value = "pointCut()", returning = "result")	public void logReturn(JoinPoint point, Object result) {		System.out.println(point.getSignature().getName() + "正常返回，@AfterReturning返回值：{"+result+"}");	}	@AfterThrowing(value = "pointCut()", throwing = "exception")	public void logThrows(JoinPoint point, Exception exception) {		System.out.println(point.getSignature().getName() + "出現異常，@AfterThrowing異常信息：{"+exception+"}");	}}

基於代碼方式實例化bean對象，@EnableAspectJAutoProxy告訴容器開啟基於註解的AOP模式：

@EnableAspectJAutoProxy@Configurationpublic class MainConfigOfAOP{	@Bean	public MathCalculator calculator() {		return new MathCalculator();	}	@Bean	public LogAspects logAspects() {		return new LogAspects();	}}

測試類：

public class IOCTestAOP{	@Test	public void test() {		AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext(MainConfigOfAOP.class);		MathCalculator calculator = applicationContext.getBean(MathCalculator.class);		calculator.div(6, 2);		System.out.println();		calculator.div(1, 0);		applicationContext.close();	}}

單元測試@Before、@After、@AfterReturning、@AfterThrowing運行結果如下：

div運行……@Before參數列表：{[6, 2]}MathCalculator.div...div運行結束@After.div正常返回，@AfterReturning返回值：{3}div運行……@Before參數列表：{[1, 0]}MathCalculator.div...div運行結束@After.div出現異常，@AfterThrowing異常信息：{java.lang.ArithmeticException: / by zero}

注意：使用AOP編程，我們需要引入aspectj模塊，但是aspectj版本和jdk有不兼容的情況，我jdk用的1.7，aspectj用的1.5.4版本，然後運行就報如下錯誤：

0 can't find referenced pointcut pointCut

org.springframework.beans.factory.BeanCreationException: Error creating bean with name 'mainConfigOfAOP': Initialization of bean failed; nested exception is java.lang.IllegalArgumentException: error at ::0 can't find referenced pointcut pointCut

然後我把aspectj模塊更換成aspectjrt-1.6.6.jar和aspectjweaver-1.6.6.jar，問題就解決了。
