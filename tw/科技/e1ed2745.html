最好的DJ音箱

2020-11-14
| 科技

作為表演DJ，要把音箱放在和打碟機、混音裝置同等重要的位置。誠然，混合音軌需要一些技巧，但是如果沒有高質量的音箱將這些音樂播放出去，那麼舞池裡的人群可能會跳著跳著就睡著了。無論您是常駐DJ還是巡演DJ，選出最好的DJ音箱都不是件容易的事兒，因為可選的音箱種類實在是太多了，今天我們將會就這點給大家提供些建議，希望這些建議會對您有所幫助。

我們還會對產品的功能做詳細的介紹，來幫助您做出選擇。因為音箱之間的差異可能非常細微，所以這會是您挑選出最適合您的DJ音箱的有利的工具。

注意：我們希望我們的建議對您來說更加實用，所以我們不會僅簡單的羅列要推薦的音箱，我們的推薦不會只考慮音箱本身。以下推薦的產品排列順序不分先後，基本上每組音箱都能滿足您的所有需求。

超低音音箱很重要

任何想要購買新的DJ音箱的DJ都可能正在尋找一款完全匹配的超低音音箱。 如果您沒有超低音音箱，您一定要考慮購買一臺。 低頻之於電子音樂或嘻哈音樂就像牛奶之於麥片一樣必不可少。 由於波長較長，超低音音箱還可以為您的聲音增添更多魅力，而且超低音音箱能讓人們更有跳舞的慾望。 因此，我們在下面的大多數推薦中都加入了一個超低音音箱。

即插即用

除了完善您的音域之外，DJ音箱還需要一些額外的配件，這些配件通常不會和音箱隨附。下面的每組推薦中都有用於將音箱堆疊到超低音音箱，在任何階段互聯它們以及在演出後保護它們等用途的配件。

需要關注的點

  • 有源/無源—雖然我們首推有源音箱，但市場中也有許多無源音箱可供選擇。如果您選擇使用無源音箱，那麼您還要為它配備一個供電的外置放大器，所以在大多數情況下，有源音箱是一個更好的選擇，因為它能為您的運輸減負。
  • 功率—影響音箱實際音量輸出（額定聲壓級或SPL）的重要因素是音箱的功率輸出，功率以瓦特為單位。根據RMS(均方根)或峰值功率的不同，額定功率會有很大的差異。RMS是放大器的連續功率的額定值，而峰值則是在非常短的時間內(有時是幾分之一秒)最高輸出功率的額定值（沒有功率損耗）。 如果您要比較規格，請確保進行對比的是RMS與RMS或峰值功率與峰值功率， 否則功率比較毫無意義。 相比較而言，RMS是衡量真實功率更好的指標。
  • 材質—如今的音箱箱體不僅可以用木頭做，還可以用注塑塑料(ABS)，ABS材質的箱體結構更穩定並且能降低音箱的總體重量，是用於巡演的音箱的最佳選擇。
  • 重量—音箱的重量是巡演DJ需要考慮的重要因素。尺寸比較大的音箱在之前是獲得比較大的音量的有利工具，但現在卻不然。找到一個強大而不龐大的音箱是最理想的。
  • 輸入/輸出—音箱輸入和輸出設置因音箱而異。 關於輸入，大多數音箱都有1/4″插孔，XLR插孔或這兩者都適用的combo插孔; 有些音箱也有藍牙無線輸入或專有的物理輸入（speakON或類似的東西，通常在無源音箱中出現）。 許多有源音箱還帶有XLR格式的輸出或直通選項，用於連接鏈中的其他音箱。 某些音箱可能還會有以太網插孔，用於連接軟件以進行配置和其他控制選項。 要注意檢查輸入和輸出接口，以確保音箱與您現有的需求和裝備匹配。
  • 便攜性—為了便於移動，有些音箱上面自帶握把和腳輪。剛開始您可能並不會覺得這種設計有多重要，但當您移動音箱成百上千次之後，您就會知道這種設計的好處了。
  • 成本—您的預算是需要您著重注意的，但最好不要因為價格犧牲質量。否則，您可能很快就需要再買一套新的音箱。
  • 箱體和外觀—您打算如何和在哪裡使用您的新DJ音箱?是在室內、室外還是兩者皆有? 是在婚禮還是俱樂部? 音樂風格是嘻哈，電子，還是流行? 當您在購買音箱時，確保您已經明確其使用的場景，根據需求進行購買。

我們的首推

EV組合

3 x Electro-Voice ZLX-12P

3 x Electro-Voice ZLX-12-CVR

2 x Electro-Voice ELX118P

2 x Electro-Voice ELX118-CVR

2 x On-Stage Stands SS7745

2 x Pro Co XLR5

EV的ZLX和ELX系列是性價比很高的DJ音箱組合。ZLX-12P是一款有源的雙功放（1000瓦峰值和250瓦RMS額定功率）音箱，配有12英寸低頻驅動器和1.5英寸鈦合金壓縮高音單元， ZLX使用塗層聚丙烯複合塑料製做箱頂和側面手柄，這種材質比木質材料更輕（僅34磅），更便於運輸。 在後面板上，您可以看到一個簡單的輸入和輸出設計，其中包含麥克風，樂器或電路板饋送的組合輸入。 此外，ZLX-12P還具有LCD顯示屏，可輕鬆調節信號處理。

ELX118P有源超低音音箱具有700瓦的峰值功率和能處理134dB最大聲壓級的18英寸EVS-18K低音單元。ELX118的箱體是15毫米彩繪膠合板材質，這種材料可以減少長期的磨損造成的箱體損壞。 如果您在設置中需要一些額外的低音，可以切換後面板上的boost開關，並且如果您過度使用了您的音箱，限制器指示燈會給您發出警告。

QSC組合

3 x QSC K10.2

3 x QSC K10 Tote

2 x QSC KS212

2 x QSC KS212C

2 x QSC KSub Pole

2 x Pro Co XLR5

QSC在此類應用中的口碑不斷增長，其K系列最受歡迎， K.2就是一個很好的例子。 K10.2音箱是配有雙功放，10英寸喇叭和1.4英寸高音單元的組合，具有令人印象深刻的2000瓦D級功率模塊 – 功率是之前K系列音箱型號的兩倍。 K.2 系列可實現多樣化功能及靈活性應用。可對多個參數進行配置，實現多種功能：舞臺監控、舞曲播放、樂器擴聲、手持話筒擴聲等；同時，可對應用場景進行儲存，以便於調用定製化參數配置，如輸入通道類型、延遲、均衡、分頻及所選界面模式等。單憑這些，這個系列的音箱就能成為DJ們的最佳選擇。

這些音箱是演出DJ的絕佳選擇的另一個原因是那對兒KS212有源超低音音箱。KS212內置雙12英寸低音單元，每隻單元帶六段帶通濾波器 ，由3600W功放模塊推動，同時由系統DSP處理功能控制，使設備前方投射的聲能比後方多15dB。 該產品除了配備有高級DSP 處理功能以外，也可對場景參數進行儲存調用（通過顯示屏、控制面板操作）， 滿足用戶在不同使用環境下的使用需求。

JBL組合

2 x JBL SRX835P

1 x JBL SRX818SP

2 x On-Stage Stands SS7745

DJ和PA音箱領域中最廣為人知的品牌之一就是JBL。 其SRX835是一款有源，配有15英寸和6.5英寸喇叭、頂部配有3英寸高音單元的音箱。 SRX835的這種設計可以使其覆蓋幾乎整個人類聽覺頻率範圍（33Hz至21kHz）。 它們有2000瓦峰值功率、1500瓦連續額定功率，這對於功能強大的Crown Audio音箱來說非常有意義。 至於設置，SRX835具有靈活的板載DSP預設和可調參數。 不過，多虧了可以完全控制您的高端PA系統的Audio Architect，HiQnet Motion Control iOS和其他獨立應用程序，JBL對音箱設計的發展更進一步。

SRX818有源超低音音箱非常強大。 每個818都有750瓦連續額定功率，1000瓦峰值D類額定功率，最大聲壓級輸出為135dB。 它由與835相同的軟件控制，當串聯使用時，您可以對系統進行最終的聲音控制。 這也意味著您可以隨身攜帶這個系統到任何需要它的地方，並且它在任何地方都能聽起來很棒。

Bose組合

Bose F1 PA Package

Bose是家庭和商用音箱創新的領導者，其F1系列音箱就是一個很好的例子。 F1系列非常適合演出DJ，因為它方便，便攜的設計和靈活的應用程序能滿足各類場地需求。例如，F1 MODEL 812音箱配有八隻Bose 專利的 2.25 英寸單元、100°水平波導、一隻高功率 12 英寸低音單元和較低的分頻點，F1 MODEL 812 音箱具有出色的最大聲壓級，性能表現明顯優於傳統音箱。 如需進一步提升低頻響應，可選的 Bose F1 超低音音箱能將大型低音音箱的所有性能融入到更加緊湊的設計之中。Bose 專利的音箱安裝支架集成在超低音音箱箱體之上，便於快速安裝。支架內含線槽，音頻與電源接線可整齊地隱藏於其中，整體視覺效果更美觀。

便攜式婚禮DJ

Yamaha StagePas 600BT便攜式PA系統

如果您正在尋找一個配有快速設置和包含混音器的婚禮DJ音箱組合，那麼Yamaha StagePas 600BT組合是您的理想之選。 600BT具備兩個美觀且輕量化的音箱以及一個可拆卸的有源調音臺，同時還包含一對音箱線和一根電源線。 10通道調音臺包含4個 mic/line 輸入以及6個 line 輸入。輸入端口3和4採用combo接口，因此可以接插XLR或者1/4插口的線纜。此外，600BT還設置了立體聲 RCA 接口，立體聲 mini 接口以及 1 個可用於連接智能手機、平板或者電腦的無線藍牙播放接入功能。Yamaha StagePas 600BT價格也很實惠，是一款很棒的婚禮DJ設備。

一個非常經濟的選擇

Mackie SRM450組合

我們推薦的的經濟型DJ設備是Mackie SRM450音箱，音箱支架和電纜。 SRM450是1000瓦，12英寸有源音箱，內置雙聲道調音臺。 為了便於調整聲音，SRM具有可選擇的音箱語音選項，如PA，DJ，監聽和獨奏模式。 為了保持聲音乾淨，每個音箱都採用了專利的聲學校正算法、雙路撥號交叉、相位校正和一個易於使用的自動反饋抑制器進行優化。 與隨附配件搭配使用，無論是對於初學者或者是老手來說，SRM450都是一個非常不錯的選擇。
