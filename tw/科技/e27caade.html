多態的作用

先看下面的代碼：

class Animal: def __init__(self, name):  self.name = name class Cat(Animal): def talk(self): print('my name is cat')class Dog(Animal): def talk(self): print('my name is dog')# 這個函數接收一個animal參數，並調用它的animal_talk方法def animal_talk(obj): obj.talk()d = Dog()c = Cat()animal_talk(d)animal_talk(c)------------------打印結果：my name is dogmy name is cat

貓、狗都繼承了動物類，並各自重寫了talk方法。animal_talk()函數接收一個animal參數，並調用它的talk方法。可以看出，無論我們給animal傳遞的是狗、貓，都能正確的調用相應的方法，打印對應的信息。這就是多態。

而多態則是為了實現另一個目的—接口重用！多態的作用，就是為了類在繼承和派生的時候，保證使用“家譜”中任一類的實例的某一屬性時的正確調用。
