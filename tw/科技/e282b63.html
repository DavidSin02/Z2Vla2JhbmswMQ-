Windows 10更新Bug禁用了文件資源管理器右鍵單擊

2020-10-29
| 科技

IT之家11月28日消息 Windows 10版本1909（2019年11月更新）正在逐步推送。2019 Windows 10更新十一月版是一個小更新，它有望帶來更好的性能，但這並不意味著一切都會順利進行。

正如我們上週報道的那樣，2019 Windows 10更新十一月版中的錯誤Bug讓文件資源管理器凍結並最終崩潰。用戶搜索文件、文檔或圖片時，文件瀏覽器的搜索欄將無響應。

雖然尚不清楚是什麼原因導致文件資源管理器出現問題，但這似乎是對微軟新的統一搜索體驗的不良反饋。

現在2019 Windows 10更新十一月版引入了另一個煩人bug，用戶無法在搜索窗格中右鍵單擊。CTRL + V仍然有效，但是資源管理器會掛起幾秒鐘，然後用戶才能輸入文本。

根據10月22日創建的另一個帖子，該問題首先由微軟社區測試人員報告，然後該公司向消費者發佈了2019年11月更新。

“我有完全相同的問題：當我右鍵單擊資源管理器搜索框時，我什麼也沒得到，沒有粘貼也沒有複製。用戶單擊後得到的只是過去搜索詞的歷史記錄，甚至無法刪除。”某用戶表示。

文件資源管理器搜索框中的右鍵單擊支持是一項必不可少的功能，用戶顯然很生氣。不幸的是，目前還沒有針對該錯誤Bug的解決方法，但你可以使用傳統的Windows搜索來查找文件。
