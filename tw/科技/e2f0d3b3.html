C｜從形參與實參的內存地址區別值傳遞與址傳遞

2020-11-14
| 科技

1 值傳遞

函數調用時，為函數的形參分配地址。

同樣的，實參也要分配地址，當然與形參的地址不同。

實質就是傳值時，實參只是簡單把自己的值賦值給形參，形參與實參之間沒有關係；

//傳數值不會改變原來值的例子。

#include <stdio.h>

void swap(int, int); //函數參數採用傳數值方式

void main()

{

int a=585, b=696;

swap(a, b); //傳數值

printf("In main: a=%d, b=%d\n", a, b); //返回主函數後

printf("In main: a=%#p, b=%#p\n", &a, &b); //返回主函數後

system("pause");

}

//將變量作為參數，以傳數值方式傳遞參數

void swap(int a, int b)

{

int temp=a;

a=b;

b=temp;

printf("In swap: a=%d, b=%d\n", a, b); //在被調函數裡

printf("In swap: a=%#p, b=%#p\n", &a, &b); //返回主函數後

}

運行結果：

In swap: a=696, b=585

In swap: a=0X0012FEEC, b=0X0012FEF0

In main: a=585, b=696

In main: a=0X0012FF44, b=0X0012FF40

2 址傳遞

在傳遞地址時，系統為指針變量分配地址，指針變量存儲的是指向實參的存儲地址，所以在被調函數中若改變形參的值，也就改變了原來變量的數值。

傳址時，實參的把自己的地址傳址給形參，在實參與形參之間通過指針指向建立了邏輯關係。

//傳遞指針改變實參值的例子。

#include <stdio.h>

void swap(int*, int*); //函數參數採用傳地址值的方式

void main()

{

int a=585, b=696;

printf("&a=%#p\t &b=%#p\n", &a, &b); //實參地址

swap(&a, &b); //傳地址值

printf("返回後數據：a=%d, b=%d\n",a, b);

}

void swap(int *p, int *t)

{

int temp;

printf("&p=%#p\t &t=%#p\n", &p, &t); //指針地址

printf(" p=%#p\t t=%#p\n", p, t); //形參地址

temp=*p; //交換兩個變量的值

*p=*t;

*t=temp;

printf("交換數據為：a=%d, b=%d\n", *p,*t);

}

運行結果：

&a=0X0012FF44 &b=0X0012FF40

&p=0X0012FEEC &t=0X0012FEF0

p=0X0012FF44 t=0X0012FF40

交換數據為：a=696, b=585

返回後數據：a=696, b=585
