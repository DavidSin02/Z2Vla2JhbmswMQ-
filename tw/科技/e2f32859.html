java基礎第四篇——數據類型及編碼規則

2020-11-14
| 科技

數據類型

Java語言是一個強調數據類型的語言，在聲明任何變量時，必須將該變量定義為一種數據類型。java中的數據類型包括基本數據類型和引用數據類型（也叫對象型）。

java中主要包括8大基本類型：4種整形、1種字符型、2種浮點型、1種布爾型。除了這幾種其他都是引用數據類型如StrongReference（強引用）、WeakReference（弱引用）等。

整形

整型就是整數類型，也就是沒有小數點的數字。在java種整型主要有四種：字節型（byte）、整數型（int）、短整型（short）、長整型（long）

  1. 字節型（byte）

byte用一個字節來表示整數值，它的範介於-128~127之間。這種類型擁有所有的進制，但無論何種進制，在輸出時系統都會自動轉化為十進制.

數據進制轉化

2. 短整型（short）

short用兩個字節來表示整數，其整數值介於-32768~32767之間，它有八進制、十進制和十六進制三種表示方法，其表示方法與字節型一樣。

3. 整數型（int）*

整數型又叫int型，用四個字節來表示整數值，在實際開發中應用最多的整型就是int型。其整數值介於-2147483648~2147483647之間。整數型擁有以上的所有進制，其方法也和字節型一樣。

4. 長整型（long）

long用8個字節表示整數型，其數值介於-9223372036854775808~9223372036854775807之間。他所有的特性與前面的幾種整型一樣，唯一不同之處在於長整型後面有一個“L”字母，如圖

字符型（char）

字符型數據是平時程序設計中使用較頻繁的類型，其佔用兩個字節。特別需要注意的是，它必須以單引號表示如‘A’表示一個字符A，而“A”雖然也只有一個字符，但因為是雙引號，所以它表示的就是字符串而不是字符。

字符型變量聲明

浮點型

浮點型數據表示有小數點的數字，共有兩種：單精度浮點型（float）和雙精度浮點型（double）.

  1. 單精度浮點型（float）

單精度浮點型佔四個字節，有效數字最長為7位數（包括小數部分），其定義變量的方式與long類似。在每個單精度數據後面都有一個標誌性符號“F”或“f”。例如：

float x=22.22f；

float y=22.22F；//兩個都是單精度浮點型

float z=22.22；//不是單精度浮點型，系統會認為是雙精度浮點型

2. 雙精度浮點型（double）

雙精度浮點型數據佔據8個字節，有效數最長為15位。

布爾型（boolean）

布爾型數據就是“是”與“否”。在程序中使用“真”與“假”來代替“是”與“否”，即“true”與“false”。如果定義了一個布爾變量但沒有賦初值，則默認為false。

數據類型轉換

數據類型的轉換分為自動轉換和強制轉換。自動轉換是程序在執行過程中“悄然”進行的轉換，不需要用戶提前聲明，一般是從位數低的類型向位數高的類型轉換；強制類型轉換則必須在代碼中聲明，轉換順序不受限制。

  1. 自動類型轉換

自動轉換按從低到高的順序轉換。不同類型數據間的優先關係如下：

低--------------------------------------------->高

byte,short,char-> int -> long -> float -> double
