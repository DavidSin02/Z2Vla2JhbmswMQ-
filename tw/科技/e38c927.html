一、什麼是函數嵌套調用

函數的嵌套指的是用一個函數調用另一個函數，從而實現需求；前面的文章中有出現過函數嵌套的例子，在實際開發過程中，基本上都會用到嵌套函數。

1、定義階段就進行了嵌套定義

定義嵌套，子函數只屬於母函數；

包括：函數對象、閉包函數、裝飾器、生成器、函數的遞歸調用都是屬於在定義階段就進行了函數嵌套。

在定義階段；在一個函數內再定義了一個函數；

二、函數嵌套時，作用域與名字的查找關係

在Python中，名稱空間與作用域分為三種：內置、全局、局部；

函數嵌套使用的情況下，作用域與名字的查找關係，遵循局部 > 全局 > 內置，這樣的由內往外查找的方式；以變量所以位置作為起始點，向外層查找名字。

例如：變量在全局時，直接在全局中查找；變量在局部時，就先在局部中查找，局部沒有，向全局中查找；


三、函數對象的介紹

函數對象就是指把函數當成變量使用；

①　把函數當作參數傳到另一個函數中，傳的就是函數名稱的內存地址；例如：fun3(fun)；

②　把函數當作另一個函數的返回值；

③　把函數賦值，當作變量使用；例如：x = fun ，然後執行 x(5)，x傳參，執行函數體功能；

④　把函數當作容器類型的一個元素；例如：list = [fun,]，調func為：list[0]，執行fun函數體list[0]()。

用了個小例子說明：

【def xu():

return 1111

def fun(x):

return x

y = fun(xu)

print(y())】

說明：函數y=fun(x)這一行表示，y=x這個函數名的內存地址

y() 就表示 xu()，所以最後得到的是x這個函數的返回值，結果是1111。
