第1題：根據我國現行建設工程總投資及工程造價的構成，

B: 固定資產投資+鋪底流動資金

C: 固定資產投資

D: 建設投資

解析：根據2019年教材第二頁圖表示選擇A和B都為建設項目總投資，選項D為工程造價的一部分

教材第1頁第一段第三句話（其中建設投資和建設利息之和對應於固定資產投資，固定資產 投資與建設項目的工程造價在量上相等）所以正確答案為C

第2題：關於設備原價的說法，正確的是（ C ）

A：進口設備原價是指其到岸價

B: 國產設備原價應通過直接查詢相關交易價格或向生產廠家詢價獲得

C: 設備原價通常包含備件費在內

D: 設備原價佔設備購置費比重增大，意味資本有機構成的提高

解析：選項A 2019版教材第5頁最後一段：進口設備原價是指進口設備的抵岸價，即設備抵達買方邊境，港口或車站，交納完各種手續費，稅費後形成的價格。

選項B 2019版教材第4頁第二節 中（一）國產設備原價的構成及計算描述國產設備原價一般指的是設備製造廠的交貨價或訂貨合同價，即出廠價格，它是一般根據生產廠或供應商的詢價，報價，合同價確定，或採用一定的方法計算確定。

選項C 2019版教材第4頁第二節中圖1.2.1下面一句話，設備原價指國內採購設備的出場價格，或國外採購設備的抵岸價格，設備原價通常包含備品備件費在內。

選項D 2019版教材第4頁第二節第一段一句話；在生產性工程建設中，設備及工器具購置費用佔工程造價比重的增大，意味著生產技術的進步和資本有機構成的提高。

第8題：根據我國現行建設項目總投資及工程造價的構成，在工程概算階段考慮的對一般自然災害處理的費用，應包含在（ B ）內

解析：2019版教材第30頁，基本預備費是指投資估算或工程概算階段預留的，由於工程實施中不可預見的工程變更及洽商，一般自然災害處理，地下障礙物處理，超規超限設備運輸等而可能增加的費用，亦可稱為工程建設不可預見費。

基本預備費由四部分組成 (工程變更洽商) (一般自然災害處理) (不可預見地下障礙物處理的費用) (超規超限設備運輸增加的費用)

第9題:某新建項目,建設期為2年,分年均衡進行貸款,第一年貸款200萬元,第二年貸款300萬元.在建設期內貸款利息只計息不支付.年利率為10%的情況下,該項目應計建設期貸款利息為( B )萬元

A：36 B: 46 C: 52 D: 70

解析:2019版教材第30頁和31頁

建設利息的計算,根據建設期資金用款計劃,在總貸款分年均衡發放的前提下可按照當年借款在年中支用考慮,即當年借款按半年計息,上年借款按照全年計息.

.即第一年借款利息為(0+200*1/2)*10%=10

第二年借款利息為(10+200+300*1/2)*10%=36

所以建設期借款利息總和為10+36=46萬元

第10題: 關於工程定額的應用,下列說法正確的是( D )

A：施工定額是編制施工圖預算的依據

B: 行業統一定額只能在本行業範圍內使用

C: 企業定額反應了施工企業的生產消耗標準,宜用於工程計價

D: 工期定額是工程定額的一種類型,但不屬於工程計價定額

解析:選項A,預算定額是編制施工圖預算的依據,施工定額是編制施工預算的依據,不可混淆

選項B,教材第41頁,行業統一定額是考慮到各行業專業工程的技術特點,以及施工生產和管理水平編制的,一般是隻在本行業和相同專業性質的範圍內使用.

選項C,教材第41頁,企業定額是施工單位根據本企業的施工技術,機械裝備和管理水平編制的人工,材料,機具臺班等的消耗標準.企業定額在企業內部使用,是企業綜合素質的標誌.

第18題:某工程採用兩票制支付方式採購某種材料，已知材料原價和運雜費的含稅價格分別為500元/t、30元/t，材料運輸損耗率、採購及保管費率分別為0.5%、3.5%。材料採購和運輸的增值稅率分別為13%、9%。則該材料的不含稅單價為( C )元/t。

A: 480.87 B 481.47 C 488.88 D 489.49

解析：2019版教材第69頁

若運輸費用為含稅價格，則需要按照兩票制和一票制兩種支付方式分別調整

兩票制支付方式，所謂兩票製材料是指材料供應商就收取的貨物銷售價款和運雜費向建築企業分別提供貨物銷售和交通運輸兩張發票的材料。在這種方式下，運雜費以接受交通運輸與服務適用稅率9%扣除增值稅進項稅額

材料原價不含稅價格=500/1.13=442.48元

運雜費不含稅價格=30/1.09=27.52元

材料的不含稅單價=（442.48+27.52）*（1+0.5%）*（1+3.5%）=488.88元

第26題:某項目根據《建設項目投資估算編審規程》（CECA/YKT-2015），採用概算法編制的估算中，工程費用為8000萬元，工程建設其他費用為800萬元，基本預備費為880萬元，價差預備費為120萬元，建設期利息為200萬元，流動資金為100萬元，則該項目建設投資估算為( B )萬元。

A: 9680 B 9800 C 9951 D 10100

解析：2019版教材第140頁（也可根據管理教材計算）
