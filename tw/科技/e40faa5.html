鏈表面試常考題目—鏈表逆序

2020-10-29
| 科技

鏈表逆序，是力扣第206道題，屬於easy級別。
https://leetcode-cn.com/problems/reverse-linked-list/

題目描述：
反轉一個單鏈表。

示例:

輸入: 1->2->3->4->5->NULL輸出: 5->4->3->2->1->NULL

鏈表基礎知識

public class ListNode {    public Object value;    public ListNode next;    public ListNode(Object value) {        this.value = value;    }}

value存儲在鏈表中的值，next指向下一個節點。

先來一道簡單的題目，認識下鏈表：
（1）定義5個節點；（2）將這5個節點連接起來；（3）定義5個節點；


 public static void main(String[] args) {        //1、定義5個節點        ListNode node0 = new ListNode(10);        ListNode node1 = new ListNode(20);        ListNode node2 = new ListNode(30);        ListNode node3 = new ListNode(40);        ListNode node4 = new ListNode(50);        //2、將5個節點穿起來        node0.next = node1;        node1.next = node2;        node2.next = node3;        node3.next = node4;        node4.next = null;        //3、遍歷鏈表        ListNode head = node0;        while(head != null){            System.out.println(head.value);            head = head.next;        }    }

鏈表逆序的基本思路
依次遍歷鏈表節點，每遍歷一個節點就反轉一個節點。

原始鏈表結構

原始結構中，首先定義一個新的鏈表節點 new_head。

第一次循環

第一次循環，我們把節點1(head)和new_head串起來，即 head.next = new_head 。但是在這之前，我們需要先把原始鏈表中節點1(head)的next指針(節點2)保存下來，然後把節點2變成原始鏈表的head節點。

代碼實現：

public class Solution {    public ListNode reverseList(ListNode head) {        //新鏈表的頭結點        ListNode newHead = null;        while(head != null){            //備份原始鏈表head節點的下一個節點            ListNode temp = head.next;            //把新鏈表的頭結點和原始鏈表頭節點串起來            head.next = newHead;            //將原始鏈表的頭結點變成新鏈接的頭節點            newHead = head;            //將第一步備份的原始頭結點的下一個節點變成原始鏈表頭結點            head = temp;        }        return newHead;    }}

簡單說，就是要先備份下一個節點，將原始頭結點和新頭節點串起來，再角色互換，原始鏈表頭結點變成新鏈表頭結點。
