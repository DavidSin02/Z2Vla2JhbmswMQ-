算法：字符串相加 （JAVA）

2020-11-14
| 科技

給定兩個字符串類型的非負整數num1和num2。求兩者的和？

不能將字符串轉化為整數計算，也不能使用java的BigDecimal庫。

解題思路，字符串轉成char類型，然後在減去‘0’，即可獲得個位數上的數字。

class Solution {    public String addStrings(String num1, String num2) {        int i = num1.length()-1;        int j = num2.length()-1;        int add = 0;        StringBuffer buffer = new StringBuffer();        while(i>=0 || j>=0 || add!=0){              int x = i>=0?num1.charAt(i)-'0':0;  //獲取每一位數字              int y = j>=0?num2.charAt(j)-'0':0;              int sum = x+y+add;  //相加，注意滿10要進行進位              add = sum/10;              buffer.append(sum%10);              i--;              j--;        }        return buffer.reverse().toString();    }}
