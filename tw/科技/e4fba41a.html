css 選擇器

2020-10-29
| 科技

CSS中的選擇我們可以分為兩種：類選擇器和ID選擇器。

類選擇器：用 .class 表示，選取帶有指定類 (class) 的元素，用於為具有相同類名的元素定義相同的樣式。

ID選擇器用：用id 表示，它指定HTML元素的唯一ID，選擇具有特定ID的元素，我們需要寫一個 # 字符，後緊跟著元素ID。


定義一個無序列表，然後通過類選擇器進行樣式設置：

<style>    li{      font-size: 20px;      line-height: 26px;    }    .li1{color: green;}    .li2{color: red;}</style><ul>    <li class="li1">同學計劃</li>    <li class="li2">學習路線</li>    <li class="li1">教程系列</li>    <li class="li2">教程內容</li>    <li class="li1">文章在線</li>    <li class="li2">VIP會員</li>    <li class="li1">關於我們</li></ul>



定義一個id選擇器：

<style>#xkd{    color: green;    font-size:60px;    font-weight: bold;}</style><p id="xkd">俠課島</p>

注意事項：

  • id屬性可以在任何HTML元素上使用
  • id值區分大小寫，它的值至少包含一個字符且不能包含空格



動手小練習

  1. 定義一段文本，然後在文本中設置一個id選擇器在某個元素上，再對這個元素進行樣式簡單設置
  2. 定義一個無需或者是有序列表，通過類選擇器對 <li>標籤進行統一樣式設置
