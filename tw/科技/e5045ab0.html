第四步：數據類型？什麼鬼？

瞭解數據類型之前，我們得先知道什麼是數據

數據：是指對客觀事件進行記錄並可以鑑別的符號，是對客觀事物的性質、狀態以及相互關係等進行記載的物理符號或這些物理符號的組合。它是可識別的、抽象的符號。

好吧，這樣的定義看著醉醉的，簡單點說吧，就是你能在電腦上看到的、看不到的一切具有實際意義的東西都叫做數據，比如說你現在看到的這篇文章，他就是數據，上面的圖片，他也是數據，為了對這個不同的數據進行區分和處理，這裡就有了【數據類型】這個詞，也就是用來對不同的數據進行分類之後，便於計算機對相應的數據進行處理。

那麼，有哪些數據類型呢？簡單來說，有以下這麼幾種：

圖片來自www.runoob.com

在這裡我們就說簡單的幾個，int，doule，float，char

  1. int：整型，就理解為整數吧
  2. float：單精度浮點型，就理解為位數比較少的小數唄
  3. doublle：雙精度浮點型，額這個就理解為位數比較長的小數
  4. char ：字符型 ，就是這樣的'a','b'這些都叫字符

上面說的這些都叫基礎數據類型，還有一種比較高級的叫引用數據類型，具體的理解會在後面有專門的解釋，現在大概可以理解為，由多種基礎數據類型組成的複雜數據類型，可以是系統中已經定義好的，也可以後面我們自行定義這樣的數據類型

第五步：哪這些數據類型怎麼用呢？

先看這一段代碼：

int a = 100;

double b = 100.00;

float c = 100.00f;

char d = 'a';

String e = "HelloWorld";

這幾句代碼的意思是定義變量，基本格式是這樣：

【數據類型】 變量名 ；

如果是這樣：

【數據類型】 變量名 = xxxxx ；則是在定義變量後，並給他賦值

這樣，我們就可以將數據保存起來，在後續我們需要使用的地方進行使用了。

第六步：保存的數據怎麼用呢？

用處挺多的，簡單的來說一下運算符這個東西吧，挺複雜的

//賦值運算符

a = 250;//在這裡，【=】 就是賦值運算符 作用是將等號 右邊 的值賦給 左邊 的變量

//四則運算符 之 加

b = a + 100;//在這裡，【+】 就是加法運算符 作用是將加號 左右兩邊的值相加

//四則運算符 之 減

b = a - 100;//在這裡，【-】 就是加法運算符 作用是將加號 左右兩邊的值相減

//四則運算符 之 乘

b = a * 100;//在這裡，【*】 就是加法運算符 作用是將加號 左右兩邊的值相乘

//四則運算符 之 除

b = a / 100;//在這裡，【/】 就是加法運算符 作用是將加號 左右兩邊的值相除
