美菲特M1106-2D單路SDI視頻會議obs錄直播超高清採集卡解析

2020-11-14
| 科技

美菲特M1106-2D單路SDI視頻會議obs錄直播超高清採集卡，支持多種操作系統，可以採集一路SDI超高清信號，支持多種輸入接口，適用於高分辨率圖像的採集，並且為用戶提供完整的SDK開發包。

美菲特M1106-2D單路SDI視頻會議obs錄直播超高清採集卡

超高清採集卡支持的操作系統

●Windows 7/8/8.1/10/2008/2008 R2/2012 (x86 & x64)

●Linux (基於 V4L2 的內核驅動源代碼，須簽訂保密協議，支持 x86，x64 & ARM 架構)

●Mac OS X（10.9及以上）

超高清採集卡支持的 APIs

●Windows

○DirectShow

○DirectKS

○Wave API/DirectSound/WASAPI

●Linux

○V4L2

○ALSA

超高清採集卡支持的軟件

●VLC VirtualDub

●OBS

●XSplit vMix

●VidBlaster

●Wirecast

●Microsoft Media Encoder

●Adobe Flash Media Encoder

●任何其它使用 DirectShow 或 V4L2 編程接口的編碼或流媒體軟件

超高清採集卡輸入接口

●BNC

●SD/HD/3G SDI

輸出接口

●PCIe Gen2 x1

●BNC (SDI 環回)

超高清採集卡輸入特性

●輸入視頻最高支持 2048x2048 分辨率

○SDI 信號相關的特性

●集成電纜均衡器，HD-SDI 信號傳輸距離最長可達 230 米

●支持 SD/HD/3Ga/3Gb/3Ga-DL/3Gb-DS 標準

●支持 2K (2048x1080)模式

●支持 RGB 4:4:4，YCbCr 4:4:4，YCbCr 4:2:2 色彩採樣

●支持 10/12-bit 的色彩深度

●支持提取 SMPTE 352 有效負荷識別符

● 支持 8 通道的音頻通道，可達 48KHz

●支持提取音頻形成信息和通道狀態數據

●對 3Gb-DS 有限的支持: 只能採集第一個流

●對 3Gb-DL 有限的支持:

○2K YCbCr 4:2:2 10-bit 1080p 50/59.94/60: 採集為 1080i 50/59.94/60

○其他的模式: 均支持

●對雙鏈路接口的有限的支持 (僅採集第一個接口)

○YCbCr 4:2:2 10-bit 1080p 50/59.94/60: 採集為 1080i 50/59.94/60

○YCbCr 4:4:4 10-bit：採集為4:2:2

○RGB 4:4:4: R/B子採樣

超高清採集卡輸出格式

●輸出信號分辨率最高可支持 2048x2048 像素

●輸出幀率最高可支持 120fps (實際輸出幀率可能會受限於 PCIe 帶寬，對於更高的畫面分辨率，即

●1280x1024 以上分辨率，實際輸出幀率可能會受限於板載視頻處理硬件的像素時脈，即 1920x1080 分辨率可達到的最大的幀率約為 80fps)

●支持 4:2:0 8-bit 輸出格式: NV12、I420、YV12

●支持 4:2:2 8-bit 輸出格式: YUY2、YUYV、UYVY

●支持 4:4:4 8-bit 輸出格式: V308、IYU2、V408、 rgb24、 rgb32、

●支持 4:4:4 10-bit 輸出格式: V410、Y410

●可通過 Pro Capture SDK (基於 DirectKS 的版本) 支持更多的輸出格式

超高清採集卡視頻處理功能

●使用兩條視頻處理流水線，每條流水線的處理帶寬可達 180M 像素/s 完全達到 10-bit 視頻處理

●視頻剪裁

●視頻縮放

●視頻去隔行

○兩場行交錯

○兩場混合

○僅頂場

○僅底場

●視頻寬高比變換

○自動或手動選擇輸入視頻寬高比

○自動或手動選擇輸出視頻寬高比

○三種寬高比變換模式: 忽略 (各向異性)、剪裁、填充 (上下或左右填充黑邊)

●視頻色彩空間轉換

○自動或手動選擇輸入色彩空間和量化範圍 自動或手動選擇輸出色彩空間、量化範圍和飽和度範圍

○支持 RGB、YCbCr 601、YCbCr 709、YCbCr 2020 色彩空間 支持有限或完整的量化範圍

○RGB24、RGB32

○支持有限、完整和擴充色域的飽和範圍

●視頻幀率變換

●視頻 OSD 合成

○支持 PNG 格式 OSD 畫面 (最大畫幅為 2048x2048)

○通過 SDK 支持動態加載 RGBA OSD 畫面

超高清採集卡一機多卡

●支持同一個系統中安裝多片卡

●板載撥碼開關，從 0 到 F，提供 16 個編號，用於對卡進行編號

●當撥碼開關設定 1 時，系統硬件設備樹中將顯示：“01：Pro Capture AIO”，以此類推

●在用戶軟件中顯示的視頻音頻設備也將包含卡的編號

超高清採集卡多路輸出流

●對於任一路輸入信號可輸出任意路輸出流

●可以對每一路輸出流單獨設置剪裁、寬高比、色彩空間、分辨率、幀率、去隔行方式、色彩調節、OSD 等屬性。

支持超低延遲

●64 行視頻延遲

●通過 SDK 實現部分完成通知模式

超高清採集卡時間戳和 A/V 同步

●基於硬件實現的 100ns 高分辨率時鐘

●音頻幀 (192 個音頻採樣) 和視頻幀都被硬件時鐘標記到達採集卡的時間戳

●硬件時鐘可跨卡同步 (通過 SDK 實現)

超高清採集卡視頻輸出 SG-DMA

●在 PCIe 2.x 的系統中每路 DMA 帶寬可達約 700MB/s

●在 PCIe 1.x 的系統中每路 DMA 帶寬可達約 400MB/s

●支持自動檢測 Intel GPU 塊狀圖像表面

●支持 AMD GPU 的 DirectGMA

●支持 Nvidia GPU 的 GPUDirect

超高清採集卡SDK

●Pro Capture SDK for DirectShow 為開發者提供了快速的整合採集卡擴展特性的途徑 (Windows)

●Pro Capture SDK for DirectKS 可以為開發者提供最大的靈活性和最高的性能 (Windows)

Windows 驅動程序自定義修改

●所有的選項都可以通過註冊表進行三級的管理：所有設備，特定型號和特定設備

● 視頻、音頻、十字交叉過濾器名稱可通過註冊表定製

超高清採集卡固件升級

●在同一系統中的多片卡可以同時升級

● 無需關閉電源即可完成卡的升級 (在大多數情況下，無需重啟系統)

超高清採集卡LED 指示燈

●LED 燈的狀態顯示了每一路的工作狀態: 空閒，輸入信號已鎖定，內存錯誤或 FPGA 配置錯誤
