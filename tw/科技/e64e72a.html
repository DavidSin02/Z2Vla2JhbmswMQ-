常用資源網站大全

2020-10-29
| 科技

資源搜索：

  • Gitlogs -- Gitlogs 是專門針對 GitHub 項目的搜索引擎，我們通過他可以快速找到想要項目。
  • 鳩摩搜書 -- 可以用來找一些技術文檔手冊，很多在百度網盤裡。
  • Stack Overflow -- 大部分編程問題都能在這裡找到答案。
  • Github -- 最大的開源項目資源網站。
  • Iconfinder -- 用來查找一些類似 logo 都小圖標。
  • TinEye -- 用圖片來檢索圖片，我們可以上傳圖片或輸入圖片的 URL 來檢索。
  • SemanticScholar -- 一個免費學術搜索引擎，其檢索結果來自於期刊、學術會議資料或者是學術機構的文獻。
  • LibreStock -- 可以檢索一些優質的高清圖片。
  • CC Search -- CC Search 上搜索到的圖片資源都是無版權的，我們可以免費的使用。
  • Pexels -- 高質量的圖片網站，可以免費使用。
  • Unsplash -- 免費高清素材網站。
  • The App Store -- 一個針對蘋果手機、iPad、Mac 設備的應用搜索工具。
  • Ebooke -- 一個基於互聯網並提供免費電子圖書下載的搜索引擎網站。
  • Goobe -- 為程序員服務的互聯網搜索引擎

    • 圖片和文檔資源

  • TinyPNG -- 免費的在線圖片壓縮工具，壓縮率高，無損畫質，直接拖拽使用，很方便。
  • picdiet -- 圖片壓縮工具，能極速壓縮 80% 的圖片大小，而不損害其質量。
  • SmartResize -- 這個工具基於 JavaScript+HTML5 編寫而成的，是一個批量裁剪縮放圖片的神器！
  • Convertio -- 在線轉換文件的簡單工具。支持超過309種不同的文檔、圖像、電子表格、電子書、文檔、演示文稿、音頻和視頻格式。
  • Online converter -- 可以轉換視頻、音頻、圖片、文檔等。
  • PDF to Word Converter -- PDF 文檔轉化工具。
  • PDF24 -- 免費且易於使用的在線 PDF 工具。
  • 在線文件轉換器 -- 免費在線轉換視頻,在線轉換音頻,在線轉換圖形,在線轉換文檔和壓縮。
  • Xmind -- 跨平臺、跨終端，擁有全功能的思維導圖軟件。
  • ProcessOn -- 免費在線作圖，實時協作，支持流程圖、思維導圖、原型圖、UML、網絡拓撲圖、組織結構圖等。
  • MindMaster -- 多平臺思維導圖軟件，可用於Windows，Mac和Linux等桌面環境，也可以在線使用或在蘋果，安卓等移動端上使用。
  • MindMeister -- 在線思維導圖工具，已有超過1000萬個用戶。
  • 百度腦圖 -- 簡潔的在線思維導圖工具。
  • Gliffy -- 可以在線創建流程圖、UML圖、線框圖、BPNM、組織架構圖等。
  • 印象筆記 -- 在你的手機和電腦上，使用印象筆記來捕捉、培養和共享靈感。
  • 有道雲筆記 -- 提供了PC端、移動端、網頁端等多端應用，用戶可以隨時隨地對線上資料進行編輯、分享以及協同。
  • OneNote -- 微軟出品等雲筆記。
  • 幕布 -- 在幕布中，大綱筆記與思維導圖可以一鍵轉換
  • 為知筆記 -- 一鍵收藏、全端全文檢索、多級目錄、Markdown
  • 石墨文檔 -- 一款輕便、簡潔的在線協作文檔工具
  • Google keep -- Google 雲筆記平臺。
  • Simplenote -- 可幫助您輕鬆記錄筆記、列表、想法等。

    • 編程工具

  • RegExr -- 正則表達式在線測試工具。
  • Regulex -- 正則表達式在線測試工具。
  • rextester -- 在線執行 PHP、Python、C、Java 等各種語言代碼。
  • Ideone -- 在線執行 PHP、Python、C、Java 等各種語言代碼。
  • GDB online Debugger -- 在線執行 PHP、Python、C、Java 等各種語言代碼。
  • JSFiddle -- 前端代碼在線執行工具。
  • CodePen -- 前端代碼在線執行工具。
  • JSON 解析器 -- 在線 JSON 工具，可以格式化 JSON 數據。
  • ToolFK -- 程序員在線工具
