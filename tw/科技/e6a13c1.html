2019年必看的22部爽片，先馬後看！

2020-10-29
| 科技

儘管離2019年還有大半年，但按照好萊塢「先定檔，再開拍」的成熟行規，19年全年主要大片的檔期，大多數都已經安排上啦。

所以扒叔特地為你們整理了，明年最值得期待的那些好萊塢頂尖大片。

一共有22部，先馬再看！

1

《地獄男爵》1月11日上映

導演: 尼爾·馬歇爾

主演: 大衛·哈伯 / 伊恩·麥柯肖恩 / 米拉·喬沃維奇 / 金大賢 / 薩莎·萊恩

《地獄男爵3》流產後，該系列迎來了重啟版，新《地獄男爵》目前被定為了R級，並敲定由《漢尼拔》、《西部世界》的導演尼爾·馬歇爾執導，《怪奇物語》裡的警官大衛·哈伯擔任新男爵。

期待值★★★★

2

《玻璃先生》1月18日上映

導演: M·奈特·沙馬蘭

主演: 詹姆斯·麥卡沃伊 / 安雅·泰勒-喬伊 / 布魯斯·威利斯 / 塞繆爾·傑克遜

別被片名唬住了，其實它是24重人格精分電影《分裂》的第2部。

相信一美精湛的演技（肉體）依然是本片最大亮點。

期待值★★★★

3

《X戰警：黑鳳凰》2月14日上映

導演: 西蒙·金伯格

主演: 蘇菲·特納 / 詹姆斯·麥卡沃伊 / 邁克爾·法斯賓德 / 詹妮弗·勞倫斯 / 傑西卡·查斯

這應該是很多人最期待的X戰警系列電影了。目前關於《黑鳳凰》的消息並不多，但據說《黑鳳凰》將一改《天啟》那樣輕鬆的故事，迴歸《第一戰》那種嚴肅黑暗的氛圍。

不過很多人期待的還是可以看到一美和老萬了吧。

期待值★★★★★

4

《馴龍高手3》3月1日上映

導演: 迪恩·德布洛斯

主演: 傑伊·巴魯切爾 / 克里斯汀·韋格 / 凱特·布蘭切特 / 傑拉德·巴特勒

《馴龍高手3 》由夢工場動畫出品，該系列是影史上最受歡迎的動畫系列之一，而這部備受期待的新片是該系列的第三部。

需要注意的是續集裡連無牙仔都有女朋友啦！

期待值★★★★★

5

《驚奇隊長》3月8日上映

導演: 安娜·波頓 / 瑞安·弗雷克

主演: 布麗·拉爾森 / 裘德·洛 / 安妮特·貝寧 / 嘉瑪·陳 / 塞繆爾·傑克遜 / 本·門德爾森 / 李·佩斯

不誇張的說《復聯3》都算是為這部電影做的鋪墊，

而《驚奇隊長》在《復聯4》之前上映前，也說明，她真是全村的希望了！

期待值★★★★★★★

6

《小飛象真人版》3月29日上映

導演：蒂姆·波頓

主演: 伊娃·格林 / 邁克爾·基頓 / 科林·法瑞爾 / 丹尼·德維託 / 道比·歐帕瑞

迪士尼經典動畫《小飛象》於1974年首映，經過76年的時間，記憶中的小飛象又回來了。

真人版同樣講述了馬戲團裡的一隻小象因耳朵巨大被動物和人類嘲笑，但最終小象發現自己耳朵可以飛翔，並一躍成為馬戲團明星……

總之蒂姆·波頓執導的電影一定要期待！

期待值★★★★★

7

《沙贊》4月5日上映

導演: 大衛·F·桑德伯格

主演: 馬克·斯特朗 / 扎克瑞·萊維 / 傑克·迪倫·格雷澤 / 亞瑟·安其

沙贊是DC漫畫旗下一位人氣極高的超級英雄，

原漫裡他從小是個孤兒，以賣報為生。

後來他遇到了一個古老的巫師並賜予了他來自神話英雄的六種超能力：智慧、力量、毅力、能力、勇氣和速度。所以每當他高喊“Shazam(沙贊)，巴特遜就可以變身為超級英雄！

事到如今，只希望DC別把這張好牌打爛啊。

期待值★★★★★

8

《復仇者聯盟4》5月3日上映

導演: 安東尼·羅素 / 喬·羅素

主演: 喬什·布洛林 / 布麗·拉爾森 / 小羅伯特·唐尼 / 克里斯·海姆斯沃斯 / 克里斯·埃文斯 / 斯嘉麗·約翰遜 / 馬克·魯弗洛 / 塞巴斯蒂安·斯坦 / 傑瑞米·雷納 / 查德維克·博斯曼 / 保羅·路德 / 凱倫·吉蘭 / 唐·錢德爾 / 戴夫·巴蒂斯塔 / 伊萬傑琳·莉莉 / 格溫妮斯·帕特洛 / 喬恩·費儒 / 泰瑞·諾塔裡 / 泰·辛普金斯 / 真田廣之 / 斯坦·李

不多說，這部《復活者聯盟》只要國內同步上映，就讓我們看爆好嗎！

期待值★★★★★★★★★★

9

《疾速追殺3》5月17日上映

導演: 查德·斯塔赫斯基

主演: 基努·裡維斯 / 哈莉·貝瑞 / 約翰·雷吉扎莫 / 蘭斯·萊迪克 / 伊恩·麥柯肖恩

《疾速追殺3》的故事會承接第二部，續集中有人向裡維斯飾演的殺手開出了價值1,400萬美元的“追殺令”，因此他開始亡命天涯。

期待值★★★★

10

《阿拉丁真人版》5月24日上映

導演: 蓋·裡奇

主演: 梅納·瑪索德 / 娜奧米·斯科特 / 威爾·史密斯 / 馬爾萬·肯扎裡 /

作為迪斯尼一系列經典動畫真人翻拍計劃之一的《阿拉丁》由英國名導蓋·裡奇執導，故事還是那個故事，並且影片仍舊會保留一些1992年動畫版的經典音樂元素。

期待值★★★★★

11

《哥斯拉2》5月31日上映

導演: 邁克爾·道赫蒂

主演: 米莉·波比·布朗 / 維拉·法米加 / 凱爾·錢德勒 / 章子怡

作為傳奇影業怪獸宇宙的重要一環，《哥斯拉2》的故事承接《金剛：骷髏島》裡的“帝王計劃”展開。

並且本次哥斯拉的老對手拉頓、摩斯拉、三頭龍基多拉都可能登場。

值得一提的是，本次新加盟的演員還有章子怡以及《怪奇物語》小11。

期待值★★★★

12

《玩具總動員4》6月21日上映

導演: 喬什·庫雷

主演: 湯姆·漢克斯 / 瓊·庫薩克 / 蒂姆·艾倫 / 邦尼·亨特 / 安妮·波茨

時隔20年，玩具總動員系列終於迎來了第4部。

第四部會講述胡迪和牧羊女的愛情故事，而觀眾喜歡的巴斯光年以及玩具小夥伴們的生活也會翻開全新篇章。

期待值★★★★★

13

《蜘蛛俠2》7月5日上映

導演: 喬·沃茨

主演: 湯姆·赫蘭德 / 贊達亞 / 雅各布·巴特朗

別緊髒！《蜘蛛俠2》緊接《復聯4》劇情，小蜘蛛肯定是要復活的，只是沒有正式海報，先拿這個飯制的頂著

目前的得知的消息是，續集中傑克·吉林哈爾有望加入，並飾演大反派神祕客。而在第一部中有精彩表現的“禿鷲”邁克爾·基頓也將回歸。

嘻嘻，明年暑假又能吸荷蘭弟啦！

期待值★★★★★★

14

《壯志凌雲2》7月12日上映

導演: 約瑟夫·科辛斯基

主演: 湯姆·克魯斯 / 方·基默

1986年上映的《壯志凌雲1》是一部空戰題材的影片，同時也是湯姆·克魯斯的成名作。

33年後，飛行員Maverick迴歸。續集背景設定在現代，在無人機科技發達的世界中，飛行員老派的飛行方式變得極為過時，他們的地位也岌岌可危。

期待值★★★★★

15

《獅子王真人版》7月19日上映

導演: 喬恩·費儒

主演: 唐納德·格洛弗 / 碧昂絲·諾爾斯 / 詹姆斯·厄爾·瓊斯

《獅子王真人版》由拍過《奇幻森林》的導演喬恩·費儒操刀，並且該片貌也是由純CG特效完成製作，聽說還會採用VR技術。

畢竟是迪士尼的經典老IP，可以期待一下的！

期待值★★★★★

16

《新變種人》8月2日上映

導演: 約什·布恩

主演: 安雅·泰勒-喬伊 / 麥茜·威廉姆斯 / 艾莉絲·布拉加 / 查理·希頓

二十世紀福斯準備拓展“X戰警”系列，所以立項了《新變種人》

《新變種人》算是《X戰警》漫畫比較冷門的分支，角色都是X教授和萬磁王的學生，能力都比較特別。可以觀望一下。

期待值★★★★

17

《霍伯斯與肖》8月2日上映

導演: 大衛·雷奇

主演: 道恩·強森 / 傑森·斯坦森

這部電影其實就是《速度與激情外傳》

影片故事也將圍繞著盧克·霍布斯（道恩·強森）和迪卡·肖（傑森·斯坦森）展開

電影臨近上映應該會改回速激系列的名字，畢竟這個IP在國內的吸精能力是有目共睹的。

期待值★★★★

18

《好萊塢往事》8月9日上映

導演: 昆汀·塔倫蒂諾

主演：萊昂納多·迪卡普里奧、布拉德·皮特、阿爾·帕西諾、詹妮弗·勞倫斯、薩繆爾·傑克遜

《好萊塢往事》是昆汀·塔倫蒂諾的第九部電影作品，這部電影的亮點全體現在陣容上了，這個要看！

期待值★★★★★★

19

《小丑回魂2》9月6日上映

導演: 安德斯·穆斯切蒂

主演: 傑西卡·查斯坦 / 比爾·斯卡斯加德 / 詹姆斯·麥卡沃伊

《小丑回魂2》設定在第一部過去的27年後，片中的小演員都長大了。

目前已確認一美會扮演成年版的比爾，傑西卡·查斯坦扮演成年的貝弗莉，比爾·哈德爾扮演成年的Richie。當然，四弟將會繼續迴歸扮演小丑潘恩懷斯！

期待值★★★★

20

《神奇女俠2》11月1日上映

導演: 派蒂·傑金斯

主演: 蓋爾·加朵 / 克里斯汀·韋格 / 佩德羅·帕斯卡

派派復活了！昨天《神2》發佈官方片場照，在《神1》裡明明和飛機一起炸了的男主史蒂夫竟然又回到了片場

，迴歸應該是沒跑了。

而這次續集的故事背景將設置在上世紀80年代美蘇冷戰時期，戴安娜的對手也來自來自蘇聯。

婦女能頂半邊天真不是隨便說說的，明年第2期待的超級英雄片，看起來！

期待值★★★★★★

21

《冰雪奇緣2》11月27日上映

導演: 克里斯·巴克 / 珍妮弗·李

主演: 伊迪娜·門澤爾 / 克里斯汀·貝爾 / 喬什·加德

《冰雪奇緣2：家人的思念》講述了在長期關閉的城門再度開啟後，安娜和艾麗莎一同度過的第一個聖誕節的故事。

而我更期待的是，這次將會有比經典的《Let it go》更棒的插曲出現！

期待值★★★★★

22

《星球大戰9》12月20日上映

導演: J·J·艾布拉姆斯

主演: 黛西·雷德利 / 亞當·德賴弗 / 奧斯卡·伊薩克 / 約翰·博耶加

據外媒NY Daily報道，傳奇角色尤達大師將回歸到《星球大戰9》和之前一樣，他將繼續以原力魂魄導師的形式出現。不多說，明年票房大牛應該就是星戰8和復聯4拼了，迪士尼真是笑開了花。

期待值★★★★★

明年最值得期待的22部大片都在這了，

光是超級英雄片就有8部！而動畫也是明年的重頭戲。

可以預見，未來的北美票房年冠，迪士尼又或成為最大贏家。
