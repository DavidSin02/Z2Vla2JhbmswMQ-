A、自敘帖 B 、自言帖 C 、中秋帖 D 、黃州寒食帖

A、史記 B 、漢書 C 、三國志 D 、資治通鑑

A、李商隱 B 、李賀 C 、李白 D 、李煜

Ａ、周 B 、元 Ｃ、商 D 、夏

A、永濟 B 、桑落 C 、汾陽 D 、綿竹

A、孟子 B 、韓非子 C 、荀子 D 、老子

A、三隻腳 B 、兩隻腳 C 、四隻腳 D 、五隻腳

A、嵇康 B 、曹植 C 、山濤 D 、阮瑀

C、折腰曾愧五斗米，負郭元無三頃田。 D 、京華結交盡奇士，意氣相期共生死。

A、月老 B 、千金小姐 C 、新郎 D 、岳父
