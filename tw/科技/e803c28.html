面試官問我什麼是「棧」，我隨手畫了10張圖來解釋

棧的概念

棧（stack）是限定僅在表的一端進行操作的數據結構，且棧是一種先進後出的數據結構，允許操作的一端稱為棧頂，不允許操作的稱為棧底，如下圖所示：

之前我們講到了鏈表，我們只能夠對其鏈表的表尾結點進行操作，並且只能進行插入一個新的結點與刪除最末尾的這個結點兩個操作，而這樣強限制性的‘鏈表’，就是我們所說的棧。

就像是一個死衚衕一樣，只有一個出口，如圖所示，有個概念：

棧的結點設計

棧分為數組棧和鏈表棧，其區別如下：

  • 數組棧使用數組進行功能的模擬，實現較為快速和便利；
  • 鏈表棧使用鏈表的思路去設計，實現相比較說較為麻煩，但是其穩定，且不易出錯；

在鏈表棧中又分為靜態鏈表棧和動態鏈表棧，其區別如下：

  • 靜態鏈表棧給定棧的空間大小，不允許超過存儲超過給定數據大小的元素；
  • 動態棧使用的是自動創建空間的方法進行創建，只要符合機器的硬件要求以及編譯器的控制，其理論上是極大的。

數組棧

其實際就是用一段連續的存儲空間來存儲棧中的數據元素，有以下特點：

  1. 元素所佔的存儲空間必須連續，這裡的連續是指的邏輯連續，而不是物理連續。
  2. 元素在存儲空間的位置是按邏輯順序存放的

我們來舉例說明，鑑於C語言數組下標都是0開始，並且棧的使用需要的空間大小難以估計，所以初始化空棧的時候，不應該設定棧的最大容量。

我們先為棧設定一個基本容量，在應用過STACK_程當中，當棧的空間不夠用時，再逐漸擴大。

設定2個常量，STACK_INIT_SIZE(存儲空間初始化分配量)和STACK_INCREMENT(存儲空間分配增量),宏定義如下

#define STACK_INIT_SIZE 1000 //數值可以根據實際情況確定#define STACK_INCREMENT 10   //數值可以根據實際情況確定

棧的定義如下

typedef struct      {    void *base;     void *top;    int stackSize;} SqSTACK;
  • base 表示棧底指針
  • top 表示棧頂指針
  • stackSize 表示棧當前可以使用的最大容量

若base的值是NULL，表示棧結構不存在;top初始值指向棧底，即top = base;

每當插入新的元素時，指針top就增1，反之刪除就減1，非空棧中的棧頂指針始終在棧頂元素的下一個指針上面。

鏈表棧

我們以鏈表棧的動態鏈表棧為例子，進行棧的設計。

首先是棧的結點，設計出兩個結構體，一個結構體Node表示結點，其中包含有一個data域和next指針，如圖所示：

Node

其中data表示數據，next指針表示下一個的指針，其指向下一個結點，通過next指針將各個結點鏈接。

接下來是我們設計的重點，為這個進行限制性的設計，我們需要額外添加一個結構體，其包括了一個永遠指向棧頭的指針top和一個計數器count記錄元素個數。

其主要功效就是設定允許操作元素的指針以及確定棧何時為空，如圖所示：

Stack

這裡我採用的是top和count組合的方法。其代碼可以表示為：

//棧的結點設計//單個結點設計，數據和下一個指針typedef struct node     {    int data;     struct node *next;} Node;

利用上面的結點創建棧，分為指向頭結點的top指針和計數用的count

typedef struct stack    {    Node *top;    int count;} Link_Stack;

棧的基本操作—入棧（壓棧）

入棧的基本順序可以用以下圖所示：

入棧(push)操作時，我們只需要找到top所指向的空間，創建一個新的結點，將新的結點的next指針指向top指針指向的空間，再將top指針轉移，並且指向新的結點，這就是是入棧操作。

其代碼可以表示為：

//入棧 pushLink_Stack *Push_stack(Link_Stack *p, int elem){    if (p == NULL)        return NULL;    Node *temp;    temp=(Node*)malloc(sizeof(Node));    //temp = new Node;    temp->data = elem;    temp->next = p->top;    p->top = temp;    p->count++;    return p;}

棧的基本操作—出棧

出棧（pop）操作，是在棧不為空的情況下，重複說一次，一定要進行判空操作，將棧頂的元素刪除，同時top指針，next向下進行移動即可的操作。

其代碼可以表示為：

//出棧 popLink_Stack *Pop_stack(Link_Stack *p){    Node *temp;    temp = p->top;    if (p->top == NULL)    {        printf("錯誤：棧為空");        return p;    }    else    {        p->top = p->top->next;        free(temp);        //delete temp;        p->count--;        return p;    }}

棧的基本操作—遍歷

這個就很常見了，也是我們調試必須的手段。

棧的遍歷相對而言比較複雜，由於棧的特殊性質，其只允許在一端進行操作，所以遍歷操作操作永遠都是逆序的。

簡單一點描述，其過程為，在棧不為空的情況下，一次從棧頂元素向下訪問，直到指針指向空（即到棧尾）為結束。

其代碼可以表示為：

//遍歷棧：輸出棧中所有元素int show_stack(Link_Stack *p){    Node *temp;    temp = p->top;    if (p->top == NULL)    {        printf("");        printf("錯誤：棧為空");        return 0;    }    while (temp != NULL)    {        printf("%d\t", temp->data);        temp = temp->next;    }    printf("\n");    return 0;}

棧數組與棧鏈表的代碼實現

最後呢，我們使用代碼來幫助我們瞭解一下：

棧數組

數組棧是一種更為快速的模擬實現棧的方法，這裡我們不多說。

模擬，就是不採用真實的鏈表設計，轉而採用數組的方式進行模擬操作。

也就是說這是一種仿真類型的操作，其可以快速的幫助我們構建代碼，分析過程，相應的實現起來也更加的便捷。

其代碼如下:

#include<stdio.h>#include<stdlib.h>#include<string.h>#define maxn 10000 //結點設計typedef struct stack{    int data[maxn];    int top;}stack; //創建stack *init(){    stack *s=(stack *)malloc(sizeof(stack));    if(s==NULL){        printf("分配內存空間失敗");        exit(0);    }    memset(s->data,0,sizeof(s->data));    //memset操作來自於庫文件string.h，其表示將整個空間進行初始化    //不理解可以查閱百度百科https://baike.baidu.com/item/memset/4747579?fr=aladdin    s->top=0;     //棧的top和bottom均為0（表示為空）    return s;} //入棧pushvoid push(stack *s,int data){    s->data[s->top]=data;    s->top++;} //出棧popvoid pop(stack *s){    if(s->top!=0){        s->data[s->top]=0;  //讓其迴歸0模擬表示未初始化即可        s->top--;    }} //模擬打印棧中元素void print_stack(stack *s){    for(int n=s->top-1;n>=0;n--){        printf("%d\t",s->data[n]);    }    printf("\n");   //習慣性換行} int main(){    stack *s=init();    int input[5]={11,22,33,44,55};  //模擬五個輸入數據    for(int i=0;i<5;i++){        push(s,input[i]);    }    print_stack(s);    /////////////    pop(s);    print_stack(s);    return 0;}

棧鏈表

#include <stdio.h>#include <stdlib.h>//棧的結點設計//單個結點設計，數據和下一個指針typedef struct node     {    int data;     struct node *next;} Node;//利用上面的結點創建棧，分為指向頭結點的top指針和計數用的counttypedef struct stack    {    Node *top;    int count;} Link_Stack; //創建棧Link_Stack *Creat_stack(){    Link_Stack *p;    //p = new Link_Stack;    p=(Link_Stack*)malloc(sizeof(Link_Stack));    if(p==NULL){        printf("創建失敗，即將退出程序");        exit(0);    } else {printf("創建成功\n"); }    p->count = 0;    p->top = NULL;    return p;} //入棧 pushLink_Stack *Push_stack(Link_Stack *p, int elem){    if (p == NULL)        return NULL;    Node *temp;    temp=(Node*)malloc(sizeof(Node));    //temp = new Node;    temp->data = elem;    temp->next = p->top;    p->top = temp;    p->count++;    return p;} //出棧 popLink_Stack *Pop_stack(Link_Stack *p){    Node *temp;    temp = p->top;    if (p->top == NULL)    {        printf("錯誤：棧為空");        return p;    }    else    {   printf("\npop success");        p->top = p->top->next;        free(temp);        //delete temp;        p->count--;        return p;    }} //遍歷棧：輸出棧中所有元素int show_stack(Link_Stack *p){    Node *temp;    temp = p->top;    if (p->top == NULL)    {        printf("");        printf("錯誤：棧為空");        return 0;    }    while (temp != NULL)    {        printf("%d\t", temp->data);        temp = temp->next;    }    printf("\n");    return 0;} int main(){ //用主函數測試一下功能 int i;    Link_Stack *p;    p = Creat_stack();    int n = 5;    int input[6] = {10,20,30,40,50,60};    /////////////以依次入棧的方式創建整個棧//////////////    for(i=0;i<n;i++){        Push_stack(p, input[i]);    }    show_stack(p);    ////////////////////出棧///////////////////////    Pop_stack(p);    show_stack(p);    return 0;}

關於棧的總結

棧-它是一種運算受限的線性表，在數制轉換，括號匹配的檢驗，表達式求值等方面都可以使用，並且較為簡便的解決問題。
