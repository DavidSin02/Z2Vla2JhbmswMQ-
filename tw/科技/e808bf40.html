C++｜用指針、引用作為函數的輸入（形參初始化為實參）、輸出

2020-11-14
| 科技

函數的形參可以是指針、引用或普通變量，相應的，函數調用時要用指針值、引用值、普通變量值將形參要初始化化為實參。

函數返回值的基本原則：

I 不要返回局部自動變量的地址；II 可以返回動態創建的堆內存的變量；III 可以返回局部靜態變量的地址；

直接看實例：

#include <iostream>using std::cout;using std::endl;double* treble(double); // Function prototypeint main(){	double num = 5.0; // Test value	double* ptr = NULL; // Pointer to returned value		ptr = treble(num);		cout << endl << "Three times num = " << 3.0*num;		cout << endl << "Result = " << *ptr; // Display 3*num	delete ptr;	cout << endl;	std::cin.get();	return 0;}// Function to treble a value - mark 1double* treble(double data){	//double result = 0;	//不要返回局部自動變量的地址	//static double result = 0;	//可以返回局部靜態變量的地址	//result = 3.0*data;	//return &result;	double* result = new double; //可以返回動態創建的堆內存的變量	*result = 3.0*data;	return result;}

output:

Three times num = 15Result = 15

可以從函數中返回一個左值引用。

// Returning a reference#include <iostream>#include <iomanip>using std::cout;using std::endl;using std::setw; double& lowest(double values[], int length); // Function prototype int main(){ double data[] = { 3.0, 10.0, 1.5, 15.0, 2.7, 23.0, 4.5, 12.0, 6.8, 13.5, 2.1, 14.0 }; int len = 12 ; // Number of elements  for(int i=0; i<len; i++) cout << setw(6) << data[i];  lowest(data, len) = 6.9; // Change lowest to 6.9 lowest(data, len) = 7.9; // Change lowest to 7.9  cout << endl; for(i=0; i<len; i++) cout << setw(6) << data[i];  cout << endl; std::cin.get(); return 0;} // Function returning a referencedouble& lowest(double a[], int len){ int j = 0; // Index of lowest element for (int i =1; i < len; i++) if(a[j] > a[i]) // Test for a lower value... j = i; // ...if so update j return a[j]; // Return reference to lowest element}

輸出：

 3 10 1.5 15 2.7 23 4.5 12 6.8 13.5 2.1 14 3 10 6.9 15 2.7 23 4.5 12 6.8 13.5 7.9 14
