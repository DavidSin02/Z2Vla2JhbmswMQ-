通過Excel VBA實現對Word中文本型窗體信息提取

2020-10-29
| 科技

工作中需要對一個模板Word進行批量輸入信息，但Word是個受保護的文檔，並且內部有很多文本型窗體域。直接用vbs替換比較困難。

這裡補充下文本型窗體域是什麼：

窗體域是用來給人填寫調查之用。

下拉式窗體是隻能選擇預置的答案，

選擇式窗體者只能選擇是或否，用開關來表示。

文字型窗體，只能輸入指定文字長度的字符。當窗體鎖定後。只有窗體可以操作，文字內容沒有辦法更改。

就開始折騰怎麼能做到自動填充，準備在Excel上做一個宏。

第一步就是確定這些文本型窗體域的類型情況。

Sub 按鈕1_Click()        Dim wdTextFormNum    wordPath = Cells(1, "j")        Set objword = GetObject(, "Word.Application")    objword.Visible = True    MsgBox (objword.ActiveDocument .Fields.Count)    For i = 1 To objword.activedocument.Fields.Count        Cells(i + 1, "j") = objword.ActiveDocument.Fields(i).Index        Cells(i + 1, "K") = objword.ActiveDocument.Fields(i).result        Cells(i + 1, "L") = objword.ActiveDocument.Fields(i).Type    Next   End Sub

通過上述代碼可以把窗體域的情況填寫出來，方便進一步分析。

創建宏，輸入上述代碼，執行宏或者點擊按鈕就可以輸出如下內容。
