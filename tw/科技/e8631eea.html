Vue3 開啟Gzip壓縮打包，使你速度更快

2020-11-14
| 科技

vue3打包文件有些文件過大，導致加載頁面時間久，體驗感不佳，可在打包時使用Gzip壓縮，將一些稍大的文件進行壓縮，從而優化加載速度。


vue


gizp壓縮是一種http請求優化方式，通過減少文件體積來提高加載速度。html、js、css文件甚至json數據都可以用它壓縮，減少文件大小。減少文件大小有兩個明顯的好處，一是可以減少存儲空間，二是通過網絡傳輸文件時，可以減少傳輸的時間。gzip 是在 Linux 系統中經常使用的一個對文件進行壓縮和解壓縮的命令，既方便又好用。

vue可以通過安裝 compression-webpack-plugin 來實現gzip壓縮，配置很容易。

  1. 安裝插件
npm i --save-dev compression-webpack-plugin
  1. 在 vue.config.js 配置gzip壓縮
const CompressionWebpackPlugin = require('compression-webpack-plugin');module.exports = {         // ...         configureWebpack: config => {           config.externals = {                     AMap: "AMap", // 高德地圖配置            }               if (process.env.NODE_ENV === 'production') {                    return {                        plugins: [new CompressionWebpackPlugin({                        test: /\.js$|\.html$|\.css/, // 匹配文件名                        threshold: 10240, // 對超過10k的數據進行壓縮                        deleteOriginalAssets: false, //是否刪除原文件                    })]               }            // ... }
  1. 配置Nginx
 server {           listen       8080;           server_name  demo;               #開啟或關閉gzip on off           gzip on;   #gzip壓縮最小文件大小，超出進行壓縮（自行調節）           gzip_min_length 5k;            gzip_buffers 4 16k; #buffer 不用修改           gzip_comp_level 8; #壓縮級別:1-10，數字越大壓縮的越好，時間也越長           gzip_types text/plain application/json application/x-javascript application/javascript text/css application/xml text/javascript application/x-httpd-php image/jpeg image/gif image/png; #  壓縮文件類型            gzip_vary on; # 和http頭有關係，加個vary頭，給代理服務器用的，有的瀏覽器支持壓縮，有的不支持，所以避免浪費不支持的也壓縮，所以根據客戶端的HTTP頭來判斷，是否需要壓縮           location / {            root   /demo/dist;            index  /index.html;           }         }
  1. 如果是部署到tomcat裡，配置tomcat也很簡單
<Connector 	connectionTimeout="20000" 	enableLookups="false" 	port="80" 	protocol="HTTP/1.1" 	redirectPort="8443" 	useBodyEncodingForURI="false"	compressableMimeType="text/html,text/xml,text/plain,text/javascript,text/csv,application/json,application/xml,application/x-javascript,application/javascript,text/css"	URIEncoding="utf-8"/>
