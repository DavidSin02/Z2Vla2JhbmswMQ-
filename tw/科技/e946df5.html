「C與指針心得」4.左移右移操作

2020-10-29
| 科技

左移操作

操作符： <<

含義：

顧名思義就是把一個數的所有位都向左移動若干位，在C用用<<運算符標記

例：

int i = 1;i = i << 2;

說明：1的二進制為00000001(這裡用8位標示，不同系統，不同的位數)，向左移兩位，即把1向左移動兩位，後面補兩個0，即 00000100,轉換成10進制就變成了4

需要注意的是，int類型為有符號的類型，最左端的1位是符號位,即0正1負,那麼移位的時候就會出現溢出

例：

int i = 0x40000000; //二進制位01000000000000000000000000000000i = i << 1;

說明：i左移1位後變為0x80000000，符號位被置為了1，其它位全是0，變成了int所能表達的最小值，32位的int為-2147483648，溢出。

如果此時i再左移1位後，丟棄了最高為後，i變成了0

右移操作

操作符: >>

含義：

顧名思義就是把一個數的所有位都向右移動若干位，在C用用>>運算符標記

右移分為邏輯右移和算術右移。

邏輯右移：

方法：

左邊移入的位用0填充

int i = 150;//二進制為10010110i = i>>2;

i向右移動兩位，則最後面兩位移除，最開始的添加兩個0，變成00100101，十進制為37

算術右移：

方法：

左邊移入的位由原先該值的符號位決定，符號位為1則移入的位均為1，符號位為0，則移入的位都為0，這樣能夠保持原數的正負數形式不變。

int i = 150;//二進制為10010110i = i>>2;

i向右移動兩位，原來的符號位即首位為1，即最後兩位移除，最開始添加兩個符號位，這裡為1，即變成11100101，十進制為229

總結：

左移時總是移位和補零。右移時無符號數是移位和補零，此時稱為邏輯右移;

而有符號數大多數情況下是移位和補最左邊的位（也就是補最高有效位），移幾位就補幾位,此時稱為算術右移
