一款炫酷的消息提示庫ALERTIFY.js

2020-10-29
| 科技

消息提示是與用戶進行交互時必不可少的組件，看膩了瀏覽器自帶的消息提示框以及bootstrap原生的alert組件，可以嘗試一下alertify.js這個前端庫，他能給你帶來耳目一新的感覺。

alertify.js官網地址：http://fabien-d.github.io/alertify.js/

github地址：https://github.com/fabien-d/alertify.js


默認彈出提示框效果

使用方法：

引入js

<script src="PATH_TO_FILE/alertify.min.js"></script>

引入css

<!-- include the core styles --><link rel="stylesheet" href="PATH_TO_FILE/alertify.css" /><!-- include a theme, can be included into the core instead of 2 separate files --><link rel="stylesheet" href="PATH_TO_FILE/alertify.default.css" />

使用默認彈出提示框

// alert dialogalertify.alert("Message");

使用默認消息提示框（消息框在右下角彈出）

// standard notification// setting the wait property to 0 will// keep the log message until it's clickedalertify.log("Notification", type, wait);

alertify.js的最大好處是可以輕鬆的自定義外觀，可以根據自己項目的配色修改相應的css，從而與 您的項目融為一體。有興趣的朋友可以加個關注，我會不斷的介紹各種有趣好玩的前端類庫，希望您能在項目中用到。
