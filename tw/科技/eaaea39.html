Excel函數(16)–lookup函數按區間查找

2020-10-29
| 科技

我發現一個有趣的現象：很多小夥伴對 vlookup 函數不陌生，平時工作中經常使用，而對 lookup 函數卻不太熟悉。

那我們今天就來說一說 lookup 函數的常用場景。

語法：

  • lookup(lookup_value,lookup_vector,result_vector)

釋義：

  • lookup（查找值，查找向量，返回向量）

應用：

  1. 區間查找
  2. 符合最後一個條件查找
  3. 多條件查找

案例 1：區間查找

下表 A 列是每場活動的實際成本，B，C 列是每個成本區間對應的係數規則，如何根據規則，快速算出 A 列每個成本的係數？

是不是要套用很多 if 函數？Oh，no no no！其實很簡單。

解決方案 1：

1. 如下圖，增加輔助列 C，填入每個成本區間的起始值

2. 在結果列輸入以下公式，向下拖動，就完成了：

=LOOKUP(A2,$C$2:$D$8)

* 請注意，$C$2:$D$8 要絕對引用

案例 2：符合最後一個條件查找

下表中的人名若有重複，則統計其最新的獎金數。

解決方案 2：

1. 在 D2 中輸入如下公式，下拉即可：

=LOOKUP(1,0/($A$2:$A$13=$A2),$B$2:$B$13)

公式釋義：

  • $A$2:$A$13=$A2：判斷 $A$2:$A$13 中的值是否等於 $A2，分別返回 true 或 false
  • 0/($A$2:$A$13=$A2)：用 0 除以 true 或 false，結果為 0 或錯誤值 #DIV/0!
  • 1,0/($A$2:$A$13=$A2)：在結果為 0 或錯誤值 #DIV/0! 的數組中查找 1 的值，查到後返回，忽略錯誤值後區域內的值全是 0，所以函數一直向下查找，直至查找到最後一個 0 的值
  • $B$2:$B$13：按最後查到的 0 值，返回對應的 $B$2:$B$13 單元格

案例 3：多條件查找

下表中多了一個“月份”列，如何查出郭德綱 2 月的獎金數？

解決方案 3：

1. 在 結果單元格中輸入以下公式即可：

=LOOKUP(1,0/(($F$2=$A$2:$A$13)*($G$2=$C$2:$C$13)),$B$2:$B$13)

公式釋義：

  • 0 和 1 的使用目的跟案例 2 類似，不難理解
  • *：表示 and，因為需要同時滿足兩個條件，所以用 * 把它們連接起來
