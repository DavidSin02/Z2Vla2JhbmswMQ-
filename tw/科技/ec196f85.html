java基礎形參實參 沒事看看

2020-11-14
| 科技

形參

定義：全稱"形式參數"，用於定義方法的時候使用的參數，目的用來接收調用該方法時傳遞的參數。

說明：只有在被調用時才會分配內存單元，在調用結束，即刻釋放所分配的內存單元。因此，只在方法內才有效。

舉個例子：

public void printInfo(String A){

System.out.println(A);

}

此處A就是一個形參，它是String類型的。

實參

定義：全稱"實際參數"，用於調用時傳遞給方法的參數，即傳遞給被調用方法的值。

說明：預先創建並賦予確定值。

舉個例子：

形參在方法頭定義中用到。比如計算兩數和的方法

public double getSum(double a,double b)//這裡的a,b就是形參

{

return a+b;

}

實參是調用方法時實際傳入的參數，比如調用上述方法

new Xxx().getSum(4,8);//這裡的4，8就是實參

再簡單明瞭的說一下：

形參就是沒有賦值的變量，實參就是有賦值的變量！
