文件自動生成1~100的30個隨機數到數列中，並對數列從小到大排序

2020-10-29
| 科技

1.理清思路

（1）先輸出1~100的30個隨機數

（2）排序

2.掌握基礎知識

（1）交換變量

（2）隨機數的輸出

(3)數組

（4）嵌套循環

（5）冒泡排序法

3.代碼

 public static void main(String[] args) { int[]a=new int[30]; //定義數組 for(int i=0;i<a.length;i++) { a[i]=(int) (Math.random()*100+1); //限制隨機數的範圍為1~100  System.out.println("隨機數為："); } for(int random:a) { //random:a指的是把隨機產生的數都放進數組中（把對隨機數大小的限制與數組長度的限制兩個限制條件結合在一起） System.out.println(random+"");  } int b; for(int j=0;j<a.length-1;j++) { //兩個for循環嵌套在一起 for(int k=0;k<a.length-(j+1);k++) { if(a[k]>a[k+1]) { b=a[k]; //交換變量 a[k]=a[k+1]; a[k+1]=b; //a[k]先輸出a[k+1]後輸出（完成排序） }  } } System.out.println("\r排序結果為："); for(int random:a) { System.out.println(random+""); } }}## 輸出結果為隨機數為：78 97 57 30 100 96 7 62 39 18 18 32 56 37 32 29 35 56 18 91 7 35 20 16 24 26 75 40 80 58 排序結果為：7 7 16 18 18 18 20 24 26 29 30 32 32 35 35 37 39 40 56 56 57 58 62 75 78 80 91 96 97 100 ``
