對偶是中國詩詞和對聯創作的常用手法，講究文句兩兩相對、字數相等、句法相似、平仄相對、意義相關。

例如：春對夏，秋對冬，暮鼓對晨鐘；雲對雨，雪對風，晚照對晴空。這樣的句子，音韻和諧、朗朗上口，容易記憶！

[ 寫人記事 ]

驛寄梅花，魚傳尺素。

物華天寶，人傑地靈。

鳥宿池邊樹，僧敲月下門。

久旱逢甘雨，他鄉遇故知。

竹喧歸浣女，蓮動下漁舟。

浮雲遊子意，落日故人情。

三杯竹葉穿心過，兩朵桃花上臉來。

遊子魂銷青塞月，美人腸斷翠樓煙。

莊生曉夢迷蝴蝶，望帝春心託杜鵑。

日晚愛行深竹裡，月明多上小橋頭。

[ 描景狀物 ]

桃灼灼，柳依依。

雪茫茫，雨霏霏。

山抹綠，燕剪柳。

霧失樓臺，月迷津渡。

亂石穿空，驚濤拍岸。

千里冰封，萬里雪飄。

明月隱高樹，長河沒曉天。

秋風吹渭水，落葉滿長安。

蟬噪林愈靜，鳥鳴山更幽。

遲日江山麗，春風花鳥香。

疏影橫斜水清淺，暗香浮動月黃昏。

池上碧苔三四點，葉底黃鸝一兩聲。

漠漠水田飛白鷺，陽陽夏木囀黃鸝。

[ 哲理修養 ]

滿招損，謙受益。

瓜熟蒂落，水到渠成。

偏聽則暗，兼聽則明。

近朱者赤，近墨者黑。

濁者自濁，清者自清。

金玉其外，敗絮其中。

一芽知春，一葉知秋。

疾風知勁草，烈火見真金。

草枯鷹眼疾，雪盡馬蹄輕。

花殘無戲蝶，藻密有潛魚。

花香風氣暖，雲淡日光新。

欲窮千里目，更上一層樓。

[ 言志抒懷 ]

不以物喜，不以己悲。

生當作人傑，死亦為鬼雄。

吟懷山川趣，觀感古今文。

放懷形骸以外，浪跡山水之間。

騁此長風快浪，欣然臨水觀山。

出淤泥而不染，濯清漣而不妖。

朝飲木蘭之墮露，夕餐秋菊之落英。

十里水光心地朗，一林花色性天空。

心志光明同日月，功勳蓋世重山河。

關山難越誰悲失路之人，

萍水相逢盡是他鄉之客。

[ 讀書風雅 ]

勝不驕，敗莫餒。

苦讀有恆，好學無時。

欲知千古事，須讀五車書。

揮毫列錦繡，落紙如雲煙。

藏古今學術，聚天地精華。

雪夜書千卷，花時酒一瓢。

白日莫閒過，青春不再來。

自有高明將學講，且教童子抱詩來。

無盡波濤歸學海，長春花木在詞林。

盡日相親惟有石，長年可槳莫如書。

銛鋒梳盡山中兔，蘸墨驚飛海上魚。

[ 時令佳節 ]

三月昏，參星夕。

杏花盛，桑葉白。

霜菊瘦，雨梅肥。

梅英盡落，柳眼初開。

酷寒盛暑，暮煙曉霧。

柳搖春白晝，梅弄月黃昏。

蟬鳴哀暮夏，鶯囀怨殘春。

落日街翠壁，暮雲佔煙鬟。

昔去雪似花，今來花如雪。

春風掩映千門柳，暖雨晴開一徑花。

沾衣欲溼杏花雨，吹面不寒楊柳風。

柳絮池塘春入夢，朵花庭院冷侵衣。

[ 風月情濃 ]

連理枝，並蒂蓮。

柔情似水，佳期如夢。

桃臉含秋，峨眉凝翠。

錦上添花，雪中送炭。

雙眸翦秋水，十指剝春筍。

窗前鶯並語，簾外燕雙飛。

月上柳梢頭，人約黃昏後。

泥融飛燕子，沙暖睡鴛鴦。

夜月一簾幽夢，春風十里柔情。

煙開蘭葉香風起，春入桃花暖氣勻。

花開並蒂鴛鴦暖，酒醉同心琥珀濃。

碧紗待月春調瑟，紅袖添香夜讀書。

染就江南春水色，結成羅帳連心花。

細雨夢迴雞塞遠，小樓吹砌玉生寒

[ 喜怒哀樂 ]

綠窗明月在，青史古人空。

淚隨流水急，愁逐野雲飛。

感時花濺淚，恨別鳥驚心。

白雲懸影望，鳥鳥切遐思。

窗竹鳴秋雨，床琴斷夜弦。

玉梅含笑意，金柳動哀情。

蝴蝶夢中家萬里，杜鵑枝上月三更。

雨中竹葉含珠淚，雪裡梅花載素冠。

勳業輝煌欣自舉，名花燦爛喜連枝。

百日星沉光不熄，一朝風起雨非輕。

[ 風趣軼聞 ]

水底月，鏡中花。

瓜熟蒂落，藕斷絲連。

雙手推開窗前月，一石擊破水中天。

鬆聲竹聲鐘磬聲聲聲自應，

山色水色煙霞色色色皆空。

佳水佳山佳風佳月千秋佳地，

痴色痴聲痴情痴夢幾輩痴人。

鶯鶯燕燕翠翠紅紅處處融融洽洽，

雨雨風風花花草草年年暮暮朝朝。

合時黃鶴重來且自把金樽看州渚十年芳草，

今日白雲尚在問誰吹玉笛落江城五月梅花。
