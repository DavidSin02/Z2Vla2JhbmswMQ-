開源C ++庫的綜合列表：

1、通用

  • Boost - 大量通用庫
  • Dlib - 網絡、線程、圖形界面、數據結構、線性代數、機器學習、XML和文本解析、數值優化、貝葉斯網絡等（Boost許可證）
  • JUCE - 廣泛、成熟、跨平臺的C ++工具包（GPL許可證）
  • Folly - Facebook開源庫
  • Abseil - 來自Google C ++庫代碼的開源集合
  • cxxomfort - C ++的Backports功能
  • Ultimate++ - 跨平臺的快速應用程序開發框架
  • CAF
  • Better Enums - 具有constexpr支持的反射枚舉
  • Smart Enum - “to_string”，“from_string”以及更多內容

2、通訊

  • libnavajo - 輕巧但功能強大的API，包括用於Web和RESTful應用程序開發的http服務器，支持SSL、身份驗證、cookie和會話管理，包括Websocket服務器支持
  • C++ RESTful framework - C ++微框架，旨在嵌入到廣泛的應用程序中
  • CPR - 現代C ++ HTTP請求庫
  • cpp-netlib - C ++網絡庫
  • cpp-redis - C ++ 11 Lightweight Redis客戶端
  • tacopie - C ++ 11 TCP庫
  • Boost.Beast - 建立在Boost.Asio之上的HTTP和WebSocket庫
  • gsoap - 用於XML數據綁定的C / C ++開發工具包
  • omniORB - C ++中最快，最完整，最便攜的CORBA ORB實現
  • Unicomm - 異步網絡，高級TCP通信框架
  • zeromq - 快速消息隊列
  • libashttp - 異步HTTP客戶端庫
  • The Silicon C++14 Web Framework - 快速而強大的Web API（MIT）

3、圖形用戶界面

  • Dear ImGui
  • FLTK
  • nana[doc] - 現代C ++風格的GUI編程
  • OWLNext
  • Switch

GTK+

  • glibmm
  • gtkmm
  • goocanvasmm
  • libglademm
  • libgnomecanvasmm
  • webkitgtk
  • flowcanvas
  • evince

Qt

  • Qt [ src ] [ doc ]
  • qwt5
  • libdbusmenu-QT
  • QuickQanava - 用於在Qt / QML應用程序中繪製有向圖的C ++ 14庫
  • QuickProperties - 用於QObject / QtObject屬性版的C ++ / QML庫

4、通用多媒體

  • SFM - 簡單快速多媒體庫
  • SDL - 通過圖形API（通常為OpenGL或Direct3D）跨平臺訪問音頻、鍵盤、鼠標、操縱桿和圖形硬件
  • SIGIL - 用於文本、形狀、輸入、音頻和2D圖像的跨平臺庫
  • Cinder
  • openFrameworks

5、圖像

  • nux
  • pangomm
  • gegl

  • Adobe/boost GIL - 高級通用庫，專注於對2D圖像進行操作的算法
  • GraphicsMagick - 以超過88種格式讀取、書寫和操作圖像。
  • Skia Graphics Engine - Google的核心圖形庫

  • Aquila - 跨平臺DSP庫

  • GLEW - OpenGL函數加載
  • GLAD - 可自定義的輕量級庫，用於加載OpenGL函數
