前篇講述了學習PHP需要的前置知識基礎，這篇主要講述PHP的介紹和硬件基礎。

PHP是什麼？PHP可以幹什麼？

  • PHP 是一種通用開源腳本語言，默認文件擴展名是 ".php"，代碼在服務器上執行，結果以純 HTML 形式返回給瀏覽器。
  • PHP 可以生成動態頁面內容；可以創建、打開、讀取、寫入、關閉服務器上的文件；可以收集表單數據；可以發送和接收 cookies；可以添加、刪除、修改您的數據庫中的數據；可以限制用戶訪問您的網站上的一些頁面；可以加密數據......
  • PHP 可在不同的平臺上運行（Windows、Linux、Unix、Mac OS X 等），目前幾乎所有的正在被使用的服務器相兼容（Apache、IIS 等）

因為PHP代碼是在服務器上執行的，所以在開始學習之前，學者需要先擁有一臺支持 PHP 和 MySQL 的 Web 主機，可以到相關平臺去購買使用已有配置的服務器，或者是在自己的電腦上安裝 Web 服務器，然後安裝 PHP 和 數據庫等。

安裝PHP 服務器組件

對於初學者建議使用集成的服務器組件，它已經包含了 PHP、Apache、Mysql 等服務,免去了開發人員將時間花費在繁瑣的配置環境過程。WampServer或者是XAMPP都可以，筆者使用的是XAMPP集成。

XAMPP

XAMPP

XAMPP是一款免費的軟件，支持Linux、Mac OS 和 Windows 系統，集成了Apache+MYSQL+PHP的服務器系統開發套件，同時還包含了管理MySQL的工具phpMyAdmin，即可對MySQL進行可視化操作。採用這種緊密的集成，XAMPP 可以運行任何程序：從個人主頁到功能全面的產品站點（雖然這僅僅用於開發目的；出於安全考慮，XAMPP 還不適於用在產品服務器上）。

具體的安裝步驟網上有很多，自行搜索並安裝即可，這裡筆者不再贅述。
