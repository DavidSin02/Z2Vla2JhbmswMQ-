WinCC 的輸入確認對話框製作方法

2020-11-14
| 科技

有時候客戶會要求在做wincc重要的數值輸入時需要進行彈窗確認，這樣會比較安全。

思路大概就是給輸入輸出域的輸入和輸出屬性分別連接一個變量，判斷輸入值變化時彈出對話框，讓操作員確認。如果操作員確認，則將輸入值賦值給輸出值，反之則什麼都不做。

這樣做法的好處是：1、操作員界面直觀，2、輸出值是真正的過程參數，如果操作員不確認，這個變量不會受到任何干擾。

下面就來講解制作方法。

1.首先在變量的內部變量中建立一個臨時變量和一個plc的寄存器變量。然後畫面中製作輸入輸出域，在屬性中的輸入屬性中選擇新建的臨時變量。在輸出屬性中選擇PLC的寄存器變量。




由於我的軟件的輸入輸出屬性是相反的所以我選的變量是相反的。正常應該是輸入選臨時變量，輸出選寄存器變量。


2.在事件對話框中選擇鍵盤中的釋放功能，右鍵選擇C動作寫入下面的腳本。編譯沒有錯誤後就可以確定運行了。

{int TempValue=GetTagWord("TempValue"); //這段意識是獲取你寫入臨時變量的數值。

char szBuffer[20]; 設置數據長度

sprintf ( szBuffer,"是否輸入數值:%d",TempValue); //這段就是對話框顯示的詢問語言

if ( nChar==13 )//if press enter 這段是獲取按回車鍵狀態

{

int xRet = MessageBox(NULL,szBuffer,"確認窗口",MB_YESNO|MB_ICONQUESTION|MB_SYSTEMMODAL);//message 這段是調用彈窗語句

if ( xRet == IDYES )//confirm operate這段是確認是否操作

{

SetTagWord("d10",TempValue);// set data 這段是按是後將臨時變量的數值寫入寄存器變量

}

}
