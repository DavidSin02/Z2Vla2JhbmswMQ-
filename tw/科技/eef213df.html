為什麼我們需要RF 24-105mm F4-7.1 IS STM鏡頭

2020-11-14
| 科技

佳能RF鏡頭系統發佈後，F2級別的變焦鏡頭以及F1.2級別的定焦鏡頭吸引了市場上絕對的關注，超群的大光圈規格和優秀的畫質表現，讓人們對RF鏡頭群充滿了厚望。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


不過在針對專業攝影師和發燒友的範疇之外，佳能仍然沒有忘記為RF系統推出易用、輕鬆、高性價比的入門級RF全畫幅鏡頭，RF 35mm F1.8鏡頭就是其中的典型代表，而在定焦鏡頭之外，今年佳能帶來了一支輕量化、性價比高的標準變焦鏡頭RF 24-105mm F4-7.1 IS STM。


輕量化標準變焦鏡頭


大部分普通相機用戶，他們對於鏡頭的要求並不高，並不需要4000線對的分辨率，也不需要十幾組的結構來矯正像差和色散，他們需要的是一支素質在水準之上，又非常輕便，可以讓自己輕鬆使用的鏡頭。


EOS RP / RF 24-105mm F4-7.1 IS STM / F8 1/2000 ISO100


RF 24-105mm F4-7.1 IS STM就是定位於此的一支廣角到中長焦的標準變焦鏡頭，最突出的特點是其重量僅為395g，差不多是F4級別24-105mm鏡頭的一半重量。同時鏡頭的和長度，也有不同程度的縮減，這支鏡頭最大直徑為76.6mm，收納長度為88.8mm，在RF鏡頭群中可以說是最為輕盈的變焦鏡頭，整體鏡頭設計思路就是給用戶提供最大程度的便攜。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/500 ISO100


從同類鏡頭的規格來說，這支鏡頭是重量、體積最為輕盈的一枚24-105mm變焦鏡頭，搭配EOS RP這樣的輕量化全畫幅機身，整體重量不超過1000g，對於旅行、日常拍攝來說攜帶非常輕鬆。


焦段覆蓋範圍廣泛


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


24-105mm的鏡頭焦距覆蓋範圍頗為「萬金油」，從廣角到中長焦，包含了24mm廣角，35mm、50mm常用焦距，85mm、105mm中長焦距，可以應對人像、靜物、風光、建築甚至微距。


EOS RP / RF 24-105mm F4-7.1 IS STM / F5.6 1/2000 ISO100


廣角端24mm是這支鏡頭比較常用的焦段，在拍攝風光、建築時，能提供84度左右的廣闊視野，廣角端最大光圈為F4，全開光圈也有不錯的畫質表現，應對旅行、日常拍攝絕無問題。這也是佳能RF鏡頭群中，可以以最輕、最高性價比的方式體驗到全畫幅+24mm廣角的鏡頭。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


長焦端105mm可以提供中長焦的視角壓縮效果，利用率同樣很高。長焦端最大光圈為F7.1，在白天拍攝時有充足的光線，實際使用並不會感到困擾，可以保持較高速度快門，並且長焦端畫質搭配EOS RP機身同樣在水準之上。


5擋手持防抖機構


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/1000 ISO100


這支鏡頭搭載了IS防抖機構，CIPA標準下提供5擋的補償效果，理論上長焦端可以使用1/4秒左右的快門速度手持拍攝。實際測試1/50秒的快門速度手持畫面非常可靠，降低到1/25秒左右，仍然有非常好的補償效果。防抖機構的加入，讓這支鏡頭在暗光環境下，尤其是長焦端的使用，有了更大的空間。


STM對焦馬達


EOS RP / RF 24-105mm F4-7.1 IS STM / F8 1/1000 ISO100


RF 24-105mm F4-7.1 IS STM鏡頭使用了步進式對焦馬達，對焦安靜迅速，合焦行程平順，應對日常靜態攝影並無壓力的前提下，可以當作視頻用鏡頭，在拍攝視頻時對焦也不會發出聲音，且對焦變化表現柔和。


極為優秀的微距性能



這支鏡頭雖然定位入門，但卻具備了非常有趣的近攝能力，自動對焦狀態下廣角端24mm最近對焦距離約為0.2米，而如果使用手動模式下則可以縮進到0.13米，長焦端最近對焦距離為0.34米，在使用廣角端時有0.5倍的放大倍率，從理論上來說可以稱之為一枚微距鏡頭了。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/500 ISO100


105mm長焦端下，最近對焦距離為0.34米，可以極為迫近主體，配合7片圓形光圈，雖然只是一枚入門標準變焦鏡頭，也可以拍攝出不錯的淺景深凸顯主體的照片。


為什麼我們需要一支輕量化變焦鏡頭


RF 24-105mm F4-7.1 IS STM的發佈實際上有些出乎人們的意料，在F2變焦、F1.2定焦的路線圖中，這支定位比RF 24-105mm F4L IS USM更加親民的鏡頭實際上更適合入門級相機用戶。


一覽鏡頭圖譜，這枚24-105mm鏡頭是最為輕量化、最為緊湊的全畫幅變焦鏡頭之一，搭配EOS RP可以給用戶提供最小負擔的全畫幅變焦鏡頭組合，手持這套組合的攝影師可以利用手裡的相機和鏡頭，去拍攝風光、建築、人文紀實、人像甚至微距靜物。


EOS RP / RF 24-105mm F4-7.1 IS STM / F7.1 1/250 ISO400


在無反範疇內，性能、素質超越單反的鏡頭有許多，但同時不少頂級無反鏡頭在便攜性上反而有所下降，RF24-105mm F4-7.1 IS STM這樣使用起來更輕鬆的鏡頭，更可以讓入門級用戶，喜歡上用相機拍照，也會提高相機用戶的使用興趣。
