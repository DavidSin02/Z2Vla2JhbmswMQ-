想讓你的工作更有效率？推薦7款Windows上的任務管理工具

2020-10-29
| 科技

此前，很少在 Windows 上接觸任務管理類的工具，我更傾向使用跨平臺的筆記類應用，比如有道雲筆記和 Google Keep。不過，自從成為少數派簽約作者後，這一習慣有了變化，經常遇到需要快速記錄選題靈感或者寫作思路，並且根據時間制定寫作計劃、進度跟蹤，逐漸感覺到筆記類應用已經不能高效地滿足我在任務管理上的需求，所以逐漸開始接觸瞭解任務管理工具。

最早接觸任務管理概念是在 Android 移動平臺，第一款深入使用的任務管理應用是 Any.do ，當初任務管理給我留下了兩大主要概念：

  • 列表式事項管理；

  • 到期提醒功能。

憑著這懵懂的概念知識，讓我們正式開始尋找 Windows 平臺上好用的任務管理工具：

Todolist

Todolist 是一款相當「標準」的任務管理工具，支持的平臺最為全面，包括了 Web、Mac、Windows、iOS、Android 以及 Chrome/Safari/Firefox，甚至還可在 Apple Watch、Android Wear 以及 Outlook/Gmail 電郵端使用這款優秀的應用。

Todolist 有專門為 Windows 10 系統準備了 UWP 安裝版本，也有 exe 版本，很好地滿足了不同 Windows 系統平臺的需求。

經過這段時間的使用，Todolist 具備以下優勢：

  • 自然語義添加任務。除了「正兒八經」地輸入任務、設置到期時間、提醒時間、添加標籤/優先級外，用戶還可以直接通過自然語音的輸入方式添加任務，Todolist 將會自動幫助你添加到期時間的任務事項；

  • 支持項目、標籤和過濾器三大形式管理任務內容。項目管理方式類似文件夾，標籤代表著任務的屬性/定義，而過濾器被視為 Todolist 的一大特色，可按照任務分配給我、優先級別、沒有到期的條件快速定位不同的任務；

  • 視覺化工作效率。Todolist Karma 數值將由 Todolist 對用戶完成任務的效率進行打分，隨著時間的推移追蹤用戶的效率並將工作效率趨勢視圖化；

  • 更詳盡的任務描述。Todolist 不僅支持文字來描述創建的任務，還可讓用戶自行添加 PDF、數據表格和照片附件來完善任務細節。

Todolist 提供了多個不同的付費方案，分別有免費基本版、高級個人版（28.99 刀/ 年）、商務團隊版（28.99 刀/用戶/年），其中多形式提醒任務到期、為任務細節添加附件、標籤和過濾器都屬於高級版提供的附件功能。Todolist 應用本身具備完善的功能，推薦給追求實用、全面，同時又有多平臺同步數據需求的用戶，通過 DrManhatt 的還沒找到合適的任務管理工具？免費領 Todoist 一年高級會員試試 可免費領取 Todoist 一年高級會員資格。

滴答清單

滴答清單所支持的平臺與 Todolist 類似，可謂是跨平臺的「巨無霸」，涵蓋了 Web、macOS、Windows、iOS、Chrome、Firefox 以及 Apple Watch、Android Wear 智能設備平臺。滴答清單更專注於多方式創建任務事項和任務提醒的環節上，其優勢主要體現在：

多方式創建任務。滴答清單支持以自然語言，語音輸入，Siri 助手創建任務，微信端便捷創建任務的方式。應用還專門提供了專屬郵箱，類似todo+XXX@mail.dida365.com的電郵地址，或者使用你的註冊郵箱發送任務到 todo@mail.dida365.com，即可創建任務到滴答清單的收集箱。

智能提醒。滴答清單具有多鬧鐘提醒、自定義重複功能和稍後提醒功能，比如多鬧鐘提醒，用戶可以分別設置提前一天、提前 5 小時、提前 1 小時等不同時間點的鬧鐘；自定義重複提醒，有助於較少用戶頻繁在固定時間設置同樣的任務，譬如每週六下午有固定的足球友誼賽，即可設置為每週六下午 2 點提醒有足球賽事，而不需要每週六前專門設置重複的任務內容。

雙向備份。滴答清單支持將所有的清單、任務、子任務、文本、提醒生成備份，將以 csv 文件保存相關數據，也支持導入其他應用程序備份，包括 Wunderlist、Omnifocus、Toodledo、iCal。

雙向訂閱。滴答清單目前支持訂閱其他日曆的事件到滴答清單，反過來，也支持將滴答清單以 URL 鏈接方式添加到 Google Calendar, Outlook 等日曆應用。

Windows 平臺的特色功能

滴答清單有針對 Windows 平臺開發了特色功能，有打開/隱藏主程序、添加任務的快捷鍵，同時還提供了任務列表和日曆視圖的桌面小部件，幫助用戶快速創建和查看任務。

除了以上特點外，滴答清單還提供了統計、摘要的選項幫助對創建的任務進行一次回顧和覆盤的特色功能。不過諸如訂閱日曆、日曆視圖、自定義智能清單、高級搜索、清單和任務動態等功能均屬於高級付費版（139 元/年），普通用戶版本在清單數量、任務數量、提醒數量、子任務數量、共享成員數量、附件數量均有所「縮水」。

Trello

加入少數派後，首次接觸了團隊協作工具 Trello，卡片組成的看板形式成為極具特色的設計風格。Trello 本身具備到期提醒的功能，我派作者 Jiahao 在[用 Trello 搭建任務系統]中，分享瞭如何使用以看板為基礎的 Trello 搭建任務系統。Trello 在跨平臺上雖然沒有達到 Todolist 和滴答清單那樣「變態」，但依然支持 Web、iOS、Android、macOS、Windows 以及 Chrome、Firefox、IE11/Edge、Safari 瀏覽器平臺，目前 Trello 僅支持 Windows 10 系統。

對於很多習慣了主動通知提醒，和列表管理任務的用戶，Trello 在消息通知和看板管理形式上存在不足的地方。而我運用 Trello 進行任務管理上會專門建立一個 Todo 的任務板（關於寫作計劃類），任務板下會有六個列表，分別是：

  • 歷史選題箱：存放除了今天收集的選題，再使用標籤分類優先級別；

  • 今天收集選題：存放今天當天收集的寫作選題；

  • 正在開展準備：存放已經開展素材收集的選題，並且在選題描述中添加大綱內容；

  • 已動筆：存放已經開始寫作的選題，將文章內容的一二級標題以清單形式展示，並標準哪部分已經完成了；

  • 推遲完成：存放由於其他因素影響了寫作進度需要推遲完成的選題；

  • 歸檔箱：顧名思義就是存放已經完成的選題。

在使用 Trello 大半年的時間裡，深刻感受到其看板管理寫作上的優勢，使用 Trello 搭建一套很完整的任務管理系統，將有別於其他任務管理工具以列表型進行展示管理，Trello 以看板作為核心特點，結合本身在團推協作、標籤、添加到期時間、與第三方軟件聯動等功能，推薦給圍繞 Trello 打造一套自己工作流的用戶。

Microsoft To-Do

今年年度最佳榜單中獲得「最三心兩意的應用開發商」，非微軟莫屬（傳送門 → 獲獎原因），Microsoft To-Do 就是其中的代表作，支持 Web、Android、iOS、Windows UWP 平臺，幾大平臺的體驗保持了很好的一致性，UWP 應用界面可以視為移動端的放大版本。

Microsoft To-Do 的優勢是：

  • 我的一天。打開應用主界面，率先聚焦的是「我的一天」界面，我的一天是 To-Do 應用默認內置的任務清單，顯示了今天待辦的任務，幫助用戶快速定位今天當天需要辦理的任務事項；

  • 重複提醒。用戶可以對任務設置重複提醒的時間，提供了每 1 天、工作日、每 1 周、每1 年或者自定義重複週期，避免多次手動添加需要重複提醒的任務；

  • 簡單清爽的界面。我在 Windows 和 Android 端都有使用 Microsoft To-Do 應用，簡單流暢的操作方式留下了不錯的印象，To-Do 還支持不同清單設置不同的皮膚，增強明顯的視覺效果。

Microsoft To-Do 應用提供了輕量化的任務管理，數據同步上並沒有受到國內特殊環境所影響，還支持 Outlook 任務中同步事項、清單，有簡單的到期提醒、搜索待辦事項和筆記內搜索的功能，推薦給深度使用微軟系列產品的用戶。

Moodo

Moo.do 是一個類似於 Workflowy 的無限層級的純文本任務管理工具，正如特倫在「Moo.do 中文入門手冊：增強版 Workflowy，一個無限層級的純文本任務管理工具」提到的，Moo.do 的 slogan 仿似 Workflowy：「Organize your way」，同樣擁有無限層級的樹形結構，但它也擁有任務管理、多面板查看、日曆等多種增強功能，就像是一個增強版的 Workflowy。Moo.do 除了類似 Workflowy 的無限層級以外，Moo.do 的多面板、通知、日曆、優先級等功能讓它也可用於任務管理，其創新性的地方體現在：

  • 無線層級的列表功能。區別於 Workflowy 只能擁有一個文檔，Moo.do 支持創建多個 Document 文件夾，用戶新建一個新文檔時，將呈現一張白紙，讓用戶無限層級地創建個人任務事項；

  • 多功能用途的 Pane 面板。Pane 是一個平鋪展示的窗格，可讓用戶將多個列表以「平行視角」的形式展示出來，用戶無需多次跳轉即可查看多個子目錄內容。除此之外，Moo.do 還提供了被稱為 Calendar Pane、Archived Pane 的多功能用途面板，比如 Calendar Pane 是一個專用於日程安排的 Pane，可將用戶在任務事項中的內容按時間顯示出來，也可以手動拖動條目到 Calendar Pane，實現快速為任務條目插件日期的目的；

  • 高度整合 Google 服務。Moo.do 支持 Google Calendar、Gmail、Google Drive、Google Contacts 之間無縫同步數據，並且 Moo.do 僅支持使用 Google 賬號進行登錄，應用的數據會保存在用戶個人的 Google Drive 賬戶中，擁有一定的安全性。

Moo.do 極具創新性地使用類似 Workflowy 的功能，圍繞無限層級列表，多功能 Pane、@ 和 #結合使用，幫助用戶構建一個更加詳盡的任務安排系統。推薦給不需要華麗界面，更願意自己「把握」任務安排的用戶。

Remember The Milk

Remember The Milk 作為有著十年曆史的任務管理工具，支持 macOS、Windows、Linux、iOS、Android、BB10、Apple Watch、Outlook，以及包括 Chrome、Firefox、Safari、Edge 瀏覽器平臺，其優點包括了強大的快捷鍵、高效的日期識別、支持子任務，用戶可以通過手機，郵件，IM 等很多方式添加任務，後期則支持電子郵件、IM 提醒、谷歌日曆、手機軟件、手機短信等方式提醒用戶。

Remember The Milk 的優勢：

  • 使用文件夾，標籤，上下文，子任務等來組織任務事項；

  • 可設置任務的優先級、完成時間、剩餘時間、是否重複、列表、標籤以及更多內容；

  • 高級搜索。支持使用標識符進行搜索，搜索你的任務和筆記，並保存你喜歡的搜索內容作為智能列表；

  • 使用文件夾，標籤，上下文，子任務等來組織任務事項；

  • 自定義排序和分組。可按照任務添加日期、優先級、名稱和列表名稱的方式進行排序和分組；

  • 強大的任務提醒。支持通過電子郵件、IM 軟件、短信、Twitter、手機或者系統桌面通知的方式提醒任務；

  • 電郵通知每日任務摘要。用戶可以將今天到期或者已經開始的任務、明天/未來 7 天開展的任務等定製的任務設置成每天發送郵件的方式。

Cortana

Cortana 作為最具 Windows 10 系統特色的功能之一，Cortana 支持手動或者語音方式添加任務事項，只需鍵入「提醒我...」或者說「你好小娜，提醒我...」就可以開始使用。Cortana 創新性提供基於時間、位置、照片的方式提醒，其中照片提醒功能，用戶將照片添加到提醒中，例如孩子最喜愛的遊戲，這樣 Cortana 就會在下次購物時作出提醒。

由於 Google Assistant、Apple Siri、Amazon Alxea 並未登錄 Windows 系統，Cortana 支持 Windows、Android、iOS 平臺，有著跨平臺的優勢，支持簡單的任務內容、時間提醒功能，語音錄入任務備忘事項有不錯的識別準確度。

由於 Any.do（我首款使用的 todo 工具）、Checkvist（強大的 Open API 接口）、Google Keep（更像「便條紙」） 、Toodledo （老牌工具） 暫時沒有官方的 Windows 客戶端，Wunderlist 在被微軟收購之後基本處於停擺狀態，所以它們都不在這次挑選推薦的應用列表範圍內，但這些工具都提供了很好的 Web 端和移動版本，歡迎大家在體驗之後留言評論它們的亮點。

蘋果 macOS/iOS 平臺上有好幾款被視為「神器」的任務管理工具，比如 OmniFocus、Things、TaskPaper、2Do，而 Windows 平臺上也不缺乏一些相當優秀的任務管理工具，它們能夠提供給你多樣的選擇，有完整任務管理系統的 Todolist、主打語音創建事項的 Cortana、無限層級形式的 Moodo。

如果你希望對任務管理工具、任務管理理念和方式有更深入的理解學習，少數派針對不同的任務管理軟件推出了一系列欄目，深入淺出地帶你走進任務管理的世界，讓你更科學、更系統地做好任務管理，歡迎加入我 π 付費教程。
