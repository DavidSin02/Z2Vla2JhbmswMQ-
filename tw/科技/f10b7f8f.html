“今年成績怎麼樣啊”

熊孩子最會搶手機玩遊戲，一不小心下N種付費遊戲，慘的還是我們這些倒黴親戚。

一旦發現他們有伸手要手機的苗頭，什麼都不要想，開口就問：

//

想玩遊戲呀？今年成績好不好啊？

//

如果考得好，那就“繼續努力，遊戲會影響成績的”；如果考得不好，那就“一定是你遊戲玩多了”

“來表演個節目吧”

春節是熊孩子的“原諒節”，各種皮都能被“他還是個孩子”這種藉口糊弄過去。但是！我們也可以反擊！

當熊孩子們蠢蠢欲動開始翻天時，只要說一句：

//

這孩子真機靈，會表演什麼節目呀？

//

你就等著七大姑八大姨各種慫恿表演，然後開心地鼓掌就好啦~
