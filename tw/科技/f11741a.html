自控貓｜儀表設計統一規定18-控制閥選型

2020-10-29
| 科技

王玉虎

儀表專業設計統一規定 儀表選型-調節閥

1) 一般選用 Globe 閥。根據工藝介質及操作條件特殊要求可選用偏心旋轉閥、單座調節閥、角型閥、蝶型閥、球閥、三通閥等形式。

2) 8”和 8”以下的調節閥優先選用單/雙座或套筒式 Globe 調節閥。10”和 10”以上口徑或低差壓情況，採用蝶型閥或偏心旋轉閥。對於介質中含有固體粉末或粘度較大的場合，採用Ｖ型球閥或偏心旋轉閥。對於噪聲較大的情況，採用籠式閥或低噪音調節閥。根據工藝介質及操作要求也可選用角型閥、三通閥、波紋管密封閥等。對於高壓差、閃蒸、空化、腐蝕、高噪聲等情況，可選用特殊的閥芯和閥體設計。

3) 調節閥選用等百分比特性、線性特性或近似等百分比特性。等百分比特性調節閥最小流量開度不應小於 10%；正常流量開度 60%~80%；最大流量開度不應超過 90%；線性特性調節閥最小流量開度不應小於 15%；正常流量開度 50%~70%；最大流量開度不應超過 80%.

4) 調節閥設計應使調節閥下游 1 米處和管道表面 1 米處的噪聲等級不超過 85dBA。 間歇使用或緊急操作的調節閥在上述位置的噪聲不超過 105dBA。調節閥的降噪措施優先選用低噪音閥內件、改變閥門結構形式等在調節閥內部節流降噪的方法。

5) 根據工藝要求(在 PID 圖上表示)，可選用帶手輪調節閥，或設旁路閥及前後切斷閥。原則上，SIS 用的調節(開關)閥不設手輪，工藝專利商有特殊要求的除外。

6) 安全儀表系統用、要求嚴密關閉的閥門選用單座閥、球閥、蝶閥等。選用氣缸執行器時帶彈簧返回裝置，氣源故障時閥應處於安全位置。安全儀表系統及順序控制用閥門應裝有限位開關，限位開關為接近式，接點類型為幹接點，且至少為單刀雙擲（SPDT）。

7) 閥體材料應按配管材料等級規定要求選取，閥體材料應不低於碳鋼。

8) 閥內組件和閥座材料一般為 316SS。高差壓、高溫等場所應選用與配管等級一致的材料。在閃蒸、空化或嚴重沖刷以及高溫、高壓差或強腐蝕性場合，應在閥內件表面堆焊或噴塗硬質合金等材料。

9) 當使用溫度超過 204℃或低於-18℃時，閥應帶散熱片或延長頸型。

10) 調節閥洩漏等級通常為 CLASS IV （ANSI/FCI 70-2），當工藝要求為嚴密關閉 （TSO）時，應選用 CLASS VI 或 CLASS V。開關閥洩漏等級應至少為 CLASS V。

11) 執行器通常選用氣動薄膜式，在要求大推力的場合可選用單動作或雙動作氣缸執行器。

對於氣源故障時要求開或關的場合，推薦優先選用帶彈簧返回的薄膜式或氣缸執行器。選擇執行機構時，應基於儀表空氣的最小壓力進行輸出力（或力矩）計算。

12) 在氣源故障時，應保證閥門處於“故障安全”位置。一般情況下，閥門的聯鎖位置和氣源故障位置一致。必要時使用儲氣罐以確保聯鎖時位於正常位置。電磁閥失電(非勵磁)聯鎖。

儀表空氣儲罐的製造應符合《固定式壓力容器安全技術監察規程》的有關規定，並應根據調節閥的安全完整性等級確定是否配置壓力開關。

13) 一般情況宜選用單層填料函結構；對於低溫、高溫或高毒性介質，應選用雙層填料函結構。介質溫度在-18℃～204℃範圍內時，宜選用 V 型 PTFE 填料；介質溫度低於-18℃或高於 204℃場合應選用柔性石墨填料或其他合適材料的填料。

14) 一般選用 4~20mA DC 帶 HART 協議輸出的智能型電/氣閥門定位器；也可選用電/氣轉換器，與氣動閥門定位器配合使用。一般選用正作用式電/氣閥門定位器。

15) 電磁閥選用 24VDC 低功率（低於 4W） ，316SS 材質。完全密封，高負載電磁線圈，絕緣等級為 H（高溫） 。三通或四通電磁閥應選用長期帶電型。

16) 自力式調節閥只適用於 N2、空氣、燃料氣、蒸汽和其它輔助用流體等調節要求不嚴的場合。

17) 壓縮機防喘振調節閥：

a) 除非數據表中有明確說明， 輸出信號到 4mA 或電磁閥失電時， 閥門從全關到全開的時間不應超過 2 秒；輸出信號到 20mA 或電磁閥勵磁時，閥門從全開到全關的時間不應超過 5 秒。

b) 防喘振調節閥宜選用線性流量特性的調節閥。

18) 用於輸送含顆粒或易暴聚介質時，開關閥宜選用旋塞閥、柱塞閥或 V 形球閥。

19) 一般情況下，閥門應採用法蘭連接。用於烴類或其他可燃介質的角行程閥門宜選用法蘭連接型；當 DN>300 時，蝶閥及偏心旋轉閥也可選用支耳（Lug）連接型，不得選用無法蘭對夾（Wafer）連接型。連接法蘭等級應與配管材料等級規定一致。

儀表專業設計統一規定 儀表選型-開關閥

1. 開關閥

1) 除非有特殊要求，開關閥的尺寸與管線尺寸相同，並且為全通徑。開關閥的要求基本可參照調節閥。

2) 開關閥的閥門形式選擇原則：

a) 低壓氣體及液體選用球閥或閘閥；

b) 含有固體顆粒介質的場合選用球閥；

c) 在大口徑、低壓差及含有懸浮顆粒物介質的場合選用蝶閥；

d) 高壓、高壓差的場合選用球閥、平行閘閥；

3) 開關閥的洩漏等級為 ANSI/FCI 70-2 CLASS V 或 CLASS VI；

4) 開關閥的動作時間應滿足工藝要求，當工藝無特殊要求時，電磁閥失電後對應開關閥的全開或全關時間不超過 3 秒/每英寸口徑，或者小於 1 秒/每英寸口徑，閥體尺寸（英寸）（以秒計）。任何閥的最大行程時間為 30 秒。緊急切斷閥最大行程不超過 10 秒。

5) 開關閥執行機構的輸出力矩應至少留有 30%的安全係數。

6) 開關閥配置全開及全關閥位開關，SIS 用的閥門應配閥位開關。

7) 開關閥的閥體應符合 API 607 或 API 6FA 耐火試驗標準。
