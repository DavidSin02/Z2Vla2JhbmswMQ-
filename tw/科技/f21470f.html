推薦十部希臘史詩級的神話電影

2020-10-29
| 科技

希臘神話是西方文學的開端，是所有西方文學的最根本的基石，而希臘神話題材電影在好萊塢影片中也是備受歡迎，多年來，上映了許多口碑佳作，特意蒐集了十部個人覺得還不錯的給大家~

1、《特洛伊》

導演：沃爾夫岡·彼德森

編劇：戴維·貝尼奧夫

主演：布拉德·皮特 / 黛安·克魯格 / 奧蘭多·布魯姆 / 朱利安·格洛弗 / 布萊恩·考克斯 / 內森·瓊斯 / 艾瑞克·巴納 / 雅各布·史密斯 / 蘿絲·拜恩 / 彼得·奧圖爾

類型：動作 / 愛情

上映日期：2004-06-12(中國大陸) / 2004-05-14(美國)

簡介：在浪漫壯觀、引人遐思的古希臘，奧林匹亞眾神與人類之間的愛恨傳奇至今為人津津樂道，而在這其中，荷馬史詩《伊里亞特》所記錄的特洛伊之戰更是後世劇作家與觀者反覆演繹的經典作品。當特洛伊王子帕里斯(奧蘭多·布魯姆 Orlando Bloom 飾)受希臘斯巴達國王之邀赴宴之際，卻迷戀上了國王的妻子海倫(黛安·克魯格 Diane Kruger 飾)，對方傾國傾城的容貌讓他不能自已，遂將海倫帶回自己的國家。

此舉引發了希臘諸國的憤怒，在邁錫尼國王阿伽門儂(布萊恩·考克斯 Brian Cox 飾)的號召下，一支強大的聯軍浩浩蕩蕩向特洛伊挺進。在隨後長達十年的戰爭中，阿喀琉斯(布拉德·皮特 Brad Pitt 飾)、奧德修斯(肖恩·賓 Sean Bean 飾)、赫克托爾(艾瑞克·巴納 Eric Bana 飾)等英雄各逞英豪，譜寫了蕩氣迴腸卻又令人扼腕唏噓的傳奇史詩。

2、《亞歷山大大帝》

導演：奧利佛·斯通

編劇：奧利佛·斯通 / Christopher Kyle

主演：科林·法瑞爾 / 安吉麗娜·朱莉 / 方·基默 / 安東尼·霍普金斯 / 傑瑞德·萊託 / 羅莎里奧·道森 / 喬納森·萊斯·梅耶斯 / 克里斯托弗·普盧默

類型：劇情 / 動作 / 愛情 / 傳記 / 歷史 / 戰爭 / 冒險

上映日期：2004-11-24(美國)

簡介：影片講述了Alexander(柯林·法瑞爾 Colin Farrell 飾)年輕時的夢想、榮耀與冒險生活，他征服世界後的孤獨生活以及神祕的死亡。他與將軍Hephaistion(傑瑞德·萊託 Jared Leto 飾)不尋常的關係也得意展現。Alexander的母親(安吉麗娜·朱莉 Angelina Jolie 飾)也在他的生命中起到不同凡響的影響力。

亞歷山大自幼聰穎過人，曾受教於亞利斯多德，稍長即被授予首府總督之職，並以平定山賊有功而任命為馬其頓大軍統帥，隨父橫掃希臘城邦之亂。融合東西文化，鼓勵民族間通婚，並倡導民族間地位平等，講希臘思想律法散佈各地，開創希臘化時代文化，死時卻年僅33歲。

3、《潘神的迷宮》

導演：吉爾莫·德爾·託羅

編劇：吉爾莫·德爾·託羅

主演：伊萬娜·巴克羅 / 塞爾希·洛佩斯 / 瑪麗維爾·貝爾杜 / 道格·瓊斯 / 阿里亞德娜·希爾

類型：劇情 / 懸疑 / 戰爭 / 奇幻

上映日期：2006-10-11(西班牙)

簡介：12歲女孩子奧菲麗婭(伊萬娜•巴克爾洛 飾)有一個凶暴的法西斯軍官繼父維達(塞吉•洛佩茲 飾)，維達的任務就是鎮壓反抗者，研究各種刑具來折磨殘害被抓來的異見人士，奧菲麗婭每天都目睹讓人不愉快的事件。眼看著冷酷的繼父和其黨羽每日作惡還以此為樂，母親又身懷六甲患病在床，被孤獨感和目睹暴行的痛苦困擾的奧菲麗婭開始沉浸在自己的幻想中聊以自慰。

一天，奧菲麗婭遇到了迷宮的守護者，潘神(道格•瓊斯 飾)。潘神告訴她，她其實是奇幻王國失蹤的公主，要想恢復身份，必須通過三個危險的考驗。

4、《角鬥士》

導演：雷德利·斯科特

編劇：大衛·弗蘭佐尼 / 約翰·洛根 / William Nicholson

主演：羅素·克勞 / 傑昆·菲尼克斯 / 康妮·尼爾森 / 理查德·哈里斯 / 傑曼·翰蘇 / 奧列佛·裡德 / 德里克·雅各比

類型：劇情 / 動作 / 歷史 / 冒險

上映日期：2000-05-05

簡介：馬克西·蒙斯(羅素·克勞 飾)是羅馬帝國戰功顯赫，受人擁戴的大將軍。老國王馬庫斯·奧利利烏斯(理查德·哈里斯 飾)對大將軍賞識有加，有意加冕於他。老國王對將軍這種超越親情的寵愛自然招來太子康莫迪烏斯的妒忌與不安，太子伺機殺害父親，搶先登上王位，並馬上下令誅殺大將軍一家。

馬克西蒙斯死裡逃生躲過暴君的魔爪，可是他所有的家人都未能倖免於難，返回家園，他只看到滿目瘡痍的破敗景象。走投無路的馬克西蒙斯只能投身為奴，從昔日的大將軍淪為一名角鬥士，在血腥廝殺中苟且求生。

憑藉英勇的氣魄和高超的武藝，馬克西蒙斯漸漸脫穎而出，晉級到羅馬的大角鬥場。這一次，他的劍直指暴君康莫迪烏斯，失去親人的痛苦，淪為奴隸的屈辱，在馬克西蒙斯的內心燃起了熊熊怒火。

5、《雷神》

導演：肯尼思·布拉納

編劇：阿什利·米勒 / 扎克·施坦茨 / 唐·佩恩 / J·邁克爾·斯特拉辛斯基 / 馬克·波託塞維奇 / 斯坦·李 / 拉里·利伯 / 傑克·科比

主演：克里斯·海姆斯沃斯 / 娜塔莉·波特曼 / 湯姆·希德勒斯頓 / 安東尼·霍普金斯 / 斯特蘭·斯卡斯加德 / 凱特·戴琳斯 / 克拉克·格雷格 / 科魯姆·費奧瑞 / 伊德里斯·艾爾巴 / 雷·史蒂文森 / 淺野忠信

類型：動作 / 奇幻 / 冒險

上映日期：2011-05-08(中國大陸) / 2011-05-06(美國)

簡介：勇敢善戰雷神托爾(克里斯•海姆斯沃斯 Chris Hemsworth 飾)，作為永恆境界阿斯加德的王位繼承人，受人敬仰。然而由於一次魯莽的行動，險些挑起一場古老的戰爭，因此被父親天神奧丁(安東尼•霍普金斯 Anthony Hopkins 飾)流放到人間。托爾在人間認識了 美麗的女科學家珍•福斯特(娜塔莉•波特曼 Natalie Portman 飾)和她的工作夥伴，在他們的幫助下，托爾找到了和自己一起被貶的雷神之錘，但此時的雷神之錘並不屬於他。

此間，托爾的弟弟洛基為了成為阿斯加德的王，居然聯合寒冰巨人，企圖在永恆境界掀起一場腥風血雨。為了維持永恆之界的法度，托爾的四位朋友也來到人間，他們決定帶托爾返回永恆之境。此時，為了阻止雷神重歸永恆境界，洛基派出巨人殺死托爾。喪失能力和法戰的托爾為了保護無辜的人類與自己的朋友，獻出了自己的生命。托爾的死打破了天神的的咒語，托爾復活並重獲雷神之力戰勝巨人。此時，天神生命危在旦夕，托爾必須與自己的朋友們重返永恆之界，維護那裡的法度。

6、《驚天戰神》

導演：塔西姆·辛

編劇：查理·帕拉伯尼得斯 / 維拉斯·帕拉伯尼得斯

主演：亨利·卡維爾 / 米基·洛克 / 約翰·赫特 / 盧克·伊萬斯 / 斯蒂芬·多爾夫 / 芙蕾達·平託 / 凱南·魯茲 / 伊莎貝爾·盧卡斯 / 約瑟夫·摩根 / 柯里·瑟威爾 / 史蒂芬·麥克哈蒂 / 羅伯特·梅耶 / 彼得·斯戴賓斯 / 丹尼爾·沙曼

類型：劇情 / 動作 / 奇幻

上映日期：2011-11-11(美國/中國大陸)

簡介：天界之戰中，奧林匹斯山的眾神擊敗了邪惡的泰坦神，並將他們囚禁於地下。傳說只有威力無窮的厄皮洛斯神弓能夠釋放泰坦，但這把神器卻在天界大戰中遺失在人間。多年後，權慾薰心、冷酷嗜血的海恩波利國王(米基·洛克 Mickey Rourke 飾)推行暴權，妄圖踏平希臘、毀滅神蹟。

西柏林修道院美麗的先知帕德拉( 芙蕾達·平託 Freida Pinto 飾)能夠預知未來，在海恩波利的囚牢中她感受到了一個受神指引的年輕人忒修斯(亨利·卡維爾 Henry Cavill 飾)將拿到神弓並和海恩波利國王聯手。此時的忒修斯卻發誓要為在海恩波利的襲擊中死去的母親報仇。

忒修斯正如預言找到了神弓，但不幸卻落入了海恩波利國王的手中，國王兵臨城下，希臘軍勢單力薄，高大的城牆在手持神弓的海恩波利國王面前也不堪一擊。恐怖的泰坦即將被釋放，希臘危在旦夕，出身低微的忒修斯雖然勇武，但在同族中卻屢遭唾棄，此時，海恩波利國王卻向忒修斯伸出了橄欖枝。

7、《諸神之戰》

導演：路易斯·萊特里爾

編劇：費爾·海伊 / 馬特·曼弗萊迪 / 特拉維斯·比徹姆 / 貝弗利·克羅斯

主演：薩姆·沃辛頓 / 傑瑪·阿特登 / 麥斯·米科爾森 / 拉爾夫·費因斯 / 連姆·尼森 / 尼古拉斯·霍爾特 / 艾莉克莎·黛沃洛斯 / 傑森·弗萊明

類型：動作 / 奇幻 / 冒險

上映日期：2010-04-16(中國大陸) / 2010-04-02(美國)

簡介：本片根據希臘神話傳說改編，講述了不堪忍受奧林匹斯眾神殘暴壓迫的阿戈斯國王亞克里斯(Jason Flemyng 飾)奮起反抗，對至高無上的神明大為不敬。在冥王哈迪斯(拉爾夫·費因斯 Ralph Fiennes 飾)的慫恿下，天神宙斯(連姆·尼森 Liam Neeson 飾)決定對愚蠢的人類施以懲罰。

遭遇海難流落到阿戈斯的青年珀修斯(薩姆·沃辛頓 Sam Worthington 飾)是宙斯和亞克里斯的皇后生下的半神，他親眼目睹哈迪斯所施下的恐怖神蹟，並得知哈迪斯將釋放北海怪獸克拉肯懲戒世人，除非亞克里斯將美麗的公主安德魯美達(艾莉克莎·黛沃洛斯 Alexa Davalos 飾)作為犧牲獻給眾神。在女神愛娥(Gemma Arterton 飾)的點撥下，珀修斯知曉了自己的身世與使命，於是決定啟程殺死北海怪獸並摧毀哈迪斯的陰謀。

8、《貝奧武夫》

導演：羅伯特·澤米吉斯

編劇：尼爾·蓋曼 / 羅傑·阿夫瑞

主演：雷·溫斯頓 / 克里斯平·格洛弗 / 安吉麗娜·朱莉 / 羅賓·懷特 / 安東尼·霍普金斯 / 約翰·馬爾科維奇 / 艾莉森·洛曼

類型：動作 / 動畫 / 奇幻 / 冒險

上映日期：2007-11-16(美國)

簡介：在英國撒克遜人長篇史詩《貝奧武夫》的描述中，主人公貝奧武夫(雷·溫斯頓 Ray Winstone 飾)是一位擁有強健體魄和驚人戰力的武士，他一身將經歷三場對其影響重大的戰役。為了拯救被巨人格蘭戴爾(克利斯丁·格拉夫 Crispin Glover 飾)掠奪的村莊，貝奧武夫帶領一群武士渡過海洋向其挑戰。

在一場浴血奮戰之後，格蘭戴爾被武士們所擊敗。但是戰鬥卻沒有結束，格蘭戴爾的母親(安吉麗娜·朱莉 Angelina Jolie 飾)為了復仇向貝奧武夫發起了挑戰。一場更大的戰役即將打響。雖然這次，勇士們將付出更加慘痛的代價，但是在無所畏懼的貝奧武夫面前，即便是再厲害的對手，也將被毀滅。多年後，與一條惡龍的終極之戰，最終將貝奧武夫的名字變成了不朽的傳奇。

9、《尼伯龍根的指環》

導演：烏利·埃德爾

編劇：烏利·埃德爾 / Diane Duane / Peter Morwood

主演：本諾·菲爾曼 / 克里斯塔娜·洛肯 / 艾麗西亞·維特 / 朱利安·山德斯 / 薩繆爾·韋斯特 / 馬克斯·馮·敘多夫 / 羅伯特·帕丁森 / Mavie Hörbiger / Aletta Bezuidenhout / Sean Higgs / 戈茲·奧托 / 拉爾夫·莫勒 / Tamsin MacCarthy / Leonard Moss / Ryan Slabbert

類型：劇情 / 動作 / 奇幻 / 冒險

上映日期：2004-11-29(德國)

簡介：根據德國民間詩史《尼白龍根之歌》和北歐《沃爾鬆格傳說》改編。 鐵匠齊格弗裡德(班諾·福曼 Benno Fürmann 飾)在一次隕石墜落後與冰島女王伯倫希爾(艾麗西亞·維特 Alicia Witt 飾)不打不相識，並且陷入愛河。之後他殺死了作惡巨龍，並用龍血沐浴讓自己刀 槍不入。可後背上一塊樹葉覆蓋的地方沒有沾到龍血成為了他的弱點。

齊格弗裡德因殺死巨龍而被國王賞識，要將自己的公主妹妹(克里斯蒂娜·裡奇 Christina Ricci 飾)嫁給他，但格弗裡德已心有所屬。為了達到目的國王在酒中加了藥使他忘記女王愛上了公主。國王請他幫忙冰島女王伯倫希爾，於是他戴上變形頭盔，裝成過國王的樣子打敗了伯倫希爾。伯倫希爾被他的“背叛”傷透了心，被迫嫁給國王。各懷心事的兩對新人在同一天結婚。齊格弗裡德最終會想起女王嗎?四人將對面如何的命運呢?

10、《新木馬屠城》

導演：約翰·肯特·哈里森

編劇：Ronnie Kern

主演：西耶娜·蓋爾利 / 馬修·馬斯登 / 盧夫斯·塞維爾

類型：劇情 / 動作 / 冒險

上映日期：2003-04-20(美國)

簡介：特洛伊的小王子帕里斯(馬修•馬斯登 飾)被通靈的姐姐預言，如果長大成人必將給特洛伊帶來滅頂災難。國王派人把他扔到山上，但卻意外被牧羊人收養。帕里斯長大後英俊瀟灑，三個女神下凡讓他說出誰最美麗，愛神答應給他世界上最美的女人海倫(希艾娜•蓋爾利 飾)，得到了金蘋果。之後，帕里斯回到特洛伊，重新當上王子。特洛伊國王派帕里斯出使斯巴達，他知道，此去凶多吉少。

海倫從小美貌出眾，沒有男人能禁得起她傾城一笑。然而，從她在水中看到帕里斯影子的那一刻，便深愛著他。然而斯巴達王卻奪走了海倫，帕里斯前往斯巴達的當天，正是海倫的婚禮，深愛海倫的帕里斯不顧一切將海倫帶走，而這也給阿伽門農(盧夫斯•塞維爾 飾)侵略特洛伊提供了藉口。戰爭持續了10年，最後將由斯巴達王與帕里斯的決鬥來決定海倫以及特洛伊的命運。
