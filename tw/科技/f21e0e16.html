孩子們的遊戲之路更加艱難了，騰訊遊戲設人臉識別新規！登陸和支付要掃臉。波士頓公司的機器狗Spot發售，定價7.45萬美元，摺合成人民幣大概是.....還有個事兒，微信上線拍一拍功能，試著拍一下你老闆，可能會被移出群~

1、北京檢測的36萬人都是利用大數據找出來的

←

入榜理由

國家衛健委疾病預防控制專家委員會專家吳浩在《新聞1+1》上表示，根據西城區的唐某某明確回憶了自己的行動軌跡，我們用大數據去驗證，是否與所說的是一致。在行動軌跡一致的情況下，利用大數據分析相關的風險人群在哪裡。最近幾天檢測的36萬人都是利用大數據找出來的，迅速得多。時間就是生命，這樣就可以跟病毒在時間上賽跑。

2、微信上線“拍一拍”功能

←

入榜理由

日前，微信更新了7.0.13版本，在iOS和Android版本都上線了“拍一拍”功能。據瞭解，這個“拍一拍”是一種提醒功能，在個人和群聊的對話場景中都可以使用。當你雙擊好友頭像時，頭像除了會抖動外，還會出現“你拍了拍'XXX '”提醒，而且對方是最新版本的話，也能收到這一提醒。值得一提的是，你雙擊自己頭像也會顯示“你拍了拍自己”的提示。

3、中國成功發射高分九號03星等3顆衛星

←

入榜理由

2020年6月17日15時19分，我國在酒泉衛星發射中心用長征二號丁運載火箭，成功將高分九號03星送入預定軌道，發射獲得圓滿成功。此次任務還搭載發射了皮星三號A星、和德五號衛星。

4、百度迴應向騰訊出售愛奇藝股權

←

入榜理由

針對“百度擬向騰訊出售愛奇藝股權”一事，在百度App上實名認證為“百度公關總監”的郭鋒做出迴應稱：“大家別亂猜了。愛奇藝是百度內容生態戰略的重要組成部分，百度會一如既往地支持愛奇藝的發展。”

5、5月高技術製造業增加值同比增長8.9%

←

入榜理由

3月底，TCL武漢華星實現100%復工，4月初供料恢復正常。“化危為機，實現逆勢增長。”和TCL武漢華星一樣，一批高新技術企業逆勢成長。國家統計局數據顯示：5月高技術製造業增加值同比增長8.9%，快於規模以上工業4.5個百分點。

6、字節跳動被曝一季度營收400億元

←

入榜理由

據路透社報道，字節跳動今年一季度營業收入達到400億元（約合56.4億美元），較上年同期增長了130%。一位知情人士透露，字節跳動為2020年訂立了2000億元的營收目標。另一位知情人士表示，字節跳動今年的目標是達到競爭對手騰訊營業額的一半。騰訊2019年的營收約3770億元。

7、波士頓動力開售Spot機器狗

←

入榜理由

機器人製造商波士頓動力宣佈將正式發售其四足機器人Spot，售價74500美元（約合人民幣52.8萬元）。美國企業可以通過該公司網站購買，這也是該公司首項線上銷售的產品。目前Spot可用於測量和數據收集，公司強調不得用於傷害或恐嚇他人。 外媒認為，該產品價格不菲，售價相當於一輛斯拉Model S長續航版。

8、樂視網：未收到賈躍亭的賠償計劃

←

入榜理由

6月17日晚間，處於退市整理期的樂視網發佈澄清公告稱，公司未收到任何來自賈躍亭或賈躍亭債務處理小組的有關公司股東賠償計劃或者安排，如公司收到相關計劃或者安排，公司將會及時公告。

9、騰訊遊戲：疑似未成年人登錄和支付需人臉識別

←

入榜理由

騰訊遊戲對外宣佈，經過多次灰度測試、技術調優後，正式升級未成年人保護措施：在對已實名未成年人“限玩、限充、宵禁”基礎上，專門針對“孩子冒用家長身份信息繞過監管”的問題，擴大人臉識別技術應用範圍，對疑似未成年人的用戶進行甄別。

10、高通發佈首款驍龍6系5G移動平臺

←

入榜理由

高通宣佈推出首款驍龍6系5G移動平臺——高通驍龍690 5G移動平臺。全新平臺旨在進一步推動全球5G體驗的廣泛普及，並提供卓越的終端側AI和暢爽的娛樂體驗。OEM/ODM廠商包括HMD Global、LG電子、摩托羅拉、夏普、TCL和聞泰，均計劃將推出搭載驍龍690的智能手機。
