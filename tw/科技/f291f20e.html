你的辦公效率低是有原因的，這些常用的電腦快捷鍵都掌握了嗎？

2020-11-14
| 科技

*本文來自花粉俱樂部論壇-玩機技巧官方

HUAWEI MateBook X Pro 已上市有一段時間，受到很多商務人士和文字工作者的喜歡。鍵盤是大家使用最多的部件，但是大家真的瞭解華為MateBook X Pro 的鍵盤嗎？一些可提高辦公效率的熱鍵和快捷鍵組合，大家都知道嗎？

不知道的話，就趕緊洗耳恭聽吧！

一、超大觸控，高效鍵盤，跟鼠標說再見

華為MateBook X Pro 大幅度加大多手勢觸控板操作面積，帶來更加廣闊的觸控板操作區域，減少束縛感，提高精度，優化防誤觸體驗。

同時， 華為MateBook X Pro進一步優化鍵盤鍵程和鍵帽尺寸，帶來更愉悅的鍵盤敲擊體驗。全新的全尺寸鍵盤可以實現防潑濺，如果不小心把水灑在鍵盤上了，關機後將鍵盤上的水倒出擦乾靜置些許時間就可以正常使用。

二、華為設計師腦洞大開，攝像頭變按鍵

華為MateBook X Pro鍵盤與此前上市的 MateBook 系列相比，最大的亮點就是隱藏式攝像頭。筆記本的攝像頭一般都會內置於屏幕上，但是華為MateBook X Pro 的設計師卻另闢蹊徑，將攝像頭移到了鍵盤上。這樣的設計，不僅讓人感覺安全無憂，同時也提高了屏幕的屏佔比。

使用攝像頭時，按一下攝像頭就會彈起，再按一下攝像頭就隱藏起來。

三、熱鍵優先，多種設置一鍵搞定

HUAWEI MateBook X Pro 鍵盤上的F1-F12鍵默認是熱鍵優先，只需按一個鍵，就可以執行對應的功能，有效提高辦公效率。比如在熱鍵優先狀態下，按F1鍵是降低屏幕亮度，按 F9 鍵可一鍵開啟/關閉WI-FI，按F10鍵可一鍵開啟電腦管家。

四、兩種方式，熱鍵與功能鍵可輕鬆切換

在一些場景下：比如遊戲中，要把 F1、F2等鍵作為快捷功能鍵使用；在瀏覽網頁中，要用F5鍵刷新網頁。要怎麼切換熱鍵與功能鍵呢？很簡單，有兩種方式可以選擇：

方式1：按下 Fn 鍵，當 Fn 鍵指示燈亮起，表示已將 F1、F2 等鍵鎖定為功能鍵模式。這時候，就可以將F1、F2作為功能鍵使用。想要切換回熱鍵優先，只需再次按 Fn 鍵， Fn 鍵指示燈熄滅表示已經為默認的熱鍵優先。

方式2：通過華為電腦管家設置“功能鍵優先”。打開華為電腦管家，點擊右上角的設置圖標，選擇“設置中心 > 功能鍵”，勾選“功能鍵優先”後，F1、F2 等鍵將作為功能鍵使用；如要切換回默認狀態，在這個界面勾選“熱鍵優先”即可。

五、快捷鍵組合，實用便捷姿勢又帥

NO.1 Win + D

上班時，稿子憋不出來，放鬆一下心情說不定不錯。找個段子看看，找首歌聽聽……結果眼睛一撇，老大向你走過來了！腫麼辦？

別緊張，按下Win + D 快捷鍵組合，秒回你一本正經的桌面。（真實目的是教你怎麼在打開一堆應用時，快速回到桌面滴~）

老闆走了之後，想切回剛才的網頁？再按一次 Win + D 就可以了！

NO.2 Win + L

上班的時候需要臨時離開座位，又不想別人看到你的電腦屏幕，怎麼辦？按下Win + L這個組合鍵，電腦即刻鎖屏。但是記住哦，電腦得先設好密碼，要不然鎖了也白鎖。

NO.3 Win + E

打開了一堆應用，又需要打開儲存在“我的電腦”中的文件怎麼辦？按下 Win + E ，就能秒出“我的電腦”界面，有沒有覺得自己姿勢超帥~

NO.4 Ctrl + N

在網上搜到很多有用的資料，為了方便日後查看，分文檔儲存下來比較好。怎麼新建文檔？教你個簡單辦法，在打開 Word 文檔時，按Ctrl + N，秒秒鐘快速新建多個文檔！

NO.5 Ctrl + S

嘔心瀝血寫的文案、做的PPT，按Ctrl + S就能一鍵保存，無需再從鍵盤切換到鼠標去點“保存”按鈕。

NO.6 Ctrl + Shift + N

學會了新建文檔，新建一個文件夾還會難嗎？答案是不難，按下組合鍵“Ctrl + Shift + N”立即出現一個新建的文件夾，並且文件夾名稱框處於激活狀態，輸入你想設置的名字就OK了。

NO.7 Ctrl + Shift + Esc

整個電腦整個卡死怎麼辦？同時按下 Ctrl + Shift + Esc 就可以調出任務管理器，然後再把不需要的程序關掉就好！

NO.8 Shift + 任意字母

這個快捷方式可以快速切換大小寫，同時按下 Shift + 任意字母，輸出的是與當前大小寫輸入狀態相反的字母。例如小寫狀態下，按下Shift + m ，就會輸出大寫的“M”，這個功能在輸入有大寫字母的英文單詞或句子時，是不是特別方便呢？

NO.9 Home、End、Page Up、Page Down

用 Word寫東西或者是敲代碼時，熟練使用 Home、End、Page Up、Page Down 鍵，事半功倍。但 MateBook X Pro 沒有這幾個鍵怎麼辦？同時按下 Fn + 上/下/左/右方向鍵，也可以有這四個按鍵的作用啦！

Fn + 左方向鍵：跳轉至行首/頂端；

Fn + 右方向鍵：跳轉至行末/底部；

Fn + 上方向鍵：向上翻頁；

Fn + 下方向鍵：向下翻頁。

NO.10 Ctrl + P

日常工作，大家通常有打印文檔的需求。有沒有打印的快捷操作呢？有，按下Ctrl + P，然後回車就可以了。如果要打印多份呢？Ctrl + P，輸入數字（要打印的份數），然後按回車鍵就好了。

NO.11 Ctrl + W/Alt+F4

工作很賣力，不知不覺開了N多網頁，怎麼關閉？

方法1：在打開的網頁中，按Ctrl + W，可以關閉當前網頁；按住Ctrl鍵不鬆，連接點擊W鍵，頁面就能連續關閉。

方法2：在打開的網頁中，按Alt + F4組合鍵，一下全關閉。不過使用此組合鍵要注意：有的瀏覽器不給提示的情況下所有網頁就都關閉了；有的瀏覽器會彈出一個提示對話框，詢問你是否關閉所有選項卡。

掌握熱鍵/功能鍵的切換、快捷鍵組合的操作，辦公效率提升5倍！5倍！5倍！
