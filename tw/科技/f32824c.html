「操作技巧」工具提示在不同條件顯示不同顏色

2020-10-29
| 科技

前幾天有一位用戶問我：“如何讓工具提示在不同條件顯示不同顏色？”。

今天我們就拿 Tableau內置的數據源示例-超市舉例：

當子類別的利潤≥0，工具提示中的利潤為綠色；

當子類別的利潤<0，工具提示中的利潤為紅色。

效果圖如下：

操作核心

創建兩個計算字段，讓它們在不同條件下分別顯示。來跟著我一步步操作吧~

操作步驟

第一步：製作柱狀圖

將字段[子類別]放到列，將字段[利潤]求和放到行。

第二步：創建兩個計算字段

第1個名稱：利潤工具提示正

代碼如下

IF SUM([利潤]) >= 0

THEN SUM([利潤])

ELSE NULL

END

第2個名稱：利潤工具提示負

代碼如下

IF SUM([利潤]) < 0

THEN SUM([利潤])

ELSE NULL

END

第三步：修改工具提示

1.將上述創建的兩個計算字段，分別拖入標記卡中的“工具提示”

2.再點擊工具提示修改文字，修改樣式如下所示：

3. 把鼠標放到標記上，彈出的工具提示已經能自己變色了哦~
