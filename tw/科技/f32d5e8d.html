新一代超廣神鏡 適馬14-24mm F2.8全面評測

2020-11-14
| 科技

十多年前，尼康發佈了一代神鏡——AF-S 14-24mm f/2.8G ED，在尼康新一代14-24mm久久不來之際，我們終於再度見證了另一支“大燈泡”的降臨，適馬於今年2月份，正式發佈了其新款Art系列超廣角變焦鏡頭14-24mm f/2.8 HSM Art，畢竟尼康14-24mm F2.8算是聲名遠揚，用過的人都說好，這也讓我們對適馬的新一代超廣角變焦14-24mm f/2.8 HSM Art充滿了興趣。

自第一款適馬ART系列鏡頭上市之後就廣受好評，適馬該系列的大部分鏡頭都擁有非常具備競爭力的價格，並且在性能方面也足以與同焦段原廠鏡頭一戰。作為適馬ART系列的新一代廣角變焦鏡頭，適馬14-24mm f/2.8 HSM Art鏡頭不但採用了f/2.8的恆定大光圈設計，還使用了更為複雜的11組17片光學結構，其中包含多達3片FLD鏡片、3片SLD鏡片以及3片非球面鏡片，其中一片非球面鏡片還是直徑達80mm的玻璃模鑄非球面鏡片，可謂用料十足。除此之外，新款鏡頭還內置有9片圓形光圈葉片，並採用了HSM自動對焦馬達，最近對焦距離僅為26cm。

適馬將該鏡頭售價控制在萬元內——￥8499，相較於佳能的旗艦級超廣角鏡頭EF 11-24mm F4L USM目前1.6萬元的市場價格來說，適馬14-24mm f/2.8 HSM Art的售價基本便宜了一半。雖然佳能的11-24mm具備更廣的焦段，但適馬14-24mm f/2.8擁有更大的光圈，如果兩者具備相近的光學素質，顯然適馬更具吸引力一些。而對於尼康的AF-S 14-24mm F2.8G ED來說（目前京東方面售價￥9999），適馬不僅在價格上擁有一定的優勢，經過了十年的光學發展，適馬新品顯然具備不小的競爭力。下面，我們先從外觀來看一下這款“神燈”。

鏡頭包裝盒+鏡頭+鏡頭蓋

適馬14-24mm f/2.8 HSM Art的包裝箱採用適馬一如既往的黑白設計，簡潔大氣。包裝盒白底黑條的部分寫著“14-24mm F2.8 DG”表示著包裝箱裡主人的身份，包裝箱上的銀灰色的圓形“A”標展現了其純正的ART系列鏡頭血統。

正面+鏡頭蓋

適馬14-24mm f/2.8 HSM Art裝上鏡頭蓋顯得無比的碩大，鏡身部分是一層細膩的磨砂塗層，增加手感的同時可以有效地防止指紋。為了增添鏡頭的工業藝術感，這支鏡頭還在鏡筒上以亮銀色的圓形“A”標標識ART系列鏡頭血統，鏡身後部更是採用了與眾不同的光滑漆面設計。

·鏡頭細節：彰顯適馬的用心之道

通過以上的圖片，各位已經瞭解了這支鏡頭的外觀情況，然而細節之處才最能體現一家廠商的用心之處。下面我們來從鏡頭的細節設計來看看這支適馬14-24mm f/2.8 HSM Art的表現。

適馬14-24mm f/2.8 HSM Art對焦指示窗

適馬14-24mm f/2.8 HSM Art配備了對焦距離指示窗，方便你判別被攝物體的距離，這支鏡頭的最近對焦距離約為26釐米，放大倍率為0.18倍，可以近距離拍攝以獲得極具戲劇性的畫面效果。

從我們使用佳能5D Mark III與其配合的情況來看，由於採用了HSM環形超聲波對焦馬達，該鏡頭對焦時安靜而迅捷。

寬大柔軟的變焦環阻尼調教得很舒服，筆者在變焦時可以非常精確的調整變焦位置。

適馬14-24mm f/2.8 HSM Art採用花瓣形遮光罩，內部有螺紋消光處理

採用一體式遮光罩可以遮擋多餘的光線防止發生鏡頭炫光，還可以起到保護鏡頭的作用。此次適馬14-24mm f/2.8 HSM Art還提供新的前端轉換服務，能夠轉換前端的花瓣形遮光罩部分，這種改變前端結構的技術，方便多攝像機在VR錄影時自由地部署鏡頭而不會產生暗角。

適馬14-24mm f/2.8 HSM Art鏡頭的卡口處有防塵防水的密封橡皮圈設計

由於該鏡頭採用內對焦設計，鏡頭在對焦時不會旋轉或伸出，因此用戶無需特別擔心在使用中鏡頭進灰的問題。並且適馬14-24mm f/2.8 HSM Art在手動對焦環、變焦環和AF/MF開關面板處都安裝有密封圈，具備和適馬Sprots鏡頭同等級的防護性能，可以讓攝影師在戶外糟糕的天氣環境下繼續安心工作。

適馬14-24mm f/2.8 HSM Art鏡頭蓋

適馬14-24mm f/2.8 HSM Art採用了花瓣形且帶有螺紋消光設計的遮光罩，可以很好的抑制畫面光暈、避免雜光進入鏡頭，同時還能阻擋雨雪濺落、保護相機和鏡頭免遭意外碰撞。其次巨大的鏡頭蓋也能很好的保護鏡頭，這就為適馬14-24mm f/2.8 HSM Art提供了雙重保護。

適馬14-24mm f/2.8 HSM Art搭配5000萬像素的佳能EOS 5Ds使用

適馬14-24mm f/2.8 HSM Art為滿足5000萬像素以上的機身畫質需求所設計，所以筆者在後續的拍攝中搭配了5000萬像素的佳能EOS 5Ds使用。

·鏡頭的參數解析

適馬14-24mm f/2.8 HSM Art採用11組17片光學結構，9片圓形光圈設計，其中包含3片FLD鏡片、3片SLD鏡片和3片非球面鏡片，其中更採用了一片大口徑的非球面鏡片。通過不同位置有效配置FLD和SLD這些高折射率鏡片的位置，進一步提升了FLD、SLD鏡片的矯正效果。這種光學系統使橫向色差和其他光學像差最小化，從而提高了鏡頭的圖像質量。整組鏡組最前面的鏡片採用了一片大口徑非球面鏡片。通過將這片超大非球面鏡片的威力與其他的鏡組相結合，有效地將畸變降至1%以內。

適馬14-24mm f/2.8 HSM Art鏡頭內部結構圖

適馬14-24mm f/2.8 HSM Art的MTF曲線

在沒有正式看測試前，我們可以通過官方給出的MTF曲線看出適馬14-24mm f/2.8 HSM Art的成像素質。這支鏡頭在14mm焦段時的整體成像表現要高於24mm焦段時的成像表現。尤其是10線/毫米與30線/毫米的實線已經接近於1，由此來看在14mm廣角端時，畫面的分辨率與對比度表現都非常不錯，而畫面的對比度普遍要比分辨率高一些。而在24mm焦段時，光圈收小後的整體表現要比光圈全開的表現要好上一些。

·14mm端各擋光圈成像表現對比

我們搭配佳能EOS 5Ds進行成像畫質對比，使用各擋光圈逐一進行拍攝，最終截取畫面中央與邊緣的部位進行放大對比。以下的兩組圖片測試均採用ISO 100進行拍攝。

圖中紅色框選區域為適馬14-24mm 12mm端畫質對比選區

通過實拍測試，適馬14-24mm f/2.8 HSM Art的14mm端在光圈全開的情況下，中心畫質表現還不錯，基本的紋理細節還可以看到，而邊緣畫質的表現則稍顯薄弱。但是光圈收一擋到F4的時候，照片邊緣的畫質表現有了非常明顯的提升，而中心畫質也得到進一步提升。在光圈F4-F11之間，適馬14-24mm f/2.8 HSM Art在14mm焦段下的分辨率與銳度表現都達到了極致，光圈縮小到F22時，照片的成像出現了因小光圈衍射而導致畫質下降的現象，可見最小光圈要比最大光圈下的成像要差一些。

·24mm端各擋光圈成像表現對比

下面我們來看看適馬14-24mm f/2.8 HSM Art這支鏡頭在24mm焦段下的各擋光圈成像表現對比，同樣採用高像素的佳能EOS 5Ds作為測試機身，使用ISO 100進行拍攝，截取畫面中心和邊緣部位進行100%放大進行觀察。

圖中藍色框選區域為適馬14-24mm 24mm焦段下畫質對比選區

通過實拍測試我們可以明顯發現，適馬14-24mm f/2.8 HSM Art在24mm端的成像表現非常優秀，畫面中心與邊緣的銳度表現都非常好，並且沒有明顯的起伏變化，當然也不可避免地在F16小光圈以後，出現了邊緣畫質下降的狀況。

·各擋光圈星芒表現

在攝影創作的過程中，不通過添加濾鏡就可以獲得星芒效果最簡單直接的方法就是縮小光圈，而星芒效果的形狀和光圈的葉片數量也有一定的關係，適馬14-24mm f/2.8 HSM Art這支鏡頭採用9片圓形光圈，因此可以獲得好看的18星效果。筆者在使用適馬這支鏡頭時支上了三腳架，採用各擋光圈逐一進行拍攝，以下是不同光圈下的星芒效果圖。

·14mm和24mm端各擋光圈色散情況表現

官方資料表示，適馬14-24mm f/2.8 HSM Art鏡頭包含三片FLD鏡片和三片SLD鏡片，通過有效配置FLD和SLD這些高折射率鏡片的位置，進一步提升FLD和SLD鏡片消除倍率色差，即高反差邊緣的紫邊。我們採用各擋光圈分別拍攝高反差的樹枝，通過100%放大截圖來觀察這支鏡頭抑制色散的能力。

圖中紅色框選區域為14mm焦段的色散對比選區

圖中紅色框選區域為24mm焦段的色散對比選區

筆者在做適馬14-24mm f/2.8 HSM Art這支鏡頭的色散測試時，找了好幾組高反差的位置進行拍攝，拍攝完在相機內進行放大觀察，均未發現有明顯的色散表現，當時真的感嘆適馬14-24mm f/2.8 HSM Art這支鏡頭包含的三片FLD鏡片和三片SLD鏡片發揮的作用十分明顯，將高反差邊緣的紫邊幾乎全都消除了。

以上這組框選出來的選區，是出現了紫邊現象的位置，不過也只在14mm的焦段下開到大光圈時有一道紫邊的存在，當光圈收到F5.6時，這道紫邊就不見了。而在24mm焦段時，筆者發現確實沒有找到紫邊的存在，由此可見適馬14-24mm f/2.8 HSM Art鏡頭對於色散出色的控制能力。

·鏡頭畸變控制表現

畸變之於廣角鏡頭，顯然要比標準鏡頭和長焦鏡頭更為重要，畸變控制得好意味著後期調整變形所損失的視角也就更小。適馬14-24mm f/2.8 HSM Art的鏡片設計中包含三片非球面鏡片，整個鏡組最前面的鏡片更是採用了一片大口徑非球面鏡片。適馬官方宣稱，通過將這片超大非球面鏡片的威力與其他的鏡組結合，適馬14-24mm f/2.8 HSM Art有效地將畸變降至1%甚至更低。

下面我們來看一下適馬14-24mm f/2.8 HSM Art在14mm和24mm兩端的畸變表現。需要說明的是在進行畸變測試時，我們關閉了機內的畸變矯正功能。

從上面兩張拍攝磚牆的照片來看，除了14mm焦段有輕微的桶形畸變以外，24mm焦段的畸變表現，肉眼已經看不出任何畸變了（拍攝時沒做到完全正面，不過仰角帶來的變化也很有層次）。由此可見三片非球面鏡片的配合使用，對於像差的控制能力表現果然有效。可以說適馬14-24mm f/2.8 HSM Art確實如官方在宣傳時所言：“此次帶來的超廣角變焦鏡頭在畸變控制方面做到了14mm端畸變小於1%，近乎零畸變的水準。”而且該鏡頭安裝在佳能機身上能夠識別鏡頭相差矯正功能，進一步提高圖像品質。相信在開啟了機內鏡頭矯正之後，適馬14-24mm f/2.8 HSM Art還能有更優秀的表現。

·鏡頭炫光控制表現

之前提到過適馬14-24mm f/2.8 HSM Art這支超廣角鏡頭使用了三片FLD鏡片和三片SLD鏡片鏡片，並且整個鏡組最前面採用了一片大口徑非球面鏡片，從設計階段就開始進行的模擬改善，幫助有效地減少了眩光和鬼影。不過在實際的拍攝過程中，筆者發現在一定場景出現鏡頭炫光的機率還蠻高的，當然這種情況也在所難免。

圖中紅色框選區域為適馬14-24mm 14mm端炫光對比選區

適馬14-24mm 14mm端照片左側炫光表現

通過以上圖片框選的位置，可以明顯發現照片四周有一道炫光，照片右側的橙紅色炫光會隨著光圈的收小而減弱，但一直到F22也未消失；而照片左側的區域也可以發現，有一道青紫色的炫光從光圈F8時開始出現並一直擴大，到光圈F16時最明顯，兩者處於此消彼長的過程。

不過在使用該鏡頭正面拍攝太陽時，也並非每次都會出現明顯的炫光情況，只有在特定的角度才會出現炫光，這也是超廣角鏡頭在所難免的現象。因此，在拍攝正面的太陽畫面時，注意調整角度即可。筆者在其他場景的拍攝中，並未發現炫光，由此可見適馬14-24mm f/2.8 HSM Art的炫光控制能力還是非常不錯的。

·鏡頭暗角控制表現

對於廣角鏡頭來說，我們最為關心的除了它在最大光圈下的成像表現以外，就是其對於廣角端畫面邊緣暗角現象的控制能力了。下面筆者通過一組藍天的拍攝，來表現適馬14-24mm f/2.8 HSM Art這支鏡頭在14mm和24mm端對於暗角的控制表現。

·評測總結：

一直以來在14-24mm這個焦段，尼康的14-24mm f/2.8都算是名聲在外，用過的人都說好，不過該鏡頭到現在推出也將近有11年之久，很多對這個焦段有需求的用戶，一直在期待這個焦段能有全新的鏡頭出現。因此，此次適馬推出同類型的鏡頭——14-24mm f/2.8 HSM Art，可以說對大家都非常具有吸引力。適馬作為一個以“黑科技”著稱的光學廠商，一直善於高銳度的鏡頭設計。不過作為一支廣角端的風光鏡頭，我們還是很關心除銳度以外的色彩、畸變以及抗眩光能力。

這些方面，在測試環節還有樣片展示中，我們已經看到了適馬14-24mm f/2.8 HSM Art這支超廣角變焦鏡頭在各方面都是可圈可點的。成像色彩很接近於肉眼在現場看到的畫面；畸變控制方面做到了14mm端近乎“零畸變”的水準；抗眩光方面，除了在直射陽光且特定角度的情況下不可避免地出現以外，其餘場景再難出現；色散的控制方面也是相當強，拍攝高反差場景時完全不用擔心色散了。並且這次適馬14-24mm f/2.8 HSM Art的主打高畫質的，筆者在拍攝的過程中感受頗深。

價格方面，適馬將該鏡頭售價控制在萬元內——￥8499，相較於同類產品佳能的EF 11-24mm F4L USM近兩萬塊的市場價格來說，雖然佳能的11-24mm具備更廣的焦段，但適馬14-24mm f/2.8擁有更大的光圈。通過前面的測試，適馬甚至還在畸變、暗角、色散控制等多個方面超越了對方，並且還便宜了將近一萬塊的價格，顯然適馬14-24mm會更具吸引力。而對於尼康的AF-S 14-24mm F2.8G ED來說（目前京東方面售價￥9999），適馬不僅在價格上有一定優勢，適馬新品顯然具備相當的競爭力。

總之，就目前適馬14-24mm f/2.8 HSM Art的表現來看，對廣角鏡頭有需求的用戶來說，它的確是一款非常值得推薦的超廣角變焦鏡頭。
