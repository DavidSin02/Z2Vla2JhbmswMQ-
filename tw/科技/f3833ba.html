卸料平臺施工方案

2020-10-29
| 科技

二、編制依據

1、《建築施工高空作業安全技術規範》

2、《建築機械使用安全技術規程》

3、《建築施工安全檢查標準》

4、《建築施工手冊》

三、平臺搭設位置及安裝順序

3.1平臺平臺體系選用與性能

1）、本工程簷口高度為22.8m，根據腳手架的搭設高度限制和卸載規定，採用懸挑式卸料平臺可不受限制。

2）、卸料平臺適用性強，配合塔吊使用，在鋼筋混凝土適當位置預埋吊環，採用鋼絲繩與結構栓拉結牢固，易於拆卸、堅固耐用可連續週轉使用。

3.2安裝順序

平臺吊裝到位→平臺與建築物固定→鋼絲繩穿吊環→ 吊環與鋼絲繩固定→ 自檢→加荷實驗→驗收 →使用

每天使用之前均需對所有部位進行檢驗，確認無誤，方可使用。

四、材料選擇

1）鋼材：主次樑均為普通碳素鋼，並採用E43焊條焊接，主樑為22#槽鋼，次樑為18a槽鋼。

2）牽引鋼絲繩為：直徑18鋼絲繩端頭設4到卡環，並在第三道卡環與第四道卡環之間設安全彎。

3）腳手板：平臺腳手板及180mm擋腳板均採用50厚木腳手板。

4）防護欄杆：採用腳手架鋼管搭設。

5）所有材料均為合格產品，強度滿足規定數據。

4.1卸料鋼平臺的安裝

當樑混凝土達到70%後方可吊掛鋼絲繩及固定平臺。平臺吊裝，需待橫樑支撐點牢固，接好鋼絲繩，調整完畢，經過檢查驗收，方可鬆卸起重吊環，上下操作。

4.2卸料鋼平臺的設置要求

新設計組裝或加工的卸料鋼平臺，在使用前應進行不低於使用施工荷載的靜載試驗和起吊試驗，試驗合格（未發現焊縫開裂、結構變形等情況）後方能投入使用。

五、安全要求

1、周圍臨邊三面用鋼管設置1.5米高護欄，護欄立杆與主次樑焊牢。護欄下設180mm高木擋腳板。

2、在鋼平臺顯著位置標明容許載重量、管理者標牌，標誌牌應可靠的固定在欄杆上。

3、鋼平臺內物料長、寬、高不得超過防護欄杆所限範圍，任何物料嚴禁放在防護欄杆上進行吊運，材料堆放對稱分散，嚴禁集中於一側堆放，堆放材料應儘快吊運下樓或運至樓內，嚴禁材料在平臺內堆放過夜。

4、鋼平臺使用時，應設專人對平臺進行檢查監督，發現鋼絲繩鏽蝕損壞應及時調換，焊縫脫焊應及時修復，發現問題時及時整改。

5、在使用過程中不允許拆除鋼平臺上的任何部件。

6、每日班前須對卸料鋼平臺進行檢查驗收合格後方可投入使用。

7、認真執行有關的吊裝及鋼結構焊接的有關規定。

六、計算書如下：

由於卸料平臺的懸挑長度和所受荷載都很大，因此必須嚴格地進行設計和驗算。

懸挑卸料平臺的計算依據《建築地基基礎設計規範》(GB 50007-2002)、《建築結構荷載規範》(GB 50009-2001)、《鋼結構設計規範》(GB 50017-2003)等編制。

(一)、參數信息:

1.荷載參數

腳手板類別：木腳手板，腳手板自重(kN/m2)：0.35；

欄杆、擋板類別：木腳手板擋板，欄杆、擋板腳手板自重(kN/m)：0.14；

施工人員等活荷載(kN/m2):2.00，最大堆放材料荷載(kN):2.00。

2.懸挑參數

內側鋼繩與牆的距離(m)：1.00，外側鋼繩與內側鋼繩之間的距離(m)：2.50；

上部拉繩點與懸挑樑牆支點的距離(m)：6.60；

鋼絲繩安全係數K：5.50，懸挑樑與牆的節點按 鉸支 計算；

只對外側鋼繩進行計算；內側鋼繩只是起到保險作用，不進行計算。

3.水平支撐樑

主樑材料類型及型號：22號槽鋼槽口豎向U ；

次樑材料類型及型號：18a號槽鋼槽口豎向U；

次樑水平間距ld(m)：1.00，建築物與次樑的最大允許距離le(m)：1.20。

4.卸料平臺參數

水平鋼樑(主樑)的懸挑長度(m)：4.20，水平鋼樑(主樑)的錨固長度(m)：1.80, 次樑懸臂Mc（m）：0.00；

平臺計算寬度(m)：1.80。

(二)、次樑的驗算:

次樑選擇 18a號槽鋼槽口豎向U ，間距1m，其截面特性為：

面積 A=25.69cm2；

慣性距 Iy=98.6cm4；

轉動慣量 Wy=20.03cm3；

回轉半徑 iy=1.96cm；

截面尺寸：b=68mm,h=180mm,t=10.5mm。

1.荷載計算

(1)、腳手板的自重標準值：本例採用木腳手板，標準值為0.35kN/m2；

Q1=0.35× 1.00= 0.35kN/m；

(2)、型鋼自重標準值：本例採用18a號槽鋼槽口豎向U，標準值為0.20 kN/m

Q2=0.20 kN/m

(3)、活荷載計算

1)施工荷載標準值：取2.00 kN/m2

Q3=2.00 kN/m2

2)最大堆放材料荷載P：2.00kN

荷載組合

Q=1.2×(0.35+0.20)+1.4×2.00×1.00=3.46kN/m

P=1.4×2.00=2.80kN

2.內力驗算

內力按照集中荷載P與均布荷載q作用下的簡支樑計算，計算簡圖如下：

最大彎矩M的計算公式（規範JGJ80-91，P31）為：

Mmax = ql2/8(1 - m2/l2)2 + pl/4

經計算得出: Mmax = (3.46×1.802/8)×(1-(0.002/1.802))2+2.80×1.80/4=2.66kN·m。

最大支座力計算公式:

R = [P + q(l+2m)]/2

經計算得出: R = (2.80 + 3.46×(1.80 + 2×0.00))/2 = 4.51kN

3.抗彎強度驗算

次樑應力：

σ = M/γxWx ≤ [f]

其中 γx -- 截面塑性發展係數，取1.05；

[f] -- 鋼材的抗壓強度設計值，[f] = 205.00 N/mm2；

次樑槽鋼的最大應力計算值 σ =2.66×103/(1.05×20.03)=126.48 N/mm2；

次樑槽鋼的最大應力計算值 σ =126.485 N/mm2 小於 次樑槽鋼的抗壓強度設計值 [f]=205 N/mm2,滿足要求!

4.整體穩定性驗算

σ = M/φbWx ≤ [f]

其中，φb -- 均勻彎曲的受彎構件整體穩定係數，按照下式計算：

φb = (570tb/lh)×(235/fy)

經過計算得到 φb=570×10.50×68.00×235/(1800.00×180.00×235.0)=1.26；

由於 φb大於0.6，按照下面公式調整：

φb' = 1.07 - 0.282/φb ≤1.0

得到 φb' = 0.845；

次樑槽鋼的穩定性驗算 σ =2.66×103/(0.845×20.030)=157.08 N/mm2；

次樑槽鋼的穩定性驗算 σ =157.078 N/mm2 小於 次樑槽鋼的抗壓強度設計值 [f]=205 N/mm2,滿足要求!

(三)、主樑的驗算:

根據現場實際情況和一般做法，卸料平臺的內鋼繩作為安全儲備不參與內力的計算。

主樑選擇 22號槽鋼槽口豎向U ，其截面特性為：

面積 A=36.24cm2；

慣性距 Iy=176.4cm4；

轉動慣量 Wy=30.05cm3；

回轉半徑 iy=2.21cm；

截面尺寸，b=79mm,h=220mm,t=11.5mm；

1.荷載驗算

(1)欄杆與擋腳手板自重標準值：本例採用木腳手板擋板，標準值為0.14kN/m；

Q1 = 0.14kN/m；

(2)槽鋼自重荷載 Q2=0.28kN/m

靜荷載設計值 q = 1.2×(Q1+Q2) = 1.2×(0.14+0.28) = 0.50kN/m；

次樑傳遞的集中荷載取次樑支座力 R；

2.內力驗算

懸挑卸料平臺示意圖

懸挑卸料平臺水平鋼樑計算簡圖

懸挑水平鋼樑支撐樑剪力圖(kN)

懸挑水平鋼樑支撐樑彎矩圖(kN·m)

懸挑水平鋼樑支撐樑變形圖(mm)

從左至右各支座反力：

R[1] = 8.95 kN；

R[2] = 6.512 kN；

R[3] = -1.162 kN。

最大支座反力為 Rmax=6.512 kN；

最大彎矩 Mmax=3.933 kN·m；

最大撓度 ν=0.099 mm。

3.抗彎強度驗算

σ = M/（γxWx） + N/A ≤ [f]

其中 γx -- 截面塑性發展係數，取1.05；

[f] -- 鋼材抗壓強度設計值，[f] = 205.00 N/mm2；

主樑槽鋼的最大應力計算值 σ =3.933×106/1.05/30050.0+6.91×103/3624.000=126.560N/mm2；

主樑槽鋼的最大應力計算值 126.560 N/mm2 小於 主樑槽鋼的抗壓強度設計值 [f]=205.00 N/mm2，滿足要求!

4.整體穩定性驗算

σ = M/（φbWx） ≤ [f]

其中 φb -- 均勻彎曲的受彎構件整體穩定係數，按照下式計算：

φb=（570tb/lh）×（235/fy）

φb=570×11.5×79.0×235/(3500.0×220.0×235.0)=0.673；

由於 φb大於0.6，應按照下面公式調整：

φb'=1.07-0.282/φb ≤1.0

可得 φb'=0.651；

主樑槽鋼的穩定性驗算 σ = 3.933×106/(0.651×30050.00)=201.15 N/mm2；

主樑槽鋼的穩定性驗算 σ = 201.15 N/mm2 小於 [f]=205.00,滿足要求!

(四)、鋼絲拉繩的內力驗算:

水平鋼樑的垂直支坐反力RCi和拉鋼繩的軸力RUi按照下面計算，

RCi = RUisinθi

其中 RCi -- 水平鋼樑的垂直支坐反力(kN)；

RUi -- 拉鋼繩的軸力(kN)；

θi -- 拉鋼繩的軸力與水平鋼樑的垂直支坐反力的夾角；

sinθi = Sin (ArcTan (3.3/(2.5+1)) = 0.686；

根據以上公式計算得到外鋼繩的拉力為：RUi = RCi / sinθi；

RUi = 8.95 / 0.686 = 13.05 kN；

(五)、鋼絲拉繩的強度驗算:

選擇6×19鋼絲繩,鋼絲繩公稱抗拉強度1550MPa，直徑18.5mm。

[Fg] = aFg/K

其中[Fg]-- 鋼絲繩的容許拉力(kN)；

Fg -- 鋼絲繩的鋼絲破斷拉力總和(kN)，查表得Fg＝199.5KN；

α -- 鋼絲繩之間的荷載不均勻係數，對6×19、6×37、6×61鋼絲繩分別取0.85、0.82和0.8。α＝0.85；

K -- 鋼絲繩使用安全係數。K＝5.5。

得到：[Fg]=30.832KN>Ru＝13.046KN。

經計算，選此型號鋼絲繩能夠滿足要求。

(六)、鋼絲拉繩拉環的強度驗算:

取鋼絲拉繩(斜拉桿)的軸力最大值RU進行計算作為拉環的拉力N為：

N=RU=13045.768N。

拉環強度計算公式為：

σ = N/A ≤ [f]

其中， [f]為拉環鋼筋抗拉強度，按照《混凝土結構設計規範》10.9.8所述在物件的自重標準值作用下，每個拉環按2個截面計算的。拉環的應力不應大於50N/mm2，故拉環鋼筋的抗拉強度設計值[f]=50.0N/mm2；

所需要的拉環最小直徑 D=[13045.8×4/(3.142×50.00×2)]1/2=12.9mm。

實際拉環選用直徑D=14mm的HPB235的鋼筋製作即可。

(七)、操作平臺安全要求:

1.卸料平臺的上部拉結點，必須設於建築物上，不得設置在腳手架等施工設備上；

2.卸料平臺安裝時，鋼絲繩應採用專用的掛鉤掛牢，建築物銳角口圍系鋼絲繩處應加補軟墊物，平臺外口應略高於內口；

3.卸料平臺左右兩側必須裝置固定的防護欄；

4.卸料平臺吊裝，需要橫樑支撐點電焊固定，接好鋼絲繩，經過檢驗後才能鬆卸起重吊鉤；

5.卸料平臺使用時，應有專人負責檢查，發現鋼絲繩有鏽蝕損壞應及時調換，焊縫脫焊應及時修復；

6.操作平臺上應顯著標明容許荷載，人員和物料總重量嚴禁超過設計容許荷載，配專人監督。
