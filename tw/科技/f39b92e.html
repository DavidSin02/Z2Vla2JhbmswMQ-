第一節 績效獎勵的基本原理

一、績效及其影響因素

個人績效是為員工通過努力所達成的對企業有價值的結果，以及他們在工作過程中所表現出來的符合企業的文化和價值觀，同時有利於企業戰略目標實現的行為。

影響績效的因素：

1、員工的知識

2、員工的能力

3、員工的工作動機

4、機會

二、激勵理論及其對績效獎勵的啟示

對絕大多數員工來說，個人績效與薪酬之間聯繫最有力，也是最直接的一種激勵手段。如何公平合理對員工績效提供報酬是企業薪酬管理中必須關注的一個非常重要的問題。

1、馬斯洛的需要層次論：馬斯洛的需求層次論認為，人的行為是受到人的內在需要激勵的。

2、赫茲伯格的雙因素理論：赫茲伯格的雙因素理論認為，員工的行為會受到保健因素和激勵因素兩種不同因素的影響。

3、期望理論:維克多·弗洛姆(Victor Vmom)提出，績效是三大知覺的函數：期望、關聯性以及效價。

4、公平理論：公平理論認為，員工不僅關心自己經過努力所獲得的報酬的絕對數量，也關心自己的報酬和其他人報酬之間的關係。

5、強化理論：強化理論認為，一個人的行為是受其目標引導的，如果員工的某種行為得到了與預期目標相符的某種報酬的強化，則員工重複性地執行相同行為的可能性會增加。

6、目標設置理論：目標設置理論主要探討了目標的具體性、挑戰性以及績效反饋對於績效的影響。

7、委託代理理論：產生風險的原因是信息不對稱和控制行為要花費成本。一個最佳選擇是不去判斷其能力和監控其行為，選擇一種使二者利益一致的契約。

三、績效獎勵計劃的特點及其實施要點

(一)績效獎勵計劃的概念及其優缺點

所謂績效獎勵計劃，是指員工的薪酬隨著個人、團隊或者組織績效的某些衡量指標所發生的變化而變化的一種薪酬設計。

優點：

1、有利於組織通過靈活調整員工的工作行為來達成企業的重要目標，從而避免員工的行為脫離組織的戰略主線而形成本位主義傾向。

2、 有利於組織根據自身的經營狀況靈活調整自己的支付水平，而不至於因為成本的壓力而陷入困境績效獎勵計劃有利於組織總體績效水平的改善 。

缺點：

1、在產出標準不公正的情況下，績效獎勵計劃很可能會流於形式。

2、績效獎勵計劃有可能導致員工之間或者使員工群體之間的競爭，而這種競爭可能不利於組織的總體利益。

3、在績效獎勵計劃的設計和執行過程中還有可能增加管理層和員工之間產生摩擦的機會。

4、有時員工收入的增加會導致企業出臺更為苛刻的產出標準，這樣就會破壞企業和員工之間的心理契約。

5、績效獎勵公式有些時候非常複雜，員工可能難以理解。

(二)績效獎勵計劃的實施要點

企業必須認識到，績效獎勵計劃只是企業整體薪酬體系中的 一個重要組成部分，但是它不能取代其他薪酬計劃。

績效獎勵計劃必須對那些圓滿完成組織績效或行為與組織目標一致的員工給予回報，而組織目標通常是和企業的戰略經營計劃和組織任務聯繫在一起的。

1、企業必須首先建立起有效的績效管理體系。

2、必須在績效和獎勵之間建立起緊密的聯繫。

3、績效獎勵計劃必須獲得有效溝通戰略的支持。

4、績效獎勵計劃需要保持一定的動態性。

第二節 績效獎勵計劃的種類

從時間維度來看，分為長期激勵計劃和短期激勵計劃；從激勵對象維度來看，分為個體激勵計劃和群體激勵計劃。

一、短期績效獎勵計劃

(一)績效加薪

績效加薪是將基本薪酬的增加與員工在某種績效評價體系中所獲得的評價等級聯繫在一起的一種績效獎勵計劃。

績效加薪計劃的三大關鍵要素：加薪的幅度、加薪的時間以及加薪的實施方式。

在實施績效加薪計劃時，通常會用到一種簡單易行的薪酬管理工具——績效加薪矩陣。

(二)一次性獎金

一次性獎金是一種一次性支付的績效加薪。

對組織的優勢：它在保持績效和薪酬掛鉤的情況下減少了因基本薪酬的累加效應所引起的固定薪酬成本增加；它可以保障組織各等級薪酬範圍的“神聖性”；具有極大的靈活性。

對員工來說，雖然一次可以拿到很多獎金，但長期來看，收入減少。

(三)月／季度浮動薪酬

是根據月或季度績效評價結果，以月績效獎金或季度績效獎金的形式對員工的業績加以認可。

企業通常可以採取以下幾個公式來計算員工個人應得的月或季度績效獎金金額(以下公式以企業採取季度考核為例)。

1、部門間季度績效工資平均單價的計算：

部門間季度績效工資平均單價 =公司季度績效工資基準額/∑（部門季度績效工資基準額×部門季度績效評價係數）

2、各部門應得季度績效工資總額的計算：

部門應得季度績效工資總額=部門季度績效工資基準額×本部門季度績效評價係數×部門間季度績效工資平均單價

3、 部門內季度績效工資平均單價的計算：

部門內季度績效工資平均單價=本部門應得季度績效工作總額 /∑(員工個人季度績效工資基準額×個人季度績效評價係數)

4、 員工實際應得季度績效工資的計算:

員工實際應得季度績效工資=員工季度績效工資基準額×個人季度績效評價係數×部門內季度績效工資平均單價

二、長期績效獎勵計劃

長期績效獎勵計劃與股票所有權計劃

長期獎勵計劃是指績效衡量週期在一年以上的對既定績效目標的達成提供獎勵(主要以股票的形式)的計劃。

1、長期績效獎勵計劃的特點

長期獎勵計劃的支付通常是以3年～5年為一個週期。長期獎勵計劃強調長期規劃和對組織的未來可能產生影響的那些決策,它能夠創造一種所有者意識,有助於企業招募、保留和激勵高績效的員工,從而為企業的長期資本積累打下良好的基礎。

2、股票所有權計劃

股票所有權計劃,實際上是指企業以股票為媒介所實施的一種長期績效獎勵計劃。傳統主要針對中高層管理人員，現在有向普通員工擴展趨勢。

常見的股票所有權計劃可以劃分為三類：現股計劃、期股計劃以及期權計劃。

2.1 現股計劃,是指通過公司獎勵的方式直接贈與，或者是參照股權的當前市場價值向員工出售股票，總之是使員工立即直接獲得實實在在的股權,但這種計劃同時會規定員工在一定的時期內必須持有股票,不得出售。

2.2 期股計劃則規定公司和員工約定在將來某一時期內以一定的價格購買一定數量的公司股權,購股價格一般參照股權的當前價格確定,該計劃同時也會對員工在購股後出售股票的期限作出規定。

2.3 期權計劃指公司給予員工在將來某一時期內以一定價格購買一定數量公司股權的權利,但員工到期時可以行使這種權利,也可以放棄這種權利,購股價格一般參照股權的當前市場價格確定,該計劃也同樣要對員工購股之後出售股票的期限作出規定。

3.股票期權計劃

所有者給以經營者（管理人員）的一種權利，允許其在特定時期內按照某一預定價格（行權價）購買本企業的股票若干股，權利不能轉讓，但所購股票可以在市場上出售。目的是激勵中高層管理人員努力工作，不斷改善經營業績，從而提高股票價格，達到公司與管理人員雙贏結果。

只是一種預期，並不需要“現金支付” ，授予股票時候還可以考慮當年績效或其它一些因素，實行浮動授予。

期權授予後一年之內，不得行使該權利，第2-4年可部分行使，有效期一般8年，最長10年，可以選在在10年中任何時候行權（實際購買股票時候叫行權），中途離開就會喪失剩餘期權。行權之後期權變成了股權，但超過10年自動作廢。

存在問題：

１、過分關注股票價格而採取一些不道德甚至違法行為。

２、股票價格上升不一定是高管個人能力和努力結果，與很多因素有關。

4、員工持股計劃

股票期權計劃主要針對高管人員，適用對象更廣泛的股票所有權計劃則是員工持股計劃。

優點：

（1）稅收和財務方面優惠；

（2）有利於穩定員工隊伍，留住人才，提高員工對組織的忠誠度；

（3）關心企業的經營成敗，有利於增強所有者意識，培養他們的自豪感；還能起到防止其他公司惡意收購或兼併的作用。

存在問題：

經濟學看，不可避免出現激勵不足和“搭便車”問題。股票業績的衡量非常複雜，與很多因素有關，且員工個人的工作成果與企業最終業績之間關心很不緊密；其次，為員工帶來利益的同時，也使得他們承受了一定風險。

三、個人績效獎勵計劃

(一)個人獎勵計劃的內涵及其適用條件

個人績效獎勵計劃，就是指針對員工個人的工作績效提供獎勵的一種報酬計劃。 企業如果想實施個人績效獎勵計劃，就必須具備下面幾個方面的條件。

1、從工作角度來看,員工個人的工作任務完成不取決於其他人的績效,即其他員工的工作狀況不會對被個人獎勵計劃覆蓋的員工的績效產生影響,組織對於員工個人的績效還必須能夠準確地加以衡量。

2、從組織狀況來看，企業所處的經營環境以及所採用的生產方法以及資本一勞動力要素組合必須是相對穩定的。

3、從管理方面來看,企業必須在整體的人力資源管理制度上強調員工個人的專業性,強調員工個人的優良績效。

(二)個人績效獎勵計劃的種類

1、直接計件工資計劃

（1）這是運用最為廣泛的一種獎勵計劃。薪酬直接根據產出水平而發生變化。

（2）確定在一定時間內(比如1小時)應當生產出的標準產出數量，然後以單位產出數量確定單 位時間工資率，最後根據實際產出水平算出實際應得薪酬。

2、標準工時計劃

所謂標準工時計劃，是指首先確定正常技術水平的工人完成某種工作任務所需要的時間，然後再確定完成這種工作任務的標準工資率。

標準工時計劃的變種——貝多計劃：

直接計件與標準工時相結合的方式 ，指的是按照中等技術熟練工人的標準定標準工時定額。同時，對於高績效員工實施高激勵，如果小於標準工時，能得到實際工時與標準工時差額的一個正比例函數的報酬。

W=r.S（T>S）

W=r.S+q(S-T)（T<=S）

r—標準小時工資 S—標準單位產量耗時

q—激勵強度

3、差額計件工資計劃

是直接計件工資制的另一種變體,由科學管理理論的創始人泰勒最先提出的,主要是指使用兩種 不同的計件工資率:一種適用於那些產量低於或等於預定標準 的員工的,而另一種則適用於產量高於預定標準的那些員工。

4、與標準工時相聯繫的可變計件工資計劃:

（1）海爾塞50--50計件工資計劃

企業通過時間研究確定完成某項任務的標準工作時間,如果員工以低於標準工時的時間完成工作,那麼因節約時間而產生的收益,則在企業和員工之間以對半的形式分享。

（2）羅曼計件工資計劃

它與海爾塞計劃類似的，所不同的是，隨著所節約的時間增加,員工所能夠分享的收益所佔的比例是上升的。

（3）甘特計件工資計劃

在確定標準工時的時候,有意將它定在工人需要付出較大的努力才能達到的水平上。不能在標準時間內完成工作的人將會得到一個有保證的工資率。但是對於那些能夠在標準時間內或者是少於標準工時的時間內完成工作的員工,計件工資率則定在標準工資率的120%這一較高水平上。

兩個主要變量的確定

工資率——確定方法有兩種

（1）以單位時間完成的產量為基礎——循環週期較短的工作

（2）以單位產出的時耗為基礎——循環週期較長的工作

工資與產出水平之間關係——確定方法有兩種

（1）工資率是常量

（2）工資率是變量

（三）對個人績效獎勵計劃的評價

1、優 點：

1.1 有助於生產率提高。

1.2 與績效加薪相比企業支付給員工的獎勵性薪酬是不會被動累積到員工的基本薪酬當中去的。

1.3 個人績效獎勵計劃降低了監督成本。

1.4 比按工時支付固定薪酬的做法能夠更好地預測和控制勞動力成本。

1.5 操作起來以及在對員工溝通的時候比較容易。

2、缺點：

2.1 個人績效獎勵計劃對於傳統製造業中的生產類員工來說是比較適用的，但是現代企業中的大多數工作—如管理性的工作和專業性的工作—都沒有可以衡量的物質產出。

2.2 個人績效獎勵計劃在設計和維持可以被員工們所接受的績效衡量標準等方面具有一種潛在的管理難題,尤其是產出標準的變動很有可能會造成員工對企業的不信任感。

2.3 個人獎勵計劃往往會導致員工只去做那些有利於他們獲得報酬的事情，而對於其他的事情則傾向於不管不問。

2.4 個人獎勵計劃可能不利於員工掌握多種不同的技能。

四、群體績效獎勵計劃

(一)群體績效獎勵計劃概述

企業實施群體獎勵計劃可能是比較有利的：

其一,從工作角度來看,工作產出是集體合作的結果,無法衡量員工個人對於產出所作出的貢獻。

其二,從組織狀況來看,在組織的目標相對穩定的情況下,個人的績效標準,需要針對環境的壓力而經常性地變化,並且生產方式以及資本和勞動力的要素組合也必須適應壓力的要求而經常作出調整。

其三,從管理方面來看,企業中存在良好的績效文化和團隊合作文化。

(二)群體績效獎勵計劃類型:

1. 利潤分享計劃

1.1 利潤分享計劃是指根據對某種組織績效指標(通常是指利潤這樣一些財務指標)的衡量結果來向員工支付報酬的一種績效獎勵模式。

1.2 直接支付或延期支付

優勢：員工的責任感、身份感和使命感會增強;在企業經營陷入低沉時有助於企業控制勞動力成本,而在經營狀況良好的時候，則為組織和員工之間的財富分享提供了方便。

缺陷：主要是在直接推動績效改善以及改變員工或團隊行為方面所起的作用卻不大。

2.收益分享計劃

收益分享計劃是企業提供的一種與員工分享因生產率提高、成本節約和質量提高而帶來的收益的績效獎勵模式。

2.1 收益分享計劃和利潤分享計劃之間的區別：

首先,收益分享計劃並不使用整個組織層次上的績效衡量指標(利潤),而是對某一群體或者部門的績效進行衡量；

其次,收益分享計劃下的獎勵支付通常比利潤分享計劃下的獎勵支付週期更短,同時更為頻繁；

最後,收益分享計劃具有真正的自籌資金的性質。

2.2 收益分享計劃中的幾個關鍵決策

一是收益衡量與角色定位問題

二是支付頻率問題

三是支付方式問題

四是設計要求問題

五是溝通問題

六是確保財務收益問題

3.成功分享計劃

成功分享計劃又被稱為目標分享計劃,它的主要內容是運用平衡記分卡方法來為某個經營單位制定目標,然後對超越目標的情況進行衡量,並根據衡量結果來對經營單位提供績效獎勵這樣一種做法。

3.1 平衡記分卡強調財務與非財務、長期與短期績效、過程與結果、投入與產出之間的平衡。

3.2 成功分享計劃的特徵：

（1）成功分享計劃需要為參與該計劃的經營單位設定操作模型,該模型需要界定出相關經營單位的核心業務流程,定出3到5個對這一核心業務流程進行衡量的關鍵績效指標,並且為每一個關鍵績效指標都制定出所要達到的目標。

（2）成功分享計劃要求經營單位中的每一位員工都要全面參與。

（3）成功分享計劃要求管理層與基層員工共同制定目標,而不是採取自上而下式的傳統目標制定方式。

（4）成功分享計劃鼓勵持續不斷的績效改進。

（5）成功分享計劃有結束的時候。

3.3 成功分享計劃設計過程中的幾個關鍵決策

（1）參與資格

（2）支付形式

（3）支付頻率

（4）支付數量與支付等級

（5）資金來源

（6）實施成功分享計劃的經營單位範圍選擇

3.4 成功分享計劃的設計程序

步驟一:建立成功分享計劃委員會

步驟二:制定經營績效指標並且確定不同指標之間的權重

步驟三:為績效指標確定公平合理的進展目標並確定獎勵的辦法

4.小群體獎勵計劃以及團隊獎勵計劃

小群體獎勵計劃是適用於規模更小的工作群體的一種群體獎勵計劃。持續時間不長，任務完成獎勵計劃消失。

在整個組織推行前，先在小群體試行，以小群體的業績為依據，常見的圍繞財務目標，也支持非財務目標。

企業對之感興趣的原因是個人獎勵計劃很容易支付給多人群體，而且更實用。可以增強凝聚力，降低管理成本，個人之間競爭削弱，團隊之間競爭加強。

第三節 特殊績效認可計劃

一、特殊績效認可計劃在組織報酬戰略中的地位與作用

特殊績效認可計劃是指一種現金或非現金的績效認可計劃,即在員工遠遠超出工作要求表現出特別的努力、實現了優秀的業績或者作出了重大貢獻的情況下,組織給予他們的小額一次性獎勵,當員工或者某個團隊出現超出預期的優秀績效,但是組織利用其他報酬形式卻無法提供報償時,績效認可計劃是一種非常有效的獎勵方式。

特殊績效認可計劃對組織報酬戰略的貢獻:

  1. 提高了整個報酬系統的靈活性和自發性。
  2. 擴大了員工在報酬系統中的參與機會,提供真正符合員工興趣的報酬。
  3. 有利於激勵那些與組織的價值觀和文化相一致的行為,強化企業的戰略目標。
  4. 實現報酬系統的成本有效性最大化。
  5. 在設計特殊績效認可計劃之前,首先應該將其放在組織的整個報酬戰略中考慮。

二、特殊績效認可計劃的設計與實施

步驟一：確定特殊績效認可計劃的目標。

步驟二：決定計劃的類型和數量。

步驟三：確定需要激勵的活動的類型和性質。

步驟四：決定誰有資格參加認可計劃。

步驟五：決定獎勵的類型和水平。

步驟六：決定獎勵的頻率。

步驟七：決定激勵的成本和資金來源。

步驟八：確定提名和獲獎者挑選過程。

步驟九：確定如何來授予獎勵品。

三、特殊績效認可計劃的審查與評價

1、計劃的運用是否到位

2、所提供的績效認可報酬對員工來說是否有價值

3、績效認可的決策程序是否過於複雜

4、員工對貨幣型績效認可的態度

5、計劃是否和組織的經營戰略保持一致

6、普通員工和經理人員在認可計劃中的公平性

7、計劃是否缺乏信度
