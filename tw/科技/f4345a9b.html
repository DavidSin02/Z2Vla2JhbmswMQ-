網絡工程｜綜合佈線整體解決方案

2020-11-14
| 科技

網絡工程集語音、數據、圖像、監控設備，各線佈線於一體的綜合系統工程，統一佈線設計、網絡設計及安裝施工和集中管理維護。為樓宇大廈和企業提供了先進、可靠的信息化管理，它是通訊、計算機網絡以及智能大廈的基礎。

根據用戶單位的需求及具體情況，結合現代網絡技術的發展水平及產品化的程度，經過充分的需求分析和市場調研，從而確定網絡建設方案，依據方案的步驟、有計劃的實施網絡建設活動。

網絡工程建設是一項綜合的系統工程，一般可分為網絡規劃和設計階段、工程組織和實施階段以及系統運行維護階段。

需求分析：

經過實地考察和分析，初步確定建設規模、定位技術水平、預計投資總額、計劃建設週期等。

網絡總體目標和設計原則：

1）確定網絡總體實現的目標：採用的網絡技術和網絡標準；分期目標、時間和進度計劃等；網絡實施成本、網絡運行成本。

2）總體設計原則：實用性、開放型、高可用性/可靠性、安全性、先進性、易用性、可擴展性原則。

拓撲結構與網絡總體規劃：

主要因素：費用、靈活性、可靠性。

網絡拓撲結構的規劃設計和網絡規模息息相關。一個規模較小的星型局域網沒有主幹網和外圍網之分。規模較大的網絡通常呈倒樹狀分層拓撲結構。

分層設計規劃的優點是可以有效的將全局通信問題分解考慮，分層還有助於分配和規劃帶寬。

核心層（核心交換機，高速傳送數據，對數據不作任何處理）——匯聚層（交換機、路由器設備，訪問層的匯接點，路由數據、分割廣播域/多點傳送域、介質轉換、安全性、遠程訪問的接入點）——接入層（交換機、集線器設備，端接設備到網絡的接入點）

資源子網的規劃設計：

服務器接入 服務器系統是網絡的核心設備，服務器在網絡中的位置直接影響網絡應用效果和網絡運行效率。

服務器一般分為兩類：

1）為全網提供公共信息服務、文件服務和通信服務，為企業網提供集中統一的數據庫服務。它由網絡中心管理維護，服務對象為網絡全局，適宜放在網管中心；

2）部門業務和網絡服務結合，主要由部門管理維護。如財務部服務器等。

服務器接入的幾種方案：

1）千兆以太網端口接入或光纖接入（接到核心交換機上，速度快，對核心交換機端口要求比較高，成本比較高）

2）並行快速以及網冗餘接入（成本低，速度會受到一些影響。）

需求分析及總體設計目標和設計原則：

網絡的連通性

企業各計算機等終端設備之間良好的連通性是需要滿足的基本條件，網絡環境就是提供需要通信的計算機設備之間互通的環境，以實現豐富多彩的網絡應用。

網絡的可靠性

許多現有網絡在初始建設時不僅要考慮到如何實現數據傳輸，還要充分考慮網絡的冗餘與可靠，否則一旦運行過程網絡發生故障，系統又不能很快恢復工作，所帶來的後果便是企業的經濟損失，影響企業的聲譽和形象。

網絡的安全性

在商品競爭日益激烈的今天，企業對網絡的安全性有非常高的要求。在很多企業在局域網和廣域網絡中傳遞的數據都是相當重要的信息，因此一定要保證數據安全保密，防止非法竊聽和惡意破壞，在網絡建設的開始就考慮採用嚴密的網絡安全措施。

網絡的可管理性

隨著網絡規模的日益擴大，網絡設備的數據和種類日益增加，網絡應用日益多樣化，網絡管理也日益重要。良好的網絡管理要重視網絡管理人力和財力的事先投入，主動控制網絡，不僅能夠進行定性管理，而且還能夠定量分析網絡流量，瞭解網絡健康狀況。有預見性地發現網絡上的問題，並將其消滅於萌芽狀態，降低網絡故障所帶來的損失，使網絡管理的投入達到事半功倍的效果。

網絡的擴展性

網絡建設為未來的發展提供良好的擴展接口是非常理智的選擇。隨著企業規模的擴大、業務的增長，網絡的擴展和升級是不可避免的問題。思科通過模塊化的網絡結構設計和模塊化的網絡產品，能為用戶的網絡提供很強的擴展和升級能力。

網絡的多媒體支持

由於視頻會議、視頻點播、 IP 電話等多媒體技術的日趨成熟，網絡傳輸的數據已不再是單一數據了，多媒體網絡傳輸成為世界網絡技術的趨勢。企業著眼於未來，對網絡的多媒體支持是有很多需求的。同時，在網絡帶寬非常寶貴的情況下，豐富的 QoS 機制，如： IP 優先、排隊、組內廣播和鏈路壓縮等優化技術能使實時的多媒體和關鍵業務得到有效的保障。

網絡的高性能

隨著互聯網的發展，上網用戶的不斷增多，訪問和數據傳輸量劇增，網絡負荷也相應加重；隨著企業對多媒體技術的廣泛應用，視頻數據、音頻數據也越來越耗費網絡帶寬。如果網絡沒有高性能，會導致系統反應緩慢，甚至在業務量突增時，發生系統崩潰、中止和異常等現象。高性能的網絡也是一些關鍵業務或特殊應用的必備條件。

網絡拓樸結構及總體規劃

企業為實現整個集團公司的多應用服務網絡平臺，需建立一個高效、可靠、安全、易擴展的網絡系統平臺。

在本方案中採用三層網絡設計的方法（核心層、匯聚層和接入層）。核心層採用兩臺具有三層交換功能的千兆模塊化交換機互為冗餘，兩臺核心交換機間用兩條多模光纖連接，通過兩條千兆光纖鏈路的捆綁實現 2G 的連接帶寬。

匯聚層使用中檔的三層交換機，交換機通過兩條光纖鏈路分別上連到兩臺核心交換機，並利用虛網中繼協議以及每個 VLAN 的生成樹（ Spanning Tree ）技術可以實現在冗餘線路上的負載均衡和上行線路發生故障時的快速恢復。同時。分佈層交換機還通過光纖或雙絞線連接接入層交換機，並通過其三層交換功能實現接入層虛網之間的交換。

接入層交換機使用二層 10/100 兆自適應以太網交換機，交換機通過光纖或雙絞線連接到分佈層交換機。對工作站密集的區域，可直接通過多臺交換機進行堆疊的方式來滿足大量 PC 接入的需求。堆疊後的交換機通過兩條光纖分別上連兩臺核心交換機，並利用虛網中繼協議以及 VLAN 的生成樹技術實現在冗餘線路上的負載均衡和上行線路發生故障時的快速恢復。

另外，在網絡中心使用一臺模塊化路由器作為中心路由器，中心路由器實現租用 ISP 的 DDN/Frame Relay/ISDN 線路連接各分公司的路由器。

網絡中的服務器單獨劃分在一個虛網中，服務器都通過雙鏈路分別連接到兩臺核心交換機上，通過核心交換機的三層交換功能實現服務器和其它虛網之間的交換。

綜合佈線系統設計要領

1．總體規劃

一般來說，國際信息通信標準是隨著科學技術的發展，逐步修訂、完善的。綜合佈線系統也是隨著新技術的發展和新產品的問世，逐步完善而趨向成熟。我們在設計智能化建築物PDS期間，要提出並研究近期和長遠的需求是非常必要的。目前，國際上各綜合佈線產品都只提出15年質量保證體系，並沒有提出多少年投資保證。為了保護建築物投資者的利益，我們可採取 “總體規劃，分佈實施，水平佈線儘量一步到位”。主幹線大多數都設置在建築物弱電井，更換或擴充比較省事；水平佈線是在建築物的天花板內或管道里，施工費比初始投資的材料費高。如果更換水平佈線，要損壞建築結構，影響整體美觀。因此，我們在設計水平佈線，儘量選用檔次較高的線纜及連接件，縮短佈線週期。

2．系統設計

綜檯布線是智能大廈建設中的一項新興技術工程項目，它不完全是建築工程中的“弱電”工程。

智能化建築是由智能化建築環境內系統集成中心利用綜合佈線系統連接和控制“3A”系統組成的。佈線系統設計是否合理，直接影響到“3A”的功能。(3A即樓宇自動化—Building Automation、辦公自動化—Office Automation、通信自動化—Communication Automation)

設計與實現一個合理綜合佈線系統一般有六個步驟：

n 獲取建築物平面圖；

n 分析用戶需求；

n 系統結構設計；

n 佈線路由設計；

n 繪製佈線施工圖；

n 編制佈線用料清單。

星型拓撲結構佈線方式，具有多元化的功能，可以使任一子系統單獨地佈線，每一子系統均為一獨立的單元組，更改任一子系統時，均不會影響其它子系統。

一個完善確定設計的佈線走線系統，其目標是，在既定時間以外，允許在有新需求的集成過程中，不必再去進行水平佈線，損壞建築裝飾而影響審美。

為了使智能建築與智能建築園區的工程設計具體化，根據實際需要，我們將綜合佈線系統分為三個設計等級：

1．基本型

適用於綜合佈線系統中配置標準較低的場合，用銅芯電纜組網。

基本型綜合佈線系統配置：

(1)每個工作區(站)有一個信息插座；

(2)每個工作區(站)的配線電纜為一條4對雙絞線，引至樓層配線架；

(3)完全採用夾接式交接硬件；

(4)每個工作區(站)的幹線電纜(即樓層配線架至設備間總配線架電線)至少有2對雙絞線。

2．增強型

適用於綜合佈線系統中中等配置標難的場合，用銅芯電纜組網。

增強型綜檯布線系統配置：

(1)每個工作區(站)有兩個以上信息插座；

(2)每個工作區(站)的配線電纜均為一條獨立的4對雙絞線，引至樓層配線架；

(3)採用夾接式(110A系列)或接插式(110P系列)交接硬件；

(4)每個工作區(站)的幹線電纜(即樓層配線架至設備問總配線架)至少有3對雙絞線。

3．綜合型

適用於綜合佈線系統中配置標準較高的場合，用光纜和銅芯電纜混合組網。

綜合型綜合佈線系統配登：

(1)在基本型和增強型綜合佈線系統的基礎上增設光纜系統；

(2)在每個基本型工作區的幹線電纜中至少配有2對雙絞線；

(3)在每個增強型工作區的幹線電纜中至少有3對雙絞線。

綜合佈線系統應能滿足所支持的數據系統的傳輸速率要求，並應選用相應等級的傳輸纜線和設備。

綜合佈線系統應能滿足所支持的語音、數據、圖像系統的傳輸標準要求。

綜合佈線系統所有設備之間連接端子、塑料絕緣的電纜或、電纜環箍應有色標。不僅各個線對是用顏色識別的，而只線束組也使用同一圖表中的色標。這樣有利於維護檢修。這也是綜合佈線系統的特點之一。

所有基本型、增強型、綜合型綜合佈線系統都能支持語音、數據、圖像等系統，能隨工程的需要轉向更高功能的佈線系統。它們之間的主要區別在於：

①支持語音和數據服務所採用的方式；

②在移動和重新佈局時實施線路管理的靈活性。

1．基本型綜檯布線系統的特點

(1)是一種富有價格競爭力的綜合佈線方案，能支持所有語音和數據的應用；

(2)應用於語音、語音／數據或高速數據；

(3)便於技術人員管理；

(4)採用氣體放電管式過壓保護和能夠自復的過流保護；

(5)能支持多種計算機系統數據的傳輸。

2．增強型綜合佈線系統的特點

增強型綜合佈線系統不僅具有增強功能．而且還可提供發展餘地。它支持語音和數據應用，並可按需要利用端子板進行管理。

(1)每個工作區行二個信息插座，不僅機動靈活，而且功能齊全，

(2)任何—個信息插座都可提供語音和高速數據應用；

(3)按需要可利用端子板進行管理；

(4)是一個能為多個數據設備製造部門環境服務的經濟有效的綜合佈線方案

(5)採用氣體放電管式過壓保護和能夠自復的過流保護。

3．綜合型綜合佈線系統的特點

綜合型綜合佈線系統的主要特點是引入光纜，可適用於規模較大的建築物或建築群，其餘特點與基本型或增強型相同。

綜合佈線系統設計要領：

1．在PDS設計起始階段，設計人員要作到：

n 評估用戶的通信要求和計算機網絡要；

n 評估用戶樓宇控制設備自動化程度

n 評估安裝設施的實際建築物或建築群環境和結構

n 確定通信、計算機網絡、樓宇控制所使用的傳輸介質

2．將初步的系統設汁方案和預算成本通知用戶單位。

3. 在收到最後合同批准書後，完成以下的系統配置、佈局藍圖和文檔記錄:

n 電纜線路由文檔

n 光纜分配及管理

n 佈局和接合細節

n 光纜鏈路，損耗預算

n 施工許可證

n 訂貨信。

如同任何一個工程一樣，系統設計方案和施工圖的詳細程度將隨工程項目複雜程度而異，並與合同條款、可用資源及工期有關。

設計文檔一定要齊全，以便能檢驗指定的PDS設計等級是否符合所規定的標準。而且在

驗收系統符合全部設計要求之前，必須備有這種設計文檔。

4．應始終確保已完成合同規定的光纜鏈路一致性測試，而且光纜鏈路損耗是可接受的。

介質及連接硬件的性能規格

在結構化佈線系統中,佈線硬件主要包括:配線架、傳輸介質、通信插座、插座板、線槽和管道等。

1)介質

主要有雙絞線和光纖,在我國主要採用雙絞線與光纜混合使用的方法。光纖主要用於高質量信息傳輸及主幹連接,按信號傳送方式可分為多模光纖和單模光纖兩種,線徑為62.5/125微米。在水平連接上主要使用多模光纖,在垂直主幹上主要使用單模光纖。

2)接頭及插座

在每個工作區至少應有兩個信息插座,一個用於語音,一個用於數據。插座的管腳組合為 :1&2、3&6、4&5、7&8。

我國基本上採用北美的結構化佈線策略,即使用雙絞線十光纖的混合佈線方式。 雙絞線又分為屏蔽線與非屏蔽線兩種。

屏蔽系統是為了保證在有干擾環境下系統的傳輸性能。抗干擾性能包括兩個方面,即系統抵禦外來電磁干擾的能力和系統本身向外發射電磁干擾的能力,對於後者,歐洲通過了電磁兼容性測試標準EMC規範。實現屏蔽的一般方法是在連接硬件外層包上金屬屏蔽層以濾除不必要的電磁波。現已有STP及S-STP兩種不同結構的屏蔽線供選擇。

屏蔽系統的屏蔽層應該接地。在頻率低於1MHz時,一點接地即可。當頻率高於1MHz時,EMC認為最好在多個位置接地。通常的做法是在每隔波長十分之一的長度處接地,且接地線的長度應小於波長的十二分之一。如果接地不良(接地電阻過大、攔地電位不均衡等),就會產生電勢差,這樣,將構成保證屏蔽系統性能的障礙和隱患。

值得注意的是，屏蔽電纜不能決定系統的整體EMC性能。屏蔽系統的整體性取決於系統中最弱的元器伯。如跳接面板、連接器信息口、設備等。因此,若屏蔽線在安裝過程中出現襲縫,則構成子屏蔽系統中最危險的環節。

佈線系統的設計與施工

在整個網絡項目工程總投資中，佈線所佔的比例不到5%，但結構化佈線卻是支持高帶寬應用和未來應用升級換代的重要保障。而且相比其它網絡配置如PC，軟件和LAN的產品，佈線的壽命是最長的。所以不僅僅是"大樓起，佈線上"，佈線系統不但成為智能大廈等大環境佈線的必然之選，在中小網絡環境中也得到普遍的應用，成為一種應用潮流。

結構化綜合佈線系統(Structured Cabling Systems,縮寫SCS)採用模塊化設計和分層星型拓撲結構。它能適應任何大樓或建築物的佈線系統。其代表產品是建築與建築群綜合佈線系統(Premises Distribution System, 縮寫PDS)。另外，還有兩種先進的系統，即智能大樓佈線系統(IBS)和工業佈線系統(IDS)。它們的原理和設計方法基本相同，差別是PDS以商務環境和辦公自動化環境為主。

PDS一般採用模塊化設計和物理分層星型拓撲結構，傳輸語音、數據、圖像以及各類控制信號。

PDS的結構可分6個獨立的子系統(模塊)。

1.工作區(終端)子系統(WORK AREA SURSYSTEM)

工作區佈線子系統由終端設備到信息插座的連線(或軟線)組線，它包括裝配軟線、連接器和連接所需的擴展軟線，並在終端設備和I／O之間搭橋，信息插座有牆上、地上、桌上、軟基型多種，標準有RJ45／RJ11的單、雙、多孔等各種類型。

2.水平佈線子系統(HORIZONTAL SUBSYSTEM)

水平佈線子系統將電纜從樓層配線架連接到各用戶工作區上的信息插座上，一般處在同

一樓層。通常可以採用5類8芯4對雙絞線，符合或超過EIA／TIA一568標準。

3.垂直幹線子系統(RISER BACKBONE SURSYSTEM)

垂直幹線子系統指各樓層配線架與主配線架間的大對數多芯銅纜或光纜組成，或二者混

用。它是綜合佈線系統的神經中樞，其主要功能是將主配線架系統與各樓層配線架系統連接起來。

4.管理子系統(ADMINISTRATION SUBSYSTEM)

管理子系統由樓層配線架組成。其主要功能是將垂直幹纜線與各樓層水平佈線子系統相

連接。佈線系統的優勢和靈活性主要體現在管理子系統上，只要簡單地跳一下線就可完成任何一個結構化佈線系統的信息插座以對任何一類智能系統的連接，極大地方便了線路重新佈局和網絡終端的調整。光纖連接時，要用光纖接續箱(LIU)，箱內可有多個ST連接器安裝孔,箱體箱內的線路彎曲設計應符合62．5／125微米多模光纖的彎曲度要求，光纖接頭用STII，由陶瓷材料製成，最大信號衰減小於0．2dB，光耦合器可作為多模光纖與網絡設備或光纖接續裝置上的連接，配線架和光纖接續箱通常設在弱電井或設備間內，用來連接其它子系統，並對它們通過跳線進行管理。

5.設備間子系統(EQUIPMENT SUBSYSTEM)

設備間子系統由主配線架和各公共設備組成。它的主要功能是將各種公共設備(如計算機主機、數字程控交換機、各種控制系統、網絡互連設備)等與主配線架連接起來。該子系統還包括電氣保護裝置等。

6.建築群向連接子系統(CAMPUS BACKBONE SUBSY5TEM)

該子系統是指主建築物中的主配線架延伸到另外一些建築物的主配線架的連接系統。與

垂直子系統類似，通常採用光纜或大對數銅纜連接。它是整個佈線系統的一部分(包括傳輸介質)並支持提供樓群之間通信所需的硬件，其中有電纜、光纜和防止電纜的浪湧電壓進入建築物的電氣保護設備。

綜合佈線要點

橋架設計合理，保證合適的線纜彎曲半徑。上下左右繞過其他線槽時，轉彎坡度要平緩，重點注意兩端線纜下垂受力後是否還能在不壓損線纜的前提下蓋上蓋板。

放線過程中主要是注意對拉力的控制，對於帶卷軸包裝的線纜，建議兩頭至少各安排一名工人，把卷軸套在自制的拉線杆上，放線端的工人先從卷軸箱內預拉出一部分線纜，供合作者在管線另一端抽取，預拉出的線不能過多，避免多根線在場地上纏結環繞。

拉線工序結束後，兩端留出的冗餘線纜要整理和保護好，盤線時要順著原來的旋轉方向，線圈直徑不要太小，有可能的話用廢線頭固定在橋架、吊頂上或紙箱內，做好標註，提醒其他人員勿動勿踩。

在整理、綁紮、安置線纜時，冗餘線纜不要太長，不要讓線纜疊加受力，線圈順勢盤整，固定扎繩不要勒得過緊。

在整個施工期間，工藝流程及時通報，各工種負責人做好溝通，發現問題馬上通知甲方，在其他後續工種開始前及時完成本工種任務。

如果安裝的是非屏蔽雙絞線，對接地要求不高，可在與機櫃相連的主線槽處接地。

線槽的規格是這樣來確定的：線槽的橫截面積留40%的富餘量以備擴充, 超5類雙絞線的橫截面積為0.3平方釐米。

線槽安裝時,應注意與強電線槽的隔離。佈線系統應避免與強電線路在無屏蔽、距離小於20cm情況下平行走3米以上。如果無法避免，該段線槽需採取屏蔽隔離措施。進入傢俱的電纜管線由最近的吊頂線槽沿隔牆下到地面，並從地面鏜槽埋管到傢俱隔斷下。

n 管槽過渡、接口不應該有毛刺，線槽過渡要平滑。

n 線管超過兩個彎頭必須留分線盒。

n 牆裝底盒安裝應該距地面30釐米以上，並與其他底盒保持等高、平行。

n 線管採用鍍鋅薄壁鋼管或PVC

機房佈線：

互聯時代的軟硬件非常重要，但是它們穩定運行的基礎——機房更重要。機房應具有高可用性，高可靠性、高安全性、可擴容性和網絡資源豐富等特點。以下將著重介紹機房佈線。

機房的佈線系統直接影響到未來機房的功能，一般佈線系統要求佈防距離儘量短而整齊，排列有序，具體的方式有“田”字形和“井”字形兩種。其中：“田”字形較適用於環形機房佈局，“井”字形較適用於縱橫式機房佈局。 它的位置可安排在地板下和吊頂兩個地方，各有特點，下面分別加以論述：

1、地板佈線：這是一種最常見的佈線方式，它充分利用了地板下的空間，但要注意地板下漏水、鼠害和散熱，還應保證在每個機櫃下方開鑿相應的穿線孔（包括地板和線槽）。

2、吊頂佈線：該佈線方式特別適合於經常需要佈線的機房，目前也非常流行，此方式中吊頂內包含了各種佈線電源、弱電佈線，在每個機櫃上方開鑿相應的穿線孔（包括地板和線槽），當然也要注意漏水、鼠害和散熱。

具體佈線的內容有：電源佈線、弱電佈線和接地佈線，其中電源佈線和弱電佈線均放在金屬佈線槽內，具體的金屬佈線槽尺寸可根據線量的多少並考慮一定的發展餘地（一般為100×50或50×50）。電源線槽和弱電線槽之間的距離應保持至少5釐米以上，互相之間不能穿越，以防止相互之間的電磁干擾。下面將一一加以闡述：

1、電源佈線：在新機房裝修進行電源佈線時，應根據整個機房的佈局和UPS的容量來安排，在規劃中的每個機櫃和設備附近，安排相應的電源插座，插座的容量應根據接入設備的功率來定，並留有一定的冗餘，一般為10A或15A。電源的線徑應根據電源插座的容量並留有一定的容量來選購。

2、弱電佈線：弱電佈線中主要包括同軸細纜、五類網線和電話線等，佈線時應注意在每個機櫃、設備後面都有相應的線纜，並應考慮以後的發展需要，各種線纜應分門別類用尼龍編織帶捆紮好。

3、接地佈線：由於新機房內都是高性能的計算機和網絡通訊設備，故對接地有著嚴格的要求，接地也是消除公共阻抗，防止電容耦合干擾，保護設備和人員的安全，保證計算機系統穩定可靠運行的重要措施。在機房地板下應佈置信號接地用的銅排，以供機房內各種接地需要,銅排再以專線方式接入該處的弱電信號接地系統。

關於6類線纜佈線施工，應特別注意以下幾點:

1、如果在2個終端間有多餘的線纜，應該按照需要的長度將其剪斷，而不應將其捲起並捆綁起來。

2、線纜的接頭處反纏繞開的線段的距離不應超過2cm，過長會引起較大的近端串擾。

3、在接頭處，線纜的外保護層需要壓在接頭內而不能在接頭外。雖然在線纜受到外界拉力時整個線纜均會受力，但若外保護層壓在接頭外，則受力的將主要是線纜和接頭連接的金屬部分。

4、在線纜接線施工時，線纜的拉力是有一定限制的，一般為9kg左右。過大的拉力會破壞線纜對絞的勻稱性。

由於六類線纜的外徑要比一般的五類線粗，為了避免線纜的纏繞（特別是在彎頭處），在管線設計時一定要注意管徑的填充度，一般內徑20mm的線管以放2根六類線為宜。

綜合佈線通道傳輸的性能指標

平衡電纜通道傳輸性能指標

按照國際佈線標準ISO/IEC11801：1995（E），給出平衡電纜傳輸通道（Balanced cabling links）的參數。除非特別強調，這些參數適應於屏蔽和非屏蔽平衡電纜的傳輸通道。描述平衡電纜通道傳輸性能的電氣特性參數有直流環路電阻、特性阻抗、衰減、近端串擾損耗、衰減與串擾之比、結構回波損耗、傳輸延遲等，與通道長度有關的參數，如衰減、直流環路電阻、傳輸延遲等；與電纜紐距有關的參數有特性阻抗、衰減、近端串擾損耗和結構回波等。不過，電纜一旦成形，這些參數只與電纜及相關連接硬件的安裝工藝有關。

1） 特性阻抗

特性阻抗是電纜及相關連接硬件組成的傳輸通道的主要特性。它根據信號傳輸的物理特性，形成對信號傳輸的阻礙作用，它用電阻與電抗一起來描述稱特性阻抗。用歐姆（Ω）來度量。平衡電纜通道的特性阻抗變化由結構回波損耗來描述。

為了確保應用系統通道的特性阻抗，就需要一個正確的設計、選擇適當的電纜和相關連接硬件。

2） 結構回波損耗（Structural Return Loss）

它是衡量通道一致性的。通道的特性阻抗隨著信號頻率的變化而變化。如果通道所用的線纜和相關連接硬件阻抗不匹配，就會造成信號反射。被反射到發送端的一部分能量會形成干擾。導致信號失真，這就降低綜合佈線的傳輸性能。在綜合佈線的任一接口測得平衡電纜回波損耗應符合或超過下表1的數據。

3） 衰減

對所有光纖通道來說，不管工作波長或光纖纖芯大小，光的反射損耗是一個重要指標。光纖最小模態帶寬指標應能支持帶寬高速應用，一些低帶寬的光纖通道通常不適合高速應用，它可以用在短距離的一些特殊系統上。多模光纖的帶寬用頻率來表示，光纖的帶寬通常是不測量的。然而，其它如光纖損耗和反射損耗測試是需要的。
