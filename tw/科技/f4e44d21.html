「中考化學」常見氣體的檢驗方法 含反應方程式

2020-11-14
| 科技

本文總結了：H2、CO、Cl2、O2、CO2、HCl、H2S、SO2、N2、NO、NO2等多種常見氣體的檢驗方法，方便中考的學生進行系統複習。

1.H2

將待檢氣體通過灼熱的CuO，CuO由黑色變為紅色，並將產物通入無水硫酸銅中，無水硫酸銅變藍，則證明通入CuO的氣體是H2。

H2+CuO =（加熱）Cu + H2O

CuSO4+5H2O =CuSO4·5H2O

2.CO

讓待檢氣體在空氣中燃燒，在火焰上方罩一干燥的小燒杯，燒杯上無水珠生成，然後將產物與澄清的石灰水接觸，澄清的石灰水變渾濁，則證明燃燒氣體為CO。

2CO + O2 = 2CO2

CO2 + Ca(OH)2 = CaCO3 + H2O

3.CO2

將氣體分別通入澄清的石灰水和品紅溶液中，如果澄清的石灰水變渾濁，且品紅不褪色，則通入的氣體為CO2。

CO2 + Ca(OH)2 = CaCO3 + H2O

4.CH4

讓待檢氣體在空氣中燃燒（火焰為淡藍色），在火焰上方罩一干燥的小燒杯，燒杯上有液滴生成，然後將產物與澄清的石灰水接觸，澄清的石灰水變渾濁，則證明燃燒氣體為CH4。

CH4 + 2O2→（點燃）CO2 + 2H2O

CO2 + Ca(OH)2 = CaCO3 + H2O

5.N2

將點燃的鎂條放在待檢氣體中燃燒，燃燒之後，向燃燒產物中加入適量的水，如果有白色的不溶固體和能使潤溼的紅色石蕊試紙變藍的刺激性氣味的氣體產生，則證明待檢的氣體為N2。

3Mg + N2 =（點燃）Mg3N2

Mg3N2 + 6H2O =3Mg(OH)2↓+ 2NH3↑

6.NO

直接將待檢氣體的瓶蓋打開，如果在瓶口有紅棕色氣體產生，則證明待檢氣體是NO。

2NO + O2 = 2NO2 (紅棕色)

7.NO2

將待檢氣體溶於水中，若待檢氣體由紅棕色變為無色，且水溶液也成無色，則證明待檢氣體為NO2。

3NO2(紅棕色) + H2O =2HNO3 + NO(無色)

8.O2

將帶火星的木條伸入待檢氣體中，如果帶火星的木條復燃，則證明待檢氣體為O2。

C + O2 = CO2

9.H2S

（1）聞待檢氣體的氣味，如果有臭雞蛋氣味，則證明待檢氣體為H2S。

（2）用溼潤的Pb(CH3COO)2試紙靠近盛待檢氣體的瓶口，如果Pb(CH3COO)2試紙變成黑色，則證明待檢氣體是H2S。

Pb(CH3COO)2 + H2S = PbS↓+ 2CH3COOH

10.SO2

將待檢氣體通入品紅溶液中，如果品紅溶液褪色，且受熱後又恢復原色，則證明待檢氣體是SO2。

11.Cl2

（1）觀察待檢氣體的顏色，如果呈黃綠色，則證明待檢氣體是Cl2。

（2）將氫氣點燃並伸入待檢氣體中，如果氫氣繼續燃燒，併產生蒼白色火焰，同時，在瓶口附近有白霧產生，則證明待檢氣體為Cl2。

H2 + Cl2 =（點燃）2HCl

（3）將溼潤的KI澱粉試紙靠近成待檢氣體的瓶口，如果KI澱粉試紙變藍，則證明待檢氣體為Cl2。（注：當有幾種氧化性氣體同時存在時，則此法不適用）

2KI + Cl2 = KCl +I2
