極大簡化操作，IDEA自定義模板代碼的運用

2020-11-14
| 科技

IDEA提供了常用的模版代碼，同時允許我們自定義模版，運用好這一功能，可以節省我們很多敲重複代碼（或者複製重複代碼）的時間，比如自定義的全局異常等在項目中常用的方法，都可以套在自定義模版中使用，非常方便！

1. 常用模板代碼快捷鍵

（1）main方法

在類體中輸入 psvm ，回車。生成如下代碼。

public static void main(String[] args) { }

（2）打印輸出

在方法體中輸入 sout ，回車。生成如下代碼。

System.out.println();

（3）普通 for 循環

在方法體中輸入 fori，回車。生成如下代碼。

for (int i = 0; i < ; i++) {          }

（4）增強 for 循環

在方法體中輸入 foreach，回車。生成如下代碼。

for (:     ) { }

（5）查看更多快捷鍵

Ctrl + J

2. 自定義模板代碼

進入設置頁面

創建自定義分組（下圖中的Java分組原來是沒有的，是本人已經建好的。可以假設還沒有這個分組，繼續往下看）

輸入分組名

創建模板代碼1

創建模板代碼2

將模板添加到Java分組

然後在方法體中輸入 sys 就會自動生成 Syste.out.println() 了;

附常用模板

/** * 1.
