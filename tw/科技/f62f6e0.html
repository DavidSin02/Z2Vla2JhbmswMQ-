邊坡綠化做法和施工工藝的選擇

2020-10-29
| 科技

在公路、鐵路、礦山、水利等工程建設中，經常伴有大量的開挖土石方，而這些土石方開挖破壞了原有植被，造成大量的裸露邊坡，導致水土流失嚴重和生態環境失衡。由於這些工程所形成的邊坡靠自然界本身的力量恢復生態平衡往往需要較長的時間，甚至根本無法恢復，因而需要採取工程措施對邊坡進行工程防護與生態綠化處理，以防止邊坡破壞、水土流失，並涵養水源、淨化空氣、美化環境。

邊坡綠化以種植草本地被為主，一般應選擇乾旱、瘠薄、根系發達、覆蓋度好、易於成活、便於管理、同時兼顧景觀效果的草本或木本植物。

根據不同地段的實際情況，可採用叢植、列植等綠化模式，儘量做到喬、灌、花、草搭配，形成不同景觀的植物群落。

邊坡生態防護植物配置的技術原則是：以水土保持為主，兼具生態景觀效果。邊坡防護要考慮對整個植被進行逐步恢復，應以林草植物為主進行生態模式配置，有利於固土護坡、防止水土流失、改善邊坡景觀和護坡效果。

選擇植物種類時，要注意保持生物多樣性，營建喬、灌、草結合的多樹種、多結構、多功能的復層生態景觀群落，有效增加綠量和綠葉面積，挖掘單位面積上的潛在生態力，提高葉面積指數。整個綠化沿線應注意立體空間上的線條變化和節奏感。

邊坡綠化常用的幾種施工工藝

1、噴砼（綠色）工藝

坡度較陡且穩定性較好的坡面，可採用此工藝。

施工工序：製作安裝坡面錨釘及洩水孔 → 掛網（加強筋連接）→ 噴砼（綠色）→ 砼養護 → 拆除腳手架

2、噴混植生工藝

首先噴射不含種子的混合料，噴射厚度6～8cm，待第一次噴射的混合土達到一定強度後，緊接著第二次噴射經過催芽處理後的種子加入過篩後的泥炭土、腐殖土、粘結劑、纖維、緩釋複合肥、保水劑攪拌均勻後的混合材料。

工序流程：坡面處理 → 鋪網釘網 → 噴混植生 → 回填種植土 → 養護

3、客土吹附工藝

在坡面較陡的巖面或岩土面上，利用噴播設備將經過一定比例混合的客土噴射到掛有鐵絲網的坡面上。

工序流程：坡面處理 → 鋪網，釘網 → 覆土，噴射客土及種子 → 養護

注意事項：

①鋪網：選用網孔為4×4cm或5×5cm的鍍PVC或鍍鋅菱形鐵絲網鋪設坡面，長度根據需要裁剪，坡頂延伸不低於50cm，開溝並用樁釘固定後回填或埋入截水溝中。坡頂固定後自上而下鋪設。左右兩片網之間的搭接寬度不小於10cm。釘網：針對石坡面情況採用“L”形Φ6～8或其它型號的鋼錨釘或竹錨釘，錨釘長度根據實際情況分別採用150～400mm。

②客土吹附：將過篩腐殖土、草纖維、泥炭土、緩釋營養肥、粘合劑、保水劑、水等混合材料用噴播機充分攪拌，然後通過噴播機將材料及種子送至坡面，噴附到預先固定在坡體上的鐵絲網上。

4、液力噴播工藝

在坡度達到小於一般土壤穩定坡角且本身具有植物生長的土壤，具備了人工回填土後液力噴播完成植被建植和恢復的條件時，可採用液力噴播工藝。液力噴播工藝是將經過催芽處理後的種子加入過篩的腐殖土、草纖維、粘合劑、保水劑、緩釋複合肥等攪拌均勻後，均勻噴射到邊坡表面上，噴射厚度2～3cm。

工序流程：坡面修整 → 覆土或客土吹附 → 液力噴播 → 養護

5、植生袋法

利用邊坡巖面本身的凹陷處，在凹陷處下部採用植生袋圍堰，植生袋內裝入按一定比例配製的耕植土、有機基質、保水劑、肥料，在圍堰內填土，種植灌木為主；或在坡面外側直接用植生袋梯形疊砌，形成一層植生袋面層，通過該植生袋內種子的生長來綠化邊坡。

工序流程：植生袋灌注 → 植生袋堆砌 → 植生袋加固 → 養護

6、魚鱗坑開挖工藝

在條件許可的邊坡，通過風鎬或小爆破來開挖適當的魚鱗坑，在坑內填土，種植喬灌木及爬藤類植物。魚鱗坑開挖的一般要求：坑0.5×0.5m，坑深0.5m，間距2m，交叉開挖呈梅花狀。

工序流程：風鎬或小爆破成坑 → 槽內裝填種植土 → 種植喬灌木及爬藤類植物 → 養護

7、種植槽工藝

在陡峭的巖質邊坡面上，利用工程錨杆固定和鋼筋混凝土樑板形成種植槽，在槽內的種植土上種植喬灌木及爬藤類植物。

工序流程：搭設腳手架 → 錨杆成孔、製作 → 鋼筋混凝土樑板槽澆築 → 槽內裝填種植土 → 種植喬灌木及爬藤類植物綠化 → 養護
