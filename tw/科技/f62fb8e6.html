1． 大型LNG船自持型TYPE B MOSS型液貨艙簡述

（1）自持型TYPE B MOSS 型液貨艙的發展歷史

第一條使用自持型TYPE B MOSS型液貨艙的LNG船是在1973年由挪威一家船廠建造的87600立方米LNG船，當時是使用的含9%鎳的合金鋼，但是隻有最初的兩條船是使用9鎳鋼建造的貨罐，之後很快就修改設計，主要以5083鋁合金來製造球型貨罐了，鋁合金的厚度一般在28-55毫米，在貨罐中部的赤道帶，厚度會達到170毫米。最初的大型MOSS LNG船是有5個36.6米直徑的球罐型液貨艙，後期隨著球罐建造技術的提高，一般使用4個直徑為40米的球罐型液貨艙，就我所知目前最大的球罐型液貨艙是日本川崎重工建造的，4個液貨艙的容積做到將近177000立方米，也就是說液貨艙的直徑將近44米，差不多已經可以說到了極限了

下面是一個常見的直徑約40米的球罐各部位艙壁厚度分佈圖，給大家一個基本概念




（2）自持型TYPE B MOSS 型液貨艙的建造方法

對於MOSS型液貨艙，以上面直徑40米的的液貨艙建造為例，其是由數十塊預成型厚鋁合金板分片分段焊接而成，一般來講，有如下兩種建造方法：

  • 剛性固定法，這種建造方法的優勢是，可以在造船廠之外的任何地方建造，待整個球罐型液貨艙建造完成後，再將其運送到船塢，整體吊裝進船體液貨艙區域內，下圖即為在一個獨立的液貨罐工廠建造的球罐型液貨艙完成後加裝垂向柱形裙圍結構的情形。




  • 片體法，這種方法是將球罐如上面分區的方法分成若干環帶，每個環帶再分解成若干片體，先製造片體，然後組成環帶，最後再將環帶組成球體，通常這種建造方法會在船體建造的同時將液貨艙分成下球體，連同裙圍結構的赤道圈和上球體三個分段進行建造，在船舶主體完成後，再將三個分段吊裝到船體液貨艙區域進行焊接組裝。


（3）自持型TYPE B MOSS 型液貨艙的絕緣層建造方法

MOSS 型液貨艙和薄膜型液貨艙在絕緣層方面的區別是，它不需要完整的次屏蔽，而僅在貨艙底部之下設置滴盤形成縮減的部分次屏蔽，用以容納可能洩漏的液貨天然氣。滴盤佈置在船體的內底中，材料為不鏽鋼或與球罐相同的5083鋁合金，滴盤頂部周邊由法蘭圍板構成，內部設置擋板，防止由於船體橫搖和縱傾引起積聚液體的運動，滴盤的容量由計算出的15天內可能的洩漏液貨量來決定。

而MOSS型液貨艙主絕緣層的建造主要有兩種型式：

  • 板塊式聚苯乙烯絕熱材料建造型式
  • 環帶發泡式聚苯乙烯絕熱材料建造型式
  • 絕熱系統的厚度根據所要求的液貨蒸發率為基礎計算確定，兩種型式的外觀如下圖所示




  • 板塊式聚苯乙烯絕熱材料以1.2米X 1.1米的聚苯乙烯預製泡沫絕熱板塊覆蓋整個球罐型液貨艙。絕熱板塊由兩層組成，在截面中和軸處由玻璃纖維將兩層絕熱版粘結，厚度約為190-230毫米，絕熱層的厚度取決於設計最大蒸發率，而最大蒸發率是由給定條件下計算出來的，通常的給定條件如下：貨物為純甲烷，空氣溫度為45攝氏度，海水溫度為32攝氏度，貨物溫度為零下162攝氏度，平穩海況。絕熱板塊以緯向排列（見上左圖），依球型的走向邊緣削斜，依球型曲率成型，與球罐以螺栓進行連接。在環之間設置由預壓成型的岩棉填充，在板之間設置彈性聚苯乙烯。絕熱層外部以粘結的方式覆蓋0.3毫米厚的鋁膜，這層鋁膜的作用是在液化天然氣微量洩漏時作為液密和氣密屏障。
  • 環帶發泡式聚苯乙烯絕熱材料為擠壓式聚苯乙烯泡沫，覆蓋除上下兩極和中間赤道帶之外的球體，採用自動連續向施工現場輸送絕熱材料的方式進行（見上右圖），並採用聚苯乙烯複合料進行熱連接，因為材料的收縮，絕熱材料與球體的接觸面採用玻璃纖維加強材料。絕緣層水平向收縮縫和為防止材料連續斷裂而設置的斷裂屏均採用聚苯乙烯複合材料。上下兩極採用1.2米X1.2米的聚苯乙烯預製板塊，裙體結構和赤道圈採用1米X1.2米的膨脹聚苯乙烯預製板塊。在最外層同樣由鋁膜覆蓋，以保證液密和氣密。


（4）自持型TYPE B MOSS 型液貨艙的支撐結構

MOSS 型液貨艙的整個球體在赤道圈外側由一個垂向柱形裙圍結構支撐（見製造方法處的圖片），柱形裙圍結構底部與船體結構焊接成一個整體，裙圍結構與球面赤道圈的連接是由具有特殊截面的連接件焊接而成。通過對裙圍結構型式的不斷優化，在目前的設計中，裙圍結構的上部材料一般為高強度鋁合金，中部為阻熱不鏽鋼，上部與中部之間為結構過度連接件，一般由鋁，鈦，鎳，不鏽鋼組成，下部為低溫鋼（見下圖），這樣減少了船體與液貨之間的熱量傳遞，進而降低了液貨蒸發率。




（5）結構過度連接件說明

上圖中的這個結構過度連接件很有意思，我試著給大家解釋一下這個連接件的情況，之所以設計這樣一個連接件是因為對於鋁和不鏽鋼的直接焊接，由於熱膨脹係數的不同而很難實現，那麼怎樣解決這個問題呢？工程師們是這樣做的，在貨罐赤道處安裝這樣一個總厚度約37毫米的連接件（見下圖），上部是鋁製的裙圍，然後是一層鈦板，接下來是一層鎳板，然後下部連接不鏽鋼裙圍，這個結構連接件一來將貨罐的鋁製結構與下部的不鏽鋼支撐裙圍連接了起來，二來起到減少熱傳導的作用，可以說一舉兩得，人的智慧真的是無窮的啊！




（6）泵塔

在每一個液貨艙的球體垂向軸線上設置泵塔（見下圖），泵塔由鋁合金製造，泵塔的存在增強了MOSS型液貨艙在部分裝載時對自由液麵的控制，所以對於MOSS型液貨艙，是沒有和薄膜型貨艙一樣的部分裝載限制的。

泵塔一般包含如下設備及組件

  • 液貨泵，液貨管線等
  • 液位等液貨艙檢測系統
  • 進入液貨艙的主要通道




（7）自持型TYPE B MOSS 型液貨艙的洩漏保護

IGC規則規定TYPE B型液貨艙的設計必須遵循“失效前洩漏”的準則，即在15天之內產生的裂紋不會擴展到結構產生突然的，災難性的失效。對於MOSS型液貨艙，最大洩漏量應小於或等於20升/小時，且這個速率將允許船舶在發現洩漏後的15天內駛達維修地點以排出貨艙內的液貨並對貨艙進行維修，對於這種少量的洩漏，MOSS型貨艙設有如下保護系統：

  • 貨艙絕緣層及主屏蔽
  • 底部設置滴盤的部分次屏蔽
  • 危險氣體檢測系統
  • 液化天然氣排出系統

在次屏蔽設置的滴盤處，有3種檢測方法用於檢測液化天然氣的洩漏，它們是：

  • 30%LEL的危險氣體檢測系統
  • 液位計
  • 零下100攝氏度的溫度探頭

MOSS型LNG船貨物圍護系統從面世至今已經超過45年，在幾十年的營運時間中證明這種貨艙圍護系統還是非常安全可靠的！

（2） 自持型TYPE C 型液貨艙的種類

由以上幾條船的貨罐型式可知，C型罐有單C型罐和雙C型罐之分，而雙體罐又有正常形態的雙體罐和異型雙體罐之分，但是在查找資料的時候發現還有一種三C型罐，我個人是從來沒有見過，不過發現了一張圖片，放到下面給大家看一下，我也是挺新奇的，個人覺得大概是為了增加貨罐容積吧，是否正確就不敢保證了。




（3） 自持型TYPE C 型液貨艙的絕緣層建造方法

自持型TYPE C 型液貨艙一直以來在小型LPG船上應用廣泛，可以說是一種比較主流的貨罐型式，罐體可以由專業廠家建造，在船體建造完成後，再將其運送到船塢，整體吊裝進船體液貨艙區域內，如上面圖片所示，與MOSS型的剛性固定法基本相同。通常C型罐僅需要做一層絕緣層，下圖是我在啟元輪駐廠監造期間進入隔離空倉時照的貨罐外表，由照片可見，使用的是類似於MOSS型的板塊式聚苯乙烯絕熱材料建造型式。



（4） 對使用自持型TYPE C 型液貨艙LNG船的幾點思考

當我在幾條使用TYPE C型液貨艙的小型LNG船工作期間，有一些個人的思考和想法，下面提出來和各位讀者討論，可能我的想法也不一定正確，所以在這裡算是拋磚引玉吧，不當之處，希望有真正的內行人士予以指正。

  • 首先一條是我在上一篇文章中提到的，即使一些使用TYPE C型貨艙的小型LNG船的貨艙可以承受一定的壓力，但是由於LNG貨溫會隨著艙壓的升高而升高，進而使得貨溫超過貨主要求的溫升標準（見本人撰寫的雜談之二內容），故此即使該貨艙可承受一定的壓力，在航行中仍應該按照全冷型液貨船控制艙壓與艙溫。
  • 其次是由於TYPE C型貨艙僅做一層絕緣層，貨艙外部與船舶內船殼之間是一個較大容積的隔離空倉，那麼為了保持該空間內氣體的乾燥，這種小型LNG船最好設計有容量較大的即可生產氮氣又能生產幹空氣的氮氣發生器，以便在正常營運期間給該空間提供足夠的幹空氣及在進廠修船前後可以生產足夠的氮氣用於貨艙惰化。
  • 第三是在我上過的這幾條小型LNG船上，對隔離空間的危險氣體探測都是使用了固定式危險氣體探頭，個人認為如果能夠使用危險氣體取樣檢測系統會更好一些，因為畢竟當需要對危險氣體探頭進行試驗或校正時，固定式的必須開倉進入該空間，則必然會帶入一定量的溼空氣，對保持該空間的乾燥不利！

3． 自持稜柱型TYPE B SPB型液貨艙

由日本石川島播磨重工公司（IHI）獨立研發的自持稜柱型（TYPE B SPB型）液貨艙設計自由度高，空間浪費小，結構緊湊，相同裝載量下船舶重量最輕，儲罐數量最少，蒸發率低，而且沒有液位裝載限制，因為上甲板空間完全不受限制，所以航行控制最容易，貨艙壓力，溫度控制簡單，不可泵送的液貨量最少（約3立方米/貨艙），且對外部及內殼絕熱的維修最容易，其實是一種非常好的貨艙型式。

但是由於造價昂貴，且該公司對其專利控制極嚴，故此一直以來只有該公司自己建造的兩條使用該類型貨艙的LNG船，後來聽說又有兩條使用該型貨艙的LNG船在建造，但因為所獲信息不是很明確，所以實際情況並不瞭解。

總結：

總之，筆者目前還沒有聽說我國國內哪家造船廠獲得了MOSS型貨艙的專利技術，自然也就沒有哪家船廠建造MOSS型LNG船，

但是對於小型LNG船的TYPE C型貨艙，國內可建造的廠家有很多，且均具有很成熟的建造技術，這個對於我國小型LNG 船的建造可以說是非常有利的，遺憾的是，正如我在第一篇文章中提到的，國內對於小型LNG船的建造在經過2014年的一個小高潮之後一直沉寂至今。就在不久前的8月10日，交通部發布《關於深入推進水運行業應用液化天然氣的意見（徵求意見稿）》，力求加快推進船舶用能結構升級和港口汙染防治，深入推進水運行業應用液化天然氣清潔能源。希望該《意見》的發佈能夠為我國小型LNG船的建造再次推出一個小高潮來！為國家能源結構調整和水運綠色發展提供重要支撐。
