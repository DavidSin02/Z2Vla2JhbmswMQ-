從零開始服務器搭建wordpress網站詳細建站教程

2020-11-14
| 科技

說起第一次使用Wordpress建站，相信大部分人的印象都是虛擬主機、空間搭網站，因為便宜、省心，自己搭個小站放上去也算足夠，但是隨著各種技術的迅速發展，原來的很多虛擬主機已經不能再滿足Wordpress配置要求，再加上各種雲服務、VPS的降價，服務器性價比已經很高，無論是wordpress企業建站還是個人博客，很多人開始使用服務器搭建網站，今天就為大家分享VPS服務器建站教程。

本篇教程將詳細講解WordPress建站流程，從服務器配置、域名解析、寶塔面板、wordpress建站、網站優化等一系列操作講解建站方法，通過本教程，你學習到的不僅僅是如何成功搭建一個Wordpress網站，更多的是學習Wordpress網站優化，Wordpress可以很強大，但是前提是你已經掌握了它。

前期準備

開始要告訴大家，Wordpress建站對於普通人來說是很簡單的，而且費用也不高。首先很重要的一點Wordpress程序是免費的，使用沒有任何限制，基礎的網站投資只需要一個域名、一個服務器就足夠了，一年幾百塊左右；即便你的網站是企業站，要求的功能比較多一些，也可以通過付費主題和插件的方式來實現大部分功能，通常幾十美元就能買到非常強大的主題和插件，避免了高昂的開發費用

因為主題和插件定製性比較高，本教程只介紹WordPress網站搭建和基礎優化，需要以下準備

  • 域名：域名，推薦騰訊雲、阿里雲、Namesilo等知名商家
  • 服務器：建議使用2G內存以上服務器，推薦騰訊雲，騰訊雲產品採購季99元/年起
  • 其他：Wordpress主題、插件等，可以建站完成後配置
  • 雲服務器購買可以進：e03.cc 查看最新的優惠活動及領取代金券

服務器LNMP環境搭建(寶塔)

想要運行Wordpress網站程序，必須要有對應的軟件，也就是服務器環境，比如我們常說的LNMP就是 Linux + Nginx + Mysql + PHP 環境，這是WordPress運行的基礎，當然你不需要懂這些，在本文中我們會使用寶塔面板這款工具幫助搭建WordPress運行環境，而且後期的維護我們基於寶塔面板也會更簡單

接下來就讓我們從使用服務器命令開始，搭建Wordpress網站的環境

安裝Xshell

由於Windows是不能直接連接到Linux服務器的，需要一個SSH的軟件，推薦使用Xshell作為遠程連接軟件，它對於個人和學校是免費使用的，可以在Xshell官網直接下載。官網可能下載的步驟有些麻煩，需要填寫郵箱。

軟件安裝過程這裡就不再多說了，沒有什麼需要注意的，xshell內含簡體中文，可以切換。

連接到服務器

安裝好了Xshell軟件之後，就可以開始連接到服務器了，提前準備好服務器的IP、賬號、密碼

打開Xshell軟件，選擇文件 – 新建，添加一個連接

輸入主機IP地址和名稱(名稱可以隨意寫)，端口一般默認為22

完成以後選擇左側菜單欄中的用戶身份認證，將用戶名和密碼填入，確定，服務器添加完成，接下來就可以連接了

選擇文件 – 打開，連接到剛剛創建的服務器

如果黑色區域出現如下圖所示字符，證明連接成功，恭喜你可以在命令行控制服務器了

安裝寶塔面板

正如官網所說，寶塔面板是一個簡單、好用的面板，它的功能就是將LNMP和服務器的各種管理集成到一個可視化的WEB環境來管理，通過面板，我們普通人不需要掌握具體的技術，只需要動動鼠標，就能像專業的人一樣使用這些技術，不管是搭建還是維護網站都非常簡單

接下來我們在服務器安裝寶塔面板，輸入下面的命令並執行(最新寶塔面板需要在centos7系統用，其他系統的命令請去寶塔官網查看安裝說明)

 yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && bash install.sh

安裝中途會腳本詢問是否將網站安裝在www目錄，直接選擇y，然後確認即可，大概會需要幾分鐘的時間

最後得到了寶塔面板的登陸信息，將這些保存下來

更新：寶塔面板為了提升安全，已將面板路徑在之前的8888端口增加了隨機入口，所以最好將登錄信息長期保存，以後面板的管理都需要用到這些信息

安裝網站環境

使用剛剛獲得到的信息，瀏覽器訪問你的後臺，登陸寶塔面板(地址一般為http://ip:8888/隨機字符串，賬號密碼也就是剛剛安裝完顯示的，這裡一定要將這些登陸信息保存好)

注意：如果使用的是阿里雲之類的雲服務器提前開放安全端口，包括8888、80、443等

首次登陸寶塔面板後臺，會彈出一鍵安裝環境，在這裡我們需要耐心設置一遍

仔細看下圖中的設置，安裝環境主要有3個點

  1. 選擇LNMP環境，節省資源
  2. 調整Mysql和PHP版本
  3. 安裝方式選擇為編譯安裝

最好是按照上圖的配置安裝，可能有朋友會問為什麼不選擇 PHP7.2，這裡主要是考慮到各種主題和插件的最大兼容，如果選擇PHP7.2的話可能會有一些奇怪的問題，當然如果後期主題和插件都兼容了也可以切換到7.2。還有一點就是如果服務器為512M內存要選擇Mysql5.5，不然壓力會很大

LNMP環境編譯安裝過程大概為半小時左右，視服務器性能而定

搭建Wordpress網站

當服務器LNMP環境安裝完成之後，我們就可以開始著手搭建Wordpress網站了，這裡就是建站的主要步驟，用過虛擬主機的朋友應當非常熟悉

新建站點

選擇網站 – 添加站點，首先填入自己的域名，一般是 domain.com 和 www.domain.com 兩種格式都要綁定，並創建FTP和數據庫

創建完成後會在網站列表中顯示，這裡面的密碼記不記無所謂，可以隨時查看，後期還要通過這裡進行網站管理

域名解析

服務器已經綁定了域名，接下來就是添加解析了，其實就是添加2條A記錄，非常簡單，這裡我用的是騰訊雲域名，其他的服務商可能稍微有些不同

登陸控制檯，選擇域名註冊 – 找到自己的域名 – 解析

和服務器綁定一致，域名也是添加2條記錄，一個是www對應 www.domain.com，另一個是@，對應domain.com，全部解析到服務器的ip地址

解析完成後訪問域名，如果顯示 恭喜, 站點創建成功，就證明解析完成，可以進入下一步的網站搭建了，如果還不能訪問，稍等幾分鐘再嘗試

注意，有些國外域名解析的話生效較慢，需要等待一天左右時間

下載Wordpress安裝包

因為需要到wordpress官網下載程序，就採用的是寶塔的遠程下載功能，先教大家如何使用

我們進入寶塔後臺 – 文件，可以看到這就是服務器的文件系統，默認的/www目錄就是所有網站的目錄，可以看到剛剛我們創建的網站 wpwp.xyz，我們點擊就能進入網站裡面

可以看到網站下還很空，可以先把2個沒用的 index.html 和 404.html 刪除



在文件的上方，我們可以看到在上傳的右邊有一個遠程下載的按鈕，點擊會彈出一個對話框

這裡我們就填入wordpress最新版的下載地址，確定之後就會下載到當前的目錄

有的朋友可能不知道如何獲取下載地址，打開wordpress中文下載，在下載按鈕上右鍵 – 複製鏈接地址

等待一會下載完成後，點擊一下刷新按鈕，就能看到Wordpress程序的壓縮包

選擇右鍵 – 解壓，直接確定

解壓完成後網站根目錄會多出一個wordpress的文件夾

但這樣不能直接使用的，我們還要繼續將wordpress文件夾內的所有文件移動到網站的根目錄

選中所有文件，然後剪切，然後到網站根目錄粘貼所有

WordPress安裝

環境和程序都就緒了，接下來就可以開始安裝Wordpress網站了

訪問 www.domain.com，進入程序安裝界面，第一步選擇 現在就開始

進入到數據庫配置界面，因為我們之前新建網站時已經創建了對應網站的數據庫，直接填寫信息就行，具體的參數可以到寶塔後臺 – 數據庫查看詳細

出現如下的提示，證明數據庫信息沒有問題

接下來是配置網站的基本信息，為了安全用戶名最好不要使用admin，如果只是網站的測試階段，要勾選禁止搜索引擎，網站搭建完畢後再開放抓取

WordPress網站搭建成功，登陸 域名/wp-login.php 管理網站

至此我們的Wordpress網站已經搭建完成，如果是搭建來玩玩的話，學習一下的話，這樣已經完全足夠了

但是如果你做一個優秀的網站，並長期堅持下去，則還需要進一步優化，下面的內容就教大家一些基礎的優化，比如配置https、固定鏈接、偽靜態設置等，在真實的環境下這些基本都是必備的

WordPress優化

WordPress部署好之後，我們就可以針對網站做一些列的優化措施了

配置https

近些年來，Chrome等瀏覽器一直在推動https網站，所有的http協議網站被標記為不安全，而在今年的7月份開始，Chrome將對非https網站進一步加入明顯的非安全標記，如果這時再不對網站進行https改造的話，那麼可能會對信任度造成一定的影響，所以說對一個面向用戶的網站來說，開啟https是非常有必要的

寶塔面板開啟https也非常簡單，它將letsencrypt的免費https證書整合到自己的功能，使原本繁瑣專業的技術變得非常簡單，只需要點擊一下按鈕，網站就能開啟全站https

進入寶塔後臺 – 網站 – 設置，進入特定網站的設置界面

我們看到有一個叫做SSL的功能，這個就是添加https證書的功能

選擇 Let’s Encrypt，輸入自己的郵箱，並在2個域名前邊打鉤，確認沒問題後點擊申請

申請之後會自動運行腳本，我們稍等一下就會看到開啟成功，寶塔已經將KEY和PEM為我們填寫完畢，而且這個比較的方便的一點是，等三個月到期後會寶塔會自動續期，完全不影響我們的網站，最後記得將強制https開啟，這樣即使有用戶訪問http網頁也會自動跳轉到https格式

還有一點就是到Wordpress後臺設置一下首頁為https開頭的網址

再看網址已經顯示小綠鎖了，https開啟成功

設置固定鏈接

除了https外，另外一項非常重要的就是URL的格式，默認的URL是非常複雜且對搜索引擎不友好的，我們必須在搜索引擎抓取之前將URL設置成一種簡單的、SEO友好的格式(後期修改將會對排名產生很大影響)

而且有些插件和功能都會依賴固定鏈接，默認的鏈接格式可能不適配，所以說設定固定鏈接是非常必要的

登陸到Wordpress後臺 – 設置 – 固定鏈接，選擇一個自己想要的格式，除了樸素類型，其他的都是可以使用的，如果你比較擅長SEO的話，自定義結構會非常有用，關於WordPress固定鏈接的詳細介紹，可以查看這篇教程

比如我選擇一個數字型，再次訪問就會出現如下的URL格式，簡單、明瞭，但是嘗試的朋友可能會發現會和我一樣頁面不顯示，這時候不用著急，是因為我們還沒有配置偽靜態的緣故

回到寶塔面板的網站設置選項中，選擇偽靜態，我們將下面一段代碼填入並保存

if (-f $request_filename/index.html){rewrite (.*) $1/index.html break;}if (-f $request_filename/index.php){rewrite (.*) $1/index.php;}if (!-f $request_filename){rewrite (.*) /index.php;}rewrite /wp-admin$ $scheme://$host$uri/ permanent;

更新：還有一種方法就是代碼框的上方有一個選擇欄，如下圖，可以把當前改為 WordPress，寶塔面板會自動為你填充偽靜態代碼

再次訪問頁面，成功顯示

刪除寶塔404頁面，使用WordPress自帶404頁面

大部分Wordpress主題是自帶404頁面的，我們最好使用主題適配的404頁面，不要使用寶塔自帶的404網頁，修改方法：

修改網站配置文件，在 error_page 404 /404.html; 前邊加上 # 註釋掉

以上步驟之後，基本wordpress搭建和一些重要設置就算完成了，另外就是建議再安裝一些必備插件以及做好站點的安全防護。

WordPress建站基礎主要就是這些，後面的話就是根據自己的站點進行各種設置，不同類型的站點使用主題和插件都是很大區別的。不過如果你能學會本篇所介紹的內容，相信你的站點就已經超過了絕大部分網站。
