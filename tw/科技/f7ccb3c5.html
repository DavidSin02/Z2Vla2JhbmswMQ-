貼片二極管型號封裝介紹

2020-11-14
| 科技

貼片二極管封裝

貼片二極管

貼片二極管封裝圖：

板子上有兩個貼片二極管的封裝無法確定，分別是1N4001和4148的。元件上週我已經出去買回來了，我是第一次用貼片二極管，第一次見的時候居然分不清到底哪個是4001，哪個是4148，不得不問賣片子的那家，老闆說，大的標著M1的是 4001，小的標著W2的是4148。

​

其實吧，要是想簡單的話，直接拿個尺子量一量，直接在protel上畫封裝就行了，不過我沒有尺子，手頭上的直尺還是借對門的，件太小，不好動手量，我認真的勁又犯了，就是想找到這種貼片二極管的標準封裝名稱及相關數據手冊，然後根據標準來畫自己的封裝庫，可以先查了一下花了一個小時時間，確定與貼片二極管有關的封裝信息如下，前面幾個我可以確定，後邊的詳細資料我沒查：

SMA / SMB / SMC / MELF / MINI-MELF / DFS / MINI-DFS /SKY / SF / HER / FR / STD / TVS / SWITCH / ZEMER / DIACES

標準封裝：

SMA <---------------->2010

SMB <---------------->2114

SMC <---------------->3220

SOD123 <---------------->1206

SOD323 <---------------->0805

SOD523 <---------------->0603

還有四種封裝名稱：

DO-214AA

DO-214AB

DO-214AC

DO-213AB<-------------->MELF

現在IN4007插件（DO-41 DIP）、貼片（DO-214AC也就是SMA）都有。一般插件的芯片是O/J的，貼片的芯片的GPP的。至於符號從M1-M7都有，其擊穿電壓分別為 50、100、200、400、600、800、1000V。

而現在又有了新的一種封裝，表面上看它是DO-214AC，但是它稱為“打扁”的，看引線的根部是圓柱，而真真的DO-214AC的引線根部是和外面一樣的平。
