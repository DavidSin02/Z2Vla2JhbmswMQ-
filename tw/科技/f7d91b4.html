UKUI 桌面環境登陸 Arch Linux

2020-10-29
| 科技

繼 Debian、Ubuntu 等國際 Linux 發行版之後，又一款 Linux 操作系統將 UKUI 作為可選桌面環境。

作者/來源： 優麒麟社區

2020 年 2 月 6 日，經過 Arch Linux 社區和優麒麟社區開發人員的共同努力，UKUI 桌面環境和優客系列應用軟件已經大部分移植至 Arch Linux 操作系統。 繼 Debian、Ubuntu 等國際 Linux 發行版之後，又一款 Linux 操作系統將 UKUI 作為可選桌面環境。中國的開源產品被越來越多的國際社區接納和採用。Arch Linux 用戶將可以從軟件倉庫中安裝並享受 UKUI 桌面環境帶來的“友好易用，簡單輕鬆” 的交互體驗。

UKUI 是一款輕量級的 Linux 桌面環境，基於 GTK 和 QT 進行開發，在國內外得到廣泛關注。近期 OMGUbuntu 等開源媒體專門對 UKUI 3.0 預覽版進行了試用和報道並給予了極高評價：UKUI 無論是從設計還是交互方面都給用戶眼前一亮的感覺。

現 UKUI 已經默認使用在麒麟軟件下的優麒麟和銀河麒麟等操作系統上，用戶可以直接在優麒麟官網，或 360、華為、阿里雲、重慶大學等開源境像站下載體驗 UKUI，並且在今年 4 月發佈的優麒麟 20.04 LTS 中首次集成 UKUI3.0，敬請期待！

Arch Linux 是一個輕量的、靈活的 Linux 發行版，遵循 K.I.S.S. 原則，除支持 UKUI 之外，還有 KDE、Gnome 等桌面環境。

在 Arch 的 UKUI 地址： https://www.archlinux.org/groups/x86_64/ukui 。

打開終端，運行以下命令即可直接安裝 UKUI 軟件包組（包含了多個軟件包）：

sudo pacman -S ukui

最後，歡迎更多的 Linux 發行版移植 UKUI 桌面環境，一起共享開源成果。可聯繫 UKUI 官方郵箱： contact@ubuntukylin.com 。
