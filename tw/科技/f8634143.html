form-generator 開源Element UI表單設計及代碼生成器

最新版本：0.1.0，MIT 協議

主要技術棧

vue2

vue-cli4

element-ui

vuedraggable

功能列表

  • 可視化設計表單
  • 一鍵生成原生的vue單頁表單代碼，100%拓展性
  • 在線編輯器修改生成的代碼，並實時預覽
  • 表單柵格化佈局
  • 支持表單校驗
  • 快速查閱組件的官方文檔

    • 項目截圖

    # 下載源碼git clone https://github.com/JakHuang/form-generator.git# 進入項目cd form-generator# 安裝依賴npm install# 本地啟動npm run dev# 本地編譯npm run build
