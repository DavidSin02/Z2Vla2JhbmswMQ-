縫紉基礎知識（十一）技巧篇

2020-11-14
| 科技

九、閉合輔件

1.搭鉤及襻

（1）怎樣裝腰頭搭鉤和襻

A 將搭鉤放在腰頭搭接部分的下側，距裡側邊緣約0.3cm處，穿過每個孔眼縫3-4針將搭鉤定位，注意不要縫到服裝正面。（下圖）

B 將搭鉤重疊到需要裝襻的部分，標記處襻的位置，在襻的每個孔裡縫3-4針。（下圖）

C 圓領鉤和鉤眼用於不搭接的腰頭。（下圖）

（2）如何製作線環扣眼

A 用針引雙股線穿過織物邊緣，縫兩道基底針跡，長度按鈕釦直徑需要來定。（下圖）

B 使針眼從基底針跡線下面和線環中穿過。（下圖）

C 針穿過線環，抽緊線環扣住基底針跡線，重複過程，只到完全蓋住基底針跡。（下圖）

D 縫兩針短小的回針針跡以便鎖住針跡線，修剪掉線頭。（下圖）

2.鈕孔

（1）確定鈕孔長度

A 測量鈕孔的寬度和高度，這兩個測得值之和加上0.3cm就是機制鈕孔的合適長度。（下圖）

B 實驗鈕孔，在一塊碎布上剪一切口，長度為鈕孔長度加上0.3cm，如果鈕釦能順利穿過該孔，長度就合適了，就可用此長度在服裝、貼邊和襯頭上縫製鈕孔。（下圖）

（2）標示鈕孔

A 將紙樣放在服裝上面，紙樣的接縫線和服裝門襟邊緣對齊，用別針穿透紙樣，把紙樣慢慢從別針頭上方拉出。（下圖）

B 用別針、水溶性標記筆或膠帶等在織物上做鈕孔標記。（下圖）

（3）機制鈕孔

A 怎樣縫製鈕孔

縫製鈕孔壓腳裝置：將織物置於鎖鈕孔壓腳下面，起點和機針對齊，壓腳中心位於中心標記的上方，機器會按下面四步進行縫製。（下圖）

包縫鈕孔：1）以窄之字形針跡縫鈕孔，剪開鈕孔並修剪鬆散紗線；2）重新將鈕孔準確定位，調整加大之字形針跡寬度，再次鎖鈕孔，讓寬之字形針跡包住窄之字形針跡。（下圖）

一步縫製的鈕孔裝置（下圖）

通用縫製鈕孔附件裝置（下圖）

B 怎樣剪開鈕孔

在鈕孔兩端的鎖眼針跡上插入別針（下圖）

用尖頭剪刀或剝線器插入鈕孔中心，小心地剪至一端頭，再剪至另一端頭。（下圖）

在切口邊緣塗上防脫散劑（下圖）

3.鈕釦

（1）標示鈕釦位置

A 標示鈕孔位置時，把服裝的門襟一側搭在另一側上，對齊中心線，用別針別住兩側衣片，使其閉合。（下圖）

B 插入別針直穿透鈕孔，到下層織物，在鈕孔中心的位置。（下圖）

C 小心地將鈕孔提升到別針上方，在別針尖處插入穿有線的縫衣針釘鈕釦。（下圖）

（2）如何釘有腳鈕釦

A 剪一節線，長76cm左右，把線拉過蜂蠟，增加線的強度，將線對摺。（下圖）

B 在織物正面，鈕釦下以短小針跡固定線，引針穿過腳孔，向下插縫衣針入織物，並引線穿過，如此重複4-6次。（下圖）

C 在鈕釦下打結或縫幾針短小針跡，以便在織物中栓住線頭，剪去線端頭。（下圖）

（3）用手工釘有眼鈕釦

A 像釘有腳鈕釦那樣穿針引線，把鈕釦放在別針標示位置，使鈕釦上的孔眼與鈕孔平行，從下面使針穿透織物並一直向上穿過鈕釦的一個孔眼，插針入另一孔眼並穿透各層織物。（下圖）

B 將一根牙籤或火柴插在線和鈕釦之間形成腳，在每個孔眼內縫3-4針，將針和線都引到服裝正面鈕釦的下方打結，去除牙籤。（下圖）

C 讓線圍繞鈕釦針跡線纏繞2-3圈再打結，修剪掉線頭。（下圖）

（4）用縫紉機釘有眼鈕釦

A 裝上釘鈕釦壓腳和專用針板蓋住送布牙或卸下送布牙，用密實的之字形針跡釘鈕釦。（下圖）

B 把鈕釦放在壓腳下，轉動手輪使機針下降，插入一個孔眼的中心，降下壓腳。（下圖）

C 插入牙籤或火柴，利用之字形針跡寬度調節器將針跡寬度調到與鈕釦孔眼間的間距寬度相等，慢慢地操作直到調至正確寬度，縫上6針或更多針。（下圖）

4.暗釦

怎樣安裝普通暗釦

（1）將一顆暗釦的凸出部分放在搭接片的反面，距邊緣0.3-0.6cm，用單股線穿過每個孔眼縫針跡，只穿過貼邊和襯頭織，切勿穿透到服裝正面，縫兩針短針跡固定住線。（下圖）

B 在下搭接片的正面標示暗釦另一部分的位置，把別針固定住暗釦凹進的那部分。（下圖）

C 使暗釦凹進部分的中心，位於標記上，所縫針跡要穿透各層織物。（下圖）
