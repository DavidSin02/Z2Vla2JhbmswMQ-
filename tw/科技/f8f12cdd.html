ASCII 字符的點陣顯示

2020-11-14
| 科技


來源：百問網

作者：韋東山

本文字數：2108，閱讀時長：4分鐘

要在 LCD 中顯示一個 ASCII 字符，即英文字母這些字符，首先是要找到字符對應的點陣。在 Linux 內核源碼中有這個文件：lib\fonts\font_8x16.c，裡面以數組形式保存各個字符的點陣，比如：

上圖左側有 16 行數值，每行 1 個字節。每一個節對應右側一行中 8 個像素：像素從右邊數起，bit0 對應第 0 個像素，bit1 對應第 1 個像素，……，bit7 對應第 7 個像素。某位的值為 1 時，表示對應的像素要被點亮；值為 0 時表示對應的像素要熄滅。

所以要顯示某個字符時，根據它的 ASCII 碼在 fontdata_8x16 數組中找到它的點陣，然後取出這 16 個字節去描畫 16 行像素。

比如字符 A 的 ASCII 值是 0x41，那麼從 fontdata_8x16[0x41*16]開始取其點陣數據。


使用 GIT 下載所有源碼後，本節源碼位於如下目錄：

核心函數是 void lcd_put_ascii(int x, int y, unsigned char c)，它在 LCD 的(x,y)位置處顯示字符 c，代碼如下圖所示：

1、獲取點陣

對於字符 c，char c，它的點陣獲取方法如下：

4693 unsigned char *dots = (unsigned char *)&fontdata_8x16[c*16];


2、描點

根據上圖，我們分析下如何利用點陣在 LCD 上顯示一個英文字母。

因為有十六行，所以首先要有一個循環 16 次的大循環，然後每一行裡有 8 位，那麼在每一個大循環裡也需要一個循環 8 次的小循環。

小循環裡的判斷單行的描點情況，如果是 1，就填充白色，如果是 0 就填充黑色，如此一來，就可以顯示出黑色底，白色輪廓的英文字母。

3、main 函數

main 函數中首先要打開 LCD 設備，獲取 Framebuffer 參數，實現 lcd_put_pixel 函數；然後調用lcd_put_ascii 即可繪製字符。

代碼如下：

4、編譯 c 文件 show_ascii.c

編譯命令：arm-linux-gnueabihf-gcc -o show_ascii show_ascii.c

注意：不同的板子，編譯工具的前綴可能不一樣。


5、上機實驗

  • 把 show_ascii 程序放到板子上，執行命令：./show_ascii
  • 如果實驗成功，我們將看到屏幕中間會顯示出一個白色的字母‘A’。


6、課後作業

  • 修改 lcd_put_ascii 函數，可以指定字符顏色。
  • 實現 lcd_put_str 函數，輸出字符串，可以換行。
  • 在 show_ascii.c 的基礎上實現漢字的顯示：要找到漢字字庫、瞭解像素排列順序、得到漢字編碼。
