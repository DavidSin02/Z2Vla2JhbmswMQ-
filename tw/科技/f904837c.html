如何使用 Windows 快捷鍵提高 5 倍效率？

2020-11-14
| 科技

當我們想要掌握 Windows 快捷鍵來提高自己工作效率的時候，首先想到的是網絡搜索，結果是，Windows 快捷鍵的組合已經多得數不勝數，完全不知道該如何下手。其實，我們只需要掌握常用的十幾種即可。

首先是 windows 徽標鍵的組合用法，

windows + E 就可以直接打開資源管理器，不用到桌面去查找“ 計算機 ”圖標。

人們使用投影儀的時候總要去控制面板設置一通，其實只需要同時摁住 windows 徽標鍵 + 字母 P 就可以直接調出設置選框；

windows +字母 R 調出的是運行對話框，我們可以通過輸入相應的命令來設置自動關機、查找電腦 IP 地址、關閉開機啟動項等；

windows +方向鍵就可以讓窗口最大化、最小化、還原和停靠屏幕左右側；

win7 、win8 、win10 的電腦系統可以直接把軟件固定在任務欄上，並通過 windows + 相應的數字鍵，快速打開。比如第一個是ie瀏覽器，那麼摁住 windows + 數字 1 ，就可以打開，第二個是 QQ ，摁住 windows + 數字 2 ，就可以打開，以此類推。

Tab ，英文全稱是 tabulator ，叫做跳格鍵，因此它的作用就是跳格。比如在填寫表格的時候，使用 tab 鍵切換就比鼠標點選快很多；

Tab 和 Ctrl 、Alt 、Windows 徽標鍵組合起來都有切換窗口的作用；

使用 ctrl + tab ，可以在軟件內的窗口之間進行切換，比如 ie 瀏覽器的標籤頁，ps 軟件的圖片處理等對話框，

使用 alt + tab ，在軟件之間進行切換，摁住 alt 鍵的同時摁一次 tab ，可以切換到最近的軟件，摁兩次或多次，就可以連續切換；

使用 windows 徽標鍵 + tab ，則可以在系統所打開的全部軟件當中以另外一種方式切換， windows 7 系統中擁有 3D 切換效果，而 win8、win10 系統中有所不同；

對於數字鍵上方的 F1 到 F12 ，我們簡單介紹幾個；

F2 通常用於重命名，比鼠標點選更快；

F5 用於刷新；

F6 用於瀏覽器當中直接顯示地址欄並快速輸入網址；

平常大家都覺得 Backspace 主要用於文檔中的刪除，其實 Backspace 的意思是回退鍵，有返回、退出的意思，因此在資源管理器中查找文件的時候，不用老是點選鼠標，直接使用 Backspace 就可以返回上級文件夾；同樣的在ie瀏覽器當中使用 Backspace 就可以直接返回之前的頁面；

關於關閉功能,

關閉所有的軟件和窗口我們都可以使用 alt + F4 ；

關閉所有軟件內窗口，比如 qq 對話窗口，可以直接使用 esc ，電腦系統或軟件彈出的提示、警告窗口都可以使用 esc 快速退出；

ctrl 組合字母鍵大家再熟悉不過啦！

ctrl + a 全選；

ctrl + z 撤銷；

ctrl + x 剪切；

ctrl + c 拷貝；

ctrl + v 粘貼；

ctrl + f 查找。

# Insert #

在文檔編輯比如 Word 文檔編輯中，你偶爾可能會遇到這樣一個問題：在文本中間插入一段文字，之後的文字就被無情地覆蓋掉了。這個時候其實只需要摁下鍵盤上的 Insert 即可。因為 Insert 就是插入的意思。

# Shift+方向鍵 #

當我們把光標定位在文字當中的某一個地方時，

摁住 Shift 並連續摁左方向鍵，就可以選中光標左邊的文字，摁幾次方向鍵，就選中幾個文字；同樣摁住 Shift 並摁右方向鍵，就能選中光標右邊的文字；

另外，摁住 Shift ，同時摁住上方向鍵或下方向鍵，就能成行地選中文字。熟練使用之後，這樣的選擇方式比轉換鼠標要快很多。

# Ctrl+O #

# Ctrl+N #

在大部分軟件當中，比如 Word、PPT、PS裡，使用 Ctrl+字母O,就能直接打開文件；使用Ctrl+字母N，就可以新建一個文檔、圖層，等等等等

# Ctrl+Shift+N #

在資源管理器窗口中 摁住 Ctrl+Shift+N 能快速新建一個文件夾，如果在IE瀏覽器當中使用，可以新建一個無痕瀏覽窗口，如果你需要瀏覽不可告人的網頁，就這麼辦。

其實不同的軟件都有不同的快捷鍵機制。我們所介紹的這些快捷鍵，足以讓你事半功倍。如果你的工作需要大量用到Word文檔，就可以專門研究和學習Word快捷鍵使用方法；如果你要剪輯視頻，比如Premiere，就可以大量學習Premiere的快捷鍵使用方法。
