Ubuntu Studio 是一個流行的 Ubuntu 官方變種 ，它是為從事音頻製作、視頻、圖形、攝影和書籍出版的創意內容創建者量身定製的。它提供了許多多媒體內容創建應用，開箱即用，體驗極佳。

在最近的 20.04 LTS 版本發佈之後，Ubuntu Studio 團隊在其 官方公告 中強調了一些非常重要的內容。而且，也許不是每個人都注意到關鍵信息，即 Ubuntu Studio 的未來。

Ubuntu Studio 20.04 將是帶有 Xfce 桌面環境 的最後一個版本。將來的所有發行版都將改用 KDE Plasma 。

為什麼 Ubuntu Studio 放棄 XFCE？

據他們的澄清，Ubuntu Studio 並不專注於任何特定的外觀/感受，而是旨在提供最佳的用戶體驗。同時，KDE 被證明是一個更好的選擇。

事實證明，Plasma 為圖形藝術家和攝影師提供了更好的工具，例如在 Gwenview、Krita 甚至文件管理器 Dolphin 中都可以看出。此外，它對 Wacom 平板的支持比其他任何桌面環境都更好。

它已經變得如此之好，以至於現在大部分 Ubuntu Studio 團隊成員都在使用通過 Ubuntu Studio 安裝程序添加了 Ubuntu Studio 的 Kubuntu 作為日常工作使用。既然我們中的這麼多人都在使用 Plasma，因此在我們的下一個版本中過渡到 Plasma 似乎是個好時機。

當然，每一個桌面環境都是針對不同的用戶量身定製的。在此，他們認為 KDE Plasma 是最適合取代 XFCE 的桌面環境，可以為所有用戶提供更好的用戶體驗。

儘管我不確定用戶對此會有何反應，因為每個用戶都有不同的偏好。如果現有用戶對 KDE 沒有問題，那就沒什麼大問題。

值得注意的是，Ubuntu Studio 還提到了為什麼 KDE 可能是它們的更好選擇：

在沒有 Akonadi 的情況下，Plasma 桌面環境的資源使用與 Xfce 一樣輕量級，甚至更輕。Fedora Jam 和 KXStudio 等其他以音頻為重點的 Linux 發行版在歷史上一直使用 KDE Plasma 桌面環境，並且在音頻方面做得很好。

此外，他們還強調了 Jason Evangelho 在福布斯雜誌上的一篇文章 ，其中一些基準測試表明 KDE 幾乎與 Xfce 一樣輕量級。儘管這是一個好的跡象，我們仍然需要等待用戶試駕 KDE 驅動的 Ubuntu Studio。只有這樣，我們才能觀察到 Ubuntu Studio 放棄 XFCE 桌面環境的決定是否正確。

更改後，Ubuntu Studio 用戶將發生什麼變化？

在帶有 KDE 的 Ubuntu Studio 20.10 及更高版本上的整體工作流程可能會受到影響（或改善）。

但是，升級過程（從 20.04 到 20.10）會導致系統損壞。因此，全新安裝 Ubuntu Studio 20.10 或更高版本將是唯一的方法。

他們還提到，他們還會不斷評估與預安裝的應用是否存在重複。所以，相信在未來的日子裡，更多的細節也會隨之而來。

Ubuntu Studio 是最近一段時間內第二個改變了主桌面環境的發行版。先前， Lubuntu 從 LXDE 切換到 LXQt。
