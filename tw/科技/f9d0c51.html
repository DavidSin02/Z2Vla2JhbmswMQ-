AWS IoT Greengrass可將 AWS 無縫擴展至邊緣設備

2020-10-29
| 科技

將本地計算、消息收發、數據緩存、同步和 ML 推理功能連接至邊緣設備

AWS IoT Greengrass 可將 AWS 無縫擴展至邊緣設備，因此可以在本地操作其生成的數據，同時仍可將雲用於管理、分析和持久存儲。藉助 AWS IoT Greengrass，連接的設備可以運行 AWS Lambda 函數、基於機器學習模型執行預測，保持設備數據同步以及與其他設備安全通信 – 甚至在沒有連接互聯網的情況下也可實現這些功能。

利用 AWS IoT Greengrass，您可以使用熟悉的語言和編程模型在雲中創建和測試設備軟件，然後將其部署到設備中。您可對 AWS IoT Greengrass 進行編程，使之可篩選設備數據，並僅將必要信息傳輸回雲中。您還可以使用 AWS IoT Greengrass 連接器連接到第三方應用程序、本地軟件和即時可用的 AWS 服務。連接器還可以用預先構建的協議適配器集成快速啟動設備，並允許您通過與 AWS Secrets Manager 的集成簡化身份驗證。

優勢

近乎實時地響應本地事件

AWS IoT Greengrass 設備可在本地操作其生成的數據，因此能夠快速響應本地事件，同時仍可將雲用於管理、分析和持久存儲。本地資源訪問功能可讓部署在 AWS IoT Greengrass Core 設備上的 AWS Lambda 函數使用本地設備資源，如相機、串行端口或 GPU，這樣設備應用程序就能夠快速訪問和處理本地數據。

脫機運行

AWS IoT Greengrass 甚至能夠允許互聯設備在間歇連接到雲的情況下運行。設備重新連接後，AWS IoT Greengrass 可將設備上的數據與 AWS IoT Core 同步，不管連接情況如何，都能夠提供無縫的功能。

安全通信

AWS IoT Greengrass 可對本地和雲通信的設備數據進行身份驗證和加密，這樣一來，在身份未經證實的情況下，設備與雲之間就不絕會進行數據交換。對於在 AWS IoT Greengrass Core 和 AWS 雲之間發送的消息，以及 AWS IoT Greengrass Core 與使用 AWS IoT Device SDK 的其他本地設備之間的消息，您還可以使用硬件保護的端到端加密。

通過 AWS Lambda 實現簡化的設備編程

您可以在雲中開發代碼，然後使用 AWS Lambda 無縫地將其部署到您的設備上。藉助 AWS IoT Greengrass，您可以在本地執行 AWS Lambda 函數，從而降低了開發嵌入式軟件的複雜性。

降低運行 IoT 應用程序的成本

通過對設備進行編程以在本地過濾數據，並只將應用程序所需的數據傳輸到雲，您能以較低的成本獲得豐富的見解。這可以減少傳輸到雲的原始數據量、降低成本，並能夠提高您發送到雲中的數據的質量。

集成高質量測試工具

AWS IoT Device Tester for AWS IoT Greengrass 是一個測試自動化工具，可以加速您對 AWS IoT Greengrass 的基於 Linux 的 IoT 設備的測試。藉助 AWS IoT Device Tester，您可以輕鬆地執行測試，以確定您的設備將是否運行 AWS IoT Greengrass 並與 AWS IoT 服務交互操作。要了解更多信息並獲得該工具，請單擊此處。

工作原理

藉助 AWS IoT Greengrass，您可以構建 IoT 解決方案，從而將不同類型的設備與雲連接起來，並實現設備互連。運行 Linux 和支持 ARM 或 x86 架構的設備可以託管 AWS IoT Greengrass Core。AWS IoT Greengrass 可在本地執行 AWS Lambda 代碼、消息收發、數據緩存和安全策略。運行 AWS IoT Greengrass Core 的設備充當樞紐，與其他運行 Amazon FreeRTOS 或已安裝 AWS IoT Device SDK 的設備進行通信。這些設備的尺寸可能不同，從較小的基於微控制器的設備到大型設備。AWS IoT Greengrass Core 設備，支持 AWS IoT Device SDK 的設備，和 Amazon FreeRTOS 設備可進行配置，以在AWS IoT Greengrass 組中相互通信。如果 AWS IoT Greengrass Core 設備與雲之間的連接斷開，AWS IoT Greengrass 組中的設備還可通過本地網絡繼續相互通信。AWS IoT Greengrass 組可能代表一個建築層、一輛卡車或整個礦區。

AWS IoT Greengrass 提供預構建的連接器，因此您無需編寫代碼即可輕鬆擴展邊緣設備功能。使用 AWS IoT Greengrass 連接器使您能快速連接到邊緣的第三方應用程序、本地軟件和 AWS 服務。

AWS IoT Greengrass 為邊緣設備提供信任私有密鑰存儲的硬件根。您可以在使用 AWS IoT Greengrass 功能的同時使用硬件保護的信息加密功能。
