Nike ISPA 推出充氣式機能外套單品

2020-10-29
| 科技


2018 年時 Nike 以 ISPA 為名推出新系列，「ISPA」各有著意義（I: IMPROVISE、即興；S: SCAVENGE、整合；P: PROTECT、保護；A: ADAPT、適應），致力於改善與都市生活環境的互動關係。系列首當其衝便以鞋款出發，推出一雙 React Runner Mid WR ISPA 鞋款，從 React Element 87 延伸，並配置鞋帶鬆緊快速調節系統。

最近 Nike ISPA 繼續創新，此番便帶來一枚機能外套單品 Inflate Jacket，內置手動充氣囊系統，在乍暖還寒的天氣底下，隨意調節其保暖度，充分提高透氣性和活動性。有關這枚單品將於 3 月 13 日於 Nike 上架，想要入手的不妨多加註意。
