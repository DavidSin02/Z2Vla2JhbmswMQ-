1. 產品概述

稱重給料機（俗稱定量皮帶給料機）是一種連續稱量給料設備，用於固體物料的定量輸送。能自動按照預定的程序，依據設定給料量，自動調節皮帶轉速，使物料流量等於設定值，以恆定的給料速率連續不斷的輸送散狀物料，可用於需要連續給料的配料場合， 在國內外電力煤炭、冶金、礦山、建材、化工、輕工、港口等行業得到廣泛應用。

3. 基本構成

稱重給料機主要有機械部分、傳感器、電氣儀表部分組成。

3.1. 機械部分：

機械部分由機架、傳動裝置、傳動輥筒、稱量裝置、防偏糾偏裝置、梨形清掃器、頭部清掃器、託輥、皮帶、卸料罩、料斗、底座、標定棒等組合而成。它是定量稱重物料的承載及輸送機構。

3.2. 傳感器：

稱重給料機有稱重、測速兩種傳感器。

1、稱重傳感器

稱重傳感器將稱重輥承受的負荷轉換為電信號，提供給稱重儀表作進一步處理。稱重給料機採用本公司生產的MTB系列稱重傳感器，傳感器容量規格、數量根據稱重給料機結構形式，稱重範圍配置。

2、測速傳感器

測速傳感器將物料的運作速度轉換成脈衝信號，從而控制稱重儀表的採樣頻率。稱重給料機選用光電脈衝式測速傳感器.

3.3. 電氣儀表部分

電氣儀表部分對傳感器或變送器輸出信號進行處理；顯示被稱物料物體的計量結果；並可輸出模擬或數字電信號，從而實現所需的自動控制。

4. 結構特點

1、 皮帶更換容易。只要張緊裝置鬆開皮帶，移動一側支架，就可取下皮帶更換。

2、 減速電機採用空心軸直接套裝於主動軸上，密封性好，結構緊湊，安裝方便，使用可靠。

3、 主被動輥採用帶座軸承安裝，安裝調整靈活方便。

4、 計量精度受皮帶影響小。由於有皮帶自動張緊裝置，張力基本恆定，稱重時稱重傳感器最大位移量僅0.2mm，所以皮帶張力對計量精度的影響可忽略不計。

5、 自動糾偏裝置，皮帶自動跟隨，不產生跑偏和隆起現象。

6、 對於不同物料和不同工藝要求，採用不同的給料料斗，並有擋料板和配套的卸料溜子。

7、 梨形清掃器、膠帶清掃器及時清除夾離物，皮帶不會因粘附物料而引起皮重變化，影響稱量精度，同時可避免皮帶損傷。

8、 配有標定掛碼支架，可隨時對計量精度進行校驗。

6. 工作原理

6.1. 計量原理

物料通過稱量段平臺將負荷加載至稱重輥上，稱重輥又將作用力傳輸給稱重傳感器，稱重傳感器輸出電壓通過放大傳輸給稱重儀表微處理器，微處理器通過一系列計算得出有關的物料實際給料量、物料的累計量等數值。

微處理器計算過程：

稱量平臺長度：Lg (單位m)

有效稱量平臺長度：Leff=Lg/2 (單位m)

稱量平臺上物料重量：QB (單位kg)

皮帶單位長度物料重量：Q=QB/ Leff (單位kg/m)

皮帶速度：V (單位m/s)

物料流量：I= 3600QV= 3600（QB/ Leff）V (單位kg/h)

連續物料的輸送時間：t(單位s)

連續物料累計量：Z=∫QVdt (單位kg)

6.2. 工作原理

通過稱重傳感器測量出輸送皮帶上物料的重量信號，及速度傳感器發出的與皮帶速度成固定比例的脈衝信號。以上兩信號通過稱重儀表轉換成數字信號同時輸送給微處理器，通過微處理器計算處理得出實際給料量，並將實際給料量不斷與設定給料量對比，通過不斷改變變頻器頻率，不斷調整皮帶的輸送速度使實際給料量符合設定給料量，從而保證稱重給料機按設定的給料流量運行。

7. 稱重傳感器使用說明

7.1. 概述

傳感器是測量（或控制）信息系統的一個組成部分，它是測量變換器的一種結構總成，包含有信號能形式的變換器，位於被測對象影響因素的作用範圍之內，並接收由該對象產生的以自然轉換為標記的信息。簡單說，傳感器是把一種形式的能量轉換成另一種形式的可以進行測量和傳輸的能量裝置。

電子稱重裝置就是以測力傳感器為基礎轉換元件，靠傳感器把非電量的重力值轉換成電量傳送給控制系統，實現非電量（重力值）的電測量。電子稱重系統中使用的傳感器叫稱重傳感器。

我公司生產的稱重傳感器均為電阻應變式稱重傳感器。

7.2. 稱重傳感器的工作原理

稱重傳感器由彈性體，電阻應變敏感元件（應變片）及變換電路組成。被測的重量作用在彈性體上，彈性體產生變形，貼在彈性體上的應變片阻值隨之發生變化，然後由變化電路將電阻變化轉變成電壓變化輸出。使用時，根據傳感器標定的數據，由傳感器的電壓變化值確定出被測重量。

簡化的橋路：

彈性體受力時，G1，G4拉伸，阻值變大，G2，G3壓縮，阻值變小，惠斯通電橋不平衡產生電壓輸出變化。

實際上，傳感器均需要各項補償（溫度，靈敏度等）才能達到一定的精度以滿足使用要求。所以實際的橋路要複雜一些。

7.3. 稱重傳感器的連接

一臺電子稱重設備有的使用一個稱重傳感器，有的使用多個稱重傳感器，在使用多個稱重傳感器時採用並聯方式連接，多個稱重傳感器可共用一激勵電源。

7.4. 稱重傳感器的合理選擇

在選擇稱重傳感器時既要考慮傳感器的靈敏度，又要考慮傳感器的過載安全係數。稱重傳感器應在其額定負載的10~80％範圍內使用。但在使用過程中瞬時超載有可能發生，會造成傳感器的損壞，應在設備中使用機械式過載防護裝置。

7.5. 稱重傳感器的故障檢驗

1、 檢查是否有垂直力分流（不柔軟的電纜、輸送管等），干擾載荷方向所需的位移。

2、 檢查是否有其他因素影響機械裝置的自由移動（熱力膨脹、汙染、較大摩擦力等）。

3、 檢查是否有潮氣進入電纜接線盒。

4、 檢查電纜連接是否正常。

5、 檢查電纜是否損壞。

如以上檢查未發現問題，應與廠家聯繫。

8. 設備使用事項

8.1. 安裝

稱重給料機只有正確的安裝才能有效的保證正常使用和計量給料的準確度。

安裝過程中的注意事項：

1、設備水平運到安裝現場後開箱，按合同和裝箱單內容檢查所裝物品是否齊全正確。

2、檢查安裝基礎是否符合要求，地腳螺栓孔及法蘭孔的位置是否正確。

3、對中安裝，要使皮帶中心與進料斗、卸料口的中心線重合，以防止物料偏載。

4、調整水平。用長度不小於1000mm的平尺和條式水平尺，在皮帶上表面按橫縱方向檢測水平度，調整水平時，可用薄鋼片墊於地腳處調整，調整好後，用螺栓緊固。再複測一次水平度，以確保安裝水平。

5、保證稱重傳感器連接良好及稱重傳感器過載限位是否有效。

6、正確安裝進料斗和卸料斗。

7、調整皮帶。環形輸送帶的鬆緊在設備出廠時以調好，再檢查一下螺桿張緊結構上的標尺線是否與機架上的設定標尺線重合。如有偏差，應通過調節張緊螺母達到規定位置。在使用時應注意觀察，使標尺線保持規定位置。

8、將進料斗閘門調至合適高度，以保證給料機使用時滿料截面輸出，避免進料忽多忽少，影響給料精度。（也可在給料機使用時調節）

9、設備停機時應往上擰緊傳感器保護螺栓M8X40，頂住傳感器底部。設備工作時應往下擰鬆傳感器保護螺栓M8X40，使保護螺栓頂部與傳感器底部脫開1-2mm。

10、系統連接, 按電氣圖要求將所有電纜連接到位。

8.2. 調試與操作

1. 機械部分調整

a) 保證標定支架與稱重託輥垂直，使標棒重量垂直作用於稱重傳感器。

b) 將安全限位螺釘與稱重傳感器底部脫開，間隙2mm。

c) 檢查主被動輥帶鎖緊套的帶座軸承鎖緊套是否鎖緊，止回墊片要有一止回片嵌入鎖緊套凹槽內。

d) 保證皮帶跑偏安全限位開關擋輥與皮帶的間隙25mm。

e) 輸送皮帶安裝位置居中，運轉無跑偏現象。

f) 調整張緊裝置拉緊皮帶，使防偏糾偏裝置指示板上端面處於機架30X30方孔水平對角線處，並保證皮帶不跑偏。

g) 稱量段三輥應在同一水平面，空載運轉託輥轉動靈活。

h) 如使用擋料裝置橡膠板下沿與環行帶上表面有2mm的間隙。

i) 檢查減速電機是否注油。

j) 保證外清掃掛板與皮帶接觸。

k) 調整速度傳感器與測速齒輪端面感應距離，使速度傳感器端面與測速齒輪端面間隙小於1.5mm。

8.3. 故障事件信息

所有秤的重要功能由內部監控，如有故障，以事件信息方式輸出。

E1 Event Max Velocity Above"：速度超過設置值。

E2 Event Min Velocity Below"：速度低於設置值。

E3 Event Max Flow Above"：流量超過設置值。

E4 Event Min Flow Below"：流量低於設置值。

E5 Event Max Load Above"：載荷超過設置值。

E6 Event Min Load Below"：載荷低於設置值。

E7 Event flow OutOfTolerance"：流量超差。

S1 System emergency button"：緊急事件。

S2 System negative ready"：跑偏。

S3 System plus board error"：插件板錯誤。

S4 System v_sensor error"：速度傳感器錯誤。

Unknown Event"：未知事件。

事件的級別可以分成一下4類。

系統警報：強調顯示、停止秤並輸出警報信號。

系統警報信息通過

按鍵清除。

普通警報：顯示穩定，不停止秤體，但需要確認。

事件：提示信息，需要確認。

如果幾種情況同時出現，則根據事件到達的先後依次顯示。

系統警報必須被確認才能正常工作，也必須被依次確認。

顯示事件文本：

調出事件顯示閱讀，並確認。

8.4. 維護及保養

本設備機械結構簡單，維護及保養工作量很小，但經常合理的維護稱重給料機能提高其的使用期限。

1、 定期為主動輥、被動輥軸承注油，確保主被動輥軸承的潤滑。

2、 定期檢查減速機油位。如缺少，應及時注油。

3、 經常檢查頭部尾部清掃器刮皮是否夾料或夾尖銳物。如有，及時清除，以防皮帶劃傷。

4、 經常檢查稱重輥是否與皮帶接觸良好。如異物夾在其中，及時清除，以防影響給料精度。

5、 皮帶長期使用，磨損不能滿足工作需要，影響稱量精度，應於更換。

6、 定期進行砝碼和實物標定，以檢驗給料精度。

8.5. 潤滑表

8.6. 機械部分的維修

1、 主動輥、被動輥軸承出現異響，檢查軸承已損壞，拆下連接螺栓，更換。

2、 稱重傳感器出現損壞（電氣稱重儀表部分檢測），拆下連接螺栓，更換。

3、 各託輥軸承出現異響，檢查軸承已損壞，拆下更換。

4、 皮帶的更換。

1) 拆下卸料斗；

2) 拆下兩側的擋料裝置；

3) 拆下驅動裝置對面一側的砝碼脫架；

4) 用兩根軟繩將尾部清掃器吊於兩側機架上；

5) 鬆動張緊機構螺母，使被動輥軸承座移至 "["形框上，拆下該側的張緊裝置；

6) 拆下頭部清掃器；

7) 拆下相應側的方支架，使稱重呈懸臂狀態；

8) 將舊皮帶從相應側抽出卸下；

9) 將新皮帶從相應側套入裝好；

10) 按上述⑦-①順序，按原位置要求裝好；

11) 調整張緊裝置拉緊皮帶，使防偏糾偏裝置指示板上端面處於機架30X30方孔水平對角線處；

12) 使皮帶清掃器橡膠板的端面與皮帶表面的間隙接觸；

13) 再檢查各部分，使設備處於完好狀態。

注意：每次調整皮帶後必須通過儀表重新調零或去皮。

設備運行時一定要為減速機注油。

C． 將秤體上電氣箱打開, 再將傳感器接線盒擰開, 擰鬆密封接頭, 將傳感器信號電纜線接至儀表, 按芯線色標與主板上的符號一一對應。
