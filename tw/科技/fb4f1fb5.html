網絡集團電話(IP PBX）融合語音系統詳解

2020-10-29
| 科技

融合語音系列 IPPBX 產品是面向小微企業、大中型企業的分支機構多功能商務辦公電話系統。產品融合了網絡電話、語音、傳真、錄音等功能，併兼容 CallManager、Broadsoft、Skype for business、 華為 IMS 和 Asterisk 等多種業務平臺和終端，可提供多樣的融合通信解決方案。

此外，手機APP 和商務助理軟件，與 融合語音系統 配套使用，支持手機 分機、即時多方會議、通話記錄、通信錄點擊撥號、客戶信息管理等智能溝通功能，助力企業總機、 員工和客戶三者之間的高效率溝通，併為企業分析核心業務的數據奠定紮實的基礎。

網絡配置

在瀏覽器地址欄輸入設備的 IP 地址（如 192.168.2.240），打開登錄界面。 在登錄界面選擇身份和語言，輸入密碼登錄設備。默認管理員密碼為 admin。

1網絡參數

點擊基本設置>網絡，根據設備所處的網絡環境，設置設備的 IP 地址和相關參數。

點擊 基本設置 > 網絡，配置通過 STUN 功能獲取設備前端路由器的公網 IP 地址。

點擊基本設置>外網訪問，啟用穿透

2自動話務員

自動話務員功能是 OM 在外線（模擬外線、SIP 外線）來電接通後向主叫方播放語音導航（歡迎詞）。OM 的語音導航分為兩類，一類是單層語音導航（後續稱為一個“語音文件”），另一類是 多層級的語音導航（後續稱為“多級 IVR”）。 按下列步驟配置自動話務員所使用的語音導航： 步驟 1 點擊基本設置>總機，進入自動話務員配置界面。 步驟 2 劃分工作時段和非工作時段（默認為全工作時段）。 步驟 3 為工作時段和非工作時段選擇相應的語音導航。

3人工話務員

外線來電用戶撥 0（默認值）後，該通來電轉給人工話務員分機處理。出廠默認人工話務員分機是 設備第一個 FXS 端口所連話機，例如設備的第一個端口是 FXS 端口，則分機號碼為 200，若第一 個 FXS 端口前有 FXO 口，以第一個端口號碼為 200 依次類推。

添加人工話務員分機或修改其他相關信息，按照如下步驟配置： 步驟 1 點擊基本設置>總機，配置接線員組相關項。

4外線設置

模擬外線配置步驟如下： 步驟 1 將電信運營商提供的電話線或其他 PBX 引出的分機線連接至 OM 的 FXO 口。

步驟 2 進入外線設置>模擬外線，對線路進行配置。

IP外線OM 除了支持標準的 SIP 規範外，還支持 Skype Connect，進行 SIP 外線的設置時，需要進行選 擇。在進行 SIP 外線設置前，您需要向網絡電話運營商申請賬號。 配置步驟如下： 步驟 1 點擊外線設置>SIP 外線。 步驟 2 點擊新增，填寫新增 SIP 外線的註冊信息。 注：若設備需要註冊到多個 SIP 服務器（包括對接 Skype connect），需先在外線設 置>SIP 外線>註冊選項中開啟多平臺。

OM 可與 IMS（IP Multimedia Subsystem，IP 多媒體系統）對接。與 IMS 系統對接時，除了在 SIP 外線頁面配置賬號註冊，還需配置 IMS 相關信息。

5分機設置

模擬分機每個 FXS 端口對應一個模擬分機號，模擬分機配置步驟如下： 步驟 1 進入分機設置>模擬分機，對模擬分機進行配置管理。

IP 話機或軟電話（電腦或手機上安裝的 SIP 電話軟件）註冊到 OM 成功後可作為 IP 分機使用。 使用 IP 分機，需先在 OM 上配置分機賬號與註冊密碼，配置步驟如下： 步驟 1 點擊分機設置> IP 分機。 步驟 2 點擊新增，輸入分機賬號、註冊密碼。

6 錄音與留言

錄音

本地錄音是將錄音文件保存在設備自帶的內置存儲或與 USB 接口連接的外置存儲設備中。遠程錄音遠程錄音方式是將錄音文件存放在外部服務器上。

留言

開啟留言功能後，當分機用戶無法接聽來電時，對方可在語音提示後留下語音留言。

本地留言

步驟 1 點擊應用服務>錄音，開啟錄音功能。留言文件存放位置與錄音 相關，即如果將錄音方式選擇為內置存儲，則留言文件也將存放於內置存儲中。

步驟 2 點擊分機設置>模擬分機/IP 分機>設置，將呼叫轉移方式選擇為全部來電轉語音留言或遇 忙/無應答時轉語音留言。

步驟 3 保存設置。

遠程留言

步驟 1 選擇錄音方式為遠程錄音，配置錄音服務器地址及端口號，並配置語音留言郵件發送服務器。

步驟 2 保存設置。

步驟 3 進入分機設置>模擬分機/IP 分機>設置，配置郵箱，該郵箱將作為接收語音留言文件的收 件箱。

步驟 4 保存設置。

2. 模擬外線接入，分機採用模擬話機

3. IP外線接入，分機採用IP話機

4. IP外線接入，分機採用模擬話機

5. 模擬外線與IP外線同時接入

6. 分機一號雙機

手機App軟電話通過4G或WiFi網絡連接到企業網絡實現與 IP-PBX的互聯互通。 IP-PBX上每個分機號碼可同時配置在1臺IP話機或模擬話機+1個手機App軟電話，實現移動分機。
