選擇題熱點題型特訓---（題型一　傳統文化描述型）

2020-10-29
| 科技

1.(2018·日照高三5月校際聯考)《茶疏》中對泡茶過程有如下記載："治壺、投茶、出浴、淋壺、燙杯、釃茶、品茶……"文中涉及下列操作方法的是( )

A.溶解 B.滲析

C.蒸餾 D.分液

答案　A

解析　上述操作包含了將茶葉裡的有機物進行萃取、溶解、過濾、供人品嚐等過程，沒有涉及的操作有滲析、蒸餾、分液，選項A正確。

2.(2018·石家莊高三模擬考試)下列詩句中，加點字(詞)所指代物質的主要成分與其他三項不相同的是( )

A.柳絮飛時花滿城 B.朝如青絲暮成雪

C.狐裘不暖錦衾薄 D.春蠶到死絲方盡

答案　A

解析　A項，柳絮飛時花滿城，柳絮主要成分是纖維素，與其他三項不相同，正確； B項，朝如青絲暮成雪，青絲主要成分是蛋白質，錯誤；C項，狐裘不暖錦衾薄，狐裘主要成分是蛋白質，錯誤；D項，春蠶到死絲方盡，絲主要成分是蛋白質，錯誤。

3.《天工開物》中記載了砒霜(As2O3) 的製取：凡燒砒，下鞠(注：在地上挖砌) 土窯，納石其上，上砌曲突(注：煙筒)，以鐵釜倒懸覆突口。其下灼炭舉火。其煙氣從曲突內薰貼釜上。"文中涉及的操作為( )

A.蒸餾 B.昇華

C.乾餾 D.萃取

答案　B

解析　A項，蒸餾一般分離的是互溶的液體，根據題意，不符合蒸餾研究的對象，錯誤；B項，昇華利用固體受熱直接轉化成氣體，遇冷重新轉化成固體，屬於物理變化，根據題意，"凡燒砒……煙氣從曲突內薰貼釜上"，符合昇華原理，正確；C項，乾餾是指煤在隔絕空氣中加強熱使之分解，根據題意，不符合乾餾的定義，錯誤；D項，根據上述分析，不符合萃取的原理，錯誤。

4.三國時期曹植在《七步詩》中這樣寫到"煮豆持作羹，漉豉以為汁。萁在釜下燃，豆在釜中泣。……"，文中"漉"涉及的化學實驗基本操作是( )

A.過濾 B.分液

C.昇華 D.蒸餾

答案　A

解析　"煮豆持作羹，漉豉以為汁"意思是煮熟豆子來做豆豉而使豆子滲出汁水。所以"漉豉以為汁"採用了過濾的方法把豆豉和豆汁分開。故A符合題意。

5.古代中國常用如圖所示裝置來煉丹、熬燒酒、制花露水等。南宋張世南《遊宦紀聞》記載了民間製取花露水的方法："錫為小甑，實花一重，香骨一重，常使花多於香。 竅甑之傍，以洩汗液，以器貯之。"該裝置利用的實驗操作方法是( )

A.蒸餾 B.過濾

C.萃取 D.昇華

答案　A

解析　題幹中描述的製取花露水的方法實際上和現代蒸餾的方法是一樣的，"竅甑之傍，以洩汗液，以器貯之"，說的就是冷凝液體的過程，故本題選A。

6.(2018·南昌市高三二模)《本草綱目》中對燒酒的製作工藝有如下記載"自元時始創其法，用濃酒和糟入甄，蒸令氣上……其清如水，味極濃烈，蓋酒露也"。文中涉及的操作方法最適用於分離( )

A.NH4Cl、I2 B.KNO3、Na2SO4

C.丁醇、苯酚鈉 D.氯化鉀、碳酸鈣

答案　C

解析　由信息可知，蒸令氣上，則利用互溶混合物的沸點差異分離，該法為蒸餾。A項，NH4Cl、I2不能用蒸餾法分離，錯誤；B項，KNO3、Na2SO4均是易溶於水的固體，不能用蒸餾法分離，錯誤；C項，丁醇、苯酚鈉的沸點相差較大，可以用蒸餾法分離，正確；D項，氯化鉀、碳酸鈣分別是易溶於水和難溶於水的固體，不能用蒸餾法分離，錯誤。

7.(2018·梅州市高三總複習質檢)我國宋代《開寶本草》中記載了中藥材鐵華粉的製作方法："取鋼煅作葉如笏或團，平面磨錯令光淨，以鹽水灑之，於醋甕中陰處埋之一百日，鐵上衣生，鐵華成矣。"中藥材鐵華粉是指( )

A.醋酸亞鐵 B.冰醋酸

C.氫氧化鐵膠體 D.鐵粉

答案　A

解析　根據題目描述，製作的過程是將鋼(主要成分是單質鐵)製成片狀或粒狀，表面淨化後，加氯化鈉溶液，在醋缸中放置較長時間，在鋼的表面會生成鐵華粉。所以鐵華粉的主要成分應該是醋酸亞鐵，選項A正確。

8.(2018·南平市高三質量檢查)下列分析不正確的是( )

A."木與木相摩則然(燃)"的"然"是化學能轉變為熱能

B."司烜氏，掌以夫遂(青銅凹鏡) 取明火於日"的"夫遂"是單質

C."著作酒醴(甜酒)，爾惟麴櫱(酒麴)"的"麴櫱"是催化劑

D."浮樑巧燒瓷，顏色比瓊玖"的"瓷"是硅酸鹽產品

答案　B

解析　A項，"木與木相摩則然(燃)"，燃燒是化學能轉變為熱能，正確；B項，"夫遂"是古人用於日下取火的凹形銅鏡，屬於合金，為混合物，不是單質，錯誤；C項，"麴櫱"是酶，為催化劑，正確；D項，"瓷"是硅酸鹽產品，屬於無機非金屬材料，正確。

9.(2018·唐山市高三三模)《本草綱目》中收載"燒酒"篇："自元時始創其法，用濃酒和糟入甑，蒸令氣上……其清如水，味極濃烈，蓋酒露也。"《本草經集註》中記載有關於鑑別消石(KNO3)和樸消(Na2SO4)之法：以火燒之，紫青煙起，雲是真消石也"。文字中兩處渉及到"法"。分別是( )

A.蒸餾　焰色反應 B.萃取 昇華

C.蒸餾　丁達爾效應 D.昇華 焰色反應

答案　A

解析　由題中所給信息，"蒸令氣上"表明該方法是利用各組分沸點不同實現物質的分離，此"法"是指蒸餾，燒酒利用的是蒸餾原理；鉀元素的焰色反應為紫色。通過以上分析，A項正確。

10.(2018·安陽市高三三模)《本草述鉤元》中的"石鹼"條目下記載"採蒿蓼之屬，開窖浸水，漉起，晒乾燒灰。以原水淋汁，每百斤入粉面二三斤，久則凝澱如石，連汁貨之四方，浣衣發麵。"下列說法錯誤的是( )

A."石鹼"的有效成分為碳酸鹽

B.記載中包括石鹼的製法與用途

C.製取過程中包括浸取、過濾、煅燒、蒸發結晶操作

D."浣衣發麵"都與石鹼的鹼性有關

答案　C

解析　A項，"採蒿蓼之屬，晒乾燒灰"，說明"石鹼"成分來自植物燒成的灰中的成分，"以原水淋汁"，說明該成分易溶於水，久則凝澱如石，浣衣發麵，能去汙亦能作為發酵劑，說明"石鹼"的有效成分為碳酸鹽，正確；B項，記載中包括石鹼的製法與用途，正確；C項，製取過程中包括浸取、過濾、煅燒等操作，但沒有蒸發結晶操作，錯誤；D項，"浣衣發麵"，碳酸鹽用作去汙和發酵劑是利用碳酸鹽水解顯鹼性，正確。

11.(2018·安徽省皖南八校高三第三次聯考)縱觀古今，化學與生活皆有著密切聯繫。下列說法錯誤的是( )

A.《本草圖經》在綠礬項載："蓋此礬色綠，味酸，燒之則赤……"。因為綠礬能電離出H＋，所以有"味酸"

B.《天工開物》記載："凡白土曰堊土，為陶家精美器用"。陶是一種傳統硅酸鹽材料

C."水聲冰下嚥，沙路雪中平"描述的是水的三態變化："冰，水為之，而寒於水"，說明冰轉化為水吸熱

D.唐末《真元妙道要略》中有云："以硫黃、雄黃合硝石並蜜燒之；焰起，燒手面及燼屋舍者"，文中描述的是黑火藥的製作過程

答案　A

解析　A項，綠礬不能電離出H＋，之所以有"味酸"，是由於FeSO4是強酸弱鹼鹽，易水解而使其水溶液呈酸性，錯誤；B項，陶瓷是傳統硅酸鹽材料，屬於無機非金屬材料，正確；C項，"水聲冰下嚥，沙路雪中平"描述的是水變成冰就沒有流動的聲音了，有三態變化，"冰，水為之，而寒於水"，說明冰轉化為水吸熱，正確；D項，黑火藥是由木炭粉(C)、硫黃(S)和硝石(KNO3)按一定比例配製而成的，根據題意可以知道，題中描述的是製備黑火藥的過程，正確。

12.我國古代的化學工藝已經有相當大的成就，以下對古代文獻中有關記載的化學解釋不正確的是( )

答案　A

解析　製取KNO3要利用其溶解度隨溫度變化較大的性質採用蒸發結晶的方法；乙烯有催熟果實的作用；分離沸點不同的混合物用蒸餾的方法；硝酸具有強氧化性，能腐蝕大多數金屬。A項，"煎煉而成"涉及蒸發結晶操作，是利用KNO3的溶解度受溫度影響較大來製取KNO3，錯誤；B項，成熟木瓜能夠釋放乙烯，促進未成熟紅柿成熟，正確；C項，燒酒的釀造方法為：加熱濃酒和糟，利用沸點不同，將酒蒸出，然後用器皿盛裝冷凝後的餾分，即為蒸餾，正確；D項，根據"五金入強水皆化、惟黃金不化強水中"判斷"強水"主要成分是硝酸。
