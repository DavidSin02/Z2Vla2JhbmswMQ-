Linux 發展史

2020-10-29
| 科技


1969年

Ken Thompson開發Unix前身 Unics。

1973年

Unix誕生。

1984年

Richard Stallman發起了開發自由軟件運動，成立了自由軟件基金會（Free Software Foundation）和GNU（GNU’s not Unix）項目，其主要目標是建立一套完整自由的可移植的類Unix操作系統。GNU的幾個重要的系統：

  • GNU C(GCC)
  • GNU C Library （glibc）
  • Bash shell

1984年

Andrew S.Tanenbaum 開發了用於教學的unix系統，命名為Minix，但是僅僅用於教學。

1989年

Andrew S.Tanenbaum將Minix系統運行於x86的pc平臺。

1990年

芬蘭赫爾辛基大學學生linus Torvalds首次接觸Minix系統。

1991年

linus Torvalds開始在Minix上編寫各種驅動程序等操作系統內核組件，使用bash，gcc等工具寫了一個內核程序，可以在386機器上運行。

1991年底

linus Torvalds公開了linux內核源碼0.02版，http://www.kernel.org，僅僅是內核。

1993年

linux 1.0版本發行，linux轉向GPL版權協議。（也有說1994年的）GPL全稱是General Public License（通用公共許可）。

GPL的核心是保證任何人有共享和修改自由軟件的自由，任何人有權取得、修改和重新發布自由軟件源代碼的權利，但必須同時給出具體更改的源代碼。相對來說另一個協議LGPL則較為寬鬆，不必公開全部源代碼。

  • Ian Murdock創建Debian系統，可以算是迄今為止最遵循GNU規範的Linux系統。

1994年

Linux 1.0 的第一個商業發行版Slackware發行。

1995年

Red Hat成立，發佈Red Hat Linux 2.0。

1997

Red Hat Linux 5.0 發佈，支持Intel,Alpha,Sparc等平臺。

1999

Linux簡體中文版發行。

2004年

  • Red Hat停止桌面版研發，投向Red Hat Enterprise Linux研發。
  • 2004年，國產系統深度前身 Hiweed Linux ，是中國第一個基於 Debian 的本地化版本，同時組建社區研發團隊。
  • 2004年10月，馬克·舍特爾沃斯 發佈Ubuntu首個版本，也是以Debian發開發藍本。

2005年10月

RHEL4發佈。

2007年3月

RHEL5發佈，其開源版本CentOS開始在中國流行。

2008年

  • 2008年，Hiweed Linux正式更名為Linux Deepin，發佈了深度操作系統第一個版本。

2010年4月

RHEL6 BETA發佈。

2014年6月

RHEL7.0發佈。

2013年

2013年3月23日，工信部與Ubuntu系統開發商Canonical聯合發佈為中國市場而設計的優麒麟（Ubuntu Kylin）操作系統。

2014年

Linux Deepin再次更名為deepin，進入全球發行版排行版前20名。


2019年11月

IBM花費340億美金收購RedHat。


從1991年成立該項目開始，Torvalds一直管理著內核發行，如今所有內核發行由來自全球1,200多個組織的10,000多個開發人員提交的源代碼合併而成，並通過一組維護人員進行審核，這些維護人員負責構成項目的各個組件-從設備驅動程序到內存管理。而最近的一次開源峰會和嵌入式Linux會議上，Torvalds表示出對未來的擔心，很多內核維護人員已經五、六十歲了，找到合適的內核核心維護者正在提上了日程。

我們也希望Linux系統仍繼續強壯地生存下去，雖然發行版很多，但要避免內核分裂，那對程序來說有可能是災難。
