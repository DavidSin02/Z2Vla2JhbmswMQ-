摩擦對於螺紋聯接的影響

2020-11-14
| 科技

螺紋聯接的原理是，施加在螺栓或螺母用的扭矩轉換成夾緊力而對聯接進行緊固。在螺紋聯接中，摩擦的影響至關重要。這裡我們將要討論摩擦對於螺紋聯接的影響，可能導致的問題，怎麼做可以解決或消除這些問題。

根據ISO 16047：2005中的10.2 和10.3中公式[1]，推算出扭矩與夾緊力的計算公式：

T=F x (0.16 x P + 0.577 x d2 x µth + 0.5 x µb x Db) （1）

T=0.16 x P x F + 0.577 x d2 x µth x F + 0.5 x µb x Db x F （2）

T：安裝扭矩；F：夾緊力；P：螺距； d2：螺牙直徑；µth ：螺紋副摩擦係數；Db：支承面摩擦直徑；µb：支承面摩擦係數。

從公式（2）可以看出，螺紋聯接中，施加在緊固件上的安裝扭矩分成了三部分[2]（圖1）：

第一部分：用於產生夾緊力（0.16 x P x F），稱為夾緊扭矩（Tp）；

第二部分：用於克服內外螺紋副的摩擦（0.577 x d2 x µth x F），稱為螺紋副摩擦扭矩（Tth）；

第三部分：用於克服緊固件與被緊固零件的支承面的摩擦（0.5 x µb x Db x F ），稱為支承面摩擦扭矩（Tb）。

安裝扭矩的分配比例是，10%的安裝扭矩轉換成夾緊力來對零件進行緊固，40%的安裝扭矩用來克服螺紋副部分的摩擦，50%的安裝扭矩用來克服支承面的摩擦 [3]。可以看出，摩擦對於螺紋緊固聯接的重要性，90%的安裝扭矩用來克服螺紋聯接中的摩擦。

圖1：螺紋聯接示意圖

螺紋聯接的實際安裝過程中，往往由於支承面或者螺紋副的摩擦的變化造成各種失效問題。摩擦變小會導致螺栓在安裝過程中出現拉伸變形或者拉斷的問題。相反的，摩擦提高會導致聯接的夾緊力降低而出現聯接鬆動或者異響問題等等。在這裡，我們統稱為聯接失效。

什麼原因導致螺紋聯接中摩擦的變化？進而導致什麼問題問題？如何避免？

1) 零件（包含緊固件和被連接零件）的生產和表面處理

緊固件的冷墩精度，螺紋加工精度，牙側粗糙度和支承面（包含緊固件和被連接零件的）的粗糙度影響到螺紋的的摩擦係數，加工精度越高，表面越光潔，摩擦係數越小，反之則摩擦係數越大。

螺栓和螺母的表面處理，可以改變螺紋和支撐面的摩擦係數。現在常見的表面處理類型有電鍍鋅，電鍍鋅鎳合金，鋅鋁塗覆，鍍銅，磷化等，不同表面處理的摩擦係數不同。

從公式(1)可以看出，摩擦係數的大波動會導致夾緊力的大波動，進而引發聯接失效問題。

相反，摩擦係數的變化範圍越小，夾緊力波動就越小。目前可以在緊固件表面時，增加額外的潤滑工藝上，對摩擦係數範圍進行調整和控制，各大主機廠摩擦係數的要求一般控制在±0.03。同時，定安裝扭矩時，儘量使用模具狀態下的零件（包含緊固件和被連接零件）進行試驗，以求最真實地模擬量產階段的安裝狀態。這樣，定出來的安裝扭矩才可靠。

2) 軸向力

按我們以往的經驗，兩種固定的材料，表面的狀態不變，那麼這兩種的摩擦係數是不變的。但是在緊固件的摩擦係數試驗過程中，我們會發現緊固件的總摩擦係數是會改變的，而且呈現的規律為摩擦係數隨軸向力的加大而降低，最後趨於穩定。如圖2所示，M12螺栓軸向力與總摩擦係數的關係。

載荷是通過接觸面積的大小和變形狀態來影響摩擦力。

常規方法加工的粗糙表面, 摩擦總是發生在一部分接觸峰點上。接觸點數目和各接觸點尺寸將隨著載荷而增加,最初是接觸點尺寸增加, 隨後載荷增加主要引起接觸點數目增加。由於實際接觸面積與載荷的非線性關係,使得摩擦係數隨著載荷的增加而降低。[4]

3) 裝配條件

a. 零件組裝過程中，有潤滑油或者潤滑脂混入導致摩擦係數降低。導致螺栓的拉伸或斷裂失效。所以在沒有特別要求的情況下，請避免在螺紋聯接的安裝環境中出現潤滑油或者潤滑脂。

b. 異物（如鐵屑，焊渣以及零件表面塗層破壞堆積等）夾雜。這種情況可能會導致的問題有兩種：一種是由於異物夾雜導致支承面和螺紋副摩擦提高，相應的夾緊力降低，聯接出現鬆弛的問題。

第二種是，零件組裝的時候，螺紋副出現卡滯和滑牙，螺栓頭部或者螺母還沒有與支承面接觸安裝就停止。

有異物夾雜的裝配，可能還會伴隨著兩個種現象：一種是“囂叫”，緊固件擰緊過程中出現很刺耳的聲音。一種是“粘滑效應”，就是在緊固件擰緊的過程中，出現擰緊突然停止（粘），然後很快擰緊又繼續（滑），擰緊曲線顯示的是忽高忽低（見圖4）。這是因為在裝配的過程中，異物的夾雜導致支承面和螺牙配合處的摩擦增大，某一瞬間需要輸入的扭矩很高而導致擰緊停頓，可實際上擰緊並沒有完成。這種情況發生的時候，聯接很容易出現鬆動。

對於焊渣和塗層這種異物的覆著，我們可以塗一層特殊的塗層（PTFE）在螺紋上以防止焊渣和塗層的附著，從根本上解決問題。另外一種方法是在外螺紋上塗一層潤滑劑來降低摩擦係數，也可以解決出現的問題。另外我們也可以通過調整安裝工具的轉速來解決“粘滑效應”問題。如圖4所示，在工具的安裝轉速為55rpm的時候，“滑移效應”出現，當工具的安裝轉速調整為25rpm的時候，“滑移效應”消失。不過，通過提高還是調低工具轉速來解決“滑移效應”，需要通過試驗來定。

4) 緊固件安裝工具的轉速

安裝工具的轉速對摩擦係數也是有影響的。通常，安裝工具轉速高的話，聯接的摩擦係數會降低。在制定螺紋聯接的安裝工藝時，我們應該通過一系列的試驗來確定安裝工具的轉速。正常的生產時，不要隨便調整安裝工具的轉速，以防止影響聯接的穩定性。

5) 零件的鏽蝕

零件的腐蝕造成摩擦係數的變化。零件腐蝕之後，會在零件表面產生一些白鏽或紅鏽或者因鏽蝕造成零件表面凹凸不平，導致摩擦係數提高，最終可能導致聯接鬆弛，或者“滑牙”的問題出現。在零件安裝前，發現零件有鏽蝕的時候請不要用於裝配。

上面介紹了影響螺紋聯接中摩擦的各種因素，可以看出螺紋聯接中，零件本身的狀態，安裝條件以及安裝工藝等都會對聯接中的摩擦造成影響。

雖然螺紋聯接中的高摩擦會降低聯接中的夾緊力，但是隻要設計合理，在保證足夠夾緊力的情況下，我們可以 通過提高摩擦來防止螺紋聯接的旋轉鬆動問題。螺紋聯接的旋轉鬆動是指內外螺紋之間發生在鬆開方向的相對轉動時發生的鬆動。防止旋轉鬆動的方法之一就是增大被聯接件表面的摩擦係數[5]。
