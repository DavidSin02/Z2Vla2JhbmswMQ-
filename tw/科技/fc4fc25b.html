乾貨分享：Windows目錄結構剖析，C盤目錄常見文件夾都有什麼用？

2020-11-14
| 科技

我們天天和文件夾打交道，但是有好多的礙眼的文件夾我們能不能刪除呢，來看看今天的文章。

先問一個問題，你對你電腦的文件系統瞭解多少？

很多人只管用，但是有些時候，想要自己清理一下文件的時候，不太敢刪除，對於很多電腦普通用戶來說，C盤是一個比較神祕的地帶，裡面通常存放著系統核心文件，如果不小心進入刪除到了系統核心文件，有可能會導致系統奔潰，因此小白用戶對C盤向來不敢亂碰。

不過電腦使用久了，會遇到C盤空間越來越小，會影響電腦運行速度，這個時候我們需要清理C盤垃圾瘦身，以釋放系統盤空間。那麼究竟C盤文件哪些可以刪除，哪些不能碰呢？

下面這篇C盤文件知識掃盲，會給大家帶來一定的幫助。

Windows目錄結構剖析，C盤目錄常見文件夾

Documents and Settings/用戶：

Windows7/10中的“用戶”文件夾其實就是XP中的Documents and Settings文件夾，這裡儲存了用戶的設置，包括用戶文檔、上網瀏覽信息、配置文件等數據。

Documents and Settings/用戶常見文件夾

默認情況下位於系統分區根目錄下，其下包括：

Administrator、All Users、Default User、用戶文件夾等。All Users針對所有用戶，這裡的更改對所有用戶有效，Administrator針對系統管理員賬戶。

  • 桌面
  • 開始菜單
  • Application Data：通用應用程序數據文件夾
  • Favorites：收藏夾
  • Local Settings：保存應用程序數據、歷史和臨時文件，可清理
  • My Documents：我的文檔。

Program Files：

應用程序文件夾，一般軟件默認都安裝在這裡，也有一些系統自帶的應用程序。在Windows 7系統中，64位用戶會多出一個Program Files（X86）文件夾，這是系統中32位軟件的安裝目錄。

Program Files和Program Files (x86)這兩個目錄都是軟件程序目錄，一般來說，這個目錄是存放安裝軟件的目錄的。其中Program Files目錄為64位程序默認安裝目錄，Program Files (x86)為32位程序默認安裝目錄。為什麼32位的系統沒有Program Files目錄呢，因為Program Files默認就是當前位數系統的默認安裝目錄啦。

那麼，64位系統Program Files (x86)目錄的作用是：沒亂用，就是用來區分的位數的，但是不安裝在這個目錄也行。

32位程序需要訪問Program Files或者System32中的系統文件時，系統會自動轉向到Program Files (x86)或者SysWoW64中。

以64位軟件為例，所有安裝程序的默認的安裝目錄為C:\Program Files。

Program Files常見文件夾

  • Common Files：共用程序文件夾，用於同系列不同程序軟件共同使用或調用數據，比如微軟和Adobe的各種軟件。
  • ComPlus Applications：微軟COM+組件使用的文件夾，刪除後可能引起COM+組件不能運行。
  • DIFX:DIFX是一種高效的XML索引方法，此文件夾與數據索引有關，不可以刪除。
  • nstallShield Installation Information：專門存儲安裝程序信息的文件夾，用於某些程序的卸載和更新。
  • MSECache:MS Office運行時自動創建。
  • MSXML6.0：微軟XML解析器程序的文件夾。
  • microsoft frontpage:FrontPage是微軟推出的一款網頁設計軟件，此文件夾通常為空。
  • Movie Maker：系統自帶的一款視頻編輯軟件。
  • Outlook Express:Outlook Express 是Windows內置的郵件收發端，不用可刪。
  • Online Services：網絡服務文件夾，不能刪。
  • Windows Media Player：系統自帶的一款多媒體播放器。
  • Xerox：用作自帶的一些圖像處理軟件的臨時空間，文件夾不能刪除，但通常為空。

Program Data:

ProgramData目錄位程序數據目錄，這個目錄是隱藏的，需要設置查看隱藏文件夾才能看到。這個目錄也是垃圾文件的重災區，很多程序的配置都在這兒，並且很多軟件卸載後並沒有刪除配置，需要我們手動刪除。

我們可以在上面的圖看到，有目錄叫{7F40DE3E-8294-4E24-B2EA-80F6C6BB173C}，看起來不太友好，這個目錄並不是所謂的病毒目錄，這個是軟件的唯一標示GUID，有些軟件為了區分，會生成一個唯一的名稱以保證唯一性，那麼這個目錄，就是軟件為區分同名軟件所創建的唯一配置文件保存目錄，並且這個軟件的註冊表，很有可能也是這個名稱。

大部分的文件夾還是挺好識別的，比如Adobe目錄保存了常用的設計軟件如PS,AI,AE等軟件配置，Tencent保存了qq和微信等軟件的配置。

Windows的系統文件夾

Windows目錄，就是我們的Windows安裝目錄了，用來放置程序的使用數據、設置等文件，不建議刪除。這個目錄裡面的東西是不能亂刪除的，刪除一個文件可能就會導致電腦無法啟動了。

在Windows目錄中，我們可以看到熟悉的軟件，你的桌面，其實是一個軟件，全名叫文件資源管理器，也就是explorer.exe，我們如果結束這個程序的進程，可能你的桌面就消失了。還有regedit.exe，註冊表編輯器，notepad.exe，記事本等系統程序。

系統臨時目錄

臨時文件目錄隱藏的很深，我們很難找到，在C:\Users\當前用戶\AppData\Local\Temp目錄下，很多垃圾文件也會在此，包括使用壓縮軟件等解壓的臨時文件也在這兒。大家可以自己去看看這個目錄是不是氾濫了！

快捷打開這個目錄，win+R調出運行窗口，輸入%tmp%，回車，即可打開。裡面的文件可以刪除。

另外C盤還有以下文件，一般是隱藏狀態

  • Temp：存放系統或其他軟件臨時文件，需經常清理
  • hiberfil.sys：系統的休眠功能所佔用的硬盤空間的文件，不建議刪除
  • ·pagefile.sys：虛擬內存頁面文件，不建議刪除
  • PerfLogs:Windows 7/10系統日誌，記錄如磁盤掃描錯誤等信息，沒有必要刪除。

以上就是C盤根目錄常見文件夾，哪些文件能刪，哪些不能刪，相信大家都知道了吧，如果這篇文章對你有幫助，別忘了點贊和關注哦~~~
