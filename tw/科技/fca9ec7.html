C語言每日一練18——遞歸方法整數逆序

2020-10-29
| 科技

題目

將一個9位的整數，利用遞歸方法將其逆序排放在一個數組中。

實現代碼

/* ============================= Author      : 折騰的小大叔 Version     : Copyright   : Your copyright notice Description : ============================= */#include <stdio.h>#include <stdlib.h>void Convert(int *buff,int num){	if(num >=10)	{		Convert(buff+1,num/10);	}	*buff=num%10;}int main(void){	int num=152436897;	int buff[10]={};	Convert(buff,num);	printf("front:%d\n",num);	printf("after:");	for(int loop=0;loop<9;loop++)	{		printf("%d",buff[loop]);	}	return 0;}

運行結果

front:152436897

after:798634251
