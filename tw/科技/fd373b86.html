EDA365：器件選型時，這幾個通識的基礎知識必須掌握

2020-10-29
| 科技


器件選型的思考的深度和廣度，非常能考驗設計者的功力，同時最後也落實到產品的質量水平上。


1、綜合考慮

// 易產生應用可靠性問題的器件


1、對外界應力敏感的器件


CMOS電路：對靜電、閂鎖、浪湧敏感

小信號放大器：對過電壓、噪聲、干擾敏感

塑料封裝器件：對溼氣、熱衝擊、溫度循環敏感


2、工作應力接近電路最大應力的器件


功率器件：功率接近極限值

高壓器件：電壓接近極限值

電源電路：電壓和電流接近極限值 （電源）

高頻器件：頻率接近極限值（射頻與高速數字）

超大規模芯片：功耗接近極限值（特別是大功率的CPU、FPGA、DSP等）


3、頻率與功率都大的器件


時鐘輸出電路：在整個電路中頻率最高，且要驅動幾乎所有數字電路模塊

總線控制與驅動電路：驅動能力強，頻率高

無線收發電路中的發射機：功率和頻率接近極限值


//選用元器件要考慮的十大要素


1、電特性：元器件除了滿足裝備功能要求之外，要能經受最大施加的電應力；


2. 工作溫度範圍：元器件的額定工作溫度範圍應等於或寬於所要經受的工作溫度範圍；


3. 工藝質量與可製造性：元器件工藝成熟且穩定可控，成品率應高於規定值，封裝應能與設備組裝工藝條件相容；


4. 穩定性：在溫度、溼度、頻率、老化等變化的情況下，參數變化在允許的範圍內；


5. 壽命：工作壽命或貯存壽命應不短於使用它們的設備的預計壽命；


6. 環境適應性：應能良好地工作於各種使用環境，特別是如潮熱、鹽霧、沙塵、酸雨、黴菌、輻射、高海拔等特殊環境；


7. 失效模式：對元器件的典型失效模式和失效機理應有充分了解；


8. 可維修性：應考慮安裝、拆卸、更換是否方便以及所需要的工具和熟練等級；


9. 可用性：供貨商多於1個，供貨週期滿足設備製造計劃進度，能保證元器件失 效時的及時更換要求等；


10. 成本：在能同時滿足所要求的性能、壽命和環境制約條件下，考慮採用性價比高的元器件；


//失效模式及其分佈


1、失效模式：元器件的失效形式，即是怎麼樣失效的？


2、失效機理：元器件的失效原因，即是為什麼失效的？


3、元器件的使用者即使不能瞭解失效機理，也應該瞭解失效模式；


4、失效模式分佈：如果元器件有多種失效模式，則各種失效模式發生的概率是進行失效分析的前提。



// 高可靠元器件的特徵


1、製造商認證：生產廠商通過了權威部門的合格認證；


2、生產線認證：產品只能在認證合格的專用生產線上生產；


3、可靠性檢驗：產品進行並通過了一系列的性能和可靠性試驗，100%篩選和質量一致性檢驗；


4、工藝控制水平：產品的生產過程得到了嚴格的控制，成品率高；


5、標準化程度：產品的生產和檢驗符合國際、國家或行業通用規範及詳細規範要求。


//品種型號的優先選用規則


優先選用標準的、通用的、系列化的元器件，慎重選用新品種和非標準器件，最大限度地壓縮元器件的品種規格和承製單位的數量。


優先選用列入元器件優選目錄。


優先選用器件製造技術成熟的產品，選用能長期、連續、穩定、大批量供貨且成品率高的定點供貨單位。


優先選用能提供完善的工藝控制數據、可靠性應用指南或使用規範的廠家產品。


在質量等級相當的前提下，優先選用集成度高的器件，少選用分立器件。



// 供貨商應提供的可靠性信息


1、詳細規範及符合的標準：國軍標、國標、行標、企標；


2、認證情況：QPL、QML、PPL、IECQ等；


3、質量等級與可靠性水平：失效率、壽命（MTTF）、抗靜電強度、抗輻照水平等；


4、可靠性試驗數據：加速與現場，環境與壽命，近期及以往，所採用的試驗方法與數據處理方法；


5、成品率數據：中測（裸片）、總測（封裝後）等；


6、質量一致性數據：批次間，晶圓間，芯片間，平均值、方差、分佈；


7、工藝穩定性數據：統計工藝控制（SPC）數據，批量生產情況；


8、採用的工藝和材料：最好能提供關鍵工藝和材料的主要參數指標；


9、使用手冊與操作規範：典型應用電路、可靠性防護方法等。


2、工藝考慮


// 以集成電路為例：


1、最小線條：0.35、0.25、0.18、0.13μm；


2、襯底材料：Si>SOI>SiGe>GaAs>SiC；


3、互連材料：Cu>Al（國外先進工藝）Al>Cu（國內現有工藝）；


4、鈍化材料：SiN>PSG>聚烯亞胺 無機>有機 ；


5、鍵合材料：Au>Cu>Al(Si）；


6、電路形式：數/模分離>數/模合一 RF/BB分離>RF/BB合一。


// CMOS芯片成品率與可靠性的關係


成品率（有時稱為質量）：出廠或老化篩選中在批量器件發現的合格器 件數 可靠性：經歷一年以上的上機時間後的失效器件數。


一般而言，器件的質量與成品率越高，可靠性越好。但質量與成品率相 同的器件，可靠性並非完全相同。


//SPC數據:合格率的表徵


1、統計工藝控制；


2、工藝準確度和工藝穩定性是決定產品成品率和可靠性的重要因素，可用統計工藝控制（SPC，Statistical Process Control）數據來定量表徵；


3、合格率的表徵參數；


4、成品率(yield）：批產品中合格品所佔百分比；


5、ppm（parts per million）：每一百萬個產品中不合格品的數量，適6合於批量大、質量穩定、成品率高的產品表徵；


7、工藝偏移和離散的表徵 ；


8、不合格品的產生主要來自元器件製造工藝不可避免地存在著的偏移和離散；


9、工藝參數的分佈通常滿足正態分佈。


3、封裝考慮


//寄生參數典型值


1、有引腳元件：寄生電感1nH/mm/引腳（越短越好），寄生電容4pF/引腳 ；


2、無引腳元件：寄生電感0.5nH/端口，寄生電容0.3pF/端口；


3、不同封裝形式寄生效應的比較（寄生參數由小到大）；


4、無引腳貼裝>表面貼裝>放射狀引腳>軸面平行引腳；


5、CSP>BGA>QFP>SMD>DIP。



電容器的寄生電感還與電容器的封裝形式有關。管腳寬長比越大，寄生電感越小。



// 有利於可靠性


1、引線極短：降低了分佈電感和電容，提高了抗干擾能力和電路速度；


2、機械強度高：提高了電路抗振動和衝擊的能力；


3、裝配一致性好：成品率高，參數離散性小。


// 不利於可靠性


1、材料不匹配性增加：某些陶瓷基材的SMT元件（如某些電阻器、電容器、 無引線芯片載體LCC）與PCB基板環氧玻璃的熱膨脹係數不匹配，引發熱 應力失效；


2、較易汙染：SMT元件與PCB板之間不易清潔，易駐留焊劑的汙染物，需採 用特殊的處理方法；


3、表面貼裝對可靠性是利遠大於弊，目前已佔了90％的比例。


// 封裝材料的比較


1、塑料封裝


優點：成本低（約為陶瓷封裝的55％），重量輕（約為陶瓷封裝的 1/2），管腳數多，高頻寄生效應弱，便於自動化生產 ；


缺點：氣密性差，吸潮，不易散熱，易老化，對靜電敏感 ；


適用性：大多數半導體分立器件與集成電路常規產品。


2、陶瓷封裝


優點：氣密性好，散熱能力強（熱導率高），高頻絕緣性能好，承受功率 大，佈線密度高；


缺點：成本高；


適用性：航空、航天、軍事等高端市場。


3、金屬封裝


優點：氣密性好，散熱能力強，具有電磁屏蔽能力，可靠性高；


缺點：成本高，管腳數有限；


適用性：小規模高可靠器件；


通常稱塑封為非氣密封裝，陶瓷和金屬為氣密封裝。


4、吸潮性問題


塑料封裝所採用環氧樹脂材料本身具有吸潮性，溼氣容易在其表面吸附。


水汽會引起塑封材料自身的蠕變，如入侵到芯片內部，則會導致腐蝕以及表面沾汙。


5、氣密性問題


塑料管殼與金屬引線框架、半導體芯片等材料的熱膨脹係數的差異要大得多(與陶瓷及金屬管殼相比)→溫度變化時會在材料界面產生相當大的機械應力→界面處產生縫隙→導致氣密性劣化 水汽在縫隙處聚集→溫度上升時迅速汽化而膨脹→界面應力進一步加大→有可 能使塑封體爆裂（“爆米花”效應）。


PCB再流焊時溫度可在5～40s內上升到205～250 ℃ ，上升梯度達到1～2 ℃/s ，容易產生上述效應。


6、溫度適應性問題


塑封材料的玻璃化轉換溫度為130～160 ℃ ，超過此溫度後塑封材料會軟化，對氣密性也有不利影響。


商用塑封器件的溫度範圍一般為0～70 ℃ 、-40~+85 ℃ 、－40～+125℃ ，難以達到軍用溫度範圍-55~+125 ℃。
