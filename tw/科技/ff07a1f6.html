前言

二叉樹遍歷（Traversing binary tree）是指從根節點觸發，按照某種次序依次訪問二叉樹中所有的結點，使得每個結點被依次訪問且僅僅被訪問一次。

二叉樹的遍歷有四種方式，分別為：先序遍歷、中序遍歷、後序遍歷、層次遍歷。

二、二叉樹的遍歷講解

接下來我們這幾種遍歷算法，一個一個的認識。

四種遍歷的概念：

（1）先序遍歷：先訪問根節點，再訪問左子樹，最後訪問右子樹；可以說為”根-左-右“。

（2）後序遍歷：先左子樹，再右子樹，最後根節點；可以說為”左-右-根“。

（3）中序遍歷：先左子樹，再根節點，最後右子樹；可以說為”左-根-右“。

（4）層序遍歷：每一層從左到右訪問每一個節點。

（2）非遞歸實現

1.首先申請一個新的棧，記為stack；

2.聲明一個結點treeNode，讓其指向root結點；

3.如果treeNode的不為空，將treeNode的值打印，並將treeNode入棧，然後讓treeNode指向4.treeNode的左結點，

5.重複步驟3，直到treenode為空；

6.然後出棧，讓treeNode指向treeNode的右孩子

7.重複步驟3，直到stack為空.

（2）非遞歸實現

1.申請一個新棧，記為stack，申請一個變量treeNode，初始時令treeNode為頭節點；

2.先把treeNode節點壓入棧中，對以treeNode節點為頭的整棵子樹來說，依次把整棵樹的左子樹壓入棧中，即不斷令treeNode=treeNode.leftChild，然後重複步驟2；

3.不斷重複步驟2，直到發現treeNode為空，此時從stack中彈出一個節點記為treeNode，打印node的值，並讓treeNode= treeNode.right，然後繼續重複步驟2；

當stack為空並且treeNode為空時結束

    /** 左-->根-->右     * 非遞歸實現中序遍歷     * @param root     */    public static void MidOrderStack(TreeNode root){        Stack<TreeNode> stack = new Stack<TreeNode>();        TreeNode treeNode = root;        while(treeNode!=null || !stack.isEmpty()){            while(treeNode != null){                stack.push(treeNode);                treeNode = treeNode.left;            }            if(!stack.isEmpty()){                treeNode = stack.pop();                System.out.print(treeNode.val+" ");                treeNode = treeNode.right;            }        }    }

（四）層次遍歷

1.首先申請一個新的隊列，記為queue；

2.將根結點root壓入queue中；

3.每次從queue中出隊，並賦值給root，作為下一次的根，然後打印root.val的值，如果root左孩子4.不為空，則將左孩子入隊；如果root的右孩子不為空，則將右孩子入隊；

重複步驟3，直到queue為空。

/*** 層次遍歷* @param root*/public static void levelOrder(TreeNode root){    LinkedList<TreeNode> queue = new LinkedList<>();        queue.add(root);        while(!queue.isEmpty()){        root = queue.pop();        System.out.print(root.val+" ");        if(root.left!=null)            queue.add(root.left);        if(root.right!=null)            queue.add(root.right);    }}

三、總結

遍歷二叉樹可以使用遞歸的方式和非遞歸的方式來遍歷。

（2）後序遍歷：先左子樹，再右子樹，最後根節點；可以說為”左-右-根“。

（3）中序遍歷：先左子樹，再根節點，最後右子樹；可以說為”左-根-右“。

（4）層序遍歷：每一層從左到右訪問每一個節點。
