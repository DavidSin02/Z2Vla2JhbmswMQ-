打造中國設計 2018年秋季錘子科技發佈會亮相眾多生態鏈新品

2020-10-29
| 科技

2018年11月6日，錘子科技在成都舉行秋季新品發佈會，此次發佈會與以往有所不同，此次發佈會並沒有像發佈手機等科技產品，而是推出全新的錘子生態鏈產品，其中產品包括與友商合作推出的暢呼吸加溼器、聲盼科技大衛和希瑞智能音箱 D1、地平線 8 號旅行箱系列等多款產品。



在發佈會現場，羅永浩首先針對有關成都分公司解散的消息進行了闢謠，羅永浩表示目前成都一切正常，上海、深圳、成都的研發資源分處在三地，比較分散，效率較低，所以進行重新整合，在成都研發中心的一部分人員整合到了北京，另一部分則作裁員處理。羅永浩還表示，上海研發中心也會在不久後整合到北京。



在闢謠之後，羅永浩首先引用了PChome電腦之家、騰訊科技等國內媒體平臺對堅果R1的點評，表述了堅固R1的市場認可度，隨後開始了此次發佈會的新品發佈歡節。



本次發佈會帶來了暢呼吸、大衛和希瑞智能音箱、以LEVEL8三個品牌的新品。在介紹暢呼吸加溼器之前，羅永浩首先針對北京的氣候平均數據進行了一個介紹，進而說明北京的氣候平均溼度不足以滿足人體舒適的45%~65%相對溼度，此時就需要一個空氣加溼器來彌補，帶來舒適的居住環境。



最新推出的暢呼吸加溼器也同樣遵循著暢呼吸系列產品的一貫特性，並且採用了全球知名的設計公司Smartisan ID Studio操刀打造，暢呼吸空氣加溼器一共由兩個型號，包括暢呼吸落地式加溼器以及暢呼吸桌面式加溼器。



兩者主要區別在於前者擁有1200毫升的超大容量，最大蓄水量高達9盛，高速擋位下可揮發7個小時。後者則蓄水4升，適合在桌面小範圍內使用。兩款產品均採用了水泵式吸水設計、冷蒸發方式，246°進風，頂部出風的立體循環設計能夠發揮出更好的加溼性能。其中暢呼吸智能落地式加溼器售價 1999 元，11月25日開始發貨，現已接受全款預定。暢呼吸桌面式加溼器售價 999 元，12月25日開始發貨。





除了兩款加溼器之外，老羅還發布了一個可監測溫度、溼度的配件——藍牙溼溫計，售價為99元。



隨後，羅永浩發佈了第二款由錘子科技控股的聲盼網絡科技有限公司打造的大衛和希瑞智能音箱，同樣是由Smartisan ID Studio操刀設計。搶眼的外觀設計吸引了現場的陣陣尖叫，機身上半部為鏡面金屬、下半部則為精密織物，類似於一個“閃念膠囊”樣式。



在智能交互方面，除了可支持呼喊“耿大衛”喚醒智能助手外，還可以採用觸摸和藍牙按鈕的方式，在無噪音環境喚醒率99.1%，有噪音時候喚醒率達96.7%。語音助手方面由獵豹傅盛的落戶星空公司提供，初期將預設四種聲音（兩個性別、兩種性格）。



在發佈會現場，羅永浩還針對這款音箱，介紹了其與錘子科技研發的子彈短信相結合。羅永浩老羅表示他經常錯過和母親通電話，但有了這個功能長按後說話，這條語音信息就可以發送到綁定的子彈短信賬號上，在子彈短信上回復後信息就回傳到大衛和希瑞智能音箱D1上，並伴隨紅色的燈光提示，再次長按就可以將語音播放。



除此之外，大衛和希瑞的智能音箱D1內置了1700萬首在線音樂，可支持用戶在空閒時多個享受不同種類的音樂播放。在價格方面，大衛和希瑞智能音箱D1售價899元，由於需要調試和優化虛擬助手體驗，要等到明年春季上市。



經歷短暫的高潮之後，最終壓軸的產品，來自用錘科戰略合資公司成都遠行客科技發佈的三款旅行箱。思巴克運動用品曾是一家為阿迪達斯等國際品牌提供服務的代工廠，具有較高的品牌特性，此次推出的旅行箱主打商務、品質和年輕用戶。



推出的地平線8號商務旅行箱分為兩個版本，均由Smartisan ID Studio設計。兩款產品均採用了陽極氧化鋁箱身、鋁鎂合金拉桿及護角、黃色“8”Logo靜音萬向輪、TSA海關鎖等設計。與標準版不同的是，旗艦版具有充電寶、前開門、防遺忘標籤和充電插頭等四個特殊功能。



地平線8號（Level 8）旅行箱旗艦版的正面可以直接打開，完美解決了出差安檢時電子產品檢查的煩惱，而內部的防遺忘標籤是用特殊設計的14個小條，配合40多個標有褲子、鑰匙、雨傘等圖標的小蓋，可以實現下拉呈現綠色的方式來提醒你不要忘記帶東西。



地平線8號（Level 8）旅行箱旗艦版的充電寶採用按下彈出的方式，同時旁邊的三合一線纜採用回力設計，用完一拉就可以收回箱體，完美提供隨時充電功能。



除了旗艦版與標準版之外，錘子科技為年輕的用戶還推出了地平線8號（Level 8）旅行箱，採用了不同的來自拜耳的PC材質，整體更加輕盈耐用。旗艦版上的黑科技並沒有得以保留，不過老羅表示地平線8號的三款產品的拉桿與萬向輪均是相同的配備，完全不必為品質可擔心。



平線8號的售價，商務旅行箱旗艦版售價1999元，2019年4月起售，商務旅行箱標準版1199元，標準版銀色售價999元。PC材質的旅行箱售價299元，後面二者都是即日起售。



在發佈會的最後，錘子科技還為堅果R1帶來了孔雀藍全新配色，產品定價3299元和3799元。



另外在此次放不會，羅永浩提到最多的詞就是Smartisan ID Studio設計，針對中國製造等問題，老羅表示希望通過科設計理念，聯合中國製造企業走出國門，讓世界見證中國設計的魅力。相信在未來，錘子科技會給我們帶來更多的設計驚喜，將中國設計引領走向全世界。
