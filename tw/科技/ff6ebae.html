詩月亮臨屏詩群“每日一詩”第117期

詩題：《峰》

出題人：愛情石

/

1.峰

文/黃少先

你高聳入雲

巍然屹立在群山之巔

春夏秋冬千萬年

愛情石點評：詩人黃少先的詩唯美大氣，不僅高聳入雲，巍然屹立，且立於天地之間千萬年！很棒！開了好頭！

/

2. 峰

文/順其自然(甘肅)

高聳在群山之巔

吶喊

震盪天宇

不畏嚴寒酷暑

懷豁達的胸襟

笑看四季勝景

陽光沐你成長

藍天伴你入眠

蟲鳥為你吟唱

溪水為你奏樂

深谷是你的驕傲

翠鬱是你的外衣

堅守一片丹心

在最高深的境界裡

迎風頂立

陶醉在雲海裡

愛情石點評：女詩人順其自然的詩秀氣，清雅，從女性角度來描述的“峰”有另外一種精彩，且在最高深的境界裡，迎風頂立，陶醉在雲海之中。美!

/

3.峰

文／風（浙江）

塔尖上的孤獨

雲知道，天知道

極高之處

往往，身不由己

更有虛無之風

成為，峰的立體

天下山峰

高低不同

你有你的峰

我有我的峰

其實，峰

是一個載體而已

愛情石點評：詩人風從哲思的角度，從深層次把握心中的“峰”寫得很棒，且令人眼睛一亮，哲理之光給人的啟迪，遠遠超過“峰”，顯示詩人作為蘭苑文學第一批簽約作家的獨立思辯能力，精彩！

/

4.峰

文/荷影雙雙

你一直是我瞧盼的頂

努力的攀登

守候著你的魂

愛情石點評：詩人荷影雙雙的微詩寫得很坦誠且深入，努力攀登(顯示詩人的辛勤！)守候著你的魂(寫得相當深情！)為詩人點贊喝彩!

/

5.峰

文/牛亞梅

每天 給自己一個高度

不懈的攀爬

達頂

愛情石點評：詩人牛亞梅以“峰”作為奮鬥目標，寫出另一種精彩，心中有夢，總會有實現之日，微詩以小見大，且文字凝鍊！棒!

/

6.峰

文/鄒以浩

你高高地挺立

俯視著山川大地

你一身傲骨

但卻沒有傲氣

因為你知道

離開腳下的土地

只是一堆堆土石

愛情石點評：詩人鄒以浩的小詩融入詩人獨特的體驗，有哲理啟迪，發人深省。精彩！

/

7.峰

文/吳延安

打一天

與藍天相識

就把夢想寫進了雲裡

攀登了多少年

山有高度

夢想無邊

無法抵達

才萌生了一種力

挺拔如峰

愛情石點評：詩人延安的詩因為“無法抵達/才萌生了一種力/挺拔如峰”詩人的詩，的確有其獨特之處。只要堅持下去，一定有更多收穫！

/

8.峰

文/成永軍

雲山霧罩

勝似仙境

各大門派爭奪之地

技高一籌

壓倒群雄

實力決定高度

要想真正成為霸主

有堅如磐石的基礎

高山仰止的氣勢

一覽眾山小的胸懷

愛情石點評：詩人成永軍最後領略到一覽眾山小的情懷，應該為詩人點贊喝彩。就是要寫出與眾不同，才能更體現自己的精彩！希望今天的“峰”成為詩人的一個良好開始。加油！

/

9.峰

文/侯國廷/山東

敞開胸懷

接受藍天的檢閱

征服 一個又一個高度

愛情石點評：詩人侯國廷以“峰”自喻，敞開胸懷，自覺接受藍天的檢閱，征服一個又一個高度！“峰”的風彩盡顯，為詩人點贊！

/

10.峰

文/幸福小婦人

心徜徉蔚藍

雲柔柔作伴

夢 在眾生仰望的高度實現

愛情石點評：詩人幸福小婦人的詩“心”，“雲”，“夢”映襯了“峰”的威儀，夢的實現，其實就是眾生仰望的高度實現。祝福詩人！

/

11. 峰

文/ 蕓 艸

仰望峨眉、我把自己站成山腳下的仙姝，

等了一個世紀的輪迴，

才有機會、沿著崎嶇的山路前行，

去尋找那靈秀空曠的山峰中的妙緣；

峰、突兀在金頂的斜坡、雷洞坪的側面，

當你看清楚了它的容顏、它已隱去了真名，

透過千變萬化的雲蒸霞蔚、已模糊了視線，

它習慣了鎮守了千年的山川、孤獨的守望；

輕輕靠近你、在奇花異草裡前路漫漫，

在古道里探祕、看見杜鵑啼血的形式，

往前一步是藤蔓的匍匐、卻牽住千仞的尺壁，

深谷悠悠、疑是滿盈卻是虛無的懸空；

從此、把峰口的珙桐花刻在心底，

一生一世、只需一次相見就足夠，

高高在上、你的樣子，

像峨眉山洗象池的皎月、長長久久。

愛情石點評：詩人蕓艸的詩，“峰”既是一種高度，詩人用其女詩人的視角融入了諸多元素，意象之美，烘托出意境之美。最後一句“像峨眉山洗象池的皎月、長長久久。”令人嚮往！真的了不起!

/

12.峰

文/徐小平

努力托起負重的身軀

高些再高些

心中的信念一直都在

與雲擁抱

與鷹接吻

做一個逍遙的君子

沙石不斷的扯裂

所有的痛

都在仰望中融化在

深藍裡的天空

愛情石點評：詩人徐小平的清新小詩，最打動我的便是詩的後三句“所有的痛/都在仰望中融化在/深藍裡的天空。”非常棒！祝賀祝福詩人!

/

13.峰

文/鄭瀚瑋

十萬風雪來戰

險處煙雲四起

勇者必勝

愛情石點評：詩人鄭瀚瑋的微詩，體現一種狹路相逢勇者勝的精神！“峰”的神勇躍然紙上，是一首很棒的微詩。祝賀詩人！

/

14.峰

文/東榮

灑脫而溫柔的女神

騰雲駕霧的漢子

一直追求你

愛情石點評：詩人東榮人格詩化，將“峰”賦予男子漢的生命的色彩，騰雲駕霧追求心中的“女神”很浪漫，很灑脫！棒!

/

15.峰

文/星空物語

沉默中愈加偉岸

站在父親的肩頭

揮一把雲彩 望斷天涯路

愛情石點評：詩人星空物語的微詩也是以“峰”喻人，寫出了屬於“峰”獨有的精彩，“揮一把雲彩/望斷天涯路”。出彩！

/

16.峰

文/無弦

鳥聲中醒來

當朝霞落入眼眸

一顆露珠，於掌心中顫抖

愛情石點評：詩人無弦的微詩帶著幾分朦朧，體現一個女詩人的細緻入微，以一顆露珠折射陽光。詩人的“峰”雖然不說，已然在詩人心中。巧妙！

/

17.峰

文/趙書周

多少次在你面前駐足，仰望

無限風光

令人神往

想飛上你

卻沒翅膀

你依然驕傲

不為任何所動

不會府下身來

為弱者卑躬

我才明白

原來可以用雙腳丈量你的高度

以及這座山與那座山的距離

峰，便是我的高度

愛情石點評：詩人趙書周以小詩來寫“峰”總體還是不錯的。詩人自信“原來可以用雙腳丈量你的高度/以及這座山與那座山的距離/峰，便是我的高度”同時體現了自信與自我意識的覺醒。棒!

/

18.峰

文/一樹繁花

可以沉默到不聞不問

所有的高高低低 都在心裡

不媚不俗 不倨不傲

愛情石點評：詩人一樹繁花寫得既坦誠又有幾分含蓄，可以沉默/高低在心，也可以“不媚不俗，不倨不傲”。棒！

/

19.峰

文/冷月無聲

風兒輕拂 云為裳

笑看世間滄桑

巋然屹立 億萬年

愛情石點評：詩人冷月無聲用擬人誇張手法，將“峰”寫得非常精彩。“風”，“雲”意象用得精準，笑看(恰到好處!)世間滄桑，巋然不動，億萬年！(誇張用到極至無窮盡！真的非常棒！)

/

20.峰

文/心靜如水

歲月

沉澱了你的厚重

在

綿延雲端的高度

俯視著萬物

愛情石點評：詩人心靜如水的詩從歲月入詩，沉澱了你的厚重，“峰”的形象躍然紙上，且在“綿延雲端的高度/俯視著萬物(俯視用得好！烘托“峰”的高度與威儀)全詩意境優美，精彩！

/

21.峰

文 /紫霞

雲堤滿山坳

放眼峰的高度

在眼裡更在心裡

愛情石點評：詩人紫霞的詩意象用得棒，意境營造得美！語言富有張力，是一首挺美的微詩！今天微詩開局，又以微詩收官。二十一位詩人從不同角度將“峰”的精彩演繹出來，真的挺好的。統一為詩人們點贊!祝賀祝福!
