【九州文學】吟梅詠雪百絕刊（第二期）

江南雪

作者：沙鷗

皚皚白雪北疆漫，飄到江南又一年。

淡抹濃妝皆美景，天公試比看誰妍？

雪

作者：沙鷗

飛雪狂風同舞蹈，臘梅茶道共香飄。

迎春五彩銀裝綴，暮色夕霞分外嬈。

暗香

作者：沙鷗

千古暗香花獨放，凜然不畏雪堆枝。

寒宵同作羅浮夢，孤月清風寄墨詩。

梅

作者：沙鷗

影瘦孤枝獨自栽，一花綻後百花開。

傲霜錚骨迷香冷，待等春風送醉來。

/

詠梅

作者：雨載鷹

寂寞深居苦自磨，丹腮玉蕊有誰呵？

芬香褪盡花飛去，冷夜芳心月照柯。

詠梅

作者：雨載鷹

寂寞寒梅志不降，孤將雪夜一肩扛。

春風若解瓊英意，早把花聲報老樁。

詠雪

作者：雨載鷹

玉屑拍肩心快哉，梨花誰見冷冬開。

揚揚灑灑輕飄絮，卻是如期應約來。

詠雪

作者：雨載鷹

寒風止步日初晴，一片蒼茫雪紮營。

放眼神州無限美，江山競是玉鋪成。

/

雪馥梅

作者：滄海一聲笑

萬木蕭條臘月天，紅梅一樹報新年。

枝頭朵朵皆春夢，盡把香思付雪前。

雪

作者：滄海一聲笑

北風呼嘯破天來，千里銀花遍地開。

玉潔無需爭沃土，人間處處似瑤臺。

踏雪尋梅

作者：滄海一聲笑

一夜銀花遍地開，寒香約我故園來。

深深淺淺因酬夢，不為清高登月臺。

梅雪吟

作者：滄海一聲笑

有雪無梅盡是愁，有梅無雪空悠悠。

梅開雪怒情方好，雪覆梅花韻自流。


​

詠雪

作者：流星雨

仙女散花飄玉帳，嫦娥引線鏽銀袍。

真顏素雅山河壯，傲骨豪情展節操。

詠雪

作者：流星雨

瑞雪紛紛詩入畫，灑鋪滿地潔瑩花。

輕歌漫舞懷豪志，施展雄威繡錦華。

早春雪景

作者：流星雨

寒風凍雨逞雄威，燕雀歸巢蝶止飛。

碧柳蒼松披白絮，桃花失意竹枝肥。

鵝毛雪

作者：流星雨

寒來一夜冬風緊，喜看鵝毛落太空。

萬樹梨花瀟灑舞，九千銀界似煙籠。

/

小雪

作者：通明

瓊花六角舞翩遷，潤物無聲大德連。

時令循環皆盡謝，笑觀俏蕊綻紅嫣。

大雪

作者：通明

北風冷冽透衣寒，飄絮飛花舞動歡。

素裹銀妝城市美，津門踏雪覓梅丹。

詠梅

作者：通明

紅粉嬌葩淡雅醇，暗香一縷醉騷人。

瑤池玉女從天降，願綻芳華笑報春。

臘梅

作者：通明

瑤池仙女降塵埃，玉蕊冰心傲骨裁。

數九寒冬香氣散，經風沐雪報春來。

/

雪

作者：霍慶來

風行雪域萬雲開，漫引騷人筆硯裁。

豈懼寒流侵瘦骨，敲詞弄韻意悠哉。

雪

作者：霍慶來

朵朵瓊花妝陌野，風行雪域萬雲開。

幽香一縷飄然過，借問東園又綻梅?

詠梅

作者：霍慶來

玉蕊冰心溢暗香，凌寒傲雪鬥風霜。

橫斜疏影誰知味，萬縷情絲醉羽裳。

詠梅

作者：霍慶來

長在深山遠俗塵，晶瑩獨抱意淳真。

嫣紅簇簇迷人眼，醉賞仙姿競入神。


​

尋梅

作者：九級旋風

雪裡芳魂自古吟，嬌柔不懼冷風侵。

吾來未做看花客，為學三分傲骨心。

看雪

作者：九級旋風

一夜瓊花遍地開，晶瑩碎玉滿亭臺。

憑窗看罷心長嘆，恨我今生不是梅。

/

吟雪

作者：心言齋主

漫天柳絮潔無瑕，萬壑千村碧玉賒。

借問塵霾欲何往，白雲深處是婆家。

詠梅

作者：心言齋主

一剪寒梅抱蕊開，迎風傲雪暗香來。

不和桃李爭嬌豔，玉骨冰肌獨佔魁。

/

雪壓白梅枝

作者：若桐

城外梅枝裹素條，細分顏色路中標。

無風卻見花靈動，早送寒香過雪橋。

驛梅

作者：若桐

悄填冷夜滿溝壕，素色妝成似雪濤。

半里寒香邀驛路，一枝梅朵寄叢篙。

/

詠梅

作者：童麗君

香如天脂色尤鮮，不與群芳共鬥妍。

獨向寒風爭氣勢，霜飛雪舞自怡然。

夜觀梅花

作者：童麗君

一陣香風拂面來，南枝破蕾待時開。

霜侵雪染方爭豔，萬里銀妝萬樹梅。

/

​詠梅

作者：心淡薄冰

蕭風瑟瑟百花凋，一陣幽香嫋嫋飄。

雨打霜欺全不懼，銜冰戲雪樂逍遙。

/

念梅

作者：心淡薄冰

銀紗曼舞素山時，一樹瓊枝綻玉姿。

溪澗淙淙花照影，冰清嫵媚惹人痴。


​

詠梅

作者：樑燕妮

凌風冒雪綻蒼枝，含韻馨香報故知。

溪畔籬旁冰骨展，何須墨客再吟詩？

雪梅報春

作者：樑燕妮

寒冰冷雪久徘徊，玉樹銀花迎面開。

端的羅浮情難禁，生機一朵報春來。

/

梅雪吟

作者：白潔

疏影橫斜送暗香，瓊花素雅碧天揚。

紅梅映雪臨寒傲，鐵骨冰機歲月長。

梅雪吟

作者：白潔

鵝毛曼舞落高天，梅雪相依玉手牽。

含笑臨風留倩影，百花爭放我為先。

/

詠雪

作者：劉金華

天空玉女下凡來，遍灑銀花任意開。

柳柏千身披素襖，春風嘻笑樂心栽。

梅

作者：劉金華

梅開岸上戰風寒，覆雪紅顏峻嶺間。

夢裡含情呈愛意，心留傲骨闖難關。

/

雪

作者：書山行者

剪開雲朵作霓裳，步履無聲入畫廊。

曼舞輕歌詩客醉，且觀大地換銀妝。

詠梅

作者：書山行者

一抹胭紅笑滿腮，風姿搖曳對窗臺。

期君踏雪今朝至，莫讓花枝寂寞開。

/

大雪

作者：山石人

紛飛大雪銀包裹，肅殺青山帶冷風。

我自吟詩冬色裡，絃音落在夕陽中。

望梅

作者：山石人

窗前極目嶺頭枝，萬卉千花凍不知。

傲骨清魂猶在守，任憑雪舞冷風馳。


​

紅梅

作者：茉莉

當年恨爾來耕種，入夢相逢可未亡？

今有繁花紅綻處，香魂一縷耐寒霜。

紅梅

作者：茉莉

北風呼嘯自疏狂，玉蝶紛飛鳥獸藏。

但有紅梅爭吐豔，冰魂鐵骨傲寒霜。

雪

作者：茉莉

凜冽寒風著力搖，漫天玉蝶任逍遙。

冰心一片藏新夢，飄向人間潤物嬌。

雪

作者：茉莉

瑟瑟西風一力搖，漫天玉蝶自飄飄。

冰心雲後裁新夢，素嶺紅霞筆墨描。

/

詠梅

作者：高應富

一夜寒風老樹紅，俏姿映入眼雲中。

興來縱酒罈心漏，提筆狂書裝醉翁。

詠雪

作者：高應富

世間亂象每相違，凜冽寒風少見吹。

大雪何嘗漫天舞，紅梅且恨報春遲。

/

詠梅

作者：城市之戀

風雨無情雪又侵，飽經磨礪亦甘心。

疏枝守得花千朵，贈與騷人作酒吟。

憐梅

作者：城市之戀

寒梅放蕊雪封湖，搖曳枝頭花影殊。

入夢芳魂千百態，有誰能解夜時孤？

/

傲雪紅梅

作者：五大洲

漫天飛絮路茫茫，村女紅裝上學堂。

冒雪踏冰驅冷凍，梅花一朵笑迎陽。

飛雪豐年

作者：五大洲

揚揚灑灑絮盈天，地凍天寒白卉妍。

一剪紅梅迎日到，漫天飛雪兆豐年。

/

梅

作者：鴿子

冷豔初開冠百芳，瘦枝冰蕊扮紅妝。

首均松竹三分傲，更勝蘭芝一縷香。

思梅

作者：鴿子

天涯羈旅自徘徊，別卻家山久不回。

歲月匆匆春又到，最思故土那枝梅。


​

春雪

作者：張樹華

東君有意獻瓊花，玉絮輕柔暖萬家。

梅笑鬆拔千里錦，多情萬物蘊新芽。

詠梅

作者：張樹華

多情瘦骨醉寒光，靚影冰心暗吐香。

雪打風吹春早報，虯枝玉樹永流芳。

/

詠梅

作者：詹國華

喜鵲登枝玉宇清，寒梅傲骨展嬌情。

皚皚白雪銀裝裹，北國嚴冬景緻盈。

雪

作者：詹國華

九天玄女舞銀花，絮落人間千萬家。

大地桑榆冰霧掛，三山仙境似蒙紗。

/

詠梅

作者：秦煥舉

寒梅不懼冷風徊，獨在嚴冬花自開。

雪覆乾枝親媚骨，飄香曠野待春來。

雪

作者：秦煥舉

曼舞銀花霄九傾，遮天蓋地落無聲。

瀟瀟灑灑千般媚，妝就梅枝朵朵盈。

/

雪梅贊

作者：張慶陽

雪壓冬雲天地白，紅梅一朵凌寒開。

誰人不曉羞花好，若有芳魂也自哀。

冬雪

作者：張慶陽

瓊妃思戀世間佳，遍野滿山開素花。

情意隨心涓萬物，迎來春暖舞紅霞。

/

梅香

作者：朝進

清風化雨潤梅開，吸引花仙下玉臺。

雪落琴絃飛日月，黃昏浮動暗香來。

詠梅

作者：朝進

梅曲寒冬笑雪來，梨花泣動石山開。

冰肌玉骨嬌姿美，玉雪清香醉翠臺。

/

大雪

作者：李冠群

朔氣幾番凝四野，梨花一夜綻千家。

無香卻是真顏色，照我歸心越海涯。

飛雪

作者：李冠群

北風捲地舞瓊英，六瓣飄飛比羽輕。

斜掠沾衣身作畫，純然無暇暗含情。


​

悟梅

作者：寶世華

尋跡常聞滿寺香，慣將五瓣混僧腸。

偏迷熱眼觀顏色，只怕無緣問古璋。

老梅

作者：寶世華

自處黃昏伴鶴雲，寒風拂面洗冬塵。

枯樁不寂虯枝探，幾綻孤芳亦動人。

/

傲骨

作者：熱冰

紅梅傲骨抗嚴霜，萬朵千枝欲洩香。

我不劈寒先吐蕾，群嬌哪敢把花揚。

憐梅

作者：熱冰

傲骨梅花吐蕾繁，梨花抱絮染心間。

孤紅墜落隨春去，白雪憐惜淚暗彈。

/

梅語

作者：江藍

孤山相狎月黃昏，驛外詩翁亦慰魂。

最是潤之能懂我，報春天下笑乾坤。

雪聲

作者：江藍

敲窗叩戶響沙沙，千里郎歸疑到家。

再起開門唯見白，玉寒撲面月西斜。

/

大雪

作者：張曉輝

雲生穹頂作冰花，不染纖塵勝白沙。

一夜寒風歸故里，揚揚灑灑惠農家。

梅

作者：張曉輝

一剪寒梅樹雪中，千劫歷盡且從容。

誰人解語愁滋味，煮酒煎茶說落紅。

/

初雪

作者：葛霞

梨花一夜九州開，砌玉鋪銀舞閣臺。

添被麥苗寒蝶醉，飛揚六出展奇才。

/

觀雪

作者：葛霞

晶瑩剔透降塵埃，萬種風情帶韻來。

素裹銀妝仙客至，冰心脈脈惹詩才。

/

梅花

作者：方生

大雪如花遍地開，冬光傍影映階臺。

凌霜幾處催春近，一角寒英弄色來。

/

早梅

作者：方生

晨起推窗雪掛枝，邊城千樹暗枯垂。

寒風吹雁鄉關遠，幸喜梅花意動時。


​

梅

作者：劉井海

鐵骨寒梅傲雪開，冰枝絮葉透香來。

北風冷瑟吹花笑，漫舞嬌姿醉玉臺。

/

梅

作者：劉井海

銀枝玉葉雪冰裁，俏麗多姿蕊競開。

一笑嫣然紅粉面，馨香暗灑透牆來。

/

思梅

作者：趙平

菊後庭園寂寞開，擲杯人月兩徘徊。

似嫌塵土沾仙履，冰雪未鋪君不來。

/

詠梅

作者：天道酬陳

獨凌寒域任冰裁，不與群芳競綠臺；

莫道冬天無暖色，紅梅似火為君開。

/

臘梅贊

作者：老冒

西風肅殺少吟哀，自有寒香傲雪開。

莫道無情風雪驟，一枝引領早春來。

/

山寺臘梅

作者：三滇

幾樹臘黃牆角出，雪花飛舞識香撲。

人蹤難渡深山禪，卻為玉魂作伴竹。

/

梅雪迎年

作者：張孝華

梅開滿嶺勝春光，一夜風吹處處香。

十里八村年味足，雪飛萬朵賦詩章。

/

雪景

梯航（河北）

塞外冰天雪放歌，江南阡野尚婆娑。

千枝萬樹瓊花放，峻嶺崇山景色多。

/

吟梅

作者：董峰

只因笑在寒風裡，便扯素袍為嫁妝。

未必幽居深谷處，迎春燦爛滿山崗。

/

老梅

作者：方偉

幾經蒼鬱傲冰風，瑞雪含欣再綻紅。

梅曲媚濃香四溢，畫詞詩賦請君翁。


​

梅

作者：流浪的心

老樹孤亭百草衰，銀沙萬里望中來。

冬寒猶在疏枝裡，幾點深紅帶雪開。

/

詠梅

作者：博凱

天生不願展情才，唯寄相思對季開。

待得寒風飛絮盡，冰心好駕碧芳來。

/

傲梅

作者：藍天

風瑟天寒雪飛揚，鋪宣研墨伴梅香。

繁花落盡何爭豔？獨俏懸崖傲冷霜。

/

落雪

作者：霧鬆

輕飄白絮掛山川，天落銀裝掩沃田。

地凍封河年已近，待春喚起萬蟲眠。

/

雪中行

作者：安寧（汴梁）

人間快事雪中行，玉樹蟠枝滿汴城。

幾日糾纏身外物，皆隨飛絮落無聲。

/

一支梅

作者：陳慧娟

山明水秀雲天外，峻嶺摩崖獨自開。

唯恐世間春色晚，拈花一笑抱香來。

/

十月雪

作者：心湖

小城十月吼西風，一夜河山盡著戎。

四野茫茫皆絮白，誰家童子戲花叢。

雪後有感
