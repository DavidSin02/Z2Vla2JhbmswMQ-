《刺客信條：奧德賽》實用銘文全方位攻略

2020-11-14
| 遊戲

《刺客信條：奧德賽》中銘文是對玩家們戰力提升的一種道具，銘文之間有一個好的搭配才能最大程度的發揮出銘文的效果，那麼遊戲中什麼銘文好用，它又好用在哪裡呢？今天小編就來給大家帶來遊戲實用銘文的全方位攻略，希望能對大家有所幫助。

實用銘文全方位攻略

一、 最實用銘文收集攻略。

本帖將給出最實用的四個【陶片銘文】的收集攻略。這四個銘文分別為【命中+15%腎上腺素】【滿血+10%暴擊率】【滿血+50%暴擊傷害】【+20%狂襲傷害】。

我在收集銘文的時候發現，收集陶片的最大困難不是【陶片銘文解密】，而是在整個希臘世界浩渺的【問號】中，找到這些陶片。只要你找到了陶片，即使你不看攻略，解密這個陶片也並不難。並且網上還有較為成熟的攻略。另一方面，如果你全清地圖，會發現，很多銘文收集來是沒用的。我曾經為了解鎖一箇中毒傷害+1秒浪費半小時時間!!!因此，我做攻略的目的有兩個：

1、 只做那些【最實用】銘文的收集攻略，提高收集效率。

2、 挖掘銘文出處(非解鎖地點)，省去清圖清問號的麻煩。

下列表格給出這四個【最實用銘文】的出處!我將在所有內容更新完後以截圖方式顯示每個地點的具體位置(再也不用清問號啦)!

二、 傷害機制及銘文分類。

在之前的【戰士大型攻略】貼中，我給出過本遊戲傷害的計算公式，但由於一些測試的誤差或者未完全擬合，與實際結果有一些偏差。我經過重複測試，進行了修正，新版傷害公式為：

武器面板 * 武器類型面板倍率 * 面板增強 * 傷害類倍率 * 動作值 * 特殊動作增強倍率 * 能力增強倍率 * 對特定敵人增傷倍率 * 暴擊倍率 * 特定增傷倍率 * 特別傷害倍率

武器類型面板倍率：刀劍/匕首/錘/斧/矛/杖 15%傷害，刀劍匕首/重型武器/矛杖 10%傷害

面板增強：戰士/獵人/刺客被動(40%)，戰士/獵人/刺客 13-15%傷害，詛咒(100%傷害25%血量)，15%傷害-15%生命，所有傷害5%，獵人/戰士25%傷害-15%防禦

傷害類倍率：盔甲穿透，火焰/劇毒傷害，30%元素傷害-30%元素抗性，生命全滿30%傷害

動作值：技能傷害倍率/平A傷害倍率

對特殊動作增傷：狂襲增傷(最高200%)/弓蓄力增傷(最高120%)

能力增傷：使用戰士/刺客/獵人能力增傷15%，使用XX(如混沌之環，毀滅射擊)能力時增傷20%

對特定敵人增傷：雅典/斯巴達/動物增傷(最高200%)

暴擊倍率：基礎暴擊倍率150%，基礎爆頭倍率250%

爆頭傷害為250%+所有暴擊爆頭詞條(包括被動)

暴擊傷害為為150%+所有暴擊傷害詞條(包括被動)

特定條件增傷(分別乘算)：匿蹤大師夜晚脫戰增傷，第六感增傷，遠距離增傷15%(30公尺)

備註：

【特別傷害倍率】：從敵人正背後攻擊敵人時出現黃字並且增傷30%。該特別傷害與暴擊爆頭不能疊加。即，背後攻擊出現暴擊爆頭時，無特殊30%傷害加成。

【爆頭傷害基礎250%測得依據】：100%爆頭暴傷詞條+90%被動下爆頭為暴擊的1.5左右(440%對300%，增加50%爆頭詞條爆頭：暴擊增加到5:3左右)。

【暴擊傷害基礎150%測得依據】：拿弓測得，150%暴傷詞條下正好為3倍傷害;0詞條被動下為1.5倍傷害。

【遠距離傷害15%乘算依據】：30公尺以上有較大概率觸發15%乘算增傷(也可能不觸發，乘算依據，火焰箭和奪命箭均為1.15倍傷害，奪命箭純物理加成不吃任何元素加成，倍率相同說明是單獨乘算)。遠距離增傷與第六感不互相排斥。可被近距離敵人發現並且射擊遠距離敵人。【遠距離增傷詞條有bug，超過30%公尺也有概率不觸發】

下面的表格給出所有類別傷害相關詞條的總結。每種類別內為加算，類別與類別之間乘算。

注意小背傷是人物天生自帶的，從背後近戰/遠程攻擊敵人時，不出現獵人暴擊爆頭，一樣跳黃字(俗稱小暴擊)，傷害為白字的1、3倍。但是可以與戰士暴擊疊加。大暴擊小暴擊直接乘算。另外戰士能力15%和20%狂襲為加算。狂襲流不推薦出15%戰士能力。

三、 基於傷害公式的傷害類銘文收益全解析

I、 非燃燒流極限優化

我們先給出一個火焰戰士基礎配置銘文：

普羅米修斯劍：35%火焰傷害，盔甲穿透30%

副武器：20%火焰傷害，30%元素傷害

弓：20%火焰傷害，詛咒/滿血30%傷害

頭盔：無

盔甲：25%暴擊傷害

手臂：5%暴擊率，50%滿血暴擊傷害

腰帶：5%暴擊率，50%滿血暴擊傷害

鞋子：無

由此我們得到，火焰戰士的最低基礎傷害倍率為

1、 面板：100% + 40%被動 + 15% + 14*7 = 253%，弓詛咒則為353%

2、 傷害類倍率： 100% + 40%火焰被動 + 35% + 20%*2 + 30%*2 = 275%，弓滿血30%則為305%

3、 暴擊率：10% + 5%*2 + 10%被動 = 30%

4、 暴擊傷害： 150% + 50%被動 + 125% = 325%

由於以上基礎倍率按照最低基礎值計算，下面是各大詞條的最高收益：

1、 10%戰士傷害(傳說中逆天的雙詞條)， 10%/253% = 4.0%，如果弓銘文為詛咒，則為2.8%

2、 20%火焰傷害(頭盔，腿甲)，20%/275% = 7.2%。如果弓銘文位滿血30%，則為20%/305% = 6.6%

3、 滿血10%暴擊率，(0.4*3.25 + 0.6)/ (0.3*3.25 + 0.7) = 13.4%

4、 滿血50%暴擊傷害， (0.3*3.75 + 0.7)/(0.3*3.25 + 0.7) = 9.0%

可以看到，收益最高的為【1滿血0%暴擊率】和【滿血50%暴擊傷害】。如果裝備不夠好手臂或者腰帶缺少一個50%暴擊傷害，則暴擊率收益下降，而暴擊傷害收益上升，但均大幅高於火焰傷害及戰士傷害。

現在我們拿一套經典的“最強”火戰士配裝舉例。主要矛盾集中在頭盔和腿甲的詞條選擇。

原“最強”：頭盔20%火焰傷害15%刀劍傷害，腿甲20%火焰傷害50%滿血暴擊傷害。此時基礎為，傷害類345%，暴擊傷害325%(假設腰帶和手臂只有一個50%暴傷)，暴擊率30%。此時將雙火焰傷害替換為2個20%滿血暴擊的收益為：305/345 * (0.5*3.25 + 0.5)/(0.3*3.25+0.7) = 112%。

【也就是說，用2個滿血暴擊詞條替換2個火焰傷害詞條，反而多出了12%的期望收益。】

因此新刀劍最強基礎配裝毋庸置疑：

頭盔：15%刀劍傷害，10%滿血暴擊率

胸甲：25%暴擊傷害

腿甲：10%滿血暴擊率，50%滿血暴擊傷害

結論是【滿血暴擊】> 【滿血暴傷】 > 【火焰傷害】 >> 【戰士傷害】

【重要】!!!：結果是驚人的，以上基礎配裝，居然不包含任何火焰/劇毒傷害。這意味著你完全可以通用配裝，同一套裝備走火/毒流。在防具純堆物理詞條的情況下，卻達到了收益的最大化。還在吹元素詞條嗎?火焰/劇毒傷害很厲害嗎?真正厲害而通用，必備的防具詞條全部是【暴擊系的】(物理系詞條)。而最垃圾的就是傳說中超稀有的【雙戰士傷害】詞條。

下面的表格給出通用戰士的各銘文收益，我們主要看【期望乘算收益】一欄，這一欄數值越高，說明這個銘文越重要!所有重要的詞條我均已標紅。值得注意的是，戰士能力傷害15%雖然純收益為15%，但限制極大，不適合平A燃燒流，因此只在特定情況下做優先選擇。狂襲傷害同理，特化過於嚴重，也是特點情況下選擇。具體的詞條選擇取決於平常的打法。

II、 Dot極限優化

關於這遊戲的持續傷害(Dot)，就不得不提及dot的傷害機制。我下面給出dot傷害計算公式：

每跳dot傷害 = 武器面板 * 傷害類倍率中的元素倍率 * dot係數 * 暴擊傷害 + dot基礎值 * 等級

火附魔的本質是將武器傷害，全部轉化為火傷，因此武器面板 * 傷害類倍率中的元素倍率其實就等於實際火面板。(注意30%元素傷害屬於元素倍率，但30%滿血傷害盔甲穿透不屬於元素倍率)。dot係數因dot種類而不同，火dot係數大幅高於毒dot係數。暴擊傷害，依舊以期望的形式進行計算。【從該公式得知，堆元素傷害或者面板傷害均可增加dot傷害】。由於我發現詛咒並不是線性增加傷害的(詛咒後4000變5000，增加25%，面板為40%，矛盾)，因此dot裡應該有一個受等級加成的基礎值。

再來看銘刻配置，頭盔銘刻20%火焰傷害，腿部銘刻【50%燃燒率】銘文，因此，剩下的腿甲的詞條選擇為20%火焰傷害，10%滿血暴擊，50%滿血暴傷。對於dot來說，傷害倍率為100% + 40% + 30% + 35% + 40% + 20% = 265%。20%/265% = 7.5%。收益依舊是不如滿血暴擊和暴傷的。因此腿甲的最優選擇依然是滿血暴擊暴傷!!!可以看到，dot流的最強基礎配裝和通用火戰士是一模一樣的。非常重要!!!

【假設你沒有刷到最強通用基礎配裝】，那麼請務必在有詞條衝突的時候優先選擇10%滿血暴擊和50%滿血暴傷。

四、戰士各流派終極配裝推薦

分析完各銘文的收益後，我們下面總結【所有輸出流戰士】的終極配裝。配裝包括包括毒流和火流，但不包括苟命流。苟命流，逃跑流，脫戰流，請使用毒配並自行配置30%防禦力，20%全抗性，10%生命等各生存銘文。

在表格中，第二欄為通用詞條(也就是一套通用必備的詞條)，後面幾欄為各戰士流派特化的銘文詞條。其中標橙的字體為銘刻的金色銘文。值得注意的是，新獅子套的滿血5%暴擊率25%滿血暴傷也是個非常非常強大的銘文(相當於10%滿血暴擊和50%滿血暴傷的綜合)

亞特蘭蒂斯匕首銘文為：15%獵人傷害，25%暴擊傷害，30%盔甲穿透，1個金色銘文位。

對比紫色刀劍/匕首銘文：14%戰士傷害，25%暴擊傷害，20%劇毒傷害，1個金色銘文位。

【亞特蘭蒂斯匕首】相對於紫武器面板優勢為5.3%。使用亞特蘭蒂斯匕首時，傷害類倍率為(100+25+20*4+30*3)= 295%。劇

毒傷害20%收益為6.8%。盔甲穿透收益為10.1%。14%戰士傷害收益為5.5%有詛咒，4.0%無詛咒。即使是【5.5%無詛咒戰士傷害+6.8%劇毒】也打不過【5.3%面板，10.1%盔甲穿透】。

因此亞特蘭蒂斯匕首雖然是獵人加成，但依舊完勝紫色毒武器!!!另外，亞特蘭蒂斯匕首的好處是可以同時攜帶：盔甲穿透，滿血30%，滿血5%暴擊25%爆傷，30%元素傷害，詛咒，5箇中4個金色銘文!!!!【完全不是紫毒武器可以比的】。還在用紫色毒武器的非狂襲戰士你們還用嗎
