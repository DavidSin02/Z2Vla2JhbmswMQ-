「公主連結Re:Dive」一期專武解析

2020-11-14
| 遊戲

文丨生活可以很美好@bigfun社區

說實話，今天的本地化筆記出來的時候驚到我了，完全沒想到專武的開啟有這麼快。不過仔細想想和臺服對比也就提前了一期（繁中服是天秤座會戰以後開啟專武的），相較於國服沒有壓縮的卡池，提前開啟的專武能給會戰帶來不一樣的體驗，這樣看來國服也有自己的特色了。

然後這篇文章本來是準備放在這期會戰之後的，因為我預測國服的專武進度和臺服進度差不多，所以現在是知道消息後爆肝出來的。當時準備的是三輪三輪專武的做（留給大家足夠的刷碎片時間），現在因為時間比較趕，就先只做第一期了。

一期專武主要為前作和這一作的御三家加上鈴鐺、充電寶和偶像。

懶人包部分

一、前言

一些基礎數據

根據本地化筆記，這次專武總共開放到110級，也就是拉滿一個人物需要58個心碎+90個人物碎片+185萬MANA。升級滿還需要474w5k的MANA。

對於我們來說最缺的資源應該是心碎，在不加倍，不重置的情況下，只靠聖蹟調查（心碎本）一天連10個碎片都刷不到。好在同期還會開放露娜塔，露娜塔中同樣會提供心碎作為獎勵（量也不是特別大）。在這種情況下，我們初期的目前是以開專武為主，不要急著強化滿。

專武和RANK、裝備一樣，並不是拉滿最好。如果專武附加的屬性有雙抗和血量的時候，很多時候也需要卡專武等級。

對環境影響的猜想

專武的提前影響最大的還是公會戰，而這主要還是出在憐（劍聖）上。

專武賦予了劍聖破甲的能力。經過了這麼多輪會戰，大家已經知道了破甲的重要性。不過憐本身的輸出並沒有很高，破甲數值也有限，所以雖然有了破甲，但是還是很難斷言會壓縮法隊的上場，臺服同期法隊被打壓的主因是TOMO和克總提前（臺服美杜莎會戰有TOMO，天秤座會戰有克總）。而且通過上次會戰的魔改我們會發現，國服的魔改還是以砍雙抗為主（且砍的魔抗值較多），在這種情況下估計還是以兩物理一法為主。

但會戰最重要的是你可以不上場，但你不能沒有，所以劍聖的練度也會作為一個考核點。

第二個影響是天秤座的雲，我們公會的指揮一直說他有印象臺服的時候因為雲魔抗過低，我們出法刀的時候用中二頂替了一個破甲（黑貓）。而專武賦予了黑貓一技能破甲的能力，是否能讓她在天秤的時候有更多的發揮呢？（這裡之所以用了問號，是因為我一點都沒這個印象啊，我的印象一直是雲是物理和法刀混著出。）

二、具體介紹

這部分介紹的順序是按上代御三家→這代御三家→剩下三人。

摺疊內放的是專武和技能的介紹。專武的數據是110級的數據。

日和（貓拳）

喵王腰帶：日和配戴的腰帶。 有著猛獸睥睨眼神的扣環， 能鼓動鬥爭本能， 令拳頭更有勁道。

專武屬性： 物理攻擊力607 ；物理暴擊129

一技能強化效果：對範圍【400】內的敵人造成 【9.0 + 9.0*技能等級 + 0.72 *atk】的物理傷害。

貓拳的專武給了她一技能額外的AOE，屬性上增加了物理攻擊力和暴擊，無需卡專武，可直接升滿。

貓拳的專武很一般，設計上確實提高了一定的傷害，但是作為第一批吃螃蟹的人，這種純傷害技能必然隨著超模角色的出現而被環境所淘汰。

雖然我在前文中在公會戰方面給了貓拳較高的優先度，但是實際在使用中估計很難有上場的機會。最主要的原因就是我前文提到的國服的環境大概率是兩物理一法，而在專武實裝的同期另一位頂底物理打手TOMO也會實裝。TOMO有個外號叫做0.9狗，這說明了TOMO在輸出端的強勢。2刀物理刀中狼狗基本定死了，扣去輔助，能上場的物理打手只有三到四個，這基本被三弓和TOMO佔領了位置。在這種情況下，貓拳大概率是無法上場的。

建議留好JJC幣觀望國服改動，之後倉管等六星實裝（然後成了JJC一把好手）

優衣

繁花之願：優衣配戴的髮箍。 造型可愛的花蕾絲與裝飾， 讓守護夥伴的治療之力更為提升。

專武屬性：物理防禦33；魔法防禦33；魔法攻擊力297；回覆量上升33

一技能強化效果：【20.0 + 20.0*技能等級 + 1.6 *atk】的魔法傷害。 降低敵方單體1230 【7.5 + 7.5*技能等級】的物理攻擊力。 持續：【12.0】 秒。

優衣的專武給了她一技能傷害上更高的倍率，以及額外的降攻效果。不痛不癢的專武效果。

簡單來說你所有能用到優衣的場合，她自身都能發揮很好了，不需要再有專武的加持；而你用不到專武的場合給優衣裝上專武也無法使用。如果說一定要給她找個穿專武的理由，那就是因為不穿專武升不了六星。

優衣專武優先度低，資源富裕情況下才去考慮。（優衣對不起）

憐（劍聖）

雄風手套：憐愛用的手套。這副攻守兼備的手套所打出的正拳突刺，不論對上什麼敵人都能夠打倒。

專武屬性：HP710；物理攻擊力613

一技能強化效果：對敵方單體造成【20.0 + 20.0*技能等級 + 1.6 *atk】的物理傷害。

降低敵方單體【0.6 + 0.6*技能等級】的物理防禦力。 持續：【12.0】 秒。

劍聖的專武使她一技能有了破甲效果。大家在看WIKI的時候要注意，有時候劍聖一技能那裡會顯示兩個傷害和兩個破甲，數值比較高的是劍聖六星以後的效果。

劍聖因為破甲效果，終於可以登上會戰的舞臺了。考慮到公會戰如果無法出三刀物理的話，那麼劍聖的出場其實並沒有你們想象中那麼多。但無疑劍聖是這期專武中，提升最大的之一。我們當時在臺服的時候劍聖主要是卡四星滿專，或則五星30專。之後劍聖退出舞臺，主要是因為有更好的破甲角色。直到六星之後，劍聖又會重新成為會戰的重要角色。

建議可以開始每天刷了，但是不用急著升滿，四星開專即可，等會戰之後再確定。（反正JJC也用不到劍聖）

可可蘿

愛梅斯護符：可可蘿配戴的花飾。受到被祀奉為神的偉大存在之加持，花中飄出的溫和香氣，可以賦予戰鬥的勇氣。

專武屬性：物理防禦33；魔法防禦33；物理攻擊力323；回覆量上升33

一技能強化效果：對敵方單體造成 【20.0 + 20.0*技能等級 + 1.6 *atk】的物理傷害。

提升自身【11.0 + 11.0*技能等級】的物理攻擊力。 持續：【12.0】 秒。

使自己的TP上升提升【15.0 】。 持續：【12.0】 秒。

可可蘿的專武使她的一技能能賦予自身攻擊力上升和TP上升。

依舊是一個非常保守的專武，並沒有給可可蘿帶來什麼質的變化。我們升級專武就僅僅是為了可可蘿能在會戰中多打一丁點傷害。隨著強力角色越來越多可可蘿的出場機會會越來越少，直到開啟六星之後，可可蘿又活躍在各種場合（她的奶實在太足了，然後其他屬性又很多）。

初期開專即可，具體看會戰實戰需求。

凱留（黑貓）

混沌魔典：凱留攜帶的魔導書。 只要控制書中滿載的浩大魔力， 就能自由操控所記載的 強大術式。

專武屬性：魔法攻擊力607；魔法暴擊65

一技能強化效果：對敵方單體造成 【20.0 + 20.0*技能等級 + 1.6 *atk】的魔法傷害。

對範圍【200】內的所有敵人造成 【4.5 + 4.5*技能等級 + 0.36 *atk】魔法傷害。

使範圍【200】內所有敵人的魔法防禦力下降23 【0.1425 + 0.1425*技能等級】。 持續：【12.0】 秒。

黑貓的專武給她的一技能加了一個4.5倍率的AOE和一個比較低的魔抗降低效果。

縱觀PCR幾次特殊內容的引入（專武和六星），黑貓從中獲得收益的方式都是作為第一批成員，擁有了爆炸屬性，從數值上比別人優秀。這就導致了將來若出現了數值更加優秀的角色或機制更加優秀的角色，她就很容易被淘汰。

專武黑貓在剛出來的這段時間，主要發揮是在JJC，但是考慮到國服會戰法隊必不可少，那麼即使是公會戰玩家對於黑貓專武的需求也比較大。之後七七香出來之後，黑貓的出場次數會再被壓縮，不過考慮到一個是塔幣，一個初期只能靠母豬石，大部分玩家還是會選擇黑貓。

專武拉滿，可活躍於JJC和公會戰，倉管後等六星翻身

佩可（吃貨）

公主之劍：貪吃佩可所使用『皇家裝備』 真正的樣貌。原本已經足夠強大，但若解放了那沉睡之力，將會賦予更強的力量。

專武屬性：物理防禦65；魔法防禦59；回覆量上升33

一技能強化效果：提升自身【1.0 + 1.0*技能等級】的魔法防禦力。 持續：【12.0】 秒。

回覆自身【10.0 + 10.0*技能等級 + 1.2*atk】的HP。

展開可吸收 【22.5 + 22.5*技能等級 】傷害量的物理魔法吸收護盾。

佩可的專武讓她一技能的提升魔法防禦力和回覆的係數都有較大的提升。賦予了新的能力：展開魔法吸收盾。

佩可的專武可以說是“錦上添花”了，佩可本身常用於PJJC的防守隊，專武的存在進一步提升了她的肉度。肉、很肉、非常肉，這個就是專武以後的佩可的唯一評價。不過在T上物理T還是布丁，法T還是抖M。佩可主要還是活躍於PJJC苟隊陰人的。

打JJC的朋友可以直接拉滿，不過初期心碎較缺，先以開專為主。

胡桃（鈴鐺）

善舉之鈴：胡桃當作護身符的鈴。從莊嚴雅正的外觀到奏出的可愛音調，都能讓怯懦的心生出勇氣。

專武屬性：HP645；物理防禦52；魔法防禦52；HP自動回覆1290

一技能強化效果：提升我方全體【0.75+0.75*技能等級】的物理防禦力。持續：【12.0】秒。

提升我方全體【0.75+0.75*技能等級】的魔法防禦力。持續：【12.0】秒。

回覆我方全體【50.0】的TP。

胡桃的專武提升了一技能原先的係數，同時賦予了小幅度的充電效果。

胡桃這個角色，並不是專武以後一下子就發光發熱的，而是慢慢隨著環境被人挖掘出來的。主要是胡桃的技能，在初期傷害不是特別高的時候，效果並不是那麼直觀展現的。隨著數據的膨脹人們慢慢發現胡桃的一技能能有效的避免我方角色被秒殺。現階段的使用還是以PJJC中的苟隊為主。

JJC玩家優先度較高，其他的可以把順序放後面一點

咲戀（充電寶）

榮耀之羽：咲戀愛用的髮飾。溫柔之中蘊含的強大正是其象徵。當翅膀振動時，能令身體從重力解放。

專武屬性：HP162；物理攻擊力678

一技能強化效果：使該技能的係數1提高【損失的HP*0.4】

對【500】範圍內的敵人造成【12.0+12.0*技能等級+0.9*atk】的物理傷害。

咲戀的專武把她的一技能係數從0.9升到了1.2，同時生命值越低傷害越高。

充電寶專武最尷尬的地方在於專武提升的輸出能力只能作為一個添頭。我們使用充電寶的場合更多是把她作為一個輔助讓她給輸出充能。這使得充電寶的專武有點類似優衣，但是因為在JJC中的高出場率，充電寶的傷害多少會幫我們打一波奇蹟團。

充電寶專武優先度低，有餘裕的情況下再升即可。

望（偶像）

閃耀麥克風：望專用的手持麥克風。在可愛的緞帶中央畫有【慈樂之音】的標誌。拿到手上就能向頂級偶像邁進。

專武屬性：HP1935；物理防禦39；魔法防禦39；迴避7

一技能強化效果：對範圍【280】內的敵人造成【8.0+8.0*技能等級+0.6*atk】的物理傷害。

使範圍【280】內的敵人一段時間內無法行動。加速0.0倍。持續：【2.0】秒。

使範圍【280】內敵人的物理攻擊力降低【1.5+1.5*技能等級】。持續：【12.0】秒。

偶像的專武增加了傷害的倍率，增加了控制時間還加了一個降攻的DEBUFF。

偶像作為一個JJC常客，本身就有非常高的強度和出場率。她的專武對她來說更像是一種錦上添花，專武的屬性增加了肉度；附加的控制效果也能更好地噁心對面。不過從實際角度來說，雖然有降攻和擊暈效果，但並不會對二號位造成傷害，所以對方在配置陣容的時候還是可以使用單T。

偶像開專後直接拉滿，但是不急著投資特別多的碎片。
