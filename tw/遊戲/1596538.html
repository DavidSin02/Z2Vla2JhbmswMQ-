輸出聖騎士懷舊服PVE指南：感受聖光的平A吧

2020-10-29
| 遊戲

一、 開始之前

聖騎士是懷舊服團隊中不可或缺的職業之一，強有力的治療能力及一系列增益光環使其成為團隊中一股出色的力量。而在懷舊服中的輸出騎士（懲戒系）更像是介於輸出與治療之間的支援型職業。在本篇文章中，我們將針對聖騎士在PVE副本中的能力進行詳細的研究。

1、 輸出騎士的職業優劣

Ø 優勢

n 強大的團隊支援能力

n 強大的恢復能力

n 護甲高不易死

Ø 劣勢

n 機動性低

n 持續作戰能力低

n 傷害技能有限

2、 輸出騎士的種族推薦

聖騎士在懷舊服中為聯盟專屬職業，所以你必須選擇加入聯盟才能創建聖騎士。

Ø 人類（推薦）

人類是聖騎士最佳的種族選擇，種族技能“劍類武器專精”和“錘類武器專精”將能額外增加你的命中率，這是矮人種族無法比擬的。其次是“外交”種族技能能加快聲望的獲取速度，而“感知”在PVP中的作用也非常不錯。

n 外交：聲望獲取速度提高10%。

n 人類精魄：精神提高5%。

n 錘類武器專精：錘和雙手錘技能提高5點。

n 劍類武器專精：劍和雙手劍技能提高5點。

n 感知：提高潛行偵查能力，持續20秒，3分鐘CD。

Ø 矮人

相比人類，矮人的種族技能對PVE並未有多大的提升。但矮人的種族技能“石像形態”在PVP中應對盜賊的毒藥或者戰士的出血時有奇效，如果你喜歡PVP的話可以選擇矮人，總體來說都不如人類。

n 尋找財寶：使矮人可以感知附近寶藏的位置並顯示在小地圖上。

n 冰霜抗性：冰霜抗性提高10點。

n 槍械專精：槍械技能提高5點。

n 石像形態：對流血、毒藥和疾病效果免疫，護甲值提高10%，持續8秒，3分鐘CD。

3、 輸出騎士的專業推薦

懷舊服中聖騎最好的專業為工程學，這個專業能為騎士彌補PVE中傷害能力不足的問題，並且還有很多物品能提供不錯的效果。

三、 天賦推薦

1、 通用PVE天賦(11/8/32)



這套天賦的加點相對固定，沒有什麼可以改動的地方。防護天賦樹下必須點出“精確”提供命中率。神聖天賦的“神聖之力”可以說是每個聖騎士的必點天賦，而“奉獻”能為我們增多一個輸出技能。唯一可以更改的地方是懲戒天賦樹第二層的“強化十字軍印記”，因為在懷舊服有著debuff上限，所以很多時候可能你會用不到十字軍印記，這個時候就可以把天賦點加在第一層的“祈福”和第四層“強化懲戒光環”。

2、 命中率達到上限後的天賦改動（18/0/33）



選擇這套天賦的前提是你裝備堆疊的命中率達到了9%，這時候我們將捨棄防護天賦下的“精確”，取而代之的是神聖天賦下的“精神集中”和“強化聖療術”，使我們的生存能力更強。


四、 技能循環

輸出騎士的循環非常簡單，主要由審批和奉獻兩個技能為主。審判將會是你最主要的輸出技能，與命令聖印或復仇搭配時能造成高額的單體傷害。

技能循環如下：

Ø 保持命令聖印

Ø 使用審判

Ø 復仇與奉獻搭配使用

Ø 當目標生命值低於20%時，儘量使用憤怒之錘

技能循環不是特別複雜，但是需要注意不能長時間的作戰，藍量根本就撐不住。
