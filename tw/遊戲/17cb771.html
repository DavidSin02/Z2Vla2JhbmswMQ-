2020-10-29
| 遊戲

一、 什麼是元素入侵？

每隔數日，在卡利姆多的安戈洛環形山、艾薩拉、冬泉谷和希利蘇斯就會刷新一些元素入侵開始。事件開始時會出現一個元素位面入口，然後會不斷地刷出一些元素入侵者和一個元素首領。

所有的元素都有機率掉落火焰元素之類的元素精華，在元素首領被殺死之前，元素入侵者會不斷地刷新，如果你是為了刷元素材料的話，切記千萬不要把首領給殺了。當元素首領被擊殺後，元素入侵事件結束，剩餘的元素入侵者將會繼續保留，直到被殺死。

二、 入侵事件類型

卡利姆多的四個地點會分別刷新不同的元素入侵事件，分別對應著四種元素：

Ø 安戈洛環形山——熾熱入侵者

Ø 艾薩拉——雷霆入侵者

Ø 冬泉谷——水流入侵者

Ø 希利蘇斯——旋風入侵者

三、 熾熱入侵者

熾熱入侵者是一個綠色的火元素，會刷新在安戈洛環形山火羽山的底部附近。



Ø 火焰男爵查爾

n 火焰男爵查爾是熾熱入侵者的首領，免疫火屬性攻擊。

n 刷新時會大喊：“火山口的居民們！我是來焚燒這片汙穢之地的！”

n 還生存時會大喊：“這個世界所謂的英雄在哪裡?害怕得不敢出來玩了?”

n 被擊殺時會大喊：“這一切終將消耗殆盡。”

Ø 掉落

n 火焰男爵查爾的節杖

n 元素之一（暗月馬戲團卡牌）

n 元素灰燼

n 火焰精華


四、 雷霆入侵者

雷霆入侵者是刷新在艾薩拉東北部的土元素，附近還有很多石巨人精英怪，需要格外地小心。



Ø 阿瓦蘭奇奧

n 阿瓦蘭奇奧是雷霆入侵者的首領，建議組隊進行挑戰。

n 刷新時會大喊：“你藏在哪裡？我找到你了！”

Ø 掉落

n 阿瓦蘭奇奧的石皮

n 元素之一（暗月馬戲團卡牌）

n 硬石戒指

n 大地精華


五、 水流入侵者

水流入侵者會刷新在冬泉谷的凱斯利爾湖附近。



Ø 泰比斯蒂亞公主

n 泰比斯蒂亞公主是水流入侵者的領袖，免疫冰霜傷害。

n 刷新時會大喊：“以這片冰冷的土地為起點，到我這來吧凡人，我會給你一個快速而痛苦的死亡。”

Ø 掉落

n 泰比斯蒂亞的寒冰項鍊

n 圖樣：雷暴手套

n 元素之一（暗月馬戲團卡牌）

n 圖樣：信念手套

n 寒冷戒指

n 水之精華


六、 旋風入侵者

旋風入侵者是刷新在希利蘇斯西北部的空氣元素。



Ø 烈風掠奪者

n 烈風掠奪者是旋風入侵者的領袖。

n 刷新時會大喊：“烈風掠奪者和他的入侵部隊出現在希利蘇斯西北部的動亂中。”

Ø 掉落

n 烈風掠奪者腰帶
