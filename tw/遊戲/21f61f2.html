新版本7.25發佈，電炎絕手不僅會吐也會“吃”了

2020-10-29
| 遊戲

可能是考慮到大家在家宅著只能無聊的打DOTA，為了給大家帶來些新鮮感，V社凌晨發佈了新版本7.25。

新版本進行了大範圍的平衡性調整，不管是物品還是英雄改動都不小。最有意思的還是給老奶奶增加了A杖效果，讓其擁有了“吃”的能力。

具體改動我們一起來看：

綜合

- 全英雄選擇的英雄挑選已經重做。現在為5輪，每輪天輝和夜魘同時選擇1名英雄，而且選擇的英雄將在下一輪揭曉。如果選擇了同一名英雄，後選的玩家將獲得額外的時間選擇另一名英雄。

- 重做了全英雄選擇中禁用英雄的機制。原來只會禁用提議英雄中一半的英雄。現在每次提議都有50%的成功機率。若被禁用的英雄少於10名，將根據玩家的積分水平相應的英雄被禁比率自動禁用英雄。

- 現在強化神符的初始產生時間為4分鐘（後續間隔還是每2分鐘）

- 強化神符在40分鐘後不再於河道兩端同時產生

- 移除了英雄擊殺的團隊金錢橡皮筋機制（重做了公式，簡化如下）

舊英雄擊殺範圍金錢：

===

定義：

- 財產總和排名系數 = 1.6 -> 0.4 基於擊殺者的財產總和在隊伍內排名

- 財產總和貧窮係數 = 1.2 -> 0.6 基於陣亡英雄的財產總和在隊伍內排名

- 翻盤係數 = 1（團隊經濟落後時），或 0（團隊經濟領先時）

1名英雄：財產總和貧窮係數 * 財產總和排名系數 * （126 + 4.5 * 陣亡英雄等級 + 翻盤係數 * （陣亡英雄財產總和 * 0.026 +70）/1））

2名英雄：財產總和貧窮係數 * 財產總和排名系數 * （63 + 3.6 * 陣亡英雄等級 + 翻盤係數 * （陣亡英雄財產總和 * 0.026 +70）/2））

3名英雄：財產總和貧窮係數 * 財產總和排名系數 * （31.5 + 2.7 * 陣亡英雄等級 + 翻盤係數 * （陣亡英雄財產總和 * 0.026 +70）/3））

4名英雄：財產總和貧窮係數 * 財產總和排名系數 * （22.5 + 1.8 * 陣亡英雄等級 + 翻盤係數 * （陣亡英雄財產總和 * 0.026 +70）/4））

5名英雄：財產總和貧窮係數 * 財產總和排名系數 * （18 + 0.9 * 陣亡英雄等級 + 翻盤係數 * （陣亡英雄財產總和 * 0.026 +70）/5））
