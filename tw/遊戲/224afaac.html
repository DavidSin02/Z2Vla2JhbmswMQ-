任務管理系統 pyteam

2020-10-29
| 遊戲

技術棧

  • koa2
  • jquery-pjax
  • bootstrap3
  • sequelize(mysql)
  • socket.io
  • nunjucks

啟動

我的開發環境：nodejs(v11) + mysql(v5.7)

  1. 創建數據庫 pyteam , 不用創建表 啟動時自動就創建表了
  2. 修改config.js裡的配置信息，base_url 和 mysql 就夠了

安裝項目依賴

npm installnpm install -g nodemon

啟動項目

npm run dev

部署正式環境

npm install -g pm2pm2 start process.yml

正式環境重啟

pm2 restart pyteam

服務端口號 3002

功能&特色

  • 項目管理
  • 項目下任務管理
  • 接口文檔管理

如果頁面內容狀態有變動，自己不用刷新頁面就可以實時更新，詳細可見動圖

使用場景

內網使用，外網碰到的問題請自行解決

另外，請在 Chrome Firefox Safari 瀏覽器上使用，不要使用其它瀏覽器，即使是國產的套殼瀏覽器也不行

如果你想測試瀏覽器的兼容性，可以查看一下技術棧裡相關技術的瀏覽器兼容性

使用方法

項目部署好之後，註冊一個用戶名為admin的用戶，默認admin為管理員（當然也可以自行配置管理員用戶，找到config.js文件，在數組admins裡添加用戶名即可），可以進入後臺

進入後臺之後，首先添加部門

其它用戶註冊就可以選擇自己是哪個部門的了，這樣在登錄之後就可以看見自己部門下的所有項目了，如果不選擇部門的話，是沒辦法創建項目的，首頁也不會有任何項目
