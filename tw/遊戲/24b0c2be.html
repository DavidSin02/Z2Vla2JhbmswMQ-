《熱血傳奇》道士技能列表(7~46級）

2020-11-14
| 遊戲

治癒術: 根據修煉等級的不同，可以恢復自己或者他人的體力(HP)，每跳5點體力。

Lv.1 : 可以在第 7 級開始修煉

Lv.2 : 可以在第 11 級開始修煉

Lv.3 : 可以在第 16 級開始修煉

精神力戰法: : 根據修煉等級的不同，可以提高戰鬥時的精確度（命中率）跟敏捷（躲閃率）

Lv.1 : 可以在第 9 級開始修煉.

Lv.2 : 可以在第 13 級開始修煉.

Lv.3 : 可以在第 19 級開始修煉.

施毒術: 根據修煉等級及道術的不同，可以延長毒藥的有效時間，綠毒減血，紅毒減防。道術隻影響毒藥時間。

Lv.1 : 可以在第 14 級開始修煉.

Lv.2 : 可以在第 17 級開始修煉.

Lv.3 : 可以在第 20 級開始修煉.

灰色毒粉

中毒的目標將會全身發綠，並且會慢慢失去它的體力。製作這種毒藥,需要4片食人樹的葉子，1個食人樹的果實，和2顆毒蜘蛛的牙齒。每份小毒粉可以使用50次。

黃色毒粉

黃色毒粉通過侵蝕目標的武器和防具來減低防禦力。毒粉同時還會破壞武器和防具的耐用性。製作這種毒藥，需要4片食人樹的葉子，1個蠍子的尾巴，以及蛆卵。每份小毒粉可以使用50次。

靈魂火符: 和火球術一樣的遠程法術，通過驅使護身符來攻擊敵人。

Lv.1 : 可以在第 18 級開始修煉.

Lv.2 : 可以在第 21 級開始修煉.

Lv.3 : 可以在第 24 級開始修煉.

召喚骷髏: 一種通過使用護身符和復活力製造骷髏的法術。製造出的骷髏將會成為英雄最忠誠的跟隨者。召喚出來的骷髏可以升級，一級骷髏可升3級，二級骷髏可升5級，，三級骷髏可升7級。

Lv.1 : 可以在第 19 級開始修煉.

Lv.2 : 可以在第 23 級開始修煉.

Lv.3 : 可以在第 26 級開始修煉.

隱身術: 隱身術使怪物無法察覺英雄的存在。使用這種法術後採用魔法遠程攻擊，是一種非常理想的單對單戰鬥方法（只能針對自己）。

Lv.1 : 可以在第 20 級開始修煉.

Lv.2 : 可以在第 23 級開始修煉.

Lv.3 : 可以在第 26 級開始修煉.

集體隱身術 : 更加強大的隱身術，能夠隱藏3X3範圍內的全體玩家。集體作戰時是一種非常有用的保護措施。

Lv.1 : 可以在第 21 級開始修煉.

Lv.2 : 可以在第 25 級開始修煉.

Lv.3 : 可以在第 29 級開始修煉.

幽靈盾 : 能夠提高魔法防禦力。他的有效時間隨著訓練等級的提高而延長（道術影響持續時間）。

Lv.1 : 可以在第 22 級開始修煉.

Lv.2 : 可以在第 24 級開始修煉.

Lv.3 : 可以在第 26 級開始修煉.

神聖戰甲術: 能夠提高防禦力。他的有效時間隨著訓練等級的提高，（道術影響持續時間）

Lv.1 : 可以在第 25 級開始修煉.

Lv.2 : 可以在第 27 級開始修煉.

Lv.3 : 可以在第 29 級開始修煉.

心靈啟示: 能夠展示目標（包括玩家和怪獸）體力。現在很少用的技能了，因為一般遊戲內掛都有血量顯示。

Lv.1 : 可以在第 26 級開始修煉.

Lv.2 : 可以在第 30 級開始修煉.

Lv.3 : 可以在第 35 級開始修煉.

困魔咒: 被限制在咒語中的怪獸因為看不見外面的情況只能繞著圈走。但是當英雄和其他人走進這個六角形的時候這個魔法會消失。當被困住的怪物受到外界攻擊時能夠躲避。即使英雄和很多人戰鬥的時候最好是一對一，困魔咒會發光以顯示邊界。

Lv.1 : 可以在第 28 級開始修煉.

Lv.2 : 可以在第 30 級開始修煉.

Lv.3 : 可以在第 32 級開始修煉.

群體治癒術: 能夠恢復自己和周圍所有玩家的體力。手殘玩家的最愛，尤其是隊友被怪物圍攻的時候，用它吧，不用再擔心奶不到隊友了。（同治愈術一樣每跳5點）

Lv.1 : 可以在第 33 級開始修煉.

Lv.2 : 可以在第 35 級開始修煉.

Lv.3 : 可以在第 38 級開始修煉.

召喚神獸: 能夠召喚一隻噴火的強大神獸作自己的隨從。升級機制同骷髏。1級神獸攻擊單位一格，2-6級攻擊前方兩碼豎線，7級神獸攻擊3格。超級肉盾，7級神獸2600點血，是個強有力的隊友。有句話怎麼說開著？人不如狗，嗯，用在這裡最合適了。

Lv.1 : 可以在第 35 級開始修煉.

Lv.2 : 可以在第 37 級開始修煉.

Lv.3 : 可以在第 40 級開始修煉.

氣功波: 過去，氣功術被認為是一種內功的修煉，而並非應用於實戰之中。直到一位方士，在潛心研究後，對氣功術進行改良，發明了氣功波，從而起到推開周圍的怪物而得以防身的作用，道士的抗拒火環，挺有趣的，不像法師的那種無腦推。

Lv.1 : 可以在第 29 級開始修煉.

Lv.2 : 可以在第 31 級開始修煉.

Lv.3 : 可以在第 34 級開始修煉.

無極真氣:利用大自然的真氣，從而引發體內潛力，不僅可以使自身能力大大提高，還可以將寵物的能量轉移給自己。這在武林流派的發展上可以說是一次里程碑。看誰不順眼，一個無極然後一個毒教他做人，那叫一個酸爽，簡直神技，嘖嘖……

Lv.1 : 可以在第 36 級開始修煉.

Lv.2 : 可以在第 39 級開始修煉.

Lv.3 : 可以在第 42 級開始修煉.

噬血術:驅使護身符，對敵人造成傷害。命中後，還可吸取對方生命，為自己回覆一定的血量。

技能施放頻率較靈魂火符有所提升，且忽略地形阻礙直接打擊目標！

缺點是，屬於魔法攻擊，會受到魔法躲避的干擾而影響命中率

Lv.1 : 你可以在第 47 級開始修煉.

Lv.2 : 你可以在第 50 級開始修煉.

Lv.3 : 你可以在第 53 級開始修煉.
