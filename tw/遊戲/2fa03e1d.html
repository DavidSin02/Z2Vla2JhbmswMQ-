從遇見，到陪伴，

從選擇禮物，到第一次的眼淚，

從告別，到鼓起勇氣繼續前行，

從溫暖的家，感受每一次愛的味道，

直到有一天，我們才知道什麼是青春。

在這滋潤心靈的節目裡，

有感動，有感悟，更有感恩...

青春不散，朗讀不停！

- 所有的遇見，都是久別後的重逢 -

// 遇見 //

「朗讀者」第一期

1.從某種意義上來看，世間一切，都是遇見。就像，冷遇見暖，就有了雨；春遇見冬，有了歲月；天遇見地，有了永恆；人遇見人，有了生命。

——董卿

2.只要你是個正直的孩子，不管你從事什麼行業，你都是我的好孩子。

——柳傳志父親對柳傳志說的話

3.如果美麗只是簡單的用來做展示，也許它的意義不是那麼大，但如果美麗，能夠轉化成一種能力，去幫助更多的人，甚至去讓自己變得更好，那它就是很有價值很有意義的了。

——董卿

4.生命哪，並不是你活了多少日子，而是你記住了多少日子，要使你過的每一天，都值得回憶。

——許淵衝

5.願你慢慢長大，願你有好運，如果沒有，希望你在不幸中學會慈悲；願你被很多人愛，如果沒有，希望你在寂寞中學會寬容。

——劉瑜「願你慢慢長大」

6.人生的不可測，在任何時候，都要抱著一份希望。

——董卿

- 陪伴，是最長情的告白 -

// 陪伴 //

「朗讀者」第二期

1.我要開花，是為了完成作為一株花的莊嚴生命，不管你們怎麼看我，我都要開花！

——林清玄「百合花開」

2.要是沒有兔子和鷓鴣，一個田野還成什麼田野呢？他們是最簡單的土生土長的動物。

——梭羅「瓦爾登湖」

3.這個世界上絕了哪種生命形式都會導致地球毀滅。獅子和螞蟻一樣偉大，小草和人類一樣重要。

——鄭淵潔

4.“媽媽，你到底為什麼愛我?”母親放下針線，用她的面額，抵住我的前額，溫柔的，不遲疑地說:“不為什麼，只因你是我的女兒!”

——楊乃斌朗讀冰心的散文「不為什麼」

- 選擇，意味著結果 -

// 選擇 //

「朗讀者」第三期

1.有人說，我們這個時代不缺機會，所以也勢必會讓每個人面臨很多的選擇，那麼，是遵從自己的內心，還是隨波逐流；是直面挑戰還是落荒而逃；是選擇喧囂一時的功利，還是恆久平靜的善良；無論如何，希望每一個人，都能做出一個在日後回想不讓自己後悔的選擇。

——董卿

2.有人說“不瘋魔，不成活”，我很享受這種若隱若現當中左右徘徊，這是我表演的語境，也是我的選擇。

——王千源

3.你用最不敬業的精神去演戲，結果世界上最單純的觀眾，用這麼炙熱的心情去觀看，真不次於在犯罪。

——王千源

4.人的一生，總是在尋找一種平衡，忠貞的人，永遠會得到忠貞；勇敢的人，最後也是用勇敢來結束。

——麥家

5.我熱愛這種渾然天成的泥濘，泥濘誕生了跋涉者，它給忍辱負重者以光明和力量，給苦難者以和平和勇氣。一個偉大的民族需要泥濘的磨礪和鍛鍊，它會使人的脊樑永遠不彎，使人在艱難的跋涉中懂得土地的可愛、博大和不可喪失，懂得祖國之於人的真正含義：當我們愛腳下的泥濘時，說明我們已經擁抱了一種精神。

6.可以說腳踏大地，泥濘，我相信一定會磨礪他們的青春，讓他們的人生煥發出與眾不同的光華，這光華也必將與我們的時代交相輝映。

——董卿

7.一個人最重要的不是擁有多少財富，和擁有多大的房子，最重要的是擁有深厚的學識和崇高的思想道德。

8.我慢慢去想奶奶講的那個神話，我慢慢相信，每一個活過的人，都能給後人的路途上添些光亮，也許是一顆巨星，也許是一把火炬，也許只是一支含淚的蠟燭。

- 天賦，是上天賜予我們最好的禮物 -

// 禮物 //

「朗讀者」第四期

4.這是幸福的一天，我漫步在花園，對於這個世界，我已一無所求。這是詩人饋贈給自己心靈的一份禮物。

——切·米沃什「禮物」

5.我不相信命運，我只相信我的手。我不相信手掌的紋路，但我相信手掌加上手指的力量。

——畢淑敏「握緊你的右手」

6.她叮囑我：“自己不倒，啥都能過去；自己倒了，誰也扶不起你。”

7.其實，當一個人真的要離開這個世界的時候，是什麼也帶不走的，唯一能帶走的就是心底裡存下來的那些記憶。

——董卿

// 那一天 //

「朗讀者」第十一期

1.那一天一定是讓你難忘的一天，也許是金榜題名時，也許是洞房花燭夜；也許是19世紀的某一天，左拉、雨果、巴爾扎克在塞納河畔的左岸咖啡館構思著傳世之作；也許是阿姆斯特朗自身的一小步、代表人類的一大步，被載入史冊。

——開場白

2.記得早先少年時，大家誠誠懇懇，說一句 是一句。

清早上火車站，長街黑暗無行人，賣豆漿的小店冒著熱氣。

從前的日色變得慢，車、馬、郵件都慢，一生只夠愛一個人。

從前的鎖也好看，鑰匙精美有樣子，你鎖了 人家就懂了。

——木心「從前慢」

3.「從前慢」，很多年輕人都喜歡，倒不是說要去過多慢的日子，而是希望有這樣一份能慢下來的從容與自由。

——董卿

4.「時間簡史」超越了普通人的價值觀、人生觀，和世界觀，它給我們增加了一個新的維度，也就是宇宙觀。霍金的超人之處在於，他打通了專家和普通人之間的壁壘，他在大學期間有兩大愛好，一個是古典音樂，還有一個就是科幻小說。這兩者讓他插上了想象的翅膀。

——北京大學世界傳記研究中心主任、國際傳記文學學會創始人 趙白生

——董卿

- 青春，是無悔 -

// 青春 //

「朗讀者」第十二期

1.青春彷彿是一種宣言，它昭示著自古英雄出少年的激情，青春也是一種姿態，心有猛虎，細嗅薔薇。青春也是一種勇氣，帶著加速度在奔跑，渴望找到未來的答案。那今天就讓我們在或許正青春，也許不再青春的人們身上，去感受青春永恆的魅力。

——開場白

2.青春即使在痛苦之中也閃耀著它的光彩！

——雨果

3.記憶就好像是一塊被蟲子啄了許多洞的木頭，這塊木頭上刻滿了我的青春往事。有蹉跎歲月，也有鯉魚跳龍門；有對生活的坦白，更有對朋友的懷念”。說這些話的人，就是著名導演。

——馮小剛

——勃蘭兌斯（丹麥）著 「人生」

5.他們保持著青年的全部特徵：愛冒險，愛生活，愛爭鬥，精力充沛，頭腦活躍，無論他們多麼年老，到死也是年輕的。好像鮭魚迎著激流，他們天賦的本性就是迎向歲月的激流。

——勃蘭兌斯（丹麥）著 「人生」
