德魯伊近戰懷舊服攻略：移速最快的輸出職業，能跟上我的速度嗎？

2020-10-29
| 遊戲

一、 開始之前

德魯伊是遊戲中的一個全能職業，既可以作為治療和坦克，又可以切換為遠程近戰輸出。如果你選擇成為一名近戰德魯伊的話，那麼你就等於能同時玩兩種形態的德魯伊——熊形態（坦克）、獵豹形態（近戰DPS）。近戰德魯伊需依靠獵豹形態專屬技能進行輸出，所以玩家也喜歡把近戰德魯伊叫做貓德。貓德的技能有點類似於盜賊，需要靠技能生成連擊點數，在依靠連擊點數施放終結技能。本篇攻略中，我們將針對坦克德魯伊在PVE副本中的能力進行詳細的研究。

1、 貓德的優劣

n 多功能性

n 實用性強

n 合理搭配天賦能同時玩近戰和坦克

n 遊戲中移速最快的職業

Ø 劣勢

n 傷害偏低

n 輸出手法繁瑣

n 裝備競爭大

2、 貓德的種族推薦

與其它職業不同的是，德魯伊在遊戲中僅兩個種族可以選擇，聯盟為暗夜精靈，部落為牛頭人。

3、 貓德的專業推薦

貓德最好的專業是工程學，該專業所製造的物品能提高你的輸出能力以及應對一些副本內的困境。其次是制皮專業，該專業所製造的魔暴龍套裝，無論是自己用還是出售都是非常不錯的選擇。同時還能製作最好的皮甲頭盔——狼頭之盔。

三、 天賦推薦

1、 通用德魯伊天賦（14/32/5）



這套天賦沒有多少可以替換的地方，基本上每個天賦都有實際的作用。首先野性天賦下的31點是必須的，因為該天賦最後一層的“獸群首領”對整個團隊的收益非常的大。恢復天賦下5點的“激怒”也會影響你的輸出。最後剩下的點數需加在平衡天賦下，“清晰預兆”和“自然變形”能為你節省不少的資源。

2、 DPS/坦克混合天賦（11/35/5）



適用於坦克和輸出之間的切換，當然相比單純的坦克或輸出天賦來說都會相對較弱，好處就是既能玩坦克又能玩輸出。
