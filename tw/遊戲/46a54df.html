so far 不單單指“很遠”

2020-10-29
| 遊戲

有同學問了一個很好的問題。我們經常能聽到 “so far” 這個短語，這是指“到目前為止”的意思，但是你知到怎麼去用這個詞嗎？如果說“到目前為止，我一直工作的很辛苦”，是不是可以用這個詞？我們將通過例句講解 so far 的幾個常用的意思和用法。

短語 “so far” 主要有三個含義：表時間、表距離和表程度。

用法講解（1）—— 表時間

“到目前為止，我一直工作的很辛苦”這句話可以說：

  • So far，I've been working very hard day and night.

此處的so far就是今天我要講解其三個意思中的第一個。它強調的是時間切入點“到目前為止”，不包括將來。我們看看例句；

  • This is the best score so far, but I will keep trying.（這是迄今為止的最好成績了，不過我會繼續努力的。）
  • So far there's been no news about our moving to Manchester.（到目前為止，沒有聽到我們搬遷到曼城的消息。）
  • So far, so good. I've got two A's and one B in my exams.（迄今為止， 一切順利。我的考試兩門得A，一門得B。）

請注意，在使用 so far來表達“迄今為止” 的時候，句子一般使用現在完成時態。

用法講解（2）——表距離

下面我們轉看 so far的第二個意思。表示“距離 to this or that point，到某處為止”。請看例句。

  • I can only walk with you so far as I'm running late.（我只能跟你走到這兒了。我要晚了。）

用法講解（3）——表程度

最後我們一起看so far的第三個意思，表示“程度 to a limited extent”. 表示“有限的、最多不過如此的”。下面看兩個例句。

  • I can only take you so far with my advice.（我只能給你這些建議了。）
  • You can only go so far on ten pounds.（靠10鎊錢能做得事情太有限了。）

好了。以上給大家講解了 so far 的三個主要意思：第一表示時間，“迄今為止”；第二，表示距離，“到某處為止”；第三，表示程度，“最多不過如此”。希望大家把例句多看兩遍，以便加深印象。
