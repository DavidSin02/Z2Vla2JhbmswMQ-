遺傳算法(Python) #5 用DEAP框架解決OneMax問題

2020-10-29
| 遊戲

遺傳算法系列的第三期介紹瞭如何不用任何框架從零開始解決OneMax問題，第四期介紹了DEAP框架的基本用法。若讀者對下文中定義或術語不熟悉，可以查看本系列的前幾篇文章。本期文章將介紹如何使用DEAP解決OneMax問題。

1.OneMax問題(OneMax problem)

OneMax問題是遺傳算法的入門問題，其內容是：如何使一段長度固定的二進制字符串所有位置上數字之和最大。

讓我們用一個長度為5的二進制字符串為例：

  • 10010 -> 和為2
  • 00111 -> 和為3
  • 11111 -> 和為5(最大值)

對一般人，顯而易見，當所有位數都為1時，該字符串的和最大，但在我們用遺傳算法解決該問題時，算法本身並不具備這樣的知識。

2.問題的解決思路

首先，我們得把這個問題轉換成一個遺傳算法問題，即：我們得定義個體、種群，選擇、雜交、突變方法、適應度函數等。假設有一個長度為100的字符串，我們可以做出以下定義：

  • 個體：個體即為問題的解，這個問題中個體可以直觀的定義為一個長度為100列表（List），列表上每個元素為0或1.
  • 種群：種群即所有個體的合集，我們可以把種群定義為所有個體組成的列表。
  • 選擇：使用錦標賽法(Tournament Selection)
  • 雜交：使用單點雜交法(Singe-Point Crossover)
  • 突變：使用位翻轉突變法(Flip Bit Mutation)
  • 適應度函數： 我們的目標是使字符串上所有數字之和最大，適應度函數可以直觀的定義為列表中所有數字之和。

若對上述定義不太瞭解的，可以回看遺傳算法系列的第二期。

3.用DEAP框架實現遺傳算法

以下將分步驟解釋每一部分的代碼，完整代碼在本文的最後可見。

3.1. 準備工作

# 1.load modulesfrom deap import base,creator,tools,algorithmsimport randomimport numpy as npimport matplotlib.pyplot as plttoolbox = base.Toolbox()

首先我們需要導入遺傳算法所必須的模組：

  • base，creator，tools，algorithms： 這四個模組是DEAP框架內最常用的模組
  • random: 用以生成隨機數
  • numpy：我們將用到 numpy 中的 mean 和 max 方程。
  • matplotlib： 用以繪圖
  • toolbox：DEAP框架中的核心，當使用deap.algorithms時，我們需要把遺傳算法的運算符存入toolbox內。 同時toolbox.register也是DEAP中最常用的方法。

3.2. 初始化算法參數

# 2.parameters:INDIVIDUAL_LENGTH = 100  # length of bit string to be optimizedPOPULATION_SIZE = 200P_CROSSOVER = 0.9  # probability for crossoverP_MUTATION = 0.1   # probability for mutating an individualMAX_GENERATIONS = 50random.seed(39)
  • INDIVIDUAL_LENGTH：個體的“長度”，即二進制字符串的長度
  • POPULATION_SIZE: 種群中個體的數目
  • P_CROSSOVER: 個體間交雜的概率
  • P_MUTATION: 個體突變的概率
  • MAX_GENERATIONS: 算法迭代次數上限
  • random.seed：為了保證每次運行的結果相同，設定了隨機種子

3.3. 定義個體與種群

# 3.create individual and populationtoolbox.register("genBinary", random.randint, 0, 1) # 1creator.create("FitnessMax", base.Fitness, weights=(1.0,)) # 2creator.create("Individual", list, fitness=creator.FitnessMax) # 3toolbox.register("createIndividual", tools.initRepeat, creator.Individual, toolbox.genBinary, INDIVIDUAL_LENGTH) # 4toolbox.register("createPopulation", tools.initRepeat, list, toolbox.createIndividual) # 5
  1. 這一列定了genBinary方程：該函數可以隨機生成0或1；通過toolbox.register()定義後，我們可以通過toolbox.genBinary()直接使用該函數。
  2. 因為在OneMax問題中，我們需要適應度的最大值，所以我們拓展base.Fitness類，並將拓展後的類命名為FitnessMax，並設定其權重為(1.0,)，代表求最大值（在其他問題中，如果需要求最小值的，可以定義權重為(-1.0,)）。
  • 在這裡注意我們定義權重是(1.0,)而不是1.0，其原因是DEAP框架支持求多個目標的的最大/最小值，所以我們必須定義權重為元組（tuple）。
  • 當在其他問題中我們的適應度方程有多個目標：如當我們需要第一個目標最大，同時第二個目標最小，且第一個目標的重要性是第二個目標的兩倍時，權重可以定義為：(1.0,-0.5)
  1. 創建Individual類，並讓FitnessMax做為Individual的fitness特性(attribute)
  1. 定義createIndividual方程：該方程可以生成長度為INDIVIDUAL_LENGTH的列表，通過調用genBinary函數，列表內每個元素隨機為0或1.
  1. 定義createPopulation方程：該方程可以生成長度待定的列表，列表中每個元素由createIndividual方程創建。
  • 注意createPopulation方程並沒有定義列表的長度，這是因為toolbox.register中用到了functools.partial方程，尚未定義的參數會以*args或**kwargs的形式傳遞給tools.initRepeat方程。

如果讀者對DEAP框架內的方程不太瞭解，看完上述解釋後還是感覺五里霧中，建議查看本系列前兩期文章，或者到DEAP官網上多瞭解下這些方程的定義。

3.4. 定義適應度函數

# 4. define evaluation functiontoolbox.register("evaluate", lambda ind: (sum(ind),))

在OneMax問題中，適應度函數即為個體（用二進制字符串表示）上所有數字之和,因此我們將適應度函數定義為(sum(ind),)。

在這裡請特別注意：

  • 因為DEAP框架支持有多個目標的適應度函數，所以我們必須把適應度函數的輸出定義為元組。
  • evaluate是關鍵詞，若要使用DEAP框架，必須使用evaluate這個單詞來命名適應度函數。

3.5. 定義遺傳算法運算符（選擇，雜交，突變）

# 5. define operatorstoolbox.register("select", tools.selTournament, tournsize=2)toolbox.register("mate", tools.cxOnePoint)toolbox.register("mutate", tools.mutFlipBit, indpb=1.0/INDIVIDUAL_LENGTH)
  • select:錦標賽法作為選擇運算符,tournsize表示錦標賽的大小
  • mate:單點雜交作為雜交運算符
  • mutate:多位翻轉突變作為突變運算符，indpb表示了每一個位置上基因突變的概率。在本文中INDIVIDUAL_LENGTH默認為100， 即每個體格由一百個二進制字符組成，每個字符突變的概率為1%

請特別注意：

select, mate, mutate都是DEAP框架中的關鍵詞，在定義運算符時，必須使用使用這幾個單詞。

3.6. 運行遺傳算法

# Genetic Algorithm flow:def main():    # create population    population = toolbox.createPopulation(n=POPULATION_SIZE)    # initialize statistics    stats = tools.Statistics(lambda ind: ind.fitness.values)    stats.register("max", np.max)    stats.register("avg", np.mean)    # Genetic Algorithm    population, logbook = algorithms.eaSimple(population, toolbox, cxpb=P_CROSSOVER,     mutpb=P_MUTATION, ngen=MAX_GENERATIONS,stats=stats, verbose=True)    # gather statistics    maxFitnessValues, meanFitnessValues = logbook.select("max", "avg")    # plot statistics:    plt.plot(maxFitnessValues, color='red',label="Max Fitness")    plt.plot(meanFitnessValues, color='green',label="Average Fitness")    plt.legend()    plt.xlabel('Generation')    plt.ylabel('Fitness')    plt.title('Max and Average Fitness over Generations')    plt.show()    print(max(population,key=lambda ind:sum(ind)))if __name__ == "__main__":    main()

在上述代碼中，我們定義了main()方程，運行該方程就可以進行遺傳算法的運算，同時生成對應的統計數據與圖表。

該方程中有以下步驟：

  1. 創建種群:種群大小為POPULATION_SIZE
  2. 初始化stats: 把個體的適應度收集進tools.Statistics對象。
  3. 通過algorithms.eaSimple方程運行遺傳算法，並將最終的種群與統計數據分別存入population和logbbok變量：
  • population：初始的種群
  • toolbox：toolbox變量，該變量內包括了我們之前定義的evaluate, select, mate 和 mutate方程
  • cxpb：雜交概率
  • mutpb：突變概率
  • ngen：最大迭代代數
  • stats：Statistics對象，用以儲存迭代過程中每一代種群的數據
  • verbose：在迭代過程中是否將Statistics輸出至標準輸出
  1. 從logbook變量中獲取每一代適應度的最大值和均值
  1. 使用收集的數據畫圖，並顯示適應度最高的個體，即最優解。

5. 小結
