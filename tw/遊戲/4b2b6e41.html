《星戰前夜：無燼星河》測試服前瞻

2020-11-14
| 遊戲

作者：NGA-小白白的寵物

今天下午維護後就將開放測試服，目前通過遊戲內的公告可知目前測試服更新的內容如下：
我會用紅字表達一些我自己的觀點。


新技能:

1、信號干擾技術*3(基礎進階專家)

能夠增加玩家使用信號干擾類裝備的效果和射程。包含索敵擾斷器、制導幹，擾器、感應抑阻器和標記裝置。

2、空間干擾裝置操作*3

能夠增加玩家使用範圍型躍遷干擾裝置的效果和範圍。包含攔截泡和躍遷擾斷力場發生器，這些技能主要是針對電子戰船和輕攔重攔，端遊也有類似的技能，可以參考端遊相關技能。


技能調整:

1、電子戰技術*3(基礎進階專家)

以前只會降低少量電子戰裝備的電容消耗。現在額外增加了掠能器和能量中和器的射程。 血襲船史詩級加強。

2、戰列艦技能和大型武器技能的科技等級，降低到8。這個必須要測試服上線後才能確定，如果是大型武器操作及升級以及戰列相關技能全面下調，呢麼T8混血戰列即可是完全體。 目前按照當前礦價及藍圖成本一艘響尾蛇30億起。


新艦船

T6 :

促進級攔截型、海燕級攔截型、強制者級攔截型、長尾鮫級攔截型，此4艘船可以安裝攔截泡，屬於特種作戰艦船。 輕攔，只能00打泡泡。

T8 :

黑鳥級攔截型、星空級攔截型、挑戰級攔截型、主宰級攔截型，能夠安裝躍遷擾斷力場發生器,也擅長電子干擾。 重攔，只能00打泡泡

T9 :

促進級攔截型I1、海燕級攔截型|1、強制者級攔截型|1、長尾鮫級攔截型|T6攔截型驅逐艦的增強型，作戰和攔截能力都有所提高。

T10：

暴君級攔截型、巨鳥級攔截型、托勒克斯級攔截型、斷崖級攔截型能夠安裝躍遷擾斷力場發生器，且防禦能力強。

在之前的解包裡似乎有個叫躍遷擾斷炸彈的加成，輕攔可以裝備， 端遊我大概棄坑了一年，所以我不確定這東西是否端遊也有， 問了下端遊也沒有，網易放飛自我的產物。具體作用是啥也不清楚，但是光憑描述應該是可以遠距離阻止你起跳的東東就對了

這次新船的改動下00將變得更加危險，但是相對的00的本地防禦也將更加完善。


艦船加成調整:

1、加成裝備變更

因為新裝備中，並不包含武器擾斷器和電子干擾器，所以這兩種加成被索敵
擾斷器和制導干擾器取代，涉及艦船有:

磨難級、獅鷲級、主宰級、主宰級隱匿型、主宰級隱匿型I1、黑鳥級、黑鳥級
隱匿型、黑鳥級隱匿型I、毒蠍級 ECM不開心了

2、加成技能變更

因為新增了”信號干擾技術"等3個技能，所以變更了部分干擾型艦船的加成技
能，涉及艦船有:

毛魯斯級、磨難級、獅鷲級、守夜者級、主宰級、黑鳥級、星空級、挑戰級

3、降低干擾射程加成

因為干擾裝備的射程很長，移除了部分護衛艦的射程加成，並降低了部分戰
列艦的最佳射程加成，改動列表(僅列出變化部分) :

毛魯斯級、磨難級、獅鷲級、守夜者級加成1 :最佳射程+5%->+0% (移除)

毒蠍級加成1 :最佳射程+25% -> +10%

4、降低干擾強度加成

縮短了部分高級隱匿型艦船在解除隱身後的鎖定延遲副作用的持續時間(buff
，降低了部分戰列艦的干擾強度加成，改動列表:

主宰級隱匿型1、黑鳥級隱匿型I1、星空級隱匿型I1、挑戰級隱匿型II

加成1 :干擾強度+10% -> +7.5%

特有加成:解除隱身鎖定延遲副作用-50%

毒蠍級加成1:干擾強度+15%->+10%

平衡性調整而已，常規操作，問題不大

5、傷害或射程調整

潛龍級傷害+175% -> +200%，還加？真就護衛當巡洋使?

響尾蛇級傷害+425% -> +450%，還加？

凝血級、夜魔俠級傷害+20% -> +25%，還加？近炮夜魔俠300DPS指日可待

阿什姆級、警惕級、復仇者級傷害+10% -> +12%，警惕和ASM的傷害基本上可以等效於T7戰巡了

巨神兵級、小鷹級、因卡薩斯級、裂谷級傷害+5% -> +7.5%

啟示級冷卻時間-3% -> -4%

啟示級海軍型冷卻時間-2% -> -3%，激光還是呢麼的完美無瑕，恭喜激光，更容易斷電了

托勒克斯級原型傷害+4% -> +5%

刺客級失準範圍+5% -> +7.5%

刺客級艦隊型失準範圍+7.5% -> +10%

甩蔥隊史詩級加強

猛鮭級、猛鮭級守衛型、猛鮭級守衛型最佳射程+5%->+10%

裸船40km的最佳射程，直接把無敵寫在了臉上，攻略再寫了

暴風級原型傷害+3%->+5%，

先驅者級原型傷害+8% -> +9%

布魯提克斯級、先驅者級傷害+5%-> +6%


武器調整:

1小型短管磁軌炮最佳射程+20%、失準距離+20%

2.小型導彈發射器飛行時間-15%

3.小型魚雷發射器飛行時間+20%

4.中型短管磁軌炮最佳射程+20%、失準距離+20%

5.中型自動加農炮跟蹤速度+10%

6.中型強襲加農炮跟蹤速度-15% 意料之內的削弱，現在強襲的跟蹤速度比磁軌還快就離譜

7.中型快速導彈發射器傷害倍數-15%，飛行時間-15% 大概還是能用。。。

8.中型導彈發射器傷害倍數+20% 不加爆炸速度和降低爆炸半徑基本上還是沒用

9.中型魚雷發射器傷害倍數+20% 除了隱轟沒人用，但是魚雷毒蜥刷T10說不定有奇效

10.大型快速導彈發射器傷害倍數+20%

11.大型魚雷發射器飛行速度+20%

12.中型無人機HP+50% PVP向的

13.大型無人機HP+100% PVP向的

l、修復了德拉米爾、塞納波、馬克瑞級不顯示躍遷加速度+50%的問題(實際
上有效果)。
