我的世界1.15.2拔刀劍終極武器合成筆記

2020-10-29
| 遊戲

最近在小夥伴的安利下，試玩了一下拔刀劍這個mod。1.15的拔刀劍跟之前版本不太一樣，經過一番折騰，終於把五把終極武器給合出來了。

下面這次合成的筆記，於己備忘，於人則是分享：

（1）mod版本號

Forge版本：forge-1.15.2-31.2.0

拔刀劍版本：SlashBlade-1.15-0.0.6

（2）五把終極武器

  • item.slashblade.yamato
  • -Chizuru- Muramasa
  • -Haze- SakuraFox
  • -Weiss- SakuraFox
  • Noble -Tukumo- Violet

（3）六種耀魂

  • Tiny Proud Soul（破碎的耀魂）
  • Proud Soul（耀魂碎片）
  • Blade Soul Ingot（耀魂鐵錠）
  • Blade Soul Sphere（耀魂寶珠）
  • Blade Soul Crystal（耀魂結晶）
  • Blade Soul Trapezohedron（耀魂偏方三八面體）

（4）耀魂升級配方

  • 2 x 破碎的耀魂 => 1 x 耀魂碎片（工作臺）
  • 2 x 耀魂碎片 + 1 x 鐵錠 => 1 x 耀魂鐵錠（工作臺）
  • 1 x 耀魂鐵錠 => 1 x 耀魂寶珠（熔爐）
  • 1 x 耀魂寶珠 => 1 x 耀魂結晶（高爐）
  • 1 x 耀魂結晶 => 1 x 耀魂偏方三八面體（煙燻爐）

（5）九種Blade Component

  • Blade Component -Bamboo-
  • Blade Component -Imitation-
  • Blade Component -White-
  • Blade Component -Iron-
  • Blade Component -Ruby-
  • Blade Component -Tukumo-
  • Blade Component -BlackFox-
  • Blade Component -WhiteFox-
  • Blade Component -Muramasa

（6）Blade Component 合成配方

  • 竹子 + 竹子 + 箱子 + 破碎的耀魂 => Blade Component -Bamboo-
  • 雞蛋 + 鐵粒 + 線 + 箱子 + 黑色燃料 + 紙 + 破碎的耀魂 => Blade Component -Imitation-
  • 耀魂鐵錠 + 耀魂鐵錠 + 箱子 + 金錠 => Blade Component -White-
  • 烈焰棒 + 耀魂寶珠 + 青金石塊 + 箱子 + 煤炭塊 + 線 + 金錠 => Blade Component -Iron-
  • 耀魂鐵錠 + 耀魂鐵錠 + 箱子 + 耀魂結晶 => Blade Component -Ruby-
  • 綠寶石塊 + 耀魂寶珠 + 鑽石塊 + 紅石塊 + 箱子 + 青金石塊 + 鐵塊 + 耀魂寶珠 + 金塊 => Blade Component -Tukumo-
  • 石英塊 + 羽毛 + 烈焰粉 + 箱子 + 耀魂結晶 + 小麥 + 黑曜石 => Blade Component -BlackFox-
  • 黑曜石 + 羽毛 + 烈焰粉 + 箱子 + 耀魂結晶 + 小麥 + 石英塊 => Blade Component -WhiteFox-
  • 靈魂沙 + 靈魂沙 + 箱子 + 耀魂偏方三八面體 => Blade Component -Muramasa-

（7）Noble -Tukumo- Violet 合成配方

  • 任意木塊 + 任意木塊 + 木劍 + 破碎的耀魂 => Anonymity -Wood-（工作臺）
  • Anonymity -Wood- + Blade Component -White- + 5級經驗 => Sharpness -White-（鐵砧）
  • Sharpness -White- => Sharpness -White- Broken（瘋狂殺怪）
  • Sharpness -White- Broken + Blade Component -Iron- + 10級經驗 => SlashBlade --（鐵砧）
  • SlashBlade -- + Blade Component -Tukumo- + 20級經驗 => Noble -Tukumo- Violet（鐵砧）

（8）-Haze- SakuraFox 合成配方

  • 任意木塊 + 任意木塊 + 木劍 + 破碎的耀魂 => Anonymity -Wood-（工作臺）
  • Anonymity -Wood- + Blade Component -Bamboo- + 1級經驗=> Anonymity -Bamboo Light-（鐵砧）
  • Anonymity -Bamboo Light- + Blade Component -Imitation- + 2級經驗 => Noted -Silver Bamboo Light-（鐵砧）
  • Noted -Silver Bamboo Light- => Noted -Silver Bamboo Light- Broken（瘋狂殺怪）
  • Noted -Silver Bamboo Light- Broken + Blade Component -Ruby- + 15級經驗 => Sharpness -nameless- Ruby（鐵砧）
  • Sharpness -nameless- Ruby + Blade Component -BlackFox- + 20級經驗 => -Haze- SakuraFox（鐵砧）

（9）-Weiss- SakuraFox 合成配方

  • 任意木塊 + 任意木塊 + 木劍 + 破碎的耀魂 => Anonymity -Wood-（工作臺）
  • Anonymity -Wood- + Blade Component -Bamboo- + 1級經驗=> Anonymity -Bamboo Light-（鐵砧）
  • Anonymity -Bamboo Light- + Blade Component -Imitation- + 2級經驗 => Noted -Silver Bamboo Light-（鐵砧）
  • Noted -Silver Bamboo Light- => Noted -Silver Bamboo Light- Broken（瘋狂殺怪）
  • Noted -Silver Bamboo Light- Broken + Blade Component -Ruby- + 15級經驗 => Sharpness -nameless- Ruby（鐵砧）
  • Sharpness -nameless- Ruby + Blade Component -WhiteFox- + 20級經驗 => -Weiss- SakuraFox（鐵砧）

（10）-Chizuru- Muramasa 合成配方

  • 任意木塊 + 任意木塊 + 木劍 + 破碎的耀魂 => Anonymity -Wood-（工作臺）
  • Anonymity -Wood- + Blade Component -White- + 5級經驗 => Sharpness -White-（鐵砧）
  • Sharpness -White- => Sharpness -White- Broken （瘋狂殺怪）
  • Sharpness -White- Broken + Blade Component -Iron- + 10級經驗 => SlashBlade --（鐵砧）
  • SlashBlade -- + Blade Component -Muramasa- + 20級經驗 => -Chizuru- Muramasa（鐵砧）

（11）item.slashblade.yamato 合成配方

  • 紫珀塊 + 紫珀塊 + 龍首 + 耀魂偏方三八面體 => item.slashblade.yamato（工作臺）
