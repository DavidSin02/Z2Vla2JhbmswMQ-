《塞爾達傳說：曠野之息》「Eldin Tower / オルディンの塔 地區」試煉之祠攻略

2020-10-29
| 遊戲

下面開始《塞爾達傳說 荒野之息》東北火山地區的試煉之祠攻略。

Mo’a Keet Shrine / モア・キシトの祠

  • 解謎關鍵：時間凝滯，磁力
  • 解謎思路：用金屬球擋住滾石

用時間凝滯停止滾下來的圓石，前進到上面的橫橋，左邊有多個圓石滾下，右側有個金屬球滾下。去右邊把滾下來的金屬球用磁力吸住，別讓它滾進岩漿，然後從金屬球滾下的跑道上去拿寶箱。

帶著金屬球返回進入祠的地方，最右邊的上方有被碎石堵住的地方，用磁力吸住金屬球上下揮動可以敲碎岩石，掉下一個寶箱，裡面是一塊紅色礦石。

吸著金屬球回到上面的橫橋，帶著金屬球是可以推住圓石的，去左側的斜坡，一路上吸住金屬球擋住滾石並推動滾石前進，就能來到終點。

Sah Dahaj Shrine / サ・ダージュの祠

  • 解謎關鍵：箭
  • 解謎思路：點燃枯葉

第一階段：用帶火的箭射向枯葉，也可以用時間凝滯能力把倒在地上的火炬打到枯葉上。

第二階段：擊敗守護者，上梯子，用箭射斷吊燈。

第三階段：用帶火的箭射向剛進門後右邊的枯葉拿寶箱。然後射下圖上畫圈的兩個吊燈，石板便會壓住按鈕，門開啟完成謎題。

Tah Muhl Shrine / タ・ムールの祠

  • 解謎關鍵：箭，磁力
  • 解謎思路：點燃物品

這個試煉之祠的位置關係到一個“Shrine Quest / 祠任務”：A Landscape of a Stable / 馬宿の風景畫。

在【Foothill Stable / 山麓の馬宿】和【Mayro / サンフ】對話可領取這個任務，他會提示你去看馬廄牆上那副畫，畫中有一個試煉之祠，也就是我們的目的地，找到它並完成其中挑戰即可完成任務。

當然，不領取這個祠任務也行，直接去試煉之祠開始挑戰：

用帶火的箭點燃正對著祠入口的枯葉，要是沒有火炎箭，旁邊有燈臺，持弓用木箭接近就能點燃。點燃枯葉後出現入口，繼續前進。

用帶火的箭點燃進去後前方三個疊起來的木箱，拿上面的寶箱。

點燃右邊兩個平放木箱後面的枯葉，拿裡面的寶箱。

之後點燃需要鑰匙才能開啟的門的左邊的枯葉，用磁力把裡面的鑰匙吸出來，開啟門進入拿寶箱。

最後點燃頂著橋的四個木箱完成謎題。

Qua Raym Shrine / キュ・ラムヒの祠

  • 解謎關鍵：磁力，遙控炸彈，箭
  • 解謎思路：天平

進入祠，正面是天平，用火箭點燃左側盤內的木箱，右側盤會下沉，打開寶箱獲得鑰匙。

天平旁邊高臺上有兩塊岩石，擋著通往終點的路，先在下面扔遙控炸彈的方式炸開它們，爆炸範圍是可以波及到的。

用鑰匙開啟祠入口對面的門，裡面有三個金屬塊，用磁力把一個金屬塊放到祠入口左邊高臺旁邊的天平，再用另一金屬塊放在那個天平旁邊，這樣玩家就可上到下降的天平上。

把自己這邊天平上的金屬塊放到另外一邊的天平上，自己所在的這邊便會上升到高臺處，拿寶箱。

之後把三塊金屬塊都拉到中央天平附近，把一個金屬塊放到右盤，玩家也站上去；其餘兩個金屬塊放到左盤。自己這邊天平上升，完成謎題。

Shae Mo’sah Shrine / シモ・イトセの祠

  • 解謎關鍵：時間凝滯，箭
  • 解謎思路：轉動齒輪

踩住前方的按鈕並用時間凝滯停止按鈕，進入閘門。

進入後左邊有寶箱，然後去右邊擊敗三個守護者，拿在角落的寶箱。

上斜坡後前進一小段發現樓梯上有個按鈕，踩上去出現一個帶火的吊燈，用箭射向它使其搖擺點燃枯葉，這樣金屬球便會進入跑道，吊燈下面還有寶箱。

之後一路前進找到齒輪前的一個按鈕，踩上去，擋住金屬球的閘門開啟，金屬球落下齒輪開啟後面的閘門，進入閘門拿寶箱裡的鑰匙。

用鑰匙開啟上斜坡後的那扇門，進入門裡樓梯上有按鈕可以不用理會它，直接踩著閘門左邊的另一個按鈕然後時間凝滯停住它，進入完成謎題。

Shora Hah Shrine / ショラ・ハの祠

可以Shae Mo’sah Shrine後面乘軌道車去。

  • 解謎關鍵：磁力，箭，火把，時間凝滯
  • 解謎思路：用藍色火焰點燃火臺

用磁力吸起帶針鐵塊，過來到十字路口，先往左走。

用磁力調整金屬臺形成臺階，上去拾得火把，並用藍火點燃火把。這裡左側高臺上有個寶箱，可以交替移動鐵塊，過去開啟寶箱。

返回十字路口點燃在十字路口的火臺，前方的閘門開啟。

舉著火把前進，裡面有三條水槍，前兩個可以繞路避開，第三個需要蹲著通過。

點燃盡頭的火臺，此時燈左邊的升降臺便會啟動，那裡有個火臺。用木箭點上藍色火焰，射中對面升降臺上的火臺，此時十字路口針臺對面的路便會接上。

沿著接上的路前進，踩下按鈕，此時會有金屬球滾下來，用磁力吸住並扔掉，繼續前進擊敗4個守護者，拿周邊的兩個寶箱。返回升降臺處點燃火把，用火把點燃與守護者戰鬥的地方的火臺，閘門開啟。

進入閘門點燃前方的火臺，然後用時間凝滯停掉其中一個水槍，然後快速先點燃停掉的水槍前的火臺再點燃另一個水槍前的火臺，閘門開啟。

帶著火把繼續前進，點燃前面的火臺，然後給木箭點上藍色火焰，等兩個浮臺快要兩點一線時射出箭同時點燃上面的火臺。這時候浮臺停止移動，並用出現上升的氣流。

利用上升的氣流來到第二個浮臺，用木箭點火，點燃左斜方對面的火臺。

去到剛點燃的火臺處，用火把點燃旁邊金屬臺上的火臺。然後來到金屬臺右邊水槍的對面，用時間凝滯停掉水槍然後迅速切換磁力將金屬臺拉到對面去。

乘著上升氣流繼續前進，金屬臺軌道旁的寶箱可以用磁鐵吸過來。去到金屬塊那裡，用磁力把金屬塊右邊的寶箱吸過來，然後用滑翔傘到金屬塊左邊的平臺上拿另一個寶箱，返回金屬塊處點燃火把。

下去發現一群守護者，不用與它們戰鬥，用火把點燃地下的枯葉燒死守護者，繼續前進拿寶箱。

來到七個火臺七個水槍的地方，需要同時點燃它們，站在中央用點燃的火把蓄兩段力再釋放即可。

Daqa Koh Shrine / ダカ・カの祠

  • 解謎關鍵：滑翔傘，時間凝滯
  • 解謎思路：到達巨石

進入祠，發現一個巨大彈板在不停地彈射著一個巨石。進入彈板等其將林克彈到空中，在空中開啟滑翔傘，降落到巨石上。

面向祠入口方向，左邊有個寶箱，等巨石再度彈起，飛到那裡拿寶箱。

用同樣的方法返回巨石，面向反方向，巨石彈起到達最高處時裝置會接觸由此打開閘門，當閘門打開時用時間凝滯停止巨石，飛向閘門完成謎題。

Kayra Mah Shrine / ケハ・ラマの祠

  • 解謎關鍵：磁力
  • 解謎思路：避開金屬球

這是一個祠任務，解放了炎之神獸在 Goron City 的 【Bladon / ブレードン】處接到。

任務要求玩家去 【Gorko Tunnel / マルゴ坑道】（也就是祠所在地附近）。

進入山洞裡跟躺著的 【Gonguon / ゴングロン】 對話。要你去幫他找食物，其實就是要你從山洞出去左轉一路下去，在岩漿池搬回肉型石頭，在途中有很多敵人，還有滾石，做好戰鬥準備。

進入祠，是裡面是斜坡，金屬球會不停得從上面滾下來，玩家只要持續避開一直到最上面即可，如果覺得吃力的話可以用磁力吸住一個金屬球擋在前面前進。

一起滾下來的還有盧比，有餘力的可以順便收了，另外一路上有兩個寶箱，一個在剛進入斜坡不久的右邊，另一個在快到最上層的左邊，從頂部乘滑翔傘飛過去即可。

Gorae Torr Shrine / グ・アチトーの祠

  • 解謎關鍵：體力，登山服，體力藥
  • 解謎思路：爬到最高處

祠的位置在頂部高臺，有上升氣流可以用滑翔傘上去，與在祠入口處的【Bayge / バケット】對話接祠任務。

任務要求玩家在限定時間內爬上剛剛乘上升氣流上去的那個高臺，並在爬的過程中收集100盧比。有兩圈體力加上登山服再有些體力藥預防可以輕鬆應對，只要沿著有盧比的地方一路爬上去即可，時間很充裕。

這裡今後就是一個攀巖的小遊戲了。
