伯牙子期，高山流水，兄弟情友情在古人眼裡到底地是什麼樣的？

一生大笑能幾回，斗酒相逢須醉倒。____岑參《涼州館中與諸判官夜集》

浮雲一別後，流水十年間。____韋應物《淮上喜會樑川故人 / 淮上喜會梁州故人》

同是天涯淪落人，相逢何必曾相識！____白居易《琵琶行 / 琵琶引》

莫愁前路無知己，天下誰人不識君。____高適《別董大二首》

世人結交須黃金，黃金不多交不深。____張謂《題長安壁主人》

洛陽親友如相問，一片冰心在玉壺。____王昌齡《芙蓉樓送辛漸》

海內存知己，天涯若比鄰。____王勃《送杜少府之任蜀州 / 送杜少府之任蜀川》

勸君更盡一杯酒，西出陽關無故人。____王維《渭城曲 / 送元二使安西》

浮雲遊子意，落日故人情。____李白《送友人》

四海皆兄弟，誰為行路人。____佚名《舊題蘇武詩 / 別詩四首·其一》

故人入我夢，明我長相憶。____杜甫《夢李白二首·其一》

過眼年華，動人幽意，相逢幾番春換。____王沂孫《法曲獻仙音·聚景亭梅次草窗韻》

唯願當歌對酒時，月光長照金樽裡。____李白《把酒問月·故人賈淳令予問之》

故人江海別，幾度隔山川。____司空曙《雲陽館與韓紳宿別》

孤帆遠影碧空盡，唯見長江天際流。____李白《黃鶴樓送孟浩然之廣陵》

故人相望若為情。別愁深夜雨，孤影小窗燈。____陳克《臨江仙·四海十年兵不解》

君不見管鮑貧時交，此道今人棄如土。____杜甫《貧交行》

不才明主棄，多病故人疏。____孟浩然《歲暮歸南山 / 歸故園作 / 歸終南山》

西窗下，風搖翠竹，疑是故人來。____秦觀《滿庭芳·碧水驚秋》

故人西辭黃鶴樓，煙花三月下揚州。____李白《黃鶴樓送孟浩然之廣陵》

相逢一醉是前緣，風雨散、飄然何處。____蘇軾《鵲橋仙·七夕》

數人世相逢，百年歡笑，能得幾回又。____何夢桂《摸魚兒·記年時人人何處》

桃花潭水深千尺，不及汪倫送我情。____李白《贈汪倫》

江南無所有，聊贈一枝春。____陸凱《贈范曄詩》

我寄愁心與明月，隨風直到夜郎西。____李白《聞王昌齡左遷龍標遙有此寄》

忽憶故人今總老。貪夢好。茫然忘了邯鄲道。____王安石《漁家傲·平岸小橋千嶂抱》

恰如燈下，故人萬里，歸來對影。____黃庭堅《品令·茶詞》

春草明年綠，王孫歸不歸？____王維《送別 / 山中送別 / 送友》

洞庭有歸客，瀟湘逢故人。____柳惲《江南曲》

古路無行客，寒山獨見君。____劉長卿《碧澗別墅喜皇甫侍御相訪》

誰知江上酒，還與故人傾。____周紫芝《臨江仙·送光州曾使君》

不知來歲牡丹時，再相逢何處。____葉清臣《賀聖朝·留別》

滿目飛花萬點，回首故人千里，把酒沃愁腸。____葛長庚《水調歌頭·江上春山遠》

開軒面場圃，把酒話桑麻。____孟浩然《過故人莊》

丈夫貧賤應未足，今日相逢無酒錢。____高適《別董大二首》

故人何在，水村山郭。____孫道絢《憶秦娥·季溫老友歸樵陽人來閒書因以為寄》

尊前故人如在，想念我、最關情。____周邦彥《綺寮怨·上馬人扶殘醉》

嵩雲秦樹久離居，雙鯉迢迢一紙書。____李商隱《寄令狐郎中》

不辭山路遠，踏雪也相過。____張九齡《答陸澧》

故人早晚上高臺。贈我江南春色、一枝梅。____舒亶《虞美人·寄公度》

欲尋芳草去，惜與故人違。____孟浩然《留別王侍御維 / 留別王維》

忽憶故人天際去，計程今日到梁州。____白居易《同李十一醉憶元九》

長安故人問我，道尋常、泥酒只依然。____辛棄疾《木蘭花慢·滁州送範倅》

何因不歸去？淮上有秋山。____韋應物《淮上喜會樑川故人 / 淮上喜會梁州故人》

偶然值林叟，談笑無還期。____王維《終南別業 / 初至山中 / 入山寄城中故人》

相逢意氣為君飲，繫馬高樓垂柳邊。____王維《少年行四首》

還怕掩、深院梨花，又作故人清淚。____吳文英《無悶·催雪》

還與去年人，共藉西湖草。____蘇軾《卜算子·感舊》

怨別自驚千里外，論交卻憶十年時。____高適《東平別前衛縣李寀少府 / 送前衛縣李宷少府》

歡笑情如舊，蕭疏鬢已斑。____韋應物《淮上喜會樑川故人 / 淮上喜會梁州故人》

河西幕中多故人，故人別來三五春。____岑參《涼州館中與諸判官夜集》

蕃漢斷消息，死生長別離。____張籍《沒蕃故人》

料因循誤了，殘氈擁雪，故人心眼。____張炎《解連環·孤雁》

雲深山塢，煙冷江皋，人生未易相逢。____吳文英《聲聲慢·友人以梅蘭瑞香水仙供客曰四香分韻得風字》

當年紫禁菸花，相逢恨不知音早。____王惲《水龍吟·送焦和之赴西夏行省》

南去北來徒自老，故人稀。____賀鑄《釣船歸·綠淨春深好染衣》

芳草已雲暮，故人殊未來。____韋莊《章臺夜思》

故人具雞黍，邀我至田家。____孟浩然《過故人莊》

今夜故人來不來，教人立盡梧桐影。____呂岩《梧桐影·落日斜》

整理不易，且看且關注。
