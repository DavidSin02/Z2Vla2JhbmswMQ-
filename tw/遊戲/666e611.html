序言

首發於NGA，轉載需註明。本文主要介紹一些劍三坦克的基礎理論及相關知識，並沒有涉及到具體每種坦克的操作和詳情。因為我始終覺得，願意仔細看一些基礎理論的玩家，能夠了解這些東西，舉一反三可以很快了解自己該做的事情，至於熟悉一個坦克職業的大致操作，有劍三基礎的玩家都用不了太久。通篇會圍繞“我為什麼要這麼做”而不是“我要怎麼做”來進行，更希望閱讀過這篇帖子的小夥伴能夠自我拓展掌握到更多的東西。作者本身也只是一個平凡的玩家，行文之中可能出現紕漏，萬望海涵。

以下內容適用於整個100級版本

一、坦克屬性作用介紹

  • 體質：1點體質提高玩家10點基礎氣血值。
  • 最終氣血值 = ( 人物氣血值 + 基礎氣血值 ) * ( 1 + 基礎氣血提高總百分比 ) + 體質 * 心法體質氣血值額外轉換系數
  • 人物氣血值：100級玩家的人物氣血值為9116。
  • 基礎氣血值：見上文，由體質提供。
  • 基礎氣血提高百分比：可以將鼠標移至技能欄左下角的心法圖標上查看本心法的心法基礎氣血提高百分比，例如：明尊琉璃體的心法基礎氣血提高百分比為47%。
  • 撼如雷、清心靜氣、超凡入聖、禪語等基礎氣血提高百分比效果均為增加，彼此之間疊加計算。這些效果與心法基礎氣血提高百分比亦為疊加計算。
  • 共戰江湖的效果為“最終氣血值提升x%”，故共戰江湖為唯一的疊乘計算。
  • 心法體質氣血值額外轉換系數：同樣可以將鼠標移至技能欄左下角的心法圖標上查看本心法的心法體質氣血值額外轉換系數，例如：明尊琉璃體的心法體質氣血值額外轉換系數為1.25。
  • 命中：每139.485點命中等級提高玩家1%命中，命中會影響玩家的偏離率。
  • 玩家的偏離率 = 所需命中 - 玩家命中
  • 一般來說，對不同等級目標所需命中如下：

  • 特殊的，“降低被命中機率”和“降低對方命中”均為疊加計算，而非疊乘。
  • 例如：攻擊處於“鳥翔碧空”(2本降低被命中機率祕籍)狀態下的唐門玩家時，所需命中+10%。當自身被銀月斬(降低命中機率祕籍)/翼絕雲天(降低命中機率祕籍)命中時，所需命中+5%。
  • 加速：每188.309點加速等級提高玩家1%加速。
  • 關於加速更為詳細的解釋，見[[理論基礎][世外蓬萊]劍網3屬性計算與收益分析]中第四部分
  • 外功/內功防禦等級：降低玩家/NPC受到的外功/內功傷害。
  • 外功 / 內功減傷 = 防禦等級 / ( 防禦等級 + 6942.8 )
  • 所有提高內外防效果(如鐵牢心法提高外防、洗髓心法提高內防、明尊奇穴極本溯源、除蕩曠劫等)均為提高防禦等級。
  • 由於分母中存在常數項，防禦等級提高10%實際上並不是使收到的傷害降低10%。
  • 外功/內功減傷最高為75%。超過75%時，按照75%計算。
  • 閃躲：使玩家有一定機率閃避來自玩家的外功攻擊和來自NPC的部分外功攻擊。如果閃避了某個外功技能，則該技能附帶效果一併被閃避。
  • 閃躲率=閃躲等級/(閃躲等級+9025.3)
  • 所有提高閃避效果，除非明確描述為提高x點閃避等級(如明尊心法)，否則均為直接提高閃躲率。因此，在提高閃避效果的加持下，閃躲率有可能達到100%。
  • 招架：使玩家有一定機率招架來自玩家的外功攻擊和來自NPC的部分外功攻擊。招架後本次傷害將被部分抵消，抵消值相當於玩家的拆招值。
  • 招架率=招架等級/(招架等級+6518)
  • 所有提高招架效果，除非明確描述為提高x點招架等級(如鐵骨心法)，否則均為直接提高招架率。因此，在提高招架效果的加持下，招架率有可能達到100%。
  • 拆招：玩家招架攻擊時所受傷害降低的數值。
  • 御勁：使玩家降低被會心攻擊的機率(以下簡稱御率)，並降低玩家被會心攻擊時實際的會心效果(以下簡稱御效)。每153.442點御勁等級提高玩家1%御率，每42.906點御勁等級提高玩家1%御效。也即：每1%御率提高玩家3.577%御效。
  • 被會心攻擊時
  • 實際會效 = 1 + ( 面板會效 - 1 ) * ( 1 - 御效 )
  • 副本中Boss的會心數值隨Boss不同而不同，初始會效為200%
  • 威脅：使玩家攻擊怪物時能夠產生更高的威脅值。
  • 技能仇恨 = 技能基礎仇恨值 * (1 + 威脅值提升率)+ ( 技能傷害 * 傷害仇恨係數 )*( 1 + 威脅值提升率 - 威脅值降低率 )
  • 通常情況下，在進入戰鬥時，怪物會攻擊仇恨列表中仇恨最高的玩家
  • 當怪物已經擁有一個目標時，需要仇恨值達到第一仇恨的120%才會更變目標進行攻擊
  • 當仇恨值不是怪物仇恨列表第一時時，使用強仇技能會立刻使自己的仇恨值和第一相同，並在持續時間內強制怪物攻擊自己
  • 多個強仇技能同時作用時，怪物會根據技能優先級攻擊使用對應技能的目標(該部分測試較為麻煩，不同職業之間存在優先級變化)
  • [例1]
  • 問：坦克在獲得無威脅氣勁時，是否還能夠繼續製造仇恨？
  • 答：能
  • 分析：根據技能仇恨公式，無威脅氣勁只會影響到技能傷害產生的仇恨部分，基礎仇恨值不受影響，故可以繼續製造少量仇恨。
  • [例2]
  • 問：需要拉背對的Boss戰時，dps仇恨達到了坦克的90%，此時Boss施放了點名技能，恰好點名該dps，會發生什麼？
  • 答：可能團滅
  • 分析：仇恨目標切換到了dps，dps仇恨被視為100%，坦克仇恨變為111%，未能達到OT條件，Boss會繼續攻擊該dps，引發的順劈可能導致團滅。
  • 無雙：每87.176點無雙等級提高玩家1%無雙，無雙會影響玩家的被識破率。
  • 和DPS不同，對於坦克來說，無雙是對仇恨提升微乎其微的屬性，因為坦克本身的技能傷害足夠低，仇恨大部分依賴的是技能的基礎仇恨而非傷害來的的額外仇恨

二、圓桌理論和有效生命值

在坦克屬性介紹中我們可以知道，除了體質和對應的氣血以外，還有很多防禦屬性會影響到坦克的生存。在這裡，我們需要提到兩個概念：[圓桌理論]和[有效生命值]

  • 圓桌理論：理論詳情可以在[→點擊這裡←]查看。
  • 對於坦克來說，承受傷害時，以下幾種狀態會被依次搬上圓桌
  • 未命中
  • 閃避
  • 招架
  • 會心
  • 直接命中
  • 實際上如果瞭解過CTM版本之前的wow，會發現這套理論是如此熟悉……嗯，沒有錯，回到十年前來看，倆遊戲的數值體系幾乎就是一樣的
  • 在劍三中，這個理論的實際意義並不是很大，因為劍三的坦克三圍不足以支持塞滿圓桌來避開會心和直接命中判定，知道這個理論就足夠了。這也是後面我們考慮屬性時忽視閃避和招架的原因
  • 有效生命值：我們可以把氣血和防禦等級進行一個換算，來衡量一個坦克的生存能力，評估是否可以在治療的承受範圍內避免猝死，這就是有效生命值。
  • 根據上文中防禦減傷公式
  • 外功 / 內功減傷 = 防禦等級 / ( 防禦等級 + 6942.8 )
  • 我們令
  • 致死所需基礎傷害 *(1-減傷)= 氣血值上限
  • 在這裡我們認為“致死所需基礎傷害”即為計算減傷前的有效生命值，兩式合併得
  • 有效生命值 = 氣血值上限 * ( 1 + 防禦等級 / 6942.8)
  • 可以看到，有效生命值和防禦等級是線性相關的

三、坦克屬性收益

談論屬性收益時，我們需要明確一點就是，坦克需要每種屬性的意義是什麼：

  • 命中：保證技能不發生偏離，這關聯到仇恨製造能力，以及強仇技能的效用性
  • 御勁：保證自身受到怪物攻擊時不會發生會心猝死事件
  • 防禦等級：保證自己的有效生命值足夠多，給治療以緩衝時間
  • 加速：加快技能的整體循環，以提升仇恨能力和生存能力
  • 威脅：保證自己的仇恨製造能力
  • 閃躲/招架：在有效生命值之外提供概率性的生存能力

在一場pve的戰鬥中，坦克的作用是為dps提供穩定輸出環境能夠過掉這個boss，那麼以下情況的危險程度排名就是：

團滅>倒T>OT

  • 團滅
  • 命中：強仇技能的偏離會導致Boss關鍵機制意外運作，直接引發團滅結果
  • 倒T
  • 防禦等級：有效生命值過低，治療沒有緩衝時間，會直接導致倒T
  • 御勁：當發生會心事件時，瞬間的高傷害導致治療沒有緩衝之間，會直接導致倒T
  • 閃躲：一定程度上可以緩解倒T，如果可以達到必定發生，則可以完全避免倒T
  • 招架：概率性的提升有效生命值的方式
  • OT
  • 命中：命中不足頻繁偏離導致仇恨過低OT
  • 加速：加速可以使仇恨製造能力提升
  • 威脅：直接關聯到仇恨高低
  • 無雙：極其微小地提高了仇恨值

不考慮心法自身特色時，坦克需要使自己的裝備按以下優先級配齊屬性：

  • 一、優先使命中達到你所需要的主流本要求，這關聯到你的仇恨製造能力，並且能夠避免強仇偏離的意外情況
  • 二、儘可能高地提升你的防禦等級，根據對應boss的傷害類型不同，可能需要準備內防/外防以應對不同場合
  • 三、以當前版本的最高難度副本為目標時，需要把御勁提到相應的水準，一般來說數值設計上御勁不足會出現猝死
  • 四、提升加速到合適的閾值，來提升技能的循環速度
  • 五、提高威脅等級來仇恨更加穩定

也即是說，坦克屬性和dps屬性不同之處在於，有下限要求，而上限並不難以追求

四、裝備屬性與配裝理念

我們需要知道一個前提：劍三的裝備屬性分數和數量容量並不是嚴格相等的，在某些情況下，同品質的屬性容器中，屬性的容量可能並不相同

裝備

除精簡裝備外，坦克裝備屬性通常由以下格式構成：

體質 + 防禦等級 / 防禦等級&閃避 / 防禦等級&招架&拆招值 + 御勁 / 命中 / 加速 + 無雙&威脅值

對於同品級的裝備，除了精簡和無界，其他的屬性容量基本是相同的。我們將防禦等級的權重定位1，有以下權重結果：

  • 防禦等級=1
  • 招架=1
  • 閃避=1
  • 御勁=3
  • 加速=1
  • 命中=1
  • 無雙=0

也就是說，我們可以簡單視為：御勁在裝備上會佔用三倍的屬性容量，而無雙在坦克裝備上不佔用屬性容量

附魔

對於同品質的附魔，我們將防禦等級的權重定位為1，有以下權重結果：

  • 防禦等級=1
  • 招架=1
  • 閃避=1
  • 御勁=1
  • 加速=1
  • 命中=1
  • 無雙=1

也就是說，我們可以簡單視為：御勁可以通過附魔較為廉價的獲得

配裝

通過對裝備和附魔的特點分析，我們基本可以得到以下兩點結論：

  • 一、儘量通過附魔優先蒐集御勁屬性
  • 二、配裝過程中屬性允許的話，可以儘量使用帶有無雙的裝備，附帶的無雙相當於是白送的屬性

由於在屬性傾向中我們摒棄了閃避和招架，因此配裝時幾乎都會採用 純外/內防+命中/御勁/加速 的裝備 (蒼雲除外，下一小節會提到)

需要注意的是，由於每個版本能夠獲取的坦克裝備在同一品級並非所有屬性組合都覆蓋了，所以往往需要向下尋找低品級的裝備來搭配自己的屬性

例1：

使用最高不超過2130的裝備來配置一套能夠用於開荒25人挑戰模式冰火島的明尊裝備

我們可以看到，該套裝備滿足了明尊加速奇穴下的四段加速閾值，同時使用附魔極大補足命中和御勁的缺口，甚至為了御勁達標，護腕和褲子還使用了上賽季的御勁附魔。

例2：

問：已知100級重製版10人鍛刀廳關底解語的攻擊為25000/1.5秒外功傷害附帶1000/層/秒最大疊加40層的毒性內功傷害，面對解語時在屬性容量有限的情況下如何搭配內外防更加合適？

答：雙T純外防，單T純內防

分析：雙T情況下因為毒性buff基本不會超過10層，我們完全不用考慮毒性內功傷害部分，重點講單T的情況

正常來說，毒性傷害隨時間推移會逐漸接近40000/秒的基礎傷害，我們需要考慮的是高壓力下坦克的存活。此時坦克會承擔的是16667/秒的外功基礎傷害和40000/秒的內功基礎傷害

我們將模型簡化為氣血值上限120000，可分配防禦等級為6942.8的坦克

根據公式：有效生命值 = 氣血值上限 * ( 1 + 防禦等級 / 6942.8)

我們令外功防禦為6942.8的x%

對外功傷害部分： 有效生命值A = 氣血值上限 * ( 1 + X%)

對內功傷害部分： 有效生命值B = 氣血值上限 * ( 2 - X%)

由於內功基礎傷害=2.4*外功基礎傷害

想要令坦克更久存活，需要使B儘可能接近2.4*A

但由於x%∈[0,1]，故取極限情況，x%=0，B=2A，此時坦克的可能存活時間最久

五、四坦特點分析與比較

首先我們可以在遊戲中取得四個坦克心法的對應數據：(已刨除非坦克相關屬性)

僅從心法加成上看，我們可以得到幾個簡單的結論：

  • 明尊有很高的閃避
  • 鐵牢有很高的外防
  • 洗髓有很高的內防和血量
  • 鐵骨衣有很高的招架和拆招

僅從數值差異上來看，內外防硬度分別是由洗髓和鐵牢領跑的，但是劍三的Boss傷害在數值要求上很少會很苛刻，反而對坦克的功能性有一定要求，對應四個坦克的優缺點如下：

  • 就團隊輔助能力來說，明教更為全面，所以一般會作為主T使用
  • 洗髓可以始終提供團隊血量上限的提升，有單保人能力，並可以解決明教T最困擾的仇恨問題，故作為副t使用很合適
  • 鐵牢提供了可觀的團隊爆發和有效的外功易傷，同時自身的適應性也不錯，無論主t副t都有一席之地，雖然大多數情況只作為副t存在
  • 蒼雲相比鐵牢的團隊輸出提升並不出彩，自身輸出和團隊增傷衝突。由於自身核心機制是關聯拆招值的護盾，裝備特性需求拆招裝，導致防禦等級低下，面對無視招架的Boss十分疲軟。 故一般作為藏寶洞主存在

[例]

問：100級重置版10人燕然峰石斑的飛石，單人承傷時基礎傷害為200000外功傷害，30000裝分左右的坦克去單吃，掉血量如何？

答：

  • 明尊：基礎傷害20W，外防減傷40%，剩餘12W，貪魔體減傷80%，剩餘2W4傷害，單吃無事發生
