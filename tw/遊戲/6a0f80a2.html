# 你需要知道的Red Wing的4款鞋型 #

● 875

如果你是首次入手 Red Wing，又不知道該選哪款好，875是你不錯的選擇。它屬於Classic系列，也是一款百搭的靴品。Moc toe的壓印縫合方形鞋頭是Red Wing最具代表性的鞋款，而Moc這個詞則來源於「moccasin」，這也是印第安 Algonquian族人對鞋子的稱呼。

875 還選用了Oro Legacy Leather，鞋頭偏翹，款式基本上和 877 是相同的；橡膠製鞋底，有小型的胎面設計，緩衝功能極高，還能在行走時能減低陷入泥地的距離，簡直是出家旅行的良伴呢~

● 8111

說到8111，還需要知道這個背景喲~在明尼蘇達州北部有個 Mesabi 鐵礦區，這裡的居民大多都是礦工。在當時礦業繁榮，但工作環境十分危險，礦工們需要經得起惡列環境考驗的工作靴~

而Red Wing就以礦工的工作環境為靈感，設計了Iron Range系列。而這個系列中最拉風的款式就是8111，它選擇了雙層的皮革設計，可以更好地保護腳趾和腳跟！琥珀色皮革也相當低調而華麗，極具復古的質感~

Nitrile Cork鞋底也更加舒適，讓人愛不釋手，這雙鞋也很受女生歡迎呢~

● 9013

Red Wing Beckman 系列可以說是Red Wing 中的貴族，而9013 是這個系列中的一員。

9013所使用的Chestnut Featherstone皮革來自於Red Wing公司自己的鞣皮廠，這種皮革加工過後蠟和油仍然很豐富，靴子擦過之後光澤度敲zàn哦！

● 1907

都說Red Wing的「1907」和 「875」是比翼雙飛，其實能1907的前身是1905。它是Red Wing推出的百年紀念款，每一款全球限量2005雙~

而1907與一般的Moc-toc系列還不太一樣，它特別使用了Red Wing No.45鞋楦、D頭的設計，使鞋頭更加平滑。為了保證其舒適度，1907還提供了一雙皮質鞋墊及兩副鞋帶，可更換使用。
