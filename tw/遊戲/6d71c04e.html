魔獸世界懷舊服法師攻略大全：懷舊服最強輸出職業是怎樣的？

2020-10-29
| 遊戲

法師會用神祕的咒語摧毀敵人。雖然他們有著威力巨大的攻擊性法術，但法師並不強壯，裝甲也很單薄，這使得他們面對近距離攻擊時非常脆弱。精明的法師會精心使用法術，將他們的敵人限制在一定距離或者控制在適當的位置。

一、 法師的基礎信息

Ø 可選種族：人類、侏儒、亡靈、巨魔。

Ø 作用：遠程傷害，AOE傷害。

Ø 資源：生命值、法力值。

Ø 護甲類型：布甲

Ø 武器：法杖、魔杖、匕首、劍和副手物品。

Ø 屬性：智力、命中、爆擊和法術傷害加成。

Ø 優勢：出色的單體傷害、AOE傷害和群體控制。通過風箏敵人提高生存能力。

Ø 劣勢：缺乏有效的戰鬥回覆能力，法力值就像生命值一樣重要。

Ø 特殊能力：可以製作食物和水，並且能為隊友開啟傳送門。

Ø 適合的專業：裁縫、附魔、工程。

二、 法師技能介紹

Ø 攻擊法術

n 火球術：火法核心技能，傷害高但消耗也高，能造成額外的火焰傷害dot。

n 寒冰箭：冰法核心技能，傷害高，能對敵人造成減速。

n 火焰衝擊：瞬發的火焰傷害技能，適用於PVP。

n 奧術飛彈：法師的第一個引導讀條技能，考慮到法力消耗問題，性價比不如火球術和寒冰箭。

n 魔爆術：法師的第一個AOE技能，在學習暴風雪前非常好用。

n 暴風雪：法師最強AOE技能，搭配“強化暴風雪”天賦，在能打傷害的同時還能減速控制敵人。

n 冰錐術：對面前錐形範圍內造成高額的冰霜傷害，搭配“碎冰”和“強化冰錐術”，清怪效率非常高。

n 烈焰風暴：AOE火焰傷害，同時還會對受到傷害的敵人施加一個火焰dot。

n 炎爆術（火焰天賦）：法師的代表技能，施法時間有點長，但傷害也是鉅額的。推薦在先手或配合變羊術時使用。

n 衝擊波（火焰天賦）：非常強大的瞬發AOE法術，造成傷害的同時減速敵人。

n 灼燒：施法時間非常短，搭配“點燃”和“強化灼燒”天賦非常適合在團本中使用。

Ø 防禦法術

n 霜甲術：法師可用的第一個魔法護甲法術。提高護甲值，能對任何近戰攻擊的敵人進行減速。

n 冰甲術：霜甲術的升級版，具有霜甲術的效果並且還額外提供了冰霜抗性。

n 魔甲術：提供額外的魔法抗性，並且能在任意情況下保持30%的法力回覆。在法師施放法術時都能保持法力回覆。

n 法力護盾：能為法師吸收一定的物理傷害，每吸收1點傷害消耗2點法力值。藍量消耗巨大，酌情使用。

n 寒冰屏障：能為法師提供短暫的無敵效果，冷卻時間過長，可通過“急速冷卻”天賦降低冷卻時間。

n 寒冰護體：吸收一定的傷害，並且在護盾的持續效果內，法師的施法就不會打斷。由於在冰霜天賦最後一層，故只適合冰法。

Ø 輔助技能

n 奧術智慧：為一個目標增加一個智力buff。可通過“奧術光輝寶典”升級為奧術光輝，為整個隊伍提供智力buff。

n 變形術：非常實用的控制類法術。

n 冰霜新星：強大的AOE控制法術。結合暴風雪能對任何近戰敵人造成毀滅性的打擊。

n 緩落術：降低墜落速度，在阿拉希盆地和安其拉非常有用。

n 解除次級詛咒：為目標移除一個詛咒，適合在PVP中應對術士。

n 偵測魔法：可以讓你查看目標敵人身上擁有的BUFF，適用於PVP。

n 魔法增效：提高目標受到的魔法傷害和治療效果。在應對只能造成物理傷害的敵人時非常有用。

n 魔法抑制：與魔法增效的效果相反，降低目標受到的魔法傷害和治療效果。

n 喚醒：8秒內恢復大量的法力值，在長時間的戰鬥中非常有用。

n 閃現術：接觸控制效果和避免墜落傷害，甚至是趕路的絕佳技能。

n 防護火焰結界：吸收火焰傷害，在應對高火焰傷害的敵人時非常有用。

n 防護冰霜傷害：吸收冰霜傷害，在應對高冰霜傷害的敵人時非常有用。

n 法術反制：法師唯一的打斷法術。

n 傳送術：允許法師傳送至陣營主城。

n 傳送門：允許法師創建一個傳送門將隊友傳送至陣營主城。

n 造食術與造水術：能製造出食物和飲料，在練級時能節約買食物和飲料的錢，可通過法師訓練師升級技能，以獲得最好的食物和飲料。

n 製作魔法寶石：能為法師製造能類似法力藥水的法力寶石。

n 氣定神閒：能使下一個施法時間低於10秒的技能變為瞬發，搭配寒冰箭、火球術、炎爆術等法師能提供額外的爆發傷害，或者搭配變形術對敵人進行快速控制。

n 奧術強化：為法師提供額外的法術傷害，但法術消耗的法力值也會提高，最好搭配寒冰箭等法力消耗低的法術。位於奧術天賦樹最後一層。

n 急速冷卻：使用後所有冰霜類法術立即結束冷卻。

n 燃燒：激活後提高所有火焰系法術的爆擊機率。在火焰天賦樹最後一層。

n 暗影抗性:暗影抗性提高10點

n 食屍：半徑5碼內有人型生物或亡靈屍體時可使用，每2秒恢復7%最大生命值，持續10秒，2分鐘CD

八、 法師PVE天賦推薦

由於當前版本的後期怪物中多數具有火抗，所以非常不建議玩火焰系的法師。

1、 奧術冰霜系31/0/20

最適合在地下城和團本中使用的天賦，這套天賦能讓你持續地使用寒冰箭，而不用過度地考慮法力值的問題，並且由於當前版本的熔火之心以及黑石山的一系列副本怪物都具備火抗，因此冰霜法術是最適合的輸出法術。

2、 冰法19/0/32

這套天賦以“深冬之寒”為核心構建，提升所有冰霜法術的暴擊率。總體造成的傷害可能不如上一套天賦，但是此天賦樹下的“寒冰屏障”和“寒冰護體”也為你提供了一定的生存能力，在團本中能減輕治療的壓力。

九、 法師PVE輸出循環

在進入戰鬥前先確認是否已經有“奧術智慧”BUFF，然後根據BOSS的情況確定是否需要為隊友提供“魔法增效”或“魔法抑制”。

Ø 單目標

n 寒冰箭

n 當“深冬之寒”疊到5層後，使用任何能增傷的物品或技能。

n 繼續寒冰箭

n 如果有學習“火焰衝擊”或“魔爆術”的話可以用於殘血收割。

Ø 多目標

n 起手暴風雪

n 當怪物靠近時使用冰霜新星→閃現術拉開距離，或者使用冰錐術。

n 繼續暴風雪

n 用魔杖或“魔爆術”進行殘血收割。

十、 法師PVE屬性優先級

Ø 命中率

n 懷舊服中法術命中率的上限為99%，在我們60級時面對63級團本BOSS時，法術命中率僅為83%。所以我們還需要額外地增加16%的命中率才能保證輸出的最大化。

Ø 法術強度

n 法術強度能為提高法術技能的傷害，但是施放時間越短的技能，法術強度的加成就越低。

n 寒冰箭：額外增加81.4%的法術傷害。

n 火球術：額外增加100%的法術傷害。

n 灼燒：額外增加42.8%的法術傷害。

Ø 爆擊率

n 由於我們的天賦中點出了“寒冰碎片”，所以爆擊率對我們的收益也是非常高的。

Ø 智力

n 法力值可以提高法力值上限和爆擊率。

n 1智力=15點法力值

n 59.5智力=1%爆擊機率

Ø 法術穿透和精神

n 法術穿透可以降低敵人的法術抗性，精神能提供法力值恢復。

n 獲取方式：世界掉落，裝備綁定。

十四、PVP天賦推薦

1、 生存向（17/0/34）

這套天賦偏向於控制對手以及提高自身的生存能力。通過各類凍結效果和法術反制進行控制，依靠碎冰和強化碎冰術進行消耗，寒冰護體、寒冰屏障和奧術活力為法師提供了極強的生存能力。

2、 進攻向（0/24/27）

這套天賦放棄了過多的生存能力，將傷害進行了最大化。冰霜新星+冰錐、衝擊波能造成大量的AOE傷害。炎爆術配合變形術，在1V1的情況下非常厲害。

十六、其它

1、 專業推薦

懷舊服法師PVE最佳專業是工程和裁縫。懷舊服的工程專業能提供不少傷害道具或者控制道具，而裁縫製作的血藤套裝，因為帶有命中等屬性，對法師前期來說非常好用。

2、 武器訓練師

在懷舊服中，法師的初始可用武器僅為法杖和魔杖，需要去武器訓練師訓練才可以使用單手劍和匕首。需要注意的是，魔杖的近戰攻擊非常方便你對低血量的怪物進行補刀，所以一把好的魔杖也是很有用的。
