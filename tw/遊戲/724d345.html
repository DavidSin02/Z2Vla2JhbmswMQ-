古玉熱潮遇上「有鳳來儀」｜中文大學楚文化玉器展

2020-10-29
| 遊戲

近來古玉在市場上愈來愈受歡迎，但收藏鑑賞不能只著眼潮流，一定要以學術為基礎。中大文物館的「有鳳來儀」，正展覽精選158組曾國和楚國玉器，時代屬兩週至戰國中晚期，所有展品均首次在內地以外展出，想學習玉器知識的朋友可別錯過喔。

楚國是西周初年封國之一。經歷代楚君開疆拓土、勵精圖治，躋身「春秋五霸」、「戰國七雄」。盛極時，勢力遍及南部大半個中國。隨之而形成強大而獨特的楚文化，於春秋、戰國之時，不僅與中原文化南北抗衡，影響更及於長江下游之吳越、黃河下游之齊魯，流風所及乃至於漢代，盛行長達八百餘載。

戰國 玉透雕對鳳珮；九連墩墓出土；4.8 x 2.5 x 0.5 cm

展覽上的玉器均在湖北出土，為湖北省博物館所藏，曾國的包括著名的曾侯乙、郭家廟；楚國的包括山灣、姚家港、曹家港、九連墩玉器。九連墩為目前出土玉器數量最多的楚國墓葬，而郭家廟墓地於2015和2016年兩度入選全國十大考古發現。

戰國早期 玉劍；曾侯乙墓出土；33.6 x 5.1 x 0.5 cm

是次展覽正以戰國中晚期九連墩玉器為中心，旨在探討和展示楚文化玉器風格特徵及其形成過程，以及與中原風格的曾國玉器之關係。展覽首次依據展品的出土位置進行展陳及組合，有助參觀人士瞭解楚國葬玉習俗及玉器的功能。

戰國 玉龍鳳合體珮；九連墩墓出土；6.8 x 2.9 x 0.3 cm

戰國 玉三人立豕珮；九連墩墓出土；5.1 x 3 x 0.4 cm

此外，為配合展覽，中大文物館亦特意出版導覽手冊，並將透過藝術體驗工作坊，以劇場導賞結合藝術創作的形式，為公眾帶來嶄新參觀體驗，加深參加者對楚國文化玉器的認識。所有活動費用全免，唯名額有限，先到先得，額滿即止。
