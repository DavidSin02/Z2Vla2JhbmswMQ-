Nike Epic React Flyknit 全新灰藍配色

2020-10-29
| 遊戲

﻿除了黑藍配色外，Nike 革新跑鞋 Epic React Flyknit 還將迎來這雙灰藍版本，特別值得一提的是，該鞋款要比原定的首發上架日期提前半個多月於 Nike+ 率先發售。設計上，靈感取自經典 Flyknit Racer 的鞋身採用淺灰色 Flyknit 打造，鞋面多處細節以白色低調呼應，並在後跟與鞋墊上融入深藍色製造對比，最後再搭載由一整塊泡棉構成的 Nike React 舒適大底。這雙灰藍配色定價為 $150 美元，將於 2 月 5 日正式登陸 Nike+，敬請期待！
