8090年代中國大陸引進的美劇（美國電視劇）（原創）

2020-11-14
| 遊戲

1. 《大西洋底來的人》，1977年上映，1980年1月在央視播出，共17集。

中國大陸引進的第一部美劇。

主演: 帕特里克·杜菲 / 阿蘭·弗吉 / 貝琳達·蒙哥瑪麗 / 瓊·瑪麗·霍恩 / 維克多·布魯諾

講述了一個神祕的晚上，海底巨浪把奇異的生物麥克·哈里斯送到岸上。當醫學界視之為死亡而無能為力時，海洋學家伊莉沙白·瑪麗博士把他放回海洋，才使他得以復活。麥克接受了一連串的試驗，以測驗他的速度、靈敏度和力量。

2. 《加里森敢死隊》,1967年上映，1980年10月在央視播出，共26集。

中國大陸引進的第二部美劇。央視播了16集後被禁播。

主演: 羅恩·哈珀 / 西薩爾·達諾瓦 / 克里斯托弗·凱瑞/魯迪·索拉里/布蘭登·波恩

劇中講述了一群來自美國的失足青年，他們行為放蕩不羈，身世惡貫滿盈，卻不斷建立功勳，成為英雄的故事。

3. 《240拯救隊》，1979年上映，1980年廣東電視臺引進播出，共13集。

主演: 約翰·貝內特·派瑞 / 馬克·哈蒙 / 喬安娜·卡西迪 / 帕梅拉·亨斯利 / 簡妮麗·哈里森

應洛杉磯警長緊急服務部的要求。 該團隊利用直升機，船隻和定製的福特野馬，設法從各種危險情況中救助不幸的公民。

4. 《特警隊》，1975年上映，80年代由廣東電視臺引進播出，共12集。

主演: 史蒂夫·福雷斯特/羅伯特·烏里奇/羅德·佩裡

該系列的特色是虛構的W.C.P.D.特種武器與戰術小組的任務。 他們是一支訓練有素，全副武裝的警務人員隊伍，其目的是在敏感局勢和可防禦地點，對武裝和危險罪犯進行協同攻擊。

5.《火星叔叔馬丁》，1963年上映，共37集。

主演: 雷·沃爾斯頓 / 比爾·貝克比 / 帕梅拉·布里頓

主要講述來自火星的Exodus因事故落到地球上，此後跟記者蒂姆·歐哈利一家同住。

6.《糊塗偵探》，1965年上映，共30集。

主演: 唐·亞當斯 / 芭芭拉·費爾頓 / 愛德華·普拉特 / 理查德·高蒂埃 / 布特·穆斯汀

麥克斯維是美國特工局的86號特工，與邪惡的混沌組織做著不懈的鬥爭。使用特種部隊的“獨特”武器，與他的搭檔-99號特工一次次的擊碎了混沌組織毀滅世界的陰謀。

此劇有個萬物皆是電話的橋段：皮鞋、皮帶、領帶扣、鋼筆、花瓶、燭臺、門把手。。。

7.《時間隧道》，1966年上映，1990年在央視播出，共30集。央視只播出了18集。

主演: 詹姆斯·達倫 / 羅伯特·科爾伯特 / 李·麥瑞威瑟

科學家託尼·紐曼和道格·菲利普斯他們發明了時間隧道，結果一次意外被陷入了時間流中，並出現在著名的不同歷史時期。

8.《不可能完成的任務》，1966年上映，共28集。

主演: 格雷格·莫里斯/彼得·盧普斯/彼得·格雷夫斯/芭芭拉·貝恩/馬丁·蘭道

這是一個由政府間諜和專家組成的團隊，他們專門負責解決最高難度的機密事件，不僅任務給的機密，而且成敗自行負責，政府完全不承認有這個組織和任務。

9.《星際旅行》，1966年上映，共29集。

主演: 倫納德·尼莫伊 / 威廉·夏特納 / 德福雷斯特·凱利 / 尼切爾·尼克斯 / 詹姆斯·杜漢

它是一個描述一艘隸屬於星際聯邦的星艦企業號上的航員以及他們“勇敢地航向前人所未至的領域”的冒險故事。

10.《代號“耶裡哥” 》，1966年上映，1994年上海電視臺引進播出，共16集。

主演: 唐·弗蘭克斯/ 約翰·萊頓

講述的是二戰時期代號“耶裡哥”的三人小組滲透到納粹佔領的敵人後方從事破壞和情報竊取活動。

11.《草原小屋》，1974年上映，1988年上海電視臺引進播出，共24集。

主演: 邁克爾·蘭登 / 凱倫·格拉斯爾 / 梅麗莎·吉爾伯特 / 林賽·格林布什

描寫主人公勞拉在威斯康星的小木屋時，到她在達科他地區結婚所渡過的美好時光，作者以充滿詩意的筆調描述了他們純樸的生活，善良的人性，平凡而動人，吸引了成千上萬的讀者。

12.《破繭飛龍》，1974年上映，上海電視臺引進播出，共22集。

主演：詹姆斯·加納/小諾亞·比裡/喬·桑托斯

關於一個名叫吉姆·羅克福德的前犯罪嫌疑人轉為私人偵探的故事。他寧願逃跑也不願打架，寧願釣魚也不願工作。 在他的案件中，洛克福德總是得到朋友丹尼斯·貝克爾，安吉爾和貝絲·達文波特的協助。

13.《神力女超人》，1975年上映，上海電視臺引進播出，共14集。

神奇女俠電視版，在劇中戴著一副特大的眼鏡在國防部任職，亳不顯眼，但當國家遇上危急時，她即變成性感的神奇女俠拯救人民。總是穿著一件星旗低胸裝在街上奔跑，蔚為奇觀。

14.《方舟二號》，1976年上映，共15集。

主演: 特里·李斯特 / 亞當 /喬斯·弗洛雷斯

故事描述的是未來的地球由於汙染等原因，人類的文明已經退化，一群科學家建造了一輛高科技的旅行車，環遊世界，試圖恢復文明。

15.《查理的天使》，1976年上映，1990年上海電視臺引進播出，共22集。

主演: 大衛·道耶 / 傑奎琳·史密斯 / 約翰·福賽思

《霹靂嬌娃》翻拍自該電視劇，該劇講述的是三個女人為一個虛構的私人調查中心工作，她們的老闆名叫查理，是一個從來不在影片中現身的角色，僅用電話與她們聯絡。

16.《根》，1977年上映，上海電視臺引進播出，共8集。

主演: 瑪雅·安吉洛 / 吉-杜·根伯卡 / 摩西·岡恩

改編自黑人作家亞歷克斯·哈利所寫的同名小說，跨度幾代人講述了他們從被販賣到尋精神之根的長達半個多世紀的黑人種族血淚史。

17.《綠巨人》，1978年上映，共12集。

主演: 盧·費裡諾 / 比爾·貝克比 / 傑克·科爾文

布魯斯·班納的父親在剛出生的班納身上進行基因改造試驗。多年後,班納在一次試驗意外中受到了伽馬射線的強力輻射，布魯斯體內的神祕力量被意外喚醒。從此之後，每當情緒激動時，布魯斯就會失去自我意識，變身成為綠巨人，並且具有超強的破壞力。

18.《帕爾斯警長》，1979年上映，1986年在央視播出，共13集。

主演: 詹姆斯·厄爾·瓊斯 / 李·張伯林 / 漢克·蓋瑞特 / 邁克爾·沃倫 / 喬格·斯坦福·布朗

伍迪·帕爾斯在加利福尼亞南部某地經營著一個大城市警察局。他指揮一個由四名特立獨行的年輕偵探組成的小隊，與妻子巴巴拉過著幸福的家庭生活。

19.《格洛麗亞》，1980年上映，1986年上海電視臺引進播出，共4集。

主演: 吉娜·羅蘭茲 / 巴克·亨利 / 朱莉·卡門 / 約翰·亞當斯 / 約翰·芬尼根

當一個小男孩的家庭被暴徒殺死時，他們的鄰居格洛里亞很不情願的成為了他的監護人。 兩人擁有黑幫想要的一本書， 隨後暴徒在整個城市追逐他們。

20.《霓裳戀曲》，1980年上映，上海電視臺引進播出，共3集。

主演: 林賽·瓦格納 / 巴瑞·波斯威克 / 瑪麗-弗朗絲·皮西爾 / 金·凱特羅爾

故事講述了億萬富翁遺孀比利、法國準時裝設計師瓦倫丁、停車場服務員/攝影師斯拜德，三個雄心勃勃的年輕人的命運。

21.《鷹冠莊園》，1981年上映，1988年上海電視臺引進播出，共18集。

主演: 洛倫佐·拉馬斯 / 冀朝理 / 簡·懷曼 / 大衛·塞爾比 / 蘇珊·沙利文

以加利福尼亞的葡萄園為背景，展現了強大的Gioberti家族之間的衝突，該家族是龐大的Falcon Crest酒莊的所有者。

22.《豪門恩怨》，1981年上映，1991年上海電視臺引進播出，共13集。

主演: 約翰·福賽思 / 琳達·伊萬斯 / 瓊·柯琳斯 / 邁克爾·納德/ 傑克·科爾曼

一個富裕的丹佛家族在石油行業的傳奇。大部分劇情都以卡林頓家族和科爾比家族這兩個有爭議的家族之間的衝突為特色。

23.《霹靂遊俠》，1982年上映，1995年在央視播出，共80集。

主演: 大衛·哈塞爾霍夫 / 理查德·貝斯哈特 / 麗貝卡·霍爾登 / 愛德華·馬哈爾

故事講述了Michael Knight駕駛著具有高度人工智能的跑車KITT，在罪犯橫行於法律之上的世界裡支持那些無辜及無助的人。

24.《斯蒂爾傳奇》，1982年上映，共22集。

主演: 皮爾斯·布魯斯南 / 斯蒂芬尼·齊姆巴麗斯特 / 詹姆斯·瑞德

斯蒂爾是個瀟灑，智慧而且浪漫的偵探。這位神祕的偵探總是和他的助手勞拉在不經意的情況下偵破了一件又一件的案子。而事實上，偵探社的真正操盤手正是勞拉。因為人們總是不不夠信任一個女偵探。

25.《曼特·豪斯頓》，1982年上映，1991年上海電視臺引進播出，共67集。

主演: 李·霍斯利 / 帕梅拉·亨斯利 / 巴迪·艾布森

曼特·豪斯頓是一位富有的德克薩斯州人，他搬到加利福尼亞來監督他的家人的海上鑽井企業，但大部分時間都花在自己的私人調查員愛好上。 這些案件通常涉及謀殺案，利用他多年的經驗和調查技巧，在他的助手C.J.的幫助下，休斯頓始終設法將壞人繩之以法。

26.《荊棘鳥》，1983年上映，上海電視臺引進播出，共4集。

主演: 理查德·查伯蘭 / 蕾切爾·沃德 / 克里斯托弗·普盧默 / 布萊恩·布朗 / 布萊特·卡倫

以早期澳大利亞移民內陸開發為背景。神父拉爾夫與新西蘭移民家庭克利裡家族交好，家中的小女兒麥琪從少女時期起便愛上了他，神父與麥琪相愛卻不能相守。

27.《俠骨柔情》，1983年上映，上海電視臺引進播出，共21集。

主演: 凱特·傑克遜 / 布魯斯·巴克林納 / 貝弗利·加蘭 / 瑪莎·史密斯 / 阿爾·羅斯西奧

特工Stetson Lee在一次躲避敵方追捕的途中將證物塞給了正巧路過的家庭主婦Amanda King。在擺脫追蹤後，Lee找到了King太太要回證物，哪知King太太玩上了癮，纏著他要幫助他們破案。迫於無奈，Lee只好答應了她的“無理”要求，兩個人開始以奇怪的組合進行合作。

28.《大飯店》，1983年上映，1990年上海電視臺引進播出，共23集。

主演:詹姆斯·布洛林 / 康妮·賽勒卡 / 莎莉·貝拉方特

是一部關於加州舊金山著名的聖格雷戈裡酒店管理層、員工和客人生活的電視劇。

29.《天龍特攻隊》，1983年上映，共13集。

主演: 喬治·佩帕德 / 德懷特·舒爾茨 / 勞倫斯·特勞德 / 德克·本內迪克特 / 梅琳達·庫萊亞

在越南服役結束後，約翰·漢尼拔·史密斯上校和他的團隊因搶劫河內銀行（他們是在命令下搶劫的）而被陷害，並被送進一所美軍監獄。他們越獄之後成為僱傭的英雄，在美國或世界上做好人警衛。

30.《天使的憤怒》，1983年上映，共3集。

主演: 傑克琳·史密斯 /肯·霍華德/阿曼·阿桑特

一位無情的律師（莫雷爾）使用一位年輕的助理地方檢察官（帕克）來解救他的客戶，因此被解僱，幾乎被取消律師資格，但她反抗成為頂級律師。

31.《飛狼》，1984年上映，上海電視臺引進播出，共11集。

主演: 簡·邁克爾·文森特 /艾力克斯·寇德 /歐內斯特·博格寧

斯特林費羅·霍克與多米尼克·桑蒂尼將乘坐先進的戰鬥直升機“飛狼”執行任務。飛狼直升機全身黑色的，性能極其凶悍，可以說是刀槍不入。

32.《神探亨特》，1984年上映，1992年上海電視臺引進播出，共153集。

主演: 弗裡德·杜萊爾 / 斯蒂芬妮·克拉默 / 查爾斯·哈拉漢

影片圍繞足智多謀的洛杉磯警探瑞克·亨特和他的默契女搭檔迪迪麥考爾展開，講述著洛杉磯警探們捍衛城市安全的故事。

33.《鑽石王朝》，1984年上映，上海電視臺引進播出，共4集。

主演: 安格哈拉德·瑞絲 / 伊安·查裡森 / 切瑞·朗吉

描述了一位淘金者的女兒的傳奇故事，幼年的經歷即讓她懂得權力的重要。她有足夠的精神，紛亂的戰爭年代成為她發跡的最佳時機；她有足夠的野心，步步為營直至締造龐大的鑽石王國。在權力的棋盤上，每一個人都是她手中的棋子，無論是她的情人還是她的兒子……

34.《埃利斯島》，1984年上映，1987年上海電視臺引進播出，共7集。

主演: 彼得·裡格特 /格雷戈裡·保羅·馬丁 / 克萊爾·布魯姆 / 瓊·格林伍德 / 艾麗絲·克里奇

為了尋求美國夢，19世紀抵達埃利斯島（埃利斯島曾是美國主要的移民檢查站）的無數移民中的一些人的生活和掙扎。

35.《成長的煩惱》，1985年上映，1990年上海電視臺引進播出，共22集。

主演: 艾倫•錫克 / 喬安娜•柯恩斯 / 柯克•卡梅隆/特蕾茜·格爾德/傑瑞米·米勒

主要講述了一個普通的美國家庭西佛一家的日常生活。

36.《家族的榮譽》，1985年上映，上海電視臺引進播出，共13集。

主演: 埃利·瓦拉赫/肯尼斯·麥克米蘭/邁克爾·馬德森/湯姆·梅森/邁克爾·伍德

從小一起長大的兩個朋友，走上了不同的道路，一個成為了警察，一個加入了黑手黨。之後兩家人分別加入警隊與黑幫，形成了敵對的警察世家與黑幫世家。

37.《浮華世家》，1985年上映，上海電視臺引進播出，共49集。

主演: 查爾頓·赫斯頓 / 約翰·詹姆斯 / 凱瑟琳·羅斯 / 愛瑪·薩姆斯 / 麥克西維爾·考菲德

《豪門恩怨》的衍生作品，以傑夫·科爾比和法倫·科爾比在加利福尼亞的冒險為特色。傑夫·科爾比接受叔叔傑森·科爾比提出的要為家族企業（ColbyCo）工作的提議。

38.《電腦娃娃》，1985年上映，1991年在大陸播出，共24集。

主演: 蒂芙尼·布里塞特 / 迪克·克里斯帝 / 瑪拉·彭寧頓 /傑裡·蘇普蘭
