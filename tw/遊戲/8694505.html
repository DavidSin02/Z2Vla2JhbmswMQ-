《精靈寶可夢 究極之日/月》新晉訓練師指南

2020-10-29
| 遊戲

《精靈寶可夢 究極之日/月》攻略索引：

御三家

遊戲中玩家最早能夠獲得三隻寶可夢被俗稱為御三家，這三隻寶可夢必定是草、水、火三種屬性，而玩家只能在其中選擇一隻。按照以往慣例，這三種寶可夢是不可能在野外捕捉到的，想要再獲得只能使用通信傳送的方式讓其他玩家送給你。

《精靈寶可夢 太陽/月亮》中的御三家分別是：

草系，木木梟→投羽梟→狙射樹梟

初始技能為“樹葉”，屬性為草/飛行，特性為茂盛；最終進化屬性為草/幽靈，特別技能為“縫影”，受到這招攻擊的對手會無法逃走或進行替換。

種族值：HP：78 / 攻擊：107 / 防禦：75 / 特攻：100 / 特防：100 / 速度：70

  • 特性：茂盛，在體力不支時草系招式攻擊增強。
  • 隱藏特性：使用近身招式不會受到自身或目標特性影響。

火系，火斑喵→炎熱喵→熾焰咆哮虎

初始技能為“火花”，屬性為火，特性為猛火。最終進化屬性為火/惡，特別技能為“ＤＤ金勾臂”，可在不受到對手能力變化影響的情況下給予對手傷害。

種族值：HP：95 / 攻擊：115 / 防禦：90 / 特攻：80 / 特防：90 / 速度：60

  • 特性：猛火，在體力不支時火系招式攻擊增強。
  • 隱藏特性：出場時所有攻擊範圍內的對手攻擊下降一級。

水系，球球海獅→花漾海獅→西獅海壬

初始技能為“水槍”，屬性為水，特性為激流。最終進化屬性為水/妖精，特別技能為“泡影的詠歎調”，受到這招攻擊的對手身上的「灼傷」狀態將會痊癒。

種族值：HP：80 / 攻擊：74 / 防禦：74 / 特攻：126 / 特防：116 / 速度：60

  • 特性：激流，在體力不支時水系招式攻擊增強，
  • 隱藏特性：所有以聲音為基礎的招式全部轉換為水屬性攻擊。

捕捉寶可夢

在選好了自己的御三家開始冒險之後，玩家就需要開始捕捉野生的寶可夢來壯大的自己的隊伍，完成自己的圖鑑。捕捉寶可夢將會需要使用精靈球，而精靈球只能對野生的寶可夢使用。也就是說，你無法捕捉已經有主人的寶可夢。

捕捉寶可夢講究的就是三個字：快、狠、準……不對，是血量低、有異常、用好球。

所謂血量低，就是儘可能的把野生的寶可夢血量控制在一個較低的水平；有異常，就是為野生的寶可夢施加各種異常狀態，比如麻痺，睡眠，混亂等等；用好球，是因為精靈球有很多種，通常越好的精靈球捕捉的概率越高，當然也有一些比較特別的精靈球會針對某些特別種類的寶可夢有加成。

以上幾種方法只是能夠提高捕捉到寶可夢的概率，並非絕對能夠捕獲。除非使用獨一無二的大師球，否則寶可夢都有一定概率從精靈球中掙脫出來。

最後，在精靈球晃動的過程中，你可以：連按A/B鍵，狂搖主機，猛點屏幕，跪地祈禱等等，但以上行為均不會有任何加成，恩。

寶可夢進化

進化，是《精靈寶可夢》的一個核心繫統。大多數寶可夢在進化之後會改變樣子，變強，甚至還會改變屬性和特性。而遊戲中寶可夢的進化方式也是異常的多樣：

等級進化

最常見的進化方式，只要寶可夢達到一定的等級之後就可以進化。不同的寶可夢進化等級各不相同，要求最低的綠毛蟲只需要7級就可以進化；而雙首暴龍則需要在64級才會進化成為三首惡龍。

道具進化

道具進化最常見的就是使用進化石。比如皮卡丘在正常情況下是不會進化的，只有對它使用雷之石才會進化成為雷丘，而伊布則需要使用火之石、雷之石、水之石才會分別進化成為火伊布、雷伊布、水伊布。

通信進化

部分寶可夢只會在兩個玩家之間交換寶可夢時候進化，比如鬼斯通→耿鬼、勇吉拉→胡地、豪力→怪力。值得一提的是，有些需要通信進化的寶可夢還需要攜帶特定的道具，比如呆呆獸→呆呆王，就需要在通信的時候讓呆呆獸攜帶王者之證才會進化。

親密度進化

這也是比較常見進化方式。這部分寶可夢只有在和玩家的親密度達到一定程度，然後在升級的時候才會進化。比如皮丘→皮卡丘。

其他進化

除了上面介紹的幾種之外，還有很多其他進化方式，但這一類寶可夢的數量非常少。比如伊布→太陽/月亮，是根據進化時間決定的；笨笨魚→美納斯，是根據美麗度決定的；黏美兒→黏美龍，要在下雨時候才會進化。最奇怪的大概還要屬好啦魷→烏賊王，需要在升級時把 3DS 主機倒過來才會進化……

屬性相剋

在遊戲中，不同的寶可夢，不同的招式都會有不同的屬性，而屬性之間的相性則會影響每次攻擊的傷害結算。下面的圖片列出了不同屬性之間的相互影響。需要注意的是，表格中只包含了一組屬性的影響，而非多重屬性。具有多重屬性的寶可夢則需要選擇攻擊方的招式的屬性和防禦方的兩個屬性對應的倍數相乘。

比如，飛行屬性的招式會對蟲、草屬性神奇寶貝造成2*2=4倍傷害，而若是地面屬性招式則只有0.5*0.5=0.25倍傷害。

不過在本作中，玩家如果對戰此前已經相遇過的寶可夢，那麼招式的效果是會直接顯示在旁邊的，其描述分別對應：

沒有效果：0x

效果不好：0.5x~1x

有效果：1x~1.5x

效果絕佳：1.5x~4x

所以即便是記不住相性表，也可以通過招式上的描述對效果進行大致判斷。

寶可夢的能力值

每隻寶可夢都有六個能力值，也稱六圍，分別是HP、攻擊、防禦、特攻、特防、速度，下面解釋這六個屬性各自的作用。

  • HP：即寶可夢的血量
  • 攻擊：影響物理攻擊招式的傷害
  • 防禦：減少受到的物理攻擊傷害
  • 特攻：影響特殊攻擊招式的傷害
  • 特防：減少受到的特殊攻擊傷害
  • 速度：速度高的寶可夢優先出手

招式的具體分類可以在招式詳情中查看，每個招式都會用上圖中的圖標表示類型。注意：極少部分招式將會以進攻方的特攻和防禦方的防禦力計算傷害。

分別對應物理、特殊、變化招式

比如逆鱗，屬性為龍，物理攻擊

速度的結算非常精確，哪怕是隻高出一點，也會比對方先出手。但是，部分招式優先度將會高於速度的計算，比如電光火石。也就是說，即便是對方速度更高，但使用技能優先度更高的一方將會先出手。此外，部分攜帶道具也會改變出招順序。

異常狀態

《精靈寶可夢 太陽/月亮》依然只有五個異常狀態，分別是中毒、灼傷、冰凍、麻痺、睡眠。中了這些異常狀態的寶可夢戰鬥時會在名字旁邊寫上異常狀態的名稱，中毒與猛毒都寫作中毒，但顏色深淺不同。異常狀態無法疊加，也不會相互抵消或替換，戰鬥結束後異常狀態也不會消失。

雖然名為異常狀態，但對寶可夢來說，並非所有的異常狀態都是負面的，也有能夠從異常狀態中獲得增益效果的寶可夢。而根據自己的需要搭配，也有主動讓自己陷入異常狀態的戰術。此外，異常狀態還會受到寶可夢的特性、天氣、環境、招式和道具的影響。

灼傷

灼傷狀態下的寶可夢每回合會按最大HP的百分比扣除體力，同時物理攻擊招式的威力減半。火屬性的寶可夢不會進入灼傷狀態。

冰凍

冰凍狀態下的寶可夢無法使用任何招式。冰屬性的寶可夢不會進入冰凍狀態，在被火屬性的招式攻擊後，冰凍狀態會解除。同時，每回合開始的時候都有25%左右的概率自動解除冰凍狀態。

麻痺

麻痺狀態下的寶可夢每回合有25%的概率無法使用招式，同時速度下降50%。電屬性的寶可夢不會進入麻痺狀態。

中毒

中毒狀態下的寶可夢每回合會按最大HP的1/8扣除體力，猛毒第一回合則為1/16的體力，隨後每回合加倍。原本鋼系和毒系寶可夢是不會中毒的，但由於“腐蝕”特性的引入，理論上所有寶可夢都會進入中毒狀態。

睡眠

睡眠狀態下的寶可夢無法使用除了“打鼾”和“夢話”之外的任何招式，睡眠狀態在2~4回合內隨機解除。除了被對手的招式催眠之外，寶可夢也有讓自己主動睡眠的招式。自己主動睡眠的寶可夢將會恢復全部HP，清除當前所有異常狀態，然後必定會在第三回合醒過來。

能力升降

在對戰中，大家經常會看到XXXXX攻擊力上升了！XXXX特防大幅上升了等等，這就是戰鬥中的能力升降變化。

戰鬥中所有能力變化都分為十二檔，上升和下降各六檔，普通狀態下所有寶可夢都默認處於普通狀態，也就是0，此時所有能力都為100%，不同檔位具體數據如下：

需要注意的是，當攻擊出現“擊中要害”的情況時，傷害結算會無視所有的不利條件，即攻擊方的攻擊和特攻如果小於0檔，則按照0檔計算，同時防守一方的防禦和特防如果大於0檔，則按照0檔計算。

狀態變化

狀態變化是區別於上面提到的異常狀態的。首先，狀態變化可以疊加；其次，大部分都可以自愈；最後，戰鬥結束時狀態變化自動消失。狀態變化單場戰鬥內有效的增益或負面效果，同時還有一些狀態變化可以對場地施加，他們會改變一些對戰時候的規則。比如，對戰場地中存在戲法空間時，速度慢的一方反而會先出手，處於滅亡之歌狀態的寶可夢，三回合之後會直接進入瀕死狀態。

白霧，就是對場地施加了狀態變化

寶可夢的性格

寶可夢的性格將會影響自己的能力值和口味，其中一項的能力值增加10%，另一項則減少10%。比如“勇敢”的性格，會提升10%的攻擊，降低10%的速度，喜歡辣、討厭澀。另外還有5種性格不會影響神奇寶貝的能力值。除了這5種之外，其它20種性格所增加和減少的能力值組合均不相同。

寶可夢的特性

一隻寶可夢可能有一到兩種特性，通常在捕捉（出生）時就決定了，部分寶可夢在進化後特性會改變，玩家可以在寶可夢的詳情中查看每隻寶可夢的特性。寶可夢大部分特性都是有益的，少部分特性會有負面效果。同時，部分寶可夢將會有不顯示出來的隱藏特性。

這隻皮卡丘的特性是靜電

Z招式

Z 招式是《精靈寶可夢 太陽/月亮》新加入的系統，屬於特別強力的招式、在一次對戰中只能使用一次。

要使用 Z 招式，寶可夢需要攜帶對應屬性的 Z 水晶。在道具欄中攜帶 Z 水晶的時候，系統會告知玩家該水晶對寶可夢是否有效，所以不同擔心帶錯。值得一提的是，部分寶可夢擁有自己的特別Z水晶，比如御三家寶可夢，皮卡丘和卡比獸等等。此外，以不同的招式為基礎使用 Z 招式的時候，效果與威力也不相同。

攻擊招式→攻擊 Z 招式，屬於將攻擊大幅度強化的Ｚ招式，即便使用“守住”或“看穿”等技能也無法免傷，部分攻擊招式甚至還會追加新的效果。

變化招式→變化 Z 招式，變化招式會藉由Ｚ力量得到增強，已有的效果將會得到強化，同時也有可能會追加新的效果。

遊戲中大部分 Z 水晶都是從各個島嶼的隊長和島主哪裡獲得的，少部分 Z 水晶會隱藏在地圖中。
