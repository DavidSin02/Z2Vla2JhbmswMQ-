起落架基本原理（減震器設計）


基本介紹


減震器的基本功能是吸收和消散衝擊動能，以使施加在機體身上的加速度降低到可接受的水平。根據所使用彈簧的類型，現有的減震器可分為兩類：使用鋼或橡膠製成的固體彈簧的減震器，以及使用含氣體或油的流體彈簧的減震器，或通常稱為油氣減震器。油-氣混合減震器具有較高的齒輪效率和重力效率，使其成為商業運輸的首選設計。


根據本節概述的分析程序，開發了算法來確定滿足給定設計條件所需的衝程和活塞長度，以及減震器的能量吸收能力。在計算著陸過載（起落架過載）時，要將減震支柱和輪胎壓縮都考慮進去，因而本文中有時候會將他們作為一個整體進行考量。


油氣減震支柱設計


油氣減震器的基本重量支撐功能，在動態條件下在能量吸收和耗散方面都具有很高的效率，它是由壓縮的空氣和油缸提供的。



單氣室減震器是商業運輸中最常用的設計，B737飛機使用的是單氣室減震器，而A320飛機使用的是雙氣室減震器（下圖所示），結構更為複雜，壓縮行程相對來說也更長，但是原理是一樣的。


這種類型的減震支柱通過推動油室和氣室（乾燥的空氣或氮），然後壓縮氣體和油來吸收能量。在壓縮過程中，根據設計的類型，油氣會保持分離狀態或混合狀態。初始衝擊後，氣壓使油通過反衝孔口回油，從而耗散了能量。


儘管壓縮孔口可能只是孔板上的一個孔，但大多數設計都具有一個穿過它的計量針（也叫做調節油針），並且通過改變針的直徑，可以改變孔的面積。調整此變化，以使支撐杆載荷在動態載荷下相當恆定。如果將其設為恆定，則齒輪效率將為100％。實際上，這是永遠無法獲得的，效率通常為80％至90％。由於在概念設計階段僅關注效率因素，不再過多贅述有關計量針設計的其他信息。

行程計算


計算行程（S）的第一步是選擇設計反作用因子（N），被稱為著陸過載（landing load factor），該過載不應與由機動或大氣干擾引起的飛行過載混淆。對於運輸類飛機，著陸過載在0.7到1.5之間變化，其中應用較為廣泛的值為1.2。



下沉速度（Vs）通常由採購機構和/或與特定類別飛機有關的法規來立法。FAA要求運輸類飛機在設計著陸重量下必須能夠承受10 ft/s的著陸衝擊，而在設計起飛重量下則能夠承受6 ft/s的著陸衝擊。實際上，由於地面效應和接地前飛機的拉平，很少發生如此大的下沉速度。

載荷和行程曲線


為了吸收在嚴重或半碰撞著陸時產生的多餘能量，減震器的設計使得即使在壓縮位置，活塞也不會完全觸底。保留的空氣量（假定為排量的10％）使減震支柱在預定的負載下可以通過額外的行程，從而通過做功吸收多餘的能量。


減震支柱在其行程期間通過對載荷-行程曲線下方的面積進行積分而得到吸收的能量，該曲線將施加的地面載荷的大小與所經過的行程相關聯。

圖｜載荷行程曲線


大多數飛機使用所謂的油氣緩衝器，其特性曲線如上圖所示。它由一個空氣彈簧和與之相連的油緩衝器組成。空氣彈簧吸收能量，阻尼器耗散能量，兩個過程同時發生。另一種緩衝器是液體彈簧，它通過承受高壓的油液吸收和耗散能量。


當空氣彈簧緩慢加載時，載荷-行程曲線遵循恆溫氣體定律，但是當快速加載時，其遵循氣體多變定律（polytropic gas law），曲線典型的幕次約為1.3 (絕熱條件下幕次為1.4)。這兩種情況載荷-行程曲線下方區域面積遠低於線彈簧的50%，但考慮油液的阻力後，曲線形狀將發生明顯變化。


起落架特性


飛機著陸是一種典型的快速加載狀態，此時，載荷-行程曲線有可能首先陡然上升，隨後變平，曲線下方區域面積A，與最大載荷F與最大行程S的乘積的比值，稱為效率η。η＝A/(F*S)。其典型值為85%。但一般而言，是給定情況下變形量的函數，液體彈簧效率相對較低，最大值一般為80% 。


緩衝器高效吸收撞擊能量存在一個缺點：小變形會引起大載荷，即不滿足小能量吸收要求。從疲勞的角度而言，這種情況顯然不利，可以通過使用雙氣室減震支柱加以解決（A320飛機使用的就是雙氣室減震支柱）。雙氣室減震支柱的特性曲線如上右圖所示。


特性曲線初步設計階段“平緩”的特性滿足小能量吸收不會引起大載荷的要求，比如滑跑時。其缺點則是總體效率較低，在設計能量吸收情況下可能只有60%，這意味著緩衝器需要更大的變形，即更長的行程。


相對飛機重心運動方向情況而言，實際絕大部分吸收的能量來自垂直方向，因此，適宜用垂向載荷和軸向行程來描述飛機起落架緩衝器的特性。實際緩衝器的特性可以通過其與豎直方向的夾角或將其併入槓桿機構中加以修正。

接地下降率200ft/min

先將垂直方向速度（即下降率）換成米制：

v = 200 ft/min = 1.016 m/s

查載荷行程曲線：

接地下降率0ft/min


此時飛機的情況又如何呢？如果飛行員在前期杆量“柔和一致”，ny保持在稍大於1g的樣子，得到了一個自己夢想的“kiss landing”，我們當然要恭喜他。然而，針對這種狀態下實際飛行數據的分析顯示——大量體感過輕的落地都有類似的情況——升力會小於重力，原因不外乎以下幾種情況：

  • 由於前期拉桿量較大，對於“拉飄”的本能反應，飛行員鬆杆，飛機姿態可能降低，從而導致升力減小；
  • 由於“平飄”時間較長，速度降低，而姿態沒有及時增加（飛行員停止拉桿），導致升力減小；
  • 由於Load <15>報中對應的VRTA值為接地前後5秒的最大值，而飛機的擾流板正常情況下會在探測到接地0.5秒左右伸出，導致升力卸載；


上面幾種升力損失的情況，如果發生在飛機接地前，後果比在飛機接地時更壞——下降率增大（觸地動能增大）。尤其是最後一種情況，擾流板放出導致的升力卸載，致使ny小於1g（不彈跳時的接地，該值通常會在0.7-0.8左右，有時會更小一點），此時就會有些“得不償失”，甚至可以說風險是比較大的——參考一起沒有談到飛行員作風問題的重著陸事件和著陸技術｜拉還是不拉？這是個問題？著陸時彈跳升力卸載後再次接地的“慘狀”。

舉個例子，假使此刻ny＝0.6g，即升力損失了40%，這時下降率為0，飛機正好接地。解方程（此時沒有動能，所以列算式跟上面兩種情況正好相反）：

得到行程d：

d = 0.42 (米)

對應的地面載荷：

Fs = F(0.42) = 493886 (牛)

此時，著陸過載：

N = Fs ／ mg = 0.84 g

則：

VRTG = ny + N = 0.6 + 0.84 = 1.44 g


故在這種情況下，Load <15>報打印出來的VRTA或者QAR在接地後的壓縮過程中記錄的VRTG值將為1.44g左右。當然這只是理論數值，實際情況呢？只要升力卸載沒這麼嚴重，結果可能比以上理論計算情況要好；但也有可能比這個更糟糕，原因可能是以上提及的幾種升力損失原因同時發生。



上圖就是一個典型的升力卸載導致著陸時過載大的例子，下面我們就來解讀一下——需要說明的是，VRTG數據正如以前推送的那樣是1/8秒記錄一次數值。

在6:41:21″375，飛機姿態達到峰值4.31°，隨後開始減小（掉機頭）——猜測此時機組可能判斷飛機已經接地了。雖然隨後飛行員有向後拉桿動作，但姿態再也沒有增加過，且為逐漸降低趨勢直至前輪接地滑跑。

在6:41:22″000，飛機右主輪接地（近地電門信號），可以看到QAR中記錄的基於RA計算的下降率已經變成0，此時飛機姿態4.22°，左主輪在0.5秒後接地。


在6:41:23″000，擾流板開始部分放出。又經過0.5秒，開始全部放出（偏轉超過10°），隨後經過大概1秒後達到完全放出狀態。擾流板在開始全部放出時，記錄的VRTG值為0.772g——此時仍可判定為飛行過載ny，代表發生升力卸載情況，飛機有向下的加速度。然而因為此時飛機主起落架已經接地，開始產生支承力。由是擾流板偏轉角度更大時，沒有記錄到準確的飛行過載ny，因而不能從記錄的數據中準確判定升力卸載情況，但毫無疑問升力卸載比顯示的更多。

在6:41:24″000，即右主輪接地兩秒後，姿態已降低至2.29°，擾流板角度35°（最大50°），接地時垂直過載VRTG達到峰值1.628g。隨後前輪在不到1秒內接地。


接地下降率600 ft/min的情況

（假定飛機質量64 500kg）


我們從上面的演算中，可以看到飛行過載和著陸過載的確會代數疊加，但好像又不是簡單的疊加。對於這種疊加情況，發生在下降率較小時，引不起太大的“水花”，接地時的垂直過載通常不會達到需要“關注”的地步，但是如果發生在下降率較大的極端情況時呢？


我們假定飛機著陸重量為64 500kg，接地下降率為600ft/min，如果在接地前ny=1g——即飛行員拉桿只是克服了拉平法則下飛機固有的低頭速率，使飛機保持了姿態（或者說保持了升力），此時會有怎樣的結果呢？重複前述的計算過程，我們可以得到：

VRTG = ny + N = 1 + 1.6 = 2.6 g


再次假定飛機著陸重量為64 500kg，但我們這次假設飛行員發現了“下沉快”，接地前粗猛帶杆，且飛機姿態增加較大——姿態必須增加，不增加就不能增加升力。下降率從初始值減小了一點，但接地下降率仍為600ft/min。接地前ny增加到了1.2g（計算機仿真模擬得到的理論最大值在1.2到1.3之間，但是QAR數據中暫時未發現超過1.2的情況）。根據飛機能量轉化公式，有：

計算可得：


VRTG = ny + N = 1.2 + 1.4 = 2.6 g


以上只是為方便計算/比較設定的條件，而我們可以想到的是，假使飛行員這一次更早一點發現了“下沉快”——當然為了方便比較，不需要太早，就0.2秒吧。這次ny獲得1.2g的飛行過載提前了0.2秒，在這0.2秒內飛機會下降2英尺，而下降率會從600ft/min時再繼續減小，根據運動學公式：

Vs = V0 + at = V0－(ny－1)gt

其中，g為重力加速度，計算出接地時的下降率為

Vs = 3.05 － (1.2－1)×9.8×0.2

= 2.658m/s = 524 ft/min

同理求得，最終的：

VRTG = ny + N = 1.2 + 1.22 = 2.42 g


我們現在也許應該可以知道了，接地前形成的飛行過載ny和起落架部分的著陸過載N，不是簡單的代數疊加關係，而較大的ny（保證不拉飄的情況下）可以有效減小起落架部分的過載。


小重量的情況


我們先假設某次接地過程中，L = W，著陸重量為最大著陸重量64 500kg，接地前瞬間的下降率為600ft/min（即3.05m/s），於是有，

E = Ek = 300000 （焦）

其中，E為起落架吸收的能量，Ek為飛機接地前的動能。


根據空客AMM手冊可知，A320系列（CEO）飛機在最大著陸重量下，以600ft/min下降率接地，對應的接地垂直過載VRTG應該為2.6。其中升力部分的飛行過載ny我們仍假定為1（本段中算例都作如此較為符合實際情況的設定），這樣起落架部分的著陸過載N為1.6。由此可得對應的起落架對飛機的作用力：

Fs = 1.6W = 1.6mg = 1011360 (牛)



飛機重量為60 000kg時


假設飛機在接地時，達到跟最大起飛重量下同樣的動能（如上圖中的陰影部分E），同樣的壓縮行程，以及同樣的地面載荷，於是有：

0.5×60000×v² = 300000 （焦）

求得：

v = 3.16m/s = 622 ft/min

而此時，根據VRTG = ny + N = 1 + Fs/W ，得到：


VRTG = 1+ 1011360÷(60000×9.8) = 2.72 g


飛機重量為50 000kg時


假設飛機接地前達到跟最大起飛重量下同樣的動能，於是有：

0.5×50000×v² = 300000 （焦）

求得：

v = 681 ft/min

此時，同理可得：

VRTG = 3.06 g


綜上所述，在假定有同樣的主起落架或飛機撞地能量下：

  • 飛機在落地重量64 500kg的情況下，造成的VRTG為2.6g的過載需要以600ft/min垂直速度接地；
  • 飛機在落地重量60 000kg的情況下，造成的VRTG為2.72g的過載需要以622ft/min垂直速度接地；
  • 飛機在落地重量50 000kg的情況下，造成的VRTG為3.06g的過載也只需要以681ft/min垂直速度接地。


在下降率超過600ft/min的情況下，儘管飛機重量小於最大著陸重量，所造成的接地垂直過載都很大，下降率並不需要增加太多，就會有極大的重著陸風險，這可以說大的接地垂直過載是與垂直方向的動能有直接關係——當然更直接的講法就是，過大的垂直速度對應著更大重著陸風險。


本期著重解釋起落架的工作原理，然後進行了示例驗算，希望朋友們瞭解飛機接地垂直過載的來龍去脈，如果有任何疑惑請留言，歡迎探討。本期涉及的算例都可以在QAR數據中找到大量的實例，而下一期我們將會進入到具體的QAR數據分析部分。
