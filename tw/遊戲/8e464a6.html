如何將你的shopify獨立站活動商品打上“折扣標籤”

2020-10-29
| 遊戲

上週有個客戶問我一個問題，有些主題的折扣商品圖片上不能顯示sale圖標，當然，如果能顯示具體的折扣力度會更好。我記得之前也有個客戶問過我這個問題，我當時給的建議是簡單粗暴，直接讓美工做圖上吧。現在發現還是挺多人有這個需求的，所以整理出一篇教程，給大家做參考。有些主題本身已經自帶這個功能可以不做參考。本篇文章以Debut主題為例，教大家如何添加"折扣標籤"。

最終顯示效果如下圖：

1.添加discount-saved文件

添加文件之前，先將當前主題備份一份，避免添加代碼錯誤導致網站不可訪問。點擊Online Store -> Theme -> Current Theme -> Actions -> Duplicate，如下圖：

在複製的主題裡，點擊Actions -> Edit Code，在代碼左側目錄中找到Snippets目錄，點擊Add a new snippet，在輸入框中填寫discount-saved.liquid，點擊創建。如下圖：

2.添加代碼到所創建的文件中

將以下代碼添加到剛剛所創建的文件中，點擊保存。

{% if product.compare_at_price > product.price %}

{% capture discount %}

{{ product.compare_at_price | minus:product.price | times:100 | divided_by: product.compare_at_price }}%

{% endcapture %}

<span style="position:absolute;top:0;right:0;background:#FF0000;display:block;padding:2px 5px;border-radius:4px;z-index:999;color:#FFFFFF;font-size:12px;"><em>{{ discount }} OFF</em></span>

{% endif %}

PS: 上述代碼中第一個加粗的色值為圖標背景色，第二個加粗的色值為字體顏色，第三個加粗的字號為字體大小，大家可以根據自己主題的配色以及圖片大小來調整樣式。

3.圖標添加到集合頁產品圖片上

上述所創建的文件為sale圖標文件，可以放在任意跟產品相關的地方。如集合頁，產品詳情頁，相關產品，推薦產品等。

將代碼放至集合頁產品圖片：

在剛剛的代碼左側目錄中，選擇Snippets->product-card-grid.liquid，不同主題黏貼代碼的文件位置不一樣，類似featured-collection.liquid，collection-template.liquid等文件。

通常標籤一般是放在圖片上面，因此，Ctrl + F搜索‘img’，將下列代碼黏貼到img標籤的上方，如下圖：

輸出代碼： {% include 'discount-saved' %}

添加完成以後點擊右上角save按鈕，保存當前文件。

此時，刷新首頁或者集合頁，就能看到我們剛剛設置好的圖標了。

根據商品不同的價格折扣，顯示具體的折扣力度，而不是統一的SALE標籤，更有說服力。同時，如果商品價格沒有折扣，標籤也不會顯示。

4.圖標添加至產品詳情頁圖片上

在代碼左側目錄中，找到Sections -> product-template.liquid，Ctrl + F搜索‘img’，找到第一個img標籤，將輸出代碼放至img標籤的上方，如下圖：

輸出代碼： {% include 'discount-saved' %}

點擊右上角Save按鈕後，刷新產品頁面，看到如下效果：

上述的sale圖標文件對每個主題都適用，只是每個主題所添加的位置不一樣，即輸出代碼位置不一樣。所以，如果其他主題有需要的，也可按照上述步驟操作。
