錢美華(1927—2010 )



寧波寧海人。中國工藝美術大師、北京特級工藝美術大師、高級工藝美術師。北京市琺琅廠第一任總工藝師、科技研究中心主持人。第一批國家級非物質文化遺產景泰藍製作技藝傳承人。師從樑思成、林徽因主研工藝美術，是新中國知識分子從事景泰藍專業設計的第一人，參與新中國第一份國禮設計。


還有如此眾多

屠呦呦（1930—）

寧波海曙人，藥學家，中國中醫研究院終身研究員兼首席研究員，青蒿素研究開發中心主任。屠呦呦是第一位獲得諾貝爾科學獎項的中國本土科學家、第一位獲得諾貝爾生理醫學獎的華人科學家。2017年1月2日，被授予2016年度國家最高科學技術獎。這是國家最高科學技術獎首次授予女性科學家。

金雅梅(1864—1934)

寧波鄞縣梅墟寶橋金家村(今屬邱隘鎮)人。中國第一位女留學生。光緒十四年回國，先後供職廈門、廣州、成都、北京等地醫院，醫術精深，求診治者甚眾。光緒三十四年，得袁世凱資助，創辦中國第一所公立護士學校——北洋女醫學堂，後又任天津婦女醫院院長。

沈貽薌（1900—1989）

寧波奉化溪口亭下村人，幼名遺香。她是中國第一所女子學校——甬江女子中學第一任國人校長，擔任時間長達二十多年。曾任奉化縣第一屆政協委員、民革奉化縣委員會委員、寧波“甬港聯誼會”副會長。


嚴彩韻（1902—1993）

寧波慈溪人，生物化學家，是中國最早從事生物化學研究的女學者。曾在聯合國兒童基金會、哥倫比亞大學人類營養研究所、紐約聖路加醫院等地任職，修訂出版《營養概論》。其丈夫吳憲則是中國生物化學領域的先驅，創立了中國的生物化學學科。其中長子吳瑞為著名分子生物學家。


陳亞珠(1936—)



寧波人，中國工程院院士，高電壓技術專家、生物醫學工程專家。現任上海交通大學教授、博土生導師，生物醫學儀器研究所所長。是我國無創傷性醫療技術領域開拓者之一。

陳賽娟（1951— ）

寧波鄞縣人。細胞遺傳學和分子遺傳學專家，中國工程院院士、發展中國家科學院院士。曾任上海交通大學國家重點實驗室主任、上海交通大學醫學院血研所執行所長、上海市科協主席、全國科協副主席。陳賽娟長期致力於白血病發病機理與治療研究，建立完整的細胞和分子遺傳學技術，實現了當時我國生物醫學領域中人類疾病新基因克隆零的突破。

黃量（1920—2013）

祖籍寧波，藥物化學家，生於上海。中科院院士。美國康奈爾大學第一位外籍化學女博士。中國第一個以蘿芙木總鹼為成分的降壓藥的參研者之一。黃量院士主要從事天然產物及抗腫瘤、計劃生育、抗病毒等新藥研究。20世紀50年代參與研製成功降壓藥蘿芙木總鹼及其利血平等有效成分，抗腫瘤新藥N-甲酰溶肉瘤素及劍麻皁甙元的利用，治療白血病的三尖杉脂鹼及其同系物的合成等。設計並指導研製的還有新藥維胺酯和甲異靛、抗病毒新藥肽丁安等。在國內較早開展手性藥物的研究。


張芷芬（1927—）

出生於寧波慈溪龍山。數學家。從事常微分方程定性理論和拓撲動力系統理論的研究，是國內這一領域的開拓者之一。歷任北京大學副教授、教授、數學系副主任，北京市數學學會副理事長。專於微分方程的定性理論和動力系統研究。對李奈方程極限環的存在性和唯n性問題有較深造詣。合著《微分方程定性理論》。

陳修良（1907—1998 ）

浙江寧波人，1927年加入中國共產黨；畢業於蘇聯莫斯科中國勞動者共產主義大學；曾任江蘇省委婦委書記，《新華報》總編輯，陳修良是我黨歷史上的第一位大城市的女市委書記；建國後，歷任中共上海市委組織部副部長、全國婦聯華東區工作委員會副主任、全國婦聯第二屆執委、浙江省委宣傳部代部長、上海社會科學院顧問。陳修良同志的丈夫是新中國首任浙江省省長沙文漢同志。

張鴻眉(1927—2005)

寧波鎮海人。1948年，出演其處女作《小城之春》，飾妹妹戴秀。建國後入上海電影製片廠，出演影片《宋景詩》等。1963年起，任導演工作，代表作有《飛向太平洋》，《珊瑚島上的死光》。中國第一部科幻故事片《珊瑚島上的死光》導演。

王丹鳳（1924—2018）

原名王玉鳳，祖籍浙江寧波，出生於上海，著名電影女演員。她在《新漁光曲》一片中主演漁家女一角轟動影壇，被影界稱為“小周璇”。出演過《護士日記》、《女理髮師》、《桃花扇》、《玉色蝴蝶》等影片。1962年被選為新中國首次推出的“二十二大影星”之一。從影40年，她在銀幕上塑造了54個藝術形象，成為高產明星。

周廣仁（1928— ）

祖籍寧波，出生於德國漢諾威。著名鋼琴家，中央音樂學院終身教授，當今中國在國際樂壇中最具影響和權威的鋼琴演奏家、教育家。是第一位在國際鋼琴比賽中獲獎的中國鋼琴家。1955年後在中央音樂學院任教並擔任鋼琴系副主任、主任等領導工作。曾多次在歐美各國的國際鋼琴比賽中擔任評委，創辦了星海、樂友兩所兒童鋼琴學校，創辦中國國際鋼琴比賽和鋼琴考級活動，被譽為“中國鋼琴教育的靈魂”。

俞麗拿（1940— ）

寧波鄞州姜山鎮人，生於上海，中國著名小提琴演奏家。上海市音樂家協會副主席、上海音樂學院教授、上海寧波經促會副會長、俞麗拿小提琴藝術基金董事長。她是小提琴協奏曲《梁山伯與祝英臺》最權威的演繹者之一。中國第一次中國音樂家在國際絃樂大賽中獲獎；中國第一位中國藝術家以自己名字命名藝術基金“俞麗拿小提琴藝術基金”。

胡曉平(1950— )

寧波北侖柴橋人，生於上海，國際女高音歌唱家。是我國改革開放以來第一位獲國際聲樂大賽第一名的歌唱家。俄羅斯天文臺曾以胡曉平的名字命名一顆小行星，載入史冊。2006年底，胡曉平毅然回到故土，參與寧波大學音樂專業的初創，成為第一個實質性助力寧波的著名甬籍音樂家。

張彩珍（1930—）

寧波鎮海人，原國家體委副主任、中國奧委會副主席。張彩珍是共和國有史以來第一位在國家體委擔任副主任的女性，而且還成為中國歷史上第一位獲得奧林匹克勳章的女性。

葉佩素(1958— )



寧波北侖人，有“亞洲七項全能女王”之稱，現為省體育局訓練競賽處處長。在1978年至1987年間，共獲全運會女子跳高冠軍2次，田徑女子五項全能(破全國紀錄)冠軍1次，亞運會田徑女子七項全能冠軍2次。

沈盛妃(1980— )

寧波北侖人，1993年進入寧波市體校進行訓練，1997年 被選入浙江省體訓隊，曾連續奪得1998年曼谷亞運會和2002年釜山亞運會女子七項全能冠軍，並多次獲得國內各類比賽冠軍。中國第一個進入奧運殿堂的女子七項全能選手。

岑偉飛(1967— )

寧波慈溪人，原中國國家地擲球隊運動員，現擔任鎮海煉化培訓中心教師。2006年10月21日，岑偉飛在第五屆世界女子地擲球錦標賽中一舉奪得單打冠軍，實現了中國地擲球運動歷史金牌的零突破。

楊波(1973— )

寧波人，原中國體操運動員、中國女子體操隊隊長。世界盃比賽中獲平衡木冠軍，榮獲國際級運動健將稱號，並獲國家體委授予的體育運動榮譽獎章。1992年，國際體操聯合會將她的一個高難度平衡木動作命名為“楊波跳”。

賀慈紅(1975— )

寧波北侖人，著名游泳運動員，現為廣州軍區體工隊游泳教練。在1991年至1998年間，共獲得全國冠軍7次，亞洲冠軍6次，世界冠軍10次，並7次打破亞洲紀錄，4次打破世界紀錄。
