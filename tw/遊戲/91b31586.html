《寶可夢劍盾》雙打速度壓制隊伍分享

2020-10-29
| 遊戲

　《寶可夢劍盾》中對自己寶可夢能力有自信的玩家們可以選擇參加排位賽，組成多種隊伍來戰勝其他玩家，下面請看由玩家“greed_zzz”分享的《寶可夢劍盾》雙打速度壓制隊伍分享，希望對大家有用。

　　風妖精 @ 氣腰

　　特性：惡作劇之心

　　等級：50

　　努力值：252 特攻 / 252 速度 / 6HP

　　膽小(+速度 -物攻) 性格

　　- 順風

　　- 戲法空間

　　- 挑釁 / 能量球

　　- 月亮之力

　　8代由於出手權的更變判定，感覺風妖精比以前更強了，所以打算建一個依靠速度去強行壓制的隊伍，那麼風妖精就是第一個選進隊的。順風保證出手權，空間在遇到空間隊的時候可以逼迫對面猜拳，挑釁是我上9級以後帶的技能，一開始帶的是能量球為了可以壓低沒極巨化的海兔獸血線，但遇到蠻多海兔獸都是1KO的。不過打上去以後感覺帶多個挑釁穩點就把能量球換成了挑釁，在10級上大師的時候也的確用得上挑釁4戰全勝上了大師。

　　鋁鋼龍 @ 突擊背心

　　特性：堅毅

　　等級：50

　　努力值：252 特攻 / 220 速度 / 38特防

　　膽小(+速度 -物攻) 性格

　　- 日光束

　　- 十萬伏特

　　- 加農光炮

　　- 流星群

　　一開始看上了這個新特性能無視看我嘛，後面實戰的時候發現好像對面用交換場地還能追著點的那隻精靈打，而且鋼龍的屬性不怕仙，弱格鬥和地面，鋁鋼龍自身物耐也高，能吃下大部分精靈無強化情況下的物攻格鬥， 極巨後的極巨龍打擊面廣能降對面一級物攻，極巨鋼加自己一級物防還能更進一步增加自己的坦度，遇到了看我嘛加布裡姆溫的組合可以首發風妖精加鋁鋼龍去強殺對面布里姆溫。日光束也是為了讓後面的鰓魚龍有舒服的輸出環境留著清海兔獸用的。

　　顫弦蠑螈 @ 氣球

　　特性：龐克搖滾

　　等級：50

　　努力值：252 特攻 / 252 速度 / 6HP

　　內斂(+特攻 -物攻) 性格

　　- 破音

　　- 爆音波

　　- 汙泥炸彈

　　- 守住

　　之前看銀風聚聚的銀風沙發現現在環境裡比較強的飛行繫有勇士鷹和鋼鎧鴉，加上4倍弱雷的暴鯉龍也能給隊友提速，於是選了這隻精靈，特性加破音可以保證雷屬性輸出有一個可觀的表現，極巨化的情況下極巨毒提升特攻也和隊伍相性比較合適，4倍弱地所以帶氣球來保證起碼能扛一輪輸出不至於直接被一發地震帶走。

　　杖尾鱗甲龍 @ 生命寶珠

　　特性：隔音

　　等級：50

　　努力值：252 特攻 / 130 特防 / 116HP / 12速度

　　內斂(+特攻 -物攻) 性格

　　- 波導彈

　　- 鱗片噪音

　　- 加農光炮

　　- 守住

　　在7代剛出來的那個環境下卡噗爹遍地走，四倍弱仙的杖尾鱗甲龍在當時環境下感覺很難出場，但是到了8代卡噗家族留在了阿羅拉，環境裡常見的仙系從一開始的仙布到現在布里姆溫長毛巨魔還有風妖精，帶隔音就是為抗仙布的巨聲和與顫弦蠑螈同場的時候顫弦蠑螈無腦爆音波壓血線自己不會掉血，而且杖尾鱗甲龍物攻特攻都不錯，走特攻不怕鬼火，實戰中也遇到由於杖尾鱗甲龍極巨化後極巨技能看不出是物攻還是特攻導致對面對杖尾鱗甲龍用鬼火，一般情況下都是由鋁鋼龍清完了對面的仙系精靈後看情況上場。

　　鰓魚龍 @ 圍巾

　　特性：強壯之顎

　　等級：50

　　努力值：252 物攻 / 252 速度 / 6HP

　　爽朗(+速度 -特攻) 性格

　　- 鰓咬

　　- 咬碎

　　- 精神之牙

　　- 龍之俯衝

　　剛開始的時候想帶的是暴鯉龍，但仔細想了想這個隊伍和鰓魚龍的相性太合適了，不帶有點可惜就帶上了。無腦鰓咬大家都懂這裡就不多說了。

　　洛託姆 (割草機) @ 巧可果

　　特性：漂浮

　　等級：50

　　努力值：252 特攻 / 252 速度 / 6HP

　　內斂(+特攻 -物攻) 性格

　　- 欺詐

　　- 伏特替換 / 十萬伏特

　　- 飛葉風暴

　　- 守住

　　洛託姆是最後進隊的一隻精靈，在它之前的是男愛管侍，想拿愛管侍當一個低配蝶蝶帶命玉打輸出，特性也能擋擊掌奇襲。結果實戰裡發現站不住，上場打一發精神強念然後就被人送到下面去休息了，而且選出也比較少。後面還遇到了一隊腹鼓卡比，發現這個隊伍很難解決卡比這個問題，所以選了洛託姆，欺詐腹鼓卡比傷害爆炸不說，飛葉風暴也能幫忙清海兔獸，總體來說還是和隊伍風格比較搭的。
