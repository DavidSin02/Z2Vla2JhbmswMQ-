項目任務管理系統

2020-10-29
| 遊戲

一個模仿Team@OSC的radmine?

重要提醒

  1. 代碼庫中沒有omni的可執行文件，請到OmniWeb中下載編譯好的二進制文件
  2. 依賴mysql庫
  • Windows：下載Mysql Connector C，將lib目錄下的libmysql.dll，放在項目根目錄，並重命名為mysql.dll
  • CentOS: yum install mariadb-devel, ln -s /usr/lib64/mysql/libmysqlclient.so /usr/lib64/libmysqlclient.so
  • MacOS: brew install mysql-connector-c
  1. 現在有些屬性只有項目管理員可以更改
  • 項目成員管理
  • 更改已存在任務的時間計劃（其他成員只能通知項目管理員協調更改時間，自己不能隨意更改）
  • 任務的驗收管理
  1. 關於驗收
  • 測試通過後狀態為【已完成】的任務才會出現在【驗收管理】中。
  • 驗收操作也是歸檔操作。驗收後，任務不再出現在任務視圖中，可在項目驗收管理及週報中查看。

安裝

  1. Clone代碼庫
  2. 將下載omni可執行文件到項目根目錄
  3. 根據平臺，提供mysql動態庫，供腳本調用，方法見 #重要提示
  4. 更改omni.ini，配置數據庫
  5. 運行omni

默認帳號

帳號: admin

密碼: team

【注】可修改scripts/controller/install/setup.lua中setup.build_in配置，自定義默認帳號
