34部2020-2021復仇者聯盟成員電影

2020-11-14
| 遊戲

早前《復仇者聯盟4：終局之戰》在世界各地都牽起一陣超級英雄風潮，漫威更是在後續公佈了很多新作品給大家帶來不同的驚喜，而在電影后各個演員是否又迴歸平靜，還是忙於其他電影的演出，今日整理了復仇者裡面11個英雄的電影動態分享給大家，都是男演員部分，女演員部分會後續再分享。

電影的製作會在不同因數的影響有所更改，以下的都是暫時小編得知的資料，如果有錯可以留言小編會重新更正，部分影片還沒有確認上映日期所有沒有感謝哦，感謝。

P1 鋼鐵俠：小羅伯特·唐尼

1《尤卡坦》2020

「編劇」:安東尼·佩卡姆 / 保羅·舒爾靈 / 特里·魯西奧

「主演」:小羅伯特·唐尼

「類型」:動作 / 冒險

「簡述」:《尤卡坦》最初的劇本誕生於上世紀60年代，創作者正是好萊塢硬漢小生史蒂夫·麥昆（Steve McQueen），麥昆希望自己能主演這部影片，但他的這一設想直到去世也沒能實現。劇本草稿還是他的家人在他去世後整理遺物時才發現的。

小編記: 先不提唐尼的參演，單單是3個編劇已經夠陣容強大，特里·魯西奧作品《怪物史瑞克》和《加勒比海盜》；安東尼·佩卡姆的《成事在天》更是小編感動得一塌糊塗；保羅·舒爾靈《越獄》編劇，這樣的陣容，小編相信一定能夠給出一部滿意的電影。

2《怪醫杜立德 Dolittle》2020

「導演」:斯蒂芬·加漢

「編劇」:斯蒂芬·加漢 / 休·洛夫廷 / 克里斯·麥凱 / Thomas Shepherd

「主演」:小羅伯特·唐尼 / 湯姆·赫蘭德 / 拉米·馬雷克 / 麥克·辛 / 傑西·巴克利

「類型」:喜劇 / 奇幻

「簡述」:小羅伯特·唐尼將出演新版《怪醫杜立德》，並和妻子蘇珊·唐尼一同擔任製片！目前片名定為“The Voyage of Doctor Dolittle”，杜立德醫生是休·洛夫廷創作的經典兒童文學作品，曾被多次改編並被搬上銀幕，這位醫生以救助動物為第一要任，每天和動物們生活在一起，還會說各種動物的語言。

「上映日期」:2020-01-17(美國)

小編記：鋼鐵俠和荷蘭弟再次合作，粉絲們一定不能錯過系列；編劇的《毒品網絡》獲得73屆奧斯卡金像獎最佳改編劇本。

3《大偵探福爾摩斯3》2020

「導演」:德克斯特·弗萊徹

「編劇」:克里斯·布蘭卡託 / 阿瑟·柯南·道爾

「主演」:小羅伯特·唐尼 / 裘德·洛 / 瑞秋·麥克亞當斯

「類型」:劇情 / 動作 / 懸疑 / 驚悚

「簡述」:小羅伯特·唐尼和裘德·洛迴歸繼續飾演福爾摩斯和華生，目前尚未確認導演人選。

「上映日期」:2021-12-22(美國)

小編記：編劇阿瑟·柯南·道爾多次參與不同版本的“福爾摩斯”製作，而克里斯·布蘭卡託最著名作品：《毒梟》系列。女主參演的《時空戀旅人》和《聚焦》也深受大家歡迎。

P2 美國隊長：克里斯·埃文斯

寡姐太美了 犧牲我們隊長吧 （哈）

1《利刃出鞘》2019

「導演」:萊恩·約翰遜

「編劇」:萊恩·約翰遜

「主演」:丹尼爾·克雷格 / 克里斯·埃文斯 / 邁克爾·珊農 / 安娜·德·阿瑪斯

「類型」:劇情 / 懸疑 / 驚悚 / 犯罪

「簡述」:富裕的犯罪小說家哈倫·特羅姆佈雷在他85歲生日時邀請他的家人到他的豪宅，希望能讓他們一家子團聚。在生日聚會期間，哈倫被家人發現死亡。偵探貝努瓦·布蘭克到案調查該案件，而家中的每個人則都成了嫌疑犯。

「上映日期」:2019-09-07(多倫多電影節) / 2019-11-27(美國)

小編記: 該劇的導演和編劇都是同一人：萊恩·約翰遜，曾執導過《星球大戰：最後的絕地武士》《絕命毒師第三/五季》《環形使者》等作品，此次美隊和丹叔一起合作，帥氣逼人真是讓人期待！

2《捍衛雅各布》2020

「導演」:莫滕·泰杜姆

「編劇」:馬克·鮑姆貝克 / William Landay

「主演」:克里斯·埃文斯 / 米歇爾·道克瑞 / Jaeden Martell

「簡述」:改編自William Landay所著同名暢銷小說，中文譯名《永遠沒有的真相》，講述助理地方檢察官Andy Barber調查一樁14歲男孩被謀殺案，發現自己的青少年兒子是嫌疑人。

小編記：該劇的導演之前和本尼合作的《模仿遊戲》在豆瓣評分8.6，同時還獲得金像獎最佳影片提名，收穫2.33億美元之多，他的《相對宇宙》小編也挺喜歡的；編劇馬克·鮑姆貝克曾參與《金剛狼2》和《猩球崛起2》製作。

3《變身怪醫》2021

「導演」:魯本·弗雷斯徹

「編劇」:安東尼·巴格羅茲 / 史蒂文·莫法特 / Chuck Mondry

「主演」:克里斯·埃文斯

「簡述」:這部《變身怪醫》是根據2007年BBC推出的同名迷你劇改編而成，故事取材於英國小說家羅伯特·路易斯·史蒂文森在1886年創造的膾炙人口的文學形象“化身博士”，哲基爾醫生在經過無數次實驗後，相信自己已經成功製造出可以將人性中善與惡分離的配方。當他將藥物注射到了自己的身上的時候，他本性中的善與惡果然被分離開來。當他的惡掌控身體時，便會精神分裂變身為愛德華·海德，到處為非作歹。

小編記：該劇導演《真愛不死》是小編喜愛的美劇之一哦，《毒液：致命守護者》也是他的作品之一；編劇史蒂文·莫法特《神祕博士》是美劇中的經典，《神探夏洛克 第四季》他也要參演製作。

P3 浩克：馬克·魯弗洛

1《黑水》2019

「導演」:託德·海因斯

「編劇」:馬修·邁克爾·卡納漢 / Mario Correa / Nathaniel Rich

「主演」:安妮·海瑟薇 / 馬克·魯弗洛 / 比爾·普爾曼 / 蒂姆·羅賓斯 / 維克多·加博

「上映日期」:2019-11-22(美國)

小編記：該導演的音樂傳記片《我不在那兒》第 64 屆威尼斯電影評委會大獎，凱特·布蘭切特憑藉該片反串表演獲得最佳女演員獎，該劇的其標新立異的形式為影片贏得了無數好評。編劇馬修·邁克爾·卡納漢作品：《殭屍世界大戰》和《深海浩劫》等。浩克這次和海瑟薇一起合作。

2《我知道這是真的》2020

「導演」:德里克·斯安弗朗斯

「編劇」:德里克·斯安弗朗斯 / 吉娜·普林斯-拜斯伍德 / Wally Lamb

「主演」:馬克·魯弗洛/梅麗莎·里奧/朱麗葉特·劉易斯/羅茜·歐唐內/雅奇·潘嘉比

「類型」:劇情

「簡述」:背景設定在20世紀康涅狄格州的三河，Dominick的兄弟Thomas患有妄想型精神分裂症，通過藥物治療，他能正常在咖啡店工作，但病情時不時會發作。Thomas認為他能做出反抗犧牲來阻止中東戰爭，在一所公立圖書館割斷了自己的右手。Dominick看透過了他並沒有將手接回去的打算，代表他將他從一所壓抑的、治療危險精神病患者的醫院釋放。

小編記：《藍色情人節》《松林外》導演德里克·斯安弗朗斯執導並編寫劇本，根據Wally Lamb所著同名暢銷小說改編，描述為 “ 史詩般的家庭傳奇故事 ”，聚焦孿生兄弟Dominick 與 Thomas Birdsey之間的故事，關於身份的探索與人性的挖掘，魯法洛將一人分飾兩角。
