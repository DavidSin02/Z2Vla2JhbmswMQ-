2. Dress

https://github.com/komeiji-satori/Dress

你可能想不到這個 17.7k star 的項目是一個面向可愛的藍孩子 (/ω＼) 的 git 學習實踐項目 。

該項目對貢獻者的要求不高，並不要求你貢獻代碼，沒有編程技能都可以參加。你可以從這裡學習從克隆項目，創建分支，提交和同步修改，到合併分支請求的整套流程，一次即可熟悉 Git/GitHub 的使用。

但是：你還要事先準備至少一張你的女裝照。

沒錯，這就是女裝大佬集結地，截止目前為止，已經有 240 多名大佬上傳過女裝照，並且有 1098 次提交。光看貢獻者的頭像，二次元風格滿滿。本來想找幾張給大家養養眼，打開後發現頂不住啊，溜了溜了。。。

體驗一下這個編程語言的Helloworld

Wenyan:

吾有一數。曰三。名之曰「甲」。為是「甲」遍。	吾有一言。曰「「公眾號【野生程序猿】」」。書之。云云。

Equivalent JavaScript:

var n = 3;for (var i = 0; i < n; i++) {	console.log("公眾號【野生程序猿】");}

公眾號【野生程序猿】公眾號【野生程序猿】公眾號【野生程序猿】
