懷舊服增強薩PVE攻略：風怒之力會毀滅你

2020-10-29
| 遊戲

一、開始之前

薩滿除去不差的輸出能力外，還具有優秀的輔助能力，憑藉風怒圖騰、戰慄圖騰、清毒圖騰等強大的圖騰能力，既能保護隊友的安全，又能顯著提升隊友的輸出能力。

1、 增強薩滿的優劣

Ø 優勢

n 風怒武器、風暴打擊對增強薩有很好的加成效果

n 最適合施放圖騰輔助隊友的天賦

n 有著打斷和驅散技能

Ø 劣勢

n 技能冷卻時間較長，持續傷害能力較弱

n 缺藍

2、 增強薩滿種族推薦

Ø 巨魔（PVE推薦）

n 野獸殺手：對野獸的傷害提高5%。

n 狂暴：根據當前生命值提高攻速和施法速度10%-30%。

n 再生：生命值回覆速度提高10%。

Ø 獸人（PVP推薦）

n 斧專精：斧和雙手斧技能提高5。

n 血性狂暴：近戰攻強提高25%，受到的治療效果降低50%。

n 堅韌：抵抗昏迷效果的機率提高25%。

Ø 牛頭人

n 栽培：草藥學技能提高15。

n 耐久：生命值上限提高5%。

n 自然抗性：自然抗性提高10點。

n 戰爭踐踏：使半徑8碼內的敵人昏迷2秒。

3、 增強薩滿專業推薦

Ø 元素薩最好的專業就是工程學，通常製造的爆炸物等一些物品在副本內有很大幫助，建議搭配採礦專業，節省購買礦物材料的錢。

Ø 或者學習鍊金和草藥學，藥物可以自給自足，並且還可以賺錢。

二、技能說明

圖騰是薩滿特有的一種能力，共有四種類型：空氣、大地、火焰和水，但每個類型的圖騰每次只能存在一個。

Ø 空氣圖騰

n 風怒圖騰：為半徑20碼內的隊友提供風怒效果，主手武器攻擊時有20%機率額外進行攻擊一次。懷舊服薩滿招牌技能，適合所有近戰職業，尤其是額外攻擊為戰士帶來的怒氣值。

n 風之優雅圖騰：使半徑20碼內的隊友增加敏捷。相比戰士和盜賊，風之優雅圖騰更適合獵人和德魯伊，因為他們不能受益於風怒圖騰。

n 自然抗性圖騰：為半徑20碼內的隊友提供自然抗性。

n 根基圖騰：每10秒為附近的隊友吸收一個有害法術，AOE法術除外。

n 崗哨圖騰：監控一定範圍內的視野，有點類似於鷹眼術。

n 風牆圖騰：減少20碼內隊友受到的遠程傷害。

Ø 大地圖騰

n 大地之力圖騰：為半徑20碼內的隊友增加力量。適合近戰隊友。

n 石膚圖騰：為半徑20碼內的隊友提供物理傷害減免。在應對BOSS時用處不大，但是在坦克扛著一群小怪時效果顯著。

n 地縛圖騰：使半徑10碼內的敵人移動速度降低。風箏怪物時用。

n 戰慄圖騰：驅散半徑30碼內隊友的恐懼、魅惑及催眠效果，在應對某些boss或PVP時很有用。

n 石爪圖騰：吸引8碼內的敵人攻擊此圖騰，單人練級的時候有用，組隊時慎用，圖騰血量不高，很容易ot。

Ø 水圖騰

n 治療之泉圖騰：每隔2秒治療半徑20碼內的所有隊員。

n 法力之泉圖騰：每隔2秒為半徑20碼內的所有隊友恢復法力值。

n 抗火圖騰：為半徑20碼內的隊友提供火焰抗性。在熔火之心和黑翼之巢中很有用。

n 祛病圖騰：每隔5秒為半徑20碼內的隊友驅散一個疾病效果。

n 清毒圖騰：每隔5秒為半徑20碼內的隊友驅散一箇中毒效果。

Ø 火焰圖騰

n 灼熱圖騰：頻繁攻擊20碼內的某一個敵人。在長時間戰鬥中傷害佔比較高。

n 火焰新星圖騰：圖騰持續時間5秒，到時後會對半徑10碼內的敵人造成火焰傷害。傷害不錯，可惜冷卻時間有點長。

n 熔岩圖騰：每隔2秒對8碼內的敵人造成火焰傷害。在火焰新星圖騰冷卻時可以作為替補使用。

n 抗寒圖騰：為半徑20碼內的隊友提供冰霜抗性。在納克薩瑪斯和PVP中應對冰法時有用。

n 火舌圖騰：為半徑20碼內的隊友主手武器附加火焰傷害。相比風怒圖騰，該圖騰作用差遠了。

Ø 治療法術

n 治療鏈：為友方目標恢復生命值，然後轉而治療附近的下一個目標，治療效果每次減少50%，共可治療3個目標。

n 次級治療波：快速地為一個目標恢復少量生命值。

n 治療波：慢速地為一個目標恢復大量生命值。

Ø 傷害法術

n 風暴打擊:獲得一次額外的攻擊機會，使你的下2次攻擊對敵人造成的自然傷害提高20%，持續12秒。

n 閃電箭：對單一目標造成傷害。

n 閃電鏈：對一定範圍內的敵人造成傷害，即使是單一目標傷害也比閃電箭要高，但是藍耗也相對較高。

n 地震術：對敵人造成傷害的同時打斷其施法。在PVE和PVP中都很有用，如果你只需要打斷效果的話，請勿將該技能升級，避免過高的藍耗。

n 烈焰震擊：對目標造成一個火焰傷害的dot。

n 冰霜震擊：對目標造成傷害並降低其移動速度，風箏時很有用。

Ø 功能法術

n 閃電之盾：受到近戰或遠程傷害時，會對攻擊者造成傷害。

n 淨化術：移除地方目標的兩個增益效果。

n 消毒術：為目標移除一箇中毒效果。在清毒圖騰無法使用時是一個很好的技能。

n 祛病術：為目標移除一個疾病效果。在祛病圖騰無法使用時是一個很好的技能。

n 幽魂之狼：移動速度提高40%，只能在戶外使用。無論是快速旅行還是風箏時都是一個非常好的技能。

n 視界術：將你的視野移動至目標區域。在PVP時作為偵查用。

n 水上行走：使友方目標能在水上進行行走。對於在水域多的地圖練級時很有用。

n 星界傳送：相當於是一個爐石，但是比爐石少了一半的冷卻時間。

n 復生：死後能自行復活並恢復少量生命值和法力值。在某些副本中因失誤死亡時很有用，避免了多餘的趕路時間。

n 先祖之魂：非戰鬥狀態時，可復活一名隊友。

三、天賦推薦

風暴打擊0/31/20



這套天賦能讓你儘可能地持續輸出，並且降低了頻繁使用圖騰所消耗的法力值，非常適合與近戰及坦克組隊。需要注意的是，由於風暴打擊會佔用一個DEBUFF的位置，所以在團本中建議替換成“自然迅捷”。

四、技能循環

單目標

Ø 保持者各種增益圖騰，在目標範圍內施放灼熱圖騰。

Ø 風暴打擊是最主要的攻擊手段。

Ø 烈焰震擊和地震術能用就用。

多目標

Ø 除了火焰新星圖騰，增強薩並沒有太好的多目標進攻手段。

五、屬性優先級

增強薩最主要的是優先提升武器熟練度，其次是需要6%的命中率，當這兩個屬性都達標後儘可能多地堆爆擊/敏捷。

Ø 武器熟練度

n 武器熟練度不足會降低傷害

Ø 命中率

Ø 爆擊

Ø 敏捷

Ø 5秒法力值恢復

Ø 智力

Ø 力量

Ø 耐力
