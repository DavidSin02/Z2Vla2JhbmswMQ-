《殺戮尖塔》更新2.0版本 新角色“觀者”等大量內容上線

2020-10-29
| 遊戲

《殺戮尖塔》2.0版本今日上線，正式推出了第四位角色“觀者”，並加入了新卡牌、新遺物、新藥水和平衡性調整等內容。

以下是本次更新的詳情：

大家好，　　2.0版本更新今天終於發佈了！本次更新帶來了新卡牌、新遺物、新藥水、平衡型調整、重做、bug修復、性能改進和第4名角色。　　您仍然可以在Steam中的BETAS菜單中選擇“v1.1 - 2019-17-07”來繼續遊玩過去的版本。

　　第4名角色　　觀者前來評估我們的高塔。她將以神聖的“姿態”與手中的預示之杖來對高塔中的魔物們進行審判。　　和其他角色一樣，觀者擁有自己的牌組和一些獨佔的特別遺物。　　感謝所有在beta版中測試第四名角色的玩家門，你們的反饋、數據和討論幫助我們能夠迅速地測試各種新點子和調整這名角色！

　　藥水研究所

　　我們這次新增了許多藥水，因此也創建了一個單獨的頁面，讓你可以一覽所有的藥水！　　你可以在主菜單的百科大全中找到藥水研究所。　　你甚至可以在其中看到所有新藥水和現有藥水的稀有度。

　　Mod　　本次更新對遊戲本體進行了不少改動，也對Mod支持進行了一些改進。與1.1版兼容的mod很可能需要更新後才能在2.0版本中繼續運作，請務必更新！

　　平衡性　　一些稀有度變化、數值變更和各種重做！

  • 召喚鈴鐺重做。
  • 懸浮風箏重做。
  • 小寶箱重做。
  • 商店遺物的基礎價格下調200 -> 150。
  • 黑暗之血遺物加強，治療量10 -> 12。
  • 永恆羽毛遺物稀有度變更，Boss->罕見。
  • 蜥蜴尾巴遺物稀有度變更，Boss->稀有。
  • 餐券遺物稀有度變更，商店->普通。
  • 星系儀遺物稀有度變更，Boss->商店。
  • 工具箱遺物加強，隨機獲得無色牌變為三選一無色牌。
  • 玩具撲翼飛機遺物稀有度變更，商店->普通。
  • 白獸雕像遺物稀有度變更，Boss->罕見。
  • 袖箭遺物加強，額外傷害3 -> 4。
  • 火焰吐息卡牌重做。
  • 重編程卡牌重做。
  • 斷魂斬+加強，傷害20 -> 22。
  • 計劃妥當耗能0 -> 1。
  • 在回合結束時獲得格擋的卡牌將會在灼傷等受傷牌之前生效。
  • 普通藥水的掉落率下降70% -> 65%。
  • 稀有藥水的掉落率提升5% -> 10%。
  • 異蛇之油重做。
  • 鮮血藥水回覆量加強10% -> 20%。
  • 鮮血藥水稀有度調整，罕見->普通。
  • 瓶中精靈加強，回覆量10% -> 30%。
  • 集中藥水稀有度調整，罕見->普通。
  • 罐裝幽靈稀有度調整，罕見->稀有。
  • 毒藥水現在變為靜默獵手獨佔。
  • 換臉事件不會再在第二階段之後出現。
  • 涅奧的無色牌選項現在變為三選一。
  • 多層護甲現在最多999層。
  • 自定模式中的特效現在更新了觀者相關的內容。
  • 永存者成就現在需要紫水晶與紫水晶+成就解鎖。

　　界面與特效

  • 現在支持4:3顯示比例。
  • 蝨蟲敵人的吐網效果增加了音效。
  • 設定中增加了“字體更大”模式，會將所有文字稍微變大。
  • 卡牌預覽中現在可以顯示一些相關的卡牌，例如無謀衝鋒會顯示暈眩。
  • 生成升級卡牌的卡牌/能力/藥水現在一律使用+來描述。
  • Boss第一勇士現在會對自己的腰帶有反應。
  • 增加了“斬殺”關鍵詞，描述殺死非爪牙敵人時生效的效果。
  • 解鎖卡牌和遺物時的字體變大。
  • 在輸入設定中鼠標滾輪速度變為3倍。
  • 加入察看按鈕，在戰鬥中選項界面出現時可以察看手牌和戰況。
  • 現在在一些遺物界面按0 - 9會播放一些音效。
  • 按ESC鍵退出其他菜單時不會再自動同時退出卡牌獎勵界面。
  • 一場戰鬥或一局遊戲中只觸發一次的遺物現在會在觸發後變灰。
  • 更改了設定畫面中“你需要重啟遊戲”的文本位置。
  • 歷史記錄等可以滾動的下拉菜單現在可以使用鼠標滾輪和方向鍵來滾動。
  • 一些藥水有了新的圖片。
  • 一些藥水現在是彩虹色的了，很酷炫。
  • 罕見藥水現在有圈狀的粒子效果。
  • 稀有藥水現在有金閃閃的特效。
  • 一些能力圖標變得更加清晰。
  • 一些遺物圖標變得更加清晰，並且查看時會顯示高清版本。
  • 一些卡牌/遺物的速度被加快和一致化。
  • 一些有特殊前提條件的卡牌（比如飛身踢）現在會有金色發光效果提示。
  • 懸停在地圖圖例的圖標上時，相關信息會出現在固定位置，不再遮擋地圖。
  • 部分遺物、卡牌、藥水和能力調整了文本，讓表達更為明確。

　　Bug修復

  • 自定模式現在即使在離線狀態下進行每日挑戰也可以解鎖。
  • 飛身踢的結算順序現在按照卡面描述進行。
  • 蛋系遺物現在在拾取時會反映在同一批卡牌獎勵的卡牌上。
  • 蛋系遺物現在會反映在潘多拉魔盒的確認界面上。
  • 混沌藥水現在可以在戰鬥外使用。
  • 修復了武裝卡牌不更新升級卡牌描述的bug。
  • 修復了打敗特定敵人後，格擋對於荊棘類能力無效的bug。
  • 修復了卡牌總覽界面點開時會自動滾屏一點的bug。
  • 修復了一些牌無法被打出時仍然會發光的bug。
  • 修復了在觸屏模式下的改名界面中需要點兩次確認/取消按鈕的bug。
  • 修復了發現卡牌在打開其他菜單後會出現跳過的bug。
  • 修復了分離在對小黑和覺醒者打出兩次時會獲得兩次能量的bug。
  • 修復了蛋系遺物無法升級“留給自己的訊息”事件中卡牌的bug。
  • 修復了無盡模式中精英分數顯示不正確的bug。
  • 修復了混沌藥水在有添水時也能生效的bug。
  • 修復了ESC鍵在卡牌獎勵畫面有可能導致地圖變白、隱藏獎勵界面等的bug。
  • 修復了逃脫計劃在不能抽牌能力生效時依然能夠獲得格擋的bug。
  • 修復了不懼妖邪+心靈堡壘不能在死亡律動前獲得格擋的bug。
  • 修復了勢不可當的傷害在殺死一名敵人後不會改變目標的bug。
  • 修復了亂戰、破滅、精煉混沌、通曉萬物有時在目標死亡時不會打出卡牌的bug。
  • 修復了商人素材初始化時內存溢出的bug。
  • 修復了無盡模式的荒疫在初始化時內存溢出的bug。
  • 修復了在進入房間時生效的遺物，在戰鬥後獎勵界面時能夠通過讀檔再次觸發的bug。
  • 修復了懷錶遺物在與先見之明等效果互動時過早觸發的bug。
  • 修復了彈回卡牌無法對奇怪的勺子遺物影響的卡牌起效的bug。
  • 修復了銅鈸遺物的傷害在殺死敵人後不會改變目標的bug。
  • 足跟勾的結算順序現在按照卡面描述進行。
  • 史萊姆和心臟敵人的音效不會再在遊戲後臺運行時播放。
  • 銅球的凍結效果現在使用隨機數種子了。
  • 轉化神龕事件的選項現在使用隨機數種子了。
  • 設計師事件的選項現在使用隨機數種子了。

　　性能優化

  • 修正了卡牌描述被初始化兩次的bug，優化啟動時間。
  • 刪除一些未使用的字體文件，減少內存使用和啟動時間。
  • 變灰的遺物現在使用著色器，而非使用灰度圖片來達成效果。

  • 現在有了越南語版本！
  • 一些時間/日期格式現在可以按語言調整。
  • 增加一些邏輯判斷，來為中日韓語言提供更好的卡牌文字格式。
  • 2.0版更新目前有如下語種！
  • 荷蘭語、世界語、法語、德語、意大利語、日語、韓語、波蘭語、俄語、中文（簡繁）、西班牙語、泰語、烏克蘭語、越南語。
