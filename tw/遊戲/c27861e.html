捷克公開賽男雙晉級、單打僅3人，燃燒吧小宇宙

2020-10-29
| 遊戲

北京時間8月24日至25日凌晨，國際乒聯世界巡迴賽捷克站，結束了正賽階段首日的比拼。

以10名小將出戰的中國乒乓球隊，在頭兩天的單打資格賽中被淘汰5人（曹巍、宋卓衡、彭飛龍、黃穎琦、張釗），正賽首日再被淘汰2人（李藝傑、袁媛），男隊只剩下薛飛、房胤池，而女隊更是隻有顧若辰一名選手了。

在24日的女雙淘汰賽，顧若辰 / 張釗戰勝了匈牙利隊的柏林特 / 奧拉迪亞。25日下午16點開始的女雙1/4決賽，顧若辰 / 張釗對陣埃霍爾姆（瑞典）/ 波塔（匈牙利）跨國組合,1比3失利（9-11,11-4,4-11,8-11）。中國女雙止步於8強，未能進入半決賽。

日本組合橋本帆乃香 / 佐藤瞳、早田希娜 / 伊藤美誠、跨國組合埃霍爾姆/ 波塔、 奧地利組合MADARASZ Dora / POLCANOVA 晉級女子雙打四強。兩對日本女子組合將在半決賽展開內戰。

在24日的男雙淘汰賽，房胤池 / 薛飛戰勝了東道主捷克選手康內克尼 / 波蘭斯基；25日下午16點40分，男雙1/4決賽，房胤池 / 薛飛對陣伊朗隊阿拉米安 / 阿拉米楊。這是一場艱苦的比賽，大比分曾經1比2落後；從第四局開始，薛房組合開始掌控比賽節奏，在懸崖邊把比分扳回2平；第五局房胤池發球連得兩分10比8拿到賽點；五局的比分依次是13-15,11-8,10-12,11-6，11-8，恭喜房胤池 / 薛飛 ，晉級四強。

房胤池 / 薛飛半決賽的對手是德國的德國弗蘭斯卡FRANZISKA /丹麥喬納森Jonathan。另一場半決賽對手是巴西 卡爾德拉諾 / 法國 西蒙高茨，日本上田仁/吉村真晴組合。

男子單打，薛飛苦戰7局，險勝烏克蘭選手寇磊；在25日22點，他將面對德國名將波爾。實力與經驗的差距決定了這將是一場硬仗，薛飛只能放手去拼。

房胤池4比0大勝奧地利選手“中國女婿”費格爾，他的下一個對手是中國臺北隊林昀儒。

顧若辰作為女乒最後一位選手，她將在25日23點30分迎戰日本隊伊藤美誠。

比賽到這個地步，國乒三位已經沒有退路。燃燒吧小宇宙，用熱血和勇氣去衝擊，在狹路中迸發激情，發揮出最好的水平，創造奇蹟！

乒乓網 胡清源
