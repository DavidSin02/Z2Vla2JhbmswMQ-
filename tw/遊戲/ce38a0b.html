鋼筆筆尖調整技巧及維護指南

2020-10-29
| 遊戲

鋼筆的很多問題都可以通過一次妥善的清潔（可參考往期文章《鋼筆清洗詳細指南及鋼筆維護保養指南》）來解決，但有些問題還是需要略高級一點的處理方法。如果你的鋼筆有點刮紙、筆尖沒對齊，或者只是寫起來不順意，即使清潔了也還是這樣，可以往下看看這些技巧和要訣。

在嘗試自行調整一支鋼筆之前，充分理解鋼筆的工作原理是很有幫助的。如果你是鋼筆領域的小白，可以先看看上期文章《鋼筆的工作原理》。

注意事項

雖然這裡列出的所有方法操作起來都相對簡單，但如果操作不當，可能會對鋼筆造成永久損傷。請注意，使用這些方法還可能會使鋼筆廠家的保修失效，你需要對任何可能造成的損失自行負責。

動作要輕緩，並且要經常測試你的鋼筆。筆尖是比較精密的器具，即使是很輕微的調整都可能會導致顯著的結果。

現在，我們來看一下鋼筆筆尖最常出現的三大問題及其解決辦法。

刮紙

書寫的時候筆尖刮紙，不僅讓人心情不悅，還會破壞紙張，被刮起來的紙纖維也可能會堵塞筆尖。刮紙通常是因為筆嘴/銥粒沒對齊或銥粒有缺陷。

你可以按照以下步驟來解決筆尖刮紙的問題。

檢查筆嘴/銥粒是否對齊第一件事就是先看看筆嘴/銥粒有沒有準確對齊，這裡強烈建議使用放大鏡，能更容易看清楚。

筆嘴/銥粒沒對齊最常見的情況是筆嘴/銥粒一邊高一邊低，對於這種情況可以參考以下方法：

  1. 把筆尖放在一個硬平面上，例如可以在桌子上放一張紙，然後用指尖輕輕按壓筆尖。

  2. 按住筆尖左右搖動大約5-10秒。

  3. 檢查一下筆尖是否對齊，如果還是沒對齊，重複第二步。

  4. 這樣做可能會導致筆尖輕微彎曲，如果必要的話，這時可以把筆尖反過來，指尖放在筆尖背面的筆舌上連續多次按壓。

另外一種沒那麼常見，但更嚴重的情況是筆嘴/銥粒一邊大一邊小。這種情況沒法輕易修好，需要聯繫賣家或廠家更換筆尖或鋼筆。

檢查銥粒是否損壞如果筆嘴/銥粒已經準確對齊，但筆尖還是刮紙，那麼就要用放大鏡檢查一下筆尖的銥粒是否有損壞。銥粒表面應該是完全光滑的，如果看到任何的裂痕、毛刺或其他不平整，應聯繫賣家或者廠商更換筆尖或鋼筆，因為這些缺陷是沒辦法輕易修好的。

打磨筆尖如果銥粒太粗糙，或者銥粒裡沿有點太鋒利，你可以通過輕度的打磨筆尖，使之更平滑一點。注意要先嚐試上文提及的方法，如果還不能解決刮紙的問題，再去考慮打磨。因為打磨銥粒損壞或者筆嘴/銥粒沒對齊的筆尖，只會把問題變得更糟。打磨工具可以是很細的grit buff sticks、Mylar或micro-mesh，但打磨筆尖是一項非常高級的操作，如果沒有經過大量研究和充分準備，不建議嘗試。

有一個操作相對比較安全的打磨技巧，不需要用到特殊工具，就是用紙袋打磨。

找一個紙袋，或者其他比較硬的紙面，給鋼筆上好墨水，用你平常的書寫角度和力度，不停地寫數字“8”，然後每隔幾分鐘左右就在普通的紙張上試寫一下這支鋼筆。這個方法可能需要一些時間才能產生明顯的效果，並且可能不會對所有的筆尖都適用。如果你發現筆尖在紙袋上寫的時候也會刮紙，必須馬上停下來——這表示筆尖的刮紙問題嚴重到不適合用這個方法打磨，刮下的紙袋纖維可能會堵塞和損壞筆尖。

出墨不流暢

墨水流量指的是墨水從鋼筆落到紙張上的速度。出墨太節制或出墨不均勻會出現飛白，或者看起來像褪色一樣，還可能導致不能“一筆出水”——在一個筆畫的開始會有一點飛白。

你可以按照以下步驟來解決筆尖出墨不流暢的問題。

檢查筆舌和筆尖是否對齊

筆尖應安裝在筆舌上，筆尖中縫應對齊筆舌上的墨水通道。如果筆尖中縫沒有對齊筆舌的墨水通道，可以試著手動調整一下，使筆尖和筆舌對齊。

整個筆尖還應緊密穩固地安裝在筆舌表面上。如果筆尖和筆舌之間有縫隙，可以按照以下步驟修正：

  1. 用指尖按住筆舌背面，把筆尖反過來頂在一個硬平面的邊沿，輕壓一下筆尖頂端。

  2. 頂在平面上，從筆尖底部到銥粒的方向慢慢地上下轉動筆尖。

  3. 重複這個步驟，每隔幾次就看一下還有沒有縫隙。

  4. 這樣做可能會導致筆尖輕微彎曲。如果必要的話，可以把筆尖反過來，指尖放在筆尖正面連續多次按壓。

檢查筆尖的中縫

確保筆尖是乾淨的，把筆尖擦乾放在燈光下，這樣可以看到燈光穿過筆尖的中縫。中縫應該向著銥粒方向輕微的逐漸變窄，但實際上筆嘴/銥粒的左右兩半應該是沒有直接接觸的。中縫太窄或者太寬會導致出墨太節制或者不均勻。

實際上這些問題都非常細微，至少要用10倍放大鏡才能準確檢查出來。

如果中縫太窄（筆嘴/銥粒的左右兩半直接接觸）：

  1. 在一個硬平面上，慢慢地向下按壓筆尖，使筆尖分叉不超過1-2mm。

  2. 按住不放幾秒，然後鬆開。

  3. 重複這個步驟，每隔幾次就檢查一下筆嘴和銥粒的左右兩半是否還有直接接觸。

  4. 這可能需要一點時間，但對於大多數筆尖都應該是適用的。不要為了加快速度就太用力按壓筆尖或者按住筆尖太長時間，因為這可能會導致中縫太寬或者對筆嘴/銥粒造成永久損害。

或者，你還可以用一塊很薄的黃銅片，緩慢地輕柔地插進中縫，就像用牙線清潔牙齒一樣。同樣的，這需要時間，必須要耐心，不能一味貪圖速度。

如果中縫太寬：

  1. 與上文提到的修理筆尖和筆舌之間有縫隙的方法一樣，把筆尖反過來頂在一個硬平面的邊沿，然後左右旋轉筆尖。

  2. 用指尖按住筆舌背面，頂在一個硬平面上，左右搖動5-10秒，適當重複這個步驟。

  3. 指尖放在筆嘴/銥粒的其中一邊，把筆尖側面頂在一個硬平面上，按住幾秒。再換另一邊，適當重複這個步驟。

  4. 用指尖捏住筆嘴/銥粒兩邊，向內壓緊5-10秒，適當重複這個步驟。這樣手指可能會不舒服，所以我們建議用一塊柔軟的薄薄的布或者橡膠墊住指尖。

檢查銥粒頂端的形狀如果一支鋼筆短時間暫停書寫之後的第一筆會有飛白，但其他時候寫起來都沒有問題，那很可能是銥粒形狀的問題。這個奇怪的問題有一個奇怪的叫法——“嬰兒屁股”銥粒問題。意思就是銥粒內側是圓弧狀的，就像嬰兒的屁股一樣（如右圖）。毛細現象會把墨水輸送到銥粒之間最窄的位置，嬰兒屁股形狀的銥粒會使墨水停在缺口處，與銥粒接觸紙面的位置之間有一條細縫。

當墨水不能直接與紙面接觸，紙張的吸水性和墨水的表面張力還是能維持墨水流動，但如果筆尖離開紙面幾秒，墨水會倒流回去，導致再次書寫的第一筆會有輕微飛白。

如果你的筆尖有“嬰兒屁股”的問題，可以嘗試以下辦法：

  1. 加大書寫力度。用力把筆尖按在紙面上會有助於墨水與紙張的接觸。

  2. 使用沒那麼光滑且吸水性強的紙張。光滑的、吸水性弱的紙張通常被認為對鋼筆好，但其實會加劇“嬰兒屁股”導致的問題。

  3. 使用流動性好的墨水。這會在下一節進一步討論。

“嬰兒屁股”筆尖是有修復方法的，但這個方法難度太高，就不在這裡講述了。

使用流動性好的墨水就像不同的鋼筆擁有不同的出墨流暢度，各種墨水的流動性也是有區別的。流動性特別好的墨水通常指溼度高的墨水。雖然給出墨不流暢的鋼筆上流動性好的墨水並不能從根本上解決問題，但可以減輕症狀。

Waterman威迪文墨水因適合在出墨不流暢的鋼筆上使用而著名。還有一些也經常被推薦的流動性好的墨水品牌包括Diamine戴阿米，Pilot百樂色彩雫和寫樂。

出墨太快/水槍

鋼筆出墨太快會很難控制，寫出來的線條很粗，要很長時間才能乾透，而且還很容易洇紙或者滲透到紙的背面。更嚴重的情況是，筆尖可能會滴墨或漏墨。

以下是筆尖出墨太快/水槍問題的解決步驟。

放置在陰涼處溫度太高會導致鋼筆內部的空氣膨脹，把墨水擠出筆尖。如果在溫度變高或者鋼筆太貼近身體的時候下才會出現漏墨或水槍的情況，那原因很可能就是溫度太高。

檢查氣密性

出墨太快/水槍問題可能是因為鋼筆某處漏氣。上墨器或墨囊的開口處和筆尖之間沒有設置密封經常會導致漏氣。如果可以，換一個不同的上墨器或墨囊看看是否還會漏氣。

收緊筆嘴

你也可以試試擠壓收緊筆嘴減少出墨量。用指尖捏住筆嘴兩邊5-10秒，適當重複這個步驟。如上文提及，這樣手指可能會不舒服，建議用一塊布或者橡膠墊著指尖。

使用流動性較弱的墨水

流動性好的墨水指的是溼度高的墨水，流動性較弱的墨水則被稱為“較幹”的墨水。一些較乾的好墨水包括Pelikan百利金4001和Lamy凌美墨水。

總結

如果你不介意自己動手並且手指沾上墨水，可以自行修復或者把鋼筆調整成你想要的樣子。關於鋼筆筆尖的種類，可以查看之前的一篇文章《鋼筆筆尖知識入門之如何選擇一款合適的筆尖》。

如果你有任何關於筆尖調整的問題或想法，可以在評論裡提出！

什麼時候需要調整鋼筆筆尖：出現清潔無法解決的筆尖刮紙、出墨不流暢或出墨過快/水槍問題。筆尖刮紙

  1. 檢查銥粒是否對齊

  2. 檢查銥粒是否有缺陷

  3. 打磨筆尖

筆尖出墨不流暢

  1. 檢查筆尖和筆舌是否對齊

  2. 檢查筆尖中縫

  3. 檢查“嬰兒屁股”

  4. 使用流動性好的墨水

筆尖出墨過快/水槍

  1. 檢查氣密性

  2. 收緊筆嘴

  3. 使用流動性較弱的墨水

流動性較好的墨水：戴阿米、百樂色彩雫、寫樂、威迪文流動性較弱的墨水：Lamy、百利金4001有用的工具：柔順的毛巾；放大鏡；黃銅薄片；粗糙的紙袋；橡膠薄片
