魔獸世界8.3預覽：新增同盟種族，神器系統更新，法師塔玩法迴歸

2020-10-29
| 遊戲

魔獸世界8.2.5才上線沒多久，8.3的PTR也馬不停蹄的到來了。此次8.3更新比8.2更新了更多了內容，新增同盟種族機械侏儒與狐人、黑龍王子攜橙披再度歸來，法師塔模式將再次迴歸等一系列重大更新。

本次8.3更新的主要內容有：

Ø 新同盟種族——機械地精、狐人

Ø 新團隊副本——尼奧羅薩

Ø 新地圖入侵事件與泰坦系統

Ø 法師塔玩法迴歸

Ø 神器項鍊更新

Ø 新的祕境詞綴

Ø 狼人與地精傳承護甲

新同盟種族——機械地精、狐人

此次更新兩個陣營將會分別新增一個同盟種族：機械侏儒（聯盟）和狐人（部落），之前被猜測會與機械侏儒對位的海地精種族被替換成更萌的狐人。（種族技能名稱為暫譯）

機械侏儒

Ø 解鎖條件：完成“麥卡貢的威脅”成就，鏽栓抵抗軍聲望崇拜。

Ø 可選職業：獵人、法師、武僧、牧師、潛行者、術士、戰士。

Ø 種族技能：

n 作戰分析：5秒提高一次主屬性（120級為50），最多可疊加8次，直到戰鬥結束後衰減。

n 超級有機光源發明家：召喚一個分身吸引敵方注意力。3分鐘CD。

n 骷髏小指：可以打開上鎖的箱子和門。

n 工藝大師：你將可以充當鐵砧、烹飪用火、礦鋤等多種專業工具。

n 重新武裝：生命值降低至20%時，立即恢復20%生命值。該效果每1.5分鐘觸發一次。

Ø 種族坐騎和傳承護甲如上圖所示。

狐人

Ø 解鎖條件：完成“深埋於沙的祕密”成就（沃頓主線任務），沃頓聲望崇拜。

Ø 可選職業：獵人、法師、武僧、牧師、潛行者、薩滿祭司、術士、戰士。

Ø 種族技能：

n 技巧之袋：從你的技巧之袋袋中選擇技能，可對友方或敵方使用。1.5分鐘CD。

n 翻動包裹：更改技巧之袋內的技能。

n 營地：設定該位置為你的營地，只能在戶外使用。5秒CD。

n 返回營地：返回設定的營地。10秒施法時間，1小時CD。

n 防火：減少1%火焰傷害。

n 敏銳嗅覺：降低受到敵人第一次攻擊的傷害。

n 狐人生存之道：殺死人形生物後獲得額外的戰利品。

Ø 種族坐騎和傳承護甲如上圖所示。

新團隊副本——尼奧羅薩

本次新團本將是12個BOSS的大型團隊副本，1號BOSS為黑王子拉希奧，尾王為古神恩佐斯。同時，熊貓人團本雷電王座的隱藏BOSS萊登也將作為BOSS出現在新團本中。

介紹語：

沉睡之城尼奧羅薩已然覺醒。經過了無數紀元後，恩佐斯終於再度登上黑暗帝國的寶座。他的大軍從黑暗的殿堂中傾瀉而出，再次令艾澤拉斯生靈塗炭。整個世界在他們的魔爪下顫抖。於是，部落和聯盟的勇士匯成了聯軍，孤注一擲地深入這支上古大軍的大本營，與其進行最終的決戰。這場戰鬥將會決定整個宇宙的命運。

新地圖入侵事件與泰坦系統

恩佐斯的使者將會入侵奧丹姆和錦繡谷兩地的泰坦設施，屆時這兩處將會被古神的能量籠罩，玩家需要前往清理這些古神的能量使其恢復如初。不過官方並未對該玩法有進一步的說明，不過暴雪總監在介紹該玩法時，提到了該事件將會改變泰坦鑄造系統。（下版本要消滅泰坦造物了嗎？#偷笑）

法師塔玩法迴歸

新的類似於法師塔挑戰的玩法“恐怖願景”。恩佐斯的勢力將會入侵奧格瑞瑪和暴風城，玩家可以1-5人前往進行挑戰。進行挑戰時會獲得拉希奧提供的幫助，獲取並強化橙披，以抵抗古神的入侵。在挑戰中還能獲得“恩佐斯的腐化碎片”，可用於強化神器項鍊。目前還未有獎勵幻化的消息，不過應該會有與法師塔類似的獎勵。

神器項鍊更新

Ø 神器最高等級將提升至85（原70級）。

Ø 新增1個次插槽。

Ø 新增2個耐力點數。

Ø 新增7個精華。

Ø 新增與恩佐斯相關的巔峰插槽。

新的祕境詞綴

新的祕境詞綴“方尖碑（Monolithic，暫譯）”。在地下城中將會出現一定數量的方尖碑，玩家點擊方尖碑後將會進入另一個位面去清除恩佐斯的僕人，如果玩家跳過這些方尖碑的話，這些恩佐斯的僕人將會在尾王出現。

另外，還有一系列的職業改動，由於目前才剛開始PTR測試，後續肯定還會繼續進行改動，就不一一列舉了。總體來說，武僧三系中，踏風和奶僧得到一定加強，酒仙醉拳相關技能有一定的削弱。鳥德削弱了月火和陽炎。暗牧削弱了吉兆的傷害加成、狂亂合唱（特質）爆擊加成、兩個dot技能的傷害。

除了上述更新外，拍賣行系統也進行了改動，玩家購物將會更便捷。英雄難度黑海岸也將會在8.3上線。還有一系列新坐騎、新裝備等相關信息我們會在後續的文章中一一進行介紹，歡迎關注我們天涯電競。
