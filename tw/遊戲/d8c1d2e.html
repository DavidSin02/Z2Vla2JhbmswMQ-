19歲女生李心草死亡事件，立案偵查！出事前這個細節被的哥曝光

2020-10-29
| 遊戲

//

昆明刑警再調查涉事酒吧

//

據上游新聞報道，記者在現場看到，昆明警方刑偵人員於14日中午12點30分左右進入熱度酒吧，多名刑偵人員進入酒吧後關上捲簾門進行調查。中午1點50分許，昆明警方刑偵人員結束調查，並對酒吧外觀等拍照後離開。


警員對上游新聞記者表示，警方目前仍然在對此案進行調查，“屍檢結果儘快在做。”對於當天早晨有媒體發佈的涉案信息，“我們沒有發過。”









10月14日下午2點左右，李心草墜江地、位於昆明市盤龍區桃源街132號的熱點網吧，昆明刑警再次赴現場勘查。（來源：上游新聞）

//

出租車司機：兩男子勸女孩下車

//

10月13日上午，澎湃新聞（www.thepaper.cn）根據李心草家屬提供的車牌號，聯繫到了9月9日晚李心草乘坐的出租車。

萬通出租車公司陸姓出租車司機表示，此前他已經接受過警方對此事的調查。據其介紹，9月9日晚，他開車經桃源街時，見一名女孩（李心草）站在酒吧門口招手叫車，女孩旁邊還站著一名男子。隨後，他將車開到路邊，女孩從後門上車坐在後排位置。

司機稱，女孩上車後，前述男子將車門打開，勸女孩下車。隨後，另一名男子從酒吧裡出來，站在車邊。兩男子均勸女孩下車，並稱“你喝多了，再玩一會兒，等下一起走”。

司機告訴澎湃新聞，女孩上車後，“沒有說過一句話，臉上沒有任何表情，就呆坐在後排座位上”。司機稱，大約坐了兩三分鐘後，女孩從後排座位的另一側下車，走向了江邊的方向，“（女孩）走的很快的樣子，兩名男子也跟著女孩向江邊方向走去”。

李心草墜江前，

到底發生了什麼？

事發後受害方家屬

有否與被警方調查的3人有接觸

這些問題亟待解答……


10月14日，在接受封面新聞採訪時，李心草親屬迴應了相關問題。

心存疑慮，無法接受調查結論


李心草母親


家屬表示，他們曾在9月15日，第一次看過那段長達三小時的視頻。


他們選擇錄下所發佈的片段，是因為在事發當天9月9日，李心草母親第一次見在場三人時，涉事男性羅某曾向家屬保證，“發誓沒有與李心草發生過沖突，沒有碰過違禁品，完全沒有碰過李心草，也沒有過語言刺激和勸酒”。


嘗試調解，並未達成任何共識


李心草家屬透露，在看到視頻前，9月11日警方曾組織李心草一方和涉事同學一方進行調解。


其中，任某及其母親，和涉事兩位男性參與了調解。在協商中，李心草舅媽曾向三人提出80到90萬的賠償金額討論，“主要用於心草媽媽身體的治療和外債償還還有精神賠償”。

據李心草母親透露，涉事兩位男性以父母不在為由表示回絕，而任某的母親則一再強調自己是單親家庭，經濟困難。


“她一直在訴苦，說自己的女兒出事之後也很不好過，今後的路也很難走，希望我們能夠換位思考，想想他們的難處”。任母的說法徹底惹怒了李心草家屬，一行人當場離席，調解草草結束，沒有達成任何共識。


李心草母親仍在和女兒微信“對話”

拒絕調解 否認花錢買熱搜


李心草家屬表示，在9月15日看到相關視頻後，他們拒絕了一切調解事宜，“我們只想要真相”。

10月14日下午，社交媒體流傳出一張微信對話截圖，稱李心草的母親出錢買熱搜。上游新聞就此求證了李心草親屬。多位親友證實，李心草母親的微信頭像並非網傳截圖中的頭像，“我們從來沒有花錢買過熱搜，從始至終都只有一個頭像。”


社交媒體流傳的微信截圖，稱李心草母親出錢買熱搜，遭李心草家屬否認。
