雲頂之奕9.14版本必須知道的事：測試排位現問世，鳥盾法師成歷史

2020-10-29
| 遊戲

國服雲頂之奕最近大家正在玩的不亦樂乎之時，國服已經在7月23日開始更新9.14新版本，這一版本可以說是一次大改，主要做的就是針對上一版本中的不平衡現象和大多數bug來進行調整，削強補弱讓遊戲更加平衡。而相對於現在的環境產生了不小的影響的，如果想要在排位開放後衝擊王者，以下這些結合改動分析的乾貨一定要看完哦。

  1. 新英雄 卡牌大師 崔斯特

再來看看雲頂中卡牌的屬性與思路

生命值：450 / 850 / 1700

法力值：50

護甲：20(降低16.7%)

魔抗：20(降低16.7%)

攻擊力：40 / 72 / 144

攻擊速度：0.75

攻擊距離：3格

技能：選牌（也就是我們常說的切牌了，不過是隨機的）

崔斯特扔出一張卡牌，黃牌傷害並暈眩目標 2/3/4 秒，紅牌傷害目標和周圍的敵人 100/150/200 點，藍牌回覆他自身和附近友軍的法力值 20/35/50 點。

定位：暫時評價為一個合格的打工者，從技能上解讀每一張牌，單體控制還可以，紅牌傷害太低，藍牌回藍作用不明顯，都不是那麼給力，最大的作用就是作為海盜，前期彌補了船長沒來之前的空白期，快速3海盜賺錢是一個新的走勢這點毋庸置疑，但是無論是中期的船長還是後期的好運姐都比卡牌實用的多，即使男槍的坦度也是超過卡牌的，而且因為是2費用，如果去追3星就沒什麼必要了，所以後期一般不會再上場，除非什麼海盜法師的野路子也許會用，所以這次的卡牌就是海盜流恰爛錢的福音，對於其他流派選擇的可能性不大。

下面介紹一下其他平衡改動，直接加套路分析，具體細節官網有公佈：

  1. 惡魔的影響

（1）法力燃燒機率：40/60/80% → 25/50/80%

避免了惡魔打工的局面，2惡魔基本沒用，這一改動就是為了讓惡魔不再成為陪襯。

（2）蜘蛛：生命和護甲都降低一點，一星和二星的小蜘蛛都減少了一隻，小蜘蛛繼承惡魔效果，所以小蜘蛛也可以法力燃燒，最重要的就是變成1費了，惡魔終於有1費了，前期打工也好，弄3星做基石都沒問題。

（3）劍魔：法力值消耗：100 → 75 傷害：350-850 → 400-1000真正的加強了一波。

（4）寡婦：生命值：550 → 600 處決血限：50% 生命值→ 65% 生命值 處決傷害倍數：300/400/500% → 300/500/700% 伊芙琳在施法期間(0.35秒)會將目標定身防止他們離開技能範圍。技能不會空了，非常大的加強，無論對於惡魔還是刺客都算增強。

（5）莫甘娜：當技能範圍內沒有敵人的時候不再會施放技能，傷害：250-450 → 300-450同樣不會空技能了，也成為惡魔中的必選牌了。

（6）火男： 法力值消耗：100 → 125，略微削弱，增加了放兩次大招的時間，影響不大，火男不做大哥的陣容一場只能放一次大招，火男大哥有裝備不在乎這點藍。

2.bug修復改動

（1）諾手：當技能範圍內沒有敵人的時候，不再會施放技能，所以必定吸血，小幅增強。

（2）狐狸： 修復了導致阿狸的技能在其陣亡後無法造成傷害的bug，現在目標在其施法時間內陣亡的話會重新選擇，目標技能目標選擇：隨機敵人→ 當前攻擊的目標。雲頂之奕第一假賽王的帽子終於可摘掉了。

（3）機器人：修復了一個導致布里茨在施法過程中陣亡會將單位拉出戰場的bug，現在附近的友軍更有可能選擇布里茨拉過來的單位作為目標，小優化，拉人過來圍毆就是了。

（4）布隆：修復了布隆的提示說明，使其能夠正確的反映出護盾接下來所減免所有傷害的百分比，傷害減免： 60-90% → 70-90%，法力值消耗：75 → 50，技能目標選擇： 最近的敵人→ 最遠的敵人。之前的布隆就像木樁一樣又笨又蠢，這回我們也許能看到打職業的布隆了。

（5）卡特：修復了一個導致卡特琳娜技能無法正確地降低敵人的治療效果的bug，不太重要。

（6）阿卡麗：修復了一系列會導致其技能偶爾無法造成傷害的bug，當技能範圍內沒有敵人的時候，不再會偶爾施放技能。不空技能，後排的脆皮們更要顫抖了。

（7）女槍：技能目標選擇：隨機敵人 → 當前攻擊目標，現在在準備動作過程中目標陣亡的話會重新選擇目標， 修復了一個導致厄運小姐偶爾會在施法時移動並改變大招的軌跡的bug，法力值消耗：75 → 100。沒實際操作一下不好說這個改動是否算改良。

（8）更新了魔法水晶箭彈道的視覺特效，魔法水晶箭現在會播放音效

修復了一個導致從共享選秀中合成升級的英雄有時會複製該英雄的bug

重新連接到雲頂之弈遊戲中不再會使得遊戲結束後的統計無法顯示

修復了一系列會導致近戰英雄原地不動以及停止移動的Bug

成噸的其他Bug修復已在上方列出

修復了一個BUG，該BUG曾導致：生命值裝備給所有2星單位提供了過多的生命值

修復了一個BUG，該BUG曾導致：2星和3星的換形師在變身後從所有來源獲得的生命值過多

其他的英雄改動比較小，就不再強調了。

3.裝備改動

裝備不再受到法術強度的加成，每個中立回合現在都會掉落獎勵。獎勵包括裝備組件，金幣和經驗值。PvE回合中，如果小兵或野怪沒有掉落裝備則會掉落金幣，真的是非酋的福音呢。 亞龍和遠古巨龍不再獲得龍類特質的收益，法師流玩家可以開始慶祝了。

（2）金鏟子：金剷剷現在會使得其相關的裝備組件屬性翻倍。例如：幽夢之靈提供40點攻擊力

（3）詛咒之刃：現在可以將目標降低至0星階，修復了一個導致詛咒之刃第二次觸發時會重置一名敵方英雄的星階的bug，新出了拉麵熊暴打小朋友玩法，攻速快的，帶aoe的都可以用用這個，效果出奇的好。

（4）復活甲：復活延遲：4 秒→ 2 秒，生命值回覆：500 → 100， Bug修復：現在守護天使會正確地清除重傷效果。這才是有價值的復活甲，給有格子的大哥來一個沒毛病。

（5）離子火花：現在會造成真實傷害而非魔法傷害，修復了一個導致離子火花會造成75點而非200點傷害的bug。這成為這一版本的神器，廣大玩家誰用誰知道呀。

（6）救贖：現在在25%生命值時觸發，不再對敵人造成傷害。終於不再是自殺救贖了，稍微有點用。

（7）盧安娜的颶風：箭矢數量：2 → 1， 箭矢傷害：50% → 25%，現在附帶普攻特效。很多騷套路這回玩不了了。

（8）大天使：雙大天使可以疊加了

（9）反甲：反彈傷害：所承受物理傷害的35%→所減免傷害的100%，傷害類型：魔法傷害→ 真實傷害。跟遊戲差不多，按照護甲和魔抗的強度來決定反彈傷害，如果是龍王或者龍女，那就變成法師剋星了。

（10）基克的先驅： 現在只為持有者左右兩格內的英雄提供攻速，攻擊速度光環：10% → 15%對於站一排的刺客來說每個人多了半個攻速弓，太合適了。
