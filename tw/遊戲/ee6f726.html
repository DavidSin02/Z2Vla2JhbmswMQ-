YEEZY BOOST 350 V2 全新地區限定發售日期確認

2020-10-29
| 遊戲


Nike 即將發佈全新 React Sertu 鞋款





自 Nike 的 React 技術在 2017 年率先在籃球鞋上應用之後，品牌接連推出了 React Element 87，React Element 55 等優秀鞋款。




而據可靠消息表明，Nike 即將為 React 家族加入新成員，以機能風格為主軸的全新 React Sertu 鞋款。




新的 React Sertu 在鞋面上運用多個區域縫製，以不同物料及製作手法建構，整體觀感變得更有細節。




據悉鞋款即將於 Nike 專門店及部分線上店鋪如 Sivasdescalzo 推出，售價約為美金 $169。

YEEZY BOOST 350 V2

全新地區限定發售日期確認





不久前 adidas Originals 於官方 IG 賬號揭示了YEEZY BOOST 350 V2全新地區限定系列的確切發售訊息。品牌將於 6 月 21 日搶先發布「Antlia」配色反光版本，6 月 22 日才發佈「Antlia」不反光版本，並僅在歐洲，俄羅斯和烏克蘭發售。




以灰粉色調登場的「Synth」配色，也將於同日期在非洲，亞洲，印度和中東地區發售，同樣具有反光及不反光版本。




至於以淺米色 Primeknit 物料構建鞋身的「Lundmark」配色，將會只在美洲地區限定發售，於 7 月 11 日搶先發售滿天星版本，於7 月 13 日大規模上架不反光版本。值得一提的是，三雙區域限定鞋款皆配備成人、大童、嬰兒三種尺碼版本。

Off-White™ x Air Jordan 1 有望發售兒童尺碼
