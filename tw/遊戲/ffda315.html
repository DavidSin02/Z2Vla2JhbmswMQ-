數據分析—缺失值處理

我們獲取到的數據（尤其是數據量比較大時），很有可能會出現數據缺失、數據異常等問題，數據處理是數據分析裡十分重要且必須的一個環節，為了在做數據分析時儘可能減少異常的出現，以及得出更精準的分析結論，因此在做數據分析之前，數據處理顯得尤其必要

閱讀條件：熟悉pandas基本操作

使用工具：Python

使用平臺：jupyter notebook

缺失值處理

缺失值判斷

python主要讀取的是csv或者excel數據，excel中單元格為空時，pandas讀取顯示的是NaN，即為缺失值

判斷缺失值數據方法：isnull，notnull

  • isnull：True表示缺失，False表示非缺失
  • notnull：True表示非缺失，False表示缺失

首先導入python數據分析必選工具包

import numpy as npimport pandas as pd__author__='莫嘆'複製代碼

生成一個表格型的二維數組df

#生成一個表格型的二維數組df=pd.DataFrame({'a':[34,6,20,np.nan,56], 'b':['','number','one','good',np.nan]})複製代碼

判斷數據中是否存在缺失值以及非缺失值數據篩選：

#判斷二維數組df是否缺失print(df.notnull(),'\n')#通過索引判斷a列是否缺失print(df['a'].notnull(),'\n')#篩選a列不存在缺失值數組print(df[df['a'].notnull()])複製代碼

缺失值刪除

通過上面的notnull布爾序列值篩選也是缺失值刪除的一種方法

缺失值的刪除需要按照具體數據情況和業務情況來處理，有時候需要刪除全部缺失數據，有時候需要刪除部分缺失數據，有時候只是需要刪除指定缺失數據。

刪除缺失值的方法：dropna(axis)

  • 默認參數axis = 0，為刪除行數據，當參數axis = 1刪除列數據（但是一般不會選擇axis=1，如果為1，則直接刪除了一整個變量數據）
  • 傳入thresh=n時保留至少有n個非NaN數據的行
#生成一個表格型二維數組df2=pd.DataFrame([[1,2,3],['',np.nan,np.nan],['a','b',np.nan],[np.nan,np.nan,np.nan],['d','j','h']], columns=list('ABC'))print(df2,'\n')#刪除所有存在缺失值的行print(df2.dropna(),'\n')#刪除部分存在缺失值的行，保留至少有n個非NaN數據的行(比如保留至少有一個非NaN數據的行)print(df2.dropna(thresh=1),'\n')#刪除某一列存在缺失值的行（刪除A列裡存在缺失值的所有的行，和上面的布爾值序列篩選相同）print(df2[df2['A'].notnull()])複製代碼

輸出如下：

缺失值填充/替換

填充方法：fillna(value=None, method=None, axis=None, inplace=False, limit=None, downcast=None, **kwargs)

  • value參數：填充值
  • method參數： pad / ffill → 用之前的數據填充，backfill / bfill → 用之後的數據填充

替換方法：replace(to_replace=None, value=None, inplace=False, limit=None, regex=False, method='pad', axis=None)

  • to_replacec參數：被替換的值
  • value參數：替換值

使用舉例如下：

import copydf3=pd.DataFrame([[1,2,3],['',np.nan,np.nan],['a','b',np.nan],['k',np.nan,np.nan],['d','j','h']], columns=list('ABC'))df4=copy.deepcopy(df3)print(df3,'\n')#缺失值全部用 0 填充print(df3.fillna(0),'\n')#method=‘pad’，使B列每一個缺失值用缺失值的前一個值填充df3['B'].fillna(method='pad',inplace=True)print(df3,'\n')#用replace替換print(df4,'\n')df4.replace(np.nan,'',inplace = True)print('將缺失值替換為\n',df4)複製代碼 A B C0 1 2 31 NaN NaN2 a b NaN3 k NaN NaN4 d j h  A B C0 1 2 31 0 02 a b 03 k 0 04 d j h  A B C0 1 2 31 2 NaN2 a b NaN3 k b NaN4 d j h  A B C0 1 2 31 NaN NaN2 a b NaN3 k NaN NaN4 d j h將缺失值替換為 A B C0 1 2 31 2 a b 3 k 4 d j h複製代碼

缺失值插值

上面講到缺失值的填充，但是在實際的數據處理過程中，對於缺失值的處理不是隨便找個數據全部填充，而是有針對性的對每個局部缺失值進行插值填充。

這裡選擇幾種常用的具有代表性的缺失值插值方法：

  • 中位數/眾數/平均值插值
  • 臨近值插值
  • 拉格朗日值插值

中位數/眾數/平均值插補

#生成一個一維數組s1=pd.Series([6,4,2,5,4,3,3,7,np.nan,3,9,np.nan,1])print(s1,'\n')med=s1.median()#中位數mod=s1.mode()#眾數avg=s1.mean() #平均值print('中位數,眾數，平均數分別為：%.2f,%.2f,%.2f'%(med,mod,avg))#以平均值為例s1.fillna(avg)複製代碼0 6.01 4.02 2.03 5.04 4.05 3.06 3.07 7.08 NaN9 3.010 9.011 NaN12 1.0dtype: float64 中位數,眾數，平均數分別為：4.00,3.00,4.270 6.0000001 4.0000002 2.0000003 5.0000004 4.0000005 3.0000006 3.0000007 7.0000008 4.2727279 3.00000010 9.00000011 4.27272712 1.000000dtype: float64複製代碼

臨近值插補

這點其實在上面缺失值填充裡提到了，主要是method這個參數，可選擇缺失值位置前一個或者和一個位置上數據填充相同的數據即可，可參考df3

fillna(value=None, method=None, axis=None, inplace=False, limit=None, downcast=None, **kwargs)

method參數： pad / ffill → 用之前的數據填充，backfill / bfill → 用之後的數據填充

拉格朗日插值法

許多實際問題中都用函數來表示某種內在聯繫或規律，而不少函數都只能通過實驗和觀測來了解。如對實踐中的某個物理量進行觀測，在若干個不同的地方得到相應的觀測值，拉格朗日插值法可以找到一個多項式，其恰好在各個觀測的點取到觀測到的值。這樣的多項式稱為拉格朗日（插值）多項式。數學上來說，拉格朗日插值法可以給出一個恰好穿過二維平面上若干個已知點的多項式函數。

我們隨意選擇一組數據（3,6）、（7,9）、（8,5）、（9,8）利用拉格朗日插值法算出過這些點的函數方程，然後輸入我們需要插值的x的值，自然可以得到y值。

#導入拉格朗日插值法計算和作圖包from scipy.interpolate import lagrangeimport matplotlib.pyplot as plt% matplotlib inline#任意創建一個有缺失值的二維數組s2=pd.DataFrame({'x':[3,7,12,8,9],'y':[6,9,np.nan,5,8]})#x值x=[3,7,8,9]#y值y=[6,9,5,8]#生成這幾個點的散點圖plt.scatter(x,y)#求出函數方程print(lagrange(x,y))#選擇一個x=12，計算插入值print('插值12為%.2f' % lagrange(x,y)(12))複製代碼
